# <a name="name-mbnprofile-element"></a>Элемент Name (Мбнпрофиле)

Элемент **Name (мбнпрофиле)** определяет имя профиля.

Это строка длиной до 255 символов.

Элемент является обязательным.

``` syntax
<xs:element name="Name"
    type="nameType"
 />
```

Элемент **Name** определяется элементом [**мбнпрофиле**](schema-mbnprofile-element.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|---------------------------------------------------|
| Минимальная версия клиента<br/> | \[Приложения UWP для классических приложений Windows 7 \|\]<br/> |
| Минимальная версия сервера<br/> | Ни одна версия не поддерживается<br/>                         |
