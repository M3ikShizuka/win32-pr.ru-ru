# <a name="providername-providertype-element"></a>Элемент ProviderName (providerType)

Элемент **providerName (ProviderType)** содержит имя сети сотовой связи.

Элемент — это строка, длина которой не может превышать 20 символов.

Этот элемент является необязательным.

``` syntax
<xs:element name="ProviderName"
    type="providerNameType"
 />
```

Элемент **providerName** определяется сложным типом [**ProviderType**](schema-providertype-complextype.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|---------------------------------------------------|
| Минимальная версия клиента<br/> | \[Приложения UWP для классических приложений Windows 7 \|\]<br/> |
| Минимальная версия сервера<br/> | Ни одна версия не поддерживается<br/>                         |
