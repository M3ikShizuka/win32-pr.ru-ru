# <a name="translating-to-vbscript"></a>Преобразование в VBScript

VBScript — это подмножество языка Microsoft Visual Basic для приложений. Поскольку VBScript предназначен для безопасного использования в клиентских приложениях, таких как веб-страницы, он не предоставляет входные и выходные данные файлов или прямой доступ к базовой операционной системе. Это предотвращает удаление или изменение файлов на компьютере с помощью веб-скрипта. полный список функций Visual Basic, которые не поддерживаются в VBScript, см. в документации по Visual Basic для приложений.

Помимо языковых возможностей, предоставляемых VBScript, сервер сценариев может также предоставлять объектную модель VBScript, к которой может обращаться ваш скрипт. Например, Internet Explorer предоставляет объектную модель, позволяющую сценариям взаимодействовать с браузером и программно управлять им. Дополнительные сведения о работе с такими объектными моделями см. в документации на сервере сценариев.
