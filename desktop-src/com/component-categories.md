# <a name="component-categories"></a>Категории компонентов

Категории компонентов OLE позволяют определять возможности программного компонента и требования, определяемые записями в реестре. В ситуации, когда контейнеру может не потребоваться поддерживать область функциональности, например DataBinding, в контейнере не нужно размещать элементы управления, требующие привязки данных для выполнения их работы. Категории компонентов позволяют идентифицировать области функций, таких как привязка данных, чтобы контейнер элемента управления мог не допустить, чтобы эти элементы управления запускали требование. Категории компонентов задаются отдельно как часть OLE и не относятся к архитектуре элементов управления ActiveX. Спецификация категорий компонентов включает интерфейсы API для управления разделами реестра категории компонентов.
