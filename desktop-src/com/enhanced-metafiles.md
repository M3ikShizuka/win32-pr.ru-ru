# <a name="enhanced-metafiles"></a>Расширенные метафайлы

Неудивительно, что расширенные метафайлы предоставляют больше функциональных возможностей, чем стандартные метафайлы; Использование расширенных метафайлов обычно упрощает код отрисовки. Контроллер домена расширенного метафайла используется точно так же, как и стандартный метафайл домена. OLE поддерживает расширенные метафайлы и обеспечивает обратную совместимость со стандартными метафайлами.

контейнеры элементов управления ActiveX должны использовать расширенные метафайлы вместо стандартных метафайлов.
