# <a name="debugobjectrpcenabled"></a>дебугобжектрпценаблед

**Дебугобжектрпценаблед** разрешает удаленную отладку COM через RPC. Если этот ключ не определен на сервере, удаленная отладка отключена.

## <a name="registry-key"></a>Ключ реестра

**HKEY \_ \_ \\ программное обеспечение на локальном компьютере \\ Microsoft \\ Windows NT \\ CurrentVersion \\ дебугобжектрпценаблед**



| Раздел реестра               | Описание                       |
|----------------------------|-----------------------------------|
| [**аедебуг**](aedebug.md) | Хранит параметры, относящиеся к отладчику. |
