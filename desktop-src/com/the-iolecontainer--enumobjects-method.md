# <a name="the-iolecontainerenumobjects-method"></a>Метод Иолеконтаинер:: Енумобжектс

Этот метод используется для перечисления всех объектов OLE, содержащихся в документе или форме, возвращающего указатель интерфейса для каждого объекта OLE. Контейнер должен возвращать указатели на каждый объект OLE, совместно использующий один и тот же контейнер. Вложенные формы и вложенные элементы управления также должны быть перечислены.

некоторые контейнеры реализуют расширяющие элементы управления, которые заключают в оболочку элементы управления, не относящиеся к ActiveX, а затем возвращают указатели на эти элементы управления расширителем при перечислении формы. такое поведение позволяет элементам управления ActiveX и ActiveX контейнерами хорошо интегрироваться с элементами управления, отличными от ActiveX, и поэтому рекомендуется, но не обязательно.
