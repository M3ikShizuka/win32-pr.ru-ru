# <a name="notifications-com"></a>Уведомления (COM)

Уведомления — это обратные вызовы, созданные объектом при обнаружении изменения его имени, состояния, данных или представления. Для контейнеров и других клиентов необходимо, чтобы уведомления отвечали соответствующим изменениям. Контейнер регистрируется для получения уведомлений путем настройки вспомогательного подключения к интересующему объекту. Другие заинтересованные клиенты могут сделать то же самое. Контейнер также создает приемник рекомендаций для получения уведомлений. При использовании соединения, установленного контейнером, объект, на котором происходит изменение, уведомляет приемник рекомендаций. При получении уведомления контейнер принимает любое действие, определенное для типа произошедших изменений.
