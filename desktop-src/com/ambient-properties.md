# <a name="ambient-properties"></a>Внешние свойства

Как минимум контейнеры элементов управления ActiveX должны поддерживать следующие внешние свойства с использованием стандартных идентификаторов DISPID.



| Внешнее свойство             | DISPID          | Комментарии и условия                                                      |
|------------------------------|-----------------|--------------------------------------------------------------------------|
| LocaleID<br/>          | — 705<br/> |   <br/>                                                          |
| Пользовательском<br/>          | — 709<br/> | Для контейнеров, имеющих разные пользователи и среды выполнения.<br/> |
| DisplayAsDefault <br/> | — 713<br/> | Для контейнеров, в которых имеется кнопка по умолчанию.<br/>      |
