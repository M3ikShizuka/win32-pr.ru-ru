# <a name="extended-properties-events-and-methods"></a>Расширенные свойства, события и методы

Для поддержки расширенных элементов управления контейнеры элементов управления ActiveX не требуются. Однако если контейнер элемента управления поддерживает расширенные свойства, он должен поддерживать следующий минимальный набор:

-   Видимый
-   Parent
-   Значение по умолчанию
-   Отменить

В настоящее время расширенные свойства, события и методы не имеют стандартных идентификаторов DISPID.
