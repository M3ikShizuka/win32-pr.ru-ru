# <a name="extended-properties-events-and-methods"></a>Расширенные свойства, события и методы

контейнеры элементов управления ActiveX не требуются для поддержки расширенных элементов управления. Однако если контейнер элемента управления поддерживает расширенные свойства, он должен поддерживать следующий минимальный набор:

-   Видимый
-   Parent
-   По умолчанию
-   Отменить

В настоящее время расширенные свойства, события и методы не имеют стандартных идентификаторов DISPID.
