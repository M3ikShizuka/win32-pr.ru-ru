# <a name="nummethods"></a>нуммесодс

Содержит число методов в связанном интерфейсе, включая методы из производных интерфейсов.

## <a name="registry-entry"></a>Запись реестра

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Interface
   {IID}
      NumMethods = value
```

## <a name="remarks"></a>Комментарии

Это значение **reg \_ SZ** .
