# <a name="exposing-methods-through-idispatch"></a>Предоставление доступа к методам через IDispatch

Хотя большинство элементов управления предоставляют и поддерживают несколько методов, элементы управления не обязательно должны предоставлять или поддерживать какие бы то ни было методы, поэтому элементу управления не требуется интерфейс **IDispatch**. Если у элемента управления есть какие-либо методы, нет никаких требований к методам, которые должен предоставить элемент управления.
