# <a name="turning-off-security"></a>Отключение безопасности

Существует несколько причин, по которым вы можете отключить безопасность, предоставляемую COM. При разработке приложений может быть полезно отключить систему безопасности изначально, чтобы вы могли сосредоточиться на других функциях в приложении. После этого можно добавить в проект функции безопасности, когда будете готовы. Для упрощения отладки может потребоваться отключить безопасность, а затем добавить функции безопасности по одной. Наконец, некоторые приложения не нуждаются в безопасности и могут работать быстрее, если проверки безопасности не выполняются.

В следующих разделах приводятся пошаговые инструкции.

-   [Отключение безопасности активации](turning-off-activation-security.md)
-   [Отключение безопасности вызовов](turning-off-call-security.md)
