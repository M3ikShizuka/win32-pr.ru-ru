# <a name="automatic-clipping"></a>Автоматическая обрезка

настоятельно рекомендуется, чтобы контейнер элемента управления ActiveX поддерживал автоматическую обрезку своих элементов управления. Это приведет к более эффективной работе для большинства элементов управления. Если автоматическая обрезка поддерживается, свойство окружения автоклипа должно поддерживаться и иметь значение **true**.

Автоматическая обрезка — это возможность контейнера, гарантирующая, что рисуемый выход элемента управления поступает только в текущую область отсечения контейнера. В контейнере, поддерживающем автоматическую обрезку, элемент управления может рисовать без учета его области отсечения, поскольку контейнер автоматически обрезает все прорисовки, которые происходят за пределами области элемента управления. Если контейнер не поддерживает автоматическую обрезку, элементы управления, созданные с помощью пакета CDK, будут создавать дополнительное родительское окно, если передается непустая область обрезки.
