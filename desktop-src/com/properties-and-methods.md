# <a name="properties-and-methods"></a>Свойства и методы

Как и любой объект OLE, элемент управления обеспечивает большую часть его функциональных возможностей посредством набора входящих интерфейсов со свойствами и методами.

Элемент управления предоставляет свойства и методы через OLE-автоматизацию, чтобы контейнеры могли обращаться к ним под управлением языка программирования, предоставляемого контейнером.

Для поддержки доступа к свойствам через пользовательский интерфейс элемент управления предоставляет страницы свойств, поддержку \_ свойств олеиверб, Просмотр свойств и привязку данных посредством изменения свойства нотфикатионс.

-   С помощью страниц свойств элемент управления может отображать его свойства независимо от его контейнера, если это необходимо.
-   При поддержке \_ свойств олеиверб элемент свойства отображается в меню контейнера. Затем конечный пользователь может выбрать элемент **Свойства** , чтобы просмотреть страницы свойств элемента управления и изменить свойства.
-   Каждый просмотр свойств поддерживает контейнер, который может отображать свойства элемента управления как часть более крупной страницы свойств, которая может включать свойства из нескольких элементов управления в контейнере.
-   С помощью уведомления об изменении свойства элемент управления может уведомлять Клиента о том, что его свойства изменились, позволяя клиенту выполнять все необходимые действия в результате.
