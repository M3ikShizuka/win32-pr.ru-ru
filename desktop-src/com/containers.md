# <a name="containers"></a>Контейнеры

контейнер элемента управления ActiveX — это контейнер OLE, который поддерживает следующие дополнительные функции:

-   Внедренные объекты из внутрипроцессного сервера
-   Активация на месте
-   ОЛЕМИСК \_ активатевхенвисибле
-   Обработка событий

ActiveX Контейнеры элементов управления должны обеспечивать поддержку всех этих функций.
