# <a name="containers"></a>Контейнеры

Контейнер элементов управления ActiveX — это контейнер OLE, который поддерживает следующие дополнительные функции:

-   Внедренные объекты из внутрипроцессного сервера
-   Активация на месте
-   ОЛЕМИСК \_ активатевхенвисибле
-   Обработка событий

Контейнеры элементов управления ActiveX должны обеспечивать поддержку всех этих функций.
