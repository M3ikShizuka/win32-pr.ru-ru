# <a name="message-reflection"></a>Отражение сообщений

Настоятельно рекомендуется, чтобы контейнер элементов управления ActiveX поддерживал отражение сообщений. Это приведет к более эффективной работе с подклассами элементов управления. Если поддерживается отражение сообщений, свойство окружения Мессажерефлект должно поддерживаться и иметь значение **true**. Если контейнер не реализует отражение сообщений, то компонент OLE CDK создает два окна для каждого элемента управления с подклассами, чтобы обеспечить отражение сообщений от имени контейнера элемента управления.
