# <a name="windowless-controls"></a>Элементы управления без окон

спецификация 96 элементов управления ActiveX включает определение для элементов управления без окон. такие элементы управления не работают в собственном окне и нуждаются в контейнере для предложения общего окна, в котором может отображаться элемент управления, см. ActiveX пакете SDK. Элементы управления без окон предназначены для совместимости с старыми контейнерами элементов управления, создавая собственное окно в этой ситуации, контейнеры элементов управления без окон должны размещать Оконные элементы управления традиционным образом без проблем. Однако может быть полезно, чтобы контейнер мог отличать те элементы управления, которые могут работать в режиме без окон, поэтому определена соответствующая категория компонента.

CATID-{1D06B600-3AE3-11cf-87B9-00AA006C8166} \_ ВИНДОВЛЕССОБЖЕКТ CATID
