# <a name="windowless-controls"></a>Элементы управления без окон

Спецификация элементов управления ActiveX 96 включает определение элементов управления без окон. Такие элементы управления не работают в собственном окне и нуждаются в контейнере, предлагающем общее окно, в котором может нарисоваться элемент управления, см. пакет SDK для ActiveX. Элементы управления без окон предназначены для совместимости с старыми контейнерами элементов управления, создавая собственное окно в этой ситуации, контейнеры элементов управления без окон должны размещать Оконные элементы управления традиционным образом без проблем. Однако может быть полезно, чтобы контейнер мог отличать те элементы управления, которые могут работать в режиме без окон, поэтому определена соответствующая категория компонента.

CATID-{1D06B600-3AE3-11cf-87B9-00AA006C8166} \_ ВИНДОВЛЕССОБЖЕКТ CATID
