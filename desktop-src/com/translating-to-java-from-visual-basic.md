# <a name="translating-to-java-from-visual-basic"></a>Перевод на Java с Visual Basic

по умолчанию Visual Basic передает параметры по ссылке. Параметры, которые должны передаваться только по значению, задаются с помощью ключевого слова **ByVal**.

Java не поддерживает многомерные массивы. Методы, которые принимают или возвращают многомерные массивы, недоступны в Java.

Java и Visual Basic немного отличаются в том, как они представляют свойства. В Java свойства представлены в виде набора функций доступа, который задает значение свойства и одно из них, которое получает значение свойства. в Visual Basic свойства представлены как один элемент, который можно использовать для получения или задания значения свойства.
