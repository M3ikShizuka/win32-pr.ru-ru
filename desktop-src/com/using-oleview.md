# <a name="using-oleview"></a>Использование OleView

В редакторе реестра отображается необработанное представление реестра, что полезно для изучения Организации реестра. Использование редактора реестра может занять много времени.

OleView предоставляет представление сведений, содержащихся в реестре более высокого уровня. Вместо того чтобы искать длинный список идентификаторов CLSID, в элементах управления "дерево" функций OleView используются понятные имена.
