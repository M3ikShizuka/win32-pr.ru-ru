# <a name="visual-basic-object-browser"></a>Обозреватель объектов Visual Basic

После добавления COM-объекта в проект в качестве ссылки или компонента можно использовать обозреватель объектов Visual Basic для просмотра сведений о библиотеке типов для этого объекта в Visual Basic.

Чтобы просмотреть COM-объект в обозревателе объектов, выполните следующие действия.

-   В меню **вид** выберите пункт **Обозреватель объектов**.

В левой части окна Обозреватель объектов отображаются все классы ссылок и компонентов, добавленных в текущий проект. В правой части содержатся методы, свойства и события этих классов.

Чтобы отфильтровать, какие классы COM-объектов отображаются, выберите библиотеку объектов из раскрывающегося списка Выбор библиотеки.

Дополнительные сведения см. в разделе [Просмотр библиотек типов компонентов ActiveX](/previous-versions/visualstudio/visual-basic-6/aa240782(v=vs.60)) в документации по Visual Basic.

В некоторых случаях в обозревателе объектов Visual Basic не отображается возвращаемый тип для функции или свойства. Возвращаемый тип не отображается, если тип возвращаемого значения является типом по умолчанию **Variant**. Если область описания в обозревателе объектов описывает элемент как функцию или свойство, а тип возвращаемого значения не показан, оператор возвращаемого типа имеет тип Variant.
