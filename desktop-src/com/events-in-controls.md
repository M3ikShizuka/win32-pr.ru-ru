# <a name="events-in-controls"></a>События в элементах управления

Хотя большинство элементов управления предоставляют и инициируют несколько событий, элементы управления не обязаны предоставлять или вызывать события, поэтому элемент управления не требует [**IConnectionPointContainer**](/windows/desktop/api/OCIdl/nn-ocidl-iconnectionpointcontainer). Если у элемента управления есть события, нет никаких требований к тому, какие события должен предоставить элемент управления.
