# <a name="aedebug"></a>аедебуг

**Аедебуг** определяет параметры удаленной отладки для com.

## <a name="registry-key"></a>Ключ реестра

**HKEY \_ \_ \\ программное обеспечение на локальном компьютере \\ Microsoft \\ Windows NT \\ CurrentVersion \\ дебугобжектрпценаблед \\ аедебуг**



| Значение реестра               | Описание                                           |
|------------------------------|-------------------------------------------------------|
| [**Автоматически**](auto.md)         | Определяет, запускается ли отладчик автоматически. |
| [**Отладчик**](debugger.md) | Хранит имя отладчика.                             |
