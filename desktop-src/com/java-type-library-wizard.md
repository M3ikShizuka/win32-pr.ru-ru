# <a name="java-type-library-wizard"></a>Мастер библиотеки типов Java

Мастер библиотеки типов Java — это компонент Visual J++ 5,0. Этот мастер создает файлы классов и сведения о библиотеке типов Java для COM-объекта.

**Создание файлов классов и сведений о библиотеке типов Java**

1.  В меню **Сервис** выберите пункт **Мастер библиотек типов Java**. Откроется список установленных компонентов COM.
2.  Установите флажки рядом с элементами, которые необходимо преобразовать.
3.  Нажмите кнопку **ОК**. Мастер библиотек типов создает файлы классов для выбранных компонентов. Он выводит инструкцию **Import** , необходимую для вставки каждого компонента в приложение или апплет Java, а также текстовый файл сводки, содержащий сведения о библиотеке типов Java для объекта.
