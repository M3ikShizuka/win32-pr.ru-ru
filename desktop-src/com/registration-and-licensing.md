# <a name="registration-and-licensing"></a>Регистрация и лицензирование

Элемент управления обычно предоставляется как внутрипроцессный сервер (.DLL), хотя он также может быть локальным или удаленным сервером (.EXE).

Элемент управления обычно поддерживает самостоятельную регистрацию и создает набор записей реестра при создании экземпляра. Элемент управления также может быть лицензирован для предотвращения несанкционированного использования. Дополнительные сведения о самостоятельной регистрации и лицензировании см. [в разделе Объектная модель компонентов](the-component-object-model.md) .

дополнительные сведения см. в разделе [ActiveX управление сведениями реестра](activex-controls-registry-information.md).
