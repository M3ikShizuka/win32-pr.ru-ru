# <a name="access-flags"></a>Флаги доступа

Эти значения указывают, являются ли записи в списке доступа описания разрешенных или запрещенных прав.



| Константа/значение                                                                                                                                                                                                                                      | Описание                                   |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------|
| <span id="ACTRL_ACCESS_ALLOWED"></span><span id="actrl_access_allowed"></span><dl> <dt>**Актрл \_ ДОСТУП \_ разрешен**</dt> <dt>0x00000000</dt> </dl> | Указывает запись, разрешенную доступом.<br/> |
| <span id="ACTRL_ACCESS_DENIED"></span><span id="actrl_access_denied"></span><dl> <dt>**Актрл \_ \_Отказано в доступе**</dt> <dt>0x10000000</dt> </dl>    | Указывает запись запрета доступа.<br/>  |
