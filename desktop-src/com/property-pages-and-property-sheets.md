# <a name="property-pages-and-property-sheets"></a>Страницы свойств и вкладки свойств

Страницы свойств OLE позволяют объекту отображать его свойства в диалоговом окне с вкладками, известном как страница свойств. Конечный пользователь может просматривать и изменять свойства объекта. Объект может отображать страницы свойств независимо от клиента, или клиент может управлять отображением страниц свойств из нескольких содержащихся объектов на одной странице свойств. Страницы свойств также предоставляют средства для уведомления клиента об изменениях в свойствах объекта.

Любой объект, который хочет предоставить пользовательский интерфейс для его изменяемых свойств, может использовать эту технологию.
