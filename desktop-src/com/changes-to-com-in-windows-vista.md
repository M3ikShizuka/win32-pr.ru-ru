# <a name="changes-to-com-in-windows-vista"></a>изменения в COM в Windows Vista

в Windows Vista были внесены следующие изменения в COM:

-   Специальное имя для повышения прав COM позволяет приложениям, выполняемым с ограниченной учетной записью пользователя (LUA), активировать классы COM с повышенными возможностями. Дополнительные сведения см. в описании [моникера повышения уровня com](the-com-elevation-moniker.md).
-   при установке приложения, выполняющего вызовы COM на компьютере Windows Vista, и планируется запускать приложение от имени администратора, необходимо установить приложение для каждого компьютера, а не для отдельного пользователя, или оно будет работать неправильно.
