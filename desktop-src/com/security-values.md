# <a name="security-values"></a>Значения безопасности

Несколько значений влияют на безопасность приложений COM, включая уровень проверки подлинности, уровень олицетворения, маскировку, удостоверение приложения, конфигурацию политики ограниченного использования программ и отслеживание ссылок.

Дополнительные сведения см. в следующих разделах:

-   [Уровень проверки подлинности](authentication-level.md) описывает, как управлять объемом защиты при обмене данными между COM-объектами.
-   [Делегирование и олицетворение](delegation-and-impersonation.md) объясняет, как маскировка влияет на удостоверение, представленное на сервере при олицетворении, и о том, как клиент предоставляет серверу полномочия, действующие от его имени.
-   [Удостоверение приложения](application-identity.md) определяет и описывает удостоверения, которые может использовать приложение.
-   [Политика ограниченного использования программ](software-restriction-policy.md) объясняет, как защитить систему от опасностей, связанных с выполнением неизвестного кода.
-   [Отслеживание ссылок](reference-tracking.md) объясняет, как можно гарантировать, что объекты не освобождаются слишком рано.
