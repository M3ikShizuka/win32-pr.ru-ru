# <a name="simple-data-binding"></a>простую привязку данных

Архитектура элементов управления ActiveX определяет механизм привязки данных, при котором элемент управления ActiveX может указывать, что одно или несколько его свойств могут быть привязаны. В большинстве случаев элемент управления с привязкой к данным не должен обязательно требовать привязки данных, чтобы его можно было вставить в контейнер, который не поддерживает привязку данных. Очевидно, что в такой ситуации функциональность элемента управления может снизиться.
