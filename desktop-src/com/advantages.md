# <a name="advantages"></a>Преимущества

Преимущества реализации приложения в качестве внутрипроцессного сервера ускоряются и объединяют некоторые преимущества обработчика объектов и локального сервера. Внутрипроцессный сервер работает быстрее по сравнению с локальными серверами по нескольким причинам. Во-первых, поскольку они меньше и выполняются в пространстве процесса приложения-контейнера, они загружаются быстрее. Во вторых, они оптимизируются для выполнения определенных задач. В третьих, взаимодействие между контейнером и сервером не зависит от удаленных вызовов процедур.
