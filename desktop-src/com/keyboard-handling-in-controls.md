# <a name="keyboard-handling-in-controls"></a>Обработка клавиатуры в элементах управления

Настоятельно рекомендуется поддержка обработки клавиатуры для следующих функций, хотя она распознает, что она неприменима ко всем контейнерам.

-   Поддержка \_ бит состояния олемиск актсликелабел и олемиск \_ актсликебуттон.
-   Реализация свойства окружения DisplayAsDefault (если оно существует, оно может возвращать **значение false**).
-   Реализация обработки вкладок, включая последовательность табуляции.

некоторые контейнеры будут использовать элементы управления ActiveX в традиционных сценариях составных документов. например, электронная таблица может позволить пользователю внедрить элемент управления ActiveX в лист. В таких сценариях контейнер будет выполнять обработку клавиатуры по-разному, так как интерфейс клавиатуры должен оставаться соответствующим ожиданиям пользователя электронной таблицы. Документация по таким продуктам должна информировать пользователей о различиях в обработке элементов управления в различных сценариях. Другие контейнеры должны правильно учитывать указанные выше функции, включая обработку назначенных клавиш.
