# <a name="ole-moniker-implementations"></a>Реализации моникера OLE

OLE предоставляет реализации нескольких моникеров в различных ситуациях, как описано в следующих разделах:

-   [Моникеры файлов](file-monikers.md)
-   [Составные моникеры](composite-monikers.md)
-   [Моникеры элементов](item-monikers.md)
-   [Специальные имена](anti-monikers.md)
-   [Моникеры указателей](pointer-monikers.md)
-   [Моникеры класса](class-monikers.md)
-   [Асинхронные моникеры](asynchronous-monikers.md)
-   [Моникеры URL-адресов](url-monikers.md)

Моникеры File, Composite и Item являются наиболее часто используемыми моникерами, так как их можно использовать для создания практически любого объекта в любом расположении. Специальные имена для защиты от моникеров и указателей в основном используются внутри OLE, но в них имеется какое-либо приложение для реализации пользовательских моникеров.
