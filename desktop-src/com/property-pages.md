# <a name="property-pages"></a>Страницы свойств

Настоятельно рекомендуется использовать поддержку страниц свойств и просмотр по свойствам, но это не обязательно. Если элемент управления реализует страницы свойств, то эти страницы должны соответствовать одному из стандартных размеров: 250x62 или 250x110 (Длус).
