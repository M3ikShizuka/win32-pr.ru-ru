# <a name="adding-an-activex-control-to-a-visual-c-project"></a>добавление элемента управления ActiveX в Visual C++ Project

следующая процедура добавляет файлы класса диспетчеризации и заголовков для элемента управления ActiveX в проект Visual C++.

**добавление элемента управления ActiveX в Visual C++ Project**

1.  в меню **Project** выберите команду **добавить в Project**. Появится контекстное меню.
2.  Щелкните **компоненты и элементы управления**. Откроется диалоговое окно **коллекция компонентов и элементов управления** .
3.  Щелкните компонент для добавления в проект. Visual C++ отображается диалоговое окно, из которого можно выбрать подмножество классов компонента для добавления в проект.
4.  Установите флажки для классов, которые необходимо добавить, и нажмите кнопку **ОК**.

Visual C++ создает файлы перенаправления и заголовков для компонента и добавляет их в проект.
