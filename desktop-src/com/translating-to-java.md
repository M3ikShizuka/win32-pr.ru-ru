# <a name="translating-to-java"></a>Перевод на Java

Система разработки Microsoft Visual J++ для Java предоставляет два средства, которые можно использовать для создания сведений о библиотеке типов для COM-объекта в синтаксисе Java, мастера библиотеки типов Java и программы командной строки Жактивекс. С помощью любого из этих средств можно создавать файлы классов Java для COM-объекта.

В Visual J++ версии 6,0 средство Жактивекс заменяет программу командной строки Жаватлб. Дополнительные сведения см. в документации по Visual J++.

Если вы используете среду разработки Java, отличную от Visual J++, обратитесь к документации продукта за информацией о средствах преобразования COM.

После создания файлов классов для компонента можно добавить классы компонентов в приложение Java с помощью инструкции **Import** .

Затем можно создать и использовать классы объекта в приложении Java. Дополнительные сведения об инструкции **импорта** Java см. в справочнике по языку программирования Java.
