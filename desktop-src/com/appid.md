# <a name="appid-appid-key"></a>AppID (ключ AppID)

Идентифицирует идентификатор GUID AppID, соответствующий именованному исполняемому файлу.

## <a name="registry-entry"></a>Запись реестра

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\AppID
   <Executable_name>
      AppID = {AppID_GUID}
```

## <a name="remarks"></a>Комментарии

Это значение **reg \_ SZ** .
