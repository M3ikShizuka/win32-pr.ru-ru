# <a name="multiple-controls-in-one-dll"></a>Несколько элементов управления в одной библиотеке DLL

Одна DLL-библиотека. ocx может быть контейнером любого количества элементов управления ActiveX, что упрощает распространение и использование набора связанных элементов управления.

При отправке нескольких элементов управления в одной библиотеке DLL обязательно включите имя поставщика в каждое имя элемента управления в пакете. Включение имен поставщиков в каждое имя элемента управления позволит пользователям легко находить элементы управления внутри пакета. Например, при отправке библиотеки DLL, реализующей три элемента управления, Con1, Con2 и Con3, имена элементов управления должны быть:

-   *Название вашей компании* Элемент управления Con1

-   *Название вашей компании* Элемент управления Con2

-   *Название вашей компании* Элемент управления Con3
