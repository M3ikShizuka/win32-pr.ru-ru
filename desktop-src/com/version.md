# <a name="version-com"></a>Версия (COM)

Указывает номер версии элемента управления.

## <a name="registry-entry"></a>Запись реестра

```
HKEY_LOCAL_MACHINE\SOFTWARE\Classes\CLSID
   {CLSID}
      Version = value
```

## <a name="remarks"></a>Комментарии

Это значение **reg \_ SZ** .

Номер версии должен совпадать с версией библиотеки типов, связанной с элементом управления.
