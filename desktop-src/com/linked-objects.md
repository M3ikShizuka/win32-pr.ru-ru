# <a name="linked-objects"></a>Связанные объекты

При вставке ссылки на объект в составной документ исходные данные или *источник связи* остаются в любом месте, где он был изначально создан, обычно в другом документе. Составной документ содержит только ссылку или *ссылку* на фактические данные, хранящиеся в источнике ссылки, а также сведения о том, как предоставить эти данные пользователю.

При активации ссылки запускается серверное приложение источника связи, которое требуется пользователю для редактирования или иным образом манипулировать данными связи. Связывание позволяет уменьшить размер составного документа. Он также полезен, если источник данных поддерживается другим пользователем и должен совместно использоваться несколькими пользователями. Если пользователь, обслуживающий источник связи, изменяет данные, это изменение автоматически обновляется во всех документах, содержащих ссылку на эти данные. Помимо создания простых ссылок, пользователи могут вкладывать ссылки и объединять связанные и внедренные объекты для создания сложных документов.
