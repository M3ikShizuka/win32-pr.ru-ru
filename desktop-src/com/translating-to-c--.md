# <a name="translating-to-c"></a>Преобразование в C++

Visual C++ предоставляет несколько средств, которые помогут в добавлении COM-объектов в проект. В проект можно добавить элемент управления ActiveX, использовать мастер классов MFC или запустить компилятор MIDL для IDL-файла.

Если вы используете среду разработки C++, отличную от Visual C++, обратитесь к документации продукта за информацией о средствах преобразования COM.

После добавления файлов включения или заглушки COM-объекта в проект приложение может использовать классы COM-объекта.

Visual C++ также предоставляет средство просмотра объектов OLE-COM, которое можно использовать для просмотра сведений о скомпилированной библиотеке типов. В средстве просмотра объектов представлены классы COM в синтаксисе C++.
