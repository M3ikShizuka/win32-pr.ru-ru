# <a name="ws_group-and-ws_tabstop-flags-in-controls"></a>\_Флаги WS Group и WS \_ TABSTOP в элементах управления

Элемент управления не должен использовать \_ Флаги WS Group и WS \_ TABSTOP внутренне. Некоторые контейнеры используют эти флаги для управления обработкой клавиатуры.
