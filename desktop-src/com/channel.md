# <a name="channel-com"></a>Канал (COM)

Канал отвечает за передачу всех сообщений между клиентом и объектом через границу процесса. Канал предназначен для прозрачного работы с различными типами каналов, совместим с использование DCE Standard RPC и поддерживает однопотоковые и многопоточные приложения.
