# <a name="control-methods"></a>Методы управления

Существует три стандартных метода, которые могут поддерживаться элементами управления: Refresh, DoClick и AboutBox. Эти стандартные методы имеют отрицательные значения DISPID, указывающие их стандартное состояние.
