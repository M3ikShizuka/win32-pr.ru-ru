# <a name="translating-to-visual-basic-from-java"></a>перевод в Visual Basic из Java

в Visual Basic в качестве внутренних типов данных используются совместимые с автоматизацией типы данных. Поэтому типы данных Java, совместимые с автоматизацией, также совместимы с Visual Basic. Типы данных, несовместимые с автоматизацией, не могут быть преобразованы в Visual Basic.

Java и Visual Basic немного отличаются в том, как они представляют свойства. В Java свойства представлены в виде набора функций доступа, который задает значение свойства и одно из них, которое получает значение свойства. в Visual Basic свойства представлены как один элемент, который можно использовать для получения или задания значения свойства.
