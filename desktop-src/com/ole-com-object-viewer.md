# <a name="ole-com-object-viewer"></a>Средство просмотра объектов OLE-COM

Средство просмотра объектов OLE-COM, Oleview.exe, — это приложение, предоставляемое Visual C++, которое отображает объекты COM, установленные на компьютере, и поддерживаемые ими интерфейсы. Это средство просмотра объектов можно использовать для просмотра библиотек типов.

Чтобы просмотреть библиотеку типов COM-объекта, выполните следующие действия.

1.  В меню **файл** средства просмотра объектов выберите пункт **Просмотреть typelib**. Откроется диалоговое окно **Открыть** .
2.  Укажите файл библиотеки типов, который нужно открыть, и нажмите кнопку **ОК**.

Средство просмотра объектов отображает интерфейсы объекта.

Дополнительные сведения см. в разделе [Использование средства просмотра объектов OLE/COM](/previous-versions/d0kh9f4c(v=vs.140)) в документации по Visual C++.
