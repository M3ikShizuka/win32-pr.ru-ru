# <a name="exposing-properties-through-idispatch"></a>Предоставление свойств через IDispatch

Хотя большинство элементов управления имеют свойства, элементы управления не обязательно должны предоставлять свойства, поэтому элементу управления не требуется интерфейс **IDispatch**. Если у элемента управления есть свойства, нет никаких требований к свойствам, которые должен предоставить элемент управления.
