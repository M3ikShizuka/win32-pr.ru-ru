# <a name="registry-hierarchy"></a>Иерархия реестра

Реестр структурирован как иерархия ключей, подразделов, именованных значений или отдельных значений по умолчанию. Подразделы могут иметь несколько подразделов и значений. Имена ключей именуются по строкам, разделенным обратной косой чертой. Каждый ключ в реестре может иметь одно или несколько значений, которые могут содержать строки, целочисленные значения или двоичные данные.
