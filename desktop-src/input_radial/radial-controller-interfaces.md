# <a name="radial-controller-interfaces"></a>Интерфейсы радиального контроллера

Разделы, содержащиеся в этом разделе, содержат справочные спецификации для интерфейсов [ввода радиального контроллера](radialcontroller-portal.md) .

## <a name="in-this-section"></a>В этом разделе

| Раздел | Описание |
|---|---|
| [**IRadialControllerConfigurationInterop**](/windows/win32/api/radialcontrollerinterop/nn-radialcontrollerinterop-iradialcontrollerconfigurationinterop)<br/> | обеспечивает взаимодействие с объектом универсальная платформа Windows (UWP) [**радиалконтроллерконфигуратион**](/uwp/api/Windows.UI.Input.RadialControllerConfiguration) и предоставляет доступ к членам **радиалконтроллерконфигуратион** для настройки меню [**радиалконтроллер**](/uwp/api/Windows.UI.Input.RadialController) .<br/> |
| [**IRadialControllerInterop**](/windows/win32/api/radialcontrollerinterop/nn-radialcontrollerinterop-iradialcontrollerinterop)<br/> | Обеспечивает взаимодействие с объектом [**РАДИАЛКОНТРОЛЛЕР**](/uwp/api/Windows.UI.Input.RadialController) UWP и предоставляет доступ к членам **радиалконтроллер** для настройки взаимодействия.<br/> |
