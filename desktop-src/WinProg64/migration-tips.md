# <a name="migration-tips"></a>Советы по миграции

При проверке кода на совместимость с 64-разрядной архитектурой следует учитывать два основных аспекта:

-   Вычисления адресов
-   Расчеты указателей

По многим причинам, разработчики могут хранить адреса в виде значения типа **ulong** . в конце концов, в 32-бит Windows, адрес, указатель и значение **ULONG** имеют длину 32 бит. однако в 64-разрядной Windows длина адреса и **ULONG** не одинакова. Хотя **ulong** остается 32-битным значением, все указатели теперь являются 64-битовыми значениями.
