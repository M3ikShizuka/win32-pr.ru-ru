# <a name="migration-tips"></a>Советы по миграции

При проверке кода на совместимость с 64-разрядной архитектурой следует учитывать два основных аспекта:

-   Вычисления адресов
-   Расчеты указателей

По многим причинам, разработчики могут хранить адреса в виде значения типа **ulong** . В конце концов, в 32-разрядных Windows адрес, указатель и значение **ulong** имеют длину 32 бит. Однако в 64-разрядной Windows длина адреса и **ulong** не одинакова. Хотя **ulong** остается 32-битным значением, все указатели теперь являются 64-битовыми значениями.
