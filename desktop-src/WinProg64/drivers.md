# <a name="drivers-programming-guide-for-64-bit-windows"></a>Драйверы (руководством по программированию для 64-разрядных Windows)

64-разрядная версия Windows предназначена для того, чтобы разработчики использовали одну базу исходного кода для своих 32-разрядных и 64-разрядных приложений. в большой степени это также справедливо для 32-разрядных и 64-разрядных Windows драйверов.

однако 32-разрядные драйверы не могут выполняться на 64-бит Windows и должны быть перенесены. для приложений пользовательского режима 64-разрядная Windows включает WOW64, который позволяет выполнять 32-разрядные Windows приложений (с потерей производительности) в системах, использующих 64-bit Windows. Для драйверов не существует эквивалентного слоя перевода.

сведения о переносе драйверов в 64-разрядный Windows см. в статье [64-разрядные проблемы](https://msdn.microsoft.com/library/aa489627.aspx) в Windowsном наборе драйверов (WDK).
