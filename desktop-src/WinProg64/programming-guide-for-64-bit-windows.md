# <a name="programming-guide-for-64-bit-windows"></a>Руководство по программированию для 64-разрядных версий Windows

Корпорация Майкрософт выпустила 64-разрядные версии операционной системы Windows. 64-разрядная версия Windows была разработана с учетом совместимости. Разработчики могут убедиться, что их существующие 32-разрядные приложения работают под управлением 64-разрядной версии Windows, или воспользоваться преимуществами 64-разрядных Windows путем переноса приложений.

## <a name="benefits-of-64-bit-windows"></a>Преимущества 64-разрядной версии Windows

64-разрядная операционная система поддерживает гораздо больше физической памяти, чем 32-разрядная операционная система. Например, большинство 32-разрядных систем Windows поддерживают не более 4 гигабайт физической памяти (до 3 ГБ адресного пространства для каждого процесса), а 64-разрядная версия Windows поддерживает до 2 терабайт физической памяти с 8 терабайтами адресного пространства для каждого процесса. Увеличенная физическая память включает следующие преимущества для приложений:

-   Каждое приложение может поддерживать больше пользователей. Все или часть каждого приложения должна быть реплицирована для каждого пользователя, что требует дополнительной памяти.
-   Каждое приложение имеет лучшую производительность. Увеличение физической памяти позволяет запускать больше приложений одновременно и оставаться полностью резидентным в основной памяти системы. Это сокращает или исключает снижение производительности при переключении страниц на диск и с диска.
-   Каждое приложение имеет больше памяти для хранения и обработки данных. Базы данных могут хранить больше данных в физической памяти системы. Доступ к данным выполняется быстрее, так как чтение с диска не требуется.
-   Приложения могут легко и надежно манипулировать большими объемами данных. По этой причине композиция видео для работы с изображением движения должна иметь 64-разрядную версию Windows. Моделирование для научных и финансовых приложений значительно выгодно из структур данных, находящихся в памяти, которые не поддерживаются в 32-разрядной версии Windows.

Существуют также важные преимущества для предприятий.

-   Повышенная производительность. Специалисты по работе с знаниями могут потратить время на обдумывание и создание, а не ожидают, пока программное обеспечение завершит свои задачи.
-   Более низкая стоимость владения. Каждый сервер может поддерживать большее количество пользователей и приложений, поэтому для вашего бизнеса потребуется меньше серверов. Это приводит к снижению расходов на управление — одной из самых высоких затрат в любой вычислительной среде.
-   Новые возможности приложений. Новые приложения могут быть разработаны без барьеров, накладываемых 32-разрядной системой Windows. Новые графические приложения сделают работу более простой и приятной. В 64-разрядных Windows задачи, ресурсоемкие с большим объемом данных, можно сделать недоступными.
