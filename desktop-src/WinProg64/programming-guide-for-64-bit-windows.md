# <a name="programming-guide-for-64-bit-windows"></a>Руководство по программированию для 64-разрядных версий Windows

корпорация майкрософт выпустила 64-разрядные версии операционной системы Windows. 64-разрядная Windows была разработана с учетом совместимости. разработчики могут гарантировать, что их существующие 32-разрядные приложения хорошо работают в 64-разрядной Windows или воспользоваться преимуществами 64-bit Windows путем переноса приложений.

## <a name="benefits-of-64-bit-windows"></a>Преимущества 64-разрядных Windows

64-разрядная операционная система поддерживает гораздо больше физической памяти, чем 32-разрядная операционная система. например, большинство 32-разрядных Windows систем поддерживают не более 4 гигабайт физической памяти (до 3 гб адресного пространства для каждого процесса), а 64-бит Windows поддерживает до 2 терабайт физической памяти с 8 терабайтами адресного пространства для каждого процесса. Увеличенная физическая память включает следующие преимущества для приложений:

-   Каждое приложение может поддерживать больше пользователей. Все или часть каждого приложения должна быть реплицирована для каждого пользователя, что требует дополнительной памяти.
-   Каждое приложение имеет лучшую производительность. Увеличение физической памяти позволяет запускать больше приложений одновременно и оставаться полностью резидентным в основной памяти системы. Это сокращает или исключает снижение производительности при переключении страниц на диск и с диска.
-   Каждое приложение имеет больше памяти для хранения и обработки данных. Базы данных могут хранить больше данных в физической памяти системы. Доступ к данным выполняется быстрее, так как чтение с диска не требуется.
-   Приложения могут легко и надежно манипулировать большими объемами данных. композиция видео для работы с изображением движения требует 64-разрядного Windows по этой причине. Моделирование для научных и финансовых приложений значительно выгодно из структур данных, находящихся в памяти, которые не поддерживаются в 32-разрядных Windows.

Существуют также важные преимущества для предприятий.

-   Повышенная производительность. Специалисты по работе с знаниями могут потратить время на обдумывание и создание, а не ожидают, пока программное обеспечение завершит свои задачи.
-   Более низкая стоимость владения. Каждый сервер может поддерживать большее количество пользователей и приложений, поэтому для вашего бизнеса потребуется меньше серверов. Это приводит к снижению расходов на управление — одной из самых высоких затрат в любой вычислительной среде.
-   Новые возможности приложений. Новые приложения могут быть разработаны без барьеров, накладываемых на 32-разрядные Windows. Новые графические приложения сделают работу более простой и приятной. Задачи с ресурсоемкими данными, которые невозможно выполнить сегодня, можно выполнять с 64-битным Windows.
