# <a name="preparing-your-application-for-64-bit-windows"></a>Подготовка приложения для 64-разрядных Windows

Существует несколько функций, упрощающих разработку приложений, которые будут работать как в 32, так и в 64-разрядной Windows. Большинство из них, например новые типы данных, описаны в разделе Подготовка к [64-разрядным Windows](getting-ready-for-64-bit-windows.md).

64-разрядный набор средств, поставляемый с Windows SDK, включает в себя 64-разрядный компилятор MIDL, Midl.exe для создания собственных 64-разрядных заглушек, а также 32-разрядные заглушки. Используйте коммутатор **/env Win64** для создания только 64-разрядных заглушек. По умолчанию создаются двойные заглушки, которые работают на обеих платформах.

Обратите внимание, что 64-разрядный MIDL поддерживает только режимы оптимизации [**/Oicf**](/windows/desktop/Midl/-oi) и [**/OS**](/windows/desktop/Midl/-os) .
