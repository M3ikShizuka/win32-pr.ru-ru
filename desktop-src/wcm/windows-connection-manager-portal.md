# <a name="windows-connection-manager"></a>Диспетчер подключений Windows

## <a name="purpose"></a>Назначение

Windows Диспетчер подключений (WCM) позволяет создавать и настраивать программное обеспечение диспетчера подключений.

## <a name="developer-audience"></a>Аудитория разработчиков

Windows Диспетчер соединений предназначен для разработчиков, использующих программное обеспечение для разработки на C/C++.

## <a name="run-time-requirements"></a>Требования к среде выполнения

API диспетчера подключений Windows поддерживается в Windows 8 и Windows Server 2012.
