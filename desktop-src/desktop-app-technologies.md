# <a name="desktop-app-technologies"></a>Технологии классических приложений

в этом разделе содержатся подробные инструкции и примеры кода, посвященные Windows функциям, доступным для классических приложений с помощью API Win32.

## <a name="in-this-section"></a>В этом разделе

| Раздел | Описание  |  
|----------------------------------|---|
| [Специальные возможности](accessibility/accessibility.md) | предоставляет рекомендации для Windows разработчиков, создающих доступные приложения, разработчики вспомогательных технологий, такие как средства чтения с экрана и экранные лупы, а также инженеры по тестированию программного обеспечения, создающие автоматизированные сценарии для тестирования Windows приложений. |
| [Пользовательский интерфейс классического приложения](windows-application-ui-development.md) | Содержит сведения, позволяющие разрабатывать графические интерфейсы пользователя для приложений, включая такие функции, как Windows и сообщения, ресурсы и элементы управления. |
| [Среда рабочего стола](user-interface.md) | содержит рекомендации по интеграции и расширению функций Windows настольных пользователей, включая панель задач, рабочий стол и проводник. |
| [Установка и обслуживание приложения](application-installing-and-servicing.md) | содержит сведения об использовании интерфейсов api и служб, предоставляемых Windows для установки и обслуживания классических приложений, а также для управления ими. |
| [Звук и видео](audio-and-video.md) | Содержит рекомендации по использованию аудио и видео функций, предоставляемых Windows. |
| [Доступ к данным и хранилище](data-access-and-storage.md) | Содержит сведения о функциях доступа к данным и хранилища, которые можно использовать в приложениях для настольных систем, включая Управление файловой системой и облачные модули синхронизации.  |
| [Устройства](devices.md) | Описание интерфейсов API для взаимодействия с устройствами и датчиками. |
| [Диагностика](diagnostics.md) | Содержит рекомендации по отладке и обработке ошибок, профилированию производительности, мониторингу сети и другим функциям диагностики. |
| [Документы и печать](printdocs/documents-and-printing.md) | описание документов и функций печати Windows, позволяющих приложениям сохранять, просматривать и печатать.  |
| [Графика и игры](graphics-and-multimedia.md) | содержит сведения о графических и игровых возможностях Windows, включая DirectX и digital imaging.  |
| [Сеть и Интернет](networking.md) | содержит рекомендации по использованию сетевых и интернет-функций Windows, включая управление сетью, api-интерфейсы HTTP и удаленный вызов процедур (RPC). |
| [Безопасность и удостоверения](security.md) | Содержит сведения о проверке подлинности, авторизации, шифровании и других функциях безопасности Windows. |
| [Системные службы](system-services.md) | Предоставляет рекомендации по основным функциям ОС, таким как процессы и потоки, службы, библиотеки динамической компоновки, COM, реестр и многое другое. |
| [Искусственный интеллект и машинное обучение](/windows/ai/) | Преобразуйте свое приложение Windows с помощью возможностей искусственного интеллекта. Средства ИИ Windows позволят вам и вашей компании достичь высоких результатов с помощью интеллектуальных решений для выполнения сложных задач. |
