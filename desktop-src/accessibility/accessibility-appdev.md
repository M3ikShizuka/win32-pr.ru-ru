# <a name="developing-accessible-applications-for-windows"></a>Разработка приложений со специальными возможностями для Windows

Каждая платформа Windows Application Framework поддерживает множество специальных возможностей, упрощающих работу пользователей с ограниченными возможностями, личными настройками или конкретными стилями работы для успешного использования любого выбранного вами устройства Windows.

В этом обзоре описываются специальные возможности, поддерживаемые в каждой из платформ, а также способы их внедрения при создании приложения.
