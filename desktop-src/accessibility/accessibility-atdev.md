# <a name="developing-assistive-technology-for-windows"></a>Разработка вспомогательных технологий для Windows

Сторонние технологические продукты, такие как средства чтения с экрана, экранные лупы и специальность оборудования, являются важными инструментами для многих пользователей Windows (и других продуктов Майкрософт).

В этом обзоре описывается создание средств поддержки специальных возможностей, совместимых с продуктами Майкрософт, для людей с видением, обучением, подвижностью и мобильностью, а также с ограничениями на языки и связи.
