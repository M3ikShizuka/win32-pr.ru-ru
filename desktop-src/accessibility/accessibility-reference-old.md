# <a name="windows-accessibility-and-automation-reference"></a>справочник по специальным возможностям и автоматизации Windows

Windows специальные возможности поддерживают две категории функций, которые помогают Windows разработчикам проектировать доступные приложения, разработчики вспомогательных технологий создают средства, такие как средства чтения с экрана и экранные лупы, а инженеры по тестированию программного обеспечения создают автоматизированные сценарии для тестирования Windows приложений: параметры пользователя и полный набор api-интерфейсов Win32.

дополнительные сведения см. в разделе [Windows специальных](../WinAuto/about-windows-accessibility-features.md)возможностей.
