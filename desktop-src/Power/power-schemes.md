# <a name="power-schemes"></a>Схемы управления питанием

Схема управления питанием — это набор параметров, управляющих энергопотреблением компьютера. Схемы управления питанием можно использовать для снижения энергопотребления отдельных устройств или всей системы. Можно настроить дополнительные функции, такие как поддержка спящего режима, время бездействия системы перед выключением отображения или состояние сна, используемое при истечении времени простоя системы.

Существует несколько способов создания схем управления питанием: приложение "Параметры электропитания" на панели управления, средство командной строки PowerCfg.exe и API схемы управления питанием. Дополнительные сведения о параметрах питания или PowerCfg.exe см. в справке, входящей в состав операционной системы.

В следующих разделах описывается поддержка схемы управления питанием, предоставляемая API схемы управления питанием.
