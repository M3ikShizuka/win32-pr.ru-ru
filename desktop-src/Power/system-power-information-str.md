# <a name="system_power_information-structure"></a>\_Структура системной \_ информации о питании

Содержит сведения о время простояе системы.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _SYSTEM_POWER_INFORMATION {
  ULONG MaxIdlenessAllowed;
  ULONG Idleness;
  ULONG TimeRemaining;
  UCHAR CoolingMode;
} SYSTEM_POWER_INFORMATION, *PSYSTEM_POWER_INFORMATION;
```



## <a name="members"></a>Члены

<dl> <dt>

**максидленессалловед**
</dt> <dd>

Время простоя, в котором система считается бездействующей, а время ожидания простоя начинается с учета, выраженное в процентах. Удаление ниже этого номера приводит к отмене таймера.

</dd> <dt>

**Неактивность**
</dt> <dd>

Текущий уровень простоя, выраженный в процентах.

</dd> <dt>

**TimeRemaining**
</dt> <dd>

Время, оставшееся до времени простоя в секундах.

</dd> <dt>

**кулингмоде**
</dt> <dd>

Текущий режим охлаждения системы. Этот элемент должен иметь одно из следующих значений.



| Значение                                                                                                                                                                                                                                 | Значение                                                                                                   |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------|
| <span id="PO_TZ_ACTIVE"></span><span id="po_tz_active"></span><dl> <dt>**PO \_ \_Неактивное**</dt> значение <dt>0</dt> </dl>                    | Сейчас система находится в активном режиме охлаждения.<br/>                                                |
| <span id="PO_TZ_INVALID_MODE"></span><span id="po_tz_invalid_mode"></span><dl> <dt>**PO \_ \_Недопустимый \_ режим**</dt> <dt>2</dt> </dl> | Система не поддерживает регулирование использования ЦП, или в системе не определена тепловая зона.<br/> |
| <span id="PO_TZ_PASSIVE"></span><span id="po_tz_passive"></span><dl> <dt>**PO \_ Режим \_ passive**</dt> <dt>1</dt> </dl>                 | В настоящее время система находится в режиме пассивного охлаждения.<br/>                                               |



 

</dd> </dl>
