# <a name="backlight-control-interface"></a>Интерфейс управления подсветкой

Интерфейс управления «подсветкой» — это стандартизированный интерфейс IOCTL для управления яркостью ЖК-подсветки.

Приложения, которым требуется программное управление яркостью подсветки или предоставлением пользователям элементов управления, следует использовать этот интерфейс, а не собственный интерфейс. в противном случае система не сможет запросить текущую аппаратную яркость и может стать несинхронизированной.

Первым шагом является запрос устройства для поддержки яркости с помощью [**\_ \_ Видеозапроса ioctl \_ поддерживаемого \_**](ioctl-video-query-supported-brightness.md) управляющего кода. Эта операция возвращает буфер, указывающий доступные уровни яркости. Затем можно запросить устройство для использования текущей яркости экрана с помощью управляющего кода, определяющего [**\_ \_ \_ \_ яркость экрана запроса видео ioctl**](ioctl-video-query-display-brightness.md) . Эта операция возвращает текущие параметры для чередования текущей (AC) яркости, прямого тока (DC) и состояния электропитания.

Чтобы изменить яркость экрана, используйте управляющий код для [**\_ \_ настройки \_ \_ яркости экрана**](ioctl-video-set-display-brightness.md) с помощью ioctl. Можно задать яркость AC, яркость контроллера домена или и то, и другое.
