# <a name="power-management-control-codes"></a>Управляющие коды управления питанием

С [батареями](battery-information.md)используются следующие управляющие коды.



| Контрольный код                                                                  | Описание                                        |
|-------------------------------------------------------------------------------|----------------------------------------------------|
| [**\_ \_ сведения о запросе от аккумулятора ioctl \_**](ioctl-battery-query-information.md) | Получает разнообразные сведения о батарее |
| [**\_ \_ состояние запроса аккумулятора \_ ioctl**](ioctl-battery-query-status.md)           | Извлекает текущее состояние аккумулятора.      |
| [**\_тег запроса на батарею ioctl \_ \_**](ioctl-battery-query-tag.md)                 | Извлекает текущий тег батареи.               |
| [**\_ \_ сведения о настройке аккумулятора ioctl \_**](ioctl-battery-set-information.md)     | Задает различные сведения о батарее.                  |



 

Следующие управляющие коды являются частью [интерфейса элемента управления "подсветка](backlight-control-interface.md)".



| Контрольный код                                                                                 | Описание                                                                   |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**\_ \_ \_ яркость экрана запроса видео \_ ioctl**](ioctl-video-query-display-brightness.md)     | Извлекает текущие уровни подсветки AC и DC и текущее состояние электропитания. |
| [**видеозапрос IOCTL — \_ \_ \_ поддерживаемая \_ яркость**](ioctl-video-query-supported-brightness.md) | Извлекает Поддерживаемые уровни подсветки.                                     |
| [**\_ \_ Настройка \_ яркости экрана для видео ioctl \_**](ioctl-video-set-display-brightness.md)         | Задает текущие уровни подсветки AC и DC                                   |
