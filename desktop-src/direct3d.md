# <a name="direct3d"></a>Direct3D

Direct3D — это интерфейс API низкого уровня для рисования примитивов с помощью конвейера отрисовки или для выполнения параллельных операций с помощью COMPUTE Shader. Дополнительные сведения см. в приведенном ниже содержимом.

Сведения о получении и установке Direct3D см. в разделе [Установка среды программирования Direct3D 12](./direct3d12/directx-12-programming-environment-set-up.md).

| [Графика Direct3D 12](./direct3d12/direct3d-12-graphics.md) | Direct3D 12 предоставляет API и платформу, позволяющие вашему приложению воспользоваться преимуществами графических и вычислительных возможностей компьютеров, оснащенных одним или несколькими GPU, совместимыми с Direct3D 12. |
