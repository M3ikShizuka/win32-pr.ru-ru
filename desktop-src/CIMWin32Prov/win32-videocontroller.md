# <a name="win32_videocontroller-class"></a>\_Класс Win32 видеоконтроллер

[Класс WMI](../wmisdk/retrieving-a-class.md) **\_ Видеоконтроллер для Win32** представляет возможности и Управление емкостью видеоконтроллера в компьютерной системе, на которой работает Windows.

оборудование, несовместимое с Windows модели драйвера (WDDM), возвращает неточные значения свойств для экземпляров этого класса.

Следующий пример синтаксиса — упрощенный MOF-код, который включает все наследуемые свойства. Свойства перечислены в алфавитном порядке, а не в MOF.
