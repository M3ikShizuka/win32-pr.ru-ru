# <a name="interrogateservice-method-of-the-win32_service-class-cimwin32-wmi-providers"></a>Метод Интеррогатесервице класса Win32_Service (поставщики WMI CIMWin32)

Метод  [класса WMI](/windows/desktop/WmiSdk/retrieving-a-class) интеррогатесервице запрашивает у службы, на которую указывает ссылка, обновление своего состояния Service Manager.

В этом разделе используется синтаксис MOF-файл (MOF). Дополнительные сведения об использовании этого метода см. [в разделе вызов метода](/windows/desktop/WmiSdk/calling-a-method).

## <a name="syntax"></a>Синтаксис


```mof
uint32 InterrogateService();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.
