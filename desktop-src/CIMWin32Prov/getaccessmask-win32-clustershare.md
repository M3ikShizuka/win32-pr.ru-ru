# <a name="getaccessmask-method-of-the-win32_clustershare-class"></a>Метод Жетакцессмаск \_ класса Win32 клустершаре

Возвращает битовую карту **UInt32** с правами доступа к общему ресурсу, удерживаемому пользователем или группой, от имени которой возвращается экземпляр.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetAccessMask();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Права доступа к общему ресурсу, удерживаемому пользователем или группой.
