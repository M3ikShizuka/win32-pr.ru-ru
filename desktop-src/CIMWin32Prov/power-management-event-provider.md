# <a name="power-management-event-provider"></a>Поставщик событий управления питанием

Поставщик управления питанием позволяет инструментарию WMI моделировать протоколы управления питанием Windows. Как поставщик событий, поставщик управления питанием предоставляет сведения классу [**Win32 \_ поверманажементевент**](win32-powermanagementevent.md) для описания событий управления питанием, вызванных изменениями состояния электропитания. Эти изменения состояния связаны с протоколами управления системой расширенного управления питанием (APM) или интерфейсом ACPI. Имя экземпляра [**\_ \_ Win32Provider**](/windows/desktop/WmiSdk/--win32provider) — « \_ \_ поставщик событий MS Power \_ Management \_ ».
