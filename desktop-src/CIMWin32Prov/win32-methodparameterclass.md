# <a name="win32_methodparameterclass-class"></a>\_Класс Win32 месодпараметеркласс

Абстрактный **\_ месодпараметеркласс Win32** , базовый [класс WMI](/windows/desktop/WmiSdk/retrieving-a-class) наследует любой класс, определенный как контейнер значений параметров, которые будут переданы в метод. Не имея собственных свойств, этот класс выступает в качестве узла классификации. Аналогичным примером является [**CIM \_ фисикалелемент**](cim-physicalelement.md). Наследование от **CIM \_ фисикалелемент** указывает, что класс описывает физическое, а не логическую сущность. В случае с **Win32 \_ месодпараметеркласс** классы, производные от него, создаются специально для передачи параметров в метод.

## <a name="members"></a>Участники

Класс **Win32 \_ месодпараметеркласс** не определяет никаких членов.
