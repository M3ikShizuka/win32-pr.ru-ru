# <a name="rename-method-of-the-cim_devicefile-class"></a>Метод Rename класса CIM \_ девицефиле

Метод **Rename** переименовывает файл устройства (или каталог), указанный в пути объекта. Переименование не поддерживается, если место назначения находится на другом диске или требуется перезапись существующего логического файла. Этот метод наследуется от [**CIM \_ LogicalFile**](cim-logicalfile.md).

> [!IMPORTANT]
> Классы CIM (модель CIM) в DMTF (распределенная задача управления) являются родительскими классами, на которых строятся классы WMI. В настоящее время WMI поддерживает только [схемы версии CIM 2. x](https://dmtf.org/standards/cim/schemas).

 

В этом разделе используется синтаксис MOF-файл (MOF). Дополнительные сведения об использовании этого метода см. [в разделе вызов метода](/windows/desktop/WmiSdk/calling-a-method).

## <a name="syntax"></a>Синтаксис


```mof
uint32 Rename(
  [in] string FileName
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Имя файла* \[ окне\]
</dt> <dd>

Полное новое имя файла (или каталога).

Пример: "c: \\ temp \\newfile.txt"

</dd> </dl>
