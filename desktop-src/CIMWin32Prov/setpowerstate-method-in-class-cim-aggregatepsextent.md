# <a name="setpowerstate-method-of-the-cim_aggregatepsextent-class"></a>Метод SetPowerState \_ класса CIM аггрегатепсекстент

Метод **SetPowerState** определяет требуемое состояние электропитания для логического устройства и когда устройство должно быть переведено в это состояние. В подклассе необходимо указать набор возможных кодов возврата, используя квалификатор **ValueMap** в методе. Строки, в которые преобразуются содержимое **ValueMap** , также должны быть указаны в подклассе как квалификатор массива **Values** . Этот метод наследуется [**от \_ CIM**](cim-logicaldevice.md)-ого модели.

> [!IMPORTANT]
> Классы CIM (модель CIM) в DMTF (распределенная задача управления) являются родительскими классами, на которых строятся классы WMI. В настоящее время WMI поддерживает только [схемы версии CIM 2. x](https://dmtf.org/standards/cim/schemas).

 

## <a name="syntax"></a>Синтаксис


```mof
uint32 SetPowerState(
  [in] uint16   PowerState,
  [in] datetime Time
);
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает 0 (нуль) в случае успеха, 1 (один), если *указанный запрос* о сбое и *время* не поддерживаются, и другое значение, если произошла какая-либо другая ошибка.
