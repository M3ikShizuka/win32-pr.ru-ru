# <a name="schannel-credentials"></a>Учетные данные SChannel

Протоколы SChannel занимают учетные данные для проверки подлинности серверов и, при необходимости, клиентов. Проверка подлинности сервера, где сервер предоставляет клиенту подтверждение своего удостоверения, требует [*протоколов безопасности*](../secgloss/s-gly.md)SChannel. Проверка подлинности клиента может быть запрошена сервером в любое время.

Учетные данные Schannel — это сертификаты [*X. 509*](../secgloss/x-gly.md) . Сведения об [*открытых*](../secgloss/p-gly.md) и [*закрытых ключах*](../secgloss/p-gly.md) из сертификатов используются для проверки подлинности сервера и, при необходимости, клиента. Эти ключи также используются для обеспечения [*целостности*](../secgloss/i-gly.md) сообщений, когда клиент и сервер обмениваются данными, необходимыми для создания и обмена [*ключами сеанса*](../secgloss/s-gly.md).

Сведения о программировании см. в разделе [Получение учетных данных SChannel](obtaining-schannel-credentials.md).
