# <a name="resource-manager-context-functions"></a>Функции контекста диспетчер ресурсов

Следующие функции устанавливают и освобождают [*контекст диспетчера ресурсов*](../secgloss/r-gly.md) , который используется функциями запросов базы данных и управления базами данных.



| Раздел                                                  | Описание                                                                                                                                          |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCardEstablishContext**](/windows/desktop/api/Winscard/nf-winscard-scardestablishcontext) | Устанавливает контекст для доступа к [*базе данных смарт-карты*](../secgloss/s-gly.md). |
| [**скардрелеасеконтекст**](/windows/desktop/api/Winscard/nf-winscard-scardreleasecontext)     | Закрывает установленный контекст.                                                                                                                       |
