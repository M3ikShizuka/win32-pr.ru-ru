# <a name="user-name-functions"></a>Функции имени пользователя

Функции имени пользователя включают в себя одну функцию [**нпжетусер**](/windows/desktop/api/Npapi/nf-npapi-npgetuser). Эта функция возвращает либо текущее имя пользователя по умолчанию, либо имя пользователя, используемое для установления сетевого подключения.
