# <a name="smart-card-database-management-functions"></a>Функции управления базами данных смарт-карт

Следующие функции управляют [*базой данных смарт-карты*](../secgloss/s-gly.md)и обновляют базу данных с помощью указанного [*контекста Resource Manager*](../secgloss/r-gly.md).

> [!Note]  
> Безопасность базы данных обеспечивается путем размещения ограничений доступа к базе данных, а не путем добавления механизмов безопасности в [*подсистему смарт-карт*](../secgloss/s-gly.md).

 



| Раздел                                                            | Описание                                                                                                                                                             |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**скардаддреадертограуп**](/windows/desktop/api/Winscard/nf-winscard-scardaddreadertogroupa)           | Добавьте [*читателя*](../secgloss/r-gly.md) в [*группу читателей*](../secgloss/r-gly.md). |
| [**скардфоржеткардтипе**](/windows/desktop/api/Winscard/nf-winscard-scardforgetcardtypea)               | Удалите смарт-карту из системы.                                                                                                                                    |
| [**скардфоржетреадер**](/windows/desktop/api/Winscard/nf-winscard-scardforgetreadera)                   | Удалите модуль чтения из системы.                                                                                                                                        |
| [**скардфоржетреадерграуп**](/windows/desktop/api/Winscard/nf-winscard-scardforgetreadergroupa)         | Удаление группы читателей из системы.                                                                                                                                  |
| [**скардинтродуцекардтипе**](/windows/desktop/api/Winscard/nf-winscard-scardintroducecardtypea)         | Ввести новую карточку в систему.                                                                                                                                     |
| [**скардинтродуцереадер**](/windows/desktop/api/Winscard/nf-winscard-scardintroducereadera)             | Познакомьтесь с новым модулем чтения в системе.                                                                                                                                   |
| [**скардинтродуцереадерграуп**](/windows/desktop/api/Winscard/nf-winscard-scardintroducereadergroupa)   | Ввести новую группу читателей в систему.                                                                                                                             |
| [**скардремовереадерфромграуп**](/windows/desktop/api/Winscard/nf-winscard-scardremovereaderfromgroupa) | Удаление модуля чтения из группы читателей.                                                                                                                                    |
