# <a name="iscarddatabase-interface"></a>Интерфейс Искарддатабасе

\[Интерфейс **искарддатабасе** доступен для использования в операционных системах, указанных в разделе требования. он недоступен для использования в Windows Server 2003 с пакетом обновления 1 (SP1) и более поздних версий, Windows Vista, Windows Server 2008 и последующих версиях операционной системы. [Модули смарт-карт](/previous-versions/windows/desktop/secsmart/smart-card-modules) предоставляют аналогичные функции.\]

Интерфейс **искарддатабасе** предоставляет методы для выполнения операций с базой данных в [*диспетчере ресурсов*](../secgloss/r-gly.md) [*смарт-карт*](../secgloss/s-gly.md) . Эти операции включают в себя список известных смарт-карт, [*читателей*](../secgloss/r-gly.md)и групп читателей, а также получение интерфейсов, поддерживаемых смарт-картой и ее [*основным поставщиком услуг*](../secgloss/p-gly.md).

> [!Note]  
> Идентификатор основного поставщика услуг — это GUID COM, который можно использовать для создания экземпляров и использования COM-объектов для конкретной карточки.

 

В следующем примере показано типичное использование интерфейса **искарддатабасе** . В этом случае интерфейс **искарддатабасе** используется для перечисления всех известных смарт-карт.

**Отправка транзакции на определенную карточку**

1.  Создайте интерфейс **искарддатабасе** .
2.  Вызовите [**листкардс**](iscarddatabase-listcards.md) , чтобы получить все известные смарт-карты на основе [*строки ATR*](../secgloss/a-gly.md) или поддерживаемых ими интерфейсов.
3.  Освободите интерфейс **искарддатабасе** .

## <a name="members"></a>Элементы

Интерфейс **искарддатабасе** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Искарддатабасе** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **искарддатабасе** содержит следующие методы.



| Метод                                                          | Описание                                                                                                                                                                                      |
|:----------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**жетпровидеркардид**](iscarddatabase-getprovidercardid.md)   | Получает идентификатор [*основного поставщика услуг*](../secgloss/p-gly.md) для конкретной смарт-карты.<br/> |
| [**листкардинтерфацес**](iscarddatabase-listcardinterfaces.md) | Получает идентификаторы интерфейса (GUID) всех интерфейсов, поддерживаемых конкретной смарт-картой.<br/>                                                                                 |
| [**ListCards**](iscarddatabase-listcards.md)                   | Извлекает все имена смарт-карт, соответствующие определенному набору идентификаторов интерфейсов (GUID) или [*строку ATR*](../secgloss/a-gly.md).<br/> |
| [**листреадерграупс**](iscarddatabase-listreadergroups.md)     | Извлекает имена [*групп читателей*](../secgloss/r-gly.md) , о которых имеет база данных Resource Manager.<br/>                        |
| [**листреадерс**](iscarddatabase-listreaders.md)               | Получите имена [*читателей*](../secgloss/r-gly.md) , для которых диспетчер ресурсов имеет свои знания.<br/>                                          |
