# <a name="authentication-for-web-pages"></a>Проверка подлинности для веб-страниц

Проверка подлинности подтверждает, кто вы.

**Цель:** Чтобы брокер веб-проверки подлинности отображался как часть приложения.

## <a name="prerequisites"></a>Предварительные требования

Нет

**Время на выполнение:** 15 минут.

## <a name="instructions"></a>Инструкции

### <a name="1-getting-started"></a>1. Начало работы

Запустите файл установки, чтобы установить веб-сайт Contoso в службы IIS 8 на локальном компьютере для размещения примеров файлов HTML и CSS.

1.  Образцы файлов HTML и CSS будут установлены в каталог Program Files в Microsoft \\ contoso.
2.  Кроме того, приложение Магазина Windows "Fabrikam Social" будет распаковаться на рабочем столе.

### <a name="2-getting-familiar"></a>2. Знакомство

Чтобы получить представление о том, как выглядят образцы страниц в брокере веб-проверки подлинности, откройте файл решения Fabrikam социальных версий Visual Studio 11 в папке Fabrikam Social на рабочем столе.

1.  Запустите приложение и нажмите кнопку "запустить", чтобы открыть образцы страниц в брокере веб-проверки подлинности.
2.  Размер приложения можно изменить на одну сторону или активировать, открыв некоторые данные в приложении.

### <a name="3-authentication"></a>3. Проверка подлинности

Когда интерфейс API веб-проверки подлинности вызывается базовым приложением для подключения к поставщику "Contoso Social", отображается страница входа. Эта страница предназначена для лучшего использования при быстром и гибком входе в систему. Он также предоставляет точки входа для некоторых других распространенных действий пользователя, таких как получение сведений о пароле, регистрация учетной записи и чтение инструкций по конфиденциальности и условиям, выполненным в браузере. ![страница входа contoso](images/wab-figure8.png)
