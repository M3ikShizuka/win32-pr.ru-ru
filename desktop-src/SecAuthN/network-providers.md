# <a name="network-providers"></a>Поставщики сети

Поставщик сети — это библиотека DLL, которая поддерживает конкретный сетевой протокол. Он также реализует [API поставщика сети](network-provider-api.md). Это позволяет ему взаимодействовать с операционной системой Windows для получения стандартных сетевых запросов, таких как запросы подключения или отключения. Для обработки этих запросов поставщик сети вызывает сетевой интерфейс API, соответствующий сетевому протоколу, поддерживаемому поставщиком сети. Иными словами, поставщик сети создает оболочку для сетевых функций в библиотеке DLL, которая предоставляет стандартный интерфейс для Windows.

Используя поставщики сетевых услуг, Windows может поддерживать множество различных типов сетевых протоколов, не зная специфических для сети сведений о каждой сети. Это важно, так как новые сетевые протоколы разрабатываются все время. При использовании поставщиков сетевых услуг для поддержки нового протокола просто требуется создать и установить новый поставщик сети.

Сведения о функциях и структурах поставщика сети см. в следующих разделах:

-   [Функции поставщика сети](authentication-functions.md)
-   [Структуры поставщиков сети](authentication-structures.md)
