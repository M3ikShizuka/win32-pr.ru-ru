# <a name="connection-oriented-contexts"></a>Контексты Connection-Oriented

В [*контексте*](/windows/desktop/SecGloss/c-gly), ориентированном на соединение, вызывающая функция отвечает за форматирование сообщений. Вызывающий объект использует [*пакет безопасности*](/windows/desktop/SecGloss/s-gly) для проверки подлинности подключений и обеспечения целостности отдельных частей сообщения.

Большинство контекстных параметров доступны для контекстов, ориентированных на соединение. Эти параметры включают взаимную проверку подлинности, обнаружение воспроизведения и обнаружение последовательности, как описано в разделе [требования к контексту](context-requirements.md).

Пакет безопасности устанавливает \_ флаг подключения флага SECPKG, \_ чтобы указать, что он поддерживает семантику, ориентированную на соединение.
