# <a name="winlogon-notification-events"></a>События уведомлений Winlogon

[*Winlogon*](../secgloss/w-gly.md) может уведомлять пакет уведомлений о следующих событиях.



| Событие                               | Описание                                                                                                                                                                                                                                                                                                         |
|-------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Блокировка**<br/>                 | Это событие возникает, когда пользователь блокирует рабочую станцию.<br/>                                                                                                                                                                                                                                                   |
| **Выполнен**<br/>               | Это событие возникает при выходе пользователя из системы. Событие **выхода из системы** выполняется синхронно, даже если параметры реестра пакета уведомлений указывают на то, что он может асинхронно управлять событиями.<br/>                                                                                         |
| **Вход в систему**<br/>                | Это событие возникает при входе пользователя в систему.<br/> Обратите внимание, что событие **logon** возникает перед восстановлением сетевых подключений пользователя. Если обработчику событий требуется доступ к сетевым подключениям пользователя, он должен обрабатывает событие **стартшелл** вместо события **logon** .<br/> |
| **Завершение работы**<br/>             | Это событие происходит непосредственно перед завершением работы системы.<br/>                                                                                                                                                                                                                                                     |
| **смарткардлогоннотифи**<br/> | Это событие возникает при входе пользователя в систему со смарт-картой.<br/>                                                                                                                                                                                                                                   |
| **стартскринсавер**<br/>     | Это событие возникает при запуске экранной заставки. Как правило, это происходит после неактивности пользователя в течение заданного периода времени.<br/> Функции, обрабатывающие это событие, не должны отображать пользовательский интерфейс. Уведомление о событии **стартскринсавер** предназначено только для информационных целей.<br/> |
| **стартшелл**<br/>           | это событие возникает после того, как пользователь вошел в систему, были установлены сетевые подключения и запущена пользовательская программа оболочки (обычно Windows Explorer).<br/>                                                                                                                |
| **Запуск**<br/>              | Это событие возникает при запуске или перезапуске системы.<br/>                                                                                                                                                                                                                                               |
| **стопскринсавер**<br/>      | Это событие возникает при остановке экранной заставки. Обычно это происходит при работе с клавиатурой или мышью.<br/> Функции, обрабатывающие это событие, не должны отображать пользовательский интерфейс. Уведомление о событии **стопскринсавер** предназначено только для информационных целей.<br/>                  |
| **Блокирован**<br/>               | Это событие возникает, когда пользователь снимает блокировку с рабочей станции или когда системный администратор переопределяет блокировку и записывает пользователя в систему.<br/>                                                                                                                                                                         |
