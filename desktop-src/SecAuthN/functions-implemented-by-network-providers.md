# <a name="functions-implemented-by-network-providers"></a>Функции, реализованные поставщиками сети

Следующие функции реализуются библиотекой DLL поставщика сети для взаимодействия с [*маршрутизатором нескольких поставщиков*](/windows/desktop/SecGloss/m-gly) (MPR). Обратите внимание, что поставщик сети должен реализовать только [функции возможностей](capabilities-functions.md) . Реализация других типов функций является необязательной и зависит от требований поставщика сети.



| Набор функций                                                                                    | Описание                                                                                                                                        |
|-------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [Функции возможностей](capabilities-functions.md)<br/>                                 | Позволяет вызывающему объекту определить возможности поставщика сети.<br/>                                                                |
| [Функции имени пользователя](user-name-functions.md)<br/>                                       | Запрашивает у поставщика сети имя пользователя, связанное с подключением.<br/>                                                            |
| [Функции перенаправления устройств](device-redirecting-functions.md)<br/>                     | Позволяет поставщику сети перенаправлять устройства, буквы дисков и порты LPT, которые являются стандартными в операционной системе Microsoft MS-DOS.<br/> |
| [Зависящие от поставщика функции диалогового окна](provider-specific-dialog-box-functions.md)<br/> | Позволяет поставщику сети отображать сведения, относящиеся к сети, или работать с ними.<br/>                                                            |
| [Административные функции](administrative-functions.md)<br/>                             | Позволяет поставщику сети отображать специальные сетевые каталоги или работать с ними.<br/>                                                             |
| [Функции перечисления](enumeration-functions.md)<br/>                                   | Позволяет пользователю просматривать сетевые ресурсы.<br/>                                                                                            |



 

Дополнительные сведения о создании и регистрации поставщика сети см. в статьях [Реализация поставщика](implementing-a-network-provider.md) сети и [Регистрация поставщиков сетевых услуг и диспетчеров учетных данных](registering-network-providers-and-credential-managers.md).
