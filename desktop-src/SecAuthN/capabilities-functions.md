# <a name="capabilities-functions"></a>Функции возможностей

API поставщика сети указывает одну функцию возможностей. Когда операционная система запрашивает сведения о возможностях поставщика сети, [*маршрутизатор с несколькими поставщиками*](/windows/desktop/SecGloss/m-gly) (MPR) вызывает функцию [**нпжеткапс**](/windows/desktop/api/Npapi/nf-npapi-npgetcaps) каждого поставщика сети, сеть которого активна.

Функция [**нпжеткапс**](/windows/desktop/api/Npapi/nf-npapi-npgetcaps) возвращает битовую маску, указывающую, какие функции API поставщика сети поддерживает поставщик сети. Функция **нпжеткапс** является единственной функцией, которую должен реализовать поставщик сети. Операционная система вызывает эту функцию, чтобы определить, какие другие функции API поставщика сети поддерживает поставщик.
