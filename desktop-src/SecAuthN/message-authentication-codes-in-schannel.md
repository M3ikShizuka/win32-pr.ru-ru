# <a name="message-authentication-codes-in-schannel"></a>Коды проверки подлинности сообщений в SChannel

[*Код проверки подлинности сообщения*](../secgloss/m-gly.md) (Mac) используется для обнаружения незаконного изменения и подделки сообщений. Отправитель сообщения создает компьютер MAC, шифруя односторонний [*хэш*](../secgloss/h-gly.md) тела сообщения, используя [*ключ сеанса*](../secgloss/s-gly.md) , совместно используемый отправителем и получателем. MAC добавляется к сообщению, отправляемому получателю. Получатель сообщения снова создает MAC, используя общий ключ сеанса и текст сообщения, и сравнивает созданный MAC с MAC, полученным от отправителя. Если эти два идентичны, то отправитель должен иметь общий ключ сеанса, и сообщение не было изменено при передаче.

В протоколах SChannel алгоритм, используемый для создания компьютера MAC, определяется [набором шифров](cipher-suites-in-schannel.md) , который используется отправителем и получателем.
