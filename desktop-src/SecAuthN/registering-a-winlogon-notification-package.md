# <a name="registering-a-winlogon-notification-package"></a>Регистрация пакета уведомлений Winlogon

Сведения о пакетах уведомлений [*Winlogon*](../secgloss/w-gly.md) хранятся в реестре. В записях реестра указывается имя пакета уведомлений, имя библиотеки DLL, в которой реализован пакет, и имена функций, которые обрабатывали события Winlogon.

При запуске Winlogon проверяет реестр и загружает все зарегистрированные пакеты уведомлений. При возникновении события Winlogon вызывает назначенную функцию обработчика событий для каждого пакета. В системе могут быть зарегистрированы несколько пакетов уведомлений о событиях.

Чтобы зарегистрировать пакет уведомлений, создайте раздел реестра с именем **Notify** в качестве подраздела следующего раздела реестра и добавьте значения, описанные в [записях реестра](registry-entries.md).
