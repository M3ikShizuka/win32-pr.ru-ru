# <a name="quality-of-protection-and-ciphers"></a>Качество защиты и шифров

Качество защиты определяет уровень безопасности, используемый при отправке и получении сообщений. Microsoft Digest выбирает уровень в зависимости от требований клиента и сервера.

При использовании дайджеста в качестве механизма SASL приложения могут указать, что сообщения должны храниться в конфиденциальном виде с помощью шифрования. Сервер указывает шифры, которые он поддерживает для зашифрованных сообщений, и клиент выбирает шифр из этого списка.

В следующих разделах подробно рассматриваются эти механизмы обеспечения безопасности:

-   [Качество защиты](quality-of-protection.md)
-   [Ciphers](ciphers.md)
