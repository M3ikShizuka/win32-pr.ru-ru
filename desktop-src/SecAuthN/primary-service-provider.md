# <a name="primary-service-provider"></a>Первичный поставщик услуг

Каждая [*смарт-карта*](../secgloss/s-gly.md) , известная системе, может иметь ЗАРЕГИСТРИРОВАННЫЙ COM-интерфейс, указанный в качестве основного поставщика услуг. При доступе к *смарт-карте* через COM этот [*первичный поставщик услуг*](../secgloss/p-gly.md) предоставляет интерфейсу управления карту. это позволяет предоставлять службы карт для широкого спектра сред программирования, включая Java и систему разработки Visual Basic майкрософт.
