# <a name="smart-card-service-providers"></a>Поставщики услуг смарт-карт

Поставщики услуг смарт-карт (СКСП) предоставляют доступ к возможностям [*смарт-карт*](../secgloss/s-gly.md) . Они могут предоставить доступ к одной возможности, например к базовым поставщикам служб, предоставляемым пакетом SDK для смарт-карт, или предоставить доступ к нескольким возможностям для выполнения более сложных задач.

Поставщики услуг предоставляют доступ через COM-интерфейсы. Пакет SDK для смарт-карт предоставляет COM-интерфейсы, используемые собственными базовыми поставщиками служб, а также несколько интерфейсов приложений, которые можно использовать при разработке пользовательских поставщиков услуг.
