# <a name="relation-to-other-services"></a>Связь с другими службами

Другие компоненты Microsoft Internet Security Framework используют [*подсистему смарт-карт*](../secgloss/s-gly.md), как показано на следующем рисунке. (Из-за ограничений экспорта в США [*поставщик служб шифрования*](../secgloss/c-gly.md) (CSP), использующий сксп для обмена данными с криптографическими запросами, должен подписать и проверить сксп.)

![отношение Microsoft Internet Security Framework к подсистеме смарт-карт](images/smartovr3.png)

Дополнительные сведения о роли подсистемы смарт-карт в Microsoft Internet Security Framework см. в разделе [https://www.microsoft.com/whdc/device/input/smartcard/default.mspx](https://www.microsoft.com/whdc/device/input/smartcard/default.mspx) .
