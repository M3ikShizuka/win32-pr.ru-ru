# <a name="private-communications-technology"></a>Технология частных коммуникаций

Технология частной связи (PCT) была заменена [*SSL*](../secgloss/s-gly.md) 3,0 и [протоколом безопасности транспортного уровня](transport-layer-security-protocol.md).

Schannel поддерживает PCT 1,0 только для обратной совместимости; его не следует использовать для новой разработки. PCT поддерживает большинство комплектов шифров, поддерживаемых SSL 2,0.
