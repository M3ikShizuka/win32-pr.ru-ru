# <a name="querying-digest-security-context-attributes"></a>Запрос атрибутов контекста безопасности дайджеста

Функция [**QueryContextAttributes (Digest)**](/windows/win32/api/rrascfg/nn-rrascfg-ieapproviderconfig) предоставляет сведения о текущих параметрах [*контекста безопасности*](../secgloss/s-gly.md).

Microsoft Digest поддерживает запросы к следующим атрибутам контекста безопасности.



| attribute                   | Описание                                                                                                                                                                                             |
|-----------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_ \_ сведения о ключе SECPKG attr \_     | Сведения, относящиеся к используемым алгоритмам подписывания и шифрования.                                                                                                                                   |
| \_ \_ сведения о пакете attr SECPKG \_ | Общие сведения, относящиеся к Microsoft Digest, такие как максимальный размер маркера, разрешенный [*пакетом безопасности*](../secgloss/s-gly.md). |
| \_размеры SECPKG attr \_         | Максимальный размер, разрешенный для данных, связанных с сообщениями, таких как подписи.                                                                                                                                |
