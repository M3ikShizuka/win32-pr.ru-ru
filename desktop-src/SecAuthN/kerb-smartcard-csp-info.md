# <a name="kerb_smartcard_csp_info-structure"></a>\_ \_ Информационная структура CSP Kerberos для смарт-карт \_

Структура **\_ сведений о \_ CSP \_** для смарт-карты Kerberos содержит сведения о [*поставщике служб шифрования*](../secgloss/c-gly.md) (CSP).

Эта структура не объявлена в общедоступном заголовке.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _KERB_SMARTCARD_CSP_INFO {
  DWORD dwCspInfoLen;
  DWORD MessageType;
  union {
    PVOID   ContextInformation;
    ULONG64 SpaceHolderForWow64;
  };
  DWORD flags;
  DWORD KeySpec;
  ULONG nCardNameOffset;
  ULONG nReaderNameOffset;
  ULONG nContainerNameOffset;
  ULONG nCSPNameOffset;
  TCHAR bBuffer;
} KERB_SMARTCARD_CSP_INFO, *PKERB_SMARTCARD_CSP_INFO;
```
