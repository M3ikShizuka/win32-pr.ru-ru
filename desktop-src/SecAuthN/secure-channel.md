# <a name="secure-channel"></a>Безопасный канал

Безопасный канал, также известный как [*SChannel*](../secgloss/s-gly.md), — это [*поставщик поддержки безопасности*](../secgloss/s-gly.md) (SSP), который содержит набор протоколов безопасности, обеспечивающих проверку подлинности и безопасную закрытую связь с помощью шифрования.

Schannel в основном используется для Интернет-приложений, требующих обмена данными по протоколу HTTP.
