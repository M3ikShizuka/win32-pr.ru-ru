# <a name="winlogon-notification-package-reference"></a>Справочник по пакету уведомлений Winlogon

В следующих разделах приведены справочные сведения о пакетах уведомлений [*Winlogon*](../secgloss/w-gly.md) .

-   В [записях реестра](registry-entries.md) перечислены ключевые значения и параметры, которые необходимо указать в системном реестре, чтобы зарегистрировать пакет уведомлений и его функции обработчика событий.
-   [Прототип функции обработчика событий](event-handler-function-prototype.md) сведения о прототипе функций обработчика событий. Каждая функция обработчика событий в пакете уведомлений должна соответствовать этому прототипу.
