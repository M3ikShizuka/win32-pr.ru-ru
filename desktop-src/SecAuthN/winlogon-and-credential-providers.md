# <a name="winlogon-and-credential-providers"></a>Winlogon и поставщики учетных данных

[*Winlogon*](../secgloss/w-gly.md) — это модуль Windows, который выполняет интерактивный вход в [*сеанс входа*](../secgloss/l-gly.md). Поведение Winlogon можно настроить путем реализации и регистрации поставщика учетных данных.

Сведения о реализации поставщика учетных данных см. в следующих разделах.



| Раздел                                                                                                           | Описание                                                                                            |
|-----------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [Интерфейс входа Windows, управляемый поставщиком учетных данных](https://go.microsoft.com/fwlink/?LinkId=717287)<br/> | Общие сведения об архитектуре Winlogon и поставщика учетных данных и пример поставщика учетных данных.<br/> |
| [Интерфейсы оболочки](../shell/samples-usingthumbnailproviders.md)<br/>                                                                | Справочник по интерфейсу поставщика учетных данных.<br/>                                                    |
| [Поставщики учетных данных в Windows 10](credential-providers-in-windows.md)<br/>                            | Сторонние поставщики учетных данных и системные поставщики учетных данных в Windows 10.<br/>             |



 

Пример реализации поставщика учетных данных см. в примере, расположенном в каталоге установки Windows SDK в разделе \\ Samples \\ Security \\ кредентиалпровидер.

**Windows Server 2003 и Windows XP:** Поставщики учетных данных не поддерживаются. Сведения о настройке Winlogon см. в разделе [Winlogon и GINA](winlogon-and-gina.md).
