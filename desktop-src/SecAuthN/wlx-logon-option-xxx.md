# <a name="wlx_logon_option_xxx"></a>\_параметр входа \_ влкс \_ xxx

\[параметр влкс \_ LOGON \_ \_ XXX константа больше не доступен для использования в Windows Server 2008 и Windows Vista.\]

Значения **\_ параметра влкс \_ logon \_ xxx** используются параметром *двоптионс* объекта [**влкслогжедаутсас**](/windows/desktop/api/Winwlx/nf-winwlx-wlxloggedoutsas).

> [!Note]  
> библиотеки dll GINA не учитываются в Windows Vista.

 

При успешном входе библиотека DLL GINA может использовать это значение для указания следующего параметра.



| Константа                                                                                                                                                                                          | Описание                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WLX_LOGON_OPT_NO_PROFILE"></span><span id="wlx_logon_opt_no_profile"></span><dl> <dt>**ВЛКС \_ входа в систему \_ \_ без \_ профиля**</dt> </dl> | Указывает, что Winlogon не должен загружать профиль для вошедшего в систему пользователя. В этом случае либо библиотека DLL GINA загрузит профиль, либо пользователю не требуется профиль.<br/> |
