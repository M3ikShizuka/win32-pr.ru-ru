# <a name="tutorial-for-authenticating-web-pages"></a>Руководство по проверке подлинности веб-страниц

В этом учебнике рассматриваются различные аспекты, с которыми должны заботиться веб-разработчики при разработке страниц для потока веб-авторизации в Windows 8. В этом разделе в этом примере демонстрируется поток авторизации из двух страниц.

**Цель:** Целью примера не является предварительное описание макета или визуального дизайна о том, что у каждого поставщика наверняка есть уникальная точка зрения, но для вызова нескольких областей, на которые стоит инвестиции, в качестве первого класса для Windows 8.

## <a name="prerequisites"></a>Предварительные условия

Чтобы использовать API брокера веб-проверки подлинности, вам потребуется:

-   Windows 8 (только x86 или AMD64)
-   Microsoft службы IIS (IIS) следует установить из панели управления в разделе "программы и компоненты" и с помощью параметра "Включение или отключение компонентов Windows".

**Общее время выполнения:** 2 минуты.
