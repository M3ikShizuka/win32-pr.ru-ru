# <a name="tutorial-for-authenticating-web-pages"></a>Руководство по проверке подлинности веб-страниц

В этом учебнике рассматриваются различные аспекты, с которыми должны заботиться веб-разработчики при проектировании страниц для потока веб-авторизации для Windows 8. В этом разделе в этом примере демонстрируется поток авторизации из двух страниц.

**Цель:** Целью образца не является предварительное описание макета или визуального дизайна о том, что у каждого поставщика наверняка есть уникальная точка зрения, но для того, чтобы вызвать несколько областей, стоит прилагаться к созданию первого класса в стиле разработки Microsoft для Windows 8.

## <a name="prerequisites"></a>Предварительные требования

Чтобы использовать API брокера веб-проверки подлинности, вам потребуется:

-   Windows 8 (только x86 или amd64)
-   Microsoft IIS (IIS) следует установить из панели управления в разделе "программы и компоненты" и с помощью параметра "включить или отключить функции Windows компонентов".

**Общее время выполнения:** 2 минуты.
