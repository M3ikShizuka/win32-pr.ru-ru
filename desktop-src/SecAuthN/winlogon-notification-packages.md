# <a name="winlogon-notification-packages"></a>Пакеты уведомлений Winlogon

Пакеты уведомлений [*Winlogon*](../secgloss/w-gly.md) — это библиотеки DLL, которые получают и обрабатывают события, созданные Winlogon. Такой пакет уведомлений можно реализовать для отслеживания событий Winlogon и реагирования на них. Это полезно для приложений, требующих дополнительной обработки во время входа или выхода, или сохранения сведений о [*состоянии*](../secgloss/s-gly.md) , которые должны быть обновлены при возникновении событий Winlogon.

Дополнительные сведения о Winlogon и GINA см. в разделе [Winlogon и GINA](winlogon-and-gina.md).
