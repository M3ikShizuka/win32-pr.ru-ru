# <a name="iscardmanagecreateinterface-method"></a>Метод Искардманаже:: Креатеинтерфаце

\[Метод **креатеинтерфаце** доступен для использования в операционных системах, указанных в разделе требования. [Модули смарт-карт](/previous-versions/windows/desktop/secsmart/smart-card-modules) предоставляют аналогичные функции.\]

Метод **креатеинтерфаце** создает указанный интерфейс.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT CreateInterface(
  [in]  LPGUID    pguidInterface,
  [in]  BSTR      bstrName,
  [in]  LONG      *pUserData,
  [out] LPUNKNOWN *ppInterface
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пгуидинтерфаце* \[ окне\]
</dt> <dd>

Значение GUID создаваемого интерфейса.

</dd> <dt>

*bstrName* \[ окне\]
</dt> <dd>

Имя создаваемого интерфейса, если GUID недоступен. Стандартные значения: "CryptoProvider".

</dd> <dt>

*пусердата* \[ окне\]
</dt> <dd>

Указатель на данные пользователя для использования при создании интерфейса.

</dd> <dt>

*ппинтерфаце* \[ заполняет\]
</dt> <dd>

Указатель на возвращаемый интерфейс.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возможны следующие возвращаемые значения:



| Код возврата                                                                                   | Описание                                                                                      |
|-----------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Operation completed successfully (Операция выполнена успешно).<br/>                                                     |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>  | Один из переданных параметров недопустим.<br/>                                         |
| <dl> <dt>**\_указатель E**</dt> </dl>     | Недопустимый указатель был передан в параметре *пгуидинтерфаце* или *пусердата* .<br/> |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти.<br/>                                                                        |



 

## <a name="remarks"></a>Remarks

Список всех методов, определенных интерфейсом [**искардманаже**](iscardmanage.md) , см. в разделе **искардманаже**.

Кроме приведенных выше кодов ошибок COM, этот интерфейс может возвращать код ошибки [*смарт-карты*](../secgloss/s-gly.md) , если для завершения запроса была вызвана функция смарт-карты. Дополнительные сведения о кодах ошибок смарт-карт см. в статье [возвращаемые значения смарт-карты](authentication-return-values.md).
