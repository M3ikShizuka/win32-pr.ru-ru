# <a name="iscard-interface"></a>Интерфейс кредитной карты

\[Интерфейс **кредитной карты** доступен для использования в операционных системах, указанных в разделе требования. [Модули смарт-карт](/previous-versions/windows/desktop/secsmart/smart-card-modules) предоставляют аналогичные функции.\]

Интерфейс **кредитной карты** позволяет открывать [*смарт-карту*](../secgloss/s-gly.md)и управлять ей. Для каждого соединения с картой требуется один соответствующий экземпляр интерфейса- **карты** .

[*Диспетчер ресурсов*](../secgloss/r-gly.md) смарт-карт должен быть доступен всякий раз, когда создается экземпляр класса « **открытка** ». Если эта служба недоступна, создание интерфейса завершится ошибкой.

В следующем примере показано типичное использование интерфейса- **карты** . Интерфейс **кредитной** карты используется для подключения к смарт-карте, отправки [*транзакции*](../secgloss/t-gly.md)и освобождения смарт-карты.

**Отправка транзакции на определенную карточку**

1.  Создайте интерфейс **кредитной карты** .
2.  Подключитесь к смарт-карте, указав [*устройство чтения*](../secgloss/r-gly.md) смарт-карт или используя ранее установленный, допустимый маркер.
3.  Создание команд транзакций с помощью [**искардкмд**](iscardcmd.md)и [**ISCardISO7816**](iscardiso7816.md) интерфейсов смарт-карт.
4.  Используйте команду "Submit **", чтобы** отправить команды транзакции для обработки со смарт-картой.
5.  Используйте для освобождения смарт **-карты.**
6.  Освободите интерфейс **кредитной карты** .

## <a name="members"></a>Элементы

Интерфейс **кредитной карты** наследует интерфейс [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . У **карточки** также есть члены следующих типов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **кредитной карты** содержит следующие методы.



| Метод                                          | Описание                                                                                                                                                                                                                     |
|:------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**аттачбихандле**](iscard-attachbyhandle.md) | Присоединяет объект к открытому и настроенному обработчику смарт-карты.<br/>                                                                                                                                                      |
| [**аттачбиреадер**](iscard-attachbyreader.md) | Открывает смарт-карту в именованном модуле чтения.<br/>                                                                                                                                                                            |
| [**Detach**](iscard-detach.md)                 | Закрывает открытое подключение к смарт-карте.<br/>                                                                                                                                                                        |
| [**локкскард**](iscard-lockscard.md)           | Потребует монопольного доступа к смарт-карте.<br/>                                                                                                                                                                           |
| [**Повторно подключиться**](iscard-reattach.md)             | Сбрасывает и повторно инициализирует смарт-карту.<br/>                                                                                                                                                                             |
| [**Транзакция**](iscard-transaction.md)       | Выполняет операцию записи и чтения для объекта команды смарт-карты ([*блок данных протокола приложения*](../secgloss/a-gly.md)).<br/> |
| [**унлоккскард**](iscard-unlockscard.md)       | Освобождает эксклюзивный доступ к смарт-карте.<br/>                                                                                                                                                                         |
