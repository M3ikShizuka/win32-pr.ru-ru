# <a name="authentication-reference"></a>Справочник по проверке подлинности

Страницы ссылок для проверки подлинности содержат подробное описание функций проверки подлинности Майкрософт, интерфейсов, объектов, структур и других программных элементов. В этих разделах содержатся сведения об элементах программирования, используемых для установления подлинности пользователей и служб. После установки удостоверения его можно использовать для попытки доступа к одному компьютеру или к ресурсам в сети компьютера.
