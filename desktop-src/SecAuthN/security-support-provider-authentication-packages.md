# <a name="security-support-providerauthentication-packages"></a>Поставщик поддержки безопасности/пакеты проверки подлинности

Windows операционная система поддерживает проверку подлинности с помощью [*пакетов безопасности*](../secgloss/s-gly.md) , которые работают как в качестве [*поставщиков поддержки безопасности*](../secgloss/s-gly.md) , так и в качестве [*пакетов проверки подлинности*](../secgloss/a-gly.md). пакеты безопасности предоставляют службы поддержки процесса входа в систему Windows пакета проверки подлинности, а также предоставляют службы проверки подлинности приложениям путем реализации набора функций, сопоставленных с [интерфейсом поставщика поддержки безопасности](sspi.md) (SSPI).

Пакет безопасности, который работает как пакет проверки подлинности и реализует функциональные возможности, необходимые для [*SSPI*](../secgloss/s-gly.md) , называется поставщиком поддержки безопасности/пакетом проверки подлинности (SSP или AP).

Дополнительные сведения о пакетах безопасности см. [в разделе пакеты SSP, предоставляемые корпорацией Майкрософт](ssp-packages-provided-by-microsoft.md). Сведения о разработке пользовательских поставщиков общих служб и ТД см. в разделе [пользовательские пакеты безопасности](custom-security-packages.md).
