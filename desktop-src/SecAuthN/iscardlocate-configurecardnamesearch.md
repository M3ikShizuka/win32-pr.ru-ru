# <a name="iscardlocateconfigurecardnamesearch-method"></a>Метод Искардлокате:: Конфигурекарднамесеарч

\[Метод **конфигурекарднамесеарч** доступен для использования в операционных системах, указанных в разделе требования. Он недоступен для использования в Windows Server 2003 с пакетом обновления 1 (SP1) и более поздней версии, Windows Vista, Windows Server 2008 и последующих версиях операционной системы. [Модули смарт-карт](/previous-versions/windows/desktop/secsmart/smart-card-modules) предоставляют аналогичные функции.\]

Метод **конфигурекарднамесеарч** указывает имена карточек, которые будут использоваться в поиске [*смарт-карты*](../secgloss/s-gly.md).

## <a name="syntax"></a>Синтаксис


```C++
HRESULT ConfigureCardNameSearch(
  [in] LPSAFEARRAY pCardNames,
  [in] LPSAFEARRAY pGroupNames,
  [in] BSTR        bstrTitle,
  [in] LONG        lFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пкарднамес* \[ окне\]
</dt> <dd>

Указатель на защищенный массив службы автоматизации имен карточек в форме BSTR.

</dd> <dt>

*пграупнамес* \[ окне\]
</dt> <dd>

Указатель на защищенный массив имен групп карточек или читателей в форме BSTR для добавления в поиск.

</dd> <dt>

*бстртитле* \[ окне\]
</dt> <dd>

Заголовок диалогового окна для общего элемента управления поиском.

</dd> <dt>

*лфлагс* \[ окне\]
</dt> <dd>

Указывает, когда отображается [*Пользовательский интерфейс*](../secgloss/u-gly.md) .



| Значение                                                                                                                                                                       | Значение                                                                                                                                                                                                                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="SC_DLG_MINIMAL_UI"></span><span id="sc_dlg_minimal_ui"></span><dl> <dt>**\_ \_ Минимальный \_ Пользовательский интерфейс SC DLG**</dt> </dl> | Отображает диалоговое окно, только если карта, поиск которой выполняется в вызывающем приложении, не найдена и недоступна для использования в модуле чтения. Это позволяет найти карту, подключить ее (через внутренний механизм диалогового окна или с помощью функций обратного вызова пользователя) и вернуть вызывающему приложению.<br/> |
| <span id="SC_DLG_NO_UI"></span><span id="sc_dlg_no_ui"></span><dl> <dt>**SC \_ DLG \_ нет \_ пользовательского интерфейса**</dt> </dl>                | Не приводит к отображению пользовательского интерфейса, независимо от результата поиска.<br/>                                                                                                                                                                                                                                                                  |
| <span id="SC_DLG_FORCE_UI"></span><span id="sc_dlg_force_ui"></span><dl> <dt>**\_ \_ Пользовательский интерфейс SC DLG Force \_ UI**</dt> </dl>       | Вызывает отображение пользовательского интерфейса независимо от результата поиска.<br/>                                                                                                                                                                                                                                                                      |



 

</dd> </dl>

Метод возвращает одно из следующих возможных значений.



| Код возврата                                                                                   | Описание                                                           |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Operation completed successfully (Операция выполнена успешно).<br/>                          |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>  | Недопустимый параметр.<br/>                                         |
| <dl> <dt>**\_указатель E**</dt> </dl>     | Недопустимый указатель был передан в *пкарднамес* или *пграупнамес*.<br/> |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти.<br/>                                             |



 

## <a name="remarks"></a>Комментарии

Чтобы открыть [*смарт-карту*](../secgloss/s-gly.md), вызовите [**финдкард**](iscardlocate-findcard.md).

Список всех методов, предоставляемых этим интерфейсом, см. в разделе [**искардлокате**](iscardlocate.md).

Кроме приведенных выше кодов ошибок COM, этот интерфейс может возвращать код ошибки смарт-карты, если для завершения запроса была вызвана функция смарт-карты. Дополнительные сведения см. в статье [возвращаемые значения смарт-карты](authentication-return-values.md).
