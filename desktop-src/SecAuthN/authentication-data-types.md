# <a name="authentication-data-types"></a>Типы данных проверки подлинности

## <a name="lsa-data-types"></a>Типы данных LSA

[*Локальный центр безопасности*](/windows/desktop/SecGloss/l-gly) (LSA) использует следующий тип данных.



| Тип данных                                            | Описание                                                                                                              |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**\_запрос клиента \_ плса**](plsa-client-request.md) | Непрозрачный тип данных, используемый локальным администратором безопасности для хранения сведений о клиенте, связанных с отдельными запросами.<br/> |



 

## <a name="sspi-data-types-and-values"></a>Типы данных и значения SSPI

[SSPI](sspi.md) использует следующие данные и перечислимые типы.



| Тип данных или значение                         | Описание                                                                      |
|--------------------------------------------|----------------------------------------------------------------------------------|
| [Дескрипторы SSPI](sspi-handles.md)           | Типы дескрипторов SSPI и указатели на эти дескрипторы.<br/>                      |
| [Коды состояния SSPI](sspi-status-codes.md) | Коды состояния возвращаются в приложениях SSPI.<br/>                           |
| [TimeStamp](timestamp.md)                 | Определенный тип для хранения сведений о допустимости срока действия маркеров безопасности.<br/> |
