# <a name="introducing-smart-cards-to-the-system"></a>Введение в систему смарт-карт

Прежде чем [*подсистема смарт-карт*](../secgloss/s-gly.md) сможет найти [*смарт-карту*](../secgloss/s-gly.md), она должна быть введена в систему. Обычно это делается с помощью средства настройки смарт-карты, предоставленного производителем карты. Средство может быть программой на гибком диске (со смарт-картой), элементом управления ActiveX, доступным на веб-сайте, и т. д.

Программа установки должна предоставить следующие сведения о карточке:

-   Его [*строка ATR*](../secgloss/a-gly.md)и необязательная маска для использования в качестве вспомогательной идентификации карты.
-   Список интерфейсов смарт-карт, поддерживаемых картой.
-   Отображаемое имя для карточки, используемое при определении карточки для пользователя. В большинстве случаев пользователь будет предоставлять его программе установки.
-   [*Основной поставщик услуг*](../secgloss/p-gly.md) , связанный с картой (необязательно) для использования при доступе к карте через интерфейсы COM.

Чтобы упростить средства установки и обеспечить целостность [*базы данных смарт-карты*](../secgloss/s-gly.md), подсистема смарт-карт предоставляет следующие две функции. [**Скардинтродуцекардтипе**](/windows/desktop/api/Winscard/nf-winscard-scardintroducecardtypea) вводит смарт-карту в базу данных и [**скардфоржеткардтипе**](/windows/desktop/api/Winscard/nf-winscard-scardforgetcardtypea) удаляет ее из базы данных.
