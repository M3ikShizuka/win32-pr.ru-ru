# <a name="faq-for-web-authentication-broker"></a>Вопросы и ответы по брокеру веб-проверки подлинности

Часто задаваемые вопросы о брокере веб-проверки подлинности.



| Вопрос                                                                                                    | Ответ                                                                                                                                                                                                            |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Как убедиться, что страница https://работает с брокером веб-проверки подлинности?                         | Попробуйте загрузить страницу в Windows Internet Explorer, прежде чем пытаться выполнить ее с помощью брокера веб-проверки подлинности. Если веб-страница загружается без ошибок, она также будет правильно отображена брокером веб-аутентификации. |
| В случае ошибки будет ли отображаться код ошибки для пользователя?                                         | Хотя пользователю будет выводиться страница ошибки, код ошибки не отображается. Он возвращается только приложению. Кроме того, можно также использовать операционные журналы.                                    |
| Как найти дополнительные сведения об обнаруженной ошибке?                                                       | Для получения дополнительных сведений используйте Операционные журналы.                                                                                                                                                                             |
| Использует ли контейнер приложения единого входа (SSO) свои файлы cookie в Internet Explorer или других браузерах? | Нет.                                                                                                                                                                                                               |
