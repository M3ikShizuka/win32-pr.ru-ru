# <a name="iscardtypeconvsizeofistream-method"></a>Метод Искардтипеконв:: Сизеофистреам

\[Метод **сизеофистреам** доступен для использования в операционных системах, указанных в разделе требования. Он недоступен для использования в Windows Server 2003 с пакетом обновления 1 (SP1) и более поздней версии, Windows Vista, Windows Server 2008 и последующих версиях операционной системы. [Модули смарт-карт](/previous-versions/windows/desktop/secsmart/smart-card-modules) предоставляют аналогичные функции.\]

Метод **сизеофистреам** определяет размер (в байтах) интерфейса **IStream** com.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SizeOfIStream(
  [in]  LPSTREAM       pStrm,
  [out] ULARGE_INTEGER *puliSize
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пстрм* \[ окне\]
</dt> <dd>

Указатель на интерфейс COM **IStream** .

</dd> <dt>

*пулисизе* \[ заполняет\]
</dt> <dd>

Указатель на большое целое число без знака, которое может содержать все 64-битное значение sizeof интерфейса **IStream** com.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает одно из следующих возможных значений.



| Код возврата                                                                                  | Описание                                                                                             |
|----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Функция прошла удачно, а *\* пулисизе* — размер (в байтах) интерфейса IStream COM.<br/> |
| <dl> <dt>**\_Ошибка E**</dt> </dl>       | Произошла неопределенная ошибка.<br/>                                                                |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Неверный параметр *пулисизе* .<br/>                                                       |
| <dl> <dt>**\_указатель E**</dt> </dl>    | Неверный параметр *пстрм* .<br/>                                                          |
| <dl> <dt>**\_непредвиденная E**</dt> </dl> | Произошла непредвиденная ошибка.<br/>                                                                   |
