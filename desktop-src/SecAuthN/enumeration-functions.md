# <a name="enumeration-functions"></a>Функции перечисления

Если поставщик сети должен поддерживать просмотр сетевых ресурсов, он должен реализовать следующие функции. Многопротокольное обращение вызывает эти функции для перечисления ресурсов.

Поставщик сети не требуется для реализации каких-либо функций перечисления. Однако если поставщик сети реализует либо [**нпопененум**](/windows/desktop/api/Npapi/nf-npapi-npopenenum), [**Нпенумресаурце**](/windows/desktop/api/Npapi/nf-npapi-npenumresource), либо [**нпклосинум**](/windows/desktop/api/Npapi/nf-npapi-npcloseenum), он также должен реализовывать другие две функции перечисления.



| Функция                                                     | Описание                                                                                                                                                         |
|--------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**нпопененум**](/windows/desktop/api/Npapi/nf-npapi-npopenenum)                             | Открывает перечисление сетевых ресурсов или существующих подключений.                                                                                                  |
| [**нпенумресаурце**](/windows/desktop/api/Npapi/nf-npapi-npenumresource)                     | Выполняет перечисление для маркера, возвращаемого функцией [**нпопененум**](/windows/desktop/api/Npapi/nf-npapi-npopenenum).                                                                                   |
| [**нпклосинум**](/windows/desktop/api/Npapi/nf-npapi-npcloseenum)                           | Закрывает перечисление.                                                                                                                                              |
| [**нпжетресаурцеинформатион**](/windows/desktop/api/Npapi/nf-npapi-npgetresourceinformation) | Определяет, является ли поставщик правильным поставщиком для ответа на запрос указанного сетевого ресурса и возвращает сведения о типе ресурса. |
| [**нпжетресаурцепарент**](/windows/desktop/api/Npapi/nf-npapi-npgetresourceparent)           | Возвращает родителя указанного сетевого ресурса в иерархии обзора.                                                                                         |
