# <a name="ietf-specifications-for-microsoft-digest-ssp"></a>Спецификации IETF для Microsoft Digest SSP

Проверка подлинности HTTP: обычная и дайджест-проверка подлинности (RFC 2617) доступна по адресу [https://www.ietf.org/rfc/rfc2617.txt](https://www.ietf.org/rfc/rfc2617.txt) .

Использование дайджеста в качестве Интернет-черновика механизма SASL доступно по адресу [https://www.ietf.org/rfc/rfc2831.txt](https://www.ietf.org/rfc/rfc2831.txt) .

Дополнительные сведения о SASL см. также в статье простая проверка подлинности и уровень безопасности (RFC 2222), доступные по адресу [https://www.ietf.org/rfc/rfc2222.txt](https://www.ietf.org/rfc/rfc2222.txt) .
