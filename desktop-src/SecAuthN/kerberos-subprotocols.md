# <a name="kerberos-subprotocols"></a>Подпротоколы Kerberos

[*Протокол Kerberos*](../secgloss/k-gly.md) состоит из трех подпротоколов.



| Подпротокол                                                                         | Значение                                                                                                                                                                                                                                                                                                            |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Проверка подлинности службы Exchange](authentication-service-exchange.md)<br/>   | В этом подпротоколе [*центр распространения ключей*](../secgloss/k-gly.md) (KDC) предоставляет клиенту [*ключ сеанса*](../secgloss/s-gly.md) входа и билет на предоставление билета (TGT).<br/> |
| [Предоставление билета службой Exchange](ticket-granting-service-exchange.md)<br/> | В этом подпротоколе KDC распространяет ключ сеанса службы и билет для службы.<br/>                                                                                                                                                                                                            |
| [Обмен данными между клиентом и сервером](client-server-exchange.md)<br/>                     | В этом подпротоколе клиент предоставляет билет на к допуску службы.<br/>                                                                                                                                                                                                                         |
