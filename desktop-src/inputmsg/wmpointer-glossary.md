# <a name="pointer-input-messages-and-notifications-glossary"></a>Глоссарии сообщений и уведомлений на входе указателя

Термины и определения, относящиеся к [входным сообщениям и уведомлениям указателя](messages-and-notifications-portal.md).

<dl> <dt>

<span id="inputmsg.inputmsg_glossary_pointer"></span><span id="INPUTMSG.INPUTMSG_GLOSSARY_POINTER"></span>**Вид**
</dt> <dd>

Любой ввод с указанием типа.

</dd> <dt>

<span id="inputmsg.inputmsg_glossary_pointer_identifier"></span><span id="INPUTMSG.INPUTMSG_GLOSSARY_POINTER_IDENTIFIER"></span>**Идентфиер указателя**
</dt> <dd>

Идентификатор, который однозначно определяет указатель в течение своего времени существования. Указатель поступает при первом обнаружении и завершает его существование, когда выходит за пределы диапазона обнаружения.

</dd> <dt>

<span id="inputmsg.inputmsg_glossary_pointer_frame"></span><span id="INPUTMSG.INPUTMSG_GLOSSARY_POINTER_FRAME"></span>**Рамка указателя**
</dt> <dd>

Одно обновление для всех указателей, связанных, в момент с заданным окном.

</dd> </dl>
