# <a name="tracelogging"></a>TraceLogging

## <a name="purpose"></a>Назначение

трацелоггинг — это новая платформа отслеживания событий Windows 10 для приложений пользовательского режима и драйверов режима ядра. трацелоггинг строится на трассировке событий для Windows (ETW) и предоставляет упрощенный способ инструментирования кода.

## <a name="developer-audience"></a>Аудитория разработчиков

Трацелоггинг предназначен для использования разработчиками приложений пользовательского режима и разработчиками драйверов в режиме ядра, которые хотят добавить трассировку в свой код.
