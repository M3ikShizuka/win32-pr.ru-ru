# <a name="universal-windows-platform-logging-example"></a>Пример ведения журнала универсальная платформа Windows

В этом примере показано, как использовать API-интерфейсы ведения журнала в пространстве имен Windows. Foundation. Diagnostics, включая LoggingChannel, Логгингактивити, Логгингсессион и Филелоггингсессион. Эти классы предназначены для ведения журнала диагностики в приложении Windows. Эти API были добавлены в Windows 8.1.

Интерфейсы API LoggingChannel и Логгингактивити были расширены в Windows 10 для поддержки написания сложных событий с помощью кодирования событий Трацелоггинг.

Пример ведения журнала универсальная платформа Windows можно скачать с сайта [GitHub](https://github.com/Microsoft/Windows-universal-samples/tree/master/Samples/Logging).
