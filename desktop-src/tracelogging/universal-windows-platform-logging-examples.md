# <a name="universal-windows-platform-logging-example"></a>пример ведения журнала универсальная платформа Windows

В этом примере показано, как использовать API ведения журнала в Windows. Пространство имен Foundation. Diagnostics, включая LoggingChannel, Логгингактивити, Логгингсессион и Филелоггингсессион. эти классы предназначены для ведения журнала диагностики в приложении Windows. Эти API были добавлены в Windows 8.1.

интерфейсы api LoggingChannel и логгингактивити расширены в Windows 10 для поддержки написания сложных событий с помощью кодирования событий трацелоггинг.

пример ведения журнала универсальная платформа Windows можно скачать из [GitHub](https://github.com/Microsoft/Windows-universal-samples/tree/master/Samples/Logging).
