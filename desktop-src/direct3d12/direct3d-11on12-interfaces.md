# <a name="11on12-interfaces"></a>Интерфейсы 11on12
Следующие интерфейсы объявляются в d3d11on12. h.

## <a name="in-this-section"></a>В этом разделе
| Раздел | Описание |
|-|-|
| [**ID3D11On12Device**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device) | Обрабатывает создание, перенос и освобождение ресурсов D3D11 для Direct3D 11on12. |
| [**ID3D11On12Device1**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device1) | Обеспечивает улучшенное взаимодействие с компонентом, который может передавать устройства Direct3D 11, но вместо этого хочет использовать Direct3D 12. |
| [**ID3D11On12Device2**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device2) | Позволяет создавать ресурсы, созданные с помощью API-интерфейсов Direct3D 11, и использовать их в Direct3D 12. Этот интерфейс расширяет [ID3D11On12Device1](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device1). |
