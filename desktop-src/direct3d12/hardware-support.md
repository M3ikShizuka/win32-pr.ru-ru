# <a name="hardware-tiers"></a>Уровни оборудования

Уровни оборудования от уровня 1 до уровня 3 увеличивают ресурсы, доступные для конвейера.

-   [Ограничения, зависящие от оборудования](#limits-dependant-on-hardware)
-   [Ограничения для неизменяемых переменных](#invariable-limits)
-   [Связанные темы](#related-topics)

## <a name="limits-dependant-on-hardware"></a>Ограничения, зависящие от оборудования



| Ресурсы, доступные для конвейера                                                                                                              | Уровень 1                                                                   | Уровень 2        | Уровень 3        |
|--------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------|---------------|---------------|
| Уровни компонентов                                                                                                                                   | 11.0 +                                                                    | 11.0 +         | 11.1 +         |
| Максимальное число дескрипторов в представлении постоянного буфера (CBV), шейдер представление ресурсов (SRV) или неупорядоченное представление доступа (UAV), используемое для отрисовки | 1 000 000                                                                | 1 000 000     | 1000000 +    |
| Максимальное число представлений буфера констант во всех таблицах дескрипторов на каждый этап шейдера                                                                | 14                                                                       | 14            | **Полная куча** |
| Максимальное число представлений ресурсов шейдера во всех таблицах дескрипторов на каждый этап шейдера                                                                | 128                                                                      | **Полная куча** | Полная куча     |
| Максимальное число неупорядоченных представлений доступа во всех таблицах дескрипторов на всех этапах                                                              | 64 для уровней компонентов 11.1 +<br/> 8 для уровня функции 11<br/> | 64            | **Полная куча** |
| Максимальное число проб во всех таблицах дескрипторов на каждый этап шейдера                                                                             | 16                                                                       | **2048** | 2048     |



 

Записи, **выделенные жирным шрифтом** , выделяют значительные улучшения на предыдущем уровне.

Существует дополнительное ограничение для оборудования уровня 1, которое применяется ко всем кучам и к оборудованию уровня 2, которое применяется к кучам CBV и UAV, что все записи кучи дескрипторов, покрываемые таблицами дескрипторов в корневой подписи, *должны* быть заполнены дескрипторами по времени выполнения шейдера, даже если в шейдере (возможно, из-за ветвления) не требуется дескриптор. Такого ограничения для оборудования уровня 3 нет. Одним из рисков для этого ограничения является использование [дескрипторов NULL](descriptors.md).

## <a name="invariable-limits"></a>Ограничения для неизменяемых переменных

Максимальное число образцов в куче видимых дескрипторов шейдера — 2048.

Максимальное число уникальных статических выборок в активных корневых форматах — 2032 (это оставляет 16 для драйверов, которым требуются собственные пробы).
