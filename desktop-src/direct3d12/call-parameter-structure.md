# <a name="call-parameter-structure"></a>Структура параметра вызова

Определяемая пользователем структура, предоставляемая как аргумент InOut для вызова [**каллшадер**](callshader-function.md) и как параметр InOut для [вызываемого шейдера](callable-shader.md). Тип структуры, используемый в вызываемом шейдере, должен соответствовать структуре, предоставленной соответствующему вызову **каллшадер** .
