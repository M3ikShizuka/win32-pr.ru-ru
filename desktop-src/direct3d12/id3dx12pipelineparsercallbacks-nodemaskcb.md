## <a name="syntax"></a>Синтаксис


```C++
void NodemaskCb(
   
        UINT
       Nodemask
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*нодемаск* 
</dt> <dd>

Тип: **uint**

</dd> </dl>
