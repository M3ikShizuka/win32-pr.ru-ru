# <a name="direct3d-12-programming-guide"></a>Руководство по программированию для Direct3D 12

Direct3D 12 предоставляет API и платформу, позволяющие приложениям воспользоваться преимуществами графических и вычислительных возможностей компьютеров, оснащенных одним или несколькими GPU, совместимыми с Direct3D 12.
