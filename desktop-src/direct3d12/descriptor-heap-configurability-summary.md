# <a name="descriptor-heap-configurability-summary"></a>Сводка по настройке кучи дескрипторов

В следующей таблице представлены сведения о поддержке шейдера и видимой кучи без шейдера.



|                               | Видимая куча дескрипторов шейдеров                                                 | Видимая куча дескрипторов без шейдеров                                                                                                                                                                                                                                                                                                                                  |
|-------------------------------|--------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Поддерживаемые типы кучи          | CBV \_ SRV \_ UAV, образец                                                         | Все                                                                                                                                                                                                                                                                                                                                                                 |
| Поддерживаемые свойства страницы ЦП | \_недоступно, запись \_ объединения                                                 | \_обратная запись                                                                                                                                                                                                                                                                                                                                                         |
| Управление местонахождение по приложениям   | Да, ответственное за приложение                                                           | Неприменимо (не отображается GPU).                                                                                                                                                                                                                                                                                                                                   |
| Поддержка редактирования дескрипторов       | Копировать только назначение, через обновление списка команд и (или) копирование ЦП, если отображается ЦП. | Только чтение и запись ЦП. Прямой доступ к GPU отсутствует. Может использоваться для немедленного копирования ЦП (в качестве источника и назначения). Можно использовать в качестве источника обновления в списке команд — это приведет к копированию дескрипторов в хранилище списка команд во время записи списка команд. При выполнении хранимая копия будет скопирована в место назначения, что должно быть видимой кучей шейдера. |
