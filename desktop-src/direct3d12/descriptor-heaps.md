# <a name="descriptor-heaps"></a>Кучи дескрипторов

Куча дескрипторов — это набор смежных выделений дескрипторов, по одному выделению для каждого из них.

| [Уровни оборудования](hardware-support.md)<br/>                                                 | Уровни оборудования от уровня 1 до уровня 3 увеличивают ресурсы, доступные для конвейера. <br/>                                                                                                                              |
| [Доступные шейдеру кучи дескрипторов](shader-visible-descriptor-heaps.md)<br/>                 | Видимые кучи дескрипторов шейдеров — это кучи дескрипторов, на которые могут ссылаться шейдеры через таблицы дескрипторов.<br/>                                                                                                              |
