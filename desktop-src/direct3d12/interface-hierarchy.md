# <a name="interface-hierarchy"></a>Иерархия интерфейса

На схеме показана иерархия наследования интерфейсов.

Большинство приложений будет использовать только интерфейсы в верхней части иерархии и не использовать унаследованные интерфейсы напрямую. [**ID3D12Pageable**](/windows/win32/api/d3d12/nn-d3d12-id3d12pageable), [**ID3D12DeviceChild**](/windows/desktop/api/d3d12/nn-d3d12-id3d12devicechild), [**ID3D12Object**](/windows/desktop/api/d3d12/nn-d3d12-id3d12object) и [**ID3D12CommandList**](/windows/desktop/api/d3d12/nn-d3d12-id3d12commandlist).

Новые интерфейсы отображаются зеленым прямоугольником.
