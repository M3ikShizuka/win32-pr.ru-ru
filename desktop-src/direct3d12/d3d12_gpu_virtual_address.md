# <a name="d3d12_gpu_virtual_address"></a>\_ \_ Виртуальный адрес GPU \_ D3D12

D3D12_GPU_VIRTUAL_ADDRESS представляет виртуальный адрес в памяти GPU. Тип данных определяется в d3d12. h следующим образом:


```
typedef UINT64 D3D12_GPU_VIRTUAL_ADDRESS;
```
