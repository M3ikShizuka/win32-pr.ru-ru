# <a name="direct3d-12-graphics"></a>Графика Direct3D 12

Это руководство по программированию содержит сведения об использовании программируемого конвейера Direct3D 12 для создания настраиваемого графического подсистемы.

Заголовки и библиотеки Direct3D 12 входят в состав пакета SDK для Windows 10. Для использования Direct3D 12 не требуется отдельная загрузка или установка.
