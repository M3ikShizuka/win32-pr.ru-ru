# <a name="directml-structures"></a>Структуры DirectML

Следующие структуры объявляются в Директмл. h.

| Раздел и описание |
|-|
| [**DML_ACTIVATION_CELU_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_activation_celu_operator_desc). Выполняет постоянную функцию активации дифферентиабле экспоненциального линейного блока (ЦЕЛУ) для каждого элемента в *инпуттенсор*, помещая результат в соответствующий элемент *аутпуттенсор*. |

| [**DML_ACTIVATION_LOG_SOFTMAX_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_activation_log_softmax_operator_desc). Описывает оператор Директмл, который выполняет функцию активации log-softmax на входе. |
| [**DML_ACTIVATION_PARAMETERIZED_RELU_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_activation_parameterized_relu_operator_desc). | [**DML_ACTIVATION_PARAMETRIC_SOFTPLUS_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_activation_parametric_softplus_operator_desc). Описывает оператор Директмл, который выполняет функцию параметрической софтплус Activation для каждого элемента во входных данных. |
| [**DML_ACTIVATION_RELU_GRAD_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_activation_relu_grad_operator_desc). Вычисляются градиенты обратного распространения для линейной единицы исправить (Релу). |
| [**DML_ACTIVATION_RELU_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_activation_relu_operator_desc). Описывает оператор Директмл, который выполняет функцию активации линейного блока исправить (Релу) для каждого элемента во входных данных. |
| [**DML_ACTIVATION_SCALED_ELU_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_activation_scaled_elu_operator_desc). Описывает оператор Директмл, который выполняет функцию активации масштабируемой экспоненциальной линейной единицы (ЕЛУ) для каждого элемента во входных данных. |
| [**DML_ACTIVATION_SCALED_TANH_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_activation_scaled_tanh_operator_desc). Описывает оператор Директмл, который выполняет масштабируемую функцию активации гиперболического тангенса для каждого элемента во входных данных. |
| [**DML_ACTIVATION_SHRINK_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_activation_shrink_operator_desc). Описывает оператор Директмл, который выполняет функцию активации елементвисе Shrink для входных данных. |
| [**DML_ACTIVATION_SIGMOID_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_activation_sigmoid_operator_desc). Описывает оператор Директмл, который выполняет функцию активации сигмоидальной для каждого элемента во входных данных. |
| [**DML_ACTIVATION_SOFTMAX_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_activation_softmax_operator_desc). Описывает оператор Директмл, который выполняет функцию активации softmax для входных данных. |
| [**DML_ACTIVATION_SOFTPLUS_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_activation_softplus_operator_desc). Описывает оператор Директмл, который выполняет функцию активации софтплус для каждого элемента во входных данных. |
| [**DML_ACTIVATION_SOFTSIGN_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_activation_softsign_operator_desc). Описывает оператор Директмл, который выполняет функцию активации софтсигн для каждого элемента во входных данных. |
| [**DML_ACTIVATION_TANH_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_activation_tanh_operator_desc). Описывает оператор Директмл, который выполняет гиперболический тангенс функции активации для каждого элемента во входных данных. |
| [**DML_ACTIVATION_THRESHOLDED_RELU_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_activation_thresholded_relu_operator_desc). Описывает оператор Директмл, который выполняет функцию активации порогового исправить линейного блока (Релу) для каждого элемента во входных данных. |
| [**DML_ADAM_OPTIMIZER_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_adam_optimizer_operator_desc). Выполняет вычисление обновленных весов (параметров) с помощью представленных градиентов на основе алгоритма ADAM (**ADA** птиве **M** омент оценкой). Этот оператор является оптимизатором и обычно используется на шаге обновления веса цикла обучения для выполнения спуска градиента. |
| [**DML_AVERAGE_POOLING_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_average_pooling_operator_desc). Описывает оператор Директмл, который выполняет среднюю функцию объединения в пул для входных данных. |
| [**DML_ARGMAX_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_argmax_operator_desc). Выводит индексы элементов с максимальным значением в одном или нескольких измерениях входного тензорные. |
| [**DML_ARGMIN_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_argmin_operator_desc). Выводит индексы элементов с минимальными значениями в одном или нескольких измерениях входного тензорные. |
| [**DML_AVERAGE_POOLING_GRAD_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_average_pooling_grad_operator_desc). Вычисляет градиенты обратного распространения для среднего количества пулов (см. [DML_AVERAGE_POOLING_OPERATOR_DESC](/windows/win32/api/directml/windows/win32/api/directml/ns-directml-dml_average_pooling_operator_desc)). |
| [**DML_BATCH_NORMALIZATION_GRAD_OPERATOR_DESC**](./directml/ns-directml-dml_batch_normalization_grad_operator_desc.md). Вычисляются градиенты обратного распространения для [нормализации пакетной обработки](/windows/win32/api/directml/ns-directml-dml_batch_normalization_operator_desc). |
| [**DML_BATCH_NORMALIZATION_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_batch_normalization_operator_desc). Описывает оператор Директмл, который выполняет функцию нормализации пакетной службы для входных данных. |
| [**DML_BINDING_DESC**](/windows/desktop/api/directml/ns-directml-dml_binding_desc). Содержит описание привязки, которое можно добавить в таблицу привязки с помощью вызова одного из методов [идмлбиндингтабле](/windows/desktop/api/directml/nn-directml-idmlbindingtable) . |
| [**DML_BINDING_PROPERTIES**](/windows/desktop/api/directml/ns-directml-dml_binding_properties). Содержит сведения о требованиях к привязке для определенного скомпилированного оператора или инициализатора оператора. |
| [**DML_BINDING_TABLE_DESC**](/windows/desktop/api/directml/ns-directml-dml_binding_table_desc). Задает параметры для [идмлдевице:: креатебиндингтабле](/windows/desktop/api/directml/nf-directml-idmldevice-createbindingtable) и [Идмлбиндингтабле:: Reset](/windows/desktop/api/directml/nf-directml-idmlbindingtable-reset). |
| [**DML_BUFFER_ARRAY_BINDING**](/windows/desktop/api/directml/ns-directml-dml_buffer_array_binding). Задает привязку к ресурсу, которая является массивом отдельных привязок буфера. |
| [**DML_BUFFER_BINDING**](/windows/desktop/api/directml/ns-directml-dml_buffer_binding). Задает привязку к ресурсу, описанную в диапазоне байтов в буфере Direct3D 12, представленном смещением и размером в [ID3D12Resource](/windows/desktop/api/d3d12/nn-d3d12-id3d12resource). |
| [**DML_BUFFER_TENSOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_buffer_tensor_desc). Описывает тензорные, который будет храниться в ресурсе буфера Direct3D 12. |
| [**DML_CAST_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_cast_operator_desc). Описывает оператор реорганизации данных Директмл, который выполняет функцию CAST f (x) = CAST (x), приведение каждого элемента во входных данных к типу с выходным тензорные и сохранение результата в соответствующем элементе в выходных данных. |
| [**DML_CONVOLUTION_INTEGER_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_convolution_integer_operator_desc). Выполняет свертывание *филтертенсор* с *инпуттенсор*. Этот оператор выполняет пересылку по целочисленным данным. |
| [**DML_CONVOLUTION_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_convolution_operator_desc). Описывает оператор умножения матрицы Директмл, который выполняет функцию свертки для входных данных. |
| [**DML_CUMULATIVE_PRODUCT_OPERATOR_DESC**](./directml/ns-directml-dml_cumulative_product_operator_desc.md). Умножает элементы тензорные вдоль оси, записывая выполняющийся высокий коэффициент произведения в выходном тензорные. |
| [**DML_CUMULATIVE_SUMMATION_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_cumulative_summation_operator_desc). Суммирует элементы тензорные вдоль оси, записывая выполняющийся счетчик суммы в выходном тензорные. |
| [**DML_DEPTH_TO_SPACE_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_depth_to_space_operator_desc). Описывает оператор реорганизации данных Директмл, который переупорядочивает (пермутес) данные из глубины в блоки пространственных данных. |
| [**DML_DEPTH_TO_SPACE1_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_depth_to_space1_operator_desc). Переупорядочивает (пермутес) данные из глубины в блоки пространственных данных. Оператор выводит копию входного тензорные, где значения из измерения глубины перемещаются в пространственные блоки в измерения высоты и ширины. |
| [**DML_DIAGONAL_MATRIX_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_diagonal_matrix_operator_desc). Описывает Директмл математический оператор, который создает матрицу, подобную Identity, с элементами на основных диагональных и нулевых местах везде. |
| [**DML_DYNAMIC_QUANTIZE_LINEAR_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_dynamic_quantize_linear_operator_desc). Вычисляет масштаб дискретизация и значения нулевой точки, необходимые для квантуем *инпуттенсор*, затем применяет этот дискретизация, записывая результат в *аутпуттенсор*. |
| [**DML_ELEMENT_WISE_ABS_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_element_wise_abs_operator_desc). Описывает Директмл математический оператор, который выполняет функцию f (x) = ABS (x * Scale + сдвиг) на уровне элементов, где условия масштабирования и смещения необязательны. |
| [**DML_ELEMENT_WISE_ACOS_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_element_wise_acos_operator_desc). Описывает тригонометрический оператор Директмл, который выполняет функцию арккосинус f (x) = ACOS (x * Scale + сдвиг), где условия масштабирования и смещения необязательны. |
| [**DML_ELEMENT_WISE_ACOSH_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_element_wise_acosh_operator_desc). Описывает тригонометрический оператор Директмл, выполняющий обратный гиперболический косинус функции f (x) = log (x + Sqrt (x * x-1)) * масштабирования и смещения, где условия масштабирования и смещения являются необязательными. |
| [**DML_ELEMENT_WISE_ADD_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_element_wise_add_operator_desc). Описывает оператор Директмл Math, который выполняет функцию добавления каждого элемента в Атенсор в соответствующий элемент в Бтенсор. |
| [**DML_ELEMENT_WISE_ADD1_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_element_wise_add1_operator_desc). Описывает оператор Директмл Math, который выполняет функцию добавления каждого элемента в <i>атенсор</i> в соответствующий элемент в <i>бтенсор</i>, f (a, b) = a + b с параметром для активации с плавким предохранителем. |
| [**DML_ELEMENT_WISE_ASIN_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_element_wise_asin_operator_desc). Описывает тригонометрический оператор Директмл, который выполняет функцию арксинус по элементу f (x) = ASIN (x * Scale + сдвиг), где условия масштабирования и смещения необязательны. |
| [**DML_ELEMENT_WISE_ASINH_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_element_wise_asinh_operator_desc). Описывает тригонометрический оператор Директмл, выполняющий обратный гиперболический синус функции f (x) = log (x + Sqrt (x * x + 1)) * масштабирования и смещения, где условия масштабирования и смещения являются необязательными. |
| [**DML_ELEMENT_WISE_ATAN_OPERATOR_DESC**](/windows/desktop/api/directml/ns-directml-dml_element_wise_atan_operator_desc). Описывает тригонометрический оператор Директмл, который выполняет функцию-арктангенс по элементу f (x) = ATAN (x * Scale + сдвиг), где условия масштабирования и смещения необязательны. |

Описывает тригонометрический оператор Директмл, выполняющий обратную гиперболический тангенс функции f (x) = (log ((1 + x)/(1-x))/2) * масштабирования и смещения, где условия масштабирования и смещения необязательны. |
| [**DML_ELEMENT_WISE_ATAN_YX_OPERATOR_DESC**](./directml/ns-directml-dml_element_wise_atan_yx_operator_desc.md). Выполняет вычисление двух аргументов-арктангенс для каждого элемента *атенсор* и *Бтенсор*, где *атенсор* — это *ось Y* , а *бтенсор* — *ось X*, помещая результат в соответствующий элемент *аутпуттенсор*. |

| [**DML_ELEMENT_WISE_BIT_NOT_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_element_wise_bit_not_operator_desc). Выдает значение счетчика побитового заполнения (число бит, заданное равным 1) для каждого элемента входного тензорные и записывает результат в выходной тензорные. |
| [**DML_ELEMENT_WISE_BIT_OR_OPERATOR_DESC**](/windows/win32/api/directml/ns-directml-dml_element_wise_bit_or_operator_desc). Вычисляются побитовое или между соответствующим элементом входных десятков и записывает результат в выходной тензорные. |

| [**DML_ELEMENT_WISE_CLIP_GRAD_OPERATOR_DESC**](./directml/ns-directml-dml_element_wise_clip_grad_operator_desc.md). Вычисление градиентов обратного распространения для [элемента, ориентированного на элемент](/windows/win32/api/directml/ns-directml-dml_element_wise_clip_operator_desc). |
