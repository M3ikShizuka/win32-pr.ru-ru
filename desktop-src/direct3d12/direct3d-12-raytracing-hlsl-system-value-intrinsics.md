# <a name="direct3d-12-raytracing-hlsl-system-value-intrinsics"></a>Встроенные функции системных значений HLSL для трассировки лучей в Direct3D 12

Системные значения извлекаются с помощью специальных встроенных функций, а не включают параметры с особой семантикой в сигнатуре функции шейдера. 

## <a name="in-this-section"></a>Содержание раздела

### <a name="ray-dispatch-system-values"></a>Значения системы диспетчеризации луча

| Раздел | Описание |
|-|-|
| [**диспатчрайсиндекс**](dispatchraysindex.md) | Возвращает текущее расположение x и y в пределах ширины и высоты, полученных с помощью встроенного системного значения **диспатчрайсдименсионс** . |
| [**DispatchRaysDimensions**](dispatchraysdimensions.md) | Значения ширины, высоты и глубины из структуры **\_ \_ \_ DESC D3D12 диспетчеризации** , указанной в исходном вызове **диспатчрайс** . |

### <a name="ray-system-values"></a>Значения системы лучей

| Раздел | Описание |
|-|-|
| [**ворлдрайоригин**](worldrayorigin.md) | Источник универсального пространства текущего луча. |
| [**ворлдрайдиректион**](worldraydirection.md) | Направление международного пространства для текущего луча. |
| [**райтмин**](raytmin.md) | Значение типа float, представляющее текущую отправную точку параметрической для луча. |
| [**райткуррент**](raytcurrent.md) | Значение типа float, представляющее текущую конечную точку параметрической для луча.  |
| [**райфлагс**](rayflags.md) | Целое число без знака, содержащее текущие флаги **ray_flag** . |

### <a name="primitiveobject-space-system-values"></a>Системные значения примитива или объектного пространства

| Раздел | Описание |
|-|-|
| [**инстанцеиндекс**](instanceindex.md) | Автоматически сформированный индекс текущего экземпляра в структуре ускорения райтраЦинг верхнего уровня. |
| [**InstanceID**](instanceid.md) | Предоставленный пользователем идентификатор для экземпляра на экземпляре структуры ускорения нижнего уровня в структуре верхнего уровня. |
| [**примитивеиндекс**](primitiveindex.md) | Автоматически сформированный индекс примитива внутри геометрии внутри экземпляра структуры ускорения нижнего уровня. |
| [**обжектрайоригин**](objectrayorigin.md) | Источник пространства объекта для текущего луча. |
| [**обжектрайдиректион**](objectraydirection.md) | Направление объектного пространства для текущего луча. |
| [**ObjectToWorld3x4**](objecttoworld3x4.md) | Матрица для преобразования из объектного пространства в мировое пространство. |
| [**ObjectToWorld4x3**](objecttoworld4x3.md) | Матрица для преобразования из объектного пространства в мировое пространство. |
| [**WorldToObject3x4**](worldtoobject3x4.md) | Матрица для преобразования из мирового пространства в объектно-пространство |
| [**WorldToObject4x3**](worldtoobject4x3.md) | Матрица для преобразования из мирового пространства в объектно-пространство |
