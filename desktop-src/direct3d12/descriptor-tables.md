# <a name="descriptor-tables"></a>Таблицы дескрипторов

Таблица дескрипторов логически представляет собой набор дескрипторов.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                 | Описание                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Общие сведения о таблицах дескрипторов](descriptor-tables-overview.md)<br/>               | Каждая таблица дескрипторов хранит дескрипторы одного или нескольких типов — СРВС, Уавс, КБВС и пробы. Таблица дескрипторов не является выделением памяти; Это просто смещение и длина в куче дескрипторов.<br/> |
| [Использование таблиц дескрипторов](using-descriptor-tables.md)<br/>                     | Таблицы дескрипторов, каждый из которых идентифицирует диапазон в куче дескрипторов, привязываются к слотам, определенным текущей корневой подписью в списке команд. <br/>                                                               |
| [Расширенное использование таблиц дескрипторов](advanced-use-of-descriptor-tables.md)<br/> | В следующих разделах приводятся сведения о расширенном использовании таблиц дескрипторов.<br/>                                                                                                                      |
