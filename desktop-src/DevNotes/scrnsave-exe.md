# <a name="scrnsaveexe"></a>SCRNSAVE.EXE

**\\Рабочий стол панели управления HKCU \\**



| Тип данных | Диапазон       | Значение по умолчанию |
|-----------|-------------|---------------|
| REG \_ SZ   | *Имя файла* | (нет)        |



 

## <a name="description"></a>Описание

Указывает имя исполняемого файла экранной заставки.

## <a name="change-method"></a>Изменение метода

Чтобы изменить значение этой записи, на панели управления дважды щелкните элемент " **экран**", перейдите на **вкладку "Заставка"** и выберите заставку из списка **экранных заставок** .

Эту запись также можно изменить с помощью функции интерфейса прикладного программирования (API) **инсталлскринсавер**, которая экспортируется Desk.cpl. Доступ к этой функции можно получить с помощью командной строки **rundll32.exe desk.cpl, инсталлскринсавер** *File*, где *File* — это имя исполняемого файла экранной заставки.

## <a name="activation-method"></a>Метод активации

Изменения, внесенные в эту запись, вступают в силу в следующий раз, когда система запускает экранную заставку. Изменения, внесенные в эту запись во время работы экранной заставки, не влияют на работу с экранной заставкой.

## <a name="notes"></a>Примечания

-   Windows операционные системы добавляют эту **запись в реестр при использовании экрана** в панели управления для выбора заставки. Если отключить все экранные заставки, выбрав **(нет)** в списке экранная заставка, операционная система удаляет эту запись из реестра и вызывает функцию [**системпараметерсинфо**](/windows/win32/api/winuser/nf-winuser-systemparametersinfoa) с параметром *уиактион* , равным SPI \_ сетскринсавеактиве, а *уипарам* равно **false**.
-   Эта запись может быть заменена параметром групповая политика в системах, поддерживающих групповая политика. Хотя параметр групповая политика **имя исполняемого файла заставки** включен, система пропускает эту запись. Конфигурация этого параметра политики хранится в записи **SCRNSAVE.EXE** , которая находится в подразделе " **политики** ".
