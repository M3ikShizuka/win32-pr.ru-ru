# <a name="ceipenable"></a>CEIPEnable

**HKLM \\ Software \\ Microsoft \\ скмклиент \\ Windows \\ цеипенабле**



| Тип данных  | Диапазон                     | Значение по умолчанию |
|------------|---------------------------|---------------|
| REG \_ DWORD | 0 (отключить) или 1 (включить) | 0             |



 

## <a name="description"></a>Описание

включает программу по улучшению качества программного обеспечения Windows.

## <a name="change-method"></a>Изменение метода

Чтобы изменить значение этой записи, на панели управления выберите **система и обслуживание**, а затем **отчеты о проблемах и решения**. щелкните **Параметры улучшения качества** программного обеспечения в разделе "см. также".

## <a name="activation-method"></a>Метод активации

Изменения, внесенные в эту запись, вступают в силу немедленно. включение этого раздела реестра приводит к включению Windows программа улучшения качества программного обеспечения для всего компьютера. отключение этого раздела реестра приводит к отключению Windows программа улучшения качества программного обеспечения для всего компьютера.

Эта запись может быть заменена параметром групповая политика в системах, поддерживающих групповая политика. в то время как параметр групповая политика включить Windows программа улучшения качества программного обеспечения CEIP включен, система пропускает эту запись. конфигурация этого параметра политики хранится в разделе **политики** в разделе **\\ политики программного обеспечения HKLM \\ \\ Microsoft \\ скмклиент \\ Windows \\ цеипенабле**.
