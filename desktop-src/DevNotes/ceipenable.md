# <a name="ceipenable"></a>CEIPEnable

**HKLM \\ Software \\ Microsoft \\ скмклиент \\ Windows \\ цеипенабле**



| Тип данных  | Диапазон                     | Значение по умолчанию |
|------------|---------------------------|---------------|
| REG \_ DWORD | 0 (отключить) или 1 (включить) | 0             |



 

## <a name="description"></a>Описание

Включает программу улучшения качества программного обеспечения Windows.

## <a name="change-method"></a>Изменение метода

Чтобы изменить значение этой записи, на панели управления выберите **система и обслуживание**, а затем **отчеты о проблемах и решения**. Щелкните **Параметры улучшения качества** программного обеспечения в разделе см. также.

## <a name="activation-method"></a>Метод активации

Изменения, внесенные в эту запись, вступают в силу немедленно. Включение этого раздела реестра приводит к включению программа улучшения качества программного обеспечения Windows для всего компьютера. Отключение этого раздела реестра приводит к отключению программа улучшения качества программного обеспечения Windows для всего компьютера.

Эта запись может быть заменена параметром групповая политика в системах, поддерживающих групповая политика. Хотя параметр групповая политика включение программы улучшения качества программного обеспечения Windows программа улучшения качества программного обеспечения включен, система пропускает эту запись. Конфигурация этого параметра политики хранится в разделе **политики** в разделе **\\ политики программного обеспечения HKLM \\ \\ Microsoft \\ скмклиент \\ Windows \\ цеипенабле**.
