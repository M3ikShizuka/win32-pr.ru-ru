# <a name="tag-types"></a>Типы тегов

| Константа/значение                                                                                                                                                                                                                            | Описание                                           |

| <span id="TAG_TYPE_BYTE"></span><span id="tag_type_byte"></span><dl> <dt>**Тег \_ Тип \_ Byte**</dt> <dt>0x2000</dt> </dl>                | Значение **типа Byte** .<br/>                          |
| <span id="TAG_TYPE_WORD"></span><span id="tag_type_word"></span><dl> <dt>**Тег \_ Введите \_ слово**</dt> <dt>0x3000</dt> </dl>                | Значение **слова** .<br/>                          |
| <span id="TAG_TYPE_DWORD"></span><span id="tag_type_dword"></span><dl> <dt>**Тег \_ Введите \_ DWORD**</dt> <dt>0x4000</dt> </dl>             | Значение **типа DWORD** .<br/>                         |
| <span id="TAG_TYPE_QWORD"></span><span id="tag_type_qword"></span><dl> <dt>**Тег \_ Введите \_ QWORD**</dt> <dt>0x5000</dt> </dl>             | Значение **улонглонг** .<br/>                     |
