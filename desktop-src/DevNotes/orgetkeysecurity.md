# <a name="orgetkeysecurity-function"></a>Функция Оржеткэйсекурити

Извлекает копию дескриптора безопасности, защищающего указанный открытый раздел реестра, в автономном кусте реестра.

## <a name="syntax"></a>Синтаксис


```C++
DWORD ORGetKeySecurity(
  _In_      ORHKEY               Handle,
  _In_      SECURITY_INFORMATION SecurityInformation,
  _Out_opt_ PSECURITY_DESCRIPTOR pSecurityDescriptor,
  _Inout_   PDWORD               lpcbSecurityDescriptor
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Обработчик* \[ окне\]
</dt> <dd>

Маркер открытого раздела реестра в автономном кусте реестра.

</dd> <dt>

*Секуритинформатион* \[ окне\]
</dt> <dd>

Значение [ \_ сведений о безопасности](../secauthz/security-information.md) , указывающее запрошенные сведения о безопасности.

</dd> <dt>

*псекуритидескриптор* \[ out, необязательно\]
</dt> <dd>

Указатель на буфер, который получает копию запрошенного дескриптора безопасности. Этот параметр может принимать значение **NULL**.

</dd> <dt>

*лпкбсекуритидескриптор* \[ в, out\]
</dt> <dd>

Указатель на переменную, указывающую размер (в байтах) буфера, на который указывает параметр *псекуритидескриптор* . Когда функция возвращает значение, переменная содержит число байтов, записанных в буфер.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается успешно, функция возвращает ошибку \_ Success.

Если функция завершается ошибкой, она возвращает ненулевой код ошибки, определенный в файле Winerror. h. [](/windows/win32/api/winbase/nf-winbase-formatmessage) \_ \_ \_ Для получения обобщенного описания ошибки можно использовать функцию FormatMessage с флагом формата Message от System.

Если буфер, указанный в параметре *псекуритидескриптор* , слишком мал, функция возвращает ошибку \_ недостаточный \_ Размер буфера, а параметр *лпкбсекуритидескриптор* содержит число байтов, необходимое для запрошенного дескриптора безопасности.
