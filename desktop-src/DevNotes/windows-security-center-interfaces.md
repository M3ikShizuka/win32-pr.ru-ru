# <a name="windows-security-center-interfaces"></a>Безопасность Windows Интерфейсы центра

для использования с Безопасность Windows Center доступны следующие интерфейсы.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                 | Описание                                                                                                                      |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**ивскпродукт**](/windows/desktop/api/Iwscapi/nn-iwscapi-iwscproduct)<br/>         | предоставляет методы для получения сведений о продукте для отдельного поставщика, взаимодействующего с центром Безопасность Windows.<br/> |
| [**ивскпродуктлист**](/windows/desktop/api/iwscapi/nn-iwscapi-iwscproductlist)<br/> | Предоставляет методы для получения сведений о продукте для выбранного типа поставщиков, установленных на компьютере.<br/>         |
