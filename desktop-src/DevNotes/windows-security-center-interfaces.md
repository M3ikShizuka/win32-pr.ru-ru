# <a name="windows-security-center-interfaces"></a>Интерфейсы центра безопасности Windows

Для использования с центром безопасности Windows доступны следующие интерфейсы.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                 | Описание                                                                                                                      |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**ивскпродукт**](/windows/desktop/api/Iwscapi/nn-iwscapi-iwscproduct)<br/>         | Предоставляет методы для получения сведений о продукте для отдельного поставщика, взаимодействующего с центром безопасности Windows.<br/> |
| [**ивскпродуктлист**](/windows/desktop/api/iwscapi/nn-iwscapi-iwscproductlist)<br/> | Предоставляет методы для получения сведений о продукте для выбранного типа поставщиков, установленных на компьютере.<br/>         |
