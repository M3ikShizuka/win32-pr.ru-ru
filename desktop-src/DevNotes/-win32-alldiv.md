# <a name="_alldiv-routine"></a>\_подпрограммы аллдив

Делит два целых числа **лонглонг** . Например, чтобы разделить два значения Int64, компилятор может создать вызов \_ подпрограммы аллдив.

## <a name="remarks"></a>Комментарии

\_подпрограммы аллдив — это вспомогательная подпрограммы для компилятора C. Вызывает ли компилятор \_ подпрограммы аллдив полностью независимо от набора оптимизации.
