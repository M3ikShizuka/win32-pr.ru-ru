# <a name="orqueryinfokey-function"></a>Функция Оркуеринфокэй

Извлекает сведения об указанном разделе реестра в автономном кусте реестра.

## <a name="syntax"></a>Синтаксис


```C++
DWORD ORQueryInfoKey(
  _In_        ORHKEY    Handle,
  _Out_opt_   PWSTR     lpClass,
  _Inout_opt_ PDWORD    lpcClass,
  _Out_opt_   PDWORD    lpcSubKeys,
  _Out_opt_   PDWORD    lpcMaxSubKeyLen,
  _Out_opt_   PDWORD    lpcMaxClassLen,
  _Out_opt_   PDWORD    lpcValues,
  _Out_opt_   PDWORD    lpcMaxValueNameLen,
  _Out_opt_   PDWORD    lpcMaxValueLen,
  _Out_opt_   PDWORD    lpcbSecurityDescriptor,
  _Out_opt_   PFILETIME lpftLastWriteTime
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Обработчик* \[ окне\]
</dt> <dd>

Маркер открытого раздела реестра в автономном кусте реестра.

</dd> <dt>

*лпкласс* \[ out, необязательно\]
</dt> <dd>

Указатель на буфер, который получает класс ключа. Этот параметр может принимать значение **NULL**.

</dd> <dt>

*лпккласс* \[ в, out, необязательно\]
</dt> <dd>

Указатель на переменную, указывающую размер буфера, на который указывает параметр *лпкласс* в символах.

Размер должен включать завершающий нуль символ. При возврате функции эта переменная содержит размер строки класса, которая хранится в буфере. Возвращенное число не содержит завершающего нуль символа. Если буфер недостаточно велик, функция возвращает ошибку \_ больше \_ данных, а переменная содержит размер строки в символах без учета завершающего нуль-символа.

Если *лпкласс* имеет **значение NULL**, *лпккласс* может иметь **значение NULL**.

Если параметр *лпкласс* является допустимым адресом, а параметр *лпккласс* — нет (например, если параметр *лпккласс* имеет **значение NULL**), функция возвращает ошибку " \_ Недопустимый \_ параметр".

</dd> <dt>

*лпксубкэйс* \[ out, необязательно\]
</dt> <dd>

Указатель на переменную, которая получает количество подразделов, содержащихся в указанном ключе. Этот параметр может принимать значение **NULL**.

</dd> <dt>

*лпкмакссубкэйлен* \[ out, необязательно\]
</dt> <dd>

Указатель на переменную, которая получает размер подраздела ключа с самым длинным именем в символах Юникода, не включая завершающий нуль-символ. Этот параметр может принимать значение **NULL**.

</dd> <dt>

*лпкмакскласслен* \[ out, необязательно\]
</dt> <dd>

Указатель на переменную, которая получает размер самой длинной строки, указывающей класс подраздела, в символах Юникода. Возвращенное число не содержит завершающего нуль символа. Этот параметр может принимать значение **NULL**.

</dd> <dt>

*лпквалуес* \[ out, необязательно\]
</dt> <dd>

Указатель на переменную, которая получает количество значений, связанных с ключом. Этот параметр может принимать значение **NULL**.

</dd> <dt>

*лпкмаксвалуенамелен* \[ out, необязательно\]
</dt> <dd>

Указатель на переменную, которая получает размер имени длинного значения ключа в символах Юникода. Размер не содержит завершающего нуль символа. Этот параметр может принимать значение **NULL**.

</dd> <dt>

*лпкмаксвалуелен* \[ out, необязательно\]
</dt> <dd>

Указатель на переменную, которая получает размер наиболее длинного компонента данных из значений ключа в байтах. Этот параметр может принимать значение **NULL**.

</dd> <dt>

*лпкбсекуритидескриптор* \[ out, необязательно\]
</dt> <dd>

Указатель на переменную, которая получает размер дескриптора безопасности ключа в байтах. Этот параметр может принимать значение **NULL**.

</dd> <dt>

*лпфтластвритетиме* \[ out, необязательно\]
</dt> <dd>

Указатель на структуру [fileTime](/windows/win32/api/minwinbase/ns-minwinbase-filetime) , которая получает время последней записи. Этот параметр может принимать значение **NULL**.

Функция задает элементы структуры [fileTime](/windows/win32/api/minwinbase/ns-minwinbase-filetime) , чтобы указать время последнего изменения ключа или любой из его записей значений.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается успешно, возвращается значение ошибки \_ Success.

Если функция завершается ошибкой, возвращаемое значение является ненулевым кодом ошибки, определенным в файле Winerror. h. [](/windows/win32/api/winbase/nf-winbase-formatmessage) \_ \_ \_ Для получения обобщенного описания ошибки можно использовать функцию FormatMessage с флагом формата Message от System.

Если буфер *лпкласс* слишком мал для получения имени класса, функция возвращает данные об ошибке \_ \_ .
