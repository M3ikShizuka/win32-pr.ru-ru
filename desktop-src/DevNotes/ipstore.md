# <a name="ipstore-interface"></a>Интерфейс Ипсторе

\[Защищенное хранилище (PStore) доступно для использования в Windows Server 2003 и Windows XP. Она доступна только для операций чтения в Windows Server 2008 и Windows Vista, но может быть недоступна в последующих версиях. PStore использует старую реализацию защиты данных. Разработчикам настоятельно рекомендуется использовать преимущества более надежной защиты данных, предоставляемые функциями [**CryptProtectData**](/windows/win32/api/dpapi/nf-dpapi-cryptprotectdata) и [**CryptUnprotectData**](/windows/win32/api/dpapi/nf-dpapi-cryptunprotectdata) .\]

\[Этот интерфейс может быть изменен или недоступен в будущих версиях Windows.\]

Предоставляет стандартные методы COM для управления элементами данных защищенного хранилища. Метод [**псторекреатеинстанце**](pstorecreateinstance.md) возвращает указатель на этот интерфейс.

## <a name="members"></a>Элементы

Интерфейс **ипсторе** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Ипсторе** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ипсторе** содержит следующие методы.



| Метод                                                   | Описание                                                                                                           |
|:---------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------|
| [**клосеитем**](ipstore-closeitem.md)                   | Закрывает указанный элемент данных из защищенного хранилища.<br/>                                                       |
| [**креатесубтипе**](ipstore-createsubtype.md)           | Создает указанный подтип в указанном типе.<br/>                                                   |
| [**CreateType**](ipstore-createtype.md)                 | Создает указанный тип с указанным именем.<br/>                                                        |
| [**DeleteItem**](ipstore-deleteitem.md)                 | Удаляет указанный элемент из защищенного хранилища.<br/>                                                     |
| [**делетесубтипе**](ipstore-deletesubtype.md)           | Удаляет указанный подтип элемента из защищенного хранилища.<br/>                                             |
| [**делететипе**](ipstore-deletetype.md)                 | Удаляет указанный тип из защищенного хранилища.<br/>                                                     |
| [**енумитемс**](ipstore-enumitems.md)                   | Возвращает указатель интерфейса подтипа для перечисления элементов в защищенной базе данных хранилища.<br/>        |
| [**енумсубтипес**](ipstore-enumsubtypes.md)             | Возвращает интерфейс для перечисления подтипов типов, зарегистрированных в настоящий момент в защищенной базе данных.<br/> |
| [**енумтипес**](ipstore-enumtypes.md)                   | Возвращает интерфейс для перечисления типов, зарегистрированных в настоящий момент в защищенной базе данных.<br/>             |
| [**GetInfo**](ipstore-getinfo.md)                       | Извлекает сведения о поставщике хранилища.<br/>                                                          |
| [**жетпровпарам**](ipstore-getprovparam.md)             | Не реализован.<br/>                                                                                           |
| [**жетсубтипеинфо**](ipstore-getsubtypeinfo.md)         | Извлекает сведения, связанные с подтипом.<br/>                                                           |
| [**GetTypeInfo**](ipstore-gettypeinfo.md)               | Извлекает сведения, связанные с типом.<br/>                                                              |
| [**опенитем**](ipstore-openitem.md)                     | Открывает элемент для нескольких обращений.<br/>                                                                       |
| [**реадакцессрулесет**](ipstore-readaccessruleset.md)   | Не реализован.<br/>                                                                                           |
| [**ReadItem**](ipstore-readitem.md)                     | Считывает указанный элемент данных из защищенного хранилища.<br/>                                                      |
| [**сетпровпарам**](ipstore-setprovparam.md)             | Задает сведения о заданных параметрах.<br/>                                                                  |
| [**вритеакцессрулесет**](ipstore-writeaccessruleset.md) | Не реализован.<br/>                                                                                           |
| [**вритеитем**](ipstore-writeitem.md)                   | Записывает элемент данных в защищенное хранилище.<br/>                                                                   |
