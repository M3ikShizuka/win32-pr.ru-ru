# <a name="sdbquerydataextagid-function"></a>Функция Сдбкуеридатаекстагид

Извлекает данные из указанных записей, принадлежащих записи EXE.

## <a name="syntax"></a>Синтаксис


```C++
DWORD WINAPI SdbQueryDataExTagID(
  _In_        PDB     pdb,
  _In_        TAGID   tiExe,
  _In_opt_    LPCTSTR lpszDataName,
  _Out_opt_   LPDWORD lpdwDataType,
  _Out_       LPVOID  lpBuffer,
  _Inout_opt_ LPDWORD lpcbBufferSize,
  _Out_       TAGID   *ptiData
);
```

Эта функция возвращает одно из следующих значений.
