# <a name="windows-security-center-functions"></a>Функции центра безопасности Windows

В центре безопасности Windows используются следующие функции.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                           | Описание                                                                                                                                                                              |
|---------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**вскжетсекуритипровидерхеалс**](/windows/desktop/api/Wscapi/nf-wscapi-wscgetsecurityproviderhealth)<br/> | Возвращает совокупное состояние работоспособности категорий поставщиков безопасности, представленных указанными значениями [**перечисления \_ \_ поставщика безопасности WSC**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider) .<br/> |
| [**вскрегистерфорчанжес**](/windows/desktop/api/Wscapi/nf-wscapi-wscregisterforchanges)<br/>               | Регистрирует функцию обратного вызова, которая должна выполняться, когда центр безопасности Windows (WSC) обнаруживает изменение, которое может повлиять на работоспособность одного из поставщиков безопасности.<br/>                    |
| [**вскунрегистерчанжес**](/windows/desktop/api/Wscapi/nf-wscapi-wscunregisterchanges)<br/>                 | Отменяет регистрацию обратного вызова, созданную путем вызова функции [**вскрегистерфорчанжес**](/windows/desktop/api/Wscapi/nf-wscapi-wscregisterforchanges) .<br/>                                               |
