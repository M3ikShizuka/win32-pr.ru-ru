# <a name="wm_ctlcolor-message"></a>\_Сообщение КТЛКОЛОР WM

сообщение **WM \_ ктлколор** используется в 16-разрядных версиях Windows для изменения цветовой схемы списков, полей со списком, окон сообщений, элементов управления "кнопка", "правка", статических элементов управления и диалоговых окон.

> [!Note]  
> сведения, относящиеся к этому сообщению и 32-разрядным версиям Windows, см. в разделе примечания.

 


```C++
  WM_CTLCOLOR

  WPARAM wParam;
  LPARAM lParam;
    
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*HWND* 
</dt> <dd>

Маркер окна назначения.

</dd> <dt>

*wParam* 
</dt> <dd>

Маркер контекстного интерфейса (DC).

</dd> <dt>

*lParam* 
</dt> <dd>

Маркер дочернего окна (элемента управления).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если приложение обрабатывает это сообщение, оно возвращает дескриптор кисти. Система использует кисть для рисования фона элемента управления.

сообщение **WM \_ ктлколор** из 16-разрядной Windows заменено более конкретными уведомлениями. В число этих замен входят следующие.

-   [**WM \_ ктлколорбтн**](../controls/wm-ctlcolorbtn.md)

-   [**WM \_ ктлколордлг**](../dlgbox/wm-ctlcolordlg.md)
-   [**WM \_ ктлколорлистбокс**](../controls/wm-ctlcolorlistbox.md)
-   [**WM \_ ктлколорскроллбар**](../controls/wm-ctlcolorscrollbar.md)
