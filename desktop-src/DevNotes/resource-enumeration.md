# <a name="resource-enumeration"></a>Перечисление ресурсов

Разработчики могут использовать перечисление ресурсов в рамках инструментирования, предоставляющего определенные типы информации о своих приложениях.

корпорация майкрософт создает инструментирование в Windowsной операционной системе, но эти функции тесно связаны с операционной системой, что они могут быть ограниченными в использовании разработчиками приложений. Корпорация Майкрософт делает доступным интерфейс прикладного программирования (API) для перечисления ресурсов, который может использоваться со следующими ресурсами инструментирования: выделением кучи и обработкой данных трассировки.

В следующих разделах описывается, как использовать этот API в приложениях.

-   [Сведения о перечислении ресурсов](about-resource-enumeration.md)
-   [Ссылка на перечисление ресурсов](resource-enumeration-reference.md)
