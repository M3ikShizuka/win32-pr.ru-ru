# <a name="pstoreenumproviders-function"></a>Функция Псторинумпровидерс

\[Защищенное хранилище (PStore) доступно для использования в Windows Server 2003 и Windows XP. Она доступна только для операций чтения в Windows Server 2008 и Windows Vista, но может быть недоступна в последующих версиях. PStore использует старую реализацию защиты данных. Разработчикам настоятельно рекомендуется использовать преимущества более надежной защиты данных, предоставляемые функциями [**CryptProtectData**](/windows/win32/api/dpapi/nf-dpapi-cryptprotectdata) и [**CryptUnprotectData**](/windows/win32/api/dpapi/nf-dpapi-cryptunprotectdata) .\]

Возвращает объект перечислителя, который можно использовать, в свою очередь, для перечисления поставщиков защищенного хранилища, установленных в данный момент в системе.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT PStoreEnumProviders(
   DWORD                dwFlags,
   IEnumPStoreProviders **ppenum
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*dwFlags* 
</dt> <dd>

Этот параметр не используется и должен быть равен нулю.

</dd> <dt>

*ппенум* 
</dt> <dd>

Указатель на указатель на интерфейс [**иенумпсторепровидерс**](ienumpstoreproviders.md) , который может использоваться для перечисления установленных поставщиков.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция возвращает **значение HRESULT**.

## <a name="remarks"></a>Комментарии

Компонент защищенного хранилища имеет архитектуру на основе поставщика. Приложения, использующие защищенное хранилище, могут указывать, какой из установленных поставщиков следует использовать при хранении и извлечении данных.

Функция **псторинумпровидерс** используется для перечисления установленных поставщиков защищенного хранилища. Каждый поставщик идентифицируется по глобальному уникальному идентификатору (GUID).

До этого времени только один поставщик защищенного хранилища когда-либо записался. Учитывая, что служба защищенного хранилища в настоящее время устарела, очень маловероятно, что все дополнительные поставщики будут созданы. В результате эта функция не должна использоваться ни в каких целях.

Эта функция не имеет связанной библиотеки импорта; его необходимо вызвать с помощью функций [**LoadLibrary**](/windows/win32/api/libloaderapi/nf-libloaderapi-loadlibrarya) и [**GetProcAddress**](/windows/win32/api/libloaderapi/nf-libloaderapi-getprocaddress) .
