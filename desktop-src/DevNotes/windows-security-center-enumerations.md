# <a name="windows-security-center-enumerations"></a>Перечисления центра безопасности Windows

В центре безопасности Windows используются следующие перечисления.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                | Описание                                                                                                      |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ состояние продукта безопасности \_ WSC**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_product_state)<br/>       | Определяет текущее состояние продукта безопасности, доступного для центра безопасности Windows. <br/> |
| [**\_поставщик безопасности \_ WSC**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider)<br/>                  | Определяет все службы, отслеживаемые центром безопасности Windows (WSC).<br/>                         |
| [**\_ \_ работоспособность поставщика безопасности WSC \_**](/windows/desktop/api/Wscapi/ne-wscapi-wsc_security_provider_health)<br/>   | Определяет возможные состояния для любой службы, отслеживаемой центром безопасности Windows (WSC).<br/>               |
| [**\_ \_ состояние подписи безопасности \_ WSC**](/windows/desktop/api/Iwscapi/ne-iwscapi-wsc_security_signature_status)<br/> | Сообщает о текущей версии продукта безопасности в центре безопасности Windows. <br/>               |
