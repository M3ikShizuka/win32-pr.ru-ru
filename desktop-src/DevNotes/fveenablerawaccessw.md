# <a name="fveenablerawaccessw-function"></a>Функция Фвинаблеравакцессв

Включает или отключает чтение и запись секторов диска.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT FveEnableRawAccessW(
  _In_ PCWSTR VolumeName,
  _In_ BOOL   Enabled
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

Переданное  \[ окне\]
</dt> <dd>

Уникальный идентификатор тома диска.

</dd> <dt>

*Включено* \[ окне\]
</dt> <dd>

Если **значение — true**, разрешает доступ к необработанному тому. Если **значение равно false**, доступ к необработанному тому запрещен. Если необработанный доступ уже включен, но это значение равно **false**, том повторно подключен и повторно проверяется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция возвращает один из следующих кодов или другой код ошибки в случае сбоя.



| Возвращаемый код и значение                                                                                                                                                           | Описание                                                                        |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                           | Функция выполнена успешно.<br/>                                            |
| <dl> <dt>**С \_ FALSE**</dt> <dt>1 (0x1)</dt> </dl>                        | Параметр Enabled имеет **значение false** , а том не находится в режиме доступа RAW.<br/> |
| <dl> <dt>**Д \_ ACCESSDENIED**</dt> <dt>2147942405 (0x80070005)</dt> </dl> | Том не может быть заблокирован.<br/>                                            |
