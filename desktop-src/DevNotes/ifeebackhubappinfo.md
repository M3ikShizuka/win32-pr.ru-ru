# <a name="ifeedbackhubappinfo-interface"></a>Интерфейс Ифидбаккхубаппинфо

Этот API недоступен для всех приложений. Если приложение не будет специально подготовлено корпорацией Майкрософт, вызовы этих API будут завершаться ошибкой во время выполнения.

## <a name="members"></a>Элементы

Интерфейс **ифидбаккхубаппинфо** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) , но не содержит дополнительных элементов.

|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows 10 \[ только классические приложения\]<br/>          |
| Минимальная версия сервера<br/> | Windows Server 2016 \[ только классические приложения\]<br/> |
