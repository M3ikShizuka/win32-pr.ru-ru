# <a name="cabinet-api-structures"></a>Структуры API CAB-файлов

В этом разделе описаны следующие структуры API CAB-файлов:



| Структура                                              | Описание                                                                                     |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**кабинетдллверсионинфо**](cabinetdllversioninfo.md) | Содержит сведения о версии для Cabinet.dll.<br/>                                    |
| [**ккаб**](/windows/desktop/api/Fci/ns-fci-ccab)                                   | Содержит сведения о CAB-файле.<br/>                                                        |
| [**ИНТЕГРИРОВАН**](/windows/win32/api/fdi_fci_types/ns-fdi_fci_types-erf)                                     | Возвращает сведения об ошибке из FCI/FDI. Вызывающий объект не должен изменять эту структуру.<br/> |
| [**фдикабинетинфо**](/windows/desktop/api/Fdi/ns-fdi-fdicabinetinfo)               | Содержит сведения о конкретном CAB-файле.<br/>                                    |
| [**фдинотификатион**](/windows/desktop/api/Fdi/ns-fdi-fdinotification)             | Укажите сведения для [**фнфдинотифи**](/windows/desktop/api/fdi/nf-fdi-fnfdinotify).<br/>                           |
| [**СЕССИИ**](session.md)                             | Содержит сведения о текущем сеансе.<br/>                                      |
