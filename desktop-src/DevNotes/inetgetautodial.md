# <a name="inetgetautodial-function"></a>Функция Инетжетаутодиал

\[Эта функция не поддерживается и может быть изменена или недоступна в будущих версиях Windows. \]

Функция **инетжетаутодиал** Возвращает параметры автонабора из реестра.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT InetGetAutodial(
  _Out_ LPBOOL lpfEnable,
  _Out_ LPSTR  lpszEntryName,
  _In_  DWORD  cbEntryNameSize
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*лпфенабле* \[ заполняет\]
</dt> <dd>

Указывает, включен ли автонабор. Значение **true** при возврате означает, что автонабор включен.

</dd> <dt>

*лпсзентринаме* \[ заполняет\]
</dt> <dd>

При возврате содержит имя записи телефонной книги, настроенной для автонабора.

</dd> <dt>

*кбентринамесизе* \[ окне\]
</dt> <dd>

Размер буфера, выделенного вызывающим объектом для имени записи телефонной книги.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция может возвращать одно из этих значений.



| Код возврата                                                                                                | Описание                                                                                                                                               |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**Ошибка при \_ успешном выполнении**</dt> </dl>              | Вызов выполнен успешно.<br/>                                                                                                                       |
| <dl> <dt>**ошибочные \_ \_ аргументы**</dt> </dl>       | *лпфенабле* или *лпсзентринаме* содержит **пустой** указатель, или значение *кбентринамесизе* равно нулю.<br/>                                         |
| <dl> <dt>**Ошибка \_ не \_ хватает \_ памяти**</dt> </dl>  | Недостаточно памяти для выделения внутренних буферов.<br/>                                                                                    |
| <dl> <dt>**Ошибка \_ недостаточного \_ буфера**</dt> </dl> | *кбентринамесизе* не указывает, что буфер, на который указывает *лпсзентринаме* , достаточно велик для получения имени записи телефонной книги.<br/> |
