# <a name="windows-security-center"></a>Центр обеспечения безопасности Windows

Безопасность Windows Center (WSC) — это комплексное средство создания отчетов, помогающее пользователям устанавливать и обслуживать защитные уровни безопасности на компьютерах. после установки уровня безопасности Безопасность Windows Center инконспикуаус, так как он отслеживает состояние работоспособности компьютера. Однако если существуют уязвимости, WSC предоставляет предупреждения и рекомендации, помогающие пользователю добиться безопасного состояния, которое будет выводится для пользователя в центре поддержки.

WSC, в свою очередь, передает эти обновления в центр поддержки, где они в конечном итоге отображаются конечному пользователю.
