# <a name="sdbinitdatabase-function"></a>Функция Сдбинитдатабасе

Открывает базу данных оболочек совместимости.

## <a name="syntax"></a>Синтаксис


```C++
HSDB WINAPI SdbInitDatabase(
  _In_ DWORD   dwFlags,
  _In_ LPCTSTR pszDatabasePath
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Этот параметр задает формат пути в параметре *псздатабасепас* . Может быть одним из указанных далее.



| Значение                                                                                                                                                                                                                                                      | Значение                                                                                                |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| <span id="HID_DOS_PATHS"></span><span id="hid_dos_paths"></span><dl> <dt>**HID \_ \_Пути к DOS**</dt> <dt>0x00000001</dt> </dl>                             | Путь в стиле MS-DOS.<br/>                                                                       |
| <span id="HID_DATABASE_FULLPATH"></span><span id="hid_database_fullpath"></span><dl> <dt>**HID \_ \_FullPath, база данных**</dt> , <dt>0x00000002</dt> </dl>     | Полный путь.<br/>                                                                                |
| <span id="HID_NO_DATABASE"></span><span id="hid_no_database"></span><dl> <dt>**HID \_ БЕЗ \_ базы данных**</dt> <dt>0x00000004</dt> </dl>                       | Параметр *псздатабасепас* игнорируется, и база данных не открывается.<br/>                       |
| <span id="HID_DATABASE_TYPE_MASK"></span><span id="hid_database_type_mask"></span><dl> <dt>**HID \_ 0xF00F0000 \_ типа \_ маски базы данных**</dt> <dt></dt> </dl> | Этот параметр указывает предопределенную базу данных. Параметр *псздатабасепас* игнорируется.<br/> |



 

Если *dwFlags* содержит **\_ \_ \_ маску типа данных HID**, этот параметр также может включать одно из следующих значений.



| Значение                                                                                                                                                                                                                                                               | Значение                                       |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------|
| <span id="SDB_DATABASE_MAIN_SHIM"></span><span id="sdb_database_main_shim"></span><dl> <dt>**SDB \_ \_Основная \_ оболочка СОВМЕСТИМОСТИ базы данных**</dt> <dt>0x80030000</dt> </dl>          | База данных оболочки совместимости приложений.<br/>         |
| <span id="SDB_DATABASE_MAIN_MSI"></span><span id="sdb_database_main_msi"></span><dl> <dt>**SDB \_ Основной 0x80020000 базы данных \_ \_ MSI**</dt> <dt></dt> </dl>             | База данных MSI.<br/>                      |
| <span id="SDB_DATABASE_MAIN_DRIVERS"></span><span id="sdb_database_main_drivers"></span><dl> <dt>**SDB \_ \_Основные \_ драйверы базы данных**</dt> <dt>0x80040000</dt> </dl> | База данных драйверов для блокировки.<br/> |



 

</dd> <dt>

*псздатабасепас* \[ окне\]
</dt> <dd>

Путь к базе данных. Этот параметр может иметь **значение NULL** , если параметр *dwFlags* указывает предопределенную базу данных.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Функция возвращает маркер открытой базы данных.
