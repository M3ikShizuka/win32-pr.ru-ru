# <a name="addhijridate"></a>аддхижридате

**\\Международная панель управления \\ HKCU**



| Тип данных | Диапазон                      | Значение по умолчанию |
|-----------|----------------------------|---------------|
| REG \_ SZ   | *Допустимое значение корректировки* | (нет)        |



 

## <a name="description"></a>Описание

Указывает корректировки даты в календаре Хиджра.

Календарь Хиджра — лунный календарь, используемый в мусульманском мире. Месяцы начинаются и заканчиваются в соответствии с Декри, основанным на наблюдении за новым лунами. Трудно правильно спрогнозировать начало и конец месяцев, и существуют региональные вариации, поэтому в календарь вносится корректировка от нуля до двух дней.

Параметр реестра **аддхижридате** сохраняет это значение корректировки следующим образом.



| Значение          | Значение                                                                                                                                                                                                                               |
|----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Аддхижридате-2 | Вычесть два дня.                                                                                                                                                                                                                    |
| аддхижридате   | Вычесть один день.                                                                                                                                                                                                                     |
| (пусто)        | Корректировка не требуется. (Если дата не была скорректирована, эта запись не отображается в реестре. Если дата была скорректирована, а затем восстановлена до исходного значения, эта запись появляется в реестре без значения.) |
| Аддхижридате + 1 | Добавьте один день.                                                                                                                                                                                                                          |
| Аддхижридате + 2 | Добавьте два дня.                                                                                                                                                                                                                         |



 

Эта запись не существует в реестре по умолчанию. Его можно добавить с помощью редактора реестра Regedit.exe или с помощью метода Change, описанного ниже.

## <a name="change-method"></a>Изменение метода

Чтобы изменить значение этой записи или добавить ее в реестр, сначала установите файлы для наборов сложных символов и языков с письмом справа налево. На панели управления щелкните **региональные и языковые параметры**, а затем перейдите на вкладку **языки** . В разделе **Дополнительная языковая поддержка** установите флажок, чтобы **установить файлы для наборов сложных символов и языков с письмом справа налево (включая тайский)**. Нажмите кнопку **ОК**.

Чтобы изменить значение этой записи или добавить ее в реестр, в панели управления щелкните **региональные и языковые параметры**. В поле **стандарты и форматы** выберите языковой стандарт, в котором используется календарь Хиджра. Нажмите кнопку " **настроить** ", а затем перейдите на вкладку " **Дата** ". В раскрывающемся списке **изменить дату календаря на:** выберите соответствующее значение. Нажмите кнопку **ОК**, а затем еще раз кнопку **ОК** .

## <a name="activation-method"></a>Метод активации

Изменения этой записи, внесенные через панель управления, вступают в силу немедленно.
