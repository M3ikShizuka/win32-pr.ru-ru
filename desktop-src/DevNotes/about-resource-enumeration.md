# <a name="about-resource-enumeration"></a>Сведения о перечислении ресурсов

Перечисление ресурсов обеспечивает закрытие и точное представление действий инструментирования, выполняемых при выполнении приложения. Сведения инструментирования могут быть абстрактными и классифицированы как набор элементов, зависящих от процесса, в рамках процесса отладки.

Средствам, таким как отладчики, часто требуется доступ к информации инструментирования, но они могут оказаться недоступными. Чтобы решить эту проблему, функция [**верифиеренумератересаурце**](/windows/desktop/api/Avrfsdk/nf-avrfsdk-verifierenumerateresource) извлекает из приложений данные этого типа, чтобы предоставить более подробные сведения о контексте для отлаживаемой проблемы.
