# <a name="csharelocknh-methods"></a>Методы Кшарелоккнх

Группа методов, используемых для управления блокировками.

## <a name="methods"></a>Методы

Ниже приведены методы, экспортируемые с помощью Rwnh.dll.



| Метод                                                                   | Описание                                                     |
|--------------------------------------------------------------------------|-----------------------------------------------------------------|
| [**ексклусивелокк**](csharelocknh--exclusivelock.md)                     | Получает блокировку потоков чтения/записи.                                  |
| [**ексклусиветопартиал**](csharelocknh--exclusivetopartial.md)           | Изменяет состояние.                                              |
| [**ексклусивеунлокк**](csharelocknh--exclusiveunlock.md)                 | Снимает блокировку.                                                |
| [**фирстпартиалтоексклусиве**](csharelocknh--firstpartialtoexclusive.md) | Используется при преобразовании частичной блокировки в монопольную блокировку.         |
| [**партиаллокк**](csharelocknh--partiallock.md)                         | Предотвращает получение блокировки более чем одним потоком. |
| [**партиалунлокк**](csharelocknh--partialunlock.md)                     | Освобождает частичную блокировку.                                        |
| [**шарелокк**](csharelocknh--sharelock.md)                             | Получает блокировку для общего режима.                                 |
| [**шареунлокк**](csharelocknh--shareunlock.md)                         | Снимает блокировку с общего режима.                               |
| [**шаредтопартиал**](csharelocknh--sharedtopartial.md)                 | Получает частичную блокировку.                                         |
| [**трексклусивелокк**](csharelocknh--tryexclusivelock.md)               | Получает монопольную блокировку.                                     |
