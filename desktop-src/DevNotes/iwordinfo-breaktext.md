# <a name="iwordinfobreaktext-method"></a>Метод Ивординфо:: Бреактекст

\[этот метод устарел и не поддерживается в Windows Vista.\]

Анализирует текст для поиска слов и предоставляет результаты для объекта [вордсинк](/previous-versions//ms691570(v=vs.85)) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT BreakText(
  [in] TEXT_SOURCE *pTextSource,
  [in] IWordSink   *pWordSink,
  [in] DWORD       fBreakMode
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*птекстсаурце* \[ окне\]
</dt> <dd>

Указатель на структуру [ \_ источника текста](/previous-versions//ms690919(v=vs.85)) , содержащую текст в Юникоде.

</dd> <dt>

*пвордсинк* \[ окне\]
</dt> <dd>

Указатель на объект [вордсинк](/previous-versions//ms691570(v=vs.85)) , который получает и обрабатывает слова, созданные этим методом. Если этот параметр имеет **значение NULL**, метод не разбивает строку на слова.

</dd> <dt>

*фбреакмоде* \[ окне\]
</dt> <dd>

Режим приостановки выполнения. Этот параметр может принимать одно из указанных ниже значений.



| Значение                                                                                                                                                                                                                                                                                   | Значение                                                                                                     |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| <span id="IWORDINFO_BREAKTEXTMODE_DICTFORM"></span><span id="iwordinfo_breaktextmode_dictform"></span><dl> <dt>**Ивординфо \_ БРЕАКТЕКСТМОДЕ \_ диктформ**</dt> <dt>0x00000002</dt> </dl> | Word разбивает текстовую строку и передает словарь слов в объект **вордсинк** .<br/> |
| <span id="IWORDINFO_BREAKTEXTMODE_SMARTSEL"></span><span id="iwordinfo_breaktextmode_smartsel"></span><dl> <dt>**Ивординфо \_ БРЕАКТЕКСТМОДЕ \_ смартсел**</dt> <dt>0x00000001</dt> </dl> | Word разбивает текстовую строку и передает слова в объект **вордсинк** .<br/>                        |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает одно из следующих значений.



| Код возврата                                                                            | Описание                                                                                             |
|----------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>   | Операция успешно выполнена. Отсутствует текст, доступный для заполнения буфера *птекстсаурце* .<br/> |
| <dl> <dt>**\_Ошибка E**</dt> </dl> | Параметр *птекстсаурце* имеет **значение NULL**.<br/>                                                     |



 

## <a name="remarks"></a>Remarks

Используйте элемент **пфнфиллтекстбуффер** структуры **\_ источника текста** для пополнения исходного текста. Этот метод должен поддерживать все возвращаемые значения функции обратного вызова **пфнфиллтекстбуффер** . При возникновении ошибки Завершите обработку текста в буфере перед обработкой ошибки.
