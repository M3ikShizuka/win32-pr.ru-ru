# <a name="id3dx10threadpumpgetworkitemcount-method"></a>Метод ID3DX10ThreadPump:: Жетворкитемкаунт

Получение количества рабочих элементов, находящихся в настоящий момент в конвейере потоков.

## <a name="syntax"></a>Синтаксис


```C++
UINT GetWorkItemCount();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **uint**](../winprog/windows-data-types.md)**

Количество рабочих элементов, которые в настоящий момент находятся в очереди в конвейере потоков.
