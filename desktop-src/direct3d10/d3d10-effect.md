# <a name="d3d10_effect-constants"></a>\_Константы эффектов D3D10

Эти константы используются при создании эффектов для определения поведения компиляции или поведения воздействия на среду выполнения.



| \#определенно                                 | Значение        | Описание                                                                                                                                                                                                                                                 |
|------------------------------------------|--------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_ \_ \_ Дочерний \_ Результат компиляции D3D10 Effect    | 1 << 0 | Скомпилируйте файл. FX для дочернего действия. Дочерние эффекты не имеют инициализаций для общих значений, так как они инициализируются в пуле эффектов.                                                                                                           |
| \_Компиляция эффектов \_ D3D10 \_ Разрешать выполнение \_ операций с высокой скоростью \_ | 1 << 1 | По умолчанию включен режим производительности. Режим производительности запрещает изменять объекты изменяемого состояния, предотвращая отображение нелитеральных выражений в определениях объектов состояния. При указании этого флага режим будет отключен и разрешено использовать изменяемые объекты состояния. |
| D3D10 \_ воздействие на \_ один \_ поток          | 1 << 3 | Не пытайтесь выполнить синхронизацию с другими потоками, загружая эффекты в один и тот же пул.                                                                                                                                                                        |



 

Эти константы определяются как макросы в d3d10effect. h.
