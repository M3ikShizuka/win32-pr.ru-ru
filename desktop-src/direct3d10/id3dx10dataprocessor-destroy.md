# <a name="id3dx10dataprocessordestroy-method"></a>ID3DX10DataProcessor: метод:D естрой

Используется [**интерфейсом ID3DX10ThreadPump**](id3dx10threadpump.md) для уничтожения процессора после завершения рабочего элемента.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Destroy();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **HRESULT**](https://msdn.microsoft.com/library/Bb401631(v=MSDN.10).aspx)**

Возвращаемое значение является одним из значений, перечисленных в [кодах возврата Direct3D 10](d3d10-graphics-reference-returnvalues.md).
