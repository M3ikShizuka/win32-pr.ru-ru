# <a name="id3dx10fontgetdc-method"></a>Метод ID3DX10Font:: GetDC

Возвращает маркер для контекста дисплея устройства (DC), для которого установлен шрифт.

## <a name="syntax"></a>Синтаксис


```C++
HDC GetDC();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **HDC**](../winprog/windows-data-types.md)**

Обработайте с КОНТРОЛЛЕРом экрана.
