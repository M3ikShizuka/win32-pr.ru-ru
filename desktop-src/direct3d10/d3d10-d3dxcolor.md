# <a name="d3dxcolor-structure-d3dx10mathh"></a>Структура D3DXCOLOR (D3DX10Math. h)

Описывает значения цвета.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct D3DXCOLOR {
  FLOAT r;
  FLOAT g;
  FLOAT b;
  FLOAT a;
} D3DXCOLOR, *LPD3DXCOLOR;
```
