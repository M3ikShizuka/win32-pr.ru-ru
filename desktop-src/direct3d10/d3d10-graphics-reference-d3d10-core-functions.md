# <a name="direct3d-10-core-functions"></a>Основные функции Direct3D 10

В этом разделе содержатся сведения о следующих основных функциях.



| Функции                                                                | Описание                                     |
|--------------------------------------------------------------------------|-------------------------------------------------|
| [**D3D10CreateBlob**](/windows/desktop/api/D3D10Misc/nf-d3d10misc-d3d10createblob)                               | Создайте буфер.                                |
| [**D3D10CreateDevice**](/windows/desktop/api/D3D10Misc/nf-d3d10misc-d3d10createdevice)                           | Создайте устройство Direct3D 10.                    |
| [**D3D10CreateDevice1**](/windows/desktop/api/D3D10_1/nf-d3d10_1-d3d10createdevice1)                         | Создайте устройство Direct3D 10,1.                  |

| [**D3D10CreateDeviceAndSwapChain1**](/windows/desktop/api/D3D10_1/nf-d3d10_1-d3d10createdeviceandswapchain1) | Создайте устройство Direct3D 10,1 и цепочку буферов. |
