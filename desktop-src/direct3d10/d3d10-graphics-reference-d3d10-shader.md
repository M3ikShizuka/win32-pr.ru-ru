# <a name="shader-reference-direct3d-10-graphics"></a>Справочник по шейдеру (графика Direct3D 10)

API Direct3D определяет несколько элементов API, которые помогают создавать программируемые шейдеры и управлять ими.

-   [Интерфейсы шейдеров](d3d10-graphics-reference-d3d10-shader-interfaces.md)
-   [Функции шейдера](d3d10-graphics-reference-d3d10-shader-functions.md)
-   [Структуры шейдеров](d3d10-graphics-reference-d3d10-shader-structures.md)
-   [Перечисления шейдеров](d3d10-graphics-reference-d3d10-shader-enums.md)
-   [Константы шейдера](d3d10-graphics-reference-d3d10-shader-constants.md)

Шейдеры — это исполняемые программы, управляющие шейдером вершин, [Этапы конвейера](d3d10-graphics-programming-guide-pipeline-stages.md)геометрического шейдера и шейдера пикселей. Шейдеры запрограммированы исключительно с помощью [HLSL](../direct3dhlsl/dx-graphics-hlsl-reference.md).
