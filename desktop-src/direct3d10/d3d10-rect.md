# <a name="d3d10_rect"></a>D3D10 \_ Rect

D3D10 \_ Rect объявляется следующим образом:

``` syntax
typedef RECT D3D10_RECT;
```

Дополнительные сведения о структуре этого прямоугольника GDI см. в разделе [Rect](/previous-versions//ms536136(v=vs.85)).

## <a name="remarks"></a>Remarks

Эта структура используется для прямоугольных прямоугольников [**ID3D10Device:: рсжетсЦиссорректс**](/windows/desktop/api/D3D10/nf-d3d10-id3d10device-rsgetscissorrects) и [**ID3D10Device:: рссетсЦиссорректс**](/windows/desktop/api/D3D10/nf-d3d10-id3d10device-rssetscissorrects).
