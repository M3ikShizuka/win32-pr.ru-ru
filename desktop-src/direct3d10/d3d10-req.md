# <a name="d3d10_req"></a>D3D10 \_ req

Константы ограничения ресурсов.



| \#определенно                                      | Значение | Описание                                                                                                                                                                                                                                                                                                                                                                                                                     |
|-----------------------------------------------|-------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_ \_ Уровни MIP D3D10 \_ req                       | 14    | Максимальное число уровней mipmap, которые может иметь ресурс текстуры.                                                                                                                                                                                                                                                                                                                                                                    |
| \_ \_ Размер ресурса требования \_ D3D10 \_ в \_ мегабайтах     | 128   | Максимальный возможный размер ресурса в мегабайтах. Приложения могут создавать ресурсы большего размера, чем максимальный размер ресурса на графическом оборудовании. Однако рекомендуется, чтобы приложения оставались ресурсами меньше, чем максимальный размер ресурса, чтобы получить максимальную степень совместимости между поставщиками графики. Среда выполнения гарантирует, что выделение памяти в пределах максимального размера ресурсов поддерживается всеми аппаратными средствами Direct3D 10. |
| \_ \_ \_ \_ Измерение оси массива D3D10 req \_ TEXTURE1D | 512   | Максимальный размер массива для массива текстуры 1D.                                                                                                                                                                                                                                                                                                                                                                                       |
| \_ \_ \_ \_ Измерение оси массива D3D10 req \_ TEXTURE2D | 512   | Максимальный размер массива 2D-текстуры.                                                                                                                                                                                                                                                                                                                                                                                       |
| \_Измерение D3D10 req \_ TEXTURE1D \_ U \_           | 8192  | Максимальная толщина одномерной текстуры.                                                                                                                                                                                                                                                                                                                                                                                                  |
| \_Измерение D3D10 req \_ TEXTURE2D \_ U \_ или \_ V \_    | 8192  | Максимальная ширина и высота двухмерной текстуры.                                                                                                                                                                                                                                                                                                                                                                                       |
| \_Измерение D3D10 req \_ TEXTURE3D \_ U \_ V \_ или \_ W \_ | 2048  | Максимальная ширина, высота и глубина трехмерной текстуры.                                                                                                                                                                                                                                                                                                                                                                               |
| \_Измерение D3D10 req \_ текстурекубе \_            | 8192  | Максимальный размер текстуры Куба.                                                                                                                                                                                                                                                                                                                                                                                                 |



 

Константы определяются в D3D10. h.
