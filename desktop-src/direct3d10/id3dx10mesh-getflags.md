# <a name="id3dx10meshgetflags-method"></a>Метод ID3DX10Mesh::

Доступ к флагам создания сетки.

## <a name="syntax"></a>Синтаксис


```C++
UINT GetFlags();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **uint**](../winprog/windows-data-types.md)**

Флаги создания, передаваемые в параметр *Options* объекта [**D3DX10CreateMesh**](d3d10-d3dx10createmesh.md) при создании сетки. См. раздел [**\_ Сетка D3DX10**](d3dx10-mesh.md).
