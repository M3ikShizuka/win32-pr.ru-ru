# <a name="effect-reference-direct3d-10-graphics"></a>Справочник по действиям (графика Direct3D 10)

API Direct3D определяет несколько элементов API, которые помогают создавать систему эффектов и управлять ею.

-   [Интерфейсы](d3d10-graphics-reference-effect-interfaces.md)
-   [Функции](d3d10-graphics-reference-effect-functions.md)
-   [Структуры](d3d10-graphics-reference-effect-structures.md)
-   [Константы](d3d10-graphics-reference-effect-constants.md)

Система влияния инкапсулирует назначения состояния в результате. Каждый результат — это коллекция состояния конвейера, которую можно разделить на назначения состояний с помощью блоков состояний, ресурсов и шейдеров. В зависимости от размера результата можно реализовать его в текстовой строке ASCII или файле эффектов (. FX). Чтобы упорядочить информацию в результате, см. [Формат эффектов](d3d10-effect-format.md).

Получив воздействие, используйте API эффектов, чтобы задать состояние на устройстве во время подготовки к просмотру. Кроме того, система эффектов поддерживает ряд API-интерфейсов отражения для получения и установки данных о последствиях. Дополнительные сведения см. в разделе [воздействие системных интерфейсов (Direct3D 10)](d3d10-graphics-programming-guide-effects-interfaces.md).
