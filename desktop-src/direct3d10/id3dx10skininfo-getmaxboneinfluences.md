# <a name="id3dx10skininfogetmaxboneinfluences-method"></a>Метод ID3DX10SkinInfo:: Жетмаксбонеинфлуенцес

Получите число вершин, которые могут максимально повлиять на кость.

## <a name="syntax"></a>Синтаксис


```C++
UINT GetMaxBoneInfluences();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **uint**](../winprog/windows-data-types.md)**

Число вершин, которое может максимально повлиять на кость.
