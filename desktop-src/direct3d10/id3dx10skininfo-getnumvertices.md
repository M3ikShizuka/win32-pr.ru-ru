# <a name="id3dx10skininfogetnumvertices-method"></a>Метод ID3DX10SkinInfo:: Жетнумвертицес

Возвращает количество вершин в ID3DX10SkinInfo.

## <a name="syntax"></a>Синтаксис


```C++
UINT GetNumVertices();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **uint**](../winprog/windows-data-types.md)**

Количество вершин в ID3DX10SkinInfo.
