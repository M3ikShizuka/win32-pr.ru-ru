# <a name="id3dx10meshgenerateattributebufferfromtable-method"></a>Метод ID3DX10Mesh:: Женератеаттрибутебуфферфромтабле

Создайте буфер атрибута на основе данных в таблице атрибутов сетки. Буфер атрибутов — это еще один формат для хранения данных в таблице атрибутов. Как буфер атрибутов, так и таблица атрибутов являются внутренними структурами данных в сетке.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GenerateAttributeBufferFromTable();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **HRESULT**](https://msdn.microsoft.com/library/Bb401631(v=MSDN.10).aspx)**

Возвращаемое значение является одним из значений, перечисленных в [кодах возврата Direct3D 10](d3d10-graphics-reference-returnvalues.md).
