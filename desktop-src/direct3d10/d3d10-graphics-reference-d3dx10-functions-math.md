## <a name="resolving-link-errors-with-d3dx-math-functions"></a><span data-ttu-id="6e4ef-205">Разрешение ошибок ссылок с помощью математических функций D3DX</span><span class="sxs-lookup"><span data-stu-id="6e4ef-205">Resolving Link Errors with D3DX Math Functions</span></span>

<span data-ttu-id="6e4ef-206">Математические функции D3DX реализуются одинаково в D3DX10 (D3DX10math. h) и D3DX9 (D3DX9math. h).</span><span class="sxs-lookup"><span data-stu-id="6e4ef-206">The D3DX math functions are implemented identically in D3DX10 (D3DX10math.h) and D3DX9 (D3DX9math.h).</span></span> <span data-ttu-id="6e4ef-207">Это может привести к ошибкам связи, если проект реализует как код DirectX 9, так и DirectX 10, и пытается связать функцию из одного заголовка с противоположной библиотекой.</span><span class="sxs-lookup"><span data-stu-id="6e4ef-207">This can cause link errors if a project implements both DirectX 9 and DirectX 10 code, and attempts to link a function from one header with the opposite library.</span></span>

<span data-ttu-id="6e4ef-208">Чтобы устранить проблему, связанную с включением обоих заголовков, D3DX10math. h включает следующие \# определения:</span><span class="sxs-lookup"><span data-stu-id="6e4ef-208">To eliminate the problem of including both headers, D3DX10math.h includes the following \#define:</span></span>


```
#ifndef __D3DX9MATH_H__
#define __D3DX9MATH_H__
```



<span data-ttu-id="6e4ef-209">Чтобы устранить возможные ошибки связи, в примерах SDK для DX сначала должна быть ссылка на библиотеки D3DX9 (D3DX9d. lib и D3DX9. lib), а затем библиотеки D3DX10 (D3DX10d. lib и D3DX10. lib).</span><span class="sxs-lookup"><span data-stu-id="6e4ef-209">To eliminate possible link errors, the DX SDK samples link to D3DX9 libraries first (D3DX9d.lib and D3DX9.lib) and then the D3DX10 libraries second (D3DX10d.lib and D3DX10.lib).</span></span> <span data-ttu-id="6e4ef-210">Эти параметры находятся в разделе проект или свойства, если используется Visual Studio.</span><span class="sxs-lookup"><span data-stu-id="6e4ef-210">These settings are under Project/Properties if you are using Visual Studio.</span></span>
