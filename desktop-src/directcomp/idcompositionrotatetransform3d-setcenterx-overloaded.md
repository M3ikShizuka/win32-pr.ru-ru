# <a name="idcompositionrotatetransform3dsetcenterx-methods"></a>Методы IDCompositionRotateTransform3D:: Сетцентеркс

Изменяет или анимируется значение свойства CenterX преобразования трехмерного вращения. Свойство CenterX Указывает координату x точки, вокруг которой выполняется поворот. Значение по умолчанию равно нулю.
