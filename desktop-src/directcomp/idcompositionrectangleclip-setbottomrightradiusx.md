# <a name="idcompositionrectangleclipsetbottomrightradiusx-methods"></a>Методы Идкомпоситионректанглеклип:: Сетботтомригхтрадиускс

Изменяет или анимируется значение свойства Боттомригхтрадиускс этого клипа. Свойство Боттомригхтрадиускс определяет радиус эллипса по оси x, округляющий правый нижний угол клипа.

| [**Сетботтомригхтрадиускс (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomrightradiusx(idcompositionanimation)) | Анимируется значение свойства Боттомригхтрадиускс этого клипа.<br/> |
