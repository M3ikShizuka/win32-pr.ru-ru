# <a name="idcompositionrectangleclipsettopleftradiusy-methods"></a>Методы Идкомпоситионректанглеклип:: Сеттоплефтрадиуси

Изменяет или анимируется значение свойства Топлефтрадиуси этого клипа. Свойство Топлефтрадиуси задает радиус эллипса по оси y, округляющий верхний левый угол клипа.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                                     | Описание                                                                |
|:---------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------|
| [**Сеттоплефтрадиуси (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settopleftradiusy(idcompositionanimation))                                     | Изменяет значение свойства Топлефтрадиуси этого клипа.<br/>  |
| [**Сеттоплефтрадиуси (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settopleftradiusy(idcompositionanimation)) | Анимируется значение свойства Топлефтрадиуси этого клипа.<br/> |
