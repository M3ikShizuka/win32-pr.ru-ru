# <a name="idcompositiontranslatetransform3dsetoffsetx-methods"></a>Методы IDCompositionTranslateTransform3D:: Сетоффсеткс

Изменяет или анимируется значение свойства Оффсеткс в эффекте преобразования 3D-преобразования. Свойство Оффсеткс задает преобразование по оси x.

| [**Сетоффсеткс (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsetx(idcompositionanimation)) | Анимируется значение свойства Оффсеткс.<br/> |
