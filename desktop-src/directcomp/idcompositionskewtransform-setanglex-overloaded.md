# <a name="idcompositionskewtransformsetanglex-methods"></a>Методы Идкомпоситионскевтрансформ:: Сетанглекс

Изменяет или анимируется значение свойства AngleX преобразования 2D-смещения. Свойство AngleX задает угол вращения (в градусах). Значение по умолчанию равно нулю.

| Метод                                                                                                     | Описание                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Сетанглекс (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setanglex(float))                                     | Изменяет значение свойства AngleX.<br/>  |
| [**Сетанглекс (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setanglex(idcompositionanimation)) | Анимируется значение свойства AngleX.<br/> |
