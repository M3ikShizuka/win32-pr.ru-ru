# <a name="idcompositiontranslatetransformsetoffsety-methods"></a>Методы Идкомпоситионтранслатетрансформ:: Сетоффсети

Изменяет или анимируется значение свойства Offset преобразования 2D-преобразования. Свойство Offset определяет преобразование по оси y.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                            | Описание                                            |
|:------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Сетоффсети (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform-setoffsety(float))                                     | Изменяет значение свойства Offset.<br/>  |
| [**Сетоффсети (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform-setoffsety(idcompositionanimation)) | Анимируется значение свойства Offset.<br/> |
