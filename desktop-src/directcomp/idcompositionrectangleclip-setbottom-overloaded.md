# <a name="idcompositionrectangleclipsetbottom-methods"></a>Методы Идкомпоситионректанглеклип:: Сетботтом

Выполняет анимацию или изменяет значение нижнего Свойства прямоугольника обрезки. Свойство Bottom Указывает координату по оси y правого нижнего угла прямоугольника обрезки.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                     | Описание                                           |
|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Сетботтом (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottom(float))                                     | Изменяет значение свойства Bottom.<br/>  |
| [**Сетботтом (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottom(idcompositionanimation)) | Анимируется значение свойства Bottom.<br/> |
