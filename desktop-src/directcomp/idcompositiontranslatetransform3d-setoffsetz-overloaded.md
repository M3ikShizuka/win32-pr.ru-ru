# <a name="idcompositiontranslatetransform3dsetoffsetz-methods"></a>Методы IDCompositionTranslateTransform3D:: Сетоффсетз

Изменяет или анимируется значение свойства Оффсетз в эффекте преобразования 3D-преобразования. Свойство Оффсетз задает преобразование вдоль оси z.

| [**Сетоффсетз (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsetz(idcompositionanimation)) | Анимируется значение свойства Оффсетз.<br/> |
