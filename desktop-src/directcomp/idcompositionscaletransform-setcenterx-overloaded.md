# <a name="idcompositionscaletransformsetcenterx-methods"></a>Методы Идкомпоситионскалетрансформ:: Сетцентеркс

Изменяет или анимируется значение свойства CenterX преобразования 2D-шкалы. Свойство CenterX Указывает координату x точки, относительно которой выполняется масштабирование.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                        | Описание                                            |
|:--------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Сетцентеркс (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setcenterx(float))                                     | Изменяет значение свойства CenterX.<br/>  |
| [**Сетцентеркс (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setcenterx(idcompositionanimation)) | Анимируется значение свойства CenterX.<br/> |
