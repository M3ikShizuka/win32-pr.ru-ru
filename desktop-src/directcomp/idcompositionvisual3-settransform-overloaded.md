# <a name="idcompositionvisual3settransform-methods"></a>Методы IDCompositionVisual3:: Сеттрансформ

Задает свойство Transform этого визуального элемента. Свойство Transform указывает трехмерное преобразование, используемое для изменения системы координат данного визуального элемента. Свойство может указывать либо матрицу преобразования размером 4 на 4, либо объект преобразования.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                  | Описание                                                                    |
|:----------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------|
| [**Сеттрансформ ( \_ Матрица D2D \_ 4x4 \_ F&)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual3-settransform(constd2d_matrix_4x4_f_))       | Устанавливает свойство Transform в указанную матрицу преобразования.<br/>      |
| [**Сеттрансформ (IDCompositionTransform3D \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual3-settransform(idcompositiontransform3d)) | Задает для свойства Transform указанный объект преобразования.<br/> |
