# <a name="idcompositionskewtransformsetcentery-methods"></a>Методы Идкомпоситионскевтрансформ:: Сетцентери

Изменяет или анимирует значение свойства Center объекта преобразования 2D-смещения. Свойство Center Указывает координату y точки, относительно которой выполняется отклонение.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                       | Описание                                            |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Сетцентери (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcentery(float))                                     | Изменяет значение свойства Center.<br/>  |
| [**Сетцентери (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcentery(idcompositionanimation)) | Выполняет анимацию значения свойства Center.<br/> |
