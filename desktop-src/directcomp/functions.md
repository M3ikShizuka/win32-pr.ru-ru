# <a name="directcomposition-functions"></a>Функции DirectComposition

В этом разделе описываются функции, предоставляемые API Microsoft DirectComposition.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                       | Описание                                                                                                                                          |
|---------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**дкомпоситионаттачмауседрагтохвнд**](/windows/desktop/api/dcomp/nf-dcomp-dcompositionattachmousedragtohwnd)<br/>   | Создает взаимодействие или Инпутсинк для перенаправления кнопки мыши вниз и всех последующих событий перемещения и вверх в заданный HWND.<br/>                      |
| [**дкомпоситионаттачмаусевхилтохвнд**](/windows/desktop/api/dcomp/nf-dcomp-dcompositionattachmousewheeltohwnd)<br/> | Создает взаимодействие или Инпутсинк для направления сообщений колесика мыши в заданный HWND. <br/>                                                        |
| [**дкомпоситионкреатедевице**](/windows/desktop/api/Dcomp/nf-dcomp-dcompositioncreatedevice)<br/>                     | Создает новый объект устройства, который можно использовать для создания других объектов DirectComposition.<br/>                                                   |
| [**DCompositionCreateDevice2**](/windows/desktop/api/Dcomp/nf-dcomp-dcompositioncreatedevice2)<br/>                   | Создает новый объект устройства, который можно использовать для создания других объектов DirectComposition.<br/>                                                   |
| [**DCompositionCreateDevice3**](/windows/desktop/api/dcomp/nf-dcomp-dcompositioncreatedevice3)<br/>                   | Создает новый объект устройства DirectComposition, который можно использовать для создания других объектов DirectComposition.<br/>                               |
| [**дкомпоситионкреатесурфацехандле**](/windows/desktop/api/Dcomp/nf-dcomp-dcompositioncreatesurfacehandle)<br/>       | Создает новый объект области композиции, который можно привязать к цепочке буфера обмена Microsoft DirectX или буферу обмена и связать с визуальным объектом.<br/> |
| [**дкомпоситионжетфраместатистикс**](/previous-versions/windows/desktop/legacy/mt589902(v=vs.85))<br/>         | Получает сведения о статистике композиции.<br/>                                                                                             |
