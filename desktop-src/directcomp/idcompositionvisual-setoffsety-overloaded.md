# <a name="idcompositionvisualsetoffsety-methods"></a>Методы Идкомпоситионвисуал:: Сетоффсети

Изменяет или анимирует значение свойства смещения этого визуального элемента, изменяя вертикальную позиции визуального элемента относительно родительского объекта.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                | Описание                                           |
|:------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Сетоффсети (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setoffsety(float))                                     | Изменяет значение свойства Offset.<br/> |
| [**Сетоффсети (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setoffsety(idcompositionanimation)) | Анимирует значение свойства Offset<br/> |
