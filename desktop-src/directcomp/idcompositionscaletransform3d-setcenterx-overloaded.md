# <a name="idcompositionscaletransform3dsetcenterx-methods"></a>Методы IDCompositionScaleTransform3D:: Сетцентеркс

Изменяет или анимируется значение свойства CenterX преобразования трехмерной шкалы. Свойство CenterX Указывает координату x точки, относительно которой выполняется масштабирование.

|:----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Сетцентеркс (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterx(float))                                     | Изменяет значение свойства CenterX.<br/>  |
| [**Сетцентеркс (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterx(idcompositionanimation)) | Анимируется значение свойства CenterX.<br/> |
