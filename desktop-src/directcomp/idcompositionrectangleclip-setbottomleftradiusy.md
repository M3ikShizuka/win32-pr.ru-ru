# <a name="idcompositionrectangleclipsetbottomleftradiusy-methods"></a>Методы Идкомпоситионректанглеклип:: Сетботтомлефтрадиуси

Изменяет или анимируется значение свойства Боттомлефтрадиуси этого клипа. Свойство Боттомлефтрадиуси задает радиус эллипса по оси y, округляющий нижний левый угол клипа.

| [**Сетботтомлефтрадиуси (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setbottomleftradiusy(idcompositionanimation)) | Анимируется значение свойства Боттомлефтрадиуси этого клипа.<br/> |
