# <a name="idcompositionskewtransformsetangley-methods"></a>Методы Идкомпоситионскевтрансформ:: Сетанглэй

Изменяет или анимируется значение свойства Angle преобразования 2D-смещения. Свойство Angle задает угол поворота в градусах. Значение по умолчанию равно нулю.

|:-----------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Сетанглэй (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setangley(float))                                     | Изменяет значение свойства Angle.<br/>  |
| [**Сетанглэй (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setangley(idcompositionanimation)) | Анимируется значение свойства Angle.<br/> |
