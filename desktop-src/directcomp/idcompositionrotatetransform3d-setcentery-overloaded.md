# <a name="idcompositionrotatetransform3dsetcentery-methods"></a>Методы IDCompositionRotateTransform3D:: Сетцентери

Изменяет или анимируется значение свойства Center для преобразования трехмерного вращения. Свойство Center Указывает координату y точки, вокруг которой выполняется поворот. Значение по умолчанию равно нулю.
