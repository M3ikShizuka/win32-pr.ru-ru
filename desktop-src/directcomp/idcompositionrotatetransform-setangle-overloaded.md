# <a name="idcompositionrotatetransformsetangle-methods"></a>Методы Идкомпоситионротатетрансформ:: Сетангле

Изменяет или анимируется значение свойства Angle преобразования поворота. Свойство Angle задает угол поворота в градусах. Значение по умолчанию равно нулю.

| Метод                                                                                                     | Описание                                          |
|:-----------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**Сетангле (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setangle(float))                                     | Изменяет значение свойства Angle.<br/>  |
| [**Сетангле (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setangle(idcompositionanimation)) | Анимируется значение свойства Angle.<br/> |
