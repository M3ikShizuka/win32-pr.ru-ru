# <a name="idcompositiontranslatetransformsetoffsetx-methods"></a>Методы Идкомпоситионтранслатетрансформ:: Сетоффсеткс

Изменяет или анимируется значение свойства Оффсеткс преобразования 2D-преобразования. Свойство Оффсеткс задает преобразование по оси x.
