# <a name="idcompositionrectangleclipsetleft-methods"></a>Методы Идкомпоситионректанглеклип:: Сетлефт

Анимируется или изменяет значение левого свойства прямоугольника обрезки. Свойство Left Указывает координату x верхнего левого угла прямоугольника обрезки.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                 | Описание                                         |
|:-------------------------------------------------------------------------------------------------------|:----------------------------------------------------|
| [**Сетлефт (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setleft(float))                                     | Изменяет значение свойства Left.<br/>  |
| [**Сетлефт (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setleft(idcompositionanimation)) | Анимируется значение свойства Left.<br/> |
