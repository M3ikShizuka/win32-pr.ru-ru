# <a name="idcompositionscaletransform3dsetscaley-methods"></a>Методы IDCompositionScaleTransform3D:: Сетскалэй

Изменяет или анимируется значение свойства Scale преобразования масштабирования. Свойство Scale определяет коэффициент масштабирования вдоль оси y.

| Метод                                                                                                        | Описание                                           |
|:--------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Сетскалэй (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscaley(float))                                     | Изменяет значение свойства Scale.<br/>  |
| [**Сетскалэй (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscaley(idcompositionanimation)) | Выполняет анимацию значения свойства Scale.<br/> |
