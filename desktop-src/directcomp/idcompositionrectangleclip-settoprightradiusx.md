# <a name="idcompositionrectangleclipsettoprightradiusx-methods"></a>Методы Идкомпоситионректанглеклип:: Сеттопригхтрадиускс

Изменяет или анимируется значение свойства Топригхтрадиускс этого клипа. Свойство Топригхтрадиускс определяет радиус эллипса по оси x, округляющий правый верхний угол клипа.
