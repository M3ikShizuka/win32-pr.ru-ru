# <a name="idcompositionrectangleclipsettoprightradiusx-methods"></a>Методы Идкомпоситионректанглеклип:: Сеттопригхтрадиускс

Изменяет или анимируется значение свойства Топригхтрадиускс этого клипа. Свойство Топригхтрадиускс определяет радиус эллипса по оси x, округляющий правый верхний угол клипа.

| Метод                                                                                                                       | Описание                                                                 |
|:-----------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------|
| [**Сеттопригхтрадиускс (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settoprightradiusx(idcompositionanimation))                                     | Изменяет значение свойства Топригхтрадиускс этого клипа.<br/>  |
| [**Сеттопригхтрадиускс (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settoprightradiusx(idcompositionanimation)) | Анимируется значение свойства Топригхтрадиускс этого клипа.<br/> |
