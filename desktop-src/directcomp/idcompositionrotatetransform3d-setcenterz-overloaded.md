# <a name="idcompositionrotatetransform3dsetcenterz-methods"></a>Методы IDCompositionRotateTransform3D:: Сетцентерз

Изменяет или анимируется значение свойства CenterZ преобразования трехмерного вращения. Свойство CenterZ Указывает координату z точки, вокруг которой выполняется поворот. Значение по умолчанию равно нулю.

| [**Сетцентерз (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterz(idcompositionanimation)) | Анимируется значение свойства CenterZ.<br/> |
