# <a name="idcompositionrotatetransform3dsetangle-methods"></a>Методы IDCompositionRotateTransform3D:: Сетангле

Изменяет или анимируется значение свойства Angle преобразования трехмерного вращения. Свойство Angle задает угол поворота в градусах. Значение по умолчанию равно нулю.
