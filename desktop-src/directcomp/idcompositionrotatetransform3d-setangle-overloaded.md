# <a name="idcompositionrotatetransform3dsetangle-methods"></a>Методы IDCompositionRotateTransform3D:: Сетангле

Изменяет или анимируется значение свойства Angle преобразования трехмерного вращения. Свойство Angle задает угол поворота в градусах. Значение по умолчанию равно нулю.

|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**Сетангле (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setangle(float))                                     | Изменяет значение свойства Angle.<br/>  |
| [**Сетангле (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setangle(idcompositionanimation)) | Анимируется значение свойства Angle.<br/> |
