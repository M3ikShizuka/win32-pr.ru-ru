# <a name="idcompositionrectangleclipsetbottomleftradiusx-methods"></a>Методы Идкомпоситионректанглеклип:: Сетботтомлефтрадиускс

Изменяет или анимируется значение свойства Боттомлефтрадиускс этого клипа. Свойство Боттомлефтрадиускс определяет радиус эллипса по оси x, округляющий нижний левый угол клипа.
