# <a name="idcompositionvisualsetclip-methods"></a>Методы Идкомпоситионвисуал:: Сетклип

Устанавливает свойство Clip этого визуального элемента в указанную прямоугольную область или объект Clip. Свойство Clip позволяет пререзать визуализацию визуального поддерева, которое находится в корне этого визуального элемента, до прямоугольной области.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                | Описание                                                            |
|:--------------------------------------------------------------------------------------|:-----------------------------------------------------------------------|
| [**Сетклип (const D2D \_ Rect \_ F&)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setclip(constd2d_rect_f_)) | Устанавливает свойство Clip в указанную прямоугольную область.<br/> |
| [**Сетклип (Идкомпоситионклип \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setclip(idcompositionclip)) | Устанавливает свойство Clip в указанный объект Clip.<br/>        |
