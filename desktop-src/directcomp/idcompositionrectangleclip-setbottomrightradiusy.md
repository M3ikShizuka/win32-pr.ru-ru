# <a name="idcompositionrectangleclipsetbottomrightradiusy-methods"></a>Методы Идкомпоситионректанглеклип:: Сетботтомригхтрадиуси

Изменяет или анимируется значение свойства Боттомригхтрадиуси этого клипа. Свойство Боттомригхтрадиуси задает радиус эллипса по оси y, округляющий правый нижний угол клипа.
