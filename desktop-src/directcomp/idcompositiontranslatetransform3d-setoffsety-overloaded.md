# <a name="idcompositiontranslatetransform3dsetoffsety-methods"></a>Методы IDCompositionTranslateTransform3D:: Сетоффсети

Изменяет или анимируется значение свойства Offset в эффекте преобразования 3D-преобразования. Свойство Offset определяет преобразование по оси y.

| Метод                                                                                                              | Описание                                            |
|:--------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Сетоффсети (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsety(float))                                     | Изменяет значение свойства Offset.<br/>  |
| [**Сетоффсети (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositiontranslatetransform3d-setoffsety(idcompositionanimation)) | Анимируется значение свойства Offset.<br/> |
