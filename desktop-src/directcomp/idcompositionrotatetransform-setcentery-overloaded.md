# <a name="idcompositionrotatetransformsetcentery-methods"></a>Методы Идкомпоситионротатетрансформ:: Сетцентери

Изменяет или анимируется значение свойства Center для преобразования 2D-вращения. Свойство Center Указывает координату y точки, вокруг которой выполняется поворот.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                         | Описание                                            |
|:---------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Сетцентери (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setcentery(float))                                     | Изменяет значение свойства Center.<br/>  |
| [**Сетцентери (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setcentery(idcompositionanimation)) | Выполняет анимацию значения свойства Center.<br/> |
