# <a name="idcompositionskewtransformsetcenterx-methods"></a>Методы Идкомпоситионскевтрансформ:: Сетцентеркс

Изменяет или анимируется значение свойства CenterX преобразования 2D-смещения. Свойство CenterX Указывает координату x точки, относительно которой выполняется отклонение.

| Метод                                                                                                       | Описание                                            |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Сетцентеркс (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcenterx(float))                                     | Изменяет значение свойства CenterX.<br/>  |
| [**Сетцентеркс (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcenterx(idcompositionanimation)) | Анимируется значение свойства CenterX.<br/> |
