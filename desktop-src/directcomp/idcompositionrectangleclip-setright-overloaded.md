# <a name="idcompositionrectangleclipsetright-methods"></a>Методы Идкомпоситионректанглеклип:: Сетригхт

Анимируется или изменяет значение правого свойства прямоугольника обрезки. Свойство Right Указывает координату по оси x правого нижнего угла прямоугольника обрезки.
