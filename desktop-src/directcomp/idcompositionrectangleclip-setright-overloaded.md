# <a name="idcompositionrectangleclipsetright-methods"></a>Методы Идкомпоситионректанглеклип:: Сетригхт

Анимируется или изменяет значение правого свойства прямоугольника обрезки. Свойство Right Указывает координату по оси x правого нижнего угла прямоугольника обрезки.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                   | Описание                                          |
|:---------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**Сетригхт (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setright(float))                                     | Изменяет значение свойства Right.<br/>  |
| [**Сетригхт (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-setright(idcompositionanimation)) | Выполняет анимацию значения правого свойства.<br/> |
