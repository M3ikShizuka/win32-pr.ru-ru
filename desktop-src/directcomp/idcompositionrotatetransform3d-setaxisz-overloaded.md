# <a name="idcompositionrotatetransform3dsetaxisz-methods"></a>Методы IDCompositionRotateTransform3D:: Сетаксисз

Изменяет или анимируется значение свойства Аксисз преобразования трехмерного вращения. Свойство Аксисз задает координату z для вектора оси вращения. Значение по умолчанию — 1,0.
