# <a name="idcompositionscaletransform3dsetcenterz-methods"></a>Методы IDCompositionScaleTransform3D:: Сетцентерз

Изменяет или анимируется значение свойства CenterZ преобразования трехмерной шкалы. Свойство CenterZ Указывает координату z точки, относительно которой выполняется масштабирование.

|:----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Сетцентерз (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterz(float))                                     | Изменяет значение свойства CenterZ.<br/>  |
| [**Сетцентерз (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcenterz(idcompositionanimation)) | Анимируется значение свойства CenterZ.<br/> |
