# <a name="idcompositionscaletransform3dsetcenterz-methods"></a>Методы IDCompositionScaleTransform3D:: Сетцентерз

Изменяет или анимируется значение свойства CenterZ преобразования трехмерной шкалы. Свойство CenterZ Указывает координату z точки, относительно которой выполняется масштабирование.
