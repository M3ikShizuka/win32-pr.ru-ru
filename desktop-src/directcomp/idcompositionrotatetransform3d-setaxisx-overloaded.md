# <a name="idcompositionrotatetransform3dsetaxisx-methods"></a>Методы IDCompositionRotateTransform3D:: Сетаксискс

Изменяет или анимируется значение свойства свойства AxisX преобразования трехмерного вращения. Свойство свойства AxisX задает координату x для вектора оси вращения. Значение по умолчанию равно нулю.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                       | Описание                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**Сетаксискс (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(float))                                     | Изменяет значение свойства свойства AxisX.<br/>  |
| [**Сетаксискс (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(idcompositionanimation)) | Анимируется значение свойства свойства AxisX.<br/> |
