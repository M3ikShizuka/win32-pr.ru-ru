# <a name="idcompositionscaletransform3dsetscalez-methods"></a>Методы IDCompositionScaleTransform3D:: Сетскалез

Изменяет или анимируется значение свойства Скалез преобразования шкалы. Свойство Скалез задает коэффициент масштабирования вдоль оси z.

|:--------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
