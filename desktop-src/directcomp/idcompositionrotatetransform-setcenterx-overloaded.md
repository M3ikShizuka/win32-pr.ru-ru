# <a name="idcompositionrotatetransformsetcenterx-methods"></a>Методы Идкомпоситионротатетрансформ:: Сетцентеркс

Изменяет или анимируется значение свойства CenterX преобразования 2D-вращения. Свойство CenterX Указывает координату x точки, вокруг которой выполняется поворот.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                         | Описание                                            |
|:---------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Сетцентеркс (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setcenterx(float))                                     | Изменяет значение свойства CenterX.<br/>  |
| [**Сетцентеркс (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform-setcenterx(idcompositionanimation)) | Анимируется значение свойства CenterX.<br/> |
