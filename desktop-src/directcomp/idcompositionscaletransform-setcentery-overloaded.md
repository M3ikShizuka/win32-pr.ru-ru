# <a name="idcompositionscaletransformsetcentery-methods"></a>Методы Идкомпоситионскалетрансформ:: Сетцентери

Изменяет или анимирует значение свойства Center преобразования 2D-шкалы. Свойство Center Указывает координату y точки, относительно которой выполняется масштабирование.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                        | Описание                                            |
|:--------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Сетцентери (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setcentery(float))                                     | Изменяет значение свойства Center.<br/>  |
| [**Сетцентери (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setcentery(idcompositionanimation)) | Выполняет анимацию значения свойства Center.<br/> |
