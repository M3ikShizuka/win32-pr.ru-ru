# <a name="idcompositionscaletransform3dsetscalex-methods"></a>Методы IDCompositionScaleTransform3D:: Сетскалекс

Изменяет или анимируется значение свойства ScaleX преобразования трехмерной шкалы. Свойство ScaleX указывает коэффициент масштабирования вдоль оси x.

| Метод                                                                                                        | Описание                                           |
|:--------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Сетскалекс (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalex(float))                                     | Изменяет значение свойства ScaleX.<br/>  |
| [**Сетскалекс (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setscalex(idcompositionanimation)) | Анимируется значение свойства ScaleX.<br/> |
