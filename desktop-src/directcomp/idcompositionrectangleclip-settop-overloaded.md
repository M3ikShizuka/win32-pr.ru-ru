# <a name="idcompositionrectangleclipsettop-methods"></a>Методы Идкомпоситионректанглеклип:: Сеттоп

Выполняет анимацию или изменяет значение свойства Top прямоугольника обрезки. Свойство Top Указывает координату по оси y верхнего левого угла прямоугольника обрезки.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                               | Описание                                        |
|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------|
| [**Сеттоп (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settop(float))                                     | Изменяет значение свойства Top.<br/>  |
| [**Сеттоп (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrectangleclip-settop(idcompositionanimation)) | Выполняет анимацию значения свойства Top.<br/> |
