# <a name="idcompositionscaletransform3dsetcentery-methods"></a>Методы IDCompositionScaleTransform3D:: Сетцентери

Изменяет или анимирует значение свойства Center преобразования трехмерной шкалы. Свойство Center Указывает координату y точки, относительно которой выполняется масштабирование.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                          | Описание                                            |
|:----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**Сетцентери (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcentery(float))                                     | Изменяет значение свойства Center.<br/>  |
| [**Сетцентери (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform3d-setcentery(idcompositionanimation)) | Выполняет анимацию значения свойства Center.<br/> |
