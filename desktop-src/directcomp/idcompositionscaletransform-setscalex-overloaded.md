# <a name="idcompositionscaletransformsetscalex-methods"></a>Методы Идкомпоситионскалетрансформ:: Сетскалекс

Изменяет или анимируется значение свойства ScaleX преобразования 2D Scale. Свойство ScaleX указывает коэффициент масштабирования вдоль оси x.

| Метод                                                                                                      | Описание                                           |
|:------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Сетскалекс (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setscalex(float))                                     | Изменяет значение свойства ScaleX.<br/>  |
| [**Сетскалекс (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setscalex(idcompositionanimation)) | Анимируется значение свойства ScaleX.<br/> |
