# <a name="idcompositionrotatetransform3dsetaxisy-methods"></a>Методы IDCompositionRotateTransform3D:: Сетаксиси

Изменяет или анимируется значение свойства Axis преобразования вращения. Свойство Axis Указывает координату y для вектора оси вращения. Значение по умолчанию равно нулю.

| Метод                                                                                                       | Описание                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**Сетаксиси (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(float))                                     | Изменяет значение свойства Axis.<br/>  |
| [**Сетаксиси (Идкомпоситионаниматион \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(idcompositionanimation)) | Анимируется значение свойства Axis.<br/> |
