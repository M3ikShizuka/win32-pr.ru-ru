# <a name="primary-group-of-a-new-object"></a>Основная группа нового объекта

Основная группа нового объекта — это основная группа из [*дескриптора безопасности*](/windows/desktop/SecGloss/s-gly) , заданного создателем объекта. Если создатель объекта не указывает основную группу, основная группа объекта является основной группой по умолчанию от основного [*маркера или токена олицетворения*](/windows/desktop/SecGloss/i-gly)создателя.
