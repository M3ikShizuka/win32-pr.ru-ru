# <a name="advantages-of-role-based-authorization"></a>Преимущества авторизации на основе ролей

API диспетчера авторизации предоставляет модель авторизации, имеющую несколько преимуществ для авторизации низкого уровня (на основе ACL):

-   Упрощает администрирование управления доступом.
-   Делает авторизацию доступной для сценариев и приложений, разработанных с помощью системы разработки Microsoft Visual Basic.
-   Предоставляет механизм для применения бизнес-логики во время выполнения при проверке доступа.
