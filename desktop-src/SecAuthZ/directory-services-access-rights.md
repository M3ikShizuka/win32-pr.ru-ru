# <a name="directory-services-access-rights"></a>Права доступа к службам каталогов

Каждому объекту Active Directory назначен [*дескриптор безопасности*](/windows/desktop/SecGloss/s-gly) . В этих дескрипторах безопасности можно задать набор прав доверенного лица, характерных для объектов службы каталогов. Эти права перечислены в следующей таблице. Дополнительные сведения см. в разделе [Управление правами доступа](/windows/desktop/AD/control-access-rights).



| Права                                | Значение                                                                                                                                                                                                                                                                                 |
|---------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| АКТРЛ \_ DS \_ Open<br/>            | Откройте объект DS.<br/>                                                                                                                                                                                                                                                            |
| АКТРЛ \_ DS \_ Создание \_ дочернего<br/>   | Создайте дочерний объект DS.<br/>                                                                                                                                                                                                                                                    |
| \_ \_ дочерний элемент актрл DS DELETE \_<br/>   | Удаление дочернего объекта DS.<br/>                                                                                                                                                                                                                                                    |
| \_список DS \_ актрл<br/>            | Перечисление объекта DS.<br/>                                                                                                                                                                                                                                                       |
| АКТРЛ \_ DS \_ Read \_ prop<br/>      | Чтение свойств объекта DS.<br/>                                                                                                                                                                                                                                          |
| АКТРЛ \_ DS \_ Write \_<br/>     | Запись свойств для объекта DS.<br/>                                                                                                                                                                                                                                            |
| АКТРЛ \_ DS \_ Self<br/>            | Доступ разрешен только после проверки проверенных прав, поддерживаемых объектом. Этот флаг можно использовать отдельно для выполнения всех проверенных проверок прав объекта или для его объединения с идентификатором определенного проверенного право на выполнение только этой проверки.<br/> |
| \_ \_ дерево удаления актрл \_ DS<br/>    | Удаляет дерево объектов DS.<br/>                                                                                                                                                                                                                                                 |
| \_ \_ объект списка DS \_ актрл<br/>    | Перечисление дерева объектов DS.<br/>                                                                                                                                                                                                                                                   |
| \_ \_ доступ к элементу управления DS актрл \_<br/> | Доступ разрешен только после выполнения расширенных проверок прав, поддерживаемых объектом. Этот флаг можно использовать отдельно для выполнения всех проверок расширенных прав на объекте или для его объединения с идентификатором определенного расширенного права, чтобы выполнить только эту проверку.<br/>    |
