# <a name="authorization-objects"></a>Объекты авторизации

В приложениях авторизации используются следующие объекты.

ActiveX Установщик предоставляет следующий объект.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                               | Описание                                                                                                                        |
|---------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**Циеаксиинсталлерсервице**](cieaxiinstallerservice.md)<br/> | Реализует интерфейсы [**иаксисервице**](ieaxiservice.md) и [**иеаксисервицекаллбакк**](ieaxiservicecallback.md) .<br/> |



 


Диспетчер авторизации предоставляет следующие объекты.



| Объект                                               | Описание                                                                          |
|------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**азаусоризатионсторе**](/windows/desktop/api/Azroles/nn-azroles-iazauthorizationstore) | Определяет контейнер, который является корнем хранилища политик авторизации.<br/> |
| [**азбизрулеконтекст**](/windows/desktop/api/Azroles/nn-azroles-iazbizrulecontext)         | Содержит сведения об операции бизнес-правила (Бизруле).<br/>           |
