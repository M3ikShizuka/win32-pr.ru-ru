# <a name="generic-access-rights"></a>Универсальные права доступа

Защищаемые объекты используют [Формат маски доступа](access-mask-format.md) , в котором четыре старших бита указывают общие права доступа. Каждый тип защищаемого объекта сопоставляет эти биты с набором стандартных и зависящих от объекта прав доступа. например, объект Windows file сопоставляет общий \_ бит чтения с \_ элементом управления для чтения и синхронизирует стандартные права доступа, а также разрешения на доступ к объектам для чтения и чтения файлов \_ \_ , \_ чтение \_ атрибутов EA и file read DATA \_ \_ . Другие типы объектов сопоставляют универсальный \_ бит чтения с любым набором прав доступа, подходящим для этого типа объекта.

С помощью универсальных прав доступа можно указать тип доступа, необходимый при открытии маркера для объекта. Обычно это проще, чем указывать все соответствующие стандартные и специальные права.

В следующей таблице показаны константы, определенные для общих прав доступа.



| Константа         | Универсальное значение            |
|------------------|----------------------------|
| УНИВЕРСАЛЬНые \_ все     | Все возможные права доступа |
| УНИВЕРСАЛЬНое \_ выполнение | Доступ для выполнения             |
| УНИВЕРСАЛЬНое \_ чтение    | Доступ на чтение.                |
| УНИВЕРСАЛЬНая \_ запись   | Доступ для записи.               |



 

Приложения, определяющие закрытые защищаемые объекты, также могут использовать универсальные права доступа.
