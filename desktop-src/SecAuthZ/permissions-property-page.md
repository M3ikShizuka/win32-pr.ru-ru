# <a name="permissions-property-page"></a>Страница свойств «разрешения»

На странице свойств **разрешения** содержатся элементы управления для расширенного редактирования [*записей ACE*](/windows/desktop/SecGloss/a-gly) в [*списке DACL*](/windows/desktop/SecGloss/d-gly)объекта. Эта страница позволяет пользователю просматривать и изменять все сведения в списке DACL, включая сведения, недоступные на [странице Основные свойства безопасности](basic-security-property-page.md). Дополнительные сведения включают [записи ACE для конкретного объекта](object-specific-aces.md) и сведения о [наследовании ACE](ace-inheritance.md).

Страница свойств **разрешения** всегда включается в страницу [свойств Расширенная безопасность](advanced-security-property-sheet.md) , отображаемая при нажатии пользователем кнопки **Дополнительно** на странице свойств базовая безопасность.
