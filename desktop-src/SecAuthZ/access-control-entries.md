# <a name="access-control-entries"></a>Записи контроля доступа

[*Запись управления доступом*](/windows/desktop/SecGloss/a-gly) (ACE) — это элемент в [*списке управления доступом*](/windows/desktop/SecGloss/a-gly) (ACL). Список ACL может содержать ноль или более записей ACE. Каждый элемент управления доступом или наблюдает за доступом к объекту с помощью указанного доверенного лица. Дополнительные сведения о добавлении, удалении или изменении записей ACE в списках ACL объекта см. в разделе [изменение списков управления доступом объекта на языке C++](modifying-the-acls-of-an-object-in-c--.md).

Существует шесть типов записей ACE, три из которых поддерживаются всеми защищаемыми объектами. Другие три типа — это объекты [ACE](object-specific-aces.md) , которые поддерживаются объектами службы каталогов.

Все типы записей ACE содержат следующие сведения об управлении доступом:

-   [Идентификатор безопасности](security-identifiers.md) (SID), определяющий [доверенное лицо](trustees.md) , к которому применяется ACE.
-   [*Маска доступа*](/windows/desktop/SecGloss/a-gly) , указывающая [права доступа](access-rights-and-access-masks.md) , контролируемые элементом управления доступом.
-   Флаг, указывающий тип записи ACE.
-   Набор битовых флагов, который определяет, могут ли дочерние контейнеры или объекты наследовать ACE от основного объекта, к которому присоединен ACL.

В следующей таблице перечислены три типа ACE, поддерживаемые всеми защищаемыми объектами.



| Тип               | Описание                                                                                                                                                                                                                                      |
|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Запрещен доступ к ACE  | Используется в [*избирательном списке управления доступом*](/windows/desktop/SecGloss/d-gly) (DACL) для запрета прав доступа к доверенному лицу.                                       |
| ACE, разрешенный доступом | Используется в списке DACL для разрешения прав доступа к доверенному лицу.                                                                                                                                                                                              |
| ACE аудита системы   | Используется в [*системном списке управления доступом*](/windows/desktop/SecGloss/s-gly) (SACL) для создания записи аудита, когда доверенное лицо пытается применить указанные права доступа. |



 

Таблицу записей ACE для конкретного объекта см. в разделе [записи ACE для конкретного объекта](object-specific-aces.md).

> [!Note]  
> Записи ACE объекта системного оповещения в настоящее время не поддерживаются.
