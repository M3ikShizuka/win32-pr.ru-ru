# <a name="allowing-anonymous-access"></a>Разрешение анонимного доступа

Политика безопасности по умолчанию разрешает анонимный локальный доступ без прав. Администраторы могут добавлять или вычитать права по мере их появлении.

Локальная группа доступа существует для приложений, имеющих те же права доступа, что и все. администраторы могут соответствующим образом увеличить или уменьшить число пользователей в этой группе с именем, которое соответствует **предварительно Windows 2000-совместимым группам доступа**.

Дополнительные сведения см. на страницах справочника по функциям, перечисленным в таблице в разделе [функции локальной группы](/windows/desktop/NetMgmt/local-group-functions) .
