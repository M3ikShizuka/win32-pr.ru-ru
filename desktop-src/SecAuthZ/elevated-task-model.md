# <a name="elevated-task-model"></a>Модель задач с повышенными правами

В модели задач с повышенными привилегиями приложение, выполняющееся в качестве обычного пользователя, выполняет операции, требующие прав администратора, запуская запланированную задачу.

**Windows Server 2003 и Windows XP:** Модель задач с повышенными правами не поддерживается.

Задачи не потребляют столько системных ресурсов, сколько служб, а задачи автоматически закрываются по завершении. Рекомендуется использовать эту модель вместо [модели службы операционной системы](operating-system-service-model.md) , если не требуется обратная совместимость с операционными системами более ранних версий.

Чтобы использовать задачу для выполнения привилегированных операций для стандартного пользовательского приложения, должны выполняться следующие условия.

-   Задача должна быть настроена для запуска от имени **System**.
-   [*Дескриптор безопасности*](/windows/desktop/SecGloss/s-gly) , связанный с задачей, должен быть настроен таким, чтобы позволить обычным пользователям запускать задачу.
-   Должна быть запущена служба планировщика заданий.

Сведения о создании и запуске задач см. в разделе [планировщик задач](/windows/desktop/TaskSchd/task-scheduler-start-page).
