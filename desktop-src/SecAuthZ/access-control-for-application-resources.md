# <a name="access-control-for-application-resources"></a>Контроль доступа для ресурсов приложения

API диспетчера авторизации и оснастка MMC, доступные в Windows Server 2003, предоставляют приложениям гибкую платформу управления доступом на основе ролей. Эта платформа включает набор объектов COM, которые представляют собой бизнес-приложения.

Следующие разделы содержат сведения об управлении доступом к ресурсам приложения.

-   [Управление доступом на основе ролей](role-based-access-control.md)
-   [Управление доступом на основе ACL](acl-based-access-control.md)
