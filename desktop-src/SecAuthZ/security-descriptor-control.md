# <a name="security_descriptor_control"></a>\_элемент управления дескриптора безопасности \_

Тип **данных \_ \_ Control дескриптора безопасности** — это набор битовых флагов, определяющих значение [*дескриптора безопасности*](/windows/desktop/SecGloss/s-gly) или его компонентов. Каждый дескриптор безопасности имеет элемент **управления** , в котором хранятся **\_ \_ Контрольные биты дескриптора безопасности** .


```C++
typedef WORD SECURITY_DESCRIPTOR_CONTROL, *PSECURITY_DESCRIPTOR_CONTROL;
```



## <a name="remarks"></a>Комментарии

Чтобы получить контрольные биты дескриптора безопасности, вызовите функцию [**жетсекуритидескрипторконтрол**](/windows/win32/api/securitybaseapi/nf-securitybaseapi-getsecuritydescriptorcontrol) . Чтобы задать контрольные биты для дескриптора безопасности, используйте функции для модификации скриптов безопасности. Список этих функций см. в разделе "см. также".

Приложения могут использовать функцию [**сетсекуритидескрипторконтрол**](/windows/win32/api/securitybaseapi/nf-securitybaseapi-setsecuritydescriptorcontrol) , чтобы задать контрольные биты, связанные с автоматическим наследованием ACE.

Значение элемента управления, полученное функцией [**жетсекуритидескрипторконтрол**](/windows/win32/api/securitybaseapi/nf-securitybaseapi-getsecuritydescriptorcontrol) , может включать сочетание битовых флагов **\_ \_ управления дескрипторами безопасности** .
