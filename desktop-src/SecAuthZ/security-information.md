# <a name="security_information"></a>\_сведения о безопасности

Тип **данных \_ Information Security** определяет, какие сведения о безопасности, связанные с объектом, задаются или запрашиваются. Эти сведения о безопасности включают:

-   Владелец объекта
-   Основная группа объекта
-   [*Список управления доступом на уровне пользователей*](/windows/desktop/SecGloss/d-gly) (DACL) объекта
-   [*Системный список управления доступом*](/windows/desktop/SecGloss/s-gly) (SACL) объекта


```C++
typedef DWORD SECURITY_INFORMATION, *PSECURITY_INFORMATION;
```



## <a name="remarks"></a>Remarks

Некоторые **элементы \_ сведений о безопасности** работают только с функцией [**сетнамедсекуритинфо**](/windows/desktop/api/Aclapi/nf-aclapi-setnamedsecurityinfoa) . Эти члены не возвращаются в структуре, возвращаемой другими функциями безопасности, такими как [**жетнамедсекуритинфо**](/windows/desktop/api/Aclapi/nf-aclapi-getnamedsecurityinfoa) или [**конвертстрингсекуритидескриптортосекуритидескриптор**](/windows/desktop/api/Sddl/nf-sddl-convertstringsecuritydescriptortosecuritydescriptora).

Каждый элемент сведений о безопасности обозначается битовым флагом. Каждый битовый флаг может принимать одно из следующих значений. Дополнительные сведения см. в разделе функции [**сетсекуритякцессмаск**](/windows/desktop/api/Securitybaseapi/nf-securitybaseapi-setsecurityaccessmask) и [**куерисекуритякцессмаск**](/windows/desktop/api/Securitybaseapi/nf-securitybaseapi-querysecurityaccessmask) .
