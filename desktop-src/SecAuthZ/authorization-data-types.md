# <a name="authorization-data-types"></a>Типы данных авторизации

В приложениях авторизации используются следующие типы данных.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                           | Описание                                                                                                                                                                                                                                                                                     |
|---------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_Маска доступа**](access-mask.md)<br/>                                  | Определяет стандартные, конкретные и универсальные права. Эти права используются в [записях управления доступом](/windows/desktop/SecGloss/a-gly) (ACE) и являются основным средством указания запрошенного или предоставленного доступа к объекту.<br/> |
| [**\_элемент управления дескриптора безопасности \_**](security-descriptor-control.md)<br/> | Набор битовых флагов, которые соответствуют значению [дескриптора безопасности](/windows/desktop/SecGloss/s-gly) или его компонентов.<br/>                                                                                                          |
| [**\_сведения о безопасности**](security-information.md)<br/>                | Определяет заданные или запрашиваемые сведения о безопасности, связанные с объектом.<br/>                                                                                                                                                                                                             |
