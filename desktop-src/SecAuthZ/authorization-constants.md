# <a name="authorization-constants-authorization"></a>Константы авторизации (авторизация)

Константы авторизации делятся на категории по использованию следующим образом.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                         | Описание                                                                                                                                                                                                                                                          |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Константы прав учетной записи**](account-rights-constants.md)<br/>       | Права учетной записи определяют тип входа, который может выполнять учетная запись пользователя. Администратор назначает права учетной записи учетным записям пользователей и групп. Права учетной записи каждого пользователя включают в себя пользователей, предоставленных пользователю, и группам, к которым принадлежит пользователь.<br/> |
| [**Константы SID контейнера приложений**](app-container-sid-constants.md)<br/> | Задиктовать центр пакетов приложений.<br/>                                                                                                                                                                                                                |
| [**Константы аудита**](auditing-constants.md)<br/>                   | Представляет категории и подкатегории событий аудита политики.<br/>                                                                                                                                                                                            |
| [**Константы идентификатора безопасности возможностей**](capability-sid-constants.md)<br/>       | Определение для хорошо известных возможностей приложений с помощью функции [**аллокатеандинитиализесид**](/windows/win32/api/securitybaseapi/nf-securitybaseapi-allocateandinitializesid) .<br/>                                                                                                                       |
| [**Константы прав доступа**](privilege-constants.md)<br/>                 | Привилегии определяют тип системных операций, которые может выполнять учетная запись пользователя. Администратор назначает привилегии учетным записям пользователей и групп. Привилегии каждого пользователя включают в себя пользователей, предоставленных пользователю, и группам, к которым принадлежит пользователь.<br/> |
