# <a name="access_mask"></a>\_Маска доступа

Тип **данных \_ Mask доступа** — это значение типа **DWORD** , определяющее стандартные, конкретные и универсальные права. Эти права используются в [*записях управления доступом*](/windows/desktop/SecGloss/a-gly) (ACE) и являются основным средством указания запрошенного или предоставленного доступа к объекту.


```C++
typedef DWORD ACCESS_MASK;
typedef ACCESS_MASK* PACCESS_MASK;
```



## <a name="remarks"></a>Комментарии

Биты в этом значении выделяются следующим образом.



| Bits             | Значение                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0 15<br/>  | Определенные права. Содержит маску доступа, относящуюся к типу объекта, связанному с маской.<br/>                                                                                                                                                                                                                                                                                                                                                                                                          |
| 16 23<br/> | Стандартные права. Содержит стандартные права доступа объекта.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| 24<br/>    | Доступ к безопасности системы (**доступ к \_ системе \_ безопасности системы**). Он используется для указания доступа к [*системному списку управления доступом*](/windows/desktop/SecGloss/s-gly) (SACL). Этот тип доступа требует, чтобы вызывающий процесс имел привилегии **\_ безопасности \_ SE** (Управление аудитом и журналом безопасности). Если этот флаг установлен в маске доступа ACE аудита доступа (успешный или неуспешный доступ), аудит доступа к SACL будет проводиться.<br/> |
| 25<br/>    | Максимально допустимое (максимально допустимое).**\_**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| 26 27<br/> | Зарезервировано.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| 28<br/>    | Универсальные все (**универсальные \_ все**).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| 29<br/>    | Универсальное выполнение (**универсальное \_ выполнение**).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| 30<br/>    | Универсальная запись (**Универсальная \_ запись**).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| 31<br/>    | Универсальное чтение (**универсальное \_ Чтение**).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |



 

Биты стандартных прав, 16 – 23, содержат стандартные права доступа объекта и могут быть сочетанием следующих предопределенных флагов.



| bit           | Flag                         | Значение                                                                                                                                                                                                                                  |
|---------------|------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 16<br/> | **DELETE**<br/>        | Удаление доступа.<br/>                                                                                                                                                                                                                |
| 17<br/> | **\_Контроль чтения**<br/> | Доступ на чтение для владельца, группы и [*списка управления доступом на уровне пользователей*](/windows/desktop/SecGloss/d-gly) (DACL) дескриптора безопасности.<br/> |
| 18<br/> | **ЗАПИСЬ \_ DAC**<br/>    | Доступ на запись к DACL.<br/>                                                                                                                                                                                                     |
| 19<br/> | **\_владелец записи**<br/>  | Доступ на запись для владельца.<br/>                                                                                                                                                                                                        |
| 20<br/> | **SYNCHRONIZE**<br/>   | Синхронизация доступа.<br/>                                                                                                                                                                                                           |



 

Следующие константы, определенные в Winnt. h, представляют конкретные и стандартные права доступа.


```C++
#define DELETE                           (0x00010000L)
#define READ_CONTROL                     (0x00020000L)
#define WRITE_DAC                        (0x00040000L)
#define WRITE_OWNER                      (0x00080000L)
#define SYNCHRONIZE                      (0x00100000L)

#define STANDARD_RIGHTS_REQUIRED         (0x000F0000L)

#define STANDARD_RIGHTS_READ             (READ_CONTROL)
#define STANDARD_RIGHTS_WRITE            (READ_CONTROL)
#define STANDARD_RIGHTS_EXECUTE          (READ_CONTROL)

#define STANDARD_RIGHTS_ALL              (0x001F0000L)

#define SPECIFIC_RIGHTS_ALL              (0x0000FFFFL)
```
