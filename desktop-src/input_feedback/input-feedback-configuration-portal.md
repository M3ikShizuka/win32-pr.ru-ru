# <a name="input-feedback-configuration"></a>Настройка обратной связи для ввода

## <a name="purpose"></a>Назначение

В подразделах этого раздела приводятся общие сведения о конфигурации отзывов о входных данных в Windows 8. Настройка обратной связи обеспечивает доступ к визуальным параметрам обратной связи для взаимодействия с пользователем.

В коллекции разделов, приведенных здесь, описывается, как использовать конфигурацию обратной связи для оптимизации взаимодействия с пользователем приложения. Например, параметры визуальной обратной связи по умолчанию могут быть ненужными или желательнои в игровом приложении из-за помех или конфликтов с самим игрового процесса.

## <a name="developer-audience"></a>Аудитория разработчиков

Обратная связь ввода Конфигуратионапис предназначена для разработчиков, создающих платформы пользовательского интерфейса, которые обеспечивают единообразное взаимодействие с пользователями в настольных приложениях.
