# <a name="input-feedback-configuration"></a>Настройка обратной связи для ввода

## <a name="purpose"></a>Назначение

В подразделах этого раздела приводится обзор конфигурации обратной связи ввода в Windows 8. Настройка обратной связи обеспечивает доступ к визуальным параметрам обратной связи для взаимодействия с пользователем.

В коллекции разделов, приведенных здесь, описывается, как использовать конфигурацию обратной связи для оптимизации взаимодействия с пользователем приложения. Например, параметры визуальной обратной связи по умолчанию могут быть ненужными или желательнои в игровом приложении из-за помех или конфликтов с самим игрового процесса.
