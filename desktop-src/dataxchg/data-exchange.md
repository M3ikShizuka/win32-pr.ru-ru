# <a name="data-exchange"></a>Обмен данными

В следующих разделах описываются основные методы обмена данными.

### <a name="data-exchange"></a>Обмен данными



| Имя                                                                                     | Описание                                                                                                                                               |
|------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Буфер обмена](clipboard.md)                                                               | Обсуждение буфера обмена. Буфер обмена — это набор функций и сообщений, которые позволяют приложениям передавать данные.<br/>                           |
| [Копирование данных](data-copy.md)                                                               | Описывает, как использовать сообщение копирования данных для перемещения данных из одного приложения в другое. <br/>                                                  |
| [платформа динамических данных Exchange](dynamic-data-exchange.md)                                       | Обсуждается реализация динамического обмена данными для приложений, которые не могут использовать библиотеку управления платформа динамических данных Exchange (ДДЕМЛ).<br/>            |
| [Библиотека управления платформа динамических данных Exchange](dynamic-data-exchange-management-library.md) | Обсуждается динамический обмен данными, который представляет собой форму межпроцессного взаимодействия, использующую общую память для обмена данными между приложениями. <br/> |
