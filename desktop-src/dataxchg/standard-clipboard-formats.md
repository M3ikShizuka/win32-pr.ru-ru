# <a name="standard-clipboard-formats"></a>Стандартные форматы буфера обмена

Форматы буфера обмена, определенные системой, называются *стандартными форматами буфера обмена*. Эти форматы буферов обмена описаны в следующей таблице.

## <a name="example"></a>Пример

```cpp
  case CF_BITMAP:
        hdcMem = CreateCompatibleDC(hdc);
        if (hdcMem != NULL)
        {
            if (OpenClipboard(hwnd))
            {
                hbm = (HBITMAP) 
                     GetClipboardData(uFormat);
                SelectObject(hdcMem, hbm);
                GetClientRect(hwnd, &rc);
 
                BitBlt(hdc, 0, 0, rc.right, rc.bottom,
                    hdcMem, 0, 0, SRCCOPY);
                CloseClipboard();
             }
             DeleteDC(hdcMem);
        }
    break;
```

Полный пример — [использование буферов обмена](using-the-clipboard.md).
