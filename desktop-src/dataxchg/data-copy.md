# <a name="data-copy"></a>Копирование данных

Функция копирования данных позволяет передавать данные из одного приложения в другое.

### <a name="in-this-section"></a>в этом разделе



| Имя                                                      | Описание                                                                                        |
|-----------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [Использование копирования данных](using-data-copy.md)<br/>         | Пример, демонстрирующий отправку данных между двумя приложениями.<br/> |
| [Справочник по копированию данных](data-copy-reference.md)<br/> | Содержит справочник по API.<br/>                                                             |



 

### <a name="data-copy-messages"></a>Сообщения копирования данных



| Имя                                           | Описание                                           |
|------------------------------------------------|-------------------------------------------------------|
| [**WM \_ COPYDATA**](wm-copydata.md)<br/> | Отправлено для передачи данных в другое приложение. <br/> |



 

### <a name="data-copy-structures"></a>Структуры копирования данных



| Имя                                                | Описание                                                                                                       |
|-----------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [**копидатаструкт**](/windows/win32/api/winuser/ns-winuser-copydatastruct)<br/> | Содержит данные, передаваемые в другое приложение с помощью сообщения [**WM \_ COPYDATA**](wm-copydata.md) . <br/> |
