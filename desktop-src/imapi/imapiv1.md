# <a name="imapiv1-support"></a>Поддержка IMAPIv1

API-интерфейс для управления образами версии 1 (IMAPIv1) позволяет выполнять этап приложения и записывать простой образ аудио или данных на устройства CD-R и CD-RW. API поддерживает форматы Жолиет и ISO 9660 для дисков аудио и данных Редбук.

> [!Note]  
> IMAPI был полностью переработан для версии 2,0. Большая часть функциональных возможностей версии 1,0 доступна в версии 2,0. При написании приложений для создания образов или разработке новых устройств и форматов рекомендуется использовать [версию 2,0](using-imapi.md) вместо версии 1,0. При запуске приложения версии 1,0 в Windows Vista интерфейсы версии 1,0 преобразуют эти вызовы в соответствующие вызовы версии 2,0, чтобы добиться одинаковой функциональности.

 

Дополнительные сведения об использовании версии 1,0 в Windows XP см. в следующих разделах:

-   [Использование API-интерфейса основного образа](using-the-image-mastering-api.md)
