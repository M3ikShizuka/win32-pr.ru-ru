# <a name="ienumfsiitemsremotenext-method"></a>Метод Иенумфсиитемс:: Ремотенекст

Поддерживает удаленный клиент, который хочет получить указанное число элементов в последовательности перечисления.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT RemoteNext(
  [in]  ULONG    celt,
  [out] IFsiItem **rgelt,
  [out] ULONG    *pceltFetched
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*celt* \[ окне\]
</dt> <dd>

Число извлекаемых элементов.

</dd> <dt>

*rgelt* \[ заполняет\]
</dt> <dd>

Массив интерфейсов [**ифсиитем**](/windows/desktop/api/imapi2fs/nn-imapi2fs-ifsiitem) . По завершении необходимо освободить каждый интерфейс в rgelt.

</dd> <dt>

*пцелтфетчед* \[ заполняет\]
</dt> <dd>

Число элементов, возвращаемых в rgelt. Если параметр *celt* равен 1, можно установить значение *Пцелтфетчед* равным **null** . В противном случае инициализируйте значение *пцелтфетчед* значением 0 перед вызовом этого метода.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

\_Если число запрошенных элементов (*celt*) возвращено успешно или число возвращаемых элементов (*пцелтфетчед*) меньше числа запрошенных элементов, то возвращается значение ОК.

Другие коды успеха могут возвращаться в результате реализации. Следующие коды ошибок обычно возвращаются при сбое операции, но не представляют единственные возможные значения ошибок:



| Код возврата                                                                                   | Описание                                                                     |
|-----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------|
| <dl> <dt>**\_указатель E**</dt> </dl>     | Недопустимый указатель.<br/> Значение: 0x80004003<br/>                   |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Не удалось выделить требуемую память.<br/> Значение: 0x8007000E<br/> |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>  | Один или несколько аргументов недопустимы.<br/> Значение: 0x80070057<br/>    |
