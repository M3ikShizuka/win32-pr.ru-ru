# <a name="glossary-imapi"></a>Глоссарий (IMAPI)

<dl> <dt>

<span id="imapi.imapi2_glossary_ANSI"></span><span id="imapi.imapi2_glossary_ansi"></span><span id="IMAPI.IMAPI2_GLOSSARY_ANSI"></span>**НАБОРА**
</dt> <dd>

Американский национальный институт стандартов (ANSI).

</dd> <dt>

<span id="imapi.imapi2_glossary_BurnEngine"></span><span id="imapi.imapi2_glossary_burnengine"></span><span id="IMAPI.IMAPI2_GLOSSARY_BURNENGINE"></span>**Записать подсистему**
</dt> <dd>

Компонент, считывающий данные из предоставленного потока данных (например, IStream или IO). Stream) и записывает эти данные на устройство чтения компакт-или DVD-дисков. Компонент должен считать доступный буфер на диске, но не содержит сведений о формате данных, подготовить диск для записи или сообщить диск о завершении процесса записи.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDDVDDevice"></span><span id="imapi.imapi2_glossary_cddvddevice"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDDVDDEVICE"></span>**Устройство чтения компакт-или DVD-дисков**
</dt> <dd>

Устройство, которое сообщает о себе как устройство типа 5. К ним относятся дисководы компакт-дисков, DVD-дисков и большинство дисков для записи компакт-дисков и дисков DVD. Кроме того, некоторые диски с магнитными носителями (например, IOMEGA) сообщают как устройства типа "тип 5" и отображаются как устройства чтения компакт-дисков в Windows.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDG"></span><span id="imapi.imapi2_glossary_cdg"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDG"></span>**CD-G** 
</dt> <dd>

Аббревиатура для графики компакт-дисков, технология, исключающая звуковые CD, позволяющая добавлять в звуковую дорожку графику с низкой скоростью. Эта технология чаще всего используется на компакт-дисках, используемых для караоке.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_R"></span><span id="imapi.imapi2_glossary_cd_r"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_R"></span>**CD-R**
</dt> <dd>

Компакт-диск однократной записи.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RandRW"></span><span id="imapi.imapi2_glossary_cd_randrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RANDRW"></span>**CD-R/RW**
</dt> <dd>

Диск CD-R или CD-RW Compact.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_ROM"></span><span id="imapi.imapi2_glossary_cd_rom"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_ROM"></span>**КОМПАКТ-ДИСК**
</dt> <dd>

Компакт-диск только для чтения.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RW"></span><span id="imapi.imapi2_glossary_cd_rw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RW"></span>**CD-RW**
</dt> <dd>

Записываемый компакт-диск с возможностью перезаписи.

</dd> <dt>

<span id="imapi.imapi2_glossaryCDText"></span><span id="imapi.imapi2_glossarycdtext"></span><span id="IMAPI.IMAPI2_GLOSSARYCDTEXT"></span>**Текст на компакт-диске** 
</dt> <dd>

Технология, исключающая с аудио компакт-дисков, которая позволяет хранить на физическом носителе сведения о низком уровне, такие как имя исполнителя и имя песни. Приложения или устройства, которые воспроизводят получившийся компакт-диск, могут затем получить эти сведения, если они существуют, для показа пользователю.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disc"></span><span id="imapi.imapi2_glossary_disc"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISC"></span>**Корр**
</dt> <dd>

Универсальный термин для циклических оптических носителей, таких как компакт-диски или DVD-диски.

</dd> <dt>

<span id="imapi.imapi2_glossary_DiscAtOnce"></span><span id="imapi.imapi2_glossary_discatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISCATONCE"></span>**Диск на один раз** 
</dt> <dd>

Метод записи на компакт-диск, где все содержимое записываемого образа записывается на диск в одном непрерывном потоке. Кроме того, в образ записи включены недоступные данные, такие как ведущий и ведущий.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disk"></span><span id="imapi.imapi2_glossary_disk"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISK"></span>**Свободного**
</dt> <dd>

Универсальный термин для круглых магнитных носителей, таких как дискета или жесткий диск.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dispatch"></span><span id="imapi.imapi2_glossary_dispatch"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISPATCH"></span>**Правляют** 
</dt> <dd>

Обозначение COM-интерфейса, в котором данный интерфейс поддерживает метод **IDispatch** вызова функции. Дополнительные сведения об этой дополнительной теме COM см. на сайте MSDN.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dual"></span><span id="imapi.imapi2_glossary_dual"></span><span id="IMAPI.IMAPI2_GLOSSARY_DUAL"></span>**Двойной** 
</dt> <dd>

Обозначение COM-интерфейса, в котором данный интерфейс поддерживает методы VTable и **IDispatch** вызова функции. Дополнительные сведения об этих дополнительных темах COM см. на сайте MSDN.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVD"></span><span id="imapi.imapi2_glossary_dvd"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVD"></span>**ОПТИЧЕСКИ**

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusR"></span><span id="imapi.imapi2_glossary_dvdplusr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSR"></span>**DVD + R**

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRDL"></span><span id="imapi.imapi2_glossary_dvdplusrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRDL"></span>**ДИСКИ DVD + R DL**

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRRW"></span><span id="imapi.imapi2_glossary_dvdplusrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRRW"></span>**DVD + R/RW**
</dt> <dd>

Диск DVD + R и диск DVD + RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRW"></span><span id="imapi.imapi2_glossary_dvdplusrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRW"></span>**DVD + RW**

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashR"></span><span id="imapi.imapi2_glossary_dvdplusdashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHR"></span>**DVD ± R**

Диск DVD + R и диск DVD-R.

<span id="imapi.imapi2_glossary_DVDPlusDashRRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRRW"></span>**DVD ± R/RW**

Компакт-диск DVD + R, диск DVD-R, диск DVD + RW и диск DVD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRW"></span>**DVD ± RW**
</dt> <dd>

Диск DVD + RW и носитель DVD + R

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashR"></span><span id="imapi.imapi2_glossary_dvddashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHR"></span>**DVD-R**

DVD-диск однократной записи. Аббревиатура произносится как «дефис тире R».

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRDL"></span><span id="imapi.imapi2_glossary_dvddashrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRDL"></span>**DL DVD-R**

Носитель с двумя слоями DVD-R.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDRAM"></span><span id="imapi.imapi2_glossary_dvdram"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDRAM"></span>**DVD-RAM**

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDROM"></span><span id="imapi.imapi2_glossary_dvdrom"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDROM"></span>**DVD-ДИСК**

DVD-диск только для чтения.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRRW"></span><span id="imapi.imapi2_glossary_dvddashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRRW"></span>**DVD-R/RW**
</dt> <dd>

Диск DVD-R и DVD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRW"></span><span id="imapi.imapi2_glossary_dvddashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRW"></span>**DVD-RW**
</dt> <dd>

Записываемый DVD-диск. Аббревиатура произносится как «тире с DVD-RW».

</dd> <dt>

<span id="imapi.imapi2_glossary_ECMA"></span><span id="imapi.imapi2_glossary_ecma"></span><span id="IMAPI.IMAPI2_GLOSSARY_ECMA"></span>**ECMA** 
</dt> <dd>

Ассоциация компаний-производителей компьютеров — это ассоциация в 1961, предназначенная для стандартизации информационных и коммуникационных систем. ECMA изменяет стандарты и технические отчеты. Все публикации ECMA доступны бесплатно.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPI"></span><span id="imapi.imapi2_glossary_imapi"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPI"></span>**IMAPI** 
</dt> <dd>

API-интерфейс для создания образов, который используется при создании и записи образов (в отличие от активных FS) для устройств CD/DVD. Этот термин обычно применяется ко всем версиям API-интерфейсов для работы с образами.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv1"></span><span id="imapi.imapi2_glossary_imapiv1"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV1"></span>**IMAPIv1** 
</dt> <dd>

версия IMAPI, впервые представленная в Windows XP. IMAPI 1,0 поддерживает носители CD-R и CD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv2"></span><span id="imapi.imapi2_glossary_imapiv2"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV2"></span>**IMAPIv2** 
</dt> <dd>

вторая версия IMAPI, выпущенная с Windows Vista.

</dd> <dt>

<span id="imapi.imapi2_glossary_INCITS"></span><span id="imapi.imapi2_glossary_incits"></span><span id="IMAPI.IMAPI2_GLOSSARY_INCITS"></span>**инЦитс**
</dt> <dd>

Международный комитет стандартов информационных технологий, организация, аккредитованное и утвержденная ANSI.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISO9660"></span><span id="imapi.imapi2_glossary_iso9660"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISO9660"></span>**ISO9660**
</dt> <dd>

Файловая система, предназначенная для первого поколения компакт-дисков. Эта файловая система имеет значительные ограничения по имени файла и сортировке.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISOImage"></span><span id="imapi.imapi2_glossary_isoimage"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISOIMAGE"></span>**ISO-образ**
</dt> <dd>

Общее имя для посекторного изображения всей области данных компакт-диска с данными или диска DVD. Имя выросло из общего использования ISO9660 в качестве файловой системы для компакт-носителя.

</dd> <dt>

<span id="imapi.imapi2_glossary_Joliet"></span><span id="imapi.imapi2_glossary_joliet"></span><span id="IMAPI.IMAPI2_GLOSSARY_JOLIET"></span>**жолиет**
</dt> <dd>

Набор стандартизованных расширений для ISO9660, поддерживающих имена файлов в Юникоде длиной до 64 символов. Дополнительные сведения см. в спецификации ECMA с номером 119.

</dd> <dt>

<span id="imapi.imapi2_glossary_MMC"></span><span id="imapi.imapi2_glossary_mmc"></span><span id="IMAPI.IMAPI2_GLOSSARY_MMC"></span>**УПРАВЛЕНИЯ**
</dt> <dd>

Набор команд мультимедиа, определяемый Рабочей группой T10 ИНЦИТС. MMC определяет команды пакетов для устройств CD и DVD. Текущие рабочие черновики MMC можно найти на веб-сайте T10.

</dd> <dt>

<span id="imapi.imapi2_glossary_MtFuji"></span><span id="imapi.imapi2_glossary_mtfuji"></span><span id="IMAPI.IMAPI2_GLOSSARY_MTFUJI"></span>**MT. Fuji**
</dt> <dd>

Набор команд для управления устройствами мультимедиа, аналогичный набору команд MMC. Некоторые элементы (например, функции или типы мультимедиа) описаны в Mt. Fuji документы, а другие — в документах MMC. Текущие рабочие черновики набора команд можно найти по адресу [FTP://FTP.AVC-Pioneer.com](ftp://ftp.avc-pioneer.com/).

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSI"></span><span id="imapi.imapi2_glossary_scsi"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSI"></span>**ИНТЕРФЕЙСА**
</dt> <dd>

Интерфейс малых компьютерных систем.

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSICommand"></span><span id="imapi.imapi2_glossary_scsicommand"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSICOMMAND"></span>**Команда SCSI**
</dt> <dd>

Любой из нескольких команд пакета, например, определенных в документе MMC. Эти команды можно отправить с помощью любого подходящего транспортного механизма, такого как SCSI, ATAPI, USB или IEEE1394.

</dd> <dt>

<span id="imapi.imapi2_glossary_SessonAtOnce"></span><span id="imapi.imapi2_glossary_sessonatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_SESSONATONCE"></span>**Сеанс — один раз**
</dt> <dd>

Метод создания компакт-диска и диска. Один сеанс записи записывает звуковые дорожки на диск, а второй сеанс записывает на диск одну запись CD-XA и закрывает диск.

</dd> <dt>

<span id="imapi.imapi2_glossary_TrackAtOnce"></span><span id="imapi.imapi2_glossary_trackatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_TRACKATONCE"></span>**Однократная трассировка**
</dt> <dd>

Метод записи на компакт-диск, где данные для одной записи записываются за один раз. Когда записываются все дорожки, сеанс закрывается в качестве завершающего шага, включающего запись ведущих и ведущих областей диска.

</dd> <dt>

<span id="imapi.imapi2_glossary_UDF"></span><span id="imapi.imapi2_glossary_udf"></span><span id="IMAPI.IMAPI2_GLOSSARY_UDF"></span>**Формат универсального диска (UDF)**
</dt> <dd>

эта файловая система определяется оста (технология оптического служба хранилища технологии), которая поддерживает диски с одним сеансом и несколькими сеансами.

</dd> <dt>

<span id="imapi.imapi2_glossary_VTable"></span><span id="imapi.imapi2_glossary_vtable"></span><span id="IMAPI.IMAPI2_GLOSSARY_VTABLE"></span>**VTable**

Обозначение COM-интерфейса, в котором данный интерфейс поддерживает метод vtable для вызова функции. Дополнительные сведения об этой дополнительной теме COM см. на сайте MSDN.

</dd> </dl>
