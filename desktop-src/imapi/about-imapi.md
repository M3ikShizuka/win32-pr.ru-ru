# <a name="about-imapi"></a>О программе IMAPI

IMAPI позволяет создавать оптические носители с помощью различных файловых систем.

## <a name="features"></a>Компоненты

IMAPI предоставляет следующие возможности:

-   Создает и записывает односеансовые и многосеансовые диски.
-   Создает и записывает загрузочные диски.
-   Создает и записывает образы необработанного компакт-диска.
-   Предоставляет низкоуровневый доступ к подсистеме записи для разработки поддержки новых устройств.
-   Предоставляет доступ к расширенным свойствам средства записи для разработки поддержки для новых устройств.
-   Предоставляет параметры проверки записи.

IMAPI поддерживает следующие форматы. Дополнительные сведения см. в разделе [форматы дисков](disc-formats.md).

-   Формат изображения ISO9660 (включает Редбук Audio).
-   Формат Жолиет.
-   Формат универсального диска (UDF).

IMAPI поддерживает следующие устройства мультимедиа.

-   Записываемый компакт-диск (CD-R). Ранее назывался однократным написанием компакт-дисков.
-   Перезаписываемый компакт-диск (CD-RW)
-   Записываемый диск DVD (DVD-R и DVD + R).
-   Перезаписываемый DVD (DVD-RW и DVD + RW).
-   DVD с произвольным доступом (DVD-RAM).
-   Записываемый DVD-диск с двумя слоями (в форматах DVD и DVD + R).
-   Записываемый диск BD (BD-R)
-   Перезаписываемый диск BD (BD-RE)
-   Выполняет запись на носитель, подобный диску.
