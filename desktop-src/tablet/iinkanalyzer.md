# <a name="iinkanalyzer-interface"></a>Интерфейс Иинканализер

Предоставляет доступ к анализу макета, разметке, классификации и распознаванию рукописного текста.

## <a name="members"></a>Элементы

Интерфейс **иинканализер** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иинканализер** также имеет следующие типы членов:

-   [Методы](#methods)

## <a name="remarks"></a>Remarks

**Иинканализер** использует данные пакетной обводки для анализа рукописного ввода и не взаимодействует с объектами коллекции [**инкдисп**](inkdisp-class.md) или [инкстрокес](/previous-versions/windows/desktop/legacy/ms703293(v=vs.85)) напрямую.

Чтобы добавить или удалить штрихи для **иинканализер** для анализа, используйте один из следующих методов.

-   [**Метод Иинканализер:: Аддстроке**](iinkanalyzer-addstroke.md)
-   [**Метод Иинканализер:: Аддстрокес**](iinkanalyzer-addstrokes.md)
-   [**Метод Иинканализер:: Ремовестроке**](iinkanalyzer-removestroke.md)
-   [**Метод Иинканализер:: Ремовестрокес**](iinkanalyzer-removestrokes.md)

Эти методы обновляют "грязную" область (см. [**метод иинканализер:: жетдиртирегион**](iinkanalyzer-getdirtyregion.md)), который является регионом, для которого анализируются росчерки при следующей операции анализа.

Для анализа рукописного ввода используйте метод метода [**иинканализер:: Analyze**](iinkanalyzer-analyze.md) или метод [**Иинканализер:: баккграунданализе**](iinkanalyzer-backgroundanalyze.md) . Во время анализа **иинканализер** выполняет анализ макета, классификацию обводки и распознавание рукописного текста.

Чтобы изменить параметры анализа макета и классификации обводки, используйте свойство [**метода иинканализер:: сетаналисисмодес**](iinkanalyzer-setanalysismodes.md) .

Во время анализа **иинканализер** получает ряд событий, включая события, созданные во время фонового анализа. [**\_ Ианалисиспроксевентс**](-ianalysisproxyevents.md) поддерживает функции прокси-сервера данных **иинканализер**. Дополнительные сведения см. [в разделе Учетная запись-посредник данных с помощью анализа рукописного ввода](data-proxy-with-ink-analysis.md). Чтобы остановить процесс анализа в обработчике событий, вызовите [**метод иинканализер:: Abort**](iinkanalyzer-abort.md).

Чтобы изменить язык, используемый анализатором рукописного ввода для распознавания рукописного текста, используйте метод [**иинканализер:: сетстрокелангуажеид**](iinkanalyzer-setstrokelanguageid.md) или [**метод Иинканализер:: сетстрокеслангуажеид**](iinkanalyzer-setstrokeslanguageid.md). Чтобы изменить, как анализатор рукописного ввода классифицирует определенные штрихи, используйте метод [**иинканализер:: сетстрокетипе**](iinkanalyzer-setstroketype.md) или [**метод Иинканализер:: сетстрокестипе**](iinkanalyzer-setstrokestype.md).

**Иинканализер** загружает сведения обо всех установленных распознавателях рукописного ввода. [**Метод иинканализер:: жетинканалисисрекогнизерсбиприорити**](iinkanalyzer-getinkanalysisrecognizersbypriority.md) Возвращает коллекцию [**иинканалисисрекогнизерс**](iinkanalysisrecognizers.md) , содержащую все доступные [**иинканалисисрекогнизер**](iinkanalysisrecognizer.md). Если более одного распознавателя рукописного ввода поддерживает конкретный язык, используйте [**метод иинканализер:: сесигхестприоритинканалисисрекогнизер**](iinkanalyzer-sethighestpriorityinkanalysisrecognizer.md) , чтобы задать, какой распознаватель рукописного ввода обрабатывает штрихи для этого языка.

Использование подсказок по анализу может повысить точность распознавания, предоставляя дополнительный контекст анализатору рукописного ввода. Дополнительные сведения о контексте могут помочь анализатору рукописного ввода ограничить количество возможных результатов распознавания. Например, можно ограничить область, определив фактоидс и ожидаемые слова или выполнив структурирование входных данных в руководством по распознаванию. Дополнительные сведения о предоставлении контекста анализатору рукописного ввода см. в следующих статьях:

-   [**Метод Иинканализер:: Креатеаналисишинт**](iinkanalyzer-createanalysishint.md)
-   [**Иинканализер: метод:D Елетеаналисишинт**](iinkanalyzer-deleteanalysishint.md)
-   [**Метод Иинканализер:: Жетаналисишинтс**](iinkanalyzer-getanalysishints.md)
-   [**Метод Иинканализер:: Жетаналисишинтсбинаме**](iinkanalyzer-getanalysishintsbyname.md)

Анализатор рукописного ввода представляет результаты анализа в виде строки или дерева объектов [**иконтекстноде**](icontextnode.md) . Чтобы получить доступ к распознанной строке, используйте [**метод иинканализер:: жетрекогнизедстринг**](iinkanalyzer-getrecognizedstring.md). Чтобы получить доступ к корню дерева узлов контекста, используйте [**метод иинканализер:: жетрутноде**](iinkanalyzer-getrootnode.md).
