# <a name="setting-context-for-windowless-controls"></a>Настройка контекста для безоконных элементов управления

Элементы управления без окон могут вызывать функцию [**сетинпутскопе**](/windows/win32/api/inputscope/nf-inputscope-setinputscope) всякий раз, когда фокус перемещается между элементами безоконного элемента управления; Однако это не рекомендуется для повышения производительности.
