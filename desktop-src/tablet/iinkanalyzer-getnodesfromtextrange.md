# <a name="iinkanalyzergetnodesfromtextrange-method"></a>Метод Иинканализер:: Жетнодесфромтекстранже

Извлекает коллекцию объектов [**иконтекстноде**](icontextnode.md) , относящихся к указанному текстовому диапазону для указанных узлов контекста.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetNodesFromTextRange(
  [in, out] LONG          *plStart,
  [in, out] LONG          *plLength,
  [out]     IContextNodes **ppContextNodes,
  [in]      IContextNodes *pNodesToSearch = defaultvalue
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*плстарт* \[ в, out\]
</dt> <dd>

Ссылка на начало текстового диапазона в *пнодестосеарч* части распознанной строки.

</dd> <dt>

*плленгс* \[ в, out\]
</dt> <dd>

Ссылка на длину текстового диапазона в *пнодестосеарч* части распознанной строки.

</dd> <dt>

*ппконтекстнодес* \[ заполняет\]
</dt> <dd>

Указатель на объекты [**иконтекстноде**](icontextnode.md) , относящиеся к указанному текстовому диапазону для указанных узлов контекста.

</dd> <dt>

*пнодестосеарч* \[ окне\]
</dt> <dd>

Объекты [**иконтекстноде**](icontextnode.md) , для которых необходимо ограничить поиск.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Описание возвращаемых значений см. в разделе [классы и интерфейсы — анализ рукописного ввода](classes-and-interfaces---ink-analysis.md).

## <a name="remarks"></a>Remarks

Указанный диапазон текста должен относиться к *пнодестосеарч* части распознанной строки [**иинканализер**](iinkanalyzer.md), а не к распознанной строке всего **иинканализер**.

Этот метод изменяет значения параметров *плстарт* и *плленгс* , расширяя диапазон текста до ближайших границ слов.

Например, если распознанная строка является "I-поздно", и этот метод вызывается с помощью значений параметра 6 для *плстарт* и 1 для *плленгс*, который соответствует букве "a" в "позднем", этот метод возвращает коллекцию, содержащую один [**иконтекстноде**](icontextnode.md), инкворд или текстворд, соответствующий слову "поздно". В этом примере этот метод также изменяет значение *плстарт* на 5, а значение *плленгс* — на 4, что соответствует слову "поздно".

> [!Note]  
> Параметр *плстарт* задается относительно распознанной строки параметра *пнодестосеарч* .
