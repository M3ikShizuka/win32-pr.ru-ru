# <a name="recognizer-enumerations"></a>Перечисления модуля распознавания

В этом разделе описаны перечисления распознавателя.

## <a name="in-this-section"></a>в этом разделе



| Перечисление                                   | Описание                                                                                |
|-----------------------------------------------|--------------------------------------------------------------------------------------------|
| [**\_разрывы Alt**](/windows/win32/api/rectypes/ne-rectypes-alt_breaks)             | Указывает, как создавать альтернативы из лучшей результирующей строки.<br/>                   |
| [**\_уровень достоверности**](/windows/win32/api/rectypes/ne-rectypes-confidence_level) | Указывает уровень достоверности, который распознаватель имеет в результатах распознавания.<br/> |
| [**\_метрики линий**](/windows/win32/api/rectypes/ne-rectypes-line_metrics)         | Представляет строки, найденные в сегменте.<br/>                                        |
| [**\_единицы свойства**](/windows/desktop/api/tpcshrd/ne-tpcshrd-property_units)     | Задает единицу измерения для свойства.<br/>                                |
