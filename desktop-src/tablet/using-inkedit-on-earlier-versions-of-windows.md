# <a name="using-inkedit-on-earlier-versions-of-windows"></a>Использование InkEdit в более ранних версиях Windows

из-за отсутствия распознавателей в выпусках Microsoft Windows XP, отличных от планшетных пк, вы не можете использовать [InkEdit](inkedit-control.md) для визуализации рукописного ввода на Windows 2000, Windows Server 2003 и выпусках, отличных от планшетных пк Windows XP, и вы не можете использовать этот элемент управления для ввода рукописного ввода в этих операционных системах.

Свойство [**инкмоде**](/windows/desktop/api/inked/nf-inked-iinkedit-get_inkmode) элемента управления имеет значение **disabled** (**Настройка Internet Explorer \_ отключено** для C++), и попытка изменить свойство не оказывает никакого влияния. Можно назначать значения другим свойствам, копировать и вставлять рукописные данные в другие приложения, но нельзя использовать этот элемент управления для приема жестов или получения рукописных данных.
