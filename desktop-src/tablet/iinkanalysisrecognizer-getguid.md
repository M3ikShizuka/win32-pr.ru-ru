# <a name="iinkanalysisrecognizergetguid-method"></a>Метод Иинканалисисрекогнизер::

Получает глобальный уникальный идентификатор (GUID) распознавателя.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetGuid(
  [out] GUID *pId
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*идентификатор процесса* \[ заполняет\]
</dt> <dd>

Идентификатор GUID, определяющий этот [**иинканалисисрекогнизер**](iinkanalysisrecognizer.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение
