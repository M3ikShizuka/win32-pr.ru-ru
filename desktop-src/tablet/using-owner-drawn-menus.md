# <a name="using-owner-drawn-menus"></a>Использование меню Owner-Drawn

При использовании меню, рисуемых владельцем, необходимо сделать имена меню доступными для поддержки речевых функций. Это можно сделать двумя способами.

-   Предоставьте имя пункта меню с помощью структуры МСААМЕНУИНФО.
-   Укажите параметр для замены графических меню стандартными текстовыми меню, если активна вспомогательная функция. Если функция [**системпараметерсинфо**](/windows/win32/api/winuser/nf-winuser-systemparametersinfoa) возвращает **значение true** с параметром *уиактион* , установленным в значение SPI \_ жетскринреадер, используйте стандартные меню. Приложение должно отслеживать \_ сообщение WM сеттингсчанже и реагировать на него, запрашивая состояние этого параметра и соответствующим образом настраивая его отображение. Например, Microsoft Visual Studio предоставляет возможность использовать стандартные меню вместо пользовательских меню, отображаемых по умолчанию.
