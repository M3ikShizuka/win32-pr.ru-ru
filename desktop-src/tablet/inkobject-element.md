# <a name="inkobject-element"></a>Инкобжект, элемент

Содержит двоичные данные в кодировке Base64 объекта Ink для [**инкворд**](inkword-element.md) или [**Drawing**](drawing-element.md).

## <a name="definition"></a>Определение

``` syntax
<xs:element name="InkObject" type="xs:base64Binary" />
```

## <a name="parent-elements"></a>Родительские элементы

[**инкворд**](inkword-element.md)

[**Рисование**](drawing-element.md)

## <a name="child-elements"></a>Дочерние элементы

Отсутствует.

## <a name="attributes"></a>Атрибуты

Отсутствует.

## <a name="element-information"></a>Сведения об элементе



|  Элемент     | Значение                                                     |
|--------------|--------------------------------------------|
| Тип элемента | **xs:base64Binary**                        |
| Пространство имен    | urn: schemas-microsoft-com: TabletPC: ричинк |
| Имя схемы  | Средство чтения журнала                             |
