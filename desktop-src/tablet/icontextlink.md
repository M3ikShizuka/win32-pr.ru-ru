# <a name="icontextlink-interface"></a>Интерфейс Иконтекстлинк

Представляет связь между двумя объектами [**иконтекстноде**](icontextnode.md) .

## <a name="members"></a>Элементы

Интерфейс **иконтекстлинк** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иконтекстлинк** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иконтекстлинк** содержит следующие методы.



| Метод                                                                  | Описание                                                                                                             |
|:------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------|
| [**жетконтекстлинкдиректион**](icontextlink-getcontextlinkdirection.md) | Извлекает тип отношения, представляемого этим **иконтекстлинк** .<br/>                                         |
| [**жетдестинатионноде**](icontextlink-getdestinationnode.md)           | Извлекает объект [**иконтекстноде**](icontextnode.md) , который является назначением для этого **иконтекстлинк**.<br/> |
| [**жетсаурценоде**](icontextlink-getsourcenode.md)                     | Извлекает объект [**иконтекстноде**](icontextnode.md) , являющийся источником для этого **иконтекстлинк**.<br/>      |



 

## <a name="remarks"></a>Комментарии

Ниже приведен пример связи, представленной объектом **иконтекстлинк** .

-   При использовании узла Аналисишинт в анализе рукописного ввода операция анализа рукописного ввода создает объект **иконтекстлинк** типа аналисишинт между узлом подсказок по анализу и узлом, содержащим запись в области подсказки. Исходный узел — это узел указания по анализу, а узел назначения — это узел, содержащий запись.
