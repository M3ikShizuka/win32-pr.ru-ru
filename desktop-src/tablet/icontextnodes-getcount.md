# <a name="icontextnodesgetcount-method"></a>Метод Иконтекстнодес:: NOCOUNT

Возвращает число объектов [**иконтекстноде**](icontextnode.md) , содержащихся в этой коллекции.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetCount(
  [out] ULONG *pulCount
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пулкаунт* \[ заполняет\]
</dt> <dd>

Число объектов [**иконтекстноде**](icontextnode.md) , содержащихся в этой коллекции.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Описание возвращаемых значений см. в разделе [классы и интерфейсы — анализ рукописного ввода](classes-and-interfaces---ink-analysis.md).
