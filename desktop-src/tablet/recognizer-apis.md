# <a name="recognizer-reference"></a>Справочник по распознавателю

Распознаватель обрабатывает:

-   Определите, какие распознаватели доступны на компьютере.
-   Укажите, какой распознаватель следует использовать.
-   Создайте контекст распознавателя.
-   Получение результатов и альтернатив распознавателя.

| Раздел                                                      | Описание                                                                                                                                                           |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [ХРЕКОАЛТ, обработчик](hrecoalt-handle.md)                     | Извлекает альтернативные значения строки и свойства, такие как уровень достоверности и метрики.<br/>                                                                  |
| [ХРЕКОКОНТЕКСТ, обработчик](hrecocontext-handle.md)             | Добавляет рукописный ввод в контекст, выполняет распознавание рукописного текста и получает результаты распознавания и альтернативные варианты.<br/>                                                    |
| [ХРЕКОГНИЗЕР, обработчик](hrecognizer-handle.md)               | Создает контекст распознавателя, извлекает его атрибуты и свойства, а также определяет, какие свойства пакетов должен выполнять распознавание распознавателем.<br/> |
| [ХРЕКОВОРДЛИСТ, обработчик](hrecowordlist-handle.md)           | Управляет списком слов, который присоединяется к контексту распознавателя. Этот список улучшает результаты распознавания.<br/>                                                   |
| [Перечисления модуля распознавания](recognizer-api-enumerations.md) | Описывает типы перечислений распознавателя.<br/>                                                                                                                |

| [Идентификаторы GUID распознавателя](recognizer-guids.md)                   | Указывает тип свойства для свойств пакета и свойств распознавателя.<br/>                                                                            |
