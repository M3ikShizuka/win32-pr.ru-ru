# <a name="using-microsoft-active-accessibility"></a>Использование Microsoft Active Accessibility

Модуль распознавания речи на планшетном ПК использует Active Accessibility для получения сведений непосредственно из объекта и получения уведомления об изменении объектов. Этот интерфейс поддерживается стандартными элементами управления Microsoft Windows и может также использоваться пользовательскими элементами управления и пользовательскими классами окон.
