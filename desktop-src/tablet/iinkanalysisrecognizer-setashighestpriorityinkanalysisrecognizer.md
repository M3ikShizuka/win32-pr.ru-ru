# <a name="iinkanalysisrecognizersetashighestpriorityinkanalysisrecognizer-method"></a>Метод Иинканалисисрекогнизер:: Сеташигхестприоритинканалисисрекогнизер

Изменяет этот [**иинканалисисрекогнизер**](iinkanalysisrecognizer.md) на наивысший приоритет в внутреннем упорядоченном списке распознавателей [**иинканализер**](iinkanalyzer.md) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetAsHighestPriorityInkAnalysisRecognizer();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.
