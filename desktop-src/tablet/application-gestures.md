# <a name="application-gestures"></a>Жесты приложения

Жесты приложения — это жесты, которые вы выбрали для поддержки вашего приложения. Они наиболее распространены в приложениях, специально предназначенных для работы с планшетным пером. Устаревшее приложение может использовать жесты, но существующие модели выбора в приложении могут усложнить поддержку жестов. Только те жесты, которые поддерживаются на панели ввода Tablet PC, доступны для устаревших приложений.

В следующих разделах описываются жесты приложения и их использование.

-   [Использование пользовательского распознавателя жестов](using-a-custom-gesture-recognizer.md)
-   [Потенциальные конфликты с жестами приложений](potential-conflicts-with-application-gestures.md)
-   [Документирование жестов, используемых приложениями](documenting-the-gestures-used-by-your-applications.md)
