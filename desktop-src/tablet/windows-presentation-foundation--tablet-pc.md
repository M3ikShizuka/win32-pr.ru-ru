# <a name="windows-presentation-foundation-tablet-pc"></a>Windows Presentation Foundation (планшетный ПК)

Функции планшетных ПК Windows Presentation Foundation (WPF) не нуждаются в специальной настройке.

Следуйте инструкциям по настройке WPF для использования компонента InkCanvas и других компонентов WPF.
