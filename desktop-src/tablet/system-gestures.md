# <a name="system-gestures"></a>Системные жесты

Системные жесты сопоставляются с традиционными сообщениями мыши. Планшетные ПК поддерживают жесты ввода системы пера, которые выводится щелчком левой кнопкой мыши, двойным щелчком, щелчком правой кнопки, перетаскиванием влево, щелчком правой кнопкой мыши и сообщениями мыши с наведением. Приложения, разработанные для планшетных ПК, должны поддерживать эти жесты так же, как они поддерживают действия мыши. Для традиционных приложений многие события, активируемые жестами, обрабатываются как традиционные сообщения мыши.
