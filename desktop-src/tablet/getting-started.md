# <a name="getting-started-tablet-pc"></a>Начало работы (планшетный ПК)

Tablet PC — это полнофункциональный персональный компьютер на базе Microsoft Windows, предназначенный для приложений с поддержкой рукописного ввода, с поддержкой пера и речи. Сочетание программного и аппаратного обеспечения на планшетном ПК позволяет этим методам взаимодействовать с пользователем, создавая полнофункциональную, интерактивную и продуктивную вычислительную среду для пользователей.

Платформа Tablet PC позволяет вводить и выводить рукописные и речевые данные на планшетном ПК, а также осуществлять обмен данными с другими компьютерами. Компоненты платформы Tablet PC, установленные с помощью пакета SDK для Windows 7, позволяют создавать приложения с поддержкой рукописного ввода и с поддержкой пера для планшетных ПК.
