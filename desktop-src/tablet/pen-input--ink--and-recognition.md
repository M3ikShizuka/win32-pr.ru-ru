# <a name="pen-input-ink-and-recognition"></a>Ввод, ввод и распознавание с помощью пера

Одной из интересных новых возможностей Tablet PC является система ввода с помощью пера. На всех компьютерах Tablet PC имеется Дигитайзер под экраном, который принимает ввод с помощью пера. Этот новый механизм ввода требует думать о том, как создавать приложения специально для планшетных ПК. Интерфейс прикладного программирования (API) платформы Tablet PC позволяет сделать это.

## <a name="collection-data-management-and-recognition"></a>Сбор, Управление данными и распознавание

Платформу для планшетных ПК можно разделить на три отдельные области:

-   Коллекция рукописных данных (объекты, используемые для сбора рукописного ввода с дигитайзера).
-   Управление рукописными данными (объекты, используемые для управления собранными красками).
-   Распознавание рукописного ввода (объекты, используемые для преобразования собранного рукописного ввода в другие типы данных, такие как текст).

На следующем рисунке на высоком уровне показано, как API сбора рукописного ввода (API), рукописный Управление данными API (рукописный ввод) и API распознавания рукописного ввода (API распознавания) работают вместе на платформе Tablet PC.
