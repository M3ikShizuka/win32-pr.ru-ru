# <a name="exposing-text-programmatically"></a>Предоставление текста программным способом

Предоставление доступа к тексту программным способом обеспечивает значительное преимущество специальных возможностей, предоставляя гораздо более надежный и более наглядный способ интерпретации информации.

Существует несколько основных методов предоставления объектов для программного предоставления текста. Однако ни один из методов не эффективен со всеми вспомогательными средствами, поэтому на экране также необходимо вывести текст, чтобы его можно было распознать. Сведения о предоставлении текста больших объемов см. [в разделе использование Active Accessibility для представления текста](using-active-accessibility-to-expose-text.md).
