# <a name="object-recognizers"></a>Распознаватели объектов

В дополнение к распознаванию текста распознаватели могут распознать класс связанных объектов. Распознаватели объектов распознают общие фигуры в соответствии с их целями. Распознаватели объектов используются для распознавания:

-   Музыкальные заметки
-   Геометрические фигуры
-   Математические уравнения
-   Элементы блоковой диаграммы

Как правило, объекты, распознаваемые таким распознавателем, находятся в двумерной пространственной или функциональной связи друг с другом. Например, в сложных отношениях в математическом уравнении распознаватель может возвращать разные результаты для верхнего предела определенного целого числа, а не числителя в дробной части.

Из-за общего характера этих связей чрезвычайно сложно определить набор интерфейсов, которые будут работать для каждого распознавателя объектов.

Технология планшетных ПК предоставляет базовую платформу для распознавателей объектов в управляемых библиотеках и интерфейсах автоматизации. Однако необходимо разработать пользовательские интерфейсы, описывающие сложные пространственные связи между распознаваемыми объектами для каждого распознавателя объектов. В частности, для распознавателей объектов платформа предоставляет базовый объект [**рекогнизерконтекст**](inkrecognizercontext-class.md) для связывания объекта [**рукописного ввода**](inkdisp-class.md) с контекстом распознавателя и для вызова распознавателя для выполнения распознавания.
