# <a name="icontextnodesaddcontextnode-method"></a>Метод Иконтекстнодес:: Аддконтекстноде

## <a name="syntax"></a>Синтаксис


```C++
HRESULT AddContextNode(
  [in] IContextNode *pContextNode
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пконтекстноде* \[ окне\]
</dt> <dd>

Добавляемый объект [**иконтекстноде**](icontextnode.md) .

</dd> </dl>
