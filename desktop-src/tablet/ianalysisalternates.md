# <a name="ianalysisalternates-interface"></a>Интерфейс Ианалисисалтернатес

Интерфейс **ианалисисалтернатес** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Ианалисисалтернатес** также имеет следующие типы членов:

-   [Методы](#methods)

Интерфейс **ианалисисалтернатес** содержит следующие методы.



| Метод                                                                   | Описание                                                                                                                                      |
|:-------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**жетаналисисалтернате**](ianalysisalternates-getanalysisalternate.md) | Извлекает объект [**ианалисисалтернате**](ianalysisalternate.md) по указанному индексу в коллекции.<br/>                   |
