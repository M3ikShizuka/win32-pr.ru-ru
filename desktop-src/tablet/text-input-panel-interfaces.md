# <a name="text-input-panel-interfaces"></a>Интерфейсы панели ввода текста

В этом разделе содержатся сведения об интерфейсах, используемых для управления внешним видом и поведением панели ввода Tablet PC.

> [!Note]  
> Интерфейсы панели ввода текста не совместимы с автоматизацией.

 

## <a name="in-this-section"></a>в этом разделе



| Интерфейс                                                                | Описание                                                                                                                                  |
|--------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------|
| [**Интерфейс Ихандвриттентекстинсертион**](/windows/desktop/api/peninputpanel/nn-peninputpanel-ihandwrittentextinsertion) | Используется в пользовательском коде ввода текста приложения для вставки текста в текстовое поле и в резервное хранилище текстовых служб.<br/> |
| [**Интерфейс Итекстинпутпанел**](/windows/desktop/api/peninputpanel/nn-peninputpanel-itextinputpanel)                     | Обеспечивает управление панелью ввода Tablet PC.<br/>                                                                                  |
| [**Интерфейс Итекстинпутпанелевентсинк**](/windows/desktop/api/peninputpanel/nn-peninputpanel-itextinputpaneleventsink)   | Определяет методы, обрабатывающие события [**интерфейса итекстинпутпанел**](/windows/desktop/api/peninputpanel/nn-peninputpanel-itextinputpanel) .<br/>                                      |
| [**Интерфейс Итекстинпутпанелрунинфо**](/windows/desktop/api/peninputpanel/nn-peninputpanel-itextinputpanelruninfo)       | Предоставляет метод, позволяющий определить, выполняется ли в данный момент панель ввода текста.<br/>                                                      |
| [**Интерфейс Итипаутокомплетеклиент**](itipautocompleteclient.md)       | Позволяет клиенту вызывать объект поставщика автоматического завершения для панели ввода текста приложения.<br/>                                      |
| [**Интерфейс Итипаутокомплетепровидер**](itipautocompleteprovider.md)   | Управляет стороны приложения для интеграции автоматического завершения автоматической сборки панели ввода текста.<br/>                                                 |
