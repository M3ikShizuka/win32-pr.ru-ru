# <a name="registering-your-recognizer-dll"></a>Регистрация библиотеки DLL распознавателя

Библиотека DLL распознавателя должна быть зарегистрирована для использования платформой Tablet PC. Распознаватель должен внести следующие изменения в реестр при установке.

Добавьте новый ключ для каждого из идентификаторов CLSID распознавателя в разделе HKEY \_ Local \_ Machine/Software/Microsoft/ТПГ/распознаватели/реестра. Добавьте один CLSID для каждого языка. В следующей таблице описаны значения, которые должны быть добавлены под каждым из ключей, содержащих идентификаторы CLSID.

> [!Note]  
> В именах этих значений учитывается регистр.

 



| Имя значения                             | Тип значения             | Описание значения                                                                                                                                                                                                                                                                                                                                                                                  |
|----------------------------------------|------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Флаги возможностей распознавателя<br/> | REG \_ DWORD<br/>  | DWORD, используемый для определения возможностей распознавателя.<br/>                                                                                                                                                                                                                                                                                                                             |
| Библиотека DLL распознавателя<br/>              | REG \_ SZ<br/>     | Полный путь к установленной библиотеке DLL распознавателя.<br/>                                                                                                                                                                                                                                                                                                                                              |
| Распознаваемые языки<br/>        | \_двоичный файл REG<br/> | Двоичный массив, описывающий язык или языки, с которыми будет работать распознаватель.<br/> В параметре переctype. h \_ определяется максимальное число языков, поддерживаемое распознавателем, и каждый язык берет слово для хранения в элементе «распознаваемые языки». Размер \_ записи реестра binary reg должен быть РАВЕН Max \_ Languages \* sizeof (Word).<br/> |
