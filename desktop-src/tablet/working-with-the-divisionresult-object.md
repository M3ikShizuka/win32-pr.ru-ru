# <a name="working-with-the-divisionresult-object"></a>Работа с объектом Дивисионресулт

Объект **дивисионресулт** представляет собой моментальный снимок анализа макета для штрихов, содержащихся в объекте- **разделителе** . Он содержит классификацию штрихов и сведения о группировании из анализа макета.

Сведения из объекта **дивисионресулт** можно получить по типу структурного элемента, например по рисованию или линиям рукописного ввода. Объект **дивисионресулт** может сохраняться после уничтожения объекта- **разделителя** . В службе автоматизации этот объект называется объектом [**интерфейса иинкдивисионресулт**](/windows/desktop/api/msinkaut15/nn-msinkaut15-iinkdivisionresult) .

Свойство [**Strokes**](/windows/desktop/api/msinkaut15/nf-msinkaut15-iinkdivisionresult-get_strokes) объекта **дивисионресулт** содержит копию коллекции **штрихов** в объекте- **разделителе** на момент создания объекта **дивисионресулт** .

Перечисление [**инкдивисионтипе**](/windows/win32/api/msinkaut15/ne-msinkaut15-inkdivisiontype) определяет типы структурных элементов, распознаваемые анализом макета.

Метод [**ресултбитипе**](/windows/desktop/api/msinkaut15/nf-msinkaut15-iinkdivisionresult-resultbytype) объекта **Дивисионресулт** Возвращает коллекцию **дивисионунитс** , содержащую все структурные элементы данного типа. Отдельный структурный элемент представлен объектом **дивисионунит** . Каждый раз при вызове метода **ресултбитипе** объект **Дивисионресулт** создает коллекцию **дивисионунитс** . Дополнительные сведения об объекте **дивисионунит** см. в разделе [Работа с объектом дивисионунит](working-with-the-divisionunit-object.md).
