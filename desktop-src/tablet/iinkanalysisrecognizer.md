# <a name="iinkanalysisrecognizer-interface"></a>Интерфейс Иинканалисисрекогнизер

Предоставляет доступ к распознавателям рукописного ввода для использования с анализом рукописного текста.

## <a name="members"></a>Элементы

Интерфейс **иинканалисисрекогнизер** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иинканалисисрекогнизер** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иинканалисисрекогнизер** содержит следующие методы.



| Метод                                                                          | Описание                                                                                                                    |
|:--------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------|
| [**Возможности**](iinkanalysisrecognizer-getcapabilities.md)               | Получает возможности распознавателя.<br/>                                                                       |
| [**GetGuid**](iinkanalysisrecognizer-getguid.md)                               | Получает глобальный уникальный идентификатор (GUID) распознавателя.<br/>                                                  |
| [**Языки**](iinkanalysisrecognizer-getlanguages.md)                     | Получает идентификаторы для языковых стандартов, которые поддерживает эта **иинканалисисрекогнизер** .<br/>                            |
| [**GetName**](iinkanalysisrecognizer-getname.md)                               | Возвращает имя распознавателя.<br/>                                                                               |
| [**жетсуппортедпропертиес**](iinkanalysisrecognizer-getsupportedproperties.md) | Получает идентификаторы GUID для свойств, которые поддерживает эта **иинканалисисрекогнизер** .<br/> |
| [**Поставщик**](iinkanalysisrecognizer-getvendor.md)                           | Возвращает имя поставщика **иинканалисисрекогнизер**.<br/>                                                        |



 

## <a name="remarks"></a>Комментарии

Распознаватель имеет определенные атрибуты и свойства, позволяющие ему выполнять распознавание. Свойства распознавателя должны быть определены до того, как будет выполнено распознавание. Типы свойств, поддерживаемых распознавателем, определяют типы распознавания, которые он может выполнять. Например, если распознаватель не поддерживает письменный рукописный ввод, он возвращает неточные результаты, когда пользователь пишет письменно.

Распознаватель также имеет встроенные функции, которые автоматически управляют многими аспектами рукописного ввода. Например, он определяет метрики линий, на которых рисуются штрихи. Можно вернуть номер строки штриха, но не нужно указывать, как эти метрики линии определяются из-за встроенной функциональности распознавателя.

[**Иинканализер**](iinkanalyzer.md) поддерживает список доступных распознавателей. Чтобы получить доступ к этому списку, используйте метод [**метода иинканализер:: жетинканалисисрекогнизерсбиприорити**](iinkanalyzer-getinkanalysisrecognizersbypriority.md) .
