# <a name="inkpicturesystemcolorschanged-event"></a>Событие InkPicture.SysТемколорсчанжед

Происходит после изменения системных цветов.

## <a name="syntax"></a>Синтаксис


```C++
void SystemColorsChanged();
```



## <a name="parameters"></a>Параметры

У этого события нет параметров.

## <a name="return-value"></a>Возвращаемое значение

Это событие не возвращает значение.

## <a name="remarks"></a>Комментарии

Этот метод события определен в интерфейсе **\_ иинкпиктуривентс** . Интерфейс **\_ иинкпиктуривентс** реализует интерфейс [**IDISPATCH**](/windows/win32/api/oaidl/nn-oaidl-idispatch) с идентификатором DISPID \_ ипесистемколорсчанжед.
