# <a name="dictionaries-and-factoids"></a>Словари и Фактоидс

Словарь — это внутренний лексикон, который распознаватель использует для определения совпадений и вариантов слов. Фактоид повышает точность распознавания, выбирая конкретный контекст для определенного поля.

Дополнительные сведения о словарях см. [в разделе Использование словарей приложений](using-application-dictionaries.md).

Дополнительные сведения о фактоидс см. [в разделе Использование контекста для повышения точности](using-context-to-improve-accuracy.md).
