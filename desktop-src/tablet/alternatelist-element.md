# <a name="alternatelist-element"></a>Алтернателист, элемент

Содержит набор [**альтернативных**](alternate-element.md) элементов, которые содержат альтернативные значения, возвращаемые распознавателем для родительского элемента [**инкворд**](inkword-element.md) . Альтернативные варианты упорядочены по достоверности распознавателя в альтернативном порядке.

## <a name="definition"></a>Определение

``` syntax
<xs:element name="AlternateList" type="AlternatesListType" />
```

## <a name="parent-elements"></a>Родительские элементы

[**инкворд**](inkword-element.md)

## <a name="child-elements"></a>Дочерние элементы

[**Альтернативный**](alternate-element.md)

## <a name="attributes"></a>Атрибуты

None
