# <a name="ianalysisalternatesgetcount-method"></a>Метод Ианалисисалтернатес:: NOCOUNT

Возвращает количество объектов [**ианалисисалтернате**](ianalysisalternate.md) , содержащихся в коллекции [**ианалисисалтернатес**](ianalysisalternates.md) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetCount(
  [out] ULONG *pulCount
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пулкаунт* \[ заполняет\]
</dt> <dd>

32-разрядное целое число без знака, равное количеству объектов [**ианалисисалтернате**](ianalysisalternate.md) , содержащихся в коллекции [**ианалисисалтернатес**](ianalysisalternates.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Описание возвращаемых значений см. в разделе [классы и интерфейсы — анализ рукописного ввода](classes-and-interfaces---ink-analysis.md).
