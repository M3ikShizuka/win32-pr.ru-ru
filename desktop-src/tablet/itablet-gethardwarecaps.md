# <a name="itabletgethardwarecaps-method"></a>Метод Итаблет:: Жесардварекапс

Возвращает значение, представляющее возможности устройства планшета.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetHardwareCaps(
  [out] DWORD *pdwCaps
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пдвкапс* \[ заполняет\]
</dt> <dd>

Битовые флаги, представляющие возможности планшетного оборудования.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Код возврата                                                                            | Описание                               |
|----------------------------------------------------------------------------------------|-------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>   | Успешно.<br/>                       |
| <dl> <dt>**\_Ошибка E**</dt> </dl> | Произошла неизвестная ошибка.<br/> |



 

## <a name="remarks"></a>Remarks

Параметр *пдвкапс* представляет собой набор битовых флагов, описывающих возможности планшетного оборудования. Эти флаги описаны в следующей таблице.



| Имя флага                         | Значение                 | Описание                                                                                                                    |
|-----------------------------------|-----------------------|--------------------------------------------------------------------------------------------------------------------------------|
| \_Встроенная СВК<br/>       | 0x00000001<br/> | Указывает, что экран и дигитайзер имеют одну и ту же поверхность.<br/>                                                    |
| СВК \_ CSR \_ должен \_ касаться<br/> | 0x00000002<br/> | Указывает, что курсор должен находиться в физическом контакте с устройством для отчета о положении.<br/>                           |
| СВК \_ жесткое \_ сходство<br/>  | 0x00000004<br/> | Указывает, что устройство может создавать события при вводе курсора и открывая диапазон физического обнаружения.<br/> |
| \_представители СВК фисид \_<br/>     | 0x00000008<br/> | Указывает, что устройство может однозначно идентифицировать активный курсор в оборудовании.<br/>                                      |
