# <a name="text-ink-reference"></a>Справочник по рукописному вводу текста

Кроме поддержки стандартных интерфейсов OLE, объект рукописного ввода текста (Тинк) также поддерживает [**иинклинеинфо**](/windows/desktop/api/msinkaut/nn-msinkaut-iinklineinfo).

Сведения об указании свойств дисплея для объекта рукописного ввода текста см. в разделе Структура [**инкметрик**](/windows/win32/api/msinkaut/ns-msinkaut-inkmetric) .
