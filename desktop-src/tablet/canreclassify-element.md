# <a name="canreclassify-element"></a>Канреклассифи, элемент

**Значение true** указывает, что рукописный ввод можно повторно проанализировать. Этот элемент имеет **значение false** , если пользователь явно установил тип рукописного ввода, используя преобразование рукописного текста в текст или изменение формы в функции журнала.

## <a name="definition"></a>Определение

``` syntax
<xs:element name="CanReclassify" type="xs:boolean" />
```

## <a name="parent-elements"></a>Родительские элементы

[**Рисование**](drawing-element.md)

## <a name="child-elements"></a>Дочерние элементы

Отсутствует.

## <a name="attributes"></a>Атрибуты

Отсутствует.
