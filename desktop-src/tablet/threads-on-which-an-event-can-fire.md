-   [События объекта InkCollector](inkcollector-object-events.md)
-   [События объекта InkOverlay](inkoverlay-object-events.md)
-   [События объекта Рекогнизерконтекст](recognizercontext-object-events.md)
-   [События коллекции штрихов](strokes-collection-events.md)
-   [События элемента управления InkEdit](inkedit-control-events.md)
-   [События элемента управления InkPicture](inkpicture-control-events.md)
-   [События объекта Пенинпутпанел](peninputpanel-object-events.md)
