# <a name="alternates"></a>Варианты

Альтернативой является возможное совпадение слов для сегмента распознавания. Распознаватель создает альтернативы и порождает их на допустимые совпадения рукописного ввода или звука с словарем или фактоид.

> [!Note]  
> Оценка достоверности в настоящее время доступна только для распознавателей Microsoft English (США) и жестов.

 

Иногда рукописные данные могут иметь неоднозначные различия между сегментами. Распознаватель сравнивает эти сегменты с словарем распознавателя, чтобы определить возможные совпадения. Когда распознаватель сравнивает сегменты, он сохраняет список возможных альтернативных совпадений и назначает уровень достоверности каждому из них, выбрав верхний вариант.

> [!Note]  
> Распознаватель не может предоставить варианты для фрагмента рукописного ввода, который меньше, чем сегмент распознавания.
