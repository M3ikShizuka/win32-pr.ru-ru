# <a name="references-tablet-pc"></a>Ссылки (планшетный ПК)

Сведения о включении распознавания речи в приложении см. в пакете Microsoft Speech SDK, который можно найти в [пакете Microsoft Speech SDK](https://www.microsoft.com/speech/download/sdk51/). Этот сайт также содержит общие сведения о программе Microsoft Speech API и рекомендациях по проектированию приложений.

Сведения о специальных возможностях Майкрософт см. в разделе [Microsoft Accessibility](https://www.microsoft.com/enable/).

Дополнительные сведения о рекомендациях Microsoft Active Accessibility для приложений на основе Windows см. в разделе [Специальные возможности](../accessibility/accessibility.md) .
