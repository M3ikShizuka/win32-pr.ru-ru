# <a name="flicks-constants"></a>Константы жестов

Ниже приведены константы жестов.



| Константа/значение                                                                                                                                                                                                                                   | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="FLICK_WM_HANDLED_MASK"></span><span id="flick_wm_handled_mask"></span><dl> <dt>**Жест \_ \_Обработанные по \_ маске маски**</dt> <dt>0x1</dt> </dl> | Значение, возвращаемое после обработки сообщения [**с \_ \_ сообщением о жесте планшета WM**](wm-tablet-flick-message.md) . Если **возвращается \_ \_ \_ Маска обработки жестов WM** , дальнейшие действия не выполняются. В противном случае Windows отправляет дальнейшие уведомления, такие как [**WM \_ аппкомманд**](/windows/desktop/inputdev/wm-appcommand), [**WM \_ VSCROLL**](/windows/desktop/Controls/wm-vscroll)или [**WM \_ KeyDown**](/windows/desktop/inputdev/wm-keydown), в зависимости от того, какое действие связано с пером. <br/> |
| <span id="NUM_FLICK_DIRECTIONS"></span><span id="num_flick_directions"></span><dl> <dt>**Число \_ \_Направления надвижений**</dt> <dt>8</dt> </dl>       | Число направлений, определенных в перечислении [**фликкдиректион**](/windows/desktop/api/tabflicks/ne-tabflicks-flickdirection) .<br/>                                                                                                                                                                                                                                                                                                                                                              |
