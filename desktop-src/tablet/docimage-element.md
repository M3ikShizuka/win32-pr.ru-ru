# <a name="docimage-element"></a>ДоЦимаже, элемент

Если прочитанная заметка журнала создана с помощью драйвера принтера составителя заметок журнала, то в этом элементе сохраняется фоновое изображение документа.

## <a name="definition"></a>Определение

``` syntax
<xs:element name="DocImage" type="xs:base64Binary" minOccurs="0" />
```

## <a name="parent-elements"></a>Родительские элементы

[**жаурналпаже**](journalpage-element.md)

## <a name="child-elements"></a>Дочерние элементы

Отсутствует.

## <a name="attributes"></a>Атрибуты

Отсутствует.

## <a name="element-information"></a>Сведения об элементе



|              |                                            |
|--------------|--------------------------------------------|
| Тип элемента | **xs:base64Binary**                        |
| Пространство имен    | urn: schemas-microsoft-com: TabletPC: ричинк |
| Имя схемы  | Средство чтения журнала                             |
