# <a name="inkanalyzer-class"></a>Класс InkAnalyzer

Реализует интерфейс [**иинканализер**](iinkanalyzer.md) .

## <a name="remarks"></a>Remarks

Этот класс реализует COM-интерфейс [**иинканализер**](iinkanalyzer.md) .

[ \_ Ианалисисевентс](-ianalysisevents.md) является источником событий по умолчанию и предоставляет стандартные события для [**иинканализер**](iinkanalyzer.md).

[**\_ Ианалисиспроксевентс**](-ianalysisproxyevents.md) предоставляет события прокси-сервера данных для [**иинканализер**](iinkanalyzer.md). Дополнительные сведения см. [в разделе Учетная запись-посредник данных с помощью анализа рукописного ввода](data-proxy-with-ink-analysis.md).
