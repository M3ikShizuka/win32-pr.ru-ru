# <a name="factoid-support"></a>Поддержка фактоид

*Фактоидс* улучшить точность распознавания, предоставляя распознавателю определение и порядок слов, чисел и знаков препинания, которые ожидались.

Дополнительные сведения о фактоидс см. [в разделе Использование контекста для повышения точности](using-context-to-improve-accuracy.md).
