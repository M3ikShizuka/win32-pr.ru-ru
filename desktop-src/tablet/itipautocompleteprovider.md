# <a name="itipautocompleteprovider-interface"></a>Интерфейс Итипаутокомплетепровидер

Управляет стороны приложения для интеграции автоматического завершения автоматической сборки панели ввода текста.

## <a name="members"></a>Элементы

Интерфейс **итипаутокомплетепровидер** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Итипаутокомплетепровидер** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **итипаутокомплетепровидер** содержит следующие методы.



| Метод                                                                  | Описание                                                                                                          |
|:------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------|
| [**Казывающи**](itipautocompleteprovider-show.md)                           | Отображает или скрывает список автозавершения.<br/>                                                                 |
| [**упдатепендингтекст**](itipautocompleteprovider-updatependingtext.md) | Используется клиентом автоматического завершения для уведомления приложения о тексте, которое пользователь направит на панель ввода.<br/> |
