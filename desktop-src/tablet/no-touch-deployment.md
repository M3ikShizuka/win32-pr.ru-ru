# <a name="no-touch-deployment"></a>Без сенсорного развертывания

Microsoft .NET Framework дает возможность развертывать приложения с веб-сервера или с файловым сервером. Эта методика, называемая «несенсорным развертыванием», сочетает в себе производительность и интерактивность приложения смарт-клиента со всеми преимуществами развертывания веб-приложения. Чтобы развернуть приложение через Интернет, щелкните правой кнопкой мыши папку, содержащую исполняемый файл, и выберите пункт **Свойства**. На вкладке « **общий доступ в Интернете** » выберите **Открыть общий доступ к этой папке**. Введите псевдоним для папки. Теперь исполняемый файл можно запустить через Интернет, используя этот псевдоним в качестве каталога на сервере. Дополнительные сведения о развертывании без вмешательства пользователя см. в разделе [интеллектуальные клиенты .NET](/archive/msdn-magazine/2002/july/net-zero-deployment-security-and-versioning-models-in-the-windows-forms-engine-help-you-create-and-deploy-smart-clients) и [развертывание без вмешательства в платформа .NET Framework](/documentation/?url=%2flibrary%2fdv_vstechart%2fhtml%2fvbtchno-touchdeploymentinnetframework.asp).

> [!Note]  
> На компьютере должен быть установлен Microsoft службы IIS (IIS), чтобы включить вкладку " **доступ к веб-** приложению" в диалоговом окне " **свойства** ".

 

Несенсорное развертывание применяется к приложениям с поддержкой рукописного ввода так же, как и любому другому приложению Windows Forms. Примеры, предоставляемые пакетом средств разработки программного обеспечения (SDK) для Microsoft Windows XP Tablet PC Edition, включают версию развертывания без участия пользователя в примере "автоматические утверждения" в разделе Образцы веб-страниц образцов. В этом примере используется [образец автоматической формы утверждений](auto-claims-form-sample.md) и предоставляется установщик, который помещает в веб-общую папку. При переходе Microsoft Internet Explorer на AutoClaims.exe в каталоге, сопоставленном с веб-папкой, приложение запускается. Дополнительные сведения о примере см. в разделе [веб-пример Несенсорного развертывания](no-touch-deployment-web-sample.md) .

> [!Note]  
> При развертывании приложения .NET через Интернет это может повлиять на приложение с помощью модели безопасности .NET. В разделе [безопасность и доверие](security-and-trust.md) описано, как безопасность работает с веб-приложениями с поддержкой рукописного ввода.
