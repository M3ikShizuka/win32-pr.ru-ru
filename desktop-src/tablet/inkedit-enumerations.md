# <a name="inkedit-enumerations"></a>Перечисления InkEdit

Этот раздел содержит перечисления, принадлежащие элементу управления InkEdit.



| Перечисление                                            | Описание                                                                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**аппеаранцеконстантс**](/windows/desktop/api/inked/ne-inked-appearanceconstants)     | Указывает, отображается ли элемент управления плоским или трехмерным.<br/>                                                                                            |
| [**бордерстилеконстантс**](/windows/desktop/api/inked/ne-inked-borderstyleconstants)   | Указывает, имеет ли элемент управления границу.<br/>                                                                                                   |
| [**инкаппликатионжестуре**](/windows/desktop/api/msinkaut/ne-msinkaut-inkapplicationgesture) | Указывает интересующий набор жестов конкретного приложения.<br/>                                                                             |
| [**инкдисплаймоде**](/windows/desktop/api/inked/ne-inked-inkdisplaymode)               | Указывает, отображается ли выделенный фрагмент как рукописный или текстовый.<br/>                                                                                         |
| [**инкедитстатус**](/windows/desktop/api/inked/ne-inked-inkeditstatus)                 | Указывает, простаивает ли элемент управления [InkEdit](inkedit-control-reference.md) , собирает рукописный ввод или распознает рукописный ввод.<br/>                           |
| [**инкинсертмоде**](/windows/desktop/api/inked/ne-inked-inkinsertmode)                 | Указывает, как рукописный ввод вставляется в элемент управления [InkEdit](inkedit-control-reference.md) .<br/>                                                      |
| [**инкмоде**](/windows/desktop/api/inked/ne-inked-inkmode)                             | Задает параметры режима сбора для рисуемого рукописного ввода: отключена ли коллекция рукописных данных, выполняется сбор рукописных данных или выполняется сбор рукописных данных и жестов.<br/> |
| [**инкмаусебуттон**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousebutton)               | Указывает, какая кнопка мыши была нажата.<br/>                                                                                                     |
| [**инкмаусепоинтер**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousepointer)             | Задает тип указателя мыши, который отображается.<br/>                                                                                             |
| [**маусебуттон**](/windows/desktop/api/inked/ne-inked-mousebutton)                     | Указывает, какая кнопка мыши была нажата.<br/>                                                                                                     |
| [**скроллбарсконстантс**](/windows/desktop/api/inked/ne-inked-scrollbarsconstants)     | Указывает, как полосы прокрутки элемента управления [InkEdit](inkedit-control-reference.md) появляются на экране.<br/>                                    |
| [**селалигнментконстантс**](/windows/desktop/api/inked/ne-inked-selalignmentconstants) | Задает выравнивание абзаца относительно полей элемента управления [InkEdit](inkedit-control-reference.md) .<br/>                     |
