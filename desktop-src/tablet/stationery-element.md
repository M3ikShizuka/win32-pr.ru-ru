# <a name="stationery-element"></a>Элемент бланка

Содержит сведения о бланке, используемом в заметке журнала, включая цвет фона или изображение, сведения о заголовке и сведения о макете строки, такие как узкие правила и правила полей.

## <a name="definition"></a>Определение

``` syntax
<xs:element name="Stationery" type="StationeryType" minOccurs="0" maxOccurs="unbounded" />
```
