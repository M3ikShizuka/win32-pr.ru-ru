# <a name="recognizer-stroke-reordering-support"></a>Поддержка переупорядочивания перьев распознавателя

Распознаватели Майкрософт для латиницы поддерживают изменение порядка штрихов.

Распознаватели рукописного ввода Майкрософт для латиницы не зависят от штрихов в строке, написанной в определенном порядке. Вместо этого распознаватели упорядочивают внутренние штрихи на основе пространственных отношений между штрихами. Например, распознаватели латиницы поддерживают общий сценарий, в котором пользователь возвращается в строку для записи левой скобки, "(" или для добавления кавычек.
