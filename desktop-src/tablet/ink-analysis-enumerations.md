# <a name="ink-analysis-enumerations"></a>Перечисления анализа рукописного ввода

В этом разделе содержатся сведения о перечислениях, используемых при анализе рукописного ввода.

## <a name="enumerations"></a>Перечисления



| Перечисление                                              | Описание                                                                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**аналисисмодес**](analysismodes.md)                   | Указывает, как [**иинканализер**](iinkanalyzer.md) выполняет анализ рукописного ввода.<br/>                                                  |
| [**аналисисварнингкоде**](analysiswarningcode.md)       | Задает набор доступных предупреждений, которые могут возникнуть во время анализа рукописного ввода.<br/>                                                    |
| [**конфирматионтипе**](confirmationtype.md)             | Указывает тип подтверждения, которое может произойти для объекта [**иконтекстноде**](icontextnode.md) .<br/>                           |
| [**контекстлинкдиректион**](contextlinkdirection.md)     | Указывает типы объектов [**иконтекстлинк**](icontextlink.md) .<br/>                                                           |
| [**рекогнитионконфиденце**](recognitionconfidence.md)   | Указывает уровень достоверности, в котором [**иинканализер**](iinkanalyzer.md) имеет точность результата распознавания.<br/> |
| [**рекогнизеркапабилитиес**](recognizercapabilities.md) | Задает атрибуты [**иинканалисисрекогнизер**](iinkanalysisrecognizer.md).<br/>                                       |
| [**строкетипе**](stroketype.md)                         | Указывает, как [**иинканализер**](iinkanalyzer.md) должен классифицировать штрих во время анализа.<br/>                               |
