# <a name="creating-a-recognizer"></a>Создание распознавателя

Если вы решили это сделать, приложение может предоставить собственные Распознаватели. В этом разделе описываются требования к созданию пользовательского распознавателя приложений для приложения, которое можно использовать с API платформы Tablet PC.

> [!Note]  
> нестандартные распознаватели приложений не используются на панели ввода Tablet PC или в журнале Microsoft Windows. Эти аксессуары используют только распознаватели, с которыми они были протестированы.
