# <a name="baseline-element"></a>Базовый элемент

Предоставляет сведения (x, y) для начальной и конечной точек базового плана абзаца в документе журнала. Координатное пространство, используемое для этих значений, — **HIMETRIC**.

## <a name="definition"></a>Определение

``` syntax
<xs:element name="Baseline" type="BaseLineType" />
```

## <a name="parent-elements"></a>Родительские элементы

[**параграфлинеметрикс**](paragraphlinemetrics-element.md)

## <a name="child-elements"></a>Дочерние элементы

Отсутствует.

## <a name="attributes"></a>Атрибуты



| attribute  | Тип                      | Обязательно | Описание                                                      | Возможные значения           |
|------------|---------------------------|----------|------------------------------------------------------------------|---------------------------|
| **старткс** | **xs:nonNegativeInteger** | Обязательно | Значение X для точки, помечающая начало базовых показателей. | Любое неотрицательное целое число. |
| **Запуск** | **xs:nonNegativeInteger** | Обязательно | Значение Y для точки, помечающая начало базовых показателей. | Любое неотрицательное целое число. |
| **ендкс**   | **xs:nonNegativeInteger** | Обязательно | Значение X для точки, помечающая конец базового плана.       | Любое неотрицательное целое число. |
