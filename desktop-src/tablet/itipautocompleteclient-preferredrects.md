# <a name="itipautocompleteclientpreferredrects-method"></a>Итипаутокомплетеклиент: метод:P Реферредректс

Позволяет клиенту предлагать место расположения списка автозавершения, чтобы избежать перекрытия панели ввода.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT PreferredRects(
  [in]      RECT *prcACList,
  [in]      RECT *prcField,
  [out]     RECT *prcModified,
  [in, out] BOOL *pfShownAboveTip
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пркаклист* \[ окне\]
</dt> <dd>

Прямоугольник в экранных координатах, указывающий предпочтительное расположение поставщика и размер пользовательского интерфейса списка автозавершения.

</dd> <dt>

*пркфиелд* \[ окне\]
</dt> <dd>

Прямоугольник в экранных координатах, указывающий расположение и размер поля с сортировкой.

</dd> <dt>

*пркмодифиед* \[ заполняет\]
</dt> <dd>

Прямоугольник, основанный на текущем состоянии Совета и расположении предпочтительного списка автозавершения и размера, заданного параметром *пркаклист*.

</dd> <dt>

*пфшовнабоветип* \[ в, out\]
</dt> <dd>

**Значение true** , если измененный прямоугольник должен отображаться над целевой областью панели ввода текста; в противном случае — **значение false**. Перед вызовом метода это значение должно быть инициализировано до предпочтительной ориентации поставщика.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Код возврата                                                                                  | Описание                                                                                                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Успешно.<br/>                                                                                                                                                                                                                                                            |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Вызовите [**метод итипаутокомплетеклиент:: рекуестшовуи**](itipautocompleteclient-requestshowui.md) , чтобы задать окно списка автозаполнения всплывающего окна перед вызовом [**метода Итипаутокомплетеклиент::P реферредректс**](itipautocompleteclient-preferredrects.md).<br/> |
| <dl> <dt>**\_Ошибка E**</dt> </dl>       | Произошла неизвестная ошибка.<br/>                                                                                                                                                                                                                                      |



 

## <a name="remarks"></a>Remarks

Это метод, который вызывается поставщиком автоматического завершения, когда он собирается отобразить пользовательский интерфейс автоматического завершения. Клиент изменяет предпочтительный прямоугольник поставщика, указанный параметром *пркаклист* , с помощью аргумента *пркмодифиед* .

Вызовите [**метод итипаутокомплетеклиент:: рекуестшовуи**](itipautocompleteclient-requestshowui.md) , чтобы задать окно списка автозаполнения всплывающего окна перед вызовом **преферредректс**. Несоблюдение этого действия приведет к ошибке **E \_ INVALIDARG** при вызове **преферредректс**.
