# <a name="hrecowordlist-handle"></a>ХРЕКОВОРДЛИСТ, обработчик

Для управления списком слов, присоединенным к контексту распознавателя, используется обработчик **хрековордлист** . Для улучшения результатов распознавания используется список слов.


```C++
typedef HANDLE HRECOWORDLIST;
```



## <a name="remarks"></a>Комментарии

Следующая функция использует **хрековордлист**.



| Функция                                         | Описание                                         |
|--------------------------------------------------|-----------------------------------------------------|
| [**аддвордстовордлист**](/windows/desktop/api/recapis/nf-recapis-addwordstowordlist) | Добавляет одно или несколько слов в список слов.<br/> |
| [**дестройвордлист**](/windows/desktop/api/recapis/nf-recapis-destroywordlist)       | Уничтожает текущий список слов.<br/>          |
| [**макевордлист**](/windows/desktop/api/recapis/nf-recapis-makewordlist)             | Создает список слов.<br/>                     |
