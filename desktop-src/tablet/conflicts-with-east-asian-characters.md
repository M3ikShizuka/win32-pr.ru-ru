# <a name="conflicts-with-east-asian-characters"></a>Конфликты с East-Asianными символами

Некоторые *жесты приложения* могут конфликтовать с восточно-азиатскими символами или сочетаниями восточно-азиатских символов. В следующей таблице перечислены возможные конфликты между жестами приложений и символами в приложениях, которые имеют определенные знаки восточно-азиатского ввода.



| жесты                 | Китайский (упрощенное письмо) | Китайский (традиционное письмо) | Японский     | Корейский       |
|-------------------------|----------------------|-----------------------|--------------|--------------|
| Down<br/>         | X<br/>         | X<br/>          | X<br/> | X<br/> |
| Правый<br/>        | X<br/>         | X<br/>          | X<br/> | X<br/> |
| DownRight<br/>    | -<br/>         | X<br/>          | -<br/> | X<br/> |
| RightDown (вправо вниз)<br/>    | -<br/>         | -<br/>          | X<br/> | X<br/> |
| Circle<br/>       | -<br/>         | -<br/>          | -<br/> | X<br/> |
| ChevronRight<br/> | -<br/>         | -<br/>          | -<br/> | X<br/> |



 

Корпорация Майкрософт продолжает зафиксироваться в разработке *жестов*. Корпорация Майкрософт понимает, что при создании приложений независимые поставщики программного обеспечения (ISV) должны знать, какие действия будут сопоставлены с жестами. [Нереализованные глифы](unimplemented-glyphs.md) перечисляет набор действий пером, которые корпорация Майкрософт планирует сопоставлять с жестами, а также жесты, зарезервированные для действий. Эти сведения предоставляются для того, чтобы вы узнали, какие действия и жесты будут предоставлены в будущем.

В дополнение к использованию жестов, предоставляемых в Microsoft Windows XP Tablet PC Edition, приложения могут создавать собственные жесты. Эти жесты затем могут быть распознаны *распознавателем Microsoft жеста* , который создает разработчик приложения. Если вы планируете реализовать любые собственные жесты, изучите [нереализованные глифы](unimplemented-glyphs.md) , чтобы избежать перекрытия с помощью жестов, которые корпорация Майкрософт планирует реализовать.
