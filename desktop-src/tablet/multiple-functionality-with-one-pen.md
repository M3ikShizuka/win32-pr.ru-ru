# <a name="multiple-functionality-with-one-pen"></a>Несколько функциональных возможностей с одним пером

Перо Tablet PC может иметь несколько конечных сторон, которые могут взаимодействовать с дигитайзером. Если оба конца пера могут создавать события, можно использовать один конец для размещения рукописного ввода, выбора и указания, а другой — для других функций, таких как удаление.

Для реализации таких функций приложение должно иметь возможность определить, какой конец пера используется, и, соответственно, изменить внешний вид курсора. Это можно сделать путем поиска состояния [**инвертированного**](/windows/desktop/api/msinkaut/nf-msinkaut-iinkcursor-get_inverted) свойства в объекте [**курсора**](/windows/desktop/api/msinkaut/nn-msinkaut-iinkcursor) .

Конкретные сведения об использовании задней части пера для стирания см. в разделе [стирание рукописного ввода с помощью пера](erasing-ink-with-the-pen.md). Кроме того, [Пример стирания рукописного ввода](ink-erasing-sample.md) демонстрирует, как использовать заднюю часть пера для стирания рукописного ввода.
