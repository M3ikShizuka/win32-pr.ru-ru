# <a name="math-input-classes-and-interfaces"></a>Классы и интерфейсы для ввода математических символов

В этом разделе содержатся классы и интерфейсы, используемые для элемента управления математическими данными.

## <a name="in-this-section"></a>в этом разделе



| Интерфейс                                                     | Описание                                                       |
|---------------------------------------------------------------|-------------------------------------------------------------------|
| [**\_имасинпутконтролевентс**](/windows/win32/api/micaut/nn-micaut-_imathinputcontrolevents) | Предоставляет обработчики событий математического элемента управления вводом.                    |
| [**имасинпутконтрол**](/windows/desktop/api/micaut/nn-micaut-imathinputcontrol)                | Предоставляет методы, которые преобразуют рукописный ввод в интерпретируемые математические выходные данные. |
