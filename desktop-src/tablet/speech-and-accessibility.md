# <a name="speech-and-accessibility"></a>Речь и специальные возможности

Чтобы приложения были доступны, сведения о содержимом экрана должны быть предоставлены с помощью служебных программ распознавания речи. Этим программам необходимо определить и присвоить определенные элементы управления, чтобы они могли распознать именованное имя элемента управления и манипулировать этим элементом управления в ответ на голосовые команды.

Например, при использовании служебной программы голосового ввода пользователь может сказать «File» (файл), затем сказать «параметры страницы», чтобы выбрать соответствующий пункт меню «команда», а затем «Альбомная» для выбора соответствующего флажка. Чтобы выполнить нужную цель, необходимо предоставить сведения об этих элементах.
