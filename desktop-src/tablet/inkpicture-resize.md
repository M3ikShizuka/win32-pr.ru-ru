# <a name="inkpictureresize-event"></a>Событие InkPicture. Resize

Происходит, когда изменяется размер элемента управления [InkPicture](inkpicture-control-reference.md) (при изменении значений свойств Width и/или Height).

## <a name="syntax"></a>Синтаксис


```C++
void Resize(
   long Left,
   long Top,
   long Right,
   long Bottom
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Слева* 
</dt> <dd>

Число пикселей от левой стороны окна, которое содержит элемент управления слева от элемента управления.

</dd> <dt>

*Top* 
</dt> <dd>

Число пикселей от верхнего края окна, которое содержит элемент управления в верхней части элемента управления.

</dd> <dt>

*Right* 
</dt> <dd>

Число пикселей от левой стороны окна, которое содержит элемент управления справа от элемента управления.

</dd> <dt>

*Нижнее* 
</dt> <dd>

Число пикселей от верхнего края окна, которое содержит элемент управления в нижней части элемента управления.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Это событие не возвращает значение.

## <a name="remarks"></a>Комментарии

Новая ширина элемента управления в пикселях будет равна разности между параметрами *right* и *Left* . Аналогичным образом Новая высота будет равна разности между *нижней* и *верхней границами*.

Этот метод события определен в интерфейсе **\_ иинкпиктуривентс** . Интерфейс **\_ иинкпиктуривентс** реализует интерфейс IDispatch с идентификатором **DISPID \_ ипересизе**.
