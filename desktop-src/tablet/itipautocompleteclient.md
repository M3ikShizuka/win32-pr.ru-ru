# <a name="itipautocompleteclient-interface"></a>Интерфейс Итипаутокомплетеклиент

Позволяет клиенту вызывать объект поставщика автоматического завершения для панели ввода текста приложения.

## <a name="members"></a>Элементы

Интерфейс **итипаутокомплетеклиент** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Итипаутокомплетеклиент** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **итипаутокомплетеклиент** содержит следующие методы.



| Метод                                                              | Описание                                                                                                                                                          |
|:--------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**адвисепровидер**](itipautocompleteclient-adviseprovider.md)     | Регистрирует поставщик в клиенте, чтобы позволить клиенту вызывать объект поставщика автозавершения приложения.<br/>                                      |
| [**преферредректс**](itipautocompleteclient-preferredrects.md)     | Позволяет клиенту предлагать место расположения списка автозавершения, чтобы избежать перекрытия панели ввода.<br/>                                               |
| [**рекуестшовуи**](itipautocompleteclient-requestshowui.md)       | Определяет, готова ли панель ввода к отображению списка автозавершения.<br/>                                                                             |
| [**унадвисепровидер**](itipautocompleteclient-unadviseprovider.md) | Отменяет регистрацию связанного поставщика.<br/>                                                                                                                      |
| [**усерселектион**](itipautocompleteclient-userselection.md)       | Уведомляет панель ввода о том, что пользователь выбрал что-либо в списке автозавершения, и отбросить оставшийся текст, который еще не был вставлен.<br/> |
