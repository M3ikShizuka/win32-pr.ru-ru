# <a name="strokebuilder-class"></a>Класс Строкебуилдер

Реализует [**интерфейс истрокебуилдер**](/windows/desktop/api/RTSCom/nn-rtscom-istrokebuilder).

## <a name="com-implementation"></a>Реализация COM

Этот класс реализует [**интерфейс истрокебуилдер**](/windows/desktop/api/RTSCom/nn-rtscom-istrokebuilder).
