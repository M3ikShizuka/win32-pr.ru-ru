# <a name="desktop-window-manager"></a>Диспетчер окон рабочего стола

Функция композиции рабочего стола, появившаяся в Windows Vista, существенно изменила способ отображения пикселов в приложениях на экране. Если композиция рабочего стола включена, отдельные окна больше не наводятся непосредственно на экран или на основное устройство отображения, как в предыдущих версиях Windows. Вместо этого их Рисование перенаправляется на экранные поверхности в видеопамяти, которые затем подготавливаются к просмотру в виде изображения на рабочем столе и отображаются на экране.

Композиция рабочего стола выполняется диспетчер окон рабочего стола (DWM). С помощью композиции рабочего стола DWM обеспечивает визуальные эффекты на настольном компьютере, а также различные функции, такие как стеклянные фреймы окна, трехмерные анимации перемещения окон, Windows Flip3D и поддержка высокого разрешения.

Диспетчер окон рабочего стола выполняется как служба Windows. Его можно включить и отключить с помощью элемента панели управления "Администрирование" в разделе "службы" как диспетчер окон рабочего стола диспетчер сеансов.

Многие функции DWM могут управляться приложением или обращаться к ним через API DWM. В следующей документации описываются функции и требования интерфейсов API DWM.
