# <a name="security-functions"></a>Функции безопасности

Функции безопасности управления сетью получают и устанавливают дескрипторы безопасности для изолированных корней DFS на основе домена и автономных расположений.

Ниже перечислены функции безопасности DFS.

| Функция                                                               | Описание                                                                                                          |
|------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**нетдфсжетсекурити**](/windows/desktop/api/lmdfs/nf-lmdfs-netdfsgetsecurity)                         | Получает дескриптор безопасности для корневого объекта указанного корня DFS.                                       |
| [**нетдфссетсекурити**](/windows/desktop/api/lmdfs/nf-lmdfs-netdfssetsecurity)                         | Находит корневой объект для указанного корня DFS и устанавливает для него указанный дескриптор безопасности.                  |
| [**нетдфсжетстдконтаинерсекурити**](/windows/desktop/api/lmdfs/nf-lmdfs-netdfssetftcontainersecurity) | Получает дескриптор безопасности для объекта контейнера указанного изолированного корневого корня DFS.                      |
| [**нетдфссетстдконтаинерсекурити**](/windows/desktop/api/lmdfs/nf-lmdfs-netdfssetstdcontainersecurity) | Находит объект контейнера для указанного автономного корня DFS и устанавливает для него указанный дескриптор безопасности. |
| [**нетдфсжетфтконтаинерсекурити**](/windows/desktop/api/lmdfs/nf-lmdfs-netdfsgetftcontainersecurity)   | Получает дескриптор безопасности для объекта контейнера указанного корня домена DFS.                           |
| [**нетдфссетфтконтаинерсекурити**](/windows/desktop/api/lmdfs/nf-lmdfs-netdfssetftcontainersecurity)   | Находит объект контейнера для указанного корня домена DFS и задает для него указанный дескриптор безопасности.      |
