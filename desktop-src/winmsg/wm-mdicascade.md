# <a name="wm_mdicascade-message"></a>\_Сообщение МДИКАСКАДЕ WM

Приложение отправляет сообщение **WM \_ мдикаскаде** в окно клиента многодокументного интерфейса (MDI) для упорядочения всех его дочерних окон в каскадном формате.


```C++
#define WM_MDICASCADE                   0x0227
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Поведение Cascade. Этот параметр может принимать одно или несколько следующих значений.



| Значение                                                                                                                                                                                                                                          | Значение                                                              |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------|
| <span id="MDITILE_SKIPDISABLED"></span><span id="mditile_skipdisabled"></span><dl> <dt>**Мдитиле \_ СКИПДИСАБЛЕД**</dt> <dt>0x0002</dt> </dl> | Запрещает каскадное отображение отключенных дочерних окон MDI. <br/> |
| <span id="MDITILE_ZORDER"></span><span id="mditile_zorder"></span><dl> <dt>**Мдитиле \_ ЗОРДЕР**</dt> <dt>0x0004</dt> </dl>                   | Упорядочивает окна в Z порядке.<br/>                          |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **bool** .

Если сообщение прошло удачно, возвращается значение **true**.

Если сообщение не выполняется, возвращается значение **false**.
