# <a name="wm_sizing-message"></a>\_Сообщение о размере WM

Отправляется в окно, для которого пользователь изменяет размер. Обрабатывая это сообщение, приложение может отслеживать размер и расположение прямоугольника перетаскивания и, при необходимости, изменять его размер или расположение.

Окно получает это сообщение через функцию [**WindowProc**](/previous-versions/windows/desktop/legacy/ms633573(v=vs.85)) .


```C++
#define WM_SIZING                       0x0214
```

## <a name="return-value"></a>Возвращаемое значение

Тип: **lResult**

Приложение должно возвращать **значение true** , если обрабатывает это сообщение.
