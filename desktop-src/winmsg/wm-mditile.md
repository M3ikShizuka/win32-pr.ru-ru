# <a name="wm_mditile-message"></a>\_Сообщение МДИТИЛЕ WM

Приложение отправляет сообщение **WM \_ мдитиле** в окно клиента многодокументного интерфейса (MDI) для упорядочения всех своих дочерних окон MDI в формате мозаики.


```C++
#define WM_MDITILE                      0x0226
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Параметр мозаичного заполнения. Этот параметр может принимать одно из следующих значений, при необходимости сочетающийся с **мдитиле \_ скипдисаблед** , чтобы отключить мозаичное заполнение отключенных дочерних окон MDI.



| Значение                                                                                                                                                                                                                                    | Значение                                |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------|
| <span id="MDITILE_HORIZONTAL"></span><span id="mditile_horizontal"></span><dl> <dt>**Мдитиле \_ ГОРИЗОНТАЛЬное**</dt> <dt>0x0001</dt> </dl> | Окна мозаики по горизонтали.<br/> |
| <span id="MDITILE_VERTICAL"></span><span id="mditile_vertical"></span><dl> <dt>**Мдитиле \_ ВЕРТИКАЛЬный**</dt> <dt>Символ 0x0000</dt> </dl>       | Окна мозаики по вертикали.<br/>   |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **bool** .

Если сообщение прошло удачно, возвращается значение **true**.

Если сообщение не выполняется, возвращается значение **false**.
