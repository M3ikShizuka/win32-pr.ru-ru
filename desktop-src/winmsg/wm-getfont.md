# <a name="wm_getfont-message"></a>Сообщение WM. \_ Font

Извлекает шрифт, с которым элемент управления в данный момент рисует свой текст.


```C++
#define WM_GETFONT                      0x0031
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется и должен быть равен нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется и должен быть равен нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **хфонт**

Возвращаемое значение является маркером шрифта, используемого элементом управления, или **значением NULL** , если элемент управления использует системный шрифт.
