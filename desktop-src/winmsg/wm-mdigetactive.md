# <a name="wm_mdigetactive-message"></a>\_Сообщение МДИЖЕТАКТИВЕ WM

Приложение отправляет сообщение **WM \_ мдижетактиве** в окно клиента многодокументного интерфейса (MDI) для получения маркера в активное дочернее окно MDI.


```C++
#define WM_MDIGETACTIVE                  0x0229
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется.

</dd> <dt>

*lParam* 
</dt> <dd>

Развернутое состояние. Если этот параметр не равен **null**, это указатель на значение, указывающее развернутое состояние дочернего окна MDI. Если значение равно **true**, окно разворачивается. значение **false** указывает, что это не так. Если этот параметр имеет **значение NULL**, параметр игнорируется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HWND**

Возвращаемое значение — это маркер активного дочернего окна MDI.
