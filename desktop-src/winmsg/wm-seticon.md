# <a name="wm_seticon-message"></a>\_Сообщение СЕТИКОН WM

Связывает новый большой или маленький значок с окном. Система отображает крупный значок в диалоговом окне ALT + TAB, а маленький значок в заголовке окна.


```C++
#define WM_SETICON                      0x0080
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Тип устанавливаемого значка. Этот параметр может принимать одно из указанных ниже значений.



| Значение                                                                                                                                                                                                       | Значение                                       |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------|
| <span id="ICON_BIG"></span><span id="icon_big"></span><dl> <dt>**Значок \_ БОЛЬШОЙ**</dt> <dt>1</dt> </dl>       | Задайте крупный значок для окна.<br/> |
| <span id="ICON_SMALL"></span><span id="icon_small"></span><dl> <dt>**Значок \_ Малый**</dt> <dt>0</dt> </dl> | Задайте маленький значок для окна.<br/> |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Маркер нового большого или маленького значка. Если этот параметр имеет **значение NULL**, значок, указанный параметром *wParam*, удаляется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **lResult**

Возвращаемое значение — это обработчик предыдущего большого или маленького значка в зависимости от значения *wParam*. Имеет **значение NULL** , если в окне ранее не было значка типа, указанного параметром *wParam*.

## <a name="remarks"></a>Комментарии

Функция [**дефвиндовпрок**](/windows/desktop/api/winuser/nf-winuser-defwindowproca) возвращает маркер предыдущего большого или маленького значка, связанного с окном, в зависимости от значения *wParam*.
