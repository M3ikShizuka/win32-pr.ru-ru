# <a name="wm_queryopen-message"></a>\_Сообщение КУЕРЙОПЕН WM

Отправляется на значок, когда пользователь запрашивает восстановление окна до его предыдущего размера и расположения.

Окно получает это сообщение через функцию [**WindowProc**](/previous-versions/windows/desktop/legacy/ms633573(v=vs.85)) .


```C++
#define WM_QUERYOPEN                    0x0013
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **lResult**

Если значок можно открыть, приложение, обрабатывающее это сообщение, должно возвращать **значение true**; в противном случае возвращается **значение false** , чтобы предотвратить открытие значка.

## <a name="remarks"></a>Комментарии

По умолчанию функция [**дефвиндовпрок**](/windows/desktop/api/winuser/nf-winuser-defwindowproca) возвращает **значение true**.

При обработке этого сообщения приложение не должно выполнять никаких действий, вызывающих изменение активации или фокуса (например, создание диалогового окна).
