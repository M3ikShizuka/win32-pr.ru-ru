# <a name="window-styles"></a>Стили окна

Если параметр **WS \_ клипсиблингс** не указан и дочерние окна перекрываются, то при рисовании в клиентской области дочернего окна можно рисовать в клиентской области соседнего дочернего окна.<br/>                                                                                                                                                                                |

Чтобы изменить стиль после создания окна, используйте функцию [**SetWindowLong**](/windows/win32/api/winuser/nf-winuser-setwindowlonga) .<br/> |
