# <a name="wm_moving-message"></a>\_Перемещение сообщения WM

Отправляется в окно, которое перемещает пользователь. Обрабатывая это сообщение, приложение может отслеживать расположение прямоугольника перетаскивания и, при необходимости, изменять его расположение.

Окно получает это сообщение через функцию [**WindowProc**](/previous-versions/windows/desktop/legacy/ms633573(v=vs.85)) .


```C++
#define WM_MOVING                       0x0216
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**Rect**](/previous-versions//dd162897(v=vs.85)) с текущей позицией окна в экранных координатах. Чтобы изменить расположение прямоугольника перетаскивания, приложение должно изменить члены этой структуры.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **lResult**

Приложение должно возвращать **значение true** , если обрабатывает это сообщение.
