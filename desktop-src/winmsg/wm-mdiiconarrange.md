# <a name="wm_mdiiconarrange-message"></a>\_Сообщение МДИИКОНАРРАНЖЕ WM

Приложение отправляет сообщение **WM \_ мдииконарранже** в окно клиента многодокументного интерфейса (MDI) для упорядочения всех минимальных дочерних окон MDI. Он не влияет на несведенные дочерние окна.


```C++
#define WM_MDIICONARRANGE               0x0228
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется; оно должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется; оно должно быть равно нулю.

</dd> </dl>
