# <a name="wm_mdirestore-message"></a>\_Сообщение МДИРЕСТОРЕ WM

Приложение отправляет сообщение **WM \_ мдиресторе** в окно клиента многодокументного интерфейса (MDI) для восстановления дочернего окна MDI из развернутого или минимального размера.


```C++
#define WM_MDIRESTORE                   0x0223
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Описатель дочернего окна MDI, подлежащей восстановлению.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **ноль**

Возвращаемое значение всегда равно нулю.
