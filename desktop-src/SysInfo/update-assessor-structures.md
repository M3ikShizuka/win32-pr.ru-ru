# <a name="update-assessor-structures"></a>Обновить структуры оценки

Поставщик метода доступа обновления использует следующие функции.



| Функция                                         | Описание                                                           |
|--------------------------------------------------|-----------------------------------------------------------------------|
| [**осупдатеассессмент**](/windows/win32/api/waasapitypes/ns-waasapitypes-osupdateassessment) | Определяет актуальность операционной системы на целевом устройстве.                |
| [**UpdateAssessment**](/windows/win32/api/waasapitypes/ns-waasapitypes-osupdateassessment)   | Содержит сведения, которые оценивают, насколько актуальна установленная ОС. |
