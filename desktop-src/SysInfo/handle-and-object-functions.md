# <a name="handle-and-object-functions"></a>Функции Handle и Object

С дескрипторами и объектами используются следующие функции.



| Функция                                             | Описание                                                                                  |
|------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**CloseHandle**](/windows/win32/api/handleapi/nf-handleapi-closehandle)                   | Закрывает открытый обработчик объекта.                                                                |
| [**компареобжександлес**](/windows/desktop/api/handleapi/nf-handleapi-compareobjecthandles) | Сравнивает два дескриптора объекта, чтобы определить, ссылаются ли они на один и тот же базовый объект ядра. |
| [**дупликатехандле**](/windows/win32/api/handleapi/nf-handleapi-duplicatehandle)           | Дублирует обработчик объекта.                                                                 |
| [**жесандлеинформатион**](/windows/win32/api/handleapi/nf-handleapi-gethandleinformation) | Извлекает определенные свойства обработчика объекта.                                            |
| [**сесандлеинформатион**](/windows/win32/api/handleapi/nf-handleapi-sethandleinformation) | Задает определенные свойства обработчика объекта.                                                 |
