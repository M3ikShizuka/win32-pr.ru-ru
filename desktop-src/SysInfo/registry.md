# <a name="registry"></a>Реестр

*Реестр* — это системная база данных, в которой приложения и системные компоненты хранят и извлекают данные конфигурации. Данные, хранящиеся в реестре, меняются в соответствии с версией Microsoft Windows. Приложения используют API реестра для извлечения, изменения или удаления данных реестра.

Не следует изменять данные реестра, не принадлежащие приложению, если это не обязательно. Если в реестре есть ошибка, система может работать неправильно. В этом случае можно восстановить реестр до состояния, в котором он находился при успешном запуске компьютера. Дополнительные сведения см. в справке по операционной системе.

Дополнительные сведения о базе данных реестра и программном доступе к данным реестра см. в следующих разделах:

-   [Сведения о реестре](about-the-registry.md)
-   [Использование реестра](using-the-registry.md)
-   [Справочник по реестру](registry-reference.md)
