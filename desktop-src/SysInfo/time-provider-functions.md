# <a name="time-provider-functions"></a>Функции поставщика времени

Следующие функции используются поставщиками времени.



| Функция                                                               | Описание                                                                                            |
|------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**алертсамплесаваилфунк**](/windows/desktop/api/Timeprov/nc-timeprov-alertsamplesavailfunc)                     | Уведомляет систему о наличии новых примеров.                                              |
| [**жеттимесисинфофунк**](/windows/desktop/api/Timeprov/nc-timeprov-gettimesysinfofunc)                           | Извлекает сведения о состоянии системного времени.                                                           |
| [**логтимепровевентфунк**](/windows/desktop/api/Timeprov/nc-timeprov-logtimeproveventfunc)                       | Регистрирует событие поставщика времени в журнале событий.                                                           |
| [**сетпровидерстатусфунк**](/windows/desktop/api/Timeprov/nc-timeprov-setproviderstatusfunc)                 | Задает сведения о состоянии поставщика времени.                                                           |
| [**сетпровидерстатусинфофрифунк**](/windows/desktop/api/Timeprov/nc-timeprov-setproviderstatusinfofreefunc) | Освобождает структуру [**сетпровидерстатусинфо**](/windows/desktop/api/Timeprov/ns-timeprov-setproviderstatusinfo) .                          |
| [**тимепровклосе**](/windows/desktop/api/Timeprov/nf-timeprov-timeprovclose)                                 | Функция обратного вызова, вызываемая диспетчером поставщиков времени для завершения работы поставщика времени.        |
| [**тимепровкомманд**](/windows/desktop/api/Timeprov/nf-timeprov-timeprovcommand)                             | Функция обратного вызова, вызываемая диспетчером поставщика времени для отправки команд поставщику времени. |
| [**тимепровопен**](/windows/desktop/api/Timeprov/nf-timeprov-timeprovopen)                                   | Функция обратного вызова, вызываемая диспетчером поставщиков времени при загрузке библиотеки DLL поставщика времени.  |
