# <a name="handle-limitations"></a>Ограничения по обработке

Некоторые объекты поддерживают только один обработчик за раз. Система предоставляет обработчик, когда приложение создает объект и делает его недействительным, когда приложение уничтожает объект. Другие объекты поддерживают несколько дескрипторов для одного объекта. Операционная система автоматически удаляет объект из памяти после закрытия последнего маркера объекта.

Общее число открытых дескрипторов в системе ограничено только объемом доступной памяти. Некоторые типы объектов поддерживают ограниченное количество дескрипторов для одного сеанса или для каждого процесса.
