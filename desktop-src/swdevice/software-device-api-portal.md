# <a name="software-device-api"></a>API программного устройства

## <a name="purpose"></a>Назначение

Вы можете использовать API программного устройства для создания устройства PnP из приложения. API позволяет перечислить устройство как дочернее по отношению к любому существующему родительскому устройству. API также позволяет регистрировать интерфейсы устройств для перечисленных устройств и задавать свойства для устройств и интерфейсов.

## <a name="developer-audience"></a>Аудитория разработчиков

API программного устройства предназначен для разработчиков, желающих публиковать функции устройств в каталоге PnP или загружать драйверы устройств.
