# <a name="starting-services-on-demand"></a>Запуск служб по требованию

Пользователь может запустить службу с помощью служебной программы панели управления "службы". Пользователь может указать аргументы для службы в поле **начальные параметры** . Программа управления службами может запустить службу и указать ее аргументы с помощью функции [**StartService**](/windows/desktop/api/Winsvc/nf-winsvc-startservicea) .

При запуске службы SCM выполняет следующие действия:

-   Получение сведений об учетной записи, хранящейся в базе данных.
-   Войдите в учетную запись службы.
-   Загрузите профиль пользователя.
-   Создайте службу в приостановленном состоянии.
-   Назначьте процессу маркер входа.
-   Разрешить выполнение процесса.
