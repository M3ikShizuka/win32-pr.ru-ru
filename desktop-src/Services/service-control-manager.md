# <a name="service-control-manager"></a>Диспетчер служб

Диспетчер управления службами (SCM) запускается при загрузке системы. Это сервер удаленного вызова процедур (RPC), который позволяет программам настройки служб и управления службами управлять службами на удаленных компьютерах.

Функции службы предоставляют интерфейс для следующих задач, выполняемых SCM:

-   Обслуживание базы данных установленных служб.
-   Запуск служб и служб драйверов при запуске системы или по требованию.
-   Перечисление установленных служб и служб драйверов.
-   Поддержание сведений о состоянии выполняющихся служб и служб драйверов.
-   Передача управляющих запросов на выполнение служб.
-   Блокировка и разблокировка базы данных службы.
