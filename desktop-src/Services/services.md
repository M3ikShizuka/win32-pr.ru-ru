# <a name="services-services"></a>Службы (службы)

*Приложение службы* соответствует правилам интерфейса диспетчера управления СЛУЖБАМИ (SCM). Он может запускаться автоматически при загрузке системы, пользователем через приложение панели управления "службы" или приложением, использующим [функции службы](service-functions.md). Службы могут выполняться, даже если ни один пользователь не вошел в систему.

*Служба драйвера* соответствует протоколам драйверов устройств. Он похож на приложение службы, но не взаимодействует с SCM. Для простоты термин « *Служба* » относится к *приложению службы* в этом обзоре.

Теперь триггеры можно использовать для управления запуском службы. Дополнительные сведения см. в разделе [Конфигурация службы](service-configuration.md).
