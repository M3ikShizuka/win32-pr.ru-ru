# <a name="single-instance-store-and-sis-backup"></a>Резервное копирование хранилища Single-Instance и SIS

*Хранилище единственных* копий или SIS — это архитектура, предназначенная для хранения повторяющихся файлов с минимальными затратами на диск, кэш и резервные носители. *Резервное копирование хранилища с одним экземпляром* или резервная копия SIS — это API, который вызывается приложениями резервного копирования для доступа к архитектуре SIS.
