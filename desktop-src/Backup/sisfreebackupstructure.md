# <a name="sisfreebackupstructure-function"></a>Функция Сисфрибаккупструктуре

Функция **сисфрибаккупструктуре** освобождает указанную структуру резервного копирования SIS.

## <a name="syntax"></a>Синтаксис


```C++
BOOL SisFreeBackupStructure(
  _In_ PVOID sisBackupStructure
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*сисбаккупструктуре* \[ окне\]
</dt> <dd>

Указатель на структуру резервного копирования SIS, возвращенную из [**сискреатебаккупструктуре**](siscreatebackupstructure.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция возвращает **значение true** , если она завершается успешно, и **false** в противном случае. Вызовите [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror) , чтобы получить дополнительные сведения о причине сбоя вызова.

## <a name="remarks"></a>Remarks

Эта функция должна вызываться после завершения операции резервного копирования для тома, определенного значением параметра *сисбаккупструктуре* .

Обратите внимание, что нельзя предположить, что это освобождает только память. Например, эта функция может также выполнять дополнительные административные операции для архитектуры SIS. Поэтому Вызывайте эту функцию, даже если операция резервного копирования будет завершена сразу же после этого.
