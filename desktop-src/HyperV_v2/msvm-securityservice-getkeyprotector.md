# <a name="getkeyprotector-method-of-the-msvm_securityservice-class"></a>Метод Жеткэйпротектор \_ класса SecurityService мсвм

Извлекает предохранитель ключа для виртуальной системы.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetKeyProtector(
  [in]  string SecuritySettingData,
  [out] uint8  KeyProtector[]
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Секуритисеттингдата* \[ окне\]
</dt> <dd>

Строка содержит встроенный экземпляр класса [**мсвм \_ секуритисеттингдата**](msvm-securitysettingdata.md) , представляющий параметры безопасности виртуальной системы.

</dd> <dt>

*Кэйпротектор* \[ заполняет\]
</dt> <dd>

Получает необработанный массив байтов, содержащий предохранитель ключа, который используется в данный момент.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

При успешном выполнении возвращает 0 или 4096. В противном случае возвращает ошибку.

**Система недоступна** (32777)
