# <a name="msvm_copyfiletoguestjobgeterror-method"></a>Мсвм \_ копифилетогуестжоб:: метод Error

Возвращает объект ошибки для задания, если оно существует. Если задание выполняется или завершается без ошибок, этот метод не возвращает объект [**\_ ошибки CIM**](/previous-versions//cc150671(v=vs.85)) . Однако если задание завершилось сбоем из-за внутренней проблемы или из-за завершения задания клиентом, возвращается экземпляр **\_ ошибки CIM** .

## <a name="syntax"></a>Синтаксис


```C++
uint32 GetError(
  [out] string Error
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Ошибка при* \[ заполняет\]
</dt> <dd>

Если рабочее состояние задания не равно 2 (ОК), этот метод возвращает встроенный экземпляр класса [**\_ ошибок мсвм**](msvm-error.md) в формате CIM-XML. Если рабочее состояние задания — 2 (ОК), возвращается **значение NULL** .

</dd> </dl>
