# <a name="revokeinteractivesessionaccess-method-of-the-msvm_terminalservice-class"></a>Метод Ревокеинтерактивесессионакцесс \_ класса Терминалсервице мсвм

Отменяет доступ к интерактивному сеансу виртуальной машины из указанного списка доверенных лиц.

## <a name="syntax"></a>Синтаксис


```mof
uint32 RevokeInteractiveSessionAccess(
  [in]  CIM_ComputerSystem REF ComputerSystem,
  [in]  string                 Trustees[],
  [out] CIM_ConcreteJob    REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ComputerSystem* \[ окне\]
</dt> <dd>

Ссылка на экземпляр класса платформы [**CIM \_**](cim-computersystem.md) , представляющий виртуальную машину, к которой будет отозван доступ.

</dd> <dt>

*Доверенные лица* \[ окне\]
</dt> <dd>

Массив строк, каждый из которых определяет доверенное лицо, права доступа которого будут отозваны. Идентификаторы доверенного лица должны быть указаны в формате, совместимом с Windows SAM, или в формате строки идентификатора безопасности Windows.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Если операция выполняется асинхронно, этот метод возвратит 4096, и этот параметр будет содержать ссылку на объект, производный от [**CIM \_ конкретежоб**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>
