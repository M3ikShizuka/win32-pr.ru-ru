# <a name="virtual-system-management-service"></a>Служба управления виртуальными системами

Профиль виртуальной системы описывает объекты, составляющие виртуальную машину. К таким объектам относятся базовая система, устройства, составляющие систему, параметры системы и ее устройств, а также служба управления, которая выполняет операции в системе.
