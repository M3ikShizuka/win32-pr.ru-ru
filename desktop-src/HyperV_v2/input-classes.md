# <a name="input-classes"></a>Входные классы

Пользовательские устройства ввода представлены этими классами. Виртуальная машина всегда имеет по одному экземпляру каждого класса, связанного с ним. Эти устройства не могут быть добавлены или удалены с виртуальной машины. Таким образом, не существует экземпляров пула ресурсов для устройств клавиатуры или мыши.

Устройства с клавиатурой и мышью связаны с виртуальной машиной с помощью ассоциации [**мсвм \_ системдевице**](msvm-systemdevice.md) . Виртуальная машина содержит как PS2, так и искусственное устройство мыши. В каждый момент времени активно только одно указывающее устройство.

Ниже перечислены классы WMI виртуализации, связанные с входными данными.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                          | Описание                                     |
|----------------------------------------------------------------|-------------------------------------------------|
| [**Мсвм \_ клавиатура**](msvm-keyboard.md)<br/>             | Представляет устройство клавиатуры.<br/>        |
| [**Мсвм \_ Ps2Mouse**](msvm-ps2mouse.md)<br/>             | Представляет устройство мыши PS2.<br/>       |
| [**Мсвм \_ синсетикмаусе**](msvm-syntheticmouse.md)<br/> | Представляет устройство искусственного мыши.<br/> |
