# <a name="destroysnapshot-method-of-the-msvm_collectionsnapshotservice-class"></a>Метод Дестройснапшот \_ класса Коллектионснапшотсервице мсвм

Уничтожает существующий моментальный снимок коллекции виртуальных систем. Этот метод может оказаться побочным действием уничтожить другие моментальные снимки, зависящие от затронутого моментального снимка.

## <a name="syntax"></a>Синтаксис


```mof
uint32 DestroySnapshot(
  [in]  CIM_Collection  REF AffectedSnapshotCollection,
  [out] CIM_ConcreteJob REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Аффектедснапшотколлектион* \[ окне\]
</dt> <dd>

Ссылка на [**\_ коллекцию CIM**](cim-collection.md) , описывающую затронутую коллекцию моментальных снимков виртуальной системы.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Необязательная ссылка, которая возвращается, если операция выполняется асинхронно. При наличии возвращаемой ссылки на экземпляр [**CIM \_ конкретежоб**](cim-concretejob.md) можно использовать для отслеживания хода выполнения и получения результата метода.

</dd> </dl>
