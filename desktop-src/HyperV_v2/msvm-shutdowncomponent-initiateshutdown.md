# <a name="initiateshutdown-method-of-the-msvm_shutdowncomponent-class"></a>Метод Инитиатешутдовн \_ класса Шутдовнкомпонент мсвм

Инициирует операцию завершения работы операционной системы на связанной дочерней виртуальной машине. Если возвращается ноль (0), то завершение работы было инициировано успешно. Любой другой код возврата указывает на состояние ошибки.

## <a name="syntax"></a>Синтаксис


```mof
uint32 InitiateShutdown(
  [in] boolean Force,
  [in] string  Reason
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Force* \[ окне\]
</dt> <dd>

Тип: **Boolean**

Флаг, который при **значении true** принудительно закрывает приложения, несмотря на наличие несохраненных данных.

</dd> <dt>

*Причина* \[ окне\]
</dt> <dd>

Тип: **строка**

Причина операции завершения работы. Это определяемая пользователем строка.

</dd> </dl>
