# <a name="getvhdsnapshotinformation-method-of-the-msvm_imagemanagementservice-class"></a>Метод Жетвхдснапшотинформатион \_ класса) мсвм

Извлекает сведения о моментальном снимке виртуального жесткого диска в файле набора виртуальных жестких дисков.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetVHDSnapshotInformation(
  [in]  string              VHDSetPath,
  [in]  string              SnapshotIds[],
  [in]  uint32              AdditionalInformation[],
  [out] string              SnapshotInformation[],
  [out] CIM_ConcreteJob REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Вхдсетпас* \[ окне\]
</dt> <dd>

Полный путь, указывающий расположение файла набора виртуальных жестких дисков.

</dd> <dt>

*Снапшотидс* \[ окне\]
</dt> <dd>

Список идентификаторов GUID, представляющих идентификаторы моментальных снимков каждого моментального снимка, для которого запрашиваются сведения. Если этот параметр имеет значение NULL, то будут получены сведения обо всех моментальных снимках.

</dd> <dt>

*Аддитионалинформатион* \[ окне\]
</dt> <dd>

Массив, указывающий, какие дополнительные сведения следует собрать о каждом моментальном снимке виртуального жесткого диска. Каждая запись в массиве является типом дополнительной информации. Если этот параметр имеет значение NULL, дополнительные сведения не извлекаются.

**Родительские пути** (2)

В случае успеха массив сведений о каждом запрошенном моментальном снимке. Каждый элемент является внедренным экземпляром [**мсвм \_ вхдснапшотинформатион**](msvm-vhdsnapshotinformation.md).
