# <a name="msvm_collectionsnapshotservice-class"></a>\_Класс мсвм коллектионснапшотсервице

Служба для создания, уничтожения и экспорта снимков коллекции виртуальных систем.

Следующий пример синтаксиса — упрощенный MOF-код, который включает все наследуемые свойства.

## <a name="syntax"></a>Синтаксис

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_CollectionSnapshotService : CIM_Service
{
};
```

## <a name="members"></a>Члены

Класс **мсвм \_ коллектионснапшотсервице** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Класс **мсвм \_ коллектионснапшотсервице** содержит следующие методы.



| Метод                                                                                    | Описание                                                                                                                                                                                                                   |
|:------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**апплиснапшот**](msvm-collectionsnapshotservice-applysnapshot.md)                     | Применяет коллекцию моментальных снимков к коллекции виртуальных компьютеров.<br/>                                                                                                                                        |
| [**конверттореференцепоинт**](msvm-collectionsnapshotservice-converttoreferencepoint.md) | Преобразование существующего моментального снимка коллекции в коллекцию опорных точек. Коллекция моментальных снимков удаляется как побочный результат. В контрольные точки можно преобразовать только моментальные снимки восстановления.<br/>                      |
| [**CreateSnapshot**](msvm-collectionsnapshotservice-createsnapshot.md)                   | Создает моментальный снимок коллекции виртуальных систем.<br/>                                                                                                                                                                 |
| [**дестройснапшот**](msvm-collectionsnapshotservice-destroysnapshot.md)                 | Уничтожает существующий моментальный снимок коллекции виртуальных систем. Этот метод может оказаться побочным действием уничтожить другие моментальные снимки, зависящие от затронутого моментального снимка.<br/>                                                   |
| [**експортснапшот**](msvm-collectionsnapshotservice-exportsnapshot.md)                   | Экспортирует коллекцию моментальных снимков для систем виртуальных компьютеров в файл. Коллекция моментальных снимков, связанные с ней параметры конфигурации и связанные с ней параметры ресурсов будут сохранены в результирующем файле.<br/> |
