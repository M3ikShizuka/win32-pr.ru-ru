# <a name="bios-classes"></a>Классы BIOS

Виртуальная BIOS — это образ программного обеспечения, который загружается в ОЗУ для настройки некоторых основных аспектов и загрузки компьютерной системы. Имеется один элемент BIOS на компьютерную систему, и этот элемент нельзя заменить или удалить. Таким словами, пул ресурсов для добавления новых элементов BIOS в виртуальную машину отсутствует. Элемент BIOS также не имеет собственного класса SettingData для описания параметров BIOS. Параметры BIOS, такие как серийные номера, порядок загрузки и состояние NUM LOCK, находятся в классе [**мсвм \_ виртуалсистемсеттингдата**](msvm-virtualsystemsettingdata.md) .

Ниже перечислены классы WMI виртуализации, связанные с BIOS. Эти классы находятся в \\ \\ . \\ Корневое \\ пространство имен виртуализации.
