# <a name="typescancodes-method-of-the-msvm_keyboard-class"></a>Метод Типесканкодес \_ класса клавиатуры мсвм

Имитирует последовательность ключей с помощью кодов сканирования.

## <a name="syntax"></a>Синтаксис


```mof
uint32 TypeScancodes(
  [in] uint8 Scancodes[]
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Сканкодес* \[ окне\]
</dt> <dd>

Тип: **Uint8 \[ \]**

Массив, содержащий коды проверки для ввода.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

Этот метод возвращает 0, если он завершается. Любое другое возвращаемое значение указывает на ошибку. Возвращаемое значение может принимать одно из следующих значений.

<dl> <dt>

**Завершено без ошибок** (0)
</dt> <dt>

**Параметры метода проверены — задание запущено** (4096)
</dt> <dt>

**Сбой** (32768)
</dt> <dt>

**Отказано в доступе** (32769)
</dt> <dt>

**Не поддерживается** (32770)
</dt> <dt>

**Состояние неизвестно** (32771)
</dt> <dt>

**Время ожидания** (32772)
</dt> <dt>

**Недопустимый параметр** (32773)
</dt> <dt>

**Система используется** (32774)
</dt> <dt>

**Недопустимое состояние для этой операции** (32775)
</dt> <dt>

**Неверный тип данных** (32776)
</dt> <dt>

**Система недоступна** (32777)
</dt> <dt>

**Недостаточно памяти** (32778)
</dt> </dl>

## <a name="remarks"></a>Комментарии
