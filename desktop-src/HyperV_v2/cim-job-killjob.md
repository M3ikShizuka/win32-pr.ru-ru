# <a name="killjob-method-of-the-cim_job-class"></a>Метод Киллжоб \_ класса заданий CIM

Метод для уничтожения этого задания и всех базовых процессов, а также для удаления любых "висячих" сопоставлений. Этот метод является устаревшим; Вместо этого используйте **рекуестчанжестате** . **Киллжоб** является устаревшим, так как нет различий между упорядоченным завершением работы и немедленной инструкцией Kill. [**Модель CIM \_ Конкретежоб**](cim-concretejob.md). **RequestStateChange**() предоставляет параметры "Terminate" и "Kill", чтобы разрешить это различие.

## <a name="syntax"></a>Синтаксис


```mof
uint32 KillJob(
  [in] boolean DeleteOnKill
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Делетеонкилл* \[ окне\]
</dt> <dd>

Указывает, должно ли задание автоматически удаляться после завершения работы. Этот параметр имеет приоритет над свойством **делетеонкомплетион** .

</dd> </dl>
