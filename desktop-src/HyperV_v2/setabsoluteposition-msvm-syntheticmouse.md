# <a name="setabsoluteposition-method-of-the-msvm_syntheticmouse-class"></a>Метод Сетабсолутепоситион \_ класса Синсетикмаусе мсвм

Задает горизонтальное и вертикальное расположение курсора мыши.

## <a name="syntax"></a>Синтаксис


```mof
uint32 SetAbsolutePosition(
  [in] sint32 horizontalPosition,
  [in] sint32 verticalPosition
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хоризонталпоситион* \[ окне\]
</dt> <dd>

Тип: **Sint32**

Новое горизонтальное расположение курсора мыши в пикселях.

</dd> <dt>

*вертикалпоситион* \[ окне\]
</dt> <dd>

Тип: **Sint32**

Новое вертикальное расположение курсора мыши в пикселях.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

Возвращаемое значение, равное нулю, указывает на успешное выполнение. Ненулевое значение указывает на ошибку изменения расположения прокрутки.

<dl> <dt>

**Завершено без ошибок** (0)
</dt> <dt>

**Параметры метода проверены — задание запущено** (4096)
</dt> <dt>

**Сбой** (32768)
</dt> <dt>

**Отказано в доступе** (32769)
</dt> <dt>

**Не поддерживается** (32770)
</dt> <dt>

**Состояние неизвестно** (32771)
</dt> <dt>

**Время ожидания** (32772)
</dt> <dt>

**Недопустимый параметр** (32773)
</dt> <dt>

**Система используется** (32774)
</dt> <dt>

**Недопустимое состояние для этой операции** (32775)
</dt> <dt>

**Неверный тип данных** (32776)
</dt> <dt>

**Система недоступна** (32777)
</dt> <dt>

**Недостаточно памяти** (32778)
</dt> </dl>
