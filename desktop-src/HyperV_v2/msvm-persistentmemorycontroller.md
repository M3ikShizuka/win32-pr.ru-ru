# <a name="msvm_persistentmemorycontroller-class"></a>\_Класс мсвм персистентмемориконтроллер

Представляет контроллер виртуальной энергонезависимой памяти.

Следующий пример синтаксиса — упрощенный MOF-код, который включает все наследуемые свойства.

## <a name="syntax"></a>Синтаксис

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_PersistentMemoryController : CIM_Controller
{
};
```

## <a name="members"></a>Члены

Класс **мсвм \_ персистентмемориконтроллер** не определяет никаких членов.
