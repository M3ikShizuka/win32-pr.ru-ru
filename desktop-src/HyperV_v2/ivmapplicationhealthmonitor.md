# <a name="ivmapplicationhealthmonitor-interface"></a>Интерфейс Ивмаппликатионхеалсмонитор

Сообщает о состоянии работоспособности приложения, которое выполняется на виртуальной машине, с компонентами интеграции Hyper-V, работающими на той же виртуальной машине. Состояние приложений, выполняющихся на виртуальной машине, отражается в  \[ \] значении свойства OperationalStatus 1 класса [**мсвм \_ хеартбеаткомпонент**](msvm-heartbeatcomponent.md) . Этот интерфейс также предоставляет способ сброса всех состояний приложений, накопленных в Hyper-V.

этот интерфейс реализуется Windows 8 компонентами интеграции Hyper-V. Экземпляр этого интерфейса получается путем создания экземпляра CLSID **397a2e5f-348c-482d-b9a3-57d383b483cd** .

## <a name="members"></a>Элементы

Интерфейс **ивмаппликатионхеалсмонитор** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмаппликатионхеалсмонитор** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмаппликатионхеалсмонитор** содержит следующие методы.



| Метод                                                                                   | Описание                                                                              |
|:-----------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------|
| [**ресеталлаппликатионстате**](ivmapplicationhealthmonitor-resetallapplicationstate.md) | Сбрасывает состояние работоспособности для всех приложений в виртуальной машине.<br/>            |
| [**сетаппликатионстате**](ivmapplicationhealthmonitor-setapplicationstate.md)           | Задает состояние работоспособности приложения, которое выполняется на виртуальной машине.<br/> |
