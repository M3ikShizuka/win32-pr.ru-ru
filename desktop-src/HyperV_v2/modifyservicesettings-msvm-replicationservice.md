# <a name="modifyservicesettings-method-of-the-msvm_replicationservice-class"></a>Метод Модифисервицесеттингс \_ класса Репликатионсервице мсвм

Изменяет параметры службы реплики Hyper-V.

## <a name="syntax"></a>Синтаксис


```mof
uint32 ModifyServiceSettings(
  [in]  string              SettingData,
  [out] CIM_ConcreteJob REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*SettingData* \[ окне\]
</dt> <dd>

Строковое представление класса [**мсвм \_ репликатионсервицесеттингдата**](msvm-replicationservicesettingdata.md) , содержащего измененные данные настройки для службы.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Если операция выполняется асинхронно, этот метод возвратит 4096, и этот параметр будет содержать ссылку на объект, производный от [**CIM \_ конкретежоб**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>
