# <a name="generatewwpn-method-of-the-msvm_virtualsystemmanagementservice-class"></a>Метод Женератеввпн \_ класса Виртуалсистемманажементсервице мсвм

Формирует набор WWN-имен портов (Ввпнс). Ввпнс создаются в предварительно настроенном диапазоне, определяемом свойствами **минимумввпнаддресс** и **максимумввпнаддресс** класса [**мсвм \_ виртуалсистемманажементсервицесеттингдата**](msvm-virtualsystemmanagementservicesettingdata.md) . Если допустимое число Ввпнс меньше запрошенного числа, остальные записи в массиве *женератедввпн* будут иметь недопустимую запись "0000000000000000", и возвращаемое значение будет означать успешность (0).

## <a name="syntax"></a>Синтаксис


```mof
uint32 GenerateWwpn(
  [in]  uint32 NumberOfWwpns,
  [out] string GeneratedWwpn[]
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Нумберофввпнс* \[ окне\]
</dt> <dd>

Число создаваемых Ввпнс.

</dd> <dt>

*Женератедввпн* \[ заполняет\]
</dt> <dd>

Массив строк, каждый из которых будет содержать созданное WWPN. Он будет отформатирован в виде строки как "01:23:45:67:89: AB: CD: EF".

</dd> </dl>
