# <a name="modifyreplicationsettings-method-of-the-msvm_replicationservice-class"></a>Метод Модифирепликатионсеттингс \_ класса Репликатионсервице мсвм

Изменяет параметры репликации для виртуальной машины. Когда клиент вызывает этот метод для реплики виртуальной машины, он изменяет параметры репликации отношения репликации с расширенной репликой.

## <a name="syntax"></a>Синтаксис


```mof
uint32 ModifyReplicationSettings(
  [in]  CIM_ComputerSystem REF ComputerSystem,
  [in]  string                 ReplicationSettingData,
  [out] CIM_ConcreteJob    REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ComputerSystem* \[ окне\]
</dt> <dd>

Ссылка на экземпляр [**CIM \_ ComputerSystem**](/windows/desktop/CIMWin32Prov/cim-computersystem) , представляющий виртуальную машину, для которой необходимо изменить параметры репликации.

</dd> <dt>

*Репликатионсеттингдата* \[ окне\]
</dt> <dd>

Строковое представление класса [**мсвм \_ репликатионсеттингдата**](msvm-replicationsettingdata.md) , определяющего новые параметры репликации.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Если операция выполняется асинхронно, этот метод возвратит 4096, и этот параметр будет содержать ссылку на объект, производный от [**CIM \_ конкретежоб**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>

## <a name="remarks"></a>Remarks

**Модифирепликатионсеттингс** принимает экземпляр [**МСВМ \_ репликатионсеттингдата**](msvm-replicationsettingdata.md) (фрсд) в качестве входных данных. По умолчанию используется связанный ФРСД для виртуальной машины в качестве поставщика услуг размещения. Входные ФРСД проверяются на допустимые параметры для каждого свойства для поставщика по умолчанию. В этой таблице перечислены различия проверки по отношению к внешнему поставщику.



| Свойство                                             | Внешние поставщики                                 |
|------------------------------------------------------|----------------------------------------------------|
| репликатионпровидер                                  | То же, что и поставщик по умолчанию                           |
| authenticationType                                   | Не учитывается                                            |
| CertificateThumbPrint                                | Не учитывается                                            |
| Рутцертификатесумбпринт (RO)                       | Не учитывается                                            |
| компрессионенаблед                                   | То же, что и поставщик по умолчанию                           |
| бипасспроксисервер                                    | То же, что и поставщик по умолчанию                           |
| рековериконнектионпоинт                              | Игнорируется \* (может измениться, если поставщик имеет требование) |
| Рековерихостсистем (RO)                              | Не учитывается                                            |
| Примариконнектионпоинт (RO)                          | То же, что и поставщик по умолчанию                           |
| Примарихостсистем (RO)                               | То же, что и поставщик по умолчанию                           |
| рековерисерверпортнумбер                             | Игнорируется \* (может измениться, если поставщик имеет требование) |
| репликатехостквпитемс                                | Не учитывается                                            |
| аппликатионконсистентснапшотинтервал                | То же, что и поставщик по умолчанию                           |
| рековерихистори                                      | То же, что и поставщик по умолчанию                           |
| инклудеддискс\[\]                                    | То же, что и поставщик по умолчанию                           |
| ауторесинчронизинаблед                             | То же, что и поставщик по умолчанию                           |
| ауторесинчронизеинтервалстарт                       | То же, что и поставщик по умолчанию                           |
| ауторесинчронизеинтерваленд                         | То же, что и поставщик по умолчанию                           |
| Енаблевритеордерпресерватионакроссдискс (не рекомендуется) | То же, что и поставщик по умолчанию                           |
| ReplicationInterval                                  | То же, что и поставщик по умолчанию                           |
