# <a name="modifypoolsettings-method-of-the-msvm_resourcepoolconfigurationservice-class"></a>Метод Модифипулсеттингс \_ класса Ресаурцепулконфигуратионсервице мсвм

Изменяет параметры дочернего пула, не связанные с распределением.

## <a name="syntax"></a>Синтаксис


```mof
uint32 ModifyPoolSettings(
  [in]  CIM_ResourcePool REF ChildPool,
  [in]  string               PoolSettings,
  [out] CIM_ConcreteJob  REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Чилдпул* \[ окне\]
</dt> <dd>

Ссылка на экземпляр класса [**CIM \_ ResourcePool**](cim-resourcepool.md) , представляющий дочерний пул, который нужно изменить.

</dd> <dt>

*Пулсеттингс* \[ окне\]
</dt> <dd>

Встроенный экземпляр класса [**мсвм \_ ресаурцепулсеттингдата**](msvm-resourcepoolsettingdata.md) , который используется для указания новых параметров пула.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Если операция выполняется асинхронно, этот метод возвратит 4096, и этот параметр будет содержать ссылку на объект, производный от [**CIM \_ конкретежоб**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>
