# <a name="getsystemcompatibilityinfo-method-of-the-msvm_virtualsystemmigrationservice-class"></a>Метод Жетсистемкомпатибилитинфо \_ класса Виртуалсистеммигратионсервице мсвм

Создает непрозрачный большой двоичный объект данных, содержащий сведения о совместимости для указанной системы. Этот метод используется совместно с методом [**чекксистемкомпатибилитинфо**](checksystemcompatibilityinfo-msvm-virtualsystemmigrationservice.md) , чтобы определить, возможна ли быстрая или динамическая миграция виртуальной машины на другую компьютерную систему, без фактической попытки миграции.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetSystemCompatibilityInfo(
  [in]  CIM_ComputerSystem REF ComputerSystem,
  [out] uint8                  CompatibilityInfo[]
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ComputerSystem* \[ окне\]
</dt> <dd>

Ссылка на класс [**мсвм \_ ComputerSystem**](msvm-computersystem.md) , представляющий виртуальную машину, для которой необходимо получить сведения о совместимости. Если этот параметр относится к системе компьютера размещения, то данные, возвращаемые в параметре *компатибилитинфо* , можно использовать, чтобы определить, можно ли быстро перенести любую из виртуальных машин на компьютере, на котором размещен компьютер, в другую систему.

</dd> <dt>

*Компатибилитинфо* \[ заполняет\]
</dt> <dd>

Непрозрачный большой двоичный объект данных, который может быть передан методу [**чекксистемкомпатибилитинфо**](checksystemcompatibilityinfo-msvm-virtualsystemmigrationservice.md) на другом компьютере компьютера для подтверждения совместимости.

</dd> </dl>
