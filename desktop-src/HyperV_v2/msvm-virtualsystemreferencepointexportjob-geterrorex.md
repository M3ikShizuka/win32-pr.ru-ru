# <a name="geterrorex-method-of-the-msvm_virtualsystemreferencepointexportjob-class"></a>Метод Жетеррорекс \_ класса Виртуалсистемреференцепоинтекспортжоб мсвм

Извлекает дополнительные сведения об ошибке. При выполнении или завершении задания без ошибок **жетеррорекс** не возвращает экземпляр **жетеррорекс** . Однако если задание завершилось сбоем из-за внутренней проблемы или из-за завершения задания клиентом, **жетеррорекс** возвращает один или несколько экземпляров [**\_ ошибок мсвм**](msvm-error.md) .

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetErrorEx(
  [out] string Errors[]
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Ошибки* \[ заполняет\]
</dt> <dd>

Если оперативное состояние задания не равно "ОК", этот параметр возвращает массив экземпляров [**\_ ошибок мсвм**](msvm-error.md) . В противном случае, если задано "ОК", этот параметр содержит **значение NULL**.

</dd> </dl>

При успешном выполнении возвращает 0; в противном случае возвращает ошибку.
