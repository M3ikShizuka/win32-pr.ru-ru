# <a name="migratevirtualsystemtosystem-method-of-the-cim_virtualsystemmigrationservice-class"></a>Метод Мигратевиртуалсистемтосистем \_ класса CIM виртуалсистеммигратионсервице

Метод для перемещения, переноса или повторного размещения виртуальной системы в целевой системе.

Описание кода возврата:

## <a name="syntax"></a>Синтаксис


```mof
uint32 MigrateVirtualSystemToSystem(
  [in]  CIM_ComputerSystem REF ComputerSystem,
  [in]  CIM_System         REF DestinationSystem,
  [in]  string                 MigrationSettingData,
  [in]  string                 NewSystemSettingData,
  [in]  string                 NewResourceSettingData[],
  [out] CIM_ComputerSystem REF NewComputerSystem,
  [out] CIM_ConcreteJob    REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ComputerSystem* \[ окне\]
</dt> <dd>

Система исходного виртуального компьютера для переноса.

</dd> <dt>

*Дестинатионсистем* \[ окне\]
</dt> <dd>

Целевая система узла вхерето перенесите виртуальную систему.

</dd> <dt>

*Мигратионсеттингдата* \[ окне\]
</dt> <dd>

Строка, содержащая встроенный экземпляр класса [**CIM \_ виртуалсистеммигратионсеттингдата**](cim-virtualsystemmigrationsettingdata.md) , представляющий параметры миграции, применимые к операции миграции.

</dd> <dt>

*Невсистемсеттингдата* \[ окне\]
</dt> <dd>

Строка, содержащая встроенный экземпляр класса [**CIM \_ виртуалсистемсеттингдата**](cim-virtualsystemsettingdata.md) , представляющий новые свойства, применимые к виртуальной системе после переноса.

</dd> <dt>

*Невресаурцесеттингдата* \[ окне\]
</dt> <dd>

Массив строк, каждый из которых содержит встроенный экземпляр класса [**CIM \_ ресаурцеаллокатионсеттингдата**](cim-resourceallocationsettingdata.md) , представляющий новые свойства, применимые к виртуальным ресурсам в области виртуальной системы после переноса.

</dd> <dt>

*Невкомпутерсистем* \[ заполняет\]
</dt> <dd>

Ссылка на экземпляр класса системы [**CIM, \_**](cim-computersystem.md) представляющий систему виртуального компьютера после переноса.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Если операция выполняется долго, может быть возвращена также [**\_ конкретежоб CIM**](cim-concretejob.md) , представляющая задание.

</dd> </dl>
