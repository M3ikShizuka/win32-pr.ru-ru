# <a name="cim_metricforme-class"></a>\_Класс CIM метрикформе

Представляет ассоциацию, в которой собираются значения метрик для управляемого элемента.

## <a name="syntax"></a>Синтаксис

``` syntax
[Association, Abstract, Version("2.7.0"), UMLPackagePath("CIM::Metrics::BaseMetric"), AMENDMENT]
class CIM_MetricForME : CIM_Dependency
{
  CIM_ManagedElement  REF Antecedent;
  CIM_BaseMetricValue REF Dependent;
};
```
