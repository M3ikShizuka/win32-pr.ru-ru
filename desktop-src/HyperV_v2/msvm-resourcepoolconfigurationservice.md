# <a name="msvm_resourcepoolconfigurationservice-class"></a>\_Класс мсвм ресаурцепулконфигуратионсервице

Обеспечивает активное управление пулами ресурсов.

Следующий синтаксис является упрощенным MOF-файлным (MOF) кодом и включает все наследуемые свойства.

## <a name="syntax"></a>Синтаксис

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider")]
class Msvm_ResourcePoolConfigurationService : CIM_Service
{
  string   InstanceID;
  string   Caption;
  string   Description;
  string   ElementName;
  datetime InstallDate;
  string   Name;
  uint16   OperationalStatus[];
  string   StatusDescriptions[];
  string   Status;
  uint16   HealthState;
  uint16   CommunicationStatus;
  uint16   DetailedStatus;
  uint16   OperatingStatus;
  uint16   PrimaryStatus;
  uint16   EnabledState = 2;
  string   OtherEnabledState;
  uint16   RequestedState = 12;
  uint16   EnabledDefault = 2;
  datetime TimeOfLastStateChange;
  uint16   AvailableRequestedStates[];
  uint16   TransitioningToState;
  string   SystemCreationClassName;
  string   SystemName;
  string   CreationClassName;
  string   PrimaryOwnerName;
  string   PrimaryOwnerContact;
  string   StartMode;
  boolean  Started;
};
```

### <a name="methods"></a>Методы

| Метод                                                                                   | Описание                                                                                  |
|:-----------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------|

| [**DeletePool**](deletepool-msvm-resourcepoolconfigurationservice.md)                   | Удаляет пул ресурсов.<br/>                                                          |
| [**модифипулресаурцес**](modifypoolresources-msvm-resourcepoolconfigurationservice.md) | Изменяет параметры ресурсов родительского пула для ресурсов, назначенных дочернему пулу.<br/> |
| [**модифипулсеттингс**](modifypoolsettings-msvm-resourcepoolconfigurationservice.md)   | Изменяет параметры дочернего пула, не связанные с распределением.<br/>             |
| **Равен**                                                                   | Этот метод не поддерживается.<br/>                                                     |
| **StartService**                                                                         | Этот метод не поддерживается.<br/>                                                     |
| **StopService**                                                                          | Этот метод не поддерживается.<br/>                                                     |
