# <a name="hyper-v-virtual-fibre-channel-api-reference"></a>Справочник по API виртуальных Fibre Channel Hyper-V

Виртуальный Fibre Channel API Hyper-V определяет следующие элементы программирования.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                    | Описание                                                                                                    |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**Мсвм \_ екстерналфкпорт**](msvm-externalfcport.md)<br/>                           | Представляет внешний порт Fibre Channel.<br/>                                                          |
| [**Мсвм \_ фкактивеконнектион**](msvm-fcactiveconnection.md)<br/>                   | Подключает порт коммутатора к конечной точке Fibre Channel, к которой подключен порт.<br/>                |
| [**Мсвм \_ фкдевицесапимплементатион**](msvm-fcdevicesapimplementation.md)<br/>     | Представляет связь между точкой доступа службы и логическим устройством, которое ее реализует.<br/> |
| [**Мсвм \_ фцендпоинт**](msvm-fcendpoint.md)<br/>                                   | Представляет логическую точку соединения для порта Fibre Channel.<br/>                                   |
| [**Мсвм \_ фкпорталлокатионсеттингдата**](msvm-fcportallocationsettingdata.md)<br/> | Представляет настроенное состояние искусственного Fibre Channel порта или порта коммутатора Fibre Channel.<br/>   |
| [**Мсвм \_ фксвитчпорт**](msvm-fcswitchport.md)<br/>                               | Представляет порт в виртуальном Fibre Channel коммутаторе.<br/>                                              |
| [**Мсвм \_ синсетикфкпорт**](msvm-syntheticfcport.md)<br/>                         | Представляет порт искусственного Fibre Channel.<br/>                                                          |
| [**Мсвм \_ синсетикфкпортсеттингдата**](msvm-syntheticfcportsettingdata.md)<br/>   | Представляет настроенное состояние искусственного Fibre Channel порта.<br/>                                  |
| [**Мсвм \_ виртуалфксвитч**](msvm-virtualfcswitch.md)<br/>                         | Представляет виртуальный Fibre Channel коммутатор.<br/>                                                          |
| [**Мсвм \_ виртуалфксвитчсеттингдата**](msvm-virtualfcswitchsettingdata.md)<br/>   | Представляет конфигурацию коммутатора виртуальной Fibre Channel.<br/>                                     |
