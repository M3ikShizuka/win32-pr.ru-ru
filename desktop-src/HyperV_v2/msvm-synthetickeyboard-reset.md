# <a name="reset-method-of-the-msvm_synthetickeyboard-class"></a>Метод Reset \_ класса мсвм синсетиккэйбоард

Сбрасывает виртуальную клавиатуру.

## <a name="syntax"></a>Синтаксис


```mof
uint32 Reset();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

При успешном выполнении возвращает 0; в противном случае возвращает ошибку.

<dl> <dt>

**Завершено без ошибок** (0)
</dt> <dt>

**Не поддерживается** (1)
</dt> </dl>
