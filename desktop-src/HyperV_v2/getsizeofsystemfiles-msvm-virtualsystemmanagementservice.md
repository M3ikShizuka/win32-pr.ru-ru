# <a name="getsizeofsystemfiles-method-of-the-msvm_virtualsystemmanagementservice-class"></a>Метод Жетсизеофсистемфилес \_ класса Виртуалсистемманажементсервице мсвм

Получает общий размер системных файлов виртуальной машины. К ним относятся файлы конфигурации и сохраненного состояния.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetSizeOfSystemFiles(
  [in]  CIM_VirtualSystemSettingData REF Vssd,
  [out] uint64                           Size
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ВССД* \[ окне\]
</dt> <dd>

Ссылка на экземпляр [**\_ виртуалсистемсеттингдата CIM**](/previous-versions//cc136954(v=vs.85)) , для которого необходимо получить размер системных файлов. Этот экземпляр может представлять либо текущий экземпляр виртуальной машины, либо экземпляр моментального снимка виртуальной машины.

</dd> <dt>

*Размер* \[ заполняет\]
</dt> <dd>

Общий размер системных файлов (в байтах).

</dd> </dl>
