# <a name="saveproperties-method-of-the-cim_logicaldevice-class"></a>Метод Савепропертиес \_ класса CIM

Запрашивает, что устройство захватывает текущую конфигурацию, настройки и/или сведения о состоянии в резервном хранилище. Цель — использовать эти сведения позже (с помощью метода Ресторепропертиес), чтобы вернуть устройство к текущему условию. Этот метод может не поддерживаться всеми устройствами. Метод должен возвращать 0 в случае успеха, 1, если запрос не поддерживается, и другое значение, если произошла какая-либо другая ошибка. В подклассе можно указать набор возможных кодов возврата, используя квалификатор ValueMap в методе. Строки, в которые содержимое ValueMap является переведенными, могут также быть указаны в подклассе как квалификатор массива Values.

## <a name="syntax"></a>Синтаксис


```mof
uint32 SaveProperties();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение 0 в случае успешного выполнения; в противном случае возвращает ошибку.
