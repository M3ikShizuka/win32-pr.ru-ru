# <a name="startservice-method-of-the-msvm_terminalservice-class"></a>Метод StartService \_ класса мсвм терминалсервице

запускает службу.

## <a name="syntax"></a>Синтаксис


```mof
uint32 StartService();
```



## <a name="parameters"></a>Параметры
