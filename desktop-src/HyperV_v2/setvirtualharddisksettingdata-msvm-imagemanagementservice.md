# <a name="setvirtualharddisksettingdata-method-of-the-msvm_imagemanagementservice-class"></a>Метод Сетвиртуалхарддисксеттингдата \_ класса) мсвм

Обновляет параметры виртуального жесткого диска.

## <a name="syntax"></a>Синтаксис


```mof
uint32 SetVirtualHardDiskSettingData(
  [in]  string              VirtualDiskSettingData,
  [out] CIM_ConcreteJob REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Виртуалдисксеттингдата* \[ окне\]
</dt> <dd>

Строковое представление класса [**мсвм \_ виртуалхарддисксеттингдата**](msvm-virtualharddisksettingdata.md) , которое указывает виртуальный жесткий диск для обновления и содержащий новые данные параметра. С помощью этого метода можно обновлять только свойства **парентпас**, **фисикалсекторсизе** или **виртуалдискид** . Вы не можете обновить эти свойства с помощью одного вызова метода. Обновить одно из этих свойств можно только с помощью одного вызова метода.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Если операция выполняется асинхронно, этот метод возвратит 4096, и этот параметр будет содержать ссылку на объект, производный от [**CIM \_ конкретежоб**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>
