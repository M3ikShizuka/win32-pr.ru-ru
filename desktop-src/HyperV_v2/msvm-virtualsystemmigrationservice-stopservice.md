# <a name="stopservice-method-of-the-msvm_virtualsystemmigrationservice-class"></a>Метод «начало» \_ класса мсвм виртуалсистеммигратионсервице

## <a name="syntax"></a>Синтаксис


```mof

```



## <a name="parameters"></a>Параметры
