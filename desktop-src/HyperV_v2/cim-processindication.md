# <a name="cim_processindication-class"></a>\_Класс CIM процессиндикатион

**Модель CIM \_ Процессиндикатион** — это абстрактный суперкласс для специализированных классов индикации, которые решают конкретные изменения и предупреждения, опубликованные поставщиками и инструментарием.

## <a name="syntax"></a>Синтаксис

``` syntax
[Indication, Version("2.6.0"), UMLPackagePath("CIM::Event"), AMENDMENT]
class CIM_ProcessIndication : CIM_Indication
{
};
```

## <a name="members"></a>Члены

Класс **CIM \_ процессиндикатион** не определяет никаких членов.
