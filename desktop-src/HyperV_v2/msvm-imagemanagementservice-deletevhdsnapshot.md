# <a name="deletevhdsnapshot-method-of-the-msvm_imagemanagementservice-class"></a>Метод Делетевхдснапшот \_ класса) мсвм

Удаляет запись моментального снимка виртуального жесткого диска в файле набора виртуальных жестких дисков.

## <a name="syntax"></a>Синтаксис


```mof
uint32 DeleteVHDSnapshot(
  [in]  string              VHDSetPath,
  [in]  string              SnapshotId,
  [in]  boolean             PersistReferenceSnapshot,
  [out] CIM_ConcreteJob REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Вхдсетпас* \[ окне\]
</dt> <dd>

Строка, содержащая путь к файлу набора виртуальных жестких дисков, содержащему рассматриваемый моментальный снимок.

</dd> <dt>

*SnapshotId* \[ окне\]
</dt> <dd>

Идентификатор GUID, представляющий идентификатор моментального снимка для удаляемого элемента моментального снимка виртуального жесткого диска.

</dd> <dt>

*Персистреференцеснапшот* \[ окне\]
</dt> <dd>

Следует ли сохранять запись моментального снимка только для ссылок после удаления моментального снимка.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Ссылка на задание (может иметь значение null, если задача завершена).

</dd> </dl>
