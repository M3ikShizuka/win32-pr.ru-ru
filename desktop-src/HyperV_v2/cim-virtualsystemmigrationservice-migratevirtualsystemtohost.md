# <a name="migratevirtualsystemtohost-method-of-the-cim_virtualsystemmigrationservice-class"></a>Метод Мигратевиртуалсистемтохост \_ класса CIM виртуалсистеммигратионсервице

Метод для перемещения, переноса или повторного размещения виртуальной системы на целевом узле, заданном сетевым именем или IP-адресом.

> [!Note]  
> Этот метод предназначен как переходное решение только после того, как будет доступно моделирование поддержки кластера.

 

## <a name="syntax"></a>Синтаксис


```mof
uint32 MigrateVirtualSystemToHost(
  [in]  CIM_ComputerSystem REF ComputerSystem,
  [in]  string                 DestinationHost,
  [in]  string                 MigrationSettingData,
  [in]  string                 NewSystemSettingData,
  [in]  string                 NewResourceSettingData[],
  [out] CIM_ConcreteJob    REF Job
);
```

<dl> <dt>
