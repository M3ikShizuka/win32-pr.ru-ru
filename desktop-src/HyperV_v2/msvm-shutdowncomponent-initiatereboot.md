# <a name="initiatereboot-method-of-the-msvm_shutdowncomponent-class"></a>Метод Инитиатеребут \_ класса Шутдовнкомпонент мсвм

Инициирует операцию перезагрузки операционной системы для связанной дочерней виртуальной машины.

Если возвращается ноль (0), перезагрузка была успешно инициирована. Любой другой код возврата указывает на состояние ошибки.

## <a name="syntax"></a>Синтаксис


```mof
uint32 InitiateReboot(
  [in] boolean Force,
  [in] string  Reason
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Force* \[ окне\]
</dt> <dd>

Флаг, который при значении TRUE принудительно закрывает приложения, несмотря на наличие несохраненных данных.

</dd> <dt>

*Причина* \[ окне\]
</dt> <dd>

Причина операции перезагрузки. Это определяемая пользователем строка.

</dd> </dl>

</dt> <dt>

</dt> <dt>

**Состояние неизвестно** (32771)
</dt> <dt>

</dt> <dt>

**Недопустимый параметр** (32773)
