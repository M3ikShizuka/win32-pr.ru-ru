# <a name="reset-method-of-the-msvm_syntheticdisplaycontroller-class"></a>Метод Reset \_ класса мсвм синсетикдисплайконтроллер

## <a name="syntax"></a>Синтаксис


```mof

```



## <a name="parameters"></a>Параметры
