# <a name="applysnapshot-method-of-the-msvm_collectionsnapshotservice-class"></a>Метод Апплиснапшот \_ класса Коллектионснапшотсервице мсвм

Применяет коллекцию моментальных снимков к коллекции виртуальных компьютеров.

## <a name="syntax"></a>Синтаксис


```mof
uint32 ApplySnapshot(
  [in]  CIM_Collection  REF SnapshotCollection,
  [out] CIM_ConcreteJob REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Снапшотколлектион* \[ окне\]
</dt> <dd>

Ссылка на [**\_ коллекцию CIM**](cim-collection.md) , представляющую коллекцию моментальных снимков, которую необходимо применить.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Необязательная ссылка, которая возвращается, если операция выполняется асинхронно. При наличии возвращаемой ссылки на экземпляр [**CIM \_ конкретежоб**](cim-concretejob.md) можно использовать для отслеживания хода выполнения и получения результата метода.

</dd> </dl>
