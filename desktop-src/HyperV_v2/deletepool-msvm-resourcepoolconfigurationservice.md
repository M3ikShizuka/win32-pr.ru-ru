# <a name="deletepool-method-of-the-msvm_resourcepoolconfigurationservice-class"></a>Метод DeletePool \_ класса Ресаурцепулконфигуратионсервице мсвм

Удаляет пул ресурсов. Для успешного удаления пула ресурсов выделение не может быть завершено, а удаление завершится с ошибкой 32774 (используется). Если пул ресурсов является корневым пулом ресурсов, все ресурсы узла возвращаются к базовой системе.

## <a name="syntax"></a>Синтаксис


```mof
uint32 DeletePool(
  [in]  CIM_ResourcePool REF Pool,
  [out] CIM_ConcreteJob  REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Пул* \[ окне\]
</dt> <dd>

Ссылка на экземпляр класса [**CIM \_ ResourcePool**](cim-resourcepool.md) , представляющий удаляемый пул.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Если операция выполняется асинхронно, этот метод возвратит 4096, и этот параметр будет содержать ссылку на объект, производный от [**CIM \_ конкретежоб**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>
