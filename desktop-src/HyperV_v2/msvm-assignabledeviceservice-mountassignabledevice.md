# <a name="mountassignabledevice-method-of-the-msvm_assignabledeviceservice-class"></a>Метод Маунтассигнабледевице \_ класса Ассигнабледевицесервице мсвм

Подключает указанное устройство PCI, чтобы его можно было использовать в системе главного компьютера.

## <a name="syntax"></a>Синтаксис


```mof
uint32 MountAssignableDevice(
  [in]  string              DeviceInstancePath,
  [in]  string              DeviceLocationPath,
  [out] string              MountedDeviceInstancePath,
  [out] CIM_ConcreteJob REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Девицеинстанцепас* \[ окне\]
</dt> <dd>

Строка, содержащая путь к экземпляру устройства для устройства.

</dd> <dt>

*Девицелокатионпас* \[ окне\]
</dt> <dd>

Строка, содержащая путь расположения устройства к устройству.

</dd> <dt>

*Маунтеддевицеинстанцепас* \[ заполняет\]
</dt> <dd>

Строка, содержащая путь к экземпляру устройства для подключенного устройства.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Ссылка на задание (может иметь значение null, если задача завершена).

</dd> </dl>
