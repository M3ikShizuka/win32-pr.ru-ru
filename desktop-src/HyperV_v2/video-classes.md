# <a name="video-classes"></a>Классы видео

Все виртуальные машины показывают наличие эмулированного видеоконтроллера S3 и виртуального видеоконтроллера с ускорением.

С каждым контроллером экрана связан объект head видео. Только один контроллер экрана может быть активным на виртуальной машине в любое время.

Для каждого активного удаленного сеанса, подключенного к виртуальной машине, имеется подключение терминала.

Ниже перечислены классы WMI виртуализации, связанные с видео.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                  | Описание                                                                                                                |
|----------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [**Мсвм \_ S3DisplayController**](msvm-s3displaycontroller.md)<br/>               | Представляет состояние эмулированного контроллера S3, присутствующего в каждой конфигурации виртуальной машины.<br/>       |
| [**Мсвм \_ синсетикдисплайконтроллер**](msvm-syntheticdisplaycontroller.md)<br/> | Представляет состояние искусственного контроллера экрана, присутствующего в каждой конфигурации виртуальной машины.<br/> |
| [**Мсвм \_ системтерминалконнектион**](msvm-systemterminalconnection.md)<br/>     | Связывает виртуальную машину с терминальным подключением.<br/>                                                        |
| [**Мсвм \_ терминалконнектион**](msvm-terminalconnection.md)<br/>                 | Указывает состояние активного удаленного сеанса, взаимодействующего с виртуальной машиной.<br/>                             |
| [**Мсвм \_ видеохеад**](msvm-videohead.md)<br/>                                   | Описывает основную поверхность рисования на контроллере экрана.<br/>                                                  |
| [**Мсвм \_ видеохеадонконтроллер**](msvm-videoheadoncontroller.md)<br/>           | Связывает головной диск с видеоконтроллером, который его содержит.<br/>                                             |
