# <a name="msvm_syntheticdisplaycontrollersettingdata-class"></a>\_Класс мсвм синсетикдисплайконтроллерсеттингдата

Описание параметров данных для виртуального искусственного контроллера экрана.

Следующий синтаксис упрощен из MOF-кода и включает все унаследованные свойства.

## <a name="syntax"></a>Синтаксис

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_SyntheticDisplayControllerSettingData : CIM_ResourceAllocationSettingData
{
  uint8  ResolutionType;
  uint16 HorizontalResolution;
  uint16 VerticalResolution;
};
```
