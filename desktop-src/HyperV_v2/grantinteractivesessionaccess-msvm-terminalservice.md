# <a name="grantinteractivesessionaccess-method-of-the-msvm_terminalservice-class"></a>Метод Грантинтерактивесессионакцесс \_ класса Терминалсервице мсвм

Предоставляет доступ к интерактивному сеансу виртуальной машины указанному списку доверенных лиц.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GrantInteractiveSessionAccess(
  [in]  CIM_ComputerSystem REF ComputerSystem,
  [in]  string                 Trustees[],
  [out] CIM_ConcreteJob    REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ComputerSystem* \[ окне\]
</dt> <dd>

Ссылка на экземпляр класса [**мсвм \_ ComputerSystem**](msvm-computersystem.md) , представляющий виртуальную машину, к которой будет предоставлен доступ.

</dd> <dt>

*Доверенные лица* \[ окне\]
</dt> <dd>

Массив строк, каждый из которых определяет доверенное лицо, которому будет предоставлен доступ к интерактивному сеансу виртуальной машины. идентификаторы доверенного лица должны быть указаны в формате Windows, совместимом с SAM, или в Windows формате строки идентификатора безопасности.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Если операция выполняется асинхронно, этот метод возвратит 4096, и этот параметр будет содержать ссылку на объект, производный от [**CIM \_ конкретежоб**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>
