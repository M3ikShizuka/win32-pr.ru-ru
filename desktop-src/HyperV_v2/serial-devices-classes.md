# <a name="serial-devices-classes"></a>Классы последовательных устройств

Последовательные устройства в виртуальной машине состоят из последовательных контроллеров и последовательных портов. Каждая виртуальная машина имеет ровно один последовательный контроллер, и каждый последовательный контроллер имеет ровно два последовательных порта.

Параметры контроллера последовательного порта нельзя настраивать; Поэтому с ним не связан ни один экземпляр данных. Кроме того, невозможно добавить или удалить последовательные контроллеры или последовательные порты с виртуальной машины. Поэтому экземпляры пула ресурсов для последовательных устройств отсутствуют.

Ниже перечислены классы WMI виртуализации, связанные с последовательными устройствами.

| [**Мсвм \_ SerialPort**](msvm-serialport.md)<br/>                                     | Представляет последовательный порт, связанный с контроллером последовательного порта.<br/>      |
