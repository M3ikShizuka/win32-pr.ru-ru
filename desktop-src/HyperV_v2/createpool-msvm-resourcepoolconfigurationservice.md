# <a name="createpool-method-of-the-msvm_resourcepoolconfigurationservice-class"></a>Метод CreatePool \_ класса Ресаурцепулконфигуратионсервице мсвм

Создает дочерний пул ресурсов. Пул ресурсов будет находиться в той же системе, что и эта служба. Полученный пул будет дочерним пулом.

## <a name="syntax"></a>Синтаксис


```mof
uint32 CreatePool(
  [in]  string               PoolSettings,
  [in]  CIM_ResourcePool REF ParentPools[],
  [in]  string               AllocationSettings[],
  [out] CIM_ResourcePool REF Pool,
  [out] CIM_ConcreteJob  REF Job
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Пулсеттингс* \[ окне\]
</dt> <dd>

Встроенный экземпляр класса [**мсвм \_ ресаурцепулсеттингдата**](msvm-resourcepoolsettingdata.md) , который используется для указания параметров пула, не связанных с распределением.

</dd> <dt>

*Парентпулс* \[ окне\]
</dt> <dd>

Массив ссылок на [**мсвм \_ ResourcePool**](msvm-resourcepool.md) , представляющих пул или пулы, из которых создается новый пул.

</dd> <dt>

*Аллокатионсеттингс* \[ окне\]
</dt> <dd>

Массив из одного или нескольких внедренных экземпляров класса [**мсвм \_ ресаурцеаллокатионсеттингдата**](msvm-resourceallocationsettingdata.md) , которые используются для указания параметров, связанных с выделением пула. Этот массив должен содержать либо один элемент для каждого элемента в массиве *парентпулс* , либо только один элемент. Если этот массив содержит один элемент, а *парентпулс* содержит более одного элемента, *алллокатионсеттингс* указывает общее распределение емкости, которое может быть удовлетворено любым из родительских пулов.

Он используется для ограничения ресурсов, которые могут быть выделены от дочернего пула к пулу, до более низкого предела, чем совокупная емкость, предоставляемая его родительскими элементами. Этот параметр не поддерживается всеми типами ресурсов. Если тип ресурса не поддерживает распределение общей емкости, этот метод возвратит 32770 (не поддерживается).

</dd> <dt>

*Пул* \[ заполняет\]
</dt> <dd>

Ссылка на полученный пул.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Если операция выполняется асинхронно, этот метод возвратит 4096, и этот параметр будет содержать ссылку на объект, производный от [**CIM \_ конкретежоб**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>
