# <a name="getsystemcertificates-method-of-the-msvm_replicationservice-class"></a>Метод Жетсистемцертификатес \_ класса Репликатионсервице мсвм

Получает системные сертификаты в системе узла.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetSystemCertificates(
  [out] string EncodedCertificates[]
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Енкодедцертификатес* \[ заполняет\]
</dt> <dd>

В случае успеха получает сертификаты, доступные в личном хранилище для локального компьютера. Каждая запись представляет собой строку сертификата в кодировке Base 64. Эту строку можно преобразовать в массив байтов, создав объект X509Certificate2.

</dd> </dl>
