# <a name="memory-classes"></a>Классы памяти

Классы памяти моделируют память для виртуальной машины.

Ниже перечислены классы WMI виртуализации, связанные с памятью.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                      | Описание                                                                                                         |
|--------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**Мсвм \_ елементаллокатедфромнуманоде**](msvm-elementallocatedfromnumanode.md)<br/> | Связывает экземпляр выделенного ресурса с физическим узлом NUMA, из которого он был выделен.<br/> |
| [**\_Память мсвм**](msvm-memory.md)<br/>                                             | Представляет память, выделенную для виртуальной машины в данный момент.<br/>                                          |
| [**Мсвм \_ меморисеттингдата**](msvm-memorysettingdata.md)<br/>                       | Представляет настроенное состояние памяти для виртуальной машины.<br/>                                     |
| [**Мсвм \_ NumaNode**](msvm-numanode.md)<br/>                                         | Представляет узел виртуальной системы на основе неоднородного доступа к памяти (NUMA).<br/>                                  |
