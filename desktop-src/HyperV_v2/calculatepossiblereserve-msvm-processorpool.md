# <a name="calculatepossiblereserve-method-of-the-msvm_processorpool-class"></a>Метод Калкулатепоссиблересерве \_ класса Процессорпул мсвм

Этот метод используется для поиска фактического резервирования с входным параметром, равным количеству процессоров виртуальной машины, для которых вычисляется резерв. Этот метод необходим, поскольку резервирование ресурсов процессора сильно зависит от числа процессоров, которые должны быть запланированы параллельно.

## <a name="syntax"></a>Синтаксис


```mof
uint32 CalculatePossibleReserve(
  [in] uint16 ProcessorCount
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ProcessorCount* \[ окне\]
</dt> <dd>

Тип: **UInt16**

Число процессоров виртуальной машины, для которых вычисляется резерв. Максимальное значение этого свойства — число логических процессоров для главного компьютера.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

Объем ресурсов ЦП, которые могут быть зарезервированы для виртуальной машины.

## <a name="remarks"></a>Remarks

Доступ к классу [**\_ процессорпул мсвм**](msvm-processorpool.md) может быть ограничен фильтром контроля учетных записей. Дополнительные сведения см. в разделе [Управление учетными записями пользователей и инструментарий WMI](/windows/desktop/WmiSdk/user-account-control-and-wmi).
