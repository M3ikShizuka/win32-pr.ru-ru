# <a name="msvm_guestcommunicationservice-class"></a>\_Класс мсвм гуесткоммуникатионсервице

Представляет гостевую службу связи. Он используется для связи с гостевым узлом с узла Hyper-V.

Следующий пример синтаксиса — упрощенный MOF-код, который включает все наследуемые свойства.

## <a name="syntax"></a>Синтаксис

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_GuestCommunicationService : Msvm_GuestService
{
};
```

## <a name="members"></a>Члены

Класс **мсвм \_ гуесткоммуникатионсервице** не определяет никаких членов.
