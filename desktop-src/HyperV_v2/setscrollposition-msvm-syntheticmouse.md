# <a name="setscrollposition-method-of-the-msvm_syntheticmouse-class"></a>Метод Сетскроллпоситион \_ класса Синсетикмаусе мсвм

Задает z-координату элемента управления "колесо" для указывающего устройства. Значения, записанные в это свойство, всегда являются относительными смещениями координат.

## <a name="syntax"></a>Синтаксис


```mof
uint32 SetScrollPosition(
  [in] sint32 scrollPositionDelta
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*скроллпоситионделта* \[ окне\]
</dt> <dd>

Тип: **Sint32**

Разность в положении прокрутки.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

Возвращаемое значение, равное нулю, указывает на успешное выполнение. Ненулевое значение указывает на ошибку изменения расположения прокрутки.
