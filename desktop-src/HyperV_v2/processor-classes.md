# <a name="processor-classes"></a>Классы процессоров

Классы процессоров моделируют процессор для виртуальной машины.

Ниже перечислены классы WMI виртуализации, связанные с процессором.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                      | Описание                                                                               |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**\_Процессор мсвм**](msvm-processor.md)<br/>                       | Представляет виртуальный процессор в виртуальной машине.<br/>                         |
| [**Мсвм \_ процессорпул**](msvm-processorpool.md)<br/>               | Статистическая обработка ресурсов процессора, которые могут быть выделены виртуальной машине.<br/> |
| [**Мсвм \_ процессорсеттингдата**](msvm-processorsettingdata.md)<br/> | Представляет параметры виртуального процессора для виртуальной машины.<br/>               |
