# <a name="msvm_guestfileservicecopyfilestoguest-method"></a>Мсвм \_ гуестфилесервице:: копифилестогуест, метод

Копирует файлы с узла Hyper-V на виртуальную машину.

## <a name="syntax"></a>Синтаксис


```C++
uint32 CopyFilesToGuest(
  [in]            string                  CopyFileToGuestSettings[],
  [out]           CIM_ConcreteJob     Job,
  [out, optional] Msvm_CopyFileToGuestJob ParentPools[]
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Копифилетогуестсеттингс* \[ окне\]
</dt> <dd>

Массив экземпляров класса [**мсвм \_ копифилетогуестсеттингдата**](msvm-copyfiletoguestsettingdata.md) , представляющий задание операции гостевой файловой службы.

</dd> <dt>

*Задание* \[ заполняет\]
</dt> <dd>

Необязательный параметр для отслеживания хода выполнения операции, который используется, если метод не может быть выполнен синхронно. Если операция выполняется асинхронно, возвращается значение 4096.

> [!Note]  
> Этот параметр был добавлен в Windows 10.

 

</dd> <dt>

*Парентпулс* \[ out, необязательно\]
</dt> <dd>

Необязательный массив ссылок на [**мсвм \_ копифилетогуестжоб**](msvm-copyfiletoguestjob.md) , которые используются для отслеживания хода выполнения операции. **Копифилестогуест** использует *парентпулс* , если он не может выполняться синхронно. Если операция выполняется асинхронно, возвращается значение 4096.

> [!Note]  
> Этот параметр был удален в Windows 10.

 

</dd> </dl>
