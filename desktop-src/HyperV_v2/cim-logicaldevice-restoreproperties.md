# <a name="restoreproperties-method-of-the-cim_logicaldevice-class"></a>Метод Ресторепропертиес \_ класса CIM

Запрашивает, что устройство повторно устанавливает конфигурацию, настройку и/или сведения о состоянии из резервного хранилища. Цель состоит в том, чтобы записать эти сведения в более раннее время (через метод Савепропертиес) и использовать его для возврата устройства к этому предыдущему условию. Этот метод может не поддерживаться всеми устройствами. Метод должен возвращать 0 в случае успеха, 1, если запрос не поддерживается, и другое значение, если произошла какая-либо другая ошибка. В подклассе можно указать набор возможных кодов возврата, используя квалификатор ValueMap в методе. Строки, в которые содержимое ValueMap является переведенными, могут также быть указаны в подклассе как квалификатор массива Values.

## <a name="syntax"></a>Синтаксис


```mof
uint32 RestoreProperties();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.
