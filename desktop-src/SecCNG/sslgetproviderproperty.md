# <a name="sslgetproviderproperty-function"></a>Функция Сслжетпровидерпроперти

Функция **сслжетпровидерпроперти** извлекает значение указанного свойства поставщика.

## <a name="syntax"></a>Синтаксис


```C++
SECURITY_STATUS WINAPI SslGetProviderProperty(
  _In_    NCRYPT_PROV_HANDLE hSslProvider,
  _In_    LPCWSTR            pszProperty,
  _Out_   PBYTE              ppbOutput,
  _Out_   DWORD              *pcbOutput,
  _Inout_ PVOID              *ppEnumState,
  _In_    DWORD              dwFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хсслпровидер* \[ окне\]
</dt> <dd>

Маркер поставщика [*SSL протокола*](/windows/desktop/SecGloss/s-gly) (SSL), для которого извлекается свойство.

</dd> <dt>

*псзпроперти* \[ окне\]
</dt> <dd>

Указатель на строку в Юникоде, завершающуюся нулем, которая содержит имя извлекаемого свойства.

</dd> <dt>

*ппбаутпут* \[ заполняет\]
</dt> <dd>

Адрес буфера, который получает значение свойства.

Вызывающая функция должна освободить этот буфер, вызвав функцию [**сслфрибуффер**](sslfreebuffer.md) .

</dd> <dt>

*пкбаутпут* \[ заполняет\]
</dt> <dd>

Размер (в байтах) буфера *пбаутпут* .

</dd> <dt>

*ппенумстате* \[ в, out\]
</dt> <dd>

Адрес указателя **void** , который получает сведения о состоянии перечисления, используемые в последующих вызовах этой функции. Эта информация имеет значение только для поставщика SSL и непрозрачна для вызывающего. Поставщик SSL использует эти сведения для определения того, какой элемент находится далее в перечислении. Если переменная, на которую указывает этот параметр, содержит **значение NULL**, перечисление запускается с самого начала.

Вызывающая функция должна освободить эту память путем вызова функции [**сслфрибуффер**](sslfreebuffer.md) .

</dd> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Этот параметр зарезервирован для использования в будущем.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается с ошибкой, она возвращает ноль.

Если функция завершается ошибкой, она возвращает ненулевое значение ошибки.

Возможные коды возврата включают, но не ограничиваются следующим.

| <dl> Не <dt>**превышать \_ Недопустимый 0x80090026L \_ Handle**</dt> <dt></dt> </dl>    | Недопустимый маркер *хсслпровидер* .<br/>                       |
| <dl> Не <dt>**превышать \_ Недопустимый \_ параметр**</dt> <dt>0x80090027L</dt> </dl> | Один из переданных параметров недопустим.<br/>                  |
