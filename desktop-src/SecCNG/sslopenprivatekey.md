# <a name="sslopenprivatekey-function"></a>Функция Сслопенприватекэй

Функция **сслопенприватекэй** открывает маркер [*закрытого ключа*](/windows/desktop/SecGloss/p-gly).

## <a name="syntax"></a>Синтаксис


```C++
SECURITY_STATUS WINAPI SslOpenPrivateKey(
  _In_  NCRYPT_PROV_HANDLE hSslProvider,
  _Out_ NCRYPT_KEY_HANDLE  *phPrivateKey,
  _In_  PCCERT_CONTEXT     pCertContext,
  _In_  DWORD              dwFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хсслпровидер* \[ окне\]
</dt> <dd>

Обработчик для экземпляра поставщика протокола [*SSL протокола*](/windows/desktop/SecGloss/s-gly) (SSL).

</dd> <dt>

*фприватекэй* \[ заполняет\]
</dt> <dd>

Адрес буфера, в который записывается маркер для закрытого ключа.

После завершения использования ключа следует освободить *фприватекэй* , вызвав функцию [**сслфриобжект**](sslfreeobject.md) .

</dd> <dt>

*пцертконтекст* \[ окне\]
</dt> <dd>

Адрес сертификата, из которого получается закрытый ключ.

</dd> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Этот параметр зарезервирован для использования в будущем.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается с ошибкой, она возвращает ноль.

Если функция завершается ошибкой, она возвращает ненулевое значение ошибки.

Возможные коды возврата включают, но не ограничиваются следующим.



| Возвращаемый код и значение                                                                                                                                                       | Описание                                                              |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------|
| <dl> Не <dt>**превышать \_ НЕТ \_**</dt> <dt>0x8009000EL</dt> памяти </dl>         | Недостаточно памяти для выделения необходимых буферов.<br/> |
| <dl> Не <dt>**превышать \_ Недопустимый 0x80090026L \_ Handle**</dt> <dt></dt> </dl>    | Недопустимый маркер *хсслпровидер* .<br/>                       |
| <dl> Не <dt>**превышать \_ Недопустимый \_ параметр**</dt> <dt>0x80090027L</dt> </dl> | Параметр *фприватекэй* или *Пцертконтекст* имеет **значение NULL**.<br/>   |



 

## <a name="remarks"></a>Remarks

Полученный закрытый ключ является частью пары из [*открытого и закрытого ключей*](/windows/desktop/SecGloss/p-gly) в [*сертификате*](/windows/desktop/SecGloss/c-gly). Эта функция просто извлекает закрытый ключ из сертификата, указанного параметром *пцертконтекст* .
