# <a name="sslgetkeyproperty-function"></a>Функция Сслжеткэйпроперти

Функция **сслжеткэйпроперти** извлекает значение именованного свойства SSL для объекта ключа поставщика [*протокола*](/windows/desktop/SecGloss/s-gly) SSL.

## <a name="syntax"></a>Синтаксис


```C++
SECURITY_STATUS WINAPI SslGetKeyProperty(
  _In_  NCRYPT_KEY_HANDLE hKey,
  _In_  LPCWSTR           pszProperty,
  _Out_ PBYTE             ppbOutput,
  _Out_ DWORD             *pcbOutput,
  _In_  DWORD             dwFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*hKey* \[ окне\]
</dt> <dd>

Маркер поставщика SSL.

</dd> <dt>

*псзпроперти* \[ окне\]
</dt> <dd>

Указатель на строку в Юникоде, завершающуюся нулем, которая содержит имя извлекаемого свойства. Это может быть один из стандартных [**идентификаторов свойств хранилища ключей**](key-storage-property-identifiers.md) или идентификатор настраиваемого свойства.

</dd> <dt>

*ппбаутпут* \[ заполняет\]
</dt> <dd>

Указатель на буфер, который получает значение свойства. Вызывающая функция должна освободить этот буфер, вызвав функцию [**сслфрибуффер**](sslfreebuffer.md) .

</dd> <dt>

*пкбаутпут* \[ заполняет\]
</dt> <dd>

Размер (в байтах) буфера *пбаутпут* .

</dd> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Этот параметр зарезервирован для использования в будущем.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается с ошибкой, она возвращает ноль.

Если функция завершается ошибкой, она возвращает ненулевое значение ошибки.

Возможные коды возврата включают, но не ограничиваются следующим.



| Возвращаемый код и значение                                                                                                                                                       | Описание                                             |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------|
| <dl> Не <dt>**превышать \_ Недопустимый 0x80090026L \_ Handle**</dt> <dt></dt> </dl>    | Один из указанных дескрипторов недопустим.<br/>    |
| <dl> Не <dt>**превышать \_ Недопустимый \_ параметр**</dt> <dt>0x80090027L</dt> </dl> | Один из переданных параметров недопустим.<br/> |
