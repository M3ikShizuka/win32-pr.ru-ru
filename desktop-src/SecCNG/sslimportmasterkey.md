# <a name="sslimportmasterkey-function"></a>Функция Сслимпортмастеркэй

Функция **сслимпортмастеркэй** выполняет операцию обмена ключами протокола SSL на [*SSL*](/windows/desktop/SecGloss/s-gly) стороне сервера.

## <a name="syntax"></a>Синтаксис


```C++
SECURITY_STATUS WINAPI SslImportMasterKey(
  _In_  NCRYPT_PROV_HANDLE hSslProvider,
  _In_  NCRYPT_KEY_HANDLE  hPrivateKey,
  _Out_ NCRYPT_KEY_HANDLE  *phMasterKey,
  _In_  DWORD              dwProtocol,
  _In_  DWORD              dwCipherSuite,
  _In_  PNCryptBufferDesc  pParameterList,
  _In_  PBYTE              pbEncryptedKey,
  _In_  DWORD              cbEncryptedKey,
  _In_  DWORD              dwFlags
);
```

## <a name="remarks"></a>Remarks

Эта функция расшифровывает предварительный основной секрет, рассчитывает главный секрет SSL и возвращает в вызывающий объект маркер этого объекта. Затем этот главный ключ можно использовать для получения ключа сеанса SSL и завершения подтверждения SSL.

> [!Note]  
> Эта функция используется при использовании алгоритма обмена ключами [*RSA*](/windows/desktop/SecGloss/r-gly) . Когда используется [*DH*](/windows/desktop/SecGloss/d-gly) , серверный код вызывает [**сслженератемастеркэй**](sslgeneratemasterkey.md) .
