# <a name="sslfreebuffer-function"></a>Функция Сслфрибуффер

Функция **сслфрибуффер** используется для освобождения памяти, выделенной одной из функций поставщика [*SSL протокола*](/windows/desktop/SecGloss/s-gly) (SSL).

## <a name="syntax"></a>Синтаксис


```C++
SECURITY_STATUS WINAPI SslFreeBuffer(
  _In_ PVOID pvInput
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пвинпут* \[ окне\]
</dt> <dd>

Указатель на буфер памяти, который необходимо освободить.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается с ошибкой, она возвращает ноль.

Если функция завершается ошибкой, она возвращает ненулевое значение ошибки.

Возможные коды возврата включают, но не ограничиваются следующим.



| Возвращаемый код и значение                                                                                                                                                       | Описание                                                                  |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------|
| <dl> Не <dt>**превышать \_ Недопустимый \_ параметр**</dt> <dt>0x80090027L</dt> </dl> | Параметр *пдвпровидеркаунт* или *Пппровидерлист* имеет **значение NULL**.<br/> |
