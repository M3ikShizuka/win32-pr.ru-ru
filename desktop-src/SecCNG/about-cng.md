# <a name="about-cng"></a>О CNG

API шифрования: следующее поколение (CNG) — это долгосрочная замена интерфейса [*CryptoAPI*](/windows/desktop/SecGloss/c-gly). CNG разработан для расширения на множестве уровней и шифрования, независимо от поведения.

Дополнительные сведения о функциях CNG см. в разделе [функции CNG](cng-features.md).

## <a name="key-import-and-export"></a>Импорт и экспорт ключей

Дополнительные сведения об импорте и экспорте ключей см. в разделе [Импорт и экспорт](key-import-and-export.md)ключей.

## <a name="data-protection-api-next-generation"></a>API защиты данных: следующее поколение

CNG предоставляет набор интерфейсов API, которые можно использовать для простого шифрования и расшифровки содержимого для участников авторизации на нескольких компьютерах. Дополнительные сведения см. в разделе [CNG DPAPI](cng-dpapi.md).
