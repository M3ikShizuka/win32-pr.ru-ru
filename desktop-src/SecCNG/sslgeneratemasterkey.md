# <a name="sslgeneratemasterkey-function"></a>Функция Сслженератемастеркэй

Функция **сслженератемастеркэй** [*выSSL*](/windows/desktop/SecGloss/s-gly) числит главный секретный ключ протокола SSL.

## <a name="syntax"></a>Синтаксис


```C++
SECURITY_STATUS WINAPI SslGenerateMasterKey(
  _In_  NCRYPT_PROV_HANDLE hSslProvider,
  _In_  NCRYPT_KEY_HANDLE  hPrivateKey,
  _In_  NCRYPT_KEY_HANDLE  hPublicKey,
  _Out_ NCRYPT_KEY_HANDLE  *phMasterKey,
  _In_  DWORD              dwProtocol,
  _In_  DWORD              dwCipherSuite,
  _In_  PNCryptBufferDesc  pParameterList,
  _Out_ PBYTE              pbOutput,
  _In_  DWORD              cbOutput,
  _Out_ DWORD              *pcbResult,
  _In_  DWORD              dwFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хсслпровидер* \[ окне\]
</dt> <dd>

Обработчик для экземпляра поставщика протокола SSL.

</dd> <dt>

*хприватекэй* \[ окне\]
</dt> <dd>

Маркер [*закрытого ключа*](/windows/desktop/SecGloss/p-gly) , используемый в Exchange.

</dd> <dt>

*хпубликкэй* \[ окне\]
</dt> <dd>

Маркер [*открытого ключа*](/windows/desktop/SecGloss/p-gly) , используемый в обмене.

</dd> <dt>

*фмастеркэй* \[ заполняет\]

*двпротокол* \[ окне\]

</dd> <dt>

*двЦиферсуите* \[ окне\]
</dt> <dd>

Один из значений [**идентификатора комплекта шифров поставщика CNG SSL**](https://msdn.microsoft.com/library/Hh971253(v=VS.85).aspx) .

</dd> <dt>

*ппараметерлист* \[ окне\]
</dt> <dd>

Указатель на массив буферов [**нкриптбуффер**](https://msdn.microsoft.com/library/Aa376245(v=VS.85).aspx) , содержащих сведения, используемые в рамках операции обмена ключами. Точный набор буферов зависит от используемого протокола и набора шифров. По крайней мере, список будет содержать буферы, содержащие клиентские и серверные случайные значения.

</dd> <dt>

*пбаутпут* \[ заполняет\]

Если этот параметр имеет **значение NULL**, эта функция возвращает необходимый размер (в байтах) в **DWORD** , на который указывает параметр *пкбресулт* .

> [!Note]  
> Этот буфер используется при выполнении обмена ключами RSA.

</dd> <dt>

*пкбресулт* \[ заполняет\]
</dt> <dd>

Указатель на значение **DWORD** , в котором будет помещено число байтов, записанных в буфер *пбаутпут* .

</dd> <dt>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается с ошибкой, она возвращает ноль.

Если функция завершается ошибкой, она возвращает ненулевое значение ошибки.
