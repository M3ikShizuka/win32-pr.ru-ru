# <a name="sslcomputeeapkeyblock-function"></a>Функция Сслкомпутиапкэйблокк

Функция **сслкомпутиапкэйблокк** вычислит блок ключей, используемый протоколом EAP.

## <a name="syntax"></a>Синтаксис


```C++
SECURITY_STATUS WINAPI SslComputeEapKeyBlock(
  _In_      NCRYPT_PROV_HANDLE hSslProvider,
  _In_      NCRYPT_KEY_HANDLE  hMasterKey,
  _In_      PBYTE              pbRandoms,
  _In_      DWORD              cbRandoms,
  _Out_opt_ PBYTE              pbOutput,
  _In_      DWORD              cbOutput,
  _Out_     DWORD              *pcbResult,
  _In_      DWORD              dwFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хсслпровидер* \[ окне\]
</dt> <dd>

Маркер экземпляра поставщика протокола [*SSL протокола*](/windows/desktop/SecGloss/s-gly) (SSL).

</dd> <dt>

*хмастеркэй* \[ окне\]
</dt> <dd>

Маркер объекта [*главного ключа*](/windows/desktop/SecGloss/m-gly) .

</dd> <dt>

*пбрандомс* \[ окне\]
</dt> <dd>

Указатель на буфер, который содержит сцепление \_ случайных и серверных \_ случайных значений сеанса SSL.

</dd> <dt>

*кбрандомс* \[ окне\]
</dt> <dd>

Длина буфера *пбрандомс* в байтах.

</dd> <dt>

*пбаутпут* \[ out, необязательно\]
</dt> <dd>

Адрес буфера, который получает большой двоичный объект ключа. Параметр *кбаутпут* содержит размер этого буфера. Если этот параметр имеет **значение NULL**, эта функция поместит требуемый размер (в байтах) в **DWORD** , на который указывает параметр *пкбресулт* .

</dd> <dt>

*кбаутпут* \[ окне\]
</dt> <dd>

Длина буфера *пбаутпут* в байтах.

</dd> <dt>

*пкбресулт* \[ заполняет\]
</dt> <dd>

Указатель на значение **DWORD** , определяющее длину хэша в байтах, записанных в буфер *пбаутпут* .

</dd> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Установите **флаг NCRYPT \_ SSL \_ - \_ сервера** , чтобы указать, что это вызов сервера.

</dd> </dl>

Если функция завершается с ошибкой, она возвращает ноль.

Если функция завершается ошибкой, она возвращает ненулевое значение ошибки.
