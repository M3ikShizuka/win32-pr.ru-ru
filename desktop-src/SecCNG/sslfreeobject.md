# <a name="sslfreeobject-function"></a>Функция Сслфриобжект

Функция **сслфриобжект** освобождает ключ, хэш или объект поставщика.

## <a name="syntax"></a>Синтаксис


```C++
SECURITY_STATUS WINAPI SslFreeObject(
  _In_ NCRYPT_HANDLE hObject,
  _In_ DWORD         dwFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хобжект* \[ окне\]
</dt> <dd>

Описатель объекта для освобождения.

</dd> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Этот параметр зарезервирован для использования в будущем.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается с ошибкой, она возвращает ноль.

Если функция завершается ошибкой, она возвращает ненулевое значение ошибки.

Возможные коды возврата включают, но не ограничиваются следующим.

|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------|
| <dl> Не <dt>**превышать \_ Недопустимый 0x80090026L \_ Handle**</dt> <dt></dt> </dl>    | Недопустимый внутренний маркер.<br/>  |
| <dl> <dt>**Состояние \_ Недопустимый 0xC0000008L \_ Handle**</dt> <dt></dt> </dl> | Предоставленный маркер недопустим.<br/> |
