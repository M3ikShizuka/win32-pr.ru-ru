# <a name="protection-providers"></a>Поставщики защиты

Начиная с Windows 8, корпорация Майкрософт приступила к распространению поставщиков, позволяющих безопасно обмениваться зашифрованными секретными данными и сообщениями на разных компьютерах. В настоящее время существует два поставщика защиты ключей. Поставщик защиты ключей (Майкрософт) позволяет защищать содержимое в группе в Active Directoryном лесу. Поставщик защиты ключей клиентов Майкрософт позволяет защищать содержимое с помощью набора учетных данных веб-служб.

Правильный выбор предохранителя автоматически выбирается, когда функция [**нкрипткреатепротектиондескриптор**](/windows/desktop/api/NCryptprotect/nf-ncryptprotect-ncryptcreateprotectiondescriptor) анализирует строку правила дескриптора защиты, которую вы указали как входные данные. Поставщик защиты ключей (Майкрософт) выбирается для строк правил, начинающихся с SID, SDDL и LOCAL. Поставщик защиты ключей клиентов Майкрософт анализирует строки правил, которые начинаются с УЧЕТных данных веб. Дополнительные сведения о строках правил см. в разделе [дескрипторы защиты](protection-descriptors.md).

> [!Note]  
> В настоящее время настраиваемые поставщики не разрешены. [CNG DPAPI](cng-dpapi.md)
