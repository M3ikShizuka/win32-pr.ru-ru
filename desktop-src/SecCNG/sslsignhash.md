# <a name="sslsignhash-function"></a>Функция Сслсигнхаш

Функция **сслсигнхаш** подписывает [*хэш*](/windows/desktop/SecGloss/h-gly) с помощью указанного [*закрытого ключа*](/windows/desktop/SecGloss/p-gly). Процесс подписывания выполняется на сервере.

## <a name="syntax"></a>Синтаксис


```C++
SECURITY_STATUS WINAPI SslSignHash(
  _In_  NCRYPT_PROV_HANDLE hSslProvider,
  _In_  NCRYPT_KEY_HANDLE  hPrivateKey,
  _In_  PBYTE              pbHashValue,
  _In_  DWORD              cbHashValue,
  _Out_ PBYTE              pbSignature,
  _In_  DWORD              cbSignature,
  _Out_ DWORD              *pcbResult,
  _In_  DWORD              dwFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хсслпровидер* \[ окне\]
</dt> <dd>

Обработчик для экземпляра поставщика протокола [*SSL протокола*](/windows/desktop/SecGloss/s-gly) (SSL).

</dd> <dt>

*хприватекэй* \[ окне\]
</dt> <dd>

Маркер закрытого ключа, используемый для подписи хэша.

</dd> <dt>

*пбхашвалуе* \[ окне\]
</dt> <dd>

Указатель на буфер, содержащий хэш для подписи.

</dd> <dt>

*кбхашвалуе* \[ окне\]
</dt> <dd>

Размер (в байтах) буфера *пбхашвалуе* .

</dd> <dt>

*пбсигнатуре* \[ заполняет\]
</dt> <dd>

Адрес буфера, который получает сигнатуру хэша. Параметр *кбсигнатуре* содержит размер этого буфера. Чтобы определить требуемый размер буфера, присвойте параметру *Пбсигнатуре* **значение NULL**. Требуемый размер буфера будет возвращен в параметре *пкбресулт* .

</dd> <dt>

*кбсигнатуре* \[ окне\]
</dt> <dd>

Размер (в байтах) буфера *пбсигнатуре* .

</dd> <dt>

*пкбресулт* \[ заполняет\]
</dt> <dd>

Указатель на значение, которое после завершения содержит фактическое число байтов, записанных в буфер *пбсигнатуре* .

</dd> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Этот параметр зарезервирован для использования в будущем.

</dd> </dl>
