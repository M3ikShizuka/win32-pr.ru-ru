# <a name="windows-networking-callbacks"></a>Обратные вызовы в сети Windows

В сети Windows используются следующие прототипы функций.



| Прототип функции                                 | Описание                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**пфнжетпрофилепас**](/previous-versions/windows/desktop/legacy/aa385358(v=vs.85))     | Прототип функции, получающей путь к профилю пользователя.                                                            |
| [**пфнпроцессполиЦиес**](/previous-versions/windows/desktop/legacy/aa385360(v=vs.85))   | Прототип функции, возвращающей сведения о политике учетных записей пользователей, связанные с конкретным процессом Windows. |
| [**пфнреконЦилепрофиле**](/previous-versions/windows/desktop/legacy/aa385364(v=vs.85)) | Прототип функции, который согласовывает локальный профиль с соответствующим глобальным профилем.                        |
