# <a name="volume-shadow-copy-service-overview"></a>Обзор служба теневого копирования томов

Служба теневого копирования томов (VSS) — это набор API-интерфейсов COM, который реализует платформу, позволяющую выполнять резервное копирование томов, пока приложения в системе продолжают записывать данные на тома.

Служба VSS обеспечивает согласованный интерфейс, который обеспечивает координацию между пользовательскими приложениями, которые обновляют данные на дисках ([*модулях записи*](vssgloss-w.md)) и резервными копиями приложений ([*запрашивающими*](vssgloss-r.md)стороны).
