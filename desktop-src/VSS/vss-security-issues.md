# <a name="vss-security-issues"></a>Проблемы безопасности VSS

Инфраструктура VSS внутренне основана на модели COM и поэтому модули записи и запрашивающие объекты должны взаимодействовать с системой безопасности на основе COM. Безопасность COM предоставляет механизмы и рекомендации, которые должен учитывать разработчик VSS:

-   [Вопросы безопасности для модулей записи](security-considerations-for-writers.md)
-   [Вопросы безопасности для запрашивающих стороны](security-considerations-for-requestors.md)
