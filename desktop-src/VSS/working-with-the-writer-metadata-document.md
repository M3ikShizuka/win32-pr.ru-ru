# <a name="working-with-the-writer-metadata-document"></a>Работа с документом метаданных модуля записи

Каждый модуль записи создает документ метаданных модуля записи. Приложение резервного копирования использует этот документ для получения сведений об этом модуле записи, данных, которыми он владеет, и восстановлении этих данных. После того как модуль записи создаст его, документ метаданных модуля записи будет доступен только для чтения в приложении резервного копирования.

Для использования документа метаданных модуля записи необходимо понимать следующее:

-   [Жизненный цикл документа метаданных модуля записи](writer-metadata-document-life-cycle.md)
-   [Содержимое документа метаданных модуля записи](writer-metadata-document-contents.md)
