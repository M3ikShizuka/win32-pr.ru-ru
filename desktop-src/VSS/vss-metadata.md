# <a name="vss-metadata"></a>Метаданные VSS

Модули записи и запрашивающие объекты сохраняют сведения, необходимые для операции резервного копирования или восстановления в собственных документах метаданных.

Эти документы, [*документ метаданных модуля записи*](vssgloss-w.md) и [*документ с компонентами резервного копирования*](vssgloss-b.md), соответственно, используются в качестве базиса для модуля записи и взаимодействия с инициатором и координации во время действий резервного копирования и восстановления.

Метаданные представлены в формате XML с помощью собственной схемы. Метаданные можно копировать на диск, на ленту или на любое другое запоминающее устройство. Он всегда должен быть сохранен на носителе резервных копий во время операции резервного копирования и должен быть получен с этого носителя в ходе операции восстановления.

**Внимание!** Конкретные сведения о формате и схеме предназначены только для системного использования. Разработчики не должны пытаться изменить или напрямую использовать XML-представление метаданных.

Модули записи и инициаторы поставляются с различными методами запросов и Set для доступа и изменения компонентов резервного копирования и документов метаданных модуля записи:

-   [Работа с документом метаданных модуля записи](working-with-the-writer-metadata-document.md)
-   [Работа с документом компонентов резервного копирования](working-with-the-backup-components-document.md)
-   [Компоненты метаданных VSS](vss-metadata-components.md)
