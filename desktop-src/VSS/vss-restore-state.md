# <a name="vss-restore-state"></a>Состояние восстановления VSS

Во время операции восстановления инициатор запроса может использовать метод [**ивссбаккупкомпонентс:: сетресторестате**](/windows/desktop/api/VsBackup/nf-vsbackup-ivssbackupcomponents-setrestorestate) для определения типа выполняемой операции восстановления. Однако большинству операций восстановления не требуется переопределять тип восстановления по умолчанию (VSS \_ ртипе \_ undefine). Модули записи должны считать этот тип восстановления, как если бы он был \_ РТИПЕ VSS \_ по \_ копированию. Дополнительные сведения о значениях типов восстановления см. в разделе Перечисление [**\_ \_ типов восстановления VSS**](/windows/desktop/api/Vss/ne-vss-vss_restore_type) .
