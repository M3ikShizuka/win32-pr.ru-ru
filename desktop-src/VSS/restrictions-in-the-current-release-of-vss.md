# <a name="restrictions-in-windows-server-2003"></a>ограничения в Windows Server 2003

некоторые функции, запланированные для выпуска VSS Windows Server 2003, не полностью поддерживаются:

-   Во время операций резервного копирования несколько экземпляров заданного класса модуля записи разрешаются только в том случае, если только один из этих экземпляров имеет состояние восстановления модуля записи (см. раздел [**VSS \_ вритерресторе \_ enum**](/windows/desktop/api/VsWriter/ne-vswriter-vss_writerrestore_enum)), кроме VSS \_ WRE \_ никогда. Если это условие выполнено, все экземпляры модуля записи будут полностью участвовать во время резервного копирования, создавая документы метаданных модуля записи и участвуя в обработке событий.
-   Во время операций восстановления только один модуль записи получит события Restore, созданные инициатором запроса. Если в нескольких экземплярах класса модуля записи для состояния восстановления модуля записи задано значение, отличное от \_ WRE VSS \_ , то только один экземпляр получит события Restore. Экземпляр, который получит эти данные, является неопределенным.
