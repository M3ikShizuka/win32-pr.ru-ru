# <a name="performservervalidation-peapextensionstype-element-v1-schema"></a>Элемент Перформсервервалидатион (Пеапекстенсионстипе) (схема v1)

Элемент **перформсервервалидатион (пеапекстенсионстипе)** указывает, выполняется ли проверка сервера.

``` syntax
<xs:element
    minOccurs="0"
    ref="extendedPeap:PerformServerValidation"
 />
```

Элемент определяется элементом [**пеапекстенсионстипе**](mspeapconnectionpropertiesv1schema-peapextensionstype-complextype.md) .

## <a name="remarks"></a>Remarks

Элемент **перформсервервалидатион** является необязательным.

## <a name="requirements"></a>Requirements (Требования)



| Роль | Минимальная поддерживаемая версия ОС |
|------|------------------------------|
| Клиент<br/> | только Windows 7 \[ настольных приложений\]<br/>              |
| Сервер<br/> | Windows \[Только для настольных приложений сервера 2008 R2\]<br/> |
