# <a name="eaphost-peer-method-api-reference"></a>Справочник по API однорангового метода EAPHost

Документация по API однорангового метода EAPHost состоит из следующих разделов.

-   [Функции настройки однорангового метода EAPHost](eaphost-peer-method-configuration-functions.md)
-   [Run-Time функции однорангового метода EAPHost](eaphost-peer-method-run-time-functions.md)
-   [Перечисления одноранговых методов EAPHost](eap-host-peer-method-enumerations.md)
-   [Структуры одноранговых методов EAPHost](eap-host-peer-method-structures.md)

Обратите внимание, что эти элементы должны быть реализованы в согласованной библиотеке DLL автором метода однорангового узла EAP. Библиотека DLL с реализацией этих элементов API загружается и вызывается клиентской версией EAPHost.
