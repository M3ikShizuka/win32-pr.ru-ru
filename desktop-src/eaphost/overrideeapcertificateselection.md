# <a name="overrideeapcertificateselection"></a>оверридиапцертификатеселектион

Раздел реестра Оверридиапцертификатеселектион определяет, будет ли клиент переопределять поведение выбора сертификата смарт-карты по умолчанию и предоставлять пользователю больше сертификатов для выбора.

## <a name="registry-entry"></a>Запись реестра

```
HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services\Rasman\PPP\EAP\13
   OverrideEAPCertificateSelection = value
```

## <a name="remarks"></a>Комментарии

Это **\_ двоичное значение reg** .



| Значение | Описание                                                                                                                                                                                          |
|-------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0x000 | Не переопределяйте поведение по умолчанию.                                                                                                                                                                |
| 0x001 | Выберите последний подключенный сертификат из смарт-карты.                                                                                                                                            |
| 0x010 | Показывает все сертификаты в пользовательском интерфейсе выбора сертификата из смарт-карты.                                                                                                                          |
| 0x100 | Показывает все сертификаты в пользовательском интерфейсе выбора сертификата из реестра.                                                                                                                            |
| 0x101 | Показывает все сертификаты в пользовательском интерфейсе выбора сертификата из реестра и последнего присоединенного сертификата со смарт-карты. Отображает последний присоединенный сертификат из смарт-карты, как выбрано. |
| 0x110 | Показывает все сертификаты в пользовательском интерфейсе выбора сертификата из реестра и смарт-карты.                                                                                                         |
