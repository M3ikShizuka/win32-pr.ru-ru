# <a name="type-baseeapparameters-element"></a>Элемент Type (Басиаппараметерс)

Элемент **Type (басиаппараметерс)** определяет элемент-заполнитель для выбранного типа метода и конфигурации конкретного метода.

``` syntax
<xs:element name="Type"
    type="integer"
 />
```

Элемент **Type** определяется сложным типом [**басиаппараметерс**](baseeapconnectionpropertiesv1schema-baseeapparameters-complextype.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Только для \[ классических приложений Windows Vista\]<br/>       |
| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2008\]<br/> |
