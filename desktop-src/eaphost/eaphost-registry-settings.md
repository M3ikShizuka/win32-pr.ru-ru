# <a name="eaphost-registry-settings"></a>Параметры реестра EAPHost

Значения реестра в следующих разделах реестра управляют аспектами функций EAPHost.



| Подраздел                                                                 | Описание                                                                                                                                                              |
|------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [бипасснеготиатион](bypassnegotiation.md)                             | Определяет, происходит ли согласование возможностей между сервером RAS и клиентом.<br/>                                                                              |
| [AssumePhase2Fragmentation](assumephase2fragmentation.md)             | Определяет, что сервер и клиент RAS предполагают фрагментацию этапа 2.<br/>                                                                                         |
| [оверридиапцертификатеселектион](overrideeapcertificateselection.md) | Определяет, будет ли клиент переопределять поведение при выборе сертификата смарт-карты по умолчанию и предоставлять пользователю больше сертификатов для выбора.<br/> |
| [пеапсерверусеаллпурпосецерт](peapserveruseallpurposecert.md)         | Определяет, используются ли для аутентификации PEAP сертификаты всех целей.<br/>                                                                                      |
| [пеапсерверакцепталлпурпосецерт](peapserveracceptallpurposecert.md)   | Определяет, принимаются ли сертификаты всех целей для проверки подлинности PEAP.<br/>                                                                                  |
| [тлссерверусеаллпурпосецерт](tlsserveruseallpurposecert.md)           | Определяет, используются ли сертификаты всех целей для проверки подлинности EAP-TLS.<br/>                                                                                   |
| [тлссерверакцепталлпурпосецерт](tlsserveracceptallpurposecert.md)     | Определяет, принимаются ли сертификаты всех целей для проверки подлинности EAP-TLS.<br/>                                                                               |
