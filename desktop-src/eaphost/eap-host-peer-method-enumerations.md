# <a name="eaphost-peer-method-enumerations"></a>Перечисления одноранговых методов EAPHost

Ниже приведены перечисления API одноранговых методов EAPHost.



| Перечисление                                                        | Описание                                                                                                                |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [**еапкредентиалтипе**](/windows/desktop/api/eaptypes/ns-eaptypes-eapcredential)                         | Определяет типы учетных данных EAP и соответствующие учетные данные.                                                          |
| [**еаппирмесодреспонсеактион**](/windows/win32/api/eapauthenticatoractiondefine/ne-eapauthenticatoractiondefine-eappeermethodresponseaction) | Определяет набор действий, которые средство проверки подлинности EAP может указывать на одностороннее или одноранговый метод EAP во время проверки подлинности.     |
| [**еаппирмесодресултреасон**](/windows/win32/api/eapauthenticatoractiondefine/ne-eapauthenticatoractiondefine-eappeermethodresultreason)     | Определяет набор результатов сеанса проверки подлинности EAP, возвращенный методом EAP Authenticator для однорангового метода EAP. |
