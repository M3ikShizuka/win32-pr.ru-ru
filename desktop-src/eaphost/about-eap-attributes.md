# <a name="eap-attributes"></a>Атрибуты EAP

Атрибут EAP — это структура [**\_ атрибута EAP**](/windows/desktop/api/eaptypes/ns-eaptypes-eap_attribute) , которая содержит предварительно определенный тип данных, связанных с сеансом проверки подлинности. Атрибуты используются для обмена данными между методами EAP и отправителей запросов или между методами и проверками подлинности EAP. Реализации метода EAP в одноранговых сетях и средствах проверки подлинности могут обмениваться этими атрибутами по сети.

Полный список поддерживаемых типов атрибутов представлен в разделе [**\_ \_ тип атрибута EAP**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_attribute_type).

## <a name="attributes-consumed-by-supplicants"></a>Атрибуты, используемые отправителей запросов

Следующие типы атрибутов используются 802.1 X отправителей запросов.

-   **еатвендорспеЦифик**

Следующие типы атрибутов используются PPP Client отправителей запросов.

-   **еатминимум**
-   **еатеаптлв**

Следующие типы атрибутов используются PPP Server отправителей запросов.

-   **еатусернаме**
-   **еатреплимессаже**
-   **еатстате**
-   **еатсессионтимеаут**
-   **еатеапмессаже**

## <a name="attributes-exported-by-methods"></a>Атрибуты, экспортированные методами

Методы EAP могут экспортировать следующие типы атрибутов.

-   **еатклеартекстпассворд**
-   **еаткуарантинесох**
-   **еатмесодид**

Следующий тип атрибута экспортируется методами [TLS](/previous-versions/windows/embedded/ms885336(v=msdn.10)) (EAP-TLS) протокола EAP.

-   **еатцертификатеоид**

Следующие типы атрибутов экспортируются методами [протокола проверки подлинности Microsoft Challenge](/previous-versions/windows/embedded/ms899190(v=msdn.10)) (MS-CHAPv2) версии 2,0.

-   **еатусернаме**
-   **еаткредентиалсчанжед**

Сервер политики сети (NPS) использует следующий тип атрибута.

-   **еатцертификатеоид**

Следующие типы атрибутов экспортируются с помощью методов [протокола PEAP](/previous-versions/windows/embedded/ms899190(v=msdn.10)) .

-   **еатусернаме**
-   **еатпеапембеддедеаптипеид**
-   **еатпеапфастроамедсессион**
-   **еаткуарантинесох**
