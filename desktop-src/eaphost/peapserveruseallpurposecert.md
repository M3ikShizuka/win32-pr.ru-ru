# <a name="peapserveruseallpurposecert"></a>пеапсерверусеаллпурпосецерт

Раздел реестра Пеапсерверусеаллпурпосецерт определяет, используются ли для проверки подлинности PEAP сертификаты всех целей.

## <a name="registry-entry"></a>Запись реестра

```
HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services\Rasman\PPP\EAP\25
   PeapServerUseAllPurposeCert = value
```

## <a name="remarks"></a>Remarks

Это значение **reg \_ DWORD** .



| Значение        | Описание                                                                                                      |
|--------------|------------------------------------------------------------------------------------------------------------------|
| 1            | Для проверки подлинности PEAP выбираются сертификаты всех целей в хранилище сертификатов клиента или сервера.     |
| Другие значения | Сертификаты всех целей в хранилище сертификатов клиента или сервера не выбраны для проверки подлинности PEAP. |



 

Если это значение реестра отсутствует, для аутентификации PEAP выбираются сертификаты всех целей в хранилище сертификатов клиента или сервера.
