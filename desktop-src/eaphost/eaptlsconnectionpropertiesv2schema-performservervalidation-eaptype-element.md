# <a name="tlsextensions-v2-schema"></a>Тлсекстенсионс (схема v2)

Элемент **тлсекстенсионс (тлсекстенсионстипе)** обеспечивает будущие улучшения схемы.

``` syntax
<xs:element name="TLSExtensions"
    type="TLSExtensionsType"
 />
```

## <a name="remarks"></a>Комментарии

Элемент **тлсекстенсионс** является необязательным.
