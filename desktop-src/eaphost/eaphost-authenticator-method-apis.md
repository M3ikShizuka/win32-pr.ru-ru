Документация по API метода проверки подлинности EAPHost состоит из следующих разделов.

-   [Функции метода проверки подлинности EAPHost](eap-host-authenticator-method-functions.md)
-   [Перечисления метода проверки подлинности EAPHost](eap-host-authenticator-method-enumerations.md)
-   [Структуры методов средства проверки подлинности EAPHost](eap-host-authenticator-method-structures.md)

Обратите внимание, что эти элементы должны быть реализованы в согласованной библиотеке DLL автором метода проверки подлинности EAP. Библиотека DLL с реализацией этих элементов API загружается и вызывается серверной версией EAPHost.
