# <a name="common-eaphost-api-enumerations"></a>Распространенные перечисления API EAPHost

Следующие перечисления являются общими для всех технологий API EAPHost.



| Перечисление                                                                  | Описание                                                                                   |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**\_тип атрибута \_ EAP**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_attribute_type)                           | Определяет набор возможных типов атрибутов EAP, доступных для сущности, проверяющей подлинность.        |
| [**\_ \_ Тип входного \_ поля \_ конфигурации EAP**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_config_input_field_type)       | Определяет набор возможных типов полей ввода, доступных при запросе учетных данных у пользователя. |
| [**\_ \_ тип свойства метода \_ EAP**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_type)              | Определяет набор возможных свойств метода EAP.                                            |
| [**\_ \_ \_ тип значения свойства метода \_ EAP**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_value_type) | Определяет набор возможных типов данных для свойств метода.                             |
| [**еапкоде**](/windows/win32/api/eapmethodtypes/ne-eapmethodtypes-eapcode)                                                   | Определяет набор типов пакетов EAP.                                                          |
