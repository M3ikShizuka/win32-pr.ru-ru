# <a name="tlsserveracceptallpurposecert"></a>тлссерверакцепталлпурпосецерт

Раздел реестра Тлссерверакцепталлпурпосецерт определяет, принимаются ли сертификаты всех целей для проверки подлинности EAP-TLS.

## <a name="registry-entry"></a>Запись реестра

```
HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services\Rasman\PPP\EAP\13
   TlsServerAcceptAllPurposeCert = value
```

## <a name="remarks"></a>Комментарии

Это значение **reg \_ DWORD** .



| Значение        | Описание                                                                                                 |
|--------------|-------------------------------------------------------------------------------------------------------------|
| 1            | Сервер и клиент принимают сертификаты всех целей, отправленные другой стороной для проверки подлинности EAP-TLS.       |
| Другие значения | Сервер и клиент не принимают сертификаты всех целей, отправленные другой стороной для проверки подлинности EAP-TLS |



 

Если это значение реестра отсутствует, сервер и клиент принимают сертификаты всех целей, отправленные другой стороной для проверки подлинности EAP-TLS.
