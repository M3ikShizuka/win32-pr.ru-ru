# <a name="about-eaphost"></a>Об EAPHost

EAPHost реализует среду (EAP) конечного автомата и протокола EAP в соответствии с [RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063).

EAPHost поддерживает [защиту доступа к сети](/windows/desktop/NAP/network-access-protection-start-page) на основе EAP (NAP).
