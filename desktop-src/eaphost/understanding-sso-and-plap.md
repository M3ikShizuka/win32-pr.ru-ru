# <a name="sso-and-plap"></a>Единый вход и PLAP

API единого Sign-On (SSO) представляет набор методов, используемых для безопасного получения учетных данных метода EAP для учетной записи сетевого пользователя или компьютера без необходимости создавать несколько экземпляров пользовательского интерфейса.

> [!Note]  
> Термин поставщик предварительных прав доступа (PLAP) является устаревшим. Если в коде или документации встречается "поставщик предварительного доступа к входу" или "PLAP", замените его на "единый вход" или "SSO" соответственно.
