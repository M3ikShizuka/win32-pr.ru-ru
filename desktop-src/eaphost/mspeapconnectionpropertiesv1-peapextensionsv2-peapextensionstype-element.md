# <a name="peapextensionsv2-peapextensionstype-element-v1-schema"></a>Элемент PeapExtensionsV2 (Пеапекстенсионстипе) (схема v1)

Элемент **PeapExtensionsV2 (пеапекстенсионстипе)** обеспечивает будущие улучшения схемы.

``` syntax
<xs:element name="PeapExtensionsV2"
    type="extendedPeap:PeapExtensionsV2"
 />
```

Элемент **PeapExtensionsV2** определяется элементом [**пеапекстенсионстипе**](mspeapconnectionpropertiesv1schema-peapextensionstype-complextype.md) .

## <a name="remarks"></a>Комментарии

Элемент **PeapExtensionsV2** является необязательным.
