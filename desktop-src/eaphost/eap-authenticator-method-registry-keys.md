# <a name="eap-authenticator-method-registry-values"></a>Значения реестра для метода средства проверки подлинности EAP

Для методов проверки подлинности EAP требуются определенные значения реестра.

## <a name="eap-authenticator-method-dll-paths"></a>Пути DLL метода проверки подлинности EAP

Следующий путь задает расположение реестра для обычных библиотек DLL метода проверки подлинности EAP.

**HKLM \\ System \\ CCS \\ Services \\ \\ методы EAPHost \\ *&lt; аусорид &gt;* \\ * &lt; еаптипеид&gt;***

Например, путь регистрации установки метода средства проверки подлинности EAP по адресу **аусорид** "311" (указывающий, что "Microsoft" является автором) и **еаптипеид** "40", выглядит следующим образом.

**HKLM \\ System \\ CCS \\ Services \\ \\ методы EAPHost \\ 311 \\ 40**

Следующий путь задает расположение реестра для расширенных библиотек DLL метода проверки подлинности EAP.

**HKLM \\ System \\ CCS \\ Services \\ EAPHost \\ Methods \\ *&lt; аусорид &gt;* \\ 254 \\ *&lt; VendorID &gt;* \\ * &lt; вендортипе&gt;***

Например, путь регистрации установки метода средства проверки подлинности EAP по адресу **аусорид** "311" (указывающий на то, что "Microsoft" является автором), **ид поставщика** "311" и **еаптипеид** "40", выглядит следующим образом.

**HKLM \\ System \\ CCS \\ Services \\ Eaphost \\ methods \\ 311 \\ 254 \\ 311 \\ 40**

> [!Note]  
> Дополнительные сведения о выделении типов методов EAP см. в разделе 6,2 статьи [RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84016).

 

## <a name="registry-values"></a>Значения реестра

Следующие значения реестра для метода проверки подлинности являются обязательными.

-   [аусентикатордллпас](#authenticatordllpath)
-   [аусентикаторфриендлинаме](#authenticatorfriendlyname)

Кроме указанных выше значений реестра, рекомендуется использовать следующий параметр реестра для метода проверки подлинности.

-   [Свойства](#properties)

Остальные параметры реестра для метода проверки подлинности являются необязательными.

-   [конфигклсид](#configclsid)
-   [стандалонесуппортед](#standalonesupported)

## <a name="authenticatordllpath"></a>аусентикатордллпас



| Постоянное значение | аусентикатордллпас                                                                                          |
|----------------|---------------------------------------------------------------------------------------------------------------|
| Тип           | \_распаковать \_ SZ                                                                                               |
| Описание    | Путь к библиотеке DLL метода проверки подлинности EAP. Например,% SystemRoot% \\ system32 \\ &lt; имя \_ DLL. \_ &gt; DLL. |



 

## <a name="authenticatorfriendlyname"></a>аусентикаторфриендлинаме



| Постоянное значение | аусентикаторфриендлинаме                                                          |
|----------------|------------------------------------------------------------------------------------|
| Тип           | REG \_ SZ                                                                            |
| Описание    | Строка, содержащая понятное (отображаемое) имя метода проверки подлинности EAP. |



 

## <a name="configclsid"></a>конфигклсид



| Постоянное значение | конфигклсид                                                                                                                           |
|----------------|---------------------------------------------------------------------------------------------------------------------------------------|
| Тип           | REG \_ SZ                                                                                                                               |
| Описание    | Строка, содержащая GUID класса конфигурации для этого метода проверки подлинности в формате {XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX} |



 

## <a name="properties"></a>Свойства



| Постоянное значение | Свойства                                                                                                                                                  |
|----------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Тип           | REG \_ DWORD                                                                                                                                                  |
| Описание    | Биты в DWORD задаются для указания поддержки свойства. Список поддерживаемых значений см. в разделе [**Свойства метода EAP**](eap-method-properties.md). |



 

## <a name="standalonesupported"></a>стандалонесуппортед



| Постоянное значение | стандалонесуппортед                                             |
|----------------|-----------------------------------------------------------------|
| Тип           | REG \_ DWORD                                                      |
| Описание    | 0, если это автономный метод проверки подлинности; 1, если это не так. |
