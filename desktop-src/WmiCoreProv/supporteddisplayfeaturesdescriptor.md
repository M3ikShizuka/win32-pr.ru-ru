# <a name="supporteddisplayfeaturesdescriptor-class"></a>Класс Суппортеддисплайфеатуресдескриптор

**Суппортеддисплайфеатуресдескриптор** представляет поддерживаемые функции отображения монитора. Сведения в этом классе соответствуют данным в определении входного видео для расширенного расширенного представления идентификационных данных по стандарту (E-EDID).

## <a name="syntax"></a>Синтаксис

``` syntax
class SupportedDisplayFeaturesDescriptor
{
  boolean ActiveOffSupported;
  uint8   DisplayType;
  boolean GTFSupported;
  boolean HasPreferredTimingMode;
  boolean sRGBSupported;
  boolean StandbySupported;
  boolean SuspendSupported;
};
```

Поддержка активных и очень низких возможностей. При получении сигнала о времени, который находится за пределами объявленного активного диапазона, экран потребляет меньше энергии. Если сигнал времени возвращается к нормальному диапазону работы, отображение будет отменено до обычной операции. Примеры повременных сигналов за пределами обычного диапазона работы не являются сигналами синхронизации или отсутствием сигнала "DE".
