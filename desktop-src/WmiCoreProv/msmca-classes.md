# <a name="msmca-classes"></a>Классы МСМКА

Классы архитектуры проверки машинного по Майкрософт (МСМКА) представляют собой набор классов WMI, которые предоставляют архитектуру проверки компьютера (MCA). Уровень абстрагирования системы (SAL) предоставляет все события, о которых сообщается в классе МСМКА. Корпорация Intel разрабатывает и владеет MCA.



| Класс                                                                               | Описание                                                                                            |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**Мсмкаевент \_ кпуеррор**](msmcaevent-cpuerror.md)                                 | Представляет событие ошибки ЦП.                                                                          |
| [**\_Заголовок мсмкаевент**](msmcaevent-header.md)                                     | Представляет общий заголовок, используемый всеми классами Мсмкаевент.                                          |
| [**Мсмкаевент \_ инвалидеррор**](msmcaevent-invaliderror.md)                         | Представляет недопустимую ошибку в архитектуре проверки памяти (MCA).                                            |
| [**Мсмкаевент \_ мемореррор**](msmcaevent-memoryerror.md)                           | Представляет событие ошибки памяти MCA.                                                                  |
| [**Мсмкаевент \_ меморипажеремовед**](msmcaevent-memorypageremoved.md)               | Указывает, что операционная система удалила физическую страницу памяти.                                 |
| [**Мсмкаевент \_ пЦибусеррор**](msmcaevent-pcibuserror.md)                           | Представляет ошибку шины PCI MCA.                                                                       |
| [**Мсмкаевент \_ пЦикомпонентеррор**](msmcaevent-pcicomponenterror.md)               | Представляет ошибку компонента MCA PCI.                                                                 |
| [**Мсмкаевент \_ платформспеЦифицеррор**](msmcaevent-platformspecificerror.md)       | Представляет ошибку, относящуюся к платформе MCA.                                                             |
| [**Мсмкаевент \_ смбиосеррор**](msmcaevent-smbioserror.md)                           | Представляет ошибку BIOS системы MCA.                                                                   |
| [**Мсмкаевент \_ свитчтокмкполлинг**](msmcaevent-switchtocmcpolling.md)             | Указывает, что исправленная обработка проверки компьютера переключена на опрос.                            |
| [**Мсмкаевент \_ системевентеррор**](msmcaevent-systemeventerror.md)                 | Представляет ошибку системного события MCA.                                                                  |
| [**мсмкаинфо**](msmcainfo.md)                                                      | Абстрактный базовый класс, из которого производятся все классы данных поставщика (архитектура проверки компьютеров). |
| [**\_Запись мсмкаинфо**](msmcainfo-entry.md)                                         | Представляет запись об исправленной проверке компьютера MCA (CMC) или исправленную ошибку платформы (CPE). |
| [**Мсмкаинфо \_ равкмцевент**](msmcainfo-rawcmcevent.md)                             | Содержит событие CMC.                                                                                  |
| [**Мсмкаинфо \_ равкорректедплатформевент**](msmcainfo-rawcorrectedplatformevent.md) | Содержит исправленное событие платформы.                                                                   |
| [**Мсмкаинфо \_ равмкадата**](msmcainfo-rawmcadata.md)                               | Указывает необработанные журналы MCA.                                                                            |
| [**Мсмкаинфо \_ равмкаевент**](msmcainfo-rawmcaevent.md)                             | Содержит событие MCA.                                                                                  |
| [**WmiEvent**](wmievent.md)                                                        | Абстрактный базовый класс, из которого производятся все классы событий класса MCA.                             |
