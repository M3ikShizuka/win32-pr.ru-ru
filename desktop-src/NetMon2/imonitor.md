# <a name="imonitor-interface"></a>Интерфейс Имонитор

Интерфейс **имонитор** предоставляет методы, управляющие операциями мониторинга. Эти методы вызываются [*службой контроля мониторинга*](m.md) (мксвк).

## <a name="members"></a>Элементы

Интерфейс **имонитор** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Имонитор** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **имонитор** содержит следующие методы.



| Метод                                        | Описание                                                                                                         |
|:----------------------------------------------|:--------------------------------------------------------------------------------------------------------------------|
| [**доконфигуре**](imonitor-doconfigure.md)   | Обновляет конфигурацию монитора.<br/>                                                                           |
| [**доинитиализе**](imonitor-doinitialize.md) | Инициализирует монитор.<br/>                                                                                   |
| [**Выкадровки**](imonitor-onframes.md)         | Указывает состояние события кадра.<br/>                                                                   |
| [**OnStart**](imonitor-onstart.md)           | Указывает, что монитор успешно настроен и начинается сбор данных.<br/> |
| [**OnStatus**](imonitor-onstatus.md)         | Указывает на событие состояния НПП.<br/>                                                                           |
| [**OnStop**](imonitor-onstop.md)             | Указывает на завершение отслеживания данных.<br/>                                                                       |
