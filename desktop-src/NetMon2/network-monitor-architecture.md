# <a name="network-monitor-architecture"></a>Архитектура сетевой монитор

Архитектуру сетевой монитор можно разделить на два набора концепций программирования, связанных с [*экспертом*](e.md) и разработкой [*анализатора*](p.md) , а также с разработчиками, связанными с разработкой [*мониторов*](m.md) .

Хотя оба компонента используют одни и те же компоненты для захвата сетевого трафика, каждый из них использует разные интерфейсы для записи данных и различных компонентов для анализа того, что было захвачено.
