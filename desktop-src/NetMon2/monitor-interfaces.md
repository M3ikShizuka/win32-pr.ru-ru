# <a name="monitor-interfaces"></a>Интерфейсы монитора

Для разработки мониторов используются следующие интерфейсы COM.



| Интерфейс                              | Описание                                                                                                                                             |
|----------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [имонитор](imonitor.md)               | Предоставляет методы, управляющие работой монитора. Этот интерфейс должен быть реализован монитором.                                                     |
| [имониторевентер](imonitoreventer.md) | Предоставляет методы, используемые для отправки событий. Этот интерфейс предоставляется [*службой управления монитором*](m.md) (мксвк). |
