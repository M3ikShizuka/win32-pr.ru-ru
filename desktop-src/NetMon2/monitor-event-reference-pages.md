# <a name="monitor-event-reference-pages"></a>Мониторинг страниц ссылок на события

Страницы ссылок на события (ERP) можно использовать для подключения монитора к сетевой монитор пользовательскому интерфейсу. Они могут использоваться для оповещения конечных пользователей об обнаруженных условиях и проблемах, а также для предоставления предлагаемых исправлений или поясняющих данных.
