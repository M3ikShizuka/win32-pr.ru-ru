# <a name="networkstatus-structure"></a>Структура NETWORKSTATUS

Структура **NETWORKSTATUS** описывает текущее состояние НПП.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _NETWORKSTATUS {
  DWORD State;
  DWORD Flags;
} NETWORKSTATUS, *LPNETWORKSTATUS;
```
