# <a name="expert-structures"></a>Экспертные структуры

Вспомогательные функции экспертов, предоставляемые сетевой монитор и распространенные вспомогательные функции, используют структуры, описанные в следующей таблице.



| Структура                                              | Описание                                                                                            |
|--------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**конфигуредексперт**](configuredexpert.md)           | Связывает библиотеку DLL эксперта с ее конфигурацией.                                                       |
| [**експертконфиг**](expertconfig.md)                   | Предоставляет необработанные данные конфигурации.                                                                       |
| [**експертенуминфо**](expertenuminfo.md)               | Предоставляет сведения о библиотеке DLL эксперта. Сетевой монитор использует сведения.                       |
| [**експертфрамедескриптор**](expertframedescriptor.md) | Предоставляет сведения о кадре.                                                                    |
| [**експертстартупинфо**](expertstartupinfo.md)         | Предоставляет сведения о запуске эксперта.                                                         |
| [**експертстатус**](expertstatus.md)                   | Предоставляет текущее состояние работающего эксперта.                                                       |
| [**филтеробжект**](filterobject.md)                   | Определяет характеристики фильтра просмотра для эксперта.                                              |
| [**нмевентдата**](nmeventdata.md)                     | Предоставляет сведения о строке, отображаемой в средстве просмотра экспертов.                                      |
| [**нмколумнинфо**](nmcolumninfo.md)                   | Предоставляет сведения, определяющие столбец в Просмотр событий.                                        |
| [**нмколумнвариант**](nmcolumnvariant.md)             | Предоставляет контейнер для всех возможных данных, вставленных в столбец в одной строке в Просмотр событий.     |
| [**нмколумнтипе**](nmcolumntype.md)                   | Точка входа в варианте, указывающая, какой элемент объединения следует использовать, и как его форматировать. |



 

Сетевой монитор также предоставляет функции экспорта (вспомогательные функции, которые могут вызываться экспертом) и перечисления.



| Сведения о                                          | См.                                                                          |
|----------------------------------------------------------------|------------------------------------------------------------------------------|
| Функции экспорта, предоставляющие точки входа для библиотеки DLL эксперта. | [Функции экспорта DLL эксперта](expert-dll-export-functions.md)               |
| Вспомогательные функции, которые могут вызываться экспертами и анализаторами.    | [Общие функции эксперта и средства синтаксического анализа](expert-and-parser-common-functions.md) |
| Вспомогательные функции, которые вызываются только экспертами.              | [Экспертные функции](expert-functions.md)                                     |
| Перечисления, используемые экспертными структурами и функциями.          | [Перечисления экспертов](expert-enumerations.md)                               |
