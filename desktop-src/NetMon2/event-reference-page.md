# <a name="event-reference-page"></a>Страница ссылки на событие

В этом разделе описывается страница ссылки на события (ERP), HTML-документ, который можно написать для просмотра событий, происходящих во время работы эксперта сетевой монитор. После определения требований для создания экспертов можно использовать ERP, чтобы помочь сетевой монитор решить сетевые проблемы.
