# <a name="b-network-monitor"></a>Б (сетевой монитор)

<dl> <dt>

<span id="_netmon_binary_large_object_gly"></span><span id="_NETMON_BINARY_LARGE_OBJECT_GLY"></span>**большой двоичный объект (BLOB)**
</dt> <dd>

Универсальная структура данных сетевой монитор. Вы можете использовать большой двоичный объект для взаимодействия и хранения сведений о конфигурации и расположении, а также для хранения сведений, относящихся к конкретному приложению, не затрагивая другие данные в большом двоичном объекте.

</dd> <dt>

<span id="_netmon_blob_gly"></span><span id="_NETMON_BLOB_GLY"></span>**ОБЪЕКТЕ**
</dt> <dd>

См. большой двоичный объект.

</dd> <dt>

<span id="_netmon_buffer_gly"></span><span id="_NETMON_BUFFER_GLY"></span>**двойной**
</dt> <dd>

Зарезервированная часть памяти, в которой данные временно ожидают завершения перемещения на запоминающее устройство или из него или в другое место в памяти.

</dd> </dl>
