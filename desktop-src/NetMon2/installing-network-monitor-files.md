# <a name="installing-network-monitor-files"></a>Установка файлов сетевой монитор

Чтобы установить и запустить сетевой монитор часть пакета средств разработки программного обеспечения (SDK) платформы, выполните следующую процедуру.

**Установка и запуск файлов пакета SDK для платформы сетевой монитор**

1.  Чтобы установить файлы, во время установки пакета SDK для платформы выберите сетевой монитор документы и пример кода.

    После завершения установки программа установки пакета Platform SDK создает пакетный файл Setenv.bat.

2.  Чтобы инициализировать необходимые переменные среды, выполните Setenv.bat.
3.  Чтобы скомпилировать образцы, запустите **NMAKE**, который включен в Microsoft Visual Studio.
