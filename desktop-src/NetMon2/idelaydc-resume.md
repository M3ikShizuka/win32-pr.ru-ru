# <a name="idelaydcresume-method"></a>Метод Иделайдк:: Resume

Метод **Resume** перезапускает отложенную запись.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT STDMETHODCALLTYPE Resume();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Если метод выполнен успешно, возвращается значение НМЕРР \_ Success.

Если метод завершается неудачно, возвращаемое значение является одним из следующих кодов ошибок:



| Код возврата                                                                                                | Описание                                                                                                                               |
|------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_запись нмерр \_ не \_ приостановлена**</dt> </dl> | Запись не приостановлена. Вызовите [**иделайдк::P Аусе**](idelaydc-pause.md) , чтобы приостановить запись.<br/>                                |
| <dl> <dt>**НМЕРР \_ не \_ подключен**</dt> </dl>       | НПП не подключен к сети. вызовите [**иделайдк:: Подключение**](idelaydc-connect.md) , чтобы подключить нпп к сети.<br/> |
| <dl> <dt>**НМЕРР \_ не \_ отложена**</dt> </dl>         | нпп подключается к сети, но не с помощью метода [**иделайдк:: Подключение**](idelaydc-connect.md) .<br/>                     |



 

## <a name="remarks"></a>Remarks

Пока запись приостановлена, новые данные не добавляются в текущий [*файл записи*](c.md) до тех пор, пока не будет вызван метод **Иделайдк:: Resume** для перезапуска записи. Если [**приостановка**](idelaydc-pause.md) и **возобновление** используются для остановки и перезапуска записи, вся записанная информация помещается в один и тот же файл записи.

При использовании [**приостановки**](idelaydc-pause.md) и **возобновления** для управления записью сетевой монитор продолжает добавлять [*статистику диалога*](c.md) к существующей статистике для текущей записи.

Чтобы прерывать запись, вызовите [**иделайдк:: останавливаться**](idelaydc-stop.md).
