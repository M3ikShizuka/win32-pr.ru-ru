# <a name="compareaddresses-function"></a>Функция Компареаддрессес

Функция **компареаддрессес** сравнивает два адреса, указывая, что один из адресов больше, меньше или равен другому адресу.

## <a name="syntax"></a>Синтаксис


```C++
int WINAPI CompareAddresses(
  _In_ LPADDRESS lpAddress1,
  _In_ LPADDRESS lpAddress2
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lpAddress1* \[ окне\]
</dt> <dd>

Указатель на первый адрес.

</dd> <dt>

*lpAddress2* \[ окне\]
</dt> <dd>

Указатель на второй адрес.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если адреса совпадают, функция возвращает ноль.

Если параметр *lpAddress1* указывает адрес, который меньше, чем адрес, заданный параметром *lpAddress2* , возвращаемое значение является отрицательным числом.

Если параметр *lpAddress1* указывает адрес, который больше, чем адрес, заданный параметром *lpAddress2* , возвращаемое значение является положительным числом.

## <a name="remarks"></a>Комментарии

Адрес, который меньше другого адреса, указывает на предыдущий кадр. Адрес, который больше, чем другой адрес, обозначает более поздний кадр.

Сетевой монитор предоставляет две другие функции, [компарефрамедестаддресс](compareframedestaddress.md) и [компарефрамесаурцеаддресс](compareframesourceaddress.md), которые можно использовать для сравнения адресов. Функция **компарефрамедестаддресс** сравнивает заданный адрес с адресом назначения кадра, а функция **компарефрамесаурцеаддресс** сравнивает заданный адрес с исходным адресом кадра.
