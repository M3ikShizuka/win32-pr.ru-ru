# <a name="capture-filters"></a>Фильтры записи

Пакет SDK для сетевой монитор включает фильтры записи, с помощью которых можно ограничить количество кадров, получаемых приложением поставщика сетевых пакетов (НПП) во время его работы. Указав, какие кадры сетевого трафика необходимо проверить с помощью пользовательского монитора или приложения НПП, можно повысить эффективность обработки.
