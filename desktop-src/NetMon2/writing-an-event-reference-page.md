# <a name="writing-an-event-reference-page"></a>Создание страницы ссылки на событие

Используйте страницы справочника по событиям (ERP), чтобы описать, как решить проблему, обнаруженную экспертом. Так как ERP состоит из текста HTML, для его создания можно использовать любой редактор HTML или текстовый редактор.

Панель обозревателя событий сетевой монитор, в которой отображается ERP, поддерживает все функциональные возможности Microsoft Internet Explorer 4,01 и более поздних версий. Внедрите поддерживаемые элементы управления или скрипты на языке в ERP для просмотра информации или запуска других функций из ERP. Например, можно создать элемент управления, который отправляет сообщения электронной почты в сетевые операции.

Дополнительные сведения о создании ERP см. в следующих разделах:

-   [Именование страницы ссылки на события](naming-an-event-reference-page.md)
-   [Тестирование страницы ссылки на событие](testing-an-event-reference-page.md)
