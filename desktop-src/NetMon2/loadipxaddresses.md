# <a name="loadipxaddresses-function"></a>Функция Лоадипксаддрессес

Функция **лоадипксаддрессес** вызывается монитором для заполнения списка IPX-адресов, взятого из строковой переменной конфигурации HTML.

## <a name="syntax"></a>Синтаксис


```C++
BOOL LoadIPXAddresses(
  _In_  const char        *pConfig,
  _In_  const char        *pVarName,
  _Out_       IPX_ADDRESS **ppAddresses,
  _Out_       DWORD       *pNumAddresses
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*pConfig* \[ окне\]
</dt> <dd>

Указатель на строку конфигурации HTML, переданную в монитор методом [имонитор::D оконфигуре](imonitor-doconfigure.md) .

</dd> <dt>

*пварнаме* \[ окне\]
</dt> <dd>

Указатель на имя переменной в строке конфигурации.

</dd> <dt>

*ппаддрессес* \[ заполняет\]
</dt> <dd>

Указатель на указатель на массив адресов. Если переменная, указанная в *пварнаме* , найдена и имеет ненулевую длину, выделяется достаточно места (с помощью **хеапаллокмемори**), а все IPX-адреса хранятся в виде массива.

</dd> <dt>

*пнумаддрессес* \[ заполняет\]
</dt> <dd>

Указатель на переменную **DWORD** , для которой задано количество IPX-адресов, полученных из строки конфигурации.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена успешно (обнаружено имя переменной, имеющее строку, отличную от нулевой длины, которая представляет IPX-адреса), возвращается значение **true**.

Если функция завершается неудачно, возвращается значение **false**.
