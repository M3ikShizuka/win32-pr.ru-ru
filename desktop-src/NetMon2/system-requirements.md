# <a name="system-requirements"></a>Требования к системе

Минимальные конфигурации компьютеров различаются; для сетевой монитор разработки рекомендуется, чтобы у разработчика был компьютер, который содержит следующие минимальные спецификации:

-   Процессор Pentium
-   32 МБ ОЗУ
-   30 МБ свободного места на жестком диске
-   Поддержка видеомониторов и видеоадаптеров для цветов 256 при разрешении 800 X 600
-   Сетевое подключение
-   Microsoft Internet Explorer 4,01
-   Microsoft Visual C++ версии 5,0

Для тестирования приложения, разработанного с помощью сетевой монитор, требуется компьютер со следующими минимальными спецификациями:

-   процессор 486 с тактовой частотой не менее 50 МГц
-   16 МБ ОЗУ
-   20 МБ свободного места на жестком диске
-   Сетевое подключение
