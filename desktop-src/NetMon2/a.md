# <a name="a-network-monitor"></a>A (сетевой монитор)

<dl> <dt>

<span id="_netmon_address_gly"></span><span id="_NETMON_ADDRESS_GLY"></span>**Address**
</dt> <dd>

Идентификатор узла сети, который отличает один узел от другого в сети. Также известен как *сетевой адрес* или *MAC-адрес*. В Microsoft Systems Management Server (SMS) адрес используется для подключения сайта и системы сайта. Отправитель использует конкретный адрес для отправки инструкций и данных на другие сайты.

</dd> <dt>

<span id="_netmon_address_pair_gly"></span><span id="_NETMON_ADDRESS_PAIR_GLY"></span>**пара адресов**
</dt> <dd>

Исходный и конечный адреса передачи данных. Назначением может быть один адрес, многоадресная рассылка или широковещательная рассылка. В захваченных данных адрес источника — это конкретный адрес. В спецификациях фильтра исходный адрес может быть конкретным адресом или любым, но не может быть широковещательным или многоадресным.

</dd> </dl>
