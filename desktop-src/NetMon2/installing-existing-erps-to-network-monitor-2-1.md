# <a name="installing-existing-erps-to-network-monitor-21"></a>Установка существующих ERP в сетевой монитор 2,1

После написания и именования страницы ссылки на события (ERP) необходимо установить ее.

**Установка страницы ссылки на событие**

1.  Создайте каталог ERP. В этих примерах демонстрируются каталоги для экспертов:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Expert\
    ```

2.  Создайте подкаталог для каждого нового эксперта, который будет отображать ERP. Следующие списки вложенных подкаталогов предназначены для экспертов:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert
    ```

3.  Скопируйте файлы ERP в соответствующий подкаталог. В следующих примерах приведены полные пути к установленному эксперту ERP:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert\MyExpert1.htm
    ```
