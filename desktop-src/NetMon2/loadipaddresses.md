# <a name="loadipaddresses-function"></a>Функция Лоадипаддрессес

Функция **лоадипаддрессес** вызывается монитором для заполнения списка IP-адресов адресами, полученными из строковой переменной конфигурации HTML.

## <a name="syntax"></a>Синтаксис


```C++
BOOL LoadIPAddresses(
  _In_  const char  *pConfig,
  _In_  const char  *pVarName,
  _Out_       DWORD **ppAddresses,
  _Out_       DWORD *pNumAddresses
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*pConfig* \[ окне\]
</dt> <dd>

Указатель на строку конфигурации HTML, переданную в монитор методом [имонитор::D оконфигуре](imonitor-doconfigure.md) .

</dd> <dt>

*пварнаме* \[ окне\]
</dt> <dd>

Указатель на имя переменной в строке конфигурации.

</dd> <dt>

*ппаддрессес* \[ заполняет\]
</dt> <dd>

Указатель на указатель на массив адресов. Если переменная, указанная в *пварнаме* , найдена и имеет ненулевую длину, функция выделяет достаточно места и сохраняет все IP-адреса в виде массива.

</dd> <dt>

*пнумаддрессес* \[ заполняет\]
</dt> <dd>

Указатель на переменную **DWORD** , для которой задано количество IP-адресов, полученных из строки конфигурации.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена успешно (имя переменной было обнаружено и имело строку ненулевой длины, которая представляет IP-адреса), возвращается значение **true**.

Если функция завершается неудачно, возвращается значение **false**.

## <a name="remarks"></a>Комментарии

IP-адреса должны быть в формате x. x. x. x (например, 127.0.0.1).
