# <a name="viewing-parsed-information"></a>Просмотр проанализированных данных

Средство просмотра кадров сетевой монитор отображает проанализированные данные на трех панелях. В верхней области отображается сводка по кадру, включающему идентифицированный протокол. В средней области отображаются свойства форматированных данных, которые могут быть организованы иерархически в виде части отображения средства синтаксического анализа. На нижней панели отображаются выделенные необработанные данные, выбранные из средней области.

Можно указать способ отображения информации в средстве просмотра при разработке собственного средства синтаксического анализа. На следующем рисунке показано, как можно отобразить сведения в средстве просмотра кадров сетевой монитор.

![средство просмотра кадров сетевого монитора](images/parseui.png)

> [!Note]  
> Средства синтаксического анализа не отображают кадры. Средства синтаксического анализа распознают поля в предоставленных сведениях, а затем уведомляют сетевой монитор, чтобы отобразить кадр. Фильтрация — это операция более высокого уровня, которая позволяет фильтровать запросы на охват средств синтаксического анализа.

 

В средстве синтаксического анализа используйте только функции, которые могут выполняться в приложениях Microsoft Win32. Перед присоединением свойств к необработанным данным средство синтаксического анализа должно сначала зарегистрировать все возможные свойства с помощью ядра сетевой монитор. Средство синтаксического анализа отправляет в ядро сообщение для создания базы данных свойств, а затем заполняет базу данных свойств любым возможным свойством для его протокола. Каждое свойство в базе данных свойств содержит такие сведения, как текстовое описание, тип данных и квалификатор, которые используются для форматирования необработанных данных, и подпрограммы форматирования, используемые для вывода данных.
