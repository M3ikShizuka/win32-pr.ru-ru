# <a name="g-network-monitor"></a>G (сетевой монитор)

<dl> <dt>

<span id="_netmon_generic_formatter_gly"></span><span id="_NETMON_GENERIC_FORMATTER_GLY"></span>**Универсальный модуль форматирования**
</dt> <dd>

Модуль форматирования данных свойств, который сетевой монитор предоставляет. Универсальный модуль форматирования может отформатировать данные, отображаемые в области сведений пользовательского интерфейса сетевой монитор. Если требуется, чтобы сетевой монитор использовал универсальный модуль форматирования, необходимо указать функцию **форматпропертинстанце** в **Инстанцедата** элемента структуры **PROPERTYINFO** свойства. См. также [Общие выходные данные модуля форматирования](generic-formatter-output.md)

</dd> </dl>
