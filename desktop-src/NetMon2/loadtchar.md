# <a name="loadtchar-function"></a>Функция Лоадтчар

Функция **лоадтчар** вызывается мониторами для задания строковой переменной строки, взятой из строки конфигурации HTML.

## <a name="syntax"></a>Синтаксис


```C++
BOOL LoadTCHAR(
  _In_  const char *pConfig,
  _In_  const char *pVarName,
  _Out_       char **ppszString
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*pConfig* \[ окне\]
</dt> <dd>

Указатель на строку конфигурации HTML, переданную в монитор методом [имонитор::D оконфигуре](imonitor-doconfigure.md) .

</dd> <dt>

*пварнаме* \[ окне\]
</dt> <dd>

Указатель на имя переменной в строке конфигурации.

</dd> <dt>

*ппсзстринг* \[ заполняет\]
</dt> <dd>

Указатель на указатель строки. Если запрошенное имя переменной найдено, строка выделяется и присваивается этому указателю. Это пользователь, ответственный за освобождение памяти, связанной со строкой.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена успешно (если имя переменной было обнаружено и имело строку ненулевой длины), возвращается значение **true**.

Если функция завершается неудачно, возвращается значение **false**.
