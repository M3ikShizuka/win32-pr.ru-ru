# <a name="expert-functions-structures-and-enumerations"></a>Экспертные функции, структуры и перечисления

В разделе "экспертные функции", "структуры" и "перечисления" содержатся сведения о функциях, структурах и перечислениях, которые могут использоваться только экспертами. В этом разделе рассматриваются следующие темы:

-   [Функции экспорта DLL эксперта](expert-dll-export-functions.md)

    В этих разделах описываются функции, которые должны быть экспортированы экспертными библиотеками DLL.

-   [Экспертные функции](expert-functions.md)

    В этих разделах описываются вспомогательные функции, которые вызываются экспертами.

-   [Экспертные структуры](expert-structures.md)

    В этих разделах описываются структуры, используемые экспертными функциями.

-   [Перечисления экспертов](expert-enumerations.md)

    В этих разделах описываются перечисления, используемые экспертными структурами и функциями.
