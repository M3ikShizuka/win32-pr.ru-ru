# <a name="clipping-a-frame"></a>Обрезка рамки

Можно указать, что драйвер обрезает кадры. (Если другие разделы фильтра записи опущены, это может быть единственная функция фильтра записи). Если поле **нфрамебитестокопи** не равно нулю (0), его значение указывает, сколько байт каждого полученного кадра следует хранить. Если поле равно нулю, весь фрейм сохраняется.

> [!Note]  
> Вы можете обрезать любой кадр, который передает другие части фильтра записи, установив **нфрамебитестокопи**.
