# <a name="findpropertyinstancerestart-function"></a>Функция Финдпропертинстанцерестарт

Функция **финдпропертинстанцерестарт** находит следующий экземпляр свойства, указанного параметром *хпроперти* .

## <a name="syntax"></a>Синтаксис


```C++
LPPROPERTYINST WINAPI FindPropertyInstanceRestart(
  _In_ HFRAME         hFrame,
  _In_ HPROPERTY      hProperty,
  _In_ LPPROPERTYINST *lpRestartKey,
  _In_ BOOL           DirForward
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хфраме* \[ окне\]
</dt> <dd>

Маркер фрейма. Маркер кадра можно извлечь с помощью вызова функции [noframes](getframe.md) .

</dd> <dt>

*хпроперти* \[ окне\]
</dt> <dd>

Описатель искомого свойства. Маркер свойства может быть получен путем вызова функции- [Свойства](getproperty.md) .

</dd> <dt>

*лпрестарткэй* \[ окне\]
</dt> <dd>

Указатель на экземпляр свойства, используемый в качестве начальной точки поиска. Если параметр *лпрестарткэй* имеет значение **null**, поиск начинается в начале кадра или в конце кадра в зависимости от значения параметра *дирфорвард* .

Если *лпрестарткэй* указывает на **null**, поиск начинается в начале кадра, если *дирфорвард* имеет **значение true** , или в конце кадра, если параметр имеет **значение false**.

</dd> <dt>

*Дирфорвард* \[ окне\]
</dt> <dd>

Индикатор направления поиска. Если значение равно **true**, Поиск перемещается от текущего положения к концу кадра. Если значение равно **false**, Поиск перемещается от текущего положения к началу кадра.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена успешно, возвращаемое значение будет следующим допустимым **лппропертинст**.

Если функция завершилась неудачно, возвращается значение **null**.

## <a name="remarks"></a>Комментарии

[*Эксперты*](e.md) и [*средства синтаксического анализа*](p.md) могут вызывать функцию **финдпропертинстанцерестарт** .
