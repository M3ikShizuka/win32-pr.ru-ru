# <a name="destroypropertydatabase-function"></a>Функция Дестройпропертидатабасе

Функция **дестройпропертидатабасе** освобождает ресурсы, используемые для создания [*базы данных свойств*](p.md)протокола.

## <a name="syntax"></a>Синтаксис


```C++
DWORD WINAPI DestroyPropertyDatabase(
  _In_ HPROTOCOL hProtocol
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хпротокол* \[ окне\]
</dt> <dd>

Обработчик для уничтожения базы данных свойств. Этот маркер передается в библиотеку DLL средства синтаксического анализа, когда сетевой монитор вызывает функцию [дерегистрации](deregister.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена успешно, возвращается значение НМЕРР \_ Success.

Если функция завершается неудачно, возвращаемое значение является кодом ошибки.



| Код возврата                                                                                              | Описание                                |
|----------------------------------------------------------------------------------------------------------|--------------------------------------------|
| <dl> <dt>**\_Внутренняя \_ Ошибка нмерр**</dt> </dl>    | Внутренняя ошибка. <br/>    |
| <dl> <dt>**НМЕРР \_ недопустимое \_ хпротокол**</dt> </dl> | Недопустимый маркер в *хпротокол*. <br/> |



 

## <a name="remarks"></a>Комментарии

Функцию **дестройпропертидатабасе** следует вызывать только при реализации функции [дерегистрации](deregister.md) экспорта для протокола. Для библиотек DLL синтаксического анализатора, поддерживающих несколько протоколов, функция **дестройпропертидатабасе** вызывается для каждой реализации **отмены регистрации**.
