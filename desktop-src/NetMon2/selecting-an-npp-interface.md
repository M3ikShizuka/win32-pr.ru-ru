# <a name="selecting-an-npp-interface"></a>Выбор интерфейса НПП

Поставщики сетевых пакетов (НППС) предоставляют интерфейсы [**иделайдк**](idelaydc.md), [**ИЕСП**](iesp.md), [**ИРТК**](irtc.md)и [**истатс**](istats.md) . Каждый из этих интерфейсов предоставляет аналогичные методы для подключения НПП к сети, сбора сетевого трафика и сбора статистических данных о перехваченных данных. Чтобы выбрать интерфейс для использования, обратитесь к следующей таблице.

> [!Note]  
> При подключении НПП к сети с помощью одного из этих интерфейсов можно использовать только методы, предоставляемые этим интерфейсом. Например, если вы подключаетесь к сети с помощью интерфейса [**ИРТК**](irtc.md) , а затем пытаетесь начать запись с помощью [**иделайдк**](idelaydc.md), вызов для запуска записи завершится ошибкой и будет возвращен код ошибки.

 



| Интерфейс                    | Назначение                                                                                                                                                                                                                                                                                                                                     |
|------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**иделайдк**](idelaydc.md) | Используйте для записи сетевого трафика и сохранения его в файле записи. Этот интерфейс используется сетевой монитор UI и другими приложениями НПП, для которых требуется хранить захваченные сведения о сети.<br/>                                                                                                                                      |
| [**иесп**](iesp.md)         | Используется для предоставления расширенной статистики в особом формате файла ESP. Этот интерфейс используется приложениями НПП, для которых требуется расширенная статистика, предоставляемая форматом ESP.<br/>                                                                                                                                                        |
| [**иртк**](irtc.md)         | Используется для сбора сетевого трафика в реальном времени и для активации событий при их возникновении. Этот интерфейс используется приложениями НПП, для которых требуются записи времени выполнения. Обратите внимание, что этот интерфейс не предоставляет статистику, предоставляемую другими интерфейсами НПП, и не позволяет вставлять фреймы в захваченный сетевой трафик.<br/> |
| [**истатс**](istats.md)     | Используется для получения статистики записи, но не для захваченных кадров.                                                                                                                                                                                                                                                                                 |
