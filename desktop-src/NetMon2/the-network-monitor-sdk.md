# <a name="network-monitor-introduction"></a>Общие сведения о сетевой монитор

Сетевой монитор часть пакета средств разработки программного обеспечения платформы (SDK) расширяет функциональные возможности сетевой монитор, помогая создавать собственные эксперты, анализаторы протоколов и поставщики сетевых пакетов (НППС). Этот раздел содержит следующую документацию:

-   Разделы по программированию, необходимые для создания экспертов и средств синтаксического анализа. Эта информация также описывает, как создать фильтр записи и использовать большие двоичные объекты и НППС.
-   Справочные сведения, связанные с API, о экспертах, средствах синтаксического анализа, больших двоичных объектах и НППС.
-   Глоссарий терминов, используемых в документации по сетевой монитор.
