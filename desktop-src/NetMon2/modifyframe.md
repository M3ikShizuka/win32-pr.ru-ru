# <a name="modifyframe-function"></a>Функция Модифифраме

Функция **модифифраме** изменяет существующий кадр на новые данные.

## <a name="syntax"></a>Синтаксис


```C++
HFRAME WINAPI ModifyFrame(
  _In_  HCAPTURE hCapture,
  _In_  DWORD    FrameNumber,
  _In_  LPBYTE   FrameData,
  _In_  DWORD    FrameLength,
  _Out_ __int64  TimeStamp
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хкаптуре* \[ окне\]
</dt> <dd>

Обработчик для записи. Дополнительные сведения о получении маркера записи см. в подразделе "Примечания" этой статьи **модифифраме** .

</dd> <dt>

*Фраменумбер* \[ окне\]
</dt> <dd>

Номер индекса кадра.

</dd> <dt>

*Фрамедата* \[ окне\]
</dt> <dd>

Указатель на массив байтов, содержащий новые данные кадра.

</dd> <dt>

*Фрамеленгс* \[ окне\]
</dt> <dd>

Длина новых данных в байтах.

</dd> <dt>

*Метка времени* \[ заполняет\]
</dt> <dd>

Метка времени, указывающая, когда был изменен кадр.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена успешно, возвращаемое значение является маркером нового кадра.

Если функция завершилась неудачно, возвращается значение **null**.

## <a name="remarks"></a>Комментарии

Если вызов выполнен успешно, функция **модифифраме** уничтожает исходный кадр.

[*Эксперты*](e.md) и [*средства синтаксического анализа*](p.md) могут вызывать функцию **модифифраме** .
