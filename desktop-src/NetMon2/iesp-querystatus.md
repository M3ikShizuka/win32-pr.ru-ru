# <a name="iespquerystatus-method"></a>Метод ИЕСП:: QueryStatus

Метод **QueryStatus** ИЗВЛЕКАЕТ состояние НПП.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT STDMETHODCALLTYPE QueryStatus(
  [out] NETWORKSTATUS *pNetworkStatus
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пнетворкстатус* \[ заполняет\]
</dt> <dd>

Указатель на возвращаемую структуру [NETWORKSTATUS](networkstatus.md) , которая указывает текущее состояние (запись, пауза, остановка и т. д.) НПП. Пользователь должен выделить и освободить память для структуры NETWORKSTATUS.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если метод выполнен успешно, возвращается значение НМЕРР \_ Success.

Если метод завершился неудачно, возвращаемое значение является следующим кодом ошибки:



| Код возврата                                                                                              | Описание                                                                                                                                                                                 |
|----------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**НМЕРР \_ Недопустимый \_ параметр**</dt> </dl> | Параметр *пнетворкстатус* не указывает на допустимую структуру [NETWORKSTATUS](networkstatus.md) . Выделите память для этой структуры и снова вызовите **ИЕСП:: QueryStatus** .<br/> |



 

## <a name="remarks"></a>Remarks

Этот метод можно вызвать в любое время после вызова [креатенппинтерфаце](createnppinterface.md) . Этот метод можно вызвать, чтобы узнать, подключен ли НПП к сети, узнать состояние текущего захвата и проверить, ожидают ли какие-либо триггеры.

Перед вызовом этого метода выделите память, необходимую для структуры [NETWORKSTATUS](networkstatus.md) , и освободите эту память, когда структура больше не нужна.
