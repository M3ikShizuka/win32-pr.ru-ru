# <a name="formatting-displayed-data"></a>Форматирование отображаемых данных

Сетевой монитор или библиотека DLL средства синтаксического анализа могут отформатировать данные, отображаемые в области сведений пользовательского интерфейса сетевой монитор.

Сетевой монитор предоставляет универсальный модуль форматирования, предоставляющий базовые службы, необходимые для вывода большинства типов данных, существующих в протоколе. Однако универсальный модуль форматирования не поддерживает все типы данных. Например, универсальный модуль форматирования не поддерживает следующее:

-   Несколько типов адресов.
-   Имена, понятные источнику и назначению.
-   Идентификаторы объектов.
-   Большие целочисленные типы данных.
-   Небольшие целочисленные типы данных переменной длины.

Следующий пример определяет две форматированные строки для свойства идентификатора привилегии.

-   В первой строке кода показаны выходные данные универсального модуля форматирования. Выходные данные основываются на типе данных.
-   Вторая строка кода показывает выходные данные пользовательского модуля форматирования со строкой для данных свойства.

``` syntax
Privilege ID (PID) = 0x0029
Privilege ID   (PID) = 0x0029   The Process has privileges
```

> [!Note]  
> По возможности используйте универсальный модуль форматирования для вывода данных, так как он позволяет управлять размером библиотеки DLL средства синтаксического анализа и предоставляет более эффективные возможности кросс-платформенных функций для библиотеки DLL синтаксического анализатора.
