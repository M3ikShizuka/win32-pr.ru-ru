# <a name="idelaydc-interface"></a>Интерфейс Иделайдк

Интерфейс **иделайдк** предоставляет методы, используемые для подключения к сети, записи сетевого трафика в файл записи, получения статистики и отключения от сети.

Этот интерфейс захватывает кадры из потока сетевых данных, а затем копирует кадры в файл записи. Этот интерфейс используется приложениями сетевой монитор и НПП. Для получения сетевых данных целевой сетевой адаптер должен поддерживать неизбирательный режим.

## <a name="members"></a>Элементы

Интерфейс **иделайдк** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иделайдк** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иделайдк** содержит следующие методы.



| Метод                                                                  | Описание                                                                                                                                             |
|:------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Настройка**](idelaydc-configure.md)                                 | Отправляет сведения о конфигурации, используемые для записи.<br/>                                                                                        |
| [**Подключить**](idelaydc-connect.md)                                     | Подключает НПП к сети.<br/>                                                                                                             |
| [**Отключить**](idelaydc-disconnect.md)                               | Отключает НПП от сети.<br/>                                                                                                        |
| [**жетконтролстате**](idelaydc-getcontrolstate.md)                     | Возвращает состояние [*записи*](c.md), которое указывает, запущена или приостановлена запись.<br/>                      |
| [**жетконверсатионстатистикс**](idelaydc-getconversationstatistics.md) | Извлекает [*сведения о*](s.md) [*сеансе*](s.md) и станции для текущей записи.<br/> |
| [**жеттоталстатистикс**](idelaydc-gettotalstatistics.md)               | Извлекает сведения о времени, буфере, драйвере и другой сетевой статистике из выполняемой в данный момент записи.<br/>                                              |
| [**Пауза**](idelaydc-pause.md)                                         | Временно останавливает текущую запись.<br/>                                                                                                       |
| [**куеристатионс**](idelaydc-querystations.md)                         | Извлекает список всех компьютеров, которые используют сетевой монитор для записи данных в подсети.<br/>                                                      |
| [**QueryStatus**](idelaydc-querystatus.md)                             | Получает состояние НПП.<br/>                                                                                                             |
| [**Возобновить**](idelaydc-resume.md)                                       | Возобновляет приостановленную запись.<br/>                                                                                                                    |
| [**Запустить**](idelaydc-start.md)                                         | Запускает запись и создает [*файл записи*](c.md).<br/>                                                           |
| [**Остановить**](idelaydc-stop.md)                                           | Останавливает текущую запись и закрывает [*файл записи*](c.md).<br/>                                                   |
