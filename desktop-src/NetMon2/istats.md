# <a name="istats-interface"></a>Интерфейс Истатс

Интерфейс **истатс** предоставляет методы, используемые для подключения НПП к сети, записи сетевого трафика, получения статистики и отключения НПП от сети. Этот интерфейс создает статистику без фреймов.

## <a name="members"></a>Элементы

Интерфейс **истатс** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Истатс** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **истатс** содержит следующие методы.



| Метод                                                                | Описание                                                                                                                                               |
|:----------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Configure**](istats-configure.md)                                 | Настраивает триггер, соответствие шаблону и размер буфера для файла записи.<br/>                                                                    |
| [**Подключение**](istats-connect.md)                                     | Подключает НПП к сети.<br/>                                                                                                               |
| [**Отключение**](istats-disconnect.md)                               | Отключает НПП от сети.<br/>                                                                                                          |
| [**жетконтролстате**](istats-getcontrolstate.md)                     | Возвращает состояние [*записи*](c.md), которое указывает, запущена или приостановлена запись.<br/>                        |
| [**жетконверсатионстатистикс**](istats-getconversationstatistics.md) | Извлекает [*сведения о*](s.md) [*сеансе*](s.md) и станции о текущем захвате.<br/> |
| [**жеттоталстатистикс**](istats-gettotalstatistics.md)               | Извлекает сведения о времени, буфере, драйвере и другой сетевой статистике из выполняемой в данный момент записи.<br/>                                                |
| [**Пауза**](istats-pause.md)                                         | Временно останавливает текущую запись.<br/>                                                                                                         |
| [**куеристатионс**](istats-querystations.md)                         | Извлекает список всех компьютеров, использующих сетевой монитор для сбора данных в подсети.<br/>                                                  |
| [**QueryStatus**](istats-querystatus.md)                             | Получает состояние НПП.<br/>                                                                                                               |
| [**Возобновить**](istats-resume.md)                                       | Перезапускает приостановленную запись.<br/>                                                                                                                     |
| [**Начать**](istats-start.md)                                         | Запускает запись.<br/>                                                                                                                            |
| [**Stop**](istats-stop.md)                                           | Останавливает текущую запись.<br/>                                                                                                                     |
