# <a name="installing-an-expert-to-network-monitor-21"></a>Установка эксперта в сетевой монитор 2,1

Чтобы установить библиотеку DLL экспертов, скопируйте ее в \\ подкаталог экспертов сетевой монитор (например, C: \\ Program Files \\ NetMon2 \\ экспертов). После установки эксперт будет распознан и доступен при следующем запуске сетевой монитор.

На следующем рисунке показана библиотека DLL Мексперт, Мексперт, построенная на основе примера кода и скопированная в \\ подкаталог экспертов.
