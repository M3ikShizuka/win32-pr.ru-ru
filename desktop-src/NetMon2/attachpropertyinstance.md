# <a name="attachpropertyinstance-function"></a>Функция Аттачпропертинстанце

Функция **аттачпропертинстанце** сопоставляет существующее свойство с конкретным расположением в распознанных данных.

## <a name="syntax"></a>Синтаксис


```C++
BOOL WINAPI AttachPropertyInstance(
  _In_ HFRAME    hFrame,
  _In_ HPROPERTY hProperty,
  _In_ DWORD     Length,
  _In_ ULPVOID   lpData,
  _In_ DWORD     HelpID,
  _In_ DWORD     IndentLevel,
  _In_ DWORD     IFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хфраме* \[ окне\]
</dt> <dd>

Обработчик для анализируемого кадра. Используйте маркер, передаваемый в библиотеку DLL средства синтаксического анализа, в параметре *хфраме* функции [**аттачпропертиес**](attachproperties.md) .

</dd> <dt>

*хпроперти* \[ окне\]
</dt> <dd>

Обработчик для структуры [**PROPERTYINFO**](propertyinfo.md) , определяющей свойство. При реализации функции [**регистрации**](register-parser.md) экспорта указывается структура **PROPERTYINFO** , определяющая свойство.

</dd> <dt>

*Длина* \[ окне\]
</dt> <dd>

Длина данных для этого экземпляра свойства.

</dd> <dt>

*лпдата* \[ окне\]
</dt> <dd>

Указатель на расположение в распознанных данных, где находится значение свойства. Используйте указатель, передаваемый в библиотеку DLL средства синтаксического анализа, в параметре *лппротокол* функции [**аттачпропертиес**](attachproperties.md) .

</dd> <dt>

*Идентификатор справки* \[ окне\]
</dt> <dd>

Идентификатор (от 0 до 2047), используемый для установки контекстно-зависимой справки для свойства.

Номер идентификатора указывается относительно файла справки, связанного с [*базой данных свойств*](p.md)протокола.

</dd> <dt>

*Индентлевел* \[ окне\]
</dt> <dd>

Уровень отступа (от 0 до 15), используемый для иерархического вывода свойства.

Сетевой монитор использует уровни от 0 до 14 для отступа свойств. Уровень 15 — это специальное значение, которое позволяет средству синтаксического анализа присоединить скрытое свойство, которое не является видимым.

</dd> <dt>

*Ифлагс* \[ окне\]

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена успешно, возвращается значение **true**.

Если функция завершается неудачно, возвращается значение **false**.

## <a name="remarks"></a>Комментарии

Функция **аттачпропертинстанце** вызывается во время реализации функции экспорта [**аттачпропертиес**](attachproperties.md) . При присоединении к данным свойства сетевой монитор создает структуру [**пропертинст**](propertyinst.md) , определяющую экземпляр присоединенного свойства.

Во время реализации [**аттачпропертиес**](attachproperties.md)вызовите **аттачпропертинстанце** , чтобы использовать данные в том виде, в котором они существуют в записи. Можно также вызвать функцию [**аттачпропертинстанцеекс**](attachpropertyinstanceex.md) для изменения данных свойства. Однако рекомендуется использовать данные в том виде, в котором они существуют в записи.
