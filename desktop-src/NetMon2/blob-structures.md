# <a name="blob-structures"></a>Структуры больших двоичных объектов

Вспомогательный API BLOB-объекта включает следующие структуры.



| Структура                                                        | Описание                                                            |
|------------------------------------------------------------------|------------------------------------------------------------------------|
| [**ADDRESS**](address.md)                                       | Содержит один поддерживаемый адрес.                                  |
| [**аддресспаир**](addresspair.md)                               | Конструирует фильтр записи.                                           |
| [**аддресстабле**](addresstable.md)                             | Указывает таблицу пар адресов.                                    |
| [**андексп**](andexp.md)                                         | Указывает массив совпадений шаблона.                                 |
| [**Таблица больших двоичных объектов \_**](blob-table.md)                                | Указывает массив больших двоичных объектов.                                           |
| [**\_значения заголовков каптурефиле \_**](capturefile-header-values.md) | Определяет структуру файла заголовка сетевой монитор.              |
| [**каптурефилтер**](capturefilter.md)                           | Указывает данные фильтра записи.                                         |
| [**ВЫРАЖЕНИЕ**](expression.md)                                 | Указывает выражение одноранговых узлов, структурированных в логических операторах и. |
| [**NOFRAMES**](frame.md)                                           | Задает кадр.                                                     |
| [**\_дескриптор кадра**](frame-descriptor.md)                    | Задает сведения о необработанных кадрах.                                       |
| [**фраметабле**](frametable.md)                                 | Задает набор кадров.                                             |
| [**УНИВЕРСАЛЬНЫЙ \_ порт**](generic-port.md)                            | Указывает номер порта, используемый в протоколах IP или IPX.                   |
| [**IPX- \_ адрес**](ipx-address.md)                              | Указывает IPX-адрес.                                              |
| [**нетворкинфо**](networkinfo.md)                               | Указывает сетевую карту.                                                       |
| [**паттернматч**](patternmatch.md)                             | Указывает элементы шаблона, используемые для вычисления кадров.                    |
| [**НЕОБХОДИМОСТЬ**](trigger.md)                                       | Указывает, что событие возникает в определенное время.                      |
| [**\_событие обновления**](update-event.md)                            | Обновляет события.                                                        |
| [**\_IP- \_ адрес Vines**](vines-ip-address.md)                   | Указывает адрес в сети Vines.                               |
