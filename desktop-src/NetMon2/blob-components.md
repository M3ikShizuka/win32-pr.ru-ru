# <a name="blob-components"></a>Компоненты BLOB-объектов

Большой двоичный объект (BLOB) включает следующие иерархические компоненты:

-   Владельцы
-   Категории
-   Теги
-   Значения

## <a name="textual-representation"></a>Текстовое представление

Для удобства большие двоичные объекты отображаются в формате владелец: Категория: Тег: значение, в котором они отображаются при сохранении в файл. Внутренняя структура большого двоичного объекта является непрозрачной, так как представляет собой указатели и таблицы. Например, ниже приведена запись из большого двоичного объекта, который можно использовать для настройки НПП:

``` syntax
NPP:Configure:BufferSize=32768
```

Владелец — НПП. Категория — configure, тег — BufferSize, а значение — 32768.
