# <a name="specifying-protocol-information"></a>Указание сведений о протоколе

Драйвер сетевой монитор может добавлять сведения в таблицу Frame, указывая, какой именно протокол (IP или IPX) был найден, и смещение до начала протокола. Чтобы добавить эти сведения в таблицу Frame, задайте фильтр записи.
