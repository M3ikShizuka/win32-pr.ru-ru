# <a name="parser-dll-architecture"></a>Архитектура библиотеки DLL средства синтаксического анализа

Архитектура библиотеки DLL средства синтаксического анализа должна предоставлять функции, показанные на следующем рисунке. Имейте в виду, что для некоторых функций требуется реализация только одной точки входа. Однако если библиотека DLL средства синтаксического анализа поддерживает несколько протоколов, то для этой функции требуется реализация нескольких точек входа.
