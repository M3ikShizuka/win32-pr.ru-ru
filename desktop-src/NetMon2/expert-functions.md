# <a name="expert-functions"></a>Экспертные функции

Следующие вспомогательные функции могут вызываться только экспертами, которые экспортируют функцию [Run](run.md) или [Configure](configure.md) .



| Функция                                         | Описание                                                                             |
|--------------------------------------------------|-----------------------------------------------------------------------------------------|
| [експертжетфраме](expertgetframe.md)             | Извлекает заданный кадр из записи.                                               |
| [експертиндикатестатус](expertindicatestatus.md) | Указывает процент завершения анализа записи экспертом.             |
| [експертжетстартупинфо](expertgetstartupinfo.md) | Извлекает сведения о запуске эксперта.                                       |
| [експертмеморисизе](expertmemorysize.md)         | Возвращает размер памяти, выделенной при вызове функции **експерталлокмемори** . |
| [експертсубмитевент](expertsubmitevent.md)       | Указывает на проблему и получает сведения о проблеме, если она существует.          |
| [експерталлокмемори](expertallocmemory.md)       | Выделяет память для эксперта.                                                        |
| [експертреаллокмемори](expertreallocmemory.md)   | Изменяет размер памяти, выделенной функцией **експерталлокмемори** .         |
| [експертфримемори](expertfreememory.md)         | Освобождает память, выделенную экспертом.                                                   |
