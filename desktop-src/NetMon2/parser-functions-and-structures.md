# <a name="parser-functions-and-structures"></a>Функции и структуры средства синтаксического анализа

В этом разделе содержатся сведения о функциях, структурах и перечислениях, связанных с синтаксическим анализатором, которые можно использовать для разработки средства синтаксического анализа. В этом разделе рассматриваются следующие задачи:

-   [Функции экспорта библиотеки DLL средства синтаксического анализа](parser-dll-export-functions.md)

    В этих разделах описываются функции, которые должны быть экспортированы с помощью библиотек DLL средства синтаксического анализа.

-   [Функции средства синтаксического анализа](parser-functions.md)

    В этих разделах описываются вспомогательные функции, которые вызываются анализаторами.

-   [Структуры, используемые функциями синтаксического анализа](parser-structures.md)

    В этих разделах описываются структуры, используемые функциями синтаксического анализатора.
