# <a name="irtc-interface"></a>Интерфейс ИРТК

Интерфейс **ИРТК** предоставляет методы, используемые для подключения НПП к сети, записи сетевого трафика, получения статистики и отключения НПП от сети. **ИРТК** получает интерфейс для локальных точек входа, которые необходимы для участия в записи в режиме реального времени. Этот интерфейс включает метод, который передает обратный вызов НПП.

## <a name="members"></a>Элементы

Интерфейс **ИРТК** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **ИРТК** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ИРТК** содержит следующие методы.



| Метод                                                              | Описание                                                                                                                                             |
|:--------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Configure**](irtc-configure.md)                                 | Задает триггер, соответствие шаблону и размер буфера записи.<br/>                                                                             |
| [**Подключение**](irtc-connect.md)                                     | Подключает НПП к сети.<br/>                                                                                                             |
| [**Отключение**](irtc-disconnect.md)                               | Отключает НПП от сети.<br/>                                                                                                        |
| [**жетконтролстате**](irtc-getcontrolstate.md)                     | Возвращает состояние [*записи*](c.md), которое указывает, запущена или приостановлена запись.<br/>                      |
| [**жетконверсатионстатистикс**](irtc-getconversationstatistics.md) | Извлекает [*сведения о*](s.md) [*сеансе*](s.md) и станции для текущей записи.<br/> |
| [**жеттоталстатистикс**](irtc-gettotalstatistics.md)               | Извлекает сведения о времени, буфере, драйвере и другой сетевой статистике из выполняемой в данный момент записи.<br/>                                              |
| [**Пауза**](irtc-pause.md)                                         | Временно останавливает текущую запись.<br/>                                                                                                       |
| [**куеристатионс**](irtc-querystations.md)                         | Извлекает список всех компьютеров в подсети, которые используют сетевой монитор для записи сетевых данных.<br/>                                        |
| [**QueryStatus**](irtc-querystatus.md)                             | Получает состояние НПП.<br/>                                                                                                             |
| [**Возобновить**](irtc-resume.md)                                       | Перезапускает приостановленную запись.<br/>                                                                                                                   |
| [**Начать**](irtc-start.md)                                         | Запускает запись.<br/>                                                                                                                            |
| [**Stop**](irtc-stop.md)                                           | Останавливает текущую запись.<br/>                                                                                                                   |
