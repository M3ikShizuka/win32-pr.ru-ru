# <a name="about-network-monitor-21"></a>Сведения о сетевой монитор 2,1

Сетевой монитор является компонентом Microsoft Systems Management Server (SMS), который используется для обнаружения и устранения неполадок в локальных сетях, глобальных сетях и последовательных связях с сервером Microsoft Remote Access Server (RAS). Сетевой монитор обеспечивает анализ данных по сети после записи.

В ходе анализа после захвата сетевой трафик сохраняется в собственном файле записи, чтобы захваченные данные можно было проанализировать позже. В этом случае анализ может быть указан в форме [*парсеров*](p.md) протоколов, которые выводят определенные типы кадров сети и отображают данные кадра в пользовательском интерфейсе сетевой монитор. или анализ может быть в виде [*экспертов*](e.md) , изучающих сетевые данные и отображая отчет (эксперты также могут управлять сетевыми данными).

Сетевой монитор предоставляет следующие типы функциональных возможностей:

-   Записывает сетевые данные в режиме реального времени или отложенного режима.
-   Предоставляет возможности фильтрации при сборе данных.
-   Использует экспертов и анализаторы для подробного анализа после записи.
