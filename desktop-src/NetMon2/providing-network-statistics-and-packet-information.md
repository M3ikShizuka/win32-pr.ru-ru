# <a name="providing-network-statistics-and-packet-information"></a>Предоставление статистики сети и сведений о пакетах

Поставщик сетевых пакетов (НПП) — это библиотека DLL, которая взаимодействует с драйвером сетевой монитор и предоставляет сетевую статистику и данные пакетов.

Ниже перечислены наиболее распространенные способы использования НПП:

-   Сетевая статистика.
-   Кадры, передаваемые в приложение в режиме реального времени.
-   Пакеты, доступные только после остановки записи.
