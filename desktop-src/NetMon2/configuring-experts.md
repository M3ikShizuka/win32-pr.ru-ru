# <a name="configuring-experts"></a>Настройка экспертов

Чтобы настроить экспертов, используйте пользовательские диалоговые окна и функцию [**Configure**](configure.md) . При написании экспертов по сетевым мониторам вы можете выбрать один из вариантов. Можно создать набор элементов конфигурации по умолчанию, которые можно использовать при первоначальном запуске эксперта, или разрешить сетевой монитор пользователям настраивать эксперт во время выполнения. Конфигурации для каждого эксперта хранятся сетевой монитор. В качестве параметра пользователи могут сбросить настройки эксперта к исходным параметрам по умолчанию. Имейте в виду, что можно обойти использование **настройки** , разработав эксперта, который передает фиксированную конфигурацию для сетевой монитор.
