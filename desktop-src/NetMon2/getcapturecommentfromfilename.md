# <a name="getcapturecommentfromfilename-function"></a>Функция Жеткаптурекомментфромфиленаме

Функция **жеткаптурекомментфромфиленаме** извлекает комментарий записи из [*файла записи*](c.md).

## <a name="syntax"></a>Синтаксис


```C++
DWORD  WINAPI GetCaptureCommentFromFilename(
  _In_ LPSTR lpFilename,
  _In_ LPSTR lpComment,
  _In_ DWORD BufferSize
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*лпфиленаме* \[ окне\]
</dt> <dd>

Указатель на имя файла записи.

</dd> <dt>

*лпкоммент* \[ окне\]
</dt> <dd>

Указатель на предварительно выделенную строку комментария.

</dd> <dt>

*BufferSize* \[ окне\]
</dt> <dd>

Размер строки.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена успешно (т. е. Если комментарий найден и скопирован или в файле записи нет комментариев), возвращается значение НМЕРР \_ Success.

Если функция завершается неудачно, возвращаемое значение является кодом ошибки.



| Код возврата                                                                                                 | Описание                                                                  |
|-------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------|
| <dl> <dt>**\_ \_ Ошибка чтения файла \_ нмерр**</dt> </dl>     | Не удается прочитать комментарий записи.<br/>                               |
| <dl> <dt>**НМЕРР \_ Недопустимый \_ \_ Формат файла**</dt> </dl> | Кадр комментария имеет недопустимый формат файла.<br/>                     |
| <dl> <dt>**\_файл нмерр \_ не \_ найден**</dt> </dl>      | Не удается найти файл, указанный параметром *лпфиленаме* .<br/> |
| <dl> <dt>**НМЕРР \_ Недопустимый \_ параметр**</dt> </dl>    | Один из параметров указан неправильно.<br/>                   |



 

## <a name="remarks"></a>Комментарии

[*Эксперты*](e.md) и [*средства синтаксического анализа*](p.md) могут вызывать функцию **жеткаптурекомментфромфиленаме** .

Чтобы получить комментарий записи в режиме реального времени, вызовите функцию [жеткаптурекоммент](getcapturecomment.md) .
