# <a name="querytable-structure"></a>Структура КУЕРИТАБЛЕ

Структура **куеритабле** предоставляет список компьютеров, которые в настоящее время используют сетевой монитор для записи сетевых данных.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _QUERYTABLE {
  DWORD        nStationQueries;
  STATIONQUERY StationQuery[1];
} QUERYTABLE, *LPQUERYTABLE;
```



## <a name="members"></a>Члены

<dl> <dt>

**нстатионкуериес**
</dt> <dd>

На входе — максимальное число компьютеров, которые сетевой монитор должны возвращаться.

В выходных данных — число структур [статионкуери](stationquery.md) , возвращенных сетевой монитор. Каждая структура **статионкуери** представляет компьютер, на котором в данный момент выполняется запись данных.

</dd> <dt>

**статионкуери**
</dt> <dd>

На входе — массив пустых структур [статионкуери](stationquery.md) , содержащих количество элементов, указанных в **нстатионкуериес**.

На выходе — заполненная [статионкуери](stationquery.md) структура для каждого компьютера, записывающего данные. Число заполненных элементов возвращается функцией **нстатионкуериес**.

</dd> </dl>

## <a name="remarks"></a>Комментарии

Память для этой структуры и массива [статионкуери](stationquery.md) должна быть выделена вызывающим приложением и освобождена после того, как сведения больше не нужны.
