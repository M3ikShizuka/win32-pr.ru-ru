# <a name="t-network-monitor"></a>T (сетевой монитор)

<dl> <dt>

<span id="_netmon_total_statistics_gly"></span><span id="_NETMON_TOTAL_STATISTICS_GLY"></span>**Общая статистика**
</dt> <dd>

Статистика, создаваемая отчетами сетевого адаптера (NIC) и сетевой монитор. Например, статистика для кадров сети включает число кадров, которые сообщает сетевой адаптер, число кадров, обрабатываемых сетевым адаптером, число кадров, переданных через фильтр записи, и число кадров, хранящихся в буфере или файле записи. Полный список этих статистических данных см. в разделе Структура **статистики** .

</dd> </dl>
