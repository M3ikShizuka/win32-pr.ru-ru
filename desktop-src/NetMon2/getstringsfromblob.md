# <a name="getstringsfromblob-function"></a>Функция Жетстрингсфромблоб

Функция **жетстрингсфромблоб** использует последовательные вызовы для получения всех строк в указанных диапазонах.

## <a name="syntax"></a>Синтаксис


```C++
DWORD GetStringsFromBlob(
  _In_        HBLOB hBlob,
  _In_  const char  *pRequestedOwnerName,
  _In_  const char  *pRequestedCategoryName,
  _In_  const char  *pRequestedTagName,
  _Out_ const char  **ppReturnedOwnerName,
  _Out_ const char  **ppReturnedCategoryName,
  _Out_ const char  **ppReturnedTagName,
  _Out_ const char  **ppReturnedString,
  _Out_       DWORD *pRestartKey
);
```

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена успешно, возвращается значение НМЕРР \_ Success.

Если функция завершается неудачно, возвращается значение НМЕРР, указывающее на проблему.

Если указанная комбинация сведений о **владельце**, **категории** и **теге** не существует, возвращаемое значение — это **нмерр \_ запись большого двоичного объекта \_ \_ \_ \_**.

При первоначальном просмотре большого двоичного объекта в пределах указанных границ функция возвращает **\_ запись большого двоичного \_ объекта \_ \_ \_ нмерр**, а параметр *престарткэй* указывает на ноль.

## <a name="remarks"></a>Комментарии

При первом вызове функции **жетстрингсфромблоб** параметр *престарткэй* указывает на переменную, содержащую нулевое значение. Предустановленные параметры можно использовать только в том *случае, если* ключ перезапуска равен нулю. В последующих вызовах, когда *престарткэй* имеет ненулевые значения, заданные *параметры игнорируются* . При первом вызове ALL может указывать на **значение NULL**, которое настраивает запрос для возврата каждой записи в большом двоичном объекте, по одному на последующий вызов.

Указание владельца ограничивает возвращаемые строки только этим владельцем. Аналогичное ограничение имеет значение true для категорий и тегов. Кроме того, при указании категории необходимо указать владельца, а если указать тег, необходимо указать категорию (и, следовательно, владельца).

Когда начальный вызов **жетстрингсфромблоб** возвращает, *престарткэй* указывает на новое значение, которое должно быть указано при следующем вызове функции для получения следующего значения.
