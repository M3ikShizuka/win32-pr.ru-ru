# <a name="about-event-reference-pages"></a>Сведения о страницах ссылок на события

Страница ссылки на события (ERP) — это HTML-документ, который предоставляет сетевой монитор сведения о событиях, обнаруженных во время работы эксперта или монитора.

ERP можно просмотреть в Просмотр событий сетевой монитор, средства управления монитора или в любом браузере, поддерживающем функции HTML Microsoft Internet Explorer 4,01 или более поздней версии. Это значит, что в ERP можно добавить поддерживаемые элементы управления или скриптовые языки.

Однако ERP отображается только в том случае, если эксперт обозначает HTML-документ по имени. При правильном обозначении ERP отображается в нижней области, когда выбрано событие в Просмотр событий. На рисунке ниже показан ERP, связанный с монитором диапазона IP-адресов.

## <a name="expert-events"></a>Мероприятия экспертов

События экспертов определяются элементом **евентидент** структуры [**нмевентдата**](nmeventdata.md) . При написании эксперта вы определяете значения **евентидент** , которые обычно пронумерованы 1, 2, 3 и т. д. Например, предположим, что эксперт создает два события (**EventIdent1** и **EventIdent2**). Если требуется, чтобы Просмотр событий отображал события отдельно, необходимо создать два отдельных документа ERP.
