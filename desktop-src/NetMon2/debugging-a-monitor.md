# <a name="debugging-a-monitor"></a>Отладка монитора

В следующей процедуре показано, как выполнить отладку монитора.

**Отладка монитора**

1.  Установите DLL монитора и файл базы данных программы (. pdb), созданный при компиляции монитора в нужное расположение. Дополнительные сведения см. [в разделе Установка монитора в сетевой монитор](installing-a-monitor-to-network-monitor.md) .
2.  Убедитесь, что служба управления монитором настроена как сервер, а не как служба, выбрав Панель управления, службы.
3.  Откройте командную строку и перейдите в каталог сетевой монитор. Тип:

    **Mcsvc.exe/RegServer**

    После завершения сеанса отладки монитора можно вернуть МКСВК к его условию по умолчанию, введя:

    **Mcsvc.exe свойством/Service**

4.  В Microsoft Visual Studio запустите Microsoft Visual C++.
5.  В меню **файл** выберите пункт **Открыть** , а затем выберите имя библиотеки DLL монитора.
6.  После загрузки Microsoft Visual C++ щелкните **проект**, **Параметры**.
7.  Перейдите на вкладку **Отладка** в разделе **исполняемый файл** для сеанса отладки и введите полный путь к Mcsvc.exe. Например, C: \\ Program Files \\ Netmon2 \\Mcsvc.exe.
8.  Задайте точки останова в коде.

> [!Note]  
> Не используйте окно сообщения для отслеживания отладки. Если МКСВК работает как служба, всплывающее окно не появится, и МКСВК будет выглядеть под зависание.
