# <a name="npp-structures"></a>Структуры НПП

В этом разделе описываются структуры НПП, используемые методами НПП. Эти структуры используются для получения статистики, предоставления состояния системы и статистических данных и указания компьютеров, на которых выполняется сетевой монитор. Эти структуры описаны в следующих таблицах.



| Структуры НПП                     | Описание                                                                                                                      |
|------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [сессионстатс](sessionstats.md)   | Предоставляет сведения о сеансе при извлечении статистики диалога.                                                         |
| [статионстатс](stationstats.md)   | Предоставляет статистику о конкретной [*станции*](s.md).                                                     |
| [STATISTICS](statistics.md)       | Предоставляет статистику сети при получении всей статистики и о том, что текущая запись приостановлена или остановлена.              |
| [куеритабле](querytable.md)       | Указывает, какие компьютеры используют сетевой монитор.                                                                             |
| [статионкуери](stationquery.md)   | Предоставляет сведения о компьютере, использующем сетевой монитор. Эта структура используется структурой **КУЕРИТАБЛЕ** НПП. |
| [NETWORKSTATUS](networkstatus.md) | Предоставляет текущее состояние сети с точки зрения состояния системы.                                                                        |
