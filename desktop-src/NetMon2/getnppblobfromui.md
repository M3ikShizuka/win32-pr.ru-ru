# <a name="getnppblobfromui-function"></a>Функция Жетнппблобфромуи

Функция **жетнппблобфромуи** выбирает регистрацию сетевого адаптера.

## <a name="syntax"></a>Синтаксис


```C++
DWORD GetNPPBlobFromUI(
  _In_  HWND  hwnd,
  _In_  HBLOB hFilterBlob,
  _Out_ HBLOB *phBlob
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*HWND* \[ окне\]
</dt> <dd>

Маркер окна, отображающего диалоговое окно **Выбор сети** .

</dd> <dt>

*хфилтерблоб* \[ окне\]
</dt> <dd>

Маркер для [*фильтра BLOB*](f.md) , используемый для ограничения отображаемых сетевых интерфейсов.

</dd> <dt>

*фблоб* \[ заполняет\]
</dt> <dd>

Указатель на маркер большого двоичного объекта, представляющий выбранную сетевую карту.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена успешно (пользователь выбирает сетевую карту), возвращаемое значение — НМЕРР \_ Success, а BLOB-объект, на который указывает *фблоб* , заполнен.

Если пользователь не выбрал сетевую карту, возвращается значение **нмерр \_ No \_ НПП \_**.

Если функция завершается неудачно, возвращается значение другого НМЕРР.

## <a name="remarks"></a>Комментарии

При вызове сетевой монитор отображает диалоговое окно **Выбор сети** , которое можно использовать для выбора сетевого адаптера. Большой двоичный объект НПП, представляющий сетевую карту, возвращается вызывающему приложению.

Если большой двоичный объект с именем *хфилтерблоб* является [*специальным BLOB-объектом*](s.md), он попытается обработать его. В качестве примера можно привести вызов, который ранее возвращал Специальный большой двоичный объект из удаленного НПП. Приложение вставило требуемый тег, \_ имя компьютера. В этом случае Finder передает этот большой двоичный объект удаленному НПП, который затем возвращает таблицу из НПП BLOB-объектов, представляющих запрошенный компьютер. Эти удаленные BLOB-объекты НПП будут отображаться в диалоговом окне.

Вызывающий объект должен вызвать функцию [дестройблоб](destroyblob.md) , которая уничтожает ВОЗВРАЩЕННЫЙ большой двоичный объект, когда он больше не требуется.
