# <a name="testing-a-monitor"></a>Тестирование монитора

После установки монитора можно протестировать его с помощью динамических данных из сетевых операций или проверить известный набор событий, записывая данные с помощью сетевой монитор, сохранив файл записи, а затем передать данные локально. (Дополнительные сведения о передаче файла записи в сеть см. в разделе сетевой монитор Справка в Интернете.)
