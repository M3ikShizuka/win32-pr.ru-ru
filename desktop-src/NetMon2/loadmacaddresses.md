# <a name="loadmacaddresses-function"></a>Функция Лоадмакаддрессес

Функция **лоадмакаддрессес** вызывается монитором для заполнения списка MAC-адресов адресами, полученными из строковой переменной конфигурации HTML.

## <a name="syntax"></a>Синтаксис


```C++
BOOL LoadMACAddresses(
  _In_  const char   *pConfig,
  _In_  const char   *pVarName,
  _Out_       LPBYTE *ppAddresses,
  _Out_       DWORD  *pNumAddresses
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*pConfig* \[ окне\]
</dt> <dd>

Указатель на строку конфигурации HTML, переданную в монитор методом [имонитор::D оконфигуре](imonitor-doconfigure.md) .

</dd> <dt>

*пварнаме* \[ окне\]
</dt> <dd>

Указатель на имя переменной в строке конфигурации.

</dd> <dt>

*ппаддрессес* \[ заполняет\]
</dt> <dd>

Указатель на указатель на массив адресов. Если переменная, указанная в *пварнаме* , найдена и имеет ненулевую длину, функция выделяет достаточно места (с помощью **хеапаллокмемори**) и сохраняет все Mac-адреса в виде массива.

</dd> <dt>

*пнумаддрессес* \[ заполняет\]
</dt> <dd>

Указатель на переменную **DWORD** , для которой задано количество MAC-адресов, полученных из строки конфигурации.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена успешно, (имя переменной было обнаружено и имело строку ненулевой длины, представляющую MAC-адреса), возвращается значение **true**.

Если функция завершается неудачно, возвращается значение **false**.
