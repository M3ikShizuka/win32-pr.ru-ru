# <a name="input-source-identification"></a>Идентификация источника входных данных

## <a name="purpose"></a>Назначение

В подразделах этого раздела приводится обзор поддержки идентификации источника входных данных в Windows 8. Идентификация источника входных данных позволяет приложениям точно и надежно обнаруживать источник входных данных, определять тип входных данных и обеспечивать взаимодействие с пользователем, наиболее подходящее для входного типа.

В этом разделе описывается, как приложение может получить эти сведения и лучше использовать его для оптимизации взаимодействия с пользователем.

## <a name="developer-audience"></a>Аудитория разработчиков

API идентификации источника входных данных предназначены для разработчиков, создающих платформы пользовательского интерфейса, которые обеспечивают единообразное взаимодействие с пользователями в настольных приложениях.
