# <a name="statistics-functions"></a>Статистические функции

Windows операционная система накапливает набор операционных статистических данных для рабочих станций и серверов с момента запуска службы рабочей станции или сервера. Чтобы получить эти статистические данные, можно вызвать следующую функцию статистики управления сетью.



| Функция                                     | Описание                                                                                 |
|----------------------------------------------|---------------------------------------------------------------------------------------------|
| [**нетстатистиксжет**](/windows/desktop/api/Lmstats/nf-lmstats-netstatisticsget) | Извлекает операционную статистику для службы. поддерживает службы рабочей станции и сервера. |



 

Функция **нетстатистиксжет** возвращает структуру [**stat \_ Workstation \_ 0**](/windows/win32/api/lmstats/ns-lmstats-stat_workstation_0-r1) при запросе статистики рабочей станции. функция возвращает структуру [**\_ сервера stat \_ 0**](/windows/desktop/api/Lmstats/ns-lmstats-stat_server_0) при запросе статистики сервера.
