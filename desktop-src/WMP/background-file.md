# <a name="background-file"></a>Фоновый файл

Каждой обложке требуется фоновый файл, который содержит изображение, которое пользователь увидит при первом отображении обложки. Этот файл определен как фоновый файл в файле определения обложки.

На следующем рисунке изображен типичный фоновый файл.

![фоновый файл](images/cesdkbak.png)

Это фоновое изображение, которое пользователи увидят первыми. Тем не менее, они также увидят два изображения Thumb в соответствующих местах на TrackBar для поиска (выше) и тома (ниже). Кроме того, все отключенные кнопки будут отображаться с использованием отключенных образов.
