# <a name="equalizersettingscurrentpresettitle"></a>ЕКУАЛИЗЕРСЕТТИНГС. Куррентпресеттитле

Атрибут **куррентпресеттитле** извлекает заголовок текущей предустановки.

``` syntax
        elementID.currentPresetTitle
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой**, предназначенной только для чтения.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
