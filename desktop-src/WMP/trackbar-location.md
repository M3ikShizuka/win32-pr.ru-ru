# <a name="trackbar-location"></a>Расположение TrackBar

Необходимо определить расположение линейки воспроизведения, которую вы хотите использовать. Необходимо указать четыре положительных целых числа, разделенные запятыми, которые определяют левую, верхнюю, ширину и высоту TrackBar (в пикселях) относительно фонового изображения.


```C++
180,46,50,30 

```
