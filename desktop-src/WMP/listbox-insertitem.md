# <a name="listboxinsertitem"></a>LISTBOX. insertItem

Метод **insertItem** Вставляет указанный текст по указанному индексу в элементе управления "список".

``` syntax
        elementID.insertItem(index, text)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*номер*
</dt> <dd>

**Число** (**Long**), содержащее индекс строки для извлечения.

</dd> <dt>

<span id="text"></span><span id="TEXT"></span>*полнотекстовым*
</dt> <dd>

**Строка** , содержащая вставляемый текст.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Remarks

При вставке строки индексы строки под вставленной строкой увеличиваются на единицу.
