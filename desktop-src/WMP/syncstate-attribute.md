# <a name="syncstate-attribute"></a>Атрибут Синкстате

атрибут **синкстате** представляет собой строковое представление 32-разрядного значения, которое проигрыватель Windows Media использует при синхронизации списков воспроизведения с портативными устройствами.

## <a name="applies-to"></a>Применяется к

-   [Звуковые элементы](audio-item-attributes.md)
-   [Другие элементы](other-item-attributes.md)
-   [Элементы фото](photo-item-attributes.md)
-   [Элементы видео](video-item-attributes.md)

## <a name="remarks"></a>Remarks

Этот атрибут состоит из 16 2-разрядных значений, каждый из которых указывает состояние синхронизации для переносимого устройства. Наиболее значимый бит (старший байт) этого 32-битного значения соответствует устройству 16. Наименее важный бит (ЛСБ) соответствует устройству 1.

значение 1-й бит каждого из двух битов указывает, синхронизировано ли содержимое с соответствующим устройством проигрыватель Windows Media. Значение 1 указывает на это. Значение 0 указывает, что это не так.

Если значение 0 – 1, ЛСБ указывает причину сбоя синхронизации. Значение 1 в ЛСБ указывает на недостаточный объем свободного места для содержимого. Значение 0 в ЛСБ указывает на другую причину, препятствующую синхронизации.

Чтобы получить состояние синхронизации данного устройства, необходимо выполнить следующие действия.

1.  Вызов **ивмпсинкдевице:: Get \_ Status** для определения, синхронизировано ли данное устройство.
2.  Если оно синхронизировано, вызовите **ивмпсинкдевице:: Get \_ партнершипиндекс** , чтобы получить индекс битовой пары устройства в атрибуте **синкстате** .
3.  Используя этот индекс, маску соответствующей битовой пары атрибута **синкстате** и проверьте результат, чтобы определить состояние синхронизации списка воспроизведения с устройством.

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .
