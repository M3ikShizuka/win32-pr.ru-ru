# <a name="ripping-a-cd"></a>Копирование компакт-диска

существует два способа копирования компакт-диска с помощью элемента управления проигрыватель Windows Media. проигрыватель Windows Media 9 может копировать компакт-диски с помощью **ивмпплайерсервицес:: сеттаскпане**. в проигрыватель Windows Media 11 появился интерфейс **ивмпкдромрип** для копирования компакт-дисков. Этот интерфейс предоставляет больше функций, чем **ивмпплайерсервицес:: сеттаскпане**, и рекомендуется использовать **ивмпкдромрип** , если он доступен.

в следующих разделах описывается копирование компакт-диска с помощью проигрыватель Windows Media.

-   [Копирование с помощью интерфейса Ивмпкдромрип](ripping-by-using-the-iwmpcdromrip-interface.md)
-   [Копирование с помощью Ивмпплайерсервицес:: Сеттаскпане](ripping-by-using-iwmpplayerservices--settaskpane.md)
