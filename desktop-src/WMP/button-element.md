# <a name="button-element"></a>BUTTON, элемент

Элемент **Button** предоставляет способ создания кнопки в обложке. Приведенные ниже атрибуты можно использовать для настройки поведения кнопки, или для удобства можно использовать предопределенную кнопку.

Элемент **Button** поддерживает следующие атрибуты.



| attribute                                         | Описание                                                                                                                                      |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [курсор](button-cursor.md)                       | Указывает или получает курсор, который появляется при наведении указателя мыши на **кнопку**.                                                |
| [дисабледимаже](button-disabledimage.md)         | Указывает или получает изображение, отображаемое при отключенной **кнопке** .                                                                  |
| [крывающей](button-down.md)                           | Указывает или получает значение, указывающее, находится ли кнопка в положении **"вверх" или "вниз"** .                                                  |
| [довнимаже](button-downimage.md)                 | Указывает или получает изображение, представляющее состояние **кнопки "вниз"**.                                                                  |
| [довнтултип](button-downtooltip.md)             | Указывает или получает текст подсказки, отображаемый при наведении указателя мыши на **кнопку** , а **кнопка** находится в состоянии "вниз" или "нажато". |
| [ховердовнимаже](button-hoverdownimage.md)       | Указывает или получает изображение, отображаемое, когда **кнопка** находится в состоянии Down, и пользователь наводит указатель мыши на него.          |
| [ховеримаже](button-hoverimage.md)               | Указывает или получает изображение, отображаемое, когда кнопка находится в состоянии **"вверх"** , и пользователь наводит указатель мыши на него.            |
| [image](button-image.md)                         | Указывает или получает изображение **кнопки** по умолчанию.                                                                                      |
| [прикрепленные](button-sticky.md)                       | Указывает или получает значение, указывающее, является ли **кнопка** переключателем, то есть является ли она кнопкой с двумя состояниями или с одним состоянием.         |
| [крытия](button-tiled.md)                         | Указывает или получает значение, указывающее, будет ли изображение **кнопки** мозаичным.                                                            |
| [транспаренциколор](button-transparencycolor.md) | Указывает или получает цвет, который будет прозрачным в изображении **кнопки** .                                                               |
| [Подсказка](button-uptooltip.md)                 | Указывает или получает текст подсказки, отображаемый при наведении указателя мыши на **кнопку** , и **кнопка** находится в состоянии "работает".                |



 

Элемент **Button** поддерживает атрибуты окружения и может реализовывать обработчики событий окружения. Дополнительные сведения см. в разделе [атрибуты окружения](ambient-attributes.md) и [обработчики событий окружения](ambient-event-handlers.md).

Стандартные кнопки — это обычные элементы **кнопки** с различными параметрами общих атрибутов, заданных по умолчанию. Доступны следующие стандартные кнопки.



| Предопределенная кнопка                    | Описание                                                                        |
|--------------------------------------|------------------------------------------------------------------------------------|
| [CLOSEBUTTON](closebutton.md)       | **Кнопка** , используемая для закрытия проигрывателя.                                             |
| [ффвдбуттон](ffwdbutton.md)         | **Кнопка** с встроенным вызовом **Player. Controls. фастфорвард** при щелчке. |
| [IMAGEBUTTON](imagebutton.md)       | **Кнопка** , используемая для вывода изображения.                                             |
| [минимизебуттон](minimizebutton.md) | **Кнопка** , используемая для сворачивания проигрывателя.                                          |
| [мутебуттон](mutebutton.md)         | **Кнопка** , используемая для выключения и отключения звука.                                   |
| [NEXTBUTTON](nextbutton.md)         | **Кнопка** с встроенным вызовом **Player. Controls. Next** при нажатии.        |
| [паусебуттон](pausebutton.md)       | **Кнопка** с встроенным вызовом **Player. Controls. Pause** при щелчке.       |
| [PLAYBUTTON](playbutton.md)         | **Кнопка** с встроенным вызовом **Player. Controls. Play** при щелчке.        |
| [превбуттон](prevbutton.md)         | **Кнопка** с встроенным вызовом **Player. Controls. Previous** при щелчке.    |
| [REPEATBUTTON](repeatbutton.md)     | **Кнопка** , которая переключает параметр REPEAT.                                       |
| [ретурнбуттон](returnbutton.md)     | **Кнопка** , которая возвращает проигрыватель Windows Media в Media Center.                |
| [ревбуттон](rewbutton.md)           | **Кнопка** с встроенным вызовом **Player. Controls. фастреверсе** при щелчке. |
| [шуффлебуттон](shufflebutton.md)   | **Кнопка** , которая переключает параметр "случайный".                                      |
| [стопбуттон](stopbutton.md)         | **Кнопка** с встроенным вызовом **Player. Controls. останавливаться** при щелчке.        |
