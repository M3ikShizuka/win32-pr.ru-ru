# <a name="onclick"></a>OnClick

Обработчик событий **OnClick** обрабатывает событие, возникающее, когда пользователь щелкает элемент.

``` syntax
onclick
```

## <a name="remarks"></a>Remarks

Этот обработчик событий неприменим к элементу **списка воспроизведения** , элементу **Popup** , элементу **Video** , элементу **ListBox** , когда *видео*. **безоконное** значение равно false или элемент **Effects** при *эффекте*. **окно** имеет значение true.
