# <a name="externalselectedtaskpane"></a>External. Селектедтаскпане

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Свойство **селектедтаскпане** указывает или получает текущую выбранную область задач.

## <a name="syntax"></a>Синтаксис

Window. external. Селектедтаскпане = *сервицетаск*

## <a name="possible-values"></a>Возможные значения

Это свойство является **строкой** для чтения и записи. Возможные значения: "ServiceTask1", "ServiceTask2" и "ServiceTask3".

## <a name="remarks"></a>Комментарии

Указание значения для этого свойства подсвечивает кнопку для этой панели. Указанная область задач не активна. Необходимо указать значение этого свойства, чтобы задать текущую кнопку области задач для веб-страницы при загрузке страницы, чтобы убедиться, что активна правильная кнопка области задач.

Чтобы сделать конкретную область задач активной, используйте метод **навигатетаскпанеурл** .
