# <a name="sliderforegroundendcolor"></a>ПОЛЗУНок. Фореграундендколор

Атрибут **фореграундендколор** указывает или получает конечный цвет переднего плана элемента управления "ползунок".

``` syntax
        elementID.foregroundEndColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей любое значение цвета Microsoft Internet Explorer. У него нет значения по умолчанию.

## <a name="remarks"></a>Комментарии

**Фореграундендколор** используется в сочетании с **foregroundColor**. Созданный эффект является градиентным эффектом от **foregroundColor** к **фореграундендколор**.
