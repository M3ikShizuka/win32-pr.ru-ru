# <a name="about-the-network-object"></a>О сетевом объекте

Объект **Network** управляет свойствами, позволяющими определить, насколько хорошо содержимое потоковой передачи через сеть. Например, можно узнать, теряются ли пакеты, и выполнить соответствующие действия. Доступ к **сетевому** объекту осуществляется только через свойство **Network** объекта **Player** . Свойство **Network** возвращает объект **Network** . Доступ к свойствам **сетевого** объекта можно получить только после его создания. Например, чтобы использовать свойство **пропускной способности** , необходимо использовать следующий код:


```C++
mybandwidth = player.network.bandwidth;

```
