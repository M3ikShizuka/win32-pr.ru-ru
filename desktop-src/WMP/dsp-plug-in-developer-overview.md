# <a name="dsp-plug-in-developer-overview"></a>Обзор подключаемого модуля DSP

С точки зрения разработчика, подключаемые модули DSP — это программные программы, которые получают аудио-или видеоданные, предоставляемые проигрывателем Windows Media непосредственно перед стадией отрисовки, затем обрабатывают эти данные и возвращают данные в проигрыватель Windows Media для подготовки к просмотру.

Следующие разделы содержат общие сведения о подключаемых модулях DSP в проигрывателе Windows Media.

-   [Входные и выходные данные](data-input-and-output.md)
-   [Формат согласования](format-negotiation.md)
-   [Предоставление пользовательского интерфейса](providing-a-user-interface.md)

-   [Требуемые интерфейсы](required-interfaces.md)
-   [Регистрация подключаемых модулей DSP](registering-dsp-plug-ins.md)
-   [Обновление существующих подключаемых модулей DSP](updating-existing-dsp-plug-ins.md)
