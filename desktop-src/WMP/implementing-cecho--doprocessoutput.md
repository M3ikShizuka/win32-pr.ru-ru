# <a name="implementing-cechodoprocessoutput"></a>Реализация Цечо::D Опроцессаутпут

Метод **допроцессаутпут** выполняет обработку цифровых сигналов. это метод, который вносит изменения в данные, предоставляемые проигрыватель Windows Media. Это результаты этого метода, которые будут слышаться в виде эхо-эффекта при завершении подключаемого модуля Echo Sample.

В этом примере подключаемый модуль будет обрабатывать только 8-разрядный или 16-разрядный звук. Необходимо внести некоторые изменения в образец кода мастера подключаемых модулей, чтобы удалить разделы, которые обрабатывают звуковые файлы более высокого уровня. Однако стоит изучить эти разделы, так как вы можете добавить собственную реализацию эха для этих форматов.

В следующих разделах подробно описаны изменения, которые необходимо внести в код.

-   [Обработка звуковых данных](processing-the-audio-data.md)
-   [Переменные для выполнения обработки](variables-to-perform-processing.md)
