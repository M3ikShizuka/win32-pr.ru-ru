# <a name="customsliderdownimage"></a>КУСТОМСЛИДЕР. Довнимаже

Атрибут **довнимаже** указывает или получает изображение состояния "вниз" настраиваемого ползунка.

``` syntax
        elementID.downImage
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей имя файла изображения.

## <a name="remarks"></a>Комментарии

Этот атрибут является необязательным. Если он не указан, будет использоваться файл, указанный в атрибуте **Image** .

**Довнимаже** представляет состояние down элемента управления **кустомслидер** . Это может быть один образ или несколько изображений, представляющих различные состояния ползунка. Если используется несколько образов, они упорядочиваются так же, как и файл **изображения** .

Поддерживаются следующие типы файлов изображений: BMP, JPG, PNG и GIF (не включая анимированные GIF).
