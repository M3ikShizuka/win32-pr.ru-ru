# <a name="controlsisavailable"></a>Элементы управления. доступно

Свойство **Available** указывает, доступен ли указанный тип сведений, или может быть выполнено указанное действие.

``` syntax
player.controls.isAvailable(
        name
        )
```

## <a name="parameters"></a>Параметры

*name*

**Строка** , содержащая одно из следующих значений.



| Строка          | Описание                                                                                                                                                       |
|-----------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| currentItem     | Определяет, может ли пользователь задать свойство **currentItem** .                                                                                                 |
| куррентмаркер   | Определяет, может ли пользователь выполнять поиск определенного маркера.                                                                                                        |
| currentPosition | Определяет, может ли пользователь искать определенную точку в файле. Некоторые файлы не поддерживают поиск.                                                       |
| фастфорвард     | Определяет, поддерживает ли файл быструю пересылку и можно ли вызывать эту функцию. Многие типы файлов (или динамические потоки) не поддерживают Фастфорвард. |
| фастреверсе     | Определяет, поддерживает ли файл Фастреверсе и можно ли вызывать эту функцию. Многие типы файлов (или динамические потоки) не поддерживают Фастреверсе.     |
| Далее            | Определяет, может ли пользователь перейти к следующей записи списка воспроизведения.                                                                                             |
| pause           | Определяет, доступен ли метод **Pause** .                                                                                                             |
| играть            | Определяет, доступен ли метод **Play** .                                                                                                              |
| Предыдущее время        | Определяет, может ли пользователь перейти к предыдущей записи списка воспроизведения.                                                                                         |
| Шаг            | Определяет, доступен ли метод **Step** во время воспроизведения.                                                                                              |
| stop            | Определяет, доступен ли метод " **останавливаться** ".                                                                                                              |



 

## <a name="return-values"></a>Возвращаемые значения

Этот метод возвращает **логическое** значение.

## <a name="examples"></a>Примеры

В следующем примере создается HTML-элемент BUTTON, который выполняет поиск начальной позиции текущего элемента мультимедиа. в JScriptном коде **для проверки** того, поддерживает ли файл операцию поиска, можно использовать параметр. Объект **Player** создан с идентификатором "Player".


```JScript
<INPUT TYPE = "BUTTON"  ID = "START"  NAME = "START"  VALUE = "Seek To Zero"

    /* If supported, seek to position zero. */
    onClick = "if (Player.controls.isAvailable('CurrentPosition'))
        Player.controls.currentPosition = 0;
">
```
