# <a name="building-a-dsp-plug-in"></a>Создание подключаемого модуля DSP

подключаемый модуль DSP можно создать с помощью Visual Studio и мастера подключаемых модулей проигрыватель Windows Media. Сведения о настройке мастера и создании подключаемого модуля см. в разделе [Создание подключаемого модуля](building-a-plug-in.md).

При запуске мастера вам будет предоставлено несколько различных типов подключаемых модулей. Выберите **подключаемый модуль DSP**.

В следующих разделах описываются функции образцов подключаемых модулей DSP аудио и видео, которые можно построить с помощью мастера.

-   [Сведения о примере подключаемого модуля DSP аудиоподсистемы](about-the-sample-audio-dsp-plug-in.md)
-   [О примере подключаемого модуля DSP видео](about-the-sample-video-dsp-plug-in.md)
