# <a name="playlistplaylistitemsvisible"></a>Список воспроизведения. Плайлиститемсвисибле

Атрибут **плайлиститемсвисибле** указывает или получает значение, указывающее, является ли область элементов списка воспроизведения видимой.

``` syntax
        elementID.playlistItemsVisible
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                              |
|-------|------------------------------------------|
| Да  | По умолчанию. Область элементов списка воспроизведения видима. |
| false | Область элементов списка воспроизведения не видна.      |



 

## <a name="remarks"></a>Remarks

Область элементы списка воспроизведения включает заголовки столбцов, содержимое столбцов и полосы прокрутки (если они есть).

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
