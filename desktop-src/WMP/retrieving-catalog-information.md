# <a name="retrieving-catalog-information"></a>Получение сведений о каталоге

Вы можете отобразить диагностическую информацию в каталоге, запустив каткомп со следующим синтаксисом:


```C++
catcomp info <catalogpath> [track|artist|album] [ID]
```



Например, следующая команда отображает сведения обо всем каталоге, включая версию, языковой стандарт и внутреннюю информацию по элементам каталога:


```C++
catcomp info C:\Catalog210\catalog.wmdb
```



Ниже отображаются сведения о дорожке с ИДЕНТИФИКАТОРом 3256:


```C++
catcomp info C:\Catalog210\catalog.wmdb track 3256
```
