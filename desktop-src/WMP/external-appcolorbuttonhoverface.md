# <a name="externalappcolorbuttonhoverface"></a>External. Аппколорбуттонховерфаце

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

свойство **аппколорбуттонховерфаце** извлекает цвет текущего нажатия кнопки для пользовательского интерфейса проигрыватель Windows Media. Этот цвет отображается, когда пользователь наводит указатель мыши на кнопку.

``` syntax
window.external.appColorButtonHoverFace
      
```

## <a name="possible-values"></a>Возможные значения

Это свойство является **строкой**, доступная только для чтения.

## <a name="remarks"></a>Remarks

пользователи могут изменять цвет пользовательского интерфейса проигрыватель Windows Media. Это свойство можно использовать для настройки внешнего вида размещенной веб-страницы в соответствии с проигрывателем.
