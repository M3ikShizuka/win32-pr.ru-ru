# <a name="externalappcolorbuttonhoverface"></a>External. Аппколорбуттонховерфаце

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Свойство **аппколорбуттонховерфаце** получает текущий цвет при наведении на кнопку для пользовательского интерфейса проигрывателя Windows Media. Этот цвет отображается, когда пользователь наводит указатель мыши на кнопку.

``` syntax
window.external.appColorButtonHoverFace
      
```

## <a name="possible-values"></a>Возможные значения

Это свойство является **строкой**, доступная только для чтения.

## <a name="remarks"></a>Комментарии

Пользователи могут изменять цвет пользовательского интерфейса проигрывателя Windows Media. Это свойство можно использовать для настройки внешнего вида размещенной веб-страницы в соответствии с проигрывателем.
