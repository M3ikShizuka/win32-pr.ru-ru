# <a name="echo-sample-properties"></a>Свойства образца Echo

Образец Echo предоставляет два свойства: время задержки и уровень влияния (сочетание задержек). Значение для уровня сигнала сухой (сухой Mix) всегда берется из значения Mix. Необходимо изменить существующий код и добавить новый код, чтобы сделать эти свойства доступными.

В следующих разделах объясняется, как изменить код свойств.

-   [Как работают свойства](how-properties-work.md)
-   [Переменные для хранения свойств](variables-to-store-properties.md)
-   [Изменение свойства Scale](modifying-the-scale-property.md)
-   [Добавление свойства "Mix!"](adding-the-wet-mix-property.md)
