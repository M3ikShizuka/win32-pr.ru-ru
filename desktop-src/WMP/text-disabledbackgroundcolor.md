# <a name="textdisabledbackgroundcolor"></a>TEXT. Дисабледбаккграундколор

Атрибут **дисабледбаккграундколор** указывает или получает цвет фона, используемый для элемента управления "текст", когда он отключен.

``` syntax
        elementID.disabledBackgroundColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей любое значение цвета Microsoft Internet Explorer.

## <a name="remarks"></a>Комментарии

Если **дисабледбаккграундколор** не указан, используется **backgroundColor** .

См. атрибут [value](text-value.md) для примера, иллюстрирующий использование атрибутов **текстового** элемента.
