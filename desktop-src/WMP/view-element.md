# <a name="view-element"></a>VIEW, элемент

Элемент **View** содержит сведения о пользовательском интерфейсе обложки и использует атрибуты, методы и обработчики событий, приведенные в следующих таблицах.

Несколько элементов **представления** могут быть определены как дочерние элементы **темы** обложки для предоставления различных интерфейсов в различных ситуациях. Элементы **представления** не могут быть потомками любого другого элемента и содержат все остальные элементы обложки. Обратите внимание, что каждое представление имеет собственную область переменной, то есть не может совместно использовать значения атрибутов с другими представлениями.

Глобальный атрибут **View** можно использовать для ссылки на конкретный элемент **представления** из любого места внутри него. Это альтернатива использованию атрибута **ID** , который должен использоваться в других элементах **представления** или в элементе **Theme** .

Элемент **View** поддерживает следующие атрибуты. Атрибуты, отмеченные звездочкой ( \* ), также поддерживаются элементом вложенного **представления** .



| attribute                                                       | Описание                                                                                                             |
|-----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [backgroundColor](view-backgroundcolor.md)\*                  | Указывает или получает цвет фона для **представления** или **подпредставления**.                                             |
| [backgroundImage](view-backgroundimage.md) \*                  | Указывает или получает фоновое изображение **представления** или **подпредставления**.                                             |
| [баккграундимажехуешифт](view-backgroundimagehueshift.md)     | Указывает или получает величину, на которую смещается цветовой тон фонового изображения.                                  |
| [баккграундимажесатуратион](view-backgroundimagesaturation.md) | Указывает или получает значение насыщенности фонового изображения.                                                    |
| [баккграундтилед](view-backgroundtiled.md)\*                  | Указывает или получает значение, указывающее, производится ли мозаичное заполнение фонового изображения **представления** или **подпредставления** .         |
| [category](view-category.md)                                   | Указывает или получает категорию, для которой будет отображаться пользовательский интерфейс.                                           |
| [фокусобжектид](view-focusobjectid.md)                         | Указывает или получает значение, указывающее, какой элемент имеет фокус клавиатуры.                                             |
| [maxHeight](view-maxheight.md)                                 | Указывает или получает максимальную высоту **представления** при изменении размера.                                                |
| [maxWidth](view-maxwidth.md)                                   | Указывает или получает максимальную ширину **представления** при изменении размера.                                                 |
| [minHeight](view-minheight.md)                                 | Указывает или получает минимальную высоту **представления** при изменении размера.                                                |
| [minWidth](view-minwidth.md)                                   | Указывает или получает минимальную ширину **представления** при изменении размера.                                                 |
| [изменяемым размером](view-resizable.md)                                 | Указывает или получает значение, указывающее, можно ли изменить размер **представления** .                                          |
| [ресизебаккграундимаже](view-resizebackgroundimage.md)         | Указывает или получает значение, указывающее, можно ли изменить размер фонового изображения.                                  |
| [scriptFile](view-scriptfile.md)                               | Указывает имена файлов сопутствующих файлов JScript.                                                                 |
| [тимеринтервал](view-timerinterval.md)                         | Указывает или получает интервал (в миллисекундах), в течение которого таймер запускает события в обработчик событий **OnTime** . |
| [title](view-title.md)                                         | Указывает или получает заголовок **представления**. Можно задать только во время разработки.                                       |
| [titleBar](view-titlebar.md)                                   | Указывает или получает значение, указывающее, отображается ли заголовок окна.                                        |
| [транспаренциколор](view-transparencycolor.md)\*              | Указывает или получает цвет прозрачности фонового изображения.                                                  |



 

Элемент **View** поддерживает следующие методы.



| Метод                                              | Описание                                                |
|-----------------------------------------------------|------------------------------------------------------------|
| [close](view-close.md)                             | Закрывает **представление**.                                       |
| [ни](view-maximize.md)                       | Разворачивает **представление**.                                    |
| [свести к минимуму](view-minimize.md)                       | Свертывает **представление**.                                    |
| [restore](view-restore.md)                         | Восстанавливает **представление**.                                     |
| [ретурнтомедиацентер](view-returntomediacenter.md) | Возвращает пользователя в полноэкранный режим проигрывателя Windows Media. |
| [size](view-size.md)                               | Изменяет размер **представления** на указанном крае.                  |



 

Элемент **View** может реализовывать следующие обработчики событий.



| Обработчик событий               | Описание                                                                |
|-----------------------------|----------------------------------------------------------------------------|
| [OnClose](view-onclose.md) | Обрабатывает событие, возникающее, когда **представление** собирается закрыться.      |
| [OnError](view-onerror.md) | Обрабатывает событие ошибки, если **Settings. енаблиррордиалогс** имеет значение false. |
| [OnLoad](view-onload.md)   | Обрабатывает событие, возникающее при первом отображении **представления** .         |
| [время ожидания](view-ontimer.md) | Обрабатывает события таймера.                                                      |



 

Элемент **View** поддерживает атрибуты окружения и может реализовывать обработчики событий окружения, за исключением случаев, когда отмечено. Дополнительные сведения см. в разделе [атрибуты окружения](ambient-attributes.md) и [обработчики событий окружения](ambient-event-handlers.md).
