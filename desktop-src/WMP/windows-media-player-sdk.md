# <a name="windows-media-player-sdk"></a>Пакет SDK для проигрывателя Windows Media

В этой документации описывается пакет средств разработки программного обеспечения (SDK) проигрывателя Microsoft Windows Media. Пакет SDK проигрывателя Windows Media является одним из компонентов Microsoft Windows SDK. К другим компонентам мультимедиа относятся пакет SDK для Microsoft Media Foundation, пакет SDK для формата Microsoft Windows Media и пакет SDK служб Microsoft Windows Media.

Windows Media Player SDK документирует технологии программирования, которые можно использовать для расширения возможностей проигрывателя Windows Media и проигрывателя Windows Media Mobile. Эти технологии описаны в следующих разделах:

> [!Note]  
> При установке Windows SDK не устанавливается проигрыватель Windows Media или проигрыватель Windows Media Mobile. Для использования материалов в этом пакете SDK должен быть установлен проигрыватель Windows Media. Проигрыватель Windows Media Mobile является частью Windows Mobile, которая доступна только на Pocket PC или смартфоне, поставляемом производителем устройства или оператором мобильной связи.
