# <a name="miscellaneous"></a>Разное

В этом разделе описываются специализированные атрибуты и другие разделы, которые используются при создании обложек.

Рассматриваются следующие темы.



| Раздел                                            | Описание                                                                                                                        |
|--------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [Ссылка на цвет](color-reference.md)           | Диаграмма цветов, названий цветов и шестнадцатеричных значений для цветов, поддерживаемых обложками.                                              |
| [Глобальные атрибуты](global-attributes.md)       | Глобальные атрибуты — это атрибуты, обеспечивающие простой доступ к определенным элементам или объектам проигрывателя из любого места внутри обложки.       |
| [Атрибуты прослушивания](listening-attributes.md) | Атрибуты, используемые для соединения одного атрибута с другим, чтобы его значение изменялось при каждом изменении значения другого атрибута. |
