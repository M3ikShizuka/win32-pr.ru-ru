# <a name="settingsplaycount"></a>Settings. Плайкаунт

Свойство **плайкаунт** указывает или получает количество воспроизводимых элементов мультимедиа.

## <a name="syntax"></a>Синтаксис

Player. Settings. Плайкаунт

## <a name="possible-values"></a>Возможные значения

Это свойство имеет **номер** для чтения и записи (**Long**) с минимальным значением, равным единице, и значением по умолчанию.

**Проигрыватель Windows Media 10 Mobile**: это свойство не поддерживается.
