# <a name="windows-media-player-conversion-plug-ins"></a>Подключаемые модули преобразования проигрывателя Windows Media

Проигрыватель Microsoft Windows Media предоставляет архитектуру для подключаемых программ, которые преобразуют цифровые форматы файлов мультимедиа.

Документация по подключаемому модулю преобразования делится на следующие разделы.



| Section                                                                                    | Описание                                                                                                                                           |
|--------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| [О подключаемых модулях преобразования](about-conversion-plug-ins.md)                                 | Содержит общие сведения об архитектуре, используемой для подключаемых модулей преобразования. Ознакомьтесь с этим разделом, чтобы узнать об общих принципах, связанных с этой технологией. |
| [Справочник по программированию подключаемых модулей преобразования](conversion-plug-ins-programming-reference.md) | Содержит подробный справочник по COM-интерфейсу, поддерживаемому пакетом SDK проигрывателя Windows Media для подключаемых модулей преобразования.                                |
