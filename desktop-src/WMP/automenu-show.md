# <a name="automenushow"></a>АВТОМЕНЮ. показывать

Метод **Показать** отображает **панель быстрого доступа**.

``` syntax
        elementID.show("Play")
        
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="_Play_"></span><span id="_play_"></span><span id="_PLAY_"></span>*Воспроизводит*
</dt> <dd>

Всегда указывайте это значение.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Remarks

**Панель быстрого доступа** отображается в расположении, указанном **левым** и **верхним** атрибутами элемента **АВТОМЕНЮ** .

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|---------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media 9 Series или более поздней версии<br/> |
