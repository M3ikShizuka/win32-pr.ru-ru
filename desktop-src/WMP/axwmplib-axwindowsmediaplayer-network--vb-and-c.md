# <a name="axwindowsmediaplayernetwork-property"></a>Аксвиндовсмедиаплайер. Network, свойство

Свойство Network (сеть) получает интерфейс Ивмпнетворк, предоставляющий доступ к сведениям о сети и параметрам.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```CSharp
public IWMPNetwork network {get;}
```


```VB

Public ReadOnly Property network As IWMPNetwork
```





## <a name="property-value"></a>Значение свойства

Интерфейс Вмплиб. Ивмпнетворк.
