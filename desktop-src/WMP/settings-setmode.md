# <a name="settingssetmode-method"></a>Метод Settings. setMode

Метод **setMode** устанавливает для различных режимов режим "активный" или "Неактивный".

## <a name="syntax"></a>Синтаксис


```JScript
Settings.setMode(
  modeName,
  state
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*моденаме* \[ окне\]
</dt> <dd>

**Строка** , указывающая имя изменяемого режима, содержащее одно из следующих значений.



| Строка     | Описание                                                                                                                                                       |
|------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ауторевинд | Режим, указывающий, следует ли перематывает дорожки в начало после воспроизведения в конец. Состояние по умолчанию — true.                                                  |
| loop       | Режим, указывающий, повторяется ли последовательность дорожек. Состояние по умолчанию — false.                                                                            |
| шовфраме  | Режим, указывающий, отображается ли ближайший кадр видеоключа в текущей позиции, когда не воспроизводится. Состояние по умолчанию — false. Не влияет на звуковые дорожки. |
| shuffle    | Режим, указывающий, воспроизводятся ли дорожки в случайном порядке. Состояние по умолчанию — false.                                                                            |



 

</dd> <dt>

*состояние* \[ окне\]
</dt> <dd>

**Логическое значение** , указывающее, активен ли новый заданный режим.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Когда активен режим Шовфраме, проигрыватель должен получить доступ к содержимому дорожки, чтобы получить кадр видео. Из-за соображений пропускной способности этот режим следует использовать с осторожностью при воспроизведении нелокального содержимого.
