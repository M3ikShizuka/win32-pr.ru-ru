# <a name="mapping-images"></a>Сопоставление изображений

Изображения сопоставления используются для конкретных элементов управления, чтобы указать, какие регионы будут отвечать на нажатия клавиш мыши, и определить, какие элементы управления получают какие события. Изображение сопоставления должно иметь ту же высоту и ширину, что и первичный образ, и должно быть создано таким образом, чтобы изображения в обоих файлах выстроились точно. По этой причине рекомендуется использовать программу редактирования графики, поддерживающую слои, так как можно быть уверенным, что изображения будут выровнены по точности.

Для различных элементов управления требуются различные типы сопоставления. Элементу **буттонелемент** требуется таблица с разными цветами для каждой кнопки. Цветные области в файле сопоставления должны соответствовать областям кнопок, которые необходимо сопоставить.
