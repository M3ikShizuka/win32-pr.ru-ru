# <a name="buttongroupmappingimage"></a>BUTTONGROUP. Маппингимаже

Атрибут **маппингимаже** указывает или получает имя изображения, представляющего карту кнопок **BUTTONGROUP**.

``` syntax
        elementID.mappingImage
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи.

## <a name="remarks"></a>Комментарии

Это обязательный атрибут, который должен быть указан.

Изображение сопоставления должно совпадать с размерами основного **изображения**. Он представляет собой карту областей, которые щелкнули в основном изображении. Создайте карту, заполняя каждую область щелчка другим цветом.

При определении каждого **буттонелемент** назначьте его цвет из схемы с помощью атрибута **маппингколор** . Например, если в основном изображении есть кнопка "присутствовать в форме", можно создать карту с областью красного знака. После этого атрибут **буттонелемент** необходимо указать как красный, чтобы сделать изображение «конец-знак» недоступным для щелчка.

Поддерживаются форматы изображений BMP, JPG, PNG и GIF (не включая анимированные GIF). Так как Жпгс являются утерянными и поэтому подвержены непредвиденным изменениям цвета, их не рекомендуется использовать для сопоставления изображений.
