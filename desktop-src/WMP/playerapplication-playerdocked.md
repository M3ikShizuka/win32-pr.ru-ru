# <a name="playerapplicationplayerdocked"></a>Плайераппликатион. Плайердоккед

свойство **плайердоккед** извлекает значение, указывающее, находится ли проигрыватель Windows Media в закрепленном состоянии.

## <a name="syntax"></a>Синтаксис

*плайераппликатион*. **плайердоккед**

## <a name="possible-values"></a>Возможные значения

Это свойство является **логическим значением**, доступным только для чтения.

## <a name="remarks"></a>Remarks

это свойство используется только при удаленном взаимодействии с элементом управления проигрыватель Windows Media.

**проигрыватель Windows Media 10 Mobile:** Это свойство всегда возвращает **значение false**.
