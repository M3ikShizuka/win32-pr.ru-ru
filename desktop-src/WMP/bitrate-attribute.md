# <a name="bitrate-attribute"></a>Атрибут "скорость"

Атрибут « **скорость** » — это битовая скорость элемента в битах в секунду.

-   [Часто используемые файлы Windows Media](commonly-used-windows-media-file-attributes.md)
-   [Другие элементы](other-item-attributes.md)

## <a name="remarks"></a>Комментарии

Этот атрибут хранится как в библиотеке, так и в файле цифрового мультимедиа.

Константа Windows Media Format SDK для этого атрибута — g \_ всзвмбитрате.

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .
