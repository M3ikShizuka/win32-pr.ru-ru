# <a name="file-name-extensions"></a>Расширения файлов

существуют определенные рекомендации по использованию расширений имен файлов для Windows метафайлов мультимедиа. Windows расширения метафайлов мультимедиа используются для поиска различных типов Windowsных файлов мультимедиа. Расширение имени файла предоставляет независимому поставщику программного обеспечения (ISV) сведения о требованиях к отрисовке приложения, использующего определенное расширение, и позволяет авторам содержимого ориентироваться на общие типы проигрывателей мультимедиа.

Рекомендации по расширению имени файла перечислены в следующей таблице. Рекомендуется использовать тип MIME файла, расположенный в заголовке файла, для идентификации типа файлов.



| Расширение имени файла | тип MIME      | содержимое файла;                                                                                                                                                                            |
|---------------------|----------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| .wma                | аудио/x-MS-WMA | Windows Файл мультимедиа только с звуковым содержимым. Обычно используется для загрузки и воспроизведения файлов или потокового содержимого.                                                                             |
| .wmv                | видео/x — MS-WMV | Windows Файл мультимедиа с аудио-или видеосодержимым. Обычно используется для загрузки и воспроизведения файлов или потокового содержимого.                                                                     |
| .asf                | видео/x — MS-ASF | Устаревшее содержимое. Обычно используется для загрузки и воспроизведения файлов или потокового содержимого. Может содержать аудио-и видеоматериалы. Обычно используется для загрузки и воспроизведения файлов или потокового содержимого. |
| .wm                 | видео/x — MS-WM  | Зарезервировано                                                                                                                                                                                |
| . мастика                | Audio/x — MS-мастика | метафайлы, которые ссылаются на Windows файлы мультимедиа с расширением. asf,. wma или. мастика, имеют расширение имени файла.                                                                                             |
| . ввкс                | видео/x — MS-ввкс | метафайлы, которые ссылаются Windows файлы мультимедиа с расширениями wma, wmv, ввкс или мастика.                                                                                       |
| .asx                | видео/x — MS-ASF | метафайлы, которые ссылаются на Windows файлы мультимедиа с расширениями wma, мастика, wmv, ввкс, asf или asx.                                                                           |
| . ВМКС                | видео/x — MS-ввкс | Зарезервировано.                                                                                                                                                                               |



 

## <a name="remarks"></a>Remarks

поддержка сценариев и управления цифровыми правами (DRM) должна поддерживаться любым приложением, которое отображает Windows файлы мультимедиа.
