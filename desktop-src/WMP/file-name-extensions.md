# <a name="file-name-extensions"></a>Расширения файлов

Существуют определенные рекомендации по использованию расширений имен файлов для метафайлов Windows Media. Расширения Windows Media Metafile Names используются для поиска различных типов файлов Windows Media. Расширение имени файла предоставляет независимому поставщику программного обеспечения (ISV) сведения о требованиях к отрисовке приложения, использующего определенное расширение, и позволяет авторам содержимого ориентироваться на общие типы проигрывателей мультимедиа.

Рекомендации по расширению имени файла перечислены в следующей таблице. Рекомендуется использовать тип MIME файла, расположенный в заголовке файла, для идентификации типа файлов.



| Расширение имени файла | тип MIME      | содержимое файла;                                                                                                                                                                            |
|---------------------|----------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| .wma                | аудио/x-MS-WMA | Файл Windows Media с только звуковым содержимым. Обычно используется для загрузки и воспроизведения файлов или потокового содержимого.                                                                             |
| .wmv                | видео/x — MS-WMV | Файл Windows Media с аудио-или видеосодержимым. Обычно используется для загрузки и воспроизведения файлов или потокового содержимого.                                                                     |
| .asf                | видео/x — MS-ASF | Устаревшее содержимое. Обычно используется для загрузки и воспроизведения файлов или потокового содержимого. Может содержать аудио-и видеоматериалы. Обычно используется для загрузки и воспроизведения файлов или потокового содержимого. |
| .wm                 | видео/x — MS-WM  | Зарезервировано                                                                                                                                                                                |
| . мастика                | Audio/x — MS-мастика | Метафайлы, которые ссылаются на файлы Windows Media с расширением ASF, WMA или мастика.                                                                                             |
| . ввкс                | видео/x — MS-ввкс | Метафайлы, которые ссылаются на файлы Windows Media с расширением. WMA,. WMV,. ввкс или. мастика.                                                                                       |
| .asx                | видео/x — MS-ASF | Метафайлы, которые ссылаются на файлы Windows Media с расширением. WMA,. мастика,. WMV,. ввкс,. ASF или. ASX.                                                                           |
| . ВМКС                | видео/x — MS-ввкс | Зарезервировано.                                                                                                                                                                               |



 

## <a name="remarks"></a>Комментарии

Поддержка сценариев и управления цифровыми правами (DRM) должна поддерживаться любым приложением, которое отображает файлы Windows Media.
