# <a name="trackbars"></a>Линейки воспроизведения

Значение TrackBar отображает изображение, которое изменяется мелкими приращениями. Ползунки используются для элементов управления "Громкость" и "расположение воспроизведения". Ползунки можно использовать для вывода сведений о текущем элементе мультимедиа или для разрешения пользователю изменять параметры или позицию воспроизведения. Например, можно использовать параметр TrackBar, чтобы разрешить пользователю настраивать том, и другую линейку, которая может показывать пользователю текущую точку воспроизведения. Ползунки имеют изображение Thumb, определяющее текущее значение параметра TrackBar.

Раздел TrackBar в файле определения обложки должен начинаться с этой строки:


```C++
[ Trackbars ]

```



Затем необходимо добавить одну или несколько строк, содержащих сведения о каждой из значений TrackBar в обложке.


```C++
    Seek        5,25,226,17    Disabled @ 4,1      SeekThumb.bmp      18,17
    Volume      32,220,172,23  Disabled @ 32,27    VolumeThumb.bmp    23,22

```



Для раздела TrackBar файла определения обложки можно использовать следующий шаблон:


```C++
//  <Type> <Location>     <Dis Image Src> <Thumb Image Src> <Thumb Size>
//  ------ ----------     --------------- ----------------- ------------

```



Чтобы получить сведения о TrackBar для каждой строки в разделе TrackBar (требуется каждая часть строки), необходимо использовать следующий порядок:

1.  [Тип TrackBar](trackbar-type.md)
2.  [Расположение TrackBar](trackbar-location.md)
3.  [Источник изображения для отключенной TrackBar](image-source-for-disabled-trackbar.md)
4.  [Источник изображения Thumb](thumb-image-source.md)
5.  [Размер бегунка](thumb-size.md)

Пример кода для TrackBar см. в [разделе Sample TrackBar](sample-trackbars-section.md).
