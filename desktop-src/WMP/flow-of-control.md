# <a name="flow-of-control"></a>Поток управления

Звуковые данные поступают в проигрыватель Windows Media постоянно через файл или поток. Эти данные передаются в визуализацию. Вы рисуете на определенной поверхности и передаете ее обратно в проигрыватель Windows Media. Этот обмен выполняется несколько раз в секунду, и пользователь получает привлекательную анимацию, которая переносит время на музыку.

Ниже приведена конкретная последовательность последовательности программы визуализации:

1.  По истечении интервала проигрыватель Windows Media создает моментальный снимок воспроизводимого звука.
2.  Проигрыватель Windows Media предоставляет данные из этого снимка в визуализацию с помощью функции **Render** и функции **рендервиндовед** .
3.  Необходимо написать код, который будет выполняться при вызове **Render** и **рендервиндовед** . Код рисуется с помощью контекста устройства, определенного проигрывателем Windows Media при отрисовке без окна, или с помощью окна, создаваемого при отрисовке окна.
4.  В регионе, указанном в текущей обложке, проигрыватель Windows Media отображает, что нарисовано в коде.
5.  Этот процесс повторяется несколько раз в секунду, создавая графические анимации, которые отправляются по времени на музыку. После прекращения воспроизведения музыки визуализация останавливается.
