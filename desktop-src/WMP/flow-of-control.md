# <a name="flow-of-control"></a>Flow элемента управления

звуковые данные поступают в проигрыватель Windows Media непрерывно через файл или поток. Эти данные передаются в визуализацию. вы рисуйте на определенной поверхности и передайте эту поверхность в проигрыватель Windows Media. Этот обмен выполняется несколько раз в секунду, и пользователь получает привлекательную анимацию, которая переносит время на музыку.

Ниже приведена конкретная последовательность последовательности программы визуализации:

1.  по истечении интервала проигрыватель Windows Media создает моментальный снимок воспроизводимого звука.
2.  проигрыватель Windows Media передает данные из этого снимка в визуализацию с помощью функции **Render** и функции **рендервиндовед** .
3.  Необходимо написать код, который будет выполняться при вызове **Render** и **рендервиндовед** . код рисуется с помощью контекста устройства, определенного проигрыватель Windows Media при отрисовке безоконного окна, или с помощью окна, создаваемого при отрисовке окна.
4.  в регионе, указанном в текущей обложке, проигрыватель Windows Media отображает, что нарисовано в коде.
5.  Этот процесс повторяется несколько раз в секунду, создавая графические анимации, которые отправляются по времени на музыку. После прекращения воспроизведения музыки визуализация останавливается.
