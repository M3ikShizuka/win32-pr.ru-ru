# <a name="creating-the-volumethumb-file"></a>Создание файла Волумесумб

Файл Волумесумб содержит изображение Thumb, которое будет использоваться с параметром TrackBar тома.

На следующем рисунке приведен файл Волумесумб.

![файл волумесумб](images/ceswmvol.png)

Он содержит два изображения Thumb. Слева находится нормальное состояние, а справа — отправленное состояние. Отключенное состояние отсутствует. Если значение TrackBar отключено, то отключенное фоновое изображение TrackBar отображается из файла Super.
