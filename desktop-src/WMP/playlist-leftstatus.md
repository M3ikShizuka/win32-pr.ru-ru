# <a name="playlistleftstatus"></a>Список воспроизведения. Лефтстатус

Атрибут **лефтстатус** указывает или получает текст состояния, отображаемый в левой части и внизу элемента **списка воспроизведения** .

``` syntax
        elementID.leftStatus
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи.

## <a name="remarks"></a>Комментарии

Этот атрибут может сочетать любой текст с конкретными ключевыми словами, которые будут отображать необходимую информацию, например общую длительность списка воспроизведения. Ключевые слова заключены в процентные символы (%) чтобы они не изменялись, используйте обычный текст.

Можно использовать следующие ключевые слова.



| Ключевое слово               | Описание                                                                                                                                                                                                  |
|-----------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| count                 | Число элементов в списке воспроизведения.                                                                                                                                                                             |
| size                  | Общий размер списка воспроизведения.                                                                                                                                                                                  |
| длительность              | Общая длительность списка воспроизведения.                                                                                                                                                                              |
| *XXX*                 | Выполняет **getItemInfo** для списка воспроизведения с *xxx* , который получает элемент.                                                                                                                                 |
| SelectedSize          | Общий размер выбранных записей в списке воспроизведения.                                                                                                                                                          |
| селектедкаунт         | Общее число выбранных записей в списке воспроизведения.                                                                                                                                                            |
| селектеддуратион      | Общая длительность выбранных записей в списке воспроизведения.                                                                                                                                                      |
| чеккедкаунт          | Общее число проверенных дорожек в списке воспроизведения.                                                                                                                                                              |
| чеккеддуратион       | Общая продолжительность проверенных дорожек в списке воспроизведения.                                                                                                                                                        |
| чеккедсизе           | Общий размер проверенных дорожек в списке воспроизведения.                                                                                                                                                            |
| дуратионстринг        | Отображает текст с описанием длительности "Общее время" или "предполагаемое время" в зависимости от того, известны ли итоговые значения. За этим текстом следует "% Duration%".                                       |
| чеккеддуратионстринг | Отображает текст, описывающий продолжительность всех отмеченных элементов в списке воспроизведения как "Общее время" или "предполагаемое время" в зависимости от того, известны ли итоговые значения. За этим текстом следует "% Duration%". |



 

Значение "Общее время:% Duration%" для списка воспроизведения, содержащего общую длительность семь минут, будет содержать "Общее время: 07:00".
