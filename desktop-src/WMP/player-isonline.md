# <a name="playerisonline"></a>Player. в сети

Свойство **ononline** извлекает значение, указывающее, подключен ли пользователь к сети.

## <a name="syntax"></a>Синтаксис

*проигрыватель* . **Подключение к Интернету**

## <a name="possible-values"></a>Возможные значения

Это свойство является **логическим значением**, доступным только для чтения.



| Значение | Описание                             |
|-------|-----------------------------------------|
| Да  | Пользователь подключен к сети.     |
| false | Пользователь не подключен к сети. |



 

## <a name="remarks"></a>Remarks

**проигрыватель Windows Media 10 Mobile:** Это свойство всегда возвращает значение true.
