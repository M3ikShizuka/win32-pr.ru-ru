# <a name="playlistitemmedia"></a>Список воспроизведения. Итеммедиа

Атрибут **итеммедиа** извлекает объект **мультимедиа** , соответствующий заданному индексу в элементе **списка воспроизведения** .

``` syntax
        elementID.itemMedia(index)
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является объектом **мультимедиа** , предназначенным только для чтения.

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*номер*
</dt> <dd>

**Число**(**длинное**), содержащее индекс элемента списка воспроизведения.

</dd> </dl>

## <a name="remarks"></a>Remarks

Свойство **итеммедиа** будет возвращать объекты мультимедиа, развернутые в элементе **списка воспроизведения** . Например, если имеется список воспроизведения, содержащий три клипа мультимедиа, которые не разворачиваются в элементе **списка воспроизведения** , **итеммедиа**(0) вернет список воспроизведения в качестве объекта мультимедиа. Если список воспроизведения расширен, **итеммедиа**(0) вернет первый клип из списка воспроизведения.
