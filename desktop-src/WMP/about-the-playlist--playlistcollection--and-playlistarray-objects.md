# <a name="about-the-playlist-playlistcollection-and-playlistarray-objects"></a>Сведения об объектах списков воспроизведения, Плайлистколлектион и Плайлистаррай

объекты **списков воспроизведения**, **плайлистколлектион** и **плайлистаррай** управляют списками воспроизведения, которые проигрыватель Windows Media могут использовать для указания порядка воспроизведения содержимого. Вы получаете объект **плайлистколлектион** из свойства **плайлистколлектион** объекта **Player** . Свойство **плайлистколлектион** возвращает объект **плайлистколлектион** . Доступ к свойствам объекта **плайлистколлектион** можно получить только после его создания. Например, чтобы создать новый список воспроизведения, сначала необходимо получить объект **плайлистколлектион** , а затем использовать метод для этого объекта.


```C++
player.playlistcollection.newplaylist('myplaylist');
```



Текущий список воспроизведения можно получить с помощью свойства **куррентплайлист** . Например, чтобы получить имя текущего списка воспроизведения, используйте следующий код:


```C++
myname = player.currentplaylist.name;
```



Объект **плайлистаррай** возвращается методами **жеталл** и **жетбинаме** объекта **плайлистколлектион** . Например, чтобы получить количество списков воспроизведения, используйте следующий код:


```C++
howmany = player.playlistcollection.getAll().count;
```



Чтобы получить известный список воспроизведения по имени, используйте следующий код:


```C++
if (player.playlistcollection.getbyname('myplaylist').count == 1) {
    pl = player.playlistcollection.getbyname('myplaylist').item(0);
}
```
