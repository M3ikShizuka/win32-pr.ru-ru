# <a name="mediacontenttypes-attribute"></a>Атрибут Медиаконтенттипес

Атрибут **медиаконтенттипес** задает тип содержимого элемента.

## <a name="applies-to"></a>Применение

-   [Списки воспроизведения](playlist-attributes-ref.md)

## <a name="remarks"></a>Комментарии

Этот атрибут может принимать одно из следующих значений:



| Значение | Значение                                |
|-------|----------------------------------------|
| 1     | Список воспроизведения содержит звуковое содержимое.   |
| 2     | Будет предоставляться.                        |
| 3     | Список воспроизведения содержит звуковое содержимое.   |
| 4     | Список воспроизведения содержит видеоматериалы.   |
| 5     | Список воспроизведения содержит содержимое изображения. |
| 6     | Список воспроизведения содержит ТВ-содержимое.      |



 

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .

|--------------------|---------------------------------------------------|
