# <a name="video-location"></a>Расположение видео

Необходимо определить расположение отображаемой области видео в обложке. Необходимо ввести четыре положительных целых числа, разделенные запятыми, которые определяют левую, верхнюю, ширину и высоту видеоизображения (в пикселях) относительно фонового изображения.


```C++
0,38,240,172 

```
