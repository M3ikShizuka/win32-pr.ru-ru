# <a name="building-the-plug-in-for-a-type-2-online-store"></a>Создание подключаемого модуля для Интернет-магазина типа 2

Подключаемый модуль интернет-магазина типа 2 можно создать с помощью мастера подключаемых модулей интернет-магазина. Мастер проектов входит в состав пакета SDK для проигрывателя Windows Media, который является частью Windows SDK. Мастер — это надстройка для Microsoft Visual Studio, которая создает проект C++ и создает пример кода, который можно использовать в качестве отправной точки для разработки собственного Интернет-магазина.

В следующих разделах объясняется, как использовать мастер подключаемых модулей интернет-магазина.

-   [Установка мастера проектов](installing-the-project-wizard.md)
-   [Создание примера проекта](creating-a-sample-project.md)
-   [Компиляция примера проекта](compiling-the-sample-project.md)
-   [Запуск примера проекта](running-the-sample-project.md)
-   [Настройка примера проекта](customizing-the-sample-project.md)
