# <a name="building-the-plug-in-for-a-type-2-online-store"></a>Создание подключаемого модуля для Интернет-магазина типа 2

Подключаемый модуль интернет-магазина типа 2 можно создать с помощью мастера подключаемых модулей интернет-магазина. мастер проектов входит в состав проигрыватель Windows Media пакета SDK, который является частью Windows SDK. мастер — это надстройка для Microsoft Visual Studio, которая создает проект C++ и создает пример кода, который можно использовать в качестве отправной точки для разработки собственного интернет-магазина.

В следующих разделах объясняется, как использовать мастер подключаемых модулей интернет-магазина.

-   [установка мастера Project](installing-the-project-wizard.md)
-   [Создание примера Project](creating-a-sample-project.md)
-   [Компиляция примера Project](compiling-the-sample-project.md)
-   [Запуск примера Project](running-the-sample-project.md)
-   [Настройка образца Project](customizing-the-sample-project.md)
