# <a name="equalizersettingsenhancedaudio"></a>ЕКУАЛИЗЕРСЕТТИНГС. Енханцедаудио

Атрибут **енханцедаудио** указывает или получает значение, указывающее, включен ли расширенный звук.

``` syntax
        elementID.enhancedAudio
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Описание | Значение                                  |
|-------------|----------------------------------------|
| true        | Включен расширенный звук.           |
| false       | По умолчанию. Расширенный звук отключен. |



 

## <a name="requirements"></a>Requirements (Требования)



| Требование | Значение |
|--------------------|---------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media 9 Series или более поздней версии<br/> |
