# <a name="slidermin"></a>SLIDER. min

Атрибут **min** указывает или получает минимальное значение диапазона, определенного элементом управления "ползунок".

``` syntax
        elementID.min
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**float**) и значением по умолчанию 0.

## <a name="remarks"></a>Remarks

Значение, указанное для параметра **min** , должно быть меньше, чем значение параметра **Max**.

См. **кустомслидер**. атрибут [поситионимаже](customslider-positionimage.md) для примера, иллюстрирующий использование атрибутов элемента **Slider** .
