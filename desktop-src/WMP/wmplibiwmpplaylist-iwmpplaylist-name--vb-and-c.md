# <a name="iwmpplaylistname-property"></a>Свойство Ивмпплайлист:: Name

Свойство **Name** Возвращает или задает имя списка воспроизведения.

## <a name="syntax"></a>Синтаксис


```CSharp
public System.String name {get; set;}
```


```VB

Public Property name As System.String
```





## <a name="property-value"></a>Значение свойства

**Строка System. String** , которая является именем списка воспроизведения.

## <a name="remarks"></a>Комментарии

Перед использованием этого свойства необходимо иметь доступ на чтение к библиотеке. Дополнительные сведения см. в разделе [доступ к библиотеке](library-access.md).
