# <a name="viewtitlebar"></a>Просмотреть. заголовок

Атрибут **заголовка** строки получает значение, указывающее, отображается ли заголовок окна.

``` syntax
        elementID.titleBar
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением**, доступным только для чтения.



| Значение | Описание                             |
|-------|-----------------------------------------|
| Да  | По умолчанию. Отобразится заголовок окна. |
| false | Заголовок окна не отображается.      |



 

## <a name="remarks"></a>Remarks

Если строка заголовка отображается, отобразятся поля элемента управления, сворачивания и закрытия. Заголовок окна будет заголовком элемента **View** .

Если параметру « **заголовок** окна» присвоено значение true и пользователь пытается изменить значение **видео. Zoom**, изменение не будет выполнено, если только Обложка не наблюдает за **масштабом** и не примет необходимые меры для изменения размера.
