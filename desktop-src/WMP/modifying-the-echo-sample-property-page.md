# <a name="modifying-the-echo-sample-property-page"></a>Изменение страницы свойств «пример эха»

реализация страницы свойств по умолчанию, предоставляемая в примере подключаемого модуля DSP для мастера подключаемых модулей проигрыватель Windows Media, содержит один элемент управления "поле ввода", который получает коэффициент масштабирования от пользователя. Необходимо изменить страницу свойств для получения двух значений свойств, используемых образцом Echo.
