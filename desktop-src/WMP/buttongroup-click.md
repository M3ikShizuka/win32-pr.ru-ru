# <a name="buttongroupclick"></a>BUTTONGROUP. Щелкните

Метод **Click** вызывает обработчик событий **OnClick** , определенный для **буттонелемент** с указанным индексом.

``` syntax
        elementID.click(button)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="button"></span><span id="BUTTON"></span>*переключатель*
</dt> <dd>

**Число** (**Long**), содержащее индекс **буттонелемент** с вызываемым обработчиком событий **OnClick** .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Remarks

Этот метод используется для предоставления альтернативного способа выполнения кода, связанного с кнопкой, содержащейся в **BUTTONGROUP**.

|--------------------|---------------------------------------------------|
