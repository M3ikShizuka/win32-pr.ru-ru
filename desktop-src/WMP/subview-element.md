# <a name="subview-element"></a>Элемент вложенного представления

Элемент вложенного **представления** предоставляет способ управления частью обложки, например, для предоставления панели управления, которая может быть скрыта, когда она не используется. Элементы вложенного **представления** всегда являются дочерними элементами родительского **представления** и могут содержать другой элемент обложки, кроме **представлений**, **тем** и **других элементов вложенного представления.**

Элемент вложенного **представления** поддерживает следующие атрибуты, которые определены в элементе **View** .



| attribute                                                       | Описание                                                                                                 |
|-----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [backgroundColor](view-backgroundcolor.md)                     | Указывает или получает цвет фона для элемента управления " **Подпредставление** ". Значение по умолчанию — None.        |
| [backgroundImage](view-backgroundimage.md)                     | Указывает или получает фоновое изображение для элемента управления " **Подпредставление** ".                                     |
| [баккграундимажехуешифт](view-backgroundimagehueshift.md)     | Указывает или получает величину, на которую смещается цветовой тон фонового изображения.                      |
| [баккграундимажесатуратион](view-backgroundimagesaturation.md) | Указывает или получает значение насыщенности фонового изображения.                                        |
| [баккграундтилед](view-backgroundtiled.md)                     | Указывает или получает значение, указывающее, находится ли мозаичное изображение элемента управления " **ПОДпредставление** ". |
| [ресизебаккграундимаже](view-resizebackgroundimage.md)         | Указывает или получает значение, указывающее, можно ли изменить размер фонового изображения.                      |
| [транспаренциколор](view-transparencycolor.md)                 | Указывает или получает цвет прозрачности фонового изображения.                                      |



 

Элемент вложенного **представления** поддерживает атрибуты окружения, за исключением случаев, где указано. Дополнительные сведения см. в разделе [атрибуты окружения](ambient-attributes.md).

Элемент вложенного **представления** может реализовывать следующие внешние обработчики событий: [онендмове](onendmove.md) и [онресизе](onresize.md).
