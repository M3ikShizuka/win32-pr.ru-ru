# <a name="nesting-metafiles"></a>Вложенные метафайлы

Метафайл может ссылаться на другой метафайл. Чтобы сослаться на другой метафайл, используйте элемент **ентриреф** . Элемент **ентриреф** в основном метафайле указывает на внешний метафайл.

элемент управления проигрыватель Windows Media обрабатывает элементы **записи** упоминаемого метафайла, как если бы они были добавлены в основной метафайл в позиции элемента **ентриреф** . Однако он пропускает все элементы **записи** в метафайле, на который указывает ссылка, для которого атрибут **СКИПИФРЕФ** имеет значение Да.

проигрыватель Windows Media 7,0, 7,1 и проигрыватель Windows Media для элементов управления Windows XP игнорируют все элементы **ентриреф** в метафайле, на который указывает ссылка. Таким словами, метафайлы могут быть вложенными только на одном уровне с использованием этих версий. Эти версии также игнорируют элемент **ASX** и его атрибуты в файле, на который указывает ссылка. проигрыватель Windows Media 9 Series или более поздней версии поддерживает вложение метафайлов размером до 5 уровней.
