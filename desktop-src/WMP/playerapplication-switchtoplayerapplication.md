# <a name="playerapplicationswitchtoplayerapplication-method"></a>Плайераппликатион. Свитчтоплайераппликатион, метод

Метод **свитчтоплайераппликатион** переключает удаленный элемент управления проигрывателя Windows Media на полный режим проигрывателя.

## <a name="syntax"></a>Синтаксис


```JScript
PlayerApplication.switchToPlayerApplication()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Этот метод используется только при удаленном взаимодействии с элементом управления проигрывателя Windows Media.

**Проигрыватель Windows Media 10 Mobile:** Этот метод не поддерживается.
