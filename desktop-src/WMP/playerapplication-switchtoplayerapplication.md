# <a name="playerapplicationswitchtoplayerapplication-method"></a>Плайераппликатион. Свитчтоплайераппликатион, метод

метод **свитчтоплайераппликатион** переключает удаленный элемент управления проигрыватель Windows Media на полный режим проигрывателя.

## <a name="syntax"></a>Синтаксис


```JScript
PlayerApplication.switchToPlayerApplication()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Remarks

этот метод используется только при удаленном взаимодействии с элементом управления проигрыватель Windows Media.

**проигрыватель Windows Media 10 Mobile:** Этот метод не поддерживается.
