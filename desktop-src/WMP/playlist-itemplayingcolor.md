# <a name="playlistitemplayingcolor"></a>Список воспроизведения. Итемплайингколор

Атрибут **итемплайингколор** указывает или получает цвет выделения, который указывает на текущий элемент списка воспроизведения.

``` syntax
        elementID.itemPlayingColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей любое значение цвета Microsoft Internet Explorer. Он имеет значение по умолчанию « \# 00FF00».

|--------------------|------------------------------------------------------|
