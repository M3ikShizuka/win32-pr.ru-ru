# <a name="axwindowsmediaplayerctlcontrols-property"></a>Аксвиндовсмедиаплайер. Ктлконтролс, свойство

Свойство Ктлконтролс получает интерфейс Ивмпконтролс, который предоставляет способ управления воспроизведением элемента мультимедиа.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```CSharp
public IWMPControls Ctlcontrols {get;}
```


```VB

Public ReadOnly Property Ctlcontrols As IWMPControls
```





## <a name="property-value"></a>Значение свойства

ВМПЛИБ. Интерфейс Ивмпконтролс.
