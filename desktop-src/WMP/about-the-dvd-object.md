# <a name="about-the-dvd-object"></a>Об объекте DVD

Объект **DVD** добавляет функции, относящиеся к DVD-носителям. В общем смысле DVD-диски обрабатываются в проигрывателе Windows Media так же, как и другие цифровые носители. Например, DVD-дисководы перечисляются с помощью объектов и наименований DVD, **а также для** работы с ними используются объекты **списков воспроизведения** и объекты **мультимедиа** . Однако некоторые функции относятся к DVD-диску, а также к объекту **DVD** . Например, DVD имеет концепцию с именем domain. Чтобы получить текущий домен для носителя DVD, введите следующее:


```C++
mydomain = player.dvd.domain;

```
