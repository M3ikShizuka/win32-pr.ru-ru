# <a name="adding-closed-captions-to-digital-media"></a>Добавление субтитров к цифровым носителям

Синхронизированный доступный медиа-обмен (SAMI) — это формат файла, предназначенный для предоставления синхронизированного текста, такого как субтитры, субтитры или аудио описания, с цифровым содержимым мультимедиа. В веб-браузере, использующем объектную модель проигрывателя Windows Media, SAMId способствует специальным возможностям. С помощью SAMI можно создавать мультимедийное содержимое, которое достигает более широкой аудитории.

В дополнение к стандартным шрифтам саамский SAMId может поддерживать другие стили текста, например различные цвета, размеры или языки, чтобы помочь различным пользователям. Саамский может быть особенно полезным для тех, кто имеет низкую концепцию или слуха. Формат SAMI также может помочь в учебных целях, таких как обучение учащихся на втором языке.

Эта статья посвящена использованию SAMId с объектной моделью элемента управления ActiveX проигрывателя Windows Media. Файлы SAMIs существуют независимо от цифровых файлов мультимедиа и не зависят от конкретного видео или звукового формата для работы. Поскольку файлы являются отдельными, элемент управления проигрывателя Windows Media обнаружит, проанализирует, синхронизирует и готовит каждый файл на клиентском компьютере. Это обеспечивает дополнительную гибкость и функциональность, поскольку позволяет редактировать отдельные файлы SAMI, выдает файл SAMI с различными форматами цифрового мультимедиа и хранение файлов SAMI в разных расположениях серверов.
