# <a name="jscript-files"></a>JScript Файла

JScript файлы загружаются с атрибутом **scriptFile** элемента **VIEW** . Они должны быть текстовыми файлами и должны использовать расширение имени файла .js. если имеется файл JScript с тем же именем, что и у файла определения обложки, JScript файл будет загружен одновременно с файлом определения обложки. например, если у вас есть файл определения обложки с именем лауре. wms и имеется файл JScript с именем laure.js, laure.js файл будет загружен автоматически.

JScript можно использовать для создания сложных функциональных возможностей в фоновом режиме. создавая функции в JScript, вы можете сделать то же самое, что и обложки. Например, можно использовать другой список воспроизведения для каждого дня недели, но в пятницу он всегда имеет один и тот же.

дополнительные сведения об использовании JScript с обложками см. [в разделе использование JScript](using-jscript.md) . обратите внимание, что Visual Basic scripting Edition (VBScript), который можно использовать при внедрении элемента управления проигрыватель Windows Media на веб-странице, не поддерживается для использования с обложками.
