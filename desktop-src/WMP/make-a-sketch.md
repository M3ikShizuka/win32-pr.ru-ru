# <a name="make-a-sketch"></a>Создать эскиз

Это не нужно делать, но может быть очень полезно в том, чтобы сделать проект необходимым для просмотра. Заранее планируя, вы можете сохранить изображение нескольких растровых файлов.

Запустите Photoshop и создайте новый образ требуемого размера. Создайте новый слой для размещения грубого эскиза. Если у вас есть эскиз и сканер, вам может потребоваться сканировать эскиз в Photoshop. На следующем рисунке показан пример грубого эскиза.

![грубый эскиз](images/ceswmskt.png)

Кнопка Плайпаусе находится в левом верхнем углу, кнопки «останавливаться», «далее» и «назад» слева, регулятор громкости находится внизу, а между кнопкой «Плайпаусе» и линейкой регулятора громкости находится бегущая строка.

при создании обложки для проигрыватель Windows Media 10 Mobile или более поздней версии вместо кнопки плайпаусе следует использовать кнопку плайпаусестоп. Дополнительная кнопка «запретить» не требуется.
