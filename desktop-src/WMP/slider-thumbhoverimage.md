# <a name="sliderthumbhoverimage"></a>ПОЛЗУНок. Сумбховеримаже

Атрибут **сумбховеримаже** указывает или получает изображение бегунка, которое появляется при наведении на него указателя мыши.

``` syntax
        elementID.thumbHoverImage
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей имя файла изображения.

## <a name="remarks"></a>Remarks

**Сумбховеримаже** является необязательным. Если он не указан, вид элемента управления "ползунок" не изменится при наведении на него указателя мыши.

Поддерживаются форматы BMP, JPG, PNG и GIF (не включая анимированные GIF).
