# <a name="ambientattributesheight"></a>Амбиентаттрибутес. Height

Атрибут **Height** указывает или получает высоту элемента управления.

``` syntax
        elementID.height
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**длинное**), представляющим высоту элемента управления в пикселях. Значение по умолчанию равно нулю или высоте изображения, указанного в атрибуте **Image** элемента управления.

## <a name="remarks"></a>Remarks

Если указанная высота меньше высоты предоставленного изображения, то изображение будет обрезано. Если высота превышает высоту изображения, то область щелчка выйдет за пределы границы изображения. Независимо от того, какое значение присвоено этому атрибуту, изображение не может увеличиваться за пределами родительского **представления** или вложенного **представления**.
