# <a name="dsp-plug-in-enumeration-types"></a>Типы перечислений подключаемых модулей DSP

Пакет SDK проигрывателя Windows Media реализует следующий тип перечисления для создания подключаемых модулей DSP.



| Тип перечисления                                        | Описание                                                                                    |
|---------------------------------------------------------|------------------------------------------------------------------------------------------------|
| [Вмпплугин \_ Cap](/previous-versions/windows/desktop/api/wmpservices/ne-wmpservices-wmpplugin_caps)                   | Используется с **ивмпплугин:: Caps** , чтобы указать, может ли подключаемый модуль выполнять преобразование между форматами. |
| [Вмпсервицес \_ стреамстате](/previous-versions/windows/desktop/api/wmpservices/ne-wmpservices-wmpservices_streamstate) | Указывает, остановлен ли поток, приостановлен или запущен.                     |
