# <a name="effectspreviouseffect"></a>EFFECTs. Превиаусеффект

Метод **превиаусеффект** Отображает предыдущую визуализацию, пропуская предустановки.

``` syntax
        elementID.previousEffect()
```

## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Этот метод отображает предыдущую визуализацию в порядке создания. Если текущая визуализация является первой в порядке создания и если **алловалл** имеет значение false, последняя визуализация становится текущей.

| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
