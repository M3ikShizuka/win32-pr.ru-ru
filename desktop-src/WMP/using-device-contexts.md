# <a name="using-device-contexts"></a>Использование контекстов устройств

Контекст устройства является стандартным обработчиком для контекста устройства. это необходимо для многих функций рисования, чтобы Microsoft Windows знал, в каком окне нужно рисовать. Например, чтобы нарисовать прямоугольник, необходимо указать контекст устройства.


```C++
HDC hdc;
::Rectangle( hdc, 1, 1, 100, 100 );

```



контекст устройства задается проигрыватель Windows Media с помощью функции **Render** . Если подключаемый модуль визуализируется с помощью окна, необходимо использовать контекст устройства этого окна. Используйте этот контекст устройства для любого инструмента рисования, для которого требуется контекст устройства.
