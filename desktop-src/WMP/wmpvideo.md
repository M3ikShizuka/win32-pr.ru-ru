# <a name="wmpvideo"></a>вмпвидео

Это предварительно определенный элемент **Video** со следующими значениями по умолчанию.

``` syntax
backgroundColor="black"
horizontalAlignment="stretch"
verticalAlignment="stretch"
```

## <a name="remarks"></a>Комментарии

При этом будет создан элемент **Video** , который будет растягивать окно видео при изменении размера обложки. Цифровое видео, отображаемое в этом окне, растягивается в соответствии с объемом доступного пространства или будет масштабироваться в соответствии с параметрами в меню **представления** проигрывателя в разделе **Размер видео**. Меню « **вид** » отображается в полноэкранном режиме проигрывателя или если в обложке для атрибута **View. заголовок** задано значение true.

Все свойства этого элемента **Video** можно переопределить, явно указав их.
