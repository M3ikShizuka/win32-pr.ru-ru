# <a name="columncolumnresizemode"></a>COLUMN. Колумнресиземоде

Атрибут **колумнресиземоде** указывает или получает режим изменения размера для этого столбца.

``` syntax
        elementID.columnResizeMode
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей одно из следующих значений.



| Значение          | Описание                                                                                                    |
|----------------|----------------------------------------------------------------------------------------------------------------|
| аутосизехеадер | По умолчанию. Столбец изменяется в соответствии со всеми данными в столбце и в заголовке.                         |
| аутосизедата   | Столбец изменяется в соответствии со всеми данными в столбце.                                                 |
| Фиксированный          | Столбец имеет фиксированный размер.                                                                                    |
| Растягивается      | Размер столбца изменяется для использования оставшегося пространства в элементе управления **списка воспроизведения** после изменения размера всех остальных столбцов. |
