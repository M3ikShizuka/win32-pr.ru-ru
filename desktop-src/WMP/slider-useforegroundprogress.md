# <a name="slideruseforegroundprogress"></a>ПОЛЗУНок. Усефореграундпрогресс

Атрибут **усефореграундпрогресс** указывает или получает значение, указывающее, будет ли использоваться основной индикатор выполнения.

``` syntax
        elementID.useForegroundProgress
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                              |
|-------|------------------------------------------|
| true  | Использовать основной ход выполнения.                 |
| false | По умолчанию. Не используйте основной ход выполнения. |



 

## <a name="remarks"></a>Комментарии

Этот атрибут позволяет использовать атрибут **фореграундпрогресс** , который в основном используется для отслеживания хода загрузки файла мультимедиа при одновременной отслеживании текущей позицией воспроизведения файла с помощью атрибута **value** .
