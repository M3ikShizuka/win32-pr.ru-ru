# <a name="iwmpmedia-vb-and-c-interface"></a>Интерфейс Ивмпмедиа (VB и C#)

Предоставляет способ задания и получения свойств элемента мультимедиа.

Интерфейс **ивмпмедиа** предоставляет следующие свойства.

## <a name="members"></a>Элементы

Интерфейс **ивмпмедиа (VB и C#)** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ивмпмедиа (VB и C#)** содержит следующие методы.



| Метод                                                                             | Описание                                                                                                   |
|:-----------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------|
| [**жетаттрибутенаме**](wmplibiwmpmedia-iwmpmedia-getattributename--vb-and-c.md)   | Возвращает имя атрибута, соответствующего указанному индексу.<br/>                            |
| [**getItemInfo**](wmplibiwmpmedia-iwmpmedia-getiteminfo--vb-and-c.md)             | Возвращает значение указанного атрибута для элемента мультимедиа.<br/>                                   |
| [**жетитеминфобятом**](wmplibiwmpmedia-iwmpmedia-getiteminfobyatom--vb-and-c.md) | Возвращает значение атрибута с указанным номером индекса.<br/>                                |
| [**жетмаркернаме**](wmplibiwmpmedia-iwmpmedia-getmarkername--vb-and-c.md)         | Возвращает имя маркера по указанному индексу.<br/>                                             |
| [**жетмаркертиме**](wmplibiwmpmedia-iwmpmedia-getmarkertime--vb-and-c.md)         | Возвращает время маркера по указанному индексу.<br/>                                             |
| [**isMemberOf**](wmplibiwmpmedia-iwmpmedia-ismemberof--vb-and-c.md)               | Возвращает значение, указывающее, является ли указанный элемент мультимедиа членом указанного списка воспроизведения.<br/> |
| [**исреадонлитем**](wmplibiwmpmedia-iwmpmedia-isreadonlyitem--vb-and-c.md)       | Возвращает значение, указывающее, можно ли изменять атрибуты указанного элемента мультимедиа.<br/>       |
| [**сетитеминфо**](wmplibiwmpmedia-iwmpmedia-setiteminfo--vb-and-c.md)             | Задает значение указанного атрибута для элемента мультимедиа.<br/>                                      |

| Свойство                                                                                      | Тип доступа          | Описание                                                                                                                                          |

| [**дуратионстринг**](wmplibiwmpmedia-iwmpmedia-durationstring--vb-and-c.md)<br/>       | Только для чтения<br/> | Возвращает значение, указывающее длительность текущего элемента мультимедиа в формате чч: мм: СС.<br/>                                                        |
| [**имажесаурцехеигхт**](wmplibiwmpmedia-iwmpmedia-imagesourceheight--vb-and-c.md)<br/> | Только для чтения<br/> | Возвращает высоту текущего элемента мультимедиа в пикселях.<br/>                                                                                      |
| [**имажесаурцевидс**](wmplibiwmpmedia-iwmpmedia-imagesourcewidth--vb-and-c.md)<br/>   | Только для чтения<br/> | Возвращает ширину текущего элемента мультимедиа в пикселях.<br/>                                                                                       |
| [**Идентично**](iwmpmedia-isidentical--vb-and-c.md)<br/>                             | Только для чтения<br/> | Возвращает значение, указывающее, совпадает ли указанный элемент мультимедиа с текущим элементом. В C# это метод **Get \_ одинаковы** .<br/> |
| [**маркеркаунт**](wmplibiwmpmedia-iwmpmedia-markercount--vb-and-c.md)<br/>             | Только для чтения<br/> | Возвращает число маркеров в элементе мультимедиа.<br/>                                                                                             |
| [**безымян**](wmplibiwmpmedia-iwmpmedia-name--vb-and-c.md)<br/>                           | Только для чтения<br/> | Возвращает или задает имя элемента мультимедиа.<br/>                                                                                                  |
| [**саурцеурл**](wmplibiwmpmedia-iwmpmedia-sourceurl--vb-and-c.md)<br/>                 | Только для чтения<br/> | Возвращает URL-адрес элемента мультимедиа.<br/>                                                                                                           |
