# <a name="editboxlinecount"></a>EDITBOX. Линекаунт

Атрибут **линекаунт** извлекает количество строк в элементе управления "поле ввода".

``` syntax
        elementID.lineCount
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** только для чтения (**длинное целое**).

## <a name="remarks"></a>Комментарии

Этот атрибут полезен только в том случае, если для **едитстиле** задано значение Multiline.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|---------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media для Windows XP или более поздней версии<br/> |
