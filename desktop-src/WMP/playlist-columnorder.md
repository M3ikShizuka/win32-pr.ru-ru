# <a name="playlistcolumnorder"></a>Список воспроизведения. Колумнордер

Атрибут **колумнордер** указывает или получает порядок столбцов списка воспроизведения.

``` syntax
        elementID.columnOrder
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, указывающей список индексов столбцов списка воспроизведения, разделенных точкой с запятой. Значение по умолчанию — 0; 1; 2; 3. Не должны присутствовать начальные и конечные точки с запятой и пробелы.
