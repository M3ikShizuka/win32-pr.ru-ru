# <a name="effectscurrenteffectpresetcount"></a>EFFECTs. Куррентеффектпресеткаунт

Атрибут **куррентеффектпресеткаунт** извлекает количество доступных предустановок для текущей визуализации.

``` syntax
        elementID.currentEffectPresetCount
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** только для чтения (**длинное целое**).

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
