# <a name="other-skin-samples"></a>Другие примеры обложки

Чтобы просмотреть образцы всех рабочих обложек, нажмите кнопку **Пуск** , выберите **программы**, **Windows Media**, **Windows Media SDK**, а затем — **примеры пакетов SDK для проигрывателя**. Для использования примеров необходимо сначала установить пакет SDK для проигрывателя Windows Media.
