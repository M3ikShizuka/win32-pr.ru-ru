# <a name="onfocus"></a>onfocus

Обработчик событий **onfocus** обрабатывает событие, происходящее при получении элементом фокуса клавиатуры.

``` syntax
onfocus
```

## <a name="remarks"></a>Remarks

Обработчик событий **onfocus** неприменим к элементу **Effects** или элементу **Popup** .

|--------------------|-----------------------------------------------------|
