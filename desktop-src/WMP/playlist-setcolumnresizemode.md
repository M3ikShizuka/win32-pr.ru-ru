# <a name="playlistsetcolumnresizemode"></a>Список воспроизведения. Сетколумнресиземоде

Метод **сетколумнресиземоде** определяет, как именно изменяются размеры индексированных столбцов.

``` syntax
        elementID.setColumnResizeMode(column, mode)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="column"></span><span id="COLUMN"></span>*рубрик*
</dt> <dd>

**Число** (**длинное**), указывающее индекс столбца, который необходимо изменить.

</dd> <dt>

<span id="mode"></span><span id="MODE"></span>*режима*
</dt> <dd>

**Строка** , указывающая режим изменения размера. Содержит одно из следующих значений:



| Значение          | Описание                                                                                                    |
|----------------|----------------------------------------------------------------------------------------------------------------|
| аутосизехеадер | Столбец изменяется в соответствии со всеми данными в столбце и в заголовке.                                  |
| аутосизедата   | Столбец изменяется в соответствии со всеми данными в столбце.                                                 |
| Фиксированный          | Столбец имеет фиксированный размер.                                                                                    |
| Растягивается      | Размер столбца изменяется для использования оставшегося пространства в элементе **списка воспроизведения** после изменения размера всех остальных столбцов. |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
