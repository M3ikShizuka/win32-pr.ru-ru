# <a name="type-2-online-stores"></a>Тип 2 Интернет-магазинов

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Проигрыватель Windows Media 11 поддерживает Интернет-магазины типа 1 и тип 2. Хранилища типа 1 более тесно интегрируются в пользовательский интерфейс проигрывателя по сравнению с типом 2 магазинов. Проигрыватель Windows Media 10 поддерживает магазины типа 2, но не имеет тип 1 хранилища. Эта документация, применимая только к типам магазинов типа 2, делится на следующие разделы.
