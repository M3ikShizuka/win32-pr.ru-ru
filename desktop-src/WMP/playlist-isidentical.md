# <a name="playlistisidentical-method"></a>Метод списка воспроизведения. одинаковы

Метод « **идентичен** » получает значение, указывающее, идентичен ли указанный объект **списка воспроизведения** текущему.

## <a name="syntax"></a>Синтаксис


```JScript
bRetVal = Playlist.isIdentical(
  playlist
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*список воспроизведения* \[ окне\]
</dt> <dd>

Объект **списка воспроизведения** для сравнения с текущим.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает **логическое значение**.
