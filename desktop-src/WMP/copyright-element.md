# <a name="copyright-element"></a>Элемент COPYRIGHT

Элемент **Copyright** определяет текстовую строку, указывающую сведения об авторских правах для элемента **ASX** или **entry** .

``` syntax
<COPYRIGHT>
    text string
</COPYRIGHT>
```

## <a name="attributes"></a>Атрибуты

Этот элемент не содержит атрибуты.

## <a name="parentchild-elements"></a>Родительские и дочерние элементы



| Иерархия       | Элементы           |
|-----------------|--------------------|
| Родительские элементы | **ASX**, **запись** |
| Дочерние элементы  | Нет               |



 

## <a name="remarks"></a>Remarks

Этот элемент определяет текстовую строку, указывающую сведения об авторских правах для элемента **ASX** или **entry** .

Когда этот элемент появляется в элементе **ASX** , строка авторского права отображается только как **Показать** информацию. Когда этот элемент появляется в элементе **entry** , он отображается как сведения об обрезке. Каждый родительский элемент **ASX** и **entry** должен содержать не более одного дочернего элемента **Copyright** . Несколько элементов **авторского права** после первого будут пропущены и не будут отображаться.

Символы для авторского права и символов регистрации товарного знака (или) могут отображаться неправильно, если метафайл не кодируется с помощью схемы кодировки UTF-8. В этом случае для правильного вывода любого из этих символов для всех пользователей можно использовать эквиваленты ASCII (c) и (R).

Если метафайл кодируется с помощью UTF-8, то авторские знаки и символы товарных знаков будут отображаться правильно.

## <a name="examples"></a>Примеры


```XML
<COPYRIGHT>Copyright (c) 1998, Microsoft Corporation</COPYRIGHT>

```
