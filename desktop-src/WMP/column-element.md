# <a name="column-element"></a>COLUMN, элемент

Элемент **Column** представляет столбец внутри элемента управления списка воспроизведения. Элементы **Column** всегда являются дочерними элементами **списка воспроизведения** .

Элемент **Column** поддерживает следующие атрибуты.



| attribute                                       | Описание                                                        |
|-------------------------------------------------|--------------------------------------------------------------------|
| [columnName](column-columnname.md)             | Указывает или получает имя столбца в элементе управления **списка воспроизведения** .  |
| [columnID](column-columnid.md)                 | Указывает или получает идентификатор столбца в элементе управления **списка воспроизведения** .    |
| [колумнресиземоде](column-columnresizemode.md) | Указывает или получает режим изменения размера для этого столбца.            |
| [колумнвидс](column-columnwidth.md)           | Указывает или получает ширину столбца в элементе управления **списка воспроизведения** . |
