# <a name="albuminfo-element"></a>Албуминфо, элемент

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

элемент **албуминфо** указывает URL-адрес веб-страницы, которая проигрыватель Windows Media отображается, когда пользователь решает просмотреть дополнительные сведения об определенном элементе мультимедиа.

``` syntax
<AlbumInfo
   URL = "URL"
/>
      
```

## <a name="attributes"></a>Атрибуты

<dl> <dt>

<span id="URL__required_"></span><span id="url__required_"></span><span id="URL__REQUIRED_"></span>**URL-адрес** (обязательно)
</dt> <dd>

URL-адрес веб-страницы, отображаемой проигрыватель Windows Media.

</dd> </dl>

## <a name="parentchild-elements"></a>Родительские и дочерние элементы



| Иерархия       | Элемент         |
|-----------------|-----------------|
| Родительские элементы | **ServiceInfo** |
| Дочерние элементы  | Нет            |



 

## <a name="remarks"></a>Remarks

когда пользователь нажимает кнопку в проигрыватель Windows Media для просмотра дополнительных сведений об определенном элементе мультимедиа, проигрыватель отправляет запрос URL-адреса с параметрами, прикрепленными с помощью знака вопроса (?) разделитель строк запроса. В следующей таблице приведены параметры, отправленные с помощью запроса URL-адреса. Другие могут присутствовать в целях совместимости с прежними версиями.



| Имя         | Значение                                                                                                                                                               |
|--------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *Album*      | Значение атрибута **WM/албумтитле** для элемента мультимедиа.                                                                                                        |
| *Исполнител*     | Значение атрибута **WM/албумартист** , если оно существует, или значение атрибута **Author** для элемента мультимедиа.                                         |
| *графическом*      | идентификатор географического расположения Windows. Идентификатор расположения задается пользователем в области **Расположение** в разделе региональные и языковые параметры панели управления. |
| *locale*     | проигрыватель Windows Media код локали.                                                                                                                                     |
| *Title*      | Значение атрибута **Title** для элемента мультимедиа.                                                                                                                |
| *уфид*       | Значение атрибута **WM/уникуефилеидентифиер** для элемента мультимедиа.                                                                                              |
| *UserLocale* | Windows код локали. Языковой стандарт задается пользователем в области **стандарты и форматы** языковых и региональных параметров на панели управления.        |
| *version*    | проигрыватель Windows Media номер версии в следующем формате: 10.0. x. xxxx или 11.0. x. xxxx.                                                                         |
