# <a name="designing-the-interface"></a>Проектирование интерфейса

После выбора функций можно разработать интерфейс. Для соответствия функциональным возможностям выбран простой интерфейс. Символы для элементов управления были выбраны из стандартных элементов управления ВИДЕОМАГНИТОФОНА.

На следующем рисунке показано, как будет выглядеть интерфейс.

![образец интерфейса](images/ceswmful.png)

-   **Кнопка Плайпаусе или Плайпаусестоп.** Пользователь будет в большей мере коснуться этого, поэтому вы можете подумать о большей кнопке. Правый верхний угол обеспечивает хорошее расположение, которое пользователь может быстро найти. Сплошная стрелка используется для обозначения воспроизведения и двух вертикальных полос (не показанных здесь) для обозначения приостановки.
    > [!Note]  
    > кнопка плайпаусестоп может использоваться только при создании обложки для проигрыватель Windows Media 10 Mobile или более поздней версии.

     

-   **Кнопка "Закрыть".** Чтобы упростить поиск, он размещается в левом верхнем углу. Квадрат используется для обозначения «останавливаться». если вы создаете обложку для проигрыватель Windows Media 10 Mobile или более поздней версии, кнопка плайпаусестоп уже предоставляет эту функцию.
-   **Кнопки "Далее" и "назад".** Так как они не будут использоваться как можно чаще, они помещаются слева. Следующая кнопка находится выше кнопки назад, так как людям, скорее всего, потребуется переместиться вперед в список воспроизведения. Используются символы двойной стрелки, так как они похожи в функции на элемент управления быстрого перемотки вперед.
-   **Ползунок громкости.** Он размещается в нижней части экрана и представляет собой простую линию с кнопкой Thumb на поверх нее.
-   **Текстовое поле области.** Он помещается под кнопкой Плайпаусе или Плайпаусестоп, чтобы его было легко увидеть.

Вы можете запланировать эту первую и поэкспериментировать с размещением каждого элемента пользовательского интерфейса. Показанная здесь схема была выбрана для простоты и простоты использования.
