# <a name="art-file-formats-mobile"></a>Форматы файлов изображений (мобильный)

Графические файлы для обложки могут быть изображениями точечных рисунков (BMP) или GIF-файлами (GIF). Также поддерживаются изображения в формате JPEG (JPG) и PNG (PNG). Не включайте в имя файла изображения пробелы и запятые.

Если вы синхронизируете файлы с портативным устройством, убедитесь, что синхронизация не изменяет расширение имени файла или не преобразует файлы изображений в другой формат.

Многие программы рисования можно использовать для создания точечных рисунков и GIF-файлов, а также для преобразования рисунков в другие форматы в точечные рисунки или GIF-файлы. Например, для простых чертежей можно использовать Microsoft Paint, который поставляется вместе с Microsoft Windows.

Необходимо обрезать пустые области изображений вокруг границ, чтобы уменьшить размер файла. С помощью смещений в разделе Bitmap можно компенсировать удаление пробелов в левой и верхней сторонах. Можно также обрезать нижний и правый пустые границы изображений, так как они не нужны для измерений. Только те файлы, которые не следует обрезать, — это рисунки Background, Сиксумб и Волумесумб.
