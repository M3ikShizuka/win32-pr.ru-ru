# <a name="playerplaylistcollectionchange-event"></a>Событие Player. Плайлистколлектиончанже

Событие **плайлистколлектиончанже** возникает при изменении в коллекции списков воспроизведения.

## <a name="syntax"></a>Синтаксис


```JScript
Player.PlaylistCollectionChange()
```



## <a name="parameters"></a>Параметры

У этого события нет параметров.

## <a name="return-value"></a>Возвращаемое значение

Это событие не возвращает значение.

## <a name="remarks"></a>Комментарии

**Проигрыватель Windows Media 10 Mobile:** Это событие не поддерживается.
