# <a name="playlistdropdownlist"></a>Список воспроизведения. dropDownList

Атрибут **dropDownList** указывает или получает значение, указывающее, какие элементы отображаются в раскрывающемся списке для данного экземпляра элемента **списка воспроизведения** .

``` syntax
        elementID.dropDownList
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей одно из следующих значений.



| Значение       | Описание                                                                                      |
|-------------|--------------------------------------------------------------------------------------------------|
| шовалбумс  | Показывает альбомы.                                                                                    |
| showAll     | По умолчанию. Отображает все доступные элементы, включая списки воспроизведения компакт-дисков, списки воспроизведения пользователей и предустановки радио. |
| шовкд      | Отображает список воспроизведения компакт-дисков.                                                                           |
| шовклипс   | Показывает все клипы.                                                                                 |
| шовкуррент | Отображает текущий несохраненный список воспроизведения.                                                             |
| шовлибрари | Отображает только списки воспроизведения библиотек.                                                                    |
| шоврадио   | Отображает предустановки радио.                                                                             |
| шовкуериес | Показывает запросы.                                                                                   |
