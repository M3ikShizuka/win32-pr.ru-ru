# <a name="wmcontentdistributortype-attribute"></a>Атрибут WM/Контентдистрибутортипе

Атрибут **WM/контентдистрибутортипе** является типом распространителя элемента.

## <a name="applies-to"></a>Применение

-   [Звуковые элементы](audio-item-attributes.md)
-   [Часто используемые атрибуты файлов Windows Media](commonly-used-windows-media-file-attributes.md)
-   [Списки воспроизведения](playlist-attributes-ref.md)
-   [Элементы видео](video-item-attributes.md)

## <a name="remarks"></a>Комментарии

Этому атрибуту будет присвоено значение "List" или "Radio". Этот атрибут хранится как в библиотеке, так и в файле цифрового мультимедиа.

**Контентдистрибутортипе** является псевдонимом для этого атрибута.

Константа Windows Media Format SDK для этого атрибута — g \_ всзвмконтентдистрибутортипе.

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .
