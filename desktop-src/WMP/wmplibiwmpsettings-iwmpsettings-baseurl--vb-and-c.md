# <a name="iwmpsettingsbaseurl-property"></a>Свойство Ивмпсеттингс:: baseURL

Свойство **baseURL** получает или задает базовый URL-адрес, используемый для разрешения относительных путей с помощью команд сценария URL-адреса, внедренных в мультимедийное содержимое.

## <a name="syntax"></a>Синтаксис


```CSharp
public System.String baseURL {get; set;}
```


```VB

Public Property baseURL As System.String
```





## <a name="property-value"></a>Значение свойства

**Строка System. String** , которая является базовым URL-адресом.

## <a name="remarks"></a>Remarks

Это свойство указывает базовый URL-адрес HTTP, который передается в качестве параметра команды для события **аксвиндовсмедиаплайер \_ вмпокксевентс \_ скрипткоммандевент** . Базовый URL-адрес объединяется с относительным URL-адресом следующим образом:

1.  В значение, полученное с помощью свойства **baseURL** , добавляется Замыкающая косая черта (/).
2.  Начальный период, обратная косая черта или символ косой черты (., \\ и/) удаляются из относительного URL-адреса.
3.  Относительный URL-адрес добавляется в конец базового URL-адреса.
4.  Все символы косой черты в результирующем полном URL-адресе указываются в том же направлении (преобразованном в прямую или обратную косую черту) в зависимости от направления первого символа косой черты, обнаруженного в новом URL-адресе.

**Примечание**

элемент управления проигрыватель Windows Media не поддерживает использование двух точек (..) в относительном URL-адресе для указания родителя текущего расположения.
