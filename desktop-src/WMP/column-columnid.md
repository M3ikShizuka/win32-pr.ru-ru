# <a name="columncolumnid"></a>COLUMN. columnID

Атрибут **columnid** указывает или получает идентификатор столбца в элементе управления **списка воспроизведения** .

``` syntax
        elementID.columnID
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи.

## <a name="remarks"></a>Комментарии

Значения **columnid** являются теми же значениями, которые используются с методом **GetItemInfo** для объекта **мультимедиа** . Список можно получить с помощью *носителя*. Свойство **аттрибутекаунт** для определения количества атрибутов, доступных для данного объекта **мультимедиа** . Номера индексов можно использовать с *носителем*. метод **жетаттрибутенаме** для определения имен атрибутов, которые, в свою очередь, могут передаваться на *носитель*. **getItemInfo**. Свойству **columnid** можно присвоить только эти значения, за исключением некоторых значений, которые не могут быть возвращены *носителем*. **жетаттрибутенаме**. Эти значения перечислены ниже.



| Значение     | Описание                                                                        |
|-----------|------------------------------------------------------------------------------------|
| name      | Отображает имя объекта **мультимедиа** .                                         |
| длительность  | Отображает длительность объекта **мультимедиа** .                                     |
| саурцеурл | Отображает URL-адрес объекта **мультимедиа** .                                          |
| status    | Отображает состояние копирования файлов.                                              |
| size      | Отображает размер файла, который представляет объект **мультимедиа** .                |
| Расширение | Отображает расширение имени файла, представляемого объектом **мультимедиа** . |
