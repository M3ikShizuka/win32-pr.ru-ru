# <a name="videofullscreen"></a>ВИДЕО. полноэкранный режим

Атрибут **«полный экран» указывает** или получает значение, указывающее, отображается ли видео в полноэкранном режиме.

``` syntax
        elementID.fullScreen
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                                          |
|-------|------------------------------------------------------|
| Да  | Видео отображается в полноэкранном режиме.                  |
| false | По умолчанию. Видео не отображается в полноэкранном режиме. |



 

## <a name="remarks"></a>Remarks

Это свойство может быть задано только во время выполнения после загрузки файла. Поэтому он должен быть установлен в обработчике событий скрипта. Для возврата к нормальному просмотру используется управляющая кнопка.
