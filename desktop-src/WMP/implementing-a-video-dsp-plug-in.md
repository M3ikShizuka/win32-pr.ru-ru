# <a name="implementing-a-video-dsp-plug-in"></a>Реализация подключаемого модуля DSP видео

Видеоадаптеры видеомониторов поддерживают набор видеоформатов. Цифровые видеокодеки также поддерживают набор видеоформатов. при попытке воспроизведения определенного файла видео проигрыватель Windows Media необходимо выбрать формат для отображения. Проигрыватель пытается найти оптимальное соответствие форматам видеокодека и форматам, поддерживаемым видеоадаптером, то есть, который дает наивысшее качество.

чтобы создать подключаемый модуль проигрыватель Windows Media DSP, обрабатывающий видео, сначала необходимо решить, какие форматы видео вы хотите обработать в подключаемом модуле. Пример подключаемого модуля DSP видео работает со следующими видеоформатами:

-   NV12
-   YV12
-   YUY2
-   UYVY
-   RGB32
-   RGB24
-   RGB555
-   RGB565

Выберите формат, который вы выбрали для обработки. Вы можете удалить форматы из примера подключаемого модуля, чтобы они не поддерживались дольше, а также добавить код для обработки дополнительных форматов.
