# <a name="sliderbackgroundendcolor"></a>ПОЛЗУНок. Баккграундендколор

Атрибут **баккграундендколор** указывает или получает конечный цвет фона элемента управления "ползунок".

``` syntax
        elementID.backgroundEndColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей любое значение цвета Microsoft Internet Explorer. У него нет значения по умолчанию.

## <a name="remarks"></a>Комментарии

**Баккграундендколор** используется в сочетании с **backgroundColor**. Созданный эффект является градиентным эффектом от **backgroundColor** к **баккграундендколор**.
