# <a name="videosettings-element"></a>ВИДЕОСЕТТИНГС, элемент

Элемент **видеосеттингс** предоставляет способ изменения различных параметров видео с помощью указанных здесь атрибутов и метода.

Элемент **видеосеттингс** поддерживает следующие атрибуты.



| attribute                                  | Описание                                                 |
|--------------------------------------------|-------------------------------------------------------------|
| [яркость](videosettings-brightness.md) | Указывает или получает настройку яркости видео. |
| [поставляется](videosettings-contrast.md)     | Указывает или получает параметр контрастности видео.   |
| [цвет](videosettings-hue.md)               | Указывает или получает параметр оттенка видео.        |
| [насыщенность](videosettings-saturation.md) | Указывает или получает параметр насыщенности видео. |



 

Элемент **видеосеттингс** поддерживает следующий метод.



| Метод                           | Описание                                    |
|----------------------------------|------------------------------------------------|
| [reset](videosettings-reset.md); | Сбрасывает все атрибуты в значения по умолчанию. |



 

Элемент **видеосеттингс** может реализовывать обработчики [событий \_ onChange атрибута](attribute-onchange.md) .

> [!Note]  
> для этого элемента требуется проигрыватель Windows Media для Windows XP или более поздней версии.
