# <a name="recordingtime-attribute"></a>Атрибут Рекордингтиме

Атрибут **рекордингтиме** — это дата и время исходной записи для элементов, в которых эта дата отличается от даты выпуска.

## <a name="applies-to"></a>Применяется к

-   [Звуковые элементы](audio-item-attributes.md)
-   [Элементы фото](photo-item-attributes.md)
-   [Элементы видео](video-item-attributes.md)

## <a name="remarks"></a>Remarks

Этот атрибут хранится только в библиотеке.

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .

## <a name="requirements"></a>Requirements (Требования)



| Требование | Значение |
|--------------------|--------------------------------------------------------------------------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media 9 Series или более поздней версии (элемент photo поддерживается только в проигрыватель Windows Media 10 или более поздней версии).<br/> |
