# <a name="erroritem-object"></a>Объект Ерроритем

Объект **ерроритем** предоставляет способ доступа к сведениям об ошибках.

Объект **ерроритем** поддерживает следующие свойства:



| Свойство                                           | Описание                                                                                     |
|----------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [выполняет](erroritem-condition.md)               | Извлекает значение, указывающее условие для ошибки.                                       |
| [кустомурл](erroritem-customurl.md)               | Извлекает URL-адрес сайта, который отображает конкретные сведения о сбое загрузки кодека. |
| [errorCode](erroritem-errorcode.md)               | Извлекает текущий код ошибки.                                                               |
| [еррорконтекст](erroritem-errorcontext.md)         | Извлекает значение, указывающее контекст ошибки.                                          |
| [errorDescription](erroritem-errordescription.md) | Извлекает описание ошибки.                                                           |
| **исправить**                                         | Зарезервировано для последующего использования.                                                                        |



 

Доступ к объекту **ерроритем** осуществляется с помощью следующих методов.



| Объект                    | Метод                   |
|---------------------------|--------------------------|
| [Ошибка](error-object.md) | [Item](error-item.md)   |
| [Носитель](media-object.md) | [error](media-error.md) |



 

В целях иллюстрации, *Player*. *Ошибка*. *элемент*(*индекс*) используется в разделах синтаксиса Reference.
