# <a name="texthoverforegroundcolor"></a>TEXT. Ховерфореграундколор

Атрибут **ховерфореграундколор** указывает или получает цвет текста, используемый для текстового элемента управления при наведении на него курсора мыши.

``` syntax
        elementID.hoverForegroundColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей любое значение цвета Microsoft Internet Explorer.

## <a name="remarks"></a>Remarks

Если **ховерфореграундколор** не указан, используется **foregroundColor** .

См. атрибут [value](text-value.md) для примера, иллюстрирующий использование атрибутов **текстового** элемента.
