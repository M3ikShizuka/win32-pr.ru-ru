# <a name="iwmpcdromburnstopburn-method"></a>Метод Ивмпкдромбурн:: Стопбурн

Метод **стопбурн** останавливает процесс записи компакт-дисков.

## <a name="syntax"></a>Синтаксис


```CSharp
public void stopBurn();
```


```VB

Public Sub stopBurn()
Implements IWMPCdromBurn.stopBurn
```

## <a name="parameters"></a>Параметры
