# <a name="text-location"></a>Расположение текста

Необходимо определить расположение текстового поля, которое вы хотите использовать. Для этого необходимо предоставить четыре положительных целых числа, разделенные запятыми, которые определяют левую, верхнюю, ширину и высоту текстового поля, отображаемого в пикселях, относительно фонового изображения.


```C++
180,46,50,30 

```
