# <a name="eventtoelement"></a>Event. Тоелемент

Атрибут **тоелемент** извлекает элемент, в который переместился фокус клавиатуры. Этот атрибут применяется только к событию **onblurия** ; для всех остальных событий его значение равно **null**.

``` syntax
event.toElement
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является объектом, предназначенным только для чтения.

|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
