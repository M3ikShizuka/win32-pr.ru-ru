# <a name="writing-event-code"></a>Написание кода события

События обрабатываются аналогично атрибутам. Необходимо присвоить событию значение, а значение — это код, который будет выполняться при наступлении события. Слово «ON» добавляется в начало имени события; Например, событие Click станет **OnClick**.

значение события указывается в двойных кавычках и начинается с слова JScript, за которым следует двоеточие. Код, который вы хотите запустить, поступает после точки с запятой и закрывающими двойными кавычками. Например, чтобы закончить воспроизведение при нажатии пользователем кнопки, введите следующий код в качестве атрибута в коде элемента **кнопки** :


```C++
onclick = "JScript: player.Controls.Stop() ; "
```



Если у вас есть код, требующий кавычек, используйте одинарные кавычки. Будьте осторожны при использовании кавычек, чтобы они были правильно сбалансированы. Ниже приведен пример использования обоих типов:


```C++
onclick = "JScript: player.URL = 'https://proseware.com/laure.wma' ; "
```



Можно также изменить атрибуты обложки при обработке внешнего события. Например, чтобы закрыть представление с именем myView, можно ввести следующее:


```C++
onclick = "JScript: myView.close() ;"
```
