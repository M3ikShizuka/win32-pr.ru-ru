# <a name="onmouseup"></a>OnMouseUp

Обработчик событий **OnMouseUp** обрабатывает событие, возникающее, когда пользователь отпускает кнопку мыши, когда указатель находится над элементом.

``` syntax
onmouseup
```

## <a name="remarks"></a>Комментарии

Этот обработчик событий неприменим к элементу **списка воспроизведения** .

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|-----------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 70 или более поздней<br/> |
