# <a name="buttonelementdown"></a>БУТТОНЕЛЕМЕНТ.

Атрибут **Down** указывает или получает значение, указывающее, находится ли элемент кнопки в положении "вверх" или "вниз".

``` syntax
        elementID.down
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                                                     |
|-------|-----------------------------------------------------------------|
| Да  | Указывает, что **буттонелемент** находится в расположении вниз.        |
| false | По умолчанию. Указывает, что **буттонелемент** находится в положении Up. |



 

## <a name="remarks"></a>Remarks

Чтобы элемент Button оставался в положении вниз, необходимо установить для параметра **закрепить** значение true. По умолчанию **закрепление** имеет значение false, и любая попытка **установить значение** true будет пропущена.

Если указано недопустимое значение, сохраняется предыдущее состояние.
