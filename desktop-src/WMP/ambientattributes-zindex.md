# <a name="ambientattributeszindex"></a>Амбиентаттрибутес. zIndex

Атрибут **zIndex** указывает или получает порядок, в котором отображается элемент управления.

``` syntax
        elementID.zIndex
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут имеет **номер** для чтения и записи (**длинный**) со значением по умолчанию 0. Диапазон состоит из длинного целого числа со знаком.

## <a name="remarks"></a>Комментарии

Фоновый рисунок **представления** или **подпредставления** имеет фиксированный нулевой индекс, равный нулю. Если требуется, чтобы элемент управления находящийся за фоном, **zIndex** должен иметь отрицательное значение.

Z-индекс **представления** или **подпредставления** является абсолютным индексом, а z-индексом элемента управления относительно индекса по оси z **представления** или **подпредставления** , содержащего его.

Атрибут **zIndex** не поддерживается элементами **браузера** и **списка воспроизведения** . Он не будет работать с элементом **Video** , если *видео*. **окно без окон** имеет значение false или элемент **Effects** , если он имеет **эффект**. для **окон** задано значение true.

Элементы **буттонелемент** используют **zIndex** **BUTTONGROUP**.
