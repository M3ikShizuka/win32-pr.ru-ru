# <a name="downloadcollection-object"></a>Объект Довнлоадколлектион

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Объект **довнлоадколлектион** содержит свойства и методы для управления коллекциями элементов загрузки. Она может использоваться веб-страницами, размещенными в проигрывателе Windows Media в полном режиме и имеющими доступ к **внешнему** объекту, например Интернет-магазинам.

Объект **довнлоадколлектион** поддерживает следующие свойства:



| Свойство                              | Описание                                  |
|---------------------------------------|----------------------------------------------|
| [count](downloadcollection-count.md) | Возвращает число незавершенных Скачиваний.   |
| [id](downloadcollection-id.md)       | Возвращает идентификатор коллекции загрузки. |



 

Объект **довнлоадколлектион** поддерживает следующие методы.



| Метод                                                | Описание                                   |
|-------------------------------------------------------|-----------------------------------------------|
| [Очистить](downloadcollection-clear.md)                 | Удаляет все элементы из коллекции загрузки. |
| [Item](downloadcollection-item.md)                   | Извлекает объект, ожидающий загрузки.          |
| [removeItem](downloadcollection-removeitem.md)       | Удаляет элемент загрузки из коллекции.    |
| [стартдовнлоад](downloadcollection-startdownload.md) | Загружает в очередь.                            |



 

Доступ к объекту **довнлоадколлектион** осуществляется с помощью следующих методов.



| Объект                                        | Метод                                                                   |
|-----------------------------------------------|--------------------------------------------------------------------------|
| [довнлоадманажер](downloadmanager-object.md) | [креатедовнлоадколлектион](downloadmanager-createdownloadcollection.md) |
| [довнлоадманажер](downloadmanager-object.md) | [жетдовнлоадколлектион](downloadmanager-getdownloadcollection.md)       |



 

В целях иллюстрации **довнлоадманажер**. **жетдовнлоадколлектион**(*CollectionID*) используется в разделах синтаксиса Reference.
