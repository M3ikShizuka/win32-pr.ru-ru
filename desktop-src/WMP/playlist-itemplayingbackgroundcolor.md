# <a name="playlistitemplayingbackgroundcolor"></a>Список воспроизведения. Итемплайингбаккграундколор

Атрибут **итемплайингбаккграундколор** указывает или получает цвет фона текущего элемента списка воспроизведения.

``` syntax
        elementID.itemPlayingBackgroundColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей любое значение цвета Microsoft Internet Explorer. Он имеет значение по умолчанию (" \# 222222").

|--------------------|------------------------------------------------------|
