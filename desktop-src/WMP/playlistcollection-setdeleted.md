# <a name="playlistcollectionsetdeleted-method"></a>Плайлистколлектион. Сетделетед, метод

Метод **сетделетед** перемещает список воспроизведения в папку Удаленные элементы.

## <a name="syntax"></a>Синтаксис


```JScript
PlaylistCollection.setDeleted(
  playlist,
  true
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*список воспроизведения* \[ окне\]
</dt> <dd>

Перемещаемый объект **списка воспроизведения** .

</dd> <dt>

*значение true* \[ окне\]
</dt> <dd>

Всегда указывайте это значение.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Remarks

**проигрыватель Windows Media 10 Mobile**: этот метод не поддерживается.
