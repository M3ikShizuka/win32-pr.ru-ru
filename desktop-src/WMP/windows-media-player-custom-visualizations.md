# <a name="windows-media-player-custom-visualizations"></a>проигрыватель Windows Media Пользовательские визуализации

Microsoft проигрыватель Windows Media предоставляет визуализации, позволяющие пользователю видеть визуальные изображения, синхронизированные с звуковым содержимым воспроизводимого мультимедиа. в проигрыватель Windows Media включены несколько стандартных визуализаций, включая пики, линейчатые и дотплане. В этом разделе пакета SDK содержатся сведения о программировании, позволяющие создавать собственные визуализации.

Документация по пользовательской визуализации делится на три раздела:



| Section                                                                                      | Описание                                                                                                                                                          |
|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [О пользовательских визуализациях](about-custom-visualizations.md)                               | Описание архитектуры пользовательских визуализаций в абстрактных терминах. ознакомьтесь с этим разделом, чтобы понять, как визуальные представления связаны с проигрыватель Windows Media. |
| [Специальное руководством по программированию визуализации](custom-visualization-programming-guide.md)         | Объясняет, что необходимо сделать для создания пользовательской визуализации. В этом разделе содержатся полезные указания и советы, которые помогут вам создать интересные визуализации.     |
| [Справочник по программированию пользовательской визуализации](custom-visualization-programming-reference.md) | Содержит ссылку на интерфейс, который необходимо реализовать, а также другие технические справочные сведения.                                                          |
