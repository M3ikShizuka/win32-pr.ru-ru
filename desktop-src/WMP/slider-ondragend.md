# <a name="sliderondragend"></a>ПОЛЗУНок. Ондраженд

Обработчик событий **ондраженд** обрабатывает событие, возникающее при отпускании левой кнопки мыши после операции перетаскивания.

``` syntax
onDragEnd
```

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
