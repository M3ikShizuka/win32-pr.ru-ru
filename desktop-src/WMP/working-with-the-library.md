# <a name="working-with-the-library"></a>Работа с библиотекой

Библиотека — это база данных сведений о мультимедийном содержимом, которое хранится на компьютере пользователя или воспроизводится. некоторые сведения отображаются в функции библиотеки в проигрыватель Windows Media. Доступ к более широкому набору сведений можно получить программным способом.

Элементы мультимедиа можно упорядочивать в списках воспроизведения. Списки воспроизведения могут создаваться пользователями или с помощью кода. объектная модель проигрыватель Windows Media предоставляет объекты и методы, которые можно использовать для управления отдельными элементами мультимедиа и списками воспроизведения.

В следующих разделах описываются методы работы с библиотекой.

-   [Доступ к библиотеке](library-access.md)
-   [Управление элементами мультимедиа](managing-media-items.md)
-   [Атрибуты элемента мультимедиа](media-item-attributes.md)
-   [Управление списками воспроизведения](managing-playlists.md)
