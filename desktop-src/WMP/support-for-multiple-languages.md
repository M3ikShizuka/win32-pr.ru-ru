# <a name="support-for-multiple-languages"></a>Поддержка нескольких языков

проигрыватель Windows Media 9 Series или более поздней версии поддерживает Windows метафайлы мультимедиа, созданные с помощью набора символов юникода. Это позволяет включать в список воспроизведения метафайлов многоязыковые метаданные. следующие правила управляют использованием многоязыковых метаданных в Windows метафайлов мультимедиа:

-   Символы должны быть закодированы с помощью схемы кодировки UTF-8.
-   Список воспроизведения метафайлов должен включать следующий **параметр** на уровне списка воспроизведения:
    ```XML
    <PARAM  NAME = "Encoding"  VALUE = "utf-8">
    
    ```

    

-   эта функция поддерживается только проигрыватель Windows Media 9 Series или более поздних версий.
-   Если список воспроизведения метафайлов не сохранен с кодировкой UTF-8 и корректным элементом **param** , он будет проанализирован с помощью кодовой страницы локального компьютера пользователя, используемой по умолчанию.
