# <a name="status-location"></a>Расположение состояния

Необходимо определить расположение представления состояния, которое вы хотите использовать. Необходимо указать четыре положительных целых числа, разделенные запятыми, которые определяют левую, верхнюю, ширину и высоту отображаемого состояния (в пикселях) относительно фонового изображения.


```C++
45,193,120,13 

```
