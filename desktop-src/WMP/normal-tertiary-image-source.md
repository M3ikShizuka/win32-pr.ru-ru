# <a name="normal-tertiary-image-source"></a>Нормальный источник третьего образа

Функция кнопки Плайпаусестоп требует, чтобы вы определили расположение обычного изображения для третьего состояния кнопки. Это будет изображение, которое пользователи видят после нажатия кнопки Плайпаусестоп, чтобы начать потоковую передачу вещания в реальном времени.

Чтобы определить этот образ, необходимо ввести тип изображения, за которым следует пробел, символ @ и еще один пробел. Затем необходимо ввести два положительных целых числа, определяющие координаты верхнего левого угла (в пикселях) изображения, которое нужно использовать внутри типа точечного рисунка.

Например, чтобы определить нормальный образ для источника "третичное изображение", если изображение находится внутри точечного рисунка, введите:


```C++
Pushed @ 286,0

```



В третичных состояниях не может быть отключенного образа. Предполагается, что для третьего и дополнительного образов используются одинаковые значения ширины и высоты.
