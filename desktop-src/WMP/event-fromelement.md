# <a name="eventfromelement"></a>Event. Фромелемент

Атрибут **фромелемент** извлекает элемент, полученный от события. Если ни один элемент не передает событие, значение равно **null**.

``` syntax
event.fromElement
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является объектом, предназначенным только для чтения.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
