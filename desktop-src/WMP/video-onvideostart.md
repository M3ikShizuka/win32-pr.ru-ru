# <a name="videoonvideostart"></a>ВИДЕО. онвидеостарт

Обработчик событий **онвидеостарт** обрабатывает событие, возникающее при загрузке видео и начале его подготовки к просмотру.

``` syntax

onvideostart 
```

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
