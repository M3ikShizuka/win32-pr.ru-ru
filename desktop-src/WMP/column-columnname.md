# <a name="columncolumnname"></a>COLUMN. columnName

Атрибут **ColumnName** указывает или получает имя столбца в элементе управления **списка воспроизведения** .

``` syntax
        elementID.columnName
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи.

## <a name="remarks"></a>Комментарии

Имя столбца — это понятное имя, которое отображается в заголовке столбца.
