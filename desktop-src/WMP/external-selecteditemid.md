# <a name="externalselecteditemid"></a>External. Селектедитемид

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

свойство **селектедитемид** извлекает идентификатор элемента мультимедиа, выбранного в данный момент в проигрыватель Windows Media.

## <a name="syntax"></a>Синтаксис

``` syntax
window.external.selectedItemID
```

## <a name="possible-values"></a>Возможные значения

Это свойство является **строкой**, доступная только для чтения.

## <a name="remarks"></a>Remarks

Это свойство используется в сочетании со свойством [External. селектедитемтипе](external-selecteditemtype.md) . Например, если **селектедитемтипе** равен кптраккид, то **СЕЛЕКТЕДИТЕМИД** — это идентификатор выбранной записи. дополнительные сведения о том, как проигрыватель Windows Media характеризует представления содержимого интернет-магазина, см. в разделе [расположение и выбранный элемент](location-and-selected-item.md).

для определенных представлений в проигрыватель Windows Media выбран определенный элемент мультимедиа. Например, предположим, что текущее представление представляет собой альбом. Альбом — это контейнер дорожек, поэтому **селектедитемтипе** равен кптраккид, а **селектедитемид** — идентификатор выбранной дорожки. Другие представления не имеют выбранного элемента мультимедиа. например, если пользователь щелкает корневой узел интернет-магазина в элементе управления "дерево", проигрыватель Windows Media отображает страницу обнаружения, предоставленную в интернет-магазине. Проигрыватель не отображает ни одного контейнера элементов мультимедиа в пользовательском интерфейсе проигрывателя. В этом случае **селектедитемтипе** равно ункновнлокатион, а **селектедитемид** равен пустой строке.
