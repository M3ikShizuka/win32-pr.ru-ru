# <a name="ambientattributeselementtype"></a>Амбиентаттрибутес. elementType

Атрибут **ElementType** Извлекает тип элемента (например, Button).

``` syntax
        elementID.elementType
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** только для чтения, указывающей имя элемента.

## <a name="remarks"></a>Комментарии

Этот атрибут полезен при определении типа элемента, который вызвал событие, и записи универсального обработчика событий для этого класса элементов.
