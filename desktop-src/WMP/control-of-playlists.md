# <a name="control-of-playlists"></a>Управление списками воспроизведения

Управление списками воспроизведения осуществляется клиентом. списки воспроизведения анализируются проигрыватель Windows Media, которая принимает действия на основе элементов списка воспроизведения. Клиент управляет процессом потоковой передачи.
