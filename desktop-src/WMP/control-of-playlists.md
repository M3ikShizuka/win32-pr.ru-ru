# <a name="control-of-playlists"></a>Управление списками воспроизведения

Управление списками воспроизведения осуществляется клиентом. Списки воспроизведения анализируются проигрывателем Windows Media, который принимает действия на основе элементов списка воспроизведения. Клиент управляет процессом потоковой передачи.
