# <a name="externalnavigatetaskpaneurl-method"></a>External. Навигатетаскпанеурл, метод

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Метод **навигатетаскпанеурл** открывает веб-страницу в указанной области задач и изменяет фокус на указанную панель.

## <a name="syntax"></a>Синтаксис


```JScript
External.NavigateTaskPaneURL(
  bstrKeyName,
  bstrTaskPane,
  bstrParams
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*бстркэйнаме* \[ окне\]
</dt> <dd>

**Строка** , содержащая имя ключа для Интернет-магазина. (обязательно)

</dd> <dt>

*бстртаскпане* \[ окне\]
</dt> <dd>

**Строка** , содержащая имя области задач, в которой открывается веб-страница. (обязательно)

</dd> <dt>

*бстрпарамс* \[ окне\]
</dt> <dd>

**Строка** , содержащая параметры строки запроса для добавления к URL-адресу, указанному элементом **navigate** документа serviceInfo. (необязательно).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Переход к панели, не поддерживаемой Вашим Интернет-магазином, может привести к изменению текущего Интернет-магазина.

Значение, указанное для *бстрпарамс* , допустимо только в том случае, если **навигатетаскпанеурл** вызывается из веб-страниц, предоставляемых Интернет-магазином.

В следующей таблице перечислены допустимые значения для *бстртаскпане* и связанная область задач для каждого из них.



| Значение            | Панель задач                      |
|------------------|--------------------------------|
| **ServiceTask1** | Первая область задач оперативного хранилища.  |
| **ServiceTask2** | Вторая область задач оперативного магазина. |
| **ServiceTask3** | Третья область задач Интернет-магазина.  |



 

Код веб-страницы должен указывать значение [External. селектедтаскпане](external-selectedtaskpane.md) при загрузке, чтобы после завершения навигации была выбрана правильная кнопка области задач.

## <a name="examples"></a>Примеры

В следующем примере кода показано, как **навигатетаскпанеурл** создает URL-адрес веб-страницы, отображаемой для **ServiceTask1**.

Пример элемента **navigate** :


```XML
<Navigate
    BaseURL = "https://www.proseware.com/online store/html/navigate.asp">
</Navigate>
```



Пример вызова метода **навигатетаскпанеурл** :


```XML
external.NavigateTaskPaneURL("Proseware", "ServiceTask1", "Pane=Store");
```



Пример итогового URL-адреса, используемого для веб-страницы, отображаемой в **ServiceTask1**:


```XML
https://www.proseware.com/online store/html/navigate.asp?Pane=Store
```
