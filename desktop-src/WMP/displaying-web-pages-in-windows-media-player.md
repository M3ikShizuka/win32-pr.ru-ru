функция проигрыватель Windows Media хтмлвиев позволяет поставщикам содержимого интернета (icp) использовать функцию **воспроизводимого** проигрывателя в полноэкранном режиме для показа веб-содержимого во время воспроизведения цифрового мультимедийного содержимого.
