# <a name="object-model-reference-for-c"></a>Справочник по объектной модели для C++

> [!Note]  
> Использование элемента управления ActiveX проигрывателя Windows Media с C++ поддерживается для Windows Media Player 9 Series или более поздней версии.

 

В справочнике по объектной модели для C++ содержится документация для мобильных интерфейсов проигрывателя Windows Media и Windows Media Player и типов перечислений.

Если вы хотите использовать удаленное взаимодействие или обложки с элементом управления проигрывателя Windows Media, вам также потребуется изучить интерфейсы **ивмпплайерсервицес** и **ивмпремотемедиасервицес** , предназначенные для использования с C++.

Демонстрацию внедрения элемента управления проигрывателя Windows Media в программу на C++ см. в разделе примеры C++, входящие в состав этого пакета SDK.

В следующих разделах описываются интерфейсы и перечисления C++, предоставляемые объектной моделью проигрывателя Windows Media.



| Section                                    | Описание                                                                                                     |
|--------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [Интерфейсы](interfaces.md)               | Список интерфейсов и методов, доступных для программистов C++, использующих элемент управления ActiveX проигрывателя Windows Media. |
| [Типы перечислений](enumeration-types.md) | Перечисляет типы перечисления, доступные в коде C++.                                                              |
