# <a name="status-alignment"></a>Выравнивание состояния

Необходимо определить способ выравнивания отображаемого состояния. В следующей таблице приведены допустимые значения.



| Значение  | Описание                                                                       |
|--------|-----------------------------------------------------------------------------------|
| Левый   | Текст выводится по левому краю (выводится вверх по левому краю экрана состояния).   |
| Center | Текст отображается в центре экрана состояния.                                       |
| Правый  | Текст выводится по правому краю (выводится с правом края экрана состояния). |
