# <a name="using-windows-media-player-with-netscape-71"></a>Использование проигрывателя Windows Media с Netscape 7.1

Веб-браузер Netscape 7,1 обеспечивает встроенную поддержку элементов управления ActiveX проигрывателя Windows Media с помощью компонента ActiveX Gecko. Это означает, что вы можете внедрить элемент управления проигрывателя Windows Media на веб-страницы для Netscape 7,1 с тем же синтаксисом, который использовался для Microsoft Internet Explorer. Netscape 7,1 также поддерживает скрипты управления и обработки событий аналогичным образом.

Если вы хотите использовать элемент управления проигрывателя Windows Media в Netscape, рекомендуется использовать Netscape 7,1.

Дополнительные сведения об использовании элемента управления проигрывателя Windows Media с Netscape 7,1 см. на [веб-сайте Netscape](https://developer.mozilla.org/docs/Archive/Windows_Media_in_Netscape).

Дополнительные сведения о компоненте ActiveX Gecko см. на [веб-сайте StreamingMedia.com](https://www.streamingmedia.com/Articles/Editorial/Featured-Articles/The-Windows-Media-ActiveX-Control-8211%3b-Not-Just-for-Internet-Explorer-Anymore-64608.aspx).

Полное скачивание этого пакета SDK включает пример, демонстрирующий обнаружение веб-браузера. Дополнительные сведения о примерах, входящих в состав пакета SDK, см. в разделе [Samples](samples.md).
