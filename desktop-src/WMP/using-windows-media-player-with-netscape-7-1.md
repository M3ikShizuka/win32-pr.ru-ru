# <a name="using-windows-media-player-with-netscape-71"></a>Использование проигрывателя Windows Media с Netscape 7.1

веб-браузер Netscape 7,1 обеспечивает встроенную поддержку элементов управления проигрыватель Windows Media ActiveX с помощью компонента ActiveX Gecko. это означает, что можно внедрить элемент управления проигрыватель Windows Media на веб-страницах для Netscape 7,1 с тем же синтаксисом, который использовался для Microsoft Internet Explorer. Netscape 7,1 также поддерживает скрипты управления и обработки событий аналогичным образом.

если вы хотите использовать элемент управления проигрыватель Windows Media в netscape, рекомендуется использовать netscape 7,1.

дополнительные сведения об использовании элемента управления проигрыватель Windows Media с Netscape 7,1 см. на [веб-сайте netscape](https://developer.mozilla.org/docs/Archive/Windows_Media_in_Netscape).

дополнительные сведения о компоненте ActiveX Gecko см. на [веб-сайте StreamingMedia.com](https://www.streamingmedia.com/Articles/Editorial/Featured-Articles/The-Windows-Media-ActiveX-Control-8211%3b-Not-Just-for-Internet-Explorer-Anymore-64608.aspx).

Полное скачивание этого пакета SDK включает пример, демонстрирующий обнаружение веб-браузера. Дополнительные сведения о примерах, входящих в состав пакета SDK, см. в разделе [Samples](samples.md).
