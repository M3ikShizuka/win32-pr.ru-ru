# <a name="nextelement"></a>некстелемент

Это предопределенный **буттонелемент** со следующими значениями по умолчанию.

``` syntax
onclick="jscript:player.controls.next()"
upToolTip="Next"
cursor="system"
enabled="wmpenabled:player.controls.next"
```

## <a name="remarks"></a>Комментарии

Все свойства этого **буттонелемент** можно переопределить, явно указав их.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
