# <a name="windows-media-playlist-elements-reference"></a>Windows Справочник по элементам списка воспроизведения мультимедиа

этот раздел содержит справочную документацию для всех Windows элементов списка воспроизведения мультимедиа. К записям ссылок относятся определения элементов, их атрибуты и значения, а также специальные условия, связанные с каждым элементом.

Поддерживаются следующие элементы списка воспроизведения.



| Элемент                                    | Описание                                                                                                                                                                                          |
|--------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [argument](argument-element.md)           | Содержит одну часть строки условия.                                                                                                                                                          |
| [body](body-element.md)                   | Содержит элементы, определяющие содержимое списка воспроизведения.                                                                                                                                        |
| [filter](filter-element.md)               | Содержит элементы, ограничивающие размер списка воспроизведения, длительность списка воспроизведения или число элементов мультимедиа в списке воспроизведения.                                                                                 |
| [экстент](fragment-element.md)           | Задает одно условие запроса, которое выбирает элементы из библиотеки.                                                                                                                            |
| [Глава](head-element.md)                   | Содержит метаданные, которые применяются ко всему списку воспроизведения.                                                                                                                                               |
| [media](media-element.md)                 | Указывает один из элементов мультимедиа в списке воспроизведения.                                                                                                                                                      |
| [метаконфигурации](meta-element.md)                   | Указывает метаданные, которые применяются ко всему списку воспроизведения.                                                                                                                                              |
| [запрос](queryset-element.md)           | Содержит элементы, которые определяют, какие элементы мультимедиа будут выбраны из библиотеки.                                                                                                                   |
| [порядк](seq-element.md)                     | Содержит элементы, которые определяют статический список воспроизведения или элементы, определяющие интеллектуальный список воспроизведения.                                                                                                            |
| [смартплайлист](smartplaylist-element.md) | Содержит динамически определенную часть списка воспроизведения.                                                                                                                                              |
| [SMIL](smil-element.md)                   | элемент **smil** всегда является элементом верхнего уровня в файле списка воспроизведения Windows Media (WPL). Он указывает, что в файле используется SMIL (синхронизированный язык интеграции мультимедиа) синтаксис и грамматика. |
| [саурцефилтер](sourcefilter-element.md)   | Содержит элементы, которые выбирают элементы из библиотеки.                                                                                                                                                  |
| [title](title-element--wpl.md)            | Указывает название списка воспроизведения.                                                                                                                                                                  |
