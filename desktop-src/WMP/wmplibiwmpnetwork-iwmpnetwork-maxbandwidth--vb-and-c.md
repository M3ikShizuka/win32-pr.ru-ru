# <a name="iwmpnetworkmaxbandwidth-property"></a>Свойство Ивмпнетворк:: maxBandwidth

Свойство **maxBandwidth** Возвращает или задает максимально допустимую пропускную способность.

## <a name="syntax"></a>Синтаксис


```CSharp
public System.Int32 maxBandwidth {get; set;}
```


```VB

Public Property maxBandwidth As System.Int32
```





## <a name="property-value"></a>Значение свойства

Значение **System. Int32** , которое является максимальной пропускной способностью.

## <a name="remarks"></a>Remarks

Это свойство не имеет значения по умолчанию. значение может быть указано, когда воспроизводится проигрыватель Windows Media, но изменение не вступит в силу до тех пор, пока текущий элемент мультимедиа не будет освобожден, открыв другой или вызвав **аксвиндовсмедиаплайер. close**. проигрыватель Windows Media пытается добиться максимально возможной пропускной способности. Если это значение не задано, то появление пропускной способности не преднамерено.

Этот параметр полезен для уменьшения объема используемой пропускной способности, особенно в случае потока с несколькими скоростями (MBR). Поток MBR содержит несколько потоков с разной скоростью. В некоторых случаях может быть желательно использовать поток с более низкой скоростью, чем требуется клиенту. В этом случае при указании более низкого бита с помощью свойства **ивмпнетворк. maxBandwidth** выбирается поток с более низким битом. Например, поток MBR может включать потоки в кодировке 20 килобит в секунду (кбит/с), 37 кбит/с и 200 кбит/с. Использование **ивмпнетворк. maxBandwidth** для указания скорости 50 000 (50 кбит/с) выберет поток 37 кбит/с, а не поток 200 кбит/с.
