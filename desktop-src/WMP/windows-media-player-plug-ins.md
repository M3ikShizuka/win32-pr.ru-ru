# <a name="windows-media-player-plug-ins"></a>проигрыватель Windows Media Подключаемые модули

Microsoft проигрыватель Windows Media предоставляет интерфейсы, позволяющие расширять функциональные возможности несколькими способами. В следующих разделах описаны поддерживаемые архитектуры подключаемых модулей и процесс создания подключаемого модуля.



| Section                                                                                                         | Описание                                                                                                                                                                                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Создание подключаемого модуля](building-a-plug-in.md)                                                                    | можно создать несколько типов подключаемых модулей с помощью Visual Studio и мастера подключаемых модулей проигрыватель Windows Media.                                                                                                                                                                                             |
| [проигрыватель Windows Media Пользовательские визуализации](windows-media-player-custom-visualizations.md)                    | проигрыватель Windows Media визуализации — это объекты модели COM, используемые для отображения визуальных изображений, которые синхронизируются с звуковой частью воспроизведения проигрывателя. Пользовательские визуализации можно создавать с помощью Microsoft Visual C++.                                             |
| [проигрыватель Windows Media Подключаемые модули пользовательского интерфейса](windows-media-player-user-interface-plug-ins.md)                | проигрыватель Windows Media подключаемые модули пользовательского интерфейса добавляют новые функции в область **Now (воспроизведение** ) в проигрывателе полного режима. Вы можете создавать подключаемые модули, использующие область визуализации, отдельное окно, область параметров, область метаданных или фоновые подключаемые модули, которые не предоставляют видимого пользовательского интерфейса. |
| [проигрыватель Windows Media Подключаемые модули DSP](windows-media-player-dsp-plug-ins.md)                                      | проигрыватель Windows Media Подключаемые модули DSP изменяют или обрабатывают аудио-и видеоданные перед отрисовкой проигрывателем. подключаемые модули DSP — это мультимедийные объекты DirectX (DMO), которые подключаются к проигрывателю через интерфейсы COM.                                                                                           |
| [проигрыватель Windows Media Модули подготовки отчетов (не рекомендуется)](/previous-versions/windows/desktop/legacy/dd564683(v=vs.85)) | декодирование подключаемых модулей проигрыватель Windows Media подготовки к просмотру (при необходимости) и отображение пользовательских данных, содержащихся в Windowsном потоке формата мультимедиа. подключаемые модули подготовки отчетов являются мультимедийными объектами DirectX (DMO), которые подключаются к проигрывателю через интерфейсы COM.                                                                  |
| [проигрыватель Windows Media Подключаемые модули преобразования](windows-media-player-conversion-plug-ins.md)                        | подключаемые модули преобразования проигрыватель Windows Media преобразуют цифровые файлы мультимедиа, созданные с помощью технологий, не предоставляемых корпорацией майкрософт, в расширенный формат систем (ASF).                                                                                                                                 |
