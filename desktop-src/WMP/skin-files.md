# <a name="skin-files"></a>Файлы обложки

При появлении обложки вы видите только поверхность. Под областью находится несколько файлов, составляющих полную обложку. Технически говоря, Обложка — это группа файлов, каждый из которых содержит конкретный тип информации. Ниже приведены типы файлов, которые можно использовать для создания полной обложки.



| Тип файла                                        | Описание                                                                                                                                                               |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Файл определения обложки](skin-definition-file.md) | Главный файл, который определяет, как будут использоваться другие файлы. Это текстовый файл с расширением WMS.                                                    |
| [Файлы изображений](art-files.md)                       | Файлы изображений, содержащие графические элементы обложки. К ним относятся файлы Bitmap, GIF, JPEG и PNG.                                                                 |
| [Файлы JScript](jscript-files.md)               | Если вы хотите создавать более сложные ответы на события, можно создать файлы скриптов с помощью Microsoft JScript. Это текстовые файлы с расширением имени файла. js. |
