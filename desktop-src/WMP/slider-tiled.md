# <a name="slidertiled"></a>ПОЛЗУНок. мозаичный

Атрибут **мозаичного заполнения** указывает или получает значение, указывающее, будет ли мозаичное изображение перемещено.

``` syntax
        elementID.tiled
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                                                                                 |
|-------|---------------------------------------------------------------------------------------------|
| true  | По умолчанию. Точечный рисунок изображения повторяется до тех пор, пока не будет заполнен весь регион элемента управления. |
| false | Изображение не будет мозаичным.                                                                |



 

## <a name="remarks"></a>Комментарии

Этот атрибут применяется только в том случае, если для определения элемента управления используется цвет переднего плана и фонового изображения. Если изображения меньше, чем определенная область ползунка, а атрибут **мозаичного заполнения** имеет значение true, изображения будут повторяться до тех пор, пока не будут заполнены все элементы управления.

Этот атрибут можно использовать вместе с атрибутом **бордерсизе** . Атрибут **бордерсизе** позволяет определить границу, которая не повторяется во время мозаичного заполнения.
