# <a name="dvdresume-method"></a>Метод DVD. Resume

Метод **Resume** возвращает в режим воспроизведения из меню в той же позиции заголовка, что и при вызове меню.

## <a name="syntax"></a>Синтаксис


```JScript
DVD.resume()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Remarks

**проигрыватель Windows Media 10 Mobile:** Этот метод всегда выполняется, но не выполняет требуемую операцию.
