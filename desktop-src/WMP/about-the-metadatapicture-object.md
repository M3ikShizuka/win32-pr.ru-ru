# <a name="about-the-metadatapicture-object"></a>Сведения об объекте Метадатапиктуре

Атрибут метаданных **WM/Picture** предоставляет подробные сведения об образах метаданных, таких как обложки альбомов. Объект **метадатапиктуре** предоставляет способ извлечения отдельных данных, связанных с изображением. Атрибут метаданных **WM/Picture** можно рассматривать как составной атрибут. При использовании *носителя*. **жетитеминфобитипе** для получения сведений об образе метаданных, извлекается объект **метадатапиктуре** со свойствами, содержащими отдельные данные.
