# <a name="editboxbackgroundcolor"></a>EDITBOX. backgroundColor

Атрибут **backgroundColor** указывает или получает цвет фона для элемента управления "поле ввода".

``` syntax
        elementID.backgroundColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей любое значение цвета Microsoft Internet Explorer, или значение None. Он имеет значение по умолчанию, равное цвету фона в Windows.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|---------------------------------------------------------|
