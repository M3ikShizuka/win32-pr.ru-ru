# <a name="user-interface-plug-ins-programming-reference"></a>Справочник по программированию подключаемых модулей пользовательского интерфейса

Пакет средств разработки программного обеспечения (SDK) проигрывателя Microsoft Windows Media поддерживает один интерфейс для подключаемых модулей пользовательского интерфейса. В следующем разделе подробно описывается этот раздел.



| Section                                                  | Описание                                                                                                   |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [вмпнотифиплугинаддремове](/previous-versions/windows/desktop/api/wmpplug/nf-wmpplug-wmpnotifypluginaddremove) | Независимая функция, используемая для уведомления проигрывателя Windows Media о том, что подключаемый модуль установлен или удален. |
| [Интерфейс Ивмпплугинуи](/previous-versions/windows/desktop/api/wmpplug/nn-wmpplug-iwmppluginui)               | Интерфейс для подключаемых модулей пользовательского интерфейса.                                                                                  |
| [Флаги регистрации](registration-flags.md)             | Список флагов параметров, влияющих на поведение подключаемых модулей пользовательского интерфейса.                                             |
