# <a name="user-interface-plug-ins-programming-reference"></a>Справочник по программированию подключаемых модулей пользовательского интерфейса

пакет средств разработки Microsoft проигрыватель Windows Media Software Development Kit (SDK) поддерживает один интерфейс для подключаемых модулей пользовательского интерфейса. В следующем разделе подробно описывается этот раздел.



| Section                                                  | Описание                                                                                                   |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [вмпнотифиплугинаддремове](/previous-versions/windows/desktop/api/wmpplug/nf-wmpplug-wmpnotifypluginaddremove) | независимая функция, используемая для уведомления проигрыватель Windows Media о том, что подключаемый модуль установлен или удален. |
| [Интерфейс Ивмпплугинуи](/previous-versions/windows/desktop/api/wmpplug/nn-wmpplug-iwmppluginui)               | Интерфейс для подключаемых модулей пользовательского интерфейса.                                                                                  |
| [Флаги регистрации](registration-flags.md)             | Список флагов параметров, влияющих на поведение подключаемых модулей пользовательского интерфейса.                                             |
