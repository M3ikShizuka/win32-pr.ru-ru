# <a name="closedcaption-object"></a>Объект Клоседкаптион

Объект **клоседкаптион** предоставляет возможность включения субтитров с помощью клипа мультимедиа. Текст субтитров находится в синхронизированном доступном файле обмена мультимедиа (SAMI).

Объект **клоседкаптион** поддерживает следующие свойства:



| Свойство                                           | Описание                                                                                          |
|----------------------------------------------------|------------------------------------------------------------------------------------------------------|
| [каптионингид](closedcaption-captioningid.md)     | Указывает или получает имя фрейма или элемента управления, отображающего субтитры.                   |
| [самифиленаме](closedcaption-samifilename.md)     | Указывает или получает имя файла, содержащего сведения, необходимые для субтитров. |
| [самиланг](closedcaption-samilang.md)             | Указывает или получает язык, отображаемый для субтитров.                                 |
| [самилангкаунт](closedcaption-samilangcount.md)   | Возвращает число языков, поддерживаемых текущим файлом SAMI.                                |
| [самистиле](closedcaption-samistyle.md)           | Указывает или получает стиль закрытых субтитров.                                                  |
| [самистилекаунт](closedcaption-samistylecount.md) | Возвращает количество стилей, поддерживаемое текущим файлом SAMI.                                   |



 

Объект **клоседкаптион** поддерживает следующие методы.



| Метод                                                 | Описание                                                                              |
|--------------------------------------------------------|------------------------------------------------------------------------------------------|
| [жетсамилангид](closedcaption-getsamilangid.md)       | Извлекает код локали (LCID) языка, поддерживаемого текущим файлом SAMI. |
| [жетсамилангнаме](closedcaption-getsamilangname.md)   | Извлекает имя языка, поддерживаемого текущим файлом SAMI.                     |
| [жетсамистиленаме](closedcaption-getsamistylename.md) | Извлекает имя стиля, поддерживаемого текущим файлом SAMI.                        |



 

Доступ к объекту **клоседкаптион** осуществляется через следующее свойство.



| Объект                      | Свойство                                  |
|-----------------------------|-------------------------------------------|
| [Игрок](player-object.md) | [клоседкаптион](player-closedcaption.md) |
