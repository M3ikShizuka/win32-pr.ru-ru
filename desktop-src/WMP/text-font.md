# <a name="text-font"></a>Шрифт текста

Необходимо определить шрифт, используемый в текстовом поле, которое вы хотите использовать. Шрифт определяется тремя значениями, разделенными запятыми, представляющими начертание, размер и плотность шрифта.

**Значения гарнитуры**

Любое имя гарнитуры можно использовать, если оно может быть установлено на компьютере пользователя. Если гарнитура не найдена на компьютере, операционная система выберет альтернативный вариант. в следующей таблице показаны гарнитуры, которые обычно находятся на устройствах на базе Windows Mobile 2003.



| Шрифт       | Описание              |
|----------------|--------------------------|
| Tahoma         | Гарнитура без засечек.   |
| Консоль луЦида | Гарнитура с засечками. |



 

**Значения размера**

Это размер шрифта в точках. Допустимо любое положительное целочисленное значение, однако рекомендуется использовать числа от 10 до 18. Размер меньше 10 может быть трудным для чтения, а размеры свыше 18 могут не оставлять достаточно места для показа нескольких букв за раз.

**Весовые значения**

В следующей таблице показаны только допустимые значения.



| Значение | Описание |
|-------|-------------|
| B     | Полужирный        |
| Нет     | Норм.      |
