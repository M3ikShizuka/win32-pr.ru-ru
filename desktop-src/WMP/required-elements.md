# <a name="required-elements"></a>Обязательные элементы

В файле определения обложки необходимо указать следующие элементы:

-   **Заголовок.** Требуется основной заголовок файла определения обложки. Сведения о версии заголовка см. в таблице в разделе [Создание файла определения обложки](creating-a-skin-definition-file.md) .
-   **Раздел описания.** раздел Description (описание) необходим при создании обложек для серии проигрыватель Windows Media 9 для Windows Mobile. Он должен быть первым разделом в файле определения обложки и должен указывать допустимые значения для измерений. Указывать значение параметра Orientation необязательно.
-   **Раздел точечных рисунков.** Необходимо указать раздел точечных рисунков. Кроме того, раздел растровых изображений должен указывать допустимые имена для фоновых, отключенных, отправленных, регионов и файлов Super Image.
-   **файлы изображений.** В рамках обложки необходимо предоставить фоновые, отключенные, отправленные, регионы и файлы суперпользователя. если вы создаете обложки для проигрыватель Windows Media 10 Mobile или более поздней версии, вам не нужно включать файлы регионов или Super image.

При создании обложки только с определенными изображениями Обложка будет видима, но не будет предоставлять пользователю никаких осмысленных функциональных возможностей. Если вы решили создать обложку без кнопок, возможно, чтобы предотвратить пропуску определенного содержимого пользователем, имейте в виду, что для аппаратных кнопок на устройстве все равно может быть возможно соответствие функциональных возможностей.

Файлы Thumb являются обязательными, и линейки не могут использоваться без бегунка.
