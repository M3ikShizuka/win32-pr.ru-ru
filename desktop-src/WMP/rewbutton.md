# <a name="rewbutton"></a>ревбуттон

Это предопределенная **кнопка** со следующими значениями по умолчанию.

``` syntax
onclick="jscript:player.controls.fastReverse()"
upToolTip="Fast Reverse"
cursor="system"
enabled="wmpenabled:player.controls.fastReverse"
```

## <a name="remarks"></a>Remarks

Все свойства этой **кнопки** можно переопределить, явно указав их.

## <a name="requirements"></a>Requirements (Требования)



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
