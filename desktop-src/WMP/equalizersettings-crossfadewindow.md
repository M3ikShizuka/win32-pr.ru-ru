# <a name="equalizersettingscrossfadewindow"></a>ЕКУАЛИЗЕРСЕТТИНГС. Кроссфадевиндов

Атрибут **кроссфадевиндов** указывает или получает объем перекрестной перекрытия в миллисекундах.

``` syntax
        elementID.crossFadeWindow
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**длинное**) в диапазоне от 0 до 10 000 со значением по умолчанию 250.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|---------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media 9 Series или более поздней версии<br/> |
