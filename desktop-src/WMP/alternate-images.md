# <a name="alternate-images"></a>Альтернативные образы

Альтернативные изображения — это изображения, которые не отображаются при первоначальном отображении обложки, но становятся видимыми в ответ на события мыши. Например, обложку можно настроить таким образом, что при наведении указателя мыши на определенную область картинка в этой области изменится. Это может информировать пользователей о том, что они могут щелкнуть область, чтобы сделать что-то. вы также можете использовать альтернативный образ, когда пользователь щелкнет область, чтобы предоставить им визуальный отзыв о том, что щелчок был получен с проигрыватель Windows Media.
