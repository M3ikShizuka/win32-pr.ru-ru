# <a name="submitting-your-visualization"></a>Отправка визуализации

Отправка визуализации с веб-сайта Вмплугинс больше не поддерживается.

## <a name="test-your-visualization"></a>Тестирование визуализации

Вы можете попробовать установить визуализацию на нескольких компьютерах, чтобы убедиться, что все работает так, как нужно. Используйте все предустановки, чтобы убедиться, что код работает правильно.

## <a name="sign-your-visualization"></a>Подпишите визуализацию

Необходимо отправить визуализацию для подписывания кода, чтобы пользователь мог решить, следует ли доверять визуализации перед установкой. Дополнительные сведения о подписывании кода см. [в разделе Введение в подписывание кода](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/ms537361(v=vs.85)).

## <a name="distributing-your-visualization"></a>Распространение визуализации

Хотя веб-сайт Вмплугинс больше не поддерживает распространение визуализации, вы по-прежнему можете распространять визуализацию с веб-сайта по своему усмотрению.
