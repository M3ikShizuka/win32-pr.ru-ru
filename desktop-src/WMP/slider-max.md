# <a name="slidermax"></a>SLIDER. Max

Атрибут **Max** указывает или получает максимальное значение диапазона, определенного элементом управления "ползунок".

``` syntax
        elementID.max
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**float**) и значением по умолчанию 100.

## <a name="remarks"></a>Комментарии

Значение, указанное для параметра **Max** , должно быть больше значения параметра **min**.

См. **кустомслидер**. атрибут [поситионимаже](customslider-positionimage.md) для примера, иллюстрирующий использование атрибутов элемента **Slider** .
