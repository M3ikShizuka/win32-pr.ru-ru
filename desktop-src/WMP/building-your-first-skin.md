# <a name="building-your-first-skin"></a>Создание первой обложки

Как более подробно описано в статье [об обложках](about-skins.md), Обложка определяется в файле определения обложки и требует наличия одного или нескольких вспомогательных графических файлов, файлов Microsoft JScript и цифровых файлов мультимедиа. В этом учебнике для очень простой обложки требуется только файл определения обложки и три графических файла. Выполните четыре шага, описанные в следующих разделах:

-   [Создание основного файла изображения](creating-the-primary-art-file.md)
-   [Создание файла сопоставления](creating-the-mapping-file.md)
-   [Создание изображения для наведения](creating-the-hover-image.md)
-   [Создание файла определения обложки](creating-the-skin-definition-file.md)
