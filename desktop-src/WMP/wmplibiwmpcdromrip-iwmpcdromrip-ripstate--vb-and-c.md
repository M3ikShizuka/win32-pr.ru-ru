# <a name="iwmpcdromripripstate-property"></a>Свойство Ивмпкдромрип:: Рипстате

Свойство **рипстате** возвращает значение перечисления, указывающее текущее состояние процесса копирования.

## <a name="syntax"></a>Синтаксис


```CSharp
public WMPRipState ripState {get; set;}
```


```VB

Public ReadOnly Property ripState As WMPRipState
```





## <a name="property-value"></a>Значение свойства

Объект **вмплиб. вмприпстате** , который является значением из перечисления **вмприпстате** , которое указывает текущее состояние.
