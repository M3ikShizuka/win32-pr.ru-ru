# <a name="handling-events"></a>Обработка событий

Кроме XML-кода, который вы пишете для инициализации атрибутов обложки, основной код, который вы пишете, является кодом JScript для управления событиями. События являются либо внешними, либо внутренними. то есть они поступают от пользователя или из проигрывателя Windows Media.

Следующие разделы содержат сведения об обработке событий.

-   [Внешние события](external-events.md)
-   [Внутренние события](internal-events.md)
-   [Написание кода события](writing-event-code.md)
-   [Вторичные события](secondary-events.md)
