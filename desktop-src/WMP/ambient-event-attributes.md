# <a name="ambient-event-attributes"></a>Атрибуты внешнего события

Следующие атрибуты являются общими для всех событий обложки. Доступ к ним осуществляется с помощью ключевого слова **event** в обработчике событий для элемента.



| attribute                              | Описание                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [алткэй](event-altkey.md)             | Получает значение, указывающее, была ли нажата клавиша ALT при возникновении события.                           |
| [переключатель](event-button.md)             | Извлекает значение, указывающее, какая кнопка мыши была нажата при возникновении события.                          |
| [клиенткс](event-clientx.md)           | Извлекает координату по оси x указателя мыши относительно области клиента окна приложения. |
| [Клиентская](event-clienty.md)           | Получает координату y указателя мыши относительно клиентской области окна приложения. |
| [ктрлкэй](event-ctrlkey.md)           | Получает значение, указывающее, была ли нажата клавиша CTRL при возникновении события.                          |
| [фромелемент](event-fromelement.md)   | Извлекает элемент, полученный от события.                                                                   |
| [keyCode](event-keycode.md)           | Извлекает код ключа ASCII, если при возникновении события была нажата клавиша.                                   |
| [оффсеткс](event-offsetx.md)           | Извлекает координату по оси x указателя мыши относительно элемента, вызывающего событие.                |
| [смещение](event-offsety.md)           | Получает координату y указателя мыши относительно элемента, вызывающего событие.                |
| [скринхеигхт](event-screenheight.md) | Получает высоту доступного размера экрана в пикселях.                                                 |
| [скринвидс](event-screenwidth.md)   | Получает ширину доступного размера экрана в пикселях.                                                  |
| [скринкс](event-screenx.md)           | Получает абсолютную координату по оси x указателя мыши относительно экрана.                         |
| [экран](event-screeny.md)           | Получает абсолютную координату y указателя мыши относительно экрана.                         |
| [шифткэй](event-shiftkey.md)         | Получает значение, указывающее, была ли нажата клавиша SHIFT при возникновении события.                         |
| [срцелемент](event-srcelement.md)     | Извлекает элемент, который вызвал событие.                                                                  |
| [тоелемент](event-toelement.md)       | Извлекает элемент, в который переместился фокус клавиатуры.                                                           |
| [x](event-x.md)                       | Извлекает координату указателя мыши по оси x в соответствии с окном приложения.                      |
| [y](event-y.md)                       | Получает координату y указателя мыши относительно окна приложения.                      |
