# <a name="image-sizes"></a>Размеры изображений

При создании иллюстраций для обложек необходимо учитывать размер отображаемой области. В следующей таблице показаны размеры областей отображения для различных конфигураций экрана.



| DPI (режим)                    | Ширина в пикселях | Высота в пикселях |
|-------------------------------|-----------------|------------------|
| 96 (Книжная)                 | 240             | 320              |
| 96 (альбомная)                | 320             | 240              |
| 192 (Книжная)                | 480             | 640              |
| 192 (альбомная)               | 640             | 220              |
| 96 (квадратный)                   | 240             | 240              |
| 192 (квадратный)                  | 480             | 480              |
| 96 (книжная для смартфонов)  | 176             | 220              |
| 131 (книжная для смартфонов) | 240             | 320              |



 

> [!Note]  
> версии, предшествующие проигрыватель Windows Media 9, для Windows Mobile 2003 поддерживают только обложки с книжным режимом 96. проигрыватель Windows Media 10 mobile или более поздняя версия поддерживает обложки 131 DPI в книжном режиме на устройствах смартфона, и эта поддержка должна продолжаться в будущих версиях проигрыватель Windows Media Mobile.
