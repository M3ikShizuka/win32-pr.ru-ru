# <a name="iwmpplaylistcollection-vb-and-c-interface"></a>Интерфейс Ивмпплайлистколлектион (VB и C#)

Предоставляет методы для управления интерфейсами **ивмпплайлист** и **ивмпплайлистаррай** в коллекции.

## <a name="members"></a>Элементы

Интерфейс **ивмпплайлистколлектион (VB и C#)** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмпплайлистколлектион (VB и C#)** содержит следующие методы.



| Метод                                                                                                 | Описание                                                                                                                    |
|:-------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------|
| [**жеталл**](wmplibiwmpplaylistcollection-iwmpplaylistcollection-getall--vb-and-c.md)                 | Возвращает интерфейс **ивмпплайлистаррай** , который предоставляет доступ ко всем спискам воспроизведения в библиотеке.<br/>             |
| [**жетбинаме**](wmplibiwmpplaylistcollection-iwmpplaylistcollection-getbyname--vb-and-c.md)           | Возвращает интерфейс **ивмпплайлистаррай** , предоставляющий доступ к спискам воспроизведения с указанным именем, если таковые имеются.<br/> |
| [**импортплайлист**](wmplibiwmpplaylistcollection-iwmpplaylistcollection-importplaylist--vb-and-c.md) | Добавляет статический список воспроизведения в библиотеку.<br/>                                                                              |
| [**isDeleted**](wmplibiwmpplaylistcollection-iwmpplaylistcollection-isdeleted--vb-and-c.md)           | Возвращает значение, указывающее, находится ли указанный список воспроизведения в папке удаленных элементов.<br/>                           |
| [**невплайлист**](wmplibiwmpplaylistcollection-iwmpplaylistcollection-newplaylist--vb-and-c.md)       | Возвращает интерфейс **ивмпплайлист** для нового пустого списка воспроизведения в библиотеке.<br/>                                     |
| [**отменит**](wmplibiwmpplaylistcollection-iwmpplaylistcollection-remove--vb-and-c.md)                 | Удаляет список воспроизведения из библиотеки.<br/>                                                                                |
| **сетделетед**                                                                                         | Больше не поддерживается.<br/>                                                                                                |



 

Получите интерфейс **ивмпплайлистколлектион** с помощью следующего свойства.



| Объект                                                                   | Свойство                                                                                 |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| [Объект Аксвиндовсмедиаплайер](axwindowsmediaplayer-object--vb-and-c.md) | [**плайлистколлектион**](axwmplib-axwindowsmediaplayer-playlistcollection--vb-and-c.md) |
