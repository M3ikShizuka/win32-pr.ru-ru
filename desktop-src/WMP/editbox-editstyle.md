# <a name="editboxeditstyle"></a>EDITBOX. Едитстиле

Атрибут **едитстиле** указывает или получает стиль элемента управления "поле ввода".

``` syntax
        elementID.editStyle
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей одно из следующих значений.



| Значение     | Описание                                                                     |
|-----------|---------------------------------------------------------------------------------|
| нормальный    | По умолчанию. Отображает нормальный текст в одной строке.                                 |
| password  | Отображает звездочки ( \* ) вместо текста. Может указываться только во время разработки. |
| верхний регистр | Отображает текст в верхнем регистре.                                                 |
| Нижний регистр | Отображает текст в нижнем регистре.                                                 |
| number    | Отображаются только числа.                                                          |
| нескольких | Отображает несколько строк текста. Может указываться только во время разработки.          |



 

## <a name="remarks"></a>Remarks

Для этого атрибута можно задать только значение "пароль" или "Многострочный" во время разработки. Если для него задано значение "Многострочный", то во время выполнения его нельзя изменить.
