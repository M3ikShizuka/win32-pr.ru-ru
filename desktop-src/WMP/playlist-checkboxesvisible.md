# <a name="playlistcheckboxesvisible"></a>Список воспроизведения. Чеккбоксесвисибле

Атрибут **чеккбоксесвисибле** указывает или получает значение, указывающее, видимы ли флажки.

``` syntax
        elementID.checkboxesVisible
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                           |
|-------|---------------------------------------|
| Да  | Флажки видимы.              |
| false | По умолчанию. Флажки не отображаются. |



 

## <a name="remarks"></a>Remarks

Флажки отображаются в крайнем левом столбце.

|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
