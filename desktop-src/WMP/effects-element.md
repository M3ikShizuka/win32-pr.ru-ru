# <a name="effects-element"></a>EFFECTs, элемент

Элемент **Effects** предоставляет способ организации и управления визуализациями с помощью следующих атрибутов и методов. Для удобства также предоставляется предопределенный элемент **Effects** .

Элемент **Effects** поддерживает следующие атрибуты.



| attribute                                                        | Описание                                                                                                                                                                                                                                          |
|------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [алловалл](effects-allowall.md)                                 | Указывает или получает значение, указывающее, включаются ли все визуализации в реестр.                                                                                                                                                 |
| [куррентеффект](effects-currenteffect.md)                       | Указывает или получает текущую визуализацию.                                                                                                                                                                                                    |
| [куррентеффектпресеткаунт](effects-currenteffectpresetcount.md) | Извлекает количество доступных предустановок для текущей визуализации.                                                                                                                                                                                 |
| [куррентеффекттитле](effects-currenteffecttitle.md)             | Извлекает отображаемое название текущей визуализации.                                                                                                                                                                                            |
| [куррентеффекттипе](effects-currenteffecttype.md)               | Возвращает имя реестра текущей визуализации.                                                                                                                                                                                            |
| [куррентпресет](effects-currentpreset.md)                       | Указывает или получает текущую предустановку текущей визуализации.                                                                                                                                                                              |
| [куррентпресеттитле](effects-currentpresettitle.md)             | Извлекает заголовок текущей предустановки текущей версии.                                                                                                                                                                              |
| [еффекткангофуллскрин](effects-effectcangofullscreen.md)       | Извлекает значение, указывающее, может ли текущая визуализация отображаться в полноэкранном режиме.                                                                                                                                                         |
| [еффекткаунт](effects-effectcount.md)                           | Возвращает число доступных визуализаций.                                                                                                                                                                                                    |
| [еффексаспропертипаже](effects-effecthaspropertypage.md)       | Получает значение, указывающее, содержит ли текущая визуализация страницу свойств.                                                                                                                                                                  |
| [fullScreen](effects-fullscreen.md)                             | Указывает или получает значение, указывающее, отображается ли текущая визуализация в полноэкранном режиме. Можно задать только во время выполнения.                                                                                                                   |
| [оконные](effects-windowed.md)                                 | Указывает или получает значение, указывающее, будет ли визуализация отображаться в окне или без окон, то есть независимо от того, будет ли весь прямоугольник элемента управления видимым в любое время или же он может быть обрезан. Можно задать только во время разработки. |



 

Элемент **Effects** поддерживает следующие методы.



| Метод                                       | Описание                                                                                                       |
|----------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [еффекттитле](effects-effecttitle.md)       | Возвращает понятное имя визуализации с указанным индексом реестра.                               |
| [еффекттипе](effects-effecttype.md)         | Возвращает имя реестра визуализации с указанным индексом реестра.                               |
| [далее](effects-next.md)                     | Отображает следующую предустановку визуализации, при необходимости переход к следующей визуализации.                            |
| [некстеффект](effects-nexteffect.md)         | Отображает первую предустановку следующего представления, пропуская промежуточные предустановки.                                |
| [некстпресет](effects-nextpreset.md)         | Отображает следующую предустановку текущей визуализации.                                                            |
| [previous](effects-previous.md)             | Отображает предыдущую предустановку визуализации, переходя к последней предустановке предыдущей предварительной версии, если это необходимо. |
| [превиаусеффект](effects-previouseffect.md) | Отображает предыдущую визуализацию, пропуская предустановки.                                                            |
| [превиауспресет](effects-previouspreset.md) | Отображает предыдущую предустановку текущей визуализации.                                                        |
| [параметры](effects-settings.md)             | Отображает страницу атрибута для текущей визуализации, если она есть.                                            |



 

Элемент **Effects** поддерживает атрибуты окружения и может реализовывать обработчики событий окружения. Дополнительные сведения см. в разделе [атрибуты окружения](ambient-attributes.md) и [обработчики событий окружения](ambient-event-handlers.md).

Стандартные эффекты — это элементы обычных **эффектов** с различными параметрами общих атрибутов, заданных по умолчанию. Доступны следующие предопределенные эффекты.



| Стандартные эффекты           | Описание                                                         |
|------------------------------|---------------------------------------------------------------------|
| [вмпеффектс](wmpeffects.md) | Элемент **Effects** , который выполняет итерацию по доступным эффектам. |
