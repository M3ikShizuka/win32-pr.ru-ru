# <a name="playlistallowcolumnsorting"></a>Список воспроизведения. Алловколумнсортинг

Атрибут **алловколумнсортинг** указывает или получает значение, указывающее, разрешено ли содержимое столбца сортировки.

``` syntax
        elementID.allowColumnSorting
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                                  |
|-------|----------------------------------------------|
| Да  | По умолчанию. Сортировка содержимого столбцов разрешена. |
| false | Сортировка содержимого столбцов не разрешена.      |



 

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
