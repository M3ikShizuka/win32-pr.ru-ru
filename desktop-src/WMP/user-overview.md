# <a name="user-overview"></a>Обзор пользователя

Для пользователя визуализации — это изображения, которые Dance время на музыку. Они предоставляют дополнительные развлечения для улучшения работы с музыкальными возможностями. Чем больше развлекательной визуализации, тем лучше!

Типичная визуализация отображается, как показано на следующем рисунке.

![Типичная визуализация](images/vizanim.gif)

## <a name="standard-visualizations"></a>Стандартные визуализации

Проигрыватель Windows Media включает несколько визуализаций, например:

-   Окружением
-   пленоптик
-   Пик

## <a name="standard-presets"></a>Стандартные предустановки

Кроме того, каждая визуализация поставляется с конкретными преднаборами, которые изменяют отображение визуализации. Например, Визуализация пик состоит из двух предустановок:

-   Пик
-   "Амеба"

Оба эффекта являются частью одной визуализации, но имеют разные предустановки. Предустановка — это набор значений для атрибутов конкретной визуализации.

## <a name="skin-attributes"></a>Атрибуты обложки

При создании обложек для проигрывателя Windows Media можно отобразить стандартные визуализации в обложках с помощью элемента **Effects** . Дополнительные сведения об использовании визуализаций в обложках см. в разделе [обложки проигрывателя Windows Media](windows-media-player-skins.md).
