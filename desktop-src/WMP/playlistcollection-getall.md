# <a name="playlistcollectiongetall-method"></a>Плайлистколлектион. Жеталл, метод

Метод **жеталл** извлекает объект **плайлистаррай** , содержащий все списки воспроизведения в библиотеке.

## <a name="syntax"></a>Синтаксис


```JScript
retVal = PlaylistCollection.getAll()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает объект **плайлистаррай** .

## <a name="remarks"></a>Remarks

Чтобы использовать этот метод, требуется доступ на чтение к библиотеке. Дополнительные сведения см. в разделе [доступ к библиотеке](library-access.md).
