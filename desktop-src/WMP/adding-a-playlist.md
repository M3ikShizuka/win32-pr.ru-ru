# <a name="adding-a-playlist"></a>Добавление списка воспроизведения

Списки воспроизведения можно использовать для выбора из коллекций аудио и видео.

Используя иллюстрацию из первой обложки, можно внести несколько изменений в файл определения обложки.

Первым шагом является использование оболочки, которая будет использоваться для большинства обложек:


```C++
<THEME>
    <VIEW
        clippingColor = "#CCCC00"
        backgroundImage = "background.bmp"
        titleBar = "false">

        <BUTTONGROUP
            mappingImage = "map.bmp"
            hoverImage = "hover.bmp"> 
                
    </VIEW>


</THEME>

```



Теперь добавьте второе **представление**, содержащее список воспроизведения. Поместите следующий код после </VIEW> кода оболочки.


```C++
     <VIEW 
          id = "playview">
          <PLAYLIST/>
     </VIEW>

```



Необходимо присвоить второму просмотру идентификатор, чтобы вы могли обращаться к нему позже. Добавьте ПЛАЙЕЛЕМЕНТ и СТОПЕЛЕМЕНТ. Эти стандартные кнопки упрощают жизнь.


```C++
        <PLAYELEMENT
            mappingColor = "#00FF00" />
                          
        <STOPELEMENT
            mappingColor = "#FF0000" />

```



Наконец, добавьте событие OnClick в ПЛАЙЕЛЕМЕНТ для отображения списка воспроизведения во втором представлении:


```C++
            onClick = "JScript: theme.openView('playview');" />

```



Аналогичные обложки рабочего списка можно просмотреть в разделе с примерами пакета SDK.
