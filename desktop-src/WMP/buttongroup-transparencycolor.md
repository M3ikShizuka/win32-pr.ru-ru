# <a name="buttongrouptransparencycolor"></a>BUTTONGROUP. Транспаренциколор

Атрибут **транспаренциколор** указывает или получает прозрачный цвет изображений **BUTTONGROUP** .

``` syntax
        elementID.transparencyColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи и не имеет значения по умолчанию, содержащей одно из следующих значений.



| Значение                                       | Описание                                                                                        |
|---------------------------------------------|----------------------------------------------------------------------------------------------------|
| Auto (Автоматически)                                        | Пиксель в положении 0, 0 в изображении превращается в прозрачный цвет.                              |
| любое значение цвета Microsoft Internet Explorer | Значение цвета Internet Explorer превращается в прозрачный цвет (например, "Red" или " \# FF0000"). |
| Нет                                        | По умолчанию. Прозрачность отсутствует.                                                                          |



 

## <a name="remarks"></a>Remarks

Прозрачный цвет в изображении позволяет отображать все, что находится за изображением, с помощью областей прозрачности. Прозрачная область может быть нажата, если не обрезаться тегом **клиппингимаже** .

Цвет может быть любым значением цвета Microsoft Internet Explorer. Если значение равно Auto, то используется цвет пикселя в положении 0, 0 в изображении.

Если указанный цвет не является одним из допустимых цветов Internet Explorer, то возвращается предупреждение, а предыдущее значение сохраняется.

Так как Жпгс являются утерянными и поэтому подвержены непредвиденному изменению цвета, они не рекомендуются при использовании **транспаренциколор** .
