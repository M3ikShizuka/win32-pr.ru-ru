# <a name="metafile-playlists"></a>Списки воспроизведения метафайлов

Списки воспроизведения метафайлов Windows Media используются для упорядочения потокового содержимого и представления. В списке воспроизведения можно просто перенаправить проигрыватель Windows Media в потоковую передачу содержимого, но чаще всего будет предоставлен список файлов для последовательного воспроизведения. Более сложный список воспроизведения может содержать повторы, коммерческие вставки и сведения о товарах в списке.

Другое преимущество списков воспроизведения заключается в том, что вместо воспроизведения потока, остановки, запуска следующего потока и последующего ожидания завершения буферизации службы Windows Media и проигрыватель Windows Media работают вместе, чтобы воспроизвести клипы один за другим с минимальным временем буферизации или прерыванием.
