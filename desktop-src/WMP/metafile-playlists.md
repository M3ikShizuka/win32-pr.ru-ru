# <a name="metafile-playlists"></a>Списки воспроизведения метафайлов

Windows Списки воспроизведения метафайлов мультимедиа используются для упорядочения потокового содержимого и представления. список воспроизведения можно просто перенаправить проигрыватель Windows Media к содержимому, которое будет передаваться в потоке, но более часто будет предоставлять список файлов для последовательного воспроизведения. Более сложный список воспроизведения может содержать повторы, коммерческие вставки и сведения о товарах в списке.

другое преимущество списков воспроизведения заключается в том, что вместо воспроизведения потока, остановки, запуска следующего потока и последующего ожидания завершения буферизации службы Windows Media и проигрыватель Windows Media работают вместе, чтобы воспроизвести клипы один за другим с минимальным временем буферизации или прерыванием.
