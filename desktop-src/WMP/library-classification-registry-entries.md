# <a name="library-classification-registry-entries"></a>Записи реестра классификации библиотеки

когда проигрыватель Windows Media обнаруживает файл мультимедиа с пользовательским расширением имени файла, он не знает, следует ли классифицировать файл как аудио, видео или какой-либо другой тип. по умолчанию проигрыватель Windows Media помещает такие файлы в другую часть носителя библиотеки.

если файлы цифрового мультимедиа имеют пользовательский формат, можно предоставить проигрыватель Windows Media с информацией о том, где файлы должны отображаться в библиотеке проигрывателя, разместив две записи в реестре на компьютере пользователя.

Одна запись помещается в следующий подраздел.

**HKEY \_ , \_ \\ программное обеспечение локального компьютера, \\ \\ расширения Microsoft MediaPlayer \\ MLS \\**

Запись реестра имеет следующий формат.



| Имя                                                  | Тип данных   | Значение                                      |
|-------------------------------------------------------|-------------|--------------------------------------------|
| Расширение имени файла без разделителя с точкой (.) | **REG \_ SZ** | Строка, указывающая расположение библиотеки |



 

Другая запись реестра находится в следующем созданном вами подразделе.

**HKey \_ \_Корневые \\** *кустомекстенсион* классов

где *кустомекстенсион* — это расширение имени файла, включая разделитель точек (.).

Запись реестра имеет следующий формат.



| Имя          | Тип данных   | Значение                                      |
|---------------|-------------|--------------------------------------------|
| PerceivedType | **REG \_ SZ** | Строка, указывающая расположение библиотеки |



 

Оба элемента реестра должны иметь одинаковое значение. Возможные значения приведены в следующей таблице.



| Значение | Описание                                                                      |
|-------|----------------------------------------------------------------------------------|
| звук | Файлы с пользовательским расширением отображаются в музыкальной части библиотеки. |
| видео | Файлы с пользовательским расширением отображаются в видеообласти библиотеки. |



 

Например, следующие записи реестра указывают, что файлы с расширением. XYZ будут отображаться в музыкальной части библиотеки:


```C++

HKEY_LOCAL_MACHINE\Software\Microsoft\MediaPlayer\MLS\Extensions
    xyz     REG_SZ     audio

HKEY_CLASSES_ROOT\.xyz
    PerceivedType     REG_SZ     audio

```



Обратите внимание, что любой код, пытающийся выполнить запись в реестр на компьютере пользователя, может выполнять запись в \_ поддерево hKey на локальном компьютере \_ только в том случае, если у текущего пользователя есть права администратора.

записи реестра классификации библиотеки поддерживаются следующими версиями проигрыватель Windows Media.

-   проигрыватель Windows Media 9 Series с исправлением 823275
-   проигрыватель Windows Media 10 и более поздних версий
