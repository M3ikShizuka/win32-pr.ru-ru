# <a name="network-object"></a>Сетевой объект

Объект **Network** предоставляет свойства и методы, используемые для доступа к статистике, связанной с качеством сетевого подключения, а также для указания и получения параметров прокси-сервера сети.

Объект **Network** поддерживает следующие свойства.



| Свойство                                           | Описание                                                                                 |
|----------------------------------------------------|---------------------------------------------------------------------------------------------|
| [Связи](network-bandwidth.md)                 | Извлекает текущую пропускную способность элемента мультимедиа.                                          |
| [Скорость](network-bitrate.md)                     | Извлекает текущую полученную скорость.                                              |
| [буфферингкаунт](network-bufferingcount.md)       | Извлекает количество попыток буферизации во время воспроизведения.                           |
| [буфферингпрогресс](network-bufferingprogress.md) | Возвращает процент завершенной буферизации.                                            |
| [буфферингтиме](network-bufferingtime.md)         | Указывает или получает количество времени буферизации в миллисекундах перед началом воспроизведения. |
| [довнлоадпрогресс](network-downloadprogress.md)   | Возвращает процент завершенной загрузки.                                             |
| [енкодедфрамерате](network-encodedframerate.md)   | Извлекает частоту кадров видео, указанную автором содержимого.                             |
| [Частота](network-framerate.md)                 | Извлекает текущую частоту кадров видео.                                                     |
| [фрамесскиппед](network-framesskipped.md)         | Возвращает общее число кадров, пропущенных во время воспроизведения.                               |
| [лостпаккетс](network-lostpackets.md)             | Возвращает число потерянных пакетов.                                                       |
| [maxBandwidth](network-maxbandwidth.md)           | Указывает или получает максимально допустимую пропускную способность.                                       |
| [максбитрате](network-maxbitrate.md)               | Возвращает максимальную возможную скорость видео.                                              |
| [рецеиведпаккетс](network-receivedpackets.md)     | Возвращает число полученных пакетов.                                                   |
| [рецептионкуалити](network-receptionquality.md)   | Возвращает процент полученных пакетов за последние 30 секунд.                        |
| [рековередпаккетс](network-recoveredpackets.md)   | Возвращает количество восстановленных пакетов.                                                  |
| [саурцепротокол](network-sourceprotocol.md)       | Возвращает исходный протокол, используемый для получения данных.                                         |



 

Объект **Network** поддерживает следующие методы.



| Метод                                                       | Описание                                                                                                          |
|--------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [жетпроксибипассфорлокал](network-getproxybypassforlocal.md) | Извлекает значение, указывающее, должен ли прокси-сервер пропускать, если сервер-источник находится в локальной сети. |
| [жетпроксексцептионлист](network-getproxyexceptionlist.md)   | Извлекает список исключений прокси-сервера.                                                                                  |
| [жетпроксинаме](network-getproxyname.md)                     | Возвращает имя используемого прокси-сервера.                                                                         |
| [жетпроксипорт](network-getproxyport.md)                     | Извлекает порт прокси-сервера для использования.                                                                                     |
| [жетпроксисеттингс](network-getproxysettings.md)             | Получает параметр прокси-сервера для данного протокола.                                                                    |
| [сетпроксибипассфорлокал](network-setproxybypassforlocal.md) | Указывает значение, указывающее, должен ли прокси-сервер обходиться, если исходный сервер находится в локальной сети. |
| [сетпроксексцептионлист](network-setproxyexceptionlist.md)   | Указывает список исключений прокси-сервера.                                                                                  |
| [сетпроксинаме](network-setproxyname.md)                     | Указывает имя прокси-сервера для использования.                                                                         |
| [сетпроксипорт](network-setproxyport.md)                     | Указывает используемый порт прокси-сервера.                                                                                     |
| [setProxySettings](network-setproxysettings.md)             | Указывает параметр прокси-сервера для данного протокола.                                                                    |



 

Доступ к объекту **сети** осуществляется через следующее свойство.



| Объект                      | Свойство                      |
|-----------------------------|-------------------------------|
| [Игрок](player-object.md) | [network](player-network.md) |
