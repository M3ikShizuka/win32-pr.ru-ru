# <a name="downloaditem-object"></a>Объект Довнлоадитем

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Объект **довнлоадитем** представляет запрос на скачивание файла. он может использоваться веб-страницами, размещенными в полном режиме проигрыватель Windows Media и имеющих доступ к **внешнему** объекту, например к службам premium.

Объект **довнлоадитем** поддерживает следующие свойства:



| Свойство                                        | Описание                                      |
|-------------------------------------------------|--------------------------------------------------|
| [довнлоадстате](downloaditem-downloadstate.md) | Получает состояние загрузки.             |
| [двигается](downloaditem-progress.md)           | Получает сведения о ходе загрузки в байтах. |
| [size](downloaditem-size.md)                   | Возвращает размер загружаемого файла.              |
| [саурцеурл](downloaditem-sourceurl.md)         | Извлекает исходный URL-адрес для скачивания.        |
| [type](downloaditem-type.md)                   | Возвращает тип загрузки.              |



 

Объект **довнлоадитем** поддерживает следующие методы.



| Метод                                      | Описание                                                |
|---------------------------------------------|------------------------------------------------------------|
| [cancel](downloaditem-cancel.md)           | Отменяет скачивание.                                      |
| [getItemInfo](downloaditem-getiteminfo.md) | Возвращает значение атрибута для элемента загрузки. |
| [pause](downloaditem-pause.md)             | Приостанавливает скачивание.                                       |
| [выход](downloaditem-resume.md)           | Возобновляет скачивание.                                      |



 

Доступ к объекту **довнлоадитем** осуществляется через следующее свойство.



| Объект                                              | Свойство                            |
|-----------------------------------------------------|-------------------------------------|
| [довнлоадколлектион](downloadcollection-object.md) | [Item](downloadcollection-item.md) |



 

В целях иллюстрации **довнлоадманажер**. **жетдовнлоадколлектион**(*CollectionID*). **элемент**(*ItemId*) используется в разделах синтаксиса Reference.
