# <a name="album-info-integration"></a>Интеграция сведений о альбоме

проигрыватель Windows Media позволяет пользователям запрашивать подробные сведения о компакт-диске или DVD-диске, с которого поступило мультимедийное содержимое, нажав кнопку, например, **дополнительные сведения**. когда пользователь нажимает кнопку, проигрыватель Windows Media 10 или более поздние вызовы в текущем музыкальном магазине для просмотра сведений об альбоме. В этом случае проигрыватель открывает панель сведения о альбоме и отображает веб-страницу, указанную атрибутом **URL-адреса** элемента **албуминфо** документа serviceInfo.

проигрыватель Windows Media добавляет строку запроса к URL-адресу, чтобы предоставить сведения о содержимом, запрошенном пользователем, в интернет-магазине. Строка запроса содержит такие атрибуты, как **Title**, **альбом** и **исполнитель**, которые Интернет-магазин может использовать для определения правильного отображаемого альбома. Справочная документация по элементу **албуминфо** содержит полный список атрибутов строки запроса и их описания.

## <a name="guidelines-for-album-info"></a>Рекомендации по использованию сведений об альбоме

При создании веб-страницы сведений о диске используйте следующие рекомендации.

-   Страница должна четко определять Интернет-магазин, предоставляющий информацию. Это можно сделать, например, с помощью наглядного отображения логотипа.
-   Страница должна содержать ссылку на заявление о конфиденциальности вашей компании.
-   По возможности избегайте перехода по страницам в компоненте «сведения о диске». Для большинства действий следует переходить к вашему Интернет-магазину.
-   Рекомендуется предоставлять ценную информацию, не требуя от пользователя устанавливать программы или входить в Интернет-магазин.
