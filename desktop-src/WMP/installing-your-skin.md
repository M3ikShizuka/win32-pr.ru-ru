# <a name="installing-your-skin"></a>Установка обложки

Для установки обложки используйте следующую процедуру.

1.  Скопируйте все файлы обложки в новую папку.
2.  Скопируйте папку, содержащую файлы обложки, в папку проигрывателя Windows Media на переносном устройстве.

    Запустите проигрыватель Windows Media Mobile. Запустится средство **выбора обложки**, которое содержит две кнопки навигации, позволяющие прокручивать доступные обложки и отображать их в предварительной версии. Имя, отображаемое для обложки, будет именем файла определения обложки.

3.  Нажмите кнопку **ОК** , чтобы выбрать обложку.
