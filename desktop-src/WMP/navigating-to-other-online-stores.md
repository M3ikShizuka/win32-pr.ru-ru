# <a name="navigating-to-other-online-stores"></a>Переход к другим Интернет-магазинам

Вы можете создавать ссылки на другие Интернет-магазины, которыми вы владеете или перекрестными ссылками на Интернет-магазины, предоставляемые другими пользователями. Для этого необходимо знать имя ключа для Интернет-магазина, к которому необходимо выполнить переход.

В следующем примере кода показан HTML-код, который создает ссылку для переключения из Интернет-магазина Proseware в функцию "ServiceTask2" в магазине Саусридже Video online.


```C++
<A HREF = 
"javascript:window.external.NavigateTaskPaneURL('SouthridgeVideo', 'ServiceTask2', 'From=Proseware&To=2')"
>Switch to Southridge Video</A>

```



когда пользователь щелкнет ссылку, проигрыватель Windows Media предлагает пользователю перейти в магазин видео саусридже. Если пользователь разрешает эту задачу, проигрыватель переключает хранилища и переходит в указанную область задач, добавляя параметры строки запроса. Параметры строки запроса, приведенные в предыдущем примере, являются пользовательскими параметрами. Это означает, что Интернет-магазин, на который вы переключиться, должен рассчитывать эти значения и их обработку. Ваш Интернет-магазин (а также любое хранилище, к которому вы переключается) может использовать параметры строки запроса любым способом.
