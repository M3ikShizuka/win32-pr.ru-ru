# <a name="echo-sample-overview"></a>Обзор образца эхо

В этом руководством создается подключаемый модуль DSP для проигрывателя Windows Media, который создает эхо в звуковой записи PCM во время воспроизведения. Ниже приведены цели для подключаемого модуля.

-   Подключаемый модуль обрабатывает 8-разрядный или 16-разрядный звук PCM.
-   Он поддерживает время задержки от 10 миллисекунд (МС) до 2000 мс (2 секунды). Это представляет практический диапазон для большинства приложений.
-   Он поддерживает смешивание исходного сигнала с помощью сигнала задержки.
-   Он предоставляет реализацию страницы свойств, которая позволяет пользователю указать значение времени задержки и значение для процентного сигнала задержки относительно общего уровня звукового сигнала.
-   Код создается путем изменения примера подключаемого модуля "аудио-подключение" мастера подключаемых модулей для проигрывателя Windows Media.

Образец echo не включен в пакет SDK проигрывателя Windows Media. Это пример, который вы создаете. Чтобы создать образец Echo, необходимо начать работу с проектом по умолчанию из мастера подключаемых модулей проигрывателя Windows Media. Можно присвоить проекту имя, которое вам нравится. в документации предполагается, что проект называется Echo. Дополнительные сведения об использовании мастера см. [в разделе Создание подключаемого модуля DSP](building-a-dsp-plug-in.md).
