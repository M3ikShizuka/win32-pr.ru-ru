# <a name="tracknametext"></a>траккнаметекст

Это Стандартный **текстовый** элемент со следующими значениями по умолчанию.

``` syntax
value="wmpprop:player.currentMedia.name"
tabstop="true"
```

## <a name="remarks"></a>Remarks

При этом будет создан **текстовый** элемент, в котором будет отображаться имя текущего носителя. Все свойства этого **текстового** элемента можно переопределить, явно указав их.

## <a name="requirements"></a>Requirements (Требования)



| Требование | Значение |
|--------------------|----------------------------------------------|
| Версия<br/> | проигрыватель Windows Media 7,0 или более поздней версии<br/> |
