# <a name="about-the-mediacollection-and-media-objects"></a>Сведения об объектах Медиаколлектион и Media

Объекты **медиаколлектион** и **Media** управляют коллекцией носителей, которая определяет расположение цифровых файлов мультимедиа, к которым имеет доступ проигрыватель Windows Media. Вы получаете объект **медиаколлектион** из свойства **медиаколлектион** объекта **Player** . Свойство **медиаколлектион** возвращает объект **медиаколлектион** . Доступ к свойствам объекта **медиаколлектион** можно получить только после его создания. Например, чтобы добавить объект **мультимедиа** (песню), используйте следующий код:


```C++
player.mediacollection.add('laure.wma');

```



Вы добавили файл Лауре. wma в коллекцию носителей.

Текущий объект **мультимедиа** можно получить с помощью свойства **куррентмедиа** **проигрывателя**. Например, этот код получает свойство **Duration** текущего объекта **мультимедиа** :


```C++
myduration = player.currentmedia.duration;

```



Существует много способов получить объект **мультимедиа** , чтобы получить доступ к его свойствам. Например, если требуется получить доступ к свойству **Duration** текущего носителя, можно использовать каждую из следующих строк кода.

Чтобы получить время воспроизведения мультимедиа, сделайте следующее:


```C++
player.currentMedia.duration;

```



Чтобы получить длительность текущего носителя в списке воспроизведения, сделайте следующее:


```C++
player.controls.currentItem.duration;

```



Чтобы получить длительность третьего элемента мультимедиа в списке воспроизведения, сделайте следующее:


```C++
player.currentPlaylist.item(2).duration;

```



Чтобы получить длительность третьего элемента мультимедиа в жанре джаз, сделайте следующее:


```C++
player.mediaCollection.getByGenre("jazz").item(2).duration;

```



Чтобы получить длительность третьего элемента мультимедиа во втором списке воспроизведения, сделайте следующее:


```C++
player.playlistCollection.getAll.item(1).item(2).duration; 
```
