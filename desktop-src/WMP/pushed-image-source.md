# <a name="pushed-image-source"></a>Источник отправленного изображения

Необходимо определить источник изображения, которое должно отображаться при принудительной отправке пользователем кнопки. Образ берется из файла, который вы определили как отправленный в разделе "точечные рисунки". Необходимо ввести тип образа, за которым следует пробел, символ @ и еще один пробел. Затем необходимо ввести два положительных целых числа, определяющие верхнюю и левую координаты (в пикселях) изображения, которое нужно использовать внутри файла изображения. Расположение, в котором будет отображаться изображение, берется из координат, определенных для кнопки относительно фонового изображения; но расположение, из которого оно берется, определяется двумя числами, указанными в параметре @, и отсчитывается от отправленного изображения, из которого считывается изображение.

Например, чтобы использовать изображение из отправленного изображения, нарасположенного в отправленном файле, левый верхний край которого равен 50, 60 следует использовать следующий код:


```C++
Pushed @ 50,60

```



Если вы не хотите отображать другое изображение, можно определить фоновый рисунок в качестве растрового изображения. Для этого укажите фоновый файл в качестве отправленного файла и укажите смещение в левом верхнем углу кнопки:


```C++
Pushed @ 50,60

```



В этом случае ни одна из кнопок не сможет отобразить отправленное изображение. Рекомендуется отображать отправленное изображение для всех кнопок, чтобы дать пользователю наглядное представление, так как не всегда ясно, дает ли касание результат, особенно при воспроизведении музыки или при отключении звука касания.
