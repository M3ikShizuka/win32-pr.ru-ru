# <a name="iwmplibraryname-property"></a>Свойство Ивмплибрари:: Name

Свойство **Name** Возвращает отображаемое имя текущей библиотеки.

## <a name="syntax"></a>Синтаксис


```CSharp
public System.String name {get; set;}
```


```VB

Public ReadOnly Property name As System.String
```





## <a name="property-value"></a>Значение свойства

**Строка System. String** , которая является именем текущей библиотеки.
