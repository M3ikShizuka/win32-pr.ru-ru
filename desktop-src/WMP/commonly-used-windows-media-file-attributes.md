# <a name="commonly-used-windows-media-file-attributes"></a>Часто используемые атрибуты файлов Windows Media

Файлы Windows Media могут содержать любое количество атрибутов, включая определяемые пользователем атрибуты.

Если файл был добавлен в библиотеку, также могут быть доступны дополнительные атрибуты, связанные с типом носителя. Дополнительные атрибуты см. [в разделе](video-item-attributes.md) [звуковые элементы](audio-item-attributes.md) или видеозаписи.

Дополнительные сведения об атрибутах файлов Windows Media см. в статье [рекомендации по использованию метаданных Windows Media](/previous-versions/ms867702(v=msdn.10)).

В следующем списке содержатся часто используемые атрибуты.

-   [Атрибут Author](author-attribute.md)
-   [Атрибут Title](title-attribute.md)
-   [Атрибут WM/Албумартист](wm-albumartist-attribute.md)
-   [Атрибут WM/Контентдистрибутор](wm-contentdistributor-attribute.md)
-   [Атрибут WM/жанра](wm-genre-attribute.md)
-   [Атрибут WM/МКДИ](wm-mcdi-attribute.md)
-   [Атрибут WM/Медиакласспримарид](wm-mediaclassprimaryid-attribute.md)
-   [Атрибут WM/Медиакласссекондарид](wm-mediaclasssecondaryid-attribute.md)
-   [Атрибут WM/provider](wm-provider-attribute.md)
-   [Атрибут WM/Траккнумбер](wm-tracknumber-attribute.md)
