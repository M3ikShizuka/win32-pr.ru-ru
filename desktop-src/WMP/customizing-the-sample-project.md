# <a name="customizing-the-sample-project"></a>Настройка примера проекта

При создании собственного Интернет-магазина вам потребуется изменить реализации следующих методов в файле с именем Йоурпрожект. cpp:

-   **Циаурпрожект:: алловплай**. Используйте эту функцию, чтобы применить бизнес-правила для воспроизведения защищенного содержимого.
-   **Циаурпрожект:: Allow кдбурн**. Эта функция используется для применения бизнес-правил, позволяющих пользователям копировать защищенное содержимое на компакт-диск.
-   **Циаурпрожект:: алловпдатрансфер**. Эта функция используется для применения бизнес-правил, позволяющих пользователям передавать защищенное содержимое на портативное устройство.
-   **Циаурпрожект:: стартбаккграундпроцессинг**. Используйте эту функцию, чтобы инициировать все необходимые задачи фоновой обработки. Например, вы можете использовать эту возможность для проверки лицензий с истекшим сроком действия.
-   **Циаурпрожект::D евицеаваилабле**. Эта функция предназначена для запуска любых задач, связанных с подключенным устройством.
-   **Циаурпрожект::P репарефорсинк**. Эта функция используется для выполнения необходимых задач непосредственно перед синхронизацией цифровых носителей с устройством.
-   **Циаурпрожект:: сервицеевент**. Эта функция используется для начала и завершения задач, которые необходимо выполнить, когда Интернет-магазин активен.
-   **Циаурпрожект:: стопбаккграундпроцессинг**. Используйте эту функцию, чтобы отключить все задачи фоновой обработки, запущенные в проигрывателе Windows Media с именем **циаурпрожект:: стартбаккграундпроцессинг**.

## <a name="working-with-media-and-playlist-objects"></a>Работа с объектами мультимедиа и списками воспроизведения

Метод **алловплай** предоставляет указатель на интерфейс **ивмпмедиа** в качестве параметра. Этот интерфейс является интерфейсом проигрывателя Windows Media, представляющим мультимедийные объекты. Вызывая методы для этого интерфейса, можно работать с атрибутами и свойствами отдельного элемента мультимедиа.

Методы **алловкдбурн** и **алловпдатрансфер** предоставляют указатель на интерфейс **ивмпплайлист** в качестве параметра. Этот интерфейс является интерфейсом проигрывателя Windows Media, представляющим объекты списков воспроизведения. Вызывая методы этого интерфейса, можно работать с атрибутами и свойствами списка воспроизведения, добавлять элементы в список воспроизведения или удалять элементы из списка воспроизведения.

Сведения о том, как удалить элемент из списка воспроизведения программным способом, см. в разделе Реализация **калловбаседиалог <T> :: онремовемедиафромплайлист**. Дополнительные сведения о работе с объектами мультимедиа и списками воспроизведения см. в разделе [объектная модель проигрывателя для языков сценариев](player-object-model-for-scripting-languages.md).

## <a name="code-that-can-be-removed"></a>Код, который можно удалить

Вам, вероятно, потребуется удалить код, который открывает диалоговые окна, так как маловероятно, что пользователи будут выбирать, какие элементы мультимедиа можно воспроизвести или скопировать. В Йоурпрожект. h удалите следующий код:

-   Объявление **калловбаседиалог**.
-   Объявление **калловбурндиалог**.
-   Объявление **калловтрансфердиалог**.

Из Йоурпрожект. cpp удалите следующий код:

-   Реализация **калловбаседиалог <T> :: онинитдиалог**.
-   Реализация **калловбаседиалог <T> :: онремовемедиафромплайлист**.
