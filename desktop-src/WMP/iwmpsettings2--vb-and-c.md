# <a name="iwmpsettings2-vb-and-c-interface"></a>Интерфейс IWMPSettings2 (VB и C#)

Предоставляет свойства и метод, дополняющий интерфейс **ивмпсеттингс** .

В дополнение к свойствам и методам, унаследованным от **ивмпсеттингс**, интерфейс **IWMPSettings2** предоставляет следующие свойства.

## <a name="members"></a>Элементы

Интерфейс **IWMPSettings2 (VB и C#)** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **IWMPSettings2 (VB и C#)** содержит следующие методы.



| Метод                                                                                                   | Описание                                                     |
|:---------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------|
| [**рекуестмедиаакцессригхтс**](wmplibiwmpsettings2-iwmpsettings2-requestmediaaccessrights--vb-and-c.md) | Запрашивает указанный уровень доступа к библиотеке.<br/> |



 

### <a name="properties"></a>Свойства

Интерфейс **IWMPSettings2 (VB и C#)** имеет следующие свойства.



| Свойство                                                                                                    | Тип доступа          | Описание                                                                               |
|:------------------------------------------------------------------------------------------------------------|:---------------------|:------------------------------------------------------------------------------------------|
| [**дефаултаудиолангуаже**](wmplibiwmpsettings2-iwmpsettings2-defaultaudiolanguage--vb-and-c.md)<br/> | Только для чтения<br/> | Возвращает код локали (LCID) для языка звука по умолчанию. <br/>              |
| [**медиаакцессригхтс**](wmplibiwmpsettings2-iwmpsettings2-mediaaccessrights--vb-and-c.md)<br/>       | Только для чтения<br/> | Возвращает значение, указывающее разрешения, предоставленные в настоящее время для доступа к библиотеке. <br/> |



 

Получите интерфейс **IWMPSettings2** путем приведения значения, возвращаемого свойством **аксвиндовсмедиаплайер. Settings** .



| Объект                                                                   | Свойство                                                             |
|--------------------------------------------------------------------------|----------------------------------------------------------------------|
| [Объект Аксвиндовсмедиаплайер](axwindowsmediaplayer-object--vb-and-c.md) | [**Параметры**](axwmplib-axwindowsmediaplayer-settings--vb-and-c.md) |
