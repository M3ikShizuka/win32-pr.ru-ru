# <a name="resizing-the-license-acquisition-dialog-box"></a>Изменение размера диалогового окна "приобретение лицензий"

Можно добавить параметры строки запроса к URL-адресу, указанному для атрибута **DRM \_ лиценсеаккурл** , чтобы указать размер для диалогового окна Windows Media Player 10 или более поздней версии. В следующей таблице перечислены параметры.



| Параметр | Описание                                        |
|-----------|----------------------------------------------------|
| *длгкс*    | Задает ширину диалогового окна в пикселях.  |
| *длги*    | Задает высоту диалогового окна в пикселях. |



 

Например, следующий URL-адрес приобретения лицензии приведет к тому, что диалоговое окно "получение лицензии" будет открываться с размером 800 на 600 пикселей:


```C++
https://www.proseware.com/license/lic_acq.htm?DlgX=800&DlgY=600

```



Максимальный размер диалогового окна никогда не превысит размеры текущего экрана минус 20 пикселей. Минимальный размер — 333 x 210 пикселей (размер по умолчанию).

Для этого компонента требуется проигрыватель Windows Media 10 или более поздней версии.
