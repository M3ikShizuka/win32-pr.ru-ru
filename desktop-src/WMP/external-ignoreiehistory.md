# <a name="externalignoreiehistory"></a>External. Игнореиехистори

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

свойство **игнореиехистори** указывает, должен ли проигрыватель Windows Media игнорировать журнал Internet Explorer.

``` syntax
window.external.ignoreIEHistory
      
```

## <a name="possible-values"></a>Возможные значения

Это свойство является **логическим значением** только для записи. **значение TRUE** указывает, что проигрыватель Windows Media должен игнорировать журнал.
