# <a name="friendlyname-element"></a>FriendlyName, элемент

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

элемент **FriendlyName** указывает текстовую строку, которую проигрыватель Windows Media отобразить пользователю для указания интернет-магазина.

``` syntax
<FriendlyName>
    text string
</FriendlyName>
```

## <a name="attributes"></a>Атрибуты

Этот элемент не содержит атрибуты.

## <a name="parentchild-elements"></a>Родительские и дочерние элементы



| Иерархия       | Элемент         |
|-----------------|-----------------|
| Родительские элементы | **ServiceInfo** |
| Дочерние элементы  | Нет            |
