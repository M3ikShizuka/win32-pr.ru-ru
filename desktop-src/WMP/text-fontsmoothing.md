# <a name="textfontsmoothing"></a>TEXT. Фонтсмусинг

Атрибут **фонтсмусинг** указывает или получает значение, указывающее, включено ли сглаживание шрифтов.

``` syntax
        elementID.fontSmoothing
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                          |
|-------|--------------------------------------|
| Да  | Сглаживание шрифтов включено.           |
| false | По умолчанию. Сглаживание шрифтов отключено. |



 

## <a name="remarks"></a>Комментарии

Сглаживание шрифтов использует функцию сглаживания операционной системы. если сглаживание шрифтов включено и операционная система поддерживает сглаживание, проигрыватель Windows Media пытается сгладить текст. Не все шрифты поддерживают сглаживание. проигрыватель Windows Media 10 использует метод сглаживания ClearType в Windows XP.

-   **Предупреждение об ошибке** Сглаживание шрифта на прозрачном цвете может привести к тому, что прозрачный цвет будет смешиваться с символами. Не рекомендуется использовать **фонтсмусинг** , если текст будет отображаться по прозрачности.

См. атрибут [value](text-value.md) для примера, иллюстрирующий использование атрибутов **текстового** элемента.
