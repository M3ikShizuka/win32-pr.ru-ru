# <a name="textdisabledforegroundcolor"></a>TEXT. Дисабледфореграундколор

Атрибут **дисабледфореграундколор** указывает или получает цвет текста, используемый для элемента управления "текст", когда он отключен.

``` syntax
        elementID.disabledForegroundColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей любое значение цвета Microsoft Internet Explorer.

## <a name="remarks"></a>Remarks

Если **дисабледфореграундколор** не указан, используется **foregroundColor** .

См. атрибут [value](text-value.md) для примера, иллюстрирующий использование атрибутов **текстового** элемента.
