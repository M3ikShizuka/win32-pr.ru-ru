# <a name="art-file-formats"></a>Форматы файлов изображений

Проигрыватель Windows Media распознает следующие форматы графических файлов для обложек:



| Формат                                  | Расширение   | Описание                                                                                        |
|-----------------------------------------|-------------|----------------------------------------------------------------------------------------------------|
| Bitmap                                  | BMP        | Рекомендуется использовать битовые изображения, так как они обеспечивают наибольшее Управление точным изображением и цветами. |
| GIF       | .gif        | Формат сжатого изображения, используемый для веб-страниц. Поддерживаются анимированные GIF.                            |
| JPEG | .jpeg, .jpg | Формат сжатого изображения, используемый для веб-страниц.                                                         |
| PNG         | .png        | Формат сжатого изображения, используемый для веб-страниц.                                                         |



 

При использовании одного из сжатых форматов файлов, определяющих цвет как прозрачный для веб-браузера, не определяйте цвет как прозрачный в файле изображения. Используйте видимый цвет для представления прозрачных областей изображения, а затем определите этот цвет как прозрачный в файле определения обложки. Например, если вы создаете GIF-файл с прозрачными областями, он не будет прозрачным в окончательном образе, и вы не сможете использовать цвет, заданный в качестве прозрачного в файле GIF, для цвета прозрачности в обложке.
