# <a name="windows-media-metafiles-overview"></a>Windows Общие сведения о метафайлах мультимедиа

наиболее важной частью успешного использования Windows метафайлов мультимедиа является правильный синтаксис для элементов метафайлов. синтаксические ошибки в Windowsном метафайле мультимедиа могут привести к невозможности поиска одного атрибута, если метафайл не распознается как допустимый и не работает.

почти так же важно — порядок, в котором элементы отображаются в метафайле мультимедиа Windows. Атрибуты некоторых элементов временно переопределяют атрибуты похожих элементов в разных разделах метафайла. Существует определенный [Порядок приоритета](order-of-precedence.md).

Windows списки воспроизведения метафайлов мультимедиа — это Windows метафайлы мультимедиа, которые предоставляют сведения, которые проигрыватель Windows Media используют для получения одноадресных потоков, многоадресных потоков и других поддерживаемых носителей из интрасети или интернета. Список воспроизведения метафайлов, по сути, является ярлыком для мультимедийного содержимого. Список воспроизведения метафайлов можно отправить в виде сообщения электронной почты, который используется в качестве ссылки на веб-страницу, созданной динамически с помощью Active Server страниц (ASP) или существовать как отдельный файл на локальном жестком диске. список воспроизведения метафайлов может ссылаться на другой список воспроизведения метафайлов, страницу ASP или файл Windows Media station (с расширением имени файла nsc). файл с расширением nsc используется для определения проигрыватель Windows Mediaной станции Windows. Базовый процесс обработки одинаков для каждого случая.
