# <a name="windows-media-metafiles-overview"></a>Общие сведения о метафайлах Windows Media

Наиболее важной частью успешного использования метафайлов Windows Media является правильный синтаксис для элементов метафайлов. Синтаксические ошибки в метафайле Windows Media могут привести к тому, что содержимое одного атрибута будет неизвестным, а метафайл не будет распознан как допустимый и не будет работать.

Почти так же важно — порядок, в котором элементы отображаются в метафайле Windows Media. Атрибуты некоторых элементов временно переопределяют атрибуты похожих элементов в разных разделах метафайла. Существует определенный [Порядок приоритета](order-of-precedence.md).

Списки воспроизведения метафайлов Windows Media представляют собой метафайлы Windows Media, которые предоставляют сведения, используемые проигрывателем Windows Media для получения одноадресных потоков, многоадресных потоков и других поддерживаемых носителей из интрасети или Интернета. Список воспроизведения метафайлов, по сути, является ярлыком для мультимедийного содержимого. Список воспроизведения метафайлов можно отправить в виде сообщения электронной почты, который используется в качестве ссылки на веб-страницу, созданной динамически с помощью Active Server страниц (ASP) или существовать как отдельный файл на локальном жестком диске. Список воспроизведения метафайлов может ссылаться на другой список воспроизведения метафайлов, страницу ASP или файл станции Windows Media (с расширением имени файла. NSC). NSC-файл используется для определения станции Windows Media в проигрывателе Windows Media. Базовый процесс обработки одинаков для каждого случая.
