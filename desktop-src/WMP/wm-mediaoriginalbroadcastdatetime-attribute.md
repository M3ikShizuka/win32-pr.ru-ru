# <a name="wmmediaoriginalbroadcastdatetime-attribute"></a>Атрибут WM/Медиаоригиналброадкастдатетиме

Атрибут **WM/медиаоригиналброадкастдатетиме** указывает дату и время первого вещания данного показа.

## <a name="applies-to"></a>Применение

-   [Элементы видео](video-item-attributes.md)

## <a name="remarks"></a>Комментарии

Константа Windows Media Format SDK для этого атрибута — g \_ всзвммедиаоригиналброадкастдатетиме.

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .
