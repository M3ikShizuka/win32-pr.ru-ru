# <a name="enumerations-for-type-1-online-stores"></a>Перечисления для Интернет-магазинов типа 1

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Для Интернет-магазинов типа 1, использующих код C++, доступны следующие типы перечисления.



| Перечисление                                            | Описание                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [вмпаккаунттипе](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpaccounttype)                   | Перечисление, определяющее типы счетов для Интернет-магазина.                                              |
| [вмпкаллбаккнотификатион](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpcallbacknotification) | Перечисление, определяющее состояния, используемые функцией обратного вызова **ивмпконтентпартнеркаллбакк:: notify** . |
| [вмппартнернотификатион](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmppartnernotification)   | Перечисление, определяющее операционные состояния Интернет-магазина.                                          |
| [вмпстреамингтипе](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpstreamingtype)               | Перечисление, указывающее тип носителя потоковой передачи.                                                  |
| [вмптасктипе](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptasktype)                         | перечисление, представляющее проигрыватель Windows Media области задач.                                             |
| [вмптемплатесизе](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptemplatesize)                 | Перечисление, представляющее размеры HTML-шаблонов.                                                         |
| [вмптрансактионтипе](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptransactiontype)           | Перечисление, представляющее тип транзакции.                                                          |
