# <a name="external-object-for-type-2-online-stores"></a>Внешний объект для Интернет-магазинов типа 2

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

**Внешний** объект может предоставлять функции веб-страниц, предоставляемых Интернет-магазином типа 2 и размещаемых в проигрывателе Windows Media.

**Внешний** объект предоставляет следующие свойства.



| Свойство                                                        | Описание                                                                               |
|-----------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [аппколорбуттонхигхлигхт](external-appcolorbuttonhighlight.md) | Извлекает цвет выделения текущей кнопки для пользовательского интерфейса проигрывателя Windows Media. |
| [аппколорбуттонховерфаце](external-appcolorbuttonhoverface.md) | Извлекает цвет пользовательского интерфейса проигрывателя Windows Media для текущего нажатия кнопки.     |
| [аппколорбуттоншадов](external-appcolorbuttonshadow.md)       | Получает текущий цвет тени для кнопки пользовательского интерфейса проигрывателя Windows Media.    |
| [аппколордарк](external-appcolordark.md)                       | Извлекает текущий темно-затененный цвет пользовательского интерфейса проигрывателя Windows Media.       |
| [аппколорлигхт](external-appcolorlight.md)                     | Извлекает цвет пользовательского интерфейса проигрывателя Windows Media с светлой тенью.      |
| [аппколормедиум](external-appcolormedium.md)                   | Извлекает текущий цвет в пользовательском интерфейсе проигрывателя Windows Media.     |
| [довнлоадманажер](external-downloadmanager.md)                 | Извлекает объект **довнлоадманажер** .                                                 |
| [селектедтаскпане](external-selectedtaskpane.md)               | Указывает или получает текущую выбранную область задач.                                  |
| [version](external-version.md)                                 | Извлекает текущую версию проигрывателя Windows Media.                                    |



 

**Внешний** объект предоставляет следующие методы.



| Метод                                                  | Описание                                                                          |
|---------------------------------------------------------|--------------------------------------------------------------------------------------|
| [навигатетаскпанеурл](external-navigatetaskpaneurl.md) | Открывает веб-страницу в указанной области задач и изменяет фокус на указанную панель. |
| [Плаймедиа (не рекомендуется)](external-playmedia.md)        | Указывает URL-адрес цифрового мультимедийного элемента для воспроизведения.                                   |



 

**Внешний** объект предоставляет следующее событие.



| Событие                                             | Описание                                                               |
|---------------------------------------------------|---------------------------------------------------------------------------|
| [онколорчанже](external-oncolorchange-event.md) | Происходит при изменении цвета пользовательского интерфейса проигрывателя Windows Media. |
