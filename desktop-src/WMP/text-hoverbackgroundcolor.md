# <a name="texthoverbackgroundcolor"></a>TEXT. Ховербаккграундколор

Атрибут **ховербаккграундколор** указывает или получает цвет фона, используемый для текстового элемента управления при наведении на него курсора мыши.

``` syntax
        elementID.hoverBackgroundColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей любое значение цвета Microsoft Internet Explorer.

## <a name="remarks"></a>Комментарии

Если **ховербаккграундколор** не указан, используется **backgroundColor** .

См. атрибут [value](text-value.md) для примера, иллюстрирующий использование атрибутов **текстового** элемента.
