# <a name="iwmplibrarymediacollection-property"></a>Свойство Ивмплибрари:: Медиаколлектион

Свойство **медиаколлектион** возвращает интерфейс **ивмпмедиаколлектион** для текущей библиотеки.

## <a name="syntax"></a>Синтаксис


```CSharp
public IWMPMediaCollection mediaCollection {get; set;}
```


```VB

Public ReadOnly Property mediaCollection As IWMPMediaCollection
```





## <a name="property-value"></a>Значение свойства

Интерфейс Вмплиб. Ивмпмедиаколлектион для текущей библиотеки.
