# <a name="prevbutton"></a>превбуттон

Это предопределенная **кнопка** со следующими значениями по умолчанию.

``` syntax
onclick="jscript:player.controls.previous()"
upToolTip="Previous"
cursor="system"
enabled="wmpenabled:player.controls.previous"
```

## <a name="remarks"></a>Remarks

Все свойства этой кнопки можно переопределить, явно указав их.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
