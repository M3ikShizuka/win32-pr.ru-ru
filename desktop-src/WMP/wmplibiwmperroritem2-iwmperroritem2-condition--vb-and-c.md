# <a name="iwmperroritem2condition-property"></a>Свойство IWMPErrorItem2:: Condition

Свойство **Condition** возвращает значение, указывающее условие для ошибки.

## <a name="syntax"></a>Синтаксис


```CSharp
public System.Int32 condition {get; set;}
```


```VB

Public ReadOnly Property condition As System.Int32
```





## <a name="property-value"></a>Значение свойства

Объект **System. Int32** , который является кодом условия.

## <a name="remarks"></a>Remarks

Код условия — это значение, которое используется корпорацией Майкрософт для предоставления дополнительных сведений сотрудникам службы технической поддержки.
