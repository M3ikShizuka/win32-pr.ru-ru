# <a name="axwindowsmediaplayerdvd-property"></a>Аксвиндовсмедиаплайер. DVD, свойство

Свойство DVD получает интерфейс ИВМПДВД, предоставляющий свойства и методы для работы с DVD-дисками.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```CSharp
public IWMPDVD dvd {get;}
```


```VB

Public ReadOnly Property dvd As IWMPDVD
```





## <a name="property-value"></a>Значение свойства

Интерфейс Вмплиб. ИВМПДВД.
