# <a name="eventaltkey"></a>Event. Алткэй

Атрибут **алткэй** извлекает значение, указывающее, была ли нажата клавиша ALT при возникновении события.

``` syntax
event.altKey
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением**, доступным только для чтения.



| Значение | Описание                                     |
|-------|-------------------------------------------------|
| true  | Указывает, что клавиша ALT находилась в положении вниз. |
| false | Указывает, что клавиша ALT находилась в положении вверх.   |



 

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
