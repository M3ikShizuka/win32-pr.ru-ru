# <a name="listboxfontsize"></a>Список LISTBOX. fontSize

Атрибут **fontSize** указывает или получает размер шрифта для элемента управления "список".

``` syntax
        elementID.fontSize
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**длинное целое**), определяющим размер шрифта в пунктах. Значение по умолчанию — 10.

| Требование | Значение |
|--------------------|---------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media для Windows XP или более поздней версии<br/> |
