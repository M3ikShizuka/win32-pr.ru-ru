# <a name="controls-element"></a>Элемент CONTROLs

Элемент **Controls** позволяет указать следующие атрибуты объекта **Controls** в обложке.



| attribute                                                           | Описание                                                                                     |
|---------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [куррентаудиолангуаже](controls-currentaudiolanguage.md)           | Указывает или получает код локали (LCID) звукового языка для воспроизведения.         |
| [куррентаудиолангуажеиндекс](controls-currentaudiolanguageindex.md) | Указывает или получает индекс, соответствующий звуковому языку воспроизведения.           |
| [currentItem](controls-currentitem.md)                             | Указывает или получает текущий элемент мультимедиа.                                                  |
| [куррентмаркер](controls-currentmarker.md)                         | Указывает или получает номер текущего маркера.                                               |
| [currentPosition](controls-currentposition.md)                     | Указывает или получает текущую точку клипа в секундах с начала.          |
| [куррентпоситионтимекоде](controls-currentpositiontimecode.md)     | Указывает или получает текущую позицию в текущем элементе мультимедиа, используя формат кода времени. |
