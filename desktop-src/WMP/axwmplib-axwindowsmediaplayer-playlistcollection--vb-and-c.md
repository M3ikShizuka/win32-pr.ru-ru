# <a name="axwindowsmediaplayerplaylistcollection-property"></a>Аксвиндовсмедиаплайер. Плайлистколлектион, свойство

Свойство Плайлистколлектион получает интерфейс **ивмпплайлистколлектион** .

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```CSharp
public IWMPPlaylistCollection playlistCollection {get;}
```


```VB

Public ReadOnly Property playlistCollection As IWMPPlaylistCollection
```





## <a name="property-value"></a>Значение свойства

Вмплиб. Интерфейс **ивмпплайлистколлектион** .

## <a name="remarks"></a>Комментарии

Чтобы получить значение этого свойства, требуется доступ на чтение к библиотеке. Дополнительные сведения см. в разделе [доступ к библиотеке](library-access.md).
