# <a name="mediaerror"></a>Media. Error

Свойство **Error** извлекает объект **ерроритем** , если у элемента мультимедиа есть условие ошибки.

## <a name="syntax"></a>Синтаксис

*проигрыватель*. *куррентмедиа*. **Ошибка при**

## <a name="possible-values"></a>Возможные значения

Это свойство является объектом **ерроритем** , предназначенным только для чтения.

## <a name="remarks"></a>Комментарии

Если не удается воспроизвести элемент мультимедиа, это свойство получает объект **ерроритем** , содержащий сведения о возникшей проблеме.
