# <a name="eventclienty"></a>событие. Клиент

**Клиентский** атрибут Получает координату y указателя мыши относительно клиентской области окна приложения.

``` syntax
event.clientY
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** только для чтения (**длинное целое**).
