# <a name="listening-attributes"></a>Атрибуты прослушивания

Атрибут прослушивания используется для соединения одного атрибута с другим, чтобы его значение изменялось при каждом изменении значения другого атрибута.

Атрибут прослушивания **вмппроп:** является наиболее полезным. Если значение одного атрибута задано как **вмппроп:** для второго атрибута, первое значение будет автоматически обновлено, чтобы отразить второе значение при каждом изменении второго значения.

**Пример**.


```C++
<TEXT
  value="wmpprop:mySlider.value"
/>

```



Таким образом, значение Мислидер, отображаемое положением ползунка, также может отображаться в виде числа в текстовом поле.

Два других атрибута прослушивания, **вмпенаблед:** и **вмпдисаблед:**, могут быть использованы, чтобы изменить атрибут **Enabled** элемента управления в зависимости от того, доступны ли его функциональные возможности в данный момент в проигрывателе. Эти атрибуты могут прослушивать только методы объекта **Controls** .

**Пример**.


```C++
<BUTTON 
  id="play" 
  enabled="wmpenabled:player.controls.play"
/>

```
