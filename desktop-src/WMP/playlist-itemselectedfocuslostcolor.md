# <a name="playlistitemselectedfocuslostcolor"></a>Список воспроизведения. Итемселектедфокуслостколор

Атрибут **итемселектедфокуслостколор** указывает или получает значение, обозначающее цвет текста выбранного элемента в списке воспроизведения при потере фокуса списком воспроизведения.

``` syntax
        elementID.itemSelectedFocusLostColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, представляющей цвет. значение по умолчанию — цвет текста кнопки Windows.
