# <a name="display-area-plug-ins-deprecated"></a>Отобразить подключаемые модули области (не рекомендуется)

эта страница описывает функцию, которая может быть недоступна в будущих версиях проигрыватель Windows Media и пакета SDK для проигрыватель Windows Media.

Подключаемые модули пользовательского интерфейса области отображения отображаются в основной области области **Воспроизведение** под названием исполнителя и сведения о дорожке, а также слева от области списка воспроизведения. В каждый момент времени может быть включен только один подключаемый модуль экранной области. Если подключаемый модуль области отображения включен, проигрыватель не может перейти в полноэкранный режим.

Подключаемые модули области отображения полезны для отображения больших объемов информации. Например, подключаемый модуль области просмотра может получать сведения о воспроизводимом в данный момент элементе мультимедиа и отображать связанные статьи, интервью и обзоры. Подключаемый модуль области просмотра может содержать окно браузера Microsoft Internet Explorer для вывода этих сведений с помощью HTML.

> [!Note]  
> размещенное окно браузера может внедрять ActiveX версию проигрыватель Windows Media, но это не рекомендуется, поскольку внедренный элемент управления не может взаимодействовать с автономным проигрывателем, содержащим его.

 

> [!Note]  
> проигрыватель Windows Media закрывает любой открытый подключаемый модуль области просмотра, когда пользователь воспроизводит содержимое, содержащее видео.
