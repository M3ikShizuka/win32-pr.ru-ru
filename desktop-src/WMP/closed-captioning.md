# <a name="closed-captioning"></a>Субтитры

элемент управления ActiveX проигрыватель Windows Media 6,4 включает интегрированную панель отображения скрытых субтитров, которая, когда она сделана видимой, включает синхронизированные субтитры доступных файлов мультимедиа для обмена (SAMI) и отображает текст закрытого заголовка. элемент управления проигрыватель Windows Media 7 или более поздней версии включает отображение субтитров samid с помощью HTML- **<DIV>** элемента. Например:


```C++
<DIV  ID = "CCDiv"></DIV>

```



Этот метод обеспечивает полную гибкость, поскольку веб-страницу можно разрабатывать таким образом, чтобы отображать скрытые субтитры в настроенном виде. отображение скрытых субтитров больше не требуется в фиксированном расположении, примыкающем к проигрыватель Windows Media пользовательскому интерфейсу.

После создания области для вывода скрытых титров используйте *клоседкаптион*. свойство **каптионингид** для указания расположения, где проигрыватель Windows Media отображает текст закрытого заголовка.


```C++
Player.closedCaption.captioningID = "CCDiv";

```



пакет средств разработки проигрыватель Windows Media Software Development Kit (SDK) содержит рабочий пример встраиваемого проигрывателя, который отображает текст закрытой подписи. чтобы получить примеры пакета sdk, скачайте и установите полный пакет sdk для проигрыватель Windows Media на веб-сайте майкрософт. Дополнительные сведения о закрытых субтитрах и SAMI см. на [веб-сайте Microsoft Accessibility](https://www.microsoft.com/enable/).
