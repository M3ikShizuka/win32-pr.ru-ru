# <a name="implementing-imediaobjectfreestreamingresources"></a>Реализация Имедиаобжект:: Фристреамингресаурцес

Важно, чтобы код выпустил выделенную память до уничтожения объекта подключаемого модуля. проигрыватель Windows Media вызывает **фристреамингресаурцес** , чтобы это можно было сделать. Для обеспечения безопасности пример, созданный мастером подключаемых модулей, содержит вызов **фристреамингресаурцес** в методе **финалрелеасе** , чтобы гарантировать освобождение памяти. Для примера Echo необходимо добавить следующий код в **фристреамингресаурцес** :


```
// Test whether a buffer exists.
if (m_pbDelayBuffer)
{
    delete m_pbDelayBuffer;
    m_pbDelayBuffer = NULL;
    m_pbDelayPointer = NULL;
    m_cbDelayBuffer = 0;
}
```
