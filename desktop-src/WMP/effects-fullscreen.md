# <a name="effectsfullscreen"></a>ЭФФЕКТЫ. полноэкранный режим

Атрибут « **полноэкранный** режим» указывает или получает значение, указывающее, отображается ли текущая визуализация в полноэкранном режиме. Этот атрибут можно задать только во время выполнения.

``` syntax
        elementID.fullScreen
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                                          |
|-------|------------------------------------------------------|
| Да  | Визуализация отображается в полноэкранном режиме.              |
| false | По умолчанию. Визуализация не отображается в полноэкранном режиме. |



 

## <a name="remarks"></a>Remarks

Визуализацию можно перевести в полноэкранный режим только при воспроизведении или приостановке мультимедиа. Если *проигрыватель*. **куррентмедиа** — видео, должен присутствовать подключаемый модуль видео. Если это звук, то должен присутствовать подключаемый модуль визуализации, поддерживающий полноэкранный режим отображения.
