# <a name="a-basic-playlist"></a>Базовый список воспроизведения

В следующем примере списка воспроизведения показан базовый набор элементов списка воспроизведения. при написании собственного кода необходимо изменить все url-адреса и имена файлов на допустимые имена файлов, доступные для проигрыватель Windows Media.

**Пример кода**


```XML
<ASX version = "3.0">
<TITLE>Basic Playlist Demo</TITLE>
    <ENTRY>
        <TITLE>An Entry in a basic playlist</TITLE>
        <AUTHOR>Microsoft Corporation</AUTHOR>
        <COPYRIGHT>(c)2000 Microsoft Corporation</COPYRIGHT>
        <REF HREF = "mms://proseware.com/path/Yourfile.wma" />
    </ENTRY>
</ASX>

```



В следующей таблице приведены сведения об использовании каждого элемента в примере кода.



| Линия                                                                                            | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
|-------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \<ASX version = "3.0">                                                                     | элемент [ASX](asx-element.md) идентифицирует клиент (проигрыватель Windows Media), что это Windows списка воспроизведения метафайла мультимедиа. Атрибут **Version** указывает номер версии элементов метафайла.                                                                                                                                                                                                                                                                                                                                           |
| \<TITLE>Демонстрация базового списка воспроизведения\</TITLE>                                                  | Элемент [Title](title-element--metafile.md) определяет заголовок списка воспроизведения в целом. проигрыватель Windows Media отображает эти метаданные в качестве заголовка для отображения.                                                                                                                                                                                                                                                                                                                                                                                               |
| \<ENTRY>                                                                                   | Начинает элемент [entry](entry-element.md) . Элемент **entry** — это способ определения определенного клипа в списке воспроизведения                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| \<TITLE>Запись в базовом списке воспроизведения\</TITLE>                                         | Определяет название клипа списка воспроизведения. Он отличается от предыдущего элемента **заголовка** , так как он вложен в элемент **entry** . проигрыватель Windows Media отображает эти метаданные в качестве заголовка клипа.                                                                                                                                                                                                                                                                                                                                         |
| \<AUTHOR>Microsoft Corporation\</AUTHOR>                                              | Элемент [Author](author-element.md) определяет автора клипа мультимедиа. проигрыватель Windows Media отображает эти метаданные в качестве **автора** клипа.                                                                                                                                                                                                                                                                                                                                                                                                         |
| \<!-- This is a comment. Change the following path to point to your Windows media file --> | Примечание. [Комментарии](comments.md) видны только при просмотре кода и имеют тот же формат, что и комментарии **XML** .                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| \<REF HREF = "mms://proseware.com/path/Yourfile.wma" />                                    | Фактический указатель на файл мультимедиа. Элемент [ref](ref-element.md) определяет строку как указатель на мультимедийное содержимое, а атрибут **href** — URL-адрес файла мультимедиа. в этом случае URL-адрес использует протокол MMS, поэтому он указывает на Windowsный сервер мультимедиа. Файлы мультимедиа на сервере мультимедиа обычно хранятся в том же расположении, что и HTML-документы.<br/> Обратите внимание на использование закрывающего XML элемента "/>" вместо " &lt; /реф &gt; ". Поскольку этот элемент не имеет дочерних элементов, он закрывается.<br/> |
| \</ENTRY>                                                                                  | Задает конец элемента **entry** .                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| \</ASX>                                                                                    | Указывает конец списка воспроизведения.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
