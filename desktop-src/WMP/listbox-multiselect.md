# <a name="listboxmultiselect"></a>LISTBOX. reselect

Атрибут множественного **выбора** указывает или получает значение, указывающее, может ли пользователь выбрать несколько строк. Можно задать только во время разработки.

``` syntax
        elementID.multiSelect
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим** , заданным во время разработки, и затем доступен только для чтения.



| Значение | Описание                                         |
|-------|-----------------------------------------------------|
| Да  | Пользователь может выбрать несколько строк.                 |
| false | По умолчанию. Пользователь может выбирать только отдельные строки. |
