# <a name="about-dsp-plug-ins"></a>О подключаемых модулях DSP

Проигрыватель Microsoft Windows Media предоставляет архитектуру для подключения подключаемого программного обеспечения, выполняющего функции обработки цифровых сигналов (DSP) в потоках аудио или видео во время воспроизведения. Эти дополнительные программы называются подключаемыми модулями DSP.

В следующих разделах приводятся общие сведения о подключаемых модулях DSP в проигрывателе Windows Media.

| [Обзор подключаемого модуля DSP](dsp-plug-in-developer-overview.md) | Содержит общие сведения, которые необходимо знать перед созданием подключаемого модуля DSP для проигрывателя Windows Media. |
| [Создание подключаемого модуля DSP](building-a-dsp-plug-in.md)                 | Описание использования мастера подключаемых модулей проигрывателя Windows Media для создания примера подключаемого модуля DSP.            |
