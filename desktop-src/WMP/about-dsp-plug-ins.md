# <a name="about-dsp-plug-ins"></a>О подключаемых модулях DSP

Microsoft проигрыватель Windows Media предоставляет архитектуру для подключения программного обеспечения, выполняющего функции обработки цифровых сигналов (DSP) в потоках аудио или видео во время воспроизведения. Эти дополнительные программы называются подключаемыми модулями DSP.
