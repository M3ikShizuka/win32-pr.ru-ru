# <a name="textscrollingdirection"></a>TEXT. Скроллингдиректион

Атрибут **скроллингдиректион** указывает или получает направление прокручиваемого текста.

``` syntax
        elementID.scrollingDirection
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи.



| Значение | Описание                         |
|-------|-------------------------------------|
| Левый  | По умолчанию. Прокрутка справа налево. |
| Правый | Прокрутите слева направо.          |



 

## <a name="remarks"></a>Remarks

Если для **прокрутки** задано значение false, **скроллингдиректион** игнорируется.

См. атрибут [value](text-value.md) для примера, иллюстрирующий использование атрибутов **текстового** элемента.

|--------------------|------------------------------------------------------|
