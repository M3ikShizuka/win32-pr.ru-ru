# <a name="ambientattributesaccdescription"></a>Амбиентаттрибутес. Аккдескриптион

Атрибут **аккдескриптион** указывает или получает описание для любого элемента.

``` syntax
        elementID.accDescription
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи со значением по умолчанию "" (пустая строка).

## <a name="remarks"></a>Комментарии

Этот атрибут используется в целях обеспечения специальных возможностей. Это позволяет программе чтения прочесть описание любого элемента.

Этот атрибут также применяется к элементам кнопки внутри элемента управления группы кнопок.
