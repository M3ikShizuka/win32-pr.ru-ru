# <a name="sliderthumbimage"></a>ПОЛЗУНок. Сумбимаже

Атрибут **сумбимаже** указывает или получает изображение, которое будет использоваться для представления текущей положения ползунка.

``` syntax
        elementID.thumbImage
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей имя файла изображения.

## <a name="remarks"></a>Комментарии

**Сумбимаже** указывает изображение, которое будет использоваться для представления текущей позицией, а также указывает, что пользователь может выполнить действие с элементом управления. Если изображение Thumb не указано, ползунок не является интерактивным.

Изображение бегунка центрируется в узких измерениях элемента управления "ползунок". Если изображение Thumb является более узким, чем элемент управления, изображение отображается в центре фона. Если изображение Thumb больше элемента управления, концы изображения будут обрезаны.

Положение ползунка задается центром изображения Thumb. Если **бордерсизе** равен нулю, то на начальном и конечном ползунке будет отображаться только половина эскиза. Чтобы предотвратить это, задайте для **бордерсизе** значение, которое больше или равно половине ширины изображения Thumb (или половины высоты, если для параметра **Direction** задано значение "вертикально").

Поддерживаются форматы BMP, JPG, PNG и GIF (не включая анимированные GIF).

См. **кустомслидер**. атрибут [поситионимаже](customslider-positionimage.md) для примера, иллюстрирующий использование атрибутов элемента **Slider** .
