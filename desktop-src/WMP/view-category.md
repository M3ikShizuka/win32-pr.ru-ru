# <a name="viewcategory"></a>Просмотр. Категория

Атрибут **Category** указывает или получает категорию, для которой предназначено **представление** .

``` syntax
        elementID.category
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей одно из следующих значений.



| Значение | Описание                           |
|-------|---------------------------------------|
| Все   | По умолчанию. Тема для всех типов носителей |
| Переключатель | Пользовательский интерфейс для воспроизведения радио             |
| CD    | Пользовательский интерфейс для воспроизведения компакт-дисков                |
| DVD-диск   | Пользовательский интерфейс для воспроизведения DVD-дисков               |
| Музыка | Пользовательский интерфейс для MP3, WAV, MIDI, WMA        |
| Видео | Пользовательский интерфейс для воспроизведения видео             |
