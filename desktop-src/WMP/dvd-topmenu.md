# <a name="dvdtopmenu-method"></a>Метод DVD. Топмену

Метод **топмену** останавливает воспроизведение заголовка и отображает верхнее (или корневое) меню для текущего заголовка.

## <a name="syntax"></a>Синтаксис


```JScript
DVD.topMenu()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Remarks

Каждый DVD-диск создан по-другому. Чтобы этот метод работал, DVD-диск должен содержать меню. Некоторые DVD-диски созданы таким образом, чтобы методы **топмену** и **титлемену** открывали одно и то же меню. Метод **топмену** обычно вызывает верхнее (или корневое) меню, но вместо этого может вызвать меню заголовка, если нет доступного корневого меню.

**проигрыватель Windows Media 10 Mobile:** Этот метод всегда выполняется, но не выполняет требуемую операцию.
