# <a name="equalizersettingsbypass"></a>ЕКУАЛИЗЕРСЕТТИНГС. обход

Атрибут **обхода** указывает или получает значение, указывающее, пропускается ли фильтр эквалайзера на графе фильтра.

``` syntax
        elementID.bypass
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                                |
|-------|--------------------------------------------|
| Да  | По умолчанию. Фильтр эквалайзера пропускается. |
| false | Используется фильтр эквалайзера.              |



 

## <a name="remarks"></a>Комментарии

Если этот атрибут не указан, предыдущее значение будет храниться.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
