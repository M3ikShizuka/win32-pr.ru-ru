# <a name="bits"></a>BITS

[фоновая интеллектуальная служба передачи](/windows/desktop/Bits/background-intelligent-transfer-service-portal) является компонентом операционной системы Windows. Служба BITS передает файлы (загрузки или передачи) между клиентом и сервером и предоставляет сведения о ходе выполнения, связанные с передачей. Если вы хотите передавать файлы с помощью кода C++, нужно использовать BITS. если вы хотите передавать файлы с помощью JScriptного кода на веб-странице интернет-магазина, используйте вместо этого диспетчер загрузки.

так как диспетчер загрузки проигрыватель Windows Media использует BITS, можно настроить задания копирования BITS таким образом, чтобы проигрыватель Windows Media управлять заданиями. для этого необходимо следовать правилам, описанным в разделе [соглашения о задании проигрыватель Windows Media BITS](windows-media-player-bits-job-convention.md) при добавлении заданий в очередь передачи bits. В этом разделе также описывается механизм извлечения сведений о заданиях BITS с веб-страницы Интернет-магазинов.
