# <a name="properties-methods-and-events"></a>Свойства, методы и события

у каждого объекта есть методы и свойства, с помощью которых можно программировать элемент управления проигрыватель Windows Media. Метод — это действие, которое может выполнить объект. Свойство — это значение данных, которое можно читать или изменять. Например, метод **Play** запускает воспроизведение содержимого, а свойство **Частота** кадров — текущую частоту кадров содержимого, которое воспроизводится.

Кроме того, объект **Player** создает события, которые дают возможность выполнять действия в определенное время. код создается в обработчике событий, который будет выполняться, когда проигрыватель Windows Media вызывает соответствующее событие. Например, можно написать код в обработчике событий **плайстатечанже** , который определяет, является ли изменение состояния таким, что носитель завершился, и если да, отобразится диалоговое окно, предлагающее пользователям повторить воспроизведение мультимедиа.

> [!Note]  
> все методы в объектной модели проигрыватель Windows Media являются асинхронными. При вызове двух методов в одной и той же процедуре второй метод не может полагаться на первый метод, который завершил свое действие.
