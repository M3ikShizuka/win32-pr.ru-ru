# <a name="networkmaxbandwidth"></a>Network. maxBandwidth

Свойство **maxBandwidth** указывает или получает максимально допустимую пропускную способность.

## <a name="syntax"></a>Синтаксис

*проигрыватель*. *сеть*. **maxBandwidth**

## <a name="possible-values"></a>Возможные значения

Это свойство имеет **номер** для чтения и записи (**Long**).

## <a name="remarks"></a>Remarks

Это свойство не имеет значения по умолчанию. его значение можно указать во время воспроизведения проигрыватель Windows Media, но изменение не вступит в силу, пока текущий элемент мультимедиа не будет освобожден путем открытия другого элемента или вызова *проигрывателя*. **закройте окно**. проигрыватель Windows Media пытается добиться максимально возможной пропускной способности. Только в случае задания значения будет продолжено снижение пропускной способности.

Этот параметр полезен для уменьшения объема используемой пропускной способности, особенно в случае потока с несколькими скоростями (MBR). Поток MBR содержит несколько потоков с разной скоростью. В некоторых случаях может быть желательно использовать поток с более низкой скоростью, чем требуется клиенту. В этом случае при настройке свойства **maxBandwidth** будет выбран поток с более низким битом.

Например, поток MBR может включать потоки в кодировке 20 килобит в секунду (кбит/с), 37 кбит/с и 200 кбит/с. Если задать для свойства **maxBandwidth** значение 50 000 (50 кбит/с), вместо потока 200 кбит/с будет выбран поток 37 кбит/с.
