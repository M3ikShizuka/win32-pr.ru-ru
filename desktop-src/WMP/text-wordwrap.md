# <a name="textwordwrap"></a>TEXT. wordWrap

Атрибут **WordWrap** указывает или получает значение, обозначающее включение или отключение переноса по словам.

``` syntax
        elementID.wordWrap
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                                                                                       |
|-------|---------------------------------------------------------------------------------------------------|
| true  | Перенос по словам включен.                                                                             |
| false | По умолчанию. Перенос по словам отключен. Если текст не умещается в элементе управления, текст обрезается. |



 

## <a name="remarks"></a>Комментарии

Элемент управления "текст" не разделяет слова друг от друга. Если слово выходит за пределы ширины элемента управления, для перемещения слова на следующую строку применяется перенос по словам. Если одно слово длиннее, чем ширина элемента управления, это слово обрезается и занимает одну строку.

Если атрибут **Width** не указан, то параметр **WordWrap** игнорируется, а размер текстового элемента управления изменяется, а не обтекает текст.

Если в определенных местах требуются разрывы строк, их необходимо явно указать в **значении** , используя "&\# 13;" или " \\ r". Если последняя форма используется при непосредственном указании значения, строка должна иметь префикс "JScript:", а фактическое значение должно быть заключено в одинарные кавычки, как показано ниже. Это необязательно, если значение задается динамически в обработчике событий.

Если для свойства **WordWrap** задано значение true, а **Подсказка** не указана, подсказка отобразит полный текст атрибута **value** . Если подсказка не нужна, установите для **подсказки** значение "" (пустая строка).

## <a name="examples"></a>Примеры


```C++
<THEME>
  <VIEW>
    <TEXT
      width = "50"
      height = "200"
      wordWrap = "true"
      backgroundColor = "blue"
      foregroundColor = "white"
      value = "This is a test.&#13;&#13;It is only a test."
    />
    <TEXT
      width = "50"
      height = "200"
      left = "100"
      wordWrap = "true"
      backgroundColor = "green"
      foregroundColor = "white"
      value = "JScript:'This is a test.\r\rIt is only a test.'"
    />
  </VIEW>
</THEME>

```
