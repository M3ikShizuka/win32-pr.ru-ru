# <a name="downloadcollectionstartdownload-method"></a>Довнлоадколлектион. Стартдовнлоад, метод

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Метод **стартдовнлоад** помещает в очередь загрузку.

## <a name="syntax"></a>Синтаксис


```JScript
retVal = DownloadCollection.startDownload(
  sourceURL,
  type
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*саурцеурл* \[ окне\]
</dt> <dd>

**Строка** , указывающая URL-адрес скачивания.

</dd> <dt>

*тип* \[ окне\]
</dt> <dd>

**Строка** , указывающая тип загрузки. Содержит одно из следующих значений:



| Значение      | Описание                                                                                                                                                                                 |
|------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| background | (Windows XP и более поздние версии) Загрузка выполняется в фоновом режиме, как только процессорное время станет доступным. Состояния загрузки сохраняются даже при завершении работы проигрывателя Windows Media или Windows XP. |
| реальное время  | (Все поддерживаемые операционные системы.) Загрузка выполняется всего один раз. Между сеансами не сохраняются состояния загрузки.                                                                            |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает объект **довнлоадитем** .

## <a name="remarks"></a>Комментарии

При запуске новой загрузки диспетчер загрузки добавляет его как элемент в коллекцию загрузки, которая инициировала скачивание.

Можно скачать только следующие форматы мультимедиа:

-   Advanced Systems Format (ASF)
-   MP3
-   Звуковые файлы в формате WMA
-   Видеофайлы в формате WMV

Параметр *саурцеурл* не поддерживает строки в кодировке Юникод. Он должен указывать на допустимое содержимое. Перенаправления не поддерживаются.

При использовании Windows XP звуковые файлы автоматически помещаются в соответствующие **музыкальные** папки на основе значений метаданных на уровне файлов. Видеофайлы помещаются в \\ музыкальный \\ файл \\ с типом *случайных чисел* \\ , где *случайное число* — случайное значение, создаваемое проигрывателем Windows Media для каждого пользователя, а *тип* — "реальное время" или "фон", в зависимости от типа скачивания.

При использовании серии проигрывателя Windows Media 9 с Windows 98 и Windows Millennium Edition (Me) аудио-и видеофайлы помещаются в \\ музыкальный \\ файл \\ скачать тип *случайных чисел* \\ , где *случайное число* — это случайное значение, создаваемое проигрывателем для каждого пользователя, а *тип* — "реальное время" или "фон", в зависимости от типа скачивания.

Обратите внимание, что файлы могут быть переименованы на основе атрибутов метаданных, содержащихся в файле и правилах, указанных пользователем в диалоговом окне **Параметры** . Файлы, которые не содержат метаданные, например **альбом** или **исполнитель**, могут быть перемещены в папки с меткой Неизвестный исполнитель или Неизвестный альбом.
