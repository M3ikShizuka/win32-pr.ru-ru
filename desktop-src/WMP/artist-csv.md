# <a name="artistcsv"></a>artist.csv

Этот файл содержит запись для каждого исполнителя в каталоге. Каждая запись представляет собой строку, состоящую из полей, разделенных табуляцией, описанных в таблице ниже. Поля должны присутствовать в указанном порядке.

Все поля в artist.csv являются обязательными.

В столбце Format в следующей таблице описывается способ форматирования каждого текстового поля в Юникоде. Он не ссылается на тип данных содержимого. Например, если в столбце Format указано целое число, это означает, что поле содержит строку в Юникоде, представляющую целочисленное значение, а не фактическое целое число.



| Поле                  | Формат                                            | Описание                                                                                                |
|------------------------|---------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| артистид               | Неотрицательное целое число. Пример: 65832              | Уникальный идентификатор исполнителя, уникальный в пределах artist.csv. Должно быть меньше 2 ^ 32.                        |
| ArtistName             | Строка в Юникоде. Пример: Дон Функ<br/>       | Отображаемое имя исполнителя.                                                                               |
| линкедженреид          | Неотрицательное целое число. Пример: 313<br/>      | GenreID основного жанра исполнителя. Должен быть основным жанром, а не субженре. Допускается только одно значение. |
| линкедсимиларартистидс | Н/Д                                               | Не используется в этом выпуске. Должен быть пустым.                                                                 |
| Популярность             | Может быть целочисленным или десятичным значением. Пример: 1252,25 | Ранжирование популярности. Может быть позицией в списке при сортировке по популярности.                             |
| фидсаваилабле         | Логическое. Формат: \[ 0 \| 1 \] Пример: 0<br/>    | Не используется в этом выпуске. Должен быть пустым.                                                                 |
