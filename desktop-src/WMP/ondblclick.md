# <a name="ondblclick"></a>ондблкликк

Обработчик событий **ондблкликк** обрабатывает событие, возникающее при двойном щелчке элемента пользователем.

``` syntax
ondblclick
```

## <a name="remarks"></a>Remarks

Этот обработчик событий неприменим к элементу **списка воспроизведения** , элементу **Popup** , элементу **Video** , когда *видео*. **безоконное** значение равно false или элемент **Effects** при *эффекте*. **окно** имеет значение true.
