# <a name="text-editors"></a>Текстовые редакторы

Необходимо использовать текстовый редактор, в котором не содержатся символы форматирования. Рекомендуется использовать простой текстовый редактор, такой как Блокнот, хотя среды разработки, такие как Microsoft Visual Studio, предлагают дополнительные функции. Не следует использовать редактор, например Microsoft Word, поскольку формат файлов Word по умолчанию сохраняет дополнительные коды форматирования, которые не будут использоваться. При использовании текстового процессора, такого как Word, необходимо сохранять работу в виде неформатированного текста каждый раз при сохранении файла.

Возможно, следует отключить перенос по словам, если это разрешено текстовым редактором, и вы можете включить возможность просмотра всех символов, чтобы убедиться, что никакие вкладки и другие управляющие символы не были случайно добавлены. Помните, что, включая символы табуляции в файле определения обложки, приведет к сбою обложки.
