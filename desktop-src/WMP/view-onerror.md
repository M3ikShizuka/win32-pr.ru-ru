# <a name="viewonerror"></a>Просмотр. OnError

Событие **OnError** обрабатывает событие ошибки, если для **Settings. енаблиррордиалогс** задано значение false.

``` syntax
onerror
```

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
