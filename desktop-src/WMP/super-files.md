# <a name="super-files"></a>Супер файлы

Суперпользователи используются для хранения отключенных изображений для значений TrackBar. Так как основное изображение TrackBar отображается в фоновом файле, и пользователь отключается к изображению Thumb, а не к изображению TrackBar, для значений TrackBar требуется только отключенный образ. Он хранится в файле, определенном Super, в разделе точечные рисунки файла определения обложки. Супер-файл также может хранить отправленные и отключенные изображения для других кнопок, таких как «Выкл.», которые не являются обязательными для кнопки типа попадания.

> [!Note]  
> Файлы Super Art не используются в обложках проигрывателя Windows Media 10 Mobile или более поздней версии, так как отключенные изображения для значений TrackBar для поиска находятся в файле сиксумб.

 

На следующем рисунке изображен типичный супер-файл.

![Супер файл](images/cesdksup.png)

При этом сохраняются отключенные изображения для значений TrackBar и кнопки Выключить звук. Эти образы являются смещениями, как определено в разделе "битовые карты".

Область фона отключенного изображения TrackBar точно соответствует соответствующей области в фоновом файле. Это важно, так как весь прямоугольник, определенный для отключенного изображения TrackBar, заменит соответствующую область в фоновом файле. Это гарантирует, что отключенное изображение TrackBar прозрачно смешивается с фоновым изображением.
