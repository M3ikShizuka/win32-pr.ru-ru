# <a name="purpose-of-a-metafile-playlist"></a>Назначение списка воспроизведения метафайла

основным предназначением списка воспроизведения метафайлов является перенаправление содержимого потокового мультимедиа из браузеров в проигрыватель Windows Media, так как большинство браузеров попытаются загрузить содержимое, а не будет выполнять его потоковую передачу. список воспроизведения метафайлов содержит сведения проигрыватель Windows Media, используемые для получения потоков, например URL-адреса. Список воспроизведения метафайлов может предоставить путь к файлам мультимедиа через Интернет, интрасеть или просто сохраненный носитель на сетевом или локальном диске. если обработка перенаправлена на проигрыватель Windows Media, вы получите больший контроль над потоковой передачей мультимедиа.

Списки воспроизведения метафайлов используются для управления процессом потоковой передачи. Объединив различные файлы мультимедиа в один поток содержимого, можно управлять и настраивать представление.
