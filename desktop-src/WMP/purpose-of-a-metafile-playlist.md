# <a name="purpose-of-a-metafile-playlist"></a>Назначение списка воспроизведения метафайла

Основным предназначением списка воспроизведения метафайлов является перенаправление мультимедийного содержимого из браузеров в проигрыватель Windows Media, так как большинство браузеров попытаются загрузить содержимое, а не будет выполнять его потоковую передачу. Список воспроизведения метафайлов предоставляет сведения, используемые проигрывателем Windows Media для получения потоков, таких как URL-адрес. Список воспроизведения метафайлов может предоставить путь к файлам мультимедиа через Интернет, интрасеть или просто сохраненный носитель на сетевом или локальном диске. При обработке, перенаправленной в проигрыватель Windows Media, вы имеете больший контроль над потоковой передачей мультимедиа.

Списки воспроизведения метафайлов используются для управления процессом потоковой передачи. Объединив различные файлы мультимедиа в один поток содержимого, можно управлять и настраивать представление.
