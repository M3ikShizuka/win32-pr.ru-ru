# <a name="text-color"></a>Цвет текста

Необходимо указать значение цвета для типа. Значение должно быть указано в виде трех положительных целых чисел, разделенных запятыми. Значения представляют собой значение красного, зеленого и синего цвета точечного рисунка и диапазон от 0 до 255.
