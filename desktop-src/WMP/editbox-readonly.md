# <a name="editboxreadonly"></a>EDITBOX. readOnly

Атрибут **ReadOnly** указывает или получает значение, указывающее, доступен ли текст в элементе управления "поле ввода" только для чтения или может быть изменен.

``` syntax
        elementID.readOnly
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим** значением для чтения и записи и значение по умолчанию false.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|---------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media для Windows XP или более поздней версии<br/> |
