# <a name="equalizersettingswowlevel"></a>ЕКУАЛИЗЕРСЕТТИНГС. Вовлевел

Атрибут **вовлевел** указывает или получает уровень влияния SRS WOW.

``` syntax
        elementID.wowLevel
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**длинное**) в диапазоне от 0 до 100 со значением по умолчанию 50.

## <a name="remarks"></a>Комментарии

Воздействие SRS WOW — это результат улучшения звука. Этот атрибут пропускается, если **енханцедаудио** имеет значение false.
