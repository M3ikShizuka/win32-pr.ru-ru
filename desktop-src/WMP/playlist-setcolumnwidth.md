# <a name="playlistsetcolumnwidth"></a>Список воспроизведения. Сетколумнвидс

Метод **сетколумнвидс** задает ширину столбца и изменяет режим изменения размера столбца на "Fixed".

``` syntax
        elementID.setColumnWidth(column, width)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="column"></span><span id="COLUMN"></span>*рубрик*
</dt> <dd>

**Число** (**Long**), указывающее индекс изменяемого столбца.

</dd> <dt>

<span id="width"></span><span id="WIDTH"></span>*Ширина*
</dt> <dd>

**Число** (**длинное**), указывающее новую ширину в пикселях.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
