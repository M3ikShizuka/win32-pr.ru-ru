# <a name="downloadmanagercreatedownloadcollection-method"></a>Довнлоадманажер. Креатедовнлоадколлектион, метод

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Метод **креатедовнлоадколлектион** создает коллекцию загрузки.

## <a name="syntax"></a>Синтаксис


```JScript
retVal = DownloadManager.createDownloadCollection()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает объект **довнлоадколлектион** .
