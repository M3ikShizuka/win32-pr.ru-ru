# <a name="listboxappenditem"></a>LISTBOX. Аппендитем

Метод **аппендитем** вставляет новый текст в конец элемента управления "список".

``` syntax
        elementID.appendItem(newVal)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="newVal"></span><span id="newval"></span><span id="NEWVAL"></span>*неввал*
</dt> <dd>

**Строка** , содержащая вставляемый текст.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

|--------------------|---------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media для Windows XP или более поздней версии<br/> |
