# <a name="dvd-object"></a>Объект DVD

Объект **DVD** предоставляет свойства и методы для работы с DVD-дисками.

Объект **DVD** поддерживает следующие свойства.



| Свойство                           | Описание                                                                                        |
|------------------------------------|----------------------------------------------------------------------------------------------------|
| [поддомен](dvd-domain.md)           | Возвращает текущий домен DVD-диска.                                                           |
| [isAvailable](dvd-isavailable.md) | Получает сведения о доступности указанного типа сведений или о том, что может быть выполнено данное действие. |



 

Объект **DVD** поддерживает следующие методы.



| Метод                         | Описание                                                                                          |
|--------------------------------|------------------------------------------------------------------------------------------------------|
| [Назад](dvd-back.md)           | Изменяет отображение из подменю на его родительское меню.                                               |
| [выход](dvd-resume.md)       | Изменение режима воспроизведения из режима меню с возобновлением в той же позиции, что и при вызове меню. |
| [титлемену](dvd-titlemenu.md) | Останавливает воспроизведение и отображает меню заголовка.                                                          |
| [топмену](dvd-topmenu.md)     | Останавливает воспроизведение и отображает корневое меню.                                                           |



 

Доступ к объекту **DVD** осуществляется через следующее свойство.



| Объект                      | Свойство              |
|-----------------------------|-----------------------|
| [Игрок](player-object.md) | [оптически](player-dvd.md) |
