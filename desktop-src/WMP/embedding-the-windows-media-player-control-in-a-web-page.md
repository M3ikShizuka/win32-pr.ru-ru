# <a name="embedding-the-windows-media-player-control-in-a-web-page"></a>внедрение элемента управления проигрыватель Windows Media в веб-страницу

при внедрении элемента управления проигрыватель Windows Media на веб-странице можно использовать базовый подход или более сложный подход.

В базовом подходе элемент управления используется только для воспроизведения фоновой музыки или для добавления видеоэкрана на страницу.

В более тщательном подходе вы сделаете страницу интерактивной, добавив код скрипта для управления этим элементом с помощью объектной модели проигрывателя. Это позволяет предоставить пользовательский интерфейс собственного дизайна или динамически реагировать на команды или события, такие как переходы мультимедиа.

дополнительные сведения см. в разделе [использование элемента управления проигрыватель Windows Media на веб-странице](using-the-windows-media-player-control-in-a-web-page.md).
