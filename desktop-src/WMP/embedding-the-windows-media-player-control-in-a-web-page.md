# <a name="embedding-the-windows-media-player-control-in-a-web-page"></a>Внедрение элемента управления проигрывателя Windows Media в веб-страницу

При внедрении элемента управления проигрывателя Windows Media на веб-страницу можно использовать базовый подход или более сложный подход.

В базовом подходе элемент управления используется только для воспроизведения фоновой музыки или для добавления видеоэкрана на страницу.

В более тщательном подходе вы сделаете страницу интерактивной, добавив код скрипта для управления этим элементом с помощью объектной модели проигрывателя. Это позволяет предоставить пользовательский интерфейс собственного дизайна или динамически реагировать на команды или события, такие как переходы мультимедиа.

Дополнительные сведения см. [в разделе Использование элемента управления проигрывателя Windows Media на веб-странице](using-the-windows-media-player-control-in-a-web-page.md).
