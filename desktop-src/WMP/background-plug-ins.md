# <a name="background-plug-ins"></a>Фоновые подключаемые модули

Фоновые подключаемые модули пользовательского интерфейса на самом деле не имеют пользовательских интерфейсов, если только они не реализуют страницы свойств. когда запускается один или несколько фоновых подключаемых модулей, в проигрыватель Windows Media отображается специальный значок, расположенный рядом с временем выполнения. Этот значок можно щелкнуть, чтобы отобразить параметры фоновых подключаемых модулей в диалоговом окне "Параметры". После этого пользователь может открыть страницу свойств любого подключаемого модуля, который поддерживает такой вариант.

Фоновые подключаемые модули полезны для предоставления автоматических служб, которые не нуждаются в вводе данных пользователем.
