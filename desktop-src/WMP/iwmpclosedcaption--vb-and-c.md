# <a name="iwmpclosedcaption-vb-and-c-interface"></a>Интерфейс Ивмпклоседкаптион (VB и C#)

Предоставляет возможность включать субтитры с цифровым файлом мультимедиа. Текст субтитров находится в синхронизированном доступном файле обмена мультимедиа (SAMI).

## <a name="members"></a>Элементы

Интерфейс **ивмпклоседкаптион (VB и C#)** не определяет никаких членов.

Интерфейс **ивмпклоседкаптион** предоставляет следующие свойства.



| Свойство                                                                             | Описание                                                                                |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [каптионингид](wmplibiwmpclosedcaption-iwmpclosedcaption-captioningid--vb-and-c.md) | Возвращает или задает имя HTML-элемента, отображающего субтитры.                       |
| [самифиленаме](wmplibiwmpclosedcaption-iwmpclosedcaption-samifilename--vb-and-c.md) | Возвращает или задает имя файла, содержащего сведения, необходимые для закрытого субтитров. |
| [самиланг](wmplibiwmpclosedcaption-iwmpclosedcaption-samilang--vb-and-c.md)         | Возвращает или задает язык, отображаемый для скрытых титров.                                 |
| [самистиле](wmplibiwmpclosedcaption-iwmpclosedcaption-samistyle--vb-and-c.md)       | Возвращает или задает стиль закрытых субтитров.                                                  |



 

Получите интерфейс **ивмпклоседкаптион** с помощью следующего свойства.



| Объект                                                            | Свойство                                                                   |
|-------------------------------------------------------------------|----------------------------------------------------------------------------|
| [аксвиндовсмедиаплайер](axwindowsmediaplayer-object--vb-and-c.md) | [клоседкаптион](axwmplib-axwindowsmediaplayer-closedcaption--vb-and-c.md) |
