# <a name="eventshiftkey"></a>Event. Шифткэй

Атрибут **шифткэй** извлекает значение, указывающее, была ли нажата клавиша SHIFT при возникновении события.

``` syntax
event.shiftKey
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением**, доступным только для чтения.



| Значение | Описание                                       |
|-------|---------------------------------------------------|
| true  | Указывает, что клавиша SHIFT находится в смещенном положении. |
| false | Указывает, что клавиша SHIFT находится в положении вверх.   |



 

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
