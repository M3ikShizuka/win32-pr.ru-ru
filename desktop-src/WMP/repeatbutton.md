# <a name="repeatbutton"></a>REPEATBUTTON

Это предопределенная **кнопка** со следующими значениями по умолчанию.

``` syntax
onclick="jscript:player.settings.setMode('loop',down);"
upToolTip="Turn Repeat On"
downToolTip="Turn Repeat Off"
down="wmpprop:player.settings.getMode('loop')"
sticky="true"
```

## <a name="remarks"></a>Remarks

При этом создается элемент управления **"Кнопка"** , который включает и выключает повторы. Всплывающие подсказки локализованы. Все свойства этой **кнопки** можно переопределить, явно указав их.
