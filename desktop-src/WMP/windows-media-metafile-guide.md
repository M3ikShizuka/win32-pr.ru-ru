# <a name="windows-media-metafile-guide"></a>Windows Мультимедийное по для метафайлов

Windowsный метафайл мультимедиа может быть простым или сложным, по мере необходимости. самый простой Windowsный метафайл мультимедиа содержит только универсальный указатель ресурсов (URL-адрес) некоторого мультимедийного содержимого на сервере. клиент, проигрыватель Windows Media, анализирует эти сведения, а затем открывает файл мультимедиа или поток, определенный в метафайле мультимедиа Windows. сложный метафайл может содержать несколько файлов или потоков, размещенных в списке воспроизведения, инструкции по воспроизведению файлов или потоков, текстовых и графических элементов, таких как заголовок, автор и текст авторских прав, персонализированная вставка рекламы в динамический поток, гиперссылки, связанные с элементами интерфейса проигрыватель Windows Media и многое другое.
