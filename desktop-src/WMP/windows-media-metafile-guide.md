# <a name="windows-media-metafile-guide"></a>Путеводитель по метафайлу Windows Media

Метафайл Windows Media может быть простым или сложным по мере необходимости. Самый простой метафайл Windows Media содержит только универсальный указатель ресурсов (URL-адрес) некоторого мультимедийного содержимого на сервере. Клиент, проигрыватель Windows Media, анализирует эти сведения, а затем открывает файл мультимедиа или поток, определенный в метафайле Windows Media. Сложный метафайл может содержать несколько файлов или потоков, расположенных в списке воспроизведения, инструкции по воспроизведению файлов или потоков, текстовых и графических элементов, таких как заголовок, автор и текст авторских прав, персонализированная вставка рекламы в поток Live, гиперссылки, связанные с элементами интерфейса проигрывателя Windows Media, и многое другое.
