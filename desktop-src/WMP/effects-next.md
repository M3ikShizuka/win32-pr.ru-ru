# <a name="effectsnext"></a>EFFECTs. Next

**Следующий** метод отображает следующую предустановку визуализации, переходя к следующей визуализации, если это необходимо.

``` syntax
        elementID.next()
```

## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Если текущая Предустановка является последней в серии всех доступных визуализаций, то первая Предустановка первой визуализации становится текущей.

| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
