# <a name="effectseffecthaspropertypage"></a>EFFECTs. Еффексаспропертипаже

Атрибут **еффексаспропертипаже** извлекает значение, указывающее, содержит ли текущая визуализация страницу свойств.

``` syntax
        elementID.effectHasPropertyPage
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением**, доступным только для чтения.



| Значение | Описание                                  |
|-------|----------------------------------------------|
| Да  | Визуализация содержит страницу свойств.           |
| false | Визуализация не имеет страницы свойств. |



 

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
