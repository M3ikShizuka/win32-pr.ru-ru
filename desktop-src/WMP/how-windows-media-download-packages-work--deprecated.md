# <a name="how-windows-media-download-packages-work-deprecated"></a>Как работают пакеты загрузки Windows Media (не рекомендуется)

Эта страница описывает функцию, которая может быть недоступна в будущих версиях проигрывателя Windows Media и Windows Media Player SDK.

Пакет загрузки Windows Media запускается с веб-сайта, когда пользователь щелкает ссылку в веб-браузере, например Microsoft Internet Explorer. Это действие открывает проигрыватель Windows Media, а затем скачивает и отменяет упаковку пакета загрузки Windows Media на жестком диске пользователя в папке по умолчанию.

После извлечения файлов из пакета скачивания Windows Media Player находит список воспроизведения метафайлов Windows Media с расширением. ASX в упакованных файлах. Если такой объект найден, проигрыватель создает список воспроизведения на основе включенного метафайла. Файлы, содержащие мультимедийное содержимое, затем добавляются в библиотеку.

Проигрыватель Windows Media также ищет элемент **обложки** в метафайле. Если элемент **Skin** содержит ссылку на файл границы с расширением WMZ, проигрыватель загружает границу **в область текущий** . Затем проигрыватель начинает воспроизводить содержимое, указанное в пакете.

На следующей схеме показано, как содержимое упаковывается в файл загрузки Windows Media, отправляется на веб-сайт, загружается и воспроизводится на клиентском компьютере с помощью проигрывателя Windows Media.
