# <a name="playlistcopying"></a>Список воспроизведения. копирование

Атрибут **Copy** извлекает значение, указывающее, находится ли элемент **списка воспроизведения** в процессе копирования.

``` syntax
        elementID.copying
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением**, доступным только для чтения.



| Значение | Описание                              |
|-------|------------------------------------------|
| true  | Выполняется копирование элемента **списка воспроизведения** .     |
| false | Элемент **списка воспроизведения** не копируется. |



 

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
