# <a name="eventctrlkey"></a>Event. Ктрлкэй

Атрибут **ктрлкэй** извлекает значение, указывающее, была ли нажата клавиша CTRL при возникновении события.

``` syntax
event.ctrlKey
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением**, доступным только для чтения.



| Значение | Описание                                      |
|-------|--------------------------------------------------|
| true  | Указывает, что клавиша CTRL находилась в положении вниз. |
| false | Указывает, что клавиша CTRL находилась в положении вверх.   |



 

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
