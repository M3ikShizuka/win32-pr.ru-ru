# <a name="iwmpcdromrip-vb-and-c-interface"></a>Интерфейс Ивмпкдромрип (VB и C#)

Предоставляет свойства и методы для управления копированием и *копированием* дорожек с аудио компакт-диска.

Копирование компакт-диска с помощью интерфейса **ивмпкдромрип** действует так же, как копирование музыки с помощью пользовательского интерфейса проигрывателя Windows Media. Скопированное содержимое автоматически добавляется в библиотеку в соответствии с предпочтениями пользователя. Дополнительные сведения о параметрах пользователя для копирования компакт-дисков см. в разделе "Копирование музыки с CD" в справке проигрывателя Windows Media.

Интерфейс **ивмпкдромрип** предоставляет следующие свойства.

## <a name="members"></a>Элементы

Интерфейс **ивмпкдромрип (VB и C#)** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ивмпкдромрип (VB и C#)** содержит следующие методы.



| Метод                                                                 | Описание                              |
|:-----------------------------------------------------------------------|:-----------------------------------------|
| [**стартрип**](wmplibiwmpcdromrip-iwmpcdromrip-startrip--vb-and-c.md) | Рипс компакт-диск.<br/>                  |
| [**стоприп**](wmplibiwmpcdromrip-iwmpcdromrip-stoprip--vb-and-c.md)   | Останавливает процесс копирования компакт-дисков.<br/> |



 

### <a name="properties"></a>Свойства

Интерфейс **ивмпкдромрип (VB и C#)** имеет следующие свойства.



| Свойство                                                                                | Тип доступа          | Описание                                                                                   |
|:----------------------------------------------------------------------------------------|:---------------------|:----------------------------------------------------------------------------------------------|
| [**риппрогресс**](wmplibiwmpcdromrip-iwmpcdromrip-ripprogress--vb-and-c.md)<br/> | Только для чтения<br/> | Возвращает ход выполнения копирования с компакт-диска в процентах завершения.<br/>                                  |
| [**рипстате**](wmplibiwmpcdromrip-iwmpcdromrip-ripstate--vb-and-c.md)<br/>       | Только для чтения<br/> | Возвращает значение перечисления, указывающее текущее состояние процесса копирования.<br/> |



 

Получите интерфейс **ивмпкдромрип** путем приведения интерфейса **ивмпкдром** компакт-диска, который вы хотите скопировать.
