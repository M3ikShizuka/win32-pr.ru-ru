# <a name="drmkeyid-attribute"></a>Атрибут Дрмкэйид

Атрибут **дрмкэйид** определяет права на использование мультимедиа для содержимого, защищенного с помощью управления цифровыми правами (DRM).

## <a name="applies-to"></a>Применяется к

-   [Звуковые элементы](audio-item-attributes.md)
-   [Элементы видео](video-item-attributes.md)

## <a name="remarks"></a>Remarks

Обычно этот атрибут используется для запроса и отображения прав на использование мультимедиа.

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .
