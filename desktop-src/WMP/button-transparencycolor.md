# <a name="buttontransparencycolor"></a>КНОПКА. Транспаренциколор

Атрибут **транспаренциколор** указывает или получает цвет, который будет прозрачным в изображениях **кнопки** .

``` syntax
        elementID.transparencyColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи и не имеет значения по умолчанию, содержащего одно из следующих значений.



| Значение                                    | Описание                                                                                               |
|------------------------------------------|-----------------------------------------------------------------------------------------------------------|
| Auto (Автоматически)                                     | Цвет пикселя в положении 0, 0 в изображении превращается в прозрачный цвет.                        |
| Любое значение формата цвета Internet Explorer | Значение формата цвета Internet Explorer превращается в прозрачный цвет (например, "Red" или " \# FF0000"). |
| Нет                                     | Прозрачность отсутствует.                                                                                          |



 

## <a name="remarks"></a>Комментарии

Прозрачный цвет в изображении позволяет отображать все, что находится за изображением, с помощью прозрачных областей. **Кнопка** по-прежнему будет принимать щелчки в прозрачной области.

Прозрачный цвет может быть любым значением цвета Internet Explorer. Если для атрибута **транспаренциколор** задано значение Auto, то используется цвет пикселя в положении 0, 0 в изображении.

Если указанный цвет не является одним из допустимых цветов IE, сохраняется предыдущее значение.

Так как Жпгс являются утерянными и поэтому подвержены непредвиденному изменению цвета, они не рекомендуются при использовании **транспаренциколор** .
