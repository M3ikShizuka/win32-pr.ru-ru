# <a name="customsliderhoverimage"></a>КУСТОМСЛИДЕР. Ховеримаже

Атрибут **ховеримаже** указывает или получает изображение, которое отображается при наведении мыши на пользовательский ползунок.

``` syntax
        elementID.hoverImage
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей имя файла изображения.

## <a name="remarks"></a>Комментарии

Этот атрибут является необязательным. Если он не указан, будет использоваться файл, указанный в атрибуте **Image** .

**Ховеримаже** представляет состояние наведения указателя мыши на элемент управления кустомслидер; то есть состояние, отображаемое при наведении курсора мыши на него. Это может быть один образ или несколько изображений, представляющих различные состояния ползунка. Если используется несколько образов, они упорядочиваются так же, как и файл **изображения**

Поддерживаются следующие типы файлов изображений: BMP, JPG, PNG и GIF (не включая анимированные GIF).
