# <a name="debugging-the-page"></a>Отладка страницы

Когда проигрыватель Windows Media размещает веб-страницу Интернет-магазина, он отключает возможность открытия диалоговых окон на веб-странице. Для отладки кода веб-страницы следует добавить код, который временно выводит выходные данные в HTML-элемент на веб-странице, например элемент DIV или TEXT.
