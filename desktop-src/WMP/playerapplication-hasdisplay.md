# <a name="playerapplicationhasdisplay"></a>Плайераппликатион. Хасдисплай

Свойство **хасдисплай** извлекает значение, указывающее, может ли видео отображаться с помощью удаленного элемента управления Player.

## <a name="syntax"></a>Синтаксис

*плайераппликатион*. **хасдисплай**

## <a name="possible-values"></a>Возможные значения

Это свойство является **логическим значением**, доступным только для чтения.

## <a name="remarks"></a>Комментарии

Это свойство используется только при удаленном взаимодействии с элементом управления проигрывателя Windows Media.

Несколько элементов управления проигрывателем Windows Media могут выполняться удаленно одновременно, но видео может отображаться в одном расположении одновременно либо в полноэкранном режиме проигрывателя, либо в одном из удаленных элементов управления проигрывателя Windows Media. Используйте это свойство, чтобы определить, является ли текущий элемент управления элементом, с помощью которого можно отобразить видео.

**Проигрыватель Windows Media 10 Mobile:** Это свойство всегда возвращает **значение true**.
