# <a name="about-device-synchronization"></a>Сведения о синхронизации устройств

в проигрыватель Windows Media 10 появилась новая модель для синхронизации мультимедийного содержимого с портативными устройствами. С точки зрения пользователя это означает, что можно указать, какие списки воспроизведения (включая автоматические списки воспроизведения) будут автоматически синхронизироваться с устройствами. Можно также вручную передавать цифровые мультимедийные данные на устройства. С точки зрения разработчика это означает, что есть новые функциональные возможности, которые можно использовать в приложениях. для этого необходимо создать удаленный экземпляр элемента управления проигрыватель Windows Media.

Существует два способа, с помощью которых пользователь может скопировать мультимедийное содержимое на устройство:

-   **Ручной перенос.** Пользователь выбирает цифровое мультимедийное содержимое в библиотеке, а затем инициирует перенос содержимого на устройство. это похоже на функциональные возможности, предоставляемые предыдущими версиями проигрыватель Windows Media. пакет SDK для проигрыватель Windows Media не предоставляет методы для передачи цифровых мультимедийных файлов на устройство.
-   **Автоматическая синхронизация.** Пользователь указывает списки воспроизведения, которые автоматически синхронизируются с устройством. это функция проигрыватель Windows Media 10 или более поздней версии. пакет SDK для проигрыватель Windows Media предоставляет функции для управления автоматической синхронизацией. Эта функция предназначена для создания пользовательского пользовательского интерфейса для приложения, позволяющего указать способ синхронизации устройства и предоставить пользователям сведения о состоянии.

чтобы автоматическая синхронизация работала, между проигрыватель Windows Media и устройством должно быть установлено специальное отношение. Это отношение называется *партнерством*.
