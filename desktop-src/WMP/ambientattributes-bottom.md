# <a name="ambientattributesbottom"></a>Амбиентаттрибутес. Bottom

**Нижний** атрибут указывает или получает нижнюю координату элемента управления.

``` syntax
        elementID.bottom
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**длинное**), представляющим расстояние в пикселях от элемента управления до нижней границы родительского **представления** или подчиненного **представления**. Он имеет нулевое значение по умолчанию. Поведение для отрицательных значений или если [амбиентаттрибутес. Height](ambientattributes-height.md) не задано, не определено.
