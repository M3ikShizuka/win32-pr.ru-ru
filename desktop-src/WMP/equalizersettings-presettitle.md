# <a name="equalizersettingspresettitle"></a>ЕКУАЛИЗЕРСЕТТИНГС. Пресеттитле

Метод **пресеттитле** извлекает имя предустановки эквалайзера с указанным индексом.

``` syntax
        elementID.presetTitle(index)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*номер*
</dt> <dd>

**Число** (**длинное**), содержащее предустановленный индекс.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает **строку**.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|---------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media 9 Series или более поздней версии<br/> |
