# <a name="about-imfgetservice"></a>О Имфжетсервице

Интерфейс **имфжетсервице** — это один из интерфейсов, который должен быть реализован подключаемым модулем DSP с двойным режимом. У него есть только один метод, **услуга**. Подключаемый модуль DSP реализует метод **WebService** , чтобы клиенты могли получать указатели на интерфейсы, поддерживаемые подключаемым модулем, когда подключаемый модуль выполняется в конвейере Media Foundation.
