# <a name="playlistdeleteselectedfromlibrary"></a>Список воспроизведения. Делетеселектедфромлибрари

Метод **делетеселектедфромлибрари** удаляет выбранный элемент из списка воспроизведения и из библиотеки.

``` syntax
        elementID.deleteSelectedFromLibrary()
```

## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="requirements"></a>Requirements (Требования)



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
