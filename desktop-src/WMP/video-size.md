# <a name="video-size"></a>Размер видео

Можно указать размер отображаемой области экрана до максимального размера экрана устройства. Однако в такой большой области отображаются все элементы управления, предоставляемые вашей обложкой, что делает их бесполезными во время воспроизведения видео. Если область отображения видео больше размеров воспроизводимого видео, то видео будет отображаться фактическим размером, а неиспользуемые части экрана видео будут отображаться в виде черной границы вокруг изображения видео. Если область отображения видео меньше размеров воспроизводимого видео, то видео по-прежнему будет отображаться по фактическому размеру, но будет отображаться в центре доступной области отображения.

В следующей таблице приведены рекомендуемые стандартные размеры видео.



| DPI | Ширина в пикселях | Высота в пикселях |
|-----|-----------------|------------------|
| 96  | 208             | 160              |
| 192 | 516             | 320              |



 

В следующей таблице приведены рекомендуемые минимальные размеры видео.



| DPI | Ширина в пикселях | Высота в пикселях |
|-----|-----------------|------------------|
| 96  | 160             | 120              |
| 192 | 320             | 240              |
