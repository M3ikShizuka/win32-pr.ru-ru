# <a name="creating-the-art"></a>Создание рисунка

Иллюстрация для этой обложки была создана с помощью Adobe Photoshop. Могут использоваться другие программы редактирования графики, такие как Жаск Paint Shop Pro или Microsoft Paint. Photoshop был выбран в качестве примера приложения для редактирования графики, которое поддерживает слои, что полезно для создания нескольких растровых файлов, которые будут выровняться друг с другом.

Для создания иллюстрации использовались следующие шаги. Можно использовать множество других графических процедур и средств. Это лишь один из способов сделать это. Важно помнить, что необходимо придерживаться конкретных файлов точечных рисунков, необходимых для обложки.
