# <a name="album-art"></a>Обложка альбома

При создании обложки для проигрывателя Windows Media 10 Mobile или более поздней версии может потребоваться отображение обложки альбома, относящейся к текущему воспроизводимому элементу мультимедиа. При проектировании обложки необходимо зарезервировать место в фоновом изображении, чтобы оно содержало обложку альбома.

Раздел обложки альбома файла определения обложки должен начинаться со следующей строки:


```C++
[ Album Art ]

```



Затем необходимо добавить сведения, указывающие расположение и размер обложки альбома в оболочке. Необходимо ввести четыре положительных целых числа, разделенные запятыми, которые определяют левую, верхнюю, ширину и высоту обложки альбома (в пикселях) относительно фонового изображения. В следующем примере показано, как задать измерения:


```C++
//  <Location>
//  ----------
   5,37,230,155

```



Если вы планируете включить в обложку раздел видео, то для экономии места можно использовать тот же размер и расположение для обложки альбома.
