# <a name="using-asp-pages-to-dynamically-create-windows-media-metafile-playlists"></a>Динамическое создание списков воспроизведения метафайлов Windows Media с помощью страниц ASP

Для динамического создания списков воспроизведения на основе сведений, предоставленных пользователями, можно использовать Active Server страниц (ASP или ASP-файлы). ASP-страница — это динамическая веб-страница, используемая совместно с Microsoft службы IIS (IIS). ASP — это среда, в которой можно сочетать HTML, сценарии и многократно используемые серверные компоненты ActiveX для создания динамических и мощных веб-решений. ASP-страницы позволяют использовать сценарии на стороне сервера для IIS с встроенной поддержкой Microsoft Visual Basic Scripting Edition (VBScript) и Microsoft JScript. В этом обсуждении предполагается, что вы знакомы с ASP и определяете переменные.

Вся информация заголовка должна содержаться в первой строке строки ASP-страницы, возвращаемой проигрывателю Windows Media.

При использовании страниц ASP для создания списков воспроизведения необходимо указать значения свойств **ContentType** и **Expires** объекта **Response** на странице ASP из-за проблем с задержкой в проигрывателе Windows Media. Значение **Response. ContentType** должно быть допустимым расширением имени файла для метафайлов Windows Media. Допустимые значения: WMA, мастика, WMV, ввкс, ASF и ASX.

Свойство **Response. Expires** определяет продолжительность времени в секундах, в течение которого проигрыватель Windows Media кэширует файл списка воспроизведения. При указании нулевого значения проигрыватель Windows Media запрашивает новый список воспроизведения с сервера каждый раз, когда пользователь обновляет страницу.

Дополнительные сведения об использовании объекта **Response** на Active Server страницах см. в пакете SDK для платформы.

Следующий код является примером страницы ASP, используемой для создания списка воспроизведения метафайла Windows Media.


```XML
<%Response.ContentType = "video/x-ms-wma"%><%Response.expires=0 %>
<ASX VERSION="3.0">
    <TITLE>Your title here</TITLE>
    <ENTRY>
        <REF HREF ="mms://adventure-works.com/pubpt/filename.wma" />
    </ENTRY>
</ASX>

```
