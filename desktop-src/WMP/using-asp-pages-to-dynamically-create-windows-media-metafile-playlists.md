# <a name="using-asp-pages-to-dynamically-create-windows-media-metafile-playlists"></a>использование страниц ASP для динамического создания Windows списков воспроизведения метафайлов мультимедиа

Для динамического создания списков воспроизведения на основе сведений, предоставленных пользователями, можно использовать Active Server страниц (ASP или ASP-файлы). ASP-страница — это динамическая веб-страница, используемая в сочетании с Microsoft IIS (IIS). ASP — это среда, в которой можно сочетать HTML, сценарии и многократно используемые ActiveX серверные компоненты для создания динамических и мощных веб-решений. asp-страницы позволяют использовать сценарии на стороне сервера для IIS с встроенной поддержкой microsoft Visual Basic scripting Edition (VBScript) и microsoft JScript. В этом обсуждении предполагается, что вы знакомы с ASP и определяете переменные.

все данные заголовка должны содержаться в первой строке строки ASP-страницы, возвращаемой в проигрыватель Windows Media.

при использовании страниц asp для создания списков воспроизведения необходимо указать значения свойств **ContentType** и **expires** объекта **Response** на странице ASP из-за проблем с задержкой проигрыватель Windows Media. значение **Response. ContentType** должно быть допустимым расширением имени файла для Windows метафайлов мультимедиа. Допустимые значения: WMA, мастика, WMV, ввкс, ASF и ASX.

свойство **Response. expires** определяет продолжительность времени в секундах, в течение которого проигрыватель Windows Media кэширует файл списка воспроизведения. если задать нулевое значение, проигрыватель Windows Media запрашивает новый список воспроизведения с сервера каждый раз, когда пользователь обновляет страницу.

Дополнительные сведения об использовании объекта **Response** на Active Server страницах см. в пакете SDK для платформы.

следующий код является примером страницы ASP, используемой для создания Windows списка воспроизведения метафайла мультимедиа.


```XML
<%Response.ContentType = "video/x-ms-wma"%><%Response.expires=0 %>
<ASX VERSION="3.0">
    <TITLE>Your title here</TITLE>
    <ENTRY>
        <REF HREF ="mms://adventure-works.com/pubpt/filename.wma" />
    </ENTRY>
</ASX>

```
