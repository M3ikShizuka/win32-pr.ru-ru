# <a name="sliderthumbdisabledimage"></a>ПОЛЗУНок. Сумбдисабледимаже

Атрибут **сумбдисабледимаже** указывает или получает изображение бегунка элемента управления Slider, которое используется при отключении элемента управления.

``` syntax
        elementID.thumbDisabledImage
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей имя файла изображения.

## <a name="remarks"></a>Remarks

**Сумбдисабледимаже** является необязательным. Если он не указан, вместо него используется **сумбимаже** .

Поддерживаются форматы BMP, JPG, PNG и GIF (не включая анимированные GIF).
