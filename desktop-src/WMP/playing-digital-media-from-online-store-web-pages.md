# <a name="playing-digital-media-from-online-store-web-pages"></a>Воспроизведение цифровых мультимедийных данных из веб-страниц интерактивного магазина

Вы можете воспроизвести мультимедийное содержимое на веб-страницах Интернет-магазина. Просто внедрите элемент управления проигрывателя Windows Media, как описано в разделе [использование элемента управления проигрывателем Windows Media на веб-странице](using-the-windows-media-player-control-in-a-web-page.md). При выполнении этой задачи в области задач службы внедренный элемент управления подключается к программе проигрывателя Windows Media аналогично удаленному режиму (см. раздел удаленный [доступ к элементу управления проигрывателя Windows Media](remoting-the-windows-media-player-control.md)). Это означает, что пользователи могут использовать элементы управления транспорта полноэкранного режима для управления воспроизведением цифровых носителей на веб-странице.

Рекомендуется установить для **uiMode** значение "нет" или "Невидимый" при внедрении элемента управления проигрывателя на веб-странице области задач службы.
