# <a name="base-element"></a>БАЗОВЫЙ элемент

**Базовый** элемент определяет строку URL-адреса, добавляемую к началу URL-адресов, отправляемых проигрывателю Windows Media.

``` syntax
<BASE
   HREF = "URL"
/>
```

## <a name="attributes"></a>Атрибуты

**Href** (обязательно)

Строка, добавляемая к URL-адресам, отправляемым проигрывателю Windows Media. Значением по умолчанию является строка NULL ("").

## <a name="parentchild-elements"></a>Родительские и дочерние элементы



| Иерархия       | Элементы           |
|-----------------|--------------------|
| Родительские элементы | **ASX**, **запись** |
| Дочерние элементы  | Нет               |



 

## <a name="remarks"></a>Remarks

Этот элемент определяет строку URL-адреса, добавленную в начало URL-адреса URL, отправляемого проигрывателю Windows Media. Зеркальное отображение URL-адресов — это механизм создания сценариев, который заставляет проигрыватель Windows Media открывать новый URL-адрес в браузере или кадре браузера при получении команды сценария.

**Базовый** элемент аналогичен домашнему каталогу для относительных ссылок. Он влияет только на URL-адреса, отправленные с помощью команд сценария, в рамках потока содержимого, который воспроизводится проигрывателем Windows Media.

**Базовый** элемент, определенный как дочерний элемент элемента **ASX** , станет значением по умолчанию для всего метафайла. **Базовый** элемент, определенный в элементе **entry** , переопределяет любой **базовый** элемент в родительском элементе **ASX** (только для этого элемента **entry** ).

> [!Note]  
> Если атрибут **href** не заканчивается символом/, проигрыватель Windows Media добавляет один объект в конец строки.

 

## <a name="examples"></a>Примеры


```XML
<BASE HREF="https://msdn.microsoft.com/" />

```
