# <a name="base-element"></a>БАЗОВЫЙ элемент

**базовый** элемент определяет строку url-адреса, добавляемую к началу url-адресов, отправляемых в проигрыватель Windows Media.

``` syntax
<BASE
   HREF = "URL"
/>
```

## <a name="attributes"></a>Атрибуты

**Href** (обязательно)

строка, добавляемая к url-адресам, отправленным в проигрыватель Windows Media. Значением по умолчанию является строка NULL ("").

## <a name="parentchild-elements"></a>Родительские и дочерние элементы



| Иерархия       | Элементы           |
|-----------------|--------------------|
| Родительские элементы | **ASX**, **запись** |
| Дочерние элементы  | Нет               |



 

## <a name="remarks"></a>Remarks

этот элемент определяет строку url-адреса, добавляемую перед url-адресами url, отправленными в проигрыватель Windows Media. зеркальное отображение url-адресов — это механизм создания сценариев, который вызывает проигрыватель Windows Media открывать новый URL-адрес в браузере или кадре браузера при получении команды сценария.

**Базовый** элемент аналогичен домашнему каталогу для относительных ссылок. он влияет только на url-адреса, отправленные с помощью команд сценария, в составе потока содержимого, который воспроизводится в проигрыватель Windows Media.

**Базовый** элемент, определенный как дочерний элемент элемента **ASX** , станет значением по умолчанию для всего метафайла. **Базовый** элемент, определенный в элементе **entry** , переопределяет любой **базовый** элемент в родительском элементе **ASX** (только для этого элемента **entry** ).

> [!Note]  
> если атрибут **HREF** не заканчивается символом/, проигрыватель Windows Media добавляет один в конец строки.

 

## <a name="examples"></a>Примеры


```XML
<BASE HREF="https://msdn.microsoft.com/" />

```
