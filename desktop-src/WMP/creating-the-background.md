# <a name="creating-the-background"></a>Создание фона

Теперь все готово для создания фонового изображения. Обеспечьте видимость слоя Rectangle, но сохраните его позади слоя, на котором выполняется рисование, и используйте его в качестве направляющего для создания изображений для обложки. Вам потребуется собрать каждый элемент в одном или нескольких слоях, чтобы упростить копирование в другой файл позже. Например, кнопка Плайпаусе должна быть скопирована и перецветна в приостановленном файле, отключенном файле и файле региона. При перемещении изображения в файл регионов убедитесь, что у копируемого рисунка нет сложных эффектов, таких как тени или багетные рамки.

Завершив создание нужного изображения, сделайте видимыми все слои, которые необходимо использовать, и скройте другие (например, эскизы и прямоугольники). Затем с помощью команды сохранить копию сохраните иллюстрацию в файл .bmp.

Следующая диаграмма является фоновым изображением.

![фоновый файл](images/ceswmbak.png)

Для создания визуальных эффектов для различных элементов использовались фаски, рельефы и тени. Обратите внимание, что отсутствует кнопка для изображения бегунка тома. Он будет создан позже.

В Photoshop есть несколько визуальных эффектов, которые можно применять к изображениям, что упрощает создание более интересных иллюстраций. Например, кнопка Плайпаусе была создана с помощью простого круга. После этого был создан скос и рельеф с эффектом тени. Сплошная стрелка была простым треугольником, созданным тремя щелчками инструмента «Перо». Затем у стрелки было присвоено внешнее представление с помощью эффекта. При использовании таких программ, как Photoshop, необязательно иметь возможность рисовать. Можно просто создавать простые изображения и применять эффекты. Вы также можете импортировать собственные фотографии или отсканированные иллюстрации.
