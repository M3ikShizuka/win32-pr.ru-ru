# <a name="equalizersettingscurrentspeakername"></a>ЕКУАЛИЗЕРСЕТТИНГС. Куррентспеакернаме

Атрибут **куррентспеакернаме** извлекает имя текущего параметра динамика.

``` syntax
        elementID.currentSpeakerName
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** только для чтения, содержащей одно из следующих значений.



| Значение           | Описание                              |
|-----------------|------------------------------------------|
| Наушники      | Текущие динамики — наушники.     |
| Обычные динамики | Текущие динамики имеют нормальный размер. |
| Крупные динамики  | Текущие динамики имеют большой размер.          |



 

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|---------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media 9 Series или более поздней версии<br/> |
