# <a name="stringcollectionisidentical-method"></a>StringCollection. тождественный метод

Метод « **идентичен** » получает значение, указывающее, совпадает ли указанный объект с текущим.

## <a name="syntax"></a>Синтаксис


```JScript
bRetVal = StringCollection.isIdentical(
  stringCollection
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*stringCollection* \[ окне\]
</dt> <dd>

Объект **StringCollection** для сравнения с текущим объектом.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает **логическое значение**.
