# <a name="onkeypress"></a>OnKeyPress

Обработчик событий **OnKeyPress** обрабатывает событие, возникающее при нажатии буквенно-цифрового ключа.

``` syntax
onkeypress
```

## <a name="remarks"></a>Комментарии

Этот обработчик событий неприменим к элементу **списка воспроизведения** , элементу **Effects** или элементу **Video** при *видео*. **безоконное** значение — false.
