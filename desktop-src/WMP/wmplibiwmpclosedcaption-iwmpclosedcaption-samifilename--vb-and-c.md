# <a name="iwmpclosedcaptionsamifilename-property"></a>Свойство Ивмпклоседкаптион:: Самифиленаме

Свойство **самифиленаме** Возвращает или задает имя файла, содержащего сведения, необходимые для закрытого субтитров.

## <a name="syntax"></a>Синтаксис


```CSharp
public System.String SAMIFileName {get; set;}
```


```VB

Public Property SAMIFileName As System.String
```





## <a name="property-value"></a>Значение свойства

**Строка System. String** , которая является именем синхронизированного доступного файла обмена мультимедиа (SAMI).

## <a name="remarks"></a>Remarks

Файл SAMI должен иметь расширение имени файла. SMI или Sami.

Если не задать значение с помощью **самифиленаме**, это свойство получает **строку** , содержащую имя файла по умолчанию или URL-адрес, связанный с текущим элементом мультимедиа. Такая ассоциация может возникнуть, когда файл SAMI указывается с помощью параметра URL-адреса *Sami* или автоматически, если имя файла Sami совпадает с именем файла цифрового носителя (за исключением расширения имени файла). Если с текущим носителем не связан файл SAMI по умолчанию, это свойство получает строку нулевой длины ("").

Когда вы задаете значение с помощью **самифиленаме**, это значение сохраняется до тех пор, пока не будет задано новое значение (или до открытия нового элемента мультимедиа с помощью параметра URL-адреса SAMI). Поэтому перед воспроизведением каждого нового элемента мультимедиа необходимо задать новое значение для этого свойства. Таким образом, новое значение для **самифиленаме** вступит в силу при открытии следующего элемента мультимедиа (или при вызове **аксвиндовсмедиаплайер. Close** ). Указание нового значения для **самифиленаме** не влияет на текущий носитель.

чтобы проигрыватель Windows Media использовать файл SAMI по умолчанию, связанный с определенным элементом мультимедиа, перед воспроизведением следующего элемента мультимедиа задайте для **самифиленаме** строку нулевой длины ("").
