# <a name="listboxselecteditem"></a>LISTBOX. selectedItem

Атрибут **SelectedItem** указывает или получает индекс элемента, выбранного в элементе управления "список".

``` syntax
        elementID.selectedItem
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**длинное целое**).

## <a name="remarks"></a>Комментарии

Это свойство будет выбирать одну строку в элементе управления "список". Выбор любой другой выбранной строки будет снят.
