# <a name="creating-the-skin-definition-file"></a>Создание файла определения обложки

Файл определения обложки — это простой текстовый файл. Файлы определения обложки можно создавать в любом текстовом редакторе, который сохраняет текстовые файлы в формате ASCII, такие как Microsoft Notepad, Microsoft Visual InterDev® или Microsoft Visual Studio. Поскольку вы будете писать XML, а не HTML, вам может не понадобиться использовать редактор для HTML, так как он может добавлять дополнительные коды, которые вам не нужны, или сообщать вам об ошибках, которые у вас нет. Вы, вероятно, захотите избежать таких текстовых редакторов, как Microsoft Word; Несмотря на то, что Word может сохранять обычные текстовые файлы, может не забывать сохранять текст при каждом сохранении.
