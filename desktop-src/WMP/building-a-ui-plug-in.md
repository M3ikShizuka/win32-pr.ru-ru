# <a name="building-a-ui-plug-in"></a>Создание подключаемого модуля пользовательского интерфейса

Создать подключаемый модуль пользовательского интерфейса можно с помощью Visual Studio и мастера подключаемых модулей проигрывателя Windows Media. Сведения о настройке мастера и создании подключаемого модуля см. в разделе [Создание подключаемого модуля](building-a-plug-in.md).

При запуске мастера вам будет предоставлено несколько различных типов подключаемых модулей. Выберите подключаемый модуль пользовательского интерфейса.

Мастер создает подключаемый модуль пользовательского интерфейса, который можно использовать в качестве отправной точки. В следующем разделе объясняется, как настроить подключаемый модуль для собственных целей.

-   [Настройка подключаемого модуля пользовательского интерфейса](customizing-the-ui-plug-in.md)
