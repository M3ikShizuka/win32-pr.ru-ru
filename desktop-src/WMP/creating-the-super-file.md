# <a name="creating-the-super-file"></a>Создание файла Super

Супер файл содержит отключенный образ для значений TrackBar.

> [!Note]  
> суперпользователи не используются в обложках для проигрыватель Windows Media 10 Mobile или более поздней версии, так как отключенные образы для значений trackbar для поиска находятся в файле сиксумб.

 

На следующем рисунке находится супер-файл.

![Супер файл](images/ceswmsup.png)

Он содержит изображение для отключенного регулятора TrackBar.
