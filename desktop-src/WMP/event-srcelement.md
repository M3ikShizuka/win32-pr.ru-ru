# <a name="eventsrcelement"></a>Event. Срцелемент

Атрибут **срцелемент** извлекает элемент, который вызвал событие.

``` syntax
event.srcElement
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является объектом, предназначенным только для чтения.

|--------------------|------------------------------------------------------|
