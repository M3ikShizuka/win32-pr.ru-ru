# <a name="marquee-location"></a>Расположение области

Необходимо определить расположение поля вывода с областью, которое будет использоваться. Необходимо ввести четыре положительных целых числа, разделенные запятыми, которые определяют левую, верхнюю, ширину и высоту текстового поля, отображаемого в пикселях, относительно фонового изображения.

Типичное значение:


```C++
180,46,50,30 

```
