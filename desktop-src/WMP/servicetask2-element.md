# <a name="servicetask2-element"></a>ServiceTask2, элемент

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Элемент **ServiceTask2** представляет вторую область задач оперативного хранилища.

``` syntax
<ServiceTask2
    URL = "URL"
/>
```

## <a name="attributes"></a>Атрибуты



| Термин                                                                                                                             | Описание                                                        |
|----------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| <span id="URL__required_"></span><span id="url__required_"></span><span id="URL__REQUIRED_"></span>**URL-адрес** (обязательно)<br/> | URL-адрес веб-страницы, отображаемой проигрыватель Windows Media.<br/> |



 

## <a name="parentchild-elements"></a>Родительские и дочерние элементы



| Иерархия       | Элемент                       |
|-----------------|-------------------------------|
| Родительские элементы | **ServiceInfo**               |
| Дочерние элементы  | **ButtonText**, **буттонтип** |



 

## <a name="remarks"></a>Remarks

**ServiceTask1** считается основной областью задач для привлечения к коммерческой деятельности. Это область задач, которая отображается, когда пользователь выбирает приобретать музыку. Используйте **ServiceTask2** для других действий в Интернет-магазине.

> [!Note]  
> в проигрыватель Windows Media 10 есть три области задач, в которых интернет-магазин может отображать свои веб-страницы. Интернет-магазин может использовать одну, две или все три области задач. проигрыватель Windows Media 11 имеет только одну область задач, которую пользователь может просматривать, щелкнув вкладку **интернет-магазины** . проигрыватель Windows Media 11 не учитывает элементы **ServiceTask2** и **ServiceTask3** .

 

Области задач в Интернет-магазинах используют один экземпляр браузера. Это означает, что не следует писать код скрипта на веб-странице, которая будет продолжать выполняться, когда пользователь перейдет из текущей задачи службы.

когда пользователь переключает области задач, проигрыватель Windows Media сохраняет URL-адреса и файлы cookie сеанса. Когда пользователь переключается обратно в область задач, проигрыватель восстанавливает URL-адрес и файлы cookie. Если пользователь выбирает использование другого Интернет-магазина, данные URL и сеанса очищаются.

В следующей таблице приведены параметры, отправленные с помощью запроса URL-адреса. Другие могут присутствовать в целях совместимости с прежними версиями.



| Имя         | Значение                                                                                                                                                               |
|--------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *графическом*      | идентификатор географического расположения Windows. Идентификатор расположения задается пользователем в области **Расположение** в разделе региональные и языковые параметры панели управления. |
| *locale*     | проигрыватель Windows Media код локали.                                                                                                                                     |
| *UserLocale* | Windows код локали. Языковой стандарт задается пользователем в области **стандарты и форматы** языковых и региональных параметров на панели управления.        |
| *version*    | проигрыватель Windows Media номер версии в следующем формате: 10.0. x. xxxx или 11.0. x. xxxx.                                                                         |
