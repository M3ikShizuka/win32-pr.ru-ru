# <a name="iwmpcdromburnburnformat-property"></a>Свойство Ивмпкдромбурн:: Бурнформат

Свойство **бурнформат** возвращает значение, указывающее тип записываемого компакт-диска.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```CSharp
public WMPBurnFormat burnFormat {get;}
```


```VB

Public ReadOnly Property burnFormat As WMPBurnFormat
```





## <a name="property-value"></a>Значение свойства

Объект **вмплиб. вмпбурнформат** , который является значением из перечисления **вмпбурнформат** , которое указывает тип записываемого компакт-диска.
