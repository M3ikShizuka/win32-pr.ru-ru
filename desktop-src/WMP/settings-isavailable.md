# <a name="settingsisavailable"></a>Параметры.

Свойство **Available** указывает, доступен ли указанный тип сведений, или может быть выполнено указанное действие.

## <a name="syntax"></a>Синтаксис

Player. Settings.-Available (имя)

## <a name="parameters"></a>Параметры

*name*

Строка, содержащая одно из следующих значений.



| Строка             | Описание                                                    |
|--------------------|----------------------------------------------------------------|
| автозапуск          | Определяет, можно ли задать свойство "Автозапуск".          |
| Balance            | Определяет, можно ли задать свойство Balance.            |
| BaseURL            | Определяет, можно ли задать свойство baseURL.            |
| дефаултфраме       | Определяет, можно ли задать свойство Дефаултфраме.       |
| енаблиррордиалогс | Определяет, можно ли задать свойство Енаблиррордиалогс. |
| Режим            | Определяет, можно ли вызывать метод метода мода.           |
| инвокеурлс         | Определяет, можно ли задать свойство Инвокеурлс.         |
| Mute               | Определяет, можно ли задать свойство "выключить звук".               |
| плайкаунт          | Определяет, можно ли задать свойство Плайкаунт.          |
| Тариф               | Определяет, можно ли задать свойство Rate.               |
| SetMode            | Определяет, можно ли вызывать метод setMode.           |
| Громкость             | Определяет, можно ли задать свойство "том".             |



 

## <a name="return-values"></a>Возвращаемые значения

Этот метод возвращает **логическое** значение.
