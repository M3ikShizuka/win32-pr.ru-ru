# <a name="about-allocating-streaming-resources"></a>О выделении ресурсов потоковой передачи

в примере подключаемого модуля DSP, созданного мастером подключаемых модулей проигрыватель Windows Media, не требуются дополнительные буферы потоковой передачи. Однако может потребоваться выделить ресурсы памяти для подключаемого модуля DSP. Например, подключаемый модуль, который создает эхо-результат, потребует дополнительного буфера для создания необходимой временной задержки.

Интерфейс **имедиаобжект** содержит два метода для решения этой ситуации. проигрыватель Windows Media вызывает **имедиаобжект:: аллокатестреамингресаурцес** , чтобы предоставить возможность создания любых требуемых буферов. проигрыватель Windows Media позже вызывает метод **имедиаобжект:: фристреамингресаурцес** , чтобы можно было освободить выделенную ранее память. В примере реализации подключаемого модуля DSP также вызывается **фристреамингресаурцес** из *Кпрожектнаме*::**финалрелеасе** , чтобы гарантировать освобождение всех ресурсов до уничтожения объекта подключаемого модуля.
