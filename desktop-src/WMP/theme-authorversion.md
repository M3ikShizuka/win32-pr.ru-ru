# <a name="themeauthorversion"></a>THEME. Аусорверсион

Атрибут **аусорверсион** указывает или получает номер версии обложки, назначенный автором.

``` syntax
theme.authorVersion
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** только для чтения и не имеет значения по умолчанию.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
