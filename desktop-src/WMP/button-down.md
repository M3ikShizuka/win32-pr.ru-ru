# <a name="buttondown"></a>КНОПКА. вниз

Атрибут **Down** указывает или получает значение, указывающее, находится ли **кнопка** в положении "вверх" или "вниз".

``` syntax
        elementID.down
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                                                   |
|-------|---------------------------------------------------------------|
| true  | Указывает, что **кнопка** находится в расположении вниз.        |
| false | По умолчанию. Указывает, что **кнопка** находится в положении вверх. |



 

## <a name="remarks"></a>Комментарии

Чтобы **кнопка** оставалась в положении вниз, необходимо задать для параметра **закрепить** значение true. По умолчанию **закрепление** имеет значение false, и любая попытка **установить значение** true будет пропущена.

Если указано недопустимое значение, сохраняется предыдущее состояние.
