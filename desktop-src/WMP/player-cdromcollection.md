# <a name="playercdromcollection"></a>Player. Кдромколлектион

Свойство Кдромколлектион извлекает объект **кдромколлектион** .

## <a name="syntax"></a>Синтаксис

*проигрыватель* . **кдромколлектион**

## <a name="possible-values"></a>Возможные значения

Это свойство является объектом **кдромколлектион** , предназначенным только для чтения.

## <a name="remarks"></a>Remarks

Чтобы получить значение этого свойства, требуется доступ на чтение к библиотеке. Дополнительные сведения см. в разделе [доступ к библиотеке](library-access.md).
