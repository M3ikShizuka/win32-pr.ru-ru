# <a name="playerwindowlessvideo"></a>Player. Виндовлессвидео

свойство **виндовлессвидео** указывает или получает значение, указывающее, визуализирует ли элемент управления проигрыватель Windows Media видео в режиме без окон.

## <a name="syntax"></a>Синтаксис

*проигрыватель*. **виндовлессвидео**

## <a name="possible-values"></a>Возможные значения

Это **логическое** значение, заданное во время разработки и доступное только для чтения.



| Значение | Описание                                      |
|-------|--------------------------------------------------|
| Да  | Видео отображается в режиме без окон.        |
| false | По умолчанию. Видео отображается в режиме с окнами. |



 

## <a name="remarks"></a>Remarks

по умолчанию встроенный элемент управления проигрыватель Windows Media визуализирует видео в отдельном окне в клиентской области. Если для **виндовлессвидео** задано значение true, элемент управления проигрывателя визуализирует видео непосредственно в клиентской области, поэтому можно применять специальные эффекты или разносить видео с текстом.

в Windows Vista отрисовка видео в режиме без окон может привести к снижению производительности.

Свойство **виндовлессвидео** не поддерживается для Netscape Navigator. Установка значения этого свойства в навигаторе может привести к непредвиденным результатам.

**проигрыватель Windows Media 10 Mobile:** Это свойство не поддерживается.
