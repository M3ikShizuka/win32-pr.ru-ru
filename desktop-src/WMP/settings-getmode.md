# <a name="settingsgetmode-method"></a>метод Параметры. мода

Метод метода **мода** определяет, активен ли режим "цикл" или "случайный".

## <a name="syntax"></a>Синтаксис


```JScript
bRetVal = Settings.getMode(
  modeName
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*моденаме* \[ окне\]
</dt> <dd>

**Строка** , указывающая имя рассматриваемого режима, содержащее одно из следующих значений.



| Строка     | Описание                                                                                                              |
|------------|--------------------------------------------------------------------------------------------------------------------------|
| ауторевинд | Дорожки перезапускаются в начале после воспроизведения в конце.                                                          |
| loop       | Последовательность дорожек повторяется самим собой.                                                                                   |
| шовфраме  | Ближайший ключевой кадр отображается в текущей позиции, когда не воспроизводится. Этот режим не подходит для звуковых дорожек. |
| shuffle    | Дорожки воспроизводятся в случайном порядке.                                                                                       |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает **логическое** значение, указывающее, активен ли заданный режим.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Версия<br/> | для режимов цикла и случайного перебора проигрыватель Windows Media версии 7,0 или более поздней. для режимов ауторевинд и шовфраме проигрыватель Windows Media 9 Series или более поздней версии.<br/> |
