# <a name="using-the-echo-sample-dsp-plug-in"></a>Использование подключаемого модуля "Echo Sample DSP"

при создании без ошибок сборки подключаемого модуля проигрыватель Windows Media echo DSP вы можете использовать его в проигрыватель Windows Media для прослушивания эффекта. при запуске проигрыватель Windows Media подключаемый модуль должен быть включен по умолчанию. Если вы воспроизводите аудио-или видеофайлы с аудио, вы должны слышать один и тот же звук, который воспроизводится на том же томе, что и исходный звук.

Можно поэкспериментировать с различными уровнями времени задержки и эффектов, изменив значения на странице свойств подключаемый модуль Echo. Попробуйте применить различные типы исходных материалов, например музыку или речь. Время задержки 200 мс с уровнем влияния 30 приведет к жестко пробелому, что звучит так, как эффект, используемый на ранних рок и в записи отката. Уровни эффектов, превышающие 50, приводят к тому, что исходный сигнал будет ниже по сравнению с эхо-выводом, который называется действием предварительного эха.
