# <a name="videostretchtofit"></a>ВИДЕО. Стретчтофит

Атрибут **стретчтофит** задает значение, указывающее, будет ли видео растягиваться по ширине и высоте, определенным для элемента управления видео.

``` syntax
        elementID.stretchToFit
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                                             |
|-------|---------------------------------------------------------|
| Да  | Видео будет растягиваться по размеру элемента управления.              |
| false | По умолчанию. Видео не будет растягиваться по размеру элемента управления. |



 

## <a name="remarks"></a>Remarks

Если ширина или высота не указаны, этот атрибут игнорируется.
