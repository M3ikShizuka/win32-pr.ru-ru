# <a name="iwmperroritem-vb-and-c-interface"></a>Интерфейс Ивмперроритем (VB и C#)

Предоставляет доступ к сведениям об ошибках.

Интерфейс **ивмперроритем** предоставляет следующие свойства.



| Свойство                                                                             | Описание                                                                                |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [кустомурл](wmplibiwmperroritem-iwmperroritem-customurl--vb-and-c.md)               | Возвращает URL-адрес сайта, который отображает конкретные сведения о сбое загрузки кодека. |
| [errorCode](wmplibiwmperroritem-iwmperroritem-errorcode--vb-and-c.md)               | Возвращает текущий код ошибки.                                                               |
| [еррорконтекст](wmplibiwmperroritem-iwmperroritem-errorcontext--vb-and-c.md)         | Возвращает значение, указывающее контекст ошибки.                                          |
| [errorDescription](wmplibiwmperroritem-iwmperroritem-errordescription--vb-and-c.md) | Возвращает описание ошибки.                                                           |
| [исправить](wmplibiwmperroritem-iwmperroritem-remedy--vb-and-c.md)                     | Зарезервировано для последующего использования.                                                                   |



 

Получите интерфейс **ивмперроритем** , используя следующее свойство.



| Интерфейс                            | Свойство                                                                   |
|--------------------------------------|----------------------------------------------------------------------------|
| [ивмперрор](iwmperror--vb-and-c.md) | [Item](iwmperror-item--vb-and-c.md) (в C# используется метод **Get \_ Item** ) |



 

## <a name="members"></a>Элементы

Интерфейс **ивмперроритем (VB и C#)** не определяет никаких членов.
