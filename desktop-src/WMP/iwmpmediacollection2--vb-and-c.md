# <a name="iwmpmediacollection2-vb-and-c-interface"></a>Интерфейс IWMPMediaCollection2 (VB и C#)

Предоставляет методы, дополняют интерфейс **ивмпмедиаколлектион** .

## <a name="members"></a>Элементы

Интерфейс **IWMPMediaCollection2 (VB и C#)** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **IWMPMediaCollection2 (VB и C#)** содержит следующие методы.



| Метод                                                                                                                     | Описание                                                                                                                                                              |
|:---------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**createQuery**](wmplibiwmpmediacollection2-iwmpmediacollection2-createquery--vb-and-c.md)                               | Возвращает интерфейс **ивмпкуери** , представляющий новый запрос.<br/>                                                                                               |
| [**жетбяттрибутеандмедиатипе**](wmplibiwmpmediacollection2-iwmpmediacollection2-getbyattributeandmediatype--vb-and-c.md) | Возвращает интерфейс **ивмпплайлист** , предоставляющий доступ к элементам мультимедиа с указанным атрибутом и типом мультимедиа.<br/>                                     |
| [**жетплайлистбикуери**](wmplibiwmpmediacollection2-iwmpmediacollection2-getplaylistbyquery--vb-and-c.md)                 | Возвращает интерфейс **ивмпплайлист** , который предоставляет доступ к элементам мультимедиа, соответствующим условиям запроса.<br/>                                                    |
| [**жетстрингколлектионбикуери**](wmplibiwmpmediacollection2-iwmpmediacollection2-getstringcollectionbyquery--vb-and-c.md) | Возвращает интерфейс **ивмпстрингколлектион** , предоставляющий доступ к набору всех строковых значений для указанного атрибута, соответствующих условиям запроса.<br/> |



 

Получите интерфейс **IWMPMediaCollection2** путем приведения значения, возвращаемого свойством [**аксвиндовсмедиаплайер. медиаколлектион**](axwmplib-axwindowsmediaplayer-mediacollection--vb-and-c.md) , или значения, возвращаемого свойством [**ивмплибрари. медиаколлектион**](wmplibiwmplibrary-iwmplibrary-mediacollection--vb-and-c.md) .
