# <a name="bitmaps-windows-media-player-sdk"></a>точечные рисунки (проигрыватель Windows Media SDK)

Необходимо использовать одно или несколько изображений в обложке, и каждое изображение должно быть определено в файле определения обложки. Если вы не определили образ в этом разделе, Обложка не сможет его использовать.

Термин "точечный рисунок" используется в ссылке в общем смысле и относится к точечным изображениям с помощью расширения .bmp, изображений GIF с расширением .gif, изображений JPEG с расширением .jpg и изображений PNG с расширением .png.

Раздел точечные рисунки файла определения обложки начинается со следующей строки:


```C++
[ Bitmaps ]

```



Затем необходимо добавить одну или несколько строк, содержащих сведения о каждом из изображений в обложке.

Типичная строка может выглядеть следующим образом:


```C++
    Background  Background.bmp  0,0

```



Для раздела точечных рисунков в файле определения обложки можно использовать следующий шаблон:


```C++
//  <Type>      <Filename>      <X,Y>
//  ------      -----------     -----

```



Для получения сведений о точечных рисунках для каждой строки в разделе "точечный рисунок" необходимо использовать следующий порядок. Каждая часть строки является обязательной.

1.  [Тип точечного рисунка](bitmap-type.md)
2.  [Имя файла](file-name.md)
3.  [Положения](coordinates.md)

Пример кода точечного рисунка см. в [разделе пример точечного рисунка](sample-bitmap-section.md).
