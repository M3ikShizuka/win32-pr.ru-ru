# <a name="buttonelementuptooltip"></a>БУТТОНЕЛЕМЕНТ. иссказка

Атрибут исправной **подсказки** указывает или получает текст подсказки, отображаемый при наведении мыши на **Буттонелемент** , а **буттонелемент** находится в состоянии up.

``` syntax
        elementID.upToolTip
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи со значением по умолчанию "" (пустая строка) с максимальной длиной 1024 символов.

## <a name="remarks"></a>Комментарии

Если для этого атрибута задано значение "" (пустая строка), подсказка не отображается.

См. атрибут [маппингколор](buttonelement-mappingcolor.md) для примера, иллюстрирующий использование этого атрибута.
