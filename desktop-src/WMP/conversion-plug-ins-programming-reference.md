# <a name="conversion-plug-ins-programming-reference"></a>Справочник по программированию подключаемых модулей преобразования

пакет средств разработки Microsoft проигрыватель Windows Media Software Development Kit (SDK) поддерживает интерфейс для подключаемых модулей преобразования, которые преобразуют один формат файлов мультимедиа в другой. Пакет SDK также определяет требования для регистрации форматов файлов для преобразования определенным подключаемым модулем. Эти сведения подробно описаны в следующих разделах.



| Section                                                                | Описание                                                                                                                                                                                           |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Регистрация подключаемых модулей преобразования](registering-conversion-plug-ins.md) | Описывает, как зарегистрировать форматы цифровых файлов мультимедиа для преобразования.                                                                                                                                  |
| [Интерфейс Ивмпконверт](/previous-versions/windows/desktop/api/wmpservices/nn-wmpservices-iwmpconvert)                               | предоставляет методы, позволяющие использовать подключаемые модули преобразования проигрыватель Windows Media для преобразования цифровых файлов мультимедиа, созданных с помощью технологий, не предоставляемых корпорацией майкрософт, в расширенный формат систем (ASF). |
