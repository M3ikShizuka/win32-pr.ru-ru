# <a name="conversion-plug-ins-programming-reference"></a>Справочник по программированию подключаемых модулей преобразования

Пакет средств разработки программного обеспечения (SDK) проигрывателя Microsoft Windows Media поддерживает интерфейс для подключаемых модулей преобразования, который преобразует один формат файла цифрового мультимедиа в другой. Пакет SDK также определяет требования для регистрации форматов файлов для преобразования определенным подключаемым модулем. Эти сведения подробно описаны в следующих разделах.



| Section                                                                | Описание                                                                                                                                                                                           |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Регистрация подключаемых модулей преобразования](registering-conversion-plug-ins.md) | Описывает, как зарегистрировать форматы цифровых файлов мультимедиа для преобразования.                                                                                                                                  |
| [Интерфейс Ивмпконверт](/previous-versions/windows/desktop/api/wmpservices/nn-wmpservices-iwmpconvert)                               | Предоставляет методы для включения подключаемых модулей преобразования проигрывателя Windows Media для преобразования цифровых файлов мультимедиа, созданных с помощью технологий, не предоставляемых корпорацией Майкрософт, в расширенный формат систем (ASF). |
