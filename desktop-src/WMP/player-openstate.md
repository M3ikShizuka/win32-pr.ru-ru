# <a name="playeropenstate"></a>Player. Опенстате

Свойство **опенстате** извлекает значение, указывающее состояние источника содержимого.

## <a name="syntax"></a>Синтаксис

*проигрыватель* . **опенстате**

## <a name="possible-values"></a>Возможные значения

Это **значение доступно только для** чтения (**Long**). Константу перечисления в стиле C можно получить с помощью префикса "вмпос" в качестве значения состояния. Например, константа для состояния Плайлистопенинг — **вмпосплайлистопенинг**.



| Значение | Состояние                   | Описание                                                                                                                                            |
|-------|-------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0     | Не определено.               | проигрыватель Windows Media находится в неопределенном состоянии.                                                                                                         |
| 1     | плайлистчангинг        | Будет загружен новый список воспроизведения.                                                                                                                    |
| 2     | плайлистлокатинг        | проигрыватель Windows Media пытается выполнить обнаружение списка воспроизведения. Список воспроизведения может быть локальным (библиотекой или метафайлом с расширением файла. ASX) или удаленным. |
| 3     | плайлистконнектинг      | Подключение к списку воспроизведения.                                                                                                                            |
| 4     | плайлистлоадинг         | Список воспроизведения найден и теперь извлекается.                                                                                                    |
| 5     | плайлистопенинг         | Получен список воспроизведения, который теперь анализируется и загружается.                                                                                        |
| 6     | плайлистопенномедиа     | Открыт список воспроизведения.                                                                                                                                      |
| 7     | плайлистчанжед         | В **куррентплайлист** был назначен новый список воспроизведения.                                                                                               |
| 8     | медиачангинг           | Будет загружен новый элемент мультимедиа.                                                                                                                |
| 9     | медиалокатинг           | проигрыватель Windows Media расположение элемента мультимедиа. Файл может быть локальным или удаленным.                                                                      |
| 10    | медиаконнектинг         | Подключение к серверу, на котором находится элемент мультимедиа.                                                                                                    |
| 11    | медиалоадинг            | Элемент мультимедиа найден и теперь извлекается.                                                                                                |
| 12    | медиаопенинг            | Элемент мультимедиа был извлечен и теперь открыт.                                                                                                 |
| 13    | медиаопен               | Теперь элемент мультимедиа открыт.                                                                                                                                |
| 14    | бегинкодекаккуиситион   | Начинается получение кодека.                                                                                                                            |
| 15    | ендкодекаккуиситион     | Получение кодека завершено.                                                                                                                         |
| 16    | бегинлиценсеаккуиситион | Получение лицензии для воспроизведения содержимого, защищенного DRM.                                                                                                     |
| 17    | ендлиценсеаккуиситион   | Получена лицензия на воспроизведение защищенного содержимого DRM.                                                                                               |
| 18    | бегининдивидуализатион  | Начало индивидуализации DRM.                                                                                                                           |
| 19    | ендиндивидуализатион    | Индивидуальное управление DRM завершено.                                                                                                              |
| 20    | медиаваитинг            | Ожидание элемента мультимедиа.                                                                                                                                |
| 21    | опенингункновнурл       | Открытие URL-адреса неизвестного типа.                                                                                                                    |



 

## <a name="remarks"></a>Remarks

состояния проигрыватель Windows Media не гарантированно выполняются в каком бы то ни было определенном порядке. Кроме того, не все состояния должны происходить во время последовательности событий. Не следует писать код, зависящий от порядка состояний.
