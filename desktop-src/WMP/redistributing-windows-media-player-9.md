# <a name="redistributing-windows-media-player-9-series"></a>распространение серии проигрыватель Windows Media 9

вы можете установить проигрыватель Windows Media 9 Series в Windows XP с помощью одной из следующих программ установки.

-   MPSetupXP.exe
-   MPSetup.exe

> [!Note]  
> MPSetup.exe является надмножеством программы установки MPSetupXP.exe. он содержит файлы, необходимые операционным системам, выпущенным до Windows XP. MPSetup.exe функционально эквивалентен MPSetupXP.exe при запуске в Windows xp, но размер файла программы установки больше, так как он не был оптимизирован для установки в операционных системах Windows XP.

 

вы можете установить проигрыватель Windows Media 9 Series на Windows 98 Second edition, Windows Millennium edition или Windows 2000 с помощью следующей программы установки.

-   MPSetup.exe

Ниже приведен пример командной строки для установки без пользовательского интерфейса и запроса перезагрузки или перезапуска.


```
MPSetup.exe /q:A /c:"setup_wm.exe /Q /R:N /P:#e"
```



> [!Note]  
> параметр/p: \# e указывает, что установочный пакет проигрыватель Windows Media должен кэшироваться во время установки проигрыватель Windows Media. Эта команда используется для управления будущими обновлениями операционной системы. Эта команда должна быть пропущена только корпоративными ИТ администраторами. Единственный случай, когда параметр/P: \# e не должен включаться в командную строку, — если вы владеете целевой системой и уверены, что целевая система никогда не будет обновлена до более поздней операционной системы. например, если вы устанавливаете проигрыватель Windows Media 9 Series на Windows 2000, а компьютер может быть обновлен до Windows XP, необходимо использовать параметр/p: \# e в командной строке. в противном случае после установки Windows XP файлы проигрыватель Windows Media будут перезаписаны файлами для проигрыватель Windows Media для Windows XP.

 

в следующей таблице приведены дополнительные параметры, которые можно использовать с программой установки проигрыватель Windows Media 9 Series.



| Параметр              | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
|------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| /номиграте             | Предотвращение миграции библиотек.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| /нестедресторе         | Создайте вложенную точку восстановления системы. используйте этот параметр, если приложение создает точку восстановления системы для вложения точки восстановления проигрыватель Windows Media в точке восстановления приложения.                                                                                                                                                                                                                                                                                                                             |
| /дисалловсистемресторе | Запретить создание точки восстановления системы. Этот флаг отключит создание точки восстановления системы. В большинстве случаев этот флаг не следует использовать для общего распространения программного обеспечения. его следует использовать только в том случае, если можно явно выбрать от имени конечного пользователя не поддерживать откат файлов проигрыватель Windows Media в более раннюю версию проигрывателя. Этот флаг следует использовать только для корпоративного развертывания или установки изготовителя оборудования (OEM). |



 

## <a name="notes"></a>Примечания

-   В параметрах командной строки учитывается регистр.
-   При подавлении запроса на перезагрузку необходимо проверить раздел реестра Инсталлресулт и выполнить обработку уведомления о перезапуске в вызывающем приложении установки.
-   проигрыватель Windows Media 9 Series также устанавливает среду выполнения Windowsного формата мультимедиа, поэтому не нужно включать пакет распространения проигрыватель Windows Media и пакет распространения среды выполнения Windows media format в одном пакете повторного распространения программного обеспечения. Поэтому при включении MPSetup.exe или MPSetupXP.exe в установку не нужно включать WMFdist.exe.
