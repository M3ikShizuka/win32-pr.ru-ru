# <a name="playlistitemselectedbackgroundfocuslostcolor"></a>Список воспроизведения. Итемселектедбаккграундфокуслостколор

Атрибут **итемселектедбаккграундфокуслостколор** указывает или получает значение, обозначающее цвет текста выбранного элемента в списке воспроизведения.

``` syntax
        elementID.itemSelectedBackgroundFocusLostColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, представляющей цвет. Значение по умолчанию — цвет значка кнопки Windows.
