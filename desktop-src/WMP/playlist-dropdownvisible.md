# <a name="playlistdropdownvisible"></a>Список воспроизведения. Дропдовнвисибле

Атрибут **дропдовнвисибле** указывает или получает значение, указывающее, является ли раскрывающийся список списка воспроизведения видимым.

``` syntax
        elementID.dropDownVisible
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                             |
|-------|-----------------------------------------|
| true  | По умолчанию. Раскрывающийся список является видимым. |
| false | Раскрывающийся список не отображается.      |



 

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
