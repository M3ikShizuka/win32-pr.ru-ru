# <a name="windows-media-player-dsp-plug-ins"></a>Подключаемые модули DSP проигрывателя Windows Media

Проигрыватель Microsoft Windows Media предоставляет архитектуру, позволяющую пользователю устанавливать и активировать подключаемые программы, которые добавляют функции обработки цифровой сигнализации (DSP). Подключаемый модуль DSP — это объект мультимедиа Microsoft DirectX (DMO) или преобразование Media Foundation (MFT), которое подключается к проигрывателю с помощью COM-интерфейсов. Типичным подключаемым модулем DSP может быть аудио-эквалайзер или элемент управления "оттенок видео". В этом разделе пакета SDK содержатся сведения о программировании, необходимые для создания собственного подключаемого модуля DSP.

Документация по подключаемому модулю DSP делится на следующие три раздела.
