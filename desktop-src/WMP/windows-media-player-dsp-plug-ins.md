# <a name="windows-media-player-dsp-plug-ins"></a>проигрыватель Windows Media Подключаемые модули DSP

Microsoft проигрыватель Windows Media предоставляет архитектуру, позволяющую пользователю устанавливать и активировать подключаемые программы, добавляющие функции обработки цифровой сигнализации (DSP). подключаемый модуль DSP — это объект мультимедиа Microsoft DirectX (DMO) или преобразование Media Foundation (MFT), которое подключается к проигрывателю с помощью COM-интерфейсов. Типичным подключаемым модулем DSP может быть аудио-эквалайзер или элемент управления "оттенок видео". В этом разделе пакета SDK содержатся сведения о программировании, необходимые для создания собственного подключаемого модуля DSP.

Документация по подключаемому модулю DSP делится на следующие три раздела.
