# <a name="serviceinfo-document"></a>Документ ServiceInfo

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Интернет-магазины должны создавать документ ServiceInfo.xml. это документ, используемый интернет-магазином для настройки проигрыватель Windows Media пользовательского интерфейса, когда проигрыватель Windows Media 10 или более поздней версии размещается интернет-магазин.

для проигрыватель Windows Media интернет-магазинов поддерживаются следующие элементы и их атрибуты.



| Элемент                                      | Описание                                                                                                                                                                                                                                                                        |
|----------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [албуминфо](albuminfo-element.md)           | указывает URL-адрес веб-страницы, которая проигрыватель Windows Media отображается, когда пользователь выбирает просмотр дополнительных сведений об определенном элементе мультимедиа. Тип 1: необязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: игнорируется<br/>                                |
| [ButtonText](buttontext-element.md)         | задает текстовую строку, которая проигрыватель Windows Media отображается для кнопки области задач. Тип 1: обязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: обязательный<br/>                                                                                             |
| [буттонтип](buttontip-element.md)           | Указывает подсказку, которая отображается, когда пользователь указывает на кнопку области задач. Тип 1: необязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                                         |
| [буйкд](buycd-element.md)                   | указывает url-адреса для веб-страниц, которые проигрыватель Windows Media отображаются, когда пользователь решает выполнить покупку. Тип 1: обязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: игнорируется<br/>                                                                      |
| [Цвет](color-element.md)                   | Задает цвет фона для кнопок навигации в Интернет-магазинах. Тип 1: необязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                                                             |
| [Описание](description-element.md)       | указывает описание интернет-магазина, которое отображается при первом запуске пользователя с установкой проигрыватель Windows Media. требуется проигрыватель Windows Media 11. тип 1: обязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/> |
| [довнлоадстатус](downloadstatus-element.md) | указывает URL-адрес, который проигрыватель Windows Media отображается в виде ссылки, чтобы разрешить пользователям просматривать состояние загрузки. Тип 1: игнорируется<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                         |
| [FriendlyName](friendlyname-element.md)     | задает текстовую строку, которую проигрыватель Windows Media отобразить пользователю для указания интернет-магазина. Тип 1: обязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: обязательный<br/>                                                                           |
| [хтмлвиев](htmlview-element.md)             | Указывает базовый URL-адрес веб-страницы Хтмлвиев. Тип 1: необязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                                                                                   |
| [Изображение](image-element.md)                   | указывает изображения, которые проигрыватель Windows Media отображаются пользователю для представления интернет-магазина. Тип 1: обязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                               |
| [Центр](infocenter-element.md)         | указывает URL-адрес веб-страницы, которая проигрыватель Windows Media отображается в представлении центра сведений в **воспроизведении** , когда интернет-магазин активен. Тип 1: обязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: игнорируется<br/>                           |
| [Установка](install-element.md)               | указывает значения, используемые программой установки проигрыватель Windows Media для установки интернет-магазина по умолчанию. Тип 1: обязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: игнорируется<br/>                                                                                          |
| [Перейти.](navigate-element.md)             | Задает базовый URL-адрес, используемый вызовами метода **External. навигатетаскпанеурл**. Тип 1: необязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                                                          |
| [ServiceInfo](serviceinfo-element.md)       | Представляет элемент Main для ServiceInfo.xml документа. Тип 1: обязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: обязательный<br/>                                                                                                                    |
| [ServiceTask1](servicetask1-element.md)     | Представляет первую область задач оперативного хранилища. Тип 1: обязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: обязательный<br/>                                                                                                                                     |
| [ServiceTask2](servicetask2-element.md)     | Представляет вторую область задач оперативного хранилища. Тип 1: игнорируется<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                                                                                     |
| [ServiceTask3](servicetask3-element.md)     | Представляет третью область задач Интернет-магазина. Тип 1: игнорируется<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                                                                                      |
