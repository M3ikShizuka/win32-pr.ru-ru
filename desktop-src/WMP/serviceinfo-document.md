# <a name="serviceinfo-document"></a>Документ ServiceInfo

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Интернет-магазины должны создавать документ ServiceInfo.xml. Это документ, используемый Интернет-магазином для настройки пользовательского интерфейса проигрывателя Windows Media, когда в проигрывателе Windows Media 10 или более поздней версии размещается Интернет-магазин.

Для Интернет-магазинов проигрывателя Windows Media поддерживаются следующие элементы и их атрибуты.



| Элемент                                      | Описание                                                                                                                                                                                                                                                                        |
|----------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [албуминфо](albuminfo-element.md)           | Задает URL-адрес веб-страницы, отображаемой проигрывателем Windows Media, когда пользователь выбирает просмотр дополнительных сведений об определенном элементе мультимедиа. Тип 1: необязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: игнорируется<br/>                                |
| [ButtonText](buttontext-element.md)         | Задает текстовую строку, отображаемую проигрывателем Windows Media для кнопки области задач. Тип 1: обязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: обязательный<br/>                                                                                             |
| [буттонтип](buttontip-element.md)           | Указывает подсказку, которая отображается, когда пользователь указывает на кнопку области задач. Тип 1: необязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                                         |
| [буйкд](buycd-element.md)                   | Указывает URL-адреса для веб-страниц, отображаемых проигрывателем Windows Media, когда пользователь решает выполнить покупку. Тип 1: обязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: игнорируется<br/>                                                                      |
| [Цвет](color-element.md)                   | Задает цвет фона для кнопок навигации в Интернет-магазинах. Тип 1: необязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                                                             |
| [Описание](description-element.md)       | Задает описание Интернет-магазина, которое отображается при первом запуске пользователя с установкой проигрывателя Windows Media. Требуется проигрыватель Windows Media 11. тип 1: обязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/> |
| [довнлоадстатус](downloadstatus-element.md) | Указывает URL-адрес, который проигрыватель Windows Media отображает как ссылку, чтобы разрешить пользователям просматривать состояние скачивания. Тип 1: игнорируется<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                         |
| [FriendlyName](friendlyname-element.md)     | Задает текстовую строку, которую проигрыватель Windows Media отображает пользователю для обнаружения интернет-магазина. Тип 1: обязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: обязательный<br/>                                                                           |
| [хтмлвиев](htmlview-element.md)             | Указывает базовый URL-адрес веб-страницы Хтмлвиев. Тип 1: необязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                                                                                   |
| [Изображение](image-element.md)                   | Указывает изображения, которые проигрыватель Windows Media отображает пользователю для представления Интернет-магазина. Тип 1: обязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                               |
| [Центр](infocenter-element.md)         | Указывает URL-адрес веб-страницы, отображаемой проигрывателем Windows Media в представлении информационного центра, который **воспроизводится** , когда Интернет-магазин активен. Тип 1: обязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: игнорируется<br/>                           |
| [Установка](install-element.md)               | Указывает значения, используемые программой установки проигрывателя Windows Media для установки Интернет-магазина по умолчанию. Тип 1: обязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: игнорируется<br/>                                                                                          |
| [Осуществляется](navigate-element.md)             | Задает базовый URL-адрес, используемый вызовами метода **External. навигатетаскпанеурл**. Тип 1: необязательный<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                                                          |
| [ServiceInfo](serviceinfo-element.md)       | Представляет элемент Main для ServiceInfo.xml документа. Тип 1: обязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: обязательный<br/>                                                                                                                    |
| [ServiceTask1](servicetask1-element.md)     | Представляет первую область задач оперативного хранилища. Тип 1: обязательный<br/> Тип 2 музыка: требуется<br/> Тип 2 Commerce: обязательный<br/>                                                                                                                                     |
| [ServiceTask2](servicetask2-element.md)     | Представляет вторую область задач оперативного хранилища. Тип 1: игнорируется<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                                                                                     |
| [ServiceTask3](servicetask3-element.md)     | Представляет третью область задач Интернет-магазина. Тип 1: игнорируется<br/> Тип 2 музыка: необязательно<br/> Тип 2 Commerce: необязательный<br/>                                                                                                                                      |
