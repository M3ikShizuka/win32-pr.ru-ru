# <a name="playlistarraycount"></a>Плайлистаррай. Count

Свойство **Count** Извлекает число списков воспроизведения в массиве списков воспроизведения.

## <a name="syntax"></a>Синтаксис

Player. Плайлистколлектион. Жеталл (). Count

## <a name="possible-values"></a>Возможные значения

Это свойство является **числом** только для чтения (**длинное целое**).

## <a name="remarks"></a>Комментарии

Чтобы получить значение этого свойства, требуется доступ на чтение к библиотеке. Дополнительные сведения см. в разделе [доступ к библиотеке](library-access.md).
