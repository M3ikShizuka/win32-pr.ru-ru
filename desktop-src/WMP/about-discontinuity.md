# <a name="about-discontinuity"></a>О прекращении поддержки

в любое время проигрыватель Windows Media может сообщить о разрыве во входном потоке, вызвав метод **имедиаобжект::D методом непрерывности** . Это происходит в начале и в конце потока, а также до каждой операции поиска или при прерывании потокового содержимого по какой-либо причине. пример подключаемого модуля DSP, создаваемого мастером подключаемых модулей проигрыватель Windows Media, не требует обработки прекращений по следующим причинам:

-   Примеры PCM являются атомарными, то есть их можно обрабатывать без учета других примеров в потоке. Некоторые видеоформаты содержат данные, зависящие от ключевых кадров и сжатых образцов.
-   Пример кода написан так, чтобы клиент всегда обрабатывал все выходные данные до того, как подключаемый модуль будет принимать больше входных данных.

Реализация Имедиаобжект по умолчанию **::D непрерывность** просто возвращает значение S \_ ОК.
