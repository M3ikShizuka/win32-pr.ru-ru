# <a name="co-branding-online-stores"></a>Co-Branding Интернет-магазинов

Производители персональных компьютеров, не работающие с музыкальным магазином, могут использовать фирменную символику с поставщиками Интернет-магазина. проигрыватель Windows Media программа установки поддерживает параметр командной строки сервицеекстра, чтобы предоставить изготовителям оборудования указание настраиваемого атрибута, который интернет-магазин может использовать для указания того, какой поставщик оборудования установил первоначальное интернет-магазин.

например, если изготовитель компьютера с именем Fabrikam хочет установить исходное интернет-магазин в музыкальное хранилище Proseware, для установки проигрыватель Windows Media 10 может использоваться следующая командная строка:


```C++
MP10Setup.exe /q:A /c:"setup_wm.exe /Q /R:N /DefaultService:Proseware /ServiceInfo:c:\Proseware\service_info_local.xml /ServiceExtra:OEM=Fabrikam"
```



когда проигрыватель Windows Media устанавливается таким образом, проигрыватель добавляет строку сервицеекстра каждый раз при открытии службы Proseware. в следующем примере показан URL-адрес, который проигрыватель Windows Media отправить в службу Proseware для получения документа ServiceInfo:


```C++
https://www.proseware.com/XML/serviceinfo.asp?OEM=Fabrikam
```



в Proseware online store можно использовать строку запроса для определения того, какая OEM-версия установлена проигрыватель Windows Media и возвращать динамически созданный документ ServiceInfo, указывающий на фирменную версию интернет-магазина.
