# <a name="eventscreenheight"></a>Event. Скринхеигхт

Атрибут **скринхеигхт** извлекает высоту доступного размера экрана в пикселях.

``` syntax
event.screenHeight
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** только для чтения (**длинное целое**).

## <a name="remarks"></a>Комментарии

Это полезно для определения объема пространства в мониторе. При наличии двух мониторов он вычисляет пространство для обоих мониторов.
