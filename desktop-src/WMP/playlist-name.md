# <a name="playlistname"></a>Playlist.name

`name`Свойство указывает или получает имя списка воспроизведения.

## <a name="syntax"></a>Синтаксис

*проигрыватель*. *куррентплайлист*. **имя**

## <a name="possible-values"></a>Возможные значения

Это свойство является **строкой** для чтения и записи.

## <a name="remarks"></a>Remarks

Чтобы получить значение этого свойства, требуется доступ на чтение к библиотеке. Чтобы указать значение, требуется полный доступ. Дополнительные сведения см. в разделе [доступ к библиотеке](library-access.md).

**проигрыватель Windows Media 10 Mobile:** Это свойство доступно только для чтения.
