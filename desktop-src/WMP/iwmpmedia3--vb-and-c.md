# <a name="iwmpmedia3-vb-and-c-interface"></a>Интерфейс IWMPMedia3 (VB и C#)

Предоставляет дополнительные методы для доступа к свойствам элемента мультимедиа.

## <a name="members"></a>Элементы

Интерфейс **IWMPMedia3 (VB и C#)** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **IWMPMedia3 (VB и C#)** содержит следующие методы.



| Метод                                                                                           | Описание                                                                                            |
|:-------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------|
| [**жетаттрибутекаунтбитипе**](wmplibiwmpmedia3-iwmpmedia3-getattributecountbytype--vb-and-c.md) | Возвращает число атрибутов, связанных с указанным типом атрибута.<br/>              |
| [**жетитеминфобитипе**](wmplibiwmpmedia3-iwmpmedia3-getiteminfobytype--vb-and-c.md)             | Возвращает значение атрибута, соответствующего указанному типу атрибута и индексу.<br/> |



 

Получите интерфейс **IWMPMedia3** путем приведения значения, возвращаемого одним из следующих свойств или методов, доступ к которым осуществляется через следующий объект или интерфейс.



| Объект или интерфейс                                               | Свойство или метод                                                                                                                |
|-------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [ивмпконтролс](iwmpcontrols--vb-and-c.md)                        | [currentItem](wmplibiwmpcontrols-iwmpcontrols-currentitem--vb-and-c.md)                                                          |
| [аксвиндовсмедиаплайер](axwindowsmediaplayer-object--vb-and-c.md) | [куррентмедиа](axwmplib-axwindowsmediaplayer-currentmedia--vb-and-c.md) , [невмедиа](axwmplib-axwindowsmediaplayer-newmedia.md) |
| [ивмпплайлист](iwmpplaylist--vb-and-c.md)                        | [Item](iwmpplaylist-item--vb-and-c.md) ( **Получение \_ элемента** в C#)                                                                   |
