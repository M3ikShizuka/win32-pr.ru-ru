# <a name="themeversion"></a>Тема. версия

атрибут **version** указывает или получает номер версии проигрыватель Windows Media, для которой была создана оболочка.

``` syntax
theme.version
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** (**float**). Указание значения для этого атрибута во время разработки или во время выполнения не оказывает никакого влияния. Значение всегда равно 1,0.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
