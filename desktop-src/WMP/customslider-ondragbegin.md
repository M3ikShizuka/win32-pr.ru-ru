# <a name="customsliderondragbegin"></a>КУСТОМСЛИДЕР. Ондрагбегин

Обработчик событий **ондрагбегин** обрабатывает событие, возникающее, когда пользователь нажимает и удерживает левую кнопку мыши вниз и начинает перетаскивать указатель мыши.

``` syntax
onDragBegin
```

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
