# <a name="viewbackgroundtiled"></a>VIEW. Баккграундтилед

Атрибут **баккграундтилед** указывает или получает значение, указывающее, производится ли мозаичное заполнение фонового изображения **представления** или **подокна.**

``` syntax
        elementID.backgroundTiled
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                                        |
|-------|----------------------------------------------------|
| Да  | Изображение повторяется горизонтально и вертикально. |
| false | По умолчанию. Образ не повторяется.                |
