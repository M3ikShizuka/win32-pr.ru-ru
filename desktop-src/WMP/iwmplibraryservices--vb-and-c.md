# <a name="iwmplibraryservices-vb-and-c-interface"></a>Интерфейс Ивмплибрарисервицес (VB и C#)

Предоставляет методы для перечисления библиотек.

## <a name="members"></a>Элементы

Интерфейс **ивмплибрарисервицес (VB и C#)** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмплибрарисервицес (VB и C#)** содержит следующие методы.



| Метод                                                                                               | Описание                                                                                                        |
|:-----------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------|
| [**жеткаунтбитипе**](wmplibiwmplibraryservices-iwmplibraryservices-getcountbytype--vb-and-c.md)     | Возвращает число доступных библиотек указанного типа.<br/>                                           |
| [**жетлибрарибитипе**](wmplibiwmplibraryservices-iwmplibraryservices-getlibrarybytype--vb-and-c.md) | Возвращает интерфейс **ивмплибрари** , представляющий библиотеку с указанным типом и индексом.<br/> |



 

Получите интерфейс **ивмплибрарисервицес** путем приведения объекта, возвращенного методом **аксвиндовсмедиаплайер. жетоккс** .
