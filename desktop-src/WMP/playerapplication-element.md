# <a name="playerapplication-element"></a>ПЛАЙЕРАППЛИКАТИОН, элемент

Элемент **плайераппликатион** может использоваться определенными обложками для доступа к объекту **плайераппликатион** . Этот элемент предоставляет аналогичные функции для глобального атрибута **плайераппликатион** . этот элемент используется только обложками, которые отображаются приложениями C++, которые внедряют удаленный экземпляр элемента управления проигрыватель Windows Media.

Элемент **плайераппликатион** поддерживает следующие атрибуты.



| attribute                                              | Описание                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [хасдисплай](playerapplication-hasdisplay-scr.md)     | получает значение, указывающее, может ли видео отображаться с помощью удаленного элемента управления проигрыватель Windows Media. |
| [плайердоккед](playerapplication-playerdocked-scr.md) | извлекает значение, указывающее, находится ли проигрыватель Windows Media в закрепленном состоянии.                          |
