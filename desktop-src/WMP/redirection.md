# <a name="redirection"></a>Перенаправление

Список воспроизведения перенаправит браузер в проигрыватель Windows Media для воспроизведения указанного потока мультимедиа или файла мультимедиа. Самый простой список воспроизведения метафайлов содержит только универсальный указатель ресурсов (URL-адрес) файла мультимедиа.

Чтобы создать базовый список воспроизведения метафайлов, откройте любой текстовый редактор, например Microsoft Notepad, и введите следующий пример кода.


```XML
<ASX version="3.0">
   <ENTRY>
      <REF HREF="PathToYourFile"/>
   </ENTRY>
</ASX>

```



Замените допустимый путь к файлу Windows Media, используя синтаксис, приведенный в следующей таблице.



| Источник содержимого                                                                                 | Синтаксис                                    |
|---------------------------------------------------------------------------------------------------|-------------------------------------------|
| Содержимое — это файл на сервере Windows Media                                                       | mms://ServerName/Path/FileName.wma        |
| Содержимое — это широковещательная многоадресная рассылка, доступ к которой осуществляется с помощью станции Windows Media.                    | https://WebServerName/Stations/kxyz.nsc    |
| Содержимое — это одноадресная рассылка, доступ к которой осуществляется из точки публикации на сервере Windows Media. | mms://ServerName/PublishingPointAlias     |
| Содержимое — это файл на веб-сервере.                                                                 | https://WebServerName/Path/Filename.wma    |
| Содержимое — это файл на локальном жестком диске                                                            | file://c: \\ путь \\ имя_файла. WMA             |
| Содержимое — это файл в общей сетевой папке.                                                              | file:// \\ \\ имя_сервера \\ путь \\ имя_файла. WMA |
| Содержимое — это файл на сервере Windows Media                                                       | mms://ServerName/Path/FileName.wmv        |



 

Сохраните созданный файл с именем файла и расширением, как описано в разделе [расширения имен файлов](file-name-extensions.md). Дважды щелкните его в проводнике Windows. Проигрыватель Windows Media должен открыть и начать потоковую передачу содержимого. Вы можете сохранить файл на веб-сервере, а также создать ссылку на него с помощью элемента **href** , или внедрить его в страницу с использованием тега **объекта** проигрывателя Windows Media.
