# <a name="redirection"></a>Перенаправление

список воспроизведения перенаправит браузер в проигрыватель Windows Media для воспроизведения указанного потока мультимедиа или файла мультимедиа. Самый простой список воспроизведения метафайлов содержит только универсальный указатель ресурсов (URL-адрес) файла мультимедиа.

чтобы создать базовый список воспроизведения метафайлов, откройте любой текстовый редактор, например Microsoft Блокнот, и введите следующий пример кода.


```XML
<ASX version="3.0">
   <ENTRY>
      <REF HREF="PathToYourFile"/>
   </ENTRY>
</ASX>

```



замените допустимый путь к файлу Windows Media, используя синтаксис, приведенный в следующей таблице.



| Источник содержимого                                                                                 | Синтаксис                                    |
|---------------------------------------------------------------------------------------------------|-------------------------------------------|
| содержимое — это файл на Windows сервере мультимедиа.                                                       | mms://ServerName/Path/FileName.wma        |
| содержимое — это широковещательная многоадресная рассылка, доступ к которой осуществляется с Windows станции мультимедиа.                    | https://WebServerName/Stations/kxyz.nsc    |
| содержимое — это одноадресная рассылка, доступ к которой осуществляется из точки публикации на Windows сервере мультимедиа. | mms://ServerName/PublishingPointAlias     |
| Содержимое — это файл на веб-сервере.                                                                 | https://WebServerName/Path/Filename.wma    |
| Содержимое — это файл на локальном жестком диске                                                            | file://c: \\ путь \\ имя_файла. WMA             |
| Содержимое — это файл в общей сетевой папке.                                                              | file:// \\ \\ имя_сервера \\ путь \\ имя_файла. WMA |
| содержимое — это файл на Windows сервере мультимедиа.                                                       | mms://ServerName/Path/FileName.wmv        |



 

Сохраните созданный файл с именем файла и расширением, как описано в разделе [расширения имен файлов](file-name-extensions.md). дважды щелкните его в обозревателе Windows. проигрыватель Windows Media должны открыть и начать потоковую передачу содержимого. вы можете сохранить файл на веб-сервере, а также создать ссылку на него с помощью элемента **HREF** , или внедрить его на страницу с использованием тега **объекта** проигрыватель Windows Media.
