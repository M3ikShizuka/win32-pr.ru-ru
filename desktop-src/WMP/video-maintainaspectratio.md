# <a name="videomaintainaspectratio"></a>ВИДЕО. Маинтаинаспектратио

Атрибут **маинтаинаспектратио** указывает или получает значение, указывающее, будет ли видео сохранять пропорции при попытке разместить его в пределах ширины и высоты, определенных для элемента управления.

``` syntax
        elementID.maintainAspectRatio
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                                              |
|-------|----------------------------------------------------------|
| true  | По умолчанию. При изменении размера видео сохраняет его пропорции. |
| false | При изменении размера видео не сохраняет его пропорции.  |

|--------------------|------------------------------------------------------|
