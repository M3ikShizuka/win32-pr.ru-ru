# <a name="metadata-area-plug-ins-deprecated"></a>Подключаемые модули области метаданных (не рекомендуется)

Эта страница описывает функцию, которая может быть недоступна в будущих версиях проигрывателя Windows Media и Windows Media Player SDK.

Подключаемые модули пользовательского интерфейса области метаданных отображаются в области "метаданные" в области " **Проигрывается** " справа от основной визуальной области и панели инструментов, а также под представлением списка воспроизведения (если оно включено). В каждый момент времени может быть включен только один подключаемый модуль области метаданных.

Подключаемые модули области метаданных полезны для отображения кратких сведений о воспроизводимом в данный момент элементе мультимедиа, простых элементах управления или гиперссылках на дополнительные сведения.
