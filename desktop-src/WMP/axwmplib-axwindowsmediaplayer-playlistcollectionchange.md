# <a name="playlistcollectionchange-event-of-the-axwindowsmediaplayer-object"></a>Событие Плайлистколлектиончанже объекта Аксвиндовсмедиаплайер

Событие Плайлистколлектиончанже возникает при изменении в коллекции списков воспроизведения.

``` syntax
[C#]
private void player_PlaylistCollectionChange(
  object sender,
  System.EventArgs e
)

[Visual Basic]
Private Sub player_PlaylistCollectionChange(
  sender As Object,
  e As System.EventArgs
) Handles player.PlaylistCollectionChange
```

## <a name="event-data"></a>Данные о событиях

Это событие не содержит данных.
