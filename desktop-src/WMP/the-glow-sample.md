# <a name="the-glow-sample"></a>Пример свечения

В этом разделе описывается создание простой визуализации с именем свечение. Используются три предустановки: красный, зеленый и синий. Был выполнен запуск мастера настраиваемой визуализации и выбрано имя свечения.

Свечение просто изменяет цвет по времени на музыку. Световые перья, так как звуковая сила волны иногда пересекает ноль в момент создания моментального снимка. Этот образец выбран, поскольку он является одним из простейших примеров.
