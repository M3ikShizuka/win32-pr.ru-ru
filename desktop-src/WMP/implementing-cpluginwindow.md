# <a name="implementing-cpluginwindow"></a>Реализация Кплугинвиндов

Класс Кплугинвиндов предоставляет пользовательский интерфейс для подключаемого модуля. В случае подключаемого модуля пользовательского интерфейса поиска этот класс содержит код для кнопки **поиска** и код, запускающий страницу поиска при нажатии кнопки.

Мастер предоставляет базовую реализацию Кплугинвиндов в файле заголовка Кплугинвиндов. h. Чтобы упростить работу, подключаемый модуль пользовательского интерфейса поиска изменит этот файл напрямую, хотя расширенные добавления обычно помещаются в отдельный файл Кплугинвиндов. cpp.

В следующих разделах описаны действия, которые необходимо выполнить для реализации Кплугинвиндов:

-   [Схема сообщений](the-message-map.md)
-   [Конструктор](the-constructor.md)
-   [Метод onpain](the-onpaint-method.md)
-   [Метод onCreate](the-oncreate-method.md)
-   [Метод onSearch](the-onsearch-method.md)
-   [Метод Лаунчпаже](the-launchpage-method.md)
