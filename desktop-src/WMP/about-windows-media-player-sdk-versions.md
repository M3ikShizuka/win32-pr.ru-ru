# <a name="about-windows-media-player-sdk-versions"></a>сведения о версиях пакета SDK проигрыватель Windows Media

текущей версией пакета SDK для проигрыватель Windows Media является версия 12. пакет sdk для проигрыватель Windows Media 12 основан на функциях, появившихся в предыдущих версиях пакета sdk для проигрыватель Windows Media, начиная с версии 7. сведения в этой документации включают функции следующих версий пакета SDK проигрыватель Windows Media:

-   пакет SDK для проигрыватель Windows Media 7
-   пакет SDK для проигрыватель Windows Media 7,1
-   пакет SDK для проигрыватель Windows Media 9 Series
-   пакет SDK для проигрыватель Windows Media 10
-   пакет SDK для проигрыватель Windows Media 11
-   пакет SDK для проигрыватель Windows Media 12

каждая версия пакета SDK связана с выпуском проигрыватель Windows Media. например, в пакете SDK для проигрыватель Windows Media 10 появились новые функции, поддерживаемые проигрыватель Windows Media 10. обратите внимание, что обновление пакета SDK проигрыватель Windows Media не было выпущено проигрыватель Windows Media для Windows XP.

В документации для каждого компонента предоставляются требования к версии, где это необходимо. если сведения о конкретной версии отсутствуют, эта функция появилась в пакете SDK для проигрыватель Windows Media 7 и поддерживается во всех последующих версиях.

сведения о том, как обновить проекты, основанные на пакете SDK проигрыватель Windows Media 6,4, чтобы использовать объектную модель для проигрыватель Windows Media 7 или более поздней версии, см. в разделе [руководство по переносу объектной модели](object-model-migration-guide.md).
