# <a name="effectswindowed"></a>ЭФФЕКТЫ. окна

Атрибут **Window** указывает или получает значение, указывающее, будет ли визуализация отображаться в окне или без окон, то есть независимо от того, будет ли весь прямоугольник элемента управления видимым в любое время или же он может быть обрезан. Этот атрибут можно задать только во время разработки.

``` syntax
        elementID.windowed
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим** , заданным во время разработки, и затем доступен только для чтения.



| Значение | Описание                              |
|-------|------------------------------------------|
| Да  | Элемент управления будет отображаться в окне.            |
| false | По умолчанию. Элемент управления будет безоконным. |



 

## <a name="remarks"></a>Remarks

Если требуется Непрямоугольное окно визуализации или если какая-либо часть окна охватывается изображением, этот атрибут должен иметь значение false. Это потребует некоторой производительности для выполнения необходимой обрезки.

Если для **окон** задано значение true, все изображения, охватывающие окно визуализации, игнорируются, а в окне визуализации отображается z-порядок самого высокого уровня.
