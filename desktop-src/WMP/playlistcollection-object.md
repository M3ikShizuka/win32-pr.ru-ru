# <a name="playlistcollection-object"></a>Объект Плайлистколлектион

Объект **плайлистколлектион** предоставляет способ организации списков воспроизведения.

Объект **плайлистколлектион** поддерживает следующие методы.



| Метод                                                  | Описание                                                                                                              |
|---------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [жеталл](playlistcollection-getall.md)                 | Извлекает объект [плайлистаррай](playlistarray-object.md) , содержащий все списки воспроизведения в библиотеке.                |
| [жетбинаме](playlistcollection-getbyname.md)           | Извлекает объект [плайлистаррай](playlistarray-object.md) , содержащий списки воспроизведения с указанным именем, если таковые имеются. |
| [импортплайлист](playlistcollection-importplaylist.md) | Добавляет статический список воспроизведения в библиотеку.                                                                                   |
| [isDeleted](playlistcollection-isdeleted.md)           | Получает значение, указывающее, находится ли указанный список воспроизведения в папке удаленных элементов.                              |
| [невплайлист](playlistcollection-newplaylist.md)       | Создает новый список воспроизведения в библиотеке.                                                                                   |
| [remove](playlistcollection-remove.md)                 | Удаляет список воспроизведения из библиотеки.                                                                                     |
| [сетделетед](playlistcollection-setdeleted.md)         | Перемещает список воспроизведения в папку Удаленные элементы.                                                                            |



 

Доступ к объекту **плайлистколлектион** осуществляется через следующее свойство.



| Объект                      | Свойство                                            |
|-----------------------------|-----------------------------------------------------|
| [Игрок](player-object.md) | [плайлистколлектион](player-playlistcollection.md) |
