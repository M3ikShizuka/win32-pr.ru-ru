# <a name="changing-the-sample-audio-dsp-plug-in-property"></a>Изменение примера свойства подключаемого модуля DSP аудиоподсистемы

Возможно, потребуется изменить свойство, создаваемое мастером подключаемых модулей проигрывателя Windows Media по умолчанию. В следующем списке приведены элементы, которые могут потребоваться изменить:

-   **Ресурс диалогового окна.** Перейдите на вкладку **ресаурцевиев** в окне рабочей области проекта. Разверните список папок, чтобы открыть папку диалогового окна. Дважды щелкните ресурс диалогового окна, чтобы открыть редактор ресурсов. Вы можете внести изменения в диалоговое окно страницы свойств, чтобы удовлетворить ваши потребности. Например, можно изменить текст в метке или заменить элемент управления "поле ввода" флажком.
-   **Объектный код страницы свойств.** Реализация по умолчанию использует переменную типа Double для хранения коэффициента масштабирования. Может потребоваться другой тип данных. Это также потребует изменения кода, сохраняющего данные в реестре, и считывания данных из реестра (включая код, считывающий данные из реестра в *кпрожектнаме*::**финалконструкт**).
-   **Переменная члена, в которой хранится значение свойства.** Эта переменная называется "m \_ фскалефактор" и объявлена как тип Double. Может потребоваться изменить имя и тип этой переменной по всему проекту.
-   **Методы get и Property.** Может потребоваться изменить имена, параметры и реализации этих методов. Не забывайте также отражать эти изменения в других местах проекта. Например, страница свойств **Apply** вызывает метод *кпрожектнаме*:: Where **\_ Scale**.
