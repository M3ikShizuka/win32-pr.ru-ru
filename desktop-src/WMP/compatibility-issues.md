# <a name="compatibility-issues"></a>Проблемы совместимости

## <a name="netscape-support"></a>Поддержка Netscape

Веб-страницы, открытые функцией Хтмлвиев, отображаются с помощью встроенного экземпляра Microsoft Internet Explorer. Для содержимого Хтмлвиев не требуется совместимость с Netscape.

## <a name="no-htmlview-support-in-earlier-versions-of-windows-media-player"></a>Отсутствие поддержки Хтмлвиев в более ранних версиях проигрывателя Windows Media

В настоящее время компонент Хтмлвиев поддерживается только проигрывателем Windows Media 9 Series или более поздней версии. В более ранних версиях проигрывателя и версий, созданных для других платформ, параметр Хтмлвиев игнорируется при анализе файла ASX. В этом случае мультимедийное содержимое воспроизводится так, как ожидалось, но веб-содержимое не отображается в **воспроизводимой** функции.
