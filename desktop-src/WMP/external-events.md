# <a name="external-events"></a>Внешние события

Когда пользователь нажимает кнопку или нажимает клавишу, вы можете реагировать на свои входные данные с помощью обработчиков событий. Обработчик событий — это раздел кода, который выполняется при каждом срабатывании события.

Элементы обложки поддерживают следующие события:

-   **загрузка**
-   **close**
-   **изменить размер**
-   **активности**
-   **Откройте**
-   **кнопки**
-   **error**
-   **вниз**
-   **Кнопка**
-   **событие**
-   **MouseOver**
-   **мышь**
-   **бытии**
-   **клавиш**
-   **KeyUp**

Дополнительные сведения о конкретных событиях см. в [справочнике по программированию обложки](skin-programming-reference.md) .

Типичный обработчик внешних событий будет наименовать событие и определить код, который будет выполняться. например, если требуется создать код для запуска проигрыватель Windows Media при нажатии пользователем кнопки, в код кнопки следует поместить следующую строку.


```C++
onclick = "JScript: player.URL = 'https://proseware.com/laure.wma' ; "

```



Будет воспроизведен файл с именем Лауре. WMA. Обратите внимание, что вы добавляете слово «on» к конкретным событиям.
