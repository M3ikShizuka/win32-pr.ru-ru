# <a name="dvdisavailable"></a>Доступный DVD-диск.

Свойство **Available** указывает, доступен ли указанный тип сведений, или может быть выполнено указанное действие.

``` syntax
player.dvd.isAvailable(
        name
        )
```

## <a name="parameters"></a>Параметры

*name*

**Строка** , содержащая одно из следующих значений.



| Строка     | Описание                                                                            |
|------------|----------------------------------------------------------------------------------------|
| Назад       | Определяет, доступен ли метод **Back** .                                   |
| оптически        | Определяет, загружен ли DVD-диск.                                                  |
| двддекодер | Определяет, установлен ли декодер DVD в системе.                             |
| resume     | Определяет, доступен ли метод **Resume** .                                 |
| титлемену  | Определяет, доступен ли метод **титлемену** .                              |
| топмену    | Определяет, доступен ли метод **топмену** . Обычно называется корневым меню. |



 

## <a name="return-values"></a>Возвращаемые значения

Этот метод возвращает **логическое значение** , доступное только для чтения, указывающее, доступен ли указанный параметр.

## <a name="remarks"></a>Remarks

функции dvd проигрыватель Windows Media не будут работать на компьютерах, на которых не установлены сторонние декодеры dvd. Можно определить, доступен ли декодер, **вызвав «** двддекодер».

Каждый DVD-диск создан по-другому. Методы, доступные во время воспроизведения DVD-диска и навигации, зависят от способа создания DVD-диска.

**проигрыватель Windows Media 10 Mobile:** Это свойство всегда возвращает **значение false**.
