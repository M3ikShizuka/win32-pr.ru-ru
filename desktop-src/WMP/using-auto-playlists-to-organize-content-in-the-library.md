# <a name="using-auto-playlists-to-organize-content-in-the-library"></a>Использование списков автоматического воспроизведения для организации содержимого в библиотеке

Вы можете использовать автоматические списки воспроизведения для Организации предоставляемого вами содержимого Premium. например, можно использовать проигрыватель Windows Media для создания автоматического списка воспроизведения, который содержит только указанное содержимое с оценкой по крайней мере на четыре звезды. затем можно добавить автоматический список воспроизведения в библиотеку в проигрыватель Windows Media, чтобы он отображался в узле "приобретенная музыка" в узле распространителя содержимого.

Для этого выполните следующие действия:

1.  Создайте Автоматический список воспроизведения.
2.  с помощью обозревателя Windows перейдите к автоматическому списку воспроизведения и извлеките файл wpl.
3.  с помощью объектной модели проигрыватель Windows Media добавьте автоматический список воспроизведения в библиотеку.
4.  Задайте атрибут **WM/контентдистрибутор** для списка воспроизведения в качестве имени ключа распространителя содержимого.
5.  Установите атрибут **синконли** для списка воспроизведения в значение true.

в следующем JScript примере кода показана функция, которая добавляет автоматический список воспроизведения с именем "избранные посещения" в узел Proseware в библиотеке:


```C++
function AddWPL()
{
    var pl = Player.mediaCollection.add("c:\\media\\Favorite Hits.wpl");
    pl.setItemInfo("WM/ContentDistributor", "Proseware");
    pl.setItemInfo("SyncOnly", true);
}
```
