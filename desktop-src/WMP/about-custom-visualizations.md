# <a name="about-custom-visualizations"></a>О пользовательских визуализациях

Проигрыватель Windows Media предоставляет интерфейс для создания пользовательских визуализаций. В этом разделе приводятся общие сведения об архитектуре визуализаций, сведения о том, как они связаны с проигрывателем Windows Media, и технические сведения, необходимые для создания пользовательских визуализаций.

Обратите внимание, что в нескольких местах кода отображается слово «результат». Визуализации — это эффекты.

Пользовательские визуализации более подробно описаны в следующих разделах.



| Раздел                                                              | Описание                                                                                |
|--------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [Обзор пользователя](user-overview.md)                                 | Описывает визуализации с точки зрения пользователя.                                          |
| [Обзор для разработчиков](developer-overview.md)                       | Описывает визуализации с точки зрения разработчика.                                     |
| [Создание визуализации](building-a-visualization.md)           | Описывает, как создать проект визуализации.                                           |
| [Реализация кода](implementing-your-code.md)               | Описывает, как изменить образец кода, созданный мастером подключаемых модулей визуализации.     |
| [Установка визуализаций](installing-visualizations.md)         | Описывает, как установить визуализацию на компьютере пользователя.                             |
