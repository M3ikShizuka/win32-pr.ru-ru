# <a name="slidercursor"></a>ПОЛЗУНок. Cursor

Атрибут **cursor** указывает или получает значение, указывающее, какой курсор отображается при наведении мыши на элемент управления Slider.

``` syntax
        elementID.cursor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи.



| Значение            | Описание                                                                                 |
|------------------|---------------------------------------------------------------------------------------------|
| система           | Зависимый от платформы курсор (обычно это стрелка).                                               |
| правый             | По умолчанию. Правый.                                                                              |
| help             | Стрелка с вопросительным знаком, указывающим на наличие справки.                                      |
| сизеалл          | 4-направленная стрелка, указывающая на север, Юг, Восток и Запад.                                   |
| сизенесв         | Двойная стрелка, указывающая на северо-восток и Запад.                                      |
| сизенс           | Двусторонняя стрелка, указывающая на север и Юг.                                              |
| сизенвсе         | Двойная стрелка, указывающая на северо-запад и Восток.                                      |
| сизеве           | Двойная стрелка, указывающая на запад и Восток.                                                |
| Стрелка вверх          | Вертикальная стрелка, указывающая вверх.                                                             |
| \*. ani или \* . cur | Любой ANI или cur-файл (должен находиться в том же каталоге, что и файл WMS, или в WMZ-файле). |
