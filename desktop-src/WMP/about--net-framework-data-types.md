# <a name="about-net-framework-data-types"></a>Сведения о типах данных платформа .NET Framework

В этом разделе содержатся сведения, необходимые для преобразования справочника по объектно-ориентированной модели объектов в базовые типы данных платформы Microsoft .NET Framework. Ссылка на сценарий проигрывателя Windows Media содержит почти всю информацию, необходимую для использования элемента управления проигрывателя Windows Media в программе на основе платформа .NET Framework, и в большинстве случаев синтаксис будет похож на тот, что используется в других языках сценариев, таких как Microsoft JScript.

Ссылка на проигрыватель Windows Media предоставляет тип данных JScript и, при необходимости, преобразование C++. Например, метод может вернуть **число** . В JScript все числа обрабатываются одинаково, но другие языки различают различные типы чисел (целые числа, плавающие точки и т. д.). Ссылка обеспечивает преобразование C++ для числовых типов данных, так как числа могут обрабатываться по-разному в C++. Например, C++ использует тип данных **int** для целочисленных арифметических операций и **float** для плавающей запятой.

В платформа .NET Framework используется немного другая система базовых типов данных. В следующей таблице показаны различия в стандартных типах данных, которые, скорее всего, будут использоваться. Дополнительные сведения о платформа .NET Framework базовых типах данных и преобразовании в другие системы типов данных см. в разделе Пошаговое описание базовых типов данных пространства имен System для разработчиков платформа .NET Framework.

Эта таблица содержит платформа .NET Framework имя класса и тип данных C#. Типы данных для других языков определяются для каждого языка в соответствующих языковых ссылках.



| Тип данных сценария | Тип данных в C++     | Класс платформа .NET Framework (тип данных C#) |
|------------------|-------------------|---------------------------------------|
| **Число**       | **int**           | **Int32** (**int**)                   |
| **Число**       | **long**          | **Int32** (**int**)                   |
| **Число**       | **double**        | **Double** (**двойной**)               |
| **Число**       | **float**         | **Single** (**float**)                |
| **String**       | **ОСВОБОЖДАЕМОЙ**          | **String** (**строка**)               |
| **Boolean**      | **VARIANT \_ bool** | **Логическое значение** (**bool**)                |
| **Объект**       | **Объект**        | **Объект** (**объект**)               |



 

При использовании Visual Studio для определения типа данных, ожидаемого для конкретной функции, можно использовать технологию Microsoft IntelliSense.
