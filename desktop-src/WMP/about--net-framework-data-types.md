# <a name="about-net-framework-data-types"></a>сведения о типах данных платформа .NET Framework

в этом разделе содержатся сведения, необходимые для преобразования справочника по объектно-ориентированной модели объектов в базовые типы данных Microsoft платформа .NET Framework. ссылка на скрипт проигрыватель Windows Media содержит почти всю информацию, необходимую для использования элемента управления проигрыватель Windows Media в программе на базе платформа .NET Framework, и в большинстве случаев синтаксис будет похож на тот, что и для других языков сценариев, таких как Microsoft JScript.

ссылка на проигрыватель Windows Media предоставляет JScript тип данных и, при необходимости, преобразование C++. Например, метод может вернуть **число** . JScript обрабатывает все числа одинаковым образом, но другие языки различают разные типы чисел (целые числа, плавающие точки и т. д.). Ссылка обеспечивает преобразование C++ для числовых типов данных, так как числа могут обрабатываться по-разному в C++. Например, C++ использует тип данных **int** для целочисленных арифметических операций и **float** для плавающей запятой.

в платформа .NET Framework используется немного другая система базовых типов данных. В следующей таблице показаны различия в стандартных типах данных, которые, скорее всего, будут использоваться. дополнительные сведения о платформа .NET Framework базовых типах данных и преобразовании в другие системы типов данных см. в разделе пошаговое описание базовых типов данных пространства имен System для разработчиков платформа .NET Framework.

эта таблица содержит платформа .NET Framework имя класса и тип данных C#. Типы данных для других языков определяются для каждого языка в соответствующих языковых ссылках.



| Тип данных сценария | Тип данных в C++     | класс платформа .NET Framework (тип данных C#) |
|------------------|-------------------|---------------------------------------|
| **Число**       | **int**           | **Int32** (**int**)                   |
| **Число**       | **long**          | **Int32** (**int**)                   |
| **Число**       | **double**        | **Double** (**двойной**)               |
| **Число**       | **float**         | **Single** (**float**)                |
| **String**       | **ОСВОБОЖДАЕМОЙ**          | **String** (**строка**)               |
| **Boolean**      | **VARIANT \_ bool** | **Логическое значение** (**bool**)                |
| **Объект**       | **Объект**        | **Объект** (**объект**)               |



 

при использовании Visual Studio можно использовать технологию Microsoft IntelliSense для определения типа данных, ожидаемого для конкретной функции.
