# <a name="equalizersettingssplinetension"></a>ЕКУАЛИЗЕРСЕТТИНГС. Сплинетенсион

Атрибут **сплинетенсион** указывает или получает натяжение сплайна для элемента управления эквалайзера.

``` syntax
        elementID.splineTension
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**float**) в диапазоне от 0,0 (без натяжения) до 10,0 (полная натяжение) со значением по умолчанию 3,0.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|---------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media 9 Series или более поздней версии<br/> |
