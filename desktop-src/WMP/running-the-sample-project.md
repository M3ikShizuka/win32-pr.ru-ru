# <a name="running-the-sample-project"></a>Запуск примера проекта

Пример COM-компонента демонстрирует функциональные возможности, предоставляемые методами **ивмпсубскриптионсервицес** , отображая диалоговое окно всякий раз, когда проигрыватель Windows Media вызывает один из методов. Чтобы увидеть это, необходимо сначала добавить в библиотеку содержимое, содержащее атрибут **контентдистрибутор** , соответствующий вашей службе. См. раздел [Добавление атрибута контентдистрибутор](adding-the-contentdistributor-attribute.md). Затем используйте проигрыватель Windows Media 10 или более поздней версии, чтобы открыть Интернет-магазин, воспроизвести содержимое уровня "Премиум" и скопировать его на устройство. Проигрыватель Windows Media будет создавать экземпляр примера COM-компонента и вызывать методы, если это уместно.
