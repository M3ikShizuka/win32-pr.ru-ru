# <a name="ffwdelement"></a>ффвделемент

Это предопределенный **буттонелемент** со следующими значениями по умолчанию.

``` syntax
onclick="jscript:player.controls.fastForward()"
upToolTip="Fast Forward"
cursor="system"
enabled="wmpenabled:player.controls.fastForward"
```

## <a name="remarks"></a>Remarks

Все свойства этого **буттонелемент** можно переопределить, явно указав их.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
