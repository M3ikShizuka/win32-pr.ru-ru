# <a name="orientation"></a>Orientation

некоторые Windows мобильные устройства можно использовать в различных физических ориентациях. Доступные ориентации:

-   **Книжная**. Устройство удерживается по вертикали, поэтому оно больше его ширины.
-   **Альбомная ориентация**. Устройство удерживается горизонтально, чтобы его ширина превышала высоту.
-   **Квадрат**. Некоторые устройства имеют представление в виде квадратной формы, то есть один режим ориентации.

Каждая создаваемая Обложка должна задавать одну ориентацию. Необходимо предоставить отдельную обложку для каждой ориентации. В средстве выбора обложки будут отображаться только обложки, созданные для текущей ориентации и размеров дисплея устройства.
