# <a name="iwmpdvdresume-method"></a>Метод ИВМПДВД:: Resume

Метод **Resume** возвращает в режим воспроизведения из режима меню, возобновляя его в той же позиции, что и при вызове меню.

## <a name="syntax"></a>Синтаксис


```CSharp
public void resume();
```


```VB

Public Sub resume()
Implements IWMPDVD.resume
```





## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
