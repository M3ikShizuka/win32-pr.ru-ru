# <a name="reference-for-type-1-online-stores"></a>Справочные материалы по интернет-магазинам типа 1

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Проигрыватель Windows Media 11 предоставляет компилятор каталога, объект и несколько интерфейсов, поддерживающих Интернет-магазины типа 1. Эти сведения подробно описаны в следующих разделах.



| Section                                                                                    | Описание                                                                                                                                                                                                                        |
|--------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Компилятор каталога для Интернет-магазинов типа 1](catalog-compiler-for-type-1-online-stores.md) | Содержит справочные материалы по компилятору, используемому Интернет-магазином для компиляции музыкального каталога.                                                                                                                               |
| [Интерфейс Ивмпконтентконтаинер](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainer)                                 | Содержит справочные страницы для методов, которые вызывает подключаемый модуль интернет-магазина для проверки контейнера содержимого. Контейнер содержимого представляет собой набор элементов мультимедиа для скачивания или покупки. Реализовано проигрывателем Windows Media. |
| [Интерфейс Ивмпконтентконтаинерлист](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentcontainerlist)                         | Содержит справочные страницы для методов, которые вызывает подключаемый модуль интернет-магазина для проверки списка контейнеров содержимого. Реализовано проигрывателем Windows Media.                                                                           |
| [Интерфейс Ивмпконтентпартнер](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartner)                                     | Содержит справочные страницы по методам, которые вызываются проигрывателем Windows Media для получения информации из Интернет-магазина и уведомления Интернет-магазина о действиях пользователя. Реализуется подключаемым модулем Интернет-магазина.                  |
| [Интерфейс Ивмпконтентпартнеркаллбакк](/previous-versions/windows/desktop/api/contentpartner/nn-contentpartner-iwmpcontentpartnercallback)                     | Содержит справочные страницы для методов, которые вызывает подключаемый модуль интернет-магазина для уведомления проигрывателя Windows Media о завершении асинхронных операций. Реализовано проигрывателем Windows Media.                                    |
| [Внешний объект для Интернет-магазинов типа 1](external-object-for-type-1-online-stores.md)   | Содержит справочные страницы по свойствам, методам и событиям, используемым скриптом на веб-страницах, предоставляемых Интернет-магазином.                                                                                                              |
| [Перечисления для Интернет-магазинов типа 1](enumerations-for-type-1-online-stores.md)         | Содержит справочные страницы для перечислений, используемых при обмене данными между проигрывателем Windows Media и подключаемым модулем Интернет-магазина.                                                                                               |
| [Структуры для Интернет-магазинов типа 1](structures-for-type-1-online-stores.md)             | Содержит справочные страницы по структурам, используемым при обмене данными между проигрывателем Windows Media и подключаемым модулем Интернет-магазина.                                                                                                 |
