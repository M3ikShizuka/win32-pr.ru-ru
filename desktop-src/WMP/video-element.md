# <a name="video-element"></a>Элемент VIDEO

Элемент **Video** предоставляет способ управления видеоокном в обложке с помощью следующих атрибутов и событий. Для удобства также предоставляется предварительно определенный элемент **Video** .

Элемент **Video** поддерживает следующие атрибуты.



| attribute                                            | Описание                                                                                                                                                                                                                              |
|------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [backgroundColor](video-backgroundcolor.md)         | Указывает или получает цвет фона элемента управления видео.                                                                                                                                                                        |
| [курсор](video-cursor.md)                           | Указывает или получает значение курсора, которое используется при наведении мыши на область видео, доступную для щелчка.                                                                                                                               |
| [fullScreen](video-fullscreen.md)                   | Указывает или получает значение, указывающее, отображается ли видео в полноэкранном режиме. Можно задать только во время выполнения.                                                                                                               |
| [маинтаинаспектратио](video-maintainaspectratio.md) | Указывает или получает значение, указывающее, будет ли видео поддерживать пропорции при попытке разместить его в пределах ширины и высоты, определенных для элемента управления.                                                                       |
| [шринктофит](video-shrinktofit.md)                 | Указывает или получает значение, указывающее, будет ли видео уменьшаться до ширины и высоты, определенных для элемента управления видео.                                                                                                           |
| [стретчтофит](video-stretchtofit.md)               | Указывает или получает значение, указывающее, будет ли видео растягиваться по ширине и высоте, определенным для элемента управления видео.                                                                                                   |
| [Сказок](video-tooltip.md)                         | Указывает или получает текст подсказки для окна видео.                                                                                                                                                                            |
| [без окон](video-windowless.md)                   | Указывает или получает значение, указывающее, будет ли элемент управления видео отображаться в окне или без окон. то есть независимо от того, будет ли весь прямоугольник элемента управления видимым в любое время или же его можно обрезать. Можно задать только во время разработки. |
| [масштаб](video-zoom.md)                               | Указывает процент, на который масштабируется видео.                                                                                                                                                                                    |



 

Элемент **Video** может реализовывать следующие обработчики событий.



| Обработчик событий                          | Описание                                                                  |
|----------------------------------------|------------------------------------------------------------------------------|
| [онвидеоенд](video-onvideoend.md)     | Обрабатывает событие, возникающее, когда видео останавливает отрисовку и выгружается. |
| [онвидеостарт](video-onvideostart.md) | Обрабатывает событие, возникающее при загрузке видео и начале его подготовки к просмотру.  |



 

Элемент **Video** поддерживает атрибуты окружения и может реализовывать обработчики событий окружения, за исключением случаев, когда отмечено. Дополнительные сведения см. в разделе [атрибуты окружения](ambient-attributes.md) и [обработчики событий окружения](ambient-event-handlers.md).

Стандартные элементы видео — это обычные элементы **видео** с различными параметрами общих атрибутов, заданных по умолчанию. Доступны следующие предварительно определенные элементы видео.



| Предопределенное видео         | Описание                                                |
|--------------------------|------------------------------------------------------------|
| [вмпвидео](wmpvideo.md) | Элемент **видео** , который растягивает видео при изменении размера. |
