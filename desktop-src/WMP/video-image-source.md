# <a name="video-image-source"></a>Источник изображения видео

Для области просмотра видео не требуется отдельный образ. Может потребоваться создать область в фоновом изображении, которая будет отображаться по умолчанию, чтобы действовать в качестве заполнителя. Сделайте прямоугольную область в фигуре с той же шириной и высотой, которую вы хотите указать в файле определения обложки. При воспроизведении видео оно будет наложено на эту область в фоновом файле.
