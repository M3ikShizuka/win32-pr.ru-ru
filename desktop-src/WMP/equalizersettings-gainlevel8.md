# <a name="equalizersettingsgainlevel8"></a>ЕКУАЛИЗЕРСЕТТИНГС. gainLevel8

Атрибут **gainLevel8** указывает или получает уровень усиления диапазона 8.

``` syntax
        elementID.gainLevel8
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**float**) со значением, которое обычно находится в диапазоне от 20 до + 20. Он имеет нулевое значение по умолчанию.

## <a name="remarks"></a>Комментарии

Этот атрибут корректирует часть спектра частот аудио в центре в 4kHz.

Если этот атрибут не указан, предыдущее значение будет храниться.
