# <a name="iwmpcdromburnburnstate-property"></a>Свойство Ивмпкдромбурн:: Бурнстате

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```CSharp
public WMPBurnState burnState {get;}
```


```VB

```
