# <a name="iwmpcdromburnburnstate-property"></a>Свойство Ивмпкдромбурн:: Бурнстате

Свойство **бурнстате** возвращает значение перечисления, указывающее текущее состояние записи.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```CSharp
public WMPBurnState burnState {get;}
```


```VB

Public ReadOnly Property burnState As WMPBurnState
```





## <a name="property-value"></a>Значение свойства

Объект **вмплиб. вмпбурнстате** , который является значением из перечисления **вмпбурнстате** , которое указывает текущее состояние.
