# <a name="onendmove"></a>онендмове

Обработчик событий **онендмове** обрабатывает событие, возникающее, когда элемент завершает операцию MoveTo.

``` syntax
onendmove
```

|--------------------|-----------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 70 или более поздней<br/> |
