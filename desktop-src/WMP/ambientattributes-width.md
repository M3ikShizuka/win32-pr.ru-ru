# <a name="ambientattributeswidth"></a>Амбиентаттрибутес. Width

Атрибут **Width** указывает или получает ширину элемента управления.

``` syntax
        elementID.width
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут представляет собой 16-разрядное **целое число** для чтения и записи (от 0 до 32 767), представляющее ширину элемента управления в пикселях. Значение по умолчанию равно нулю или ширине изображения, указанного в атрибуте **Image** элемента управления.

## <a name="remarks"></a>Remarks

Если заданная ширина является более узким, чем ширина предоставленного изображения, то изображение будет обрезано. Если ширина превышает ширину изображения, то область щелчка будет выходить за границы изображения. Независимо от того, какое значение присвоено этому атрибуту, изображение не может увеличиваться за пределами родительского **представления** или вложенного **представления**.
