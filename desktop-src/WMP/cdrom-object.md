# <a name="cdrom-object"></a>Объект CDROM

Объект **CDROM** предоставляет способ доступа к компакт-диску или DVD-диску на его диске.

Объект **CDROM** поддерживает следующие свойства.



| Свойство                                   | Описание                                                                                                                                             |
|--------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [дривеспеЦифиер](cdrom-drivespecifier.md) | Возвращает букву компакт-диска или DVD-дисковода.                                                                                                                   |
| [список воспроизведения](cdrom-playlist.md)             | Извлекает объект [списка воспроизведения](playlist-object.md) , который представляет дорожки на компакт-диске в данный момент в ДИСКОВОДе компакт-дисков или записи заголовка корневого уровня для DVD. |



 

Объект **CDROM** поддерживает следующий метод.



| Метод                   | Описание                          |
|--------------------------|--------------------------------------|
| [выдвинут](cdrom-eject.md) | Извлекает компакт-диск или DVD-диск из накопителя. |



 

Доступ к объекту **CDROM** осуществляется с помощью следующего метода.



| Объект                                        | Метод                           |
|-----------------------------------------------|----------------------------------|
| [кдромколлектион](cdromcollection-object.md) | [Item](cdromcollection-item.md) |



 

В целях иллюстрации в разделах синтаксиса Reference используется проигрыватель. Кдромколлектион. Item (*индекс*).
