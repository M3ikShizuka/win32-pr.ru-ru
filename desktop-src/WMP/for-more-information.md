# <a name="for-more-information-wmp-sdk"></a>Дополнительные сведения (пакет SDK для WMP)

-   Дополнительные сведения о разработке решений для цифрового мультимедиа см. в [центре разработчиков Microsoft Windows Media Center](https://msdn.microsoft.com/windowsmedia/default.aspx).
-   Дополнительные сведения о Windows Media см. на [веб-сайте Windows Media](https://support.microsoft.com/help/18612/windows-media-player).
-   Дополнительные сведения о проигрывателе Windows Media см. на [странице проигрывателя Windows Media](https://support.microsoft.com/help/18612/windows-media-player) на веб-сайте Windows Media.
-   Чтобы скачать Windows SDK, включающий пакет SDK для проигрывателя Windows Media, перейдите на страницу [Windows SDK](https://msdn.microsoft.com/windows/bb980924.aspx) в центре разработчиков Windows.
-   Дополнительные сведения о программировании с помощью технологий Майкрософт, включая технологии Windows Media и пакеты средств разработки программного обеспечения, см. в [библиотеке MSDN]().
-   Текущей версией пакета SDK для проигрывателя Windows Media является версия 12. Сведения о проблемах, связанных с пакетом SDK для проигрывателя Windows Media 11, см. в [файле readme пакета SDK для проигрывателя Windows Media 11](/previous-versions/aa905044(v=msdn.10)).
