# <a name="choosing-the-functions"></a>Выбор функций

Цель этой обложки — предоставить базовую функциональность для воспроизведения звука. Будут использованы следующие функции:



| Функция                     | Описание                                                                                                                                                                                                                                                                                                                                                                                                                          |
|------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Плайпаусе или Плайпаусестоп\* | Запуск элемента мультимедиа имеет простую важность. при создании обложки для проигрыватель Windows Media 10 Mobile или более поздней версии следует использовать плайпаусестоп. при работе с более ранней версией проигрыватель Windows Media Mobile необходимо использовать плайпаусе. Обе функции предоставляют дополнительные функциональные возможности приостановки, но только Плайпаусестоп позволяет остановить вещание в реальном времени, когда функция приостановки недоступна. |
| Stop                         | Чтобы закончить воспроизведение элемента, необходимо добавить параметр "останавливаться". если вы создаете обложку для проигрыватель Windows Media 10 Mobile или более поздней версии, функция плайпаусестоп уже предоставляет эту функцию.                                                                                                                                                                                                                                          |
| Следующая                         | Если у вас есть список воспроизведения, вы сможете выбрать следующий элемент. Это необходимо для простой навигации по цифровым носителям.                                                                                                                                                                                                                                                                                                       |
| Предыдущий                         | Хотя вы можете использовать Next для прохода по списку воспроизведения, пока вы не поступаете к началу, добавление в начале, что позволит пользователю перейти назад и вперед при навигации по списку воспроизведения.                                                                                                                                                                                                                                   |
| Волумеуп или Волумедовн\*     | Вам потребуется предоставить пользователю возможность включения или уменьшения объема тома.                                                                                                                                                                                                                                                                                                                                                    |



 

\*доступно для проигрыватель Windows Media 10 Mobile или более поздней версии.
