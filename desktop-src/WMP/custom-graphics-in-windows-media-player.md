# <a name="custom-graphics-in-windows-media-player"></a>Пользовательская графика в проигрывателе Windows Media

С помощью сценариев метафайлов Windows Media можно добавлять графические элементы в проигрыватель Windows Media при воспроизведении потока. Существует один базовый тип графики, которую можно добавить.



| Тип   | Описание                                                                                                                                                                                                                        |
|--------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| BANNER | С помощью элемента **баннера** можно разместить баннер (32 пикселей высотой 194 пикселей в ширину) в проигрывателе Windows Media в нижней части области экрана. Затем можно использовать дочерний элемент для добавления гиперссылки в баннер. |
