# <a name="image-source-for-disabled-trackbar"></a>Источник изображения для отключенной TrackBar

Необходимо определить источник изображения, которое должно отображаться при отключении TrackBar, используя тип точечного рисунка, из которого нужно создать изображение. отображаемое изображение обычно находится внутри супертипа или если вы создаете обложку для проигрыватель Windows Media 10 Mobile, образ будет находиться в файле сиксумб. Необходимо ввести тип образа, за которым следует пробел, символ @ и еще один пробел. Затем необходимо ввести два положительных целых числа, определяющие координаты верхнего левого угла (в пикселях) изображения, которое нужно использовать внутри типа точечного рисунка.

Например, чтобы использовать изображение из точечного рисунка Сиксумб с верхним и левым положением 50, 60 пикселей, введите:


```C++
Disabled @ 50,60

```



Необходимо определить расположение изображения для отключенного изображения TrackBar. Если вы не хотите отображать новый образ, можно определить фоновое изображение в качестве отключенного изображения со смещением 0, 0:


```C++
Disabled @ 50,60

```



В предыдущем примере 50, 60 означает расположение кнопки, с которой вы работаете, в файле Сиксумб (в данном случае это то же расположение, что и кнопка на фоновом рисунке). Однако настоятельно рекомендуется отображать отключенный образ для всех значений TrackBar, чтобы дать пользователю визуальную индикацию, поскольку ползунки не будут работать при определенных условиях. Например, если текущий список воспроизведения пуст, могут быть отключены линейки по поиску.
