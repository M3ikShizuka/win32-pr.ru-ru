# <a name="installing-while-offline"></a>Установка в автономном режиме

Пользователи могут установить проигрыватель Windows Media, не подключенный к Интернету. В этом случае программа установки проигрывателя Windows Media находит ServiceInfo.xml документ, указанный параметром командной строки *serviceInfo* . Если **ключевой** атрибут соответствует параметру командной строки *дефаултсервице* , программа установки применяет сведения из следующих элементов к проигрывателю Windows Media:

-   FriendlyName. Понятное имя Интернет-магазина отображается в пользовательском интерфейсе проигрывателя Windows Media.
-   Цвет. Указанные цвета применяются к панели задач и кнопкам.
-   ServiceTask1, ServiceTask2 и ServiceTask3. Дочерние элементы ButtonText и Буттонтип задаются. Атрибут URL-адреса не задан. URL-адрес задается после того, как пользователь подключается к Интернету, и проигрыватель Windows Media извлекает свой список интернет-магазинов обычным образом.
-   Изображение. Задаются атрибуты Менуурл и Сервицеларжеурл. Эти URL-адреса должны указывать на образы, установленные на жестком диске пользователя с помощью протокола file://.

Когда пользователь пытается просмотреть Интернет-магазин, проигрыватель Windows Media выводит сообщение с уведомлением о том, что требуется подключение к Интернету.
