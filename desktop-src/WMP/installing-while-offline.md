# <a name="installing-while-offline"></a>Установка в автономном режиме

пользователи могут устанавливать проигрыватель Windows Media, не подключаясь к интернету. в этом случае проигрыватель Windows Media программа установки находит ServiceInfo.xml документ, указанный параметром командной строки *ServiceInfo* . если **ключевой** атрибут соответствует параметру командной строки *дефаултсервице* , программа установки применяет сведения из следующих элементов для проигрыватель Windows Media:

-   FriendlyName. понятное имя интернет-магазина отображается в проигрыватель Windows Media пользовательском интерфейсе.
-   Цвет. Указанные цвета применяются к панели задач и кнопкам.
-   ServiceTask1, ServiceTask2 и ServiceTask3. Дочерние элементы ButtonText и Буттонтип задаются. Атрибут URL-адреса не задан. URL-адрес задается после того, как пользователь подключается к интернету и проигрыватель Windows Media получает свой список интернет-магазинов обычным образом.
-   Изображение. Задаются атрибуты Менуурл и Сервицеларжеурл. Эти URL-адреса должны указывать на образы, установленные на жестком диске пользователя с помощью протокола file://.

когда пользователь пытается просмотреть интернет-магазин, проигрыватель Windows Media отображает сообщение, информирующее пользователя о необходимости подключения к интернету.
