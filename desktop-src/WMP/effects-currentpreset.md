# <a name="effectscurrentpreset"></a>EFFECTs. Куррентпресет

Атрибут **куррентпресет** указывает или получает текущую предустановку текущей визуализации.

``` syntax
        elementID.currentPreset
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут имеет **номер** для чтения и записи (**длинный**), указывающий индекс предустановки. Индексы начинаются с нуля, который также является значением по умолчанию.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
