# <a name="textjustification"></a>TEXT. обоснование

Атрибут **обоснования** указывает или получает выравнивание текста внутри элемента управления "текст".

``` syntax
        elementID.justification
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи.



| Значение  | Описание                                                   |
|--------|---------------------------------------------------------------|
| Левый   | По умолчанию. Выровняйте текст по левому краю элемента управления "текст".     |
| Правый  | Выровняйте текст по правому краю элемента управления "текст".             |
| Center | Выравнивает текст по горизонтали по центру элемента управления "текст". |



 

## <a name="remarks"></a>Remarks

См. атрибут [value](text-value.md) для примера, иллюстрирующий использование атрибутов **текстового** элемента.
