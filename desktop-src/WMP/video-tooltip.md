# <a name="videotooltip"></a>ВИДЕО. подсказка

Атрибут **ToolTip** указывает или получает текст подсказки для окна видео.

``` syntax
        elementID.toolTip
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, длина которой не должна превышать 1024 символов. У него нет значения по умолчанию.

## <a name="remarks"></a>Remarks

Если для этого атрибута задано значение "" (пустая строка), подсказка не отображается.
