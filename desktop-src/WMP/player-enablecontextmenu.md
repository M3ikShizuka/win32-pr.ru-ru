# <a name="playerenablecontextmenu"></a>Player. Енаблеконтекстмену

Свойство **енаблеконтекстмену** указывает или получает значение, указывающее, следует ли включать контекстное меню, которое появляется при нажатии правой кнопки мыши.

## <a name="syntax"></a>Синтаксис

*проигрыватель*. **енаблеконтекстмену**

## <a name="possible-values"></a>Возможные значения

Это свойство является логическим значением для чтения и записи.



| Значение | Описание                       |
|-------|-----------------------------------|
| Да  | По умолчанию. Включите контекстное меню. |
| false | Не включайте контекстное меню.   |



 

## <a name="remarks"></a>Remarks

во время воспроизведения в полноэкранном режиме проигрыватель Windows Media скрывает курсор мыши, когда **енаблеконтекстмену** равняется false, а **uiMode** равно "none".

**проигрыватель Windows Media 10 Mobile:** Это свойство не поддерживается.
