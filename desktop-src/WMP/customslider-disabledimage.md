# <a name="customsliderdisabledimage"></a>КУСТОМСЛИДЕР. Дисабледимаже

Атрибут **дисабледимаже** указывает или получает изображение ползунка, используемого при отключенном ползунке.

``` syntax
        elementID.disabledImage
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей имя файла изображения.

## <a name="remarks"></a>Remarks

Этот атрибут является необязательным. Если он не указан, будет использоваться файл, указанный в атрибуте **Image** .

**Дисабледимаже** представляет отключенное состояние элемента управления **кустомслидер** . Это может быть один образ или несколько изображений, представляющих различные состояния ползунка. Если используется несколько образов, они упорядочиваются так же, как и файл **изображения** .

Поддерживаются следующие типы файлов изображений: BMP, JPG, PNG и GIF (не включая анимированные GIF).
