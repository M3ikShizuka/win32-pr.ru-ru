# <a name="iwmpmetadatatext-vb-and-c-interface"></a>Интерфейс Ивмпметадататекст (VB и C#)

Предоставляет свойства для получения сведений о сложных атрибутах текстовых метаданных.

Интерфейс **ивмпметадататекст** предоставляет следующие свойства.



| Свойство                                                                         | Описание                              |
|----------------------------------------------------------------------------------|------------------------------------------|
| [Описание](wmplibiwmpmetadatatext-iwmpmetadatatext-description--vb-and-c.md) | Возвращает описание текста метаданных. |
| [text](wmplibiwmpmetadatatext-iwmpmetadatatext-text--vb-and-c.md)               | Возвращает текст метаданных.                  |



 

Получите интерфейс **ивмпметадататекст** , передав имя атрибута, например **WM/усервебурл** , в следующий метод и приведя возвращаемый объект.



| Интерфейс                              | Свойство                                                                         |
|----------------------------------------|----------------------------------------------------------------------------------|
| [IWMPMedia3](iwmpmedia3--vb-and-c.md) | [жетитеминфобитипе](wmplibiwmpmedia3-iwmpmedia3-getiteminfobytype--vb-and-c.md) |



 

## <a name="members"></a>Элементы

Интерфейс **ивмпметадататекст (VB и C#)** не определяет никаких членов.
