# <a name="windows-media-download-packages-deprecated"></a>Пакеты загрузки Windows Media (устарели)

Эта страница описывает функцию, которая может быть недоступна в будущих версиях проигрывателя Windows Media и Windows Media Player SDK.

Пакеты загрузки Windows Media объединяют границы проигрывателя Windows Media, сведения о списках воспроизведения и мультимедийное содержимое в одном загружаемом файле с расширением WMD. Пакет загрузки Windows Media может включать в себя весь альбом музыкальных видео, в котором также отображается реклама в виде графического фирменного стиля и ссылки на веб-сайт Интернет-магазина.

Пользователи могут загрузить пакет загрузки Windows Media с веб-сайта, просто щелкнув ссылку. После загрузки пакета на компьютер пользователя проигрыватель Windows Media автоматически извлекает файлы в пакете, затем добавляет упакованный список воспроизведения в раскрывающийся список списки воспроизведения, добавляет содержимое в библиотеку, отображает обложку границы на панели **Воспроизведение** в проигрывателе Windows Media в полноэкранном режиме, а затем воспроизводит исходный элемент в списке воспроизведения.

Пакеты загрузки Windows Media позволяют получить следующие преимущества.

-   При однократном щелчке скачивается и извлекается несколько файлов на компьютер пользователя.
-   Содержимое начнется сразу после загрузки.
-   Настраиваемые границы могут отображать сведения о рекламе и фирменной символике.
-   Упакованные списки воспроизведения находятся в библиотеке.
-   Перенаправление веб-сайта может происходить из проигрывателя Windows Media в связанный сайт.
-   Доступны различные интерактивные элементы управления границами.
-   Упаковка поддерживается для нескольких типов звуковых и видеофайлов.

Все эти преимущества можно использовать вместе. Например, один пакет загрузки Windows Media может предоставить пользователям возможность просматривать песни песен по мере их воспроизведения, отображать видео, сопровождающее песни, просматривать рекламные сведения о распространителе записей, просматривать обложку альбома, посещать веб-сайт вентилятора и каталогизировать содержимое в библиотеке.
