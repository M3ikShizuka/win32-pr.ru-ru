# <a name="windows-media-download-packages-deprecated"></a>Windows Пакеты скачивания носителя (не рекомендуется)

эта страница описывает функцию, которая может быть недоступна в будущих версиях проигрыватель Windows Media и пакета SDK для проигрыватель Windows Media.

Windows пакеты для скачивания мультимедиа объединяют проигрыватель Windows Media границы, сведения о списках воспроизведения и мультимедийное содержимое в один загружаемый файл с расширением имени файла. wmd. пакет загрузки Windowsного носителя может включать в себя весь альбом музыкальных видео, в котором также отображается реклама в виде графического фирменного стиля и ссылки на веб-сайт интернет-магазина.

пользователи могут загрузить пакет скачивания Windows Media с веб-сайта, просто щелкнув ссылку. после загрузки пакета на компьютер пользователя проигрыватель Windows Media извлекает файлы внутри пакета автоматически, затем добавляет упакованный список воспроизведения в раскрывающийся список списки воспроизведения, добавляет содержимое в библиотеку, отображает обложку границы на панели « **воспроизведение** » в полноэкранном режиме проигрыватель Windows Media, а затем воспроизводит исходный элемент в списке воспроизведения.

Windows Пакеты загрузки мультимедиа предоставляют следующие преимущества:

-   При однократном щелчке скачивается и извлекается несколько файлов на компьютер пользователя.
-   Содержимое начнется сразу после загрузки.
-   Настраиваемые границы могут отображать сведения о рекламе и фирменной символике.
-   Упакованные списки воспроизведения находятся в библиотеке.
-   перенаправление веб-сайта может происходить из проигрыватель Windows Media в связанный сайт.
-   Доступны различные интерактивные элементы управления границами.
-   Упаковка поддерживается для нескольких типов звуковых и видеофайлов.

Все эти преимущества можно использовать вместе. например, один пакет загрузки носителя Windows может предоставить пользователям возможность просматривать песни в песен по мере их воспроизведения, отображать видео, сопровождающее песни, просматривать рекламные сведения о распространителе записей, просматривать обложку альбома, посещать веб-сайт вентилятора и каталогизировать содержимое в библиотеке.
