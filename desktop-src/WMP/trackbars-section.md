# <a name="trackbars-section"></a>Раздел TrackBar

Раздел TrackBar содержит сведения о любых элементах TrackBar, которые необходимо включить.


```C++
[ Trackbars ]

//  <Function>  <Location>     <Dis Image Src>  <Thumb Image Src>  <Thumb Size>
//  ----------  ----------     ---------------  -----------------  ------------
    Volume      40,200,170,40  Disabled @ 15,10    VolumeThumb.bmp    23,22

```



Здесь определена только линейка громкости. Дополнительные сведения о разделе TrackBar в файле определения обложки см. в статье [TrackBar](trackbars.md) в справочнике по обложкам.
