# <a name="editboxtextlimit"></a>EDITBOX. Текстлимит

Атрибут **текстлимит** указывает или получает максимальное число символов, которое пользователь может ввести в элементе управления "поле ввода".

``` syntax
        elementID.textLimit
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут имеет **номер** для чтения и записи (**длинный**) со значением по умолчанию 0, что означает отсутствие ограничения.
