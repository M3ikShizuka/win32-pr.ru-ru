# <a name="eventscreenwidth"></a>Event. Скринвидс

Атрибут **скринвидс** извлекает ширину доступного размера экрана в пикселях.

``` syntax
event.screenWidth
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** только для чтения (**длинное целое**).

## <a name="remarks"></a>Remarks

Это полезно для определения объема пространства в мониторе. При наличии двух мониторов он вычисляет пространство для обоих мониторов.
