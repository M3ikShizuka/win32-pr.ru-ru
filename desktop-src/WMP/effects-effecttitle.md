# <a name="effectseffecttitle"></a>EFFECTs. Еффекттитле

Метод **еффекттитле** извлекает отображаемое название визуализации с указанным индексом реестра.

``` syntax
        elementID.effectTitle(index)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*номер*
</dt> <dd>

**Число** (**Long**), содержащее индекс реестра визуализации.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает **строку**.

## <a name="remarks"></a>Комментарии

Этот метод используется для отображения заголовков визуализации в пользовательском интерфейсе.
