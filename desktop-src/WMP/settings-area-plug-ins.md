# <a name="settings-area-plug-ins"></a>Параметры Подключаемые модули области

для Microsoft проигрыватель Windows Media 11 и более ранних версий подключаемые модули пользовательского интерфейса области параметров отображаются в области "инструменты" панели " **воспроизведение** " под главной областью отображения и над элементами управления транспорта. для Microsoft проигрыватель Windows Media 12 и более поздних версий подключаемые модули пользовательского интерфейса области параметров отображаются в виде отдельных окон. В каждый момент времени может быть включен только один подключаемый модуль области параметров.

подключаемые модули Параметры области полезны для отображения пользовательских панелей элементов управления, аналогичных элементам управления "графический эквалайзер" или "параметры видео".
