# <a name="settings-area-plug-ins"></a>Подключаемые модули области параметров

Для проигрывателя Microsoft Windows Media 11 и более ранних версий подключаемые модули пользовательского интерфейса области параметров отображаются в области "Инструменты" панели " **Воспроизведение** " под главной областью отображения и над элементами управления транспорта. Для проигрывателя Microsoft Windows Media 12 и более поздних версий подключаемые модули пользовательского интерфейса области параметров отображаются в виде отдельных окон. В каждый момент времени может быть включен только один подключаемый модуль области параметров.

Подключаемые модули области параметров полезны для отображения пользовательских панелей элементов управления, аналогичных элементам управления "графический эквалайзер" или "Параметры видео".
