# <a name="effectseffectcangofullscreen"></a>EFFECTs. Еффекткангофуллскрин

Атрибут **еффекткангофуллскрин** извлекает значение, указывающее, может ли текущая визуализация отображаться в полноэкранном режиме.

``` syntax
        elementID.effectCanGoFullScreen
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением**, доступным только для чтения.



| Значение | Описание                                    |
|-------|------------------------------------------------|
| true  | Визуализация может отображаться в полноэкранном режиме.    |
| false | Визуализация не может быть отображена в полноэкранном режиме. |



 

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|---------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media для Windows XP или более поздней версии<br/> |
