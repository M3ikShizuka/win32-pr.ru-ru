# <a name="axwindowsmediaplayerclosedcaption-property"></a>Аксвиндовсмедиаплайер. Клоседкаптион, свойство

Свойство Клоседкаптион возвращает интерфейс **ивмпклоседкаптион** для текущего элемента мультимедиа.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```CSharp
public IWMPClosedCaption closedCaption {get;}
```


```VB

Public ReadOnly Property closedCaption As IWMPClosedCaption
```





## <a name="property-value"></a>Значение свойства

ВМПЛИБ. Интерфейс Ивмпклоседкаптион.
