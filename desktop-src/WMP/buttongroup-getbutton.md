# <a name="buttongroupgetbutton"></a>BUTTONGROUP. Кнопка

Метод  **буттонелемент** извлекает объект с указанным индексом.

``` syntax
        elementID.getButton(button)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="button"></span><span id="BUTTON"></span>*переключатель*
</dt> <dd>

**Число** (**Long**), содержащее индекс извлекаемого **буттонелемент** .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает объект, соответствующий элементу **буттонелемент** .

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|---------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media 9 Series или более поздней версии<br/> |
