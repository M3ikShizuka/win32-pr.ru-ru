# <a name="separate-window-plug-ins"></a>Отдельные подключаемые модули окна

Отдельные подключаемые модули пользовательского интерфейса окна отображаются в собственных окнах. Эти окна могут быть исправлены в размерах или могут быть изменены. Одновременно можно открыть несколько отдельных окон подключаемых модулей пользовательского интерфейса.

Отдельные подключаемые модули пользовательского интерфейса окна полезны, когда требуется, чтобы предоставляемые сведения или элементы управления оставались видимыми, когда пользователь выходит из области **воспроизведения** или загружает другой подключаемый модуль пользовательского интерфейса. все отдельные подключаемые модули пользовательского интерфейса окна, выполняемые при закрытии проигрыватель Windows Media, будут перезагружены и отображены в том же размере и положении при повторном запуске проигрывателя.
