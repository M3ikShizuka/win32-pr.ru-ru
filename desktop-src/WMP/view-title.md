# <a name="viewtitle"></a>Просмотреть. заголовок

Атрибут **Title** указывает или получает заголовок **представления**.

``` syntax
        elementID.title
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут представляет собой **строку** , указанную во время разработки, и затем доступна только для чтения. Он может иметь длину до 255 символов и не имеет значения по умолчанию.

## <a name="remarks"></a>Remarks

Заданный или полученный заголовок является отображаемым именем темы.
