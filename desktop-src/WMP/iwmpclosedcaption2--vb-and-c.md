# <a name="iwmpclosedcaption2-vb-and-c-interface"></a>Интерфейс IWMPClosedCaption2 (VB и C#)

Предоставляет свойства и методы для субтитров, дополняющих интерфейс **ивмпклоседкаптион** .

В дополнение к свойствам, унаследованным от **ивмпклоседкаптион**, интерфейс **IWMPClosedCaption2** предоставляет следующие свойства.

## <a name="members"></a>Элементы

Интерфейс **IWMPClosedCaption2 (VB и C#)** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **IWMPClosedCaption2 (VB и C#)** содержит следующие методы.



| Метод                                                                                             | Описание                                                                                       |
|:---------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------|
| [**жетсамилангид**](wmplibiwmpclosedcaption2-iwmpclosedcaption2-getsamilangid--vb-and-c.md)       | Возвращает код локали (LCID) языка, поддерживаемого текущим файлом SAMI.<br/> |
| [**жетсамилангнаме**](wmplibiwmpclosedcaption2-iwmpclosedcaption2-getsamilangname--vb-and-c.md)   | Возвращает имя языка, поддерживаемого текущим файлом SAMI.<br/>                     |
| [**жетсамистиленаме**](wmplibiwmpclosedcaption2-iwmpclosedcaption2-getsamistylename--vb-and-c.md) | Возвращает имя стиля, поддерживаемого текущим файлом SAMI.<br/>                        |



 

### <a name="properties"></a>Свойства

Интерфейс **IWMPClosedCaption2 (VB и C#)** имеет следующие свойства.



| Свойство                                                                                                | Тип доступа           | Описание                                                                         |
|:--------------------------------------------------------------------------------------------------------|:----------------------|:------------------------------------------------------------------------------------|
| [**самилангкаунт**](wmplibiwmpclosedcaption2-iwmpclosedcaption2-samilangcount--vb-and-c.md)<br/> | Чтение/запись<br/> | Возвращает или задает число языков, поддерживаемых текущим файлом SAMI.<br/> |
| [**самистилекаунт**](wmplibiwmpclosedcaption2-samistylecount.md)<br/>                            | Чтение/запись<br/> | Возвращает или задает количество стилей, поддерживаемое текущим файлом SAMI.<br/>    |



 

Получите интерфейс **IWMPClosedCaption2** путем приведения значения, возвращаемого свойством [**аксвиндовсмедиаплайер. клоседкаптион**](axwmplib-axwindowsmediaplayer-closedcaption--vb-and-c.md) .
