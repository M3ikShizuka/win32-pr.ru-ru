# <a name="playlistcollectionimportplaylist-method"></a>Плайлистколлектион. Импортплайлист, метод

Метод **импортплайлист** добавляет статический список воспроизведения в библиотеку.

## <a name="syntax"></a>Синтаксис


```JScript
retVal = PlaylistCollection.importPlaylist(
  playlist
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*список воспроизведения* \[ окне\]
</dt> <dd>

Добавляемый объект **списка воспроизведения** .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает добавленный объект **списка воспроизведения** .

## <a name="remarks"></a>Комментарии

Списки воспроизведения, которые не содержат элементов мультимедиа, не могут быть добавлены в библиотеку с помощью этого метода. Чтобы создать пустой список воспроизведения в библиотеке, используйте метод **невплайлист** . Затем можно заполнить полученный список воспроизведения элементами мультимедиа с помощью *списка воспроизведения*. **аппендитем** или *список воспроизведения*. **insertItem**.

Если передать этот метод автоматическому списку воспроизведения, запрос выполняется один раз, а результат добавляется в библиотеку в виде статического списка воспроизведения. Чтобы добавить автоматический список воспроизведения в библиотеку и сохранить его автоматическое поведение, используйте *медиаколлектион*. **Добавить**. Дополнительные сведения см. в разделе [статические и автоматические списки воспроизведения](static-and-auto-playlists.md).

Чтобы использовать этот метод, требуется полный доступ к библиотеке. Дополнительные сведения см. в разделе [доступ к библиотеке](library-access.md).
