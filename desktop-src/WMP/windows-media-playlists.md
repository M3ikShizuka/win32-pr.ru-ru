# <a name="windows-media-playlists"></a>Списки воспроизведения Windows Media

Этот справочник документирует списки воспроизведения Windows Media. Список воспроизведения Windows Media (WPL) — это файл, в котором используются элементы, описанные в этом справочнике, чтобы определить динамический «смарт-» список элементов мультимедиа или статический набор элементов мультимедиа.

Этот справочный материал содержит следующие разделы.
