# <a name="settingsdefaultaudiolanguage"></a>Параметры. дефаултаудиолангуаже

свойство **дефаултаудиолангуаже** извлекает код языка для аудио по умолчанию, указанного в проигрыватель Windows Media.

## <a name="syntax"></a>Синтаксис

Player. Settings. Дефаултаудиолангуаже

## <a name="possible-values"></a>Возможные значения

Это свойство является **числом** только для чтения (**длинное целое**).

## <a name="remarks"></a>Remarks

Код языка (LCID) однозначно определяет определенный диалект языка, который называется локальным языком.
