# <a name="settingsdefaultaudiolanguage"></a>Settings. Дефаултаудиолангуаже

Свойство **дефаултаудиолангуаже** извлекает код языка для звука по умолчанию, указанного в проигрывателе Windows Media.

## <a name="syntax"></a>Синтаксис

Player. Settings. Дефаултаудиолангуаже

## <a name="possible-values"></a>Возможные значения

Это свойство является **числом** только для чтения (**длинное целое**).

## <a name="remarks"></a>Комментарии

Код языка (LCID) однозначно определяет определенный диалект языка, который называется локальным языком.
