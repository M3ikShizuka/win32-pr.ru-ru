# <a name="managing-playlists"></a>Управление списками воспроизведения

Объект **списка воспроизведения** представляет набор элементов мультимедиа. Свойства и методы объекта используются для управления элементами мультимедиа, которые он содержит, и для получения метаданных о списке воспроизведения. Для управления списками воспроизведения используется объект **плайлистколлектион** и объект **медиаколлектион** .

в следующих разделах описывается управление списками воспроизведения с помощью объектной модели элемента управления проигрыватель Windows Media.

-   [Атрибуты списка воспроизведения](playlist-attributes.md)
-   [Списки воспроизведения и элементы мультимедиа](playlists-and-media-items.md)
-   [Списки воспроизведения и объект Плайлистколлектион](playlists-and-the-playlistcollection-object.md)
-   [Статические и автоматические списки воспроизведения](static-and-auto-playlists.md)
-   [Списки воспроизведения и объект Медиаколлектион](playlists-and-the-mediacollection-object.md)
