# <a name="working-with-streaming-resources"></a>Работа с потоковой передачей ресурсов

в примере проекта подключаемого модуля DSP для аудио, созданного мастером подключаемых модулей проигрыватель Windows Media, не требуются потоковые ресурсы для выделения подключаемым модулем. Однако в образце Echo требуется отдельный буфер для хранения звуковых данных в течение определенного периода времени для создания эффекта задержки. Управление буфером осуществляется двумя методами: **имедиаобжект:: аллокатестреамингресаурцес**, который создает буфер, и **Имедиаобжект:: фристреамингресаурцес**, который освобождает буфер. Методы **имедиаобжект** реализуются в Echo. cpp.
