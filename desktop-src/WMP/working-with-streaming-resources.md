# <a name="working-with-streaming-resources"></a>Работа с потоковой передачей ресурсов

Пример проекта подключаемого модуля DSP для аудио, создаваемого мастером подключаемых модулей проигрывателя Windows Media, не требует выделения ресурсов потоковой передачи подключаемым модулем. Однако в образце Echo требуется отдельный буфер для хранения звуковых данных в течение определенного периода времени для создания эффекта задержки. Управление буфером осуществляется двумя методами: **имедиаобжект:: аллокатестреамингресаурцес**, который создает буфер, и **Имедиаобжект:: фристреамингресаурцес**, который освобождает буфер. Методы **имедиаобжект** реализуются в Echo. cpp.

Следующие разделы содержат дополнительные сведения об управлении буферами.

-   [Реализация Имедиаобжект:: Аллокатестреамингресаурцес](implementing-imediaobject--allocatestreamingresources.md)
-   [Реализация Имедиаобжект:: Фристреамингресаурцес](implementing-imediaobject--freestreamingresources.md)
