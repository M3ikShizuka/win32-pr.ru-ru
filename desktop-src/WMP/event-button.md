# <a name="eventbutton"></a>событие. Кнопка

Атрибут **Button** получает значение, указывающее, какая кнопка мыши была нажата при возникновении события.

``` syntax
event.button
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** только для чтения (**длинное целое**).



| Значение | Описание                      |
|-------|----------------------------------|
| 0     | Нет кнопок мыши.      |
| 1     | Левая кнопка мыши была нажата.  |
| 2     | Правая кнопка мыши была нажата. |
| 3     | Обе кнопки мыши были отключены.    |

|--------------------|------------------------------------------------------|
