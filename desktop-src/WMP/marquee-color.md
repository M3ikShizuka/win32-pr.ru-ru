# <a name="marquee-color"></a>Цвет бегущей строки

Необходимо указать значение цвета для типа. Значение должно быть указано с использованием трех положительных целых чисел, разделенных запятыми. Значения представляют красный, зеленый и синий цвет для растрового цвета, а диапазоны — от 0 до 255.
