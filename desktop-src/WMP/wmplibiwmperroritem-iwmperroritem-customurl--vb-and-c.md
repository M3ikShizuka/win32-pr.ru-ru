# <a name="iwmperroritemcustomurl-property"></a>Свойство Ивмперроритем:: Кустомурл

Свойство **кустомурл** получает URL-адрес сайта, который отображает конкретные сведения об ошибке загрузки кодека.

## <a name="syntax"></a>Синтаксис


```CSharp
public System.String customUrl {get; set;}
```


```VB

Public ReadOnly Property customUrl As System.String
```





## <a name="property-value"></a>Значение свойства

**Строка System. String** , которая является пользовательским URL-адресом.
