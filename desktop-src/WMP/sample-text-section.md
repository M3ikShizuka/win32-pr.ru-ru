# <a name="sample-text-section"></a>Пример текстового раздела

В следующих строках показан типичный Текстовый раздел файла определения обложки:


```C++
//  <Type>       <Location>     <Align>  <Font>          <Color>

//  ------       ----------     -------  ------          -------

    Time         180,46,50,30   Right    Tahoma,16,N     255,255,255

    Playlist     3,46,175,20    Left     Tahoma,10,N     255,255,255

    Track#       10,68,40,20    Left     Tahoma,14,N     255,255,255

```



В предыдущем примере определяются три текстовых поля, отображающие текущее время в элементе мультимедиа, текущий список воспроизведения и номер текущей записи.
