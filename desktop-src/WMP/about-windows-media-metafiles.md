# <a name="about-windows-media-metafiles"></a>сведения о метафайлах мультимедиа Windows

метафайлы мультимедиа Microsoft Windows представляют собой текстовые файлы, содержащие сведения о файловом потоке и его представлении. Windows Метафайлы мультимедиа основаны на синтаксисе язык XML (XML) и могут быть закодированы в формате ANSI или UNICODE (UTF-8). Они состоят из различных элементов со связанными тегами и атрибутами. каждый элемент в метафайле Windows мультимедиа определяет определенный параметр или действие в проигрыватель Windows Media. Некоторые элементы должны располагаться в определенном месте файла относительно других элементов. некоторые элементы имеют обязательные теги и атрибуты, которые должны быть определены в метафайле мультимедиа Windows.

Windows пакеты загрузки мультимедиа позволяют объединять проигрыватель Windows Media границы, данные списка воспроизведения и мультимедийное содержимое в один загружаемый файл.
