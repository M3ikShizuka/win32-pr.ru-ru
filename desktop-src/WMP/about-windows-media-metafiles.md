# <a name="about-windows-media-metafiles"></a>О метафайлах Windows Media

Метафайлы Microsoft Windows Media — это текстовые файлы, содержащие сведения о файловом потоке и его представлении. Метафайлы Windows Media основаны на синтаксисе язык XML (XML) и могут быть закодированы в формате ANSI или UNICODE (UTF-8). Они состоят из различных элементов со связанными тегами и атрибутами. Каждый элемент в метафайле Windows Media определяет определенный параметр или действие в проигрывателе Windows Media. Некоторые элементы должны располагаться в определенном месте файла относительно других элементов. Некоторые элементы имеют обязательные Теги и атрибуты, которые должны быть определены в метафайле Windows Media.

Пакеты загрузки Windows Media позволяют объединять границы проигрывателя Windows Media, данные списка воспроизведения и мультимедийное содержимое в один загружаемый файл.
