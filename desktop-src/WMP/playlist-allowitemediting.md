# <a name="playlistallowitemediting"></a>Список воспроизведения. Алловитемедитинг

Атрибут **алловитемедитинг** указывает или получает значение, указывающее, будут ли элементы в списке воспроизведения поддерживать редактирование на месте.

``` syntax
        elementID.allowItemEditing
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                           |
|-------|---------------------------------------|
| true  | По умолчанию. Разрешено редактирование на месте. |
| false | Редактирование на месте запрещено.      |



 

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
