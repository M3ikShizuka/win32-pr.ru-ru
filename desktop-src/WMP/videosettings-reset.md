# <a name="videosettingsreset"></a>ВИДЕОСЕТТИНГС. Reset

Метод **Reset** сбрасывает все атрибуты до значений по умолчанию, равных нулю.

``` syntax
        elementID.reset()
```

## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="requirements"></a>Requirements (Требования)



| Требование | Значение |
|--------------------|-----------------------------------------------------------------|
| Версия<br/> | версия проигрыватель Windows Media для Windows XP или более поздней версии<br/> |
