# <a name="organization-of-the-documentation"></a>Организация документации

Пакет SDK для проигрывателя Windows Media документирует широкий спектр технологий, связанных с проигрывателем Windows Media. Чтобы упростить обнаружение необходимой информации, документация разделена на разделы, которые помогут быстро находить темы о конкретной функции или технологии.

Там, где это уместно, основные разделы включают в себя три подраздела: сведения, руководство по программированию и справочные материалы. В разделе о программе описывается конкретная функция или технология, помогающие понять, как ее можно использовать. В разделе Руководство по программированию добавлены учебники, посвященные этой технологии, с пошаговыми инструкциями. В разделе Reference содержатся сведения о интерфейсах, объектах, методах, свойствах, событиях, элементах и атрибутах, доступных для использования.

Следующая таблица поможет вам понять, где искать необходимую вам помощь.
