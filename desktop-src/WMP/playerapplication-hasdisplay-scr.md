# <a name="playerapplicationhasdisplay"></a>ПЛАЙЕРАППЛИКАТИОН. Хасдисплай

атрибут **хасдисплай** извлекает значение, указывающее, может ли видео отображаться с помощью удаленного элемента управления проигрыватель Windows Media.

``` syntax
        elementID.hasDisplay
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением**, доступным только для чтения.



| Значение | Описание               |
|-------|---------------------------|
| True  | Видео может отображаться.    |
| Нет | Не удается отобразить видео. |



 

## <a name="remarks"></a>Remarks

этот атрибут используется только при удаленном взаимодействии с элементом управления проигрыватель Windows Media.

несколько элементов управления проигрыватель Windows Media могут выполняться удаленно одновременно, но видео может отображаться в одном расположении одновременно либо в полноэкранном режиме проигрывателя, либо в одном из удаленных элементов управления. Используйте это свойство, чтобы определить, является ли текущий элемент управления элементом, с помощью которого можно отобразить видео.
