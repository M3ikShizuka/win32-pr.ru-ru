# <a name="playerapplicationhasdisplay"></a>ПЛАЙЕРАППЛИКАТИОН. Хасдисплай

Атрибут **хасдисплай** извлекает значение, указывающее, может ли видео отображаться через удаленный элемент управления проигрывателя Windows Media.

``` syntax
        elementID.hasDisplay
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением**, доступным только для чтения.



| Значение | Описание               |
|-------|---------------------------|
| True  | Видео может отображаться.    |
| Неверно | Не удается отобразить видео. |



 

## <a name="remarks"></a>Комментарии

Этот атрибут используется только при удаленном взаимодействии с элементом управления проигрывателя Windows Media.

Несколько элементов управления проигрывателем Windows Media могут выполняться удаленно одновременно, но видео может отображаться в одном расположении одновременно либо в полноэкранном режиме проигрывателя, либо в одном из удаленных элементов управления. Используйте это свойство, чтобы определить, является ли текущий элемент управления элементом, с помощью которого можно отобразить видео.
