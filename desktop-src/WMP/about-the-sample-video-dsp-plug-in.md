# <a name="about-the-sample-video-dsp-plug-in"></a>О примере подключаемого модуля DSP видео

Подключаемый модуль схемы DSP видео — это простая реализация обработки, которая масштабирует насыщенность цвета видео на коэффициент, предоставленный пользователем на странице свойств. Подключаемый модуль принимает значения от 0 до 1. Значение по умолчанию — 1. Значение 1 не влияет на изображение видео; другие коэффициенты масштабирования отменяют насыщенность цвета изображения. Например, значение 0,5 приведет к насыщенности цвета в 50%. Нулевое значение приводит к полутону видео в градациях серого.

Пример подключаемого модуля DSP видео работает со следующими видеоформатами:

-   NV12
-   YV12
-   YUY2
-   UYVY
-   RGB32
-   RGB24
-   RGB555
-   RGB565
