# <a name="cdromcollection-object"></a>Объект Кдромколлектион

Объект **кдромколлектион** предоставляет способ организации и доступа к коллекции компакт-дисков или DVD-дисководов.

Объект **кдромколлектион** поддерживает следующее свойство.



| Свойство                           | Описание                                                        |
|------------------------------------|--------------------------------------------------------------------|
| [count](cdromcollection-count.md) | Возвращает количество доступных дисководов компакт-дисков и дисков DVD в системе. |



 

Объект **кдромколлектион** поддерживает следующие методы.



| Метод                                                         | Описание                                                                               |
|----------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [жетбидривеспеЦифиер](cdromcollection-getbydrivespecifier.md) | Извлекает объект [CDROM](cdrom-object.md) , связанный с определенной буквой диска. |
| [Item](cdromcollection-item.md)                               | Извлекает объект [CDROM](cdrom-object.md) по указанному индексу.                        |



 

Доступ к объекту **кдромколлектион** осуществляется через следующее свойство.



| Объект                      | Свойство                                      |
|-----------------------------|-----------------------------------------------|
| [Игрок](player-object.md) | [кдромколлектион](player-cdromcollection.md) |
