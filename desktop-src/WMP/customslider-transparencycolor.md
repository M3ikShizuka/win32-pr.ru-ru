# <a name="customslidertransparencycolor"></a>КУСТОМСЛИДЕР. Транспаренциколор

Атрибут **транспаренциколор** указывает или получает цвет прозрачности настраиваемых изображений ползунка.

``` syntax
        elementID.transparencyColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей любое значение цвета Microsoft Internet Explorer. У него нет значения по умолчанию.

## <a name="remarks"></a>Комментарии

Любая часть пользовательских изображений с ползунком, содержащих **транспаренциколор** , позволяет отображать фон с помощью.

Так как Жпгс являются утерянными и поэтому подвержены непредвиденному изменению цвета, они не рекомендуются при использовании **транспаренциколор** .
