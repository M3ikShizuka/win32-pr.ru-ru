# <a name="editboxvalue"></a>EDITBOX. Value

Атрибут **value** указывает или получает текст, отображаемый в элементе управления "поле ввода".

``` syntax
        elementID.value
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи со значением по умолчанию "" (пустая строка).

## <a name="requirements"></a>Requirements (Требования)



| Требование | Значение |
|--------------------|---------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media для Windows XP или более поздней версии<br/> |
