# <a name="equalizersettingsnormalizationpeak"></a>ЕКУАЛИЗЕРСЕТТИНГС. Нормализатионпеак

Атрибут **нормализатионпеак** получает значение пиковой нормализации.

``` syntax
        elementID.normalizationPeak
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** только для чтения (**float**).

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|---------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media 9 Series или более поздней версии<br/> |
