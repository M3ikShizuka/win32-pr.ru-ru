# <a name="creating-a-skin-definition-file"></a>Создание файла определения обложки

Файл определения обложки — это текстовый файл, который определяет обложку и все ее составные части. Расширение имени файла должно быть СКН.

Каждый файл определения обложки должен начинаться со следующей строки, указывающей номер версии файла обложки. в следующей таблице приведены проигрыватель Windows Media версии и строка кода, которые должны использоваться для определения каждого в файле определения обложки. Хотя некоторые числа в строках кода не являются предполагаемыми, они являются правильными, как показано в следующей таблице.



| версия проигрыватель Windows Media | Первая строка файла определения обложки |
|------------------------------|------------------------------------|
| 1.01.1<br/>            | \[Файл обложки Pocket WMP версии 1.0\]      |
| 1.2                          | \[Файл обложки Pocket WMP версии 1.2\]      |
| 7.0                          | \[Файл обложки Pocket WMP версии 2.0\]      |
| 7.1                          | \[Файл обложки Pocket WMP версии 8.0\]      |
| Серия 9                     | \[Файл обложки Pocket WMP v 9.0.1\]    |
| 10 Mobile                    | \[Файл обложки Pocket WMP v 9.0.1\]    |
| 10,1 Mobile                  | \[Файл обложки Pocket WMP v 9.0.1\]    |



 

номер версии 9.0.1 предназначен для обложек, созданных специально для проигрыватель Windows Media 9 Series или более поздних версий. обложки с более ранним номером версии открываются проигрыватель Windows Media 9 Series или более поздней версии в режиме книжной ориентации, 96 точек на дюйм (DPI).

Файл определения обложки состоит из нескольких разделов. Каждый раздел определяет определенную область обложки. Разделы должны быть размещены в следующем порядке:

1.  Описание
2.  Растровые изображения
3.  видео;
4.  через кнопки;
5.  Состояние
6.  Текст
7.  Marquee
8.  Линейки воспроизведения

Каждый раздел начинается с имени раздела в квадратных скобках, например:


```C++
[ Bitmaps ]

```



> [!Note]  
> Файл определения обложки будет проанализирован неправильно, если между скобками и именем раздела не указаны пробелы.

 

Затем одна или несколько строк определяют отдельные изображения, кнопки и т. д. Например, раздел растровых изображений может включать в себя следующее:


```C++
    Background  Background.bmp  0,0
    Disabled    Disabled.bmp    0,0
    Pushed      Pushed.bmp      0,0
    Region      Region.bmp      0,0
    Super       Super.bmp       0,0

```



Не обязательно выровнять значения в столбцах, но это поможет сделать код более удобочитаемым. Дополнительные сведения о каждом разделе файла определения обложки см. в [справочнике по обложкам](skin-reference.md).

> [!Note]  
> Не используйте символы табуляции в любом месте файла. Вместо этого используйте лишние пробелы. Это очень важно, поскольку нажатие клавиши TAB во время записи или редактирования файла определения обложки приведет к сбою всей обложки. Каждая строка должна быть заполнена и не может продолжаться на второй строке. кроме того, параметры Region и Super не рекомендуются для обложек, используемых с проигрыватель Windows Media 10 Mobile или более поздней версии.
