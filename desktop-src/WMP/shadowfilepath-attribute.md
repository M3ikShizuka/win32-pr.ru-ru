# <a name="shadowfilepath-attribute"></a>Атрибут Шадовфилепас

Атрибут **шадовфилепас** представляет собой полный путь к теневому файлу.

## <a name="applies-to"></a>Применение

-   [Звуковые элементы](audio-item-attributes.md)

## <a name="remarks"></a>Комментарии

*Теневой файл* создается как часть преобразования формата файла. Это копия файла, используемая проигрывателем при синхронизации пользователем содержимого компьютера с устройством, которое требует, чтобы содержимое было в исходном формате. Этот атрибут задан для преобразованного файла, указывающего расположение теневого файла.

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .
