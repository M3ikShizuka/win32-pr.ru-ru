# <a name="rules-for-each-line"></a>Правила для каждой строки

Каждое определение в файле определения обложки должно находиться в отдельной строке. Необходимо ввести текст в строки, используя следующие правила.

-   Каждая строка должна заканчиваться символом возврата из абзаца. Некоторые строки могут быть длиннее, чем видимая длина строки в текстовом редакторе, но не разрывать строки. Вы можете отключить перенос строки, чтобы убедиться, что строки не нарушают работу.
-   Используйте только видимые символы, как определено в ссылке на обложку.
-   Строка состоит из одного или нескольких элементов, разделенных пробелами.
-   Можно ввести столько пробелов между элементами, сколько требуется для удобочитаемости.
-   Не используйте символы табуляции в любом месте файла.
-   Строки, начинающиеся с двух последовательных косых черт (//), игнорируются. Используйте их для комментариев. Пустые строки также игнорируются.
-   Добавьте пробелы между скобками и именем раздела.
