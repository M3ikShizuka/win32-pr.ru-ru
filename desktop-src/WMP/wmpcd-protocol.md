# <a name="wmpcd-protocol"></a>Протокол ВМПКД

Протокол ВМПКД позволяет указывать дорожки на компакт-диске с помощью синтаксиса URL. Это общий синтаксис протокола:


```C++
wmpcd://drive/track
```



Сегмент *диска* — это индекс ДИСКОВОДА компакт-дисков. Сегмент *Track* — это номер курса. В следующем примере демонстрируется использование протокола ВМПКД.


```C++
player.url = "wmpcd://0/4";
```



В примере воспроизводится четвертая запись на диске на первом компакт-диске (диск, индекс которого равен 0).
