# <a name="buttongroupcursor"></a>BUTTONGROUP. Cursor

Атрибут **cursor** указывает или получает тип курсора, который появляется при наведении указателя мыши на кнопку в **BUTTONGROUP**.

``` syntax
        elementID.cursor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей одно из следующих значений.



| Значение            | Описание                                                                                 |
|------------------|---------------------------------------------------------------------------------------------|
| система           | По умолчанию. Зависимый от платформы курсор (обычно это стрелка).                                      |
| правый             | Правый.                                                                                       |
| help             | Стрелка с вопросительным знаком, указывающим на наличие справки.                                      |
| сизеалл          | 4-направленная стрелка, указывающая на север, Юг, Восток и Запад.                                   |
| сизенесв         | Двойная стрелка, указывающая на северо-восток и Запад.                                      |
| сизенс           | Двусторонняя стрелка, указывающая на север и Юг.                                              |
| сизенвсе         | Двойная стрелка, указывающая на северо-запад и Восток.                                      |
| сизеве           | Двойная стрелка, указывающая на запад и Восток.                                                |
| Стрелка вверх          | Вертикальная стрелка, указывающая вверх.                                                             |
| \*. ani или \* . cur | Любой ANI или cur-файл (должен находиться в том же каталоге, что и файл WMS, или в WMZ-файле). |



 

## <a name="remarks"></a>Комментарии

Указанный курсор применяется ко всем кнопкам в **BUTTONGROUP**.

Если указано недопустимое значение курсора, оно остается в ранее заданном значении.

Пути к именам файлов курсоров игнорируются, поэтому файл курсора должен находиться в каталоге по умолчанию.
