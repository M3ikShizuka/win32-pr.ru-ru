# <a name="ambientattributesalphablend"></a>Амбиентаттрибутес. Алфабленд

Атрибут **алфабленд** указывает или получает значение для альфа-смешения любого **представления**, **подпредставления** или мини-приложения пользовательского интерфейса.

``` syntax
        elementID.alphaBlend
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**длинное**) со значением в диапазоне от 0 (без непрозрачности) до 255 (полная непрозрачность) и значением по умолчанию 255.

## <a name="remarks"></a>Комментарии

Этот атрибут позволяет элементу отображаться полупрозрачным, в зависимости от объема непрозрачности. Чем меньше непрозрачность, тем более прозрачным будет элемент. Каждый элемент обложки может иметь отдельное значение прозрачности, за исключением элементов Button в элементе управления **BUTTONGROUP** . Если **алфабленд** задан в **представлении**, будет задано непрозрачность всей обложки. Альфа-смешение не будет работать для оконных элементов управления, включая **списки воспроизведения**, **эффекты**, **списки**, **всплывающие подсказки**, **EDITBOX** и **видео** (если **окно без окон** имеет значение false). Если для **представления** задано значение **алфабленд** , вся Обложка становится прозрачной. Атрибуты **транспаренциколор** , используемые несколькими элементами, не поддерживаются в **алфабленд**.

При использовании **алфабленд** с **текстовым** элементом, для которого не указан **backgroundColor** , будет использоваться цвет фона черного цвета. Если цвет переднего плана также является черным (значением по умолчанию для *текста*).**foregroundColor**), текст может стать нечитаемым. Чтобы избежать этого, всегда указывайте атрибут **backgroundColor** или задайте для **foregroundColor** цвет, отличный от черного.

> [!Note]  
> Этот атрибут не поддерживается в Windows 98.
