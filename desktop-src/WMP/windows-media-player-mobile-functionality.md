# <a name="windows-media-player-mobile-functionality"></a>проигрыватель Windows Media Мобильные функции

Для пользователя обложки доступны следующие действия:

-   Касание или нажатие кнопок
-   Скользящие TrackBar
-   Чтение прокручиваемого текста
-   Просмотр видео
-   Просмотр обложки альбома
-   Элементы носителя рейтинга

В следующих разделах описываются типы элементов, которые можно создать для реализации этих действий пользователя.

-   [Функции кнопок](button-functions.md)
-   [Типы TrackBar](trackbar-types.md)
-   [Типы отображаемых текста](text-display-types.md)
-   [Экран видео](the-video-display.md)
-   [Orientation](orientation.md)
