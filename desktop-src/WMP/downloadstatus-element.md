# <a name="downloadstatus-element"></a>Довнлоадстатус, элемент

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Элемент **довнлоадстатус** указывает URL-адрес, который проигрыватель Windows Media отображает как ссылку, чтобы разрешить пользователям просматривать состояние скачивания.

``` syntax
<DownloadStatus
    URL = "URL"
/>
```

## <a name="attributes"></a>Атрибуты

<dl> <dt>

<span id="URL"></span><span id="url"></span>URL-адрес
</dt> <dd>

URL-адрес веб-страницы, отображаемой Интернет-магазином для отображения состояния скачивания пользователю.

</dd> </dl>

## <a name="parentchild-elements"></a>Родительские и дочерние элементы



| Иерархия       | Элемент         |
|-----------------|-----------------|
| Родительские элементы | **ServiceInfo** |
| Дочерние элементы  | Нет            |



 

## <a name="remarks"></a>Remarks

Проигрыватель Windows Media отображает сообщение для пользователей, когда выполняется скачивание. Если текущие активные службы определяют URL-адрес состояния загрузки, пользователь может щелкнуть текст сообщения. Когда пользователь щелкает сообщение, проигрыватель переходит по URL-адресу, указанному в элементе **довнлоадстатус** , чтобы текущий активный магазин мог предоставить сведения о выполняемых скачиваниях.
