# <a name="advantages-of-using-htmlview"></a>Преимущества использования Хтмлвиев

Функция Хтмлвиев упрощает создание интегрированного веб-интерфейса пользователя, который воспроизводит мультимедийное содержимое с помощью привычного вида Интернета. при объединении веб-содержимого с цифровым медиа-содержимым в проигрыватель Windows Media пользовательском интерфейсе результат может оказать больше воздействия на пользователя, чем при отдельном просмотре элементов. Содержимое цифрового носителя дополняется интересной связанной информацией, а веб-страница может содержать ссылки на похожие материалы, создавая новые бизнес-возможности.

## <a name="htmlview-provides-a-better-user-experience"></a>Хтмлвиев обеспечивает более удобный пользовательский интерфейс

Пользователи хотят получать доступ к информации, которая может быть предоставлена с помощью цифрового мультимедийного содержимого, но не хотят работать с бесконечными последовательностью всплывающих объявлений. Использование всплывающих окон браузера для отображения рекламы в Интернете стало настолько распространенным, что есть программное обеспечение, препятствующее открытию этих окон. Это программное обеспечение может иметь нежелательный побочный результат, препятствующий отображению легальных веб-страниц, иногда запрещая всю презентацию с цифровым мультимедиа.

При использовании функции Хтмлвиев пользователям больше не нужно работать с всплывающими окнами. вместо того чтобы открывать отдельное окно браузера для предоставления пользователю дополнительных сведений, можно отобразить пользовательское веб-содержимое в **воспроизводимой** функции пользовательского интерфейса проигрыватель Windows Media, когда проигрыватель воспроизводит содержимое аудио или видео, которое требуется пользователю. пользователи могут управлять воспроизведением с помощью элементов управления проигрыватель Windows Media. Поскольку содержимое воспроизводится в полноэкранном проигрывателе, программное обеспечение, разработанное для предотвращения всплывающих объявлений, не будет препятствовать содержимому пользователей.

## <a name="htmlview-content-is-easy-to-create"></a>Хтмлвиев содержимое легко создать

Как предполагает имя функции, вы создаете веб-содержимое, отображаемое с помощью функции Хтмлвиев, с помощью HTML (HTML). Если вы уже создали содержимое для Интернета, это означает, что вам не нужно изучать новый язык программирования для простого создания богатого содержимого с помощью функции Хтмлвиев. если у вас уже есть веб-страницы со встроенным элементом управления проигрыватель Windows Media ActiveX, эти страницы можно будет отобразить в пользовательском интерфейсе проигрывателя, просто указав эти веб-страницы из метафайла Windows Media (файл с расширением asx) с помощью специального параметра.

проигрыватель Windows Media использует встроенный экземпляр Microsoft Internet Explorer для вывода содержимого хтмлвиев. Это означает, что при создании веб-страниц не нужно учитывать разные обозреватели Интернета, несколько моделей сценариев или разные языки сценариев. Даже если пользователь не использует Internet Explorer в качестве браузера по умолчанию, веб-страница по-прежнему отображается в проигрывателе правильно.

возможно, что программа, отличная от проигрыватель Windows Media, может зарегистрироваться в качестве программы по умолчанию для открытия файлов. asx. объектная модель проигрыватель Windows Media 9 Series или более поздней версии включает новый метод **опенплайер**, который позволяет указать универсальный указатель ресурсов (URL) для содержимого, которое нужно воспроизвести. при использовании метода **опенплайер** проигрыватель Windows Media всегда открывается в полноэкранном режиме для воспроизведения указанного содержимого. с помощью этого метода можно убедиться, что содержимое хтмлвиев отображается в проигрыватель Windows Media, а не в каком-либо другом приложении, которое заняло сопоставление типа файла. asx.
