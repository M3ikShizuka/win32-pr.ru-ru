# <a name="listboxsetselectedstate"></a>LISTBOX. Сетселектедстате

Метод **сетселектедстате** выбирает или отменяет выбор элемента с указанным индексом.

``` syntax
        elementID.setSelectedState(index, selected)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*номер*
</dt> <dd>

**Число** (**Long**), содержащее индекс элемента, который нужно выбрать или отменить.

</dd> <dt>

<span id="selected"></span><span id="SELECTED"></span>*выбрать*
</dt> <dd>

**Логическое** значение, указывающее, должен ли элемент быть выбран (true) или невыбранным (false).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Remarks

Используется для выбора или выделения нескольких строк.
