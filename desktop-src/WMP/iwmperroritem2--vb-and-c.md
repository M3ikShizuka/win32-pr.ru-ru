# <a name="iwmperroritem2-vb-and-c-interface"></a>Интерфейс IWMPErrorItem2 (VB и C#)

Предоставляет свойство для получения кода условия ошибки.

В дополнение к свойствам, унаследованным от **ивмперроритем**, интерфейс **IWMPErrorItem2** предоставляет следующее свойство.



| Свойство                                                                 | Описание                                          |
|--------------------------------------------------------------------------|------------------------------------------------------|
| [выполняет](wmplibiwmperroritem2-iwmperroritem2-condition--vb-and-c.md) | Возвращает значение, указывающее условие для ошибки. |



 

Получите интерфейс **IWMPErrorItem2** путем приведения значения, возвращаемого свойством **Item** (в C#, **метода Get \_ Item**) интерфейса **ивмперроритем** .

## <a name="members"></a>Элементы

Интерфейс **IWMPErrorItem2 (VB и C#)** не определяет никаких членов.
