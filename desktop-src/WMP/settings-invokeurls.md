# <a name="settingsinvokeurls"></a>Settings. Инвокеурлс

Свойство **инвокеурлс** указывает или получает значение, указывающее, должны ли события URL-адреса запускать веб-браузер.

## <a name="syntax"></a>Синтаксис

Player. Settings. Инвокеурлс

## <a name="possible-values"></a>Возможные значения

Это свойство является **логическим значением** для чтения и записи.



| Значение | Описание                                  |
|-------|----------------------------------------------|
| true  | По умолчанию. События URL-адреса должны запускать браузер. |
| false | События URL-адреса не должны запускать браузер.      |



 

## <a name="remarks"></a>Комментарии

Файлы мультимедиа могут содержать URL-адреса. При отправке URL-адреса элементу управления проигрывателя Windows Media он сначала передается обработчику событий **команду скрипта** независимо от значения в **инвокеурлс**. После выхода из **команду скрипта** проигрыватель Windows Media проверяет **инвокеурлс** , чтобы определить, следует ли запускать браузер по умолчанию с URL-адресом. Вы можете выборочно отображать URL-адреса, проверив их в **команду скрипта** и установив нужные **инвокеурлс** .

**Проигрыватель Windows Media 10 Mobile**: это свойство принимает или возвращает значение false.
