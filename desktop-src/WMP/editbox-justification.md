# <a name="editboxjustification"></a>EDITBOX. обоснование

Атрибут **обоснования** указывает или получает выравнивание текста внутри элемента управления EditBox.

``` syntax
        elementID.justification
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей одно из следующих значений.



| Значение  | Описание                                                        |
|--------|--------------------------------------------------------------------|
| Левый   | По умолчанию. Выровняйте текст по левому краю элемента управления "поле ввода". |
| Правый  | Выровняйте текст по правому краю элемента управления "поле ввода".         |
| Center | Выравнивает текст по горизонтали по центру элемента управления "поле ввода".  |



 

## <a name="remarks"></a>Remarks

Имеется поле, равное двум пикселям.
