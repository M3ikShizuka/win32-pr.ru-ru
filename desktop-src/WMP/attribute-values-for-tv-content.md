# <a name="attribute-values-for-tv-content"></a>Значения атрибутов для ТВ-содержимого

В этом разделе объект **Player** был определен следующим образом:


```C++
AxWMPLib.AxWindowsMediaPlayer Player;
using WMPLib;

```



Проигрыватель Windows Media 10 или более поздней версии может организовывать ТЕЛЕВИЗИОНное содержимое в библиотеке. Проигрыватель Windows Media обрабатывает ТЕЛЕВИЗИОНное содержимое как подкатегорию видеосодержимого. Чтобы видеосодержимое отображалось на ТВ-узлах в библиотеке, задайте атрибуты **WM/медиакласспримарид** и **WM/медиакласссекондарид** со значениями, приведенными в следующей таблице, с помощью *носителя*. метод **сетитеминфо** :



| attribute                    | Значение                                |
|------------------------------|--------------------------------------|
| **WM/Медиакласспримарид**   | DB9830BD-3AB3-4FAB-8A37-1A995F7FF74B |
| **WM/Медиакласссекондарид** | BA7F258A-62F7-47A9-B21F-4651C42A000E |



 

Эти значения также можно использовать для определения того, содержит ли конкретный элемент мультимедиа ТВ-содержимое с помощью *носителя*. **getItemInfo** или *Media*. методы **жетитеминфобитипе** .

Не забывайте использовать значения **GUID** в качестве **строковых** значений при указании или извлечении этих значений.

В следующем примере кода C# атрибуты класса мультимедиа устанавливаются таким образом, чтобы элемент мультимедиа был идентифицирован как ТВ-содержимое.


```C++
// Initialize the media object.
// This code assumes only 1 item named MyFile.
IWMPMedia3 media = (IWMPMedia3)Player.mediaCollection.getByName("MyFile").item(0);

// Set the primary media-class identifier.
media.setItemInfo("WM/MediaClassPrimaryID", "DB9830BD-3AB3-4FAB-8A37-1A995F7FF74B");

// Set the secondary media-class identifier.
media.setItemInfo("WM/MediaClassSecondaryID", "BA7F258A-62F7-47A9-B21F-4651C42A000E");

```



Дополнительные сведения о возможных значениях атрибутов класса мультимедиа см. в разделе рекомендации по [использованию метаданных мультимедиа Windows Media](/previous-versions/ms867702(v=msdn.10)).
