# <a name="installing-visualizations"></a>Установка визуализаций

Необходимо указать процесс установки для пользователя визуализации. Необходимо также предоставить процесс удаления для пользователя. текущая версия проигрыватель Windows Media не устанавливает визуализации из пользовательского интерфейса.

## <a name="installing-to-the-visualization-folder"></a>Установка в папку визуализации

рекомендуется установить все визуализации в подпапке визуализации папки, в которой установлена проигрыватель Windows Media.

## <a name="registering-your-visualization"></a>Регистрация визуализации

Визуализации представляют собой DLL-библиотеки COM и следуют всем обычным правилам установки и удаления. Для регистрации визуализации можно использовать regsvr32.exe или другие средства установки.
