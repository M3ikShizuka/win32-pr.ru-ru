# <a name="bitmaps-section"></a>Раздел точечных рисунков

Далее необходимо иметь раздел, который определяет каждый из файлов растрового изображения. Каждый тип точечного рисунка, который вы будете использовать, должен иметь назначенный ему файл, хотя можно использовать несколько типов, использующих один и тот же точечный рисунок.


```C++
[ Bitmaps ]

//  <Name>      <File name>     <X,Y>
//  ------      -----------     -----
    Background  Background.bmp  0,0
    Disabled    Disabled.bmp    0,0
    Pushed      Pushed.bmp      0,0
    Region      Region.bmp      0,0
    Super       Super.bmp       0,0

```



Раздел растровых изображений должен начинаться с точечных рисунков Word в квадратных скобках, а затем по строке для каждого типа точечного рисунка, который необходимо определить. В этом примере определено пять типов точечных рисунков. Дополнительные сведения см. в разделе [точечные](bitmaps.md) рисунки в справочнике по обложкам.

> [!Note]  
> в проигрыватель Windows Media 10 мобильных или более поздних обложек не рекомендуется использовать Region и Super bitmap.
