# <a name="ratings"></a>Рейтинги

При создании обложки для проигрывателя Windows Media 10,1 Mobile можно отобразить рейтинг, связанный с содержимым Windows Media Audio, которое сейчас воспроизводится. Оценка отображается в виде звезды с цифрой. Звездочка будет пронумерована от одной до пяти, обозначая текущую оценку этого содержимого. Если содержимое не оценивается, звездочка будет пронумерована нуль.

Раздел рейтингов файла определения обложки начинается со следующей строки:


```C++
[ Ratings ]

```



Затем необходимо добавить строку, содержащую сведения о размере и расположении значка оценки в обложке.


```C++
147, 3, 22, 21       ratings96S.gif

```



Для раздела рейтингов файла определения обложки можно использовать следующий шаблон:


```C++
// <Location>           <Image>
// ---------------      --------------------
```



Можно также назначить значение оценки элементу мультимедиа с помощью пользовательского интерфейса проигрывателя Windows Media 10,1 Mobile, а при следующем синхронизации устройства с компьютером новое значение рейтинга будет распространяться на этот элемент мультимедиа, если он находится в библиотеке проигрывателя Windows Media 10.
