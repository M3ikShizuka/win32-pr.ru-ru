# <a name="windows-media-metafile-elements-reference"></a>Windows Справочник по элементам метафайлов мультимедиа

в этом разделе содержится справочная документация по всем Windowsным элементам метафайлов мультимедиа для клиентских приложений. К записям ссылок относятся определения элементов, их атрибуты и значения, а также специальные условия, связанные с каждым элементом.

Для клиентских приложений поддерживаются следующие элементы метафайлов.



| Элемент                                           | Описание                                                                                                            |
|---------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [ВЫДЕЛЯЕТ](abstract-element.md)                  | Содержит текст, описывающий связанный элемент **ASX**, **Banner** или **entry** .                                 |
| [ASX](asx-element.md)                            | определяет файл как метафайл мультимедиа Windows.                                                                            |
| [КАЛЕНДАРЯ](author-element.md)                      | содержит имя автора клипа мультимедиа или метафайла мультимедиа Windows.                                           |
| [BANNER](banner-element.md)                      | задает URL-адрес для рисунка, отображаемого на панели отображения проигрыватель Windows Media.                               |
| [ИЗ](base-element.md)                          | указывает строку, добавляемую к началу url-адресов, отправленных в проигрыватель Windows Media.                                 |
| [Комментарии](comments.md)                          | Указывает синтаксис XML для комментариев ( <!--...--> ).                                                            |
| [Информация](copyright-element.md)                | Содержит сведения об авторских правах для элемента **ASX** или **entry** .                                                |
| [DURATION](duration-element.md)                  | указывает время, в течение которого проигрыватель Windows Media отображает поток.                                                    |
| [ендмаркер](endmarker-element.md)                | указывает маркер, по которому проигрыватель Windows Media прекращает отрисовку потока.                                           |
| [ОПЕРАЦИИ](entry-element.md)                        | Задает путь для клипа мультимедиа.                                                                                   |
| [ентриреф](entryref-element.md)                  | ссылки на элементы **записи** во внешнем Windows метафайле мультимедиа с расширением. asx,. мастика,. ввкс или. вмкс. |
| [ЖУРНАЛЕ](event-element.md)                        | определяет поведение или действие, предпринимаемое проигрыватель Windows Media при получении команды сценария, помеченной как событие.      |
| [логурл](logurl-element.md)                      | указывает проигрыватель Windows Media отправить данные журнала по указанному URL-адресу.                                            |
| [мореинфо](moreinfo-element.md)                  | Указывает URL-адрес для сайта, адрес электронной почты или команду сценария, связанные с показом, обрезками или баннером.               |
| [БАЙТ](param-element.md)                        | Указывает значение пользовательского параметра, связанного с списком воспроизведения или элементом списка воспроизведения.                      |
| [превиевдуратион](previewduration-element.md)    | Указывает продолжительность воспроизведения клипа в режиме предварительного просмотра.                                                         |
| [ЗНАЧ](ref-element.md)                            | Указывает URL-адрес для фрагмента мультимедийного содержимого.                                                                  |
| [ПРИДЕТСЯ](repeat-element.md)                      | указывает, сколько раз проигрыватель Windows Media повторяет один или несколько элементов **ENTRY** или **ентриреф** .             |
| [ОБЛОЖКа (не рекомендуется)](skin-element--deprecated.md) | Задает URL-адрес внедренной обложки.                                                                                   |
| [стартмаркер](startmarker-element.md)            | указывает маркер, по которому проигрыватель Windows Media начинает отрисовку потока.                                          |
| [STARTTIME](starttime-element.md)                | указывает время, когда проигрыватель Windows Media начнет отрисовку потока.                                        |
| [ТИТУЛЬ](title-element--metafile.md)              | Содержит заголовок элемента **ASX** или **entry** .                                                                 |
