# <a name="windows-media-metafile-elements-reference"></a>Справочник по элементам метафайлов Windows Media

Этот раздел содержит справочную документацию по всем элементам метафайлов Windows Media для клиентских приложений. К записям ссылок относятся определения элементов, их атрибуты и значения, а также специальные условия, связанные с каждым элементом.

Для клиентских приложений поддерживаются следующие элементы метафайлов.



| Элемент                                           | Описание                                                                                                            |
|---------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [ВЫДЕЛЯЕТ](abstract-element.md)                  | Содержит текст, описывающий связанный элемент **ASX**, **Banner** или **entry** .                                 |
| [ASX](asx-element.md)                            | Определяет файл как метафайл Windows Media.                                                                            |
| [КАЛЕНДАРЯ](author-element.md)                      | Содержит имя автора клипа мультимедиа или метафайла Windows Media.                                           |
| [BANNER](banner-element.md)                      | Задает URL-адрес для рисунка, отображаемого на панели отображения проигрывателя Windows Media.                               |
| [ИЗ](base-element.md)                          | Указывает строку, добавляемую к началу URL-адресов, отправляемых проигрывателю Windows Media.                                 |
| [Комментарии](comments.md)                          | Указывает синтаксис XML для комментариев ( <!--...--> ).                                                            |
| [Информация](copyright-element.md)                | Содержит сведения об авторских правах для элемента **ASX** или **entry** .                                                |
| [DURATION](duration-element.md)                  | Указывает длительность времени, в течение которого проигрыватель Windows Media визуализирует поток.                                                    |
| [ендмаркер](endmarker-element.md)                | Указывает маркер, по которому проигрыватель Windows Media останавливает отрисовку потока.                                           |
| [ОПЕРАЦИИ](entry-element.md)                        | Задает путь для клипа мультимедиа.                                                                                   |
| [ентриреф](entryref-element.md)                  | Ссылки на элементы **записи** во внешнем метафайле Windows Media с расширением. ASX,. мастика,. ввкс или. ВМКС. |
| [ЖУРНАЛЕ](event-element.md)                        | Определяет поведение или действие, предпринимаемое проигрывателем Windows Media при получении команды сценария, помеченной как событие.      |
| [логурл](logurl-element.md)                      | Предписывает проигрывателю Windows Media отправлять любые данные журнала по указанному URL-адресу.                                            |
| [мореинфо](moreinfo-element.md)                  | Указывает URL-адрес для сайта, адрес электронной почты или команду сценария, связанные с показом, обрезками или баннером.               |
| [БАЙТ](param-element.md)                        | Указывает значение пользовательского параметра, связанного с списком воспроизведения или элементом списка воспроизведения.                      |
| [превиевдуратион](previewduration-element.md)    | Указывает продолжительность воспроизведения клипа в режиме предварительного просмотра.                                                         |
| [ЗНАЧ](ref-element.md)                            | Указывает URL-адрес для фрагмента мультимедийного содержимого.                                                                  |
| [ПРИДЕТСЯ](repeat-element.md)                      | Указывает, сколько раз проигрыватель Windows Media повторяет один или несколько элементов **entry** или **ентриреф** .             |
| [ОБЛОЖКа (не рекомендуется)](skin-element--deprecated.md) | Задает URL-адрес внедренной обложки.                                                                                   |
| [стартмаркер](startmarker-element.md)            | Указывает маркер, по которому проигрыватель Windows Media начинает отрисовку потока.                                          |
| [STARTTIME](starttime-element.md)                | Указывает время, когда проигрыватель Windows Media начнет отрисовку потока.                                        |
| [ТИТУЛЬ](title-element--metafile.md)              | Содержит заголовок элемента **ASX** или **entry** .                                                                 |
