# <a name="initializing-the-page"></a>Инициализация страницы

При загрузке веб-страницы выполняется функция **init** . Этот код сначала извлекает все данные, хранящиеся в предыдущем сеансе. Дополнительные сведения см. в разделе [обслуживание сведений о сеансах](maintaining-session-information.md). Затем он создает объект **довнлоадманажер** и сохраняет его в глобальной переменной с именем g \_ оманажер.


```C++
g_oManager = external.DownloadManager;

```



Затем функция подключает событие **онколорчанже** к функции с именем **онаппколор** , а затем вызывает функцию напрямую для инициализации цвета фона. см. раздел [сопоставление цветов проигрыватель Windows Media](matching-the-windows-media-player-colors.md).

Наконец, функция запускает таймер с интервалом в 1 секунду и инициализирует списки, отображающие ожидающие загрузки коллекции и элементы загрузки. Это важно, так как в последний раз при закрытии Интернет-магазина были запущены сеансы, и эти сведения необходимо отобразить снова.
