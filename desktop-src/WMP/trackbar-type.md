# <a name="trackbar-type"></a>Тип TrackBar

Необходимо определить тип TrackBar, который необходимо использовать. В следующей таблице показаны допустимые типы.



| Тип   | Описание                                                                                                                                                                                                                                                                                                     |
|--------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Seek   | Управляет позицией воспроизведения. Если значение равно нулю, то элемент мультимедиа находится в начале. Значение увеличивается до достижения конца элемента мультимедиа. Пользователь может сдвинуть изображение бегунка, чтобы изменить значение, а при воспроизведении элемента мультимедиа изображение бегунка переместится в соответствии с текущей позицией воспроизведения. |
| Громкость | Управляет звуковым томом. Значение варьируется от 0 до 100. Пользователь может сдвинуть изображение бегунка, чтобы изменить том. Если том изменится, изображение эскиза будет перемещено в соответствие с новым томом.                                                                                                                   |
