# <a name="wmtracknumber-attribute"></a>Атрибут WM/Траккнумбер

Атрибут **WM/траккнумбер** является номером записи в альбоме, в котором он был первоначально выпущен.

## <a name="applies-to"></a>Применение

-   [Звуковые элементы](audio-item-attributes.md)
-   [Часто используемые атрибуты файлов Windows Media](commonly-used-windows-media-file-attributes.md)

## <a name="remarks"></a>Комментарии

Этот атрибут хранится как в библиотеке, так и в файле цифрового мультимедиа.

Номера отслеживаний для альбома начинаются с 1.

**Оригиналиндекс** и **оригиналиндекслефт** являются псевдонимами для этого атрибута.

Константа Windows Media Format SDK для этого атрибута — g \_ всзвмтраккнумбер.

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .
