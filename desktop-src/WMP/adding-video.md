# <a name="adding-video"></a>Добавление видео

Вы можете добавить видео в файл, просто используя элемент **Video** и определив место, где должно размещаться видеоокно.

Используйте тот же код, что и в разделе Выбор файлов. все, что нужно сделать, — это добавить элемент **Video** с атрибутами верхнего, левого, ширины и высоты.


```C++
        <VIDEO
            top = "10"
            left = "80"
            width = "180"
            height = "180"/>

```



Затем при воспроизведении видео оно отобразится в окне. Иллюстрация для примера видео была изменена, чтобы сделать обложку немного крупнее. Поскольку слои использовались в Photoshop, кнопки были легко перемещены, а новый набор был очень быстро создан. Было изменено только фоновое изображение. Заливка использовалась в пустом слое, и был добавлен эффект рельефа и выпуклости.

Аналогичные рабочие обложки можно увидеть в разделе с примерами пакета SDK.
