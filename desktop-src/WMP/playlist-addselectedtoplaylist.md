# <a name="playlistaddselectedtoplaylist"></a>Список воспроизведения. Аддселектедтоплайлист

Метод **аддселектедтоплайлист** добавляет выбранный элемент в список воспроизведения.

``` syntax
        elementID.addSelectedToPlaylist(playlist)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="playlist"></span><span id="PLAYLIST"></span>*списком*
</dt> <dd>

Объект **списка воспроизведения** , который получит выбранный элемент.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | Проигрыватель Windows Media версии 7,0 или более поздней<br/> |
