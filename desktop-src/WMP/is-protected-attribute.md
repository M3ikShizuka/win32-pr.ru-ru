# <a name="is_protected-attribute"></a>\_Защищенный атрибут

Атрибут **\_ protected** указывает, защищено ли содержимое с помощью управления цифровыми правами (DRM).

## <a name="applies-to"></a>Применение

-   [Звуковые элементы](audio-item-attributes.md)
-   [Часто используемые файлы Windows Media](commonly-used-windows-media-file-attributes.md)
-   [Элементы видео](video-item-attributes.md)

## <a name="remarks"></a>Комментарии

Этот атрибут хранится как в библиотеке, так и в файле цифрового мультимедиа.

**Дигиталлисекуре** является псевдонимом для этого атрибута.

Константа Windows Media Format SDK для этого атрибута — g \_ всзвмпротектед.

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .
