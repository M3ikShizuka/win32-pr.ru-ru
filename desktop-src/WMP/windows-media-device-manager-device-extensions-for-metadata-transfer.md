# <a name="windows-media-device-manager-device-extensions-for-metadata-transfer"></a>Расширения устройств Windows Media диспетчер устройств для передачи метаданных

Чтобы включить ускоренную пересылку метаданных, производители устройств, которые не поддерживают MTP, должны выполнить следующие действия в исходном коде:

-   Определение **\_ \_ \_ поддержки устройств WMP вмдм**.
-   Включите вмпдевицес. h, который устанавливается как часть пакета SDK для проигрывателя Windows Media.

Вмпдевицес. h определяет следующие структуры.



| Структура                                                                                 | Описание                                                                                                                                       |
|-------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| [Проигрыватель \_ WMP \_ вмдм \_ цикл \_ обработки метаданных \_ PC2DEVICE](/previous-versions/windows/desktop/api/wmpdevices/ns-wmpdevices-wmp_wmdm_metadata_round_trip_pc2device) | Структура, используемая проигрывателем Windows Media для запроса сведений о синхронизации метаданных с портативных устройств, которые не поддерживают MTP. |
| [Проигрыватель \_ WMP \_ вмдм \_ цикл \_ обработки метаданных \_ DEVICE2PC](/previous-versions/windows/desktop/api/wmpdevices/ns-wmpdevices-wmp_wmdm_metadata_round_trip_device2pc) | Структура, используемая проигрывателем Windows Media для получения данных ускоренной синхронизации метаданных с портативных устройств, которые не поддерживают MTP. |



 

Чтобы запросить от устройства сведения об измененных метаданных, проигрыватель Windows Media 10 или более поздней версии вызывает метод Windows Media диспетчер устройств Method **IWMDMDevice3::D евицеиоконтрол**. При выполнении этого вызова игрок выполняет определенные действия следующим образом:

-   Первый параметр, *двиоконтролкоде*, содержит константу **\_ \_ \_ цикла \_ обработки метаданных WMP ioctl**. Эта константа определена в вмпдевицес. h.
-   Второй параметр, *лпинбуффер*, указывает на структуру **проигрывателя WMP вмдм в виде \_ \_ \_ кругового пути \_ \_ PC2DEVICE** .
-   Третий параметр, *нинбуфферсизе*, содержит размер входного буфера.
-   Четвертый параметр, *лпаутбуффер*, указывает на структуру вмдм пути к **\_ \_ \_ циклической \_ обработки \_ метаданных проигрывателя WMP DEVICE2PC** . Устройство должно заполнить эту структуру сведениями об изменениях.
-   Пятый параметр *пнаутбуфферсизе* получает размер выходного буфера.
