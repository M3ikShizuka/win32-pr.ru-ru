# <a name="iwmpcdromripstartrip-method"></a>Метод Ивмпкдромрип:: Стартрип

Метод **стартрип** Рипс компакт-диска.

## <a name="syntax"></a>Синтаксис


```CSharp
public void startRip();
```


```VB

Public Sub startRip()
Implements IWMPCdromRip.startRip
```





## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Копирование компакт-диска с помощью интерфейса **ивмпкдромрип** действует так же, как копирование музыки с помощью пользовательского интерфейса проигрывателя Windows Media. Скопированное содержимое автоматически добавляется в библиотеку в соответствии с предпочтениями пользователя. Дополнительные сведения о параметрах пользователя для копирования компакт-дисков см. в разделе "Копирование музыки с CD" в справке проигрывателя Windows Media.
