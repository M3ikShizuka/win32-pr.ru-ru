# <a name="pixelaspectratioy"></a>пикселаспектратиой

Атрибут **пикселаспектратиой** является вертикальной частью пропорций видео.

## <a name="applies-to"></a>Применяется к

-   [Элементы видео](video-item-attributes.md)

## <a name="remarks"></a>Remarks

Для видео с пропорциями 16:9 **пикселаспектратиой** будет равно «9».
