# <a name="ambientattributesmovesizeto"></a>Амбиентаттрибутес. Мовесизето

Метод **мовесизето** перемещает элемент управления и задает новый размер элемента управления в новом расположении. Элемент управления перемещается и изменяет размер в анимированном виде за указанный период времени.

``` syntax
        elementID.moveSizeTo(newX, newY, newWidth, newHeight, moveTime, fSlide)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="newX"></span><span id="newx"></span><span id="NEWX"></span>*невкс*
</dt> <dd>

**Число** (**длинное**), определяющее новое значение **левого** атрибута элемента управления.

</dd> <dt>

<span id="newY"></span><span id="newy"></span><span id="NEWY"></span>*неви*
</dt> <dd>

**Число** (**длинное**), указывающее новое значение для **верхнего** атрибута элемента управления.

</dd> <dt>

<span id="newWidth"></span><span id="newwidth"></span><span id="NEWWIDTH"></span>*неввидс*
</dt> <dd>

**Число** (**длинное**), определяющее новое значение атрибута **Width** элемента управления.

</dd> <dt>

<span id="newHeight"></span><span id="newheight"></span><span id="NEWHEIGHT"></span>*невхеигхт*
</dt> <dd>

**Число** (**длинное**), указывающее новое значение для атрибута **Height** элемента управления.

</dd> <dt>

<span id="moveTime"></span><span id="movetime"></span><span id="MOVETIME"></span>*моветиме*
</dt> <dd>

**Число** (**длинное**), указывающее время в миллисекундах, которое требуется для перехода элемента управления в новое место.

</dd> <dt>

<span id="fSlide"></span><span id="fslide"></span><span id="FSLIDE"></span>*фслиде*
</dt> <dd>

**Логическое значение** , указывающее тип движения, создаваемого при перемещении элемента управления.



| Значение | Описание                                                             |
|-------|-------------------------------------------------------------------------|
| Да  | Движение является нелинейным (скользящее движение, которое ускоряется и замедляется). |
| false | Движение является линейным.                                                       |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Движение элемента управления может быть линейным или нелинейным. Линейное движение означает, что элемент управления перемещается по постоянной скорости в новое место, что приводит к внезапному запуску и остановке. Если указано нелинейное движение, создается скользящее движение, которое ускоряется от нуля в начале движения и замедляется до нуля в конце, что достигается гладкой остановкой.
