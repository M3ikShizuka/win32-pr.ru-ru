# <a name="download-manager"></a>Диспетчер загрузки

Диспетчер загрузки проигрывателя Microsoft Windows Media позволяет Интернет-магазинам типа 2 управлять процессом загрузки содержимого из области задач "проигрыватель Windows Media в Интернет-магазине". В области задач Интернет-магазина размещается веб-страница, которая отображается, когда пользователь выбирает использование Интернет-магазина.

Это введение в диспетчер загрузки содержит следующие разделы.

| [Архитектура диспетчера загрузки](download-manager-architecture.md) | Содержит сводку по объектной модели диспетчера загрузки.        |
