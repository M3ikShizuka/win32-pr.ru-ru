# <a name="slider-element"></a>Элемент SLIDER

Элемент **Slider** предоставляет способ создания и управления простым горизонтальным или вертикальным ползунком. Он поддерживает следующие атрибуты и обработчики событий. Для удобства также предоставляются предопределенные элементы **Slider** .

Элемент **Slider** поддерживает следующие атрибуты.



| attribute                                                 | Описание                                                                                                       |
|-----------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [backgroundColor](slider-backgroundcolor.md)             | Указывает или получает цвет фона элемента управления "ползунок".                                                |
| [баккграундендколор](slider-backgroundendcolor.md)       | Указывает или получает конечный цвет фона элемента управления "ползунок".                                         |
| [баккграундховеримаже](slider-backgroundhoverimage.md)   | Указывает или получает фоновое изображение ползунка, отображаемого при наведении на него указателя мыши.      |
| [backgroundImage](slider-backgroundimage.md)             | Указывает или получает фоновое изображение по умолчанию для ползунка.                                                |
| [бордерсизе](slider-bordersize.md)                       | Указывает или получает размер границы в пикселях.                                                                 |
| [курсор](slider-cursor.md)                               | Указывает или получает значение, указывающее, какой тип курсора отображается при наведении мыши на ползунок. |
| [direction](slider-direction.md)                         | Указывает или получает направление, в котором размещаются изображения ползунка.                                             |
| [дисабледколор](slider-disabledcolor.md)                 | Указывает или получает отключенный цвет элемента управления "ползунок".                                                  |
| [дисабледимаже](slider-disabledimage.md)                 | Указывает или получает изображение ползунка, отображаемого при отключенном элементе управления.                         |
| [foregroundColor](slider-foregroundcolor.md)             | Указывает или получает цвет переднего плана для элемента управления "ползунок".                                                |
| [фореграундендколор](slider-foregroundendcolor.md)       | Указывает или получает конечный цвет переднего плана для элемента управления "ползунок".                                         |
| [фореграундховеримаже](slider-foregroundhoverimage.md)   | Указывает или получает фоновое изображение ползунка, отображаемого при наведении на него указателя мыши.      |
| [фореграундимаже](slider-foregroundimage.md)             | Указывает или получает изображение переднего плана по умолчанию для ползунка.                                                |
| [фореграундпрогресс](slider-foregroundprogress.md)       | Указывает или получает текущую положение индикатора выполнения переднего плана в процентах от области ползунка.    |
| [max](slider-max.md)                                     | Указывает или получает максимальное значение диапазона, определенного элементом управления "ползунок".                              |
| [min](slider-min.md)                                     | Указывает или получает минимальное значение диапазона, определенного элементом управления "ползунок".                              |
| [слайд](slider-slide.md)                                 | Указывает или получает значение, указывающее, будет ли изображение переднего плана поверх фонового изображения.          |
| [сумбдисабледимаже](slider-thumbdisabledimage.md)       | Указывает или получает отключенное изображение бегунка элемента управления "ползунок".                                            |
| [сумбдовнимаже](slider-thumbdownimage.md)               | Указывает или получает изображение, представляющее состояние бегунка.                                        |
| [сумбховеримаже](slider-thumbhoverimage.md)             | Указывает или получает изображение бегунка, которое появляется при наведении на него указателя мыши.                  |
| [сумбимаже](slider-thumbimage.md)                       | Указывает или получает изображение, которое будет использоваться для представления текущей положения ползунка.               |
| [крытия](slider-tiled.md)                                 | Указывает или получает значение, указывающее, будут ли изображения на ползунке перемещаться по мозаике.                                |
| [Сказок](slider-tooltip.md)                             | Указывает или получает текст подсказки для элемента управления "ползунок".                                                   |
| [транспаренциколор](slider-transparencycolor.md)         | Указывает или получает прозрачный цвет изображений ползунка.                                                |
| [усефореграундпрогресс](slider-useforegroundprogress.md) | Указывает или получает значение, указывающее, будет ли использоваться основной индикатор выполнения.                       |
| [value](slider-value.md)                                 | Указывает и получает текущую положение ползунка.                                                       |



 

Элемент **Slider** может реализовывать следующие обработчики событий.



| Обработчик событий                                   | Описание                                                                                                          |
|-------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [ондрагбегин](slider-ondragbegin.md)           | Обрабатывает событие, возникающее, когда пользователь нажимает и удерживает левую кнопку мыши вниз и начинает перетаскивание мышью. |
| [ондраженд](slider-ondragend.md)               | Обрабатывает событие, возникающее при отпускании левой кнопки мыши после операции перетаскивания.                      |
| [онпоситиончанже](slider-onpositionchange.md) | Обрабатывает событие, возникающее при изменении положения ползунка в результате нажатия или перетаскивания пользователем.   |



 

Элемент **Slider** поддерживает атрибуты окружения и может реализовывать обработчики событий окружения. Дополнительные сведения см. в разделе [атрибуты окружения](ambient-attributes.md) и [обработчики событий окружения](ambient-event-handlers.md).

Стандартные ползунки являются обычными элементами **Slider** с различными параметрами общих атрибутов, заданными по умолчанию. Доступны следующие стандартные ползунки.



| Предопределенный ПОЛЗУНок                  | Описание                                                    |
|------------------------------------|----------------------------------------------------------------|
| [баланцеслидер](balanceslider.md) | **Ползунок** , используемый для настройки баланса звука.                        |
| [сикслидер](seekslider.md)       | **Ползунок** , используемый для поиска любой позицией в файле мультимедиа. |
| [волумеслидер](volumeslider.md)   | **Ползунок** , используемый для настройки громкости звука.                         |
