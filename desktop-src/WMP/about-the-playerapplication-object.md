# <a name="about-the-playerapplication-object"></a>Сведения об объекте Плайераппликатион

объект **плайераппликатион** используется для удаленного взаимодействия проигрыватель Windows Media. он предоставляет функциональные возможности, необходимые для переключения между удаленным элементом управления проигрыватель Windows Media и полным режимом проигрывателя.

удаленное взаимодействие позволяет элементу управления проигрыватель Windows Media, внедренному в приложение C++, использовать тот же модуль воспроизведения, что и проигрыватель. Это означает, что обычно вы будете использовать объект **плайераппликатион** в коде C++ с помощью COM-интерфейсов. однако существует особый случай, в котором можно внедрить элемент управления проигрыватель Windows Media, отображающий обложку проигрыватель Windows Media в качестве пользовательского интерфейса. в этом случае **плайераппликатион** можно программировать с помощью JScript в коде обложки.
