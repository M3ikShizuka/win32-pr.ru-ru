# <a name="creating-the-pushed-file"></a>Создание отправленного файла

Вы можете взять созданные элементы и повторно использовать их для создания других необходимых образов. Объединив слои и выкрасить результаты, можно создать помещенный точечный рисунок, содержащий изображения, которые должны отображаться при нажатии пользователем кнопки. Этот файл также содержит дополнительный образ для приостановленного состояния кнопки Плайпаусе.

Следующий рисунок — это отправленный файл.

![Отправленный файл](images/ceswmpus.png)

Две кнопки справа относятся к нормальному приостановленному состоянию и состоянию отложенной кнопки Плайпаусе. Цвет кнопок является желтым, за исключением нормального состояния "Плайпаусе" кнопки "в качестве хорошего контраста с зеленым цветом, используемым для нормального состояния всех кнопок.
