# <a name="wmparentalrating-attribute"></a>Атрибут WM/Паренталратинг

Атрибут **WM/паренталратинг** является родительским рейтингом содержимого.

## <a name="applies-to"></a>Применение

-   [Звуковые элементы](audio-item-attributes.md)
-   [Часто используемые атрибуты файлов Windows Media](commonly-used-windows-media-file-attributes.md)
-   [DVD-диски](dvd-attributes.md)
-   [Элементы видео](video-item-attributes.md)

## <a name="remarks"></a>Комментарии

Этот атрибут хранится как в библиотеке (или в кэше), так и в файле цифрового носителя.

Константа Windows Media Format SDK для этого атрибута — g \_ всзвмпаренталратинг.

**Мпааратинг** является псевдонимом для этого атрибута.

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .
