# <a name="listboxreadonly"></a>LISTBOX. readOnly

Атрибут **ReadOnly** указывает или получает значение, указывающее, доступен ли текст только для чтения или может быть выбран пользователем. Можно задать только во время разработки.

``` syntax
        elementID.readOnly
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим значением** для чтения и записи.



| Значение | Описание                                |
|-------|--------------------------------------------|
| Да  | Текст доступен только для чтения.                         |
| false | По умолчанию. Пользователь может выбрать текст. |

|--------------------|---------------------------------------------------------|
