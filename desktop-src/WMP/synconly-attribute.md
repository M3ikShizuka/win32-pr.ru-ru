# <a name="synconly-attribute"></a>Атрибут Синконли

атрибут **синконли** представляет собой строковое представление **логического** значения, которое проигрыватель Windows Media использует, чтобы определить, доступен ли список воспроизведения только для синхронизации.

## <a name="applies-to"></a>Применяется к

-   [Списки воспроизведения](playlist-attributes-ref.md)

## <a name="remarks"></a>Remarks

Значение 1 указывает, что список воспроизведения доступен только для синхронизации и не может отображаться в узле **автосписок воспроизведения** . Нулевое значение указывает, что список воспроизведения может отображаться в узле **автосписок воспроизведения** .

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .
