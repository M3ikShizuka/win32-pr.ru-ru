# <a name="playlistdisableditemcolor"></a>Список воспроизведения. Дисабледитемколор

Атрибут **дисабледитемколор** указывает или получает цвет отключенной записи компакт-дисков или содержимое в сети в автономном режиме.

``` syntax
        elementID.disabledItemColor
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей любое значение цвета Microsoft Internet Explorer. Он имеет значение по умолчанию «грайтекст».

## <a name="requirements"></a>Requirements (Требования)



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
