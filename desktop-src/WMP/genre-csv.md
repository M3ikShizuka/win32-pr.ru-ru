# <a name="genrecsv"></a>genre.csv

Этот файл содержит запись для каждого жанра, представленного в каталоге. Каждая запись представляет собой строку, состоящую из полей, разделенных табуляцией, описанных в таблице ниже. Поля должны присутствовать в указанном порядке.

Все поля в genre.csv являются обязательными.

В столбце Format в следующей таблице описывается способ форматирования каждого текстового поля в Юникоде. Он не ссылается на тип данных содержимого. Например, если в столбце Format указано целое число, это означает, что поле содержит строку в Юникоде, представляющую целочисленное значение, а не фактическое целое число.



| Поле             | Формат                                                    | Описание                                                                                                                                        |
|-------------------|-----------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| Идентификатор жанра \_         | Неотрицательное целое число. Пример: 22<br/>               | Идентификатор жанра, уникальный в пределах genre.csv. Должно быть меньше 2 ^ 32. Допускается не более 64 жанров.                                             |
| женретитле        | Строка в Юникоде. Пример: рок<br/>                   | Отображаемое имя жанра.                                                                                                                                |
| фидсареаваилабле | Логический. формат: \[ 0 \| 1\]<br/> Пример: 0<br/> | Указывает, возможно ли поведение "Радио Play", переходя к каналу.                                                                          |
| хаскомпосер       | Логический. формат: \[ 0 \| 1\]<br/> Пример: 1<br/> | Значение true, если этот жанр должен сохранять сведения о Composer в каталоге. Если значение не задано, сведения Composer игнорируются и не компилируются в каталог. |
