# <a name="dropdownplaylist"></a>дропдовнплайлист

Это предопределенный элемент **списка воспроизведения** со следующими значениями по умолчанию.

``` syntax
playlistItemsVisible="false"
```

## <a name="remarks"></a>Комментарии

При этом будет создан простой раскрывающийся **список воспроизведения** без отображения элементов. Все свойства этого **списка воспроизведения** можно переопределить, явно указав их.

|--------------------|----------------------------------------------|
