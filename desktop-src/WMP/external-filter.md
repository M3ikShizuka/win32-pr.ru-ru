# <a name="externalfilter"></a>Внешний фильтр

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Свойство **Filter** извлекает фильтр поиска, используемый проигрывателем Windows Media в настоящий момент.

``` syntax
window.external.filter
      
```

## <a name="possible-values"></a>Возможные значения

Это свойство является **строкой**, доступная только для чтения.

## <a name="remarks"></a>Комментарии

Фильтр поиска — это текст, который пользователь вводит в элемент управления колесиком мыши игрока.
