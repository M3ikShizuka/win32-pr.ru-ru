# <a name="ffwdbutton"></a>ффвдбуттон

Это предопределенная **кнопка** со следующими значениями по умолчанию.


```C++
onclick="jscript:player.controls.fastForward()"
upToolTip="Fast Forward"
cursor="system"
enabled="wmpenabled:player.controls.fastForward"

```



## <a name="remarks"></a>Комментарии

Все свойства этой **кнопки** можно переопределить, явно указав их.
