# <a name="imagebutton"></a>IMAGEBUTTON

Это предопределенная **кнопка** со следующим значением по умолчанию.

``` syntax
cursor="Hand"
```

## <a name="remarks"></a>Remarks

Все кнопки могут содержать изображения. Этот элемент полезен для уточнения кода, чтобы указать кнопку, которая в основном используется в качестве контейнера изображений.

Все свойства этой **кнопки** можно переопределить, явно указав их.
