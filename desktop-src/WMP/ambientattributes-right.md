# <a name="ambientattributesright"></a>Амбиентаттрибутес. Right

**Правый** атрибут указывает или получает правую координату элемента управления.

``` syntax
        elementID.right
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**длинным**), представляющим расстояние в пикселях от элемента управления до правого края родительского **представления** или подчиненного **представления**. Поведение для отрицательных значений или если [амбиентаттрибутес. Width](ambientattributes-width.md) не задано, не определено.
