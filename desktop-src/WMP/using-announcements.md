# <a name="using-announcements"></a>Использование объявлений

Объявление — это файл, содержащий сведения о URL-адресе для потока мультимедиа, включая IP-адрес многоадресной рассылки, порт, формат потока и другие параметры станции. объявления создаются Windows администратором носителей при создании одноадресного или многоадресного потока публикации. Клиент может быстро загрузить файл объявления, а затем перейти к файлу потокового мультимедиа.

Для точки одноадресной рассылки открывается поток мультимедиа пункта публикации. Для точки многоадресной рассылки URL-адрес извлекается из файла станции вещания с расширением NSC, а доступ к потоковой передаче осуществляется. В отличие от одноадресного потока сведения о заголовке не содержатся в потоке многоадресной рассылки. Эти сведения поступают из файла станции вещания с расширением NSC. проигрыватель Windows Media, как правило, сначала открывает файл извещения, который используется для списков воспроизведения метафайлов, указывающий на расположение файла станции вещания.

Пример кода


```XML
<ASX VERSION="3.0">
    <TITLE>title</TITLE>
    <ENTRY>
        <REF HREF="mms://proseware.com/pubpoint"/>
    </ENTRY>
</ASX>

```
