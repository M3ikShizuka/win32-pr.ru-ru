# <a name="creating-custom-sliders"></a>Создание настраиваемых ползунков

Вы можете создавать настраиваемые ползунки в любой форме. В этом примере выбирается простая полоса, но фактическая фигура может быть любой. Ниже приведен код для элемента **кустомслидер** :


```C++
<CustomSlider 
  top="160"
  left="130"
  min="0"
  max="100"
  toolTip="volume control"
  image="slider.bmp"
  positionImage="graymap.bmp"
  enabled="true"
  value="wmpprop:player.settings.volume"
  value_onchange="player.settings.volume = value" />

```



Настраивает начальное значение для ползунка. Появились два новых точечных рисунка. Один из них — точечный рисунок в градациях серого (slider.bmp), который определяет, какие значения будут использоваться при щелчке, а другая (slider.bmp), определяющая изображение, которое будет отображаться при нажатии на определенную часть оттенков серого.

Начальное значение определяется путем прослушивания тома с помощью вмппроп, а затем том может быть изменен, когда пользователь щелкает часть ползунка, которая активирует изменение значения.

Аналогичную обложку рабочего ползунка можно увидеть в разделе примеров пакета SDK.
