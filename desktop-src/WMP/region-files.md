# <a name="region-files"></a>Файлы регионов

Файлы регионов требуются, если используется любой тип кнопки нажатия (2PushHit, Пушхит или Тогглехит).

Файлы регионов используются для определения областей, которые будут реагировать на касание, которое также называется нажатием, на определенной кнопке. Для каждого переключателя область в битовой карте области получает определенный веб-цвет (например \# , FF0000, который является значением сплошного красного цвета). Номер цвета задается в определении кнопки региона. Когда пользователь отображает обложку, изображение кнопки переносится на задний план с помощью координат области в битовой карте области.

Например, можно нарисовать красный круг в месте, соответствующем расположению кнопки Далее, и окрасить его в красный цвет ( \# FF0000). Затем в определении кнопки можно присвоить значение RGB, равное 255, 0, 0 (то есть RGB-эквивалент \# FF0000). В этом случае кнопка Далее будет реагировать только на касания (попадания) в красном круге.

Кнопки нажатия используются, если требуется определить фигуры, отличные от прямоугольников. Необходимо по-прежнему определить координаты каждой кнопки, чтобы дополнительные образы, такие как отправленные и отключенные, могли быть правильно обнаружены. На практике каждая кнопка ограничена прямоугольником, и эти мнимые ограничивающие прямоугольники не должны перекрываться.

> [!Note]  
> Графические файлы не нужны в обложек проигрывателя Windows Media 10 для мобильных устройств, так как типы кнопок не поддерживаются в проигрывателе Windows Media 10 Mobile или более поздней версии.
