# <a name="text-alignment"></a>Выравнивание текста

Необходимо определить выравнивание текстового поля, которое будет использоваться. Значения, приведенные в следующей таблице, являются допустимыми.



| Значение  | Описание                |
|--------|----------------------------|
| Левый   | Текст выровняйтеся по левому краю.  |
| Center | Текст выравнивается по центру.      |
| Правый  | Текст выровняйтеся по правому краю. |
