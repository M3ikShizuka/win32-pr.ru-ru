# <a name="iwmpcontrols-vb-and-c-interface"></a>Интерфейс Ивмпконтролс (VB и C#)

предоставляет способ управления воспроизведением элемента мультимедиа за счет представления элементов управления транспорта проигрыватель Windows Media, таких как воспроизведение, остановка и пауза.

Интерфейс **ивмпконтролс** предоставляет следующие свойства.

## <a name="members"></a>Элементы

Интерфейс **ивмпконтролс (VB и C#)** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ивмпконтролс (VB и C#)** содержит следующие методы.



| Метод                                                                       | Описание                                                                                 |
|:-----------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------|
| [**фастфорвард**](wmplibiwmpcontrols-iwmpcontrols-fastforward--vb-and-c.md) | Запускает быстрое воспроизведение элемента мультимедиа в прямом направлении.<br/>                     |
| [**фастреверсе**](wmplibiwmpcontrols-iwmpcontrols-fastreverse--vb-and-c.md) | Запускает быстрое воспроизведение элемента мультимедиа в обратном направлении.<br/>                     |
| [**Далее**](wmplibiwmpcontrols-iwmpcontrols-next--vb-and-c.md)               | Задает следующий элемент списка воспроизведения в качестве текущего элемента.<br/>                          |
| [**работу**](wmplibiwmpcontrols-iwmpcontrols-pause--vb-and-c.md)             | Приостанавливает воспроизведение элемента мультимедиа.<br/>                                               |
| [**воспроизводит**](wmplibiwmpcontrols-iwmpcontrols-play--vb-and-c.md)               | Начинает воспроизведение текущего элемента мультимедиа или возобновляет воспроизведение приостановленного элемента.<br/> |
| [**плайитем**](wmplibiwmpcontrols-iwmpcontrols-playitem--vb-and-c.md)       | Воспроизводит указанный элемент мультимедиа.<br/>                                                  |
| [**прошлом**](wmplibiwmpcontrols-iwmpcontrols-previous--vb-and-c.md)       | Устанавливает предыдущий элемент списка воспроизведения в качестве текущего элемента.<br/>                      |
| [**позиции**](wmplibiwmpcontrols-iwmpcontrols-stop--vb-and-c.md)               | Останавливает воспроизведение элемента мультимедиа.<br/>                                                |
