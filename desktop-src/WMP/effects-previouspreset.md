# <a name="effectspreviouspreset"></a>EFFECTs. Превиауспресет

Метод **превиауспресет** Отображает предыдущую предустановку текущей визуализации.

``` syntax
        elementID.previousPreset()
```

## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Если текущая Предустановка является первой в списке для текущей визуализации, последняя Предустановка становится текущей.
