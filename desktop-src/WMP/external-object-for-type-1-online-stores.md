# <a name="external-object-for-type-1-online-stores"></a>Внешний объект для Интернет-магазинов типа 1

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

**внешний** объект предоставляет функции веб-страниц, предоставляемые интернет-магазином, размещенным в проигрыватель Windows Media.

**Внешний** объект предоставляет следующие свойства для Интернет-магазинов типа 1.



| Свойство                                                                                  | Описание                                                                                                                              |
|-------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [accountType](external-accounttype.md)                                                   | Возвращает тип учетной записи текущего пользователя.                                                                                          |
| [аппколорбуттонхигхлигхт](external-appcolorbuttonhighlight.md)                           | извлекает цвет выделения текущей кнопки для пользовательского интерфейса проигрыватель Windows Media.                                                |
| [аппколорбуттонховерфаце](external-appcolorbuttonhoverface.md)                           | возвращает текущий цвет для наведения указателя на кнопку проигрыватель Windows Media пользовательского интерфейса.                                                    |
| [аппколорбуттоншадов](external-appcolorbuttonshadow.md)                                 | извлекает цвет тени текущей кнопки для проигрыватель Windows Media пользовательского интерфейса.                                                   |
| [аппколордарк](external-appcolordark.md)                                                 | извлекает текущий темно-затененный цвет проигрыватель Windows Media пользовательского интерфейса.                                                      |
| [аппколорлигхт](external-appcolorlight.md)                                               | извлекает текущий цвет светло-затененного пользовательского интерфейса проигрыватель Windows Media.                                                     |
| [аппколормедиум](external-appcolormedium.md)                                             | извлекает текущий средний затененный цвет проигрыватель Windows Media пользовательского интерфейса.                                                    |
| [баскеттитле](external-baskettitle.md)                                                   | получает заголовок кнопки на панели списка (также называемой корзиной) в проигрыватель Windows Media.                                     |
| [filter](external-filter.md)                                                             | получает фильтр поиска, используемый в данный момент проигрыватель Windows Media.                                                                    |
| [игнореиехистори](external-ignoreiehistory.md)                                           | указывает, должен ли проигрыватель Windows Media игнорировать журнал Internet Explorer.                                                          |
| [либрарилокатионид](external-librarylocationid.md)                                       | Возвращает идентификатор определенного элемента мультимедиа, который в настоящий момент отображается в представлении проигрывателя.                                      |
| [либрарилокатионтипе](external-librarylocationtype.md)                                   | извлекает [константу расположения библиотеки](library-location-constants.md) , которая указывает тип текущего представления в проигрыватель Windows Media. |
| [плугинруннинг](external-pluginrunning.md)                                               | Извлекает значение, указывающее, выполняется ли подключаемый модуль интернет-магазина.                                                          |
| [селектедитемид](external-selecteditemid.md)                                             | возвращает идентификатор элемента мультимедиа, выбранного в данный момент в проигрыватель Windows Media.                                           |
| [селектедитемтипе](external-selecteditemtype.md)                                         | извлекает тип элемента мультимедиа, выбранного в данный момент в проигрыватель Windows Media.                                                 |
| [задачи](external-task.md)                                                                 | Возвращает имя текущей области задач.                                                                                             |
| [темплатебеингдисплайединлокаллибрари](external-templatebeingdisplayedinlocallibrary.md) | Указывает, отображается ли веб-канал, представленный текущей страницей обнаружения, в элементе управления представления дерева локальной библиотеки.          |
| [усерлогжедин](external-userloggedin.md)                                                 | Получает значение, указывающее, вошел ли пользователь в Интернет-магазин.                                                          |
| [version](external-version.md)                                                           | извлекает текущую версию проигрыватель Windows Media.                                                                                   |
| [виевпараметерс](external-viewparameters.md)                                             | извлекает параметры, связанные с текущим представлением в проигрыватель Windows Media.                                                           |



 

**Внешний** объект предоставляет следующие методы для Интернет-магазинов типа 1.



| Метод                                                            | Описание                                                                                                                  |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [аддтобаскет](external-addtobasket.md)                           | добавляет элементы мультимедиа на панель списка (также называемую корзиной) в проигрыватель Windows Media.                                          |
| [аттемптлогин](external-attemptlogin.md)                         | Отображает диалоговое окно, чтобы пользователь мог попытаться войти в Интернет-магазин.                                                 |
| [authenticate](external-authenticate.md)                         | Инициирует попытку проверки подлинности пользователя.                                                                               |
| [купить](external-buy.md)                                           | Инициирует покупку набора элементов мультимедиа.                                                                              |
| [канцелнавигате](external-cancelnavigate.md)                     | информирует проигрыватель Windows Media о том, что не следует отображать новую страницу обнаружения, даже если представление изменилось в проигрывателе. |
| [чанжевиев](external-changeview.md)                             | изменяет представление в проигрыватель Windows Media.                                                                                    |
| [чанжевиевонлинелист](external-changeviewonlinelist.md)         | изменяет представление в проигрыватель Windows Media для отображения списка, динамически созданного интернет-магазином.                        |
| [download](external-download.md)                                 | Инициирует скачивание набора элементов мультимедиа.                                                                              |
| [воспроизводит](external-play.md)                                         | указывает проигрыватель Windows Media воспроизвести набор элементов мультимедиа.                                                                 |
| [савекуррентвиевтолибрари](external-savecurrentviewtolibrary.md) | Создает список воспроизведения из элементов мультимедиа в текущем представлении и сохраняет список воспроизведения в локальной библиотеке.                     |
| [sendMessage](external-sendmessage.md)                           | Отправляет сообщение подключаемому модулю Интернет-магазина.                                                                               |
| [showPopup](external-showpopup.md)                               | указывает проигрыватель Windows Media отображать всплывающую веб-страницу; то есть веб-страница, которая отображается в отдельном окне.            |



 

**Внешний** объект предоставляет следующие события для Интернет-магазинов типа 1.



| Событие                                                                         | Описание                                                                             |
|-------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [ончанжевиеверрор](external-onchangeviewerror-event.md)                     | Происходит, когда вызов внешнего метода **. чанжевиев** приводит к ошибке.           |
| [ончанжевиевонлинелистеррор](external-onchangeviewonlinelisterror-event.md) | Происходит, когда вызов внешнего метода **. чанжевиевонлинелист** приводит к ошибке. |
| [онколорчанже](external-oncolorchange-event.md)                             | происходит при изменении цвета проигрыватель Windows Media пользовательского интерфейса.               |
| [онлогинчанже](external-onloginchange-event.md)                             | Происходит при изменении состояния входа пользователя или при сбое попытки входа в систему.        |
| [онсендмессажекомплете](external-onsendmessagecomplete-event.md)             | Происходит, когда Интернет-магазин завершил обработку сообщения.                         |
| [онвиевчанже](external-onviewchange-event.md)                               | происходит при изменении представления в проигрыватель Windows Media.                                   |
