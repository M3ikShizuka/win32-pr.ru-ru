# <a name="external-object-for-type-1-online-stores"></a>Внешний объект для Интернет-магазинов типа 1

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

**Внешний** объект предоставляет функции веб-страниц, предоставляемые Интернет-магазином, размещенный в проигрывателе Windows Media.

**Внешний** объект предоставляет следующие свойства для Интернет-магазинов типа 1.



| Свойство                                                                                  | Описание                                                                                                                              |
|-------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [accountType](external-accounttype.md)                                                   | Возвращает тип учетной записи текущего пользователя.                                                                                          |
| [аппколорбуттонхигхлигхт](external-appcolorbuttonhighlight.md)                           | Извлекает цвет выделения текущей кнопки для пользовательского интерфейса проигрывателя Windows Media.                                                |
| [аппколорбуттонховерфаце](external-appcolorbuttonhoverface.md)                           | Извлекает цвет пользовательского интерфейса проигрывателя Windows Media для текущего нажатия кнопки.                                                    |
| [аппколорбуттоншадов](external-appcolorbuttonshadow.md)                                 | Получает текущий цвет тени для кнопки пользовательского интерфейса проигрывателя Windows Media.                                                   |
| [аппколордарк](external-appcolordark.md)                                                 | Извлекает текущий темно-затененный цвет пользовательского интерфейса проигрывателя Windows Media.                                                      |
| [аппколорлигхт](external-appcolorlight.md)                                               | Извлекает цвет пользовательского интерфейса проигрывателя Windows Media с светлой тенью.                                                     |
| [аппколормедиум](external-appcolormedium.md)                                             | Извлекает текущий цвет в пользовательском интерфейсе проигрывателя Windows Media.                                                    |
| [баскеттитле](external-baskettitle.md)                                                   | Получает заголовок кнопки на панели списка (также называемой корзиной) в проигрывателе Windows Media.                                     |
| [filter](external-filter.md)                                                             | Получает фильтр поиска, используемый проигрывателем Windows Media в настоящий момент.                                                                    |
| [игнореиехистори](external-ignoreiehistory.md)                                           | Указывает, должен ли проигрыватель Windows Media игнорировать журнал Internet Explorer.                                                          |
| [либрарилокатионид](external-librarylocationid.md)                                       | Возвращает идентификатор определенного элемента мультимедиа, который в настоящий момент отображается в представлении проигрывателя.                                      |
| [либрарилокатионтипе](external-librarylocationtype.md)                                   | Извлекает [константу расположения библиотеки](library-location-constants.md) , которая указывает тип текущего представления в проигрывателе Windows Media. |
| [плугинруннинг](external-pluginrunning.md)                                               | Извлекает значение, указывающее, выполняется ли подключаемый модуль интернет-магазина.                                                          |
| [селектедитемид](external-selecteditemid.md)                                             | Возвращает идентификатор элемента мультимедиа, выбранного в данный момент в проигрывателе Windows Media.                                           |
| [селектедитемтипе](external-selecteditemtype.md)                                         | Извлекает тип элемента мультимедиа, выбранного в данный момент в проигрывателе Windows Media.                                                 |
| [задачи](external-task.md)                                                                 | Возвращает имя текущей области задач.                                                                                             |
| [темплатебеингдисплайединлокаллибрари](external-templatebeingdisplayedinlocallibrary.md) | Указывает, отображается ли веб-канал, представленный текущей страницей обнаружения, в элементе управления представления дерева локальной библиотеки.          |
| [усерлогжедин](external-userloggedin.md)                                                 | Получает значение, указывающее, вошел ли пользователь в Интернет-магазин.                                                          |
| [version](external-version.md)                                                           | Извлекает текущую версию проигрывателя Windows Media.                                                                                   |
| [виевпараметерс](external-viewparameters.md)                                             | Извлекает параметры, связанные с текущим представлением в проигрывателе Windows Media.                                                           |



 

**Внешний** объект предоставляет следующие методы для Интернет-магазинов типа 1.



| Метод                                                            | Описание                                                                                                                  |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [аддтобаскет](external-addtobasket.md)                           | Добавляет элементы мультимедиа на панель списка (также называемую корзиной) в проигрывателе Windows Media.                                          |
| [аттемптлогин](external-attemptlogin.md)                         | Отображает диалоговое окно, чтобы пользователь мог попытаться войти в Интернет-магазин.                                                 |
| [authenticate](external-authenticate.md)                         | Инициирует попытку проверки подлинности пользователя.                                                                               |
| [купить](external-buy.md)                                           | Инициирует покупку набора элементов мультимедиа.                                                                              |
| [канцелнавигате](external-cancelnavigate.md)                     | Информирует проигрыватель Windows Media, что не следует отображать новую страницу обнаружения, даже если представление изменилось в проигрывателе. |
| [чанжевиев](external-changeview.md)                             | Изменяет представление в проигрывателе Windows Media.                                                                                    |
| [чанжевиевонлинелист](external-changeviewonlinelist.md)         | Изменяет представление в проигрывателе Windows Media для отображения списка, созданного динамически в Интернет-магазине.                        |
| [download](external-download.md)                                 | Инициирует скачивание набора элементов мультимедиа.                                                                              |
| [воспроизводит](external-play.md)                                         | Указывает проигрывателю Windows Media воспроизвести набор элементов мультимедиа.                                                                 |
| [савекуррентвиевтолибрари](external-savecurrentviewtolibrary.md) | Создает список воспроизведения из элементов мультимедиа в текущем представлении и сохраняет список воспроизведения в локальной библиотеке.                     |
| [sendMessage](external-sendmessage.md)                           | Отправляет сообщение подключаемому модулю Интернет-магазина.                                                                               |
| [showPopup](external-showpopup.md)                               | Указывает проигрывателю Windows Media отображать всплывающую веб-страницу. то есть веб-страница, которая отображается в отдельном окне.            |



 

**Внешний** объект предоставляет следующие события для Интернет-магазинов типа 1.



| Событие                                                                         | Описание                                                                             |
|-------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [ончанжевиеверрор](external-onchangeviewerror-event.md)                     | Происходит, когда вызов внешнего метода **. чанжевиев** приводит к ошибке.           |
| [ончанжевиевонлинелистеррор](external-onchangeviewonlinelisterror-event.md) | Происходит, когда вызов внешнего метода **. чанжевиевонлинелист** приводит к ошибке. |
| [онколорчанже](external-oncolorchange-event.md)                             | Происходит при изменении цвета пользовательского интерфейса проигрывателя Windows Media.               |
| [онлогинчанже](external-onloginchange-event.md)                             | Происходит при изменении состояния входа пользователя или при сбое попытки входа в систему.        |
| [онсендмессажекомплете](external-onsendmessagecomplete-event.md)             | Происходит, когда Интернет-магазин завершил обработку сообщения.                         |
| [онвиевчанже](external-onviewchange-event.md)                               | Происходит при изменении представления в проигрывателе Windows Media.                                   |
