# <a name="coordinates"></a>Координаты

Это определяет координаты x и y изображения относительно фонового файла. Требуется два положительных целых числа, разделенные запятыми. В зависимости от типа определяемого вами изображения необходимо использовать два типа координат. В следующей таблице показаны допустимые значения координат для типов точечных рисунков.



| Тип точечного рисунка               | Координаты значений                                                             |
|---------------------------|-------------------------------------------------------------------------------|
| Фон или супер       | 0, 0                                                                           |
| Отключено, Push-уведомления или регион | Левый верхний угол изображения относительно фонового изображения в пикселях. |
