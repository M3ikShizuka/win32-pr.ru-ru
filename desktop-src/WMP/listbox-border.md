# <a name="listboxborder"></a>LISTBOX. Border

Атрибут **border** указывает или получает значение, указывающее, имеет ли элемент управления "список" границу.

``` syntax
        elementID.border
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **логическим** значением.



| Значение | Описание                                  |
|-------|----------------------------------------------|
| true  | По умолчанию. Элемент управления "список" имеет границу.  |
| false | Элемент управления "список" не имеет границы. |



 

## <a name="remarks"></a>Комментарии

Граница отображается как утопленная граница из двух пикселей вокруг элемента управления "список" с системными цветами.

|--------------------|---------------------------------------------------------|
