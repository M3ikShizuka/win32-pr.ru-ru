# <a name="registry-entries-for-tracking-installation-progress"></a>Записи реестра для отслеживания хода установки

программа установки проигрыватель Windows Media 11, программа установки \_wm.exe, записывает в реестр следующие записи, чтобы программы установки могли отслеживать ход выполнения программы установки проигрыватель Windows Media.


```C++
[HKEY_LOCAL_MACHINE\Software\Microsoft\MediaPlayer\Setup]
"Progress_CurrentDialog" = dword:value
"Progress_MaxDialog" = dword:value
"Progress_CurrentInstall" = dword:value
"Progress_MaxInstall" = dword:value
```



В приведенном выше синтаксисе реестра *value* является заполнителем для целочисленного значения.

куррентдиалог хода выполнения \_ указывает, какое диалоговое окно проигрыватель Windows Media программа установки в данный момент отображается. максдиалог хода выполнения \_ указывает общее количество диалоговых окон, которые будут отображаться Windows Media. например, если \_ куррентдиалог = 2 и progress \_ максдиалог = 5, проигрыватель Windows Media в настоящий момент отображает второе диалоговое окно в последовательности, равной пяти.

\_Куррентинсталл и ход выполнения \_ максинсталл, взятые вместе, указывают процент завершения текущего диалогового окна. Например, если \_ куррентинсталл = 6 и Progress \_ максинсталл = 25, то текущим диалоговым окном является 6/25 (то есть 24%) завершено.
