# <a name="buttongroup-element"></a>BUTTONGROUP, элемент

Элемент **BUTTONGROUP** предоставляет способ группировки нескольких кнопок в обложке. Эти кнопки можно указать с помощью элементов **буттонелемент** в качестве потомков элемента **BUTTONGROUP** .

Элемент **BUTTONGROUP** поддерживает следующие атрибуты.



| attribute                                              | Описание                                                                                                                                                                                                                     |
|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [буттонкаунт](buttongroup-buttoncount.md)             | Получает количество кнопок в **BUTTONGROUP**.                                                                                                                                                                         |
| [курсор](buttongroup-cursor.md)                       | Указывает или получает тип курсора, который появляется при наведении указателя мыши на кнопку в **BUTTONGROUP**.                                                                                                                  |
| [дисабледимаже](buttongroup-disabledimage.md)         | Указывает или получает имя изображения, представляющего отключенное состояние кнопок в **BUTTONGROUP**.                                                                                                             |
| [довнимаже](buttongroup-downimage.md)                 | Указывает или получает имя изображения, представляющего состояние **BUTTONGROUP**.                                                                                                                                |
| [ховердовнимаже](buttongroup-hoverdownimage.md)       | Указывает или получает имя изображения, представляющего состояние наведения указателя кнопки в **BUTTONGROUP**. Состояние наведения указателя мыши возникает, когда кнопка находится в состоянии Down, и пользователь наводит на нее указатель мышью. |
| [ховеримаже](buttongroup-hoverimage.md)               | Указывает или получает имя изображения, представляющего состояние наведения указателя на кнопку в **BUTTONGROUP**. Состояние наведения наведением возникает, когда кнопка находится в состоянии "вверх", и пользователь наводит на нее указатель мыши.             |
| [хуешифт](buttongroup-hueshift.md)                   | Указывает или получает величину, на которую смещается оттенок изображений **BUTTONGROUP** .                                                                                                                                    |
| [image](buttongroup-image.md)                         | Указывает или получает имя изображения, представляющего кнопки **BUTTONGROUP**.                                                                                                                                     |
| [маппингимаже](buttongroup-mappingimage.md)           | Указывает или получает имя изображения, представляющего карту кнопок **BUTTONGROUP**.                                                                                                                                |
| [radio](buttongroup-radio.md);                         | Указывает или получает значение, указывающее, состоит ли **BUTTONGROUP** из переключателей.                                                                                                                             |
| [насыщенность](buttongroup-saturation.md)               | Указывает или получает значение насыщенности изображений **BUTTONGROUP** .                                                                                                                                                      |
| [шовбаккграунд](buttongroup-showbackground.md)       | Указывает или получает значение, указывающее, отображает ли **BUTTONGROUP** только кнопки, или отображает полный точечный рисунок, указанный в атрибуте **Image** .                                                              |
| [транспаренциколор](buttongroup-transparencycolor.md) | Указывает или получает прозрачный цвет изображений **BUTTONGROUP** .                                                                                                                                                     |



 

Элемент **BUTTONGROUP** поддерживает следующие методы.



| Метод                                 | Описание                                                                                     |
|----------------------------------------|-------------------------------------------------------------------------------------------------|
| [Откройте](buttongroup-click.md)         | Вызывает обработчик событий **OnClick** , определенный для **буттонелемент** с указанным индексом. |
| [Кнопка](buttongroup-getbutton.md) | Извлекает **буттонелемент** с указанным индексом.                                       |



 

Элемент **BUTTONGROUP** поддерживает атрибуты окружения и может реализовывать обработчики событий окружения. Дополнительные сведения см. в разделе [атрибуты окружения](ambient-attributes.md) и [обработчики событий окружения](ambient-event-handlers.md).
