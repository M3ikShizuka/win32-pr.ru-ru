# <a name="about-the-sample-audio-dsp-plug-in"></a>Сведения о примере подключаемого модуля DSP аудиоподсистемы

Пример подключаемого модуля DSP аудиоподсистемы предоставляет простую реализацию обработки, которая масштабирует амплитуду звука по фактору, предоставленному пользователем на странице свойств. Подключаемый модуль принимает значения от 0 до 1. Значение по умолчанию — 1. Значение 1 не влияет на звук; другие коэффициенты масштабирования — это множители для звуковых образцов. Например, значение 0,5 приведет к снижению объема тома в 6 шкала. Нулевое значение приводит к отсутствию тишины.

Пример подключаемого модуля DSP для аудиоподсистемы работает с аудио-или моно PCM.
