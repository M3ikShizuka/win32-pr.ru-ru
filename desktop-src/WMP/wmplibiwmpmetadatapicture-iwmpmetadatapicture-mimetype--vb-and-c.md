# <a name="iwmpmetadatapicturemimetype-property"></a>Свойство Ивмпметадатапиктуре:: mimeType

`mimeType`Свойство получает тип MIME изображения, представленного атрибутом метаданных.

## <a name="syntax"></a>Синтаксис


```CSharp
public System.String mimeType {get; set;}
```


```VB

Public ReadOnly Property mimeType As System.String
```





## <a name="property-value"></a>Значение свойства

**Строка System. String** , которая является типом MIME изображения.

## <a name="remarks"></a>Комментарии

Перед использованием этого свойства необходимо иметь доступ на чтение к библиотеке. Дополнительные сведения см. в разделе [доступ к библиотеке](library-access.md).
