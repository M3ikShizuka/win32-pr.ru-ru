# <a name="buttondowntooltip"></a>КНОПКА. Довнтултип

Атрибут **довнтултип** указывает или получает текст подсказки, отображаемый при наведении указателя мыши на **кнопку** , и **кнопка** находится в состоянии Down.

``` syntax
        elementID.downToolTip
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи со значением по умолчанию "" (пустая строка) и максимальной длиной 1024 символов.

## <a name="remarks"></a>Комментарии

Если для этого атрибута задано значение "" (пустая строка), подсказка не отображается.
