# <a name="durationtext"></a>дуратионтекст

Это Стандартный **текстовый** элемент со следующими значениями по умолчанию.

``` syntax
value="wmpprop:player.currentMedia.DurationString"
tabstop="true"
justification="right"
```

## <a name="remarks"></a>Комментарии

При этом будет создан **текстовый** элемент, в котором будет отображаться длительность воспроизведения мультимедиа. Все свойства этого **текстового** элемента можно переопределить, явно указав их.
