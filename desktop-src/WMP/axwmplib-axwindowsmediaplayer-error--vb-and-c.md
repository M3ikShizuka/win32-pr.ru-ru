# <a name="axwindowsmediaplayererror-property"></a>Аксвиндовсмедиаплайер. Error, свойство

Свойство Error получает интерфейс Ивмперрор, предоставляющий доступ к коллекции интерфейсов Ивмперроритем.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```CSharp
public IWMPError Error {get;}
```


```VB

Public ReadOnly Property Error As IWMPError
```





## <a name="property-value"></a>Значение свойства

Интерфейс Вмплиб. Ивмперрор.
