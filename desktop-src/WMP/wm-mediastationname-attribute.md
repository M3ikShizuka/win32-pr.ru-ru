# <a name="wmmediastationname-attribute"></a>Атрибут WM/Медиастатионнаме

Атрибут **WM/статионнаме** указывает имя станции, на которой было первоначально вещание данного показа.

## <a name="applies-to"></a>Применение

-   [Элементы видео](video-item-attributes.md)

## <a name="remarks"></a>Комментарии

Константа Windows Media Format SDK для этого атрибута — g \_ всзвммедиастатионнаме.

Чтобы определить, можно ли изменить значение этого атрибута, используйте метод [Media. исреадонлитем](media-isreadonlyitem.md) .
