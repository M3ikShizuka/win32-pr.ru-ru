# <a name="externaltask"></a>Внешний. задача

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Свойство **Task** извлекает имя текущей области задач.

## <a name="syntax"></a>Синтаксис

``` syntax
window.external.task
```

## <a name="possible-values"></a>Возможные значения

Это свойство является **строкой**, доступная только для чтения. Возможные значения: обзор, запись и синхронизация.
