# <a name="using-the-windows-media-player-control-in-a-web-page"></a>использование элемента управления проигрыватель Windows Media на веб-странице

внедрение элемента управления проигрыватель Windows Media на веб-странице позволяет полностью настроить способ взаимодействия пользователя с элементом управления. Вы можете использовать интерфейс, предоставляемый элементом управления, или скрыть его и отобразить собственный пользовательский интерфейс. можно указать несколько свойств проигрыватель Windows Media элементов управления в точке встраивания элемента управления или задать свойства проигрывателя и вызвать методы проигрывателя в коде скрипта.

> [!Note]  
> мобильный элемент управления проигрыватель Windows Media 10 содержит функциональные возможности, основанные на подмножестве функциональных возможностей, предоставляемых в классической версии элемента управления проигрыватель Windows Media. Таким образом, его можно внедрить в веб-страницу Pocket Internet Explorer таким же образом, как и элемент управления Desktop на веб-странице Internet Explorer. чтобы узнать, поддерживается ли конкретный метод, свойство или событие для мобильного элемента управления проигрыватель Windows Media 10, см. раздел [справочник по объектной модели для создания сценариев](object-model-reference-for-scripting.md).
