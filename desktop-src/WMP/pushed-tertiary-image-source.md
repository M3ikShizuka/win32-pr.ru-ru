# <a name="pushed-tertiary-image-source"></a>Источник отправленного третьего образа

Функция кнопки Плайпаусестоп требует, чтобы вы определили расположение отправленного изображения для третьего состояния кнопки. Это будет изображение, которое пользователи видят при нажатии кнопки Плайпаусестоп при потоковой передаче широковещательной передачи в реальном времени.

Чтобы определить этот образ, необходимо ввести тип изображения, за которым следует пробел, символ @ и еще один пробел. Затем необходимо ввести два положительных целых числа, определяющие координаты верхнего левого угла (в пикселях) изображения, которое нужно использовать внутри типа точечного рисунка.

Например, чтобы определить отправленный образ для источника третьего образа, если изображение находится внутри точечного рисунка, введите:


```C++
Pushed @ 324,0

```



В третичных состояниях не может быть отключенного образа. Предполагается, что для третьего и дополнительного образов используются одинаковые значения ширины и высоты.
