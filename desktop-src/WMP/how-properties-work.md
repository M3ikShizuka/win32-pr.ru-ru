# <a name="how-properties-work"></a>Как работают свойства

Значения свойств хранятся в переменных членов.

Свойства становятся доступными парами методов. Один метод предоставляет реализацию для указания значения свойства; его имя начинается с «вставить \_ ». Другой метод предоставляет реализацию для получения значения свойства. его имя начинается с Get \_ . Определение интерфейса находится в файле Echo. idl. Прототипы методов свойств находятся в Echo. h. Они реализуются в Echo. cpp.

В следующих трех разделах показано, как изменить существующие методы свойств в соответствии со своими потребностями и как добавить методы для дополнительного свойства.

-   [Изменение свойства Scale](modifying-the-scale-property.md)
-   [Добавление свойства "Mix!"](adding-the-wet-mix-property.md)
