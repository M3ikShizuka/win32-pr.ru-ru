# <a name="mediacollectionchange-event-of-the-axwindowsmediaplayer-object"></a>Событие Медиаколлектиончанже объекта Аксвиндовсмедиаплайер

Событие Медиаколлектиончанже возникает при изменении коллекции носителей.

``` syntax
[C#]
private void player_MediaCollectionChange(
  object sender,
  System.EventArgs e
)

[Visual Basic]
Private Sub player_MediaCollectionChange(
  sender As Object,
  e As System.EventArgs
) Handles player.MediaCollectionChange
```

## <a name="event-data"></a>Данные о событиях

Это событие не содержит данных.
