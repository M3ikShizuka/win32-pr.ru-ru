# <a name="effectsnextpreset"></a>EFFECTs. Некстпресет

Метод **некстпресет** отображает следующую предустановку текущей визуализации.

``` syntax
        elementID.nextPreset()
```

## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Remarks

Если текущая Предустановка является последней в списке для текущей визуализации, первая Предустановка становится текущей.
