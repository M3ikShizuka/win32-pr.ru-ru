# <a name="pausebutton"></a>паусебуттон

Это предопределенная **кнопка** со следующими значениями по умолчанию.

``` syntax
onclick="jscript:player.controls.pause()" 
upToolTip="Pause"
cursor="system"
enabled="wmpenabled:player.controls.pause"
```

## <a name="remarks"></a>Комментарии

Все свойства этой **кнопки** можно переопределить, явно указав их.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
