# <a name="reference-for-type-2-online-stores"></a>Справочные материалы по интернет-магазинам типа 2

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

проигрыватель Windows Media 10 или более поздней версии предоставляет объекты, интерфейсы, свойства и методы, поддерживающие интернет-магазины типа 2. Эти сведения подробно описаны в следующих разделах.
