# <a name="applying-an-update-to-a-catalog"></a>Применение обновления к каталогу

> [!Note]  
> Это обычно не выполняется, за исключением целей диагностики — например, для проверки допустимости файла различий. каталог, созданный таким образом, не сжат в форме и не должен передаваться в проигрыватель Windows Media.

 

Новый файл каталога можно создать с помощью приведенного ниже синтаксиса, где *инпуткаталог* — это расположение исходного каталога, а *инпутдифф* — расположение файла различий для применения. Файлы каталога должны быть в несжатом виде.


```C++
catcomp applydiff <inputcatalog> <inputdiff>
```



Например, в следующем примере создается новый каталог из C: \\ Catalog210 \\ Catalog. Вмдб и C: \\ Catalog210 \\ Catalog. diff.


```C++
catcomp applydiff C:\Catalog210\catalog.wmdb C:\Catalog210\catalog.diff
```



Если компиляция прошла успешно, catcomp.exe создает следующие выходные файлы.



| Имя файла        | Описание       |
|------------------|-------------------|
| Catalog. вмдб. New | Новый файл каталога. |
