# <a name="introduction-to-the-online-stores-welcome-kit"></a>Знакомство с начальным набором для Интернет-магазинов

начальный набор для интернет-магазинов содержит сведения о том, как присоединять электронное хранилище digital media на доске для проигрыватель Windows Media. проигрыватель Windows Media поддерживает два типа интернет-магазинов: тип 1 и тип 2. сведения о разнице между хранилищами типа 1 и 2 см. в статье [проигрыватель Windows Media интернет-магазинов](windows-media-player-online-stores.md).

Процесс адаптации, описанный в следующих разделах, относится к хранилищам типа 1 и 2, за исключением некоторых различий, которые отмечены.

## <a name="eligibility"></a>Право на участие

чтобы иметь возможность использовать интернет-магазин, интегрированный с проигрыватель Windows Media, ваша компания должна соответствовать требованиям к адаптации и требованиям к обслуживанию, указанным в соглашении об интернет-магазинах. Типы требований в соглашении включают, но не ограничиваются следующим.

-   ваша организация должна иметь права на распространение интернет-содержимого с помощью проигрыватель Windows Media.
-   ваша компания должна согласиться с тем, чтобы для пользователей проигрыватель Windows Media было доступно определенное количество дорожек.
-   Ваша организация должна предоставить определенный уровень непрерывной работы службы.

Некоторые особые требования соглашения выделяются в [требованиях к магазинам Интернет-музыки](requirements-for-online-music-stores.md).

## <a name="contacting-microsoft"></a>Обращение в Майкрософт

если у вас возникли вопросы относительно интернет-магазинов и процесса их включения, обратитесь к виртуальной группе проигрыватель Windows Media Services по адресу mpsvctm@microsoft.com .
