# <a name="sliderthumbdownimage"></a>ПОЛЗУНок. Сумбдовнимаже

Атрибут **сумбдовнимаже** указывает или получает изображение, представляющее состояние бегунка.

``` syntax
        elementID.thumbDownImage
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **строкой** для чтения и записи, содержащей имя файла изображения.

## <a name="remarks"></a>Комментарии

**Сумбдовнимаже** является необязательным и является изображением, используемым для представления состояния отключения **сумбимаже**. Если **сумбдовнимаже** не предоставлен, ползунок будет использовать **сумбимаже**.

Поддерживаются форматы BMP, JPG, PNG и GIF (не включая анимированные GIF).
