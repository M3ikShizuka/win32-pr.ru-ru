# <a name="customslidermax"></a>КУСТОМСЛИДЕР. Max

Атрибут **Max** указывает или получает максимальное значение диапазона, определенного настраиваемым ползунком.

``` syntax
        elementID.max
```

## <a name="possible-values"></a>Возможные значения

Этот атрибут является **числом** для чтения и записи (**float**) и значением по умолчанию 100.

## <a name="remarks"></a>Комментарии

Значение **Max** должно быть больше значения **min**.
