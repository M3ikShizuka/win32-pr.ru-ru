# <a name="serviceinfo-element"></a>ServiceInfo, элемент

> [!Note]  
> В этом разделе описываются функции, предназначенные для использования в Интернет-магазинах. Использование этой функции вне контекста Интернет-магазина не поддерживается.

 

Элемент **serviceInfo** является основным элементом для ServiceInfo.xml документа.

``` syntax
<ServiceInfo
    Version = 1.00
    Key = "service key"
    ContentPartner = "true|false"
/>
```

## <a name="attributes"></a>Атрибуты



| Термин                                                                                                                                             | Описание                                                                                                                                                                                                                                                    |
|--------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="Version__optional_"></span><span id="version__optional_"></span><span id="VERSION__OPTIONAL_"></span>**Версия** (необязательно)<br/> | Версия файла ServiceInfo.xml. для проигрыватель Windows Media поддерживается версия 1,00.<br/>                                                                                                                                                            |
| <span id="Key__required_"></span><span id="key__required_"></span><span id="KEY__REQUIRED_"></span>**Ключ** (обязательно)<br/>                 | Строка ключа службы, однозначно определяющая Интернет-магазин.<br/>                                                                                                                                                                                   |
| <span id="ContentPartner"></span><span id="contentpartner"></span><span id="CONTENTPARTNER"></span>**контентпартнер**<br/>                 | Указывает, является ли Интернет-магазин хранилищем типа 1. Значение "true" указывает, что хранилищем является хранилище типа 1. Значение "false" указывает, что хранилище не является хранилищем типа 1; то есть, это хранилище типа 2. Значение по умолчанию — «false».<br/> |

## <a name="remarks"></a>Remarks

В следующей таблице приведены параметры, отправленные с помощью запроса URL-адреса. Другие могут присутствовать в целях совместимости с прежними версиями. запрос также будет включать все параметры, указанные с помощью параметра командной строки сервицеекстра программы установки проигрыватель Windows Media.



| Имя         | Значение                                                                                                                                                               |
|--------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *графическом*      | идентификатор географического расположения Windows. Идентификатор расположения задается пользователем в области **Расположение** в разделе региональные и языковые параметры панели управления. |
| *locale*     | проигрыватель Windows Media код локали.                                                                                                                                     |
| *UserLocale* | Windows код локали. Языковой стандарт задается пользователем в области **стандарты и форматы** языковых и региональных параметров на панели управления.        |
| *version*    | проигрыватель Windows Media номер версии в следующем формате: 10.0. x. xxxx или 11.0. x. xxxx.                                                                         |
