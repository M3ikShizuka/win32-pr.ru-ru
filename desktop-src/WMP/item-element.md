# <a name="item-element"></a>ITEM, элемент

Элемент **Item** представляет элемент в списке или всплывающих элементах управления. Элементы **Item** всегда являются дочерними элементами элементов **ListBox** или **Popup** .

Элемент **Item** поддерживает следующие атрибуты.



| attribute               | Описание                                                                  |
|-------------------------|------------------------------------------------------------------------------|
| [value](item-value.md) | Указывает или получает текст, отображаемый для поля со списком или всплывающего элемента. |



 

> [!Note]  
> для этого элемента требуется проигрыватель Windows Media для Windows XP или более поздней версии.
