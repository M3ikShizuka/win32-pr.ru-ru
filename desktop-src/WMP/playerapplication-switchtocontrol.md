# <a name="playerapplicationswitchtocontrol-method"></a>Плайераппликатион. Свитчтоконтрол, метод

метод **свитчтоконтрол** переключает удаленный элемент управления проигрыватель Windows Media в закрепленное состояние.

## <a name="syntax"></a>Синтаксис


```JScript
PlayerApplication.switchToControl()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Remarks

этот метод используется только при удаленном взаимодействии с элементом управления проигрыватель Windows Media.

**проигрыватель Windows Media 10 Mobile:** Этот метод не поддерживается.
