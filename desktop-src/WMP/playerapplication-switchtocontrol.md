# <a name="playerapplicationswitchtocontrol-method"></a>Плайераппликатион. Свитчтоконтрол, метод

Метод **свитчтоконтрол** переключает удаленный элемент управления проигрывателя Windows Media на закрепленное состояние.

## <a name="syntax"></a>Синтаксис


```JScript
PlayerApplication.switchToControl()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Этот метод используется только при удаленном взаимодействии с элементом управления проигрывателя Windows Media.

**Проигрыватель Windows Media 10 Mobile:** Этот метод не поддерживается.
