# <a name="status-windows-media-player-sdk"></a>Состояние (пакет SDK для проигрывателя Windows Media)

Вы можете использовать отображение состояния в обложке. Если это так, элемент Status должен быть определен в файле определения обложки. В качестве альтернативы можно также отобразить эти сведения с помощью атрибута Status в элементе text.

Раздел Status файла определения обложки начинается со следующей строки:


```C++
[ Status ]

```



Затем необходимо добавить одну или несколько строк, содержащих сведения о состоянии, отображаемых в обложке.


```C++
    On           45,193,120,13    Left    Tahoma,8,B        0,255,  0

```



Для раздела Status в файле определения обложки можно использовать следующий шаблон:


```C++
//  <Item>       <Location>     <Align>  <Font>          <Color>
//  ------       ----------     -------  ------          -------

```



Необходимо использовать порядок, показанный в предыдущем шаблоне, для отображения сведений о состоянии для каждой строки в разделе состояние. Каждая часть строки является обязательной. В следующих разделах описываются все элементы.

-   [Элемент состояния](status-item.md)
-   [Расположение состояния](status-location.md)
-   [Выравнивание состояния](status-alignment.md)
-   [Шрифт состояния](status-font.md)
-   [Цвет состояния](status-color.md)

Пример кода состояния см. в [разделе Sample Status](sample-status-section.md).
