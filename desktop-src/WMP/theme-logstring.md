# <a name="themelogstring"></a>THEME. Логстринг

Метод **логстринг** записывает определяемую пользователем строку в файл ошибок, если ведение журнала включено.

``` syntax
        theme.logString(message)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="message"></span><span id="MESSAGE"></span>*Сообщение*
</dt> <dd>

**Строка** , указывающая сообщение для записи в журнал.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|------------------------------------------------------|
| Версия<br/> | проигрыватель Windows Media версии 7,0 или более поздней<br/> |
