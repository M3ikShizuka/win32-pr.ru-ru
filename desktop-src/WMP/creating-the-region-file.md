# <a name="creating-the-region-file"></a>Создание файла регионов

Вам потребуется создать файл области, который показывает, какие области реагируют на касания. Можно просто скопировать базовые уровни каждого элемента в новый слой и окрасить их в соответствии с цветовыми номерами, которые будут использоваться в файле определения обложки. Убедитесь, что изображения, создаваемые на этом слое, являются сплошными и никакие эффекты не применяются. Вы, вероятно, захотите записать цветовые номера для каждого изображения. Цветовые номера можно получить с помощью средства выбора цвета Photoshop. Необходимо, чтобы число RGB было равно трем десятичным числам. Каждое число в диапазоне от 0 до 255. Например, сплошной красный цвет будет 255, 0, 0.

> [!Note]  
> Файлы регионов не используются в обложек проигрывателя Windows Media 10 для мобильных устройств, так как типы кнопок не поддерживаются в проигрывателе Windows Media 10 Mobile или более поздней версии.
