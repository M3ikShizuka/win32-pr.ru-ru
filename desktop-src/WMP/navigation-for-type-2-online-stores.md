# <a name="navigation-for-type-2-online-stores"></a>Навигация для Интернет-магазинов типа 2

Веб-страницы, размещенные в проигрывателе Windows Media 10 или более поздней версии, например в панелях задач службы "Интернет-магазин", могут использовать метод [External. навигатетаскпанеурл](external-navigatetaskpaneurl.md) для управления навигацией в проигрывателе Windows Media. Навигация может осуществляться следующими способами.

-   Между панелями задач службы. См. раздел [Переход между областями задач службы](navigating-between-service-task-panes.md) .
-   Из таких функций, как представление центра информации, веб-страницы Хтмлвиев и страница сведений о альбоме для панелей задач служб. См. раздел [переходы из функций проигрывателя Windows Media](navigating-from-windows-media-player-features.md).
-   Между Интернет-магазинами. Ознакомьтесь [с навигацией по другим Интернет-магазинам](navigating-to-other-online-stores.md).
