# <a name="dsp-plug-in-user-overview"></a>Обзор подключаемого пользователя DSP

для пользователя подключаемые модули DSP — это надстройки, которые по каким-либо причинам изменяют способ воспроизведения пользователем аудио или видео, проигрыватель Windows Media. Результат использования подключаемого модуля DSP может быть очевидным, как, например, цвета в видеоизображении отменяются, так что он выглядит как изображение негативно или что-то очень незаметно, например, при светлой отспаЦиауснессе звука.

пользователи могут устанавливать и удалять подключаемые модули DSP с помощью вкладки " **подключаемые модули** " диалогового окна "параметры" в проигрыватель Windows Media. После этого пользователи могут включать и отключать установленные подключаемые модули с помощью флажков на вкладке " **подключаемые модули** ". подключаемые модули DSP могут реализовывать страницу свойств, к которой пользователи также могут обращаться из вкладки " **подключаемые модули** ". Страница свойств предоставляет пользователю способ изменения параметров, влияющих на способ работы подключаемого модуля DSP. Например, страница свойств может включать элемент управления "ползунок", который позволяет пользователю изменить время Decay для измененного результата.
