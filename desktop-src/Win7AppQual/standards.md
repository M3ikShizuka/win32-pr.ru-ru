# <a name="standards"></a>Стандарты

Windows Internet explorer 8 содержит новый обработчик макетов и визуализации, который полностью реализует CSS 2,1 и включает дополнительную совместимость с HTML 4,0 (в отличие от предыдущих версий Windows Internet Explorer). Широкая поддержка веб-стандартов в Internet Explorer 8 может привести к неправильной работе некоторых веб-приложений, поскольку они не написаны для Microsoft Internet Explorer 6 или не соответствуют стандартам. Приложения могут завершаться сбоем разными способами, включая изображения или текст, которые отображаются в разных местах или в целых приложениях, которые визуализируются таким образом, чтобы приложение стало непригодным для использования.

Internet Explorer 8 также содержит Улучшенный механизм JavaScript. Этот новый обработчик скриптов повышает производительность, поддерживает новые методы, обеспечивает повышенную безопасность и изменяет некоторые функции для работы в соответствии с более стандартными стандартами. Улучшенная производительность не приводит к проблемам совместимости приложений, но другие изменения могут привести к тому, что некоторые веб-приложения перестанут работать должным образом. Дополнительные сведения о проблемах совместимости, связанных с изменениями JavaScript, см. в разделе [Совместимость сайтов и Internet Explorer 8](/archive/blogs/ie/site-compatibility-and-ie8).
