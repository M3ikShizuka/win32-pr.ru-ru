# <a name="fiddler-web-debugger-tool"></a>Средство веб-отладчика Fiddler

Средство Fiddler помогает выполнять отладку веб-приложений путем сбора сетевого трафика между Интернетом и тестовыми компьютерами. Средство позволяет проверять входящие и исходящие данные для отслеживания и изменения запросов и ответов до того, как браузер получит их. Fiddler также включает в себя мощную подсистему создания сценариев на основе событий, которую можно расширить с помощью любого платформа .NET Framework языка.

Fiddler и параметры воспроизведения HTTP помогают устранять неполадки на стороне клиента в веб-приложениях путем создания автономной копии тестового сайта. С помощью этих средств можно создавать автономные образы для просмотра, а затем упаковывать и анализировать результаты для получения более подробных сведений об отладке.

Чтобы скачать надстройку Fiddler, перейдите на страницу [надстроек Internet Explorer](https://www.microsoft.com/iegallery) . Дополнительные сведения об устранении неполадок с помощью Fiddler и связанных с ними средств см. в статье [как использовать Fiddler и воспроизведение HTTP, чтобы иметь автономную копию](https://blogs.msdn.com/b/ricardo/archive/2010/06/22/how-to-user-fiddler-and-http-replay-to-have-an-offline-copy-of-your-site.aspx) записи блога сайта и [проверку подлинности для устранения неполадок с](https://blogs.msdn.com/b/ieinternals/archive/2009/11/22/clipboard-paste-bug-in-windows-7-password-prompt.aspx) помощью записи блога Fiddler.
