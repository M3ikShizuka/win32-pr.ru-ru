# <a name="application-compatibility-testing-using-virtual-pc-images"></a>Тестирование совместимости приложений с помощью образов Virtual PC

Чтобы начать тестирование совместимости веб-приложений, необходимо установить тестовую среду. Процесс создания и настройки компьютеров для этой задачи занимает много времени и часто является дорогостоящим.

Чтобы упростить тестирование совместимости, можно использовать бесплатные предварительно созданные образы виртуальных ПК (VPC) для обеспечения совместимости приложений. Образы VPC — это предварительно настроенные образы жестких дисков, которые можно загрузить на существующие компьютеры, на которых работает [программное обеспечение Virtual PC](https://support.microsoft.com/help/14223/windows-xp-end-of-support). эти образы позволяют тщательно протестировать различные версии Microsoft Windows и Windows Internet Explorer с существующими приложениями в течение короткого промежутка времени.

образы VPC доступны для всех текущих поддерживаемых версий Windows и содержат множество ресурсов и приложений для тестирования совместимости. Чтобы найти тестовые образы, см. [образ VPC для совместимости приложений Internet Explorer](https://www.microsoft.com/Downloads/details.aspx?FamilyID=21eabb90-958f-4b64-b5f1-73d0a413c8ef&amp;displaylang=en).
