# <a name="versioning-windows-7-and-windows-server-2008-r2-application-quality-cookbook"></a>управление версиями (Windows 7 и Windows Server 2008 R2 качество приложения, Cookbook)

одной из наиболее распространенных проблем совместимости приложений Windows Internet Explorer являются строки агента пользователя (UA) и векторы версий. Windows В Internet Explorer 8 появилась новая строка UA, которая помогает приложениям определить, какая версия запущена. Помимо простого увеличения значения MSIE, связанного с Internet Explorer в строке UA, Internet Explorer 8 добавляет значение Trident/4.0, помогающее определить, работает ли Internet Explorer 8 в режиме представления совместимости. Эти изменения и проверки жестко закодированных версий могут привести к проблемам совместимости между браузерами.
