# <a name="internet-explorer-8-developer-tools"></a>Средства для разработчиков Internet Explorer 8

Windows Internet Explorer 8 включает функцию Средства для разработчиков. эта функция позволяет быстро выполнять отладку Microsoft JScript, исследовать Windows поведение Internet Explorer или быстро выполнять итерацию по созданию прототипа или тестированию нового решения.

В следующих разделах описываются важные характеристики Средства для разработчиков в Internet Explorer 8.

## <a name="integrated-and-simple-to-use"></a>Интеграция и простота использования

Средства для разработчиков включены в каждую установку Internet Explorer 8, поэтому можно выполнять отладку проблем на любом компьютере, где работает Internet Explorer 8. Кроме того, средства загружаются только при необходимости, чтобы ограничить их влияние на производительность браузера. С помощью Средства для разработчиков можно быстро отлаживать только текущий процесс Internet Explorer, а не отладку для всего обозревателя Internet Explorer.

## <a name="provide-a-visual-interface-to-the-platform"></a>Предоставление визуального интерфейса для платформы

Вместо реконструирования работы веб-сайта или изменения сайта для вывода отладочной информации Средства для разработчиков разрешите обозревателю Internet Explorer просмотреть его представление сайта. Это представление сокращает время, затрачиваемое на отладку динамических сайтов, когда проверка исходного кода не полезна или изучает поведение, характерное для обозревателя Internet Explorer, в котором не может помочь универсальный инструмент разработки.

## <a name="enable-fast-experimentation"></a>Включить быстрое экспериментирование

При создании прототипа нового проекта или исправления в предыдущих версиях Internet Explorer, скорее всего, вы изменяете источник, сохраняете его, обновляете страницу в браузере и повторяйте. Средства для разработчиков упростить этот сценарий, позволяя редактировать сайт в браузере и немедленно просматривать изменения.

## <a name="optimize-application-performance"></a>Оптимизация производительности приложения

Чтобы выявление и устранение проблем с производительностью, вы, скорее всего, перебираетесь по одному сценарию за раз. С помощью профилировщика скриптов в Средства для разработчиков можно выполнять сбор статистики, включая время выполнения и число вызовов функции JavaScript. Эти сведения можно использовать при тестировании приложения и использовании отчета профиля для быстрого выявления и устранения узких мест производительности.

Чтобы получить доступ к Средства для разработчиков, нажмите клавишу F12 во время просмотра веб-страницы или выберите **средства для разработчиков** в меню **Сервис** .
