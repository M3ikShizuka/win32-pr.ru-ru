# <a name="understanding-the-application-compatibility-challenge"></a>Основные сведения о вызове совместимости приложений

Windows Internet Explorer 8 является браузером по умолчанию в операционной системе Windows 7. Internet Explorer 8 улучшает многие функции, в том числе улучшенную архитектуру платформы, предназначенную для упрощения и ускорения работы в Интернете, а также для упрощения настройки и управления для ИТ-специалистов и создания инновационного содержимого для веб-разработчиков.

некоторые из этих функций используют новые механизмы визуализации и создания сценариев, которые являются основными элементами, которые Windows Internet Explorer использует для отображения содержимого HTML и CSS и выполнения скриптов. Изменения архитектуры и усовершенствования платформы представляют проблемы совместимости приложений, которые могут возникнуть при переносе приложений в Internet Explorer 8 из предыдущего браузера.

Internet Explorer 8 включает в себя некоторую обратную совместимость, но некоторые изменения (включая поддержку поддержки веб-стандартов и функций безопасности), которые могут потребовать обновления веб-приложений, созданных для Microsoft Internet Explorer 6 или предыдущих версий.

Дополнительные сведения о тестировании и устранении проблем совместимости для Internet Explorer 8 см. [в статье Устранение совместимости приложений при переходе на Internet Explorer 8 — Информация для специалистов по информационным технологиям](/previous-versions/windows/it-pro/windows-7/ff955640(v=ws.10)).
