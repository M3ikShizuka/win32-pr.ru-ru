# <a name="internet-explorer-8-compatibility-wizard-aggiorno"></a>Мастер совместимости Internet Explorer 8 (Aggiorno)

Мастер совместимости в Windows Internet Explorer 8 автоматически обновляет веб-сайты для корректной визуализации в Internet Explorer 8 без нарушения совместимости в старых браузерах. Мастер автоматизирует задачу нацеливания версий браузера, сохраняя внешний вид и функциональность страниц. Дополнительные сведения об этом мастере см. в разделе [Мастер совместимости Internet Explorer 8 быстрое начало](https://www.mobilize.net/).
