# <a name="windows-server-2008-r2-logo-program-for-software"></a>Windows Программа для получения логотипа Server 2008 R2 для программного обеспечения

## <a name="affected-platforms"></a>Затронутые платформы

 **серверы —** Windows Server 2008 R2  

### <a name="description"></a>Описание

программа Windows Server 2008 R2 Logo была снята с учета.

Windows отправку тестов сервера 2008 R2 2012 больше не принимаются ни для компетенций Microsoft Partner Network (MPN), ни для логотипа сертифицированного для Windows сервера.

Windows Server 2012 в настоящее время R2 является предпочтительной платформой для логотипа сертифицированного для Windows сервера. дополнительные сведения см. в статье о [программе сертификации для эмблемы облачного приложения Enterprise](https://azure.microsoft.com/overview/commercial-marketplace/) .
