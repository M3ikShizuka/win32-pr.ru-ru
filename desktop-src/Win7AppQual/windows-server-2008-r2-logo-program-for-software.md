# <a name="windows-server-2008-r2-logo-program-for-software"></a>Программа Windows Server 2008 R2 Logo Program для программного обеспечения

## <a name="affected-platforms"></a>Затронутые платформы

 **Серверы —** Windows Server 2008 R2  

### <a name="description"></a>Описание

Программа Windows Server 2008 R2 Logo была снята с учета.

Отправку тестов Windows Server 2008 R2 2012 больше не принимаются ни для компетенций Microsoft Partner Network (MPN), ни для логотипа сертифицированного для Windows Server.

В настоящее время Windows Server 2012 R2 является предпочтительной платформой для логотипа сертифицировано для Windows Server. Дополнительные сведения см. в разделе [Программа сертификации корпоративного приложения для облачных приложений](https://azure.microsoft.com/overview/commercial-marketplace/) .
