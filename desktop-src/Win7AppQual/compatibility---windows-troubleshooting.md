# <a name="windows-troubleshooting"></a>Устранение неполадок Windows

## <a name="affected-platforms"></a>Затронутые платформы

**Клиенты** — Windows 7  
**Серверы** — Windows Server 2008 R2  









## <a name="description"></a>Описание

Новый центр поддержки Windows 7 (прежнее название — центр решений), устранение неполадок Windows, предоставляет систематическую процедуру устранения неполадок для пользователя. Центр действий — один из пяти значков, закрепленных в систрай. Устранение неполадок в Windows позволяет просматривать и искать встроенные пакеты устранения неполадок, а также устранять неполадки пакетов, которые хранятся на сервере Microsoft в Интернете — лучше при подключении (БВК). Можно выбрать и запустить пакет устранения неполадок, чтобы попытаться устранить проблему. Если не удается определить разрешение проблемы, вы можете найти справку, содержимое сообщества, статьи о поддержке или другие пакеты устранения неполадок для соответствующего связанного содержимого. Если не предоставить ответ на проблему, можно восстановить компьютер до момента возникновения проблемы или получить помощь через удаленный помощник. Цель — позволить легко и быстро найти решение проблемы.

## <a name="usage"></a>Использование

Центр действий четко видим и доступен из нескольких расположений. Вы можете запустить его из контекстного меню центра действий в систрай на панели управления в качестве ярлыка в разделе "система и обслуживание", на главной странице центра действий и из содержимого справки.

Устранение неполадок пакетов основано на сценариях PowerShell. Процесс создания и публикации пакета устранения неполадок будет сделан общедоступным, чтобы позволить поставщикам вычислительной техники, независимым поставщикам программного обеспечения и ИТ-специалистам разрабатывать и поставлять собственные материалы по устранению неполадок.

Для обеспечения единообразного взаимодействия с пользователем обязательно следуйте рекомендациям и рекомендациям, описанным в разделе Инструментарий по устранению неполадок Windows, при проектировании собственных пакетов устранения неполадок.
