# <a name="rendering-and-layout-changes"></a>Изменения отрисовки и макета

Windows Internet Explorer 8 более строго интерпретирует разметку HTML по сравнению с предыдущими версиями, и это изменение может привести к неправильному отображению веб-страниц. например, если сайт правильно отображает список или таблицу в Windows internet explorer 7, он может отображаться в обозревателе internet explorer 8.

Эти типы проблем можно устранить, уменьшив веб-страницу, чтобы найти неправильно сформированный код HTML. при устранении большинства проблем с отображением эти изменения не создают проблем в предыдущих версиях Windows Internet Explorer, поскольку предыдущие версии просто пропускают излишнее HTML. Дополнительные сведения об этих изменениях см. в записи блога о [совместимости сайтов и IE8](/archive/blogs/ie/site-compatibility-and-ie8) .
