# <a name="file-library-replaces-document-folder"></a>Библиотека файлов заменяет папку документа

## <a name="affected-platforms"></a>Затронутые платформы

**клиенты** — Windows 7  
**серверы** — Windows Server 2008 R2  









## <a name="feature-impact"></a>Воздействие на функции

**Серьезность** — средний  
**Частота** — высокая  











## <a name="description"></a>Описание

Библиотеки обеспечивают централизованную работу с папками для хранения файлов, поиска и доступа в нескольких расположениях, как локальных, так и удаленных.

Расположения по умолчанию, используемые общими диалоговыми файлами (например, Open и Save), были изменены из папки документов в библиотеку документов. Пользовательский интерфейс не изменяется, но теперь пользователь сможет просматривать, просматривать и искать библиотеку с помощью различных представлений упорядочения. Файлы будут сохраняться в папке сохранения по умолчанию, если пользователь не изменит расположение сохранения по умолчанию или не укажет другую папку.

Разработчики могут создавать собственные библиотеки или добавлять расположения в существующие библиотеки с помощью интерфейса Ишелллибрари. Пользователи могут находить библиотеки с помощью известной системы папок (например, FOLDERID \_ документслибрари).

## <a name="manifestation-of-impact"></a>Влияние на манифесты

Библиотека сама по себе является файлом, а не папкой. Таким образом, операции с путями могут привести к ошибкам из-за попытки приложения объединить файлы в файлы.

## <a name="solution"></a>Решение

При использовании Ифиледиалог необходимо использовать метод «методом» вместо комбинации файлов «. Folder» и «ИмяФайла», как в предыдущих версиях операционной системы. Используйте интерфейсы API оболочки, где это возможно, чтобы взаимодействовать с элементами в пространстве имен оболочки и управлять ими (например, интерфейса IShellItem).

## <a name="leveraging-feature-capabilities"></a>Использование возможностей функций

Если вы хотите создать собственные библиотеки или добавить расположения в существующие библиотеки, необходимо использовать API Ишелллибрари. Библиотеки сами являются папками оболочки, поэтому их можно перечислить так же, как и любую другую папку оболочки.

## <a name="compatibility-performance-reliability-and-usability-testing"></a>Совместимость, производительность, надежность и тестирование на удобство использования

При использовании общего диалогового окна «файл» пользователи смогут сохранять данные непосредственно в своих библиотеках.
