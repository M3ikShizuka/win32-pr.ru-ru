# <a name="sync-with-sid-attribute"></a>Sync — атрибут с идентификатором SID

Для синхронизации встроенного объекта Group/локальной политики SAM это локальная группа, которой соответствует объект.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Синхронизация с ИД безопасности                        |
| LDAP-отображаемое имя | синквиссид                          |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.667               |
| System-ID — GUID    | 037651e5-441d-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**Строка (SID)**](s-string-sid.md)  |
