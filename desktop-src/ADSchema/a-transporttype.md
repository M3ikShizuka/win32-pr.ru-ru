# <a name="transport-type-attribute"></a>Атрибут Transport-Type

Различающееся имя для типа транспорта, используемого для объединения сайтов. Это значение может указывать на транспорт IP или SMTP.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Transport-Type                          |
| LDAP-отображаемое имя | transportType                           |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | При подключении сайтов.                  |
| Attribute-Id      | 1.2.840.113556.1.4.791                  |
| System-ID — GUID    | 26d97374-6070-11d1-a9c6-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
