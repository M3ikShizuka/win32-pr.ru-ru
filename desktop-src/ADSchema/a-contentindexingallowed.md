# <a name="content-indexing-allowed-attribute"></a>Атрибут content-Indexing-Allowed

Указывает, может ли объект тома индексировать содержимое.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Индексация содержимого — разрешено             |
| LDAP-отображаемое имя | контентиндексингалловед               |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.24                |
| System-ID — GUID    | bf967943-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
