В смешанном режиме MSMQ время репликации по умолчанию между сайтами.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MSMQ-Interval2                       |
| LDAP-отображаемое имя | mSMQInterval2                        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1309              |
| System-ID — GUID    | 99b88f52-3b7b-11d2-90cc-00c04fd91ab1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
