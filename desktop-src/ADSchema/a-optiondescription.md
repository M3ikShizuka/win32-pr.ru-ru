# <a name="option-description-attribute"></a>Атрибут Option-Description

Этот атрибут содержит описание параметра, установленного на DHCP-сервере.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Option-Description                          |
| LDAP-отображаемое имя | оптиондескриптион                           |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.712                      |
| System-ID — GUID    | 963d274d-48be-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
