# <a name="title-attribute-ad-schema"></a>Атрибут Title (схема AD)

Содержит название должности пользователя. Это свойство обычно используется для указания формального названия задания, например высшего программиста, а не класса для занятий, такого как программист. Обычно он не используется для названий суффиксов, таких как ЕСК. или DDS.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------|
| CN                | Название                                                                     |
| LDAP-отображаемое имя | title                                                                     |
| Размер              | \-                                                                        |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                    |
| Частота обновления  | При создании записи пользователя и при необходимости изменения названия. |
| Attribute-Id      | 2.5.4.12                                                                  |
| System-ID — GUID    | bf967a55-0de6-11d0-a285-00aa003049e2                                      |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                               |
