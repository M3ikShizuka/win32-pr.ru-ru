# <a name="local-policy-reference-attribute"></a>Атрибут локальной политики — ссылка

| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Локальная политика — ссылка                  |
| LDAP-отображаемое имя | локалполициреференце                    |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.457                  |
| System-ID — GUID    | 80a67e4d-9f22-11d0-afdd-00c04fd930c9    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
