# <a name="seq-notification-attribute"></a>Атрибут Seq-Notification

Этот атрибут содержит счетчик, который увеличивается ежедневно. Это значение счетчика передается службе отслеживания ссылок, которая добавляет значение к его томам и связывает исходные файлы при их обновлении. Этот параметр поддерживается контроллером домена.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Seq-Notification                     |
| LDAP-отображаемое имя | секнотификатион                      |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.504               |
| System-ID — GUID    | ddac0cf2-af8f-11d0-afeb-00c04fd930c9 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
