# <a name="options-location-attribute"></a>Атрибут Options-Location

Для DHCP расположение параметров содержит различающееся имя для альтернативных сайтов, содержащих сведения о параметрах.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Options-Location                     |
| LDAP-отображаемое имя | оптионслокатион                      |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.713               |
| System-ID — GUID    | 963d274e-48be-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
