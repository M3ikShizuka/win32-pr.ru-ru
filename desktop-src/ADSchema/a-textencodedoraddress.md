# <a name="text-encoded-or-address-attribute"></a>Атрибут с текстовым кодированием или адресом

Этот атрибут используется для поддержки адресов X. 400 в текстовом формате.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | С текстовым кодированием или адресом                     |
| LDAP-отображаемое имя | текстенкодедораддресс                        |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.2                   |
| System-ID — GUID    | a8df7489-c5ea-11d1-bbcb-0080c76670c0        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
