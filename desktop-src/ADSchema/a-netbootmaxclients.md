# <a name="netboot-max-clients-attribute"></a>атрибут нетбут-Max-Clients

Атрибут **нетбут-Max-Clients** зарезервирован для внутреннего использования.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | нетбут-Max-клиенты                  |
| LDAP-отображаемое имя | нетбутмаксклиентс                    |
| Размер              | \-                                   |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.851               |
| System-ID — GUID    | 07383078-91df-11d1-aebc-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
