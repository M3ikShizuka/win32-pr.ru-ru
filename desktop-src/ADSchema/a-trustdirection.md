# <a name="trust-direction-attribute"></a>Атрибут Trust-Direction

Направление доверия.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Trust-Direction                      |
| LDAP-отображаемое имя | трустдиректион                       |
| Размер              | 4 байта                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | При создании нового доверия.         |
| Attribute-Id      | 1.2.840.113556.1.4.132               |
| System-ID — GUID    | bf967a5c-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
