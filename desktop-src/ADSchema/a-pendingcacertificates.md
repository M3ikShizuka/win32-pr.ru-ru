# <a name="pending-ca-certificates-attribute"></a>Атрибут Pending — CA-Certificates

Сертификаты, которые скоро вступят в силу для этого центра сертификации.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Ожидающие — CA — сертификаты                               |
| LDAP-отображаемое имя | пендингкацертификатес                                 |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.693                                |
| System-ID — GUID    | 963d273c-48be-11d1-a9c3-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
