| Ввод | Значение |
|-------------------|--------------------------------------|

| LDAP-отображаемое имя | msDS-Локкаутобсерватионвиндов        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2017              |
| System-ID — GUID    | b05bda89-76af-468a-b892-1be55558ecc8 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
