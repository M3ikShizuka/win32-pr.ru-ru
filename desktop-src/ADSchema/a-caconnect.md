# <a name="ca-connect-attribute"></a>Атрибут CA-Connect

Строка подключения для привязки к центру сертификации.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | CA-Connect                                  |
| LDAP-отображаемое имя | каконнект                                   |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.687                      |
| System-ID — GUID    | 963d2735-48be-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
