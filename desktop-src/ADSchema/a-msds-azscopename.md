# <a name="ms-ds-az-scope-name-attribute"></a>Атрибут ms-DS-AZ-Scope-Name

Строка, однозначно определяющая объект области.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-AZ-Scope-Name                         |
| LDAP-отображаемое имя | msDS-Азскопенаме                            |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор Азролес                               |
| Частота обновления  | Во время инициализации или изменения политики.     |
| Attribute-Id      | 1.2.840.113556.1.4.1799                     |
| System-ID — GUID    | 515a6b06-2617-4173-8099-d5605df043c6        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
