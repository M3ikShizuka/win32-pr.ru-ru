# <a name="flat-name-attribute"></a>Атрибут Flat-Name

для доменов Windows NT неструктурированным именем является имя NetBIOS. для ссылок с доменами, не Windows NT, неструктурированным именем является идентифицирующее имя этого домена или **значение NULL**.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Flat-Name                                   |
| LDAP-отображаемое имя | флатнаме                                    |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.511                      |
| System-ID — GUID    | b7b13117-b82e-11d0-afee-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
