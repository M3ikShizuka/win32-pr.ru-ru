# <a name="ms-wmi-intdefault-attribute"></a>атрибут MS-WMI-Интдефаулт

Значение по умолчанию для объектов целочисленных параметров WMI 32-bit.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-WMI-Интдефаулт                    |
| LDAP-отображаемое имя | Мсвми — Интдефаулт                     |
| Размер              | 4 байта                              |
| Привилегия обновления  | Администратор групповая политика           |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1628              |
| System-ID — GUID    | 1b0c07f8-76dd-4060-a1e1-70084619dc90 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
