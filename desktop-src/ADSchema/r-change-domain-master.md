# <a name="change-domain-master-extended-right"></a>Изменение-домен-основное расширенное право

Расширенное право, необходимое для изменения владельца роли FSMO именования доменов.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Изменение — домен-хозяин                 |
| Display-Name | Изменение главного домена                 |
| Rights-GUID  | 014bf69c-7b3b-11d1-85f6-08002be74fab |
