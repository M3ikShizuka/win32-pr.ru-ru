Атрибут **Scope-flags** в настоящее время не используется.

|-------------------|--------------------------------------|
| CN                | Scope-Flags                          |
| LDAP-отображаемое имя | скопефлагс                           |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1354              |
