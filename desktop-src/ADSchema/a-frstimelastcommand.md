# <a name="frs-time-last-command-attribute"></a>Атрибут FRS-Time-Last-Command

Время, в которое была выполнена последняя команда.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------|
| CN                | FRS-Time-Last-Command                                         |
| LDAP-отображаемое имя | фрстимеласткомманд                                            |
| Размер              | \-                                                            |
| Привилегия обновления  | \-                                                            |
| Частота обновления  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.880                                        |
| System-ID — GUID    | 2a132583-9373-11d1-aebc-0000f80367c1                          |
| Синтаксис            | [**String(обобщенное_время)**](s-string-generalized-time.md) |
