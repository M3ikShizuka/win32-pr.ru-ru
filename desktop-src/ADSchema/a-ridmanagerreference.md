# <a name="rid-manager-reference-attribute"></a>RID-Manager-ссылка атрибут

Различающееся имя для диспетчера RID объекта.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Справочник по RID-Manager                   |
| LDAP-отображаемое имя | ридманажерреференце                     |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | При каждом изменении диспетчера RID.       |
| Attribute-Id      | 1.2.840.113556.1.4.368                  |
| System-ID — GUID    | 66171886-8f3c-11d0-afda-00c04fd930c9    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
