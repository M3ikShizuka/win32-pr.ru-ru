# <a name="parent-ca-attribute"></a>Атрибут parent — CA

Различающееся имя объекта центра сертификации (ЦС) для родительского ЦС.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Родительский ЦС                               |
| LDAP-отображаемое имя | парентка                                |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.557                  |
| System-ID — GUID    | 5245801b-ca6a-11d0-afff-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
