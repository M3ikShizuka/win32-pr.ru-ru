# <a name="show-in-advanced-view-only-attribute"></a>Атрибут "Показать в режиме" — "только просмотр"

**Значение true** , если этот атрибут должен быть видимым в расширенном режиме пользовательского интерфейса.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------------------------------|
| CN                | "Показать" — "Расширенный просмотр"                                                              |
| LDAP-отображаемое имя | showInAdvancedViewOnly                                                                  |
| Размер              | 4 байта                                                                                 |
| Привилегия обновления  | Администратор домена                                                                    |
| Частота обновления  | При создании объекта или при необходимости изменения аспектов просмотра атрибута. |
| Attribute-Id      | 1.2.840.113556.1.2.169                                                                  |
| System-ID — GUID    | bf967984-0de6-11d0-a285-00aa003049e2                                                    |
| Синтаксис            | [**Логическая**](s-boolean.md)                                                            |
