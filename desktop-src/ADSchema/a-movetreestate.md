|-------------------|-------------------------------------------------------|
| CN                | Перемещение-дерево-состояние                                       |
| LDAP-отображаемое имя | моветристате                                         |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1305                               |
| System-ID — GUID    | 1f2ac2c8-3b71-11d2-90cc-00c04fd91ab1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
