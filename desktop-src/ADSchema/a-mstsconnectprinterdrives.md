# <a name="ms-ts-connect-printer-drives-attribute"></a>атрибут MS-TS-Connect-Printer-Drives

Диски подключения сеанса служб терминалов указывает, следует ли повторно подключаться к подключенным принтерам клиента при входе в систему.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-TS-Connect-Printer-Drives         |
| LDAP-отображаемое имя | мстсконнектпринтердривес             |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1987              |
| System-ID — GUID    | 8ce6a937-871b-4c92-b285-d99d4036681c |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
