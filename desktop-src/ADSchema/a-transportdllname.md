# <a name="transport-dll-name-attribute"></a>Атрибут имени транспорта-DLL

Имя библиотеки DLL, которая будет управлять транспортом.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Transport-DLL-Name                          |
| LDAP-отображаемое имя | транспортдллнаме                            |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | При подключении сайтов.                      |
| Attribute-Id      | 1.2.840.113556.1.4.789                      |
| System-ID — GUID    | 26d97372-6070-11d1-a9c6-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
