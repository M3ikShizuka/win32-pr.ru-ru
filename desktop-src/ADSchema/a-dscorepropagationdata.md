# <a name="ds-core-propagation-data-attribute"></a>DS-Core-Propagate-атрибут данных

Атрибут **DS-Core-Propagate-Data** предназначен только для внутреннего использования.

|-------------------|---------------------------------------------------------------|
| CN                | DS-Core-распространение-данные                                      |
| LDAP-отображаемое имя | дскорепропагатиондата                                         |
| Размер              | \-                                                            |
| Привилегия обновления  | \-                                                            |
| Частота обновления  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.1357                                       |
| System-ID — GUID    | d167aa4b-8b08-11d2-9939-0000f87a57d4                          |
