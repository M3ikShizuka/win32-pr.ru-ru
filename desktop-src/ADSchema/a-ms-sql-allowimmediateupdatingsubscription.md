# <a name="ms-sql-allowimmediateupdatingsubscription-attribute"></a>Атрибут MS-SQL-Алловиммедиатеупдатингсубскриптион

Значение true, если публикация разрешает синхронизированные транзакции обновляемые подписки.



| Ввод | Значение |
|-------------------|-------------------------------------------|
| CN                | MS-SQL-Алловиммедиатеупдатингсубскриптион |
| LDAP-отображаемое имя | mS-SQL-Алловиммедиатеупдатингсубскриптион |
| Размер              | 4 байта                                   |
| Привилегия обновления  | Это значение задается системой.          |
| Частота обновления  | При установке репликации.                |
| Attribute-Id      | 1.2.840.113556.1.4.1404                   |
| System-ID — GUID    | c4186b6e-d34b-11d2-999a-0000f87a57d4      |
| Синтаксис            | [**Логическая**](s-boolean.md)              |
