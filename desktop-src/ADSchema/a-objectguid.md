# <a name="object-guid-attribute"></a>Атрибут Object-Guid

Уникальный идентификатор объекта.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------|
| CN                | Object-Guid                                                         |
| LDAP-отображаемое имя | objectGUID                                                          |
| Размер              | 16 байт                                                            |
| Привилегия обновления  | Это значение задается системой.                                    |
| Частота обновления  | Это значение задается при создании объекта и не может быть изменено. |
| Attribute-Id      | 1.2.840.113556.1.4.2                                                |
| System-ID — GUID    | bf9679e7-0de6-11d0-a285-00aa003049e2                                |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md)               |
