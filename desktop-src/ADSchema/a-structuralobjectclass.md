# <a name="structural-object-class-attribute"></a>Атрибут "структурный объект — класс"

Этот сконструированный атрибут хранит список классов, содержащихся в иерархии классов, включая абстрактные классы. Этот список содержит динамически связанные вспомогательные классы.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------|
| CN                | Структурный объектный класс                                         |
| LDAP-отображаемое имя | структуралобжекткласс                                           |
| Размер              | \-                                                              |
| Привилегия обновления  | Это значение задается системой.                                |
| Частота обновления  | При создании класса.                                              |
| Attribute-Id      | 2.5.21.9                                                        |
| System-ID — GUID    | 3860949f-f6a8-4b38-9950-81ecb6bc2982                            |
| Синтаксис            | [**String(идентификатор_объекта)**](s-string-object-identifier.md) |
