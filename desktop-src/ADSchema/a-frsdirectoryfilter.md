# <a name="frs-directory-filter-attribute"></a>Атрибут фильтра FRS-Directory

Список каталогов, исключаемых из репликации файлов, например TEMP или obj.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | FRS-каталог — фильтр                        |
| LDAP-отображаемое имя | фрсдиректорифилтер                          |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.484                      |
| System-ID — GUID    | 1be8f171-a9ff-11d0-afe2-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
