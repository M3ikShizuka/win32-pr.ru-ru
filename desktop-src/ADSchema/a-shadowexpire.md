# <a name="shadowexpire-attribute"></a>атрибут Шадовекспире

Содержит абсолютную дату окончания срока действия учетной записи.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | шадовекспире                         |
| LDAP-отображаемое имя | шадовекспире                         |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.10                     |
| System-ID — GUID    | 75159a00-1fff-4cf4-8bff-4ef2695cf643 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
