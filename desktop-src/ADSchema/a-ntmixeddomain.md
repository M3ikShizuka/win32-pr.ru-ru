# <a name="nt-mixed-domain-attribute"></a>NT-смешанный атрибут домена

Указывает, что домен работает в собственном режиме или в смешанном режиме. Этот атрибут находится в объекте Домаинднс (Head) для домена.



| Ввод | Значение |
|-------------------|---------------------------------------|
| CN                | NT — смешанный домен                       |
| LDAP-отображаемое имя | нтмикседдомаин                         |
| Размер              | 4 байта. Смешанный режим 1, собственный режим 0. |
| Привилегия обновления  | \-                                    |
| Частота обновления  | \-                                    |
| Attribute-Id      | 1.2.840.113556.1.4.357                |
| System-ID — GUID    | 3e97891f-8c01-11d0-afda-00c04fd930c9  |
| Синтаксис            | [**Перечисление**](s-enumeration.md)  |
