# <a name="ms-com-defaultpartitionlink-attribute"></a>атрибут MS-COM-Дефаултпартитионлинк

| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-COM-Дефаултпартитионлинк             |
| LDAP-отображаемое имя | Мском — Дефаултпартитионлинк              |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1427                 |
| System-ID — GUID    | 998b10f7-aa1a-4364-b867-753d197fe670    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
