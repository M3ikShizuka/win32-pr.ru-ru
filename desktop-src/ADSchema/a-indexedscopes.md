# <a name="indexedscopes-attribute"></a>Атрибут Индекседскопес

Список областей каталога (например, C: \\ или D: \) , которые индексируются.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | индекседскопес                               |
| LDAP-отображаемое имя | индекседскопес                               |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.681                      |
| System-ID — GUID    | 7bfdcb87-4807-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
