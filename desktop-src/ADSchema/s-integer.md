# <a name="integer-syntax"></a>Целочисленный синтаксис

Представляет значение 32-битового целого числа со знаком.



| Ввод | Значение |
|--------------|---------------------------------------------------------------------------|
| Имя         | Целочисленный тип                                                                   |
| Идентификатор синтаксиса    | 2.5.5.9                                                                   |
| ИДЕНТИФИКАТОР OM        | 2                                                                         |
| Тип MAPI    | LONG                                                                      |
| Тип ADS     | АДСТИПЕ \_ целое число                                                          |
| Тип варианта | VT \_ I4                                                                    |
| Тип SDS     | [System.Int32](/dotnet/api/system.int32) |
