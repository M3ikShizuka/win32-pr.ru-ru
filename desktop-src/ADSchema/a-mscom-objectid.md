# <a name="ms-com-objectid-attribute"></a>атрибут MS-COM-ObjectId

Идентификатор GUID, используемый для обнаружения раздела COM+ или набор разделов COM+.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MS-COM-ObjectId                                       |
| LDAP-отображаемое имя | Мском-ObjectId                                        |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1428                               |
| System-ID — GUID    | 430f678b-889f-41f2-9843-203b5a65572f                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
