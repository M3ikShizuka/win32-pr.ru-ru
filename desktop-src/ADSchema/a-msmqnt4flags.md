# <a name="msmq-nt4-flags-attribute"></a>Атрибут "MSMQ-Nt4-Flags"

Атрибут **MSMQ-Nt4-flags** содержит сведения о смешанном режиме MSMQ.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MSMQ-Nt4-флаги                       |
| LDAP-отображаемое имя | mSMQNt4Flags                         |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.964               |
| System-ID — GUID    | eb38a158-d57f-11d1-90a2-00c04fd91ab1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
