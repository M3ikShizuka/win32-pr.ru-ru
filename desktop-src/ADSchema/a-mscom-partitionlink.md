# <a name="ms-com-partitionlink-attribute"></a>атрибут MS-COM-Партитионлинк

Ссылка, используемая для связи с набором разделов COM+ со связанными разделами COM+.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-COM-Партитионлинк                    |
| LDAP-отображаемое имя | Мском — Партитионлинк                     |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1423                 |
| System-ID — GUID    | 09abac62-043f-4702-ac2b-6ca15eee5754    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
