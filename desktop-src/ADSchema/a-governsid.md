# <a name="governs-id-attribute"></a>Атрибут ID атрибута

Уникальный идентификатор объекта класса, определенного данным объектом Class-Schema.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------|
| CN                | Идентификатор, управляющий                                                      |
| LDAP-отображаемое имя | governsID                                                       |
| Размер              | \-                                                              |
| Привилегия обновления  | \-                                                              |
| Частота обновления  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.22                                           |
| System-ID — GUID    | bf96797d-0de6-11d0-a285-00aa003049e2                            |
| Синтаксис            | [**String(идентификатор_объекта)**](s-string-object-identifier.md) |
