# <a name="ms-ds-schema-extensions-attribute"></a>Атрибут ms-DS-Schema-Extensions

Двоичный BLOB-объект, используемый для хранения сведений о расширениях объектов схемы.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Расширения MS-DS-Schema-Extensions                               |
| LDAP-отображаемое имя | msDs-расширения схемы                                |
| Размер              | \-                                                    |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | При расширении схемы.                          |
| Attribute-Id      | 1.2.840.113556.1.4.1440                               |
| System-ID — GUID    | b39a61be-ed07-4cab-9a4a-4963ed0141e1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
