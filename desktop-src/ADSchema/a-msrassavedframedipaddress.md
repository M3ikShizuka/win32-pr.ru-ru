# <a name="msrassavedframedipaddress-attribute"></a>атрибут Мсрассаведфрамедипаддресс

Атрибут **мсрассаведфрамедипаддресс** используется внутренним образом. Не изменяйте это значение напрямую.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | мсрассаведфрамедипаддресс            |
| LDAP-отображаемое имя | мсрассаведфрамедипаддресс            |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1190              |
| System-ID — GUID    | db0c90c6-c1f2-11d1-bbc5-0080c76670c0 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
