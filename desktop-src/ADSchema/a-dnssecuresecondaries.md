# <a name="dns-secure-secondaries-attribute"></a>Атрибут DNS-Secure-вторичный

Атрибут **DNS-Secure-вторичных** в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | DNS-Secure-вторичные службы               |
| LDAP-отображаемое имя | днссекуресекондариес                 |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.380               |
| System-ID — GUID    | e0fa1e67-9b45-11d0-afdd-00c04fd930c9 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
