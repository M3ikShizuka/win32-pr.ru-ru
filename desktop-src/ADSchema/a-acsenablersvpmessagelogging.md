# <a name="acs-enable-rsvp-message-logging-attribute"></a>ACS-Enable-RSVP-сообщение-атрибут ведения журнала

Значение true, если включено ведение журнала RSVP.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-Enable-RSVP-сообщение — ведение журнала      |
| LDAP-отображаемое имя | аксенаблерсвпмессажелоггинг          |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.768               |
| System-ID — GUID    | 7f561285-5301-11d1-a9c5-0000f80367c1 |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
