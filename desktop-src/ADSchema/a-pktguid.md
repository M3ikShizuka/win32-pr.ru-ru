# <a name="pkt-guid-attribute"></a>Атрибут PKT-Guid

Уникальный идентификатор данной распределенная файловая система таблицы знаний секционирования.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | PKT-Guid                                              |
| LDAP-отображаемое имя | пктгуид                                               |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.205                                |
| System-ID — GUID    | 8447f9f0-1027-11d0-a05f-00aa006c33ed                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
