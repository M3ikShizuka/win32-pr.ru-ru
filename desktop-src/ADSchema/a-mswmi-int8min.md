# <a name="ms-wmi-int8min-attribute"></a>атрибут MS-WMI-int8Min

Минимальное значение для объекта целочисленного параметра WMI 64-bit.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-WMI-int8Min                       |
| LDAP-отображаемое имя | Мсвми — Int8Min                        |
| Размер              | 8 байт                              |
| Привилегия обновления  | Администратор групповая политика           |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1634              |
| System-ID — GUID    | ed1489d1-54cc-4066-b368-a00daa2664f1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
