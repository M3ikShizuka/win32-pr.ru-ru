|-------------------|--------------------------------------|
| CN                | Print-min-X-экстент                   |
| LDAP-отображаемое имя | принтминксекстент                      |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.279               |
| System-ID — GUID    | ba305f71-47e3-11d0-a1a6-00c04fd930c9 |
