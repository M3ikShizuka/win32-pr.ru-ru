# <a name="ms-ds-behavior-version-attribute"></a>атрибут версии MS-DS-Behavior

Этот атрибут используется для отслеживания версии поведения домена или леса. Это монотонно увеличивающийся номер, который используется для реализации определенных Active Directoryных функций.



| Ввод | Значение |
|-------------------|------------------------------------------------|
| CN                | MS-DS-поведение — версия                         |
| LDAP-отображаемое имя | msDS — поведение — версия                          |
| Размер              | \-                                             |
| Привилегия обновления  | Это значение задается администратором домена. |
| Частота обновления  | \-                                             |
| Attribute-Id      | 1.2.840.113556.1.4.1459                        |
| System-ID — GUID    | d31a8757-2447-4545-8081-3bb610cacbf2           |
| Синтаксис            | [**Перечисление**](s-enumeration.md)           |
