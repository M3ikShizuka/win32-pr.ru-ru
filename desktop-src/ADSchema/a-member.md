# <a name="member-attribute"></a>Атрибут элемента

Список пользователей, принадлежащих группе.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Член                                                |
| LDAP-отображаемое имя | член                                                |
| Размер              | \-                                                    |
| Привилегия обновления  | Администратор домена                                  |
| Частота обновления  | Каждый раз, когда пользователь добавляется в группу или удаляется из нее. |
| Attribute-Id      | 2.5.4.31                                              |
| System-ID — GUID    | bf9679c0-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md)               |
