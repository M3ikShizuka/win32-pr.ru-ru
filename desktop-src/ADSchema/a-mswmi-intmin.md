# <a name="ms-wmi-intmin-attribute"></a>атрибут MS-WMI-Интмин

Минимальное значение для объекта целочисленного параметра WMI 32-bit.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-WMI-Интмин                        |
| LDAP-отображаемое имя | Мсвми — Интмин                         |
| Размер              | 4 байта                              |
| Привилегия обновления  | Администратор групповая политика           |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1630              |
| System-ID — GUID    | 68c2e3ba-9837-4c70-98e0-f0c33695d023 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
