# <a name="ms-dfsr-contentsetguid-attribute"></a>атрибут MS-DFSR-Контентсетгуид

Содержит идентификатор GUID набора содержимого репликации распределенная файловая система (DFS).



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MS-DFSR-Контентсетгуид                                |
| LDAP-отображаемое имя | Мсдфср — Контентсетгуид                                 |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.18                            |
| System-ID — GUID    | 1035a8e1-67a8-4c21-b7bb-031cdf99d7a0                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
