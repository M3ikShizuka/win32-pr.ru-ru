# <a name="default-hiding-value-attribute"></a>Атрибут по умолчанию-скрытие значения

Логическое значение, указывающее параметр по умолчанию для свойства Шовинадванцедвиевонли новых экземпляров этого класса.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | По умолчанию-скрытие значения                 |
| LDAP-отображаемое имя | дефаулсидингвалуе                   |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.518               |
| System-ID — GUID    | b7b13116-b82e-11d0-afee-0000f80367c1 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
