# <a name="ms-wmi-targetclass-attribute"></a>атрибут MS-WMI-TargetClass

Имя класса создаваемого объекта политики.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-TargetClass                          |
| LDAP-отображаемое имя | Мсвми — TargetClass                           |
| Размер              | Менее двадцати символов.                |
| Привилегия обновления  | Администратор групповая политика                  |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1645                     |
| System-ID — GUID    | 95b6d8d6-c9e8-4661-a2bc-6a5cabc04c62        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
