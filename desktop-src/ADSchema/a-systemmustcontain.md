# <a name="system-must-contain-attribute"></a>System-должен-содержать атрибут

Список обязательных атрибутов для класса. Эти атрибуты должны быть указаны при создании экземпляра класса. Список атрибутов может быть изменен только системой.



| Ввод | Значение |
|-------------------|-------------------------------------------------------------------------------|
| CN                | System — должно содержать                                                           |
| LDAP-отображаемое имя | системмустконтаин                                                             |
| Размер              | \-                                                                            |
| Привилегия обновления  | Администратор схемы                                                          |
| Частота обновления  | При создании класса или добавлении нового обязательного атрибута в класс. |
| Attribute-Id      | 1.2.840.113556.1.4.197                                                        |
| System-ID — GUID    | bf967a45-0de6-11d0-a285-00aa003049e2                                          |
| Синтаксис            | [**String(идентификатор_объекта)**](s-string-object-identifier.md)               |
