# <a name="ms-ds-dnsrootalias-attribute"></a>Атрибут ms-DS-Днсруталиас

Используется для хранения псевдонима домена.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-Днсруталиас                          |
| LDAP-отображаемое имя | msDS-Днсруталиас                           |
| Размер              | Максимальный размер 255 байт.                     |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | Только во время реструктуризации домена.             |
| Attribute-Id      | 1.2.840.113556.1.4.1719                     |
| System-ID — GUID    | 2143acca-eead-4d29-b591-85fa49ce9173        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
