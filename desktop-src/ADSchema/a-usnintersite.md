# <a name="usn-intersite-attribute"></a>Атрибут USN-Intersite

Порядковый номер обновления (USN) для межсайтовой репликации.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | USN-Intersite                        |
| LDAP-отображаемое имя | уснинтерсите                         |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.469               |
| System-ID — GUID    | a8df7498-c5ea-11d1-bbcb-0080c76670c0 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
