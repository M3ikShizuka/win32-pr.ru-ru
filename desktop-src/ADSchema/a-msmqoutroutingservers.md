# <a name="msmq-out-routing-servers-attribute"></a>Атрибут MSMQ-out-Routing-Servers

DN-ссылки на серверы маршрутизации MSMQ, через которые должен маршрутизироваться весь исходящий трафик этого компьютера.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MSMQ-out-Routing-Servers                |
| LDAP-отображаемое имя | мсмкаутраутингсерверс                   |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.928                  |
| System-ID — GUID    | 9a0dc32b-c100-11d1-bbc5-0080c76670c0    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
