# <a name="nc-name-attribute"></a>Атрибут NC-Name

Различающееся имя контекста именования для объекта.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | NC-Name                                 |
| LDAP-отображаемое имя | Параметрами                                  |
| Размер              | \-                                      |
| Привилегия обновления  | Администратор домена                    |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.16                   |
| System-ID — GUID    | bf9679d6-0de6-11d0-a285-00aa003049e2    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
