# <a name="ms-ds-reveal-ondemand-group-attribute"></a>Атрибут ms-DS-раскрыть-OnDemand-Group

Используется с контроллером RODC для определения пользователей, компьютеров и групп, которым разрешено кэширование паролей на контроллере домена только для чтения.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-раскрыть-OnDemand-Group             |
| LDAP-отображаемое имя | msDS-Ревеалондемандграуп                |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1928                 |
| System-ID — GUID    | 303d9f4a-1dd6-4b38-8fc5-33afe8c988ad    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
