# <a name="name-service-flags-attribute"></a>Атрибут Name-Service-flags

Флаги конфигурации для службы имен RPC.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Name-Service-flags                   |
| LDAP-отображаемое имя | намесервицефлагс                     |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.753               |
| System-ID — GUID    | 80212840-4bdc-11d1-a9c4-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
