# <a name="ms-radius-savedframedinterfaceid-attribute"></a>атрибут MS-RADIUS-Саведфрамединтерфацеид

Указывает идентификатор интерфейса IPv6, который должен быть настроен для пользователя.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-RADIUS-Саведфрамединтерфацеид     |
| LDAP-отображаемое имя | Мсрадиус — Саведфрамединтерфацеид      |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1914              |
| System-ID — GUID    | a4da7289-92a3-42e5-b6b6-dad16d280ac9 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
