# <a name="dns-notify-secondaries-attribute"></a>Атрибут DNS-Notify-получателей

Атрибут **DNS-Notify-получателей** в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | DNS-Notify-получатели               |
| LDAP-отображаемое имя | днснотифисекондариес                 |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.381               |
| System-ID — GUID    | e0fa1e68-9b45-11d0-afdd-00c04fd930c9 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
