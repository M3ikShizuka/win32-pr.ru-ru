# <a name="msmq-site-id-attribute"></a>Атрибут MSMQ-site-ID

Атрибут **MSMQ-site-ID** содержит сведения о смешанном режиме MSMQ.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-site-ID                                          |
| LDAP-отображаемое имя | мсмкситеид                                            |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.953                                |
| System-ID — GUID    | 9a0dc340-c100-11d1-bbc5-0080c76670c0                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
