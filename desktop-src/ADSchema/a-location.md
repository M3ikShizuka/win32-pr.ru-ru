| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.222                      |
| System-ID — GUID    | 09dcb79f-165f-11d0-a064-00aa006c33ed        |
