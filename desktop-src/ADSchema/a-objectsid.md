# <a name="object-sid-attribute"></a>Атрибут Object-Sid

Двоичное значение, указывающее идентификатор безопасности (SID) пользователя. Идентификатор безопасности — это уникальное значение, используемое для распознавания пользователя в качестве субъекта безопасности.



| Ввод | Значение |
|-------------------|--------------------------------------------------------------|
| CN                | Object-Sid                                                   |
| LDAP-отображаемое имя | objectSid                                                    |
| Размер              | \-                                                           |
| Привилегия обновления  | Это значение задается системой.                             |
| Частота обновления  | Это значение задается системой при создании учетной записи. |
| Attribute-Id      | 1.2.840.113556.1.4.146                                       |
| System-ID — GUID    | bf9679e8-0de6-11d0-a285-00aa003049e2                         |
| Синтаксис            | [**Строка (SID)**](s-string-sid.md)                          |
