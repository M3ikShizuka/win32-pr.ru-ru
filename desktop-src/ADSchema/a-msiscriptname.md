# <a name="msi-script-name-attribute"></a>Атрибут MSI-Script-Name

Имя файла скрипта установщика Microsoft Installer для этого приложения.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Имя MSI-Script-Name                             |
| LDAP-отображаемое имя | мсискриптнаме                               |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.845                      |
| System-ID — GUID    | 96a7dd62-9118-11d1-aebc-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
