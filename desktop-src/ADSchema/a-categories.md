# <a name="categories-attribute"></a>Атрибут Categories

Список идентификаторов категорий (GUID) для категорий, применяемых к этому приложению.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Категории                                  |
| LDAP-отображаемое имя | Категории                                  |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.672                      |
| System-ID — GUID    | 7bfdcb7e-4807-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
