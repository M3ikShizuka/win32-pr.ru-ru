# <a name="enumerationdelivery-mechanism-syntax"></a>Синтаксис перечисления (механизм доставки)

Синтаксис перечисления (механизм доставки).



| Ввод | Значение |
|--------------|---------------------------------------------------------------------------|
| Имя         | Перечисление (механизм доставки)                                           |
| Идентификатор синтаксиса    | 2.5.5.9                                                                   |
| ИДЕНТИФИКАТОР OM        | 10                                                                        |
| Тип MAPI    | LONG                                                                      |
| Тип ADS     | АДСТИПЕ \_ целое число                                                          |
| Тип варианта | VT \_ I4                                                                    |
| Тип SDS     | [System.Int32](/dotnet/api/system.int32) |



## <a name="remarks"></a>Комментарии

Определяется ITU. Active Directory рассматривает это как целое число.
