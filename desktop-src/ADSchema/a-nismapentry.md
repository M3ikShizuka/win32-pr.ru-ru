# <a name="nismapentry-attribute"></a>атрибут Нисмапентри

Содержит одну запись Map для нестандартной схемы.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | нисмапентри                          |
| LDAP-отображаемое имя | нисмапентри                          |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.27                     |
| System-ID — GUID    | 4a95216e-fcc0-402e-b57f-5971626148a9 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
