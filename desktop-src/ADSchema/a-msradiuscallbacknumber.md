# <a name="msradiuscallbacknumber-attribute"></a>атрибут Мсрадиускаллбаккнумбер

Атрибут **мсрадиускаллбаккнумбер** используется внутренним образом. Не изменяйте это значение напрямую.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | мсрадиускаллбаккнумбер               |
| LDAP-отображаемое имя | мсрадиускаллбаккнумбер               |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1145              |
| System-ID — GUID    | db0c909c-c1f2-11d1-bbc5-0080c76670c0 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
