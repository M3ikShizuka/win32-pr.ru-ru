# <a name="ms-ds-az-application-version-attribute"></a>Атрибут ms-DS-AZ-Application-Version

Номер версии, указывающий, что Азаппликатион обновлен.



| Ввод | Значение |
|-------------------|------------------------------------------------------|
| CN                | MS-DS-AZ-версия приложения                         |
| LDAP-отображаемое имя | msDS-Азаппликатионверсион                            |
| Размер              | \-                                                   |
| Привилегия обновления  | Администратор Азролес                                        |
| Частота обновления  | При изменении номера версии приложения. |
| Attribute-Id      | 1.2.840.113556.1.4.1817                              |
| System-ID — GUID    | 7184a120-3ac4-47ae-848f-fe0ab20784d4                 |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)          |
