# <a name="labeleduri-attribute"></a>атрибут Лабеледури

Универсальный идентификатор ресурса, за которым следует метка. Метка используется для описания ресурса, на который указывает URI, и служит в качестве отображаемого имени, читаемого человеком.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | лабеледури                                  |
| LDAP-отображаемое имя | лабеледури                                  |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.3.6.1.4.1.250.1.57                        |
| System-ID — GUID    | c569bb46-c680-44bc-a273-e6c227d71b45        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
