# <a name="ms-ds-other-settings-attribute"></a>атрибут ms-DS-Other-Параметры

Этот атрибут с несколькими значениями используется для хранения любых настраиваемых параметров для DS, хранимых в формате NAME = VALUE.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | ms-DS-Other-Параметры                        |
| LDAP-отображаемое имя | msDS-Other-Параметры                         |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1621                     |
| System-ID — GUID    | 79d2f34c-9d7d-42bb-838f-866b3e4400e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
