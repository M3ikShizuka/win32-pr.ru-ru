# <a name="msmq-dependent-client-service-attribute"></a>Атрибут "MSMQ-Dependent-Client-Service"

Указывает, предоставляет ли этот сервер службы зависимых клиентов.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MSMQ-Dependent-Client-Service        |
| LDAP-отображаемое имя | мсмкдепендентклиентсервице           |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1239              |
| System-ID — GUID    | 2df90d83-009f-11d2-aa4c-00c04fd7d83a |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
