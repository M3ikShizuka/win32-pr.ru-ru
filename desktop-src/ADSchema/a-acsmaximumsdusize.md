# <a name="acs-maximum-sdu-size-attribute"></a>ACS-Maximum-СДУ-размер атрибута

Атрибут **ACS-Maximum-СДУ-size** предназначен только для внутреннего использования. На основе RFC2210.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-Maximum-СДУ — размер                 |
| LDAP-отображаемое имя | аксмаксимумсдусизе                    |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1314              |
| System-ID — GUID    | 87a2d8f9-3b90-11d2-90cc-00c04fd91ab1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
