| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-Members-for-AZ-Role-BL            |
| LDAP-отображаемое имя | msDS-Мемберсфоразролебл                 |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | Это значение задается системой.        |
| Attribute-Id      | 1.2.840.113556.1.4.1807                 |
| System-ID — GUID    | ececcd20-a7e0-4688-9ccf-02ece5e287f5    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
