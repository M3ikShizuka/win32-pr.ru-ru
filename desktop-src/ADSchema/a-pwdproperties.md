# <a name="pwd-properties-attribute"></a>Атрибут Pwd-Properties

Свойства пароля. Часть политики домена. Битовое значение для обозначения сложности и ограничений хранилища.



| Ввод | Значение |
|-------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | Pwd-Properties                                                                                                                                                                                                           |
| LDAP-отображаемое имя | пвдпропертиес                                                                                                                                                                                                            |
| Размер              | Целое число. Пароль в ДОМЕНЕ \_ \_ сложный 1, \_ пароль домена \_ нет \_ ПРОВЕР \_ изменение 2, \_ пароль домена \_ без \_ четкого \_ изменения 4, \_ Администраторы блокировки домена \_ 8, \_ хранилище паролей домена, Открытый текст \_ \_ 16, отказ домена, \_ \_ изменение пароля \_ 32 |
| Привилегия обновления  | Администратор домена                                                                                                                                                                                                     |
| Частота обновления  | При изменении политики для пользователя.                                                                                                                                                                                      |
| Attribute-Id      | 1.2.840.113556.1.4.93                                                                                                                                                                                                    |
| System-ID — GUID    | bf967a0b-0de6-11d0-a285-00aa003049e2                                                                                                                                                                                     |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                                                                                                                                                                                     |
