# <a name="msmq-peek-dead-letter-extended-right"></a>очередь MSMQ-Просмотр-недоставленное письмо расширенное право

Позволяет просматривать сообщения в очереди недоставленных сообщений.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | MSMQ-Просмотр недоставленных писем                |
| Display-Name | Показать недоставленное письмо                     |
| Rights-GUID  | 4b6e08c1-df3c-11d1-9c86-006008764d0e |
