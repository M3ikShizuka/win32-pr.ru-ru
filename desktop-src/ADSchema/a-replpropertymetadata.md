# <a name="repl-property-meta-data-attribute"></a>REPL-свойство-мета-атрибут данных

Отслеживает сведения о состоянии внутренней репликации для объектов DS. Сведения, которые можно извлечь в общедоступной форме через общедоступный API Дсрепликажетинфо (). Имеется во всех объектах DS.



| Ввод | Значение |
|-------------------|------------------------------------------------------------------------------|
| CN                | REPL-Property-Meta-Data                                                      |
| LDAP-отображаемое имя | реплпропертиметадата                                                         |
| Размер              | Длина пропорциональна количеству реплицируемых атрибутов объекта. |
| Привилегия обновления  | Это значение задается системой.                                             |
| Частота обновления  | Изменения в ответ на любые реплицируемые изменения в объекте.                |
| Attribute-Id      | 1.2.840.113556.1.4.3                                                         |
| System-ID — GUID    | 281416c0-1968-11d0-a28f-00aa003049e2                                         |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md)                        |
