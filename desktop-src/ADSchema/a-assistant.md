| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Помощник                               |
| LDAP-отображаемое имя | assistant                               |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.652                  |
| System-ID — GUID    | 0296c11c-40da-11d1-a9c0-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
