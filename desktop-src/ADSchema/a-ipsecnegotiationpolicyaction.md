# <a name="ipsec-negotiation-policy-action-attribute"></a>Атрибут "IPSEC-согласование-политика-действие"

Атрибут **IPSec-согласовать-Policy-Action** предназначен только для внутреннего использования.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | IPSEC-согласование — политика — действие             |
| LDAP-отображаемое имя | ипсекнеготиатионполициактион                |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.888                      |
| System-ID — GUID    | 07383075-91df-11d1-aebc-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
