# <a name="frs-root-path-attribute"></a>FRS-root-путь атрибут

Путь к корню реплицированной файловой системы дерева.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | FRS-root-путь                               |
| LDAP-отображаемое имя | фрсрутпас                                 |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.487                      |
| System-ID — GUID    | 1be8f174-a9ff-11d0-afe2-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
