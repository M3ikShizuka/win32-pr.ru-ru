# <a name="ms-radius-framedipv6prefix-attribute"></a>атрибут MS-RADIUS-FramedIpv6Prefix

Указывает префикс IPv6 (и соответствующий маршрут), который должен быть настроен для пользователя.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-RADIUS-FramedIpv6Prefix           |
| LDAP-отображаемое имя | Мсрадиус — FramedIpv6Prefix            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1915              |
| System-ID — GUID    | f63ed610-d67c-494d-87be-cd1e24359a38 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
