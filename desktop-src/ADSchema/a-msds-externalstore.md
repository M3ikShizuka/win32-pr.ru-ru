# <a name="ms-ds-external-store-attribute"></a>Атрибут ms-DS-External-Store

Строка, определяющая расположение внешнего хранилища, например базы данных.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-External-Store                        |
| LDAP-отображаемое имя | msDS-Екстерналсторе                          |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1834                     |
| System-ID — GUID    | 604877cd-9cdb-47c7-b03d-3daadb044910        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
