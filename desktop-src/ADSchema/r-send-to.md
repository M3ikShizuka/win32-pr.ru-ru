# <a name="send-to-extended-right"></a>Send-To расширенное право

|--------------|--------------------------------------|
| CN           | Send-To                              |
| Display-Name | Отправить                              |
| Rights-GUID  | ab721a55-1e2f-11d0-9819-00aa0040529b |
