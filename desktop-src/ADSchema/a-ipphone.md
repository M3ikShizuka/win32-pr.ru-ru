# <a name="phone-ip-primary-attribute"></a>атрибут Телефон-Ip-Primary

Адрес TCP/IP для телефона. Используется телефонией.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Телефон-Ip-Primary                                                                 |
| LDAP-отображаемое имя | ipPhone                                                                          |
| Размер              | \-                                                                               |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                           |
| Частота обновления  | При создании записи пользователя и при необходимости изменения номера телефона. |
| Attribute-Id      | 1.2.840.113556.1.4.721                                                           |
| System-ID — GUID    | 4d146e4a-48d4-11d1-a9c3-0000f80367c1                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
