# <a name="ms-ds-consistency-guid-attribute"></a>MS-DS-целостность-атрибут GUID

Этот атрибут используется для проверки согласованности между каталогом и другим объектом, базой данных или приложением путем сравнения идентификаторов GUID.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MS-DS-Consistencу-Guid                                |
| LDAP-отображаемое имя | mS-DS-ConsistencyGuid                                 |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1360                               |
| System-ID — GUID    | 23773dc2-b63a-11d2-90e1-00c04fd91ab1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
