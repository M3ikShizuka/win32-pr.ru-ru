|-------------------|--------------------------------------|
| CN                | Search-Flags                         |
| LDAP-отображаемое имя | searchFlags                          |
| Размер              | \-                                   |
| Привилегия обновления  | Администратор схемы                 |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.334               |
| System-ID — GUID    | bf967a2d-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |

## <a name="remarks"></a>Remarks

Этот атрибут может быть нулевым или сочетанием одного или нескольких из следующих значений.



| Значение            | Описание                                                                                                                                                                                                                                                                                                                                                               |
|------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)   | Создайте индекс для атрибута.                                                                                                                                                                                                                                                                                                                                        |
| 2 (0x00000002)   | Создайте индекс для атрибута в каждом контейнере.                                                                                                                                                                                                                                                                                                                      |
| 4 (0x00000004)   | Добавьте этот атрибут в набор разрешения неоднозначных имен (ANR). Используется для поиска объекта при наличии только частичной информации. Например, если фильтр LDAP имеет значение (ANR = Джефф), поиск будет находить каждый объект, в котором имя, фамилия, адрес электронной почты или другой атрибут ANR равны ДЖЕФФу. Чтобы этот индекс вступил в силу, необходимо установить бит 0. |
| 8 (0x00000008)   | Сохраните этот атрибут в объекте захоронения для удаленных объектов.                                                                                                                                                                                                                                                                                                      |
| 16 (0x00000010)  | Скопируйте значение этого атрибута при копировании объекта.                                                                                                                                                                                                                                                                                                              |
| 32 (0x00000020)  | Создайте индекс кортежа для атрибута. Это улучшит Поиск, когда в начале строки поиска появится подстановочный знак. Например, (SN = \* МИС).                                                                                                                                                                                                                |
| 64 (0x00000040)   | Поддерживается начиная с ADAM. Создает индекс, значительно помогающий VLV производительность на произвольных атрибутах.                                                                                                                                                                                                                                                                  |
| 128 (0x00000080) | Пометьте атрибут как конфиденциальный. Игнорируется для атрибутов базовой схемы (systemFlags = 0x10).                                                                                                                                                                                                                                                                                    |
| 64 (0x00000040)  | поддерживается начиная с Windows Server 2008. Создайте индекс, чтобы улучшить производительность поиска VLV для этого атрибута.                                                                                                                                                                                                                                                        |
