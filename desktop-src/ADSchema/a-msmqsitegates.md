# <a name="msmq-site-gates-attribute"></a>Атрибут MSMQ-site-Gates

Список DNs-серверов маршрутизации MSMQ, через который должен маршрутизироваться весь трафик между сайтами.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MSMQ-site-Gates                         |
| LDAP-отображаемое имя | мсмкситегатес                           |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.945                  |
| System-ID — GUID    | 9a0dc339-c100-11d1-bbc5-0080c76670c0    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
