# <a name="mastered-by-attribute"></a>Атрибут Mastered-By

Обратная ссылка для атрибута с параметром-Master-NC. различающееся имя для объектов NTDS Параметры.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Mastered-By                             |
| LDAP-отображаемое имя | мастередби                              |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | При создании контекста именования.     |
| Attribute-Id      | 1.2.840.113556.1.4.1409                 |
| System-ID — GUID    | e48e64e0-12c9-11d3-9102-00c04fd91ab1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
