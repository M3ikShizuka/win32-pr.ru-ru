# <a name="trust-posix-offset-attribute"></a>Отношение доверия — атрибут POSIX-offset

Смещение переносимого интерфейса операционной системы (POSIX) для доверенного домена.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Отношение доверия-POSIX-offset                   |
| LDAP-отображаемое имя | трустпосиксоффсет                     |
| Размер              | 4 байта                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | При создании нового доверия.         |
| Attribute-Id      | 1.2.840.113556.1.4.134               |
| System-ID — GUID    | bf967a5e-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
