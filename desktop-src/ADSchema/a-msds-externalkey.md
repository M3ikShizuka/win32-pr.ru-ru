# <a name="ms-ds-external-key-attribute"></a>Атрибут ms-DS-External-Key

| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-External-Key                          |
| LDAP-отображаемое имя | msDS-Екстерналкэй                            |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1833                     |
| System-ID — GUID    | b92fd528-38ac-40d4-818d-0433380837c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
