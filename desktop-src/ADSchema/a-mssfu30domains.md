# <a name="mssfu-30-domains-attribute"></a>атрибут Мссфу-30-Domains

содержит список доменов UNIX nis, которые были перенесены в один и тот же домен AD nis.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Мссфу-30-домены                     |
| LDAP-отображаемое имя | msSFU30Domains                       |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.340          |
| System-ID — GUID    | 93095ed3-6f30-4bdd-b734-65d569f5f7c9 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
