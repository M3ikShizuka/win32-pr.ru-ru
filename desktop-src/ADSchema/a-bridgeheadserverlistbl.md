# <a name="bridgehead-server-list-bl-attribute"></a>Плацдарм-Server-List-BL, атрибут

Список серверов, являющихся серверами-плацдармами для репликации.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Плацдарм-Server-List-BL               |
| LDAP-отображаемое имя | бриджехеадсерверлистбл                  |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | При настройке сайта.               |
| Attribute-Id      | 1.2.840.113556.1.4.820                  |
| System-ID — GUID    | d50c2cdb-8951-11d1-aebc-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
