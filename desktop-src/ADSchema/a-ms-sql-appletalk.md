# <a name="ms-sql-appletalk-attribute"></a>атрибут MS-SQL-AppleTalk

Точка подключения AppleTalk.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-AppleTalk                            |
| LDAP-отображаемое имя | mS-SQL-AppleTalk                            |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | При запуске системы.                          |
| Attribute-Id      | 1.2.840.113556.1.4.1378                     |
| System-ID — GUID    | 8fda89f4-ccee-11d2-9993-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
