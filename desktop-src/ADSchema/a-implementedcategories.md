# <a name="implemented-categories-attribute"></a>Атрибут Implemented-Categories

Этот атрибут содержит список идентификаторов категорий компонентов, реализуемых этим объектом.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Implemented-Categories                                |
| LDAP-отображаемое имя | имплементедкатегориес                                 |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.320                                |
| System-ID — GUID    | 7d6c0e92-7e20-11d0-afd6-00c04fd930c9                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
