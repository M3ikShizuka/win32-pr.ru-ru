# <a name="acs-max-no-of-account-files-attribute"></a>Атрибут ACS-Max-No-of-Account-Files

Максимальное число файлов учетной записи RSVP.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-Max-без учета--файлов          |
| LDAP-отображаемое имя | аксмакснуфаккаунтфилес               |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.901               |
| System-ID — GUID    | f0722310-aef5-11d1-bdcf-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
