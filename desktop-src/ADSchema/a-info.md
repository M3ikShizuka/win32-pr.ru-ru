# <a name="comment-attribute-ad-schema"></a>Атрибут Comment (схема AD)

Комментарии пользователя. Эта строка может быть пустой строкой.



| Ввод | Значение |
|-------------------|--------------------------------------------------------------------------|
| CN                | Комментировать                                                                  |
| LDAP-отображаемое имя | сведения                                                                     |
| Размер              | \-                                                                       |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                   |
| Частота обновления  | Каждый раз, когда пользователю или администратору необходимо добавить комментарии к учетной записи. |
| Attribute-Id      | 1.2.840.113556.1.2.81                                                    |
| System-ID — GUID    | bf96793e-0de6-11d0-a285-00aa003049e2                                     |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                              |
