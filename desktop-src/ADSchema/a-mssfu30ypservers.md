# <a name="mssfu-30-yp-servers-attribute"></a>атрибут Мссфу-30-ИП-Servers

Содержит список NIS-серверов в домене NIS.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Мссфу-30-ИП-серверы                  |
| LDAP-отображаемое имя | msSFU30YpServers                     |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.341          |
| System-ID — GUID    | 084a944b-e150-4bfe-9345-40e1aedaebba |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
