# <a name="frs-ds-poll-attribute"></a>Атрибут FRS-DS-poll

Интервал опроса DS для модуля репликации файлов.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | FRS-DS-poll                          |
| LDAP-отображаемое имя | фрсдсполл                            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.490               |
| System-ID — GUID    | 1be8f177-a9ff-11d0-afe2-00c04fd930c9 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
