# <a name="open-address-book-extended-right"></a>Open-Address-Book, расширенное право

Расширенное право проверено при открытии объекта адресной книги для представлений адресной книги.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Справочник по открытым адресам                    |
| Display-Name | Открыть список адресов                    |
| Rights-GUID  | a1990816-4298-11d1-ade2-00c04fd8d5cd |
