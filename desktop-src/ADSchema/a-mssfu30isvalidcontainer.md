# <a name="mssfu-30-is-valid-container-attribute"></a>Мссфу-30 — является допустимым атрибутом контейнера

| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |

| Синтаксис            | [**Перечисление**](s-enumeration.md) |
