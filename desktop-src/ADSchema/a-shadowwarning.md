# <a name="shadowwarning-attribute"></a>атрибут Шадовварнинг

Содержит число дней до истечения срока действия пароля для предупреждения пользователя.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | шадовварнинг                        |
| LDAP-отображаемое имя | шадовварнинг                        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.8                      |
| System-ID — GUID    | 7ae89c9c-2976-4a46-bb8a-340f88560117 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
