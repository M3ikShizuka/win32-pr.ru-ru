# <a name="ms-wmi-genus-attribute"></a>атрибут MS-WMI-Женус

Определяет тип объекта кодировки.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------|
| CN                | MS-WMI-Женус                                                    |
| LDAP-отображаемое имя | Мсвми — Женус                                                     |
| Размер              | 4 байта. 1 обозначает класс. 2 обозначает экземпляр.          |
| Привилегия обновления  | Администратор домена                                            |
| Частота обновления  | Каждый раз при добавлении или изменении класса, содержащего атрибут. |
| Attribute-Id      | 1.2.840.113556.1.4.1677                                         |
| System-ID — GUID    | 50c8673a-8f56-4614-9308-9e1340fb9af3                            |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                            |
