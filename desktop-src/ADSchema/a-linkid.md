# <a name="link-id-attribute"></a>Атрибут Link-ID

Целое число, указывающее, что атрибут является связанным атрибутом. Четное целое число — это прямая ссылка, а нечетное целое — обратная.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Идентификатор ссылки                              |
| LDAP-отображаемое имя | Параметра                               |
| Размер              | \-                                   |
| Привилегия обновления  | Администратор схемы                 |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.50                |
| System-ID — GUID    | bf96799b-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
