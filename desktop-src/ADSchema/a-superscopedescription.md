# <a name="super-scope-description-attribute"></a>Атрибут Super-Scope-Description

Этот атрибут предоставляет описание суперобласти.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Супер-Scope — описание                     |
| LDAP-отображаемое имя | суперскопедескриптион                       |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.711                      |
| System-ID — GUID    | 963d274c-48be-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
