# <a name="print-duplex-supported-attribute"></a>Print — поддерживаемый атрибут

Указывает тип дуплексной поддержки принтера.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Печать — дуплексная — поддерживается                                |
| LDAP-отображаемое имя | printDuplexSupported                                  |
| Размер              | 4 байта. Поддержка дуплекса = 2. Симплексная = 1 или 0. |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1311                               |
| System-ID — GUID    | 281416cc-1968-11d0-a28f-00aa003049e2                  |
| Синтаксис            | [**Логическая**](s-boolean.md)                          |
