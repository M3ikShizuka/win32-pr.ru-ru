# <a name="acs-enable-acs-service-attribute"></a>Атрибут ACS-Enable-ACS-Service

Значение true, если служба ACS должна быть включена.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-Enable-ACS-Service               |
| LDAP-отображаемое имя | аксенаблеакссервице                  |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.770               |
| System-ID — GUID    | 7f561287-5301-11d1-a9c5-0000f80367c1 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
