# <a name="acs-max-token-bucket-per-flow-attribute"></a>Атрибут ACS-Max-Token-контейнеров-Flow

Атрибут **ACS-Max-Token-контейнеров-per-Flow** предназначен только для внутреннего использования. На основе RFC2210.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-Max-Token-контейнер-на поток        |
| LDAP-отображаемое имя | аксмакстокенбуккетперфлов             |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1313              |
| System-ID — GUID    | 81f6e0df-3b90-11d2-90cc-00c04fd91ab1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
