# <a name="uas-compat-attribute"></a>Атрибут UAS-Compat

Указывает, будет ли диспетчер учетных записей безопасности применять размеры данных, чтобы сделать Active Directory совместимыми с системой учетных записей пользователей (UAS) Ланманажер. Если это значение равно 0, ограничения не применяются. Если это значение равно 1, применяются следующие ограничения.



| Значение                          | Длина                         |
|--------------------------------|--------------------------------|
| Пароль<br/>            | от 0 до 14 символов<br/>  |
| Имя учетной записи<br/>        | от 0 до 20 символов<br/>  |
| Имя домена<br/>         | от 0 до 15 символов<br/>  |
| Имя компьютера<br/>       | от 0 до 15 символов<br/>  |
| Комментарии<br/>            | от 0 до 48 символов<br/>  |
| Домашний каталог<br/>      | от 0 до 256 символов<br/> |
| Путь к сценарию<br/>         | от 0 до 256 символов<br/> |
| Единицы времени в неделю<br/> | 168 бит (21 байт)<br/> |



 



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | UAS-Compat                           |
| LDAP-отображаемое имя | уаскомпат                            |
| Размер              | \-                                   |
| Привилегия обновления  | Выполняется администратором.       |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.155               |
| System-ID — GUID    | bf967a61-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
