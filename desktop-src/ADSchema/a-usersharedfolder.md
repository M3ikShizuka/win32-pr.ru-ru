# <a name="user-shared-folder-attribute"></a>Атрибут общей папки для пользователя

Указывает UNC-путь к папке общих документов пользователя. Путь должен представлять собой сетевой UNC-путь к **\\\\**  *_\\_* _общему_ *_\\_* _каталогу_ сервера форм. Это значение может быть пустой строкой.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------------------------|
| CN                | Общая папка пользователя                                                                |
| LDAP-отображаемое имя | усершаредфолдер                                                                  |
| Размер              | \-                                                                                |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                            |
| Частота обновления  | При создании записи пользователя и при необходимости изменения общей папки. |
| Attribute-Id      | 1.2.840.113556.1.4.751                                                            |
| System-ID — GUID    | 9a9a021f-4a5b-11d1-a9c3-0000f80367c1                                              |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                       |
