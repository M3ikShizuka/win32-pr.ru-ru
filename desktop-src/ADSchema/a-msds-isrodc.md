# <a name="ms-ds-isrodc-attribute"></a>Атрибут ms-DS-в RODC

Показывает, является ли контроллер домена только для чтения.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-доrodc                         |
| LDAP-отображаемое имя | msDS-доrodc                          |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1960              |
| System-ID — GUID    | a8e8aa23-3e67-4af1-9d7a-2f1a1d633ac9 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
