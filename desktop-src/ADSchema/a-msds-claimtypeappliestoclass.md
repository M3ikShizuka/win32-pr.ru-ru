# <a name="ms-ds-claim-type-applies-to-class-attribute"></a>атрибут применения MS-DS-ClaimSet-Type-to-Class

Для объекта типа утверждения этот связанный атрибут указывает на классы субъектов безопасности Active Directory, для которых должны выдаваться утверждения. (Например, ссылка на класс пользователя).



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-заявка-тип — применяется к классу       |
| LDAP-отображаемое имя | msDS-Клаимтипеапплиестокласс            |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2100                 |
| System-ID — GUID    | 6afb0e4c-d876-437c-aeb6-c3e41454c272    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
