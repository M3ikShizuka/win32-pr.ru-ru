# <a name="smtp-mail-address-attribute"></a>Атрибут адреса SMTP-Mail

Атрибут универсального почтового адреса. Используется в поле как необязательный атрибут объектов сервера, где он используется репликацией DS на основе почты (если настроены компьютеры).



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Адрес SMTP-Mail                           |
| LDAP-отображаемое имя | mailAddress                                 |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | Почти никогда                                |
| Attribute-Id      | 1.2.840.113556.1.4.786                      |
| System-ID — GUID    | 26d9736f-6070-11d1-a9c6-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
