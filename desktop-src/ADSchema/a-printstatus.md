# <a name="print-status-attribute"></a>Атрибут Print-Status

Состояние из диспетчера очереди печати. В настоящее время не используется.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Print-Status                                |
| LDAP-отображаемое имя | принтстатус                                 |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.273                      |
| System-ID — GUID    | ba305f6b-47e3-11d0-a1a6-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
