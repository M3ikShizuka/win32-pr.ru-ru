# <a name="ms-sql-lastdiagnosticdate-attribute"></a>Атрибут MS-SQL-Ластдиагностикдате

Последняя дата выполнения инструкции DBCC CHECKDB.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Ластдиагностикдате                   |
| LDAP-отображаемое имя | mS-SQL-Ластдиагностикдате                   |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | При выполнении инструкции DBCC CHECKDB.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1399                     |
| System-ID — GUID    | f6d6dd88-ccee-11d2-9993-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
