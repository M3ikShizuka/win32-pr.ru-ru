# <a name="home-directory-attribute"></a>Атрибут Home-Directory

Корневой каталог для учетной записи. Если [**хомедриве**](a-homedrive.md) задано и указывает букву диска, **ХОМЕДИРЕКТОРИ** должен быть UNC-путем. В противном случае **хомедиректори** — это полный локальный путь, включая букву диска (например, папка *буква_диска ***: \\**_Каталог_* _\\_ * ). Это значение может быть пустой строкой.



| Ввод | Значение |
|-------------------|------------------------------------------------------------------------------------|
| CN                | Home-Directory                                                                     |
| LDAP-отображаемое имя | хомедиректори                                                                      |
| Размер              | \-                                                                                 |
| Привилегия обновления  | Администратор домена                                                               |
| Частота обновления  | При создании записи пользователя и при необходимости изменения домашнего каталога. |
| Attribute-Id      | 1.2.840.113556.1.4.44                                                              |
| System-ID — GUID    | bf967985-0de6-11d0-a285-00aa003049e2                                               |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                        |
