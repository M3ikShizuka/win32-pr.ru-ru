# <a name="group-type-attribute"></a>Атрибут Group-Type

Содержит набор флагов, определяющих тип и область объекта группы. Возможные значения для этого атрибута см. в разделе Примечания.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Group-Type                           |
| LDAP-отображаемое имя | groupType                            |
| Размер              | \-                                   |
| Привилегия обновления  | Администратор домена                 |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.750               |
| System-ID — GUID    | 9a9a021e-4a5b-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |

## <a name="remarks"></a>Комментарии

Этот атрибут может быть нулевым или сочетанием одного или нескольких из следующих значений.



| Значение                   | Описание                                                                                  |
|-------------------------|----------------------------------------------------------------------------------------------|
| 1 (0x00000001)          | Указывает группу, созданную системой.                                             |
| 2 (0x00000002)          | Указывает группу с глобальной областью.                                                         |
| 4 (0x00000004)          | Указывает группу с локальной областью действия домена.                                                   |
| 8 (0x00000008)          | Задает группу с универсальной областью действия.                                                      |
| 16 (0x00000010)         | Указывает \_ группу Basic для приложения для диспетчера авторизации Windows Server.                      |
| 32 (0x00000020)         | Указывает \_ группу запросов приложений для диспетчера авторизации Windows Server.                      |
| 2147483648 (0x80000000) | Указывает группу безопасности. Если этот флаг не установлен, группа является группой рассылки. |



 

Дополнительные сведения о типе и области группы см. в разделах [типы групп](/previous-versions/windows/it-pro/windows-server-2003/cc781446(v=ws.10)) и [область групп](/previous-versions/windows/it-pro/windows-server-2003/cc755692(v=ws.10)) в Microsoft TechNet.
