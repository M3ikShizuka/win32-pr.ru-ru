# <a name="upn-suffixes-attribute"></a>Атрибут UPN-Suffixes

Список суффиксов имени участника-пользователя для домена.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | UPN-Suffixes                                |
| LDAP-отображаемое имя | упнсуффиксес                                 |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.890                      |
| System-ID — GUID    | 032160bf-9824-11d1-aec0-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
