# <a name="ms-pki-ra-application-policies-attribute"></a>атрибут MS-PKI-RA-Application-Policies

Требуемый OID политики приложения RA в сигнатурах счетчика запроса на сертификат.



| Ввод | Значение |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-RA-приложения — политики                                                             |
| LDAP-отображаемое имя | Мспки-RA-приложение-политики                                                              |
| Размер              | \-                                                                                         |
| Привилегия обновления  | Администратор предприятия                                                                   |
| Частота обновления  | При создании нового шаблона или изменении атрибутов существующих шаблонов. |
| Attribute-Id      | 1.2.840.113556.1.4.1675                                                                    |
| System-ID — GUID    | 3c91fbbf-4773-4ccd-a87b-85d53e7bcf6a                                                       |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                                |
