# <a name="system-may-contain-attribute"></a>System-может-содержать атрибут

Список необязательных атрибутов для класса. Список атрибутов может быть изменен только системой.



| Ввод | Значение |
|-------------------|------------------------------------------------------------------------------|
| CN                | System — может содержать                                                           |
| LDAP-отображаемое имя | системмайконтаин                                                             |
| Размер              | \-                                                                           |
| Привилегия обновления  | Администратор схемы                                                         |
| Частота обновления  | При создании класса или добавлении к классу нового необязательного атрибута. |
| Attribute-Id      | 1.2.840.113556.1.4.196                                                       |
| System-ID — GUID    | bf967a44-0de6-11d0-a285-00aa003049e2                                         |
| Синтаксис            | [**String(идентификатор_объекта)**](s-string-object-identifier.md)              |
