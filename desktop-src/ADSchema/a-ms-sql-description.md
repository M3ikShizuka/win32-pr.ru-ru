# <a name="ms-sql-description-attribute"></a>Атрибут MS-SQL-Description

Описание базы данных SQL Server.



| Ввод | Значение |
|-------------------|--------------------------------------------------|
| CN                | MS-SQL — описание                               |
| LDAP-отображаемое имя | mS-SQL — описание                               |
| Размер              | \-                                               |
| Привилегия обновления  | Администратор домена                             |
| Частота обновления  | Когда пользователь решает изменить описание. |
| Attribute-Id      | 1.2.840.113556.1.4.1390                          |
| System-ID — GUID    | 8386603c-ccef-11d2-9993-0000f87a57d4             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)      |
