# <a name="gpc-wql-filter-attribute"></a>ГПК-WQL-атрибут Filter

Используется для хранения строки, содержащей идентификатор GUID для фильтра и пути к пространству имен WMI.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------|
| CN                | ГПК-WQL-Filter                                                                  |
| LDAP-отображаемое имя | гпквклфилтер                                                                    |
| Размер              | \-                                                                              |
| Привилегия обновления  | Администратор домена                                                            |
| Частота обновления  | Только в том случае, если администратор изменяет свойство Filter в пользовательском интерфейсе групповая политика. |
| Attribute-Id      | 1.2.840.113556.1.4.1694                                                         |
| System-ID — GUID    | 7bd4c7a6-1add-4436-8c04-3999a880154c                                            |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                     |
