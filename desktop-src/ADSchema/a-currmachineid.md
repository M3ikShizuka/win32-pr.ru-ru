# <a name="curr-machine-id-attribute"></a>Атрибут-Machine-ID

ИДЕНТИФИКАТОР компьютера, на котором находится объект Link-Track-Vol-Entry.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Идентификатор компьютера                                       |
| LDAP-отображаемое имя | куррмачинеид                                         |
| Размер              | \-                                                    |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.337                                |
| System-ID — GUID    | 1f0075fe-7e40-11d0-afd6-00c04fd930c9                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
