| LDAP-отображаемое имя | MS-NET-IEEE-8023-GP-Полицидата              |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1955                     |
