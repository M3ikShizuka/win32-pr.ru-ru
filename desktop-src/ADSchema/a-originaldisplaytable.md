| LDAP-отображаемое имя | оригиналдисплайтабле                                  |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.445                                |
| System-ID — GUID    | 5fd424ce-1262-11d0-a060-00aa006c33ed                  |
