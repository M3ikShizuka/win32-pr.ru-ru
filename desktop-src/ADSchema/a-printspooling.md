# <a name="print-spooling-attribute"></a>Атрибут Print-Spooling

Строка, представляющая тип очереди печати принтера.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------|
| CN                | Print-Spooling                                                       |
| LDAP-отображаемое имя | принтспулинг                                                        |
| Размер              | Возможные значения: Принтдирект, Принтвхилеспулинг, Принтафтерспулед. |
| Привилегия обновления  | \-                                                                   |
| Частота обновления  | \-                                                                   |
| Attribute-Id      | 1.2.840.113556.1.4.274                                               |
| System-ID — GUID    | ba305f6c-47e3-11d0-a1a6-00c04fd930c9                                 |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                          |
