# <a name="msmq-site-1-attribute"></a>Атрибут MSMQ-Site-1

Различающееся имя первого сайта подключенной пары.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MSMQ-Site-1                             |
| LDAP-отображаемое имя | mSMQSite1                               |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | При каждом создании строки MSMQ-site-Line. |
| Attribute-Id      | 1.2.840.113556.1.4.943                  |
| System-ID — GUID    | 9a0dc337-c100-11d1-bbc5-0080c76670c0    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
