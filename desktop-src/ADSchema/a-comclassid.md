# <a name="com-classid-attribute"></a>Атрибут COM-ClassID

Этот атрибут хранит список идентификаторов классов, реализованных в этом пакете приложения.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | COM-ClassID                                 |
| LDAP-отображаемое имя | комклассид                                  |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.19                       |
| System-ID — GUID    | bf96793b-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
