# <a name="ms-sql-allowknownpullsubscription-attribute"></a>атрибут MS-SQL-алловкновнпуллсубскриптион

Значение true, если публикация позволяет зарегистрированным подписчикам подписываться.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-SQL-алловкновнпуллсубскриптион    |
| LDAP-отображаемое имя | mS-SQL-алловкновнпуллсубскриптион    |
| Размер              | 4 байта                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | При установке репликации.           |
| Attribute-Id      | 1.2.840.113556.1.4.1403              |
| System-ID — GUID    | c3bb7054-d34b-11d2-999a-0000f87a57d4 |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
