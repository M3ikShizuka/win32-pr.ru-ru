# <a name="user-change-password-extended-right"></a>Пользователь-Change-пароль расширен вправо

Разрешает изменение пароля для учетной записи пользователя.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Изменение пароля пользователя                 |
| Display-Name | Изменить пароль                      |
| Rights-GUID  | ab721a53-1e2f-11d0-9819-00aa0040529b |
