# <a name="default-object-category-attribute"></a>Атрибут по умолчанию — объект-Категория

Категория объектов, используемая для объекта, если он не указан.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | По умолчанию — объект — Категория                 |
| LDAP-отображаемое имя | defaultObjectCategory                   |
| Размер              | \-                                      |
| Привилегия обновления  | Администратор схемы                    |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.783                  |
| System-ID — GUID    | 26d97367-6070-11d1-a9c6-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
