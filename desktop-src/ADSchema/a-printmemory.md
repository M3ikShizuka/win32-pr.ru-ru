# <a name="print-memory-attribute"></a>Атрибут Print-Memory

Объем памяти, установленной на принтере.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Print-Memory                         |
| LDAP-отображаемое имя | printMemory                          |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.282               |
| System-ID — GUID    | ba305f74-47e3-11d0-a1a6-00c04fd930c9 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
