# <a name="system-auxiliary-class-attribute"></a>Атрибут System-вспомогательного класса

Список вспомогательных классов, которые не могут быть изменены пользователем.



| Ввод | Значение |
|-------------------|--------------------------------------------------------------------|
| CN                | Системный-вспомогательный класс                                             |
| LDAP-отображаемое имя | системауксилиарикласс                                               |
| Размер              | \-                                                                 |
| Привилегия обновления  | Администратор схемы                                               |
| Частота обновления  | При создании класса или добавлении к нему нового вспомогательного класса. |
| Attribute-Id      | 1.2.840.113556.1.4.198                                             |
| System-ID — GUID    | bf967a43-0de6-11d0-a285-00aa003049e2                               |
| Синтаксис            | [**String(идентификатор_объекта)**](s-string-object-identifier.md)    |
