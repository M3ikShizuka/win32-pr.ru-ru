| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Предпочитаемое подразделение                            |
| LDAP-отображаемое имя | преферредау                             |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.97                   |
| System-ID — GUID    | bf9679ff-0de6-11d0-a285-00aa003049e2    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
