# <a name="is-defunct-attribute"></a>Атрибут Is-Defunct

Если **значение — true**, класс или атрибут больше не может использоваться. Старые версии этого объекта могут существовать, но создать новые нельзя.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Is-Defunct                           |
| LDAP-отображаемое имя | Уничтоженный                            |
| Размер              | \-                                   |
| Привилегия обновления  | Администратор схемы                 |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.661               |
| System-ID — GUID    | 28630ebe-41d5-11d1-a9c1-0000f80367c1 |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
