| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-COM-Усерпартитионсетлинк             |
| LDAP-отображаемое имя | Мском — Усерпартитионсетлинк              |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1426                 |
| System-ID — GUID    | 8e940c8a-e477-4367-b08d-ff2ff942dcd7    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
