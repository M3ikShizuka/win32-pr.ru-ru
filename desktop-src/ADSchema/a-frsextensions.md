# <a name="frs-extensions-attribute"></a>Атрибут FRS-Extensions

Двоичные данные, используемые при репликации файлов.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | FRS-Extensions                                        |
| LDAP-отображаемое имя | фрсекстенсионс                                         |
| Размер              | \-                                                    |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.536                                |
| System-ID — GUID    | 52458020-ca6a-11D0-аффф-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
