# <a name="stringnt-sec-desc-syntax"></a>Синтаксис строки (NT-sec-DESC)

Строка октета, содержащая дескриптор безопасности Windows NT или Windows 2000.



| Ввод | Значение |
|--------------|------------------------------------------------------------------------------------------------|
| Имя         | String(NT-Sec-Desc)                                                                            |
| Идентификатор синтаксиса    | 2.5.5.15                                                                                       |
| ИДЕНТИФИКАТОР OM        | 66                                                                                             |
| Тип MAPI    | BINARY                                                                                         |
| Тип ADS     | \_ \_ дескриптор безопасности адстипе \_ NT                                                              |
| Тип варианта | \_Диспетчер VT                                                                                   |
| Тип SDS     | COM-объект, который можно привести к [**иадссекуритидескриптор**](/windows/desktop/api/iads/nn-iads-iadssecuritydescriptor). |
