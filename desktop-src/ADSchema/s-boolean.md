# <a name="boolean-syntax"></a>Логический синтаксис

Представляет логическое значение.



| Ввод | Значение |
|--------------|--------------------------------------------------------------------------|
| Имя         | Логическое                                                                  |
| Идентификатор синтаксиса    | 2.5.5.8                                                                  |
| ИДЕНТИФИКАТОР OM        | 1                                                                        |
| Тип MAPI    | BOOLEAN                                                                  |
| Тип ADS     | АДСТИПЕ \_ Boolean                                                         |
| Тип варианта | Логическое значение VT \_                                                                 |
| Тип SDS     | [System.Boolean](/dotnet/api/system.boolean) |



## <a name="remarks"></a>Комментарии

Для запросов, содержащих атрибуты логического синтаксиса в фильтре, укажите значение TRUE или FALSE (например, "мибулаттр = TRUE").
