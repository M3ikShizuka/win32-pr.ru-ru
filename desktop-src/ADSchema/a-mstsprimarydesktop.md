# <a name="ms-ts-primary-desktop-attribute"></a>атрибут MS-TS-PRIMARY-Desktop

Этот атрибут представляет прямую ссылку на основной рабочий стол пользователя.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-TS-основной рабочий стол                   |
| LDAP-отображаемое имя | мстспримаридесктоп                      |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2073                 |
| System-ID — GUID    | 29259694-09e4-4237-9f72-9306ebe63ab2    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
