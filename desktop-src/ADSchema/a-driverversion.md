# <a name="driver-version-attribute"></a>Атрибут Driver-Version

Номер версии драйвера устройства.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Driver-Version                       |
| LDAP-отображаемое имя | driverVersion                        |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.276               |
| System-ID — GUID    | ba305f6e-47e3-11d0-a1a6-00c04fd930c9 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
