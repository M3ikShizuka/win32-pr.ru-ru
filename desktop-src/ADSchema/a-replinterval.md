# <a name="repl-interval-attribute"></a>Атрибут Repl-Interval

Атрибут Site-Link объектов, который определяет интервал (в минутах) между циклами репликации между сайтами в списке сайтов. Должен быть кратен 15 минутам (детализация репликации межсайтовых каталогов), не менее 15 минут и не более 10 080 минут (одна неделя).



| Ввод | Значение |
|-------------------|------------------------------------------------|
| CN                | Repl-Interval                                  |
| LDAP-отображаемое имя | реплинтервал                                   |
| Размер              | 4 байта                                        |
| Привилегия обновления  | Администратор предприятия                       |
| Частота обновления  | При необходимости изменения интервала репликации. |
| Attribute-Id      | 1.2.840.113556.1.4.1336                        |
| System-ID — GUID    | 45ba9d1a-56fa-11d2-90d0-00c04fd91ab1           |
| Синтаксис            | [**Перечисление**](s-enumeration.md)           |
