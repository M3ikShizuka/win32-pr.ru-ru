# <a name="acs-non-reserved-tx-size-attribute"></a>ACS-не зарезервированный — атрибут размера TX

Размер контейнера маркера, который приложение может использовать перед резервированием.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS — не зарезервировано — размер TX             |
| LDAP-отображаемое имя | акснонресерведткссизе                 |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.898               |
| System-ID — GUID    | f072230d-aef5-11d1-bdcf-0000f80367c1 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
