# <a name="ms-ts-work-directory-attribute"></a>атрибут MS-TS-рабочий каталог

Рабочий каталог сеанса служб терминалов указывает путь к рабочему каталогу для пользователя.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-TS-рабочий каталог                        |
| LDAP-отображаемое имя | мстсворкдиректори                           |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1989                     |
| System-ID — GUID    | a744f666-3d3c-4cc8-834b-9d4f6f687b8b        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
