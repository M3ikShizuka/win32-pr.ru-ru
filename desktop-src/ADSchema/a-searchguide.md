# <a name="search-guide-attribute"></a>Атрибут Search-Guide

Указывает сведения о предлагаемых критериях поиска, которые могут включаться в некоторые записи, которые должны быть удобным базовым объектом для операции поиска, например в стране или регионе или организации.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Search-Guide                                          |
| LDAP-отображаемое имя | сеарчгуиде                                           |
| Размер              | \-                                                    |
| Привилегия обновления  | Администратор домена                                  |
| Частота обновления  | Всякий раз, когда требуется новое условие поиска.               |
| Attribute-Id      | 2.5.4.14                                              |
| System-ID — GUID    | bf967a2e-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
