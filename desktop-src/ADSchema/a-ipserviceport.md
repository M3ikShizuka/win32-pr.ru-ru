# <a name="ipserviceport-attribute"></a>атрибут Ипсервицепорт

Содержит раздел схемы служб, указывающий порт, на котором доступна служба UNIX.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ипсервицепорт                        |
| LDAP-отображаемое имя | ипсервицепорт                        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.15                     |
| System-ID — GUID    | ff2daebf-f463-495a-8405-3e483641eaa2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
