# <a name="ipsec-data-type-attribute"></a>Атрибут типа данных IPsec

Атрибут **IPSec-Data-Type** предназначен только для внутреннего использования.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | IPSec-Data-Type                      |
| LDAP-отображаемое имя | ипсекдататипе                        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.622               |
| System-ID — GUID    | b40ff81e-427a-11d1-a9c2-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
