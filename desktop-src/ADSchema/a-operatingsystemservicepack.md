# <a name="operating-system-service-pack-attribute"></a>Атрибут "операционная система-Пакетная служба"

Строка идентификатора пакета обновления операционной системы (например, SP3).



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Операционная система-Service-Pack               |
| LDAP-отображаемое имя | оператингсистемсервицепакк                  |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.365                      |
| System-ID — GUID    | 3e978927-8c01-11d0-afda-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
