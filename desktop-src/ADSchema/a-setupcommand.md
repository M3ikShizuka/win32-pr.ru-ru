# <a name="setup-command-attribute"></a>Атрибут Setup-Command

Этот атрибут указывает, требуется ли команда установки для настройки этого приложения.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Setup-Command                               |
| LDAP-отображаемое имя | сетупкомманд                                |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.325                      |
| System-ID — GUID    | 7d6c0e97-7e20-11d0-afd6-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
