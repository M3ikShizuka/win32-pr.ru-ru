# <a name="pki-critical-extensions-attribute"></a>Атрибут PKI-Critical-Extensions

Список критических расширений в шаблоне сертификата.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | PKI — критические расширения                     |
| LDAP-отображаемое имя | пкикритикалекстенсионс                       |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1330                     |
| System-ID — GUID    | fc5a9106-3b9d-11d2-90cc-00c04fd91ab1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
