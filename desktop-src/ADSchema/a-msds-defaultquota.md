# <a name="ms-ds-default-quota-attribute"></a>атрибут квоты MS-DS-Default-

Квота по умолчанию, которая будет применяться к субъекту безопасности, который создает объект в NC, если не существует спецификации квоты, охватывающей субъект безопасности.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-Default-квота                  |
| LDAP-отображаемое имя | msDS-Дефаулткуота                    |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1846              |
| System-ID — GUID    | 6818f726-674b-441b-8a3a-f40596374cea |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
