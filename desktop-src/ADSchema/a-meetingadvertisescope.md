# <a name="meetingadvertisescope-attribute"></a>атрибут Митингадвертисескопе

Указывает, следует ли объявлять запись за пределами корпоративного шлюза или прокси-сервера.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | митингадвертисескопе                       |
| LDAP-отображаемое имя | митингадвертисескопе                       |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.582                      |
| System-ID — GUID    | 11b6cc8b-48c4-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
