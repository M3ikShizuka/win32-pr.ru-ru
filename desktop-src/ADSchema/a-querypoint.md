# <a name="querypoint-attribute"></a>Атрибут Куерипоинт

URL-адрес или UNC страницы запроса или другой внешний интерфейс для доступа к каталогу.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | куерипоинт                                  |
| LDAP-отображаемое имя | куерипоинт                                  |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.680                      |
| System-ID — GUID    | 7bfdcb86-4807-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
