# <a name="msmq-journal-quota-attribute"></a>Атрибут квоты MSMQ-Journal

Квота сообщений журнала MSMQ для компьютера.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Очередь MSMQ — журнал — квота                   |
| LDAP-отображаемое имя | мсмкжаурналкуота                     |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.921               |
| System-ID — GUID    | 9a0dc324-c100-11d1-bbc5-0080c76670c0 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
