# <a name="dns-property-attribute"></a>Атрибут DNS-Property

Используется для хранения двоичных параметров (свойств) в объектах зоны DNS.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | DNS-Property                                          |
| LDAP-отображаемое имя | днспроперти                                           |
| Размер              | \-                                                    |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | При каждом изменении записей зоны DNS.                     |
| Attribute-Id      | 1.2.840.113556.1.4.1306                               |
| System-ID — GUID    | 675a15fe-3b70-11d2-90cc-00c04fd91ab1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
