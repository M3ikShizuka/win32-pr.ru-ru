# <a name="documenttitle-attribute"></a>атрибут Документтитле

Тип атрибута Документтитле задает заголовок документа.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | документтитле                               |
| LDAP-отображаемое имя | документтитле                               |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.12                  |
| System-ID — GUID    | de265a9c-ff2c-47b9-91dc-6e6fe2c43062        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
