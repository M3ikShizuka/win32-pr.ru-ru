# <a name="msmq-peek-computer-journal-extended-right"></a>MSMQ-Peek-Computer-журнал, расширенное право

Позволяет просматривать сообщения в очереди журнала компьютера.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | MSMQ-просмотр-компьютер-журнал           |
| Display-Name | Просмотр журнала компьютера                |
| Rights-GUID  | 4b6e08c3-df3c-11d1-9c86-006008764d0e |
