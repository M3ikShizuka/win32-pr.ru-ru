# <a name="msmq-routing-service-attribute"></a>Атрибут MSMQ-Routing-Service

Указывает, предоставляет ли этот сервер службы маршрутизации.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MSMQ-Routing-Service                 |
| LDAP-отображаемое имя | мсмкраутингсервице                   |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1237              |
| System-ID — GUID    | 2df90d81-009f-11d2-aa4c-00c04fd7d83a |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
