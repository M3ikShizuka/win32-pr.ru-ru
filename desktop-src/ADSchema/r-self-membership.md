# <a name="self-membership-validated-writes"></a>Self-Membership проверенных записей

Проверено разрешение на запись для включения обновления членства в группе с точки зрения добавления или удаления собственной учетной записи.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Self-Membership                      |
| Display-Name | Добавить или удалить самого себя в качестве члена            |
| Rights-GUID  | bf9679c0-0de6-11d0-a285-00aa003049e2 |
