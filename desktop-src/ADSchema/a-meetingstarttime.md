# <a name="meetingstarttime-attribute"></a>атрибут Митингстарттиме

| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | митингстарттиме                                                                 |
| LDAP-отображаемое имя | митингстарттиме                                                                 |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.587                                                           |
| System-ID — GUID    | 11b6cc90-48c4-11d1-a9c3-0000f80367c1                                             |
| Синтаксис            | [**String(обобщенное_время)**](s-string-generalized-time.md)                    |
