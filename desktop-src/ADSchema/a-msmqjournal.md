# <a name="msmq-journal-attribute"></a>Атрибут MSMQ-Journal

Указывает, должны ли сообщения, полученные в виде очереди, храниться в журнале.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MSMQ-Journal                         |
| LDAP-отображаемое имя | мсмкжаурнал                          |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.918               |
| System-ID — GUID    | 9a0dc321-c100-11d1-bbc5-0080c76670c0 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
