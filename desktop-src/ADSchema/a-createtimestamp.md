# <a name="create-time-stamp-attribute"></a>Атрибут создания метки времени

Дата создания этого объекта. Это значение реплицируется.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------|
| CN                | Метка времени создания                                             |
| LDAP-отображаемое имя | креатетиместамп                                               |
| Размер              | 8 байт                                                       |
| Привилегия обновления  | Это значение задается системой.                              |
| Частота обновления  | При создании объекта.                                   |
| Attribute-Id      | 2.5.18.1                                                      |
| System-ID — GUID    | 2df90d73-009f-11d2-aa4c-00c04fd7d83a                          |
| Синтаксис            | [**String(обобщенное_время)**](s-string-generalized-time.md) |
