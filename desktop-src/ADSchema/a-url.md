# <a name="www-page-other-attribute"></a>WWW-Page-другой атрибут

Список альтернативных веб-страниц.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------------------|
| CN                | WWW-Page — другие                                                              |
| LDAP-отображаемое имя | url                                                                         |
| Размер              | \-                                                                          |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                      |
| Частота обновления  | Когда создается запись пользователя и каждый раз, когда необходимо изменить веб-страницу. |
| Attribute-Id      | 1.2.840.113556.1.4.749                                                      |
| System-ID — GUID    | 9a9a0221-4a5b-11d1-a9c3-0000f80367c1                                        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                 |
