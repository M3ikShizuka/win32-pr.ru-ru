# <a name="dns-root-attribute"></a>Атрибут Dns-Root

Самое верхнему доменному имени DNS, назначенному разделу домена или каталога. Это значение задается для объекта crossRef и используется, помимо прочего, для создания ссылок. При поиске по всему доменному дереву поиск должен быть инициирован в объекте Dns-Root. Этот атрибут может быть многозначным, в этом случае создается несколько ссылок.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Dns-Root                                    |
| LDAP-отображаемое имя | днсрут                                     |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.28                       |
| System-ID — GUID    | bf967959-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
