# <a name="rdn-attribute"></a>Атрибут RDN

Относительное различающееся имя (RDN) объекта. RDN — это относительная часть различающегося имени (DN), однозначно определяющая объект LDAP.



| Ввод | Значение |
|-------------------|------------------------------------------------|
| CN                | RDN                                            |
| LDAP-отображаемое имя | name                                           |
| Размер              | \-                                             |
| Привилегия обновления  | Это значение задается администратором схемы. |
| Частота обновления  | \-                                             |
| Attribute-Id      | 1.2.840.113556.1.4.1                           |
| System-ID — GUID    | bf967a0e-0de6-11d0-a285-00aa003049e2           |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)    |
