# <a name="netboot-tools-attribute"></a>атрибут нетбут-Tools

Атрибут **нетбут-Tools** зарезервирован для внутреннего использования.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | нетбут — средства                               |
| LDAP-отображаемое имя | нетбуттулс                                |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.858                      |
| System-ID — GUID    | 0738307f-91df-11d1-aebc-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
