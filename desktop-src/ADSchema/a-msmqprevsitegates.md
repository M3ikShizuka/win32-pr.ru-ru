|-------------------|-----------------------------------------|
| CN                | MSMQ-prev-site-Gates                    |
| LDAP-отображаемое имя | мсмкпревситегатес                       |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1225                 |
| System-ID — GUID    | 2df90d75-009f-11d2-aa4c-00c04fd7d83a    |
