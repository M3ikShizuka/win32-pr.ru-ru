# <a name="package-name-attribute"></a>Атрибут Package-Name

Этот атрибут содержит имя развертывания для приложения.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Package-Name                                |
| LDAP-отображаемое имя | packageName,                                 |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.326                      |
| System-ID — GUID    | 7d6c0e98-7e20-11d0-afd6-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
