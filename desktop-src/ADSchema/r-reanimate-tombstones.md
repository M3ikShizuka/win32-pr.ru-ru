# <a name="reanimate-tombstones-extended-right"></a>Reanimate-Tombstones расширенное право

Право на управление доступом, позволяющее восстанавливать удаленные элементы схемы.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Reanimate-Tombstones                 |
| Display-Name | Восстановить отметки полного удаления                 |
| Rights-GUID  | 45ec5156-db7e-47bb-b53f-dbeb2d03c40f |
