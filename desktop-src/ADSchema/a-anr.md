# <a name="anr-attribute"></a>Атрибут ANR

Неоднозначный атрибут разрешения имени, используемый при выборе между объектами.



| Ввод | Значение |
|-------------------|-------------------------------------------------------------------|
| CN                | ANR                                                               |
| LDAP-отображаемое имя | aNR                                                               |
| Размер              | \-                                                                |
| Привилегия обновления  | Администратор схемы                                              |
| Частота обновления  | При необходимости добавления или удаления атрибута из списка ANR. |
| Attribute-Id      | 1.2.840.113556.1.4.1208                                           |
| System-ID — GUID    | 45b01500-c419-11d1-bbc9-0080c76670c0                              |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                       |
