# <a name="frs-service-command-attribute"></a>Атрибут службы FRS-Service-Command

Строка в Юникоде, которую администратор может задать для передачи команды каждому члену набора реплик. Не используется.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Служба FRS-Service-Command                         |
| LDAP-отображаемое имя | фрссервицекомманд                           |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.500                      |
| System-ID — GUID    | ddac0cee-af8f-11d0-afeb-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
