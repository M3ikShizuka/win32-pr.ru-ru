# <a name="msrassavedcallbacknumber-attribute"></a>атрибут Мсрассаведкаллбаккнумбер

Атрибут [**мсрассаведкаллбаккнумбер**](a-msradiusservicetype.md) используется внутренним образом. Не изменяйте это значение напрямую.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | мсрассаведкаллбаккнумбер             |
| LDAP-отображаемое имя | мсрассаведкаллбаккнумбер             |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1189              |
| System-ID — GUID    | db0c90c5-c1f2-11d1-bbc5-0080c76670c0 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
