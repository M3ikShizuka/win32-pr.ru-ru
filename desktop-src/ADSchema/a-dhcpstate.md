# <a name="dhcp-state-attribute"></a>атрибут состояния DHCP

Атрибут **состояния DHCP** в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | DHCP-состояние                           |
| LDAP-отображаемое имя | дхкпстате                            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.717               |
| System-ID — GUID    | 963d2752-48be-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
