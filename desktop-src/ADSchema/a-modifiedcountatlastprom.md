# <a name="modified-count-at-last-prom-attribute"></a>Атрибут "Modified-on-Last-Пром"

Серийный номер журнала изменений сетевого входа в систему при последнем повышении уровня.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Изменено — количество-за последние — Пром          |
| LDAP-отображаемое имя | модифиедкаунтатластпром              |
| Размер              | \-                                   |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.81                |
| System-ID — GUID    | bf9679c6-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
