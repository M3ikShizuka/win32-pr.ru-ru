# <a name="superior-dns-root-attribute"></a>Атрибут «старший-DNS-Root»

Это системный атрибут, используемый для создания ссылок.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Старший DNS-корневой узел                           |
| LDAP-отображаемое имя | супериорднсрут                             |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.532                      |
| System-ID — GUID    | 5245801d-ca6a-11d0-afff-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
