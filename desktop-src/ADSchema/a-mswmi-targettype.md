# <a name="ms-wmi-targettype-attribute"></a>атрибут MS-WMI-TargetType

Ссылка WMI на определение типа для объекта политики.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-TargetType                           |
| LDAP-отображаемое имя | Мсвми-TargetType                            |
| Размер              | Менее 100 символов.                   |
| Привилегия обновления  | Администратор групповая политика                  |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1649                     |
| System-ID — GUID    | ca2a281e-262b-4ff7-b419-bc123352a4e9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
