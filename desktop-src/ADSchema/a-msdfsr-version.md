# <a name="ms-dfsr-version-attribute"></a>атрибут MS-DFSR-Version

Содержит номер версии службы репликации распределенная файловая система (DFS).



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DFSR-Version                             |
| LDAP-отображаемое имя | Мсдфср — версия                              |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.1                   |
| System-ID — GUID    | 1a861408-38c3-49ea-ba75-85481a77c655        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
