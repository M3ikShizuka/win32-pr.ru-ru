# <a name="houseidentifier-attribute"></a>атрибут Хаусеидентифиер

Указывает лингвистическую конструкцию, используемую для указания конкретного здания, например номер дома или название дома относительно улицы, места, города, городов и т. д.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | хаусеидентифиер                             |
| LDAP-отображаемое имя | хаусеидентифиер                             |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 2.5.4.51                                    |
| System-ID — GUID    | a45398b7-c44a-4eb6-82d3-13c10946dbfe        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
