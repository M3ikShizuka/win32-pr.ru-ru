# <a name="ms-authz-resource-condition-attribute"></a>атрибут MS-authz-Resource-Condition

| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2153                     |
| System-ID — GUID    | 80997877-f874-4c68-864d-6e508a83bdbd        |
