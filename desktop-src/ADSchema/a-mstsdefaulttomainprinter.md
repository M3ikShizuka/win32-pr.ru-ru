# <a name="ms-ts-default-to-main-printer-attribute"></a>атрибут MS-TS-Default-to-Main-Printer

Для служб терминалов по умолчанию используется основной принтер. указывает, следует ли автоматически печатать на принтере клиента по умолчанию.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-TS-по умолчанию — принтер        |
| LDAP-отображаемое имя | мстсдефаулттомаинпринтер             |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1988              |
| System-ID — GUID    | c0ffe2bd-cacf-4dc7-88d5-61e9e95766f6 |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
