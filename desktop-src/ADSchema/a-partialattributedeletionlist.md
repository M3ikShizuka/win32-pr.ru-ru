# <a name="partial-attribute-deletion-list-attribute"></a>Атрибут разделяемого атрибута-удаления списка

Отслеживает состояние внутренней репликации частичных реплик (то есть в глобальных каталогах). Атрибут объекта NC частичной реплики. Используется, когда сборщик мусора находится в процессе удаления атрибутов из объектов в контекстах именования частичной реплики.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Разделяемый атрибут-удаление-список                       |
| LDAP-отображаемое имя | партиалаттрибутеделетионлист                          |
| Размер              | \-                                                    |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | Во время репликации                                    |
| Attribute-Id      | 1.2.840.113556.1.4.663                                |
| System-ID — GUID    | 28630ec0-41d5-11d1-a9c1-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
