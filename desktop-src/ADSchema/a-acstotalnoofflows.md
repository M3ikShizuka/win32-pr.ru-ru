|-------------------|--------------------------------------|
| CN                | ACS-Total — нет потоков                |
| LDAP-отображаемое имя | акстоталнуффловс                    |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.763               |
| System-ID — GUID    | 7f561280-5301-11d1-a9c5-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
