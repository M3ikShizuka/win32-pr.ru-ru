# <a name="generate-rsop-logging-extended-right"></a>Generate-RSoP — ведение журнала расширено справа

Пользователь, имеющий права в подразделении или домене, сможет создавать данные RSoP в режиме ведения журнала для пользователей или компьютеров в подразделении.



| Ввод | Значение |
|--------------|--------------------------------------------|
| CN           | Generate-RSoP — ведение журнала                      |
| Display-Name | Генерация RSoP (Протоколирование) |
| Rights-GUID  | b7b1b3de-ab09-4242-9e30-9980e5d322f7       |
