# <a name="sam-domain-updates-attribute"></a>Атрибут SAM-Domain-Updates

Содержит битовую маску для выполнения операций SAM на Active Directory.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | SAM-обновление домена                                    |
| LDAP-отображаемое имя | самдомаинупдатес                                      |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1969                               |
| System-ID — GUID    | 04d2d114-f799-4e9b-bcdc-90e8f5ba7ebe                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
