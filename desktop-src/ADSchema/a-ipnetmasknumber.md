# <a name="ipnetmasknumber-attribute"></a>атрибут Ипнетмаскнумбер

Содержит IP-маску с точками в формате десятичной запятой, в которой опущены начальные нули.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ипнетмаскнумбер                      |
| LDAP-отображаемое имя | ипнетмаскнумбер                      |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.21                     |
| System-ID — GUID    | 6ff64fcd-462e-4f62-b44a-9a5347659eb9 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
