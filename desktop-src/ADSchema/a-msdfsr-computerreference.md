# <a name="ms-dfsr-computerreference-attribute"></a>атрибут MS-DFSR-Компутерреференце

Содержит прямую ссылку на объект [**Computer**](c-computer.md) .



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DFSR-Компутерреференце               |
| LDAP-отображаемое имя | Мсдфср — Компутерреференце                |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.101             |
| System-ID — GUID    | 6c7b5785-3d21-41bf-8a8a-627941544d5a    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
