|-------------------|-------------------------------------------------------|

| LDAP-отображаемое имя | паренткацертификатечаин                              |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.685                                |
| System-ID — GUID    | 963d2733-48be-11d1-a9c3-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
