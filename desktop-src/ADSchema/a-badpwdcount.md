# <a name="bad-pwd-count-attribute"></a>Недопустимый атрибут-PWD-Count

Число попыток входа пользователя в учетную запись с помощью неправильного пароля. Значение 0 указывает, что значение неизвестно.



| Ввод | Значение |
|-------------------|-------------------------------------------|
| CN                | Bad-PWD-число                             |
| LDAP-отображаемое имя | badPwdCount                               |
| Размер              | 4 байта                                   |
| Привилегия обновления  | Это значение задается системой.          |
| Частота обновления  | Каждый раз, когда пользователь вводит неправильный пароль. |
| Attribute-Id      | 1.2.840.113556.1.4.12                     |
| System-ID — GUID    | bf96792e-0de6-11d0-a285-00aa003049e2      |
| Синтаксис            | [**Перечисление**](s-enumeration.md)      |

## <a name="remarks"></a>Комментарии

Этот атрибут не реплицируется и хранится отдельно на каждом контроллере домена в домене.

Этот атрибут сбрасывается на конкретном контроллере домена, когда пользователь успешно входит в систему на этом контроллере домена.
