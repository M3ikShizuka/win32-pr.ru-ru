# <a name="msnpcalledstationid-attribute"></a>атрибут Мснпкалледстатионид

Атрибут **мснпкалледстатионид** используется внутренним образом. Не изменяйте это значение напрямую.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | мснпкалледстатионид                  |
| LDAP-отображаемое имя | мснпкалледстатионид                  |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1123              |
| System-ID — GUID    | db0c9089-c1f2-11d1-bbc5-0080c76670c0 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
