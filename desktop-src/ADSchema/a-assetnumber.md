# <a name="asset-number-attribute"></a>Атрибут Asset-Number

Номер отслеживания для объекта.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Asset-Number                                |
| LDAP-отображаемое имя | assetNumber                                 |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.283                      |
| System-ID — GUID    | ba305f75-47e3-11d0-a1a6-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
