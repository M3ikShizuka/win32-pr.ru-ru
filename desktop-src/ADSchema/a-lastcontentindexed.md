# <a name="last-content-indexed-attribute"></a>Атрибут Last-Content-index

Время последнего индексированного содержимого этого тома.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Последний — индексируемый по содержимому                 |
| LDAP-отображаемое имя | ластконтентиндексед                   |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.50                |
| System-ID — GUID    | bf967995-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
