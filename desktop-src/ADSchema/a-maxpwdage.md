# <a name="max-pwd-age-attribute"></a>Атрибут max-PWD-Age

Максимальное количество времени (в 100-наносекундных интервалах), допустимое для пароля. Это значение хранится в виде большого целого числа, представляющего число 100-наносекундных интервалов с момента, когда пароль был задан до истечения срока действия пароля.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Max-PWD-Age                          |
| LDAP-отображаемое имя | макспвдаже                            |
| Размер              | \-                                   |
| Привилегия обновления  | Администратор домена                 |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.74                |
| System-ID — GUID    | bf9679bb-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
