# <a name="locality-name-attribute"></a>Атрибут Locality-Name

Представляет имя локализации, например населенный или город.



| Ввод | Значение |
|-------------------|------------------------------------------------------------------------------|
| CN                | Locality-Name                                                                |
| LDAP-отображаемое имя | l                                                                            |
| Размер              | \-                                                                           |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                       |
| Частота обновления  | Когда создается запись пользователя и каждый раз, когда необходимо изменить локальность. |
| Attribute-Id      | 2.5.4.7                                                                      |
| System-ID — GUID    | bf9679a2-0de6-11d0-a285-00aa003049e2                                         |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                  |
