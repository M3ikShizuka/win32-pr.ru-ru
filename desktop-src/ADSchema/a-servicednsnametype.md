# <a name="service-dns-name-type-attribute"></a>Атрибут Service-DNS-Name-Type

Тип записи DNS для поиска в этой службе. Например, A или SRV.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Service-DNS-Name-Type                       |
| LDAP-отображаемое имя | сервицеднснаметипе                          |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.659                      |
| System-ID — GUID    | 28630eba-41d5-11d1-a9c1-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
