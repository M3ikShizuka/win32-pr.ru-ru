# <a name="domain-policy-object-attribute"></a>Домен-политика-атрибут объекта

Ссылка на объект политики, определяющий политику локального центра безопасности для домена узла.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Домен — политика-объект                    |
| LDAP-отображаемое имя | домаинполициобжект                      |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.32                   |
| System-ID — GUID    | bf96795d-0de6-11d0-a285-00aa003049e2    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
