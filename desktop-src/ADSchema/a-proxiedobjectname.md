# <a name="proxied-object-name-attribute"></a>Атрибут прокси — object-name

Этот атрибут используется внутренне Active Directory для наблюдения за перемещением между доменами.



| Ввод | Значение |
|-------------------|-------------------------------------------------|
| CN                | Прокси-объект-имя                             |
| LDAP-отображаемое имя | проксиедобжектнаме                               |
| Размер              | \-                                              |
| Привилегия обновления  | Используется системой.                     |
| Частота обновления  | \-                                              |
| Attribute-Id      | 1.2.840.113556.1.4.1249                         |
| System-ID — GUID    | e1aea402-cd5b-11d0-afff-0000f80367c1            |
| Синтаксис            | [**Object(двоичный_DN)**](s-object-dn-binary.md) |
