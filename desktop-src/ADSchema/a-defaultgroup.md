# <a name="default-group-attribute"></a>Атрибут Default-Group

Группа, которой назначен этот объект при его создании.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Default-Group                           |
| LDAP-отображаемое имя | дефаултграуп                            |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.480                  |
| System-ID — GUID    | 720bc4e2-a54a-11d0-afdf-00c04fd930c9    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
