# <a name="ms-ds-nc-ro-replica-locations-attribute"></a>Атрибут ms-DS-NC-RO-Replica-Locations

Связанный атрибут для объекта перекрестной ссылки для секции. Выводит список контроллеров домена, которые должны размещать секцию в режиме только для чтения.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-NC-RO-Replica-Locations           |
| LDAP-отображаемое имя | msDS-NC-RO-Replica-Locations            |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1967                 |
| System-ID — GUID    | 3df793df-9858-4417-A701-735a1ecebf74    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
