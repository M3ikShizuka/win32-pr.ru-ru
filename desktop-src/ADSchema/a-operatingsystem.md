# <a name="operating-system-attribute"></a>Атрибут Operating-System

Имя операционной системы, например Windows Vista Enterprise.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Operating-System                            |
| LDAP-отображаемое имя | operatingSystem                             |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.363                      |
| System-ID — GUID    | 3e978925-8c01-11d0-afda-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
