# <a name="acs-minimum-latency-attribute"></a>ACS — атрибут минимальной задержки

Атрибут **ACS-Minimum-задержка** предназначен только для внутреннего использования. На основе RFC2210.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS — Минимальная задержка                  |
| LDAP-отображаемое имя | аксминимумлатенци                    |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1316              |
| System-ID — GUID    | 9517fefb-3b90-11d2-90cc-00c04fd91ab1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
