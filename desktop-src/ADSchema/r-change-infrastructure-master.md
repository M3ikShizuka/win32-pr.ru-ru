# <a name="change-infrastructure-master-extended-right"></a>Изменение — инфраструктура-основное расширенное право

Расширенное право, необходимое для изменения владельца роли FSMO инфраструктуры.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Изменение — инфраструктура — главный сервер         |
| Display-Name | Изменение хозяина инфраструктуры         |
| Rights-GUID  | cc17b1fb-33d9-11d2-97d4-00c04fd8d5cd |
