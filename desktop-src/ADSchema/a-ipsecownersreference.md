# <a name="ipsec-owners-reference-attribute"></a>Атрибут ссылки IPSec-Owners

Атрибут **IPSec-Owners-Reference** предназначен только для внутреннего использования.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Справочник по IPsec-владельцам                  |
| LDAP-отображаемое имя | ипсековнерсреференце                    |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.624                  |
| System-ID — GUID    | b40ff824-427a-11d1-a9c2-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
