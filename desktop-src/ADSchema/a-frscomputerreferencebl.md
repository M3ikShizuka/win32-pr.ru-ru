# <a name="frs-computer-reference-bl-attribute"></a>Атрибут FRS-Computer-Reference-BL

Ссылка на наборы реплик, к которым принадлежит этот компьютер.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | FRS-Computer-Reference-BL               |
| LDAP-отображаемое имя | фрскомпутерреференцебл                  |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.870                  |
| System-ID — GUID    | 2a132579-9373-11d1-aebc-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
