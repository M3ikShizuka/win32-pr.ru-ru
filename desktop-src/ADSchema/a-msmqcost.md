| CN                | MSMQ-Cost                            |
| LDAP-отображаемое имя | мсмккост                             |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.946               |
| System-ID — GUID    | 9a0dc33a-c100-11d1-bbc5-0080c76670c0 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
