# <a name="pending-parent-ca-attribute"></a>Атрибут ожидающего наследования CA

Ссылка на центры сертификации, которые выдавали ожидающие сертификаты для данного центра сертификации.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Ожидающий — родительский ЦС                       |
| LDAP-отображаемое имя | пендингпарентка                         |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.695                  |
| System-ID — GUID    | 963d273e-48be-11d1-a9c3-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
