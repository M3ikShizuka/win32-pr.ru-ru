# <a name="phone-isdn-primary-attribute"></a>Телефон-ISDN-первичный атрибут

Основной ISDN.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Телефон-ISDN-Primary                                                               |
| LDAP-отображаемое имя | примаринтернатионалисдннумбер                                                   |
| Размер              | \-                                                                               |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                           |
| Частота обновления  | При создании записи пользователя и при необходимости изменения номера телефона. |
| Attribute-Id      | 1.2.840.113556.1.4.649                                                           |
| System-ID — GUID    | 0296c11f-40da-11d1-a9c0-0000f80367c1                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
