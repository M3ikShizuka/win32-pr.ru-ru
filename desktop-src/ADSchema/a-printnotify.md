# <a name="print-notify-attribute"></a>Атрибут Print-Notify

Предоставляемая пользователем строка, указывающая Контактное лицо для уведомления.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Print-Notify                                |
| LDAP-отображаемое имя | printNotify                                 |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.272                      |
| System-ID — GUID    | ba305f6a-47e3-11d0-a1a6-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
