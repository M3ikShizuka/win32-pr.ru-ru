Синтаксис перечисления (предпочтительный метод доставки).



| Ввод | Значение |
|--------------|---------------------------------------------------------------------------|
| Имя         | Перечисление (предпочтительный метод-Delivery)                                    |
| Идентификатор синтаксиса    | 2.5.5.9                                                                   |
| ИДЕНТИФИКАТОР OM        | 10                                                                        |
| Тип MAPI    | LONG                                                                      |
| Тип ADS     | АДСТИПЕ \_ целое число                                                          |
| Тип варианта | VT \_ I4                                                                    |
| Тип SDS     | [System.Int32](/dotnet/api/system.int32) |
