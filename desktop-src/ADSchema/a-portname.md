# <a name="port-name-attribute"></a>Атрибут Port-Name

Список имен портов. Например, для портов принтеров или последовательных портов.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Port-Name                                   |
| LDAP-отображаемое имя | portName                                    |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.228                      |
| System-ID — GUID    | 281416c4-1968-11d0-a28f-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
