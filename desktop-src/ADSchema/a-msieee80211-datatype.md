# <a name="ms-ieee-80211-data-type-attribute"></a>MS-IEEE-80211-атрибут типа данных

Внутренний используемый тип данных для msieee80211 BLOB-объекта данных.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | MS-IEEE-80211-тип данных                                                          |
| LDAP-отображаемое имя | msieee80211 — тип данных                                                             |
| Размер              | \-                                                                               |
| Привилегия обновления  | Администратор домена                                                             |
| Частота обновления  | Каждый раз, когда администратор домена изменяет политику беспроводной сети для домена или подразделения. |
| Attribute-Id      | 1.2.840.113556.1.4.1822                                                          |
| System-ID — GUID    | 6558b180-35da-4efe-beed-521f8f48cafb                                             |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                                             |
