# <a name="ldap-admin-limits-attribute"></a>Атрибут LDAP-Admin-limits

Содержит набор пар «атрибут-значение», определяющих административные ограничения сервера LDAP.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | LDAP-Admin-limits                           |
| LDAP-отображаемое имя | лдападминлимитс                             |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.843                      |
| System-ID — GUID    | 7359a352-90f7-11d1-aebc-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
