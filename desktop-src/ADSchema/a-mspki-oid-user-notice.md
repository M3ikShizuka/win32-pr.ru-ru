# <a name="ms-pki-oid-user-notice-attribute"></a>атрибут MS-PKI-OID-User-Notice

Уведомление пользователя для OID политики издателя предприятия.



| Ввод | Значение |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-OID-пользователь — уведомление                                                                     |
| LDAP-отображаемое имя | Мспки-OID-пользователь-уведомление                                                                      |
| Размер              | \-                                                                                         |
| Привилегия обновления  | Администратор предприятия                                                                   |
| Частота обновления  | При создании нового шаблона или изменении атрибутов существующих шаблонов. |
| Attribute-Id      | 1.2.840.113556.1.4.1673                                                                    |
| System-ID — GUID    | 04c4da7a-e114-4e69-88de-e293f2d3b395                                                       |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                                |
