# <a name="dns-allow-xfr-attribute"></a>Атрибут DNS-Allow-КСФР

Атрибут **DNS-Allow-ксфр** в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | DNS-Allow-КСФР                        |
| LDAP-отображаемое имя | днсалловксфр                          |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.379               |
| System-ID — GUID    | e0fa1e66-9b45-11d0-afdd-00c04fd930c9 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
