Содержит суффикс имени очереди (если таковой имеется).



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MSMQ-Queue-Name-ext                         |
| LDAP-отображаемое имя | мсмккуеуенамикст                            |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1243                     |
| System-ID — GUID    | 2df90d87-009f-11d2-aa4c-00c04fd7d83a        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
