# <a name="dhcp-mask-attribute"></a>атрибут маски DHCP

Атрибут **маски DHCP** в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | DHCP-маска                            |
| LDAP-отображаемое имя | дхкпмаск                             |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.706               |
| System-ID — GUID    | 963d2747-48be-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
