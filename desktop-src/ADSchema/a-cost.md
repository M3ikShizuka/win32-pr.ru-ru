# <a name="cost-attribute"></a>Атрибут стоимости

Содержит относительную стоимость маршрутизации сообщений через конкретный соединитель сайта.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Стоимость                                 |
| LDAP-отображаемое имя | cost                                 |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.135               |
| System-ID — GUID    | bf967944-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
