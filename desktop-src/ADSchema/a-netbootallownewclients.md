# <a name="netboot-allow-new-clients-attribute"></a>атрибут нетбут-Allow-New-Clients

Атрибут **нетбут-Allow-New-Clients** зарезервирован для внутреннего использования.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | нетбут-Allow-New-Clients            |
| LDAP-отображаемое имя | нетбуталловневклиентс               |
| Размер              | \-                                   |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.849               |
| System-ID — GUID    | 07383076-91df-11d1-aebc-0000f80367c1 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
