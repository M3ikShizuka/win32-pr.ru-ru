# <a name="supported-application-context-attribute"></a>Поддерживается — атрибут контекста приложения

Указывает идентификаторы объектов контекстов приложений, поддерживаемых приложением OSI.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Поддерживается — контекст приложения                         |
| LDAP-отображаемое имя | суппортедаппликатионконтекст                           |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 2.5.4.30                                              |
| System-ID — GUID    | 1677588f-47f3-11d1-a9c3-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
