# <a name="gpc-user-extension-names-attribute"></a>Атрибут ГПК-User-Extension-Names

Используется объектом групповая политика для политик пользователя.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------|
| CN                | ГПК-User-Extensions-Names                                                        |
| LDAP-отображаемое имя | гпкусерекстенсионнамес                                                           |
| Размер              | Зависит от числа клиентских расширений, имеющих политику в этом объекте групповой политики. |
| Привилегия обновления  | Администратор домена или политики.                                                 |
| Частота обновления  | Каждый раз при обновлении объекта групповой политики с помощью ГПЕ.                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1349                                                         |
| System-ID — GUID    | 42a75fc6-783f-11d2-9916-0000f87a57d4                                            |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                     |
