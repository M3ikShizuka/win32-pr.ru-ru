| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-TS-Endpoint-подключаемый модуль                       |
| LDAP-отображаемое имя | мстсендпоинтплугин                          |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2072                     |
| System-ID — GUID    | 3c08b569-801F-4158-b17b-e363d6ae696a        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
