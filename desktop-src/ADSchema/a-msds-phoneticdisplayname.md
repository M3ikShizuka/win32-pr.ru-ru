# <a name="ms-ds-phonetic-display-name-attribute"></a>Атрибут ms-DS-фонетического отображаемого имени

Фонетическое отображаемое имя объекта. В случае отсутствия фонетического отображаемого имени используется существующее отображаемое имя.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-фонетическо-отображаемое имя                 |
| LDAP-отображаемое имя | msDS-PhoneticDisplayName                    |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1946                     |
| System-ID — GUID    | e21a94e4-2d66-4ce5-b30d-0ef87a776ff0        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
