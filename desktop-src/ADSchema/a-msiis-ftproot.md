# <a name="ms-iis-ftp-root-attribute"></a>атрибут MS-IIS-FTP-root

Этот атрибут определяет общую папку файлового сервера. Он используется совместно с MS-IIS-FTP-dir для определения домашнего каталога пользователя FTP.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------------------------------------------------------|
| CN                | MS-IIS-FTP-root                                                                                                                 |
| LDAP-отображаемое имя | Мсиис — Фтпрут                                                                                                                   |
| Размер              | 30-50 символов (15-25 символов Юникода для каждого свойства)                                                                   |
| Привилегия обновления  | Администратор домена & локальная система                                                                                             |
| Частота обновления  | Это свойство задается, когда администратор создает веб-сайт и настраивает изоляцию FTP. Это редко меняется после этого. |
| Attribute-Id      | 1.2.840.113556.1.4.1785                                                                                                         |
| System-ID — GUID    | 2a7827a4-1483-49a5-9d84-52e3812156b4                                                                                            |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                                                                     |
