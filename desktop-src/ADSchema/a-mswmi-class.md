# <a name="ms-wmi-class-attribute"></a>атрибут класса MS-WMI-Class

Имя объекта класса WMI в связанной кодировке (например, Win32 \_ ComputerSystem). В качестве кодировки может использоваться экземпляр или объект класса.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------|
| CN                | Класс MS-WMI-                                                    |
| LDAP-отображаемое имя | Класс Мсвми                                                     |
| Размер              | \-                                                              |
| Привилегия обновления  | Администратор домена                                            |
| Частота обновления  | Каждый раз при добавлении или изменении класса, содержащего атрибут. |
| Attribute-Id      | 1.2.840.113556.1.4.1676                                         |
| System-ID — GUID    | 90c1925f-4a24-4b07-b202-be32eb3c8b74                            |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                     |
