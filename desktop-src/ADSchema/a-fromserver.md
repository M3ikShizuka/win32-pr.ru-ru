# <a name="from-server-attribute"></a>Атрибут From-Server

Различающееся имя исходного сервера репликации.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | From-Server                             |
| LDAP-отображаемое имя | фромсервер                              |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.40                   |
| System-ID — GUID    | bf967979-0de6-11d0-a285-00aa003049e2    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
