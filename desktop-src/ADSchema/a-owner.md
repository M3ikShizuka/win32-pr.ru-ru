# <a name="owner-attribute"></a>Атрибут Owner

Различающееся имя объекта, который владеет объектом.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Владелец                                   |
| LDAP-отображаемое имя | владелец                                   |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 2.5.4.32                                |
| System-ID — GUID    | bf9679f3-0de6-11d0-a285-00aa003049e2    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
