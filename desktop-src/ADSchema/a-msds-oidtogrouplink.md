# <a name="ms-ds-oidtogroup-link-attribute"></a>Атрибут ms-DS-Оидтограуп-Link

Для OID определяет объект группы, соответствующий политике выдачи, представленной этим идентификатором объекта.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-Оидтограуп-Link                   |
| LDAP-отображаемое имя | msDS-Оидтограуплинк                     |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2051                 |
| System-ID — GUID    | f9c9a57c-3941-438d-bebf-0edaf2aca187    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
