# <a name="proxy-addresses-attribute"></a>Атрибут Proxy-Addresses

адрес прокси-сервера — это адрес, по которому объект получателя Microsoft Exchange Server распознается в внешней почтовой системе. Адреса прокси-сервера необходимы для всех объектов получателей, таких как пользовательские получатели и списки рассылки.



| Ввод | Значение |
|-------------------|------------------------------------------------------------------------------------------------------|
| CN                | Proxy-Addresses                                                                                      |
| LDAP-отображаемое имя | proxyAddresses                                                                                       |
| Размер              | \-                                                                                                   |
| Привилегия обновления  | Это значение задается системой.                                                                     |
| Частота обновления  | Создается библиотекой DLL, предоставляемой объектом Address-Type Directory при установке типа адреса. |
| Attribute-Id      | 1.2.840.113556.1.2.210                                                                               |
| System-ID — GUID    | bf967a06-0de6-11d0-a285-00aa003049e2                                                                 |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                                          |
