# <a name="ms-ts-allow-logon-attribute"></a>атрибут MS-TS-Allow-logon

Указывает, разрешено ли пользователю входить на сервер терминалов. Значение равно 1, если вход разрешен, и 0, если вход запрещен.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-TS-Allow-logon                    |
| LDAP-отображаемое имя | мстсалловлогон                       |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1979              |
| System-ID — GUID    | 3a0cd464-bc54-40e7-93ae-a646a6ecc4b4 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
