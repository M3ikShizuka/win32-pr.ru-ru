# <a name="om-object-class-attribute"></a>OM-атрибут класса Object

Уникальный идентификатор объекта для атрибута или класса.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | OM-Object-Class                                       |
| LDAP-отображаемое имя | омобжекткласс                                         |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.218                                |
| System-ID — GUID    | bf9679ec-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
