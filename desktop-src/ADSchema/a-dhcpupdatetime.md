# <a name="dhcp-update-time-attribute"></a>DHCP-время обновления-атрибут

Атрибут " **DHCP-время обновления** " в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | DHCP-время обновления                     |
| LDAP-отображаемое имя | дхкпупдатетиме                       |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.720               |
| System-ID — GUID    | 963d2755-48be-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
