|-------------------|---------------------------------------------|
| CN                | Привязка службы — сведения                 |
| LDAP-отображаемое имя | сервицебиндингинформатион                   |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.510                      |
| System-ID — GUID    | b7b1311c-b82e-11d0-afee-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
