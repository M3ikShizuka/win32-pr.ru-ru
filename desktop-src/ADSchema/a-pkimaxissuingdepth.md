# <a name="pki-max-issuing-depth-attribute"></a>PKI-Max-атрибут глубины

Максимальная длина цепочки сертификатов, выданной сертификатом.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | PKI-max-глубина выдачи                |
| LDAP-отображаемое имя | пкимаксиссуингдепс                   |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1329              |
| System-ID — GUID    | f0bfdefa-3b9d-11d2-90cc-00c04fd91ab1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
