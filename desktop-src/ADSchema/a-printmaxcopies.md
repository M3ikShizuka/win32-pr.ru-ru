# <a name="print-max-copies-attribute"></a>Print — атрибут max-Копирующийs

Максимальное число копий, которое может печатать устройство.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Print-Max-копии                     |
| LDAP-отображаемое имя | принтмакскопиес                       |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.241               |
| System-ID — GUID    | 281416d1-1968-11d0-a28f-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
