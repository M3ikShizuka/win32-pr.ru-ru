# <a name="ds-ui-admin-notification-attribute"></a>Атрибут DS-UI-Admin-Notification

Это список идентификаторов GUID COM-объектов, поддерживающих интерфейс обратного вызова, Дсадмин вызовы при возникновении действия над объектом через пользовательский интерфейс.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | DS-UI-Admin — уведомление                    |
| LDAP-отображаемое имя | дсуиадминнотификатион                       |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1343                     |
| System-ID — GUID    | f6ea0a94-6f91-11d2-9905-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
