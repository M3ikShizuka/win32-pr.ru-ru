| CN                | Root-Trust                              |
| LDAP-отображаемое имя | руттруст                               |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.674                  |
| System-ID — GUID    | 7bfdcb80-4807-11d1-a9c3-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
