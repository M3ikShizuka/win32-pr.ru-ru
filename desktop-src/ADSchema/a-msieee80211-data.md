# <a name="ms-ieee-80211-data-attribute"></a>MS-IEEE-80211-атрибут данных

Содержит список предпочитаемых конфигураций сети в групповая политика для беспроводных сетей.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | MS-IEEE-80211-Data                                                               |
| LDAP-отображаемое имя | msieee80211 — данные                                                                 |
| Размер              | \-                                                                               |
| Привилегия обновления  | Администратор домена                                                             |
| Частота обновления  | Каждый раз, когда администратор домена изменяет политику беспроводной сети для домена или подразделения. |
| Attribute-Id      | 1.2.840.113556.1.4.1821                                                          |
| System-ID — GUID    | 0e0d0938-2658-4580-a9f6-7a0ac7b566cb                                             |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md)                            |
