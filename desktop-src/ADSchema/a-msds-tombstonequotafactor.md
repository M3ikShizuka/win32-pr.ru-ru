# <a name="ms-ds-tombstone-quota-factor-attribute"></a>Атрибут ms-DS-захоронение — Quota

| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-захоронение-фактор квоты         |
| LDAP-отображаемое имя | msDS-Томбстонекуотафактор            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1847              |
| System-ID — GUID    | 461744d7-f3b6-45ba-8753-fb9552a5df32 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
