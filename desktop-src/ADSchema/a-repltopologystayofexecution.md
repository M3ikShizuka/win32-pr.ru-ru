# <a name="repl-topology-stay-of-execution-attribute"></a>REPL-топология-сохранение атрибута выполнения

Задержка между удалением объекта сервера и его окончательной удалением из топологии репликации.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | REPL-топология-продолжение выполнения      |
| LDAP-отображаемое имя | реплтопологистайофексекутион          |
| Размер              | 4 байта                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | При каждом удалении объекта сервера. |
| Attribute-Id      | 1.2.840.113556.1.4.677               |
| System-ID — GUID    | 7bfdcb83-4807-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
