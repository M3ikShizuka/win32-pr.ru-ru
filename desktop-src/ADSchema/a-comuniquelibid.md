# <a name="com-unique-libid-attribute"></a>Атрибут COM-Unique-LIBID

Однозначный строковый идентификатор LIBID для библиотеки типов.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | COM-Unique-LIBID                            |
| LDAP-отображаемое имя | комуникуелибид                              |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.250                      |
| System-ID — GUID    | 281416da-1968-11d0-a28f-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
