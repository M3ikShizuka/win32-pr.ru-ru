# <a name="app-schema-version-attribute"></a>Атрибут App-Schema-Version

Этот атрибут хранит версию схемы хранилища классов. Он используется для обеспечения правильного поведения в изменениях схемы.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Версия App-Schema-Version                   |
| LDAP-отображаемое имя | аппсчемаверсион                     |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.848               |
| System-ID — GUID    | 96a7dd65-9118-11d1-aebc-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
