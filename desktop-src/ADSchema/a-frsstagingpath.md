# <a name="frs-staging-path-attribute"></a>Атрибут «промежуточный путь FRS»

Путь к промежуточной области репликации файлов.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Каталог FRS-промежуточного хранения                            |
| LDAP-отображаемое имя | фрсстагингпас                              |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.488                      |
| System-ID — GUID    | 1be8f175-a9ff-11d0-afe2-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
