# <a name="network-address-attribute"></a>Атрибут Network-Address

TCP/IP-адрес сегмента сети. Также называется адресом подсети.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Network-Address                             |
| LDAP-отображаемое имя | networkAddress                              |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.459                      |
| System-ID — GUID    | bf9679d9-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Teletex)**](s-string-teletex.md) |
