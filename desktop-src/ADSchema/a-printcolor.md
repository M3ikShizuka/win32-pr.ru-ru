# <a name="print-color-attribute"></a>Атрибут Print-Color

**Значение true** , если принтер может печатать в цвете.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Print-Color                          |
| LDAP-отображаемое имя | printColor                           |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.243               |
| System-ID — GUID    | 281416d3-1968-11d0-a28f-00aa003049e2 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
