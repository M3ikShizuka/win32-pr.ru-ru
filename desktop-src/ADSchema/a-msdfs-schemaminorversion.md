# <a name="ms-dfs-schema-minor-version-attribute"></a>атрибут MS-DFS-Schema-Minor-Version

Содержит дополнительный номер версии схемы метаданных DFS.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DFS-Schema-Minor — версия          |
| LDAP-отображаемое имя | Мсдфс — Счемаминорверсион             |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2031              |
| System-ID — GUID    | fef9a725-e8f1-43ab-bd86-6a0115ce9e38 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
