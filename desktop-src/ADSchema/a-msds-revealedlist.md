# <a name="ms-ds-revealed-list-attribute"></a>Атрибут ms-DS-выводит список

Определяет субъекты безопасности, чьи текущие пароли учетных записей компьютеров были реплицированы на RODC.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MS-DS-выявили список                                   |
| LDAP-отображаемое имя | msDS-Ревеаледлист                                     |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1940                               |
| System-ID — GUID    | cbdad11c-7fec-387b-6219-3a0627d9af81                  |
| Синтаксис            | [**Object(точка_доступа)**](s-object-access-point.md) |
