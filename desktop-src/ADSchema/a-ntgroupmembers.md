# <a name="nt-group-members-attribute"></a>Атрибут NT-Group-Members

Этот атрибут не используется.

Атрибут [**member**](a-member.md) используется для хранения членов группы.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | NT-Group-Members                                      |
| LDAP-отображаемое имя | нтграупмемберс                                        |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.89                                 |
| System-ID — GUID    | bf9679df-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
