# <a name="ms-dfsr-rdcminfilesizeinkb-attribute"></a>атрибут MS-DFSR-Рдкминфилесизеинкб

Содержит минимальный размер файла в килобайтах, к которому применяется RDC.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-Рдкминфилесизеинкб           |
| LDAP-отображаемое имя | Мсдфср — Рдкминфилесизеинкб            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.20           |
| System-ID — GUID    | f402a330-ace5-4dc1-8cc9-74d900bf8ae0 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
