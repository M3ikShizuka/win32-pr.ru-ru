|-------------------|---------------------------------------------|
| CN                | COM-other-Prog-ID                           |
| LDAP-отображаемое имя | комосерпрогид                              |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.253                      |
