# <a name="ms-sql-registeredowner-attribute"></a>Атрибут MS-SQL-Регистередовнер

Зарегистрированный владелец текущего экземпляра SQL Server.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Регистередовнер                      |
| LDAP-отображаемое имя | mS-SQL-Регистередовнер                      |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | При установке системы.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1364                     |
| System-ID — GUID    | 48fd44ea-ccee-11d2-9993-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
