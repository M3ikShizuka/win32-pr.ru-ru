# <a name="dns-record-attribute"></a>Атрибут Dns-Record

Используется для хранения двоичных записей DNS-ресурсов в объектах DNS.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Dns-Record                                            |
| LDAP-отображаемое имя | днсрекорд                                             |
| Размер              | \-                                                    |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | При изменении записей ресурсов DNS.                 |
| Attribute-Id      | 1.2.840.113556.1.4.382                                |
| System-ID — GUID    | e0fa1e69-9b45-11d0-afdd-00c04fd930c9                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
