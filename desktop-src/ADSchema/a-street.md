# <a name="street-address-attribute"></a>Атрибут Street-Address

Почтовый адрес.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Street-Address                                                                   |
| LDAP-отображаемое имя | street                                                                           |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 2.5.4.9                                                                          |
| System-ID — GUID    | bf967a3a-0de6-11d0-a285-00aa003049e2                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
