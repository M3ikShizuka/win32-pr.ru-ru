# <a name="dhcp-unique-key-attribute"></a>атрибут уникального ключа DHCP

Атрибут **DHCP-Unique-Key** в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Ключ DHCP-Unique                      |
| LDAP-отображаемое имя | дхкпуникуекэй                        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.698               |
| System-ID — GUID    | 963d273a-48be-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
