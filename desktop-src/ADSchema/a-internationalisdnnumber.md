# <a name="international-isdn-number-attribute"></a>Международный-ISDN-числовой атрибут

Указывает Международный номер ISDN, связанный с объектом.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Международный-ISDN-номер                   |
| LDAP-отображаемое имя | интернатионалисдннумбер                     |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 2.5.4.25                                    |
| System-ID — GUID    | bf96798d-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(числовой)**](s-string-numeric.md) |
