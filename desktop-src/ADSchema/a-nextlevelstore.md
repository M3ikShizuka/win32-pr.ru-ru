# <a name="next-level-store-attribute"></a>Атрибут Store следующего уровня

Этот атрибут указывает на следующее хранилище классов для поиска.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Хранилище следующего уровня                        |
| LDAP-отображаемое имя | некстлевелсторе                          |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.214                  |
| System-ID — GUID    | bf9679da-0de6-11d0-a285-00aa003049e2    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
