# <a name="ms-sql-location-attribute"></a>MS-SQL-атрибут Location

Определяемая пользователем строка. Значение по умолчанию — Location.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-SQL — расположение                             |
| LDAP-отображаемое имя | mS-SQL — расположение                             |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | При установке системы.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1366                     |
| System-ID — GUID    | 561c9644-ccee-11d2-9993-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
