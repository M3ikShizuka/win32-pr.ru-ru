|-------------------|-------------------------------------------------------|
| CN                | Адрес-ввод-отображение-таблица                           |
| LDAP-отображаемое имя | аддрессентридисплайтабле                              |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.324                                |
| System-ID — GUID    | 5fd42461-1262-11d0-a060-00aa006c33ed                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
