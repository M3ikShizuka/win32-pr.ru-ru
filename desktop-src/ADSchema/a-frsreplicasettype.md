# <a name="frs-replica-set-type-attribute"></a>FRS-реплика-набор-тип атрибута

Это код, указывающий, является ли этот набор репликой SYSVOL, набором реплик DFS или другим набором реплик.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | FRS-Replica-Set-Type                 |
| LDAP-отображаемое имя | фрсрепликасеттипе                    |
| Размер              | \-                                   |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.31                |
| System-ID — GUID    | 26d9736b-6070-11d1-a9c6-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
