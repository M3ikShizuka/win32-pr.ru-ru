# <a name="ms-ds-secondary-krbtgt-number-attribute"></a>Атрибут ms-DS-Secondary-KrbTgt-Number

Определяет идентификационный номер протокола, связанного с дополнительным доменом.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-Secondary-KrbTgt-номер        |
| LDAP-отображаемое имя | msDS-Секондарикрбтгтнумбер           |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1929              |
| System-ID — GUID    | aa156612-2396-467e-ad6a-28d23fdb1865 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
