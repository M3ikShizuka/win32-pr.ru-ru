# <a name="well-known-objects-attribute"></a>Атрибут "хорошо известные объекты"

Этот атрибут содержит список хорошо известных контейнеров объектов по идентификатору GUID и различающееся имя. Хорошо известные объекты являются системными контейнерами. Эти сведения используются для получения объекта после перемещения с использованием только идентификатора GUID и имени домена. При перемещении объекта система автоматически обновляет часть различающегося имени в значениях известных объектов, которые ссылаются на объект. Файл NTDSAPI. h содержит следующие определения, которые можно использовать для получения объекта (идентификаторы GUID, связанные с этими объектами, содержатся в NTDSAPI. h):

-   контейнер пользователей с идентификаторами GUID \_ \_ \_ W
-   GUID \_ компутрс \_ контейнер \_ W
-   Идентификатор \_ системы GUID, \_ контейнер \_
-   GUID \_ \_ контроллеров домена, \_ контейнер \_
-   \_контейнер инфраструктуры \_ GUID \_ W
-   GUID \_ удаленных \_ объектов, \_ контейнер \_
-   GUID \_ LOSTANDFOUND \_ контейнера \_ W
-   GUID \_ FOREIGNSECURITYPRINCIPALS \_ контейнер \_ W
-   \_контейнер данных программы для идентификатора GUID \_ \_ \_ W
-   GUID \_ \_ \_ контейнер данных программы \_ Майкрософт \_ W
-   GUID \_ . \_ контейнер квот \_ NTDS \_ W



| Ввод | Значение |
|-------------------|-------------------------------------------------|
| CN                | Хорошо известные объекты                              |
| LDAP-отображаемое имя | веллкновнобжектс                                |
| Размер              | \-                                              |
| Привилегия обновления  | Это значение задается системой.                |
| Частота обновления  | При перемещении одного из системных контейнеров. |
| Attribute-Id      | 1.2.840.113556.1.4.618                          |
| System-ID — GUID    | 05308983-7688-11d1-aded-00c04fd8d5cd            |
| Синтаксис            | [**Object(двоичный_DN)**](s-object-dn-binary.md) |
