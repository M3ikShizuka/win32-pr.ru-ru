# <a name="when-created-attribute"></a>Атрибут When-Created

Дата создания этого объекта. Это значение реплицируется и находится в глобальном каталоге.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------|
| CN                | When-Created                                                  |
| LDAP-отображаемое имя | whenCreated                                                   |
| Размер              | 8 байт                                                       |
| Привилегия обновления  | Это значение задается системой.                              |
| Частота обновления  | При создании объекта.                                   |
| Attribute-Id      | 1.2.840.113556.1.2.2                                          |
| System-ID — GUID    | bf967a78-0de6-11d0-a285-00aa003049e2                          |
| Синтаксис            | [**String(обобщенное_время)**](s-string-generalized-time.md) |
