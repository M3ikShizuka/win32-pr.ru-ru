# <a name="ms-ds-has-instantiated-ncs-attribute"></a>Атрибут ms-DS-с экземпляром-NC

Сведения о состоянии репликации DS, аналогичные (и надмножество) существующих атрибутов Хасмастернкс и Хаспартиалрепликанкс. Используется в KCC при настройке партнеров репликации.



| Ввод | Значение |
|-------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | MS-DS-создан экземпляр — NC                                                                                                                                                                                  |
| LDAP-отображаемое имя | msDS-Хасинстантиатеднкс                                                                                                                                                                                     |
| Размер              | 4 байта                                                                                                                                                                                                     |
| Привилегия обновления  | Это значение задается системой.                                                                                                                                                                            |
| Частота обновления  | Примерно вдвое чаще, чем Хасмастернкс/Хаспартиалрепликанкс. Эти существующие атрибуты обновляются только в том случае, если контроллер домена добавляет или удаляет секцию (например, при переходе с контроллера домена на GC или наоборот). |
| Attribute-Id      | 1.2.840.113556.1.4.1709                                                                                                                                                                                     |
| System-ID — GUID    | 11e9a5bc-4517-4049-af9c-51554fb0fc09                                                                                                                                                                        |
| Синтаксис            | [**Object(двоичный_DN)**](s-object-dn-binary.md)                                                                                                                                                             |
