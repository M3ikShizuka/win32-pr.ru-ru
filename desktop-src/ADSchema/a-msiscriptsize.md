# <a name="msi-script-size-attribute"></a>Атрибут размера MSI-скрипта

Размер файла скрипта установщика Microsoft для этого приложения.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Размер MSI-скрипта                      |
| LDAP-отображаемое имя | мсискриптсизе                        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.846               |
| System-ID — GUID    | 96a7dd63-9118-11d1-aebc-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
