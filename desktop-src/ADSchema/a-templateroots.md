# <a name="template-roots-attribute"></a>Атрибут Template-Roots

Этот атрибут используется в контейнере конфигурации Exchange для указания места хранения контейнеров шаблонов. Эти сведения используются поставщиком Active Directory MAPI.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Template-Roots                          |
| LDAP-отображаемое имя | темплатерутс                           |
| Размер              | \-                                      |
| Привилегия обновления  | Используется системой.             |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1346                 |
| System-ID — GUID    | ed9de9a0-7041-11d2-9905-0000f87a57d4    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
