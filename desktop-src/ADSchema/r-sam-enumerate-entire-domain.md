# <a name="sam-enumerate-entire-domain-extended-right"></a>SAM-перечисление-всего-домен расширенное право

Это право доступа позволяет ограничить круг пользователей, которым разрешено использовать API нижнего уровня, например Неткуеридисплайинформатион и Нетусер/Граупенум, и перечислить весь домен.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | SAM-перечисление — весь домен          |
| Display-Name | Перечисление всего домена SAM          |
| Rights-GUID  | 91d67418-0135-4acc-8d79-c08e857cfbec |
