# <a name="machine-password-change-interval-attribute"></a>Атрибут Machine-Password-Change-Interval

Атрибут **Machine-Password-Change-Interval** в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Компьютер-смена пароля-интервал     |
| LDAP-отображаемое имя | мачинепассвордчанжеинтервал        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.520               |
| System-ID — GUID    | c9b6358e-bb38-11d0-afef-0000f80367c1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
