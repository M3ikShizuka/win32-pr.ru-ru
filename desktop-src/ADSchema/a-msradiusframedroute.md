# <a name="msradiusframedroute-attribute"></a>атрибут Мсрадиусфрамедрауте

Атрибут **мсрадиусфрамедрауте** используется внутренним образом. Не изменяйте это значение напрямую.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | мсрадиусфрамедрауте                  |
| LDAP-отображаемое имя | мсрадиусфрамедрауте                  |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1158              |
| System-ID — GUID    | db0c90a9-c1f2-11d1-bbc5-0080c76670c0 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
