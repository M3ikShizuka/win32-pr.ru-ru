# <a name="documentidentifier-attribute"></a>атрибут Документидентифиер

Тип атрибута Документидентифиер указывает уникальный идентификатор для документа.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | документидентифиер                          |
| LDAP-отображаемое имя | документидентифиер                          |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.11                  |
| System-ID — GUID    | 0b21ce82-ff63-46d9-90fb-c8b9f24e97b9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
