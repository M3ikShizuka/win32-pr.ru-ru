| Ввод | Значение |
|-------------------|----------------------------------------------|
| CN                | MS-DS-Password-обратимое шифрование — включено |
| LDAP-отображаемое имя | msDS-Пассвордреверсиблинкриптионенаблед     |
| Размер              | \-                                           |
| Привилегия обновления  | \-                                           |
| Частота обновления  | \-                                           |
| Attribute-Id      | 1.2.840.113556.1.4.2016                      |
| System-ID — GUID    | 75ccdd8f-af6c-4487-bb4b-69e4d38a959c         |
| Синтаксис            | [**Логическая**](s-boolean.md)                 |
