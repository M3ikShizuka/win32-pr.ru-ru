# <a name="print-end-time-attribute"></a>Print — атрибут времени окончания

Время, когда очередь печати останавливает задания обслуживания.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Печать — время окончания                       |
| LDAP-отображаемое имя | принтендтиме                         |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.234               |
| System-ID — GUID    | 281416ca-1968-11d0-a28f-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
