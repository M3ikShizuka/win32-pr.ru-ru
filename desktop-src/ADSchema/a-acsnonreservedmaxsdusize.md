# <a name="acs-non-reserved-max-sdu-size-attribute"></a>ACS-не зарезервированный-Max-СДУ-size атрибут

Атрибут **ACS-не зарезервирован-Max-СДУ-size** предназначен только для внутреннего использования. На основе RFC2814.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-не зарезервированный-Max-СДУ-size        |
| LDAP-отображаемое имя | акснонресерведмакссдусизе             |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1320              |
| System-ID — GUID    | aec2cfe3-3b90-11d2-90cc-00c04fd91ab1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
