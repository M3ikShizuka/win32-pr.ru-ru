# <a name="tree-name-attribute"></a>Атрибут Tree-Name

| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Tree-Name                                   |
| LDAP-отображаемое имя | тринаме                                    |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.660                      |
| System-ID — GUID    | 28630ebd-41d5-11d1-a9c1-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
