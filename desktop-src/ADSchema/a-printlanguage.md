# <a name="print-language-attribute"></a>Атрибут Print-Language

Поддерживаемый язык описания страниц (например, PostScript, PCL).



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Print-Language                              |
| LDAP-отображаемое имя | принтлангуаже                               |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.246                      |
| System-ID — GUID    | 281416d6-1968-11d0-a28f-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
