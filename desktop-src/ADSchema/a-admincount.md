# <a name="admin-count-attribute"></a>Атрибут Admin-Count

Указывает, что списки ACL для данного объекта изменились на более безопасное значение системой, так как он был членом одной из административных групп (напрямую или транзитно).



| Ввод | Значение |
|-------------------|-----------------------------------------------------|
| CN                | Admin-Count                                         |
| LDAP-отображаемое имя | админкаунт                                          |
| Размер              | 4 байта                                             |
| Привилегия обновления  | Это значение задается системой.                    |
| Частота обновления  | При добавлении объекта в административную группу. |
| Attribute-Id      | 1.2.840.113556.1.4.150                              |
| System-ID — GUID    | bf967918-0de6-11d0-a285-00aa003049e2                |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                |
