# <a name="picture-attribute"></a>Атрибут рисунка

Изображение пользователя. Рекомендуется использовать формат с поддержкой пробелов, например JPEG или GIF.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Picture                                               |
| LDAP-отображаемое имя | thumbnailPhoto;                                        |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 2.16.840.1.113730.3.1.35                              |
| System-ID — GUID    | 8d3bca50-1d7e-11d0-a081-00aa006c33ed                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
