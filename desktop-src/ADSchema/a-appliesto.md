# <a name="applies-to-attribute"></a>Атрибут Applies-To

Содержит список классов объектов, к которым применяется расширенное право. В списке класс объекта представлен свойством schemaIDGUID для своего объекта Счемакласс.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Applies-To                                  |
| LDAP-отображаемое имя | appliesTo                                   |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.341                      |
| System-ID — GUID    | 8297931d-86d3-11d0-afda-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
