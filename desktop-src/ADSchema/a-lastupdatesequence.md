# <a name="last-update-sequence-attribute"></a>Атрибут Sequence последнего обновления

Этот атрибут содержит порядковый номер обновления для последнего элемента в хранилище классов, который был изменен.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Последняя-обновление — последовательность                        |
| LDAP-отображаемое имя | ластупдатесекуенце                          |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.330                      |
| System-ID — GUID    | 7d6c0e9c-7e20-11d0-afd6-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
