# <a name="policy-replication-flags-attribute"></a>Политика-репликация — атрибут Flags

Определяет, какие свойства LSA реплицируются на клиенты.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Политика-репликация-флаги             |
| LDAP-отображаемое имя | полицирепликатионфлагс               |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.633               |
| System-ID — GUID    | 19405b96-3cfa-11d1-a9c0-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
