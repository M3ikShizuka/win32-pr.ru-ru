# <a name="ms-sql-serviceaccount-attribute"></a>Атрибут MS-SQL-учетная запись службы

Определяемая пользователем строка. Значение по умолчанию — учетная запись службы.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-учетная запись службы                       |
| LDAP-отображаемое имя | mS-SQL-учетная запись службы                       |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | При установке системы.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1369                     |
| System-ID — GUID    | 64933a3e-ccee-11d2-9993-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
