# <a name="ms-pki-ra-policies-attribute"></a>атрибут MS-PKI-RA-Policies

Содержит список обязательных идентификаторов объектов политик из центров регистрации, которые подписывают запрос на регистрацию.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-RA-политики                                                                                |
| LDAP-отображаемое имя | Мспки-RA-политики                                                                                 |
| Размер              | 64 байт больше числа записей.                                                             |
| Привилегия обновления  | Администратор домена                                                                              |
| Частота обновления  | При изменении, создании или клонировании объекта шаблона сертификата (MS-PKI-Certificate-Template). |
| Attribute-Id      | 1.2.840.113556.1.4.1438                                                                           |
| System-ID — GUID    | d546ae22-0951-4d47-817e-1c9f96faad46                                                              |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                                       |
