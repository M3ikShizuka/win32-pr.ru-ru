# <a name="object-class-category-attribute"></a>Атрибут "объект-класс-категория"

Этот атрибут содержит тип класса, например abstract, вспомогательный или структурированный.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------|
| CN                | Объект-класс-категория                                                           |
| LDAP-отображаемое имя | objectClassCategory                                                             |
| Размер              | 4 байта. Структур 1, абстрактно 2, дополнительный 3. Класс 88, 0 не должен использоваться. |
| Привилегия обновления  | Это значение задается конструктором объекта.                                |
| Частота обновления  | Это значение никогда не должно изменяться.                                                 |
| Attribute-Id      | 1.2.840.113556.1.2.370                                                          |
| System-ID — GUID    | bf9679e6-0de6-11d0-a285-00aa003049e2                                            |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                                            |
