# <a name="ms-ds-repl-authentication-mode-attribute"></a>Атрибут ms-DS-REPL-Authentication-Mode

Атрибут **MS-DS-REPL-Authentication-Mode** используется для указания метода проверки подлинности, используемого для проверки подлинности партнеров репликации. Этот атрибут применяется к разделу конфигурации экземпляра ADAM.

Для этого атрибута возможны следующие значения.

| Значение        | Метод проверки подлинности                          | Описание                                                                                                                                                                    |
|--------------|------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0<br/> | Согласование (сквозная проверка)<br/>             | Все экземпляры ADAM в наборе конфигурации используют идентичные имя учетной записи и пароль в качестве учетной записи службы ADAM.<br/>                                                 |
| 1<br/> | На стадии изучения<br/>                          | Сначала выполняется попытка проверки подлинности Kerberos (с использованием имен SPN). В случае сбоя Kerberos выполняется попытка проверки подлинности NTLM. В случае сбоя NTLM экземпляры ADAM не будут реплицироваться.<br/> |
| 2<br/> | Взаимная проверка подлинности Kerberos<br/> | Необходима проверка подлинности Kerberos с использованием имен участников безопасности службы (SPN). Если проверка подлинности Kerberos завершается сбоем, экземпляры ADAM не реплицируются.<br/>                |



 

В следующей таблице содержатся программные идентификаторы для значений этого атрибута.

| Значение        | Идентификатор (из NTDSAPI. h)                                               |
|--------------|---------------------------------------------------------------------------|
| 0<br/> | **\_ \_ режим проверки подлинности ADAM REPL \_ \_ — согласование \_ передачи \_**<br/> |
| 1<br/> | **\_ \_ согласование режима проверки подлинности ADAM REPL \_ \_**<br/>                |
| 2<br/> | **\_ \_ \_ \_ требуется взаимная проверка \_ подлинности режима ADAM REPL \_**<br/>   |
