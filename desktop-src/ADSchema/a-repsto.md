# <a name="reps-to-attribute"></a>Атрибут Reps-To

Выводит список серверов, которые каталог будет уведомлять об изменениях и серверах, на которые каталог будет отправлять изменения по запросу для определенного контекста именования.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Reps-To                                               |
| LDAP-отображаемое имя | репсто                                                |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.83                                 |
| System-ID — GUID    | bf967a1e-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
