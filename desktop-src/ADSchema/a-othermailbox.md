# <a name="other-mailbox-attribute"></a>Атрибут Other-Mailbox

Содержит другие дополнительные адреса электронной почты в форме, например ККМАИЛ: Бруцекивер.



| Ввод | Значение |
|-------------------|----------------------------------------------------------|
| CN                | Other-Mailbox                                            |
| LDAP-отображаемое имя | осермаилбокс                                             |
| Размер              | \-                                                       |
| Привилегия обновления  | Администратор домена                                     |
| Частота обновления  | Каждый раз, когда пользователю выдается дополнительный адрес электронной почты. |
| Attribute-Id      | 1.2.840.113556.1.4.651                                   |
| System-ID — GUID    | 0296c123-40da-11d1-a9c0-0000f80367c1                     |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)              |
