# <a name="ms-ds-has-master-ncs-attribute"></a>Атрибут ms-DS-содержит-Master-NC

Содержит контексты именования, удерживаемые контроллером домена. Этот атрибут не является устаревшим с-Master-NC.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-содержит-Master-NC                    |
| LDAP-отображаемое имя | msDS-Хасмастернкс                       |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.         |
| Частота обновления  | При каждом создании нового контекста именования.          |
| Attribute-Id      | 1.2.840.113556.1.4.1836                 |
| System-ID — GUID    | ae2de0e2-59d7-4d47-8d47-ed4dfe4357ad    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
