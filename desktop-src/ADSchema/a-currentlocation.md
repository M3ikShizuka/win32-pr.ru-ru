# <a name="current-location-attribute"></a>Атрибут Current-Location

Расположение компьютера для объекта, который был перемещен.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Current-Location                                      |
| LDAP-отображаемое имя | currentLocation                                       |
| Размер              | \-                                                    |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.335                                |
| System-ID — GUID    | 1f0075fc-7e40-11d0-afd6-00c04fd930c9                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
