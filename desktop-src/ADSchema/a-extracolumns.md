# <a name="extra-columns-attribute"></a>Атрибут Extra-Columns

Это многозначный атрибут, значения которого состоят из 5 кортежа: (имя атрибута), (заголовок столбца), (видимость по умолчанию (0, 1)), (ширина столбца (1 для автоматической ширины)), 0 (зарезервировано для будущего использования должна быть нулевой). Это значение используется в консоли "пользователи и компьютеры Active Directory".



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | Extra-Columns                                                                                                                                                        |
| LDAP-отображаемое имя | екстраколумнс                                                                                                                                                         |
| Размер              | Каждое значение будет иметь размер строки из 5 кортежа выше. По умолчанию для объекта, отображаемого по умолчанию в контейнере ДисплайспеЦифиер, будет отображаться 22 значения. |
| Привилегия обновления  | Администратор домена                                                                                                                                                 |
| Частота обновления  | Это будет обновлено только в том случае, если установлена служба Exchange.                                                                                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1687                                                                                                                                              |
| System-ID — GUID    | d24e2846-1dd9-4bcf-99d7-a6227cc86da7                                                                                                                                 |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                                                                                                          |
