# <a name="pkt-attribute"></a>Атрибут PKT

Таблица знаний раздела DFS. Описывает структуру распределенная файловая система иерархии.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | PKT                                                   |
| LDAP-отображаемое имя | pKT                                                   |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.206                                |
| System-ID — GUID    | 8447f9f1-1027-11d0-a05f-00aa006c33ed                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
