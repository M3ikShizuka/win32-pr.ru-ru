# <a name="ms-pki-private-key-flag-attribute"></a>атрибут MS-PKI-Private-Key-Flag

Содержит флаги, связанные с закрытым ключом.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-закрытый-Key-флаг                                                                           |
| LDAP-отображаемое имя | Мспки-закрытый-ключ-флаг                                                                            |
| Размер              | 4 байта                                                                                           |
| Привилегия обновления  | Администратор домена                                                                              |
| Частота обновления  | При изменении, создании или клонировании объекта шаблона сертификата (MS-PKI-Certificate-Template). |
| Attribute-Id      | 1.2.840.113556.1.4.1431                                                                           |
| System-ID — GUID    | bab04ac2-0435-4709-9307-28380e7c7001                                                              |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                                                              |
