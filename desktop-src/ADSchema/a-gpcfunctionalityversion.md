| LDAP-отображаемое имя | гпкфунктионалитиверсион              |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.893               |
| System-ID — GUID    | f30e3bc0-9ff0-11d1-b603-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
