# <a name="email-information-property-set"></a>Набор свойств Email-Information

Набор свойств, содержащий атрибуты пользователя, которые описывают сведения, относящиеся к электронной почте пользователя.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Email-Information                    |
| Display-Name | параметры Телефон и почты               |
| Rights-GUID  | e45795b2-9455-11d1-aebd-0000f80367c1 |
