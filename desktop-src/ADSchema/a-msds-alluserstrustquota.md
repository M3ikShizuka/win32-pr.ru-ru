# <a name="ms-ds-all-users-trust-quota-attribute"></a>Атрибут MS-DS-All-Users-Trust-Quota

Используется для принудительного применения общей квоты пользователей к общему количеству Trusted-Domain объектов, созданных с помощью право доступа New Control, CREATE-Inbound-лесами-Trust.



| Ввод | Значение |
|-------------------|-------------------------------------------|
| CN                | MS-DS-All-Users-Trust — квота               |
| LDAP-отображаемое имя | msDS-Аллусерструсткуота                   |
| Размер              | \-                                        |
| Привилегия обновления  | Администратор домена                      |
| Частота обновления  | При создании леса и редко после него. |
| Attribute-Id      | 1.2.840.113556.1.4.1789                   |
| System-ID — GUID    | d3aa4a5c-4e03-4810-97aa-2b339e7a434b      |
| Синтаксис            | [**Перечисление**](s-enumeration.md)      |
