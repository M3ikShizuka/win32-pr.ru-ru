Содержит значение ограждения корневого каталога.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-Рутфенце                    |
| LDAP-отображаемое имя | Мсдфср — Рутфенце                     |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.22           |
| System-ID — GUID    | 51928e94-2cd8-4abe-b552-e50412444370 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
