# <a name="phone-mobile-primary-attribute"></a>атрибут Телефон-Mobile — Primary

Основной номер мобильного телефона.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Телефон — Mobile — Primary                                                             |
| LDAP-отображаемое имя | mobile                                                                           |
| Размер              | \-                                                                               |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                           |
| Частота обновления  | При создании записи пользователя и при необходимости изменения номера телефона. |
| Attribute-Id      | 0.9.2342.19200300.100.1.41                                                       |
| System-ID — GUID    | f0f8ffa3-1191-11d0-a060-00aa006c33ed                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
