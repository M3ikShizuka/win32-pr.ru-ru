# <a name="certificate-revocation-list-attribute"></a>Атрибут списка отзыва сертификатов

Представляет список отозванных сертификатов.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Список отзыва сертификатов                           |
| LDAP-отображаемое имя | цертификатеревокатионлист                             |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 2.5.4.39                                              |
| System-ID — GUID    | 1677579f-47f3-11d1-a9c3-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
