# <a name="msmq-authenticate-attribute"></a>Атрибут MSMQ-Authenticate

Указывает, принимает ли очередь только сообщения, прошедшие проверку подлинности.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MSMQ-Authenticate                    |
| LDAP-отображаемое имя | мсмкаусентикате                     |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.923               |
| System-ID — GUID    | 9a0dc326-c100-11d1-bbc5-0080c76670c0 |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
