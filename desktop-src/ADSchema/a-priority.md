# <a name="priority-attribute"></a>Атрибут Priority

Текущий приоритет (процесса, задания печати и т. д.).



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Приоритет                             |
| LDAP-отображаемое имя | priority                             |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.231               |
| System-ID — GUID    | 281416c7-1968-11d0-a28f-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
