# <a name="primary-group-token-attribute"></a>Атрибут-маркер первичной группы

Вычисленный атрибут, используемый для получения списка членства в группе, например для пользователей домена. Полное членство таких групп не сохраняется явно для целей масштабирования.



| Ввод | Значение |
|-------------------|--------------------------------------------|
| CN                | Маркер первичной группы                        |
| LDAP-отображаемое имя | примариграуптокен                          |
| Размер              | 4 байта                                    |
| Привилегия обновления  | Это значение задается системой.           |
| Частота обновления  | Каждый раз при изменении основной группы объектов. |
| Attribute-Id      | 1.2.840.113556.1.4.1412                    |
| System-ID — GUID    | c0ed8738-7efd-4481-84d9-66d2db8be369       |
| Синтаксис            | [**Перечисление**](s-enumeration.md)       |
