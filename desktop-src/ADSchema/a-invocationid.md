# <a name="invocation-id-attribute"></a>Атрибут Invocation-Id

Используется для уникальной идентификации каждого каталога Microsoft Exchange Server в Организации.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Invocation-Id                                         |
| LDAP-отображаемое имя | invocationId                                          |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | При установке Exchange Server.                |
| Attribute-Id      | 1.2.840.113556.1.2.115                                |
| System-ID — GUID    | bf96798e-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
