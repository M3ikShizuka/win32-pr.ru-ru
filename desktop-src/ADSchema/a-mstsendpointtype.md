# <a name="ms-ts-endpoint-type-attribute"></a>атрибут MS-TS-Endpoint-Type

Этот атрибут определяет, является ли компьютер физическим компьютером или виртуальной машиной.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-TS-Endpoint-Type                  |
| LDAP-отображаемое имя | мстсендпоинттипе                     |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2071              |
| System-ID — GUID    | 377ade80-e2d8-46c5-9bcd-6d9dec93b35e |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
