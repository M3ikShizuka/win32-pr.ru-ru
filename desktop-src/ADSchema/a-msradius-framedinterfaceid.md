# <a name="ms-radius-framedinterfaceid-attribute"></a>атрибут MS-RADIUS-Фрамединтерфацеид

Указывает идентификатор интерфейса IPv6, который должен быть настроен для пользователя.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-RADIUS-Фрамединтерфацеид          |
| LDAP-отображаемое имя | Мсрадиус — Фрамединтерфацеид           |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1913              |
| System-ID — GUID    | a6f24a23-d65c-4d65-a64f-35fb6873c2b9 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
