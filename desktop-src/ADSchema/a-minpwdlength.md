# <a name="min-pwd-length-attribute"></a>Атрибут min-PWD-length

Минимальное число символов, которое должен содержать пароль.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Min-PWD-length                       |
| LDAP-отображаемое имя | минпвдленгс                         |
| Размер              | \-                                   |
| Привилегия обновления  | Администратор домена                 |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.79                |
| System-ID — GUID    | bf9679c3-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
