# <a name="personal-title-attribute"></a>Атрибут Personal-Title

Название пользователя.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Personal-Title                                                                   |
| LDAP-отображаемое имя | персоналтитле                                                                    |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.2.615                                                           |
| System-ID — GUID    | 16775858-47f3-11d1-a9c3-0000f80367c1                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
