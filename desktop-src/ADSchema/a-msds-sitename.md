# <a name="ms-ds-sitename-attribute"></a>Атрибут ms-DS-SiteName

Список имен сайтов, соответствующих контроллеру домена.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-SiteName                              |
| LDAP-отображаемое имя | msDS-SiteName                               |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1961                     |
| System-ID — GUID    | 98a7f36d-3595-448a-9e6f-6b8965baed9c        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
