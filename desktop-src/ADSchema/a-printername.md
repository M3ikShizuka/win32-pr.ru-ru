# <a name="printer-name-attribute"></a>Атрибут Printer-Name

Отображаемое имя присоединенного принтера.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Printer-Name                                |
| LDAP-отображаемое имя | printerName                                 |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.300                      |
| System-ID — GUID    | 244b296e-5abd-11d0-afd2-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
