# <a name="display-name-attribute"></a>Атрибут Display-Name

Отображаемое имя объекта. Обычно это сочетание имени, инициала и фамилии пользователя.



| Ввод | Значение |
|-------------------|------------------------------------------------------------------------------|
| CN                | Display-Name                                                                 |
| LDAP-отображаемое имя | displayName                                                                  |
| Размер              | \-                                                                           |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                       |
| Частота обновления  | При создании записи пользователя и необходимости изменения отображаемого имени. |
| Attribute-Id      | 1.2.840.113556.1.2.13                                                        |
| System-ID — GUID    | bf967953-0de6-11d0-a285-00aa003049e2                                         |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                  |
