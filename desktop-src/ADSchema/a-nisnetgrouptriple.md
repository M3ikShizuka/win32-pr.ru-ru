# <a name="nisnetgrouptriple-attribute"></a>атрибут Ниснетграуптрипле

Представляет одну запись из схемы сетевой группы.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ниснетграуптрипле                    |
| LDAP-отображаемое имя | ниснетграуптрипле                    |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.14                     |
| System-ID — GUID    | a8032e74-30ef-4ff5-affc-0fc217783fec |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
