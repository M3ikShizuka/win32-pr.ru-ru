# <a name="msi-file-list-attribute"></a>MSI-файл-атрибут списка

Этот атрибут содержит список файлов установщика Microsoft, таких как базовый MSI-файл (.msi) и файлы преобразования MST (. MST).



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MSI-файл-список                               |
| LDAP-отображаемое имя | мсифилелист                                 |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.671                      |
| System-ID — GUID    | 7bfdcb7d-4807-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
