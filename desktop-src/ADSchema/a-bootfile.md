# <a name="bootfile-attribute"></a>атрибут Бутфиле

Содержит имя загрузочного образа для поддержки управления удостоверениями UNIX.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | бутфиле                             |
| LDAP-отображаемое имя | бутфиле                             |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.24                     |
| System-ID — GUID    | e3f3cb4e-0f20-42eb-9703-d2ff26e52667 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
