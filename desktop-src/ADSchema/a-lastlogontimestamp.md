# <a name="last-logon-timestamp-attribute"></a>Атрибут метки времени последнего входа

Время последнего входа пользователя в домен. Это значение хранится в виде большого целого числа, представляющего число 100-наносекундных интервалов с 1 января 1601 (UTC). При каждом входе пользователя в систему значение этого атрибута считывается с контроллера домена. Если значение старше \[ `current_time - msDS-LogonTimeSyncInterval` \] , значение обновляется. Начальное обновление после повышения функционального уровня домена вычисляется в течение 14 дней минус случайный процент от 5 дней.

| LDAP-отображаемое имя | lastLogonTimestamp                                                                                                     |
| Размер              | \-                                                                                                                     |
| Привилегия обновления  | Это значение задается системой.                                                                                       |
| Частота обновления  | При входе пользователя в систему и если это значение старше текущего времени минус значение msDS-ЛогонтимесинЦинтервал. |
