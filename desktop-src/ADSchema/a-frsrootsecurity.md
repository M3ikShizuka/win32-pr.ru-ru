# <a name="frs-root-security-attribute"></a>FRS-root-атрибут безопасности

Дескриптор безопасности корня набора реплик для репликации файлов.



| Ввод | Значение |
|-------------------|-----------------------------------------------------|
| CN                | FRS-root-Security                                   |
| LDAP-отображаемое имя | фрсрутсекурити                                     |
| Размер              | \-                                                  |
| Привилегия обновления  | \-                                                  |
| Частота обновления  | \-                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.535                              |
| System-ID — GUID    | 5245801f-ca6a-11d0-afff-0000f80367c1                |
| Синтаксис            | [**String(NT-Sec-Desc)**](s-string-nt-sec-desc.md) |
