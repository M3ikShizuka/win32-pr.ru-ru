# <a name="acs-event-log-level-attribute"></a>ACS-Event-атрибут уровня журнала

Уровень ведения журнала RSVP.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | ACS-Event-уровень журнала                     |
| LDAP-отображаемое имя | аксевентлоглевел                        |
| Размер              | 4 байта могут иметь значения 0, 1, 2 и 3. |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.769                  |
| System-ID — GUID    | 7f561286-5301-11d1-a9c5-0000f80367c1    |
| Синтаксис            | [**Перечисление**](s-enumeration.md)    |
