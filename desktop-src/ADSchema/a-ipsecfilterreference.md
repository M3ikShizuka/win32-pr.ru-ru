# <a name="ipsec-filter-reference-attribute"></a>Атрибут ссылки IPSec-Filtering

Атрибут **IPSec-Filtering-Reference** предназначен только для внутреннего использования.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Справочник по IPsec-Filtering                  |
| LDAP-отображаемое имя | ипсекфилтерреференце                    |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.629                  |
| System-ID — GUID    | b40ff823-427a-11d1-a9c2-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
