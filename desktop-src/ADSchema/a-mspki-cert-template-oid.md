# <a name="ms-pki-cert-template-oid-attribute"></a>атрибут MS-PKI-CERT-Template-OID

Указывает идентификатор объекта для шаблона сертификата.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-CERT-Template-OID                                                                          |
| LDAP-отображаемое имя | Мспки-CERT-Template-OID                                                                           |
| Размер              | 64 байт                                                                                          |
| Привилегия обновления  | Администратор домена                                                                              |
| Частота обновления  | При изменении, создании или клонировании объекта шаблона сертификата (MS-PKI-Certificate-Template). |
| Attribute-Id      | 1.2.840.113556.1.4.1436                                                                           |
| System-ID — GUID    | 3164c36a-ba26-468c-8bda-c1e5cc256728                                                              |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                                       |
