# <a name="ms-sql-language-attribute"></a>MS-SQL — атрибут Language

Язык текущего экземпляра SQL Server.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Language                             |
| LDAP-отображаемое имя | mS-SQL-Language                             |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | При установке системы.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1389                     |
| System-ID — GUID    | c57f72f4-ccee-11d2-9993-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
