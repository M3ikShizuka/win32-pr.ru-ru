# <a name="acs-non-reserved-tx-limit-attribute"></a>ACS — незарезервированный-TX — атрибут Limit

Максимальная пропускная способность, которую пользовательское приложение может передать, прежде чем заместить резервирование.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-не зарезервировано-TX-Limit            |
| LDAP-отображаемое имя | акснонресерведткслимит                |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.780               |
| System-ID — GUID    | 1cb355a2-56d0-11d1-a9c6-0000f80367c1 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
