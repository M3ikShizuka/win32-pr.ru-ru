# <a name="ms-pki-certificate-application-policy-attribute"></a>MS-PKI-Certificate-приложение-атрибут политики

OID политики приложения в сертификате.



| Ввод | Значение |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-Certificate-приложение-политика                                                      |
| LDAP-отображаемое имя | Мспки-Certificate-приложение-политика                                                       |
| Размер              | \-                                                                                         |
| Привилегия обновления  | Администратор предприятия                                                                   |
| Частота обновления  | При создании нового шаблона или изменении атрибутов существующих шаблонов. |
| Attribute-Id      | 1.2.840.113556.1.4.1674                                                                    |
| System-ID — GUID    | dbd90548-aa37-4202-9966-8c537ba5ce32                                                       |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                                |
