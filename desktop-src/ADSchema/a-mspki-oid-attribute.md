# <a name="ms-pki-oid-attribute-attribute"></a>атрибут MS-PKI-OID-Attribute

Атрибут для корпоративного OID.



| Ввод | Значение |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-OID-атрибут                                                                       |
| LDAP-отображаемое имя | Мспки-OID-атрибут                                                                        |
| Размер              | 4 байта                                                                                    |
| Привилегия обновления  | Администратор предприятия                                                                   |
| Частота обновления  | При создании нового шаблона или изменении атрибутов существующих шаблонов. |
| Attribute-Id      | 1.2.840.113556.1.4.1671                                                                    |
| System-ID — GUID    | 8c9e1288-5028-4f4f-a704-76d026f246ef                                                       |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                                                       |
