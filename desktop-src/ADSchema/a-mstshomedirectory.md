# <a name="ms-ts-home-directory-attribute"></a>атрибут MS-TS-Home-Directory

Корневой каталог служб терминалов указывает домашний каталог для пользователя.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-TS-Home-Directory                        |
| LDAP-отображаемое имя | мстшомедиректори                           |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1977                     |
| System-ID — GUID    | 5d3510f0-c4e7-4122-b91f-a20add90e246        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
