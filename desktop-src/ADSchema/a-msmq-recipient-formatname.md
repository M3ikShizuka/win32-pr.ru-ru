# <a name="msmq-recipient-formatname-attribute"></a>Атрибут MSMQ-Recipient-FormatName

Используется в качестве имени формата получателя в классе MSMQ-Custom-получател.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MSMQ-Recipient-FormatName                   |
| LDAP-отображаемое имя | msMQ-Recipient-FormatName                   |
| Размер              | от 1 до 255 символов.                        |
| Привилегия обновления  | Владелец очереди.                            |
| Частота обновления  | Только после перемещения внешней очереди.    |
| Attribute-Id      | 1.2.840.113556.1.4.1695                     |
| System-ID — GUID    | 3bfe6748-b544-485a-b067-1b310c4334bf        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
