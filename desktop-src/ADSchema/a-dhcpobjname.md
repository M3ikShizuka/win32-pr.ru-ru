# <a name="dhcp-obj-name-attribute"></a>атрибут DHCP-obj-Name

Атрибут **DHCP-obj-Name** в настоящее время не используется.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | DHCP-obj-Name                               |
| LDAP-отображаемое имя | дхкпобжнаме                                 |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.702                      |
| System-ID — GUID    | 963d2743-48be-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
