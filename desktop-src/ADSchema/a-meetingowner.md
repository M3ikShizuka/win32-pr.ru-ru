# <a name="meetingowner-attribute"></a>атрибут Митинговнер

Лицо, ответственное за собрание.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | митинговнер                                                                     |
| LDAP-отображаемое имя | митинговнер                                                                     |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.579                                                           |
| System-ID — GUID    | 11b6cc88-48c4-11d1-a9c3-0000f80367c1                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
