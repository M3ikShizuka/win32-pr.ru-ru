# <a name="delta-revocation-list-attribute"></a>Атрибут списка изменений отзыва

Список сертификатов, отозванных с момента последнего разностного обновления.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Список отзыва изменений                                 |
| LDAP-отображаемое имя | делтаревокатионлист                                   |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 2.5.4.53                                              |
| System-ID — GUID    | 167757b5-47f3-11d1-a9c3-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
