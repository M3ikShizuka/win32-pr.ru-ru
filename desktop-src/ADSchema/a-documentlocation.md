# <a name="documentlocation-attribute"></a>атрибут Документлокатион

Тип атрибута Документлокатион указывает расположение исходного документа.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | документлокатион                            |
| LDAP-отображаемое имя | документлокатион                            |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.15                  |
| System-ID — GUID    | b958b14e-ac6d-4ec4-8892-be70b69f7281        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
