# <a name="ms-ds-sd-reference-domain-attribute"></a>Атрибут ms-DS-SD-Reference-domain

Имя домена, используемого для преобразования дескрипторов безопасности для контекста, не являющегося доменом.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-SD-Reference-domain               |
| LDAP-отображаемое имя | msDS-Сдреференцедомаин                  |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1711                 |
| System-ID — GUID    | 4c51e316-f628-43a5-b06b-ffb695fcb4f3    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
