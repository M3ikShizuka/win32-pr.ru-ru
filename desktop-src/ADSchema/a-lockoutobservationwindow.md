# <a name="lock-out-observation-window-attribute"></a>Атрибут блокировки — наблюдение — окно

Диапазон времени в 100-наносекундных интервалах, в течение которого система увеличивает неверное число входов в систему.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Блокировка — наблюдение — окно          |
| LDAP-отображаемое имя | локкаутобсерватионвиндов             |
| Размер              | \-                                   |
| Привилегия обновления  | Администратор домена                 |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.61                |
| System-ID — GUID    | bf9679a4-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
