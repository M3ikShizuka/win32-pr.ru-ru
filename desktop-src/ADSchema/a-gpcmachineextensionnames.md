# <a name="gpc-machine-extension-names-attribute"></a>Атрибут ГПК-Machine-Extension-Names

Используется объектом групповая политика для политик компьютера.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------|
| CN                | ГПК-Machine-Extension-Names                                                     |
| LDAP-отображаемое имя | гпкмачиникстенсионнамес                                                        |
| Размер              | Зависит от числа клиентских расширений, имеющих политику в этом объекте групповой политики. |
| Привилегия обновления  | Администратор домена или политики.                                                 |
| Частота обновления  | Каждый раз при обновлении объекта групповой политики с помощью ГПЕ.                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1348                                                         |
| System-ID — GUID    | 32ff8ecc-783f-11d2-9916-0000f87a57d4                                            |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                     |
