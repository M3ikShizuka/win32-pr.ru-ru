| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-необязательный параметр-Feature-flags         |
| LDAP-отображаемое имя | msDS-Оптионалфеатурефлагс            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2063              |
| System-ID — GUID    | 8a0560c1-97b9-4811-9db7-dc061598965b |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
