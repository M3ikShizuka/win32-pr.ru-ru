# <a name="sam-account-type-attribute"></a>Атрибут SAM-Account-Type

Этот атрибут содержит сведения о каждом объекте типа учетной записи. Вы можете перечислить список типов учетных записей или использовать API экранной информации для создания списка. Так как компьютеры, обычные учетные записи пользователей и учетные записи доверия могут перечисляться как объекты пользователя, значения этих учетных записей должны быть непрерывного диапазона.

Ниже приведены возможные значения для этого атрибута.

-   SAM \_ доменное \_ объект 0x0
-   SAM \_ Group \_ Object 0x10000000
-   SAM \_ : \_ \_ \_ 0x10000001 объекта группы безопасности
-   SAM: \_ псевдоним \_ объекта 0x20000000
-   SAM \_ не \_ \_ 0x20000001 объект псевдонима безопасности \_
-   \_ \_ 0x30000000 объект пользователя SAM
-   SAM \_ Обычная \_ \_ учетная запись пользователя 0x30000000
-   SAM \_ \_ 0x30000001 учетной записи компьютера
-   SAM: \_ доверие \_ учетной записи 0x30000002
-   \_ \_ 0x40000000 группы основных приложений SAM \_
-   \_ \_ 0x40000001 группы запросов приложения SAM \_
-   \_Тип учетной записи SAM \_ \_ Max 0x7FFFFFFF



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------|
| CN                | SAM-тип учетной записи                                                |
| LDAP-отображаемое имя | sAMAccountType                                                  |
| Размер              | \-                                                              |
| Привилегия обновления  | Это значение задается системой.                                |
| Частота обновления  | Это значение задается операционной системой при создании объекта. |
| Attribute-Id      | 1.2.840.113556.1.4.302                                          |
| System-ID — GUID    | 6e7b626c-64f2-11d0-afd2-00c04fd930c9                            |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                            |
