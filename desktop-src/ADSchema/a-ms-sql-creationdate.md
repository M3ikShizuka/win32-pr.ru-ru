# <a name="ms-sql-creationdate-attribute"></a>Атрибут MS-SQL-CreationDate

Дата создания базы данных.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-CreationDate                         |
| LDAP-отображаемое имя | mS-SQL-CreationDate                         |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | При создании новой базы данных.             |
| Attribute-Id      | 1.2.840.113556.1.4.1397                     |
| System-ID — GUID    | ede14754-ccee-11d2-9993-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
