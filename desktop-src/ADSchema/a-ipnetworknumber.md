# <a name="ipnetworknumber-attribute"></a>атрибут Ипнетворкнумбер

Содержит номер сети IP в точечно-десятичной нотации, в которой опущены начальные нули.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ипнетворкнумбер                      |
| LDAP-отображаемое имя | ипнетворкнумбер                      |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.20                     |
| System-ID — GUID    | 4e3854f4-3087-42a4-a813-bb0c528958d3 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
