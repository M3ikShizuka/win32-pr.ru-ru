# <a name="admin-display-name-attribute"></a>Атрибут "admin-Name"

Имя, отображаемое на экранах администратора.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Имя администратора-отображение                          |
| LDAP-отображаемое имя | adminDisplayName                            |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.194                      |
| System-ID — GUID    | bf96791a-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
