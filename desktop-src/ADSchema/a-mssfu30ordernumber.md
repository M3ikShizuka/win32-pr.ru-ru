# <a name="mssfu-30-order-number-attribute"></a>атрибут Мссфу-30-Order-Number

Содержит значение, используемое NIS для проверки изменения схемы. При каждом изменении данных, хранящихся в объекте [**мссфу-30-Domain-info**](c-mssfu30domaininfo.md) , это значение увеличивается. Это значение используется для контроля изменений данных между вызовами **ипксфер** .



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Мссфу-30-порядок-номер                       |
| LDAP-отображаемое имя | msSFU30OrderNumber                          |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.308                 |
| System-ID — GUID    | 02625f05-d1ee-4f9f-b366-55266becb95c        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
