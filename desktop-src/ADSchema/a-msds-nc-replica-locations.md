# <a name="ms-ds-nc-replica-locations-attribute"></a>Атрибут ms-DS-NC-Replica-Locations

Список серверов, являющихся наборами реплик для соответствующего контекста именования, не являющегося доменом.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-NC-Replica-Locations              |
| LDAP-отображаемое имя | msDS-NC-Replica-Locations               |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1661                 |
| System-ID — GUID    | 97de9615-b537-46bc-ac0f-10720f3909f3    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
