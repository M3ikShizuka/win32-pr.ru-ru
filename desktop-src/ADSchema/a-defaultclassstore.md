| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Хранилище классов по умолчанию                     |
| LDAP-отображаемое имя | дефаулткласссторе                       |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.213                  |
| System-ID — GUID    | bf967948-0de6-11d0-a285-00aa003049e2    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
