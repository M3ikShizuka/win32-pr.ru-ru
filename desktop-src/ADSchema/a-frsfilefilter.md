# <a name="frs-file-filter-attribute"></a>Атрибут FRS-File-Filter

Список расширений имен файлов, исключенных из репликации файлов.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | FRS-File-Filter                             |
| LDAP-отображаемое имя | фрсфилефилтер                               |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | При установке репликации.                  |
| Attribute-Id      | 1.2.840.113556.1.4.483                      |
| System-ID — GUID    | 1be8f170-a9ff-11d0-afe2-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
