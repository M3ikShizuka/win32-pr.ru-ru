# <a name="meetingoriginator-attribute"></a>атрибут Митингоригинатор

Пользователь, который запланировал собрание.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | митингоригинатор                                                                |
| LDAP-отображаемое имя | митингоригинатор                                                                |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.577                                                           |
| System-ID — GUID    | 11b6cc86-48c4-11d1-a9c3-0000f80367c1                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
