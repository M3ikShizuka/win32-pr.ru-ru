# <a name="ms-ds-az-class-id-attribute"></a>Атрибут ms-DS-AZ-Class-ID

Идентификатор класса, необходимый пользовательскому интерфейсу Азролес для объекта Азаппликатион.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-AZ-Class-ID                           |
| LDAP-отображаемое имя | msDS-Азклассид                              |
| Размер              | 36 символов                               |
| Привилегия обновления  | Администратор Азролес                               |
| Частота обновления  | Во время инициализации или изменения политики.     |
| Attribute-Id      | 1.2.840.113556.1.4.1816                     |
| System-ID — GUID    | 013a7277-5c2d-49ef-a7de-b765b36a3f6f        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
