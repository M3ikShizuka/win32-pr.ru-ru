# <a name="extended-chars-allowed-attribute"></a>Расширенный атрибут-chars-Allowed

Указывает, разрешены ли расширенные символы в значении этого атрибута. Применяется только к атрибутам строковых атрибутов IA5, numeric, печатаемых и Teletex.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Расширенные символы-разрешено               |
| LDAP-отображаемое имя | екстендедчарсалловед                 |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.380               |
| System-ID — GUID    | bf967966-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
