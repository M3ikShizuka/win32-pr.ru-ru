# <a name="department-attribute"></a>Атрибут Department

Содержит имя отдела, в котором работает пользователь.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Отдел                                  |
| LDAP-отображаемое имя | department                                  |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена или владелец учетной записи.      |
| Частота обновления  | Каждый раз, когда необходимо изменить отдел.    |
| Attribute-Id      | 1.2.840.113556.1.2.141                      |
| System-ID — GUID    | bf96794f-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
