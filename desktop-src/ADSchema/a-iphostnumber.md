# <a name="iphostnumber-attribute"></a>атрибут Ифостнумбер

Содержит IP-адрес узла с точками в десятичной нотации, в котором опущены начальные нули.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ipHostNumber                         |
| LDAP-отображаемое имя | ipHostNumber                         |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.19                     |
| System-ID — GUID    | de8bb721-85dc-4fde-b687-9657688e667e |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
