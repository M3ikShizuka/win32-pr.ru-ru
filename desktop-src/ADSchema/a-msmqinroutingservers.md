# <a name="msmq-in-routing-servers-attribute"></a>Атрибут MSMQ-in-Routing-Servers

DN-ссылки на серверы маршрутизации MSMQ, через которые должен маршрутизироваться весь входящий трафик к этому компьютеру.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MSMQ-in-Routing-серверы                 |
| LDAP-отображаемое имя | мсмкинраутингсерверс                    |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.929                  |
| System-ID — GUID    | 9a0dc32c-c100-11d1-bbc5-0080c76670c0    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
