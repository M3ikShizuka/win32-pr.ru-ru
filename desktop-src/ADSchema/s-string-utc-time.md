# <a name="stringutc-time-syntax"></a>Синтаксис String (время в формате UTC)

Формат строки времени, определенный стандартом ASN. 1. Дополнительные сведения см. в разделе ISO 8601 и X680. Используйте этот синтаксис для хранения значений времени в UTC-Time формате.



| Ввод | Значение |
|--------------|----------------------------------------------------------------------------|
| Имя         | String(время_UTC)                                                           |
| Идентификатор синтаксиса    | 2.5.5.11                                                                   |
| ИДЕНТИФИКАТОР OM        | 23                                                                         |
| Тип MAPI    | систиме                                                                    |
| Тип ADS     | АДСТИПЕ \_ время в формате UTC \_                                                         |
| Тип варианта | \_Дата VT                                                                   |
| Тип SDS     | [System.DateTime](/dotnet/api/system.datetime) |
