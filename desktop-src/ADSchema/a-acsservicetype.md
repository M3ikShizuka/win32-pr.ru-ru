# <a name="acs-service-type-attribute"></a>ACS-Service-тип — атрибут

Тип службы ACS. Управляемая нагрузка или гарантированная пропускная способность.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-Service-Type                     |
| LDAP-отображаемое имя | акссервицетипе                       |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.762               |
| System-ID — GUID    | 7f56127f-5301-11d1-a9c5-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
