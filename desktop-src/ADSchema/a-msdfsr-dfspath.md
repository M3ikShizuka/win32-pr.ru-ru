# <a name="ms-dfsr-dfspath-attribute"></a>атрибут MS-DFSR-Дфспас

Содержит полный путь к связанной распределенная файловая система (DFS) для поддержки службы репликация DFS.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DFSR-Дфспас                             |
| LDAP-отображаемое имя | Мсдфср — Дфспас                              |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.21                  |
| System-ID — GUID    | 2cc903e2-398c-443b-ac86-ff6b01eac7ba        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
