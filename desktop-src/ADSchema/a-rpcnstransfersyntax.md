# <a name="rpc-ns-transfer-syntax-attribute"></a>атрибут RPC-NS-reby-Syntax

UUID синтаксиса перемещения, поддерживаемого текущей записью.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | RPC-NS-передает-синтаксис                      |
| LDAP-отображаемое имя | рпкнстрансферсинтакс                         |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.314                      |
| System-ID — GUID    | 29401c4a-7a27-11d0-afd6-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
