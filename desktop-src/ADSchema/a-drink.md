# <a name="drink-attribute"></a>напитки, атрибут

Тип атрибута напитки (напитки) определяет любимый напитки объекта (или человека).



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | напиток                                       |
| LDAP-отображаемое имя | напиток                                       |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.5                   |
| System-ID — GUID    | 1a1aa5b5-262e-4df6-af04-2cf6b0d80048        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
