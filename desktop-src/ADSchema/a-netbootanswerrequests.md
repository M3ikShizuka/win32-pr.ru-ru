# <a name="netboot-answer-requests-attribute"></a>атрибут нетбут-Answer-Requests

Позволяет серверу RIS принимать любые запросы RIS.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | нетбут — ответы — запросы              |
| LDAP-отображаемое имя | нетбутансверрекуестс                |
| Размер              | \-                                   |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.853               |
| System-ID — GUID    | 0738307a-91df-11d1-aebc-0000f80367c1 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
