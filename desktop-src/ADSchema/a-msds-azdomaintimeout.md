# <a name="ms-ds-az-domain-timeout-attribute"></a>Атрибут ms-DS-AZ-Domain-timeout

Время в миллисекундах, после которого домен будет недоступен и до повторной попытки.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-AZ-Domain-timeout                 |
| LDAP-отображаемое имя | msDS-Аздомаинтимеаут                    |
| Размер              | \-                                      |
| Привилегия обновления  | Администратор Азролес                           |
| Частота обновления  | Во время инициализации или изменения политики. |
| Attribute-Id      | 1.2.840.113556.1.4.1795                 |
| System-ID — GUID    | 6448f56a-ca70-4e2e-b0af-d20e4ce653d0    |
| Синтаксис            | [**Перечисление**](s-enumeration.md)    |
