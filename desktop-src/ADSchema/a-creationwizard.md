|-------------------|---------------------------------------------|
| CN                | Creation-Wizard                             |
| LDAP-отображаемое имя | креатионвизард                              |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.498                      |
| System-ID — GUID    | 4d8601ed-ac85-11d0-afe3-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
