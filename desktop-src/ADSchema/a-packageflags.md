# <a name="package-flags-attribute"></a>Атрибут Package-Flags

Битовое поле, содержащее флаги состояния развертывания для приложения.

Этот атрибут может быть нулевым или сочетанием одного или нескольких из следующих значений.

| Значение                 | Описание                                                                                                                                                                                                                                                                                                    |
|-----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0x00000004<br/> | Перед назначением необходимо удалить неуправляемую версию этого приложения. **Windows XP с пакетом обновления SP1 и более поздней версии:** Этот флаг не поддерживается.<br/> <br/>                                                                                                                                          |
| 0x00000008<br/> | Это опубликованное приложение.<br/>                                                                                                                                                                                                                                                                    |
| 0x00000010<br/> | Этот пакет был развернут после бета-версии 3 Windows 2000.<br/>                                                                                                                                                                                                                                             |
| 0x00000020<br/> | Это приложение можно установить с помощью компонента " **Установка и удаление программ** " панели управления.<br/>                                                                                                                                                                                                 |
| 0x00000040<br/> | Это приложение может быть установлено по запросу.<br/>                                                                                                                                                                                                                                                   |
| 0x00000080<br/> | Это приложение является потерянным. Опубликованное приложение может стать потерянным, если администратор удалит приложение из списка развертываемых приложений.<br/>                                                                                                                                    |
| 0x00000100<br/> | Это приложение следует рассматривать как удаленное.<br/>                                                                                                                                                                                                                                                  |
| 0x00000200<br/> | Это приложение доступно только для пилотного развертывания.<br/>                                                                                                                                                                                                                                                      |
| 0x00000400<br/> | Это назначенное приложение. Назначенное приложение не может быть полностью удалено пользователем. Если пользователь пытается удалить приложение с помощью компонента " **Установка и удаление программ** " панели управления, приложение будет повторно объявлено на компьютере после завершения удаления.<br/> |
| 0x00000800<br/> | Это приложение потеряно при удалении политики. Если администратор удаляет политику, связанную с этим приложением, администратор больше не будет управлять развертыванием приложения, но установленное приложение продолжит работать.<br/>                          |
| 0x00001000<br/> | Это приложение удаляется при удалении политики развертывания.<br/>                                                                                                                                                                                                                              |
| 0x00002000<br/> | Будет выполнена полная установка назначенных пользователю приложений.<br/>                                                                                                                                                                                                                                |
| 0x00004000<br/> | Более старые версии этого приложения необходимо обновить до этой версии.<br/>                                                                                                                                                                                                                                |
| 0x00008000<br/> | Этот пакет поддерживает только минимальный пользовательский интерфейс с индикатором выполнения процесса установки.<br/>                                                                                                                                                                                               |
| 0x00010000<br/> | Это пакет для 32-разрядных версий Windows, которые не должны выполняться в Windows XP Professional x64 Edition или 64-разрядных версиях Windows Server 2003.<br/>                                                                                                                                      |
| 0x00020000<br/> | Этот пакет подходит для любого языка.<br/>                                                                                                                                                                                                                                                          |
| 0x00040000<br/> | Этот пакет содержит обновления.<br/>                                                                                                                                                                                                                                                                          |
| 0x00080000<br/> | Этот пакет содержит полный пользовательский интерфейс для процесса установки.<br/>                                                                                                                                                                                                                                |
| 0x00100000<br/> | Классы для этого приложения сохраняются во время операции повторного развертывания, если приложение развертывается повторно при переименовании домена.<br/>                                                                                                                                                                       |
