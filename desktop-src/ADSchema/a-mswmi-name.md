# <a name="ms-wmi-name-attribute"></a>атрибут MS-WMI-Name

Отображаемое имя для объектов политики верхнего уровня. Используется в глобальном каталоге.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-Name                                 |
| LDAP-отображаемое имя | Мсвми — имя                                  |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор групповая политика                  |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1639                     |
| System-ID — GUID    | c6c8ace5-7e81-42af-ad72-77412c5941c4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
