# <a name="print-rate-attribute"></a>Атрибут Print-Rate

Скорость печати, предоставляемая драйвером.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Print-Rate                           |
| LDAP-отображаемое имя | printRate                            |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.285               |
| System-ID — GUID    | ba305f77-47e3-11d0-a1a6-00c04fd930c9 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
