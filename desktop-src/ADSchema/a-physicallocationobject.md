# <a name="physical-location-object-attribute"></a>Атрибут физического расположения — объект

Используется для подключения устройства (например, принтера, компьютера и т. д.) к физическому расположению.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Физическое расположение — объект                |
| LDAP-отображаемое имя | physicalLocationObject                  |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.514                  |
| System-ID — GUID    | b7b13119-b82e-11d0-afee-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
