# <a name="ipsec-negotiation-policy-type-attribute"></a>Атрибут типа "IPSEC-согласование-политика"

Атрибут **IPSec-согласовать-Policy-Type** предназначен только для внутреннего использования.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | IPSEC-согласование — тип политики               |
| LDAP-отображаемое имя | ипсекнеготиатионполицитипе                  |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.887                      |
| System-ID — GUID    | 07383074-91df-11d1-aebc-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
