# <a name="remote-source-attribute"></a>Атрибут Remote-Source

Обратный указатель на внешние объекты.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Remote-Source                               |
| LDAP-отображаемое имя | ремотесаурце                                |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.107                      |
| System-ID — GUID    | bf967a14-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
