# <a name="ms-ds-members-for-az-role-attribute"></a>атрибут роли MS-DS-Members-for-AZ-Role

Список групп приложений-участников или пользователей, связанных с AZ-Role.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-Members-for-AZ-Role               |
| LDAP-отображаемое имя | msDS-Мемберсфоразроле                   |
| Размер              | \-                                      |
| Привилегия обновления  | Администратор Азролес                           |
| Частота обновления  | Во время инициализации или изменения политики. |
| Attribute-Id      | 1.2.840.113556.1.4.1806                 |
| System-ID — GUID    | cbf7e6cd-85a4-4314-8939-8bfe80597835    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
