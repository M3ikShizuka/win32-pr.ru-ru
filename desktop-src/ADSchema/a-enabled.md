# <a name="enabled-attribute"></a>Включенный атрибут

Этот атрибут используется для обозначения того, включен ли данный режим crossRef.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Активировано                              |
| LDAP-отображаемое имя | Активировано                              |
| Размер              | \-                                   |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.557               |
| System-ID — GUID    | a8df73f2-c5ea-11d1-bbcb-0080c76670c0 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
