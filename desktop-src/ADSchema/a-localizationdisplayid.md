# <a name="localization-display-id-attribute"></a>Атрибут Localization-ID локализации

Используется для индексации в файле Extrts.mc для получения локализованного displayName для объектов в целях пользовательского интерфейса.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Локализация — идентификатор экрана              |
| LDAP-отображаемое имя | локализатиондисплайид                |
| Размер              | 4 байта                              |
| Привилегия обновления  | Администратор схемы                 |
| Частота обновления  | Идентификатор никогда не должен изменяться.       |
| Attribute-Id      | 1.2.840.113556.1.4.1353              |
| System-ID — GUID    | a746f0d1-78d0-11d2-9916-0000f87a57d4 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
