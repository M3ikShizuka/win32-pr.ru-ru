# <a name="phone-mobile-other-attribute"></a>Телефон-Mobile — другой атрибут

Список альтернативных номеров мобильных телефонов.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Телефон-Mobile — другие                                                               |
| LDAP-отображаемое имя | otherMobile                                                                      |
| Размер              | \-                                                                               |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                           |
| Частота обновления  | При создании записи пользователя и при необходимости изменения номера телефона. |
| Attribute-Id      | 1.2.840.113556.1.4.647                                                           |
| System-ID — GUID    | 0296c11e-40da-11d1-a9c0-0000f80367c1                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
