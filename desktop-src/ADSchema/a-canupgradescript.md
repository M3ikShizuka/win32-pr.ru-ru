# <a name="can-upgrade-script-attribute"></a>Can-Upgrade-атрибут скрипта

Этот атрибут хранит список пакетов приложений, которые могут быть обновлены, или которые могут обновить этот пакет приложения.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Can-Upgrade-Скрипт                          |
| LDAP-отображаемое имя | канупградескрипт                            |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.815                      |
| System-ID — GUID    | d9e18314-8939-11d1-aebc-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
