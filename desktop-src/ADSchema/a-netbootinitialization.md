# <a name="netboot-initialization-attribute"></a>Атрибут Netboot-Initialization

Путь загрузки по умолчанию для бездисковой загрузки.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Netboot-Initialization                      |
| LDAP-отображаемое имя | нетбутинитиализатион                       |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.358                      |
| System-ID — GUID    | 3e978920-8c01-11d0-afda-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
