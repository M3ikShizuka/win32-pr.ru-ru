# <a name="inter-site-topology-failover-attribute"></a>Репликация между сайтами-топология — атрибут отработки отказа

Указывает, сколько времени должно пройти с момента последнего сохранения активности для генератора межсайтовой топологии, чтобы считаться неработающим.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Топология между сайтами — отработка отказа         |
| LDAP-отображаемое имя | интерситетопологифаиловер            |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1248              |
| System-ID — GUID    | b7c69e60-2cc7-11d2-854e-00a0c983f608 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
