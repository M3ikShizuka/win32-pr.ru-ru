# <a name="certificate-enrollment-extended-right"></a>Certificate-Enrollment расширенное право

| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Certificate-Enrollment               |
| Display-Name | Enroll (Регистрация)                               |
| Rights-GUID  | 0e10c968-78fb-11d2-90d4-00c04f79dc55 |
