# <a name="ms-iis-ftp-dir-attribute"></a>атрибут MS-IIS-FTP-dir

Этот атрибут определяет домашнюю папку пользователя по отношению к общей папке файлового сервера. Он используется совместно с MS-IIS-FTP-root для определения домашнего каталога пользователя FTP.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------------------------------------------------------|
| CN                | MS-IIS-FTP-dir                                                                                                                  |
| LDAP-отображаемое имя | Мсиис — Фтпдир                                                                                                                    |
| Размер              | 30-50 символов (15-25 символов Юникода для каждого свойства)                                                                   |
| Привилегия обновления  | Администратор домена & локальная система                                                                                             |
| Частота обновления  | Это свойство задается, когда администратор создает веб-сайт и настраивает изоляцию FTP. Это редко меняется после этого. |
| Attribute-Id      | 1.2.840.113556.1.4.1786                                                                                                         |
| System-ID — GUID    | 8a5c99e9-2230-46eb-b8e8-e59d712eb9ee                                                                                            |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                                                                     |
