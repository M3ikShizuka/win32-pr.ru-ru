# <a name="ms-ds-phonetic-last-name-attribute"></a>Атрибут ms-DS-фонетического последнего имени

Содержит фонетическое фамилию человека.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-фонетическое имя (фамилия)                    |
| LDAP-отображаемое имя | msDS-Фонетикластнаме                       |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1943                     |
| System-ID — GUID    | f217e4ec-0836-4b90-88af-2f5d4bbda2bc        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
