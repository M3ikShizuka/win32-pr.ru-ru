# <a name="service-instance-version-attribute"></a>Атрибут Service-instance-Version

Версия службы Winsock.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Версия-экземпляра службы                              |
| LDAP-отображаемое имя | сервицеинстанцеверсион                                |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.199                                |
| System-ID — GUID    | bf967a37-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
