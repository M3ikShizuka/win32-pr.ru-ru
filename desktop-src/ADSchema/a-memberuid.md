# <a name="memberuid-attribute"></a>атрибут Мемберуид

Содержит имена входа членов группы.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | мемберуид                            |
| LDAP-отображаемое имя | мемберуид                            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.12                     |
| System-ID — GUID    | 03dab236-672e-4f61-ab64-f77d2dc2ffab |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
