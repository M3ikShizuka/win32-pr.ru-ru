# <a name="ms-ds-creator-sid-attribute"></a>Атрибут MS-DS-Creator-SID

Идентификатор безопасности создателя объекта, содержащего этот атрибут.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-Creator-SID                    |
| LDAP-отображаемое имя | mS-DS-Креаторсид                     |
| Размер              | \-                                   |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | При создании нового объекта.    |
| Attribute-Id      | 1.2.840.113556.1.4.1410              |
| System-ID — GUID    | c5e60132-1480-11d3-91c1-0000f87a57d4 |
| Синтаксис            | [**Строка (SID)**](s-string-sid.md)  |
