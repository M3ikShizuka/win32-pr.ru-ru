# <a name="managed-objects-attribute"></a>Атрибут Managed-Objects

Содержит список объектов, управляемых пользователем. В списке перечислены объекты, для которых свойство managedBy свойства имеет значение этого пользователя. Каждый элемент в списке является связанной ссылкой на управляемый объект.



| Ввод | Значение |
|-------------------|------------------------------------------------------------------------------------|
| CN                | Managed-Objects                                                                    |
| LDAP-отображаемое имя | managedObjects                                                                     |
| Размер              | \-                                                                                 |
| Привилегия обновления  | Это значение задается системой.                                                   |
| Частота обновления  | При создании записи пользователя и при необходимости изменения управляемых объектов. |
| Attribute-Id      | 1.2.840.113556.1.4.654                                                             |
| System-ID — GUID    | 0296c124-40da-11d1-a9c0-0000f80367c1                                               |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md)                                            |
