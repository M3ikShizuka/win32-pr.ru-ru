# <a name="ipsec-policy-reference-attribute"></a>Атрибут "политика IPSec-ссылка"

Отличительное имя связанной политики безопасности протокола IP (IPsec).



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Справочник по IPsec-политикам                  |
| LDAP-отображаемое имя | ипсекполициреференце                    |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.517                  |
| System-ID — GUID    | b7b13118-b82e-11d0-afee-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
