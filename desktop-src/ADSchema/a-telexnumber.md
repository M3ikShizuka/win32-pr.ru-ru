# <a name="telex-number-attribute"></a>Атрибут Telex-Number

Список альтернативных номеров телекса.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Telex-Number                                                                     |
| LDAP-отображаемое имя | телекснумбер                                                                      |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 2.5.4.21                                                                         |
| System-ID — GUID    | bf967a4b-0de6-11d0-a285-00aa003049e2                                             |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md)                            |
