# <a name="frs-replica-set-guid-attribute"></a>Атрибут FRS-реплика Set-GUID

Идентификатор GUID, определяющий набор реплик FRS.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | FRS-Replica-Set-GUID                                  |
| LDAP-отображаемое имя | фрсрепликасетгуид                                     |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.533                                |
| System-ID — GUID    | 5245801a-ca6a-11d0-afff-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
