# <a name="msmq-sites-attribute"></a>Атрибут MSMQ-Sites

Список сайтов, к которым принадлежит компьютер (массив узлов Обжектгуидс, обычно не более одного GUID).



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Sites                                            |
| LDAP-отображаемое имя | мсмкситес                                             |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.927                                |
| System-ID — GUID    | 9a0dc32a-c100-11d1-bbc5-0080c76670c0                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
