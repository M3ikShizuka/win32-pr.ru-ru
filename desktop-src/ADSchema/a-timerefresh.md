# <a name="time-refresh-attribute"></a>Атрибут Time-Refresh

Этот атрибут имеет интервал, в течение которого запись ресурса, содержащаяся в интегрированной зоне Active Directory, должна быть обновлена для DNS-сервера. Интервал по умолчанию составляет 7 дней.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Time-Refresh                         |
| LDAP-отображаемое имя | тимерефреш                          |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.503               |
| System-ID — GUID    | ddac0cf1-af8f-11d0-afeb-00c04fd930c9 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
