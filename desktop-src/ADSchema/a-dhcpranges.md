# <a name="dhcp-ranges-attribute"></a>DHCP-атрибут Ranges

Атрибут **диапазонов DHCP** в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | DHCP-диапазоны                          |
| LDAP-отображаемое имя | дхкпранжес                           |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.707               |
| System-ID — GUID    | 963d2748-48be-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
