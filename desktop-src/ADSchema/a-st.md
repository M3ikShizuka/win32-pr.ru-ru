# <a name="state-or-province-name-attribute"></a>Атрибут State-или-провинция-Name

Имя штата или провинции пользователя.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | State-или-провинция-Name                                                           |
| LDAP-отображаемое имя | st                                                                               |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 2.5.4.8                                                                          |
| System-ID — GUID    | bf967a39-0de6-11d0-a285-00aa003049e2                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
