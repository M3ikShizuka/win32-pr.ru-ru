# <a name="from-entry-attribute"></a>Атрибут From-Entry

Это сконструированный атрибут, который имеет **значение true** , если объект доступен для записи, и **значение false** , если он доступен только для чтения, например экземпляр реплики GC.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | From-Entry                           |
| LDAP-отображаемое имя | фроментри                            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.910               |
| System-ID — GUID    | 9a7ad949-ca53-11d1-bbd0-0080c76670c0 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
