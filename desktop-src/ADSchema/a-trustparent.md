# <a name="trust-parent-attribute"></a>Атрибут Trust-Parent

Родительский элемент в иерархии доверия KERBEROS.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Trust-Parent                            |
| LDAP-отображаемое имя | трустпарент                             |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | При создании дочернего домена.         |
| Attribute-Id      | 1.2.840.113556.1.4.471                  |
| System-ID — GUID    | b000ea7a-a086-11d0-afdd-00c04fd930c9    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
