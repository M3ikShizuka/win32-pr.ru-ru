# <a name="ms-sql-database-attribute"></a>MS-SQL-атрибут базы данных

имя базы данных SQL Server, участвующей в репликации.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-SQL — база данных                             |
| LDAP-отображаемое имя | mS-SQL — база данных                             |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | При установке репликации.                  |
| Attribute-Id      | 1.2.840.113556.1.4.1393                     |
| System-ID — GUID    | d5a0dbdc-ccee-11d2-9993-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
