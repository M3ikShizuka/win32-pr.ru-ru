| LDAP-отображаемое имя | msDS-Реплвалуеметадата                      |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1708                     |
| System-ID — GUID    | 2f5c8145-e1bd-410b-8957-8bfa81d5acfd        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
