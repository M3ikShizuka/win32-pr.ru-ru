# <a name="dhcp-sites-attribute"></a>DHCP-sites, атрибут

Атрибут **DHCP-Sites** в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | DHCP-сайты                           |
| LDAP-отображаемое имя | дхкпситес                            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.708               |
| System-ID — GUID    | 963d2749-48be-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
