# <a name="ms-ts-profile-path-attribute"></a>атрибут MS-TS-Profile-Path

Путь к профилю служб терминалов указывает перемещаемый или обязательный путь к профилю, который будет использоваться при входе пользователя на сервер терминалов. Путь к профилю указан в следующем формате сетевого пути: **\\\\** _ServerName_ *_\\_* _профилесфолдернаме_ *_\\_* _имя пользователя_.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-TS-Profile-Path                          |
| LDAP-отображаемое имя | мстспрофилепас                             |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1976                     |
| System-ID — GUID    | e65c30db-316c-4060-a3a0-387b083f09cd        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
