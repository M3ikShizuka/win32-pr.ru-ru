# <a name="acs-minimum-delay-variation-attribute"></a>ACS-Минимальная-задержка-атрибут варианта

Атрибут **ACS-Minimum-Delay-variation** предназначен только для внутреннего использования. На основе RFC2210.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-Минимальная-задержка-вариант          |
| LDAP-отображаемое имя | аксминимумделайвариатион             |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1317              |
| System-ID — GUID    | 9c65329b-3b90-11d2-90cc-00c04fd91ab1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
