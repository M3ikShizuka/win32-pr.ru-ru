# <a name="ms-ds-replication-notify-subsequent-dsa-delay-attribute"></a>Атрибут ms-DS-Replication-notify-последующего-DSA-Delay

Этот атрибут управляет задержкой между уведомлениями каждого последующего участника реплики для NC.



| Ввод | Значение |
|-------------------|-----------------------------------------------|
| CN                | MS-DS-Replication-notify-последующий-DSA-Delay |
| LDAP-отображаемое имя | msDS-Replication-notify-последующий-DSA-Delay  |
| Размер              | \-                                            |
| Привилегия обновления  | Это значение задается системой.              |
| Частота обновления  | \-                                            |
| Attribute-Id      | 1.2.840.113556.1.4.1664                       |
| System-ID — GUID    | d63db385-dd92-4b52-b1d8-0d3ecc0e86b6          |
| Синтаксис            | [**Перечисление**](s-enumeration.md)          |
