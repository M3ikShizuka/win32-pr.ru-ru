# <a name="ms-dfs-namespace-identity-guid-v2-attribute"></a>атрибут MS-DFS-Namespace-Identity-GUID-v2

Задается только при создании пространства имен. Стабильны при переименовании или перемещении, пока пространство имен не заменяется другим пространством имен с таким же именем.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MS-DFS-Namespace-Identity-GUID-v2                     |
| LDAP-отображаемое имя | Мсдфс — NamespaceIdentityGUIDv2                         |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.2033                               |
| System-ID — GUID    | 200432ce-ec5f-4931-A525-d7f4afe34e68                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
