# <a name="mscope-id-attribute"></a>Атрибут Mscope-Id

Указывает, что на указанном DHCP-сервере присутствует многоадресная область.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Mscope-Id                            |
| LDAP-отображаемое имя | мскопеид                             |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.716               |
| System-ID — GUID    | 963d2751-48be-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
