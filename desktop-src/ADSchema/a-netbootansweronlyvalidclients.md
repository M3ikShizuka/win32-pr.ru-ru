# <a name="netboot-answer-only-valid-clients-attribute"></a>атрибут нетбут-Answer-only-Valid-Clients

Определяет, отвечает ли сервер всем или только предварительно подготовленным клиентским компьютерам.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | нетбут — только для ответов — допустимые — клиенты    |
| LDAP-отображаемое имя | нетбутансверонливалидклиентс        |
| Размер              | \-                                   |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.854               |
| System-ID — GUID    | 0738307b-91df-11d1-aebc-0000f80367c1 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
