# <a name="dmd-name-attribute"></a>Атрибут DMD-Name

Имя, используемое для задания секции схемы. Не используется AD.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | DMD-Name                                    |
| LDAP-отображаемое имя | дмднаме                                     |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.598                      |
| System-ID — GUID    | 167757b9-47f3-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
