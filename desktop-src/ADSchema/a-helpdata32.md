# <a name="help-data32-attribute"></a>Атрибут Help-Data32

этот атрибут использовался в формате файла справки Win32 для Exchange 4,0. Он не используется для других версий Exchange.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Help-Data32                                           |
| LDAP-отображаемое имя | helpData32                                            |
| Размер              | \-                                                    |
| Привилегия обновления  | Используется системой.                           |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.9                                  |
| System-ID — GUID    | 5fd424a8-1262-11d0-a060-00aa006c33ed                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
