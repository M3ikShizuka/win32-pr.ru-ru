# <a name="friendly-names-attribute"></a>Атрибут Friendly-Names

Список определений отображаемого имени по умолчанию, поддерживаемых каталогом.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Friendly-Names                              |
| LDAP-отображаемое имя | фриендлинамес                               |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.682                      |
| System-ID — GUID    | 7bfdcb88-4807-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
