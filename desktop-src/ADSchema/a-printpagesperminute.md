# <a name="print-pages-per-minute-attribute"></a>Атрибут print-Pages-per-minute

Скорость печати, предоставляемая драйвером, в страницах в минуту.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Печать-страниц в минуту               |
| LDAP-отображаемое имя | printPagesPerMinute                  |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.631               |
| System-ID — GUID    | 19405b97-3cfa-11d1-a9c0-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
