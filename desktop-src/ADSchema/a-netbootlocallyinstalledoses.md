# <a name="netboot-locally-installed-oses-attribute"></a>нетбут — локально установленный атрибут ОС

Атрибут **нетбут-локально-installed-ОС** зарезервирован для внутреннего использования.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | нетбут — локально установленные — ОС              |
| LDAP-отображаемое имя | нетбутлокаллинсталледосес                 |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.859                      |
| System-ID — GUID    | 07383080-91df-11d1-aebc-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
