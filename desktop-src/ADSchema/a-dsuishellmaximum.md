# <a name="ds-ui-shell-maximum-attribute"></a>Атрибут DS-UI-Shell-Maximum

Это максимальное число объектов, которое будет отображаться в контейнере ИНТЕРФЕЙСом оболочки по умолчанию.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | DS-UI-Shell-максимум                  |
| LDAP-отображаемое имя | дсуишеллмаксимум                     |
| Размер              | \-                                   |
| Привилегия обновления  | Администратор домена                 |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1345              |
| System-ID — GUID    | fcca766a-6f91-11d2-9905-0000f87a57d4 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
