# <a name="ms-sql-allowanonymoussubscription-attribute"></a>Атрибут MS-SQL-Аллованонимауссубскриптион

Значение true, если публикация разрешает анонимным подписчикам подписываться на него.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-SQL-Аллованонимауссубскриптион    |
| LDAP-отображаемое имя | mS-SQL-Аллованонимауссубскриптион    |
| Размер              | 4 байта                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | При установке репликации.           |
| Attribute-Id      | 1.2.840.113556.1.4.1394              |
| System-ID — GUID    | db77be4a-ccee-11d2-9993-0000f87a57d4 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
