Содержит время существования прокси-объекта.

|-------------------|--------------------------------------|
| CN                | Proxy-Lifetime                       |
| LDAP-отображаемое имя | проксилифетиме                        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.103               |
| System-ID — GUID    | bf967a07-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
