# <a name="inter-site-topology-generator-attribute"></a>Атрибут межсайтовой топологии — генератор

Этот атрибут будет использоваться для поддержки отработки отказа на компьютере, обозначенном как тот, который выполняет создание межсайтовой топологии с помощью средства проверки согласованности знаний на данном сайте.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Генератор межсайтовых топологий           |
| LDAP-отображаемое имя | интерситетопологиженератор              |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1246                 |
| System-ID — GUID    | b7c69e5e-2cc7-11d2-854e-00a0c983f608    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |



## <a name="implementations"></a>Варианты реализации решения
