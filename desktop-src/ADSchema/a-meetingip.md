# <a name="meetingip-attribute"></a>атрибут Митингип

Адрес TCP/IP для собрания.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | митингип                                                                        |
| LDAP-отображаемое имя | митингип                                                                        |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.580                                                           |
| System-ID — GUID    | 11b6cc89-48c4-11d1-a9c3-0000f80367c1                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
