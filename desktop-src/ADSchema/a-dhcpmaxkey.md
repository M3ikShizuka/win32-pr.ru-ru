# <a name="dhcp-maxkey-attribute"></a>атрибут DHCP-Макскэй

Атрибут **DHCP-макскэй** в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | DHCP-Макскэй                          |
| LDAP-отображаемое имя | дхкпмакскэй                           |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.719               |
| System-ID — GUID    | 963d2754-48be-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
