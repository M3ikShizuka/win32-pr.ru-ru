# <a name="ms-ds-logon-time-sync-interval-attribute"></a>MS-DS-вход-время-синхронизация-атрибут интервала

Этот атрибут определяет степень гранулярности (в днях), с которой время последнего входа пользователя или компьютера, записанное в атрибут lastLogonTimestamp, реплицируется на все контроллеры домена в домене.



| Ввод | Значение |
|-------------------|------------------------------------------------------------------------------------------------------------|
| CN                | MS-DS-вход в систему-время-синхронизация-интервал                                                                             |
| LDAP-отображаемое имя | msDS-ЛогонтимесинЦинтервал                                                                                 |
| Размер              | 4 байта                                                                                                    |
| Привилегия обновления  | Администратор домена                                                                                       |
| Частота обновления  | В редких случаях, поскольку это параметр политики, он обновляется только в том случае, если требуется изменить политику на уровне домена. |
| Attribute-Id      | 1.2.840.113556.1.4.1784                                                                                    |
| System-ID — GUID    | ad7940f8-e43a-4a42-83bc-d688e59ea605                                                                       |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                                                                       |
