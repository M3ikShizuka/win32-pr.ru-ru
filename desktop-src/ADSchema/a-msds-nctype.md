| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-NC-Type                        |
| LDAP-отображаемое имя | msDS-Нктипе                          |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2024              |
| System-ID — GUID    | 5a2eacd7-cc2b-48cf-9d9a-b6f1a0024de9 |
