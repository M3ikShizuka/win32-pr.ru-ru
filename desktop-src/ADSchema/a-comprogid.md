# <a name="com-progid-attribute"></a>Атрибут COM-ProgID

Этот атрибут хранит список идентификаторов программных объектов COM, реализованных в этом пакете приложения.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | COM-ProgID                                                                       |
| LDAP-отображаемое имя | компрогид                                                                        |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.21                                                            |
| System-ID — GUID    | bf96793d-0de6-11d0-a285-00aa003049e2                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
