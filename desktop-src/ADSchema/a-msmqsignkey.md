|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Sign-ключ                                         |
| LDAP-отображаемое имя | мсмксигнкэй                                           |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.937                                |
| System-ID — GUID    | 9a0dc332-c100-11d1-bbc5-0080c76670c0                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
