# <a name="msmq-site-foreign-attribute"></a>Атрибут MSMQ-site-инородный

Логическое значение, указывающее, является ли это инородным сайтом MSMQ.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MSMQ-site-инородный                    |
| LDAP-отображаемое имя | мсмкситефореигн                      |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.961               |
| System-ID — GUID    | fd129d8a-d57e-11d1-90a2-00c04fd91ab1 |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
