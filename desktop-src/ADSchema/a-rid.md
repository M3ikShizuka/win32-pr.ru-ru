# <a name="rid-attribute"></a>Атрибут RID

Относительный идентификатор объекта.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Избежать                                         |
| LDAP-отображаемое имя | rid                                         |
| Размер              | 4 байта                                     |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | При создании объекта, которому требуется RID. |
| Attribute-Id      | 1.2.840.113556.1.4.153                      |
| System-ID — GUID    | bf967a22-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**Перечисление**](s-enumeration.md)        |
