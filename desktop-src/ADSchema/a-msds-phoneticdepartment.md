# <a name="ms-ds-phonetic-department-attribute"></a>Атрибут ms-DS-фонетического отдела

Содержит фонетическое название отдела, в котором работает пользователь.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-фонетический отдел                   |
| LDAP-отображаемое имя | msDS-Фонетикдепартмент                     |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1944                     |
| System-ID — GUID    | 6cd53daf-003e-49e7-a702-6fa896e7a6ef        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
