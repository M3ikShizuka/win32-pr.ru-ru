# <a name="postal-code-attribute"></a>Атрибут Postal-Code

Почтовый индекс для доставки почты.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------------------|
| CN                | Postal-Code                                                                 |
| LDAP-отображаемое имя | postalCode                                                                  |
| Размер              | \-                                                                          |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                      |
| Частота обновления  | При создании записи пользователя и при необходимости изменения адреса. |
| Attribute-Id      | 2.5.4.17                                                                    |
| System-ID — GUID    | bf9679fd-0de6-11d0-a285-00aa003049e2                                        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                 |
