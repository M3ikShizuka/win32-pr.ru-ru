# <a name="ms-ds-keyversionnumber-attribute"></a>Атрибут ms-DS-Кэйверсионнумбер

Номер версии Kerberos текущего ключа для этой учетной записи. Это сконструированный атрибут.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-Кэйверсионнумбер               |
| LDAP-отображаемое имя | msDS-Кэйверсионнумбер                |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1782              |
| System-ID — GUID    | c523e9c0-33b5-4ac8-8923-b57b927f42f6 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
