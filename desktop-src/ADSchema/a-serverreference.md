# <a name="server-reference-attribute"></a>Атрибут Server-Reference

Найдено в объекте компьютера сайта. Содержит различающееся имя контроллера домена в контексте именования домена.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Server-Reference                        |
| LDAP-отображаемое имя | серверреференце                         |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | При создании сайта.             |
| Attribute-Id      | 1.2.840.113556.1.4.515                  |
| System-ID — GUID    | 26d9736d-6070-11d1-a9c6-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
