# <a name="ms-pki-oid-cps-attribute"></a>атрибут MS-PKI-OID-CPS

CPS (инструкция политики сертификата) для OID политики издателя предприятия.



| Ввод | Значение |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | MS-PKI-OID-CPS                                                                             |
| LDAP-отображаемое имя | Мспки-OID-CPS                                                                              |
| Размер              | \-                                                                                         |
| Привилегия обновления  | Администратор предприятия                                                                   |
| Частота обновления  | При создании нового шаблона или изменении атрибутов существующих шаблонов. |
| Attribute-Id      | 1.2.840.113556.1.4.1672                                                                    |
| System-ID — GUID    | 5f49940e-a79f-4a51-bb6f-3d446a54dc6b                                                       |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                                |
