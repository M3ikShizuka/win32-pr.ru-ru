# <a name="logon-hours-attribute"></a>Атрибут Logon-Hours

Часы, в которые пользователю разрешено выполнять вход в домен.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Logon-Hours                                           |
| LDAP-отображаемое имя | логонхаурс                                            |
| Размер              | \-                                                    |
| Привилегия обновления  | Администратор домена                                  |
| Частота обновления  | Каждый раз, когда необходимо изменить время входа в учетную запись.   |
| Attribute-Id      | 1.2.840.113556.1.4.64                                 |
| System-ID — GUID    | bf9679ab-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
