# <a name="ms-wmi-normalizedclass-attribute"></a>атрибут MS-WMI-Нормализедкласс

Имя основного класса политики WMI.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-Нормализедкласс                      |
| LDAP-отображаемое имя | Мсвми — Нормализедкласс                       |
| Размер              | Менее двадцати символов.                |
| Привилегия обновления  | Администратор групповая политика                  |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1640                     |
| System-ID — GUID    | eaba628f-eb8e-4fe9-83fc-693be695559b        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
