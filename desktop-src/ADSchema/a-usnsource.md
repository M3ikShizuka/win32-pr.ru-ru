# <a name="usn-source-attribute"></a>Атрибут USN-Source

Значение атрибута [**USN-Changed**](a-usnchanged.md) объекта из удаленного каталога, который реплицирует изменение на локальный сервер.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------------------------------------------|
| CN                | USN-Source                                                                                          |
| LDAP-отображаемое имя | уснсаурце                                                                                           |
| Размер              | 8 байт                                                                                             |
| Привилегия обновления  | Это значение задается системой.                                                                    |
| Частота обновления  | При изменении объекта в удаленном каталоге, который реплицирует изменения в локальный каталог. |
| Attribute-Id      | 1.2.840.113556.1.4.896                                                                              |
| System-ID — GUID    | 167758ad-47f3-11d1-a9c3-0000f80367c1                                                                |
| Синтаксис            | [**Интервал**](s-interval.md)                                                                      |
