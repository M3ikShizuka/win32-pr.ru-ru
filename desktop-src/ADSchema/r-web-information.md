# <a name="web-information-property-set"></a>Набор свойств Web-Information

Набор свойств, содержащий атрибуты пользователя, которые описывают сведения, связанные с веб-пользователем.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Web-Information                      |
| Display-Name | Веб-информация                      |
| Rights-GUID  | e45795b3-9455-11d1-aebd-0000f80367c1 |
