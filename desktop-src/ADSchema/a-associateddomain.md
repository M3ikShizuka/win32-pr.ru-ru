# <a name="associateddomain-attribute"></a>атрибут АссоЦиатеддомаин

Тип атрибута АссоЦиатеддомаин указывает домен DNS, связанный с объектом.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ассоЦиатеддомаин                     |
| LDAP-отображаемое имя | ассоЦиатеддомаин                     |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 0.9.2342.19200300.100.1.37           |
| System-ID — GUID    | 3320fc38-c379-4c17-a510-1bdf6133c5da |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
