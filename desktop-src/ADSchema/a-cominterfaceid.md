# <a name="com-interfaceid-attribute"></a>Атрибут COM-InterfaceID

Этот атрибут хранит список интерфейсов, реализованных в этом пакете приложения.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | COM-InterfaceID                                                                  |
| LDAP-отображаемое имя | коминтерфацеид                                                                   |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.20                                                            |
| System-ID — GUID    | bf96793c-0de6-11d0-a285-00aa003049e2                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
