Ссылки на объекты в других пространствах имен ADSI.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Wbem-Path                                   |
| LDAP-отображаемое имя | вбемпас                                    |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.301                      |
| System-ID — GUID    | 244b2970-5abd-11d0-afd2-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
