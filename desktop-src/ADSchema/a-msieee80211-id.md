# <a name="ms-ieee-80211-id-attribute"></a>MS-IEEE-80211-атрибут ID

Идентификатор, используемый для объекта политики беспроводной связи в AD.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | MS-IEEE-80211-ID                                                                 |
| LDAP-отображаемое имя | msieee80211 — идентификатор                                                                   |
| Размер              | \-                                                                               |
| Привилегия обновления  | Администратор домена                                                             |
| Частота обновления  | Каждый раз, когда администратор домена изменяет политику беспроводной сети для домена или подразделения. |
| Attribute-Id      | 1.2.840.113556.1.4.1823                                                          |
| System-ID — GUID    | 7f73ef75-14c9-4c23-81de-dd07a06f9e8b                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
