# <a name="dhcp-servers-attribute"></a>атрибут DHCP-Servers

Содержит список серверов, разрешенных на предприятии.



| Ввод | Значение |
|-------------------|--------------------------------------------------------|
| CN                | DHCP-серверы                                           |
| LDAP-отображаемое имя | дхкпсерверс                                            |
| Размер              | \-                                                     |
| Привилегия обновления  | Администратор домена.                                  |
| Частота обновления  | При добавлении или удалении нового сервера из леса. |
| Attribute-Id      | 1.2.840.113556.1.4.704                                 |
| System-ID — GUID    | 963d2745-48be-11d1-a9c3-0000f80367c1                   |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)                    |
