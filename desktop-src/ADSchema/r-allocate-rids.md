# <a name="allocate-rids-extended-right"></a>Allocate-Rids расширенное право

Расширенное право, необходимое для запроса пула RID.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Allocate-Rids                        |
| Display-Name | Выделить идентификаторы RID                        |
| Rights-GUID  | 1abd7cf8-0a99-11d1-adbb-00c04fd8d5cd |
