| LDAP-отображаемое имя | catalogs                                    |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.675                      |
| System-ID — GUID    | 7bfdcb81-4807-11d1-a9c3-0000f80367c1        |
