# <a name="object-count-attribute"></a>Атрибут Object-Count

Квота на записанные файлы для заданного тома.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Object-Count                         |
| LDAP-отображаемое имя | objectCount                          |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.506               |
| System-ID — GUID    | 34aaa216-b699-11d0-afee-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
