# <a name="ms-net-ieee-8023-gp-policyguid-attribute"></a>атрибут MS-NET-IEEE-8023-GP-Полицигуид

Содержит идентификатор GUID, определяющий конкретный объект групповая политика 802,3 в домене.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-NET-IEEE-8023-GP-Полицигуид              |
| LDAP-отображаемое имя | MS-NET-IEEE-8023-GP-Полицигуид              |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1954                     |
| System-ID — GUID    | 94a7b05a-b8b2-4f59-9c25-39e69baa1684        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
