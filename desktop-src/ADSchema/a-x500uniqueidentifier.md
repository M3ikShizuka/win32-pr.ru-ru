# <a name="x500uniqueidentifier-attribute"></a>атрибут x500uniqueIdentifier

Используется для различения объектов при повторном использовании различающегося имени. Это другой тип атрибута из типов UID и uniqueIdentifier.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | x500uniqueIdentifier                                  |
| LDAP-отображаемое имя | x500uniqueIdentifier                                  |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 2.5.4.45                                              |
| System-ID — GUID    | d07da11f-8a3d-42b6-b0aa-76c962be719a                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
