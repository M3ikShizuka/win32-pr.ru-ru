# <a name="frs-control-data-creation-attribute"></a>Атрибут создания данных FRS-Control-Data

Пара "предупреждение" или "уровень ошибки" для создания файловых данных (в мегабайтах в секунду).



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | FRS-Control-создание данных                   |
| LDAP-отображаемое имя | фрсконтролдатакреатион                      |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.871                      |
| System-ID — GUID    | 2a13257a-9373-11d1-aebc-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
