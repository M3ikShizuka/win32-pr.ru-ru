# <a name="ms-ts-reconnection-action-attribute"></a>MS-TS-повторное подключение — атрибут действия

Действие повторного подключения сеанса служб терминалов указывает, разрешено ли повторное подключение к отключенному сеансу служб терминалов с любого клиентского компьютера.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-TS-повторное подключение — действие            |
| LDAP-отображаемое имя | мстсреконнектионактион               |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1984              |
| System-ID — GUID    | 366ed7ca-3e18-4c7f-abae-351a01e4b4f7 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
