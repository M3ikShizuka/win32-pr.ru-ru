# <a name="additional-trusted-service-names-attribute"></a>Атрибут "дополнительно-Trusted-Service-Names"

Список служб в домене, которые могут быть доверенными. Не используется AD.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Дополнительные-доверенные имена служб            |
| LDAP-отображаемое имя | аддитионалтрустедсервиценамес               |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.889                      |
| System-ID — GUID    | 032160be-9824-11d1-aec0-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
