Следующее поле RID, используемое распределителем смешанного режима.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Next-RID                             |
| LDAP-отображаемое имя | некстрид                              |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.88                |
| System-ID — GUID    | bf9679db-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
