# <a name="jpegphoto-attribute"></a>атрибут Жпегфото

Используется для хранения одного или нескольких изображений человека, использующего формат обмена файлами JPEG \[ JFIF \] .



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | жпегфото                                             |
| LDAP-отображаемое имя | жпегфото                                             |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 0.9.2342.19200300.100.1.60                            |
| System-ID — GUID    | bac80572-09c4-4fa9-9ae6-7628d7adbe0e                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
