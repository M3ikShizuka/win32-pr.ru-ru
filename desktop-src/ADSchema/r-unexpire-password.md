# <a name="unexpire-password-extended-right"></a>Unexpire-Password расширенное право

Право на расширенное управление доступом, позволяющее пользователю восстановить пароль с истекшим сроком действия для объекта пользователя.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | отмена истечения срока пароля;                    |
| Display-Name | Пароль с истекшим сроком действия                    |
| Rights-GUID  | ccc2dc7d-a6ad-4a7a-8846-c04e3cc53501 |
