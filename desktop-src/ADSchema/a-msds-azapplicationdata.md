# <a name="ms-ds-az-application-data-attribute"></a>Атрибут ms-DS-AZ-Application-Data

| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-AZ-Application-Data                   |
| LDAP-отображаемое имя | msDS-Азаппликатиондата                      |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1819                     |
| System-ID — GUID    | 503fc3e8-1cc6-461a-99a3-9eee04f402a7        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
