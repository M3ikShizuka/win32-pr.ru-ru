# <a name="documentauthor-attribute"></a>атрибут Документаусор

Тип атрибута Документаусор указывает различающееся имя автора документа.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | документаусор                          |
| LDAP-отображаемое имя | документаусор                          |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 0.9.2342.19200300.100.1.14              |
| System-ID — GUID    | f18a8e19-af5f-4478-b096-6f35c27eb83f    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
