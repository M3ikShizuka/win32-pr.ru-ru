# <a name="server-reference-bl-attribute"></a>Атрибут Server-Reference-BL

Найдено в контексте именования домена. Различающееся имя компьютера в папке sites.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Server-Reference-BL                     |
| LDAP-отображаемое имя | серверреференцебл                       |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.516                  |
| System-ID — GUID    | 26d9736e-6070-11d1-a9c6-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
