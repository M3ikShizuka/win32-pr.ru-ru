# <a name="bridgehead-transport-list-attribute"></a>-Плацдарм-атрибут списка

Транспорты, для которых этот сервер является плацдармом.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Плацдарм-Transport-List               |
| LDAP-отображаемое имя | бриджехеадтранспортлист                 |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | При настройке сайта.               |
| Attribute-Id      | 1.2.840.113556.1.4.819                  |
| System-ID — GUID    | d50c2cda-8951-11d1-aebc-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
