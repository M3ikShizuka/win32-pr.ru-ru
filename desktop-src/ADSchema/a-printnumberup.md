# <a name="print-number-up-attribute"></a>Print — атрибут-up

Число изображений страниц на листе.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Печать — нумерация                      |
| LDAP-отображаемое имя | принтнумберуп                        |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.290               |
| System-ID — GUID    | 3bcbfcf4-4d3d-11d0-a1a6-00c04fd930c9 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
