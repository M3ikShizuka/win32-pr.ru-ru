# <a name="sync-with-object-attribute"></a>Sync — атрибут объекта

Различающееся имя синхронизированного объекта для синхронизации встроенной группы SAM или локальной политики.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Синхронизация с помощью объекта                        |
| LDAP-отображаемое имя | синквисобжект                          |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.664                  |
| System-ID — GUID    | 037651e2-441d-11d1-a9c3-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
