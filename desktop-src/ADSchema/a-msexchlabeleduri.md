# <a name="ms-exch-labeleduri-attribute"></a>атрибут MS-дов-Лабеледури

URI Exchange, за которым следует метка. Метка используется для описания ресурса, на который указывает URI, и предназначенного для удобного для восприятия имени.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-дов-Лабеледури                          |
| LDAP-отображаемое имя | мсексчлабеледури                            |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.593                      |
| System-ID — GUID    | 16775820-47f3-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
