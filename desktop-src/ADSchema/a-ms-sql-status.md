# <a name="ms-sql-status-attribute"></a>MS-SQL — атрибут Status

состояние сервера SQL. Состояние имеет **значение true** , если сервер работает.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-SQL — состояние                        |
| LDAP-отображаемое имя | mS-SQL — состояние                        |
| Размер              | 8 байт                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | При запуске системы.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1380              |
| System-ID — GUID    | 9a7d4770-ccee-11d2-9993-0000f87a57d4 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
