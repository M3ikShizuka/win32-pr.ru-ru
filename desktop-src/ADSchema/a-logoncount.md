# <a name="logon-count-attribute"></a>Атрибут Logon-Count

Количество раз, когда учетная запись была успешно зарегистрирована. Значение 0 указывает, что значение неизвестно.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Logon-Count                          |
| LDAP-отображаемое имя | логонкаунт                           |
| Размер              | 4 байта                              |
| Привилегия обновления  | Администратор домена                 |
| Частота обновления  | Каждый раз при входе пользователя в систему.          |
| Attribute-Id      | 1.2.840.113556.1.4.169               |
| System-ID — GUID    | bf9679aa-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |

## <a name="remarks"></a>Комментарии

Этот атрибут не реплицируется и поддерживается на каждом контроллере домена в домене. Чтобы получить точное значение общего числа успешных попыток входа в домен, необходимо запрашивать каждый контроллер домена в домене и использовать сумму значений. Помните, что атрибут не реплицируется, поэтому контроллеры домена, которые были удалены, могут подсчитать входные данные для пользователя, и они будут отсутствовать в счетчике.

> [!IMPORTANT]
> Из-за совместимости с 16-разрядными версиями диспетчера LAN атрибут имеет верхний предел 65535. После достижения этого предела вы не сможете использовать его в качестве индикатора активности пользователей на этом контроллере домена.
