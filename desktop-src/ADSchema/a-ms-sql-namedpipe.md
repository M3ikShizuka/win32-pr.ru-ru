# <a name="ms-sql-namedpipe-attribute"></a>Атрибут MS-SQL-помощью канала NamedPipe

Соединение именованного канала.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-помощью канала NamedPipe                            |
| LDAP-отображаемое имя | mS-SQL-помощью канала NamedPipe                            |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | При запуске системы.                          |
| Attribute-Id      | 1.2.840.113556.1.4.1374                     |
| System-ID — GUID    | 7b91c840-ccee-11d2-9993-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
