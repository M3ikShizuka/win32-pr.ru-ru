# <a name="usn-dsa-last-obj-removed-attribute"></a>Атрибут USN-DSA-Last-obj-removed

Содержит порядковый номер обновления (USN) для последнего системного объекта, который был удален с сервера.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | USN-DSA-Last-obj-removed             |
| LDAP-отображаемое имя | усндсаластобжремовед                 |
| Размер              | 8 байт                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | При каждом изменении объекта каталога. |
| Attribute-Id      | 1.2.840.113556.1.2.267               |
| System-ID — GUID    | bf967a71-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
