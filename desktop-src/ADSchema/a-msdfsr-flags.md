| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Флаги MS-DFSR-                        |
| LDAP-отображаемое имя | Мсдфср — флаги                         |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.16           |
| System-ID — GUID    | fe515695-3f61-45c8-9bfa-19c148c57b09 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
