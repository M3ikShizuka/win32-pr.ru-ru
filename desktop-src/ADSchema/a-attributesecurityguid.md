# <a name="attribute-security-guid-attribute"></a>Атрибут-Security-GUID

Идентификатор GUID, используемый для применения учетных данных безопасности к набору объектов.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Attribute-Security-GUID                               |
| LDAP-отображаемое имя | attributeSecurityGUID                                 |
| Размер              | \-                                                    |
| Привилегия обновления  | Администратор схемы                                  |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.149                                |
| System-ID — GUID    | bf967924-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
