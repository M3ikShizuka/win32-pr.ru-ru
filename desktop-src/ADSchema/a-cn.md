# <a name="common-name-attribute"></a>Атрибут Common-Name

Имя, представляющее объект. Используется для выполнения поиска.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Common-Name                                 |
| LDAP-отображаемое имя | cn                                          |
| Размер              | Средний размер — 64 байт.                   |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | Только при создании нового объекта.          |
| Attribute-Id      | 2.5.4.3                                     |
| System-ID — GUID    | bf96793f-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
