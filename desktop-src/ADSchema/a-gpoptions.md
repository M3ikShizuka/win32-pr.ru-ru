# <a name="gp-options-attribute"></a>Атрибут GP-Options

| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | GP-Options                           |
| LDAP-отображаемое имя | гпоптионс                            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.892               |
| System-ID — GUID    | f30e3bbf-9ff0-11d1-b603-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
