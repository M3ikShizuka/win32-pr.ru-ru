# <a name="x121-address-attribute"></a>Атрибут X121-Address

| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | X121-Address                                |
| LDAP-отображаемое имя | x121Address                                 |
| Размер              | 15 байт                                    |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 2.5.4.24                                    |
| System-ID — GUID    | bf967a7b-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(числовой)**](s-string-numeric.md) |
