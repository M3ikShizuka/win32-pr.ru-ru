# <a name="acs-non-reserved-min-policed-size-attribute"></a>ACS-не зарезервированный-min-Attribute-атрибут size

Атрибут **ACS-не зарезервирован-min-с заданной функцией-size** предназначен только для внутреннего использования. На основе RFC2814.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-не зарезервировано-min-with-size    |
| LDAP-отображаемое имя | акснонресерведминполицедсизе         |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1321              |
| System-ID — GUID    | b6873917-3b90-11d2-90cc-00c04fd91ab1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
