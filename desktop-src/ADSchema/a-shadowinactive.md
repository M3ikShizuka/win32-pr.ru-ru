# <a name="shadowinactive-attribute"></a>атрибут Шадовинактиве

Содержит количество дней бездействия, разрешенное для пользователя.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | шадовинактиве                       |
| LDAP-отображаемое имя | шадовинактиве                       |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.9                      |
| System-ID — GUID    | 86871d1f-3310-4312-8efd-af49dcfb2671 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
