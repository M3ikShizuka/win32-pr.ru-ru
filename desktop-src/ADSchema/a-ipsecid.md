# <a name="ipsec-id-attribute"></a>Атрибут идентификатора IPSec

Атрибут **IPSec-ID** предназначен только для внутреннего использования.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Идентификатор IPSec                                    |
| LDAP-отображаемое имя | ипсеЦид                                     |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.621                      |
| System-ID — GUID    | b40ff81d-427a-11d1-a9c2-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
