Идентификаторы объектов, экспортированные указанным сервером.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | RPC-NS-Object-ID                            |
| LDAP-отображаемое имя | рпкнсобжектид                               |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.312                      |
| System-ID — GUID    | 29401c48-7a27-11d0-afd6-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
