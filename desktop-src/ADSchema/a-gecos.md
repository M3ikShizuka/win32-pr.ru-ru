# <a name="gecos-attribute"></a>атрибут жекос

Содержит сведения, которые хранятся в поле ЖЕКОС.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | жекос                                |
| LDAP-отображаемое имя | жекос                                |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.2                      |
| System-ID — GUID    | a3e03f1f-1d55-4253-a0af-30c2a784e46e |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
