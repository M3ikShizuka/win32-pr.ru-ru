# <a name="msmq-owner-id-attribute"></a>Атрибут MSMQ-Owner-ID

Атрибут **MSMQ-Owner-ID** содержит сведения о смешанном режиме MSMQ.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Owner-ID                                         |
| LDAP-отображаемое имя | мсмковнерид                                           |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.925                                |
| System-ID — GUID    | 9a0dc328-c100-11d1-bbc5-0080c76670c0                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
