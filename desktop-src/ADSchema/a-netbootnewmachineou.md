# <a name="netboot-new-machine-ou-attribute"></a>нетбут — атрибут "создать-компьютер-OU"

Указывает, где будет создана учетная запись нового клиентского компьютера.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | нетбут-New-Machine-OU                  |
| LDAP-отображаемое имя | нетбутневмачинеау                     |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.856                  |
| System-ID — GUID    | 0738307d-91df-11d1-aebc-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
