# <a name="associatedname-attribute"></a>атрибут АссоЦиатеднаме

Тип атрибута АссоЦиатеднаме указывает запись в Организации DIT, связанную с доменом DNS.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | ассоЦиатеднаме                          |
| LDAP-отображаемое имя | ассоЦиатеднаме                          |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 0.9.2342.19200300.100.1.38              |
| System-ID — GUID    | f7fbfc45-85ab-42a4-a435-780e62f7858b    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
