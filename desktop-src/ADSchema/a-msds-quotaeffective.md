# <a name="ms-ds-quota-effective-attribute"></a>MS-DS-Quota-действующий атрибут

Эффективная квота субъекта безопасности, вычисленная на основе назначенных квот для раздела каталога.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-Quota-эффективная                |
| LDAP-отображаемое имя | msDS-Куотаеффективе                  |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1848              |
| System-ID — GUID    | 6655b152-101c-48b4-b347-e1fcebc60157 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
