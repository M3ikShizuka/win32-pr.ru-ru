# <a name="generate-rsop-planning-extended-right"></a>Создание-RSoP-планирование расширенного права

Пользователь, имеющий права в подразделении или домене, сможет создавать данные RSoP в режиме планирования для пользователей или компьютеров в подразделении.



| Ввод | Значение |
|--------------|---------------------------------------------|
| CN           | Создание-RSoP-планирование                      |
| Display-Name | Генерация RSoP (Планирование) |
| Rights-GUID  | b7b1b3dd-ab09-4242-9e30-9980e5d322f7        |
