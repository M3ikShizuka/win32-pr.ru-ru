# <a name="update-password-not-required-bit-extended-right"></a>Обновление-пароль — необязательный — бит расширенное право

Право на расширенное управление доступом, позволяющее пользователю включать или отключать параметр "пароль не требуется" для объектов "пользователь".



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Обновление-пароль — необязательный бит     |
| Display-Name | Бит обновления пароля не требуется     |
| Rights-GUID  | 280f369c-67c7-438e-ae98-1d46f3c6f541 |
