# <a name="transport-address-attribute-attribute"></a>Атрибут-адрес транспорта

Имя типа адреса для транспорта.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------|
| CN                | Адрес-атрибут транспорта                                     |
| LDAP-отображаемое имя | транспортаддрессаттрибуте                                       |
| Размер              | \-                                                              |
| Привилегия обновления  | Это значение задается системой.                                |
| Частота обновления  | При подключении сайтов.                                          |
| Attribute-Id      | 1.2.840.113556.1.4.895                                          |
| System-ID — GUID    | c1dc867c-a261-11d1-b606-0000f80367c1                            |
| Синтаксис            | [**String(идентификатор_объекта)**](s-string-object-identifier.md) |
