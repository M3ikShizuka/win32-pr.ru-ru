# <a name="acs-max-peak-bandwidth-attribute"></a>Атрибут ACS-max-Пиковая пропускная способность

| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-max-Пиковая пропускная способность               |
| LDAP-отображаемое имя | аксмакспеакбандвидс                  |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.767               |
| System-ID — GUID    | 7f561284-5301-11d1-a9c5-0000f80367c1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
