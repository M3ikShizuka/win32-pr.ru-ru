# <a name="ms-sql-alias-attribute"></a>MS-SQL-псевдоним атрибута

Псевдоним, используемый для подключения к базе данных. Значение по умолчанию — Alias.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Псевдоним MS-SQL                                |
| LDAP-отображаемое имя | Псевдоним mS-SQL                                |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1395                     |
| System-ID — GUID    | e0c6baae-ccee-11d2-9993-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
