| CN                | униксусерпассворд                                      |
| LDAP-отображаемое имя | униксусерпассворд                                      |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1910                               |
| System-ID — GUID    | 612cb747-c0e8-4f92-9221-fdd5f15b550d                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
