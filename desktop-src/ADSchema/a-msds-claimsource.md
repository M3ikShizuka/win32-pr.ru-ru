# <a name="ms-ds-claim-source-attribute"></a>Атрибут ms-DS-ClaimSet-Source

Для типа утверждения этот атрибут указывает источник типа утверждения. Например, источником может быть Certificate.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-заявка-источник                          |
| LDAP-отображаемое имя | msDS-Клаимсаурце                            |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2157                     |
| System-ID — GUID    | fa32f2a6-f28b-47d0-bf91-663e8f910a72        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
