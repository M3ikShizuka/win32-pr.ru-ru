# <a name="ms-wmi-targetnamespace-attribute"></a>атрибут MS-WMI-TargetNameSpace

Пространство имен, в котором создается объект.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-TargetNameSpace                      |
| LDAP-отображаемое имя | Мсвми-TargetNameSpace                       |
| Размер              | Менее двадцати символов.                |
| Привилегия обновления  | Администратор групповая политика                  |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1646                     |
| System-ID — GUID    | 1c4ab61f-3420-44e5-849d-8b5dbf60feb7        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
