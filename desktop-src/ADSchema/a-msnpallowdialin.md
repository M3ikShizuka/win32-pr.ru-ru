# <a name="msnpallowdialin-attribute"></a>атрибут Мснпалловдиалин

Указывает, имеет ли учетная запись разрешение на удаленный доступ к серверу удаленного доступа. Не изменяйте это значение напрямую. Используйте соответствующую [функцию администрирования RAS](/windows/desktop/RRAS/ras-administration-functions) для изменения этого значения.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | мснпалловдиалин                      |
| LDAP-отображаемое имя | мснпалловдиалин                      |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1119              |
| System-ID — GUID    | db0c9085-c1f2-11d1-bbc5-0080c76670c0 |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
