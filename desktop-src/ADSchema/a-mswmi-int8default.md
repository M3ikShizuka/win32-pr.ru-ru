# <a name="ms-wmi-int8default-attribute"></a>атрибут MS-WMI-int8Default

Значение по умолчанию для объектов параметров WMI 64-разрядного целого числа.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-WMI-int8Default                   |
| LDAP-отображаемое имя | Мсвми — Int8Default                    |
| Размер              | 8 байт                              |
| Привилегия обновления  | Администратор групповая политика           |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1632              |
| System-ID — GUID    | f4d8085a-8c5b-4785-959b-dc585566e445 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
