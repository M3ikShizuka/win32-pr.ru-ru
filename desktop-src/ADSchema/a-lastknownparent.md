# <a name="last-known-parent-attribute"></a>Атрибут последнего известного родителя

Различающееся имя (DN) последнего известного родителя потерянного объекта.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Последний-известный-родительский                       |
| LDAP-отображаемое имя | lastKnownParent                         |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | Каждый раз, когда объект потерян.         |
| Attribute-Id      | 1.2.840.113556.1.4.781                  |
| System-ID — GUID    | 52ab8670-5709-11d1-a9c6-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
