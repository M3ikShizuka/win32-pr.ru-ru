# <a name="required-categories-attribute"></a>Атрибут Required-Categories

Этот атрибут содержит список идентификаторов категорий компонентов, необходимых для запуска объекта (например, приложения).



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Required-Categories                                   |
| LDAP-отображаемое имя | рекуиредкатегориес                                    |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.321                                |
| System-ID — GUID    | 7d6c0e93-7e20-11d0-afd6-00c04fd930c9                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
