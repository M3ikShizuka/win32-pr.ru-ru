# <a name="photo-attribute"></a>атрибут Photo

Объект, закодированный в G3 Fax, как описано в рекомендациях T. 4, с оболочкой ASN. 1, обеспечивающей совместимость с X. 400 Бодипарт, как определено в X. 420.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | фотография                                                 |
| LDAP-отображаемое имя | фотография                                                 |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 0.9.2342.19200300.100.1.7                             |
| System-ID — GUID    | 9c979768-ba1a-4c08-9632-c6a5c1ed649a                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
