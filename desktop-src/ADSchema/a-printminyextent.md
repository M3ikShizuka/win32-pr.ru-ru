|-------------------|--------------------------------------|
| CN                | Print-min-Y-экстент                   |
| LDAP-отображаемое имя | принтминекстент                      |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.280               |
| System-ID — GUID    | ba305f72-47e3-11d0-a1a6-00c04fd930c9 |
