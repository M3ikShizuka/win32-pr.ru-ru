|-------------------|--------------------------------------|
| CN                | MS-DS-минимальный-пароль — возраст           |
| LDAP-отображаемое имя | msDS-Минимумпассвордаже              |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2012              |
