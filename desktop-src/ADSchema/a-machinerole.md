# <a name="machine-role-attribute"></a>Атрибут Machine-Role

Роль для компьютера: контроллер домена, сервер или рабочая станция.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Machine-Role                         |
| LDAP-отображаемое имя | мачинероле                          |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.71                |
| System-ID — GUID    | bf9679b2-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
