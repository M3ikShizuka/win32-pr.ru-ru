# <a name="employee-number-attribute"></a>Атрибут Employee-Number

Номер, назначенный сотруднику, отличному от идентификатора.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Employee-Number                             |
| LDAP-отображаемое имя | employeeNumber                              |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.610                      |
| System-ID — GUID    | a8df73ef-c5ea-11d1-bbcb-0080c76670c0        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
