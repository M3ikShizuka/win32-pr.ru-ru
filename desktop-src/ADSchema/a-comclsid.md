# <a name="com-clsid-attribute"></a>Атрибут COM-CLSID

Этот атрибут содержит идентификатор GUID, связанный с этим классом объектов.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | CLSID COM                                   |
| LDAP-отображаемое имя | комклсид                                    |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.249                      |
| System-ID — GUID    | 281416d9-1968-11d0-a28f-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
