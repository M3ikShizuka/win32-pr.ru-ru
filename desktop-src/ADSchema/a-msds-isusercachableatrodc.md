# <a name="ms-ds-is-user-cachable-at-rodc-attribute"></a>Атрибут ms-DS-"-User-кэшируемые-at-RODC"

|-------------------|--------------------------------------|
| CN                | MS-DS-"-пользователь-кэшируемые-at-RODC"       |
| LDAP-отображаемое имя | msDS-Исусеркачаблеатродк            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2025              |
| System-ID — GUID    | fe01245a-341f-4556-951f-48c033a89050 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
