# <a name="ms-net-ieee-80211-gp-policyguid-attribute"></a>атрибут MS-NET-IEEE-80211-GP-Полицигуид

Содержит идентификатор GUID, определяющий конкретный объект групповая политика 802,11 в домене.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-NET-IEEE-80211-GP-Полицигуид             |
| LDAP-отображаемое имя | MS-NET-IEEE-80211-GP-Полицигуид             |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1951                     |
| System-ID — GUID    | 35697062-1eaf-448b-ac1e-388e0be4fdee        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
