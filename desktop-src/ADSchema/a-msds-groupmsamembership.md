# <a name="ms-ds-groupmsamembership-attribute"></a>Атрибут ms-DS-Граупмсамембершип

Этот атрибут используется для проверки доступа, чтобы определить, имеет ли запрашивающий запрос разрешение на получение пароля для группы MSA.



| Ввод | Значение |
|-------------------|-----------------------------------------------------|
| CN                | MS-DS-Граупмсамембершип                            |
| LDAP-отображаемое имя | msDS-Граупмсамембершип                             |
| Размер              | \-                                                  |
| Привилегия обновления  | \-                                                  |
| Частота обновления  | \-                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.2200                             |
| System-ID — GUID    | 888eedd6-ce04-df40-b462-b8a50e41ba38                |
| Синтаксис            | [**String(NT-Sec-Desc)**](s-string-nt-sec-desc.md) |
