# <a name="shadowmax-attribute"></a>атрибут Шадовмакс

Содержит максимальное число дней, в течение которых пароль может быть действительным.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | шадовмакс                            |
| LDAP-отображаемое имя | шадовмакс                            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.7                      |
| System-ID — GUID    | f285c952-50dd-449e-9160-3b880d99988d |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
