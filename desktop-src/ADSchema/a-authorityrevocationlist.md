|-------------------|-------------------------------------------------------|
| CN                | Список отзыва центров сертификации                             |
| LDAP-отображаемое имя | аусоритиревокатионлист                               |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 2.5.4.38                                              |
| System-ID — GUID    | 1677578d-47f3-11d1-a9c3-0000f80367c1                  |
