# <a name="ms-dfsr-options-attribute"></a>атрибут MS-DFSR-Options

Содержит параметры объекта репликации распределенная файловая система (DFS).



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-параметры                      |
| LDAP-отображаемое имя | Мсдфср — параметры                       |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.17           |
| System-ID — GUID    | d6d67084-C720-417d-8647-b696237a114c |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
