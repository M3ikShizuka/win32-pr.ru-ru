# <a name="ms-ds-never-reveal-group-attribute"></a>Атрибут ms-DS-Never-раскрыть-Group

Используется с контроллером RODC для определения пользователей, компьютеров и групп, которые не могут кэшировать пароли в RODC.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-Never-раскрыть — группа                |
| LDAP-отображаемое имя | msDS-Неверревеалграуп                   |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1926                 |
| System-ID — GUID    | 15585999-fd49-4d66-b25d-eeb96aba8174    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
