# <a name="omt-guid-attribute"></a>Атрибут OMT-Guid

Уникальный идентификатор для записи ссылки-Track-Object-Move.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | OMT-Guid                                              |
| LDAP-отображаемое имя | омтгуид                                               |
| Размер              | 16 байт                                              |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | При каждом изменении ссылки на файл.                 |
| Attribute-Id      | 1.2.840.113556.1.4.505                                |
| System-ID — GUID    | ddac0cf3-af8f-11d0-afeb-00c04fd930c9                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
