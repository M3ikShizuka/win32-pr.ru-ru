# <a name="ms-ds-preferred-gc-site-attribute"></a>Атрибут ms-DS-предпочтителен-GC-site

Атрибут **MS-DS-предпочтителен-GC-site** используется диспетчером учетных записей безопасности для расширения группы во время оценки маркера.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-предпочтительный — GC-site                 |
| LDAP-отображаемое имя | msDS-предпочтителен — GC-site                  |
| Размер              | \-                                      |
| Привилегия обновления  | Администратор домена                    |
| Частота обновления  | По усмотрению администратора.      |
| Attribute-Id      | 1.2.840.113556.1.4.1444                 |
| System-ID — GUID    | d921b50a-0ab2-42cd-87f6-09cf83a91854    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
