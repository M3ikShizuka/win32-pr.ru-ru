# <a name="frs-level-limit-attribute"></a>Атрибут ограничения уровня FRS

Ограничьте глубину дерева каталогов для репликации файлов.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Репликация на уровне FRS                      |
| LDAP-отображаемое имя | фрслевеллимит                        |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.534               |
| System-ID — GUID    | 5245801e-ca6a-11d0-afff-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
