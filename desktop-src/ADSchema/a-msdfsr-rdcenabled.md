# <a name="ms-dfsr-rdcenabled-attribute"></a>атрибут MS-DFSR-Рдценаблед

Содержит значение, указывающее, включен или отключен RDC.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-Рдценаблед                   |
| LDAP-отображаемое имя | Мсдфср — Рдценаблед                    |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.19           |
| System-ID — GUID    | e3b44e05-f4a7-4078-a730-f48670a743f8 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
