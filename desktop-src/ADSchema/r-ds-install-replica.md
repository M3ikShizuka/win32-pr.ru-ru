# <a name="ds-install-replica-extended-right"></a>DS-Install-реплика расширена справа

Расширенное право, необходимое для установки реплики.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | DS-Install-реплика                   |
| Display-Name | Добавить или удалить реплику в домене         |
| Rights-GUID  | 9923a32a-3607-11d2-b9be-0000f87a36b2 |
