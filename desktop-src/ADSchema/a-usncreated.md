# <a name="usn-created-attribute"></a>Атрибут USN-Created

Последовательный номер обновления (USN), назначенный при создании объекта. См. также [**изменения USN**](a-usnchanged.md).



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | USN-Created                          |
| LDAP-отображаемое имя | uSNCreated                           |
| Размер              | 8 байт                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | При создании объекта.          |
| Attribute-Id      | 1.2.840.113556.1.2.19                |
| System-ID — GUID    | bf967a70-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
