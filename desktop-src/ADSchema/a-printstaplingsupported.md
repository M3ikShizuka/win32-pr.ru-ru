# <a name="print-stapling-supported-attribute"></a>Print-степлер-атрибут, поддерживаемый

**Значение true** , если принтер поддерживает сшивание. Предоставлено драйвером.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Печать — поддержка сшивания             |
| LDAP-отображаемое имя | printStaplingSupported               |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.281               |
| System-ID — GUID    | ba305f73-47e3-11d0-a1a6-00c04fd930c9 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
