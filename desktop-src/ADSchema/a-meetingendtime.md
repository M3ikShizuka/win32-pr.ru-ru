| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | митинжендтиме                                                                   |
| LDAP-отображаемое имя | митинжендтиме                                                                   |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.588                                                           |
| System-ID — GUID    | 11b6cc91-48c4-11d1-a9c3-0000f80367c1                                             |
| Синтаксис            | [**String(обобщенное_время)**](s-string-generalized-time.md)                    |
