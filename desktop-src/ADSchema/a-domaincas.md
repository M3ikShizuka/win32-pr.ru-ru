# <a name="domain-certificate-authorities-attribute"></a>Атрибут domain-Certificate-органов

Список центров сертификации для данного домена.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Домен — центры сертификации          |
| LDAP-отображаемое имя | домаинкас                               |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.668                  |
| System-ID — GUID    | 7bfdcb7a-4807-11d1-a9c3-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
