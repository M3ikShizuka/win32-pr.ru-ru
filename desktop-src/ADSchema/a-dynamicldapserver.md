# <a name="dynamic-ldap-server-attribute"></a>Атрибут dynamic-LDAP-Server

DNS-имя сервера, которое передают динамические свойства этой учетной записи.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Dynamic-LDAP-Server                     |
| LDAP-отображаемое имя | динамиклдапсервер                       |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.537                  |
| System-ID — GUID    | 52458021-ca6a-11D0-аффф-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
