# <a name="ca-web-url-attribute"></a>Атрибут CA-WEB-URL

URL-адрес для HTTP-подключения к центру сертификации.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | CA-WEB-URL                                  |
| LDAP-отображаемое имя | кавебурл                                    |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.688                      |
| System-ID — GUID    | 963d2736-48be-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
