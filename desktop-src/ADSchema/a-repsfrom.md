# <a name="reps-from-attribute"></a>Атрибут Reps-From

Выводит список серверов, на которых каталог будет принимать изменения для определенного контекста именования.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Reps-From                                             |
| LDAP-отображаемое имя | репсфром                                              |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.91                                 |
| System-ID — GUID    | bf967a1d-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
