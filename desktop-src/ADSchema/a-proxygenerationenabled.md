# <a name="proxy-generation-enabled-attribute"></a>Прокси — атрибут с поддержкой создания

**Значение true** , если создание прокси-сервера включено.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Создание прокси-сервера с поддержкой создания             |
| LDAP-отображаемое имя | проксиженератионенаблед               |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.523               |
| System-ID — GUID    | 5fd424d6-1262-11d0-a060-00aa006c33ed |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
