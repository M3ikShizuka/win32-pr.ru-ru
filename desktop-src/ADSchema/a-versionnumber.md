# <a name="version-number-attribute"></a>Атрибут Version-Number

Номер версии общего назначения.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Version-Number                       |
| LDAP-отображаемое имя | versionNumber                        |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.141               |
| System-ID — GUID    | bf967a76-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
