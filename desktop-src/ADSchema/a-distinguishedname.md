# <a name="obj-dist-name-attribute"></a>Атрибут obj-расп-Name

То же, что и различающееся имя объекта. Используется Exchange.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Obj-расп-имя                           |
| LDAP-отображаемое имя | distinguishedName                       |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | При каждом создании или перемещении объекта. |
| Attribute-Id      | 2.5.4.49                                |
| System-ID — GUID    | bf9679e4-0de6-11d0-a285-00aa003049e2    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
