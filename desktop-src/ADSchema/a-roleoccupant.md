# <a name="role-occupant-attribute"></a>Атрибут Role-Occupant

Различающееся имя объекта, удовлетворяющего роли Организации.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Role-Occupant                           |
| LDAP-отображаемое имя | ролеоккупант                            |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 2.5.4.33                                |
| System-ID — GUID    | a8df7465-c5ea-11d1-bbcb-0080c76670c0    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
