# <a name="ms-exch-assistant-name-attribute"></a>Имя атрибута MS-дов-Assistant

Содержит имя помощника для учетной записи.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-дов-Assistant-Name                      |
| LDAP-отображаемое имя | msExchAssistantName                         |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.444                      |
| System-ID — GUID    | a8df7394-c5ea-11d1-bbcb-0080c76670c0        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
