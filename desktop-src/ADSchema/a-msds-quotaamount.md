# <a name="ms-ds-quota-amount-attribute"></a>Атрибут ms-DS-Quota-Amount

Назначенная квота в терминах количества объектов, принадлежащих базе данных.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-Quota-сумма                   |
| LDAP-отображаемое имя | msDS-Куотаамаунт                     |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1845              |
| System-ID — GUID    | fbb9a00d-3a8c-4233-9cf9-7189264903a1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
