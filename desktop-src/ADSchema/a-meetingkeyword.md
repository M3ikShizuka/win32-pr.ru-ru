| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.568                      |
| System-ID — GUID    | 11b6cc7f-48c4-11d1-a9c3-0000f80367c1        |
