# <a name="meetingscope-attribute"></a>атрибут Митингскопе

| CN                | митингскопе                                |
| LDAP-отображаемое имя | митингскопе                                |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.581                      |
| System-ID — GUID    | 11b6cc8a-48c4-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
