# <a name="non-security-member-attribute"></a>Атрибут элемента, не относящегося к безопасности

Члены группы, не являющиеся безопасностью. Используется для списков рассылки Exchange.



| Ввод | Значение |
|-------------------|--------------------------------------------------|
| CN                | Не-Security-Member                              |
| LDAP-отображаемое имя | нонсекуритимембер                                |
| Размер              | \-                                               |
| Привилегия обновления  | Администратор домена                             |
| Частота обновления  | Каждый раз, когда пользователь добавляется или удаляется из списка рассылки. |
| Attribute-Id      | 1.2.840.113556.1.4.530                           |
| System-ID — GUID    | 52458018-ca6a-11D0-аффф-0000f80367c1             |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md)          |
