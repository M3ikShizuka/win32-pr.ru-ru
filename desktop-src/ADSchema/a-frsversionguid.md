# <a name="frs-version-guid-attribute"></a>Атрибут FRS-Version-GUID

Если этот параметр указан, изменение этого значения означает, что в этом наборе реплик внесено изменение конфигурации.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | FRS-Version-GUID                                      |
| LDAP-отображаемое имя | фрсверсионгуид                                        |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.43                                 |
| System-ID — GUID    | 26d9736c-6070-11d1-a9c6-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
