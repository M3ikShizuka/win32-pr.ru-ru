# <a name="ms-ds-az-script-timeout-attribute"></a>Атрибут ms-DS-AZ-Script-timeout

Максимальное время ожидания скриптом завершения аудита определенной политики (в миллисекундах).



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-AZ-Script-timeout                 |
| LDAP-отображаемое имя | msDS-Азскрипттимеаут                    |
| Размер              | \-                                      |
| Привилегия обновления  | Администратор Азролес                           |
| Частота обновления  | Во время инициализации или изменения политики. |
| Attribute-Id      | 1.2.840.113556.1.4.1797                 |
| System-ID — GUID    | 87d0fb41-2c8b-41f6-b972-11fdfd50d6b0    |
| Синтаксис            | [**Перечисление**](s-enumeration.md)    |
