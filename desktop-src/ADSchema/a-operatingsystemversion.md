# <a name="operating-system-version-attribute"></a>Атрибут версии операционной системы

Строка версии операционной системы, например 4,0.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Версия операционной системы                    |
| LDAP-отображаемое имя | operatingSystemVersion                      |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.364                      |
| System-ID — GUID    | 3e978926-8c01-11d0-afda-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
