|-------------------|-----------------------------------------|

| LDAP-отображаемое имя | ридсетреференцес                        |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.669                  |
| System-ID — GUID    | 7bfdcb7b-4807-11d1-a9c3-0000f80367c1    |
