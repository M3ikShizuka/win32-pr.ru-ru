# <a name="ms-pki-supersede-templates-attribute"></a>MS-PKI-замена-атрибут Templates

Указывает имена шаблонов сертификатов, заменяемых текущим шаблоном.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-замена — шаблоны                                                                        |
| LDAP-отображаемое имя | Мспки — Замена шаблонов                                                                         |
| Размер              | 64 байт                                                                                          |
| Привилегия обновления  | Администратор домена                                                                              |
| Частота обновления  | При изменении, создании или клонировании объекта шаблона сертификата (MS-PKI-Certificate-Template). |
| Attribute-Id      | 1.2.840.113556.1.4.1437                                                                           |
| System-ID — GUID    | 9de8ae7d-7a5b-421d-b5e4-061f79dfd5d7                                                              |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                                       |
