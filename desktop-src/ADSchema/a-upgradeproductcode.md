# <a name="upgrade-product-code-attribute"></a>Upgrade — атрибут Product-Code

Этот атрибут содержит код продукта других пакетов, таких как приложения, которые могут быть обновлены этим пакетом или могут обновлять этот пакет.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Обновление — код продукта                                  |
| LDAP-отображаемое имя | упградепродукткоде                                    |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.813                                |
| System-ID — GUID    | d9e18312-8939-11d1-aebc-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
