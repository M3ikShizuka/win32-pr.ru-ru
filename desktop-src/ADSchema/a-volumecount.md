# <a name="volume-count-attribute"></a>Атрибут Volume-Count

Квота на записанные тома для данного компьютера.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Volume-Count                         |
| LDAP-отображаемое имя | волумекаунт                          |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.507               |
| System-ID — GUID    | 34aaa217-b699-11d0-afee-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
