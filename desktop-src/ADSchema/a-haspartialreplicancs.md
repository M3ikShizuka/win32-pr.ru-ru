# <a name="has-partial-replica-ncs-attribute"></a>Атрибут WITH-PARTIAL-реплики NC

Одноуровневый элемент с атрибутом-Master-NC. Имеет-partial-реплика-NCS отражает различающееся имя всех контекстов именования других доменов, которые были реплицированы в глобальный каталог.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Имеет-partial-реплика-NC                 |
| LDAP-отображаемое имя | хаспартиалрепликанкс                    |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.15                   |
| System-ID — GUID    | bf967981-0de6-11d0-a285-00aa003049e2    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
