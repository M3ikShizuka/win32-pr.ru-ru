# <a name="ms-dfs-schema-major-version-attribute"></a>атрибут MS-DFS-Schema-Major-Version

Содержит основной номер версии схемы метаданных DFS.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DFS-Schema-Major-Version          |
| LDAP-отображаемое имя | Мсдфс — Счемамажорверсион             |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2030              |
| System-ID — GUID    | ec6d7855-704a-4f61-9aa6-c49a7c1d54c7 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
