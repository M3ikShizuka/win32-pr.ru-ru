# <a name="is-critical-system-object-attribute"></a>Атрибут "является критически важным" — "системный объект"

Если **значение — true**, то объект, на котором размещен этот атрибут, должен быть реплицирован во время установки новой реплики.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Является критически важным — системный объект            |
| LDAP-отображаемое имя | искритикалсистемобжект               |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.868               |
| System-ID — GUID    | 00fbf30d-91fe-11d1-aebc-0000f80367c1 |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
