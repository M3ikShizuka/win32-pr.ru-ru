# <a name="ms-ds-quota-trustee-attribute"></a>Атрибут ms-DS-Quota-доверенного лица

| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-Quota-доверенное лицо                  |
| LDAP-отображаемое имя | msDS-Куотатрусти                    |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1844              |
| System-ID — GUID    | 16378906-4ea5-49be-a8d1-bfd41dff4f65 |
| Синтаксис            | [**Строка (SID)**](s-string-sid.md)  |
