| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-Стрингдефаулт                        |
| LDAP-отображаемое имя | Мсвми — Стрингдефаулт                         |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор групповая политика                  |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1636                     |
| System-ID — GUID    | 152e42b6-37c5-4f55-ab48-1606384a9aea        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
