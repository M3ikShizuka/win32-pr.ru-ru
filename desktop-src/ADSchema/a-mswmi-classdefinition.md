# <a name="ms-wmi-classdefinition-attribute"></a>атрибут MS-WMI-Классдефинитион

Содержит определение класса, енстатед в некоторых пространствах имен WMI.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-Классдефинитион                      |
| LDAP-отображаемое имя | Мсвми — Классдефинитион                       |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1625                     |
| System-ID — GUID    | 2b9c0ebc-c272-45cb-99d2-4d0e691632e0        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
