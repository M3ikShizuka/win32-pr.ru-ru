# <a name="flags-attribute"></a>Атрибут Flags

Используется объектом для хранения сведений о битах.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Флаги                                |
| LDAP-отображаемое имя | flags                                |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.38                |
| System-ID — GUID    | bf967976-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
