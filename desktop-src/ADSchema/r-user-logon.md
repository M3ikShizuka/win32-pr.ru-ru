# <a name="user-logon-property-set"></a>Набор свойств User-Logon

Набор свойств, содержащий атрибуты пользователя, которые описывают сведения о входе пользователя в систему.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | User-Logon                           |
| Display-Name | Сведения о входе                    |
| Rights-GUID  | 5f202010-79a5-11d0-9020-00c04fc2d4cf |
