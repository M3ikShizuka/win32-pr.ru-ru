# <a name="inter-site-topology-renew-attribute"></a>Атрибут межсайтовой топологии-продление

Этот класс указывает, как часто генератор межсайтовой топологии обновляет сообщение о состоянии активности, отправляемое на контроллеры домена, находящиеся на том же сайте.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Топология между сайтами — продление            |
| LDAP-отображаемое имя | интерситетопологиренев               |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1247              |
| System-ID — GUID    | b7c69e5f-2cc7-11d2-854e-00a0c983f608 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
