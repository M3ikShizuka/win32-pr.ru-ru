# <a name="ms-tapi-ip-address-attribute"></a>атрибут MS-TAPI-IP-Address

IP-адреса клиентского компьютера TAPI, в который вошел пользователь. Этот атрибут можно использовать в качестве универсального атрибута для хранения IP-адресов компьютеров.



| Ввод | Значение |
|-------------------|------------------------------------------------------------|
| CN                | MS-TAPI-IP-Address                                         |
| LDAP-отображаемое имя | Мстапи-IpAddress                                           |
| Размер              | Каждый IP-адрес хранится в виде строки в нотации. B. C. D. |
| Привилегия обновления  | Специальные привилегии не требуются.                            |
| Частота обновления  | Обычно очень низкий.                                        |
| Attribute-Id      | 1.2.840.113556.1.4.1701                                    |
| System-ID — GUID    | efd7d7f7-178e-4767-87fa-f8a16b840544                       |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                |
