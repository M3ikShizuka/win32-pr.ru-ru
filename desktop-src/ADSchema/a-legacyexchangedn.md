# <a name="legacy-exchange-dn-attribute"></a>Устаревший атрибут Exchange-DN

Различающееся имя, которое ранее использовалось Exchange.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Устаревший — Exchange-DN                          |
| LDAP-отображаемое имя | legacyExchangeDN                            |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.655                      |
| System-ID — GUID    | 28630ebc-41d5-11d1-a9c1-0000f80367c1        |
| Синтаксис            | [**String(Teletex)**](s-string-teletex.md) |
