# <a name="ms-dfsr-stagingpath-attribute"></a>атрибут MS-DFSR-Стагингпас

Содержит полный путь к промежуточному каталогу.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DFSR-Стагингпас                         |
| LDAP-отображаемое имя | Мсдфср — Стагингпас                          |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.5                   |
| System-ID — GUID    | 86b9a69e-f0a6-405d-99bb-77d977992c2a        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
