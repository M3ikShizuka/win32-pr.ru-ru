# <a name="object-category-attribute"></a>Атрибут Object-Category

Имя класса объекта, используемое для группирования объектов этого или производных классов.



| Ввод | Значение |
|-------------------|--------------------------------------------------|
| CN                | Object-Category                                  |
| LDAP-отображаемое имя | objectCategory                                   |
| Размер              | Примерно 20 байт в среднем.                       |
| Привилегия обновления  | Это значение задается конструктором объекта. |
| Частота обновления  | Это значение никогда не должно изменяться.                  |
| Attribute-Id      | 1.2.840.113556.1.4.782                           |
| System-ID — GUID    | 26d97369-6070-11d1-a9c6-0000f80367c1             |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md)          |
