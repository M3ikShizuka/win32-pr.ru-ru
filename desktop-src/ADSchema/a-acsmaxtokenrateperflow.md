| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-Max-Token-Rate — на Flow          |
| LDAP-отображаемое имя | аксмакстокенратеперфлов               |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.758               |
| System-ID — GUID    | 7f56127b-5301-11d1-a9c5-0000f80367c1 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
