# <a name="msmq-sign-certificates-mig-attribute"></a>Атрибут MSMQ-Sign-Certificates-MIG

В смешанном режиме MSMQ атрибут содержит предыдущее значение Мсмксигнцертификатес. MSMQ поддерживает миграцию из DS MSMQ 1,0 в службы Windows 2000 DS, а смешанный режим задает состояние, в котором некоторые из серверов DS не были обновлены до Windows 2000.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Sign-Certificates-MIG                            |
| LDAP-отображаемое имя | мсмксигнцертификатесмиг                               |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.967                                |
| System-ID — GUID    | 3881b8ea-da3b-11d1-90a5-00c04fd91ab1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
