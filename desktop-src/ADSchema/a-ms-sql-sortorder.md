# <a name="ms-sql-sortorder-attribute"></a>Атрибут MS-SQL-SortOrder

Порядок сортировки текущего экземпляра SQL Server.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-SortOrder                            |
| LDAP-отображаемое имя | mS-SQL-SortOrder                            |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | При установке системы.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1371                     |
| System-ID — GUID    | 6ddc42c0-ccee-11d2-9993-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
