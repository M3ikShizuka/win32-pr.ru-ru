# <a name="dit-content-rules-attribute"></a>Атрибут DIT-Content-Rules

Это указывает допустимое содержимое записей определенного класса структурного объекта с помощью идентификации необязательного набора вспомогательных классов объектов, обязательных, необязательных и исключенных атрибутов. Коллективные атрибуты включены в правила DIT-Content-Rules, как определено в RFC 2251.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | DIT-Content-правила                           |
| LDAP-отображаемое имя | дитконтентрулес                             |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | При создании нового класса.            |
| Attribute-Id      | 2.5.21.2                                    |
| System-ID — GUID    | 9a7ad946-ca53-11d1-bbd0-0080c76670c0        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
