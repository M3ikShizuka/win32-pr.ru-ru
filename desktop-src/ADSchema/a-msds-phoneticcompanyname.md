# <a name="ms-ds-phonetic-company-name-attribute"></a>Атрибут ms-DS-фонетического имени компании

Содержит фонетическое название компании, в которой работает пользователь.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-Фонетическое название компании                 |
| LDAP-отображаемое имя | msDS-Фонетиккомпанинаме                    |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1945                     |
| System-ID — GUID    | 5bd5208d-e5f4-46ae-a514-543bc9c47659        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
