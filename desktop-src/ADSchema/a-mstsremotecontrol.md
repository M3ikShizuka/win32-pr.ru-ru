# <a name="ms-ts-remote-control-attribute"></a>MS-TS-атрибут Remote-Control

Удаленное управление службами терминалов указывает, следует ли разрешить удаленное наблюдение или удаленное управление сеансом служб терминалов пользователя.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-TS-Remote-Control                 |
| LDAP-отображаемое имя | мстсремотеконтрол                    |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1980              |
| System-ID — GUID    | 15177226-8642-468b-8c48-03ddfd004982 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
