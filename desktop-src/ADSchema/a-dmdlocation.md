# <a name="dmd-location-attribute"></a>Атрибут DMD-Location

Различающееся имя секции схемы.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | DMD-Location                            |
| LDAP-отображаемое имя | дмдлокатион                             |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.36                   |
| System-ID — GUID    | f0f8ff8b-1191-11d0-a060-00aa006c33ed    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
