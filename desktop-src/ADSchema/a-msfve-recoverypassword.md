# <a name="ms-fve-recoverypassword-attribute"></a>атрибут MS-ФВЕ-Рековерипассворд

Содержит пароль, необходимый для восстановления тома полного шифрования тома (ФВЕ).



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-ФВЕ-Рековерипассворд                     |
| LDAP-отображаемое имя | Мсфве — Рековерипассворд                      |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1964                     |
| System-ID — GUID    | 43061ac1-c8ad-4ccc-b785-2bfac20fc60a        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
