# <a name="account-name-history-attribute"></a>Атрибут учетной записи Name-History

Период времени, в течение которого учетная запись была активна.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Имя учетной записи — журнал                        |
| LDAP-отображаемое имя | аккаунтнамехистори                          |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1307                     |
| System-ID — GUID    | 031952ec-3b72-11d2-90cc-00c04fd91ab1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
