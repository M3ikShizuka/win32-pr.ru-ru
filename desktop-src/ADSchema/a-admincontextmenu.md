# <a name="admin-context-menu-attribute"></a>Административный — атрибут контекстного меню

Порядковый номер и идентификатор GUID контекстного меню, которое будет использоваться на экранах администрирования.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Контекстное меню администратора                          |
| LDAP-отображаемое имя | админконтекстмену                            |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.614                      |
| System-ID — GUID    | 553fd038-f32e-11d0-b0bc-00c04fd8dca6        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
