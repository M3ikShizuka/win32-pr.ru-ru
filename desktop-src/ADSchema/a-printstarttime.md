# <a name="print-start-time-attribute"></a>Print — атрибут времени запуска

Время начала обслуживания заданий в очереди печати.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Печать — время запуска                     |
| LDAP-отображаемое имя | принтстарттиме                       |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.233               |
| System-ID — GUID    | 281416c9-1968-11d0-a28f-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
