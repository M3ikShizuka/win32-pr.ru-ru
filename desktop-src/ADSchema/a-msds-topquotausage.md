# <a name="ms-ds-top-quota-usage-attribute"></a>Атрибут ms-DS-Top-Quota-Usage

Список пользователей с наибольшей квотой, в настоящее время находящихся в базе данных каталога, упорядоченный по снижению использования квоты.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-верхняя квота — использование                       |
| LDAP-отображаемое имя | msDS-Топкуотаусаже                          |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1850                     |
| System-ID — GUID    | 7b7cce4f-f1f5-4bb6-b7eb-23504af19e75        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
