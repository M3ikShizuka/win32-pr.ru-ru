# <a name="ms-dfsr-schedule-attribute"></a>атрибут MS-DFSR-Schedule

Содержит расписание репликации для службы репликации распределенная файловая система (DFS).



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MS-DFSR-Schedule                                      |
| LDAP-отображаемое имя | Мсдфср — расписание                                       |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.14                            |
| System-ID — GUID    | 4699f15f-a71f-48e2-9ff5-5897c0759205                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
