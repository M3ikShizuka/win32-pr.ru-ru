# <a name="reports-attribute"></a>Атрибут Reports

Содержит список пользователей, непосредственно сообщающих пользователю. Пользователи, перечисленные как отчеты, имеют свойство "Диспетчер свойств", установленное для этого пользователя. Каждый элемент в списке представляет собой связанную ссылку на объект, представляющий пользователя.



| Ввод | Значение |
|-------------------|-------------------------------------------------|
| CN                | Отчеты                                         |
| LDAP-отображаемое имя | directReports                                   |
| Размер              | \-                                              |
| Привилегия обновления  | Это значение задается системой.                |
| Частота обновления  | При каждом изменении непосредственных подчиненных пользователей. |
| Attribute-Id      | 1.2.840.113556.1.2.436                          |
| System-ID — GUID    | bf967a1c-0de6-11d0-a285-00aa003049e2            |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md)         |
