# <a name="acs-non-reserved-peak-rate-attribute"></a>ACS-не зарезервированный — атрибут частоты пиковой нагрузки

Атрибут **ACS-не зарезервирован — Пиковая ставка** предназначен только для внутреннего использования. На основе RFC2814.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS — не зарезервировано — Пиковая ставка           |
| LDAP-отображаемое имя | акснонресерведпеакрате               |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1318              |
| System-ID — GUID    | a331a73f-3b90-11d2-90cc-00c04fd91ab1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
