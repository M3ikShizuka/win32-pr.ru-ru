# <a name="help-file-name-attribute"></a>Help-File-Name, атрибут

Этот атрибут использовался для Exchange 4,0. Он содержал имя, которое должно использоваться для файла, когда поставщик загрузил данные справки на клиентский компьютер. Он не используется для других версий Exchange.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Help-File-Name                              |
| LDAP-отображаемое имя | хелпфиленаме                                |
| Размер              | \-                                          |
| Привилегия обновления  | Используется системой.                 |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.327                      |
| System-ID — GUID    | 5fd424a9-1262-11d0-a060-00aa006c33ed        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
