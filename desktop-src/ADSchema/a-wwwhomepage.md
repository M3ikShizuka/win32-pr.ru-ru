# <a name="www-home-page-attribute"></a>WWW — атрибут главной страницы

Веб-страница, которая является первичной целевой страницей веб-сайта.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------------------|
| CN                | WWW — Главная страница                                                               |
| LDAP-отображаемое имя | wWWHomePage                                                                 |
| Размер              | \-                                                                          |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                      |
| Частота обновления  | Когда создается запись пользователя и каждый раз, когда необходимо изменить веб-страницу. |
| Attribute-Id      | 1.2.840.113556.1.2.464                                                      |
| System-ID — GUID    | bf967a7a-0de6-11d0-a285-00aa003049e2                                        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                 |
