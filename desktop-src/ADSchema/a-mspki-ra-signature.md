# <a name="ms-pki-ra-signature-attribute"></a>атрибут MS-PKI-RA-Signature

Указывает количество подписей центра регистрации, необходимых для запроса на регистрацию.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-RA-Signature                                                                               |
| LDAP-отображаемое имя | Мспки-RA-Signature                                                                                |
| Размер              | 4 байта                                                                                           |
| Привилегия обновления  | Администратор домена                                                                              |
| Частота обновления  | При изменении, создании или клонировании объекта шаблона сертификата (MS-PKI-Certificate-Template). |
| Attribute-Id      | 1.2.840.113556.1.4.1429                                                                           |
| System-ID — GUID    | fe17e04b-937d-4f7e-8e0e-9292c8d5683e                                                              |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                                                              |
