# <a name="post-office-box-attribute"></a>Атрибут "после помещения"

Номер почтового ящика для этого объекта.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------------------|
| CN                | Пост-Office-Box                                                             |
| LDAP-отображаемое имя | postOfficeBox                                                               |
| Размер              | \-                                                                          |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                      |
| Частота обновления  | При создании записи пользователя и при необходимости изменения адреса. |
| Attribute-Id      | 2.5.4.18                                                                    |
| System-ID — GUID    | bf9679fb-0de6-11d0-a285-00aa003049e2                                        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                 |
