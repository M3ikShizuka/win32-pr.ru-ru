# <a name="global-address-list2-attribute"></a>Атрибут Global-Address-list2

Этот атрибут используется в контейнере Microsoft Exchange для хранения различающегося имени только что созданного глобального списка адресов (GAL).



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Global-Address-list2                    |
| LDAP-отображаемое имя | globalAddressList2                      |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2047                 |
| System-ID — GUID    | 4898f63d-4112-477c-8826-3ca00bd8277d    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
