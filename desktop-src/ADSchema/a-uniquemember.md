# <a name="uniquemember-attribute"></a>атрибут Уникуемембер

Различающееся имя для члена группы. Используется Граупофуникуенамес.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | уникуемембер                            |
| LDAP-отображаемое имя | уникуемембер                            |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 2.5.4.50                                |
| System-ID — GUID    | 8f888726-f80a-44d7-b1ee-cb9df21392c8    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
