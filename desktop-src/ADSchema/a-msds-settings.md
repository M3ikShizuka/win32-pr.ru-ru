# <a name="ms-ds-settings-attribute"></a>Атрибут ms-DS-Settings

Используется для хранения параметров объекта. Его использование определяется только владельцем объекта. Мы рекомендуем использовать его для хранения пар "имя-значение". Например, цвет = синий.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-параметры                              |
| LDAP-отображаемое имя | msDS-параметры                               |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | При каждом изменении значений параметров.       |
| Attribute-Id      | 1.2.840.113556.1.4.1697                     |
| System-ID — GUID    | 0e1b47d7-40a3-4b48-8d1b-4cac0c1cdf21        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
