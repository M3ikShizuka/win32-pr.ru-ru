# <a name="physical-delivery-office-name-attribute"></a>атрибут физической доставки-Office-Name

Содержит расположение офиса в организации пользователя.



| Ввод | Значение |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | физическая доставка — Office-имя                                                       |
| LDAP-отображаемое имя | physicalDeliveryOfficeName                                                          |
| Размер              | \-                                                                                  |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                              |
| Частота обновления  | Когда создается запись пользователя и каждый раз, когда необходимо изменить расположение офиса. |
| Attribute-Id      | 2.5.4.19                                                                            |
| System-ID — GUID    | bf9679f7-0de6-11d0-a285-00aa003049e2                                                |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                         |
