# <a name="netbios-name-attribute"></a>Атрибут NETBIOS-Name

Имя объекта, который будет использоваться по протоколу NetBIOS.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | NETBIOS-Name                                |
| LDAP-отображаемое имя | nETBIOSName                                 |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.87                       |
| System-ID — GUID    | bf9679d8-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
