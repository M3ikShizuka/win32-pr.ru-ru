# <a name="ms-com-partitionsetlink-attribute"></a>атрибут MS-COM-Партитионсетлинк

Ссылка, используемая для связывания раздела COM+ с объектом-набором разделов COM+.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-COM-Партитионсетлинк                 |
| LDAP-отображаемое имя | Мском — Партитионсетлинк                  |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1424                 |
| System-ID — GUID    | 67f121dc-7d02-4c7d-82f5-9ad4c950ac34    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
