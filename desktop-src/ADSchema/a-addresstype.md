# <a name="address-type-attribute"></a>Атрибут Address-Type

Символьная строка, описывающая формат адреса пользователя. Типы адресов сопоставляются с форматами адресов. То есть, просмотрев тип адреса получателя, клиентские приложения могут определить, как отформатировать адрес, подходящий для получателя.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------------------------------------------------|
| CN                | Address-Type                                                                                                              |
| LDAP-отображаемое имя | addressType                                                                                                               |
| Размер              | Типы адресов: 3COM, АТРИ, ККМАИЛ, COMPUSERVE, EX, FAX, MSFAX, MCI, МХС, MS, MSA, MSN, ПРОФС, SMTP, СНАДС, ТЕЛЕКСА, X400, X500 |
| Привилегия обновления  | Это значение задается системой.                                                                                          |
| Частота обновления  | Задается при создании объекта.                                                                                           |
| Attribute-Id      | 1.2.840.113556.1.2.350                                                                                                    |
| System-ID — GUID    | 5fd42464-1262-11d0-a060-00aa006c33ed                                                                                      |
| Синтаксис            | [**String(Teletex)**](s-string-teletex.md)                                                                               |
