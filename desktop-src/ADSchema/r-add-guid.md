# <a name="add-guid-extended-right"></a>Расширенное право Add-GUID

Расширенное право, необходимое в корне сетевого контроллера, для добавления объекта с указанным GUID.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Add-GUID                             |
| Display-Name | Добавление GUID                             |
| Rights-GUID  | 440820ad-65b4-11d1-a3da-0000f875ae0d |
