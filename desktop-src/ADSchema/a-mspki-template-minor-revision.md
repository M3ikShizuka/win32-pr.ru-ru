# <a name="ms-pki-template-minor-revision-attribute"></a>атрибут MS-PKI-Template-минор-Revision

Отслеживает атрибуты в изменяемом классе. Однако это не вызовет автоматическую регистрацию.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-Template — дополнительный номер редакции                                                                    |
| LDAP-отображаемое имя | Мспки-Template — дополнительный номер редакции                                                                     |
| Размер              | 4 байта                                                                                           |
| Привилегия обновления  | Администратор домена                                                                              |
| Частота обновления  | При изменении, создании или клонировании объекта шаблона сертификата (MS-PKI-Certificate-Template). |
| Attribute-Id      | 1.2.840.113556.1.4.1435                                                                           |
| System-ID — GUID    | 13f5236c-1884-46b1-b5d0-484e38990d58                                                              |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                                                              |
