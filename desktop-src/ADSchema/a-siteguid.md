# <a name="site-guid-attribute"></a>Атрибут site-GUID

Уникальный идентификатор для сайта.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Сайт — GUID                                             |
| LDAP-отображаемое имя | ситегуид                                              |
| Размер              | 16 байт                                              |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | При создании нового сайта.                       |
| Attribute-Id      | 1.2.840.113556.1.4.362                                |
| System-ID — GUID    | 3e978924-8c01-11d0-afda-00c04fd930c9                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
