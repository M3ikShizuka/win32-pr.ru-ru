# <a name="is-deleted-attribute"></a>Атрибут Is-Deleted

Если **значение — true**, этот объект был помечен для удаления и не может быть создан. После истечения периода захоронения он будет удален из системы.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Is-Deleted                           |
| LDAP-отображаемое имя | isDeleted                            |
| Размер              | \-                                   |
| Привилегия обновления  | Администратор домена                 |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.48                |
| System-ID — GUID    | bf96798f-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
