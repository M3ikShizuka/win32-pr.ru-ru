# <a name="site-link-list-attribute"></a>Атрибут "сайт-ссылка-список"

Список связей сайтов, связанных с этим мостом.



| Ввод | Значение |
|-------------------|--------------------------------------------------------------|
| CN                | Site-Link-List                                               |
| LDAP-отображаемое имя | сителинклист                                                 |
| Размер              | \-                                                           |
| Привилегия обновления  | Администратор домена                                         |
| Частота обновления  | При добавлении или удалении связи сайтов из моста. |
| Attribute-Id      | 1.2.840.113556.1.4.822                                       |
| System-ID — GUID    | d50c2cdd-8951-11d1-aebc-0000f80367c1                         |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md)                      |
