# <a name="package-type-attribute"></a>Атрибут Package-Type

Этот атрибут описывает тип установки, необходимый для пакета приложения, например MSI, EXE, CAB.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Package-Type                         |
| LDAP-отображаемое имя | packageType                          |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.324               |
| System-ID — GUID    | 7d6c0e96-7e20-11d0-afd6-00c04fd930c9 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
