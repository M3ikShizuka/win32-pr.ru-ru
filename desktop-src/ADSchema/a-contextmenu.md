# <a name="context-menu-attribute"></a>Атрибут Context-Menu

Порядковый номер и идентификатор GUID контекстного меню, используемого для объекта.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Context-Menu                                |
| LDAP-отображаемое имя | Меню                                 |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор схемы                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.499                      |
| System-ID — GUID    | 4d8601ee-ac85-11d0-afe3-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
