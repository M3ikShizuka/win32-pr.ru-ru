# <a name="extended-attribute-info-attribute"></a>Атрибут расширенного атрибута-info

Многозначное свойство, содержащее строки, представляющие дополнительные сведения для каждого атрибута.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Расширенный атрибут-сведения                     |
| LDAP-отображаемое имя | екстендедаттрибутеинфо                       |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.909                      |
| System-ID — GUID    | 9a7ad947-ca53-11d1-bbd0-0080c76670c0        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
