# <a name="presentation-address-attribute"></a>Атрибут Presentation-Address

Указывает адрес представления, связанный с объектом, представляющим сущность приложения OSI.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------------|
| CN                | Presentation-Address                                                  |
| LDAP-отображаемое имя | пресентатионаддресс                                                   |
| Размер              | \-                                                                    |
| Привилегия обновления  | \-                                                                    |
| Частота обновления  | \-                                                                    |
| Attribute-Id      | 2.5.4.29                                                              |
| System-ID — GUID    | a8df744b-c5ea-11d1-bbcb-0080c76670c0                                  |
| Синтаксис            | [**Object(адрес_представления)**](s-object-presentation-address.md) |
