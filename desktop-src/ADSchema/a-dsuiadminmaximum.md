# <a name="ds-ui-admin-maximum-attribute"></a>DS-UI-Admin-максимальный атрибут

Это максимальное число объектов, которые могут отображаться в контейнере администратором по умолчанию.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | DS-UI-Admin — максимум                  |
| LDAP-отображаемое имя | дсуиадминмаксимум                     |
| Размер              | \-                                   |
| Привилегия обновления  | Администратор домена                 |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1344              |
| System-ID — GUID    | ee8d0ae0-6f91-11d2-9905-0000f87a57d4 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
