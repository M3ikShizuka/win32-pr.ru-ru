# <a name="min-pwd-age-attribute"></a>Атрибут min-PWD-Age

Минимальное время в 100-наносекундных интервалах, в течение которого пароль действителен.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Min-PWD-Age                          |
| LDAP-отображаемое имя | минпвдаже                            |
| Размер              | \-                                   |
| Привилегия обновления  | Администратор домена                 |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.78                |
| System-ID — GUID    | bf9679c2-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
