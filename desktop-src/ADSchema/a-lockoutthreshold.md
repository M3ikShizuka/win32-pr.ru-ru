# <a name="lockout-threshold-attribute"></a>Атрибут Lockout-Threshold

Число недопустимых попыток входа в систему, которые разрешены до блокировки учетной записи.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Lockout-Threshold                    |
| LDAP-отображаемое имя | lockoutThreshold                     |
| Размер              | \-                                   |
| Привилегия обновления  | Администратор домена                 |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.73                |
| System-ID — GUID    | bf9679a6-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
