# <a name="help-data16-attribute"></a>Атрибут Help-Data16

этот атрибут использовался для формата файла справки Win16 для Exchange 4,0. Он не используется для других версий Exchange.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Help-Data16                                           |
| LDAP-отображаемое имя | helpData16                                            |
| Размер              | \-                                                    |
| Привилегия обновления  | Используется системой.                           |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.402                                |
| System-ID — GUID    | 5fd424a7-1262-11d0-a060-00aa006c33ed                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
