# <a name="must-contain-attribute"></a>Атрибут Must-Contain

Список обязательных атрибутов для класса. Эти атрибуты должны быть указаны при создании экземпляра класса.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------|
| CN                | Must-Contain                                                    |
| LDAP-отображаемое имя | mustContain                                                     |
| Размер              | \-                                                              |
| Привилегия обновления  | Администратор схемы                                            |
| Частота обновления  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.24                                           |
| System-ID — GUID    | bf9679d3-0de6-11d0-a285-00aa003049e2                            |
| Синтаксис            | [**String(идентификатор_объекта)**](s-string-object-identifier.md) |
