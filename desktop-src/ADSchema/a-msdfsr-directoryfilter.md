|-------------------|---------------------------------------------|
| CN                | MS-DFSR-Директорифилтер                     |
| LDAP-отображаемое имя | Мсдфср — Директорифилтер                      |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.13                  |
| System-ID — GUID    | 93c7b477-1f2e-4b40-b7bf-007e8d038ccf        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
