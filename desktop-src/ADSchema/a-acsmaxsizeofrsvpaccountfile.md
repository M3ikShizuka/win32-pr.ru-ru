# <a name="acs-max-size-of-rsvp-account-file-attribute"></a>Атрибут ACS-Max-size-of-RSVP-Account-File

Максимальный размер файла учетной записи RSVP (в байтах).



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-Max-size-of-RSVP-файл учетной записи    |
| LDAP-отображаемое имя | аксмакссизеофрсвпаккаунтфиле          |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.902               |
| System-ID — GUID    | f0722311-aef5-11d1-bdcf-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
