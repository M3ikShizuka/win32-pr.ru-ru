# <a name="generated-connection-attribute"></a>Атрибут Generated-Connection

**Значение true** , если это соединение было создано с помощью автоматического создания топологии.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Generated-Connection                 |
| LDAP-отображаемое имя | женератедконнектион                  |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.41                |
| System-ID — GUID    | bf96797a-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
