# <a name="authentication-options-attribute"></a>Атрибут Authentication-Options

Параметры проверки подлинности, используемые в ADSI для привязки к объектам служб каталогов.



| Ввод | Значение |
|-------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | Authentication-Options                                                                                                                                                                                                                                                       |
| LDAP-отображаемое имя | authenticationOptions                                                                                                                                                                                                                                                        |
| Размер              | 4 байта. значения, определенные в IADS. h: общедоступная \_ \_ проверка подлинности ads, объявления, использование шифрования 0x2, объявления, использование \_ \_ \_ \_ протокола SSL 0x2, ads \_ READONLY \_ SERVER 0x4, \_ \_ \_ \_ \_ \_ \_ \_ \_ \_ рекламные объявления, FAST запрос на рекламу, объявление |
| Привилегия обновления  | \-                                                                                                                                                                                                                                                                           |
| Частота обновления  | \-                                                                                                                                                                                                                                                                           |
| Attribute-Id      | 1.2.840.113556.1.4.11                                                                                                                                                                                                                                                        |
| System-ID — GUID    | bf967928-0de6-11d0-a285-00aa003049e2                                                                                                                                                                                                                                         |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                                                                                                                                                                                                                                         |
