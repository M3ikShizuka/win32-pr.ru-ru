# <a name="country-name-attribute"></a>Атрибут Country-Name

Страна или регион в адресе пользователя. Страна или регион представлены в виде кода из двух символов на основе ISO-3166.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Country-Name                                |
| LDAP-отображаемое имя | c                                           |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 2.5.4.6                                     |
| System-ID — GUID    | bf967945-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
