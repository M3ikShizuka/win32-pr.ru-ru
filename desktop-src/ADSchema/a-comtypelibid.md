# <a name="com-typelib-id-attribute"></a>Атрибут COM-typelib-ID

Этот атрибут хранит список идентификаторов библиотек типов, содержащихся в этом пакете приложения.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Модель COM-typelib-ID                                                                   |
| LDAP-отображаемое имя | комтипелибид                                                                     |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.254                                                           |
| System-ID — GUID    | 281416de-1968-11d0-a28f-00aa003049e2                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
