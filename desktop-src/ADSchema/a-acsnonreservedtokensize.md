# <a name="acs-non-reserved-token-size-attribute"></a>ACS-не зарезервировано — атрибут размера токена

Атрибут **ACS-не зарезервирован — маркер-size** предназначен только для внутреннего использования. На основе RFC2814.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS — не зарезервировано — размер токена          |
| LDAP-отображаемое имя | акснонресерведтокенсизе              |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1319              |
| System-ID — GUID    | a916d7c9-3b90-11d2-90cc-00c04fd91ab1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
