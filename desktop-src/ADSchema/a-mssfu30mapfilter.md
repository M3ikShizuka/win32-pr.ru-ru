# <a name="mssfu-30-map-filter-attribute"></a>атрибут Мссфу-30-Map-Filter

Содержит строку, используемую для фильтрации данных на карте. Может содержать ключи карт, доменное имя и другие типы данных.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Мссфу-30 — Map-Filter                         |
| LDAP-отображаемое имя | msSFU30MapFilter                            |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.306                 |
| System-ID — GUID    | b7b16e01-024f-4e23-ad0d-71f1a406b684        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
