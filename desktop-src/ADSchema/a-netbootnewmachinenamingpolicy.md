# <a name="netboot-new-machine-naming-policy-attribute"></a>нетбут-создание атрибута-компьютера-имя политики

Указывает схему именования, которую будут использовать новые учетные записи клиентского компьютера.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | нетбут — создание политики именования компьютеров           |
| LDAP-отображаемое имя | нетбутневмачиненамингполици               |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.855                      |
| System-ID — GUID    | 0738307c-91df-11d1-aebc-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
