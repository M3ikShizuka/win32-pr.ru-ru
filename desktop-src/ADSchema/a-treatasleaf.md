# <a name="treat-as-leaf-attribute"></a>Атрибут "рассматривать как конечный лист"

Описатели отображения. Этот флаг имеет значение **true** , чтобы связанный класс отображался как конечный класс, даже если он имеет дочерние элементы.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Обрабатывать как конечный                        |
| LDAP-отображаемое имя | треатаслеаф                          |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.806               |
| System-ID — GUID    | 8fd044e3-771f-11d1-aeae-0000f80367c1 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
