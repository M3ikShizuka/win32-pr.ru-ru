# <a name="supplemental-credentials-attribute"></a>Атрибут Supplemental-Credentials

Сохраненные учетные данные для использования при проверке подлинности. Зашифрованная версия пароля пользователя. Этот атрибут не доступен для чтения или записи.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Supplemental-Credentials                              |
| LDAP-отображаемое имя | супплементалкредентиалс                               |
| Размер              | \-                                                    |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | При изменении пароля пользователя.                     |
| Attribute-Id      | 1.2.840.113556.1.4.125                                |
| System-ID — GUID    | bf967a3f-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
