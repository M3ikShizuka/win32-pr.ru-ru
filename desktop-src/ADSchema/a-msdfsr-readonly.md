# <a name="ms-dfsr-readonly-attribute"></a>атрибут MS-DFSR-ReadOnly

Указывает, доступно ли содержимое только для чтения или для чтения и записи.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-ReadOnly                     |
| LDAP-отображаемое имя | Мсдфср — только для чтения                      |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.28           |
| System-ID — GUID    | 5ac48021-e447-46e7-9d23-92c0c6a90dfb |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
