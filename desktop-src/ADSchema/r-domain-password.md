# <a name="domain-password-property-set"></a>Набор свойств Domain-Password

Набор свойств, содержащий атрибуты, связанные с блокировкой и сроком действия пароля учетной записи пользователя.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Domain-Password                      |
| Display-Name | Политики блокировки & паролей домена   |
| Rights-GUID  | c7407360-20bf-11d0-a768-00aa006e0529 |
