# <a name="code-page-attribute"></a>Атрибут Code-Page

Указывает кодовую страницу для выбранного языка пользователя. Это значение не используется в Windows 2000.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Code-Page                                             |
| LDAP-отображаемое имя | Страница                                              |
| Размер              | 4 байта                                               |
| Привилегия обновления  | Администратор домена                                  |
| Частота обновления  | Когда пользователь хочет изменить язык по умолчанию. |
| Attribute-Id      | 1.2.840.113556.1.4.16                                 |
| System-ID — GUID    | bf967938-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                  |
