# <a name="rid-used-pool-attribute"></a>RID — используется атрибут пула

Пулы RID, которые использовались контроллером домена.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | RID — используется для пула                        |
| LDAP-отображаемое имя | ридуседпул                          |
| Размер              | 8 байт                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | При очистке пула RID.      |
| Attribute-Id      | 1.2.840.113556.1.4.373               |
| System-ID — GUID    | 6617188b-8f3c-11d0-afda-00c04fd930c9 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
