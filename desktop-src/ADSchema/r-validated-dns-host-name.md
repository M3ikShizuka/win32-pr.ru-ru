# <a name="validated-dns-host-name-validated-writes"></a>Проверено — DNS-имя узла — проверенные записи

| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Проверено — DNS-Host-Name              |
| Display-Name | Удостоверенная запись на узел с DNS-именем     |
| Rights-GUID  | 72e39547-7b18-11d1-adef-00c04fd8d5cd |
