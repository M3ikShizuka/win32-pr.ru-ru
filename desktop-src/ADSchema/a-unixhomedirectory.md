# <a name="unixhomedirectory-attribute"></a>атрибут Униксхомедиректори

Содержит абсолютный путь к домашнему каталогу.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | униксхомедиректори                    |
| LDAP-отображаемое имя | униксхомедиректори                    |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.3                      |
| System-ID — GUID    | bc2dba12-000f-464d-bf1d-0808465d8843 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
