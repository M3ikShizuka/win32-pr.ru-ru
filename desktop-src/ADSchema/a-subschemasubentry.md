# <a name="subschemasubentry-attribute"></a>Атрибут Субсчемасубентри

Различающееся имя расположения объекта подсхемы, в котором определен класс или атрибут.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | субсчемасубентри                                                                |
| LDAP-отображаемое имя | субсчемасубентри                                                                |
| Размер              | \-                                                                               |
| Привилегия обновления  | Администратор схемы                                                             |
| Частота обновления  | При загрузке Active Directory или при определении нового класса или атрибута. |
| Attribute-Id      | 2.5.18.10                                                                        |
| System-ID — GUID    | 9a7ad94d-ca53-11d1-bbd0-0080c76670c0                                             |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md)                                          |
