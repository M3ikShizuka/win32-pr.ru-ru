# <a name="ms-ds-authenticatedto-accountlist-attribute"></a>Атрибут ms-DS-Аусентикатедто-Аккаунтлист

| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-Аусентикатедто-Аккаунтлист       |
| LDAP-отображаемое имя | msDS-Аусентикатедтоаккаунтлист         |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1957                 |
| System-ID — GUID    | e8b2c971-a6df-47bc-8d6f-62770d527aa5    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
