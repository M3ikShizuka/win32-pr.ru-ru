# <a name="msradiusservicetype-attribute"></a>атрибут Мсрадиуссервицетипе

Атрибут **мсрадиуссервицетипе** используется внутренним образом. Не изменяйте это значение напрямую.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | мсрадиуссервицетипе                  |
| LDAP-отображаемое имя | мсрадиуссервицетипе                  |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1171              |
| System-ID — GUID    | db0c90b6-c1f2-11d1-bbc5-0080c76670c0 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
