# <a name="ms-ds-source-object-dn-attribute"></a>Атрибут ms-DS-Source-Object-DN

Содержит строковое представление различающегося имени объекта в другом лесу, соответствующем этому объекту.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-Source-Object-DN                      |
| LDAP-отображаемое имя | msDS-Саурцеобжектдн                         |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1879                     |
| System-ID — GUID    | 773e93af-d3b4-48d4-b3f9-06457602d3d0        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
