# <a name="ms-ds-per-user-trust-tombstones-quota-attribute"></a>Атрибут квоты MS-DS-на пользователя — доверие — захоронение

Используется для принудительного применения квоты на пользователя для удаления объектов Trusted-Domain, когда Авторизация основана на идентификаторе безопасности пользователя.



| Ввод | Значение |
|-------------------|-------------------------------------------|
| CN                | MS-DS-на пользователя-захоронение — квота     |
| LDAP-отображаемое имя | msDS-Перусертрусттомбстонескуота          |
| Размер              | \-                                        |
| Привилегия обновления  | Администратор домена                      |
| Частота обновления  | При создании леса и редко после него. |
| Attribute-Id      | 1.2.840.113556.1.4.1790                   |
| System-ID — GUID    | 8b70a6c6-50f9-4fa3-a71e-1ce03040449b      |
| Синтаксис            | [**Перечисление**](s-enumeration.md)      |
