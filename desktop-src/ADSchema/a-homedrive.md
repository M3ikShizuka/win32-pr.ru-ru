# <a name="home-drive-attribute"></a>Атрибут Home-Drive

Указывает букву диска, с которой будет сопоставлен UNC-путь, заданный параметром [**хомедиректори**](a-homedirectory.md). Буква диска должна быть указана в формате *буква_диска * * *:**, где *буква_диска* — буква диска для отображения. Буква « *буква_диска* » должна быть отдельной буквы в верхнем регистре и двоеточием (:) является обязательным.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------------------------|
| CN                | Home-Drive                                                                        |
| LDAP-отображаемое имя | хомедриве                                                                         |
| Размер              | 2 байта                                                                           |
| Привилегия обновления  | Это значение устанавливается администратором домена.                                         |
| Частота обновления  | При создании записи пользователя и при необходимости изменения домашнего диска. |
| Attribute-Id      | 1.2.840.113556.1.4.45                                                             |
| System-ID — GUID    | bf967986-0de6-11d0-a285-00aa003049e2                                              |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                       |
