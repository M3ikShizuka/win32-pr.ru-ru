# <a name="netboot-server-attribute"></a>атрибут нетбут-Server

Различающееся имя сервера Нетбут.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | нетбут — сервер                          |
| LDAP-отображаемое имя | нетбутсервер                           |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.860                  |
| System-ID — GUID    | 07383081-91df-11d1-aebc-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
