# <a name="ms-radius-savedframedipv6route-attribute"></a>атрибут MS-RADIUS-SavedFramedIpv6Route

Предоставляет сведения о маршрутизации, которые должны быть настроены для пользователя на NAS.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-RADIUS-SavedFramedIpv6Route       |
| LDAP-отображаемое имя | Мсрадиус — SavedFramedIpv6Route        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1918              |
| System-ID — GUID    | 9666bb5c-df9d-4d41-b437-2eec7e27c9b3 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
