# <a name="audio-attribute"></a>атрибут Audio

Тип атрибута Audio позволяет хранить звуки в каталоге.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | звук                                                 |
| LDAP-отображаемое имя | звук                                                 |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 0.9.2342.19200300.100.1.55                            |
| System-ID — GUID    | d0e1d224-e1a0-42ce-a2da-793ba5244f35                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
