|-------------------|--------------------------------------|
| CN                | нисмапнаме                           |
| LDAP-отображаемое имя | нисмапнаме                           |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.26                     |
| System-ID — GUID    | 969d3c79-0e9a-4d95-b0ac-bdde7ff8f3a1 |
