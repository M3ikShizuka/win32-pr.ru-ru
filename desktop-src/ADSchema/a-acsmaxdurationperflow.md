Максимальная длительность одного потока в секундах.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-Max-Duration-на Flow            |
| LDAP-отображаемое имя | аксмаксдуратионперфлов                |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.761               |
| System-ID — GUID    | 7f56127e-5301-11d1-a9c5-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
