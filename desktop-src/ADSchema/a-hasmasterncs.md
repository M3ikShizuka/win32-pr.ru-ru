# <a name="has-master-ncs-attribute"></a>Атрибут содержит-Master-NC

| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Содержит-Master-NC                          |
| LDAP-отображаемое имя | хасмастернкс                            |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.2.14                   |
| System-ID — GUID    | bf967982-0de6-11d0-a285-00aa003049e2    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
