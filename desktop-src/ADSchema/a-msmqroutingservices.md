# <a name="msmq-routing-services-attribute"></a>Атрибут MSMQ-Routing-Services

Указывает, предоставляет ли MSMQ, установленный на этом компьютере, службы маршрутизации MSMQ.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MSMQ-Routing-Services                |
| LDAP-отображаемое имя | мсмкраутингсервицес                  |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1227              |
| System-ID — GUID    | 2df90d77-009f-11d2-aa4c-00c04fd7d83a |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
