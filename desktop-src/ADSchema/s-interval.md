# <a name="interval-syntax"></a>Синтаксис интервала

Представляет значение интервала времени. Фактические единицы времени зависят от того, какой атрибут использует этот синтаксис. Этот синтаксис идентичен синтаксису [**ларжеинтежер**](s-largeinteger.md) , за исключением того, что значения High и Low являются целыми числами без знака.



| Ввод | Значение |
|--------------|------------------------------------------------------------------------------------|
| Имя         | Интервал                                                                           |
| Идентификатор синтаксиса    | 2.5.5.16                                                                           |
| ИДЕНТИФИКАТОР OM        | 65                                                                                 |
| Тип MAPI    | \-                                                                                 |
| Тип ADS     | АДСТИПЕ \_ большое \_ целое                                                            |
| Тип варианта | \_Диспетчер VT                                                                       |
| Тип SDS     | COM-объект, который можно привести к [**иадсларжеинтежер**](/windows/desktop/api/iads/nn-iads-iadslargeinteger). |
