# <a name="domain-identifier-attribute"></a>Атрибут Domain-Identifier

Идентификатор безопасности домена, определяющий домен.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Domain-Identifier                    |
| LDAP-отображаемое имя | домаинидентифиер                     |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.755               |
| System-ID — GUID    | 7f561278-5301-11d1-a9c5-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
