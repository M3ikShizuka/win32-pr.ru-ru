# <a name="ms-fve-volumeguid-attribute"></a>атрибут MS-ФВЕ-VolumeGuid

Содержит идентификатор GUID, связанный с Томом диска, поддерживаемым BitLocker.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MS-ФВЕ-VolumeGuid                                     |
| LDAP-отображаемое имя | Мсфве — VolumeGuid                                      |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1998                               |
| System-ID — GUID    | 85e5a5cf-dcee-4075-9cfd-ac9db6a2f245                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
