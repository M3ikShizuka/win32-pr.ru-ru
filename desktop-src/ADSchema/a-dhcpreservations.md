# <a name="dhcp-reservations-attribute"></a>атрибут резервирования DHCP

Атрибут **резервирования DHCP** в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Резервирования DHCP                    |
| LDAP-отображаемое имя | дхкпресерватионс                     |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.709               |
| System-ID — GUID    | 963d274a-48be-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
