# <a name="frs-working-path-attribute"></a>Атрибут FRS-worked Path

Путь к базе данных репликации файлов.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | FRS-Working-Path                            |
| LDAP-отображаемое имя | фрсворкингпас                              |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.486                      |
| System-ID — GUID    | 1be8f173-a9ff-11d0-afe2-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
