# <a name="do-garbage-collection-extended-right"></a>Do-мусор-коллекция расширена справа

Расширенное право для принудительной сборки мусора службой каталогов. Право на управление службой каталогов для принудительного сбора мусора.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Do-мусора-сбор                |
| Display-Name | Выполнить сбор мусора                |
| Rights-GUID  | fec364e0-0a98-11d1-adbb-00c04fd8d5cd |
