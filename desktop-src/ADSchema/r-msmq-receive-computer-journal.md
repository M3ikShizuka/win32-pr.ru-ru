# <a name="msmq-receive-computer-journal-extended-right"></a>MSMQ-Receive-Computer-журнал, расширенное право

Позволяет получать сообщения из очереди журнала компьютера.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | MSMQ-Receive-Computer-Journal        |
| Display-Name | Получение журнала компьютера             |
| Rights-GUID  | 4b6e08c2-df3c-11d1-9c86-006008764d0e |
