# <a name="user-password-attribute"></a>Атрибут User-Password

Пароль пользователя в формате UTF-8. Это атрибут, доступный только для записи.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | User-Password                                         |
| LDAP-отображаемое имя | userPassword                                          |
| Размер              | \-                                                    |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 2.5.4.35                                              |
| System-ID — GUID    | bf967a6e-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
