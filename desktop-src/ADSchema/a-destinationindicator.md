|-------------------|--------------------------------------|
| CN                | Destination-Indicator                |
| LDAP-отображаемое имя | дестинатиониндикатор                 |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 2.5.4.27                             |
| System-ID — GUID    | bf967951-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
