# <a name="ms-wmi-intmax-attribute"></a>атрибут MS-WMI-Интмакс

Максимальное значение для объекта целочисленного параметра WMI 32-bit.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-WMI-Интмакс                        |
| LDAP-отображаемое имя | Мсвми — Интмакс                         |
| Размер              | 4 байта                              |
| Привилегия обновления  | Администратор групповая политика           |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1629              |
| System-ID — GUID    | fb920c2c-f294-4426-8ac1-d24b42aa2bce |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
