# <a name="dhcp-identification-attribute"></a>Атрибут идентификации DHCP

Атрибут **идентификации DHCP** в настоящее время не используется.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | DHCP-идентификация                         |
| LDAP-отображаемое имя | дхкпидентификатион                          |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.701                      |
| System-ID — GUID    | 963d2742-48be-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
