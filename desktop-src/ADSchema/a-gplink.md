# <a name="gp-link-attribute"></a>Атрибут GP-Link

Отсортированный список параметров групповая политика. Каждый параметр имеет значение **типа DWORD**. Использовать строку в Юникоде удобно.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | GP-Link                                     |
| LDAP-отображаемое имя | gPLink                                      |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.891                      |
| System-ID — GUID    | f30e3bbe-9ff0-11d1-b603-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
