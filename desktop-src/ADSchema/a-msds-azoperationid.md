# <a name="ms-ds-az-operation-id-attribute"></a>Атрибут ms-DS-AZ-Operation-ID

Идентификатор, зависящий от приложения, который делает операцию уникальной для приложения.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-AZ-Operation-ID                   |
| LDAP-отображаемое имя | msDS-Азоператионид                      |
| Размер              | \-                                      |
| Привилегия обновления  | Администратор Азролес                           |
| Частота обновления  | Во время инициализации или изменения политики. |
| Attribute-Id      | 1.2.840.113556.1.4.1800                 |
| System-ID — GUID    | a5f3b553-5d76-4cbe-ba3f-4312152cab18    |
| Синтаксис            | [**Перечисление**](s-enumeration.md)    |
