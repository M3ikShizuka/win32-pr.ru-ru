# <a name="lm-pwd-history-attribute"></a>Атрибут LM-PWD-History

Журнал паролей пользователя в одностороннем формате LAN Manager (LM). Протокол LM OWF используется для обеспечения совместимости с LAN Manager 2. *x* клиентов, Windows 95 и Windows 98.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | LM-PWD-History                                        |
| LDAP-отображаемое имя | лмпвдхистори                                          |
| Размер              | \-                                                    |
| Привилегия обновления  | Администратор домена                                  |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.160                                |
| System-ID — GUID    | bf96799d-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
