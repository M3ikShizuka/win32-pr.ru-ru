# <a name="frs-member-reference-attribute"></a>Атрибут FRS-Member-Reference

Ссылка на объект Member для этого подписчика.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | FRS — ссылка на член                    |
| LDAP-отображаемое имя | фрсмемберреференце                      |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.875                  |
| System-ID — GUID    | 2a13257e-9373-11d1-aebc-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
