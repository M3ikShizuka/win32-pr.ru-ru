# <a name="site-object-bl-attribute"></a>Атрибут site-Object-BL

Список различающихся имен для подсетей, принадлежащих этому сайту.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Site-Object-BL                          |
| LDAP-отображаемое имя | ситеобжектбл                            |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | При каждом создании или удалении сайта.  |
| Attribute-Id      | 1.2.840.113556.1.4.513                  |
| System-ID — GUID    | 3e10944d-c354-11d0-aff8-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
