# <a name="rdn-att-id-attribute"></a>Атрибут RDN-Атри-ID

RDN для атрибута, который используется для именования класса.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------|
| CN                | RDN-Атри-ID                                                      |
| LDAP-отображаемое имя | rDNAttID                                                        |
| Размер              | \-                                                              |
| Привилегия обновления  | Администратор схемы                                            |
| Частота обновления  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.26                                           |
| System-ID — GUID    | bf967a0f-0de6-11d0-a285-00aa003049e2                            |
| Синтаксис            | [**String(идентификатор_объекта)**](s-string-object-identifier.md) |
