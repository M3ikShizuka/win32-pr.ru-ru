# <a name="ms-wmi-query-attribute"></a>MS-WMI-запрос атрибута

Один WQL-запрос.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-Query                                |
| LDAP-отображаемое имя | Мсвми — запрос                                 |
| Размер              | Менее 250 символов.                   |
| Привилегия обновления  | Администратор групповая политика                  |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1642                     |
| System-ID — GUID    | 65fff93e-35e3-45a3-85ae-876c6718297f        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
