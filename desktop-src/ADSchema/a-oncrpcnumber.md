# <a name="oncrpcnumber-attribute"></a>атрибут Онкрпкнумбер

содержит часть схемы rpc и хранит номер RPC для UNIX rpc.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | онкрпкнумбер                         |
| LDAP-отображаемое имя | онкрпкнумбер                         |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.18                     |
| System-ID — GUID    | 966825f5-01d9-4a5c-a011-d15ae84efa55 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
