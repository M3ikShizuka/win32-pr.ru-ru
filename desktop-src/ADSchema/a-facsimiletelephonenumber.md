# <a name="facsimile-telephone-number-attribute"></a>Атрибут «Факс-телефон-номер»

Содержит номер телефона компьютера, на котором установлен рабочий Факс.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Факс-телефон — номер                  |
| LDAP-отображаемое имя | facsimileTelephoneNumber                    |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена или владелец учетной записи.      |
| Частота обновления  | Каждый раз, когда номер факса необходимо изменить.    |
| Attribute-Id      | 2.5.4.23                                    |
| System-ID — GUID    | bf967974-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
