# <a name="other-name-attribute"></a>Атрибут Other-Name

Дополнительные имена для пользователя. Например, отчество, патронимик, матронимик или другие.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Other-Name                                  |
| LDAP-отображаемое имя | middleName                                  |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 2.16.840.1.113730.3.1.34                    |
| System-ID — GUID    | bf9679f2-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
