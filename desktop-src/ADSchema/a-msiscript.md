# <a name="msi-script-attribute"></a>Атрибут Msi-Script

Скрипт объявления установщика Майкрософт для этого приложения.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Msi-Script                                            |
| LDAP-отображаемое имя | мсискрипт                                             |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.814                                |
| System-ID — GUID    | d9e18313-8939-11d1-aebc-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
