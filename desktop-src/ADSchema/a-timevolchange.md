# <a name="time-vol-change-attribute"></a>Атрибут «время-Vol-изменить»

Этот атрибут указывает время последнего изменения файла на удаленном томе хранилища.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Изменение времени и громкости                      |
| LDAP-отображаемое имя | тимеволчанже                        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.502               |
| System-ID — GUID    | ddac0cf0-af8f-11d0-afeb-00c04fd930c9 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
