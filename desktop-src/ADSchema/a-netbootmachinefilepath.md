# <a name="netboot-machine-file-path-attribute"></a>Нетбут-Machine-File-путь атрибут

Этот атрибут указывает сервер, который отвечает клиенту. Начиная с операционной системы Windows Server 2003, она может указывать Startrom.com, которые получает клиент.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Нетбут-Machine-File-Path                   |
| LDAP-отображаемое имя | нетбутмачинефилепас                      |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.361                      |
| System-ID — GUID    | 3e978923-8c01-11d0-afda-00c04fd930c9        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
