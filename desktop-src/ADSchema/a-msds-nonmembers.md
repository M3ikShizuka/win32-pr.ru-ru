# <a name="ms-ds-non-members-attribute"></a>Атрибут ms-DS-non-members

Этот атрибут служит той же цели, что и атрибут, не относящийся к безопасности, но с применением правил области.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-не-члены                       |
| LDAP-отображаемое имя | msDS — не члены                         |
| Размер              | \-                                      |
| Привилегия обновления  | Администратор Азролес                           |
| Частота обновления  | При инициализации и изменении политики.    |
| Attribute-Id      | 1.2.840.113556.1.4.1793                 |
| System-ID — GUID    | cafcb1de-f23c-46b5-adf7-1e64957bd5db    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
