# <a name="ms-wmi-targetobject-attribute"></a>атрибут MS-WMI-TargetObject

Он содержит одну или несколько двоичных последовательностей, представляющих скомпилированные объекты WMI.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MS-WMI-TargetObject                                   |
| LDAP-отображаемое имя | Мсвми — TargetObject                                    |
| Размер              | \-                                                    |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1647                               |
| System-ID — GUID    | c44f67a5-7de5-4a1f-92d9-662b57364b77                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
