# <a name="ds-replication-monitor-topology-extended-right"></a>DS-Replication-Monitor — топология расширена справа

Право доступа расширенного управления, которое позволяет считывать данные мониторинга репликации, такие как состояние репликации и метаданные объектов.

|--------------|--------------------------------------|
| CN           | DS-Replication-Monitor — топология      |
| Display-Name | Мониторинг репликации Active Directory |
