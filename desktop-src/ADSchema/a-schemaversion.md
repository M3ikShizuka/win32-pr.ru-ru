# <a name="schema-version-attribute"></a>Атрибут Schema-Version

Номер версии схемы.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Schema-Version                       |
| LDAP-отображаемое имя | schemaVersion                        |
| Размер              | 4 байта                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | Каждый раз при освобождении схемы.    |
| Attribute-Id      | 1.2.840.113556.1.2.471               |
| System-ID — GUID    | bf967a2c-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
