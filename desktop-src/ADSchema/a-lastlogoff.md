# <a name="last-logoff-attribute"></a>Атрибут Last-Logoff

Этот атрибут не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Last-Logoff                          |
| LDAP-отображаемое имя | ластлогофф                           |
| Размер              | 8 байт                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | При каждом выходе пользователя из системы.         |
| Attribute-Id      | 1.2.840.113556.1.4.51                |
| System-ID — GUID    | bf967996-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
