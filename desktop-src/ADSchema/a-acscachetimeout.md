# <a name="acs-cache-timeout-attribute"></a>ACS-Кэш-атрибут времени ожидания

Период времени до истечения срока действия объектов ACS, кэшируемых службой ACS.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS — Кэш — время ожидания                    |
| LDAP-отображаемое имя | акскачетимеаут                      |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.779               |
| System-ID — GUID    | 1cb355a1-56d0-11d1-a9c6-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
