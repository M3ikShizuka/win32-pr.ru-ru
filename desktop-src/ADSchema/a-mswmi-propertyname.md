# <a name="ms-wmi-propertyname-attribute"></a>атрибут MS-WMI-PropertyName

Имя целевого объекта политики для объекта параметра.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-PropertyName                         |
| LDAP-отображаемое имя | Мсвми-PropertyName                          |
| Размер              | Менее двадцати символов.                |
| Привилегия обновления  | Администратор групповая политика                  |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1641                     |
| System-ID — GUID    | ab920883-e7f8-4d72-b4a0-c0449897509d        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
