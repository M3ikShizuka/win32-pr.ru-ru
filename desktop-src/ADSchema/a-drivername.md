# <a name="driver-name-attribute"></a>Атрибут Driver-Name

Имя драйвера устройства.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Driver-Name                                 |
| LDAP-отображаемое имя | driverName                                  |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.229                      |
| System-ID — GUID    | 281416c5-1968-11d0-a28f-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
