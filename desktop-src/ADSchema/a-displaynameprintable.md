# <a name="display-name-printable-attribute"></a>Отображаемое имя — атрибут, поддерживающий печать

Отображаемое имя для печати объекта. Отображаемое имя для печати обычно представляет собой сочетание имени, отчество и фамилии пользователя.



| Ввод | Значение |
|-------------------|----------------------------------------|
| CN                | Отображаемое имя — печатаемое                 |
| LDAP-отображаемое имя | дисплайнамепринтабле                   |
| Размер              | \-                                     |
| Привилегия обновления  | Администратор домена или владелец учетной записи. |
| Частота обновления  | \-                                     |
| Attribute-Id      | 1.2.840.113556.1.2.353                 |
| System-ID — GUID    | bf967954-0de6-11d0-a285-00aa003049e2   |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)    |
