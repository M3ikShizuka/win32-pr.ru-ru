# <a name="ms-dfs-link-identity-guid-v2-attribute"></a>атрибут MS-DFS-Link-Identity-GUID-v2

Задается только при создании ссылки. Стабильны при переименовании или перемещении до тех пор, пока ссылка не заменяется другой ссылкой с таким же именем.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MS-DFS-Link-Identity-GUID-v2                          |
| LDAP-отображаемое имя | Мсдфс — LinkIdentityGUIDv2                              |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.2041                               |
| System-ID — GUID    | edb027f3-5726-4dee-8d4e-dbf07e1ad1f1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
