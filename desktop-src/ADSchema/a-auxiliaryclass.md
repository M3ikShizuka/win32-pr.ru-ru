Список вспомогательных классов, которые будут связаны с этим классом.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------|
| CN                | Auxiliary-Class                                                 |
| LDAP-отображаемое имя | auxiliaryClass                                                  |
| Размер              | \-                                                              |
| Привилегия обновления  | Администратор схемы                                            |
| Частота обновления  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.351                                          |
| System-ID — GUID    | bf96792c-0de6-11d0-a285-00aa003049e2                            |
| Синтаксис            | [**String(идентификатор_объекта)**](s-string-object-identifier.md) |
