# <a name="user-comment-attribute"></a>Атрибут User-Comment

Комментарии пользователя.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | User-Comment                                |
| LDAP-отображаемое имя | comment                                     |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена или владелец учетной записи.      |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.156                      |
| System-ID — GUID    | bf967a6a-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
