# <a name="ms-ds-isgc-attribute"></a>Атрибут ms-DS-Исгк

Определяет состояние глобального каталога на контроллере домена.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-Исгк                           |
| LDAP-отображаемое имя | msDS-Исгк                            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1959              |
| System-ID — GUID    | 1df5cf33-0fe5-499e-90e1-e94b42718a46 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
