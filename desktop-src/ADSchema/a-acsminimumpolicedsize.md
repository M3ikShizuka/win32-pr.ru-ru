# <a name="acs-minimum-policed-size-attribute"></a>ACS-минимальная-политика-атрибут размера

Атрибут " **ACS-Minimum-with-size** " предназначен только для внутреннего использования. На основе RFC2210.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS — минимальная политика-размер             |
| LDAP-отображаемое имя | аксминимумполицедсизе                |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1315              |
| System-ID — GUID    | 8d0e7195-3b90-11d2-90cc-00c04fd91ab1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
