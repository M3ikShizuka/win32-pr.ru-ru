# <a name="phone-pager-other-attribute"></a>Телефон-пейджер-другой атрибут

Список альтернативных номеров пейджеров.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Телефон-пейджер — другие                                                                |
| LDAP-отображаемое имя | otherPager                                                                       |
| Размер              | \-                                                                               |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                           |
| Частота обновления  | При создании записи пользователя и при необходимости изменения номера телефона. |
| Attribute-Id      | 1.2.840.113556.1.2.118                                                           |
| System-ID — GUID    | f0f8ffa4-1191-11d0-a060-00aa006c33ed                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
