# <a name="notification-list-attribute"></a>Атрибут Notification-List

Атрибут **списка уведомлений** в настоящее время не используется.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Notification-List                       |
| LDAP-отображаемое имя | нотификатионлист                        |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.303                  |
| System-ID — GUID    | 19195a56-6da0-11d0-afd3-00c04fd930c9    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
