# <a name="acs-identity-name-attribute"></a>ACS — атрибут Identity-Name

Этот атрибут содержит различающееся имя пользователя или подразделения и является идентификатором пользователя или группы пользователей, к которым применяется эта политика качества обслуживания.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | ACS-Identity-Name                           |
| LDAP-отображаемое имя | аксидентитинаме                             |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.784                      |
| System-ID — GUID    | dab029b6-ddf7-11d1-90a5-00c04fd91ab1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
