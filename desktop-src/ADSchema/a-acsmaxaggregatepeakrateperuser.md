# <a name="acs-max-aggregate-peak-rate-per-user-attribute"></a>Атрибут ACS-Max-Aggregate-Пиковая ставка на пользователя

| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-Max-Aggregate-Пиковая ставка-на пользователя |
| LDAP-отображаемое имя | аксмаксаггрегатепеакратеперусер       |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.897               |
| System-ID — GUID    | f072230c-aef5-11d1-bdcf-0000f80367c1 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
