# <a name="print-orientations-supported-attribute"></a>Print — ориентация — поддерживаемый атрибут

Поворот страницы для альбомной печати.



| Ввод | Значение |
|-------------------|-------------------------------------------------------------|
| CN                | Ориентация на печать — поддерживается                                |
| LDAP-отображаемое имя | принториентатионссуппортед                                  |
| Размер              | 4 байта. Возможные значения: 0, 90, 270 и 0 = нет альбомной ориентации. |
| Привилегия обновления  | \-                                                          |
| Частота обновления  | \-                                                          |
| Attribute-Id      | 1.2.840.113556.1.4.240                                      |
| System-ID — GUID    | 281416d0-1968-11d0-a28f-00aa003049e2                        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                 |
