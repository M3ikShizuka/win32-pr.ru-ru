# <a name="schema-id-guid-attribute"></a>Атрибут Schema-ID-GUID

Уникальный идентификатор атрибута.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------|
| CN                | Schema-ID-GUID                                                      |
| LDAP-отображаемое имя | schemaIDGUID                                                        |
| Размер              | 16 байт                                                            |
| Привилегия обновления  | Это значение задается системой.                                    |
| Частота обновления  | Это значение задается при создании объекта и не может быть изменено. |
| Attribute-Id      | 1.2.840.113556.1.4.148                                              |
| System-ID — GUID    | bf967923-0de6-11d0-a285-00aa003049e2                                |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md)               |
