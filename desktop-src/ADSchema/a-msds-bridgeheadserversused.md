| Ввод | Значение |
|-------------------|-------------------------------------------------|
| CN                | MS-DS-плацдарм-серверы — используется                   |
| LDAP-отображаемое имя | msDS-Бриджехеадсерверсусед                      |
| Размер              | \-                                              |
| Привилегия обновления  | \-                                              |
| Частота обновления  | \-                                              |
| Attribute-Id      | 1.2.840.113556.1.4.2049                         |
| System-ID — GUID    | 3ced1465-7b71-2541-8780-1e1ea6243a82            |
| Синтаксис            | [**Object(двоичный_DN)**](s-object-dn-binary.md) |
