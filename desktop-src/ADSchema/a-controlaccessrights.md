# <a name="control-access-rights-attribute"></a>Control-Access-Rights, атрибут

Используется службой безопасности DS для определения пользователей, которые могут выполнять определенные операции с объектом узла.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Управление — права доступа                                 |
| LDAP-отображаемое имя | контролакцессригхтс                                   |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.200                                |
| System-ID — GUID    | 6da8a4fc-0e52-11d0-a286-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
