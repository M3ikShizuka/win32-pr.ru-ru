# <a name="ms-ts-home-drive-attribute"></a>атрибут MS-TS-Home-Drive

Домашний диск служб терминалов указывает домашний диск для пользователя.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-TS-Home-Drive                            |
| LDAP-отображаемое имя | мстшомедриве                               |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1978                     |
| System-ID — GUID    | 5f0a24d9-dffa-4cd9-acbf-a0680c03731e        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
