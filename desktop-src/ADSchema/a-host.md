# <a name="host-attribute"></a>атрибут узла

Указывает главный компьютер.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | узел                                        |
| LDAP-отображаемое имя | узел                                        |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.9                   |
| System-ID — GUID    | 6043df71-fa48-46cf-ab7c-cbd54644b22d        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
