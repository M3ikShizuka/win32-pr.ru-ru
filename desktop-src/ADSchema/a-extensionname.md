# <a name="extension-name-attribute"></a>Атрибут Extension-Name

Имя страницы свойств, используемой для расширения пользовательского интерфейса объекта каталога.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Extension-Name                              |
| LDAP-отображаемое имя | extensionName                               |
| Размер              | \-                                          |
| Привилегия обновления  | Создатель объекта расширения.        |
| Частота обновления  | При создании нового расширения пользовательского интерфейса.     |
| Attribute-Id      | 1.2.840.113556.1.2.227                      |
| System-ID — GUID    | bf967972-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
