# <a name="com-treat-as-class-id-attribute"></a>Атрибут COM-Treat-As-Class-ID

Treat-As строковый идентификатор CLSID для класса узла.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | COM-интерпретировать-As-Class-ID                       |
| LDAP-отображаемое имя | комтреатасклассид                           |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.251                      |
| System-ID — GUID    | 281416db-1968-11d0-a28f-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
