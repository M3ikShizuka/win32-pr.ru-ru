# <a name="ms-ds-cached-membership-attribute"></a>атрибут членства MS-DS-Cache-Membership

Атрибут " **MS-DS-Cache-Membership** " используется диспетчером учетных записей безопасности для расширения группы во время оценки маркера.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MS-DS-Cache-Membership                               |
| LDAP-отображаемое имя | msDS — кэширование членства                                |
| Размер              | Двоичный BLOB-объект размером до 3 килобайт.                     |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1441                               |
| System-ID — GUID    | 69cab008-cdd4-4bc9-bab8-0ff37efe1b20                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
