# <a name="ms-dfsr-extension-attribute"></a>атрибут MS-DFSR-Extension

Содержит атрибут расширения репликации распределенная файловая система (DFS).



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MS-DFSR-Extension                                     |
| LDAP-отображаемое имя | Мсдфср — расширение                                      |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.2                             |
| System-ID — GUID    | 78f011ec-a766-4b19-adcf-7b81ed781a4d                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
