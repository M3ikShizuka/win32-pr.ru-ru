# <a name="enabled-connection-attribute"></a>Атрибут Enabled-Connection

Указывает, доступно ли подключение для использования.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Enabled-Connection                   |
| LDAP-отображаемое имя | енабледконнектион                    |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.36                |
| System-ID — GUID    | bf967963-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
