# <a name="msmq-sign-certificates-attribute"></a>Атрибут MSMQ-Sign-Certificates

Этот атрибут содержит несколько сертификатов. Пользователь может создать сертификат для каждого компьютера. Для каждого сертификата также сохраняется дайджест.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------------|
| CN                | MSMQ-Sign-Certificates                                                                 |
| LDAP-отображаемое имя | мсмксигнцертификатес                                                                   |
| Размер              | Тип атрибута — большой двоичный объект, размер не ограничен, и данные хранятся в собственном формате. |
| Привилегия обновления  | \-                                                                                     |
| Частота обновления  | \-                                                                                     |
| Attribute-Id      | 1.2.840.113556.1.4.947                                                                 |
| System-ID — GUID    | 9a0dc33b-c100-11d1-bbc5-0080c76670c0                                                   |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md)                                  |
