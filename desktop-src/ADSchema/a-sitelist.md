# <a name="site-list-attribute"></a>Атрибут Site-List

Список сайтов, подключенных к этому объекту ссылки.



| Ввод | Значение |
|-------------------|----------------------------------------------|
| CN                | Site-List                                    |
| LDAP-отображаемое имя | сителист                                     |
| Размер              | \-                                           |
| Привилегия обновления  | Администратор домена                         |
| Частота обновления  | При каждом добавлении или удалении сайта из. |
| Attribute-Id      | 1.2.840.113556.1.4.821                       |
| System-ID — GUID    | d50c2cdc-8951-11d1-aebc-0000f80367c1         |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md)      |
