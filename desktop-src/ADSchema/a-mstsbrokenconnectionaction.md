# <a name="ms-ts-broken-connection-action-attribute"></a>атрибут MS-TS-сломанных подключений — действие

Действие подключения "неработающий сеанс служб терминалов" указывает действие, выполняемое при достижении предела для сеанса служб терминалов.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-TS-нарушено подключение — действие       |
| LDAP-отображаемое имя | мстсброкенконнектионактион           |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1985              |
| System-ID — GUID    | 1cf41bba-5604-463e-94d6-1a1287b72ca3 |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
