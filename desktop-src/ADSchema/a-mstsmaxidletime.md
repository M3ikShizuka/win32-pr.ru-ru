# <a name="ms-ts-max-idle-time-attribute"></a>атрибут MS-TS-Max-Idle время простоя

Максимальное время простоя сеанса служб терминалов — это максимальный период времени (в минутах), в течение которого сеанс служб терминалов может оставаться бездействующим.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-TS-Max-время простоя                  |
| LDAP-отображаемое имя | мстсмаксидлетиме                      |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1983              |
| System-ID — GUID    | ff739e9c-6bb7-460e-b221-e250f3de0f95 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
