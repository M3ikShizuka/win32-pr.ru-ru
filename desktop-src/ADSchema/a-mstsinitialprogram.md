# <a name="ms-ts-initial-program-attribute"></a>MS-TS-Initial-атрибут программы

Начальная программа сеанса служб терминалов указывает путь и имя файла приложения, которое пользователь хочет запустить автоматически при входе пользователя на сервер терминалов.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-TS-Initial-программа                       |
| LDAP-отображаемое имя | мстсинитиалпрограм                          |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1990                     |
| System-ID — GUID    | 9201ac6f-1d69-4dfb-802e-d95510109599        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
