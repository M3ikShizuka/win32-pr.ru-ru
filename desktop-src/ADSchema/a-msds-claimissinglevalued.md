# <a name="ms-ds-claim-is-single-valued-attribute"></a>Атрибут ms-DS-заявка с одним значением

Для объекта типа утверждения этот атрибут определяет, может ли тип утверждения или свойство ресурса содержать только одно значение.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-заявка-является однозначным         |
| LDAP-отображаемое имя | msDS-Клаимиссинглевалуед             |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2160              |
| System-ID — GUID    | cd789fb9-96b4-4648-8219-ca378161af38 |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
