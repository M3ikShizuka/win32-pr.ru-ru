# <a name="fsmo-role-owner-attribute"></a>Атрибут FSMO-Role-Owner

Гибкая операция Single-Master: различающееся имя контроллера домена, в котором можно изменить схему.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | FSMO-Role-Owner                         |
| LDAP-отображаемое имя | фсморолеовнер                           |
| Размер              | \-                                      |
| Привилегия обновления  | Администратор схемы                    |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.369                  |
| System-ID — GUID    | 66171887-8f3c-11d0-afda-00c04fd930c9    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
