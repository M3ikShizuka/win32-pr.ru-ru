# <a name="ms-ds-krbtgt-link-attribute"></a>Атрибут ms-DS-KrbTgt-Link

Используется с контроллером RODC для определения \_ учетной записи KRBTGT XXXX, соответствующей каждому контроллеру домена только для чтения.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-KrbTgt-Link                       |
| LDAP-отображаемое имя | msDS-Крбтгтлинк                         |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1923                 |
| System-ID — GUID    | 778ff5c9-6f4e-4b74-856a-d68383313910    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
