# <a name="ms-ds-az-ldap-query-attribute"></a>Атрибут ms-DS-AZ-LDAP-Query

Строка, определяющая запрос LDAP (максимальная длина 4096), который определяет членство объекта-пользователя в группе.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-AZ-LDAP-Query                         |
| LDAP-отображаемое имя | msDS-Азлдапкуери                            |
| Размер              | 4096 символов                             |
| Привилегия обновления  | Администратор Азролес                               |
| Частота обновления  | Во время инициализации или изменения политики.     |
| Attribute-Id      | 1.2.840.113556.1.4.1792                     |
| System-ID — GUID    | 5e53368b-fc94-45c8-9d7d-daf31ee7112d        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
