Ссылка на запись сервера RPC или другую группу RPC.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | RPC-NS-Group                                |
| LDAP-отображаемое имя | рпкнсграуп                                  |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.114                      |
| System-ID — GUID    | bf967a24-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
