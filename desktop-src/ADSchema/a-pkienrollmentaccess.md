# <a name="pki-enrollment-access-attribute"></a>PKI — атрибут доступа для регистрации

Атрибут **PKI — регистрация доступа** предназначен только для внутреннего использования.



| Ввод | Значение |
|-------------------|-----------------------------------------------------|
| CN                | PKI — доступ к регистрации                               |
| LDAP-отображаемое имя | пкиенроллментакцесс                                 |
| Размер              | \-                                                  |
| Привилегия обновления  | \-                                                  |
| Частота обновления  | \-                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.1335                             |
| System-ID — GUID    | 926be278-56f9-11d2-90d0-00c04fd91ab1                |
| Синтаксис            | [**String(NT-Sec-Desc)**](s-string-nt-sec-desc.md) |
