# <a name="remote-server-name-attribute"></a>Атрибут Remote-Server-Name

Используется везде, где необходимо хранить одно или несколько имен компьютеров.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Имя удаленного сервера                          |
| LDAP-отображаемое имя | ремотесервернаме                            |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.105                      |
| System-ID — GUID    | bf967a12-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
