# <a name="generation-qualifier-attribute"></a>Атрибут Generation-Qualifier

Указывает создание пользователя. Например, Jr. или II.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Generation-Qualifier                        |
| LDAP-отображаемое имя | женератионкуалифиер                         |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 2.5.4.44                                    |
| System-ID — GUID    | 16775804-47f3-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
