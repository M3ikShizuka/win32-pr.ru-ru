# <a name="last-backup-restoration-time-attribute"></a>Последний-резервный — атрибут времени восстановления

Когда произошло последнее восстановление системы.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Последнее резервное копирование — время восстановления         |
| LDAP-отображаемое имя | ластбаккупресторатионтиме            |
| Размер              | 8 байт                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.519               |
| System-ID — GUID    | 1fbb0be8-ba63-11d0-afef-0000f80367c1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
