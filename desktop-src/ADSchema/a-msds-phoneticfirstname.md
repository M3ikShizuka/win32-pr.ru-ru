# <a name="ms-ds-phonetic-first-name-attribute"></a>Атрибут ms-DS-фонетического имени

Содержит фонетическое имя или имя пользователя.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-фонетическо-First-Name                   |
| LDAP-отображаемое имя | msDS-Фонетикфирстнаме                      |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1942                     |
| System-ID — GUID    | 4b1cba4e-302f-4134-ac7c-f01f6c797843        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
