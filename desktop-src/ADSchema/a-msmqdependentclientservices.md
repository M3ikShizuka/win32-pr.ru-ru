# <a name="msmq-dependent-client-services-attribute"></a>Атрибут "MSMQ-Dependent-Client-Services"

Указывает, предоставляет ли MSMQ, установленный на этом компьютере, службы клиента, зависимые от MSMQ.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MSMQ-Dependent-Client-Services       |
| LDAP-отображаемое имя | мсмкдепендентклиентсервицес          |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1226              |
| System-ID — GUID    | 2df90d76-009f-11d2-aa4c-00c04fd7d83a |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
