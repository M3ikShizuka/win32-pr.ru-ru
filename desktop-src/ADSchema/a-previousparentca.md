# <a name="previous-parent-ca-attribute"></a>Атрибут "предыдущий — родительский ЦС"

Ссылка на центры сертификации, которые выдавали последний сертификат с истекшим сроком действия для центра сертификации.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Предыдущий-родительский ЦС                      |
| LDAP-отображаемое имя | превиауспарентка                        |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.694                  |
| System-ID — GUID    | 963d273d-48be-11d1-a9c3-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
