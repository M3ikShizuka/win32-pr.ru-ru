# <a name="ldap-ipdeny-list-attribute"></a>Атрибут списка LDAP-Ипдени-List

Содержит список двоичных IP-адресов, которым запрещен доступ к серверу LDAP.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Список LDAP-Ипдени-List                                      |
| LDAP-отображаемое имя | лдапипденилист                                        |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.844                                |
| System-ID — GUID    | 7359a353-90f7-11d1-aebc-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
