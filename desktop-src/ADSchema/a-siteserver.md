# <a name="site-server-attribute"></a>Атрибут Site-Server

Лицензирование основного сервера для данного сайта.



| Ввод | Значение |
|-------------------|----------------------------------------------|
| CN                | Site-Server                                  |
| LDAP-отображаемое имя | siteServer                                   |
| Размер              | \-                                           |
| Привилегия обновления  | Администратор домена                         |
| Частота обновления  | Каждый раз, когда необходимо изменить сайт лицензирования. |
| Attribute-Id      | 1.2.840.113556.1.4.494                       |
| System-ID — GUID    | 1be8f17c-a9ff-11d0-afe2-00c04fd930c9         |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md)      |
