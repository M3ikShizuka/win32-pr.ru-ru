# <a name="mapi-id-attribute"></a>Атрибут идентификатора MAPI

Целое число, по которому клиенты MAPI определяют этот атрибут.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ИДЕНТИФИКАТОР MAPI                              |
| LDAP-отображаемое имя | mAPIID                               |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.49                |
| System-ID — GUID    | bf9679b7-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
