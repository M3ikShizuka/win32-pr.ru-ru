# <a name="domain-replica-attribute"></a>Атрибут Domain-Replica

строковый атрибут юникода предоставляет список контроллеров домена репликации Windows NT 4,0.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Domain-Replica                              |
| LDAP-отображаемое имя | домаинреплика                               |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.158                      |
| System-ID — GUID    | bf96795e-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
