# <a name="ms-dfsr-rootpath-attribute"></a>атрибут MS-DFSR-RootPath

Содержит полный путь к корневому каталогу.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DFSR-RootPath                            |
| LDAP-отображаемое имя | Мсдфср — RootPath                             |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.3                   |
| System-ID — GUID    | d7d5e8c1-e61f-464f-9fcf-20bbe0a2ec54        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
