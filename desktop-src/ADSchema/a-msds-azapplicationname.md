# <a name="ms-ds-az-application-name-attribute"></a>Атрибут ms-DS-AZ-Application-Name

Строка, однозначно идентифицирующая объект приложения.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-AZ-имя приложения                   |
| LDAP-отображаемое имя | msDS-Азаппликатионнаме                      |
| Размер              | 128 символов                              |
| Привилегия обновления  | Администратор Азролес                               |
| Частота обновления  | Во время инициализации или изменения политики.     |
| Attribute-Id      | 1.2.840.113556.1.4.1798                     |
| System-ID — GUID    | db5b0728-6208-4876-83b7-95d3e5695275        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
