# <a name="partial-attribute-set-attribute"></a>Атрибут разделяемого атрибута Set

Отслеживает состояние внутренней репликации частичных реплик (то есть в глобальных каталогах). Атрибут объекта NC частичной реплики. Определяет набор атрибутов, имеющихся в определенном NC частичной реплики.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Разделяемый атрибут-set                                 |
| LDAP-отображаемое имя | партиалаттрибутесет                                   |
| Размер              | \-                                                    |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | Во время репликации                                    |
| Attribute-Id      | 1.2.840.113556.1.4.640                                |
| System-ID — GUID    | 19405b9e-3cfa-11d1-a9c0-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
