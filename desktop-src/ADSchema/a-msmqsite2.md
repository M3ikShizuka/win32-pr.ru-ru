# <a name="msmq-site-2-attribute"></a>Атрибут MSMQ-Site-2

Различающееся имя второго сайта подключенной пары.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MSMQ-Site-2                             |
| LDAP-отображаемое имя | mSMQSite2                               |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | При каждом создании строки MSMQ-site-Line. |
| Attribute-Id      | 1.2.840.113556.1.4.944                  |
| System-ID — GUID    | 9a0dc338-c100-11d1-bbc5-0080c76670c0    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
