# <a name="validated-spn-validated-writes"></a>Проверено — записей с проверкой имени участника-службы

Проверено разрешение на запись для включения параметра атрибута имени субъекта-службы, соответствующего имени узла DNS компьютера.



| Ввод | Значение |
|--------------|-------------------------------------------|
| CN           | Проверено — имя участника-службы                             |
| Display-Name | Удостоверенная запись в имя участника-службы |
| Rights-GUID  | f3a64788-5306-11d1-a9c5-0000f80367c1      |
