# <a name="frs-update-timeout-attribute"></a>FRS-Update — атрибут timeout

Максимальное время (в минутах) ожидания завершения обновления.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | FRS-Update — время ожидания                   |
| LDAP-отображаемое имя | фрсупдатетимеаут                     |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.485               |
| System-ID — GUID    | 1be8f172-a9ff-11d0-afe2-00c04fd930c9 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
