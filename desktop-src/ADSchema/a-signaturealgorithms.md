# <a name="signature-algorithms-attribute"></a>Атрибут Signature-Algorithms

Этот атрибут указывает тип алгоритма, который должен использоваться для декодирования цифровой подписи в процессе проверки подлинности.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Signature-Algorithms                        |
| LDAP-отображаемое имя | сигнатуреалгорисмс                         |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.824                      |
| System-ID — GUID    | 2a39c5b2-8960-11d1-aebc-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
