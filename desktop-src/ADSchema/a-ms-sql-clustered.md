# <a name="ms-sql-clustered-attribute"></a>MS-SQL-кластеризованный атрибут

Значение true, если сервер кластеризован.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-SQL-Clustered                     |
| LDAP-отображаемое имя | mS-SQL-Clustered                     |
| Размер              | 4 байта                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | При запуске системы.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1373              |
| System-ID — GUID    | 7778bd90-ccee-11d2-9993-0000f87a57d4 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
