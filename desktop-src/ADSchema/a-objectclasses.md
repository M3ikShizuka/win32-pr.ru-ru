# <a name="object-classes-attribute"></a>Атрибут Object-Classes

Многозначное свойство, содержащее строки, представляющие каждый класс в схеме. Каждое значение содержит governsID, lDAPDisplayName, mustContain, mayContain и т. д.



| Ввод | Значение |
|-------------------|--------------------------------------------------|
| CN                | Object-Classes                                   |
| LDAP-отображаемое имя | обжектклассес                                    |
| Размер              | Примерно 20 байт в среднем на имя класса.        |
| Привилегия обновления  | Это значение задается конструктором объекта. |
| Частота обновления  | При изменении структуры класса.        |
| Attribute-Id      | 2.5.21.6                                         |
| System-ID — GUID    | 9a7ad94b-ca53-11d1-bbd0-0080c76670c0             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)      |
