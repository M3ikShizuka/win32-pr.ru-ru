# <a name="address-syntax-attribute"></a>Атрибут Address-Syntax

Грамматика для кодирования свойств отображаемой таблицы в виде строки.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Address-Syntax                                        |
| LDAP-отображаемое имя | аддресссинтакс                                         |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.255                                |
| System-ID — GUID    | 5fd42463-1262-11d0-a060-00aa006c33ed                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
