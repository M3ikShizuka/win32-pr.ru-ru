# <a name="ms-ds-revealed-users-attribute"></a>Атрибут ms-DS-Directory-Users

Используется с контроллером RODC для обнаружения объектов пользователя, секреты которых были предоставлены для RODC.



| Ввод | Значение |
|-------------------|-------------------------------------------------|
| CN                | MS-DS-выявили пользователей                            |
| LDAP-отображаемое имя | msDS-Ревеаледусерс                              |
| Размер              | \-                                              |
| Привилегия обновления  | \-                                              |
| Частота обновления  | \-                                              |
| Attribute-Id      | 1.2.840.113556.1.4.1924                         |
| System-ID — GUID    | 185c7821-3749-443a-bd6a-288899071adb            |
| Синтаксис            | [**Object(двоичный_DN)**](s-object-dn-binary.md) |
