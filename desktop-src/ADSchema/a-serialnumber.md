# <a name="serial-number-attribute"></a>Атрибут Serial-Number

Часть спецификации X. 500. Не используется Active Directory.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Serial-Number                        |
| LDAP-отображаемое имя | серийный номер                         |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 2.5.4.5                              |
| System-ID — GUID    | bf967a32-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
