# <a name="extended-class-info-attribute"></a>Атрибут расширенного класса info

Многозначное свойство, содержащее строки, представляющие дополнительные сведения для каждого класса. Каждое значение содержит governsID, lDAPDisplayName и schemaIDGUID класса.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Расширенные сведения о классе                         |
| LDAP-отображаемое имя | екстендедклассинфо                           |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.908                      |
| System-ID — GUID    | 9a7ad948-ca53-11d1-bbd0-0080c76670c0        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
