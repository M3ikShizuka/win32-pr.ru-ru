# <a name="employee-id-attribute"></a>Атрибут Employee-ID

Идентификатор сотрудника.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Идентификатор сотрудника                                 |
| LDAP-отображаемое имя | employeeID                                  |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.35                       |
| System-ID — GUID    | bf967962-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
