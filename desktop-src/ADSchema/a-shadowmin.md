# <a name="shadowmin-attribute"></a>атрибут Шадовмин

Содержит минимальное число дней между теневыми изменениями.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | шадовмин                            |
| LDAP-отображаемое имя | шадовмин                            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.6                      |
| System-ID — GUID    | a76b8737-e5a1-4568-b057-dc12e04be4b2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
