# <a name="acs-priority-attribute"></a>Атрибут ACS-Priority

Относительный приоритет последовательности для этого пользователя.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-Priority                         |
| LDAP-отображаемое имя | аксприорити                          |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.764               |
| System-ID — GUID    | 7f561281-5301-11d1-a9c5-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
