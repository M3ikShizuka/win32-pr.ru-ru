|-------------------|---------------------------------------------|
| CN                | MS-PKI-Site-Name                            |
| LDAP-отображаемое имя | Мспки-Site-Name                             |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2077                     |
| System-ID — GUID    | 0cd8711f-0afc-4926-a4b1-09b08d3d436c        |
