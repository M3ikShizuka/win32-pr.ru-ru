# <a name="msnpcallingstationid-attribute"></a>атрибут Мснпкаллингстатионид

Атрибут **мснпкаллингстатионид** используется внутренним образом. Не изменяйте это значение напрямую.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | мснпкаллингстатионид                 |
| LDAP-отображаемое имя | мснпкаллингстатионид                 |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1124              |
| System-ID — GUID    | db0c908a-c1f2-11d1-bbc5-0080c76670c0 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
