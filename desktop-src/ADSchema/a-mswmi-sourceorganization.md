# <a name="ms-wmi-sourceorganization-attribute"></a>атрибут MS-WMI-Саурцеорганизатион

Бизнес-организация, которая изначально создала объект политики.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-Саурцеорганизатион                   |
| LDAP-отображаемое имя | Мсвми — Саурцеорганизатион                    |
| Размер              | Менее 50 символов.                 |
| Привилегия обновления  | Администратор групповая политика                  |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1644                     |
| System-ID — GUID    | 34f7ed6c-615d-418d-aa00-549a7d7be03e        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
