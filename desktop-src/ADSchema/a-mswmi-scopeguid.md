# <a name="ms-wmi-scopeguid-attribute"></a>атрибут MS-WMI-Скопегуид

Идентификатор GUID для области, в которой находится связанная кодировка.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------|
| CN                | MS-WMI-Скопегуид                                                |
| LDAP-отображаемое имя | Мсвми — Скопегуид                                                 |
| Размер              | 16 байт                                                        |
| Привилегия обновления  | Администратор домена                                            |
| Частота обновления  | Каждый раз при добавлении или изменении класса, содержащего атрибут. |
| Attribute-Id      | 1.2.840.113556.1.4.1686                                         |
| System-ID — GUID    | 87b78d51-405f-4b7f-80ed-2bd28786f48d                            |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                     |
