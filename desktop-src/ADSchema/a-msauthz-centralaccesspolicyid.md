# <a name="ms-authz-central-access-policy-id-attribute"></a>атрибут MS-authz-Central-Access-Policy-ID

Для централизованной политики доступа этот атрибут определяет идентификатор GUID, который может использоваться для определения набора политик при применении к ресурсу.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-authz-Центральный-Access — Policy-ID    |
| LDAP-отображаемое имя | Мсаусз — Централакцессполициид        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2154              |
| System-ID — GUID    | 62f29b60-be74-4630-9456-2f6691993a86 |
| Синтаксис            | [**Строка (SID)**](s-string-sid.md)  |
