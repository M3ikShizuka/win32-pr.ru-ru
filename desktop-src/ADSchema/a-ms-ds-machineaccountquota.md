# <a name="ms-ds-machine-account-quota-attribute"></a>Атрибут MS-DS-Machine-Account-Quota

Число учетных записей компьютеров, которые пользователь может создать в домене.



| Ввод | Значение |
|-------------------|--------------------------------------------------|
| CN                | MS-DS-Machine-Account — квота                      |
| LDAP-отображаемое имя | MS-DS-Мачинеаккаунткуота                        |
| Размер              | 4 байта                                          |
| Привилегия обновления  | Администратор домена                             |
| Частота обновления  | Каждый раз, когда необходимо изменить квоту для домена. |
| Attribute-Id      | 1.2.840.113556.1.4.1411                          |
| System-ID — GUID    | d064fb68-1480-11d3-91c1-0000f87a57d4             |
| Синтаксис            | [**Перечисление**](s-enumeration.md)             |
