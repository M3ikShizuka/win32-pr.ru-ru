# <a name="netboot-scp-bl-attribute"></a>атрибут нетбут-SCP-BL

Список точек подключения службы, которые ссылаются на этот сервер Нетбут.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | нетбут-SCP-BL                          |
| LDAP-отображаемое имя | нетбутскпбл                            |
| Размер              | \-                                      |
| Привилегия обновления  | Это значение задается системой.        |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.864                  |
| System-ID — GUID    | 07383082-91df-11d1-aebc-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
