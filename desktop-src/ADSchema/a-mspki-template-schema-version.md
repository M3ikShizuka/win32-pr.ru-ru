# <a name="ms-pki-template-schema-version-attribute"></a>атрибут MS-PKI-Template-Schema-Version

Отслеживает обновления схемы класса PKI-Certificate-Template.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-Template-Schema-Version                                                                    |
| LDAP-отображаемое имя | Мспки-Template-Schema-Version                                                                     |
| Размер              | 4 байта                                                                                           |
| Привилегия обновления  | Администратор домена                                                                              |
| Частота обновления  | При изменении, создании или клонировании объекта шаблона сертификата (MS-PKI-Certificate-Template). |
| Attribute-Id      | 1.2.840.113556.1.4.1434                                                                           |
| System-ID — GUID    | 0c15e9f5-491d-4594-918f-32813a091da9                                                              |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                                                              |
