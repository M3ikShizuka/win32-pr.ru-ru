# <a name="domain-other-parameters-property-set"></a>Набор свойств "домен — другие параметры"

Набор свойств, который позволяет элементу управления получить список атрибутов домена.



| Ввод | Значение |
|--------------|------------------------------------------|
| CN           | Домен — другие параметры                  |
| Display-Name | Другие параметры домена (для использования SAM) |
| Rights-GUID  | b8119fd0-04f6-4762-ab7a-4986c76b3f9a     |
