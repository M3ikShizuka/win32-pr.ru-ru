# <a name="print-keep-printed-jobs-attribute"></a>Атрибут Printing-держитесь-Jobs

**Значение true** , если напечатанные задания сохраняются.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Печать — сохранения — задания              |
| LDAP-отображаемое имя | printKeepPrintedJobs                 |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.275               |
| System-ID — GUID    | ba305f6d-47e3-11d0-a1a6-00c04fd930c9 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
