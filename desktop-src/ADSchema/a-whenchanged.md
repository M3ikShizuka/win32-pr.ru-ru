# <a name="when-changed-attribute"></a>Атрибут When-Changed

Дата последнего изменения этого объекта. Это значение не реплицируется и существует в глобальном каталоге.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------|
| CN                | When-Changed                                                  |
| LDAP-отображаемое имя | whenChanged                                                   |
| Размер              | 8 байт                                                       |
| Привилегия обновления  | Это значение задается системой.                              |
| Частота обновления  | При каждом изменении объекта.                              |
| Attribute-Id      | 1.2.840.113556.1.2.3                                          |
| System-ID — GUID    | bf967a77-0de6-11d0-a285-00aa003049e2                          |
| Синтаксис            | [**String(обобщенное_время)**](s-string-generalized-time.md) |
