# <a name="dsa-signature-attribute"></a>Атрибут DSA-Signature

DSA-Signature объекта — это идентификатор вызова последнего каталога для изменения объекта.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | DSA-Signature                                         |
| LDAP-отображаемое имя | дсасигнатуре                                          |
| Размер              | \-                                                    |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | При каждом изменении объекта.                       |
| Attribute-Id      | 1.2.840.113556.1.2.74                                 |
| System-ID — GUID    | 167757bc-47f3-11d1-a9c3-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
