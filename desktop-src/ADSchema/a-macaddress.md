# <a name="macaddress-attribute"></a>атрибут macAddress

Содержит MAC-адрес в максимальном шестнадцатеричном формате, разделенном двоеточием.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | macAddress                           |
| LDAP-отображаемое имя | macAddress                           |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.22                     |
| System-ID — GUID    | e6a522dd-9770-43e1-89de-1de5044328f7 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
