# <a name="gidnumber-attribute"></a>атрибут gidNumber

Содержит целочисленное значение, уникально идентифицирующее группу в административном домене.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | gidNumber                            |
| LDAP-отображаемое имя | gidNumber                            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.1                      |
| System-ID — GUID    | c5b95f0c-ec9e-41c4-849c-b46597ed6696 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
