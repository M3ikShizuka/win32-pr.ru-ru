# <a name="ms-pki-minimal-key-size-attribute"></a>атрибут MS-PKI-минимальный-Key-size

Указывает минимальный размер закрытого ключа.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-минимальный-ключ-размер                                                                           |
| LDAP-отображаемое имя | Мспки — минимальный размер ключа                                                                            |
| Размер              | 4 байта                                                                                           |
| Привилегия обновления  | Администратор домена                                                                              |
| Частота обновления  | При изменении, создании или клонировании объекта шаблона сертификата (MS-PKI-Certificate-Template). |
| Attribute-Id      | 1.2.840.113556.1.4.1433                                                                           |
| System-ID — GUID    | e96a63f5-417f-46d3-be52-db7703c503df                                                              |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                                                              |
