# <a name="ms-rras-vendor-attribute-entry-attribute"></a>атрибут MS-RRAS-Vendor-Attribute-Entry

Строка, позволяющая поставщикам добавлять в DS атрибуты маршрутизатора для использования в запросах в формате AttributeName: vendorID: AttributeType.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-RRAS-Vendor-Attribute-Entry              |
| LDAP-отображаемое имя | мсррасвендораттрибутинтри                  |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.883                      |
| System-ID — GUID    | f39b98ac-938d-11d1-aebd-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
