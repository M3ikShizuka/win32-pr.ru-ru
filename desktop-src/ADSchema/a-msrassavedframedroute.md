# <a name="msrassavedframedroute-attribute"></a>атрибут Мсрассаведфрамедрауте

Атрибут **мсрассаведфрамедрауте** используется внутренним образом. Не изменяйте это значение напрямую.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | мсрассаведфрамедрауте                |
| LDAP-отображаемое имя | мсрассаведфрамедрауте                |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1191              |
| System-ID — GUID    | db0c90c7-c1f2-11d1-bbc5-0080c76670c0 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
