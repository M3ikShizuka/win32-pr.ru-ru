# <a name="ms-ts-max-disconnection-time-attribute"></a>Атрибут времени MS-TS-Max-unconnection-Time

Максимальное время отключения сеанса служб терминалов — это максимальный период времени (в минутах), в течение которого отключенный сеанс служб терминалов остается активным на сервере терминалов.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-TS-Max-отключение-время         |
| LDAP-отображаемое имя | мстсмаксдисконнектионтиме             |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1981              |
| System-ID — GUID    | 326f7089-53d8-4784-b814-46d8535110d2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
