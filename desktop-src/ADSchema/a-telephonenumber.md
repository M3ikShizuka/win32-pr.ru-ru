# <a name="telephone-number-attribute"></a>Атрибут Telephone-Number

Основной номер телефона.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Telephone-Number                                                                 |
| LDAP-отображаемое имя | TelephoneNumber                                                                  |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 2.5.4.20                                                                         |
| System-ID — GUID    | bf967a49-0de6-11d0-a285-00aa003049e2                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
