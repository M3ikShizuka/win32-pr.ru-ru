# <a name="service-principal-name-attribute"></a>Атрибут Service-Principal-Name

Список имен участников, используемых для взаимной проверки подлинности с экземпляром службы на этом компьютере.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Имя участника-службы                      |
| LDAP-отображаемое имя | servicePrincipalName                        |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.771                      |
| System-ID — GUID    | f3a64788-5306-11d1-a9c5-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
