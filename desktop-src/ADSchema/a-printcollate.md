# <a name="print-collate-attribute"></a>Атрибут Print-Collate

**Значение true** , если у принтера есть параметры сортировки ячеек.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Print-Collate                        |
| LDAP-отображаемое имя | printCollate                         |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.242               |
| System-ID — GUID    | 281416d2-1968-11d0-a28f-00aa003049e2 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
