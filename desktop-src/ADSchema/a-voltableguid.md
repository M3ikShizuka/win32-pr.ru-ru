# <a name="vol-table-guid-attribute"></a>Атрибут Vol-Table-GUID

Уникальный идентификатор для записи таблицы Link-Track-Volume.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Vol-Таблица-GUID                                        |
| LDAP-отображаемое имя | волтаблегуид                                          |
| Размер              | 16 байт                                              |
| Привилегия обновления  | Это значение задается системой.                      |
| Частота обновления  | При создании новой ссылки на файл.             |
| Attribute-Id      | 1.2.840.113556.1.4.336                                |
| System-ID — GUID    | 1f0075fd-7e40-11d0-afd6-00c04fd930c9                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
