# <a name="ms-rras-attribute-attribute"></a>атрибут MS-RRAS-Attribute

Строка, содержащая идентификатор vendorID: Number, установленный на маршрутизаторе. VendorID — это поставщик SMI, а номер — номер атрибута MS-RRAS, определенный в объекте словарь удостоверений маршрутизатора.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-RRAS-атрибут                           |
| LDAP-отображаемое имя | мсррасаттрибуте                             |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.884                      |
| System-ID — GUID    | f39b98ad-938d-11d1-aebd-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
