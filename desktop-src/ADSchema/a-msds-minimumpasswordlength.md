# <a name="ms-ds-minimum-password-length-attribute"></a>Атрибут ms-DS-Minimum-password-length

Минимальная длина паролей учетных записей пользователей.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DS-минимальная длина пароля        |
| LDAP-отображаемое имя | msDS-Минимумпассвордленгс           |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2013              |
| System-ID — GUID    | b21b3439-4c3a-441c-bb5f-08f20e9b315e |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
