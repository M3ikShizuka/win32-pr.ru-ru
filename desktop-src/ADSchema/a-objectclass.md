# <a name="object-class-attribute"></a>Атрибут Object-Class

Список классов, производным от которого является этот класс.



| Ввод | Значение |
|-------------------|-----------------------------------------------------------------|
| CN                | Object-Class                                                    |
| LDAP-отображаемое имя | objectClass                                                     |
| Размер              | Примерно 20 байт в среднем.                                      |
| Привилегия обновления  | Это значение задается конструктором объекта.                |
| Частота обновления  | Это значение никогда не должно изменяться.                                 |
| Attribute-Id      | 2.5.4.0                                                         |
| System-ID — GUID    | bf9679e5-0de6-11d0-a285-00aa003049e2                            |
| Синтаксис            | [**String(идентификатор_объекта)**](s-string-object-identifier.md) |
