# <a name="primary-group-id-attribute"></a>Атрибут PRIMARY-Group-ID

Содержит относительный идентификатор (RID) для основной группы пользователя. По умолчанию это RID для группы "Пользователи домена".



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ИДЕНТИФИКАТОР первичной группы                     |
| LDAP-отображаемое имя | примариграупид                       |
| Размер              | 4 байта                              |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.98                |
| System-ID — GUID    | bf967a00-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
