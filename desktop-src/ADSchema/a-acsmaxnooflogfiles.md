# <a name="acs-max-no-of-log-files-attribute"></a>Атрибут ACS-Max-No-of-log-files

Максимальное число файлов журнала RSVP.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-Max-No-of-log-files              |
| LDAP-отображаемое имя | аксмакснуфлогфилес                   |
| Размер              | 4 байта                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.774               |
| System-ID — GUID    | 1cb3559c-56d0-11d1-a9c6-0000f80367c1 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
