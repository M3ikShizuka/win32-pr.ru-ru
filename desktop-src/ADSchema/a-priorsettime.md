# <a name="prior-set-time-attribute"></a>Атрибут до времени установки

Предыдущее время, заданное для секрета.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Ранее установленное время                       |
| LDAP-отображаемое имя | приорсеттиме                         |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.99                |
| System-ID — GUID    | bf967a01-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
