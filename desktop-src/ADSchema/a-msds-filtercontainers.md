# <a name="ms-ds-filter-containers-attribute"></a>Атрибут ms-DS-Filter-Containers

Строка с несколькими значениями, содержащая имена классов, используемых для определения типов контейнеров, которые должны отображаться при фильтрации с помощью оснастки "пользователи и компьютеры" Active Directory.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-Filter-Containers                     |
| LDAP-отображаемое имя | msDS-Филтерконтаинерс                       |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | При изменении списка типов контейнеров.   |
| Attribute-Id      | 1.2.840.113556.1.4.1703                     |
| System-ID — GUID    | fb00dcdf-ac37-483a-9c12-ac53a6603033        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
