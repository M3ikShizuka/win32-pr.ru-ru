# <a name="domain-policy-reference-attribute"></a>Атрибут "домен-политика-ссылка"

Различающееся имя объекта политики домена, из которого копируется объект политики.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Справочник по политикам домена                 |
| LDAP-отображаемое имя | домаинполициреференце                   |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.422                  |
| System-ID — GUID    | 80a67e2a-9f22-11d0-afdd-00c04fd930c9    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
