# <a name="ms-radius-framedipv6route-attribute"></a>атрибут MS-RADIUS-FramedIpv6Route

Предоставляет сведения о маршрутизации, которые должны быть настроены для пользователя на NAS.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-RADIUS-FramedIpv6Route            |
| LDAP-отображаемое имя | Мсрадиус — FramedIpv6Route             |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1917              |
| System-ID — GUID    | 5a5aa804-3083-4863-94e5-018a79a22ec0 |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
