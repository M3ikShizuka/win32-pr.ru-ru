# <a name="ms-wmi-mof-attribute"></a>атрибут MS-WMI-MOF

Содержит определение MOF для некоторых объектов WMI.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-MOF                                  |
| LDAP-отображаемое имя | Мсвми-MOF                                   |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор групповая политика                  |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1638                     |
| System-ID — GUID    | 6736809f-2064-443e-a145-81262b1f1366        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
