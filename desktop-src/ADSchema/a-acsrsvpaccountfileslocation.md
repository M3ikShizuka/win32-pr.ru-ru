# <a name="acs-rsvp-account-files-location-attribute"></a>Атрибут расположения ACS-RSVP-Account-Files-Location

Расположение файлов учетной записи RSVP в каталоге. По умолчанию используется подкаталог System32, расположенный в системном каталоге по умолчанию.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | ACS-RSVP-Account-Files-Location             |
| LDAP-отображаемое имя | аксрсвпаккаунтфилеслокатион                 |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.900                      |
| System-ID — GUID    | f072230f-aef5-11d1-bdcf-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
