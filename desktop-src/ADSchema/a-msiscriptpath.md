# <a name="msi-script-path-attribute"></a>Атрибут "MSI-Script-Path"

Путь к файлу скрипта установщика Microsoft для этого приложения.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MSI-Script-путь                             |
| LDAP-отображаемое имя | мсискриптпас                               |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.15                       |
| System-ID — GUID    | bf967937-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
