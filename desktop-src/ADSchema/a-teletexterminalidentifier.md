# <a name="teletex-terminal-identifier-attribute"></a>Teletex-атрибут идентификатора терминала

Указывает идентификатор терминала Teletex и (необязательно) параметры для терминала Teletex, связанного с объектом.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Teletex — идентификатор терминала                           |
| LDAP-отображаемое имя | телетекстерминалидентифиер                             |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 2.5.4.22                                              |
| System-ID — GUID    | bf967a4a-0de6-11d0-a285-00aa003049e2                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
