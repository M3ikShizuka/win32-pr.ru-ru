# <a name="pki-overlap-period-attribute"></a>PKI-перекрытие-атрибут периода

Период времени, в течение которого сертификат должен быть продлен до истечения срока его действия.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | PKI — перекрытие — точка                                    |
| LDAP-отображаемое имя | пкиоверлаппериод                                      |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1332                               |
| System-ID — GUID    | 1219a3ec-3b9e-11d2-90cc-00c04fd91ab1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
