# <a name="msmq-ds-service-attribute"></a>Атрибут MSMQ-DS-Service

Указывает, предоставляет ли этот сервер службы DS.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MSMQ-DS-Service                      |
| LDAP-отображаемое имя | мсмкдссервице                        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1238              |
| System-ID — GUID    | 2df90d82-009f-11d2-aa4c-00c04fd7d83a |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
