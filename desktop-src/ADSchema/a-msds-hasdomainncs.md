# <a name="ms-ds-has-domain-ncs-attribute"></a>Атрибут ms-DS-with-Domain-NC

Сведения о репликации службы каталогов, в которых содержатся сведения о контекстах именования доменов, имеющихся на определенном сервере.



| Ввод | Значение |
|-------------------|-----------------------------------------------------|
| CN                | MS-DS-имеет-Domain-NC                                |
| LDAP-отображаемое имя | msDS-Хасдомаиннкс                                   |
| Размер              | \-                                                  |
| Привилегия обновления  | Это значение задается системой.                     |
| Частота обновления  | Он задается DCPromo и не изменяется в дальнейшем. |
| Attribute-Id      | 1.2.840.113556.1.4.1820                             |
| System-ID — GUID    | 6f17e347-a842-4498-b8b3-15e007da4fed                |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md)             |
