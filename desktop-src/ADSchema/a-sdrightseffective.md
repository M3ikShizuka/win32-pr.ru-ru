# <a name="sd-rights-effective-attribute"></a>SD-Rights-действующий атрибут

Этот сконструированный атрибут возвращает одиночное значение **DWORD** , которое может иметь до трех наборов битов:

-   \_ \_ сведения о безопасности владельца
-   \_ \_ сведения о безопасности DACL
-   \_ \_ сведения о безопасности SACL

Если задан бит, то пользователь имеет доступ на запись к соответствующей части дескриптора безопасности. Владелец означает как владельца, так и группу.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | SD-Rights-эффективен                  |
| LDAP-отображаемое имя | сдригхтсеффективе                    |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1304              |
| System-ID — GUID    | c3dbafa6-33df-11d2-98b2-0000f87a57d4 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
