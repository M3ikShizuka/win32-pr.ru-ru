# <a name="msmq-receive-journal-extended-right"></a>MSMQ-Receive-журнал — расширенное право

Позволяет получать сообщения из журнала очереди.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | MSMQ-получение-журнал                 |
| Display-Name | Получение журнала                      |
| Rights-GUID  | 06bd3203-df3e-11d1-9c86-006008764d0e |
