# <a name="ms-ds-entry-time-to-die-attribute"></a>атрибут "время до кристалла" MS-DS-Entry

Содержит абсолютный срок действия динамического объекта в каталоге.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------|
| CN                | MS-DS-ввод-время до кристалла                                       |
| LDAP-отображаемое имя | msDS-запись-время до кристалла                                        |
| Размер              | \-                                                            |
| Привилегия обновления  | Это значение задается системой.                              |
| Частота обновления  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.1622                                       |
| System-ID — GUID    | e1e9bad7-c6dd-4101-a843-794cec85b038                          |
| Синтаксис            | [**String(обобщенное_время)**](s-string-generalized-time.md) |
