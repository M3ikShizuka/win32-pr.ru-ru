# <a name="acs-aggregate-token-rate-per-user-attribute"></a>Атрибут "ACS" — "Статистическая обработка"-частота "на пользователя"

Максимальная частота маркеров, которую любой пользователь может иметь для всех потоков.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS-агрегирование-частота маркеров на пользователя    |
| LDAP-отображаемое имя | аксаггрегатетокенратеперусер         |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.760               |
| System-ID — GUID    | 7f56127d-5301-11d1-a9c5-0000f80367c1 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
