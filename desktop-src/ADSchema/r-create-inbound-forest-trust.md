# <a name="create-inbound-forest-trust-extended-right"></a>Create-Inbound-лесами-Trust расширенное право

Право на расширенное управление доступом, которое позволяет пользователям создавать только входящие отношения доверия между лесами, добавляя их в соответствующую группу.



| Ввод | Значение |
|--------------|--------------------------------------|
| CN           | Create-Inbound-лесами-Trust          |
| Display-Name | Создание входящего доверия лесов          |
| Rights-GUID  | e2a36dc9-ae17-47c3-b58b-be34c55ba633 |
