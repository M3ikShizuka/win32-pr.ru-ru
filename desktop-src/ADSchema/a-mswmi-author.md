# <a name="ms-wmi-author-attribute"></a>атрибут MS-WMI-Author

Автор экземпляра класса.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-Author                               |
| LDAP-отображаемое имя | Мсвми — автор                                |
| Размер              | Меньше 25 символов.                    |
| Привилегия обновления  | Администратор групповая политика                  |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1623                     |
| System-ID — GUID    | 6366c0c1-6972-4e66-b3a5-1d52ad0c0547        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
