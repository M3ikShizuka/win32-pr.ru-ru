# <a name="ipsec-data-attribute"></a>Атрибут Ipsec-Data

Атрибут **IPSec-Data** предназначен только для внутреннего использования.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Ipsec-Data                                            |
| LDAP-отображаемое имя | ипсекдата                                             |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.623                                |
| System-ID — GUID    | b40ff81f-427a-11d1-a9c2-0000f80367c1                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
