# <a name="ms-ds-az-task-is-role-definition-attribute"></a>Атрибут ms-DS-AZ-Task-=-Role-definition

Логическое поле, указывающее, является ли Азтаск классической задачей или определением роли.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-AZ-Task-=-Role-definition        |
| LDAP-отображаемое имя | msDS-Азтаскисроледефинитион             |
| Размер              | \-                                      |
| Привилегия обновления  | Администратор Азролес                           |
| Частота обновления  | Во время инициализации или изменения политики. |
| Attribute-Id      | 1.2.840.113556.1.4.1818                 |
| System-ID — GUID    | 7b078544-6c82-4fe9-872f-ff48ad2b2e26    |
| Синтаксис            | [**Логическая**](s-boolean.md)            |
