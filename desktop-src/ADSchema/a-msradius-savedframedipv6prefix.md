# <a name="ms-radius-savedframedipv6prefix-attribute"></a>атрибут MS-RADIUS-SavedFramedIpv6Prefix

Указывает префикс IPv6 (и соответствующий маршрут), который должен быть настроен для пользователя.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-RADIUS-SavedFramedIpv6Prefix      |
| LDAP-отображаемое имя | Мсрадиус — SavedFramedIpv6Prefix       |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1916              |
| System-ID — GUID    | 0965a062-b1e1-403b-b48d-5c0eb0e952cc |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
