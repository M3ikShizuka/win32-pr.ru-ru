# <a name="ms-wmi-targetpath-attribute"></a>MS-WMI-TargetPath, атрибут

Список пар "ключ-значение" для уникальной идентификации объекта WMI.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-TargetPath                           |
| LDAP-отображаемое имя | Мсвми-TargetPath                            |
| Размер              | Менее 100 символов.                   |
| Привилегия обновления  | Администратор групповая политика                  |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1648                     |
| System-ID — GUID    | 5006a79a-6bfe-4561-9f52-13cf4dd3e560        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
