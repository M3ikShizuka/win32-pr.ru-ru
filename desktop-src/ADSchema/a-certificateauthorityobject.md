# <a name="certificate-authority-object-attribute"></a>Certificate-Authority-атрибут объекта

Ссылка на центр сертификации, связанный с точкой распространения списка отзыва сертификатов.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Certificate-Authority-объект            |
| LDAP-отображаемое имя | цертификатеаусоритйобжект              |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.684                  |
| System-ID — GUID    | 963d2732-48be-11d1-a9c3-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
