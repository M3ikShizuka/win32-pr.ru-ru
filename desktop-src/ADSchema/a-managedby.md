# <a name="managed-by-attribute"></a>Атрибут Managed-By

Различающееся имя пользователя, назначенного для управления этим объектом.



| Ввод | Значение |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | Managed-By                                                                          |
| LDAP-отображаемое имя | managedBy                                                                           |
| Размер              | \-                                                                                  |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                              |
| Частота обновления  | При создании записи пользователя и при необходимости изменения управляемых объектов. |
| Attribute-Id      | 1.2.840.113556.1.4.653                                                              |
| System-ID — GUID    | 0296c120-40da-11d1-a9c0-0000f80367c1                                                |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md)                                             |
