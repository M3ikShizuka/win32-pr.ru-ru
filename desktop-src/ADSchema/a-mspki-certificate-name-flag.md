# <a name="ms-pki-certificate-name-flag-attribute"></a>атрибут MS-PKI-Certificate-Name-Flag

Содержит флаги, связанные с созданием имени субъекта в выданном сертификате.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-Certificate-Name — флаг                                                                      |
| LDAP-отображаемое имя | Мспки-Certificate-Name — флаг                                                                       |
| Размер              | 4 байта                                                                                           |
| Привилегия обновления  | Администратор домена                                                                              |
| Частота обновления  | При изменении, создании или клонировании объекта шаблона сертификата (MS-PKI-Certificate-Template). |
| Attribute-Id      | 1.2.840.113556.1.4.1432                                                                           |
| System-ID — GUID    | ea1dddc4-60ff-416e-8cc0-17cee534bce7                                                              |
| Синтаксис            | [**Перечисление**](s-enumeration.md)                                                              |
