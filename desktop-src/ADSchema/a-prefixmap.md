|-------------------|-------------------------------------------------------|
| CN                | Prefix-Map                                            |
| LDAP-отображаемое имя | префиксмап                                             |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.538                                |
| System-ID — GUID    | 52458022-ca6a-11D0-аффф-0000f80367c1                  |
