# <a name="short-server-name-attribute"></a>Атрибут короткого имени сервера

Пред-Windows 2000 имя сервера, совместимое с серверами печати.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Короткое имя сервера                           |
| LDAP-отображаемое имя | shortServerName                             |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1209                     |
| System-ID — GUID    | 45b01501-c419-11d1-bbc9-0080c76670c0        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
