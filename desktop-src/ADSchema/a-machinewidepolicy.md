# <a name="machine-wide-policy-attribute"></a>Атрибут политики на уровне компьютера

Используется для репликации расширяемой политики пользователя на клиентах.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Политика на уровне компьютера                                   |
| LDAP-отображаемое имя | мачиневидеполици                                     |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.459                                |
| System-ID — GUID    | 80a67e4f-9f22-11d0-afdd-00c04fd930c9                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
