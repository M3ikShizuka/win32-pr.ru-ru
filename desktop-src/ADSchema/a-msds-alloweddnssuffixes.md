# <a name="ms-ds-allowed-dns-suffixes-attribute"></a>Атрибут ms-DS-Allowed-DNS-суффиксов

Список разрешенных суффиксов для атрибута dNSHostName в объектах Computer.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-DS-Allowed-DNS-суффиксы                  |
| LDAP-отображаемое имя | msDS-AllowedDNSSuffixes                     |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1710                     |
| System-ID — GUID    | 8469441b-9ac4-4e45-8205-bd219dbf672d        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
