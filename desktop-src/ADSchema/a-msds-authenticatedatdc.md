# <a name="ms-ds-authenticatedat-dc-attribute"></a>Атрибут ms-DS-Аусентикатедат-DC

Прямая ссылка на [**MS-DS-аусентикатедто-аккаунтлист**](a-msds-authenticatedtoaccountlist.md). Указывает контроллер домена, на который пользователь прошел проверку подлинности.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-Аусентикатедат-DC                |
| LDAP-отображаемое имя | msDS-Аусентикатедатдк                  |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1958                 |
| System-ID — GUID    | 3e1ee99c-6604-4489-89d9-84798a89515a    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
