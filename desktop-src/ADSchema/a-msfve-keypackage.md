# <a name="ms-fve-keypackage-attribute"></a>атрибут MS-ФВЕ-Кэйпаккаже

Содержит ключ шифрования BitLocker тома, защищенный соответствующим паролем восстановления.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | MS-ФВЕ-Кэйпаккаже                                     |
| LDAP-отображаемое имя | Мсфве — Кэйпаккаже                                      |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1999                               |
| System-ID — GUID    | 1fd55ea8-88a7-47dc-8129-0daa97186a54                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
