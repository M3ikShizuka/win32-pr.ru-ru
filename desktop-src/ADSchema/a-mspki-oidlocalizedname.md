# <a name="ms-pki-oid-localizedname-attribute"></a>атрибут MS-PKI-OID-LocalizedName

Содержит список отображаемых имен, используемых для описания OID по языку и региональным параметрам.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-PKI-OID-LocalizedName                    |
| LDAP-отображаемое имя | Мспки — Оидлокализеднаме                      |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1712                     |
| System-ID — GUID    | 7d59a816-bb05-4a72-971f-5c1331f67559        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
