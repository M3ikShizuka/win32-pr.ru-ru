# <a name="oem-information-attribute"></a>Атрибут OEM-Information

Для хранения информации OEM. Больше не используется. Здесь для обеспечения обратной совместимости.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | OEM-Information                             |
| LDAP-отображаемое имя | оеминформатион                              |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.151                      |
| System-ID — GUID    | bf9679ea-0de6-11d0-a285-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
