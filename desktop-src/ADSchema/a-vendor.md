# <a name="vendor-attribute"></a>Атрибут Vendor

Этот атрибут идентифицирует поставщика для приложения.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Vendor                                      |
| LDAP-отображаемое имя | vendor                                      |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.255                      |
| System-ID — GUID    | 281416df-1968-11d0-a28f-00aa003049e2        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
