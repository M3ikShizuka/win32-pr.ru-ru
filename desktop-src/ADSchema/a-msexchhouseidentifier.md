|-------------------|---------------------------------------------|
| CN                | MS-дов-House-identifier                    |
| LDAP-отображаемое имя | мсексчхаусеидентифиер                       |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.596                      |
| System-ID — GUID    | a8df7407-c5ea-11d1-bbcb-0080c76670c0        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
