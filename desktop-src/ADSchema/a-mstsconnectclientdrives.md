# <a name="ms-ts-connect-client-drives-attribute"></a>атрибут MS-TS-Connect-Client-Drives

| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-TS-Connect-Client-Drives          |
| LDAP-отображаемое имя | мстсконнектклиентдривес              |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1986              |
| System-ID — GUID    | 23572aaf-29dd-44ea-b0fa-7e8438b9a4a3 |
| Синтаксис            | [**Логическая**](s-boolean.md)         |
