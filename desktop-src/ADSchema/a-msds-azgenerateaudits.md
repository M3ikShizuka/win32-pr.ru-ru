# <a name="ms-ds-az-generate-audits-attribute"></a>Атрибут ms-DS-AZ-Generate-audits

Логическое поле, указывающее, нужно ли включать аудит времени выполнения (включить аудит для проверок доступа и т. д.).



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | MS-DS-AZ-Generate-аудиты                |
| LDAP-отображаемое имя | msDS-Азженератеаудитс                   |
| Размер              | \-                                      |
| Привилегия обновления  | Администратор Азролес                           |
| Частота обновления  | Во время инициализации или изменения политики. |
| Attribute-Id      | 1.2.840.113556.1.4.1805                 |
| System-ID — GUID    | f90abab0-186c-4418-bb85-88447c87222a    |
| Синтаксис            | [**Логическая**](s-boolean.md)            |
