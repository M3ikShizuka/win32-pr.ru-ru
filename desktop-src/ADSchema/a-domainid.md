# <a name="domain-id-attribute"></a>Атрибут domain-ID

Ссылка на домен, связанный с центром сертификации.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | Идентификатор домена                               |
| LDAP-отображаемое имя | домаинид                                |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.686                  |
| System-ID — GUID    | 963d2734-48be-11d1-a9c3-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
