# <a name="service-dns-name-attribute"></a>Service-DNS-атрибут Name

DNS-имя для поиска сервера, на котором запущена эта служба.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Service-DNS-имя                            |
| LDAP-отображаемое имя | serviceDNSName                              |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.657                      |
| System-ID — GUID    | 28630eb8-41d5-11d1-a9c1-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
