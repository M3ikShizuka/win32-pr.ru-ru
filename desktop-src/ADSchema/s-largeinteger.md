# <a name="largeinteger-syntax"></a>Синтаксис Ларжеинтежер

Представляет значение 64-битового целого числа со знаком.



| Ввод | Значение |
|--------------|------------------------------------------------------------------------------------|
| Имя         | LargeInteger                                                                       |
| Идентификатор синтаксиса    | 2.5.5.16                                                                           |
| ИДЕНТИФИКАТОР OM        | 65                                                                                 |
| Тип MAPI    | \-                                                                                 |
| Тип ADS     | АДСТИПЕ \_ большое \_ целое                                                            |
| Тип варианта | \_Диспетчер VT                                                                       |
| Тип SDS     | COM-объект, который можно привести к [**иадсларжеинтежер**](/windows/desktop/api/iads/nn-iads-iadslargeinteger). |
