# <a name="acs-allocable-rsvp-bandwidth-attribute"></a>ACS-с атрибутом пропускной способности и RSVP

Максимальная пропускная способность, которую можно зарезервировать.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | ACS — с поддержкой-RSVP — пропускная способность         |
| LDAP-отображаемое имя | аксаллокаблерсвпбандвидс            |
| Размер              | 8 байт                              |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.766               |
| System-ID — GUID    | 7f561283-5301-11d1-a9c5-0000f80367c1 |
| Синтаксис            | [**Интервал**](s-interval.md)       |
