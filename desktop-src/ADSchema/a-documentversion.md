# <a name="documentversion-attribute"></a>атрибут Документверсион

Тип атрибута Документверсион указывает номер версии документа.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | документверсион                             |
| LDAP-отображаемое имя | документверсион                             |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.13                  |
| System-ID — GUID    | 94b3a8a9-d613-4cec-9aad-5fbcc1046b43        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
