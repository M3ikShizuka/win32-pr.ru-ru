# <a name="ipsec-name-attribute"></a>Атрибут Ipsec-Name

Атрибут **IPSec-Name** предназначен только для внутреннего использования.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Ipsec-Name                                  |
| LDAP-отображаемое имя | ипсекнаме                                   |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.620                      |
| System-ID — GUID    | b40ff81c-427a-11d1-a9c2-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
