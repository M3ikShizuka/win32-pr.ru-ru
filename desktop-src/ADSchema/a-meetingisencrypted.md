# <a name="meetingisencrypted-attribute"></a>атрибут Митингисенкриптед

Это **справедливо** , если собрание должно быть зашифровано.



| Ввод | Значение |
|-------------------|----------------------------------------------------------------------------------|
| CN                | митингисенкриптед                                                               |
| LDAP-отображаемое имя | митингисенкриптед                                                               |
| Размер              | \-                                                                               |
| Привилегия обновления  | Любой пользователь может обновить этот объект на основе безопасности создаваемого объекта. |
| Частота обновления  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.585                                                           |
| System-ID — GUID    | 11b6cc8e-48c4-11d1-a9c3-0000f80367c1                                             |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                      |
