| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | нетбут-IntelliMirror-ОС                  |
| LDAP-отображаемое имя | нетбутинтеллимирроросес                    |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.857                      |
| System-ID — GUID    | 0738307e-91df-11d1-aebc-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
