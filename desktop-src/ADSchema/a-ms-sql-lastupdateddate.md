# <a name="ms-sql-lastupdateddate-attribute"></a>атрибут MS-SQL-ластупдатеддате

это значение представляет время, которое SQL последнего обновления Active Directory.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-ластупдатеддате                      |
| LDAP-отображаемое имя | mS-SQL-ластупдатеддате                      |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | При запуске системы или при обновлении AD.    |
| Attribute-Id      | 1.2.840.113556.1.4.1381                     |
| System-ID — GUID    | 9fcc43d4-ccee-11d2-9993-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
