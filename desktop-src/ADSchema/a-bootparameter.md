# <a name="bootparameter-attribute"></a>атрибут Бутпараметер

Предоставляет данные, необходимые для запуска бесдискового клиента.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | бутпараметер                        |
| LDAP-отображаемое имя | бутпараметер                        |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.23                     |
| System-ID — GUID    | d72a0750-8c7c-416e-8714-e65f11e908be |
| Синтаксис            | [**String(IA5)**](s-string-ia5.md)  |
