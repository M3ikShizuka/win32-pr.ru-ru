# <a name="ms-sql-name-attribute"></a>Атрибут MS-SQL-Name

Имя экземпляра SQL Server. Значение по умолчанию — MSSQLSERVER.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Имя MS-SQL                                 |
| LDAP-отображаемое имя | Имя mS-SQL                                 |
| Размер              | \-                                          |
| Привилегия обновления  | Это значение задается системой.            |
| Частота обновления  | При установке системы.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1363                     |
| System-ID — GUID    | 3532dfd8-ccee-11d2-9993-0000f87a57d4        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
