# <a name="msmq-queue-type-attribute"></a>MSMQ-Queue-Type, атрибут

Идентификатор GUID, описывающий тип службы, предоставляемой очередью.



| Ввод | Значение |
|-------------------|-------------------------------------------------------|
| CN                | Очередь MSMQ — тип                                       |
| LDAP-отображаемое имя | мсмккуеуетипе                                         |
| Размер              | \-                                                    |
| Привилегия обновления  | \-                                                    |
| Частота обновления  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.917                                |
| System-ID — GUID    | 9a0dc320-c100-11d1-bbc5-0080c76670c0                  |
| Синтаксис            | [**Object(ссылка_на_реплику)**](s-object-replica-link.md) |
