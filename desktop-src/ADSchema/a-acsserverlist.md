# <a name="acs-server-list-attribute"></a>ACS-сервер-атрибут списка

Содержит DNS-имена серверов Windows NT, которым разрешено выполнять ACS.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | ACS-Server-список                             |
| LDAP-отображаемое имя | акссерверлист                               |
| Размер              | \-                                          |
| Привилегия обновления  | Администратор домена                        |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1312                     |
| System-ID — GUID    | 7cbd59a5-3b90-11d2-90cc-00c04fd91ab1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
