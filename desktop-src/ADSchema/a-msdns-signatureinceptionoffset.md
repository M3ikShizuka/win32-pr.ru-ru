# <a name="ms-dns-signature-inception-offset-attribute"></a>атрибут MS-DNS-Signature-InAttribute-offset

Атрибут, который определяет время в секундах, которое должно начинаться в течение прошлых периодов действия подписи DNSSEC при подписывании зоны DNS.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | MS-DNS-"начало подписи"-смещение    |
| LDAP-отображаемое имя | msDN — Сигнатуреинцептионоффсет       |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2141              |
| System-ID — GUID    | 03d4c32e-e217-4a61-9699-7bbc4729a026 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
