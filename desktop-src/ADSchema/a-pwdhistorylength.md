# <a name="pwd-history-length-attribute"></a>PWD-атрибут длины журнала

Число старых паролей, которые необходимо сохранить.



| Ввод | Значение |
|-------------------|------------------------------------------|
| CN                | PWD-History-длина                       |
| LDAP-отображаемое имя | пвдхисториленгс                         |
| Размер              | 4 байта                                  |
| Привилегия обновления  | Администратор домена                     |
| Частота обновления  | При необходимости изменения политики учетных записей. |
| Attribute-Id      | 1.2.840.113556.1.4.95                    |
| System-ID — GUID    | bf967a09-0de6-11d0-a285-00aa003049e2     |
| Синтаксис            | [**Перечисление**](s-enumeration.md)     |
