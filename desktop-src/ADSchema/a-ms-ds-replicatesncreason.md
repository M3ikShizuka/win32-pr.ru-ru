# <a name="ms-ds-replicates-nc-reason-attribute"></a>Атрибут MS-DS-replicate-NC-Reason

Атрибут объекта Нтдсконнектион, указывающий, как (или в этом случае) показывается, что соединение будет использоваться в топологии репликации. Параметр имеет несколько значений и имеет синтаксис Дистнаме + binary, где двоичная часть является битовым значением целого размера.



| Ввод | Значение |
|-------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | MS-DS-replicate-NC-причина                                                                                                                 |
| LDAP-отображаемое имя | mS-DS-Репликатеснкреасон                                                                                                                   |
| Размер              | Значение для двоичной части: 0 = нет \_ причины, 1 = \_ топология сборки мусора, 2 = Кольцевая \_ топология, 4 = сокращение \_ \_ ТОПОЛОГИИ прыжков, 8 = устаревшая \_ \_ топология серверов. |
| Привилегия обновления  | Это значение задается системой.                                                                                                           |
| Частота обновления  | Может измениться в ответ на изменения в топологии сети.                                                                                 |
| Attribute-Id      | 1.2.840.113556.1.4.1408                                                                                                                    |
| System-ID — GUID    | 0ea12b84-08b3-11d3-91bc-0000f87a57d4                                                                                                       |
| Синтаксис            | [**Object(двоичный_DN)**](s-object-dn-binary.md)                                                                                            |
