# <a name="phone-office-other-attribute"></a>Телефон-Office-другой атрибут

Список альтернативных номеров офисных телефонов.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------------------------|
| CN                | Телефон-Office-Other                                                              |
| LDAP-отображаемое имя | otherTelephone                                                                  |
| Размер              | \-                                                                              |
| Привилегия обновления  | Администратор домена или владелец учетной записи.                                          |
| Частота обновления  | При создании записи пользователя и при необходимости изменения номера телефона. |
| Attribute-Id      | 1.2.840.113556.1.2.18                                                           |
| System-ID — GUID    | f0f8ffa5-1191-11d0-a060-00aa006c33ed                                            |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md)                                     |
