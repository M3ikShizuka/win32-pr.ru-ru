# <a name="meetingprotocol-attribute"></a>атрибут Митингпротокол

Протоколы, используемые для собрания, например H. 320 или T. 120.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | митингпротокол                             |
| LDAP-отображаемое имя | митингпротокол                             |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.570                      |
| System-ID — GUID    | 11b6cc81-48c4-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
