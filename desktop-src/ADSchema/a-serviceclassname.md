# <a name="service-class-name-attribute"></a>Атрибут Service-Class-Name

Строковое имя службы, которую представляет точка администрирования.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Имя-класса службы                          |
| LDAP-отображаемое имя | serviceClassName                            |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.509                      |
| System-ID — GUID    | b7b1311d-b82e-11d0-afee-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
