| LDAP-отображаемое имя | классдисплайнаме                            |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.610                      |
| System-ID — GUID    | 548e1c22-dea6-11d0-b010-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
