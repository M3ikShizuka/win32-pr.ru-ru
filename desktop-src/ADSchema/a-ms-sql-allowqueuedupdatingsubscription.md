# <a name="ms-sql-allowqueuedupdatingsubscription-attribute"></a>атрибут MS-SQL-алловкуеуедупдатингсубскриптион

Значение true, чтобы разрешить транзакции в очереди при обновлении подписок.



| Ввод | Значение |
|-------------------|----------------------------------------|
| CN                | MS-SQL-алловкуеуедупдатингсубскриптион |
| LDAP-отображаемое имя | mS-SQL-алловкуеуедупдатингсубскриптион |
| Размер              | 4 байта                                |
| Привилегия обновления  | Это значение задается системой.       |
| Частота обновления  | При установке репликации.             |
| Attribute-Id      | 1.2.840.113556.1.4.1405                |
| System-ID — GUID    | c458ca80-d34b-11d2-999a-0000f87a57d4   |
| Синтаксис            | [**Логическое**](s-boolean.md)           |
