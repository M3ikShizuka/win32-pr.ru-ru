# <a name="modify-time-stamp-attribute"></a>Атрибут-метка времени изменения

Вычисленный атрибут, представляющий дату последнего изменения этого объекта. Это значение не реплицируется.



| Ввод | Значение |
|-------------------|---------------------------------------------------------------|
| CN                | Метка времени изменения                                             |
| LDAP-отображаемое имя | modifyTimeStamp                                               |
| Размер              | 8 байт                                                       |
| Привилегия обновления  | Это значение задается системой.                              |
| Частота обновления  | \-                                                            |
| Attribute-Id      | 2.5.18.2                                                      |
| System-ID — GUID    | 9a7ad94a-ca53-11d1-bbd0-0080c76670c0                          |
| Синтаксис            | [**String(обобщенное_время)**](s-string-generalized-time.md) |
