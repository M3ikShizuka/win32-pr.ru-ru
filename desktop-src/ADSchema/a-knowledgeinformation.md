# <a name="knowledge-information-attribute"></a>Атрибут Knowledge-Information

Указывает общее для пользователя накопленное описание набора знаний, созданного конкретным агентом системы каталогов.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | Knowledge-Information                       |
| LDAP-отображаемое имя | кновледжеинформатион                        |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 2.5.4.2                                     |
| System-ID — GUID    | 1677581f-47f3-11d1-a9c3-0000f80367c1        |
| Синтаксис            | [**String(Teletex)**](s-string-teletex.md) |
