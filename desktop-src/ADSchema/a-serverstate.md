# <a name="server-state-attribute"></a>Атрибут Server-State

Указывает, включен ли сервер или отключен. Значение 1 указывает на то, что сервер включен. Значение 2 указывает на то, что сервер отключен. Все остальные значения недопустимы.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | Server-State                         |
| LDAP-отображаемое имя | serverState                          |
| Размер              | \-                                   |
| Привилегия обновления  | Это значение задается системой.     |
| Частота обновления  | При изменении политики для пользователя.  |
| Attribute-Id      | 1.2.840.113556.1.4.154               |
| System-ID — GUID    | bf967a34-0de6-11d0-a285-00aa003049e2 |
| Синтаксис            | [**Перечисление**](s-enumeration.md) |
