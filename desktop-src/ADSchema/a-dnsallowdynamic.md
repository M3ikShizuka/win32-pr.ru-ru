# <a name="dns-allow-dynamic-attribute"></a>DNS-Allow — динамический атрибут

В настоящее время атрибут **DNS-Allow-Dynamic** не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | DNS — разрешить — динамический                    |
| LDAP-отображаемое имя | днсалловдинамик                      |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.378               |
| System-ID — GUID    | e0fa1e65-9b45-11d0-afdd-00c04fd930c9 |
| Синтаксис            | [**Логическое**](s-boolean.md)         |
