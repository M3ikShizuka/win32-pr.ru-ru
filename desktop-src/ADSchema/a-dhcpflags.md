# <a name="dhcp-flags-attribute"></a>атрибут DHCP-flags

Атрибут **DHCP-flags** в настоящее время не используется.



| Ввод | Значение |
|-------------------|--------------------------------------|
| CN                | DHCP-флаги                           |
| LDAP-отображаемое имя | дхкпфлагс                            |
| Размер              | \-                                   |
| Привилегия обновления  | \-                                   |
| Частота обновления  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.700               |
| System-ID — GUID    | 963d2741-48be-11d1-a9c3-0000f80367c1 |
| Синтаксис            | [**Пределах**](s-interval.md)       |
