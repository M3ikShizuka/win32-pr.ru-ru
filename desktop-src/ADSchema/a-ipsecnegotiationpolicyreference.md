# <a name="ipsec-negotiation-policy-reference-attribute"></a>Атрибут "согласование IPSec-политика-ссылка"

Атрибут **IPSec-согласовать-Policy-Reference** предназначен только для внутреннего использования.



| Ввод | Значение |
|-------------------|-----------------------------------------|
| CN                | IPSec-согласование — политика-Справочник      |
| LDAP-отображаемое имя | ипсекнеготиатионполициреференце         |
| Размер              | \-                                      |
| Привилегия обновления  | \-                                      |
| Частота обновления  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.628                  |
| System-ID — GUID    | b40ff822-427a-11d1-a9c2-0000f80367c1    |
| Синтаксис            | [**Object(DS-DN)**](s-object-ds-dn.md) |
