# <a name="msmq-site-name-ex-attribute"></a>Атрибут MSMQ-Site-Name-ex

Атрибут **MSMQ-Site-Name-ex** содержит сведения о смешанном режиме MSMQ.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | MSMQ-Site-Name-ex                           |
| LDAP-отображаемое имя | мсмкситенамикс                              |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1416                     |
| System-ID — GUID    | 422144fa-c17f-4649-94d6-9731ed2784ed        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
