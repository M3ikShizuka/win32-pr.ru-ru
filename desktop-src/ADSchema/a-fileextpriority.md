# <a name="file-ext-priority-attribute"></a>Атрибут File-ext-Priority

Этот атрибут содержит список расширений имен файлов в пакете приложения и связанные с ними приоритеты.



| Ввод | Значение |
|-------------------|---------------------------------------------|
| CN                | File-ext-Priority                           |
| LDAP-отображаемое имя | филикстприорити                             |
| Размер              | \-                                          |
| Привилегия обновления  | \-                                          |
| Частота обновления  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.816                      |
| System-ID — GUID    | d9e18315-8939-11d1-aebc-0000f80367c1        |
| Синтаксис            | [**String(Юникод)**](s-string-unicode.md) |
