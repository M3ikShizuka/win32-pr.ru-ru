# <a name="p"></a>С

## <a name="primary-colors"></a>Основные цвета

Три самых ключевых цвета в цветовом пространстве, которые можно комбинировать для получения всех других цветов.

## <a name="profile-connection-space-pcs"></a>Место для подключения профиля (ПК)

Независимое от устройства цветовое пространство, используемое для преобразования цветов и сопоставления цветов.
