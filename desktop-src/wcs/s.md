# <a name="s"></a>S

## <a name="saturation-chroma"></a>насыщенность (чрома)

Обычно чрома — это объем серого, смешанный в виде оттенка. Ноль чрома создает оттенок серого. Full чрома создает чистый оттенок.

## <a name="shade"></a>текстуры

Заливка цвета выполняется путем добавления черного к его оттенку.

## <a name="soft-proofing"></a>Программная проверка

Предварительный просмотр или Проверка изображения на экране монитора.

## <a name="srgb"></a>sRGB

Стандартное цветовое пространство RGB, предлагаемое корпорацией Майкрософт и Hewlett-Packard, которое является независимым от устройства. Этот стандарт утвержден в качестве формального международного стандарта для мультимедийных систем и Екуипментколаур измерения и Манажементпарт 2-1: цветное Манажементдефаулт RGB-Цветное пространство. Он доступен непосредственно из IEC по адресу https://www.iec.ch/ .

## <a name="subtractive-primary-colors"></a>вычитаемые основные цвета

Цвета, которые можно вычесть из белого, чтобы смешать новый цвет. Основные цвета с вычитанием — голубой, желтый и пурпурный.
