# <a name="wcs-10-availability"></a>Доступность WCS 1.0

ICM1 доступен для Microsoft Windows 95. он недоступен для MS DOS, Windows 3. *x*, Windows NT 4,0 или более ранней версии.

ICM2 доступен для Microsoft Windows 98, Windows Millennium Edition (Me), Windows 2000 и Windows XP. он недоступен для MS DOS, Windows 3. *x*, Windows 95 или Windows NT 4,0 или более ранней версии.

WCS 1,0 доступен для Microsoft Windows Vista. он недоступен для Windows XP и более ранних версий.
