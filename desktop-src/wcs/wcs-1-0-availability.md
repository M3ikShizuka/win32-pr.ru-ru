# <a name="wcs-10-availability"></a>Доступность WCS 1.0

ICM1 доступен для Microsoft Windows 95. Она недоступна для MS DOS, Windows 3. *x*, Windows NT 4,0 или более ранней версии.

ICM2 доступен для Microsoft Windows 98, Windows Millennium Edition (Me), Windows 2000 и Windows XP. Она недоступна для MS DOS, Windows 3. *x*, Windows 95 или windows NT 4,0 или более ранней версии.

Для Microsoft Windows Vista доступен WCS 1,0. Она недоступна для Windows XP и более ранних версий.
