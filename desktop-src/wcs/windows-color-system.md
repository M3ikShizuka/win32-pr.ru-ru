# <a name="windows-color-system"></a>Цветовая система Windows

## <a name="purpose"></a>Назначение

Схемы управления цветом реализованы в операционных системах Microsoft Windows, чтобы улучшить отрисовку цветового содержимого во всех формах цифрового взаимодействия.

Схема управления цветом, которая используется начиная с Windows 2000, Windows XP и Windows Server 2003, называется управление цветом изображений (ICM) 2,0. Схема управления цветом, используемая начиная с Windows Vista, называется цветовой системой Windows (WCS) 1,0. Схема управления цветом WCS 1,0 представляет собой надмножество интерфейсов API и функций ICM 2,0.

## <a name="where-applicable"></a>Где применимо

ICM можно использовать во всех приложениях в операционных системах Windows 2000 и более поздних версий. WCS можно использовать во всех приложениях Windows Vista и более поздних операционных систем.

## <a name="developer-audience"></a>Аудитория разработчиков

API-интерфейс WCS предназначен для использования программистами C/C++. Вы знакомы с графическим пользовательским интерфейсом Windows, управляемой на основе сообщений архитектурой, и вам нужны основные понятия управления цветом.

## <a name="run-time-requirements"></a>Требования к среде выполнения

Для приложений, использующих ICM API, требуется Windows 2000 или более поздняя версия. Для приложений, использующих API-интерфейс WCS, требуется Windows Vista или более поздняя версия. Сведения о конкретных данных времени выполнения для функции см. в разделе "требования" на странице справки по этой функции.
