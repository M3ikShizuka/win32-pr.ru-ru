# <a name="windows-color-system"></a>Цветовая система Windows

## <a name="purpose"></a>Назначение

схемы управления цветом реализованы в операционных системах Microsoft Windows для улучшения отрисовки цветового содержимого во всех формах цифрового взаимодействия.

схема управления цветом, которая используется начиная с Windows 2000, Windows XP и Windows Server 2003, называется управление цветом изображений (ICM) 2,0. схема управления цветом, которая используется, начиная с Windows Vista, называется Windowsой цветовая система (WCS) 1,0. Схема управления цветом WCS 1,0 представляет собой надмножество интерфейсов API и функций ICM 2,0.

## <a name="where-applicable"></a>Где применимо

ICM можно использовать во всех приложениях на Windows 2000 и более поздних операционных системах. WCS можно использовать во всех приложениях на Windows Vista и более поздних операционных системах.

## <a name="developer-audience"></a>Аудитория разработчиков

API-интерфейс WCS предназначен для использования программистами C/C++. вы знакомы с Windows графическим пользовательским интерфейсом, управляемой на основе сообщений архитектурой, а также необходимыми знаниями о концепциях управления цветом.

## <a name="run-time-requirements"></a>Требования к среде выполнения

для приложений, использующих API ICM, требуется Windows 2000 или более поздней версии. для приложений, использующих API-интерфейс WCS, требуется Windows Vista или более поздней версии. Сведения о конкретных данных времени выполнения для функции см. в разделе "требования" на странице справки по этой функции.
