# <a name="using-color-management-on-the-internet"></a>Использование управления цветом в Интернете

Цветные изображения и графика — это все большее важное средство обмена информацией с пользователями Интернета. Сведения о преобразовании цвета могут быть внедрены в некоторые форматы файлов. Используемые сейчас методики внедрения не являются компактными. Вопросы пропускной способности Интернета часто делают их непрактичными. Вопросы Интернета включают следующие темы:

-   [sRGB: стандартное цветовое пространство](srgb--a-standard-color-space.md)
-   [WCS 1,0 значения по умолчанию для цветового пространства ввода и выходного профиля](srgb--a-standard-color-space.md)
-   [Профили sRGB и Embedded](srgb--a-standard-color-space.md)
