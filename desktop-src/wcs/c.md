# <a name="c"></a>C

## <a name="camp"></a>CAMP

Формат профиля модели цветового оформления — это формат файла на основе XML, который содержит условия просмотра, необходимые для описания связи между значениями ЦИЕКСИЗ DMP и значениями Циежаб.

## <a name="cmyk"></a>ИЗОБРАЖЕНИЯ

Голубое, Пурпурное, желтое и Черное цветовое пространство. Часто это реализуется на принтерах.

## <a name="colorant"></a>колорант

Нечто, особенно пигмент или краска, это цвета или изменяет цветовой тон чего-то другого.

## <a name="color-channel"></a>цветовой канал

Компонент цветового пространства. Например, цветовое пространство RGB имеет цветовые каналы красного, зеленого и синего цветов.

## <a name="color-conversion"></a>Преобразование цветов

Процесс преобразования цветов из одного цветового пространства в другое.

## <a name="color-management-module-cmm"></a>Модуль управления цветом (CMM)

Модуль кода, использующий профили устройств для выполнения преобразования цветов и сопоставления цветов.

## <a name="color-mapping"></a>Сопоставление цветов

См. раздел сопоставление цветов.

## <a name="color-matching"></a>Сопоставление цветов

Сопоставление преобразованного цвета с наиболее близким к нему цветом цветового пространства назначения.

## <a name="color-model"></a>Цветовая модель

См. раздел цветовое пространство.

## <a name="color-space"></a>цветовое пространство

Сопоставление цветовых компонентов на геометрической системе координат в трех измерениях.
