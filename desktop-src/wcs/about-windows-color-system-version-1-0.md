# <a name="about-windows-color-system-version-10"></a>Сведения о цветовой системе Windows версии 1.0

технология цветовой системы (WCS) Microsoft Windows гарантирует, что цвет изображения, графического или текстового объекта будет отображаться как можно ближе к исходному намерению на любом устройстве, несмотря на различия в технологиях создания образов и цветовых возможностях устройств. Независимо от того, сканируете изображение или какой-либо рисунок на цветном сканере, скачивает его через Интернет, просматриваете или редактируете на экране или выводите его на бумагу, пленку или другие мультимедийные файлы, WCS 1,0 позволяет постоянно и точно следить за тем, чтобы цвета соответствовали.

wcs 1,0 является неотъемлемой частью Microsoft Windows так как wcs 1,0 встроены в семейство операционных систем Windows в виде набора функций API Win32, оно легко доступно для любого приложения, драйвера устройства, средства калибровки устройства или модуля управления цветом (CMM).

версия 1,0 для управления цветом изображений (ICM) была предоставлена в Microsoft Windows 95 и предоставляет базовые возможности управления цветом в Windows контексте устройств.

ICM версии 2,0 входит в состав Windows 98, Windows Millennium Edition, Windows 2000 и Windows XP и включенных функций, в которых реализовано управление цветом за пределами контекстов устройств. Эти функции подходят для более требовательных требований к управлению цветом и предоставляют приложениям больший контроль над процессом управления цветом.

WCS версии 1,0 входит в состав Windows Vista и включает множество новых функций, обеспечивающих значительные улучшения в флексиблити, прозрачности, предсказуемости и расширяемости для поставщиков.

Какие типы приложений получат преимущество при использовании WCS 1,0? практически любое приложение, работающее в среде Windows Vista, уже сегодня. Базовая возможность управления цветом становится обязательным требованием для приложений всех типов.

Улучшение отображения и печати цвета в последние годы до момента, когда изображения с фотографией и сложная цветовая графика обычно используются не только при публикации на настольном компьютере, но и в широком спектре приложений для работы с данными и представлениями. такие технологии объектов, как COM и ActiveX, имеют внедренные объекты color практически во всех видах пространства данных.

Каталоги одежды, Интернет-картинки, фотоальбомы семьи, эмблемы для бизнеса, диаграммы, графики, презентации, предварительный просмотр и имитация цветов — всего лишь несколько примеров повседневных приложений, где важно иметь цвет.

В результате все больше и больше пользователей требуют, чтобы их приложения могли точно отображать цвет. Качественная визуализация с плохими цветами руинс изображения и графики, которые все чаще важны для пользователей.

На базовом уровне почти любое приложение должно иметь возможность настроить цвет автоматически, чтобы на разных мониторах и принтерах его выходные данные были одинаковыми. WCS 1,0 предоставляет набор функций для предоставления такого рода управления цветом, который является прозрачным для пользователя, и требует небольших издержек в приложении.

На более высоком уровне WCS 1,0 предоставляет дополнительные функции, обеспечивающие более сложное и управляемое управление цветом. Приложения для публикации графики и настольных систем потребуют этих дополнительных возможностей, чтобы пользователи могли точно работать с согласованным цветом на многих устройствах в рамках рабочего процесса.

Как правило, производители программного обеспечения, продукты которых работают с цветными входными или выходными данными и производителями оборудования цветных периферийных устройств, найдут WCS 1,0 ключевую технологию для упрощения доставки успешных продуктов. Оглавление раздела:
