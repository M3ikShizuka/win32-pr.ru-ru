# <a name="h"></a>H

## <a name="hard-proofing"></a>жесткое подтверждение

Предварительный просмотр или проверка образа путем печати его твердой копии.

## <a name="hls"></a>HLS

Цветовое пространство, часто используемое исполнителями. Его компоненты — это оттенок, освещение и насыщенность (чрома).

## <a name="hsv"></a>HSV

Цветовое пространство, часто используемое исполнителями. Его компоненты — это оттенок, насыщенность (чрома) и значение (освещение).

## <a name="hue"></a>цвет

Чистый цвет. Оттенок — это определенное расположение в видимой части электромагнитного спектра.
