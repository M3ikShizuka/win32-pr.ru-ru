# <a name="additive-primary-colors"></a>Аддитивные основные цвета

Метод аддитивного смешения цветов основан на предположении, которое начинается с черного. То есть если другие цвета отсутствуют (изображение является черным) и вы добавляете красный цвет, изображение отображается красным. Если затем добавить синий цвет, изображение будет пурпурным.

Три аддитивных [основных цвета](p.md) — красный, зеленый и синий. На следующем рисунке показана аддитивная основная смешение цветов.
