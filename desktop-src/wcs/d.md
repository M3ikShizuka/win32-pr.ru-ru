# <a name="d"></a>D

## <a name="dac"></a>DAC

Преобразование цифрового в аналоговый.

## <a name="device-link-profile"></a>Профиль связи устройства

Файл, содержащий объединение преобразований цветов, которые часто используются.

## <a name="device-profile"></a>профиль устройства

Файл, содержащий сведения о том, как характеризуется цветовой возможности устройства. Профили устройств ICC преобразуют цвета из цветового охвата устройства в ПК. Он также содержит сведения, необходимые для преобразования цветов из компьютеров в цветовой охват устройства. Профили устройств XML WCS предоставляют данные измерения и алгоритмические ссылки для преобразования между собственным пространством устройства и ЦИЕКСИЗ.

## <a name="dmp"></a>DMP

Формат профиля модели цветового устройства — это формат файла на основе XML, который содержит измерения, необходимые для описания связи между значениями цветового пространства собственного устройства и значениями ЦИЕКСИЗ.
