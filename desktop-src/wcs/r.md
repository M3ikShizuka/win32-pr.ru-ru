# <a name="r"></a>R

## <a name="rendering-intent"></a>Цель отрисовки

Подходы к отрисовке цвета из одного цветового пространства в другое. Механизм рендеринга, используемый в профилях ICC, определяется международным консорциумом по цветам (ICC).

## <a name="rgb"></a>RGB

Красное, зеленое и синее цветовое пространство.

## <a name="rgb-triple"></a>RGB, тройной

Количество, содержащее значения цветового пространства RGB.
