# <a name="g"></a>G

## <a name="gamma-correction"></a>гамма-коррекция

Собственные характеристики физического тона устройства часто отличаются на разных типах устройств. Коэффициент коррекции гаммы компенсирует эти различия.

## <a name="gamma-curve"></a>гамма-кривая

Коллекция коррекций гаммы для всего диапазона цветов. При отображении на диаграмме они формируют кривую.

## <a name="gamma-ramp"></a>гамма-шкала

См. раздел гамма-кривая.

## <a name="gamut"></a>спектр

Набор цветов из цветового пространства, которые может создать устройство.

## <a name="gmmp"></a>гммп

Формат профиля модели карты цветовой палитры — это формат XML-файла, который управляет сопоставлением цветов между различными диапазонами исходных и целевых значений устройств.
