# <a name="cmm-transform-creation-flags"></a>Флаги создания преобразования CMM

Кммс используйте флаги создания преобразования в качестве подсказок для создания преобразования цветов. Для определения оптимального использования этих флагов необходимо использовать модуль CMM.

Все функции, которые используют эти флаги, передают или получают значения флагов, используя параметр с именем *dwFlags*. В приведенной ниже таблице должно быть задано значение высокого порядка **слова** *dwFlags* .



| Константа                    | Описание                                                                                                                                  |
|-----------------------------|----------------------------------------------------------------------------------------------------------------------------------------------|
| ВКЛЮЧИТЬ \_ проверку палитры \_     | Используйте это преобразование для проверки цветовой палитры.                                                                                                       |
| ИСПОЛЬЗОВАТЬ \_ относительную \_ цветопередачу | Не сохраняйте белую точку. Если выходной диапазон не поддерживает данный цвет, используйте ближайший поддерживаемый цвет. См. раздел Обработка целей. |
| Быстрый \_ перевод             | Поиск только по цвету. Не Выменяйте интерполяцию цвета.                                                                                            |
| пресервеблакк               | Вставляет соответствующий ГММП "черного поколения" в качестве последней ГММП в последовательности преобразования.                                                     |
| WCS \_ всегда                 | Используйте путь к коду WCS даже для преобразований ICC.                                                                                               |
| последовательное \_ Преобразование       | Флаг создания преобразования для создания последовательного (не оптимизированного) преобразования цвета.                                                           |



 

**Слово** низкого порядка может иметь одно из следующих значений констант.



| Константа     | Описание                                                                                        |
|--------------|----------------------------------------------------------------------------------------------------|
| \_режим цветопробы  | Для предварительного просмотра изображения будет использоваться преобразование. Низкое качество изображения.                                    |
| НОРМАЛЬНЫЙ \_ режим | Преобразование будет использоваться для обычного вывода изображения. Среднее качество изображения.                            |
| Лучший \_ режим   | Преобразование будет использоваться для отображения изображения самого высокого качества на целевом устройстве. |



 

При переходе из \_ режима цветопробы в наилучший \_ режим качество вывода обычно повышается, а скорость преобразования отклоняется.
