# <a name="color-space-constants"></a>Константы цветового пространства

Значение подписи, указывающее цветовое пространство, в котором определяются данные профиля. Может принимать любое из следующих значений.



| Класс Profile     | Подпись         |
|-------------------|-------------------|
| XYZ               | ПРОСТРАНСТВО \_ XYZ        |
| Лаборатория               | МОДУЛЬное \_ занятие        |
| лув               | \_Лув пространства        |
| икбкр             | \_ИКБКР пространства      |
| икси               | \_ИКСИ пространства        |
| RGB               | ПРОСТРАНСТВО \_ RGB        |
| Серый масштаб        | \_серый пробел       |
| HSV               | \_HSV пространства        |
| HLS               | \_HLS пространства        |
| ИЗОБРАЖЕНИЯ              | ПРОСТРАНСТВО \_ CMYK       |
| кми               | \_КМИ пространства        |
| Универсальный канал 2 | \_Канал 2 \_ |
| Универсальный канал 3 | \_Канал 3 \_ |
| Универсальный 4 канала | \_Канал 4 \_ |
| Универсальный 5 канал | \_Канал 5 \_ |
| Универсальный канал 6 | \_Канал 6 \_ |
| Универсальный канал 7 | \_Канал 7 \_ |
| Универсальный 8 канал | \_Канал 8 \_ |
