# <a name="security-considerations-windows-color-system"></a>вопросы безопасности: Windows цветовая система

В этом документе содержатся сведения о вопросах безопасности, связанных с управлением цветом изображений. Этот документ не предоставляет все, что нужно знать о проблемах безопасности. вместо этого используйте его в качестве отправной точки и справочной информации для этой области технологий.

## <a name="non-microsoft-cmms-can-run-in-system-context"></a>Сторонние Кммс могут работать в контексте системы

Модули управления цветом (не Майкрософт) (Кммс) должны рассматриваться как драйверы принтеров сторонних производителей, так как они могут работать в контексте системы для операций печати.
