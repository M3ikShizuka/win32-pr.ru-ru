# <a name="obsolete-wcs-functions"></a>Устаревшие функции WCS

В этом разделе описаны функции, устаревшие в WCS 1,0. В настоящее время они хранятся для обеспечения обратной совместимости. Однако они будут удалены из API в последующих выпусках WCS.



|     Функция                               |      Описание                             |
|--------------------------------------------|----------------------------------------------|
| [**упдатеикмрегкэй**](/windows/desktop/api/Wingdi/nf-wingdi-updateicmregkeya) | *Устарело*: управляет цветовыми профилями и кммс. |
