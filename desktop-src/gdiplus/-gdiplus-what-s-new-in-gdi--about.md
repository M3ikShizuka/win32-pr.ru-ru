# <a name="whats-new-in-gdi"></a>Новые возможности GDI+

Windows GDI+ отличается от Windows интерфейс графических устройств (GDI) несколькими способами. Во-первых, GDI+ расширяет возможности GDI, предоставляя новые возможности, такие как градиентные кисти и альфа-смешение. Во вторых, модель программирования пересмотрена, чтобы упростить программирование графики и сделать его более гибким.

-   [Изменения в модели программирования](-gdiplus-changes-in-the-programming-model-about.md)
