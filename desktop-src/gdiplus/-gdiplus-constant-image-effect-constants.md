# <a name="image-effect-constants"></a>Константы эффектов изображения

Класс [**Effect**](/windows/desktop/api/gdipluseffects/nl-gdipluseffects-effect) имеет несколько потомков, которые можно использовать для применения цветовых эффектов или корректировок к точечному рисунку. Каждый из потомков идентифицируется по идентификатору GUID. Следующие константы, определенные в Гдиплусеффектс. h, представляют идентификаторы GUID, которые определяют различные эффекты.

-   блуреффектгуид
-   бригхтнессконтрастеффектгуид
-   колорбаланцееффектгуид
-   колоркурвиффектгуид
-   колорлутеффектгуид
-   колорматриксеффектгуид
-   хуесатуратионлигхтнессеффектгуид
-   левелсеффектгуид
-   редэйекорректионеффектгуид
-   шарпенеффектгуид
-   тинтеффектгуид
