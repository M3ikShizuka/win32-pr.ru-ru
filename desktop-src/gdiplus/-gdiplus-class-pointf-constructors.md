# <a name="pointfpointf-constructors"></a>Конструкторы PointF. PointF

В этом разделе перечислены конструкторы класса [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) . Полный список классов см. в разделе **класс PointF**.

### <a name="overload-list"></a>Список перегрузок



| Конструктор                                                    | Описание                                                                                                                                                                       |
|:---------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PointF (SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inconstsizef_))   | Создает объект [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) , используя объект [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) для указания элементов данных **X** и **Y** .<br/> |
| [**PointF (PointF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inconstpointf_)) | Создает новый объект [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) и копирует данные из другого объекта **PointF** .<br/>                                                 |
| [**PointF (ВЕЩЕСТВЕНная, реальная)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf(inreal_inreal)) | Создает объект [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) , используя два вещественных числа для указания элементов данных **X** и **Y** .<br/>                                     |
| [**PointF ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-pointf-pointf)              | Создает объект [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) и инициализирует элементы данных **X** и **Y** равными нулю. Это конструктор по умолчанию.<br/>              |
