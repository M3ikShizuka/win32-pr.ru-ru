В этом разделе перечислены методы Сетцентерпоинт класса [**пасградиентбруш**](/windows/win32/api/gdipluspath/nl-gdipluspath-pathgradientbrush) . Полный список методов для класса **пасградиентбруш** см. в разделе [методы пасградиентбруш](-gdiplus-class-pathgradientbrush-methods.md).

| Метод                                                                                           | Описание                                                                                                                                                                                                                                                                                                                                    |
|:-------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Сетцентерпоинт (Point&)**](/windows/win32/api/gdipluspath/nf-gdipluspath-pathgradientbrush-setcenterpoint(inconstpoint_))   | Метод [**пасградиентбруш:: сетцентерпоинт**](/windows/win32/api/gdipluspath/nf-gdipluspath-pathgradientbrush-setcenterpoint(inconstpoint_)) задает центральную точку кисти градиента контура. По умолчанию Центральная точка находится в центроид контура границы кисти, но можно установить центральную точку на любое расположение внутри или за пределами пути.<br/>  |
| [**Сетцентерпоинт (PointF&)**](/previous-versions//ms535095(v=vs.85)) | Метод [**пасградиентбруш:: сетцентерпоинт**](/previous-versions//ms535095(v=vs.85)) задает центральную точку кисти градиента контура. По умолчанию Центральная точка находится в центроид контура границы кисти, но можно установить центральную точку на любое расположение внутри или за пределами пути.<br/> |
