# <a name="transformations"></a>Преобразования

Аффинное преобразование включает вращение, масштабирование, отражение, наклон и преобразование. В Windows GDI+ класс [**Matrix**](/windows/desktop/api/gdiplusmatrix/nl-gdiplusmatrix-matrix) предоставляет основу для выполнения аффинных преобразований на векторных рисунках, изображениях и тексте.

-   [Значение порядка преобразований](-gdiplus-why-transformation-order-is-significant-use.md)
