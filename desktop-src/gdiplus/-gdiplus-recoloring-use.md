# <a name="recoloring"></a>Перекрашивание

Изменение цвета — это процесс настройки цветов изображения. Некоторые примеры перекрашивания изменяют один цвет на другой, настраивая интенсивность цвета относительно другого цвета, настраивая яркость или контрастность всех цветов и преобразуя цвета в оттенки серого цвета.

-   [Преобразование цветов](-gdiplus-translating-colors-use.md)
-   [Масштабирование цветов](-gdiplus-scaling-colors-use.md)
-   [Вращение цветов](-gdiplus-rotating-colors-use.md)
-   [Наклон цветов](-gdiplus-shearing-colors-use.md)
