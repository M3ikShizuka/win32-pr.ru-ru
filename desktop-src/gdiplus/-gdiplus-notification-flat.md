# <a name="notification-functions"></a>Функции уведомлений



| Плоская функция                                                                  | Метод-оболочка                            | Комментарии                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
|--------------------------------------------------------------------------------|-------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Гпстатус ВИНГДИПАПИ Гдиплуснотификатионхук (выходной \_ токен PTR \* )<br/> | Не вызывается методами-оболочками.<br/> | Функция [**гдиплусстартуп**](/windows/desktop/api/Gdiplusinit/nf-gdiplusinit-gdiplusstartup) возвращает указатель на структуру [**гдиплусстартупаутпут**](/windows/desktop/api/Gdiplusinit/ns-gdiplusinit-gdiplusstartupoutput) (в своем выходном параметре). Один из членов структуры — это указатель на функцию обработчика уведомлений, имеющую ту же сигнатуру, что и Гдиплуснотификатионхук.<br/> Функцию обработчика уведомлений можно вызвать двумя способами. можно использовать указатель, возвращаемый функцией [**гдиплусстартуп**](/windows/desktop/api/Gdiplusinit/nf-gdiplusinit-gdiplusstartup) , или вызвать гдиплуснотификатионхук. На самом деле, Гдиплуснотификатионхук просто проверяет, подавляется ли фоновый поток, а затем вызывает функцию обработчика уведомлений, возвращаемую **гдиплусстартуп**.<br/> Параметр token получает идентификатор, который позже должен передать соответствующий вызов функции unhookа уведомления.<br/>                                                                                                                                         |
| VOID ВИНГДИПАПИ Гдиплуснотификатионунхук ( \_ токен типа ulong)<br/>         | Не вызывается методами-оболочками.<br/> | Функция [**гдиплусстартуп**](/windows/desktop/api/Gdiplusinit/nf-gdiplusinit-gdiplusstartup) возвращает указатель на структуру [**гдиплусстартупаутпут**](/windows/desktop/api/Gdiplusinit/ns-gdiplusinit-gdiplusstartupoutput) (в своем выходном параметре). Один из членов структуры — это указатель на функцию разперехватчика уведомления, которая имеет ту же сигнатуру, что и Гдиплуснотификатионунхук.<br/> Существует два способа вызова функции уведомления о неловушках. можно использовать указатель, возвращаемый функцией [**гдиплусстартуп**](/windows/desktop/api/Gdiplusinit/nf-gdiplusinit-gdiplusstartup) , или вызвать гдиплуснотификатионунхук. На самом деле, Гдиплуснотификатионунхук просто проверяет, подавляется ли фоновый поток, а затем вызывает функцию unhookа уведомления, возвращенную **гдиплусстартуп**.<br/> При вызове функции unhookа уведомления передайте токен, полученный ранее, от соответствующего вызова функции обработчика уведомлений. Если этого не сделать, будут обнаружены утечки ресурсов, которые не будут очищены до завершения процесса.<br/> |
