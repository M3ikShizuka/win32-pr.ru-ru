# <a name="creating-a-linear-gradient"></a>Создание линейного градиента

GDI+ предоставляет горизонтальные, вертикальные и диагональные линейные градиенты. По умолчанию цвет линейного градиента изменяется единообразно. Однако линейный градиент можно настроить таким образом, чтобы цвет был изменен неоднородным образом.

-   [Горизонтальные линейные градиенты](#horizontal-linear-gradients)
-   [Настройка линейных градиентов](#customizing-linear-gradients)
-   [Диагональные линейные градиенты](#diagonal-linear-gradients)

## <a name="horizontal-linear-gradients"></a>Горизонтальные линейные градиенты

В следующем примере кисть горизонтального линейного градиента используется для заливки линии, эллипса и прямоугольника.


```
LinearGradientBrush linGrBrush(
   Point(0, 10),
   Point(200, 10),
   Color(255, 255, 0, 0),   // opaque red
   Color(255, 0, 0, 255));  // opaque blue

Pen pen(&linGrBrush);

graphics.DrawLine(&pen, 0, 10, 200, 10);
graphics.FillEllipse(&linGrBrush, 0, 30, 200, 100);
graphics.FillRectangle(&linGrBrush, 0, 155, 500, 30);
```



Конструктор [**LinearGradientBrush**](/windows/desktop/api/gdiplusbrush/nl-gdiplusbrush-lineargradientbrush) получает четыре аргумента: две точки и два цвета. Первая точка (0, 10) связана с первым цветом (красным), а вторая точка (200, 10) связана со вторым цветом (синим). Как и следовало ожидания, линия, нарисованная от (0,0) до (200, 10), постепенно меняется от красного к синему.

10 в точках (50, 10) и (200, 10) не важны. Важно то, что две точки имеют одинаковую вторую координату — линия, соединяющая их, является горизонтальной. Эллипс и прямоугольник также постепенно меняются от красного к голубому, так как горизонтальная координата — от 0 до 200.

На следующем рисунке показана линия, эллипс и прямоугольник. Обратите внимание, что цветовой градиент повторяется, так как горизонтальная координата возрастает за пределами 200.

## <a name="customizing-linear-gradients"></a>Настройка линейных градиентов

В предыдущем примере компоненты цвета меняются линейно по мере перемещения от горизонтальной координаты 0 к горизонтальной координате 200. Например, точка, первая координата которой находится на расстоянии от 0 до 200, будет иметь синий компонент, наполовину от 0 до 255.

GDI+ позволяет настроить способ, которым цвет меняется от одного края градиента к другому. Предположим, что необходимо создать градиентную кисть, которая меняется с черного на красный в соответствии со следующей таблицей.



| Координата по горизонтали | Компоненты RGB |
|-----------------------|----------------|
| 0                     | (0, 0, 0)      |
| 40                    | (128, 0, 0)    |
| 200                   | (255, 0, 0)    |



 

Обратите внимание, что красный компонент имеет половину интенсивность, когда горизонтальная координата составляет всего 20% от 0 до 200.

В следующем примере вызывается метод [**LinearGradientBrush:: сетбленд**](/windows/desktop/api/Gdiplusbrush/nf-gdiplusbrush-lineargradientbrush-setblend) объекта [**LinearGradientBrush**](/windows/desktop/api/gdiplusbrush/nl-gdiplusbrush-lineargradientbrush) , чтобы связать три относительных интенситиес с тремя относительными позициями. Как и в предыдущей таблице, относительная интенсивность 0,5 связана с относительной позицией 0,2. Код заполняет эллипс и прямоугольник с помощью градиентной кисти.


```
LinearGradientBrush linGrBrush(
   Point(0, 10),
   Point(200, 10),
   Color(255, 0, 0, 0),     // opaque black 
   Color(255, 255, 0, 0));  // opaque red

REAL relativeIntensities[] = {0.0f, 0.5f, 1.0f};
REAL relativePositions[]   = {0.0f, 0.2f, 1.0f};

linGrBrush.SetBlend(relativeIntensities, relativePositions, 3);

graphics.FillEllipse(&linGrBrush, 0, 30, 200, 100);
graphics.FillRectangle(&linGrBrush, 0, 155, 500, 30);
```



На следующем рисунке показан итоговый эллипс и прямоугольник.

## <a name="diagonal-linear-gradients"></a>Диагональные линейные градиенты

Градиенты в предыдущих примерах были горизонтальными; Это значит, что цвет меняется постепенно по мере перемещения вдоль любой горизонтальной линии. Можно также определить вертикальные градиенты и диагональные градиенты. Следующий код передает точки (0, 0) и (200, 100) в конструктор [**LinearGradientBrush**](/windows/desktop/api/gdiplusbrush/nl-gdiplusbrush-lineargradientbrush) . Цвет синего цвета связан с (0, 0), а зеленый цвет связан с (200, 100). Линия (с толщиной пера 10) и эллипс заполняются с помощью кисти линейного градиента.


```
LinearGradientBrush linGrBrush(
   Point(0, 0),
   Point(200, 100),
   Color(255, 0, 0, 255),   // opaque blue
   Color(255, 0, 255, 0));  // opaque green

Pen pen(&linGrBrush, 10);

graphics.DrawLine(&pen, 0, 0, 600, 300);
graphics.FillEllipse(&linGrBrush, 10, 100, 200, 100);
```



На следующем рисунке показана линия и эллипс. Обратите внимание, что цвет эллипса постепенно меняется по мере перемещения вдоль любой линии, которая находится параллельно с линией, проходящего через (0, 0) и (200, 100).

![Иллюстрация, показывающая диагональный градиент, который заполняет эллипс и диагональную линию](images/lineargradient3.png)
