# <a name="pointpoint-constructors"></a>Конструкторы точки. Point

В этом разделе перечислены конструкторы класса [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) . Полный список классов см. в разделе **класс Point**.

### <a name="overload-list"></a>Список перегрузок



| Конструктор                                                | Описание                                                                                                                                                                      |
|:-----------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Точка (размер&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inconstsize_))   | Создает объект [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) , используя объект [**size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) для инициализации элементов данных **X** и **Y** .<br/> |
| [**Точка (точка&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inconstpoint_)) | Создает новый объект [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) и копирует элементы данных из другого объекта **Point** .<br/>                                           |
| [**Point (INT, INT)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inint_inint))  | Создает объект [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) , используя два целых числа для инициализации элементов данных **X** и **Y** .<br/>                                       |
| [**Point ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point)             | Создает объект [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) и инициализирует элементы данных **X** и **Y** равными нулю. Это конструктор по умолчанию.<br/>               |
