# <a name="colorcolor-constructors"></a>Конструкторы Color. Color

В этом разделе перечислены конструкторы класса [**Color**](/windows/win32/api/gdipluscolor/nl-gdipluscolor-color) . Полный список классов см. в разделе **класс Color**.

| Конструктор                                                               | Описание                                                                                                                                                                                                         |
|:--------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Цвет (ARGB)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb))                   | Создает объект [**Color:: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb)) , используя значение **ARGB** .<br/>                                                                                                    |
| [**Цвет (байт, байт, байт)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte))        | Создает объект [**Color:: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte)) , используя указанные значения для красного, зеленого и синего компонентов. Этот конструктор задает для альфа-компонента значение 255 (непрозрачный).<br/> |
| [**Цвет (байт, байт, байт)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) | Создает объект [**Color:: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) , используя указанные значения для альфа-, красного, зеленого и синего компонентов.<br/>                                                   |
| [**Color ()**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color)                            | Создает объект [**Color:: Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color) и инициализирует его непрозрачным черным цветом.
