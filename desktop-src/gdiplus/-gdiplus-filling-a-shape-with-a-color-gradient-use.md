# <a name="filling-a-shape-with-a-color-gradient"></a>Заливка фигуры цветным градиентом

Можно заполнить фигуру с постепенно изменяющимся цветом с помощью градиентной кисти. Windows GDI+ предоставляет кисти линейных градиентов и градиенты вдоль пути. На следующем рисунке показан прямоугольник, заполненный линейной градиентной кистью, и эллипс, заполненный градиентом вдоль пути.

![Иллюстрация, показывающая прямоугольник, заполненный сверху слева направо вниз, и эллипс, заполняемый от центра к краю](images/gradient2.png)

Дополнительные сведения о градиентных кистях см. [в разделе Заполнение фигур с помощью градиентной кисти](-gdiplus-filling-shapes-with-a-gradient-brush-use.md).
