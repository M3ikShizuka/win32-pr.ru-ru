# <a name="using-regions-gdi"></a>Использование регионов (GDI+)

класс Windows GDI+ [**Region**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-region) позволяет определить пользовательскую фигуру. Фигура может состоять из линий, многоугольников и кривых.

Два распространенных применения областей — проверка попадания и обрезка. Проверка нажатия определяет, была ли нажата мышь в определенной области экрана. Обрезка — это ограничивает рисование в определенном регионе.

В следующих разделах более подробно рассматриваются регионы:

-   [Проверка нажатия с использованием региона](-gdiplus-hit-testing-with-a-region-use.md)
-   [Обрезка с использованием региона](-gdiplus-clipping-with-a-region-use.md)
