# <a name="using-a-brush-to-fill-shapes"></a>Использование кисти для заливки фигур

объект [**кисти**](/windows/desktop/api/gdiplusbrush/nl-gdiplusbrush-brush) Windows GDI+ используется для заполнения внутренней части замкнутой фигуры. GDI+ определяет несколько стилей заливки: сплошной цвет, шаблон штриховки, текстура изображения и градиент цвета.

В следующих разделах более подробно рассматривается использование кистей.

-   [Заполнение фигуры сплошным цветом](-gdiplus-filling-a-shape-with-a-solid-color-use.md)
-   [Заполнение фигуры шаблоном штриховки](-gdiplus-filling-a-shape-with-a-hatch-pattern-use.md)
-   [Заполнение фигуры текстурой изображения](-gdiplus-filling-a-shape-with-an-image-texture-use.md)
-   [Мозаичное заполнение фигуры изображением](-gdiplus-tiling-a-shape-with-an-image-use.md)
-   [Заливка фигуры цветным градиентом](-gdiplus-filling-a-shape-with-a-color-gradient-use.md)
