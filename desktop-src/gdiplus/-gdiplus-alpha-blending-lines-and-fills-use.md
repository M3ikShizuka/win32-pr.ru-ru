# <a name="alpha-blending-lines-and-fills"></a>Альфа-смешение цвета для линий и заливок

в Windows GDI+ цвет — это 32-разрядное значение с 8 битами на каждый для альфа-, красного, зеленого и синего цветов. Альфа-значение обозначает прозрачность цвета — степень, до которой смешивается цвет с цветом фона. Значения альфа находятся в диапазоне от 0 до 255, где 0 представляет полностью прозрачный цвет, а 255 — полностью непрозрачный цвет.

Альфа-смешение — это попиксельное смешение данных исходного и фонового цветов. Каждый из трех компонентов (красный, зеленый, синий) данного исходного цвета смешивается с соответствующим компонентом цвета фона в соответствии со следующей формулой:

Дисплайколор = Саурцеколор × альфа/255 + backgroundColor x (255 – альфа)/255

Например, предположим, что красный компонент исходного цвета имеет значение 150, а красный компонент цвета фона — 100. Если альфа-значение равно 200, то красный компонент результирующего цвета вычисляется следующим образом:

150 × 200 / 255 + 100 × (255 – 200) / 255 = 139
