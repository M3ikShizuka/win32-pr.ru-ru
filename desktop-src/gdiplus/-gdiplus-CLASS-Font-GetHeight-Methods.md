# <a name="fontgetheight-methods"></a>Методы Font. методом Height

В этом разделе перечислены методы с методом Height класса [**Font**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-font) . Полный список методов для класса **Font** см. в разделе [методы шрифтов](-gdiplus-class-font-methods.md).

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                   | Описание                                                                                                                                                                                                                                                                                                                                                                                                                              |
|:-------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Высота (REAL)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-font-getheight(inreal))            | Метод [**font:: ковысоту**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-font-getheight(inreal)) получает междустрочный шаг (в пикселях) этого шрифта. Межстрочный интервал — это расстояние по вертикали между двумя опорными линиями двух соседних строк в тексте. Таким словами, междустрочный интервал включает пробел между строками, а также высоту самого символа.<br/>                                                                                      |
| [**Высота (графический \* )**](/previous-versions//ms536201(v=vs.85)) | Метод [**font::-Height**](/previous-versions//ms536201(v=vs.85)) получает Междустрочный зазор для этого шрифта в текущей единице указанного объекта [**Graphics**](/windows/win32/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) . Межстрочный интервал — это расстояние по вертикали между двумя опорными линиями двух соседних строк в тексте. Таким словами, междустрочный интервал включает пробел между строками, а также высоту самого символа.<br/> |
