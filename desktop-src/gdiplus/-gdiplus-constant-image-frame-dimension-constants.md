# <a name="image-frame-dimension-constants"></a>Константы измерения кадра изображения

Форматы файлов изображений формата GIF и TIFF позволяют хранить несколько кадров в одном файле изображения.... Для анимации используется несколько кадров в файле GIF, поэтому кадры говорят о том, что они находятся в измерении времени. Несколько кадров в TIFF-файле обычно используются как отдельные страницы, поэтому говорят, что кадры находятся в измерении страницы.

Следующие константы, определенные в Гдиплусимагинг. h, представляют идентификаторы GUID, которые определяют размеры страницы и времени.

-   фрамедименсионпаже
-   фрамедименсионтиме
