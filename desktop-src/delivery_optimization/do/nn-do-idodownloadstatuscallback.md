# <a name="idodownloadstatuscallback-interface"></a>Интерфейс Идодовнлоадстатускаллбакк

Интерфейс **идодовнлоадстатускаллбакк** используется для получения уведомлений о скачивании.

## <a name="methods"></a>Методы

Интерфейс **идодовнлоадстатускаллбакк** содержит следующие методы.

| Метод | Описание |
| ---- |:---- |
| [Идодовнлоадстатускаллбакк:: Онстатусчанжед](./nf-do-idodownloadstatuscallback-onstatuschanged.md) | Вызывает реализацию этого метода при каждом изменении состояния загрузки. |

| ---- |:---- |

| **Минимальная версия сервера** | Windows Server, \[ только приложения Win32 версии 1809\] |
