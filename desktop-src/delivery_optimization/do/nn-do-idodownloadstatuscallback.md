# <a name="idodownloadstatuscallback-interface"></a>Интерфейс Идодовнлоадстатускаллбакк

Интерфейс **идодовнлоадстатускаллбакк** используется для получения уведомлений о скачивании.

## <a name="methods"></a>Методы

Интерфейс **идодовнлоадстатускаллбакк** содержит следующие методы.

| Метод | Описание |
| ---- |:---- |
| [Идодовнлоадстатускаллбакк:: Онстатусчанжед](./nf-do-idodownloadstatuscallback-onstatuschanged.md) | Оптимизация доставки вызывает реализацию этого метода при каждом изменении состояния загрузки. |
