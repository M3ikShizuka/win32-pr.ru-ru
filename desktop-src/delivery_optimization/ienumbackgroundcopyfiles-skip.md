# <a name="ienumbackgroundcopyfilesskip-method"></a>Метод Иенумбаккграундкопифилес:: Skip

Пропускает следующее заданное число элементов в последовательности перечисления. Если в последовательности осталось меньше элементов, чем запрошенное число элементов для пропуска, он пропускается после последнего элемента последовательности.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Skip(
  [in] ULONG celt
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*celt* \[ окне\]
</dt> <dd>

Число пропускаемых элементов.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает следующие значения **HRESULT** , а также другие.



| Код возврата                                                                              | Описание                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------|
| <dl> <dt>S_OK * * * *</dt> </dl> | Число запрошенных элементов успешно пропущено.<br/> |
| <dl> <dt>**S_FALSE**</dt> </dl>  | Пропущено меньшее число запрошенных элементов.<br/>    |
