# <a name="ienumbackgroundcopyfilesreset-method"></a>Метод Иенумбаккграундкопифилес:: Reset

Сбрасывает последовательность перечисления в начало.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Reset();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает **S_OK** при успешном или одном из стандартных значений **HRESULT** com при ошибке.
