# <a name="do-enumerations"></a>Перечисления DO

[Интерфейсы оптимизации доставки (Do)](do-interfaces.md) используют следующие перечисления.

## <a name="in-this-section"></a>В этом разделе

| Раздел | Описание |
|-|-|
| [**BG_ERROR_CONTEXT**](bg-error-context.md) | Перечисление **BG_ERROR_CONTEXT** определяет постоянные значения, указывающие контекст, в котором произошла ошибка. |
| [**BG_JOB_PRIORITY**](bg-job-priority-.md) | Перечисление **BG_JOB_PRIORITY** определяет постоянные значения, указывающие уровень приоритета задания.  |
| [**BG_JOB_STATE**](bg-job-state-.md) | Перечисление **BG_JOB_STATE** определяет константные значения для различных состояний задания. |
| [**BG_JOB_TYPE**](bg-job-type.md) | Перечисление **BG_JOB_TYPE** определяет постоянные значения, указывающие тип задания перемещения, например download. |
| [**BITS_FILE_PROPERTY_ID**](bits-file-property-id-.md) | В перечислении **BITS_FILE_PROPERTY_ID** указываются значения, определяющие значения идентификаторов, соответствующие свойствам **баккграундкопифиле** . |
| [**BITS_JOB_PROPERTY_ID**](bits-job-property-id.md) | Перечисление **BITS_JOB_PROPERTY_ID** указывает идентификатор свойства для задания Do. Это перечисление используется в объединении [**BITS_JOB_PROPERTY_VALUE**](bits-job-property-value-.md) для определения типа значения, содержащегося в объединении. |
| [**BITS_JOB_TRANSFER_POLICY**](bits-job-transfer-policy-.md) | Перечисление **BITS_JOB_TRANSFER_POLICY** определяет значения идентификаторов, соответствующие свойствам Do. |
| [**деливерйоптимизатионфилепроперти**](deliveryoptimizationfileproperty.md) | Задает идентификатор необязательного свойства для файла DO. |
| [**додовнлоадкостполици**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadcostpolicy.md) | Указывает идентификатор параметров политик затрат, связанных со свойством **DODownloadProperty_CostPolicy** . |
| [**додовнлоадпроперти**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadproperty.md) | Указывает идентификатор свойств для операции скачивания DO. |
| [**додовнлоадпропертекс**](./dodownloadinternal/ne-dodownloadinternal-dodownloadpropertyex.md) | Указывает идентификатор расширенных свойств для операции "выполнить скачивание". |
| [**додовнлоадстате**](./deliveryoptimizationdownloadtypes/ne-deliveryoptimizationdownloadtypes-dodownloadstate.md) | Указывает идентификатор текущего состояния загрузки, который является частью структуры **DO_DOWNLOAD_STATUS** . |
| [**DownloadMode**](downloadmode.md) | Определяет различные режимы загрузки, используемые оптимизацией доставки.  |
| [**свармстатус**](swarmstatus.md) | Определяет состояние файла в рамках клиента оптимизации доставки. |
