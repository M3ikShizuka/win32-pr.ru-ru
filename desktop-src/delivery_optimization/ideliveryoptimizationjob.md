# <a name="ideliveryoptimizationjob-interface"></a>Интерфейс Иделиверйоптимизатионжоб

Используйте интерфейс **иделиверйоптимизатионжоб** для скачивания диапазонов файла.

## <a name="members"></a>Элементы

Интерфейс **иделиверйоптимизатионжоб** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иделиверйоптимизатионжоб** также имеет следующие типы членов:

- [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иделиверйоптимизатионжоб** содержит следующие методы.



| Метод                                                                  | Описание                                                                                         |
|:------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| [**аддфилевисранжес**](ideliveryoptimizationjob-addfilewithranges.md) | Добавляет файл в задание загрузки и задает диапазоны файлов, которые необходимо загрузить.<br/> |
