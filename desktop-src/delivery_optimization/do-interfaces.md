# <a name="delivery-optimization-interfaces"></a>Интерфейсы оптимизации доставки

Используйте следующие интерфейсы оптимизации доставки для передачи файлов и мониторинга заданий в очереди передачи.

| Интерфейс | Описание |
|-|-|
| [**ибаккграундкопикаллбакк**](ibackgroundcopycallback.md) | Клиенты реализуют интерфейс [**ибаккграундкопикаллбакк**](ibackgroundcopycallback.md) , чтобы получить уведомление о завершении задания, было ли оно изменено или находится в ошибке. |
| [**ибаккграундкоперрор**](ibackgroundcopyerror.md) | Извлекает сведения об ошибке задания. |
| [**ибаккграундкопифиле**](ibackgroundcopyfile.md) | Извлекает имена локальных и удаленных файлов для запроса на перемещение файла в задании и его ход выполнения. |
| [**IBackgroundCopyFile2**](ibackgroundcopyfile2.md) | Задает новое удаленное имя для файла и получает список загружаемых диапазонов. |
| [**IBackgroundCopyFile5**](ibackgroundcopyfile5.md) | Предоставляет методы универсального свойства Get и Set для свойств Баккграундкопифиле. |
| [**Использованием метода ibackgroundcopyjob**](ibackgroundcopyjob-.md) | Добавляет файлы в задание, задает уровень приоритета задания, определяет состояние задания, запускает и останавливает задание. |
| [**IBackgroundCopyJob5**](ibackgroundcopyjob5.md) | Запрашивает или задает несколько необязательных поведений задания. |
| [**ибаккграундкопиманажер**](ibackgroundcopymanager.md) | Создает задания перемещения, извлекает объект перечислителя заданий из очереди и получает отдельные задания из очереди. |
| [**иделиверйоптимизатионжоб**](ideliveryoptimizationjob.md) | Используйте для скачивания диапазонов файла. |
| [**иделиверйоптимизатионфиле**](ideliveryoptimizationfile.md) | Используется для задания состояния определенного файла. |
| [**идодовнлоад**](./do/nn-do-idodownload.md) | Используется для запуска загрузки и управления им. |
| [**идодовнлоадинтернал**](./dodownloadinternal/nn-dodownloadinternal-idodownloadinternal.md) | Используется для получения или задания расширенных свойств загрузки. |
| [**идодовнлоадстатускаллбакк**](./do/nn-do-idodownloadstatuscallback.md) | Используется для получения уведомлений о скачивании. |
| [**идоманажер**](./do/nn-do-idomanager.md) | Используется для создания новой загрузки и перечисления существующих Скачиваний. |
| [**иенумбаккграундкопифилес**](ienumbackgroundcopyfiles-.md) | Перечисляет файлы в задании. |
