# <a name="ienumbackgroundcopyfilesnext-method"></a>Метод Иенумбаккграундкопифилес:: Next

Возвращает заданное число элементов последовательности перечисления. Если в последовательности осталось меньше запрошенного числа элементов, то он извлекает оставшиеся элементы.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Next(
  [in]  ULONG               celt,
  [out] IBackgroundCopyFile **rgelt,
  [out] ULONG               *pceltFetched
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*celt* \[ окне\]
</dt> <dd>

Число запрошенных элементов.

</dd> <dt>

*rgelt* \[ заполняет\]
</dt> <dd>

Массив объектов [**ибаккграундкопифиле**](ibackgroundcopyfile.md) . По завершении необходимо освободить каждый объект в *rgelt* .

</dd> <dt>

*пцелтфетчед* \[ заполняет\]
</dt> <dd>

Число элементов, возвращаемых в *rgelt*. Если параметр *celt* равен 1, можно установить значение *Пцелтфетчед* равным **null** . В противном случае инициализируйте значение *пцелтфетчед* значением 0 перед вызовом этого метода.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает следующие значения **HRESULT** , а также другие.



| Код возврата                                                                              | Описание                                                        |
|------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| <dl> <dt>S_OK * * * *</dt> </dl> | Число запрошенных элементов успешно возвращено.<br/> |
| <dl> <dt>**S_FALSE**</dt> </dl>  | Возвращаемое значение меньше числа запрошенных элементов.<br/>    |
