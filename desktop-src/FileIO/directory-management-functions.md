# <a name="directory-management-functions"></a>Функции управления каталогом

В управлении каталогами используются следующие функции.

## <a name="in-this-section"></a>Содержание раздела



| Функция                                                                      | Описание                                                                                                                                                               |
|-------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateDirectory**](/windows/desktop/api/FileAPI/nf-fileapi-createdirectorya)<br/>                         | Создает каталог.<br/>                                                                                                                                       |
| [**креатедиректорекс**](/windows/desktop/api/WinBase/nf-winbase-createdirectoryexa)<br/>                     | Создает новый каталог с атрибутами указанного каталога шаблонов.<br/>                                                                                 |
| [**креатедиректоритрансактед**](/windows/desktop/api/WinBase/nf-winbase-createdirectorytransacteda)<br/>     | Создает новый каталог в качестве транзакционной операции с атрибутами указанного каталога шаблона.<br/>                                                      |
| [**финдклосечанженотификатион**](/windows/desktop/api/FileAPI/nf-fileapi-findclosechangenotification)<br/> | Останавливает наблюдение за обработчиком уведомлений об изменениях.<br/>                                                                                                                   |
| [**финдфирстчанженотификатион**](/windows/desktop/api/FileAPI/nf-fileapi-findfirstchangenotificationa)<br/> | Создает обработчик уведомлений об изменениях и настраивает начальные условия фильтра уведомлений об изменении.<br/>                                                                |
| [**финднекстчанженотификатион**](/windows/desktop/api/FileAPI/nf-fileapi-findnextchangenotification)<br/>   | Запрашивает обработку уведомления об изменении операционной системой в следующий раз, когда обнаруживается соответствующее изменение.<br/>                                         |
| [**GetCurrentDirectory**](/windows/desktop/api/WinBase/nf-winbase-getcurrentdirectory)<br/>                 | Извлекает текущий каталог для текущего процесса.<br/>                                                                                                       |
| [**реаддиректоричанжесексв**](/windows/desktop/api/WinBase/nf-winbase-readdirectorychangesexw)<br/>         | Извлекает сведения, описывающие изменения в указанном каталоге, которые могут включать расширенные сведения, если указан этот тип сведений.<br/> |
| [**реаддиректоричанжесв**](/windows/desktop/api/WinBase/nf-winbase-readdirectorychangesw)<br/>             | Извлекает сведения, описывающие изменения в указанном каталоге.<br/>                                                                               |
| [**ремоведиректори**](/windows/desktop/api/FileAPI/nf-fileapi-removedirectorya)<br/>                         | Удаляет существующий пустой каталог.<br/>                                                                                                                           |
| [**ремоведиректоритрансактед**](/windows/desktop/api/WinBase/nf-winbase-removedirectorytransacteda)<br/>     | Удаляет существующий пустой каталог в качестве транзакционной операции.<br/>                                                                                                 |
| [**сеткуррентдиректори**](/windows/desktop/api/WinBase/nf-winbase-setcurrentdirectory)<br/>                 | Изменяет текущий каталог для текущего процесса.<br/>                                                                                                         |
