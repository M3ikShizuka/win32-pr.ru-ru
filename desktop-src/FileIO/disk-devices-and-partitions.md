# <a name="disk-devices-and-partitions"></a>Дисковые устройства и разделы

Жесткий диск состоит из набора вращающиеся с накоплением, каждый из которых содержит электромагнитные данные, которые хранятся в концентрических кругах или в *дорожках*. Каждый Негнущийся имеет две головки — по одной на каждой стороне негнущийся, которая считывает или записывает данные по мере прокрутки диска. *Жесткий* диск управляет размещением, чтением и записью жесткого диска. Обратите внимание, что заголовки всех вращающиеся размещаются как единое целое.

Наименьшая адресная единица записи — это *сектор*. *Цилиндр* определяется как набор дорожек, которые отображаются в одном расположении на каждом Негнущийся. Например, на следующей диаграмме показан жесткий диск с четырьмя вращающиеся. Цилиндр X состоит из восьми дорожек (следом X от каждой стороны каждого Негнущийся).

![жесткий диск, включая дорожки, секторы и вращающиеся](images/diskdevice.png)

Жесткий диск может содержать один или несколько логических регионов, называемых *разделами*. Разделы создаются, когда пользователь форматирует жесткий диск в качестве *базового диска*. Windows также поддерживает *динамические диски*, которые не рассматриваются в этом разделе. Дополнительные сведения о базовых дисках и динамических дисках см. в разделе [базовые и динамические](basic-and-dynamic-disks.md)диски.

Создание нескольких разделов на диске обеспечивает внешний вид отдельных жестких дисков. Например, система с одним жестким диском с одним разделом содержит один том, обозначенный системой как диск C. Система с жестким диском с двумя разделами обычно содержит диски C и D. Наличие нескольких разделов на жестком диске может упростить управление системой, например, организацию файлов или поддержку нескольких пользователей.

Первый физический сектор на базовом диске содержит структуру данных, известную как основная загрузочная запись (MBR). MBR содержит следующие сведения:

-   Программа загрузки (размер до 442 байт)
-   Подпись диска (уникальное 4-байтовое число)
-   Таблица разделов (до четырех записей)
-   Маркер конца MBR (всегда 0x55AA)
