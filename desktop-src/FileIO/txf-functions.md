# <a name="txf-functions"></a>Функции TxF

Транзакционная NTFS (TxF) предоставляет следующие функции.

## <a name="in-this-section"></a>Содержание раздела



| Функция                                                                      | Описание                                                                                                                  |
|-------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [**тксфлогкреатефилереадконтекст**](/windows/desktop/api/TxfW32/nf-txfw32-txflogcreatefilereadcontext)<br/> | Создает контекст, используемый для чтения записей репликации.<br/>                                                         |
| [**тксфлогдестройреадконтекст**](/windows/desktop/api/TxfW32/nf-txfw32-txflogdestroyreadcontext)<br/>       | Закрывает контекст чтения, созданный функцией [**тксфлогкреатефилереадконтекст**](/windows/desktop/api/TxfW32/nf-txfw32-txflogcreatefilereadcontext) .<br/> |
| [**тксфлогреадрекордс**](/windows/desktop/api/TxfW32/nf-txfw32-txflogreadrecords)<br/>                     | Считывает записи о повторяемости из журнала.<br/>                                                                              |
