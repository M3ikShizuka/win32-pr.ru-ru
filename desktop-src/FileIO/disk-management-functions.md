# <a name="disk-management-functions"></a>Функции управления дисками

В оснастке «Управление дисками» используются следующие функции.

## <a name="in-this-section"></a>Содержание раздела



| Функция                                                    | Описание                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**жетдискфриспаце**](/windows/desktop/api/FileAPI/nf-fileapi-getdiskfreespacea)<br/>     | Извлекает сведения о указанном диске, включая объем свободного места на диске.<br/>                                                                                                                                                              |
| [**жетдискфриспацеекс**](/windows/desktop/api/FileAPI/nf-fileapi-getdiskfreespaceexa)<br/> | Извлекает сведения о объеме пространства, доступного на томе диска, общем объеме пространства, общем объеме свободного пространства и общем объеме свободного пространства, доступного пользователю, связанному с вызывающим потоком.<br/> |



 

Другие функции, используемые в управлении дисками.



| Функция                         | Описание                                                                                                                                                                                                                        |
|----------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateFile**](/windows/desktop/api/FileAPI/nf-fileapi-createfilea) | Создает или открывает файл или устройство ввода-вывода. Чаще всего используются следующие устройства ввода-вывода: файл, файловый поток, каталог, физический диск, том, буфер консоли, ленточный накопитель, ресурс связи, почтовый слот и канал.<br/> |
| [**DeleteFile**](/windows/desktop/api/FileAPI/nf-fileapi-deletefilea) | Удаляет существующий файл.<br/>                                                                                                                                                                                               |
