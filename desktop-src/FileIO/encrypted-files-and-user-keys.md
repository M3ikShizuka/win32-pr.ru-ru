# <a name="encrypted-files-and-user-keys"></a>Зашифрованные файлы и ключи пользователей

Чтобы создать новый ключ для пользователя, используйте функцию [**сетусерфилинкриптионкэй**](/windows/desktop/api/Winefs/nf-winefs-setuserfileencryptionkey) . Чтобы добавить ключи пользователей в зашифрованный файл, используйте функцию [**аддусерстоенкриптедфиле**](/windows/desktop/api/Winefs/nf-winefs-adduserstoencryptedfile) . Чтобы запросить ключи пользователя для зашифрованного файла, используйте функцию [**куерюсерсоненкриптедфиле**](/windows/desktop/api/Winefs/nf-winefs-queryusersonencryptedfile) . Чтобы удалить ключи пользователей из зашифрованного файла, используйте функцию [**ремовеусерсфроменкриптедфиле**](/windows/desktop/api/Winefs/nf-winefs-removeusersfromencryptedfile) .
