# <a name="about-disk-management"></a>Сведения об управлении дисками

Файловая система обеспечивает абстракцию физических характеристик устройств хранения, чтобы приложения могли просто выполнять запись и чтение из файлов. Однако хранилище на предприятии в значительной степени зависит от концепции дисков.
