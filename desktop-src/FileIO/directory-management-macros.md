# <a name="directory-management-macros"></a>Макросы управления каталогом

В точках повторного анализа используются следующие макросы.

## <a name="in-this-section"></a>Содержание раздела



| Макрос                                                                     | Описание                                                                                                                             |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**исрепарсетагмикрософт**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagmicrosoft)<br/>         | Определяет, указывает ли тег точки повторного анализа точку повторного анализа (Майкрософт).<br/>                                                  |
| [**исрепарсетагнамесуррогате**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagnamesurrogate)<br/> | Определяет, является ли связанная точка повторной обработки тега суррогатом для другой именованной сущности (например, подключенной папки).<br/> |
