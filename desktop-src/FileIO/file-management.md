# <a name="file-management-local-file-systems"></a>Управление файлами (локальные файловые системы)

*Объект File* предоставляет представление ресурса (физического устройства или ресурса, расположенного на физическом устройстве), которое может управляться системой ввода-вывода. Как и другие объекты, они обеспечивают общий доступ к ресурсу, имеют имена, защищены с помощью безопасности на основе объектов и поддерживают синхронизацию. Система ввода-вывода также позволяет выполнять чтение или запись в ресурс.
