# <a name="disk-management"></a>Управление дисками

*Жесткий диск* — это жесткий диск внутри компьютера, который хранит и обеспечивает сравнительно быстрый доступ к большим объемам данных. Это тип хранилища, наиболее часто используемый в Windows. Система также поддерживает съемные носители.
