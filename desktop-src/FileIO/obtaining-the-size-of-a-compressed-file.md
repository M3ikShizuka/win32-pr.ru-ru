# <a name="obtaining-the-size-of-a-compressed-file"></a>Получение размера сжатого файла

Чтобы получить сжатый размер файла, используйте функцию [**жеткомпресседфилесизе**](/windows/desktop/api/fileapi/nf-fileapi-getcompressedfilesizea) . Если файл сжат, его сжатый размер будет меньше его несжатого размера. Для определения несжатого размера файла используйте функцию [**GetFileSize**](/windows/desktop/api/FileAPI/nf-fileapi-getfilesize) .
