# <a name="file-access-rights-constants"></a>Константы прав доступа к файлам

Допустимые права доступа к файлам и каталогам включают в себя **Удаление**, **\_ Управление чтением**, **запись \_ DAC**, **запись \_ владельца** и **синхронизацию** [стандартных прав доступа](/windows/desktop/SecAuthZ/standard-access-rights). В следующей таблице перечислены права доступа, относящиеся к файлам и каталогам.

| <span id="FILE_ADD_SUBDIRECTORY"></span><span id="file_add_subdirectory"></span><dl> <dt>**Файл \_ ДОБАВИТЬ \_ подкаталог**</dt> <dt>4</dt> </dl>              | Для каталога — право на создание подкаталога.<br/>                                                                                                                                                                                                                                                                                                                    |

(Для локальных файлов операции записи не перезаписывают существующие данные, если этот флаг задан без **\_ \_ данных записи файла**.)
