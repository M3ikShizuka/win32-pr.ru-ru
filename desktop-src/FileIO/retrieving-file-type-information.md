# <a name="retrieving-file-type-information"></a>Получение сведений о типе файла

Функция [**жетфилетипе**](/windows/desktop/api/FileAPI/nf-fileapi-getfiletype) Извлекает тип файла: диск, символ (например, консоль), Pipe или Unknown. Функция [**жетбинаритипе**](/windows/desktop/api/WinBase/nf-winbase-getbinarytypea) определяет, является ли файл исполняемым, и если да, то он имеет тип исполняемого файла. Список поддерживаемых типов исполняемых файлов см. в разделе **жетбинаритипе** .
