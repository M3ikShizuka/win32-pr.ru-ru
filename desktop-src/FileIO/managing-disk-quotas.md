# <a name="managing-disk-quotas"></a>Управление квотами на диск

Файловая система NTFS поддерживает *дисковые квоты*, которые позволяют администраторам управлять объемом данных, которые каждый пользователь может хранить на томе файловой системы NTFS. Администраторы могут при необходимости настроить систему для регистрации событий, когда пользователи близки к своей квоте, а также отклонять дополнительное место на диске для пользователей, которые превышают их квоты. Администраторы также могут создавать отчеты и использовать монитор событий для отслеживания проблем с квотами.

Определить, поддерживает ли файловая система квоты диска, можно путем вызова функции [**жетволумеинформатион**](/windows/desktop/api/FileAPI/nf-fileapi-getvolumeinformationa) и проверки флага bit для **файловой системы \_ \_ квот** .
