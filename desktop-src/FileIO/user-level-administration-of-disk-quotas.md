# <a name="user-level-administration-of-disk-quotas"></a>Администрирование дисковых квот на уровне пользователя

Квоты диска прозрачны для пользователя. Когда пользователь запрашивает объем свободного места на диске, система сообщает только о доступной квоте, доступной пользователю. Если пользователь превышает эту квоту, система возвращает сообщение **об ошибке \_ \_ переполнения диска** , точно так же, как было бы означать, что диск заполнен.

Чтобы получить больше свободного места на диске после превышения квоты, пользователь должен выполнить одно из следующих действий.

-   Удалите некоторые файлы.
-   Попросите другого пользователя запросить владение некоторыми файлами.
-   Повысьте квоту у администратора.

Программы, которым требуется получить фактический объем свободного места на диске, могут вызвать функцию [**жетдискфриспацеекс**](/windows/desktop/api/FileAPI/nf-fileapi-getdiskfreespaceexa) и взглянуть на параметр *тоталнумбероффрибитес* .
