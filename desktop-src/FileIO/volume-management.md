# <a name="volume-management"></a>Управление томами

Самым высоким уровнем организации в файловой системе является *том*. Файловая система находится на томе. Том содержит по крайней мере одну *секцию*, которая является логическим подразделением физического диска (Дополнительные сведения см. в разделе [дисковые устройства и разделы](disk-devices-and-partitions.md)). Том, содержащий данные, которые находятся в одной секции, называется *простым томом*, а том, содержащий данные, которые находятся в нескольких разделах, называются томом с несколькими *разделами*.
