| Константа                                                                        | Описание                                                                                                                                                            |
|---------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Константы прав доступа к файлам**](file-access-rights-constants.md)<br/> | Список прав доступа, характерных для файлов и каталогов.<br/>                                                                                         |
| [**Константы атрибутов файлов**](file-attribute-constants.md)<br/>         | Атрибуты файлов — это значения метаданных, которые хранятся в файловой системе на диске и используются системой и доступны для разработчиков с помощью различных API-интерфейсов файлового ввода/вывода.<br/> |
