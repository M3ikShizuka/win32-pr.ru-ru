# <a name="com-object-constructor-strings"></a>Строки конструктора объектов COM+

Строки конструктора объектов COM+ поддерживают универсальное программирование. Они позволяют создавать один компонент с определенной степенью общего вида, чтобы его можно было настроить для конкретной задачи с помощью строк конструктора, указанных администратором.

В разделах, приведенных в следующей таблице, содержатся сведения об использовании строк конструктора объектов COM+ в качестве фона и задач.
