# <a name="contacting-support"></a>Обращение в службу поддержки

Если проблема не устранена, обратитесь в службу технической поддержки Майкрософт со следующими сведениями:

-   Топографии приложения, в котором произошла ошибка, например описание приложений, компонентов и интерфейсов.
-   Журнал событий приложений на всех компьютерах.
-   Описание способа воспроизведения ошибки, если это возможно.
