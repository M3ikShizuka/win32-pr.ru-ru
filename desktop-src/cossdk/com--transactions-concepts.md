# <a name="com-transactions-concepts"></a>Концепции транзакций COM+

Хотя COM+ обрабатывает много трудоемких сведений о программировании, связанных с обработкой транзакций, полезно иметь концептуальное представление о теории транзакций, когда необходимо программировать транзакции в COM+.

В разделах, приведенных в следующей таблице, рассматриваются понятия, относящиеся к обработке транзакций.



| Раздел                                                                                                                     | Описание                                                                                                                                                                                 |
|---------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Свойства ACID](acid-properties.md)<br/>                                                                         | Описывает атомарные, последовательные, изолированные и устойчивые свойства транзакций.<br/>                                                                                              |
| [Настройка транзакций](configuring-transactions.md)<br/>                                                       | Описывает значения атрибутов транзакций, которые можно назначить компонентам для определения степени защиты транзакций, которую необходимо применить.<br/>                              |
| [Настройка уровней изоляции транзакций](configuring-transaction-isolation-levels.md)<br/>                       | Описывает уровни изоляции, которые можно назначить транзакциям, что позволяет увеличивать или уменьшать параллелизм в зависимости от требований к производительности и масштабируемости.<br/> |
| [Управление автоматическими транзакциями в COM+](managing-automatic-transactions-in-com-.md)<br/>                         | Обзор автоматизированного процесса транзакций, поддерживаемого COM+. <br/>                                                                                                                |

| [Приведите собственную транзакцию (BYOT)](bring-your-own-transaction--byot-.md)<br/>                                     | Описывает, как разрешить компоненту наследовать внешнюю транзакцию.<br/>                                                                                                              |
