# <a name="specifying-an-object-constructor-string-for-a-component"></a>Указание строки конструктора объекта для компонента

Чтобы указать строку конструктора объекта для компонента, выполните следующие действия.

1.  В области сведений средства администрирования "службы компонентов" щелкните правой кнопкой мыши компонент, который необходимо настроить, и выберите пункт **Свойства**.
2.  В диалоговом окне Свойства компонента перейдите на вкладку **Активация** .
3.  Чтобы включить использование строки конструктора объектов, установите флажок **включить конструирование объектов** .
4.  В поле **строка конструктора** введите строку конструирования (например, DSN = аккаунтингсервер). Имеется возможность ввести пустую строку.

> [!Note]  
> Строки конструктора объектов не следует использовать для хранения конфиденциальных данных.
