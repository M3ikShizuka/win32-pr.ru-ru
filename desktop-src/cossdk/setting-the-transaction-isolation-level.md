# <a name="setting-the-transaction-isolation-level"></a>Установка уровня изоляции транзакций

Можно вручную задать уровень изоляции транзакций для компонентов с помощью средства администрирования "службы компонентов" или программно настроить уровень изоляции транзакции для компонента с помощью [интерфейсов администрирования com+](com--administration-interfaces.md).

Дополнительные сведения об уровнях изоляции транзакций см. в разделе [Настройка уровней изоляции транзакций](configuring-transaction-isolation-levels.md).

**Установка уровня изоляции транзакции с помощью средства администрирования "службы компонентов"**

1.  В дереве консоли щелкните правой кнопкой мыши компонент, который необходимо настроить, и выберите пункт **Свойства**.

2.  В диалоговом окне Свойства компонента перейдите на вкладку **транзакции** .

3.  В разделе **уровень изоляции транзакции** выберите нужное значение из раскрывающегося списка. Значение по умолчанию для всех компонентов **сериализуется**.

    > [!Note]  
    > Если в разделе **Поддержка транзакций** выбрано значение **отключено** или **не поддерживается** , то нельзя задать уровень изоляции транзакции.

     

4.  Нажмите **OK**.

Эту процедуру необходимо повторить для каждого компонента.
