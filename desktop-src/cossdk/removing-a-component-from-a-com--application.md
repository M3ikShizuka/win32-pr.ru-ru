# <a name="removing-a-component-from-a-com-application"></a>Удаление компонента из приложения COM+

Для удаления компонентов из приложения COM+ можно использовать средство администрирования "службы компонентов".

> [!Note]  
> Будьте осторожны при удалении и переустановке отдельных компонентов из приложения COM+. Этот процесс требует подробных сведений о классах компонента, включая их атрибуты, конфигурации безопасности для интерфейсов и методов, а также другие параметры.

 

**Удаление компонента из приложения COM+**

1.  В дереве консоли средства администрирования "службы компонентов" выберите приложение, содержащее компонент, который необходимо удалить.

2.  Откройте папку **Components** приложения.

3.  Выберите компонент, который необходимо удалить.

4.  В меню **действие** выберите команду **Удалить**. Можно также выбрать компонент, щелкнуть его правой кнопкой мыши и выбрать команду **Удалить**.

5.  Нажмите кнопку **Да** при появлении запроса на подтверждение действия.
