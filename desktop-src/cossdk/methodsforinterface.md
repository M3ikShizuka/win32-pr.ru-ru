# <a name="methodsforinterface-collection"></a>Коллекция Месодсфоринтерфаце

Содержит объект для каждого метода в интерфейсе, к которому относится коллекция.

Эта коллекция не поддерживает методы [**Add**](/windows/desktop/api/ComAdmin/nf-comadmin-icatalogcollection-add) и [**Remove**](/windows/desktop/api/ComAdmin/nf-comadmin-icatalogcollection-remove) объекта [**комадминкаталогколлектион**](comadmincatalogcollection.md) .

## <a name="members"></a>Элементы

Коллекция **месодсфоринтерфаце** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) , но не содержит дополнительных элементов.

## <a name="related-collections"></a>Связанные коллекции

Можно переходить от этой коллекции к любой из следующих коллекций:

-   [**ErrorInfo**](errorinfo.md)
-   [**PropertyInfo**](propertyinfo.md)
-   [**релатедколлектионинфо**](relatedcollectioninfo.md)
-   [**ролесформесод**](rolesformethod.md)

Вы можете переходить к этой коллекции из следующих коллекций:

-   [**интерфацесфоркомпонент**](interfacesforcomponent.md)

## <a name="properties"></a>Свойства

Объект [**комадминкаталогобжект**](comadmincatalogobject.md) в коллекции поддерживает следующие свойства:

-   [Автозаполнения](#autocomplete)
-   [ЭТОМУ](#clsid)
-   [Описание](#description)
-   [IID](#iid)
-   [Index](#index)
-   [Name](#name)

### <a name="autocomplete"></a>AutoComplete



| Ввод | Значение |
|----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Описание    | Определяет, будет ли автоматически деактивироваться объект при завершении метода, без обязательного вызова Сеткомплете или SetAbort. Это влияет только на установку по умолчанию бита «done» JIT-активации при запуске метода. Этот бит можно сбросить (как с помощью Сеткомплете или SetAbort) в методе, чтобы переопределить значение по умолчанию. |
| Access         | ReadWrite                                                                                                                                                                                                                                                                                                                                  |
| Тип           | Bool                                                                                                                                                                                                                                                                                                                                       |
| Значение по умолчанию        | Неверно                                                                                                                                                                                                                                                                                                                                      |
| Минимальная система | Windows 2000                                                                                                                                                                                                                                                                                                                               |
