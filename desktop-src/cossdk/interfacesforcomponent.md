# <a name="interfacesforcomponent-collection"></a>Коллекция Интерфацесфоркомпонент

Содержит объект для каждого интерфейса, предоставляемого компонентом, к которому относится коллекция.

Эта коллекция не поддерживает методы [**Add**](/windows/desktop/api/ComAdmin/nf-comadmin-icatalogcollection-add) и [**Remove**](/windows/desktop/api/ComAdmin/nf-comadmin-icatalogcollection-remove) объекта [**комадминкаталогколлектион**](comadmincatalogcollection.md) .

## <a name="members"></a>Элементы

Коллекция **интерфацесфоркомпонент** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) , но не содержит дополнительных элементов.

## <a name="related-collections"></a>Связанные коллекции

Можно переходить от этой коллекции к любой из следующих коллекций:

-   [**ErrorInfo**](errorinfo.md)
-   [**месодсфоринтерфаце**](methodsforinterface.md)
-   [**PropertyInfo**](propertyinfo.md)
-   [**релатедколлектионинфо**](relatedcollectioninfo.md)
-   [**ролесфоринтерфаце**](rolesforinterface.md)

Вы можете переходить к этой коллекции из следующих коллекций:

-   [**Компоненты**](components.md)

## <a name="properties"></a>Свойства

Объект [**комадминкаталогобжект**](comadmincatalogobject.md) в коллекции поддерживает следующие свойства:

-   [ЭТОМУ](#clsid)
-   [Описание](#description)
-   [IID](#iid)
-   [Name](#name)
-   [куеуинженаблед](#queuingenabled)
-   [куеуеингсуппортед](#queueingsupported)

### <a name="clsid"></a>CLSID



| Ввод | Значение |
|----------------|---------------------------|
| Описание    | Идентификатор GUID для компонента. |
| Access         | ReadOnly                  |
| Тип           | Строка                    |
| По умолчанию        | Н/Д                       |
| Минимальная система | Windows 2000              |



 

### <a name="description"></a>Описание



| Ввод | Значение |
|----------------|----------------------------------|
| Описание    | Описание интерфейса. |
| Access         | ReadWrite                        |
| Тип           | Строка                           |
| По умолчанию        | ""                               |
| Минимальная система | Windows 2000                     |



 

### <a name="iid"></a>IID



| Ввод | Значение |
|----------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| Описание    | Идентификатор GUID для интерфейса. Это свойство возвращается при вызове метода свойства [**ключа**](/windows/desktop/api/ComAdmin/nf-comadmin-icatalogobject-get_key) для объекта этой коллекции. |
| Access         | ReadOnly                                                                                                                                                  |
| Тип           | Строка                                                                                                                                                    |
| По умолчанию        | Н/Д                                                                                                                                                       |
| Минимальная система | Windows 2000                                                                                                                                              |



 

### <a name="name"></a>Имя



| Ввод | Значение |
|----------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Описание    | Имя интерфейса. Это свойство возвращается при вызове метода свойства [**Name**](/windows/desktop/api/ComAdmin/nf-comadmin-icatalogobject-get_name) для объекта этой коллекции. |
| Access         | ReadOnly                                                                                                                                                    |
| Тип           | Строка                                                                                                                                                      |
| По умолчанию        | Н/Д                                                                                                                                                         |
| Минимальная система | Windows 2000                                                                                                                                                |



 

### <a name="queuingenabled"></a>куеуинженаблед



| Ввод | Значение |
|----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Описание    | Помечает интерфейс как куеуабле и может быть установлен с помощью средства администрирования SDK или служб компонентов. Однако флаг **включения очереди** может быть установлен только для интерфейса с установленным флагом " **Поддерживаемые очереди** ". |
| Access         | ReadWrite                                                                                                                                                                                                                          |
| Тип           | Bool                                                                                                                                                                                                                               |
| Значение по умолчанию        | Неверно                                                                                                                                                                                                                              |
| Минимальная система | Windows 2000                                                                                                                                                                                                                       |



 

### <a name="queueingsupported"></a>куеуеингсуппортед



| Ввод | Значение |
|----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Описание    | Интерфейс поддерживает очереди. Для интерфейса, поддерживающего очереди, все методы должны иметь только параметры in. **Поддерживаемая поддержка очередей** доступна в виде свойства, доступного только для чтения. |
| Access         | ReadOnly                                                                                                                                                               |
| Тип           | Bool                                                                                                                                                                   |
| Значение по умолчанию        | Неверно                                                                                                                                                                  |
| Минимальная система | Windows 2000                                                                                                                                                           |
