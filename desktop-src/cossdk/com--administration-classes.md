# <a name="com-administration-classes"></a>Классы администрирования COM+

Ниже перечислены классы администрирования COM+.



| Класс                                                          | Описание                                                                                                                                            |
|----------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**комадминкаталог**](comadmincatalog.md)                     | Обращается к данным, хранящимся в каталоге COM+.                                                                                                  |
| [**комадминкаталогколлектион**](comadmincatalogcollection.md) | Представляет любую коллекцию в каталоге COM+. Используйте его для перечисления, добавления, удаления и извлечения элементов в коллекции, а также для доступа к связанным коллекциям. |
| [**комадминкаталогобжект**](comadmincatalogobject.md)         | Представляет элементы в коллекциях в каталоге COM+. Используйте его для извлечения и изменения свойств, предоставляемых элементом в коллекции.                      |
