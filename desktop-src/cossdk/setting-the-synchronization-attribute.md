# <a name="setting-the-synchronization-attribute"></a>Настройка атрибута синхронизации

Задайте атрибут синхронизации с помощью средства администрирования "службы компонентов" или библиотеки администрирования COM+. (См. раздел [Автоматизация администрирования com+](automating-com--administration.md).)

**Задание атрибута с помощью средства администрирования "службы компонентов"**

1.  После установки компонента откройте диалоговое окно **Свойства** компонента.

2.  Перейдите на вкладку **параллелизм** .

3.  В разделе **Поддержка синхронизации** щелкните переключатель рядом со значением, которое необходимо выбрать для компонента.

4.  Нажмите **OK**.
