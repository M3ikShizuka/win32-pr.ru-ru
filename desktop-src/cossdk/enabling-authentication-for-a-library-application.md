# <a name="enabling-authentication-for-a-library-application"></a>Включение проверки подлинности для библиотечного приложения

Так как приложения библиотеки COM+ размещаются в других процессах, их требования к безопасности могут отличаться от требований, предъявляемых к серверным приложениям. Если приложение библиотеки будет размещено в браузере, может потребоваться получение обратных вызовов, не прошедших проверку подлинности.

Чтобы решить эту необходимость, можно отключить проверку подлинности, чтобы ведущий процесс не выполнял проверку безопасности для вызывающих объектов приложения библиотеки. При отключении проверки подлинности вызовы приложения библиотеки эффективно проходят без проверки подлинности — все вызовы будут выполнены успешно. Дополнительные сведения см. в разделе [Библиотека Application Security](library-application-security.md).

**Включение или отключение проверки подлинности для библиотечного приложения**

1.  Щелкните правой кнопкой мыши приложение COM+, для которого включается или отключается проверка подлинности, и выберите пункт **Свойства**.

2.  В диалоговом окне Свойства приложения перейдите на вкладку **Безопасность** .

3.  В разделе **Проверка** подлинности установите флажок **включить проверку подлинности** , чтобы включить проверку подлинности. Снятие флажка приведет к отключению проверки подлинности.

4.  Нажмите кнопку **ОК**.
