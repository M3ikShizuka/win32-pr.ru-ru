# <a name="com-crm-interfaces"></a>Интерфейсы COM+ CRM

Интерфейсы CRM должны обеспечивать поддержку для работников CRM и компенсаторов CRM, разработанных с помощью Visual Basic и Visual C++.

Можно использовать компенсирующий диспетчер ресурсов COM+ (CRM) для быстрой и простой интеграции ресурсов приложений с транзакциями Microsoft координатор распределенных транзакций (DTC).

Для компонентов, написанных с помощью Visual Basic, проще создавать записи журнала в виде коллекции вариантов. Кроме того, Visual Basic компоненты являются потоковыми потоками, что подразумевает возможность маршалирования интерфейсов из многопоточного апартамента в однопотоковое подразделение. Компоненты CRM, разработанные с помощью Visual C++, могут также использовать модель потоков подразделений, хотя вместо этого рекомендуется использовать как потоковую модель.

Интерфейсы, описанные в следующей таблице, содержат подробные справочные сведения для разработчиков COM+ Крмс.



| Интерфейсы CRM                                             | Описание                                                                                                               |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**икрмкомпенсатор**](/windows/desktop/api/ComSvcs/nn-comsvcs-icrmcompensator)                 | Этот интерфейс предоставляет неструктурированные записи журнала в Visual C++.                                                           |
| [**икрмкомпенсаторвариантс**](/windows/desktop/api/ComSvcs/nn-comsvcs-icrmcompensatorvariants) | Этот интерфейс предоставляет данные структурированного журнала компенсатору CRM при использовании Visual Basic.                            |
| [**икрмформатлогрекордс**](/windows/desktop/api/ComSvcs/nn-comsvcs-icrmformatlogrecords)       | Этот интерфейс преобразует записи журнала в отображаемый формат, чтобы их можно было представить с помощью универсального средства мониторинга. |
| [**икрмлогконтрол**](/windows/desktop/api/ComSvcs/nn-comsvcs-icrmlogcontrol)                   | Этот интерфейс используется работником CRM и компенсатором CRM для записи записей в журнал и обеспечения их надежности.           |
| [**икрммонитор**](/windows/desktop/api/ComSvcs/nn-comsvcs-icrmmonitor)                         | Этот интерфейс фиксирует моментальный снимок текущего состояния CRM и содержит конкретный клерк CRM.                          |
| [**икрммониторклеркс**](/windows/desktop/api/ComSvcs/nn-comsvcs-icrmmonitorclerks)             | Этот интерфейс получает сведения о состоянии клерков.                                                             |
| [**икрммониторлогрекордс**](/windows/desktop/api/ComSvcs/nn-comsvcs-icrmmonitorlogrecords)     | Этот интерфейс отслеживает отдельные записи журнала, поддерживаемые определенным клерком CRM для заданной транзакции.            |
