# <a name="com-application-pooling"></a>Группирование приложений COM+

Пул приложений COM+ позволяет выполнять однопотоковые процессы масштабирования и также может помочь в восстановлении после сбоев в отдельных процессах, предоставляя другим работающим процессам возможность обработки активаций.
