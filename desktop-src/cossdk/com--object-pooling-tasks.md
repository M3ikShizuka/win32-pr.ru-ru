# <a name="com-object-pooling-tasks"></a>Задачи пула объектов COM+

Вы можете настроить компонент для объединения в пул с помощью средства администрирования "службы компонентов" или административных функций COM+. Перед настройкой необходимо убедиться, что компонент правильно написан для поддержки объединения в пул.

В следующих подразделах этого раздела приводятся пошаговые инструкции по использованию пула объектов COM+.



| Раздел                                                                                       | Описание                                                                                                                                 |
|---------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [Настройка компонента для объединения в пул](configuring-a-component-to-be-pooled.md)<br/> | Описывает, как создать компонент в составе пула.<br/>                                                                                      |
| [Мониторинг статистики объектов](monitoring-object-statistics.md)<br/>                 | Описывает, как отслеживать объекты, чтобы можно было точно настроить характеристики пула объектов для достижения желаемой производительности. <br/> |
