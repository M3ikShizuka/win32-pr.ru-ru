# <a name="com-resource-dispenser-interfaces"></a>Интерфейсы распределителя ресурсов COM+

Компоненты приложения используют распределитель ресурсов для доступа к общей информации. Интерфейсы, описанные в следующей таблице, содержат подробные справочные сведения для разработчиков распределительных ресурсов.



| Интерфейсы                                                | Описание                                                                                                               |
|-----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**идиспенсердривер**](/windows/desktop/api/ComSvcs/nn-comsvcs-idispenserdriver)<br/>   | Этот интерфейс вызывается держательом распределителя ресурсов для создания, прикрепления, вычисления и уничтожения ресурса.<br/> |
| [**идиспенсерманажер**](/windows/desktop/api/ComSvcs/nn-comsvcs-idispensermanager)<br/> | Распределитель ресурсов использует этот интерфейс для подключения к диспетчеру распределителя.<br/>                                    |
| [**ихолдер**](/windows/desktop/api/ComSvcs/nn-comsvcs-iholder)<br/>                     | Этот интерфейс используется для подготовки и обработки ресурсов.<br/>                                                        |
