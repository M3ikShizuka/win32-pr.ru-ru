# <a name="com-compensating-resource-manager"></a>Компенсирующие диспетчер ресурсов COM+

Компенсирующий диспетчер ресурсов COM+ (CRM) предоставляет быстрый и простой способ интеграции ресурсов приложений с транзакциями Microsoft координатор распределенных транзакций (DTC). Использование CRM — это альтернатива разработке полнофункционального диспетчера ресурсов Microsoft Transaction Services (MTS).
