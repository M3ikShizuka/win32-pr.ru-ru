# <a name="com-low-memory-activation-gates"></a>Шлюзы активации COM+ Low-Memory

Служба "шлюзы активации нехватки памяти COM+" автоматически проверяет память перед созданием сервера или объекта COM+. Если процентная доля виртуальной памяти, доступная для приложения, падает ниже фиксированного порога, активация завершается ошибкой до создания объекта. Таким образом, служба шлюза активации нехватки памяти значительно повышает надежность системы.

В следующих разделах содержатся сведения о фоновом и задаче службы шлюза активации нехватки памяти COM+.
