# <a name="copying-components"></a>Копирование компонентов

Компонент можно скопировать из одного приложения COM+ в другое. После копирования компонента в другое приложение COM+ этот компонент можно настроить не так, как исходный компонент.

**Копирование компонента из одного приложения COM+ в другое**

1.  В области сведений средства администрирования "службы компонентов" щелкните правой кнопкой мыши компонент, который необходимо скопировать, и выберите команду **Копировать**.

2.  В диалоговом окне **копирование компонента** в области **Выбор назначения** выберите приложение COM+, в которое будет скопирован компонент.

3.  Введите новый идентификатор ProgID для скопированного компонента.

    > [!Note]  
    > Идентификатор ProgID для исходного компонента отображается для справки и не может быть изменен.

     

4.  Введите новый идентификатор CLSID для скопированного компонента. COM+ отображает автоматически созданный CLSID. В большинстве случаев это достаточно для уникальной идентификации скопированного компонента. Введите новый идентификатор CLSID, только если вы хотите изменить его, предоставленный COM+.

5.  Нажмите кнопку **ОК**.
