# <a name="partition-implementation"></a>Реализация секционирования

При установке на сервере приложений COM+ автоматически создает секцию, называемую *глобальным разделом*. Глобальный раздел включает стандартный набор приложений COM+. Приложения COM+, установленные в глобальном разделе, автоматически становятся доступными для всех пользователей на сервере. Если у вас есть другие приложения COM+, которые вы хотите сделать доступными для всех пользователей на сервере, их можно добавить в глобальный раздел.

В дополнение к глобальному разделу можно создавать другие разделы для пользователей только на этом сервере или для пользователей в домене. Создание дополнительных разделов и установка в них нескольких конфигураций приложения COM+ позволяет пользователям одновременно выполнять эти конфигурации приложений на одном компьютере. Кроме того, установка приложений COM+ в раздел, отличный от глобального раздела, позволяет управлять доступом пользователей к этим приложениям.

**Windows XP:** Возможность создания, настройки или делегирования разделов COM+ недоступна. Глобальный раздел является единственным доступным разделом COM+.
