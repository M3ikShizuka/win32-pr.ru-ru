# <a name="com-administration-interfaces"></a>Интерфейсы администрирования COM+

Ниже перечислены интерфейсы администрирования COM+.



| Интерфейс                                        | Описание                                                                                                                                                                                      |
|--------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**икаталогколлектион**](/windows/desktop/api/ComAdmin/nn-comadmin-icatalogcollection) | Позволяет перечислять, добавлять, удалять и извлекать элементы в любой коллекции каталога COM+ и обращаться к связанным коллекциям.                                                               |
| [**икаталогобжект**](/windows/desktop/api/ComAdmin/nn-comadmin-icatalogobject)         | Позволяет получать и размещать свойства, предоставляемые объектами в каталоге COM+.                                                                                                                    |
| [**икомадминкаталог**](/windows/desktop/api/ComAdmin/nn-comadmin-icomadmincatalog)     | Инициирует сеанс для программного администрирования COM+, доступа к коллекциям в каталоге, установки приложений и компонентов COM+, запуска и завершения служб, а также подключения к удаленным серверам. |
| [**ICOMAdminCatalog2**](/windows/desktop/api/ComAdmin/nn-comadmin-icomadmincatalog2)   | Управляет взаимодействием приложений, компонентов и секций.                                                                                                                           |
