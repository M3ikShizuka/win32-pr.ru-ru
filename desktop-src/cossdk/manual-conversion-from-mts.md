# <a name="manual-conversion-from-mts"></a>Ручное преобразование из MTS

может потребоваться изолировать преобразование пакетов MTS в приложения COM+, чтобы оно находилось вне процесса установки Windows. Следующая процедура предлагает альтернативный подход:

1.  Экспортируйте пакеты MTS с помощью функции экспорта пакетов MTS 2,0 (в результате чего файл пакета MTS и коллекция других файлов).
2.  удалите пакеты MTS и Windows обновления или выполните чистую установку Windows.
3.  установите файлы пакета MTS (. pak) на компьютере Windows 2000 или более поздней версии с помощью мастера установки приложения "службы компонентов". Кроме того, необходимо сбросить удостоверение "Запуск от имени" приложения в системном реестре в разделе **hKey \_ Local \_ Machine \\ Software \\ Classes \\ AppID \\ runas**.

> [!Note]  
> файл журнала мтстоком создается только процедурой автоматического преобразования (с помощью Windows установки или запуска служебной программы мтстоком). Этот файл не создается при выполнении процедуры преобразования вручную.
