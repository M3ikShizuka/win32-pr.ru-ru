# <a name="enabling-role-based-authorization-checking"></a>Включение проверки авторизации Role-Based

Чтобы использовать безопасность на основе ролей в приложении COM+, необходимо сначала включить проверку авторизации на основе ролей для приложения. Это гарантирует, что пользователи, обращающиеся к приложению, проверяются на роль, к которой они принадлежат, прежде чем они смогут выполнять какие-либо действия в приложении. Если проверка авторизации на основе ролей не включена, безопасность на основе ролей для приложения не используется.

**Включение проверки авторизации на основе ролей**

1.  Щелкните правой кнопкой мыши приложение COM+, для которого включается проверка авторизации на основе ролей, и выберите пункт **Свойства**.

2.  В диалоговом окне Свойства приложения перейдите на вкладку **Безопасность** .

3.  В разделе " **авторизация**" установите флажок **Принудительная проверка доступа для этого приложения** . Снятие флажка означает, что для приложения не используется безопасность на основе ролей.

4.  Нажмите **OK**.

Выбранный параметр авторизации вступает в силу при следующем запуске приложения.
