# <a name="com-synchronization"></a>Синхронизация COM+

Служба синхронизации COM+ обеспечивает синхронизацию, которая передается из компонента в компонент и запрещает нескольким вызывающим сторонам вводить компонент в любое заданное время. Синхронизация определяет, когда именно потоки могут направлять вызовы объекту.
