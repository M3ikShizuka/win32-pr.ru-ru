# <a name="the-policy-object-type"></a>Тип объекта политики

Объект [**политики**](policy-object.md) является объектом контейнера. В этом контейнере могут находиться следующие типы объектов:

-   Объекты [**учетной записи**](account-object.md)
-   Объект [**трустеддомаин**](trusteddomain-object.md)
-   Секретный объект

Существует только один экземпляр объекта [**Policy**](policy-object.md) . Они не могут ни создаваться, ни уничтожаться.
