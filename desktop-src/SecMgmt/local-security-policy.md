# <a name="local-security-policy"></a>Локальная политика безопасности

Локальная политика безопасности системы — это набор сведений о безопасности локального компьютера.

Сведения о локальной политике безопасности включают в себя следующее:

-   Домены, имеющие доверие для проверки подлинности при попытке входа.
-   Какие учетные записи пользователей могут получать доступ к системе и как. Например, интерактивно, через сеть или как услуга.
-   Права и [*привилегии*](/windows/desktop/SecGloss/p-gly) , назначенные учетным записям.
-   Политика аудита безопасности.

[*Локальный центр безопасности*](/windows/desktop/SecGloss/l-gly) (LSA) сохраняет сведения о локальной политике в наборе [объектов политики LSA](lsa-policy-objects.md).
