# <a name="lsa-policy"></a>Политика LSA

[*локальный администратор безопасности*](/windows/desktop/SecGloss/l-gly) (LSA) — это защищенная подсистема Windows, которая хранит сведения обо всех аспектах локальной безопасности системы, называемых [локальной политикой безопасности](local-security-policy.md) системы. В дополнение к сведениям о политике, LSA предоставляет службы для перевода между именами и [*идентификаторами безопасности*](/windows/desktop/SecGloss/s-gly) (SID).
