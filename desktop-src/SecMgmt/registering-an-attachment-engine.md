# <a name="registering-an-attachment-engine"></a>Регистрация подсистемы вложений

Прежде чем подсистема настройки безопасности сможет вызвать обработчик вложений для обработки задач, зависящих от службы, необходимо установить подсистему вложений и зарегистрировать ее в подсистеме настройки безопасности.

**Установка и регистрация библиотеки DLL подсистемы вложений**

1.  Скопируйте библиотеку DLL подсистемы вложений в каталог в системе. Предпочтительный каталог —% WINDIR% \\ secedit \\ . Если этот каталог не существует, его следует создать.
2.  Создайте раздел реестра в следующем узле. *Имя службы* — это зарегистрированное имя для вложения. оно должно совпадать с именем службы, используемым в диспетчере управления службами, модулем, который управляет остановкой и запуском служб. Имя также должно быть уникальным, поэтому оно не конфликтует с именами других вложений.

    ```
    HKEY_LOCAL_MACHINE
       Microsoft
          Windows NT
             CurrentVersion
                SeCEdit
                   Service
                      service name
    ```

3.  Создайте следующее строковое значение в новом разделе реестра. *путь к подсистеме вложений* — это полный путь к модулю подсистемы вложений.<dl> <dd>**Сервицеаттачментпас**  =  *путь к подсистеме вложений*<dl> <dt>

Тип данных
</dt> <dd>REG_SZ</dd> </dl> </dd> </dl>
