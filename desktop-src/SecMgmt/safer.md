# <a name="safer"></a>Безопаснее

Функции в более безопасном API предоставляют любое приложение, запускающее программы из внешних источников, возможность запрашивать политику безопасности для утверждения перед запуском исполняемого файла. Эти функции можно вызывать перед загрузкой и запуском исполняемого или активного содержимого. Типы приложений, в которых используется более безопасный API, включают в себя приложения, обрабатывающие вложения (например, почтовые клиенты и мгновенные сообщения, которые могут передавать файлы) и интерпретаторы сценариев. поддержка функций в более безопасном интерфейсе API начинается с Windows XP. Дополнительные сведения см. в разделе [более безопасные функции](management-functions.md).
