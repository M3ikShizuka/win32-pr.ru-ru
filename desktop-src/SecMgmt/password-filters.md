# <a name="password-filters"></a>Фильтры паролей

Фильтры паролей позволяют реализовать политику паролей и уведомления об изменениях.

Когда выполняется запрос на изменение пароля, [*локальный администратор безопасности*](/windows/desktop/SecGloss/l-gly) вызывает фильтры паролей, зарегистрированные в системе. Каждый фильтр паролей вызывается дважды: сначала проверяется новый пароль, а затем, после того, как все фильтры проверяют новый пароль, чтобы уведомить о внесенных изменениях. Данный процесс показан на следующем рисунке.

![запрос на смену пароля](images/pswdfilte.png)

Уведомление о смене пароля используется для синхронизации изменений паролей с базами данных внешней учетной записи.

Фильтры паролей используются для принудительного применения политики паролей. Фильтры проверяют новые пароли и указывают, соответствует ли новый пароль реализованной политике паролей.

Общие сведения об использовании фильтров паролей см. в разделе [Использование фильтров паролей](using-password-filters.md).

Список функций фильтрации паролей см. в разделе [функции фильтрации паролей](management-functions.md).
