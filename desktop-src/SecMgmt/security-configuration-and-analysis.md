# <a name="security-configuration-and-analysis"></a>Анализ и настройка безопасности

Настройка и анализ безопасности — это Изолированная оснастка, которую пользователи могут использовать для импорта одной или нескольких сохраненных конфигураций в личную базу данных безопасности. При импорте конфигураций создается база данных безопасности, зависящая от компьютера, в которой хранится составная конфигурация. Эту составную конфигурацию можно применить к компьютеру и проанализировать текущую конфигурацию системы по сравнению с сохраненной составной конфигурацией, хранящейся в базе данных.

[Шаблоны безопасности](security-templates.md) , а также настройки и анализа безопасности составляют оснастки настройки безопасности.
