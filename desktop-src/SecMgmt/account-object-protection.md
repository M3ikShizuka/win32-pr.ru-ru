# <a name="account-object-protection"></a>Защита объектов учетных записей

Объекты [**учетной записи**](account-object.md) защищаются следующим образом:

-   В **мире** группы есть универсальное \_ выполнение.
-   У **локального \_ администратора** группы есть доступ для удаления, общего \_ чтения, универсальной \_ записи, общего \_ выполнения и записи \_ списка DACL.
-   Группа **"локальный \_ Администратор"** назначается как владелец и основная группа объектов учетной записи.
