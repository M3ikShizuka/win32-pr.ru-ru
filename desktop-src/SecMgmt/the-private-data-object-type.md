# <a name="the-private-data-object-type"></a>Тип закрытого объекта данных

Для хранения строк данных в зашифрованном виде можно использовать закрытые объекты данных. Система может содержать несколько частных объектов данных. Закрытые объекты данных предоставляются в основном для поддержки хранения паролей учетных записей сервера. Данные, хранящиеся в частных объектах данных, шифруются с помощью созданного системного ключа.
