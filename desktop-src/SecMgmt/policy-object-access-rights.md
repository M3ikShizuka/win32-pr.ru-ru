# <a name="policy-object-access-rights"></a>Права доступа к объектам политики

Объект [**политики**](policy-object.md) имеет следующие типы доступа для конкретного объекта:



| Тип доступа                         | Описание                                                                                                                                                                                                                                                                                                                                                        |
|-------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_ \_ Локальная информация представления политики \_    | Этот тип доступа необходим для чтения сведений о различных политиках безопасности целевой системы. К ним относятся квота по умолчанию, аудит, сведения о состоянии сервера и роли, а также сведения о доверии. Этот тип доступа также необходим для перечисления доверенных доменов, учетных записей и [*привилегий*](/windows/desktop/SecGloss/p-gly). |
| ПОЛИТИКА \_ получения \_ частной \_ информации   | Этот тип доступа необходим для просмотра конфиденциальной информации, например имен учетных записей, установленных для отношений доверенных доменов.                                                                                                                                                                                                                              |
| \_Администратор доверия \_ политики                | Этот тип доступа необходим для изменения сведений о домене учетной записи или основном домене.                                                                                                                                                                                                                                                                             |
| \_ \_ \_ ограничение квоты по умолчанию для набора политик \_ | Задайте системные квоты по умолчанию, применяемые к учетным записям пользователей.                                                                                                                                                                                                                                                                                                   |
| ПОЛИТИКА \_ создания \_ секрета              | Этот тип доступа необходим для создания нового закрытого объекта данных.                                                                                                                                                                                                                                                                                                    |
| ПОЛИТИКА \_ создания \_ учетной записи             | Этот тип доступа необходим для создания нового объекта [**учетной записи**](account-object.md) .                                                                                                                                                                                                                                                                               |
| \_ \_ требования к аудиту для набора политик \_    | Этот тип доступа необходим для обновления требований аудита системы.                                                                                                                                                                                                                                                                                      |
| \_ \_ Администратор журнала аудита \_ политики           | Этот тип доступа необходим для изменения характеристик журнала аудита, например его максимального размера или срока хранения записей аудита, а также для очистки журналов.                                                                                                                                                                                               |
| \_Просмотр политики \_ \_ сведения о аудите    | Этот тип доступа необходим для просмотра сведений о требованиях аудита или аудита.                                                                                                                                                                                                                                                                                  |
| \_Администратор сервера \_ политики               | Этот тип доступа необходим для изменения сведений о состоянии сервера или роли (главной или реплике). Также необходимо изменить сведения об источнике реплики и имени учетной записи.                                                                                                                                                                                           |
| \_имена уточняющего запроса политики \_               | Этот тип доступа необходим для преобразования между именами и идентификаторами безопасности.                                                                                                                                                                                                                                                                                                    |
| \_привилегия создания \_ политики           | Пока не поддерживается.                                                                                                                                                                                                                                                                                                                                                 |



 

## <a name="generic-access-masks"></a>Универсальные маски доступа

Объект [**политики**](policy-object.md) публикует следующие сопоставления из универсальных типов доступа в определенные типы доступа:

``` syntax
    GENERIC_READ    STANDARD_RIGHTS_READ |
                    POLICY_VIEW_AUDIT_INFORMATION |
                    POLICY_GET_PRIVATE_INFORMATION

    GENERIC_WRITE   STANDARD_RIGHTS_WRITE |
                    POLICY_TRUST_ADMIN |
                    POLICY_CREATE_ACCOUNT |
                    POLICY_CREATE_SECRET |
                    POLICY_CREATE_PRIVILEGE |
                    POLICY_SET_DEFAULT_QUOTA_LIMITS |
                    POLICY_SET_AUDIT_REQUIREMENTS |
                    POLICY_AUDIT_LOG_ADMIN |
                                        POLICY_SERVER_ADMIN

    GENERIC_EXECUTE STANDARD_RIGHTS_EXECUTE |
                    POLICY_VIEW_LOCAL_INFORMATION |
                    POLICY_LOOKUP_NAMES
```

## <a name="standard-access-types"></a>Стандартные типы доступа

Этот объект не поддерживает (необязательно) СИНХРОНИЗАЦИю стандартного типа доступа. Поддерживаются все необходимые типы доступа. Маска всех поддерживаемых типов доступа для этого типа объектов:

``` syntax
    POLICY_ALL_ACCESS STANDARD_RIGHTS_REQUIRED |
                    POLICY_VIEW_LOCAL_INFORMATION |
                    POLICY_VIEW_AUDIT_INFORMATION |
                    POLICY_GET_PRIVATE_INFORMATION |
                    POLICY_TRUST_ADMIN |
                    POLICY_CREATE_ACCOUNT |
                    POLICY_CREATE_SECRET |
                    POLICY_CREATE_PRIVILEGE |
                    POLICY_SET_DEFAULT_QUOTA_LIMITS |
                    POLICY_SET_AUDIT_REQUIREMENTS |
                    POLICY_AUDIT_LOG_ADMIN |
                    POLICY_SERVER_ADMIN
                    POLICY_LOOKUP_NAMES
```
