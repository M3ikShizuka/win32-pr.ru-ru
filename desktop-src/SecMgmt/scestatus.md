# <a name="scestatus"></a>сцестатус

Тип данных **сцестатус** используется в API набора средств настройки безопасности для получения сведений о результатах вызова функции. Дополнительные сведения об определенных значениях кода состояния см. в разделе [возвращаемые значения вложений](management-return-values.md).


```C++
typedef DWORD SCESTATUS;
```
