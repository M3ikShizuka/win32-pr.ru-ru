# <a name="modifying-configuration-information-in-the-database"></a>Изменение сведений о конфигурации в базе данных

Расширение оснастки «вложение» должно позволить пользователям изменять сведения о конфигурации через узел анализа. Это позволяет пользователям вносить изменения, а затем повторно применять конфигурацию с течением времени. Логика для этого должна быть идентична изменению информации в файлах конфигурации. Внесенные изменения вступят в силу в отношении сохраненной конфигурации в базе данных безопасности.
