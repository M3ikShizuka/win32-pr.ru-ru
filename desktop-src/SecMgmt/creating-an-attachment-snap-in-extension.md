# <a name="creating-an-attachment-snap-in-extension"></a>Создание расширения оснастки «вложение»

Расширение оснастки «вложение» предоставляет интерфейс, который пользователи могут использовать для изменения параметров конфигурации, зависящих от службы. Расширение оснастки "вложение" должно соответствовать требованиям MMC, чтобы быть допустимым расширением оснастки. Дополнительные сведения об этих требованиях см. в документации по [консоли управления Microsoft Management](/previous-versions/windows/desktop/mmc/microsoft-management-console-start-page) .

В дополнение к интерфейсам, необходимым для MMC, расширение оснастки вложения должно реализовывать COM-интерфейс [**исцесвкаттачментперсистинфо**](/windows/desktop/api/Scesvc/nn-scesvc-iscesvcattachmentpersistinfo). Оснастки настройки безопасности вызывают методы этого интерфейса, чтобы определить, изменились ли данные конфигурации, и, если да, обновить базу данных безопасности. Оснастка "вложение" должна сохранять любые изменения конфигурации до тех пор, пока оснастки настройки безопасности не извлекают эти данные.

Расширение оснастки вложений должно обеспечивать следующие функциональные возможности.

-   [Отображение сведений о конфигурации и анализе](displaying-configuration-and-analysis-information.md)
-   [Изменение сведений о конфигурации в пользовательском интерфейсе](modifying-configuration-information-in-the-user-interface.md)
-   [Изменение сведений о конфигурации в базе данных безопасности](modifying-configuration-information-in-the-database.md)

Чтобы помочь вашему расширению оснастки при выполнении этих задач, оснастки настройки безопасности реализуют COM-интерфейс [**исцесвкаттачментдата**](/windows/desktop/api/Scesvc/nn-scesvc-iscesvcattachmentdata), предоставляющий методы, которые может вызывать расширение оснастки для инициализации и запроса сведений из базы данных безопасности.

После создания расширения оснастки вложения необходимо зарегистрировать его с помощью оснасток настройки безопасности, как описано в разделе [Регистрация расширения оснастки вложений](registering-an-attachment-snap-in-extension.md).
