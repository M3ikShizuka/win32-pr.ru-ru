# <a name="lsa-policy-objects"></a>Объекты политики LSA

LSA сохраняет сведения о локальной политике безопасности в наборе объектов. Приложение может запрашивать или изменять локальную политику безопасности, обращаясь к этим объектам.

Набор состоит из следующих четырех объектов:

-   [Политика](policy-object.md) содержит сведения о глобальных политиках.
-   [Трустеддомаин](trusteddomain-object.md) содержит сведения о доверенном домене.
-   [Учетная запись](account-object.md) содержит сведения о пользователе, группе или учетной записи локальной группы.
-   [Личные данные](private-data-object.md) содержат защищенную информацию, например пароли учетной записи сервера. Эти сведения хранятся в виде зашифрованных строк.

Дополнительные сведения об использовании функций политики LSA для управления данными, хранящимися в этих объектах, см. в разделе [Использование политики LSA](using-lsa-policy.md).
