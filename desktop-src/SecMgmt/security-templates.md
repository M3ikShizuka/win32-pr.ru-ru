# <a name="security-templates"></a>Шаблоны безопасности

Шаблоны безопасности — это изолированное средство оснастки, которое пользователи могут использовать для определения конфигураций безопасности, не зависящих от компьютера. Эти конфигурации сохраняются в виде текстовых INF-файлов.

Шаблоны безопасности, а также [настройки и анализа безопасности](security-configuration-and-analysis.md) составляют оснастки настройки безопасности.
