# <a name="using-service-security-attachments"></a>Использование вложений безопасности службы

В этом разделе описывается создание компонентов вложений, необходимых для поддержки конфигурации, зависящей от службы, в оснастках настройки безопасности. Вам потребуется реализовать и зарегистрировать два компонента: оснастку расширения для вывода пользовательского интерфейса и подсистемы вложений для обработки запросов конфигурации и анализа.

-   [Регистрация расширения оснастки "вложение"](registering-an-attachment-snap-in-extension.md)
-   [Создание подсистемы вложений](creating-an-attachment-engine.md)
-   [Регистрация подсистемы вложений](registering-an-attachment-engine.md)
