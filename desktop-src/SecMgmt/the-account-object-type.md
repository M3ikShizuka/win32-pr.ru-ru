# <a name="the-account-object-type"></a>Тип объекта Account

Объекты [**Account**](account-object.md) используются для обозначения [*привилегий*](/windows/desktop/SecGloss/p-gly)или прав доступа к системе, которые должны быть бестовед для пользователя или членов групп или локальных групп.

Объекты [**учетной записи**](account-object.md) находятся в объекте [**политики**](policy-object.md) . Может быть несколько этих объектов или ни одного.

Объект [**учетной записи**](account-object.md) не имеет типов данных, относящихся к объекту.
