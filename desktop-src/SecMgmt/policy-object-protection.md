# <a name="policy-object-protection"></a>Защита объектов политики

Объект [**политики**](policy-object.md) защищен по умолчанию со следующими параметрами:

-   В мире локальной группы есть универсальный \_ доступ для выполнения.
-   Известной системе идентификации предоставляется \_ доступ ко всем политикам \_ .
-   ЛОКАЛЬНЫЙ администратор локальной группы \_ имеет универсальное \_ чтение, универсальную \_ запись и универсальный \_ доступ для выполнения.
-   Группа "локальный \_ Администратор" назначена как владелец и основная группа этого объекта.

Невозможно создать или уничтожить объект [**политики**](policy-object.md) .
