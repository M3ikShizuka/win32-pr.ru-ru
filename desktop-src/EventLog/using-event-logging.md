# <a name="using-event-logging"></a>Использование ведения журнала событий

Приложения могут использовать API ведения журнала событий для отчетов и просмотра событий. Дополнительные сведения о событиях отчетов см. в разделе [отчеты о событиях](reporting-an-event.md).

Сведения о просмотре событий, записанных в файлы журналов, см. в разделе [запросы к сообщениям источника событий](querying-for-event-source-messages.md) и [Получение уведомлений о событиях](receiving-event-notification.md). Для просмотра событий также можно использовать Просмотр событий.
