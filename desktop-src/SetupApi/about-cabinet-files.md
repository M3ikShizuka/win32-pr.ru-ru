# <a name="about-cabinet-files"></a>О CAB-файлах

CAB — это один файл, который обычно имеет суффикс. CAB, который хранит сжатые файлы в библиотеке файлов. Сжатый файл может быть распределен по нескольким CAB-файлам. Во время установки приложение установки распаковывает файлы, хранящиеся в CAB-файле, и копирует их в систему пользователя.

Если используются CAB-файлы, функции установки используют имя CAB-файла в качестве файл тегов для исходного диска, на котором он находится.
