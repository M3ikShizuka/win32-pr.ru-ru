# <a name="releasing-other-system-resources"></a>Освобождение других системных ресурсов

Если используется исходный список или файл журнала, ресурсы должны быть освобождены путем вызова [**сетупфрисаурцелист**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista) или [**сетуптерминатефилелог**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog).
