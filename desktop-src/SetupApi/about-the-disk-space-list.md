# <a name="about-the-disk-space-list"></a>О списке Disk-Space

Список дискового пространства позволяет вычислить место на диске, необходимое для завершения операций установки. После создания списка дискового пространства и добавления в него файловых операций можно запросить список, чтобы определить целевые диски, указанные в операциях с файлами, и место на диске, необходимое для каждого диска.

Сравнивая пространство, необходимое для свободного места на целевых дисках, можно программным образом определить, достаточно ли свободного места для текущей установки.

Вы можете добавить или удалить файловые операции в списке дискового пространства и повторно рассчитать необходимый объем дискового пространства. Эта функция позволяет, например, написать программу, которая взаимодействует с пользователем, позволяя ему выбрать компоненты, которые требуется установить, в зависимости от требуемого места на диске.

Функции списка дискового пространства автоматически проверяют наличие существующих копий файлов на целевом диске. Если была найдена Предыдущая версия файла, функции списка дискового пространства вычисляют дополнительное пространство, необходимое для завершения операций с файлами.

Например, если операция копирования указывает, что 6000-байтовый файл будет скопирован на диск C, а на диске C уже существует 2000-байтовый файл, то функции дискового пространства будут возвращать требуемый объем в 4000 байт. Дополнительное пространство используется для замены более старой версии файла новой версией.

В списке дискового пространства функции округляют размер файлов в границах кластерных дисков.
