# <a name="spfilenotify_renameerror-message"></a>\_Сообщение спфиленотифи ренамиррор

Уведомление **спфиленотифи \_ ренамиррор** отправляется в подпрограммы обратного вызова при возникновении ошибки во время операции переименования файла.


```C++
SPFILENOTIFY_RENAMEERROR
  Param1 = (UINT) FilePathInfo;
  Param2 = (UINT) 0;
            
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Параметр1* 
</dt> <dd>

Указатель на структуру [**FilePath**](/windows/desktop/api/Setupapi/ns-setupapi-filepaths_a) . Элемент **Win32Error** структуры **filePaths** указывает ошибку, которая произошла во время операции переименования.

</dd> <dt>

*Param2* 
</dt> <dd>

Не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Подпрограммы обратного вызова должны возвращать один из следующих элементов.



| Код возврата                                                                                  | Описание                                                                                                                                                                                                                                                       |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**ФИЛЕОП \_ Повтор**</dt> </dl> | Пользователь выбрал попытку повторного выполнения операции переименования.<br/>                                                                                                                                                                                                  |
| <dl> <dt>**ФИЛЕОП \_ пропустить**</dt> </dl>  | Пользователь решил пропустить операцию переименования файла.<br/>                                                                                                                                                                                                      |
| <dl> <dt>**прервать ФИЛЕОП \_**</dt> </dl> | Останавливает фиксацию очереди. [**Сетупкоммитфилекуеуе**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea) возвращает **значение false**. [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror) Возвращает расширенные сведения об ошибке, такие как ошибка \_ отмены (если пользователь был отменен) или ошибка \_ не \_ хватает \_ памяти.<br/> |
