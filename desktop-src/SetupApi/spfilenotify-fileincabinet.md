# <a name="spfilenotify_fileincabinet-message"></a>\_Сообщение спфиленотифи филеинкабинет

Уведомление **спфиленотифи \_ филеинкабинет** отправляется в подпрограммы обратного вызова с помощью [**сетупитератекабинет**](/windows/desktop/api/Setupapi/nf-setupapi-setupiteratecabineta) для каждого файла, найденного в CAB-файле. Подпрограммы обратного вызова должны возвращать значение, указывающее, следует ли извлекать файл.


```C++
SPFILENOTIFY_FILEINCABINET
  Param1 = (UINT) FileInCabinetInfo;
  Param2 = (UINT) CabinetFile;
            
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Параметр1* 
</dt> <dd>

Указатель на [**файл \_ в \_ \_ информационной структуре CAB**](/windows/desktop/api/Setupapi/ns-setupapi-file_in_cabinet_info_a) -файла, содержащий сведения о файле из CAB-файла.

</dd> <dt>

*Param2* 
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая содержит имя файла CAB-файла.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Подпрограммы обратного вызова должны возвращать одно из следующих действий.



| Код возврата                                                                                 | Описание                                  |
|---------------------------------------------------------------------------------------------|----------------------------------------------|
| <dl> <dt>**ФИЛЕОП \_ пропустить**</dt> </dl> | Не извлекайте файл, пропустите его.<br/> |
| <dl> <dt>**ФИЛЕОП \_ доит**</dt> </dl> | Извлеките файл.<br/>                 |



 

Если подпрограммы обратного вызова возвращают ФИЛЕОП \_ доит, то имя, используемое для извлеченного файла, должно быть указано в **фуллтаржетнаме** члене [**файла \_ в структуре \_ \_ сведений о CAB-файле**](/windows/desktop/api/Setupapi/ns-setupapi-file_in_cabinet_info_a) , переданном в подпрограммы в параметре *param1*.

> [!Note]  
> Нет подпрограммы обратного вызова CAB-файла по умолчанию. Приложение установки должно предоставить подпрограммы обратного вызова для управления уведомлениями, отправленными [**сетупитератекабинет**](/windows/desktop/api/Setupapi/nf-setupapi-setupiteratecabineta).
