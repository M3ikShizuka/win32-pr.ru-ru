# <a name="spfilenotify_startregistration-message"></a>\_Сообщение спфиленотифи стартрегистратион

При использовании директивы **регистердллс** INF для самостоятельной регистрации библиотек DLL вызывающие объекты [**сетупинсталлфроминфсектион**](/windows/desktop/api/Setupapi/nf-setupapi-setupinstallfrominfsectiona) могут получать уведомления о каждом файле при регистрации или отмене регистрации. Чтобы отправить уведомление **спфиленотифи \_ стартрегистратион** в подпрограммы обратного вызова один раз перед регистрацией файла, включите \_ \_ в параметр " *flags* " в **сетупинсталлфроминфсектион**. Чтобы отправить уведомление об отмене регистрации, включите \_ регистеркаллбаккаваре и регуляторы \_ унрегсвр в параметре *flags* .

Подпрограммы обратного вызова, заданные параметром *мсгхандлер* параметра [**сетупинсталлфроминфсектион**](/windows/desktop/api/Setupapi/nf-setupapi-setupinstallfrominfsectiona) , должны быть [ \_ \_ обратным вызовом типа PSP File](/windows/win32/api/setupapi/nc-setupapi-psp_file_callback_a). Задайте для параметра *контекста* тот же *контекст* , который указан в **сетупинсталлфроминфсектион**. Задайте для параметра *уведомления* значение **спфиленотифи \_ стартрегистратион**.


```C++
SPFILENOTIFY_STARTREGISTRATION
  Param1 = (UINT_PTR) pointer to file information;
  Param2 = (UINT_PTR) file registration or unregistration;
            
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Параметр1* 
</dt> <dd>

Указатель на структуру [**\_ \_ \_ состояния управления регистром SP**](/windows/desktop/api/Setupapi/ns-setupapi-sp_register_control_statusa) , содержащую сведения о регистрируемом или незарегистрированном файле. Элементу **кбсизе** должен быть присвоен размер структуры. Элементу **filename** следует присвоить полный путь к регистрируемому файлу. **Win32Error** не используется, и для него следует задать значение No \_ Error. **Фаилурекоде** не используется, и для него должно быть задано значение Спрег \_ Success.

</dd> <dt>

*Param2* 
</dt> <dd>

Если файл регистрируется, для *Param2* должен быть задан указатель на ненулевое значение. При отмене регистрации файла *Param2* должен быть установлен в указатель на нуль.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

После получения уведомления функция обратного вызова может вернуть одно из следующих значений.



| Код возврата                                                                                  | Описание                                                                                        |
|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| <dl> <dt>**прервать ФИЛЕОП \_**</dt> </dl> | Не регистрировать и не регистрировать файл, а также прерывать обработку раздела INF.<br/>             |
| <dl> <dt>**ФИЛЕОП \_ доит**</dt> </dl>  | Зарегистрируйте или отмените регистрацию файла и продолжайте обработку раздела INF.<br/>                |
| <dl> <dt>**\_Пропуск файла**</dt> </dl>    | Пропустить регистрацию или отменить регистрацию файла, но продолжить обработку раздела INF<br/> |
