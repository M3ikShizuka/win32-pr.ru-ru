# <a name="spfilenotify_needmedia-message"></a>\_Сообщение спфиленотифи нидмедиа

Уведомление **спфиленотифи \_ нидмедиа** отправляется в подпрограммы обратного вызова, когда требуется новый носитель или новый CAB-файл.


```C++
SPFILENOTIFY_NEEDMEDIA
  Param1 = (UINT) SourceMediaInfo;
  Param2 = (UINT) NewPathInfo;
            
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Параметр1* 
</dt> <dd>

Указатель на [**исходную структуру \_ носителя**](/windows/desktop/api/Setupapi/ns-setupapi-source_media_a) .

</dd> <dt>

*Param2* 
</dt> <dd>

Указатель на массив символов для хранения новых сведений о пути, предоставленных пользователем. Размер буфера — это массив **TCHAR** из элементов максимального \_ пути. Сведения о пути, возвращаемые приложением установки, не должны превышать этот размер.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Подпрограммы обратного вызова должны возвращать один из следующих элементов.



| Код возврата                                                                                    | Описание                                                                                                                                                                                                             |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**ФИЛЕОП \_ NewPath**</dt> </dl> | Носитель существует, а запрошенный файл доступен по пути, указанному в буфере, на который указывает *Param2*.<br/>                                                                                         |
| <dl> <dt>**ФИЛЕОП \_ пропустить**</dt> </dl>    | Пропустить запрошенный файл<br/>                                                                                                                                                                                      |
| <dl> <dt>**прервать ФИЛЕОП \_**</dt> </dl>   | Прервать обработку очереди. Функция [**сетупкоммитфилекуеуе**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea) возвращает **значение false**. GetLastError Возвращает расширенные сведения об ошибке, такие как ошибка \_ отмены, если пользователь отменил действие.<br/> |
| <dl> <dt>**ФИЛЕОП — ДОИТ**</dt> </dl>     | Носитель доступен.<br/>                                                                                                                                                                                      |
