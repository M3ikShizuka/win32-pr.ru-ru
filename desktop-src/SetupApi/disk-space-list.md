# <a name="disk-space-list"></a>Список Disk-Space

API установки предоставляет функциональные возможности списка дискового пространства. Список дискового пространства представляет собой список файловых операций, которые можно запросить, чтобы определить объем дискового пространства, необходимый для этих операций.

В следующих разделах подробно описаны списки дискового пространства, показано, как они вычисляют необходимое пространство и как использовать их в приложении установки:

-   [Использование списка Disk-Space](using-the-disk-space-list.md)
-   [Справочник по списку дискового пространства](disk-space-list-reference.md)
