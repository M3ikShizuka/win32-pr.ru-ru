# <a name="using-cabinet-files"></a>Использование CAB-файлов

Функции установки обработают ящики внутренним образом. Чтобы явно перечислить и извлечь файлы из CAB-файла, можно вызвать функцию [**сетупитератекабинет**](/windows/desktop/api/Setupapi/nf-setupapi-setupiteratecabineta) .

В следующем разделе описывается внутренняя обработка CAB-файла для функций установки и использование [**сетупитератекабинет**](/windows/desktop/api/Setupapi/nf-setupapi-setupiteratecabineta). Также обсуждаются уведомления, отправленные **сетупитератекабинет** , и необходимый формат подпрограммы обратного вызова CAB-файла для обработки этих уведомлений.

-   [Извлечение файлов из ящиков](extracting-files-from-cabinets.md)
-   [Создание подпрограммы обратного вызова CAB-файла](creating-a-cabinet-callback-routine.md)
