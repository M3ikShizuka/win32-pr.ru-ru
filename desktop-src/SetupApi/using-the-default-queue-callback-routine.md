# <a name="using-the-default-queue-callback-routine"></a>Использование подпрограммы обратного вызова очереди по умолчанию

Подпрограммы обратного вызова очереди по умолчанию можно указать для обработки уведомлений, отправляемых [**сетупкоммитфилекуеуе**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea), или вызывать с помощью пользовательской подпрограммы обратного вызова, которая основана на функциональных возможностях подпрограммы обратного вызова очереди по умолчанию.

В следующих разделах описывается инициализация и завершение контекста, используемого подпрограммыми обратного вызова очереди по умолчанию. В разделах также описывается использование подпрограммы обратного вызова очереди по умолчанию с [**сетупкоммитфилекуеуе**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea) или пользовательской подпрограммы обратного вызова.

-   [Инициализация и завершение контекста обратного вызова](initializing-and-terminating-the-callback-context.md)
-   [Вызов подпрограммы обратного вызова очереди по умолчанию](calling-the-default-queue-callback-routine.md)
