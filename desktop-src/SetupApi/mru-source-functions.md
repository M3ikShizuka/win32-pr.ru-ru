# <a name="mru-source-functions"></a>Исходные функции MRU

Следующие функции используются с списками исходных файлов MRU.



| Функция                                                                 | Описание                                                                                       |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**сетупаддтосаурцелист**](/windows/desktop/api/Setupapi/nf-setupapi-setupaddtosourcelista)                     | Добавляет запись в исходный список.                                                                   |
| [**сетупканцелтемпорарисаурцелист**](/windows/desktop/api/Setupapi/nf-setupapi-setupcanceltemporarysourcelist) | Отмена любого временного списка или поведения без просмотра и восстановление стандартного поведения списка.        |
| [**сетупфрисаурцелист**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista)                       | Освобождает ресурсы, выделенные для исходного списка.                                                       |
| [**сетупкуерисаурцелист**](/windows/desktop/api/Setupapi/nf-setupapi-setupquerysourcelista)                     | Запрашивает текущий список источников установки.                                                 |
| [**сетупремовефромсаурцелист**](/windows/desktop/api/Setupapi/nf-setupapi-setupremovefromsourcelista)           | Удаляет запись из списка источников установки.                                                |
| [**сетупсетсаурцелист**](/windows/desktop/api/Setupapi/nf-setupapi-setupsetsourcelista)                         | Задает список источников установки в списке MRU, список MRU пользователя или временный список. |
