# <a name="inf-sourcedisksnames-and-sourcedisksfiles-sections-example"></a>Примеры разделов Саурцедискснамес и Саурцедисксфилес в INF

Раздел **САУРЦЕДИСКСНАМЕС** INF-файла определяет диски, содержащие исходные файлы, которые устанавливаются. Раздел **саурцедисксфилес** определяет исходные файлы и исходные диски, содержащие их. обратите внимание, что платформы MIPS, Alpha и PPC не поддерживаются Windows 2000 или Windows XP.

начиная с Windows XP, в разделе **саурцедискснамес** может быть указан тег и cab-файл. Например, следующий раздел **саурцедискснамес** может использоваться с съемными или стационарными носителями. При использовании съемного носителя раздел пример проверяет наличие носителя, сначала проверяя наличие тега. Если используется фиксированный носитель, в разделе пример для проверки наличия носителя сначала проверяется наличие CAB-файла. После проверки наличия CAB-файла система пытается скопировать файлы из CAB-файла и запрашивает файлы, которые не удалось скопировать.

``` syntax
[SourceDisksNames]
1="Dajava" , "Dajava.cab",,, 0x10,"Dajava.tag"
```

дополнительные сведения о **саурцедискснамес** или **саурцедисксфилес** см. в разделах «общие рекомендации по inf-файлу» и «разделы и директивы inf-файла» пакета средств разработки драйверов Microsoft Windows 2000.
