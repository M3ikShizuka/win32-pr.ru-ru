# <a name="spfilenotify_endregistration-message"></a>\_Сообщение спфиленотифи ендрегистратион

При использовании директивы **регистердллс** INF для самостоятельной регистрации библиотек DLL вызывающие объекты [**сетупинсталлфроминфсектион**](/windows/desktop/api/Setupapi/nf-setupapi-setupinstallfrominfsectiona) могут получать уведомления о каждом файле при регистрации или отмене регистрации. Чтобы отправить **спфиленотифи \_ ендрегистратион** уведомление в подпрограммы обратного вызова после регистрации или отмены регистрации файла, включите \_ \_ в параметр *flags* в **сетупинсталлфроминфсектион** значение Spin регистеркаллбаккаваре и Spin регсвр. Чтобы отправить уведомление об отмене регистрации, включите \_ регистеркаллбаккаваре и регуляторы \_ унрегсвр в параметре *flags* .

Подпрограммы обратного вызова, заданные параметром *мсгхандлер* параметра [**сетупинсталлфроминфсектион**](/windows/desktop/api/Setupapi/nf-setupapi-setupinstallfrominfsectiona) , должны быть [ \_ \_ обратным вызовом типа PSP File](/windows/win32/api/setupapi/nc-setupapi-psp_file_callback_a). Задайте для параметра *контекста* тот же *контекст* , который указан в **сетупинсталлфроминфсектион**. Задайте для параметра *уведомления* значение **спфиленотифи \_ ендрегистратион**.


```C++
SPFILENOTIFY_ENDREGISTRATION
  Param1 = (UINT_PTR) pointer to file information;
  Param2 = (UINT_PTR) file registration or unregistration;
            
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Параметр1* 
</dt> <dd>

Указатель на структуру [**\_ \_ \_ состояния управления регистром SP**](/windows/desktop/api/Setupapi/ns-setupapi-sp_register_control_statusa) , содержащую сведения о регистрируемом или незарегистрированном файле. Элементу **кбсизе** должен быть присвоен размер структуры. Для **filename** необходимо указать полный путь к регистрируемому файлу. Для **Win32Error** должен быть задан [код системной ошибки](/windows/desktop/Debug/system-error-codes) , указывающий код расширенной ошибки. Для **фаилурекоде** следует задать один из допустимых кодов сбоя, указывающий результат регистрации. Допустимые коды ошибок см. в разделе [**\_ \_ \_ состояние управления регистром SP**](/windows/desktop/api/Setupapi/ns-setupapi-sp_register_control_statusa).

</dd> <dt>

*Param2* 
</dt> <dd>

Если файл регистрируется, для *Param2* должен быть задан указатель на ненулевое значение. При отмене регистрации файла *Param2* должен быть установлен в указатель на нуль.

</dd> </dl>
