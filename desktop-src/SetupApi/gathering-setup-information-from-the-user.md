# <a name="gathering-setup-information-from-the-user"></a>Сбор сведений о программе установки от пользователя

Настоятельно рекомендуется, чтобы приложение установки включало пользовательский интерфейс, который получает все необходимые сведения от пользователя. Эта функция не предоставляется функциями установки и должна быть реализована в приложении установки, обычно в виде мастера установки.
