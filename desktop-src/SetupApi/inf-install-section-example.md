# <a name="inf-install-section-example"></a>Пример раздела установки INF-файла

Раздел **Установка** INF-файла определяет шаги процедуры установки. Каждая строка раздела **установки** состоит из двух частей. В левой части знака равенства (=) находится ключ. В правой части — это список из одного или нескольких заголовков разделов. Ключ задает тип разделов, которые перечислены справа. Описание формата этого раздела см. в разделе "разделы и директивы INF-файлов" пакета средств разработки драйверов для Microsoft Windows 2000.

Ниже приведен пример раздела **Install** .

``` syntax
[MyInstallSection]
Copyfiles=DataFiles, ProgramFiles
Delfiles=OldFiles
UpdateInis=NewIniInfo
AddReg=NewRegistryInfo, MoreNewRegistryInfo
DelReg=OldRegistryInfo, MoreOldRegistryInfo
```

В этом примере для ключа **copyfiles** заданы значения "Files" и "ProgramFiles". Этот параметр задает два раздела **копий файлов** в INF-файле, которые содержат имена исходных файлов, используемых операциями копирования. Место назначения для скопированных файлов задается в разделе **дестинатиондирс** в INF-файле.

Ключу **делфилес** присваивается значение «олдфилес». Здесь указывается раздел **Удаление файлов** , содержащий сведения, относящиеся к операциям удаления файлов.

Ключ **упдатеинис** указывает разделы **Update ini-файла** , содержащие сведения об обновлении записей в ini-файле.

Ключи **аддрег** и **делрег** указывают разделы реестра **Add** и **Delete** , содержащие сведения о добавлении или удалении данных реестра.
