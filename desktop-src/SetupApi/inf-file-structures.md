# <a name="inf-file-structures"></a>Структуры INF-файлов

Для файлов INF используются следующие структуры.



| Структура                                              | Описание                        |
|--------------------------------------------------------|------------------------------------|
| [**инфконтекст**](/windows/desktop/api/Setupapi/ns-setupapi-infcontext)                   | Позиционированный контекст в INF-файле. |
| [**\_сведения об INF-файле SP \_**](/windows/desktop/api/Setupapi/ns-setupapi-sp_inf_information) | Сведения о INF-файле.    |



 

> [!Note]  
> Структура ИНФКОНТЕКСТ используется внутренне функциями программы установки и не должна изменяться или ссылаться на них в приложениях установки. Он содержится здесь только в информационных целях.
