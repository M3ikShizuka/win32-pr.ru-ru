# <a name="disk-prompting-and-error-handling-functions"></a>Функции запросов к диску и обработки ошибок

Следующие функции предоставляют запросы к дискам и пользовательский интерфейс обработки ошибок.



| Функция                                         | Описание                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------|
| [**сетупкоперрор**](/windows/desktop/api/Setupapi/nf-setupapi-setupcopyerrora)         | Создает диалоговое окно, информирующее пользователя об ошибке копирования.                          |
| [**сетупделетиррор**](/windows/desktop/api/Setupapi/nf-setupapi-setupdeleteerrora)     | Создает диалоговое окно, информирующее пользователя об ошибке удаления.                      |
| [**сетуппромптфордиск**](/windows/desktop/api/Setupapi/nf-setupapi-setuppromptfordiska) | Создает диалоговое окно, предлагающее пользователю указать диск или расположение исходного файла мультимедиа. |
| [**сетупренамиррор**](/windows/desktop/api/Setupapi/nf-setupapi-setuprenameerrora)     | Создает диалоговое окно, информирующее пользователя об ошибке переименования.                        |
