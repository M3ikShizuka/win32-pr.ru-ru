# <a name="file-log-functions"></a>Функции журнала файлов

В журналах файлов используются следующие функции.



| Функция                                                   | Описание                                 |
|------------------------------------------------------------|---------------------------------------------|
| [**сетупинитиализефилелог**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitializefileloga)   | Инициализирует файл журнала для использования.             |
| [**SetupLogFile**](/windows/desktop/api/Setupapi/nf-setupapi-setuplogfilea)                       | Добавляет запись в файл журнала.              |
| [**сетупкуерифилелог**](/windows/desktop/api/Setupapi/nf-setupapi-setupqueryfileloga)             | Извлекает сведения из файла журнала.      |
| [**сетупремовефилеложентри**](/windows/desktop/api/Setupapi/nf-setupapi-setupremovefilelogentrya) | Удаляет запись из файла журнала.           |
| [**сетуптерминатефилелог**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog)     | Освобождает ресурсы, выделенные для файла журнала. |
