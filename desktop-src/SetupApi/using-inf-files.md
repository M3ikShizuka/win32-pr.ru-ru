# <a name="using-inf-files"></a>Использование INF-файлов

После создания INF-файла для процесса установки функции установки могут получить доступ к данным, которые будут использоваться при копировании, удалении или переименовании файлов и обновлении записей ИНИС и реестра.

-   [Открытие и закрытие INF-файла](opening-and-closing-an-inf-file.md)
-   [Получение сведений из INF-файла](retrieving-information-from-an-inf-file.md)
-   [Установка из INF-файла](installing-from-an-inf-file.md)
-   [Указание дескриптора безопасности из INF-файла](specifying-a-security-descriptor-from-an-inf-file.md)
