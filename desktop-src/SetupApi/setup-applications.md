# <a name="setup-applications"></a>Настройка приложений

Как правило, перед созданием приложения установки создается *INF-файл* . INF-файл — это текстовый файл, содержащий сведения, используемые приложением установки во время установки. Примеры сведений, которые могут храниться в INF-файле, включают в себя изменения в INI-файлах и реестре, имена файлов и расположения исходного файла на исходном носителе.

После создания INF-файла вы пишете исходный код для приложения установки. Вы можете использовать функции программы установки в приложении установки для доступа к данным в INF-файле и выполнения операций установки.

В следующих разделах описывается использование INF-файлов и функции обработки установки.

-   [Сведения о файлах INF](about-inf-files.md)
-   [Использование INF-файлов](using-inf-files.md)
-   [Функции установки INF-файла](inf-file-setup-functions.md)

Дополнительные сведения об использовании функций программы установки в приложении см. в разделе [Создание приложений установки](creating-setup-applications.md).
