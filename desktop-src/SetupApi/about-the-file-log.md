# <a name="about-the-file-log"></a>Сведения о журнале файлов

В журнале файлов записываются сведения о файлах, скопированных в систему во время установки. Его можно использовать для аварийного восстановления или в качестве диагностического средства.

Файл, используемый для установки файлов журнала, может быть системным журналом или любым другим указанным журналом файлов.
