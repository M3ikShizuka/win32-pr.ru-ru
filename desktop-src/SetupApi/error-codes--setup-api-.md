# <a name="error-codes-setup-api"></a>Коды ошибок (API установки)

Следующие коды ошибок относятся к API установки.



| Ошибки разбора INF-файла              | Описание                                                                                                         |
|---------------------------------|---------------------------------------------------------------------------------------------------------------------|
| Ошибка \_ ожидаемое \_ \_ имя раздела  | Ожидалось имя раздела, и оно не найдено.                                                                          |
| Ошибка \_ неправильной \_ \_ строки имени раздела \_ | Недопустимый формат имени раздела. (Например, имя, не заканчивающееся правой скобкой ( \] ). |
| \_ \_ \_ слишком \_ ДЛИННое имя раздела ошибки | Длина имени раздела превышает максимальную длину в \_ сект \_ имени \_ Len.                                               |
| Ошибка \_ общего \_ синтаксиса          | Неверный общий синтаксис.                                                                                    |



 



| Ошибки времени выполнения INF         | Описание                                                                                                                                                                                                                                    |
|----------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ОШИБОЧный \_ \_ \_ Стиль INF   | Тип INF-файла не указан в вызове функции. Например, эта ошибка может возвращаться функцией [**сетупопенаппендинффиле**](/windows/desktop/api/Setupapi/nf-setupapi-setupopenappendinffilea) , если INF-файл предназначен для ранней версии Windows. |
| \_раздел ошибки \_ не \_ найден | Раздел не найден в INF-файле.                                                                                                                                                                                                     |
| \_строка ошибки \_ не \_ найдена    | Строка не найдена в разделе INF.                                                                                                                                                                                                     |
