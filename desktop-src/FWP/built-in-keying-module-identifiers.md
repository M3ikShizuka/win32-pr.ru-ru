# <a name="built-in-keying-module-identifiers"></a>Идентификаторы встроенного модуля создания ключей

Идентификаторы модулей ключей, встроенных в платформу фильтрации Windows (WFP), представляются с помощью идентификатора GUID.

Эти идентификаторы определяются следующим образом.

<dl> <dt>

<span id="FWPM_KEYING_MODULE_IKE"></span><span id="fwpm_keying_module_ike"></span>**\_ \_ IKE модуля ключей \_ фвпм**
</dt> <dd> <dl> <dt>



Модуль ключа протокол IKE (IKE).


</dt> </dl> </dd> <dt>

<span id="FWPM_KEYING_MODULE_IKEV2"></span><span id="fwpm_keying_module_ikev2"></span>**\_Модуль ключей \_ фвпм \_ IKEV2**
</dt> <dd> <dl> <dt>



Модуль ключа протокол IKE версии 2 (IKEv2).

> [!Note]  
> Доступно только в Windows Server 2008 R2 и Windows 7.

 


</dt> </dl> </dd> <dt>

<span id="FWPM_KEYING_MODULE_AUTHIP"></span><span id="fwpm_keying_module_authip"></span>**\_модуль ключей \_ фвпм \_ AUTHIP**
</dt> <dd> <dl> <dt>



Модуль ключа AuthIP.


</dt> </dl> </dd> </dl>
