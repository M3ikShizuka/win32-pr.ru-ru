# <a name="page-space-to-device-space-transformations"></a>Преобразование Page-Space в Device-Space

Преобразование «страница-место в устройство» определяет режим сопоставления для всех выходных данных графики, связанных с определенным контроллером домена. *Режим сопоставления* — это преобразование масштабирования, которое указывает размер единиц, используемых для операций рисования. Режим сопоставления может также выполнять преобразование. В некоторых случаях режим сопоставления изменяет ориентацию оси x и оси y в пространстве устройства. В следующих разделах описываются режимы сопоставления.

-   [Режимы сопоставления и переводы](mapping-modes-and-translations.md)
-   [Стандартные режимы сопоставления](predefined-mapping-modes.md)
-   [Режимы сопоставления, определяемые приложением](application-defined-mapping-modes.md)
