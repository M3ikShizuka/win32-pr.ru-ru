# <a name="pen-attributes"></a>Атрибуты пера

Существует семь атрибутов пера, определяющих тип пера и его характеристики: ширина, стиль, цвет, узор, штриховки, стиль конца и стиль объединения. Как косметические, так и геометрические перья имеют атрибуты ширины, стиля и цвета. Только геометрические перья имеют атрибуты шаблон, штриховки, стиль конца и стиль объединения. Шаблон и необязательный атрибут штриховки обычно связаны с кистью, но также можно использовать с геометрическими перьями.
