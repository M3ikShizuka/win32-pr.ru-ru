# <a name="device-space-to-physical-device-transformation"></a>Преобразование Device-Space Physical-Device

Преобразование "устройство — место для физического устройства" является уникальным в нескольких отношениях. Например, оно ограничено переводом и управляется системой. Единственное назначение этого преобразования заключается в том, чтобы убедиться, что источник пространства устройства сопоставлен с соответствующей точкой физического устройства. Нет ни одной функции для задания этого преобразования, ни функции для получения связанных данных.
