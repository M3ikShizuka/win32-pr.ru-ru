# <a name="painting-regions"></a>Области рисования

Приложение заполняет внутреннюю часть области, используя кисть, выбранную в данный момент в контексте устройства функцией [**паинтргн**](/windows/desktop/api/Wingdi/nf-wingdi-paintrgn) . Эта функция использует текущие режимы заливки многоугольников (чередование и обмотка).
