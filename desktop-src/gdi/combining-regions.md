# <a name="combining-regions"></a>Объединение регионов

Приложение объединяет два региона, вызывая функцию [**комбинергн**](/windows/desktop/api/Wingdi/nf-wingdi-combinergn) . С помощью этой функции приложение может сочетать пересекающиеся части двух регионов, все, кроме пересекающихся частей двух регионов, две исходные области целиком и т. д. Ниже приведены пять значений, определяющих сочетания областей.



| Значение     | Значение                                                                               |
|-----------|---------------------------------------------------------------------------------------|
| РГН \_ и  | Пересекающиеся части двух исходных регионов определяют новый регион.                   |
| \_копирование РГН | Копия первого (из двух исходных регионов) определяет новый регион.               |
| РГН \_ diff | Часть первого региона, который не пересекается со вторым, определяет новый регион. |
| РГН \_ или   | Два исходных региона определяют новый регион.                                         |
| РГН \_ XOR  | Эти части двух исходных регионов, которые не перекрываются, определяют новый регион.      |
