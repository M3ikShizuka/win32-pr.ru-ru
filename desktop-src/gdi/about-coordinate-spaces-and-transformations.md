# <a name="about-coordinate-spaces-and-transformations"></a>Сведения об пространствах координат и преобразованиях

Координатные пространства и преобразования используются в следующих типах приложений:

-   Приложения для настольной публикации (для «масштабирования» частей страницы или для показа смежных страниц в окне).
-   Приложения для автоматизированного проектирования (CAD) (для поворота объектов, масштабирования чертежей или создания перспективных представлений).
-   Приложения электронных таблиц (для перемещения и изменения размера диаграмм).
