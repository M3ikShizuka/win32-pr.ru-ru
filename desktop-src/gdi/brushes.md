# <a name="brushes-windows-gdi"></a>Кисти (Windows GDI)

*Кисть* — это графическое средство, используемое приложениями для рисования внутренней части многоугольников, эллипсов и контуров. Графические приложения используют кисти для рисования фигур; приложения обработки текста используют кисти для рисования правил; приложения для автоматизированного проектирования (CAD) используют кисти для рисования внутренних областей представлений в разных разделах. приложения электронных таблиц используют кисти для рисования фрагментов круговых диаграмм и линий на линейчатых диаграммах.
