# <a name="about-brushes"></a>О кистях

Существует два типа кистей: логическое и физическое. *Логическая кисть* — это описание идеального точечного рисунка, используемого приложением для рисования фигур. *Физическая кисть* — это фактический точечный рисунок, который создается драйвером устройства на основе определения логической кисти приложения. Дополнительные сведения о растровых изображениях см. в разделе [точечные рисунки](bitmaps.md).

Когда приложение вызывает одну из функций, которые создают кисть, он получает дескриптор, идентифицирующий логическую кисть. Когда приложение передает этот дескриптор функции [**SelectObject**](/windows/desktop/api/Wingdi/nf-wingdi-selectobject) , драйвер устройства для соответствующего дисплея или принтера создает физическую кисть.
