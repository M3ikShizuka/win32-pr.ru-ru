# <a name="about-pies"></a>О кругах

*Круг* — это область, ограниченная пересечением кривой эллипса и двумя радиальными. На следующем рисунке показана круговая диаграмма, изображенная с помощью функции [**круговой диаграммы**](/windows/desktop/api/Wingdi/nf-wingdi-pie) .

![Иллюстрация с эллипсом с затененной круговой диаграммой](images/csfsh-03.png)

При вызове [**круговой диаграммы**](/windows/win32/api/wingdi/nf-wingdi-pie)приложение предоставляет координаты верхнего левого и нижнего правого угла ограничивающего прямоугольника эллипса, а также координаты двух точек, определяющих два радиальных элемента.

Когда система рисует изогнутую часть круговой диаграммы, она использует текущее направление дуги для данного контекста устройства. Направление дуги по умолчанию — против часовой стрелки. Приложение может сбросить направление дуги, вызвав функцию [**сетаркдиректион**](/windows/desktop/api/Wingdi/nf-wingdi-setarcdirection) .
