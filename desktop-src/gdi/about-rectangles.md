# <a name="about-rectangles"></a>О прямоугольниках

Прямоугольники используются для области обрезки курсора, недопустимой части клиентской области, области для отображения форматированного текста или области прокрутки. Приложения также могут использовать прямоугольники для заполнения, фрейма или инвертирования части клиентской области определенной кистью, а также для получения координат окна или клиентской области окна.

-   [Координаты прямоугольника](rectangle-coordinates.md)
-   [Операции с прямоугольниками](rectangle-operations.md)
