# <a name="smooth-shading"></a>Плавная Заливка

*Гладкая заливка* — это метод заливки области цветом градиента. Включая сведения о цвете вместе с границами примитива рисования, указывает цветовой градиент. GDI линейно интерполирует цвет внутри примитива, переданного в конечные точки цвета. Сведения о цвете и вершине включаются в сведения о положении в структуре [**тривертекс**](/windows/desktop/api/Wingdi/ns-wingdi-trivertex) .

Используйте функцию [**градиентфилл**](/windows/desktop/api/WinGdi/nf-wingdi-gradientfill) для заполнения структуры треугольника или прямоугольника. Чтобы заполнить треугольник с помощью плавной заливки, вызовите **градиентфилл** с тремя конечными точками треугольника. Чтобы заполнить прямоугольник с помощью плавной заливки, вызовите **градиентфилл** с координатами верхнего левого и нижнего правого прямоугольника. **Градиентфилл** ссылается на структуры [**тривертекс**](/windows/desktop/api/Wingdi/ns-wingdi-trivertex), [**\_ прямоугольника градиента**](/windows/desktop/api/Wingdi/ns-wingdi-gradient_rect)и [**градиентного \_ треугольника**](/windows/desktop/api/Wingdi/ns-wingdi-gradient_triangle) .

Пример см. в разделе [Рисование затененного треугольника](drawing-a-shaded-triangle.md).
