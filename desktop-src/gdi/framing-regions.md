# <a name="framing-regions"></a>Регионы кадрирования

Приложение рисует границу вокруг области, вызывая функцию [**фрамергн**](/windows/desktop/api/Wingdi/nf-wingdi-framergn) и задавая ширину границы и шаблон кисти, используемые системой при рисовании рамки.
