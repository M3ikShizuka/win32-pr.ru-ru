# <a name="formatting-text-windows-gdi"></a>форматирование текста (Windows GDI)

Функции форматирования можно разделить на три категории: те, которые извлекают или устанавливают [атрибуты форматирования текста](text-formatting-attributes.md) для контекста устройства, которые извлекают [ширину символов](character-widths.md)и те, которые получают [ширину и высоту строки](string-widths-and-heights.md).
