# <a name="palette-animation"></a>Анимация палитры

Анимация палитры — это метод моделирования движения путем быстрого изменения цветов выбранных записей в цветовой палитре. Приложение может выполнять анимацию палитры, создавая логическую палитру, содержащую "зарезервированные" записи, а затем используя функцию [**аниматепалетте**](/windows/desktop/api/Wingdi/nf-wingdi-animatepalette) для изменения цветов в этих зарезервированных записях.

Приложение создает зарезервированный элемент в логической палитре, устанавливая для элемента **пефлагс** структуры [**палеттинтри**](/previous-versions//dd162769(v=vs.85)) \_ флаг зарезервированного компьютера. После выбора и реализации этой логической палитры приложение может вызвать функцию [**аниматепалетте**](/windows/desktop/api/Wingdi/nf-wingdi-animatepalette) , чтобы изменить одну или несколько зарезервированных записей. Если данная палитра связана с активным окном, система немедленно обновляет цвета на экране.
