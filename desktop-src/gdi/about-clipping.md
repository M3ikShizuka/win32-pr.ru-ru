# <a name="about-clipping"></a>О обрезке

Обрезка используется приложениями различными способами. Текстовые редакторы и приложения для работы с электронными таблицами зазажимируют ввод с клавиатуры, чтобы они не отображались на полях страницы или электронной таблицы. Графические изображения, разработанные на основе компьютеров (CAD) и графические приложения обрисуются, чтобы не перезаписывать границы рисунка или рисунка.

*Область обрезки* — это область с краями, которые являются прямыми линиями или кривыми. *Контур обрезки* — это область с краями, которые являются прямыми линиями, кривыми Безье или комбинациями обоих. Дополнительные сведения о регионах см. в разделе [регионы](regions.md). Дополнительные сведения о путях см. в разделе [paths](paths.md).

В этом обзоре рассматриваются следующие темы:

-   [Вырезанные области](clipping-regions.md)
-   [Пути обрезки](clip-paths.md)
