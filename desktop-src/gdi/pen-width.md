# <a name="pen-width"></a>Ширина пера

Атрибут width задает ширину косметических перьев в единицах устройства. Однако при использовании с геометрическим пером он указывает ширину пера в логических единицах. Дополнительные сведения об единицах устройств см. в разделе [координатные пространства и преобразования](coordinate-spaces-and-transformations.md).

В настоящее время система ограничивает ширину косметических перьев одним пикселем. Однако будущие версии могут удалить это ограничение.
