# <a name="character-widths"></a>Ширина символов

Приложениям необходимо получить данные ширины символов, когда они выполняют такие задачи, как подгонка строк текста к ширине страницы или столбца. Существует четыре функции, которые приложение может использовать для извлечения данных ширины символов. Две из этих функций извлекают ширину символов и две из этих функций для получения фактических данных по ширине символов.

Приложение может использовать функции [GetCharWidth32](/windows/desktop/api/Wingdi/nf-wingdi-getcharwidth32a) и [жетчарвидсфлоат](/windows/desktop/api/Wingdi/nf-wingdi-getcharwidthfloata) для получения предварительной ширины отдельных символов или символов в текстовой строке. Ширина — это расстояние, на которое курсор на дисплее или печатающем головке на принтере должен быть перемещен перед печатью следующего символа в текстовой строке. Функция [**GetCharWidth32**](/windows/win32/api/wingdi/nf-wingdi-getcharwidth32a) Возвращает ширину в виде целочисленного значения. Если требуется большая точность, приложение может использовать функцию [**жетчарвидсфлоат**](/windows/win32/api/wingdi/nf-wingdi-getcharwidthfloata) для получения дробных значений по расширенной ширине.

Приложение может получать фактические данные по ширине символов с помощью функций [жетчарабквидсс](/windows/desktop/api/Wingdi/nf-wingdi-getcharabcwidthsa) и [**жетчарабквидссфлоат**](/windows/desktop/api/Wingdi/nf-wingdi-getcharabcwidthsfloata) . Функция **жетчарабквидссфлоат** работает со всеми шрифтами. Функция [**жетчарабквидсс**](/windows/win32/api/wingdi/nf-wingdi-getcharabcwidthsa) работает только с шрифтами TrueType и OpenType. Дополнительные сведения о шрифтах TrueType и OpenType см. в разделе [растровые, векторные, шрифты TrueType и OpenType](raster--vector--truetype--and-opentype-fonts.md).

Интервал — это ширина, добавляемая к текущей позиции перед тем, как поместить символ. Интервал B — это ширина самого символа. Пробел в C является пробелом справа от символа. Общая ширина аванса определяется путем вычисления суммы значений A + B + C. Ячейка символов представляет собой воображаемый прямоугольник, который окружает каждый символ или символ в шрифте. Поскольку символы могут зависнуть или зависать в ячейке символов, один или оба шага A и C могут быть отрицательным числом.
