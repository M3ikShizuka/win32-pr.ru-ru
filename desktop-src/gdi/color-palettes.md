# <a name="color-palettes-windows-gdi"></a>цветовые палитры (Windows GDI)

Цветовая палитра — это массив, который содержит значения цвета, определяющие цвета, которые в настоящее время можно отображать или рисовать на устройстве вывода. Цветовые палитры используются устройствами, которые способны создавать много цветов, но они могут только отображать или нарисовать подмножество из них в любое заданное время. Для таких устройств система поддерживает *системную палитру* для мониторинга текущих цветов устройства и управления ими. Приложения не имеют прямого доступа к системной палитре. Вместо этого система связывает палитру по умолчанию с каждым контекстом устройства. Приложения могут использовать цвета в палитре по умолчанию или определять собственные цвета, создавая *логические палитры* и связывая их с отдельными контекстами устройств.

Приложение может определить, поддерживает ли устройство цветовые палитры, проверив \_ бит палитры RC в значении растеркапс, возвращаемом функцией [**жетдевицекапс**](/windows/desktop/api/Wingdi/nf-wingdi-getdevicecaps) .
