# <a name="device-contexts"></a>Контексты устройств

*Контекст устройства* — это структура, определяющая набор графических объектов и связанных с ними атрибутов, а также графические режимы, влияющие на выходные данные. *Графические объекты* включают в себя перо для рисования линий, кисть для рисования и заливки, точечный рисунок для копирования или прокрутки фрагментов экрана, палитру для определения набора доступных цветов, региона для обрезки и других операций, а также траектории для операций рисования и рисования. Оставшаяся часть этого раздела состоит из трех следующих областей.

-   [О контекстах устройств](about-device-contexts.md)
-   [Использование функций контекста устройства](using-the-device-context-functions.md)
-   [Справочник по контексту устройства](device-context-reference.md)
