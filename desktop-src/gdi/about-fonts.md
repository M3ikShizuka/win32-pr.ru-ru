# <a name="about-fonts"></a>О шрифтах

Для людей, обученных механикой композиции документ или знакомой со стандартным оформлением, некоторые типографские термины, используемые в этом обзоре, могут быть ненеобычными. Большинство отличий между стандартным оформлением и текстом в Microsoft Windows отражает изменения в технологии. Исходные типографские термины основаны на композиции с использованием горячего металла, в то время как используемые здесь термины, которые отображаются в качестве имен членов для структур вывода шрифта и текста, отражают новую технологию на основе вывода лазерного принтера и композиции, выполняемой на персональном компьютере с помощью программного обеспечения для публикации настольных систем.
