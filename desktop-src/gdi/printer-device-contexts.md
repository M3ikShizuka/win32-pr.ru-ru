# <a name="printer-device-contexts-windows-gdi"></a>Контексты печатающих устройств (Windows GDI)

Контроллер домена может использоваться при печати на матричном принтере, на принтере с рукописным вводом, на лазерном принтере или плоттере. Приложение создает контроллер домена, вызывая функцию [**креатедк**](/windows/desktop/api/Wingdi/nf-wingdi-createdca) и предоставляя соответствующие аргументы (имя драйвера принтера, имя принтера, имя файла или устройства для физического выходного носителя и другие данные инициализации). После завершения печати приложения он удаляет контроллер домена, вызывая функцию [**делетедк**](/windows/desktop/api/Wingdi/nf-wingdi-deletedc) . Приложение должно удалять (а не выпустить) контроллер домена принтера; Функция [**релеаседк**](/windows/desktop/api/Winuser/nf-winuser-releasedc) завершается ошибкой, когда приложение пытается использовать его для выпуска контроллера домена.

Дополнительные сведения см. в разделе [вывод принтера](../printdocs/printer-output.md).
