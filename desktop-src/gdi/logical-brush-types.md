# <a name="logical-brush-types"></a>Логические типы кистей

Существует четыре типа логических кистей: « [сплошная](solid-brush.md)», « [Склад](stock-brush.md)», « [штриховки](hatch-brush.md)» и « [узор](pattern-brush.md)». Эти кисти показаны на следующем рисунке.

![Иллюстрация, показывающая четыре круга, каждый из которых заполняется другим типом кисти: сплошная, складская, штриховка и шаблон](images/csbru-02.png)

У каждого типа акции и штриховки есть несколько предопределенных кистей.

Функция [**креатебрушиндирект**](/windows/desktop/api/Wingdi/nf-wingdi-createbrushindirect) создает логическую кисть с указанным стилем, цветом и шаблоном.
