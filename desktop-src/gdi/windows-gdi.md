# <a name="windows-gdi"></a>Windows GDI

## <a name="purpose"></a>Назначение

Интерфейс графических устройств (GDI) Microsoft Windows позволяет приложениям использовать графику и форматированный текст как на экране видео, так и на принтере. Приложения на основе Windows не обращаются к графическому оборудованию напрямую. Вместо этого GDI взаимодействует с драйверами устройств от имени приложений.

## <a name="where-applicable"></a>Где применимо

GDI можно использовать во всех приложениях на основе Windows.

## <a name="developer-audience"></a>Аудитория разработчиков

Этот API предназначен для использования программистами C/C++. Знание [архитектуры, управляемой сообщениями](../learnwin32/window-messages.md) Windows, является обязательным.

## <a name="run-time-requirements"></a>Требования к среде выполнения
