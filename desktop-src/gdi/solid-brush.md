# <a name="solid-brush"></a>Сплошная кисть

*Сплошная кисть* — это логическая кисть, которая содержит 64 пикселов того же цвета. Приложение может создать сплошную логическую кисть, вызвав функцию [**креатесолидбруш**](/windows/desktop/api/Wingdi/nf-wingdi-createsolidbrush) , указав цвет требуемой кисти. После создания сплошной кисти приложение может выбрать его в контексте устройства и использовать для рисования заполненных фигур.
