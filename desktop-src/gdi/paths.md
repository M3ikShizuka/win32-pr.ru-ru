# <a name="paths-windows-gdi"></a>пути (Windows GDI)

*Контур* представляет собой одну или несколько фигур (или фигур), которые заполнены, обрабатываются или заполняются. Пути используются в приложениях рисования и рисования. Приложения для автоматизированного проектирования (CAD) используют пути для создания уникальных областей обрезки, для рисования контуров нестандартных фигур и для заполнения внутренних областей нестандартных фигур. Неравномерные фигуры — это фигура, состоящая из кривых Безье и прямых линий. (Обычные фигуры включают многоточие, круги, прямоугольники и многоугольники.)
