# <a name="filled-shapes"></a>Закрашенные фигуры

*Заполненные фигуры* — это геометрические формы, которые обрабатываются с помощью текущего пера и заполняются с помощью текущей кисти. Имеется пять заполненных фигур:

-   Эллипс
-   Хорда
-   Круговая
-   Многоугольник
-   Прямоугольник

В этом обзоре описываются заполненные фигуры.

-   [Использование закрашенных фигур](using-filled-shapes.md)
-   [Заполненная ссылка на фигуру](filled-shape-reference.md)
