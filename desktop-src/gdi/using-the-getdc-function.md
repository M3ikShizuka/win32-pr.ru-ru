# <a name="using-the-getdc-function"></a>Использование функции GetDC

Функция [**GetDC**](/windows/desktop/api/Winuser/nf-winuser-getdc) используется для выполнения рисования, которое должно выполняться мгновенно, а не при обработке сообщения [**WM \_ Paint**](wm-paint.md) . Такая прорисовка обычно в ответ на какое-либо действие пользователем, например создание выделения или рисование с помощью мыши. В таких случаях пользователь должен получить мгновенный отзыв и не должен вынужден прерывать выбор или прорисовку, чтобы приложение отображало результат. В следующих разделах показано, как использовать GetDC для рисования в окне:

-   [Рисование с помощью мыши](drawing-with-the-mouse.md)
-   [Рисование с интервалами времени](drawing-at-timed-intervals.md)
