# <a name="stock-brush"></a>Стандартная кисть

В интерфейсе графических устройств (GDI) поддерживаются семь стандартных логических стандартных объектов. Существует также 21 стандартный логический запасной кисти, поддерживаемый интерфейсом управления окнами (пользователь).

На следующем рисунке показаны прямоугольники, нарисованные с помощью семи предопределенных стандартных кистей.

![Иллюстрация, демонстрирующая семь квадратиков: один черный, три оттенка серого и три, которые выглядят пустыми](images/csbru-03.png)

Приложение может получить дескриптор, идентифицирующий одну из семи стандартных кистей, вызвав функцию [**жетстоккобжект**](/windows/desktop/api/Wingdi/nf-wingdi-getstockobject) , указав тип кисти.

21 стандартные кисти, поддерживаемые интерфейсом управления окнами, соответствуют цветам элементов окон, таким как меню, полосы прокрутки и кнопки. Приложение может получить маркер, идентифицирующий одну из этих кистей, вызвав функцию [**жетсисколорбруш**](/windows/desktop/api/Winuser/nf-winuser-getsyscolorbrush) и указав значение системного цвета. Приложение может получить цвет, соответствующий определенному элементу окна, вызвав функцию [**жетсисколор**](/windows/win32/api/winuser/nf-winuser-getsyscolor) . Приложение может задать цвет, соответствующий элементу окна, вызвав функцию [**сетсисколорс**](/windows/win32/api/winuser/nf-winuser-setsyscolors) .
