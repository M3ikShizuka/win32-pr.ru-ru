# <a name="multiple-monitor-considerations-for-older-programs"></a>Рекомендации по нескольким мониторам для старых программ

Как правило, более старые приложения не требуют изменений для работы в нескольких системах мониторинга. В большинстве случаев система выглядит только с одним дисплеем. Системные метрики соответствуют основным приложениям отображения и полноэкранного режима, отображаемым на основном компьютере. Система обрабатывает минимизацию, максимизации, меню и диалоговые окна.

Однако некоторые программы будут иметь проблемы. Некоторые, которые могут иметь проблемы, — это приложения удаленного управления, имеющие прямой доступ к оборудованию, и те, которые установили пакет GDI или видеодрайвер. В таких случаях пользователю может потребоваться отключить монитор за пределами основного монитора.
