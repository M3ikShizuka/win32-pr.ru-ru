# <a name="pen-end-cap"></a>Концевая точка пера

Атрибут End Cap задает форму геометрического пера: Round, квадратного или плоского. На следующем рисунке показаны параллельные линии, рисуемые с помощью каждого типа конца.

![Иллюстрация, демонстрирующая три горизонтальных линии, каждая с другим типом конца](images/cspen-04.png)

Скругленные и квадратные концы выходят за пределы начальной и конечной точек линии, нарисованной с помощью геометрического пера; фиксированная конечная точка не имеет.
