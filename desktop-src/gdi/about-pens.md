# <a name="about-pens"></a>О перьях

Существует два типа пера: косметические и геометрические. *Косметические перья* используются для приложений, которым требуются линии фиксированной ширины и линий, которые быстро рисуются. Приложение САПР, например, использует косметический перо для создания скрытых, разделов, центров и линий измерений между .015 и .022 дюйма в ширину независимо от коэффициента масштабирования. *Геометрический перо* используется с приложениями, которым требуются масштабируемые линии, линии с уникальными стилями конца или объединения, а также строки, ширина которых превышает один пиксель. Например, приложение для работы с электронными таблицами использует геометрическое перо для определения каждой из столбцов линейчатой диаграммы в виде широкой линии.
