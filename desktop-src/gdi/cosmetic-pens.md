# <a name="cosmetic-pens"></a>Косметические перья

Размеры косметических планшета указываются в единицах устройства. Таким образом, линии, нарисованные с помощью косметических перьев, всегда имеют фиксированную ширину. Линии, нарисованные с помощью косметических перьев, обычно выводятся 3 – 10 раз быстрее, чем линии, рисуемые с помощью геометрического пера. Косметические перья имеют три атрибута: ширина, стиль и цвет. Дополнительные сведения об этих атрибутах см. в разделе [атрибуты пера](pen-attributes.md).

Чтобы создать косметический перо, используйте функцию [**креатепен**](/windows/desktop/api/Wingdi/nf-wingdi-createpen), [**креатепениндирект**](/windows/desktop/api/Wingdi/nf-wingdi-createpenindirect)или [**ексткреатепен**](/windows/desktop/api/Wingdi/nf-wingdi-extcreatepen) . Чтобы получить один из трех перьевых косметических элементов, управляемых системой, используйте функцию [**жетстоккобжект**](/windows/desktop/api/Wingdi/nf-wingdi-getstockobject) .

После создания пера (или получения маркера для одного из геоакций) выберите перо в контексте устройства (DC) приложения с помощью функции [**SelectObject**](/windows/desktop/api/Wingdi/nf-wingdi-selectobject) . С этого момента приложение использует это перо для любых операций рисования строк в клиентской области.
