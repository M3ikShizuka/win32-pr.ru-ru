# <a name="font-families"></a>Семейства шрифтов

*Семейство шрифтов* — это набор шрифтов со стандартными характеристиками толщины и засечек. Существует пять семейств шрифтов. Шестое семейство позволяет приложению использовать шрифт по умолчанию. В следующей таблице описаны семейства шрифтов.



| Семейство шрифтов | Описание                                                                                                                                   |
|-------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| Декоративный  | Задает шрифт новизна. Примером является старый английский.                                                                                          |
| Dontcare    | Указывает универсальное имя семейства. Это имя используется, если сведения о шрифте не существуют или не имеют значения. Используется шрифт по умолчанию. |
| Современная      | Задает шрифт с засечками с засечек или без них. Шрифты с пробелом обычно являются современными; Примеры: пика, Elite и Courier New.         |
| Символы       | Задает пропорциональный шрифт с засечками. В качестве примера используется Times New Roman.                                                                     |
| Скрипт      | Указывает шрифт, который должен выглядеть как рукописный ввод; Примерами могут служить script и письменный.                                              |
| Швейцарский       | Задает пропорциональный шрифт без засечек. Например, Arial.                                                                            |



 

Эти семейства шрифтов соответствуют константам, найденным в файле Вингди. h: FF \_ декоративные, FF \_ DONTCARE, FF \_ современный, FF \_ Roman, FF \_ script и FF \_ Швейцарии. Приложение использует эти константы при создании шрифта, выборе шрифта или извлечении сведений о шрифте.

Шрифты в семействе различаются по размеру (10 пунктов, 24 точки и т. д.) и стилю (обычный, курсив и т. д.).
