# <a name="color-basics"></a>Основные сведения о цвете

Цветовые возможности устройств, таких как экраны и принтеры, могут варьироваться от монохромного к тысячам цветов. Так как приложению может потребоваться создать выходные данные для устройств в этом диапазоне, он должен быть готов к обработке различных возможностей цвета.

Приложение может определить количество цветов, доступных для данного устройства, используя функцию [**жетдевицекапс**](/windows/desktop/api/Wingdi/nf-wingdi-getdevicecaps) для получения значения нумколорс. Это значение указывает количество цветов, доступных для использования приложением. Обычно это число соответствует физическому свойству выходного устройства, например число красок на принтере или количество различных цветовых сигналов, которые видеоадаптер может передавать монитору.

Хотя значение НУМКОЛОРС указывает количество цветов, оно не определяет доступные цвета. Приложение может определить, какие цвета доступны, перечисляя все перья, имеющие \_ сплошной тип PS. Так как драйвер устройства, поддерживающий данное устройство, обычно имеет полный спектр перьев, а система требует, чтобы у твердотельных перьев были только цвета, которые может создать устройство, перечисление этих перьев часто эквивалентно перечислению цветов. Приложение может перечислить перья с помощью функции [**енумобжектс**](/windows/desktop/api/Wingdi/nf-wingdi-enumobjects) . Пример кода см. в разделе [перечисление цветов](enumerating-colors.md).
