# <a name="raster-operation-codes"></a>Коды растровых операций

Коды операций с растровыми операциями определяют, как интерфейс графических устройств (GDI) объединяет биты из выбранного пера с битами в битовой карте назначения.

В этом обзоре перечислены и описаны двоичные и ternary растровые операции, используемые GDI. [Операция двоичного растрового](binary-raster-operations.md) изображения включает два операнда: перо и конечный точечный рисунок. [Точечная операция ternary](ternary-raster-operations.md) включает три операнда: исходный точечный рисунок, кисть и точечный рисунок назначения. Бинарные и ternary растровые операции используют логические операторы.
