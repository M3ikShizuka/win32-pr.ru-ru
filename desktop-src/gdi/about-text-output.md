# <a name="about-text-output"></a>О выводе текста

Текстовый вывод является наиболее распространенным типом графических данных, найденных в клиентской области. они используются приложениями различными способами. Приложения обработки и настольной публикации создают документы с форматированным текстом; приложения электронных таблиц используют текст, числа и символы для указания формул, столбцов меток и значений списка; приложения базы данных создают записи и отображают запросы с текстом, а приложения САПР используют текст для меток объектов и отображают размеры.

Существуют функции форматирования и отрисовки текста в клиентской области приложения и на странице бумаги принтера. Эти функции можно разделить на две категории: те, которые отформатируют текст (или подготавливают его для вывода), и те, которые на самом деле рисуют текст. Функции форматирования выровняйте текст, установить межзнаковый интервал, задать цвета фона текста и текста, а также выровнять текст. Функции рисования вырисуют отдельные символы (или символы) или целые строки текста.

При работе в Microsoft Windows фиксированные разрывы строк задаются парой символов возврата каретки и перевода строки ( \\ r \\ n).
