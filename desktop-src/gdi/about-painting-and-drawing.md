# <a name="about-painting-and-drawing"></a>О рисовании и рисовании

Почти все приложения используют экран для отображения данных, которыми они управляют. Приложение рисует изображения, отображает рисунки и записывает текст, чтобы пользователь мог просматривать данные по мере их создания, изменения и печати. Microsoft Windows предоставляет широкую поддержку для рисования и рисования, но из-за природы многозадачных операционных систем приложения должны взаимодействовать друг с другом при доступе к экрану.

Чтобы обеспечить бесперебойную работу всех приложений, система управляет всем выводом на экран. Приложения используют Windows в качестве основного выходного устройства, а не самого экрана. Система предоставляет контексты устройств, которые уникально соответствуют Windows. Приложения используют контексты устройств для направления выходных данных в указанные окна. Рисование в окне (направление вывода в него) не позволяет приложению противоречить выходным данным других приложений и позволяет приложениям сосуществовать друг с другом, одновременно используя все возможности графики системы.
