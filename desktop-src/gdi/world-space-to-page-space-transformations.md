# <a name="world-space-to-page-space-transformations"></a>Мировое пространство для Page-Space преобразований

Преобразование в мировое пространство для преобразований страничного пространства поддерживает перевод и масштабирование. Кроме того, они поддерживают возможности вращения, наклона и отражения. В следующих разделах описываются эти преобразования, демонстрируются их эффекты и предоставляются алгоритмы, используемые для их достижения.

-   [Перевод](translation.md)
-   [Масштабирование](scaling.md)
-   [Поворот](rotation.md)
-   [Сдвиг](shear.md)
-   [Отражение](reflection.md)
-   [Объединение преобразований «из Интернета в мир»](combined-world-to-page-space-transformations.md)
