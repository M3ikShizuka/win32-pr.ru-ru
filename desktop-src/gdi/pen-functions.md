# <a name="pen-functions-windows-gdi"></a>функции пера (Windows GDI)

Для перьев используются следующие функции.



| Функция                                       | Описание                                                                                            |
|------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**креатепен**](/windows/desktop/api/Wingdi/nf-wingdi-createpen)                 | Создает логическое перо с указанным стилем, шириной и цветом.                                  |
| [**креатепениндирект**](/windows/desktop/api/Wingdi/nf-wingdi-createpenindirect) | Создает логическое косметические перо со стилем, шириной и цветом, указанными в структуре.          |
| [**ексткреатепен**](/windows/desktop/api/Wingdi/nf-wingdi-extcreatepen)           | Создает логическое косметические или геометрические перья с указанным стилем, шириной и атрибутами кисти. |
| [**сетдкпенколор**](/windows/desktop/api/Wingdi/nf-wingdi-setdcpencolor)         | Задает цвет пера контекста текущего устройства.                                                             |
