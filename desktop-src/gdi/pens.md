# <a name="pens"></a>Перья

Перо — это графическое средство, которое приложение может использовать для рисования линий и кривых. Графические приложения используют перья для рисования произвольных линий, прямых линий и кривых. Приложения для автоматизированного проектирования (CAD) используют перья для рисования видимых строк, скрытых линий, строк разделов, центрированных линий и т. д. Приложения обработки и настольной публикации используют перья для рисования границ и правил. Приложения для работы с электронными таблицами используют перья для обозначения тенденций в графиках, а диаграммы — для диаграмм и круговых диаграмм.
