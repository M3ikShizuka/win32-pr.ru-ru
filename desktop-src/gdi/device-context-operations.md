# <a name="device-context-operations"></a>Операции контекста устройства

Приложение может выполнять следующие операции в контексте устройства:

-   Перечисление существующих графических объектов.
-   Выберите новые графические объекты.
-   Удаление существующих графических объектов.
-   Сохраните текущие графические объекты, их атрибуты и графические режимы.
-   Восстановите ранее сохраненные графические объекты, их атрибуты и графические режимы.

Кроме того, приложение может использовать контекст устройства для:

-   Определите, как преобразуются выходные данные графики.
-   Отмена длительных операций рисования (начатых потоком в многопоточном приложении).
-   Сброс принтера до определенного состояния.
