# <a name="path-functions-windows-gdi"></a>функции пути (Windows GDI)

Следующие функции используются для создания, изменения или рисования контуров.



| Функция                                       | Описание                                                                                                                                                      |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**абортпас**](/windows/desktop/api/Wingdi/nf-wingdi-abortpath)                 | Закрывает и удаляет все пути в указанном контексте устройства.                                                                                                   |
| [**бегинпас**](/windows/desktop/api/Wingdi/nf-wingdi-beginpath)                 | Открывает скобку пути в указанном контексте устройства.                                                                                                            |
| [**клосефигуре**](/windows/desktop/api/Wingdi/nf-wingdi-closefigure)             | Закрывает открытую фигуру в контуре.                                                                                                                                 |
| [**ендпас**](/windows/desktop/api/Wingdi/nf-wingdi-endpath)                     | Закрывает скобку пути и выбирает путь, определенный скобкой, в указанный контекст устройства.                                                             |
| [**филлпас**](/windows/desktop/api/Wingdi/nf-wingdi-fillpath)                   | Закрывает все открытые фигуры в текущем контуре и заполняет внутреннюю часть пути, используя текущую кисть и режим заливки многоугольников.                                   |
| [**флаттенпас**](/windows/desktop/api/Wingdi/nf-wingdi-flattenpath)             | Преобразует все кривые в пути, выбранном в текущем контексте устройства (DC), преобразуя каждую кривую в последовательность линий.                            |
| [**жетмитерлимит**](/windows/desktop/api/Wingdi/nf-wingdi-getmiterlimit)         | Возвращает предел среза для указанного контекста устройства.                                                                                                      |
| [**GetPath**](/windows/desktop/api/Wingdi/nf-wingdi-getpath)                     | Получает координаты, определяющие конечные точки линий и контрольные точки кривых, найденные по пути, выбранному в заданном контексте устройства. |
| [**пасторегион**](/windows/desktop/api/Wingdi/nf-wingdi-pathtoregion)           | Создает регион из пути, выбранного в заданном контексте устройства.                                                                               |
| [**сетмитерлимит**](/windows/desktop/api/Wingdi/nf-wingdi-setmiterlimit)         | Задает ограничение длины угловых соединений для указанного контекста устройства.                                                                                   |
| [**строкеандфиллпас**](/windows/desktop/api/Wingdi/nf-wingdi-strokeandfillpath) | Закрывает все открытые фигуры в контуре, обводки контура с помощью текущего пера и заполняет внутреннюю часть с помощью текущей кисти.                  |
| [**строкепас**](/windows/desktop/api/Wingdi/nf-wingdi-strokepath)               | Отображает указанный путь с помощью текущего пера.                                                                                                             |
| [**виденпас**](/windows/desktop/api/Wingdi/nf-wingdi-widenpath)                 | Переопределяет текущий путь как область, которая будет окрашена, если контур был штриховым с помощью пера, выбранного в данный момент в данном контексте устройства.            |
