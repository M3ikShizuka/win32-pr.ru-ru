# <a name="comparing-regions"></a>Сравнение регионов

Приложение сравнивает два региона, чтобы определить, идентичны ли они, вызвав функцию [**екуалргн**](/windows/desktop/api/Wingdi/nf-wingdi-equalrgn) . Екуалргн считает два региона идентичными, если они равны размеру и форме.
