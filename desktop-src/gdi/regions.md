# <a name="regions-windows-gdi"></a>регионы (Windows GDI)

*Регион* — это прямоугольник, многоугольник или эллипс (или сочетание двух или более этих фигур), которые могут быть заполнены, окрашены, инвертированы, заключены в фреймы и использованы для проверки попадания (проверка положения курсора).
