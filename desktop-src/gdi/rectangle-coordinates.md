# <a name="rectangle-coordinates"></a>Координаты прямоугольника

Приложение должно использовать структуру [**Rect**](/previous-versions//dd162897(v=vs.85)) для определения прямоугольника. Структура задает координаты двух точек: верхнего левого и нижнего правого угла прямоугольника. Стороны прямоугольника расширяются из этих двух точек и параллельны с осью x и осью y.

Значения координат для прямоугольника выражаются как целые числа со знаком. Значение координаты правой стороны прямоугольника должно быть больше, чем у левой стороны. Аналогично, значение координаты нижнего значения должно быть больше верхнего.

Так как приложения могут использовать прямоугольники для различных целей, функции Rectangle не используют явную единицу измерения. Вместо этого все координаты прямоугольника и измерения задаются в виде логических значений со знаком. Режим сопоставления и функция, в которых используется прямоугольник, определяют единицы измерения. Дополнительные сведения о координатах и режимах сопоставления см. в разделе [координатные пространства и преобразования](coordinate-spaces-and-transformations.md).
