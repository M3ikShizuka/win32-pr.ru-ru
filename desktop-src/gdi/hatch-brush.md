# <a name="hatch-brush"></a>Кисть штриховки

GDI поддерживает шесть предопределенных логических кистей штриховки. Следующие прямоугольники были окрашены с использованием шести стандартных кистей штриховки.

![Иллюстрация, на которой показаны шесть квадратиков, один из которых заполнен каждой из шести кистей штриховки](images/hatchbrush.png)

Приложение может создать кисть штриховки, вызвав функцию [**креатехатчбруш**](/windows/desktop/api/Wingdi/nf-wingdi-createhatchbrush) , указав один из шести стилей штриховки.
