# <a name="coordinate-spaces-and-transformations"></a>Координатные пространства и преобразования

Приложения используют пространство координат и преобразования для масштабирования, вращения, преобразования, наклона и отражения выходных данных графики. *Координатное пространство* — это плоское пространство, которое находит двухмерные объекты с помощью двух осей, перпендикулярных друг другу. Существует четыре координатных пространства: мир, страница, устройство и физическое устройство (клиентская область, Рабочий стол или страница бумаги принтера).

*Преобразование* — это алгоритм, который изменяет ("преобразует") размер, ориентацию и форму объектов. Преобразования также переносят графический объект из одного пространства координат в другое. В конечном итоге объект отображается на физическом устройстве (обычно это экран или принтер).
