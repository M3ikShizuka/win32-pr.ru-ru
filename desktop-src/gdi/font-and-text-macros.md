# <a name="font-and-text-macros"></a>Макросы шрифта и текста

Для шрифтов и текста используются следующие макросы.



| Макрос                            | Описание                                            |
|----------------------------------|--------------------------------------------------------|
| [**делетефонт**](/windows/desktop/api/Windowsx/nf-windowsx-deletefont) | Удаляет указанный шрифт.                              |
| [**селектфонт**](/windows/desktop/api/Windowsx/nf-windowsx-selectfont) | Выбирает шрифт в указанном контексте устройства (DC). |
