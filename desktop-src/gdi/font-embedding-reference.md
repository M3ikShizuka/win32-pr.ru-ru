# <a name="font-embedding-reference"></a>Справочник по внедрению шрифтов

Библиотека служб внедрения шрифтов предоставляет механизм для объединения шрифтов TrueType и Microsoft OpenType в документ или файл. Как правило, документ, содержащий внедренные шрифты, нуждается в шрифтах для подготовки документа к просмотру на другом компьютере. Внедрение шрифта гарантирует, что шрифт, указанный в файле, будет присутствовать на компьютере, получающем файл. Однако некоторые шрифты не могут быть перемещены на другие компьютеры из-за проблем с автором прав, ограничивающих распространение.

Для служб внедрения шрифтов используются следующие элементы.

-   [Функции внедрения шрифтов](font-embedding-functions.md)
-   [Структуры внедрения шрифтов](font-embedding-services-structures.md)
-   [Сообщения об ошибках внедрения шрифтов](font-embedding-function-error-messages.md)
