# <a name="about-paths"></a>О путях

Путь — это один из объектов, связанных с контекстом устройства (DC). Однако в отличие от объектов по умолчанию (перо, кисть и шрифт), которые входят в состав любого нового контроллера домена, путь по умолчанию отсутствует. Дополнительные сведения о контроллерах домена см. в разделе [контексты устройств](device-contexts.md).
