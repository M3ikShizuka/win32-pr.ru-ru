# <a name="using-regions-to-clip-output"></a>Использование регионов для вывода клипов

Этот раздел содержит один пример, демонстрирующий, как можно использовать регионы для определения того, как может выглядеть часть выходных данных клиентской области. Регионы, используемые для этой цели, называются областями обрезки.

Пример для этого раздела взят из приложения, позволяющего пользователю захватить весь рабочий стол в виде точечного рисунка, а затем изолировать и сохранить часть этого образа как файл .BMP.

Выбрав команду **определить область** отсечения в меню приложения, пользователь может выбрать область обрезки, щелкнув левую кнопку мыши и перетащив указатель мыши. Когда пользователь перетаскивает мышь, приложение рисует прямоугольник, соответствующий новому региону отсечения.

Если щелкнуть **клип** в меню приложения, пользователь может перерисовать изолированную часть изображения в пределах границ указанного прямоугольника.

В этом разделе содержатся сведения по следующим темам.

-   [Определение области обрезки](defining-the-clipping-region.md)
-   [Вырезание выходных данных](clipping-output.md)
