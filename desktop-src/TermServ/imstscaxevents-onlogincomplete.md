# <a name="imstscaxeventsonlogincomplete-method"></a>Метод Имстскаксевентс:: Онлогинкомплете

Вызывается, когда клиентский элемент управления успешно выполнил вход на сервер узла сеансов удаленный рабочий стол (узел сеансов удаленных рабочих столов), как показано в диалоговом окне входа в систему Windows.

## <a name="syntax"></a>Синтаксис


```C++
void OnLoginComplete();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Реализуйте этот метод в приемнике событий, чтобы получить уведомление о том, что элемент управления выполнил вход.

## <a name="examples"></a>Примеры

В следующем примере показано, как справиться с этим событием с помощью Visual Basic кода скрипта. В этом примере предполагается, что управляющий объект называется "Мсрдпклиент".

Дополнительные сведения о веб-подключение к удаленному рабочему столу см. в разделе [требования для веб-подключение к удаленному рабочему столу](requirements-for-remote-desktop-web-connection.md).


```VB
Sub MsRdpClient.OnLoginComplete()
    Msgbox("Login has completed")
End sub
```
