# <a name="imsrdpclientshell-interface"></a>Интерфейс Имсрдпклиентшелл

Параметры клиента подключение к удаленному рабочему столу (RDC), используемые для запуска клиента из удаленный рабочий стол Веб-доступ (RD Веб-доступ) или других веб-порталов.

## <a name="members"></a>Элементы

Интерфейс **имсрдпклиентшелл** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Имсрдпклиентшелл** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **имсрдпклиентшелл** содержит следующие методы.



| Метод                                                     | Описание                                                  |
|:-----------------------------------------------------------|:-------------------------------------------------------------|
| [**жетрдппроперти**](/previous-versions/windows/desktop/legacy/aa381303(v=vs.85)) | Извлекает одно свойство RDP.<br/>                  |
| [**Launch**](imsrdpclientshell-launch.md)                 | Запускает содержимое удаленного файла с веб-портала.<br/> |
| [**сетрдппроперти**](/previous-versions/windows/desktop/legacy/aa381312(v=vs.85)) | Задает одно свойство RDP.<br/>                       |



 

### <a name="properties"></a>Свойства

Интерфейс **имсрдпклиентшелл** имеет следующие свойства.



| Свойство                                                                                              | Тип доступа           | Описание                                                                                               |
|:------------------------------------------------------------------------------------------------------|:----------------------|:----------------------------------------------------------------------------------------------------------|
| [**исремотепрограмклиентинсталлед**](imsrdpclientshell-isremoteprogramclientinstalled.md)<br/> | Только для чтения<br/>  | Возвращает значение, указывающее, поддерживает ли клиент подключение к удаленному рабочему столу (RDC) функции RemoteApp.<br/> |
| [**публикмоде**](imsrdpclientshell-publicmode.md)<br/>                                         | Чтение/запись<br/> | Получает, включен ли общий режим.<br/>                                                      |
| [**рдпфилеконтентс**](imsrdpclientshell-rdpfilecontents.md)<br/>                               | Чтение/запись<br/> | Потоковая версия файла конфигурации RDP.<br/>                                                  |
