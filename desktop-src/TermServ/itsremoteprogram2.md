# <a name="itsremoteprogram2-interface"></a>Интерфейс ITSRemoteProgram2

Определяет свойства для использования с RemoteApp.

## <a name="members"></a>Элементы

Интерфейс **ITSRemoteProgram2** наследует от [**итсремотепрограм**](itsremoteprogram.md). **ITSRemoteProgram2** также имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Интерфейс **ITSRemoteProgram2** имеет следующие свойства.



| Свойство                                                                                  | Тип доступа           | Описание                                                 |
|:------------------------------------------------------------------------------------------|:----------------------|:------------------------------------------------------------|
| [**ремотеаппликатионаргс**](itsremoteprogram2-remoteapplicationargs.md)<br/>       | Только на запись<br/> | Аргументы командной строки для RemoteApp.<br/>    |
| [**ремотеаппликатионнаме**](itsremoteprogram2-remoteapplicationname.md)<br/>       | Только на запись<br/> | Имя RemoteApp.<br/>                       |
| [**ремотеаппликатионпрограм**](itsremoteprogram2-remoteapplicationprogram.md)<br/> | Только на запись<br/> | Путь и имя файла удаленного приложения RemoteApp.<br/> |
