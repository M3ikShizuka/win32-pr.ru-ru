# <a name="remoteapp-and-desktop-connection-management-service-structures"></a>Структуры служб управление подключениями к удаленным рабочим столам и приложениям RemoteApp

API службы управление подключениями к удаленным рабочим столам и приложениям RemoteApp предоставляет следующие структуры.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**плугинресаурце**](/windows/win32/api/tspubplugincom/ns-tspubplugincom-pluginresource)
</dt> <dd>

Содержит сведения о ресурсе, который можно назначить пользователям в подключения к удаленным рабочим столам и приложениям RemoteApp.

</dd> <dt>

[**pluginResource2**](/windows/win32/api/tspubplugin2com/ns-tspubplugin2com-pluginresource2)
</dt> <dd>

Содержит дополнительные сведения о ресурсе, который можно назначить пользователям в подключения к удаленным рабочим столам и приложениям RemoteApp.

</dd> <dt>

[**pluginResource2FileAssociation**](/windows/win32/api/tspubplugin2com/ns-tspubplugin2com-pluginresource2fileassociation)
</dt> <dd>

Содержит сведения о сопоставлении файлов в подключения к удаленным рабочим столам и приложениям RemoteApp.

</dd> </dl>
