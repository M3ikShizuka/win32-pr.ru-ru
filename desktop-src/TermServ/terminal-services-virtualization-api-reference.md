# <a name="remote-desktop-virtualization-api-reference"></a>Справочник по API виртуализации удаленный рабочий стол

Удаленный рабочий стол API виртуализации поддерживает интерфейсы и структуры, которые можно использовать для создания пользовательских подключаемых модулей, переопределяющих стандартную логику перенаправления подключение к удаленному рабочему столу Broker (RDCB). Настраиваемые подключаемые модули могут перенаправлять подключения к серверам узла сеансов удаленный рабочий стол, а также к виртуальным машинам, виртуальным рабочим столам, блейд-серверам и физическим настольным компьютерам.
