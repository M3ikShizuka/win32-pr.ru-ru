# <a name="imsrdpclientshellshowtrustedsitesmanagementdialog-method"></a>Метод Имсрдпклиентшелл:: Шовтрустедситесманажементдиалог

Отображает диалоговое окно список надежных сайтов.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT ShowTrustedSitesManagementDialog();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Если этот метод завершается успешно, возвращается значение **S \_ ОК**. В противном случае возвращается код ошибки **HRESULT** .
