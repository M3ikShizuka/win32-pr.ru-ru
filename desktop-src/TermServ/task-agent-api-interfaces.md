# <a name="personal-desktop-task-agent-api-interfaces"></a>Интерфейсы API агента задачи персональных рабочих столов

Интерфейс API агента задачи персонального рабочего стола поддерживает следующие интерфейсы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**ирдвтаскплугиннотифисинк**](irdvtaskpluginnotifysink.md)
</dt> <dd>

Интерфейс [**ирдвтаскплугиннотифисинк**](irdvtaskpluginnotifysink.md) используется агентом задач для взаимодействия с агентом триггера.

</dd> <dt>

[**ирдвтаскплугин**](irdvtaskplugin.md)
</dt> <dd>

Интерфейс [**ирдвтаскплугин**](irdvtaskplugin.md) реализуется *агентом задачи* обновления виртуальной машины, что позволяет агенту задач управлять обновлениями системы для виртуальной машины.

</dd> </dl>
