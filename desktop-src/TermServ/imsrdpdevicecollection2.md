# <a name="imsrdpdevicecollection2-interface"></a>Интерфейс IMsRdpDeviceCollection2

Представляет коллекцию объектов Device. Это усовершенствование интерфейса [**имсрдпдевицеколлектион**](imsrdpdevicecollection.md) .

## <a name="members"></a>Элементы

Интерфейс **IMsRdpDeviceCollection2** наследует от [**имсрдпдевицеколлектион**](imsrdpdevicecollection.md). **IMsRdpDeviceCollection2** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **IMsRdpDeviceCollection2** содержит следующие методы.



| Метод                                                                         | Описание                                                                                                 |
|:-------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------|
| [**адддевицебинстанцеид**](imsrdpdevicecollection2-adddevicebyinstanceid.md) | Добавляет устройство, не имеющее список, в коллекцию устройств.<br/>                                                |
| [**редиректнов**](imsrdpdevicecollection2-redirectnow.md)                     | Принудительное перенаправление или удаление устройств, которые были выбраны или отменены из коллекции.<br/> |
