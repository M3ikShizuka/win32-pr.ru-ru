# <a name="win32_tsvm-methods"></a>\_Методы тсвм Win32

Класс [**\_ методов Win32 тсвм**](win32-tsvm.md) предоставляет следующие методы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**Метод Куерйоффлинеинформатион**](queryofflineinformation-win32-tsvm.md)
</dt> <dd>

Извлекает свойства текущего сервера виртуальных рабочих столов (VDS), но только в том случае, если сервер находится в автономном режиме.

</dd> <dt>

[**Метод Думпвминфо**](dumpvminfo-win32-tsvm.md)
</dt> <dd>

Этот член предназначен для внутренних целей тестирования и не должен использоваться в коде.

</dd> <dt>

[**Метод экспорта**](export-win32-tsvm.md)
</dt> <dd>

Извлекает сведения мониторинга сеанса виртуальной машины.

</dd> </dl>
