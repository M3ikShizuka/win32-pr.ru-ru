# <a name="iremotedesktopclienttouchpointer-properties"></a>Свойства Иремотедесктопклиенттаучпоинтер

Интерфейс [**иремотедесктопклиенттаучпоинтер**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclienttouchpointer) предоставляет следующие свойства.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**Enabled, свойство**](iremotedesktopclienttouchpointer-enabled.md)
</dt> <dd>

Включена ли функция сенсорного указателя в клиентском элементе управления контейнера приложения RDP.

</dd> <dt>

[**Евентсенаблед, свойство**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_eventsenabled)
</dt> <dd>

Включены ли уведомления о событии сенсорного указателя для клиентского элемента управления контейнера приложения RDP.

</dd> <dt>

[**Поинтерспид, свойство**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclienttouchpointer-get_pointerspeed)
</dt> <dd>

Скорость перемещения курсора сенсорного указателя на виртуальный рабочий стол относительно скорости жеста на стороне клиента.

</dd> </dl>
