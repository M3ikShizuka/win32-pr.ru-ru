# <a name="graphics-virtual-channels-interfaces"></a>Интерфейсы виртуальных каналов графики

API виртуальных каналов графики поддерживает следующие интерфейсы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**иврдсграфиксчаннел**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannel)
</dt> <dd>

этот интерфейс используется RemoteFX графических служб для отправки и получения данных в канал виртуальной графики.

</dd> <dt>

[**иврдсграфиксчаннелевентс**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannelevents)
</dt> <dd>

Этот интерфейс получает уведомления, связанные с виртуальным каналом графики.

</dd> <dt>

[**иврдсграфиксчаннелманажер**](/windows/desktop/api/wrdsgraphicschannels/nn-wrdsgraphicschannels-iwrdsgraphicschannelmanager)
</dt> <dd>

этот интерфейс используется интерфейсом API RemoteFX graphics services для создания графических виртуальных каналов, необходимых для удаленного взаимодействия с данными графики.

</dd> </dl>
