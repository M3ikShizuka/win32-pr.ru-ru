# <a name="iworkspacescriptable2-methods"></a>Методы IWorkspaceScriptable2

Интерфейс [**IWorkspaceScriptable2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable2) поддерживает следующие методы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**Метод Стартворкспацеекс**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-startworkspaceex)
</dt> <dd>

Связывает учетные данные пользователя и сертификаты с ИДЕНТИФИКАТОРом подключения; также содержит дополнительные элементы безопасности и пользовательского интерфейса.

</dd> <dt>

[**Метод Ресаурцедисмиссед**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-resourcedismissed)
</dt> <dd>

Предупреждает пользователя о том, что ресурс был отключен или иным образом отклонен.

</dd> </dl>
