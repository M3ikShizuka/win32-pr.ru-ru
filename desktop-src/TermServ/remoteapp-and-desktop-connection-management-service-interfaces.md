# <a name="remoteapp-and-desktop-connection-management-service-interfaces"></a>Интерфейсы служб управление подключениями к удаленным рабочим столам и приложениям RemoteApp

API управление подключениями к удаленным рабочим столам и приложениям RemoteApp службы предоставляет следующие интерфейсы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**итспубплугин**](/windows/desktop/api/tspubplugincom/nn-tspubplugincom-itspubplugin)
</dt> <dd>

Предоставляет свойства и методы, предоставляющие сведения о ресурсах, доступных пользователям подключений к удаленным рабочим столам и приложениям RemoteApp.

</dd> <dt>

[**ItsPubPlugin2**](/windows/desktop/api/tspubplugin2com/nn-tspubplugin2com-itspubplugin2)
</dt> <dd>

Указывает методы, предоставляющие сведения о ресурсах, доступных пользователям подключений к удаленным рабочим столам и приложениям RemoteApp.

</dd> </dl>
