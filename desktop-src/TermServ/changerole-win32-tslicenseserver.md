# <a name="changerole-method-of-the-win32_tslicenseserver-class"></a>Метод ChangeRole \_ класса Win32 тслиценсесервер

Изменяет область обнаружения сервера лицензирования удаленный рабочий стол. Область обнаружения определяет, какие серверы узла сеансов удаленный рабочий стол (узлы сеансов удаленных рабочих столов) в сети могут автоматически обнаружить сервер лицензирования.

## <a name="syntax"></a>Синтаксис


```mof
uint32 ChangeRole(
  [in] uint32 ServerRole
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ServerRole* \[ окне\]
</dt> <dd>

Область обнаружения для сервера лицензирования удаленный рабочий стол. Можно задать одно из следующих значений.

<dt>

0
</dt> <dd>

Серверы узла сеансов удаленных рабочих столов в одной рабочей группе могут обнаружить сервер лицензирования.

</dd> <dt>

1
</dt> <dd>

Серверы узла сеансов удаленных рабочих столов в одном домене могут обнаружить сервер лицензирования. Чтобы задать это значение, необходимо войти в систему с учетной записью администратора домена.

</dd> <dt>

2
</dt> <dd>

Серверы узла сеансов удаленных рабочих столов из нескольких доменов в одном лесу могут обнаружить сервер лицензирования. Чтобы задать это значение, необходимо войти в систему от имени администратора предприятия.

</dd> </dl> </dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если метод завершается с ошибкой, он возвращает ноль. Если метод завершился неудачно, он возвращает ненулевое значение. Список кодов ошибок см. в разделе [службы удаленных рабочих столов коды ошибок поставщика WMI](terminal-services-wmi-provider-error-codes.md).
