# <a name="imsrdpclientadvancedsettingsdedicatedterminal-property"></a>Имсрдпклиентадванцедсеттингс: свойство Едикатедтерминал:D

Данное свойство не поддерживается.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_DedicatedTerminal(
  [in]  LONG dedicatedTerminal
);

HRESULT get_DedicatedTerminal(
  [out] LONG *pdedicatedTerminal
);
```



## <a name="property-value"></a>Значение свойства

Задайте для этого параметра значение 0, чтобы отключить функцию, или ненулевое, чтобы включить эту функцию.

## <a name="error-codes"></a>Коды ошибок

Возвращает **\_ значение false**.
