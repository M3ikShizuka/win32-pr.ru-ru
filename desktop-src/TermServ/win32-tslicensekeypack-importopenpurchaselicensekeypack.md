# <a name="importopenpurchaselicensekeypack-method-of-the-win32_tslicensekeypack-class"></a>Метод Импортопенпурчаселиценсекэйпакк \_ класса Win32 тслиценсекэйпакк

Импорт, с другого сервера лицензий удаленный рабочий стол — Open License службы удаленных рабочих столов пакет лицензионных ключей.

## <a name="syntax"></a>Синтаксис


```mof
uint32 ImportOpenPurchaseLicenseKeyPack(
  [in]  string sLicenseNumber,
  [in]  string sAuthorizationNumber,
  [in]  uint32 ProductVersion,
  [in]  uint32 ProductType,
  [in]  uint32 LicenseCount,
  [in]  string sSourceLSName,
  [in]  string sSourceLSProductId,
  [out] uint32 KeyPackId
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*слиценсенумбер* \[ окне\]
</dt> <dd>

8-символьная числовая строка, предоставленная вместе с пакетом лицензионных ключей. Параметр *слиценсенумбер* не может содержать дефисы.

</dd> <dt>

*саусоризатионнумбер* \[ окне\]
</dt> <dd>

Строка, состоящая из 15 символов и имеющая ключ лицензии. Параметр *саусоризатионнумбер* не может содержать дефисы.

</dd> <dt>

*ProductVersion* \[ окне\]
</dt> <dd>

Номер версии продукта.

<dt>

0
</dt> <dd>

Не поддерживается.

</dd> <dt>

1
</dt> <dd>

Не поддерживается.

</dd> <dt>

2
</dt> <dd>

Windows Server 2008

</dd> </dl> </dd> <dt>

*Продукттипе* \[ окне\]
</dt> <dd>

Тип продукта.

<dt>

0
</dt> <dd>

Тип продукта пакета лицензионных ключей службы удаленных рабочих столов для каждого устройства. Таким образом, каждому устройству, которое подключается к серверу узла сеансов удаленных рабочих столов, должна быть назначена лицензия.

</dd> <dt>

1
</dt> <dd>

Тип продукта пакета лицензионных ключей службы удаленных рабочих столов для каждого пользователя. Поэтому у каждого пользователя, подключающегося к серверу узла сеансов удаленных рабочих столов, должна быть лицензия.

</dd> <dt>

2
</dt> <dd>

Недопустимый тип продукта.

</dd> </dl> </dd> <dt>

*Лиценсекаунт* \[ окне\]
</dt> <dd>

Число импортируемых лицензий

</dd> <dt>

*ссаурцелснаме* \[ окне\]
</dt> <dd>

Имя исходного удаленный рабочий стол сервера лицензирования. Это либо полное различающееся имя, либо IP-адрес сервера.

</dd> <dt>

*ссаурцелспродуктид* \[ окне\]
</dt> <dd>

Идентификатор сервера лицензий удаленный рабочий стол. — Это буквенно-цифровая строка, состоящая из 35 символов, которая не может содержать дефисы.

</dd> <dt>

*Кэйпаккид* \[ заполняет\]
</dt> <dd>

Получает идентификатор пакета ключей.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если метод завершается с ошибкой, он возвращает ноль. Если метод завершился неудачно, он возвращает ненулевое значение. Список кодов ошибок см. в разделе [службы удаленных рабочих столов коды ошибок поставщика WMI](terminal-services-wmi-provider-error-codes.md).
