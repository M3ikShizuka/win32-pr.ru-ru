# <a name="scriptable-virtual-channels"></a>Виртуальные каналы с помощью сценариев

Дополнительные сведения об использовании виртуальных каналов, поддерживающих сценарии, см. [в разделе Реализация сценариев с использованием скриптов с помощью веб-подключение к удаленному рабочему столу](implementing-scriptable-virtual-channels-using-remote-desktop-web-connection.md).

Дополнительные сведения о виртуальных каналах см. в разделе [службы удаленных рабочих столов виртуальные каналы](terminal-services-virtual-channels.md).

**Windows XP:** В состав Windows XP входит веб-подключение к удаленному рабочему столу, который является обновленной версией служб терминалов расширенный клиент (TSAC). Веб-подключение к удаленному рабочему столу поддерживает [протокол удаленного рабочего стола](remote-desktop-protocol.md) (RDP) 5,1 и обратно СОВМЕСТИМА с RDP 5,0.
