# <a name="using-the-remote-desktop-activex-control"></a>использование элемента управления ActiveX удаленный рабочий стол

в следующих разделах содержатся сведения об использовании элемента управления удаленный рабочий стол ActiveX для настройки пользовательского интерфейса службы удаленных рабочих столов.

элемент управления удаленный рабочий стол ActiveX доступен в следующих формах:

-   **Элемент управления для сценариев**— реализует интерфейсы, которые являются надежными для сценариев. Эта форма элемента управления может использоваться веб-клиентами или сценариями (например, VBScript).
-   **Необрабатываемый элемент управления**— предоставляет дополнительные функциональные возможности, которые не являются надежными для сценариев. Эту форму элемента управления можно использовать из машинного или управляемого кода, но эта форма не может использоваться веб-клиентами или узлами, использующими только скрипты.

Следующий список содержит **идентификаторы CLSID** для различных элементов управления для различных версий операционной системы. Каждый **идентификатор CLSID** совместим с более поздними версиями системы. например, **идентификатор CLSID** для элемента управления, поддерживающего сценарии, в Windows Vista будет работать в более поздних версиях системы, таких как Windows 7.
