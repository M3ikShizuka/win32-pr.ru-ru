# <a name="whats-new-in-windows-server-2008-r2"></a>новые возможности Windows Server 2008 R2

Windows Сервер 2008 R2 предоставляет следующие новые программные элементы для службы удаленных рабочих столов.




| | | Службы терминалов теперь службы удаленных рабочих столов<br /> | Службы терминалов были переименованы службы удаленных рабочих столов.
