# <a name="whats-new-in-windows-server-2008-r2"></a>Новые возможности Windows Server 2008 R2

В Windows Server 2008 R2 появились следующие новые программные элементы для службы удаленных рабочих столов.



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<tbody>
<tr class="odd">
<td>Службы терминалов теперь службы удаленных рабочих столов<br/></td>
<td>Службы терминалов были переименованы службы удаленных рабочих столов. Сервер, на котором установлена служба роли узла сеансов удаленный рабочий стол (удаленных рабочих столов), теперь называется сервером узла сеансов удаленных рабочих столов, а не сервером терминалов.<br/>
<ul>
<li><a href="terminal-services-is-now-remote-desktop-services.md">Службы терминалов теперь службы удаленных рабочих столов</a></li>
</ul></td>
</tr>
<tr class="even">
<td>API виртуализации удаленный рабочий стол<br/></td>
<td>Удаленный рабочий стол API виртуализации предоставляет перечисления, интерфейсы и структуры, которые можно использовать для создания пользовательских подключаемых модулей, переопределяющих стандартную логику перенаправления подключение к удаленному рабочему столу Broker (RDCB). RDCB (ранее назывался посредником сеансов служб терминалов) был расширен для поддержки подключений к виртуальным машинам.<br/>
<ul>
<li><a href="using-the-remote-desktop-virtualization-api.md">Использование API виртуализации удаленный рабочий стол</a></li>
<li><a href="terminal-services-virtualization-api-reference.md">Справочник по API виртуализации удаленный рабочий стол</a></li>
</ul></td>
</tr>
<tr class="odd">
<td>Поставщик протокол удаленного рабочего стола<br/></td>
<td>API поставщика протокол удаленного рабочего стола можно использовать для создания протокола, который настраивает взаимодействие между службами службы удаленных рабочих столов и настольными клиентами.<br/>
<ul>
<li><a href="creating-a-custom-remote-protocol.md">Создание поставщика протокол удаленного рабочего стола</a></li>
<li><a href="custom-remote-protocol-reference.md">Справочник по поставщику протокол удаленного рабочего стола</a></li>
</ul></td>
</tr>
<tr class="even">
<td>службы удаленных рабочих столов API Аудиоендпоинт<br/></td>
<td>Службы удаленных рабочих столов API Аудиоендпоинт поддерживает перечисление, интерфейсы и структуры для регистрации конечной точки аудио и передачи данных.<br/>
<ul>
<li><a href="terminal-services-audioendpoint-api-reference.md">Справочник по API службы удаленных рабочих столов Аудиоендпоинт</a></li>
</ul></td>
</tr>
<tr class="odd">
<td>API службы управление подключениями к удаленным рабочим столам и приложениям RemoteApp<br/></td>
<td>API службы управление подключениями к удаленным рабочим столам и приложениям RemoteApp поддерживает интерфейсы и структуры, которые можно использовать для выполнения настраиваемой фильтрации ресурсов и обеспечения поддержки типов файлов, которые изначально не поддерживаются в Windows Server 2008 R2.<br/>
<ul>
<li><a href="centralized-publishing-api-reference.md">Справочник по API службы управление подключениями к удаленным рабочим столам и приложениям RemoteApp</a></li>
</ul></td>
</tr>
</tbody>
</table>
