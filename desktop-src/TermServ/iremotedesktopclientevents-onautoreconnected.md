# <a name="iremotedesktopclienteventsonautoreconnected-method"></a>Метод Иремотедесктопклиентевентс:: Онаутореконнектед

Вызывается при автоматическом повторном подключении клиентского элемента управления к удаленному сеансу.

## <a name="syntax"></a>Синтаксис


```C++
void OnAutoReconnected();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
