# <a name="imstscaxeventsondevicesbuttonpressed-method"></a>Метод Имстскаксевентс:: Ондевицесбуттонпрессед

Вызывается при нажатии кнопки устройства на панели подключения.

## <a name="syntax"></a>Синтаксис


```C++
void OnDevicesButtonPressed();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Кнопка устройства отображается на панели подключения, если перенаправление USB включено как на клиенте, так и на сервере.
