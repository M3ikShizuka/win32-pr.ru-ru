# <a name="iconnectionbrokerrequest-interface"></a>Интерфейс Иконнектионброкеррекуест

Предоставляет методы, необходимые для получения результатов асинхронного метода [**иконнектионброкерклиент:: жеттаржетинфо**](iconnectionbrokerclient-gettargetinfo.md) .

Этот интерфейс реализуется системой. Экземпляр этого интерфейса предоставляется вызывающему объекту с помощью метода [**иконнектионброкерклиент:: жеттаржетинфо**](iconnectionbrokerclient-gettargetinfo.md) .

## <a name="members"></a>Элементы

Интерфейс **иконнектионброкеррекуест** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иконнектионброкеррекуест** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иконнектионброкеррекуест** содержит следующие методы.



| Метод                                                      | Описание                                                           |
|:------------------------------------------------------------|:----------------------------------------------------------------------|
| [**CheckStatus**](iconnectionbrokerrequest-checkstatus.md) | Вызывается для определения состояния асинхронного запроса.<br/> |
