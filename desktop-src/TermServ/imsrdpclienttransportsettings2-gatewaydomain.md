# <a name="imsrdpclienttransportsettings2gatewaydomain-property"></a>Свойство IMsRdpClientTransportSettings2:: Гатевайдомаин

Указывает или получает доменное имя пользователя, предоставленного на сервер шлюза удаленный рабочий стол (шлюз удаленных рабочих столов).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_GatewayDomain(
  [in]  BSTR proxyDomain
);

HRESULT get_GatewayDomain(
  [out] BSTR *pProxyDomain
);
```



## <a name="property-value"></a>Значение свойства

Доменное имя, которое предоставляется для подключения к серверу шлюза удаленных рабочих столов.

## <a name="error-codes"></a>Коды ошибок

При успешном выполнении возвращает значение **\_ ОК** .
