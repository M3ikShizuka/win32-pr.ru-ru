# <a name="imsrdpclientadvancedsettings8-interface"></a>Интерфейс IMsRdpClientAdvancedSettings8

Предоставляет методы и свойства, управляющие дополнительными параметрами элемента управления ActiveX удаленный рабочий стол.

Чтобы получить экземпляр этого интерфейса, используйте свойство [**IMsRdpClient8:: AdvancedSettings9**](imsrdpclient8-advancedsettings9.md) .

## <a name="members"></a>Элементы

Интерфейс **IMsRdpClientAdvancedSettings8** наследует от [**IMsRdpClientAdvancedSettings7**](imsrdpclientadvancedsettings7.md). **IMsRdpClientAdvancedSettings8** также имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Интерфейс **IMsRdpClientAdvancedSettings8** имеет следующие свойства.



| Свойство                                                                                  | Тип доступа           | Описание                                                           |
|:------------------------------------------------------------------------------------------|:----------------------|:----------------------------------------------------------------------|
| [**бандвидсдетектион**](imsrdpclientadvancedsettings8-bandwidthdetection.md)<br/> | Чтение/запись<br/> | Указывает, будут ли изменения пропускной способности обнаружены автоматически.<br/> |
