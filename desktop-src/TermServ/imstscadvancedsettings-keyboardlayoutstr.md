# <a name="imstscadvancedsettingskeyboardlayoutstr-property"></a>Свойство Имстскадванцедсеттингс:: Кэйбоардлайаутстр

Задает имя активного идентификатора языка ввода (прежнее название раскладки клавиатуры), используемого для соединения.

Если это свойство не задано, элемент управления использует макет по умолчанию, возвращаемый функцией [**жеткэйбоардлайаут**](/windows/desktop/api/winuser/nf-winuser-getkeyboardlayout) .

Это свойство доступно только на запись.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_KeyBoardLayoutStr(
  [in] BSTR KeyBoardLayoutStr
);
```



## <a name="property-value"></a>Значение свойства

Имя активного идентификатора языка ввода.

## <a name="error-codes"></a>Коды ошибок

При успешном выполнении возвращает значение **\_ ОК** .

## <a name="remarks"></a>Комментарии

Свойство является шестнадцатеричным числом из восьми цифр в форме строки. Младшие четыре цифры обозначают идентификатор языка, а верхние четыре цифры обозначают вариант клавиатуры на этом языке. Например, "00000409" представляет стандартную клавиатуру США по умолчанию, так как "0409" является идентификатором английского языка (США). Вариант по Двораку английской клавиатуры США имеет идентификатор "00010409". Доступные раскладки клавиатуры, перечисленные по идентификаторам раскладки клавиатуры, можно найти в реестре в разделе

```
HKEY_LOCAL_MACHINE
   SYSTEM
      ControlSet001
         Control
            Keyboard Layouts
```

Дополнительные сведения о веб-подключение к удаленному рабочему столу см. в разделе [требования для веб-подключение к удаленному рабочему столу](requirements-for-remote-desktop-web-connection.md).
