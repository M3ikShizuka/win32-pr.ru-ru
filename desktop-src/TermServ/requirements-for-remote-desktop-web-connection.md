# <a name="requirements-for-remote-desktop-web-connection"></a>Требования к веб-подключение к удаленному рабочему столу

Для веб-подключение к удаленному рабочему столу требуется две системы, настроенные следующим образом:

-   сервер узла под Windows и IIS 4,0 или более поздней версии.
-   клиент, который может скачивать и запускать элемент управления ActiveX удаленный рабочий стол. любой клиент, использующий Microsoft Internet Explorer 4,01 (желательно с пакетом обновления 2) или более поздней версии, может запустить элемент управления удаленный рабочий стол ActiveX.

Пользователь, устанавливающий программное обеспечение, должен обладать достаточными правами для установки программного обеспечения. например, в Windows XP пользователь должен быть членом группы опытных пользователей или администраторов.
