# <a name="imsrdpdrivecollection-interface"></a>Интерфейс Имсрдпдривеколлектион

Представляет коллекцию объектов Drive.

## <a name="members"></a>Элементы

Интерфейс **имсрдпдривеколлектион** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Имсрдпдривеколлектион** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **имсрдпдривеколлектион** содержит следующие методы.



| Метод                                                     | Описание                                                 |
|:-----------------------------------------------------------|:------------------------------------------------------------|
| [**рескандривес**](imsrdpdrivecollection-rescandrives.md) | Обновляет список объектов в коллекции.<br/> |



 

### <a name="properties"></a>Свойства

Интерфейс **имсрдпдривеколлектион** имеет следующие свойства.



| Свойство                                                              | Тип доступа          | Описание                                                  |
|:----------------------------------------------------------------------|:---------------------|:-------------------------------------------------------------|
| [**дривебиндекс**](imsrdpdrivecollection-drivebyindex.md)<br/> | Только для чтения<br/> | Извлекает диск по указанному индексу.<br/>       |
| [**дривекаунт**](imsrdpdrivecollection-drivecount.md)<br/>     | Только для чтения<br/> | Возвращает число объектов в коллекции.<br/> |
