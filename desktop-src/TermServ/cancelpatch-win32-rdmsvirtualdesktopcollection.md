# <a name="cancelpatch-method-of-the-win32_rdmsvirtualdesktopcollection-class"></a>Метод Канцелпатч \_ класса Win32 рдмсвиртуалдесктопколлектион

Отменяет задание подготовки обновлений программного обеспечения для виртуальных машин в коллекции виртуальных рабочих столов.

## <a name="syntax"></a>Синтаксис


```mof
uint32 CancelPatch();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение 0 при успешном выполнении, в противном случае возвращает код ошибки WMI.
