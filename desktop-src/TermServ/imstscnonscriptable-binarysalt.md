# <a name="imstscnonscriptablebinarysalt-property"></a>Свойство Имстскнонскриптабле:: Бинарисалт

Это свойство больше не доступно для использования.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_BinarySalt(
  [in]  BSTR newSalt
);

HRESULT get_BinarySalt(
  [out] BSTR *pSalt
);
```



## <a name="property-value"></a>Значение свойства

Новая часть двоичной соли для пароля в двоичном формате.

## <a name="error-codes"></a>Коды ошибок
