# <a name="imsrdpclient5msrdpclientshell-property"></a>Свойство IMsRdpClient5:: Мсрдпклиентшелл

Получает интерфейс параметра клиента с скриптом [**имсрдпклиентшелл**](imsrdpclientshell.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_MsRdpClientShell(
  [out] IMsRdpClientShell **ppLauncher
);
```



## <a name="property-value"></a>Значение свойства

Указатель интерфейса [**имсрдпклиентшелл**](imsrdpclientshell.md) .
