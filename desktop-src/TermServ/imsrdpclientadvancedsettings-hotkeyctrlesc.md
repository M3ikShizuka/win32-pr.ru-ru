# <a name="imsrdpclientadvancedsettingshotkeyctrlesc-property"></a>Свойство Имсрдпклиентадванцедсеттингс:: Хоткэйктрлеск

Указывает код виртуального ключа, добавляемый к ALT для определения замены сочетания клавиш для CTRL + ESC.

Это свойство допустимо, только если свойство [**кэйбоардхукмоде**](imsrdpclientsecuredsettings-keyboardhookmode.md) не включено.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_HotKeyCtrlEsc(
  [in]  LONG hotKeyCtrlEsc
);

HRESULT get_HotKeyCtrlEsc(
  [out] LONG *photKeyCtrlEsc
);
```



## <a name="property-value"></a>Значение свойства

Новый код виртуального ключа. **VK \_ Домашняя страница** — это значение по умолчанию, а в качестве результирующей последовательности — Alt + Home.

## <a name="error-codes"></a>Коды ошибок

При успешном выполнении возвращает значение **\_ ОК** .

## <a name="remarks"></a>Комментарии
