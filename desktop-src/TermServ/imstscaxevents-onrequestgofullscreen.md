# <a name="imstscaxeventsonrequestgofullscreen-method"></a>Метод Имстскаксевентс:: Онрекуестгофуллскрин

Вызывается, когда клиент запрашивает переключение в полноэкранный режим и вызывается метод [**имстскадванцедсеттингс::p UT \_ контаинерхандледфуллскрин**](imstscadvancedsettings-containerhandledfullscreen.md) , чтобы установить свойство **контаинерхандледфуллскрин** в ненулевое значение.

## <a name="syntax"></a>Синтаксис


```C++
void OnRequestGoFullScreen();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

В полноэкранном режиме, поддерживающем контейнеры, контейнер должен войти в стандартный полноэкранный режим в ответ на это событие.

Дополнительные сведения о веб-подключение к удаленному рабочему столу см. в разделе [требования для веб-подключение к удаленному рабочему столу](requirements-for-remote-desktop-web-connection.md).
