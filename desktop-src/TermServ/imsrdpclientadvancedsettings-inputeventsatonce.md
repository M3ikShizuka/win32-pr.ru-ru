# <a name="imsrdpclientadvancedsettingsinputeventsatonce-property"></a>Свойство Имсрдпклиентадванцедсеттингс:: Инпутевентсатонце

Данное свойство не поддерживается.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_InputEventsAtOnce(
  [in]  LONG inputEventsAtOnce
);

HRESULT get_InputEventsAtOnce(
  [out] LONG *pinputEventsAtOnce
);
```



## <a name="property-value"></a>Значение свойства

Новое число событий ввода. Значение по умолчанию равно 10.

## <a name="error-codes"></a>Коды ошибок

Возвращает **\_ значение false**.
