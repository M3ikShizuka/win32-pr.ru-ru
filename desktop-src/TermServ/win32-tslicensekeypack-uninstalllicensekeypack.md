# <a name="uninstalllicensekeypack-method-of-the-win32_tslicensekeypack-class"></a>Метод Унинсталллиценсекэйпакк \_ класса Win32 тслиценсекэйпакк

Удаляет службы удаленных рабочих столов пакет лицензионных ключей.

## <a name="syntax"></a>Синтаксис


```mof
uint32 UninstallLicenseKeyPack(
  [in] unit32 ProductVersion,
  [in] unit32 ProductType,
  [in] uint32 LicenseCount
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ProductVersion* \[ окне\]
</dt> <dd>

Идентификатор версии продукта для службы удаленных рабочих столов пакета лицензионных ключей.

<dt>

0
</dt> <dd>

Не поддерживается.

</dd> <dt>

1
</dt> <dd>

Не поддерживается.

</dd> <dt>

2
</dt> <dd>

Windows Server 2008

</dd> </dl> </dd> <dt>

*Продукттипе* \[ окне\]
</dt> <dd>

Тип продукта службы удаленных рабочих столов пакета лицензионных ключей.

<dt>

0
</dt> <dd>

Тип продукта пакета лицензионных ключей службы удаленных рабочих столов для каждого устройства. Таким образом, каждому устройству, которое подключается к серверу узла сеансов удаленных рабочих столов, должна быть назначена лицензия.

</dd> <dt>

1
</dt> <dd>

Тип продукта пакета лицензионных ключей службы удаленных рабочих столов для каждого пользователя. Поэтому у каждого пользователя, подключающегося к серверу узла сеансов удаленных рабочих столов, должна быть лицензия.

</dd> <dt>

2
</dt> <dd>

Недопустимый тип продукта.

</dd> </dl> </dd> <dt>

*Лиценсекаунт* \[ окне\]
</dt> <dd>

Число удаляемых лицензий.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если метод завершается с ошибкой, он возвращает ноль. Если метод завершился неудачно, он возвращает ненулевое значение. Список кодов ошибок см. в разделе [службы удаленных рабочих столов коды ошибок поставщика WMI](terminal-services-wmi-provider-error-codes.md).
