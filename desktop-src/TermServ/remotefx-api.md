# <a name="remotefx-media-redirection-api"></a>API перенаправления мультимедиа RemoteFX

API перенаправления мультимедиа RemoteFX используется в сеансе удаленный рабочий стол для обнаружения областей сервера, на которых отображается быстрое изменение содержимого, например видео. Это содержимое может быть закодировано видео и отправлено клиенту в закодированном формате.
