# <a name="imsrdppreferredredirectioninfo-interface"></a>Интерфейс Имсрдппреферредредиректионинфо

Предоставляет свойство для управления использованием сервера перенаправления.

## <a name="members"></a>Элементы

Интерфейс **имсрдппреферредредиректионинфо** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Имсрдппреферредредиректионинфо** также имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Элемент Property

Интерфейс **имсрдппреферредредиректионинфо** имеет следующие свойства.



| Свойство.                                                                                               | Тип доступа           | Описание                                                          |
|:-------------------------------------------------------------------------------------------------------|:----------------------|:---------------------------------------------------------------------|
| [**усередиректионсервернаме**](imsrdppreferredredirectioninfo-useredirectionservername.md)<br/> | Чтение/запись<br/> | Возвращает или задает значение, указывающее, следует ли использовать имя сервера перенаправления.<br/> |
