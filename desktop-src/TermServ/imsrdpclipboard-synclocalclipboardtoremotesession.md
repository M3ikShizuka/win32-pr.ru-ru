# <a name="imsrdpclipboardsynclocalclipboardtoremotesession-method"></a>Метод Имсрдпклипбоард:: Синклокалклипбоардторемотесессион

Синхронизирует локальный буфер обмена с удаленным сеансом.

## <a name="syntax"></a>Синтаксис

```C++
HRESULT SyncLocalClipboardToRemoteSession();
```

## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

В случае успеха возвратите значение **\_ ОК** .
