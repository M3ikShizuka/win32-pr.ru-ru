# <a name="remote-desktop-connection-broker-classes"></a>Классы подключение к удаленному рабочему столу Broker

Поставщик WMI подключение к удаленному рабочему столу Broker (RDCB) предоставляет следующие классы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**\_Сессионброкерфарм Win32**](win32-sessionbrokerfarm.md)
</dt> <dd>

Определяет запрос для фермы посредников сеансов.

</dd> <dt>

[**\_Сессионброкерфармаккаунт Win32**](win32-sessionbrokerfarmaccount.md)
</dt> <dd>

Класс [**Win32 \_ сессионброкерфармаккаунт**](win32-sessionbrokerfarmaccount.md) больше не доступен для использования в Windows Server 2012.

</dd> <dt>

[**\_Сессионброкерсервицепропертиес Win32**](win32-sessionbrokerserviceproperties.md)
</dt> <dd>

Определяет запрос для службы посредника сеансов.

</dd> <dt>

[**\_Сессионброкертаржет Win32**](win32-sessionbrokertarget.md)
</dt> <dd>

Определяет запрос для целевого объекта посредника сеанса.

</dd> <dt>

[**\_Сессионброкертаржетевент Win32**](win32-sessionbrokertargetevent.md)
</dt> <dd>

Представляет изменение целевого объекта посредника сеансов.

</dd> <dt>

[**\_Сессиондиректориклустер Win32**](win32-sessiondirectorycluster.md)
</dt> <dd>

Предоставляет свойства для просмотра свойств фермы в RDCB.

</dd> <dt>

[**\_Сессиондиректорисервер Win32**](win32-sessiondirectoryserver.md)
</dt> <dd>

Предоставляет свойства для просмотра свойств сервера RDCB.

</dd> <dt>

[**\_Сессиондиректорисессион Win32**](win32-sessiondirectorysession.md)
</dt> <dd>

Предоставляет свойства для просмотра свойств сеанса RDCB.

</dd> <dt>

[**\_Сессиондиректоривиртуалдесктопсервер Win32**](win32-sessiondirectoryvirtualdesktopserver.md)
</dt> <dd>

Представляет Узел виртуализации удаленных рабочих столов сервер (узел виртуализации удаленных рабочих столов), присоединенный к посреднику сеансов.

</dd> <dt>

[**\_Сессиондиректоривммплугин Win32**](win32-sessiondirectoryvmmplugin.md)
</dt> <dd>

Представляет подключаемый модуль Virtual Machine Manager (VMM), зарегистрированный в посреднике сеансов.

</dd> </dl>

> [!Note]  
> в Windows Server 2008 R2 имя посредника сеансов служб терминалов изменено на RDCB.
