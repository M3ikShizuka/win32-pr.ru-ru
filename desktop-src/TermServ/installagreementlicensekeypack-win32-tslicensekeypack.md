# <a name="installagreementlicensekeypack-method-of-the-win32_tslicensekeypack-class"></a>Метод Инсталлагриментлиценсекэйпакк \_ класса Win32 тслиценсекэйпакк

Устанавливает службы удаленных рабочих столов пакет лицензионных ключей, приобретенный по лицензионному соглашению, и автоматически подключается через Интернет для проверки лицензии на пакет ключей.

## <a name="syntax"></a>Синтаксис


```mof
uint32 InstallAgreementLicenseKeyPack(
  [in]  uint32 AgreementType,
  [in]  string sAgreementNumber,
  [in]  uint32 ProductVersion,
  [in]  uint32 ProductType,
  [in]  uint32 LicenseCount,
  [out] uint32 KeyPackId
);
```

## <a name="parameters"></a>Параметры

*AgreementType* \[ окне\]

Тип соглашения.

| Значение | Описание |
|-------|-------------|
| 0 | Пакет лицензионных ключей относится к выбранному соглашению о корпоративном лицензировании (для клиентов с 250 или более компьютеров). Параметр *сагриментнумбер* — номер регистрации (семь цифр), найденный в подписанном соглашении. |
| 1 | пакет лицензионных ключей относится к Enterprise соглашению корпоративного лицензирования для клиентов с 250 или более компьютеров. Параметр *сагриментнумбер* — номер регистрации (семь цифр), найденный в подписанном соглашении. |
| 2 | Пакет лицензионных ключей относится к соглашению о корпоративном лицензировании для образовательных учреждений. Параметр *сагриментнумбер* — номер регистрации (семь цифр), найденный в подписанном соглашении. |
| 3 | Пакет лицензионных ключей входит в состав учебного соглашения корпоративного лицензирования для основного и дополнительного учреждений. Параметр *сагриментнумбер* — номер регистрации (семь цифр), найденный в подписанном соглашении. |
| 4 | Пакет лицензионных ключей предоставляется лицензионным соглашением поставщика услуг для поставщиков услуг по ежемесячному лицензированию программного обеспечения Майкрософт. Параметр *сагриментнумбер* — номер регистрации (семь цифр), найденный в подписанном соглашении. |
| 5 | Пакет лицензионных ключей относится к другому лицензионному соглашению, такому как открытая стоимость, открытая лицензия на несколько лет и открытая лицензия на подписку. Параметр *сагриментнумбер* — это номер соглашения, предоставленный в сведениях о программе. |

*сагриментнумбер* \[ окне\]

Номер соглашения или номер регистрации. Параметр *сагриментнумбер* — это семь цифр числовая строка без дефисов.

*ProductVersion* \[ окне\]

Номер версии продукта.

| Значение | Описание |
|-------|-------------|
| 0 | Не поддерживается |
| 1 | Не поддерживается |
| 2 | Windows server 2008/Windows server 2008 R2 |
| 4 | Windows Server 2012 и Windows Server 2012 R2 |
| 5 | Windows Server 2016 |
| 6 | Windows Server 2019 |

*Продукттипе* \[ окне\]

Тип продукта.

| Значение | Описание |
|-------|-------------|
| 0 | Тип продукта пакета лицензионных ключей службы удаленных рабочих столов для каждого устройства. Таким образом, каждому устройству, которое подключается к серверу узла сеансов удаленных рабочих столов, должна быть назначена лицензия. |
| 1 | Тип продукта пакета лицензионных ключей службы удаленных рабочих столов для каждого пользователя. Поэтому у каждого пользователя, подключающегося к серверу узла сеансов удаленных рабочих столов, должна быть лицензия. |
| 2 | Недопустимый тип продукта. |

*Лиценсекаунт* \[ окне\]

Число устанавливаемых лицензий.

*Кэйпаккид* \[ заполняет\]

Получает идентификатор пакета ключей.

## <a name="return-value"></a>Возвращаемое значение

Если метод завершается с ошибкой, он возвращает ноль. Если метод завершился неудачно, он возвращает ненулевое значение. Список кодов ошибок см. в разделе [службы удаленных рабочих столов коды ошибок поставщика WMI](terminal-services-wmi-provider-error-codes.md).
