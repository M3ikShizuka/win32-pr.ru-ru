# <a name="imsrdpclientadvancedsettingssmoothscroll-property"></a>Свойство Имсрдпклиентадванцедсеттингс:: Смусскролл

Данное свойство не поддерживается.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_SmoothScroll(
  [in]  LONG smoothScroll
);

HRESULT get_SmoothScroll(
  [out] LONG *psmoothScroll
);
```



## <a name="property-value"></a>Значение свойства

Присвойте этому параметру значение 0, чтобы отключить плавную прокрутку или ненулевое число, чтобы включить плавную прокрутку.

## <a name="error-codes"></a>Коды ошибок

Возвращает **\_ значение false**.
