# <a name="iconnectionbrokerclient-interface"></a>Интерфейс Иконнектионброкерклиент

Предоставляет методы, необходимые для использования клиента компонента подключение к удаленному рабочему столу Broker.

Этот интерфейс реализуется системой. Чтобы получить экземпляр этого интерфейса, используйте функцию [**кбкреатеклиентинстанце**](cbcreateclientinstance.md) .

## <a name="members"></a>Элементы

Интерфейс **иконнектионброкерклиент** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иконнектионброкерклиент** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иконнектионброкерклиент** содержит следующие методы.



| Метод                                                         | Описание                                                                                          |
|:---------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| [**жеттаржетинфо**](iconnectionbrokerclient-gettargetinfo.md) | Запрашивает сведения о конечном компьютере, на котором должно быть перенаправлено подключение.<br/> |
