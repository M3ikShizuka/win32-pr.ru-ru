# <a name="iremotedesktopclientactions-methods"></a>Методы Иремотедесктопклиентактионс

Интерфейс [**иремотедесктопклиентактионс**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclientactions) предоставляет следующие методы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**Метод Ексекутеремотеактион**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-executeremoteaction)
</dt> <dd>

Вызывает выполнение действия в удаленном сеансе.

</dd> <dt>

[**Метод Snapshot**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-getsnapshot)
</dt> <dd>

Вызывает создание моментального снимка рабочего стола клиента контейнера приложения протокол удаленного рабочего стола (RDP).

</dd> <dt>

[**Метод Ресумескринупдатес**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-resumescreenupdates)
</dt> <dd>

Возобновляет обновления экрана, отправляемые клиенту.

</dd> <dt>

[**Метод Суспендскринупдатес**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientactions-suspendscreenupdates)
</dt> <dd>

Приостанавливает обновления экрана, отправляемые клиенту.

</dd> </dl>
