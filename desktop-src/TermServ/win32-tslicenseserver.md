# <a name="win32_tslicenseserver-class"></a>\_Класс Win32 тслиценсесервер

Предоставляет методы и свойства для просмотра и настройки лицензирования удаленный рабочий стол (лицензирования удаленных рабочих столов) на сервере лицензирования удаленный рабочий стол.

## <a name="syntax"></a>Синтаксис

``` syntax
[singleton, dynamic, provider("Win32_WIN32_TERMSERVLICENSING_Prov"), AMENDMENT]
class Win32_TSLicenseServer
{
  string FirstName;
  string LastName;
  string Company;
  string CountryRegion;
  string eMail;
  string OrgUnit;
  string Address;
  string City;
  string State;
  string PostalCode;
  uint32 ServerRole;
  string DatabasePath;
  string ProductId;
  string Version;
  string VersionNumber;
};
```

## <a name="remarks"></a>Remarks

Этот класс является [Singleton](/windows/desktop/WmiSdk/standard-wmi-qualifiers) -классом и может иметь только один экземпляр. Все методы, содержащиеся в этом классе, являются статическими.

Для использования этого класса необходимо быть членом группы администраторов. Если вызывающий объект не является членом группы администраторов, возвращаемые свойства будут пустыми.

файлы MOF-файл (MOF) содержат определения для классов инструментарий управления Windows (WMI) (WMI). файлы MOF не устанавливаются в составе пакета средств разработки программного обеспечения Microsoft Windows Software Development Kit (SDK). Они устанавливаются на сервере при добавлении связанной роли с помощью диспетчер сервера. Дополнительные сведения о файлах MOF см. в разделе [MOF-файл (MOF)](/windows/desktop/WmiSdk/managed-object-format--mof-).
