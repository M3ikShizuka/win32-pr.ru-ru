# <a name="itssbprovisioning-methods"></a>Методы Итссбпровисионинг

Интерфейсы [**итссбпровисионинг**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbprovisioning) предоставляют следующие методы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**Метод CancelJob**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioning-canceljob)
</dt> <dd>

Отменяет задание подготовки.

</dd> <dt>

[**Метод Креатевиртуалмачинес**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioning-createvirtualmachines)
</dt> <dd>

Асинхронно создает виртуальную машину.

</dd> <dt>

[**Метод Делетевиртуалмачинес**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioning-deletevirtualmachines)
</dt> <dd>

Асинхронно удаляет виртуальную машину.

</dd> <dt>

[**Метод Патчвиртуалмачинес**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbprovisioning-patchvirtualmachines)
</dt> <dd>

Асинхронно выполняет исправления виртуальной машины.

</dd> </dl>
