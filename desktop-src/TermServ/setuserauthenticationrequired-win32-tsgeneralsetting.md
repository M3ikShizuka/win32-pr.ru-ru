# <a name="setuserauthenticationrequired-method-of-the-win32_tsgeneralsetting-class"></a>Метод Сетусераусентикатионрекуиред \_ класса Win32 тсженералсеттинг

Включает или отключает требование, которое пользователи должны пройти проверку подлинности во время соединения, задав значение свойства **усераусентикатионрекуиред** в классе [**Win32 \_ тсженералсеттинг**](win32-tsgeneralsetting.md) . Если **значение равно true**, то при подключении **усераусентикатионрекуиред** требуется проверка подлинности пользователя во время подключения, чтобы повысить защиту сервера от сетевых атак. Подключаться могут только клиенты протокол удаленного рабочего стола (RDP), поддерживающие RDP версии 6,0 или выше. Чтобы избежать нарушений работы удаленных пользователей, перед включением свойства рекомендуется развернуть клиенты RDP, поддерживающие соответствующую версию протокола.

## <a name="syntax"></a>Синтаксис


```mof
uint32 SetUserAuthenticationRequired(
  [in] uint32 UserAuthenticationRequired
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Усераусентикатионрекуиред* \[ окне\]
</dt> <dd>

Указывает, требуется ли проверка подлинности пользователя.

<dt>

0 (0x0)
</dt> <dd>

Отключить требование, когда пользователь должен пройти проверку подлинности

</dd> <dt>

1 (0x1)
</dt> <dd>

Включает требование, которое пользователь должен пройти проверку подлинности.

</dd> </dl> </dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение 0 при успешном выполнении, в противном случае возвращает код ошибки WMI. Список этих значений см. в разделе [службы удаленных рабочих столов коды ошибок поставщика WMI](terminal-services-wmi-provider-error-codes.md) .

## <a name="remarks"></a>Комментарии

Файлы MOF-файл (MOF) содержат определения для классов инструментарий управления Windows (WMI) (WMI). MOF-файлы не устанавливаются в составе пакета средств разработки программного обеспечения Microsoft Windows (SDK). Они устанавливаются на сервере при добавлении связанной роли с помощью диспетчер сервера. Дополнительные сведения о файлах MOF см. в разделе [MOF-файл (MOF)](/windows/desktop/WmiSdk/managed-object-format--mof-).
