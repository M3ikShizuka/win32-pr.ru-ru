# <a name="imstscaxeventsonautoreconnected-method"></a>Метод Имстскаксевентс:: Онаутореконнектед

Вызывается при автоматическом повторном подключении клиентского элемента управления к удаленному сеансу.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT OnAutoReconnected();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Если этот метод завершается успешно, возвращается значение **S \_ ОК**. В противном случае возвращается код ошибки **HRESULT** .
