# <a name="imsrdpdrive-interface"></a>Интерфейс Имсрдпдриве

Содержит сведения об объекте Drive.

## <a name="members"></a>Элементы

Интерфейс **имсрдпдриве** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Имсрдпдриве** также имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Интерфейс **имсрдпдриве** имеет следующие свойства.



| Свойство                                                            | Тип доступа           | Описание                                              |
|:--------------------------------------------------------------------|:----------------------|:---------------------------------------------------------|
| [**Имя**](imsrdpdrive-name.md)<br/>                         | Только для чтения<br/>  | Возвращает имя диска.<br/>              |
| [**редиректионстате**](imsrdpdrive-redirectionstate.md)<br/> | Чтение/запись<br/> | Указывает состояние перенаправления диска.<br/> |
