# <a name="remoteapp-and-desktop-connection-management-service-api-reference"></a>Справочник по API службы управление подключениями к удаленным рабочим столам и приложениям RemoteApp

API службы управление подключениями к удаленным рабочим столам и приложениям RemoteApp поддерживает интерфейс и структуру, предоставляющие сведения о ресурсах, доступных пользователям удаленных подключений к удаленным рабочим столам и приложениям RemoteApp. Реализация этого интерфейса может выполнять пользовательскую фильтрацию ресурсов и обеспечивать поддержку типов файлов, которые в настоящее время не поддерживаются. Дополнительные сведения см. в следующих статьях.
