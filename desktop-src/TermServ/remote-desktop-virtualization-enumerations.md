# <a name="remote-desktop-virtualization-enumerations"></a>удаленный рабочий стол перечисления виртуализации

Удаленный рабочий стол API виртуализации поддерживает следующие типы перечислений.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**\_тип изменения целевого объекта \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_change_type)
</dt> <dd>

Указывает тип изменения, произошедшего в целевом объекте.

</dd> <dt>

[**ЦЕЛЕВое \_ состояние**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_state)
</dt> <dd>

Указывает состояние целевого объекта.

</dd> <dt>

[**Тип целевого объекта \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_type)
</dt> <dd>

Указывает, принадлежит ли целевой объект пулу или ферме.

</dd> <dt>

[**\_состояние тссессион**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-tssession_state)
</dt> <dd>

Указывает состояние сеанса.

</dd> <dt>

[**ТССД \_ AddrV46Type**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-tssd_addrv46type)
</dt> <dd>

Указывает тип IP-адреса.

</dd> <dt>

[**TS \_ SB — \_ Сортировка \_ по**](/windows/desktop/api/sbtsv/ne-sbtsv-ts_sb_sort_by)
</dt> <dd>

Задает порядок сортировки. Он используется в качестве параметра в методе [**енумератетаржетс**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbresourcepluginstore-enumeratetargets) .

</dd> </dl>
