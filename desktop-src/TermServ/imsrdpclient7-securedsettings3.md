# <a name="imsrdpclient7securedsettings3-property"></a>Свойство IMsRdpClient7:: SecuredSettings3

Извлекает объект, который поддерживает интерфейс [**IMsRdpClientSecuredSettings2**](imsrdpclientsecuredsettings2.md) .

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_SecuredSettings3(
  [out] IMsRdpClientSecuredSettings2 **ppSecuredSettings
);
```



## <a name="property-value"></a>Значение свойства

Адрес указателя интерфейса [**IMsRdpClientSecuredSettings2**](imsrdpclientsecuredsettings2.md) , который получает объект.
