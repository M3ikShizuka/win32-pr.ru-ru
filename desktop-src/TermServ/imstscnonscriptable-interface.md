# <a name="imstscnonscriptable-interface"></a>Интерфейс Имстскнонскриптабле

Содержит свойства и методы, связанные с применением пароля к элементу управления ActiveX удаленный рабочий стол.

В основном используется интерфейс **имстскнонскриптабле** для настройки автоматического доступа к паролю удаленный рабочий стол для серверов узла сеансов сеансов (удаленных рабочих столов) в сценариях, где удаленный рабочий стол элемент управления ActiveX размещается в пользовательском контейнере. При настройке автоматического входа пользователь не получает диалоговое окно входа в Windows во время подключения.

На некоторых платформах методы интерфейса **имстскнонскриптабле** можно использовать для указания паролей в одном из трех поддерживаемых форматов:

-   Открытый текст
-   Переносимая кодировка
-   Двоичный (непортовый) закодированный

Обратите внимание, что пароль в закодированном формате состоит из двух частей:

-   Часть закодированного пароля.
-   Часть значения соли.

Для задания закодированного пароля требуются обе части. Ни часть зашифрованного пароля, ни часть значения соли не должны считаться безопасными.

Доступ с помощью скрипта к паролям с открытым текстом можно получить через свойство [**клеартекстпассворд**](imsrdpclientadvancedsettings-cleartextpassword.md) интерфейса, поддерживающего сценарии [**имсрдпклиентадванцедсеттингс**](imsrdpclientadvancedsettings-interface.md).

Доступ к интерфейсу **имстскнонскриптабле** можно получить только через таблицу vtable.

## <a name="members"></a>Элементы

Интерфейс **имстскнонскриптабле** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Имстскнонскриптабле** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **имстскнонскриптабле** содержит следующие методы.



| Метод                                                     | Описание                                           |
|:-----------------------------------------------------------|:------------------------------------------------------|
| [**ресетпассворд**](imstscnonscriptable-resetpassword.md) | Сбрасывает все состояния паролей в элементе управления.<br/> |



 

### <a name="properties"></a>Свойства

Интерфейс **имстскнонскриптабле** имеет следующие свойства.



| Свойство                                                                      | Тип доступа           | Описание                                                                  |
|:------------------------------------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------|
| [**бинарипассворд**](imstscnonscriptable-binarypassword.md)<br/>       | Чтение/запись<br/> | Данное свойство не поддерживается.<br/>                                   |
| [**бинарисалт**](imstscnonscriptable-binarysalt.md)<br/>               | Чтение/запись<br/> | Данное свойство не поддерживается.<br/>                                   |
| [**клеартекстпассворд**](imstscnonscriptable-cleartextpassword.md)<br/> | Только на запись<br/> | Удаленный рабочий стол пароль элемента управления ActiveX в текстовом формате.<br/> |
| [**портаблепассворд**](imstscnonscriptable-portablepassword.md)<br/>   | Чтение/запись<br/> | Данное свойство не поддерживается.<br/>                                   |
| [**портаблесалт**](imstscnonscriptable-portablesalt.md)<br/>           | Чтение/запись<br/> | Данное свойство не поддерживается.<br/>                                   |



 

## <a name="remarks"></a>Комментарии

Указание пароля для элемента управления ActiveX удаленный рабочий стол является необязательным. При указании пароля для элемента управления следует применить к элементу управления только один из трех предыдущих форматов, прежде чем инициировать соединение с помощью вызова метода [**Connect**](imstscax-connect.md) .

> [!Note]  
> Вы также можете включить автоматический вход на сервер с помощью средства настройки службы удаленных рабочих столов (TSCC. msc). Администратор может использовать это средство для назначения определенного пароля для подключения в ситуации, когда требуется автоматический вход в систему.

 

Дополнительные сведения о веб-подключение к удаленному рабочему столу см. в разделе [требования для веб-подключение к удаленному рабочему столу](requirements-for-remote-desktop-web-connection.md).

Интерфейс **имстскнонскриптабле** расширен следующими интерфейсами. каждый новый интерфейс наследует все методы и свойства предыдущих интерфейсов:

-   [**имсрдпклиентнонскриптабле**](imsrdpclientnonscriptable-interface.md)
-   [**IMsRdpClientNonScriptable2**](imsrdpclientnonscriptable2.md)
-   [**IMsRdpClientNonScriptable3**](imsrdpclientnonscriptable3.md)
-   [**IMsRdpClientNonScriptable4**](imsrdpclientnonscriptable4.md)
-   [**IMsRdpClientNonScriptable5**](imsrdpclientnonscriptable5.md)
