# <a name="imsrdpclientnonscriptable2-interface"></a>Интерфейс IMsRdpClientNonScriptable2

Предоставляет доступ к необрабатываемым свойствам удаленного сеанса клиента на удаленный рабочий стол элементе управления ActiveX. Является производным от интерфейса [**имсрдпклиентнонскриптабле**](imsrdpclientnonscriptable-interface.md) . Доступ к методам этого интерфейса возможен только через таблицу vtable; они недоступны для использования в сценариях клиентов.

## <a name="members"></a>Элементы

Интерфейс **IMsRdpClientNonScriptable2** наследует от [**имсрдпклиентнонскриптабле**](imsrdpclientnonscriptable-interface.md). **IMsRdpClientNonScriptable2** также имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Интерфейс **IMsRdpClientNonScriptable2** имеет следующие свойства.



| Свойство                                                                                   | Тип доступа           | Описание                                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------------------|:----------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**уипарентвиндовхандле**](imsrdpclientnonscriptable2-uiparentwindowhandle.md)<br/> | Чтение/запись<br/> | Маркер окна, который будет родительским окном для элемента управления. Это позволяет корректно модальным окнам, отображаемым элементом управления, по отношению к любым окнам, отображаемым родительским приложением.<br/> |



 

## <a name="remarks"></a>Комментарии

Этот интерфейс расширен следующими интерфейсами, при этом каждый новый интерфейс наследует все методы и свойства предыдущих интерфейсов:

-   [**IMsRdpClientNonScriptable3**](imsrdpclientnonscriptable3.md)
-   [**IMsRdpClientNonScriptable4**](imsrdpclientnonscriptable4.md)
-   [**IMsRdpClientNonScriptable5**](imsrdpclientnonscriptable5.md)

Дополнительные сведения о веб-подключение к удаленному рабочему столу см. в разделе [требования для веб-подключение к удаленному рабочему столу](requirements-for-remote-desktop-web-connection.md).
