# <a name="adsi-extension-for-remote-desktop-services-user-configuration"></a>Расширение ADSI для пользовательской конфигурации службы удаленных рабочих столов

Расширение интерфейса службы Active Directory (ADSI) — это специальный COM-объект, позволяющий расширение COM-объекта. Расширения ADSI могут добавлять функции в объект каталога и упростить задачи администрирования.

Расширение ADSI для пользовательской конфигурации службы удаленных рабочих столов расширяет объект ADSI, позволяющий администраторам домена создавать сценарии, автоматизирующие обслуживание свойств пользователей, относящихся к службы удаленных рабочих столов.

Расширение ADSI для пользовательской конфигурации службы удаленных рабочих столов входит в Windows Server.

-   [Справочник по расширению ADSI для службы удаленных рабочих столов пользовательской конфигурации](reference-for-the-adsi-extension-for-terminal-services-user-configuration.md)
