# <a name="cim_elementsetting-class-remote-desktop-services"></a>Класс CIM_ElementSetting (службы удаленных рабочих столов)

Представляет ассоциацию между управляемыми системными элементами и определенным для них классом параметров.

Следующий синтаксис упрощен из MOF-кода и включает все унаследованные свойства.

## <a name="syntax"></a>Синтаксис

``` syntax
[Abstract, UUID("{8502C518-5FBB-11D2-AAC1-006008C78BC7}"), AMENDMENT]
class CIM_ElementSetting : CIM_ManagedSystemElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
};
```
