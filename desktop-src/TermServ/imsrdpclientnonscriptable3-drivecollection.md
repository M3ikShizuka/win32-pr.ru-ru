# <a name="imsrdpclientnonscriptable3drivecollection-property"></a>IMsRdpClientNonScriptable3: свойство Ривеколлектион:D

Извлекает коллекцию объектов Drive для перенаправления.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_DriveCollection(
  [out] IMsRdpDriveCollection **ppDriveCollection
);
```



## <a name="property-value"></a>Значение свойства

Извлекает коллекцию объектов Drive типа [**имсрдпдриве**](imsrdpdrive.md).
