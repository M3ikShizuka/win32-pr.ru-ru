# <a name="iworkspacereportmessage-methods"></a>Методы Иворкспацерепортмессаже

Интерфейс [**иворкспацерепортмессаже**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacereportmessage) поддерживает следующие методы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**Метод Регистереррорлогмессаже**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-registererrorlogmessage)
</dt> <dd>

Регистрирует указанное сообщение об ошибке для использования в пользовательском интерфейсе.

</dd> <dt>

[**Метод Исеррормессажерегистеред**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-iserrormessageregistered)
</dt> <dd>

Определяет, зарегистрировано ли указанное сообщение об ошибке в указанной рабочей области.

</dd> <dt>

[**Метод Регистерерроревент**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacereportmessage-registererrorevent)
</dt> <dd>

Регистрирует указанное сообщение о событии ошибки для использования в пользовательском интерфейсе.

</dd> </dl>
