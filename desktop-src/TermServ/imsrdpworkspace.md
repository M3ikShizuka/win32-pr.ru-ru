# <a name="imsrdpworkspace-interface"></a>Интерфейс Имсрдпворкспаце

Предоставляет методы, управляющие учетными данными и соединениями подключения к удаленным рабочим столам и приложениям RemoteApp. Этот интерфейс реализуется элементом управления службы удаленных рабочих столов Веб-доступ. Этот элемент управления является оболочкой для подключение к удаленному рабочему столу клиента (MsTscAx.dll), а также прокси-сервера среды выполнения подключений RemoteApp и Desktop Connections (Tswbprxy.exe).

## <a name="members"></a>Элементы

Интерфейс **имсрдпворкспаце** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Имсрдпворкспаце** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **имсрдпворкспаце** содержит следующие методы.



| Метод                                                                                   | Описание                                                                                                                                                           |
|:-----------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**клеарворкспацекредентиал**](/previous-versions/windows/desktop/legacy/ee351596(v=vs.85))             | Удаляет учетные данные пользователя, связанные с указанным ИДЕНТИФИКАТОРом подключения.<br/>                                                                                  |
| [**дисконнектворкспаце**](/previous-versions/windows/desktop/legacy/ee351597(v=vs.85))                       | Отключает все существующие соединения, связанные с указанным ИДЕНТИФИКАТОРом подключения, и удаляет соответствующие учетные данные пользователя из хранилища учетных данных.<br/> |
| [**исворкспацекредентиалспеЦифиед**](/previous-versions/windows/desktop/legacy/ee351598(v=vs.85)) | Определяет, существуют ли учетные данные пользователя для указанного идентификатора соединения.<br/>                                                                                 |
| [**исворкспацессоенаблед**](/previous-versions/windows/desktop/legacy/ee351599(v=vs.85))                   | Определяет, включена ли поддержка единого входа (SSO) для подключения к удаленным рабочим столам и приложениям RemoteApp.<br/>                                                                   |
| [**С проверкой подлинности**](/previous-versions/windows/desktop/legacy/ee351600(v=vs.85))                               | Помечает проверку подлинности учетных данных пользователя для идентификатора подключения, а затем отображает уведомление о подключении в области уведомлений панели задач. <br/>     |
| [**стартворкспаце**](/previous-versions/windows/desktop/legacy/ee351601(v=vs.85))                                 | Связывает учетные данные пользователя и сертификаты с ИДЕНТИФИКАТОРом подключения.<br/>                                                                                         |
