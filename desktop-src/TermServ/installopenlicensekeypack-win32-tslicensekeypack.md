# <a name="installopenlicensekeypack-method-of-the-win32_tslicensekeypack-class"></a>Метод Инсталлопенлиценсекэйпакк \_ класса Win32 тслиценсекэйпакк

Устанавливает лицензию Open License службы удаленных рабочих столов лицензионный пакет.

## <a name="syntax"></a>Синтаксис


```mof
uint32 InstallOpenLicenseKeyPack(
  [in]  string sLicenseNumber,
  [in]  string sAuthorizationNumber,
  [in]  uint32 ProductVersion,
  [in]  uint32 ProductType,
  [in]  uint32 LicenseCount,
  [out] uint32 KeyPackId
);
```

## <a name="parameters"></a>Параметры

*слиценсенумбер* \[ окне\]

8-символьная числовая строка, предоставленная вместе с пакетом лицензионных ключей. Параметр *слиценсенумбер* не может содержать дефисы.

*саусоризатионнумбер* \[ окне\]

Строка, состоящая из 15 символов и имеющая ключ лицензии. Параметр *саусоризатионнумбер* не может содержать дефисы.

*ProductVersion* \[ окне\]

Номер версии продукта.

| Значение | Описание |
|-------|-------------|
| 0 | Не поддерживается |
| 1 | Не поддерживается |
| 2 | Windows Server 2008/Windows Server 2008 R2 |
| 4 | Windows Server 2012/Windows Server 2012 R2 |
| 5 | Windows Server 2016 |
| 6 | Windows Server 2019 |

*Продукттипе* \[ окне\]
</dt> <dd>

Тип продукта.

| Значение | Описание |
|-------|-------------|
| 0 | Тип продукта пакета лицензионных ключей службы удаленных рабочих столов для каждого устройства. Таким образом, каждому устройству, которое подключается к серверу узла сеансов удаленных рабочих столов, должна быть назначена лицензия. |
| 1 | Тип продукта пакета лицензионных ключей службы удаленных рабочих столов для каждого пользователя. Поэтому у каждого пользователя, подключающегося к серверу узла сеансов удаленных рабочих столов, должна быть лицензия. |
| 2 | Недопустимый тип продукта. |

*Лиценсекаунт* \[ окне\]

Число устанавливаемых лицензий.

*Кэйпаккид* \[ заполняет\]

Получает идентификатор пакета ключей.

## <a name="return-value"></a>Возвращаемое значение

Если метод завершается с ошибкой, он возвращает ноль. Если метод завершился неудачно, он возвращает ненулевое значение. Список кодов ошибок см. в разделе [службы удаленных рабочих столов коды ошибок поставщика WMI](terminal-services-wmi-provider-error-codes.md).

## <a name="remarks"></a>Комментарии

Для вызова этого метода необходимо быть членом группы администраторов.

Файлы MOF-файл (MOF) содержат определения для классов инструментарий управления Windows (WMI) (WMI). MOF-файлы не устанавливаются в составе пакета средств разработки программного обеспечения Microsoft Windows (SDK). Они устанавливаются на сервере при добавлении связанной роли с помощью диспетчер сервера. Дополнительные сведения о файлах MOF см. в разделе [MOF-файл (MOF)](/windows/desktop/WmiSdk/managed-object-format--mof-).
