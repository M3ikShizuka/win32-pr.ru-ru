# <a name="imsrdpclientadvancedsettingsmaxeventcount-property"></a>Свойство Имсрдпклиентадванцедсеттингс:: Максевенткаунт

Данное свойство не поддерживается.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_maxEventCount(
  [in]  LONG maxEventCount
);

HRESULT get_maxEventCount(
  [out] LONG pmaxEventCount
);
```



## <a name="property-value"></a>Значение свойства

Новое число событий. Значение по умолчанию — 100.

## <a name="error-codes"></a>Коды ошибок

Возвращает **\_ значение false**.
