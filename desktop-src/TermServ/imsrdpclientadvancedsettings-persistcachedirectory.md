# <a name="imsrdpclientadvancedsettingspersistcachedirectory-property"></a>Имсрдпклиентадванцедсеттингс: свойство Ерсисткачедиректори:P

Данное свойство не поддерживается.

Это свойство доступно только на запись.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_PersistCacheDirectory(
  [in] BSTR PersistCacheDirectory
);
```



## <a name="property-value"></a>Значение свойства

Новый путь.

## <a name="error-codes"></a>Коды ошибок

Возвращает **\_ значение false**.
