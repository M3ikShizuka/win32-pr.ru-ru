# <a name="remote-desktop-services-audioendpoint-enumeration-types"></a>Типы перечисления службы удаленных рабочих столов Аудиоендпоинт

С API Аудиоендпоинт используются следующие типы перечислений.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**\_ \_ Флаги расположения СЗ**](/windows/desktop/api/Audioengineendpoint/ne-audioengineendpoint-ae_position_flags)
</dt> <dd>

Определяет константы для [**структуры \_ текущей \_ позицией AE**](/windows/desktop/api/Audioengineendpoint/ns-audioengineendpoint-ae_current_position) . Эти константы описывают степень допустимости текущей должности.

</dd> <dt>

[**\_ \_ Флаги буфера API**](/windows/desktop/api/Audioapotypes/ne-audioapotypes-apo_buffer_flags)
</dt> <dd>

Определяет флаги проверки буфера для структуры [**\_ \_ свойств соединения API**](/windows/desktop/api/Audioapotypes/ns-audioapotypes-apo_connection_property) , связанной с каждым соединением API.

</dd> </dl>

## <a name="remarks"></a>Комментарии

Службы удаленных рабочих столов API Аудиоендпоинт для использования в сценариях удаленный рабочий стол; Он не предназначен для клиентских приложений.
