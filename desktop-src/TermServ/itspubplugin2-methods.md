# <a name="itspubplugin2-methods"></a>Методы ItsPubPlugin2

Интерфейс [**ItsPubPlugin2**](/windows/desktop/api/tspubplugin2com/nn-tspubplugin2com-itspubplugin2) предоставляет следующие методы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**Метод Делетеперсоналдесктопассигнмент**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-deletepersonaldesktopassignment)
</dt> <dd>

Вызывается для удаления сопоставления между указанным пользователем и виртуальной машиной в личной коллекции виртуальных рабочих столов.

</dd> <dt>

[**Метод GetResource2**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2)
</dt> <dd>

Этот метод зарезервирован и всегда должен возвращать **E \_ нотимпл**.

</dd> <dt>

[**Метод GetResource2List**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-getresource2list)
</dt> <dd>

Извлекает список ресурсов, назначенных указанному пользователю.

</dd> <dt>

[**Метод Ресолвеперсоналдесктоп**](/windows/desktop/api/tspubplugin2com/nf-tspubplugin2com-itspubplugin2-resolvepersonaldesktop)
</dt> <dd>

Вызывается для разрешения сопоставления между указанным пользователем и виртуальной машиной в личной коллекции виртуальных рабочих столов.

</dd> </dl>
