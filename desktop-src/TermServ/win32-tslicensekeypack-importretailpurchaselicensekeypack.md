# <a name="importretailpurchaselicensekeypack-method-of-the-win32_tslicensekeypack-class"></a>Метод Импортретаилпурчаселиценсекэйпакк \_ класса Win32 тслиценсекэйпакк

Импорт с другого сервера лицензий удаленный рабочий стол, службы удаленных рабочих столов пакета лицензионных ключей, приобретенного по розничному каналу.

## <a name="syntax"></a>Синтаксис


```mof
uint32 ImportRetailPurchaseLicenseKeyPack(
  [in]  string sLicenseCode,
  [in]  string sSourceLSName,
  [in]  string sSourceLSProductId,
  [out] uint32 KeyPackId
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*слиценсекоде* \[ окне\]
</dt> <dd>

Содержит 25-символьный код лицензии. В качестве входных данных должно быть задано только 25-символьная строка символов. Переносы не добавляются.

</dd> <dt>

*ссаурцелснаме* \[ окне\]
</dt> <dd>

Имя исходного удаленный рабочий стол сервера лицензирования. Это либо полное различающееся имя, либо IP-адрес сервера.

</dd> <dt>

*ссаурцелспродуктид* \[ окне\]
</dt> <dd>

Идентификатор сервера лицензий удаленный рабочий стол. — Это буквенно-цифровая строка, состоящая из 35 символов, которая не может содержать дефисы.

</dd> <dt>

*Кэйпаккид* \[ заполняет\]
</dt> <dd>

Получает идентификатор пакета ключей.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если метод завершается с ошибкой, он возвращает ноль. Если метод завершился неудачно, он возвращает ненулевое значение. Список кодов ошибок см. в разделе [службы удаленных рабочих столов коды ошибок поставщика WMI](terminal-services-wmi-provider-error-codes.md).
