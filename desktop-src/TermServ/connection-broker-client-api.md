# <a name="remote-desktop-connection-broker-client-api"></a>API клиента компонента подключение к удаленному рабочему столу Broker

подключение к удаленному рабочему столу Broker является компонентом развертывания инфраструктура виртуальных рабочих столов (VDI), который для удаленного подключения найдет подходящий целевой объект из пула компьютеров, на которых подключение может быть перенаправлено, и возвращает IP-адрес целевого объекта. API клиента посредника подключений позволяет сторонним поставщикам протоколов использовать брокер подключений для ускорения обработки подключений, использующих протоколы для виртуальных машин или серверов удаленный рабочий стол в ферме.
