# <a name="iremotedesktopclienteventsonnetworkstatuschanged-method"></a>Метод Иремотедесктопклиентевентс:: Оннетворкстатусчанжед

Вызывается при изменении состояния сети.

## <a name="syntax"></a>Синтаксис


```C++
void OnNetworkStatusChanged(
  [in] unsigned long qualityLevel,
  [in] long          bandwidth,
  [in] long          rtt
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*куалитилевел* \[ окне\]
</dt> <dd>

Новый уровень качества соединения. Уровень качества определяется на основе значений пропускной способности и времени приема-передачи (RTT).

Одно из следующих значений.



| Значение                                                                        | Значение                                                     |
|------------------------------------------------------------------------------|-------------------------------------------------------------|
| <dl> <dt>0</dt> </dl> | Не удалось определить уровень качества.<br/>       |
| <dl> <dt>1</dt> </dl> | Низкое качество соединения (одна панель).<br/>        |
| <dl> <dt>2</dt> </dl> | Качество соединения справедливо (две полосы).<br/>       |
| <dl> <dt>3</dt> </dl> | Качество подключения хорошее (три полосы).<br/>     |
| <dl> <dt>4</dt> </dl> | Качество соединения отлично (четыре полосы).<br/> |



 

</dd> <dt>

*пропускная способность* \[ окне\]
</dt> <dd>

Указывает новую пропускную способность подключения в килобитах в секунду (кбит/с).

</dd> <dt>

*RTT* \[ окне\]
</dt> <dd>

Указывает новое время обращения к соединению (задержка) в миллисекундах.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
