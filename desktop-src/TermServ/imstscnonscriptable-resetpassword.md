# <a name="imstscnonscriptableresetpassword-method"></a>Метод Имстскнонскриптабле:: Ресетпассворд

сбрасывает все состояния паролей в элементе управления удаленный рабочий стол ActiveX. Этот метод можно вызвать для очистки паролей, хранящихся в одном из трех поддерживаемых форматов: обычный текст, переносимая кодировка или двоичный (неподдерживающий) формат. Обратите внимание, что зашифрованные пароли не следует считать безопасными.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT ResetPassword();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

В случае успеха возвратите значение **\_ ОК** .

## <a name="remarks"></a>Remarks

Этот метод можно вызвать для сброса пароля элемента управления после отключения. Это гарантирует, что при последующих соединениях, использующих один и тот же экземпляр элемента управления, не будет автоматически входить в систему с установленным ранее паролем.

этот метод можно вызвать только в том случае, если элемент управления удаленный рабочий стол ActiveX не находится в состоянии connected. Этот метод возвращает **E \_ Fail** , если он вызывается при соединении элемента управления. Чтобы проверить состояние Connected, вызовите метод [**Get \_ Connected**](imstscax-connected.md) через интерфейс [**имстскакс**](imstscax-interface.md) или один из интерфейсов, наследующих от него.

Дополнительные сведения о веб-подключение к удаленному рабочему столу см. в разделе [требования для веб-подключение к удаленному рабочему столу](requirements-for-remote-desktop-web-connection.md).
