# <a name="whats-new-in-windows-7"></a>новые возможности Windows 7

в Windows 7 появились следующие новые программные элементы для службы удаленных рабочих столов.




| Требование | Значение |
|--------|-------|
| интерфейсы удаленный рабочий стол ActiveX элементов управления<br /> | следующие интерфейсы (и связанные с ними свойства) были добавлены в <a href="remote-desktop-web-connection-reference.md">интерфейс управления удаленный рабочий стол ActiveX</a>.<br /><ul><li><a href="imsrdpclientadvancedsettings7.md"><strong>Интерфейс IMsRdpClientAdvancedSettings7</strong></a></li></ul> | 
| API управления Веб-доступ службы удаленных рабочих столов<br /> | Службы удаленных рабочих столов Веб-доступ элемент управления предоставляет интерфейсы, поддерживающие публикацию пользовательских ресурсов в подключения к удаленным рабочим столам и приложениям RemoteApp.<br /><ul><li><a href="/windows/desktop/TermServ/remote-desktop-services-web-access-control-interfaces">API управления Веб-доступ службы удаленных рабочих столов</a></li></ul> | 
| API среды выполнения подключения к удаленным рабочим столам и приложениям RemoteApp<br /> | API среды выполнения подключения к удаленным рабочим столам и приложениям RemoteApp поддерживает разработку пользовательских клиентов в подключения к удаленным рабочим столам и приложениям RemoteApp.<br /><ul><li><a href="/windows/desktop/TermServ/remoteapp-and-desktop-connection-runtime-interfaces">API среды выполнения подключения к удаленным рабочим столам и приложениям RemoteApp</a></li></ul> |
