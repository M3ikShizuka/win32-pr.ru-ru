# <a name="imsrdpclientadvancedsettingshotkeyctrlaltdel-property"></a>Свойство Имсрдпклиентадванцедсеттингс:: Хоткэйктрлалтдел

Указывает код виртуального ключа, добавляемый в сочетание клавиш CTRL + ALT для определения замены сочетания клавиш CTRL + ALT + DELETE, также называемой последовательностью защиты (SAS).

> [!Note]  
> CTRL + ALT + DELETE никогда не перенаправляется на удаленный сервер, даже если свойство [кэйбоардхукмоде](imsrdpclientsecuredsettings-keyboardhookmode.md) включено. CTRL + ALT + DELETE — локальная последовательность SAS.

 

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_HotKeyCtrlAltDel(
  [in]  LONG hotKeyCtrlAltDel
);

HRESULT get_HotKeyCtrlAltDel(
  [out] LONG *photKeyCtrlAltDel
);
```



## <a name="property-value"></a>Значение свойства

Новый код виртуального ключа. **VK \_ Значение END** используется по умолчанию.

## <a name="error-codes"></a>Коды ошибок

При успешном выполнении возвращает значение **\_ ОК** .

## <a name="remarks"></a>Remarks

Дополнительные сведения о веб-подключение к удаленному рабочему столу см. в разделе [требования для веб-подключение к удаленному рабочему столу](requirements-for-remote-desktop-web-connection.md).
