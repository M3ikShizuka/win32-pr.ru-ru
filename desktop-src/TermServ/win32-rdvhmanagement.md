# <a name="win32_rdvhmanagement-class"></a>\_Класс Win32 рдвхманажемент

Описание службы управления удаленный рабочий стол виртуального узла (РДВХ).

Следующий пример синтаксиса — упрощенный MOF-код, который включает все наследуемые свойства.

## <a name="syntax"></a>Синтаксис

``` syntax
[dynamic, provider("Win32_TSVmHost_Prov"), AMENDMENT]
class Win32_RdvhManagement
{
};
```
