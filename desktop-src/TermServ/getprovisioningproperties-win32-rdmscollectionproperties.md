# <a name="getprovisioningproperties-method-of-the-win32_rdmscollectionproperties-class"></a>Метод Жетпровисионингпропертиес \_ класса Win32 рдмсколлектионпропертиес

Извлекает свойства подготовки коллекции виртуальных рабочих столов.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetProvisioningProperties(
  [out] string  PoolVhdType,
  [out] string  MasterVmLocation,
  [out] string  NamePrefix,
  [out] uint32  NameStartIndex,
  [out] string  LocalVmLocation,
  [out] string  LocalGoldVmLocation,
  [out] boolean RunFromSMB,
  [out] string  SMBLocation,
  [out] boolean SMBStreaming,
  [out] string  VmDomain,
  [out] string  VmOU,
  [out] string  ProductKey
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Пулвхдтипе* \[ заполняет\]
</dt> <dd>

Указывает формат виртуального жесткого диска (виртуальный жесткий диск) пула виртуальных машин в коллекции.

<dt>

Клонировать
</dt> <dd>

Клонированный виртуальный жесткий диск.

</dd> <dt>

диффдиск
</dt> <dd>

Разностный виртуальный жесткий диск.

</dd> </dl> </dd> <dt>

*Мастервмлокатион* \[ заполняет\]
</dt> <dd>

Получает путь к образу главной виртуальной машины.

</dd> <dt>

*NamePrefix* \[ заполняет\]
</dt> <dd>

Получает префикс имени, добавляемый к началу имен виртуальных машин.

</dd> <dt>

*Наместартиндекс* \[ заполняет\]
</dt> <dd>

Начальный индекс.

</dd> <dt>

*Локалвмлокатион* \[ заполняет\]
</dt> <dd>

Получает локальный путь к виртуальным машинам на серверах Hyper-V. Если этот параметр имеет значение **null** или если он пуст, будет использоваться каталог виртуальной машины по умолчанию.

</dd> <dt>

*Локалголдвмлокатион* \[ заполняет\]
</dt> <dd>

Получает локальный путь к золотому образу VHD, если для параметра *пулвхдтпе* задано значение "диффдиск". Если этот параметр имеет значение **null** или если он пуст, будет использоваться каталог виртуальной машины по умолчанию.

</dd> <dt>

*Рунфромсмб* \[ заполняет\]
</dt> <dd>

Получает значение, указывающее, следует ли запускать коллекцию из общего ресурса SMB. **Значение true** , чтобы запускать виртуальные машины из общего ресурса СБМ; ином **Значение false**.

</dd> <dt>

*Смблокатион* \[ заполняет\]
</dt> <dd>

Получает путь к общему ресурсу SMB, если включен общий доступ к SMB.

</dd> <dt>

*Смбстреаминг* \[ заполняет\]
</dt> <dd>

Получает значение, указывающее, включена ли потоковая передача SMB. **Значение true** , если общий доступ к SMB включен; ином **Значение false**.

</dd> <dt>

*Вмдомаин* \[ заполняет\]
</dt> <dd>

Получает доменное имя виртуальных машин в коллекции.

</dd> <dt>

*Вмау* \[ заполняет\]
</dt> <dd>

Получает Active Directory подразделение виртуальных машин в коллекции.

</dd> <dt>

*ProductKey* \[ заполняет\]
</dt> <dd>

Получает ключи продукта операционной системы для виртуальных машин в коллекции.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение 0 при успешном выполнении, в противном случае возвращает код ошибки WMI.
