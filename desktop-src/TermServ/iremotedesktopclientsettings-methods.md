# <a name="iremotedesktopclientsettings-methods"></a>Методы Иремотедесктопклиентсеттингс

Интерфейс [**иремотедесктопклиентсеттингс**](/windows/win32/api/rdpappcontainerclient/nn-rdpappcontainerclient-iremotedesktopclientsettings) предоставляет следующие методы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**Метод Апплисеттингс**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-applysettings)
</dt> <dd>

Сохраняет указанное содержимое в RDP-файле.

</dd> <dt>

[**Метод Жетрдппроперти**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-getrdpproperty)
</dt> <dd>

Извлекает одно именованное значение свойства RDP.

</dd> <dt>

[**Метод Ретриевесеттингс**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-retrievesettings)
</dt> <dd>

Извлекает весь RDP-файл в виде строки.

</dd> <dt>

[**Метод Сетрдппроперти**](/windows/win32/api/rdpappcontainerclient/nf-rdpappcontainerclient-iremotedesktopclientsettings-setrdpproperty)
</dt> <dd>

Задает значение одного именованного свойства RDP.

</dd> </dl>
