# <a name="cim_setting-class-remote-desktop-services"></a>Класс CIM_Setting (службы удаленных рабочих столов)

Представляет связанные с конфигурацией и операционные параметры для одного или нескольких управляемых системных элементов.

Следующий синтаксис упрощен из MOF-кода и включает все унаследованные свойства.

## <a name="syntax"></a>Синтаксис

``` syntax
[Abstract, UUID("{8502C518-5FBB-11D2-AAC1-006008C78BC7}"), AMENDMENT]
class CIM_Setting : CIM_ManagedSystemElement
{
  string   Caption;
  string   Description;
  datetime InstallDate;
  string   Name;
  string   Status;
};
```
