# <a name="imstscaxeventsonfocusreleased-method"></a>Метод Имстскаксевентс:: Онфокусрелеасед

Вызывается при нажатии сочетания клавиш для фокуса в выпуске. Например, это событие вызывается, когда пользователь нажимает клавишу CTRL + ALT + стрелка влево или сочетание клавиш CTRL + ALT + стрелка вправо.

Это событие позволяет контейнеру элемента управления ActiveX убрать элемент управления с элемента управления ActiveX. Например, это полезно в ситуации, когда у вас нет мыши, а специальные сочетания клавиш, такие как ALT + TAB, перенаправляются на сервер. В этом случае нет способа вернуть фокус клавиатуры обратно на локальный рабочий стол. Однако с помощью сочетания клавиш CTRL + ALT + стрелка влево или CTRL + ALT + стрелка вправо контейнер ActiveX может прослушивать это событие и реагировать на него, отключив фокус от элемента управления ActiveX.

## <a name="syntax"></a>Синтаксис


```C++
void OnFocusReleased(
  [in] INT iDirection
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*идиректион* \[ окне\]
</dt> <dd>

Параметр направления 1 (CTRL + ALT + стрелка вправо) или 1 (CTRL + ALT + стрелка влево).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Это событие доступно, если на клиенте используется подключение к удаленному рабочему столу 6,0.
