# <a name="win32_tssessiondirectory-methods"></a>\_Методы тссессиондиректори Win32

Класс [**Win32 \_ тссессиондиректори**](win32-tssessiondirectory.md) предоставляет следующие методы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**Метод Креатеусердисктемплате**](createuserdisktemplate-win32-tssessiondirectory.md)
</dt> <dd>

Создает шаблон диска пользователя.

</dd> <dt>

[**Метод Дисаблеусервхд**](disableuservhd-win32-tssessiondirectory.md)
</dt> <dd>

Отключает виртуальный жесткий диск профиля пользователя.

</dd> <dt>

[**Метод Енаблеусервхд**](enableuservhd-win32-tssessiondirectory.md)
</dt> <dd>

Включает виртуальный жесткий диск профиля пользователя на сервере узлов удаленных рабочих столов.

</dd> <dt>

[**Метод Жеткуррентредиректаблеаддрессес**](getcurrentredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Получает список доступных адресов DNS, которые можно использовать для перенаправления.

</dd> <dt>

[**Метод Жетредиректаблеаддрессес**](getredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Получает полный список доступных адресов DNS.

</dd> <dt>

[**Метод Пингсессиондиректори**](pingsessiondirectory-win32-tssessiondirectory.md)
</dt> <dd>

Проверяет, доступен ли сервер подключение к удаленному рабочему столу Broker (RDCB).

</dd> <dt>

[**Метод Сеткуррентредиректаблеаддрессес**](setcurrentredirectableaddresses-win32-tssessiondirectory.md)
</dt> <dd>

Задает настроенный список доступных адресов DNS, которые могут использоваться для перенаправления.

</dd> <dt>

[**Метод СетлоадбаланЦингстате**](setloadbalancingstate-win32-tssessiondirectory.md)
</dt> <dd>

Задает значение, указывающее, будет ли сервер участвовать в балансировке нагрузки RDCB.

</dd> <dt>

[**Метод Сетсервервеигхт**](setserverweight-win32-tssessiondirectory.md)
</dt> <dd>

Задает значение веса сервера для балансировки нагрузки RDCB.

</dd> <dt>

[**Метод Сетсессиондиректоряктиве**](win32-tssessiondirectory-setsessiondirectoryactive.md)
</dt> <dd>

Метод **сетсессиондиректоряктиве** задает свойство **сессиондиректоряктиве** .

</dd> <dt>

[**Метод Сетсессиондиректорекспосесерверип**](win32-tssessiondirectory-setsessiondirectoryexposeserverip.md)
</dt> <dd>

Задает свойство **сессиондиректорекспосесерверип** .

</dd> <dt>

[**Метод Сетсессиондиректорипроперти**](win32-tssessiondirectory-setsessiondirectoryproperty.md)
</dt> <dd>

Задает свойство **сессиондиректорилокатион** или свойство **сессиондиректориклустернаме** для класса.

</dd> <dt>

[**Метод Сеттсредиректормоде**](settsredirectormode-win32-tssessiondirectory.md)
</dt> <dd>

Задает значение, указывающее, будет ли сервер действовать в качестве перенаправления службы удаленных рабочих столов.

</dd> </dl>
