API виртуализации удаленный рабочий стол поддерживает следующие структуры.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**\_Отображение клиента**](/windows/win32/api/sessdirpublictypes/ns-sessdirpublictypes-client_display)
</dt> <dd>

Содержит сведения о дисплее клиента подключение к удаленному рабочему столу (RDC).

</dd> <dt>

[**ТССД \_ ConnectionPoint**](/windows/win32/api/sessdirpublictypes/ns-sessdirpublictypes-tssd_connectionpoint)
</dt> <dd>

Определяет IP-адрес целевого объекта.

</dd> </dl>
