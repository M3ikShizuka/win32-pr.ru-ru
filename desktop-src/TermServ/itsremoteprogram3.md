# <a name="itsremoteprogram3-interface"></a>Интерфейс ITSRemoteProgram3

Определяет свойства и методы для использования с RemoteApp.

## <a name="members"></a>Элементы

Интерфейс **ITSRemoteProgram3** наследует от **ITSRemoteProgram2**. **ITSRemoteProgram3** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ITSRemoteProgram3** содержит следующие методы.



| Метод                                                     | Описание                                                  |
|:-----------------------------------------------------------|:-------------------------------------------------------------|
| [**серверстартапп**](itsremoteprogram3-serverstartapp.md) | Указывает приложение, запускаемое в удаленном сеансе. <br/> |
