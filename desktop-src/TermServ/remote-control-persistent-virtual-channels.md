# <a name="remote-control-persistent-virtual-channels"></a>Постоянные виртуальные каналы удаленного управления

В Windows XP библиотека DLL может указывать один или несколько виртуальных каналов в качестве *постоянного удаленного управления*. Постоянный виртуальный канал удаленного управления не закрывается при подключении или отключении удаленного управления для сеанса, подключенного к клиенту. Данные по-прежнему будут передаваться и приниматься через этот канал. Виртуальные каналы, не указанные библиотекой DLL в качестве постоянного удаленного управления, будут закрыты в этом сценарии.

Постоянные виртуальные каналы удаленного управления указываются во время вызова **виртуалчаннелинит**. Конкретные сведения об этом см. на странице справочника по [**виртуалчаннелинит**](/windows/desktop/api/Cchannel/nc-cchannel-virtualchannelinit) .
