# <a name="itssbtaskpluginnotifysink-methods"></a>Методы Итссбтаскплугиннотифисинк

Интерфейсы [**итссбтаскплугиннотифисинк**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbtaskpluginnotifysink) предоставляют следующие методы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**Метод Онделететасктиме**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-ondeletetasktime)
</dt> <dd>

Уведомляет подключение к удаленному рабочему столу Broker (RDCB) о том, что задача была удалена из очереди.

</dd> <dt>

[**Метод Онрепорттаскс**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onreporttasks)
</dt> <dd>

Оповещает RDCB о новом отчете о задаче.

</dd> <dt>

[**Метод Онсеттасктиме**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onsettasktime)
</dt> <dd>

Уведомляет RDCB о том, что задача запланирована.

</dd> <dt>

[**Метод Онупдатетаскстатус**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskpluginnotifysink-onupdatetaskstatus)
</dt> <dd>

Уведомляет RDCB о том, что состояние задачи изменилось.

</dd> </dl>
