# <a name="synchronizeallpublishingdata-method-of-the-win32_rdmsmanagementdata-class"></a>Метод Синчронизеаллпублишингдата \_ класса Win32 рдмсманажементдата

Синхронизирует все данные публикации для служб удаленный рабочий стол Management Services (RDMS).

## <a name="syntax"></a>Синтаксис


```mof
uint32 SynchronizeAllPublishingData();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение 0 при успешном выполнении, в противном случае возвращает код ошибки WMI.
