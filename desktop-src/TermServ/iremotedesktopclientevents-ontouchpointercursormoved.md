# <a name="iremotedesktopclienteventsontouchpointercursormoved-method"></a>Метод Иремотедесктопклиентевентс:: Онтаучпоинтеркурсормовед

## <a name="syntax"></a>Синтаксис


```C++
void OnTouchPointerCursorMoved(
  [in] long x,
  [in] long y
);
```
