# <a name="remote-desktop-services-permissions"></a>службы удаленных рабочих столов разрешения

Вы можете использовать разрешения, предоставленные для службы удаленных рабочих столов, чтобы управлять доступом пользователей и групп к серверу. Описание типов разрешений по умолчанию и более подробные сведения о службы удаленных рабочих столов разрешений в целом см. в документации, прилагаемой к средству администрирования настройки службы удаленных рабочих столов. сведения о настройке этих разрешений для Windows Server 2008 см. в разделе [настройка разрешений для службы удаленных рабочих столовных соединений](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc753032(v=ws.11)).

Ниже приведен список разрешений, которые можно задать, а также задачи, разрешенные разрешениями.



| Значение                        | Значение                                                                                                                                                               |
|------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Сведения о запросе<br/> | Запрос сведений о сеансах и серверах.<br/>                                                                                                                |
| Задать сведения<br/>   | Настройте свойства соединения.<br/>                                                                                                                           |
| Удаленное управление<br/>    | Просмотр или активное управление сеансом другого пользователя.<br/>                                                                                                           |
| Вход в систему<br/>             | Войдите в сеанс на сервере.<br/>                                                                                                                         |
| Выполнен<br/>            | Выйдите из сеанса пользователя. Имейте в виду, что выход пользователя из системы без предупреждения может привести к утрате данных на клиенте.<br/>                                  |
| Message<br/>           | Отправка сообщения в сеансы другого пользователя.<br/>                                                                                                                 |
| Подключение<br/>           | Подключение в другой сеанс.<br/>                                                                                                                                |
| Отключение<br/>        | Отключение сеанса.<br/>                                                                                                                                      |
| Виртуальные каналы<br/>  | Используйте виртуальные каналы. Имейте в виду, что отключение виртуальных каналов отключает некоторые службы удаленных рабочих столов функции, такие как буфер обмена и перенаправление принтеров.<br/> |
| Reset<br/>             | Завершение сеанса. Имейте в виду, что завершение сеанса без предупреждения может привести к утрате данных на клиенте.<br/>                                                    |



 

Разрешение на вход в систему требуется пользователю для входа в новый сеанс службы удаленных рабочих столов. Все остальные службы удаленных рабочих столов разрешения применяются для управления сеансом службы удаленных рабочих столов другого пользователя.

Службы удаленных рабочих столов разрешения могут быть предоставлены или заданы для отдельных пользователей или групп. Пользователи также могут наследовать разрешения в результате того, что является членом группы. Однако отказ от разрешения переопределяет унаследованное разрешение. Например, членам группы удаленный рабочий стол Users (РДУ) по умолчанию предоставляется разрешение на запрос. Если администратор устанавливает разрешение запроса "запретить" для этого пользователя, он не сможет выполнять запросы к сеансу другого пользователя. После входа пользователя в сеанс ему предоставляются все остальные разрешения службы удаленных рабочих столов для своего сеанса.
