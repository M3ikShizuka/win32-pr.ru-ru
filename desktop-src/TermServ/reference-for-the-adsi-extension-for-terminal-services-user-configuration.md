# <a name="reference-for-the-adsi-extension-for-remote-desktop-services-user-configuration"></a>Справочник по расширению ADSI для службы удаленных рабочих столов пользовательской конфигурации

Расширение ADSI для службы удаленных рабочих столов пользовательской конфигурации позволяет администраторам домена создавать сценарии для автоматизации обслуживания свойств пользователей, относящихся к службы удаленных рабочих столов. Он включает в себя следующий сдвоенный интерфейс.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**иадстсусерекс**](/windows/desktop/api/Tsuserex/nn-tsuserex-iadstsuserex)
</dt> <dd>

Используется для проверки и настройки службы удаленных рабочих столов свойств пользователя.

</dd> </dl>
