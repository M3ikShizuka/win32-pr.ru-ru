# <a name="imsrdpdevicev2-properties"></a>Свойства IMsRdpDeviceV2

Интерфейс [**IMsRdpDeviceV2**](imsrdpdevicev2.md) предоставляет следующие свойства.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**Кмклассгуид, свойство**](imsrdpdevicev2-cmclassguid.md)
</dt> <dd>

Содержит GUID класса установки Configuration Manager для устройства.

</dd> <dt>

[**Кмдевицеинстанце, свойство**](imsrdpdevicev2-cmdeviceinstance.md)
</dt> <dd>

Содержит экземпляр устройства Configuration Manager.

</dd> <dt>

[**Девицетекст, свойство**](imsrdpdevicev2-devicetext.md)
</dt> <dd>

Содержит текст устройства.

</dd> <dt>

[**Дривелеттербитмап, свойство**](imsrdpdevicev2-driveletterbitmap.md)
</dt> <dd>

Содержит битовое значение, представляющее собой карту букв диска, содержащихся в устройстве.

</dd> <dt>

[**Искомпоситедевице, свойство**](imsrdpdevicev2-iscompositedevice.md)
</dt> <dd>

Указывает, является ли устройство составным.

</dd> <dt>

[**Исоптионалдевице, свойство**](imsrdpdevicev2-isoptionaldevice.md)
</dt> <dd>

Указывает, является ли устройство необязательным для перенаправления USB.

</dd> <dt>

[**Исусбдевице, свойство**](imsrdpdevicev2-isusbdevice.md)
</dt> <dd>

Указывает, что устройство предназначено для перенаправления USB.

</dd> </dl>
