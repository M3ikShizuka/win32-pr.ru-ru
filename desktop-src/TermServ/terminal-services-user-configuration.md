# <a name="remote-desktop-services-user-configuration"></a>службы удаленных рабочих столов Конфигурация пользователя

Функции [**втскуерюсерконфиг**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) и [**втссетусерконфиг**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga) позволяют задавать и извлекать сведения о конфигурации для указанной учетной записи пользователя. Тип [**перечисления \_ \_ класса ВТС config**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class) определяет параметры конфигурации пользователя.
