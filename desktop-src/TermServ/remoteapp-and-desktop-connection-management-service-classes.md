# <a name="remoteapp-and-desktop-connection-management-service-classes"></a>Классы служб управление подключениями к удаленным рабочим столам и приложениям RemoteApp

Поставщик WMI службы управление подключениями к удаленным рабочим столам и приложениям RemoteApp предоставляет следующие классы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**\_Централпублишингчанжеевент Win32**](win32-centralpublishingchangeevent.md)
</dt> <dd>

Событие, представляющее изменение центральных параметров RDV.

</dd> <dt>

[**\_Рдцентралпублишеддеплойментсеттингс Win32**](win32-rdcentralpublisheddeploymentsettings.md)
</dt> <dd>

Содержит параметры развертывания, используемые для создания RDP-файлов для ресурсов, опубликованных из фермы.

</dd> <dt>

[**\_Рдцентралпублишедфарм Win32**](win32-rdcentralpublishedfarm.md)
</dt> <dd>

Список ферм, из которых были опубликованы настольные компьютеры или приложения.

</dd> <dt>

[**\_РдцентралпублишедфилеассоЦиатион Win32**](win32-rdcentralpublishedfileassociation.md)
</dt> <dd>

Сведения о расширении файла, связанном с приложением

</dd> <dt>

[**\_Рдцентралпублишедремотеаппликатион Win32**](win32-rdcentralpublishedremoteapplication.md)
</dt> <dd>

Описывает приложение, опубликованное на другом компьютере для удаленного использования через службы терминалов.

</dd> <dt>

[**\_Рдцентралпублишедремотедесктоп Win32**](win32-rdcentralpublishedremotedesktop.md)
</dt> <dd>

Рабочий стол, опубликованный на другом компьютере для удаленного использования через службы терминалов

</dd> <dt>

[**\_Рдперсоналдесктопассигнмент Win32**](win32-rdpersonaldesktopassignment.md)
</dt> <dd>

Список назначений персональных рабочих столов.

</dd> <dt>

[**\_Тскпубплугин Win32**](win32-tscpubplugin.md)
</dt> <dd>

Представляет подключаемый модуль, настроенный для использования в службе управление подключениями к удаленным рабочим столам и приложениям RemoteApp.

</dd> <dt>

[**\_Тслегациплугин Win32**](win32-tslegacyplugin.md)
</dt> <dd>

Представляет удаленный рабочий стол сервер, который встроенные подключаемые модули службы управление подключениями к удаленным рабочим столам и приложениям RemoteApp будут запрашивать программы RemoteApp.

</dd> <dt>

[**\_Рабочая область Win32**](win32-workspace.md)
</dt> <dd>

Указывает службы удаленных рабочих столов сведения о конфигурации рабочей области.

</dd> </dl>
