# <a name="setallowtsconnections-method-of-the-win32_terminalservicesetting-class"></a>Метод Сеталловтсконнектионс \_ класса Win32 терминалсервицесеттинг

Метод **сеталловтсконнектионс** разрешает или запрещает новые подключения к удаленному рабочему столу. Этот метод соответствующим образом изменит свойство **алловтсконнектионс** для класса.

## <a name="syntax"></a>Синтаксис


```mof
uint32 SetAllowTSConnections(
  [in] uint32 AllowTSConnections,
  [in] uint32 ModifyFirewallException
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Алловтсконнектионс* \[ окне\]
</dt> <dd>

Указывает, разрешены ли новые подключения к удаленному рабочему столу. Это должно быть одно из следующих значений.

<dt>

<span id="0"></span>

<span id="0"></span>**0,0**


</dt> <dd>

Новые подключения не допускаются. Если параметр *модифифиреваллексцептион* имеет значение 1, то исключение брандмауэра удаленный рабочий стол отключено.

</dd> <dt>

<span id="1"></span>

<span id="1"></span>**1**


</dt> <dd>

Новые подключения разрешены. Если параметр *модифифиреваллексцептион* имеет значение 1, то удаленный рабочий стол исключение брандмауэра включено.

</dd> </dl> </dd> <dt>

*Модифифиреваллексцептион* \[ окне\]
</dt> <dd>

Указывает, будет ли параметр исключения брандмауэра для удаленный рабочий стол изменен в состояние, указанное параметром *алловтсконнектионс* . Это должно быть одно из следующих значений.

<dt>

<span id="0"></span>

<span id="0"></span>**0,0**


</dt> <dd>

Не изменяйте параметр исключения брандмауэра.

</dd> <dt>

<span id="1"></span>

<span id="1"></span>**1**


</dt> <dd>

Измените параметр исключения брандмауэра.

</dd> </dl> </dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает успешное завершение; в противном случае возвращает код ошибки WMI. Список этих значений см. в разделе [службы удаленных рабочих столов коды ошибок поставщика WMI](terminal-services-wmi-provider-error-codes.md) . Метод возвращает ошибку, если параметр находится в разделе Управление групповой политикой.
