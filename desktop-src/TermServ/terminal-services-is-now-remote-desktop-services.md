# <a name="terminal-services-is-now-remote-desktop-services"></a>Службы терминалов теперь службы удаленных рабочих столов

Службы терминалов были переименованы службы удаленных рабочих столов.

в Windows Server 2008 R2 все службы ролей службы удаленных рабочих столов были переименованы. В следующей таблице перечислены имя и новое имя каждой службы роли службы удаленных рабочих столов.



| Предыдущее имя                                                   | имя в Windows Server 2008 R2                                         |
|-----------------------------------------------------------------|------------------------------------------------------------------------|
| Службы терминалов <br/>                                   | Службы удаленных рабочих столов<br/>                                     |
| Сервер терминалов<br/>                                      | Узел сеансов удаленных рабочих столов<br/>               |
| Лицензирование службы терминалов (лицензирование служб терминалов)<br/>          | Лицензирование удаленных рабочих столов<br/>                     |
| Шлюз служб терминалов<br/>               | Шлюз удаленных рабочих столов<br/>                         |
| Посредник сеансов служб терминалов<br/> | Брокер подключение к удаленному рабочему столу (RDCB)<br/>     |
| Веб-доступ к службам терминалов<br/>         | Веб-доступ удаленный рабочий стол (RD Веб-доступ)<br/>                  |
| Виртуализация служб терминалов<br/>                     | Узел виртуализации удаленных рабочих столов<br/> |



 

Кроме того, обратите внимание, что сервер, на котором установлена служба роли узла сеансов удаленных рабочих столов, теперь называется сервером узла сеансов удаленных рабочих столов, а не сервером терминалов.
