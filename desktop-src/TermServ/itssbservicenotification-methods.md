# <a name="itssbservicenotification-methods"></a>Методы Итссбсервиценотификатион

Интерфейсы [**итссбсервиценотификатион**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbservicenotification) предоставляют следующие методы.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**Метод Нотифисервицефаилуре**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicefailure)
</dt> <dd>

Уведомляет зарегистрированные подключаемые модули о том, что служба подключение к удаленному рабочему столу Broker (RDCB) остановлена.

</dd> <dt>

[**Метод Нотифисервицесукцесс**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicesuccess)
</dt> <dd>

Уведомляет зарегистрированные подключаемые модули, запущенные службой RDCB.

</dd> </dl>
