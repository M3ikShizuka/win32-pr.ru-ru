# <a name="dynamic-virtual-channels-reference"></a>Справочник по динамическим виртуальным каналам

Следующие клиентские интерфейсы динамического виртуального канала (DVC) поддерживаются службы удаленных рабочих столов (ранее назывались службами терминалов).

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[Клиентские API DVC](dvc-client-apis.md)
</dt> <dd>

Клиентские API динамических виртуальных каналов (DVC) реализуются специально для клиента подключение к удаленному рабочему столу (RDC) подключения.

</dd> <dt>

[API сервера DVC](dvc-server-apis.md)
</dt> <dd>

Динамические API сервера виртуальных каналов (DVC) реализуются с помощью сервера подключение к удаленному рабочему столу (RDC) подключения.

</dd> </dl>
