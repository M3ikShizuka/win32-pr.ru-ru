# <a name="remote-desktop-services-management-applications"></a>Приложения управления службы удаленных рабочих столов

Службы удаленных рабочих столов поддерживает следующие приложения управления.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[Расширение ADSI для пользовательской конфигурации службы удаленных рабочих столов](adsi-extension-for-terminal-services-user-configuration.md)
</dt> <dd>

Расширение интерфейса службы Active Directory (ADSI) — это специальный COM-объект, позволяющий расширение COM-объекта.

</dd> <dt>

[службы удаленных рабочих столов разрешения](terminal-services-permissions.md)
</dt> <dd>

Вы можете использовать разрешения, предоставленные для службы удаленных рабочих столов, чтобы управлять доступом пользователей и групп к серверу.

</dd> </dl>

[Поставщик wmi службы удаленных рабочих столов](terminal-services-wmi-provider.md) предоставляет доступ WMI к сведениям, которые администраторы могут настроить с помощью оснастки Службы удаленных рабочих столов Настройка или подключения консоли управления (MMC).

[Расширение ADSI для пользовательской конфигурации службы удаленных рабочих столов](adsi-extension-for-terminal-services-user-configuration.md) расширяет объект ADSI, позволяющий администраторам домена создавать сценарии, автоматизирующие обслуживание службы удаленных рабочих столовных свойств пользователя.

Чтобы получить список разрешений, которые могут быть заданы, и задачи, разрешенные разрешениями, [службы удаленных рабочих столов разрешения](terminal-services-permissions.md).

Дополнительные сведения см. в [справочнике по поставщику WMI службы удаленных рабочих столов](terminal-services-wmi-provider-reference.md) и [использовании расширения ADSI для службы удаленных рабочих столов пользовательской конфигурации](using-the-adsi-extension-for-terminal-services-user-configuration.md).
