# <a name="disabling-remote-desktop-services-features"></a>Отключение функций службы удаленных рабочих столов

для повышения безопасности можно отключить службы удаленных рабочих столов такие функции, как перенаправление буфера обмена и перенаправление принтеров, для клиентов, подключающихся к серверам узла сеансов удаленный рабочий стол (узел сеансов удаленных рабочих столов) с помощью элемента управления удаленный рабочий стол ActiveX.

**Отключение функций службы удаленных рабочих столов**

1.  Измените реестр клиентского компьютера и добавьте следующие ключи:

    -   **HKEY \_ локальный \_ компьютер \\ программное обеспечение \\ Microsoft \\ Terminal Server \\ дисаблеклипбоардредиректион**
    -   **HKEY \_ локальный \_ компьютер \\ программное обеспечение \\ Microsoft \\ Terminal Server \\ дисабледривередиректион**
    -   **HKEY \_ локальный \_ компьютер \\ программное обеспечение \\ Microsoft \\ Terminal Server \\ дисаблепринтерредиректион**

2.  Задайте для обоих ключей значение **reg \_ DWORD** 1.
