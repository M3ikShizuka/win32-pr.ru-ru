# <a name="remote-desktop-management-services-provider"></a>Поставщик служб удаленный рабочий стол Management Services

Поставщик служб управления удаленный рабочий стол (RDMS) управляет средами инфраструктуры виртуальных рабочих столов (VDI).

<dl> <dt>

[**\_Рдмсколлектионпропертиес Win32**](win32-rdmscollectionproperties.md)
</dt> <dd>

Управляет свойствами коллекции виртуальных рабочих столов.

</dd> <dt>

[**\_Рдмсдеплойментсеттингс Win32**](win32-rdmsdeploymentsettings.md)
</dt> <dd>

Управляет параметрами развертывания для коллекции виртуальных рабочих столов.

</dd> <dt>

[**\_Рдмсдесктопассигнмент Win32**](win32-rdmsdesktopassignment.md)
</dt> <dd>

Описание назначения пользовательского рабочего стола коллекции удаленных рабочих столов.

</dd> <dt>

[**\_Рдмсенвиронмент Win32**](win32-rdmsenvironment.md)
</dt> <dd>

Управляет средой служб управления удаленный рабочий стол (RDMS).

</dd> <dt>

[**\_РдмсфилетипеассоЦиатион Win32**](win32-rdmsfiletypeassociation.md)

[**\_Рдмсжоинедноде Win32**](win32-rdmsjoinednode.md)

</dd> <dt>

[**\_Рдмсманажементдата Win32**](win32-rdmsmanagementdata.md)
</dt> <dd>

[**\_Рдмспублишедаппликатион Win32**](win32-rdmspublishedapplication.md)
</dt> <dd>

Управляет опубликованным приложением.

[**\_Рдмсвиртуалдесктоп Win32**](win32-rdmsvirtualdesktop.md)

[**\_Рдмсвиртуалдесктопколлектион Win32**](win32-rdmsvirtualdesktopcollection.md)

</dd> <dt>

[**\_Рдшколлектион Win32**](win32-rdshcollection.md)

[**\_Рдшсервер Win32**](win32-rdshserver.md)
