# <a name="itsremoteprogram2remoteapplicationargs-property"></a>Свойство ITSRemoteProgram2:: Ремотеаппликатионаргс

Аргументы командной строки для RemoteApp.

Это свойство доступно только на запись.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_RemoteApplicationArgs(
  [in] BSTR bstrRemoteApplicationArgs
);
```



## <a name="property-value"></a>Значение свойства

Аргументы командной строки RemoteApp.
