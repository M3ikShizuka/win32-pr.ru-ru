# <a name="how-to-optimize-your-msdn-library-search"></a>Оптимизация поиска в библиотеке MSDN

Чтобы выполнить поиск в MSDN, введите ключевое слово или символьную строку в поле **поиска MSDN с помощью Bing** . После поиска на веб-сайте [Microsoft Developer Network](https://msdn.microsoft.com/default.aspx) (MSDN) можно сузить результаты поиска, щелкнув Параметры, доступные в столбце **Уточнение поиска** в разделе **по источнику** или **разделу**. Эти параметры становятся доступными после первой попытки поиска.

**Только для возврата разделов в разделе "Разработка приложений Windows" библиотеки MSDN**

1.  В списке **по разделам** выберите **Win32 & com**. Появится дополнительный список с именем **текущие уточнения** . **Текущая строка уточнения** имеет **Win32 &** в ней com.
2.  В списке **по источнику** щелкните **Документация & статьи**. Список " **текущие усовершенствования** " теперь содержит **документацию & статей**.

При поиске могут быть доступны дополнительные фильтры, например **Безопасность Windows**, **графика и мультимедиа** Windows, **а также обмен сообщениями и совместная работа**.

**Поиск только тем в пакете SDK для безопасности**

-   В списке **по разделам** выберите **Безопасность Windows**. Список **текущих усовершенствований** теперь включает **Безопасность Windows**.

При вводе строки поиска варианты автоматически отображаются в поле поиска. Чтобы найти содержимое пакета SDK, найдите в предложениях **API** или **Windows** .

Если щелкнуть раздел в результатах поиска, можно быстро определить, содержит ли раздел содержимое пакета SDK, просмотрев путь перехода, который отображается в верхней части раздела. Например, при поиске **криптографии** создаются предложения, включающие **API шифрования**. При выборе **API криптографии** предложений создается панель результатов поиска, включающая **функции шифрования (Windows)**. Щелкнув элемент **криптографические функции (Windows)**, вы переведете в раздел, содержащий следующий путь навигации в верхней части раздела:

**MSDN > Библиотека MSDN > Windows Development > безопасность > шифрование > Справочник по шифрованию > криптографические функции >**

Все содержимое пакета SDK для системы безопасности содержит эту часть пути перехода:

**MSDN > Библиотека MSDN > Windows Development > Security >**

**Удаление фильтров результатов поиска из поиска MSDN**

-   В списке **текущие уточнения** щелкните **(x)** , следующий за фильтром поиска, который нужно удалить. Например, чтобы удалить **документацию & статей** из **текущих усовершенствований**, щелкните **(x)** , следующий за **документацией & статей**. Можно также нажать кнопку **удалить все** , чтобы удалить все фильтры из поиска MSDN.
