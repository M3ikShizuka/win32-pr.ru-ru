# <a name="setprocessdefaultcpusets-function"></a>Функция Сетпроцессдефаулткпусетс

Задает назначение наборов ЦП по умолчанию для потоков в указанном процессе. Создаваемые потоки, не имеющие наборов ЦП, явно заданные с помощью [**сетсреадселектедкпусетс**](setthreadselectedcpusets.md), будут наследовать наборы, заданные **сетпроцессдефаулткпусетс** , автоматически.

## <a name="syntax"></a>Синтаксис


```C++
BOOL WINAPI SetProcessDefaultCpuSets(
  _In_     HANDLE Process,
  _In_opt_ ULONG  CpuSetIds,
  _In_     ULONG  CpuSetIdCound
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Обработка* \[ окне\]
</dt> <dd>

Указывает процесс, для которого задаются наборы ЦП по умолчанию. Этот обработчик должен иметь право на \_ \_ \_ доступ к ограниченным сведениям. Значение, возвращаемое функцией [**жеткуррентпроцесс**](/windows/win32/api/processthreadsapi/nf-processthreadsapi-getcurrentprocess) , также можно указать здесь.

</dd> <dt>

*Кпусетидс* \[ в необязательное\]
</dt> <dd>

Указывает список идентификаторов наборов ЦП, которые необходимо задать в качестве набора ЦП процесса по умолчанию. Если это значение равно NULL, **сетпроцессдефаулткпусетс** удаляет любое назначение.

</dd> <dt>

*Кпусетидкаунд* \[ окне\]
</dt> <dd>

Указывает количество идентификаторов в списке, переданном в аргументе **кпусетидс** . Если это значение равно NULL, то оно должно быть равно 0.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не может завершиться ошибкой, если переданы допустимые параметры.
