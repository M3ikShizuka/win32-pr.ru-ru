# <a name="setthreadselectedcpusets-function"></a>Функция Сетсреадселектедкпусетс

Задает назначение выбранных наборов ЦП для указанного потока. Это назначение переопределяет назначение процесса по умолчанию, если оно задано.

## <a name="syntax"></a>Синтаксис


```C++
BOOL WINAPI SetThreadSelectedCpuSets(
  _In_       HANDLE Thread,
  _In_ const ULONG  *CpuSetIds,
  _In_       ULONG  CpuSetIdCount
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Поток команд* \[ окне\]
</dt> <dd>

Указывает поток, в котором следует задать назначение набора ЦП. Этот обработчик должен иметь право на \_ \_ доступ к ограниченным данным в наборе потоков \_ . Можно также использовать значение, возвращаемое функцией [**жеткуррентсреад**](/windows/win32/api/processthreadsapi/nf-processthreadsapi-getcurrentthread) .

</dd> <dt>

*Кпусетидс* \[ окне\]
</dt> <dd>

Указывает список идентификаторов наборов ЦП, которые должны быть заданы в качестве выбранного в потоке набора ЦП. Если это значение равно NULL, API удаляет любое назначение, при этом возвращается к обработке назначения по умолчанию, если оно задано.

</dd> <dt>

*Кпусетидкаунт* \[ окне\]
</dt> <dd>

Указывает количество идентификаторов в списке, переданном в аргументе **кпусетидс** . Если это значение равно NULL, то оно должно быть равно 0.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не может завершиться ошибкой, если переданы допустимые параметры.
