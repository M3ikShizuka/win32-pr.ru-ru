# <a name="windows-mixed-reality-com-interfaces"></a>COM-интерфейсы Windows Mixed Reality

Разделы, содержащиеся в этом разделе, содержат справочные спецификации для COM-интерфейсов Windows Mixed Reality.

## <a name="in-this-section"></a>В этом разделе

| Раздел | Описание |
|-|-|
| [**испатиалинтерактионманажеринтероп**](/windows/desktop/api/spatialinteractionmanagerinterop/nn-spatialinteractionmanagerinterop-ispatialinteractionmanagerinterop) | Обеспечивает взаимодействие универсальная платформа Windows с объектом [спатиалинтерактионманажер](/uwp/api/Windows.UI.Input.Spatial.SpatialInteractionManager) (UWP), а также предоставляет доступ к членам **спатиалинтерактионманажер** для доступа к входным данным пользователя из руки, контроллеров движения и системных команд. |
| [**ихолографикспацеинтероп**](/windows/desktop/api/holographicspaceinterop/nn-holographicspaceinterop-iholographicspaceinterop) | Обеспечивает возможность взаимодействия универсальная платформа Windows с объектом [холографикспаце](/uwp/api/windows.graphics.holographic.holographicspace) (UWP), а также предоставляет доступ к членам **холографикспаце** , представляющим holographic-сцену. |
