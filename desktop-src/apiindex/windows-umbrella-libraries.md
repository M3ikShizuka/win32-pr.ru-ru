# <a name="windows-umbrella-libraries"></a>Общие библиотеки Windows

*Библиотека "тег* -объект" — это единая Библиотека статической компоновки, которая экспортирует подмножество интерфейсов API Win32. например, библиотека с именем **OneCore. lib** предоставляет экспорты для подмножества api-интерфейсов Win32, общих для всех устройств Windows 10.

Интерфейсы API в библиотеке-кодека могут быть реализованы в разных модулях (будь то [набор API](windows-apisets.md) или библиотека DLL), но библиотека "тег-объект" абстрагирует подробности от вас, делая приложение более переносимым в разных версиях операционных систем. Просто Свяжите свое классическое приложение или драйвер с библиотекой-Отделом, содержащей набор API-интерфейсов, которые вас интересуют, и все, что нужно сделать. 

| Библиотека | Описание |
|------------------------|-------------------------|
| OneCore. lib | эта библиотека предоставляет экспорты для подмножества api-интерфейсов Win32, которые являются общими для всех Windows 10 устройств. свяжите свое классическое приложение или драйвер с OneCore. lib (и без других библиотек) для доступа к этим api. если вы свяжете приложение или драйвер с OneCore. lib и в этой библиотеке вызываются только api-интерфейсы Win32, приложение или драйвер будут успешно загружены на всех Windows 10 устройствах.         |
| OneCore_apiset. lib | эта библиотека предоставляет то же покрытие, что и OneCore. lib, но использует [прямую пересылку API set](api-set-loader-operation.md#direct-forwarding). обратите внимание, что использование этой библиотеки будет совместимо только с версией Windows 10, указанной в версии SDK или более поздних версий.  |
| Онекореуап. lib | эта библиотека предоставляет экспорты для подмножества api-интерфейсов Win32, которые являются общими для всех Windows 10 устройств, поддерживающих среда выполнения Windows (WinRT). Свяжите свое классическое приложение или драйвер с Онекореуап. lib (и без других библиотек) для доступа к этим API. если вы свяжете приложение или драйвер с онекореуап. lib и в этой библиотеке вызываются только api-интерфейсы Win32, приложение или драйвер будут успешно загружены на всех Windows 10 устройствах, поддерживающих UWP. |
| OneCoreUAP_apiset. lib | Эта библиотека предоставляет то же покрытие, что и Онекореуап. lib, но использует метод [прямого перенаправления API](api-set-loader-operation.md#direct-forwarding) . обратите внимание, что использование этой библиотеки будет совместимо только с версией Windows 10, указанной в требуемом пакете SDK, или более поздней версии.  |
