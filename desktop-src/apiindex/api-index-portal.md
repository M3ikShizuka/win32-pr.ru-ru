## <a name="win32-windows-api"></a>Win32 (API Windows)

api Win32 (также называемый Windows api) — это собственная платформа для Windows приложений. Этот API лучше подходит для настольных приложений, которым требуется прямой доступ к системным функциям и оборудованию. Windows API можно использовать во всех классических приложениях, а одни и те же функции обычно поддерживаются в 32-и 64-разрядных Windows.

## <a name="windows-runtime-winrt"></a>Windows Среда выполнения (WinRT)

WinRT — это ведущая платформа для Windows 10 приложений и игр, включая классические приложения. API WinRT подходит как для машинного C++, так и для управляемых настольных приложений, для которых требуется сложный пользовательский интерфейс, Настройка стилей и сценарии с большим объемом графики.
