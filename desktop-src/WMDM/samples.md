# <a name="wmdm-samples"></a>Примеры ВМДМ

Пакет Windows Media диспетчер устройств SDK содержит два компилируемых примера: Пример графического классического приложения, позволяющего просматривать подключенные устройства, создавать списки воспроизведения и перетаскивать файлы на устройство и обратно. и пример поставщика услуг, который обменивается данными с виртуальными устройствами на компьютере (которые фактически являются жесткими дисками на компьютере). В документации объясняется, как скомпилировать и запустить эти проекты, а также приводятся сведения о том, как они работают.

Эти два проекта описаны в следующих разделах:

-   [Пример классического приложения](sample-desktop-application.md)
-   [Образец поставщика услуг](sample-service-provider.md)
