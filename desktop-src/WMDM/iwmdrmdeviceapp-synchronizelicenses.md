# <a name="iwmdrmdeviceappsynchronizelicenses-method"></a>Метод Ивмдрмдевицеапп:: Синчронизелиценсес

Метод **синчронизелиценсес** обновляет лицензии на устройстве, когда срок их действия приближается к истечению.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SynchronizeLicenses(
  [in] IWMDMDevice    *pDevice,
  [in] IWMDMProgress3 *pProgressCallback,
  [in] DWORD          cMinCountThreshold,
  [in] DWORD          cMinHoursThreshold
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пдевице* \[ окне\]
</dt> <dd>

Указатель на объект [**ивмдмдевице**](/windows/desktop/api/mswmdm/nn-mswmdm-iwmdmdevice) .

</dd> <dt>

*ппрогресскаллбакк* \[ окне\]
</dt> <dd>

Обратный вызов хода выполнения, который получит ход выполнения всех шагов, которые может потребоваться выполнить. Шаг определяется параметром *EventID* метода [**IWMDMProgress3**](/windows/desktop/api/mswmdm/nn-mswmdm-iwmdmprogress3) с именем.

</dd> <dt>

*кминкаунтсрешолд* \[ окне\]
</dt> <dd>

Необязательный минимально оставшийся Счетчик воспроизведения для лицензии устройства.

</dd> <dt>

*кминхаурссрешолд* \[ окне\]
</dt> <dd>

Необязательно минимальное количество оставшихся часов для лицензии устройства.

</dd> </dl>

## <a name="remarks"></a>Комментарии

Этот вызов может выполняться только на устройстве, которое поддерживает Windows Media DRM 10 для переносных устройств. Необходимо указать хотя бы один параметр порога.
