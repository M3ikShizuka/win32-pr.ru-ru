# <a name="code-files-for-the-source-provider-sample"></a>Файлы кода для примера поставщика источника

Пример проекта поставщика источника включает следующие файлы исходного кода со связанными заголовками:



| Файл                   | Описание                                                                                                                                                                                                     |
|------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| хдсппч. cpp            | Включает стандартные файлы ATL.                                                                                                                                                                                    |
| Key. c                  | Содержит фиктивный ключ проверки подлинности.                                                                                                                                                                            |
| логхелп. cpp            | Содержит функции, которые заводят в журнал действия и ошибки с помощью класса **вмдмлогжер** , который реализован в WMDMLOG.dll системном файле.                                                                         |
| Мдсервицепровидер. cpp  | Реализует класс Кмдсервицепровидер, реализующий интерфейсы [**имдсервицепровидер**](/windows/desktop/api/mswmdm/nn-mswmdm-imdserviceprovider) и икомпонентаусентикате.                                                             |
| МДСП. cpp               | Точка входа DLL и регистрационный код.                                                                                                                                                                          |
| Мдспдевице. cpp         | Реализует класс Кмдспдевице, реализующий интерфейсы [**IMDSPDevice2**](/windows/desktop/api/mswmdm/nn-mswmdm-imdspdevice2), [**имдспдевицеконтрол**](/windows/desktop/api/mswmdm/nn-mswmdm-imdspdevicecontrol)и **испеЦифипропертипажес** .                          |
| Мдспенумдевице. cpp     | Реализует класс Кмдспенумдевице, реализующий интерфейс [**имдспенумдевице**](/windows/desktop/api/mswmdm/nn-mswmdm-imdspenumdevice) .                                                                                                  |
| Мдспенумстораже. cpp    | Реализует класс Кмдспенумстораже, реализующий интерфейс [**имдспенумстораже**](/windows/desktop/api/mswmdm/nn-mswmdm-imdspenumstorage) .                                                                                               |
| Мдспстораже. cpp        | Реализует класс Кмдспстораже, реализующий интерфейсы [**IMDSPStorage2**](/windows/desktop/api/mswmdm/nn-mswmdm-imdspstorage2), [**имдспобжектинфо**](/windows/desktop/api/mswmdm/nn-mswmdm-imdspobjectinfo)и [**имдспобжект**](/windows/desktop/api/mswmdm/nn-mswmdm-imdspobject) .                    |
| Мдспсторажеглобалс. cpp | Реализует класс Кмдспсторажеглобалс, реализующий интерфейс [**имдспсторажеглобалс**](/windows/desktop/api/mswmdm/nn-mswmdm-imdspstorageglobals) .                                                                                      |
| Мдспутил. cpp           | Содержит различные служебные функции для управления устройствами и файлами.                                                                                                                                              |
| заменяющий. cpp           | Реализует класс Кпроппаже, наследующий классы ATL **ипропертипажеимпл** (для реализации ипропертипаже) и **CDialogImpl**, который наследует класс ATL CDialogImpl (для управления окнами и сообщениями). |
