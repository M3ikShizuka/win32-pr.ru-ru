# <a name="iwmdrmdevicegetsynclist-method"></a>Метод Ивмдрмдевице:: Жетсинклист

Метод **жетсинклист** извлекает список синхронизации лицензий на устройстве. Синхронизация лицензий позволяет главному компьютеру передавать обновленные лицензии на устройство в соответствии с указанными критериями.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetSyncList(
  [in]  DWORD cMinCountThreshold,
  [in]  DWORD cMinHoursThreshold,
  [out] BYTE  **ppbSyncList,
  [out] DWORD *pcbSyncList
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*кминкаунтсрешолд* \[ окне\]
</dt> <dd>

Пороговое значение минимального числа.

</dd> <dt>

*кминхаурссрешолд* \[ окне\]
</dt> <dd>

Минимальное пороговое значение часов.

</dd> <dt>

*ппбсинклист* \[ заполняет\]
</dt> <dd>

Получен список синхронизации лицензий.

</dd> <dt>

*пкбсинклист* \[ заполняет\]
</dt> <dd>

Размер списка синхронизации лицензий в байтах.

</dd> </dl>

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |
