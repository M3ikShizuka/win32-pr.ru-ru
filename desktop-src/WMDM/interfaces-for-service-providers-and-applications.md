# <a name="interfaces-for-service-providers-and-applications"></a>Интерфейсы для поставщиков служб и приложений

следующие интерфейсы могут использоваться поставщиками служб и приложениями, которые используют Windows Media диспетчер устройств SDK.



| Интерфейс                                                | Описание                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**ивмдмлогжер**](/windows/desktop/api/wmdmlog/nn-wmdmlog-iwmdmlogger)                       | Записывает записи в общий файл журнала.                                                        |
| [**икомпонентаусентикате**](/windows/desktop/api/mswmdm/nn-mswmdm-icomponentauthenticate) | обеспечивает безопасный зашифрованный обмен данными между модулями Windows Media диспетчер устройств. |
