# <a name="interfaces-for-service-providers-and-applications"></a>Интерфейсы для поставщиков служб и приложений

Для поставщиков служб и приложений, использующих пакет SDK для Windows Media диспетчер устройств, могут использоваться следующие интерфейсы.



| Интерфейс                                                | Описание                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**ивмдмлогжер**](/windows/desktop/api/wmdmlog/nn-wmdmlog-iwmdmlogger)                       | Записывает записи в общий файл журнала.                                                        |
| [**икомпонентаусентикате**](/windows/desktop/api/mswmdm/nn-mswmdm-icomponentauthenticate) | Обеспечивает безопасное зашифрованное взаимодействие между модулями Windows Media диспетчер устройств. |
