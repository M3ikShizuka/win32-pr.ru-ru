# <a name="using-secure-authenticated-channels"></a>Использование защищенных каналов с проверкой подлинности

Windows Media диспетчер устройств обеспечивает проверку подлинности и безопасный обмен данными между компонентами, предоставляя два вспомогательных класса: [ксекуречаннелклиент](csecurechannelclient-class.md) (для приложений) и [ксекуречаннелсервер](csecurechannelserver-class.md) (для поставщиков услуг) и один интерфейс [**икомпонентаусентикате**](/windows/desktop/api/mswmdm/nn-mswmdm-icomponentauthenticate) (для обоих). Вместе они составляют API для использования защищенных каналов с проверкой подлинности (SAC). Специальная консоль администрирования (SAC) обрабатывает следующие три задачи для поставщиков услуг или приложений, использующих Windows Media диспетчер устройств:

-   [Проверка подлинности компонента](component-authentication.md)
-   [Шифрование и расшифровка](encryption-and-decryption.md)
-   [Проверка подлинности сообщений](message-authentication.md)

Поставщик приложения или службы должен поддерживать проверку подлинности, шифрование и расшифровку компонента. Проверка подлинности сообщений является необязательной.
