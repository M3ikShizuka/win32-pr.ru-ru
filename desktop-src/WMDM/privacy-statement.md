# <a name="privacy-statement-windows-media-device-manager-11-sdk"></a>Заявление о конфиденциальности (пакет SDK для Windows Media диспетчер устройств 11)

Разработчики, создающие диспетчер устройств приложения Windows Media, должны четко взаимодействовать с пользователями, как это может повлиять на их конфиденциальность. Чтобы помочь вам создать заявление о конфиденциальности для конечного пользователя, корпорация Майкрософт разместила на [веб-странице о конфиденциальности Windows Media](https://www.microsoft.com/windows/windowsmedia/privacy/10ssdk.aspx)рекомендации по конфиденциальности.
