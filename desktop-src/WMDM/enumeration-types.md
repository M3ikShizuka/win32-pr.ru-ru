# <a name="wmdm-enumerations"></a>Перечисления ВМДМ

В этом разделе документируется функция пакета SDK для Windows Media диспетчер устройств. Рекомендуется выполнить миграцию приложения для использования API переносных устройств Windows. Дополнительные сведения см. в пакете SDK для переносных устройств Windows.

Диспетчер устройств Windows Media определяет следующие перечисления.



| Тип перечисления                                                                  | Описание                                                                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**\_ \_ \_ форма допустимых \_ значений \_ вмдм enum Prop**](wmdm-enum-prop-valid-values-form.md) | Описывает возможные формы допустимых значений для свойства.                                        |
| [**ВМДМ \_ Поиск \_ области**](wmdm-find-scope.md)                                      | Определяет область объекта хранилища.                                                        |
| [**ВМДМ \_ форматкоде**](wmdm-formatcode.md)                                       | Определяет список кодов форматов, описывающих типы содержимого, передаваемые на устройство и с устройства. |
| [**\_тип сеанса \_ вмдм**](wmdm-session-type.md)                                  | Определяет тип сеанса.                                                                       |
| [**\_ \_ режим перечисления вмдм в хранилище \_**](wmdm-storage-enum-mode.md)                       | Определяет способ перечисления содержимого в хранилище.                                     |
| [**\_ \_ тип данных тега вмдм**](wmdm-tag-datatype.md)                                  | Определяет тип данных.                                                                            |
| [**вмдммессаже**](wmdmmessage.md)                                                | Определяет типы сообщений и состояния.                                                               |
