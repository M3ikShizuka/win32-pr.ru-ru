# <a name="wmdm-programming-guide"></a>ВМДМ по программированию

в этом разделе описываются действия, которые должны выполняться с помощью приложения или поставщика услуг, созданного на Windows носителях диспетчер устройств.

-   [Задачи, общие для приложений и поставщиков служб](tasks-common-to-applications-and-service-providers.md) , описывают задачи, выполняемые приложениями и поставщиками служб, такие как ведение журнала, шифрование и расшифровка, а также проверка подлинности.
-   [создание Windows мультимедиа диспетчер устройств приложение](creating-a-windows-media-device-manager-application.md) описывает все действия, которые обычно необходимо выполнить приложению.
-   [Создание поставщика услуг](creating-a-service-provider.md) описывает все действия, которые обычно необходимо выполнить поставщику услуг.
