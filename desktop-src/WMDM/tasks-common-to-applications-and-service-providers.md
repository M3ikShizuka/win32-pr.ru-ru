# <a name="tasks-common-to-applications-and-service-providers"></a>Задачи, общие для приложений и поставщиков служб

в этом разделе описаны следующие задачи, которые относятся к приложениям и поставщикам услуг, построенным на Windows диспетчер устройств носителей.

-   [Компиляция IDL-файлов, поставляемых с пакетом SDK](compiling-the-idl-files-supplied-with-the-sdk.md) , описывает, какие IDL-файлы используются для создания файлов заголовков, если вы решили не использовать файлы заголовков, предоставляемые вместе с пакетом SDK.
-   [включение ведения журнала](enabling-logging.md) описывает объект ведения журнала, предоставляемый Windows Media диспетчер устройств для записи данных из выполняющегося объекта.
-   [Использование защищенных каналов, прошедших проверку](using-secure-authenticated-channels.md) подлинности, описывает защищенные каналы, которые используются для шифрования, расшифровки и проверки подлинности компонентов и сообщений. все, что основано на Windows Media диспетчер устройств, должны иметь по крайней мере возможность выполнять проверку подлинности компонента.
-   [обработка защищенного содержимого](handling-protected-content.md) описывает, как ваше приложение может обрабатывать содержимое, защищенное с помощью Windows Media digital rights management (DRM).
-   [Получение и установка метаданных и атрибутов](getting-and-setting-metadata-and-attributes.md) . описывает, как приложение может извлекать и устанавливать атрибуты и метаданные для хранилища или устройства.
