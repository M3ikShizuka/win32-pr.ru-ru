# <a name="windows-media-drm-implemented-interfaces"></a>Интерфейсы DRM-Implemented Windows Media

Следующие интерфейсы не предназначены для реализации сторонними поставщиками услуг, но реализуются и вызываются внутри Windows Media DRM и Windows Media диспетчер устройств. Реализация этих интерфейсов предотвратит обработку содержимого, защищенного с использованием DRM. Они задокументированы только для полноты.

-   [**Интерфейс Ивмдрмдевице**](iwmdrmdevice.md)
-   [**Интерфейс IWMDRMDevice2**](iwmdrmdevice2.md)
