# <a name="windows-media-drm-implemented-interfaces"></a>Windows Интерфейсы DRM-Implemented мультимедиа

следующие интерфейсы не предназначены для реализации сторонними поставщиками услуг, но реализуются и вызываются внутренним образом Windows media DRM и Windows media диспетчер устройств. Реализация этих интерфейсов предотвратит обработку содержимого, защищенного с использованием DRM. Они задокументированы только для полноты.

-   [**Интерфейс Ивмдрмдевице**](iwmdrmdevice.md)
-   [**Интерфейс IWMDRMDevice2**](iwmdrmdevice2.md)
