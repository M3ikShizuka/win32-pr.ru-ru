# <a name="wmdm-reference"></a>Справочник по ВМДМ

пакет SDK для Windows Media диспетчер устройств состоит из коллекции интерфейсов, структур, перечислений и констант. В разделе Reference интерфейсы разбиваются на группы в соответствии с типом объекта, который их использует.



| Section                                                                                                    | Описание                                                                                                                                                                                         |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Интерфейсы для приложений](interfaces-for-applications.md)                                             | описывает интерфейсы, которые используются или реализуются только приложениями для настольных систем, проигрыватель Windows Media подключаемых модулей или COM-объектами, которым требуется высокоуровневый доступ к портативному устройству.                     |
| [Интерфейсы для поставщиков услуг](interfaces-for-service-providers.md)                                   | Описание интерфейсов, которые используются или реализуются только поставщиками служб, которые обрабатывают фактическое взаимодействие низкого уровня с портативным устройством.                                                |
| [Windows Интерфейсы DRM-Implemented мультимедиа](windows-media-drm-implemented-interfaces.md)                   | описывает интерфейсы, которые не предназначены для реализации сторонними поставщиками услуг, но реализуются и вызываются внутренним образом Windows media DRM и Windows media диспетчер устройств. |
| [Интерфейсы для поставщиков служб и приложений](interfaces-for-service-providers-and-applications.md) | Описывает интерфейсы, которые могут использоваться приложениями или поставщиками служб.                                                                                                               |
| [Интерфейсы для поставщиков защищенного содержимого](interfaces-for-secure-content-providers.md)                     | Описывает интерфейсы, которые должны быть реализованы устройством или приложением, которое будет использовать содержимое, защищенное с помощью DRM.                                                                                       |
| [Структуры](structures.md)                                                                               | описывает структуры, используемые для Windows методов диспетчер устройств мультимедиа.                                                                                                                                 |
| [Типы перечислений](enumeration-types.md)                                                                 | описывает перечисления, используемые для Windows методов мультимедиа диспетчер устройств.                                                                                                                               |
| [Константы метаданных](metadata-constants.md)                                                               | описывает константы, определенные с помощью Windows Media диспетчер устройств SDK для настройки или получения различных свойств метаданных для устройства или объекта на устройстве.                                 |
| [Код ошибки](error-codes.md)                                                                             | описание кодов ошибок, которые могут быть возвращены методами Windows Media диспетчер устройств SDK.                                                                                                         |
