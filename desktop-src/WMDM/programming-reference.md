# <a name="wmdm-reference"></a>Справочник по ВМДМ

Пакет Windows Media диспетчер устройств SDK состоит из коллекции интерфейсов, структур, перечислений и констант. В разделе Reference интерфейсы разбиваются на группы в соответствии с типом объекта, который их использует.



| Section                                                                                                    | Описание                                                                                                                                                                                         |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Интерфейсы для приложений](interfaces-for-applications.md)                                             | Описывает интерфейсы, которые используются или реализуются только приложениями для настольных систем, подключаемых модулей проигрывателя Windows Media или COM-объектами, которым требуется высокоуровневый доступ к портативному устройству.                     |
| [Интерфейсы для поставщиков услуг](interfaces-for-service-providers.md)                                   | Описание интерфейсов, которые используются или реализуются только поставщиками служб, которые обрабатывают фактическое взаимодействие низкого уровня с портативным устройством.                                                |
| [Интерфейсы DRM-Implemented Windows Media](windows-media-drm-implemented-interfaces.md)                   | Описывает интерфейсы, которые не предназначены для реализации сторонними поставщиками услуг, но реализуются и вызываются внутри Windows Media DRM и Windows Media диспетчер устройств. |
| [Интерфейсы для поставщиков служб и приложений](interfaces-for-service-providers-and-applications.md) | Описывает интерфейсы, которые могут использоваться приложениями или поставщиками служб.                                                                                                               |
| [Интерфейсы для поставщиков защищенного содержимого](interfaces-for-secure-content-providers.md)                     | Описывает интерфейсы, которые должны быть реализованы устройством или приложением, которое будет использовать содержимое, защищенное с помощью DRM.                                                                                       |
| [Структуры](structures.md)                                                                               | Описывает структуры, используемые для методов диспетчер устройств Windows Media.                                                                                                                                 |
| [Типы перечислений](enumeration-types.md)                                                                 | Описывает перечисления, используемые для методов диспетчер устройств Windows Media.                                                                                                                               |
| [Константы метаданных](metadata-constants.md)                                                               | Описывает константы, определенные с помощью пакета SDK Windows Media диспетчер устройств для настройки или получения различных свойств метаданных для устройства или объекта на устройстве.                                 |
| [Код ошибки](error-codes.md)                                                                             | Описание кодов ошибок, которые могут возвращаться методами Windows Media диспетчер устройств SDK.                                                                                                         |
