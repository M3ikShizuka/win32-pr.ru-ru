# <a name="wmdmrights-structure"></a>Структура ВМДМРИГХТС

Структура **вмдмригхтс** описывает права на использование содержимого.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct __WMDMRIGHTS {
  UINT         cbSize;
  DWORD        dwContentType;
  DWORD        fuFlags;
  DWORD        fuRights;
  DWORD        dwAppSec;
  DWORD        dwPlaybackCount;
  WMDMDATETIME ExpirationDate;
} WMDMRIGHTS, *PWMDMRIGHTS;
```



## <a name="members"></a>Члены

<dl> <dt>

**кбсизе**
</dt> <dd>

Размер структуры в байтах.

</dd> <dt>

**двконтенттипе**
</dt> <dd>

**DWORD** , содержащий тип содержимого.

</dd> <dt>

**фуфлагс**
</dt> <dd>

Битовое поле, указывающее параметры прав, используемые для содержимого.



| Значение                        | Описание                                  |
|------------------------------|----------------------------------------------|
| ВМДМ \_ Rights \_ плайбакккаунт  | Количество раз, когда файл можно воспроизвести. |
| ВМДМ \_ Rights \_ EXPIRATIONDATE | Дата окончания срока действия файла.                 |
| ВМДМ \_ Rights \_ фрисериалидс  | Бесплатный серийный идентификатор файла.          |
| \_ \_ Группа GROUPID Rights вмдм  | Идентификатор файла.                      |
| ВМДМ \_ Rights \_ намедсериалидс | Именованный последовательный идентификатор файла.         |
