# <a name="iwmdrmdevice2-interface"></a>Интерфейс IWMDRMDevice2

Этот интерфейс не предназначен для реализации поставщиком услуг, но предоставляется в целях полной документации.

Интерфейс **IWMDRMDevice2** расширяет **ивмдрмдевице** , предоставляя методы для получения состояния лицензии и списка частичной синхронизации.

## <a name="members"></a>Элементы

Интерфейс **IWMDRMDevice2** наследует от [**ивмдрмдевице**](iwmdrmdevice.md). **IWMDRMDevice2** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **IWMDRMDevice2** содержит следующие методы.



| Метод                                                         | Описание                                     |
|:---------------------------------------------------------------|:------------------------------------------------|
| [**жетлиценсестате**](iwmdrmdevice2-getlicensestate.md)       | Возвращает состояние лицензии.<br/>              |
| [**жетпартиалсинклист**](iwmdrmdevice2-getpartialsynclist.md) | Возвращает список частичной синхронизации.<br/> |
