# <a name="sample-desktop-application"></a>Пример классического приложения

Диспетчер устройств Windows Media поставляется с примером классического приложения под названием вмдмапп. Это приложение имеет графический пользовательский интерфейс, позволяющий просматривать подключенные устройства, создавать списки воспроизведения на устройстве и перетаскивать файлы мультимедиа на устройство.

Этот пример приложения состоит из двух частей:

-   wmdapp.exe, исполняемый файл, который отображает окно и выполняет большую часть пользовательского интерфейса и основных функций программы, и
-   proghelp.dll, вспомогательная библиотека DLL, которая выполняет служебные функции для приложения. Эту библиотеку DLL необходимо зарегистрировать после сборки проекта.

Чтобы скомпилировать пример приложения, можно использовать Visual Studio. В следующем разделе описано, как это сделать.

-   [Компиляция примера приложения с помощью Visual Studio](compiling-the-sample-application-using-visual-studio.md)

После компиляции проекта зарегистрируйте файл proghelp.dll. Чтобы зарегистрировать эту библиотеку DLL, откройте командную строку и перейдите в каталог, содержащий эту библиотеку DLL, и введите `regsvr32 proghelp.dll` .
