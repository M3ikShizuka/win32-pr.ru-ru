# <a name="compiling-the-sample-service-provider-using-nmake"></a>Компиляция образца поставщика услуг с помощью NMAKE

Чтобы скомпилировать пример поставщика службы с помощью NMAKE, Откройте командное окно Visual Studio, перейдите к <*путь установки пакета SDK* > \\ WMFSDK95 \\ вмдм \\ МДСП и введите `nmake` в командной строке.
