# <a name="xaudio2-macros"></a>XAudio2 макросы

В этом разделе содержатся сведения о макросах, предоставляемых Microsoft XAudio2 API.



| Макрос                          | Описание                                                                                                                                               |
|--------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ксапоаллок**](/windows/desktop/api/XAPO/nf-xapo-xapoalloc) | Макрос выделения памяти, используемый методами ИКСАПО, которые должны выделить структуры произвольного размера, которые впоследствии возвращаются приложению.<br/> |
| [**ксапофри**](/windows/desktop/api/XAPO/nf-xapo-xapofree)   | Макрос, используемый для освобождения памяти, выделенной с помощью макроса Ксапоаллок.<br/>                                                                                  |
