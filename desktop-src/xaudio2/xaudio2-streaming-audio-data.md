# <a name="xaudio2-streaming-audio-data"></a>XAudio2 потоковая передача звуковых данных

Потоковая передача — это процесс сохранения только небольшой части воспроизводимого звукового файла в памяти. Это позволяет воспроизводить большие звуковые файлы, например фоновую музыку, и не занимать большой объем памяти.

При потоковой передаче звукового файла его данные считываются с диска в виде фрагментов, а не загружаются сразу весь файл. Потоковая передача осуществляется асинхронным чтением звуковых данных в очередь буферов диска. Каждый буфер заполняется, а затем отправляется на исходный голоса. После того, как голосовое воспроизведение пойдет из буфера, буфер снова станет доступным для чтения. Таким образом, циклический перебор буферов позволяет воспроизводить большой звуковой файл, пока загружается только часть его данных. Код потоковой передачи должен размещаться в отдельном потоке, где он может находиться в спящем режиме при ожидании завершения длительных операций с диском и аудио. Класс обратного вызова используется для пробуждения потока путем активации событий после завершения операций аудио.

Пример того, как можно выполнить потоковую передачу с помощью XAudio2, см. [в разделе как передавать звук с диска](how-to--stream-a-sound-from-disk.md).
