# <a name="jet_enumcolumnvalue-class"></a>Класс JET_ENUMCOLUMNVALUE

[Жетенумератеколумнс (JET_SESID, JET_TABLEID, Int32, \[ \] , int32, \[ \] , JET_PFNREALLOC, IntPtr, Int32, енумератеколумнсгрбит)](./api.jetenumeratecolumns-method.md) возвращает массив структур JET_ENUMCOLUMNVALUE.
