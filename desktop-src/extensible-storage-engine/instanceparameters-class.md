# <a name="instanceparameters-class"></a>Класс Инстанцепараметерс

Этот класс предоставляет свойства для установки и получения системных параметров в экземпляре ESENT. Этот класс предоставляет статические свойства для установки и получения системных параметров ESENT для каждого экземпляра.
