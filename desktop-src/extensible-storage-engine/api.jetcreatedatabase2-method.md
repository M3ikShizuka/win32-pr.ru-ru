# <a name="apijetcreatedatabase2-method"></a>API. JetCreateDatabase2, метод

Создает и прикрепляет файл базы данных с указанным максимальным размером базы данных.

**Пространство имен:**  [Microsoft. ISAM. ESENT. Interop](./microsoft.isam.esent.interop-namespace.md)  
**Сборка:**  Microsoft. ISAM. ESENT. Interop (в Microsoft.Isam.Esent.Interop.dll)

## <a name="syntax"></a>Синтаксис

``` vb
'Declaration
Public Shared Sub JetCreateDatabase2 ( _
    sesid As JET_SESID, _
    database As String, _
    maxPages As Integer, _
    <OutAttribute> ByRef dbid As JET_DBID, _
    grbit As CreateDatabaseGrbit _
)
'Usage
Dim sesid As JET_SESID
Dim database As String

Dim dbid As JET_DBID
```

``` csharp
public static void JetCreateDatabase2(
```

#### <a name="parameters"></a>Параметры

  - сесид  
    Тип: [Microsoft.ISAM.ESENT.Interop.JET_SESID](./jet-sesid-structure.md)  
    
    Используемый сеанс.

<!-- end list -->

  - База данных  
    Тип: [System. String](/dotnet/api/system.string)  
    
    Путь к создаваемому файлу базы данных.

<!-- end list -->

  - макспажес  
    Тип: [System. Int32](/dotnet/api/system.int32)  
    
    Максимальный размер базы данных в страницах базы данных. Передача значения 0 означает отсутствие принудительного применения максимального значения.

<!-- end list -->

  - dbid  
    Тип: [Microsoft.ISAM.ESENT.Interop.JET_DBID](./jet-dbid-structure.md)  
    
    Возвращает DBID новой базы данных.
