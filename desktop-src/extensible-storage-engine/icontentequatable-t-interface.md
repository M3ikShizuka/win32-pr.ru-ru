# <a name="icontentequatablet-interface"></a>\<T\>Интерфейс иконтентекуатабле

Интерфейс для объектов, содержимое которых может быть сравнено друг с другом. Он должен использоваться для сравнения на равенство изменяемых ссылочных объектов, если переопределение Equals () и GetHashCode () не является хорошей идеей.

**Пространство имен:**  [Microsoft. ISAM. ESENT. Interop](./microsoft.isam.esent.interop-namespace.md)  
**Сборка:**  Microsoft. ISAM. ESENT. Interop (в Microsoft.Isam.Esent.Interop.dll)

## <a name="syntax"></a>Синтаксис

``` vb
'Declaration
Public Interface IContentEquatable(Of T)
'Usage
Dim instance As IContentEquatable(Of T)
```

``` csharp
public interface IContentEquatable<T>
```

#### <a name="type-parameters"></a>Параметры типа

  - T  
    Тип объектов для комапре.
