# <a name="windows7grbitsforcenewlog-field"></a>Windows7Grbits. Форценевлог, поле

Принудительно создать новый файл журнала. Этот параметр может использоваться, даже если сеанс в данный момент не находится в транзакции. Этот параметр нельзя использовать в сочетании с любым другим параметром.
