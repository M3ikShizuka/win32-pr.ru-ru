# <a name="instanceparameterstempdirectory-property"></a>Инстанцепараметерс. TempDirectory, свойство

Возвращает или задает относительный или абсолютный путь файловой системы к папке, которая будет содержать временную базу данных для экземпляра.

**Пространство имен:**  [Microsoft. ISAM. ESENT. Interop](./microsoft.isam.esent.interop-namespace.md)  
**Сборка:**  Microsoft. ISAM. ESENT. Interop (в Microsoft.Isam.Esent.Interop.dll)

    Set
