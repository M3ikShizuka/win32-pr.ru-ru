# <a name="windows7paramwaypointlatency-field"></a>Windows7Param. Вайпоинтлатенци, поле

Этот параметр задает количество журналов, в которых ESENT будет откладывать сбросы базы данных. Это можно использовать для повышения возможности восстановления базы данных, если ошибки приводят к потере файлов журнала.

**Пространство имен:**  [Microsoft. ISAM. ESENT. Interop. Windows7](./microsoft.isam.esent.interop.windows7-namespace.md)  
**Сборка:**  Microsoft. ISAM. ESENT. Interop (в Microsoft.Isam.Esent.Interop.dll)
