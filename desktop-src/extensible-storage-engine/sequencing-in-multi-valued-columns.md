# <a name="sequencing-in-multi-valued-columns"></a>Последовательность в столбцах с несколькими значениями


_**Применимо к:** Windows | Windows Сервером_

## <a name="sequencing-in-multi-valued-columns"></a>Последовательность в столбцах с несколькими значениями

Значения в столбце с несколькими значениями идентифицируются по номеру индекса, который называется *итагсекуенце*. Это число является ссылкой на одно значение столбца. Этот номер используется, когда новые значения задаются, извлекаются или перечисляются в столбце. *Итагсекуенце* используется в различных структурах, включая:

  - [JET_RETINFO](./jet-retinfo-structure.md)

  - [JET_SETINFO](./jet-setinfo-structure.md)

  - [JET_SETCOLUMN](./jet-setcolumn-structure.md)

  - [JET_RETRIEVECOLUMN](./jet-retrievecolumn-structure.md)

  - [JET_ENUMCOLUMNVALUE](./jet-enumcolumnvalue-structure.md)

Этот *итагсекуенце* начинается с 1 для каждого значения в столбце с несколькими значениями. Порядковый номер увеличивается при добавлении новых значений в столбец. Установка значения в столбце с параметром *итагсекуенце* , равным 0, означает, что значение будет добавлено к набору значений, уже наданных в этом столбце. Использование *итагсекуенце* больше, чем количество значений, заданных для столбца, будет иметь одинаковый результат. Если указать *итагсекуенце* существующего значения, это значение будет перезаписано без вставки нового значения. Установка существующего значения равным NULL приведет к удалению этого значения из набора значений, уже наданных в этом столбце. Все последующие значения в столбце будут переноситься вниз на один слот, что последующий доступ к этим значениям с помощью *итагсекуенце* будет иметь номер, который еще не использовался ранее.

На следующей диаграмме показана *итагсекуенце* в столбце с несколькими значениями. Столбец с именем ColA имеет три записи: val1, val2 и Val3 с *итагсекуенце* 1, 2 и 3 соответственно.
