# <a name="jet_dateserial"></a>JET_DATESERIAL


_**Применимо к:** Windows | Windows Server_

## <a name="jet_dateserial"></a>JET_DATESERIAL

Тип данных **JET_DATESERIAL** представляет дату в дробных днях, начиная с 100 года.

```cpp
    typedef double JET_DATESERIAL;
```

### <a name="data-types"></a>Типы данных

JET_DATESERIAL

Содержит число с плавающей запятой двойной точности (8 байт). Это то же самое, что и тип [даты](https://msdn.microsoft.com/library/82ab7w69(VS.71).aspx) C++.
