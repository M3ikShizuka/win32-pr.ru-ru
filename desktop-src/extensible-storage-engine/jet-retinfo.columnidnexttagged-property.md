# <a name="jet_retinfocolumnidnexttagged-property"></a>Свойство JET_RETINFO. Колумниднексттагжед

Получает значение columnid полученного помеченного, многозначного или разреженного столбца, когда все отмеченные столбцы извлекаются путем передачи 0 в качестве значения columnid в Жетретриевеколумн.

**Пространство имен:**  [Microsoft. ISAM. ESENT. Interop](./microsoft.isam.esent.interop-namespace.md)  
**Сборка:**  Microsoft. ISAM. ESENT. Interop (в Microsoft.Isam.Esent.Interop.dll)
