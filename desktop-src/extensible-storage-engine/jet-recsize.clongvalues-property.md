# <a name="jet_recsizeclongvalues-property"></a>Свойство JET_RECSIZE. Клонгвалуес

Возвращает общее количество значений long, хранящихся в дереве Long-Value для этой записи. Это не относится к внутренним длинным значениям.

**Пространство имен:**  [Microsoft. ISAM. ESENT. Interop. Vista](./microsoft.isam.esent.interop.vista-namespace.md)  
**Сборка:**  Microsoft. ISAM. ESENT. Interop (в Microsoft.Isam.Esent.Interop.dll)
