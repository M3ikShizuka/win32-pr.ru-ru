# <a name="instanceparameterscreatepathifnotexist-property"></a>Инстанцепараметерс. Креатепасифнотексист, свойство

**Пространство имен:**  [Microsoft. ISAM. ESENT. Interop](./microsoft.isam.esent.interop-namespace.md)
