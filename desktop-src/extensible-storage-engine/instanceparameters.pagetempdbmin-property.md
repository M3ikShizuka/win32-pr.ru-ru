# <a name="instanceparameterspagetempdbmin-property"></a>Инстанцепараметерс. Пажетемпдбмин, свойство

Возвращает или задает начальный размер временной базы данных. Размер находится в страницах базы данных. Нулевой размер означает, что следует использовать стандартный размер обычной базы данных. Часто нежелательно, чтобы небольшие приложения настроили временную базу данных как можно меньше. Если задать для этого параметра значение [пажетемпдбсмаллест](./systemparameters.pagetempdbsmallest-field.md) , будет достигнута наименьшая временная база данных.

**Пространство имен:**  [Microsoft. ISAM. ESENT. Interop](./microsoft.isam.esent.interop-namespace.md)  
**Сборка:**  Microsoft. ISAM. ESENT. Interop (в Microsoft.Isam.Esent.Interop.dll)
