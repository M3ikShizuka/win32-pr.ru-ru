# <a name="apijetrollback-method"></a>API. Жетроллбакк, метод

Отменяет изменения, внесенные в состояние базы данных, и возвращает его в последнюю точку сохранения. Жетроллбакк также закроет курсоры, открытые во время точки сохранения. Если внешняя точка сохранения отменена, сеанс завершит транзакцию.

**Пространство имен:**  [Microsoft. ISAM. ESENT. Interop](./microsoft.isam.esent.interop-namespace.md)  
**Сборка:**  Microsoft. ISAM. ESENT. Interop (в Microsoft.Isam.Esent.Interop.dll)
