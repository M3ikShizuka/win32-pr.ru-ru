# <a name="apisetcolumn-method-jet_sesid-jet_tableid-jet_columnid-byte"></a>Метод API. Сетколумн (JET_SESID, JET_TABLEID, JET_COLUMNID, Byte)

**Пространство имен:**  [Microsoft. ISAM. ESENT. Interop](./microsoft.isam.esent.interop-namespace.md)
