# <a name="columnvalueofstructtlength-property"></a>Колумнвалуеофструкт \<T\> . Свойство Length

Возвращает длину в байтах значения столбца, равного нулю, если столбец имеет значение null, в противном случае соответствует размеру для этого столбца фиксированного размера.

**Пространство имен:**  [Microsoft. ISAM. ESENT. Interop](./microsoft.isam.esent.interop-namespace.md)  
**Сборка:**  Microsoft. ISAM. ESENT. Interop (в Microsoft.Isam.Esent.Interop.dll)
