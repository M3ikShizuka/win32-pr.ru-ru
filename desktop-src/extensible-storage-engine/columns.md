# <a name="columns"></a>Столбцы


_**Применимо к:** Windows | Windows Сервером_

## <a name="columns"></a>Столбцы

Таблицу можно создать либо с начальным набором столбцов, вызвав [жеткреатетаблеколумниндекс](./jetcreatetablecolumnindex-function.md) , либо без начального набора столбцов путем вызова [жеткреатетабле](./jetcreatetable-function.md). Таблицы в подсистеме ESE могут содержать до 127 столбцов фиксированной длины, 128 столбцов переменной длины и 64 993 столбцов с тегами. Столбцы идентифицируются по имени и ИДЕНТИФИКАТОРу и могут динамически добавляться в таблицу с помощью [жетаддколумн](./jetaddcolumn-function.md). Столбцы создаются с определенным типом данных и необязательным набором атрибутов, например, имеет ли столбец фиксированную длину или может принимать значение NULL.

Тип столбца определяет данные, которые могут храниться в столбце, и многие свойства столбца, включая его порядок для индексирования. ESE поддерживает широкий спектр типов столбцов, от 1 до 2 ГБ (2146483647 символов ASCII или 1073741823 символов Юникода). Полный список типов данных столбцов, поддерживаемых ESE, см. в [JET_COLTYP](./jet-coltyp.md) разделе. В следующих разделах обсуждаются некоторые типы столбцов, поддерживаемые ESE:

  - [Столбцы с тегами, фиксированными и переменными](./tagged-fixed-and-variable-columns.md)

  - [Версия, автоматический инкремент и столбцы с подстолбцами](./version-auto-increment-and-escrow-columns.md)

  - [Столбцы с длинными значениями](./long-value-columns.md)

  - [Разреженные столбцы с несколькими значениями](./multi-valued-sparse-columns.md)

  - [Определяемые пользователем столбцы](./user-defined-columns.md)

  - [Использование столбцов в таблице](./using-columns-in-a-table.md)
