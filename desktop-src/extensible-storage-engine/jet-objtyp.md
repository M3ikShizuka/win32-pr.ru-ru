# <a name="jet_objtyp"></a>JET_OBJTYP


_**Применимо к:** Windows | Windows Сервером_

## <a name="jet_objtyp"></a>JET_OBJTYP

**JET_OBJTYP** группа констант представляет тип объекта базы данных. В настоящее время поддерживаются только таблицы.


| <p>Константа/значение</p> | <p>Описание</p> | 
|-----------------------|--------------------|
| <p>JET_objtypNil<br />0</p> | <p>Представляет все типы объектов. В настоящее время поддерживаются только таблицы.</p> | 
| <p>JET_objtypTable<br />1</p> | <p>Представляет таблицу.</p> |
