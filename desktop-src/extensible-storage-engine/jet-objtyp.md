# <a name="jet_objtyp"></a>JET_OBJTYP


_**Применимо к:** Windows | Windows Server_

## <a name="jet_objtyp"></a>JET_OBJTYP

**JET_OBJTYP** группа констант представляет тип объекта базы данных. В настоящее время поддерживаются только таблицы.
