# <a name="methods-imanipulationprocessor"></a>Методы (IManipulationProcessor)

В этом разделе содержатся методы для интерфейса [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) .

Интерфейс [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) представляет следующие методы.



| Метод                                                                      | Описание                                                                              |
|-----------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| [**комплетеманипулатион**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-completemanipulation) | Этот метод вызывается, когда разработчик решает закончить манипуляцию.                |
| [**жетангуларвелоЦити**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getangularvelocity)     | Вычисляет скорость вращения, в которую перемещается целевой объект.                  |
| [**жетекспансионвелоЦити**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getexpansionvelocity) | Вычисляет частоту, с которой происходит расширение целевого объекта.                              |
| [**жетвелоЦитикс**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getvelocityx)                 | Вычисляет и возвращает горизонтальную скорость для целевого объекта.                    |
| [**Скорость**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-getvelocityy)                 | Вычисляет и возвращает вертикальную скорость.                                            |
| [**процессдовн**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processdown)                   | Передает данные в обработчик манипуляции, связанный с целевым объектом.                       |
| [**процессмове**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processmove)                   | Передает данные в обработчик манипуляции, связанный с целевым объектом.                       |
| [**процессуп**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processup)                       | Передает данные в обработчик манипуляции, связанный с целевым объектом.                       |
| [**процессдовнвистиме**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processdownwithtime)   | Данные каналов, включая метку времени для обработчика манипуляций, связанного с целевым объектом. |
| [**процессмовевистиме**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processmovewithtime)   | Данные каналов, включая метку времени для обработчика манипуляций, связанного с целевым объектом. |
| [**процессупвистиме**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-processupwithtime)       | Данные каналов, включая метку времени для обработчика манипуляций, связанного с целевым объектом. |
