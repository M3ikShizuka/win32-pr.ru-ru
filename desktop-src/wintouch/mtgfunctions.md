# <a name="functions-windows-touch-gestures"></a>функции (Windows сенсорные жесты)

этот раздел содержит функции для Windows сенсорных жестов.

для Windows сенсорных жестов используются следующие функции.



| Функция                                                 | Описание                                                                                             |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [**клосежестуреинфохандле**](/windows/desktop/api/winuser/nf-winuser-closegestureinfohandle) | Закрывает ресурсы, связанные с полученным вручную маркером, в структуру сведений о жесте.    |
| [**жетжестурикстрааргс**](/windows/desktop/api/winuser/nf-winuser-getgestureextraargs)       | Извлекает дополнительные сведения о жесте из его обработчика [**GESTUREINFO**](/windows/win32/api/winuser/ns-winuser-gestureinfo) .    |
| [**жетжестуреинфо**](/windows/desktop/api/winuser/nf-winuser-getgestureinfo)                 | Вручную извлекает структуру сведений о жесте, заданную маркером для структуры сведений о жесте. |
| [**жетжестуреконфиг**](/windows/desktop/api/winuser/nf-winuser-getgestureconfig)             | Извлекает сведения о конфигурации жестов для окна.                                           |
| [**сетжестуреконфиг**](/windows/desktop/api/winuser/nf-winuser-setgestureconfig)             | Задает конфигурацию жестов для окна.                                                            |
