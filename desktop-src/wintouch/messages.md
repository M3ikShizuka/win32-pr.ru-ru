# <a name="messages-touch-input"></a>Сообщения (сенсорный ввод)

В этом разделе содержатся сообщения для входных данных Мультисенсорная.

Для входных данных Мультисенсорная используются следующие сообщения.



| Сообщение                           | Описание                                                                                                         |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**WM \_ Touch**](wm-touchdown.md) | Указывает, что одна или несколько сенсорных точек, таких как палец или перо, затронули сенсорную поверхность дигитайзера. |
