# <a name="windows-touch-samples"></a>Примеры Windows Touch

Описывает путь к образцам Windows Touch и предоставляет краткие обзоры образцов.

При установке [образцов Windows 7 Touch]()можно выбрать, где устанавливаются образцы. Например, при установке на диск C пакетом SDK по умолчанию будет "C: \\ Program Files \\ Microsoft SDKs \\ Windows \\ v 7.0 \\ Samples". В каталоге Samples находится папка Touch. В этой папке содержатся все примеры Windows Touch.

Следующие примеры для Windows Touch поставляются с [пакетом SDK для Windows 7]().

- [Пример управления касанием Windows (Мтманипулатион)](windows-touch-manipulation-sample--mtmanipulation-.md)
