# <a name="windows-touch-samples"></a>Windows Примеры сенсорного ввода

описывает путь к примерам сенсорного ввода Windows и предоставляет краткие обзоры образцов.

при установке [образцов Windows 7 Touch]()можно выбрать, где устанавливаются образцы. например, при установке на диск c пакетом SDK по умолчанию будет "C: \\ Program files \\ Microsoft sdks \\ Windows \\ v 7.0 \\ samples". В каталоге Samples находится папка Touch. в этой папке содержатся все примеры Windows Touch.
