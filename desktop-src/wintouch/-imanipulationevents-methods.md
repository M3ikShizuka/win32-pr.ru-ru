# <a name="methods-_imanipulationevents"></a>Методы (_IManipulationEvents)

В этом разделе содержатся методы для интерфейса [**\_ иманипулатионевентс**](/windows/win32/api/manipulations/nn-manipulations-_imanipulationevents) .



| Метод                                                                      | Описание                                                   |
|-----------------------------------------------------------------------------|---------------------------------------------------------------|
| [**ManipulationStarted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationstarted)     | Обрабатывает событие при начале манипуляции или инерции.    |
| [**ManipulationDelta**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationdelta)         | Обрабатывает события, происходящие при изменении управляемого объекта. |
| [**ManipulationCompleted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationcompleted) | Обрабатывает событие при завершении манипуляции или инерции.      |
