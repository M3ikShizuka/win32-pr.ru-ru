# <a name="properties-imanipulationprocessor"></a>Свойства (IManipulationProcessor)

Этот раздел содержит свойства интерфейса [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) .

Интерфейс IManipulationProcessor представляет следующие свойства.



| Свойство                                                                            | Описание                                                                                                |
|-------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**минимумскалеротатерадиус**](imanipulationprocessor-minimumscalerotateradius.md) | Указывает, насколько велика дистанция контактов на жесте масштабирования или вращения для запуска манипуляций. |
| [**пивотпоинткс**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_pivotpointx)                           | Это свойство является горизонтальным центром объекта.                                                      |
| [**пивотпоинти**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_pivotpointy)                           | Это свойство является вертикальным центром объекта.                                                        |
| [**пивотрадиус**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_pivotradius)                           | Это свойство используется для определения того, какой объем вращения используется в манипуляциях с одним пальцем.                |
| [**суппортедманипулатионс**](/windows/desktop/api/manipulations/nf-manipulations-imanipulationprocessor-get_supportedmanipulations)     | Это свойство позволяет указать, какие манипуляции поддерживаются объектом.                          |
