# <a name="functions-boundary-feedback"></a>Функции (обратная связь)

В этом разделе описаны функции граничных отзывов.



| Функция                                               | Описание                                                                                        |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| [**бегинпаннингфидбакк**](/windows/win32/api/uxtheme/nf-uxtheme-beginpanningfeedback)     | Эта функция вызывается пользователем перед использованием обратной связи с границей.                               |
| [**ендпаннингфидбакк**](/windows/win32/api/uxtheme/nf-uxtheme-endpanningfeedback)     | Эта функция вызывается после завершения обратной связи и при необходимости запускает анимацию. |
| [**упдатепаннингфидбакк**](/windows/win32/api/uxtheme/nf-uxtheme-updatepanningfeedback) | Эта функция вызывается во время обратной связи с границей и при необходимости запускает анимацию.             |
