# <a name="touch-input"></a>Сенсорный ввод

## <a name="purpose"></a>Назначение

Windows поддерживает сенсорный ввод с помощью модели взаимодействия, предоставляемой комплексной платформой и платформы пользовательского интерфейса Windows touch.

## <a name="developer-audience"></a>Аудитория разработчиков

функции Windows сенсорного ввода предназначены для использования опытными разработчиками на C/C++, знакомыми с основными понятиями программирования компонентов (COM), а также общими сведениями о программировании пользовательского интерфейса и общих понятиях сенсорного взаимодействия.

## <a name="run-time-requirements"></a>Требования к среде выполнения

Windows 7
