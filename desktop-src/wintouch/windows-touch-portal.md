# <a name="touch-input"></a>Сенсорный ввод

## <a name="purpose"></a>Назначение

Windows поддерживает сенсорный ввод через модель взаимодействия, предоставляемую комплексной платформой и платформой пользовательского интерфейса Windows Touch.

## <a name="developer-audience"></a>Аудитория разработчиков

Функции технологии Windows Touch предназначены для использования опытными разработчиками на C/C++, знакомыми с основными понятиями программирования в модели COM, программировании пользовательского интерфейса и общими понятиями сенсорного взаимодействия.

## <a name="run-time-requirements"></a>Требования к среде выполнения

Windows 7
