# <a name="structures-windows-touch-gestures"></a>структуры (Windows сенсорные жесты)

этот раздел содержит структуры для Windows сенсорных жестов.

В этом разделе указаны следующие структуры.



| Структура                                          | Описание                                                                              |
|----------------------------------------------------|------------------------------------------------------------------------------------------|
| [**жестуреконфиг**](/windows/win32/api/winuser/ns-winuser-gestureconfig)             | Используется для задания конфигурации включения или отключения сообщений жестов.      |
| [**GESTUREINFO**](/windows/win32/api/winuser/ns-winuser-gestureinfo)                 | Содержит сведения о жесте.                                                    |
| [**жестуренотифиструкт**](/windows/win32/api/winuser/ns-winuser-gesturenotifystruct) | Содержит сведения об уведомлении [**\_ жестуренотифи WM**](wm-gesturenotify.md) . |
