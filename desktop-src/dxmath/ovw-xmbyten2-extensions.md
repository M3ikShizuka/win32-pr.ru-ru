# <a name="xmbyten2-extensions"></a>Расширения XMBYTEN2

[**XMBYTEN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmbyten2) описывает двумерный вектор для хранения подписанных нормализованных значений как целые 8-разрядные (1 байтовые) целочисленные значения.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
