# <a name="xmuint3-operators"></a>Операторы XMUINT3

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела

|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [**Оператор =**](/windows/win32/api/directxmath/nf-directxmath-xmuint3-operator-assign)<br/> | Назначает данные компонента вектора от одного экземпляра `XMUINT3` к текущему экземпляру `XMUINT3` . <br/> |
