| Раздел                                                                                 | Описание                                                                   |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|

| [**XMVector4AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4anglebetweennormalsest)<br/> | Вычисляет угол радианы между двумя нормализованными векторами 4D.<br/>      |
| [**XMVector4AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector4anglebetweenvectors)<br/>       | Вычисление угла радианы между двумя векторами 4D.<br/>                   |
| [**XMVector4ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector4clamplength)<br/>                       | Задает длину вектора 4D в заданный диапазон.<br/>                 |
| [**XMVector4ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector4clamplengthv)<br/>                     | Задает длину вектора 4D в заданный диапазон.<br/>                 |

| [**XMVector4InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector4inbounds)<br/>                             | Проверяет, находятся ли компоненты вектора 4D в пределах заданных границ.<br/> |
| [**XMVector4Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector4length)<br/>                                 | Вычисление длины вектора 4D.<br/>                                |
| [**XMVector4LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4lengthest)<br/>                           | Оценивает длину вектора 4D.<br/>                               |
| [**XMVector4LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector4lengthsq)<br/>                             | Вычисляет квадрат длины вектора 4D.<br/>                  |
| [**XMVector4Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector4normalize)<br/>                           | Возвращает нормализованную версию вектора 4D.<br/>                     |
