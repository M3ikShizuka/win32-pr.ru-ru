# <a name="xmubyten2-extensions"></a>Расширения XMUBYTEN2

[**XMUBYTEN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmubyten2) описывает двумерный вектор для хранения беззнаковых нормализованных значений в виде целых чисел без знака 8 бит (1 байт).

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
