# <a name="xmshortn2-extensions"></a>Расширения XMSHORTN2

[**XMSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn2) описывает двумерный вектор для хранения подписанных нормализованных значений как 16-разрядных целых чисел со знаком (тип `int16_t` ).

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
