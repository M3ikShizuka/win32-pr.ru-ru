# <a name="xmfloat2-constructors"></a>Конструкторы XMFLOAT2

Конструкторы, предоставляемые `XMFLOAT2` .

Конструирует экземпляры структуры [**XMFLOAT2**](/windows/win32/api/directxmath/ns-directxmath-xmfloat2) .

> [!Note]  
> Перечисленные здесь конструкторы доступны только в C++.

| Конструктор | Описание |

| [<strong>XMFLOAT2 (float, float)</strong>](/windows/win32/api/directxmath/nf-directxmath-xmfloat2-xmfloat2(float_float)) | Инициализирует новый экземпляр <code>XMFLOAT2</code> из двух <code>float</code> аргументов. <br /> Этот конструктор инициализирует новый экземпляр класса [<strong>XMFLOAT2</strong>](/windows/win32/api/directxmath/ns-directxmath-xmfloat2) из двух <code>float</code> аргументов. <br /><blockquote>[!Note]<br />Этот конструктор доступен только в C++.</blockquote><br /> |
