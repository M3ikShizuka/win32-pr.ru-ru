# <a name="xmint3-operators"></a>Операторы XMINT3

Структура [**XMINT3**](/windows/win32/api/directxmath/ns-directxmath-xmint3) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                             | Описание                                                                                                      |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**Оператор =**](/windows/win32/api/directxmath/nf-directxmath-xmint3-operator-assign)<br/> | Назначает данные компонента вектора от одного экземпляра `XMINT3` к текущему экземпляру `XMINT3` . <br/> |
