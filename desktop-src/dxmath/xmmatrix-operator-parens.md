# <a name="xmmatrix-operator--operators"></a>Операторы КСММАТРИКС operator ()

Обращается к конкретным элементам матрицы, на которые ссылается строка и столбец из текущего экземпляра `XMMATRIX` .

Обращается к конкретным элементам матрицы, на которые ссылается строка и столбец из текущего экземпляра [ **ксмматрикс**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix)

|----------|-------------|
