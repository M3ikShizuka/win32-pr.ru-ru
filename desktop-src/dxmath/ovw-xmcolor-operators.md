# <a name="xmcolor-operators"></a>Операторы КСМКОЛОР

Структура [**ксмколор**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmcolor) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                            | Описание                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**Оператор UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmcolor-operatoruint32_t)<br/> | Возвращает экземпляр, `uint32_t` содержащий компоненты `XMCOLOR` экземпляра в упакованном формате. <br/> |
| [**Оператор =**](xmcolor-operator-eq.md)<br/>               | Присваивает данные компонента вектора текущему экземпляру `XMCOLOR` .<br/>                                        |
