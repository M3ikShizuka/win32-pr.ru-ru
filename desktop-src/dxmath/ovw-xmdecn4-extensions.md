# <a name="xmdecn4-extensions"></a>Расширения XMDECN4

[**XMDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdecn4) описывает вектор 4d для хранения подписанных нормализованных значений в виде 10-разрядных компонентов x-, y и z, а также 2-разрядного, подписанного w.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
