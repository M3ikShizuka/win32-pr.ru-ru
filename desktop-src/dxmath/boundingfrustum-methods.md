| Метод                                                                  | Описание                                                                                                    |

| [**Пересекает**](boundingfrustum-intersects.md)<br/>             | Проверяет [**баундингфрустум**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) для пересечения с другим объектом.<br/>          |
| [**Преобразование**](boundingfrustum-transform.md)<br/>               | Преобразует [**баундингфрустум**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).<br/>                                          |

| [**креатефромматрикс**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-createfrommatrix)<br/> | Создает объект [**баундингфрустум**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) из указанной матрицы проекции.<br/>          |

| [**\_назначение Op**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-operator-assign(boundingfrustum__))<br/>      | Копирует значения из другого [**баундингфрустум**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).<br/>                              |
