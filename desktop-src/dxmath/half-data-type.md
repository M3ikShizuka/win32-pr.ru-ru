# <a name="half-data-type"></a>ПОЛОВИНА типа данных

Псевдоним для **UInt16 \_ t** с 16-разрядным числом с плавающей запятой, состоящим из бита знака, 5-битного смещения и 10-разрядной мантисса.


```C++
typedef uint16_t HALF;
```



## <a name="remarks"></a>Комментарии

ПОЛОВИНА типа данных эквивалентна формату binary16 в формате IEEE 754.

ПОЛОВИНная \_ минута = 6.10352 e-5F

ПОЛОВИНА \_ Max = 65504. f

Дополнительные сведения о половина типа данных см. в разделе [Формат с плавающей запятой половинной точности](https://en.wikipedia.org/wiki/Half_precision_floating-point_format).

**Пространство имен**: Используйте DirectX::P аккедвектор

### <a name="platform-requirements"></a>Требования к платформе

Microsoft Visual Studio 2010 или Microsoft Visual Studio 2012 с Windows SDK для Windows 8. Поддерживается для классических приложений Win32, приложений для Магазина Windows и Windows Phone 8 приложений.
