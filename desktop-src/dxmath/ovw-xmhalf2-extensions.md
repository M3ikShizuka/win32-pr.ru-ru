# <a name="xmhalf2-extensions"></a>Расширения XMHALF2

[**XMHALF2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmhalf2) описывает двумерный вектор, состоящий из двух значений с плавающей запятой половинной точности (16-разрядный).

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
