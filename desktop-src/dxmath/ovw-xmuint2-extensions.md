# <a name="xmuint2-extensions"></a>Расширения XMUINT2

[**XMUINT2**](/windows/win32/api/directxmath/ns-directxmath-xmuint2) описывает двумерный вектор, состоящий из двух целочисленных значений без знака.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
