# <a name="xmshortn4-extensions"></a>Расширения XMSHORTN4

[**XMSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn4) описывает вектор 4d для хранения подписанных нормализованных значений как 16-разрядных целых чисел со знаком (тип `int16_t` ).

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
