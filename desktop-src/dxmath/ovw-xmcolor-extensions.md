# <a name="xmcolor-extensions"></a>Расширения КСМКОЛОР

[**Ксмколор**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmcolor) описывает цветовой вектор 32-разрядного красного зеленого цвета (ARGB), где каждый цветовой канал задается 8 битами.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
