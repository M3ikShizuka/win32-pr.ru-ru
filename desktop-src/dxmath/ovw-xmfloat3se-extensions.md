# <a name="xmfloat3se-extensions"></a>Расширения XMFLOAT3SE

Трехмерный вектор из трех компонентов с плавающей запятой с 9-битовыми мантиссами, каждый из которых имеет один и тот же показатель «5 бит».

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
