# <a name="directxmath"></a>DirectXMath

## <a name="purpose"></a>Назначение

API Директксмас предоставляет SIMD-удобные типы и функции C++ для обычных линейных и графических математических операций, общих для приложений DirectX. библиотека предоставляет оптимизированные версии для Windows 32-bit (x86), Windows 64-bit (x64) и Windows в ARM/ARM64 с помощью функций SSE, AVX и arm-NEON в компиляторе Visual C++.

Для разработчиков, начинающихся с директксмас, можно использовать оболочку симплемас в *наборе средств DirectX* для [DirectX 11](https://go.microsoft.com/fwlink/?LinkId=248929)  /  [DirectX12](https://go.microsoft.com/fwlink/?LinkID=615561) в качестве отправной точки.

## <a name="developer-audience"></a>Аудитория разработчиков

библиотека директксмас разработана для разработчиков на C++, работающих в играх и графиках DirectX, в приложениях Windows Store, в играх Xbox и традиционных настольных приложениях для Windows.

## <a name="obtaining-directxmath"></a>Получение Директксмас
 
заголовки директксмас поставляются в Windows SDK, поставляемые с Visual Studio 2012 или более поздней версии, и во всех встроенных заголовках нет библиотеки DLL или статической библиотеки для связи. Он также доступен в виде пакета на [NuGet](https://www.nuget.org/packages/directxmath/).

Директксмас — это открытый исходный код с [лицензией MIT](https://opensource.org/licenses/MIT) , размещенной на [GitHub](https://github.com/Microsoft/DirectXMath).
