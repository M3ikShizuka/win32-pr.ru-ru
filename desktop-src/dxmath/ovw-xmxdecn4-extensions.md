# <a name="xmxdecn4-extensions"></a>Расширения XMXDECN4

[**XMXDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdecn4) определяет вектор 4d для хранения подписанных нормализованных значений в виде 10-битных компонентов x-, y и z, а также неподписанных нормализованных значений в виде 2-разрядного неподписанного бесэлементного компонента.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
