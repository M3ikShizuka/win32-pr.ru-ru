# <a name="xmfloat3-constructors"></a>Конструкторы XMFLOAT3

Конструкторы, предоставляемые `XMFLOAT3` .

Конструирует экземпляры структуры [**XMFLOAT3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3) .

> [!Note]  
> Перечисленные здесь конструкторы доступны только в C++.
