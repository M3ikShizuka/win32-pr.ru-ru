# <a name="xmfloat4x3-operators"></a>Операторы XMFLOAT4X3

Структура [**XMFLOAT4X3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4x3) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                      | Описание                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**Оператор =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat4x3-operator-assign)<br/>      | Назначает данные компонента вектора от одного экземпляра `XMFLOAT4X3` к текущему экземпляру `XMFLOAT4X3` . <br/> |
| [**оператор ()**](xmfloat4x3-operator-parens.md)<br/> | Обращается к конкретным элементам матрицы, на которые ссылается строка и столбец из текущего экземпляра `XMFLOAT4X3` . <br/>    |
