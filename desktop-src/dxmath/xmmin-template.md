# <a name="xmmin-template"></a>Шаблон Ксммин

Сравнивает два экземпляра типа числовых данных или два экземпляра объекта, которые поддерживают перегрузку <, и возвращает меньше одного из двух экземпляров. Тип данных аргументов и возвращаемое значение одинаковы.

## <a name="syntax"></a>Синтаксис

``` syntax
template<class T> T XMMin(
  [in]  T a,
  [in]  T b
);
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="a"></span><span id="A"></span>*конкретного*
</dt> <dd>

\[в \] задает первый из двух объектов.

</dd> <dt>

<span id="b"></span><span id="B"></span>*&*
</dt> <dd>

\[в \] задает два из двух объектов.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает меньшее из двух входных объектов.

## <a name="remarks"></a>Комментарии

`XMMin` является шаблоном, а тип T указывается при создании экземпляра шаблона.

> [!Note]  
> `XMMin`Шаблон является новым для директксмас и недоступен для кснамас 2. x. `XMMin` доступен в виде макроса в Кснамас 2. x.

 

> [!Note]  
> В идеале используйте std:: min вместо `XMMin` . Чтобы избежать конфликтов с заголовками Windows с std:: min, необходимо \# определить номинмакс перед включением заголовков Windows.

 

**Пространство имен**. Использование DirectX

Microsoft Visual Studio 2010 или Microsoft Visual Studio 2012 с Windows SDK для Windows 8. Поддерживается для классических приложений Win32, приложений для Магазина Windows и Windows Phone 8 приложений.
