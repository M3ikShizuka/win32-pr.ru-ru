# <a name="xmxdecn4-operators"></a>Операторы XMXDECN4

Структура [**XMXDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdecn4) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                             | Описание                                                                                                            |
|---------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**Оператор UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmxdecn4-operatoruint32_t)<br/> | Возвращает экземпляр, `uint32_t` содержащий компоненты `XMXDECN4` экземпляра в упакованном формате. <br/> |
| [**Оператор =**](xmxdecn4-operator-eq.md)<br/>               | Присваивает данные компонента вектора текущему экземпляру `XMXDECN4` . <br/>                                       |
