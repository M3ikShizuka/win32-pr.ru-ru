# <a name="xmfloat3se-operators"></a>Операторы XMFLOAT3SE

Структура [**XMFLOAT3SE**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmfloat3se) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                               | Описание                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**Оператор UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmfloat3se-operatoruint32_t)<br/> | Возвращает экземпляр, `uint32_t` содержащий компоненты `XMFLOAT3SE` экземпляра в упакованном формате.<br/> |
| [**Оператор =**](xmfloat3se-operator-eq.md)<br/>               | Присваивает данные компонента вектора текущему экземпляру `XMFLOAT3SE` .<br/>                                       |
