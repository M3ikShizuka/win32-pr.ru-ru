# <a name="xmu555-operators"></a>Операторы XMU555

Структура [**XMU555**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu555) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                           | Описание                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**Оператор UInt16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmu555-operatoruint16_t)<br/> | Возвращает экземпляр, `uint16_t` содержащий компоненты `XMU555` экземпляра в упакованном формате. <br/> |
| [**Оператор =**](xmu555-operator-eq.md)<br/>               | Присваивает данные компонента вектора текущему экземпляру `XMU555` . <br/>                                       |
