# <a name="xmmatrix-operators"></a>Операторы КСММАТРИКС

Структура [**ксмматрикс**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                    | Описание                                                                                                                                                                                   |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**станции \***](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult)<br/>    | Выполняет умножение матрицы текущего экземпляра `XMMATRIX` другим экземпляром `XMMATRIX` .<br/>                                                                          |
| [**станции \*=**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult-assign)<br/> | Выполняет умножение матрицы текущего экземпляра `XMMATRIX` другим экземпляром `XMMATRIX` и возвращает ссылку на текущий экземпляр, который был обновлен. <br/> |
| [**Оператор =**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-assign)<br/>      | Назначает данные матрицы одного экземпляра `XMMATRIX` текущему экземпляру `XMMATRIX` и возвращает объект `reference` в текущий экземпляр. <br/>                                    |
| [**оператор ()**](xmmatrix-operator-parens.md)<br/> | Обращается к конкретным элементам матрицы, на которые ссылается строка и столбец из текущего экземпляра `XMMATRIX` . <br/>                                                                           |
