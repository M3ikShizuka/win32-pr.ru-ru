# <a name="xmint4-extensions"></a>Расширения XMINT4

[**XMINT4**](/windows/win32/api/directxmath/ns-directxmath-xmint4) описывает структуру, содержащую вектор 4d, состоящий из четырех целочисленных значений.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
