# <a name="xmushortn4-extensions"></a>Расширения XMUSHORTN4

[**XMUSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn4) описывает вектор 4d для хранения беззнаковых нормализованных значений как 16-разрядных целых чисел со знаком (тип `uint16_t` ).

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
