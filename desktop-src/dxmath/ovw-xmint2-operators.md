# <a name="xmint2-operators"></a>Операторы XMINT2

Структура [**XMINT2**](/windows/win32/api/directxmath/ns-directxmath-xmint2) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                             | Описание                                                                                                     |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**Оператор =**](/windows/win32/api/directxmath/nf-directxmath-xmint2-operator-assign)<br/> | Назначает данные компонента вектора от одного экземпляра `XMINT2` к текущему экземпляру `XMINT2` .<br/> |
