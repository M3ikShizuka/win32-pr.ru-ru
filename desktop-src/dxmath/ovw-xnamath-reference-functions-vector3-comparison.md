# <a name="directxmath-library-3d-vector-comparison-functions"></a>Функции сравнения трехмерных векторных библиотек Директксмас

Содержит список функций сравнения трехмерных векторов.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                   | Описание                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equal)<br/>                     | Проверяет, равны ли два трехмерных вектора.<br/>                                                                                                                                                                                                        |
| [**XMVector3EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equalint)<br/>               | Проверяет равенство двух трехмерных векторов, рассматривая каждый компонент как целое число без знака.<br/>                                                                                                                                                        |
| [**XMVector3EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equalintr)<br/>             | Проверяет равенство двух трехмерных векторов, рассматривая каждый компонент как целое число без знака. Кроме того, эта функция возвращает значение сравнения, которое можно исследовать с помощью таких функций, как [**ксмкомпарисоналлтруе**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVector3EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3equalr)<br/>                   | Проверяет, равны ли два трехмерных вектора. Кроме того, эта функция возвращает значение сравнения, которое можно исследовать с помощью таких функций, как [**ксмкомпарисоналлтруе**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                 |
| [**XMVector3Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greater)<br/>                 | Проверяет, больше ли один трехмерный вектор, чем другой трехмерный вектор.<br/>                                                                                                                                                                                 |
| [**XMVector3GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greaterorequal)<br/>   | Проверяет, является ли один трехмерный вектор большим или равным другому трехмерному вектору.<br/>                                                                                                                                                                     |
| [**XMVector3GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greaterorequalr)<br/> | Проверяет, является ли один трехмерный вектор больше или равен другому трехмерному вектору, и возвращает значение сравнения, которое можно исследовать с помощью таких функций, как [**ксмкомпарисоналлтруе**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                      |
| [**XMVector3GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector3greaterr)<br/>               | Проверяет, больше ли один трехмерный вектор, чем другой трехмерный вектор, и возвращает значение сравнения, которое можно исследовать с помощью таких функций, как [**ксмкомпарисоналлтруе**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                  |
| [**XMVector3IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector3isinfinite)<br/>           | Проверяет, является ли любой компонент трехмерного вектора положительным или отрицательным бесконечностью.<br/>                                                                                                                                                                   |
| [**XMVector3IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector3isnan)<br/>                     | Проверяет, является ли любой компонент трехмерного вектора NaN.<br/>                                                                                                                                                                                           |
| [**XMVector3Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector3less)<br/>                       | Проверяет, меньше ли один трехмерный вектор, чем другой трехмерный вектор.<br/>                                                                                                                                                                                    |
| [**XMVector3LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lessorequal)<br/>         | Проверяет, является ли один трехмерный вектор меньшим или равным другому трехмерному вектору.<br/>                                                                                                                                                                        |
| [**XMVector3NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3nearequal)<br/>             | Проверяет, приближается ли один трехмерный вектор к другому трехмерному вектору.<br/>                                                                                                                                                                                         |
| [**XMVector3NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector3notequal)<br/>               | Проверяет, не равны ли два трехмерных вектора.<br/>                                                                                                                                                                                                    |
| [**XMVector3NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector3notequalint)<br/>         | Проверьте, не равны ли два трехмерных вектора, обрабатывая каждый компонент как целое число без знака.<br/>                                                                                                                                                     |
