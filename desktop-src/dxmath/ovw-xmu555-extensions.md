# <a name="xmu555-extensions"></a>Расширения XMU555

[**XMU555**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu555) описывает вектор 4d с компонентами x, y и z, представленными в виде 5-разрядных целочисленных значений без знака, а компонент w-Component — в виде одноразрядного целочисленного значения.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
