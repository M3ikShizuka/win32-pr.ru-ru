# <a name="component-wise-vector-functions"></a>Функции с векторным компонентом

Содержит список функций-векторов, ориентированных на компонент.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                             | Описание                                                                                                                              |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**ксмвекторинсерт**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinsert)<br/>               | Поворачивает вектор влево на заданное число 32-разрядных компонентов и вставляет выбранные элементы этого результата в другой вектор.<br/> |
| [**ксмвектормержекси**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergexy)<br/>             | Создает новый вектор, объединяя компоненты x и y двух векторов.<br/>                                                      |
| [**ксмвектормержезв**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergezw)<br/>             | Создает новый вектор, комбинируя z и w-компоненты двух векторов.<br/>                                                      |
| [**ксмвекторпермуте**](/windows/win32/api/directxmath/nf-directxmath-xmvectorpermute)<br/>             | Пермутес компоненты двух векторов для создания нового вектора.<br/>                                                                |
| [**ксмвекторротателефт**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateleft)<br/>       | Поворачивает вектор влево на заданное число 32-разрядных элементов.<br/>                                                                 |
| [**ксмвекторротатеригхт**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateright)<br/>     | Поворачивает вектор вправо на заданное число 32-разрядных элементов.<br/>                                                                |
| [**ксмвекторселект**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect)<br/>               | Выполняет выбор каждого компонента между двумя входными векторами и возвращает результирующий вектор.<br/>                                |
| [**ксмвекторселектконтрол**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselectcontrol)<br/> | Определяет вектор управления для использования в [**ксмвекторселект**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect).<br/>                                                 |
| [**ксмвекторшифтлефт**](/windows/win32/api/directxmath/nf-directxmath-xmvectorshiftleft)<br/>         | Сдвигает вектор влево на заданное число 32-разрядных элементов, заполняя освобождаемые элементы элементами из второго вектора.<br/>   |
| [**ксмвекторсплатв**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatw)<br/>               | Реплицирует компонент «w» вектора в все компоненты.<br/>                                                              |
| [**ксмвекторсплаткс**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatx)<br/>               | Реплицирует компонент x вектора во все компоненты.<br/>                                                              |
| [**ксмвекторсплати**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplaty)<br/>               | Реплицирует компонент y вектора во все компоненты.<br/>                                                              |
| [**ксмвекторсплатз**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatz)<br/>               | Реплицирует компонент z вектора на все компоненты.<br/>                                                              |
| [**ксмвекторсвиззле**](/windows/win32/api/directxmath/nf-directxmath-xmvectorswizzle)<br/>             | Свиззлес вектор.<br/>                                                                                                            |
