# <a name="xmu565-operators"></a>Операторы XMU565

Структура [**XMU565**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu565) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                           | Описание                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**Оператор UInt16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmu565-operatoruint16_t)<br/> | Возвращает экземпляр, `uint16_t` содержащий компоненты `XMU555` экземпляра в упакованном формате. <br/> |
| [**Оператор =**](xmu565-operator-eq.md)<br/>               | Присваивает данные компонента вектора текущему экземпляру `XMU565` . <br/>                                       |
