# <a name="xmhalf4-extensions"></a>Расширения XMHALF4

[**XMHALF4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmhalf4) описывает вектор 4d, состоящий из четырех значений с плавающей запятой половинной точности (16-разрядный).

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
