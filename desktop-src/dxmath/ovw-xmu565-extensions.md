# <a name="xmu565-extensions"></a>Расширения XMU565

[**XMU565**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu565) описывает вектор 4d с 5-битовыми неподписанными компонентами x и z и 6-битным неподписанным компонентом y.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
