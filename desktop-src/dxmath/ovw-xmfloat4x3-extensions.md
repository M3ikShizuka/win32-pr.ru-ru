# <a name="xmfloat4x3-extensions"></a>Расширения XMFLOAT4X3

[**XMFLOAT4X3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4x3) описывает \* матрицу с плавающей точкой размером 4 3.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
