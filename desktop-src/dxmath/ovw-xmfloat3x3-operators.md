# <a name="xmfloat3x3-operators"></a>Операторы XMFLOAT3X3

Структура [**XMFLOAT3X3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3x3) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                      | Описание                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**Оператор =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat3x3-operator-assign)<br/>      | Назначает данные компонента вектора от одного экземпляра `XMFLOAT3X3` к текущему экземпляру `XMFLOAT3X3` . <br/> |
| [**оператор ()**](xmfloat3x3-operator-parens.md)<br/> | Обращается к конкретным элементам матрицы, на которые ссылается строка и столбец из текущего экземпляра `XMFLOAT3X3` . <br/>    |
