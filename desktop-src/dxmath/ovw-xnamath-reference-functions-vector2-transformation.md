# <a name="directxmath-library-2d-vector-transformation-functions"></a>Функции преобразования 2D-вектора библиотеки Директксмас

Содержит список функций преобразования 2D-вектора.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                               | Описание                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transform)<br/>                         | Преобразует двумерный вектор по матрице.<br/>                                                                                                   |
| [**XMVector2TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformcoord)<br/>               | Преобразует двумерный вектор по заданной матрице, проецирование результата обратно в w = 1.<br/>                                                      |
| [**XMVector2TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformcoordstream)<br/>   | Преобразует поток двумерных векторов в заданную матрицу, формируя результирующие векторы таким, что их w-координаты равны 1,0.<br/> |
| [**XMVector2TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformnormal)<br/>             | Преобразует двумерный вектор, нормализованный заданной матрицей.<br/>                                                                                  |
| [**XMVector2TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformnormalstream)<br/> | Преобразует поток двумерных векторов нормали в заданную матрицу.<br/>                                                                           |
| [**XMVector2TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector2transformstream)<br/>             | Преобразует поток двумерных векторов в заданную матрицу.<br/>                                                                                  |
