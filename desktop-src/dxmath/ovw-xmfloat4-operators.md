# <a name="xmfloat4-operators"></a>Операторы XMFLOAT4

Структура [**XMFLOAT4**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                               | Описание                                                                                                         |
|-------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**Оператор =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat4-operator-assign(xmfloat4__))<br/> | Назначает данные компонента вектора от одного экземпляра `XMFLOAT4` к текущему экземпляру `XMFLOAT4` .<br/> |
