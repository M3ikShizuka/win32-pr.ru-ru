# <a name="xmint3-extensions"></a>Расширения XMINT3

[**XMINT3**](/windows/win32/api/directxmath/ns-directxmath-xmint3) описывает структуру, содержащую трехмерный вектор, состоящий из трех целочисленных значений.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
