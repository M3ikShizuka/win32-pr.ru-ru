# <a name="xmudec4-operators"></a>Операторы XMUDEC4

Структура [**XMUDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudec4) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                            | Описание                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**Оператор UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmudec4-operatoruint32_t)<br/> | Возвращает экземпляр, `uint32_t` содержащий компоненты `XMUDEC4` экземпляра в упакованном формате. <br/> |
| [**Оператор =**](xmudec4-operator-eq.md)<br/>               | Присваивает данные компонента вектора текущему экземпляру `XMUDEC4` . <br/>                                       |
