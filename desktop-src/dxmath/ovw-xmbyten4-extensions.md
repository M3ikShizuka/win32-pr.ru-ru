# <a name="xmbyten4-extensions"></a>Расширения XMBYTEN4

[**XMBYTEN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmbyten4) описывает трехмерный вектор для хранения подписанных нормализованных значений в виде 8-разрядных целых чисел со знаком (1 байт).

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
