# <a name="xmfloat3pk-extensions"></a>Расширения XMFLOAT3PK

Трехмерный вектор с компонентами X и Y, сохраненный в виде 11-разрядного числа с плавающей запятой, и компонент Z, хранящийся в виде 10-разрядного значения с плавающей запятой.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
