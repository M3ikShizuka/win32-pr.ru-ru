# <a name="directxmath-library-2d-vector-comparison-functions"></a>Функции сравнения 2D-векторной библиотеки Директксмас

Содержит список функций сравнения 2D-вектора.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                   | Описание                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector2Equal**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equal)<br/>                     | Проверяет, равны ли два 2D-вектора.<br/>                                                                                                                                                                                                        |
| [**XMVector2EqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalint)<br/>               | Проверяет равенство двух двумерных векторов, рассматривая каждый компонент как целое число без знака.<br/>                                                                                                                                                        |
| [**XMVector2EqualIntR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalintr)<br/>             | Проверяет равенство двух двумерных векторов, рассматривая каждый компонент как целое число без знака. Кроме того, эта функция возвращает значение сравнения, которое можно исследовать с помощью таких функций, как [**ксмкомпарисоналлтруе**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/> |
| [**XMVector2EqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2equalr)<br/>                   | Проверяет, равны ли два 2D-вектора. Кроме того, эта функция возвращает значение сравнения, которое можно исследовать с помощью таких функций, как [**ксмкомпарисоналлтруе**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                 |
| [**XMVector2Greater**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greater)<br/>                 | Проверяет, больше ли один 2D-вектор, чем другой 2D-вектор.<br/>                                                                                                                                                                                 |
| [**XMVector2GreaterOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterorequal)<br/>   | Проверяет, что один 2D-вектор больше или равен другому 2D-вектору.<br/>                                                                                                                                                                     |
| [**XMVector2GreaterOrEqualR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterorequalr)<br/> | Проверяет, является ли один 2D-вектор больше или равен другому 2D-вектору, и возвращает значение сравнения, которое можно исследовать с помощью таких функций, как [**ксмкомпарисоналлтруе**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                      |
| [**XMVector2GreaterR**](/windows/win32/api/directxmath/nf-directxmath-xmvector2greaterr)<br/>               | Проверяет, больше ли один 2D-вектор, чем другой 2D-вектор, и возвращает значение сравнения, которое можно исследовать с помощью таких функций, как [**ксмкомпарисоналлтруе**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue).<br/>                                                  |
| [**XMVector2IsInfinite**](/windows/win32/api/directxmath/nf-directxmath-xmvector2isinfinite)<br/>           | Проверяет, является ли любой компонент 2D-вектора положительным или отрицательным бесконечностью.<br/>                                                                                                                                                                   |
| [**XMVector2IsNaN**](/windows/win32/api/directxmath/nf-directxmath-xmvector2isnan)<br/>                     | Проверяет, является ли любой компонент 2D-вектором NaN.<br/>                                                                                                                                                                                           |
| [**XMVector2Less**](/windows/win32/api/directxmath/nf-directxmath-xmvector2less)<br/>                       | Проверяет, меньше ли один 2D-вектора, чем другой 2D-вектор.<br/>                                                                                                                                                                                    |
| [**XMVector2LessOrEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2lessorequal)<br/>         | Проверяет, является ли один 2D-вектор меньшим или равным другим 2D-вектором.<br/>                                                                                                                                                                        |
| [**XMVector2NearEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2nearequal)<br/>             | Проверяет, приближается ли один двумерный вектор к другому 2D-вектору.<br/>                                                                                                                                                                                         |
| [**XMVector2NotEqual**](/windows/win32/api/directxmath/nf-directxmath-xmvector2notequal)<br/>               | Проверяет, не равны ли два 2D-вектора.<br/>                                                                                                                                                                                                    |
| [**XMVector2NotEqualInt**](/windows/win32/api/directxmath/nf-directxmath-xmvector2notequalint)<br/>         | Проверьте, не равны ли два вектора, обрабатывая каждый компонент как целое число без знака.<br/>                                                                                                                                                        |
