# <a name="directxmath-library-conversion-functions"></a>Функции преобразования библиотеки Директксмас

Перечисляет функции преобразования, предоставляемые Директксмас.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                       | Описание                                                                                                                                                          |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ксмконвертфлоаттохалф**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconvertfloattohalf)<br/>             | Преобразует значение с плавающей запятой одиночной точности в значение с плавающей запятой половинной точности.<br/>                                                                |
| [**ксмконвертфлоаттохалфстреам**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconvertfloattohalfstream)<br/> | Преобразует поток значений с плавающей запятой одиночной точности в поток значений с плавающей запятой половинной точности.<br/>                                          |
| [**ксмконверсалфтофлоат**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconverthalftofloat)<br/>             | Преобразует значение с плавающей запятой половинной точности в значение с плавающей запятой одиночной точности.<br/>                                                                |
| [**ксмконверсалфтофлоатстреам**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmconverthalftofloatstream)<br/> | Преобразует поток значений с плавающей запятой половинной точности в поток значений с плавающей запятой одиночной точности.<br/>                                          |
| [**ксмконверттодегрис**](/windows/desktop/api/DirectXMath/nf-directxmath-xmconverttodegrees)<br/>                 | Преобразует угол, измеряемый в радианах, в единицу измерения в градусах.<br/>                                                                                       |
| [**ксмконвертторадианс**](/windows/desktop/api/DirectXMath/nf-directxmath-xmconverttoradians)<br/>                 | Преобразует угол, измеряемый в градусах, в единицу измерения в радианах.<br/>                                                                                       |
| [**ксмконвертвекторфлоаттоинт**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorfloattoint)<br/>   | Преобразует [**ксмвектор**](xmvector-data-type.md) с компонентами с **плавающей запятой** в **ксмвектор** с компонентами **Int32 \_ t** и применяет однородное смещение.<br/>  |
| [**ксмконвертвекторфлоаттауинт**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorfloattouint)<br/> | Преобразует [**ксмвектор**](xmvector-data-type.md) с компонентами с **плавающей запятой** в **ксмвектор** с компонентами **UInt32 \_ t** и применяет однородное смещение.<br/> |
| [**ксмконвертвекторинттофлоат**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectorinttofloat)<br/>   | Преобразует [**ксмвектор**](xmvector-data-type.md) с компонентами **Int32 \_ t** в **ксмвектор** с компонентами **float** и применяет однородное смещение.<br/>  |
| [**ксмконвертвекторуинттофлоат**](/windows/win32/api/directxmath/nf-directxmath-xmconvertvectoruinttofloat)<br/> | Преобразует [**ксмвектор**](xmvector-data-type.md) с компонентами **UInt32 \_ t** в **ксмвектор** с компонентами **float** и применяет однородное смещение.<br/> |
