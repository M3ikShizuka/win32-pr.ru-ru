# <a name="xmushortn2-extensions"></a>Расширения XMUSHORTN2

[**XMUSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn2) описывает двумерный вектор для хранения беззнаковых нормализованных значений как 16-разрядных целых чисел без знака (тип `int16_t` ).

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
