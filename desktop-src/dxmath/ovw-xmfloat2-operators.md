# <a name="xmfloat2-operators"></a>Операторы XMFLOAT2

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела

|-------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**Оператор =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat2-operator-assign)<br/> | Назначает данные компонента вектора от одного экземпляра `XMFLOAT2` к текущему экземпляру `XMFLOAT2` .<br/> |
