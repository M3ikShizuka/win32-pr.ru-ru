# <a name="xmubyte4-extensions"></a>Расширения XMUBYTE4

Описывает вектор 4D, состоящие из целых чисел без знака, 8 бит (1 байт) в длине.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
