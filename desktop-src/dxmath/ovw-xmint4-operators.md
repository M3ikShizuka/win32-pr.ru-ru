# <a name="xmint4-operators"></a>Операторы XMINT4

Структура [**XMINT4**](/windows/win32/api/directxmath/ns-directxmath-xmint4) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                             | Описание                                                                                                     |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**Оператор =**](/windows/win32/api/directxmath/nf-directxmath-xmint4-operator-assign)<br/> | Назначает данные компонента вектора от одного экземпляра `XMINT4` к текущему экземпляру `XMINT4` .<br/> |
