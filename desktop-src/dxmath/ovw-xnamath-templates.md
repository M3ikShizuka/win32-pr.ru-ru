# <a name="directxmath-library-template-functions"></a>Функции шаблона библиотеки Директксмас

Содержит список функций шаблонов, предоставляемых Директксмас.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                  | Описание                                                                                                                                                                                                                               |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ксммакс**](xmmax-template.md)<br/>                             | Сравнивает два экземпляра типа числовых данных или два экземпляра объекта, которые поддерживают перегрузку <, и возвращает больше одного из двух экземпляров. Тип данных аргументов и возвращаемое значение одинаковы.<br/>  |
| [**ксммин**](xmmin-template.md)<br/>                             | Сравнивает два экземпляра типа числовых данных или два экземпляра объекта, которые поддерживают перегрузку <, и возвращает меньше одного из двух экземпляров. Тип данных аргументов и возвращаемое значение одинаковы.<br/> |
| [**ксмвекторпермуте**](xmvectorpermute-template.md)<br/>         | Пермутес компоненты двух векторов для создания нового вектора.<br/>                                                                                                                                                                 |
| [**ксмвекторротателефт**](xmvectorrotateleft-template.md)<br/>   | Поворачивает вектор влево на заданное число 32-разрядных элементов.<br/>                                                                                                                                                                  |
| [**ксмвекторротатеригхт**](xmvectorrotateright-template.md)<br/> | Поворачивает вектор вправо на заданное число 32-разрядных элементов.<br/>                                                                                                                                                                 |
| [**ксмвекторшифтлефт**](xmvectorshiftleft-template.md)<br/>     | Сдвигает вектор влево на заданное число 32-разрядных элементов, заполняя освобождаемые элементы элементами из второго вектора.<br/>                                                                                                    |
| [**ксмвекторсвиззле**](xmvectorswizzle-template.md)<br/>         | Свиззлес вектор.<br/>                                                                                                                                                                                                             |
| [**ксмвекторинсерт**](xmvectorinsert-template.md)<br/>           | Поворачивает вектор влево на заданное число 32-разрядных компонентов и вставляет выбранные элементы этого результата в другой вектор.<br/>                                                                                                  |
