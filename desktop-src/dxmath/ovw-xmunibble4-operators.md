# <a name="xmunibble4-operators"></a>Операторы XMUNIBBLE4

Структура [**XMUNIBBLE4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmunibble4) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                               | Описание                                                                                                              |
|-----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**Оператор UInt16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmunibble4-operatoruint16_t)<br/> | Возвращает экземпляр, `uint16_t` содержащий компоненты `XMUNIBBLE4` экземпляра в упакованном формате. <br/> |
| [**Оператор =**](xmunibble4-operator-eq.md)<br/>               | Присваивает данные компонента вектора текущему экземпляру `XMUNIBBLE4` . <br/>                                       |
