# <a name="xmbyte2-extensions"></a>Расширения XMBYTE2

Описывает двумерный вектор, состоящие из целых чисел со знаком, 8 бит (1 байт) в длине.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
