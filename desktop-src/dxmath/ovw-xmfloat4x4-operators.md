# <a name="xmfloat4x4-operators"></a>Операторы XMFLOAT4X4

Структура [**XMFLOAT4X4**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4x4) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                      | Описание                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**Оператор =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat4x4-operator-assign)<br/>      | Назначает данные компонента вектора от одного экземпляра `XMFLOAT4X4` к текущему экземпляру `XMFLOAT4X4` . <br/> |
| [**оператор ()**](xmfloat4x4-operator-parens.md)<br/> | Обращается к конкретным элементам матрицы, на которые ссылается строка и столбец из текущего экземпляра `XMFLOAT4X4` . <br/>    |
