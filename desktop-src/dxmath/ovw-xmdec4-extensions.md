# <a name="xmdec4-extensions"></a>Расширения XMDEC4

[**XMDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdec4)описывает вектор 4d с компонентами x, y и z, представленными в виде 10-разрядных целочисленных значений со знаком, а компонент w-Component — как целое число со знаком в 2 бита.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
