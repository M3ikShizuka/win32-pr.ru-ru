# <a name="xmushort2-extensions"></a>Расширения XMUSHORT2

[**XMUSHORT2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushort2) описывает вектор a2D для хранения беззнаковых нормализованных значений в виде 16-разрядных целых чисел без знака (тип `uint16_t` ).

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
