# <a name="operator--operators"></a>operator + операторы

Операторы сложения.

### <a name="overload-list"></a>Список перегрузок




| Оператор | Описание | 
|----------|-------------|
| <a href="/previous-versions/windows/desktop/legacy/ee421395(v=vs.85)"><strong>КСМВЕКТОР:: operator + (КСМВЕКТОР)</strong></a> | Производительность операции идентификации в <code>XMVECTOR</code> экземпляре.<br /> Метод <code>operator +</code> принимает экземпляр <a href="xmvector-data-type.md"><strong>типа данных ксмвектор</strong></a> и возвращает новый экземпляр <code>XMVECTOR</code> с оператором Identity, применяемым к каждому компоненту.<br /><blockquote>[!Note]<br />Этот оператор доступен только в C++.</blockquote><br /> | 
| <a href="/previous-versions/windows/desktop/legacy/ee421396(v=vs.85)"><strong>КСМВЕКТОР:: operator + (КСМВЕКТОР, КСМВЕКТОР)</strong></a> | Добавляет два экземпляра <code>XMVECTOR</code> , возвращая результат в новый экземпляр.<br /> Метод <code>operator +</code> добавляет каждый компонент двух экземпляров <a href="xmvector-data-type.md"><strong>типа данных ксмвектор</strong></a>, а затем возвращает результат в новом <code>XMVECTOR</code> экземпляре. <br /><blockquote>[!Note]<br />Этот оператор доступен только в C++.</blockquote><br /> |
