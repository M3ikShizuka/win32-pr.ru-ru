# <a name="xmudecn4-extensions"></a>Расширения XMUDECN4

[**XMUDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudecn4) Вектор 4D для хранения беззнаковых нормализованных значений в виде 10-разрядных неподписанных компонентов x-, y и z, а также 2-разрядного неподписанного w – компонента.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
