# <a name="xmmatrix-extensions"></a>Расширения КСММАТРИКС

[**Ксмматрикс**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) описывает \* матрицу 4 4, выравниваемая по 16-байтовой границе, которая сопоставляется с четырьмя аппаратными регистрами вектора.

> [!Note]  
> Перечисленные здесь функции доступны только в C++.
