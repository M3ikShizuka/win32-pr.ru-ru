# <a name="xmudecn4-operators"></a>Операторы XMUDECN4

Структура [**XMUDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudecn4) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                             | Описание                                                                                                            |
|---------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**Оператор UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmudecn4-operatoruint32_t)<br/> | Возвращает экземпляр, `uint32_t` содержащий компоненты `XMUDECN4` экземпляра в упакованном формате. <br/> |
| [**Оператор =**](xmudecn4-operator-eq.md)<br/>               | Присваивает данные компонента вектора текущему экземпляру `XMUDECN4` . <br/>                                       |
