# <a name="xmfloat4-extensions"></a>Расширения XMFLOAT4

[**XMFLOAT4**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4) описывает структуру, содержащую вектор 4d, состоящий из четырех значений с плавающей запятой одиночной точности.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
