# <a name="xmdec4-operators"></a>Операторы XMDEC4

Структура [**XMDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdec4) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                           | Описание                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**Оператор UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmdec4-operatoruint32_t)<br/> | Возвращает экземпляр, `uint32_t` содержащий компоненты `XMDEC4` экземпляра в упакованном формате. <br/> |
| [**Оператор =**](xmdec4-operator-eq.md)<br/>               | Присваивает данные компонента вектора текущему экземпляру `XMDEC4` . <br/>                                       |
