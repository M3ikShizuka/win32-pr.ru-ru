# <a name="xmdecn4-operators"></a>Операторы XMDECN4

Структура [**XMDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmdecn4) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                            | Описание                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**Оператор UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmdecn4-operatoruint32_t)<br/> | Возвращает экземпляр, `uint32_t` содержащий компоненты `XMDECN4` экземпляра в упакованном формате. <br/> |
| [**Оператор =**](xmdecn4-operator-eq.md)<br/>               | Присваивает данные компонента вектора текущему экземпляру `XMDECN4` . <br/>                                       |
