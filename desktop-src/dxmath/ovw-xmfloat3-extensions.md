# <a name="xmfloat3-extensions"></a>Расширения XMFLOAT3

[**XMFLOAT3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3) описывает структуру, содержащую трехмерный вектор, состоящий из трех значений с плавающей запятой одиночной точности.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
