# <a name="directxmath-library-utility-functions"></a>Служебные программы библиотеки Директксмас

Содержит список служебных функций, предоставляемых Директксмас.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                       | Описание                                                                                                                |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [**ксмкомпарисоналлфалсе**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonallfalse)<br/>             | Проверяет значение сравнения, чтобы определить, имеет ли все сравниваемые компоненты значение false.<br/>                            |
| [**ксмкомпарисоналлинбаундс**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonallinbounds)<br/>       | Проверяет значение сравнения, чтобы определить, находятся ли все сравниваемые компоненты в пределах установленных границ.<br/>                |
| [**ксмкомпарисоналлтруе**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonalltrue)<br/>               | Проверяет значение сравнения, чтобы определить, все ли сравниваемые компоненты имеют значение true.<br/>                             |
| [**ксмкомпарисонанифалсе**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanyfalse)<br/>             | Проверяет значение сравнения, чтобы определить, имеет ли какой либо сравниваемый компонент значение false.<br/>                            |
| [**ксмкомпарисонанйоутофбаундс**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanyoutofbounds)<br/> | Проверяет значение сравнения, чтобы определить, находятся ли какие либо сравниваемые компоненты за пределами набора границ.<br/>           |
| [**ксмкомпарисонанитруе**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonanytrue)<br/>               | Проверяет значение сравнения, чтобы определить, имеет ли один из сравниваемых компонентов значение true.<br/>                             |
| [**ксмкомпарисонмиксед**](/windows/desktop/api/DirectXMath/nf-directxmath-xmcomparisonmixed)<br/>                   | Проверяет значение сравнения, чтобы определить, были ли сравниваемые компоненты смешанные результаты — некоторые из них имеют значение true и некоторые значения false.<br/> |
| [**ксмфреснелтерм**](/windows/win32/api/directxmath/nf-directxmath-xmfresnelterm)<br/>                           | Вычисляет термин Френелю для неполярного источника.<br/>                                                              |
| [**ксмверификпусуппорт**](/windows/win32/api/directxmath/nf-directxmath-xmverifycpusupport)<br/>                 | Указывает, поддерживает ли библиотека Директксмас текущую платформу.<br/>                                             |
