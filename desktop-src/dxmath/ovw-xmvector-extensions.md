# <a name="xmvector-extensions"></a>Расширения КСМВЕКТОР

[**Ксмвектор**](xmvector-data-type.md) — это переносимый тип, используемый для представления вектора 4 32-разрядных компонентов с плавающей запятой или целых чисел, каждый из которых оптимально соответствует и сопоставляется с регистром аппаратного вектора.

> [!Note]  
> Перечисленные здесь функции доступны только в C++.
