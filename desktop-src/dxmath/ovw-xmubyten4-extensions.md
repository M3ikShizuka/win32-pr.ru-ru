# <a name="xmubyten4-extensions"></a>Расширения XMUBYTEN4

[**XMUBYTEN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmubyten4) описывает трехмерный вектор для хранения беззнаковых нормализованных значений как целых чисел без знака 8 бит (1 байт).

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
