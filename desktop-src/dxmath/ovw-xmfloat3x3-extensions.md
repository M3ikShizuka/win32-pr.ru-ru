# <a name="xmfloat3x3-extensions"></a>Расширения XMFLOAT3X3

[**XMFLOAT3X3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3x3) описывает 3 \* 3 матрицы с плавающей точкой.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
