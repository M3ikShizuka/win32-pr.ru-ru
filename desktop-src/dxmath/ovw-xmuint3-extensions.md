# <a name="xmuint3-extensions"></a>Расширения XMUINT3

[**XMUINT3**](/windows/win32/api/directxmath/ns-directxmath-xmuint3) описывает структуру, содержащую трехмерный вектор, состоящий из трех целочисленных значений без знака.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
