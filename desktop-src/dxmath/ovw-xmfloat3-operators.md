# <a name="xmfloat3-operators"></a>Операторы XMFLOAT3

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела

|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**Оператор =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat3-operator-assign)<br/> | Назначает данные компонента вектора от одного экземпляра `XMFLOAT3` к текущему экземпляру `XMFLOAT3` . <br/> |
