# <a name="xmunibble4-extensions"></a>Расширения XMUNIBBLE4

[**XMUNIBBLE4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmunibble4) описывает вектор 4d с четырьмя неподписанными 4-разрядными целыми компонентами целых чисел.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
