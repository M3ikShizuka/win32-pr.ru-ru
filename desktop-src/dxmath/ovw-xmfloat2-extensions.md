# <a name="xmfloat2-extensions"></a>Расширения XMFLOAT2

[**XMFLOAT2**](/windows/win32/api/directxmath/ns-directxmath-xmfloat2) описывает двумерный вектор, состоящий из двух значений с плавающей запятой одиночной точности.

> [!Note]  
> Перечисленные здесь функции доступны только при разработке с использованием C++.
