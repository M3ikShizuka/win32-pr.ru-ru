# <a name="xmxdec4-operators"></a>Операторы XMXDEC4

Структура [**XMXDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdec4) предоставляет следующие операторы.

> [!Note]  
> Указанные здесь операторы доступны только в C++.

 

## <a name="in-this-section"></a>Содержание раздела



| Методы                                                            | Описание                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**Оператор UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmxdec4-operatoruint32_t)<br/> | Возвращает экземпляр, `uint32_t` содержащий компоненты `XMXDEC4` экземпляра в упакованном формате. <br/> |
| [**Оператор =**](xmxdec4-operator-eq.md)<br/>               | Присваивает данные компонента вектора текущему экземпляру `XMXDEC4` . <br/>                                       |
