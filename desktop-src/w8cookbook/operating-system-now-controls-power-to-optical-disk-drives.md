# <a name="operating-system-now-controls-power-to-optical-disk-drives"></a>Теперь операционная система управляет питанием оптических дисков.

## <a name="platforms"></a>Платформы

**Клиенты** — Windows 8  
**Серверы** — Windows Server 2012  


## <a name="description"></a>Описание

В предыдущих версиях Windows питание оптического дисковода не было управляемым, если оптический дисковод не использовался. Теперь, если на оптическом диске нет носителя (НЕЧЕТного), операционная система выключает питание на оптический диск. Эта функция называется 0 Power НЕЧЕТная (ЗПОДД). Эта функция применима только к дисководам оптических дисков, использующим соединитель SATA форм-фактора slimline (с последовательным подключением).

## <a name="manifestation"></a>Проявление

В этом новом поведении не обнаружено негативных последствий. Однако следует иметь в виду, что это может привести к непредвиденному поведению мультимедийного программного обеспечения.

## <a name="mitigation"></a>Меры по снижению риска

Чтобы вернуться к состоянию Always on, отключите эту функцию в реестре. Абсолютный путь к значению реестра:

`HKLM\SYSTEM\CurrentControlSet\Services\cdrom\Parameters\ZeroPowerODDEnabled`

Его тип — DWORD (32 бит), и если его значение равно 0, то ЗПОДД отключена; Если это любое другое значение, то ЗПОДД включен.

## <a name="tests"></a>Тесты

Протестируйте программное обеспечение на основе мультимедиа для всех аномалий, возникающих из-за этой новой функции. [Сообщите корпорации Майкрософт](mailto:OptIssue@microsoft.com) о любых проблемах, обнаруженных с помощью этой новой функции.
