# <a name="windows-8-and-windows-server-2012-client-and-server-compatibility"></a>совместимость Windows 8 и Windows Server 2012 клиента и сервера

В этом разделе описаны изменения в операционной системе, о которых следует уделить особое внимание из-за возможных влияния на существующие приложения и о том, как новые приложения должны разрабатываться на клиентах, на серверах или на клиентах и серверах. Ниже приведен список разделов, рассмотренных в этом разделе, сгруппированных по функциональным областям:

**AppCompat**

-   Обновление правил обнаружения приложений безопасности
-   Определение состояния оболочки совместимости
-   Серверные приложения должны быть способны работать без графической оболочки сервера.
-   Компоненты сервера RDS удаляются из Windows 8
-   Модель сопоставления типов файлов и протоколов
-   Модератор активности на рабочем столе
-   платформа .NET Framework 4,5 по умолчанию и платформа .NET Framework 3,5 являются необязательными
-   Классические приложения могут быть невидимыми после запуска веб-браузера по умолчанию
-   Режим высокой контрастности
-   Манифест приложения (исполняемый)
-   Текущая модель в очереди устарела
-   Сценарии помощника по совместимости программ для Windows 8
-   Мини-приложения рабочего стола удалены

**служба хранилища и файловая система**

-   Обновление совместимости диска в расширенном формате (4 КБ)
-   Тонкая подготовка логических устройств
-   расширенное хранилище теперь необязательно для среда предустановки Windows (WinPE) и SKU сервера.
-   служба виртуальных дисков переходит на API-интерфейс Windows служба хранилища управления на основе инструментарий управления Windows (WMI) (WMI)
-   Пользовательский интерфейс предыдущих версий удален для локальных томов
-   СторахЦи заменяет МСАХЦИ
-   нерекомендуемое резервное копирование и восстановление Windows 7
-   Разгрузка передачи данных

**Другое**

-   диспетчер окон рабочего стола всегда включено
-   Direct2D не поддерживает отрисовку в "форматированные" метафайлы в Internet Explorer 9
-   Изменения в поддержке устаревшего оборудования DX9
-   MSAA недоступен для приложений магазина Windows
-   Порт 3 устарел для драйверов NDIS 6,30
