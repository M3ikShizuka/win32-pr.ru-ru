# <a name="new-features-and-enhancements"></a>Новые функции и усовершенствования

В этом разделе документа описываются улучшения и новые возможности, которые могут улучшить работу пользователей и разработчиков. Однако некоторые из них влияют на то, что разработчики должны знать о существующих и новых продуктах. Ниже приведен список новых функций и улучшений, сгруппированных по функциональным областям:

## <a name="security"></a>Безопасность

-   Ранний запуск антивредоносного ПО
-   Требования к подписывания функций безопасной загрузки для драйверов режима ядра
-   Измеряемая загрузка

## <a name="performance"></a>Производительность

-   Автозагружаемые приложения
-   Отказоустойчивые файловые системы
-   Поддержка API файлового сервера
-   Автоматическое обслуживание

## <a name="usability"></a>Удобство использования

-   Редакторы на базе сторонних методов ввода

## <a name="storage-and-file-system"></a>Хранилище и файловая система

-   Отслеживание измененных диапазонов файла в Windows 8.1
-   Новый API позволяет приложениям отсылать подсказки "обрезать и отменить сопоставление" на носитель хранилища
-   Multipath I/O теперь поддерживает расширенные блоки запросов хранилища
-   Отказоустойчивые файловые системы
-   Поддержка API файлового сервера
-   Новая функция истории файлов
-   Теперь операционная система управляет питанием оптических дисков.

## <a name="usb"></a>USB

-   Поддержка USB 3,0
