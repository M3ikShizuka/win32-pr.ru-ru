# <a name="file-server-api-support"></a>Поддержка API файлового сервера

## <a name="platform"></a>Платформа

**Серверы** — Windows Server 2012 

## <a name="description"></a>Описание

Документация, в которой содержатся сведения о том, какие API поддерживаются 2,2 для протокола SMB 2,2, а также файловых систем отказоустойчивых или общих томов кластера (ReFS/CSVFS), будут доступны для разработчиков в указанном ниже ресурсе. Документация разбита на следующие категории:

-   Функции управления файлами
-   Функции управления каталогом
-   Функции управления томами
-   Функции безопасности
-   Коды поддержки файлов и каталогов
-   Коды управления томами
-   Файлы распределения памяти.

## <a name="usage"></a>Использование

Некоторые API не поддерживаются для всех протоколов или во всех сценариях, в то время как другие API-интерфейсы являются устаревшими в Windows Server 2012. Ознакомьтесь со списком интерфейсов API, которые поддерживаются, чтобы обеспечить непрерывную поддержку приложений.
