# <a name="support-for-usb-30"></a>Поддержка USB 3,0

## <a name="platforms"></a>Платформы

**Клиенты** — Windows 8  
**Серверы** — Windows Server 2012  


## <a name="description"></a>Описание

В Windows 8 и Windows Server 2012 Добавлена поддержка USB 3,0. Мы сделали это, добавив новый стек программного обеспечения для питания контроллера узла USB 3,0, который называется расширяемым хост-контроллером (КСХК). Мы поддерживаем четность интерфейса, уровень IRQL, контекст вызывающего объекта, состояние ошибки, периодичность повтора и т. д. При взаимодействии с устройством, подключенным через USB 2,0 по сравнению с USB 3,0, не должно быть разницы.

Однако при написании драйвера для нового USB-устройства 3,0, определенно, следует использовать новые возможности USB 3,0.

## <a name="manifestation"></a>Проявление

Передача устройств выполняется быстрее, и снижается энергопотребление с компьютера на USB-устройствах в целом. Существует риск того, что существующие USB-устройства могут работать неправильно при подключении к порту USB 3,0. Это не должно происходить и не ожидалось, но, поскольку программное обеспечение, обеспечивающее USB 3,0, является новым, это является риском.
