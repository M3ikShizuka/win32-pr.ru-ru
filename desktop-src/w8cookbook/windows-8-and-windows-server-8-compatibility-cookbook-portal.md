# <a name="windows-and-windows-server-compatibility-cookbook-windows-10-windows-8-windows-81-windows-server-2012-and-windows-server-2012-r2"></a>совместимость Windows и Windows Server cookbook: Windows 10, Windows 8, Windows 8.1, Windows Server 2012 и Windows Server 2012 R2

## <a name="purpose"></a>Назначение

в *Cookbook* мы предоставляем сведения об изменениях и новых возможностях Windows 10, Windows 8, Windows 8.1, Windows server 2012 и Windows серверных операционных систем 2012 R2. Мы также предоставляем рекомендации по проверке совместимости существующих и запланированных программ с новыми операционными системами. Предполагается, что вы знакомы с предыдущими версиями Windows.

Содержимое применяется к:

-   Windows 10
-   Windows 8.1
-   Windows Server 2012 R2
-   Windows 8
-   Windows Server 2012
-   Windows 7
-   Windows Server 2008 R2
