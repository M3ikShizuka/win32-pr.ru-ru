# <a name="windows-and-windows-server-compatibility-cookbook-windows-10-windows-8-windows-81-windows-server-2012-and-windows-server-2012-r2"></a>Windows и Windows Server Compatibility Cookbook: Windows 10, Windows 8, Windows 8.1, Windows Server 2012 и Windows Server 2012 R2

## <a name="purpose"></a>Назначение

В *Cookbook* мы предоставляем сведения об изменениях и новых возможностях операционных систем Windows 10, Windows 8, Windows 8.1, windows Server 2012 и windows Server 2012 R2. Мы также предоставляем рекомендации по проверке совместимости существующих и запланированных программ с новыми операционными системами. Предполагается, что вы знакомы с предыдущими версиями Windows.

Содержимое применяется к:

-   Windows 10
-   Windows 8.1
-   Windows Server 2012 R2
-   Windows 8
-   Windows Server 2012
-   Windows 7
-   Windows Server 2008 R2
