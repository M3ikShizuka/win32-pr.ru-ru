# <a name="windows-store-apps"></a>Приложения для Магазина Windows

## <a name="resources"></a>Ресурсы

Перечисленные ниже ресурсы призваны помочь при переносе приложений, встроенных в Windows 8, в Windows 8.1.

-   [обновление приложения с Windows 8 до Windows 8.1](/previous-versions/windows/dn376326(v=win.10)) &mdash; технический документ, содержащий рекомендации и советы по переносу приложения с Windows 8 на Windows 8.1, от начала до конца. содержит рекомендации, рекомендуемые методы и рекомендации по повторной отправке приложения в хранилище Windows.
-   [Новые интерфейсы API и функции для разработчиков](https://msdn.microsoft.com/library/windows/apps/dn751496.aspx) &mdash; узнайте, Windows 8.1, изучив новые интерфейсы api и функции.
-   [перенацеливание приложения хранилища Windows 8 на Windows 8.1](/previous-versions/windows/apps/dn263114(v=win.10)) &mdash; пошаговое руководство по переносу приложения из Windows 8 в Windows 8.1.
-   [Обновленные примеры](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples) &mdash; примеры для Windows 8.1, демонстрирующие использование новых api и функций, а так же способы устранения устаревших проблем.
-   [Программирование DirectX](https://msdn.microsoft.com/library/windows/apps/bg182880.aspx) &mdash; Новые функции и усовершенствования для игр и графических приложений.
-   сведения об изменениях api в Windows 8.1, связанных с программированием с помощью javascript и Internet Explorer, см. в разделе [api changes for Windows 8.1 (приложения для магазина Windows, использующие JavaScript и HTML)](/previous-versions/windows/apps/dn263112(v=win.10)).
-   сведения об изменениях api в Windows 8.1, связанных с программированием на C#/VB/C + + и xaml, см. в разделе [изменения api для Windows 8.1 (приложения магазина Windows с помощью c#/VB/C + + и xaml)](/previous-versions/windows/apps/dn263110(v=win.10)) .
