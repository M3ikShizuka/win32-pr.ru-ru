# <a name="windows-store-apps"></a>Приложения для Магазина Windows

## <a name="resources"></a>Ресурсы

Перечисленные ниже ресурсы призваны помочь при переносе приложений, встроенных в Windows 8, в Windows 8.1.

-   [Обновление приложения с Windows 8 на Windows 8.1](/previous-versions/windows/dn376326(v=win.10)) &mdash; Технический документ, содержащий рекомендации и советы по переносу приложения с Windows 8 на Windows 8.1, от начала до конца. Содержит рекомендации, Рекомендуемые методы и рекомендации по повторной отправке приложения в магазин Windows.
-   [Новые интерфейсы API и функции для разработчиков](https://msdn.microsoft.com/library/windows/apps/dn751496.aspx) &mdash; Узнайте, Windows 8.1, изучив новые интерфейсы API и функции.
-   [Перенацеливание приложения Магазина Windows 8 на Windows 8.1](/previous-versions/windows/apps/dn263114(v=win.10)) &mdash; Пошаговое руководство по переносу приложения из Windows 8 в Windows 8.1.
-   [Обновленные примеры](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples) &mdash; Примеры для Windows 8.1, демонстрирующие использование новых API и функций, а так же способы устранения устаревших проблем.
-   [Программирование DirectX](https://msdn.microsoft.com/library/windows/apps/bg182880.aspx) &mdash; Новые функции и усовершенствования для игр и графических приложений.
-   Сведения об изменениях API в Windows 8.1, связанных с программированием с помощью JavaScript и Internet Explorer, см. в разделе [изменения API для Windows 8.1 (приложения для Магазина Windows, использующие JavaScript и HTML)](/previous-versions/windows/apps/dn263112(v=win.10)).
-   Сведения об изменениях API в Windows 8.1, связанных с программированием на C#/VB/C + + и XAML, см. в разделе [изменения API для Windows 8.1 (приложения Магазина Windows на c#/VB/C + + и XAML)](/previous-versions/windows/apps/dn263110(v=win.10)) .
