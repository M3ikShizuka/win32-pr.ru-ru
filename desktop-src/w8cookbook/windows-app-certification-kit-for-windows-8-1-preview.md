# <a name="windows-app-certification-kit"></a>Комплект сертификации приложений для Windows

## <a name="platforms"></a>Платформы

**клиенты —** Windows 7 \| Windows 8 \| Windows 8.1  


## <a name="description"></a>Описание

Windows комплект сертификации приложений 3,1 входит в состав Windows SDK для Windows 8.1. этот пакет можно использовать для предварительной сертификации Windows приложений магазина для Windows 8 и Windows 8.1 перед подключением, а также для программ Windows 7, Windows 8 и Windows 8.1 сертификации настольных приложений и Windows 8.1 Телефон приложений.

комплект сертификации Windows приложений 3,1 был обновлен, чтобы предоставить разработчикам простой пользовательский интерфейс, включая выполнение тестов параллельно, чтобы сэкономить время, выборочный выбор тестов и расширенные отчеты, чтобы присвоить несколько улучшений.

**Примечание.** при появлении запроса на обновление обновите версию Windows набора сертификации приложений, чтобы воспользоваться преимуществами новейших функций, входящих в комплект. когда приложение тестируется на основе последней версии Windows набора сертификации приложений, оно обеспечивает лучшую совместимость с политиками хранения Windows и программами сертификации настольных систем.
