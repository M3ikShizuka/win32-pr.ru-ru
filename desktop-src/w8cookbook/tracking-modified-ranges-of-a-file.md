# <a name="tracking-modified-ranges-of-a-file"></a>Отслеживание измененных диапазонов файла

## <a name="platforms"></a>Платформы

**Клиенты —** Windows 8.1 (все номера SKU)  
**Серверы —** Windows Server 2012 R2  

## <a name="description"></a>Описание

Команда NT File System (NTFS) добавила в Windows новую функцию. В журнале USN будет выведена запись с порядковым номером обновления (USN), содержащая измененные диапазоны для файла после закрытия. \_ \_ Для записи этих измененных диапазонов файла введен новый тип записи USN запись v4.

Эта функция не включена по умолчанию. Пользователи должны вызвать команду управления файловой системой (ФСКТЛ), чтобы включить ее. Однако, так как другие компоненты Windows могут включать отслеживание диапазонов, пользователи и разработчики могут принять во внимание, что эта функция всегда включена. Windows позволяет разработчикам запрашивать журнал USN, чтобы определить, включено ли отслеживание диапазона.

Документация MSDN будет предоставлена позже, чтобы сообщить разработчикам о том, как получить доступ \_ к \_ записям USN записи версии 4.

## <a name="manifestation"></a>Проявление

Все существующие приложения, использующие журнал USN, продолжают работать без проблем совместимости.
