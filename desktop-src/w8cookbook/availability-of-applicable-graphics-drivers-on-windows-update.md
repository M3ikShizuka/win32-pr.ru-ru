# <a name="availability-of-applicable-graphics-drivers-on-windows-update"></a>доступность применимых драйверов графики на Центр обновления Windows

доступность этих драйверов на Центр обновления Windows (WU) определяет, будет ли пользователь автоматически предлагать обновление от Windows 7, Windows 8 или Windows 8.1 до Windows 10. если при сканировании оборудования обнаружено отсутствие графического драйвера на Центр обновления Windows для графического адаптера на компьютере, пользователи не предлагали обновленную ос Windows 10. тем не менее пользователи по-прежнему могут получать Windows 10 через другие источники и вручную выполнять обновление.

некоторым пользователям не было уверений в том, почему они не предлагали обновление, когда другие люди были, несмотря на то, что в помощник по обновлению было объяснено, что для оборудования не был доступен графический драйвер.

Кроме того, некоторые пользователи принудительно находили обновление, а затем обнаружили, что их графические возможности и производительность были вызваны отсутствием драйвера оборудования.

## <a name="mitigations"></a>Устранение проблем

чтобы устранить эту ошибку, пользователи могут вручную установить старый драйвер, т. е. Windows 7, Windows 8 или Windows 8.1, перейдя на веб-сайт IHV или OEM и загрузив драйвер явным образом. Это необходимо сделать после обновления и не гарантирует работу. Нет поддерживаемого решения, если подходящий графический драйвер недоступен в WU. Пользователь должен решить, нужно ли:

-   Остаться в предыдущей ОС;
-   Примите ограничения для драйвера программного обеспечения, Microsoft Basic видеоадаптер (МСБДА); ни
-   приобретите новую графическую карту с поддерживаемым драйвером Windows 10.

## <a name="solutions"></a>Решения

очень важно, чтобы независимые поставщики программного обеспечения и изготовители оборудования перегрузили свои Windows 10 графические драйверы в WU для любого оборудования, которое они должны поддерживать.

Обратите внимание, что оборудование, которое достигло конца Live (конца строки), может не иметь драйверов в WU. В этом случае нет решения — пользователь должен выбрать один из параметров в разделе Устранение рисков выше.
