# <a name="enumeration-resume-handles"></a>Дескрипторы возобновления перечисления

Дескрипторы возобновления перечисления — это идентификаторы фактического ключа возобновления, содержащегося в данных экземпляра для функции. Это необходимо для обеспечения безопасности, взаимодействия и упрощения кода вызывающей стороны для функции.

Если для указателя на обработчик возобновления передается **значение NULL** , то обработчик не сохраняется и поиск перечисления не может быть продолжен. Это полезно в тех случаях, когда приложению не требуется перечислять все элементы.

Если при вызове перечисления возвращается ошибка, то обработчик возобновления должен рассматриваться как недопустимый и не используется для последующих вызовов перечисления. В этом случае необходимо перезапустить перечисление с начала.
