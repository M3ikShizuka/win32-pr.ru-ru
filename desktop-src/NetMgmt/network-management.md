# <a name="network-management"></a>Управление сетью

## <a name="purpose"></a>Назначение

Функции управления сетью обеспечивают возможность управления учетными записями пользователей и сетевыми ресурсами. Многие возможности, предоставляемые функциями управления сетью, не предоставляются другими сетевыми функциями. Однако если возможности предоставляются другим набором функций, документация по функциям управления сетью будет ссылаться на другие функции, которые можно использовать для той же задачи.

## <a name="developer-audience"></a>Аудитория разработчиков

Функции управления сетью предназначены для использования программистами C/C++. программисты также должны быть знакомы с концепциями Windowsных сетей

## <a name="run-time-requirements"></a>Требования к среде выполнения

функции управления сетью можно использовать на всех платформах Windows. Если существуют определенные реализации или возможности ограничений платформы управления сетью, они четко отмечены в документации.
