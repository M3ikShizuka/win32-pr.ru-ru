# <a name="netservicecontrol"></a>нетсервицеконтрол

Функция **нетсервицеконтрол** устарела. Он включен для обеспечения совместимости с 16-разрядными версиями Windows. Другие приложения должны использовать [функции службы](/windows/desktop/Services/service-functions).
