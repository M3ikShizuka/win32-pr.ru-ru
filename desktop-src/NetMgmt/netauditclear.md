# <a name="netauditclear"></a>нетаудитклеар

Функция **нетаудитклеар** устарела. Он включен для обеспечения совместимости с 16-разрядными версиями Windows. Другие приложения должны использовать [ведение журнала событий](/windows/desktop/EventLog/event-logging).
