# <a name="netservicegetinfo"></a>нетсервицежетинфо

Функция **нетсервицежетинфо** устарела. Он включен для обеспечения совместимости с 16-разрядными версиями Windows. Другие приложения должны использовать [функции службы](/windows/desktop/Services/service-functions).
