# <a name="obsolete-information-fields"></a>Поля устаревших сведений

Многие из информационных полей в основных информационных структурах будут устаревшими. Эти поля останутся в информационной структуре для обеспечения совместимости с 16-разрядными версиями Windows и будут возвращать интеллектуальные значения по умолчанию в других системах Windows.
