# <a name="nls-support"></a>Поддержка NLS

Функции управления сетью принимают строки Юникода в качестве входных данных и предоставляют строки Юникода в выходных данных. Если ваше приложение обычно работает со строками ANSI, необходимо соблюдать осторожность при преобразовании в Юникод, где это необходимо.
