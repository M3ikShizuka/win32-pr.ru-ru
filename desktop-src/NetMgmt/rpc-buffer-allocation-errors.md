# <a name="rpc-buffer-allocation-errors"></a>Ошибки выделения буфера RPC

Так как библиотека времени выполнения RPC выделяет память для буферов отправки и получения, функция должна ждать ошибки выделения памяти RPC. В случае ошибки выделения RPC, возобновляемый обработчик становится недействительным. Это требование обусловлено тем, что возобновляемые функции не могут перематываться.
