# <a name="application-scenarios-for-sens"></a>Сценарии приложений для Сенс

Некоторые типы приложений могут использовать функции подключения и службы уведомлений, предоставляемые службой уведомлений о системных событиях.

-   Приложение, для которого требуется состояние сетевого подключения, например приложение, использующее службы каталогов.
-   Приложение, которое адаптирует свои операции в зависимости от уровня подключения и качества сетевых служб, таких как веб-приложение Internet Explorer, которое работает на более низком уровне при низкой пропускной способности подключения.
-   Приложение, которое может выполнять отложенные операции, такие как программа электронной почты, которая может ставить сообщения в очередь в автономном режиме и отсылать их при установке соединения.
