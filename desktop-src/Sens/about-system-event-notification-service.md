# <a name="about-system-event-notification-service"></a>О службе уведомлений о системных событиях

Приложениям, предназначенным для использования мобильными пользователями, требуется уникальный набор функций подключения и уведомлений, а в прошлых отдельных приложениях для внутренней реализации этих функций требовалось использовать уникальные. Служба уведомлений о системных событиях (Сенс) теперь предоставляет эти возможности в операционной системе, создавая единый интерфейс подключения и уведомления для приложений. Сенс, диспетчер синхронизации и кэширование на стороне клиента объединяют, чтобы обеспечить инфраструктуру для полной поддержки мобильных вычислений.

Подразделы этого раздела содержат общие сведения о Сенс.

-   [Конфигурации мобильных вычислений для Сенс](mobile-computing-configurations-for-sens.md)

    Содержит несколько конфигураций мобильных компьютеров, которые могут использовать преимущества приложений с поддержкой Сенс.

-   [Сценарии приложений для Сенс](application-scenarios-for-sens.md)

    Содержит несколько типов приложений, которые могут выиграть при использовании функций Сенс.

-   [Уведомления](notifications.md)

    Список системных событий, Сенс мониторы и диспетчеризации.

-   [Архитектура Сенс](sens-architecture.md)

    Описание внутренней архитектуры Сенс.
