# <a name="mobile-computing-configurations-for-sens"></a>Конфигурации мобильных вычислений для Сенс

Функции подключения и уведомления полезны для компьютеров, настроенных следующим образом.

-   Мобильный компьютер, используемый на стыковочный станции в сети с высокой пропускной способностью, который может иногда использовать коммутируемое подключение.
-   Мобильный компьютер, использующий только коммутируемое соединение.
-   Настольный компьютер, использующий только коммутируемое подключение.
-   Настольный компьютер, подключенный к сети с высокой пропускной способностью, с проблемами задержки.

В каждой из этих конфигураций данные о пропускной способности и задержке подключения могут использоваться приложением для динамической оптимизации своих операций по доступности сети.
