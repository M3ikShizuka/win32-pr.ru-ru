| Раздел                                                           | Описание                                                                                                                                          |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_Параметры контекста \_ TBS**](/windows/win32/api/tbs/ns-tbs-tbs_context_params)<br/>   | Указывает версию реализации контекста TBS.<br/>                                                                                  |
| [**\_PARAMS2 контекста \_ TBS**](/windows/win32/api/tbs/ns-tbs-tbs_context_params2)<br/> | Указывает версию реализации контекста TBS. Эту структуру необходимо использовать, если приложение работает с обеими версиями TPM.<br/> |
