# <a name="tbs-data-types"></a>Типы данных TBS

TBS определяет следующие типы данных.



| Тип данных                                                                                                | Описание                                                                                                                                                            |
|----------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TBS_HCONTEXT"></span><span id="tbs_hcontext"></span>**TBS \_ хконтекст**                          | Обработчик объекта контекста.<br/>                                                                                                                                      |
| <span id="TBS_COMMAND_PRIORITY"></span><span id="tbs_command_priority"></span>**\_приоритет команды \_ TBS** | Приоритет, связанный с командами.<br/>                                                                                                                          |
| <span id="TBS_COMMAND_LOCALITY"></span><span id="tbs_command_locality"></span>**\_ \_ Локализация команд TBS** | Локальность, из которой отправляется команда.<br/>                                                                                                            |
| <span id="TBS_OWNERAUTH_TYPE"></span><span id="tbs_ownerauth_type"></span>**\_тип ОВНЕРАУС \_ TBS**       | Тип проверки подлинности владельца.<br/> **Windows server 2008 R2, Windows 7, Windows Server 2008 и Windows Vista:** Этот тип данных недоступен.<br/> |
