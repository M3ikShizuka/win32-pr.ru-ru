# <a name="inapcomponentconfig2invokeuifromconfigblob-method"></a>Метод INapComponentConfig2:: Инвокеуифромконфигблоб

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инвокеуифромконфигблоб** реализуется средствами проверки работоспособности системы (SHV), необходимыми для загрузки конфигурации удаленного или локального компьютера в памяти и вывода пользовательского интерфейса, позволяющего манипулировать данными конфигурации. Компонент конфигурации должен поддерживать использование [**инапкомпонентконфиг**](inapcomponentconfig.md) через DCOM.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT InvokeUIFromConfigBlob(
  [in, unique] HWND   hwndParent,
  [in]         UINT16 inbCount,
  [in]         BYTE   *inData,
  [out]        UINT16 *outbCount,
  [out]        BYTE   **outdata,
  [out]        BOOL   *fConfigChanged
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хвндпарент* \[ окне\]
</dt> <dd>

Маркер родительского окна или окно-владельца. Необходимо указать допустимый описатель окна.

</dd> <dt>

*инбкаунт* \[ окне\]
</dt> <dd>

Размер *данных* в байтах.

</dd> <dt>

*данные* \[ окне\]
</dt> <dd>

Указатель на буфер, в котором хранятся исходные данные конфигурации. Эти данные экспортируются с удаленного или локального компьютера.

</dd> <dt>

*аутбкаунт* \[ заполняет\]
</dt> <dd>

Размер *данных* в байтах.

</dd> <dt>

*данные* \[ заполняет\]
</dt> <dd>

Указатель на буфер, в котором хранятся данные конфигурации, измененные пользовательским интерфейсом конфигурации SHV. Эти данные импортируются удаленным или локальным компьютером.

</dd> <dt>

*фконфигчанжед* \[ заполняет\]
</dt> <dd>

Указатель на **логическое** значение, равное **true** , если конфигурация была изменена во время операции пользовательского интерфейса, и **false** в противном случае.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение \_ ОК, если успешно, или один из стандартных кодов ошибок Windows.

## <a name="remarks"></a>Комментарии

Если используется для локального компьютера, эту функцию можно использовать для каждой конфигурации SHV политики. Он предоставляет способ вызова пользовательского интерфейса SHV и изменения существующей конфигурации SHV.
