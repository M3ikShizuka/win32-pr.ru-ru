# <a name="inapenforcementclientconnectionsetflags-method"></a>Метод Инапенфорцементклиентконнектион:: Сетфлагс

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапенфорцементклиентконнектион:: сетфлагс** используется для различения ответов первого времени от ответов из-за того, что сохрекуестс кэшируется принудительно.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetFlags(
  [in] UINT8 flags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Флаги* \[ окне\]
</dt> <dd>

Флаги, определяющие, является ли [**сохреспонсе**](/windows/win32/api/naptypes/ns-naptypes-soh) причиной кэширования **сохрекуест**. Если *Флаги* имеют значение [**фрешсохрекуест**](nap-type-constants.md), это новый запрос; в противном случае это кэшированный запрос.

</dd> </dl>
