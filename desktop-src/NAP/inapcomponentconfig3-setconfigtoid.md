# <a name="inapcomponentconfig3setconfigtoid-method"></a>Метод INapComponentConfig3:: Сетконфигтоид

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **сетконфигтоид** реализуется средствами проверки работоспособности системы (SHV) для предоставления способа установки данных конфигурации для конкретного идентификатора конфигурации.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetConfigToID(
  [in] UINT32 configID,
  [in] UINT16 count,
  [in] BYTE   *data
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*конфигид* \[ окне\]
</dt> <dd>

Значение, представляющее конфигурацию. *Конфигид* назначается службой сервера политики сети (NPS) и уникальна в SHV. Если *конфигид* имеет значение 0, задается конфигурация по умолчанию SHV.

</dd> <dt>

*количество* \[ окне\]
</dt> <dd>

Размер (в байтах) данных конфигурации в *данных*.

</dd> <dt>

*данные* \[ окне\]
</dt> <dd>

На входе — массив БАЙТОВ, который содержит набор данных конфигурации для *конфигид*.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает один из следующих кодов ошибок в зависимости от результата этой операции.



| Код возврата                                                                                                    | Описание                             |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> </dl>                          | Операция прошла успешно.<br/> |
| <dl> <dt>**\_ \_ \_ \_ не \_ найдена Конфигурация NAP E SHV**</dt> </dl> | Не удается найти *конфигид* .<br/>  |



 

## <a name="remarks"></a>Remarks

Перед вызовом этого метода необходимо использовать метод [**документе newconfig**](inapcomponentconfig3-newconfig.md) для выделения данных конфигурации для *конфигид* .
