# <a name="inapenforcementclientconnectiongetmaxsize-method"></a>Метод Инапенфорцементклиентконнектион:: Жетмакссизе

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапенфорцементклиентконнектион:: жетмакссизе** получает максимальный размер пакета SoH для этого клиента.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetMaxSize(
  [out] ProtocolMaxSize *maxSize
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*MAXSIZE* \[ заполняет\]
</dt> <dd>

Указатель на [**протоколмакссизе**](nap-datatypes.md) , указывающий максимальный размер пакета SoH в байтах.

</dd> </dl>

Также могут возвращаться другие коды ошибок, относящиеся к COM.



| Код возврата                                                                                     | Описание                                                        |
|-------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> </dl>           | Операция успешно завершена.<br/>                                    |
| <dl> <dt>**Д \_ ACCESSDENIED**</dt> </dl> | Ошибка разрешений, доступ запрещен.<br/>                       |
| <dl> <dt>**Д \_ OUTOFMEMORY**</dt> </dl>  | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/> |
