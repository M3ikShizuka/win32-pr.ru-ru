# <a name="inapcomponentconfig3getconfigfromid-method"></a>Метод INapComponentConfig3:: Жетконфигфромид

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **жетконфигфромид** реализуется средствами проверки работоспособности системы (SHV) для предоставления данных конфигурации для определенного идентификатора конфигурации.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetConfigFromID(
  [in]  UINT32 configID,
  [out] UINT16 *count,
  [out] BYTE   **outdata
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*конфигид* \[ окне\]
</dt> <dd>

Значение, представляющее конфигурацию. Если *конфигид* имеет значение **0**, SHV должен вернуть данные конфигурации по умолчанию в *данные*.

</dd> <dt>

*количество* \[ заполняет\]
</dt> <dd>

Размер (в байтах) данных конфигурации, возвращаемых в *данных*.

</dd> <dt>

*данные* \[ заполняет\]
</dt> <dd>

При возвращении — массив БАЙТОВ, содержащий данные конфигурации, представленные параметром *конфигид*.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает один из следующих кодов ошибок в зависимости от результата этой операции.



| Код возврата                                                                                                    | Описание                             |
|----------------------------------------------------------------------------------------------------------------|-----------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> </dl>                          | Операция прошла успешно.<br/> |
| <dl> <dt>**\_ \_ \_ \_ не \_ найдена Конфигурация NAP E SHV**</dt> </dl> | Не удается найти *конфигид* .<br/>  |



 

## <a name="remarks"></a>Комментарии

Перед вызовом этого метода необходимо использовать метод [**документе newconfig**](inapcomponentconfig3-newconfig.md) для выделения данных конфигурации для *конфигид* .
