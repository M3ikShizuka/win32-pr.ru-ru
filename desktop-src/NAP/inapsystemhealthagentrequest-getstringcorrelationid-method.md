# <a name="inapsystemhealthagentrequestgetstringcorrelationid-method"></a>Метод Инапсистемхеалсажентрекуест:: Жетстрингкоррелатионид

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапсистемхеалсажентрекуест:: жетстрингкоррелатионид** используется агентами работоспособности системы для записи идентификатора корреляции.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetStringCorrelationId(
  [out] StringCorrelationId **correlationId
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*correlationId* \[ заполняет\]
</dt> <dd>

Указатель на указатель на уникальный идентификатор [**correlationId**](/windows/win32/api/naptypes/ns-naptypes-correlationid) для этого обмена состояния работоспособности.

</dd> </dl>
