# <a name="inapcomponentconfig2-interface"></a>Интерфейс INapComponentConfig2

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Интерфейс **INapComponentConfig2** предоставляет методы настройки системы защиты доступа к сети для средств проверки работоспособности системы (SHV) для удаленной настройки пользовательского интерфейса сервера политики сети (NPS).

> [!Note]  
> Этот интерфейс наследует все методы [**инапкомпонентконфиг**](inapcomponentconfig.md) и использовать вместо него.

 

## <a name="members"></a>Элементы

Интерфейс **INapComponentConfig2** наследует от [**инапкомпонентконфиг**](inapcomponentconfig.md). **INapComponentConfig2** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **INapComponentConfig2** содержит следующие методы.



| Метод                                                                                                | Описание                                                                                                                                                                |
|:------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**INapComponentConfig2:: Инвокеуиформачине**](inapcomponentconfig2-invokeuiformachine.md)           | Реализуется средствами SHV по мере необходимости для управления удаленной конфигурацией непосредственно на указанном компьютере.<br/>                                                                 |
| [**INapComponentConfig2:: Инвокеуифромконфигблоб**](inapcomponentconfig2-invokeuifromconfigblob.md)   | Реализуется средствами SHV по мере необходимости для загрузки конфигурации удаленного компьютера в памяти и вывода пользовательского интерфейса, позволяющего манипулировать данными конфигурации.<br/> |
| [**INapComponentConfig2:: Исремотеконфигсуппортед**](inapcomponentconfig2-isremoteconfigsupported.md) | Реализуется средствами SHV, чтобы указать, поддерживается ли удаленная конфигурация.<br/>                                                                                      |



 

## <a name="remarks"></a>Remarks

Этот интерфейс не должен реализовываться агентами работоспособности системы (SHA) или клиентами принудительного применения карантина (кекс).
