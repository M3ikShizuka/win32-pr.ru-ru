# <a name="inapsystemhealthagentcallbackgetfixupinfo-method"></a>Метод Инапсистемхеалсаженткаллбакк:: Жетфиксупинфо

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапсистемхеалсаженткаллбакк:: жетфиксупинфо** вызывается напажент для определения состояния агента работоспособности системы во время обработки [**сохреспонсе**](/windows/win32/api/naptypes/ns-naptypes-soh).

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetFixupInfo(
  [out] FixupInfo **info
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*сведения о* \[ заполняет\]
</dt> <dd>

Указатель на указатель на структуру [**фиксупинфо**](/windows/win32/api/naptypes/ns-naptypes-fixupinfo) , которая содержит состояние исправления агента.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Код возврата                                                                          | Описание                   |
|--------------------------------------------------------------------------------------|-------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Указывает на успешное завершение.<br/> |



 

## <a name="remarks"></a>Комментарии

Этот метод обратного вызова объявляется системой защиты доступа к сети и реализуется модулем записи SHA.

Агент работоспособности системы должен вернуть значение **S \_ ОК** немедленно без блокировки.
