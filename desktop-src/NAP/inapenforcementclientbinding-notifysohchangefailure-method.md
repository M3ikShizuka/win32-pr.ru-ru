# <a name="inapenforcementclientbindingnotifysohchangefailure-method"></a>Метод Инапенфорцементклиентбиндинг:: Нотифисохчанжефаилуре

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапенфорцементклиентбиндинг:: нотифисохчанжефаилуре** используется клиентом принудительного применения для информирования напажент о том, что ему не удалось обработать предыдущий [**Инапенфорцементклиенткаллбакк:: NotifySoHChange**](inapenforcementclientcallback-notifysohchange-method.md).

## <a name="syntax"></a>Синтаксис


```C++
HRESULT NotifySoHChangeFailure();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Также могут возвращаться другие коды ошибок, относящиеся к COM.



| Код возврата                                                                                             | Описание                                                        |
|---------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> </dl>                   | Операция прошла успешно.<br/>                            |
| <dl> <dt>**Д \_ ACCESSDENIED**</dt> </dl>         | Ошибка разрешений, доступ запрещен.<br/>                       |
| <dl> <dt>**Д \_ OUTOFMEMORY**</dt> </dl>          | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/> |
| <dl> <dt>**Защита доступа к сети \_ E \_ не \_ инициализирована**</dt> </dl> | Перед инициализацией не был инициализирован.<br/>       |



 

## <a name="remarks"></a>Комментарии

В результате вызова этого метода Напажент повторные попытки применить изменение состояния работоспособности позже, вызвав [**инапенфорцементклиенткаллбакк:: нотифисохчанже**](inapenforcementclientcallback-notifysohchange-method.md) еще раз. После того как клиент принудительного применения вызвал [**инапенфорцементклиентбиндинг:: жетсохрекуест**](inapenforcementclientbinding-getsohrequest-method.md), он должен применить изменение, т. е. сбои не обрабатываются напажент после этой точки.

Клиент принудительного применения должен вызвать метод [**инапенфорцементклиентбиндинг:: Initialize**](inapenforcementclientbinding-initialize-method.md) перед вызовом этого или любого другого метода интерфейса [**инапенфорцементклиентбиндинг**](inapenforcementclientbinding.md) .
