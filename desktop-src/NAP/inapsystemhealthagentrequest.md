# <a name="inapsystemhealthagentrequest-interface"></a>Интерфейс Инапсистемхеалсажентрекуест

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Интерфейс **инапсистемхеалсажентрекуест** предоставляет методы, которые SHA использует для обмена данными и координации обработки в системе защиты доступа к сети.

## <a name="members"></a>Элементы

Интерфейс **инапсистемхеалсажентрекуест** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Инапсистемхеалсажентрекуест** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **инапсистемхеалсажентрекуест** содержит следующие методы.



| Метод                                                                                                                     | Описание                                                                                      |
|:---------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------|
| [**Инапсистемхеалсажентрекуест:: Жеткачесохфлаг**](inapsystemhealthagentrequest-getcachesohflag-method.md)               | Используется только Напажент.<br/>                                                            |
| [**Инапсистемхеалсажентрекуест:: Жеткоррелатионид**](inapsystemhealthagentrequest-getcorrelationid-method.md)             | Используется агентами работоспособности системы для корреляции SoH и [**сохреспонсе**](/windows/win32/api/naptypes/ns-naptypes-soh).<br/> |
| [**Инапсистемхеалсажентрекуест:: Жетсохрекуест**](inapsystemhealthagentrequest-getsohrequest-method.md)                   | Используется SHA для получения SoH, ранее кэшированного Напажент.<br/>                           |
| [**Инапсистемхеалсажентрекуест:: Жетсохреспонсе**](inapsystemhealthagentrequest-getsohresponse-method.md)                 | Используется агентом работоспособности для получения [**сохреспонсе**](/windows/win32/api/naptypes/ns-naptypes-soh).<br/>         |
| [**Инапсистемхеалсажентрекуест:: Жетстрингкоррелатионид**](inapsystemhealthagentrequest-getstringcorrelationid-method.md) | Используется агентами работоспособности системы для регистрации идентификатора корреляции.<br/>                               |
| [**Инапсистемхеалсажентрекуест:: Сетсохрекуест**](inapsystemhealthagentrequest-setsohrequest-method.md)                   | Используется агентами работоспособности для записи своего запроса SoH.<br/>                                     |
