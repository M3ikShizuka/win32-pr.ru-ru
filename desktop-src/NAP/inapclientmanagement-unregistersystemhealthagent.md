# <a name="inapclientmanagementunregistersystemhealthagent-method"></a>Метод Инапклиентманажемент:: Унрегистерсистемхеалсажент

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **унрегистерсистемхеалсажент** отменяет регистрацию SHA в системе защиты доступа к сети.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT UnregisterSystemHealthAgent(
  [in] SystemHealthEntityId id
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*идентификатор* \[ в\]
</dt> <dd>

Объект [**системхеалсентитид**](nap-datatypes.md) , определяющий отмену регистрации агента работоспособности системы.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает код состояния HRESULT, включая, помимо прочего, один из следующих.



| Код возврата                                                                                         | Описание                                                        |
|-----------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                | Операция выполнена успешно.<br/>                                   |
| <dl> <dt>**E \_ ACCESSDENIED**</dt> </dl>      | Ошибка разрешений, доступ запрещен.<br/>                       |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl>       | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/> |
| <dl> <dt>**\_ \_ по-прежнему \_ привязанный к NAP E**</dt> </dl> | SHA остается привязанным и не может быть отменена.<br/>    |
