# <a name="inapcomponentinfogetfriendlyname-method"></a>Метод Инапкомпонентинфо:: Жетфриендлинаме

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод обратного вызова **инапкомпонентинфо:: жетфриендлинаме** используется системой защиты доступа к сети для получения понятного имени клиента работоспособности.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetFriendlyName(
  [out] MessageId *friendlyName
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*FriendlyName* \[ заполняет\]
</dt> <dd>

Указатель на [**MessageId**](nap-datatypes.md) , содержащий идентификатор ресурса понятного имени.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает один из этих кодов ошибок в зависимости от результата этой операции.



| Код возврата                                                                                     | Описание                                                        |
|-------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> </dl>           | Операция прошла успешно.<br/>                            |
| <dl> <dt>**Д \_ ACCESSDENIED**</dt> </dl> | Ошибка разрешений, доступ запрещен.<br/>                       |
| <dl> <dt>**Д \_ OUTOFMEMORY**</dt> </dl>  | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/> |
