# <a name="inapenforcementclientconnectionsetsohresponse-method"></a>Метод Инапенфорцементклиентконнектион:: Сетсохреспонсе

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапенфорцементклиентконнектион:: сетсохреспонсе** задает SoH-Response и используется клиентом принудительного применения при получении пакета.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetSoHResponse(
  [in] const NetworkSoHResponse *sohResponse
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*сохреспонсе* \[ окне\]
</dt> <dd>

Указатель на уникальную структуру [**нетворксохреспонсе**](/windows/win32/api/naptypes/ns-naptypes-networksoh) , которая представляет собой большой двоичный объект непрозрачных данных для принудительного применения.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Также могут возвращаться другие коды ошибок, относящиеся к COM.



| Код возврата                                                                                     | Описание                                                        |
|-------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> </dl>           | Пакет SoH является допустимым.<br/>                                |
| <dl> <dt>**Д \_ ACCESSDENIED**</dt> </dl> | Ошибка разрешений, доступ запрещен.<br/>                       |
| <dl> <dt>**Д \_ OUTOFMEMORY**</dt> </dl>  | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/> |
