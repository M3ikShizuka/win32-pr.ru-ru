# <a name="inapclientmanagementunregisterenforcementclient-method"></a>Метод Инапклиентманажемент:: Унрегистеренфорцементклиент

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **унрегистеренфорцементклиент** отменяет регистрацию клиента принудительного применения в системе защиты доступа к сети.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT UnregisterEnforcementClient(
  [in] EnforcementEntityId id
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*идентификатор* \[ в\]
</dt> <dd>

Значение [**енфорцементентитид**](nap-datatypes.md) , идентифицирующее клиент принудительного применения для отмены регистрации.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает код состояния HRESULT, включая, помимо прочего, один из следующих.



| Код возврата                                                                                         | Описание                                                                    |
|-----------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                | Операция выполнена успешно.<br/>                                               |
| <dl> <dt>**E \_ ACCESSDENIED**</dt> </dl>      | Ошибка разрешений, доступ запрещен.<br/>                                   |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl>       | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/>             |
| <dl> <dt>**\_ \_ по-прежнему \_ привязанный к NAP E**</dt> </dl> | Не удалось отменить регистрацию клиента принудительного применения и он останется привязанным.<br/> |
