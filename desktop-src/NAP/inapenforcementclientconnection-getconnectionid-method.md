# <a name="inapenforcementclientconnectiongetconnectionid-method"></a>Метод Инапенфорцементклиентконнектион:: Жетконнектионид

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапенфорцементклиентконнектион:: жетконнектионид** используется для получения уникального идентификатора соединения клиента.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetConnectionId(
  [out] ConnectionId **connectionId
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*connectionId* \[ заполняет\]
</dt> <dd>

Указатель на указатель на [**ConnectionId**](nap-datatypes.md) , однозначно определяющий это соединение.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Также могут возвращаться другие коды ошибок, относящиеся к COM.



| Код возврата                                                                                     | Описание                                                        |
|-------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> </dl>           | Операция успешно завершена.<br/>                                    |
| <dl> <dt>**Д \_ ACCESSDENIED**</dt> </dl> | Ошибка разрешений, доступ запрещен.<br/>                       |
| <dl> <dt>**Д \_ OUTOFMEMORY**</dt> </dl>  | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/> |



 

## <a name="remarks"></a>Remarks

Идентификатор соединения в основном используется в целях ведения журнала.
