# <a name="uninitializenapagentnotifier-function"></a>Функция Унинитиализенапажентнотифиер

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Функция **унинитиализенапажентнотифиер** отменяет подписывание вызывающего процесса от уведомлений об изменении состояния напажент и уведомлений об изменении состояния карантина. Эти уведомления предоставляются службой Напажент.

## <a name="syntax"></a>Синтаксис


```C++
NAPAPI VOID WINAPI UninitializeNapAgentNotifier(
  _In_ NapNotifyType type
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*тип* \[ окне\]
</dt> <dd>

Значение [**напнотифитипе**](/windows/win32/api/naptypes/ne-naptypes-napnotifytype) , указывающее тип уведомлений службы, с которых отменяется подписка.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не имеет возвращаемых значений.

## <a name="remarks"></a>Комментарии

Эта функция не является потокобезопасной.

Каждый процесс, подписанный на уведомления службы Напажент указанного *типа* , должен вызывать **унинитиализенапажентнотифиер** для отмены подписки на уведомления. Если процесс подписан на более одного типа уведомлений, он должен вызывать **унинитиализенапажентнотифиер** один раз для каждого типа уведомления.

Эта функция завершается со сбоем автоматически, если процесс ранее не вызывал [**инитиализенапажентнотифиер**](initializenapagentnotifier.md) для типа уведомления.
