# <a name="inapsystemhealthagentbindinginitialize-method"></a>Метод Инапсистемхеалсажентбиндинг:: Initialize

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапсистемхеалсажентбиндинг:: Initialize** инициализирует агент работоспособности системы (SHA) и привязывает SHA к службе напажент. Этот метод должен вызываться перед вызовом любого другого метода в интерфейсе [**INapSystemHealthAgentBinding2**](inapsystemhealthagentbinding2.md) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Initialize(
  [in] SystemHealthEntityId          id,
  [in] INapSystemHealthAgentCallback *callback
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*идентификатор* \[ в\]
</dt> <dd>

Уникальный [системхеалсентитид](nap-datatypes.md) , СОДЕРЖАЩИЙ идентификатор SHA, привязанный к службе напажент.

</dd> <dt>

*обратный вызов* \[ окне\]
</dt> <dd>

COM-указатель на интерфейс [**инапсистемхеалсаженткаллбакк**](inapsystemhealthagentcallback.md) , используемый напажент для обратного вызова агента работоспособности с уведомлением или процессом. Напажент содержит ссылку на объект, связанный с этим интерфейсом, пока не будет вызвана [**неИнициализация**](inapsystemhealthagentbinding-uninitialize-method.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Также могут возвращаться другие коды ошибок, относящиеся к COM.



| Код возврата                                                                                                | Описание                                                                                                                    |
|------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> </dl>                      | Операция успешно завершена.<br/>                                                                                                |
| <dl> <dt>**Д \_ ACCESSDENIED**</dt> </dl>            | Ошибка разрешений, доступ запрещен.<br/>                                                                                   |
| <dl> <dt>**Д \_ OUTOFMEMORY**</dt> </dl>             | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/>                                                             |
| <dl> <dt>**Ошибка \_ уже \_ инициализирована**</dt> </dl> | Если SHA был инициализирован ранее, возвращается эта ошибка.<br/>                                                      |
| <dl> <dt>**Защита доступа к сети \_ E \_ не \_ зарегистрирована**</dt> </dl>     | Если SHA не был зарегистрирован ранее, возвращается эта ошибка.<br/>                                                      |
| <dl> <dt>**RPC \_ E \_ отключен**</dt> </dl>        | Напажент остановлен. После перезапуска этот объект будет автоматически восстановлен и повторно привязан к Напажент.<br/> |



 

## <a name="remarks"></a>Комментарии

Напажент не запускает обмен данными о состоянии работоспособности в результате инициализации. Агент работоспособности системы должен вызвать [**нотифисохчанже**](inapsystemhealthagentbinding-notifysohchange-method.md) , чтобы запросить обмен пакетами SoH после инициализации с помощью напажент.
