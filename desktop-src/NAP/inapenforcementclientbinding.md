# <a name="inapenforcementclientbinding-interface"></a>Интерфейс Инапенфорцементклиентбиндинг

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

**Инапенфорцементклиентбиндинг** предоставляет методы, используемые клиентами для взаимодействия с напажент.

## <a name="members"></a>Элементы

Интерфейс **инапенфорцементклиентбиндинг** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Инапенфорцементклиентбиндинг** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **инапенфорцементклиентбиндинг** содержит следующие методы.



| Метод                                                                                                                           | Описание                                                                                                                                                                                                           |
|:---------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Инапенфорцементклиентбиндинг:: CreateConnection**](inapenforcementclientbinding-createconnection-method.md)                   | Используется принудительными триггерами для создания объектов подключения.<br/>                                                                                                                                                            |
| [**Инапенфорцементклиентбиндинг:: Жетсохрекуест**](inapenforcementclientbinding-getsohrequest-method.md)                         | Используется клиентом принудительного применения, когда ему требуется запрос SoH для определенного соединения.<br/>                                                                                                                   |
| [**Инапенфорцементклиентбиндинг:: Initialize**](inapenforcementclientbinding-initialize-method.md)                               | Запускает службу Напажент. Клиент принудительного применения должен вызвать этот метод перед вызовом любого другого метода этого интерфейса.<br/>                                                                            |
| [**Инапенфорцементклиентбиндинг:: Нотификоннектионстатедовн**](inapenforcementclientbinding-notifyconnectionstatedown-method.md) | Используется для информирования Напажент о том, что подключение к клиенту принудительного применения было прервано.<br/>                                                                                                                      |
| [**Инапенфорцементклиентбиндинг:: Нотифисохчанжефаилуре**](inapenforcementclientbinding-notifysohchangefailure-method.md)       | Используется клиентом принудительного применения для информирования Напажент о том, что ему не удалось обработать предыдущий [**инапенфорцементклиенткаллбакк:: нотифисохчанже**](inapenforcementclientcallback-notifysohchange-method.md).<br/> |
| [**Инапенфорцементклиентбиндинг::P Роцесссохреспонсе**](inapenforcementclientbinding-processsohresponse-method.md)               | Используется клиентами принудительного применения при получении SoH-Responseного большого двоичного объекта данных с сервера принудительного применения.<br/>                                                                                                       |
| [**Инапенфорцементклиентбиндинг:: Uninitialize**](inapenforcementclientbinding-uninitialize-method.md)                           | Завершает службу Напажент для этого клиентского подключения.<br/>                                                                                                                                                 |



 

## <a name="remarks"></a>Комментарии

Все API-интерфейсы в этом интерфейсе будут возвращать RPC \_ E \_ Disconnected, если напажент остановлен. После перезапуска этот объект будет автоматически восстановлен и повторно привязан к Напажент.
