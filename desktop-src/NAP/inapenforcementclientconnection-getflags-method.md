# <a name="inapenforcementclientconnectiongetflags-method"></a>Метод Инапенфорцементклиентконнектион::

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапенфорцементклиентконнектион::** сохрекуестс используется для различения ответов первого времени от ответов из-за, кэшируемых с помощью триггеров.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetFlags(
  [out] UINT8 *flags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Флаги* \[ заполняет\]
</dt> <dd>

Указатель на флаг, определяющий, является ли [**сохреспонсе**](/windows/win32/api/naptypes/ns-naptypes-soh) из-за кэшированного **сохрекуест**. Если *Флаги* имеют значение [**фрешсохрекуест**](nap-type-constants.md), это новый запрос; в противном случае это кэшированный запрос.

</dd> </dl>
