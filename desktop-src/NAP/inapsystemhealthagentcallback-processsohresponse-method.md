# <a name="inapsystemhealthagentcallbackprocesssohresponse-method"></a>Инапсистемхеалсаженткаллбакк: метод:P Роцесссохреспонсе

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапсистемхеалсаженткаллбакк::P роцесссохреспонсе** вызывается, когда Напажент получает [**сохреспонсе**](/windows/win32/api/naptypes/ns-naptypes-soh) , предназначенный для этого агента работоспособности.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT ProcessSoHResponse(
  [in] INapSystemHealthAgentRequest *request
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*запрос* \[ окне\]
</dt> <dd>

Указатель COM на объект [**инапсистемхеалсажентрекуест**](inapsystemhealthagentrequest.md) , который идентифицирует объект запроса.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Код возврата                                                                                            | Описание                                                                              |
|--------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                   | Указывает на успешное завершение.<br/>                                                            |
| <dl> <dt>**\_ \_ Недопустимый \_ пакет NAP E**</dt> </dl> | Возвращается этой реализацией, если ответ имеет неправильный формат.<br/> |



 

## <a name="remarks"></a>Комментарии

Этот метод обратного вызова объявляется системой защиты доступа к сети и реализуется модулем записи SHA.

Когда Напажент получает [**сохреспонсе**](/windows/win32/api/naptypes/ns-naptypes-soh) , предназначенный для этого агента работоспособности, он вызывает этот метод. Агент работоспособности должен запросить Сохреспонсе из объекта Request. После завершения этого вызова он не должен содержать ссылки на объект запроса.

Метод **инапсистемхеалсаженткаллбакк::P роцесссохреспонсе** не должен блокироваться. Если требуется обработка исправления, любая реализация **процесссохреспонсе** должна запустить новый поток для выполнения процесса исправления. Напажент должен вызвать [**инапсистемхеалсаженткаллбакк:: жетфиксупинфо**](inapsystemhealthagentcallback-getfixupinfo-method.md) , чтобы определить состояние исправления SHA.

Этот метод должен возвращать **\_ \_ Недопустимый \_ пакет NAP E** , если ответ имеет неправильный формат.
