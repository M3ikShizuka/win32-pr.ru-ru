# <a name="inapsohprocessorinitialize-method"></a>Метод Инапсохпроцессор:: Initialize

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапсохпроцессор:: Initialize** инициализирует пакет протокола и систему процессора работоспособности. Этот метод должен вызываться только один раз.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Initialize(
  [in]  const SoH                  *soh,
  [in]        BOOL                 isRequest,
  [out]       SystemHealthEntityId *id
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*состояние работоспособности* \[ окне\]
</dt> <dd>

Указатель на пакет [**SoH**](/windows/win32/api/naptypes/ns-naptypes-soh) для обработки.

</dd> <dt>

*запрос* \[ окне\]
</dt> <dd>

**Логическое** **значение** , равное true, если пакет является [**сохрекуест**](/windows/win32/api/naptypes/ns-naptypes-soh) , и **false** , если это **сохреспонсе**.

</dd> <dt>

*идентификатор* \[ исходящей\]
</dt> <dd>

Уникальный [системхеалсентитид](nap-datatypes.md) , СОДЕРЖАЩИЙ идентификатор SHA или SHV, который разстроил пакет.

</dd> </dl>
