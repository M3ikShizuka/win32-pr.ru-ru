# <a name="inapsystemhealthvalidationrequest-interface"></a>Интерфейс Инапсистемхеалсвалидатионрекуест

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Интерфейс **инапсистемхеалсвалидатионрекуест** предоставляет методы, используемые для поддержки средств проверки работоспособности системы, которые определяются разработчиком приложения.

> [!Note]  
> [**INapSystemHealthValidationRequest2**](inapsystemhealthvalidationrequest2.md) наследует все методы этого интерфейса, и вместо него следует использовать.

 

## <a name="members"></a>Элементы

Интерфейс **инапсистемхеалсвалидатионрекуест** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Инапсистемхеалсвалидатионрекуест** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **инапсистемхеалсвалидатионрекуест** содержит следующие методы.



| Метод                                                                                                                               | Описание                                                                                                           |
|:-------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------|
| [**Инапсистемхеалсвалидатионрекуест:: Жеткоррелатионид**](inapsystemhealthvalidationrequest-getcorrelationid-method.md)             | Используется проверяющими элементами управления для получения идентификатора корреляции. Затем проверяющий элемент управления должен записать эти сведения в журнал.<br/>           |
| [**Инапсистемхеалсвалидатионрекуест:: Жетмачиненаме**](inapsystemhealthvalidationrequest-getmachinename-method.md)                 | Используется проверяющими элементами управления для получения имени компьютера. Затем проверяющий элемент управления должен записать эти сведения в журнал.<br/>             |
| [**Инапсистемхеалсвалидатионрекуест:: Жетприватедата**](inapsystemhealthvalidationrequest-getprivatedata-method.md)                 | Позволяет Напсервер получать сведения о состоянии.<br/>                                                        |
| [**Инапсистемхеалсвалидатионрекуест:: Жетсохрекуест**](inapsystemhealthvalidationrequest-getsohrequest-method.md)                   | Позволяет проверяющим элементам извлекать и проверять сведения Сохрекуест.<br/>                                     |
| [**Инапсистемхеалсвалидатионрекуест:: Жетсохреспонсе**](inapsystemhealthvalidationrequest-getsohresponse-method.md)                 | Возвращает объект Сохреспонсе.<br/>                                                                               |
| [**Инапсистемхеалсвалидатионрекуест:: Жетстрингкоррелатионид**](inapsystemhealthvalidationrequest-getstringcorrelationid-method.md) | Используется проверяющими элементами управления для получения уникального идентификатора Exchange. Затем проверяющий элемент управления должен записать эти сведения в журнал.<br/> |
| [**Инапсистемхеалсвалидатионрекуест:: Сетприватедата**](inapsystemhealthvalidationrequest-setprivatedata-method.md)                 | Позволяет Напсервер хранить сведения о состоянии.<br/>                                                           |
| [**Инапсистемхеалсвалидатионрекуест:: Сетсохреспонсе**](inapsystemhealthvalidationrequest-setsohresponse-method.md)                 | Позволяет средствам проверки задавать Сохреспонсе.<br/>                                                                  |
