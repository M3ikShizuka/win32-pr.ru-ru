# <a name="inapsystemhealthagentrequestsetsohrequest-method"></a>Метод Инапсистемхеалсажентрекуест:: Сетсохрекуест

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапсистемхеалсажентрекуест:: сетсохрекуест** используется агентами работоспособности для записи своего запроса SoH, полученного в результате вызова [**Инапсистемхеалсаженткаллбакк:: жетсохрекуест**](inapsystemhealthagentcallback-getsohrequest-method.md).

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetSoHRequest(
  [in] const SoHRequest *sohRequest,
  [in]       BOOL       cacheSohForLaterUse
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*сохрекуест* \[ окне\]
</dt> <dd>

Указатель на пакет [**сохрекуест**](/windows/win32/api/naptypes/ns-naptypes-soh) .

</dd> <dt>

*качесохфорлатерусе* \[ окне\]
</dt> <dd>

**Логическое** значение, равное **true** , если напажент должен кэшировать [**SoH**](/windows/win32/api/naptypes/ns-naptypes-soh) и **false** в противном случае.

</dd> </dl>
