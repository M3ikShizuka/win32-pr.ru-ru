# <a name="inapclientmanagement-interface"></a>Интерфейс Инапклиентманажемент

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Интерфейс **инапклиентманажемент** предоставляет методы для управления клиентами NAP.

> [!Note]  
> [**INapClientManagement2**](inapclientmanagement2.md) наследует все методы этого интерфейса, и вместо него следует использовать.

 

## <a name="members"></a>Элементы

Интерфейс **инапклиентманажемент** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Инапклиентманажемент** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **инапклиентманажемент** содержит следующие методы.



| Метод                                                                                                                | Описание                                                                   |
|:----------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------|
| [**Инапклиентманажемент:: Жетнапклиентинфо**](inapclientmanagement-getnapclientinfo.md)                               | Извлекает сведения о клиенте NAP.<br/>                          |
| [**Инапклиентманажемент:: Жетрегистереденфорцементклиентс**](inapclientmanagement-getregisteredenforcementclients.md) | Извлекает сведения о зарегистрированных клиентах принудительной установки.<br/>    |
| [**Инапклиентманажемент:: Жетрегистередсистемхеалсажентс**](inapclientmanagement-getregisteredsystemhealthagents.md) | Получение сведений о зарегистрированном SHA.<br/>                       |
| [**Инапклиентманажемент:: Жетсистемисолатионинфо**](inapclientmanagement-getsystemisolationinfo.md)                   | Извлекает сведения о состоянии изоляции клиента NAP.<br/> |
| [**Инапклиентманажемент:: Регистеренфорцементклиент**](inapclientmanagement-registerenforcementclient.md)             | Регистрирует клиент принудительного применения в системе защиты доступа к сети.<br/>               |
| [**Инапклиентманажемент:: Регистерсистемхеалсажент**](inapclientmanagement-registersystemhealthagent.md)             | Регистрирует SHA в системе защиты доступа к сети.<br/>                              |
| [**Инапклиентманажемент:: Унрегистеренфорцементклиент**](inapclientmanagement-unregisterenforcementclient.md)         | Отменяет регистрацию клиента принудительного применения в системе защиты доступа к сети.<br/>             |
| [**Инапклиентманажемент:: Унрегистерсистемхеалсажент**](inapclientmanagement-unregistersystemhealthagent.md)         | Отменяет регистрацию SHA в системе защиты доступа к сети.<br/>                            |
