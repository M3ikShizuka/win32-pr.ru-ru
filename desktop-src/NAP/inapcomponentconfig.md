# <a name="inapcomponentconfig-interface"></a>Интерфейс Инапкомпонентконфиг

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Интерфейс **инапкомпонентконфиг** предоставляет методы настройки системы защиты доступа к сети для средств проверки работоспособности системы (SHV).

> [!Note]  
> [**INapComponentConfig2**](inapcomponentconfig2.md) и [**INapComponentConfig3**](inapcomponentconfig3.md) наследуют все методы этого интерфейса и должны использоваться вместо них.

 

## <a name="members"></a>Элементы

Интерфейс **инапкомпонентконфиг** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Инапкомпонентконфиг** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **инапкомпонентконфиг** содержит следующие методы.



| Метод                                                                          | Описание                                                                          |
|:--------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------|
| [**Инапкомпонентконфиг:: config**](inapcomponentconfig-getconfig.md)         | Извлекает конфигурацию компонента SHV.<br/>                                |
| [**Инапкомпонентконфиг:: Инвокеуи**](inapcomponentconfig-invokeui.md)           | Запускает настраиваемый пользовательский интерфейс для компонента SHV.<br/>              |
| [**Инапкомпонентконфиг:: Исуисуппортед**](inapcomponentconfig-isuisupported.md) | Указывает, поддерживает ли компонент SHV настроенный пользовательский интерфейс.<br/> |
| [**Инапкомпонентконфиг:: Сетконфиг**](inapcomponentconfig-setconfig.md)         | Задает конфигурацию компонента SHV.<br/>                                     |



 

## <a name="remarks"></a>Remarks

Этот интерфейс не должен реализовываться агентами работоспособности системы (SHA) или клиентами принудительного применения карантина (кекс).
