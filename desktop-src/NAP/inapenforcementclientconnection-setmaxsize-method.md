# <a name="inapenforcementclientconnectionsetmaxsize-method"></a>Метод Инапенфорцементклиентконнектион:: Сетмакссизе

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапенфорцементклиентконнектион:: сетмакссизе** задает максимальный размер пакета SoH для этого клиента.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetMaxSize(
  [in] ProtocolMaxSize maxSize
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*MAXSIZE* \[ окне\]
</dt> <dd>

Структура [**протоколмакссизе**](nap-datatypes.md) , указывающая максимальный размер пакета SOH (в байтах).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Также могут возвращаться другие коды ошибок, относящиеся к COM.



| Код возврата                                                                                     | Описание                                                        |
|-------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> </dl>           | Операция успешно завершена.<br/>                                    |
| <dl> <dt>**Д \_ ACCESSDENIED**</dt> </dl> | Ошибка разрешений, доступ запрещен.<br/>                       |
| <dl> <dt>**Д \_ OUTOFMEMORY**</dt> </dl>  | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/> |



 

## <a name="remarks"></a>Remarks

Это значение задается клиентом принудительного применения.
