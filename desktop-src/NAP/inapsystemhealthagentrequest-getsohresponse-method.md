# <a name="inapsystemhealthagentrequestgetsohresponse-method"></a>Метод Инапсистемхеалсажентрекуест:: Жетсохреспонсе

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапсистемхеалсажентрекуест:: жетсохреспонсе** используется агентом работоспособности для получения своего большого двоичного объекта сохреспонсе, когда Напажент вызывает [**Инапсистемхеалсаженткаллбакк::P rocesssohresponse**](inapsystemhealthagentcallback-processsohresponse-method.md).

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetSoHResponse(
  [out] SoHResponse **sohResponse,
  [out] UINT8       *flags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*сохреспонсе* \[ заполняет\]
</dt> <dd>

Указатель на указатель на пакет [**сохреспонсе**](/windows/win32/api/naptypes/ns-naptypes-soh) .

</dd> <dt>

*Флаги* \[ заполняет\]
</dt> <dd>

Указатель на флаг, позволяющий выполнять исправление SHA, если установлен бит [**шафиксуп**](nap-type-constants.md) , в противном случае — исправление отключено.



| Возможные значения                                                                                                                                                          | Значение                                                                                                                                                                                                                   |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="shaFixup"></span><span id="shafixup"></span><span id="SHAFIXUP"></span><dl> <dt>**шафиксуп**</dt> </dl> | Ожидается, что SHA будет выполнять исправление на основе ответа. Если этот флаг не установлен, SHA не должен выполнять исправление, даже если [**сохреспонсе**](/windows/win32/api/naptypes/ns-naptypes-soh) указывает на неработоспособность.<br/> |



 

</dd> </dl>
