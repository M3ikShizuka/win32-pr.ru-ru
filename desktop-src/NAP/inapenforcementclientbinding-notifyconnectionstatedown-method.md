# <a name="inapenforcementclientbindingnotifyconnectionstatedown-method"></a>Метод Инапенфорцементклиентбиндинг:: Нотификоннектионстатедовн

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапенфорцементклиентбиндинг:: нотификоннектионстатедовн** используется для информирования напажент о том, что подключение к клиенту принудительного применения было прервано.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT NotifyConnectionStateDown(
  [in] INapEnforcementClientConnection *downCxn
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*довнкксн* \[ окне\]
</dt> <dd>

Указатель COM на интерфейс [**инапенфорцементклиентконнектион**](inapenforcementclientconnection.md) соединения.

</dd> </dl>

Также могут возвращаться другие коды ошибок, относящиеся к COM.



| Код возврата                                                                                             | Описание                                                        |

| <dl> <dt>**С \_ ОК**</dt> </dl>                   | Операция прошла успешно.<br/>                            |
| <dl> <dt>**Д \_ ACCESSDENIED**</dt> </dl>         | Ошибка разрешений, доступ запрещен.<br/>                       |
| <dl> <dt>**Д \_ OUTOFMEMORY**</dt> </dl>          | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/> |

## <a name="remarks"></a>Комментарии

При отключении любого из подключений, установленных клиентом принудительного применения, клиент принудительного применения должен удалить подключение из активного списка и уведомить Напажент с помощью этого метода. Как только этот вызов возвращает, объект соединения может быть освобожден и освобожден. Напажент не будет содержать ссылки на объект Connection.

В результате этого уведомления Напажент обновляет состояние NAP системы соответствующим образом.

Клиент принудительного применения должен вызвать метод [**инапенфорцементклиентбиндинг:: Initialize**](inapenforcementclientbinding-initialize-method.md) перед вызовом этого или любого другого метода интерфейса [**инапенфорцементклиентбиндинг**](inapenforcementclientbinding.md) .
