# <a name="inapservercallbackoncomplete-method"></a>Метод Инапсерверкаллбакк:: OnComplete

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапсерверкаллбакк:: OnComplete** используется SHV для сигнализации о завершении асинхронного запроса.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT OnComplete(
  [in] INapSystemHealthValidationRequest *request,
  [in] HRESULT                           errorCode
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*запрос* \[ окне\]
</dt> <dd>

Указатель на объект [**инапсистемхеалсвалидатионрекуест**](inapsystemhealthvalidationrequest.md) , представляющий запрос на проверку.

</dd> <dt>

*код ошибки* \[ окне\]
</dt> <dd>

[**Код ошибки NAP**](nap-error-constants.md) , указывающий причину, по которой не удалось выполнить проверку.

> [!Note]  
> Как правило, возвращаемое значение метода [**инапсистемхеалсвалидатионрекуест:: сетсохреспонсе**](inapsystemhealthvalidationrequest-setsohresponse-method.md) передается этому параметру. Однако, если **сетсохреспонсе** не удалось вызвать из-за сбоя повторной обработки, передается значение, возвращенное неудачной командой.

 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Также могут возвращаться другие коды ошибок, относящиеся к COM.



| Код возврата                                                                                     | Описание                                                        |
|-------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> </dl>           | Операция успешно завершена.<br/>                                    |
| <dl> <dt>**Д \_ ACCESSDENIED**</dt> </dl> | Ошибка разрешений, доступ запрещен.<br/>                       |
| <dl> <dt>**Д \_ OUTOFMEMORY**</dt> </dl>  | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/> |



 

## <a name="remarks"></a>Remarks

Проверяющие элементы управления должны возвращать \_ значение "ОК", если проверка [**сохрекуест**](/windows/win32/api/naptypes/ns-naptypes-soh) может быть завершена независимо от того, прошел ли **сохрекуест** проверку работоспособности.
