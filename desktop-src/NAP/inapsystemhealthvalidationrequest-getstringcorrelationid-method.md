# <a name="inapsystemhealthvalidationrequestgetstringcorrelationid-method"></a>Метод Инапсистемхеалсвалидатионрекуест:: Жетстрингкоррелатионид

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапсистемхеалсвалидатионрекуест:: жетстрингкоррелатионид** используется средствами проверки работоспособности системы (SHV), которые должны записывать эти сведения в журнал.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetStringCorrelationId(
  [out] StringCorrelationId **correlationId
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*correlationId* \[ заполняет\]
</dt> <dd>

Указатель на указатель на уникальный [**стрингкоррелатионид**](nap-type-constants.md) для обмена SoH.

</dd> </dl>

Также могут возвращаться другие коды ошибок, относящиеся к COM.



| Код возврата                                                                                     | Описание                                                        |
|-------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> </dl>           | Операция успешно завершена.<br/>                                    |
| <dl> <dt>**Д \_ ACCESSDENIED**</dt> </dl> | Ошибка разрешений, доступ запрещен.<br/>                       |
| <dl> <dt>**Д \_ OUTOFMEMORY**</dt> </dl>  | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/> |
