# <a name="inapcomponentconfig3deleteconfig-method"></a>INapComponentConfig3: метод:D Елетеконфиг

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **делетеконфиг** реализуется средствами проверки работоспособности системы (SHV) для предоставления способа удаления данных конфигурации для определенного идентификатора конфигурации. Идентификатор может быть использован повторно после удаления данных конфигурации.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT DeleteConfig(
   UINT32 configID
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*конфигид* 
</dt> <dd>

Значение типа, представляющее данные конфигурации для удаления.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает один из следующих кодов ошибок в зависимости от результата этой операции.



| Код возврата                                                                                                    | Описание                                                                  |
|----------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> </dl>                          | Операция прошла успешно.<br/>                                      |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>                   | *Конфигид* — 0 (значение, зарезервированное для конфигурации по умолчанию).<br/> |
| <dl> <dt>**\_ \_ \_ \_ не \_ найдена Конфигурация NAP E SHV**</dt> </dl> | Не удается найти *конфигид* .<br/>                                       |
