# <a name="inapcertrelyingpartysubscribecertbygroup-method"></a>Метод Инапцертрелингпарти:: Субскрибецертбиграуп

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **субскрибецертбиграуп** подписывается на сервер сертификатов работоспособности (HCS). Прежде чем подписаться на подписку, HCS уже должен быть зарегистрирован.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SubscribeCertByGroup(
  [in]        EnforcementEntityId  id,
  [in]  const BSTR                subscriberName,
  [in]  const  VARIANT            *reserved,
  [out]       BOOL                *certExists
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

 *идентификатор* \[ в\]
</dt> <dd>

Объект [**енфорцементентитид**](nap-datatypes.md) , содержащий идентификатор клиента принудительного применения, который подписывается на подписку.

</dd> <dt>

*subscriberName* \[ окне\]
</dt> <dd>

Имя подписчика.

> [!Note]  
> В настоящее время этот параметр используется только в целях ведения журнала.

 

</dd> <dt>

*зарезервировано* \[ окне\]
</dt> <dd>

Должен иметь **значение NULL**.

</dd> <dt>

*цертексистс* \[ заполняет\]
</dt> <dd>

Указывает, что сертификат работоспособности из этого HCS уже поддерживается Напажент и имеется в хранилище компьютера. Если **значение — true**, сертификат работоспособности уже сохраняется и находится в наличии. Если задано **значение false**, сертификат работоспособности в настоящее время не поддерживается и отсутствует.

</dd> </dl>

| Код возврата                                                                                     | Описание                                                        |
|-------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> </dl>           | Операция прошла успешно.<br/>                            |
| <dl> <dt>**Д \_ ACCESSDENIED**</dt> </dl> | Ошибка разрешений, доступ запрещен.<br/>                       |
| <dl> <dt>**Д \_ OUTOFMEMORY**</dt> </dl>  | Не удалось выполнить операцию из – за пределов системных ресурсов.<br/> |
