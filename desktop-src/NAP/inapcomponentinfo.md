# <a name="inapcomponentinfo-interface"></a>Интерфейс Инапкомпонентинфо

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Интерфейс **инапкомпонентинфо** предоставляет методы, которые должны быть реализованы подключаемыми компонентами (например, SHA и SHV), чтобы система NAP могла взаимодействовать с ними. Система защиты доступа к сети вызывает реализацию этих методов для получения статических административных данных (например, понятного имени или локализованных строк).

## <a name="members"></a>Элементы

Интерфейс **инапкомпонентинфо** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Инапкомпонентинфо** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **инапкомпонентинфо** содержит следующие методы.



| Метод                                                                                                         | Описание                                                                                                   |
|:---------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------|
| [**Инапкомпонентинфо:: Конвертерроркодетомессажеид**](inapcomponentinfo-converterrorcodetomessageid-method.md) | Используется системой защиты доступа к сети для запроса клиента работоспособности при преобразовании кода ошибки HRESULT в идентификатор сообщения.<br/> |
| [**Инапкомпонентинфо::, описание**](inapcomponentinfo-getdescription-method.md)                           | Используется системой защиты доступа к сети для получения описания клиента работоспособности.<br/>                                  |
| [**Инапкомпонентинфо:: Жетфриендлинаме**](inapcomponentinfo-getfriendlyname-method.md)                         | Используется системой защиты доступа к сети для получения понятного имени клиента работоспособности.<br/>                                |
| [**Инапкомпонентинфо:: "Icon"**](inapcomponentinfo-geticon-method.md)                                         | Используется системой защиты доступа к сети для получения значка клиента работоспособности.<br/>                                         |
| [**Инапкомпонентинфо:: Жетлокализедстринг**](inapcomponentinfo-getlocalizedstring-method.md)                   | Используется системой защиты доступа к сети для получения локализованных строк.<br/>                                                   |
| [**Инапкомпонентинфо:: Жетвендорнаме**](inapcomponentinfo-getvendorname-method.md)                             | Используется системой защиты доступа к сети для получения имени поставщика клиента работоспособности.<br/>                                  |
| [**Инапкомпонентинфо::/версия**](inapcomponentinfo-getversion-method.md)                                   | Используется системой защиты доступа к сети для получения версии клиента работоспособности.<br/>                                      |
