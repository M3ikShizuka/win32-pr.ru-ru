# <a name="inapsystemhealthagentrequestgetcachesohflag-method"></a>Метод Инапсистемхеалсажентрекуест:: Жеткачесохфлаг

> [!Note]  
> Платформа защиты доступа к сети недоступна начиная с Windows 10

 

Метод **инапсистемхеалсажентрекуест:: жеткачесохфлаг** используется только напажент.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetCacheSoHFlag(
   BOOL *cacheSohForLaterUse
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*качесохфорлатерусе* 
</dt> <dd>

**Логическое** значение, равное **true** , если напажент должен кэшировать [**SoH**](/windows/win32/api/naptypes/ns-naptypes-soh) и **false** в противном случае.

</dd> </dl>
