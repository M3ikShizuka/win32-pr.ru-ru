# <a name="platform-update-for-windows-vista"></a>обновление платформы для Windows Vista

## <a name="purpose"></a>Назначение

обновление платформы для Windows Vista и обновление платформы для Windows Server 2008 — это обновления операционной системы конечных пользователей, которые поддерживают использование выбранных технологий Windows 7 в предыдущих версиях Windows операционной системы. обновления включают набор библиотек времени выполнения, которые позволяют разработчикам приложений ориентироваться на текущие выпуски, Windows 7 и Windows server 2008 R2, а также предыдущие версии Windows Vista и Windows server 2008.

## <a name="developer-audience"></a>Аудитория разработчиков

API, поддерживаемый обновлением платформы для Windows Vista и обновления платформы для Windows Server 2008, предназначен для использования опытными разработчиками C/C++.

Рекомендуемые профиЦиенЦиес:

-   Программирование COM
-   Windows Программирование API

## <a name="run-time-requirements"></a>Требования к среде выполнения

для Windows Vista конечные пользователи получают обновление платформы для Windows Vista с Центр обновления Windows. для Windows server 2008 конечные пользователи получают обновление платформы для Windows Server 2008 с Центр обновления Windows.
