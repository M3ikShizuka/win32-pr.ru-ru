# <a name="istringcollection-methods"></a>Методы Истрингколлектион

Интерфейс [**истрингколлектион**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) определяет следующие методы.



| Метод                                         | Описание                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**Включить**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-add)           | Добавляет элемент в коллекцию.                                |
| [**Открытым**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-clear)       | Удаляет все элементы из коллекции.                  |
| [**Копировать**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-copy)         | Создает копию коллекции с глубоким чтением и записью.              |
| [**Вставляет**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-insert)     | Вставляет элемент в коллекцию в указанную позицию. |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-removeat) | Удаляет из коллекции элемент с заданным индексом.   |
