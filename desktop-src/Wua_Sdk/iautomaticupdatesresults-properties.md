# <a name="iautomaticupdatesresults-properties"></a>Свойства Иаутоматикупдатесресултс

Интерфейс [**иаутоматикупдатесресултс**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatesresults) определяет следующие свойства.



| Свойство                                                                                    | Описание                                                                                                                                             |
|---------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ластинсталлатионсукцессдате**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatesresults-get_lastinstallationsuccessdate) | Возвращает дату последнего времени и время в формате UTC, когда автоматическое обновление успешно установили обновления, даже если произошел сбой. |
| [**ластсеарчсукцессдате**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatesresults-get_lastsearchsuccessdate)             | Возвращает дату последнего времени и время в формате UTC, когда Аутоматикупдатес успешно выполнил Поиск обновлений.                                   |
