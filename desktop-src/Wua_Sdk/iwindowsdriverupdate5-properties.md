# <a name="iwindowsdriverupdate5-properties"></a>Свойства IWindowsDriverUpdate5

Интерфейс [**IWindowsDriverUpdate5**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate5) определяет следующие свойства.



| Свойство                                                     | Описание                                                                                                                                               |
|--------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Автозагрузка**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate5-get_autodownload)   | Возвращает значение [**аутодовнлоадмоде**](/windows/win32/api/wuapi/ne-wuapi-autodownloadmode) , указывающее режим автоматической загрузки обновления.                                          |
| [**Автовыбор**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate5-get_autoselection) | Возвращает значение [**аутоселектионмоде**](/windows/win32/api/wuapi/ne-wuapi-autoselectionmode) , указывающее режим автоматического выбора обновления на панели управления Центр обновления Windows. |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующих интерфейсах.

-   [**ивиндовсдриверупдате**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
-   [**IWindowsDriverUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate3)
-   [**IWindowsDriverUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate4)
