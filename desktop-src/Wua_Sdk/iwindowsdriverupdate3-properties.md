# <a name="iwindowsdriverupdate3-properties"></a>Свойства IWindowsDriverUpdate3

Интерфейс **IWindowsDriverUpdate3** определяет следующее свойство.



| Свойство                                               | Описание                                                                                                             |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**бровсеонли**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate3-get_browseonly) | Возвращает логическое значение, указывающее, может ли обновление обнаруживаться только при просмотре доступных обновлений. |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующем интерфейсе.

-   [**ивиндовсдриверупдате**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
