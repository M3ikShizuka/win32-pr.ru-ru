# <a name="isearchjob-properties"></a>Свойства Исеарчжоб

Интерфейс [**исеарчжоб**](/windows/desktop/api/Wuapi/nn-wuapi-isearchjob) определяет следующие свойства.



| Свойство                                      | Описание                                                                                                                                                 |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AsyncState**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-get_asyncstate)   | Возвращает объект состояния, относящегося к вызывающему объекту, который передается в метод [**иупдатесеарч. бегинсеарч**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) .                         |
| [**IsCompleted**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-get_iscompleted) | Возвращает логическое значение, указывающее, полностью ли обрабатывается вызов метода [**иупдатесеарч. бегинсеарч**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) . |
