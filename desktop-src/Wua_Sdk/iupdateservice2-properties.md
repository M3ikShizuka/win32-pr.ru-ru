# <a name="iupdateservice2-properties"></a>Свойства IUpdateService2

Интерфейс **IUpdateService2** определяет следующее свойство.



| Свойство                                                         | Описание                                                                                                                                                                         |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**исдефаултаусервице**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice2-get_isdefaultauservice) | Возвращает логическое значение, указывающее, зарегистрирована ли служба в автоматическое обновление и используется ли в данный момент службой автоматическое обновление в качестве службы по умолчанию. |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующем интерфейсе.

-   [**иупдатесервице**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservice)
