# <a name="iupdatesearcher3-methods"></a>Методы IUpdateSearcher3

Интерфейс [**IUpdateSearcher3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher3) определяет следующий метод.



| Метод                                              | Описание                                                                                                                                    |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SearchScope**](/previous-versions/windows/desktop/ee694850(v=vs.85)) | Возвращает и задает значение перечисления [**SearchScope**](/windows/win32/api/wuapi/ne-wuapi-searchscope) , которое указывает область обновлений, возвращаемых поиском. |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующих интерфейсах.

-   [**иупдатесеарчер**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher)
-   [**IUpdateSearcher2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher2)
