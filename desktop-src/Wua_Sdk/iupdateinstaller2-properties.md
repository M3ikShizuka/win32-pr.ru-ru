# <a name="iupdateinstaller2-properties"></a>Свойства IUpdateInstaller2

Интерфейс [**IUpdateInstaller2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller2) определяет следующее свойство.



| Свойство                                           | Описание                                                                                                                       |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**форцекуиет**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller2-get_forcequiet) | возвращает и задает логическое значение, указывающее, принудительно ли устанавливается установщик Windows для установки обновлений без взаимодействия с пользователем. |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующем интерфейсе.

-   [**иупдатеинсталлер**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller)
