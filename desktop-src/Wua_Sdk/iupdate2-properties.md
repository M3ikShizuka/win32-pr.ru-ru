# <a name="iupdate2-properties"></a>Свойства IUpdate2

Интерфейс [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2) определяет следующие свойства.



| Свойство                                          | Описание                                                                                                                                             |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**квеидс**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_cveids)                 | Возвращает коллекцию общих уязвимостей и идентификаторов (CVE), связанных с обновлением.                                                |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_ispresent)           | Возвращает логическое значение, указывающее, имеется ли на компьютере обновление.                                                                         |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_rebootrequired) | Возвращает логическое значение, указывающее, требуется ли перезагрузка системы на компьютере для завершения установки или удаления обновления. |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующем интерфейсе:

-   [**иупдате**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
