# <a name="iautomaticupdates-methods"></a>Методы Иаутоматикупдатес

Интерфейс [**иаутоматикупдатес**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) определяет следующие методы.



| Метод                                                             | Описание                                                         |
|--------------------------------------------------------------------|---------------------------------------------------------------------|
| [**детектнов**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-detectnow)                   | Начинает автоматическое обновление, если оно еще не запущено.            |
| [**енаблесервице**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-enableservice)           | Включает все компоненты, которые автоматическое обновление требуется.         |
| [**Пауза**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-pause)                           | Приостанавливает автоматическое обновление.                                          |
| [**Возобновить**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-resume)                         | Перезапускает автоматическое обновление, если оно приостановлено.                        |
| [**шовсеттингсдиалог**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-showsettingsdialog) | Отображает диалоговое окно, содержащее параметры для автоматическое обновление. |
