# <a name="iupdateidentity-properties"></a>Свойства Иупдатеидентити

Интерфейс [**иупдатеидентити**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateidentity) определяет следующие свойства.



| Свойство                                                 | Описание                                            |
|----------------------------------------------------------|--------------------------------------------------------|
| [**RevisionNumber**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateidentity-get_revisionnumber) | Возвращает номер редакции обновления.                 |
| [**UpdateID**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateidentity-get_updateid)             | Возвращает независимый от редакции идентификатор обновления. |
