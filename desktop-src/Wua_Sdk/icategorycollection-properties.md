# <a name="icategorycollection-properties"></a>Свойства Икатегориколлектион

Интерфейс [**икатегориколлектион**](/windows/desktop/api/Wuapi/nn-wuapi-icategorycollection) определяет следующие свойства.



| Свойство                                          | Описание                                                                                                                     |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-icategorycollection-get__newenum) | Возвращает интерфейс [IEnumVARIANT](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) , который можно использовать для перечисления коллекции. |
| [**Расчета**](/windows/desktop/api/Wuapi/nf-wuapi-icategorycollection-get_count)        | Получает количество элементов коллекции.                                                                                  |
| [**Элемент**](/windows/desktop/api/Wuapi/nf-wuapi-icategorycollection-get_item)          | Возвращает интерфейс [**икатегори**](/windows/desktop/api/Wuapi/nn-wuapi-icategory) из коллекции.                                                           |
