# <a name="isearchresult-properties"></a>Свойства Исеарчресулт

Интерфейс [**исеарчресулт**](/windows/desktop/api/Wuapi/nn-wuapi-isearchresult) определяет следующие свойства.



| Свойство                                               | Описание                                                                                                   |
|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_resultcode)         | Возвращает перечисление [**оператионресулткоде**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) , которое указывает результат поиска. |
| [**руткатегориес**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_rootcategories) | Возвращает коллекцию интерфейсов корневых категорий, доступных на компьютере в данный момент.             |
| [**Обновляем**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_updates)               | Возвращает коллекцию интерфейсов обновлений, которые являются результатом поиска.                                        |
| [**Предупреждения**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_warnings)             | Возвращает коллекцию предупреждений, которые появляются в результате поиска.                                                  |
