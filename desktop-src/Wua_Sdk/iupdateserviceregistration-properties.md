# <a name="iupdateserviceregistration-properties"></a>Свойства Иупдатесервицерегистратион

Интерфейс **иупдатесервицерегистратион** определяет следующие свойства.



| Свойство                                                                                      | Описание                                                                                                                                          |
|-----------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**испендингрегистратионвисау**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateserviceregistration-get_ispendingregistrationwithau) | Возвращает логическое значение, указывающее, будет ли служба также зарегистрирована в автоматическое обновление при добавлении.                                  |
| [**RegistrationState**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateserviceregistration-get_registrationstate)                     | Возвращает значение [**упдатесервицерегистратионстате**](/windows/win32/api/wuapi/ne-wuapi-updateserviceregistrationstate) , указывающее текущее состояние регистрации службы. |
| [**Служба**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateserviceregistration-get_service)                                         | Возвращает указатель на интерфейс [**IUpdateService2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservice2) . Это свойство является свойством по умолчанию.                                    |
