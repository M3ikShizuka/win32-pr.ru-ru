# <a name="iupdateexception-properties"></a>Свойства Иупдатиксцептион

Интерфейс [**иупдатиксцептион**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexception) определяет следующие свойства.



| Свойство                                    | Описание                                                |
|---------------------------------------------|------------------------------------------------------------|
| [**Контекст**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_context) | Возвращает контекст исключения.                         |
| [**HResult**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_hresult) | возвращает код **HRESULT** на основе Windows для исключения. |
| [**Message**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_message) | Возвращает сообщение, описывающее текущее исключение.       |
