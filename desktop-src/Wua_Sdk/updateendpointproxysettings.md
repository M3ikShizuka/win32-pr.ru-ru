# <a name="updateendpointproxysettings-structure"></a>Структура Упдатиндпоинтпроксисеттингс

Структура **упдатиндпоинтпроксисеттингс** определяет параметры прокси-сервера, используемые при запросе маркера.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct tagUpdateEndpointProxySettings {
  BSTR szProxyAddr;
  BSTR szBypassList;
  BSTR szUserName;
  BSTR szPassword;
} UpdateEndpointProxySettings;
```



## <a name="members"></a>Члены

<dl> <dt>

**сзпроксяддр**
</dt> <dd>

DNS-имя или IP-адрес прокси-сервера для использования (например, "proxy.somecorp.com" или "192.168.0.4") или пустая строка, если прокси-сервер не используется.

</dd> <dt>

**сзбипасслист**
</dt> <dd>

Список адресов узлов, которые должны обходить прокси-сервер, или пустую строку, если все адреса узлов должны использовать прокси-сервер.

WUA не использует эти данные, если **сзпроксяддр** пуст.

</dd> <dt>

**сзусернаме**
</dt> <dd>

Имя пользователя, используемое для проверки подлинности на прокси-сервере, или пустая строка, если проверка подлинности не требуется.

WUA не использует эти данные, если **сзпроксяддр** пуст.

</dd> <dt>

**сзпассворд**
</dt> <dd>

Пароль, используемый для проверки подлинности на прокси-сервере, или пустая строка, если проверка подлинности не требуется.

WUA не использует эти данные, если **сзпроксяддр** пуст.

</dd> </dl>
