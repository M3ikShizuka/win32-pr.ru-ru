# <a name="iupdateservicemanager2-methods"></a>Методы IUpdateServiceManager2

Интерфейс [**IUpdateServiceManager2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager2) определяет следующие методы.



| Метод                                                                                      | Описание                                                                                                                                                                                |
|---------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddService2**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicemanager2-addservice2)                           | Регистрирует службу с помощью WUA, не требуя CAB-файла авторизации и возвращает указатель на интерфейс [**иупдатесервицерегистратион**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicecollection) . |
| [**куерисервицерегистратион**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicemanager2-queryserviceregistration) | Возвращает указатель на интерфейс [**иупдатесервицерегистратион**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateserviceregistration) .                                                                                        |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующем интерфейсе.

-   [**иупдатесервицеманажер**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager)
