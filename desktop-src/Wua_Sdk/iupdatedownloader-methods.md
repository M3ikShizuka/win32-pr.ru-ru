# <a name="iupdatedownloader-methods"></a>Методы Иупдатедовнлоадер

Интерфейс [**иупдатедовнлоадер**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloader) определяет следующие методы.



| Метод                                                   | Описание                                                                                |
|----------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**BeginDownload**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-begindownload) | Запускает асинхронную загрузку файлов содержимого, связанных с обновлениями. |
| [**Загрузка**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-download)           | Запускает синхронную загрузку файлов содержимого, связанных с обновлениями.   |
| [**Метод**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-enddownload)     | Завершает асинхронную загрузку.                                                        |
