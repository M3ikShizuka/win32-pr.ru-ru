# <a name="idownloadchangedcallbackargs-properties"></a>Свойства Идовнлоадчанжедкаллбаккаргс

Интерфейс [**идовнлоадпрогрессчанжедкаллбаккаргс**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadprogresschangedcallbackargs) определяет следующее свойство.



| Свойство                                                          | Описание                                                                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [**Ход выполнения**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadprogresschangedcallbackargs-get_progress) | Возвращает интерфейс, содержащий ход выполнения асинхронной загрузки на момент, когда был сделан обратный вызов. |
