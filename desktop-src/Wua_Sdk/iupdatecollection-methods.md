# <a name="iupdatecollection-methods"></a>Методы Иупдатеколлектион

Интерфейс [**иупдатеколлектион**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatecollection) определяет следующие методы.



| Метод                                         | Описание                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**Включить**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-add)           | Добавляет элемент в коллекцию.                                |
| [**Clear**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-clear)       | Удаляет все элементы из коллекции.                  |
| [**Копировать**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-copy)         | Создает поверхностную копию коллекции, доступную для чтения и записи.           |
| [**Вставить**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-insert)     | Вставляет элемент в коллекцию в указанную позицию. |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-removeat) | Удаляет из коллекции элемент с заданным индексом.   |
