# <a name="iupdate5-properties"></a>Свойства IUpdate5

Интерфейс [**IUpdate5**](/windows/win32/api/wuapi/nn-wuapi-iupdate5) определяет следующие свойства.



| Свойство                                        | Описание                                                                                            |
|-------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**Автозагрузка**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate5-get_autodownload)   | Возвращает значение, указывающее режим автоматической загрузки обновления.                                         |
| [**Автовыбор**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate5-get_autoselection) | возвращает значение, указывающее режим автоматического выбора обновления на панели управления Центр обновления Windows. |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующих интерфейсах:

-   [**IUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate4)
-   [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3)
-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**иупдате**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
