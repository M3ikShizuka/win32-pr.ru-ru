# <a name="iupdatedownloadcontent2-properties"></a>Свойства IUpdateDownloadContent2

Интерфейс [**IUpdateDownloadContent2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadcontent2) определяет следующее свойство.



| Свойство                                                                             | Описание                                                                                     |
|--------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**исделтакомпресседконтент**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontent2-get_isdeltacompressedcontent) | Возвращает логическое значение, указывающее, является ли обновление двоичным обновлением или полным обновлением файла. |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующем интерфейсе.

-   [**иупдатедовнлоадконтент**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatecollection)
