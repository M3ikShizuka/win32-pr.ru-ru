| Свойство                                                   | Описание                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**сервицеенаблед**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_serviceenabled) | Возвращает логическое значение, указывающее, доступны ли все компоненты, которые автоматическое обновление необходимы. |
| [**Параметры**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-get_settings)             | Возвращает параметры конфигурации для автоматическое обновление.                                                        |
