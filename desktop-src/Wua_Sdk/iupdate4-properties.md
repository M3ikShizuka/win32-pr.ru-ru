# <a name="iupdate4-properties"></a>Свойства IUpdate4

Интерфейс [**IUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate4) определяет следующее свойство.



| Свойство                            | Описание                                                            |
|-------------------------------------|------------------------------------------------------------------------|
| [**Изучение**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate4-get_peruser) | Возвращает логическое значение, указывающее, является ли это обновлением для каждого пользователя. |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующих интерфейсах:

-   [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3)
-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**иупдате**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
