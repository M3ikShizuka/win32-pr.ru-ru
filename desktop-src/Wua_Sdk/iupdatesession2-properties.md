# <a name="iupdatesession2-properties"></a>Свойства IUpdateSession2

Интерфейс [**IUpdateSession2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession2) определяет следующее свойство.



| Свойство                                         | Описание                                                                   |
|--------------------------------------------------|-------------------------------------------------------------------------------|
| [**UserLocale**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession2-get_userlocale) | Возвращает и задает предпочтительный языковой стандарт, для которого извлекаются сведения об обновлении. |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующем интерфейсе.

-   [**иупдатесессион**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession)
