# <a name="istringcollection-properties"></a>Свойства Истрингколлектион

Интерфейс [**истрингколлектион**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) определяет следующие свойства.



| Свойство                                        | Описание                                                                                                                     |
|-------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get__newenum) | Возвращает интерфейс [IEnumVARIANT](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) , который можно использовать для перечисления коллекции. |
| [**Count**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_count)        | Получает количество элементов коллекции.                                                                                  |
| [**Компонент**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_item)          | Возвращает или задает строку в коллекции.                                                                                        |
| [**Доступно**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_readonly)  | Возвращает логическое значение, указывающее, доступна ли коллекция только для чтения.                                                        |
