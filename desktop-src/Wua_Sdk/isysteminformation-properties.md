# <a name="isysteminformation-properties"></a>Свойства Исистеминформатион

Интерфейс [**исистеминформатион**](/windows/desktop/api/Wuapi/nn-wuapi-isysteminformation) определяет следующие свойства.



| Свойство                                                                    | Описание                                                                                                                                     |
|-----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**оемхардваресуппортлинк**](/windows/desktop/api/Wuapi/nf-wuapi-isysteminformation-get_oemhardwaresupportlink) | Получает гиперссылку на сведения о технической поддержке для оборудования OEM.                                                                             |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-isysteminformation-get_rebootrequired)                 | Возвращает логическое значение, указывающее, требуется ли перезагрузка системы для завершения установки или удаления одного или нескольких обновлений. |
