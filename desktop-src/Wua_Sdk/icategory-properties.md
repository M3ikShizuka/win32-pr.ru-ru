# <a name="icategory-properties"></a>Свойства Икатегори

Интерфейс [**икатегори**](/windows/desktop/api/Wuapi/nn-wuapi-icategory) определяет следующие свойства.



| Свойство                                     | Описание                                                                                       |
|----------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**CategoryID**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_categoryid)   | Возвращает идентификатор категории.                                                              |
| [**Дети**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_children)       | Возвращает коллекцию интерфейсов, содержащую дочерние категории этой категории.                 |
| [**Описание**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_description) | Возвращает описание категории.                                                             |
| [**Образ**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_image)             | Возвращает интерфейс, содержащий сведения об изображении, связанном с категорией. |
| [**Имя**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_name)               | Возвращает локализованное имя категории.                                                          |
| [**Заказ**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_order)             | Возвращает рекомендуемый порядок вывода этой категории среди категорий того же уровня.                 |
| [**Parent**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_parent)           | Возвращает интерфейс, описывающий родительскую категорию этой категории.                            |
| [**Тип**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_type)               | Возвращает тип категории.                                                                    |
| [**Обновления**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_updates)         | Возвращает интерфейс, содержащий коллекцию обновлений, которые непосредственно принадлежат к категории.  |
