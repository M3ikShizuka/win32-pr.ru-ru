# <a name="iupdatesearcher-methods"></a>Методы Иупдатесеарчер

Интерфейс [**иупдатесеарчер**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher) определяет следующие методы.



| Метод                                                              | Описание                                                                                                               |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**бегинсеарч**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch)                   | Начинает выполнение асинхронного поиска обновлений. Поиск использует параметры поиска, настроенные в настоящее время. |
| [**ендсеарч**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-endsearch)                       | Завершает асинхронный поиск обновлений.                                                                             |
| [**EscapeString**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-escapestring)                 | Преобразует строку в строку, которая может использоваться как литеральное значение в строке условий поиска.                          |
| [**жеттоталхисторикаунт**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-gettotalhistorycount) | Возвращает число событий обновления на компьютере.                                                                      |
| [**куерихистори**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-queryhistory)                | Синхронно запрашивает у компьютера журнал событий обновления.                                                      |
| [**Поиск**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-search)                             | Выполняет синхронный поиск обновлений. Поиск использует параметры поиска, настроенные в настоящее время.              |
