# <a name="iwindowsdriverupdateentry-properties"></a>Свойства Ивиндовсдриверупдатинтри

Интерфейс [**ивиндовсдриверупдатинтри**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdateentry) определяет следующие свойства.



| Свойство                                                                     | Описание                                                                                                          |
|------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**девицепроблемнумбер**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_deviceproblemnumber) | Возвращает номер проблемы устройства, которое соответствует обновлению драйвера Windows.                                    |
| [**девицестатус**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_devicestatus)               | Возвращает состояние устройства, которое соответствует обновлению драйвера Windows.                                            |
| [**дриверкласс**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverclass)                 | Возвращает класс обновления драйвера Windows.                                                                         |
| [**дриверхардвареид**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverhardwareid)       | Возвращает оборудование или совместимый идентификатор, который должен совпадать с обновлением драйвера Windows для установки. |
| [**дривермануфактурер**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermanufacturer)   | Получает инвариантное от языка имя производителя обновления драйвера Windows.                                   |
| [**дривермодел**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermodel)                 | Возвращает неизменяемое имя модели языка устройства, для которого предназначено обновление драйвера Windows.                |
| [**дриверпровидер**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverprovider)           | Возвращает инвариантное от языка имя поставщика обновления драйвера Windows.                                       |
| [**дривервердате**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driververdate)             | Возвращает дату версии драйвера для обновления драйвера Windows.                                                           |
