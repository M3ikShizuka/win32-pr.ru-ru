# <a name="iautomaticupdatessettings3-properties"></a>Свойства IAutomaticUpdatesSettings3

Интерфейс [**IAutomaticUpdatesSettings3**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings3) определяет следующие свойства.



| Свойство                                                                                  | Описание                                                                                                                                     |
|-------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**феатуредупдатесенаблед**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_featuredupdatesenabled)       | Возвращает и задает логическое значение, указывающее, следует ли отображать уведомления для популярных обновлений.                                             |
| [**нонадминистраторселеватед**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings3-get_nonadministratorselevated) | Возвращает и задает логическое значение, указывающее, могут ли неадминистраторы выполнять некоторые действия, связанные с обновлением, без утверждения администратором. |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующих интерфейсах.

-   [**иаутоматикупдатессеттингс**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
-   [**IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2)
