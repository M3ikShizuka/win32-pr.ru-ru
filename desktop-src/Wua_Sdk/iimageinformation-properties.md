# <a name="iimageinformation-properties"></a>Свойства Иимажеинформатион

Интерфейс [**иимажеинформатион**](/windows/desktop/api/Wuapi/nn-wuapi-iimageinformation) определяет следующие свойства.



| Свойство                                     | Описание                              |
|----------------------------------------------|------------------------------------------|
| [**алттекст**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_alttext) | Возвращает альтернативный текст для изображения.   |
| [**Высота**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_height)   | Возвращает высоту изображения в пикселях. |
| [**Источник**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_source)   | Возвращает исходное расположение изображения.   |
| [**Ширина**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_width)     | Возвращает ширину изображения в пикселях.  |
