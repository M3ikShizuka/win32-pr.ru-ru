# <a name="iwebproxy-methods"></a>Методы IWebProxy

Интерфейс [**IWebProxy**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) определяет следующие методы.



| Метод                                                                         | Описание                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**промптфоркредентиалс**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentials)                 | Запрашивает у пользователя пароль, используемый для проверки подлинности прокси.  |
| [**промптфоркредентиалсфромхвнд**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentialsfromhwnd) | Запрашивает у пользователя пароль, используемый для проверки подлинности прокси.  |
| [**SetPassword**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-setpassword)                                   | Задает пароль для отправки на прокси-сервер для проверки подлинности. |
