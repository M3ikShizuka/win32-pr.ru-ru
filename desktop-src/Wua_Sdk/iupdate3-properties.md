# <a name="iupdate3-properties"></a>Свойства IUpdate3

Интерфейс [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3) определяет следующее свойство.



| Свойство                                  | Описание:                                                                                                             |
|-------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**бровсеонли**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate3-get_browseonly) | Возвращает логическое значение, указывающее, может ли обновление обнаруживаться только при просмотре доступных обновлений. |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующих интерфейсах:

-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**иупдате**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
