# <a name="iinstallationprogresschangedcallbackargs-properties"></a>Свойства Иинсталлатионпрогрессчанжедкаллбаккаргс

Интерфейс [**иинсталлатионпрогрессчанжедкаллбаккаргс**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogresschangedcallbackargs) определяет следующее свойство.



| Свойство                                                              | Описание:                                                                                                                        |
|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**Ход выполнения**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogresschangedcallbackargs-get_progress) | Возвращает интерфейс, который содержит ход выполнения асинхронной установки или удаления во время обратного вызова. |
