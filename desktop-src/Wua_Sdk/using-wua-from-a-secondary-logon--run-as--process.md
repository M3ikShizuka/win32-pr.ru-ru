# <a name="using-wua-from-a-secondary-logon-run-as-process"></a>Использование агента WUA из процесса вторичного входа в систему (Запуск от имени)

обновления, требующие взаимодействия с пользователем, не могут быть установлены приложениями Центр обновления Windows Agent (wua), если приложения wua выполняются в контексте службы вторичного входа в систему.

Если процесс, вызывающий WUA, выполняется в контексте службы запуска от имени или службы вторичного входа в систему, попытка установить обновление, требующее взаимодействия с пользователем, завершится неудачей и возвратит ошибку **Wu \_ E \_ без \_ интерактивного \_ пользователя** .

в Microsoft Windows можно запускать программы от имени пользователя, который отличается от пользователя, выполнившего вход в систему. Для этого необходимо запустить службу вторичного входа в систему.
