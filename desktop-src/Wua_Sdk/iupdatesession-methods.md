# <a name="iupdatesession-methods"></a>Методы Иупдатесессион

Интерфейс [**иупдатесессион**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession) определяет следующие методы.



| Метод                                                                  | Описание                                                                           |
|-------------------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**креатеупдатедовнлоадер**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-createupdatedownloader) | Возвращает интерфейс [**иупдатедовнлоадер**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloader) для этого сеанса. |
| [**креатеупдатеинсталлер**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-createupdateinstaller)   | Возвращает интерфейс [**иупдатеинсталлер**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller) для этого сеанса.   |
| [**креатеупдатесеарчер**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession-createupdatesearcher)     | Возвращает интерфейс [**иупдатесеарчер**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher) для этого сеанса.     |
