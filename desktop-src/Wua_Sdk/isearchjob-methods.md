# <a name="isearchjob-methods"></a>Методы Исеарчжоб

Интерфейс [**исеарчжоб**](/windows/desktop/api/Wuapi/nn-wuapi-isearchjob) определяет следующие методы.



| Метод                                          | Описание                                                                          |
|-------------------------------------------------|--------------------------------------------------------------------------------------|
| [**Чистку**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-cleanup)           | Ожидает завершения асинхронной операции, а затем освобождает все обратные вызовы. |
| [**рекуестаборт**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-requestabort) | Выполняет запрос на отмену асинхронного поиска.                                    |
