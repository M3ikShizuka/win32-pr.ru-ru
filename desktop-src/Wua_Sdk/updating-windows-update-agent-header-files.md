# <a name="updating-windows-update-agent-header-files"></a>Обновление файлов заголовков агента Центр обновления Windows

Ранее API-интерфейсы агента Центр обновления Windows были загружены отдельно. Теперь они включены в состав Windows SDK. Чтобы скачать Windows SDK, перейдите на [эту страницу](https://developer.microsoft.com/windows/downloads/windows-10-sdk).

Дополнительные сведения об обновлении WUA см. в разделе [обновление агента Центр обновления Windows](updating-the-windows-update-agent.md).
