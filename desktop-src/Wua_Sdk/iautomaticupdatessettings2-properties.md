# <a name="iautomaticupdatessettings2-properties"></a>Свойства IAutomaticUpdatesSettings2

Интерфейс [**IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2) определяет следующее свойство.



| Свойство                                                                                  | Описание                                                                                                                                                         |
|-------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**инклудерекоммендедупдатес**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings2-get_includerecommendedupdates) | Возвращает и задает логическое значение, указывающее, следует ли включать необязательные или Рекомендуемые обновления при поиске обновлений и установки обновлений. |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующем интерфейсе.

-   [**иаутоматикупдатессеттингс**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
