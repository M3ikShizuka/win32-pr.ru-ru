# <a name="iwindowsdriverupdate-properties"></a>Свойства Ивиндовсдриверупдате

Интерфейс [**ивиндовсдриверупдате**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) определяет следующие свойства.



| Свойство                                                                                                    | Описание                                                                                           |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [**Девицепроблемнумбер**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_deviceproblemnumber)[**адрес**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_address) | возвращает номер проблемы устройства, которое соответствует обновлению драйвера Windows.                     |
| [**девицестатус**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_devicestatus)                                                   | возвращает состояние устройства, соответствующее обновлению драйвера Windows.                             |
| [**дриверкласс**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverclass)                                                     | возвращает класс обновления Windows драйвера.                                                          |
| [**дриверхардвареид**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverhardwareid)                                           | возвращает идентификатор оборудования или совместимый идентификатор, которому должен соответствовать Windows обновление драйвера для установки.    |
| [**дривермануфактурер**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermanufacturer)                                       | возвращает инвариантное от языка имя производителя Windows обновления драйвера.                    |
| [**дривермодел**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermodel)                                                     | возвращает имя неизменяемой модели языка устройства, для которого предназначено обновление драйвера Windows. |
| [**дриверпровидер**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverprovider)                                               | возвращает инвариантное от языка имя поставщика Windows обновления драйвера.                        |
| [**дривервердате**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driververdate)                                                 | возвращает дату версии драйвера Windows обновления драйвера.                                            |
