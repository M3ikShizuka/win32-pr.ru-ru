# <a name="iwindowsdriverupdate-properties"></a>Свойства Ивиндовсдриверупдате

Интерфейс [**ивиндовсдриверупдате**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) определяет следующие свойства.



| Свойство                                                                                                    | Описание                                                                                           |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [**Девицепроблемнумбер**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_deviceproblemnumber)[**адрес**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_address) | Возвращает номер проблемы устройства, которое соответствует обновлению драйвера Windows.                     |
| [**девицестатус**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_devicestatus)                                                   | Возвращает состояние устройства, которое соответствует обновлению драйвера Windows.                             |
| [**дриверкласс**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverclass)                                                     | Возвращает класс обновления драйвера Windows.                                                          |
| [**дриверхардвареид**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverhardwareid)                                           | Возвращает идентификатор оборудования или совместимый идентификатор, которому должен соответствовать обновление драйвера Windows для установки.    |
| [**дривермануфактурер**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermanufacturer)                                       | Получает инвариантное от языка имя производителя обновления драйвера Windows.                    |
| [**дривермодел**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermodel)                                                     | Возвращает неизменяемое имя модели языка устройства, для которого предназначено обновление драйвера Windows. |
| [**дриверпровидер**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverprovider)                                               | Возвращает инвариантное от языка имя поставщика обновления драйвера Windows.                        |
| [**дривервердате**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driververdate)                                                 | Возвращает дату версии драйвера для обновления драйвера Windows.                                            |
