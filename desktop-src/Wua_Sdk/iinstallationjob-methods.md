# <a name="iinstallationjob-methods"></a>Методы Иинсталлатионжоб

Интерфейс [**иинсталлатионжоб**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationjob) определяет следующие методы.



| Метод                                                | Описание                                                                                                                                           |
|-------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Чистку**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationjob-cleanup)           | Ожидает завершения асинхронной операции, а затем освобождает все обратные вызовы.                                                              |
| [**GetProgress**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationjob-getprogress)   | Возвращает интерфейс [**иинсталлатионпрогресс**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogress) , описывающий текущий ход выполнения установки или удаления. |
| [**рекуестаборт**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationjob-requestabort) | Выполняет запрос на отмену установки или удаления.                                                                                         |
