# <a name="iwindowsdriverupdate4-properties"></a>Свойства IWindowsDriverUpdate4

Интерфейс [**IWindowsDriverUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate4) определяет следующие свойства.



| Свойство                                                                               | Описание                                                                 |
|----------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**Изучение**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate4-get_peruser)                                       | Возвращает логическое значение, указывающее, является ли обновление обновлением для каждого пользователя. |
| [**виндовсдриверупдатинтриес**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate4-get_windowsdriverupdateentries) | Возвращает записи обновления драйверов, применимые к обновлению.          |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующем интерфейсе.

-   [**ивиндовсдриверупдате**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
-   [**IWindowsDriverUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate3)
