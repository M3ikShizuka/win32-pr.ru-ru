# <a name="iwindowsdriverupdate2-properties"></a>Свойства IWindowsDriverUpdate2

Интерфейс [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) определяет следующие свойства.



| Свойство                                                       | Описание                                                                                                             |
|----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**квеидс**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_cveids)                 | Содержит коллекцию стандартных идентификаторов уязвимостей и раскрытия (CVE), связанных с обновлением. |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_ispresent)           | Возвращает логическое значение, указывающее, установлено ли обновление на компьютере.                                     |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_rebootrequired) | Возвращает логическое значение, указывающее, должен ли компьютер быть перезагружен после установки или удаления обновления.    |



 

Сведения о членах, наследуемых этим интерфейсом, см. в следующем интерфейсе.

-   [**ивиндовсдриверупдате**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
