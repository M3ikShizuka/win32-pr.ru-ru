# <a name="iupdateinstaller-methods"></a>Методы Иупдатеинсталлер

Интерфейс [**иупдатеинсталлер**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller) определяет следующие методы.



| Метод                                                    | Описание                                                                          |
|-----------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**бегининсталл**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-begininstall)     | Запускает асинхронную установку обновлений.                                  |
| [**бегинунинсталл**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-beginuninstall) | Запускает асинхронное удаление обновлений.                                |
| [**ендинсталл**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-endinstall)         | Завершает асинхронную установку обновлений.                               |
| [**ендунинсталл**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-enduninstall)     | Завершает асинхронное удаление обновлений.                             |
| [**Взнос**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-install)               | Запускает синхронную установку обновлений.                                    |
| [**рунвизард**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-runwizard)           | Запуск мастера, направляющего локального пользователя на действия по установке обновлений. |
| [**Удалить**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-uninstall)           | Запускает синхронное удаление обновлений.                                  |
