# <a name="idownloadjob-methods"></a>Методы Идовнлоаджоб

Интерфейс [**идовнлоаджоб**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadjob) определяет следующие методы.



| Метод                                            | Описание                                                                                                            |
|---------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**Чистку**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadjob-cleanup)           | Ожидает завершения асинхронной операции и освобождает все обратные вызовы.                                        |
| [**GetProgress**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadjob-getprogress)   | Возвращает интерфейс [**идовнлоадпрогресс**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadprogress) , который описывает текущий ход выполнения загрузки. |
| [**рекуестаборт**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadjob-requestabort) | Выполняет запрос на завершение асинхронной загрузки.                                                                       |
