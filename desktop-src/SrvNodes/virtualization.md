# <a name="virtualization"></a>Виртуализация

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[Microsoft Virtual Server](/previous-versions/windows/desktop/msvs/microsoft-virtual-server-portal)
</dt> <dd>

Microsoft Virtual Server 2005 позволяет создавать отдельные виртуальные машины поверх рабочего стола Microsoft Windows, где можно установить практически любую операционную систему архитектуры Intel. Каждая виртуальная машина эмулирует полную аппаратную систему — от процессора до сетевой карты — в автономной изолированной программной среде, обеспечивая одновременную работу несовместимых систем.

</dd> <dt>

[Windows Virtual PC](/windows/desktop/VPC/virtual-pc-portal)
</dt> <dd>

Windows Virtual PC — это новейшая технология виртуализации Майкрософт; Она позволяет запускать множество приложений для повышения производительности в виртуальной среде Windows с одним щелчком непосредственно из Windows 7. Вы можете создать отдельные виртуальные машины поверх рабочего стола Windows 7. Каждая виртуальная машина эмулирует полную аппаратную систему — от процессора до сетевой карты — в автономной изолированной программной среде, обеспечивая одновременную работу несовместимых систем.

</dd> </dl>
