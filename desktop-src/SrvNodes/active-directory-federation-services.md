# <a name="active-directory-federation-services"></a>Службы федерации Active Directory (AD FS)

Active Directory служба федерации (AD FS) — это служба проверки подлинности и авторизации, которая работает на сервере Windows. AD FS предоставляет пользователям доступ с единым входом между средами предприятия и веб-приложениями.
