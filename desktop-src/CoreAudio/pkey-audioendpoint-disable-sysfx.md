# <a name="pkey_audioendpoint_disable_sysfx"></a>PKEY \_ аудиоендпоинт \_ Отключить \_ сисфкс

Свойство **PKEY \_ аудиоендпоинт \_ disable \_ сисфкс** указывает, включены ли системные эффекты в потоке общего режима, который передается на устройство конечной точки аудио или с него.

Системные эффекты реализуются как объекты обработки звука (APOs), которые можно вставить в аудиопоток. APOs — это программные модули, выполняющие функции обработки звука, такие как управление томами и преобразование форматов. Отключение системных эффектов для устройства конечной точки позволяет соответствующему потоку проходить через узел «APOs без изменений».

дополнительные сведения о системных эффектах в Windows Vista см. в технических документах под названием "пользовательские звуковые эффекты в Windows Vista" и "повторное использование Windows Vista audio system effects" на странице [технологии аудио устройства для Windows](https://www.microsoft.com/whdc/device/audio/default.mspx) веб-сайта.

Это свойство применимо только к локальным результатам и глобальным эффектам, которые были установлены с помощью INF-файла, установившего звуковой адаптер, к которому подключено устройство конечной точки. Кроме того, это свойство влияет только на целевое устройство конечной точки — оно не влияет на системные эффекты для других устройств конечных точек, даже если они подключаются к одному адаптеру.

Элементу **VT** структуры **пропвариант** присвоено значение **VT \_ UI4**.

Элемент **улвал** структуры **пропвариант** устанавливается в значение **Endpoint \_ сисфкс \_ включено** , если системные эффекты включены, или в **конечную точку \_ сисфкс \_ отключены** , если они отключены.
