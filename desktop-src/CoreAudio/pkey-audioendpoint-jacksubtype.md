# <a name="pkey_audioendpoint_jacksubtype"></a>PKEY \_ аудиоендпоинт \_ жакксубтипе

Свойство **PKEY \_ аудиоендпоинт \_ жакксубтипе** содержит идентификатор GUID категории вывода для устройства конечной точки аудио. В файле заголовка Ксмедиа. h определяются идентификаторы GUID. каждый GUID указывает тип соединения. Эти идентификаторы GUID также имеют связанные категории закрепления. Например, файл заголовков Ксмедиа. h определяет идентификатор **GUID \_ кснодетипе \_ дисплайпорт** для порта дисплея, который подключается с помощью ПИН-кода KS, определенного GUID **пиннаме \_ дисплайпорт \_ out**.

дополнительные сведения см. в описании свойств категории pin-кодов в документации по Windows DDK.

Элементу **VT** структуры **пропвариант** присвоено значение **VT \_ LPWSTR**.

Элемент **пвсзвал** структуры **пропвариант** указывает на строку расширенных символов, заканчивающуюся нулем и содержащую GUID категории.
