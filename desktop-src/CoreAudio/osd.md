# <a name="osd"></a>ОПЕРАЦИОННОЙ

В этом примере используются основные API аудио, чтобы реализовать экран на экране, в котором показаны изменения тома в потоке вывода, который воспроизводится через устройство конечной точки отрисовки звука по умолчанию. Экран отображается, когда пользователь корректирует уровень громкости в программе управления громкостью Windows Sndvol.exe и исчезает после того, как уровень тома остается неизменным в течение короткого периода.

## <a name="building-the-sample"></a>Построение образца

1.  Откройте командную оболочку для Windows SDK и перейдите в каталог примеров OSD.
2.  Выполните команду "Start OSD. sln" в каталоге OSD, чтобы открыть проект OSD в окне Visual Studio.
3.  В окне выберите конфигурацию решения для **отладки** или **выпуска** , выберите меню **Сборка** в строке меню и выберите параметр **Сборка** . Если не открыть Visual Studio из командной оболочки для пакета SDK, Visual Studio не сможет получить доступ к среде сборки пакета SDK. В этом случае выборка не будет построена, если вы явно не настроили переменную среды Мссдк, которая используется в файле проекта, OSD. vcproj.

## <a name="running-the-sample"></a>Запуск примера

1.  Запустите исполняемый файл OSD, OSD.exe, в Windows Vista или более поздней версии. Обратите внимание, что для приложения не отображается значок или окно панели задач, но можно увидеть, что процесс выполняется с помощью TaskMgr.exe.
2.  Запустите sndvol.exe, чтобы изменить или выключить том, или измените громкость с помощью клавиш управления клавиатуры или элемента управления HID. Отобразится пользовательский интерфейс OSD.
3.  Чтобы выйти из приложения, запустите TaskMgr.exe, выделите OSD.exe процесс и нажмите кнопку **завершить процесс**.
