# <a name="spatial_audio_xxx-constants"></a>\_ \_ Константы пространственного аудио XXX

\_ \_ Константы пространственного аудио XXX определяют значения, связанные с функциями пространственного звука.



| Константа/значение                                                                                                                                                                                                                                                                                       | Описание                                                                                                                                                                                                                                                                     |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="SPATIAL_AUDIO_POSITION"></span><span id="spatial_audio_position"></span><dl> <dt>**Пространственный \_ \_Расположение аудио**</dt> <dt>200</dt> </dl>                                                   | Стандартная определяемая корпорацией Майкрософт команда пространственных метаданных, которая представляет позицию прослушивателя в стандартной модели, используемой [**испатиалаудиоклиент**](/windows/desktop/api/spatialaudioclient/nn-spatialaudioclient-ispatialaudioclient) , где заголовок прослушивателя находится в координатах (0, 0, 0), определенном в метрах.<br/> |
| <span id="SPATIAL_AUDIO_POSITION_BYTE_COUNT"></span><span id="spatial_audio_position_byte_count"></span><dl> <dt>**Пространственный \_ \_ \_ \_ Число байтов в байтах для позиций**</dt> <dt>(float) \* 3</dt> </dl> | Указывает размер значения **пространственного расположения для \_ звука \_** .<br/>                                                                                                                                                                                                        |
| <span id="SPATIAL_AUDIO_STANDARD_COMMANDS_START"></span><span id="spatial_audio_standard_commands_start"></span><dl> <dt>**Пространственный \_ \_Стандартные \_ команды AUDIO \_ Start**</dt> <dt>200</dt> </dl>    | Указывает начало диапазона команд зарезервированных пространственных метаданных (Майкрософт). Команды пользовательских метаданных ограничены идентификаторами команд 0-199. Команды 200-255 зарезервированы для использования корпорацией Майкрософт.<br/>                                                           |
