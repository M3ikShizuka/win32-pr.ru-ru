# <a name="core-audio-constants"></a>Константы аудиоподсистемы Core

В этом разделе описываются константы, используемые основными API-интерфейсами аудиоподсистемы в Windows Vista и более поздних версиях.

| Константа                                                                            | Описание                                                                         |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**АУДКЛНТ \_ сессионфлагс \_ xxx**](audclnt-sessionflags-xxx-constants.md)<br/> | Флаги для сеанса аудио, который открывается во время инициализации потока.<br/> |
| [**АУДКЛНТ \_ стреамфлагс \_ xxx**](audclnt-streamflags-xxx-constants.md)              | Флаги инициализации для аудиопотока.                                           |
| [**\_состояние устройства \_ xxx**](device-state-xxx-constants.md)                            | Флаги состояния для устройства конечной точки аудио.                                           |
| [**Поддержка оборудования КОНЕЧных точек \_ \_ \_ xxx**](endpoint-hardware-support-xxx-constants.md) | Флаги поддержки оборудования для устройства конечной точки аудио.                                |
