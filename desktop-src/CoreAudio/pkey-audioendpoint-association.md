# <a name="pkey_audioendpoint_association"></a>\_Связь АУДИОЕНДПОИНТ \_ PKEY

Свойство **\_ \_ взаимосвязи аудиоендпоинт "PKEY** " связывает категорию закрепления "потоковая передача" (KS) с устройством конечной точки аудио. INF-файл, устанавливающий звуковой адаптер, назначает категорию закрепления каждому ПИН-коду KS в адаптере. ПИН-код KS на устройстве адаптера представляет точку, в которую поток аудио попадает или покидает устройство. Категория закрепления — это идентификатор GUID, указывающий тип функции, выполняемой ПИН-кодом KS. Например, файл заголовков Ксмедиа. h определяет PIN-код категории GUID КСНОДЕТИПЕ \_ Microphone для указания ПИН-кода KS, который подключается к микрофону, а кснодетипе \_ наушники — для указания ПИН-кода KS, который подключается к наушникам. Дополнительные сведения см. в описании свойств категории ПИН-кодов в документации по Windows DDK.

Элементу **VT** структуры **пропвариант** присвоено значение VT \_ LPWSTR.

Элемент **пвсзвал** структуры **пропвариант** указывает на строку расширенных символов, заканчивающуюся НУЛЕМ, которая содержит GUID категории ПИН-кода в KS.
