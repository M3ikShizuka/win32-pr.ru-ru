# <a name="pkey_audioendpoint_formfactor"></a>PKEY \_ аудиоендпоинт \_ формфактор

Свойство **PKEY \_ аудиоендпоинт \_ формфактор** указывает конструктивный параметр устройства конечной точки аудио. Форм фактор указывает физические атрибуты устройства аудио конечных точек, которыми управляет пользователь.

Элементу **VT** структуры **пропвариант** присвоено значение VT \_ UI4.

Элемент **уинтвал** структуры **пропвариант** содержит значение перечисления, приведенное к типу uint. Для него задано одно из следующих значений перечисления [**ендпоинтформфактор**](/windows/win32/api/mmdeviceapi/ne-mmdeviceapi-endpointformfactor) :

-   ремотенетворкдевице
-   Speakers
-   линелевел
-   Наушники
-   Микрофон
-   Headset
-   Трубк
-   ункновндигиталпасссраугх
-   SPDIF
-   HDMI
-   ункновнформфактор
