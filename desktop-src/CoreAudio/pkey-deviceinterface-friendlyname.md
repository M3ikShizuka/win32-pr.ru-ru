# <a name="pkey_deviceinterface_friendlyname"></a>PKEY \_ девицеинтерфаце \_ FriendlyName

Свойство **PKEY \_ девицеинтерфаце \_ FriendlyName** содержит понятное имя звукового адаптера, к которому подключено устройство конечной точки (например, "XYZ Audio Adapter").

Элементу **VT** структуры **пропвариант** присвоено значение VT \_ LPWSTR.

Элемент **пвсзвал** структуры **пропвариант** указывает на строку расширенных символов, заканчивающуюся нулем, которая содержит понятное имя.
