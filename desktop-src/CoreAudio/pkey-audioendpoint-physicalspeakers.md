# <a name="pkey_audioendpoint_physicalspeakers"></a>PKEY \_ аудиоендпоинт \_ фисикалспеакерс

Свойство **PKEY \_ аудиоендпоинт \_ фисикалспеакерс** указывает маску конфигурации канала для устройства конечной точки аудио. Маска указывает физическую конфигурацию набора докладчиков и задает назначение каналов для колонок. Дополнительные сведения о масках конфигурации канала см. в следующих статьях:

1.  описание \_ \_ свойства настройки кспроперти AUDIO CHANNEL \_ в документации по Windows DDK.
2.  технический документ под названием "поддержка аудио драйверов для конфигураций динамиков домашнего кинотеатра" в статье [технологии аудио-устройств для Windows](https://www.microsoft.com/whdc/device/audio/default.mspx) .

Элементу **VT** структуры **пропвариант** присвоено значение VT \_ UI4.

Элемент **уинтвал** структуры **пропвариант** содержит маску конфигурации канала, которая приведена к типу **uint**.
