# <a name="position-attribute-shapevml"></a>Атрибут расположения (Shape) (VML)

В этом разделе описывается функция VML, которая является устаревшей в Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). Сведения, рекомендации и рекомендации по текущей версии Windows Internet Explorer см. в [центре разработчиков Internet Explorer](https://msdn.microsoft.com/ie/).

 

Определяет тип позиционирования, используемый для размещения элемента. Read/write. **Строка**.

**Применимо к**:

[Фигурная](shape-element--vml.md)

**Синтаксис тега**

<v: Style *элемента* = "Расположение: *выражение* " >

**Синтаксис сценария**

*element* . Style. Disposition = "*выражение*"

*выражение* = *element*. Style. положением

**Замечания**

Атрибут « **позиционирование** » аналогичен стандартному атрибуту « **позиционирование** HTML», используемому со таблицами стилей.

К этим значениям относятся следующие.



| Значение    | Описание                                                                                                                                                                                                               |
|----------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| static   | По умолчанию. Элемент располагается в соответствии с нормальным потоком страницы. Атрибуты **верхнего** и **левого угла** игнорируются. Если объект привязан встроенным, который происходит только в Microsoft Word, используется это значение. |
| absolute | Элемент располагается относительно родителя с использованием атрибутов **верхнего** и **левого** элементов. Это значение используется для перемещаемых объектов Microsoft Word и Microsoft Excel и слайдов Microsoft PowerPoint.                  |
| relative | Элемент располагается в соответствии с нормальным потоком страницы, но используются **верхние** и **левые** атрибуты. Перекрытие перекрывающихся элементов регулируется атрибутом **Z-Index** .                       |



 

*Стандартный атрибут VML*

**Пример**

Расположение прямоугольника отображается с использованием *относительного* стиля.


```HTML
   <v:rect id=myrect fillcolor="red"
   style="position:relative;top:1;left:1;width:20;height:20">
   </v:rect>
```



[Пример атрибута позиционирования](/previous-versions/bb264090(v=vs.85)). (Требуется Microsoft Internet Explorer 5 или более поздней версии.)
