# <a name="vml-textboxrect-attribute"></a>Атрибут Текстбоксрект VML

В этом разделе описывается функция VML, которая является устаревшей в Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). Сведения, рекомендации и рекомендации по текущей версии Windows Internet Explorer см. в [центре разработчиков Internet Explorer](https://msdn.microsoft.com/ie/).

 

Определяет одно или несколько текстовых полей внутри фигуры. Read/write. **Строка**.

**Применимо к**:

[Путь](msdn-online-vml-path-element.md)

**Синтаксис тега**

<v: *element* текстбоксрект = " *выражение* " >

**Синтаксис сценария**

*element* . текстбоксрект = "*выражение*"

*выражение* = *element*. текстбоксрект

**Замечания**

Текстовое поле определяется одним или несколькими наборами чисел, которые указывают (по порядку) левую, верхнюю, правую и нижнюю точки прямоугольника. Несколько наборов разделяются точкой с запятой. Значением по умолчанию является то же значение измерения, что и содержащий прямоугольник. Если определено более одного текстового поля, разделенные запятыми наборы, определяющие каждое текстовое поле, разделяются точкой с запятой. Обычно текстовые поля поступают в виде наборов из 1, 2, 3 или 6 прямоугольников на фигуре.

*Стандартный атрибут VML*

**Пример**

В левом верхнем углу фигуры определяется и помещается 5 единиц в поле с 95 единиц на 95 единиц.


```HTML
   <v:shape id="rect01"
   coordorigin="0 0" coordsize="200 200"
   strokecolor="red"
   style="top:1;left:1;width:50;height:50">
   <v:path id= "mypath" v="m 1,1 l 1,200, 200,200, 200,1 x e"
   textboxrect="5 5 100 100"/>
   </v:shape>
```
