# <a name="vml-gain-attribute"></a>Атрибут усиления VML

в этом разделе описывается функция VML, которая является устаревшей по отношению к Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). сведения, рекомендации и рекомендации относительно текущей версии Windows Internet explorer см. в [центре разработчиков internet explorer](https://msdn.microsoft.com/ie/).

 

Определяет интенсивность всех цветов в изображении. Read/write. **Вгнумбер**.

**Применимо к**:

[ImageData](msdn-online-vml-imagedata-element.md)

**Синтаксис тега**

<v: *element* прибыль = " *выражение* " >

**Синтаксис сценария**

*element* . прибыль = "*выражение*"

*выражение* = *element*. выигрыш

**Замечания**

Этот атрибут определяет степень яркости белого цвета, влияя на все остальные цвета. Диапазон значений — от 0 до бесконечности. Значение по умолчанию — 1,0. Значение 0 показывает отсутствие изображения. Значения, превышающие 1, доменяют изображение, а значения меньше 1 делают изображение серым.

Этот атрибут можно использовать для создания интересных эффектов.

*Стандартный атрибут VML*

**Пример**

Изображение будет отображаться со всеми цветами, которые будут относиться к серому.


```HTML
   <v:shape id="rect01"
   coordorigin="0 0" coordsize="200 200"
   strokecolor="red"
   style="top:1;left:1;width:300;height:200"
   path="m 1,1 l 1,200, 200,200, 200,1 x e">
   <v:imagedata gain=".5" src="kids.jpg"/>
   </v:shape>
```
