# <a name="vml-target-attribute"></a>Целевой атрибут VML

В этом разделе описывается функция VML, которая является устаревшей в Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). Сведения, рекомендации и рекомендации по текущей версии Windows Internet Explorer см. в [центре разработчиков Internet Explorer](https://msdn.microsoft.com/ie/).

 

Определяет фрейм или окно, в котором отображается URL-адрес. Read/write. **Строка**.

**Применимо к**:

[Фигурная](shape-element--vml.md)

**Синтаксис тега**

<v: *элемент* Target = " *выражение* " >

**Замечания**

**Целевой** атрибут аналогичен стандартному **целевому** атрибуту HTML привязок.

К этим значениям относятся следующие.



| Значение      | Описание                                                                                                                                                                                                                                                      |
|------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| TargetName | Строка, содержащая имя фрейма или окна, в который загружается документ.                                                                                                                                                                                 |
| \_свобод    | Указывает, что связанный документ загружается в новое пустое окно. Это окно не называется.                                                                                                                                                                  |
| \_носител    | Начиная с версии Internet Explorer 6 для Windows XP с пакетом обновления 2 (SP2) \_ атрибут Media устарел и больше не поддерживается. Первый доступ в Internet Explorer 6, это значение указывает, что связанный документ должен быть загружен на панель медиа.                |
| \_источника   | Указывает, что связанный документ загружается в ближайший родительский элемент документа, содержащего ссылку.                                                                                                                                                      |
| \_осуществлять   | Начиная с версии Internet Explorer 7 для Windows XP с пакетом обновления 2 (SP2), \_ атрибут поиска устарел и больше не поддерживается. Первый доступ в Internet Explorer 6, это значение указывает, что связанный документ был загружен в панель поиска в браузере. |
| \_самообслуживания     | Указывает, что связанный документ загружается в окно, в котором была нажата ссылка (активное окно).                                                                                                                                                  |
| \_Вверх      | Указывает, что связанный документ загружается в самое верхнее окно.                                                                                                                                                                                            |



 

*Стандартный атрибут VML*

**Пример**

При щелчке прямоугольника браузер загружает домашнюю страницу корпорации Майкрософт в том же окне.


```HTML
   <v:rect id=myrect fillcolor="red"
   href="https://www.microsoft.com" target="_self"
   style="position:relative;top:1;left:1;width:20;height:20">
   </v:rect>
```



[Пример целевого атрибута](/previous-versions/bb264096(v=vs.85)). (Требуется Microsoft Internet Explorer 5 или более поздней версии.)
