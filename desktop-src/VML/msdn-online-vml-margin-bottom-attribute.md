# <a name="vml-margin-bottom-attribute"></a>Атрибут VML Margin-Bottom

В этом разделе описывается функция VML, которая является устаревшей в Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). Сведения, рекомендации и рекомендации по текущей версии Windows Internet Explorer см. в [центре разработчиков Internet Explorer](https://msdn.microsoft.com/ie/).

 

Задает нижнюю границу содержащего прямоугольника фигуры относительно привязки фигуры. Read/write. **Строка**.

**Применимо к**:

[Фигурная](shape-element--vml.md)

**Синтаксис тега**

<v: поле *элемента* -Bottom = " *выражение* " >

**Синтаксис сценария**

*element* . marginbottom = "*выражение*"

*выражение* = *element*. marginbottom

**Замечания**

Атрибут **Margin-Bottom** аналогичен стандартному атрибуту **поля** HTML, используемому в таблицах стилей.

Обратите внимание, что для написания скрипта вместо **поля по нижнему** краю используется **marginbottom** . Также обратите внимание, что если **положение** **абсолютное**, поле не изменится.

К этим значениям относятся следующие.



| Значение      | Описание                                                                                                                                                                                       |
|------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Auto (Автоматически)       | Расположение элемента по умолчанию в потоке страницы.                                                                                                                                           |
| units      | По умолчанию. Число с абсолютными единицами измерения (cm, mm, in, PT, PC или px) или обозначение относительного числа единиц (EM или ex). Если единицы не заданы, предполагается использование точек (ПКС). Значение по умолчанию — 0. |
| процент | Значение, выраженное в процентах от высоты родительского объекта.                                                                                                                                    |



 

*Стандартный атрибут VML*

**См. также:**

[единиц(ы)](msdn-online-vml-units.md)

**Пример**

Для нижнего поля задается значение 25 пикселей.


```HTML
   <v:rect id=myrect fillcolor="red"
   style="position:relative;top:1;left:1;width:20;height:20;margin-bottom:25px">
   </v:rect>
```



[Пример атрибута Margin-Bottom](/previous-versions/bb229675(v=vs.85)). (Требуется Microsoft Internet Explorer 5 или более поздней версии.)
