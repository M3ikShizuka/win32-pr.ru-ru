# <a name="vml-ivgadjustments-data-type"></a>Тип данных Ивгаджустментс VML

в этом разделе описывается функция VML, которая является устаревшей по отношению к Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). сведения, рекомендации и рекомендации относительно текущей версии Windows Internet explorer см. в [центре разработчиков internet explorer](https://msdn.microsoft.com/ie/).

 

Коллекция корректировок для фигуры, которую можно использовать в формулах. Корректировки можно использовать как временные заполнители или по любой причине для использования переменных. В коллекции доступно только 8 корректировок.



| Атрибуты | Описание                                                                                                                                                                                                                                                                                                                            |
|------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Длина     | **Integer**. Число корректировок. Не может быть больше 8.                                                                                                                                                                                                                                                                          |
| Item       | **Длинное целое**. Массив корректировок, индексируемых от 0 до 7. Обратите внимание, что корректировки могут быть указаны с SPARC. то есть значения промежуточных массивов могут не всегда заполняться. Например, элементы 1, 3 и 5 могут иметь значения длиной 3, для которых задан элемент (0), элемент (2) и элемент (4). Чтобы узнать, существует ли элемент, используйте атрибут EXISTS. |
| Exists     | **Ивгтристате**. Определяет, существует ли указанная корректировка. Обратите внимание, что необходимо использовать индекс. то есть для получения существования элемента должен использоваться существующий (Item).                                                                                                                                                           |
| Значение      | **Строка**. Текстовое представление числовых значений с запятыми между каждым числом.                                                                                                                                                                                                                                                    |
