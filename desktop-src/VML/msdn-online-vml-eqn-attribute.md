# <a name="vml-eqn-attribute"></a>Атрибут ЕКН VML

В этом разделе описывается функция VML, которая является устаревшей в Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). Сведения, рекомендации и рекомендации по текущей версии Windows Internet Explorer см. в [центре разработчиков Internet Explorer](https://msdn.microsoft.com/ie/).

 

Определяет формулу, используемую формулой. Read/write. **Строка**.

**Применимо к**:

[F](msdn-online-vml-f-element.md) (вложенный элемент [формул](msdn-online-vml-formulas-element.md))

**Синтаксис тега**

<v: *element* ЕКН = " *выражение* " >

**Синтаксис сценария**

*element* . ЕКН = "*выражение*"

*выражение* = *element*. ЕКН

**Замечания**

Уравнения определяются вычислением текстового выражения, которое имеет общую форму операции, за которым следует до трех аргументов. Каждый аргумент может иметь следующие типы:

-   корректировка (например, \# 2)
-   другая формула (например, @2 )
-   фиксированные числа (например, 2)
-   предопределенные значения

В таблице ниже определяются формулы, которые можно использовать с необязательными аргументами, заданными именами *v*, *P1* и *P2*. Шаблон формулы:

<f eqn=" *operation* \[*v* \] \[*p1* \] \[*p2* \]"/>



| Операция | Параметры | Exact  | Результат                    | Описание                                                                    |
|-----------|--------|--------|---------------------------|--------------------------------------------------------------------------------|
| Val       | 1      | да    | v                         | Определяет значение Guide из другого значения.                                   |
| Sum       | 3      | да    | v + P1 – P2               | Используется для сложения и вычитания.                                             |
| product   | 3      | Округляет | v \* P1/P2              | Используется для умножения и деления.                                          |
| mid       | 2      | ц    | (v + P1)/2               | Усреднен.                                                                       |
| abs       | 1      | да    | ABS (v)                    | Абсолютное значение.                                                                |
| мин       | 2      | да    | min (v, P1)                 | Меньшее значение v и P1.                                                  |
| max       | 2      | да    | Max (v, P1)                 | Большее значение v и P1.                                                 |
| if        | 3      | да    | v > 0? P1: P2        | Условное тестирование.                                                           |
| mod       | 3      | Нет     | Sqrt (v ^ 2 + P1 ^ 2 + P2 ^ 2)   | Значение модуля.                                                                 |
| atan2     | 2      | Нет     | atan2 (P1, v)               | Полярное значение в градусах \* 2 ^ 16 (единицы с автодемоном).                                     |
| sin       | 2      | Нет     | v \* sin (P1)              | Sin, аргумент в градусах \* 2 ^ 16 ( [единицы с блоком](msdn-online-vml-units.md) фильтра).     |
| cos       | 2      | Нет     | v \* cos (P1)              | COS, аргумент в градусах \* 2 ^ 16 ( [единицы с блоком](msdn-online-vml-units.md) фильтра).     |
| cosatan2  | 3      | Нет     | v \* cos (atan2 (P2, P1))    | Сохраняет полную точность в промежуточном вычислении.                           |
| sinatan2  | 3      | Нет     | v \* sin (atan2 (P2, P1))    | Сохраняет полную точность в промежуточном вычислении.                           |
| sqrt      | 1      | Нет     | Sqrt (v)                   | Результат является положительным и округляется вниз.                                            |
| сумангле  | 3      | да    | v + P1 \* 2 ^ 16 + P2 \* 2 ^ 16 | v масштабируется на 2 ^ 16; P1 и P2 — градусы.<br/>                            |
| ellipse   | 3      | Нет     | P2 \* Sqrt (1-(v/P1) ^ 2)    | Эллипс.                                                                       |
| tan       | 2      | Нет     | v \* Tan (P1)              | Тангенс, аргумент в градусах \* 2 ^ 16 ( [единицы с блоком](msdn-online-vml-units.md) фильтра). |



 

Обратите внимание, что уравнение состоит только из операций и чисел. математические символы опущены. Например, уравнение

ЕКН = "Sum 5 9 3"

Возвращает эквивалент метода

5 + 9-3

для возвращаемого значения 11. Если отсутствуют операнды, значение не используется. Например,

ЕКН = "Sum 5 9"

Возвращает эквивалент метода

5 + 9

и не будет игнорировать отсутствующий операнд.

*Стандартный атрибут VML*

**Пример**

Следующая формула выдает результат 6 (сумма обоих чисел, деленная на 2), что, если это первая формула, может быть извлечена символом " @0 ".


```HTML
    <v:f eqn="mid 5 7"/>
```
