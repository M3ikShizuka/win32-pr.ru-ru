# <a name="vml-tableproperties-attribute"></a>Атрибут Таблепропертиес VML

В этом разделе описывается функция VML, которая является устаревшей в Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). Сведения, рекомендации и рекомендации по текущей версии Windows Internet Explorer см. в [центре разработчиков Internet Explorer](https://msdn.microsoft.com/ie/).

 

Определяет свойства таблицы. Read/write. **Integer**.

**Применимо к**:

[Фигурная](shape-element--vml.md)

**Синтаксис тега**

<v: *element* о:таблепропертиес = " *выражение* " >

**Замечания**

Используется в Microsoft PowerPoint для собственных таблиц. По умолчанию установлено значение 0. Используются только первые три бита этого целого числа.

Несмотря на то, что значение хранится в фигуре, атрибут полезен только в том случае, если таблица состоит из сгруппированных фигур. Биты могут быть объединены.

Включаются следующие битовые значения.



| bit | Описание                              |
|-----|------------------------------------------|
| 1   | Задайте, если группа фигур является таблицей.   |
| 2   | Установите, если фигура является заполнителем.       |
| 3   | Установите, если текст таблицы является двунаправленным. |
