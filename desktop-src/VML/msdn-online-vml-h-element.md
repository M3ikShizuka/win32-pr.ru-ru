# <a name="vml-h-element"></a>Элемент VML H

В этом разделе описывается функция VML, которая является устаревшей в Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). Сведения, рекомендации и рекомендации по текущей версии Windows Internet Explorer см. в [центре разработчиков Internet Explorer](https://msdn.microsoft.com/ie/).

 

Определяет маркер для фигуры.

Следующие атрибуты изменяют обработчик.



| attribute                                                | Описание                                                    |
|----------------------------------------------------------|----------------------------------------------------------------|
| [инвкс](msdn-online-vml-invx-attribute.md)               | Определяет, инвертирована ли *Координата x* маркера. |
| [инви](msdn-online-vml-invy-attribute.md)               | Определяет, инвертирована ли *координата y* маркера. |
| [Схема](msdn-online-vml-map-attribute.md)                 | Определяет диапазон сопоставления для маркера.                         |
| [Полярная](msdn-online-vml-polar-attribute.md)             | Определяет центральную точку для полярных дескрипторов.                 |
| [Позиция](position-attribute--h--vml.md)               | Задает значения *x* и *y* для маркера.                |
| [радиусранже](msdn-online-vml-radiusrange-attribute.md) | Определяет диапазон полярного маркера.                           |
| [Параметр](msdn-online-vml-switch-attribute.md)           | Определяет, меняются ли направления обработчиков.          |
| [ксранже](msdn-online-vml-xrange-attribute.md)           | Определяет диапазон *x* для маркера.                             |
| [иранже](msdn-online-vml-yrange-attribute.md)           | Определяет диапазон *y* для маркера.                             |



 

**Замечания**

Этот элемент должен быть определен внутри элемента [Shape](shape-element--vml.md) .
