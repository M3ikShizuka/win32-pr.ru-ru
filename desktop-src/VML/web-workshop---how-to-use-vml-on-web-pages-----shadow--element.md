# <a name="using-the-shadow-element"></a>Использование элемента Shadow

в этом разделе описывается функция VML, которая является устаревшей по отношению к Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). сведения, рекомендации и рекомендации относительно текущей версии Windows Internet explorer см. в [центре разработчиков internet explorer](https://msdn.microsoft.com/ie/).

 

В этом разделе будет показано, как использовать `<shadow>` вложенный элемент для рисования фигуры с различными эффектами тени.

`<shadow>`Вложенный элемент можно поместить внутри `<shape>` , `<shapetype>` или любого предопределенного элемента Shape, чтобы нарисовать фигуру с тенью. Затем можно использовать атрибуты свойств `<shadow>` вложенного элемента для настройки тени.

Например, чтобы создать фигуру с тенью, как показано на следующем рисунке, в `<BODY>` области веб-страницы можно ввести следующие строки:

![shape1.gif (887 байт)](images/shape1.gif)


```HTML
<v:oval style='width:120pt;height:80pt;' fillcolor="red">
<v:shadow on="t" type="perspective"
origin=".5,.5" offset="0,0"
matrix=",-92680f,,,,-95367431641e-17"/>
</v:oval>
```





-   `on="t"` и `type="perspective"` указывают на включение тени с помощью типа перспективы.
-   **Источник** и **смещение** указывают, где следует нарисовать тень.
-   `matrix="..."` Указывает матрицу преобразования перспективы.

Дополнительные сведения об этом элементе см. в [спецификации VML](https://www.w3.org/TR/NOTE-VML#-toc416858396) .
