# <a name="vml-shapetype-element"></a>Элемент Шапетипе VML

в этом разделе описывается функция VML, которая является устаревшей по отношению к Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). сведения, рекомендации и рекомендации относительно текущей версии Windows Internet explorer см. в [центре разработчиков internet explorer](https://msdn.microsoft.com/ie/).

 

Определяет фигуру, которую можно использовать для создания других фигур.

Следующий атрибут изменяет **шапетипе**.



| attribute                                      | Описание                                             |
|------------------------------------------------|---------------------------------------------------------|
| [Master](msdn-online-vml-master-attribute.md) | Определяет, является ли **шапетипе** главным элементом. |



 

**Замечания**

**Шапетипе** идентичен элементу **Shape** , за исключением того, что он не может ссылаться на другой элемент **шапетипе** .

Если элемент **Shape** ссылается на **шапетипе**, **фигура** может дублировать некоторые свойства, уже указанные в **шапетипе**. В этих случаях атрибуты в **фигуре** переопределяют значения **шапетипе**.

Атрибут **Type** не может использоваться с **шапетипе**.

Атрибуты позиционирования CSS (например, " **Top**", " **Left**" и т. д.) не передаются в **фигуру** из **шапетипе**.

Чтобы использовать этот элемент, создайте **шапетипе** с указанным атрибутом [ID](id-attribute--vml.md) . Затем создайте **фигуру** и сослаться на **шапетипе** с помощью атрибута **Type** . Дополнительные сведения см. в описании атрибута [Type](type-attribute--shape--vml.md) .
