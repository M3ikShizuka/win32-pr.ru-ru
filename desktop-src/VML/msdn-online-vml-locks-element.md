# <a name="vml-locks-element"></a>Элемент locks VML

В этом разделе описывается функция VML, которая является устаревшей в Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). Сведения, рекомендации и рекомендации по текущей версии Windows Internet Explorer см. в [центре разработчиков Internet Explorer](https://msdn.microsoft.com/ie/).

 

Определяет блокировку для фигуры.

Следующие атрибуты изменяют блокировку.



| attribute                                                    | Описание                                                                 |
|--------------------------------------------------------------|-----------------------------------------------------------------------------|
| [аджуссандлес](msdn-online-vml-adjusthandles-attribute.md) | Определяет, можно ли изменять дескрипторы фигуры.                    |
| [AspectRatio](msdn-online-vml-aspectratio-attribute.md)     | Определяет, можно ли изменить пропорции фигуры с помощью редактора. |
| [Обрезка](msdn-online-vml-cropping-attribute.md)           | Определяет, разрешена ли обрезка в редакторе.                   |
| [WLAN](ext-attribute--lock--vml.md)                          | Определяет поведение блокирующих действий для графического редактора.             |
| [Группирование](msdn-online-vml-grouping-attribute.md)           | Определяет, можно ли группировать фигуры в редакторе.                      |
| [Позиция](position-attribute--lock--vml.md)                | Определяет, заблокировано ли расположение фигуры в редакторе.          |
| [Поворот](rotation-attribute--lock--vml.md)                | Определяет, разрешено ли вращение фигур в редакторе.         |
| [Выбор](msdn-online-vml-selection-attribute.md)         | Определяет, является ли фигура выбираемой в редакторе.                    |
| [шапетипе](msdn-online-vml-shapetype-attribute.md)         | Определяет, может ли тип автофигур изменяться редактором.         |
| [Text](msdn-online-vml-text-attribute.md)                   | Определяет, можно ли редактировать текст, присоединенный к фигуре.              |
| [Вершины](msdn-online-vml-vertices-attribute.md)           | Определяет, можно ли изменить вершины пути редактором.      |



 

**Замечания**

Этот элемент должен быть определен внутри элемента [Shape](shape-element--vml.md) .

Элемент **locks** является Microsoft Office РАСШИРЕНИЕМ для VML.
