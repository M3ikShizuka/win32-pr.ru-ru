# <a name="vml-callout-element"></a>Элемент выноски VML

в этом разделе описывается функция VML, которая является устаревшей по отношению к Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). сведения, рекомендации и рекомендации относительно текущей версии Windows Internet explorer см. в [центре разработчиков internet explorer](https://msdn.microsoft.com/ie/).

 

Определяет выноску для фигуры.

Следующие атрибуты изменяют внешний вызов.



| attribute                                                        | Описание                                                                             |
|------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [акцентбар](msdn-online-vml-accentbar-attribute.md)             | Определяет, будет ли вызываемая черта использоваться с выноской.                         |
| [Под](angle-attribute--callout--vml.md)                       | Определяет угол, который задается выноской относительно ограничивающего прямоугольника фигуры. |
| [расстояние;](msdn-online-vml-distance-attribute.md)               | Определяет расстояние перетаскивания в выноске.                                                 |
| [Тени](msdn-online-vml-drop-attribute.md)                       | Определяет место, куда будет помещено удаление выноски.                                  |
| [дропауто](msdn-online-vml-dropauto-attribute.md)               | Определяет, будет ли выноска автоматически удаляться.                             |
| [WLAN](ext-attribute--callout--vml.md)                           | Определяет, как обрабатываются выноски.                                                     |
| [Контур](msdn-online-vml-gap-attribute.md)                         | Определяет расстояние в выноски от ограничивающего прямоугольника.                        |
| [Длина](msdn-online-vml-length-attribute.md)                   | Определяет длину выноски.                                                      |
| [ленгсспеЦифиед](msdn-online-vml-lengthspecified-attribute.md) | Определяет, будет ли использоваться атрибут **длины** для вызова.               |
| [минускс](msdn-online-vml-minusx-attribute.md)                   | Определяет, переворачивается ли выноска вдоль оси x.                              |
| [Минус](msdn-online-vml-minusy-attribute.md)                   | Определяет, переворачивается ли выноска вдоль оси y.                              |
| [On](on-attribute--callout--vml.md) (Вкл.)                             | Определяет, является ли фигура выноской.                                                |
| [текстбордер](msdn-online-vml-textborder-attribute.md)           | Определяет, будет ли выноска иметь текстовую границу.                                   |
| [Тип](type-attribute--callout--vml.md)                         | Определяет тип выноски.                                                            |



 

**Замечания**

Этот элемент должен быть определен внутри элемента [Shape](shape-element--vml.md) .

элемент **выноски** является расширением Microsoft Office для VML.
