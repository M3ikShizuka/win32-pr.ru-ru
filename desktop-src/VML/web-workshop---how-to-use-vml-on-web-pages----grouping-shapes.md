# <a name="grouping-shapes"></a>Группирование фигур

В этом разделе описывается функция VML, которая является устаревшей в Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). Сведения, рекомендации и рекомендации по текущей версии Windows Internet Explorer см. в [центре разработчиков Internet Explorer](https://msdn.microsoft.com/ie/).

 

Как вы узнали, вы можете легко рисовать отдельные фигуры с помощью VML. В этом разделе будут объяснены преимущества группирования фигур и группирования фигур.

Если у вас много фигур, которые необходимо масштабировать, переместить или поворачивать вместе, можно было бы задать атрибуты отдельно для каждой фигуры. Кроме того, вы порождаете поля для ошибок. Было бы лучше, если бы вы могли сгруппировать фигуры, а затем задать атрибуты для всей группы.

В VML можно использовать `<group>` элемент для группировки многих фигур вместе, чтобы их можно было преобразовать как одну единицу. Например, как показано в следующем представлении VML, можно легко сгруппировать две фигуры.


```HTML
<v:group id="GroupA" style='position:relative;left:10pt;top:20pt;
width:150pt;height:100pt; ...>
   <v:shape id="Shape1"...></v:shape>
   <v:shape id="Shape2"...></v:shape>
</v:group>
```



Если фигуры сгруппированы, они используют [Локальное пространство координат](web-workshop---how-to-use-vml-on-web-pages----local-coordinate-space.md) группы. Таким образом, фигуры внутри группы можно масштабировать и перемещать вместе. В разделе Использование локального пространства координат вы увидите дополнительные примеры.

Внутри группы можно создать любое количество фигур или групп. Если группа находится внутри другой группы, она называется "вложенная группа". На уровни вложенности нет ограничений.

Например, в следующих строках показана 3-уровневая вложенная группа. Shape3 и Shape4 находятся в Граупк. Shape2 и Граупк находятся в Граупб. Shape1 и Граупб находятся в группе a.


```HTML
<body>
   <v:group id="GroupA"...>
      <v:shape id="Shape1"...></v:shape>
      <v:group id="GroupB"...>
         <v:shape id="Shape2"...></v:shape>
         <v:group id="GroupC"...>
            <v:shape id="Shape3"...></v:shape>
            <v:shape id="Shape4"...></v:shape>
         </v:group>
      </v:group>
   </v:group>
</body>
```



Дополнительные сведения об этом элементе см. в [спецификации VML](https://www.w3.org/TR/NOTE-VML#-toc416858388) .
