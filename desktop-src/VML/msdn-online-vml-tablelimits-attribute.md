# <a name="vml-tablelimits-attribute"></a>Атрибут Таблелимитс VML

В этом разделе описывается функция VML, которая является устаревшей в Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). Сведения, рекомендации и рекомендации по текущей версии Windows Internet Explorer см. в [центре разработчиков Internet Explorer](https://msdn.microsoft.com/ie/).

 

Список значений минимальной высоты для каждой строки в таблице. Read/write. **Вгленгсаррай**.

**Применимо к**:

[Фигурная](shape-element--vml.md)

**Синтаксис тега**

<v: *element* о:таблелимитс = " *выражение* " >

**Замечания**

Используется в Microsoft PowerPoint для собственных таблиц. Значение по умолчанию — **null**.

Несмотря на то, что значение хранится в фигуре, атрибут полезен только в том случае, если таблица состоит из сгруппированных фигур. Когда текст добавляется в ячейки таблицы, высота строки может увеличиться. Атрибут **таблелимитс** сохраняет исходную высоту строки таким образом, что при удалении текста высота строки не попадает под исходным значением.
