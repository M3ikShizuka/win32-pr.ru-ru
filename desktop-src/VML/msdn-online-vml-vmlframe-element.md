# <a name="vml-vmlframe-element"></a>Элемент VMLFrame VML 

В этом разделе описывается функция VML, которая является устаревшей в Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). Сведения, рекомендации и рекомендации по текущей версии Windows Internet Explorer см. в [центре разработчиков Internet Explorer](https://msdn.microsoft.com/ie/).

 

Определяет кадр для внешней фигуры.

Следующие атрибуты изменяют кадр.



| attribute                                     | Описание                                                           |
|-----------------------------------------------|-----------------------------------------------------------------------|
| [Усечения](msdn-online-vml-clip-attribute.md)    | Определяет, будет ли изображение обрезано.                         |
| [Идентификатор](id-attribute--vmlframe--vml.md)         | Определяет уникальный идентификатор для кадра.                            |
| [Лета](origin-attribute--vmlframe--vml.md) | Указывает источник кадра.                                    |
| [Размер](size-attribute--vmlframe.md)          | Задает размер рамки.                                      |
| [Src](src-attribute--vmlframe--vml.md)       | Указывает источник данных, которые будут отображаться в кадре. |
| [Style](msdn-online-vml-style-attribute.md)  | Определяет обычные стили CSS, которые можно использовать для размещения рамки. |



 

**Замечания**

Исходные данные, отображаемые в кадре, могут содержаться на одной веб-странице или быть частью другой страницы. С помощью сценариев источник можно динамически изменить, а характеристики экрана можно изменить.

Объекты внутри **вмлфраме** имеют масштаб масштабирования. То есть они масштабируются как метафайлы, где весовые значения, тени и текстовые области масштабируются пропорционально. Это отличается от группы, где размер и расположение объекта масштабируются пропорционально, но линии, тени и т. д. не поддерживаются.

Ниже приведен минимальный код, необходимый для загрузки изображения в кадр.


```HTML
   <v:vmlframe
     style="top:1;left:1;width:50;height:50">
     src="external.vml#rect01"/>
   </v:vmlframe>
```



Если файл является внешним, то он должен быть XML-файлом. Следующий код является минимальным кодом, необходимым для указания внешнего исходного файла, который должен содержать идентифицируемую форму. Этот образец содержит фигуру изображения, в которой отображается точечный рисунок.


```HTML
   <xml xmlns:v = "urn:schemas-microsoft-com:vml">
   <v:image id="rect01"
     style="height:100;width:100;top:50;left:50">
     src="kids.jpg">
   </v:rect>
   </xml>
```



> [!Note]  
> В предварительных версиях VML этот элемент назывался **вмлканвас** .
