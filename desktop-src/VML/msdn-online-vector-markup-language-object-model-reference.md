## <a name="introduction"></a>Введение

[Язык VML (VML)](https://www.w3.org/TR/NOTE-datetime.html) — это текстовый язык, который использует [XML](https://www.w3.org/TR/REC-xml.mdl) для расширения HTML с целью отображения векторных графических данных. VML модель DOM (DOM) определяет программный интерфейс для управления элементами документа. Это позволяет пользователю динамически создавать и изменять векторную графику с помощью интерфейса, не зависящего от платформы и языка. Модель DOM VML соответствует спецификации [модель DOM](https://www.w3.org/TR/REC-DOM-Level-1/) .

VML использует элемент [Shape](#shape-element) в качестве базового стандартного блока для векторных графических изображений. После создания фигуры можно изменить форму с помощью атрибутов или вложенных подэлементов. Например, чтобы изменить цвет фигуры, назначьте атрибуту **FillColor** значение цвета.


```HTML
myshape.fillcolor = "red"
```



Некоторые атрибуты фигуры также являются [подэлементами](/windows) и имеют собственные атрибуты, включая следующие:

-   [Фон](#background-element)
-   [Выдавливания](#extrusion-element)
-   [Заполнить](#fill-element)
-   [Группа](#group-element)
-   [имажедата](#imagedata-element)
-   [Путь](#path-element)
-   [Shadow](#shadow-element)
-   [Отклонение](#skew-element)
-   [Водок](#stroke-element)
-   [текстпас](#textpath-element)

Объектная модель VML использует несколько [типов данных](/windows) для определения параметров. Типы данных с префиксом "VG" являются перечислениями, а с префиксом "ИВГ" — объектами. Щелкните здесь для вывода списка. Дополнительные типы с указанными параметрами перечислены в списке.

## <a name="example"></a>Пример

В следующем коде на языке VBScript показано, как создать простую фигуру:


```HTML
        Set MyRect = Document.CreateElement("v:Rect")
        Set R = MyDiv.AppendChild(MyRect)
        R.Style.Position = "absolute"
        R.Style.Width = 20
        R.Style.Height = 20
        R.Style.Top = 50
        R.Style.Left = 50
        R.FillColor = "red"
```



В приведенном выше примере фигура создается с помощью модель DOM метода **CreateElement**. Фигура представляет собой предопределенную фигуру в формате VML. Несмотря на то, что объект существует, он не может быть частью документа, пока он не будет присоединен к документу. С помощью метода **AppendChild** Rect становится дочерним элементом элемента **div** с именем мидив. Несколько минимальных атрибутов стиля («**Расположение**», « **Ширина**», « **Высота**», « **сверху**», « **слева**») устанавливаются для присвоения форме определенного размера. Наконец, цвет присваивается атрибуту **FillColor** . Обратите внимание, что можно использовать любой язык сценариев или любой язык, который может работать с интерфейсами модель DOM.

## <a name="setting-up-vml"></a>Настройка VML

Одна реализация VML осуществляется через Microsoft Internet Explorer 5,0 или более поздней версии. Чтобы правильно настроить объект отрисовки на веб-странице, необходимо выполнить следующие добавления:

1.  Схема должна быть настроена в начальной <HTML> тег следующим образом:
    ```HTML
    <HTML xmlns:v="urn:schemas-microsoft-com:vml">
    ```

    

2.  Режим подготовки к просмотру должен быть частью стиля документа:
    ```HTML
    <STYLE>
    v\:* { behavior: url(#default#VML); display:inline-block}
    </STYLE>
    ```

    

Ниже показан пример веб-страницы HTML, настроенной правильно для VML, отображающей динамическое создание фигуры.


```HTML
<HTML xmlns:v="urn:schemas-microsoft-com:vml">
<HEAD>
<STYLE>
v\:* { behavior: url(#default#VML); display:inline-block}
</STYLE>
<TITLE>VML Sample</TITLE>
</HEAD>
<BODY>
<DIV id="MyDiv"></DIV>
<SCRIPT ID="MYSCRIPT" LANGUAGE="VBScript">
<!--
Set MyRect = Document.CreateElement("v:Rect")
Set R = MyDiv.AppendChild(MyRect)
R.Style.Position = "absolute"
R.Style.Width = 20
R.Style.Height = 20
R.Style.Top = 50
R.Style.Left = 50
R.FillColor = "red"
-->
</SCRIPT>
</BODY>
</HTML>
```



Обратите внимание, что в бета-версиях требуется тег объекта ActiveX и другой стиль поведения.

Число с плавающей запятой с диапазоном от-32 766,0 до 32 766,0.

Целое число с диапазоном от-Infinity до бесконечности.

Коллекция корректировок для фигуры, которую можно использовать для изменения размеров фигуры. Корректировки можно использовать как временные заполнители или по любой причине для использования переменных. В коллекции доступно только 8 корректировок.

**Тип данных length**

Число с плавающей запятой с диапазоном от 0 до бесконечности.

**Тип данных Measure**

Число с плавающей запятой от-Infinity до бесконечности.

**Тип данных String**

Символьные данные любой длины.

**вгблакквхитемоде**

Режим рендеринга для черного и белого. Возможны следующие значения:

-   **Цвет**
-   **Автоматически**
-   **Оттенки серого**
-   **лигхтграйскале**
-   **инверсеграй**
-   **грайаутлине**
-   **блакктекстандлинес**
-   **HighContrast**
-   **Черный**
-   **Белый**
-   **Не вырисовано**

**Тип данных Вгфрактион**

Число с плавающей запятой с диапазоном от 0,0 до 1,0. Доли также могут быть указаны в процентах; Например, "50%".

**Тип данных Вгтристате**

Перечисление, используемое для значений, которые могут быть одного из трех состояний:

-   **TRUE**
-   **FALSE**
-   **MIXED**
