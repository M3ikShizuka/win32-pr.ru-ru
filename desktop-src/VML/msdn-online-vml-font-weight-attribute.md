# <a name="vml-font-weight-attribute"></a>Атрибут VML Font-Weight

в этом разделе описывается функция VML, которая является устаревшей по отношению к Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). сведения, рекомендации и рекомендации относительно текущей версии Windows Internet explorer см. в [центре разработчиков internet explorer](https://msdn.microsoft.com/ie/).

 

Определяет толщину букв шрифта. Read/write. **Строка**.

**Применимо к**:

[текстпас](msdn-online-vml-textpath-element.md)

**Синтаксис тега**

<v: Style *элемента* = "Font-весом: *Expression* " >

**Синтаксис сценария**

*element* . Style. FontWeight = "*выражение*"

*выражение* = *element*. Style. FontWeight

**Замечания**

Значения совпадают со стандартными атрибутами стиля HTML. К этим значениям относятся следующие.



| Значение   | Описание                                                                 |
|---------|-----------------------------------------------------------------------------|
| нормальный  | Нормальный. По умолчанию.                                                            |
| полужирный    | Полужирный.                                                                       |
| bolder  | Более тяжелее обычного.                                                        |
| lighter | Светлее обычного.                                                        |
| 100     | По крайней мере, в качестве веса 200.                                        |
| 200     | По крайней мере выделены полужирным шрифтом, равным 100, и, по крайней мере, в качестве веса 300. |
| 300     | По крайней мере выделены полужирным шрифтом, равным 200, и, по крайней мере, в качестве веса 400. |
| 400     | Нормальный.                                                                     |
| 500     | По крайней мере выделены полужирным шрифтом, равным 400, и, по крайней мере, в качестве веса 600. |
| 600     | По крайней мере выделены полужирным шрифтом, равным 500, и, по крайней мере, в качестве веса 700. |
| 700     | Полужирный.                                                                       |
| 800     | По крайней мере выделены полужирным шрифтом, равным 700, и, по крайней мере, в качестве веса 900. |
| 900     | По крайней мере, выделено полужирным шрифтом 800.                                         |



 

*Стандартный атрибут VML*

**Пример**

Плотность шрифта выделена полужирным шрифтом.


```HTML
   <v:line from="50 100" to="400 100">
   <v:fill on="True" color="red"/>
   <v:path textpathok="True"/>
   <v:textpath on="True" string="VML Text"
   style="font:normal normal bold 36pt Arial"/>
   </v:line>
```
