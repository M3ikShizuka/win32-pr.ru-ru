# <a name="allowincell-attribute"></a>Атрибут Алловинцелл

Определяет, можно ли поместить фигуру в таблицу. Read/write. **Вгтристате**.

**Применимо к**:

[Фигурная](shape-element--vml.md)

**Синтаксис тега**

v:*element* о:алловинцелл = "*выражение*" >

**Замечания**

Используется в Microsoft Word. Значение по умолчанию равно **False**.

*Атрибут расширений Microsoft Office*
