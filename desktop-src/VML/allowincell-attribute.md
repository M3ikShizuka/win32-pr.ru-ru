# <a name="allowincell-attribute"></a>Атрибут Алловинцелл

Определяет, можно ли поместить фигуру в таблицу. Read/write. **Вгтристате**.

**Применимо к**:

[Фигура](shape-element--vml.md)

**Синтаксис тега**

v:*element* о:алловинцелл = "*выражение*" >

**Замечания**

Используется Microsoft Word. Значение по умолчанию равно **False**.

*Microsoft Office Extensions, атрибут*
