# <a name="vml-dashstyle-attribute"></a>Атрибут DashStyle VML

в этом разделе описывается функция VML, которая является устаревшей по отношению к Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). сведения, рекомендации и рекомендации относительно текущей версии Windows Internet explorer см. в [центре разработчиков internet explorer](https://msdn.microsoft.com/ie/).

 

Задает шаблон точки и штриха для штриха. Read/write. **Вглинедашстиле**.

**Применимо к**:

[Водок](msdn-online-vml-stroke-element.md)

**Синтаксис тега**

<v: *element* DashStyle = " *выражение* " >

**Синтаксис сценария**

*element* . DashStyle = "*выражение*"

*выражение* = *element*. DashStyle

**Замечания**

К этим значениям относятся следующие.

-   Solid (по умолчанию)
-   шортдаш
-   шортдот
-   шортдашдот
-   шортдашдотдот
-   Точки
-   Штрих
-   лонгдаш
-   DashDot
-   лонгдашдот
-   лонгдашдотдот

Атрибут **DashStyle** позволяет пользователю указать пользовательский шаблон штриха. Это делается с помощью ряда чисел. Стили штриха определяются с точки зрения длины тире (рисуемой части штриха) и длины пробела между тире. Длины задаются относительно толщины линии: длина "1" равна толщине линии. Стиль **ендкап** применяется к каждому штриху, а стиль стрелки — нет. Строка сначала определяет длину тире, а затем длину пробела. Это может повторяться для формирования сложных стилей штрихов. Строка всегда должна содержать пару цифр. Если оно содержит нечетное число чисел, Последнее может быть не учитывается.

В следующей таблице перечислены некоторые типичные значения и описание предполагаемого воздействия. "0" означает точку, которая должна быть фаурфолд симметричной (при этом круглые концы должны быть кругом). Если конец строки имеет значение «Flat», средство просмотра должно выбрать по возможности встроенный штрих по операционной системе (иными словами). что быстро нарисовано.) В следующей таблице приведены некоторые примеры.



| DashStyle     | Описание                                                                                          |
|---------------|------------------------------------------------------------------------------------------------------|
| "2 2"         | Короткие тире. Каждое тире и расстояние между ними вдвое больше ширины строки.                        |
| "0 2"         | Количество. Расстояние между вдвое больше ширины строки.                                                  |
| "2 2 0 2"     | Короткий штрих пунктир.                                                                                      |
| "2 2 0 2 0 2" | Короткий Штрих пунктирная точка.                                                                                  |
| "1 2"         | Оператор. Каждое тире — это ширина линии, а ширина каждого пробела вдвое превышает ширину строки.            |
| "4 2"         | Черточками. Каждое тире в четыре раза превышает ширину строки, тогда как каждое пространство вдвое превышает ширину строки. |
| "8 2"         | Длинный тире.                                                                                           |
| "4 2 1 2"     | Тире, точка.                                                                                            |
| "8 2 1 2"     | Длинный штрих пунктир.                                                                                       |
| "8 2 1 2 1 2" | Длинный штрихпунктир с точкой.                                                                                   |



 

*Стандартный атрибут VML*

**Пример**

Фигура имеет стиль пунктира для чередующихся штрихов и точек.


```HTML
   <v:shape id="rect01"
   strokecolor="red" fillcolor="red"
   style="top:20;left:20;width:30;height:30"
   path="m 1,1 l 1,200,200,200, 200,1 x e">
   <v:stroke dashstyle="dashdot"/>
   </v:shape>
```
