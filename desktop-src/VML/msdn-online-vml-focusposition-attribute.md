# <a name="vml-focusposition-attribute"></a>Атрибут Фокуспоситион VML

В этом разделе описывается функция VML, которая является устаревшей в Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). Сведения, рекомендации и рекомендации по текущей версии Windows Internet Explorer см. в [центре разработчиков Internet Explorer](https://msdn.microsoft.com/ie/).

 

Определяет центр радиального градиента. Read/write. **VgVector2D**.

**Применимо к**:

[Заполнить](msdn-online-vml-fill-element.md)

**Синтаксис тега**

<v: *element* фокуспоситион = " *выражение* " >

**Синтаксис сценария**

*element* . фокуспоситион = "*выражение*"

*выражение* = *element*. фокуспоситион

**Замечания**

Указывает радиальные градиенты Центральной поситионфор. Вектор представляет собой часть ширины и высоты фигуры. Первый — это процентная доля от заполнения до левого края; Второй — это процентная доля от заполнения до верха. Значение по умолчанию — 0, 0. Чтобы поместить радиальную заливку в центре фигуры, используйте значение 50%, 50%.

*Стандартный атрибут VML*

**Пример**

Радиальная заливка будет центрирована таким образом, что синий цвет будет начинаться в центре и истекает наружу, постепенно меняя его на красную, на время достижения внешних краев и углов.


```HTML
   <v:shape id="rect01"
   coordorigin="0 0" coordsize="200 200"
   strokecolor="red"
   style="top:1;left:1;width:50;height:50"
   path="m 1,1 l 1,200, 200,200, 200,1 x e">
   <v:fill type="GradientRadial" color="red" color2="blue"
   focusposition="50%,50%"/>
   </v:shape>
```
