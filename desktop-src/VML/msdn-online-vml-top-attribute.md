# <a name="vml-top-attribute"></a>Атрибут Top VML

В этом разделе описывается функция VML, которая является устаревшей в Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). Сведения, рекомендации и рекомендации по текущей версии Windows Internet Explorer см. в [центре разработчиков Internet Explorer](https://msdn.microsoft.com/ie/).

 

Определяет расположение фигуры относительно элемента, расположенного над ним в потоке страницы. Read/write. **Строка**.

**Применимо к**:

[Фигурная](shape-element--vml.md)

**Синтаксис тега**

<v: Style *элемента* = "Top: *Expression* " >

**Синтаксис сценария**

*element* . Style.Top = "*выражение*"

*выражение* = *element*. Style.Top

**Замечания**

Атрибут **Top** аналогичен стандартному атрибуту HTML **Top** для Styles.

Единицы могут быть сопоставлены с родительским элементом или могут быть в абсолютных единицах. Этот атрибут не может быть записан для фигур, закрепленных встроенным.

К этим значениям относятся следующие.



| Значение      | Описание                                                                                                                                                      |
|------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Auto (Автоматически)       | Расположение элемента по умолчанию в потоке страницы.                                                                                                          |
| units      | Число с абсолютными единицами измерения (cm, mm, in, PT, PC или px) или обозначение относительного числа единиц (EM или ex). Если единицы не заданы, предполагается использование точек (ПКС). |
| процент | Значение, выраженное в процентах от высоты родительского объекта.                                                                                                   |



 

*Стандартный атрибут VML*

**См. также:**

[единиц(ы)](msdn-online-vml-units.md)

**Пример**

Верхний край фигуры находится на 5 пикселях под объектом, расположенном перед ним в потоке страницы.


```HTML
   <v:rect id=myrect fillcolor="red"
   style="position:relative;top:5;left:1;width:20;height:20">
   </v:rect>
```



[Пример атрибута Top](/previous-versions/bb264098(v=vs.85)). (Требуется Microsoft Internet Explorer 5 или более поздней версии.)
