# <a name="vml-z-index-attribute"></a>Атрибут индекса VML Z

в этом разделе описывается функция VML, которая является устаревшей по отношению к Windows Internet Explorer 9. Веб-страницы и приложения, использующие VML, должны быть перенесены в формат SVG или другие широко поддерживаемые стандарты.

> [!Note]  
> По состоянию на Декабрь 2011 этот раздел был архивирован. В результате он больше не поддерживается. Дополнительные сведения см. в разделе [архивированное содержимое](/previous-versions/windows/internet-explorer/ie-developer/). сведения, рекомендации и рекомендации относительно текущей версии Windows Internet explorer см. в [центре разработчиков internet explorer](https://msdn.microsoft.com/ie/).

 

Определяет порядок вывода перекрывающихся фигур. Read/write. **Строка**.

**Применимо к**:

[Фигура](shape-element--vml.md)

**Синтаксис тега**

<v: Style *элемента* = "z-index: *выражение* " >

**Синтаксис сценария**

*element* . Style. ZIndex = "*выражение*"

*выражение* = *element*. Style. ZIndex

**Замечания**

Атрибут **z-index** аналогичен стандартному атрибуту HTML **Z-Index** для стилей.

К этим значениям относятся следующие.



| Значение | Описание                                                                                                                                                                                            |
|-------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Auto (Автоматически)  | Порядок, в котором фигуры отображаются на странице HTML, будет использоваться снизу вверх.                                                                                                                         |
| порядок | Число, представляющее приоритет стека. Фигура с более высоким числом будет отображаться так, как если бы она вереоверлаппинг (в «лицевой») фигуре с меньшим числом. Можно использовать отрицательные числа. |



 

*Стандартный атрибут VML*

**Пример**

Красная фигура будет отображаться в "лицевой" синей фигуре, так как она имеет более высокий z-индекс.


```HTML
   <v:rect id=bluerect fillcolor="blue"
   style="position:relative;top:1;left:1;width:20;height:20;z-index:1">
   </v:rect>
   <v:rect id=redrect fillcolor="red"
   style="position:relative;top:10;left:10;width:20;height:20;z-index:2">
   </v:rect>
```



[Пример атрибута Z-Index](/previous-versions/ms530275(v=vs.85)). (Требуется Microsoft Internet Explorer 5 или более поздней версии.)
