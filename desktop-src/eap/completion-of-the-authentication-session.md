# <a name="completion-of-the-authentication-session"></a>Завершение сеанса проверки подлинности

После завершения сеанса аутентификации служба проверки подлинности вызывает функцию [**расеапенд**](/previous-versions/windows/desktop/legacy/aa363521(v=vs.85)) , чтобы разрешить протоколу проверки подлинности освободить рабочий буфер. Это действие принимается независимо от того, была ли проверка подлинности успешной. Вызов функции **расеапенд** гарантирует, что никакие дальнейшие вызовы к протоколу проверки подлинности с помощью этого конкретного пользователя или контекста не будут выполнены без предварительного вызова [**расеапбегин**](/previous-versions/windows/desktop/legacy/aa363520(v=vs.85)).
