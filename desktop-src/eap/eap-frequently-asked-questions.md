# <a name="eap-frequently-asked-questions"></a>Часто задаваемые вопросы о EAP

В следующем разделе приведены ответы на часто задаваемые вопросы о API-интерфейсах EAP.



| Вопрос                                                                                        | Ответ                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Каково время существования проверки подлинности EAP?                                                  | В типичной ситуации проверка подлинности состоит из всех объектов, происходящих между вызовами функций [**рапеапбегин**](/previous-versions/windows/desktop/legacy/aa363520(v=vs.85)) и [**расеапенд**](/previous-versions/windows/desktop/legacy/aa363521(v=vs.85)) . Когда пользователь выбирает настройку поставщика EAP в оснастке RRAS, проверка подлинности включает все, что происходит между вызовами методов [**Initialize**](/previous-versions/windows/desktop/api/Rrascfg/nf-rrascfg-ieapproviderconfig-initialize) и [**Uninitialize**](/previous-versions/windows/desktop/api/Rrascfg/nf-rrascfg-ieapproviderconfig-uninitialize) .<br/> |
| Что такое "Групповая политика"?                                                                         | Описание групповой политики см. в разделе [Групповая политика Collection](/previous-versions/windows/it-pro/windows-server-2003/cc779838(v=ws.10)).                                                                                                                                                                                                                                                                                                                                                    |
| Могут ли функции EAP переопределять политику конфигурации, заданную групповой политикой?                      | Нет, никогда. Если групповая политика используется, параметры групповой политики всегда будут переопределять параметры конфигурации EAP.                                                                                                                                                                                                                                                                                                                                                         |
| Мне нужно предупредить пользователей о недопустимых попытках ввода ПИН-кода. Можно ли записать недопустимый ПИН-код? | Когда пользователь вводит неправильный ПИН-код, расширенная проверка подлинности Protocol-Transport уровень безопасности (EAP-TLS) отправит коды ошибок запрашивающему сторона VPN. После возврата кода ошибки запрашивающий объект может реализовать предпочтительную логику повтора.                                                                                                                                                                                                                    |
| Что такое безопасность уровня EAP-Transport (EAP-TLS)?                                                 | EAP-TLS — это протокол клиент-сервер, в котором для клиента и сервера обычно используются отдельные профили сертификатов. Дополнительные сведения см. в стандарте [IETF RTC 2716](/previous-versions/windows/embedded/ms885336(v=msdn.10)).<br/>                                                                                                                                                                                                                                        |
