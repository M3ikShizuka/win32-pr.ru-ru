# <a name="touch-hit-testing-structures"></a>Структуры проверки нажатия касания

В подразделах этого раздела приведены справочные сведения по структурам [проверки попадания в касание](touch-hit-testing-portal.md) .

## <a name="in-this-section"></a>В этом разделе

| Раздел | Описание |
| --- | --- |
| [**TOUCH_HIT_TESTING_INPUT**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_input)<br/> | Содержит сведения об области сенсорного контакта, сообщаемой сенсорным дигитайзером.<br/> |
| [**TOUCH_HIT_TESTING_PROXIMITY_EVALUATION**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_proximity_evaluation)<br/> | Содержит оценку проверки нажатия, показывающую, является ли объект наиболее вероятной целью области сенсорного контакта относительно других объектов, пересекающих зону касания.<br/> |
