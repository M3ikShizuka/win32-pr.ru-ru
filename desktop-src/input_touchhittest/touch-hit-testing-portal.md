# <a name="touch-hit-testing"></a>Проверка нажатия касания

## <a name="purpose"></a>Назначение

В подразделах этого раздела приводится обзор поддержки проверки нажатия сенсорного ввода в Windows. Проверка попадания касания позволяет определить целевой объект на основе того, попадает ли область содержимого элемента пользовательского интерфейса в область контакта или включает контактную точку.

При проверке нажатия касания используется сложная геометрия контакта для всей сенсорной области, а не отдельной интерполяции x-y. Использование всей геометрии контакта значительно повышает точность и точность при определении наиболее вероятной цели сенсорного ввода.

API проверки нажатия касания предназначены для разработчиков, создающих платформы пользовательского интерфейса, которые обеспечивают единообразное взаимодействие с пользователями в настольных приложениях.
