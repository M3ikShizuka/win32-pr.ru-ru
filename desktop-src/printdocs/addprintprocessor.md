# <a name="addprintprocessor-function"></a>Функция Аддпринтпроцессор

Функция **аддпринтпроцессор** устанавливает обработчик заданий печати на указанном сервере и добавляет имя обработчика печати в список поддерживаемых обработчиков печати.

## <a name="syntax"></a>Синтаксис


```C++
BOOL AddPrintProcessor(
  _In_ LPTSTR pName,
  _In_ LPTSTR pEnvironment,
  _In_ LPTSTR pPathName,
  _In_ LPTSTR pPrintProcessorName
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*pName* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя сервера, на котором должен быть установлен обработчик печати. Если этот параметр имеет **значение NULL**, обработчик заданий печати устанавливается локально.

</dd> <dt>

*пенвиронмент* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает среду (например, Windows x86, Windows IA64 или Windows x64). Если этот параметр имеет **значение NULL**, используется текущая среда вызывающего или клиентского объекта (не целевого или сервера).

</dd> <dt>

*ппаснаме* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя файла, содержащего обработчик заданий печати. Этот файл должен находиться в каталоге системного процессора печати.

</dd> <dt>

*ппринтпроцессорнаме* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя обработчика печати.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена, возвращаемое значение будет ненулевым.

Если функция выполняется неудачно, возвращается нулевое значение.

## <a name="remarks"></a>Комментарии

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.

 

Вызывающий объект должен иметь [селоаддриверпривилеже](/windows/desktop/SecAuthZ/authorization-constants).

Перед вызовом функции **аддпринтпроцессор** приложение должно убедиться, что файл, содержащий обработчик заданий, хранится в системном каталоге системного принтера. Приложение может получить имя каталога системного процессора, вызвав функцию [**жетпринтпроцессордиректори**](getprintprocessordirectory.md) .

Приложение может определить имена существующих обработчиков печати, вызвав функцию [**енумпринтпроцессорс**](enumprintprocessors.md) .
