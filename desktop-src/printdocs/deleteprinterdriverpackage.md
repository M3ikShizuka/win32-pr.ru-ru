# <a name="deleteprinterdriverpackage-function"></a>Функция Делетепринтердриверпаккаже

Удаляет пакет драйвера принтера из хранилища драйверов.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT DeletePrinterDriverPackage(
  _In_ LPCTSTR pszServer,
  _In_ LPCTSTR pszInfPath,
  _In_ LPCTSTR pszEnvironment
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*псзсервер* \[ окне\]
</dt> <dd>

Указатель на константу, завершающуюся нулем строку, указывающую имя сервера печати, с которого удаляется пакет драйверов. Значение указателя **null** означает локальный компьютер.

</dd> <dt>

*псзинфпас* \[ окне\]
</dt> <dd>

Указатель на константу, заканчивающуюся нулем строкой, которая указывает путь к INF-файлу драйвера \* .

</dd> <dt>

*псзенвиронмент* \[ окне\]
</dt> <dd>

указатель на константу, завершающуюся нулем строку, указывающую архитектуру процессора (например, Windows NT x86). Может иметь **значение NULL**.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

\_ОК, если операция выполнена успешно.

E \_ ACCESSDENIED, если пакет был доставлен с Windows.

\_Код HRESULT (ошибка \_ при \_ \_ использовании пакета драйвера печати \_ \_ ), если пакет используется.

В противном случае **HRESULT** будет содержать код ошибки.

Дополнительные сведения о кодах ошибок COM см. в разделе [Обработка ошибок](../com/error-handling-in-com.md).

## <a name="remarks"></a>Remarks

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.

 

Обычно хранилище драйверов —% WINDIR% \\ INF или% WINDIR% \\ system32 \\ дриверсторе \\ филерепоситори.

пакет драйверов, поставляемый с Windows, нельзя удалить с помощью этой функции.

Пользователь должен иметь права администратора принтера.
