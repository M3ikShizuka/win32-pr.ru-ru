# <a name="parameter-constructs"></a>Конструкции параметров

Этот раздел не является актуальным. Самые актуальные сведения см. в [спецификации печати схемы](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Параметры, используемые в схеме печати, кратко описывают допустимый диапазон значений, а не дискретное перечисление значений. Платформа схемы печати определяет три элемента, характерные для параметров: элементы Параметердеф описывают параметр и его характеристики, Параметеринит элементы определяют конкретный экземпляр или значение для параметра, а ParameterRef элементы ссылаются на определенный элемент Параметеринит, определенный в других местах.
