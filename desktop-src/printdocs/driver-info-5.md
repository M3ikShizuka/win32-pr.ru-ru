# <a name="driver_info_5-structure"></a>\_Сведения о драйвере \_ 5 структура

Структура **драйвера \_ \_ 5** содержит сведения о драйвере принтера.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _DRIVER_INFO_5 {
  DWORD  cVersion;
  LPTSTR pName;
  LPTSTR pEnvironment;
  LPTSTR pDriverPath;
  LPTSTR pDataFile;
  LPTSTR pConfigFile;
  DWORD  dwDriverAttributes;
  DWORD  dwConfigVersion;
  DWORD  dwDriverVersion;
} DRIVER_INFO_5, *PDRIVER_INFO_5;
```



## <a name="members"></a>Члены

<dl> <dt>

**кверсион**
</dt> <dd>

Версия операционной системы, для которой был записан драйвер. Поддерживаемое значение равно 3.

</dd> <dt>

**pName**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя драйвера (например, QMS 810).

</dd> <dt>

**пенвиронмент**
</dt> <dd>

указатель на строку, завершающуюся нулем, которая указывает среду, для которой был записан драйвер (например, Windows x86, Windows IA64 и Windows x64).

</dd> <dt>

**пдриверпас**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя файла или полный путь и имя файла для файла, содержащего драйвер устройства (например, C: \\ drivers \\Pscript.dll).

</dd> <dt>

**пдатафиле**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя файла или полный путь и имя файла для файла, содержащего данные драйвера (например, C: \\ Drivers \\ Qms810. PPD).

</dd> <dt>

**пконфигфиле**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя файла или полный путь и имя файла для библиотеки динамической компоновки конфигурации драйвера устройства (например, C: \\ drivers \\Pscrptui.dll).

</dd> <dt>

**двдривераттрибутес**
</dt> <dd>

Атрибуты драйвера, например УМПД/КМПД.

</dd> <dt>

**двконфигверсион**
</dt> <dd>

Количество раз, когда файл конфигурации для этого драйвера был обновлен или понижен с момента последнего перезапуска диспетчера очереди печати.

</dd> <dt>

**двдриверверсион**
</dt> <dd>

Число обновлений или понижения уровня файла драйвера для этого драйвера с момента последнего перезапуска диспетчера очереди печати.

</dd> </dl>
