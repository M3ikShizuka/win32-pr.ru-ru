# <a name="schema-imposed-limitations"></a>Ограничения Schema-Imposed

Этот раздел не является актуальным. Самые актуальные сведения см. в [спецификации печати схемы](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Схема PrintCapabilities предоставляет авторам PrintCapabilities значительное количество гибкости и выразительности, которые можно использовать в описаниях устройств. Однако зависимости конфигурации накладывают ограничение на эту гибкость и выразительность. Экземпляры Параметердеф, список элементов компонента, список экземпляров параметров в каждой функции, а также экземпляры Скоредпроперти в каждом параметре не должны содержать зависимости конфигурации. Поставщик документов PrintCapabilities должен обнаружить недопустимые сочетания конфигураций и разрешить их.
