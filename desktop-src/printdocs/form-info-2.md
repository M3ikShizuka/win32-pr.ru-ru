# <a name="form_info_2-structure"></a>\_Структура сведения о форме \_ 2

Содержит сведения о локализуемых формах печати.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _FORM_INFO_2 {
  DWORD   Flags;
  LPTSTR  pName;
  SIZEL   Size;
  RECTL   ImageableArea;
  LPCSTR  pKeyword;
  DWORD   StringType;
  LPCTSTR pMuiDll;
  DWORD   dwResourceId;
  LPCTSTR pDisplayName;
  LANGID  wLangId;
} FORM_INFO_2, *PFORM_INFO_2;
```



## <a name="members"></a>Члены

<dl> <dt>

**Флаги**

Свойства формы. Определены следующие значения, но можно задать только одно из них. При возврате из **формы \_ сведений \_ 2** функцией- [**формой**](getform.md) или [**енумформс**](enumforms.md) **Флаги** устанавливаются в текущее значение в базе данных Forms.

| \_Встроенная форма | Если этот флаг bit установлен, форма является частью диспетчера очереди печати. Определения форм с этим набором флагов не отображаются в реестре. ## <a name="remarks"></a>Remarks

При вызове [**аддформ**](addform.md) или [**сетформ**](setform.md):

-   Если **стрингтипе** имеет значение String \_ None, то оба **пмуидлл** и **пдисплайнаме** должны быть **равны NULL** , а оба **двресаурцеид** и **влангид** должны быть равны 0.
-   Если **стрингтипе** имеет строковый \_ муидлл, **Пдисплайнаме** должен иметь **значение NULL** , а **влангид** должен быть равен 0.
-   Если **стрингтипе** имеет строковый \_ лангпаир, **Пмуидлл** должен иметь **значение NULL** , а **двресаурцеид** должен быть равен 0.

Для **формы \_ сведения \_ 2** , возвращенные вызовом функции [](getform.md) [**енумформс**](enumforms.md):

-   Если **стрингтипе** является строкой \_ муидлл и String \_ лангпаир, **пмуидлл**, **пдисплайнаме**, **двресаурцеид** и **влангид** , все они имеют допустимые значения.
-   Если **стрингтипе** — \_ только String Муидлл, то **пмуидлл** и **двресаурцеид** будут иметь допустимые значения. **пдисплайнаме** будет иметь **значение NULL** , а **влангид** — 0.
-   Если **стрингтипе** — \_ только String Лангпаир, то **пдисплайнаме** и **влангид** будут иметь допустимые значения. **пмуидлл** будет иметь **значение NULL** , а **двресаурцеид** — 0.
