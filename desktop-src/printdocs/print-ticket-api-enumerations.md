# <a name="print-ticket-api-enumerations"></a>Перечисление API билетов на печать

API билета печати использует следующие перечисления.

## <a name="in-this-section"></a>Содержание раздела



| Перечисление                                                   | Описание                                                                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**едефаултдевмодетипе**](/windows/win32/api/prntvpt/ne-prntvpt-edefaultdevmodetype)<br/> | Позволяет пользователям указать, какой [**DEVMODE**](/windows/win32/api/wingdi/ns-wingdi-devmodea) следует использовать в качестве источника значений по умолчанию, если в билете на печать не указаны все возможные параметры.<br/> |
| [**епринттиккетскопе**](/windows/desktop/api/prntvpt/ne-prntvpt-eprintticketscope)<br/>     | Задает область действия билета на печать.<br/>                                                                                                                       |
