# <a name="important-property-instances"></a>Важные экземпляры свойств

Этот раздел не является актуальным. Самые актуальные сведения см. в [спецификации печати схемы](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Чтобы клиент PrintCapabilities мог создать разумное значение PrintTicket, документ PrintCapabilities должен предоставлять определенные свойства экземпляров компонентов, а также экземпляры параметров в этой функции. Универсальный модуль пользовательского интерфейса требует такой информации для создания пользовательского интерфейса. Это, в свою очередь, требует, чтобы ключевые слова схемы Print определяли несколько экземпляров свойств, которые отображаются как дочерние элементы функций и элементов Option.

Элементы компонента могут содержать следующее свойство.



| Свойство                  | Значения                                 | Назначение                                                                                                                                                                                                                                                                                                       |
|---------------------------|----------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| селектионтипе <br/> | пикконе<br/> пиккмани<br/> | Указывает количество параметров, которые могут быть выбраны для этой функции в определенный момент времени (Пикконе) или несколько (Пиккмани). Клиент может использовать эти сведения для создания PrintTicket. Эта информация влияет на поведение пользовательского интерфейса, а также на проверку PrintTicket поставщиком.<br/> |



 

Элементы Option могут содержать следующее свойство.



| Свойство                   | Значения                           | Назначение                                                                                                                                                                                                                                                                                                                                                                 |
|----------------------------|----------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| идентитйоптион <br/> | True<br/> Неверно<br/> | Выбор свойства Идентитйоптион интерпретируется как "отключить эту функцию". Функция, которая содержит свойство Селектионтипе, значение которого равно Пиккмани, также должно содержать параметр, имеющий свойство Идентитйоптион. Код пользовательского интерфейса (или клиент, создающий PrintTicket) должен отменить выбор любого другого параметра, если выбрано свойство Идентитйоптион.<br/> |



 

Элементы feature, Option и Параметердеф могут содержать следующее свойство.



| Свойство              | Значения                          | Назначение                                                                                                                                                                                                                                                                                                                                     |
|-----------------------|---------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| дисплайуи <br/> | Показать<br/> Скрыть<br/> | Указывает, должен ли родительский элемент отображаться в пользовательском интерфейсе. Показать указывает, что элемент должен отображаться в пользовательском интерфейсе. Скрыть указывает, что элемент не должен отображаться в пользовательском интерфейсе. Если это свойство не определено для элемента, то интерпретация по умолчанию показывает, что элемент отображается.<br/> |



 

Элементы feature, Option и Параметердеф могут содержать следующее свойство.



| Свойство                | Значение             | Назначение                                                                                                                                                                                                                                                                                                                                                                                                               |
|-------------------------|-------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| DisplayName <br/> | Строка<br/> | Задает отображаемую строку для родительского элемента, переопределяя отображаемое имя по умолчанию. Может использоваться поставщиками PrintCapabilities для представления локализованного и понятного отображаемого имени. Значение отображаемого имени по умолчанию — это атрибут имени для родительского элемента. <br/> В случае использования элементов Option, если атрибут name не указан, должно присутствовать свойство DisplayName.<br/> |
