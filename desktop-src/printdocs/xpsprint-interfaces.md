# <a name="xps-print-api-interfaces"></a>Интерфейсы API печати XPS

\[Интерфейсы, описанные в этом разделе, являются устаревшими. Вместо этого клиентские приложения должны использовать [API пакета печати документов](./tailored-app-printing-api.md) .\]

\[Икспспринтжоб не поддерживается и может быть изменен или недоступен в будущем. \]

\[Икспспринтжобстреам не поддерживается и может быть изменен или недоступен в будущем. \]

## <a name="in-this-section"></a>Содержание раздела



| Интерфейс                                                   | Описание                                                                               |
|-------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**икспспринтжоб**](/windows/desktop/api/XpsPrint/nn-xpsprint-ixpsprintjob)<br/>             | Предоставляет доступ к выполняемому в данный момент заданию печати.<br/>                  |
| [**икспспринтжобстреам**](/windows/desktop/api/XpsPrint/nn-xpsprint-ixpsprintjobstream)<br/> | Потоковый интерфейс только для записи, в который приложение записывает данные задания печати.<br/> |
