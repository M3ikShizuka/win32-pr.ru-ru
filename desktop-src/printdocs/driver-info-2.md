# <a name="driver_info_2-structure"></a>\_Структура сведений о драйвере \_ 2

Структура **драйвера \_ \_ 2** определяет драйвер принтера, номер версии драйвера, среду, для которой был записан драйвер, имя файла, в котором хранится драйвер, и т. д.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _DRIVER_INFO_2 {
  DWORD  cVersion;
  LPTSTR pName;
  LPTSTR pEnvironment;
  LPTSTR pDriverPath;
  LPTSTR pDataFile;
  LPTSTR pConfigFile;
} DRIVER_INFO_2, *PDRIVER_INFO_2;
```



## <a name="members"></a>Члены

<dl> <dt>

**кверсион**
</dt> <dd>

Версия операционной системы, для которой был записан драйвер. Поддерживаемое значение равно 3.

</dd> <dt>

**pName**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя драйвера (например, "QMS 810").

</dd> <dt>

**пенвиронмент**
</dt> <dd>

указатель на строку, завершающуюся нулем, которая указывает среду, для которой был записан драйвер (например, Windows x86, Windows IA64 и Windows x64).

</dd> <dt>

**пдриверпас**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя файла или полный путь и имя файла для файла, содержащего драйвер устройства (например, "c: \\ drivers \\pscript.dll").

</dd> <dt>

**пдатафиле**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя файла или полный путь и имя файла для файла, содержащего данные драйвера (например, "c: \\ Drivers \\ Qms810. PPD").

</dd> <dt>

**пконфигфиле**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя файла или полный путь и имя файла для .dll конфигурации драйвера устройства (например, "c: \\ drivers \\Pscrptui.dll").

</dd> </dl>
