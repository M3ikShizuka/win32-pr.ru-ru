# <a name="xps-document-structures"></a>Структуры документов XPS

## <a name="contents"></a>Содержимое

## <a name="in-this-section"></a>Содержание раздела



| Структура                                                   | Описание                                                                 |
|-------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**\_Цвет XPS**](xps-color.md)<br/>                  | Структура, описывающая одно значение цвета.<br/>                 |
| [**тире в формате XPS \_**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_dash)<br/>                    | Эта структура описывает элемент штриха контура.<br/>               |
| [**\_индекс глифа XPS \_**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_glyph_index)<br/>     | Описывает размещение и расположение глифа.<br/>                 |
| [**\_сопоставление глифов XPS \_**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_glyph_mapping)<br/> | Описывает сопоставление глифа с индексом.<br/>                              |
| [**\_Матрица XPS**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_matrix)<br/>                | Описывает два левого столбца матрицы размером 3 на 3.<br/>               |
| [**\_точка XPS**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_point)<br/>                  | Представляет пару координат X и Y в двухмерном пространстве.<br/> |
| [**\_прямоугольник XPS**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_rect)<br/>                    | Описывает ширину, высоту и расположение прямоугольника.<br/>        |
| [**\_Размер XPS**](/windows/win32/api/xpsobjectmodel/ns-xpsobjectmodel-xps_size)<br/>                    | Описывает размер объекта.<br/>                                 |
