# <a name="printticket-syntax-and-schema"></a>Синтаксис и схема PrintTicket

Этот раздел не является актуальным. Самые актуальные сведения см. в [спецификации печати схемы](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

PrintTicket выражаются в синтаксисе платформы схемы печати на основе XML. Таким образом, читатель должен проверить [сводку типов элементов](summary-of-element-types.md) на наличие требуемого синтаксиса.

Схема PrintTicket тесно параллельно использует платформу схемы печати с точки зрения типов элементов и в общей структуре, выраженной родительскими и дочерними элементами. Обратите внимание, что элементы Параметердеф не разрешены в PrintTicket, а элементы Параметеринит разрешены только в PrintTicket.
