# <a name="responsibilities-of-consumers-of-the-printcapabilities-document"></a>Обязанности потребителей документа PrintCapabilities

Этот раздел не является актуальным. Самые актуальные сведения см. в [спецификации печати схемы](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Потребители документов PrintCapabilities имеют определенные обязательства, которые они должны правилами, прежде чем они смогут использовать документ PrintCapabilities. Следующие требования применяются к клиентам PrintCapabilities документа.

-   Они не должны отклонять или не обрабатывать синтаксически допустимые PrintCapabilities; то есть любой документ PrintCapabilities, который соответствует схеме PrintCapabilities.

-   Они должны иметь возможность обойти непредвиденное присутствие или отсутствие частного определенного содержимого. Это включает в себя закрытое содержимое, которое отображается в экземплярах элементов, определяемых схемой печати.

-   Они должны иметь возможность обойти необязательное содержимое схемы печати, которое отсутствует.

-   Они должны знать, когда следует запрашивать новый документ.

    -   Значения экземпляров свойств зависят от конфигурации (следовательно, зависит от моментального снимка). При доступе к экземпляру свойства необходимо обновить моментальный снимок.

    -   Экземпляры компонентов, параметров и Скоредпроперти, которые должны быть скопированы в PrintTicket, являются статическими. Это значит, что они не (и не должны быть) зависят от конфигурации устройства. При доступе к экземплярам этих типов не требуется получать новый документ PrintCapabilities при изменении конфигурации.

-   Потребители PrintCapabilities, которые являются клиентами ПОЛЬЗОВАТЕЛЬСКОГО интерфейса, должны иметь возможность использовать информацию в документе PrintCapabilities для создания пользовательского интерфейса и создания допустимого PrintTicket из выбора пользователя. Сюда входит знание того, какие экземпляры Параметеринит должны быть указаны, и проверка указанных экземпляров.
