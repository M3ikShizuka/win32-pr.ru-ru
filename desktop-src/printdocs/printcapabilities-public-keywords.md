# <a name="printcapabilities-public-keywords"></a>Общедоступные ключевые слова PrintCapabilities

Этот раздел не является актуальным. Самые актуальные сведения см. в [спецификации печати схемы](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

В следующем разделе указаны настраиваемые пользователем элементы и определения параметров, которые могут быть применимы к документу PrintCapabilities.

## <a name="user-configurable-elements-usage"></a>Использование настраиваемых элементов

Настраиваемые пользователем элементы представляют собой общедоступные ключевые слова схемы печати, которые могут присутствовать в документе PrintCapabilities или PrintTicket. Они предназначены для представления настраиваемых атрибутов устройств, поддерживаемых конкретным устройством, или для передачи параметров печати приложением или пользователем.

Настраиваемые пользователем элементы могут ссылаться на элементы ссылки на параметр. Дополнительные сведения см. в разделе " [элементы справочника по параметрам](parameter-reference-elements.md) ".

## <a name="parameter-definitions-usage"></a>Использование определений параметров

Определения параметров принимают форму типов элементов Параметердеф в документе возможностей печати. Типы элементов Параметердеф инициализируются в PrintTicket с помощью типа элемента Параметеринит. Значение параметра будет указано с помощью элемента Параметеринит. Настраиваемое пользователем ключевое слово может ссылаться на определение параметра с помощью типа элемента ParameterRef. Дополнительные сведения см. в разделе [конструкции параметров](parameter-constructs.md) .
