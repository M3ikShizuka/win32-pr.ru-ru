# <a name="form_info_1-structure"></a>Структура FORM_INFO_1

Структура **FORM_INFO_1** содержит сведения о печатной форме. Эти сведения включают источник формы печати, ее имя, размеры и размеры области печати.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _FORM_INFO_1 {
  DWORD  Flags;
  LPTSTR pName;
  SIZEL  Size;
  RECTL  ImageableArea;
} FORM_INFO_1, *PFORM_INFO_1;
```



## <a name="members"></a>Члены

<dl> <dt>

**Флаги**
</dt> <dd>

Свойства формы. Определены следующие значения.



| Значение         | Значение                                                                                                                      |
|---------------|------------------------------------------------------------------------------------------------------------------------------|
| FORM_USER    | Если задан этот битовый флаг, форма была определена пользователем. Формы с установленным флагом определяются в реестре.        |
| FORM_BUILTIN | Если этот флаг bit установлен, форма является частью диспетчера очереди печати. Определения форм с этим набором флагов не отображаются в реестре. |
| FORM_PRINTER | Если этот битовый флаг установлен, форма связана с определенным принтером и его определение отображается в реестре.          |



 

</dd> <dt>

**pName**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя формы. Длина имени формы не может превышать 31 символ.

</dd> <dt>

**Размер**
</dt> <dd>

Ширина и высота (в тысячах миллиметров) формы.

</dd> <dt>

**имажеаблеареа**
</dt> <dd>

Ширина и высота (в тысячах миллиметров) формы.

</dd> </dl>
