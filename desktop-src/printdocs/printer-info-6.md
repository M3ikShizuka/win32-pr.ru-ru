# <a name="printer_info_6-structure"></a>\_Структура сведений о принтере \_ 6

В **\_ сведениях \_ о принтере 6** указывается значение состояния принтера.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _PRINTER_INFO_6 {
  DWORD dwStatus;
} PRINTER_INFO_6, *PPRINTER_INFO_6;
```
