# <a name="getdefaultprinter-function"></a>Функция Жетдефаултпринтер

Функция **жетдефаултпринтер** извлекает имя принтера принтера по умолчанию для текущего пользователя на локальном компьютере.

## <a name="syntax"></a>Синтаксис


```C++
BOOL GetDefaultPrinter(
  _In_    LPTSTR  pszBuffer,
  _Inout_ LPDWORD pcchBuffer
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*псзбуффер* \[ окне\]
</dt> <dd>

Указатель на буфер, который получает строку символов, завершающуюся нулем, которая содержит имя принтера по умолчанию. Если этот параметр имеет **значение NULL**, функция завершается ошибкой, и переменная, на которую указывает *пкчбуффер* , возвращает требуемый размер буфера в символах.

</dd> <dt>

*пкчбуффер* \[ в, out\]
</dt> <dd>

В поле входные данные задает размер буфера *псзбуффер* в символах. В выходных данных получает размер строки имени принтера в символах, включая завершающий символ null.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается удачно, возвращаемое значение является ненулевым значением, а переменная, на которую указывает *пкчбуффер* , содержит количество символов, скопированных в буфер *псзбуффер* , включая завершающий нуль-символ.

Если функция выполняется неудачно, возвращается нулевое значение.



| Значение                       | Значение                                                                                                                        |
|-----------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| Ошибка \_ недостаточного \_ буфера | Буфер *псзбуффер* слишком мал. Переменная, на которую указывает *пкчбуффер* , содержит требуемый размер буфера в символах. |
| \_файл ошибок \_ не \_ найден     | Принтер по умолчанию отсутствует.                                                                                                   |



 

## <a name="remarks"></a>Комментарии

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.
