# <a name="about-xps-document-api"></a>Об API документов XPS

[API документа XPS](documents-xps.md) реализует объектную модель XPS и позволяет разработчикам создавать объекты-функции XPS, управлять содержимым документов XPS в собственных \\ \\ программах Windows и сохранять OM XPS в файл или поток в виде документа XPS. Разработчики модулей конвейера фильтра XPSDrv также могут использовать API документов XPS для управления содержимым документов XPS в фильтре драйвера принтера XPSDrv.

## <a name="xps-document-api-overview"></a>Общие сведения об API документов XPS

Объектная модель XPS — это информационная модель документа XPS. Информационная модель документа XPS отличается от модели разметки, используемой внутри частей документа. Объектная модель XPS описывает организацию внутренних компонентов, образующих документ XPS, а модель разметки описывает содержимое этих компонентов.

В программе экземпляр модели объектов XPS создается как объектная модель XPS. МОДЕЛЬ XPS, по сути, представляет собой версию содержимого XPS-документа в памяти. Несмотря на то, что в логической организации используется документ XPS, объектная модель XPS не считается документом XPS, пока он не будет сериализован в файл или поток.

Точная структура разметки недоступна для модели XPS, когда документ XPS десериализуется из разметки в OM-объект XPS. Например, если свойство было представлено как элемент или атрибут в разметке, значение свойства объекта документа будет представлено OM DOM точно таким же образом.

[API документов XPS](documents-xps.md) можно разделить на следующие категории:

-   [Интерфейсы магистральной модели XPS](xps-om-trunk-interfaces.md)

    Магистральные интерфейсы обеспечивают доступ к компонентам верхнего уровня структуры документа XPS. Эти интерфейсы также предоставляют средства для сериализации OM-объекта XPS и десериализации документа XPS.

-   [Интерфейсы страницы OM XPS](xps-object-model-page-interfaces.md)

    Интерфейсы страницы обеспечивают доступ к содержимому страницы в документе XPS. Интерфейсы, описывающие содержимое страницы, также включаются в интерфейсы страницы.

-   [Цифровые подписи модели XPS](using-the-xps-digital-signatures.md)

    МОДЕЛЬ XPS поддерживает цифровые подписи. Однако вы можете получить доступ к цифровым подписям в документе XPS напрямую, не создавая модель XPS. Дополнительные сведения о доступе к цифровым подписям XPS без модели XPS см. в статье [API цифровой подписи XPS](xps-digital-signatures.md).

-   [Интерфейсы билета на печать Operations Manager XPS](xps-object-model-print-ticket-interfaces.md)

    Документы XPS поддерживают печать билетов на уровне пакета (задания), документа и страницы. В билетах печати содержатся сведения о форматировании содержимого документов для печати или просмотра.
