# <a name="addport-function"></a>Функция Аддпорт

Функция **аддпорт** добавляет имя порта в список поддерживаемых портов. Функция **аддпорт** экспортируется монитором порта.

## <a name="syntax"></a>Синтаксис


```C++
BOOL AddPort(
  _In_ LPTSTR pName,
  _In_ HWND   hWnd,
  _In_ LPTSTR pMonitorName
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*pName* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя сервера, к которому подключен порт. Если этот параметр имеет **значение NULL**, то порт является локальным.

</dd> <dt>

*HWND* \[ окне\]
</dt> <dd>

Обработчик родительского окна диалогового окна **аддпорт** .

</dd> <dt>

*пмониторнаме* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает монитор, связанный с портом.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена, возвращаемое значение будет ненулевым.

Если функция выполняется неудачно, возвращается нулевое значение.

## <a name="remarks"></a>Remarks

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.

 

Функция **аддпорт** просматривает сеть, чтобы найти существующие порты, и отображает диалоговое окно для пользователя. Функция **аддпорт** должна проверить имя порта, указанное пользователем, вызвав [**енумпортс**](enumports.md) , чтобы убедиться, что дублирующиеся имена не существуют.

Вызывающая функция **аддпорт** должна иметь доступ к серверу \_ для \_ администрирования доступа к серверу, к которому подключен порт.

Чтобы добавить порт без отображения диалогового окна, вызовите функцию **ксквдата** вместо **аддпорт**. дополнительные сведения о **ксквдата** см. в пакете sdk для Microsoft Windows Driver Kit (DDK).
