# <a name="xps-om-trunk-interfaces"></a>Интерфейсы магистральной модели XPS

Интерфейсы уровня магистрали API XPS-документов описывают структуру документа XPS, который представляет его в виде модели XPS.

<dl>

[Интерфейсы пакета OM в XPS](xps-object-model-package-interfaces.md) Интерфейсы пакета описывают функции и операции, действующие на верхнем уровне OM.  
[Интерфейсы документа OM модели XPS](xps-object-model-document-interfaces.md) Интерфейсы документов описывают организацию и последовательность Фикседдокументсов пакета и документов Фикседпажес.  
[Интерфейсы частей модели XPS](working-with-xps-object-model-part-interfaces.md) Интерфейсы частей описывают ресурсы, используемые документом и его компонентами.  
</dl>
