# <a name="enumforms-function"></a>Функция Енумформс

Функция **енумформс** перечисляет формы, поддерживаемые указанным принтером.

## <a name="syntax"></a>Синтаксис


```C++
BOOL EnumForms(
  _In_  HANDLE  hPrinter,
  _In_  DWORD   Level,
  _Out_ LPBYTE  pForm,
  _In_  DWORD   cbBuf,
  _Out_ LPDWORD pcbNeeded,
  _Out_ LPDWORD pcReturned
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хпринтер* \[ окне\]
</dt> <dd>

Обработчик для принтера, для которого необходимо перечислить формы. Используйте функцию [**опенпринтер**](openprinter.md) или [**аддпринтер**](addprinter.md) для получения маркера принтера.

</dd> <dt>

*Уровень* \[ окне\]
</dt> <dd>

Указывает версию структуры, на которую указывает *пформ* . Это значение должно быть равно 1 или 2.

</dd> <dt>

*пформ* \[ заполняет\]
</dt> <dd>

Указатель на одну или несколько [**структур \_ сведений о форме \_ 1**](form-info-1.md) или в одну или несколько структур [**\_ сведений о форме \_ 2**](form-info-2.md) . Все структуры будут иметь одинаковый уровень.

</dd> <dt>

*кббуф* \[ окне\]
</dt> <dd>

Задает размер (в байтах) буфера, на который *пформ* указывает.

</dd> <dt>

*пкбнидед* \[ заполняет\]
</dt> <dd>

Указатель на переменную, которая получает число байтов, скопированных в массив, в который *пформ* указывает (если операция выполнена успешно), или требуемое число байтов (в случае сбоя, так как *кббуф* слишком мал).

</dd> <dt>

*пкретурнед* \[ заполняет\]
</dt> <dd>

Указатель на переменную, которая получает число структур, скопированных в массив, в который *пформ* Points.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена, возвращаемое значение будет ненулевым.

Если функция выполняется неудачно, возвращается нулевое значение.

## <a name="remarks"></a>Remarks

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.

 

Если вызывающий объект является удаленным, а *уровень* равен 2, то значение **стрингтипе** в возвращенных структурах [**формы \_ info \_**](form-info-2.md) всегда будет **строковым \_ лангпаир**.

в Windows Vista данные формы, возвращаемые **енумформс** , извлекаются из локального кэша, когда *хпринтер* ссылается на удаленный сервер печати или на принтер, размещенный на сервере печати, и существует по крайней мере одно открытое соединение с принтером на удаленном сервере печати. Во всех остальных конфигурациях данные формы запрашиваются с удаленного сервера печати.
