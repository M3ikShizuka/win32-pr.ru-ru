# <a name="how-to-render-print-job-data"></a>Как отобразить данные задания печати

В этом разделе описывается Визуализация данных программы для печати. В этом разделе подробно объясняется, как визуализировать определенные графические или текстовые изображения. Вместо этого он описывает, как управлять данными программы и визуализировать их как документ XPS, который отправляется на принтер с помощью [API печати XPS](xps-printing.md).

## <a name="overview"></a>Обзор

Отрисовка данных заданий печати для печати предполагает получение данных программы, таких как текст, изображения и форматирование, и их преобразование в формат, совместимый с принтером назначения. Программа, отправляющая данные на принтер, должна отправлять их в формате, который требуется драйверу принтера.

Используйте [API печати XPS](xps-printing.md) для отправки данных на принтер, и для этого требуется, чтобы данные были отформатированы как документ XPS. Для создания содержимого XPS-документа, необходимого API печати XPS, образец программы использует [API документа XPS](/previous-versions/windows/desktop/dd316976(v=vs.85)).

Если содержимое программы имеет формат, несовместимый с принтером, он должен быть визуализирован перед отправкой на принтер. Если программа использует [API документов XPS](/previous-versions/windows/desktop/dd316976(v=vs.85)) для управления содержимым программы, содержимое программы будет иметь формат, который можно отправить непосредственно в [API печати XPS](xps-printing.md) и не будет требовать дополнительной подготовки к просмотру перед отправкой на принтер.
