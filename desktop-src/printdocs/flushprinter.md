# <a name="flushprinter-function"></a>Функция Флушпринтер

Функция **флушпринтер** отправляет буфер на принтер, чтобы очистить его из переходного состояния.

## <a name="syntax"></a>Синтаксис


```C++
BOOL FlushPrinter(
  _In_  HANDLE  hPrinter,
  _In_  LPVOID  pBuf,
  _In_  DWORD   cbBuf,
  _Out_ LPDWORD pcWritten,
  _In_  DWORD   cSleep
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хпринтер* \[ окне\]
</dt> <dd>

Маркер объекта принтера. Это должен быть тот же маркер, который использовался в предыдущем вызове [**вритепринтер**](writeprinter.md) драйвером принтера.

</dd> <dt>

*пбуф* \[ окне\]
</dt> <dd>

Указатель на массив байтов, содержащий данные, записываемые на принтер.

</dd> <dt>

*кббуф* \[ окне\]
</dt> <dd>

Размер массива в байтах, на который указывает *пбуф*.

</dd> <dt>

*пквриттен* \[ заполняет\]
</dt> <dd>

Указатель на значение, которое получает число байтов данных, записанных на принтер.

</dd> <dt>

*кслип* \[ окне\]
</dt> <dd>

Время в миллисекундах, в течение которого строка ввода-вывода для порта принтера должна оставаться неактивной.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполняется успешно, возвращается ненулевое значение.

Если функция выполняется неудачно, возвращается нулевое значение.

## <a name="remarks"></a>Комментарии

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.

 

**Флушпринтер** должен вызываться только в случае сбоя [**вритепринтер**](writeprinter.md) , что приводит к переходу принтера в неизменное состояние. Например, принтер может перейти в переходное состояние, когда задание прерывается и драйвер принтера частично отправил некоторые необработанные данные на принтер.

**Флушпринтер** также может указать период простоя, в течение которого диспетчер очереди печати не планирует задания на соответствующий порт принтера.
