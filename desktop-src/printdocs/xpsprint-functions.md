# <a name="xps-print-api-functions"></a>Функции API печати XPS

\[Функции, описанные в этом разделе, являются устаревшими. Вместо этого клиентские приложения должны использовать ссылку на [API пакета печати документов](./tailored-app-printing-api.md) .\]

\[Старткспспринтжоб не поддерживается и может быть изменен или недоступен в будущем. \]

\[StartXpsPrintJob1 не поддерживается и может быть изменен или недоступен в будущем. \]

## <a name="in-this-section"></a>Содержание раздела



| Компонент                                                  | Описание                                                                   |
|-----------------------------------------------------------|-------------------------------------------------------------------------------|
| [**старткспспринтжоб**](/windows/desktop/api/XpsPrint/nf-xpsprint-startxpsprintjob)<br/>   | Начинает печать потока документа XPS на принтер.<br/>               |
| [**StartXpsPrintJob1**](/windows/desktop/api/XpsPrint/nf-xpsprint-startxpsprintjob1)<br/> | Создает задание печати для отправки содержимого XPS-документа на принтер.<br/> |
