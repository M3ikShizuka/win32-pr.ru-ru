# <a name="setdefaultprinter-function"></a>Функция SetDefaultPrinter

Функция **SetDefaultPrinter** задает имя принтера принтера по умолчанию для текущего пользователя на локальном компьютере.

## <a name="syntax"></a>Синтаксис


```C++
BOOL SetDefaultPrinter(
  _In_ LPCTSTR pszPrinter
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*псзпринтер* \[ окне\]
</dt> <dd>

Указатель на строку с завершающим нулем, содержащую имя принтера по умолчанию. Для подключения к удаленному принтеру используется формат имени **\\\\** _сервера_ *_\\_* _принтернаме_. Для локального принтера используется формат имени *принтернаме*.

Если этот параметр имеет **значение NULL** или является пустой строкой, то есть "", **SetDefaultPrinter** выберет принтер по умолчанию на одном из установленных принтеров. Если принтер по умолчанию уже существует, вызов **SetDefaultPrinter** с **нулевым значением** или пустой строкой в этом параметре может изменить принтер по умолчанию.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена, возвращаемое значение будет ненулевым.

Если функция выполняется неудачно, возвращается нулевое значение.

## <a name="remarks"></a>Комментарии

При использовании этого метода необходимо указать допустимый принтер, драйвер и порт. Если они недопустимы, то API-интерфейсы не завершаются ошибкой, но результат не определен. Это может привести к тому, что другие программы снова настроили принтер на предыдущий допустимый принтер. [**Енумпринтерс**](enumprinters.md) можно использовать для получения имени принтера, имени драйвера и имени порта всех доступных принтеров.

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.
