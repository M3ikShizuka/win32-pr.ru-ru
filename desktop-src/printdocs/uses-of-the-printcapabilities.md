# <a name="uses-of-the-printcapabilities"></a>Использование PrintCapabilities

Этот раздел не является актуальным. Самые актуальные сведения см. в [спецификации печати схемы](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

PrintCapabilities предназначены для представления настраиваемых атрибутов устройства в виде конструкций "функция-параметр" или конструкции параметров. Эта информация определяет пространство конфигурации устройства и может использоваться клиентом пользовательского интерфейса для создания пользовательского интерфейса. Ключевые слова схемы Print определяют набор стандартных экземпляров компонентов, экземпляры параметров для стандартных экземпляров компонента и экземпляры Параметердеф.

Функции, параметры и конструкции параметров в PrintCapabilities предназначены для хранения экземпляров свойств или Скоредпроперти, описывающих устройство и поддерживаемых подсистемой диспетчера очереди. Эти экземпляры свойств и Скоредпроперти являются общими для клиентов устройства и содержат сведения, предоставляемые функциями Win32 Девкапс. Ключевые слова схемы Print определяют набор стандартных свойств и экземпляров Скоредпроперти.

Следует подчеркнуть, что документ PrintCapabilities предназначен для представления только однозначных данных: данные, которые не зависят от конкретной конфигурации устройства. Документ PrintCapabilities содержит только моментальный снимок данных, зависящих от конфигурации.
