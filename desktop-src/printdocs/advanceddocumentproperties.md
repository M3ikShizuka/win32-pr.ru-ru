# <a name="advanceddocumentproperties-function"></a>Функция Адванцеддокументпропертиес

Функция **адванцеддокументпропертиес** отображает диалоговое окно настройки принтера для указанного принтера, позволяя пользователю настроить этот принтер.

Эта функция является особым случаем функции [**DocumentProperties**](documentproperties.md) . Дополнительные сведения см. в разделе "Примечания".

## <a name="syntax"></a>Синтаксис


```C++
LONG AdvancedDocumentProperties(
  _In_  HWND     hWnd,
  _In_  HANDLE   hPrinter,
  _In_  LPTSTR   pDeviceName,
  _Out_ PDEVMODE pDevModeOutput,
  _In_  PDEVMODE pDevModeInput
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*HWND* \[ окне\]
</dt> <dd>

Маркер родительского окна диалогового окна настройки принтера.

</dd> <dt>

*хпринтер* \[ окне\]
</dt> <dd>

Маркер объекта принтера. Используйте функцию [**опенпринтер**](openprinter.md) или [**аддпринтер**](addprinter.md) для получения маркера принтера.

</dd> <dt>

*пдевиценаме* \[ окне\]
</dt> <dd>

Указатель на строку с завершающим нулем, указывающую имя устройства, для которого должно отображаться диалоговое окно настройки принтера.

</dd> <dt>

*пдевмодеаутпут* \[ заполняет\]
</dt> <dd>

Указатель на структуру [**DEVMODE**](/windows/win32/api/wingdi/ns-wingdi-devmodea) , которая будет содержать данные конфигурации, указанные пользователем.

</dd> <dt>

*пдевмодеинпут* \[ окне\]
</dt> <dd>

Указатель на структуру [**DEVMODE**](/windows/win32/api/wingdi/ns-wingdi-devmodea) , содержащую данные конфигурации, используемые для инициализации элементов управления диалогового окна настройки принтера.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция [**DocumentProperties**](documentproperties.md) с этими параметрами выполнена успешно, возвращаемое значение **адванцеддокументпропертиес** равно 1. В противном случае возвращаемое значение равно нулю.

## <a name="remarks"></a>Remarks

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.

 

Эта функция может отображать диалоговое окно Настройка принтера только для того, чтобы пользователь мог его настроить. Для получения дополнительных элементов управления используйте [**DocumentProperties**](documentproperties.md). Входные параметры для этой функции передаются непосредственно в **DocumentProperties** , а значение *фмоде* — DM \_ в \_ буфере \| DM \_ в \_ запросе \| DM \_ out \_ . В отличие от **DocumentProperties**, эта функция возвращает только 1 или 0. Поэтому невозможно определить требуемый размер [**DEVMODE**](/windows/win32/api/wingdi/ns-wingdi-devmodea) , установив для *пдевмоде* значение 0.

Приложение может получить имя, на которое указывает параметр *пдевиценаме* , вызвав функцию « [**PrintOut**](getprinter.md) », а затем проверив элемент **ппринтернаме** в структуре « [**\_ сведения о принтере \_ 2**](printer-info-2.md) ».
