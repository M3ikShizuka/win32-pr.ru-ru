# <a name="enumprinterdataex-function"></a>Функция Енумпринтердатаекс

Функция **енумпринтердатаекс** перечисляет все имена значений и данные для указанного принтера и ключа.

Данные принтера хранятся в реестре. При перечислении данных принтера не следует вызывать функции реестра, которые могут изменить данные.

## <a name="syntax"></a>Синтаксис


```C++
DWORD EnumPrinterDataEx(
  _In_  HANDLE  hPrinter,
  _In_  LPCTSTR pKeyName,
  _Out_ LPBYTE  pEnumValues,
  _In_  DWORD   cbEnumValues,
  _Out_ LPDWORD pcbEnumValues,
  _Out_ LPDWORD pnEnumValues
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хпринтер* \[ окне\]
</dt> <dd>

Маркер принтера, для которого функция получает данные конфигурации. Используйте функцию [**опенпринтер**](openprinter.md) или [**аддпринтер**](addprinter.md) для получения маркера принтера.

</dd> <dt>

*пкэйнаме* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает ключ, содержащий значения для перечисления. Используйте символ обратной косой черты ( \\ ) в качестве разделителя, чтобы указать путь с одним или несколькими подразделами. **Енумпринтердатаекс** перечисляет все значения ключа, но не перечисляет значения подразделов указанного ключа. Используйте функцию [**енумпринтеркэй**](enumprinterkey.md) для перечисления подразделов.

Если *пкэйнаме* имеет **значение NULL** или является пустой строкой, **енумпринтердатаекс** возвращает ошибку \_ Недопустимый \_ параметр.

</dd> <dt>

*пенумвалуес* \[ заполняет\]
</dt> <dd>

Указатель на буфер, который получает массив структур [**\_ \_ значений перечислений принтера**](printer-enum-values.md) . Каждая структура содержит имя, тип, данные и размеры значения в разделе ключа.

</dd> <dt>

*кбенумвалуес* \[ окне\]
</dt> <dd>

Размер (в байтах) буфера, на который указывает *пкбенумвалуес*. Если для *кбенумвалуес* задано значение 0, то параметр *пкбенумвалуес* возвращает требуемый размер буфера.

</dd> <dt>

*пкбенумвалуес* \[ заполняет\]
</dt> <dd>

Указатель на переменную, которая получает размер извлеченных данных конфигурации в байтах. Если размер буфера, указанный параметром *кбенумвалуес* , слишком мал, функция возвращает ошибку \_ больше \_ данных, а *пкбенумвалуес* указывает требуемый размер буфера.

</dd> <dt>

*пненумвалуес* \[ заполняет\]
</dt> <dd>

Указатель на переменную, которая получает число структур [**\_ \_ значений перечислений принтера**](printer-enum-values.md) , возвращаемых в *пенумвалуес*.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается успешно, возвращается значение ошибки \_ Success.

Если функция завершается ошибкой, возвращаемое значение является кодом системной ошибки.

## <a name="remarks"></a>Remarks

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.

 

**Енумпринтердатаекс** извлекает данные конфигурации принтера, установленные функциями [**сетпринтердатаекс**](setprinterdataex.md) и [**сетпринтердата**](setprinterdata.md) .
