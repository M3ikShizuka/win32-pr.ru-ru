# <a name="printer_connection_info_1-structure"></a>\_ \_ Структура 1. сведения о подключении принтера \_

Представляет сведения о соединении с принтером.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _PRINTER_CONNECTION_INFO_1 {
  DWORD  dwFlags;
  LPTSTR pszDriverName;
} PRINTER_CONNECTION_INFO_1, *PPRINTER_CONNECTION_INFO_1;
```



## <a name="members"></a>Члены

<dl> <dt>

**dwFlags**
</dt> <dd>

Определены следующие значения:



| Значение                                      | Значение                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|--------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_Несоответствие подключения к принтеру \_ (0x00000020) | Если этот флаг bit установлен, подключение принтера не соответствует. Пользователь может предоставить локальный драйвер печати как *псздривернаме* и использовать его для подготовки к просмотру вместо использования драйвера, установленного на принтере сервера, к которому подключен пользователь.<br/>                                                                                                                                                                                                                                    |
| \_Подключение принтера \_ без \_ пользовательского интерфейса (0x00000040)   | Если этот флаг bit установлен, то этот вызов не может отобразить диалоговое окно. Если необходимо отобразить диалоговое окно для установки драйвера принтера с сервера, а этот флаг установлен, то драйвер принтера не будет установлен, подключение принтера не будет добавлено, и вызов завершится ошибкой.<br/> **Windows 7:** В Windows 7 и более поздних версиях Windows если этот флаг установлен и пользователь работает в режиме с повышенными правами, диалоговое окно " **доверять этому принтеру** " не отображается.<br/> |



 

</dd> <dt>

**псздривернаме**
</dt> <dd>

Указатель на имя драйвера.

</dd> </dl>
