# <a name="configuration-dependency-in-the-printcapabilities-schema"></a>Зависимость конфигурации в схеме PrintCapabilities

Этот раздел не является актуальным. Самые актуальные сведения см. в [спецификации печати схемы](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Схема PrintCapabilities тесно параллельна с платформой схемы печати с точки зрения используемых элементов и общей структуры, выраженной родительскими и дочерними элементами. Зависимости конфигурации, относящиеся к PrintCapabilities, перечислены здесь как расширение для общей инфраструктуры. Зависимости конфигурации относятся к тому факту, что некоторые элементы, включая их содержимое, могут быть изменены или даже удалены из одного документа PrintCapabilities на следующий в результате зависимости от конфигурации. Даже если родительский элемент должен быть независимым от конфигурации, его дочерние элементы могут иметь зависимости. Такие зависимости выражаются с помощью \* конструкций переключения или \* вариантов в файлах ГПД.

В качестве примера зависимости конфигурации учитывайте значения, связанные с каждым свойством в документе PrintCapabilities. Каждый документ PrintCapabilities может отличаться в определенных экземплярах свойств и в экземплярах значений, назначенных этим экземплярам свойств.



| Тип элемента                                                                                                                                                                             | Зависимость конфигурации                                                                                                                                                                                                      |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Компонент <br/> Параметр<br/> параметеринит<br/> ParameterRef<br/> PrintCapabilities<br/> PrintTicket<br/> Свойство<br/> скоредпроперти<br/> | Эти элементы не должны иметь зависимостей конфигурации.<br/>                                                                                                                                                           |
| параметердеф <br/>                                                                                                                                                                 | Элементы Параметердеф и элементы, содержащиеся в них, не должны иметь зависимостей конфигурации.<br/>                                                                                        |
| Свойство <br/>                                                                                                                                                                     | Элементы свойств могут иметь зависимости конфигурации, за исключением случаев, когда они появляются в элементах Параметердеф.<br/>                                                                                                       |
| Значение <br/>                                                                                                                                                                        | Элементы значения, которые отображаются в элементах Скоредпроперти, не должны иметь зависимостей конфигурации. Элементы значения, отображаемые в элементе Property, могут иметь произвольную зависимость от конфигурации.<br/> |
