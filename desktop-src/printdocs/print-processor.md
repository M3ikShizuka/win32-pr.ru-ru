# <a name="print-processor"></a>Обработчик заданий печати

Диспетчер очереди печати отслеживает текущие задания печати и Целевой принтер, чтобы определить соответствующее время для печати задания. После того как диспетчер очереди печати определит, что задание должно быть напечатано, оно вызывает обработчик заданий печати. Обработчик заданий печати — это подключаемый модуль, который обрабатывает данные заданий печати.

Используйте следующие функции для работы с обработчиками печати.



| Функция                                                           | Описание                                                          |
|--------------------------------------------------------------------|----------------------------------------------------------------------|
| [**аддпринтпроцессор**](addprintprocessor.md)                     | Устанавливает обработчик заданий печати на указанном сервере.                    |
| [**делетепринтпроцессор**](deleteprintprocessor.md)               | Удаляет процессор принтера с указанного сервера.                 |
| [**енумпринтпроцессордататипес**](enumprintprocessordatatypes.md) | Перечисляет типы данных, поддерживаемые указанным обработчиком печати. |
| [**енумпринтпроцессорс**](enumprintprocessors.md)                 | Перечисляет обработчики печати, установленные на указанном сервере.     |
| [**жетпринтпроцессордиректори**](getprintprocessordirectory.md)   | Получает путь к обработчику заданий печати на указанном сервере.  |
