# <a name="whats-new-for-printing"></a>Новые возможности печати

Windows 8 поддерживает [Печать приложений для Магазина Windows с помощью JavaScript и HTML](/previous-versions/windows/apps/hh465225(v=win.10)) и [Печать для приложений Магазина Windows с помощью C#/VB/C + + и XAML](/previous-versions/windows/apps/hh465196(v=win.10)).

Windows 8 также включает новый API на основе COM, обеспечивающий полную поддержку [опенкспс печати](app-support-for-openxps-printing.md) и доступа к частям новых драйверов принтеров, поддерживаемых Windows 8. Дополнительные сведения о новом API см. в разделе [Печать пакета документа API](tailored-app-printing-api.md).

Программа "Входящие" драйвера печати Windows гарантирует, что в Windows 8 включена поддержка высокой доли популярных принтеров.

В следующих разделах описываются функции печати и документирования, появившиеся в более ранних версиях Windows.
