# <a name="whats-new-for-printing"></a>Новые возможности печати

Windows 8 поддерживает [печать для приложений Windows store с помощью JavaScript и HTML](/previous-versions/windows/apps/hh465225(v=win.10)) и [печати для Windows приложений магазина с помощью C#/VB/C + + и XAML](/previous-versions/windows/apps/hh465196(v=win.10)).

Windows 8 также включает новый API на основе COM, обеспечивающий полную поддержку [опенкспс печати](app-support-for-openxps-printing.md) и доступа к частям новых драйверов принтера, которые Windows 8 поддерживаются. Дополнительные сведения о новом API см. в разделе [Печать пакета документа API](tailored-app-printing-api.md).

Windows программа "входящие" драйвера принтера гарантирует, что Windows 8 включает поддержку высокой доли популярных принтеров.

В следующих разделах описываются функции печати и документирования, появившиеся в более ранних версиях Windows.
