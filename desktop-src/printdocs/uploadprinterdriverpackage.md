# <a name="uploadprinterdriverpackage-function"></a>Функция Уплоадпринтердриверпаккаже

Передает драйвер принтера в хранилище драйверов сервера печати, чтобы его можно было установить путем вызова [**инсталлпринтердриверфромпаккаже**](installprinterdriverfrompackage.md).

## <a name="syntax"></a>Синтаксис


```C++
HRESULT UploadPrinterDriverPackage(
  _In_    LPCTSTR pszServer,
  _In_    LPCTSTR pszInfPath,
  _In_    LPCTSTR pszEnvironment,
  _In_    DWORD   dwFlags,
  _In_    HWND    hwnd,
  _Out_   LPTSTR  pszDestInfPath,
  _Inout_ PULONG  pcchDestInfPath
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*псзсервер* \[ окне\]
</dt> <dd>

Указатель на константу, заканчивающуюся нулем строкой, которая указывает имя сервера печати. Если сервер является локальным, используйте **значение NULL** .

</dd> <dt>

*псзинфпас* \[ окне\]
</dt> <dd>

Указатель на константу, заканчивающуюся нулем строкой, которая указывает исходный путь к INF-файлу драйвера.

</dd> <dt>

*псзенвиронмент* \[ окне\]
</dt> <dd>

указатель на константу, заканчивающуюся нулем строкой, которая указывает архитектуру процессора сервера (например, Windows NT x86). Может иметь **значение NULL**.

</dd> <dt>

*dwFlags* \[ окне\]

## <a name="return-value"></a>Возвращаемое значение

Если операция выполнена удачно, возвращаемое значение будет S_OK, в противном случае **HRESULT** будет содержать код ошибки.

Дополнительные сведения о кодах ошибок COM см. в разделе [Обработка ошибок](../com/error-handling-in-com.md).

## <a name="remarks"></a>Remarks

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.

 

Хранилище драйверов обычно имеет значение% WINDIR% \\ INF или% WINDIR% \\ system32 \\ дриверсторе \\ филерепоситори.

Можно отправить только один пакет за раз. Если пакет зависит от других, он должен быть отправлен отдельно.

Можно отправлять только подписанные пакеты драйверов.
