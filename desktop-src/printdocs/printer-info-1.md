# <a name="printer_info_1-structure"></a>\_Структура сведений о принтере \_ 1

В **структуре \_ сведения \_ о принтере 1** указаны общие сведения о принтере.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _PRINTER_INFO_1 {
  DWORD  Flags;
  LPTSTR pDescription;
  LPTSTR pName;
  LPTSTR pComment;
} PRINTER_INFO_1, *PPRINTER_INFO_1;
```

<dl> <dt>
