# <a name="xps-document-api"></a>API документов XPS

API документов XPS — это собственный API Windows, позволяющий программам создавать, читать, редактировать, сохранять и печатать XPS-документы. API документов XPS поддерживается в Windows 7 и более поздних версиях Windows и может использоваться в программах пользовательского режима и драйверах принтера XPSDrv.

API документов XPS, JScript как XPS, так и Опенкспс, предназначен для программистов C++, создающих собственные приложения Windows. Дополнительные сведения о спецификации Open XPS см. в [статье о формате Open XML Specification (опенкспс)](https://www.ecma-international.org/publications/standards/Ecma-388.htm).
