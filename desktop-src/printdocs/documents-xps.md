# <a name="xps-document-api"></a>API документов XPS

api документов xps — это собственный Windows api, позволяющий программам создавать, читать, редактировать, сохранять и печатать XPS-документы. API документов XPS поддерживается в Windows 7 и более поздних версиях Windows и может использоваться в программах пользовательского режима и драйверах принтера XPSDrv.

API документов xps, jscript как XPS, так и опенкспс, предназначен для программистов C++, создающих собственные Windows приложения. Дополнительные сведения о спецификации Open XPS см. в [статье о формате Open XML Specification (опенкспс)](https://www.ecma-international.org/publications/standards/Ecma-388.htm).
