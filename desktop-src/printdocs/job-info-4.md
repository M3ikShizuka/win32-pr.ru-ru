# <a name="job_info_4-structure"></a>\_Структура сведений о задании \_ 4

Описывает полный набор значений, связанных с заданием, и поддерживает большие файлы очереди с размерами, которые выражаются с 64 битами.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _JOB_INFO_4 {
  DWORD                JobId;
  LPTSTR               pPrinterName;
  LPTSTR               pMachineName;
  LPTSTR               pUserName;
  LPTSTR               pDocument;
  LPTSTR               pNotifyName;
  LPTSTR               pDatatype;
  LPTSTR               pPrintProcessor;
  LPTSTR               pParameters;
  LPTSTR               pDriverName;
  LPDEVMODE            pDevMode;
  LPTSTR               pStatus;
  PSECURITY_DESCRIPTOR pSecurityDescriptor;
  DWORD                Status;
  DWORD                Priority;
  DWORD                Position;
  DWORD                StartTime;
  DWORD                UntilTime;
  DWORD                TotalPages;
  DWORD                Size;
  SYSTEMTIME           Submitted;
  DWORD                Time;
  DWORD                PagesPrinted;
  LONG                 SizeHigh;
} JOB_INFO_4, *PJOB_INFO_4;
```



## <a name="members"></a>Члены

<dl> <dt>

**JobId**
</dt> <dd>

Значение идентификатора задания.

</dd> <dt>

**ппринтернаме**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя принтера, для которого задано постановка в очередь.

</dd> <dt>

**пмачиненаме**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя компьютера, создавшего задание печати.

</dd> <dt>

**пусернаме**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя пользователя, владеющего заданием печати.

</dd> <dt>

**пдокумент**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя задания печати (например, MS-WORD: Review.doc).

</dd> <dt>

**пнотифинаме**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя пользователя, который должен получать уведомления при печати задания или при возникновении ошибки во время печати задания.

</dd> <dt>

**пдататипе**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает тип данных, используемых для записи задания печати.

</dd> <dt>

**ппринтпроцессор**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя обработчика печати, который должен использоваться для печати задания.

</dd> <dt>

**ппараметерс**
</dt> <dd>

Указатель на строку, завершающуюся нулем, указывающую параметры процессора печати.

</dd> <dt>

**пдривернаме**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает имя драйвера принтера, который должен использоваться для обработки задания печати.

</dd> <dt>

**пдевмоде**
</dt> <dd>

Указатель на структуру [**DEVMODE**](/windows/win32/api/wingdi/ns-wingdi-devmodea) , которая содержит данные об инициализации устройства и среде для драйвера принтера.

</dd> <dt>

**пстатус**
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает состояние задания печати. Этот элемент должен быть проверен до **состояния** , и, если **Пстатус** имеет **значение NULL**, состояние определяется содержимым элемента Status.

</dd> <dt>

**псекуритидескриптор**
</dt> <dd>

Значение этого элемента равно **null**. Извлечение и настройка дескрипторов безопасности документов в этом выпуске не поддерживается.

</dd> <dt>

**Состояние**
</dt> <dd>

Состояние задания. Этот элемент может иметь одно или несколько следующих значений:

| Значение                           | Значение                                                      |
|---------------------------------|--------------------------------------------------------------|
| \_состояние задания \_ заблокировано \_ девк      | Драйвер не может напечатать задание.                             |
| \_состояние задания \_ удалено            | Задание удалено.                                        |
| \_удаление состояния \_ задания           | Идет удаление задания.                                        |
| \_Ошибка состояния \_ задания              | С заданием связана ошибка.                         |
| состояние задания в \_ \_ автономном режиме            | Принтер находится в автономном режиме.                                          |
| \_Бумага состояния \_ задания           | В принтере нет бумаги.                                     |
| \_состояние задания \_ приостановлено             | Задание приостановлено.                                               |
| \_состояние задания \_ напечатано            | Задание напечатано.                                             |
| \_Печать состояния \_ задания           | Идет печать задания.                                             |
| \_перезапуск состояния \_ задания            | Задание перезапущено.                                      |
| \_очередь состояний \_ заданий           | Задание находится в очереди.                                             |
| \_состояние задания \_ — \_ вмешательство пользователя | В принтере есть ошибка, которая требует от пользователя выполнить какие-либо действия. |



 

в Windows XP и более поздних версиях Windows также можно использовать следующие значения:

| Значение                 | Значение                                                                                       |
|-----------------------|-----------------------------------------------------------------------------------------------|
| \_состояние задания \_ завершено | Задание отправляется на принтер, но может быть еще не напечатано. Дополнительные сведения см. в разделе "Примечания". |
| состояние задания — " \_ \_ сохранены" | Задание было сохранено в очереди печати после печати.                              |



 

</dd> <dt>

**Приоритет**
</dt> <dd>

Приоритет задания. Этот элемент может принимать одно из следующих значений или находиться в диапазоне от 1 до 99 (от МИНИМАЛЬного \_ приоритета до максимального \_ приоритета).



| Значение         | Значение           |
|---------------|-------------------|
| Минимальный \_ приоритет | Минимальный приоритет. |
| МАКСИМАЛЬНЫЙ \_ приоритет | Максимальный приоритет. |
| \_приоритет DEF | Приоритет по умолчанию. |



 

</dd> <dt>

**Позиция**
</dt> <dd>

Расположение задания в очереди печати.

</dd> <dt>

**StartTime**
</dt> <dd>

Самое раннее время, когда может быть распечатано задание.

</dd> <dt>

**унтилтиме**
</dt> <dd>

Последнее время, когда задание может быть напечатано.

</dd> <dt>

**TotalPages**
</dt> <dd>

Количество страниц, необходимых для задания. Это значение может быть равно нулю, если задание печати не содержит сведений об разделителе страниц.

</dd> <dt>

**Размер**
</dt> <dd>

Младшие четыре байта задания (в байтах). См. также элемент **сизехигх** ниже.

</dd> <dt>

**Отправлено**
</dt> <dd>

Структура [**SYSTEMTIME**](/windows/desktop/api/minwinbase/ns-minwinbase-systemtime) , указывающая время отправки задания.

Это значение времени указано в формате универсальной координаты времени (UTC). Перед отображением его необходимо преобразовать в местное значение времени. Для выполнения преобразования можно использовать функцию [**филетиметолокалфилетиме**](/windows/desktop/api/fileapi/nf-fileapi-filetimetolocalfiletime) .

</dd> <dt>

**Время**
</dt> <dd>

Общее время (в миллисекундах), прошедшее с момента начала печати задания.

</dd> <dt>

**пажеспринтед**
</dt> <dd>

Число напечатанных страниц. Это значение может быть равно нулю, если задание печати не содержит сведений об разделителе страниц.

</dd> <dt>

**сизехигх**
</dt> <dd>

Старшие четыре байта для задания (в байтах). См. также элемент **size** выше.

</dd> </dl>

## <a name="remarks"></a>Remarks

Мониторы портов, не поддерживающие Труиндофжоб, задают задание как \_ состояние задания, \_ выводимое сразу после отправки задания на принтер.
