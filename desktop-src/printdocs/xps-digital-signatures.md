# <a name="xps-digital-signature-api"></a>API цифровой подписи XPS

API цифровой подписи XPS позволяет пользователю подписать документ, проверить удостоверение подписавшего и указать, изменился ли документ XPS с момента подписания. API цифровой подписи XPS основан на технологии цифровой подписи, которая используется в соглашениях Open Packaging, которые указаны в первом выпуске, часть 2, «Open Packaging Conventions» [стандартных форматов файлов ECMA-376, Office Open XML](https://www.ecma-international.org/publications/standards/Ecma-376.htm).

## <a name="platform-update-for-windows-vista"></a>Обновление платформы для Windows Vista

Интерфейсы цифровой подписи XPS, описанные в этом разделе, не поддерживаются обновлением платформы для Windows Vista или обновления платформы для Windows Server 2008. Приложение, которому требуются эти интерфейсы, должно выполняться в Windows 7 или более поздней версии или в Windows Server 2008 R2 или более поздней версии. В противном случае приложение может не предоставить пользователю полную функциональность приложения.
