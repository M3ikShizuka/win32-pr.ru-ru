# <a name="port_info_3-structure"></a>\_Сведения о порте \_ 3 структура

В структуре " **\_ сведения о порте \_ 3** " указывается значение состояния порта принтера.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _PORT_INFO_3 {
  DWORD  dwStatus;
  LPTSTR pszStatus;
  DWORD  dwSeverity;
} PORT_INFO_3, *PPORT_INFO_3;
```



## <a name="members"></a>Члены

<dl> <dt>

**Dwstatus устанавливается**
</dt> <dd>

Новое значение состояния порта. Это значение используется только в том случае, если элемент **псзстатус** имеет **значение NULL**.

Этот элемент может принимать одно из следующих значений.



| Значение                            | Значение                                             |
|----------------------------------|-----------------------------------------------------|
| 0                                | Очищает состояние порта принтера.                     |
| состояние порта в \_ \_ автономном режиме            | Принтер порта отключен.                      |
| \_состояние порта \_ \_ Застревание бумаги         | В принтере порта есть Застревание бумаги.                 |
| \_Бумага состояния \_ порта \_         | В принтере порта нет бумаги.                 |
| \_ \_ выходной лоток состояния \_ порта \_ полон  | Выходной лоток принтер'с для порта заполнен.            |
| \_проблема с \_ бумагой о состоянии порта \_     | На принтере порта имеется проблема с бумагой.             |
| \_состояние порта \_ без \_ тонера          | У принтера порта нет тонера.                 |
| \_ \_ открыта дверца состояния порта \_         | Открыта дверца принтера порта.             |
| \_состояние порта \_ \_ вмешательство пользователя | Для принтера порта требуется вмешательство пользователя.      |
| состояние порта " \_ \_ недостаточно \_ \_ памяти"    | В принтере порта не хватает памяти.                |
| \_ \_ низкое тонер состояния \_ порта         | На принтере порта не хватает тонера.                 |
| \_прогрев состояния \_ порта \_        | Принтер порта прогрев.                   |
| состояние порта "Энергосбережение" \_ \_ \_        | Принтер порта находится в режиме экономии энергии. |



 

</dd> <dt>

**псзстатус**
</dt> <dd>

Указатель на новую строку значения состояния порта принтера, которую нужно задать. Используйте этот элемент, если нет подходящего значения состояния в списке для **dwstatus устанавливается**.

</dd> <dt>

**двсеверити**
</dt> <dd>

Серьезность значения состояния порта.

Этот элемент может принимать одно из следующих значений.



| Значение                       | Значение                                   |
|-----------------------------|-------------------------------------------|
| \_ \_ Ошибка типа состояния \_ порта   | Значение "состояние порта" указывает на ошибку. |
| \_предупреждение о \_ ТИПЕ \_ состояния порта | Значение "состояние порта" является предупреждением.       |

</dd> </dl>

## <a name="remarks"></a>Комментарии

Если для параметра состояние порта принтера задано значение \_ \_ Ошибка тип состояния порта \_ , диспетчер очереди печати прекращает отправку заданий на порт. Диспетчер очереди печати не возобновляет отправку заданий в порт, пока не будет выполнен другой вызов [**сетпорт**](setport.md) для очистки состояния.
