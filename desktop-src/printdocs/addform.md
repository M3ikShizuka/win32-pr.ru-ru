# <a name="addform-function"></a>Функция Аддформ

Функция **аддформ** добавляет форму в список доступных форм, которые можно выбрать для указанного принтера.

## <a name="syntax"></a>Синтаксис


```C++
BOOL AddForm(
  _In_ HANDLE hPrinter,
  _In_ DWORD  Level,
  _In_ LPBYTE pForm
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хпринтер* \[ окне\]
</dt> <dd>

Маркер принтера, поддерживающий печать с указанной формой. Используйте функцию [**опенпринтер**](openprinter.md) или [**аддпринтер**](addprinter.md) для получения маркера принтера.

</dd> <dt>

*Уровень* \[ окне\]
</dt> <dd>

Уровень структуры, на которую указывает *пформ* . Это значение должно быть равно 1 или 2.

</dd> <dt>

*пформ* \[ окне\]
</dt> <dd>

Указатель на [**\_ информационную структуру формы \_ 1**](form-info-1.md) или [**\_ сведения о форме \_ 2**](form-info-2.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена, возвращаемое значение будет ненулевым.

Если функция выполняется неудачно, возвращается нулевое значение.

## <a name="remarks"></a>Комментарии

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.

 

Приложение может определить, какие формы доступны для принтера, вызвав функцию [**енумформс**](enumforms.md) .

Если *пформ* указывает на [**\_ сведения о форме \_ 2**](form-info-2.md), то **аддформ** завершится ошибкой, если либо форма с указанным именем уже существует, либо значение *пкэйворд* структуры уже существует.
