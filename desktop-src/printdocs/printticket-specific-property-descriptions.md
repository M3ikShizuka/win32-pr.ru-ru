# <a name="printticket-specific-property-descriptions"></a>Описания свойств PrintTicket

Этот раздел не является актуальным. Самые актуальные сведения см. в [спецификации печати схемы](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

В этом разделе описываются общедоступные ключевые слова, которые могут отображаться в PrintTicket.

Не обязательно поддерживать все ключевые слова. Разработчики PrintTicket могут поддерживать функциональность только в том случае, если она применима к конкретной реализации.

## <a name="element-type-implementations-for-printticket-specific-property-descriptions"></a>Реализации типов элементов для описаний свойств PrintTicket

Важно обратить внимание на родительские и дочерние элементы, разрешенные в структурном содержимом для каждого ключевого слова. Дополнительные сведения см. в разделе [Сводка по типам элементов](./summary-of-element-types.md) .
