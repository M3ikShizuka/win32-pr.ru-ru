# <a name="print-document-package-api-interfaces"></a>Интерфейсы API для печати пакета документов

Следующие интерфейсы используются приложением для управления пакетом печати документа.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                       | Описание                                                                                                                                                                                                                 |
|---------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ипринтдокументпаккажестатусевент**](/windows/desktop/api/Documenttarget/nn-documenttarget-iprintdocumentpackagestatusevent)<br/>     | Представляет ход выполнения задания печати.<br/>                                                                                                                                                                        |
| [**ипринтдокументпаккажетаржет**](/windows/win32/api/documenttarget/nn-documenttarget-iprintdocumentpackagetarget)<br/>               | Позволяет пользователям перечислять Поддерживаемые типы целевых объектов пакета и создавать их с заданным ИДЕНТИФИКАТОРом типа. **Ипринтдокументпаккажетаржет** также поддерживает отслеживание процесса печати пакета и его отмены.<br/> |
| [**ипринтдокументпаккажетаржетфактори**](/windows/desktop/api/documenttarget/nn-documenttarget-iprintdocumentpackagetargetfactory)<br/> | Используется с [ипринтдокументпаккажетаржет](/windows/win32/api/documenttarget/nn-documenttarget-iprintdocumentpackagetarget) для запуска задания печати.<br/>                                                                                                               |
