# <a name="scoredproperty"></a>скоредпроперти

Этот раздел не является актуальным. Самые актуальные сведения см. в [спецификации печати схемы](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Элемент Скоредпроперти объявляет свойство, которое является встроенным для определения параметра. Такие свойства следует сравнивать при оценке того, насколько близко запрошенный параметр соответствует параметру, поддерживаемому устройством.

## <a name="element-tag"></a>Тег элемента

<ScoredProperty>

## <a name="xml-attributes"></a>XML-атрибуты

В следующей таблице перечислены атрибуты XML, которые могут быть связаны с этим элементом.



| Атрибут XML   | Сведения                                                                                                                 |
|-----------------|-------------------------------------------------------------------------------------------------------------------------|
| name<br/> | Содержит атрибут Name Скоредпроперти, стандартного свойства или закрытого свойства. <br/> |



 

Дополнительные сведения см. в разделе [XML Attributes](xml-attributes.md) .

## <a name="element-information"></a>Сведения об элементе

В следующей таблице перечислены элементы, которые могут быть родительскими для этого элемента, элементы, которые могут быть дочерними для этого элемента, и любые ограничения на сам элемент.



| Категория                   | Сведения                                                                                                                                                                  |
|----------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Родительские элементы<br/> | *Параметр*<br/> *скоредпроперти*<br/>                                                                                                                          |
| Дочерние элементы<br/>  | Можно использовать<br/> *ParameterRef* (один)<br/> или<br/> *Значение* (одно)<br/> *Свойство* (ноль или более)<br/> *Скоредпроперти* (ноль или более)<br/> |
| Этот элемент<br/>    | Символьные данные не допускаются.<br/> Дублирование дочерних одноуровневых элементов не разрешено.<br/>                                                                        |



 

## <a name="configuration-dependencies"></a>Зависимости конфигурации

Элемент Скоредпроперти не может иметь зависимостей конфигурации.

## <a name="example"></a>Пример

Объявите элемент Скоредпроперти с именем Медиасизевидс и значением 11.

``` syntax
<psf:ScoredProperty name="psk:MediaSizeWidth">
   <psf:Value xsi:type="integer">11</psf:Value>
</psf:ScoredProperty>
```
