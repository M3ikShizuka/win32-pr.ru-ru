# <a name="print-spooler-api"></a>API диспетчера очереди печати

API диспетчера очереди печати предоставляет интерфейс диспетчера очереди печати для приложений, управляющих принтерами и заданиями печати.

API диспетчера очереди печати используется приложением в рамках его программирования, а не непосредственно конечными пользователями.

| [Ссылка на асинхронную печать уведомлений](asynchronous-printing-notification.md)<br/> | Описание функций, интерфейсов и перечислений, используемых для асинхронного взаимодействия между приложениями и компонентами, размещенными в диспетчере очереди печати, такими как драйверы принтеров и мониторы портов.<br/> |
| [Справочник по установке драйвера принтера](printer-driver-installation-reference.md)<br/>     | Описание функций, устанавливающих и настроив драйверы принтеров на компьютере.<br/>                                                                                                                            |
