# <a name="parameterref-elements"></a>Элементы ParameterRef

Этот раздел не является актуальным. Самые актуальные сведения см. в [спецификации печати схемы](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Элементы ParameterRef специально применяются к элементам Option, что позволяет этому элементу Option ссылаться на определенный элемент Параметердеф. Для этих элементов Option элемент Скоредпроперти, характеризующий параметр, не жестко закодирован в документе PrintCapabilities как значение, а является переменной. Чтобы включить эту вариативность, эти элементы Скоредпроперти содержат элемент ParameterRef. Элемент Скоредпроперти не может содержать как элемент value, так и элемент ParameterRef. Элементы Option, содержащие один или несколько элементов ParameterRef, называются параметризованными элементами Option.

Платформа схемы печати позволяет элементу Option содержать любое количество элементов Скоредпроперти, ссылающихся на элементы параметров, а также любое количество элементов Скоредпроперти, содержащих элементы значения. Платформа также позволяет любому количеству элементов компонента содержать параметризованные элементы Option, а для каждого элемента Feature разрешено любое количество параметризованных элементов Option. Кроме того, одна и та же конструкция параметра может называться несколькими различными элементами Option, элементами Option, которые могут принадлежать к одному или разным функциональным элементам.
