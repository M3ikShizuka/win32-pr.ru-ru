# <a name="printer_notify_options_type-structure"></a>\_ \_ Структура типа параметров уведомления принтера \_

Структура **\_ \_ \_ типа параметров уведомления принтера** указывает набор полей принтера или сведений о задании, которые будут отслеживаться объектом уведомления об изменении принтера.

Вызов функции [**финдфирстпринтерчанженотификатион**](findfirstprinterchangenotification.md) указывает структуру [**\_ \_ параметров принтера notify**](printer-notify-options.md) , которая содержит массив **\_ \_ параметров \_ уведомлений принтера** .

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _PRINTER_NOTIFY_OPTIONS_TYPE {
  WORD  Type;
  WORD  Reserved0;
  DWORD Reserved1;
  DWORD Reserved2;
  DWORD Count;
  PWORD pFields;
} PRINTER_NOTIFY_OPTIONS_TYPE, *PPRINTER_NOTIFY_OPTIONS_TYPE;
```
