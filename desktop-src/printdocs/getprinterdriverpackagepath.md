# <a name="getprinterdriverpackagepath-function"></a>Функция Жетпринтердриверпаккажепас

Извлекает путь к указанному пакету драйвера принтера на сервере печати.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetPrinterDriverPackagePath(
  _In_    LPCTSTR pszServer,
  _In_    LPCTSTR pszEnvironment,
  _In_    LPCTSTR pszLanguage,
  _In_    LPCTSTR pszPackageID,
  _Inout_ LPTSTR  pszDriverPackageCab,
  _In_    DWORD   cchDriverPackageCab,
  _Out_   LPDWORD pcchRequiredSize
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*псзсервер* \[ окне\]
</dt> <dd>

Указатель на константу, заканчивающуюся нулем строкой, которая указывает имя сервера печати. Для локального компьютера используйте **значение NULL** .

</dd> <dt>

*псзенвиронмент* \[ окне\]
</dt> <dd>

Указатель на константу, заканчивающуюся нулем строкой, которая указывает архитектуру процессора (например, Windows NT x86). Может иметь **значение NULL**.

</dd> <dt>

*псзлангуаже* \[ окне\]
</dt> <dd>

Указатель на константу, заканчивающуюся нулем строкой, которая указывает язык [многоязычного пользовательского интерфейса](/windows/desktop/Intl/mui-resource-management) для устанавливаемого драйвера. Может иметь **значение NULL**.

</dd> <dt>

*псзпаккажеид* \[ окне\]
</dt> <dd>

Указатель на константу, заканчивающуюся нулем строкой, которая указывает идентификатор пакета драйвера.

</dd> <dt>

*псздриверпаккажекаб* \[ в, out\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает путь к CAB-файлу для пакета драйверов. Может иметь **значение NULL**. См. заметки.

</dd> <dt>

*кчдриверпаккажекаб* \[ окне\]
</dt> <dd>

Размер (в символах) буфера *псздриверпаккажекаб* . Может иметь **значение NULL**.

</dd> <dt>

*пкчрекуиредсизе* \[ заполняет\]
</dt> <dd>

Указатель на требуемый размер буфера *псздриверпаккажекаб* .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если операция выполнена успешно, возвращается значение S \_ ОК, в противном случае **HRESULT** будет содержать код ошибки.

Дополнительные сведения о кодах ошибок COM см. в разделе [Обработка ошибок](../com/error-handling-in-com.md).

## <a name="remarks"></a>Комментарии

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.

 

Чтобы получить значение для *кчдриверпаккажекаб*, вызовите функцию со значением **null** в качестве значения параметра *псздриверпаккажекаб*. Используйте значение, возвращенное в *пкчрекуиредсизе* , как значение *кчдриверпаккажекаб* , и снова вызовите функцию.

*Псзпаккажеид* обычно получается из вызова [**жеткорепринтердриверс**](getcoreprinterdrivers.md).
