# <a name="getprinterdataex-function"></a>Функция Жетпринтердатаекс

Функция **жетпринтердатаекс** получает данные конфигурации для указанного принтера или сервера печати. **Жетпринтердатаекс** может извлекать значения, хранимые в функции [**сетпринтердата**](setprinterdata.md) . Кроме того, **жетпринтердатаекс** может извлекать значения, хранимые в функции [**сетпринтердатаекс**](setprinterdataex.md) в указанном ключе.

## <a name="syntax"></a>Синтаксис


```C++
DWORD GetPrinterDataEx(
  _In_  HANDLE  hPrinter,
  _In_  LPCTSTR pKeyName,
  _In_  LPCTSTR pValueName,
  _Out_ LPDWORD pType,
  _Out_ LPBYTE  pData,
  _In_  DWORD   nSize,
  _Out_ LPDWORD pcbNeeded
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хпринтер* \[ окне\]
</dt> <dd>

Маркер принтера или сервера печати, для которого функция получает данные конфигурации. Используйте функцию [**опенпринтер**](openprinter.md), [**OpenPrinter2**](openprinter2.md)или [**аддпринтер**](addprinter.md) для получения маркера принтера.

</dd> <dt>

*пкэйнаме* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает ключ, содержащий извлекаемое значение. Используйте символ обратной косой черты ( \\ ) в качестве разделителя, чтобы указать путь с одним или несколькими подразделами.

Если *хпринтер* является обработчиком для принтера, *Пкэйнаме* имеет **значение NULL** или является пустой строкой, **жетпринтердатаекс** возвращает **ошибку \_ Недопустимый \_ параметр**.

Если *хпринтер* является обработчиком для сервера печати, *пкэйнаме* игнорируется.

</dd> <dt>

*пвалуенаме* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая определяет извлекаемые данные.

Для принтеров эта строка указывает имя значения под ключом *пкэйнаме* .

Для серверов печати эта строка является одной из предопределенных строк, перечисленных в следующем разделе "Примечания".

</dd> <dt>

*птипе* \[ заполняет\]
</dt> <dd>

Указатель на переменную, которая получает тип данных, хранящихся в значении. Функция возвращает тип, указанный в вызове [**сетпринтердатаекс**](setprinterdataex.md) при сохранении данных. Если сведения не нужны, этот параметр может иметь **значение NULL** . **Жетпринтердатаекс** передает *птипе* в качестве параметра *лпдвтипе* вызова функции [**процедура RegQueryValueEx**](/windows/desktop/api/winreg/nf-winreg-regqueryvalueexa) .

</dd> <dt>

*pData* \[ заполняет\]
</dt> <dd>

Указатель на буфер, который получает данные конфигурации.

</dd> <dt>

*нсизе* \[ окне\]
</dt> <dd>

Размер (в байтах) буфера, на который указывает *pData*.

</dd> <dt>

*пкбнидед* \[ заполняет\]
</dt> <dd>

Указатель на переменную, которая получает размер данных конфигурации в байтах. Если размер буфера, указанный параметром *нсизе* , слишком мал, функция возвращает **ошибку \_ больше \_ данных**, а *пкбнидед* указывает требуемый размер буфера.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается успешно, возвращается значение **ошибки \_ Success**.

Если функция завершается ошибкой, возвращается значение ошибки.

## <a name="remarks"></a>Remarks

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.
