# <a name="deleteprinterdata-function"></a>Функция Делетепринтердата

Функция **делетепринтердата** удаляет указанные данные конфигурации для принтера. Данные конфигурации принтера состоят из набора именованных и типизированных значений. Функция **делетепринтердата** удаляет одно из этих значений, указанное по имени значения.

Вызов **делетепринтердата** эквивалентен вызову функции [**делетепринтердатаекс**](deleteprinterdataex.md) с параметром *Пкэйнаме* , имеющим значение "принтердривердата".

## <a name="syntax"></a>Синтаксис


```C++
DWORD DeletePrinterData(
  _In_ HANDLE hPrinter,
  _In_ LPTSTR pValueName
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хпринтер* \[ окне\]
</dt> <dd>

Описатель принтера, данные конфигурации которого необходимо удалить. Используйте функцию [**опенпринтер**](openprinter.md) или [**аддпринтер**](addprinter.md) для получения маркера принтера.

</dd> <dt>

*пвалуенаме* \[ окне\]
</dt> <dd>

Указатель на имя удаляемого значения данных конфигурации, заканчивающееся нулем.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается успешно, возвращается значение ошибки \_ Success.

Если функция завершается ошибкой, возвращаемое значение является кодом системной ошибки.

## <a name="remarks"></a>Remarks

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.
