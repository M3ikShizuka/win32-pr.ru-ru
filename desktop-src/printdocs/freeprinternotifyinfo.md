# <a name="freeprinternotifyinfo-function"></a>Функция Фрипринтернотифинфо

Функция **фрипринтернотифинфо** освобождает выделенный системой буфер, созданный функцией [**финднекстпринтерчанженотификатион**](findnextprinterchangenotification.md) .

## <a name="syntax"></a>Синтаксис


```C++
BOOL FreePrinterNotifyInfo(
  _In_ PPRINTER_NOTIFY_INFO pPrinterNotifyInfo
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ппринтернотифинфо* \[ окне\]
</dt> <dd>

Указатель на буфер [**\_ \_ сведений об уведомлении принтера**](printer-notify-info.md) , возвращенный из вызова функции [**финднекстпринтерчанженотификатион**](findnextprinterchangenotification.md) . **Фрипринтернотифинфо** освобождает этот буфер.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена, возвращаемое значение будет ненулевым.

Если функция выполняется неудачно, возвращается нулевое значение.

## <a name="remarks"></a>Remarks

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.
