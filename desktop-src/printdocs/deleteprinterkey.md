# <a name="deleteprinterkey-function"></a>Функция Делетепринтеркэй

Функция **делетепринтеркэй** удаляет указанный ключ и все его подразделы для указанного принтера.

## <a name="syntax"></a>Синтаксис


```C++
DWORD DeletePrinterKey(
  _In_ HANDLE  hPrinter,
  _In_ LPCTSTR pKeyName
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хпринтер* \[ окне\]
</dt> <dd>

Описатель принтера, для которого функция удаляет ключ. Используйте функцию [**опенпринтер**](openprinter.md) или [**аддпринтер**](addprinter.md) для получения маркера принтера.

</dd> <dt>

*пкэйнаме* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая указывает удаляемый ключ. Используйте символ обратной косой черты ( \\ ) в качестве разделителя, чтобы указать путь с одним или несколькими подразделами.

Если *пкэйнаме* является пустой строкой (""), **делетепринтеркэй** удаляет все ключи ниже ключа верхнего уровня для принтера. Если *пкэйнаме* имеет **значение NULL**, **делетепринтеркэй** возвращает ошибку \_ Недопустимый \_ параметр.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается успешно, возвращается значение ошибки \_ Success.

Если функция завершается ошибкой, возвращаемое значение является кодом системной ошибки.

## <a name="remarks"></a>Remarks

> [!Note]  
> Это блокирующая или синхронная функция, которая может не возвращать значение немедленно. Скорость возврата этой функцией зависит от факторов времени выполнения, таких как состояние сети, Конфигурация сервера печати и факторы реализации драйвера принтера, которые трудно предсказать при написании приложения. Вызов этой функции из потока, который управляет взаимодействием с пользовательским интерфейсом, может привести к невозможности реагирования приложения.
