# <a name="user-configurable-elements"></a>Настраиваемые пользователем элементы

Этот раздел не является актуальным. Самые актуальные сведения см. в [спецификации печати схемы](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

В этом разделе описываются общедоступные ключевые слова, которые могут присутствовать в документе PrintCapabilities или PrintTicket.

Не обязательно поддерживать все ключевые слова и элементы ключевых слов. Разработчики PrintTicket и PrintCapabilities должны поддерживать функциональность только в том случае, если она применима к целевому устройству и (или) драйверу.

## <a name="element-type-implementations-for-user-configurable-elements"></a>Реализации типов элементов для настраиваемых пользователем элементов

Важно обратить внимание на родительские и дочерние элементы, разрешенные в структурном содержимом для каждого ключевого слова. Дополнительные сведения см. в разделе [Сводка по типам элементов](./summary-of-element-types.md) .
