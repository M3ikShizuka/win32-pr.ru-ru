# <a name="port-monitors"></a>Мониторы портов

После того как драйвер устройства преобразует весь файл журнала в команды необработанного устройства, файл преобразованных команд передается обратно в Диспетчер очереди печати. Диспетчер очереди отправки команд низкого уровня в монитор. Монитор порта — это библиотека DLL, которая передает команды необработанного устройства по сети, через параллельный порт или через последовательный порт на устройство принтера. Для поддержки передачи данных устройства через другие порты связи можно установить настраиваемые мониторы портов.

Используйте следующие функции для работы с мониторами портов.



| Функция                               | Описание                                                                         |
|----------------------------------------|-------------------------------------------------------------------------------------|
| [**аддмонитор**](addmonitor.md)       | Устанавливает монитор локального порта и связывает файлы конфигурации, данных и монитора. |
| [**делетемонитор**](deletemonitor.md) | Удаляет монитор портов, добавленный функцией [**аддмонитор**](addmonitor.md) .      |
| [**енуммониторс**](enummonitors.md)   | Перечисление мониторов портов, установленных на указанном сервере.                       |
