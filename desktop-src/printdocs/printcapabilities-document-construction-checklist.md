    -   <span data-ttu-id="4ee1d-116">Для нестандартных экземпляров компонентов следует определить характеристики, которые можно использовать для различения одного варианта от другого.</span><span class="sxs-lookup"><span data-stu-id="4ee1d-116">For nonstandard Feature instances, identify the characteristics that can be used to distinguish one Option from another.</span></span> <span data-ttu-id="4ee1d-117">Представьте каждую из этих характеристик с помощью элемента Скоредпроперти, а в каждом экземпляре параметра назначьте каждому Скоредпроперти значение, относящееся к этому параметру.</span><span class="sxs-lookup"><span data-stu-id="4ee1d-117">Represent each such characteristic by a ScoredProperty element, and in each Option instance, assign each ScoredProperty a Value that is specific to that Option.</span></span> <span data-ttu-id="4ee1d-118">Убедитесь в наличии достаточного числа элементов Скоредпроперти, чтобы каждый параметр для данной функции был уникальным.</span><span class="sxs-lookup"><span data-stu-id="4ee1d-118">Ensure that there are enough ScoredProperty elements so that each Option for a given Feature is unique.</span></span> <span data-ttu-id="4ee1d-119">Нестандартные экземпляры функций и параметров по своей природе являются подпортами.</span><span class="sxs-lookup"><span data-stu-id="4ee1d-119">Nonstandard Feature and Option instances are by their nature nonportable.</span></span> <span data-ttu-id="4ee1d-120">То есть другой драйвер не сможет найти эквивалентную функцию или параметр для сопоставления с нестандартным компонентом или параметром, указанным в PrintTicket, создаваемом вашим драйвером.</span><span class="sxs-lookup"><span data-stu-id="4ee1d-120">That is, another driver will not be able to find any equivalent Feature or Option to match a nonstandard Feature or Option specified in the PrintTicket that your driver creates.</span></span>
