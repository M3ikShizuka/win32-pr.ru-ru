# <a name="sensor_category_all"></a>\_Категория датчика \_ все

**Ключи свойств, определяемые платформой**

Ключи свойств, определяемые платформой для этой категории, основаны на \_ типе данных датчика \_ \_ общего \_ идентификатора GUID:

{DB5E0CF2-CF1F-4C18-B46C-D86011D62150}

| <span id="SENSOR_DATA_TYPE_TIMESTAMP"></span><span id="sensor_data_type_timestamp"></span><dl> <dt>**Датчик \_ \_Тип данных \_ timestamp**</dt> <dt>(PID = 2)</dt> </dl> | **VT \_ fileTime**<br/> Требуется для всех отчетов данных.
