# <a name="the-sensor-collection-object"></a>Объект коллекции датчиков

Объект коллекции датчиков содержит группу объектов датчика.

Иногда требуется работать с датчиками как с группой. Например, вы можете использовать диспетчер датчиков для обнаружения всех доступных датчиков движения. Объект коллекции датчиков предоставляет методы для работы с группой объектов датчика. Интерфейс [**исенсорколлектион**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection) можно использовать для вызова методов, которые подсчитывает, добавляют, удаляют и извлекают датчики в коллекции.
