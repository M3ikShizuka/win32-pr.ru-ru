# <a name="sensor_category_scanner"></a>\_сканер категории \_ датчика

\_ \_ Категория сканер категории датчика содержит датчики, предоставляющие сведения, получаемые при сканировании.

**Типы датчиков, определяемые платформой**

В эту категорию входят следующие типы датчиков, определяемые платформой.



| Тип датчика                                                                                                                                                                                                                                                                                           | Описание                                                     |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------|
| <span id="SENSOR_TYPE_BARCODE_SCANNER"></span><span id="sensor_type_barcode_scanner"></span><dl> <dt>**Датчик \_ Тип \_ \_ сканера штрихкодов**</dt> <dt>{990B3D8F-85BB-45FF-914D-998C04F372DF}</dt> </dl> | Датчики, использующие оптическое сканирование для чтения штрихкодов.<br/> |
| <span id="SENSOR_TYPE_RFID_SCANNER"></span><span id="sensor_type_rfid_scanner"></span><dl> <dt>**Датчик \_ Введите \_ \_ Сканер RFID**</dt> <dt>{44328EF5-02DD-4E8D-AD5D-9249832B2ECA}</dt> </dl>          | Датчики проверки кода с радиочастотой.<br/>                 |



**Поля данных, определяемые платформой**

Ключи свойств, определяемые платформой для этой категории, основываются на \_ \_ \_ GUID сканера типа данных датчика \_ :

{D7A59A3C-3421-44AB-8D3A-9DE8AB6C4CAE}

Эта категория включает следующие поля данных, определяемые платформой.



| Имя поля данных и идентификатор процесса                                                                                                                                                                                                                                                                     | Описание                                                            |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------|
| <span id="SENSOR_DATA_TYPE_RFID_TAG_40_BIT"></span><span id="sensor_data_type_rfid_tag_40_bit"></span><dl> <dt>**Датчик \_ \_Тип данных \_ тег технологии RFID \_ \_ 40 \_ bit**</dt> <dt>(PID = 2)</dt> </dl> | **VT \_ UI8**<br/> 40-разрядное значение тега идентификатора радиочастоты.<br/> |
