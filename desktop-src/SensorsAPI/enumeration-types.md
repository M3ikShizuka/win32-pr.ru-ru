# <a name="enumeration-types"></a>Типы перечисления

API датчика определяет следующие типы перечислений.



| Перечисление                                                                | Описание                                      |
|----------------------------------------------------------------------------|--------------------------------------------------|
| [**сенсорконнектионтипе**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorconnectiontype) | Определяет типы подключений устройств датчика.      |
| [**сенсорстате**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorstate)                   | Определяет возможные рабочие состояния датчиков. |
