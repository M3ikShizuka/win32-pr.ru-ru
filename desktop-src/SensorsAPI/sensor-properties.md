# <a name="sensor-properties"></a>Свойства датчика

Платформа датчиков и местоположений определяет константы, определяющие свойства датчиков. Производители датчиков также могут определять собственные свойства.

Платформа определяет следующие значения **PROPERTYKEY** для свойств датчика. Эти свойства доступны только для чтения, если не указано иное.

Каждое определяемое платформой свойство датчика **PROPERTYKEY** основано на общем **GUID** с именем \_ Свойства датчика \_ Common \_ GUID:

{7F8383EC-D3EC-495C-A8CF-B8BBE85C2920}.

> [!IMPORTANT]
> Не используйте это базовое значение для определения собственных ключей свойств.

 

Значения свойств, предназначенных для чтения и записи, могут быть заданы клиентским приложением. Значения свойств, определенных как статические, не должны меняться со временем. Указанные свойства должны поддерживаться датчиком.
