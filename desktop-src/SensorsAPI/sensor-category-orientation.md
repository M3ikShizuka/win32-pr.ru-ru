# <a name="sensor_category_orientation"></a>\_ориентация категории \_ датчика

Категория « \_ ориентация категории датчика» \_ содержит датчики, предоставляющие сведения о физической ориентации. Компасы обеспечивают ориентацию навигации, например, на основе магнитного Севера. Наклон или повышение инклинометерс меры. Датчики расстояния измеряют расстояние от какого бы то ни было объекта с датчиком.
