# <a name="the-sensor-data-report-object"></a>Объект отчета с данными датчика

Объект отчета с данными датчика содержит данные датчика.

Чтобы датчик был полезен, он должен предоставить осмысленные данные. Количество и частота создания данных меняются от датчика к датчику. Например, датчик, который определяет, открыта ли дверца, создаст небольшой объем **логических** данных, а датчик движения может постоянно формировать несколько элементов данных. Чтобы стандартизировать способ получения данных программой, API датчика использует объект отчета с данными датчика.

Вы можете получить доступ к данным в отчете с данными датчика через интерфейс [**исенсордатарепорт**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensordatareport) . Этот интерфейс позволяет извлекать отметку времени отчета о данных, чтобы можно было определить, является ли информация в отчете полезной. Данные можно получить двумя способами: как значение отдельного поля данных или как набор значений. Чтобы получить данные в виде отдельного значения, вызовите метод [**жетсенсорвалуе**](/windows/win32/api/sensorsapi/nf-sensorsapi-isensordatareport-getsensorvalue) . Чтобы получить несколько значений, вызовите метод [**жетсенсорвалуес**](/windows/win32/api/sensorsapi/nf-sensorsapi-isensordatareport-getsensorvalues) .

Укажите тип данных или полей данных, которые необходимо получить из отчета, используя константу **PROPERTYKEY** . Ключи свойств для полей данных типов стандартных датчиков определяются в sensors. h.
