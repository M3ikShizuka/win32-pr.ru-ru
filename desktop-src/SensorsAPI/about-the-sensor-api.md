# <a name="about-the-sensor-api"></a>Сведения об API датчика

API датчика предоставляет свои функциональные возможности через набор интерфейсов COM. API датчика предоставляет некоторые из этих интерфейсов, что означает возможность вызова методов для них. Другие интерфейсы определяются API, чтобы можно было реализовать методы, которые API-интерфейсу датчика должны предоставлять для программы уведомления о событиях.
