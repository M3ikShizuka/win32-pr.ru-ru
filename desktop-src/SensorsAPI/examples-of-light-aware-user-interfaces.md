# <a name="examples-of-light-aware-user-interfaces"></a>Примеры пользовательских интерфейсов Light-Aware

Теперь, когда вы понимаете некоторые основные принципы оптимизации пользовательского интерфейса для различных условий освещения, вы можете взглянуть на некоторые примеры того, как эта оптимизация делает различия при просмотре содержимого туризм. Следующие образы являются параллельными сравнениями ноутбуков в прямом солнечном свете.

Приведенные ниже изображения сравнивают пользовательский интерфейс с обычным ИНТЕРФЕЙСом на ноутбуке в прямом солнечном излучении. Первая фотография показывает, что содержимое обычно выглядит как туризм, когда ноутбук использует параметры экрана использования аккумулятора. На второй фотографии показано сочетание адаптивной яркости и пользовательского интерфейса с поддержкой освещения, а также способы совместной работы этих функций для повышения удобочитаемости экрана.

![нормальный пользовательский интерфейс с яркостью 40%](images/ui-unaware.png)![Пользовательский интерфейс с поддержкой светлой информации при 100% яркости](images/ui-lightaware.png)

На следующих фотографиях показана программа навигации, как видно из туризм с выключенным и включенным освещением. Содержимое непригодно для использования в туризм, тогда как содержимое на воздухе легко разборчиво. Также обратите внимание на то, как будут сведены отражения при использовании цветовой схемы «черно-белая», как показано во второй фотографии.

![Стандартный пользовательский интерфейс навигации](images/nav-photo-normal.png)![Пользовательский интерфейс навигации с поддержкой источников](images/nav-photo-contrast.png)
