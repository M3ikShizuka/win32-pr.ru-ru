# <a name="sensor_category_biometric"></a>\_БИОметрическая Категория датчика \_

\_ \_ Категория биометрических МЕТРИК категории датчика содержит датчики, предоставляющие сведения о живых существ.

В эту категорию входят следующие типы датчиков, определяемые платформой.



| Тип датчика                                                                                                                                                                                                                                                                                           | Описание                                     |

| <span id="SENSOR_TYPE_HUMAN_PROXIMITY"></span><span id="sensor_type_human_proximity"></span><dl> <dt>**Датчик \_ Введите \_ \_ близость человека**</dt> <dt>{5220DAE9-3179-4430-9F90-06266D2A34DE}</dt> </dl> | Датчики, которые определяют близость людей.<br/> |
| <span id="SENSOR_TYPE_TOUCH"></span><span id="sensor_type_touch"></span><dl> <dt>**Датчик \_ Введите \_ сенсорный**</dt> ввод <dt>{17DB3018-06C4-4F7D-81AF-9274B7599C27}</dt> </dl>                                | Сенсорные датчики.<br/>                       |
