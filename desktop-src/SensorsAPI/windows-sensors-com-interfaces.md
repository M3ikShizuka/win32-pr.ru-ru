# <a name="com-interfaces-sensor-api"></a>COM-интерфейсы (API датчика)

API датчика предоставляет следующие интерфейсы COM.



| Интерфейс                                              | Описание                                                                                                      |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**илокатионпермиссионс**](/windows/desktop/api/sensorsapi/nn-sensorsapi-ilocationpermissions)   | Предоставляет состояние системного параметра, с помощью которого пользователи могут изменять параметры расположения.                         |
| [**илогикалсенсорманажер**](/previous-versions/windows/desktop/legacy/dd318934(v=vs.85)) | Предоставляет методы, используемые авторам логических датчиков для управления подключениями логического датчика.                            |
| [**исенсор**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensor)                             | Представляет датчик.                                                                                             |
| [**исенсорколлектион**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection)         | Представляет коллекцию датчиков.                                                                              |
| [**исенсордатарепорт**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensordatareport)         | Представляет отчет о данных датчика.                                                                                 |
| [**исенсоревентс**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorevents)                 | Интерфейс обратного вызова для получения событий датчика.                                                              |
| [**исенсорманажер**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensormanager)               | Предоставляет методы для обнаружения и получения доступных датчиков и метода для запроса событий диспетчера датчиков. |
| [**исенсорманажеревентс**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensormanagerevents)   | Интерфейс обратного вызова для получения событий диспетчера датчиков.                                                      |
