# <a name="the-sensor-object"></a>Объект датчика

Объект датчика представляет определенный датчик.

API датчика представляет каждый датчик как объект датчика. Вы можете работать с определенным объектом датчика через его интерфейс [**исенсор**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensor) . Этот интерфейс позволяет выполнять следующие задачи:

-   Получите метаданные о датчике, например его идентификатор, тип или понятное имя.
-   Получение сведений о конкретных данных, которые может предоставить датчик.
-   Синхронное извлечение данных.
-   Получение сведений о состоянии, например о том, готов ли датчик к работе.
-   Укажите, какие события будут получены программой.
-   Укажите интерфейс обратного вызова, который API-интерфейс датчика может использовать для предоставления программы уведомлениям о событиях.
