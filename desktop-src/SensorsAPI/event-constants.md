# <a name="event-constants-sensorsh"></a>Константы событий (sensors. h)

датчик Windows и платформа расположения определяют константы для событий драйвера. Манфуактуререс датчика также может определять собственные константы.

**Типы событий датчика**

Платформа определяет следующие идентификаторы типов событий датчика.



| Тип события датчика                                                                                                                                                                                                                                                                                                    | Описание                                                                                                                                                                                                                                                                                                           |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="SENSOR_EVENT_ACCELEROMETER_SHAKE"></span><span id="sensor_event_accelerometer_shake"></span><dl> <dt>**Датчик \_ \_АКСЕЛЕРОМЕТР событий \_ встряхните**</dt> <dt>{825F5A94-0F48-4396-9CA0-6ECB5C99D915}</dt> </dl> | Указывает, что устройство было Шакен.<br/>                                                                                                                                                                                                                                                                      |
| <span id="SENSOR_EVENT_DATA_UPDATED"></span><span id="sensor_event_data_updated"></span><dl> <dt>**Датчик \_ \_Данные события \_ обновлены**</dt> <dt>{2ED0F2A4-0087-41D3-87DB-6773370B3C88}</dt> </dl>                      | Указывает, что новые данные доступны.<br/>                                                                                                                                                                                                                                                                      |
| <span id="SENSOR_EVENT_PROPERTY_CHANGED"></span><span id="sensor_event_property_changed"></span><dl> <dt>**Датчик \_ \_Свойство события \_ изменено**</dt> <dt>{2358F099-84C9-4D3D-90DF-C2421E2B2045}</dt> </dl>          | Указывает, что значение свойства изменилось. Проверьте интерфейс [ипортабледевицевалуес](/previous-versions//ms740012(v=vs.85)) , передаваемый через параметр *певентдата* , в значение [**oneven**](/windows/win32/api/sensorsapi/nf-sensorsapi-isensorevents-onevent), чтобы определить, какое свойство изменилось и его новое значение.<br/> |
| <span id="SENSOR_EVENT_STATE_CHANGED"></span><span id="sensor_event_state_changed"></span><dl> <dt>**Датчик \_ \_Состояние события \_ изменено**</dt> <dt>{BFD96016-6BD7-4560-AD34-F2F6607E8F81}</dt> </dl>                   | Указывает изменение операционного состояния, например \_ состояние датчика \_ Инициализация до \_ состояния датчика \_ Готово.<br/>                                                                                                                                                                                            |



**Пропертикэйс события датчика**

Ключи свойств, определяемые платформой для событий, основаны на следующем идентификаторе GUID:

{64346E30-8728-4B34-BDF6-4F52442C5C28}

Платформа датчика определяет следующие **PROPERTYKEY**, которые определяют параметры событий датчика.



| PROPERTYKEY событий датчика и PID                                                                                                                                                                                                                                                      | Описание                                                                                                                                                                         |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="SENSOR_EVENT_PARAMETER_EVENT_ID"></span><span id="sensor_event_parameter_event_id"></span><dl> <dt>**Датчик \_ \_ \_ \_ ИД события параметра события**</dt> <dt>(PID = 2)</dt> </dl> | Указывает, что значение **идентификатора GUID** в [ипортабледевицевалуес](/previous-versions//ms740012(v=vs.85)) является идентификатором типа события, например \_ \_ обновленными данными о событии датчика \_ .<br/> |
| <span id="SENSOR_EVENT_PARAMETER_STATE"></span><span id="sensor_event_parameter_state"></span><dl> <dt>**Датчик \_ \_ \_ Состояние параметра события**</dt> <dt>(PID = 3)</dt> </dl>           | Указывает, что целочисленное значение без знака в **ипортабледевицевалуес** является состоянием датчика, например, \_ состояние датчика \_ Готово.<br/>                                                  |



**Ошибка датчика Пропертикэйс**

Ключи свойств, определяемые платформой, для ошибок будут основываться на следующем GUID:

{77112BCD-FCE1-4f43-B8B8-A88256ADB4B3}

Платформа датчика резервирует этот идентификатор GUID для будущего использования.
