# <a name="cloud-sync-engines"></a>Облачные модули синхронизации

См. также [пример зеркального отражения облака](./build-a-cloud-file-sync-engine.md#cloud-mirror-sample).

Начиная с Windows 10 версии 1709, Windows предоставляет *API облачных файлов*. Этот API состоит из нескольких собственных API Win32 и WinRT, которые являются формализацией поддержки для облачных механизмов синхронизации, и выполняет такие задачи, как создание файлов заполнителей и каталогов и управление ими. Пользователи этого API обычно являются поставщиками синхронизации, а в некоторых экстентах — приложениями Windows.
