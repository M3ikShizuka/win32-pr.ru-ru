# <a name="winsat-mof-classes"></a>Классы WinSAT MOF

\[Классы WinSAT MOF могут быть изменены или недоступны для выпусков после Windows 8.1.\]

Средство оценки системы Windows (WinSAT) определяет следующий класс инструментарий управления Windows (WMI) (WMI) MOF-файл (MOF), который существует в \\ корневом \\ пространстве имен CIMV2:

-   [**Win32 \_ WinSAT**](win32-winsat.md)

Этот класс можно использовать для получения сводных сведений о последней формальной оценке.
