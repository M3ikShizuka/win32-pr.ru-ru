# <a name="dictionary"></a>Словарь

[**Ипропертистораже:: вритепропертинамес**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writepropertynames) реализуется с помощью [словаря идентификатора свойства 0](/windows/desktop/Stg/reserved-property-identifiers). Свойство с ИДЕНТИФИКАТОРом 0 недоступно через [**ипропертистораже:: реадмултипле**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-readmultiple) или [**Ипропертистораже:: вритемултипле**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writemultiple).
