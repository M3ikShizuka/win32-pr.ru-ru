# <a name="shared-access-flags"></a>Флаги общего доступа

Флаги общего доступа определяют степень доступа других приложений к объекту, открываемому приложением. Можно запретить доступ на чтение, запретить запись или запретить доступ. Можно также явно указать, что запрещен доступ к типу доступа.
