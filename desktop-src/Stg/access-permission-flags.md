# <a name="access-permission-flags"></a>Флаги разрешений доступа

Флаги разрешений доступа укажите тип доступа клиентского приложения к открытому объекту: чтение, запись или чтение и запись. Разрешение на чтение позволяет приложению считывать содержимое потока, но не записывать изменения в поток. Разрешение на запись позволяет приложению вызывать функцию, которая фиксирует изменения в хранилище объекта, но не считывает содержимое потока. Разрешение на чтение и запись позволяет приложению считывать потоки объекта или записывать в хранилище объекта.
