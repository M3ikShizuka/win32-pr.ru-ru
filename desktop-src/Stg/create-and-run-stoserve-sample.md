# <a name="create-and-run-stoserve-sample"></a>Создание и запуск примера Стосерве

Перед запуском клиента необходимо скомпилировать образец клиента и другие связанные образцы. Дополнительные сведения о построении образцов см. в разделе Создание [образцов](how-to-build-samples.md). Если вы уже создали соответствующие примеры, STOCLIEN.EXE — клиентский исполняемый файл для выполнения примера **стосерве** .

## <a name="code-tour"></a>Обзор кода

В следующей таблице перечислены файлы, относящиеся к примеру **стосерве** .



| Файлы        | Описание                                                                                                               |
|--------------|---------------------------------------------------------------------------------------------------------------------------|
| STOSERVE.TXT | Краткое описание образца                                                                                                  |
| ФАЙЛЫ     | Универсальный файл makefile для создания образца кода STOSERVE.DLL этого занятия.                                            |
| СТОСЕРВЕ. Высоты   | Включаемый файл для объявления как импортированного или определяющего экспортирования функций службы в STOSERVE.DLL.                 |
| СТОСЕРВЕ. CPP | Основной файл реализации для STOSERVE.DLL. Имеет функцию DllMain и функции сервера COM (например, DllGetClassObject). |
| СТОСЕРВЕ. DEF | Файл определения модуля. Экспортирует функции серверного корпуса.                                                             |
| СТОСЕРВЕ. RC  | Файл определения ресурса DLL для исполняемого файла.                                                                      |
| СТОСЕРВЕ. ICO | Ресурс значка для исполняемого файла.                                                                                     |
| Сервером. Высоты     | Включаемый файл для объекта серверного элемента управления C++.                                                                       |
| Сервером. CPP   | Файл реализации для объекта серверного элемента управления C++.                                                                |
| Установлен. Высоты    | Включаемый файл для COM-объектов фабрики класса сервера.                                                              |
| Установлен. CPP  | Файл реализации фабрик класса сервера.                                                                 |
| Соединиться. Высоты    | Включаемый файл для классов перечислителя точек соединения, точек соединения и перечислителей подключений.                |
| Соединиться. CPP  | Файл реализации для перечислителя точки соединения, точки соединения и объектов перечислителя подключений.         |
| Letter. Высоты      | Включаемый файл для класса кобумажного COM-объекта.                                                                        |
| Letter. CPP    | Файл реализации для класса кобумажного COM-объекта и точек соединения.                                       |
| СТОСЕРВЕ. DSP | Microsoft Visual Studio Project файл.                                                                                     |

-   Использование интерфейса [**ипаперсинк**](ipapersink-methods.md) для клиентских уведомлений.
