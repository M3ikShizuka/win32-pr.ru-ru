# <a name="count-of-properties"></a>Число свойств

Этот тип данных **DWORD** позволяет получить количество значений свойств в разделе. Набор свойств может содержать любое количество значений свойств. Читатели должны иметь возможность обрабатывать случай, когда есть нулевые свойства.
