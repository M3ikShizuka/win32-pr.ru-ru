# <a name="allocating-format-identifiers"></a>Выделение идентификаторов формата

Фмтидс создаются и представляются так же, как CLSID COM и идентификаторы интерфейсов. Чтобы создать уникальный FMTID, используйте программу UUIDGEN.EXE, входящую в® API Win32.
