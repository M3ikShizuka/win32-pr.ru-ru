# <a name="size-of-section"></a>Размер раздела

Тип данных размер раздела указывает размер раздела в байтах. Поскольку размер раздела составляет первые 4 байта, разделы можно копировать в виде массива байтов. Размер раздела всегда должен быть кратен четырем.

Например, пустой раздел, т. е. один из которых имеет нулевые свойства, будет иметь число байтов, равное восьми ( **DWORD** -числа байтов и **DWORD** счетчика свойств). Сам раздел будет содержать 8 байт: **08 00 00 00 00 00 00 00**.
