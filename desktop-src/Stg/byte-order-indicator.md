# <a name="byte-order-indicator"></a>Индикатор Byte-Order

Индикатор порядка байтов является типом данных **Word** и всегда должен содержать значение 0xFFFE. Это то же самое, что и индикатор порядка байтов Юникода. Это значение всегда записывается в Intel Byte и, таким образом, отображается в файле или потоке как 0xFE, 0xFF.
