# <a name="structures-structured-storage"></a>Структуры (структурированное хранилище)

Структуры являются главным набором элементов данных или членов, которые упрощают организационную схему для интерпретации данных или выполнения операций над ними. Структурированные структуры хранилища описаны в следующих разделах:

-   [**пропспек**](/windows/win32/api/propidlbase/ns-propidlbase-propspec)
-   [**пропвариант**](/windows/win32/api/propidlbase/ns-propidlbase-propvariant)
-   [**ремснб**](/windows/win32/api/objidl/ns-objidl-remsnb)
-   [**снб**](snb.md)
-   [**статпропсетстг**](/windows/win32/api/propidlbase/nn-propidlbase-ienumstatpropsetstg)
-   [**статпропстг**](/windows/win32/api/propidlbase/nn-propidlbase-ienumstatpropstg)
-   [**статстг**](/windows/win32/api/objidl/ns-objidl-statstg)
-   [**стгоптионс**](/windows/win32/api/coml2api/ns-coml2api-stgoptions)
-   [**сторажелайаут**](/windows/win32/api/objidl/ns-objidl-storagelayout)
