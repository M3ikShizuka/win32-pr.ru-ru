# <a name="class-identifier"></a>Идентификатор класса

Идентификатор класса (CLSID) — это класс, который может отображать и (или) предоставлять программный доступ к значениям свойств. Если такого класса нет, рекомендуется задать это значение так же, как FMTID (см. ниже). Кроме того, можно задать для этого параметра значение все нули; Однако использование FMTID обеспечивает большую гибкость в будущем.
