# <a name="storage-creation-flags"></a>Флаги создания хранилища

Флаги создания хранилища укажите, что должно делать COM, если существующий объект хранилища, поток или локкбитес имеет то же имя, что и создаваемый объект хранилища или объекта потока. По умолчанию возвращается сообщение об ошибке, а новый объект не создается. В данном вызове создания можно использовать только один из этих флагов.
