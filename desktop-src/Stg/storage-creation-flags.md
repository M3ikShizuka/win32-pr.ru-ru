# <a name="storage-creation-flags"></a>служба хранилища Флаги создания

флаги создания служба хранилища определяют, что должно делать COM, если существующий объект хранилища, поток или локкбитес имеет то же имя, что и создаваемый объект хранилища или объекта потока. По умолчанию возвращается сообщение об ошибке, а новый объект не создается. В данном вызове создания можно использовать только один из этих флагов.
