# <a name="reserved"></a>Зарезервировано

Значение типа данных **DWORD** зарезервировано для будущего использования. Для модулей записи наборов свойств это значение должно быть равно 1; читатели наборов свойств должны гарантировать, что это значение не меньше 1. Исключением из этого правила является то, что для [наборов свойств документсуммаринформатион и пользователяопределенные](the-documentsummaryinformation-and-userdefined-property-sets.md)это значение может быть равно 2.
