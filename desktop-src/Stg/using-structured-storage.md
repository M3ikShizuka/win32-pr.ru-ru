# <a name="using-structured-storage"></a>использование структурированных служба хранилища

В этом разделе описывается использование структурированного хранилища в COM. В нем описывается управление свойствами и наборами свойств. Включены два структурированных примера кода хранилища: **стоклиен** и **стосерве**.

В обоих примерах содержатся файлы учебников, содержащие инструкции по сборке и документацию по поддержке. они связаны с другим учебником, в котором содержатся общие сведения о создании примеров в модели COM.

В этом разделе содержатся следующие темы:

-   [Создание образцов](how-to-build-samples.md)
-   [структурированные примеры служба хранилища](structured-storage-samples.md)
-   [Управление свойствами](managing-properties.md)
-   [Управление наборами свойств](managing-property-sets.md)
