# <a name="enumerations-windows-update-agent"></a>Перечисления (агент Центр обновления Windows)

Перечисления в структурированном хранилище — это типы данных, которые состоят из последовательности именованных значений в заданном порядке. Перечислимый тип данных позволяет описывать член класса объектов в естественном языке. Перечисления наиболее полезны в контексте, где известны все возможные значения переменной, поэтому выражение этой переменной в языке программирования может быть выражено в словах на естественном языке.

Структурированные перечисления хранилищ описаны в следующих разделах.

-   [**LOCKTYPE**](/windows/win32/api/objidl/ne-objidl-locktype)
-   [**статфлаг**](/windows/win32/api/wtypes/ne-wtypes-statflag)
-   [**стгк**](/windows/win32/api/wtypes/ne-wtypes-stgc)
-   [**стгфмт**](/previous-versions/windows/desktop/legacy/aa380330(v=vs.85))
-   [**стгмове**](/windows/win32/api/wtypes/ne-wtypes-stgmove)
-   [**стгти**](/windows/win32/api/objidl/ne-objidl-stgty)
-   [**ПОТОКовый \_ Поиск**](/windows/win32/api/objidl/ne-objidl-stream_seek)
