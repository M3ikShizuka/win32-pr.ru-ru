# <a name="cguipaper-methods"></a>Методы Кгуипапер

Ниже приведены методы Кгуипапер. Все эти методы реализуются в ГУИПАПЕР. CPP.



| Метод                                                         | Описание                                                                                                           |
|----------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| BOOL init (HINSTANCE Хинст, HWND дескриптора HWND, TCHAR \* псзкмдлинефиле); | Инициализирует Гуипапер. Запрашивает сервер на создание объекта собумаги.                                                     |
| HRESULT Дравон (void);                                          | Документ блокирует бумагу для рисования только этим клиентом.                                                                   |
| HRESULT Дравофф (void);                                         | Разблокирует бумагу, чтобы позволить другим клиентам рисовать.                                                                         |
| HRESULT Клеарвин (void);                                        | Очищает окно просмотра, но оставляет данные рукописного ввода.                                                                           |
| HRESULT Паинтвин (void);                                        | Очищает окно и перерисовывает текущие данные рукописного ввода.                                                                     |
| HRESULT Erase (void);                                           | Удаляет текущее содержимое документа и очищает окно отображения.                                                             |
| Изменение размера HRESULT (WORD Ввидс, WORD Вхеигхт);                     | Изменяет размер окна просмотра.                                                                                           |
| HRESULT Инквидс (SHORT Нинквидс);                             | Задает текущую ширину пера для рисования.                                                                                   |
| HRESULT Инкколор (COLORREF Кринкколор);                         | Задает текущий цвет чернил для рисования.                                                                                   |
| HRESULT Инксавинг (BOOL Бинксавинг);                            | Включает и выключает рукописные данные, сохраняемые в содокументе.                                                                          |
| HRESULT Инкстарт (КРАТКОСРОЧНая nX, SHORT-Йорк);                          | Запускает последовательность рукописного ввода.                                                                                          |
| HRESULT Инкдрав (КРАТКОСРОЧНая nX, SHORT-Йорк);                           | Рисует данные последовательностей рукописного ввода.                                                                                              |
| HRESULT Инкстоп (КРАТКОСРОЧНая nX, SHORT-Йорк);                           | Останавливает последовательность рисования рукописных данных.                                                                                           |
| HRESULT Коннектпаперсинк (void);                                | Подключает объект клиента Паперсинк к источнику серверной бумаги.                                                    |
| HRESULT Дисконнектпаперсинк (void);                             | Отсоедините объект Паперсинк клиента от источника серверной бумаги.                                                |
| Загрузка HRESULT (void);                                            | Загружает данные рукописного ввода из текущего составного файла.                                                                            |
| HRESULT-сохранение (void);                                            | Сохраняет существующие данные рукописного ввода в текущем составном файле.                                                                     |
| HRESULT Асксаве (void);                                         | Проверяет, изменился ли рисунок. Если это так, отображает диалоговое окно, предлагающее пользователю сохранить изменения и ответить соответствующим образом. |
| HRESULT Open (void);                                            | Показывает общее диалоговое окно Win32. Открывает существующий составной файл бумажных данных.                                               |
| HRESULT SaveAs (void);                                          | Показывает общее диалоговое окно Win32. Сохраняет текущие данные бумаги в переименованном файле.                                              |
| COLORREF Пиккколор (void);                                      | Показывает диалоговое окно Win32 бщие. Запрашивает у пользователя выбор нового цвета пера.                                                      |



 

Метод **init** создает объект сопечатной бумаги на основе сервера и назначает элемент кгуипапер m \_ пипапер.

Методы **асксаве**, **Open**, **SaveAs** и **пиккколор** предоставляют привычное поведение графического интерфейса пользователя с помощью общих диалоговых окон Win32. Например, метод **Open** использует диалоговое окно Win32 Open File Name, чтобы попросить пользователя указать имя файла для открытия.

Методы **Load** и **Save** будут подробно рассмотрены далее в этом обзоре.

**Инксавинг**, **инкстарт**, **инкдрав** и **инкстоп** являются центральными методами для функций рисования приложения **стоклиен** . **Стоклиен** использует эти методы кгуипапер для записи, отображения и хранения интерактивных данных рисования, как это происходит при использовании пользовательского элемента управления. Они выполняют двойную роль рисования изображенного изображения в клиентском окне, а также передают данные рисования в собумагу на сервере. Собумага преобразует данные рисования в пакеты рукописных данных для хранения.
