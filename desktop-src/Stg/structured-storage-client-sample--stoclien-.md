# <a name="structured-storage-client-sample-stoclien"></a>Пример структурированного клиента хранилища (Стоклиен)

В примере **стоклиен** представлено простое приложение для рисования. Это приложение позволяет пользователю использовать мышь или планшетное устройство для рисования произвольной формы в окне клиента. Можно выбрать цвет и ширину электронного рукописного ввода, а рисунки можно сохранить в файлах.

Сборка и запуск образца **стоклиен** предоставляет необходимые сведения для понимания структуры структурированного клиента хранилища, ее использования и возможностей.
