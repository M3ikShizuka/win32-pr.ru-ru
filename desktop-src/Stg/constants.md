# <a name="constants-structured-storage"></a>Константы (структурированное хранилище)

Константы в структурированном хранилище представляют собой последовательность именованных значений. Константы позволяют описывать объекты на естественном языке.

Константы структурированного хранилища описаны в следующих разделах:

-   [Константы ПРОПСЕТФЛАГ](propsetflag-constants.md)
-   [Константы СТГМ](stgm-constants.md)
