# <a name="gluquadrictexture-function"></a>Функция Глукуадриктекстуре

Функция **глукуадриктекстуре** указывает, следует ли затекстурировать куадрикс.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI gluQuadricTexture(
   GLUquadric *quadObject,
   GLboolean  textureCoords
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*куадобжект* 
</dt> <dd>

Объект куадрик (созданный с помощью [**глуневкуадрик**](glunewquadric.md)).

</dd> <dt>

*текстурекурдс* 
</dt> <dd>

Флаг, указывающий, должны ли создаваться координаты текстуры. Допустимы следующие значения.



| Значение                                                                                                                                          | Значение                                                                    |
|------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------|
| <span id="GL_TRUE"></span><span id="gl_true"></span><dl> <dt>**значение по ГК \_**</dt> </dl>    | Создание координат текстуры.<br/>                                   |
| <span id="GL_FALSE"></span><span id="gl_false"></span><dl> <dt>**GL, \_ значение false**</dt> </dl> | Не создавайте координаты текстуры. Это значение по умолчанию.<br/> |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Функция **глукуадриктекстуре** указывает, должны ли создаваться координаты текстуры для куадрикс, отображаемого с помощью **куадобжект**.

Способ формирования координат текстуры зависит от конкретного куадрик, подготовленного к просмотру.
