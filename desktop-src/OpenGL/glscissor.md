# <a name="glscissor-function"></a>Функция ГлсЦиссор

Функция **глсЦиссор** определяет прямоугольный прямоугольник.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glScissor(
   GLint   x,
   GLint   y,
   GLsizei width,
   GLsizei height
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*x* 
</dt> <dd>

Координата x (вертикальная ось) для левого нижнего угла прямоугольной области.

</dd> <dt>

*y* 
</dt> <dd>

Координата y (горизонтальная ось) для левого нижнего угла прямоугольной области. Вместе, x и y указывают левый нижний угол прямоугольной области. Изначально (0, 0).

</dd> <dt>

*width* 
</dt> <dd>

Ширина прямоугольной рамки.

</dd> <dt>

*height* 
</dt> <dd>

Высота прямоугольной рамки. При *первом* присоединении контекста OpenGL к окну *Ширина* и *Высота* устанавливаются в размеры этого окна.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Функция **глсЦиссор** определяет прямоугольник, называемый прямоугольной рамкой, в координатах окна. Первые два параметра, *x* и *y*, указывают нижний левый угол бокса. Параметры *Width* и *Height* определяют ширину и высоту поля.

Ножницы включаются и отключаются с помощью [**гленабле**](glenable.md) и **глдисабле** с аргументом "GL- \_ ножницный \_ Тест". Пока ножницы включены, команды рисования могут изменять только пиксели, находящиеся внутри прямоугольной области. Координаты окон имеют целочисленные значения в общих углах буфера кадров пикселей, поэтому **глсЦиссор**(0, 0, 1, 1) позволяет изменять только нижний левый пиксел в окне, а **глсЦиссор**(0, 0, 0, 0) запрещает изменение во всех пикселях окна.

Если ножницы отключены, то это так, как если бы прямоугольная рамка включала все окно.

Следующие функции извлекают сведения, относящиеся к **глсЦиссор**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом GL \_ "прямоугольная распашная \_ "

[**глисенаблед**](glisenabled.md) с аргументом, выполнив \_ \_ пробное вырезание
