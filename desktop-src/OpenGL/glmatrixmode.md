# <a name="glmatrixmode-function"></a>Функция Глматриксмоде

Функция **глматриксмоде** указывает, какая матрица является текущей матрицей.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glMatrixMode(
   GLenum mode
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*mode* 
</dt> <dd>

Стек матриц, который является целевым объектом для последующих операций с матрицей. Параметр *mode* может принимать одно из трех значений.



| Значение                                                                                                                                                         | Значение                                                                         |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------|
| <span id="GL_MODELVIEW"></span><span id="gl_modelview"></span><dl> <dt>**\_МОДЕЛВИЕВ GL**</dt> </dl>    | Применяет последующие операции с матрицей к стеку матрицы моделвиев.<br/>  |
| <span id="GL_PROJECTION"></span><span id="gl_projection"></span><dl> <dt>**\_ПРОЕКЦИЯ GL**</dt> </dl> | Применяет последующие операции с матрицей к стеку матрицы проекции.<br/> |
| <span id="GL_TEXTURE"></span><span id="gl_texture"></span><dl> <dt>**\_текстура GL**</dt> </dl>          | Применяет последующие операции с матрицей к стеку матриц текстуры.<br/>    |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

| <dl> <dt>**\_недействительное \_ перечисление GL**</dt> </dl>      | *режим* не является допустимым значением.<br/>                                                                                          |
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Remarks

Функция **глматриксмоде** устанавливает текущий режим матрицы.

Следующая функция получает сведения, связанные с **глматриксмоде**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом в \_ режиме матрицы GL \_
