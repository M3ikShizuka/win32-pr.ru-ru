# <a name="opengl-error-codes"></a>Коды ошибок OpenGL

OpenGL содержит следующие коды ошибок.



| Код ошибки             | Описание                                 |
|------------------------|---------------------------------------------|
| \_недействительное \_ перечисление GL      | Аргумент **гленум** выходит за пределы допустимого диапазона.           |
| \_недопустимое \_ значение GL     | Числовой аргумент выходит за пределы допустимого диапазона.              |
| \_Недопустимая \_ Операция GL | Операция недопустима в текущем состоянии.         |
| \_переполнение стека GL \_    | Вызов функции приведет к переполнению стека.      |
| \_ \_ потеря значимости СТЕКа GL   | Вызов функции приведет к потере значимости стека.     |
| \_ \_ \_ неиспользуемая память в GL    | Недостаточно памяти для выполнения функции. |
