# <a name="glarrayelement-function"></a>Функция Гларрайелемент

Функция **гларрайелемент** указывает элементы массива, используемые для отрисовки вершины.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glArrayElement(
   GLint index
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*index* 
</dt> <dd>

Индекс в включенных массивах.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Используйте функцию **гларрайелемент** в парах [**глбегин**](glbegin.md) и [**гленд**](glend.md) , чтобы указать данные вершин и атрибутов для примитивов, линий и многоугольников. Функция **гларрайелемент** указывает данные для отдельной вершины, используя данные вершин и атрибутов, расположенные по *индексу* включенных массивов вершин.

**Гларрайелемент** можно использовать для создания примитивов путем индексирования данных вершин, а не потоковой передачи через массивы данных в порядке «первый к последнему». Поскольку **гларрайелемент** указывает только одну вершину, можно явно указать атрибуты для отдельных примитивов. Например, для каждого отдельного треугольника можно задать одну нормальную обычную.

При включении вызовов **гларрайелемент** в списках вывода необходимые данные массива, определяемые указателями на массивы и включающие значения, также добавляются в список просмотра. Указатели на массив и включение значений определяются при создании списков вывода, а не при выполнении списков вывода.

Данные статического массива можно считывать и кэшировать в любое время с помощью **гларрайелемент**. При изменении элементов статического массива без повторного указания массива результаты всех последующих вызовов **гларрайелемент** не определены.

При вызове **гларрайелемент** без предварительного вызова **гленаблеклиентстате**( \_ массив вершин \_ в GL) не происходит прорисовка, но атрибуты, соответствующие включенным массивам, изменяются. Хотя при указании массива в парах **глбегин** и **гленд** не возникает ошибок, результаты не определены.

> [!Note]  
> Функция **гларрайелемент** доступна только в OpenGL версии 1,1 или более поздней.
