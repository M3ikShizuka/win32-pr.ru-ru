# <a name="gllightf-function"></a>Функция Гллигхтф

Функция **гллигхтф** возвращает значения светлого исходного параметра.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glLightf(
   GLenum  light,
   GLenum  pname,
   GLfloat param
);
```

</dd> <dt>

## <a name="remarks"></a>Комментарии

Функция **гллигхтф** задает значение или значения отдельных параметров источника освещения. Параметр *Light* именует источник света, а — символическое имя формы \_ «GL»*i*, где 0 = <  « \_ максимальные \_ огни».

Параметр *pname* указывает один из исходных параметров источника, который опять же содержит символьное имя. Параметр *param* является либо одиночным значением, либо указателем на массив, который содержит новые значения.

Вычисление освещения включается и отключается с помощью [**гленабле**](glenable.md) и [**глдисабле**](gldisable.md) с аргументом \_ освещение GL. Если включено освещение, источники света, которые включены, вносят вклад в расчет освещения. Источник освещения *i* включен и отключен с помощью **гленабле** и **глдисабле** с аргументом GL \_ Light *i*.

Это всегда относится к нефинансовому \_ освещению *i* = GL \_ LIGHT0 + *i*.
