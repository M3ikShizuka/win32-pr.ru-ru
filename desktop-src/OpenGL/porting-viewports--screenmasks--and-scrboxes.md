# <a name="porting-viewports-screenmasks-and-scrboxes"></a>Перенос окна просмотра, Скринмаскс и Скрбоксес

Следующие функции окна просмотра IRI GL не имеют эквивалента OpenGL:

-   **решапевиевпорт**
-   **скрбокс**
-   **жетскрбокс**

С помощью функции **просмотра** IRI GL вы указываете координаты x (в пикселях) слева и справа от прямоугольника просмотра и координаты y для верхней и нижней части. Однако с помощью функции OpenGL [**глвиевпорт**](glviewport.md) вы указываете координаты x и y (в пикселях) левого нижнего угла прямоугольника окна просмотра, а также его ширину и высоту.

В следующей таблице перечислены функции окна просмотра IRI GL и их эквивалентные функции OpenGL.



| Функция IRI GL                          | Функция OpenGL                                                                                         | Значение                      |
|-------------------------------------------|---------------------------------------------------------------------------------------------------------|------------------------------|
| **окно просмотра** (слева, справа, снизу, сверху) | [**глвиевпорт**](glviewport.md) (x, y, ширина, высота)                                                | Задает окно просмотра.           |
| **попвиевпортпушвиевпорт**<br/>    | [**глпопаттриб**](glpopattrib.md)[**глпушаттриб**](glpushattrib.md) ( \_ бит окна просмотра GL \_ )<br/> | Отправляет и выводит стек.   |
| **окно просмотра**                           | [**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) ( \_ окно просмотра GL)               | Возвращает размеры окна просмотра. |
