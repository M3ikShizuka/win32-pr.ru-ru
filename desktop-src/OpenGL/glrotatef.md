# <a name="glrotatef-function"></a>Функция Глротатеф

Функция **глротатеф** Умножает текущую матрицу на матрицу вращения.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glRotatef(
   GLfloat angle,
   GLfloat x,
   GLfloat y,
   GLfloat z
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*angle* 
</dt> <dd>

Угол поворота (в градусах).

</dd> <dt>

*x* 
</dt> <dd>

Координата *x* вектора.

</dd> <dt>

*y* 
</dt> <dd>

Координата *y* вектора.

</dd> <dt>

*гармошкой* 
</dt> <dd>

Координата *z* вектора.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Функция **глротатеф** выдает матрицу, которая выполняет поворот *угла* в градусах против часовой стрелки относительно вектора от начала координат до точки (*x*, *y*, *z*).

Текущая матрица (см. [**глматриксмоде**](glmatrixmode.md)) умножается на эту матрицу вращения, при этом продукт заменяется текущей матрицей. То есть, если M — это текущая матрица, а R — матрица перевода, то M заменяется на M R.

Если режим матрицы — GL \_ моделвиев или GL \_ , то вызывается поворот всех объектов, нарисованных после **глротатеф** . Используйте [**глпушматрикс**](glpushmatrix.md) и [**глпопматрикс**](glpopmatrix.md) , чтобы сохранить и восстановить повернутую систему координат.

Следующие функции извлекают сведения, относящиеся к **глротатеф**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ режим рендеринга GL \_

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом в \_ режиме матрицы GL \_

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ МОДЕЛВИЕВ \_ Матрица GL

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ матрица проекции главной книги \_

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ \_ Матрица текстуры GL
