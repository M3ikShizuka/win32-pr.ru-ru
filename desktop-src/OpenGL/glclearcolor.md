# <a name="glclearcolor-function"></a>Функция Глклеарколор

Функция **глклеарколор** указывает четкие значения для буферов цветов.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glClearColor(
   GLclampf red,
   GLclampf green,
   GLclampf blue,
   GLclampf alpha
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*отмечен* 
</dt> <dd>

Красное значение, которое [**глклеар**](glclear.md) использует для очистки буферов цветов. Значение по умолчанию равно нулю.

</dd> <dt>

*зеленого* 
</dt> <dd>

Зеленое значение, которое [**глклеар**](glclear.md) использует для очистки буферов цветов. Значение по умолчанию равно нулю.

</dd> <dt>

*выделен* 
</dt> <dd>

Синее значение, которое [**глклеар**](glclear.md) использует для очистки буферов цветов. Значение по умолчанию равно нулю.

</dd> <dt>

*канал альфа-версии*; 
</dt> <dd>

Альфа-значение, используемое [**глклеар**](glclear.md) для очистки буферов цветов. Значение по умолчанию равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Функция **глклеарколор** задает красный, зеленый, синий и альфа-значения, используемые [**глклеар**](glclear.md) для очистки буферов цветов. Значения, заданные параметром **глклеарколор** , записываются в диапазон \[ 0, 1 \] .

Следующие функции извлекают сведения, относящиеся к функции **глклеарколор** :

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ « \_ Удаление накопленной \_ стоимости GL»

**глжет** с аргументом \_ " \_ очистить \_ значение цвета GL"
