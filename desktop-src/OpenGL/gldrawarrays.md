# <a name="gldrawarrays-function"></a>Функция Глдраваррайс

Функция **глдраваррайс** указывает несколько примитивов для отображения.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glDrawArrays(
   GLenum  mode,
   GLint   first,
   GLsizei count
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*mode* 
</dt> <dd>

Тип отображаемых примитивов. Следующие константы задают допустимые типы примитивов: \_ точки GL, \_ Линейная \_ полоса строк GL, \_ линейный \_ цикл строки, строки GL, ленту с треугольным треугольником, \_ \_ \_ \_ \_ \_ вершинный Вентилятор, треугольники GL, \_ четыре чередующихся элемента, четыре неметрических элемента \_ \_ и \_ многоугольник GL.

</dd> <dt>

*first* 
</dt> <dd>

Начальный индекс в включенных массивах.

</dd> <dt>

*count* 
</dt> <dd>

Число индексов для подготовки к просмотру.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

С помощью **глдраваррайс** можно указать несколько геометрических примитивов для отображения. Вместо вызова отдельных функций OpenGL для передачи каждой отдельной вершины, обычной или цветовой области можно указать отдельные массивы вершин, нормалей и цветов, чтобы определить последовательность примитивов (одинаковый тип) с одним вызовом **глдраваррайс**.

При вызове **глдраваррайс** *число* последовательных элементов из каждого разрешенного массива используется для создания последовательности геометрических примитивов, начиная с *первого* элемента. Параметр *mode* определяет тип создаваемого примитива и использование элементов массива для создания примитивов.

После возврата **глдраваррайс** значения атрибутов вершин, измененных **глдраваррайс** , не определены. Например, если \_ \_ включен массив цветов GL, значение текущего цвета не определено после возврата **глдраваррайс** . Атрибуты, не измененные **глдраваррайс** , остаются определенными. Если \_ массив ВЕРШИН GL \_ не включен, геометрические примитивы не создаются, но атрибуты, соответствующие включенным массивам, изменяются.

**Глдраваррайс** можно включить в списки вывода. При включении **глдраваррайс** в список отображений необходимые данные массива, определяемые указателями на массивы и включенными, создаются и записываются в список отображаемых данных. Значения указателей на массивы и включения определяются во время создания списков вывода.

Данные статического массива можно считывать в любое время. Если какие-либо статические элементы массива изменяются и массив не указан снова, результаты всех последующих вызовов **глдраваррайс** будут неопределенными.

Хотя при указании массива несколько раз в пределах пар [**глбегин**](glbegin.md) и [**гленд**](glend.md) не возникает ошибок, результаты не определены.
