# <a name="glunurbsproperty-function"></a>Функция Глунурбспроперти

Функция **глунурбспроперти** задает неравномерное рациональное свойство B-сплайна ([нурбс](using-nurbs-curves-and-surfaces.md)).

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI gluNurbsProperty(
   GLUnurbs *nobj,
   GLenum   property,
   GLfloat  value
);
```

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Используйте **глунурбспроперти** для управления свойствами, хранящимися в объекте нурбс. Эти свойства влияют на способ отрисовки кривой НУРБС.
