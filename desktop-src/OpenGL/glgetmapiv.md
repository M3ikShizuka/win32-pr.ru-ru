# <a name="glgetmapiv-function"></a>Функция Глжетмапив

Функции [**глжетмапдв**](glgetmapdv.md), [**глжетмапфв**](glgetmapfv.md)и **глжетмапив** возвращают параметры оценщика.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glGetMapiv(
   GLenum target,
   GLenum query,
   GLint  *v
);
```

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Функции **глжетмап** возвращают параметры оценщика. (Функции **glMap1** и **glMap2** определяют оценивающие.) *Целевой* параметр указывает карту, *запрос* выбирает конкретный параметр, а *v* указывает на хранилище, в котором будут возвращаться значения.

Допустимые значения для *целевого* параметра описаны в [**glMap1**](glmap1.md) и [**glMap2**](glmap2.md).

Если возникает ошибка, содержимое *v* не вносится никаких изменений.
