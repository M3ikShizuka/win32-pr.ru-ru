# <a name="glprioritizetextures-function"></a>Функция Глприоритизетекстурес

Функция **глприоритизетекстурес** задает приоритет проживания для текстур.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glPrioritizeTextures(
         GLsizei  n,
   const GLuint   *textures,
   const GLclampf *priorities
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*n* 
</dt> <dd>

Число текстур для определения приоритетов.

</dd> <dt>

*текстур* 
</dt> <dd>

Указатель на первый элемент массива, содержащий имена текстур, для которых необходимо задать приоритеты.

</dd> <dt>

*приоритеты* 
</dt> <dd>

Указатель на первый элемент массива, содержащий приоритеты текстуры. Приоритет, заданный в *элементе параметра prioritys, применяется* к текстуре, названной соответствующим элементом параметра *текстуры* .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Функция **глприоритизетекстурес** назначает приоритеты текстуры *n* , указанные в параметре *приоритеты* , с текстурами *n* , названными в параметре *текстуры* . На компьютерах с ограниченным объемом памяти текстуры OpenGL устанавливает "Рабочий набор" текстур, которые находятся в памяти текстуры. Эти текстуры могут быть привязаны к цели текстуры гораздо эффективнее, чем нерезидентные текстуры.

При указании приоритета для каждой текстуры функция **глприоритизетекстурес** позволяет определить, какие текстуры должны быть резидентными.

*Приоритеты текстур задаются* в диапазоне \[ 0,0, 1,0 \] перед присвоением. Ноль указывает на самый низкий приоритет; таким, текстуры с нулевым приоритетом, скорее всего, будут резидентными. Значение 1,0 указывает на самый высокий приоритет; таким, текстуры с приоритетом 1,0, скорее всего, будут резидентными. Однако текстуры не гарантируют резидентные, пока они не будут привязаны.

Функция **глприоритизетекстурес** игнорирует попытки определения приоритета текстуры 0 или любого названия текстуры, которое не соответствует существующей текстуре. Ни одна из функций, названных с помощью параметра *текстуры* , не должна быть привязана к цели текстуры.

Если в настоящее время привязана текстура, можно также использовать функцию [**глтекспараметер**](gltexparameter-functions.md) для установки ее приоритета. Это единственный способ задать приоритет текстуры по умолчанию.

**Глприоритизетекстурес** можно включить в списки вывода.

Следующая функция получает приоритет текстуры, привязанной к текущему моменту, связанной с **глприоритизетекстурес**:

-   [**глжеттекспараметер**](glgettexparameter.md) с именем параметра \_ приоритет текстуры \_ GL

> [!Note]  
> Функция **глприоритизетекстурес** доступна только в OpenGL версии 1,1 или более поздней.
