# <a name="gluloadsamplingmatrices-function"></a>Функция Глулоадсамплингматрицес

Функция **глулоадсамплингматрицес** загружает неоднородное рациональное распределение B-сплайна ([нурбс](using-nurbs-curves-and-surfaces.md)) и матрицы отбора.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI gluLoadSamplingMatrices(
         GLUnurbs *nobj,
   const GLfloat  modelMatrix[16],
   const GLfloat  projMatrix[16],
   const GLint    viewport[4]
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*нобж* 
</dt> <dd>

Объект НУРБС (созданный с помощью [**глуневнурбсрендерер**](glunewnurbsrenderer.md)).

</dd> <dt>

*моделматрикс* 
</dt> <dd>

Матрица моделвиев (как в вызове [**глжетфлоатв**](glgetfloatv.md) ).

</dd> <dt>

*прожматрикс* 
</dt> <dd>

Матрица проекции (как в вызове **глжетфлоатв** ).

</dd> <dt>

*просмотра* 
</dt> <dd>

Окно просмотра (как в вызове [**глжетинтежерв**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) ).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Функция **глулоадсамплингматрицес** использует *моделматрикс*, *прожматрикс* и *окно просмотра* , чтобы повторно вычислить матрицы выборки и отбора, хранящиеся в *нобж*. Матрица выборки определяет, насколько точно НУРБС кривую или поверхность, чтобы удовлетворить допуск выборки (как определено в \_ \_ свойстве допуска Glu выборки). Матрица отбора используется для принятия решения о том, следует ли исключать НУРБС кривую или поверхность до подготовки к просмотру (если включено свойство исходящего GLU \_ ).

Функция **глулоадсамплингматрицес** необходима только в том случае, если \_ \_ свойство МАТРИЦы автоматической загрузки Glu отключено \_ (см. [**глунурбспроперти**](glunurbsproperty.md)). Несмотря на то \_ \_ , что свойство матрицы автозагрузки Glu можно оставить \_ включенным, это требует кругового обращения к серверу OpenGL для получения текущих значений матрицы моделвиев, матрицы проекции и окна просмотра.)
