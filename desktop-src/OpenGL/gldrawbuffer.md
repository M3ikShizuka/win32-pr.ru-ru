# <a name="gldrawbuffer-function"></a>Функция Глдравбуффер

Функция **глдравбуффер** определяет, в какие буферы цветов должны быть выведены.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glDrawBuffer(
   GLenum mode
);
```

## <a name="remarks"></a>Remarks

Когда цвета записываются в буфера кадров, они записываются в буферы цветов, заданные параметром **глдравбуффер**.

Если для рисования выбрано более одного цветового буфера, то операции смешения или логических операций вычисляются и применяются независимо для каждого цветового буфера и могут формировать разные результаты в каждом буфере.

Однообластные контексты включают только левые буферы, а стереоскопик контексты включают как левые, так и правый буферы. Аналогичным образом, однобуферные контексты включают только интерфейсы переднего плана, а контексты с двойной буферизацией включают в себя как передний, так и задний буфер. Контекст выбирается при инициализации OpenGL.

Это всегда так, что GL \_ AUX *i* = GL \_ AUX0 + *i*.

Следующие функции извлекают сведения, относящиеся к функции **глдравбуффер** :

**глжет** с аргументом "дополнительные \_ \_ буферы GL"
