# <a name="assembling-primitives"></a>Сборка примитивов

После выполнения всех необходимых вычислений вершины объединяются в [примитивные](primitives.md) точки, сегменты линий или многоугольники вместе с соответствующим флагом границы, цветом и сведениями о текстурах для каждой вершины.
