# <a name="copying-an-opengl-image-to-the-clipboard"></a>Копирование изображения OpenGL в буфер обмена

Несмотря на то, что текущая версия Microsoft (OpenGL) в Windows не поддерживает буфер обмена напрямую, можно скопировать образ Windows OpenGL в буфер обмена.

**Копирование изображения OpenGL в буфер обмена**

1.  Нарисуйте изображение в битовой карте памяти или в расширенном метафайле.
2.  Скопируйте этот рисунок в буфер обмена.
