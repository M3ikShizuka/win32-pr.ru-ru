# <a name="stencil-test"></a>Тест набора элементов

Условие теста трафарета отклоняет фрагмент на основе результата сравнения значения в буфере шаблона и ссылочного значения. Функция [**глстенЦилфунк**](glstencilfunc.md) указывает функцию сравнения и ссылочное значение. Если фрагмент передается или не проходит проверку трафарета, значение в буфере шаблона изменяется в соответствии с инструкциями, указанными в [**глстенЦилоп**](glstencilop.md).
