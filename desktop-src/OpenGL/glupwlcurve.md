# <a name="glupwlcurve-function"></a>Функция Глупвлкурве

Функция **глупвлкурве** описывает кусочно-линейную кривую неоднородной кривой B-сплайна ([нурбс](using-nurbs-curves-and-surfaces.md)).

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI gluPwlCurve(
   GLUnurbs *nobj,
   GLint    count,
   GLfloat  *array,
   GLint    stride,
   GLenum   type
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*нобж* 
</dt> <dd>

Объект НУРБС (созданный с помощью [**глуневнурбсрендерер**](glunewnurbsrenderer.md)).

</dd> <dt>

*count* 
</dt> <dd>

Число точек на кривой.

</dd> <dt>

*array*. 
</dt> <dd>

Массив, содержащий точки кривой.

</dd> <dt>

*шага* 
</dt> <dd>

Смещение (число значений с плавающей запятой одиночной точности) между точками кривой.

</dd> <dt>

*type* 
</dt> <dd>

Тип кривой. Должен быть либо GLU \_ «Карта1» \_ Trim \_ 2, либо \_ Glu «Карта1» \_ Trim \_ 3.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Функция **глупвлкурве** описывает кривую линейной обрезки кусочно-для поверхности нурбс. Линейная кривая кусочно-состоит из списка координат точек в области параметров для усечения поверхности НУРБС. Эти точки связаны с сегментами линии для формирования кривой. Если кривая приближена к реальной кривой, точки должны быть близки, чтобы результирующий контур был изогнут в разрешении, используемом в приложении.

Если *тип* — Glu \_ «Карта1» \_ Trim \_ 2, он описывает кривую в двухмерном пространстве параметров (*u* и *v*). Если это GLU \_ «Карта1» \_ Trim \_ 3, то он описывает кривую в двухмерном одномерном пространстве параметров (*u*, *v* и *w*). Дополнительные сведения об усечении кривых см. в разделе [**глубегинтрим**](glubegintrim.md).
