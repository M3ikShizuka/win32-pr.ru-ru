# <a name="glgetcolortableext-function"></a>Функция Глжетколортабликст

Функция **глжетколортабликст** получает данные цветовой таблицы для текущей целевой палитры текстуры.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glGetColorTableEXT(
         GLenum target,
         GLenum format,
         GLenum type,
   const GLvoid *data
);
```

## <a name="remarks"></a>Комментарии

Функция **глжетколортабликст** возвращает фактические данные таблицы цветов, указанные в [**глколортабликст**](glcolortableext.md) и [**глколорсубтабликст**](glcolorsubtableext.md).

Функция **глжетколортабликст** — это функция расширения, которая не является частью стандартной библиотеки OpenGL, но является частью расширения текстуры с помощью \_ палитры типа "Расш \_ \_ . Чтобы проверить, поддерживает ли ваша реализация OpenGL **глжетколортабликст**, вызовите [**глжетстринг**](glgetstring.md)( \_ расширения GL). Если он возвращает \_ \_ текстуру с Многоцветовой палитрой GL \_ , поддерживается **глжетколортабликст** . Чтобы получить адрес функции расширения, вызовите [**вглжетпрокаддресс**](/windows/desktop/api/wingdi/nf-wingdi-wglgetprocaddress).
