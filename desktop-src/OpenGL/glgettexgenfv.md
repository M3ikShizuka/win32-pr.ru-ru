# <a name="glgettexgenfv-function"></a>Функция Глжеттексженфв

Функции [**глжеттексжендв**](glgettexgendv.md), **глжеттексженфв** и [**глжеттексженив**](glgettexgeniv.md) возвращают параметры создания координат текстуры.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glGetTexGenfv(
   GLenum  coord,
   GLenum  pname,
   GLfloat *params
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*курд* 
</dt> <dd>

Координата текстуры. Необходимо указать GL \_ S, GL \_ T, GL \_ R или GL \_ Q.

</dd> <dt>

*pname* 
</dt> <dd>

Символическое имя возвращаемых значений. Должен быть либо \_ \_ \_ режимом формирования текстуры GL, либо именем одного из уравнений плоскости создания текстуры: \_ «плоскость объектов GL» или «плоскость «планового \_ \_ взгляда \_ ». Ниже приведены эти значения.



| Значение                                                                                                                                                                             | Значение                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="GL_TEXTURE_GEN_MODE"></span><span id="gl_texture_gen_mode"></span><dl> <dt>**\_ \_ режим ГЕНЕРАЦИИ текстуры \_ GL**</dt> </dl> | Параметр *params* возвращает функцию создания текстуры с одним значением, символьную константу.<br/>                                                                                                                                                                                                                                                                                                                                                            |
| <span id="GL_OBJECT_PLANE"></span><span id="gl_object_plane"></span><dl> <dt>**\_плоскость объектов \_ GL**</dt> </dl>              | Параметр *params* возвращает четыре коэффициента уравнений плоскости, определяющие создание объектных линейных координат. При запросе целочисленные значения сопоставляются непосредственно из внутреннего представления с плавающей запятой.<br/>                                                                                                                                                                                                                                    |
| <span id="GL_EYE_PLANE"></span><span id="gl_eye_plane"></span><dl> <dt>**\_плоскость для глаз в ГК \_**</dt> </dl>                       | Параметр *params* возвращает четыре коэффициента уравнений плоскости, которые определяют Создание линейного создания координат глаз. При запросе целочисленные значения сопоставляются непосредственно из внутреннего представления с плавающей запятой. Возвращаемые значения — это те, которые поддерживаются в координатах глаза. Они не равны значениям, заданным с помощью [**глтексжен**](gltexgen-functions.md), если только матрица моделвиев не была определена во время вызова **глтексжен** .<br/> |

*params* 
</dt> <dd>

Возвращает запрошенные данные.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

| Имя                                                                                                  | Значение                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_недействительное \_ перечисление GL**</dt> </dl>      | *курд* или *pname* не является допустимым значением.<br/>                                                                              |
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Комментарии

Функция **глжеттексжен** *возвращает в* параметрах выбранных параметров функции создания координат текстуры, указанной с помощью **глтексжен**. Параметр *курд* называет одно из координат текстуры (*s*, *t*, *r*, *q*), используя символьную константу GL \_ , GL \_ t, GL \_ r или GL \_ q.

Если возникает ошибка, содержимое *параметров* не изменяется.
