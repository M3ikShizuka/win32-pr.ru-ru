# <a name="using-antialiasing-functions"></a>Использование функций сглаживания

В следующей таблице перечислены функции сглаживания в ГК, а также их эквивалентные функции OpenGL.



| Функция IRI GL | Функция OpenGL                                      | Значение                           |
|------------------|------------------------------------------------------|-----------------------------------|
| **пнтсмус**    | [**гленабле**](glenable.md) ( \_ Smooth для точки GL \_ )   | Включает сглаживание точек.   |
| **линесмус**   | **гленабле**( \_ \_ гладкая линия GL)                     | Включает сглаживание линий.    |
| **Smooth**   | [**гленабле**](glenable.md) ( \_ \_ несглаженный многоугольник GL) | Включает сглаживание многоугольников. |



 

Используйте эквивалентные вызовы [**глдисабле**](gldisable.md) для отключения сглаживания.

В IRI GL можно управлять качеством сглаживания, вызывая:


```C++
linesmooth(SML_ON + SML_SMOOTHER);
```



OpenGL предоставляет аналогичные контролусе [**глхинт**](glhint.md):


```C++
glHint(GL_POINT_SMOOTH_HINT, hintMode); 
glHint(GL_LINE_SMOOTH_HINT, hintMode); 
glHint(GL_POLYGON_SMOOTH_HINT, hintMode);
```



где *хинтмоде* является одним из следующих:

-   \_НИЦЕСТ GL (используйте наилучшее качество сглаживания.)
-   \_Самая быстрая (используйте наиболее эффективное сглаживание).
-   \_Неа \_ GL

IRI GL также разрешает конечное исправление путем вызова:


```C++
linesmooth(SML_ON +  SML_END_CORRECT);
```



OpenGL не имеет эквивалента для этой функции.
