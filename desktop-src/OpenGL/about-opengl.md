# <a name="appendix-opengl"></a>Приложение (OpenGL)

OpenGL, изначально разработанный Silicon Graphics (СГИ) для своих графических рабочих станций, позволяет приложениям создавать высококачественные цветные изображения независимо от систем с окнами, операционных систем и оборудования.

На сегодняшний день доска проверки архитектуры OpenGL (АРБ), в настоящее время отвечает за определение OpenGL. К членам АРБ относятся Встроенная графическая система, корпорация Майкрософт, Intel, IBM и корпорация Digital Equipment.

Официальным справочным документом по OpenGL версии 1 является *справочное руководство по OpenGL* на доске проверки архитектуры OpenGL (ISBN 0-201-63276-4). Официальное справочное руководством по началу изучение OpenGL версии 1 — это *Справочник по программированию OpenGL*, основными Неидер, Tom Дэвис и кирпич вау (ISBN 0-201-63274-8). Обе книги публикуются с помощью Addison-Wesley.
