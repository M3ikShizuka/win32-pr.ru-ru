# <a name="glclear-function"></a>Функция Глклеар

Функция **глклеар** очищает буферы до предустановленных значений.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glClear(
   GLbitfield mask
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*mask* 
</dt> <dd>

Побитовые операторы или для масок, которые указывают буферы для очистки. Ниже приведены четыре маски.



| Значение                                                                                                                                                                                   | Значение                                                     |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------|
| <span id="GL_COLOR_BUFFER_BIT"></span><span id="gl_color_buffer_bit"></span><dl> <dt>**\_ \_ бит буфера цвета \_ GL**</dt> </dl>       | Буферы, в настоящее время включенные для записи цвета.<br/> |
| <span id="GL_DEPTH_BUFFER_BIT"></span><span id="gl_depth_buffer_bit"></span><dl> <dt>**\_ \_ бит буфера глубины \_ GL**</dt> </dl>       | Буфер глубины.<br/>                                |
| <span id="GL_ACCUM_BUFFER_BIT"></span><span id="gl_accum_buffer_bit"></span><dl> <dt>**\_бит накопленного \_ буфера GL \_**</dt> </dl>       | Буфер накопления.<br/>                         |
| <span id="GL_STENCIL_BUFFER_BIT"></span><span id="gl_stencil_buffer_bit"></span><dl> <dt>**\_ \_ бит буфера шаблона \_ GL**</dt> </dl> | Буфер шаблона.<br/>                              |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Функция **глклеар** задает для области битплане в окне значения, ранее выбранные с помощью [**глклеарколор**](glclearcolor.md), [**глклеариндекс**](glclearindex.md), [**глклеардепс**](glcleardepth.md), [**глклеарстенЦил**](glclearstencil.md)и [**глклеараккум**](glclearaccum.md). Можно очистить несколько цветовых буферов одновременно, выбрав более одного буфера за раз с помощью [**глдравбуффер**](gldrawbuffer.md).

Тест «точка-владение», «ножницы», «дизеринг» и «buffer вритемаскс» влияют на работу **глклеар**. Ножница развязывается к снятой области. Функция **глклеар** пропускает альфа-функцию, функцию Blend, логическую операцию, набор элементов, сопоставление текстур и буфер *z*.

Функция **глклеар** принимает один аргумент (*Mask*), который является побитовым или из нескольких значений, указывающих, какой буфер нужно очистить.

Значение, на которое очищается каждый буфер, зависит от значения Clear для этого буфера.

Если буфер отсутствует, вызов **глклеар** , направленный в этот буфер, не действует.

Следующие функции извлекают сведения, относящиеся к **глклеар**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ « \_ Удаление накопленной \_ стоимости GL»

**глжет** с аргументом \_ " \_ Очистка \_ значения глубины GL"

**глжет** с аргументом \_ " \_ очистить \_ значение индекса GL"

**глжет** с аргументом \_ " \_ очистить \_ значение цвета GL"

**глжет** с аргументом \_ " \_ очистить значение трафарета GL" \_
