# <a name="gllogicop-function"></a>Функция Гллогикоп

Функция **гллогикоп** задает операцию логического пикселя для подготовки к просмотру цветового индекса.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glLogicOp(
   GLenum opcode
);
```

## <a name="remarks"></a>Комментарии

Функция **гллогикоп** задает логическую операцию, которая, если она включена, применяется между входящим индексом цвета и индексом цвета в соответствующем расположении в буфера кадров. Логическая операция включена или отключена с помощью [**гленабле**](glenable.md) и **глдисабле** с использованием символьной константы главной операции GL \_ \_ .

Параметр *кода операции* — это символьная константа, выбранная из списка ниже. В объяснении логических операций *s* представляет индекс входящего цвета, а *d* — индекс в буфера кадров. Используются стандартные операторы языка C. Как и в этих побитовых операторах, логическая операция применяется независимо к каждой паре битов исходного и конечного индексов.

Операции логического пикселя не применяются к буферам цветов RGBA.

Если для рисования включено более одного буфера цветового индекса, логические операции выполняются отдельно для каждого включенного буфера с использованием содержимого этого буфера для индекса назначения (см. [**глдравбуффер**](gldrawbuffer.md)).

Параметр *кода операции* должен быть одним из 16 допустимых значений. Другие значения приводят к ошибке.

Следующие функции извлекают сведения, относящиеся к **гллогикоп**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ режим ЛОГИКи операций GL \_ \_

[**глисенаблед**](glisenabled.md) с аргументом \_ ЛОГИКа операции GL \_
