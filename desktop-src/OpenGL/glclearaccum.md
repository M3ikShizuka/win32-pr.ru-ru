# <a name="glclearaccum-function"></a>Функция Глклеараккум

Функция **глклеараккум** указывает четкие значения для буфера накопления.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glClearAccum(
   GLfloat red,
   GLfloat green,
   GLfloat blue,
   GLfloat alpha
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*отмечен* 
</dt> <dd>

Значение красного цвета, используемое при очистке буфера накопления. Значение по умолчанию равно нулю.

</dd> <dt>

*зеленого* 
</dt> <dd>

Зеленое значение, используемое при очистке буфера накопления. Значение по умолчанию равно нулю.

</dd> <dt>

*выделен* 
</dt> <dd>

Значение синего цвета, используемое при очистке буфера накопления. Значение по умолчанию равно нулю.

</dd> <dt>

*буквы* 
</dt> <dd>

Альфа-значение, используемое при очистке буфера накопления. Значение по умолчанию равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

| Имя                                                                                                  | Значение                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Комментарии

Функция **глклеараккум** задает красный, зеленый, синий и альфа-значения, используемые [**глклеар**](glclear.md) для очистки буфера накопления.

Значения, заданные параметром **глклеараккум** , записываются в диапазон \[ 1, 1 \] .

Следующая функция получает сведения, связанные с **глклеараккум**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ « \_ Удаление накопленной \_ стоимости GL»
