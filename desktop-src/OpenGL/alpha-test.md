# <a name="alpha-test"></a>Альфа-тест

Альфа-тест (выполняется только в режиме RGBA) отбрасывает фрагмент в зависимости от результата сравнения альфа-значения фрагмента и константного значения ссылки. Вы указываете функцию сравнения и ссылочное значение с помощью [**глалфафунк**](glalphafunc.md).
