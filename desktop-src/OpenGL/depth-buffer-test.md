# <a name="depth-buffer-test"></a>Тест буфера глубины

Тест буфера глубины отклоняет фрагмент при сбое сравнения глубины; [**глдепсфунк**](gldepthfunc.md) указывает функцию сравнения. Если включен набор элементов, результат сравнения глубины также влияет на значение обновления буфера набора элементов.
