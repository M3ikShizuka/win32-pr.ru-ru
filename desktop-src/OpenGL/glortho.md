# <a name="glortho-function"></a>Функция Глорсо

Функция **глорсо** Умножает текущую матрицу на ортогональную матрицу.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glOrtho(
   GLdouble left,
   GLdouble right,
   GLdouble bottom,
   GLdouble top,
   GLdouble zNear,
   GLdouble zFar
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*слева* 
</dt> <dd>

Координаты левой вертикальной плоскости обрезки.

</dd> <dt>

*Правильно* 
</dt> <dd>

Координаты вертикальной плоскости обрезки серигхт.

</dd> <dt>

*Нижний* 
</dt> <dd>

Координаты нижней горизонтальной плоскости отсечения.

</dd> <dt>

*В начало* 
</dt> <dd>

Координаты верхних горизонтальных планов отсечения.

</dd> <dt>

*знеар* 
</dt> <dd>

Расстояния до плоскости усечения глубины. Это расстояние является отрицательным, если плоскость находится за пределами средства просмотра.

</dd> <dt>

*зфар* 
</dt> <dd>

Расстояния на плоскости усечения глубины. Это расстояние является отрицательным, если плоскость находится за пределами средства просмотра.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Функция **глорсо** описывает матрицу перспективы, которая создает параллельную проекцию. Параметры (*слева*, *снизу*, *вблизи*) и (*справа*, *сверху*, *вблизи*) указывают точки на близкой плоскости, которые сопоставлены с нижним левым и верхним правым углами окна соответственно, предполагая, что глаз находится в (0, 0, 0). Параметр *FAR* задает расположение дальней плоскости. Оба *знеар* и *зфар* могут быть либо положительными, либо отрицательными. Соответствующая матрица показана на рисунке ниже.

![Схема, показывающая матрицу перспективы, которую описывает функция Глорсо.](images/ortho1.png)

where

![Уравнения, описывающие матрицу перспективы.](images/ortho2.png)

Текущая матрица умножается на эту матрицу с результатом замены текущей матрицы. То есть, если M — это текущая матрица, а O — матрица Орсо, то M заменяется на M O.

Используйте [**глпушматрикс**](glpushmatrix.md) и **глпопматрикс** для сохранения и восстановления текущего стека матриц. Чтобы задать текущую матрицу, используйте [**глматриксмоде**](glmatrixmode.md) .

Следующие функции извлекают сведения, относящиеся к **глорсо**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом в \_ режиме матрицы GL \_

**глжет** с аргументом \_ МОДЕЛВИЕВ \_ Матрица GL

**глжет** с аргументом \_ матрица проекции главной книги \_

**глжет** с аргументом \_ \_ Матрица текстуры GL
