# <a name="opengl-graphic-control"></a>Графический элемент управления OpenGL

OpenGL обеспечивает довольно прямое управление фундаментальными операциями двух-и трехмерной графики. Это включает в себя спецификацию таких параметров, как матрицы преобразования, коэффициенты освещения, методы сглаживания и операторы обновления пикселей. Однако он не предоставляет средств для описания или моделирования сложных геометрических объектов. Таким образом, команды OpenGL, которые вы выдаете, указывают, как должен быть создан определенный результат (какая процедура должна следовать), а не то, как именно должен выглядеть результат. Это значит, что OpenGL является фундаментальной процедурой, а не описательным. Чтобы полностью понять, как использовать OpenGL, полезно знать порядок, в котором он выполняет операции.
