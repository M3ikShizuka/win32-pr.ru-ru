# <a name="porting-spheres"></a>Перенос шарик

При переносе шарик в OpenGL учитывайте следующие моменты:

-   Нельзя управлять типами примитивов, используемых для рисования сферы. Можно управлять точностью рисования другим способом: используйте параметры срезы и стеки. Срезы — это долготу; стеки — широту.
-   Шарик рисуются по центру источника. Вместо указания местоположения, как это делается с помощью функции IRI GL **сфдрав** , перед вызовом функции [**глусфере**](glusphere.md) Glu с переводом.
-   Библиотека сфер пока недоступна для OpenGL.

В следующей таблице перечислены функции IRI GL для рисования шарик и их эквивалентные функции GLU, если они доступны.



| Функция IRI GL | Функция GLU                                 | Значение                                       |
|------------------|----------------------------------------------|-----------------------------------------------|
| **сфобж**       | [**глуневкуадрик**](glunewquadric.md)       | Создает новый объект sphere.                  |
| **сффри**      | [**глуделетекуадрик**](gludeletequadric.md) | Удаляет объект Sphere и используемую свободную память.   |
| **сфдрав**      | [**глусфере**](glusphere.md)               | Рисует сферу.                               |
| **сфмоде**      |                                              | Задает атрибуты сферы.                       |
| **сфротматрикс** |                                              | Управляет ориентацией на сферу.                  |
| **сфгнполис**   |                                              | Возвращает число многоугольников в текущей сфере. |
