# <a name="gltexgenf-function"></a>Функция Глтексженф

Управляет созданием координат текстуры.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glTexGenf(
   GLenum  coord,
   GLenum  pname,
   GLfloat param
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*курд* 
</dt> <dd>

Координата текстуры. Должно быть одним из следующих: GL \_ S, GL \_ T, GL \_ R или GL \_ Q.

</dd> <dt>

**pname** 
</dt> <dd>

Символическое имя функции создания координаты текстуры.

</dd> <dt>

*param* 
</dt> <dd>

Однозначный параметр создания текстуры, один из \_ \_ линейных объектов GL, \_ линейный глаз в ГК или геообъектная \_ \_ \_ геосхема.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Функция **глтексжен** выбирает функцию создания координат текстуры или предоставляет коэффициенты для одной из функций. Параметр *курд* называет один из координат текстуры (s, t, r, q) и должен быть одним из следующих символов: GL \_ s, GL \_ t, GL \_ r или GL \_ q. Параметр *pname* должен быть одной из трех символьных констант: \_ \_ режим генерации текстуры GL, план \_ \_ объектов GL \_ или плоскость "GL" \_ \_ . Если *pname* имеет \_ режим Gen текстура GL \_ \_ , *параметр param* задает режим, одну из \_ \_ линейных объектных шкал, \_ \_ линейного взгляда в ГК или карту- \_ сферу GL \_ . Если *pname* является либо плоскостью \_ объектной \_ плоскости \_ \_ , либо плоскостью в главной области, *параметр param* содержит коэффициенты для соответствующей функции формирования текстуры.
