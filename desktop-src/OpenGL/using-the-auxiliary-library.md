# <a name="using-the-auxiliary-library"></a>Использование вспомогательной библиотеки

Используя OpenGL, полупроводниковая графическая система (СГИ) создала вспомогательную библиотеку для написания простых примеров программ для программного кода *OpenGL*. Исходный код для вспомогательной библиотеки предоставляется с помощью пакета Microsoft Platform SDK, а также примеров OpenGL. Чтобы понять, как вспомогательная библиотека была разработана из функций и подпрограмм OpenGL, изучите исходный код. Вспомогательные функции библиотеки можно использовать в собственных программах. Описание вспомогательной библиотеки см. в разделе с *руководством по программированию OpenGL*.
