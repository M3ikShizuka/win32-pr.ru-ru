# <a name="gltranslated-function"></a>Функция Глтранслатед

Функция **глтранслатед** Умножает текущую матрицу на матрицу перевода.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glTranslated(
   GLdouble x,
   GLdouble y,
   GLdouble z
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*x* 
</dt> <dd>

Координата *x* вектора перевода.

</dd> <dt>

*y* 
</dt> <dd>

Координата *y* вектора перевода.

</dd> <dt>

*гармошкой* 
</dt> <dd>

Координата *z* вектора перевода.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Функция **глтранслатед** создает перевод, указанный в (*x*, *y*, *z*). Вектор перевода используется для расчета матрицы перевода 4x4:

![Схема, показывающая матрицу перевода 4x4, указанную x, y, z.](images/trans01.png)

Текущая матрица (см. [**глматриксмоде**](glmatrixmode.md)) умножается на эту матрицу преобразования, при этом продукт заменяется текущей матрицей. То есть, если M — это текущая матрица, а T — матрица перевода, то M заменяется на M T.

Если режим матрицы — GL \_ моделвиев или GL \_ , то будут переведены все объекты, нарисованные после **глтранслатед** . Используйте [**глпушматрикс**](glpushmatrix.md) и **глпопматрикс** для сохранения и восстановления непреобразованной системы координат.

Следующие функции извлекают сведения, относящиеся к [**глтранслатед**](gltranslate.md):

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом в \_ режиме матрицы GL \_

**глжет** с аргументом \_ МОДЕЛВИЕВ \_ Матрица GL

**глжет** с аргументом \_ матрица проекции главной книги \_

**глжет** с аргументом \_ \_ Матрица текстуры GL
