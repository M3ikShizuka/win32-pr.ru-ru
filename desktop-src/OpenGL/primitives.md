# <a name="primitives-opengl"></a>Примитивы (OpenGL)

На следующем этапе обработки примитивы преобразуются в фрагменты пикселей в следующих шагах:

-   Примитивы обрезаются соответствующим образом.
-   Необходимо выполнить соответствующие корректировки цвета и данных текстуры, а соответствующие координаты преобразуются в координаты окна.
-   Растрирование преобразует обрезанные примитивы в фрагменты пикселей.

Дополнительные сведения см. в следующих разделах:

-   [Вырезая](clipping.md)
-   [Преобразование в координаты окна](transforming-to-window-coordinates.md)
-   [Растрирование](rasterizing.md)
-   [Справочник по примитивам](primitives-reference.md)
