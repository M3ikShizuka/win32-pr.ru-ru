# <a name="glloadname-function"></a>Функция Гллоаднаме

Функция **гллоаднаме** загружает имя в стек имен.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glLoadName(
   GLuint name
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*name* 
</dt> <dd>

Имя, которое заменит верхнее значение в стеке имен.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

Функция [**глжетеррор**](glgeterror.md) может получить следующие коды ошибок.



| Имя                                                                                                  | Значение                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана, пока стек имен не был пустым.<br/>                                                                    |
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Комментарии

Функция **гллоаднаме** заставляет *имя* заменить значение в верхней части стека имен, которое изначально пустое. В режиме выбора используется стек имен, чтобы обеспечить уникальную идентификацию наборов команд отрисовки. Он состоит из упорядоченного набора целых чисел без знака.

Стек имен всегда пуст, пока режим рендеринга не является GL \_ SELECT. Вызовы **гллоаднаме** , когда режим рендеринга не является GL \_ SELECT, игнорируются.

Следующие функции извлекают сведения, относящиеся к **гллоаднаме**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ \_ Длина стека имен GL \_

**глжет** с аргументом \_ , \_ \_ глубина стека максимальных имен \_
