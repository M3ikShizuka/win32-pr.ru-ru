# <a name="glgettexenvfv-function"></a>Функция Глжеттексенвфв

Функции **глжеттексенвфв** и [**глжеттексенвив**](glgettexenviv.md) возвращают параметры среды текстуры.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glGetTexEnvfv(
   GLenum  target,
   GLenum  pname,
   GLfloat *params
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*target* 
</dt> <dd>

Среда текстуры. Должна быть текстура GL в виде \_ \_ env.

</dd> <dt>

*pname* 
</dt> <dd>

Символьное имя параметра среды текстуры. Принимаются следующие значения.



| Значение                                                                                                                                                                                | Значение                                                                                                                                                                                                                                                                                                                                              |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="GL_TEXTURE_ENV_MODE"></span><span id="gl_texture_env_mode"></span><dl> <dt>**\_режим текстуры \_ в главной модели \_**</dt> </dl>    | Параметр *params* возвращает однозначный режим текстуры в виде символьной константы.<br/>                                                                                                                                                                                                                                           |
| <span id="GL_TEXTURE_ENV_COLOR"></span><span id="gl_texture_env_color"></span><dl> <dt>**\_Цвет текстуры в главной \_ env \_**</dt> </dl> | Параметр *params* возвращает четыре значения типа Integer или с плавающей запятой, которые являются цветом среды текстуры. Целочисленные значения при запросе линейно сопоставляются с внутренним представлением с плавающей запятой, таким образом, 1,0 сопоставляется с наиболее положительным целым числом, а-1,0 сопоставляется с наиболее отрицательным целым числом.<br/> |



 

</dd> <dt>

*params* 
</dt> <dd>

Возвращает запрошенные данные.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

| Имя                                                                                                  | Значение                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_недействительное \_ перечисление GL**</dt> </dl>      | значение *Target* или *pname* не было принято.<br/>                                                                             |
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Remarks

Функция **глжеттексенв** возвращает в *параметре params* выбранные значения для среды текстуры, которая была указана с помощью [**глтексенв**](gltexenv-functions.md). *Целевой* параметр указывает среду текстуры. В настоящее время определена и поддерживается только одна Текстурная среда: текстура GL в виде \_ \_ env.

Параметр *pname* присваивает имя конкретному параметру среды текстуры.

Если возникает ошибка, содержимое *параметров* не изменяется.
