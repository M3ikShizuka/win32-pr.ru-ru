|---------------------------------|-------------------------------------------------------------------------------------------|----------------------------------------|

| **пиккселект**<br/>       | [**глрендермоде**](glrendermode.md) ( \_ Выбор GL)                                       | Переключается в режим выбора или подбора. |
| **ендпиккендселект**<br/> | [**глрендермоде**](glrendermode.md) ( \_ рендеринг GL)                                       | Переключается в режим рендеринга.            |
| **пикксизе**                    | [**глупиккматрикс**](glupickmatrix.md)[**глселектбуффер**](glselectbuffer.md)<br/> | Задает возвращаемый массив.                 |
| **инитнамес**                   | [**глинитнамес**](glinitnames.md)                                                        |                                        |
| **пушнамепопнаме**<br/>  | [**глпушнаме**](glpushname.md)[**глпопнаме**](glpopname.md)<br/>                 |                                        |
| **лоаднаме**                    | [**гллоаднаме**](glloadname.md)                                                          |                                        |
