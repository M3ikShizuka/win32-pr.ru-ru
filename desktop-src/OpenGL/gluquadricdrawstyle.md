# <a name="gluquadricdrawstyle-function"></a>Функция Глукуадрикдравстиле

Функция **глукуадрикдравстиле** задает стиль рисования, необходимый для куадрикс.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI gluQuadricDrawStyle(
   GLUquadric *quadObject,
   GLenum     drawStyle
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*куадобжект* 
</dt> <dd>

Объект куадрик (созданный с помощью [**глуневкуадрик**](glunewquadric.md)).

</dd> <dt>

*дравстиле* 
</dt> <dd>

Требуемый стиль рисования. Допустимы следующие значения.



| Значение                                                                                                                                                            | Значение                                                                                                                                                                                                                |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="GLU_FILL"></span><span id="glu_fill"></span><dl> <dt>**GLU \_ Заливка**</dt> </dl>                   | Куадрикс подготавливаются к просмотру с помощью примитивов многоугольников. Многоугольники рисуются против часовой стрелки по отношению к их нормальным значениям (как определено с помощью [**глукуадрикориентатион**](gluquadricorientation.md)).<br/> |
| <span id="GLU_LINE"></span><span id="glu_line"></span><dl> <dt>**GLU \_ строка**</dt> </dl>                   | Куадрикс подготавливаются к просмотру как набор строк.<br/>                                                                                                                                                                    |
| <span id="GLU_SILHOUETTE"></span><span id="glu_silhouette"></span><dl> <dt>**GLU \_ силуэт**</dt> </dl> | Куадрикс подготавливаются к просмотру как набор строк, за исключением того, что границы, разделяющие копланар грани, не рисуются.<br/>                                                                                                     |
| <span id="GLU_POINT"></span><span id="glu_point"></span><dl> <dt>**\_точка Glu**</dt> </dl>                | Куадрикс подготавливаются к просмотру как набор точек.<br/>                                                                                                                                                                   |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Функция **глукуадрикдравстиле** задает стиль рисования для куадрикс, отображаемого с помощью **куадобжект**.
