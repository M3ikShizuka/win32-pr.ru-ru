# <a name="glbindtexture-function"></a>Функция Глбиндтекстуре

Функция **глбиндтекстуре** позволяет создать именованную текстуру, привязанную к цели текстуры.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glBindTexture(
   GLenum target,
   GLuint texture
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*target* 
</dt> <dd>

Целевой объект, к которому привязана текстура. Должно иметь значение GL \_ текстуры \_ 1d или \_ 2D текстура GL \_ .

</dd> <dt>

*поддержки* 
</dt> <dd>

Имя текстуры; Имя текстуры в настоящее время не может использоваться.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

| Имя                                                                                                  | Значение                                                                                                                                                                                                            |
|-------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_недействительное \_ перечисление GL**</dt> </dl>      | *Целевой объект* параметра не является допустимым значением.<br/>                                                                                                                                                       |
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | *Текстура* параметра не имеет такой же размерности, что и *целевой объект*, или функция была вызвана между вызовом [**Глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Remarks

Функция **глбиндтекстуре** позволяет создать именованную текстуру. При вызове **глбиндтекстуре** с *ЦЕЛЕВЫМ объектом* \_ текстуры GL \_ 1d или \_ 2D текстуры GL \_ , а для параметра *текстура* — имя новой текстуры, которая была создана, привязывает название текстуры к соответствующей цели текстуры. Если текстура привязана к целевому объекту, то предыдущая привязка для этого целевого объекта больше не действует.

Названия текстур — это целые числа без знака с нулевым зарезервированным значением, представляющие текстуру по умолчанию для каждой цели текстуры. Названия текстур и соответствующие содержимое текстур являются локальными для общего пространства отображения, отображаемого в текущем контексте визуализации OpenGL. два контекста отрисовки используют только названия текстур, если они также совместно используют списки отображения. С помощью [**глжентекстурес**](glgentextures.md)можно создать набор новых текстурных имен.

При первой привязке текстуры предполагается размер целевого объекта текстуры. Текстура, привязанная к \_ текстуре \_ 1d, преобразуется в одномерный, а текстура, привязанная к \_ ДВУХМЕРНОЙ текстуре GL, \_ преобразуется в двухмерный. Операции, выполняемые на целевом объекте текстуры, также влияют на текстуру, привязанную к целевому объекту. При запросе к цели текстуры возвращаемое значение является состоянием текстуры, привязанной к ней. Цели текстуры становятся псевдонимами для текстур, которые в настоящее время привязаны к ним.

При привязке текстуры с помощью **глбиндтекстуре** привязка остается активной до тех пор, пока другая текстура не будет привязана к тому же целевому объекту или вы не удалите связанную текстуру с помощью функции [**глделететекстурес**](gldeletetextures.md) . После создания именованной текстуры ее можно привязать к цели текстуры с такой же размерностью, как и при необходимости.

Обычно гораздо быстрее использовать **глбиндтекстуре** для привязки существующей именованной текстуры к одному из целей текстуры, чем перезагружать изображение текстуры с помощью [**glTexImage1D**](glteximage1d.md) или [**glTexImage2D**](glteximage2d.md). Для дополнительного управления производительностью текстур используйте [**глприоритизетекстурес**](glprioritizetextures.md).

В списки вывода можно включить вызовы **глбиндтекстуре** .

> [!Note]  
> Функция **глбиндтекстуре** доступна только в OpenGL версии 1,1 или более поздней.
