# <a name="gltexparameteri-function"></a>Функция Глтекспараметери

Задает параметры текстуры.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glTexParameteri(
   GLenum target,
   GLenum pname,
   GLint  param
);
```

## <a name="remarks"></a>Комментарии

Сопоставление текстур — это метод, который применяет изображение к поверхности объекта, как если бы изображение было декал или целлофане сжимать. Изображение создается в пространстве текстуры с помощью системы координат (*s*, *t*). Текстура — это одно-или двухмерный изображение и набор параметров, определяющих, как выводятся образцы из изображения.

Функция **глтекспараметер** присваивает значение или значения в параметрах текстуры указанному в качестве pname. Целевой параметр определяет целевую текстуру, либо \_ 2D-текстура \_ 1d, либо \_ ДВУХМЕРНАЯ текстура GL \_ .

По мере того как в процессе минификации будет вычислено больше элементов текстуры, будут очевидны и менее артефакты для присвоения псевдонимов. Хотя функции, \_ присоединяемые к главной и главной \_ линейной минификации, могут быть быстрее, чем остальные четыре, они задают только один или четыре элемента текстуры для определения значения текстуры отображаемого пикселя и могут формировать моире или неоднородные переходы. Значение по умолчанию для \_ фильтра min для текстуры GL \_ \_ — это \_ Ближайшая к \_ mipmap \_ Линейная шкала.

Предположим, что текстурирование включено (путем вызова [**гленабле**](glenable.md) с аргументом GL \_ текстуры \_ 1d или \_ двухмерной текстурой GL \_ ), а \_ для фильтра "min текстур ГК \_ \_ " задана одна из функций, требующих mipmap. Если размеры изображений текстуры, определенных в данный момент (с предыдущими вызовами [**glTexImage1D**](glteximage1d.md) или [**glTexImage2D**](glteximage2d.md)), не соответствуют правильной последовательности для MIP-карты или меньше изображений текстуры, чем требуется, или набор изображений текстуры имеет разное количество компонентов текстуры, то это так, как если бы сопоставление текстур было отключено. Линейная фильтрация обращается к четырем ближайшим элементам текстуры только в двумерной текстуре. В 1-D текстурах Линейная фильтрация обращается к двум ближайшим элементам текстуры. Следующая функция получает сведения, связанные с **глтекспараметерф**, **глтекспараметери**, **глтекспараметерфв** и **глтекспараметерив**:
