# <a name="glmaterialfv-function"></a>Функция Глматериалфв

Функция **глматериалфв** задает параметры материала для модели освещения.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glMaterialfv(
         GLenum  face,
         GLenum  pname,
   const GLfloat *params
);
```

## <a name="remarks"></a>Remarks

Функция [**глматериалфв**](glmaterialf.md) присваивает значения материальным параметрам. Существует два соответствующих набора параметров материала. Первый, *внешний* набор, используется для затенения точек, линий, точечных рисунков и всех многоугольников (если двустороннее освещение отключено) или только для внешних многоугольников (если включено двустороннее освещение). Другой набор, с *обратным переходом*, используется для затенения задних многоугольников только при включении двустороннего освещения. Дополнительные сведения об односторонним и двустороннем вычислениях освещения см. в разделе [**гллигхтмодел**](gllightmodel-functions.md) .

Функция [**глматериалфв**](glmaterialf.md) принимает три аргумента. Первый, *лицевой стороной* указывает, будут ли изменены изначальные материалы, первоначальные \_ \_ \_ \_ и \_ задние плановые материалы GL. Второй параметр, *pname*, указывает, какое из нескольких параметров в одном или обоих наборах будет изменено. Третья, *param*, указывает, какое значение будет присвоено указанному параметру.

Параметры материала используются в уравнении освещения, которое при необходимости применяется к каждой вершине. Уравнение рассматривается в [**гллигхтмодел**](gllightmodel-functions.md).

Параметры материала можно обновить в любое время. В частности, [**глматериалфв**](glmaterialf.md) можно вызывать между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md). Если для каждой вершины необходимо изменить только один параметр материала, [**глколорматериал**](glcolormaterial.md) является предпочтительным по сравнению с **глматериалфв**.
