# <a name="glunurbssurface-function"></a>Функция Глунурбссурфаце

Функция **глунурбссурфаце** определяет форму неравномерного рационального постороннего прямоугольника B-сплайна ([нурбс](using-nurbs-curves-and-surfaces.md)).

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI gluNurbsSurface(
   GLUnurbs *nobj,
   GLint    sknot_count,
   float    *sknot,
   GLint    tknot_count,
   GLfloat  *tknot,
   GLint    s_stride,
   GLint    t_stride,
   GLfloat  *ctlarray,
   GLint    sorder,
   GLint    torder,
   GLenum   type
);
```

*нобж*

Объект НУРБС (созданный с помощью [**глуневнурбсрендерер**](glunewnurbsrenderer.md)).

</dd> <dt>

*\_число скнот* 
</dt> <dd>

*\_Шаг с шагом* 
</dt> <dd>

Смещение (в виде числа отдельных значений преЦисионфлоатинг точек) между последовательными контрольными точками в параметре параметрической *u* в *ктларрай*.

</dd> <dt>

*t \_ шаг*

Смещение (в одиночных преЦисионфлоатинг значениях) между последовательными контрольными точками в направлении параметрической *v* в *ктларрай*.

*ктларрай*

Массив, содержащий контрольные точки для поверхности НУРБС. Смещение между последовательными контрольными точками в направлениях *"параметрической"* и " *v* " определяется с помощью *s \_ stride* и *t \_ stride*.

Порядок поверхности НУРБС в направлении параметрической *v* . Этот порядок больше, чем в степени, поэтому у поверхности, которая является кубическим  в версии v *, имеется порядковый* номер 4.

</dd> <dt>

*type* 
</dt> <dd>

Тип поверхности. Параметр *типа* может быть любым из допустимых двух типов оценщиков (например, GL \_ MAP2 \_ Vertex \_ 3 или GL \_ MAP2 \_ Color \_ 4).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Используйте **глунурбссурфаце** в определении поверхности нурбс, чтобы описать форму поверхности нурбс (перед обрезками). Чтобы отметить начало определения поверхности НУРБС, используйте функцию [**глубегинсурфаце**](glubeginsurface.md) . Чтобы отметить конец определения поверхности НУРБС, используйте функцию [**глуендсурфаце**](gluendsurface.md) . Вызовите **глунурбссурфаце** только в определении поверхности нурбс.

Вы связываете позиционированные, текстурные и цветные координаты с поверхностью, предоставляя каждую из них как отдельную **глунурбссурфаце** между парой **глубегинсурфаце** / **глуендсурфаце** . В одной паре **глубегинсурфаце** / **глуендсурфаце** можно установить только один вызов **глунурбссурфаце** для данных о цвете, положении и текстуре. Сделайте ровно один вызов, чтобы описать расположение поверхности ( *тип* GL \_ MAP2 \_ Vertex \_ 3 или GL \_ MAP2 \_ Vertex \_ 4).

Можно обрезать НУРБС поверхность с помощью функций [**глунурбскурве**](glunurbscurve.md) и [**глупвлкурве**](glupwlcurve.md) между вызовами [**глубегинтрим**](glubegintrim.md) и [**глуендтрим**](gluendtrim.md).

**Глунурбссурфаце** с *скнот \_ Count* Кнотс в направлении *u* и *ткнот \_ Count* Кнотс в версии *v* с Orders *сордер* и *тордер* должны иметь контрольные точки (*скнот \_ Count*  - *сордер*) мултипиед by (*ткнот \_ Count*  - *тордер*).
