# <a name="clipping-opengl"></a>Обрезка (OpenGL)

Обрезка происходит в два этапа:

1.  **Просмотр отсечения для тома Клиппингаппликатион.** Сразу после сборки примитивов они обрезаются в координатах глаза по мере необходимости для всех обтравочных плоскостей, определенных с помощью [**глклипплане**](glclipplane.md). (Для OpenGL требуется поддержка по крайней мере шести плоскостей отсечения для конкретных приложений.)
2.  Примитивы преобразуются матрицей проекции в координаты клипа и обрезаются соответствующим объемом представления. Эту матрицу можно контролировать с помощью функций преобразования матрицы (см. раздел Преобразование [матрицы](matrix-transformations.md)), но обычно это задается с помощью [**глфрустум**](glfrustum.md) или [**глорсо**](glortho.md).

Точки, сегменты линии и многоугольники обрабатываются по-разному во время обрезки:

-   Точки либо сохраняются в исходном состоянии (если они находятся внутри тома обрезки), либо отменяются (если они находятся за пределами громкости).
-   Если части сегментов линий или многоугольников находятся за пределами громкости клипа, в точках обрезки создаются новые вершины.
-   Для многоугольников все границы могут быть построены между новыми вершинами, созданными в точках обрезки.
-   Для сегментов линий и многоугольников, которые обрезаются, для всех новых вершин назначается флаг границы, цвет и сведения о текстуре.
