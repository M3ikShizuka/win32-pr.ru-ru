# <a name="translating-tevdef"></a>Преобразование тевдеф

В следующем примере кода представлено определение среды диафрагмы IRI, в котором указывается \_ параметр среды декал для ТВ-вещания.


```C++
float tevprops[] = {TV_DECAL, TV_NULL}; 
 
tevdef(1, 0, tevprops);
```



и тот же код, переведенный в OpenGL:


```C++
glTexEnvfv(GL_TEXTURE_ENV, GL_TEXTURE_ENV_MODE, GL_DECAL);
```



В следующей таблице перечислены параметры окружения "Текстура" для ДИАФРАГМы и их эквиваленты параметров OpenGL.



| Параметр IRI GL     | Параметр OpenGL             |
|-----------------------|------------------------------|
| ТЕЛЕВИЗИОНная \_ модуляция          | GL для \_ модуляции                 |
| ТВ- \_ декал             | \_ДЕКАЛ GL                    |
| Телевизор \_ Blend             | ГК \_ Blend                    |
| ТВ- \_ Цвет             | \_Цвет текстуры в главной \_ env \_      |
| \_альфа-канал ТВ             | Нет прямого эквивалента OpenGL. |
| Выбор ТВ- \_ компонента \_ | Нет прямого эквивалента OpenGL. |



 

Дополнительные сведения о параметрах среды текстуры см. в разделе [**глтексенв**](gltexenv-functions.md).
