# <a name="glindexd-function"></a>Функция Глиндексд

Функция **глиндексд** задает текущий индекс цвета.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glIndexd(
   GLdouble c
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*c* 
</dt> <dd>

Новое значение для текущего индекса цвета.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Функция **глиндексд** обновляет текущий (однозначный) цветовой индекс. Он принимает один аргумент: новое значение для текущего индекса цвета.

Текущий индекс хранится в виде значения с плавающей запятой. Целочисленные значения преобразуются непосредственно в значения с плавающей запятой без специального сопоставления.

Значения индекса за пределами представимого диапазона буфера цветового индекса не обрабатываются. Однако перед тем, как индекс переносится (если он включен) и записывается в буфера кадров, он преобразуется в формат с фиксированной запятой. Все биты в целой части результирующего значения с фиксированной запятой, которые не соответствуют битам в буфера кадров, маскируются.

Текущий индекс можно обновить в любое время. В частности, **глиндексд** можно вызывать между вызовом [**глбегин**](/windows/desktop/OpenGL/glbegin) и соответствующим вызовом [**гленд**](glend.md).

Следующая функция получает сведения, связанные с **глиндексд**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ текущий \_ индекс GL
