# <a name="porting-feedback-functions"></a>Перенос функций обратной связи

При использовании IRI GL способ обработки отзывов зависит от компьютера, на котором выполняется IRI GL. OpenGL стандартизируют функции обратной связи, чтобы вы могли полагаться на согласованную обратную связь между различными аппаратными платформами. В следующей таблице перечислены функции обратной связи для IRI и их эквивалентные функции OpenGL.



| Функция IRI GL | Функция OpenGL                                                                                            | Значение                                       |
|------------------|------------------------------------------------------------------------------------------------------------|-----------------------------------------------|
| **оставлять**     | [**глрендермоде**](glrendermode.md) ( \_ обратная связь в ГК)                                                      | Переключается в режим обратной связи.                    |
| **ендфидбакк**  | [**глрендермоде**](glrendermode.md) ( \_ рендеринг GL)[**глфидбаккбуффер**](glfeedbackbuffer.md)<br/> | Переключается в режим рендеринга.                   |
| **Сылка**  | [**глпасссраугх**](glpassthrough.md)                                                                     | Помещает маркер маркера в буфер обратной связи. |
