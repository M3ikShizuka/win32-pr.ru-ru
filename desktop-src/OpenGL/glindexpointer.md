# <a name="glindexpointer-function"></a>Функция Глиндекспоинтер

Функция **глиндекспоинтер** определяет массив цветовых индексов.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glIndexPointer(
         GLenum  type,
         GLsizei stride,
   const GLvoid  *pointer
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*type* 
</dt> <dd>

Тип данных каждого цветового индекса в массиве, используя следующие символьные константы: GL \_ Short, GL \_ int, GL \_ с плавающей запятой, ГК \_ Double.

</dd> <dt>

*шага* 
</dt> <dd>

Смещение в байтах между последовательными индексами цвета. Если *шаг* равен нулю, индексы цвета жестко упакованы в массив.

</dd> <dt>

*указатель* 
</dt> <dd>

Указатель на первый цветовой индекс в массиве.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

Функция [**глжетеррор**](glgeterror.md) может получить следующие коды ошибок.



| Имя                                                                                              | Значение                                      |
|---------------------------------------------------------------------------------------------------|----------------------------------------------|
| <dl> <dt>**\_недействительное \_ перечисление GL**</dt> </dl>  | *тип* не является допустимым значением.<br/> |
| <dl> <dt>**\_недопустимое \_ значение GL**</dt> </dl> | *шаг* или *число* были отрицательными.<br/> |



## <a name="remarks"></a>Комментарии

Функция **глиндекспоинтер** задает расположение и данные массива цветовых индексов, используемых при подготовке к просмотру. Параметр *типа* задает тип данных каждого цветового индекса, а *шаг* смещения определяет смещение в байтах от одного цветового индекса к другому, позволяя использовать упаковку вершин и атрибуты в одном массиве или хранилище в отдельных массивах. В некоторых реализациях хранение вершин и атрибутов в одном массиве может быть более эффективным, чем использование отдельных массивов. Дополнительные сведения см. в разделе [**глинтерлеаведаррайс**](glinterleavedarrays.md).

Массив цветового индекса включается при указании \_ \_ константы массива индекса GL с помощью [**гленаблеклиентстате**](glenableclientstate.md). Если этот параметр включен, [**глдраваррайс**](gldrawarrays.md) и [**гларрайелемент**](glarrayelement.md) используют массив цветового индекса. По умолчанию массив цветового индекса отключен.

Невозможно включить **глиндекспоинтер** в списки вывода.

При указании массива цветового индекса с помощью **глиндекспоинтер** значения всех параметров массива цветового индекса функции сохраняются в состоянии на стороне клиента, а статические элементы массива могут быть кэшированы. Так как параметры массива цветового индекса являются состояниями на стороне клиента, их значения не сохраняются и не восстанавливаются [**глпушаттриб**](glpushattrib.md) и **глпопаттриб**.

Несмотря на то что ошибка не возникает при вызове **глиндекспоинтер** в парах [**глбегин**](glbegin.md) и **гленд** , результаты не определены.

Следующие функции извлекают сведения, относящиеся к **глиндекспоинтер**:

[**глисенаблед**](glisenabled.md) с аргументом \_ Array индекса GL \_

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ index \_ массива индекса \_ stride

**глжет** с аргументом \_ \_ число МАССИВов индексов GL \_

**глжет** с аргументом \_ \_ тип массива индексов GL \_

**глжет** с аргументом \_ \_ Размер массива \_ индекса GL

[**глжетпоинтерв**](glgetpointerv.md) с аргументом \_ \_ указатель массива \_ индекса GL
