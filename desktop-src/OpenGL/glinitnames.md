# <a name="glinitnames-function"></a>Функция Глинитнамес

Функция **глинитнамес** инициализирует стек имен.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glInitNames(void);
```



## <a name="parameters"></a>Параметры

У этой функции нет параметров.

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

С помощью функции [**глжетеррор**](glgeterror.md) можно получить следующий код ошибки.



| Имя                                                                                                  | Значение                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Remarks

Функция **глинитнамес** приводит к инициализации стека имен в пустое состояние по умолчанию. В режиме выбора используется стек имен, чтобы обеспечить уникальную идентификацию наборов команд отрисовки. Он состоит из упорядоченного набора целых чисел без знака.

Стек имен всегда пуст, пока режим рендеринга не является GL \_ SELECT. Вызовы **глинитнамес** , когда режим рендеринга не является GL \_ SELECT, игнорируются.

Следующие функции извлекают сведения, относящиеся к **глинитнамес**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ \_ Длина стека имен GL \_

**глжет** с аргументом \_ , \_ \_ глубина стека максимальных имен \_
