# <a name="glgentextures-function"></a>Функция Глжентекстурес

Функция **глжентекстурес** создает имена текстур.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glGenTextures(
   GLsizei n,
   GLuint  *textures
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*n* 
</dt> <dd>

Число создаваемых текстурных названий.

</dd> <dt>

*текстур* 
</dt> <dd>

Указатель на первый элемент массива, в котором хранятся созданные имена текстур.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

Функция [**глжетеррор**](glgeterror.md) может получить следующие коды ошибок.



| Имя                                                                                                  | Значение                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_недопустимое \_ значение GL**</dt> </dl>     | *n* было отрицательным значением.<br/>                                                                                                  |
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Remarks

Функция **глжентекстурес** возвращает *n* названий текстур в параметре *текстуры* . Названия текстур не обязательно являются непрерывным набором целых чисел, однако ни одно из возвращаемых имен не может быть использовано непосредственно перед вызовом функции **глжентекстурес** . Созданные текстуры предполагают размерность цели текстуры, к которой они сначала привязаны с помощью функции [**глбиндтекстуре**](glbindtexture.md) . Имена текстур, возвращаемые функцией **глжентекстурес** , не возвращаются при последующих вызовах **глжентекстурес** , если они не были сначала удалены путем вызова [**глделететекстурес**](gldeletetextures.md).

Невозможно включить **глжентекстурес** в списки вывода.

> [!Note]  
> Функция **глжентекстурес** доступна только в OpenGL версии 1,1 или более поздней.

 

Следующая функция получает сведения, связанные с **глжентекстурес**:

-   [**глистекстуре**](glistexture.md)
