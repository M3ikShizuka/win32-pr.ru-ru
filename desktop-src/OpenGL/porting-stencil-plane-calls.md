# <a name="porting-stencil-plane-calls"></a>Перенос вызовов плоскости набора элементов

В OpenGL вы выделяете плоскости трафаретов, запрашивая соответствующий формат пикселей с помощью OpenGL **ауксинитдисплаймоде** или [**чусепикселформат**](/windows/desktop/api/wingdi/nf-wingdi-choosepixelformat). функции. В следующей таблице перечислены функции IRI GL, которые влияют на плоскости трафаретов и их эквивалентные функции OpenGL.



| Функция IRI GL             | Функция OpenGL                                         | Значение                                                |
|------------------------------|---------------------------------------------------------|--------------------------------------------------------|
| **стенсизе**                 | **чусепикселформат**                                   |                                                        |
| **набор элементов**( **true**,...) | [**гленабле**](glenable.md) ( \_ тест шаблона GL \_ )      | Включает тесты наборов элементов.                                 |
| **набор элементов**                  | [**глстенЦилоп**](glstencilop.md)                      | Задает действия теста набора элементов.                             |
| **набор элементов**(... Func,...)  | [**глстенЦилфунк**](glstencilfunc.md)                  | Задает значение функции и ссылки для тестирования набора элементов. |
| **свритемаск**               | [**глстенЦилмаск**](glstencilmask.md)                  | Указывает, какие биты набора элементов могут быть записаны.           |
|                              | [**глклеарстенЦил**](glclearstencil.md)                | Задает значение Clear для буфера шаблона.      |
| **склеар**                   | [**глклеар**](glclear.md) ( \_ \_ бит буфера шаблона GL \_ ) |                                                        |



 

Функции сравнения наборов элементов и операции успешного и неуспешного набора элементов почти эквивалентны в OpenGL и IRI GL. Флаги функции трафарета IRI GL начинаются с «SF». Флаги OpenGL с GL. Флаги операции "пройденные/неудачные" в IRI GL начинаются с ST; Флаги OpenGL с GL.
