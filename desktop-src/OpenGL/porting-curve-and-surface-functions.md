# <a name="porting-curve-and-surface-functions"></a>Перенос функций кривой и поверхностей

OpenGL не поддерживает эквиваленты функций IRI GL для кривых и исправлений на поверхности. Вам потребуется переписать код, если он включает любой из следующих вызовов:

-   **дефбасис**
-   **курвебасис**, **курвепреЦисион**, **КРВ**, **крвн**, **ркрв**, **ркрвн** и **курвеит**
-   **патчбасис**, **патчкурвес**, **патчпреЦисион**, **Patch** и **рпатч**

-   [Перенос кривых НУРБС](porting-nurbs-curves.md)
-   [Перенос кривых обрезки](porting-trimming-curves.md)
-   [Перенос поверхностей НУРБС](porting-nurbs-surfaces.md)
