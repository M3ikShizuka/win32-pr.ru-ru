# <a name="glcalllists-function"></a>Функция Глкалллистс

Функция **глкалллистс** выполняет список списков вывода.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glCallLists(
         GLsizei n,
         GLenum  type,
   const GLvoid  *lists
);
```

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Функция **глкалллистс** приводит к тому, что каждый список экранов в списке имен передается как *списки* для выполнения. В результате функции, сохраненные в каждом списке вывода, выполняются по порядку, так же, как если бы они были вызваны без использования списка дисплеев. Имена списков вывода, которые не были определены, игнорируются.

Функция **глкалллистс** предоставляет эффективные средства для исполнения списков вывода. Параметр *n* указывает количество списков с различными форматами имен (заданные параметром *типа* ) **глкалллистс** .

Список имен списков вывода не завершается нулем. Вместо *n* указывает, сколько имен следует взять из *списков*.

Функция [**гллистбасе**](gllistbase.md) делает дополнительный уровень косвенного обращения доступным. Функция **гллистбасе** задает смещение без знака, которое добавляется к каждому имени списка вывода, указанному в *списках* перед выполнением списка отображений.

Функция **глкалллистс** может отображаться в списке отображения. Чтобы избежать возможности бесконечной рекурсии, полученной в результате вызова друг друга из списков вывода, на уровне вложенности списков отображаемых данных во время выполнения списка отображается ограничение. Это ограничение должно быть не менее 64 и зависит от реализации.

Состояние OpenGL не сохраняется и не восстанавливается в результате вызова **глкалллистс**. Поэтому изменения, внесенные в состояние OpenGL во время выполнения списков отображения, остаются после завершения выполнения. Используйте [**глпушаттриб**](glpushattrib.md), [**глпопаттриб**](glpopattrib.md), [**глпушматрикс**](glpushmatrix.md)и [**глпопматрикс**](glpopmatrix.md) , чтобы сохранить состояние OpenGL во всех вызовах **глкалллистс** .

Вы можете выполнять списки вывода между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md), если в списке отображаются только те функции, которые разрешены в этом интервале.
