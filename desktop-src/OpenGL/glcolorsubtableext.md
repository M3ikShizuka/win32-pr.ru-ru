# <a name="glcolorsubtableext-function"></a>Функция Глколорсубтабликст

Функция **глколорсубтабликст** указывает часть палитры целевой текстуры для замены.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glColorSubTableEXT(
         GLenum  target,
         GLsizei start,
         GLsizei count,
         GLenum  format,
         GLenum  type,
   const GLvoid  *data
);
```

</dd> <dt>

*start* 
</dt> <dd>

Начальная запись индекса палитры для изменяемой палитры.

</dd> <dt>

*count* 
</dt> <dd>

Число элементов индекса палитры для изменяемой палитры, начиная с *Start*. Параметр *Count* определяет диапазон изменяемых записей в индексе палитры.

## <a name="remarks"></a>Комментарии

Функция **глколорсубтабликст** указывает части палитры текущей целевой текстуры для замены. В отличие от [**глколортабликст**](glcolortableext.md), нельзя указать *целевой* параметр в качестве палитры текстуры-посредника.

> [!Note]  
> Функция **глколорсубтабликст** — это функция расширения, которая не является частью стандартной библиотеки OpenGL, но является частью расширения текстуры с помощью \_ палитры типа "Расш \_ \_ . Чтобы проверить, поддерживает ли ваша реализация OpenGL **глколорсубтабликст**, вызовите [**глжетстринг**](glgetstring.md)( \_ расширения GL). Если он возвращает \_ \_ текстуру с Многоцветовой палитрой GL \_ , поддерживается **глколорсубтабликст** . Чтобы получить адрес функции расширения, вызовите [**вглжетпрокаддресс**](/windows/desktop/api/wingdi/nf-wingdi-wglgetprocaddress).
