# <a name="pixels"></a>Перекрытых

Фрагменты преобразуются в Пиксели в буфера кадров. Буфера кадров организован в набор логических буферов, которые представляют собой цвет, глубину, трафарет и буферы накопления. Цветовой буфер сам по себе состоит из передних левый, передних прав, заднего левого, заднего право и некоторое количество вспомогательных буферов. Вы можете выдавать функции для управления этими буферами и напрямую считывать или копировать пиксели из них. (Обратите внимание, что конкретный контекст OpenGL, который вы используете, может не предоставлять все эти буферы.)

-   [Ссылки на Пиксели](pixels-reference.md)
