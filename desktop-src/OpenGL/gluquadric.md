# <a name="gluquadriccallback-function"></a>Функция Глукуадриккаллбакк

Функция **глукуадриккаллбакк** определяет обратный вызов для объекта куадрик.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI gluQuadricCallback(
   GLUquadric *qobj,
   GLenum     which,
   void (CALLBACK *fn)()
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*кобж* 
</dt> <dd>

Объект куадрик (созданный с помощью [**глуневкуадрик**](glunewquadric.md)).

</dd> <dt>

*какую* 
</dt> <dd>

Определяемый обратный вызов. Единственное допустимое значение — GLU \_ Error.



| Значение                                                                                                                                             | Значение                                                                                                                                                                                                                                                                                                 |
|---------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="GLU_ERROR"></span><span id="glu_error"></span><dl> <dt>**GLU, \_ Ошибка**</dt> </dl> | Функция **глукуадриккаллбакк** вызывается при возникновении ошибки. Его единственный аргумент имеет тип **гленум** и указывает конкретную ошибку, которая возникла. Символьные строки, описывающие эти ошибки, можно получить с помощью вызова [**глуеррорстринг**](gluerrorstring.md) .<br/> |



 

</dd> <dt>

*FN* 
</dt> <dd>

Вызываемая функция.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Используйте **глукуадриккаллбакк** для определения нового обратного вызова, который будет использоваться объектом куадрик. Если указанный обратный вызов уже определен, он заменяется. Если *fn* имеет **значение NULL**, любой существующий обратный вызов удаляется.
