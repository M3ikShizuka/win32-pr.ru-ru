# <a name="glunurbscallback-function"></a>Функция Глунурбскаллбакк

Функция **глунурбскаллбакк** определяет обратный вызов для неравномерного рационального объекта B-сплайна ([нурбс](using-nurbs-curves-and-surfaces.md)).

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI gluNurbsCallback(
   GLUnurbs *nobj,
   GLenum   which,
   void (CALLBACK *fn)()
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*нобж* 
</dt> <dd>

Объект НУРБС (созданный с помощью [**глуневнурбсрендерер**](glunewnurbsrenderer.md)).

</dd> <dt>

*какую* 
</dt> <dd>

Определяемый обратный вызов. Единственное допустимое значение — GLU \_ Error. Значение \_ ошибки Glu означает, что функция error вызывается при обнаружении ошибки. Его единственный аргумент имеет тип **гленум** и указывает конкретную ошибку, которая возникла. Существует 37 ошибок, уникальных для НУРБС, с именем GLU \_ нурбс \_ ERROR1 до Glu \_ нурбс \_ ERROR37. Символьные строки, описывающие эти ошибки, можно получить с помощью [**глуеррорстринг**](gluerrorstring.md).

</dd> <dt>

*FN* 
</dt> <dd>

Указатель на функцию обратного вызова.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Используйте **глунурбскаллбакк** для определения обратного вызова, который будет использоваться объектом нурбс. Если указанный обратный вызов уже определен, он заменяется. Если *fn* имеет **значение NULL**, то любой существующий обратный вызов удаляется.
