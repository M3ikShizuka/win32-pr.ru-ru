# <a name="glviewport-function"></a>Функция Глвиевпорт

Функция **глвиевпорт** задает окно просмотра.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glViewport(
   GLint   x,
   GLint   y,
   GLsizei width,
   GLsizei height
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*x* 
</dt> <dd>

Нижний левый угол прямоугольника окна просмотра в пикселях. Значение по умолчанию — 0,0.

</dd> <dt>

*y* 
</dt> <dd>

Нижний левый угол прямоугольника окна просмотра в пикселях. Значение по умолчанию — 0,0.

</dd> <dt>

*width* 
</dt> <dd>

Ширина окна просмотра. При первом присоединении контекста OpenGL к окну *Ширина* и *Высота* устанавливаются в размеры этого окна.

</dd> <dt>

*height* 
</dt> <dd>

Высота окна просмотра. При первом присоединении контекста OpenGL к окну *Ширина* и *Высота* устанавливаются в размеры этого окна.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

| <dl> <dt>**\_недопустимое \_ значение GL**</dt> </dl>     | Либо *Ширина* , либо *Высота* были отрицательными.<br/>                                                                                   |
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Remarks

Функция **глвиевпорт** указывает аффинное преобразование *x* и *y* из нормализованных координат устройства в координаты окна. Let (*x*<sub>ND</sub> , *y*<sub>ND</sub> ) должны быть нормализованными координатами устройства. Затем вычисляются координаты окна (*x*<sub>w</sub> , *y*<sub>w</sub> ) следующим образом:

![Уравнение, показывающее вычисления координат окна.](images/view01.png)

Ширина и высота окна просмотра автоматически заносятся в диапазон, зависящий от реализации. Запрос к этому диапазону осуществляется путем вызова **глжет** с аргументом GL \_ Max \_ окна просмотра \_ .

Следующие функции извлекают сведения, относящиеся к **глвиевпорт**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ окна просмотра аргументов GL

**глжет** с аргументом " \_ Максимальное \_ окно просмотра" \_
