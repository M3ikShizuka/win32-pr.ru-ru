# <a name="glrectd-function"></a>Функция Глректд

Функция **глректд** рисует прямоугольник.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glRectd(
   GLdouble x1,
   GLdouble y1,
   GLdouble x2,
   GLdouble y2
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*x1* 
</dt> <dd>

Координата *x* вершины прямоугольника.

</dd> <dt>

*Y1* 
</dt> <dd>

Координата *y* вершины прямоугольника.

</dd> <dt>

*штук* 
</dt> <dd>

Координата *x* противоположной вершины прямоугольника.

</dd> <dt>

*Y2* 
</dt> <dd>

Координата *y* противоположной вершины прямоугольника.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Функция **глректд** поддерживает эффективную спецификацию прямоугольников в двух угловых точках. Каждая команда Rectangle принимает четыре аргумента, организованные как две последовательные пары (*x*, *y*) или как два указателя на массивы, каждый из которых содержит пару (*x*, *y*). Полученный прямоугольник определяется в плоскости *z* = 0.

Функция **глректд**(*x1,* *Y1,* *x2,* *Y2*) в точности эквивалентна следующей последовательности:

**глбегин**( \_ многоугольник GL);

**glVertex2**( *x1,* *Y1* );

**glVertex2**( *x2,* *Y1* );

**glVertex2**( *x2,* *Y2* );

**glVertex2**( *x1,* *Y2* );

**гленд**();

Обратите внимание, что если вторая вершина находится выше и правее первой вершины, прямоугольник создается с поворотом против часовой стрелки.
