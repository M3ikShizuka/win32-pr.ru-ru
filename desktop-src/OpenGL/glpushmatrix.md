# <a name="glpushmatrix-function"></a>Функция Глпушматрикс

Функции **глпушматрикс** и [**глпопматрикс**](glpopmatrix.md) принудительно отправляют и отправляют текущий стек матриц.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glPushMatrix(void);
```



## <a name="parameters"></a>Параметры

У этой функции нет параметров.

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

Отправка полного стека матрицы или открытие стека матриц, содержащего только одну матрицу, является ошибкой. В любом случае устанавливается флаг ошибки, а в состояние OpenGL другие изменения не вносятся.

Функция [**глжетеррор**](glgeterror.md) может получить следующие коды ошибок.



| Имя                                                                                                  | Значение                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_переполнение стека GL \_**</dt> </dl>    | Функция была вызвана, пока текущий стек матриц заполнен.<br/>                                                           |
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Remarks

Существует стек матриц для каждого режима матрицы. В \_ моделвиев режиме GL глубина стека составляет не менее 32. В двух других режимах — \_ проекция и \_ текстура GL, глубина — не менее 2. Текущая матрица в любом режиме — это матрица, расположенная в верхней части стека для этого режима.

Функция **глпушматрикс** помещает текущий стек матриц вниз на единицу, что дублирует текущую матрицу. То есть после вызова **глпушматрикс** матрица, расположенная вверху стека, идентична расположенной ниже. Функция **глпопматрикс** выводит текущий стек матриц, заменяя текущую матрицу на одну под ней в стеке. Изначально каждый из стеков содержит одну матрицу, матрицу идентификации.

Следующие функции извлекают сведения, относящиеся к **глпушматрикс** и **глпопматрикс**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом в \_ режиме матрицы GL \_

**глжет** с аргументом \_ МОДЕЛВИЕВ \_ Матрица GL

**глжет** с аргументом \_ матрица проекции главной книги \_

**глжет** с аргументом \_ \_ Матрица текстуры GL

**глжет** с аргументом GL \_ моделвиев \_ Stack \_

**глжет** с аргументом \_ " \_ глубина стека главной книги GL" \_

**глжет** с аргументом \_ \_ глубина текстуры стека GL \_

**глжет** с аргументом \_ GL \_ Максимальная \_ глубина моделвиев стека \_

**глжет** с аргументом \_ Максимальная \_ \_ глубина стека проекции \_

**глжет** с аргументом \_ " \_ Максимальная \_ глубина текстуры стека \_ "
