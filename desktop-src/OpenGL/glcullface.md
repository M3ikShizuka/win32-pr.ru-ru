# <a name="glcullface-function"></a>Функция Глкуллфаце

Функция **глкуллфаце** указывает, можно ли исключать внешние или фоновые аспекты.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glCullFace(
   GLenum mode
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*mode* 
</dt> <dd>

Указывает, являются ли аспекты с внешними или фоновыми серверными кандидатами для отбора. Символьные константы GL \_ Front, GL \_ назад и GL \_ Front \_ и \_ Back принимаются. Значение по умолчанию — GL \_ назад.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Функция **глкуллфаце** указывает, исключаются ли аспекты с внешним или фоновым *режимом*(как указано в режиме) при включении отбора аспектов. Вы включаете и отключаете отбор аспектов с помощью [**гленабле**](glenable.md) и [**глдисабле**](gldisable.md) с аргументом «поверхность применения GL \_ \_ ». К аспектам относятся треугольники, куадрилатералс, многоугольники и прямоугольники.

Функция [**глфронтфаце**](glfrontface.md) указывает, какой из аспектов по часовой стрелке и против часовой стрелки является внешним и обратным.

Если *режим* — GL \_ спереди \_ и \_ назад, то никакие аспекты не рисуются, а другие примитивы, такие как точки и линии, рисуются.

Следующие функции извлекают сведения, относящиеся к **глкуллфаце**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ " \_ режим отбора" для параметра GL \_

[**глисенаблед**](glisenabled.md) с аргументом " \_ отбор, \_ лицо"
