# <a name="porting-trimming-curves"></a>Перенос кривых обрезки

Кривые с обрезкими OpenGL очень похожи на кривые с изгибом в ГК. В следующей таблице перечислены функции IRI GL для определения усеченных кривых и их эквивалентных функций OpenGL.



| Функция IRI GL | Функция OpenGL                        | Значение                              |
|------------------|----------------------------------------|--------------------------------------|
| **бгнтрим**      | [**глубегинтрим**](glubegintrim.md)   | Начинает усечение определения кривой.    |
| **пвлкурве**     | [**глупвлкурве**](glupwlcurve.md)     | Определяет линейную кривую кусочно-.    |
| **нурбскурве**   | [**глунурбскурве**](glunurbscurve.md) | Задает атрибуты кривой обрезки. |
| **ендтрим**      | [**глуендтрим**](gluendtrim.md)       | Завершает определение кривой обрезки.      |
