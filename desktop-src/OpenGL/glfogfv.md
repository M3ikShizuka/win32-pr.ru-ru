# <a name="glfogfv-function"></a>Функция Глфогфв

Функция **глфогфв** задает параметры тумана.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glFogfv(
         GLenum  pname,
   const GLfloat *params
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*pname* 
</dt> <dd>

Задает параметр тумана.

Принимает одно из следующих значений.



| Значение                                                                                                                                                             | Значение                                                                                                                                                                                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="GL_FOG_MODE"></span><span id="gl_fog_mode"></span><dl> <dt>**\_режим тумана GL \_**</dt> </dl>          | Параметр *params* — это значение с плавающей запятой, которое указывает уравнение, используемое для вычисления коэффициента смешения тумана, *f*. Допустимы три символьных константы: GL \_ , GL \_ и GL \_ EXP2. Уравнения, соответствующие этим символьным константам, определены в следующем разделе "Примечания". По умолчанию для тумана используется режим "GL \_ exp".<br/>                                                                           |
| <span id="GL_FOG_DENSITY"></span><span id="gl_fog_density"></span><dl> <dt>**\_плотность тумана GL \_**</dt> </dl> | Параметр *params* — это значение с плавающей запятой, определяющее *плотность*, плотность тумана, используемую в обоих экспоненциальных уравнениях. Принимаются только неотрицательные плотности. Плотность тумана по умолчанию — 1,0.<br/>                                                                                                                                                                                                              |
| <span id="GL_FOG_START"></span><span id="gl_fog_start"></span><dl> <dt>**Начало работы с \_ туманом GL \_**</dt> </dl>       | Параметр *params* — это значение с плавающей запятой, которое указывает *Start*, ближайшее расстояние, используемое в уравнении линейного тумана. Ближайшее расстояние по умолчанию — 0,0.<br/>                                                                                                                                                                                                                                                            |
| <span id="GL_FOG_END"></span><span id="gl_fog_end"></span><dl> <dt>**Окончание GL в главной \_ \_ части**</dt> </dl>             | Параметр *params* — это значение с плавающей запятой, которое указывает *конец*, расстояние, используемое в уравнении линейного тумана. По умолчанию расстояние равно 1,0.<br/>                                                                                                                                                                                                                                                                |
| <span id="GL_FOG_INDEX"></span><span id="gl_fog_index"></span><dl> <dt>**\_индекс тумана GL \_**</dt> </dl>       | Параметр *params* — это значение с плавающей запятой, которое указывает *i*<sub>f</sub> , цветовой индекс тумана. | <span id="GL_FOG_COLOR"></span><span id="gl_fog_color"></span><dl> <dt>**\_Цвет тумана GL \_**</dt> </dl>       | Параметр *params* содержит четыре значения с плавающей запятой, определяющие *C*<sub>f</sub> , цвет тумана. Целочисленные значения сопоставляются линейно, так что наиболее положительное значение, представленное в результате, сопоставляется с 1,0, а наиболее отрицательное значение — в-1,0. Значения с плавающей запятой сопоставляются напрямую. После преобразования все компоненты цвета записываются в диапазон \[ 0, 1 \] . Цвет тумана по умолчанию — (0, 0, 0, 0).<br/> |

</dd> </dl>

## <a name="remarks"></a>Remarks

Вы включаете и отключаете туман с помощью [**гленабле**](glenable.md) и [**глдисабле**](gldisable.md), используя аргумент GL в виде \_ тумана. При включении туман влияет на растровую геометрию, точечные рисунки и блоки пикселей, но не на операции очистки буфера.

Функция **глфогфв** присваивает значение или значения в *params* параметру тумана, заданному аргументом *pname*.

Туман смешивает цвет тумана с каждым растровым цветом фрагмента пикселя с помощью коэффициента смешения *f*. Коэффициент *f* может быть вычислен одним из трех способов в зависимости от режима тумана. Пусть *z* — это расстояние в координатах глаз от начала до фогжед фрагмента. Уравнение \_ линейного тумана GL:

![Уравнение, показывающее значение GL_LINEAR тумана.](images/fog01.png)

Уравнение для наглавной \_ тумана GL имеет следующее:

![Уравнение, показывающее значение коэффициента смешения в GL_EXP режиме тумана.](images/fog02.png)

Уравнение для \_ EXP2 тумана GL имеет следующее значение:

![Уравнение, показывающее значение коэффициента смешения в GL_EXP2 режиме тумана.](images/fog03.png)

Независимо от режима тумана, *f* замещается в диапазон \[ 0, 1 \] после его расчета. Затем, если OpenGL находится в режиме цвета RGBA, цвет кода *C*<sub>r</sub> заменяется на

![Уравнение, показывающее цвет фрагмента фогжед в виде функции коэффициента смешения и цвета тумана.](images/fog04.png)

В режиме индексирования Color индекс фрагмента, который *я* заменил <sub>r</sub> , заменяется на

![Уравнение, показывающее цветовой индекс фрагмента фогжед в качестве функции коэффициента смешения и индексированного цвета.](images/fog05.png)
