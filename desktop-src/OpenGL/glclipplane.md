# <a name="glclipplane-function"></a>Функция Глклипплане

Функция **глклипплане** указывает плоскость, в которой обрезана Вся геометрия.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glClipPlane(
         GLenum   plane,
   const GLdouble *equation
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*плоскости* 
</dt> <dd>

Размещается плоскость обрезки. Символьные имена в виде плоскости Clip в виде главной области \_ \_ *i*, где *i* — целое число от 0 до \_ максимального количества \_ \_ плоскостей, равное 1, принимаются.

</dd> <dt>

*Дробь* 
</dt> <dd>

Адрес массива из четырех значений с плавающей запятой двойной точности. Эти значения интерпретируется как уравнение плоскости.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Геометрия всегда обрезается относительно границ шести плоскостей фрустум в *x*, *y* и *z*. Функция **глклипплане** позволяет определить дополнительные плоскости, не обязательно Перпендикулярные оси *x*, оси *y* или оси *z*, для которых обрезается Вся геометрия. Можно указать до GL максимальных плоскостей для \_ \_ вырезок \_ , где \_ GL \_ \_ не менее шести плоскостей. Так как результирующая область обрезки является пересечением определенных половинных пробелов, оно всегда выпуклой.

Функция **глклипплане** задает половину пространства, используя уравнение плоскости из четырех компонентов. При вызове **глклипплане***Формула* преобразуется инверсией матрицы моделвиев и хранится в результирующих координатах глаза. Последующие изменения матрицы моделвиев не оказывают влияния на хранимые компоненты уравнений плоскости. Если скалярное произведение координат глаза для вершины с компонентами уравнения в хранимой плоскости имеет положительный или нулевое значение, вершина находится в относительно этой плоскости обрезки. В противном случае он истекает.

Используйте функции [**гленабле**](glenable.md) и [**глдисабле**](gldisable.md) для включения и отключения отсечения плоскостей. Вызовите обтравочные плоскости с аргументом GL \_ - \_ плоскостью *i*, где *i* — это номер плоскости.

По умолчанию все обтравочные плоскости определены как (0, 0, 0, 0) в координатах глаза и отключены.

Это всегда так, что GL — это « \_ \_  \_ \_ PLANE0».
