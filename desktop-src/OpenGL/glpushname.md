# <a name="glpushname-function"></a>Функция Глпушнаме

Функции **глпушнаме** и [**глпопнаме**](glpopname.md) отправляют и выводят на экран стека имен.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glPushName(
   GLuint name
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*name* 
</dt> <dd>

Имя, которое будет отправлено в стек имен.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

Функция [**глжетеррор**](glgeterror.md) может получить следующие коды ошибок.



| Имя                                                                                                  | Значение                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_переполнение стека GL \_**</dt> </dl>    | Функция была вызвана, пока текущий стек матриц заполнен.<br/>                                                           |
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Комментарии

Функция **глпушнаме** приводит к тому, что имя помещается в стек имен, который изначально пуст. Функция [**глпопнаме**](glpopname.md) выводит одно имя из верхней части стека. В режиме выбора используется стек имен, чтобы обеспечить уникальную идентификацию наборов команд отрисовки. Он состоит из упорядоченного набора целых чисел без знака.

Стек имен всегда пуст, пока режим рендеринга не является GL \_ SELECT. Вызовы **глпушнаме** или [**глпопнаме**](glpopname.md) , когда режим рендеринга не является GL \_ SELECT, игнорируются.

Следующие функции извлекают сведения, относящиеся к **глпушнаме** и [**глпопнаме**](glpopname.md):

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ \_ Длина стека имен GL \_

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ , \_ \_ глубина стека максимальных имен \_
