# <a name="translating-texgen"></a>Преобразование тексжен

Функция IRI GL **тексжен** преобразуется в [глтексжен](gltexgen-functions.md) для OpenGL.

При использовании IRI GL вы вызываете **тексжен** дважды: один раз, чтобы одновременно задать уравнение режима и плоскости, и один раз для включения создания координат текстуры. Пример:


```C++
texgen(TX_S, TG_LINEAR, planeParams); 
texgen(TX_S, TG_ON, NULL);
```



С помощью OpenGL вы выполняете три вызова: две к **глтексжен** (один раз, чтобы задать режим, и один раз, чтобы задать уравнение плоскости), а второй — [**гленабле**](glenable.md). Например, для кода OpenGL, эквивалентного приведенному выше коду в ГК, используется:


```C++
glTexGen(GL_S, GLTEXTURE_GEN_MODE, modeName); 
glTextGen(GL_S, GL_OBJECT_PLANE, planeParameters); 
glEnable(GL_TEXTURE_GEN_S);
```
