# <a name="reading-color-values-from-the-framebuffer"></a>Считывание значений цвета из буфера кадров

При использовании функций, считывающих значения цвета из буфера кадров, следует учитывать различия между чтением значений RGBA и значений цветовых индексов на true-цветных устройствах и на устройствах на основе палитры.

На true-цветном устройстве:

-   Значения RGBA ограничиваются каналом на устройстве.
-   Значения в цветовых индексах хранятся в буфера кадров в виде значений RGBA. При использовании этих значений необходимо выполнить обратное преобразование из RGBA в логический индекс палитры. Если два логических индекса имеют одинаковые значения RGBA, может возвращаться неверный индекс.

На устройстве на основе палитры:

-   Значения RGBA считываются из индекса в системной палитре. Логический индекс получается из обратной таблицы, а компоненты RGBA извлекаются.
-   Значения цветовых индексов считываются из индекса в системную палитру, а обратная таблица используется для получения индекса логической палитры.
