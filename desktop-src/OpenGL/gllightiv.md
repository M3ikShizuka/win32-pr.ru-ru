# <a name="gllightiv-function"></a>Функция Гллигхтив

Функция **гллигхтив** возвращает значения светлого исходного параметра.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glLightiv(
         GLenum light,
         GLenum pname,
   const GLint  *params
);
```

## <a name="remarks"></a>Комментарии

Функция **гллигхтив** задает значение или значения отдельных параметров источника освещения. Параметр *Light* именует источник света, а — символическое имя формы \_ «GL»*i*, где 0 = <  « \_ максимальные \_ огни».

Параметр *pname* указывает один из исходных параметров источника, который опять же содержит символьное имя. Параметр *params* является либо одиночным значением, либо указателем на массив, который содержит новые значения.

Вычисление освещения включается и отключается с помощью [**гленабле**](glenable.md) и [**глдисабле**](gldisable.md) с аргументом \_ освещение GL. Если включено освещение, источники света, которые включены, вносят вклад в расчет освещения. Источник освещения *i* включен и отключен с помощью **гленабле** и **глдисабле** с аргументом GL \_ Light *i*.

Это всегда относится к нефинансовому \_ освещению *i* = GL \_ LIGHT0 + *i*.

Следующие функции извлекают сведения, относящиеся к функции **гллигхтив** :

[**глжетлигхт**](glgetlight.md)

[**глисенаблед**](glisenabled.md) с аргументом \_ освещение GL
