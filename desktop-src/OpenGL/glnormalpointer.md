# <a name="glnormalpointer-function"></a>Функция Глнормалпоинтер

Функция **глнормалпоинтер** определяет массив нормалей.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glNormalPointer(
         GLenum  type,
         GLsizei stride,
   const GLvoid  *pointer
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*type* 
</dt> <dd>

Тип данных каждой координаты в массиве с использованием следующих символьных констант: GL, Short, GL, GL \_ \_ и GL, с \_ \_ плавающей запятой \_ .

</dd> <dt>

*шага* 
</dt> <dd>

Смещение в байтах между последовательными нормальными значениями. Если *шаг* равен нулю, нормали тесно упакованы в массив.

</dd> <dt>

*вид* 
</dt> <dd>

Указатель на первое значение нормали в массиве.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

Функция [**глжетеррор**](glgeterror.md) может получить следующие коды ошибок.



| Имя                                                                                                  | Значение                                      |
|-------------------------------------------------------------------------------------------------------|----------------------------------------------|
| <dl> <dt>**\_недействительное \_ перечисление GL**</dt> </dl>      | *тип* не является допустимым значением.<br/> |
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | *шаг* или *число* были отрицательными.<br/> |



## <a name="remarks"></a>Remarks

Функция **глнормалпоинтер** задает расположение и данные массива нормалей для использования при подготовке к просмотру. Параметр *типа* задает тип данных каждой стандартной координаты. Параметр *stride* определяет смещение в байтах от одного обычного до следующего, включая упаковку вершин и атрибуты в одном массиве или хранилище в отдельных массивах. В некоторых реализациях хранение вершин и атрибутов в одном массиве может быть более эффективным, чем использование отдельных массивов. Дополнительные сведения см. в разделе [**глинтерлеаведаррайс**](glinterleavedarrays.md) .

Стандартный массив включается при указании \_ обычной \_ константы массива GL с помощью [**гленаблеклиентстате**](glenableclientstate.md). Если параметр включен, [**глдраваррайс**](gldrawarrays.md), [**глдравелементс**](gldrawelements.md) и [**гларрайелемент**](glarrayelement.md) используют стандартный массив. По умолчанию стандартный массив отключен.

Невозможно включить **глнормалпоинтер** в списки вывода.

При указании обычного массива с помощью **глнормалпоинтер** значения всех обычных параметров массива функции сохраняются в состоянии на стороне клиента. Поскольку обычные параметры массива сохраняются в состоянии на стороне клиента, их значения не сохраняются и не восстанавливаются [**глпушаттриб**](glpushattrib.md) и [**глпопаттриб**](glpopattrib.md).

Несмотря на то что ошибка не возникает при вызове **глнормалпоинтер** в парах [**глбегин**](glbegin.md) и [**гленд**](glend.md) , результаты не определены.

С **глнормалпоинтер** связаны следующие функции:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом \_ обычного \_ массива \_ stride

**глжет** с аргументом \_ стандартное \_ число массивов GL \_

**глжет** с аргументом \_ обычного \_ типа массива GL \_

**глжетпоинтерв** с аргументом \_ обычного \_ указателя массива GL \_

[**глисенаблед**](glisenabled.md) с аргументом \_ обычного \_ массива GL
