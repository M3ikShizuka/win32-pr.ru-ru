# <a name="glunextcontour-function"></a>Функция Глунекстконтаур

\[Функция **глунекстконтаур** устарела и предоставляется только для обратной совместимости. Функция **глунекстконтаур** сопоставляется с [**глутессендконтаур**](glutessendcontour.md) , за которым следует [**глутессбегинконтаур**](glutessbegincontour.md).\]

Функция **глунекстконтаур** отмечает начало другого контура.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI gluNextContour(
   GLUtesselator *tess,
   GLenum        type
);
```

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Используйте функцию **глунекстконтаур** для описания многоугольников с несколькими контурами. После описания первого контура с помощью ряда вызовов [**глутессвертекс**](glutessvertex.md) вызов **глунекстконтаур** указывает, что предыдущий профиль завершен и что начинается следующий профиль. Выполните еще одну серию вызовов **глутессвертекс** , чтобы описать новый профиль. Повторите эту процедуру, пока не будут описаны все контуры.

Параметр *типа* определяет, какой тип контура следует за.

Чтобы определить тип первого профиля, можно вызвать **глунекстконтаур** , прежде чем описать первый профиль. Если не вызвать **глунекстконтаур** перед первым контуром, первый контур помечается как Glu \_ наружный.

## <a name="examples"></a>Примеры

Вы можете описать грани четырехсторонней с треугольным отверстием в нем следующим образом:

``` syntax
gluBeginPolygon(tess); 
    gluTessVertex(tess, v1, v1); 
    gluTessVertex(tess, v2, v2); 
    gluTessVertex(tess, v3, v3); 
    gluTessVertex(tess, v4, v4);  
gluNextContour(tess, GLU_INTERIOR); 
    gluTessVertex(tess, v5, v5); 
    gluTessVertex(tess, v6, v6); 
    gluTessVertex(tess, v7, v7);  
gluEndPolygon(tess);
```
