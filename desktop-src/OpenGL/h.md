<dl> <dt>

<span id="opengl_half_space"></span><span id="OPENGL_HALF_SPACE"></span>**половина пробела**
</dt> <dd>

Результат деления плоскости на пространство. Плоскость делит пространство на две половины пробела.

</dd> <dt>

<span id="opengl_homogenous_coordinates"></span><span id="OPENGL_HOMOGENOUS_COORDINATES"></span>**однородные координаты**
</dt> <dd>

Набор координат n + 1, используемый для представления точек в n-мерных многомерных пространствах. Точки в проецированном пространстве можно рассматривать как точки в евклидово пространстве вместе с некоторыми точками в бесконечности. Координаты являются однородными, так как масштабирование каждой из координат с помощью одной ненулевой константы не изменяет точку, на которую ссылается координата. Однородные координаты полезны при вычислении проектной геометрии, и, таким словами, в компьютерной графике, где сцены должны быть проецированы на окно.

</dd> </dl>
