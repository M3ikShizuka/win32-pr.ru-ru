# <a name="glaretexturesresident-function"></a>Функция Гларетекстуресресидент

Функция **гларетекстуресресидент** определяет, являются ли указанные объекты текстуры резидентными в памяти текстуры.

## <a name="syntax"></a>Синтаксис


```C++
GLboolean WINAPI glAreTexturesResident(
         GLsizei   n,
   const GLuint    *textures,
         GLboolean *residences
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*n* 
</dt> <dd>

Число текстур для запроса.

</dd> <dt>

*текстур* 
</dt> <dd>

Адрес массива, содержащего имена текстур для запроса.

</dd> <dt>

*места жительства* 
</dt> <dd>

Адрес массива, в котором возвращается состояние проживания текстуры. Состояние проживания текстуры, названное элементом *текстур* , возвращается в соответствующем *элементе.*

</dd> </dl>

## <a name="remarks"></a>Remarks

На компьютерах с ограниченным объемом памяти текстуры OpenGL устанавливает рабочий набор текстур, находящихся в памяти текстуры. Эти текстуры могут быть привязаны к цели текстуры гораздо эффективнее, чем нерезидентные текстуры.

Функция **гларетекстуресресидент** запрашивает состояние проживания *текстур текстур,* названных элементами *текстур*. Если все именованные текстуры находятся в резидентном виде, **гларетекстуресресидент** ВОЗВРАЩАЕТ \_ значение GL true,  а содержимое расположений не затрагивается. Если какая-либо из названных текстур не является резидентной, **гларетекстуресресидент** ВОЗВРАЩАЕТ \_ значение ГК false, а подробное состояние возвращается в  *n* элементов.

Если *элемент расположений* имеет \_ значение GL true, то текстура, названная соответствующим элементом *текстур* , накладывается в память текстуры.

Чтобы запросить состояние проживания для одной ограничивающей текстуры, вызовите [**глжеттекспараметер**](glgettexparameter.md) с *целевым* параметром, для которого задан целевой текстурой, к которой привязан целевой объект, и установите для параметра *pname* значение \_ резидентная текстура GL \_ . Этот метод необходимо использовать для запроса состояния текстуры по умолчанию в резидентном состоянии.

Невозможно включить **гларетекстуресресидент** в списки вывода.

Функция **гларетекстуресресидент** возвращает состояние местонахождение текстур во время вызова. Это не гарантирует, что текстуры будут оставаться резидентными в любое другое время.

Если текстуры находятся в виртуальной памяти (нет памяти текстуры), они считаются всегда резидентными.

> [!Note]  
> Функция **гларетекстуресресидент** доступна только в OpenGL версии 1,1 или более поздней.
