# <a name="glgenlists-function"></a>Функция Глженлистс

Функция **глженлистс** создает непрерывный набор пустых списков вывода.

## <a name="syntax"></a>Синтаксис


```C++
GLuint WINAPI glGenLists(
   GLsizei range
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*range* 
</dt> <dd>

Количество смежных пустых списков вывода, которые должны быть созданы.

</dd> </dl>

## <a name="error-codes"></a>Коды ошибок

Функция [**глжетеррор**](glgeterror.md) может получить следующие коды ошибок.



| Имя                                                                                                  | Значение                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_недопустимое \_ значение GL**</dt> </dl>     | *диапазон* был отрицательным.<br/>                                                                                                      |
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Комментарии

Функция **глженлистс** имеет один аргумент, *Range*. Он возвращает целое число *n* , такое как *диапазон* смежных пустых списков вывода с именами *n*, *n* + 1,. . ., *n* + (*Range* -1) создаются. Если значение *Range* равно нулю, то при отсутствии доступной группы из *диапазона* смежных имен или при возникновении ошибки, списки отображений не создаются и возвращается ноль.

Следующая функция получает сведения, связанные с **глженлистс**:

[**глислист**](glislist.md)
