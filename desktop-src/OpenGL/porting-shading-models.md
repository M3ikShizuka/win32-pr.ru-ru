| **шадемодел**(метод Гуро)                            | **глшадемодел**( \_ Smooth GL)                                                                     | Плавное затенение.         |
| **жетсм**                                          | [**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) ( \_ модель шейдера GL \_ )      | Возвращает текущую модель затенения. |
| **дизеринг (** \_ вкл.) **смешение**(DT \_ Off) <br/> | [**гленабле**](glenable.md) ( \_ дизеринг GL)[**глдисабле**](gldisable.md)( \_ дизеринг GL)<br/> | Включает или выключает Дизеринг.      |
