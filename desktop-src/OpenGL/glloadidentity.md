# <a name="glloadidentity-function"></a>Функция Гллоадидентити

Функция **гллоадидентити** заменяет текущую матрицу матрицей идентификаторов.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI glLoadIdentity(void);
```



## <a name="parameters"></a>Параметры

У этой функции нет параметров.

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

С помощью функции [**глжетеррор**](glgeterror.md) можно получить следующий код ошибки.



| Имя                                                                                                  | Значение                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_Недопустимая \_ Операция GL**</dt> </dl> | Функция была вызвана между вызовом [**глбегин**](glbegin.md) и соответствующим вызовом [**гленд**](glend.md).<br/> |



## <a name="remarks"></a>Remarks

Функция **гллоадидентити** заменяет текущую матрицу матрицей идентификаторов. Он семантически эквивалентен вызову [**гллоадматрикс**](glloadmatrix.md) со следующей матрицей идентификаторов.

![Схема, показывающая матрицу идентификаторов, Гллоадидентити вызовы.](images/load01.png)

Однако в некоторых случаях это более эффективно.

Следующие функции извлекают сведения, относящиеся к **гллоадидентити**:

[**глжет**](glgetbooleanv--glgetdoublev--glgetfloatv--glgetintegerv.md) с аргументом в \_ режиме матрицы GL \_

**глжет** с аргументом \_ МОДЕЛВИЕВ \_ Матрица GL

**глжет** с аргументом \_ матрица проекции главной книги \_

**глжет** с аргументом \_ \_ Матрица текстуры GL
