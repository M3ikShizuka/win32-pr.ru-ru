# <a name="porting-to-opengl"></a>Перенос в OpenGL

OpenGL предназначен для обеспечения совместимости в операционных системах и оборудовании. Такая схема упрощает для программистов перенос программ OpenGL с одной системы на другую. Хотя каждая операционная система имеет уникальные требования, большая часть кода OpenGL в текущих программах может использоваться как есть. чтобы перенести приложение OpenGL на Windows майкрософт, необходимо внести изменения в программы для работы с системами Windows окон.

в целом приложения переносятся на OpenGL для Windows с одной из двух платформ:

-   Приложения OpenGL, разработанные для системы Window X и библиотеки X (Кслиб)
-   Приложения IRI GL
