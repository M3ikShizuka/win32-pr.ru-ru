# <a name="header-files"></a>Файлы заголовков

Приложения, использующие:

-   Основные функции OpenGL должны включать заголовочный файл <GL \\ GL. h>.
-   Библиотека служебной программы OpenGL должна включать заголовочный файл <GL \\ Glu. h>.
-   Вспомогательная библиотека по программированию OpenGL должна содержать заголовочный файл <GL \\ глаукс. h>.
-   Функции ВГЛ должны включать заголовочный файл Windows. h.
-   Функции Windows, поддерживающие реализацию OpenGL Майкрософт в Windows, должны включать заголовочный файл Windows. h.
