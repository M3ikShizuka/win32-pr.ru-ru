# <a name="porting-triangles"></a>Перенос треугольников

В OpenGL можно нарисовать три типа треугольников: отдельные треугольники, треугольники и вентиляторы треугольников.

OpenGL не имеет эквивалента функции IRI GL **сваптмеш** . Вы можете добиться того же результата, используя сочетание треугольников, треугольников и вентиляторов треугольников.

В следующей таблице перечислены функции IRI GL для рисования треугольников и их эквивалентные функции OpenGL.



| Функция IRI GL           | Эквивалентный параметр Глбегин | Значение                                       |
|----------------------------|------------------------------|-----------------------------------------------|
|                            | \_треугольники GL                | Тройки вершин, интерпретируемые как треугольники. |
| **бгнтмеш**, **ендтмеш** | \_полоса треугольников GL \_          | Связанные полосы треугольников.                   |
|                            | \_вентилятор с треугольником GL \_            | Связанные вентиляторы треугольников.                     |
