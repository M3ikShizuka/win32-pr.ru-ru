# <a name="bluetooth-and-wsaqueryset"></a>Bluetooth и всакуерисет

В следующих разделах описывается использование структуры ВСАКУЕРИСЕТ в каждой из этих операций.

-   [Bluetooth и всакуерисет для запроса устройства](bluetooth-and-wsaqueryset-for-device-inquiry.md)
-   [Bluetooth и всакуерисет для запроса на обслуживание](bluetooth-and-wsaqueryset-for-service-inquiry.md)
-   [Bluetooth и всакуерисет для Set Service](bluetooth-and-wsaqueryset-for-set-service.md)

дополнительные сведения и полный справочник по [**всакуерисет**](/windows/desktop/api/winsock2/ns-winsock2-wsaquerysetw) или другие элементы сокетов Windows см. в разделе Windows sockets раздела Software Development Kit (пакет SDK для платформы).
