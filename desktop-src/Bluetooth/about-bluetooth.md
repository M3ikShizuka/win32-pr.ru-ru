# <a name="about-bluetooth"></a>О Bluetooth

Bluetooth — это стандартный промышленный протокол, обеспечивающий беспроводное подключение для множества устройств, включая компьютеры, принтеры, мобильные телефоны и карманные устройства.

ключевые Bluetooth функции включают:

-   Экономичный и экономичный протокол с низким энергопотреблением, обеспечивающий поддержку стандартных отраслевых стандартов и принятие решений по всему миру.
-   Определенный и знакомый программный интерфейс, который разработчики могут использовать для быстрой разработки приложений или работы с ними.
-   официальный веб-узел и корпоративная организация, которая описывает, способствует и стандартизует Bluetooth технологии. Дополнительные сведения см. в разделе [www.Bluetooth.com](https://www.bluetooth.com/).

Bluetooth на Windows предоставляет основные службы, аналогичные тем, которые предоставляются протоколом управления передачей (tcp-компонент tcp/IP). как и многие сетевые протоколы и службы, Bluetooth подключения и обмена данными программируется через вызовы функций Windows сокетов, используя стандартные методы программирования сокетов Windows и конкретные расширения Bluetooth. однако, поскольку существуют значительные различия между проводной, фиксированной сетью и беспроводной сетью с прямыми сетями, Bluetooth предоставляет расширения, такие как обнаружение службы, устройства и уведомление, позволяющие приложениям правильно работать в беспроводной среде. Эти расширения также Pave способ простого переноса на аналогичные технологии, такие как IrDA или будущие Беспроводные транспорта.

корпорация майкрософт предоставляет поддержку Bluetooth в Windows xp с пакетом обновления 1 (SP1) и более поздних версий, на Windows XP Embedded с пакетом обновления 2 (sp2) и Windows CE. Bluetooth приложения, работающие в Windows XP, должны иметь возможность работать в образе среды выполнения Windows XP Embedded, который включает необходимые зависимости. дополнительные сведения о Windows XP embedded см. в справочной документации по Windows xp embedded на сайте MSDN. дополнительные сведения о Windows CE программировании см. в пакете SDK для Windows CE.

корпорация майкрософт предлагает два подхода к программированию Bluetooth Windows.

-   использование интерфейса сокетов Windows
-   управление устройствами напрямую с помощью несокетных Bluetooth интерфейсов

В этом разделе содержатся общие сведения о обоих подходах в следующих разделах. дополнительные сведения об использовании Windows API-интерфейсов сокетов для программирования Bluetooth см. в разделе [Bluetoothное программирование с Windows сокетами](bluetooth-programming-with-windows-sockets.md).
