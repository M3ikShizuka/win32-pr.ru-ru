# <a name="managing-bluetooth-devices-and-services"></a>управление устройствами и службами Bluetooth

в этом разделе описано, как использовать API Bluetooth для непосредственного управления Bluetooth устройствами и службами.

сведения об использовании интерфейса сокетов Windows для программирования Bluetooth на Windows см. в разделе [поддержка сокетов Windows для Bluetooth](windows-sockets-support-for-bluetooth.md).

> [!Note]  
> Bluetooth не мешает функциям управления питанием прерывать Bluetooth передачи. приложения с поддержкой Bluetooth могут отслеживать сообщения о питании, чтобы предотвратить такое прерывание. Дополнительные сведения см. [в разделе Использование функции управления питанием](/windows/desktop/Power/using-power-management).
