# <a name="bluetooth-and-getsockname"></a>Bluetooth и жетсоккнаме

Bluetooth использует функцию [**жетсоккнаме**](/windows/desktop/api/winsock/nf-winsock-getsockname) для получения адреса и номера порта серверного устройства, выделенного сокету посредством предыдущего вызова функции [**BIND**](/windows/desktop/api/winsock/nf-winsock-bind) .

Дополнительные сведения см. в разделе [сокеты Windows](/windows/desktop/WinSock/windows-sockets-start-page-2).
