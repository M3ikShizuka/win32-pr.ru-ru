# <a name="bluetooth-and-getsockname"></a>Bluetooth и жетсоккнаме

Bluetooth использует функцию [**жетсоккнаме**](/windows/desktop/api/winsock/nf-winsock-getsockname) для получения адреса устройства сервера и номера порта, выделенного сокету через предыдущий вызов функции [**bind**](/windows/desktop/api/winsock/nf-winsock-bind) .
