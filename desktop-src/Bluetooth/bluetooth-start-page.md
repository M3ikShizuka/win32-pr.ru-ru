# <a name="bluetooth"></a>Bluetooth

## <a name="purpose"></a>Назначение

Bluetooth является стандартным промышленным протоколом, обеспечивающим беспроводное подключение к компьютерам, карманным устройствам, мобильным телефонам и другим устройствам.

## <a name="where-applicable"></a>Где применимо

Bluetooth интерфейс программирования приложений позволяет разработчикам использовать существующие знания о сетевом программировании для быстрой разработки приложений или работы с ними.

## <a name="developer-audience"></a>Аудитория разработчиков

Bluetooth предназначен для использования программистами C/C++. некоторые Bluetooth функции доступны с сокетами Windows. знакомство с Microsoft Windows сетями и программированием Windowsных соединителей является обязательным.

## <a name="run-time-requirements"></a>Требования к среде выполнения

поддержка Microsoft Bluetooth начинается с Windows XP с пакетом обновления 1 (SP1).

поддержка Bluetooth 2,1 предлагается в Windows Vista с пакетом обновления 2 (SP2) и Windows 7.
