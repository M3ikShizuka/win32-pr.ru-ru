# <a name="wm_psd_minmarginrect-message"></a>\_Сообщение WM PSD \_ минмаргинрект

Уведомляет процедуру-обработчик [*пажепаинсук*](/windows/win32/api/commdlg/nc-commdlg-lppagepainthook) о координатах прямоугольника поля на странице образца. Диалоговое окно « **Параметры страницы** » отправляет это сообщение, когда будет выводиться содержимое образца страницы.


```C++
#define WM_USER                  0x0400
#define WM_PSD_MINMARGINRECT    (WM_USER+2)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Маркер контекста устройства для примера страницы.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**Rect**](/previous-versions//dd162897(v=vs.85)) , содержащую координаты (в пикселях) прямоугольника минимального поля.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если процедура-обработчик возвращает **значение true**, то диалоговое окно не отправляет никаких сообщений и не рисует его на образце страницы до тех пор, пока системе не потребуется перерисовать образец страницы.

Если процедура-обработчик возвращает **значение false**, то диалоговое окно отправляет оставшиеся сообщения последовательности отображения.

## <a name="remarks"></a>Комментарии

Диалоговое окно **Параметры страницы** содержит изображение образца страницы, показывающее, как выбранные пользователем параметры влияют на внешний вид напечатанных данных. При вызове функции [**пажесетупдлг**](/previous-versions/windows/desktop/legacy/ms646937(v=vs.85)) можно предоставить процедуру-обработчик [*пажепаинсук*](/windows/win32/api/commdlg/nc-commdlg-lppagepainthook) для настройки внешнего вида образца страницы. Каждый раз, когда диалоговое окно собирается вывести содержимое образца страницы, диалоговое окно отправляет последовательность сообщений в процедуру-обработчик.
