# <a name="wm_psd_pagesetupdlg-message"></a>\_Сообщение WM PSD \_ пажесетупдлг

Уведомляет процедуру-обработчик [**пажепаинсук**](/windows/win32/api/commdlg/nc-commdlg-lppagepainthook) о том, что диалоговое окно " **Параметры страницы** " собирается вывести содержимое образца страницы. Процедура-обработчик может использовать это сообщение для выполнения задач инициализации, связанных с рисованием содержимого образца страницы.


```C++
#define WM_USER                  0x0400
#define WM_PSD_PAGESETUPDLG     (WM_USER  )
```

## <a name="return-value"></a>Возвращаемое значение

Если процедура-обработчик возвращает **значение true**, то диалоговое окно не отправляет никаких сообщений и не рисует его на образце страницы до тех пор, пока системе не потребуется перерисовать образец страницы.

Если процедура-обработчик возвращает **значение false**, то диалоговое окно отправляет оставшиеся сообщения последовательности отображения.

## <a name="remarks"></a>Комментарии

Диалоговое окно **Параметры страницы** содержит изображение образца страницы, показывающее, как выбранные пользователем параметры влияют на внешний вид напечатанных данных. При вызове функции [**пажесетупдлг**](/previous-versions/windows/desktop/legacy/ms646937(v=vs.85)) можно предоставить процедуру-обработчик [*пажепаинсук*](/windows/win32/api/commdlg/nc-commdlg-lppagepainthook) для настройки внешнего вида образца страницы. Каждый раз, когда диалоговое окно собирается вывести содержимое образца страницы, диалоговое окно отправляет последовательность сообщений в процедуру-обработчик.

Первые три сообщения последовательности (**WM \_ PSD \_ пажесетупдлг**, [**WM \_ PSD \_ фуллпажерект**](wm-psd-fullpagerect.md)или [**WM \_ PSD \_ минмаргинрект**](wm-psd-minmarginrect.md)) предоставляют сведения, которые процедура-ловушка может использовать для рисования содержимого образца страницы. Оставшиеся сообщения ([**WM \_ PSD \_ маргинрект**](wm-psd-marginrect.md), [**WM \_ PSD \_ гриктекстрект**](wm-psd-greektextrect.md), [**WM \_ PSD \_ енвстампрект**](wm-psd-envstamprect.md), [**WM \_ PSD \_ яфуллпажерект**](wm-psd-yafullpagerect.md)) уведомляют процедуру-обработчик о том, что диалоговое окно собирается отобразить определенную часть образца страницы. Это позволяет процедуре-обработчику выборочно рисовать части образца страницы.
