# <a name="wm_psd_yafullpagerect-message"></a>\_Сообщение WM PSD \_ яфуллпажерект

Уведомляет процедуру-обработчик диалогового окна **Параметры страницы** , [*пажепаинсук*](/windows/win32/api/commdlg/nc-commdlg-lppagepainthook), что диалоговое окно собирается отобразить часть возвращаемого адреса на странице образца конверта.


```C++
#define WM_USER                  0x0400
#define WM_PSD_YAFULLPAGERECT   (WM_USER+6)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Маркер контекста устройства для примера страницы.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**Rect**](/previous-versions//dd162897(v=vs.85)) , содержащую координаты образца страницы (в пикселях).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если процедура-обработчик возвращает **значение true**, то диалоговое окно не рисует часть возвращаемого адреса на странице образца конверта.

Если процедура-обработчик возвращает **значение false**, в диалоговом окне выводится часть возвращаемого адреса из образца страницы "Конверт".

Если тип бумаги не является конвертом, возвращаемое значение не действует.

## <a name="remarks"></a>Комментарии

Диалоговое окно **Параметры страницы** содержит изображение образца страницы, показывающее, как выбранные пользователем параметры влияют на внешний вид напечатанных данных. При вызове функции [**пажесетупдлг**](/previous-versions/windows/desktop/legacy/ms646937(v=vs.85)) можно предоставить процедуру-обработчик [*пажепаинсук*](/windows/win32/api/commdlg/nc-commdlg-lppagepainthook) для настройки внешнего вида образца страницы. Каждый раз, когда диалоговое окно собирается вывести содержимое образца страницы, диалоговое окно отправляет последовательность сообщений в процедуру-обработчик.
