# <a name="dm_reposition-message"></a>\_Сообщение о перемещении DM

Перемещает диалоговое окно верхнего уровня таким образом, чтобы оно поместилось в область рабочего стола. Приложение может отправить это сообщение в диалоговое окно после изменения его размера, чтобы убедиться, что все диалоговое окно остается видимым.


```C++
#define WM_USER              0x0400
#define DM_REPOSITION       (WM_USER+2)
```

</dt> <dd>

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется и должен быть равен нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Это сообщение не имеет возвращаемого значения.

## <a name="remarks"></a>Remarks

Это сообщение не действует, если диалоговое окно является дочерним окном.
