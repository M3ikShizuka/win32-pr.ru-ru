# <a name="wm_enteridle-message"></a>\_Сообщение ЕНТЕРИДЛЕ WM

Отправляется в окно владельца модального диалогового окна или меню, которое переходит в состояние простоя. Модальное диалоговое окно или меню переходит в состояние простоя, если в очереди нет сообщений, ожидающих обработки одного или нескольких предыдущих сообщений.


```C++
#define WM_ENTERIDLE                    0x0121
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр может принимать одно из указанных ниже значений.



| Значение                                                                                                                                                                                                                   | Значение                                                          |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------|
| <span id="MSGF_DIALOGBOX"></span><span id="msgf_dialogbox"></span><dl> <dt>**Мсгф \_ ДИАЛОГБОКС**</dt> <dt>0</dt> </dl> | Система бездействует, так как отображается диалоговое окно.<br/> |
| <span id="MSGF_MENU"></span><span id="msgf_menu"></span><dl> <dt>**Мсгф \_ МЕНЮ**</dt> <dt>2</dt> </dl>                | Система бездействует, поскольку отображается меню.<br/>       |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Маркер диалогового окна (если *wParam* — **мсгф \_ диалогбокс**) или окно, содержащее отображаемое меню (если *wParam* — **мсгф \_ меню**).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Приложение должно вернуть нуль, если оно обрабатывает это сообщение.

## <a name="remarks"></a>Remarks

Вы можете отключить сообщение **WM \_ ентеридле** для диалогового окна, создав диалоговое окно с помощью стиля **DS \_ ноидлемсг** .
