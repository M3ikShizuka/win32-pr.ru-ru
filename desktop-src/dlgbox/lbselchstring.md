# <a name="lbselchstring-message"></a>Сообщение ЛБСЕЛЧСТРИНГ

\[Начиная с Windows Vista, диалоговые окна " **Открыть** " и " **Сохранить как** " были заменены [диалоговым окном общих элементов](/previous-versions/windows/desktop/legacy/bb776913(v=vs.85)). Рекомендуется использовать API-интерфейс общего элемента, а не эти диалоговые окна из библиотеки общих диалоговых окон.\]

Диалоговое окно **Открыть** или **Сохранить как** отправляет зарегистрированное сообщение **лбселчстринг** в процедуру-обработчик при изменении выбора в любом из списков или полей со списком диалогового окна.


```C++
#define LBSELCHSTRING TEXT("commdlg_LBSelChangedNotify")
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Идентификатор списка или поля со списком, в котором был изменен выбор.

</dd> <dt>

*lParam* 
</dt> <dd>

Слово нижнего порядка задает номер позиции выбранной строки в списке или поле со списком. Слово в высоком порядке указывает тип изменения выбора. Этот параметр может принимать одно из указанных ниже значений.



| Значение                                                                                                                                                                                                                       | Значение                                                                            |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------|
| <span id="CD_LBSELCHANGE"></span><span id="cd_lbselchange"></span><dl> <dt>**Компакт-диск \_ ЛБСЕЛЧАНЖЕ**</dt> <dt>0</dt> </dl>     | Элемент является единственным элементом, выбранным в списке с единственным выбором.<br/>      |
| <span id="CD_LBSELADD"></span><span id="cd_lbseladd"></span><dl> <dt>**Компакт-диск \_ ЛБСЕЛАДД**</dt> <dt>2</dt> </dl>              | Элемент является одним из элементов, выбранных в списке с множественным выбором.<br/> |
| <span id="CD_LBSELSUB"></span><span id="cd_lbselsub"></span><dl> <dt>**Компакт-диск \_ ЛБСЕЛСУБ**</dt> <dt>1</dt> </dl>              | Элемент больше не выбирается в списке с множественным выбором.<br/>        |
| <span id="CD_LBSELNOITEMS"></span><span id="cd_lbselnoitems"></span><dl> <dt>**Компакт-диск \_ ЛБСЕЛНОИТЕМС**</dt> <dt>-1</dt> </dl> | В списке с множественным выбором не существует элементов.<br/>                        |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Это сообщение не имеет возвращаемого значения.

## <a name="remarks"></a>Комментарии

Процедура-обработчик должна указать константу **лбселчстринг** в вызове функции [**регистервиндовмессаже**](/windows/desktop/api/winuser/nf-winuser-registerwindowmessagea) , чтобы получить идентификатор сообщения, отправленного диалоговым окном.
