# <a name="wm_psd_envstamprect-message"></a>\_Сообщение WM PSD \_ енвстампрект

Уведомляет процедуру-обработчик диалогового окна **Параметры страницы** , [*пажепаинсук*](/windows/win32/api/commdlg/nc-commdlg-lppagepainthook), что диалоговое окно собирается отобразить прямоугольник метки конверта страницы образца.


```C++
#define WM_USER                  0x0400
#define WM_PSD_ENVSTAMPRECT     (WM_USER+5)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Маркер контекста устройства для примера страницы.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**Rect**](/previous-versions//dd162897(v=vs.85)) , которая содержит координаты (в пикселях) прямоугольника метки конверта.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если процедура-обработчик возвращает **значение true**, то диалоговое окно не рисует часть метки конверта на странице выборки.

Если процедура-обработчик возвращает **значение false**, то в диалоговом окне на странице выводится часть метки конверта.

## <a name="remarks"></a>Комментарии

Диалоговое окно **Параметры страницы** содержит изображение образца страницы, показывающее, как выбранные пользователем параметры влияют на внешний вид напечатанных данных. При вызове функции [**пажесетупдлг**](/previous-versions/windows/desktop/legacy/ms646937(v=vs.85)) можно предоставить процедуру-обработчик [*пажепаинсук*](/windows/win32/api/commdlg/nc-commdlg-lppagepainthook) для настройки внешнего вида образца страницы. Каждый раз, когда диалоговое окно собирается вывести содержимое образца страницы, диалоговое окно отправляет последовательность сообщений в процедуру-обработчик.

Процедура-обработчик получает это сообщение, только если выбранный тип бумаги является конвертом.
