# <a name="server-data-objects-interfaces"></a>Интерфейсы объектов данных сервера

API серверных объектов данных (SDO) предоставляет следующие интерфейсы.




| Интерфейс | Назначение | 
|-----------|---------|
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdo"><strong>исдо</strong></a> | Работа с объектом SDO.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdocollection"><strong>исдоколлектион</strong></a> | Управлять коллекцией объектов SDO;<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdodictionaryold"><strong>исдодиктионарйолд</strong></a> | Работа с словарем атрибутов.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdomachine"><strong>исдомачине</strong></a> | Управление компьютером SDO и получение объектов SDO.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdoservicecontrol"><strong>исдосервицеконтрол</strong></a> | Управление службой, управляемой с помощью SDO, то есть службы проверки подлинности в Интернете (IAS) или сервера удаленного доступа (RAS).<br /><blockquote>[!Note]<br />служба проверки подлинности в интернете (IAS) переименовала сервер политики сети (NPS), начиная с Windows Server 2008.</blockquote><br /> |
