# <a name="elements-of-the-sdo-api"></a>Элементы API SDO

API SDO разработан вокруг объектной модели, состоящей из объектов и коллекций объектов. В объектной модели SDO объекты и коллекции упорядочены в иерархии, то есть объекты и коллекции содержат другие объекты и коллекции.

Для управления объектами используется извлечение и установка свойств объекта.

В следующих разделах описывается использование объектов, свойств объектов и коллекций API SDO.

-   [Иерархия объектной модели](/windows/desktop/Nps/sdo-object-model-hierarchy)
-   [Атрибуты](/windows/desktop/Nps/sdo-sdo-supported-attributes)
