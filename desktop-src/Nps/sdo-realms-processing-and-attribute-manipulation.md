# <a name="realms-processing-and-attribute-manipulation"></a>Обработка сфер и управление атрибутами

> [!Note]  
> служба проверки подлинности в интернете (IAS) переименовала сервер политики сети (NPS), начиная с Windows Server 2008. Содержимое этого раздела относится как к IAS, так и к NPS. По всему тексту NPS используется для ссылки на все версии службы, включая версии, изначально называемые IAS.

 

Обработка сфер, которая также называется манипуляцией с атрибутами, относится к преобразованию имени пользователя, запросившего доступ. Также можно манипулировать ИДЕНТИФИКАТОРом вызывающей станции и ИДЕНТИФИКАТОРом вызываемой станции. Правила обработки сфер являются частью коллекции атрибутов профиля прокси-сервера.

Для каждой манипуляции необходимо создать два атрибута для [правил манипуляции](/windows/desktop/Nps/sdo-manipulation-rule) . Каждый из этих атрибутов является строковым значением. Первое правило содержит регулярное выражение, представляющее шаблон для сопоставления. Второе правило содержит регулярное выражение, представляющее замещающий текст. Необходимо также создать атрибут " [манипуляция-Target](/windows/desktop/Nps/sdo-manipulation-target) ". Этот атрибут является перечислением, которое указывает, какая часть данных пользователя будет управлять.

Порядок, в котором создаются атрибуты, важен. Сервер политики сети обрабатывает атрибуты в том порядке, в котором они были созданы.

В следующей таблице показан пример набора атрибутов манипуляции.



| Имя                 | Тип         | Строковый параметр     |
|----------------------|--------------|------------------|
| мсманипулатионруле   | **VT \_ BSTR** | "@company.com"   |
| мсманипулатионруле   | **VT \_ BSTR** | "@microsoft.com" |
| мсманипулатионруле   | **VT \_ BSTR** | "^.+@"           |
| мсманипулатионруле   | **VT \_ BSTR** | "guest@"         |
| мсманипулатионтаржет | **VT \_ I4**   | "1"              |
