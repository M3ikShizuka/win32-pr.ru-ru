# <a name="manipulation-target"></a>Manipulation-Target

> [!Note]  
> служба проверки подлинности в интернете (IAS) переименовала сервер политики сети (NPS), начиная с Windows Server 2008.

 

Целевые данные, к которым применяется правило обработки атрибутов. Управление атрибутами ранее называлось "обработка сфер". Дополнительные сведения см. в разделе [обработка и управление сферами](/windows/desktop/Nps/sdo-realms-processing-and-attribute-manipulation).

**Имя LDAP:** мсманипулатионтаржет

**Идентификатор:** 4113 = [ **\_ \_ \_ целевой объект манипуляции с атрибутами IAS**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Синтаксис:** Перечислитель

* * Значения: * *

Определяется центром IP-назначенных номеров (IANA) в документе [типов RADIUS](https://www.iana.org/assignments/radius-types) , основанном на [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt).

-   User-Name (1)
-   Именуемый-Station-ID (30)
-   Вызывающий-Station-ID (31)
