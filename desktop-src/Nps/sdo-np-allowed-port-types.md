# <a name="np-allowed-port-types"></a>NP-Allowed-Types-типы

Типы портов, разрешенные для соединения.

**Имя LDAP:** мснпалловедпорттипес

**Идентификатор:** 4104 = [ **атрибут IAS с \_ \_ \_ разрешенными \_ \_ типами портов NP**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Синтаксис:** Перечислитель

**Значения:  **

Определяется центром IP-назначенных номеров (IANA) в документе [типов RADIUS](https://www.iana.org/assignments/radius-types) , основанном на [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt).

-   Async (модем) (0)
-   Синхронизация (линия T1) (1)
-   Синхронизация ISDN (2)
-   ISDN Async V. 120 (3)
-   ISDN Async V. 110 (4)
-   Виртуальная сеть (VPN) (5)
-   ПИАФС (6)
-   ХДЛК очистить канал (7)
-   X. 25 (8)
-   X. 75 (9)
-   G. 3 Факс (10)
-   СДСЛ-симметричный DSL (11)
-   ADSL — Асимметричная модуляция для фазы амплитуды бесконтактного DSL (12)
-   ADSL-ДМТ-асимметричное дискретное многофакторное DSL (13)
-   ИДСЛ — линия цифрового подписчика ISDN (14)
-   Ethernet (15)
-   Ксдсл — строка цифрового подписчика неизвестного типа (16)
-   Кабель (17)
-   Беспроводная сеть (18)
-   Wireless-IEEE 802,11 (19)
-   Token Ring (20)
-   FDDI (21)
