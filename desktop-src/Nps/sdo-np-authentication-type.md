# <a name="np-authentication-type"></a>NP-Authentication-тип

Типы проверки подлинности, разрешенные для соединения.

**Имя LDAP:** msNPAuthenticationType2

**Идентификатор:** 4105 = [ **\_ атрибут IAS \_ \_ \_ тип проверки подлинности NP**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Синтаксис:** Цело

* * Значения: * *

Определяется в перечислимом типе [**проверки подлинности \_**](/windows/desktop/api/SdoIas/ne-sdoias-authentication_type) , объявленном в общедоступном файле заголовка сдоиас. h.

-   **Служба IAS \_ Проверка подлинности \_ недопустима** (0)
-   **Служба IAS \_ Проверка подлинности \_ PAP** (1)
-   **Служба IAS \_ Проверка подлинности \_ MD5CHAP** (2)
-   **Служба IAS \_ Проверка подлинности \_ mschap** (3)
-   **Служба IAS \_ Проверка подлинности \_ MSCHAP2** (4)
-   **Служба IAS \_ Проверка подлинности \_ EAP** (5)
-   **Служба IAS \_ Проверка подлинности \_ ARAP** (6)
-   **Служба IAS \_ Проверка подлинности \_ нет** (7)
-   **Служба IAS \_ \_Пользовательская проверка ПОдлинности** (8)
-   **Служба IAS \_ Проверка подлинности \_ mschap \_ КПВ** (9)
-   **Служба IAS \_ AUTH \_ MSCHAP2 \_ КПВ** (10)
-   **Служба IAS \_ Проверка подлинности \_ PEAP** (11)
