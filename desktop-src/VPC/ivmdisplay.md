# <a name="ivmdisplay-interface"></a>Интерфейс Ивмдисплай

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Управляет параметрами экрана виртуальной машины. Интерфейс **ивмдисплай** для виртуальной машины можно получить с помощью свойства [**ивмвиртуалмачине::D Play**](ivmvirtualmachine-display.md) .

## <a name="members"></a>Элементы

Интерфейс **ивмдисплай** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмдисплай** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ивмдисплай** содержит следующие методы.



| Метод                                                       | Описание                                                                                             |
|:-------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------|
| [**\_GenerateThumbnail**](ivmdisplay--generatethumbnail.md) | Извлекает массив пикселей, представляющий эскиз изображения экрана виртуальной машины.<br/> |
| [**сетдименсионс**](ivmdisplay-setdimensions.md)            | Задает высоту и ширину дисплея виртуальной машины в пикселях.<br/>                       |



 

### <a name="properties"></a>Свойства

Интерфейс **ивмдисплай** имеет следующие свойства.



| Свойство                                             | Тип доступа          | Описание                                                                                   |
|:-----------------------------------------------------|:---------------------|:----------------------------------------------------------------------------------------------|
| [**канресизе**](ivmdisplay-canresize.md)<br/> | Только для чтения<br/> | Указывает, допускает ли гостевое изменение разрешение изменений.<br/>                             |
| [**Высота**](ivmdisplay-height.md)<br/>       | Только для чтения<br/> | Высота дисплея виртуальной машины в пикселях.<br/>                            |
| [**Thumbnail**](ivmdisplay-thumbnail.md)<br/> | Только для чтения<br/> | Массив пикселей, представляющий эскиз изображения экрана виртуальной машины.<br/> |
| [**видеомоде**](ivmdisplay-videomode.md)<br/> | Только для чтения<br/> | Текущий режим видео (текст, VGA, SVGA и т. д.).<br/>                               |
| [**Ширина**](ivmdisplay-width.md)<br/>         | Только для чтения<br/> | Ширина дисплея виртуальной машины в пикселях.<br/>                             |
