# <a name="ivmvirtualmachineadddvdromdrive-method"></a>Метод Ивмвиртуалмачине:: Адддвдромдриве

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Добавляет новый компакт-диск или DVD-дисковод в виртуальную машину.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT AddDVDROMDrive(
  [in]          long        busNumber,
  [in]          long        deviceNumber,
  [out, retval] IVMDVDDrive **dvdDrive
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*буснумбер* \[ окне\]
</dt> <dd>

Шина, к которой будет подключен диск.



| Значение                                                                        | Значение                                                  |
|------------------------------------------------------------------------------|----------------------------------------------------------|
| <dl> <dt>0</dt> </dl> | Диск будет подключен к первой шине.<br/>  |
| <dl> <dt>1</dt> </dl> | Диск будет подключен к второй шине.<br/> |



 

</dd> <dt>

*девиценумбер* \[ окне\]
</dt> <dd>

Устройство, к которому будет подключен диск.



| Значение                                                                        | Значение                                                                |
|------------------------------------------------------------------------------|------------------------------------------------------------------------|
| <dl> <dt>0</dt> </dl> | Диск будет подключен к первому устройству на шине.<br/>  |
| <dl> <dt>1</dt> </dl> | Диск будет подключен ко второму устройству на шине.<br/> |



 

</dd> <dt>

*двддриве* \[ out, retval\]
</dt> <dd>

Объект [**ивмдвддриве**](ivmdvddrive.md) .

</dd> </dl>

## <a name="remarks"></a>Remarks

В остановленную виртуальную машину можно добавить только новый компакт-диск или DVD-дисковод.
