# <a name="ivmmouseusingabsolutecoordinates-property"></a>Свойство Ивммаусе:: Усингабсолутекурдинатес

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Возвращает, представляют ли координаты мыши абсолютные или относительные координаты.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_UsingAbsoluteCoordinates(
  [in]          VARIANT_BOOL usingAbsoluteCoordinates
);

HRESULT get_UsingAbsoluteCoordinates(
  [out, retval] VARIANT_BOOL *usingAbsoluteCoordinates
);
```



## <a name="property-value"></a>Значение свойства

**Вариант \_ Значение TRUE** , чтобы настроить устройство мыши для использования абсолютных координат, **вариант \_ false** — для использования относительных координат.

## <a name="remarks"></a>Комментарии

Это свойство является локальным только для этого объекта и по умолчанию будет иметь **вариант \_ false** для нового объекта [**ивммаусе**](ivmmouse.md) . Обратите внимание, что для использования абсолютных координат необходимо установить компоненты интеграции.
