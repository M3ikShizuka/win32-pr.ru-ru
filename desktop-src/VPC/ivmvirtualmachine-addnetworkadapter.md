# <a name="ivmvirtualmachineaddnetworkadapter-method"></a>Метод Ивмвиртуалмачине:: Адднетворкадаптер

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Добавляет сетевой интерфейс к виртуальной машине.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT AddNetworkAdapter(
  [out, retval] IVMNetworkAdapter **networkAdapter
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*сетевого адаптера* \[ out, retval\]
</dt> <dd>

Объект [**ивмнетворкадаптер**](ivmnetworkadapter.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                            | Описание                                                     |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                  | Операция выполнена успешно.<br/>                        |
| <dl> <dt>**Д \_**</dt> <dt>0x80004003</dt> указателя </dl>                    | Параметр *сетевого адаптера* имеет **значение NULL**.<br/>          |
| <dl> <dt>**Виртуальная машина \_ \_Неизвестный \_ 0xA0040207 виртуальной машины**</dt> <dt></dt> </dl>            | Конфигурация неизвестна.<br/>                        |
| <dl> <dt>**Виртуальная машина \_ E \_ преф \_ недопустимое \_ значение**</dt> <dt>0xA0040301</dt> </dl>   | Можно добавить не более четырех (4) сетевых адаптеров.<br/> |
| <dl> <dt>**Виртуальная машина \_ \_Виртуальная машина \_ с установленным \_ или \_ сохраненным**</dt> <dt>0xA004020B</dt> </dl> | Виртуальная машина находится в работающем или сохраненном состоянии.<br/>  |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>            | Произошла непредвиденная ошибка.<br/>                    |



 

## <a name="remarks"></a>Remarks

В остановленную виртуальную машину можно добавить только новый сетевой интерфейс. Созданный сетевой адаптер не подключен к виртуальной сети.
