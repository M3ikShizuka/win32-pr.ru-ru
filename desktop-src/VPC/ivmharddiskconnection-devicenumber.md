# <a name="ivmharddiskconnectiondevicenumber-property"></a>Ивмхарддискконнектион: свойство Евиценумбер:D

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Возвращает номер устройства, к которому подключен образ диска.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_DeviceNumber(
  [out, retval] long *vmDeviceNumber
);
```



## <a name="property-value"></a>Значение свойства

Номер устройства, соответствующий этому соединению.
