# <a name="ivmvirtualpcgetharddisk-method"></a>Метод Ивмвиртуалпк:: Жесарддиск

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Извлекает объект, соответствующий существующему файлу образа диска.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetHardDisk(
  [in]          BSTR        imagePath,
  [out, retval] IVMHardDisk **hardDisk
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*imagePath* \[ окне\]
</dt> <dd>

Полный путь к существующему файлу образа диска.

</dd> <dt>

*жесткого* \[ диска out, retval\]
</dt> <dd>

Объект [**ивмхарддиск**](ivmharddisk.md) , соответствующий этому образу диска.

</dd> </dl>
