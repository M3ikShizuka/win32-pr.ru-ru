# <a name="ivmfloppydrivereleaseimage-method"></a>Метод Ивмфлоппидриве:: Релеасеимаже

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Выпускают образ гибкого носителя на узле с дисковода гибких дисков.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT ReleaseImage();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                          | Описание                                                                            |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                | Операция выполнена успешно.<br/>                                               |
| <dl> <dt>**Виртуальная машина \_ \_Неизвестный \_ 0xA0040207 виртуальной машины**</dt> <dt></dt> </dl>          | Конфигурация этой виртуальной машины недопустима или не найдена.<br/> |
| <dl> <dt>**Виртуальная машина \_ \_ \_ Неверный \_ тип носителя**</dt> <dt>0xA00400728</dt> </dl>  | Носитель, подключенный к этому дисководу гибких дисков, не является образом дискеты.<br/>         |
| <dl> <dt>**Виртуальная машина \_ E \_ нет \_ \_**</dt> <dt>0xA00400652ных</dt> носителей </dl> | К этому дисководу гибких дисков не подключен носитель.<br/>                            |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>          | Произошла непредвиденная ошибка.<br/>                                           |
