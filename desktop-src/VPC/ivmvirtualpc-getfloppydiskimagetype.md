# <a name="ivmvirtualpcgetfloppydiskimagetype-method"></a>Метод Ивмвиртуалпк:: Жетфлоппидискимажетипе

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Возвращает тип существующего файла образа гибкого диска.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetFloppyDiskImageType(
  [in]          BSTR                  imagePath,
  [out, retval] VMFloppyDiskImageType *imageType
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*imagePath* \[ окне\]
</dt> <dd>

Полный путь к файлу образа диска.

</dd> <dt>

*imageType* \[ out, retval\]
</dt> <dd>

Тип образа диска. Список значений см. в разделе [**вмфлоппидискимажетипе**](vmfloppydiskimagetype.md).

</dd> </dl>
