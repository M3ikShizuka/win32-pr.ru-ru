# <a name="ivmharddisksizeonhost-property"></a>Свойство Ивмхарддиск:: Сизеонхост

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Получает размер файла виртуального жесткого диска на главном компьютере.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_SizeOnHost(
  [out, retval] VARIANT *fileSize
);
```



## <a name="property-value"></a>Значение свойства

Размер файла образа жесткого диска в байтах. Это значение является **разновидностью** типа **\_ Decimal**.
