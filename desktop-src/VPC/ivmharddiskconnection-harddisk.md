# <a name="ivmharddiskconnectionharddisk-property"></a>Свойство Ивмхарддискконнектион:: жесткого диска

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Извлекает объект жесткого диска, соответствующий этому соединению.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_HardDisk(
  [out, retval] IVMHardDisk **hardDisk
);
```



## <a name="property-value"></a>Значение свойства

Объект [**ивмхарддиск**](ivmharddisk.md) , описывающий жесткий диск, связанный с этим соединением.
