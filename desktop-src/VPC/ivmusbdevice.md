# <a name="ivmusbdevice-interface"></a>Интерфейс Ивмусбдевице

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет интерфейс USB-устройства, подключенного к узлу. USB-устройство можно подключить к виртуальной машине для использования устройства внутри виртуальной машины.

## <a name="members"></a>Элементы

Интерфейс **ивмусбдевице** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмусбдевице** также имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Интерфейс **ивмусбдевице** имеет следующие свойства.



| Свойство                                                                 | Тип доступа          | Описание                                                                     |
|:-------------------------------------------------------------------------|:---------------------|:--------------------------------------------------------------------------------|
| [**аттачедтовм**](ivmusbdevice-attachedtovm.md)<br/>             | Только для чтения<br/> | Имя виртуальной машины, к которой подключено USB-устройство.<br/> |
| [**DeviceClass**](ivmusbdevice-deviceclass.md)<br/>               | Только для чтения<br/> | Класс устройства USB-устройства.<br/>                                  |
| [**девицестринг**](ivmusbdevice-devicestring.md)<br/>             | Только для чтения<br/> | Имя USB-устройства.<br/>                                          |
| [**хубид**](ivmusbdevice-hubid.md)<br/>                           | Только для чтения<br/> | Идентификатор концентратора, к которому подключено устройство.<br/>          |
| [**мануфактурерстринг**](ivmusbdevice-manufacturerstring.md)<br/> | Только для чтения<br/> | Имя изготовителя USB-устройства.<br/>                             |
| [**Порту**](ivmusbdevice-port.md)<br/>                             | Только для чтения<br/> | Идентификатор порта, к которому подключено устройство.<br/>         |
