# <a name="ivmdvddrive-interface"></a>Интерфейс Ивмдвддриве

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Управляет дисководом компакт-дисков или DVD-дисков в виртуальной машине. **Ивмдвддриве** может уведомлять клиентов о событиях с помощью исходящего интерфейса [**ивмдвддрививентс**](ivmdvddriveevents.md) .

На виртуальную машину можно добавить новый дисковод компакт-или DVD-дисков с помощью метода [**ивмвиртуалмачине:: адддвдромдриве**](ivmvirtualmachine-adddvdromdrive.md) . Вы можете удалить существующий компакт-диск или DVD-дисковод из виртуальной машины с помощью метода [**ивмвиртуалмачине:: ремоведвдромдриве**](ivmvirtualmachine-removedvdromdrive.md) . Объект **ивмдвддриве** можно получить из объекта [**ивмдвддривеколлектион**](ivmdvddrivecollection.md) , возвращенного из свойства [**ивмвиртуалмачине::D вдромдривес**](ivmvirtualmachine-dvdromdrives.md) .

## <a name="members"></a>Элементы

Интерфейс **ивмдвддриве** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмдвддриве** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ивмдвддриве** содержит следующие методы.



| Метод                                                   | Описание                                                                             |
|:---------------------------------------------------------|:----------------------------------------------------------------------------------------|
| [**аттачхостдриве**](ivmdvddrive-attachhostdrive.md)   | Подключает диск узла к DVD-дисководу в виртуальной машине.<br/>           |
| [**аттачимаже**](ivmdvddrive-attachimage.md)           | Подключает образ ISO к DVD-дисководу в виртуальной машине.<br/>           |
| [**релеасехостдриве**](ivmdvddrive-releasehostdrive.md) | Освобождает захваченный диск узла с DVD-диска.<br/>                           |
| [**релеасеимаже**](ivmdvddrive-releaseimage.md)         | Освобождает захваченный образ ISO с DVD-диска.<br/>                            |
| [**сетбуслокатион**](ivmdvddrive-setbuslocation.md)     | Подключает DVD-дисковод к указанному расположению шины на виртуальной машине.<br/> |



 

### <a name="properties"></a>Свойства

Интерфейс **ивмдвддриве** имеет следующие свойства.



| Свойство                                                          | Тип доступа          | Описание                                                                        |
|:------------------------------------------------------------------|:---------------------|:-----------------------------------------------------------------------------------|
| [**Вложение**](ivmdvddrive-attachment.md)<br/>           | Только для чтения<br/> | Тип носителя, подключенного к DVD-дисководу в виртуальной машине.<br/> |
| [**буснумбер**](ivmdvddrive-busnumber.md)<br/>             | Только для чтения<br/> | Номер шины, к которой подключено устройство.<br/>                        |
| [**девиценумбер**](ivmdvddrive-devicenumber.md)<br/>       | Только для чтения<br/> | Номер устройства, к которому подключен диск.<br/>                      |
| [**хостдривелеттер**](ivmdvddrive-hostdriveletter.md)<br/> | Только для чтения<br/> | Буква диска узла, заданная для этого устройства.<br/>                       |
| [**Изображения**](ivmdvddrive-imagefile.md)<br/>             | Только для чтения<br/> | Полный путь к набору файлов изображений для этого устройства.<br/>                    |
