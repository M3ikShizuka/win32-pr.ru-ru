# <a name="ivmharddiskconnection-interface"></a>Интерфейс Ивмхарддискконнектион

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет подключение для жесткого диска в виртуальной машине. Объект **ивмхарддискконнектион** возвращается из метода [**Ивмвиртуалмачине:: AddHardDiskConnection**](ivmvirtualmachine-addharddiskconnection.md) . Можно также получить объект **ивмхарддискконнектион** из объекта [**ивмхарддискконнектионколлектион**](ivmharddiskconnectioncollection.md) , возвращенного из свойства [**ивмвиртуалмачине:: харддискконнектионс**](ivmvirtualmachine-harddiskconnections.md) .

## <a name="members"></a>Элементы

Интерфейс **ивмхарддискконнектион** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмхарддискконнектион** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ивмхарддискконнектион** содержит следующие методы.



| Метод                                                         | Описание                                                           |
|:---------------------------------------------------------------|:----------------------------------------------------------------------|
| [**сетбуслокатион**](ivmharddiskconnection-setbuslocation.md) | Задает расположение шины, к которой подключен этот жесткий диск.<br/> |



 

### <a name="properties"></a>Свойства

Интерфейс **ивмхарддискконнектион** имеет следующие свойства.



| Свойство                                                              | Тип доступа          | Описание                                                                       |
|:----------------------------------------------------------------------|:---------------------|:----------------------------------------------------------------------------------|
| [**буснумбер**](ivmharddiskconnection-busnumber.md)<br/>       | Только для чтения<br/> | Номер шины, к которой подключен образ диска.<br/>                   |
| [**девиценумбер**](ivmharddiskconnection-devicenumber.md)<br/> | Только для чтения<br/> | Номер устройства, к которому подключен образ диска.<br/>                |
| [**Отключения**](ivmharddiskconnection-harddisk.md)<br/>         | Только для чтения<br/> | Объект жесткого диска, соответствующий этому соединению.<br/>                   |
| [**ундохарддиск**](ivmharddiskconnection-undoharddisk.md)<br/> | Только для чтения<br/> | Объект жесткого диска, соответствующий образу диска отмены этого подключения.<br/> |
