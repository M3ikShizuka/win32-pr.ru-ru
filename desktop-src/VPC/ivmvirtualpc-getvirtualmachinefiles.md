# <a name="ivmvirtualpcgetvirtualmachinefiles-method"></a>Метод Ивмвиртуалпк:: Жетвиртуалмачинефилес

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Извлекает массив известных файлов конфигурации виртуальной машины.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetVirtualMachineFiles(
  [in]          VARIANT      inAdditionalSearchPaths,
  [in]          VARIANT_BOOL inExcludedRegisteredVMs,
  [out, retval] VARIANT      *outVirtualMachineFileList
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*инаддитионалсеарчпасс* \[ окне\]
</dt> <dd>

Эти пути будут искаться вместе с путями, заданными в свойствах [**ивмвиртуалпк:: сеарчпасс**](ivmvirtualpc-searchpaths.md) и [**Ивмвиртуалпк::D ефаултвмконфигуратионпас**](ivmvirtualpc-defaultvmconfigurationpath.md) .

</dd> <dt>

*инексклудедрегистередвмс* \[ окне\]
</dt> <dd>

**Значение true** , если зарегистрированные виртуальные машины должны быть исключены из массива, возвращаемого параметром *Аутвиртуалмачинефилелист* , и **false** в противном случае.

</dd> <dt>

*аутвиртуалмачинефилелист* \[ out, retval\]
</dt> <dd>

Массив строк пути для файлов конфигурации виртуальной машины, найденных по указанным путям поиска.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                                        | Описание                                                                                     |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                              | Операция выполнена успешно.<br/>                                                        |
| <dl> <dt>**Д \_**</dt> <dt>0x80004003</dt> указателя </dl>                                | Параметр *аутвиртуалмачинефилелист* имеет **значение NULL**.<br/>                               |
| <dl> <dt>**Д \_ INVALIDARG**</dt> <dt>0x80000003</dt> </dl>                             | Параметр *инаддитионалсеарчпасс* не является массивом строк.<br/>                  |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>                        | Произошла непредвиденная ошибка.<br/>                                                    |

## <a name="remarks"></a>Remarks

Пути поиска, используемые для получения массива файлов конфигурации, будут содержать наборы, ранее [**ивмвиртуалпк:: сеарчпасс**](ivmvirtualpc-searchpaths.md) и [**Ивмвиртуалпк::D ефаултвмконфигуратионпас**](ivmvirtualpc-defaultvmconfigurationpath.md) в дополнение к тем, которые указаны параметром *инаддитионалсеарчпасс* .
