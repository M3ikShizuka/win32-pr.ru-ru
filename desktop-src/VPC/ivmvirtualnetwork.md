# <a name="ivmvirtualnetwork-interface"></a>Интерфейс Ивмвиртуалнетворк

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет виртуальную сеть. Объекты **ивмвиртуалнетворк** возвращаются из метода [**ивмвиртуалпк**](ivmvirtualpc.md) [**финдвиртуалнетворк**](ivmvirtualpc-findvirtualnetwork.md). Можно также получить объект **ивмвиртуалнетворк** из объекта [**ивмвиртуалнетворкколлектион**](ivmvirtualnetworkcollection.md) , возвращенного из свойства [**ивмвиртуалпк:: VirtualNetworks**](ivmvirtualpc-virtualnetworks.md) .

Виртуальная сеть состоит из виртуального коммутатора, который выполняет всю внутреннюю маршрутизацию и несколько соединений, подключенных к виртуальному коммутатору. Эти подключения могут быть реальным подключением к внешнему узлу, внутренней сетью или общей сетью (NAT).

## <a name="members"></a>Элементы

Интерфейс **ивмвиртуалнетворк** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмвиртуалнетворк** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ивмвиртуалнетворк** содержит следующие методы.



| Метод                                | Описание                                                          |
|:--------------------------------------|:---------------------------------------------------------------------|
| [**\_ID**](ivmvirtualnetwork--id.md) | Извлекает внутренний идентификатор виртуальной сети.<br/> |



 

### <a name="properties"></a>Свойства

Интерфейс **ивмвиртуалнетворк** имеет следующие свойства.



| Свойство                                                                | Тип доступа          | Описание                                                                    |
|:------------------------------------------------------------------------|:---------------------|:-------------------------------------------------------------------------------|
| [**хостадаптер**](ivmvirtualnetwork-hostadapter.md)<br/>         | Только для чтения<br/> | Имя адаптера, к которому подключена виртуальная сеть.<br/>  |
| [**Мультимедиа**](/previous-versions/windows/desktop/legacy/dd796707(v=vs.85))<br/>             | Только для чтения<br/> | Определяет, является ли экземпляр виртуальной сети беспроводным.<br/> |
| [**Имя**](ivmvirtualnetwork-name.md)<br/>                       | Только для чтения<br/> | Уникальное имя экземпляра виртуальной сети.<br/>                    |
| [**Адаптера**](ivmvirtualnetwork-networkadapters.md)<br/> | Только для чтения<br/> | Перечисляемая коллекция сетевых адаптеров, подключенных к виртуальной сети.<br/>   |
