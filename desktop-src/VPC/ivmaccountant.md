# <a name="ivmaccountant-interface"></a>Интерфейс Ивмаккаунтант

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Предоставляет доступ к сведениям, связанным с учетом, для виртуальной машины. Чтобы получить **ивмаккаунтант** для виртуальной машины, используйте свойство [**Ивмвиртуалмачине:: бухгалтер**](ivmvirtualmachine-accountant.md) .

## <a name="members"></a>Элементы

Интерфейс **ивмаккаунтант** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмаккаунтант** также имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Элемент Property

Интерфейс **ивмаккаунтант** имеет следующие свойства.



| Свойство.                                                                        | Тип доступа          | Описание                                                                                             |
|:--------------------------------------------------------------------------------|:---------------------|:--------------------------------------------------------------------------------------------------------|
| [**График**](ivmaccountant-cpuutilization.md)<br/>               | Только для чтения<br/> | Процент текущей загрузки ЦП для этой виртуальной машины.<br/>                          |
| [**кпуутилизатионхистори**](ivmaccountant-cpuutilizationhistory.md)<br/> | Только для чтения<br/> | Последняя загрузка ЦП этой виртуальной машины (в виде массива процентных значений).<br/>       |
| [**дискбитесреад**](ivmaccountant-diskbytesread.md)<br/>                 | Только для чтения<br/> | Общее число байтов, считанных всеми контроллерами хранилища для этой виртуальной машины.<br/>          |
| [**дискбитесвриттен**](ivmaccountant-diskbyteswritten.md)<br/>           | Только для чтения<br/> | Общее число байтов, записанных всеми контроллерами хранилища для этой виртуальной машины.<br/>       |
| [**нетворкбитесрецеивед**](ivmaccountant-networkbytesreceived.md)<br/>   | Только для чтения<br/> | Общее число байтов, полученных всеми виртуальными сетевыми адаптерами для этой виртуальной машины.<br/> |
| [**нетворкбитессент**](ivmaccountant-networkbytessent.md)<br/>           | Только для чтения<br/> | Общее число байтов, отправленных всеми виртуальными сетевыми адаптерами для этой виртуальной машины.<br/>     |
| [**Просто**](ivmaccountant-uptime.md)<br/>                               | Только для чтения<br/> | Количество секунд, в течение которых виртуальная машина была запущена.<br/>                             |
