# <a name="ivmmousegetbutton-method"></a>Метод Ивммаусе:: "Button"

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Извлекает текущее состояние (вверх или вниз) указанной кнопки мыши.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetButton(
  [in]          VMMouseButton buttonIndex,
  [out, retval] VARIANT_BOOL  *isDown
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*буттониндекс* \[ окне\]
</dt> <dd>

Индекс кнопки, состояние которой запрашивается. Список значений см. в разделе [**вммаусебуттон**](vmmousebutton.md).

</dd> <dt>

*раскрывающийся список* \[ out, retval\]
</dt> <dd>

Состояние указанной кнопки. **Значение true** , если кнопка в настоящее время отключена в виртуальной машине; в противном случае — **значение false** .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                        | Описание                                                                                     |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                              | Операция выполнена успешно.<br/>                                                        |
| <dl> <dt>**Д \_**</dt> <dt>0x80004003</dt> указателя </dl>                | Параметр " *Down* " имеет **значение NULL**.<br/>                                                  |
| <dl> <dt>**Д \_ INVALIDARG**</dt> <dt>0x80000003</dt> </dl>             | Недопустимый параметр *буттониндекс* .<br/>                                            |
| <dl> <dt>**Виртуальная машина \_ \_Виртуальная машина E \_ не \_ работает**</dt> <dt>0xA0040206</dt> </dl>   | Виртуальная машина, к которой подключено это устройство с мышью, в данный момент не выполняется.<br/> |
| <dl> <dt>**Виртуальная машина \_ Электронная \_ мышь \_ не \_ активна**</dt> <dt>0xA0040800</dt> </dl> | Устройство мыши не включено.<br/>                                            |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>        | Произошла непредвиденная ошибка.<br/>                                                    |
