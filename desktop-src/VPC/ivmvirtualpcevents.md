# <a name="ivmvirtualpcevents-interface"></a>Интерфейс Ивмвиртуалпцевентс

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет интерфейс исходящего события для интерфейса [**ивмвиртуалпк**](ivmvirtualpc.md) . Клиент реализует эти методы для получения событий, отправленных из [**ивмвиртуалпк**](ivmvirtualpc.md).

## <a name="members"></a>Элементы

Интерфейс **ивмвиртуалпцевентс** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмвиртуалпцевентс** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмвиртуалпцевентс** содержит следующие методы.



| Метод                                                        | Описание                                                                  |
|:--------------------------------------------------------------|:-----------------------------------------------------------------------------|
| [**оневентлогжед**](ivmvirtualpcevents-oneventlogged.md)     | Получает уведомление о том, что Windows Virtual PC регистрирует событие.<br/>      |
| [**онвмстатечанже**](ivmvirtualpcevents-onvmstatechange.md) | Получает уведомление о том, что состояние виртуальной машины изменилось.<br/> |
