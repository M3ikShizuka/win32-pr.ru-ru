# <a name="ivmhostinfoprocessorfeaturesstring-property"></a>Ивмхостинфо: свойство Роцессорфеатуресстринг:P

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Возвращает список возможностей, поддерживаемых процессором узла.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_ProcessorFeaturesString(
  [out, retval] BSTR *featuresString
);
```



## <a name="property-value"></a>Значение свойства

Разделенный запятыми список функций. Примером может быть "MMX, SSE, SSE2, x86-64".



| Значение                                                                                 | Значение                                                             |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------|
| <dl> <dt>"AMD-V"</dt> </dl>    | Поддерживает расширения виртуализации AMD.<br/>              |
| <dl> <dt>"Intel VT"</dt> </dl> | Поддерживает расширения технологии виртуализации Intel.<br/> |
| <dl> <dt>"ХАВД"</dt> </dl>     | Аппаратная виртуализация отключена.<br/>            |
| <dl> <dt>ИМЕЕТЕ</dt> </dl>     | Виртуализация с поддержкой оборудования включена.<br/>             |
| <dl> <dt>ТЕХНОЛОГИЕЙ</dt> </dl>      | Поддерживает расширения MMX.<br/>                             |
| <dl> <dt>ИСПОЛЬЗУЕМЫЙ</dt> </dl>      | Поддерживает расширения Streaming SIMD.<br/>                  |
| <dl> <dt>ПОДДЕРЖИВАЮЩ</dt> </dl>     | Поддерживает расширения Streaming SIMD Extensions 2.<br/>                |
| <dl> <dt>SSE3</dt> </dl>     | Поддерживает потоковые SIMD расширения 3.<br/>                |
| <dl> <dt>"ТКСТЕ"</dt> </dl>     | Поддерживает расширения технологии доверенного выполнения.<br/>    |
| <dl> <dt>"x86-64"</dt> </dl>   | Поддерживает расширения x86-64.<br/>                              |
