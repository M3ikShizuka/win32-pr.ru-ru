# <a name="ivmnetworkadapter_id-method"></a>Метод Ивмнетворкадаптер:: \_ ID

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Извлекает внутренний идентификатор этого сетевого интерфейса.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT _ID(
  [out] long *identifier
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*идентификатор* \[ заполняет\]
</dt> <dd>

Внутренний идентификатор.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                 | Описание                                     |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                       | Операция выполнена успешно.<br/>        |
| <dl> <dt>**Д \_**</dt> <dt>0x80004003</dt> указателя </dl>         | Параметр имеет **значение NULL**.<br/>           |
| <dl> <dt>**Виртуальная машина \_ \_Неизвестный \_ 0xA0040207 виртуальной машины**</dt> <dt></dt> </dl> | Не удается найти виртуальную машину.<br/> |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl> | В операции произошла неизвестная ошибка.<br/>  |
