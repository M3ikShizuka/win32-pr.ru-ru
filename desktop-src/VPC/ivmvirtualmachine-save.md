# <a name="ivmvirtualmachinesave-method"></a>Метод Ивмвиртуалмачине:: Save

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Сохраняет состояние виртуальной машины.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Save(
  [out, retval] IVMTask **saveTask
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*саветаск* \[ out, retval\]
</dt> <dd>

Объект [**ивмтаск**](ivmtask.md) , используемый для отслеживания хода выполнения последовательности сохранения состояния виртуальной машины.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                                          | Описание                                                                              |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                                | Операция выполнена успешно.<br/>                                                 |
| <dl> <dt>**Д \_ Сбой**</dt> <dt>0x80004005</dt> </dl>                                     | Не удалось сохранить виртуальную машину, так как диски отмены были помечены для удаления.<br/>    |
| <dl> <dt>**Д \_**</dt> <dt>0x80004003</dt> указателя </dl>                                  | Параметр имеет **значение NULL**.<br/>                                                    |
| <dl> <dt>**Значение HRESULT \_ ИЗ \_ Win32 (ошибка \_ \_ отказа в доступе)**</dt> <dt>0x80070005</dt> </dl> | Вызывающий объект должен иметь разрешения на выполнение для сохранения состояния этой виртуальной машины.<br/> |
| <dl> <dt>**Виртуальная машина \_ \_Виртуальная машина E \_ не \_ работает**</dt> <dt>0xA0040206</dt> </dl>                     | Виртуальная машина не запущена.<br/>                                                        |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>                          | Произошла непредвиденная ошибка.<br/>                                             |



 

## <a name="remarks"></a>Комментарии

Виртуальная машина будет отключена, когда задача **сохранения** достигнет завершения. В свойстве [**ивмвиртуалмачине:: State**](ivmvirtualmachine-state.md) будет содержаться **\_ Сохранение вмвмстате** во время сохранения, после чего **вмвмстате \_ сохраняется** после завершения сохранения и выключения виртуальной машины.
