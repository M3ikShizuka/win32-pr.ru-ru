# <a name="ivmguestoslogoff-method"></a>Метод Ивмгуестос:: logoff

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Выполнит выход всех пользователей из операционной системы на виртуальной машине.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Logoff(
  [out, retval] IVMTask **outLogoffTask
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*аутлогоффтаск* \[ out, retval\]
</dt> <dd>

Объект [**ивмтаск**](ivmtask.md) , используемый для отслеживания хода выполнения последовательности выхода.

</dd> </dl>

## <a name="remarks"></a>Комментарии

`HRESULT_FROM_WIN32(ERROR_NO_SUCH_LOGON_SESSION)` (0x80070520) возвращается с помощью свойства [**Error**](ivmtask-error.md) возвращенного объекта [**ивмтаск**](ivmtask.md) в гостевой операционной системе нет сеансов входа в систему.
