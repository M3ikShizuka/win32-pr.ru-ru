# <a name="ivmvirtualmachinestartup-method"></a>Метод Ивмвиртуалмачине:: Startup

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Запускает виртуальную машину из неинициализированного или сохраненного состояния.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Startup(
  [out, retval] IVMTask **startupTask
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*стартуптаск* \[ out, retval\]
</dt> <dd>

Объект [**ивмтаск**](ivmtask.md) , используемый для отслеживания хода выполнения последовательности запуска виртуальной машины.

</dd> </dl>
