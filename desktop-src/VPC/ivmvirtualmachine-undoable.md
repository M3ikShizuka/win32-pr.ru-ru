# <a name="ivmvirtualmachineundoable-property"></a>Свойство Ивмвиртуалмачине:: Undoed

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет, включены ли диски отмены для жестких дисков, подключенных к виртуальной машине.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_Undoable(
  [in]          VARIANT_BOOL enableUndo
);

HRESULT get_Undoable(
  [out, retval] VARIANT_BOOL *isUndoable
);
```



## <a name="property-value"></a>Значение свойства

Указывает **вариант \_ true** , если диски отмены включены для жестких дисков, ПОДКЛЮЧЕННЫХ к виртуальной машине, и **вариант \_ false** в противном случае.

## <a name="remarks"></a>Комментарии

Если отключены диски отмены, все существующие файлы диска отмены будут удалены.

Это свойство нельзя задать, если виртуальная машина запущена или сохранена.
