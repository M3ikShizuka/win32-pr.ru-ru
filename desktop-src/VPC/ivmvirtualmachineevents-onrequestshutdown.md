# <a name="ivmvirtualmachineeventsonrequestshutdown-method"></a>Метод Ивмвиртуалмачинивентс:: Онрекуестшутдовн

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Получает уведомление о том, что был выполнен запрос на завершение работы.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT OnRequestShutdown();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Если этот метод завершается успешно, возвращается значение **S \_ ОК**. В противном случае возвращается код ошибки **HRESULT** .

## <a name="remarks"></a>Remarks

Этот метод вызывается при каждом завершении сеанса виртуальной машины. Клиентская программа должна реализовать этот метод интерфейса для получения уведомления о событии **вмвиртуалмачинивент \_ рекуестшутдовн** , исходящем от [**ивмвиртуалмачине**](ivmvirtualmachine.md).

Это уведомление о событии отправляется только в том случае, если сеанс работы с виртуальными машинами скоро завершит работу. Подпрограммы завершения работы операционной системы, такие как [**ивмгуестос:: Shutdown**](ivmguestos-shutdown.md), не вызывают это событие напрямую, кроме случаев, когда они также пытаются выполнить программное выключение виртуального оборудования.
