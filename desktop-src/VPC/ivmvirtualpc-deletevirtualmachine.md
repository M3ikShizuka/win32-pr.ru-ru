# <a name="ivmvirtualpcdeletevirtualmachine-method"></a>Ивмвиртуалпк: метод:D Елетевиртуалмачине

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Удаляет конфигурацию виртуальной машины.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT DeleteVirtualMachine(
  [in] IVMVirtualMachine *virtualMachine
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*virtualMachine* \[ окне\]
</dt> <dd>

Указатель на объект [**ивмвиртуалмачине**](ivmvirtualmachine.md) , представляющий конфигурацию виртуальной машины, которую необходимо удалить.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                                        | Описание                                                                                     |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                              | Операция выполнена успешно.<br/>                                                        |
| <dl> <dt>**С \_ ЛОЖЬ**</dt> <dt>1</dt> </dl>                                           | Не удалось найти указанную конфигурацию.<br/>                                      |
| <dl> <dt>**Д \_**</dt> <dt>0x80004003</dt> указателя </dl>                                | Параметр *virtualMachine* имеет **значение NULL**.<br/>                                         |
| <dl> <dt>**Виртуальная машина \_ \_Виртуальная машина E \_ с**</dt> <dt>0xA0040500</dt> </dl>                        | Виртуальная машина запущена.<br/>                                                      |
| <dl> <dt>**Виртуальная машина \_ E \_ аппаратная \_ виртуализация \_ отключена**</dt> <dt>0xA0040951</dt> </dl> | Процессор не поддерживает расширения виртуализации с аппаратным ускорением (ЗАКОНЧИТЕ).<br/> |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>                        | Произошла непредвиденная ошибка.<br/>                                                    |



 

## <a name="remarks"></a>Remarks

Можно удалить только остановленные виртуальные машины. Обратите внимание, что все существующие данные о сохраненном состоянии или дисках отмены для этой конфигурации будут удалены в дополнение к файлу конфигурации.
