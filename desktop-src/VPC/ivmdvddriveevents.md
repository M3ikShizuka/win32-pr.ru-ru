# <a name="ivmdvddriveevents-interface"></a>Интерфейс Ивмдвддрививентс

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет интерфейс исходящего события для интерфейса [**ивмдвддриве**](ivmdvddrive.md) .

## <a name="members"></a>Элементы

Интерфейс **ивмдвддрививентс** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмдвддрививентс** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмдвддрививентс** содержит следующие методы.



| Метод                                                   | Описание                                                                   |
|:---------------------------------------------------------|:------------------------------------------------------------------------------|
| [**онмедиаежект**](ivmdvddriveevents-onmediaeject.md)   | Получает уведомление о том, что носитель был извлечен с диска.<br/>  |
| [**онмедиаинсерт**](ivmdvddriveevents-onmediainsert.md) | Получает уведомление о том, что носитель вставлен в диск.<br/> |
