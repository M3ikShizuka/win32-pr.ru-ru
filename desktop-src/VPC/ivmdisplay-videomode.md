# <a name="ivmdisplayvideomode-property"></a>Свойство Ивмдисплай:: Видеомоде

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Извлекает текущий режим видео.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_VideoMode(
  [out, retval] VMDisplayVideoMode *videoMode
);
```



## <a name="property-value"></a>Значение свойства

Текущий режим видео операционной системы на виртуальной машине. Список значений см. в разделе [**вмдисплайвидеомоде**](vmdisplayvideomode.md).
