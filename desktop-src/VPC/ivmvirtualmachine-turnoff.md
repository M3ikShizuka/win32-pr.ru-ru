# <a name="ivmvirtualmachineturnoff-method"></a>Метод Ивмвиртуалмачине:: Турнофф

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Отключит виртуальную машину.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT TurnOff(
  [out, retval] IVMTask **turnOffTask
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*турноффтаск* \[ out, retval\]
</dt> <dd>

Объект [**ивмтаск**](ivmtask.md) , используемый для отслеживания хода выполнения последовательности турнофф виртуальной машины.

</dd> </dl>

## <a name="remarks"></a>Комментарии

Этот метод выключает виртуальную машину таким же образом, как и отключение питания на физическом компьютере.
