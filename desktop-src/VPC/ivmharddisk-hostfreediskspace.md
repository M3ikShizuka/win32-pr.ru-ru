# <a name="ivmharddiskhostfreediskspace-property"></a>Свойство Ивмхарддиск:: Хостфридискспаце

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Получение оставшегося свободного места на диске, доступного на узле для виртуального жесткого диска.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_HostFreeDiskSpace(
  [out, retval] VARIANT *freeBytes
);
```



## <a name="property-value"></a>Значение свойства

Оставшийся объем свободного места на диске на главном компьютере для текущего файла образа жесткого диска. Это значение является **разновидностью** типа \_ Decimal.
