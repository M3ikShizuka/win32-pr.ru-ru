# <a name="ivmharddiskcompact-method"></a>Метод Ивмхарддиск:: Compact

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Сжимает динамически расширяемый образ виртуального жесткого диска.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Compact(
  [out, retval] IVMTask **compactTask
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*компакттаск* \[ out, retval\]
</dt> <dd>

Объект [**ивмтаск**](ivmtask.md) , который используется для отслеживания завершения процесса сжатия.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                                              | Описание                                                                                                                                     |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                                    | Операция выполнена успешно.<br/>                                                                                                        |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>                              | Произошла непредвиденная ошибка.<br/>                                                                                                    |
| <dl> <dt>**Д \_**</dt> <dt>0x80004003</dt> указателя </dl>                                      | Параметр имеет **значение NULL**.<br/>                                                                                                           |
| <dl> <dt>**Значение HRESULT \_ ИЗ \_ Win32 ( \_ \_ нарушение общего доступа к ошибкам)**</dt> <dt>0x80070020</dt> </dl> | Образ виртуального жесткого диска, на который ссылается этот объект [**ивмхарддиск**](ivmharddisk.md) , используется.<br/>                                  |
| <dl> <dt>**Значение HRESULT \_ ИЗ \_ Win32 (на \_ диске с ошибкой \_ Full)**</dt> <dt>0x80070070</dt> </dl>         | На томе узла недостаточно места для создания временного файла, необходимого для сжатия этого образа виртуального жесткого диска.<br/>     |
| <dl> <dt>**Виртуальная машина \_ \_Приложение E \_ завершает работу \_**</dt> <dt>0xA0040209</dt> </dl>                      | Невозможно сжать образ виртуального жесткого диска, так как приложение завершает работу.<br/>                                            |

| <dl> <dt>**Виртуальная машина \_ \_Неверный \_ \_ \_ тип**</dt> <dt>0xA004067Bного</dt> образа HD </dl>                   | Образ виртуального жесткого диска, на который ссылается этот объект [**ивмхарддиск**](ivmharddisk.md) , должен быть типом изображения **вмдисктипединамик** .<br/> |
| <dl> <dt>**Виртуальная машина \_ E \_ Недопустимый \_ \_ файл HD**</dt> <dt>0xA0040682</dt> </dl>                        | Возможно, образ виртуального жесткого диска, на который ссылается этот объект [**ивмхарддиск**](ivmharddisk.md) , не является допустимым образом.<br/>          |



 

## <a name="remarks"></a>Комментарии

Чтобы сжать динамически расширяемый образ жесткого диска, необходимо сначала обнулить его свободное пространство на образе диска.
