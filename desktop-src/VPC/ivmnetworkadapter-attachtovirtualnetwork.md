# <a name="ivmnetworkadapterattachtovirtualnetwork-method"></a>Метод Ивмнетворкадаптер:: Аттачтовиртуалнетворк

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Подключает сетевой интерфейс к указанной виртуальной сети.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT AttachToVirtualNetwork(
  [in] IVMVirtualNetwork *virtualNetwork
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*virtualNetwork* \[ окне\]
</dt> <dd>

Виртуальная сеть, к которой будет подключен этот виртуальный сетевой адаптер. См. [**ивмвиртуалнетворк**](ivmvirtualnetwork.md).

</dd> </dl>
