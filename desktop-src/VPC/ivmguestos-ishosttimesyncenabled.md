# <a name="ivmguestosishosttimesyncenabled-property"></a>Свойство Ивмгуестос:: Ишосттимесинценаблед

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Указывает, должны ли компоненты интеграции в этой виртуальной машине синхронизировать часы гостевой виртуальной машины с часами узла.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_IsHostTimeSyncEnabled(
  [in]          VARIANT_BOOL shouldEnable
);

HRESULT get_IsHostTimeSyncEnabled(
  [out, retval] VARIANT_BOOL *isEnabled
);
```



## <a name="property-value"></a>Значение свойства

Используйте **вариант \_ true** , если компоненты интеграции в этой виртуальной машине должны синхронизировать часы гостевой виртуальной машины с часами узла и **вариант \_ false** в противном случае.

## <a name="remarks"></a>Remarks

Этот параметр нельзя изменить, пока виртуальная машина активна (то есть запущена или находится в сохраненном состоянии).
