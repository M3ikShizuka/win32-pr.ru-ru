# <a name="ivmtaskwaitforcompletion-method"></a>Метод Ивмтаск:: Ваитфоркомплетион

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Ожидает завершения задачи или до истечения указанного интервала времени ожидания.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT WaitForCompletion(
  [in] long timeout
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*время ожидания* \[ окне\]
</dt> <dd>

Время в миллисекундах, в течение которого этот метод будет ожидать завершения задачи перед возвратом управления вызывающему объекту. Значение-1 указывает, что метод будет ожидать завершения задачи без истечения времени ожидания. Другие допустимые значения времени ожидания находятся в диапазоне от 0 до 4 000 000 миллисекунд.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                 | Описание                                      |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                       | Операция выполнена успешно.<br/>         |
| <dl> <dt>**Д \_ INVALIDARG**</dt> <dt>0x80000003</dt> </dl>      | Недопустимый параметр *времени ожидания* .<br/> |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl> | Произошла непредвиденная ошибка.<br/>     |



 

## <a name="remarks"></a>Комментарии

Метод **ваитфоркомплетион** помещает текущий поток выполнения в спящий режим до тех пор, пока не вернет значение. Указание бесконечного ожидания (timeout =-1) не рекомендуется, если не крайне важно, чтобы задача завершилась при каких-либо обстоятельствах.
