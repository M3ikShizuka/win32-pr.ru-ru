# <a name="ivmharddiskconnectionsetbuslocation-method"></a>Метод Ивмхарддискконнектион:: Сетбуслокатион

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Задает расположение шины, к которой подключен этот жесткий диск.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetBusLocation(
  [in] long vmBusNumber,
  [in] long vmDeviceNumber
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*вмбуснумбер* \[ окне\]
</dt> <dd>

Номер шины, которая будет использоваться.

</dd> <dt>

*вмдевиценумбер* \[ окне\]
</dt> <dd>

Номер используемого устройства.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                               | Описание                                                                                                 |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                     | Операция выполнена успешно.<br/>                                                                    |
| <dl> <dt>**Д \_ INVALIDARG**</dt> <dt>0x80000003</dt> </dl>                    | Указано недопустимое расположение шины.<br/>                                                         |
| <dl> <dt>**Виртуальная машина \_ \_Виртуальная машина \_ с установленным \_ или \_ сохраненным**</dt> <dt>0xA004020B</dt> </dl>    | Не удалось задать расположение шины, так как виртуальная машина находится в работающем или сохраненном состоянии.<br/>    |
| <dl> <dt>**Виртуальная машина \_ \_Шина диска \_ E \_ \_ с \_ помощью**</dt> <dt>0xA00400503</dt> </dl> | Не удалось установить расположение шины, так как в данный момент другое устройство подключено к этому расположению.<br/> |
| <dl> <dt>**Виртуальная машина \_ \_Диск E \_ Недопустимый**</dt> <dt>0xA0040502</dt> </dl>            | Текущий диск является недопустимым.<br/>                                                                  |
| <dl> <dt>**Виртуальная машина \_ \_Неизвестный \_ 0xA0040207 виртуальной машины**</dt> <dt></dt> </dl>               | Недопустимая Текущая виртуальная машина.<br/>                                                        |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>               | Произошла непредвиденная ошибка.<br/>                                                                |
