# <a name="ivmvirtualpcfindvirtualmachine-method"></a>Метод Ивмвиртуалпк:: Финдвиртуалмачине

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Извлекает объект виртуальной машины, соответствующий запрошенной конфигурации.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT FindVirtualMachine(
  [in]          BSTR              configurationName,
  [out, retval] IVMVirtualMachine **virtualMachine
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*configurationName* \[ окне\]
</dt> <dd>

Имя виртуальной машины, которую нужно найти.

</dd> <dt>

*virtualMachine* \[ out, retval\]
</dt> <dd>

Указатель на новый объект [**ивмвиртуалмачине**](ivmvirtualmachine.md) , представляющий эту виртуальную машину.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                                        | Описание                                                                                     |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                              | Операция выполнена успешно.<br/>                                                        |
| <dl> <dt>**С \_ ЛОЖЬ**</dt> <dt>1</dt> </dl>                                           | Не удалось найти указанную конфигурацию.<br/>                                      |
| <dl> <dt>**Д \_**</dt> <dt>0x80004003</dt> указателя </dl>                                | Параметр *configurationName* недопустим, или *VirtualMachine* имеет **значение NULL**.<br/>       |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>                        | Произошла непредвиденная ошибка.<br/>                                                    |
| <dl> <dt>**Виртуальная машина \_ E \_ аппаратная \_ виртуализация \_ отключена**</dt> <dt>0xA0040951</dt> </dl> | Процессор не поддерживает расширения виртуализации с аппаратным ускорением (ЗАКОНЧИТЕ).<br/> |



 

## <a name="remarks"></a>Комментарии

В именах виртуальных машин не учитывается регистр, например "MyVM" и "MyVM" ссылаются на одну и ту же виртуальную машину.
