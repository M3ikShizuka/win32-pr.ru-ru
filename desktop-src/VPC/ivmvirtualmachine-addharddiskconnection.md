# <a name="ivmvirtualmachineaddharddiskconnection-method"></a>Метод Ивмвиртуалмачине:: AddHardDiskConnection

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Добавляет новое подключение жесткого диска к виртуальной машине.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT AddHardDiskConnection(
  [in]          BSTR                  hardDiskPath,
  [in]          long                  busNumber,
  [in]          long                  deviceNumber,
  [out, retval] IVMHardDiskConnection **hardDiskConnection
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*харддискпас* \[ окне\]
</dt> <dd>

Полный путь к файлу виртуального жесткого диска (VHD) для подключения.

</dd> <dt>

*буснумбер* \[ окне\]
</dt> <dd>

Шина, к которой будет подключен диск.



| Значение                                                                        | Значение                                                  |
|------------------------------------------------------------------------------|----------------------------------------------------------|
| <dl> <dt>0</dt> </dl> | Диск будет подключен к первой шине.<br/>  |
| <dl> <dt>1</dt> </dl> | Диск будет подключен к второй шине.<br/> |



 

</dd> <dt>

*девиценумбер* \[ окне\]
</dt> <dd>

Устройство, к которому будет подключен диск.



| Значение                                                                        | Значение                                                                |
|------------------------------------------------------------------------------|------------------------------------------------------------------------|
| <dl> <dt>0</dt> </dl> | Диск будет подключен к первому устройству на шине.<br/>  |
| <dl> <dt>1</dt> </dl> | Диск будет подключен ко второму устройству на шине.<br/> |



 

</dd> <dt>

*харддискконнектион* \[ out, retval\]
</dt> <dd>

Объект [**ивмхарддискконнектион**](ivmharddiskconnection.md) .

</dd> </dl>
