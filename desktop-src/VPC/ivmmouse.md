# <a name="ivmmouse-interface"></a>Интерфейс Ивммаусе

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Управляет устройством мыши в пределах виртуальной машины. **Ивммаусе** для виртуальной машины можно получить с помощью свойства [**Ивмвиртуалмачине:: Mouse**](ivmvirtualmachine-mouse.md) . Координаты устройства мыши можно представить в абсолютных координатах или в разностных координатах. Используйте свойство [**усингабсолутекурдинатес**](ivmmouse-usingabsolutecoordinates.md) , чтобы различать два метода представления координат. Обратите внимание, что получение текущей позиции курсора и использование абсолютных координат поддерживаются только в том случае, если на гостевой операционной системе установлены компоненты интеграции.

## <a name="members"></a>Элементы

Интерфейс **ивммаусе** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивммаусе** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ивммаусе** содержит следующие методы.



| Метод                                  | Описание                                                                        |
|:----------------------------------------|:-----------------------------------------------------------------------------------|
| [**Щелкните**](ivmmouse-click.md)         | Имитирует нажатие кнопки мыши.<br/>                                         |
| [**Кнопка**](ivmmouse-getbutton.md) | Извлекает текущее состояние (вверх или вниз) указанной кнопки мыши.<br/> |
| [**сетбуттон**](ivmmouse-setbutton.md) | Задает текущее состояние (вверх или вниз) указанной кнопки мыши.<br/>      |



 

### <a name="properties"></a>Свойства

Интерфейс **ивммаусе** имеет следующие свойства.



| Свойство                                                                         | Тип доступа           | Описание                                                                                |
|:---------------------------------------------------------------------------------|:----------------------|:-------------------------------------------------------------------------------------------|
| [**хоризонталпоситион**](ivmmouse-horizontalposition.md)<br/>             | Чтение/запись<br/> | Абсолютная координата по оси x мыши.<br/>                                         |
| [**скроллвхилпоситион**](ivmmouse-scrollwheelposition.md)<br/>           | Только на запись<br/> | Z-координата мыши (относительная).<br/>                                  |
| [**усингабсолутекурдинатес**](ivmmouse-usingabsolutecoordinates.md)<br/> | Чтение/запись<br/> | Указывает, представляют ли координаты мыши абсолютные или относительные координаты.<br/> |
| [**вертикалпоситион**](ivmmouse-verticalposition.md)<br/>                 | Чтение/запись<br/> | Абсолютная координата y мыши.<br/>                                         |
