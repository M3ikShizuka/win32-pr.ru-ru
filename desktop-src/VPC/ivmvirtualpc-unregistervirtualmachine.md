# <a name="ivmvirtualpcunregistervirtualmachine-method"></a>Метод Ивмвиртуалпк:: Унрегистервиртуалмачине

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Отмена регистрации конфигурации виртуальной машины без удаления файла конфигурации.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT UnregisterVirtualMachine(
  [in] IVMVirtualMachine *virtualMachine
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*virtualMachine* \[ окне\]
</dt> <dd>

Указатель на объект [**ивмвиртуалмачине**](ivmvirtualmachine.md) , представляющий конфигурацию виртуальной машины, для которой необходимо отменить регистрацию.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                                        | Описание                                                                                     |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                              | Операция выполнена успешно.<br/>                                                        |
| <dl> <dt>**Д \_**</dt> <dt>0x80004003</dt> указателя </dl>                                | Параметр *virtualMachine* имеет **значение NULL**.<br/>                                         |
| <dl> <dt>**Виртуальная машина \_ \_Виртуальная машина E \_ с**</dt> <dt>0xA0040500</dt> </dl>                        | Виртуальная машина запущена.<br/>                                                                   |
| <dl> <dt>**Виртуальная машина \_ E \_ аппаратная \_ виртуализация \_ отключена**</dt> <dt>0xA0040951</dt> </dl> | Процессор не поддерживает расширения виртуализации с аппаратным ускорением (ЗАКОНЧИТЕ).<br/> |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>                        | Произошла непредвиденная ошибка.<br/>                                                    |



 

## <a name="remarks"></a>Комментарии

Отменить регистрацию можно только для остановленных виртуальных машин. Существующие данные о сохраненном состоянии или дисках отмены для этой конфигурации будут сохранены при отмене регистрации виртуальной машины.
