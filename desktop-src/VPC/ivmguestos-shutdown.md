# <a name="ivmguestosshutdown-method"></a>Метод Ивмгуестос:: Shutdown

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Завершает работу гостевой операционной системы в виртуальной машине.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Shutdown(
  [in]          VARIANT_BOOL isForced,
  [out, retval] IVMTask      **outShutdownTask
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*принудительно* \[ окне\]
</dt> <dd>

**Вариант \_ Значение TRUE** , если завершение работы должно быть принудительным; в противном случае — значение **\_ false** .

</dd> <dt>

*аутшутдовнтаск* \[ out, retval\]
</dt> <dd>

Объект [**ивмтаск**](ivmtask.md) , который используется для отслеживания завершения процесса завершения работы.

</dd> </dl>

## <a name="remarks"></a>Комментарии

При вызове этого метода виртуальная машина должна быть запущена, а компонент интеграции должен быть установлен. Этот метод поддерживается только для гостевых операционных систем под управлением Windows.
