# <a name="ivmvirtualmachinerdppipename-property"></a>Свойство Ивмвиртуалмачине:: Рдппипенаме

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Возвращает имя именованного канала RDP-подключения, используемого для видео и ввода.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_RdpPipeName(
  [out, retval] BSTR *RdpPipeName
);
```



## <a name="property-value"></a>Значение свойства

Имя именованного канала RDP-подключения, используемого для видео и ввода.

## <a name="error-codes"></a>Коды ошибок

Если метод завершается успешно, возвращается значение **S \_ ОК**. В противном случае возвращается код ошибки **HRESULT** .



| Имя/значение                                                                                                                                                         | Значение                                           |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------|
| <dl> <dt>С \_ ОК</dt> <dt>0</dt> </dl>                            | Операция выполнена успешно.<br/>          |
| <dl> <dt>Д \_ </dt> <dt>0x80004003</dt> указателя </dl>              | Параметр Рдппипенаме имеет **значение NULL**.<br/> |
| <dl> <dt>Виртуальная машина \_ \_Виртуальная машина E \_ не \_ работает</dt> <dt>0xA0040206</dt> </dl> | Виртуальная машина не запущена.<br/>    |
| <dl> <dt> \_ E \_ Exception</dt> <dt>0x80020009</dt> </dl>      | Произошла неизвестная ошибка.<br/>             |
