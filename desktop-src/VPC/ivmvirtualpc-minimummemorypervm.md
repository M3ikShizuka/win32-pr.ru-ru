# <a name="ivmvirtualpcminimummemorypervm-property"></a>Свойство Ивмвиртуалпк:: Минимуммеморипервм

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Возвращает минимально допустимое количество физической памяти на одну виртуальную машину в мегабайтах.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_MinimumMemoryPerVM(
  [out, retval] long *megabytesOfMemory
);
```



## <a name="property-value"></a>Значение свойства

Минимальное допустимое количество (в мегабайтах) физической памяти на виртуальную машину.
