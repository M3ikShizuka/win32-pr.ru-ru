# <a name="ivmparallelport-interface"></a>Интерфейс Ивмпараллелпорт

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет параллельный порт внутри виртуальной машины. Этот интерфейс позволяет настраивать параллельные порты внутри виртуальной машины. Объект **ивмпараллелпорт** можно получить из объекта [**ивмпараллелпортколлектион**](ivmparallelportcollection.md) , возвращенного из свойства [**ивмвиртуалмачине::P араллелпортс**](ivmvirtualmachine-parallelports.md) .

## <a name="members"></a>Элементы

Интерфейс **ивмпараллелпорт** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмпараллелпорт** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ивмпараллелпорт** содержит следующие методы.



| Метод                              | Описание                                                        |
|:------------------------------------|:-------------------------------------------------------------------|
| [**\_ID**](ivmparallelport--id.md) | Извлекает внутренний идентификатор параллельного порта.<br/> |



 

### <a name="properties"></a>Свойства

Интерфейс **ивмпараллелпорт** имеет следующие свойства.



| Свойство                                        | Тип доступа           | Описание                               |
|:------------------------------------------------|:----------------------|:------------------------------------------|
| [**Имя**](ivmparallelport-name.md)<br/> | Чтение/запись<br/> | Имя параллельного порта.<br/> |
