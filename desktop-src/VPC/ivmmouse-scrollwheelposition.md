# <a name="ivmmousescrollwheelposition-property"></a>Свойство Ивммаусе:: Скроллвхилпоситион

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Настраивает z-координату мыши (только относительную).

Это свойство доступно только на запись.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_ScrollWheelPosition(
  [in] long position
);
```



## <a name="property-value"></a>Значение свойства

Координата z, указывающая количество пикселей, на которое мышь перемещается вдоль оси z.
