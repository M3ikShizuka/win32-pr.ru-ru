# <a name="ivmserialportcollection-interface"></a>Интерфейс Ивмсериалпортколлектион

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет коллекцию последовательных портов в пределах виртуальной машины. Чтобы получить объект **ивмсериалпортколлектион** , используйте свойство [**Ивмвиртуалмачине:: сериалпортс**](ivmvirtualmachine-serialports.md) .

## <a name="members"></a>Элементы

Интерфейс **ивмсериалпортколлектион** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмсериалпортколлектион** также имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Интерфейс **ивмсериалпортколлектион** имеет следующие свойства.



| Свойство                                                         | Тип доступа          | Описание                                                                |
|:-----------------------------------------------------------------|:---------------------|:---------------------------------------------------------------------------|
| [**\_NewEnum**](ivmserialportcollection--newenum.md)<br/> | Только для чтения<br/> | Перечислитель для коллекции.<br/>                               |
| [**Расчета**](ivmserialportcollection-count.md)<br/>        | Только для чтения<br/> | Число последовательных портов в этой коллекции.<br/>                  |
| [**Элемент**](ivmserialportcollection-item.md)<br/>          | Только для чтения<br/> | Объект последовательного порта, соответствующий указанному индексу.<br/> |
