# <a name="ivmguestosisheartbeating-property"></a>Свойство Ивмгуестос::

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет, имеет ли виртуальная машина пульс.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_IsHeartbeating(
  [out, retval] VARIANT_BOOL *heartBeating
);
```



## <a name="property-value"></a>Значение свойства

**Вариант \_ Значение TRUE** , если период пульса обнаружен; в противном случае — значение **\_ false** .

## <a name="remarks"></a>Remarks

при установке компонентов интеграции в операционной системе на виртуальной машине для Windows virtual PC в сеансе виртуальной машины отправляется обычный такт или пульс. Если операционная система на виртуальной машине сильно загружена, возможно, виртуальная машина будет принимать меньше периодических сигналов, чем ожидалось. Если пульс не обнаружен, возможно, что гостевая операционная система не отвечает на запросы или происходит сбой.

По умолчанию виртуальная машина выдает десять тактов пульса в минуту. если тактовые импульсы не обнаруживаются за всю минуту, Windows virtual PC попытается перезапустить сеанс виртуальной машины каждые десять секунд до двух минут. Это поведение управляется следующими значениями ключей в файле конфигурации сеанса виртуальной машины.



| Ключ конфигурации                                            | Значение по умолчанию       | Описание                                                                                                                             |
|--------------------------------------------------------------|---------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| интеграция/Microsoft/пульс/время<br/>              | 60<br/> | Длина блока времени, используемого для создания тактов пульса в секундах.<br/>                                             |
| интеграция/Microsoft/пульс/ставка<br/>              | 10<br/> | Число тактов, созданных в каждом блоке времени пульса.<br/>                                                                  |
| интеграция/Microsoft/пульс/ \_ интервал сбоев<br/> | 10<br/> | Время в секундах между попытками перезапуска, по истечении которого тактовые импульсы не будут получены в течение определенного блока времени пульса.<br/> |
| интеграция/Microsoft/пульс/неудачные \_ попытки<br/> | 12<br/> | Число выполненных попыток перезапуска.<br/>                                                                                         |
