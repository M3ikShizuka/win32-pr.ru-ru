# <a name="ivmvirtualpccreatedynamicvirtualharddisk-method"></a>Метод Ивмвиртуалпк:: Креатединамиквиртуалхарддиск

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Создает динамическое изменение размера виртуального жесткого диска.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT CreateDynamicVirtualHardDisk(
  [in]          BSTR    imagePath,
  [in]          long    size,
  [out, retval] IVMTask **diskTask
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*imagePath* \[ окне\]
</dt> <dd>

Полный путь к новому файлу образа диска. Содержащаяся папка будет создана, если она не существует.

</dd> <dt>

*Размер* \[ окне\]
</dt> <dd>

Размер изображения в мегабайтах. Это значение не может превышать 2 088 960 МБ (2040GB).

</dd> <dt>

*дисктаск* \[ out, retval\]
</dt> <dd>

Объект [**ивмтаск**](ivmtask.md) , используемый для отслеживания создания образа.

</dd> </dl>
