# <a name="ivmserialportconfigure-method"></a>Метод Ивмсериалпорт:: Configure

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Настраивает последовательный порт.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Configure(
  [in] VMSerialPortType portType,
  [in] BSTR             portName,
  [in] VARIANT_BOOL     vmConnectImmediately
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*portType* \[ окне\]
</dt> <dd>

Тип последовательного порта. Список значений см. в разделе [**вмсериалпорттипе**](vmserialporttype.md).

</dd> <dt>

*портнаме* \[ окне\]
</dt> <dd>

Имя последовательного порта. Например, "COM1" для **вмсериалпорт \_ хостпорт**, "C: \\SerialPort.txt" для **вмсериалпорт \_ TextFile** или " \\ \\ *ServerName* \\ pipe \\ *pipeName*" для **вмсериалпорт \_ помощью канала NamedPipe**.

</dd> <dt>

*вмконнектиммедиатели* \[ окне\]
</dt> <dd>

**Значение true** , если последовательный порт узла должен быть открыт сразу же после запуска виртуальной машины; в противном случае — **значение false** . Игнорируется, если *portType* не **вмсериалпорт \_ хостпорт**.

</dd> </dl>
