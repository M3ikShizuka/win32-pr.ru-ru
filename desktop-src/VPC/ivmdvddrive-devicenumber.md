# <a name="ivmdvddrivedevicenumber-property"></a>Ивмдвддриве: свойство Евиценумбер:D

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Возвращает номер устройства, к которому подключен этот диск.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_DeviceNumber(
  [out, retval] long *vmDeviceNumber
);
```



## <a name="property-value"></a>Значение свойства

Номер устройства. Например, на шине IDE это значение будет представлять собой либо основное, либо дополнительное расположение.
