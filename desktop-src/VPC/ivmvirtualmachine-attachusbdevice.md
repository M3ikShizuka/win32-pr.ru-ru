# <a name="ivmvirtualmachineattachusbdevice-method"></a>Метод Ивмвиртуалмачине:: Аттачусбдевице

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Подключает USB-устройство к виртуальной машине.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT AttachUSBDevice(
  [in] IVMUSBDevice *inUSBDevice
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*инусбдевице* \[ окне\]
</dt> <dd>

Указатель [**ивмусбдевице**](ivmusbdevice.md) , представляющий USB-устройство, подключенное к узлу.

</dd> </dl>
