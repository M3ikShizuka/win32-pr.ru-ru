# <a name="ivmguestosgetosversioninfo-method"></a>Метод Ивмгуестос:: Жетосверсионинфо

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Извлекает сведения о версии гостевой операционной системы, работающей на виртуальной машине.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetOsVersionInfo(
  [out, retval] GUESTOSVERSIONINFOEX **guestOsVersionInfo
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*гуестосверсионинфо* \[ out, retval\]
</dt> <dd>

Указатель на структуру [**гуестосверсионинфоекс**](guestosversioninfoex.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                                       | Описание                                                               |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                             | Операция выполнена успешно.<br/>                                  |
| <dl> <dt>**Д \_**</dt> <dt>0x80004003</dt> указателя </dl>                               | Параметр имеет **значение NULL**.<br/>                                     |
| <dl> <dt>**Значение HRESULT \_ ИЗ \_ Win32 (Error \_ OUTOFMEMORY)**</dt> <dt>0x8007000E</dt> </dl> | Недостаточно памяти для выполнения этого запроса.<br/> |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>                       | Произошла непредвиденная ошибка.<br/>                              |
| <dl> <dt>**Виртуальная машина \_ \_Виртуальная машина E \_ не \_ работает**</dt> <dt>0xA0040206</dt> </dl>                  | Виртуальная машина не запущена.<br/>                                         |
| <dl> <dt>**Виртуальная машина \_ Добавление компонента "E" \_ \_ \_ не \_**</dt> доступно <dt>0xA0040505</dt> </dl>    | Компоненты интеграции не установлены в этой виртуальной машине.<br/>           |
