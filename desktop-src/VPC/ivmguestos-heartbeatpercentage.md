# <a name="ivmguestosheartbeatpercentage-property"></a>Свойство Ивмгуестос:: Хеартбеатперцентаже

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Возвращает процент ожидаемых периодических сигналов, полученных за последнюю минуту.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_HeartbeatPercentage(
  [out, retval] long *heartbeatPercentage
);
```



## <a name="property-value"></a>Значение свойства

Процент ожидаемых периодических сигналов, полученных за последнюю минуту.

## <a name="remarks"></a>Комментарии

Компоненты интеграции будут отсылать периодический пульс на виртуальный ПК Windows, пока выполняется гостевая операционная система. Если операционная система на виртуальной машине сильно загружена, возможно, Windows Virtual PC будет принимать меньше периодических сигналов, чем ожидалось. Если процент пульса падает до нуля, возможно, что гостевая операционная система не отвечает на запросы или происходит сбой. Виртуальная машина должна быть запущена (то есть полностью загружена и не завершается), а компоненты интеграции должны быть установлены при вызове этого свойства.
