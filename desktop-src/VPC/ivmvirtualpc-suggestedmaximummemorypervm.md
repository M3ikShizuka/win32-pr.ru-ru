# <a name="ivmvirtualpcsuggestedmaximummemorypervm-property"></a>Свойство Ивмвиртуалпк:: Сугжестедмаксимуммеморипервм

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Извлекает рекомендуемое максимально допустимое количество физической памяти на виртуальную машину в мегабайтах, чтобы избежать нехватки памяти на узле.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_SuggestedMaximumMemoryPerVM(
  [out, retval] long *megabytesOfMemory
);
```



## <a name="property-value"></a>Значение свойства

Рекомендуемое максимально допустимое количество (в мегабайтах) физической памяти на виртуальную машину.
