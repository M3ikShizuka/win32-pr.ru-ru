# <a name="ivmparallelportcollection-interface"></a>Интерфейс Ивмпараллелпортколлектион

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет коллекцию параллельных портов в виртуальной машине. Чтобы получить объект **ивмпараллелпортколлектион** , используйте свойство [**Ивмвиртуалмачине::P араллелпортс**](ivmvirtualmachine-parallelports.md) .

## <a name="members"></a>Элементы

Интерфейс **ивмпараллелпортколлектион** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмпараллелпортколлектион** также имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Интерфейс **ивмпараллелпортколлектион** имеет следующие свойства.



| Свойство                                                           | Тип доступа          | Описание                                                                  |
|:-------------------------------------------------------------------|:---------------------|:-----------------------------------------------------------------------------|
| [**\_NewEnum**](ivmparallelportcollection--newenum.md)<br/> | Только для чтения<br/> | Перечислитель для коллекции.<br/>                                 |
| [**Расчета**](ivmparallelportcollection-count.md)<br/>        | Только для чтения<br/> | Число параллельных портов в этой коллекции.<br/>                  |
| [**Элемент**](ivmparallelportcollection-item.md)<br/>          | Только для чтения<br/> | Объект параллельного порта, соответствующий указанному индексу.<br/> |
