# <a name="ivmvirtualpcmaximumnetworkadapterspervm-property"></a>Свойство Ивмвиртуалпк:: Максимумнетворкадаптерспервм

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Возвращает максимальное число сетевых интерфейсов на виртуальную машину.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_MaximumNetworkAdaptersPerVM(
  [out, retval] long *maxNetworkAdapters
);
```



## <a name="property-value"></a>Значение свойства

Максимальное число сетевых интерфейсов на виртуальную машину.
