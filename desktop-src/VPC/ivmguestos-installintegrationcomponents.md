# <a name="ivmguestosinstallintegrationcomponents-method"></a>Метод Ивмгуестос:: Инсталлинтегратионкомпонентс

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Находит и устанавливает последние компоненты интеграции в операционную систему на виртуальной машине.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT InstallIntegrationComponents();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                                            | Описание                                                                            |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                                  | Операция выполнена успешно.<br/>                                               |
| <dl> <dt>**Виртуальная машина \_ \_Виртуальная машина E \_ не \_ работает**</dt> <dt>0xA0040206</dt> </dl>                       | Для этой операции должна быть запущена виртуальная машина.<br/>                     |
| <dl> <dt>**Виртуальная машина \_ \_Неизвестный \_ 0xA0040207 виртуальной машины**</dt> <dt></dt> </dl>                            | Не удалось найти виртуальную машину.<br/>                                     |
| <dl> <dt>**Виртуальная машина \_ \_Диск E \_ Недопустимый**</dt> <dt>0xA0040502</dt> </dl>                         | Виртуальная машина не имеет пустого DVD-дисковода.<br/>                       |
| <dl> <dt>**Виртуальная машина \_ \_ \_ \_ Ошибка отключения носителя**</dt> <dt>0xA0040508</dt> </dl>                   | Сбой при попытке отключить носитель от DVD-дисковода виртуальной машины.<br/> |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>                            | Произошла непредвиденная ошибка.<br/>                                           |
| <dl> <dt>**Значение HRESULT \_ ИЗ \_ Win32 ( \_ файл ошибки \_ не \_ найден)**</dt> <dt>0x80070002</dt> </dl> | Системе не удается найти ISO-файл для компонентов интеграции.<br/>         |
