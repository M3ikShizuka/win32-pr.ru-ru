# <a name="ivmvirtualmachinesavedstatefilepath-property"></a>Свойство Ивмвиртуалмачине:: Саведстатефилепас

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Получает полный путь к сохраненному файлу состояния.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_SavedStateFilePath(
  [out, retval] BSTR *savedStateFilePath
);
```



## <a name="property-value"></a>Значение свойства

Полный путь к сохраненному файлу состояния виртуальной машины.
