# <a name="ivmdvddrivesetbuslocation-method"></a>Метод Ивмдвддриве:: Сетбуслокатион

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Подключает DVD-дисковод к указанному расположению шины на виртуальной машине.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetBusLocation(
  [in] long busNumber,
  [in] long deviceNumber
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*буснумбер* \[ окне\]
</dt> <dd>

Номер шины, к которой должен быть подключен диск. Например, на шине IDE это число указывает, следует ли использовать номер основной или дополнительной шины.

</dd> <dt>

*девиценумбер* \[ окне\]
</dt> <dd>

Номер устройства, к которому будет присоединен этот диск. Например, на шине IDE это число будет представлять необходимость использования основного или дополнительного расположения устройства.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                            | Описание                                                                                                                             |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                  | Операция выполнена успешно.<br/>                                                                                                |
| <dl> <dt>**Д \_ INVALIDARG**</dt> <dt>0x80000003</dt> </dl>                 | Указано недопустимое расположение шины.<br/>                                                                                     |
| <dl> <dt>**Д \_ Сбой**</dt> <dt>0x80004005</dt> </dl>                       | Произошла непредвиденная ошибка.<br/>                                                                                            |
| <dl> <dt>**Виртуальная машина \_ \_Виртуальная машина \_ с установленным \_ или \_ сохраненным**</dt> <dt>0xA004020B</dt> </dl> | Невозможно задать расположение шины, пока виртуальная машина работает или находится в сохраненном состоянии.<br/>                                     |
| <dl> <dt>**\_ \_ \_ \_ используемая шина виртуальной машины \_ E**</dt> </dl>                                                                      | Другое устройство уже подключено к указанному расположению шины.<br/>                                                            |
| <dl> <dt>**Виртуальная машина \_ \_Диск E \_ Недопустимый**</dt> <dt>0xA0040502</dt> </dl>         | Не удалось инициализировать текущий диск.<br/>                                                                                  |
| <dl> <dt>**Виртуальная машина \_ \_Неизвестный \_ 0xA0040207 виртуальной машины**</dt> <dt></dt> </dl>            | Не удалось записать изменения в файл настроек, так как не удалось определить виртуальную машину для этого диска.<br/> |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>            | Произошла непредвиденная ошибка.<br/>                                                                                            |
