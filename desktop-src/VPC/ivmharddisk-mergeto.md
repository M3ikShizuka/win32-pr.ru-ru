# <a name="ivmharddiskmergeto-method"></a>Метод Ивмхарддиск:: Мержето

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Слияние разностного виртуального жесткого диска со всеми его родительскими элементами (вплоть до корневого родительского виртуального жесткого диска и включая его) в новый файл жесткого диска.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT MergeTo(
  [in]          BSTR           newDiskImagePath,
  [in]          VMHardDiskType newDiskImageType,
  [out, retval] IVMTask        **mergeTask
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*невдискимажепас* \[ окне\]
</dt> <dd>

Путь к новому целевому образу диска, в который будут объединены выбранные образы дисков.

</dd> <dt>

*невдискимажетипе* \[ окне\]
</dt> <dd>

Тип нового образа целевого диска. Для нового образа целевого диска допустимы следующие типы образов: **вмдисктипе \_ dynamic** и **вмдисктипе \_ FixedSize**. Дополнительные сведения см. в разделе [**вмхарддисктипе**](vmharddisktype.md).

</dd> <dt>

*мержетаск* \[ out, retval\]
</dt> <dd>

Объект [**ивмтаск**](ivmtask.md) , который используется для отслеживания завершения процесса слияния.

</dd> </dl>
