# <a name="ivmguestosscreenlocked-property"></a>Свойство Ивмгуестос:: Скринлоккед

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Указывает, заблокирован ли экран в операционной системе на виртуальной машине.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_ScreenLocked(
  [out, retval] VARIANT_BOOL *guestScreenLocked
);
```



## <a name="property-value"></a>Значение свойства

**Вариант \_ TRUE** , если экран заблокирован в гостевой операционной системе, в противном случае — значение **\_ false** .
