# <a name="ivmvirtualmachineevents-interface"></a>Интерфейс Ивмвиртуалмачинивентс

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет интерфейс исходящего события для интерфейса [**ивмвиртуалмачине**](ivmvirtualmachine.md) . Клиент реализует эти методы для получения событий, отправленных из [**ивмвиртуалмачине**](ivmvirtualmachine.md).

## <a name="members"></a>Элементы

Интерфейс **ивмвиртуалмачинивентс** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмвиртуалмачинивентс** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмвиртуалмачинивентс** содержит следующие методы.



| Метод                                                                                   | Описание                                                                                                                                     |
|:-----------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**онаддитионсаваилабле**](ivmvirtualmachineevents-onadditionsavailable.md)             | Получает уведомление о том, что компоненты интеграции доступны на виртуальной машине.<br/>                                                |
| [**онаддитионсунинсталлед**](ivmvirtualmachineevents-onadditionsuninstalled.md)         | Получает уведомление об удалении компонентов интеграции на виртуальной машине.<br/>                                              |
| [**онконфигуратиончанжед**](ivmvirtualmachineevents-onconfigurationchanged.md)         | Получает уведомление о том, что значение в конфигурации этой виртуальной машины изменилось.<br/>                                        |
| [**ондискаутофспаце**](ivmvirtualmachineevents-ondiskoutofspace.md)                     | Получает уведомление о том, что на диске, требуемом для виртуальной машины, недостаточно места, а виртуальная машина не может быть запущена.<br/>           |
| [**оненханцедвидеомодечанжед**](ivmvirtualmachineevents-onenhancedvideomodechanged.md) | Получает уведомление о том, что изменилась поддержка виртуальной машины для расширенного видеорежима.<br/>                                          |
| [**онгуестлогофф**](ivmvirtualmachineevents-onguestlogoff.md)                           | Получает уведомление о том, что пользователь вышел из операционной системы на виртуальной машине.<br/>                                                    |
| [**онгуестшутдовн**](ivmvirtualmachineevents-onguestshutdown.md)                       | Получает уведомление о завершении работы гостевой операционной системы.<br/>                                                                     |
| [**онхеартбеатстоппед**](ivmvirtualmachineevents-onheartbeatstopped.md)                 | Получает уведомление о том, что пульс виртуальной машины остановлен. Обычно это свидетельствует о сбое операционной системы на виртуальной машине.<br/> |
| [**онрекуестшутдовн**](ivmvirtualmachineevents-onrequestshutdown.md)                   | Получает уведомление о том, что был выполнен запрос на завершение работы.<br/>                                                                         |
| [**OnReset**](ivmvirtualmachineevents-onreset.md)                                       | Получает уведомление о том, что виртуальная машина сброшена.<br/>                                                                         |
| [**онстатечанже**](ivmvirtualmachineevents-onstatechange.md)                           | Получает уведомление о том, что состояние виртуальной машины изменилось.<br/>                                                                    |
| [**онтриплефаулт**](ivmvirtualmachineevents-ontriplefault.md)                           | Получает уведомление о том, что на виртуальной машине возникла тройная ошибка.<br/>                                                                     |
