# <a name="ivmvirtualpcfindvirtualnetwork-method"></a>Метод Ивмвиртуалпк:: Финдвиртуалнетворк

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Извлекает объект виртуальной сети, соответствующий запрошенному имени.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT FindVirtualNetwork(
  [in]          BSTR              virtualNetworkName,
  [out, retval] IVMVirtualNetwork **virtualNetwork
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*virtualNetworkName* \[ окне\]
</dt> <dd>

Имя искомой виртуальной сети.

</dd> <dt>

*virtualNetwork* \[ out, retval\]
</dt> <dd>

Указатель на новый объект [**ивмвиртуалнетворк**](ivmvirtualnetwork.md) , представляющий эту виртуальную сеть.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                                            | Описание                                                                                     |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                                  | Операция выполнена успешно.<br/>                                                        |
| <dl> <dt>**С \_ ЛОЖЬ**</dt> <dt>1</dt> </dl>                                               | Не удалось найти указанную виртуальную сеть.<br/>                                    |
| <dl> <dt>**Д \_**</dt> <dt>0x80004003</dt> указателя </dl>                                    | Параметр *virtualNetwork* имеет **значение NULL** , или виртуальная сеть не найдена.<br/>   |
| <dl> <dt>**Д \_ INVALIDARG**</dt> <dt>0x80000003</dt> </dl>                                 | Параметр *virtualNetworkName* недопустим или является пустой строкой.<br/>               |
| <dl> <dt>**Значение HRESULT \_ ИЗ \_ Win32 ( \_ файл ошибки \_ не \_ найден)**</dt> <dt>0x80070002</dt> </dl> | Не удалось найти файл виртуальной сети.<br/>                                         |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>                            | Произошла непредвиденная ошибка.<br/>                                                    |
| <dl> <dt>**Виртуальная машина \_ E \_ аппаратная \_ виртуализация \_ отключена**</dt> <dt>0xA0040951</dt> </dl>     | Процессор не поддерживает расширения виртуализации с аппаратным ускорением (ЗАКОНЧИТЕ).<br/> |



 

## <a name="remarks"></a>Комментарии

Имена виртуальных сетей не учитывают регистр, например "Минетворк" и "минетворк" ссылаются на одну и ту же виртуальную сеть.
