# <a name="ivmharddiskconnectionbusnumber-property"></a>Свойство Ивмхарддискконнектион:: Буснумбер

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Возвращает номер шины, к которой подключен образ диска.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_BusNumber(
  [out, retval] long *vmBusNumber
);
```



## <a name="property-value"></a>Значение свойства

Номер шины, соответствующий этому соединению.
