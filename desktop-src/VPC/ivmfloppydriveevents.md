# <a name="ivmfloppydriveevents-interface"></a>Интерфейс Ивмфлоппидрививентс

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет интерфейс исходящего события для интерфейса [**ивмфлоппидриве**](ivmfloppydrive.md) . Клиент реализует эти методы для получения событий, отправленных из **ивмфлоппидриве**.

## <a name="members"></a>Элементы

Интерфейс **ивмфлоппидрививентс** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмфлоппидрививентс** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмфлоппидрививентс** содержит следующие методы.



| Метод                                                      | Описание                                                                   |
|:------------------------------------------------------------|:------------------------------------------------------------------------------|
| [**онмедиаежект**](ivmfloppydriveevents-onmediaeject.md)   | Получает уведомление о том, что носитель был извлечен с диска.<br/>  |
| [**онмедиаинсерт**](ivmfloppydriveevents-onmediainsert.md) | Получает уведомление о том, что носитель вставлен в диск.<br/> |
