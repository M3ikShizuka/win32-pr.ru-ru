# <a name="ivmdvddriveattachment-property"></a>Свойство Ивмдвддриве:: вложение

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Возвращает тип носителя, подключенного к DVD-дисководу в виртуальной машине.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_Attachment(
  [out, retval] VMDVDDriveAttachmentType *driveAttachment
);
```



## <a name="property-value"></a>Значение свойства

Тип подключенного носителя. Список значений см. в разделе [**вмдвддривеаттачменттипе**](vmdvddriveattachmenttype.md).
