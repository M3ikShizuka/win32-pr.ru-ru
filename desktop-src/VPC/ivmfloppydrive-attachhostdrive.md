# <a name="ivmfloppydriveattachhostdrive-method"></a>Метод Ивмфлоппидриве:: Аттачхостдриве

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Подключает физический диск на узле к дисководу гибких дисков виртуальной машины.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT AttachHostDrive(
  [in] BSTR driveLetter
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*буква_диска* \[ окне\]
</dt> <dd>

Буква диска физического дисковода гибких дисков на хост-компьютере, который должен быть подключен.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                 | Описание                                                                                                  |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                       | Операция выполнена успешно.<br/>                                                                     |
| <dl> <dt>**Д \_ INVALIDARG**</dt> <dt>0x80000003</dt> </dl>      | Параметр *буква_диска* имеет **значение NULL**, не является допустимым дисководом гибких дисков, или указанный путь пуст.<br/> |
| <dl> <dt>**Виртуальная машина \_ \_Неизвестный \_ 0xA0040207 виртуальной машины**</dt> <dt></dt> </dl> | Конфигурация этой виртуальной машины недопустима или не найдена.<br/>                       |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl> | Произошла непредвиденная ошибка.<br/>                                                                 |
