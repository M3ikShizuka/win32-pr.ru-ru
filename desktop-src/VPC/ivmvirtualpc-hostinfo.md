# <a name="ivmvirtualpchostinfo-property"></a>Свойство Ивмвиртуалпк:: Хостинфо

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Извлекает сведения о физическом компьютере.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_HostInfo(
  [out, retval] IVMHostInfo **hostInfo
);
```



## <a name="property-value"></a>Значение свойства

Объект [**ивмхостинфо**](ivmhostinfo.md) , содержащий сведения о хосте компьютера.
