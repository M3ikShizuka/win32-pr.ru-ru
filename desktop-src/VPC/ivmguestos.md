# <a name="ivmguestos-interface"></a>Интерфейс Ивмгуестос

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет гостевую операционную систему, работающую на виртуальной машине. Этот интерфейс позволяет взаимодействовать с компонентами интеграции, работающими в гостевой операционной системе. **Ивмгуестос** для виртуальной машины можно получить с помощью свойства [**Ивмвиртуалмачине:: гуестос**](ivmvirtualmachine-guestos.md) .

## <a name="members"></a>Элементы

Интерфейс **ивмгуестос** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмгуестос** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ивмгуестос** содержит следующие методы.



| Метод                                                                          | Описание                                                                                             |
|:--------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------|
| [**жетосверсионинфо**](ivmguestos-getosversioninfo.md)                         | Извлекает сведения о версии гостевой операционной системы, работающей на виртуальной машине.<br/> |
| [**Параметром PARAMETERIZATION**](ivmguestos-getparameter.md)                                 | Извлекает именованный параметр в гостевой системе.<br/>                                                |
| [**инсталлинтегратионкомпонентс**](ivmguestos-installintegrationcomponents.md) | Находит и устанавливает последние компоненты интеграции в операционную систему на виртуальной машине.<br/>      |
| [**исусерлогжедон**](ivmguestos-isuserloggedon.md)                             | Определяет, имеется ли запрошенный сеанс.<br/>                                         |
| [**Выполнен**](ivmguestos-logoff.md)                                             | Выполнит выход всех пользователей из операционной системы на виртуальной машине.<br/>                                          |
| [**Перезагрузить**](ivmguestos-restart.md)                                           | Перезапускает операционную систему на виртуальной машине.<br/>                                                         |
| [**сетпараметер**](ivmguestos-setparameter.md)                                 | Задает именованный параметр в гостевой системе.<br/>                                                     |
| [**Закрытия**](ivmguestos-shutdown.md)                                         | Завершает работу гостевой операционной системы.<br/>                                                       |
