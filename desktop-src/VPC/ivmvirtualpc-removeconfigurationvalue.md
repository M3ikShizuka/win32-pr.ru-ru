# <a name="ivmvirtualpcremoveconfigurationvalue-method"></a>Метод Ивмвиртуалпк:: Ремовеконфигуратионвалуе

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Удаляет значение указанного параметра конфигурации.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT RemoveConfigurationValue(
  [in] BSTR preferenceKey
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*преференцекэй* \[ окне\]
</dt> <dd>

Ключ, используемый для задания предпочтения, который хранится в файле конфигурации.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                                        | Описание                                                                                     |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                              | Операция выполнена успешно.<br/>                                                        |
| <dl> <dt>**Д \_**</dt> <dt>0x80004003</dt> указателя </dl>                                | Параметр *преференцекэй* имеет **значение NULL**.<br/>                                           |
| <dl> <dt>**Д \_ INVALIDARG**</dt> <dt>0x80000003</dt> </dl>                             | Параметр *преференцекэй* недопустим или является пустой строкой.<br/>                    |
| <dl> <dt>**Виртуальная машина \_ E \_ преф \_ не \_ Найдено**</dt> <dt>0xA0040300</dt> </dl>                   | Предпочтение не найдено.<br/>                                                        |
| <dl> <dt>**Виртуальная машина \_ E \_ аппаратная \_ виртуализация \_ отключена**</dt> <dt>0xA0040951</dt> </dl> | Процессор не поддерживает расширения виртуализации с аппаратным ускорением (ЗАКОНЧИТЕ).<br/> |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>                        | Произошла непредвиденная ошибка.<br/>                                                    |



 

## <a name="remarks"></a>Комментарии

Этот метод обеспечивает доступ низкого уровня к любому значению предпочтения для текущего пользователя. Его можно использовать для удаления значений предпочтений для определяемых пользователем ключей.
