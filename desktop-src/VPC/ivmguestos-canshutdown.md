# <a name="ivmguestoscanshutdown-property"></a>Свойство Ивмгуестос:: Каншутдовн

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Указывает, может ли гостевая операционная система завершать работу (требуются компоненты интеграции).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_CanShutdown(
  [out, retval] VARIANT_BOOL *canShutdown
);
```



## <a name="property-value"></a>Значение свойства

**Вариант \_ Значение TRUE** , если операционная система поддерживает операцию завершения работы и **вариант \_ false** в противном случае.
