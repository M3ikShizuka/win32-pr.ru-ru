# <a name="ivmtask-interface"></a>Интерфейс Ивмтаск

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Используйте интерфейс **ивмтаск** для мониторинга и управления асинхронными задачами для различных методов COM.

## <a name="members"></a>Элементы

Интерфейс **ивмтаск** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмтаск** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ивмтаск** содержит следующие методы.



| Метод                                                 | Описание                                                                                 |
|:-------------------------------------------------------|:--------------------------------------------------------------------------------------------|
| [**Отменить**](ivmtask-cancel.md)                       | Отменяет задачу.<br/>                                                                |
| [**ваитфоркомплетион**](ivmtask-waitforcompletion.md) | Ожидает завершения задачи или до истечения указанного интервала времени ожидания.<br/> |



 

### <a name="properties"></a>Свойства

Интерфейс **ивмтаск** имеет следующие свойства.



| Свойство                                                        | Тип доступа          | Описание                                                        |
|:----------------------------------------------------------------|:---------------------|:-------------------------------------------------------------------|
| [**Описание**](ivmtask-description.md)<br/>           | Только для чтения<br/> | Описание задачи.<br/>                              |
| [**Ошибка**](ivmtask-error.md)<br/>                       | Только для чтения<br/> | Ошибка, записанная для этой задачи.<br/>                       |
| [**ErrorDescription**](ivmtask-errordescription.md)<br/> | Только для чтения<br/> | Локализованное описание ошибки, записанное для этой задачи.<br/> |
| [**ID**](ivmtask-id.md)<br/>                             | Только для чтения<br/> | Уникальный идентификатор для этой задачи.<br/>                      |
| [**Доменять**](ivmtask-iscancelable.md)<br/>         | Только для чтения<br/> | Указывает, может ли задача быть отменена.<br/>             |
| [**IsComplete**](ivmtask-iscomplete.md)<br/>             | Только для чтения<br/> | Указывает, завершена ли задача.<br/>               |
| [**перценткомплетед**](ivmtask-percentcompleted.md)<br/> | Только для чтения<br/> | Процент завершения задачи.<br/>                  |
| [**Результат**](ivmtask-result.md)<br/>                     | Только для чтения<br/> | Результат задачи.<br/>                                 |



 

## <a name="remarks"></a>Комментарии

Объект **ивмтаск** возвращается методами, которые могут потребовать значительного времени на выполнение. Это позволяет приложению отслеживать ход выполнения требуемой операции без принудительной блокировки дальнейших операций, ожидая завершения операции.

Следующие методы возвращают объект **ивмтаск** , который можно использовать для отслеживания хода выполнения:

-   [**Ивмгуестос:: выход из системы**](ivmguestos-logoff.md)
-   [**Ивмгуестос:: Restart**](ivmguestos-restart.md)
-   [**Ивмгуестос:: Shutdown**](ivmguestos-shutdown.md)
-   [**Ивмхарддиск:: Compact**](ivmharddisk-compact.md)
-   [**Ивмхарддиск:: Convert**](ivmharddisk-convert.md)
-   [**Ивмхарддиск:: Merge**](ivmharddisk-merge.md)
-   [**Ивмхарддиск:: Мержето**](ivmharddisk-mergeto.md)
-   [**Ивмвиртуалмачине:: Мержеундодискс**](ivmvirtualmachine-mergeundodisks.md)
-   [**Ивмвиртуалмачине:: Reset**](ivmvirtualmachine-reset.md)
-   [**Ивмвиртуалмачине:: Save**](ivmvirtualmachine-save.md)
-   [**Ивмвиртуалмачине:: Startup**](ivmvirtualmachine-startup.md)
-   [**Ивмвиртуалмачине:: Startup2**](ivmvirtualmachine-startup2.md)
-   [**Ивмвиртуалмачине:: Турнофф**](ivmvirtualmachine-turnoff.md)
-   [**Ивмвиртуалпк:: КреатедифференЦингвиртуалхарддиск**](ivmvirtualpc-createdifferencingvirtualharddisk.md)
-   [**Ивмвиртуалпк:: Креатединамиквиртуалхарддиск**](ivmvirtualpc-createdynamicvirtualharddisk.md)
-   [**Ивмвиртуалпк:: Креатефикседвиртуалхарддиск**](ivmvirtualpc-createfixedvirtualharddisk.md)
