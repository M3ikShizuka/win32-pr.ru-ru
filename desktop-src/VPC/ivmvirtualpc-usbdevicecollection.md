# <a name="ivmvirtualpcusbdevicecollection-property"></a>Свойство Ивмвиртуалпк:: Усбдевицеколлектион

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Возвращает перечисляемую коллекцию всех USB-устройств, подключенных к узлу.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_USBDeviceCollection(
  [out, retval] IVMUSBDeviceCollection **usbDeviceCollection
);
```



## <a name="property-value"></a>Значение свойства

Ссылка на указатель [**ивмусбдевицеколлектион**](ivmusbdevicecollection.md) , представляющий коллекцию объектов [**ивмусбдевице**](ivmusbdevice.md) .
