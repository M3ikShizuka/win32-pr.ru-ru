# <a name="ivmdisplaycanresize-property"></a>Свойство Ивмдисплай:: Канресизе

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет, допускает ли гостевое изменение разрешение изменений.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_CanResize(
  [out, retval] VARIANT_BOOL *canResize
);
```



## <a name="property-value"></a>Значение свойства

**Вариант \_ Значение TRUE** , если гостевая ОС допускает изменения разрешения и **вариант \_ false** в противном случае.
