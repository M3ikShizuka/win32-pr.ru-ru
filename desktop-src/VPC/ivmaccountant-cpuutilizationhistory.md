# <a name="ivmaccountantcpuutilizationhistory-property"></a>Свойство Ивмаккаунтант:: Кпуутилизатионхистори

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Получает последнюю загрузку ЦП этой виртуальной машины (в виде массива процентных значений).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_CPUUtilizationHistory(
  [out, retval] VARIANT *percentageUtilization
);
```



## <a name="property-value"></a>Значение свойства

Последнее использование ЦП этой виртуальной машиной. Эти данные возвращаются в виде массива элементов 60, представляющих процент использования ЦП в секунду за последнюю минуту. Тип Variant — это VT \_ array \| VT \_ UI1.
