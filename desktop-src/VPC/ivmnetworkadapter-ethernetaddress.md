# <a name="ivmnetworkadapterethernetaddress-property"></a>Свойство Ивмнетворкадаптер:: Есернетаддресс

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Извлекает и устанавливает адрес Ethernet (MAC) сетевого интерфейса.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_EthernetAddress(
  [in]          BSTR ethernetAddress
);

HRESULT get_EthernetAddress(
  [out, retval] BSTR *ethernetAddress
);
```



## <a name="property-value"></a>Значение свойства

MAC-адрес виртуального сетевого адаптера. Он должен иметь форму "*XX* - *XX* - *XX* - *XX*, XX -  - ", где каждый *X* является шестнадцатеричной цифрой.
