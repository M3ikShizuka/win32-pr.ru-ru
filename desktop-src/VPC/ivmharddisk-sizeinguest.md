# <a name="ivmharddisksizeinguest-property"></a>Свойство Ивмхарддиск:: Сизеингуест

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Получение размера виртуального жесткого диска в гостевой операционной системе.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_SizeInGuest(
  [out, retval] VARIANT *fileSize
);
```



## <a name="property-value"></a>Значение свойства

Размер образа жесткого диска (в байтах). Это значение является **разновидностью** типа \_ Decimal.
