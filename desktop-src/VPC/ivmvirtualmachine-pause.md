# <a name="ivmvirtualmachinepause-method"></a>Ивмвиртуалмачине: метод:P Аусе

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Приостанавливает виртуальную машину.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Pause();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Возвращаемый код и значение                                                                                                                                                                          | Описание                                                                  |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0</dt> </dl>                                                | Операция выполнена успешно.<br/>                                     |
| <dl> <dt>**Д \_ Сбой**</dt> <dt>0x80004005</dt> </dl>                                     | Виртуальная машина уже приостановлена.<br/>                                         |
| <dl> <dt>**Значение HRESULT \_ ИЗ \_ Win32 (ошибка \_ \_ отказа в доступе)**</dt> <dt>0x80070005</dt> </dl> | Чтобы приостановить эту виртуальную машину, вызывающая сторона должна иметь разрешения на выполнение.<br/> |
| <dl> <dt>**Виртуальная машина \_ \_Истекло время \_ ожидания**</dt> <dt>0xA0040202</dt> </dl>                           | Операция не завершилась своевременно.<br/>                |
| <dl> <dt>**Виртуальная машина \_ \_Виртуальная машина E \_ не \_ работает**</dt> <dt>0xA0040206</dt> </dl>                     | Виртуальная машина не запущена.<br/>                               |
| <dl> <dt>**Виртуальная машина \_ \_Неизвестный \_ 0xA0040207 виртуальной машины**</dt> <dt></dt> </dl>                          | Конфигурация неизвестна.<br/>                                     |
| <dl> <dt>**\_ E \_ Exception**</dt> <dt>0x80020009</dt> </dl>                          | Произошла непредвиденная ошибка.<br/>                                 |



 

## <a name="remarks"></a>Remarks

**Приостановка** не сохраняет текущее состояние виртуальной машины.
