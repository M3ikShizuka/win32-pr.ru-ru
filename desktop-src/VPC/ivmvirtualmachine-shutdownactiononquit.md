# <a name="ivmvirtualmachineshutdownactiononquit-property"></a>Свойство Ивмвиртуалмачине:: Шутдовнактиононкуит

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

извлекает и задает действие, выполняемое на этой виртуальной машине, если она выполняется при завершении работы Windows virtual PC.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_ShutdownActionOnQuit(
  [in]          VMShutdownAction shutdownAction
);

HRESULT get_ShutdownActionOnQuit(
  [out, retval] VMShutdownAction *shutdownAction
);
```



## <a name="property-value"></a>Значение свойства

указывает, как завершить работу этой виртуальной машины, если она работает при завершении работы Windows Virtual PC. Список значений см. в разделе [**вмшутдовнактион**](vmshutdownaction.md).

## <a name="remarks"></a>Remarks

по умолчанию работающие виртуальные машины сохраняются при завершении работы Windows Virtual PC. Действие завершения работы **вмшутдовнактион \_ Save** (0) сохранит состояние виртуальной машины. Действие **вмшутдовнактион \_ турнофф** (1) отключит виртуальную машину. Действие **\_ Завершение работы вмшутдовнактион** (2) приведет к завершению работы операционной системы на виртуальной машине, если компоненты интеграции доступны и в противном случае сохранит виртуальную машину.
