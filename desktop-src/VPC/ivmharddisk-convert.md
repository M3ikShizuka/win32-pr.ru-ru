# <a name="ivmharddiskconvert-method"></a>Метод Ивмхарддиск:: Convert

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Преобразует виртуальный жесткий диск фиксированного размера в динамически расширяемый виртуальный жесткий диск или преобразует динамически расширяемый виртуальный жесткий диск в виртуальный жесткий диск фиксированного размера.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Convert(
  [in]          BSTR           convertedDiskImagePath,
  [in]          VMHardDiskType convertedDiskImageType,
  [out, retval] IVMTask        **convertTask
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*конвертеддискимажепас* \[ окне\]
</dt> <dd>

Путь к файлу образа целевого диска.

</dd> <dt>

*конвертеддискимажетипе* \[ окне\]
</dt> <dd>

Тип образа целевого диска. Список значений см. в разделе [**вмхарддисктипе**](vmharddisktype.md).

</dd> <dt>

*конверттаск* \[ out, retval\]
</dt> <dd>

Объект [**ивмтаск**](ivmtask.md) , используемый для отслеживания завершения процесса преобразования.

</dd> </dl>
