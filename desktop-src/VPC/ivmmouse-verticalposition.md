# <a name="ivmmouseverticalposition-property"></a>Свойство Ивммаусе:: Вертикалпоситион

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Получает абсолютную координату y указателя мыши.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_VerticalPosition(
  [in]          long position
);

HRESULT get_VerticalPosition(
  [out, retval] long *position
);
```



## <a name="property-value"></a>Значение свойства

Координата y, указывающая новое положение мыши. При использовании абсолютных координат это значение указывает новую координату y мыши. При использовании относительных координат это значение указывает количество пикселей, в которое указатель мыши должен перемещаться по оси y.
