# <a name="ivmharddisk-interface"></a>Интерфейс Ивмхарддиск

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Предоставляет доступ к образу жесткого диска. Доступ к нему можно получить с помощью свойства [**ивмхарддискконнектион:: жесткого**](ivmharddiskconnection-harddisk.md) диска или метода [**Ивмвиртуалпк:: жесарддиск**](ivmvirtualpc-getharddisk.md) .

## <a name="members"></a>Элементы

Интерфейс **ивмхарддиск** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Ивмхарддиск** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ивмхарддиск** содержит следующие методы.



| Метод                                 | Описание                                                                                                                                                 |
|:---------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Свернут**](ivmharddisk-compact.md) | Сжимает динамически расширяемый образ виртуального жесткого диска.<br/>                                                                                        |
| [**Convert**](ivmharddisk-convert.md) | Преобразует любой виртуальный жесткий диск в динамически расширяемый виртуальный жесткий диск или виртуальный жесткий диск фиксированного размера.<br/>                            |
| [**AutoMerge**](ivmharddisk-merge.md)     | Слияние разностного виртуального жесткого диска с образом его родительского диска.<br/>                                                                              |
| [**мержето**](ivmharddisk-mergeto.md) | Слияние разностного виртуального жесткого диска со всеми его родительскими элементами (вплоть до корневого родительского виртуального жесткого диска и включая его) в новый файл жесткого диска.<br/> |



 

### <a name="properties"></a>Свойства

Интерфейс **ивмхарддиск** имеет следующие свойства.



| Свойство                                                              | Тип доступа           | Описание                                                                                    |
|:----------------------------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------|
| [**Файл**](ivmharddisk-file.md)<br/>                           | Только для чтения<br/>  | Полный путь к файлу виртуального жесткого диска.<br/>                                   |
| [**хостфридискспаце**](ivmharddisk-hostfreediskspace.md)<br/> | Только для чтения<br/>  | Объем оставшегося места на диске, доступного на узле для виртуального жесткого диска.<br/> |
| [**Parent**](ivmharddisk-parent.md)<br/>                       | Чтение/запись<br/> | Родительский объект разностного виртуального жесткого диска.<br/>                                   |
| [**сизеингуест**](ivmharddisk-sizeinguest.md)<br/>             | Только для чтения<br/>  | Размер виртуального жесткого диска в гостевой операционной системе.<br/>                    |
| [**сизеонхост**](ivmharddisk-sizeonhost.md)<br/>               | Только для чтения<br/>  | Размер файла виртуального жесткого диска на главном компьютере.<br/>                        |
| [**Тип**](ivmharddisk-type.md)<br/>                           | Только для чтения<br/>  | Тип виртуального жесткого диска.<br/>                                                  |
