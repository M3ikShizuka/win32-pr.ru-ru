# <a name="ivmguestoscsdversion-property"></a>Свойство Ивмгуестос:: Ксдверсион

\[Windows Virtual PC больше не доступен для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Получает Ксдверсион гостевой операционной системы, работающей на виртуальной машине.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_CSDVersion(
  [out, retval] BSTR *csdVersion
);
```



## <a name="property-value"></a>Значение свойства

Строка, завершающаяся нулем, например "пакет обновления 3", которая указывает на последний установленный в системе пакет обновления. Если пакет обновления не установлен, строка пуста.
