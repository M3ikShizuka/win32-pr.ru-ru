# <a name="ivmdvddrivebusnumber-property"></a>Свойство Ивмдвддриве:: Буснумбер

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Возвращает номер шины, к которой подключено устройство.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_BusNumber(
  [out, retval] long *vmBusNumber
);
```



## <a name="property-value"></a>Значение свойства

Номер шины. Например, на шине IDE это значение будет представлять собой либо основное, либо дополнительное расположение.
