# <a name="ivmguestosisuserloggedon-method"></a>Метод Ивмгуестос:: Исусерлогжедон

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет, имеется ли запрошенный сеанс.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT IsUserLoggedOn(
  [in]          long         inRailSession,
  [out, retval] VARIANT_BOOL *outSessionPresent
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*инраилсессион* \[ окне\]
</dt> <dd>

Задайте значение 0 для сеанса шины или 1 для сеанса RDP.

</dd> <dt>

*аутсессионпресент* \[ out, retval\]
</dt> <dd>

Задайте значение **\_ true** , если сеанс имеется, и **вариант \_ false** в противном случае.

</dd> </dl>
