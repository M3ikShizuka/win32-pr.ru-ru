# <a name="ivmvirtualmachinehassse2-property"></a>Свойство Ивмвиртуалмачине:: HasSSE2

\[Windows Virtual PC больше не доступна для использования в Windows 8. Вместо этого используйте [поставщик WMI Hyper-V (v2)](/windows/desktop/HyperV_v2/windows-virtualization-portal).\]

Определяет, поддерживает ли процессор набор инструкций SSE2.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_HasSSE2(
  [out, retval] VARIANT_BOOL *sse2Enabled
);
```



## <a name="property-value"></a>Значение свойства

**Значение true** , если процессор поддерживал набор инструкций SSE2; в противном случае — **значение false** .
