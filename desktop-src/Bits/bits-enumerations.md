# <a name="bits-enumerations"></a>Перечисления BITS

[Интерфейсы фоновая интеллектуальная служба передачи (BITS)](bits-interfaces.md) используют следующие перечисления.

## <a name="in-this-section"></a>В этом разделе

| Раздел | Описание |
|-|-|
| [**BG_AUTH_SCHEME**](/windows/win32/api/bits1_5/ne-bits1_5-bg_auth_scheme) | Определяет константы, указывающие схему проверки подлинности, используемую, когда прокси или сервер запрашивает проверку подлинности пользователя. |
| [**BG_AUTH_TARGET**](/windows/win32/api/bits1_5/ne-bits1_5-bg_auth_target) | Определяет константы, указывающие, используются ли учетные данные для запросов проверки подлинности пользователя или прокси-сервера. |
| [**BG_CERT_STORE_LOCATION**](/windows/win32/api/bits2_5/ne-bits2_5-bg_cert_store_location) | Определяет константы, указывающие расположение хранилища сертификатов. |
| [**BG_ERROR_CONTEXT**](/windows/win32/api/bits/ne-bits-bg_error_context) | Определяет константы, указывающие контекст, в котором произошла ошибка. |
| [**BG_JOB_PRIORITY**](/windows/win32/api/bits/ne-bits-bg_job_priority) | Определяет константы, указывающие уровень приоритета задания.  |
| [**BG_JOB_PROXY_USAGE**](/windows/win32/api/bits/ne-bits-bg_job_proxy_usage) | Определяет константы, указывающие, какой прокси-сервер следует использовать для передачи файлов. Для каждого задания можно задать разные параметры прокси-сервера. |
| [**BG_JOB_STATE**](/windows/win32/api/bits/ne-bits-bg_job_state) | Определяет константы, указывающие различные состояния задания. |
| [**BG_JOB_TYPE**](/windows/win32/api/bits/ne-bits-bg_job_type) | Определяет константы, указывающие тип задания перемещения, например скачивание. |
| [**BITS_COST_STATE**](./bits-cost-state.md) | Определяет константы, указывающие состояние стоимости BITS. |
| [**BITS_FILE_PROPERTY_ID**](/windows/win32/api/bits5_0/ne-bits5_0-bits_file_property_id) | Определяет константы, указывающие значения ИДЕНТИФИКАТОРов, соответствующие свойствам файла фоновой копии. См. раздел [IBackgroundCopyFile5:: Property](../delivery_optimization/ibackgroundcopyfile5-getproperty.md) и [SetProperty](../delivery_optimization/ibackgroundcopyfile5-setproperty.md). |
| [**BITS_JOB_PROPERTY_ID**](/windows/win32/api/bits5_0/ne-bits5_0-bits_job_property_id) | Определяет константы, указывающие идентификатор свойства для задания BITS. |
| [**BITS_JOB_TRANSFER_POLICY**](/windows/win32/api/bits5_0/ne-bits5_0-bits_job_transfer_policy) | Определяет константы, указывающие значения ИДЕНТИФИКАТОРов, соответствующие свойствам BITS. |
