# <a name="upload-limits"></a>Upload Диапазона

Чтобы ограничить размер отправки, задайте свойство расширения IIS [**битсмаксимумуплоадсизе**](bits-iis-extension-properties.md) . В IIS 7 может также потребоваться изменить атрибут **maxAllowedContentLength** . По умолчанию предел загрузки IIS 7 составляет 30 000 000 байт. Дополнительные сведения и сведения об изменении параметров IIS по умолчанию см. в разделе [KB942074](https://support.microsoft.com//kb/942074).
