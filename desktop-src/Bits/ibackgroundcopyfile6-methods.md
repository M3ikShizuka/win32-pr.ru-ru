# <a name="ibackgroundcopyfile6-methods"></a>Методы IBackgroundCopyFile6

Интерфейс [**IBackgroundCopyFile6**](/windows/desktop/api/bits10_1/nn-bits10_1-ibackgroundcopyfile6) предоставляет следующие методы.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                           | Описание                                                                   |
|-------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**Метод Упдатедовнлоадпоситион**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-updatedownloadposition)<br/> | Указывает точку, с которой следует установить приоритет загрузки отсутствующих данных. <br/> |
| [**Метод Рекуестфилеранжес**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-requestfileranges)<br/>           | Добавляет новый набор диапазонов файлов для приоритета загрузки. <br/>     |
| [**Метод Жетфилледфилеранжес**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-getfilledfileranges)<br/>       | Возвращает набор загруженных диапазонов файлов.<br/>          |
