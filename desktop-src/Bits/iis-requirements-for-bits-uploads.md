# <a name="iis-requirements-for-bits-uploads"></a>Требования IIS для отправки BITS

Для отправки BITS требуется IIS 6,0 на Windows Server 2003 и IIS 7,0 в Windows Server 2008; Служба BITS не поддерживает IIS 5,1 в Windows XP. Виртуальный каталог IIS должен быть включен для отправки и настроен для расширений IIS BITS.

**Основные установки Windows Server:** Расширения BITS для IIS не поддерживаются.

В Windows Server 2008 используйте **Диспетчер сервера** , чтобы установить компонент серверные расширения BITS. В **Диспетчер сервера** щелкните **компоненты** в левой области. В **мастере добавления компонентов** проверьте серверные расширения BITS. Обратите внимание, что необходимо установить роли совместимости управления IIS 6.

В Windows Server 2003 используйте **Мастер компонентов Windows** для установки расширения сервера BITS. В **панели управления** выберите **Установка и удаление программ**. Затем выберите **Добавить или удалить компоненты Windows** , чтобы открыть **Мастер компонентов Windows**. Серверное расширение BITS является подкомпонентом службы IIS (IIS), который является подкомпонентом сервера веб-приложений.

> [!Note]  
> Если служба IISAdmin перезапущена, Рабочий процесс IIS необходимо перезапустить, прежде чем служба BITS сможет возобновить передачу. Рабочий процесс IIS можно вручную перезапустить с помощью служебной программы командной строки **IISReset.exe** .

 

Сведения о настройке служб IIS для отправки см. в разделе [Настройка сервера для отправки](setting-up-the-server-for-uploads.md).

Дополнительные сведения о свойствах, которые служба BITS добавляет в IIS, см. в разделе [Параметры сервера BITS для заданий отправки](bits-server-settings-for-upload-jobs.md).
