# <a name="iis-requirements-for-bits-uploads"></a>Требования IIS для отправки BITS

для отправки BITS требуется iis 6,0 на Windows Server 2003, а iis 7,0 на Windows Server 2008; служба BITS не поддерживает IIS 5,1 в Windows XP. Виртуальный каталог IIS должен быть включен для отправки и настроен для расширений IIS BITS.

**основные установки Windows Server:** Расширения BITS для IIS не поддерживаются.

на Windows Server 2008 используйте **диспетчер сервера** , чтобы установить компонент серверные расширения BITS. В **Диспетчер сервера** щелкните **компоненты** в левой области. В **мастере добавления компонентов** проверьте серверные расширения BITS. Обратите внимание, что необходимо установить роли совместимости управления IIS 6.

в Windows Server 2003 используйте **мастер Windows компонентов** для установки расширения сервера BITS. В **панели управления** выберите **Установка и удаление программ**. затем выберите **добавить или удалить Windows компоненты** , чтобы открыть **мастер Windows компонентов**. серверное расширение BITS является подкомпонентом службы IIS (IIS), который является подкомпонентом сервера веб-приложений.

> [!Note]  
> Если служба IISAdmin перезапущена, Рабочий процесс IIS необходимо перезапустить, прежде чем служба BITS сможет возобновить передачу. Рабочий процесс IIS можно вручную перезапустить с помощью служебной программы командной строки **IISReset.exe** .

 

Сведения о настройке служб IIS для отправки см. в разделе [Настройка сервера для отправки](setting-up-the-server-for-uploads.md).

дополнительные сведения о свойствах, которые служба bits добавляет в IIS, см. в разделе [Параметры сервера BITS для заданий Upload](bits-server-settings-for-upload-jobs.md).
