# <a name="bits-response-packets"></a>Пакеты ответов BITS

В следующей таблице перечислены пакеты ответа, отправляемые клиенту BITS для заданий отправки и отправки и ответа. В таблице перечислены пакеты в типичной последовательности, которые они отправляют клиенту.



| Пакет ответа                                      | Назначение                                                                                                                                                                                     |
|------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Подтверждение проверки связи](ack-for-ping.md)                     | Подтверждает запрос [проверки связи](ping.md) .                                                                                                                                                  |
| [Подтверждение для создания сеанса](ack-for-create-session.md) | Подтверждает запрос на [Создание сеанса](create-session.md) и возвращает идентификатор сеанса, используемый клиентом BITS во всех последующих запросах для идентификации этого сеанса передачи файлов. |
| [Подтверждение для фрагмента](ack-for-fragment.md)             | Подтверждает запрос [фрагмента](fragment.md) и записывает фрагмент в файл передачи на сервере.                                                                                 |
| [Подтверждение для закрытия сеанса](ack-for-close-session.md)   | Подтверждает запрос на [закрытие сеанса](close-session.md) и освобождает все ресурсы, связанные с этим сеансом.                                                                         |
| [Подтверждение для отмены сеанса](ack-for-cancel-session.md) | Подтверждает запрос на [отмену сеанса](cancel-session.md) и освобождает все ресурсы, связанные с этим сеансом.                                                                       |
