# <a name="bitsadmin-tool"></a>Средство Битсадмин

Битсадмин — это программа командной строки, которую можно использовать для создания заданий скачивания или отправки и отслеживания хода их выполнения. полную, исчерпывающую документацию по средству и его командам см. в разделе [битсадмин](/windows-server/administration/windows-commands/bitsadmin) and [битсадмин examples](/windows-server/administration/windows-commands/bitsadmin-examples) статьи Windows IT Pro Center.

Например, чтобы получить список всех заданий BITS, можно выполнить команду `bitsadmin /list /verbose` . Ниже приведен снимок экрана с примерами выходных данных для этой команды.
