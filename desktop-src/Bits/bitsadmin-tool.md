# <a name="bitsadmin-tool"></a>Средство Битсадмин

Битсадмин — это программа командной строки, которую можно использовать для создания заданий скачивания или отправки и отслеживания хода их выполнения. Полную, исчерпывающую документацию по средству и его командам см. в разделе [примеры](/windows-server/administration/windows-commands/bitsadmin-examples) [битсадмин](/windows-server/administration/windows-commands/bitsadmin) и БИТСАДМИН в центре Windows IT Pro Center.

Например, чтобы получить список всех заданий BITS, можно выполнить команду `bitsadmin /list /verbose` . Ниже приведен снимок экрана с примерами выходных данных для этой команды.
