# <a name="bits-startup-type"></a>Тип запуска BITS

**Тип запуска** BITS — это отложенный автоматический запуск (при наличии активных заданий BITS) или запуск по требованию (если активные задания отсутствуют). Версии Windows до обновления за ноябрь использовали только тип запуска отложенного автоматического запуска; в версиях, предшествовавших версии Vista, используется ручной тип запуска.

Не следует задавать для параметра **Тип запуска** значение отключено. Отключение BITS может привести к нарушению работы приложений, использующих BITS для передачи файлов.
