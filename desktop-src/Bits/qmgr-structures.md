# <a name="qmgr-structures"></a>Структуры КМГР

\[Структуры диспетчера очереди (КМГР) доступны для использования в операционных системах, указанных в разделе требования раздела. В последующих версиях они могут быть изменены или недоступны. Вместо этого используйте [структуры и объединения BITS](bits-c---structures-and-unions.md).\]

[Интерфейсы](qmgr-interfaces.md) диспетчера очереди (кмгр) используют следующие структуры.



| structure                                     | Назначение                                                                   |
|-----------------------------------------------|---------------------------------------------------------------------------|
| [**филесетинфо**](/windows/desktop/api/Qmgr/ns-qmgr-filesetinfo)<br/> | Определяет удаленные и локальные имена загружаемого файла.<br/> |
| **время ФОНОВого \_ копирования \_**<br/>        | Не поддерживается.<br/>                                                 |
