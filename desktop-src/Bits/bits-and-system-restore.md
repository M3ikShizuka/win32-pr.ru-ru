# <a name="bits-and-system-restore"></a>Службы BITS и восстановление системы

Не все версии BITS используют один и тот же формат для хранения заданий. Если вы вернете компьютер к точке восстановления до обновления BITS, старая версия BITS не сможет считать текущие сведения о задании. В этом случае служба BITS удалит список заданий и создаст новый список пустых заданий. В результате временные файлы, связанные с списком предыдущих заданий, не очищаются; чтобы освободить место на диске, необходимо удалить файлы вручную.

пользователи, знакомые с командной строкой Windows, могут избежать этой проблемы с помощью [средства битсадмин](bitsadmin-tool.md) , чтобы очистить список заданий BITS перед выполнением восстановления системы. Чтобы очистить список заданий BITS, выполните следующую команду:

**битсадмин/Reset/аллусерс**

Обратите внимание, что для выполнения этой команды необходимы права администратора.
