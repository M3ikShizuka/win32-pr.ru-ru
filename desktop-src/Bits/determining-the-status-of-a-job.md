# <a name="determining-the-status-of-a-job"></a>Определение состояния задания

Состояние задания относится к изменениям в состоянии задания (например, состояние задания изменяется с состояния \_ задания BG \_ \_ Передача в \_ состояние задания BG \_ \_ ) и ход передачи файлов. Чтобы записать изменения в состояние задания, используйте следующие параметры.

-   [Опрос состояния задания](polling-for-the-status-of-the-job.md).
-   [Зарегистрируйтесь для получения уведомления о событии](registering-a-com-callback.md).
-   [Зарегистрируйтесь, чтобы выполнить программу](registering-to-execute-a-program.md) на основе события.

Сведения о ходе выполнения задания см. в разделе [Определение хода выполнения задания](determining-the-progress-of-a-job.md).
