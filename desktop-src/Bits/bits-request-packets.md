# <a name="bits-request-packets"></a>Пакетов запросов BITS

Пакеты запросов описывают запросы клиентов. В данный момент времени может быть только один необработанный запрос. перед отправкой другого запроса необходимо получить [Подтверждение](bits-response-packets.md) для текущего запроса от сервера.

В следующей таблице перечислены пакеты запроса, отправляемые на сервер BITS для заданий отправки и отправки и ответа. В таблице перечислены пакеты в типичной последовательности, которые они отправляют на сервер.



| Пакет запроса                       | Назначение                                                                                                                                                        |
|--------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Проверок](ping.md)                     | Устанавливает соединение и согласовывает безопасность с сервером.                                                                                              |
| [Создание сеанса](create-session.md) | Запрашивает сеанс отправки с сервером BITS.                                                                                                               |
| [Fragment](fragment.md)             | Отправляет фрагмент файла на сервер BITS. Число отправленных запросов фрагментов зависит от выбранного размера фрагмента и размера файла отправки. |
| [Закрыть — сеанс](close-session.md)   | Завершает сеанс передачи файла с сервером BITS.                                                                                                             |
| [Отмена — сеанс](cancel-session.md) | Завершает сеанс передачи файла с сервером BITS. Как правило, пакет Cancel-Session отправляется, если пользователь отменил задание.                                 |



 

Пакет Ping является необязательным. Вместо отправки ping-пакета можно использовать пакет Create-Session для установления соединения и согласования безопасности. Однако для этой цели более эффективно использовать пакет Ping.
