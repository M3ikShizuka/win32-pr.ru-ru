# <a name="using-the-iis-ui-to-configure-the-virtual-directory"></a>Использование пользовательского интерфейса IIS для настройки виртуального каталога

Чтобы использовать пользовательский интерфейс IIS для настройки виртуального каталога для отправки BITS, выберите пункт **Свойства** в контекстном меню виртуального каталога. Нажмите вкладку **расширение BITS** , чтобы открыть страницу свойств BITS (необходимо установить расширение сервера BITS ISAPI). Измените значения свойств по умолчанию соответствующим образом и включите каталог для отправки. невозможно передать файлы, пока каталог не будет включен. Дополнительные сведения о задании значений свойств см. в справке в Интернете.
