# <a name="whats-new-bits"></a>Новые возможности BITS

начиная с первого выпуска в Windows XP, фоновая интеллектуальная служба передачи (BITS) постоянно улучшился, добавляя более мощные элементы управления для разработчика и администратора для контроля и управления загрузкой. Добавлен обширный набор командлетов PowerShell; Он может подключаться к нескольким типам HTTP-серверов; Это более тщательная пропускная способность и стоимость сети пользователя, чем раньше. 

В следующей таблице приведены новые возможности для каждого выпуска фоновая интеллектуальная служба передачи (BITS).
