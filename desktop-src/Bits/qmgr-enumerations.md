# <a name="qmgr-enumerations"></a>Перечисления КМГР

\[Перечисления диспетчера очереди (КМГР) доступны для использования в операционных системах, указанных в разделе требований раздела. В последующих версиях они могут быть изменены или недоступны. Вместо этого используйте [перечисления BITS](bits-enumerations.md).\]

[Интерфейсы](qmgr-interfaces.md) диспетчера очереди (кмгр) используют следующее перечисление.



| Перечисление                               | Описание                                                                              |
|-------------------------------------------|------------------------------------------------------------------------------------------|
| [**грауппроп**](/windows/desktop/api/Qmgr/ne-qmgr-groupprop)<br/> | Определяет константные значения для получения и установки значений свойств группы.<br/> |
