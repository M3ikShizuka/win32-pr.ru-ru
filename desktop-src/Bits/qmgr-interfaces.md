# <a name="qmgr-interfaces"></a>Интерфейсы КМГР

\[Интерфейсы диспетчера очереди (КМГР) доступны для использования в операционных системах, указанных в разделе требования раздела. В последующих версиях они могут быть изменены или недоступны. Вместо этого используйте [интерфейсы BITS](bits-interfaces.md).\]

Используйте следующие интерфейсы диспетчера очередей (КМГР) для загрузки файлов и отслеживания заданий в очереди загрузки.



| Интерфейс                                                                 | Описание                                                                                                                                                   |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IBackgroundCopyCallback1**](/windows/desktop/api/Qmgr/nn-qmgr-ibackgroundcopycallback1)<br/>   | Реализуйте, чтобы получить уведомление при возникновении событий.<br/>                                                                                               |
| [**ибаккграундкопиграуп**](/windows/desktop/api/Qmgr/nn-qmgr-ibackgroundcopygroup)<br/>           | Используйте для управления группой. Например, добавьте в группу задание, задайте свойства группы, а затем запустите и прервите группу в очереди загрузки.<br/> |
| [**IBackgroundCopyJob1**](/windows/desktop/api/Qmgr/nn-qmgr-ibackgroundcopyjob1)<br/>             | Используйте для добавления файлов в задание и получения состояния задания.<br/>                                                                                         |
| [**ибаккграундкопикмгр**](/windows/desktop/api/Qmgr/nn-qmgr-ibackgroundcopyqmgr)<br/>             | Используйте для создания новой группы, получения существующей группы или перечисления всех групп в очереди.<br/>                                                       |
| [**иенумбаккграундкопиграупс**](/windows/desktop/api/Qmgr/nn-qmgr-ienumbackgroundcopygroups)<br/> | Используйте для получения списка групп в очереди загрузки.<br/>                                                                                            |
| [**IEnumBackgroundCopyJobs1**](/windows/desktop/api/Qmgr/nn-qmgr-ienumbackgroundcopyjobs1)<br/>   | Используйте для получения списка заданий в группе.<br/>                                                                                                       |
