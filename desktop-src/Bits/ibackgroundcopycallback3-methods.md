# <a name="ibackgroundcopycallback3-methods"></a>Методы IBackgroundCopyCallback3

Интерфейс [**IBackgroundCopyCallback3**](/windows/desktop/api/Bits10_1/nn-bits10_1-ibackgroundcopycallback3) предоставляет следующие методы.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                             | Описание                                                                                                                                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Метод Филеранжестрансферред**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopycallback3-filerangestransferred)<br/> | Служба BITS вызывает реализацию метода [**филеранжестрансферред**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopycallback3-filerangestransferred) при загрузке одного или нескольких диапазонов файлов. Диапазоны файлов добавляются в задание с помощью метода [**IBackgroundCopyFile6:: рекуестфилеранжес**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-requestfileranges) .<br/> |
