# <a name="mailslot-functions"></a>Функции слота

С маилслотс можно использовать следующие функции.



| Функция                                   | Описание                                                                  |
|--------------------------------------------|------------------------------------------------------------------------------|
| [**креатемаилслот**](/windows/desktop/api/Winbase/nf-winbase-createmailslota)   | Создает почтовый слот с указанным именем.                                  |
| [**жетмаилслотинфо**](/windows/desktop/api/Winbase/nf-winbase-getmailslotinfo) | Извлекает сведения об указанном почтовом слоте.                          |
| [**сетмаилслотинфо**](/windows/desktop/api/Winbase/nf-winbase-setmailslotinfo) | Задает значение времени ожидания, используемое указанным слотом для операции чтения. |
