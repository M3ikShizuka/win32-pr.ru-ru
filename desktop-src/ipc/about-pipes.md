# <a name="about-pipes"></a>О каналах

Существует два типа каналов: [анонимные каналы](anonymous-pipes.md) и [именованные каналы](named-pipes.md). Анонимные каналы занимают меньше ресурсов, чем именованные каналы, но предлагают ограниченные службы.

Термин «канал», как здесь используется, подразумевает, что канал используется в качестве информационного канала. По сути, канал имеет две конечные точки. Односторонний канал позволяет процессу на одном конце выполнить запись в канал и позволяет процессу на другом конце считаться из канала. Двусторонний (или дуплексный) канал позволяет процессу выполнять чтение и запись с конца канала.

-   [Анонимные каналы](anonymous-pipes.md)
-   [Именованные каналы](named-pipes.md)
