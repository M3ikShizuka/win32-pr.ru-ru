# <a name="anonymous-pipes"></a>Анонимные каналы

*Анонимный канал* является неименованным односторонним каналом, который обычно передает данные между родительским процессом и дочерним процессом. Анонимные каналы всегда являются локальными; они не могут использоваться для связи по сети.

-   [Наследование обработчика канала](pipe-handle-inheritance.md)
-   [Безопасность анонимных каналов и права доступа](anonymous-pipe-security-and-access-rights.md)
