# <a name="network-dde-functions"></a>Сетевые функции DDE

\[Сетевой DDE больше не поддерживается. Nddeapi.dll имеется в Windows Vista, но все вызовы функций возвращают ндде \_ не \_ реализованы.\]

Для сетевого DDE используются следующие функции.



| Функция                                                   | Описание                                                                                                           |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**нддежетеррорстринг**](nddegeterrorstring.md)           | Преобразует код ошибки, возвращенный сетевой функцией DDE, в строку ошибки, объясняющую возвращенный код ошибки. |
| [**нддежетшаресекурити**](nddegetsharesecurity.md)       | Извлекает дескриптор безопасности, связанный с общим ресурсом DDE.                                                      |
| [**нддежеттрустедшаре**](nddegettrustedshare.md)         | Извлекает параметры, связанные с общим ресурсом DDE, который находится в списке доверенных общих ресурсов пользователя сервера.                |
| [**нддеисвалидапптопиклист**](nddeisvalidapptopiclist.md) | Определяет, использует ли строка приложения и раздела ("*AppName* \| *топикнаме*") правильный синтаксис.                 |
| [**нддеисвалидшаренаме**](nddeisvalidsharename.md)       | Определяет, использует ли имя общего ресурса правильный синтаксис.                                                               |
| [**нддесетшаресекурити**](nddesetsharesecurity.md)       | Задает дескриптор безопасности, связанный с общим ресурсом DDE.                                                           |
| [**нддесеттрустедшаре**](nddesettrustedshare.md)         | Предоставляет указанное состояние доверия общего ресурса DDE в контексте текущего пользователя.                                      |
| [**нддешареадд**](nddeshareadd.md)                       | Создает и добавляет новый общий ресурс DDE в диспетчер общей базы данных DDE (DSDM).                                            |
| [**нддешаредел**](nddesharedel.md)                       | Удаляет общую папку DDE из DSDM.                                                                                    |
| [**нддешаринум**](nddeshareenum.md)                     | Возвращает список доступных общих ресурсов DDE.                                                                           |
| [**нддешарежетинфо**](nddesharegetinfo.md)               | Извлекает сведения об общем ресурсе DDE.                                                                                      |
| [**нддешаресетинфо**](nddesharesetinfo.md)               | Задает общие сведения об общем ресурсе DDE.                                                                                           |
| [**нддетрустедшаринум**](nddetrustedshareenum.md)       | Возвращает имена всех общих сетевых ресурсов DDE, которые являются доверенными в контексте вызывающего процесса.                 |
