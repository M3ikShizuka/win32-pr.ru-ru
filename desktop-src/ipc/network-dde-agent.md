# <a name="network-dde-agent"></a>Агент сетевого DDE

\[Сетевой DDE больше не поддерживается. Nddeapi.dll имеется в Windows Vista, но все вызовы функций возвращают ндде \_ не \_ реализованы.\]

Агент сетевого DDE запускает сетевой DDE, если обнаруживает активность DDE по локальной сети. Он не обнаруживает удаленный клиент, пытающийся подключиться. Поэтому, прежде чем клиент сможет успешно подключиться, необходимо запустить сетевой DDE на компьютере сервера. Обратите внимание, что по умолчанию сетевой DDE не запускается. Чтобы запустить сетевой DDE, запустите NETDDE.EXE. этот файл находится в каталоге Windows.

Агент сетевого DDE также запускает приложения, необходимые для сетевого DDE. После запуска сетевого DDE сеансы DDE контролируются через окно сетевого DDE, связанное с одним из приложений сетевого DDE. Это приложение выступает в качестве прокси-сервера. Он взаимодействует со всеми локальными и удаленными приложениями DDE.
