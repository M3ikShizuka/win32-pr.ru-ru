# <a name="pipes-interprocess-communications"></a>Каналы (взаимодействие между процессами)

*Канал* — это раздел общей памяти, используемый процессами для обмена данными. Процесс, создающий канал, является *сервером канала*. Процесс, который подключается к каналу, является *клиентом канала*. Один процесс записывает сведения в канал, а другой процесс считывает информацию из канала. В этом обзоре описывается создание, управление и использование каналов.
