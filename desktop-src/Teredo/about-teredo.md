# <a name="about-teredo"></a>О Teredo

Из-за ограниченного числа доступных адресов IPv4 поставщикам услуг Интернета (ISP) потребуется начать переход на IPv6. Windows Vista обеспечит полную поддержку IPv6, позволяя разработчикам обеспечить готовность своих приложений к завтрашнему сроку, включив поддержку IPv6 уже сегодня. С помощью Teredo настольные компьютеры Windows Vista могут подключаться к IPv6, даже если сети ISP являются только IPv4.

Таким технологиям, как одноранговые, требуется сквозное подключение для полного взаимодействия с пользователем. Многие дома имеют устройства преобразования сетевых адресов (NAT), которые ограничивают эти технологии и разработчики приложений должны сосредоточиться на поиске эффективных способов решения этих проблем. Однако поддержка Teredo в Windows Vista снижает нагрузку на разработчиков приложений за счет простого обхода NAT.
