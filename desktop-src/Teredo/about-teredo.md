# <a name="about-teredo"></a>О Teredo

Из-за ограниченного числа доступных адресов IPv4 поставщикам услуг Интернета (ISP) потребуется начать переход на IPv6. Windows Vista обеспечит полную поддержку IPv6, позволяя разработчикам обеспечить готовность своих приложений к завтрашнему сроку, включив поддержку IPv6 уже сегодня. с помощью Teredo настольные компьютеры Windows Vista могут подключаться к IPv6, даже если сети ISP — только IPv4.

Таким технологиям, как одноранговые, требуется сквозное подключение для полного взаимодействия с пользователем. Многие дома имеют устройства преобразования сетевых адресов (NAT), которые ограничивают эти технологии и разработчики приложений должны сосредоточиться на поиске эффективных способов решения этих проблем. однако поддержка Teredo в Windows Vista снижает нагрузку на разработчиков приложений, предоставляя простой обход NAT.
