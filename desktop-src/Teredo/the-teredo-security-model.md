# <a name="the-teredo-security-model"></a>Модель безопасности Teredo

технология [Teredo](about-teredo.md) , связанная с Безопасность Windows Center и [Windows брандмауэром](/previous-versions/windows/desktop/ics/windows-firewall-start-page), обеспечивает повышенную безопасность для конечного пользователя. для Windows брандмауэра трафик ipv6, отправленный и полученный через интерфейс Teredo, обрабатывается так же, как и собственный трафик IPv6. Интерфейс Teredo распознается как другой мини-порт в системе.

Дополнительные сведения об использовании см. в статье [Реализация модели безопасности Teredo](implementing-the-teredo-security-model.md).
