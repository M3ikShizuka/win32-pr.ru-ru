# <a name="the-teredo-security-model"></a>Модель безопасности Teredo

Технология [Teredo](about-teredo.md) , если она связана с центром безопасности Windows и [брандмауэром Windows](/previous-versions/windows/desktop/ics/windows-firewall-start-page), обеспечивает повышенную безопасность для конечного пользователя. Для брандмауэра Windows трафик IPv6, отправленный и полученный через интерфейс Teredo, обрабатывается так же, как и исходный трафик IPv6. Интерфейс Teredo распознается как другой мини-порт в системе.

Дополнительные сведения об использовании см. в статье [Реализация модели безопасности Teredo](implementing-the-teredo-security-model.md).
