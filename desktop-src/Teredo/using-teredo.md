# <a name="using-teredo"></a>Использование Teredo

При разработке приложения, использующего [Teredo](about-teredo.md), цель состоит в том, чтобы обеспечить подключение IPv6 и не сосредоточиться на использовании Teredo. Например, Конференц-зал Windows и удаленный помощник — это два приложения, которые используют преимущества сквозного подключения, предоставляемого протоколом IPv6. Эти приложения могут получать подключения с помощью собственных адресов IPv6, 6to4 или Teredo. Для этих приложений и других компонентов, которые могут быть разработаны для IPv6, Teredo обрабатывается как любой другой протокол, обеспечивающий подключение.
