# <a name="ipv6-connectivity-and-teredo"></a>Подключение IPv6 и Teredo

Большинство сетей — только IPv4, а сквозное подключение IPv6 легко получить. Кроме того, большинство устройств NAT даже ограничивают сквозное подключение по протоколу IPv4. В Windows Vista возможны сквозные подключения IPv6 между любыми двумя узлами. Разработчикам больше не нужно беспокоиться о сквозном подключении, а создавать новые сценарии приложений, позволяющие им воспользоваться преимуществами такого подключения.

Цель Teredo, как технология туннелирования IPv6, заключается в том, чтобы сделать сквозное подключение IPv6 простым и автоматическим. Во многих случаях разработчикам даже не нужно знать, что в Windows Vista существуют такие технологии перехода, как Teredo. Windows Vista имеет возможность использовать Teredo в фоновом режиме как простой способ подключения IPv6.
