# <a name="directx-graphics-articles"></a>Статьи, посвященные графике DirectX

## <a name="purpose"></a>Назначение

в этом разделе содержатся технические статьи с описанием деформации и Windows 7, добавленной поддержкой режима перелистывания и связанной с ней статистики в Direct3D 9Ex и диспетчер окон рабочего стола. в этом разделе также содержатся технические статьи о Windows графических интерфейсах api, переносе интерфейсов api Direct3D 9 в api-интерфейсы графической инфраструктуры Microsoft DirectX (DXGI) и о том, как развернуть Direct3D 11.

Дополнительные сведения о Direct3D см. в разделе [Direct3D](/windows/desktop/direct3d).

## <a name="developer-audience"></a>Аудитория разработчиков

Эти технические статьи предназначены для разработчиков графических приложений DirectX.
