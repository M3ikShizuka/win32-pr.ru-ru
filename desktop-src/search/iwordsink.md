# <a name="iwordsink-interface"></a>Интерфейс Ивордсинк

Обрабатывает слова, определяемые пределами слов, во время индексирования и во время выполнения запроса.

## <a name="members"></a>Элементы

Интерфейс **ивордсинк** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Ивордсинк** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивордсинк** содержит следующие методы.



| Метод                                             | Описание                                                                                                                             |
|:---------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------|
| [**ендалтфрасе**](iwordsink-endaltphrase.md)     | Указывает конец конечной фразы в последовательности альтернативных фраз, формируемых средством разбиения по словам во время индексирования.<br/>  |
| [**путалтворд**](iwordsink-putaltword.md)         | Помещает в объект **ивордсинк** альтернативное слово и его расположение.<br/>                                                       |
| [**путбреак**](iwordsink-putbreak.md)             | Помещает разрыв после предыдущего слова.<br/>                                                                                       |
| [**путворд**](iwordsink-putword.md)               | Помещает слово и его расположение в объекте **ивордсинк** .<br/>                                                                    |
| [**старталтфрасе**](iwordsink-startaltphrase.md) | Указывает границу между фразами в последовательности альтернативных фраз, формируемых средством разбиения по словам во время индексирования.<br/> |



 

## <a name="remarks"></a>Remarks

Windows Поиск создает и инициализирует экземпляры объекта **ивордсинк** . Объект **ивордсинк** получает параметр *фкуери* во время инициализации и использует этот параметр для определения контекста разбиения по словам, в котором используется объект.

Реализации [**ивордбреакер**](/windows/win32/api/indexsrv/nn-indexsrv-iwordbreaker) получают указатель на объект **ивордсинк** в методе [**ивордбреакер:: бреактекст**](/windows/win32/api/indexsrv/nf-indexsrv-iwordbreaker-breaktext) .
