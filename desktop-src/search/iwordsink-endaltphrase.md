# <a name="iwordsinkendaltphrase-method"></a>Метод Ивордсинк:: Ендалтфрасе

Указывает конец конечной фразы в последовательности альтернативных фраз, формируемых средством разбиения по словам во время индексирования.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT EndAltPhrase();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Код возврата                                                                                           | Описание                                                                            |
|-------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| <dl> <dt>**ВБРЕАК \_ \_ только запрос \_ E**</dt> </dl> | [**Ендалтфрасе**](iwordsink-endaltphrase.md) вызывается во время запроса.<br/> |



 

## <a name="remarks"></a>Комментарии

Реализации [**ивордбреакер**](/windows/win32/api/indexsrv/nn-indexsrv-iwordbreaker) вызывают **Ивордсинк:: Ендалтфрасе** из метода [**ивордбреакер:: бреактекст**](/windows/win32/api/indexsrv/nf-indexsrv-iwordbreaker-breaktext) , чтобы завершить последовательность альтернативных фраз. Метод [**ивордсинк:: старталтфрасе**](iwordsink-startaltphrase.md) вызывается для обозначения конца одной фразы и начала другой в последовательности фраз. Только последняя фраза в последовательности завершается вызовом **ендалтфрасе**.
