# <a name="iwordsinkputaltword-method"></a>Ивордсинк: метод:P Уталтворд

Помещает в объект [**ивордсинк**](iwordsink.md) альтернативное слово и его расположение.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT PutAltWord(
  [in]       ULONG cwc,
  [in] const WCHAR *pwcInBuf,
  [in]       ULONG cwcSrcLen,
  [in]       ULONG cwcSrcPos
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*КВК* \[ окне\]
</dt> <dd>

Число символов в *пвЦинбуф*.

</dd> <dt>

*пвЦинбуф* \[ окне\]
</dt> <dd>

Указатель на буфер, содержащий альтернативную форму слова из исходного текста. Этот параметр не изменяется **путалтворд**. При необходимости можно передать параметр *птекстсаурце* из [**Ивордбреакер:: бреактекст**](/windows/win32/api/indexsrv/nf-indexsrv-iwordbreaker-breaktext) .

</dd> <dt>

*квксрклен* \[ окне\]
</dt> <dd>

Количество символов в исходном текстовом буфере (обозначенное параметром *птекстсаурце* в [**Ивордбреакер:: бреактекст**](/windows/win32/api/indexsrv/nf-indexsrv-iwordbreaker-breaktext)), соответствующее слову, содержащемуся в *пвЦинбуф*.

</dd> <dt>

*квксркпос* \[ окне\]
</dt> <dd>

Начальное расположение слова в *пвЦинбуф* в исходном текстовом буфере (указывается параметром *птекстсаурце* в [**ивордбреакер:: бреактекст**](/windows/win32/api/indexsrv/nf-indexsrv-iwordbreaker-breaktext)).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Код возврата                                                                                              | Описание                                                                                                                                               |
|----------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                     | Операция успешно завершена. Также указывает, что больше не осталось никакого текста для обработки.<br/>                                            |
| <dl> <dt>**Языковая \_ \_крупное \_ слово S**</dt> </dl> | Значение *КВК* больше, чем значение *улмакстокенсизе* , указанное в [**ивордбреакер:: init**](/windows/win32/api/indexsrv/nf-indexsrv-iwordbreaker-init). <br/> |



 

## <a name="remarks"></a>Комментарии

**Путалтворд** помещает в [**ивордсинк**](iwordsink.md)альтернативную форму слова. Слово помещается в ту же точку, что и исходное слово в источнике текста (*птекстсаурце* в [**Ивордбреакер:: бреактекст**](/windows/win32/api/indexsrv/nf-indexsrv-iwordbreaker-breaktext)). По умолчанию **путалтворд** завершает слова с **вордрепм Break \_ \_ Еов** , заданными в перечислимом типе [**вордреп \_ \_ типа Break**](/previous-versions/windows/desktop/legacy/ff819130(v=vs.85)) .
