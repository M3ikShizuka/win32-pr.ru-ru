# <a name="protocol-handlers-interfaces"></a>Интерфейсы обработчиков протоколов

этот Windows раздел справки по поиску состоит из интерфейсов обработчика протокола.


## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                   | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**иитемпропертибаг**](iitempropertybag.md)<br/>                                 | Определяет методы для получения сведений о свойствах элемента поиска. этот интерфейс поддерживается только в Windows XP и Windows Server 2003 и больше не должен использоваться.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**иоплоккстатус**](/windows/desktop/api/Searchapi/nn-searchapi-ioplockstatus)<br/>                               | предоставляет методы для проверки уступающей блокировки, используемой Microsoft Windows Desktop Search (WDS) для элементов во время индексирования. Если другой процесс блокирует файл с помощью несовместимого способа, WDS теряет блокировку и разрешает другому процессу использовать этот файл. Этот механизм позволяет WDS работать в фоновом режиме. Следовательно, службы WDS должны проверить свои блокировки, чтобы гарантировать, что другой процесс не будет иметь приоритет, а WDS индексирует элемент. <br/> Сторонний объект [**иурлакцессор**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) может реализовать этот интерфейс, если базовое хранилище данных предоставляет механизм для контроля параллельного доступа к элементам. Если этот интерфейс предоставляется **иурлакцессор**, службы WDS проверяют [**иоплоккстатус**](/windows/desktop/api/Searchapi/nn-searchapi-ioplockstatus) при индексировании элементов из этого хранилища. <br/> |
| [**ипротоколхандлерсервицес**](/previous-versions/windows/desktop/legacy/bb231447(v=vs.85))<br/>         |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**ипротоколхандлерсите**](/windows/desktop/api/Searchapi/nn-searchapi-iprotocolhandlersite)<br/>                 | Предоставляет методы для объекта [**иурлакцессор**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) обработчика протокола для запроса управляющей программы фильтра для соответствующего фильтра для элемента URL-адреса. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**исеарчпротокол**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocol)<br/>                           | Предоставляет методы для вызова, инициализации и управления объектами [**иурлакцессор**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) . Методы в этом интерфейсе вызываются узлом протокола при обработке URL-адресов из средства сбора данных. <br/> Обработчик протокола реализует протокол для доступа к источнику содержимого в собственном формате. Используйте этот интерфейс для реализации пользовательского обработчика протокола, чтобы расширить источники данных, которые могут быть проиндексированы. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISearchProtocol2**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocol2)<br/>                         | Предоставляет методы для вызова, инициализации и управления объектами [**иурлакцессор**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) . Методы в этом интерфейсе вызываются узлом протокола при обработке URL-адресов из средства сбора данных. <br/> Обработчик протокола реализует протокол для доступа к источнику содержимого в собственном формате. Используйте этот интерфейс для реализации пользовательского обработчика протокола, чтобы расширить источники данных, которые могут быть проиндексированы. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**исеарчпротоколсреадконтекст**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocolthreadcontext)<br/> | Этот необязательный интерфейс позволяет обработчику протокола выполнить действие в потоке, используемом для фильтрации в узле протокола. При запуске узла протокола сначала инициализируются все обработчики протокола, а затем создаются потоки фильтрации. Методы этого интерфейса позволяют обработчикам протоколов управлять ресурсами, используемыми потоком фильтрации.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**исеарчпротоколуи**](-search-isearchprotocolui.md)<br/>                       | Предоставляет метод для вызова объектов [**исеарчитем**](-search-isearchitem.md) . Методы в этом интерфейсе вызываются узлом протокола при обработке URL-адресов из средства сбора данных. Обработчик протокола реализует протокол для доступа к источнику содержимого в собственном формате, и этот интерфейс реализует пользовательский обработчик протокола для расширения источников данных, которые могут быть проиндексированы.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**иурлакцессор**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor)<br/>                                 | Предоставляет методы для обработки отдельного элемента в источнике содержимого, URL-адрес которого предоставляется средством сбора данных для узла фильтра. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IUrlAccessor2**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor2)<br/>                               | Расширяет функциональные возможности интерфейса [**иурлакцессор**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IUrlAccessor3**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor3)<br/>                               | Расширяет функциональные возможности интерфейса [**IUrlAccessor2**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor2) с помощью метода [**IUrlAccessor3:: жетимперсонатионсидблобс**](/windows/desktop/api/Searchapi/nf-searchapi-iurlaccessor3-getimpersonationsidblobs) для идентификации идентификаторов безопасности пользователя (SID) для указанного URL-адреса.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IUrlAccessor4**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor4)<br/>                               | Расширяет функциональные возможности интерфейса [**IUrlAccessor3**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor3) с помощью метода [**IUrlAccessor4:: шаулдиндекситемконтент**](/windows/desktop/api/Searchapi/nf-searchapi-iurlaccessor4-shouldindexitemcontent) , который определяет, следует ли индексировать содержимое элемента. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
