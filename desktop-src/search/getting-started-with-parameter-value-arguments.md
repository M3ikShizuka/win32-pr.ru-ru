# <a name="getting-started-with-parameter-value-arguments"></a>Начало работы с Parameter-Valueными аргументами

**Поиск-МС** ? [протокол приложения](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa767916(v=vs.85)) — это соглашение для запроса индекса поиска Windows. Протокол позволяет приложениям, например проводнику, запрашивать индекс с аргументами-значениями параметров, включая аргументы свойств, ранее сохраненные поисковые запросы, синтаксис расширенных запросов (АКС), синтаксис естественного запроса (НКС) и идентификаторы кода языка (LCID) как для индексатора, так и для самого запроса.

Этот раздел организован следующим образом:

- [Об аргументах Parameter-Value](#about-parameter-value-arguments)
- [Примеры](#examples)
- [См. также](#related-topics)

## <a name="about-parameter-value-arguments"></a>Об аргументах Parameter-Value

Протокол Search-MS использует следующий стандартный синтаксис в кодировке URL:

```
search-ms:parameter=value[&parameter=value]&
```

Синтаксис начинается с определения самого протокола (Поиск-MS:). Пары "параметр-значение" — это аргументы, передаваемые в механизм поиска, как описано в следующей таблице.

| Параметр                                                    | Значение                                                         | Описание                                                                                                                                                                                                                                                                | Версия                  |
|--------------------------------------------------------------|---------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------|
| query                                                        | Текст в кодировке URL                                              | Текст запроса, указанный пользователем.                                                                                                                                                                                                                                        | Windows XP и более поздние версии    |
| [инпутлокале](-search-3x-wds-qryidx-localeidentifiers.md)   | Любой допустимый код языка                                                | КОД языка, определяющий язык ввода для запроса.                                                                                                                                                                                                                 | Windows XP и более поздние версии    |
| [кэйвордлокале](-search-3x-wds-qryidx-localeidentifiers.md) | Любой допустимый код языка                                                | КОД языка, определяющий язык международной версии индексатора. Значение по умолчанию — 1033 (EN-US).                                                                                                                                                            | Windows XP и более поздние версии    |
| [иерархической](-search-3x-wds-qryidx-crumb.md)                     | АКС, инструкция                                                 | Этот аргумент ограничивает область поиска. В Windows Vista и более поздних версиях Поиск-MS поддерживает полную АКС, а также специальную реализацию `location` аргумента. В Windows XP Поиск-MS также поддерживает полную АКС, за исключением специальной реализации `kind` и `store` . | Windows XP и более поздние версии    |
| [Syntax](-search-3x-wds-qryidx-syntaxargument.md)           | НКС, АКС (без учета регистра)                                 | Синтаксис запроса, используемый для поиска в индексе: естественный синтаксис запроса или синтаксис расширенных запросов (АКС). АКС является значением по умолчанию и всегда считается проанализированным и поддерживаемым.                                                                                                    | Windows Vista и более поздние версии |
| [стаккедби](-search-3x-wds-qryidx-stackedby.md)             | Любое допустимое свойство из системы свойств                   | Свойство, указывающее столбец, по которому должны быть упорядочены результаты.                                                                                                                                                                                                                  | Windows Vista и более поздние версии |
| [subquery](-search-3x-wds-qryidx-subquery.md)               | Полностью указанный путь к сохраненному файлу поиска ( \* . Search-MS) | Результаты вложенного запроса используются в качестве источника запроса. То есть термины запроса ищутся по результатам вложенного запроса.                                                                                                                           | Windows Vista и более поздние версии |
| displayname                                                  | Строка в кодировке URL                                            | Имя текущего поиска.                                                                                                                                                                                                                                            | Windows Vista и более поздние версии |


Дополнительные сведения см. в разделе [Регистрация приложения по протоколу URL-адреса](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa767914(v=vs.85)).

## <a name="examples"></a>Примеры

```
search-ms:query=microsoft&
search-ms:query=vacation&subquery=mydepartment.search-ms&
search-ms:query=seattle&crumb=kind:pics&
search-ms:query=seattle&crumb=folder:C:\MyFolder&
```
