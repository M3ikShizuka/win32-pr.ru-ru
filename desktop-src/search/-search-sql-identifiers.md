# <a name="identifiers"></a>Идентификаторы

Идентификаторы указывают имена столбцов (иногда называемых свойствами), каталоги и псевдонимы. Например, System. ItemName и System. DateCreated являются идентификаторами двух определяемых системой столбцов свойств. Литералы, напротив, указывают строковые и числовые значения.


Можно создавать идентификаторы длиной до 128 символов и в одном из двух типов, в отличие от символов, используемых в имени идентификатора:

-   **Обычные идентификаторы** содержат только символы a – z, A – z, 0-9, символ подчеркивания ( \_ ) и точку (.) и начинаются с буквы. Обычные идентификаторы не нужно заключать в двойные кавычки ("").
-   **Идентификаторы с разделителями** могут содержать любой допустимый символ Юникода и должны быть заключены в двойные кавычки.

> [!Note]  
> В предложениях FREETEXT и CONTAINS можно использовать звездочку () в \* качестве специального идентификатора столбца, если необходимо указать, что в Windows Search будут включены все индексированные свойства в запросе. Хотя это не обычный идентификатор, для него не требуются двойные кавычки.
