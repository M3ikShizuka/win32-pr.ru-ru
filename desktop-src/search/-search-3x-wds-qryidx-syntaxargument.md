# <a name="syntax-argument-windows-search"></a>аргумент синтаксиса (поиск Windows)

`syntax`Аргумент позволяет задать синтаксис запроса. Допустимые значения — это либо синтаксис естественного запроса (НКС), либо расширенный синтаксис запросов (АКС). АКС является значением по умолчанию и всегда считается проанализированным и поддерживаемым.

Этот раздел организован следующим образом:

-   [Пример](#example)
-   [Связанные темы](#related-topics)

## <a name="example"></a>Пример


```
 search-ms:query=from:paolo&syntax:AQS
```
