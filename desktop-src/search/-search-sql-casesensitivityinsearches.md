# <a name="case-sensitivity-in-searches"></a>Чувствительность к регистру при поиске

В запросах Microsoft Windows Search регистр не учитывается. Сюда входят ключевые слова запроса (SELECT совпадает с выбором, выделением и выделением) и условиями поиска. Если идентификатор или литерал использует символы Юникода с семантикой сопоставления регистра, поисковая система считает все варианты эквивалентными.
