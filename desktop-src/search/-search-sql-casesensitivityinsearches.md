# <a name="case-sensitivity-in-searches"></a>Чувствительность к регистру при поиске

поисковые запросы Microsoft Windows не учитывают регистр. Сюда входят ключевые слова запроса (SELECT совпадает с выбором, выделением и выделением) и условиями поиска. Если идентификатор или литерал использует символы Юникода с семантикой сопоставления регистра, поисковая система считает все варианты эквивалентными.
