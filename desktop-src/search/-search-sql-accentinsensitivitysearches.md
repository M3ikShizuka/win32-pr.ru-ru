# <a name="diacritic-sensitivity-in-searches"></a>Чувствительность диакритических знаков в поиске

по умолчанию поисковая система Windows и каталог не зависят от диакритических знаков, которые представляют собой диакритические знаки, добавленные к буквам для изменения значения или произношения слова. однако Windows поиск позволяет изменить его для каталога, используя [диспетчер каталогов](-search-3x-wds-mngidx-catalog-manager.md) для установки чувствительности диакритических знаков. Например, если параметр чувствительность диакритических знаков имеет значение **false**, каталог будет считаться «Resume» и «ресумé», как если бы они были одним словом.

На уровне запроса термины запроса с диакритическими знаками в FREETEXT и предложениями CONTAINS передаются в подсистему и затем нормализованы (так же, как и во время индексирования) для сопоставления. Сопоставление с каталогом всегда учитывает диакритические знаки.

> [!Note]  
> Это не так для диапазонов символов 0x2e81-f8ff и 0x1100-0x11ff.
