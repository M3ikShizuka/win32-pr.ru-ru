# <a name="order-by-clause"></a>Предложение ORDER BY

Предложение ORDER BY сортирует результаты на основе значения одного или нескольких указанных столбцов. Ниже приведен синтаксис предложения ORDER BY:


```
ORDER BY <column> [<direction>] [,<column> [<direction>]]
```



Спецификатор столбца должен быть допустимым столбцом. Можно использовать описатель столбца для ссылки на столбцы по порядку их появления в запросе. Первый столбец в запросе пронумерован 1. В предложении ORDER BY можно включить несколько столбцов, разделенных запятыми.

Необязательный спецификатор направления может быть либо "ASC" для возрастания (от низкого до высокого), либо "DESC" для нисходящего (от максимального к меньшему). Если не указать описатель направления, используется значение по умолчанию (по возрастанию). Если указать более одного столбца, но не указывать все направления, то направление, заданное последним, применяется к каждому столбцу до тех пор, пока не будет явно изменено направление.

Например, в следующем предложении ORDER BY столбцы A, B, C и G сортируются в возрастающем порядке, а столбцы D, E и F сортируются в порядке убывания.


```
ORDER BY A ASC, B, C, D DESC, E, F, G ASC
```
