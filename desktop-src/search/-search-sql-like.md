# <a name="like-predicate"></a>Предикат LIKE

Предикат LIKE выполняет сравнение сопоставления с шаблоном для указанного столбца. Для этого используется следующий синтаксис:


```
...WHERE <column> LIKE '<wildcard_literal>'
```



&lt;Столбец &gt; может быть обычным или [идентификатором](-search-sql-identifiers.md)с разделителями. Столбец ограничен свойствами в хранилище свойств.

> литерал с подстановочным знаком <\_ является строковым литералом. Он заключается в кавычки и при необходимости может содержать подстановочные знаки. При необходимости строка соответствия может содержать несколько подстановочных знаков. В следующей таблице описаны подстановочные знаки, распознаваемые предикатом LIKE.



| Подстановочный знак                | Описание                                                                                                                                                                                     | Пример                                                                                                                                                                                                              |
|-------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| % (в процентах)             | Соответствует нулю или большему числу символов.                                                                                                                                                         | "comp% r" соответствует "Comp", за которой следует ноль или более символов, заканчивающихся на r.                                                                                                                                  |
| и символ подчеркивания (\_).         | Соответствует любому отдельному символу.                                                                                                                                                                   | "Comp \_ заве" соответствует "Comp", за которым следует только один символ, за которым следует "заве".                                                                                                                              |
| \[\](квадратные скобки) | Соответствует любому отдельному символу в указанном диапазоне или наборе. Например, \[ a-z \] указывает диапазон; \[ AEIOU \] задает набор гласных.                                                  | "Comp \[ a-z \] Re" соответствует "Comp", за которым следует один символ в диапазоне от a до z, за которым следует ". "Comp \[ AO \] " соответствует "Comp", за которым следует один символ, который должен быть либо a, либо o.<br/> |
| \[^ \] курсор          | Соответствует любому отдельному символу, который не входит в указанный диапазон или набор. Например, \[ ^ a-z \] указывает диапазон, исключающий от a до z; \[ ^ AEIOU \] указывает набор, исключающий гласные. | "Comp \[ ^ u \] " соответствует "Comp", за которым следует любой отдельный символ, отличный от u.                                                                                                                                        |



 

При создании предикатов с несколькими диапазонами диапазоны должны быть упорядочены по порядку.

> [!Note]  
> Чтобы сопоставить символы-шаблоны как литеральные символы для сопоставления, а не в качестве подстановочных знаков, поместите символ в квадратные скобки. Например, чтобы сопоставить знак процента, используйте "". \[ % \]

 

## <a name="examples"></a>Примеры


```
...WHERE System.ItemNameDisplay LIKE 'financ%'
```
