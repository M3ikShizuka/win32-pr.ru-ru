# <a name="windows-search"></a>Поиск Windows

Windows Search — это платформа для поиска на рабочем столе, которая обладает возможностями мгновенного поиска для большинства распространенных типов файлов и данных, таких как электронная почта, контакты, встречи в календаре, документы, фотографии, мультимедиа и другие форматы, которые могут быть расширены сторонними разработчиками. Эти возможности позволяют пользователям находить, администрировать и организовывать увеличивающийся объем данных, общих в домашних и корпоративных средах.
