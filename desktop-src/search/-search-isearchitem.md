# <a name="isearchitem-interface"></a>Интерфейс Исеарчитем

Предоставляет методы, определяющие взаимодействие между пользовательским интерфейсом и объектами пространства имен поиска, созданными индексатором.

## <a name="members"></a>Элементы

Интерфейс **исеарчитем** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Исеарчитем** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **исеарчитем** содержит следующие методы.



| Метод                                                         | Описание                                                                                                                               |
|:---------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------|
| [**жетпарентфолдер**](-search-isearchitem-getparentfolder.md) | Возвращает объект **исеарчитем** , если URL-адрес представляет фактический источник данных оболочки (также известный как расширение пространства имен оболочки).<br/> |
| [**жетуиобжектоф**](/previous-versions/windows/desktop/legacy/dd756721(v=vs.85))     | Возвращает объект пользовательского интерфейса **исеарчитем**.<br/>                                                                        |



 

## <a name="remarks"></a>Remarks

[**исеарчитем:: жетпарентфолдер**](-search-isearchitem-getparentfolder.md) поддерживается только в Windows XP и Windows Server 2003 и больше не следует использовать.

для предварительного просмотра вложений с помощью обработчика протоколов стороннего производителя на компьютерах под управлением Windows XP или Windows Server 2003 может потребоваться использовать интерфейс **исеарчитем** и следующие api: интерфейсы [**иитемпревиеверекст**](-search-iitempreviewerext.md), [**иитемпропертибаг**](iitempropertybag.md)и [**исеарчпротоколуи**](-search-isearchprotocolui.md) , структура [**линкинфо**](-search-linkinfo.md) и [**перечисление**](-search-linktype.md) типов ссылок.
