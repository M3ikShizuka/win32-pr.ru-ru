# <a name="document-and-system-locale-settings"></a>языковой стандарт документа и системы Параметры

Если операционная система или даже приложение настроены на использование определенного языка и языкового стандарта, затрагиваются многие параметры. Эти параметры включают числовой формат, формат даты, формат валюты, сопоставление верхнего и нижнего регистра, порядок сортировки словаря, разметку и другие. несмотря на то что эти параметры помогают Windows поиска обеспечить отличную локализованную поддержку, непредвиденные результаты могут возникнуть при поиске документов из одного языкового стандарта с помощью системы, для которой настроен другой языковой стандарт.

Когда объект [**IFilter**](/windows/win32/api/filter/nn-filter-ifilter) обрабатывает свойства текста и содержимое документа, он сообщает языку этого документа с индексатором содержимого. используя эти сведения, Windows поиска может применить соответствующее средство разбиения по словам.
