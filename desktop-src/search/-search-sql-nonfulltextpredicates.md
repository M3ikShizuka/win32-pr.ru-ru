# <a name="non-full-text-predicates"></a>Неполнотекстовые предикаты

Язык запросов Microsoft Windows Search поддерживает шесть предикатов, не являющихся полнотекстовым поиском. Предикаты, описанные в этом разделе, перечислены в следующей таблице.



| Неполнотекстовый предикат                                                    | Описание                                                                                                                                                                             |
|----------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [ВСЕ побитовое и побитовое](all-bitwise.md)                            | — Это набор ключевых слов, посвященных проверке битов в целочисленном типе.                                                                                                               |
| [DATEADD, функция](-search-sql-dateadd.md)                                | Выполняет вычисления времени и даты для соответствующих свойств с типами дат. Функция DATEADD используется для получения даты и времени в течение указанного промежутка времени до текущего.     |
| [Предикат LIKE](-search-sql-like.md)                                     | Сравнивает значения столбцов, используя простое сопоставление шаблонов с подстановочными знаками.                                                                                                          |
| [Сравнение литеральных значений](-search-sql-literalvaluecomparison.md)         | Сравнивает значения столбцов со строками, датами, метками времени, числовыми значениями и другими литеральными значениями. Этот предикат поддерживает неравенство, например, больше (' > ') и меньше (' < '). |
| [Сравнения с несколькими значениями (МАССИВы)](-search-sql-multivaluedcomparisons.md) | Сравнивает Многозначные столбцы с многозначным массивом литералов.                                                                                                                   |
| [Предикат NULL](-search-sql-null.md)                                     | Обнаруживает значения столбцов, которые не определены для документа.                                                                                                                              |



 

> [!IMPORTANT]
> Поисковые запросы с помощью предиката **null** могут потребовать, чтобы поиск Windows проверял весь каталог, что может замедлить производительность запроса.
