# <a name="using-localized-searches"></a>Использование локализованных поисков

Microsoft Windows Search поддерживает полный поиск в Юникоде. Можно индексировать документы, использующие Юникод, и создавать поисковые запросы, использующие идентификаторы Юникода и литералы. Однако параметр языка приложения или операционной системы может иметь более тесное воздействие на работу службы поиска Windows.

Этот раздел содержит следующие подразделы:

-   [Параметры языка документа и системы](-search-sql-docandlocalesystemsettings.md)
-   [Указание языков](-search-sql-specifyinglanguages.md)
