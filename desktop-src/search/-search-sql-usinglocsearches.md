# <a name="using-localized-searches"></a>Использование локализованных поисков

Microsoft Windows Search поддерживает полный поиск в юникоде. Можно индексировать документы, использующие Юникод, и создавать поисковые запросы, использующие идентификаторы Юникода и литералы. однако языковой параметр приложения или операционной системы может иметь более тесное влияние на работу Windows поиска.

Этот раздел содержит следующие подразделы:

-   [языковой стандарт документа и системы Параметры](-search-sql-docandlocalesystemsettings.md)
-   [Указание языков](-search-sql-specifyinglanguages.md)
