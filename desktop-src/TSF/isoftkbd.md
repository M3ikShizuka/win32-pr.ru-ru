# <a name="isoftkbd-interface"></a>Интерфейс Исофткбд

Интерфейс **исофткбд** используется приложениями и текстовыми службами для реализации экранной клавиатуры.

## <a name="members"></a>Элементы

Интерфейс **исофткбд** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Исофткбд** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **исофткбд** содержит следующие методы.



| Метод                                                                                        | Описание                                                                                                   |
|:----------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------|
| [**адвисесофткэйбоардевентсинк**](isoftkbd-advisesoftkeyboardeventsink.md)                   | Устанавливает приемник событий программируемой клавиатуры для работы с Онкэйселектион уведомлениями с помощью экранной клавиатуры.<br/> |
| [**креатесофткэйбоардлайаутфромресаурце**](isoftkbd-createsoftkeyboardlayoutfromresource.md) | Создает раскладку экранной клавиатуры из указанного ресурса.<br/>                                        |
| [**креатесофткэйбоардлайаутфромксмлфиле**](isoftkbd-createsoftkeyboardlayoutfromxmlfile.md)   | Создает раскладку экранной клавиатуры из указанного XML-файла.<br/>                                        |
| [**креатесофткэйбоардвиндов**](isoftkbd-createsoftkeyboardwindow.md)                         | Создает окно с экранной клавиатурой.<br/>                                                                    |
| [**дестройсофткэйбоардвиндов**](isoftkbd-destroysoftkeyboardwindow.md)                       | Уничтожает окно с экранной клавиатурой.<br/>                                                                   |
| [**енумсофткэйбоард**](isoftkbd-enumsoftkeyboard.md)                                         | Перечисляет возможные мягкие клавиатуры, поддерживающие указанный язык.<br/>                        |
| [**жетсофткэйбоардколорс**](isoftkbd-getsoftkeyboardcolors.md)                               | Получает цвет экранной клавиатуры, соответствующий заданному типу цвета.<br/>                        |
| [**жетсофткэйбоардпоссизе**](isoftkbd-getsoftkeyboardpossize.md)                             | Извлекает начальную точку и размер экранной клавиатуры.<br/>                                       |
| [**жетсофткэйбоардтекстфонт**](isoftkbd-getsoftkeyboardtextfont.md)                           | Извлекает шрифт текста, используемый мягкой клавиатурой.<br/>                                                   |
| [**жетсофткэйбоардтипемоде**](isoftkbd-getsoftkeyboardtypemode.md)                           | Получает режим типа для экранной клавиатуры.<br/>                                                       |
| [**Инициализация**](isoftkbd-initialize.md)                                                     | Инициализирует все необходимые поля для мягкой клавиатуры и создает стандартные раскладки клавиатуры.<br/> |
| [**селектсофткэйбоард**](isoftkbd-selectsoftkeyboard.md)                                     | Выбирает указанную мягкую клавиатуру.<br/>                                                               |
| [**сеткэйбоардлабелтекст**](isoftkbd-setkeyboardlabeltext.md)                                 | Задает текст метки для экранной клавиатуры.<br/>                                           |
| [**сеткэйбоардлабелтексткомбинатион**](isoftkbd-setkeyboardlabeltextcombination.md)           | Задает сочетание метки и текста, используемого для описания экранной клавиатуры.<br/>                             |
| [**сетсофткэйбоардколорс**](isoftkbd-setsoftkeyboardcolors.md)                               | Задает цвет экранной клавиатуры, соответствующий заданному типу цвета.<br/>                             |
| [**сетсофткэйбоардпоссизе**](isoftkbd-setsoftkeyboardpossize.md)                             | Задает начальное расположение и размер экранной клавиатуры.<br/>                                            |
| [**сетсофткэйбоардтекстфонт**](isoftkbd-setsoftkeyboardtextfont.md)                           | Задает шрифт текста, используемый мягкой клавиатурой.<br/>                                                        |
| [**сетсофткэйбоардтипемоде**](isoftkbd-setsoftkeyboardtypemode.md)                           | Задает режим типа для экранной клавиатуры.<br/>                                                            |
| [**шовкэйсфоркэйсканмоде**](isoftkbd-showkeysforkeyscanmode.md)                             | Отображает ключи, используемые для режима сканирования ключа для экранной клавиатуры.<br/>                                  |
| [**шовсофткэйбоард**](isoftkbd-showsoftkeyboard.md)                                         | Отображает экранную клавиатуру.<br/>                                                                          |
| [**унадвисесофткэйбоардевентсинк**](isoftkbd-unadvisesoftkeyboardeventsink.md)               | Удаляет многофункциональный приемник событий клавиатуры.<br/>                                                                |
