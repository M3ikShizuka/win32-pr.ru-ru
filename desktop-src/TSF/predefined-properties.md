# <a name="predefined-properties"></a>Предопределенные свойства

Следующие значения определяют свойства, определяемые TSF. В него включаются формат данных и содержимое каждого типа свойства.

## <a name="properties"></a>Свойства



| Свойство                        | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|---------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_атрибут prop для GUID \_**       | Содержит значение [**тфгуидатом**](tfguidatom.md) , представляющее **идентификатор GUID** атрибута дисплея. [**Итфкатегоримгр::-GUID**](/windows/desktop/api/Msctf/nf-msctf-itfcategorymgr-getguid) используется для преобразования этого значения в **GUID**. Дополнительные сведения см. [в разделе Использование атрибутов вывода](using-display-attributes.md).                                                                                                                                                                                                                                                                                                                                                                             |
| **GUID \_ prop \_ текстовнер**       | Содержит значение [**тфгуидатом**](tfguidatom.md) , представляющее идентификатор класса ( **CLSID** ) текстовой службы, которой принадлежит текст. [**Итфкатегоримгр::-GUID**](/windows/desktop/api/Msctf/nf-msctf-itfcategorymgr-getguid) используется для преобразования этого значения в **идентификатор CLSID**.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **GUID \_ prop \_ LangID**          | Содержит значение **типа DWORD** , содержащее идентификатор языка ( **LangID** ) текста в нижнем слове.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| **\_чтение свойства Prop \_**         | Содержит фонетический текст для текста, покрываемого свойством. Это значение может отличаться от фактического текста. Приложения Магазина Windows не поддерживают это свойство.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **\_ \_ составление свойств GUID**       | Содержит логическое значение, отличное от нуля, если текст является частью композиции, или ноль в противном случае. Если это свойство имеет значение VT \_ , то можно предположить, что текст не является частью композиции.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **GUID \_ prop \_ модебиас**        | Содержит значение [**тфгуидатом**](tfguidatom.md) , представляющее тип поддерживаемой защиты режима. [**Итфкатегоримгр::-GUID**](/windows/desktop/api/Msctf/nf-msctf-itfcategorymgr-getguid) используется для преобразования этого значения в **GUID**. Это может быть одно из [**значений смещения режима**](mode-bias-values.md).                                                                                                                                                                                                                                                                                                                                                                                                  |
| **Для свойства "Prop" (GUID) \_ \_ нравится мой лифеаттице**       | Содержит указатель на объект [**итфлмлаттице**](/windows/desktop/api/Ctffunc/nn-ctffunc-itflmlattice) .                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **\_ \_ альтернативные ТКБ для GUID Prop \_** | **Начиная с Windows 8:** Содержит значение **DWORD** , заданное сенсорной клавиатурой. Это свойство может использоваться элементами управления и приложениями редактирования, поддерживающими TSF, для указания природы текста в диапазоне текста, который охватывает свойство, например, если текст в диапазоне является результатом вставки текстового предложения или автозамены. <br/> Характер текста в диапазоне текста, который охватывает свойство, также расширяется до типа альтернатив, возвращаемого интерфейсом [**итффнреконверсион**](/windows/desktop/api/Ctffunc/nn-ctffunc-itffnreconversion) для этого текстового диапазона в документе.<br/> Возможные значения этого свойства см. в следующих примечаниях.<br/> |



 

## <a name="remarks"></a>Комментарии

Свойство **\_ ТКБ свойства \_ \_ альтернативы GUID** может принимать одно из следующих значений.



| Имя                                     | Значение      | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------|------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_Стандартные варианты ТКБ \_                | 0x00000001 | Указывает, что сенсорная клавиатура создала список возможных альтернативных слов для текста в диапазоне, охватываемом свойством, и что ни текстовый диапазон, ни альтернативы не являются автозаменой или текстовым предложением.                                                                                                                                                                                                              |
| ТКБ \_ альтернативы \_ для \_ автозамены     | 0x00000002 | Указывает, что сенсорная клавиатура создала альтернативное слово, которое должно автоматически заменять текст в диапазоне текста, охватываемом свойством.<br/> Сенсорная клавиатура не будет применять автозамену без указания этого элемента управления или приложения редактирования. Для применения исправления к тексту в документе следует использовать интерфейс повторного преобразования ([**итффнреконверсион**](/windows/desktop/api/Ctffunc/nn-ctffunc-itffnreconversion)).<br/> |
| \_альтернативные варианты ТКБ \_ для \_ прогнозирования         | 0x00000003 | Указывает, что текстовый диапазон, охваченный свойством, является текстовым предложением, созданным сенсорной клавиатурой, и вставляется в документ пользователем.<br/> Дополнительные альтернативные прогнозы также можно сохранить как свойство в документе.<br/>                                                                                                                                                                     |
| \_ \_ применение автокоррекции к АЛЬТЕРНАТИВам ТКБ \_ | 0x00000004 | Указывает, что текстовый диапазон, охваченный свойством, является автозаменой, обеспечиваемой сенсорной клавиатурой, и применяется через интерфейс [**итффнреконверсион**](/windows/desktop/api/Ctffunc/nn-ctffunc-itffnreconversion) .<br/> Это значение может использоваться для элементов управления или приложений, в которых ТКБ \_ альтернативы \_ для \_ автозамены, чтобы предотвратить повторное применение автозамены.<br/>                                                                               |
