# <a name="ts_ias_-constants"></a>\_Константы IAS служб терминалов \_ \*

\_Константы IAS служб терминалов \_ \* используются как флаги битов для управления вставкой текста или внедренных объектов в выделенном фрагменте или точке вставки.



| Константа/значение                                                                                                                                                                                                                       | Описание                                                                                                                                   |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TS_IAS_NOQUERY"></span><span id="ts_ias_noquery"></span><dl> <dt>**TS \_ \_НЕЗАПРОС IAS**</dt> <dt>(0x1)</dt> </dl>       | Текст вставляется, а при выходе в качестве указателя на диапазон устанавливается **значение NULL** . Не может сочетаться с \_ \_ флагом TS IAS куерйонли.<br/>          |
| <span id="TS_IAS_QUERYONLY"></span><span id="ts_ias_queryonly"></span><dl> <dt>**TS \_ IAS \_ куерйонли**</dt> <dt>(0x2)</dt> </dl> | Не выполняйте вставку. Вызывающий объект требует, чтобы был установлен только указатель на диапазон. Не может использоваться вместе с \_ \_ флагом незапроса IAS служб терминалов.<br/> |
