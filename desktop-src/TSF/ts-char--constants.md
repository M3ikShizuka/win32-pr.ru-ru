# <a name="ts_char_-constants"></a>\_Константы char служб терминалов \_ \*

\_Константы char служб терминалов \_ \* описывают тип символа.



| Константа/значение                                                                                                                                                                                                                                   | Описание                                                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| <span id="TS_CHAR_EMBEDDED"></span><span id="ts_char_embedded"></span><dl> <dt>**TS \_ CHAR \_ Embedded**</dt> <dt>(0xfffc)</dt> </dl>          | Символ представляет собой символ замены объекта в Юникоде 2,1.<br/>                             |
| <span id="TS_CHAR_REGION"></span><span id="ts_char_region"></span><dl> <dt>**TS \_ \_Область char**</dt> <dt>(0)</dt> </dl>                     | Символ является границей области.<br/>                                                      |
| <span id="TS_CHAR_REPLACEMENT"></span><span id="ts_char_replacement"></span><dl> <dt>**TS \_ \_Замена символов**</dt> <dt>(0xfffd)</dt> </dl> | Символ является символом-заполнителем скрытого текста или символом замены в Юникоде.<br/> |
