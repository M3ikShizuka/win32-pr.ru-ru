# <a name="tf_lbmenuf_-constants"></a>TF \_ лбменуф \_ \* константы

Константы ** \_ tf \_ \* лбменуф* _ используются в методе [итфмену:: аддменуитем](/windows/desktop/api/Ctfutb/nf-ctfutb-itfmenu-addmenuitem) для указания характеристик пункта меню на языковой панели.



| Константа/значение                                                                                                                                                                                                                                         | Описание                                     |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------|
| <span id="TF_LBMENUF_CHECKED"></span><span id="tf_lbmenuf_checked"></span><dl> <dt>_ * TF \_ ЛБМЕНУФ \_ Проверено * *</dt> <dt>0x01</dt> </dl>                | Пункт меню установлен.<br/>            |
| <span id="TF_LBMENUF_SUBMENU"></span><span id="tf_lbmenuf_submenu"></span><dl> <dt>**Tf \_ \_Подменю лбменуф**</dt> <dt>0x02</dt> </dl>                | Пункт меню является подменю.<br/>          |
| <span id="TF_LBMENUF_SEPARATOR"></span><span id="tf_lbmenuf_separator"></span><dl> <dt>**Tf \_ 0x04 \_ Разделитель лбменуф**</dt> <dt></dt> </dl>          | Пункт меню представляет собой разделитель.<br/>        |
| <span id="TF_LBMENUF_RADIOCHECKED"></span><span id="tf_lbmenuf_radiochecked"></span><dl> <dt>**Tf \_ ЛБМЕНУФ \_ радиочеккед**</dt> <dt>0x08</dt> </dl> | Пункт меню является переключателем.<br/> |
| <span id="TF_LBMENUF_GRAYED"></span><span id="tf_lbmenuf_grayed"></span><dl> <dt>**Tf \_ ЛБМЕНУФ \_ серый цвет**</dt> <dt>0x10</dt> </dl>                   | Пункт меню отключен.<br/>           |
