# <a name="flags-for-conversion-mode"></a>Флаги для режима преобразования

Следующие флаги используются в качестве значения [ \_ \_ \_ Инпутмоде \_ преобразования секций GUID](predefined-compartments.md). Это эквивалентно значениям [IME \_ КМОДЕ](../intl/ime-conversion-mode-values.md) для IMM32.



| Флаг                             | Значение  | Описание                                                     |
|----------------------------------|--------|-----------------------------------------------------------------|
| TF \_ конверсионмоде, \_ буквенно-цифровой | 0x0000 | Значение 1, если БУКВЕНно-цифровой режим.                                  |
| TF \_ конверсионмоде \_ native       | 0x0001 | Задайте значение 1, если собственный режим —. 0, если БУКВЕНно-цифровой режим.                |
| TF \_ конверсионмоде \_ катакана     | 0x0002 | Задайте значение 1 в режиме катакана. 0, если режим хирагана.                  |
| TF \_ конверсионмоде \_ фуллшапе    | 0x0008 | Задайте значение 1, если режим полной формы; 0, если режим половинной фигуры.              |
| TF \_ конверсионмоде \_ Roman        | 0x0010 | Задайте значение 1, чтобы предотвратить обработку преобразований IME. 0, если нет. |
| TF \_ конверсионмоде \_ CHARCODE     | 0x0020 | Значение 1, если режим ввода кода символа; 0, если нет.                |
| TF \_ конверсионмоде \_ софткэйбоард | 0x0080 | Задайте значение 1 в режиме экранной клавиатуры. 0, если нет.                       |
| TF \_ конверсионмоде, \_ Преобразование | 0x0100 | Задайте значение 1, чтобы предотвратить обработку преобразований IME. 0, если нет. |
| TF \_ конверсионмоде \_ symbol       | 0x0400 | В режиме преобразования символов задайте значение 1. 0, если нет.                   |
| TF \_ конверсионмоде \_ fixed        | 0x0800 | При фиксированном режиме преобразования задайте значение 1. 0, если нет.                    |



 

В качестве значения параметра [ \_ \_ \_ инпутмоде \_ Секции GUID](predefined-compartments.md)используются следующие значения. Это эквивалентно значениям [IME \_ СМОДЕ](../intl/ime-composition-string-values.md) для IMM32.



| Флаг                            | Значение  | Описание                                                                |
|---------------------------------|--------|----------------------------------------------------------------------------|
| TF \_ сентенцемоде \_ None          | 0x0000 | Нет сведений о предложении.                                               |
| TF \_ сентенцемоде \_ плауралклаусе | 0x0001 | Редактор IME использует сведения о предложении plural для выполнения обработки преобразования. |
| TF \_ сентенцемоде \_ синглеконверт | 0x0002 | Редактор IME выполняет обработку преобразования в односимвольном режиме.        |
| TF \_ сентенцемоде \_ Automatic     | 0x0004 | Редактор IME выполняет обработку преобразования в автоматическом режиме.               |
| TF \_ сентенцемоде \_ фрасепредикт | 0x0008 | Редактор IME использует сведения о фразах для прогнозирования следующего символа.             |
| TF \_ сентенцемоде \_ CONVERSATION  | 0x0010 | Редактор IME использует режим диалога. Это полезно для приложений разговора.      |
