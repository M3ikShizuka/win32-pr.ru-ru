# <a name="tf_lbi_style_-constants"></a>\_ \_ Константы в стиле TF ЛБИ \_ \*

**\_ \_ Константы TF ЛБИ Style \_ \* *_ можно использовать в* члене _ двстиле** структуры [tf \_ лангбаритеминфо](/windows/desktop/api/Ctfutb/ns-ctfutb-tf_langbariteminfo) для указания стиля элемента языковой панели.

Если этот стиль сочетается с \_ кнопкой TF ЛБИ \_ Style \_ БТН \_ , в дополнение к тексту будет отображаться стрелка раскрывающегося списка для элемента. Стрелка раскрывающегося списка функционирует как кнопка меню, и при нажатии она вызывает вызов **итфлангбаритембуттон:: инитмену** . Щелчок по текстовой части кнопки приведет к вызову **итфлангбаритембуттон:: OnClick** .
