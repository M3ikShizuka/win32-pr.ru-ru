# <a name="about-text-services-framework"></a>Сведения о платформе текстовых служб

Microsoft Windows Text Services Framework (TSF) — это прикладной программный интерфейс (API), обеспечивающий расширенный, независимый от источника текстовый ввод. TSF предоставляет свою инфраструктуру приложениям и текстовым службам через набор интерфейсов COM.

В следующих разделах приводятся сводные сведения о функциях TSF.

-   [Зачем использовать платформу текстовых служб?](why-use-text-services-framework.md)
-   [Архитектура](architecture.md)
-   [Как настроить платформу текстовых служб](how-to-set-up-tsf.md)
