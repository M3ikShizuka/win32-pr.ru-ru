# <a name="itfcontextrenderingmarkupgetrenderingmarkup-method"></a>Метод Итфконтекстрендерингмаркуп:: Жетрендерингмаркуп

Метод **итфконтекстрендерингмаркуп:: жетрендерингмаркуп** извлекает перечислитель для разметки отрисовки для заданного диапазона.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetRenderingMarkup(
  [in]  TfEditCookie           ec,
  [in]  DWORD                  dwFlags,
  [in]  ITfRange               *pRangeCover,
  [out] IEnumTfRenderingMarkup **ppEnum
);
```

| Значение                                                                                                                                                                                         | Значение                                                                                      |
