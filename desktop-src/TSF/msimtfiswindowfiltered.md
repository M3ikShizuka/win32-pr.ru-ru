# <a name="msimtfiswindowfiltered-function"></a>Функция Мсимтфисвиндовфилтеред

Функция **мсимтфисвиндовфилтеред** проверяет, фильтруется ли заданное окно по АИММ (активным диспетчером метода ввода).

## <a name="syntax"></a>Синтаксис


```C++
BOOL CALLBACK MsimtfIsWindowFiltered(
  _In_ HWND hwnd
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*HWND* \[ окне\]
</dt> <dd>

Проверяемый описатель окна.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение



| Код возврата                                                                          | Описание                                                               |
|--------------------------------------------------------------------------------------|---------------------------------------------------------------------------|
| <dl> <dt>**УСЛОВИЯ**</dt> </dl>  | Если это окно фильтруется по активному диспетчеру методов ввода.<br/>     |
| <dl> <dt>**ISFALSE**</dt> </dl> | Если это окно не фильтруется с помощью активного диспетчера методов ввода.<br/> |



 

## <a name="remarks"></a>Комментарии

Окно может быть отфильтровано по Иактивеиммапп:: Филтерклиентвиндовс.
