# <a name="isoftkbdsetkeyboardlabeltext-method"></a>Метод Исофткбд:: Сеткэйбоардлабелтекст

Метод **исофткбд:: сеткэйбоардлабелтекст** задает текст метки из макета для экранной клавиатуры.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetKeyboardLabelText(
  [in] HKL hKl
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*hKl* \[ окне\]
</dt> <dd>

Маркер, используемый для получения установленного макета для экранной клавиатуры.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                |
|----------------------------------------------------------------------------------------------|--------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>      |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Недопустимый параметр *hKl* .<br/> |
