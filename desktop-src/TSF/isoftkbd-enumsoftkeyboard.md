# <a name="isoftkbdenumsoftkeyboard-method"></a>Метод Исофткбд:: Енумсофткэйбоард

Метод **исофткбд:: енумсофткэйбоард** перечисляет возможные мягкие клавиатуры, поддерживающие указанный язык.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT EnumSoftKeyboard(
  [in]  LANGID langid,
  [out] DWORD  *lpdwKeyboard
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*LangID* \[ окне\]
</dt> <dd>

Идентификатор языка.

</dd> <dt>

*лпдвкэйбоард* \[ заполняет\]
</dt> <dd>

Указатель на буфер, в котором этот метод получает идентификаторы доступных мягких клавиатур.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                   |
|----------------------------------------------------------------------------------------------|-----------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>         |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Недопустимый параметр *LangID* .<br/> |
