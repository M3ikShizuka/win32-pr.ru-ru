# <a name="predefined-list-attributes"></a>Стандартные атрибуты списка

Следующие значения обозначают атрибуты списка, полученные с помощью метода [ITfContext:: жетапппроперти](/windows/desktop/api/msctf/nf-msctf-itfcontext-getappproperty) . В него включаются формат данных и содержимое каждого типа свойства.

## <a name="properties"></a>Свойства



| Свойство                          | Описание                                                                                     |
|-----------------------------------|-------------------------------------------------------------------------------------------------|
| \_Список тсаттрид                    | Не используется.                                                                                       |
| \_Левелиндел списка \_ тсаттрид        | Содержит уровень индекса списка. 1 — это внешний уровень, 2 — следующий уровень и т. д. |
| \_Тип списка \_ тсаттрид              | Не используется.                                                                                       |
| \_Тип списка \_ тсаттрид \_ Арабский      | Содержит ненулевое значение, если список является арабским числом или ноль в противном случае.               |
| \_ \_ Маркер типа списка \_ тсаттрид      | Содержит ненулевое значение, если список является маркированным списком, или ноль в противном случае.                      |
| ТСАТТРИД \_ \_ тип списка \_ ловерлеттер | Содержит ненулевое значение, если список является списком с прописными буквами или ноль в противном случае.            |
| ТСАТТРИД \_ \_ тип списка \_ ловерроман  | Содержит ненулевое значение, если список имеет римские цифры в нижнем регистре или в противном случае ноль.       |
| ТСАТТРИД \_ \_ тип списка \_ упперлеттер | Содержит ненулевое значение, если список является списком с прописными буквами или ноль в противном случае.          |
| ТСАТТРИД \_ \_ тип списка \_ упперроман  | Содержит ненулевое значение, если список является списком цифр в верхнем регистре или ноль в противном случае.      |
