# <a name="predefined-language-bar-items"></a>Стандартные элементы языковой панели

Следующие значения [**GUID**](/windows/desktop/TSF/predefined-lang-bar-items) обозначают предопределенные элементы языковой панели. Эти значения используются с [**элементом**](/windows/desktop/api/Ctfutb/nf-ctfutb-itflangbaritemmgr-getitem) Item для получения определенного элемента языковой панели.

<dl> <dt>

<span id="_______________GUID_LBI_SAPILAYR_CFGMENUBUTTON_____________"></span><span id="_______________guid_lbi_sapilayr_cfgmenubutton_____________"></span> GUID \_ ЛБИ \_ сапилайр \_ кфгменубуттон 
</dt> <dd>

Пункт меню языковой панели инструментов для речи.

</dd> <dt>

<span id="_______________GUID_TFCAT_TIP_HANDWRITING_____________"></span><span id="_______________guid_tfcat_tip_handwriting_____________"></span>\_Ввод GUID тфкат \_ Tip \_ 
</dt> <dd>

Элемент типа рукописного ввода.

</dd> <dt>

<span id="_______________GUID_TFCAT_TIP_KEYBOARD_____________"></span><span id="_______________guid_tfcat_tip_keyboard_____________"></span> GUID \_ тфкат \_ Tip, \_ клавиатура 
</dt> <dd>

Элемент типа клавиатуры.

</dd> </dl>
