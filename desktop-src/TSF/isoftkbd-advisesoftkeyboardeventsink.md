# <a name="isoftkbdadvisesoftkeyboardeventsink-method"></a>Метод Исофткбд:: Адвисесофткэйбоардевентсинк

Метод **исофткбд:: адвисесофткэйбоардевентсинк** устанавливает приемник событий экранной клавиатуры для управления уведомлениями онкэйселектион от экранной клавиатуры.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT AdviseSoftKeyboardEventSink(
  [in]  DWORD    dwKeyboardId,
  [in]  REFIID   riid,
  [in]  IUnknown *punk,
  [out] DWORD    *pdwCookie
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*двкэйбоардид* \[ окне\]
</dt> <dd>

Идентификатор программируемой клавиатуры.

</dd> <dt>

*riid* \[ окне\]
</dt> <dd>

Идентификатор интерфейса для интерфейса приемника.

</dd> <dt>

*Punk* \[ окне\]
</dt> <dd>

Указатель на [IUnknown](/windows/win32/api/unknwn/nn-unknwn-iunknown) для интерфейса приемника, заданного параметром *riid*. Этот параметр не может иметь значение **null**.

</dd> <dt>

*пдвкукие* \[ заполняет\]
</dt> <dd>

Указатель на буфер, в котором этот метод получает программную клавиатуру "cookie", используемую для соединения с клиентом. Файл cookie должен быть уникальным для каждого подключения.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>          |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Один или несколько параметров недопустимы.<br/> |
