# <a name="lattice-types"></a>Типы Lattice

Следующие константы указывают определенные типы свойств Lattice. Эти значения используются с методами [итфлмлаттице:: QueryType](/windows/desktop/api/Ctffunc/nf-ctffunc-itflmlattice-querytype) и [Итфлмлаттице:: енумлаттицеелементс](/windows/desktop/api/Ctffunc/nf-ctffunc-itflmlattice-enumlatticeelements) .

<dl> <dt>

<span id="GUID_LMLATTICE_VER1_0"></span><span id="guid_lmlattice_ver1_0"></span>**GUID, \_ любовь мой лифеаттице \_ VER1 \_ 0**
</dt> <dd>

Указывает, что свойство Lattice имеет версию 1,0.

</dd> </dl>
