# <a name="d-text-services-framework"></a>D (платформа текстовых служб)

<dl> <dt>

<span id="_tsf_document_lock_gly"></span><span id="_TSF_DOCUMENT_LOCK_GLY"></span>**блокировка документа**
</dt> <dd>

Состояние документа, в котором диспетчер TSF может считывать или изменять текст без изменения текста другими средствами. Блокировка документа может быть доступна только для чтения или для чтения и записи.

</dd> </dl>
