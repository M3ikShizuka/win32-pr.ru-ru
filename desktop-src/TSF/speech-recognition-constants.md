# <a name="speech-recognition-constants"></a>Константы распознавания речи

При работе со службой текста распознавания речи используются следующие значения.



| Константа/значение                                                                                                                                                                                                                                         | Описание                                                                                                                                                                                                                                                 |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TF_DICTATION_ON"></span><span id="tf_dictation_on"></span><dl> <dt>**Tf \_ ДИКТОВКа \_ в**</dt> <dt>0x00000001</dt> </dl>                   | Если этот бит равен 1, то Диктовка речи активна. В противном случае Диктовка речи неактивна. Это значение не может быть объединено с помощью \_ команды tf \_ в. Используется с Секцией диктатионстат "речь" о [ \_ секции \_ \_ GUID](predefined-compartments.md) .<br/> |
| <span id="TF_DICTATION_ENABLED"></span><span id="tf_dictation_enabled"></span><dl> <dt>**Tf \_ ДИКТОФОН \_ включил поддержку**</dt> <dt>0x00000002</dt> </dl>    | Если этот бит равен 1, распознавание речи включено. В противном случае Диктовка речи отключена. Используется с Секцией диктатионстат "речь" о [ \_ секции \_ \_ GUID](predefined-compartments.md) .<br/>                                                       |
| <span id="TF_COMMANDING_ENABLED"></span><span id="tf_commanding_enabled"></span><dl> <dt>**Tf \_ КОМАНДная \_ Поддержка**</dt> <dt>0x00000004</dt> </dl> | Если этот бит равен 1, то Голосовая команда включена. В противном случае голосовая команда отключена. Используется с Секцией диктатионстат "речь" о [ \_ секции \_ \_ GUID](predefined-compartments.md) .<br/>                                                           |
| <span id="TF_COMMANDING_ON"></span><span id="tf_commanding_on"></span><dl> <dt>**Tf \_ КОМАНДная команда \_ на**</dt> <dt>0x00000008</dt> </dl>                | Если этот бит равен 1, то Голосовая команда активна. В противном случае голосовая команда неактивна. Это значение нельзя сочетать с TF \_ Dictation \_ On. Используется с Секцией диктатионстат "речь" о [ \_ секции \_ \_ GUID](predefined-compartments.md) .<br/>      |
| <span id="TF_SPEECHUI_SHOWN"></span><span id="tf_speechui_shown"></span><dl> <dt>**Tf \_ СПИЧУИ \_ показана**</dt> <dt>0x00000010</dt> </dl>             | В настоящий момент не используется.<br/>                                                                                                                                                                                                                              |
| <span id="TF_SHOW_BALLOON"></span><span id="tf_show_balloon"></span><dl> <dt>**Tf \_ ПОКАЗЫВАТЬ \_ всплывающее сообщение**</dt> <dt>0x00000001</dt> </dl>                   | В настоящий момент не используется. Используется с Секцией [ \_ \_ \_ \_ состояния пользовательского интерфейса "речевой интерфейс" для секции GUID](predefined-compartments.md) .<br/>                                                                                                                              |
| <span id="TF_DISABLE_BALLOON"></span><span id="tf_disable_balloon"></span><dl> <dt>**Tf \_ ОТКЛЮЧИТЬ \_ всплывающий**</dt> <dt>0x00000002</dt> </dl>          | Если этот бит равен 1, отображение всплывающего окна для текущего речевого режима отключено. В противном случае будет включен режим отображения всплывающего окна для текущего речевого режима. Используется с Секцией [ \_ \_ \_ \_ состояния пользовательского интерфейса "речевой интерфейс" для секции GUID](predefined-compartments.md) .<br/>    |
| <span id="TF_DISABLE_SPEECH"></span><span id="tf_disable_speech"></span><dl> <dt>**Tf \_ ОТКЛЮЧИТЬ \_ речь**</dt> <dt>0x00000001</dt> </dl>             | Если этот бит равен 1, речевой ввод отключен. В противном случае речевой ввод включается. Используется с Секцией [GUID, \_ \_ \_ отключенной от речи](predefined-compartments.md) .<br/>                                                                    |
| <span id="TF_DISABLE_DICTATION"></span><span id="tf_disable_dictation"></span><dl> <dt>**Tf \_ ОТКЛЮЧЕНИЕ \_ диктофона**</dt> <dt>0x00000002</dt> </dl>    | Если этот бит равен 1, Диктовка речи отключена. В противном случае режим диктовки речи включен. Используется с Секцией [GUID, \_ \_ \_ отключенной от речи](predefined-compartments.md) .<br/>                                                            |
| <span id="TF_DISABLE_COMMANDING"></span><span id="tf_disable_commanding"></span><dl> <dt>**Tf \_ ОТКЛЮЧИТЬ \_ командную команду**</dt> <dt>0x00000004</dt> </dl> | Если этот бит равен 1, команда распознавания речи отключена. В противном случае голосовая команда включена. Используется с Секцией [GUID, \_ \_ \_ отключенной от речи](predefined-compartments.md) .<br/>                                                                |
