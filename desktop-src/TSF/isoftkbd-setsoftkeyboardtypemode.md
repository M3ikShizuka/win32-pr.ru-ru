# <a name="isoftkbdsetsoftkeyboardtypemode-method"></a>Метод Исофткбд:: Сетсофткэйбоардтипемоде

Метод **исофткбд:: сетсофткэйбоардтипемоде** задает режим типа для экранной клавиатуры.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetSoftKeyboardTypeMode(
  [in] TYPEMODE TypeMode
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Типемоде* \[ окне\]
</dt> <dd>

Режим типа. Для перечисления [**типемоде**](typemode.md) определены возможные значения.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                     |
|----------------------------------------------------------------------------------------------|-------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>           |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Недопустимый параметр *типемоде* .<br/> |
