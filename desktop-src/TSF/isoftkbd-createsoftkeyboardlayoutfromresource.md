# <a name="isoftkbdcreatesoftkeyboardlayoutfromresource-method"></a>Метод Исофткбд:: Креатесофткэйбоардлайаутфромресаурце

Метод **исофткбд:: креатесофткэйббоардлайаутфромресаурце** создает раскладку с экранной клавиатурой из указанного ресурса.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT CreateSoftKeyboardLayoutFromResource(
  [in]  WCHAR *lpszResFile,
  [in]  WCHAR *lpszResType,
  [in]  WCHAR *lpszXMLResString,
  [out] DWORD *lpdwLayoutCookie
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*лпсзресфиле* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, указывающую имя файла ресурсов.

</dd> <dt>

*лпсзрестипе* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, указывающую тип ресурса.

</dd> <dt>

*лпсзксмлресстринг* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, обозначающую ресурс.

</dd> <dt>

*лпдвлайауткукие* \[ заполняет\]
</dt> <dd>

Указатель на буфер, в котором этот метод получает куки-файл раскладки клавиатуры.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>          |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Один или несколько параметров недопустимы.<br/> |
