# <a name="isoftkbdgetsoftkeyboardtextfont-method"></a>Метод Исофткбд:: Жетсофткэйбоардтекстфонт

Метод **исофткбд:: жетсофткэйбоардтекстфонт** извлекает шрифт текста, используемый программируемой клавиатурой.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetSoftKeyboardTextFont(
  [out] LOGFONTW *pLogFont
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*плогфонт* \[ заполняет\]
</dt> <dd>

Указатель на буфер, в котором этот метод получает структуру [**логфонтв**](/windows/win32/api/wingdi/ns-wingdi-logfonta) , определяющую шрифт текста для экранной клавиатуры.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                     |
|----------------------------------------------------------------------------------------------|-------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>           |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Недопустимый параметр *плогфонт* .<br/> |
