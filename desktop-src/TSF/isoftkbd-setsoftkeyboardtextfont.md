# <a name="isoftkbdsetsoftkeyboardtextfont-method"></a>Метод Исофткбд:: Сетсофткэйбоардтекстфонт

Метод **исофткбд:: сетсофткэйбоардтекстфонт** задает шрифт текста, используемый программируемой клавиатурой.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetSoftKeyboardTextFont(
  [in] LOGFONTW *pLogFont
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*плогфонт* \[ окне\]
</dt> <dd>

Указатель на структуру [**логфонтв**](/windows/win32/api/wingdi/ns-wingdi-logfonta) , определяющую шрифт текста для экранной клавиатуры.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                     |
|----------------------------------------------------------------------------------------------|-------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>           |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Недопустимый параметр *плогфонт* .<br/> |
