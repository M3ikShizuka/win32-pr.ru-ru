# <a name="itfcontextrenderingmarkup-interface"></a>Интерфейс Итфконтекстрендерингмаркуп

Интерфейс **итфконтекстрендерингмаркуп** реализуется диспетчером TSF и используется приложениями. Этот интерфейс можно получить с помощью Икуеринтерфаце из объекта [ITfContext](/windows/desktop/api/Msctf/nn-msctf-itfcontext) . Этот интерфейс помогает приложению перечислять данные отрисовки.



| Методы Итфконтекстрендерингмаркуп                                                | Описание                                                           |
|----------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| [жетрендерингмаркуп](itfcontextrenderingmarkup-getrenderingmarkup.md)           | Извлекает перечислитель разметки отрисовки для заданного диапазона. |
| [финднекстрендерингмаркуп](itfcontextrenderingmarkup-findnextrenderingmarkup.md) | Эта функция не реализована. Он всегда возвращает E \_ нотимпл.       |



 

## <a name="members"></a>Элементы

Интерфейс **итфконтекстрендерингмаркуп** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) , но не содержит дополнительных элементов.

## <a name="remarks"></a>Комментарии

> [!Note]  
> Этот интерфейс в настоящее время не находится в файлах общедоступного заголовка. Чтобы использовать этот API, необходимо скомпилировать [прототип](prototypes.md)с помощью MIDL.
