# <a name="isoftkbdsetsoftkeyboardpossize-method"></a>Метод Исофткбд:: Сетсофткэйбоардпоссизе

Метод **исофткбд:: сетсофткэйбоардпоссизе** задает начальную точку и размер экранной клавиатуры.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetSoftKeyboardPosSize(
  [in] POINT StartPoint,
  [in] WORD  width,
  [in] WORD  height
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*StartPoint* \[ окне\]
</dt> <dd>

Структура [точек](/previous-versions//dd162805(v=vs.85)) , указывающая координаты левого верхнего положения экранной клавиатуры.

</dd> <dt>

*Ширина* \[ окне\]
</dt> <dd>

Ширина экранной клавиатуры.

</dd> <dt>

*Высота* \[ окне\]
</dt> <dd>

Высота экранной клавиатуры.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                  |
|----------------------------------------------------------------------------------------------|----------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>        |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Один из параметров недопустим.<br/> |
