# <a name="isoftkbdgetsoftkeyboardtypemode-method"></a>Метод Исофткбд:: Жетсофткэйбоардтипемоде

Метод **исофткбд:: жетсофткэйбоардтипемоде** получает режим типа для экранной клавиатуры.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetSoftKeyboardTypeMode(
  [out] TYPEMODE *lpTypeMode
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*лптипемоде* \[ заполняет\]
</dt> <dd>

Указатель на буфер, в котором этот метод извлекает режим типа. Для перечисления [**типемоде**](typemode.md) определены возможные значения.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                       |
|----------------------------------------------------------------------------------------------|---------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>             |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Недопустимый параметр *лптипемоде* .<br/> |
