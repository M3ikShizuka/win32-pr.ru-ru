# <a name="ts_ch_-constants"></a>\_ \_ \* Константы TS CH

\_ \_ \* Константы TS CH используются внутренне TSF для указания на то, что произошло удаление текста.



| Константа/значение                                                                                                                                                                                                                               | Описание                                                        |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------|
| <span id="TS_CH_PRECEDING_DEL"></span><span id="ts_ch_preceding_del"></span><dl> <dt>**TS \_ CH \_ перед \_ Del**</dt> <dt>(1)</dt> </dl> | Текст, предшествующий позиции символа, был удален.<br/> |
| <span id="TS_CH_FOLLOWING_DEL"></span><span id="ts_ch_following_del"></span><dl> <dt>**TS \_ CH \_ после \_ Del**</dt> <dt>(2)</dt> </dl> | Текст после позиции символа был удален.<br/> |
