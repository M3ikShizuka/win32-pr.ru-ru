# <a name="other-predefined-attributes"></a>Другие предопределенные атрибуты

Следующие значения указывают различные атрибуты, полученные с помощью метода [ITfContext:: жетапппроперти](/windows/desktop/api/msctf/nf-msctf-itfcontext-getappproperty) . В него включаются формат данных и содержимое каждого типа свойства.

## <a name="properties"></a>Свойства



| Свойство                             | Описание                                                                       |
|--------------------------------------|-----------------------------------------------------------------------------------|
| **\_Приложение тсаттрид**                    | Не используется.                                                                         |
| **\_Инкорректграммар приложения \_ тсаттрид**  | Содержит ненулевое значение, если текст содержит грамматическую ошибку или ноль в противном случае.  |
| **\_Инкорректспеллинг приложения \_ тсаттрид** | Содержит ненулевое значение, если текст содержит орфографические ошибки или ноль в противном случае. |
| **ТСАТТРИД \_ другие**                 | Не используется.                                                                         |
