# <a name="predefined-text-attributes"></a>Стандартные атрибуты текста

Следующие значения обозначают текстовые атрибуты, полученные с помощью метода [**ITfContext:: жетапппроперти**](/windows/desktop/api/msctf/nf-msctf-itfcontext-getappproperty) . В него включаются формат данных и содержимое каждого типа свойства.

## <a name="properties"></a>Свойства



| Свойство                                     | Описание                                                                                                                                              |
|----------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| ТСАТТРИД \_ текст                               | Не используется.                                                                                                                                                |
| \_Выравнивание текста \_ тсаттрид                    | Не используется.                                                                                                                                                |
| \_ \_ Центр выравнивания текста тсаттрид \_            | Содержит ненулевое значение, если текст выравнивается по центру или 0 в противном случае.                                                                                      |
| \_Выравнивание текста тсаттрид по \_ \_ ширине           | Содержит ненулевое значение, если текст выровнен или равен нулю в противном случае.                                                                                     |
| \_Выравнивание текста \_ тсаттрид \_ слева              | Содержит ненулевое значение, если текст по левому краю или ноль в противном случае.                                                                                  |
| \_Выравнивание текста \_ тсаттрид \_ вправо             | Содержит ненулевое значение, если текст по правому краю или ноль в противном случае.                                                                                 |
| ТСАТТРИД \_ Text \_ ембеддедобжект               | Содержит ненулевое значение, если текст является внедренным объектом, или ноль в противном случае.                                                                            |
| \_Перенос текста \_ тсаттрид                  | Содержит ненулевое значение, если текст является дефисом или ноль в противном случае.                                                                                    |
| \_Язык текста \_ тсаттрид                     | Содержит идентификатор языка **LangID** текста.                                                                                                 |
| \_Текстовая \_ ссылка на тсаттрид                         | Содержит указатель на объект ссылки. Вызывающий объект должен использовать метод QueryInterface для получения требуемого интерфейса, например **иуниформресаурцелокатор**. |
| \_Ориентация текста \_ тсаттрид                  | Задает угол в десятых долях градуса между базовой строкой текста и осью x устройства.                                                          |
| ТСАТТРИД \_ текстовый \_ абзац                         | Не используется.                                                                                                                                                |
| ТСАТТРИД \_ текстовый \_ абзац \_ фирстлинеиндент        | Содержит число точек, на которые добавляется отступ первой строки абзаца.                                                                            |
| ТСАТТРИД \_ текстовый \_ абзац \_ лефтиндент             | Содержит число точек, для которых отступ абзаца находится слева.                                                                              |
| ТСАТТРИД \_ текстовый \_ абзац \_ линеспаЦинг            | Не используется.                                                                                                                                                |
| ТСАТТРИД \_ текстовый \_ абзац \_ линеспаЦинг по \_ крайней мере   | Содержит минимальное число строк для интервала между строками абзаца.                                                                              |
| ТСАТТРИД \_ текстовый \_ абзац \_ линеспаЦинг \_ Double    | Содержит ненулевое значение, если абзац является двойным или нулем в противном случае.                                                                            |
| ТСАТТРИД \_ текстовый \_ абзац \_ линеспаЦинг в \_ точности   | Содержит точное число линий для интервала между строками абзаца.                                                                                |
| ТСАТТРИД \_ текст \_ абзаца \_ линеспаЦинг \_ несколько  | Содержит число строк для многострочного интервала абзаца.                                                                             |
| ТСАТТРИД \_ текстовый \_ абзац \_ линеспаЦинг \_ онептфиве | Содержит ненулевое значение, если абзац является одним и одной половинной линией или нулем в противном случае.                                                             |
| ТСАТТРИД \_ текстовый \_ \_ линеспаЦинг \_ один абзац    | Содержит ненулевое значение, если абзац является одиночным или нулем в противном случае.                                                                            |
| ТСАТТРИД \_ текстовый \_ абзац \_ ригхтиндент            | Содержит число точек, для которых отступ абзаца находится справа.                                                                             |
| ТСАТТРИД \_ текстовый \_ абзац \_ спацеафтер             | Содержит количество точек после абзаца.                                                                                            |
| ТСАТТРИД \_ текстовый \_ абзац \_ спацебефоре            | Содержит число точек после интервала перед абзацем.                                                                                           |
| ТСАТТРИД \_ Text \_ ReadOnly                     | Содержит ноль, если текст доступен только для чтения или не равен нулю в противном случае.                                                                                             |
| ТСАТТРИД \_ Text \_ RightToLeft                  | Содержит ноль, если текст читается справа налево или равен нулю в противном случае.                                                                                 |
| ТСАТТРИД \_ Text \_ вертикалвритинг              | Указывает, является ли текст вертикальным или горизонтальным. Содержит ноль, если текст является горизонтальным или ненулевым, если текст является вертикальным.                             |
