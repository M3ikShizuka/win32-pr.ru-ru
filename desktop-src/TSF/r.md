# <a name="r-text-services-framework"></a>R (платформа текстовых служб)

<dl> <dt>

<span id="_tsf_range_gly"></span><span id="_TSF_RANGE_GLY"></span>**разнообраз**
</dt> <dd>

Объект, описывающий диапазон текста, который существует в текстовом потоке. Диапазон используется для чтения, записи и обработки текста. Также известен как *диапазон текста*.

См. также *Region*.

</dd> <dt>

<span id="_tsf_reconversion_gly"></span><span id="_TSF_RECONVERSION_GLY"></span>**Повторное преобразование влечет**
</dt> <dd>

Процесс, позволяющий пользователю выбрать альтернативный текст для вставленного текста.

</dd> <dt>

<span id="_tsf_region_gly"></span><span id="_TSF_REGION_GLY"></span>**регионе**
</dt> <dd>

Диапазон текста, обработанный как уникальный диапазон редактирования на основе форматирования.

</dd> </dl>
