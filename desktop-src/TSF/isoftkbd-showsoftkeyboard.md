# <a name="isoftkbdshowsoftkeyboard-method"></a>Метод Исофткбд:: Шовсофткэйбоард

Метод **исофткбд:: шовсофткэйбоард** отображает экранную клавиатуру.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT ShowSoftKeyboard(
  [in] INT iShow
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ишов* \[ окне\]
</dt> <dd>

Целое число, указывающее тип отображаемой экранной клавиатуры.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                | Описание                           |
|--------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод успешно выполнен.<br/> |
