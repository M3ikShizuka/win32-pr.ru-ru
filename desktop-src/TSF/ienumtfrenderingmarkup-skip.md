# <a name="ienumtfrenderingmarkupskip-method"></a>Метод Иенумтфрендерингмаркуп:: Skip

Метод **иенумтфрендерингмаркуп:: Skip** получает от текущей позицией указанное число элементов в последовательности перечисления.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Skip(
  [in] ULONG ulCount
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*улкаунт* \[ окне\]
</dt> <dd>

\[в параметре \] указывает количество элементов для пропуска.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                   | Описание                                                                                                        |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>    | Метод успешно выполнен.<br/>                                                                              |
| <dl> <dt>**S \_ false**</dt> </dl> | Метод достиг конца перечисления, прежде чем заданное число элементов может быть пропущено.<br/> |
