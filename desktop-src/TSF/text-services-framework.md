# <a name="text-services-framework-text-services-framework"></a>Платформа текстовых служб (платформа текстовых служб)

## <a name="purpose"></a>Назначение

Microsoft Windows Text Services Framework (TSF) — это системная служба, доступная в качестве распространяемого пакета для Windows. TSF предоставляет простую и масштабируемую платформу для доставки расширенного ввода текста и технологий естественного языка. TSF можно включить в приложениях или как текстовую службу TSF. Служба текстового ввода TSF обеспечивает многоязычную поддержку и предоставляет текстовые службы, такие как процессоры клавиатуры, распознавание рукописного текста и распознавание речи.

## <a name="where-applicable"></a>Где применимо

платформа текстовых служб применима для компьютеров на основе Windows, использующих текстовые службы и Windows XP или более поздних версий операционной системы.

## <a name="developer-audience"></a>Аудитория разработчиков

Платформа текстовых служб разработана для использования с языками программирования C/C++ программистами в модели COM. программисты должны быть знакомы с текстовыми службами для компьютеров на базе Windows. Рекомендуется знание распознавания рукописного текста, распознавания речи и программирования для многоязычной поддержки.

Что необходимо узнать перед чтением этого руководством

В целях работы с платформой текстовых служб термин "приложение" относится к приложению с поддержкой TSF, термин "текстовая служба" относится к текстовой службе TSF, а термин "менеджер" относится к диспетчеру TSF. Каждый термин применяется, как указано здесь, если не указано иное. Поставщики текстовых служб должны предоставлять цифровые подписи для двоичных исполняемых файлов.
