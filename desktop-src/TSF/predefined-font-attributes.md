# <a name="predefined-font-attributes"></a>Стандартные атрибуты шрифтов

Следующие значения обозначают атрибуты шрифта, полученные с помощью метода [ITfContext:: жетапппроперти](/windows/desktop/api/msctf/nf-msctf-itfcontext-getappproperty) . В него включаются формат данных и содержимое каждого типа свойства.

## <a name="properties"></a>Свойства



| Свойство                                                 | Описание                                                                                                                 |
|----------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| **\_Шрифт тсаттрид**                                       | Не используется.                                                                                                                   |
| **ТСАТТРИД \_ Font \_ преимущственно**                             | Содержит название шрифта текста.                                                                                    |
| **ТСАТТРИД \_ Font \_ сизептс**                              | Содержит кегль шрифта текста.                                                                                   |
| **\_Стиль ШРИФТА \_ тсаттрид**                                | Не используется.                                                                                                                   |
| **\_ \_ Анимация стиля шрифта \_ тсаттрид**                     | Не используется.                                                                                                                   |
| **ТСАТТРИД \_ \_ анимация стиля \_ шрифта \_ блинкингбаккграунд** | Содержит ненулевое значение, если текст имеет указанную анимацию или ноль в противном случае.                                         |
| **ТСАТТРИД \_ \_ анимация стиля \_ шрифта \_ ласвегаслигхтс**     | Содержит ненулевое значение, если текст имеет указанную анимацию или ноль в противном случае.                                         |
| **ТСАТТРИД \_ \_ анимация стиля \_ шрифта \_ марчингблаккантс**  | Содержит ненулевое значение, если текст имеет указанную анимацию или ноль в противном случае.                                         |
| **ТСАТТРИД \_ \_ анимация стиля \_ шрифта \_ марчингредантс**    | Содержит ненулевое значение, если текст имеет указанную анимацию или ноль в противном случае.                                         |
| **ТСАТТРИД \_ \_ анимация стиля \_ шрифта \_ Шиммер**            | Содержит ненулевое значение, если текст имеет указанную анимацию или ноль в противном случае.                                         |
| **ТСАТТРИД \_ \_ анимация стиля \_ шрифта \_ спарклетекст**        | Содержит ненулевое значение, если текст имеет указанную анимацию или ноль в противном случае.                                         |
| **ТСАТТРИД \_ \_ анимация стиля \_ шрифта \_ випедовн**           | Содержит ненулевое значение, если текст имеет указанную анимацию или ноль в противном случае.                                         |
| **ТСАТТРИД \_ \_ анимация стиля \_ шрифта \_ виперигхт**          | Содержит ненулевое значение, если текст имеет указанную анимацию или ноль в противном случае.                                         |
| **\_Стиль ШРИФТА \_ тсаттрид \_ BackgroundColor**               | Содержит значение RGB фона текста. Содержит 0xFF000000, если цвет текста является автоматическим.                          |
| **\_ \_ Мигающий стиль шрифта тсаттрид \_**                         | Содержит ненулевое значение, если текст мигает или ноль в противном случае.                                                         |
| **ТСАТТРИД начертание \_ шрифта \_ \_**                          | Содержит ненулевое значение, если текст имеет полужирное начертание или ноль в противном случае.                                                             |
| **\_Стиль ШРИФТА тсаттрид с \_ \_ прописной буквы**                    | Содержит ненулевое значение, если текст является прописным или нулем в противном случае.                                                      |
| **\_ \_ Цвет стиля шрифта \_ тсаттрид**                         | Содержит значение RGB цвета текста. Содержит 0xFF000000, если цвет текста является автоматическим.                               |
| **\_ \_ Приподнятый стиль шрифта тсаттрид \_**                        | Содержит ненулевое значение, если текст приподнят, или ноль в противном случае.                                                         |
| **\_Стиль ШРИФТА \_ тсаттрид \_ утоплен**                       | Содержит ненулевое значение, если текст утоплен или ноль в противном случае.                                                         |
| **\_ \_ Высота стиля шрифта \_ тсаттрид**                        | Содержит высоту шрифта. Дополнительные сведения см. в описании элемента **лфхеигхт** структуры LOGFONT.                       |
| **\_Стиль ШРИФТА \_ тсаттрид \_ скрыт**                        | Содержит ненулевое значение, если текст скрыт или ноль в противном случае.                                                           |
| **ТСАТТРИД начертание \_ шрифта \_ \_**                        | Содержит ненулевое значение, если текст является курсивом или нулем в противном случае.                                                           |
| **\_Кернинг для \_ стиля \_ шрифта тсаттрид**                       | Содержит минимальный размер кернинга. Дополнительные сведения см. в разделе Итекстфонт:: onкернинг.                                         |
| **ТСАТТРИД начертание \_ шрифта в \_ \_ нижнем регистре**                     | Содержит ненулевое значение, если текст является строчным или нулем в противном случае.                                                        |
| **\_Стиль ШРИФТА тсаттрид, \_ \_ выставляемый**                      | Содержит ненулевое значение, если текст является контурным или нулем в противном случае.                                                         |
| **ТСАТТРИД \_ \_ стиля шрифта \_**                      | Содержит ненулевое значение, если текст является перечеркиванием или нулем в противном случае.                                                        |
| **ТСАТТРИД \_ \_ начертание шрифта с \_ двойной линией \_**              | Содержит ненулевое значение, если текст имеет двойное перечеркивание или ноль в противном случае.                                                 |
| **ТСАТТРИД \_ \_ начертание шрифта " \_ \_ простая линия"**              | Содержит ненулевое значение, если текст является одиночным перечеркиванием или нулем в противном случае.                                                 |
| **\_ \_ Расположение стиля шрифта \_ тсаттрид**                      | Содержит расположение шрифта.                                                                                                 |
| **ТСАТТРИД \_ стиль шрифта с \_ \_ защитой**                     | Содержит ненулевое значение, если текст защищен или в противном случае — ноль.                                                        |
| **\_ \_ Тень стиля шрифта \_ тсаттрид**                        | Содержит ненулевое значение, если текст затенен или в противном случае — ноль.                                                         |
| **\_Стиль ШРИФТА \_ тсаттрид \_ SmallCaps**                     | Содержит ненулевое значение, если текст является строчным или нулем в противном случае.                                                        |
| **ТСАТТРИД \_ \_ стиля шрифта \_**                       | Содержит сведения о промежутке между шрифтами.                                                                                                  |
| **ТСАТТРИД \_ \_ начертание \_ шрифта**                 | Не используется.                                                                                                                   |
| **\_ \_ \_ Двойной ЗАчеркивание шрифта \_ тсаттрид**         | Содержит ненулевое значение, если текст дважды зачеркиванием или нулем в противном случае.                                             |
| **\_Стиль ШРИФТА \_ тсаттрид \_ Зачеркнутый \_ одинарный**         | Содержит ненулевое значение, если текст является одинарным зачеркиванием или нулем в противном случае.                                             |
| **\_Подстрочный шрифт стиля ШРИФТА тсаттрид \_ \_**                     | Содержит ненулевое значение, если текст является подстрочным или нулем в противном случае.                                                        |
| **\_ \_ Надстрочный стиль шрифта \_ тсаттрид**                   | Содержит ненулевое значение, если текст является надстрочными или ноль в противном случае.                                                      |
| **\_ \_ Подчеркнутый стиль шрифта тсаттрид \_**                     | Содержит ненулевое значение, если текст подчеркнут или равен нулю в противном случае.                                                       |
| **\_ \_ \_ Двойной подчеркнутый стиль шрифта тсаттрид \_**             | Содержит ненулевое значение, если текст является двойным подчеркнутым или нулем в противном случае.                                                |
| **\_Стиль ШРИФТА \_ тсаттрид \_ Подчеркнутый \_ одинарный**             | Содержит ненулевое значение, если текст имеет одинарную подчеркивание или ноль в противном случае.                                                |
| **ТСАТТРИД \_ стиль шрифта в \_ \_ верхнем регистре**                     | Содержит ненулевое значение, если текст является прописным или нулем в противном случае.                                                        |
| **\_ \_ Вес стиля шрифта \_ тсаттрид**                        | Содержит насыщенность шрифта. Дополнительные сведения о возможных значениях см. в описании элемента **лфвеигхт** структуры LOGFONT. |
