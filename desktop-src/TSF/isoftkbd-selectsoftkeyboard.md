# <a name="isoftkbdselectsoftkeyboard-method"></a>Метод Исофткбд:: Селектсофткэйбоард

Метод **исофткбд:: селектсофткэйбоард** выбирает указанную мягкую клавиатуру.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SelectSoftKeyboard(
  [in] DWORD dwKeyboardId
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*двкэйбоардид* \[ окне\]
</dt> <dd>

Идентификатор выбираемой загружаемой клавиатуры.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                         |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>               |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Недопустимый параметр *двкэйбоардид* .<br/> |
