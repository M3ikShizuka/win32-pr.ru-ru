# <a name="ts_shift_-constants"></a>\_Константы сдвига служб терминалов \_ \*

\_Константы сдвига TS \_ \* используются интерфейсом **ианчор** для обработки скрытого текста и подсчета символов.



| Константа/значение                                                                                                                                                                                                                                       | Описание                                                                                                                                                                                                                                     |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TS_SHIFT_COUNT_HIDDEN"></span><span id="ts_shift_count_hidden"></span><dl> <dt>**TS \_ Счетчик СДВИГов \_ \_ скрыт**</dt> <dt>(0x1)</dt> </dl> | Указывает, что привязка будет перемещена на следующую границу области, включая границу скрытой области текста. Если не задано, привязка будет смещена после любого соседнего скрытого текста, пока не будет найдена область видимого текста.<br/> |
| <span id="TS_SHIFT_HALT_HIDDEN"></span><span id="ts_shift_halt_hidden"></span><dl> <dt>**TS \_ SHIFT \_ Остановка \_ скрытия**</dt> <dt>(0x2)</dt> </dl>    | Не используется.<br/>                                                                                                                                                                                                                            |
| <span id="TS_SHIFT_HALT_VISIBLE"></span><span id="ts_shift_halt_visible"></span><dl> <dt>**TS \_ СДВИГ \_ остановлен \_ видимым**</dt> <dt>(0x4)</dt> </dl> | Не используется.<br/>                                                                                                                                                                                                                            |
| <span id="TS_SHIFT_COUNT_ONLY"></span><span id="ts_shift_count_only"></span><dl> <dt>**TS \_ \_ \_ Только количество сдвигов**</dt> <dt>(0x8)</dt> </dl>       | Привязка не смещается.<br/>                                                                                                                                                                                                           |
