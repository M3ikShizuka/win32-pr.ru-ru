# <a name="isoftkbdgetsoftkeyboardpossize-method"></a>Метод Исофткбд:: Жетсофткэйбоардпоссизе

Метод **исофткбд:: жетсофткэйбоардпоссизе** извлекает начальную точку и размер экранной клавиатуры.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetSoftKeyboardPosSize(
  [out] POINT *lpStartPoint,
  [out] WORD  *lpwidth,
  [out] WORD  *lpheight
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*лпстартпоинт* \[ заполняет\]
</dt> <dd>

Указатель на буфер, в котором этот метод получает структуру [точек](/previous-versions//dd162805(v=vs.85)) , указывающую координаты верхнего левого положения экранной клавиатуры.

</dd> <dt>

*лпвидс* \[ заполняет\]
</dt> <dd>

Указатель на буфер, в котором этот метод получает ширину экранной клавиатуры.

</dd> <dt>

*лфеигхт* \[ заполняет\]
</dt> <dd>

Указатель на буфер, в котором этот метод получает высоту экранной клавиатуры.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                  |
|----------------------------------------------------------------------------------------------|----------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>        |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Один из параметров недопустим.<br/> |
