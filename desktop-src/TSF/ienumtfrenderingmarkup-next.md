# <a name="ienumtfrenderingmarkupnext-method"></a>Метод Иенумтфрендерингмаркуп:: Next

Метод **иенумтфрендерингмаркуп:: Next** получает от текущей позицией указанное число элементов в последовательности перечисления.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Next(
  [out] ULONG ulCount,
  [out]       ppElement,
  [out] ULONG *pcFetched
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*улкаунт* \[ заполняет\]
</dt> <dd>

\[out \] указывает количество элементов для получения.

</dd> <dt>

*ппелемент* \[ заполняет\]
</dt> <dd>

\[\]указатель out на массив структур [tf \_ рендерингмаркуп](/windows/desktop/TSF/tf-renderingmarkup) . Этот массив должен содержать по крайней мере *улкаунт* элементов в размере.

</dd> <dt>

*пкфетчед* \[ заполняет\]
</dt> <dd>

\[\]указатель out на значение ulong, получающее число фактически полученных элементов. Это значение может быть меньше запрошенного количества элементов. Этот параметр может принимать значение **NULL**.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                                                                                         |
|----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>                                                                               |
| <dl> <dt>**S \_ false**</dt> </dl>      | Метод достиг конца перечисления до получения указанного числа элементов.<br/> |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Один или несколько параметров недопустимы.<br/>                                                                      |
