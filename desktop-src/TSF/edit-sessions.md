# <a name="edit-sessions"></a>Сеансы изменения

Когда служба текстового ввода должна получить, изменить или задать текст в контексте, она должна запросить *сеанс изменения*. Когда запрос на изменение запрашивается, диспетчер TSF выполняет согласование с владельцем целевого контекста для блокировки документа запрошенного типа. При предоставлении блокировки документа Диспетчер TSF позволяет текстовой службе читать и записывать текст в контекст.
