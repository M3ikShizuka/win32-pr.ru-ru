# <a name="isoftkbdunadvisesoftkeyboardeventsink-method"></a>Метод Исофткбд:: Унадвисесофткэйбоардевентсинк

Метод **исофткбд:: унадвисесофткэйбоардевентсинк** удаляет приемник событий программируемой клавиатуры.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT UnadviseSoftKeyboardEventSink(
  [in] TfClientId tid
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*tid* \[ окне\]
</dt> <dd>

Идентификатор клиента, владеющего приемником событий программируемой клавиатуры. Это значение было передано при установке приемника событий с помощью [исофткбд:: адвисесофткэйбоардевентсинк](isoftkbd-advisesoftkeyboardeventsink.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                                   | Описание                                                   |
|---------------------------------------------------------------------------------------------------------|---------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                    | Метод успешно выполнен.<br/>                         |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>            | Недопустимый параметр *tid* .<br/>                    |
| <dl> <dt>**подключение \_ к \_ соединению E**</dt> </dl> | Приемник уведомлений, идентифицируемый *tid* , не найден.<br/> |
