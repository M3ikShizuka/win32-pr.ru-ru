# <a name="isoftkbdshowkeysforkeyscanmode-method"></a>Метод Исофткбд:: Шовкэйсфоркэйсканмоде

Метод **исофткбд:: шовкэйсфоркэйсканмоде** отображает ключи, используемые для режима просмотра ключа для экранной клавиатуры.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT ShowKeysForKeyScanMode(
  [in] KEYID *lpKeyID,
  [in] INT   iKeyNum,
  [in] BOOL  fHighL
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*лпкэйид* \[ окне\]
</dt> <dd>

Указатель на массив элементов KEYID, указывающий идентификаторы ключей для вывода.

</dd> <dt>

*икэйнум* \[ окне\]
</dt> <dd>

Число отображаемых ключей.

</dd> <dt>

*фхигхл* \[ окне\]
</dt> <dd>

Значение TRUE, если метод должен выделять ключи, и **false** в противном случае.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                  |
|----------------------------------------------------------------------------------------------|----------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>        |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Один из параметров недопустим.<br/> |
