# <a name="isoftkbdcreatesoftkeyboardwindow-method"></a>Метод Исофткбд:: Креатесофткэйбоардвиндов

Метод **исофткбд:: креатесофткэйбоардвиндов** создает окно с экранной клавиатурой.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT CreateSoftKeyboardWindow(
  [in] HWND          hOwner,
  [in] TITLEBAR_TYPE Titlebar_type,
  [in] INT           xPos,
  [in] INT           yPos,
  [in] INT           width,
  [in] INT           height
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ховнер* \[ окне\]
</dt> <dd>

Маркер окна, в котором будет содержаться мягкая клавиатура.

</dd> <dt>

*\_ Тип заголовка* \[ в\]
</dt> <dd>

Тип заголовка окна экранной клавиатуры. Возможные типы определяются перечислением [**\_ типа в строке заголовка**](titlebar-type.md) .

</dd> <dt>

*кспос* \[ окне\]
</dt> <dd>

Координата по оси x верхнего левого угла окна экранной клавиатуры.

</dd> <dt>

*ИПОС* \[ окне\]
</dt> <dd>

Координата y левого верхнего угла окна экранной клавиатуры.

</dd> <dt>

*Ширина* \[ окне\]
</dt> <dd>

Ширина окна экранной клавиатуры.

</dd> <dt>

*Высота* \[ окне\]
</dt> <dd>

Высота окна экранной клавиатуры.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>          |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Один или несколько параметров недопустимы.<br/> |
