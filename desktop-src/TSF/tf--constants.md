# <a name="tf_-constants"></a>\_ \* КОНСТАНТы TF

Вызовы функций TSF Atom используют эти константы для управления процессами ввода потока.



| Константа/значение                                                                                                                                                                                                                                                                       | Описание                                                                                          |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="TF_PROCESS_ATOM"></span><span id="tf_process_atom"></span><dl> <dt>**Tf \_ Обработка текста \_ Atom**</dt> <dt>( \_ КТФ \_ Process \_ Atom \_ )</dt> </dl>                               | Используется для уведомления функций Atom о инициации процесса ввода потока.<br/>                  |
| <span id="TF_ENABLE_PROCESS_ATOM"></span><span id="tf_enable_process_atom"></span><dl> <dt>**Tf \_ ВКЛЮЧИТЬ текст \_ процесса \_ Atom**</dt> <dt>( \_ КТФ \_ включить \_ Process \_ Atom \_ )</dt> </dl> | Используется для уведомления TSF с помощью функций Atom, которые включали процесс ввода потока.<br/> |
