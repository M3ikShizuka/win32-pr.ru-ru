# <a name="isoftkbdsetkeyboardlabeltextcombination-method"></a>Метод Исофткбд:: Сеткэйбоардлабелтексткомбинатион

Метод **исофткбд:: сеткэйбоардлабелтексткомбинатион** задает сочетание метки и текста, используемого для описания мягкой клавиатуры.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetKeyboardLabelTextCombination(
  [in] DWORD nModifierCombination
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*нмодифиеркомбинатион* \[ окне\]
</dt> <dd>

Сочетание метки и текста, используемого для описания экранной клавиатуры.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                        | Описание                                                 |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод успешно выполнен.<br/>                       |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Недопустимый параметр *нмодифиеркомбинатион* .<br/> |
