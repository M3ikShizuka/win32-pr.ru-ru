# <a name="values-for-guid_compartment_transitoryextension"></a>Значения для \_ транситорекстенсион секции \_ GUID

Следующие значения \_ \_ Секции ТРАНСИТОРЕКСТЕНСИОН секции GUID используются для управления поведением расширения транзитного модуля.

<dl> <dt>

<span id="TF_TRANSITORYEXTENSION_NONE"></span><span id="tf_transitoryextension_none"></span>**TF \_ транситорекстенсион \_ None**
</dt> <dd>

Это значение останавливает расширение транзитного расширения.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_FLOATING"></span><span id="tf_transitoryextension_floating"></span>**TF \_ транситорекстенсион \_ float**
</dt> <dd>

Это значение запускает транзитное расширение с плавающим пользовательским интерфейсом.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_ATSELECTION"></span><span id="tf_transitoryextension_atselection"></span>**TF \_ транситорекстенсион \_ атселектион**
</dt> <dd>

Это значение запускает расширение транзитного приложения с помощью всплывающего пользовательского интерфейса в IP-адресе или при выборе родительского диспетчера документов.

</dd> </dl>
