# <a name="structures-and-data-types"></a>Структуры и типы данных

В этом разделе содержатся сведения о структурах и типах данных, используемых в Microsoft Active Accessibility.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                             | Описание                                                                                                                                                              |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**мсааменуинфо**](/windows/win32/api/oleacc/ns-oleacc-msaamenuinfo)<br/>   | Используется разработчиками сервера для предоставления имен элементов меню, рисуемых владельцем.<br/>                                                                                      |
| [**хвиневенсук**](hwineventhook.md)<br/> | Используется для объявления функции обработчика событий окна.<br/>                                                                                                                 |
| [**виневентпрок**](/previous-versions/windows/desktop/legacy/dd373882(v=vs.85))<br/>   | Используется для объявления указателя на определяемую приложением функцию обратного вызова, которая вызывается системой в ответ на события, формируемые доступным объектом.<br/> |
