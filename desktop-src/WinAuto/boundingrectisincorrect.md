# <a name="boundingrectisincorrect"></a>баундингректисинкоррект

## <a name="text"></a>Текст

{0}Граница сообщаемого прямоугольника не согласуется с проверкой нажатия {1} параметра Rect в точке {2} .

## <a name="type"></a>Тип

Ошибка

## <a name="description"></a>Описание

Ограничивающий прямоугольник элемента не согласуется с ожидаемыми результатами проверки нажатия. Например, проверка нажатия точки за пределами ограничивающего прямоугольника элемента не возвращает другой элемент.
