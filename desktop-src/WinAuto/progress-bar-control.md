# <a name="progress-bar-control-msaa-ui-element-reference"></a>Элемент управления "индикатор выполнения" (Справочник по элементам пользовательского интерфейса MSAA)

> [!Note]  
> В этом разделе описываются **управляющие объекты индикатора выполнения** в целях справочника по элементам ПОЛЬЗОВАТЕЛЬСКОГО интерфейса MSAA. Создание объектов **управления "индикатор выполнения** " в различных ПЛАТФОРМАХ пользовательского интерфейса не описывается здесь. См. справочную документацию по API для платформы пользовательского интерфейса, которую вы используете.

 

Элементы управления "индикатор выполнения" указывают на ход выполнения длительной операции, например загрузку файла из Интернета. Обычно ход выполнения выражается в процентах от нуля (0) до 100 (100).

Имя класса окна для элемента управления "индикатор выполнения" — это \_ класс Progress, который определен как "мсктлс \_ Progress" в коммктрл. h.

## <a name="iaccessible-methods"></a>Методы IAccessible

Элементы управления "индикатор выполнения" поддерживают следующие методы [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) :

-   [**акчиттест**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)
-   [**акклокатион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)
-   [**аккнавигате**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate)
-   [**аккселект**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect)
