# <a name="content-of-descriptive-properties"></a>Содержимое описательных свойств

Интерфейс [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) предоставляет описательные свойства, описывающие различные аспекты объекта. Некоторые из этих свойств относятся к содержимому; другие свойства имеют содержимое, состоящее из описательного текста, предоставляемого сервером. Тип сведений для каждого свойства зависит от объекта.

В следующих разделах описываются сведения, которые клиенты получают из этих свойств. Они также предоставляют серверы с предложениями по выбору содержимого.

-   [Свойство Name](name-property.md)
-   [Свойство Role](role-property.md)
-   [Свойство State](state-property.md)
-   [Value, свойство](value-property.md)
-   [Свойство Description](description-property.md)
-   [Свойство справки](help-property.md)
-   [Хелптопик, свойство](helptopic-property.md)
-   [Кэйбоардшорткут, свойство](keyboardshortcut-property.md)
-   [DefaultAction, свойство](defaultaction-property.md)

При проектировании объектов со специальными возможностями разработчики сервера также должны обратиться к следующим разделам:

-   [Выбор свойств для поддержки](choosing-which-properties-to-support.md)
-   [Выбор содержимого для описательных свойств](choosing-the-content-for-descriptive-properties.md)

Сведения о параметрах и возвращаемых значениях этих свойств см. в разделе [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) справочника по Microsoft Active Accessibility [C/C++](c-c---reference.md).
