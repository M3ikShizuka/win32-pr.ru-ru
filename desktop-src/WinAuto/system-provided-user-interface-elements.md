# <a name="system-provided-user-interface-elements"></a>System-Provided элементов пользовательского интерфейса

Microsoft Active Accessibility предоставляет поддержку для большинства стандартных и стандартных элементов управления. На следующем рисунке показано типичное окно и некоторые предоставляемые системой Active Accessibility элементы пользовательского интерфейса, такие как строки заголовка, меню, поля со списком, элементы управления панели инструментов, элементы управления иерархического представления, строки состояния, размер освоиться, элементы управления "список" и полосы прокрутки.

![снимок экрана, показывающий несколько различных элементов пользовательского интерфейса](images/explorer.gif)

Microsoft Active Accessibility предоставляет предоставляемые системой элементы пользовательского интерфейса серверным приложениям, не требуя от разработчика сервера реализовать интерфейс [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) . Любое приложение, содержащее эти элементы, автоматически наследует их доступность.

Список элементов управления и других предоставляемых системой элементов пользовательского интерфейса, поддерживаемых Microsoft Active Accessibility, см. [в приложении a: Справочник по элементам пользовательского интерфейса, которые поддерживаются](appendix-a--supported-user-interface-elements-reference.md).
