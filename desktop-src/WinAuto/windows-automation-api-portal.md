# <a name="windows-accessibility-api-overview"></a>Windows Общие сведения об API специальных возможностей

в этом разделе приводятся общие обзоры и подробные справочные материалы по api для Microsoft Windows Automation api 3,0, включающего Windows реализацию [спецификации автоматизации пользовательского интерфейса](./uiauto-specandcommunitypromise.md)майкрософт и Microsoft Active Accessibility (устаревшая функция, появившаяся в Windows 95 как надстройка платформы).
