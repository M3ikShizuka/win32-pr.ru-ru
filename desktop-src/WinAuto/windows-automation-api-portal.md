# <a name="windows-accessibility-api-overview"></a>Общие сведения об API специальных возможностей Windows

В этом разделе приведены общие обзоры и подробные справочные материалы по API для Microsoft Windows Automation API 3,0, в том числе реализация [спецификации Microsoft UI Automation](./uiauto-specandcommunitypromise.md)и Microsoft Active Accessibility (устаревшая функция, появившаяся в Windows 95 как надстройка платформы).

Мы описываем сходства и различия между Microsoft Active Accessibility и автоматизацией пользовательского интерфейса, компонентами и функциями, обеспечивающими совместную работу двух технологий, и предоставляет рекомендации по выбору реализуемой технологии для конкретного сценария.
