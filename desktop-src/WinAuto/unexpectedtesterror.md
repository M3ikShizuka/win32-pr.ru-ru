# <a name="unexpectedtesterror"></a>унекспектедтестеррор

## <a name="text"></a>Текст

:(Исключения {0} )

## <a name="type"></a>Тип

Ошибка

## <a name="description"></a>Описание

Исключение, не связанное с проверками, было вызвано Аккчеккер. Сведения о проблемах см. в диалоговом окне Аккчеккер **о программе** для получения контактной информации.
