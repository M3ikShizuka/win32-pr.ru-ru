# <a name="choosing-the-content-for-descriptive-properties"></a>Выбор содержимого для описательных свойств

Хотя содержимое некоторых свойств является специфичным, содержимое для других свойств состоит из описательного текста, который можно предоставить разработчику сервера. Кроме того, тип сведений для каждого свойства зависит от объекта. В следующей таблице приведены рекомендации по выбору содержимого этих описательных свойств.



| Свойство                                        | Предложение содержимого                                                                                                                                                                                                                                                    |
|-------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Имя**](name-property.md)                   | Метка, которая кратко описывает и определяет объект в его контейнере, например текст в кнопке "Кнопка", имя пункта меню или метку, отображаемую рядом с элементом управления "поле ввода".                                                                              |
| [**DefaultAction**](defaultaction-property.md) | Взаимодействие по умолчанию с объектом. Строка, возвращаемая этим свойством, представляет собой одну команду, например "Press" для кнопки панели инструментов, или короткую фразу, которая начинается с глагола.                                                                               |
| [**Значение**](value-property.md)                 | Сведения, содержащиеся в объекте, например текст в элементе управления "поле ввода" или имя файла ссылки HTML. Содержимое свойства [**value**](value-property.md) может измениться, в то время как содержимое свойства [**имя**](name-property.md) не изменяется. |
| [**Описание**](description-property.md)     | Описывает внешний вид объекта.                                                                                                                                                                                                                                    |
| [**Справка**](help-property.md)                   | Подсказка или сведения в стиле подсказки используются для описания того, что делает объект, и как его использовать.                                                                                                                                                                   |
| [**хелптопик**](helptopic-property.md)         | Идентификатор контекста раздела [WinHelp](/windows/win32/api/winuser/nf-winuser-winhelpa) , который документирует объект.                                                                                                                                                 |



 

Используйте одно из стандартных значений [роли объекта](object-roles.md) для [**роли**](role-property.md) и соответствующее сочетание [констант состояния объекта](object-state-constants.md) для [**состояния**](state-property.md).

Содержимое свойств пользовательских элементов управления должно соответствовать содержимому свойств элементов управления, предоставляемых системой, которые моделируются пользовательскими элементами управления. Сведения о свойствах, поддерживаемых элементами управления, предоставляемыми системой, см. [в приложении а. Справочник по элементам пользовательского интерфейса](appendix-a--supported-user-interface-elements-reference.md).
