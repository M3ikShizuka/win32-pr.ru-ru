# <a name="boundingrectnothittestable"></a>баундингректноситтестабле

## <a name="text"></a>Текст

Отчетный прямоугольник не содержит точку, в которой этот элемент можно получить с помощью проверки нажатия.

## <a name="type"></a>Тип

Предупреждение

## <a name="description"></a>Описание

Ограничивающий прямоугольник элемента недоступен при проверке нажатия. Например, он находится за пределами физических координат экрана (или видимого региона) рабочего стола.
