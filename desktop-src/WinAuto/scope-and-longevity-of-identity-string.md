# <a name="scope-and-longevity-of-identity-string"></a>Область и долговечность строки идентификатора

Строка идентификации определяет конкретный экземпляр элемента пользовательского интерфейса. Несколько экземпляров элемента со специальными возможностями будут иметь разные строки идентификаторов. Поэтому обычно не рекомендуется сохранять строку идентификатора.
