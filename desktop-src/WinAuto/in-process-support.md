# <a name="in-process-support"></a>Поддержка In-Process

Текущая реализация динамической аннотации полностью обрабатывается и, следовательно, позволяет закомментировать процессы, владеющие этими элементами пользовательского интерфейса, только элементами пользовательского интерфейса. Один процесс не может закомментировать элементы пользовательского интерфейса другого процесса.

Обратите внимание, что это влияет только на действие настройки заметки. Он не мешает клиентам обращаться к свойствам (с заметками или иным способом) элементов пользовательского интерфейса в других процессах.
