# <a name="combo-box-msaa-ui-element-reference"></a>Поле со списком (Справочник по элементам пользовательского интерфейса MSAA)

> [!Note]  
> В этом разделе описываются объекты **поля со списком** в целях справочника по элементам ПОЛЬЗОВАТЕЛЬСКОГО интерфейса MSAA. Создание объектов полей **со списками** в различных ПЛАТФОРМАХ пользовательского интерфейса не описывается здесь. См. справочную документацию по API для платформы пользовательского интерфейса, которую вы используете.

 

Поле со списком представляет собой список, объединенный со статическим элементом управления или элементом управления "Поле ввода", который отображает выбранный элемент в списке поля со списком. Часть элемента управления со списком отображается всегда или раскрывается только в том случае, когда пользователь выбирает стрелку раскрывающегося списка (которая является кнопкой кнопки) рядом с элементом управления. Если поле выбора является элементом управления редактирования, пользователь может ввести информацию, не указанную в списке. в противном случае пользователь может выбрать только элементы из списка.

Имя класса окна для поля со списком — "COMBOBOX".

Содержимое свойств [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) зависит от того, какие из следующих частей поля со списком запрашиваются клиентом:

-   Окно поля со списком
-   Элемент управления "поле ввода" или "статический текст"
-   Стрелка раскрывающегося списка (кнопка "Отправить")
-   Поле со списком
-   Элементы списка в списке

## <a name="iaccessible-methods"></a>Методы IAccessible

Поля со списком поддерживают следующие методы [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) :

-   [**аккдодефаултактион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction)
-   [**акчиттест**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)
-   [**акклокатион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)
-   [**аккнавигате**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate)
-   [**аккселект**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect)
