# <a name="controltypenoexpectedpatternssupported"></a>контролтипеноекспектедпаттернссуппортед

## <a name="text"></a>Текст

Элемент с ControlType {0} должен поддерживать хотя бы один из этих шаблонов: {1} но этот элемент не

## <a name="type"></a>Тип

Error

## <a name="description"></a>Описание

Поддержка модели автоматизации пользовательского интерфейса элемента не соответствует спецификации модели автоматизации пользовательского интерфейса, поскольку элемент должен поддерживать по крайней мере один из указанных в списке шаблонов элементов управления, но не имеет. В результате этот элемент может быть неверно предоставлен для вспомогательных технологий, что может оказать серьезное влияние на взаимодействие с пользователем.

## <a name="possible-causes"></a>Возможные причины

-   Неполная реализация модели автоматизации пользовательского интерфейса.
-   Свойство ControlType задано неправильно.

## <a name="remarks"></a>Remarks

Аккчеккер регистрирует это сообщение об ошибке для неопределенного индикатора выполнения. Можно проигнорировать это сообщение и (или) добавить его в список подавлений.
