# <a name="text-control-type"></a>Тип элемента управления "текст"

В этом разделе содержатся сведения о поддержке автоматизации пользовательского интерфейса Майкрософт для типа элемента управления **Text** .

Элемент управления "текст" — это базовый элемент пользовательского интерфейса, представляющий фрагмент текста на экране.

В следующих разделах определяется необходимая древовидная структура модели автоматизации пользовательского интерфейса, свойства, шаблоны элементов управления и события для типа элемента управления **Text** . Требования к автоматизации пользовательского интерфейса применяются ко всем элементам управления "дерево", в которых платформа или платформа пользовательского интерфейса интегрирует поддержку модели автоматизации пользовательского интерфейса для типов элементов управления и шаблонов элементов управления.

В этом разделе содержатся следующие подразделы.

-   [Типичная древовидная структура](#typical-tree-structure)
-   [Соответствующие свойства](#relevant-properties)
-   [Обязательные шаблоны элементов управления](#required-control-patterns)
-   [Обязательные события](#required-events)
-   [Связанные темы](#related-topics)

Дополнительные сведения о дереве автоматизации пользовательского интерфейса см. в разделе [Общие сведения о дереве модели автоматизации пользовательского интерфейса](uiauto-treeoverview.md).

Элемент управления "Текст" может использоваться один в качестве метки или статического текста в форме. Он также может содержаться в структуре одного из следующих элементов:

-   [DataItem](uiauto-supportdataitemcontroltype.md)
-   [ListItem](uiauto-supportlistitemcontroltype.md)
-   [TreeItem](uiauto-supporttreeitemcontroltype.md)

Текстовые элементы управления могут не отображаться в представлении содержимого дерева модели автоматизации пользовательского интерфейса, поскольку текст часто отображается с помощью свойства **Name** другого элемента управления. Например, текст, используемый для обозначения элемента управления "поле со списком", предоставляется через свойство **имени** элемента управления. Поскольку элемент управления "поле со списком" находится в представлении содержимого дерева модели автоматизации пользовательского интерфейса, элемент управления "текст" не должен быть там. Элемент управления "текст" может иметь дочерние элементы в представлении содержимого, если имеется внедренный объект, например гиперссылка.

| [**UIA \_ аутоматионидпропертид**](uiauto-automation-element-propids.md)                 | См. примечания. | Значение этого свойства должно быть уникальным среди всех одноранговых элементов в необработанном представлении дерева модели автоматизации пользовательского интерфейса.                                                                                                                                                                                                                                        |
| [**UIA \_ баундингректанглепропертид**](uiauto-automation-element-propids.md)       | См. примечания. | Внешний прямоугольник, содержащий весь элемент управления.                                                                                                                                                                                                                                                                                            |
| [**UIA \_ кликкаблепоинтпропертид**](uiauto-automation-element-propids.md)             | См. примечания. | Поддерживается при наличии ограничивающего прямоугольника. Если не все точки внутри ограничивающего прямоугольника являются щелчками, а элемент выполняет специализированное тестирование нажатия, переопределите и предоставьте точку для щелчка.                                                                                                                                                |
| [**UIA \_ контролтипепропертид**](uiauto-automation-element-propids.md)                   | **Text**   |                                                                                                                                                                                                                                                                                                                                                     |
| [**UIA \_ исконтентелементпропертид**](uiauto-automation-element-propids.md)         | Зависит    | Элемент управления "текст" является содержимым, если он содержит сведения, не представленные в свойстве имени другого элемента управления.                                                                                                                                                                                                                                              |
| [**UIA \_ исконтролелементпропертид**](uiauto-automation-element-propids.md)         | TRUE       | Элемент управления "Текст" всегда должен быть элементом управления.                                                                                                                                                                                                                                                                                                          |
| [**UIA \_ искэйбоардфокусаблепропертид**](uiauto-automation-element-propids.md)   | См. примечания. | Если элемент управления может получать фокус клавиатуры, он должен поддерживать это свойство.                                                                                                                                                                                                                                                                           |
| [**UIA \_ лабеледбипропертид**](uiauto-automation-element-propids.md)                       | NULL       | Элементы управления "Текст" не имеют меток со статическим текстом.                                                                                                                                                                                                                                                                                                      |
| [**UIA \_ локализедконтролтипепропертид**](uiauto-automation-element-propids.md) | См. примечания. | Локализованная строка, соответствующая типу элемента управления **Text** . Значение по умолчанию — "Text" для en-US или English (США).                                                                                                                                                                                                                      |
| [**UIA \_ намепропертид**](uiauto-automation-element-propids.md)                                 | См. примечания. | Именем текстового элемента управления может быть отображаемый текст. Однако если элемент управления также поддерживает [текстовый](uiauto-implementingtextandtextrange.md) шаблон и текст имеет большой объем, не используйте его в качестве значения **Name** . Вместо этого укажите более короткое значение **имени** , производное от других свойств элемента управления. |



 

## <a name="required-control-patterns"></a>Обязательные шаблоны элементов управления

В следующей таблице перечислены шаблоны элементов управления модели автоматизации пользовательского интерфейса, которые должны поддерживаться элементами управления "текст". Дополнительные сведения о шаблонах элементов управления см. в разделе [UI Automation Control Patterns Overview](uiauto-controlpatternsoverview.md).



| Шаблон элемента управления                                         | Поддержка | Примечания                                                                                                                                                                                                                                                                  |
|---------------------------------------------------------|---------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IGridItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igriditemprovider)   | Зависит | Если элемент управления "текст" содержится в элементе управления "Таблица", должен поддерживаться шаблон элемента управления [GridItem](uiauto-implementinggriditem.md) .                                                                                                                            |
| [**ITableItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableitemprovider) | Зависит | Если элемент управления "текст" содержится в элементе управления "Таблица", должен поддерживаться шаблон элемента управления [TableItem](uiauto-implementingtableitem.md) .                                                                                                                          |
| [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider)           | Зависит | Текст должен поддерживать шаблон элемента управления [Text](uiauto-implementingtextandtextrange.md) для улучшения специальных возможностей; Однако это не обязательно. Шаблон элемента управления Text целесообразно использовать, если текст имеет множество стилей и атрибутов (например, цвет, полужирный шрифт и курсив). |
| [**IValueProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ivalueprovider)         | Никогда   | Элемент управления "текст" никогда не поддерживает шаблон элемента управления [value](uiauto-implementingvalue.md) . Если текст доступен для редактирования, то это тип элемента управления [Edit](uiauto-supporteditcontroltype.md) .                                                                                    |
