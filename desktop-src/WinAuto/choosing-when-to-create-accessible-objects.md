# <a name="choosing-when-to-create-accessible-objects"></a>Выбор времени создания объектов со специальными возможностями

Разработчики сервера могут создавать все объекты, доступные в контейнере, при создании контейнера, а также динамически создавать объекты со специальными возможностями.

Например, представьте себе диалоговое окно с несколькими пользовательскими элементами управления. Сервер создает объекты со специальными возможностями для всех пользовательских элементов управления при отображении диалогового окна. Однако если один из элементов управления является пользовательским полем со списком, сервер ждет, пока пользователь не выберет его, чтобы создать специальные объекты для элементов, содержащихся в поле со списком.

Если родительский объект создает объекты со специальными возможностями динамически, приложения используют меньше памяти, чем если бы все доступные объекты были созданы заранее.
