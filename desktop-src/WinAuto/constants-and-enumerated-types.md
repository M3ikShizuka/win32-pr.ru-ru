# <a name="constants-active-accessibility-user-interface-services"></a>Константы (Active Accessibility служб пользовательского интерфейса)

В этом разделе содержатся сведения о константах, используемых в Microsoft Active Accessibility.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                               | Описание                                                                                                                                                                                                                                                                                                                                                                           |
|---------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Константы событий**](event-constants.md)<br/>               | В этом разделе описываются события, создаваемые операционной системой и серверными приложениями.<br/>                                                                                                                                                                                                                                                                     |
| [**Константы навигации**](navigation-constants.md)<br/>     | В этом разделе описываются константные значения, определенные в олеакк. h, которые указывают на *пространственный* (вверх, вниз, слева и справа) или *логический* (первый дочерний, последний, следующий и предыдущий) направление, которое наблюдается, когда клиенты используют [**IAccessible:: аккнавигате**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate) для перехода от одного элемента пользовательского интерфейса к другому в пределах того же контейнера.<br/> |
| [**Идентификаторы объектов**](object-identifiers.md)<br/>         | В этом разделе описываются идентификаторы объектов Microsoft Active Accessibility, 32-разрядные значения, которые указывают *категории* объектов, доступных в окне.<br/>                                                                                                                                                                                                                |
| [**Роли объектов**](object-roles.md)<br/>                     | В этом разделе описываются константные значения, используемые для описания ролей различных объектов пользовательского интерфейса в приложении.<br/>                                                                                                                                                                                                                                                               |
| [**Константы состояния объектов**](object-state-constants.md)<br/> | В этом разделе описаны константные значения, используемые для описания состояния объектов в пользовательском интерфейсе приложения.<br/>                                                                                                                                                                                                                                                                       |
| [**Константы СЕЛФЛАГ**](selflag.md)<br/>                     | В этом разделе описываются константные значения, используемые для указания способа выбора доступного объекта или фокуса.<br/>                                                                                                                                                                                                                                                     |
