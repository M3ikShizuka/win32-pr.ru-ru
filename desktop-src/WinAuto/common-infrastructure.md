# <a name="common-infrastructure"></a>Общая инфраструктура

Windows предлагает две спецификации API для специальных возможностей пользовательского интерфейса и автоматизации тестирования программного обеспечения: microsoft Active Accessibility и microsoft UI automation. В этом разделе описываются функции и компоненты, общие для Microsoft Active Accessibility и модели автоматизации пользовательского интерфейса, которые позволяют им взаимодействовать.
