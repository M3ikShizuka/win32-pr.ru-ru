# <a name="active-accessibility-objects"></a>Active Accessibility объекты

В терминологии Microsoft Active Accessibility имеются *Доступные объекты* и *простые элементы*. Хотя большинство приложений содержат оба объекта, доступные объекты являются более распространенными, чем простые элементы. В этом разделе определяются и обсуждаются доступные объекты и простые элементы, а также предоставляются соответствующие примеры пользовательского интерфейса.
