# <a name="inconsistentstate-inconsistentproperties"></a>Инконсистентстате, Инконсистентпропертиес

## <a name="text"></a>Текст

Элемент управления имеет несоответствующие {0} состояния или свойства. {1}

## <a name="type"></a>Тип

Error

## <a name="description"></a>Описание

Элемент сообщает о несоответствиях или конфликтующих состояниях MSAA или свойств модели автоматизации пользовательского интерфейса. Например, если метод [**Get \_ аккстате**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accstate) возвращает любое из следующих сочетаний.

-   \_Развернутая система состояний \_ и \_ Система состояний \_ свернуты
-   \_Система состояний \_ выбрана и не является выбранной \_ системой состояния \_
-   Система \_ состояний \_ с \_ фокусом и без системы состояний \_

Эта проблема вызывает проблемы для пользователей, которые используют средство чтения с экрана и клавиатуру для навигации, поскольку пользователь может сообщить о неправильном состоянии элемента.

## <a name="possible-causes"></a>Возможные причины

Для элемента или его родителя задано неправильное состояние MSAA.
