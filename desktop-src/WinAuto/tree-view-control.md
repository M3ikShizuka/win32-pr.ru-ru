# <a name="tree-view-control-msaa-ui-element-reference"></a>Элемент управления иерархического представления (Справочник по элементам пользовательского интерфейса MSAA)

> [!Note]  
> В этом разделе описываются **управляющие объекты представления дерева** для ссылки на элемент ПОЛЬЗОВАТЕЛЬСКОГО интерфейса MSAA. Создание **управляющих объектов древовидного представления** в различных ПЛАТФОРМАХ пользовательского интерфейса не описывается здесь. См. справочную документацию по API для платформы пользовательского интерфейса, которую вы используете.

 

Элемент управления иерархического представления отображает иерархический список элементов, использующих линии для соединения связанных элементов в иерархии. Каждый элемент состоит из метки и необязательного точечного рисунка. Проводник Windows использует элемент управления иерархического представления для отображения каталогов.

Имя класса Window для элемента управления представления в виде дерева — это WC \_ TREEVIEW, который определен как "систривиев" в коммктрл. h.

## <a name="iaccessible-methods"></a>Методы IAccessible

Элемент управления иерархического представления поддерживает следующие методы [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) :



| Метод                                                                    | Комментарии                                                                                                                                                                                                                                                              |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**аккдодефаултактион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) | Сам объект представления в виде дерева не имеет метода [**аккдодефаултактион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) . В зависимости от свойства **State** элемента в представлении в виде дерева **аккдодефаултактион** либо развертывает, либо сворачивает указанный элемент. |
| [**акчиттест**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)                 |                                                                                                                                                                                                                                                                       |
| [**акклокатион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)               |                                                                                                                                                                                                                                                                       |

## <a name="notes"></a>Примечания

Свойство **акклокатион** элементов представления в виде дерева возвращает расположение только текста в элементе. Координаты, возвращаемые функцией **акклокатион** для элементов представления в виде дерева, не включают значок, расположенный слева от текста.
