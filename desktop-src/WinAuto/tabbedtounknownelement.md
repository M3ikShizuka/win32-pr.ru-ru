# <a name="tabbedtounknownelement"></a>таббедтаункновнелемент

## <a name="text"></a>Текст

Переход к элементу за пределами целевого HWND.

## <a name="type"></a>Тип

Error

## <a name="description"></a>Описание

Использование стандартной навигации с помощью клавиатуры (TAB или SHIFT + TAB) приводит к сдвигу фокуса за пределами HWND целевого объекта проверки.

Аккчеккер перемещает дерево элементов в HWND верхнего уровня для выбранного целевого объекта проверки перед выполнением каких-либо задач проверки. Это сокращает количество ложных ошибок, создаваемых, если HWND, выбранный для проверки, является частью более сложной клиентской области.

## <a name="possible-causes"></a>Возможные причины

-   Целью проверки не требуется реализация с вкладками для обеспечения полной функциональности.
-   Взаимодействие с пользователем во время проверки, например перемещение фокуса на нецелевой HWND, мешает процессу проверки.
