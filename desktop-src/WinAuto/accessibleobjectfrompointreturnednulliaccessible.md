# <a name="accessibleobjectfrompointreturnednulliaccessible"></a>акцессиблеобжектфромпоинтретурнеднуллиакцессибле

## <a name="text"></a>Текст

Акцессиблеобжектфромпоинт ( {0} , {1} ) вернул значение null

## <a name="type"></a>Тип

Ошибка

## <a name="description"></a>Описание

Адрес интерфейса [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) элемента пользовательского интерфейса, полученного для заданных координат, имеет значение null.

## <a name="possible-causes"></a>Возможные причины

-   Взаимодействие с пользователем во время проверки, например перемещение фокуса на нецелевой HWND, мешает процессу проверки.
-   Неправильная или недопустимая реализация MSAA.
