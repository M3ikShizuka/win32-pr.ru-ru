# <a name="appearstonotsupporttabbing"></a>аппеарстонотсуппорттаббинг

## <a name="text"></a>Текст

Приложение не поддерживает переходы.

## <a name="type"></a>Тип

Error

## <a name="description"></a>Описание

Приложение не поддерживает стандартную навигацию с помощью клавиатуры (TAB или SHIFT + TAB).

Эта проблема приводит к проблемам с удобством использования для людей, которые используют средство чтения с экрана или клавиатуру для навигации, поскольку ни один из элементов в приложении не может быть обнаруживаемым.

## <a name="possible-causes"></a>Возможные причины

-   Платформа пользовательского интерфейса приложения не поддерживает MSAA.
-   MSAA не реализован ни в одном элементе управления в приложении.
