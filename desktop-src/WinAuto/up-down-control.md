# <a name="up-down-control-msaa-ui-element-reference"></a>Элемент управления Up-Down (Справочник по элементам пользовательского интерфейса MSAA)

> [!Note]  
> В этом разделе описываются объекты **управления "вверх/вниз** ", которые предназначены для ссылки на элемент ПОЛЬЗОВАТЕЛЬСКОГО интерфейса MSAA. Создание объектов **управления "вверх-вниз** " в различных ПЛАТФОРМАХ пользовательского интерфейса не описывается здесь. См. справочную документацию по API для платформы пользовательского интерфейса, которую вы используете.

 

Элемент управления "вверх/вниз", также известный как элемент управления "Счетчик", объединяет пару кнопок, отображаемых в виде стрелок, с элементом управления "Кнопка ввода". Щелкая стрелки, вы увеличиваете или уменьшаете значение в элементе управления "поле ввода".

Имя класса Window для элемента управления "вверх/вниз" является \_ классом Up, который определен как "мсктлс \_ updown32" в коммктрл. h.

## <a name="iaccessible-methods"></a>Методы IAccessible

Элемент управления "вверх-вниз" поддерживает следующие методы [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) :

-   [**акчиттест**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)
-   [**акклокатион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)
-   [**аккнавигате**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate)
-   [**аккселект**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect)
