# <a name="multiple-annotations-on-a-single-accessible-element"></a>Несколько заметок для одного доступного элемента

В настоящее время доступный элемент поддерживает только одну заметку для каждого свойства. Последовательные заметки в свойстве заменяют и переопределяют предыдущие аннотации этого свойства.
