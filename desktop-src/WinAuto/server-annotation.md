# <a name="server-annotation"></a>Заметки сервера

В этом разделе приводятся сведения об использовании серверных заметок.

## <a name="summary"></a>Сводка

Вы определяете класс, реализующий [**иаккпропсервер**](/windows/desktop/api/oleacc/nn-oleacc-iaccpropserver), создаете его экземпляр и сообщаете системе, что необходимо переопределить определенные свойства в определенных элементах пользовательского интерфейса. Когда клиент запрашивает одно из свойств из одного из элементов пользовательского интерфейса, вызывается объект, и ему предоставляется возможность предоставить значение. Если объект возвращает значение, это значение передается обратно клиенту. Если объект не возвращает значение, клиенту возвращается значение по умолчанию для этого элемента пользовательского интерфейса.

## <a name="when-to-use-this-technique"></a>Когда следует использовать этот метод

Используйте этот метод, если требуется переопределить свойства [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) для объекта. Этот метод гораздо проще, чем предыдущие методы **IAccessible** . Дополнительные сведения см. [в разделе альтернативы динамической аннотации](alternatives-to-dynamic-annotation.md).

Нельзя использовать заметки сервера для изменения структуры предоставленного объекта. Чтобы изменить структуру объекта, необходимо реализовать полный указатель интерфейса [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) .
