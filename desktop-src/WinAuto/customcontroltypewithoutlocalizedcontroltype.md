# <a name="customcontroltypewithoutlocalizedcontroltype"></a>кустомконтролтипевисаутлокализедконтролтипе

## <a name="text"></a>Текст

Для элемента не задано свойство Локализедконтролтипе

## <a name="type"></a>Тип

Error

## <a name="description"></a>Описание

Свойство [**локализедконтролтипе**](/windows/desktop/api/UIAutomationClient/nf-uiautomationclient-iuiautomationelement-get_currentlocalizedcontroltype) элемента не задано.

Большинство средств чтения с экрана считывают локализованный тип элемента управления для описания природы элемента. Если локализованный тип элемента управления не задан, средство чтения с экрана будет возвращаться к чтению основного типа элемента управления, который всегда находится на английском языке и может иметь смысл не иметь смысла для всех пользователей.
