# <a name="uiaduplicateaccesskey"></a>уиадупликатеакцесскэй

## <a name="text"></a>Текст

Существует одноуровневый элемент автоматизации с таким же ключом доступа или сочетанием клавиш

## <a name="type"></a>Тип

Error

## <a name="description"></a>Описание

Сочетание клавиш доступа или сочетания клавиш, назначенное элементу, также назначается другому элементу в целевом объекте проверки.

Эта проблема вызывает проблемы для тех, кто полагается на клавиатуру для навигации, так как может вызываться несколько элементов или циклировать фокус, используя один и тот же ярлык или сочетание клавиш.
