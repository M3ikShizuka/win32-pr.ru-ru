# <a name="scrollbar-control-type"></a>Тип элемента управления ScrollBar

В этом разделе содержатся сведения о поддержке модели автоматизации пользовательского интерфейса Майкрософт для типа элемента управления **ScrollBar** .

С помощью элементов управления "Полоса прокрутки" пользователи могут прокручивать содержимое внутри окна или контейнера элементов. Элемент управления состоит из набора кнопок и элемента управления Thumb.

В следующих разделах определяется необходимая древовидная структура модели автоматизации пользовательского интерфейса, свойства, шаблоны элементов управления и события для типа элемента управления **ScrollBar** . Требования к автоматизации пользовательского интерфейса применяются ко всем элементам управления "полоса прокрутки", в которых платформа или платформа пользовательского интерфейса интегрирует поддержку автоматизации пользовательского интерфейса для типов элементов управления и шаблонов элементов управления.

В этом разделе содержатся следующие подразделы.

-   [Типичная древовидная структура](#typical-tree-structure)
-   [Соответствующие свойства](#relevant-properties)
-   [Обязательные шаблоны элементов управления](#required-control-patterns)
-   [Обязательные события](#required-events)
-   [См. также](#related-topics)

Дополнительные сведения о дереве автоматизации пользовательского интерфейса см. в разделе [Общие сведения о дереве модели автоматизации пользовательского интерфейса](uiauto-treeoverview.md).

Элемент управления "полоса прокрутки" может иметь от нуля до пяти дочерних элементов. Так как поддерево содержит более одного элемента управления "Кнопка", элементу необходимо задать конкретное значение [**UIA \_ аутоматионидпропертид**](uiauto-automation-element-propids.md) для каждого элемента, чтобы сделать их доступными для автоматизированных средств тестирования.

## <a name="relevant-properties"></a>Соответствующие свойства

В следующей таблице перечислены свойства модели автоматизации пользовательского интерфейса, значения или определения которых особенно важны для элементов управления "полоса прокрутки". Обратите внимание, что элемент управления "полоса прокрутки" не имеет содержимого; его функциональность предоставляется с помощью шаблона элемента управления [Scroll](uiauto-implementingscroll.md) , который поддерживается в контейнере для прокрутки.

Дополнительные сведения о свойствах модели автоматизации пользовательского интерфейса см. в разделе [Получение свойств элементов модели автоматизации пользовательского интерфейса](uiauto-propertiesforclients.md).



| Свойство модели автоматизации пользовательского интерфейса                                                                                              | Значение         | Примечания                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|---------------------------------------------------------------------------------------------------------------------|---------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**UIA \_ аутоматионидпропертид**](uiauto-automation-element-propids.md)                 | См. примечания.    | Значение этого свойства должно быть уникальным среди всех одноранговых элементов в необработанном представлении дерева модели автоматизации пользовательского интерфейса.                                                                                                                                                                                                                                                                                                                                                                    |
| [**UIA \_ баундингректанглепропертид**](uiauto-automation-element-propids.md)       | См. примечания.    | Внешний прямоугольник, содержащий весь элемент управления.                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**UIA \_ кликкаблепоинтпропертид**](uiauto-automation-element-propids.md)             | не число           | Элемент управления "Полоса прокрутки" не имеет активных точек.                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**UIA \_ контролтипепропертид**](uiauto-automation-element-propids.md)                   | **Элемента** | Это значение одинаково для всех инфраструктур. Полосы прокрутки, которые работают как ползунки, должны использовать тип элемента управления [Slider](uiauto-supportslidercontroltype.md) .                                                                                                                                                                                                                                                                                                                        |
| [**UIA \_ исконтентелементпропертид**](uiauto-automation-element-propids.md)         | FALSE         | Элемент управления "Полоса прокрутки" никогда не является элементом содержимого. Если полоса прокрутки является автономным элементом управления, он должен выполнить тип элемента управления [Slider](uiauto-supportslidercontroltype.md) и вернуть [**UIA \_ слидерконтролтипеид**](uiauto-controltype-ids.md) для свойства [**Иуиаутоматионелемент:: куррентконтролтипе**](/windows/desktop/api/UIAutomationClient/nf-uiautomationclient-iuiautomationelement-get_currentcontroltype) (или [**качедконтролтипе**](/windows/desktop/api/UIAutomationClient/nf-uiautomationclient-iuiautomationelement-get_cachedcontroltype)). |
| [**UIA \_ исконтролелементпропертид**](uiauto-automation-element-propids.md)         | true          | Элемент управления "полоса прокрутки" всегда включается в представление элемента управления дерева модели автоматизации пользовательского интерфейса.                                                                                                                                                                                                                                                                                                                                                                                        |
| [**UIA \_ искэйбоардфокусаблепропертид**](uiauto-automation-element-propids.md)   | См. примечания.    | Если элемент управления может получать фокус клавиатуры, он должен поддерживать это свойство. Элемент управления "полоса прокрутки" редко принимает фокус, но когда он выполняется, фокус должен оставаться на самом элементе управления полосы прокрутки, а не на дочерних кнопках или Thumb. Пользователь должен иметь возможность выполнять все действия прокрутки с помощью клавиш стрелка вверх и стрелка вниз (или стрелка вправо и стрелка влево), а также клавиши PAGE UP и PAGE DOWN.                                                                |
| [**UIA \_ лабеледбипропертид**](uiauto-automation-element-propids.md)                       | NULL          | Полосы прокрутки не имеют меток.                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**UIA \_ локализедконтролтипепропертид**](uiauto-automation-element-propids.md) | См. примечания.    | Локализованная строка, соответствующая типу элемента управления **ScrollBar** . Значение по умолчанию — "полоса прокрутки" для en-US или English (США).                                                                                                                                                                                                                                                                                                                                       |
| [**UIA \_ намепропертид**](uiauto-automation-element-propids.md)                                 | NULL          | Элемент управления "полоса прокрутки" не имеет элементов содержимого, и свойство [**UIA \_ намепропертид**](uiauto-automation-element-propids.md) не требуется устанавливать.                                                                                                                                                                                                                                                                                           |
| [**UIA \_ ориентатионпропертид**](uiauto-automation-element-propids.md)                   | См. примечания.    | Элемент управления "Полоса прокрутки" всегда должен представлять горизонтальную или вертикальную ориентацию.                                                                                                                                                                                                                                                                                                                                                                                               |



 

## <a name="required-control-patterns"></a>Обязательные шаблоны элементов управления

В следующей таблице перечислены шаблоны элементов управления модели автоматизации пользовательского интерфейса, которые должны поддерживаться всеми элементами управления "полоса прокрутки". Дополнительные сведения о шаблонах элементов управления см. в разделе [UI Automation Control Patterns Overview](uiauto-controlpatternsoverview.md).

> [!Note]  
> Если полоса прокрутки используется в качестве элемента управления только для манипуляций с мышью, шаблоны элементов управления не поддерживаются. Если он используется в качестве элемента управления "ползунок" в приложении, ему должен быть присвоен тип элемента управления [Slider](uiauto-supportslidercontroltype.md) .

 



| Шаблон элемента управления                                           | Поддержка | Примечания                                                                                                                                                                                                                          |
|-----------------------------------------------------------|---------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IRangeValueProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irangevalueprovider) | Зависит | Шаблон элемента управления [RangeValue](uiauto-implementingrangevalue.md) должен поддерживаться, только если шаблон элемента управления [Scroll](uiauto-implementingscroll.md) не поддерживается в контейнере с полосой прокрутки. |
| [**IScrollProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollprovider)         | Никогда   | Шаблон элемента управления [Scroll](uiauto-implementingscroll.md) никогда не поддерживается в полосе прокрутки напрямую.                                                                                                                     |
