# <a name="determine-the-installed-version-of-active-accessibility"></a>Определение установленной версии Active Accessibility

Чтобы узнать, какая версия Microsoft Active Accessibility (MSAA) установлена в системе, следует проверить версию файла Oleacc.dll в папке system32 на жестком диске. Следующая таблица сопоставляет версию Microsoft Active Accessibility с версией Oleacc.dll.

| Версия Active Accessibility     | Версия Oleacc.dll                                                                       |
|----------------------------------|------------------------------------------------------------------------------------------|
| 1.0                              | 4.01.0970.0                                                                              |
| 1.2                              | 4.02.2900.0                                                                              |
| 1,3                              | 4.02.3100.0                                                                              |
| 2.0                              | 5.0.0.0                                                                                  |
| 3,0 (Windows API автоматизации 3,0) | 7.0. x. y, где *x* — это номер сборки, а *y* — номер QFE (Быстрая разработка исправлений) |
