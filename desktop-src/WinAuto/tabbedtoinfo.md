# <a name="tabbedtoinfo"></a>таббедтоинфо

## <a name="text"></a>Текст

**Сообщение 1**

Создание дополнительного фокусного события

**Сообщение 2**

В начало

## <a name="type"></a>Тип

Informational

## <a name="description"></a>Описание

**Сообщение 1**

Несколько событий фокуса были запущены одним элементом при использовании стандартной навигации с помощью клавиатуры (TAB или SHIFT + TAB). Ложные события отбрасываются.

**Сообщение 2**

Фокус перемещается назад к начальному элементу в последовательности табуляции. Переход на этот элемент завершен, так как последовательность табуляции выглядит круглой.
