# <a name="uiaelementdoesnotparenttoroot"></a>уиаелементдоеснотпарентторут

## <a name="text"></a>Текст

Родительская цепочка элемента не переходит к корневому элементу

## <a name="type"></a>Тип

Предупреждение

## <a name="description"></a>Описание

Перемещение из заданного элемента в корень дерева пользовательского интерфейса разорвано.

Эта проблема может привести к непредсказуемым результатам для приложения, которое пытается переходить по дереву пользовательского интерфейса.

## <a name="possible-causes"></a>Возможные причины

Неверная реализация родительской дочерней иерархии дерева пользовательского интерфейса.
