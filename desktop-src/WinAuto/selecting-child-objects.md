# <a name="selecting-child-objects"></a>Выбор дочерних объектов

Клиенты вызывают метод [**IAccessible:: аккселект**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect) для изменения выделения или фокуса клавиатуры между дочерними элементами объекта. [Константы селфлаг](selflag.md) , указанные с помощью вызова, определяют выполняемую операцию.

Если метод [**IAccessible:: аккселект**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect) вызывается с флагом [**селфлаг \_ такефокус**](selflag.md) для дочернего объекта, имеющего **HWND**, флаг вступает в силу только в том случае, если родительский объект объекта имеет фокус.

## <a name="performing-complex-selection-operations"></a>Выполнение сложных операций выбора

Ниже описано, какие значения СЕЛФЛАГ необходимо указать при вызове метода [**IAccessible:: аккселект**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect) для выполнения сложных операций выбора.

**Имитация щелчка**

-   [**Селфлаг \_ такефокус**](selflag.md) \| [ **селфлаг \_ такеселектион**](selflag.md)

**Выбор целевого элемента путем имитации щелчка CTRL + щелчок**

-   [**Селфлаг \_ такефокус**](selflag.md) \| [ **селфлаг \_ аддселектион**](selflag.md)

**Отмена выбора целевого элемента путем имитации нажатия клавиш CTRL + щелчок**

-   [**Селфлаг \_ такефокус**](selflag.md) \| [ **селфлаг \_ ремовеселектион**](selflag.md)

**Имитация НАЖАТия SHIFT + щелчок**

-   [**Селфлаг \_ такефокус**](selflag.md) \| [ **селфлаг \_ екстендселектион**](selflag.md)

**Выбор диапазона объектов и помещение фокуса на последний объект**

1.  Укажите [**селфлаг \_ такефокус**](selflag.md) на начальном объекте, чтобы задать привязку выбора.
2.  Вызовите метод [**IAccessible:: аккселект**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect) еще раз и укажите [**селфлаг \_ екстендселектион**](selflag.md) \| [**селфлаг \_ такефокус**](selflag.md) на последнем объекте.

**Чтобы отменить выбор всех объектов**

1.  Укажите [**селфлаг \_ такеселектион**](selflag.md) для любого объекта. Этот флаг отменяет выбор всех выбранных объектов, кроме одного выбранного.
2.  Вызовите метод [**IAccessible:: аккселект**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect) еще раз и укажите [**селфлаг \_ ремовеселектион**](selflag.md) для оставшегося объекта.
