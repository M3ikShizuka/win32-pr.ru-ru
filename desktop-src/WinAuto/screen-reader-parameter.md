# <a name="screen-reader-parameter"></a>Параметр средства чтения с экрана

Параметр средства чтения с экрана указывает, должно ли приложение предоставлять текстовую информацию в ситуациях, когда оно в противном случае будет представлять информацию графически.

Этот параметр обычно задается специальными средствами, такими как средства чтения с экрана. Приложения используют флаги **SPI \_ Жетскринреадер** и **SPI \_ сетскринреадер** с функцией [**системпараметерсинфо**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa) для получения и задания параметра средства чтения с экрана.

> [!Note]  
> экранный диктор, средство чтения с экрана, включенное в Windows, не устанавливает флаги **spi \_ сетскринреадер** или **spi \_ жетскринреадер** .
