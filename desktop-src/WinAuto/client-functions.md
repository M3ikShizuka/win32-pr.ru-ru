# <a name="client-functions"></a>Функции клиента

В этом разделе содержатся сведения о клиентских функциях, используемых с Microsoft Active Accessibility.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                             | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|-----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**акцессиблечилдрен**](/windows/desktop/api/Oleacc/nf-oleacc-accessiblechildren)<br/>                       | Получает идентификатор дочернего элемента или [**IDispatch**](idispatch-interface.md) каждого потомка в доступном объекте контейнера.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**акцессиблеобжектфромевент**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfromevent)<br/>         | Извлекает адрес интерфейса [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) для объекта, создавшего событие, которое в настоящее время обрабатывается функцией обработчика событий клиента.<br/>                                                                                                                                                                                                                                                                                                                                                                             |
| [**акцессиблеобжектфромпоинт**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfrompoint)<br/>         | Извлекает адрес указателя интерфейса [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) для объекта, отображаемого в указанной точке экрана.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**акцессиблеобжектфромвиндов**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfromwindow)<br/>       | Извлекает адрес указанного интерфейса для объекта, связанного с указанным окном.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**жетолеаккверсионинфо**](/windows/desktop/api/Oleacc/nf-oleacc-getoleaccversioninfo)<br/>                   | Возвращает номер версии и номер сборки Oleacc.dll файла Microsoft Active Accessibility.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**жетпроцесшандлефромхвнд**](getprocesshandlefromhwnd.md)<br/>           | Извлекает маркер процесса из маркера окна.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**жетролетекст**](/windows/desktop/api/Oleacc/nf-oleacc-getroletexta)<br/>                                     | Извлекает локализованную строку, описывающую роль объекта для указанного значения роли.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**жетстатетекст**](/windows/desktop/api/Oleacc/nf-oleacc-getstatetexta)<br/>                                   | Извлекает локализованную строку, описывающую состояние объекта для одного предопределенного флага битового состояния. Так как значения состояния являются сочетанием одного или нескольких битовых флагов, клиенты вызывают эту функцию более одного раза для получения всех строк состояния.<br/>                                                                                                                                                                                                                                                                                                                      |
| [**сетвиневенсук**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook)<br/>                             | Задает функцию обработчика событий для диапазона событий.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**унхуквиневент**](/windows/desktop/api/Winuser/nf-winuser-unhookwinevent)<br/>                               | Удаляет функцию обработчика событий, созданную предыдущим вызовом метода [**сетвиневенсук**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**виндовфромакцессиблеобжект**](/windows/desktop/api/Oleacc/nf-oleacc-windowfromaccessibleobject)<br/>       | Извлекает маркер окна, соответствующий конкретному экземпляру интерфейса [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [*Функция обратного вызова Виневентпрок*](/windows/desktop/api/Winuser/nc-winuser-wineventproc)<br/> | Определяемая приложением функция обратного вызова (или обработчика), которая вызывается системой в ответ на события, формируемые доступным объектом. Функция-обработчик обрабатывает уведомления о событиях по мере необходимости. Клиенты устанавливают функцию-обработчик и запрашивают определенные типы уведомлений о событиях путем вызова [**сетвиневенсук**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook).<br/> Тип [**виневентпрок**](/previous-versions/windows/desktop/legacy/dd373882(v=vs.85)) определяет указатель на эту функцию обратного вызова. [*Виневентпрок*](/windows/desktop/api/winuser/nc-winuser-wineventproc) — это заполнитель для имени определяемой приложением функции.<br/> |
