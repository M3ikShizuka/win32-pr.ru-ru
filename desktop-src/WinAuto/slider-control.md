# <a name="slider-control-msaa-ui-element-reference"></a>Элемент управления Slider (Справочник по элементам пользовательского интерфейса MSAA)

> [!Note]  
> В этом разделе описываются **управляющие** объекты, предназначенные для ссылок на элементы ПОЛЬЗОВАТЕЛЬСКОГО интерфейса MSAA. Создание **управляющих объектов-ползунков** в различных ПЛАТФОРМАХ пользовательского интерфейса не описывается здесь. См. справочную документацию по API для платформы пользовательского интерфейса, которую вы используете.

 

Элемент управления "ползунок", который также называется элементом управления "линейка", позволяет пользователю выбирать из диапазона значений, перемещая ползунок. Элементы управления громкостью в операционной системе Windows представляют собой ползунки.

Имя класса Window для элемента управления Slider имеет класс TRACKBAR \_ , который определен как "мсктлс \_ TrackBar" в коммктрл. h.

Содержимое свойств [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) зависит от того, является ли ползунок вертикальным или горизонтальным и какая из следующих частей элемента управления Slider запрашивается клиентом:

-   Окно Slider
-   Бегунок
-   Затененная область выше (или до
-   Затененная область ниже (или справа от ползунка) бегунка

## <a name="iaccessible-methods"></a>Методы IAccessible

Элемент управления "ползунок" поддерживает следующие методы [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) :

-   [**акчиттест**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)
-   [**акклокатион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)
-   [**аккнавигате**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate)
-   [**аккселект**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect)

## <a name="iaccessible-properties"></a>Свойства IAccessible

Элемент управления "ползунок" поддерживает следующие свойства [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) :

-   [**получить \_ аккчилд**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accchild)
-   [**получить \_ аккчилдкаунт**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accchildcount)
-   [**получить \_ аккдескриптион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accdescription)
-   [**получить \_ акчелп**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_acchelp)
-   [**получить \_ акчелптопик**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_acchelptopic)
-   [**Get \_ акккэйбоардшорткут**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_acckeyboardshortcut)— свойство **кэйбоардшорткут** является клавишей доступа ползунка, которая представляет собой подчеркнутый символ в тексте метки для ползунка. Возвращаемая строка содержит символ клавиши доступа, добавленный к строке "Alt +".
-   [**Get \_ аккнаме**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accname)— свойство **Name** зависит от части запрашиваемого ползунка.

    Части вертикального ползунка имеют следующие имена:

    

    | Элемент Slider                    | Имя                                |
    |--------------------------------|-------------------------------------|
    | Окно Slider                  | Элемент управления "статический текст", используемый в качестве метки |
    | Бегунок                   | Разместить                          |
    | Затененный участок над бегунком ползунка | "Стр."                           |
    | Затененная область под бегунком ползунка | "На страницу вниз"                         |

    

     

    Части горизонтального ползунка имеют следующие имена:

    

    | Элемент Slider                              | Имя                                |

    | Бегунок                             | Разместить                          |
    | Затененная область слева от бегунка  | "Страница слева"                         |
    | Затененная область справа от бегунка | "Страница вправо"                        |

    

     

-   [**Get \_ Аккпарент**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accparent)— **родительское** свойство кнопок со стрелками, бегунок прокрутки и затененная область с любой стороны бегунка — это ползунок. Свойство **Parent** окна Slider является окном ( [**\_ \_ окно системы ролей**](object-roles.md) ), которое окружает элемент управления и имеет **то же имя** свойства и класса окна.
-   [**Get \_ аккроле**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accrole)— свойство **Role** зависит от части запрашиваемого ползунка. 

    | Элемент Slider                                     | [Роль](object-roles.md)                                                |

    | Окно Slider                                   | [**\_ \_ ползунок системы роли**](object-roles.md)         |
    | Бегунок                                    | [**\_индикатор системы \_ роли**](object-roles.md)   |
    | Затененные области с любой стороны бегунка | [**\_кнопка системы \_ роли**](object-roles.md) |

    

     

-   [**Get \_ аккстате**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accstate)—[значения](object-state-constants.md) для свойства **State** зависят от части запрашиваемого ползунка. 

    | Элемент Slider                                     | Возможные значения состояния                                                                                                                                                                                                                                                                                                                                                                                                           |

    | Окно Slider                                   | [**Состояние \_ Системная \_ невидимая**](object-state-constants.md) система \| [**состояний \_ \_ недоступна**](object-state-constants.md) система состояний с \| [**\_ \_**](object-state-constants.md) \| [**\_ \_ фокусом**](object-state-constants.md) системы состояния, \| [**\_ \_ нормальное состояние системы**](object-state-constants.md) |

    | Затененные области с любой стороны бегунка | Ноль (0), это означает, что объект является видимым [**, \_ или \_ система состояния "**](object-state-constants.md) \| [**\_ \_ недоступная**](object-state-constants.md) система состояния" не имеет \| [**\_ \_ нормальной системы состояния**](object-state-constants.md)                                                                                                                       |

    

     

-   [**Get \_ акквалуе**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accvalue)— свойство **value** для ползунка указывает положение бегунка и представляет собой строку, содержащую целое число от 0 до "100".
