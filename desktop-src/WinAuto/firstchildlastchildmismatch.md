# <a name="firstchildlastchildmismatch"></a>фирстчилдластчилдмисматч

## <a name="text"></a>Текст

При переходе по вызову {0} из протестированного узла и последующем многократном вызове {1} мы завершили следующий дочерний элемент: {2} . Это не соответствует результату вызова {3} на проверенном узле, который вернул: {4} . Вместо этого дочерний элемент должен сообщить как его родительский узел тестирования.

## <a name="type"></a>Тип

Ошибка

## <a name="description"></a>Описание

Возникла проблема при переходе между дочерними элементами элемента. 1. Неправильная или недопустимая реализация модели автоматизации пользовательского интерфейса.

Эта проблема может вызвать проблемы с навигацией для автоматизированных средств, так как обходные элементы могут быть неустойчивыми и непредсказуемыми.

## <a name="possible-causes"></a>Возможные причины

Неправильная или недопустимая реализация модели автоматизации пользовательского интерфейса.
