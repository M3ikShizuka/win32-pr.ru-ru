# <a name="helptopic-property"></a>Хелптопик, свойство

Свойство **хелптопик** получает сведения, используемые с функцией Microsoft Win32 [WinHelp](/windows/win32/api/winuser/nf-winuser-winhelpa) , для вывода раздела справки, связанного с объектом. Это свойство поддерживается не всеми объектами.

Свойство **хелптопик** извлекается путем вызова метода [**IAccessible:: Get \_ акчелптопик**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_acchelptopic).
