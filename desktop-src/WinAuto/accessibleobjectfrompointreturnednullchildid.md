# <a name="accessibleobjectfrompointreturnednullchildid"></a>акцессиблеобжектфромпоинтретурнеднуллчилдид

## <a name="text"></a>Текст

Акцессиблеобжектфромпоинт ( {0} , {1} ) вернул значение NULL чилдид

## <a name="type"></a>Тип

Error

## <a name="description"></a>Описание

Чилдид интерфейса [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) объекта, полученного для заданных координат, имеет значение null.

## <a name="possible-causes"></a>Возможные причины

Взаимодействие с пользователем во время проверки, например перемещение фокуса на нецелевой HWND, мешает процессу проверки.
