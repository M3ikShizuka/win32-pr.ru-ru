# <a name="visual-basic-sample-programs"></a>Visual Basic примеров программ

В состав бюллетеня *Advanced microsoft Visual Basic 6,0, 2-го выпуска* (Мандельброта Set International, Ltd.) входит глава, в которой обсуждается Microsoft Active Accessibility. В этой главе приведен пример кода, демонстрирующий использование ключевых элементов Microsoft Active Accessibility API. (Microsoft Press, ISBN 1-57231-893-7) (Эта книга может быть недоступна в некоторых языках и странах.)
