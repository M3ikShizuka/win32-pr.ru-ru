# <a name="hot-key-control-msaa-ui-element-reference"></a>Элемент управления горячими ключами (Справочник по элементам пользовательского интерфейса MSAA)

Элементы управления горячими ключами позволяют пользователям вводить сочетания клавиш, используемые в качестве горячих клавиш, что позволяет быстро выполнять какие-либо действия. Элемент управления "горячие клавиши" отображает вводимые пользователем нажатия клавиш и гарантирует, что пользователь выбрал допустимое сочетание клавиш.

Имя класса Window для элемента управления "горячий ключ" — это сочетание \_ класса, которое определено как "мсктлс \_ hotkey32" в коммктрл. h.

## <a name="iaccessible-methods"></a>Методы IAccessible

Элементы управления горячими ключами поддерживают следующие методы [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) :

-   [**акчиттест**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)
-   [**акклокатион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)
-   [**аккнавигате**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate)
-   [**аккселект**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect)
