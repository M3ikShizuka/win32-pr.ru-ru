# <a name="event-constants-winuserh"></a>Константы событий (Winuser. h)

В этом разделе описываются события, создаваемые операционной системой и серверными приложениями. Константы перечислены в алфавитном порядке.

Перед использованием этих событий клиентские приложения должны использовать [доступное наблюдатель событий](accessible-event-watcher.md) , чтобы убедиться, что эти события используются элементами пользовательского интерфейса.

Дополнительные сведения о событиях в целом см. в разделе [что такое виневентс?](what-are-winevents.md) и [события уровня системы и объекта](system-level-and-object-level-events.md). Дополнительные сведения о событиях, отправляемых системой, см. [в приложении A: Поддерживаемые ссылки на элементы пользовательского интерфейса](appendix-a--supported-user-interface-elements-reference.md).
