# <a name="in-context-and-out-of-context-hook-functions"></a>In-Context и функции-обработчики вне контекста

При регистрации функции-обработчика с помощью [**сетвиневенсук**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook)клиенты указывают, находится ли функция обработчика *в контексте* или *вне контекста*. В этих терминах описывается расположение памяти функции-обработчика относительно адресного пространства сервера.
