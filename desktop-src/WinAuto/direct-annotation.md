# <a name="direct-annotation"></a>Непосредственная Аннотация

В этом разделе содержатся сведения об использовании прямых заметок.

## <a name="summary"></a>Сводка

Непосредственная Аннотация — это простейшая форма динамической аннотации. Он используется для переопределения значения одного или ограниченного числа свойств элемента, доступного для доступа. Он наиболее удобен для доступных элементов, свойства заметок которых не зависят от состояния элемента управления и не нуждаются в дополнительной поддержке, предоставляемой [заметкой значений](value-map-annotation.md) и [заметками сервера](server-annotation.md).

## <a name="when-to-use-this-technique"></a>Когда следует использовать этот метод

Используйте этот метод, если требуется переопределить значение одного или ограниченного числа свойств для элемента, доступного для доступа.

Дополнительные сведения см. [в разделе Использование прямых заметок](using-direct-annotation.md).
