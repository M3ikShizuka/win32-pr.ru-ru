# <a name="aria-grid-structure-error"></a>Ошибка структуры сетки ARIA

## <a name="text"></a>Текст

Элемент с ролью **сетки** не имеет соответствующей структуры сетки или доступной разметки.

## <a name="type"></a>Тип

Error

## <a name="description"></a>Описание

Эта ошибка применяется к пользовательским элементам, для которых атрибут **Role** имеет значение Grid. Он не применяется к тегам таблицы HTML, которые имеют неявную **роль** "Grid".

У элемента, у которого атрибут **Role** имеет значение "Grid", должна быть структура, определенная ролью сетки "веб-приложения специальных возможностей" (ожидать-ARIA), включая доступное имя для сетки и вложенные элементы заголовка.

Чтобы устранить эту ошибку, проверьте реализацию, чтобы убедиться, что она соответствует спецификации ожидать-ARIA. В частности, убедитесь, что структура соответствует следующим правилам.

-   **Сетка** содержит элементы **Row** или **группы строк** .
-   **группы строк** содержит элементы **Row** .
-   элементы **Row** содержат элементы **колумнхеадер** или **гридцелл** или **ровхеадер** .

Доступное имя должно быть определено для элементов **Grid**, **колумнхеадер**, **гридцелл** и **ровхеадер** с помощью одного из следующих атрибутов.

-   атрибут [**ARIA-лабелледби**](https://developer.mozilla.org/docs/Web/Accessibility/ARIA) для ссылки на элемент, содержащий текст.
-   атрибут [**ARIA-Label**](https://developer.mozilla.org/docs/Web/Accessibility/ARIA) для непосредственного задания имени для доступа.
-   [**заголовок**](https://developer.mozilla.org/docs/Web/HTML/Global_attributes/title) для создания подсказки, которая в то же время используется в качестве имени.
