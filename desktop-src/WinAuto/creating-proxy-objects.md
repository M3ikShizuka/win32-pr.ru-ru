# <a name="creating-proxy-objects"></a>Создание прокси-объектов

В дополнение к проектированию классов, реализующих свойства и методы [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) , разработчикам сервера также может потребоваться разработать *прокси-* объекты, отслеживающие жизненный интервал доступных объектов. Если доступный объект в приложении доступен в течение всего времени выполнения приложения, создание прокси-объекта не требуется. Если можно уничтожить доступный объект во время работы приложения, необходимо создать прокси-объект.
