# <a name="droptarget-control-pattern"></a>Шаблон элемента управления Дроптаржет

Содержит правила и соглашения по реализации шаблона элемента управления **дроптаржет** с помощью [**идроптаржетпровидер**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-idroptargetprovider), включая сведения о свойствах и методах. Шаблон элемента управления **дроптаржет** используется для поддержки элементов управления, которые могут быть целевым объектом операции перетаскивания.

## <a name="implementation-guidelines-and-conventions"></a>Правила и соглашения реализации

При реализации шаблона элемента управления **дроптаржет** используйте следующие правила и соглашения.

-   Шаблон **дроптаржет** должен поддерживаться во время выполнения операции перетаскивания. Она может поддерживаться даже в том случае, если операция перетаскивания не выполняется.
-   Свойство [**идроптаржетпровидер::D роптаржетеффект**](/windows/desktop/api/UIAutomationCore/nf-uiautomationcore-idroptargetprovider-get_droptargeteffect) является обязательным.
-   Свойство [**идроптаржетпровидер::D роптаржетеффектс**](/windows/desktop/api/UIAutomationCore/nf-uiautomationcore-idroptargetprovider-get_droptargeteffects) является обязательным, если для целевого объекта существует несколько возможных эффектов перетаскивания.
-   Элемент должен вызывать события изменения свойства для свойств **дроптаржетеффект** ([**UIA \_ дроптаржетдроптаржетеффектпропертид**](uiauto-control-pattern-propids.md)) и **дроптаржетеффектс** ([**UIA \_ дроптаржетдроптаржетеффектспропертид**](uiauto-control-pattern-propids.md)) при их изменении.

## <a name="required-members-for-idroptargetprovider"></a>Обязательные члены для **идроптаржетпровидер**

Для реализации интерфейса [**идроптаржетпровидер**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idroptargetprovider) требуются следующие свойства и методы.



| Обязательные члены                                                                              | Тип члена | Примечания                                                                    |
|-----------------------------------------------------------------------------------------------|-------------|--------------------------------------------------------------------------|
| [**дроптаржетеффект**](/windows/desktop/api/UIAutomationCore/nf-uiautomationcore-idroptargetprovider-get_droptargeteffect)                       | Свойство    | Нет                                                                     |
| [**дроптаржетеффектс**](/windows/desktop/api/UIAutomationCore/nf-uiautomationcore-idroptargetprovider-get_droptargeteffects)                     | Свойство    | Требуется, если цель перетаскивания поддерживает несколько возможных эффектов перетаскивания. |
| [**UIA \_ дроптаржет \_ дражентеревентид**](uiauto-event-ids.md) | Событие       | Нет                                                                     |
| [**UIA \_ дроптаржет \_ драглеавивентид**](uiauto-event-ids.md) | Событие       | Нет                                                                     |
| [**UIA \_ дроптаржет \_ дроппедевентид**](uiauto-event-ids.md)     | Событие       | Нет                                                                     |
