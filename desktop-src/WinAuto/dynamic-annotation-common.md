# <a name="dynamic-annotation"></a>Динамическая Аннотация

Динамическая Аннотация ранее была доступна только для серверов и клиентов Microsoft Active Accessibility. Теперь этот прием доступен для всех клиентов автоматизации пользовательского интерфейса, а также для некоторых реализаций сервера и модели автоматизации пользовательского интерфейса MSAA. Кроме того, заметки, вносимые в пользовательские элементы управления с помощью динамической аннотации (все три метода), доступны клиентам автоматизации пользовательского интерфейса.

Дополнительные сведения см. в разделе [Динамическая Аннотация API](dynamic-annotation-api.md).
