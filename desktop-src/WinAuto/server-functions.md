# <a name="server-functions-windows-accessibility-features"></a>функции сервера (Windows специальные возможности)

В этом разделе содержатся сведения о функциях сервера, используемых в Microsoft Active Accessibility.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                     | Описание                                                                                                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**аккнотифитаучинтерактион**](/windows/desktop/api/Oleacc/nf-oleacc-accnotifytouchinteraction)<br/> | позволяет приложению с поддержкой специальных возможностей (AT) уведомлять систему о взаимодействии пользовательского интерфейса через API-интерфейс автоматизации Windows (например, в Microsoft UI automation) в результате сенсорного жеста от пользователя. Это позволяет вспомогательной технологии уведомлять о целевом приложении и системе, с которой пользователь взаимодействует с Touch.<br/> |
| [**акксетруннингутилитистате**](/windows/desktop/api/Oleacc/nf-oleacc-accsetrunningutilitystate)<br/> | Устанавливает системные значения, указывающие, влияет ли текущее состояние приложения на функциональные возможности (AT) на функциональность, обычно предоставляемую системой. <br/>                                                                                                                                                                                 |
| [**креатестдакцессиблеобжект**](/windows/desktop/api/Oleacc/nf-oleacc-createstdaccessibleobject)<br/> | Создает доступный объект с методами и свойствами указанного типа элемента пользовательского интерфейса, предоставляемого системой.<br/>                                                                                                                                                                                                                      |
| [**креатестдакцессиблепрокси**](/windows/desktop/api/Oleacc/nf-oleacc-createstdaccessibleproxya)<br/>   | Создает объект со специальными возможностями, который имеет свойства и методы указанного класса, предоставляемого системным элементом пользовательского интерфейса.<br/>                                                                                                                                                                                                                 |
| [**Функции lresultfromobject**](/windows/desktop/api/Oleacc/nf-oleacc-lresultfromobject)<br/>                 | Возвращает ссылку, аналогичную маркеру, с указанным объектом. Серверы возвращают эту ссылку при обработке [**WM \_ GetObject**](wm-getobject.md).<br/>                                                                                                                                                                                              |
| [**обжектфромлресулт**](/windows/desktop/api/Oleacc/nf-oleacc-objectfromlresult)<br/>                 | Извлекает запрошенный указатель интерфейса для доступного объекта на основе ранее созданной ссылки на объект.<br/> Эта функция предназначена для внутреннего использования корпорацией Майкрософт Active Accessibility и описана только в ознакомительных целях. Эта функция не должна вызывать ни клиенты, ни серверы.<br/>                               |
| [**исвиневенсукинсталлед**](/windows/desktop/api/Winuser/nf-winuser-iswineventhookinstalled)<br/>     | Определяет наличие установленного обработчика WinEvent, который может получать уведомления о указанном событии.<br/>                                                                                                                                                                                                                                                |
| [**нотифивиневент**](/windows/desktop/api/Winuser/nf-winuser-notifywinevent)<br/>                       | Сообщает системе, что произошло предопределенное событие. Если все клиентские приложения зарегистрировали функцию-обработчик для события, система вызывает функцию-ловушку клиента.<br/>                                                                                                                                                                        |
