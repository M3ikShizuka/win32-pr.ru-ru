# <a name="toomanychildren"></a>туманичилдрен

## <a name="text"></a>Текст

Остановлен Поиск дополнительных элементов того же уровня после поиска {0} элементов того же уровня. Возможно, неверное дерево

## <a name="type"></a>Тип

Предупреждение

## <a name="description"></a>Описание

Дерево элементов может быть циклическим и неограниченным глубиной дерева.

Эта проблема может вызвать проблемы с навигацией для автоматизированных инструментов, так как они вводят циклическую ссылку или бесконечный цикл.

## <a name="possible-causes"></a>Возможные причины

-   Проект приложения или элемента управления может быть слишком сложным. Это предупреждение может быть получено для элементов управления "представление списка", имеющих большое количество элементов. В таких случаях это предупреждение обычно можно игнорировать.
-   Неправильная или недопустимая реализация MSAA.
