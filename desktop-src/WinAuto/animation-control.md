# <a name="animation-control-msaa-ui-element-reference"></a>Элемент управления анимации (Справочник по элементам пользовательского интерфейса MSAA)

Элементы управления анимации автоматически отображают ролик AVI. Элементы управления анимацией обычно отображаются при копировании файлов или при выполнении какой-либо другой длительной задачи.

Имя класса Window для элемента управления Animation — анимация \_ Class, которая определена как «SysAnimate32» в коммктрл. h.

## <a name="iaccessible-methods"></a>Методы IAccessible

Элементы управления анимации поддерживают следующие методы [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) :

-   [**акчиттест**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)
-   [**акклокатион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)
-   [**аккнавигате**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate)
-   [**аккселект**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect)

## <a name="iaccessible-properties"></a>Свойства IAccessible

Элементы управления анимации поддерживают следующие свойства [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) :
