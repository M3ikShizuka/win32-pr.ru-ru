# <a name="object-roles"></a>Роли объектов

В этом разделе описываются константные значения, используемые для описания ролей различных объектов пользовательского интерфейса в приложении. Константы роли определяются в файле Олеакк. h.

Следует использовать только перечисленные константы ролей, перечисленные здесь; не добавляйте настраиваемые роли или роли, которые не являются предопределенными.

Прежде чем использовать эти роли объектов, разработчики клиентских приложений должны использовать средство [проверки](inspect-objects.md) , чтобы убедиться, что роли объектов используются элементами пользовательского интерфейса.

Чтобы получить роль объекта, клиенты вызывают метод [**IAccessible:: Get \_ аккроле**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accrole) , который должен возвращать одно из значений, описанных в таблице. Чтобы получить локализованную строку, описывающую роль объекта, клиенты вызывают функцию [**жетролетекст**](/windows/desktop/api/Oleacc/nf-oleacc-getroletexta) со значением Role. Локализованные строки для значений ролей находятся в файле oleaccrc.dll.
