# <a name="list-box-msaa-ui-element-reference"></a>List Box (Справочник по элементам пользовательского интерфейса MSAA)

> [!Note]  
> В этом разделе описываются объекты **списков** для ссылки на элемент ПОЛЬЗОВАТЕЛЬСКОГО интерфейса MSAA. Создание объектов **списка** в различных ПЛАТФОРМАХ пользовательского интерфейса не описывается здесь. См. справочную документацию по API для платформы пользовательского интерфейса, которую вы используете.

 

В списках отображаются списки, из которых пользователь может выбрать один или несколько элементов. Поле со списком создается с вертикальной полосой прокрутки, горизонтальной панелью управления или обоими.

Имя класса окна для списка — "LISTBOX".

## <a name="iaccessible-methods"></a>Методы IAccessible

Списки поддерживают следующие методы [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) :



| Метод                                                                    | Комментарии                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**аккдодефаултактион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) | Сам объект списка не имеет свойства **DefaultAction** ; Однако [**аккдодефаултактион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) дважды щелкает элементы списка.                                                                                    |
| [**акчиттест**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)                 | Метод [**акчиттест**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest) возвращает нуль, если проверенная точка находится в списке, но не на элементе в списке. Если проверенная точка находится на элементе в списке, **акчиттест** возвращает индекс элемента списка. |
| [**акклокатион**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)               |                                                                                                                                                                                                                                                                            |
| [**аккнавигате**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate)               |                                                                                                                                                                                                                                                                            |
| [**аккселект**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect)                   |                                                                                                                                                                                                                                                                            |
