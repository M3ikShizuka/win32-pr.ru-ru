# <a name="directwrite-functions"></a>Функции DirectWrite

DirectWrite предоставляет следующую функцию. Все остальные операции выполняются с помощью объектов интерфейса DirectWrite, созданных объектом [**идвритефактори**](/windows/win32/api/dwrite/nn-dwrite-idwritefactory) , созданным с помощью функции ниже.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                         | Описание                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**двритекреатефактори**](/windows/win32/api/dwrite/nf-dwrite-dwritecreatefactory)<br/> | Создает объект фабрики DirectWrite, который используется для последующего создания отдельных объектов DirectWrite.<br/> |
