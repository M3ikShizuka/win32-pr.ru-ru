# <a name="directwrite-macros"></a>DirectWrite макросы

DirectWrite определяет следующие макросы.

## <a name="in-this-section"></a>В этом разделе

| Раздел | Описание |
|-|-|
| [**DWRITE_MAKE_FONT_AXIS_TAG**](/windows/win32/api/dwrite_3/nf-dwrite_3-dwrite_make_font_axis_tag) | Создает тег OpenType для оси шрифта. |
| [**ДВРИТЕ \_ создать \_ \_ тег OPENTYPE**](/windows/win32/api/dwrite/nf-dwrite-dwrite_make_opentype_tag) | Создает тег OpenType как 32-разрядное целое число, таким образом, что первый символ в теге является наименьшим байтом (по крайней мере значительным для архитектур с прямым порядком байтов), который можно использовать для сравнения с тегами в файле шрифта. Этот макрос совместим с [**\_ \_ \_ тегом функции шрифта дврите**](/windows/win32/api/dwrite/ne-dwrite-dwrite_font_feature_tag). |
