# <a name="idwritelocalfontfileloader-interface"></a>Интерфейс Идврителокалфонтфилелоадер

Встроенная реализация интерфейса [**идвритефонтфилелоадер**](/windows/win32/api/dwrite/nn-dwrite-idwritefontfileloader) , которая работает с локальными файлами шрифтов и предоставляет сведения о файле локального шрифта из ключа ссылки на файл шрифта. Ссылки на файлы шрифтов, созданные с помощью [**креатефонтфилереференце**](/windows/win32/api/dwrite/nf-dwrite-idwritefactory-createfontfilereference) , используют этот загрузчик файлов шрифтов.

## <a name="members"></a>Элементы

Интерфейс **идврителокалфонтфилелоадер** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Идврителокалфонтфилелоадер** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **идврителокалфонтфилелоадер** содержит следующие методы.



| Метод                                                                                  | Описание                                                                               |
|:----------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------|
| [**жетфилепасфромкэй**](idwritelocalfontfileloader-getfilepathfromkey.md)             | Получает абсолютный путь к файлу шрифта из ключа ссылки на файл шрифта.<br/>          |
| [**жетфилепасленгсфромкэй**](idwritelocalfontfileloader-getfilepathlengthfromkey.md) | Получает длину абсолютного пути к файлу из ключа ссылки на файл шрифта.<br/> |
| [**жетластвритетимефромкэй**](idwritelocalfontfileloader-getlastwritetimefromkey.md)   | Получает время последней записи файла из ключа ссылки на файл шрифта.<br/>      |
