# <a name="idwritefontsetbuilderaddfontfacereference-methods"></a>Методы Идвритефонтсетбуилдер:: Аддфонтфацереференце

Добавляет ссылку на шрифт в создаваемый набор.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                                                           | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|:-------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Аддфонтфацереференце (Идвритефонтфацереференце \* )**](/windows/win32/api/dwrite_3/nf-dwrite_3-idwritefontsetbuilder-addfontfacereference(idwritefontfacereference))                                         | Добавляет ссылку на шрифт в создаваемый набор. Необходимые метаданные будут автоматически извлечены из шрифта при вызове Креатефонтсет.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| [**Аддфонтфацереференце (Идвритефонтфацереференце \* , свойство const \_ дврите \_ Font \* , UINT32)**](/windows/win32/api/dwrite_3/nf-dwrite_3-idwritefontsetbuilder-addfontfacereference(idwritefontfacereference_dwrite_font_propertyconst_uint32)) | Добавляет ссылку на шрифт в создаваемый набор. Вызывающий объект предоставляет достаточно информации для поиска, избегая необходимости открывать потенциально нелокальный шрифт. Все свойства, не предоставляемые вызывающим объектом, будут отсутствовать, и эти свойства будут недоступны в качестве фильтров в Жетматчингфонтс. GetPropertyValues для отсутствующих свойств вернет пустой список строк. Передаваемые свойства обычно должны соответствовать фактическому содержимому шрифта, но они не должны быть. Например, можно создать псевдоним для шрифта, используя другое имя или уникальный идентификатор, или задать пользовательские теги, отсутствующие в фактическом шрифте.<br/> |
