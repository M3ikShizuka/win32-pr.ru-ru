# <a name="dwrite_rendering_mode-enumerations"></a>\_Перечисления в режиме рендеринга дврите \_

Начиная с Windows 8, перечисление **\_ \_ режима рендеринга дврите** добавило новые значения перечисления и нерекомендуемые другие.

Начиная с Windows 8, перечисление [**\_ \_ \_ режима сглаживания текста дврите**](/windows/win32/api/Dwrite_1/ne-dwrite_1-dwrite_text_antialias_mode) определяет, визуализируется ли текст с помощью технологии ClearType. В результате все режимы рендеринга ClearType в перечислении **дврите \_ \_ режима рендеринга** являются устаревшими. Эти значения перечислений теперь сопоставляются с новыми режимами рендеринга.

В таблице ниже показаны старые значения перечисления и новые значения, с которыми они сопоставляются. Описание новых значений см. в разделе [**дврите \_ Rendering \_ mode**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode).



| Старый режим                                                                                | Новый режим                                                                                |
|-----------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**классический интерфейс ДВРИТЕ в \_ режиме рендеринга \_ \_ CLEARTYPE \_ \_**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode)       | [**\_классический режим отрисовки дврите \_ \_ GDI \_**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode)                  |
| [**ДВРИТЕ в \_ режиме рендеринга, \_ \_ \_ \_ естественный GDI**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode)       | [**ДВРИТЕ \_ режима рендеринга \_ \_ GDI \_ Natural**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode)                  |
| [**ДВРИТЕ \_ режим рендеринга \_ \_ CLEARTYPE — \_ естественный**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode)            | [**ДВРИТЕ \_ режим рендеринга \_ \_ CLEARTYPE — \_ естественный**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode)            |
| [**ДВРИТЕ \_ режим рендеринга \_ \_ CLEARTYPE \_ Natural \_ симметрично**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode) | [**ДВРИТЕ \_ режим рендеринга \_ \_ CLEARTYPE \_ Natural \_ симметрично**](/windows/win32/api/dwrite/ne-dwrite-dwrite_rendering_mode) |
