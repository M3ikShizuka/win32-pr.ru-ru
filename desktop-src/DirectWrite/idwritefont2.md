# <a name="idwritefont2-interface"></a>Интерфейс IDWriteFont2

Представляет физический шрифт в коллекции шрифтов.

Этот интерфейс добавляет возможность проверки того, является ли путь визуализации цвета потенциально необходимым.

## <a name="members"></a>Элементы

Интерфейс **IDWriteFont2** наследует от [**IDWriteFont1**](/windows/win32/api/dwrite_1/nn-dwrite_1-idwritefont1). **IDWriteFont2** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **IDWriteFont2** содержит следующие методы.



| Метод                                          | Описание                                                                        |
|:------------------------------------------------|:-----------------------------------------------------------------------------------|
| [**исколорфонт**](idwritefont2-iscolorfont.md) | Позволяет определить, является ли путь визуализации цвета потенциально необходимым.<br/> |
