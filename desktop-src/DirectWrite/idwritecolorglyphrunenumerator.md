# <a name="idwritecolorglyphrunenumerator-interface"></a>Интерфейс Идвритеколорглифруненумератор

Этот интерфейс позволяет приложению выполнять перечисление по цвету глифа цвета. Перечислитель перечисляет слои в обратном порядке для соответствующего слоя.

## <a name="members"></a>Элементы

Интерфейс **идвритеколорглифруненумератор** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Идвритеколорглифруненумератор** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **идвритеколорглифруненумератор** содержит следующие методы.



| Метод                                                                | Описание                                                 |
|:----------------------------------------------------------------------|:------------------------------------------------------------|
| [**жеткуррентрун**](/windows/win32/api/dwrite_2/nf-dwrite_2-idwritecolorglyphrunenumerator-getcurrentrun) | Возвращает текущее выполнение глифа в перечислителе.<br/> |
| [**MoveNext**](idwritecolorglyphrunenumerator-movenext.md)           | Переход к следующему глифу выполняется в перечислителе.<br/>    |
