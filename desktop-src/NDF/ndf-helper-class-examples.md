# <a name="ndf-helper-class-extension-examples"></a>Примеры расширения вспомогательного класса NDF

Эти примеры применяются только в том случае, если разработчики считают, что необходимо расширить функциональные возможности вспомогательного класса Microsoft NDF, который объявлен как расширяемый. Как правило, это необязательно, но в некоторых случаях уникальные диагностические возможности представляют себя в связи с определенными требованиями к оборудованию или программному обеспечению компонента.

Два приведенных ниже примера связаны друг с другом, прежде чем анализировать вторую. Первый предоставляет реализацию [расширения вспомогательного класса ndf](ndf-helper-class-example.md) с помощью класса с именем симплехелперфилекласс.

Второе, [вспомогательное расширение класса ndf с](ndf-helper-class-example-with-handoff.md)передачей, предоставляет пример расширения, которое не выполняет определенные диагностические задачи, но создает низкую гипотезу о работоспособности для симплехелперфилекласс в первом примере.
