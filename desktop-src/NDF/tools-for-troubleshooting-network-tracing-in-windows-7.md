# <a name="tools-for-troubleshooting-using-network-tracing-in-windows-7"></a>средства для устранения неполадок с помощью трассировки сети в Windows 7

Существует ряд средств, которые можно использовать для включения и управления трассировкой сети и просмотра результирующих данных. Некоторые из средств, которые могут оказаться полезными, Netsh.exe, сетевой монитор, Просмотр событий, **netsh trace Convert**, Tracerpt.exe и Xperf.exe.

В этом разделе мы рассмотрим, как использовать Netsh.exe и сетевой монитор, чтобы помочь в устранении неполадок трассировки сети.
