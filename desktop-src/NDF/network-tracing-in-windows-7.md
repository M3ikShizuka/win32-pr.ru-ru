# <a name="network-tracing-in-windows-7"></a>трассировка сети в Windows 7

По мере роста сложности сетевого стека часто бывает трудно быстро отслеживать и диагностировать проблемы в стеке. Windows 7 разворачивается в инфраструктуре диагностики сети (NDF), чтобы обеспечить быстрый способ устранения проблем с сетевым подключением, включив сбор всех необходимых данных за один шаг и используя [трассировку событий для Windows (ETW)](../etw/event-tracing-portal.md) для записи сетевых событий & пакетов в одном файле.

Связанные события и пакеты группируются для конкретного действия, например для обзора веб-сайта, по различным компонентам в сетевом стеке. Выходными данными этого процесса является файл журнала трассировки событий (ETL). Разрешая всем событиям, связанным с конкретным действием, одновременно просматриваться в этом файле, время, необходимое для изоляции и диагностики сетевых проблем, может значительно снизиться.
