# <a name="using-ndf"></a>Использование NDF

инфраструктура диагностики сети (NDF) входит в состав Windows Vista, Windows Server 2008 и более поздних версий. NDF может использоваться разработчиками программного обеспечения для диагностики и устранения сетевых проблем, которые могут привести к сбою в работе приложений. Разработчики могут использовать предоставляемые корпорацией Майкрософт интерфейсы API, которые соединяют свои приложения с NDF. В некоторых случаях разработчики также могут создавать расширения классов поддержки NDF для диагностики проблем, связанных с сетевым программным обеспечением.
