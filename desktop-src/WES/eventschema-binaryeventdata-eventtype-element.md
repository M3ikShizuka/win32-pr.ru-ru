# <a name="binaryeventdata-eventtype-element"></a>Элемент Бинаревентдата (EventType)

Содержит данные события как двоичный BLOB-объект.

``` syntax
<xs:element name="BinaryEventData"
    type="hexBinary"
 />
```

Элемент **бинаревентдата** определяется сложным типом [**EventType**](eventschema-eventtype-complextype.md) .

|-------------------------------------|------------------------------------------------------|

| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
