# <a name="userdata-eventtype-element"></a>UserData (EventType), элемент

Содержит данные о событии.

``` syntax

    type="UserDataType"
 />
```

Элемент **UserData** определяется сложным типом [**EventType**](eventschema-eventtype-complextype.md) .
