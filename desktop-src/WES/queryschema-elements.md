# <a name="query-schema-elements"></a>Элементы схемы запроса

Ниже перечислены элементы, которые определяет схема запроса. В этом разделе содержатся имена элементов, которые будут использоваться в запросе. Однако, чтобы получить подробные сведения для каждого элемента, см. сложный тип, содержащий элемент.



| Элемент                                                                  | Описание                                                                                                                     |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**куерилист**](queryschema-querylist-element.md)                       | Корневой узел запроса.<br/>                                                                                          |
| [**Запрос (Куерилисттипе)**](queryschema-query-querylisttype-element.md) | Определяет набор селекторов и подавления, которые используются для включения или исключения событий из результирующего набора.<br/> |
| [**SELECT (QueryType)**](queryschema-select-querytype-element.md)       | Запрос XPath, определяющий события, включаемые в результирующий набор запроса.<br/>                                        |
| [**Подавлять (QueryType)**](queryschema-suppress-querytype-element.md)   | Запрос XPath, определяющий события, исключаемые из результирующего набора запроса.<br/>                                      |
