# <a name="windows-event-log-tools"></a>Windows Средства журнала событий

Windows В журнале событий содержатся следующие средства, используемые для построения поставщика.



| Средство                                                           | Описание                                                                                   |
|----------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**Компилятор сообщений (MC.exe)**](message-compiler--mc-exe-.md)  | Служебная программа командной строки, используемая для компиляции манифестов инструментирования и текстовых файлов сообщений. |
| WevtUtil.exe                                                   | Программа командной строки, используемая в основном для регистрации поставщика на компьютере. Кроме того, его можно использовать для получения метаданных о поставщике, его событиях и каналах, на которые он записывает события, а также для запроса событий из канала или файла журнала. Средство WevtUtil.exe входит в папку% WINDIR% \\ System32. Для получения сведений об использовании введите "wevtutil/?" в командной строке.<br/> Эта команда ограничена членами группы "Администраторы" и должна выполняться с повышенными привилегиями.|
