# <a name="map-patternmaptype-element"></a>Map (Паттернмаптипе), элемент

Определяет регулярные выражения, используемые для поиска соответствующей строки в строке сообщения и ее изменения.

``` syntax
<xs:element name="map"
    type="PatternMapValueType"
 />
```

Элемент **Map** определяется сложным типом [**паттернмаптипе**](eventmanifestschema-patternmaptype-complextype.md) .

## <a name="requirements"></a>Требования

|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows \[Только классические приложения Vista\]<br/>       |
| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
