# <a name="patternmaplisttype-complex-type"></a>Сложный тип Паттернмаплисттипе

Не используется. Определяет список пар регулярных выражений, которые используются для изменения строки сообщения.

``` syntax
<xs:complexType name="PatternMapListType">
    <xs:sequence>
        <xs:element name="patternMap"
            type="PatternMapType"
            maxOccurs="unbounded"
         />
    </xs:sequence>
</xs:complexType>
```

## <a name="child-elements"></a>Дочерние элементы



| Элемент                                                                         | Тип                                                                     | Описание                                                                                                                                                                                                                                                                                                              |
|---------------------------------------------------------------------------------|--------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**паттернмап**](eventmanifestschema-patternmap-patternmaplisttype-element.md) | [**паттернмаптипе**](eventmanifestschema-patternmaptype-complextype.md) | Определяет сопоставление между двумя регулярными выражениями. Одно выражение используется для поиска соответствующей строки в строке сообщения, а другая — для изменения строки перед тем, как служба снова поместит ее в строку сообщения. Используется первое сопоставление шаблонов, которое соответствует, и другие шаблоны не используются.<br/> |
