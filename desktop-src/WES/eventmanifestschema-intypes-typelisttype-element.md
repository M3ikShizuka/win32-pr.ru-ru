# <a name="intypes-typelisttype-element"></a>Элемент Types (Типелисттипе)

Содержит список определений типов входных данных.

``` syntax
<xs:element name="inTypes"
    type="InputTypeListType"
 />
```

Элемент **types** определяется сложным типом [**типелисттипе**](eventmanifestschema-typelisttype-complextype.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows \[Только классические приложения Vista\]<br/>       |
| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
