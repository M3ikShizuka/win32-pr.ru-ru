# <a name="controlguid-channelpublishingtype-element"></a>Контролгуид (Чаннелпублишингтипе), элемент

Определяет идентификатор GUID сеанса для сеанса ETW, содержащего события WPP.

``` syntax
<xs:element name="controlGuid"
    type="GUIDType"
 />
```

Элемент **контролгуид** определяется сложным типом [**чаннелпублишингтипе**](eventmanifestschema-channelpublishingtype-complextype.md) .
