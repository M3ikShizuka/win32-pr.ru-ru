# <a name="opcodes-tasktype-element"></a>opcode, элемент (TaskType)

Определяет список кодов операций, относящихся к задачам.

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

Элемент **кодов операций** определяется сложным типом [**TaskType**](eventmanifestschema-tasktype-complextype.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Только для \[ классических приложений Windows Vista\]<br/>       |
| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2008\]<br/> |
