# <a name="opcodes-tasktype-element"></a>opcode, элемент (TaskType)

Определяет список кодов операций, относящихся к задачам.

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

Элемент **кодов операций** определяется сложным типом [**TaskType**](eventmanifestschema-tasktype-complextype.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows \[Только классические приложения Vista\]<br/>       |
| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
