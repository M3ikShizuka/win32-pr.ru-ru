# <a name="level-channelpublishingtype-element"></a>Элемент Level (Чаннелпублишингтипе)

Степень серьезности событий, записываемых в канал.

``` syntax
<xs:element name="level"
    type="UInt8Type"
 />
```

Элемент **Level** определяется сложным типом [**чаннелпублишингтипе**](eventmanifestschema-channelpublishingtype-complextype.md) .

| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
