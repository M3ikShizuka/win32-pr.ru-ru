# <a name="windows-event-log-structures"></a>Структуры журнала событий Windows

В журнале событий Windows определены следующие структуры.



| Структура                                | Описание                                                    |
|------------------------------------------|----------------------------------------------------------------|
| [**\_ \_ имя входа RPC evt**](/windows/desktop/api/WinEvt/ns-winevt-evt_rpc_login) | Содержит сведения, используемые для подключения к удаленному компьютеру. |
| [**\_тип evt**](/windows/desktop/api/WinEvt/ns-winevt-evt_variant)      | Содержит данные события или значения свойств.                        |
