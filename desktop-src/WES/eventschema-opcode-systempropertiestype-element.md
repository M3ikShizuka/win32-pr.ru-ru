# <a name="opcode-systempropertiestype-element"></a>Opcode, элемент (Системпропертиестипе)

Код операции, определенный в событии.

``` syntax
<xs:element name="Opcode"
    type="unsignedByte"
 />
```

Элемент **кода операции** определяется сложным типом [**системпропертиестипе**](eventschema-systempropertiestype-complextype.md) .
