# <a name="eventmanifest-schema-simple-types"></a>Простые типы схемы Евентманифест

Ниже приведены простые типы, определяемые схемой Евентманифест.



| Простой тип                                                       | Описание                                                                                                                                |
|-------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [**каунттипе**](eventmanifestschema-counttype-simpletype.md)     | Определяет тип счетчика, который используется для указания количества элементов в массиве.<br/>                                                |
| [**ксимболтипе**](eventmanifestschema-csymboltype-simpletype.md) | Определяет допустимое имя символа C/C++.<br/>                                                                                              |
| [**Равно**](eventmanifestschema-filepath-simpletype.md)       | Определяет строку, содержащую полный путь к файлу.<br/>                                                                |
| [**гуидтипе**](eventmanifestschema-guidtype-simpletype.md)       | Определяет тип глобального уникального идентификатора в формате реестра.<br/>                                                                  |
| [**HexInt8Type**](eventmanifestschema-hex8type-simpletype.md)    | Определяет 1-байтовый шестнадцатеричный тип.<br/>                                                                                              |
| [**HexInt16Type**](eventmanifestschema-hex16type-simpletype.md)  | Определяет 2-байтовый шестнадцатеричный тип.<br/>                                                                                              |
| [**HexInt32Type**](eventmanifestschema-hex32type-simpletype.md)  | Определяет 4-байтовый шестнадцатеричный тип.<br/>                                                                                              |
| [**HexInt64Type**](eventmanifestschema-hex64type-simpletype.md)  | Определяет 8-байтовый шестнадцатеричный тип.<br/>                                                                                             |
| [**UInt8Type**](eventmanifestschema-hexint8type-simpletype.md)   | Определяет тип Byte без знака.<br/>                                                                                                  |
| [**UInt16Type**](eventmanifestschema-hexint16type-simpletype.md) | Определяет короткий тип без знака.<br/>                                                                                                 |
| [**UInt32Type**](eventmanifestschema-hexint32type-simpletype.md) | Определяет целочисленный тип без знака.<br/>                                                                                               |
| [**UInt64Type**](eventmanifestschema-hexint64type-simpletype.md) | Определяет тип long без знака.<br/>                                                                                                  |
| [**ленгстипе**](eventmanifestschema-lengthtype-simpletype.md)   | Определяет тип длины, используемый для указания числа байтов или символов в элементе данных переменной длины.<br/>                 |
| [**кнамелист**](eventmanifestschema-qnamelist-simpletype.md)     | Определяет список полных имен.<br/>                                                                                              |
| [**стртаблереф**](eventmanifestschema-strtableref-simpletype.md) | Определяет строку, которая ссылается на строку сообщения, определенную в таблице строк в манифесте или в файле сообщения (. MC).<br/> |
