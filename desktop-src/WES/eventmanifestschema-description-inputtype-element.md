# <a name="description-inputtype-element"></a>Description (InputType), элемент

Описание типа входных данных.

``` syntax

    type="string"
 />
```

Элемент **Description** определяется сложным типом [**InputType**](eventmanifestschema-inputtype-complextype.md) .
