# <a name="task-tasklisttype-element"></a>Элемент Task (Тасклисттипе)

Определяет компонент или подкомпонент приложения.

``` syntax
<xs:element name="task"
    type="TaskType"
 />
```

Элемент **Task** определяется сложным типом [**тасклисттипе**](eventmanifestschema-tasklisttype-complextype.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows \[Только классические приложения Vista\]<br/>       |
| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
