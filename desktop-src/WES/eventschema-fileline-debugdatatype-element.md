# <a name="fileline-debugdatatype-element"></a>Филелине (Дебугдататипе), элемент

Имя исходного файла и строка в исходном файле, которая записала в журнал сообщение трассировки.

``` syntax
<xs:element name="FileLine"
    type="string"
 />
```

Элемент **филелине** определяется сложным типом [**дебугдататипе**](eventschema-debugdatatype-complextype.md) .

|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows \[Только классические приложения Vista\]<br/>       |
| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
