# <a name="version-systempropertiestype-element"></a>Version (Системпропертиестипе), элемент

Содержит номер версии определения события.

``` syntax
<xs:element name="Version"
    type="unsignedByte"
 />
```

Элемент **Version** определяется сложным типом [**системпропертиестипе**](eventschema-systempropertiestype-complextype.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows \[Только классические приложения Vista\]<br/>       |
| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
