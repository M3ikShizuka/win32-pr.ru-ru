# <a name="data-structdefinitiontype-element"></a>Элемент Data (Структдефинитионтипе)

Определяет элемент данных, который необходимо включить в структуру.

``` syntax
<xs:element name="data"
    type="DataDefinitionType"
 />
```

Элемент **Data** определяется сложным типом [**структдефинитионтипе**](eventmanifestschema-structdefinitiontype-complextype.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Только для \[ классических приложений Windows Vista\]<br/>       |
| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2008\]<br/> |
