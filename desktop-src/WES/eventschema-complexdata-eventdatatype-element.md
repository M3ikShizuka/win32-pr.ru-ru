# <a name="complexdata-eventdatatype-element"></a>Комплексдата (Евентдататипе), элемент

Структура, определенная в шаблоне для события.

``` syntax
<xs:element name="ComplexData"
    type="ComplexDataType"
 />
```

Элемент **комплексдата** определяется сложным типом [**евентдататипе**](eventschema-eventdatatype-complextype.md) .
