# <a name="opcode-renderinginfotype-element"></a>Opcode, элемент (Рендерингинфотипе)

Строка отображаемого сообщения кода операции, указанного в событии.

``` syntax
<xs:element name="Opcode"
    type="string"
 />
```

Элемент **кода операции** определяется сложным типом [**рендерингинфотипе**](eventschema-renderingtype-complextype.md) .
