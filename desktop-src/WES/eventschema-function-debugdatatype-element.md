# <a name="function-debugdatatype-element"></a>Элемент Function (Дебугдататипе)

Имя функции, которая зарегистрировала сообщение трассировки.

``` syntax
<xs:element name="Function"
    type="string"
    minOccurs="0"
 />
```

Элемент **Function** определяется сложным типом [**дебугдататипе**](eventschema-debugdatatype-complextype.md) .
