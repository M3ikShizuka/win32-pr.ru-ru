# <a name="intype-inputtypelisttype-element"></a>Элемент Type (Инпуттипелисттипе)

Определяет тип входных данных.

``` syntax
<xs:element name="inType"
    type="InputType"
 />
```

Элемент " **тип** " определяется сложным типом [**инпуттипелисттипе**](eventmanifestschema-inputtypelisttype-complextype.md) .

| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Только для \[ классических приложений Windows Vista\]<br/>       |
| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2008\]<br/> |
