# <a name="task-renderinginfotype-element"></a>Элемент Task (Рендерингинфотипе)

Строка отображаемого сообщения задачи, указанной в событии.

``` syntax
<xs:element name="Task"
    type="string"
 />
```

Элемент **Task** определяется сложным типом [**рендерингинфотипе**](eventschema-renderingtype-complextype.md) .
