# <a name="compiling-an-instrumentation-manifest"></a>Компиляция манифеста инструментирования

После [записи манифеста](writing-an-instrumentation-manifest.md)используйте компилятор сообщений для проверки манифеста и создания файлов ресурсов и заголовков, включаемых в поставщик. Дополнительные сведения об использовании компилятора см. в разделе [**компилятор сообщений**](message-compiler--mc-exe-.md). Включите в проект файлы заголовков и ресурсов, которые компилятор создает в проекте, содержащем оставшуюся часть исходного кода поставщика. Сведения о том, как записать события, определенные в манифесте, см. в разделе [Разработка поставщика](developing-a-provider.md).
