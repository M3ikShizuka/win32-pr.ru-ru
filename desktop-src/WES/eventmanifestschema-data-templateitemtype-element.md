# <a name="data-templateitemtype-element"></a>Элемент Data (Темплатеитемтипе)

Определяет элемент данных, который необходимо включить в событие.

``` syntax
<xs:element name="data"
    type="DataDefinitionType"
 />
```

Элемент **Data** определяется сложным типом [**темплатеитемтипе**](eventmanifestschema-templateitemtype-complextype.md) .

## <a name="requirements"></a>Требования

|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows \[Только классические приложения Vista\]<br/>       |
| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
