# <a name="patternmap-patternmaplisttype-element"></a>Паттернмап (Паттернмаплисттипе), элемент

Определяет сопоставление между двумя регулярными выражениями. Одно выражение используется для поиска соответствующей строки в строке сообщения, а другая — для изменения строки перед тем, как служба снова поместит ее в строку сообщения.

``` syntax
<xs:element name="patternMap"
    type="PatternMapType"
 />
```

Элемент **паттернмап** определяется сложным типом [**паттернмаплисттипе**](eventmanifestschema-patternmaplisttype-complextype.md) .
