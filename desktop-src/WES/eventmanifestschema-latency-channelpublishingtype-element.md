# <a name="latency-channelpublishingtype-element"></a>Элемент задержки (Чаннелпублишингтипе)

Время ожидания перед очисткой буферов (в миллисекундах).

``` syntax
<xs:element name="latency"
    type="unsignedInt"
 />
```

Элемент **задержки** определяется сложным типом [**чаннелпублишингтипе**](eventmanifestschema-channelpublishingtype-complextype.md) .
