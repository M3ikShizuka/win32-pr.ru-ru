# <a name="computer-systempropertiestype-element"></a>Computer (Системпропертиестипе), элемент

имя компьютера, на котором произошло событие.

``` syntax
<xs:element name="Computer"
    type="string"
 />
```

Элемент **Computer** определяется сложным типом [**системпропертиестипе**](eventschema-systempropertiestype-complextype.md) .
