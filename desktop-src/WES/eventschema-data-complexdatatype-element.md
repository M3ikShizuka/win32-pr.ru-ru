# <a name="data-complexdatatype-element"></a>Элемент Data (Комплексдататипе)

Список элементов данных в структуре. Список элементов находится в том же порядке, что и определено в шаблоне.

``` syntax
<xs:element name="Data"
    type="DataType"
 />
```

Элемент **Data** определяется сложным типом [**комплексдататипе**](eventschema-complexdatatype-complextype.md) .
