# <a name="eventdata-eventtype-element"></a>Элемент EventData (EventType)

Содержит данные о событии.

``` syntax
<xs:element name="EventData"
    type="EventDataType"
 />
```

Элемент **EVENTDATA** определяется сложным типом [**EventType**](eventschema-eventtype-complextype.md) .
