# <a name="keyword-keywords-element"></a>Ключевое слово (keywords), элемент

Содержит отображаемое ключевое слово.

``` syntax
<xs:element name="Keyword"
    type="string"
 />
```

Элемент **keyword** определяется элементом [**keywords**](eventschema-keywords-renderingtype-element.md) .

| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Только для \[ классических приложений Windows Vista\]<br/>       |
| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2008\]<br/> |
