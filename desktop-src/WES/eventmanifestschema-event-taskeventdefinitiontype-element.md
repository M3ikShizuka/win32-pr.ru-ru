# <a name="event-taskeventdefinitiontype-element"></a>Элемент Event (Таскевентдефинитионтипе)

\[начиная с компилятора сообщений, который поставляется с версией Windows 7 Windows SDK, этот элемент больше не доступен.\]

Определяет событие для события, относящегося к задаче.

``` syntax
<xs:element name="event"
    type="EventDefinitionType"
 />
```

Элемент **event** определяется сложным типом [**таскевентдефинитионтипе**](eventmanifestschema-taskeventdefinitiontype-complextype.md) .

| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows \[Только классические приложения Vista\]<br/>       |
| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
