# <a name="provider-renderinginfotype-element"></a>Элемент Provider (Рендерингинфотипе)

Строка отображаемого сообщения для поставщика.

``` syntax
<xs:element name="Provider"
    type="string"
 />
```

Элемент **provider** определяется сложным типом [**рендерингинфотипе**](eventschema-renderingtype-complextype.md) .
