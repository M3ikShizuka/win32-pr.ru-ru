# <a name="component-debugdatatype-element"></a>Элемент Component (Дебугдататипе)

Имя компонента, который записал в журнал сообщение трассировки.

``` syntax
<xs:element name="Component"
    type="string"
 />
```

Элемент **Component** определяется сложным типом [**дебугдататипе**](eventschema-debugdatatype-complextype.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Только для \[ классических приложений Windows Vista\]<br/>       |
| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2008\]<br/> |
