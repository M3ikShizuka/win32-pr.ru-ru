# <a name="bitmaptype-complex-type"></a>Сложный тип Битмаптипе

Определяет список сопоставлений имен и значений между битовыми значениями и строковыми значениями.

``` syntax
<xs:complexType name="BitMapType">
    <xs:sequence>
        <xs:element name="map"
            type="BitMapValueType"
            maxOccurs="unbounded"
         />
    </xs:sequence>
    <xs:attribute name="name"
        type="string"
     />
    <xs:attribute name="symbol"
        type="CSymbolType"
        use="optional"
     />
</xs:complexType>
```

## <a name="child-elements"></a>Дочерние элементы



| Элемент                                                   | Тип                                                                       | Описание                                                            |
|-----------------------------------------------------------|----------------------------------------------------------------------------|------------------------------------------------------------------------|
| [**карта**](eventmanifestschema-map-bitmaptype-element.md) | [**битмапвалуетипе**](eventmanifestschema-bitmapvaluetype-complextype.md) | Определяет сопоставление между битовым значением и строковым значением.<br/> |



## <a name="attributes"></a>Атрибуты



| Имя   | Тип                                                              | Описание                                                                                                                                                                                                                                                                                                         |
|--------|-------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| name   | строка                                                            | Имя битовой карты.<br/>                                                                                                                                                                                                                                                                                  |
| символ | [**ксимболтипе**](eventmanifestschema-csymboltype-simpletype.md) | Символ, используемый для ссылки на сопоставления в приложении. [**Компилятор сообщений (MC.exe)**](message-compiler--mc-exe-.md) использует символ для создания константы для Map в файле заголовка, создаваемого компилятором. Если не указать символ, компилятор создаст его.<br/> |
