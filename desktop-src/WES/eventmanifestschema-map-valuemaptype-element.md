# <a name="map-valuemaptype-element"></a>Map (Валуемаптипе), элемент

Определяет сопоставление между целочисленным значением и строковым значением.

``` syntax
<xs:element name="map"
    type="ValueMapValueType"
 />
```

Элемент **Map** определяется сложным типом [**валуемаптипе**](eventmanifestschema-valuemaptype-complextype.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Только для \[ классических приложений Windows Vista\]<br/>       |
| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2008\]<br/> |
