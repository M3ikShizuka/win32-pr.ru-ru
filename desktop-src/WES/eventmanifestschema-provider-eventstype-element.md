# <a name="provider-eventstype-element"></a>Элемент Provider (Евентстипе)

Определяет поставщик и метаданные, используемые для определения его событий.

``` syntax
<xs:element name="provider"
    type="ProviderType"
 />
```

Элемент **provider** определяется сложным типом [**евентстипе**](eventmanifestschema-eventstype-complextype.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Только для \[ классических приложений Windows Vista\]<br/>       |
| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2008\]<br/> |
