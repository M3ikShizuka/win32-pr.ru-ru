# <a name="level-systempropertiestype-element"></a>Элемент Level (Системпропертиестипе)

Содержит степень серьезности события.

``` syntax
<xs:element name="Level"
    type="unsignedByte"
 />
```

Элемент **Level** определяется сложным типом [**системпропертиестипе**](eventschema-systempropertiestype-complextype.md) .
