# <a name="message-renderinginfotype-element"></a>Элемент Message (Рендерингинфотипе)

Содержит сообщение о событии, которое отображается для события.

``` syntax
<xs:element name="Message"
    type="string"
 />
```

Элемент **Message** определяется сложным типом [**рендерингинфотипе**](eventschema-renderingtype-complextype.md) .

## <a name="requirements"></a>Требования

|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows \[Только классические приложения Vista\]<br/>       |
| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
