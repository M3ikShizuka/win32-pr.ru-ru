# <a name="namedqueries-providertype-element"></a>Намедкуериес (ProviderType), элемент

Не используется. Определяет список именованных запросов, которые запрашивают строку сообщения о событии для значения и выполняют указанное действие, если оно найдено.

``` syntax
<xs:element name="namedQueries"
    type="NamedQueryType"
 />
```

Элемент **намедкуериес** определяется сложным типом [**ProviderType**](eventmanifestschema-providertype-complextype.md) .
