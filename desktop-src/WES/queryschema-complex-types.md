# <a name="query-schema-complex-types"></a>Сложные типы схемы запросов

Ниже приведены сложные типы, определяемые схемой запроса.



| Сложный тип                                                   | Описание                                                                                                                                                         |
|----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**куерилисттипе**](queryschema-querylisttype-complextype.md) | Определяет список запросов, которые могут содержать набор селекторов и подавления, которые используются для включения или исключения событий из результирующего набора. <br/> |
| [**QueryType**](queryschema-querytype-complextype.md)         | Определяет набор запросов Selector и суппрессор, которые используются для включения или исключения событий из результирующего набора.<br/>                               |
