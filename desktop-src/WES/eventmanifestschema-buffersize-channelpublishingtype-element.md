# <a name="buffersize-channelpublishingtype-element"></a>bufferSize (Чаннелпублишингтипе), элемент

Содержит объем памяти (в килобайтах), выделенной для буферов сеанса.

``` syntax
<xs:element name="bufferSize"
    type="unsignedInt"
 />
```

Элемент **bufferSize** определяется сложным типом [**чаннелпублишингтипе**](eventmanifestschema-channelpublishingtype-complextype.md) .
