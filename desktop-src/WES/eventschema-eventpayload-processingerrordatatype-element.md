# <a name="eventpayload-processingerrordatatype-element"></a>Евентпайлоад (Процессинжеррордататипе), элемент

Содержит двоичные данные событий для события, которое привело к ошибке при обработке данных события.

``` syntax
<xs:element name="EventPayload"
    type="hexBinary"
 />
```

Элемент **евентпайлоад** определяется сложным типом [**процессинжеррордататипе**](eventschema-processingerrordatatype-complextype.md) .
