# <a name="channel-systempropertiestype-element"></a>Элемент Channel (Системпропертиестипе)

Канал, в который было записано событие.

``` syntax
<xs:element name="Channel"
    type="anyURI"
    minOccurs="0"
 />
```

Элемент **Channel** определяется сложным типом [**системпропертиестипе**](eventschema-systempropertiestype-complextype.md) .
