# <a name="sequencenumber-debugdatatype-element"></a>SequenceNumber (Дебугдататипе), элемент

Локальный или глобальный порядковый номер сообщения трассировки.

``` syntax
<xs:element name="SequenceNumber"
    type="unsignedInt"
    minOccurs="0"
 />
```

Элемент **SequenceNumber** определяется сложным типом [**дебугдататипе**](eventschema-debugdatatype-complextype.md) .
