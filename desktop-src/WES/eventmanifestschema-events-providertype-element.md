# <a name="events-providertype-element"></a>Элемент Events (ProviderType)

Содержит список событий, определяемых поставщиком.

``` syntax
<xs:element name="events"
    type="DefinitionType"
 />
```

Элемент **Events** определяется сложным типом [**ProviderType**](eventmanifestschema-providertype-complextype.md) .

| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows \[Только классические приложения Vista\]<br/>       |
| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
