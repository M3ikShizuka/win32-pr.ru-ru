# <a name="data-eventdatatype-element"></a>Элемент Data (Евентдататипе)

Элемент данных верхнего уровня, определенный в шаблоне для события.

``` syntax
<xs:element name="Data"
    type="DataFieldType"
 />
```

Элемент **Data** определяется сложным типом [**евентдататипе**](eventschema-eventdatatype-complextype.md) .
