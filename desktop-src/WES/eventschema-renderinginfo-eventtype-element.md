# <a name="renderinginfo-eventtype-element"></a>Элемент Рендерингинфо (EventType)

Содержит строки отображаемых сообщений для события (включая строку сообщения события и строки сообщения для любого свойства события, например уровень, задача и код операции).

``` syntax
<xs:element name="RenderingInfo"
    type="RenderingType"
 />
```

Элемент **рендерингинфо** определяется сложным типом [**EventType**](eventschema-eventtype-complextype.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Только для \[ классических приложений Windows Vista\]<br/>       |
| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2008\]<br/> |
