</dd> <dt>

<span id="_security_x.500_gly"></span><span id="_SECURITY_X.500_GLY"></span>**X. 500**

</dd> <dt>

<span id="_security_x.509_gly"></span><span id="_SECURITY_X.509_GLY"></span>**X. 509**
</dt> <dd>

Международный стандарт сертификатов, определяющий их обязательные части.

</dd> <dt>

<span id="_security_x509_asn_encoding_gly"></span><span id="_SECURITY_X509_ASN_ENCODING_GLY"></span>**\_ \_ Кодировка X509 ASN**

Тип кодировки, указывающий кодировку сертификата. Типы кодирования сертификатов хранятся в младшем слове типа **DWORD** (значение: 0x00000001). Этот тип кодировки функционально аналогичен ШИФРОВАНию \_ \_ кодировки ASN.

</dd> <dt>

<span id="_security_xor_gly"></span><span id="_SECURITY_XOR_GLY"></span>**XOR**
</dt> <dd>

Монопольный **или**. Логическая операция, возвращающая значение true, только если один из операндов имеет значение true, а другой — значение false. Если оба операнда одинаковы (true или false), операция возвращает значение false.

</dd> </dl>
