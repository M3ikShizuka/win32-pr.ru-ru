<dl> <dt>

<span id="_security_ucs_gly"></span><span id="_SECURITY_UCS_GLY"></span>**КОДИРОВК**
</dt> <dd>

См. раздел *универсальные наборы символов*.

</dd> <dt>

<span id="_security_unicode_gly"></span><span id="_SECURITY_UNICODE_GLY"></span>**Юникод**
</dt> <dd>

Общедоступный стандарт кодировки символов, позволяющий хранить в каждой строке дополнительные сведения, определяя 16-разрядные символьные строки вместо стандартных 8-разрядных символьных строк. Юникод обеспечивает универсальный обмен данными и улучшает обработку многоязычного текста. Строки в Юникоде также называются *расширенными строками*.

</dd> <dt>

<span id="_security_universal_character_set_gly"></span><span id="_SECURITY_UNIVERSAL_CHARACTER_SET_GLY"></span>**Универсальный набор символов**
</dt> <dd>

КОДИРОВК 16-разрядный набор символов.

</dd> <dt>

<span id="_security_universal_well_known_security_identifier_gly"></span><span id="_SECURITY_UNIVERSAL_WELL_KNOWN_SECURITY_IDENTIFIER_GLY"></span>**универсальный известный идентификатор безопасности**
</dt> <dd>

[*Идентификатор безопасности*](s-gly.md) (SID), известный во всех системах, которые соответствуют модели безопасности, основанной на прошедших проверку подлинности пользователях и управлении доступом на уровне пользователей.

</dd> <dt>

<span id="_security_upn_gly"></span><span id="_SECURITY_UPN_GLY"></span>**УЧАСТНИКА**
</dt> <dd>

См. раздел *имя участника-пользователя*.

</dd> <dt>

<span id="_security_user_gly"></span><span id="_SECURITY_USER_GLY"></span>**нажат**
</dt> <dd>

Пользователь системы определяется процедурой входа в систему, которая устанавливает процесс, с помощью которого приложения могут работать и обращаться к другим объектам, относящимся к безопасности. Примером процедуры входа в систему является использование смарт-карты, возможно, в сочетании с паролем или личным идентификационным номером (ПИН-код).

</dd> <dt>

<span id="_security_user_interface_smart_card_gly"></span><span id="_SECURITY_USER_INTERFACE_SMART_CARD_GLY"></span>**Пользовательский интерфейс (смарт-карта)**
</dt> <dd>

Общее диалоговое окно, которое позволяет пользователю подключиться к смарт-карте и использовать его в приложении. С помощью этого диалогового окна пользователь может указать конкретную карточку или найти смарт-карту, которую нужно открыть.

</dd> <dt>

<span id="_security_user_principal_name_gly"></span><span id="_SECURITY_USER_PRINCIPAL_NAME_GLY"></span>**имя участника-пользователя**
</dt> <dd>

УЧАСТНИКА Имя учетной записи пользователя (иногда называется именем для входа пользователя) и доменное имя, идентифицирующее домен, в котором находится учетная запись пользователя. Это стандартный способ входа в домен Windows. Формат: someone@example.com (как для адреса электронной почты).

</dd> </dl>
