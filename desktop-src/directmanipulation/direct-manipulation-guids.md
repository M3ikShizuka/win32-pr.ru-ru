## <a name="master-class-ids"></a>Главный класс — идентификаторы

| Идентификатор GUID                                     | Описание                                                                                                                                                                                                                                                                                       |
|------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **54E211B6-3650-4F75-8334-FA359598E1C5** | Класс Директманипулатионманажер. Этот объект предоставляет доступ ко всем функциям [прямого управления](direct-manipulation-portal.md) и интерфейсам API, доступным для приложения.                                                                                                                         |
| **79DEA627-A08A-43AC-8EF5-6900B9299126** | Класс Дкомпманипулатионкомпоситор. Это реализация [**идиректманипулатионкомпоситор**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationcompositor) , которая создает оболочку для DirectComposition. Через объект-компоновщик Директманипулатион может применять выходные данные путем установки преобразований непосредственно в дереве Дкомп. |

## <a name="secondary-content-class-ids"></a>Вторичный класс содержимого — идентификаторы

| Идентификатор GUID                                  | Описание                                                                                                                      |
|---------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| **\_ВЕРТИКАЛИНДИКАТОРКОНТЕНТ CLSID**   | Индикатор вертикального панорамирования. Визуальный элемент, который показывает текущую положение в содержимом, которое выходит за пределы экрана по вертикали.     |
| **\_ХОРИЗОНТАЛИНДИКАТОРКОНТЕНТ CLSID** | Индикатор горизонтального панорамирования. Визуальный элемент, отображающий текущую положение в содержимом, которое выходит за пределы экрана по горизонтали. |
| **\_ВИРТУАЛВИЕВПОРТКОНТЕНТ CLSID**     | Виртуальное окно просмотра. Виртуальное окно просмотра можно использовать для связи с элементами с фиксированной позицией для окна просмотра с настроенным масштабом.          |

## <a name="behavior-objects-class-ids"></a>Идентификаторы классов объектов поведения
