# <a name="direct-manipulation-interfaces"></a>Интерфейсы прямого управления

Разделы, содержащиеся в этом разделе, содержат справочные спецификации для интерфейсов [прямого управления](direct-manipulation-portal.md) .

> [!Note]  
> При реализации объекта [прямой манипуляции](direct-manipulation-portal.md) убедитесь, что реализация [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) поддерживает многопоточность с помощью потокобезопасных подсчета ссылок. Дополнительные сведения см. в разделе [**интерлоккединкремент**](/windows/win32/api/winnt/nf-winnt-interlockedincrement) и [**интерлоккеддекремент**](/windows/win32/api/winnt/nf-winnt-interlockeddecrement).
