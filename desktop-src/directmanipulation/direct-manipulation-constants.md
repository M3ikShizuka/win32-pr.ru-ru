# <a name="direct-manipulation-constants"></a>Константы прямой манипуляции

В этом разделе приведены справочные спецификации для констант [прямого управления](direct-manipulation-portal.md) .

| Константа/значение                                                                                                                                                                                                                                                                         | Описание                                                                                    |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------|
| <span id="DIRECTMANIPULATION_KEYBOARDFOCUS"></span><span id="directmanipulation_keyboardfocus"></span><dl> <dt>**Директманипулатион \_ КЭЙБОАРДФОКУС**</dt> <dt>0xFFFFFFFE</dt> </dl> | Указывает идентификатор псуедо-указателя для эмуляции сенсорного контакта с помощью ввода с клавиатуры.<br/> |
| <span id="DIRECTMANIPULATION_MOUSEFOCUS"></span><span id="directmanipulation_mousefocus"></span><dl> <dt>**Директманипулатион \_ МАУСЕФОКУС**</dt> <dt>0xFFFFFFFD</dt> </dl>          | Указывает идентификатор псуедо-указателя для эмуляции сенсорного контакта с помощью ввода мыши.<br/>    |
| <span id="DIRECTMANIPULATION_MINIMUM_ZOOM"></span><span id="directmanipulation_minimum_zoom"></span><dl> <dt>**Директманипулатион \_ Минимальный \_ масштаб**</dt> <dt>0,1</dt> </dl>          | Указывает минимально допустимую границу масштабирования, равное 10%.<br/>                               |
