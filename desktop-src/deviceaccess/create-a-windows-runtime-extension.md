# <a name="create-a-windows-runtime-component-for-device-access"></a>создание среда выполнения Windows компонента для доступа к устройству

можно создать компонент среда выполнения Windows, чтобы заключить в оболочку компонент, обращающийся к вашему драйверу. затем можно написать приложение для устройства магазина Windows для устройства на C# или JavaScript.

## <a name="instructions"></a>Инструкции

См. [Пример пользовательского доступа к драйверу](https://github.com/microsoftarchive/msdn-code-gallery-microsoft/tree/411c271e537727d737a53fa2cbe99eaecac00cc0/Official%20Windows%20Platform%20Sample/Custom%20driver%20access%20sample). в этом примере реализуется среда выполнения Windows компонент для провидингсе функций специализированного устройства и используются функции в приложении для устройства магазина Windows.
