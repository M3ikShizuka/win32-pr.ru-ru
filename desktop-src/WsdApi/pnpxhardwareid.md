# <a name="pnpxhardwareid-element"></a>Пнпксхардвареид, элемент

Указывает идентификатор оборудования PnP-X для службы. Устройство PnP соответствует этому элементу с описанием оборудования, указанным в \[ разделе ПНПКСДЕВИЦЕ \] INF-файла устройства. На основе этих сведений Служба PnP выбирает и загружает наиболее подходящий драйвер устройства.

## <a name="usage"></a>Использование

``` syntax
<PnPXHardwareId/>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                             | Описание                                                                            |
|-------------------------------------|----------------------------------------------------------------------------------------|
| [**внутрипроцессных**](hosted.md)<br/> | Определяет элементы для служб, определенных узлом службы. <br/> <br/> |



## <a name="remarks"></a>Комментарии

Чтобы указать более одного идентификатора оборудования, разделяйте идентификаторы пробелами, например "Пнпкс \_ самплесервице \_ Хвид \_ 1 пнпкс \_ самплесервице \_ хвид \_ 2 пнпкс \_ SampleService1 \_ хвид \_ 3".
