# <a name="proxybuilderdeclarations-element"></a>Проксибуилдердекларатионс, элемент

Создает объявления для функций, которые создают типизированные прокси-серверы.

## <a name="usage"></a>Использование

``` syntax
<proxyBuilderDeclarations>
  child elements
</proxyBuilderDeclarations>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы



| Элемент                                     | Описание                                                                        |
|---------------------------------------------|------------------------------------------------------------------------------------|
| [**proxyClass**](proxyclass.md)<br/> | Имя класса, создаваемого из функции построителя прокси.<br/> <br/> |



### <a name="child-element-sequence"></a>Последовательность дочерних элементов

``` syntax
proxyClass
```

## <a name="parent-elements"></a>Родительские элементы



| Элемент                         | Описание                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**File**](file.md)<br/> | Выводит файл из генератора кода.<br/> <br/> |



## <a name="element-information"></a>Сведения об элементе



|                                     |               |
|-------------------------------------|---------------|
| Минимальная поддерживаемая система<br/> | Windows Vista |
| Может быть пустым                        | Нет            |
