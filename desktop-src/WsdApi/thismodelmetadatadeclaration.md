# <a name="thismodelmetadatadeclaration-element"></a>Сисмоделметадатадекларатион, элемент

Создает прямую декларацию для константы C для метаданных производителя, указанных в элементе [**сисмоделметадата**](thismodelmetadata.md) . Это объявление используется в файле заголовка.

## <a name="usage"></a>Использование

``` syntax
<thisModelMetadataDefinition/>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                         | Описание                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**File**](file.md)<br/> | Выводит файл из генератора кода.<br/> <br/> |
