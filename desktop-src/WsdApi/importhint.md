# <a name="importhint-element"></a>Импорсинт, элемент

Указывает расположение загрузки для \<wsdl:import> директивы, которая не указывает расположение явным образом.

## <a name="usage"></a>Использование

``` syntax
<importHint>
  child elements
</importHint>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы



| Элемент                                   | Описание                                                                                                                       |
|-------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**места**](location.md)<br/>   | Расположение импортируемого файла. Расположением может быть относительный путь, абсолютный путь или URL-адрес HTTP.<br/> <br/> |
| [**Имен**](namespace.md)<br/> | Импортируемое пространство имен. Оно должно соответствовать пространству имен, указанному в \<wsdl:import> элементе.<br/> <br/>     |



### <a name="child-element-sequence"></a>Последовательность дочерних элементов

``` syntax
(
  nameSpace, 
  location
)
```

## <a name="parent-elements"></a>Родительские элементы



| Элемент                         | Описание                                                                       |
|---------------------------------|-----------------------------------------------------------------------------------|
| [**файле**](wsdl.md)<br/> | Указывает WSDL-файл для обработки сведений о контракте.<br/> <br/> |



## <a name="element-information"></a>Сведения об элементе



| Метка | Значение |
|-------------------------------------|---------------|
| Минимальная поддерживаемая система<br/> | Windows Vista |
| Может быть пустым                        | нет            |
