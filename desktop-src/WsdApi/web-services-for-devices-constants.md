# <a name="web-services-on-devices-constants"></a>Константы веб-служб на устройствах

Веб-службы на устройствах используют следующие константы.



| Константа                                                                                                                                                                                                                        | Описание                                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WSD_DEFAULT_HOSTING_ADDRESS"></span><span id="wsd_default_hosting_address"></span><dl> <dt>**\_адрес размещения по умолчанию для WSD \_ \_**</dt> </dl>                       | Адрес по умолчанию (в формате [UrlPrefix](/windows/desktop/Http/urlprefix-strings) ), который будет использоваться узлом WSDAPI для прослушивания запросов через порт 5357. <br/>                                                 |
| <span id="WSD_DEFAULT_SECURE_HOSTING_ADDRESS"></span><span id="wsd_default_secure_hosting_address"></span><dl> <dt>**\_ \_ \_ адрес безопасного размещения по УМОЛЧАНИю для WSD \_**</dt> </dl> | Защищенный адрес по умолчанию (в формате [UrlPrefix](/windows/desktop/Http/urlprefix-strings) ), который будет использоваться узлом WSDAPI для прослушивания запросов через порт 5358. <br/>                                          |
| <span id="WSD_DEFAULT_EVENTING_ADDRESS__"></span><span id="wsd_default_eventing_address__"></span><dl> <dt>**WSD \_ \_ \_ адрес событий по умолчанию**</dt> </dl>               | Адрес по умолчанию (в формате [UrlPrefix](/windows/desktop/Http/urlprefix-strings) ), который будет использоваться узлом WSDAPI для прослушивания событий на порте 5358 для защищенных узлов и портов 5357 для других узлов. <br/> |
