# <a name="macroprefix-element"></a>Макропрефикс, элемент

Определяет префикс, который будет использоваться в созданном коде для имен макросов в пространстве имен.

## <a name="usage"></a>Использование

``` syntax
<macroPrefix/>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                                   | Описание                                                        |
|-------------------------------------------|--------------------------------------------------------------------|
| [**Имен**](namespace.md)<br/> | Пространство имен, используемое для создания кода.<br/> <br/> |



## <a name="remarks"></a>Remarks

Этот элемент переопределяет префикс URI по умолчанию, используемый для созданных макросов. Например, если префиксом макроса является "AV \_ ", а имя — "тюнер", макрос, созданный для полного имени, будет иметь значение "AV \_ тюнер".

По умолчанию созданный код создает предпочтительный префикс макроса из универсального кода ресурса (URI).
