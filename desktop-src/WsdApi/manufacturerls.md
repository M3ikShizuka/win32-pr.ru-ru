# <a name="manufacturerls-element"></a>Мануфактурерлс, элемент

Указывает локализованную версию имени производителя.

## <a name="usage"></a>Использование

``` syntax
<manufacturerLS
  Language = "language identifier string"
  Data = "localized manufacturer name string"/>
```

## <a name="attributes"></a>Атрибуты



| attribute               | Тип                                          | Обязательно       | Описание                                                             |
|-------------------------|-----------------------------------------------|----------------|-------------------------------------------------------------------------|
| **Данные**<br/>     | Строка имени локализованного изготовителя<br/> | Да<br/> | Локализованное название изготовителя.<br/> <br/>                 |
| **Язык**<br/> | Строка идентификатора языка<br/>         | Да<br/> | Язык локализованного имени производителя.<br/> <br/> |



## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                                                   | Описание                                                                                          |
|-----------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| [**сисмоделметадата**](thismodelmetadata.md)<br/> | Определяет производителя и метаданные модели для реализуемого устройства.<br/> <br/> |



## <a name="element-information"></a>Сведения об элементе



| Метка | Значение |
|-------------------------------------|---------------|
| Минимальная поддерживаемая система<br/> | Windows Vista |
| Может быть пустым                        | Да           |
