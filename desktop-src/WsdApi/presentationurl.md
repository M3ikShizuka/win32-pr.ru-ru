# <a name="presentationurl-element"></a>Пресентатионурл, элемент

Указывает URL-адрес данных представления модели устройства.

## <a name="usage"></a>Использование

``` syntax
<presentationURL/>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                                                   | Описание                                                                                          |
|-----------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| [**сисмоделметадата**](thismodelmetadata.md)<br/> | Определяет производителя и метаданные модели для реализуемого устройства.<br/> <br/> |

|-------------------------------------|---------------|
