# <a name="excludeimport-element"></a>Ексклудеимпорт, элемент

Предотвращает создание операторов импорта для указанных целевых объектов с именем в элементе <WSDL: Import> в WSDL-файле. Это можно использовать для того, чтобы Всдкодежен импортировать хорошо известные целевые объекты, такие как спецификации WS-Addressing и WS-Eventing, несмотря на то, что на эти целевые объекты ссылаются в WSDL.

Этому элементу должно быть присвоено пространство имен с именем в элементе <WSDL: Import>, который необходимо исключить.

## <a name="usage"></a>Использование

``` syntax
<excludeImport/>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.
