# <a name="excludeimport-element"></a>Ексклудеимпорт, элемент

Предотвращает создание операторов импорта для указанных целевых объектов в \<wsdl:import> ЭЛЕМЕНТЕ WSDL-файла. Это можно использовать для того, чтобы Всдкодежен импортировать хорошо известные целевые объекты, такие как спецификации WS-Addressing и WS-Eventing, несмотря на то, что на эти целевые объекты ссылаются в WSDL.

Значение этого элемента должно быть задано для пространства имен, указанного в \<wsdl:import> элементе, который необходимо исключить.

## <a name="usage"></a>Использование

``` syntax
<excludeImport/>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.
