# <a name="operationdeallocator-element"></a>Оператиондеаллокатор, элемент

Указывает дераспределитель типов, используемый функцией-заглушкой операции.

## <a name="usage"></a>Использование

``` syntax
<operationDeallocator
  operation = "character_string"
  deallocator = "character_string"/>
```
