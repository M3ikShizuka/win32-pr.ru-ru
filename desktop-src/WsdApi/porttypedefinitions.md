# <a name="porttypedefinitions-element"></a>Порттипедефинитионс, элемент

Создает константы C для типов портов.

## <a name="usage"></a>Использование

``` syntax
<portTypeDefinitions>
  child elements
</portTypeDefinitions>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы



| Элемент                                                   | Описание                                                                                                                                                                       |
|-----------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Обеспечения**](codename.md)<br/>                   | Указывает имя, используемое для типа порта в созданном коде. По умолчанию имя кода создается на основе полного имени типа порта.<br/> <br/>         |
| [**евентстубфунктион**](eventstubfunction.md)<br/> | Указывает, должны ли ссылки на функции-заглушки включаться в структуры операций в определениях типов портов для операций уведомления.<br/> <br/>        |
| [**операцию**](operation.md)<br/>                 | Указывает операцию, для которой создается код.<br/> <br/>                                                                                                  |
| [**Порта**](porttype.md)<br/>                   | Указывает тип порта, для которого создается код.<br/> <br/>                                                                                                 |
| [**стубфунктион**](stubfunction.md)<br/>           | Указывает, должны ли ссылки на функции-заглушки включаться в структуры операций в определениях типов портов для односторонних и двусторонних операций.<br/> <br/> |



### <a name="child-element-sequence"></a>Последовательность дочерних элементов

``` syntax
(
  portType?, 
  operation*, 
  stubFunction?, 
  eventStubFunction?, 
  codeName?
)
```

## <a name="parent-elements"></a>Родительские элементы



| Элемент                         | Описание                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**File**](file.md)<br/> | Выводит файл из генератора кода.<br/> <br/> |



## <a name="remarks"></a>Примечания

Этот элемент обычно используется в исходных файлах на языке C для предоставления констант типа порта, объявленных с помощью [**порттипедекларатионс**](porttypedeclarations.md).
