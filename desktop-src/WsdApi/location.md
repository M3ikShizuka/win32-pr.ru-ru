# <a name="location-element"></a>location - элемент

Указывает URL-адрес, указанный в качестве указания импорта для WSDL или XSD, на который ссылается другой язык WSDL.

## <a name="usage"></a>Использование

``` syntax
<location/>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                                     | Описание                                                                                                                             |
|---------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**импорсинт**](importhint.md)<br/> | Указывает расположение загрузки для \<wsdl:import> директивы, которая не указывает расположение явным образом.<br/> <br/> |



## <a name="element-information"></a>Сведения об элементе



| Метка | Значение |
|-------------------------------------|---------------|
| Минимальная поддерживаемая система<br/> | Windows Vista |
| Может быть пустым                        | Да           |
