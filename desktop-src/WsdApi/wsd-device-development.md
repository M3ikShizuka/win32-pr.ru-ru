# <a name="wsd-device-development"></a>Разработка устройств WSD

производители устройств и другие разработчики устройств могут создавать устройства, взаимодействующие с клиентами и узлами wsdapi на основе Windows, без использования wsdapi.

Помимо разработки функций устройства, разработчики устройств должны реализовать [профиль устройств для веб-служб](https://specs.xmlsoap.org/ws/2006/02/devprof/) (DPWS).

по возможности используйте WSDAPI, чтобы создать клиентское приложение WSD на основе Windows или ведущее устройство. Использование WSDAPI сокращает сложность разработки и экономит время. Дополнительные сведения см. [в статье Разработка приложений WSD на Windows](wsd-application-development-on-windows.md).
