# <a name="namespace-element"></a>элемент nameSpace

Описывает пространство имен. Это пространство имен может использоваться для создания кода или для обработки \<wsdl:import> директив.

## <a name="usage"></a>Использование

``` syntax
<nameSpace
  uri = "character_string">
  child elements
</nameSpace>
```

## <a name="attributes"></a>Атрибуты



| attribute          | Тип                         | Обязательно       | Описание                                             |
|--------------------|------------------------------|----------------|---------------------------------------------------------|
| **uri**<br/> | Символьная \_ строка<br/> | Да<br/> | Уникальный URI пространства имен.<br/> <br/> |



## <a name="child-elements"></a>Дочерние элементы



| Элемент                                               | Описание                                                                                              |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [**Обеспечения**](codename.md)<br/>               | Имя, используемое для обнаружения пространства имен в созданном коде.<br/> <br/>                  |
| [**макропрефикс**](macroprefix.md)<br/>         | Префикс, который будет использоваться в созданном коде для имен макросов в пространстве имен.<br/> <br/> |
| [**безымян**](name.md)<br/>                       | Полное имя в пространстве имен.<br/> <br/>                                                |
| [**преферредпрефикс**](preferredprefix.md)<br/> | Префикс, с которым должно быть сопоставлено пространство имен, чтобы сделать XML более удобочитаемым.<br/> <br/> |



### <a name="child-element-sequence"></a>Последовательность дочерних элементов

``` syntax
(
  preferredPrefix?, 
  macroPrefix?, 
  codeName?, 
  name*
)
```

| Элемент                                     | Описание                                                                          |
|---------------------------------------------|--------------------------------------------------------------------------------------|

## <a name="remarks"></a>Remarks

Этот элемент может использоваться для предоставления генератору кода дополнительных сведений о пространствах имен, которые встречаются в файлах WSDL и XSD, или для добавления новых пространств имен.

Пространства имен, подразумеваемые отражением типа или указанные в WSDL-и XSD-файлах, автоматически анализируются генератором кода и не обязательно должны быть определены в скрипте явным образом. В некоторых случаях этот элемент можно использовать для добавления дополнительных свойств или имен к пространству имен, на которое ссылается отражение типов или WSDL/XSD. Генератор кода не учитывает это в качестве конфликта.

В следующем XML-коде показан элемент nameSpace (для краткости опущены дочерние элементы).

``` syntax
<nameSpace uri="https://www.example.com/namespace">
  ...
</nameSpace>
```
