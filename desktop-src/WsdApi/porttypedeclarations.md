# <a name="porttypedeclarations-element"></a>Порттипедекларатионс, элемент

Создает объявления констант C для типов портов.

## <a name="usage"></a>Использование

``` syntax
<portTypeDeclarations>
  child elements
</portTypeDeclarations>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы



| Элемент                                   | Описание                                                                                                                                                               |
|-------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Обеспечения**](codename.md)<br/>   | Указывает имя, используемое для типа порта в созданном коде. По умолчанию имя кода создается на основе полного имени типа порта.<br/> <br/> |
| [**операцию**](operation.md)<br/> | Указывает операцию, для которой создается код.<br/> <br/>                                                                                          |
| [**Порта**](porttype.md)<br/>   | Указывает тип порта, для которого создается код.<br/> <br/>                                                                                         |



### <a name="child-element-sequence"></a>Последовательность дочерних элементов

``` syntax
(
  portType?, 
  operation*, 
  codeName?
)
```

## <a name="parent-elements"></a>Родительские элементы



| Элемент                         | Описание                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**File**](file.md)<br/> | Выводит файл из генератора кода.<br/> <br/> |



## <a name="remarks"></a>Комментарии

На объявления типа порта ссылается приложение и большая часть созданного кода. Этот элемент используется для создания включаемых файлов.

|-------------------------------------|---------------|
