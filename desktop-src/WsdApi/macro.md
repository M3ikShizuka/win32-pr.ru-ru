# <a name="macro-element"></a>Macro, элемент

Определяет текст или CDATA для повторного использования элементом [**include**](include.md) .

## <a name="usage"></a>Использование

``` syntax
<macro
  name = "character_string"/>
```

## <a name="attributes"></a>Атрибуты



| attribute           | Тип                         | Обязательно       | Описание                                   |
|---------------------|------------------------------|----------------|-----------------------------------------------|
| **name**<br/> | Символьная \_ строка<br/> | Да<br/> | Имя макроса.<br/> <br/> |



## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                                     | Описание                                                                         |
|---------------------------------------------|-------------------------------------------------------------------------------------|
| [**всдкодежен**](wsdcodegen.md)<br/> | Корневой элемент XML-файла скрипта генератора кода WSDAPI.<br/> <br/> |



## <a name="remarks"></a>Примечания

Всдкодежен определяет макрос с именем **донотмодифи**. При включении этого макроса созданный код содержит блок комментариев с высокой степенью видимости, который указывает разработчикам не изменять созданный код.

В следующем коде XML показано, как включить макрос **донотмодифи** . Этот XML-код можно добавить в XML-файл конфигурации для Всдкодежен.

``` syntax
<include macro="DoNotModify">
```
