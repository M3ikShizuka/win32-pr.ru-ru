# <a name="deallocator-element"></a>элемент распределителя

Указывает тип дераспределителя, используемый функцией-заглушкой.

## <a name="usage"></a>Использование

``` syntax
<deallocator/>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                                               | Описание                                                                                   |
|-------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**стубдефинитионс**](stubdefinitions.md)<br/> | Создает реализации для функций-заглушек для операций с типом порта.<br/> <br/> |



## <a name="remarks"></a>Комментарии

Тип дераспределителя должен быть заключен в пару &lt; тегов дераспределителя &gt; </deallocator> . Следующие строки являются допустимыми значениями для дераспределения:

-   нет
-   всдфрилинкедмемори
-   CoTaskMemFree
-   свободный
-   удалить
-   делетеаррай
-   Выпуск

## <a name="element-information"></a>Сведения об элементе



| Метка | Значение |
|-------------------------------------|---------------|
| Минимальная поддерживаемая система<br/> | Windows Vista |
| Может быть пустым                        | Да           |
