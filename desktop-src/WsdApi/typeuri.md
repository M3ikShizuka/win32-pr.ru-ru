# <a name="typeuri-element"></a>Типеури, элемент

Указывает тип, включаемый из XSD-файла.

## <a name="usage"></a>Использование

``` syntax
<typeUri
  type = "character_string"
  uri = "character_string"/>
```

## <a name="attributes"></a>Атрибуты



| attribute           | Тип                         | Обязательно       | Описание                                                            |
|---------------------|------------------------------|----------------|------------------------------------------------------------------------|
| **type**<br/> | Символьная \_ строка<br/> | Да<br/> | Имя типа.<br/> <br/>                           |
| **uri**<br/>  | Символьная \_ строка<br/> | Да<br/> | Пространство имен типа. Должен быть допустимым URI.<br/> <br/> |



## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                       | Описание                                                                       |
|-------------------------------|-----------------------------------------------------------------------------------|
| [**схеме**](xsd.md)<br/> | Указывает XSD-файл для обработки сведений о контракте.<br/> <br/> |



## <a name="element-information"></a>Сведения об элементе



| Метка | Значение |
|-------------------------------------|---------------|
| Минимальная поддерживаемая система<br/> | Windows Vista |
| Может быть пустым                        | Да           |
