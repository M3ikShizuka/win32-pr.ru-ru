# <a name="using-web-services-on-devices"></a>Использование веб-служб на устройствах

API веб-служб Майкрософт на устройствах (WSDAPI) поддерживает реализацию управляемых клиентом устройств и служб, а также узлы устройств, соответствующие [профилю устройств для веб-служб](https://specs.xmlsoap.org/ws/2006/02/devprof/) (DPWS). WSDAPI может использоваться для разработки как клиентских, так и серверных реализаций (устройств).

По возможности используйте WSDAPI, чтобы создать клиентское приложение WSD на основе Windows или ведущее устройство. Использование WSDAPI сокращает сложность разработки и экономит время. Дополнительные сведения см. [в статье Разработка приложений WSD в Windows](wsd-application-development-on-windows.md) и [WSDAPI Development Tools](wsdapi-development-tools.md).

Можно также создавать устройства WSD, которые не работают под управлением Windows и совместимы с клиентами и компьютерами WSD на основе Windows. Эти устройства должны соответствовать определенным требованиям для обеспечения взаимодействия. Дополнительные сведения см. в разделе [Разработка устройств WSD](wsd-device-development.md).
