# <a name="using-web-services-on-devices"></a>Использование веб-служб на устройствах

API веб-служб Майкрософт на устройствах (WSDAPI) поддерживает реализацию управляемых клиентом устройств и служб, а также узлы устройств, соответствующие [профилю устройств для веб-служб](https://specs.xmlsoap.org/ws/2006/02/devprof/) (DPWS). WSDAPI может использоваться для разработки как клиентских, так и серверных реализаций (устройств).

по возможности используйте WSDAPI, чтобы создать клиентское приложение WSD на основе Windows или ведущее устройство. Использование WSDAPI сокращает сложность разработки и экономит время. дополнительные сведения см. [в статье разработка приложений WSD на Windows](wsd-application-development-on-windows.md) и [WSDAPI development tools](wsdapi-development-tools.md).

можно также создать устройства WSD, которые не работают Windows и совместимы с клиентами WSD на основе Windows и узлами устройств. Эти устройства должны соответствовать определенным требованиям для обеспечения взаимодействия. Дополнительные сведения см. в разделе [Разработка устройств WSD](wsd-device-development.md).
