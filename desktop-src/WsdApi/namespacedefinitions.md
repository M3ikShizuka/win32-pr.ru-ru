# <a name="namespacedefinitions-element"></a>Намеспацедефинитионс, элемент

Создает определения языка C для таблиц пространств имен.

## <a name="usage"></a>Использование

``` syntax
<namespaceDefinitions/>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                         | Описание                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**File**](file.md)<br/> | Выводит файл из генератора кода.<br/> <br/> |



## <a name="remarks"></a>Комментарии

Этот элемент обычно используется в исходных файлах на языке C для предоставления таблиц пространства имен, объявленных с помощью [**намеспацедекларатионс**](namespacedeclarations.md).

## <a name="element-information"></a>Сведения об элементе



|                                     |               |
|-------------------------------------|---------------|
| Минимальная поддерживаемая система<br/> | Windows Vista |
| Может быть пустым                        | Да           |
