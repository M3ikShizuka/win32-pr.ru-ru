# <a name="hostedservice-element"></a>элемент hostedService

Определяет службу, размещаемую в функции конструктора узла.

## <a name="usage"></a>Использование

``` syntax
<hostedService>
  child elements
</hostedService>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы



| Элемент                                     | Описание                                                                                                                                                               |
|---------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Обеспечения**](codename.md)<br/>     | Указывает имя, используемое для типа порта в созданном коде. По умолчанию имя кода создается на основе полного имени типа порта.<br/> <br/> |
| [**Порта**](porttype.md)<br/>     | Тип порта, для которого создается код.<br/> <br/>                                                                                                       |
| [**proxyClass**](proxyclass.md)<br/> | Имя класса, создаваемого из функции построителя.<br/> <br/>                                                                                                      |
| [**ServiceID**](serviceid.md)<br/>   | URI, представляющий идентификатор службы.<br/> <br/>                                                                                                      |



### <a name="child-element-sequence"></a>Последовательность дочерних элементов

``` syntax
codeName(
  ServiceID, 
  proxyClass, 
  portType+
)
```

## <a name="parent-elements"></a>Родительские элементы



| Элемент                         | Описание                                                                  |
|---------------------------------|------------------------------------------------------------------------------|
| [**файл**](file.md)<br/> | Спецификация выходного файла для генератора кода.<br/> <br/> |



## <a name="element-information"></a>Сведения об элементе



| Метка | Значение |
|-------------------------------------|---------------|
| Минимальная поддерживаемая система<br/> | Windows Vista |
| Может быть пустым                        | Нет            |
