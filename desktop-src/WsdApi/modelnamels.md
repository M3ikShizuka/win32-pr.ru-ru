# <a name="modelnamels-element"></a>Моделнамелс, элемент

Указывает локализованную версию имени устройства.

## <a name="usage"></a>Использование

``` syntax
<modelNameLS
  Language = "language identifier string"
  Data = "localized model name string"/>
```

## <a name="attributes"></a>Атрибуты



| attribute               | Тип                                   | Обязательно       | Описание                                                      |
|-------------------------|----------------------------------------|----------------|------------------------------------------------------------------|
| **Данные**<br/>     | Строка имени локализованной модели<br/> | Да<br/> | Имя локализованной модели.<br/> <br/>                 |
| **Язык**<br/> | Строка идентификатора языка<br/>  | Да<br/> | Язык имени локализованной модели.<br/> <br/> |



## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                                                   | Описание                                                                                          |
|-----------------------------------------------------------|------------------------------------------------------------------------------------------------------|
| [**сисмоделметадата**](thismodelmetadata.md)<br/> | Определяет производителя и метаданные модели для реализуемого устройства.<br/> <br/> |



## <a name="element-information"></a>Сведения об элементе



| Метка | Значение |
|-------------------------------------|---------------|
| Минимальная поддерживаемая система<br/> | Windows Vista |
| Может быть пустым                        | Да           |
