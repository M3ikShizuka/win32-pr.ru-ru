# <a name="codename-element"></a>элемент кодового элемента

Определяет имя, которое будет использоваться для определения пространства имен в созданном коде.

## <a name="usage"></a>Использование

``` syntax
<codeName/>
```

## <a name="attributes"></a>Атрибуты

Атрибуты отсутствуют.

## <a name="child-elements"></a>Дочерние элементы

Нет дочерних элементов.

## <a name="parent-elements"></a>Родительские элементы



| Элемент                                                         | Описание                                                              |
|-----------------------------------------------------------------|--------------------------------------------------------------------------|
| [**Имен**](namespace.md)<br/>                       | Пространство имен, используемое для создания кода.<br/> <br/>       |
| [**порттипедекларатионс**](porttypedeclarations.md)<br/> | Создает объявления констант C для типов портов.<br/> <br/> |
| [**порттипедефинитионс**](porttypedefinitions.md)<br/>   | Создает константы C для типов портов.<br/> <br/>             |



## <a name="remarks"></a>Примечания

Этот элемент переопределяет имя кода по умолчанию, используемое для созданного кода. По умолчанию созданный код создает имя кода из URI или полного имени.
