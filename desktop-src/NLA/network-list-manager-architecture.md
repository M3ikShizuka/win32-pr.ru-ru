# <a name="network-list-manager-architecture"></a>Архитектура диспетчера списка сетей

Диспетчер списка сетей запускается как служба в контексте Svchost.exe и запускается во время процедуры загрузки компьютера. Служба диспетчера списков сетей поддерживает таблицу доступных сетей и сетевых атрибутов, получаемых приложениями через API диспетчера списка сетей. Диспетчер списков сетей предоставляет основные функции для фильтрации и запроса службы диспетчера списков сетей для конкретных сетей и получения атрибутов этих сетей. На следующей схеме показана архитектура диспетчера списка сетей и взаимодействие между службой диспетчера списка сетей и клиентским приложением.
