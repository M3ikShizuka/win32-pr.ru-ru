# <a name="nlm-structures"></a>Структуры NLM

Следующие структуры поддерживают API диспетчера списка сетей.



| Структура                                                           | Описание                                                          |
|---------------------------------------------------------------------|----------------------------------------------------------------------|
| [**\_ \_ сведения о имитируемом ПРОфиле NLM \_**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_simulated_profile_info) | Представляет сведения о профиле, используемые при моделировании затрат.    |
| [**\_SOCKADDR NLM**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_sockaddr)                               | Представляет адрес назначения IPv4/IPv6.                        |
| [**\_данные об использовании NLM \_**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_usage_data)                          | Представляет сведения, указывающие на использование данных в плане данных. |
| [**\_ \_ состояние плана данных \_ NLM**](https://www.bing.com/search?q=**NLM\_DATA\_PLAN\_STATUS**)             | Представляет текущее состояние плана данных.                        |
