## <a name="remarks"></a>Комментарии

Поставщик WinNT, предоставляемый корпорацией Майкрософт, не поддерживает все методы свойств [**IADsUser**](/windows/desktop/api/Iads/nn-iads-iadsuser) , представленные выше. Однако поставщик поддерживает другие свойства, доступ к которым можно получить с помощью метода [**iAds:: Get**](/windows/desktop/api/Iads/nf-iads-iads-get) или [**iAds::P UT**](/windows/desktop/api/Iads/nf-iads-iads-put) . Дополнительные сведения и список неподдерживаемых свойств и примеры кода см. в разделе [объект пользователя WinNT](winnt-user-object.md) в [поставщике ADSI WinNT](adsi-winnt-provider.md).

Дополнительные сведения о функциях поставщика LDAP ADSI, характерных для объекта класса User, см. в разделе [объект пользователя LDAP](ldap-user-object.md) в [поставщике ADSI LDAP](adsi-ldap-provider.md). Раздел содержит [**IADsUser**](/windows/desktop/api/Iads/nn-iads-iadsuser), а также примеры кода для управления учетной записью пользователя.

## <a name="examples"></a>Примеры

В следующем примере кода показано, как выполнить привязку к объекту учетной записи пользователя и получить полное имя пользователя.


```VB
Dim usr As IADsUser
Dim sFullName as String

On Error GoTo Cleanup
Set usr = GetObject("WinNT://Fabrikam/JeffSmith,user")
sFullName = usr.FullName

Cleanup:
    If (Err.Number<>0) Then
        MsgBox("An error has occurred. " & Err.Number)
    End If

    Set usr = Nothing
```



В следующем примере кода показано, как выполнить привязку к объекту учетной записи пользователя и получить полное имя пользователя.


```C++
IADsUser *GetUserObject(LPWSTR uPath)
{
    IADsUser *pUser;
    HRESULT hr = ADsGetObject(uPath,IID_IADsUser,(void**)&pUser);
    if (FAILED(hr)) {return NULL;}
    BSTR bstr;
    hr = pUser->get_FullName(&bstr);
    printf("User: %S\n", bstr);
    SysFreeString(bstr);
    return pUser;
}
```
