# <a name="cpropscpp"></a>КПРОПС. CPP

В примере компонента поставщика пример реализации кэша свойств можно найти в кпропс. cpp. Поддерживаемые методы перечислены в следующей таблице.



| Метод                                           | Описание                                                                                                         |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| **Кпропертикаче:: AddProperty**                  | Расширьте кэш свойств, добавив новый.                                                                      |
| **Кпропертикаче:: упдатепроперти**               | Найдите свойство, освободите его содержимое и используйте вместо него новые значения. Затем пометьте измененный кэш для этого свойства. |
| **Кпропертикаче:: финдпроперти**                 | Поиск этого свойства по имени; Сохраните индекс.                                                                      |
| **Кпропертикаче:: Property**                  | Найдите свойство в кэше, если доступно, в противном случае **— Call.** Задайте индекс и скопируйте новые значения.  |
| **Кпропертикаче::p утпроперти**                  | Найдите свойство. Бесплатная и поставьте новые значения.                                                       |
| **Кпропертикаче:: Кпропертикаче**               | Стандартный конструктор.                                                                                               |
| **Кпропертикаче:: ~ Кпропертикаче**              | Стандартный деструктор.                                                                                                |
| **Кпропертикаче:: креатепропертикаче**          | Создайте кэш.                                                                                                   |
| **Кпропертикаче:: унбаунджетпроперти**           | Найдите свойство в кэше и задайте для него эти значения.                                                          |
| **Кпропертикаче:: Сампледсмаршаллпропертиес**   | Маршалирует данные и значения свойств.                                                                                   |
| **Кпропертикаче:: маршаллпроперти**             | Маршалирует свойство.                                                                                                 |
| **Кпропертикаче:: Сампледсунмаршаллпропертиес** | Распаковать данные и значения свойств.                                                                                 |
| **Кпропертикаче:: унмаршаллпроперти**           | Распаковать свойство.                                                                                               |
