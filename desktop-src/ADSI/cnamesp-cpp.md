# <a name="cnamespcpp"></a>КНАМЕСП. CPP

В примере компонента поставщика код, управляющий временем существования объекта пространства имен компонента поставщика в примере, находится в кнамесп. cpp. Поддерживаемые методы перечислены в следующей таблице.



| Метод                                                   | Описание                                            |
|----------------------------------------------------------|--------------------------------------------------------|
| **Ксампледснамеспаце:: Ксампледснамеспаце**               | Стандартный конструктор.                                  |
| **Ксампледснамеспаце:: ~ Ксампледснамеспаце**              | Стандартный деструктор.                                   |
| Стандартные методы [**iAds**](/windows/desktop/api/Iads/nn-iads-iads) .                   |                                                        |
| Стандартные методы [**иадсконтаинер**](/windows/desktop/api/Iads/nn-iads-iadscontainer) . |                                                        |
| **Ксампледснамеспаце:: Аллокатенамеспацеобжект**          | Создайте объект пространства имен и загрузите его данные типа.      |
| **Ксампледснамеспаце:: Креатенамеспаце**                  | Выделение, инициализация и проверка объекта пространства имен. |
| **Ксампледснамеспаце:: QueryInterface**                   | Проверьте указанный IID для этого объекта.                   |
