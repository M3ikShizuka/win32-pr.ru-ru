-   [Изменение пользователя не может изменить пароль (поставщик LDAP)](modifying-user-cannot-change-password-ldap-provider.md)

Проще читать и изменять параметр User не может изменять пароль с помощью поставщика WinNT. Дополнительные сведения см. в разделе [пользователь не может изменить пароль (поставщик WinNT)](winnt-user-cannot-change-password.md).
