# <a name="memorycpp"></a>Свободной. CPP

В примере компонента поставщика пример кода, отображающий выделение памяти и освобождение ресурсов, находится в памяти. cpp. Поддерживаемые подпрограммы перечислены в следующей таблице.



| Элемент                | Описание                                                           |
|---------------------|-----------------------------------------------------------------------|
| **аллокпровмем**    | Выделение указанной памяти.                                            |
| **фрипровмем**     | Указана свободная память.                                                |
| **реаллокпровмем**  | Выделение непрерывной памяти.                                           |
| **аллокпровстр**    | Выделение строки LPWSTR.                                            |
| **фрипровстр**     | Свободная строка, если она еще не освобождена.                                     |
| **реаллокпровстр**  | Выделение непрерывной памяти.                                           |
| **проваллокстринг** | Проверяет строку и первый параметр. Если ОК, выполнит выделение. |
