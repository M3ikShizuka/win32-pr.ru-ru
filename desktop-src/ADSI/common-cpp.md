# <a name="commoncpp"></a>Разделяем. CPP

В примере компонента поставщика общий пример кода для всех универсальных объектов находится в Common. cpp. Объекты наследуют эти методы и свойства с помощью правил наследования C++. Эти методы включают в себя создание строк ADsPath, типы методов данных и другие процедуры обслуживания.
