# <a name="scenario-the-fabrikam-corporation"></a>Сценарий: Корпорация Fabrikam

Компания Fabrikam обновила свой домен с Windows NT 4,0 на Windows 2000. Во время установки Джо Ворден, администратор сети, получил запрос на указание DNS-имени домена. Джо ввел «fabrikam.com». Существующие пользователи, группы и компьютеры переносятся в Active Directory в новый домен: fabrikam.com.
