# <a name="packcpp"></a>Распаковк. CPP

В примере компонента поставщика пример кода для упаковки и распаковки типов данных в вариантах находится в виде файла Pack. cpp. Следующие функции используются, когда свойства загружаются в кэш свойств и удаляются из него:

-   **паккстрингинвариант**
-   **унпаккстрингинвариант**
-   **пакклонгинвариант**
-   **унпакклонгфромвариант**
-   **паккдатеинвариант**
-   **унпаккдатеинвариант**
-   **Пакквариант \_ булинвариант**
-   **Унпакквариант \_ булфромвариант**
-   **пакквариантинвариант**
-   **унпакквариантфромвариант**
