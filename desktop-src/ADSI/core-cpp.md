# <a name="corecpp"></a>Центральный. CPP

В примере компонента поставщика пример кода, используемый для инициализации основных свойств базового объекта, находится в ядре. cpp. Поддерживаемые методы перечислены в следующей таблице.



| Метод                                   | Описание                                                                                              |
|------------------------------------------|----------------------------------------------------------------------------------------------------------|
| **Ккореадсобжект:: Инитиализекореобжект** | Инициализируйте базовые свойства [**iAds**](/windows/desktop/api/Iads/nn-iads-iads) для этого объекта, что удобно для создания объекта. |
| **Ккореадсобжект:: Ккореадсобжект**       | Стандартный создатель.                                                                                        |
| **Ккореадсобжект:: ~ Ккореадсобжект**      | Стандартный деструктор.                                                                                     |
