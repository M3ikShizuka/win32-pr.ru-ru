# <a name="errors-and-error-trapping"></a>Ошибки и перехват ошибок

ADSI создает множество различных ошибок, от интерфейсов ADSI до COM, связанных со стандартными ошибками Win32. Этот раздел руководства содержит стратегии обработки ошибок в сценариях ADSI и помогает понять различные коды ошибок. Рассматриваются следующие темы.

-   [Перехват ошибок в ADSI](how-to-trap-adsi-errors.md)
-   [Распространенные ошибки](common-errors.md)
