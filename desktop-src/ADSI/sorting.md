# <a name="sorting-adsi"></a>Сортировка (ADSI)

Клиент может запросить сервер для сортировки результирующего набора. Можно указать:

-   Порядок сортировки: можно указать порядок сортировки по возрастанию или по убыванию.
-   Атрибуты сортировки: в строке сортировки можно указать несколько атрибутов. Например, выполните сортировку по фамилии, а затем — по имени.

При указании порядка сортировки следует попытаться использовать один из индексированных атрибутов. В противном случае сервер должен вычислить полный результирующий набор перед отправкой клиенту. Это справедливо, даже если вы запросили страничный поиск и указали размер страницы.

> [!Note]  
> Не все серверы каталогов поддерживают сортировку нескольких атрибутов или порядок сортировки.
