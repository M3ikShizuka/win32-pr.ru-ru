# <a name="cschobjcpp"></a>КСЧОБЖ. CPP

В примере компонента поставщика пример кода, который управляет временем существования объектов схемы, находится в ксчобж. cpp. Поддерживаемые методы перечислены в следующей таблице.



| Метод                                                   | Описание                                    |
|----------------------------------------------------------|------------------------------------------------|
| **Ксампледссчема:: Креатесчема**                        | Стандартный конструктор.                          |
| **Ксампледссчема:: ~ Ксампледссчема**                    | Стандартный деструктор.                           |
| Стандартные методы [**iAds**](/windows/desktop/api/Iads/nn-iads-iads) .                   |                                                |
| Стандартные методы [**иадсконтаинер**](/windows/desktop/api/Iads/nn-iads-iadscontainer) . |                                                |
| **Ксампледссчема:: QueryInterface**                      | Проверьте заданный идентификатор интерфейса для этого объекта.  |
| **Ксампледссчема:: Аллокатесчема**                      | Создайте объект схемы и загрузите его данные типа. |
