# <a name="who-will-use-active-directory-service-interfaces"></a>Кто Будет использовать Active Directory интерфейсы служб?

Сетевые администраторы будут использовать ADSI для автоматизации распространенных административных задач, таких как добавление пользователей и групп, Управление принтерами и настройка разрешений для сетевых ресурсов.

Независимые поставщики программного обеспечения (ISV) и разработчики конечных пользователей будут использовать ADSI для "Directory Enable" своих продуктов и приложений. Службы могут публиковать себя в каталоге, клиенты могут использовать каталог для поиска служб, и оба они могут использовать каталог для поиска других интересующих объектов и управления ими. Так как интерфейсы служб Active Directory не зависят от базовых служб каталогов, продукты и приложения, включенные в каталог, будут успешно взаимодействовать в нескольких средах сети и каталогов.
