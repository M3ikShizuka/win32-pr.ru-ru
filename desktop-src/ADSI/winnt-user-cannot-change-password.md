# <a name="user-cannot-change-password-winnt-provider"></a>Пользователь не может изменить пароль (поставщик WinNT)

Возможность изменения пароля пользователем — это разрешение, которое может быть предоставлено или запрещено. Дополнительные сведения о программном чтении и изменении этого разрешения с помощью поставщика WinNT см. в следующих статьях:

-   [Чтение пользователя не может изменить пароль (поставщик WinNT)](reading-user-cannot-change-password-winnt-provider.md)
-   [Изменение пользователя не может изменить пароль (поставщик WinNT)](modifying-user-cannot-change-password-winnt-provider.md)

Атрибут **усерфлагс** объекта User доступен только при использовании поставщика WinNT.

Также можно прочитать и изменить это разрешение с помощью поставщика LDAP. Дополнительные сведения см. в разделе [пользователь не может изменить пароль (поставщик LDAP)](user-cannot-change-password.md).
