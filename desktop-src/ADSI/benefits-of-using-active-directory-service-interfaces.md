# <a name="benefits-of-using-active-directory-service-interfaces"></a>Преимущества использования интерфейсов служб Active Directory

ADSI предоставляет системным администраторам множество преимуществ, как описано в следующей таблице.



| Компонент                   | Преимущество                                                                                                                                                                                                                                                                                                                             |
|---------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Открыть                      | Любой поставщик каталога может реализовать поставщик ADSI; Пользователи получают свободу выбора в службах каталогов без ущерба для управляемости.                                                                                                                                                                                        |
| Независимо от DS            | Административные приложения тесно привязаны к службе каталогов данного поставщика. Администраторы и другие приложения с поддержкой каталогов могут быть разработаны без необходимости понимания интерфейсов API каталога конкретного поставщика. Одно и то же приложение может работать с несколькими каталогами. Затраты времени на разработку и поддержку сокращаются. |
| Поддержка нескольких языков | Объекты ADSI обеспечивают простой доступ к службам каталогов через объектную модель компонента. приложения COM могут быть написаны на таких языках, как microsoft Visual Basic, microsoft Visual Basic scripting Edition (VBScript) и C/C++.                                                                                             |
| Простая модель программирования  | ADSI состоит из небольшого, простого в изучении набора интерфейсов.                                                                                                                                                                                                                                                                          |
| Поддержкой скриптов                | любой язык, совместимый с автоматизацией (например, Visual Basic, VBScript, Perl, рекскс и др.), можно использовать для разработки приложений службы каталогов. Администраторы и разработчики могут использовать уже известные инструменты. Повышение производительности — снижение затрат времени на разработку и поддержку.                               |
| Функционально функционально         | Независимые поставщики программного обеспечения и сложные конечные пользователи могут разрабатывать серьезные приложения, используя те же модели ADSI, которые используются для простых сценариев администрирования.                                                                                                                                                                         |
| возможности расширения.                | Поставщики каталогов, независимые поставщики программного обеспечения и конечные пользователи могут расширять интерфейсы ADSI с помощью новых объектов и функций, чтобы добавить ценность или удовлетворить уникальные потребности.                                                                                                                                                                                                          |
| OLE DB с поддержкой              | Интерфейсы ADSI предоставляют интерфейс OLE DB, чтобы программисты, знакомые с программированием баз данных с помощью OLE DB, могли быстро работать.                                                                                                                                                                                                  |
