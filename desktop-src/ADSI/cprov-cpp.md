# <a name="cprovcpp"></a>КПРОВ. CPP

В примере компонента поставщика пример кода, используемый для реализации объекта поставщика ADs, находится в кпров. cpp. Поддерживаемые методы перечислены в следующей таблице.



| Элемент                                      | Описание                                         |
|-------------------------------------------|-----------------------------------------------------|
| **Ксампледспровидер:: Ксампледспровидер**  | Стандартный создатель.                                   |
| **Ксампледспровидер:: ~ Ксампледспровидер** | Стандартный деструктор.                                |
| **Ксампледспровидер:: Create**             | создание объекта.                                  |
| **Ксампледспровидер:: QueryInterface**     | Проверьте наличие поддерживаемых интерфейсов.                     |
| **Ксампледспровидер::P Арседисплайнаме**   | Разрешите путь.                                   |
| **Ксампледспровидер:: Ресолвепаснаме**    | Получите объект и создайте для него специальное имя указателя. |
