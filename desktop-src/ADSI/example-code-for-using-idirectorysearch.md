# <a name="example-code-for-using-idirectorysearch"></a>Пример кода для использования IDirectorySearch

Пакет SDK для платформы включает в себя пример кода, Дссрч, который предоставляет простое приложение командной строки, использующее интерфейс [**IDirectorySearch**](/windows/desktop/api/Iads/nn-iads-idirectorysearch) для запроса к службе каталогов. Полный пример кода включен в пакет Platform SDK в каталоге Samples \\ нетдс \\ ADSI \\ Samples \\ General \\ дссрч.
