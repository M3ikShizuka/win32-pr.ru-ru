# <a name="generic-com-error-codes"></a>Универсальные коды ошибок COM

Универсальные коды ошибок COM выражают состояние операции, созданное всеми модулями COM. В следующей таблице приведены некоторые примеры таких кодов ошибок на любой платформе Win32.



| Код ошибки     | Шестнадцатеричное значение  | Описание              |
|----------------|------------|--------------------------|
| \_прервать       | 0x80004004 | Операция аварийно завершена.       |
| \_Ошибка E        | 0x80004005 | Незаданная ошибка.       |
| \_интерфейс E | 0x80004002 | Интерфейс не поддерживается. |
| E \_ нотимпл     | 0x80004001 | Не реализован.         |
| \_указатель E     | 0x80004003 | Недопустимый указатель.         |
| \_непредвиденная E  | 0x8000FFFF | Разрушительный сбой.    |
