# <a name="guidcpp"></a>Устройства. CPP

В примере компонента поставщика идентификаторы GUID, необходимые для объектов примера поставщика, находятся в GUID. cpp. Идентификаторы GUID определяются для следующих элементов:

-   \_САМПЛЕДСНАМЕСПАЦЕ CLSID
-   \_САМПЛЕДСПРОВИДЕР CLSID
-   Идентификатор LIBID \_ сампледсоле
-   \_САМПЛЕДСЖЕНОБЖЕКТ CLSID
-   \_САМПЛЕДССЧЕМА CLSID
-   \_САМПЛЕДСКЛАСС CLSID
-   \_САМПЛЕДСПРОПЕРТИ CLSID
