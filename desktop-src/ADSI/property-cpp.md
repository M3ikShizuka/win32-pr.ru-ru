# <a name="propertycpp"></a>Свойства. CPP

В примере компонента поставщика примеры кода для хранения различных типов данных в **вариантных** типах данных, чтобы их можно было сохранить как свойства объекта в свойстве. cpp. Поддерживаемые подпрограммы перечислены в следующей таблице.



| Подпрограмма                                                           | Описание                  |
|-------------------------------------------------------------------|------------------------------|
| **получить \_ \_** Свойство " / **помещает \_ BSTR \_** " свойства BSTR                   | Для строковых свойств.       |
| **получить \_ \_** Длинное / **\_ \_ свойство размещения** Long                   | Для целочисленных свойств типа long. |
| **получить \_ \_** / **\_ \_ Свойство даты размещения** свойства Date                   | Для свойств формата даты.  |
| **получить \_ Значение типа Variant \_ bool \_** логическое / **\_ \_ \_ свойство** | Для логических свойств.      |
| **получить \_ \_** Свойство варианта / **размещения \_ \_** варианта свойства             | Для **вариантов** свойств.  |
