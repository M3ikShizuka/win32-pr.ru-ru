# <a name="cclsobjcpp"></a>ККЛСОБЖ. CPP

В примере компонента поставщика функции для объекта класса схемы содержатся в кклсобж. cpp.

Класс **ксампледскласс** определен в этом файле. **Ксампледскласс** определяется методами и свойствами, перечисленными в следующей таблице.



| Метод                      | Описание                                                                                                                                                                                                |
|-----------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **ксампледскласс**          | Стандартный конструктор.                                                                                                                                                                                      |
| **~ Ксампледскласс**         | Стандартный деструктор.                                                                                                                                                                                       |
| **креатекласс**             | Создайте объект класса схемы ADs. Определение атрибутов уточняющих запросов путем вызова **сампледсжетклассдефинитион**.                                                                                                 |
| **креатекласс**             | Создайте объект класса схемы с учетом определений атрибутов, задав известные атрибуты, такие как перечисленные в [**иадскласс:: мандаторяттрибутес**](iadsclass-property-methods.md).                     |
| **аллокатеклассобжект**     | Создайте объект класса схемы и загрузите его данные типа.                                                                                                                                                       |
| **QueryInterface**          | Возвращает запрошенный указатель интерфейса, если он доступен.                                                                                                                                                      |
| Стандартные методы IADs.      | Стандартные методы интерфейса [**iAds**](/windows/desktop/api/Iads/nn-iads-iads) , входящие в этот файл.                                                                                                                                     |
| Стандартные методы Иадскласс. | Стандартные методы интерфейса [**иадскласс**](/windows/desktop/api/Iads/nn-iads-iadsclass) , входящие в этот файл.                                                                                                                           |
| **креатепропертилист**      | Создайте список свойств, связанных с этим классом схемы, вызвав **креатепропертентри**.                                                                                                          |
| **креатепропертентри**     | Создайте один объект Property в этом классе схемы.                                                                                                                                                           |
| **фрипропертентри**       | Освободите запись, созданную в **креатепропертентри**.                                                                                                                                                            |
| **макевариантфромпроплист** | Создайте массив вариантов из списка свойств, созданного с помощью **креатепропертилист**. Может использоваться в реализации [**иадскласс:: мандаторяттрибутес**](iadsclass-property-methods.md) и т. д. |
