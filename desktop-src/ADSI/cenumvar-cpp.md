# <a name="cenumvarcpp"></a>ЦЕНУМВАР. CPP

В примере компонента поставщика Базовая реализация для производных классов Ксксксенумвариант можно найти в ценумвар. cpp. Связанные методы перечислены в следующей таблице.



| Метод                                          | Описание                                                                           |
|-------------------------------------------------|---------------------------------------------------------------------------------------|
| **Ксампледсенумвариант:: Ксампледсенумвариант**  | Стандартный конструктор.                                                                 |
| **Ксампледсенумвариант:: ~ Ксампледсенумвариант** | Стандартный деструктор.                                                                  |
| **Ксампледсенумвариант:: QueryInterface**        | Стандартная реализация [**IUnknown:: QueryInterface**](/windows/win32/api/unknwn/nf-unknwn-iunknown-queryinterface(q)) . |
| **Ксампледсенумвариант:: AddRef**                | Стандартная реализация [**IUnknown:: AddRef**](/windows/win32/api/unknwn/nf-unknwn-iunknown-addref) .                 |
| **Ксампледсенумвариант:: Release**               | Стандартная реализация [**IUnknown:: Release**](/windows/win32/api/unknwn/nf-unknwn-iunknown-release) .               |
| **Ксампледсенумвариант:: Skip**                  | Стандартная реализация **иенумкскскскс:: Skip** .                                          |
| **Ксампледсенумвариант:: Reset**                 | Стандартная реализация **иенумкскскскс:: Reset** .                                         |
| **Ксампледсенумвариант:: Clone**                 | Стандартная реализация **иенумкскскскс:: Clone** .                                         |
