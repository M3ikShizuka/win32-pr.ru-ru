# <a name="cenumschcpp"></a>ЦЕНУМСЧ. CPP

В примере компонента поставщика перечисление объекта схемы использует методы из ценумсч. cpp, перечисленные в следующей таблице.



| Метод                                        | Описание                                                                                                          |
|-----------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| **Ксампледссчемаенум:: Create**               | Создайте объект, чтобы разрешить перечисление объекта класса схемы ADSI.                                                |
| **Ксампледссчемаенум:: Ксампледссчемаенум**  | Стандартный конструктор.                                                                                                |
| **Ксампледссчемаенум:: ~ Ксампледссчемаенум** | Стандартный деструктор.                                                                                                 |
| **Ксампледссчемаенум:: Next**                 | Извлечение указанного числа элементов из указанного объекта схемы.                                          |
| **Ксампледссчемаенум:: Енумобжектс**          | Управление получением указателей интерфейсов на объекты указанного типа объекта.                               |
| **Ксампледссчемаенум:: Енумобжектс**          | Управление получением указателей интерфейса на объекты типа объекта по умолчанию.                                  |
| **Ксампледссчемаенум:: Енумклассес**          | Управление получением указателей интерфейса только на объекты класса схемы, содержащиеся в этом объекте.                  |
| **Ксампледссчемаенум:: Жетклассобжект**       | Получение определения следующего класса схемы; Если он найден, создайте объект класса схемы и возвратите указатель интерфейса. |
| **Ксампледссчемаенум:: Енумпропертиес**       | Управление получением указателей интерфейса на объекты свойств, которые содержатся только в этом объекте.                      |
| **Ксампледссчемаенум:: Жетпропертйобжект**    | Получение определения следующего свойства; Если он найден, создайте объект класса схемы и возвратите указатель интерфейса.     |
| **Ксампледссчемаенум:: Енумсинтаксес**         | Управление получением указателей интерфейса на объекты синтаксиса, которые содержатся только в этом объекте.                        |
| **Ксампледссчемаенум:: Жетсинтаксобжект**      | Получите следующее определение синтаксиса. Если он найден, создайте объект класса схемы и возвратите указатель интерфейса.       |
