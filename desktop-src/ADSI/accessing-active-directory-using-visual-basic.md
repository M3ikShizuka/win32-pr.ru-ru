# <a name="accessing-active-directory-using-visual-basic"></a>Доступ к Active Directory с помощью Visual Basic

Одной из наиболее эффективных функций, которые стали доступны в операционной системе Microsoft Windows 2000, является служба Microsoft Active Directory Directory. При входе в домен Windows 2000 Active Directory помещается в действие. для поиска ближайшего цветового принтера в здании можно использовать Active Directory. Он может использоваться для поиска в адресной книге или для поиска пользователей, работающих в здании 40. С Active Directory можно использовать смарт-карту для входа в домен Windows 2000. Можно даже присоединять данные из программного обеспечения базы данных Microsoft SQL Server и Active Directory.

Многие другие технологии Майкрософт, такие как сервер очереди сообщений Майкрософт (MSMQ), COM (хранилище классов), протокол IPsec, групповая политика объекты (GPO) и Exchange, интегрированы с Active Directory.

В этом разделе описано, как использовать интерфейсы Active Directory служб (ADSI) для доступа к Active Directory. Использование сценария для вымышленной компании — корпорации Fabrikam — вы узнаете, как выполнять некоторые базовые задачи администрирования.

По мере выполнения этого сценария примеры кода в каждом разделе будут создаваться по отдельности.

Для краткости все примеры кода написаны с помощью системы разработки Microsoft Visual Basic. Дополнительные сведения о преобразовании C++ см. [в статье сопоставление кода Visual Basic ADSI с кодом c++](mapping-adsi-visual-basic-code-to-c-code.md).
