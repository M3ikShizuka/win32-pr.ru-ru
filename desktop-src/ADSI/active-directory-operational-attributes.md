# <a name="active-directory-operational-attributes"></a>Active Directory операционные атрибуты

Операционные атрибуты — это свойства, которые хранятся в каталоге и используются в административных целях. Например, можно сохранить операционный атрибут для записи в журнал даты и времени изменения другого атрибута. Операционные атрибуты должны быть явно извлечены с помощью вызова [**жетинфоекс**](/windows/desktop/api/Iads/nf-iads-iads-getinfoex).
