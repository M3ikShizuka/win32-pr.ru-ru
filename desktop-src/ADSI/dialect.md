# <a name="dialect"></a>Dialect

Два интерфейса ADSI для поиска в каталоге, ADO и OLE DB используют строку поиска для указания параметров поиска. Параметры поиска включают базу поиска, фильтр поиска, извлекаемые атрибуты и глубину поиска.

Строку поиска можно указать в одном из двух диалектов:

-   [Диалект LDAP](ldap-dialect.md)
-   [диалект SQL](sql-dialect.md)
