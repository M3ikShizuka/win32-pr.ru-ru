# <a name="regdsapicpp"></a>РЕГДСАПИ. CPP

В примере компонента поставщика функции, представляющие интерфейс API, который напрямую обращается к собственной операционной системе, находятся в Регдсапи. cpp. Пример компонента поставщика реализует свою службу каталогов в реестре. Чтобы написать поставщик, обращающийся к вашей собственной службе каталогов, создайте собственные реализации этого API. Поддерживаемые функции перечислены в следующей таблице.



| Метод                             | Описание                                                                                                                                                                                    |
|------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **сампледсопенобжект**             | Открыть этот объект по имени. Если параметр типа класса схемы имеет **значение NULL**, заполните найденный тип. Получение маркера для объекта.                                                             |
| **сампледсклосеобжект**            | Используйте маркер, полученный с помощью **сампледсопенобжект**.                                                                                                                                            |
| **сампледсрдненум**                | Получение маркера для объекта перечислителя для управления перечислением относительных различающихся имен (Рднс) из объекта контейнера.                                                          |
| **сампледснекстрдн**                | С помощью маркера, полученного **сампледсрдненум**, получите следующее относительное различающееся имя от этого объекта контейнера.                                                                        |
| **сампледсфриенум**               | Освободите объект перечислителя, выделенный в **сампледсрдненум**.                                                                                                                                   |
| **сампледсмодифйобжект**           | Изменение свойств объекта в службе каталогов по заданному маркеру объекта и списка атрибутов и их значений.                                                              |
| **сампледсреадобжект**             | Прочтите свойства объекта из службы каталогов. Сопоставьте синтаксис из собственного хранилища с соответствующими значениями синтаксиса ADS. Соответствующим образом обрабатывайте свойства с несколькими значениями. |
| **сампледсжетпропертидефинитион**  | В схеме найдите все определения свойств и их атрибуты для этого типа объекта класса схемы.                                                                                     |
| **сампледсжетпропертидефинитион**  | В схеме найдите это свойство и его атрибуты по имени.                                                                                                                               |
| **сампледсфрипропертидефинитион** | Свободная память, выделенная **жетпропертидефинитион**.                                                                                                                                            |
| **сампледсжеттипетекст**            | Возвращает тип класса схемы объекта в текстовом формате.                                                                                                                                         |
| **сампледсжеттипе**                | Возвращает тип класса схемы объекта.                                                                                                                                                        |
| **сампледсжетпропертинфо**        | При наличии маркера для объекта класса схемы и имени свойства извлекаются сведения о свойстве, такие как синтаксис и т. д.                                                                      |
| **FreeList**                       | Освободите память, используемую \_ СПИСКОМ LPWSTR.                                                                                                                                                        |
| **сампледсжетклассдефинитион**     | Получение набора всех определений классов схемы и связанных с ними данных из схемы.                                                                                                    |
| **сампледсжетклассдефинитион**     | Получение данных о конкретном классе схемы в схеме.                                                                                                                                   |
| **сампледсжетклассинфо**           | С учетом имени класса схемы найдите связанные с ним данные, такие как обязательные свойства.                                                                                                      |
| **сампледсаддобжект**              | Добавьте объект в службу каталогов.                                                                                                                                                        |
| **сампледсремовеобжект**           | Удалите объект из службы каталогов.                                                                                                                                                   |
| **сампледскреатебуффер**           | Создайте буфер памяти для данных атрибутов и данных операций.                                                                                                                                  |
| **сампледсфрибуффер**             | Освободите буфер, созданный в **сампледскреатебуффер**.                                                                                                                                           |
