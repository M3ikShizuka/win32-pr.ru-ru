# <a name="provider-overview"></a>Общие сведения о поставщике

Помимо [минимальных требований к поставщику](provider-minimum-requirements.md), другие сведения об использовании включают:

-   [Интерфейсы схемы](schema-interfaces.md) обсуждают интерфейсы управления схемой с точки зрения поставщика.
-   [Перечисление объектов контейнеров](enumerating-container-objects.md) описывает требования для поддержки перечислений.
-   [Сведения реестра поставщика](provider-registry-information.md) содержит записи реестра, используемые поставщиком ADSI.
-   [Поддержка запросов](support-for-queries.md) описывает требования для поддержки поиска в каталоге.
-   [Проблемы реализации поставщиков ADSI](implementation-issues-for-adsi-providers.md) включают примечания к поставщикам различных проблем с программным обеспечением.
