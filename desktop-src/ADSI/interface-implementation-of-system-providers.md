# <a name="interface-implementation-of-system-providers"></a>Реализация интерфейса поставщиков систем

ADSI включает несколько поставщиков служб каталогов, а именно, LDAP и WinNT. Различные поставщики могут выбрать реализацию или не реализацию интерфейса. Разные поставщики служб каталогов поддерживают различные свойства и методы. Дополнительные сведения о поддержке конкретного поставщика службы каталогов см. в разделе [поставщики систем ADSI](adsi-system-providers.md).
