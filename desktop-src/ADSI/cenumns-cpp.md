# <a name="cenumnscpp"></a>ЦЕНУМНС. CPP

В примере компонента поставщика перечисление объекта пространства имен использует методы из ценумнс. cpp, перечисленные в следующей таблице.



| Метод                                              | Описание                                                                                                                                               |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Ксампледснамеспацеенум:: Create**                  | Создайте объект, чтобы разрешить перечисление объекта пространства имен ADS.                                                                                         |
| **Ксампледснамеспацеенум:: Ксампледснамеспацеенум**  | Стандартный конструктор.                                                                                                                                     |
| **Ксампледснамеспацеенум:: ~ Ксампледснамеспацеенум** | Стандартный деструктор.                                                                                                                                      |
| **Ксампледснамеспацеенум:: Next**                    | Извлечение указанного числа элементов из указанного объекта пространства имен.                                                                            |
| **Ксампледснамеспацеенум:: Енумобжектс**             | Управление получением указателей интерфейса на объекты.                                                                                                  |
| **Ксампледснамеспацеенум:: Фетчобжектс**            | Получение набора указателей [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) .                                                                          |
| **Ксампледснамеспацеенум:: Фетчнекстобжект**         | Получить следующий объект. Если он найден, создайте универсальный объект Active Directory и извлеките его указатель [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . |
