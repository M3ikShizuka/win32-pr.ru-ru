# <a name="globalscpp"></a>Globals. CPP

В примере компонента поставщика используемые глобальные значения можно найти в Globals. cpp. Поддерживаемые глобальные значения перечислены в следующей таблице.



| Элемент                                          | Описание                                                                                                                                                                             |
|-----------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| g \_ сзпровидернаме                             | Следующий                                                                                                                                                                                |
| кэйвордлист                                   | "пользователь", "Группа", "подразделение"                                                                                                                                                  |
| Сопоставления синтаксиса из примера DS с вариантами ADS | Сопоставляет синтаксис "String" и вариант VT \_ BSTR, а также "integer" и VT \_ i4. Представляет сопоставление собственного формата данных с форматами, поддерживаемыми службой автоматизации, и наоборот. |
