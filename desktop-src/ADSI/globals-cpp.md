# <a name="globalscpp"></a>Globals. CPP

В примере компонента поставщика используемые глобальные значения можно найти в Globals. cpp. Поддерживаемые глобальные значения перечислены в следующей таблице.



| Элемент                                          | Описание                                                                                                                                                                             |
|-----------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| g \_ сзпровидернаме                             | Следующий                                                                                                                                                                                |
| кэйвордлист                                   | "пользователь", "Группа", "подразделение"                                                                                                                                                  |
| Сопоставления синтаксиса из примера DS с вариантами ADS | Карты между синтаксисом "String" и вариантом vt \_ BSTR, а также "Integer" и VT \_ I4. Представляет сопоставление собственного формата данных с форматами, поддерживаемыми службой автоматизации, и наоборот. |
