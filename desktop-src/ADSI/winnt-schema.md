# <a name="winnt-schema"></a>Схема WinNT

в схеме WinNT определяется набор объектов ADSI и их иерархия классов для сети Windows NT 4,0. [Объекты ADSI в Winnt](adsi-objects-of-winnt.md) включают пользователя, компьютер, домен, группу, ресурс, сеанс, службу и другие. В разделе " [Иерархия классов объектов](winnt-object-class-hierarchy.md) " ниже обсуждаются иерархические связи между этими объектами.

Каждый объект реализует набор интерфейсов ADSI. Однако в реализованном интерфейсе поддерживаются не все свойства и методы. [Обязательные и необязательные свойства в Winnt Schema](winnt-schemaampaposs-mandatory-and-optional-properties.md) содержат список свойств, поддерживаемых классами объектов Winnt.
