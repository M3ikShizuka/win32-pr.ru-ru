# <a name="client-shell-api-additional-functions"></a>Дополнительные функции API оболочки клиента

В следующей таблице приведены общие сведения о неосновных функциях для программного интерфейса (API) клиентской оболочки служба удаленного управления Windows (WinRM).



| Дополнительные функции                                                   | Описание                       |
|------------------------------------------------------------------------|-----------------------------------|
| [**всманжетеррормессаже**](/windows/desktop/api/Wsman/nf-wsman-wsmangeterrormessage)                   | Возвращает сообщения об ошибках.              |
| [**всманжетсессионоптионасдворд**](/windows/desktop/api/Wsman/nf-wsman-wsmangetsessionoptionasdword)   | Возвращает параметры сеанса в виде DWORD.  |
| [**всманжетсессионоптионасстринг**](/windows/desktop/api/Wsman/nf-wsman-wsmangetsessionoptionasstring) | Возвращает параметры сеанса в виде строки. |
| [**всмансетсессионоптион**](/windows/desktop/api/Wsman/nf-wsman-wsmansetsessionoption)                 | Задает параметры сеанса.             |
