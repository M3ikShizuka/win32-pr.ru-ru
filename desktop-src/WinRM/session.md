# <a name="session-object-wsmandisph"></a>Объект Session (Всмандисп. h)

Определяет параметры операций и сеанса. любой служба удаленного управления Windows операции требует создания **сеанса** , который подключается к удаленному компьютеру, [*базовому контроллеру управления*](windows-remote-management-glossary.md) (BMC) или локальному компьютеру. Сетевые операции WinRM включают получение, запись или перечисление данных или вызов методов. Методы объекта **сеанса** зеркально отражают основные операции, определенные в протоколе WS-Management.

## <a name="members"></a>Элементы

Объект **сеанса** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **Session** имеет следующие методы.



| Метод                                 | Описание                                                                                                                 |
|:---------------------------------------|:----------------------------------------------------------------------------------------------------------------------------|
| [**Создания**](session-create.md)       | Создает новый экземпляр ресурса и возвращает универсальный код ресурса (URI) нового объекта.<br/>                                      |
| [**Удален**](session-delete.md)       | Удаляет ресурс, указанный в URI ресурса.<br/>                                                              |
| [**Перечислить**](session-enumerate.md) | Перечисляет ресурсы журнала коллекции, таблицы или сообщения.<br/>                                                         |
| [**Получить**](session-get.md)             | Извлекает ресурс из службы и возвращает XML-представление текущего экземпляра ресурса.<br/> |
| [**Указывается**](session-identify.md)   | Запрашивает удаленный компьютер, чтобы определить, поддерживает ли он протокол WS-Management<br/>                                 |
| [**Invoke**](session-invoke.md)       | Вызывает метод, возвращающий результаты вызова метода.<br/>                                                    |
| [**PUT**](session-put.md)             | Обновление ресурса.<br/>                                                                                              |



 

### <a name="properties"></a>Свойства

Объект **сеанса** имеет следующие свойства.



| Свойство                                            | Тип доступа           | Описание                                                                                                                                                                                                                 |
|:----------------------------------------------------|:----------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**батчитемс**](session-batchitems.md)<br/> | Чтение/запись<br/> | Задает и возвращает количество элементов в каждом пакете перечисления. Это значение нельзя изменить во время перечисления. По умолчанию значение по умолчанию равно неограниченному количеству элементов. Поставщик ресурсов может установить ограничение.<br/> |
| [**План**](session-error.md)<br/>           | Только для чтения<br/>  | Возвращает дополнительные сведения об ошибке в потоке XML.<br/>                                                                                                                                                              |
| [**Счетчик**](session-timeout.md)<br/>       | Чтение/запись<br/> | Задает и возвращает максимальное время ожидания (в миллисекундах) для клиентского приложения.<br/>                                                                                                                   |



 

## <a name="remarks"></a>Remarks

Объект **Session** соответствует интерфейсу [**IWSManSession**](/windows/desktop/api/WSManDisp/nn-wsmandisp-iwsmansession) .

## <a name="examples"></a>Примеры

В следующем примере кода VBScript показано, как создать объект **сеанса** .


```VB
' Create a WSMan object.
Dim objWsman 
Set objWsman = CreateObject( "WSMAN.Automation" )

' Create Session object.
Dim objSession
Set objSession = objWsman.CreateSession
```
