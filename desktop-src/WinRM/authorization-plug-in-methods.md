# <a name="authorization-plug-in-methods"></a>Методы подключаемого модуля авторизации

Все точки входа подключаемых модулей авторизации имеют механизм обратного вызова для сообщения об успешном или неуспешном вызове. Некоторые механизмы обратного вызова вызываются несколько раз до тех пор, пока не будут переданы все результаты.

В следующей таблице приведены общие сведения о методах обратного вызова авторизации.



| Метод                                                                           | Описание                                                          |
|----------------------------------------------------------------------------------|----------------------------------------------------------------------|
| [**всманплугинаусзоператионкомплете**](/windows/desktop/api/Wsman/nf-wsman-wsmanpluginauthzoperationcomplete)   | Сообщает об успешной или неудачной операции пользователя.                |
| [**всманплугинаусзкуерикуотакомплете**](/windows/desktop/api/Wsman/nf-wsman-wsmanpluginauthzqueryquotacomplete) | Сообщает сведения о квоте, относящиеся к указанному пользователю.      |
| [**всманплугинаусзусеркомплете**](/windows/desktop/api/Wsman/nf-wsman-wsmanpluginauthzusercomplete)             | Сообщает об успешной или неудачной авторизации подключения пользователя. |
