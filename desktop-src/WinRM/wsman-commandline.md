# <a name="wsmancommandline-property"></a>WSMan. CommandLine, свойство

Возвращает командную строку процесса, который загружает компонент автоматизации.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
WSMan.CommandLine As BSTR
```



## <a name="property-value"></a>Значение свойства

Строка, содержащая командную строку.
