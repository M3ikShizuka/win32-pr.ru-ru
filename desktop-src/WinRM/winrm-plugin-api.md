# <a name="winrm-plug-in-api"></a>API подключаемого модуля WinRM

Windows Подключаемые модули удаленного управления — это собственные библиотеки динамической компоновки (DLL), которые подключаются и расширяют функциональные возможности WinRM. Интерфейс прикладного программирования (API) подключаемого модуля WinRM предоставляет функциональные возможности, позволяющие пользователю создавать подключаемые модули, реализуя определенные API для поддерживаемых URI и операций с [*ресурсами*](windows-remote-management-glossary.md) . после того как подключаемые модули настроены для службы winrm или службы IIS (iis), они загружаются в узел WinRM или узел iis соответственно. Удаленные запросы направляются на эти точки входа подключаемых модулей для выполнения операций.

В разделе Справочник по интерфейсу API подключаемого модуля WinRM содержатся подробные сведения о следующих элементах API:

-   [Точки входа подключаемого модуля авторизации](authorization-plug-in-entry-points.md)
-   [Методы подключаемого модуля авторизации](authorization-plug-in-methods.md)
-   [Точки входа подключаемых модулей операций](operations-plug-in-entry-points.md)
-   [Методы подключаемых модулей операций](operations-plug-in-methods.md)
-   [Структуры](winrm-plug-in-api-structures.md)
