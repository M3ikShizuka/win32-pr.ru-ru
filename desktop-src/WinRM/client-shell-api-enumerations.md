# <a name="client-shell-api-enumerations"></a>Перечисления API клиентской оболочки

В следующей таблице приведены общие сведения о перечислениях в интерфейсе прикладного программирования (API) для клиентской оболочки служба удаленного управления Windows (WinRM).



| Перечисления                                                 | Описание                                                                                      |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**всманаусентикатионфлагс**](/windows/desktop/api/Wsman/ne-wsman-wsmanauthenticationflags) | Определяет схему проверки подлинности, используемую для проверки подлинности сервера и прокси.               |
| [**всманкаллбаккфлагс**](/windows/desktop/api/Wsman/ne-wsman-wsmancallbackflags)             | Определяет набор флагов, используемых всеми функциями обратного вызова.                                           |
| [**всмандататипе**](/windows/desktop/api/Wsman/ne-wsman-wsmandatatype)                       | Задает текущий тип данных объединения в структуре [**\_ данных WSMan**](/windows/desktop/api/Wsman/ns-wsman-wsman_data) . |
| [**всманпроксякцесстипе**](/windows/desktop/api/Wsman/ne-wsman-wsmanproxyaccesstype)         | Определяет тип доступа прокси-сервера.                                                                   |
| [**WSManSessionOption**](/windows/desktop/api/Wsman/ne-wsman-wsmansessionoption)             | Определяет набор расширенных параметров для сеанса.                                               |
