# <a name="resourcelocatorerror-property"></a>ResourceLocator. Error, свойство

Возвращает XML-представление дополнительных сведений об ошибке.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
ResourceLocator.Error As BSTR
```



## <a name="property-value"></a>Значение свойства

XML-представление дополнительных сведений об ошибке.
