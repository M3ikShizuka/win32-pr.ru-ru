# <a name="locale_itlzero"></a>итлзеро ЛОКАЛи \_

Описатель для начальных нулей в полях времени. В приложении предпочтительнее использовать локальную константу [ \_ СТИМЕФОРМАТ](locale-stime-constants.md) вместо locale \_ итлзеро.



| Значение | Значение                    |
|-------|----------------------------|
| 0     | Нет начальных нулей для часов |
| 1     | Начальные нули для часов    |
