# <a name="locale_use_cp_acp"></a>Использование языкового стандарта \_ \_ CP \_ ACP

**Windows Me/98, windows 2000:** Системная кодовая страница Windows ANSI по умолчанию (ACP) вместо кодовой страницы языкового стандарта, используемой для перевода строки. Список кодовой страницы ANSI и других кодовых страниц см. в разделе [идентификаторы страниц](code-page-identifiers.md) .
