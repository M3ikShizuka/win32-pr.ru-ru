# <a name="locale_ssort_constants"></a>\_константы ССОРТ языкового стандарта \_

В этом разделе определяются \_ константы ССОРТ локали \* , используемые NLS.



| Значение               | Значение                                                                                                                                                                                                           |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ссортлокале ЛОКАЛи \_ | **Windows 7 и более поздних версий:** Имя языкового стандарта, используемого для сортировки или работы с регистром.                                                                                                                                |
| ссортнаме ЛОКАЛи \_   | **Windows Me/98, Windows 2000:** Полное локализованное имя сортировки для указанного идентификатора языкового стандарта, зависящее от языка оболочки. Эта константа используется для определения режима работы с регистром и сортировкой. |
