# <a name="transliteration-services"></a>Службы транслитерации

Службы транслитерации ELS сопоставляют текст UTF-16 от одной системы записи к другой системе письма. Каждая служба фактически представляет собой данные, применяемые к определенному набору входных и выходных сценариев Юникода, а сам Транслитерация является внутренним по отношению к платформе ELS. Приложение может дополнительно перечислить доступные службы для конкретных входных и выходных скриптов и выбрать требуемую службу.

Платформа поддерживает метаданные для служб транслитерации ELS. Метаданные для каждой службы предоставляют описание службы и список входных и выходных скриптов, поддерживаемых службой. Метаданные представлены структурой [**\_ \_ сведений службы сопоставления**](/windows/desktop/api/Elscore/ns-elscore-mapping_service_info) , полученной функцией [**маппингжетсервицес**](/windows/desktop/api/Elscore/nf-elscore-mappinggetservices) .

## <a name="input-to-a-transliteration-service"></a>Входные данные для службы транслитерации

В качестве входных данных для службы транслитерации используется текст UTF-16 в одной системе записи.

## <a name="output-of-a-transliteration-service"></a>Выходные данные службы транслитерации

В качестве выходных данных службы транслитерации используется текст UTF-16, сопоставленный со второй системой записи. Если подходящее сопоставление транслитерации для любого фрагмента входного текста не поддерживается, то фрагмент остается неизменным.

## <a name="transliteration-service-operation"></a>Операция службы транслитерации

Служба транслитерации сопоставляет текст Юникода из входного скрипта с выходным скриптом, символом или термином по условию, соответствующим образом. Приложение может выбрать получение определенной службы транслитерации, указав входные и выходные скрипты при вызове [**маппингжетсервицес**](/windows/desktop/api/Elscore/nf-elscore-mappinggetservices)или указав идентификатор GUID службы. Кроме того, для приложения можно перечислить все доступные службы транслитерации, указав категорию службы «Транслитерация» при вызове **маппингжетсервицес**. В этом случае приложение вызывает каждую службу и сравнивает результаты с исходным текстом, чтобы узнать, были ли результаты изменены операцией определенной службы.

Приложение может запрашивать распознавание текста для службы ELSного транслитерации с помощью вызова [**маппингрекогнизетекст**](/windows/desktop/api/Elscore/nf-elscore-mappingrecognizetext). При получении запроса служба транслитерации выделяет буфер для хранения данных, а затем выполняет распознавание текста для каждой кодовой точки во входной строке, предоставляемой приложением.

> [!Note]  
> Исходный текст и перетранслитерированный текст могут иметь разную длину.

 

## <a name="transliteration-service-guids"></a>Идентификаторы GUID службы транслитерации

Идентификаторы GUID для служб транслитерации объявляются в Елссрвк. h, как показано в следующем коде.


```C++
// {A3A8333B-F4FC-42f6-A0C4-0462FE7317CB}
static const GUID ELS_GUID_TRANSLITERATION_HANT_TO_HANS =
    { 0xA3A8333B, 0xF4FC, 0x42f6, { 0xA0, 0xC4, 0x04, 0x62, 0xFE, 0x73, 0x17, 0xCB } };

// {3CACCDC8-5590-42dc-9A7B-B5A6B5B3B63B}
static const GUID ELS_GUID_TRANSLITERATION_HANS_TO_HANT =
    { 0x3CACCDC8, 0x5590, 0x42dc, { 0x9A, 0x7B, 0xB5, 0xA6, 0xB5, 0xB3, 0xB6, 0x3B } };

// {D8B983B1-F8BF-4a2b-BCD5-5B5EA20613E1}
static const GUID ELS_GUID_TRANSLITERATION_MALAYALAM_TO_LATIN =
    { 0xD8B983B1, 0xF8BF, 0x4a2b, { 0xBC, 0xD5, 0x5B, 0x5E, 0xA2, 0x06, 0x13, 0xE1 } };

// {C4A4DCFE-2661-4d02-9835-F48187109803}
static const GUID ELS_GUID_TRANSLITERATION_DEVANAGARI_TO_LATIN =
    { 0xC4A4DCFE, 0x2661, 0x4d02, { 0x98, 0x35, 0xF4, 0x81, 0x87, 0x10, 0x98, 0x03 } };

// {3DD12A98-5AFD-4903-A13F-E17E6C0BFE01}
static const GUID ELS_GUID_TRANSLITERATION_CYRILLIC_TO_LATIN =
    { 0x3DD12A98, 0x5AFD, 0x4903, { 0xA1, 0x3F, 0xE1, 0x7E, 0x6C, 0x0B, 0xFE, 0x01 } };

// {F4DFD825-91A4-489f-855E-9AD9BEE55727}
static const GUID ELS_GUID_TRANSLITERATION_BENGALI_TO_LATIN =
    { 0xF4DFD825, 0x91A4, 0x489f, { 0x85, 0x5E, 0x9A, 0xD9, 0xBE, 0xE5, 0x57, 0x27 } };
```
