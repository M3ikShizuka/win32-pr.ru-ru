# <a name="locale_name-constants"></a>\_Константы имени языкового стандарта \*

В этом разделе определяются \_ константы имени языкового стандарта \* , используемые NLS.




| Значение | Значение | 
|-------|---------|
| LOCALE_NAME_INVARIANT | Имя инвариантного языкового стандарта, предоставляющего стабильные данные о языковых стандартах и календарях. | 
| LOCALE_NAME_MAX_LENGTH | Максимальная длина имени языкового стандарта. Максимально допустимое число символов для этой строки — 85, включая завершающий нуль-символ.<blockquote>[!Note]<br />Приложение должно использовать константу для максимальной длины имени локали, а не жестко закодировать значение "85".</blockquote><br /> | 
| LOCALE_NAME_SYSTEM_DEFAULT | Имя текущей национальной настройки операционной системы. | 
| LOCALE_NAME_USER_DEFAULT | Имя текущего пользовательского языкового стандарта, совпадающее с предпочтениями, заданными в разделе региональные и языковые параметры панели управления. Этот языковой стандарт может отличаться от языкового стандарта текущего языка пользовательского интерфейса. |
