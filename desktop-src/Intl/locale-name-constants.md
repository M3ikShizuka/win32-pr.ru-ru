# <a name="locale_name-constants"></a>\_Константы имени языкового стандарта \*

В этом разделе определяются \_ константы имени языкового стандарта \* , используемые NLS.



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Значение</th>
<th>Значение</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td>LOCALE_NAME_INVARIANT</td>
<td>Имя инвариантного языкового стандарта, предоставляющего стабильные данные о языковых стандартах и календарях.</td>
</tr>
<tr class="even">
<td>LOCALE_NAME_MAX_LENGTH</td>
<td>Максимальная длина имени языкового стандарта. Максимально допустимое число символов для этой строки — 85, включая завершающий нуль-символ.
<blockquote>
[!Note]<br />
Приложение должно использовать константу для максимальной длины имени локали, а не жестко кодировать значение &quot; 85 &quot; .
</blockquote>
<br/></td>
</tr>
<tr class="odd">
<td>LOCALE_NAME_SYSTEM_DEFAULT</td>
<td>Имя текущей национальной настройки операционной системы.</td>
</tr>
<tr class="even">
<td>LOCALE_NAME_USER_DEFAULT</td>
<td>Имя текущего пользовательского языкового стандарта, совпадающее с предпочтениями, заданными в разделе региональные и языковые параметры панели управления. Этот языковой стандарт может отличаться от языкового стандарта текущего языка пользовательского интерфейса.</td>
</tr>
</tbody>
</table>
