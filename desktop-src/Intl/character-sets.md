# <a name="character-sets"></a>Кодировки

"Набор символов" — это сопоставление символов с их кодовыми значениями. Чаще всего в настоящее время для компьютеров используется [Юникод](unicode.md), глобальный стандарт для кодировки символов. на внутреннем уровне Windows приложения используют кодировку юникод в кодировке UTF-16. В UTF-16 большинство символов идентифицируются с помощью двухбайтовых кодов. Менее часто используемые добавочные символы представлены парой символов-заместителей, которая представляет собой пару из двух байтовых кодов. Дополнительные сведения см. в разделе [суррогаты и добавочные символы](surrogates-and-supplementary-characters.md).

некоторые Windows приложения должны работать с более старыми наборами символов, которые являются собственными для Windows Me/98/95. [Windows кодовые страницы](code-pages.md) позволяют приложению работать с этими кодировками. Эти наборы символов можно разделить на следующие:

-   [Однобайтовые кодировки](single-byte-character-sets.md) (SBCS). В SBCS каждый символ идентифицируется по значению на один байт.
-   Многобайтовые наборы символов, в частности [двухбайтовые](double-byte-character-sets.md) кодировки (DBCS). Многобайтовые кодировки предоставляют средства для представления большого количества символов на многих азиатских языках.
