# <a name="locale_sname"></a>SNAME ЛОКАЛи \_

**Windows Vista и более поздние версии:** Имя языкового стандарта, многокомпонентный тег для уникальной идентификации локали. Максимально допустимое число символов для этой строки — 85, включая завершающий нуль-символ. Тег основан на соглашениях о маркировке языков RFC 4646. Используемый шаблон описан в разделе [имена языковых стандартов](locale-names.md).
