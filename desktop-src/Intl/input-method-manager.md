# <a name="input-method-manager"></a>Диспетчер метода ввода

Диспетчер методов ввода (IMM) — это технология, используемая приложением для взаимодействия с редактором метода ввода (IME), который выполняется как служба. Редактор IME позволяет пользователям компьютеров вводить сложные символы и символы, например японские символы кандзи, с помощью стандартной клавиатуры.

В этом разделе описывается API-интерфейс IMM и объясняется, как использовать функциональные возможности для создания окон IME и управления ими. Она включает следующие разделы:

-   [О диспетчере методов ввода](about-input-method-manager.md)
-   [Использование диспетчера методов ввода](using-input-method-manager.md)
-   [Справочник по диспетчеру методов ввода](input-method-manager-reference.md)
