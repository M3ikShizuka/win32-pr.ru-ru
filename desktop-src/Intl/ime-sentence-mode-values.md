# <a name="ime-sentence-mode-values"></a>Значения режима предложений IME

Эти значения используются с функциями [**иммжетконверсионстатус**](/windows/desktop/api/Imm/nf-imm-immgetconversionstatus) и [**иммсетконверсионстатус**](/windows/desktop/api/Imm/nf-imm-immsetconversionstatus) .



| Константа                  | Определение                                                                 |
|---------------------------|----------------------------------------------------------------------------|
| редактор IME \_ смоде \_ автоматически     | Редактор IME выполняет обработку преобразования в автоматическом режиме.               |
| IME \_ смоде \_ None          | Нет сведений о предложении.                                               |
| редактор IME \_ смоде \_ фрасепредикт | Редактор IME использует сведения о фразах для прогнозирования следующего символа.             |
| редактор IME \_ смоде \_ плуралклаусе  | Редактор IME использует сведения о предложении plural для выполнения обработки преобразования. |
| редактор IME \_ смоде \_ синглеконверт | Редактор IME выполняет обработку преобразования в односимвольном режиме.        |
| \_диалог СМОДЕ \_ IME  | Редактор IME использует режим диалога. Это полезно для приложений разговора.      |



 

Биты с 16 по 31 зарезервированы для использования IME.
