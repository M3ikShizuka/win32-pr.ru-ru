# <a name="candidate-lists"></a>Списки кандидатов

Список кандидатов представляется структурой [**кандидателист**](/windows/win32/api/imm/ns-imm-candidatelist) , содержащей массив строк, указывающий символы или символьные строки, из которых пользователь может выбрать. Приложение, поддерживающее IME, может получать сведения списка кандидатов с помощью функций [**иммжеткандидателисткаунт**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelistcounta) и [**иммжеткандидателист**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelista) .
