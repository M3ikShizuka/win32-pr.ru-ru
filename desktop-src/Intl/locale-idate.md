# <a name="locale_idate"></a>Идате ЛОКАЛи \_

Описатель краткого порядка сортировки формата даты. Спецификатор должен иметь одно из следующих значений. Значения, заданные пользователем, не допускаются. В приложении рекомендуется использовать [языковой стандарт \_ сшортдате](locale-sshortdate.md) вместо \_ Идате locale.



| Значение | Значение        |
|-------|----------------|
| 0     | Месяц-день-год |
| 1     | День-месяц-год |
| 2     | Год-месяц-день |
