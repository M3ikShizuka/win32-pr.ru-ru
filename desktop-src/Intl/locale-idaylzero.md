# <a name="locale_idaylzero"></a>идайлзеро ЛОКАЛи \_

Описатель для начальных нулей в полях Day только для краткой даты. Максимально допустимое число символов для этой строки равно двум, включая завершающий нуль-символ. В приложении предпочтительнее использовать локальную константу [ \_ СШОРТДАТЕ](locale-sshortdate.md) вместо locale \_ идайлзеро.



| Значение | Значение                   |
|-------|---------------------------|
| 0     | Нет начальных нулей для дней |
| 1     | Начальные нули в днях    |
