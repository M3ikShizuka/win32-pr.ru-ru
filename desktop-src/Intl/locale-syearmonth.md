# <a name="locale_syearmonth"></a>сеармонс ЛОКАЛи \_

**Windows Me/98, windows 2000:** Строка форматирования годового месяца для языкового стандарта. Максимально допустимое число символов для этой строки — 80, включая завершающий нуль-символ. Эта строка показывает правильный формат строки даты, которая содержит только год и месяц, используя формат изображений в формате "день", " [месяц", "год" и "Эра](day--month--year--and-era-format-pictures.md)".
