# <a name="locale_scurrency"></a>скурренци ЛОКАЛи \_

Строка, используемая в качестве местного символа денежной суммы. Максимально допустимое число символов для этой строки равно 13, включая завершающий нуль-символ. Например, в США это символ "$".
