# <a name="locale_sdate"></a>ЯЗЫКовой стандарт " \_ sdata"

Символы для разделителя даты. Максимально допустимое число символов для этой строки равно четырем, включая завершающий нуль-символ.

**Windows Vista и более поздних версий:** Эта константа является устаревшей. Вместо этого используйте [языковой стандарт \_ сшортдате](locale-sshortdate.md) . Пользовательский языковой стандарт может не иметь одного однородного разделителя. Например, допустим формат, например "12/31, 2006".
