# <a name="language-identifiers"></a>Идентификаторы языка

Идентификатор языка — это стандартное Международный цифровой сокращенный номер [языка](locales-and-languages.md) в стране или географическом регионе. Каждый язык имеет уникальный идентификатор языка (тип данных LANGID), 16-разрядное значение, состоящее из основного идентификатора языка и идентификатора языка. Дополнительные сведения об идентификаторах языков см. в разделе [константы и строки идентификатора языка](language-identifier-constants-and-strings.md).

Идентификатор языка создается с помощью макроса [**макелангид**](/windows/desktop/api/Winnt/nf-winnt-makelangid) . На следующем рисунке показан формат битов в идентификаторе языка.

``` syntax
+-------------------------+-------------------------+
|     SubLanguage ID      |   Primary Language ID   |
+-------------------------+-------------------------+
15                    10  9                         0   bit
```

Ниже приведены предопределенные идентификаторы языка.

-   \_ \_ по умолчанию используется язык lang. Язык операционной системы по умолчанию.
-   \_пользователь lang \_ по умолчанию. Язык текущего пользователя.

Приложение может извлекать текущие идентификаторы языков с помощью функций [многоязыкового интерфейса пользователя](multilingual-user-interface.md) .
