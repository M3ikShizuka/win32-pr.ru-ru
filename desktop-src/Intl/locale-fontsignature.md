# <a name="locale_fontsignature"></a>фонтсигнатуре ЛОКАЛи \_

**Windows Me/98/95, Windows NT 4,0:** Конкретный битовый шаблон, определяющий связь между покрытием символов, необходимым для поддержки языкового стандарта и содержимого шрифта. Обратите внимание, что региональные \_ фонтсигнатуре данные имеют другую форму из всех других сведений о языковых стандартах. Все остальные сведения о языковых стандартах могут быть выражены в виде строки или в виде числа. Данные фонтсигнатуре языкового стандарта \_ извлекаются в структуре [локалесигнатуре](/windows/win32/api/wingdi/ns-wingdi-localesignature) .
