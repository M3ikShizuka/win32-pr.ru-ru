# <a name="language-names"></a>Имена языков

Имя языка использует именно то же форматирование, что и [имя локали](locale-names.md). Например, имя языка для английского языка (Белиз) — EN-BZ. Каждое имя языка соответствует [идентификатору языка](language-identifiers.md).

Основные различия между форматированием для имени языка и форматирования имени языкового стандарта:

-   Имя языка никогда не включает порядок сортировки. Таким образом, имя языка, соответствующее имени локали "hu-HU \_ течнл" (венгерский язык, Венгрия, с технической сортировкой), — это просто "hu-HU".
-   Имя языка может быть "нейтральным" в отношении языкового стандарта. Например, "en" (английский) — это допустимое нейтральное название языка, отличное от EN-BZ (английский, Белиз).
