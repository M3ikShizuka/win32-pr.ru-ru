# <a name="locale_inegativepercent"></a>инегативеперцент ЛОКАЛи \_

**Windows 7 и более поздние версии:** Шаблон форматирования отрицательного процента для языкового стандарта. Может быть указан только один шаблон. Если для языкового стандарта используется более одного формата, выберите вариант предпочтительно. Например, если в качестве отрицательного процентного значения отображается «-9%» для «отрицательных девяти процентов», то для этой константы нужно выбрать значение 0.



| Значение | Формат                                                    |
|-------|-----------------------------------------------------------|
| 0     | Отрицательный знак, число, пробел, процент; Например,-\# % |
| 1     | Отрицательный знак, число, процент; Например,-\#%         |
| 2     | Отрицательный знак, процент, число; Например,-%\#         |
| 3     | Процент, знак минус, число; Например,%-\#         |
| 4     | Процент, число, отрицательный знак; Например,%\#-         |
| 5     | Число, знак минус, процент; Например \#-%         |
| 6     | Число, процент, знак минус; Например \#%-         |
| 7     | Отрицательный знак, процент, пробел, число; Например,-% \# |
| 8     | Число, пробел, процент, знак минус; Например \# %- |
| 9     | Процент, пробел, число, знак минус; Например,% \#- |
| 10    | Процент, пробел, знак минус, число; Например,%-\# |
| 11    | Число, отрицательный знак, пробел, процент; Например, \# -% |
