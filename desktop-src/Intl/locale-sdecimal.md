# <a name="locale_sdecimal"></a>сдеЦимал ЛОКАЛи \_

Символы, используемые для десятичного разделителя, например "." в "3,14" или "," в "3, 14". Максимально допустимое число символов для этой строки равно четырем, включая завершающий нуль-символ.
