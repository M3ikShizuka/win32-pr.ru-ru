# <a name="supporting-system-language-settings"></a>Поддержка языковых параметров системы

Предпочтительный способ установки языковых настроек приложения MUI — поддержка того же набора языков интерфейса пользователя, что и Целевая операционная система. Приложение просто зависит от операционной системы, чтобы выбрать правильный язык пользовательского интерфейса для вывода. Связанный пример приведен в [примере приложения MUI: параметры системы](mui-system-settings-application-sample.md).
