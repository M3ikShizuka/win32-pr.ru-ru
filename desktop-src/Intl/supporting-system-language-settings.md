# <a name="supporting-system-language-settings"></a>поддержка системных языков Параметры

Предпочтительный способ установки языковых настроек приложения MUI — поддержка того же набора языков интерфейса пользователя, что и Целевая операционная система. Приложение просто зависит от операционной системы, чтобы выбрать правильный язык пользовательского интерфейса для вывода. связанный пример приведен в [примере приложения MUI: System Параметры приложение](mui-system-settings-application-sample.md).
