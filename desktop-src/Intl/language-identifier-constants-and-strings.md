# <a name="language-identifier-constants-and-strings"></a>Константы и строки идентификатора языка

> [!IMPORTANT]
> Константы идентификатора языка являются устаревшими, и их использование не рекомендуется. Использование имен языковых стандартов вместо идентификаторов языкового стандарта всегда является предпочтительным.

Описание идентификаторов языков см. в разделе [идентификатор языка](language-identifiers.md) .

Идентификатор первичного или подязыка может быть определен пользователем или заранее. Сведения о предопределенных основных идентификаторах языка с допустимыми идентификаторами языков см. в разделе [[MS-LCID]: Справочник по идентификатору языка (LCID) Windows](/openspecs/windows_protocols/ms-lcid/70feba9f-294e-491e-b6eb-56532684c37f).

> [!Note]  
> Если идентификатор языка отсутствует для использования с основным идентификатором языка, приложение должно использовать подязык **\_ по умолчанию**. Для ресурсов, которые одинаковы для всех языков основного языка, следует использовать **\_ нейтральный** языковой набор.

Определяемый пользователем основной идентификатор языка имеет значение в диапазоне от 0x0200 до 0x03ff. Все остальные значения зарезервированы для использования операционной системой.

Определяемый пользователем идентификатор языка имеет значение в диапазоне от 0x20 до 0x3F. Все остальные значения зарезервированы для использования операционной системой.
