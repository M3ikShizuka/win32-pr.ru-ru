# <a name="locale_siso-constants"></a>\_Константы SISO языкового стандарта \*

В этом разделе определяются \_ константы SISO локали \* , используемые NLS.



| Значение                     | Значение                                                                                                                                                                                                                                                                                                                                                                              |
|---------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| SISO3166CTRYNAME ЛОКАЛи \_  | **Windows Me/98, Windows NT 4,0:** Название страны или региона, основанное на стандарте ISO 3166, например "US" для США. Это также может возвращать число, например "029" для Карибские острова. Максимально допустимое число символов для этой строки равно девяти, включая завершающий нуль-символ.                                                                                        |
| SISO3166CTRYNAME2 ЛОКАЛи \_ | **Windows Vista и более поздние версии:** Трехбуквенное имя региона ISO (ISO 3166 3 — код письма для страны или региона), например "США" для США. Это также может возвращать число, например "029" для Карибские острова. Максимально допустимое число символов для этой строки равно девяти, включая завершающий нуль-символ.                                                            |
| SISO639LANGNAME ЛОКАЛи \_   | **Windows Me/98, Windows NT 4,0:** Сокращенное название языка, полностью основанное на значениях стандарта ISO 639, в нижнем регистре, например en для английского языка. Это может быть трехбуквенный код для языков, у которых нет 2-буквенного кода, например "хав" для Гавайского. Максимально допустимое число символов для этой строки равно девяти, включая завершающий нуль-символ. |
| SISO639LANGNAME2 ЛОКАЛи \_  | **Windows Vista и более поздние версии:** Трехбуквенное имя языка ISO в виде строчных букв (ISO 639-2 с тремя буквами для языка), например "ENG" (английский язык). Максимально допустимое число символов для этой строки равно девяти, включая завершающий нуль-символ.                                                                                                                  |
