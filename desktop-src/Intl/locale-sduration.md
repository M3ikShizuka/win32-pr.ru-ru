# <a name="locale_sduration"></a>сдуратион ЛОКАЛи \_

**Windows Vista и более поздние версии:** Формат длительности времени, состоящий из формата изображений, перечисленных в следующей таблице. Формат аналогичен формату для [языкового стандарта \_ стимеформат](locale-stime-constants.md). Как и для языкового стандарта \_ стимеформат, этот формат может также включать любую строку символов, заключенную в одинарные кавычки. Форматы могут включать, например, "ч: СС", или "d 'd" х'х "". В сравнении с языковыми стандартами \_ стимеформат для долей секунды существуют дополнительные форматы изображений. Так как этот формат имеет значение Duration, а не время, он не задает 12-или 24-часовую систему часов или включает индикатор AM/PM. Эту константу можно использовать, например, для приложения с несколькими носителями, которое отображает время файла или приложение для спортивных событий, отображающее время окончания.



| Значение | Значение                                                                                                                                                                                                                             |
|-------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| h     | Часы без начальных нулей для однозначных часов                                                                                                                                                                                  |
| hh    | Часы с начальными нулями для часов, состоящих из одной цифры                                                                                                                                                                                     |
| m     | Минуты без начальных нулей для минут с одной цифрой                                                                                                                                                                              |
| ММ    | Минуты с начальными нулями для минут с одной цифрой                                                                                                                                                                                 |
| s     | Секунды без начальных нулей для однозначных секунд                                                                                                                                                                              |
| сс    | Секунды с начальными нулями для однозначных секунд                                                                                                                                                                                 |
| f     | Десятые доли секунды                                                                                                                                                                                                                  |
| ff    | Сотые доли секунды                                                                                                                                                                                                              |
| fff   | Доли секунды; символ "f" может возникать до девяти последовательных попыток (ффффффффф), хотя поддержка таймеров частоты ограничена 100 наносекунд; Если указаны девять символов, последние две цифры всегда равны нулю. |
