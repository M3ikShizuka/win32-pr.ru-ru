# <a name="unicode-and-character-sets"></a>Юникод и кодировки

Microsoft Windows обеспечивает поддержку множества различных письменных языков международного рынка в [юникоде](unicode.md) и традиционных наборах символов.

Юникод — это стандартный стандарт кодировки символов, который предоставляет уникальный номер для представления каждого символа, используемого в современных вычислениях, включая технические символы и специальные символы, используемые при публикации. Юникод требуется для современных стандартов, таких как XML и ECMAScript (JavaScript), и является официальным механизмом реализации ISO/IEC 10646. Поддерживается многими операционными системами, всеми современными браузерами и многими другими продуктами. новые Windows приложения должны использовать юникод, чтобы избежать несоответствий между различными кодовыми страницами и упростить локализацию.

традиционные наборы символов — это предыдущие стандарты кодирования символов, такие как Windows [кодовые страницы](code-pages.md) , использующие 8-битные значения кода или сочетания 8-разрядных значений для представления символов, используемых в определенном языке или географическом регионе.

В этом обзоре описываются функции наборов символов и объясняется, как использовать их в приложениях.
