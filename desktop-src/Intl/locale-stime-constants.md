# <a name="locale_stime-constants"></a>\_Константы СТИМЕ языкового стандарта \*

В этом разделе определяются \_ константы СТИМЕ локали \* , используемые NLS.



| Значение               | Значение                                                                                                                                                                                                                                                                                                                                                                                   |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| стиме ЛОКАЛи \_       | Символ (ы) для разделителя времени. Максимально допустимое число символов для этой строки равно четырем, включая завершающий нуль-символ. <br/> **Windows Vista и более поздние версии:** Эта константа является устаревшей. Вместо этого используйте ЯЗЫКовой стандарт \_ стимеформат. Пользовательский языковой стандарт может не иметь одного однородного разделителя. Например, допустим формат "03:56 ' 23".<br/> |
| стимеформат ЛОКАЛи \_ | Строки форматирования времени для языкового стандарта. Максимально допустимое число символов для этой строки — 80, включая завершающий нуль-символ. Строка может состоять из комбинации [часа, минуты и второго формата изображений](hour--minute--and-second-format-pictures.md).                                                                                                      |
