# <a name="multilingual-user-interface"></a>Многоязыковой интерфейс пользователя

многоязычный пользовательский интерфейс (MUI) обеспечивает локализацию пользовательских интерфейсов для глобализованных приложений. MUI также поддерживает создание ресурсов для любого количества языков пользовательского интерфейса.

В этой документации описывается MUI и объясняется, как добавить функциональные возможности MUI в полностью глобализованное приложение. Однако он не пытается описать все сведения о глобализованной разработке приложений на Windows. Дополнительные сведения об этом типе см. в [центре разработчиков Microsoft Go Global](https://msdn.microsoft.com/goglobal).

Документация многоязыкового интерфейса пользователя включает в себя следующие разделы:

-   [о многоязычный пользовательский интерфейс](about-multilingual-user-interface.md)
-   [использование многоязычный пользовательский интерфейс](using-multilingual-user-interface.md)
-   [многоязычный пользовательский интерфейс IsReference](multilingual-user-interface-reference.md)
