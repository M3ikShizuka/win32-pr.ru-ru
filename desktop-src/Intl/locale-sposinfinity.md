# <a name="locale_sposinfinity"></a>спосинфинити ЛОКАЛи \_

**Windows Vista и более поздних версий:** Строковое значение для "положительная бесконечность", например "бесконечность" для языкового стандарта "Английский (США)". Количество символов в строке не ограничено.
