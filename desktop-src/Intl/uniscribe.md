# <a name="uniscribe"></a>Uniscribe

Uniscribe — это набор API-интерфейсов, обеспечивающих высокий уровень контроля для тонкого оформления и обработки сложных наборов символов. Как для сложных наборов символов, так и для простых сценариев с тонкими типографскими эффектами требуется специальная обработка для отображения и редактирования, поскольку символы ("глифы") не размещаются простым способом. В сложных скриптах правила, определяющие формирование и размещение глифов, задаются и сохранятся в [стандарте Юникода](https://www.unicode.org/).
