# <a name="locale_senglish-constants"></a>\_Константы СЕНГЛИШ языкового стандарта \*

В этом разделе определяются \_ константы СЕНГЛИШ локали \* , используемые NLS.



| Значение                        | Значение                                                                                                                                                                                                                                                  |
|------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| сенглишкаунтринаме ЛОКАЛи \_  | **Windows 7 и более поздние версии:** Английское название страны или региона, например "Германия" для Deutschland. Это имя всегда ограничено символами, которые можно сопоставить с подмножеством символов ASCII 127. <br/>                                      |
| сенглишдисплайнаме ЛОКАЛи \_  | **Windows 7 и более поздние версии:** Отображаемое имя языкового стандарта на английском языке. Обычно отображаемое имя состоит из языка и страны или региона, например немецкого (Германия) для Deutsch (Deutschland). <br/>                                           |
| сенглишлангуаженаме ЛОКАЛи \_ | **Windows 7 и более поздние версии:** Название языка на английском языке, например немецкий для Deutsch, от международного стандарта ISO 639. Это имя всегда ограничено символами, которые можно сопоставить с подмножеством символов ASCII 127. <br/> |
