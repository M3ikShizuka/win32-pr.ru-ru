# <a name="extended-linguistic-services"></a>Расширенные лингвистические службы

Расширенные лингвистические службы (ELS) поддерживают текстовые службы, в том числе определение языка, определение скриптов и службы транслитерации. Документация по ELS включает в себя следующие разделы:

-   [О расширенных лингвистических службах](about-extended-linguistic-services.md)
-   [Использование расширенных лингвистических служб](using-extended-linguistic-services.md)
-   [Расширенная ссылка на расширенные лингвистические службы](extended-linguistic-services-reference.md)
