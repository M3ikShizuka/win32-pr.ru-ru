# <a name="locale_iuseutf8legacyacp"></a>IUSEUTF8LEGACYACP ЛОКАЛи \_

**Начиная со следующей версии Windows:** Вместо этого рекомендуется использовать Юникод в приложениях. На компьютерах, где пользователь настроен для использования кодовой страницы UTF-8, LOCALE_IUSEUTF8LEGACYACP является устаревшей кодовой страницей Windows, которая была бы использована на нестандартном компьютере Windows, отличном от UTF-8.
