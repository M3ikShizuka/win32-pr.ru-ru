# <a name="locale_iuseutf8legacyacp"></a>IUSEUTF8LEGACYACP ЛОКАЛи \_

**Начиная со следующей версии Windows:** Вместо этого рекомендуется использовать Юникод в приложениях. на компьютерах, где пользователь настроен для использования кодовой страницы UTF-8, LOCALE_IUSEUTF8LEGACYACP — это устаревшая Windows кодовая страница, которая была бы использована на устаревшем Windows компьютере, отличном от UTF-8.
