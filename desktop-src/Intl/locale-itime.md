# <a name="locale_itime"></a>итиме ЛОКАЛи \_

Спецификация формата времени. Спецификация может иметь одно из следующих значений. В приложении предпочтительнее использовать локальную константу [ \_ СТИМЕФОРМАТ](locale-stime-constants.md) вместо locale \_ итиме.



| Значение | Описание          |
|-------|----------------------|
| 0     | 12-часовой формат AM/PM |
| 1     | 24-часовой формат       |
