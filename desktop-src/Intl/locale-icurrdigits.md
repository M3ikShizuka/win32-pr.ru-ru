# <a name="locale_icurrdigits"></a>икуррдигитс ЛОКАЛи \_

Количество цифр дробной части для местного денежного формата. Максимально допустимое число символов для этой строки равно двум, включая значения 0-9 и завершающий нуль-символ. Например, 200,00 имеет значение 2, поскольку существует две дробные цифры, а 200 имеет значение 0, так как дробные части не отображаются.
