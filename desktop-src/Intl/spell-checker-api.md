# <a name="spell-checking-api"></a>API проверки орфографии

API проверки орфографии позволяет разработчикам использовать средства проверки орфографии для проверки текста, получения предложений и поддержки параметров и пользовательских словарей. Разработчики также могут создавать средства проверки орфографии, интегрированные в платформу проверки орфографии в качестве поставщиков.

-   [Об API проверки орфографии](about-the-spell-checker-api.md)
-   [Справочник по API проверки орфографии](spell-checker-api-reference.md)
