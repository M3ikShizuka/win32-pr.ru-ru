# <a name="locale_sdayname-constants"></a>\_Константы сдайнаме языкового стандарта \*

В этом разделе определяются \_ константы СДАЙНАМЕ локали \* , используемые NLS.



| Значение             | Значение                                                                                                                                 |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| SDAYNAME1 ЛОКАЛи \_ | Собственное длинное имя в понедельник. Максимально допустимое число символов для этой строки — 80, включая завершающий нуль-символ.    |
| SDAYNAME2 ЛОКАЛи \_ | Собственное полное имя для вторника. Максимально допустимое число символов для этой строки — 80, включая завершающий нуль-символ.   |
| SDAYNAME3 ЛОКАЛи \_ | Собственное полное имя среды. Максимально допустимое число символов для этой строки — 80, включая завершающий нуль-символ. |
| SDAYNAME4 ЛОКАЛи \_ | Собственное длинное имя в четверг. Максимально допустимое число символов для этой строки — 80, включая завершающий нуль-символ.  |
| SDAYNAME5 ЛОКАЛи \_ | Собственное длинное имя для пятницы. Максимально допустимое число символов для этой строки — 80, включая завершающий нуль-символ.    |
| SDAYNAME6 ЛОКАЛи \_ | Собственное длинное имя для субботы. Максимально допустимое число символов для этой строки — 80, включая завершающий нуль-символ.  |
| SDAYNAME7 ЛОКАЛи \_ | Собственное длинное имя для Sunday. Максимально допустимое число символов для этой строки — 80, включая завершающий нуль-символ.    |
