Чтобы получить форму женитиве названия месяца, приложение вызывает <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformata">жетдатеформат</a> или <a href="/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformatex">Жетдатеформатекс</a> с датой ддмммм и удаляет две цифры из начала полученной строки.</blockquote><br /> |
