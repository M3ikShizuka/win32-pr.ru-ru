# <a name="ime-composition-string-values"></a>Строковые значения композиции IME

Эти значения используются с [**\_ \_ композицией IME**](wm-ime-composition.md) [**иммжеткомпоситионстринг**](/windows/desktop/api/Imm/nf-imm-immgetcompositionstringa) и WM.



| Значение                 | Описание                                                                                |
|-----------------------|--------------------------------------------------------------------------------------------|
| \_КОМПАТТР GC         | Извлечение или обновление атрибута строки композиции.                                |
| \_КОМПКЛАУСЕ GC       | Получение или обновление сведений о предложении строки композиции.                           |
| \_КОМПРЕАДАТТР GC     | Извлечение или обновление атрибутов строки чтения текущей композиции.        |
| \_КОМПРЕАДКЛАУСЕ GC   | Извлекает или обновляет сведения о предложении строки для чтения строки композиции. |
| \_КОМПРЕАДСТР GC      | Извлечение или обновление строки чтения текущей композиции.                          |
| \_КОМПСТР GC          | Получение или обновление текущей строки композиции.                                         |
| \_КУРСОРПОС GC        | Получение или обновление позиции курсора в строке композиции.                              |
| \_ДЕЛТАСТАРТ GC       | Возвращает или обновляет начальную точку любых изменений в строке композиции.             |
| \_РЕСУЛТКЛАУСЕ GC     | Получение или обновление сведений о предложении результирующей строки.                                |
| \_РЕСУЛТРЕАДКЛАУСЕ GC | Получение или обновление сведений о предложении строки для чтения.                               |
| \_РЕСУЛТРЕАДСТР GC    | Извлечение или обновление строки чтения.                                                     |
| \_РЕСУЛТСТР GC        | Извлечение или обновление строки результата композиции.                                   |
