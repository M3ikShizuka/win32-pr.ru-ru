# <a name="locale_ireadinglayout"></a>иреадинглайаут ЛОКАЛи \_

**Windows 7 и более поздних версий:** Макет для чтения текста. Возможные значения определены в следующей таблице.



| Значение | Значение                                                                                                                                                             |
|-------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0     | Чтение слева направо, как для английского языка (США).                                                                                                 |
| 1     | Чтение справа налево, как для арабских языков.                                                                                                                     |
| 2     | Прочтите вертикально сверху вниз, столбцы располагаются справа налево, или прочтите горизонтальные строки слева направо, как для японского языка (Япония). |
| 3     | Вертикально читается сверху вниз, а столбцы — слева направо, как и для языкового стандарта Монгольский (монгольский).                                                  |
