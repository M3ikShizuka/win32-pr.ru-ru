# <a name="locale_sgrouping"></a>сграупинг ЛОКАЛи \_

Размеры для каждой группы цифр слева от десятичного разделителя. Максимально допустимое число символов для этой строки равно десяти, включая завершающий нуль-символ. Для каждой группы требуется явный размер, а размеры разделяются точками с запятой. Если Последнее значение равно 0, предыдущее значение повторяется. Например, чтобы сгруппировать тысячи, укажите значение 3; 0. Индийские национальные стандарты группируют первые тысячи и затем группируются по сотням. Например, 12, 34, 56789 представляется 3; 2; 0.

Дополнительные примеры:



| Спецификация | Результирующая строка   |
|---------------|--------------------|
| 3; 0           | 3 000 000 000 000  |
| 3; 2; 0         | 30, 00, 00, 00, 00000 |
| 3             | 3 000 000 000 000     |
| 3; 2           | 30000000, 00000    |
