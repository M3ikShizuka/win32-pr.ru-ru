# <a name="availability-of-mui"></a>Доступность многоязыкового интерфейса пользователя

Следующие версии Windows включают API и конструкции для обеспечения собственной поддержки разработки многоязычных приложений с использованием технологии многоязыкового интерфейса пользователя:

-   Windows 7 и более поздние версии.
-   Windows Vista.

Библиотека нижнего уровня [**лоадмуилибрари**](/windows/desktop/api/Muiload/nf-muiload-loadmuilibrarya)предназначена для создания многоязычных приложений для версий Windows до Windows Vista и обеспечивает поведение, совместимое с приложениями многоязыкового интерфейса пользователя записанный для Windows Vista и более поздних версий Windows.
