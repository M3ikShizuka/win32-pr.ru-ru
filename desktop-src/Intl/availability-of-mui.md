# <a name="availability-of-mui"></a>Доступность многоязыкового интерфейса пользователя

следующие версии Windows включают api и конструкции для обеспечения собственной поддержки разработки многоязычных приложений с использованием технологии многоязыкового интерфейса пользователя:

-   Windows 7 и более поздней версии.
-   Windows Vista.

библиотека нижнего уровня [**лоадмуилибрари**](/windows/desktop/api/Muiload/nf-muiload-loadmuilibrarya)предназначена для создания многоязычных приложений для версий Windows до Windows Vista и обеспечивает поведение, совместимое с приложениями MUI, записанный для Windows Vista и более поздних версий Windows.
