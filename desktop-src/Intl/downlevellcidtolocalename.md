# <a name="downlevellcidtolocalename-function"></a>Функция ДовнлевеллЦидтолокаленаме

Преобразует [идентификатор локали](locale-identifiers.md) в [имя языкового стандарта](locale-names.md).

> [!Note]  
> Эта функция используется только приложениями, работающими в операционных системах, предшествующих Windows Vista. Для его использования требуется загружаемый пакет. Приложения, работающие только в Windows Vista и более поздних версиях, должны вызывать [**лЦидтолокаленаме**](/windows/desktop/api/Winnls/nf-winnls-lcidtolocalename) для получения имени языкового стандарта.

 

## <a name="syntax"></a>Синтаксис


```C++
int DownlevelLCIDToLocaleName(
  _In_  LCID   Locale,
  _Out_ LPWSTR lpName,
  _In_  int    cchName,
  _In_  DWORD  dwFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Языковой стандарт* \[ окне\]
</dt> <dd>

Идентификатор локали для преобразования. Для создания идентификатора языкового стандарта можно использовать макрос [**макелЦид**](/windows/desktop/api/Winnt/nf-winnt-makelcid) . Эта функция не поддерживает нейтральные национальные настройки или следующие конкретные значения идентификатора локали.

-   [система языкового стандарта \_ \_ по умолчанию](locale-system-default.md)
-   [\_Пользовательская Национальная настройка \_ по умолчанию](locale-user-default.md)
-   [Пользовательский ЯЗЫКовой стандарт \_ \_ по умолчанию](locale-custom-constants.md)
-   [Пользовательский пользовательский интерфейс языкового стандарта \_ \_ \_ по умолчанию](locale-custom-constants.md)
-   [Пользовательский ЯЗЫКовой стандарт не \_ \_ указан](locale-custom-constants.md)

</dd> <dt>

*лпнаме* \[ заполняет\]
</dt> <dd>

Указатель на буфер, в котором эта функция получает имя локали. Функция получает **значение NULL** , если *кчнаме* имеет значение 0.

</dd> <dt>

*кчнаме* \[ окне\]
</dt> <dd>

Размер (в кодовых позициях UTF-16) в буфере имени языкового стандарта. Приложение задает для этого параметра значение 0, чтобы получить требуемый размер в буфере имен языкового стандарта.

</dd> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Флаги, указывающие тип получаемого имени. Значение по умолчанию — \_ имя локали нижнего уровня \_ .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает количество кодовых точек UTF-16 в имени языкового стандарта, включая завершающий нуль-символ, в случае успеха. Если функция выполнена удачно и значение *кчнаме* равно 0, то возвращаемое значение будет иметь необходимый размер в символах (включая символы NULL) для буфера имен языкового стандарта.

Функция возвращает 0, если она не выполнена. Чтобы получить расширенные сведения об ошибке, приложение может вызвать [**GetLastError**](/windows/win32/api/errhandlingapi/nf-errhandlingapi-getlasterror), которая может возвращать один из следующих кодов ошибок:

-   Ошибка \_ : недостаточный \_ Размер буфера. Указанный размер буфера не достаточен или имеет неправильное значение **null**.
-   Ошибка \_ : недопустимые \_ флаги. Недопустимое значение *dwFlags* .
-   Ошибка \_ : недопустимый \_ параметр. Любое из значений параметров было недопустимым.
