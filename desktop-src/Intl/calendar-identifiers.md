# <a name="calendar-identifiers"></a>Идентификаторы календаря

В этом разделе определяются идентификаторы календаря (тип данных КАЛИД), используемые для указания различных календарей. Приложения могут использовать эти идентификаторы при использовании следующих функций многоязыковой поддержки и функций обратного вызова, которые имеют параметры, принимающие тип данных КАЛИД:

-   [**конвертсистемтиметокалдатетиме**](convertsystemtimetocaldatetime.md)
-   [**енумкалендаринфо**](/windows/desktop/api/Winnls/nf-winnls-enumcalendarinfoa)
-   [**енумкалендаринфоекс**](/windows/desktop/api/Winnls/nf-winnls-enumcalendarinfoexa)
-   [**енумкалендаринфоексекс**](/windows/desktop/api/Winnls/nf-winnls-enumcalendarinfoexex)
-   [**енумкалендаринфопроцекс**](/previous-versions/windows/desktop/legacy/dd317807(v=vs.85))
-   [**енумдатеформатспроцекс**](/previous-versions/windows/desktop/legacy/dd317814(v=vs.85))
-   [**жеткалендаринфо**](/windows/desktop/api/Winnls/nf-winnls-getcalendarinfoa)
-   [**жеткалендаринфоекс**](/windows/desktop/api/Winnls/nf-winnls-getcalendarinfoex)
-   [**жеткалендарсуппортеддатеранже**](getcalendarsupporteddaterange.md)
-   [**искалендарлеапеар**](iscalendarleapyear.md)
-   [**сеткалендаринфо**](/windows/desktop/api/Winnls/nf-winnls-setcalendarinfoa)

Определены следующие значения. Все остальные значения зарезервированы. Эти значения нельзя сочетать друг с другом.
