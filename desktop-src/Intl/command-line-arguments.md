# <a name="command-line-arguments"></a>Аргументы командной строки

Приложение может использовать функцию Win32 [GetCommandLine](/windows/win32/api/processenv/nf-processenv-getcommandlinea) для получения аргументов командной строки в Юникоде. Приложение должно вызывать функцию Win32 в качестве функции Юникода.
