2-значный или 4-значный столетие только для краткой даты. Век может иметь одно из следующих значений. В приложении рекомендуется использовать [языковой стандарт \_ сшортдате](locale-sshortdate.md) вместо \_ ицентури locale.



| Значение | Значение                     |

| 1     | Полный 4-значный век        |
