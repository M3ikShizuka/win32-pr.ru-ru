# <a name="locale_iconstructedlocale"></a>иконструктедлокале ЛОКАЛи \_

Идентификатор для запроса, если языковой стандарт — "сконструированный" языковой стандарт. Использовать этот LCTYPE не рекомендуется.

это определяет языковой стандарт, для которого Windows много не имеют полных сведений и во время выполнения им приходится «конструировать» сведения. обычно сведения, предоставляемые LOCALE_ICONSTRUCTEDLOCALE, имеют ограниченный объем использования, так как Windows предоставит столько данных, сколько доступно для каждого языкового стандарта. Поэтому использовать этот LCTYPE не рекомендуется.


| Значение | Значение                 |
|-------|-------------------------|
| 0     | Не создано         |
| 1     | Является сконструированным языковым стандартом |


Примером может быть запрос "de-US" или немецкий в США. NLS будет использовать данные на немецком языке, которые он может найти, а также данные США региона, которые она может найти. 

Это может быть не так, как, например, система, скорее всего, не будет содержать сведения об имени США на немецком языке. Однако если приложение или пользователь помещает контекст "de-US", то возвращаемые данные лучше доступны. 

Приложения, использующие LOCALE_ICONSTRUCTEDLOCALE для отклонения национальных настроек и переходят на другой языковой стандарт, обычно в конечном итоге имеют более серьезную работу, например, в нашем примере — de-DE или EN-US. Ни один из них не близок к исходному запросу немецкого языка с СШАным регионом.
