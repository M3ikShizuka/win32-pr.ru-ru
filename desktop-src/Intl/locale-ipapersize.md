# <a name="locale_ipapersize"></a>ипаперсизе ЛОКАЛи \_

**Windows 2000:** Размер бумаги по умолчанию, связанный с языковым стандартом. Размер обычно имеет одно из следующих значений, хотя для него можно задать любой из определенных размеров бумаги, распознаваемых диспетчером очереди печати.



| Значение | Значение   |
|-------|-----------|
| 1     | Письмо США |
| 5     | Юридическое лицо США  |
| 8     | A3        |
| 9     | A4        |
