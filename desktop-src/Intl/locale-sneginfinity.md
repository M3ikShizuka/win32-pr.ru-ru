# <a name="locale_sneginfinity"></a>снегинфинити ЛОКАЛи \_

**Windows Vista и более поздних версий:** Строковое значение для "отрицательной бесконечности", например "-Infinity" для языкового стандарта "Английский (США)". Количество символов в этой строке не ограничено.
