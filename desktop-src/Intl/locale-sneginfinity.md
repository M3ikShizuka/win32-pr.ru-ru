# <a name="locale_sneginfinity"></a>снегинфинити ЛОКАЛи \_

**Windows Vista и более поздние версии:** Строковое значение для "отрицательной бесконечности", например "-Infinity" для языкового стандарта "Английский (США)". Количество символов в этой строке не ограничено.
