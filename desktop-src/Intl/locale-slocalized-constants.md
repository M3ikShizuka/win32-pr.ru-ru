# <a name="locale_slocalized-constants"></a>\_Константы слокализед языкового стандарта \*

В этом разделе определяются \_ константы СЛОКАЛИЗЕД локали \* , используемые NLS.



| Значение                          | Значение                                                                                                                                                                                                                                                                                                                                     |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| слокализедкаунтринаме ЛОКАЛи \_  | **Windows 7 и более поздних версий:** Полное локализованное название страны или региона, например Deutschland для Германии. Максимально допустимое число символов для этой строки — 80, включая завершающий нуль-символ. Поскольку это имя основано на локализации продукта, оно изменяется для каждой локализованной версии. <br/>           |
| слокализеддисплайнаме ЛОКАЛи \_  | **Windows 7 и более поздних версий:** Полное локализованное имя языкового стандарта для языка пользовательского интерфейса, например Deutsch (Deutschland) для немецкого (Германия), ограничение числа символов, разрешенное для этой строки, не ограничено. Поскольку это имя основано на локализации продукта, оно изменяется для каждой локализованной версии. <br/> |
| слокализедлангуаженаме ЛОКАЛи \_ | **Windows Vista:** Полное локализованное первичное имя языка пользовательского интерфейса, входящее в локализованное отображаемое имя, например Deutsch, представляющее немецкий. Поскольку это имя основано на локализации продукта, оно изменяется для каждой локализованной версии. <br/>                                                               |
