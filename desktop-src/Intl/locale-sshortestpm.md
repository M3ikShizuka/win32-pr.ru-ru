# <a name="locale_sshortestpm"></a>сшортестпм ЛОКАЛи \_

Строка для кратчайшего допустимого индикатора PM для языкового стандарта. Для английского языка, который может иметь значение "p"; Однако другие языки могут иметь больше кодовыми точками.
