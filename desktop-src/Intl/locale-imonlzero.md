# <a name="locale_imonlzero"></a>имонлзеро ЛОКАЛи \_

Описатель для начальных нулей в полях месяца только для краткой даты. В приложении предпочтительнее использовать локальную константу [ \_ СШОРТДАТЕ](locale-sshortdate.md) вместо locale \_ имонлзеро.



| Значение | Значение                     |
|-------|-----------------------------|
| 0     | Нет начальных нулей для месяцев |
| 1     | Начальные нули для месяцев    |
