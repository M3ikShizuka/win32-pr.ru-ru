# <a name="microsoft-script-detection"></a>Обнаружение сценариев Майкрософт

Служба обнаружения скриптов ELS называется обнаружением сценариев Майкрософт. Эта служба позволяет приложениям обнаруживать скрипты, в которых записывается текст. Поддержка национальных языков (NLS) службы обнаружения скриптов является функцией [жетстрингскриптс](/windows/desktop/api/Winnls/nf-winnls-getstringscripts) . Однако служба ELS дополнительно извлекает текстовые диапазоны, принадлежащие каждой системе записи.

## <a name="input-to-microsoft-script-detection"></a>Входные данные для обнаружения сценариев Майкрософт

Входными данными для службы обнаружения скриптов Майкрософт является текст UTF-16, для которого служба определяет диапазоны скриптов.

## <a name="output-of-microsoft-script-detection"></a>Выходные данные обнаружения сценариев (Майкрософт)

Выходные данные службы обнаружения скриптов Майкрософт — это массив диапазонов, каждый из которых содержит строку UTF-16, заканчивающуюся нулем, с именем связанной системы записи, заданным в Юникоде. Служба выводит обычные общие (Зиии) и унаследованные (Кааи) символы как принадлежащие предыдущему диапазону скриптов. Начало общих и наследуемых символов указывается как принадлежащий следующему диапазону скриптов. Если все символы во входном тексте являются общими или наследуются, выходные данные службы представляют собой один диапазон с пустой строкой в качестве данных.

## <a name="microsoft-script-detection-operation"></a>Операция обнаружения скрипта (Майкрософт)

Служба обнаружения сценариев (Майкрософт) сопоставляет кодовые точки, относящиеся к общему диапазону, с предыдущей системой записи. Кроме того, служба может сопоставлять кодовые точки со следующей системой записи, если кодовые точки находятся в начале входной строки. Приложению не придется работать с общим диапазоном.

## <a name="microsoft-script-detection-guid"></a>Идентификатор GUID обнаружения сценариев (Майкрософт)

Идентификатор GUID для службы Microsoft распознавание языка объявляется в Елссрвк. h, как показано в следующем коде.


```C++
// {2D64B439-6CAF-4f6b-B688-E5D0F4FAA7D7}
static const GUID ELS_GUID_SCRIPT_DETECTION =
    { 0x2D64B439, 0x6CAF, 0x4F6B, { 0xB6, 0x88, 0xE5, 0xD0, 0xF4, 0xFA, 0xA7, 0xD7 } };
```
