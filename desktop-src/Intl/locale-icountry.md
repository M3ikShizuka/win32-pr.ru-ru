# <a name="locale_idialingcode-and-locale_icountry"></a>ЯЗЫКовой стандарт \_ идиалингкоде и \_ икаунтри locale

Код страны или региона, основанный на международных кодах телефонов, которые также называются кодами страны и региона IBM. Максимально допустимое число символов для этой строки равно шести, включая завершающий нуль-символ.

Windows 10 добавил ЯЗЫКовой стандарт \_ идиалингкоде в качестве более точно именованного синонима для ИКАУНТРИ локали \_ . Мы рекомендуем использовать новое имя, чтобы улучшить удобочитаемость кода.

> [!Note]  
> Международные коды не являются уникальными для определенной страны. Например, в некоторых странах, включая США и Канада, в качестве международного кода используется «1».
