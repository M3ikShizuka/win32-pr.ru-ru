# <a name="about-national-language-support"></a>Сведения о поддержке национальных языков

Функции поддержки национальных языков (NLS) позволяют приложениям выполнять следующие задачи:

-   Задать языковой стандарт для пользователя
-   Указание языка, на котором работает пользователь
-   Получение строк, представляющих время, даты и другие сведения, правильно отформатированные для указанного языка и локали

NLS также включает поддержку раскладок клавиатуры и шрифтов, зависящих от языка. Дополнительные сведения см. в разделе [Диспетчер методов ввода](input-method-manager.md) и [международные шрифты и текст](international-fonts-and-text-display.md).

-   [Языковые стандарты и языки](locales-and-languages.md)
-   [Терминология NLS](nls-terminology.md)
-   [Сортировка](sorting.md)
