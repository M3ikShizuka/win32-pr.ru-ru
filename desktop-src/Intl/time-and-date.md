# <a name="time-and-date"></a>Время и Дата

NLS поддерживает получение строк для любого времени или даты в формате, подходящем для текущего языкового стандарта. в Windows Vista и более поздних версиях NLS также поддерживает получение строк длительности для всех языков, включая пользовательские языковые стандарты. Инструкции по разработке см. в разделе [Получение сведений о времени и датах](retrieving-time-and-date-information.md).
