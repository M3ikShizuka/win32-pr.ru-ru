# <a name="locale_ioptionalcalendar"></a>иоптионалкалендар ЛОКАЛи \_

Необязательный тип календаря, доступный для языкового стандарта. Тип календаря может представлять только необязательный календарь, доступный для соответствующего языкового стандарта. Чтобы получить все необязательные календари, доступные для языкового стандарта, приложение может использовать следующие функции NLS:

-   [енумкалендаринфо](/windows/desktop/api/Winnls/nf-winnls-enumcalendarinfoa)
-   [енумкалендаринфоекс](/windows/desktop/api/Winnls/nf-winnls-enumcalendarinfoexa)
-   [енумкалендаринфоексекс](/windows/desktop/api/Winnls/nf-winnls-enumcalendarinfoexex)
