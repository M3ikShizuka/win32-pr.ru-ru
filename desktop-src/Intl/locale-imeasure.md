# <a name="locale_imeasure"></a>имеасуре ЛОКАЛи \_

Система измерения. Максимально допустимое число символов для этой строки равно двум, включая завершающий нуль-символ. Это значение равно 0, если используется система метрик (Систéме International Д'унитс или С.И.), и 1, если используется СШАная система.
