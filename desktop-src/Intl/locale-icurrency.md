# <a name="locale_icurrency"></a>икурренци ЛОКАЛи \_

Расположение символа денежной единицы в режиме положительной валюты.



| Значение | Значение                                            |
|-------|----------------------------------------------------|
| 0     | Префикс, без разделения, например $1,1           |
| 1     | Суффикс, без разделения, например $1,1           |
| 2     | Префикс, 1-символьное разделение, например $1,1 |
| 3     | Суффикс, 1-символьное разделение, например $1,1 |
