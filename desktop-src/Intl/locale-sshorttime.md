# <a name="locale_sshorttime"></a>сшорттиме ЛОКАЛи \_

**Windows 7 и более поздних версий:** Строка краткого форматирования времени для языкового стандарта. Шаблоны обычно являются производными путем удаления значения SS (секунды) из шаблона длинного формата времени. Например, если используется длинный формат времени "ч: СС TT", короткий формат времени скорее всего будет "ч TT". Эта константа может указывать несколько форматов в списке, разделенном точкой с запятой. Однако предпочтительным коротким форматом времени должно быть первое указанное значение.
