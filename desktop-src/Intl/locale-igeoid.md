# <a name="locale_igeoid"></a>ижеоид ЛОКАЛи \_

32-разрядное число со знаком, уникально идентифицирующее [географическое расположение](table-of-geographical-locations.md). Приложение использует эту константу для предоставления клиентам служб, зависящих от языкового стандарта. Например, его можно использовать в качестве ключа для доступа к записи базы данных, содержащей конкретные сведения о стране или регионе.
