# <a name="microsoft-language-detection"></a>Microsoft распознавание языка

Служба обнаружения языка ELS называется Microsoft распознавание языка. Эта служба использует технологию, запатентованную корпорацией Майкрософт, чтобы позволить приложениям определять язык, на котором написан конкретный текст.

## <a name="input-to-microsoft-language-detection"></a>Входные данные в Microsoft распознавание языка

Входными данными для службы Microsoft распознавание языка является текст UTF-16 (Нормализованная форма C). Служба должна определить язык для этого текста.

## <a name="output-of-microsoft-language-detection"></a>Выходные данные Microsoft распознавание языка

Служба Microsoft распознавание языка извлекает строки в формате UTF-16 с двойным нулем, которые представлены с помощью имен, разделенных символами NULL. Список сортируется по релевантности. Для большинства языков используются нейтральные имена. Однако для некоторых, например, SR-Цирл, SR-ЛАТН, zh-Hant и zh-Ханс, используются полные имена.

## <a name="microsoft-language-detection-operation"></a>Операция Microsoft распознавание языка

Служба Microsoft распознавание языка проверяет скрипт Юникода текста, предоставленного приложением. Он разделяет текст на основе обнаруженных им скриптов, а затем определяет язык, на котором записан каждый сегмент. Если сценарий указывает на один язык, этот язык будет гарантированно представлен в списке выходных данных языков. Служба использует запатентованный алгоритм для определения релевантности каждого поддерживаемого языка.

## <a name="microsoft-language-detection-guid"></a>Идентификатор GUID распознавание языка Microsoft

Идентификатор GUID для службы Microsoft распознавание языка объявляется в Елссрвк. h, как показано в следующем коде.


```C++
// {CF7E00B1-909B-4d95-A8F4-611F7C377702}
static const GUID ELS_GUID_LANGUAGE_DETECTION =
    { 0xCF7E00B1, 0x909B, 0x4D95, { 0xA8, 0xF4, 0x61, 0x1F, 0x7C, 0x37, 0x77, 0x02 } };
```
