# <a name="locale_spositivesign"></a>споситивесигн ЛОКАЛи \_

Локализованное строковое значение для положительного знака языкового стандарта. Например, **языковой стандарт \_ споситивесигн** считается "+" для английского языка (США). Обратите внимание, что наличие данных для этой строки не означает, что при форматировании числа обязательно используется положительный знак. Если эти данные пусты или пусты, предполагается, что используется символ "+". Максимально допустимое число символов для этой строки равно пяти, включая завершающий нуль-символ.
