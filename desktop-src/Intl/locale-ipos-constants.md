# <a name="locale_ipos-constants"></a>\_Константы ИПОС языкового стандарта \*

В этом разделе определяются \_ константы ИПОС локали \* , используемые NLS.



| Значение                   | Значение                                                                                                                                                                   |
|-------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ипоссепбиспаце ЛОКАЛи \_  | Разделение символа денежной суммы в положительном денежном значении. Это значение равно 1, если символ денежной суммы отделяется пробелом от положительного числа, в противном случае — 0.        |
| ипоссигнпосн ЛОКАЛи \_    | Индекс форматирования для положительных значений. Индекс использует те же значения, что и [языковой \_ инегсигнпосн](locale-ineg-constants.md), за исключением того, что не использует нулевой индекс.    |
| ипоссимпрецедес ЛОКАЛи \_ | Расположение символа денежной суммы в положительном денежном значении. Это значение равно 1, если символ денежной суммы предшествует положительному значению, или 0, если символ соответствует сумме. |
