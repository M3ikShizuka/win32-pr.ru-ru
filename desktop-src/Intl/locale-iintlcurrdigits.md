# <a name="locale_iintlcurrdigits"></a>иинтлкуррдигитс ЛОКАЛи \_

\[ЯЗЫКовой стандарт \_ иинтлкуррдигитс не поддерживается и может быть изменен или недоступен в будущем. Вместо этого используйте [языковой стандарт \_ икуррдигитс](locale-icurrdigits.md).\]

Количество цифр дробной части для международного денежного формата. Максимально допустимое число символов для этой строки равно трем, включая завершающий нуль-символ. Например, количество цифр дробной части для международного денежного формата в долларах США равно 2, как в $345,25.
