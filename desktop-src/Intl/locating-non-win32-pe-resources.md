# <a name="locating-non-win32-pe-resources"></a>Поиск ресурсов, отличных от Win32 PE

Для размещения ресурсов, отличных от Win32 PE, приложение должно сначала вызвать функцию [**жетфилемуипас**](/windows/desktop/api/Winnls/nf-winnls-getfilemuipath) для размещения файла ресурсов, зависящего от языка, из которого загружаются ресурсы. Если приложение имеет следующие системные параметры языка, оно должно вызывать функцию с \_ именем языка MUI \_ \| \_ \_ предпочтительные \_ языки интерфейса пользователя MUI \_ , указанные для *DwFlags* , и **значение NULL** , указанное для *пвсзлангуаже*. Если приложение имеет следующие параметры языка приложения, оно использует **жетфилемуипас** , чтобы определить, существует ли зависящий от языка файл, указав язык в параметре *пвсзлангуаже* .

После вызова [**жетфилемуипас**](/windows/desktop/api/Winnls/nf-winnls-getfilemuipath)приложение должно определить пользовательские функции, чтобы загрузить модуль ресурсов и загрузить из него определенные ресурсы. Например, если вы используете .txt или .xml файл ресурсов, приложение должно использовать для загрузки файла файл TXT или средство синтаксического анализа XML, а затем анализировать содержимое файла для каждого необходимого ресурса.
