# <a name="locale_skeyboardstoinstall"></a>скэйбоардстоинсталл ЛОКАЛи \_

**Windows Vista и более поздние версии:** Разделенный точками с запятой список клавиатур, которые могут устанавливаться для языкового стандарта и использоваться внутри Windows. Количество символов в этой строке не ограничено. Чтобы получить имя активного идентификатора локали ввода (прежнее название раскладки клавиатуры), приложение может вызвать функцию [**жеткэйбоардлайаутнаме**](/windows/win32/api/winuser/nf-winuser-getkeyboardlayoutnamea) .
