# <a name="locale_sthousand"></a>ссаусанд ЛОКАЛи \_

Символы, используемые для разделения групп разрядов влево от десятичного разделителя. Максимально допустимое число символов для этой строки равно четырем, включая завершающий нуль-символ. Обычно эти группы представляют тысячи. Однако в зависимости от значения, указанного для параметра [locale \_ сграупинг](locale-sgrouping.md), они могут представлять что-то другое.
