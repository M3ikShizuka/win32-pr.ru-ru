# <a name="locale_smonthday"></a>смонсдай ЛОКАЛи \_

**Windows 7 и более поздних версий:** Строка форматирования для отображения только месяца и дня. Форматирование аналогично заданному для [языкового стандарта \_ слонгдате](locale-slongdate.md). Например, если шаблон "месяц/день" представляет собой полное название месяца, за которым следует номер дня с начальными нулями, как в "Сентябрь 03", строка формата будет иметь формат "мммм дд". Строка может состоять из сочетания форматов «день» и «месяц» и любой строки символов, заключенной в одинарные кавычки. Символы в одинарных кавычках остаются в указанном виде, например "de" для испанского языка (Испания). Языковой стандарт может указывать только один формат месяца/дня.
