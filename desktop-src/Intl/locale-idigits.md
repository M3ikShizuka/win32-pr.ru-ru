# <a name="locale_idigits"></a>идигитс ЛОКАЛи \_

Число цифр дробной части, помещенных после десятичного разделителя. Максимально допустимое число символов для этой строки равно двум, включая завершающий нуль-символ. Например, 2 для 5,00, 1 для 5,0.
