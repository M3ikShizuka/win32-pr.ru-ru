# <a name="international-fonts-and-text-display"></a>Международные шрифты и отображение текста

международные шрифты и текст — поддержка платформ для международных шрифтов, международного текста и типографского оформления на платформе Windows.

международные шрифты и текст создаются на основе основных технологий оформления, текста и расширенного чтения Windows для предоставления шрифтов и текста для пользователей Windowsной платформы на всем мире. Технология улучшает сквозные сценарии вывода шрифта и текста.

Включены следующие технологии:

-   [Управление международным шрифтом](about-international-fonts-and-text.md). Представляет основные функциональные возможности международных шрифтов.
-   [С помощью MS Shell Dlg и MS Shell Dlg 2](using-ms-shell-dlg-and-ms-shell-dlg-2.md). общие сведения о шрифтах ms shell dlg и ms shell dlg 2 и сопоставлении шрифтов для ms shell dlg и ms shell dlg 2 в разных версиях Windows.
-   [Перечисление международных шрифтов и выбор](using-international-fonts-and-text.md). Описание возможных способов перечисления международных шрифтов, позволяющих пользователям выбрать нужный шрифт.
-   [Международный вывод текста](creating-your-own-format-selection-user-interface.md). Описание возможных способов просмотра международного текста.
-   [Uniscribe](uniscribe.md). Обеспечивает высокий уровень контроля для тонкого оформления и обработки сложных наборов символов.
