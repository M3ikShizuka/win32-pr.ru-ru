# <a name="locale_seng-constants"></a>\_Константы Сенг языкового стандарта \*

В этом разделе определяются \_ константы Сенг локали \* , используемые NLS.



| Значение                | Значение                                                                                                                                                                   |
|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| сенгкаунтри ЛОКАЛи \_  | Не рекомендуется для Windows 7 и более поздних версий. Полное название страны или региона на английском языке. См. раздел [locale \_ сенглишкаунтринаме](locale-senglish-constants.md).                            |
| сенгкуррнаме ЛОКАЛи \_ | **Windows Me/98, windows 2000:** Полное название валюты, связанной с языковым стандартом. Количество символов в этой строке не ограничено. |
| сенглангуаже ЛОКАЛи \_ | Не рекомендуется для Windows 7 и более поздних версий. Полное название языка по стандарту ISO 639. См. раздел [locale \_ сенглишлангуаженаме](locale-senglish-constants.md).           |
