# <a name="locale_smon-constants"></a>\_Константы СМОН языкового стандарта \*

В этом разделе определяются \_ локальные \* константы смон, используемые NLS в представлениях денежных значений.



| Значение                   | Значение                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|-------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| смондеЦималсеп ЛОКАЛи \_  | Символы, используемые в качестве десятичного разделителя. Максимально допустимое число символов для этой строки равно четырем, включая завершающий нуль-символ. Например, если денежная сумма отображается как "$3,40", так же как "три доллара и 40 центов" отображается в США, десятичный разделитель ".".                                                                                                                                             |
| смонграупинг ЛОКАЛи \_    | Размеры для каждой группы денежных знаков, слева от десятичного разделителя. Максимально допустимое число символов для этой строки равно десяти, включая завершающий нуль-символ. Для каждой группы требуется явный размер, а размеры разделяются точками с запятой. Если Последнее значение равно 0, предыдущее значение повторяется. Например, чтобы сгруппировать тысячи, укажите значение 3; 0. Индийские языки группируют первые тысячи, а затем группируются по сотням. Например, 12, 34, 56789 представляется 3; 2; 0. |
| смонсаусандсеп ЛОКАЛи \_ | Символы, используемые как разделитель денежных знаков между группами цифр слева от десятичного разделителя. Максимально допустимое число символов для этой строки равно четырем, включая завершающий нуль-символ. Как правило, группы представляют тысячи. Однако в зависимости от значения, указанного для параметра LOCALE \_ смонграупинг, они могут представлять что-то другое.                                                                                                                                 |
