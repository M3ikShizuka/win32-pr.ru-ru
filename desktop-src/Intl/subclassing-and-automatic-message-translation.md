# <a name="subclassing-and-automatic-message-translation"></a>Подклассы и автоматическое преобразование сообщений

Создание подклассов — это метод, позволяющий приложению перехватывать и обрабатывать сообщения, отправленные или отправляемые в определенное окно, прежде чем процедура окна сможет их обработать. Операционная система автоматически преобразует сообщения в [кодовую страницу Windows (ANSI)](code-pages.md) или в форму [Юникода](unicode.md) в зависимости от формы функции, которая подклассировать процедуру окна.

Следующий вызов функции [**сетвиндовлонга**](/windows/win32/api/winuser/nf-winuser-setwindowlonga) подклассировать текущую процедуру окна, связанную с окном, определяемым параметром *HWND* . Кроме того, приложение может использовать [**сетвиндовлонгптра**](/windows/win32/api/winuser/nf-winuser-setwindowlongptra). Новая процедура окна **неввндпрок**, получит сообщения с текстом в формате кодовой страницы Windows.


```C++
OldWndProc = (WNDPROC) SetWindowLongA(hWnd,
             GWL_WNDPROC, (LONG)NewWndProc); 
```



Когда **неввндпрок** заканчивает обработку сообщения, он использует функцию [**каллвиндовпрок**](/windows/win32/api/winuser/nf-winuser-callwindowproca) следующим образом для передачи сообщения в **олдвндпрок**.


```C++
CallWindowProc(OldWndProc, hWnd, uMessage, wParam, lParam);
```



Если **олдвндпрок** был создан с использованием стиля класса Unicode, сообщения транслируются из формы кодовой страницы Windows, полученной из **неввндпрок** в Юникод.

Аналогичным образом, вызов функции [**сетвиндовлонгв**](/windows/win32/api/winuser/nf-winuser-setwindowlonga) или [**сетвиндовлонгптрв**](/windows/win32/api/winuser/nf-winuser-setwindowlongptra) подклассировать текущую процедуру окна с процедурой окна, ожидающей текстовые сообщения в Юникоде. При необходимости преобразование сообщения выполняется во время обработки функции [**каллвиндовпрок**](/windows/win32/api/winuser/nf-winuser-callwindowproca) .

Дополнительные сведения о подклассах см. в разделе [процедуры окна](../winmsg/window-procedures.md).
