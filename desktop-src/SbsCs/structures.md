# <a name="side-by-side-assembly-structures"></a>Параллельные структуры сборок

Параллельный API сборки использует следующие структуры.



| Структура                                                       | Описание                                                                            |
|-----------------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**\_сведения о сборке**](/windows/desktop/api/Winsxs/ns-winsxs-assembly_info)                        | Содержит сведения о сборке в параллельном хранилище сборок системы.    |
| [**\_Справочник по установке Fusion \_**](/windows/win32/api/winsxs/ns-winsxs-fusion_install_reference) | Содержит сведения о приложении, которое ссылается на параллельную сборку. |
