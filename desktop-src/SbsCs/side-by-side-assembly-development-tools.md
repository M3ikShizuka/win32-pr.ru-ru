# <a name="side-by-side-assembly-development-tools"></a>Средства разработки параллельных сборок

Следующие средства доступны только в пакете средств разработки программного обеспечения (SDK) для Microsoft Windows.



| Служебная программа                                | Описание                                                   |
|----------------------------------------|---------------------------------------------------------------|
| [Manifestchk.vbs](manifestchk-vbs.md) | Средство проверки манифеста.                                   |
| [Mt.exe](mt-exe.md)                   | Средство построения манифеста, которое создает подписанные файлы и каталоги. |
| [Pktextract.exe](pktextract-exe.md)   | Извлекает маркеры открытого ключа из файлов сертификатов.            |
