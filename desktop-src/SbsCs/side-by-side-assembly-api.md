# <a name="side-by-side-assembly-api"></a>Параллельный API сборки

Параллельные сборки API устанавливают, удаляют или запрашивают параллельные сборки из параллельного хранилища сборок системы. Это также называется хранилищем winSxS в Windows XP и хранилищем компонентов инфраструктуры обслуживания компонентов (CSI), начиная с Windows Vista и Windows Server 2008.

Части параллельных API-интерфейсов сборок описаны в следующих разделах:

-   [Параллельные перечисления сборок](enumerations.md)
-   [Параллельные функции сборки](side-by-side-assembly-functions.md)
-   [Параллельные интерфейсы сборки](side-by-side-assembly-interfaces.md)
-   [Параллельные структуры сборок](structures.md)
