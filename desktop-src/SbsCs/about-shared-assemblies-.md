# <a name="about-shared-assemblies"></a>Общие сведения о общих сборках

Общая сборка — это сборка, доступная для использования в нескольких приложениях на компьютере. В Windows Vista и Windows XP [Параллельные сборки](about-side-by-side-assemblies-.md) можно устанавливать как общие сборки. Общие параллельные сборки не регистрируются глобально в системе, но они глобально доступны для приложений, которые задают зависимость от сборки в [манифестах](manifests.md). Несколько версий параллельных сборок могут совместно использоваться разными приложениями, выполняющимися одновременно. Дополнительные сведения см. в разделе [о изолированных приложениях и параллельных сборках](about-isolated-applications-and-side-by-side-assemblies.md). Например, [Поддерживаемые параллельные сборки Microsoft](supported-microsoft-side-by-side-assemblies.md) , поставляемые с Windows XP, обычно используются в качестве общих сборок несколькими приложениями.

Общие параллельные сборки устанавливаются в папку WinSxS. Издатели сборок, приложения и администраторы могут изменять версию зависимостей параллельных сборок после развертывания с помощью [конфигурации](configuration.md). Совместно используемые параллельные сборки могут устанавливаться обновлением операционной системы или пакетом установщик Windows, который устанавливает или обновляет приложение. Дополнительные сведения см. в разделе [Установка сборок Win32](../msi/installation-of-win32-assemblies.md).

До Windows XP общие сборки были зарегистрированы глобально и установлены в системной папке Windows. В этом случае последняя установленная версия сборки доступна для любого приложения, которое выполняет привязку к ней. Параллельную сборку также можно установить как закрытую сборку для монопольного использования приложения. Дополнительные сведения см. в разделе [Закрытые сборки](/windows/desktop/Msi/private-assemblies).
