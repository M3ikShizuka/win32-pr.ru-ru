# <a name="microsoftwindowsactctx-object"></a>Объект Microsoft. Windows. Акткткс

Объект **Microsoft. Windows. акткткс** ссылается на манифесты и предоставляет обработчикам сценариев возможность доступа к параллельным сборкам. Объект **Microsoft. Windows. акткткс** можно использовать для создания экземпляра параллельной сборки с COM-компонентами.

Объект **Microsoft. Windows. акткткс** поставляется как сборка в Windows Server 2003. Она также может быть установлена приложениями, которые используют установщик Windows для установки и включают его в пакет установки в качестве модуля слияния.

## <a name="members"></a>Элементы

Объект **Microsoft. Windows. акткткс** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **Microsoft. Windows. акткткс** содержит следующие методы.



| Метод                               | Описание                                                                     |
|:-------------------------------------|:--------------------------------------------------------------------------------|
| [**CreateObject**](createobject.md) | Создает объект в контексте текущего манифеста.<br/>            |
| [**GetObject**](getobject.md)       | Возвращает экземпляр существующего объекта **Microsoft. Windows. акткткс** .<br/> |

Объект **Microsoft. Windows. акткткс** имеет следующие свойства.



| Свойство                                | Тип доступа          | Описание                                    |
|:----------------------------------------|:---------------------|:-----------------------------------------------|
| [**Явление**](manifest.md)<br/> | Только для чтения<br/> | Возвращает активный контекст активации.<br/> |
