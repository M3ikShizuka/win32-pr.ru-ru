# <a name="microsoftwindowsactctx-object"></a>NNTP. Windows. Объект Акткткс

**Microsoft. Windows. Объект Акткткс** ссылается на манифесты и предоставляет обработчикам сценариев возможность доступа к параллельным сборкам. **Microsoft. Windows. Объект Акткткс** можно использовать для создания экземпляра параллельной сборки с COM-компонентами.

**Microsoft. Windows. объект акткткс** входит в сборку в Windows Server 2003. она также может быть установлена приложениями, которые используют установщик Windows для установки и включают его в пакет установки в качестве модуля слияния.
