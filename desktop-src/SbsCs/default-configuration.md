# <a name="default-configuration"></a>Конфигурация по умолчанию

Версии сборок, указанные в [манифестах сборок](assembly-manifests.md) и [манифестах приложений](application-manifests.md) , представляют собой конфигурацию по умолчанию, которую приложение должно использовать при запуске приложения. приложения и администраторы могут переопределить конфигурацию сборки по умолчанию, установив [конфигурацию издателя](publisher-configuration.md) или [конфигурации для каждого приложения](per-application-configuration.md) на Windows Server 2003.
