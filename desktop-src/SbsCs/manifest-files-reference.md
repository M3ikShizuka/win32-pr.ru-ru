# <a name="manifest-files-reference"></a>Справочник по файлам манифеста

Параллельные сборки используются со следующими типами манифестов и файлов конфигурации. Манифесты и конфигурации — это XML-файлы.



| Manifest                                                               | Описание                                                                                                                                                                                                   |
|------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Манифесты сборки](assembly-manifests.md)                           | Описывает имена, версии, ресурсы и зависимости сборок параллельных сборок.                                                                                                               |
| [Манифесты приложений](application-manifests.md)                     | Описывает имена и версии общих параллельных сборок, с которыми приложение должно выполнять привязку во время выполнения, а также может содержать метаданные для частных параллельных сборок, используемых приложением. |
| [Файлы конфигурации приложений](application-configuration-files.md) | Перенаправляет версии сборки зависимостей сборок, используя [конфигурацию для каждого приложения](per-application-configuration.md).                                                                            |
| [Publisher Файлы конфигурации](publisher-configuration-files.md)     | Перенаправляет версии сборки зависимостей сборок на основе каждой сборки, используя [конфигурацию издателя](publisher-configuration.md).                                                              |



 

Список схемы файла манифеста см. в разделе [Схема файла манифеста](manifest-file-schema.md).

Список схемы файла конфигурации приложения см. в разделе [Схема файла конфигурации приложения](application-configuration-file-schema.md).

список схемы файла конфигурации издателя см. в разделе [Publisher схема файла конфигурации](publisher-configuration-file-schema.md).
