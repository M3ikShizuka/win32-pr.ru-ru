# <a name="side-by-side-assembly-interfaces"></a>Параллельные интерфейсы сборки

Параллельный API сборки использует следующие интерфейсы.



| Интерфейс                                        | Описание                                                                                                                                            |
|--------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAssemblyCache**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycache)         | Используется для установки, удаления или запроса параллельной сборки.                                                                                          |
| [**IAssemblyCacheItem**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycacheitem) | Используется для установки параллельных сборок в Параллельное хранилище с использованием установки на основе потока.                                                 |
| [**IAssemblyName**](/windows/desktop/api/winsxs/nn-winsxs-iassemblyname)           | Представляет имя параллельной сборки. Параллельное имя сборки состоит из набора пар «имя-значение», описывающих параллельную сборку. |
