Параллельный API сборки использует следующие функции для доступа к параллельным интерфейсам сборки.



| Функция                                                     | Описание                                                                    |
|--------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**креатеассембликаче**](/windows/desktop/api/Winsxs/nf-winsxs-createassemblycache)           | Получает экземпляр интерфейса [**IAssemblyCache**](/windows/desktop/api/winsxs/nn-winsxs-iassemblycache) . |
| [**креатеассемблинамеобжект**](/windows/desktop/api/Winsxs/nf-winsxs-createassemblynameobject) | Получает экземпляр интерфейса [**IAssemblyName**](/windows/desktop/api/winsxs/nn-winsxs-iassemblyname) .   |
