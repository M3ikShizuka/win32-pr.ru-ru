# <a name="supported-microsoft-side-by-side-assemblies"></a>Поддерживаемые параллельные сборки Майкрософт

Ниже перечислены параллельные сборки, поддерживаемые корпорацией Майкрософт. Пакеты обновления Windows могут включать совместимые версии этих сборок с исправлениями безопасности или ошибок.

Доступные распространяемые пакеты можно найти в [центре загрузки Майкрософт](https://www.microsoft.com/Downloads/).

## <a name="windows-server-2003"></a>Windows Server 2003

Эти параллельные сборки поддерживаются корпорацией Майкрософт начиная с Windows Server 2003.

-   WinHTTP 5,1
-   Сборки службы автоматизации Microsoft Isolation, позволяющие использовать параллельные сборки с скриптами.
-   Общие элементы управления оболочки версии 6,0 (Comctl32.dll)
-   GDI Plus версии 1,0 (GDIplus.dll)
-   Библиотеки времени выполнения Visual C++ версии 6,0

## <a name="windows-xp"></a>Windows XP

Эти параллельные сборки поддерживались корпорацией Майкрософт начиная с Windows XP.

-   Общие элементы управления оболочки версии 6,0 (Comctl32.dll)
-   GDI Plus версии 1,0 (GDIplus.dll)
-   Библиотеки времени выполнения Visual C++ версии 6,0
