# <a name="i-isolated-applications-and-side-by-side-assemblies"></a>I (изолированные приложения и параллельные сборки)

<dl> <dt>

<span id="_win32_isolated_application_gly"></span><span id="_WIN32_ISOLATED_APPLICATION_GLY"></span>**изолированное приложение**
</dt> <dd>

Приложения, использующие параллельные сборки, но не использующие их сборки. Изолированное приложение всегда сопровождается файлом манифеста приложения. Манифест указывает версии общих параллельных сборок, к которым приложение привязывается во время выполнения, и может содержать некоторые метаданные для частных параллельных сборок. Манифест содержит сведения, которые обычно хранятся в реестре; полностью изолированное приложение может быть независимым от реестра.

</dd> </dl>
