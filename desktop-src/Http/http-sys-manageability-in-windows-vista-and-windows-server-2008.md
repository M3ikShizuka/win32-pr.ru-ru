# <a name="httpsys-manageability-in-windows-vista-and-windows-server-2008"></a>HTTP.sys управляемости в Windows Vista и Windows Server 2008

В Windows Vista и Windows Server 2008 ИТ-специалисты найдут улучшенные функции управления для компонента API сервера HTTP, а также поддержку ведения журнала HTTP, представленную в Windows Server 2003. Чтобы диагностировать и настроить компонент API сервера HTTP, ИТ-специалисты могут использовать три средства:

-   Новые функции трассировки ETW
-   Счетчики производительности
-   Команды netsh Configuration

API сервера HTTP позволяет приложениям взаимодействовать по протоколу HTTP и включать поддержку SSL, чтобы приложения могли обмениваться данными через защищенные HTTP-соединения. API сервера HTTP поддерживается в операционных системах Windows Server 2003 и в Windows XP с пакетом обновления 2 (SP2).
