# <a name="deleting-a-reservation"></a>Удаление резервирования

Процесс удаления резервирования начинается с проверки доступа, которая идентична описанной выше. Идентификатор пользователя вызывающего процесса должен иметь права доступа к *родительскому узлу* удаляемой записи резервирования.
