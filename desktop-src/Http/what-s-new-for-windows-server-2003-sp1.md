# <a name="whats-new-for-windows-server-2003-sp1"></a>новые возможности Windows Server 2003 с пакетом обновления 1 (SP1)

для Windows Server 2003 с пакетом обновления 1 (SP1) доступны следующие функции:

-   [Поддержка SSL в режиме ядра.](kernel-mode-ssl.md)
    -   Windows Сервер 2003 с пакетом обновления 1 (SP1) добавляет параметр для выполнения операций SSL в режиме ядра для повышения производительности.
-   Поддержка выполнения 32 разрядных приложений в 64-разрядной операционной системе (WOW64).
-   [Улучшения в ведении журнала ошибок HTTP.](error-logging-in-windows-server-2003-sp1.md)
    -   Поддержка заголовков в стиле W3C.
    -   Доступны дополнительные поля ведения журнала.
    -   Операция переключения на файл журнала, основанную на дате и времени.
-   [Исключения синтаксического анализа HTTP.](parsing-exceptions.md)
    -   Поддерживает исключения HTTP/1.1 для обратной совместимости.
-   [Усовершенствования средства синтаксического анализа HTTP.](parser-enhancements.md)
