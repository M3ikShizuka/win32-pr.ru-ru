# <a name="httpsys-manageability-scenarios"></a>Сценарии управляемости HTTP.sys

В следующих сценариях приведены примеры того, как эти средства могут помочь ИТ-специалистам диагностировать и устранять проблемы, связанные с компонентом HTTP Server API (HTTP.sys). Средства, используемые в этих примерах, доступны в Windows Vista и Windows Server 2008.

-   [Сценарий 1. пример времени ожидания HTTP с использованием трассировки ETW и команд Netsh](scenario-1--http-timeout-example-using-etw-tracing-and-netsh-commands.md)
-   [Сценарий 2. Пример средства синтаксического анализа с использованием трассировки ETW](scenario-2--parser-example-using-etw-tracing.md)
-   [Сценарий 3. счетчики производительности](scenario-3--performance-counters.md)

Сценарии в этом документе являются примерами использования средств управляемости компонента API сервера HTTP. Эти примеры не предназначены для предложения конкретных изменений конфигурации.
