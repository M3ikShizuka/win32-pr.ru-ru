# <a name="formatting-legend"></a>Условные обозначения форматирования

В разделе используются следующие соглашения о форматировании.

| Формат                                                                    | Значение                                                        |
|---------------------------------------------------------------------------|----------------------------------------------------------------|
| Курсив                                                                    | Сведения, которые должен предоставить пользователь                          |
| Жирный                                                                      | Элементы, которые пользователь должен ввести в точности так, как показано              |
| Многоточие (...)                                                            | Параметр, который может повторяться несколько раз в командной строке |
| Между скобками ( \[ \] )                                                   | Необязательные элементы                                                 |
| Между фигурными скобками ( {} ); варианты, разделенные вертикальной чертой ( \| ). Пример: {четных и \| нечетных} | Набор вариантов, из которых пользователь должен выбрать только один        |
| Шрифт Courier                                                              | Программный код или данные, выводимые программой                                         |
