# <a name="http-version-20-features"></a>Функции HTTP версии 2,0

Для HTTP-API версии 2,0 поддерживается Следующая функция.

-   Объекты конфигурации сеанса сервера и группы URL-адресов
-   Свойства конфигурации, предоставляемые для объектов конфигурации
-   Расширенные возможности управления очередями запросов и настройки
-   Полный режим ядра SSL
-   Кэш ответов режима ядра
-   Проверка подлинности на стороне сервера
-   Управление версиями с областью действия объекта
-   Ведение журнала на стороне сервера
-   Корректное завершение работы очереди запросов
-   Запуск запроса в очереди запросов
-   Совместное использование портов
-   трассировка событий для поддержки Windows
-   Счетчики производительности
-   Поддержка международного доменного имени (IDN)
-   Служебная программа настройки оболочки сети
