# <a name="show-cachestate"></a>show cachestate

Перечисляет все ресурсы и связанные с ними свойства, которые кэшируются в кэше HTTP-ответов, или отображает один ресурс и связанные с ним свойства.

``` syntax
show cachestate [[url=]string]
 
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="__url__string_"></span><span id="__URL__STRING_"></span>**\[\[URL-адрес = \] ***строка***\]**
</dt> <dd>

Полный URL-адрес. Если не указано, подразумевает использование всех URL-адресов. URL-адрес также может быть префиксом для зарегистрированных URL-адресов.

</dd> </dl>

## <a name="examples"></a>Примеры

**показывать URL-адрес качестате =https://www.contoso.com:80/myresource**
