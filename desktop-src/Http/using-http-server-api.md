# <a name="using-http-server-api"></a>Использование API сервера HTTP

Исходные файлы программы приложения включают файл заголовка HTTP. h для доступа к прототипам функций и определения структур для API-сервера HTTP. Разработчики могут использовать файл библиотеки Httpapi. lib для создания приложений, использующих API-сервер HTTP. Во время выполнения приложения связываются с Httpapi.dll.
