# <a name="show-iplisten"></a>show iplisten

Перечисляет все IP-адреса в списке прослушивания IP-адресов. Список прослушивания IP-адресов используется для определения области списка адресов, к которым привязывается служба HTTP. "0.0.0.0" означает любой IPv4-адрес, а "::" — любой IPv6-адрес.

``` syntax
show iplisten
 
```
