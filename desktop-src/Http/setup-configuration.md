# <a name="setup-configuration"></a>Конфигурация установки

Для настройки программы установки требуются права администратора и они сохраняются при перезапуске. Как правило, приложение установки, работающее с правами администратора, выполняет такую постоянную настройку во время установки, чтобы приложения могли работать с более низким уровнем привилегий, хотя Постоянная конфигурация может быть выполнена в любое время. Конфигурация установки может включать любое из следующих четырех действий:

-   Создание резервирования URL-адреса. API HTTP позволяет приложению установки зарезервировать префиксы URL-адресов для очередей запросов, связанных с конкретными приложениями. Чтобы зарезервировать URL-адрес, приложение установки вызывает функцию [**хттпсетсервицеконфигуратион**](/windows/desktop/api/Http/nf-http-httpsetserviceconfiguration) с параметром *конфигид* , для которого задано значение **хттпсервицеконфигурлаклинфо** , и передает указатель в параметре *пконфигинформатион* в структуру [**\_ Set HTTP Service \_ config \_ URLACL \_**](/windows/desktop/api/Http/ns-http-http_service_config_urlacl_set) , содержащую сведения о регистрации. Дополнительные сведения см. в разделе [Резервирование пространств имен, регистрации и маршрутизация](namespace-reservations-registrations-and-routing.md).
-   Настройка SSL. Чтобы настроить SSL, администратор вызывает функцию [**хттпсетсервицеконфигуратион**](/windows/desktop/api/Http/nf-http-httpsetserviceconfiguration) с параметром *конфигид* , установленным в **хттпсервицеконфигсслцертинфо** , и передает указатель в параметре *Пконфигинформатион* в [**структуру \_ \_ \_ \_ набора SSL настройки HTTP-службы**](/windows/desktop/api/Http/ns-http-http_service_config_ssl_set) , содержащую сведения о сертификате SSL.
-   Настройка других постоянных конфигураций на основе HTTP-сервера, таких как IP-адреса, на которых HTTP-сервер будет прослушиваться, и значение времени ожидания на уровне сервера. См. раздел [список прослушивания IP-адресов](ip-listen-list.md) и [**\_ \_ \_ \_ Настройка времени ожидания конфигурации службы HTTP**](/windows/desktop/api/Http/ns-http-http_service_config_timeout_set).
