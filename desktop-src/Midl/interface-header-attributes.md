# <a name="interface-header-attributes"></a>Атрибуты заголовка интерфейса

Включите эти атрибуты в заголовок интерфейса, чтобы передать сведения обо всем интерфейсе.



| attribute                                   | Использование                                                                                                                                                                                                                                            |
|---------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**асинхронный \_ UUID**](async-uuid.md)           | Направляет компилятор MIDL для определения синхронных и асинхронных версий COM-интерфейса.                                                                                                                                               |
| [**UUID**](uuid.md)                        | Обозначает 128-разрядное значение, которое отличает определенный интерфейс от других. Фактическое значение может представлять GUID, CLSID или IID.                                                                                                 |
| [**Языковые**](local.md)                      | Указывает компилятору MIDL создавать только файлы заголовков. Интерфейс должен иметь либо [**UUID**](uuid.md) , либо [**локальный**](local.md) атрибут.                                                                                             |
| [**MS \_ Union**](-ms-union.md)              | Управляет выравниванием ОНД неинкапсулированных объединений. Используется для обратной совместимости с интерфейсами, созданными на основе MIDL 1,0 или 2,0.                                                                                                                   |
| [**object**](object.md)                    | Определяет интерфейс как COM-интерфейс и направляет компилятору MIDL запрос на создание кода прокси или заглушки вместо заглушек клиента и сервера RPC.                                                                                                    |
| [**Версия**](version.md)                  | Определяет конкретную версию интерфейса в случаях, когда существует несколько версий интерфейса. Поскольку COM-интерфейсы являются неизменяемыми, нельзя использовать атрибут [**Version**](version.md) в интерфейсе [**объектов**](object.md) . |
| [**Указатель \_ по умолчанию**](pointer-default.md) | Указывает тип указателя по умолчанию для всех указателей, кроме тех, которые входят в списки параметров. Тип по умолчанию может быть [**уникальным**](unique.md), [**ref**](ref.md)или [**ptr**](ptr.md).                                                   |
| [**конечной**](endpoint.md)                | Задает статическую (хорошо известную) конечную точку, на которой серверное приложение будет прослушивать удаленные вызовы процедур.                                                                                                                                   |



 

См. раздел [атрибуты библиотеки типов](type-library-attributes.md) для атрибутов интерфейса, таких как [**Dual**](dual.md) и [**oleautomation**](oleautomation.md), которые относятся к интерфейсам, определенным или упоминаемым в операторе Library.
