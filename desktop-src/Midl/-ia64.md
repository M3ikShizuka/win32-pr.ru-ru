# <a name="ia64-switch"></a>/ia64, параметр

Параметр **/ia64** направляет компилятор MIDL для создания файлов заглушек или файла библиотеки типов для архитектуры Intel 64-bit.

``` syntax
midl /ia64
```

## <a name="remarks"></a>Remarks

> [!Note]  
> невозможно использовать MIDL.exe для создания 64-разрядного tlb на Windows 2000.
