# <a name="data-type-attributes"></a>Атрибуты типа данных

Эти атрибуты можно применить к типам данных в операторе [**typedef**](typedef.md) , чтобы дополнительно определить использование или воздействие типа данных.



| attribute                                 | Использование                                                                                                                                                                                                                                                                                                                      |
|-------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Контекстный \_ маркер**](context-handle.md) | Определяет маркер привязки, который хранит сведения о состоянии (контекста) на определенном сервере между вызовами удаленных процедур из определенного клиента. Недопустимый для функций [**объектного**](object.md) интерфейса.                                                                                                         |
| [**handle**](handle.md)                  | Указывает настраиваемый тип обработчика, относящийся к приложению.                                                                                                                                                                                                                                                                |
| [**MS \_ Union**](-ms-union.md)            | Управляет выравниванием ОНД неинкапсулированных объединений. Используйте в [**typedef**](typedef.md)для обеспечения обратной совместимости с интерфейсами, созданными с помощью MIDL 1,0 или 2,0.                                                                                                                                                            |
| [**дать**](pipe.md)                      | Разрешает передачу открытого потока типизированных данных по удаленному вызову процедуры. Параметр [**in**](in.md) pipe позволяет серверу получать поток данных от клиента во время удаленного вызова процедуры. Параметр [**out**](-out.md) pipe позволяет серверу передать поток данных обратно клиенту. |
| [**передать \_ как**](transmit-as.md)       | Указывает, как тип данных будет передаваться по сети, используемый для настраиваемого маршалирования.                                                                                                                                                                                                                                  |
| [**\_перечисление v1**](v1-enum.md)               | Указывает, что указанный перечисляемый тип передается как 32-разрядная сущность, а не как 16-разрядное значение по умолчанию.                                                                                                                                                                                                              |
| [**проводное \_ маршалирование**](wire-marshal.md)     | Аналогично [**передаче \_**](transmit-as.md) , но вы реализуете подпрограммы для изменения размера, маршалирования, распаковки и освобождения данных.                                                                                                                                                                                              |
