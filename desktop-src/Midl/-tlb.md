# <a name="tlb-switch"></a>параметр/tlb

Параметр **/tlb** указывает имя файла для библиотеки типов, создаваемой компилятором MIDL.

``` syntax
midl /tlb filename
```

## <a name="switch-options"></a>Параметры коммутатора

<dl> <dt>

*filename* 
</dt> <dd>

Указывает имя выходного файла библиотеки типов (TLB). По умолчанию, если параметр **/tlb** не используется, файл TLB имеет то же имя, что и IDL-файл, с расширением. TLB.

</dd> </dl>

## <a name="examples"></a>Примеры

**MIDL/TLB newname. tlb**
