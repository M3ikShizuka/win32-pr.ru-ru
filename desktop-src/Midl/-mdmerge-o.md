# <a name="o-switch-mdmerge"></a>параметр/o (МДМЕРЖЕ)

Параметр **/o** указывает выходной каталог для составных файлов метаданных.

``` syntax
mdmerge /o output_directory
```

## <a name="switch-options"></a>Параметры коммутатора

<dl> <dt>

*выходной \_ Каталог* 
</dt> <dd>

Указывает каталог, содержащий объединенные файлы метаданных.

</dd> </dl>
