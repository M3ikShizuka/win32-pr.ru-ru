# <a name="midl-and-com"></a>MIDL и COM

Все COM-объекты реализуют один или несколько интерфейсов. При создании пользовательского COM-объекта необходимо описать интерфейс или интерфейсы в IDL-файле. В подразделах этого раздела представлены сведения об использовании MIDL для определения COM-интерфейсов.

-   [Файлы, созданные для COM-интерфейса](files-generated-for-a-com-interface.md)
-   [Маршалирование типов данных OLE](marshaling-ole-data-types.md)

Дополнительные сведения об определении COM-интерфейсов с помощью MIDL см. в разделе [Определение COM-интерфейсов](../com/defining-com-interfaces.md). Дополнительные сведения о создании созданных файлов для различных платформ см. в разделе [нацеленность на заглушки для определенных 32-разрядных или 64-разрядных платформ](targeting-stubs-for-specific-32-bit-or-64-bit-platforms.md).
