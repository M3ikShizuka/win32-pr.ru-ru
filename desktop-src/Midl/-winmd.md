# <a name="winmd-switch"></a>/WinMD, параметр

Параметр [**/WinMD**](-metadata-dir.md) указывает имя выходного файла метаданных (WinMD).

``` syntax
midlrt /winmd metadata_filename
```

## <a name="switch-options"></a>Параметры коммутатора

<dl> <dt>

*\_имя файла метаданных* 
</dt> <dd>

Указывает имя файла метаданных.

</dd> </dl>
