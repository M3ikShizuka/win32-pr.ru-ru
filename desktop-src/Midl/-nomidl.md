# <a name="nomidl-switch"></a>/номидл, параметр

Параметр **/номидл** подавляет выполнение MIDL.EXE после обработки IDL-файла среда выполнения Windows.

``` syntax
midlrt /nomidl 
```

## <a name="switch-options"></a>Параметры коммутатора

Этот параметр не имеет параметров.

## <a name="remarks"></a>Remarks

## <a name="examples"></a>Примеры

**midlrt/номидл**
