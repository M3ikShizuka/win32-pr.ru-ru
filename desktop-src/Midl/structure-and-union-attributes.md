# <a name="structure-and-union-attributes"></a>Атрибуты структуры и объединения

Используйте атрибуты **\_ переключателя** , \* чтобы указать характеристики объединения в удаленном вызове процедуры. Используйте атрибут [**Ignore**](ignore.md) , чтобы назначить определенную структуру или члены объединения в качестве локальных для клиентского приложения.



| attribute                           | Использование                                                                                                                         |
|-------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**ключом**](switch.md)            | Выбирает discriminant для инкапсулированного объединения.                                                                           |
| [**параметр \_ имеет**](switch-is.md)     | Определяет discriminant для неинкапсулированного объединения.                                                                      |
| [**Тип коммутатора \_**](switch-type.md) | Определяет тип discriminant для неинкапсулированного объединения.                                                          |
| [**обращать**](ignore.md)            | Указывает, что указатель, содержащийся в структуре или объединении, и объект, указанный указателем, не передаются. |



 

Можно также использовать атрибуты « [массив» и «размер указателя](array-and-sized-pointer-attributes.md) » для определения характеристик элементов структуры или объединения.
