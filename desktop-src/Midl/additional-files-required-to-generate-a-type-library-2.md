# <a name="additional-files-required-to-generate-a-type-library"></a>Дополнительные файлы, необходимые для создания библиотеки типов

Для компиляции IDL-файла, содержащего инструкцию Library, файлы DLL OLE и OLE-автоматизации должны находиться в системе. Эти файлы устанавливаются автоматически.
