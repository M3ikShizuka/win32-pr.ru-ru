# <a name="ns_prefix-switch"></a>\_переключатель префикса/НС

Ключ **\_ префикса/НС** добавляет префикс ABI ко всем типам.

``` syntax
midlrt /ns_prefix 
```

## <a name="switch-options"></a>Параметры коммутатора

Этот параметр не имеет параметров.
