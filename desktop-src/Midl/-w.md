# <a name="w-switch"></a>Параметр/w

Параметр **/w** указывает уровень предупреждений компилятора MIDL. Уровень предупреждения указывает на серьезность предупреждения.

``` syntax
midl /W level
```

## <a name="switch-options"></a>Параметры коммутатора

<dl> <dt>

*level* 
</dt> <dd>

Задает уровень предупреждений — целое число в диапазоне от 0 до 4. Между параметром **/w** нет пробелов и цифрой, указывающей на значение уровня предупреждения.

</dd> </dl>

## <a name="remarks"></a>Remarks

Уровни предупреждений находятся в диапазоне от 1 до 4 и имеют нулевое значение для вывода сведений о предупреждении. Предупреждение о наивысшей серьезности имеет уровень 1. В следующей таблице описаны предупреждения для каждого уровня предупреждений.



| Уровень предупреждений | Описание                                             | Пример                                                                   |
|---------------|---------------------------------------------------------|---------------------------------------------------------------------------|
| W0            | Нет предупреждений.                                            |                                                                           |
| W1            | Серьезные предупреждения, которые могут вызвать ошибки приложения.      | Не заданы обработчики привязки, неатрибуты указатели, конфликтующие переключатели. |
| W2            | Может вызвать проблемы в операционной среде пользователя. | Длина идентификатора превышает 31 символ. Не указана ARM объединения по умолчанию.  |
| W3            | Зарезервировано.                                               |                                                                           |
| W4            | Самый низкий уровень предупреждений.                                   | Конструкции C, отличные от ANSI.                                                    |



 

Предупреждения отличаются от ошибок. Ошибки приводят к остановке обработки IDL-файла компилятором MIDL. Предупреждения вызывают вывод информационного сообщения компилятором MIDL и продолжение обработки IDL-файла.

Уровень предупреждений, заданный параметром **/w** , можно использовать с параметром [**/WX**](-wx.md) , чтобы компилятор MIDL приостановить обработку IDL-файла.

Параметр **/w** ведет себя так же, как параметр [**/warn**](-warn.md) .

## <a name="examples"></a>Примеры

**MIDL/W2 filename. idl**

**MIDL/W4 Bar. idl**
