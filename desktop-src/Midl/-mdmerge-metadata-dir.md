# <a name="metadata_dir-switch-mdmerge"></a>параметр/metadata_dir (МДМЕРЖЕ)

Параметр **\_ dir/Метадата** указывает каталог, содержащий файлы метаданных платформы.

``` syntax
mdmerge /metadata_dir metadata_directory
```

## <a name="switch-options"></a>Параметры коммутатора

<dl> <dt>

*\_Каталог метаданных* 
</dt> <dd>

Указывает каталог, содержащий файлы метаданных платформы.

</dd> </dl>

## <a name="remarks"></a>Remarks

используйте этот параметр, чтобы указать расположение основного файла метаданных для Windows, который называется Windows. winmd.
