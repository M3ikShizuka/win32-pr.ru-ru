# <a name="i-switch"></a>/i, параметр

Параметр **/i** указывает каталог, содержащий файлы метаданных для слияния.

``` syntax
mdmerge /i metadata_directory
```

## <a name="switch-options"></a>Параметры коммутатора

<dl> <dt>

*\_Каталог метаданных* 
</dt> <dd>

Указывает каталог, содержащий файлы метаданных для слияния.

</dd> </dl>
