# <a name="performance-attributes"></a>Атрибуты производительности

Используйте следующие атрибуты в IDL-файле, чтобы уменьшить размер кода-заглушки или иным образом повысить производительность.



| attribute                             | Использование                                                                                                                                                |
|---------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**обращать**](ignore.md)              | Указывает, что указатель, содержащийся в структуре или объединении, и объект, указанный указателем, не передаются.                        |
| [**Языковые**](local.md)                | Определяет локальную функцию для приложения, для которой MIDL не должен создавать код-заглушку.                                           |
| [**проводное \_ маршалирование**](wire-marshal.md) | Определяет тип данных как более простой тип для передачи по сети и позволяет реализовать упаковку и деупаковку подпрограмм для типа. |
