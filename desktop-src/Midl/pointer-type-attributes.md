# <a name="pointer-type-attributes"></a>Атрибуты типа указателя

Следующие атрибуты определяют характеристики указателей.



| attribute                                   | Использование                                                                                                                                                                                                |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ptr**](ptr.md)                          | Назначает указатель как полный указатель со всеми возможностями указателя языка C, включая псевдонимы.                                                                                       |
| [**ref**](ref.md)                          | Обозначает простейший тип указателя в MIDL — один, который просто предоставляет адрес некоторых данных. Ссылочные указатели никогда не могут иметь значение null.                                                             |
| [**однозначно**](unique.md)                    | Позволяет указателю иметь значение null, но не поддерживает псевдонимы.                                                                                                                                               |
| [**Указатель \_ по умолчанию**](pointer-default.md) | Применяется к интерфейсу для указания типа указателя по умолчанию для всех указателей в этом интерфейсе, за исключением указателей на параметры верхнего уровня, которые по умолчанию автоматически задаются как [**Ссылочные**](ref.md) указатели. |
| [**IID \_ —**](iid-is.md)                   | Предоставляет идентификатор интерфейса COM, который является объектом указателя.                                                                                                            |
| [**Строка**](string.md)                    | Указывает, что указатель указывает на строку.                                                                                                                                                       |
