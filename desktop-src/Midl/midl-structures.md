# <a name="midl-structures"></a>Структуры MIDL

Обычная семантика C применяется к полям базовых типов. Поля с более сложными типами, такими как указатели, массивы и другие сконструированные типы, могут быть изменены [**\_ атрибутами**](/windows/desktop/Rpc/field-attributes)типа или поля. Дополнительные сведения см. в разделе [**Структура**](struct.md).
