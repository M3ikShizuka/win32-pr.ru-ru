# <a name="partial-switch"></a>/партиал, параметр

Параметр **/партиал** указывает, что неразрешенные типы разрешаются для файлов метаданных, перечисленных с помощью параметра [**/Метадата \_ dir**](-mdmerge-metadata-dir.md) .

``` syntax
mdmerge /partial 
```

## <a name="switch-options"></a>Параметры коммутатора

Этот параметр не имеет параметров.
