# <a name="directional-attributes"></a>Атрибуты направления

Примените к параметрам атрибуты направления, чтобы указать направление передачи данных. Оба атрибута могут применяться к одному параметру, но это увеличивает издержки, связанные с вызовом.



| attribute              | Использование                                                                 |
|------------------------|-----------------------------------------------------------------------|
| [**окне**](in.md)       | Параметр передается из вызывающей стороны в вызываемую функцию. |
| [**заполняет**](out-idl.md) | Параметр передается из вызванной функции в вызывающий объект. |
