# <a name="nomd-switch"></a>/номд, параметр

Параметр **/номд** указывает, что файл метаданных (. winmd) не создается.

``` syntax
midlrt /nomd
```

## <a name="switch-options"></a>Параметры коммутатора

Этот параметр не имеет параметров.
