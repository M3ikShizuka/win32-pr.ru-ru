# <a name="o-switch"></a>параметр/o

Параметр **/o** перенаправляет выходные данные, например сообщения об ошибках и предупреждения, из стандартных выходных данных в указанный файл.

``` syntax
midl /o outputfile
```

## <a name="switch-options"></a>Параметры коммутатора

<dl> <dt>

*OutputFile* 
</dt> <dd>

Указывает имя файла, в который компилятор MIDL будет перенаправлять выходные данные, например сообщения об ошибках и предупреждения.

</dd> </dl>
