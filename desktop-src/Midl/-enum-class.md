# <a name="enum_class-switch"></a>/енум, \_ переключатель класса

Переключатель **\_ класса/енум** позволяет использовать конструкцию класса **enum** C++.

``` syntax
midlrt /enum_class 
```

## <a name="switch-options"></a>Параметры коммутатора

Этот параметр не имеет параметров.
