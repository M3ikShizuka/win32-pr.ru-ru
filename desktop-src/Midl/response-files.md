# <a name="response-files"></a>Файлы ответов

В качестве альтернативы размещению всех параметров в командной строке компилятор MIDL принимает файлы ответов, содержащие переключатели и аргументы. Файл ответов — это текстовый файл, содержащий один или несколько параметров командной строки компилятора MIDL. В отличие от командной строки, файл ответов допускает несколько строк параметров и имен файлов. Это может быть полезно из-за ограничений среды сборки или в качестве удобства для процесса сборки. Файл ответов MIDL можно указать следующим образом:

 **\@ имя файла** MIDL

<dl> <dt>

<span id="filename"></span><span id="FILENAME"></span>*файлов*
</dt> <dd>

Указывает имя файла ответов. Имя файла ответа должно сразу следовать за символом @ без пробелов между символом @ и именем файла ответа.

</dd> </dl>

Параметры в файле ответов обрабатываются так, как если бы они присутствовали в этом месте в командной строке MIDL. Каждый аргумент в файле ответов должен начинаться и заканчиваться на одной строке. Нельзя использовать символ обратной косой черты ( \) для сцепления линий).

MIDL поддерживает аргументы командной строки, включающие один или несколько файлов ответов в сочетании с другими параметрами командной строки:

**MIDL-Оикф @midl1.rsp -envwin32 @midl2.rsp ИТФ. idl**

Компилятор MIDL не поддерживает вложенные файлы ответов.
