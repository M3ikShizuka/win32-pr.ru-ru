# <a name="syntax_check-switch"></a>\_параметр проверки/синтакс

Параметр **\_ проверки/синтакс** указывает, что компилятор проверяет только синтаксис и не создает выходные файлы.

``` syntax
midl /syntax_check
midl /Zs
```

## <a name="switch-options"></a>Параметры коммутатора

Этот параметр не имеет параметров.

## <a name="remarks"></a>Комментарии

Этот параметр переопределяет все другие параметры, которые задают сведения о выходных файлах.

Также можно указать режим проверки синтаксиса с помощью параметра компилятора MIDL [**/ZS**](-zs.md).

## <a name="examples"></a>Примеры

**MIDL/ZS filename. idl**

**MIDL/синтакс \_ проверить filename. idl**
