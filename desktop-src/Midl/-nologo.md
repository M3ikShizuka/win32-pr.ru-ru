# <a name="nologo-switch"></a>Параметр/nologo

Параметр **/nologo** отключает отображение баннера авторского права.

``` syntax
midl /nologo
```

## <a name="switch-options"></a>Параметры коммутатора

Этот параметр не имеет параметров.
