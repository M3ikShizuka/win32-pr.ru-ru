# <a name="no_cpp-nocpp"></a>/но \_ cpp,/нокпп

Параметр **/но \_ cpp** указывает, что компилятор MIDL не вызывает препроцессору C для ПРЕДВАРИТЕЛЬной обработки IDL-файла.

Параметр **/но \_ cpp** имеет приоритет над параметрами [**/КПП \_ cmd**](-cpp-cmd.md) и [**/КПП \_ OPT**](-cpp-opt.md) .

## <a name="switch-options"></a>Параметры коммутатора

``` syntax
midl /no_cpp
midl /nocpp
```

## <a name="example"></a>Пример

**MIDL/но \_ cpp filename. idl**
