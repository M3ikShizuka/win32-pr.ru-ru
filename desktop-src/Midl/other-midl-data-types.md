# <a name="other-midl-data-types"></a>Другие типы данных MIDL

MIDL распознает несколько типов данных COM и автоматизации, таких как **BSTR**, **Variant** и **SAFEARRAY**, и дескрипторы com, такие как **хметафиле \_ PICT**, **хенхметафиле**, **хметафиле**, **хбитмап**, **хпалетте** и **хглобал**. Дополнительные сведения, включая имена файлов импорта, необходимых для использования этих типов, см. в разделе [маршалирование типов данных OLE](marshaling-ole-data-types.md).
