# <a name="metadata_dir-switch-midlrt"></a>параметр/metadata_dir (MIDLRT)

Параметр **\_ dir/Метадата** указывает один или несколько каталогов, содержащих файлы метаданных платформы.

``` syntax
midlrt /metadata_dir metadata_directory
```

## <a name="switch-options"></a>Параметры коммутатора

<dl> <dt>

*\_Каталог метаданных* 
</dt> <dd>

Указывает один или несколько каталогов, содержащих файлы метаданных платформы.

</dd> </dl>

## <a name="remarks"></a>Комментарии

Используйте этот параметр, чтобы указать расположение основного файла метаданных для Windows, который называется Windows. winmd.

## <a name="examples"></a>Примеры

**midlrt/Метадата \_ dir dir1 Dir2**
