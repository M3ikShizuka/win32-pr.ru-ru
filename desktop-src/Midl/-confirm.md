# <a name="confirm-switch"></a>/конфирм, параметр

Параметр **/конфирм** указывает компилятору отображать все параметры компилятора MIDL без обработки входных IDL и необязательных файлов ACF.

``` syntax
midl /confirm
```

## <a name="switch-options"></a>Параметры коммутатора

Этот параметр не имеет параметров.

## <a name="examples"></a>Примеры

**MIDL/конфирм**

**MIDL/конфирм @response.rsp filename. idl**
