# <a name="nonencapsulated-unions"></a>Неинкапсулированные объединения

Неинкапсулированное объединение — это объединение, которое не содержится в структуре с его discriminant. Неинкапсулированное объединение обозначает наличие **\[ \_ типа \] переключателя** атрибута Type, а параметр атрибута поля **\[ \_ имеет значение \]**.

Описание формы неинкапсулированного объединения см. в разделе [**Union**](union.md).

Для обеспечения взаимодействия между платформами все объединения должны быть одинаковыми для разных платформ.

Дополнительные сведения см. в разделе [инкапсулированные объединения](encapsulated-unions.md), [**объединение**](union.md), **\[** [**коммутатор \_**](switch-is.md) **\]** и **\[** [**\_ Тип переключателя**](switch-type.md) **\]** .
