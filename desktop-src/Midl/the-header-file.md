# <a name="the-header-file"></a>Файл заголовка

Файл заголовка содержит определения всех типов данных и операций, объявленных в IDL-файле, а также все типы данных и операции, объявленные в файлах, включенных в \# директиву include. Типы данных из файлов, импортированных с помощью директивы Import, не реплицируются в файл заголовка. Вместо этого созданный файл заголовка содержит строку include для H файла, созданного из импортированного файла. Файл заголовка должен включаться во все модули приложений, вызывающие определенные операции, реализовывать определенные операции или манипулировать определенными типами.

Компилятор MIDL переключает [**/Header**](-header.md) и [**/out**](-out.md) на файл заголовка.
