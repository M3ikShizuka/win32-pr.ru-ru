# <a name="about-this-guide"></a>Сведения об этом пошаговом руководству

Этот раздел представляет собой справочник по язык MIDL (MIDL) и руководству пользователя для компилятора MIDL, который создает файлы языка C из IDL-файлов.

В разделе [Использование КОМПИЛЯТОРА MIDL](using-the-midl-compiler-2.md) перечислены требования для языка C-Compiler и перед процессором, с которым должен взаимодействовать MIDL. В этом разделе также описаны файлы, создаваемые компилятором MIDL для заглушек RPC, заглушек DCOM и библиотеки типов OLE Automation.

В разделе [определения интерфейсов и библиотеки типов](interface-definitions-and-type-libraries.md) содержатся сведения о создании файлов определения интерфейса (IDL) и конфигурации приложения (ACF) для MIDL, включая справочные страницы для атрибутов IDL и ACF. Также есть сведения об импорте файлов и библиотек типов и импорте системных файлов заголовков.

В разделе [справочника по MIDL Command-Line](midl-command-line-reference.md) содержатся справочные сведения для каждого переключателя командной строки и параметра Switch, распознаваемых компилятором MIDL.

Раздел [Справочник по языку MIDL](midl-language-reference.md) содержит запись ссылки для каждого ключевого слова в язык MIDL.

В разделе [ошибки и предупреждения компилятора MIDL](midl-compiler-errors-and-warnings.md) перечислены сообщения об ошибках и предупреждения, которые могут создаваться компилятором MIDL.
