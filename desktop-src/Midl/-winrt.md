# <a name="winrt-switch"></a>/WinRT, параметр

Параметр [**/WinRT**](-winmd.md) включает семантику среда выполнения Windows.

``` syntax
midlrt /winrt 
```

## <a name="switch-options"></a>Параметры коммутатора

Этот параметр не имеет параметров.
