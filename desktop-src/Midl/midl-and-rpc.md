# <a name="midl-and-rpc"></a>MIDL и RPC

При разработке приложения RPC для создания IDL-файла, определяющего интерфейс между клиентом и сервером, используется текстовый редактор. Компилятор MIDL используется для создания исходных файлов, которые требуются клиентским и серверным программам. В следующих разделах представлены сведения об использовании компилятора MIDL для приложений RPC.

-   [Назначение заглушек для конкретных 32-разрядных или 64-разрядных платформ](targeting-stubs-for-specific-32-bit-or-64-bit-platforms.md)

Дополнительные сведения см. в разделе [Создание приложений RPC](/windows/desktop/Rpc/building-rpc-applications).
