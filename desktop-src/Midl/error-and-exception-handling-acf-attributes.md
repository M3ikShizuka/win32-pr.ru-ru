# <a name="error-and-exception-handling-acf-attributes"></a>Атрибуты ACF обработки ошибок и исключений

Используйте следующие атрибуты для обработки ошибок.



| attribute                                                                | Использование                                                                                                                                                                                                                                                                                                  |
|--------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_**](comm-status.md)[**\_ состояние ошибки** состояния связи](fault-status.md) | Позволить клиентскому приложению корректно обрабатывать исключения, вызывая возвращение клиенту ошибок связи и сервера в качестве значений параметров. Затем клиентское приложение может вызвать функцию времени выполнения RPC [**дцеерроринктекст**](/windows/desktop/api/rpcdce/nf-rpcdce-dceerrorinqtext) для ретрансляции сообщения об ошибке пользователю. |
