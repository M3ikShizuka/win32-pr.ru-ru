# <a name="v-switch"></a>Параметр/v

Параметр **/v** указывает, что ссылки на типы метаданных проверены.

``` syntax
mdmerge /v 
```

## <a name="switch-options"></a>Параметры коммутатора

Этот параметр не имеет параметров.
