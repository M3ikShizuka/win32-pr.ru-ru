# <a name="cc-compiler-considerations"></a>C/C++-рекомендации по компиляторам

Компилятор MIDL должен взаимодействовать с препроцессором C компилятора и C. MIDL не принимает синтаксис C++ для входных данных. Созданный h-файл содержит определения в стиле C и C++ для интерфейсов. В следующих разделах описываются требования для компилятора C.

-   [Вопросы упаковки C-компилятора](c-compiler-packing-issues.md)
-   [Определения для прокси-сервера и заглушек C-компилятора](c-compiler-definitions-for-proxy-stubs.md)
