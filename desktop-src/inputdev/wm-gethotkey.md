# <a name="wm_gethotkey-message"></a>Сообщение WM с \_ сочетанием клавиш

Отправлено, чтобы определить горячую клавишу, связанную с окном.


```C++
#define WM_GETHOTKEY                    0x0033
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение — это код виртуального ключа и модификаторы для сочетания клавиш, или **значение NULL** , если с окном не связана ни одна горячая клавиша. Код виртуального ключа находится в младшем байте возвращаемого значения, а модификаторы — в высоком байте. Модификаторы могут быть сочетанием следующих флагов из Коммктрл. h.



| Возвращаемый код и значение                                                                                                                                         | Описание             |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------|
| <dl> <dt>**Хоткэйф \_ ALT**</dt> <dt>0x04</dt> </dl>     | ALT - клавиша<br/>      |
| <dl> <dt>**Хоткэйф \_ Управление**</dt> <dt>0x02</dt> </dl> | Клавиша CTRL<br/>     |
| <dl> <dt>**Хоткэйф \_ EXT**</dt> <dt>0x08</dt> </dl>     | Расширенный ключ<br/> |
| <dl> <dt>**Хоткэйф \_ SHIFT**</dt> <dt>0x01</dt> </dl>   | Клавиша SHIFT<br/>    |



 

## <a name="remarks"></a>Комментарии

Эти горячие клавиши не связаны с горячими ключами, заданными функцией [**RegisterHotKey**](/windows/win32/api/winuser/nf-winuser-registerhotkey) .
