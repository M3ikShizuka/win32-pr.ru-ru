# <a name="wm_sethotkey-message"></a>\_Сообщение СЕСОТКЭЙ WM

Отправляется в окно для связывания горячих клавиш с окном. Когда пользователь нажимает клавишу горячего ключа, система активирует окно.


```C++
#define WM_SETHOTKEY                    0x0032
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Слово нижнего порядка указывает код виртуального ключа, связываемый с окном.

Слово с высоким порядковым значением может быть одним или несколькими из следующих значений в Коммктрл. h.

При установке параметра *wParam* в **значение NULL** удаляется горячая клавиша, связанная с окном.



| Значение                                                                                                                                                                                                                         | Значение                 |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------|
| <span id="HOTKEYF_ALT"></span><span id="hotkeyf_alt"></span><dl> <dt>**Хоткэйф \_ ALT**</dt> <dt>0x04</dt> </dl>             | ALT - клавиша<br/>      |
| <span id="HOTKEYF_CONTROL"></span><span id="hotkeyf_control"></span><dl> <dt>**Хоткэйф \_ Управление**</dt> <dt>0x02</dt> </dl> | Клавиша CTRL<br/>     |
| <span id="HOTKEYF_EXT"></span><span id="hotkeyf_ext"></span><dl> <dt>**Хоткэйф \_ EXT**</dt> <dt>0x08</dt> </dl>             | Расширенный ключ<br/> |
| <span id="HOTKEYF_SHIFT"></span><span id="hotkeyf_shift"></span><dl> <dt>**Хоткэйф \_ SHIFT**</dt> <dt>0x01</dt> </dl>       | Клавиша SHIFT<br/>    |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение является одним из следующих.



| Возвращаемое значение                                                                  | Описание                                                                             |
|-------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| <dl> <dt>-1</dt> </dl> | Функция не выполнена; Недопустимый горячая клавиша.<br/>                        |
| <dl> <dt>0</dt> </dl>  | Функция не выполнена; Недопустимое окно.<br/>                         |
| <dl> <dt>1</dt> </dl>  | Функция выполнена успешно, и никакие другие окна не имеют такой же горячей клавиши.<br/>        |
| <dl> <dt>2</dt> </dl>  | Функция выполнена успешно, но другое окно уже имеет ту же горячую клавишу.<br/> |



 

## <a name="remarks"></a>Комментарии

Горячая клавиша не может быть связана с дочерним окном.

**VK \_** Клавиши Escape **, \_ VK** и **VK являются \_** недопустимыми сочетаниями клавиш.

Когда пользователь нажимает сочетание клавиш, система создает сообщение [**WM \_ сискомманд**](/windows/desktop/menurc/wm-syscommand) с параметром *wParam* , равным **\_ сочетанию клавиш SC** и *lParam* , равным маркеру окна. Если это сообщение передается в [**дефвиндовпрок**](/windows/desktop/api/winuser/nf-winuser-defwindowproca), система переместит Последнее активное всплывающее окно окна (если оно существует) или само окно (если всплывающее окно отсутствует) на передний план.

Окно может иметь только одну горячую клавишу. Если с окном уже связано сочетание клавиш, новый горячая клавиша заменяется на старый. Если в нескольких окнах есть одна и та же горячая клавиша, окно, активируемое горячим ключом, является случайным.

Эти Горячие ключи не связаны с горячими ключами, установленными [**RegisterHotKey**](/windows/win32/api/winuser/nf-winuser-registerhotkey).
