# <a name="winbio_db-constants"></a>\_Константы винбио DB

При вызове [**винбиупенсессион**](/windows/desktop/api/Winbio/nf-winbio-winbioopensession) можно использовать следующие константы для указания базы данных, которая будет использоваться в системном пуле.



| Константа                                                                                                                                                                         | Описание                                                                                                                                                                                          |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WINBIO_DB_DEFAULT"></span><span id="winbio_db_default"></span><dl> <dt>**ВИНБИО \_ DB \_ по умолчанию**</dt> </dl>       | Каждая биометрическая единица в пуле датчиков использует базу данных по умолчанию, указанную в конфигурации биометрического модуля по умолчанию.<br/>                                                                   |
| <span id="WINBIO_DB_BOOTSTRAP"></span><span id="winbio_db_bootstrap"></span><dl> <dt>**\_начальная загрузка базы данных винбио \_**</dt> </dl> | Может использоваться в сценариях перед запуском Windows. Как правило, база данных является частью микросхемы датчика или входит в состав BIOS и может использоваться только для регистрации и удаления шаблона.<br/> |
| <span id="WINBIO_DB_ONCHIP"></span><span id="winbio_db_onchip"></span><dl> <dt>**Микросхема ВИНБИО \_ DB \_**</dt> </dl>          | База данных находится в микросхеме датчика.<br/>                                                                                                                                                  |
