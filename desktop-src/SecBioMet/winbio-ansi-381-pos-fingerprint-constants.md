## <a name="remarks"></a>Remarks

Неуказанные ВИНБИОные \_ \_ POS \_ - \_ \* константы не входят в спецификацию ANSI 381. Эти константы можно использовать для регистрации до десяти пальцев (на ИД безопасности учетной записи), не имеющих связанных сведений о положении.

> [!IMPORTANT]
>
> Не пытайтесь проверить значения констант отпечатков пальцев. служба биометрии Windows проверит указанные значения констант отпечатков пальцев перед их передачей в реализацию.
