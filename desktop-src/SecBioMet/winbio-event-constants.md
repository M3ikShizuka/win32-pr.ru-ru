# <a name="winbio_event-constants"></a>\_Константы событий винбио

Следующие константы можно использовать в функции [**винбиорегистеревентмонитор**](/windows/desktop/api/Winbio/nf-winbio-winbioregistereventmonitor) для указания типов уведомлений о событиях поставщика услуг для отслеживания.



| Константа                                                                                                                                                                                                                        | Описание                                                                                                                                                                                                                                                                                     |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WINBIO_EVENT_FP_UNCLAIMED"></span><span id="winbio_event_fp_unclaimed"></span><dl> <dt>**\_НЕутвержденное событие винбио \_ FP \_**</dt> </dl>                             | Датчик обнаружил пальца, который не запрашивался приложением или окном, которое в данный момент находится в фокусе. Биометрическая платформа Windows вызывает функцию обратного вызова для указания на то, что произошел палец, но не пытается определить отпечаток.<br/> |
| <span id="WINBIO_EVENT_FP_UNCLAIMED_IDENTIFY"></span><span id="winbio_event_fp_unclaimed_identify"></span><dl> <dt>**неутвержденная ВИНБИО для \_ события \_ FP \_ \_**</dt> </dl> | Датчик обнаружил пальца, который не запрашивался приложением или окном, которое в данный момент находится в фокусе. Биометрическая платформа Windows пытается опознать отпечаток и передает результат этого процесса функции обратного вызова.<br/>                        |
