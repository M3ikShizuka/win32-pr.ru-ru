# <a name="winbio_orientation-constants"></a>\_Константы ориентации винбио

Следующие константы указывают возможные ориентации камеры, которые компонент датчика указывает как обязательный.



| Константа/значение                                                                                                                                                                                                                                                           | Описание                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------|
| <span id="WINBIO_ORIENTATION_UNSPECIFIED"></span><span id="winbio_orientation_unspecified"></span><dl> <dt>**Винбио \_ ОРИЕНТАЦИЯ не \_ указана**</dt> <dt>0</dt> </dl> | Не указана обязательная ориентация для камеры.<br/> |
| <span id="WINBIO_ORIENTATION_LANDSCAPE"></span><span id="winbio_orientation_landscape"></span><dl> <dt>**Винбио \_ \_Альбомная ориентация**</dt> <dt>1</dt> </dl>       | Для камеры требуется альбомная ориентация.<br/>    |
| <span id="WINBIO_ORIENTATION_PORTRAIT"></span><span id="winbio_orientation_portrait"></span><dl> <dt>**Винбио \_ ОРИЕНТАЦИЯ \_ Книжная**</dt> <dt>2</dt> </dl>          | Для камеры требуется книжная ориентация.<br/>     |
| <span id="WINBIO_ORIENTATION_ANY"></span><span id="winbio_orientation_any"></span><dl> <dt>**Винбио \_ ОРИЕНТАЦИЯ \_**</dt> — <dt>3</dt> </dl>                         | Для камеры разрешена любая ориентация.<br/>             |
