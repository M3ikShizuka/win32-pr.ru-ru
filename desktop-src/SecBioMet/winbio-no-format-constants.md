# <a name="winbio_no_format-constants"></a>ВИНБИО \_ без \_ констант формата

Если члены [**\_ зарегистрированной структуры \_ формата винбио**](winbio-registered-format.md) не могут быть равны нулю, можно использовать следующие константы для проверки на наличие ошибок.



| Константа/значение                                                                                                                                                                                                                                                                   | Описание                                                                                          |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="WINBIO_NO_FORMAT_OWNER_AVAILABLE"></span><span id="winbio_no_format_owner_available"></span><dl> <dt>**Винбио \_ НЕТ \_ \_ \_ доступного владельца формата**</dt> <dt>0</dt> </dl> | Нет доступного значения владельца ИБИА (Международная биометрическая Ассоциация отрасли).<br/> |
| <span id="WINBIO_NO_FORMAT_TYPE_AVAILABLE"></span><span id="winbio_no_format_type_available"></span><dl> <dt>**Винбио \_ НЕТ \_ \_ \_ доступных типов форматов**</dt> <dt>0</dt> </dl>    | Тип формата, которому назначен владелец, недоступен.<br/>                                               |
