# <a name="private-sensor-pool"></a>Частный пул датчиков

Частный пул датчиков — это набор биометрических блоков, зарезервированных для монопольного использования клиентским приложением. Частные пулы поддерживают собственные методы проверки подлинности и позволяют клиентскому приложению получать доступ к биометрической единице с помощью определяемых поставщиком команд управления. Биометрические единицы в частном пуле датчиков:

-   Доступны только для клиентского приложения, создавшего пул.
-   Отправляйте уведомления о событиях, формируемые в результате выполнения биометрических операций непосредственно в приложение, не обращаясь к текущему фокусу окна.
-   Используйте идентификаторы GUID для поиска биометрических шаблонов.
-   Совместное использование одной выбранной базы данных шаблона приложения.

При использовании клиентского приложения необходимо использовать частные пулы датчиков:

-   Управляет коллекцией выделенных биометрических единиц, использующих зависящую от приложения базу данных шаблонов. Например, рассмотрим приложение для просмотра сотрудников, в котором сотрудники сообщают о поступлении на работу, прочитающи палец на сканере отпечатков пальцев. у сотрудников нет учетных записей Windows на компьютере, на котором выполняется приложение. Вместо этого их отпечатки определяются идентификаторами GUID, которыми управляет приложение посещаемости.
-   Собирает биометрические примеры, а не просто сопоставляет образцы с идентификаторами безопасности.
-   Переводит оборудование биометрического модуля в режим обслуживания для обновления встроенного по.
-   Отправляет определяемые поставщиком команды управления на оборудование или программное обеспечение биометрического модуля.
-   Пытается настроить биометрический модуль с подключением хранилища для работы в расширенном режиме, но модуль не может выполнить необходимые операции хэширования.
-   Запускается из удаленный рабочий стол сеанса клиента.

> [!Note]  
> Приложения могут создавать частные пулы датчиков только для биометрического отпечатка. Если приложение пытается создать какое-либо другое (например, лицо), запрос завершится ошибкой.
