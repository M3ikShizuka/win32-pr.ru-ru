# <a name="client-application-enumerations"></a>Перечисления клиентских приложений

API биометрическая платформа Windows поддерживает следующие типы перечисления.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                      | Описание                                                                                                                                       |
|--------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ \_ действие политики АНТИфишинга винбио \_**](winbio-anti-spoof-policy-action.md)<br/>  | Указывает типы действий, выполняемых для политики антиподмены пользователя. <br/>                                                        |
| [**\_асинхронный \_ метод уведомления винбио \_**](/windows/desktop/api/Winbio/ne-winbio-winbio_async_notification_method)<br/> | Определяет константы, указывающие, как уведомления о завершении асинхронных операций должны доставляться в клиентское приложение.<br/> |
| [**\_формат учетных данных винбио \_**](winbio-credential-format.md)<br/>                  | Определяет флаги, которые можно использовать для указания формата учетных данных конечного пользователя.<br/>                                                              |
| [**\_состояние учетных данных винбио \_**](winbio-credential-state.md)<br/>                    | Определяет значения, указывающие, связаны ли учетные данные с биометрических данных конечного пользователя.<br/>                          |
| [**\_тип учетных данных винбио \_**](winbio-credential-type.md)<br/>                      | Определяет флаги, которые можно использовать для фильтрации по типу учетных данных.<br/>                                                                       |
| [**\_источник политики \_ винбио**](winbio-policy-source.md)<br/>                          | Список возможных источников сведений о политике для обнаружения подмены для биометрических факторов.<br/>                                  |
