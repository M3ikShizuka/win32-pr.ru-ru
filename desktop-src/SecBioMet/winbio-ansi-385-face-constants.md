# <a name="winbio_ansi_385_face-constants"></a>\_ \_ Константы винбио ANSI 385 \_

Следующие константы являются **значениями \_ \_ ПОДтипа БИОметрической метрики винбио** , которые можно использовать для указания двух типов изображений переднего плана, как определено в ANSI ИнЦитс 385-2004: "формат распознавания лиц для обмена данными": полное разрешение и низкое разрешение. На практике биометрическая платформа будет использовать только образы с полным разрешением для распознавания лиц.



| Константа/значение                                                                                                                                                                                                                                                                          | Описание                                                |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------|
| <span id="WINBIO_ANSI_385_FACE_TYPE_UNKNOWN"></span><span id="winbio_ansi_385_face_type_unknown"></span><dl> <dt>**Винбио \_ \_Неизвестный \_ \_ тип шрифта \_ ANSI 385**</dt> <dt></dt> </dl>    | Тип изображения лицевой стороны неизвестен.<br/>         |
| <span id="WINBIO_ANSI_385_FACE_FRONTAL_FULL"></span><span id="winbio_ansi_385_face_frontal_full"></span><dl> <dt>**Винбио \_ \_Интерфейс ANSI 385 \_ лицевой стороной, \_ \_ полный**</dt> <dt>1</dt> </dl>    | Тип образа лицевой стороны — полное разрешение.<br/> |
| <span id="WINBIO_ANSI_385_FACE_FRONTAL_TOKEN"></span><span id="winbio_ansi_385_face_frontal_token"></span><dl> <dt>**Винбио \_ \_ \_ Лицевой \_ \_ маркер 2 ANSI 385**</dt> <dt></dt> </dl> | Тип изображения лицевой стороны имеет низкую разрешающую способность.<br/>  |
