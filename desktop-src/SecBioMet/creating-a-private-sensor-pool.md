# <a name="creating-a-private-sensor-pool"></a>Создание частного пула датчиков

Частный пул датчиков — это набор биометрических блоков, зарезервированных для монопольного использования клиентским приложением. Частные пулы поддерживают собственные методы проверки подлинности и позволяют клиентскому приложению получать доступ к биометрической единице с помощью определяемых поставщиком команд управления.

В следующих разделах содержатся примеры кода, демонстрирующие создание частного пула датчиков.
