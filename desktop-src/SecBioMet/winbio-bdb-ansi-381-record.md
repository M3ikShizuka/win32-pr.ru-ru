# <a name="winbio_bdb_ansi_381_record-structure"></a>ВИНБИО \_ БДБ \_ ANSI \_ 381, \_ Структура записи

Структура **\_ записи винбио \_ БДБ \_ ANSI \_ 381** содержит сведения о одном отпечатке пальцев или Palm от конечного пользователя. Коллекция этих структур включена в каждую структуру [**\_ \_ \_ \_ заголовков винбио БДБ ANSI 381**](winbio-bdb-ansi-381-header.md) .

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _WINBIO_BDB_ANSI_381_RECORD {
  ULONG                    BlockLength;
  USHORT                   HorizontalLineLength;
  USHORT                   VerticalLineLength;
  WINBIO_BIOMETRIC_SUBTYPE Position;
  UCHAR                    CountOfViews;
  UCHAR                    ViewNumber;
  UCHAR                    ImageQuality;
  UCHAR                    ImpressionType;
  UCHAR                    Reserved;
} WINBIO_BDB_ANSI_381_RECORD;
```
