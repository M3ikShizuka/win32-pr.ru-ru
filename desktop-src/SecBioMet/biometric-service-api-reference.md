# <a name="windows-biometric-framework-api-reference"></a>Windows Справочник по API биометрической платформы

Windows Справочные страницы по API биометрической платформы содержат подробное описание функций, структур и возможностей, которые можно использовать для создания клиентских приложений и подключаемых модулей. Справочные страницы делятся на следующие группы:

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                       | Описание                                                                                                                                         |
|-----------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [Справочник по клиентскому приложению](client-application-reference.md)<br/> | функции, структуры и другие типы, которые можно использовать для создания клиентских приложений, взаимодействующих с биометрическая платформа Windows.<br/> |
| [Справочник по подключаемым модулям](plug-in-reference.md)<br/>                       | Функции адаптера, функции-оболочки и структуры для создания настраиваемых адаптеров подключаемых модулей трех типов: подсистемы, датчика и хранилища.<br/>   |
