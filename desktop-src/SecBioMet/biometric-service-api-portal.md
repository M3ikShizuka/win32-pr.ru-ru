# <a name="windows-biometric-framework"></a>Биометрическая платформа Windows

## <a name="purpose"></a>Назначение

Вы можете использовать биометрическая платформа Windows API для создания клиентских приложений, которые безопасно захватывают, сохраняют и сравнивают биометрические данные конечного пользователя.

## <a name="developer-audience"></a>Аудитория разработчиков

Разработчики, использующие этот API, должны быть знакомы с языками программирования C и C++ и средой программирования на основе Windows.

## <a name="run-time-requirements"></a>Требования к среде выполнения

Биометрическая платформа Windows API поддерживается начиная с Windows Server 2008 R2 и Windows 7. Сведения о требованиях времени выполнения для определенного программного элемента см. в разделе "требования" на странице справочника по этому элементу.
