# <a name="plug-in-reference"></a>Справочник по подключаемым модулям

Биометрические устройства изготовлены в широком спектре типов и конфигураций. Биометрическая платформа Windows включает расширяемую архитектуру, позволяющую работать с этими множеством, создавая пользовательские подключаемые модули. В центре этой архитектуры находится программный объект, называемый биометрической единицей. Биометрическую единицу можно представить как абстракцию, которая представляет собой биометрический устройство для платформы. Программные компоненты подключаемых модулей, называемые адаптерами, соединяют биометрическую единицу с соответствующим оборудованием. Существует три типа адаптеров, которые можно создать. Адаптер подсистемы создает Биометрические шаблоны из захваченных примеров, сопоставляет примеры с существующими шаблонами и шаблоны индексов. Адаптер датчика создает оболочку для биометрического устройства и предоставляет стандартный интерфейс для настройки датчика, отслеживания образцов и управления передачей биометрических данных механизму обработки. Адаптер хранилища управляет шаблонами баз данных.
