# <a name="winbio_bir-structure"></a>\_Структура Бир винбио

Структура **винбио \_ Бир** представляет запись биометрической информации (Бир). Информационная запись содержит блоки заголовка, данных и подписи.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _WINBIO_BIR {
  WINBIO_BIR_DATA HeaderBlock;
  WINBIO_BIR_DATA StandardDataBlock;
  WINBIO_BIR_DATA VendorDataBlock;
  WINBIO_BIR_DATA SignatureBlock;
} WINBIO_BIR;
```



## <a name="members"></a>Члены

<dl> <dt>

**хеадерблокк**
</dt> <dd>

Структура [**\_ \_ данных винбио Бир**](winbio-bir-data.md) , которая содержит размер, в байтах и смещение заголовка Бир. Заголовок содержит сведения, описывающие содержимое информационной записи.

</dd> <dt>

**стандарддатаблокк**
</dt> <dd>

Структура [**\_ \_ данных винбио Бир**](winbio-bir-data.md) , которая содержит размер, в байтах и смещение обработанных или необработанных биометрических данных, СОЗДАННЫХ биометрическая платформа Windowsом (ВБФ).

</dd> <dt>

**вендордатаблокк**
</dt> <dd>

Структура [**\_ \_ данных винбио Бир**](winbio-bir-data.md) , которая содержит размер, в байтах, а также смещение обработанных или необработанных биометрических данных, предоставленных датчиками и программным обеспечением поставщика.

</dd> <dt>

**сигнатуреблокк**
</dt> <dd>

Необязательная структура [**\_ \_ данных винбио Бир**](winbio-bir-data.md) , которая содержит размер, в байтах и смещение кода проверки подлинности сообщений цифровой подписи (Mac), который может использоваться для проверки целостности Бир. При наличии подписи или MAC должна охватываться заголовок и блоки данных.

</dd> </dl>

## <a name="remarks"></a>Комментарии

Использование смещений вместо указателей позволяет легко сериализовать Бир и для менее сложного перевода между 32 и 64-разрядными средами или между пользователем и режимом ядра.

Бир совместим с общей биометрической структурой формата Exchange (КБЕФФ), определенной директивой NIST 6529-A.

Если эта структура содержит значение *стандарддатаблокк* , то параметру *Type* заголовка, указанного в параметре *хеадерблокк* , должен быть задан **\_ \_ \_ \_ тип винбио ANSI 381**. Это единственный стандартный формат данных, поддерживаемый текущей версией биометрическая платформа Windows.
