# <a name="isownercleardisabled-method-of-the-win32_tpm-class"></a>Метод Исовнерклеардисаблед \_ класса TPM Win32

Метод **исовнерклеардисаблед** класса [**\_ TPM Win32**](win32-tpm.md) указывает, ограничен ли владелец устройства очисткой устройства.

## <a name="syntax"></a>Синтаксис


```mof
uint32 IsOwnerClearDisabled(
  [out] boolean IsOwnerClearDisabled
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Исовнерклеардисаблед* \[ заполняет\]
</dt> <dd>

Тип: **Boolean**

Если **значение — true**, владелец устройства не может очистить устройство. Только физически представленный пользователь может очистить устройство. Если задано **значение false**, владелец устройства может очистить устройство.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

Могут возвращаться все ошибки доверенного платформенного модуля, а также ошибки, относящиеся к базовым службам TPM.

Ниже приведены общие коды возврата.



| Возвращаемый код и значение                                                                                                                                 | Описание                           |
|---------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl> | Метод успешно выполнен.<br/> |
