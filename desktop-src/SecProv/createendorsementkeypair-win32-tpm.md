# <a name="createendorsementkeypair-method-of-the-win32_tpm-class"></a>Метод Креатиндорсементкэйпаир \_ класса TPM Win32

Метод **креатиндорсементкэйпаир** класса [**\_ Tpm Win32**](win32-tpm.md) создает пару 2048-битного ключа подтверждения в доверенном платформенном модуле. Для успешного выполнения метода [**такеовнершип**](takeownership-win32-tpm.md) должна быть доступна пара ключей подтверждения. С помощью метода [**исендорсементкэйпаирпресент**](isendorsementkeypairpresent-win32-tpm.md) можно определить, существует ли уже ключ подтверждения в доверенном платформенном модуле.

## <a name="syntax"></a>Синтаксис


```mof
uint32 CreateEndorsementKeyPair();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

Могут возвращаться все ошибки доверенного платформенного модуля, а также ошибки, относящиеся к базовым службам TPM.

В следующей таблице перечислены некоторые распространенные коды возврата.



| Возвращаемый код и значение                                                                                                                                                                  | Описание                                                    |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                  | Метод успешно выполнен.<br/>                          |
| <dl> <dt> **TPM \_ E \_ отключил \_ cmd-командлет**</dt> <dt>2150105096 (0x80280008)</dt> </dl> | Пара ключей подтверждения уже существует в этом доверенном платформенном модуле.<br/> |
