# <a name="preparevolume-method-of-the-win32_encryptablevolume-class"></a>Метод Препареволуме \_ класса Win32 енкриптаблеволуме

Метод **препареволуме** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) создает том BitLocker с указанным типом файловой системы тома обнаружения. Этот метод должен вызываться до того, как том можно будет защитить с помощью любого из методов **протекткэйвис \*** .

## <a name="syntax"></a>Синтаксис

```mof
uint32 PrepareVolume(
  [in] string DiscoveryVolumeType,
  [in] uint32 ForceEncryptionType
);
```

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.

| Возвращаемый код и значение      | Описание                     |
|------------------------|---------------------------------|
| **S_OK** <br/> 0 (0x0) | Метод успешно выполнен.<br/> |

## <a name="remarks"></a>Remarks

Если вы не вызываете этот метод при включении тома BitLocker, он аналогичен вызову этого метода со значением по умолчанию в параметре *дисковериволуметипе* .
