# <a name="isowned-method-of-the-win32_tpm-class"></a>Метод владеет \_ классом TPM Win32

Метод, **принадлежащий** классу [**\_ TPM Win32**](win32-tpm.md) , указывает, имеет ли устройство владелец. Это значение изменяется методом [**такеовнершип**](takeownership-win32-tpm.md) .

## <a name="syntax"></a>Синтаксис


```mof
uint32 IsOwned(
  [out] boolean IsOwned
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Владелец* \[ заполняет\]
</dt> <dd>

Тип: **Boolean**

Если **значение — true**, устройство имеет владельца.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

Могут возвращаться все ошибки доверенного платформенного модуля, а также ошибки, относящиеся к базовым службам TPM.

Ниже приведены общие коды возврата.



| Возвращаемый код и значение                                                                                                                                 | Описание                           |
|---------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl> | Метод успешно выполнен.<br/> |
