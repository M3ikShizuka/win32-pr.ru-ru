# <a name="getkeyprotectortype-method-of-the-win32_encryptablevolume-class"></a>Метод Жеткэйпротектортипе \_ класса Win32 енкриптаблеволуме

Метод **жеткэйпротектортипе** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) указывает тип предохранителя ключа.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetKeyProtectorType(
  [in]  string VolumeKeyProtectorID,
  [out] uint32 KeyProtectorType
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Волумекэйпротекторид* \[ окне\]
</dt> <dd>

Тип: **строка**

Уникальный строковый идентификатор, используемый для управления предохранителем ключа зашифрованного тома.

</dd> <dt>

*Кэйпротектортипе* \[ заполняет\]
</dt> <dd>

Тип: **UInt32**

Целое число без знака, указывающее тип предохранителя ключа.



| Значение                                                                         | Значение                                              |
|-------------------------------------------------------------------------------|------------------------------------------------------|
| <dl> <dt>0</dt> </dl>  | Неизвестный или другой тип предохранителя<br/>           |
| <dl> <dt>1</dt> </dl>  | Доверенный платформенный модуль (TPM)<br/>             |
| <dl> <dt>2</dt> </dl>  | Внешний ключ<br/>                              |
| <dl> <dt>3</dt> </dl>  | Числовой пароль<br/>                        |
| <dl> <dt>4</dt> </dl>  | TPM и ПИН-код<br/>                               |
| <dl> <dt>5</dt> </dl>  | TPM и ключ запуска<br/>                       |
| <dl> <dt>6</dt> </dl>  | TPM, ПИН-код и ключ запуска<br/>               |
| <dl> <dt>7</dt> </dl>  | Открытый ключ<br/>                                |
| <dl> <dt>8</dt> </dl>  | Парольная фраза<br/>                                |
| <dl> <dt>9</dt> </dl>  | Сертификат TPM<br/>                           |
| <dl> <dt>10</dt> </dl> | Средство защиты для следующего поколения CryptoAPI (CNG)<br/> |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.



| Возвращаемый код и значение                                                                                                                                                                  | Описание                                                                                   |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                  | Метод успешно выполнен.<br/>                                                         |
| <dl> <dt>**Д \_ INVALIDARG**</dt> <dt>2147942487 (0x80070057)</dt> </dl>          | Параметр *волумекэйпротекторид* не ссылается на допустимый *кэйпротектортипе*.<br/> |
| <dl> <dt>**ФВЕ \_ E \_ не \_ активировано**</dt> <dt>2150694920 (0x80310008)</dt> </dl> | Средство BitLocker не включено в томе. Добавьте предохранитель ключа, чтобы включить BitLocker. <br/>  |
