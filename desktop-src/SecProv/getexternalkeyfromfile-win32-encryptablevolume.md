# <a name="getexternalkeyfromfile-method-of-the-win32_encryptablevolume-class"></a>Метод Жетекстерналкэйфромфиле \_ класса Win32 енкриптаблеволуме

Метод **жетекстерналкэйфромфиле** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) возвращает внешний ключ из файла, созданного с помощью [**савикстерналкэйтофиле**](saveexternalkeytofile-win32-encryptablevolume.md), учитывая расположение этого файла.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetExternalKeyFromFile(
  [in]  string PathWithFileName,
  [out] uint8  ExternalKey[]
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Пасвисфиленаме* \[ окне\]
</dt> <dd>

Тип: **строка**

Строка, указывающая расположение файла, содержащего внешний ключ.

</dd> <dt>

*Екстерналкэй* \[ заполняет\]
</dt> <dd>

Тип: **Uint8 \[ \]**

Массив байтов, который является 256-битным внешним ключом, содержащимся в файле, который можно использовать для разблокировки тома.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.



| Возвращаемый код и значение                                                                                                                                                                   | Описание                                            |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                   | Метод успешно выполнен.<br/>                  |
| <dl> <dt>**Д \_ INVALIDARG**</dt> <dt>2147942487 (0x80070057)</dt> </dl>           | Файл не содержит внешний ключ.<br/>  |
| <dl> <dt>**Ошибка при \_ ФАЙЛ \_ не \_ найден**</dt> <dt>2147942402 (0x80070002)</dt> </dl> | Не удается найти файл в указанном расположении.<br/> |



 

## <a name="remarks"></a>Remarks

файлы MOF-файл (MOF) содержат определения для классов инструментарий управления Windows (WMI) (WMI). MOF-файлы не устанавливаются в составе Windows SDK. Они устанавливаются на сервере при добавлении связанной роли с помощью диспетчер сервера. Дополнительные сведения о файлах MOF см. в разделе [MOF-файл (MOF)](../wmisdk/managed-object-format--mof-.md).
