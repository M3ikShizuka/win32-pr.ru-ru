# <a name="findvalidcertificates-method-of-the-win32_encryptablevolume-class"></a>Метод Финдвалидцертификатес \_ класса Win32 енкриптаблеволуме

Метод **финдвалидцертификатес** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) перечисляет все сертификаты в системе, соответствующие указанным критериям, и возвращает список отпечатков. Возвращаемый список содержит только сертификаты с допустимым [*идентификатором объекта*](../secgloss/o-gly.md) (OID). OID может быть значением по умолчанию или его можно указать в групповая политика.

## <a name="syntax"></a>Синтаксис


```mof
uint32 FindValidCertificates(
  [out] string CertificateThumbprint[]
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*CertificateThumbprint* \[ заполняет\]
</dt> <dd>

Тип: **строка \[ \]**

Массив строк, содержащий список допустимых сертификатов.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.



| Возвращаемый код и значение                                                                                                                                                                           | Описание                                          |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                           | Метод успешно выполнен.<br/>                |
| <dl> <dt>**ФВЕ \_ \_Недопустимый \_ \_ код BitLocker OID**</dt> <dt>2150695022 (0x8031006E)</dt> </dl> | Недопустимый доступный OID BitLocker.<br/> |
