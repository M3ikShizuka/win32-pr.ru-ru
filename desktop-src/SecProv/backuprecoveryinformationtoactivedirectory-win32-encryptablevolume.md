# <a name="backuprecoveryinformationtoactivedirectory-method-of-the-win32_encryptablevolume-class"></a>Метод Баккупрековеринформатионтоактиведиректори \_ класса Win32 енкриптаблеволуме

Метод **баккупрековеринформатионтоактиведиректори** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) создает резервные копии данных восстановления в Active Directory. Этот метод требует наличия на томе средства защиты числового пароля. Групповая политика также необходимо настроить для включения резервного копирования сведений о восстановлении в Active Directory.

## <a name="syntax"></a>Синтаксис


```mof
uint32 BackupRecoveryInformationToActiveDirectory(
  [in] string VolumeKeyProtectorID
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Волумекэйпротекторид* \[ окне\]
</dt> <dd>

Тип: **строка**

Уникальный строковый идентификатор, используемый для управления предохранителем ключа зашифрованного тома. Предохранитель ключа должен быть числовым предохранителем пароля.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.



| Возвращаемый код и значение                                                                                                                                                                            | Описание                                                                                                              |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                            | Метод успешно выполнен.<br/>                                                                                    |
| <dl> <dt>**С \_ FALSE**</dt> <dt>1 (0x1)</dt> </dl>                                         | Групповая политика не позволяет Active Directory хранения сведений о восстановлении.<br/>                         |
| <dl> <dt>**ФВЕ \_ E \_ не \_ активировано**</dt> <dt>2150694920 (0x80310008)</dt> </dl>           | Средство BitLocker не включено в томе. Добавьте предохранитель ключа, чтобы включить BitLocker. <br/>                             |
| <dl> <dt>**ФВЕ \_ \_Недопустимый \_ \_ тип предохранителя**</dt> <dt>2150694970 (0x8031003A)</dt> </dl> | Указанный предохранитель ключа не является ключом числового предохранителя ключа. Необходимо ввести средство защиты числового пароля. <br/> |
