# <a name="getkeyprotectorplatformvalidationprofile-method-of-the-win32_encryptablevolume-class"></a>Метод Жеткэйпротекторплатформвалидатионпрофиле \_ класса Win32 енкриптаблеволуме

Метод **жеткэйпротекторплатформвалидатионпрофиле** извлекает профиль проверки платформы для данного предохранителя ключа соответствующего типа.

Идентификатор предохранителя ключа должен ссылаться на предохранитель ключа типа "TPM", "TPM и ПИН-код", "TPM, ПИН-код и ключ запуска", "TPM и ключ запуска".

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetKeyProtectorPlatformValidationProfile(
  [in]  string VolumeKeyProtectorID,
  [out] uint8  PlatformValidationProfile[]
);
```
