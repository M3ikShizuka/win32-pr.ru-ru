# <a name="win32_tpmgetsrkadthumbprint-method"></a>\_Метод Win32 TPM:: жетсркадсумбпринт

Возвращает отпечаток корневого ключа хранилища для данного модуля открытой части корневого ключа хранилища TPM. Отпечаток используется для индексации корневых ключей хранилища на Active Directory сервере, если Active Directoryная резервная копия сведений о авторизации владельца доверенного платформенного модуля включена через параметр групповая политика.

> [!Note]  
> Начиная с Windows 10 версии 1607, Авторизация владельца доверенного платформенного модуля никогда не зарезервирована на Active Directory.

 

Этот метод доступен только локальным администраторам.

## <a name="syntax"></a>Синтаксис


```C++
uint32 GetSrkADThumbprint(
  [in]  uint8 SrkPublicKeyModulus,
  [out] uint8 SrkADThumbprint
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Сркпубликкэймодулус* \[ окне\]
</dt> <dd>

Модуль открытой части корневого ключа хранилища TPM. Его также можно получить с помощью метода [**жетсркпубликкэймодулус**](win32-tpm-getsrkpublickeymodulus.md) класса [ \_ TPM Win32](win32-tpm.md) .

</dd> <dt>

*Сркадсумбпринт* \[ заполняет\]
</dt> <dd>

Возвращает 20-байтовый массив, содержащий отпечаток корневого ключа хранилища для данного модуля.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Могут возвращаться все ошибки доверенного платформенного модуля, а также ошибки, относящиеся к [базовым службам TPM](../tbs/tbs-return-codes.md) .

Ниже приведены общие коды возврата.



| Возвращаемый код и значение                                                                                                                                 | Описание                           |
|---------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl> | Метод успешно выполнен.<br/> |
