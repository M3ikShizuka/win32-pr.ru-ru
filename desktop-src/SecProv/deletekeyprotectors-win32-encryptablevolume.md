# <a name="deletekeyprotectors-method-of-the-win32_encryptablevolume-class"></a>Метод Делетекэйпротекторс \_ класса Win32 енкриптаблеволуме

Метод **делетекэйпротекторс** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) удаляет все предохранители ключа для тома.

Если незашифрованный том содержит предохранители ключа, то при успешном выполнении **делетекэйпротекторс** том возвращается к стандартной файловой системе NTFS.

Если том еще не полностью расшифрован, используйте [**дисаблекэйпротекторс**](disablekeyprotectors-win32-encryptablevolume.md) перед запуском **делетекэйпротекторс** , чтобы убедиться, что зашифрованные части тома остаются доступными.

## <a name="syntax"></a>Синтаксис


```mof
uint32 DeleteKeyProtectors();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.



| Возвращаемый код и значение                                                                                                                                                                          | Описание                                                                                                                                                                                                                                                                                                               |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                          | Метод успешно выполнен.<br/>                                                                                                                                                                                                                                                                                     |
| <dl> <dt>**ФВЕ \_ E с \_ заблокированным \_ томом**</dt> <dt>2150694912 (0x80310000)</dt> </dl>         | Том заблокирован.<br/>                                                                                                                                                                                                                                                                                          |
| <dl> <dt>**ФВЕ \_ 2150694941 \_ \_ требуется ключ E**</dt> <dt>(0x8031001D)</dt> </dl>          | Последний предохранитель ключа для частично или полностью зашифрованного тома нельзя удалить, если включены предохранители ключа. Используйте [**дисаблекэйпротекторс**](disablekeyprotectors-win32-encryptablevolume.md) перед удалением последнего предохранителя ключа, чтобы убедиться, что зашифрованные части тома остаются доступными. <br/> |
| <dl> <dt>**ФВЕ \_ E \_ \_ с привязкой к тому \_ уже**</dt> <dt>2150694943 (0x8031001F)</dt> </dl> | Невозможно удалить предохранители ключа, так как один из них используется для автоматической разблокировки тома. <br/> Используйте [**дисаблеаутаунлокк**](disableautounlock-win32-encryptablevolume.md) , чтобы отключить автоматическое снятие блокировки перед выполнением этого метода.<br/>                                                       |
