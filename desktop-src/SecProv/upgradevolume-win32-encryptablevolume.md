# <a name="upgradevolume-method-of-the-win32_encryptablevolume-class"></a>Метод Упградеволуме \_ класса Win32 енкриптаблеволуме

метод **упградеволуме** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) обновляет том из формата Windows Vista в формат Windows 7. Это необратимая операция.

## <a name="syntax"></a>Синтаксис


```mof
uint32 UpgradeVolume();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.

Если том уже разблокирован и другие ошибки не возникают, этот метод возвращает ноль.



| Возвращаемый код и значение                                                                                                                                                                  | Описание                                                                                  |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                  | Метод успешно выполнен.<br/>                                                        |
| <dl> <dt>**Д \_ INVALIDARG**</dt> <dt>214794287 (0xCCD802F)</dt> </dl>            | Один или несколько аргументов являются недопустимыми.<br/>                                       |
| <dl> <dt>**ФВЕ \_ E с \_ заблокированным \_ томом**</dt> <dt>2150694912 (0x80310000)</dt> </dl> | Том заблокирован.<br/>                                                             |
| <dl> <dt>**ФВЕ \_ E \_ не \_ активировано**</dt> <dt>2150694920 (0x80310008)</dt> </dl> | Средство BitLocker не включено в томе. Добавьте предохранитель ключа, чтобы включить BitLocker. <br/> |
