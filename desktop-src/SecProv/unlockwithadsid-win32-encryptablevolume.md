# <a name="unlockwithadsid-method-of-the-win32_encryptablevolume-class"></a>Метод Унлокквисадсид \_ класса Win32 енкриптаблеволуме

Метод **унлокквисадсид** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) использует указанную строку идентификатора безопасности Active Directory, чтобы получить производный ключ и разблокировать зашифрованный том.

> [!Note]  
> Если диск поддерживает аппаратное шифрование, эта функция устанавливает для полосы состояние "разблокирован".

 

## <a name="syntax"></a>Синтаксис


```mof
uint32 UnlockWithAdSid(
  [in]  string SidString
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Сидстринг* \[ окне\]
</dt> <dd>

Строка, содержащая идентификатор безопасности Active Directory.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.



| Возвращаемый код и значение                                                                                                                                 | Описание                           |
|---------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl> | Метод успешно выполнен.<br/> |



 

## <a name="remarks"></a>Комментарии

Файлы MOF-файл (MOF) содержат определения для классов инструментарий управления Windows (WMI) (WMI). MOF-файлы не устанавливаются в составе Windows SDK. Они устанавливаются на сервере при добавлении связанной роли с помощью диспетчер сервера. Дополнительные сведения о файлах MOF см. в разделе [MOF-файл (MOF)](../wmisdk/managed-object-format--mof-.md).
