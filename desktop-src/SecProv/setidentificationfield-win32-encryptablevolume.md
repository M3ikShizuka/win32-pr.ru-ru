# <a name="setidentificationfield-method-of-the-win32_encryptablevolume-class"></a>Метод Сетидентификатионфиелд \_ класса Win32 енкриптаблеволуме

Метод **сетидентификатионфиелд** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) устанавливает указанную строку идентификатора в метаданных тома.

## <a name="syntax"></a>Синтаксис


```mof
uint32 SetIdentificationField(
  [in, optional] string IdentificationField
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Идентификатионфиелд* \[ в необязательное\]
</dt> <dd>

Тип: **строка**

Строка, указывающая поле идентификации, назначенное тому. Если необязательная строка отсутствует, используются значения набора реестра. Если строка имеется и не пуста, используется указанное значение. Параметр *идентификатионфиелд* не учитывает регистр.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.



| Возвращаемый код и значение                                                                                                                                                                  | Описание                                                                                                     |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                  | Метод успешно выполнен.<br/>                                                                           |
| <dl> <dt>**ФВЕ \_ E с \_ заблокированным \_ томом**</dt> <dt>2150694912 (0x80310000)</dt> </dl> | Этот диск заблокирован шифрование диска BitLocker. Необходимо разблокировать этот том с панели управления. <br/> |
| <dl> <dt>**ФВЕ \_ E \_ не \_ активировано**</dt> <dt>2150694920 (0x80310008)</dt> </dl> | Средство BitLocker не включено в томе. Добавьте предохранитель ключа, чтобы включить BitLocker. <br/>                    |
