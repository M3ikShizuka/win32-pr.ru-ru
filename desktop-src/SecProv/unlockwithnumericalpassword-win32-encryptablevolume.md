# <a name="unlockwithnumericalpassword-method-of-the-win32_encryptablevolume-class"></a>Метод Унлокквиснумерикалпассворд \_ класса Win32 енкриптаблеволуме

Метод **унлокквиснумерикалпассворд** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) использует предоставленный числовой пароль для доступа к содержимому тома данных.

Ключ шифрования тома должен быть защищен с помощью одного или нескольких предохранителей ключа типа "числовой пароль" (с помощью метода [**протекткэйвиснумерикалпассворд**](protectkeywithnumericalpassword-win32-encryptablevolume.md) ), чтобы иметь возможность разблокировать том с помощью этого метода.

> [!Note]  
> Если диск поддерживает аппаратное шифрование, эта функция устанавливает для полосы состояние "разблокирован".

 

## <a name="syntax"></a>Синтаксис


```mof
uint32 UnlockWithNumericalPassword(
  [in] string NumericalPassword
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Нумерикалпассворд* \[ окне\]
</dt> <dd>

Тип: **строка**

Строка, указывающая числовой пароль.

Числовой пароль должен содержать 48 цифр. Эти цифры можно разделить на 8 групп из 6 цифр, с последней цифрой в каждой группе, указывающей на значение контрольной суммы для группы. Каждая группа из 6 цифр должна быть кратной 11 и должна быть меньше 65536. При условии, что группа из шести цифр помечена как x1, x2, X3, X4, x5 и X6, контрольная сумма X6 вычисляется как – x1 + x2 – X3 + X4 – x5 mod 11.

Группы цифр можно разделять пробелами или дефисами. Таким образом, "XXXXXX-XXXXXX-XXXXXX-XXXXXX-XXXXXX-XXXXXX-XXXXXX-XXXXXX" или "XXXXXX XXXXXX XXXXXX XXXXXX (xxxxxx) XXXXXX XXXXXX" может также содержать допустимые числовые пароли.

</dd> </dl>

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.

Если том уже разблокирован и другие ошибки не возникают, этот метод возвращает значение 0.

|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                             | Метод успешно выполнен.<br/>                                                                                                                                                                                          |
| <dl> <dt>**ФВЕ \_ E \_ не \_ активировано**</dt> <dt>2150694920 (0x80310008)</dt> </dl>            | Средство BitLocker не включено в томе. Добавьте предохранитель ключа, чтобы включить BitLocker. <br/>                                                                                                                                   |
