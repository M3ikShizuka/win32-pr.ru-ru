# <a name="unlockwithexternalkey-method-of-the-win32_encryptablevolume-class"></a>Метод Унлокквисекстерналкэй \_ класса Win32 енкриптаблеволуме

Метод **унлокквисекстерналкэй** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) использует предоставленный внешний ключ для доступа к содержимому тома данных.

Ключ шифрования тома должен быть защищен с помощью одного или нескольких предохранителей ключа типа "внешний ключ" с использованием метода [**протекткэйвисекстерналкэй**](protectkeywithexternalkey-win32-encryptablevolume.md) , чтобы иметь возможность разблокировать том с помощью этого метода.

> [!Note]  
> Если диск поддерживает аппаратное шифрование, эта функция устанавливает для полосы состояние "разблокирован".

 

## <a name="syntax"></a>Синтаксис


```mof
uint32 UnlockWithExternalKey(
  [in] uint8 ExternalKey[]
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Екстерналкэй* \[ окне\]
</dt> <dd>

Тип: **Uint8 \[ \]**

Массив байтов, указывающий 256-разрядный внешний ключ, используемый для разблокировки тома. Этот ключ можно получить, вызвав метод [**жетекстерналкэйфромфиле**](getexternalkeyfromfile-win32-encryptablevolume.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.

Если том уже разблокирован, этот метод возвращает значение 0.



| Возвращаемый код и значение                                                                                                                                                                  | Описание                                                                                                                                 |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                  | Метод успешно выполнен.<br/>                                                                                                       |
| <dl> <dt>**Ошибка при \_ Значение не \_ Найдено**</dt> <dt>(0x)</dt> </dl>          | Том не имеет предохранителя ключа типа "внешний ключ".<br/>                                                             |
| <dl> <dt>**Ошибка при \_ Недопустимый \_ пароль**</dt> <dt>не задано значение (0x)</dt> </dl>   | Один или несколько предохранителей ключа типа "внешний ключ" существуют, но указанный параметр *екстерналкэй* не может разблокировать том.<br/> |
| <dl> <dt>**Д \_ INVALIDARG**</dt> <dt>2147942487 (0x80070057)</dt> </dl>          | Параметр *екстерналкэй* не является массивом размером 4.<br/>                                                                           |
| <dl> <dt>**ФВЕ \_ E \_ не \_ активировано**</dt> <dt>2150694920 (0x80310008)</dt> </dl> | Средство BitLocker не включено в томе. Добавьте предохранитель ключа, чтобы включить BitLocker. <br/>                                                |
