# <a name="getencryptionmethod-method-of-the-win32_encryptablevolume-class"></a>Метод Жетенкриптионмесод \_ класса Win32 енкриптаблеволуме

Метод **жетенкриптионмесод** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) указывает алгоритм шифрования и размер ключа, используемые в томе.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetEncryptionMethod(
  [out] uint32 EncryptionMethod,
  [out] string SelfEncryptionDriveEncryptionMethod
);
```
