# <a name="unlockwithcertificatethumbprint-method-of-the-win32_encryptablevolume-class"></a>Метод Унлокквисцертификатесумбпринт \_ класса Win32 енкриптаблеволуме

Метод **унлокквисцертификатесумбпринт** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) использует предоставленный отпечаток [*сертификата*](../secgloss/c-gly.md) для получения производного ключа и разблокировки зашифрованного тома.

> [!Note]  
> Если диск поддерживает аппаратное шифрование, эта функция устанавливает для полосы состояние "разблокирован".

 

## <a name="syntax"></a>Синтаксис


```mof
uint32 UnlockWithCertificateThumbprint(
  [in] string CertThumbprint,
  [in] string PIN
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*CertThumbprint* \[ окне\]
</dt> <dd>

Тип: **строка**

Значение отпечатка, равное 0, принимается, и в нем задается поиск соответствующего сертификата в локальном хранилище. При обнаружении одного сертификата BitLocker Поиск будет успешным. Если не найдено ни одного сертификата, метод завершается с ошибкой.

</dd> <dt>

*Закрепить* \[ окне\]
</dt> <dd>

Тип: **строка**

Указанная пользователем личная строка идентификации. Эта строка должна состоять из последовательности из 4 до 20 цифр. Эта строка используется для автоматической проверки подлинности [*поставщика хранилища ключей*](../secgloss/k-gly.md) (KSP) при использовании со [*смарт-картой*](../secgloss/s-gly.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.



| Возвращаемый код и значение                                                                                                                                                                            | Описание                                                                                                                                                                                                                                                              |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                            | Метод успешно выполнен.<br/>                                                                                                                                                                                                                                    |
| <dl> <dt>**ФВЕ \_ E \_ не \_ активировано**</dt> <dt>2150694920 (0x80310008)</dt> </dl>           | Средство BitLocker не включено в томе. Добавьте предохранитель ключа, чтобы включить BitLocker. <br/>                                                                                                                                                                             |
| <dl> <dt>**ФВЕ \_ E \_ Failed \_ authentication**</dt> <dt>2150694951 (0x80310027)</dt> </dl>   | Невозможно разблокировать том, используя предоставленные сведения. <br/>                                                                                                                                                                                             |
| <dl> <dt>**ФВЕ \_ \_ \_ Не \_ найден предохранитель "E**</dt> <dt>" 2150694963 (0x80310033)</dt> </dl>    | Указанный предохранитель ключа не существует на томе. Необходимо ввести другой предохранитель ключа.<br/>                                                                                                                                                                |
| <dl> <dt>**ФВЕ \_ E \_ PRIVATEKEY \_ AUTH \_ Failed**</dt> <dt>2150695060 (0x80310094)</dt> </dl> | Не удалось авторизовать [*закрытый ключ*](../secgloss/p-gly.md) , связанный с указанным сертификатом. Не указана авторизация закрытого ключа, или предоставлена недопустимая авторизация.<br/> |
