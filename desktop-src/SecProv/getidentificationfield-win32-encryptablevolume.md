# <a name="getidentificationfield-method-of-the-win32_encryptablevolume-class"></a>Метод Жетидентификатионфиелд \_ класса Win32 енкриптаблеволуме

Метод **жетидентификатионфиелд** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) возвращает строку идентификатора, доступную в метаданных тома.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetIdentificationField(
  [out] string IdentificationField
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Идентификатионфиелд* \[ заполняет\]
</dt> <dd>

Тип: **строка**

Строка, указывающая поле идентификации, назначенное тому.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.



| Возвращаемый код и значение                                                                                                                                                                  | Описание                                                                                                     |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                  | Метод успешно выполнен.<br/>                                                                           |
| <dl> <dt>**ФВЕ \_ E с \_ заблокированным \_ томом**</dt> <dt>2150694912 (0x80310000)</dt> </dl> | Этот диск заблокирован шифрование диска BitLocker. Необходимо разблокировать этот том с панели управления. <br/> |
| <dl> <dt>**ФВЕ \_ E \_ не \_ активировано**</dt> <dt>2150694920 (0x80310008)</dt> </dl> | Средство BitLocker не включено в томе. Добавьте предохранитель ключа, чтобы включить BitLocker. <br/>                    |
