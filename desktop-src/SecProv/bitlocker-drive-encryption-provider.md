# <a name="bitlocker-drive-encryption-provider"></a>Поставщик шифрования диска BitLocker

интерфейс поставщика WMI предназначен для управления и настройки шифрование диска BitLocker (BDE) на Windows server 2008 R2, Windows Server 2008 и только для конкретных версий Windows 7, Windows vista Enterprise и Windows vista Ultimate. Он предназначен для чтения с помощью сценариев записи, элементов пользовательского интерфейса или других средств администрирования для BDE, а также для пользователей, желающих больше узнать о возможностях BDE.
