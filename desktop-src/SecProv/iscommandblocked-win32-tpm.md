# <a name="iscommandblocked-method-of-the-win32_tpm-class"></a>Метод Искоммандблоккед \_ класса TPM Win32

## <a name="syntax"></a>Синтаксис


```mof
uint32 IsCommandBlocked(
  [in]  uint32 CommandOrdinal,
  [out] uint32 IsCommandBlocked
);
```

## <a name="remarks"></a>Комментарии

Этот метод считывает следующие разделы реестра:

-   Групповая политика список заблокированных команд TPM:

    **Ключ \_ \_** \\ **Политики программного обеспечения** локального компьютера \\  \\  \\ **список блоккедкоммандс доверенного платформенного модуля** \\  \\  Майкрософт

-   Список заблокированных команд TPM по умолчанию:

    **Ключ \_ \_** \\ **Программное обеспечение** локального компьютера ( \\ **Microsoft** \\ **TPM** \\ **блоккедкоммандс** \\ **List** )

-   Групповая политика параметр, чтобы игнорировать список заблокированных команд TPM по умолчанию:

    **Ключ \_ \_** \\ **Политики программного обеспечения** локального компьютера ( \\  \\ **Microsoft** \\ **TPM** \\ **блоккедкоммандс** \\ **игноредефаултлист** )

-   Локальный список заблокированных команд TPM:

    **Ключ \_ CurrentControlSet служб на локальном \_ компьютере**. \\  \\  \\  \\  \\ **Параметры** SharedAccess \\ **TPM** \\ **блоккедкоммандс** \\ **список**

-   Групповая политика параметр, чтобы игнорировать локальный список заблокированных команд TPM:

    **Ключ \_ \_** \\ **Политики программного обеспечения** локального компьютера ( \\  \\ **Microsoft** \\ **TPM** \\ **блоккедкоммандс** \\ **игнорелокаллист** )

Возвращаемое значение метода может быть получено следующим образом:

-   Предположим, что *искоммандблоккедтемп* — это 3-разрядное значение, изначально равное 0.
-   Если параметр групповая политика игнорировать список заблокированных команд TPM по умолчанию не существует или равен 0, а значение в разделе реестра, соответствующее списку заблокированных команд TPM по умолчанию, — *коммандординал*, то для параметра наименее значащий бит *искоммандблоккедтемп* устанавливается значение 1.
-   Если параметр групповая политика игнорировать локальный список заблокированных команд TPM не существует или равен 0, а значение в разделе реестра, соответствующее локальному списку заблокированных команд доверенного платформенного модуля, — *коммандординал*, то для второго наименее значимого бита *искоммандблоккедтемп* устанавливается значение 1.
-   Если в разделах реестра, соответствующих групповая политика списку заблокированных команд доверенного платформенного модуля (TPM), указано значение *коммандординал*, то наиболее значимый бит *искоммандблоккедтемп* имеет значение 1.
-   *Искоммандблоккед* возвращает *искоммандблоккедтемп*.

Файлы MOF-файл (MOF) содержат определения для классов инструментарий управления Windows (WMI) (WMI). MOF-файлы не устанавливаются в составе Windows SDK. Они устанавливаются на сервере при добавлении связанной роли с помощью диспетчер сервера. Дополнительные сведения о файлах MOF см. в разделе [MOF-файл (MOF)](../wmisdk/managed-object-format--mof-.md).
