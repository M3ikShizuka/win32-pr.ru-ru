# <a name="gethardwareencryptionstatus-method-of-the-win32_encryptablevolume-class"></a>Метод Жесардваринкриптионстатус \_ класса Win32 енкриптаблеволуме

Метод **жесардваринкриптионстатус** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) определяет, расположен ли том на диске, поддерживающем или поддерживающем шифрование оборудования.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetHardwareEncryptionStatus(
  [out] uint32 HardwareEncryptionStatus
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Хардваринкриптионстатус* \[ заполняет\]
</dt> <dd>

Указывает, может ли диск поддерживать шифрование оборудования. Это может быть одно из следующих значений.



| Значение                                                                                                                                                                                                                                               | Значение                                             |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------|
| <span id="Not_supported"></span><span id="not_supported"></span><span id="NOT_SUPPORTED"></span><dl> <dt>**Не поддерживается**</dt> <dt>0</dt> </dl> | Аппаратное шифрование не поддерживается.<br/>    |
| <span id="No_protection"></span><span id="no_protection"></span><span id="NO_PROTECTION"></span><dl> <dt>**Нет защиты**</dt> <dt>1</dt> </dl> | Шифрование диска не поддерживается.<br/>       |
| <span id="Uses_software"></span><span id="uses_software"></span><span id="USES_SOFTWARE"></span><dl> <dt>**Использует программное обеспечение**</dt> <dt>2</dt> </dl> | Метод шифрования основан на программном обеспечении.<br/> |
| <span id="Uses_hardware"></span><span id="uses_hardware"></span><span id="USES_HARDWARE"></span><dl> <dt>**Использует оборудование**</dt> <dt>3</dt> </dl> | Диск поддерживает аппаратное шифрование.<br/>  |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция возвращает ноль (0), если том совместим с аппаратным шифрованием BitLocker. В противном случае функция возвращает ошибку.



| Возвращаемый код и значение                                                                                                                                 | Описание                                                             |
|---------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl> | Том совместим с аппаратным шифрованием BitLocker.<br/> |
