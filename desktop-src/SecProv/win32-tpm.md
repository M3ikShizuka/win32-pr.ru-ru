# <a name="win32_tpm-class"></a>\_Класс TPM Win32

Класс **\_ TPM Win32** представляет доверенный платформенный модуль (TPM) (доверенный платформенный модуль), микросхема безопасности оборудования, которая предоставляет корень доверия для компьютерной системы.

## <a name="syntax"></a>Синтаксис

``` syntax
class Win32_Tpm
{
  boolean IsActivated_InitialValue;
  boolean IsEnabled_InitialValue;
  boolean IsOwned_InitialValue;
  string  SpecVersion;
  string  ManufacturerVersion;
  string  ManufacturerVersionInfo;
  uint32  ManufacturerId;
  string  PhysicalPresenceVersionInfo;
};
```
