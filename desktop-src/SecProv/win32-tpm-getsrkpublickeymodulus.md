# <a name="win32_tpmgetsrkpublickeymodulus-method"></a>\_Метод Win32 TPM:: жетсркпубликкэймодулус

возвращает модуль открытой части доверенного платформенного служба хранилища корневого ключа.

Этот метод доступен только локальным администраторам.

## <a name="syntax"></a>Синтаксис


```C++
uint32 GetSrkPublicKeyModulus(
  [out] uint8 SrkPublicKeyModulus
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Сркпубликкэймодулус* \[ заполняет\]
</dt> <dd>

возвращает массив из 256 байт, содержащий модуль открытой части служба хранилища корневого ключа TPM

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Могут возвращаться все ошибки доверенного платформенного модуля, а также ошибки, относящиеся к [базовым службам TPM](../tbs/tbs-return-codes.md) .

Ниже приведены общие коды возврата.



| Возвращаемый код и значение                                                                                                                                 | Описание                           |
|---------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl> | Метод успешно выполнен.<br/> |
