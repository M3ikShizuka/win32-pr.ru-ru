# <a name="getkeyprotectorcertificate-method-of-the-win32_encryptablevolume-class"></a>Метод Жеткэйпротекторцертификате \_ класса Win32 енкриптаблеволуме

Метод **жеткэйпротекторцертификате** класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) извлекает [*открытый ключ*](../secgloss/p-gly.md) и отпечаток [*сертификата*](../secgloss/c-gly.md) для предохранителя открытого ключа.

## <a name="syntax"></a>Синтаксис


```mof
uint32 GetKeyProtectorCertificate(
  [in]  string VolumeKeyProtectorID,
  [out] uint8  PublicKey[],
  [out] string CertThumbprint,
  [out] uint32 CertType
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Волумекэйпротекторид* \[ окне\]
</dt> <dd>

Тип: **строка**

Уникальный строковый идентификатор, используемый для управления предохранителем ключа зашифрованного тома.

</dd> <dt>

*PublicKey* \[ заполняет\]
</dt> <dd>

Тип: **Uint8 \[ \]**

Массив байтов, указывающий открытый ключ.

</dd> <dt>

*CertThumbprint* \[ заполняет\]
</dt> <dd>

Тип: **строка**

Строка, указывающая отпечаток сертификата.

</dd> <dt>

*Типом* \[ заполняет\]
</dt> <dd>

Тип: **UInt32**

Целое число без знака, указывающее тип предохранителя ключа. Это целое число используется для различения агента восстановления данных (DRA) и сертификатов пользователей.



| Значение                                                                        | Значение                                  |
|------------------------------------------------------------------------------|------------------------------------------|
| <dl> <dt>1</dt> </dl> | Сертификат является агентом DRA.<br/>     |
| <dl> <dt>2</dt> </dl> | Сертификат не является агентом DRA.<br/> |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **UInt32**

При сбое этот метод возвращает один из следующих кодов или другой код ошибки.



| Возвращаемый код и значение                                                                                                                                                                                       | Описание                                                                                                     |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**С \_ ОК**</dt> <dt>0 (0x0)</dt> </dl>                                                       | Метод успешно выполнен.<br/>                                                                           |
| <dl> <dt>**Д \_ INVALIDARG**</dt> <dt>2147942487 (0x80070057)</dt> </dl>                               | Указанный предохранитель ключа не является предохранителем ключа. Необходимо ввести другой предохранитель ключа.<br/>            |
| <dl> <dt>**ФВЕ \_ E с \_ заблокированным \_ томом**</dt> <dt>2150694912 (0x80310000)</dt> </dl>                      | Этот диск заблокирован шифрование диска BitLocker. Необходимо разблокировать этот том с панели управления. <br/> |
| <dl> <dt>**ФВЕ \_ E \_ не \_ активировано**</dt> <dt>2150694920 (0x80310008)</dt> </dl>                      | Средство BitLocker не включено в томе. Добавьте предохранитель ключа, чтобы включить BitLocker. <br/>                    |
| <dl> <dt>**ФВЕ \_ \_ \_ \_ \_ Требуется сертификат пользователя политики**</dt> <dt>2150695027 (0x80310073)</dt> </dl> | Групповая политика требует использования сертификата пользователя, например смарт-карты.<br/>                           |



 

## <a name="remarks"></a>Комментарии

Файлы MOF-файл (MOF) содержат определения для классов инструментарий управления Windows (WMI) (WMI). MOF-файлы не устанавливаются в составе Windows SDK. Они устанавливаются на сервере при добавлении связанной роли с помощью диспетчер сервера. Дополнительные сведения о файлах MOF см. в разделе [MOF-файл (MOF)](../wmisdk/managed-object-format--mof-.md).
