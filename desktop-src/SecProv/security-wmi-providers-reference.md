# <a name="security-wmi-providers-reference"></a>Справочник по поставщикам WMI для системы безопасности

Справочные страницы WMI безопасности содержат подробное описание зашифрованного поставщика WMI для тома и поставщика доверенный платформенный модуль (TPM) (TPM).

Поставщики WMI для системы безопасности включают следующие поставщики.



| Section                                                                        | Описание                                                                                                                                  |
|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------|
| [Поставщик шифрования диска BitLocker](bitlocker-drive-encryption-provider.md) | Документация по API для управления шифрование диска BitLocker (BDE) с помощью класса [**Win32 \_ енкриптаблеволуме**](win32-encryptablevolume.md) . |
| [Поставщик доверенный платформенный модуль (TPM)](trusted-platform-module-provider.md)       | Документация по API для управления доверенный платформенный модуль (TPM) (TPM) с помощью [**класса \_ TPM Win32**](win32-tpm.md) .                            |
