# <a name="removeblockedcommand-method-of-the-win32_tpm-class"></a>Метод Ремовеблоккедкомманд \_ класса TPM Win32

Метод **ремовеблоккедкомманд** класса [**\_ TPM Win32**](win32-tpm.md) удаляет команду TPM из локального списка команд, заблокированных на компьютере.

Возможно, потребуется настроить групповая политика, чтобы разблокировать команду. Например, если вы хотите разрешить команду, которая заблокирована по умолчанию, необходимо сначала настроить групповая политика, чтобы игнорировать список заблокированных команд по умолчанию. Используйте метод [**искоммандблоккед**](iscommandblocked-win32-tpm.md) , чтобы определить, как команда блокируется с помощью локального списка или групповой политики.

> [!Caution]  
> Операционная система предотвращает запуск предопределенного набора команд по умолчанию. Изменения этого значения по умолчанию могут повлиять на безопасность, конфиденциальность или функциональность компьютера.

 

## <a name="syntax"></a>Синтаксис


```mof
uint32 RemoveBlockedCommand(
  [in] uint32 CommandOrdinal
);
```

## <a name="remarks"></a>Комментарии

Файлы MOF-файл (MOF) содержат определения для классов инструментарий управления Windows (WMI) (WMI). MOF-файлы не устанавливаются в составе Windows SDK. Они устанавливаются на сервере при добавлении связанной роли с помощью диспетчер сервера. Дополнительные сведения о файлах MOF см. в разделе [MOF-файл (MOF)](../wmisdk/managed-object-format--mof-.md).
