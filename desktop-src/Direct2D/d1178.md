# <a name="d1178-custom-dash-style-requires-dash-array"></a>D1178: для пользовательского типа штриха требуется массив штрихов

Для создания стиля обводки с \_ пользовательским стилем штриха \_ требуется массив тире.






 

## <a name="possible-causes"></a>Возможные причины

Когда приложение попыталось создать ID2D1StrokeStyle, в нем использовался \_ \_ настраиваемый атрибут стиля тире D2D1, \_ но был ЗАДАН массив штрихов null.

## <a name="possible-fixes"></a>Возможные исправления

Если для приложения требуются пользовательские тире, убедитесь, что указан допустимый массив штрихов. Пример см. в разделе Креатестрокестиле. Если допускается использование встроенного стиля штриха, оставьте массив дефисов РАВНым NULL и используйте описатель, например D2D1 штрих- \_ \_ \_ пунктир.
