# <a name="d1171-redundant-call-to-map-bitmap"></a>D1171: избыточный вызов точечного рисунка карты

Предпринята попытка сопоставить уже сопоставленный точечный рисунок.






 

## <a name="possible-causes"></a>Возможные причины

Direct2D может выводить это сообщение и сообщать об ошибке, если приложение пытается сопоставить точечный рисунок, который уже сопоставлен с помощью ID2D1Bitmap1:: Map.

## <a name="possible-fixes"></a>Возможные исправления

Так же, как ID2D1RenderTarget:: EndDraw должен следовать за вызовами ID2D1RenderTarget:: Бегиндрав, убедитесь, что ID2D1Bitmap1:: uncall следует вызов ID2D1Bitmap1:: Map. Если приложение использует обратные вызовы или пути рекурсивного кода для отображения точечных рисунков, убедитесь в правильности его поведения.
