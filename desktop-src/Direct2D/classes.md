# <a name="direct2d-helper-classes"></a>Вспомогательные классы Direct2D

Пространство имен D2D1 определяет следующие классы.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                       | Описание                                                                                                                              |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**колорф**](/windows/win32/api/d2d1helper/nl-d2d1helper-colorf)<br/>         | Определяет набор констант, представляющих известные цвета, и предоставляет удобные методы для создания экземпляров различных цветов.<br/>  |
| [**Matrix3x2F**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f)<br/> | Класс [**Matrix3x2F**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f) представляет матрицу объемом 3 на 2 и предоставляет удобные методы для создания матриц.<br/> |
| [**Matrix4x3F**](matrix4x3f.md)<br/> | Класс [**Matrix4x3F**](matrix4x3f.md) представляет матрицу 4 на 3 и предоставляет удобные методы для создания матриц.<br/> |
| [**Matrix4x4F**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)<br/> | Класс [**Matrix4x4F**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f) представляет матрицу 4 на 4 и предоставляет удобные методы для создания матриц.<br/> |
| [**Matrix5x4F**](matrix5x4f.md)<br/> | Класс [**Matrix5x4F**](matrix5x4f.md) представляет матрицу с 5 по 4 и предоставляет удобные методы для создания матриц.<br/> |
