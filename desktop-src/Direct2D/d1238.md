# <a name="d1238-source-transform-single-channel-srgb"></a>D1238: преобразование исходного кода с одним каналом SRGB

ID2D1SourceTransform не позволяет сочетать D2D1 \_ buffer \_ \_ sRGB с D2D1 \_ Channel \_ Depth \_ 1.






 

## <a name="possible-causes"></a>Возможные причины

Приложение указало \_ точность D2D1 \_ буфера \_ sRGB и D2D1 \_ Channel \_ \_ 1 на ID2D1SourceTransform.

## <a name="possible-fixes"></a>Возможные исправления

\_ \_ \_ \_ \_ Вместо этого следует использовать по умолчанию глубину канала D2D1 или номер 4 канала D2D1 \_ .
