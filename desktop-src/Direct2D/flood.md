# <a name="flood-effect"></a>Результат переполнения

Используйте результат переполнения для создания точечного рисунка на основе заданного цвета и альфа-значения. Этот результат можно использовать, если требуется конкретный цвет в качестве входного для действия, например цвет фона.

> [!Note]  
> Этот результат передается по указанному значению цвета. Необходимо вручную выполнить предварительное умножение значений, если планируется передать выходные данные эффектам, которые предполагают предварительно умноженные входные данные.

 

Идентификатором CLSID для этого действия является CLSID \_ D2D1Flood.

Результат переполнения не имеет входного изображения.

## <a name="example-image"></a>Пример изображения

![Пример изображения, которое выводится зеленым цветом.](images/20-flood.png)


```C++
ComPtr<ID2D1Effect> floodEffect;
m_d2dContext->CreateEffect(CLSID_D2D1Flood, &floodEffect);

floodEffect->SetValue(D2D1_FLOOD_PROP_COLOR, D2D1::Vector4F(0.0f, 1.0f, 0.0f, 1.0f));

m_d2dContext->BeginDraw();
m_d2dContext->DrawImage(floodEffect.Get());
m_d2dContext->EndDraw();
```



## <a name="effect-properties"></a>Свойства эффектов



| Отображаемое имя и перечисление индекса                   | Описание                                                                                                                                                                                                                                                                                                                                                                                     |
|------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Цвет<br/> \_ \_ Цвет Prop D2D1 \_ Flood<br/> | Цвет и непрозрачность точечного рисунка. Это свойство является D2D1 \_ векторным \_ 4Fом. Отдельные значения для каждого канала имеют тип FLOAT, без привязки и без единицы. Этот результат не изменяет значения для каналов.<br/> Значения RGBA для каждого канала в диапазоне от 0 до 1.<br/> Тип — D2D1 \_ vector \_ 4F.<br/> Значение по умолчанию — {0.0 f, 0,0 f, 0,0 f, 1,0 f}.<br/> |



 

## <a name="output-bitmap"></a>Битовая карта вывода

Этот результат приводит к созданию точечного рисунка с логически бесконечным размером.
