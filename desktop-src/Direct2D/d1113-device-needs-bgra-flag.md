# <a name="d1113-device-needs-bgra-flag"></a>D1113: устройству требуется флаг BGRA

Устройство Direct3D не было создано с помощью D3D10 \_ Create \_ Device \_ BGRA \_ support и, следовательно, несовместимо с Direct2D.



|             |         |
|-------------|---------|
| Уровень ошибки | Предупреждение |



 

## <a name="possible-causes"></a>Возможные причины

Устройство Direct3D не было создано с помощью D3D10 \_ Create \_ Device \_ BGRA \_ support.
