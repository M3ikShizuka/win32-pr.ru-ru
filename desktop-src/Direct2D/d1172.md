# <a name="d1172-invalid-to-unmap-nonmapped-bitmap"></a>D1172: недопустимо сопоставление несопоставленного точечного рисунка

Предпринята попытка отменить сопоставление несопоставленного точечного рисунка.






 

## <a name="possible-causes"></a>Возможные причины

Direct2D может выводить это сообщение и сообщать об ошибке, если приложение пытается отменить сопоставление битовой карты, которая еще не сопоставлена с помощью ID2D1Bitmap1:: Map.

## <a name="possible-fixes"></a>Возможные исправления

Так же, как ID2D1RenderTarget:: EndDraw должен следовать за вызовами ID2D1RenderTarget:: Бегиндрав, убедитесь, что ID2D1Bitmap1:: uncall следует вызов ID2D1Bitmap1:: Map. Не вызывайте ID2D1Bitmap1:: uncall несколько раз, чтобы отменить сопоставление точечного рисунка.
