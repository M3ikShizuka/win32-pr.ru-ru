# <a name="d1183-createwicbitmaprendertarget-with-incompatible-feature-level"></a>D1183: Креатевикбитмапрендертаржет с несовместимым уровнем функции

Невозможно передать Direct2D уровня компонентов, отличного от D2D1 \_ \_ \_ по умолчанию, в креатевикбитмапрендертаржет.






 

## <a name="possible-causes"></a>Возможные причины

Если приложение с именем ID2D1Factory:: Креатевикбитмапрендертаржет, поле Минлевел \_ \_ аргумента D2D1 свойства целевого объекта прорисовки \_ не было D2D1 \_ на \_ уровне компонентов \_ по умолчанию.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что на уровне компонента, используемого при создании целевого объекта рендеринга WIC-интероппед, используется значение по умолчанию. Взаимодействие с WIC включает в себя только программные функции. семантика уровня компонента важна для аппаратного поведения.
