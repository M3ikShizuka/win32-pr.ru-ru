# <a name="d1167-read-map-option-requires-cpu-readable-bitmap"></a>D1167: для параметра "чтение карты" требуется доступный для чтения точечный рисунок ЦП

\_ \_ Чтение параметров карты D2D1 \_ допустимо только для точечного рисунка с доступным для чтения ЦП






 

## <a name="possible-causes"></a>Возможные причины

Direct2D может выводить это сообщение, если приложение вызывает ID2D1Bitmap1:: Map с \_ \_ атрибутом D2D1 Map \_ Read, но точечный рисунок не читается в ЦП.

## <a name="possible-fixes"></a>Возможные исправления

Если приложению требуется доступ на чтение с помощью ID2D1Bitmap1:: Map, создайте точечный рисунок с \_ помощью \_ параметра D2D1 Bitmap параметры \_ \_ чтения ЦП.
