# <a name="d1203-copyfrombitmap-with-multisampled-source"></a>D1203: Копифромбитмап с источником с многовыборочной выборкой

Копифромбитмап использовался с многовыборочным точечным рисунком источника. Вызов будет по-прежнему выдан, но скопированный результат может не представлять шаблон, который ожидался для многообразца.






 

## <a name="possible-causes"></a>Возможные причины

Приложение взаимодействует с многовыборочной текстурой Direct3D как точечным рисунком Direct2D и называется Копифромбитмап на точечном рисунке.

## <a name="possible-fixes"></a>Возможные исправления

Избегайте использования многовыборочных текстур в качестве точечных рисунков, если важна Копифромбитмап функция. Скопированный результат не может представлять шаблон, который ожидался для многообразца.
