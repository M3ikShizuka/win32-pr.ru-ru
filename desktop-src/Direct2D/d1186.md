# <a name="d1186-dc-render-target-requires-binddc"></a>D1186: для целевого объекта прорисовки контроллера домена требуется Бинддк

Не допускается использование целевого объекта прорисовки контроллера домена до вызова метода Бинддк.






 

## <a name="possible-causes"></a>Возможные причины

Приложение создало ID2D1DCRenderTarget и использовало его для вызова Бегиндрав, но еще не вызывало его метод Бинддк. Или, Бинддк завершилась ошибкой.

## <a name="possible-fixes"></a>Возможные исправления

Прежде чем использовать целевой объект прорисовки DC-интероппед, приложения должны вызвать Бинддк, чтобы связать целевой объект с контекстом устройства GDI. Целевой объект не считается допустимым, пока не будет назначен контекст устройства GDI. Пример см. в разделе ID2D1DCRenderTarget:: Бинддк. Если приложение уже вызывает Бинддк, убедитесь, что Бинддк не вернул ошибку.
