# <a name="d1249-inaccessible-target"></a>D1249: недоступная цель

Обнаружена попытка нарисовать на недоступный целевой объект.






 

## <a name="possible-causes"></a>Возможные причины

Обнаружена попытка нарисовать на недоступный целевой объект. Одна из причин, по которой целевой объект может быть недоступен, — это, если он был создан из общей текстуры, а текстура не была получена.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что все целевые объекты доступны для подготовки к просмотру с помощью Direct2D.
