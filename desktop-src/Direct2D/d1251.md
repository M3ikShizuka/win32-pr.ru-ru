# <a name="d1251-d3d-state-issue"></a>D1251: ошибка состояния D3D

Состояние устройства Direct3D изменилось, когда Direct2D все еще использовал его. Это может свидетельствовать о нарушении потоков, поскольку только один поток может использовать устройство за раз.






 

## <a name="possible-causes"></a>Возможные причины

Приложение может использовать одну потоковую фабрику D2D одновременно в двух разных потоках.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что приложение правильно соблюдает политики потоков Direct2D's.
