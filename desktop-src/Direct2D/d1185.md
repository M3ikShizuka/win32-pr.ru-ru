# <a name="d1185-fillgeometry-opacity-mask-brush-restriction"></a>D1185: ограничение кисти маски непрозрачности Филлжеометри

Если Филлжеометри используется с маской непрозрачности, основная кисть должна быть маскированной растровой кистью.






 

## <a name="possible-causes"></a>Возможные причины

При использовании ID2D1RenderTarget:: Филлжеометри основная кисть была типом, отличной от ID2D1BitmapBrush. Или первичная кисть имеет режим расширения x или y, отличный от D2D1 в \_ режиме расширения \_ \_ .

## <a name="possible-fixes"></a>Возможные исправления

Если приложение использует Филлжеометри с маской непрозрачности, убедитесь, что основная кисть (не путать с кистью непрозрачности) является ID2D1BitmapBrush с маскированными режимами выделения. Если поведение маскировки непрозрачности не имеет значения, оставьте для аргумента кисти непрозрачности значение NULL, чтобы удалить эти ограничения для основной кисти.
