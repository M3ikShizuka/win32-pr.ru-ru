# <a name="d1117-bitmap-is-not-a-target"></a>D1117: точечный рисунок не является целевым объектом

Сбой Сеттаржет, так как точечный рисунок не имеет целевого параметра.






 

## <a name="possible-causes"></a>Возможные причины

Чтобы задать ID2D1Bitmap в качестве целевого объекта, он должен быть создан с помощью \_ \_ \_ флага целевого параметра D2D1 Bitmap. Этот флаг доступен в виде поля D2D1 \_ Bitmap \_ PROPERTIES1, используемого с ID2D1DeviceContext:: креатебитмап.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что приложение создает конечный точечный рисунок \_ , используя параметры битовой карты D2D1 \_ \_ . Если он создает целевой точечный рисунок с помощью ID2D1RenderTarget:: Креатебитмап, измените его, чтобы он использовал ID2D1DeviceContext:: Креатебитмап. Если указан целевой флаг, Direct2D также проверяет совместимость атрибутов Direct3D текстуры с его использованием в качестве целевого объекта.
