# <a name="d1155-fillopacitymask-requires-aliased-rendering"></a>D1155: для ФиллопаЦитимаск требуется отрисовка с псевдонимами

Для ФиллопаЦитимаск требуется, чтобы в качестве режима сглаживания был задан режим \_ \_ сглаживания D2D1 \_ .





 

## <a name="possible-causes"></a>Возможные причины

Когда приложение вызывает ID2D1RenderTarget:: ФиллопаЦитимаск, режим сглаживания рендертаржет должен быть D2D1 режим сглаживания с \_ \_ \_ псевдонимом. Поведение целевого объекта рендеринга по умолчанию заключается в использовании сглаживания, поэтому необходимо вызвать метод ID2D1RenderTarget:: Сетантиалиасмоде перед заполнением маски непрозрачности.

## <a name="possible-fixes"></a>Возможные исправления

Используйте ID2D1RenderTarget:: Сетантиалиасмоде, чтобы в рендертаржет использовалась отрисовка с псевдонимами при заполнении маски непрозрачности. Если для последующих операций требуется сглаживание, присвойте режиму сглаживания целевого объекта значение D2D1 для \_ \_ \_ каждого \_ примитива. Убедитесь, что вызовы ID2D1RenderTarget:: Сетантиалиасмоде и ID2D1RenderTarget:: ФиллопаЦитимаск выдаются в одном и том же целевом объекте.
