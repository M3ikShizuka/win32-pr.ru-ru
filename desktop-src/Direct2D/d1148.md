# <a name="d1148-invalid-operation-on-closed-print-control"></a>D1148: Недопустимая операция с закрытым элементом управления печатью

Direct2D Print: [**addPage**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-addpage) или [**Close**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-close) вызывается после закрытия [**ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol)






 

## <a name="possible-causes"></a>Возможные причины

Это сообщение может появиться, если приложение вызывает ID2D1PrintControl:: AddPage в элементе управления печатью, который был закрыт. Direct2D считает объекты элементов управления печати неизменяемыми после закрытия, аналогично геометрией путей и спискам команд.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что приложение не пытается добавить страницы в закрытый элемент управления печатью. Измените логику приложения таким образом, чтобы все обязательные страницы отправлялись в элемент управления печатью перед закрытием.
