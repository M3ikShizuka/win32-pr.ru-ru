# <a name="d1150-unsupported-mip-level-selection"></a>D1150: неподдерживаемый выбор уровня MIP

Предоставленный IDXGISurface2 имеет несколько уровней MIP, но выбор уровня MIP не поддерживается на этом уровне устройства (D3D9).






 

## <a name="possible-causes"></a>Возможные причины

Текстуры Direct3D интероппед могут быть сопоставлены MIP; Direct2D будет выполняться только на одном уровне MIP за раз. Другие уровни MIP родительской текстуры не будут затронуты или визуализированы Direct2D. Однако устройства уровня "функция 9" не поддерживают выбор уровня MIP и не могут использовать эту возможность взаимодействия.

## <a name="possible-fixes"></a>Возможные исправления

Для приложений, которые должны быть правильно запущены на устройствах уровня Feature 9, рассмотрите возможность изменения логики отрисовки, чтобы избежать необходимости в взаимодействии текстур мипмаппед. По возможности используйте вместо него промежуточную. Чтобы избежать последствий, связанных с этим ограничением, приложение может выбрать его реализацию в качестве резервного пути, относящегося к уровню компонентов 9.
