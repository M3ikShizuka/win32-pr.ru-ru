# <a name="matrix5x4f-class"></a>Класс Matrix5x4F

Класс **Matrix5x4F** представляет матрицу с 5 по 4 и предоставляет удобные методы для создания матриц.

## <a name="members"></a>Элементы

Класс **Matrix5x4F** наследует от [**D2D1 \_ Matrix \_ 5x4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f). **Matrix5x4F** также имеет следующие типы членов:

-   [Конструкторы](#constructors)
