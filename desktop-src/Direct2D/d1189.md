# <a name="d1189-cpu-read-bitmap-option-was-used-incorrectly"></a>D1189: неправильное использование параметра "ЦП Read Bitmap"

\_Параметры битовой карты D2D1 \_ \_ \_ для чтения ЦП должны сочетаться с \_ параметрами D2D1 Bitmap \_ \_ \_






 

## <a name="possible-causes"></a>Возможные причины

В приложении указано несовместимое сочетание \_ флагов параметров точечного рисунка D2D1 \_ .

## <a name="possible-fixes"></a>Возможные исправления

Удалите один из флагов, чтобы создать допустимый набор \_ параметров битовой карты D2D1 \_ .
