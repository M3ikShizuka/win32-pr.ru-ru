# <a name="d1162-fillmesh-requires-aliased-rendering"></a>D1162: для Филлмеш требуется отрисовка с псевдонимами

Для Филлмеш требуется, чтобы в качестве режима сглаживания был задан режим \_ \_ сглаживания D2D1 \_ .






 

## <a name="possible-causes"></a>Возможные причины

Когда приложение вызывает ID2D1RenderTarget:: Филлмеш, режим сглаживания рендертаржет должен быть D2D1 режим сглаживания с \_ \_ \_ псевдонимом. Поведение целевого объекта рендеринга по умолчанию заключается в использовании сглаживания, поэтому необходимо вызвать ID2D1RenderTarget:: Сетантиалиасмоде перед заполнением сеток.

## <a name="possible-fixes"></a>Возможные исправления

Используйте ID2D1RenderTarget:: Сетантиалиасмоде, чтобы гарантировать, что рендертаржет использует отображение с псевдонимом при заполнении сеток. Если для последующих операций требуется сглаживание, присвойте режиму сглаживания целевого объекта значение D2D1 для \_ \_ \_ каждого \_ примитива. Убедитесь, что вызовы ID2D1RenderTarget:: Сетантиалиасмоде и ID2D1RenderTarget:: Филлмеш выдаются в одном и том же целевом объекте.
