# <a name="d1230-unknown-buffer-precision-support"></a>D1230: неизвестная поддержка точности буфера

Вызов ИсбуфферпреЦисионсуппортед был выполнен с \_ \_ неизвестной точностью буфера D2D1 \_ . Этот API возвращает значение false, хотя \_ \_ \_ неизвестная точность буфера D2D1 может быть допустимой в качестве входных данных для некоторых интерфейсов API.






 

## <a name="possible-causes"></a>Возможные причины

Вызов ИсбуфферпреЦисионсуппортед был выполнен с \_ \_ неизвестной точностью буфера D2D1 \_ .

## <a name="possible-fixes"></a>Возможные исправления

\_ \_ \_ Чтобы определить, является ли неизвестная точность буфера D2D1 неизвестной точностью для данного API, прочитайте документацию по этому API, а не пытаясь определить этот программный интерфейс.
