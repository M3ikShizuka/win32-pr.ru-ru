# <a name="d1158-pixels-unit-mode-unsupported-with-print"></a>D1158: режим единицы измерения в пикселях не поддерживается при печати

Direct2D Print: в списке команд, отправленном для печати, содержится неподдерживаемая команда Сетунитмоде (ПИКСЕЛЕЙ).





 

## <a name="possible-causes"></a>Возможные причины

Direct2Dная печать не разрешает режим единицы в ПИКСЕЛях. Единицей по умолчанию является "DIP".

## <a name="possible-fixes"></a>Возможные исправления

Чтобы устранить это сообщение, не устанавливайте в качестве единицы измерения контекста устройства ПИКСЕЛы во время создания печатного содержимого. Если необходимо применить глобальное преобразование масштабирования, которое влияет на все операции рисования примитивов, подумайте, достаточно ли эквивалентного вызова ID2D1RenderTarget:: Сетдпи или ID2D1RenderTarget:: Сеттрансформ.
