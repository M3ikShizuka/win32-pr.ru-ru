# <a name="d1232-32bpp-rgba-causes-conversion"></a>D1232:32 бита RGBA приводит к преобразованию

Рассмотрите возможность использования BGR, а не RGB для форматов 32 бита, чтобы можно было избежать дополнительных преобразований формата.
