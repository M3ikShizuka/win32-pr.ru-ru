# <a name="d1234-null-resource-brush"></a>D1234: кисть ресурсов со значением NULL

Растровую кисть или кисть изображения нельзя использовать для рисования, если ее связанный точечный рисунок или изображение имеет значение NULL.






 

## <a name="possible-causes"></a>Возможные причины

Приложение попыталось отобразить примитив с помощью ID2D1BitmapBrush или ID2D1ImageBrush, точечный рисунок или изображение которого имело значение NULL.

## <a name="possible-fixes"></a>Возможные исправления

Перед их использованием убедитесь, что для точечных рисунков или кистей изображений назначен допустимый точечный рисунок или изображение. Приложение может сделать это во время создания или позже с помощью ID2D1BitmapBrush:: Сетбитмап или ID2D1ImageBrush:: Сетимаже. Direct2D разрешает ресурсы кисти NULL, чтобы приложения могли освобождать ресурсы кисти после их использования для подготовки к просмотру. Приложения могут сделать это, если ресурс точечного рисунка или изображения особенно дорогостоящий, и сценарий требует немедленного освобождения. Если задать для ресурса кисти значение NULL, Direct2D не сможет сохранить необработанную ссылку на него. Однако если приложение присваивает ресурс кисти значению NULL, то не следует пытаться использовать кисть до тех пор, пока ей не будет назначен новый допустимый ресурс.
