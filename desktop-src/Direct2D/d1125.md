# <a name="d1125-dxgi-format-mismatch"></a>D1125: несоответствие формата DXGI

Формат пикселей, переданный этому API, несовместим с форматом пикселей \[ поверхности \] .

## <a name="possible-causes"></a>Возможные причины

Для таких операций, как ID2D1RenderTarget:: Креатебитмапфромвикбитмап или ID2D1RenderTarget:: Креатедксгисурфацерендертаржет, требуется поверхность и указанный формат поверхности. Direct2D проверяет, совместим ли указанный тип поверхности с поверхностью. Если он несовместим, Direct2D выводит это сообщение.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что указанный \_ Формат DXGI соответствует типу базовой поверхности или совместим с. Для ресурсов поверхности Direct3D, созданных приложением, используйте тот же формат DXGI, который \_ использовался для создания ресурса. Дополнительные рекомендации см. в разделе [Поддерживаемые форматы пикселей и альфа-режимы](supported-pixel-formats-and-alpha-modes.md) .
