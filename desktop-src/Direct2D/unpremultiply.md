# <a name="unpremultiply-effect"></a>Унпремултипли, результат

Используйте этот результат для преобразования изображения из предварительно умноженного альфа-канала в унпремултиплиед Alpha. Этот результат заменяет каждый входной пиксель `P = {R, G, B, A}` `P' = {R/A, G/A, B/A, A}` на в выходных данных.

У этого действия нет свойств.

Идентификатором CLSID для этого действия является CLSID \_ D2D1Unpremultiply.

## <a name="output-bitmap"></a>Битовая карта вывода

Размер выходного растрового изображения совпадает с размером входного растрового изображения.
