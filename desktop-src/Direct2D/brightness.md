# <a name="brightness-effect"></a>Эффекты яркости

Используйте эффекты яркости для управления яркостью изображения.

Идентификатором CLSID для этого действия является CLSID \_ D2D1Brightness.

-   [Пример изображения](#example-image)
-   [Свойства эффектов](#effect-properties)
-   [Битовая карта вывода](#output-bitmap)
-   [Requirements](#requirements)
-   [Связанные темы](#related-topics)

## <a name="example-image"></a>Пример изображения



| До                                                      |
|-------------------------------------------------------------|
| ![изображение перед результатом.](images/default-before.jpg)  |
| После                                                       |
| ![изображение после преобразования.](images/34-brightness.png) |



 


```C++
ComPtr<ID2D1Effect> brightnessEffect;
m_d2dContext->CreateEffect(CLSID_D2D1Brightness, &brightnessEffect);

brightnessEffect->SetValue(D2D1_BRIGHTNESS_PROP_BLACK_POINT, D2D1::Vector2F(0.0f, 0.2f));

m_d2dContext->BeginDraw();
m_d2dContext->DrawImage(brightnessEffect.Get());
m_d2dContext->EndDraw();
```



## <a name="effect-properties"></a>Свойства эффектов



| Отображаемое имя свойства                                                 | Тип и значение по умолчанию                              | Описание                                                                                                                                                                                                                                                                 |
|-----------------------------------------------------------------------|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| вхитепоинт<br/> \_ \_ \_ Белая \_ точка prop для D2D1 яркости<br/> | D2D1 \_ vector \_<br/> {1.0 f, 1.0 f}<br/> | Верхняя часть кривой перемещения яркости. Белая точка настраивает внешний вид более ярких частей изображения. Это свойство относится как к значению x, так и к значению y в этом порядке. Каждое из значений этого свойства находится в диапазоне от 0 до 1 включительно. |
| блаккпоинт<br/> \_ \_ \_ Черная \_ точка Prop яркости D2D1<br/> | D2D1 \_ vector \_<br/> {0.0 f, 0,0 f}<br/> | Нижняя часть кривой перемещения яркости. Черная точка настраивает внешний вид более темных частей изображения. Это свойство относится как к значению x, так и к значению y в этом порядке. Каждое из значений этого свойства находится в диапазоне от 0 до 1 включительно.   |



 

Этот результат использует указанные белые и черные точки для создания функции перемещения, используемой для корректировки точечного рисунка. В следующем уравнении описывается функция пересылки. Входные интенситиес определяются между 0 и 1.

![алгоритм яркости](images/brightness-formula1.png)

Алгоритм эффектов реализует уравнение, создающее функцию пересылки. Мы используем эту функцию для корректировки пикселов изображения. Значения x и y черной точки и белой точки являются координатами в двух измерениях, Соединенных для преобразования. Каждая часть окончательного выражения вывода:

1.  Преобразует данные изображения из линейного пространства в нелинейное, используя это уравнение:![вспомогательная функция 1](images/brightness-formula2.png)

2.  Корректирует изображение в соответствии со следующими значениями:
    -   *входные данные* — это значения интенсивности точечного рисунка от 0 до 1.

    -   *Белая пт (x, y)* расположение кривой преобразования для более светлой пиксельной интенситиес.

    -   *Черный PT (x, y)* — это расположение кривой преобразования для интенситиес пикселов DIMM.

3.  Преобразует данные изображения обратно в линейное пространство с помощью этого уравнения: ![вспомогательная функция 2](images/brightness-formula3.png)

Здесь показаны окончательное выходное уравнение и части компонентов.
