# <a name="d1144-effect-is-not-a-target"></a>D1144: воздействие не является целевым объектом

Сбой Сеттаржет, так как в качестве целевого объекта был передан результат.






 

## <a name="possible-causes"></a>Возможные причины

ID2D1DeviceContext:: Сеттаржет принимает параметр ID2D1Image \* . Изображения Direct2D могут быть точечными рисунками, списками команд или эффектами. Direct2D считает только точечные рисунки и списки команд допустимыми целевыми объектами.

## <a name="possible-fixes"></a>Возможные исправления

Измените приложение так, чтобы оно использовало точечный рисунок или список команд в качестве целевого объекта контекста устройства. Direct2D не выполняет инициализацию эффектов с помощью Сеттаржет. Вместо этого, чтобы выбрать содержимое воздействия, приложение создает результат с помощью ID2D1DeviceContext:: Креатиффект и использует соответствующий идентификатор GUID соответствующего действия.
