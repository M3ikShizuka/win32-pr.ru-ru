# <a name="d1128-dxgi-format-not-supported-in-any-api"></a>D1128: формат DXGI не поддерживается ни в одном API

Формат DXGI, переданный в этот API, не поддерживается Direct2D.



|             |         |
|-------------|---------|
| Уровень ошибки | Предупреждение |



 

## <a name="possible-causes"></a>Возможные причины

Некоторые форматы Surface несовместимы с Direct2D операциями независимо от того, как они используются. Это сообщение означает, что приложение должно выбрать другой формат поверхности.

## <a name="possible-fixes"></a>Возможные исправления

См. Дополнительные сведения о поддерживаемых форматах [пикселей и альфа-режимах](supported-pixel-formats-and-alpha-modes.md) , касающихся поддерживаемых форматов. Если приложение должно быть представлено в одном канале и пытается использовать неподдерживаемый формат с красным каналом, рассмотрите возможность изменения его для отображения в формате альфа-канала.
