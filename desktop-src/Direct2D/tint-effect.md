# <a name="tint-effect"></a>Оттенок

Это приведет к оттенок исходного изображения путем умножения исходного изображения на указанный цвет. Он имеет один вход.

Идентификатором CLSID для этого действия является CLSID \_ D2D1Tint.

## <a name="effect-properties"></a>Свойства эффектов



| Отображаемое имя и перечисление индекса                    | Тип и значение по умолчанию                              | Описание                                                      |
|-------------------------------------------------------|-----------------------------------------------------|------------------------------------------------------------------|
| \_Цвет ColorD2D1 оттенка \_ \_ цвета<br/>               | D2D1 \_ vector \_ 4F (1.0 f, 1.0 f, 1.0 f, 1.0 f)<br/> | Цвета из исходного изображения умножаются на это значение.       |
| \_ \_ \_ Выходные данные среза ClampOutputD2D1 оттенка \_<br/> | булфалсе<br/>                                | Указывает, следует ли отменять выходные значения до диапазона \[ 0, 1 \] . |
