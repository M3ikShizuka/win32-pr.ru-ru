# <a name="d1131-dxgi-format-not-gdi-compatible"></a>D1131: формат DXGI не совместим с GDI

В этот API был передан параметр, совместимый с GDI, но формат DXGI не совместим с GDI.






 

## <a name="possible-causes"></a>Возможные причины

Использование \_ параметра D2D1 Bitmap \_ Параметры, \_ \_ совместимые с GDI во время создания растрового изображения, обеспечивает совместимость битовой карты с визуализацией GDI, но вводит дополнительные ограничения на формат поверхности точечного рисунка. Как правило, Визуализация GDI совместима с \_ форматом DXGI \_ B8G8R8A8 \_ UNORM и представлением DXGI \_ \_ B8G8R8A8 \_ UNORM \_ sRGB.

## <a name="possible-fixes"></a>Возможные исправления

Чтобы устранить эту проблему, убедитесь, что формат, заданный для точечного рисунка, совместимого с GDI, имеет формат BGRA из четырех каналов, как указано выше. Если точечный рисунок не требует взаимодействия с GDI, опустите \_ совместимый с GDI спецификатор.
