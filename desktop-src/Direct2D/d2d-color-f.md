# <a name="d2d_color_f"></a>\_Цветовая \_ F D2D

Описывает красный, зеленый, синий и альфа-компоненты цвета.


```C++
typedef D3DCOLORVALUE D2D_COLOR_F;
```

Дополнительные сведения см. в разделе [**D3DCOLORVALUE**](/previous-versions/windows/desktop/legacy/dd368193(v=vs.85)).
