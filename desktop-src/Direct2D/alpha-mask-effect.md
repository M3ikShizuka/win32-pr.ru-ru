# <a name="alpha-mask-effect"></a>Воздействие альфа-маски

Этот результат применяет альфа-маску к изображению. Он имеет два входа, называемые назначением и маской. Цветовые значения в целевом изображении умножаются на альфа-канал изображения маски.

Идентификатором CLSID для этого действия является CLSID \_ D2D1AlphaMask.

## <a name="effect-properties"></a>Свойства эффектов

Этот результат не имеет свойств, относящихся к определенному результату.
