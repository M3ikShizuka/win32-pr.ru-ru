# <a name="d1146-invalid-render-controls"></a>D1146: недопустимые элементы управления прорисовкой

Указанные элементы управления отрисовки имели недопустимые параметры и будут проигнорированы.






 

## <a name="possible-causes"></a>Возможные причины

Это сообщение может появиться во время ID2D1DeviceContext:: Сетрендерингконтролс, если приложение указывает недопустимое значение точности буфера или нулевой размер плитки.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что приложение использует допустимое значение точности буфера. См. раздел [**\_ \_ точность буфера D2D1**](/windows/desktop/api/D2d1_1/ne-d2d1_1-d2d1_buffer_precision). Убедитесь, что приложение указывает ненулевой размер плитки.
