# <a name="d1250-inaccessible-source"></a>D1250: недоступный источник

Обнаружена попытка рисования из недоступного точечного рисунка.






 

## <a name="possible-causes"></a>Возможные причины

Обнаружена попытка прорисовки из недоступного источника. Одна из причин того, что источник может быть инацессибле, если он был создан из общей текстуры, а текстура не была получена.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что все точечные рисунки доступны, прежде чем рисовать их с помощью Direct2D.
