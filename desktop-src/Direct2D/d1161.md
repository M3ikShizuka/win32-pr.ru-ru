# <a name="d1161-copyfrommemory-requires-cpu-readable-bitmap"></a>D1161: для Копифроммемори требуется доступное для чтения растровое изображение ЦП

Этот метод не разрешен для точечного рисунка, созданного с помощью [**\_ параметров точечного рисунка D2D1, \_ \_ \_ считанных ЦП**](/windows/desktop/api/D2d1_1/ne-d2d1_1-d2d1_bitmap_options).






 

## <a name="possible-causes"></a>Возможные причины

Это сообщение может появиться при вызове [**ID2D1Bitmap:: копифроммемори**](/windows/win32/api/d2d1/nf-d2d1-id2d1bitmap-copyfrommemory). Если точечный рисунок был создан с \_ помощью \_ параметра D2D1 Bitmap \_ \_ Reread, Direct2D сообщит об ошибке, и на отладочном уровне будет выведено это сообщение.

## <a name="possible-fixes"></a>Возможные исправления

Если необходимо, чтобы приложение вызывало [**ID2D1Bitmap:: копифроммемори**](/windows/win32/api/d2d1/nf-d2d1-id2d1bitmap-copyfrommemory), убедитесь, что точечный рисунок не создан как доступный для чтения ЦП. Если невозможно изменить атрибуты поверхности точечного рисунка, подумайте, есть ли эквивалентная операция отрисовки для требуемого результата.
