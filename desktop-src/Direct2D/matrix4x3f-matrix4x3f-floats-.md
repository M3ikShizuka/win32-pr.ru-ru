# <a name="matrix4x3fmatrix4x3ffloat-float-float-float-float-float-float-float-float-float-float-float-constructor"></a>Конструктор Matrix4x3F:: Matrix4x3F (FLOAT, FLOAT, FLOAT, FLOAT, float, FLOAT, FLOAT, FLOAT, float, float, float, float)

Создает новый экземпляр класса [**Matrix4x3F**](matrix4x3f.md) , инициализируемый всеми значениями матрицы с плавающей запятой.

## <a name="syntax"></a>Синтаксис


```C++
inline Matrix4x3F(
   FLOAT m11,
   FLOAT m12,
   FLOAT m13,
   FLOAT m21,
   FLOAT m22,
   FLOAT m23,
   FLOAT m31,
   FLOAT m32,
   FLOAT m33,
   FLOAT m41,
   FLOAT m42,
   FLOAT m43
);
```
