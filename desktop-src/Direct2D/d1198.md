# <a name="d1198-copy-to-read-only-bitmap"></a>D1198: копирование в Read-Only точечный рисунок

Предпринята попытка копирования данных в точечный рисунок, доступный только для чтения или неизменяемый.






 

## <a name="possible-causes"></a>Возможные причины

Предпринята попытка скопировать данные в точечный рисунок, доступный только для чтения или неизменяемый.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что все точечные рисунки, записанные в, не создаются как неизменяемые.
