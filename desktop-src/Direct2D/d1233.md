# <a name="d1233-begindraw-invalid-from-previous-error"></a>D1233: недопустимый Бегиндрав из предыдущей ошибки

Бегиндрав был вызван, но будет проигнорирован, так как контекст устройства находится в недопустимом состоянии.






 

## <a name="possible-causes"></a>Возможные причины

Приложение с именем Бегиндрав, но контекст устройства находится в недопустимом состоянии из-за предыдущей операции.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что приложение отвечает соответствующим ошибкам, возвращаемым сбросом или EndDraw, при необходимости воссоздание устройства. Если команда привела к тому, что контекст устройства находится в недопустимом состоянии, то соответствующий сброс или EndDraw возвратит ошибку.
