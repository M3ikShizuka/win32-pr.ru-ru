# <a name="d1116-unexpected-hardware-creation-error"></a>D1116: непредвиденная ошибка создания оборудования

\[  \] При попытке создать целевой объект Direct3D обнаружен непредвиденный код ошибки.

</dd> </dl>

## <a name="possible-causes"></a>Возможные причины

Это сообщение появляется, когда Direct2D не удалось создать аппаратный целевой ресурс с параметрами, указанными в приложении. Вероятная причина заключается в том, что приложение запросило аппаратную цель, в результате которой Direct2D должен стать владельцем распределения целевой текстуры, например с помощью [**ID2D1Factory:: креатехвндрендертаржет**](/windows/win32/api/d2d1/nf-d2d1-id2d1factory-createhwndrendertarget(constd2d1_render_target_properties__constd2d1_hwnd_render_target_properties__id2d1hwndrendertarget)), а запрошенный размер был слишком большим. Каждый уровень функций Direct3D накладывает ограничение на размер текстуры. См. раздел [уровни функций Direct3D](/previous-versions/windows/apps/hh994923(v=win.10)).

## <a name="possible-fixes"></a>Возможные исправления

При выделении целевого объекта убедитесь, что размер целевого объекта не превышает максимальный размер, допустимый уровнем компонента. Для более точного устранения неполадок выделения целевых объектов рассмотрите возможность изменения приложения, чтобы выделить целевую текстуру, и убедитесь, что выделение выполнено успешно. Чтобы непосредственно использовать эту текстуру в качестве целевого объекта, ее можно использовать как объект [**ID2D1Bitmap**](/windows/win32/api/d2d1/nn-d2d1-id2d1bitmap) , а в качестве аргумента — в [**ID2D1DeviceContext:: сеттаржет**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1devicecontext-settarget)— **ID2D1Bitmap** .
