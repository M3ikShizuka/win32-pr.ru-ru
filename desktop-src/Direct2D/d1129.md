# <a name="d1129-dxgi-format-incompatible-with-alpha-mode"></a>D1129: формат DXGI несовместим с режимом альфа

Сочетание режима альфа и формата DXGI несовместимо друг с другом.





 

## <a name="possible-causes"></a>Возможные причины

Для операций Direct2D, использующих формат поверхности и альфа-режим, бывают случаи, когда формат поверхности и альфа-режим не согласуются. Например, формат поверхности, такой как формат DXGI \_ \_ a8 \_ UNORM, не имеет СМЫСЛА использовать с Ignore альфа, так как он отображается только в альфа-канале. Кроме того, прямая альфа-версия не поддерживается в некоторых \_ форматах DXGI, но предварительно умноженный альфа-канал —.

## <a name="possible-fixes"></a>Возможные исправления

См. раздел [Поддерживаемые форматы пикселей и альфа-режимы](supported-pixel-formats-and-alpha-modes.md) для указания того, какие сочетания форматов поддерживаются, а какие нет. Измените приложение, чтобы использовать поддерживаемое сочетание форматов.
