# <a name="d2d1_matrix_4x3_f"></a>D2D1 \_ Матрица \_ 4X3 \_ F

Представляет матрицу 4 на 3.


```C++
typedef D2D_MATRIX_4X3_F D2D1_MATRIX_4X3_F;
```



## <a name="remarks"></a>Remarks

**D2D1 \_ MATRIX \_ 4X3** — это новое имя для структуры [**\_ матрицы D2D \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f) . Список полей, предоставляемых матрицей, см. в [**разделе \_ Таблица D2D \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f).
