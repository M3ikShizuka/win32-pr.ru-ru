# <a name="matrix4x3f-class"></a>Класс Matrix4x3F

Класс **Matrix4x3F** представляет матрицу 4 на 3 и предоставляет удобные методы для создания матриц.

## <a name="members"></a>Элементы

Класс **Matrix4x3F** наследует от [**D2D1 \_ Matrix \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f). **Matrix4x3F** также имеет следующие типы членов:

-   [Конструкторы](#constructors)

### <a name="constructors"></a>Конструкторы

Класс **Matrix4x3F** содержит эти конструкторы.



| Конструктор                                                                                                                                                                                           | Описание                                                                                                                        |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**Matrix4x3F()**](matrix4x3f-matrix4x3f--.md)                                                                                                                                                       | Создает новый экземпляр неинициализированного класса **Matrix4x3F** .<br/>                                                   |
| [**Matrix4x3F (FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, float, float, float, float) (FLOAT, FLOAT, float, float, FLOAT, FLOAT, float, float, FLOAT, FLOAT, float, float)**](matrix4x3f-matrix4x3f-floats-.md) | Создает новый экземпляр класса **Matrix4x3F** , инициализируемый всеми значениями матрицы с плавающей запятой.<br/> |
