# <a name="d1009-unexpected-hardware-creation-error"></a>D1009: непредвиденная ошибка создания оборудования

\[  \] При попытке создать целевой объект Direct3D произошла непредвиденная ошибка.

Номер ошибки.

|-------------|---------|
| Уровень ошибки | Предупреждение |



 

## <a name="possible-causes"></a>Возможные причины

-   Целевой объект был больше максимального размера битовой карты.
-   Недостаточно видеопамяти при создании целевого объекта рендеринга.
-   Не удалось создать устройство Direct3D.
-   Приложение работает на платформе IA64.
