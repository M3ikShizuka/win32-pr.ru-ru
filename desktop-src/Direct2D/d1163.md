# <a name="d1163-print-bitmap-too-big"></a>D1163: слишком большой точечный рисунок

Точечный рисунок слишком велик для печати, его можно разбить на мелкие плитки.






 

## <a name="possible-causes"></a>Возможные причины

Direct2Dная печать накладывает ограничение на размеры печатных рисунков. Это ограничение составляет 65535 единиц, зависящих от устройства. Если в приложении имеется точечный рисунок, размер которого превышает указанный в печатном содержимом, Direct2D сообщит об ошибке, а отладочный слой выдаст это сообщение.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что точечные рисунки, используемые в операциях печати, не превышают предельный размер Direct2D печати. Рассмотрите возможность секционирования точечного рисунка на несколько плиток.
