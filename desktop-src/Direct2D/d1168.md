# <a name="d1168-write-map-option-requires-cpu-writable-bitmap"></a>D1168: для параметра схемы записи требуется точечный рисунок, доступный для записи ЦП

\_ \_ Возможность записи параметров карты D2D1 \_ допустима только для точечного рисунка ЦП с возможностью записи






 

## <a name="possible-causes"></a>Возможные причины

Direct2D может выводить это сообщение, если приложение вызывает ID2D1Bitmap1:: Map с \_ \_ атрибутом D2D1 Map \_ Write, но точечный рисунок не поддерживает запись ЦП.

## <a name="possible-fixes"></a>Возможные исправления

Если приложению требуется доступ на запись с помощью ID2D1Bitmap1:: Map, убедитесь, что в базовом ресурсе Direct3D есть атрибут D3D11 " \_ доступ на запись ЦП" \_ \_ .
