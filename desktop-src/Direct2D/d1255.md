# <a name="d1255-tile-opacity-metadata-conflict"></a>D1255: конфликт метаданных непрозрачности плитки

Непрозрачный прямоугольник, вычисляемый для графа эффектов, перекрытой областью, помеченной как прозрачная. Это может произойти из-за неправильного использования преобразования коррекции в Atlas или границ.
