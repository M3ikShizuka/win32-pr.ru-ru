# <a name="d1176-negative-dpi"></a>D1176: отрицательное значение DPI

При создании рендертаржет возникает ошибка при указании отрицательного значения DPI.






 

## <a name="possible-causes"></a>Возможные причины

Приложение попыталось указать отрицательное значение DPI либо с помощью ID2D1RenderTarget:: Сетдпи, либо путем создания целевого объекта со свойством отрицательного DPI.

## <a name="possible-fixes"></a>Возможные исправления

При создании или изменении целевых объектов рендеринга выберите допустимые неотрицательные значения DPI. Если значение DPI по умолчанию приемлемо, не следует вызывать Сетдпи или указывать параметр DPI при создании целевого объекта. Указание значения DPI 0 эквивалентно использованию значения DPI по умолчанию.
