# <a name="d1239-source-transform-bitmap-invalid-call"></a>D1239: недопустимый вызов исходного растрового изображения преобразования

Битовая карта, передаваемая в ID2D1SourceTransform::D RAW, не поддерживает этот метод.






 

## <a name="possible-causes"></a>Возможные причины

Приложение вызвало неподдерживаемый метод для точечного рисунка, переданного в ID2D1SourceTransform::D RAW.

## <a name="possible-fixes"></a>Возможные исправления

Чтобы обновить этот тип точечного рисунка, используйте ID2D1Bitmap1:: Map.
