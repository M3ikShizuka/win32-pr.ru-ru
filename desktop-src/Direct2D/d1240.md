# <a name="d1240-source-tranform-still-mapped"></a>D1240: исходный преобразование по-прежнему сопоставлен

Битовая карта, передаваемая в ID2D1SourceTransform::D RAW, не была отменена.






 

## <a name="possible-causes"></a>Возможные причины

Приложение с именем ID2D1Bitmap1:: Map не вызывало ID2D1Bitmap1:: uncall для точечного рисунка, переданного ID2D1SourceTransform::D RAW.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что метод сопоставления вызывается всякий раз, когда вызывается сопоставление.
