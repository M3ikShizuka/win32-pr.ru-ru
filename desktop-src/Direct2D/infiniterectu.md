# <a name="infiniterectu-function"></a>Функция Инфинитеректу

Создает прямоугольник, для левого верхнего угла которого задано значение (0U, 0U), а в нижнем правом углу — значение (бесконечность, бесконечность).

## <a name="syntax"></a>Синтаксис


```C++
D2D1_RECT_U InfiniteRectU(void);
```



## <a name="parameters"></a>Параметры

У этой функции нет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **D2D1 \_ Rect \_ U**](d2d1-rect-u.md)**

Новый прямоугольник.
