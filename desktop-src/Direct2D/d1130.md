# <a name="d1130-dxgi-format-disallowed-by-caps"></a>D1130: формат DXGI запрещен прописными буквами

У устройства нет возможности поддерживать заданный формат DXGI.






 

## <a name="possible-causes"></a>Возможные причины

Для операций Direct2D, которые выполняются на оборудовании и используют формат поверхности, формат поверхности должен поддерживаться на уровне компонентов оборудования. Например, если оборудование имеет уровень функций 9, он может не поддерживать форматы однобуферной или SRGB, такие как \_ Формат DXGI \_ B8G8R8A8 \_ UNORM \_ sRGB.

## <a name="possible-fixes"></a>Возможные исправления

Рекомендации по различиям между уровнями функций и поддерживаемыми форматами см. в разделе [уровни функций Direct3D](/previous-versions/windows/apps/hh994923(v=win.10)) . Рассмотрите возможность использования [**метода ID3D11Device:: чеккформатсуппорт**](/windows/desktop/api/d3d11/nf-d3d11-id3d11device-checkformatsupport) , чтобы убедиться, что требуемый формат поддерживается оборудованием, прежде чем использовать его с Direct2D.
