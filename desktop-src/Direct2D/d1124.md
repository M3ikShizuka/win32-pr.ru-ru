# <a name="d1124-dxgi-format-unknown-with-null-target-bitmap"></a>D1124: неизвестный формат DXGI с конечным точечным рисунком null

\_Неизвестный формат DXGI \_ не допускается в качестве параметра для этого API, если целевой объект имеет значение null или является списком команд.






 

## <a name="possible-causes"></a>Возможные причины

Для таких операций, как ID2D1RenderTarget:: Креатекомпатиблерендертаржет, целевому объекту отрисовки источника должно быть назначено целевое растровое изображение. Без целевого растрового изображения Direct2D не может определить, какой формат поверхности будет иметь новый ресурс целевого объекта прорисовки.

## <a name="possible-fixes"></a>Возможные исправления

Чтобы решить эту эту задачи, убедитесь, что соответствующий ID2D1DeviceContext имеет целевой точечный рисунок, назначенный с помощью ID2D1DeviceContext:: Сеттаржет. Или, если приложению просто требуется возможность отображения на двух разных целевых поверхностях, рассмотрите возможность создания ID2D1Bitmap и задания его в качестве целевого объекта контекста устройства вместо использования ID2D1RenderTarget:: Креатекомпатиблерендертаржет.
