# <a name="saturation-effect"></a>Воздействие на насыщенность

Используйте этот результат, чтобы изменить насыщенность изображения. Воздействие на насыщенность — это специализация [цветовой матрицы](color-matrix.md) .

Идентификатором CLSID для этого действия является CLSID \_ D2D1Saturation.

-   [Пример изображения](#example-image)
-   [Свойства эффектов](#effect-properties)
-   [Требования](#requirements)
-   [Связанные темы](#related-topics)

## <a name="example-image"></a>Пример изображения

В этом примере показаны входные и выходные изображения для эффектов насыщенности с насыщенностью 0%.



| До                                                      |
|-------------------------------------------------------------|
| ![изображение перед результатом.](images/default-before.jpg)  |
| После                                                       |
| ![изображение после преобразования.](images/16-saturation.png) |



 


```C++
ComPtr<ID2D1Effect> saturationEffect;
m_d2dContext->CreateEffect(CLSID_D2D1Saturation, &saturationEffect);

saturationEffect->SetInput(0, bitmap);

saturationEffect->SetValue(D2D1_SATURATION_PROP_SATURATION, 0.0f);

m_d2dContext->BeginDraw();
m_d2dContext->DrawImage(saturationEffect.Get());
m_d2dContext->EndDraw();
```



Этот результат вычисляет цветовую матрицу на основе значения насыщенности *в* уравнении здесь, которое указывается с помощью \_ \_ \_ Свойства насыщенности D2D1 насыщенности. Уравнение матрицы показано здесь.

![Формула для вычисления матрицы насыщенности.](images/saturation-formula.png)

Созданная матрица зависит только от значения насыщенности. Если требуется определенная матрица, можно использовать [цветовую матрицу](color-matrix.md) .

Этот результат использует и выводит предварительно умноженные альфа-изображения. Этот результат не будет работать с прямыми альфа-изображениями, если они не полностью непрозрачны.

## <a name="effect-properties"></a>Свойства эффектов



| Отображаемое имя и перечисление индекса                                  | Тип и значение по умолчанию           | Описание                                                                                                                                                                                                                      |
|---------------------------------------------------------------------|----------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Насыщенность<br/> \_Насыщенность D2D1 насыщенности перегрузки \_ \_<br/> | FLOAT<br/> 0,5 f<br/> | Насыщенность изображения. Можно задать для насыщенности значение от 0 до 1. Если задать для него значение 1, выходное изображение будет полностью насыщенным. Если задать для него значение 0, выходное изображение будет монохромным. Значение насыщенности является бесмодульным. |
