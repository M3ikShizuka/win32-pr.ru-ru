# <a name="d2d1_matrix_5x4_f"></a>D2D1 \_ Матрица \_ 5x4 \_ F

Представляет матрицу с 5 по 4.


```C++
typedef D2D_MATRIX_5X4_F D2D1_MATRIX_5X4_F;
```



## <a name="remarks"></a>Комментарии

**D2D1 \_ MATRIX \_ 5x4** — это новое имя для структуры [**\_ матрицы D2D \_ 5x4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f) . Список полей, предоставляемых матрицей, см. в [**разделе \_ Таблица D2D \_ 5x4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f).
