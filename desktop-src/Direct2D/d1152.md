# <a name="d1152-command-list-closed-twice"></a>D1152: список команд закрыт дважды

Предпринята попытка закрыть список команд за второй раз.






 

## <a name="possible-causes"></a>Возможные причины

После заполнения списка команд его необходимо закрыть только один раз. Direct2D учитывает последующие вызовы метода ID2D1CommandList:: Close, чтобы стать ошибкой.

## <a name="possible-fixes"></a>Возможные исправления

Чтобы избежать этого сообщения, убедитесь, что список команд закрыт только один раз. Если приложение использует обратные вызовы или пути рекурсивного кода для взаимодействия с списками команд, убедитесь в правильности его поведения.
