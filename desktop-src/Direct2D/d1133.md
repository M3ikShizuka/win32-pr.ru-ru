# <a name="d1133-buffer-precision-unknown-not-allowed"></a>D1133: неизвестная точность буфера

\_ \_ Неизвестная точность буфера D2D1 \_ не допускается в качестве параметра для этого API.






 

## <a name="possible-causes"></a>Возможные причины

Direct2D позволяет приложению указать требуемую точность буфера для некоторых операций. Например, точность, используемая для конвейера создания образов, указывается с помощью ID2D1DeviceContext:: Сетрендерингконтролс. Неизвестный \_ \_ определитель точности буфера позволяет приложению превышать этот параметр, и Direct2D будет использовать значение по умолчанию, но этот спецификатор не разрешен для некоторых операций. В частности, он может быть не разрешен во время создания коллекции ограничителей градиента.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что \_ \_ неизвестный ОПИСАТЕЛЬ точности буфера не используется с ID2D1DeviceContext:: креатеградиентстопколлектион. Используйте известное значение точности; Рассмотрите возможность использования точности целевого формата поверхности для операции.
