# <a name="d1196-a8-shared-bitmap"></a>D1196: общий точечный рисунок A8

Поверхность DXGI формата DXGI \_ \_ a8 \_ UNORM не поддерживается в этом API.





 

## <a name="possible-causes"></a>Возможные причины

\_Формат DXGI \_ a8 \_ UNORM не поддерживается в качестве формата точечного рисунка взаимодействия DXGI, даже если он поддерживается в других API-интерфейсах.

## <a name="possible-fixes"></a>Возможные исправления

Создание точечного рисунка взаимодействия DXGI в другом формате.
