# <a name="d1182-operation-on-unclosed-path-geometry"></a>D1182: операция над незамкнутой геометрической траекторией

Произошла операция над геометрией пути, которая не была закрыта.






 

## <a name="possible-causes"></a>Возможные причины

Приложение начала заполнение ID2D1PathGeometry, но не вызывало ID2D1SimplifiedGeometrySink:: Close в своем приемнике. Или приложение вызвало Close, но было возвращено значение ошибки.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что все геометрические пути заполнены правильно, вызвав Close в своем приемнике и убедившись, что Close не возвращал ошибку. Пример создания и заполнения объекта Geometry пути см. в справочнике по ID2D1PathGeometry.
