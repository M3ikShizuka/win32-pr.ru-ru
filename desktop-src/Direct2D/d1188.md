# <a name="d1188-cpu-read-and-target-bitmap-options-are-incompatible"></a>D1188: параметры чтения и целевого растрового изображения несовместимы

\_Параметры точечного рисунка D2D1 \_ \_ \_ не совместимы с \_ параметрами битовой карты D2D1 \_ \_ .






 

## <a name="possible-causes"></a>Возможные причины

В приложении указано несовместимое сочетание \_ флагов параметров точечного рисунка D2D1 \_ .

## <a name="possible-fixes"></a>Возможные исправления

Удалите \_ Параметры точечного рисунка D2D1 \_ \_ Загрузка ЦП \_ или D2D1 \_ точечные рисунки \_ \_ целевого объекта из флагов. Если приложению нужны оба варианта, необходимо создать два отдельных точечных рисунка.
