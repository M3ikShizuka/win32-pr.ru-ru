# <a name="direct2d-glossary"></a>Глоссарий Direct2D

<dl> <dt>

<span id="direct2d.direct2d_glossary_a8_target"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_A8_TARGET"></span>**Целевой объект A8**
</dt> <dd>

Целевой объект отрисовки, использующий формат пикселей, представляющий альфа-канал с восемью битами. Целевые объекты A8 полезны для рисования геометрических объектов в виде масок.

</dd> <dt>

<span id="direct2d.direct2d_glossary_aliasing"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_ALIASING"></span>**сглаживания**
</dt> <dd>

В компьютерной графике неровный вид кривых или диагональных линий. Псевдонимы наиболее видны при более низких разрешениях.

</dd> <dt>

<span id="direct2d.direct2d_glossary_antialiasing"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_ANTIALIASING"></span>**сглаживания**
</dt> <dd>

Методика сглаживания зубчатого внешнего вида кривых или диагональных линий. Методы сглаживания включают окружающие пикселы с промежуточными оттенками и управление размером и горизонтальным выравниванием пикселов.

</dd> <dt>

<span id="direct2d.direct2d_glossary_batch"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BATCH"></span>**разделах**
</dt> <dd>

Набор сгруппированных запросов или транзакций.

</dd> <dt>

<span id="direct2d.direct2d_glossary_bitmap"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BITMAP"></span>**массив**
</dt> <dd>

Представление символов или графиков по отдельным пикселям. Пиксели могут быть упорядочены по горизонтали, в строках или по вертикали в столбцах. Каждый пиксель может быть представлен одним или несколькими битами.

</dd> <dt>

<span id="direct2d.direct2d_glossary_bits_per_pixel__bpp_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BITS_PER_PIXEL__BPP_"></span>**бит на пиксель (в бит)**
</dt> <dd>

Число битов, используемых для хранения и вывода цветовых данных для одного пикселя. Это стандартная единица измерения для глубины бита (также называемая глубиной цвета). К общим глубинам бит относятся 8, 16 и 32.

</dd> <dt>

<span id="direct2d.direct2d_glossary_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_BRUSH"></span>**Brush**
</dt> <dd>

Ресурс Direct2D, который закрашивает бесконечную плоскость сплошным цветом, градиентом или точечным рисунком. Кисть обычно используется для обводки и заливки геометрии.

</dd> <dt>

<span id="direct2d.direct2d_glossary_cleartype"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_CLEARTYPE"></span>**Шрифтов**
</dt> <dd>

Технология отображения шрифтов, значительно улучшающая разрешение экрана шрифта, так как буквы на экране компьютера отображаются гладкими, а не зубчатыми. Технология ClearType улучшает внешний вид текста на ЖК-мониторах с цифровым интерфейсом, например мониторами ноутбука и высококачественными мониторами с плоскими панелями.

</dd> <dt>

<span id="direct2d.direct2d_glossary_dc"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DC"></span>**Постоянный**
</dt> <dd>

См. определение для: контекст устройства.

</dd> <dt>

<span id="direct2d.direct2d_glossary_device_context__dc_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DEVICE_CONTEXT__DC_"></span>**контекст устройства (DC)**
</dt> <dd>

Контекст устройства GDI. См. также: интерфейс графических устройств.

</dd> <dt>

<span id="direct2d.direct2d_glossary_direct2d"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECT2D"></span>**Direct2D**
</dt> <dd>

API-интерфейс рендеринга с аппаратным ускорением и режимом 2-D.

</dd> <dt>

<span id="direct2d.direct2d_glossary_direct3d"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECT3D"></span>**Direct3D**
</dt> <dd>

Платформа аппаратного ускорения и среда выполнения для трехмерной графики в Windows.

</dd> <dt>

<span id="direct2d.direct2d_glossary_directwrite"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECTWRITE"></span>**DirectWrite**
</dt> <dd>

Текстовый API, обеспечивающий отрисовку текста, управление шрифтами и поддержку макета текста, который не зависит от конкретной системы отрисовки.

</dd> <dt>

<span id="direct2d.direct2d_glossary_directx_graphics_infrastructure__dxgi_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DIRECTX_GRAPHICS_INFRASTRUCTURE__DXGI_"></span>**Графическая инфраструктура DirectX (DXGI)**
</dt> <dd>

Инфраструктура, которая управляет задачами низкого уровня, связанными с графикой, которые не зависят от среды выполнения DirectX Graphics. DXGI предоставляет общую платформу для графических компонентов.

</dd> <dt>

<span id="direct2d.direct2d_glossary_dxgi"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_DXGI"></span>**DXGI**
</dt> <dd>

См. определение для: графическая инфраструктура DirectX (DXGI).

</dd> <dt>

<span id="direct2d.direct2d_glossary_figure"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_FIGURE"></span>**замкнут**
</dt> <dd>

Открытая или закрытая фигура, определяемая начальной точкой и коллекцией сегментов.

</dd> <dt>

<span id="direct2d.direct2d_glossary_flatten"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_FLATTEN"></span>**расположение**
</dt> <dd>

Для создания многоугольной аппроксимации (потенциально изогнутой) геометрии.

</dd> <dt>

<span id="direct2d.direct2d_glossary_gdi"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GDI"></span>**ИСПОЛЬЗОВАНИЕМ**
</dt> <dd>

См. определение для: интерфейс графических устройств (GDI).

</dd> <dt>

<span id="direct2d.direct2d_glossary_geometric_mask"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GEOMETRIC_MASK"></span>**геометрическая маска**
</dt> <dd>

Клип или фрагмент, определенный объектом ID2D1Geometry, который маскирует слой при прорисовке объектом прорисовки. Геометрическая маска может быть либо псевдонимом, либо иметь границы вложенных точек.

</dd> <dt>

<span id="direct2d.direct2d_glossary_geometry_transform"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GEOMETRY_TRANSFORM"></span>**геометрическое преобразование**
</dt> <dd>

Преобразование, которое применяется к геометрии перед обводки или заливкой.

</dd> <dt>

<span id="direct2d.direct2d_glossary_glyph"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GLYPH"></span>**изображения**
</dt> <dd>

Графическое представление символа, части символа или последовательности символов.

</dd> <dt>

<span id="direct2d.direct2d_glossary_glyph_run"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GLYPH_RUN"></span>**выполнение глифа**
</dt> <dd>

Непрерывный запуск глифов, имеющих общий формат.

</dd> <dt>

<span id="direct2d.direct2d_glossary_gradient_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRADIENT_BRUSH"></span>**Градиентная кисть**
</dt> <dd>

Кисть, которая закрашивает область постепенного продвижения от одного цвета к другому или от одного тени до другого оттенка того же цвета.

</dd> <dt>

<span id="direct2d.direct2d_glossary_gradient_stop"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRADIENT_STOP"></span>**прерывать градиент**
</dt> <dd>

Определенный цвет, определенный для расположения в пределах области градиента. Ограничители градиента используются для определения цвета в определенных местах вдоль пути градиента.

</dd> <dt>

<span id="direct2d.direct2d_glossary_grahics_primitive"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRAHICS_PRIMITIVE"></span>**примитив грахикс**
</dt> <dd>

В компьютерной графике это фигура, например линия, круг, кривая или многоугольник, который графический адаптер может рисовать, хранить и манипулировать как дискретная сущность.

</dd> <dt>

<span id="direct2d.direct2d_glossary_graphics_device_interface__gdi_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_GRAPHICS_DEVICE_INTERFACE__GDI_"></span>**Интерфейс графических устройств (GDI)**
</dt> <dd>

Графический API, позволяющий приложениям отсылать графические команды на устройство вывода или печати, как правило, без учета его характеристик или возможностей. Microsoft Win32 GDI получает запросы на визуализацию из приложений и передает эти запросы в GDI режима ядра.

</dd> <dt>

<span id="direct2d.direct2d_glossary_handle_to_a_device_context__hdc_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_HANDLE_TO_A_DEVICE_CONTEXT__HDC_"></span>**обработчик контекста устройства (HDC)**
</dt> <dd>

Ссылка на контекст устройства GDI.

</dd> <dt>

<span id="direct2d.direct2d_glossary_hdc"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_HDC"></span>**HDC**
</dt> <dd>

См. определение для: Handle в контексте устройства.

</dd> <dt>

<span id="direct2d.direct2d_glossary_mesh"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_MESH"></span>**сетчат**
</dt> <dd>

Коллекция вершин, определяющих набор треугольников, образующих фигуру.

</dd> <dt>

<span id="direct2d.direct2d_glossary_multi-sample_anti-aliasing__msaa_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_MULTI-SAMPLE_ANTI-ALIASING__MSAA_"></span>**Многопримерное сглаживание (MSAA)**
</dt> <dd>

Методика сглаживания, которая обрабатывает всю сцену.

</dd> <dt>

<span id="direct2d.direct2d_glossary_opacity_mask"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_OPACITY_MASK"></span>**Маска непрозрачности**
</dt> <dd>

Маска, описываемая кистью или точечным рисунком, которая применяется к другому объекту, чтобы сделать этот объект частично или полностью прозрачным. Маска непрозрачности использует сведения о альфа-канале для указания того, как исходные пикселы объекта смешиваются в конечном целевом объекте назначения. Прозрачные части маски указывают области, в которых базовое изображение скрыто, а непрозрачные части маски указывают, где разрешается маскированный объект.

</dd> <dt>

<span id="direct2d.direct2d_glossary_per_primitive_anti-aliasing__ppaa_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_PER_PRIMITIVE_ANTI-ALIASING__PPAA_"></span>**Для каждого примитивного сглаживания (ППАА)**
</dt> <dd>

Метод сглаживания, применяемый к каждому отдельному графическому примитиву, а не ко всей сцене.

</dd> <dt>

<span id="direct2d.direct2d_glossary_radial_gradient_brush"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_RADIAL_GRADIENT_BRUSH"></span>**кисть радиального градиента**
</dt> <dd>

Градиент, где начальная точка определяется фокальной точкой, а конечная точка определяется градиентом.

</dd> <dt>

<span id="direct2d.direct2d_glossary_render_target"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_RENDER_TARGET"></span>**Целевой объект прорисовки**
</dt> <dd>

Ресурс Direct2D, который выдает команды рисования. Существуют различные типы целевых объектов отрисовки, каждая из которых готовится к просмотру в другом месте назначения. Например, ID2D1HwndRenderTarget подготавливается к просмотру в окне, а ID2D1BitmapRenderTarget — к точечному рисунку.

</dd> <dt>

<span id="direct2d.direct2d_glossary_segment"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_SEGMENT"></span>**сегментаци**
</dt> <dd>

Часть геометрического пути, определяемая начальной и конечной точками.

</dd> <dt>

<span id="direct2d.direct2d_glossary_tessellate"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TESSELLATE"></span>**проводить тесселяцию**
</dt> <dd>

Для разбиения фигуры на коллекцию треугольников.

</dd> <dt>

<span id="direct2d.direct2d_glossary_transform"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TRANSFORM"></span>**Преобразует**
</dt> <dd>

Матрица, представляющая линейное сопоставление в ряде измерений. Direct2D использует матрицу размером 2 на 3, которая ограничивает API этими преобразованиями, которые являются аффинное.

</dd> <dt>

<span id="direct2d.direct2d_glossary_translation"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_TRANSLATION"></span>**операций**
</dt> <dd>

Процесс использования матрицы преобразования для перемещения объекта.

</dd> <dt>

<span id="direct2d.direct2d_glossary_vertex"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_VERTEX"></span>**положений**
</dt> <dd>

Самая высокая точка кривой, точка, в которой заканчивается кривая, или точка, в которой два сегмента соответствуют многоугольнику или произвольному контуру.

</dd> <dt>

<span id="direct2d.direct2d_glossary_vertex_shader"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_VERTEX_SHADER"></span>**Шейдер вершин**
</dt> <dd>

Тип программируемого шейдера, который содержит набор инструкций ассемблера, которые выполняются в GPU на вершину и на пиксель для текущего вызова Draw. Шейдер вершин разгружает ресурсоемкие вычисления с ЦП на GPU.

</dd> <dt>

<span id="direct2d.direct2d_glossary_windows_imaging_component__wic_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_WINDOWS_IMAGING_COMPONENT__WIC_"></span>**Компонент Windows Imaging Component (WIC)**
</dt> <dd>

API, позволяющий приложениям (1) отображать и редактировать любой формат изображения, для которого установлен кодек, совместимый с WIC, а также (2) считывать и записывать метаданные или файлы изображений.

</dd> <dt>

<span id="direct2d.direct2d_glossary_xml_paper_specification__xps_"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_XML_PAPER_SPECIFICATION__XPS_"></span>**Спецификация формата XML (XPS)**
</dt> <dd>

Формат документа, описываемый спецификацией XML Paper, который можно использовать для хранения документов, их обработки для печати и печати.

</dd> <dt>

<span id="direct2d.direct2d_glossary_xps"></span><span id="DIRECT2D.DIRECT2D_GLOSSARY_XPS"></span>**ОБСЛУЖИВАНИИ**
</dt> <dd>

См. определение для: спецификация формата XML Paper (XPS).

</dd> </dl>
