# <a name="d1228-not-original-target"></a>D1228: непервоначальный целевой объект

Вызов ID2D1HwndRenderTarget:: resize или ID2D1DCRenderTarget:: Бинддк был выполнен, пока в контексте устройства задан целевой объект, отличный от исходного.
