# <a name="d1169-discard-map-option-requires-dynamic-texture"></a>D1169: для параметра "отменить карту" требуется динамическая текстура

\_Параметры D2D1 карты "Отмена" \_ \_ допустимы только с точечным рисунком, созданным из динамической текстуры.






 

## <a name="possible-causes"></a>Возможные причины

Direct2D может выводить это сообщение, если приложение вызывает ID2D1Bitmap1:: Map с \_ атрибутом D2D1 "отменить карту" \_ \_ , но базовый ресурс Direct3D для точечного рисунка не имеет атрибута \_ " \_ Динамическое использование D3D11".

## <a name="possible-fixes"></a>Возможные исправления

Если приложению требуется доступ к точечному рисунку с помощью ID2D1Bitmap1:: Map, убедитесь, что в базовом ресурсе Direct3D имеется \_ \_ динамический атрибут использования D3D11.
