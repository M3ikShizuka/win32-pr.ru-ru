# <a name="d1174-pushlayer-to-command-list-will-ignore-layer-object"></a>D1174: в списке команд Пушлайер будет пропущен объект слоя

При помещении слоя (не равного NULL) в список команд объект слоя будет проигнорирован.





 

## <a name="possible-causes"></a>Возможные причины

Если команда Пушлайер использует объект ID2D1Layer, а целевой объект является списком команд, Direct2D будет вести себя так, как если бы слой не был указан, и не будет накладывать фиксированное границы на уровне, если список команд будет отображен позже.

## <a name="possible-fixes"></a>Возможные исправления

Приложения, которые предполагают команд потоковой передачи в список команд, должны использовать объекты слоев со значением NULL или учитывать, что параметры ограничения размера, указанные в объекте ID2D1Layer, не будут учитываться.
