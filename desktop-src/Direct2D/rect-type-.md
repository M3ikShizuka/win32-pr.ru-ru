# <a name="recttype-function"></a>Rect, <Type> функция

Создает прямоугольную структуру, в которой хранятся координаты, используя указанный тип данных.

``` syntax
template<typename Type>
typename TypeTraits<Type>::Rect Rect(
  Type left,
  Type top,
  Type right,
  Type bottom
);
```

## <a name="template-parameters"></a>Параметры шаблона



| Параметр | Описание                                                                                                |
|-----------|------------------------------------------------------------------------------------------------------------|
| Тип      | Тип данных, используемый для хранения размеров прямоугольника. Возможными значениями являются **float** и **UINT32**. |



 

## <a name="parameters"></a>Параметры



| Параметр | Описание:                                             |
|-----------|---------------------------------------------------------|
| Левое      | Координата по оси x верхнего левого угла прямоугольника.  |
| top       | Координата по оси y верхнего левого угла прямоугольника.  |
| right     | Координата по оси x правого нижнего угла прямоугольника. |
| снизу    | Координата по оси y правого нижнего угла прямоугольника. |



 

## <a name="return-value"></a>Возвращаемое значение

Структура прямоугольника, которая содержит указанные координаты.
