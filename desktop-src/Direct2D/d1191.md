# <a name="d1191-gdi-compatible-bitmap-option-was-used-incorrectly"></a>D1191: параметр точечного рисунка, совместимого с GDI, использовался неправильно

\_Параметры битовой карты D2D1 \_ \_ \_ совместимые с GDI должны сочетаться с \_ параметрами битовой карты D2D1 \_ \_






 

## <a name="possible-causes"></a>Возможные причины

В приложении указано несовместимое сочетание \_ флагов параметров точечного рисунка D2D1 \_ .

## <a name="possible-fixes"></a>Возможные исправления

Добавьте \_ Параметры битовой карты D2D1 \_ \_ в \_ флаги параметров растрового изображения D2D1 \_ .
