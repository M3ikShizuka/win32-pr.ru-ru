# <a name="d1166-discard-map-option-requires-write-map-option"></a>D1166: для параметра "отменить карту" требуется параметр "записать карту"

Параметры D2D1 Map reon \_ \_ \_ должны быть объединены \_ с \_ параметрами \_ записи D2D1 Map






 

## <a name="possible-causes"></a>Возможные причины

Direct2D может выводить это сообщение, если приложение вызывает ID2D1Bitmap1:: Map с \_ параметрами Map D2D1 \_ \_ отменить, но не D2D1 \_ \_ Параметры Map \_ .

## <a name="possible-fixes"></a>Возможные исправления

Для использования параметра "Отмена" требуется запись, чтобы быть полезной операцией в приложении. \_ \_ Описание параметров Map см. в разделе Параметры D2D1 Map. При вызове ID2D1Bitmap1:: Map определите, какой тип доступа требуется приложению.
