# <a name="d1165-read-and-discard-map-options-are-incompatible"></a>D1165: параметры схемы Read и Discard несовместимы

\_ \_ Чтение параметров D2D1 Map \_ несовместимо с параметрами " \_ \_ отменить карту \_ D2D1"





 

## <a name="possible-causes"></a>Возможные причины

Direct2D может выводить это сообщение, если приложение вызывает ID2D1Bitmap1:: Map с \_ параметрами Map D2D1 для \_ \_ Read и D2D1 \_ Map параметры \_ \_ Discard.

## <a name="possible-fixes"></a>Возможные исправления

Параметры READ и DISCARD не согласуются друг с другом, поскольку предоставление доступа на чтение к неопределенным данным имеет ограниченную полезность для приложения. \_ \_ Описание параметров Map см. в разделе Параметры D2D1 Map. При вызове ID2D1Bitmap1:: Map определите, какой тип доступа требуется приложению.
