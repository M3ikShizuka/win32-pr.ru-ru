# <a name="premultiply-effect"></a>Преумножить результат

Используйте этот результат для преобразования изображения из унпремултиплиед Alpha в предварительно умноженный альфа-канал. Этот результат заменяет каждый входной пиксель `P = {R, G, B, A}` `P' = {R*A, G*A, B*A, A}` на в выходных данных.

У этого действия нет свойств.

Идентификатором CLSID для этого действия является CLSID \_ D2D1Premultiply.

## <a name="output-bitmap"></a>Битовая карта вывода

Размер выходного растрового изображения совпадает с размером входного растрового изображения.
