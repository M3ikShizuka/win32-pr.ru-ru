# <a name="d1149-print-detected-an-invalid-unicode-character"></a>D1149: Print обнаружил недопустимый символ Юникода

Direct2D Print: обнаружен недопустимый символ Юникода \_ в \_ описании для описания запуска глифа дврите \_ . String. Он был заменен на пробел (0x20).






 

## <a name="possible-causes"></a>Возможные причины

Direct2D Print считает недопустимые символы, указанные в [документации по Юникоду](https://unicode.org/faq/utf_bom.html#utf16-7), а также определенные символы ASCII за пределами видимого диапазона, которые не могут быть распечатаны. Использование этих символов не приводит к ошибке. Вместо этого Direct2Dная печать заменит эти символы пробелами.

## <a name="possible-fixes"></a>Возможные исправления

Чтобы избежать этого уведомления, не используйте неправильно сформированные символы Юникода или непечатаемый ASCII-код при рисовании текста для печати.
