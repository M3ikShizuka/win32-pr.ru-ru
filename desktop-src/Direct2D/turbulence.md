# <a name="turbulence-effect"></a>Турбуленце, результат

Используйте турбуленценый результат для создания точечного рисунка на основе функции шума Perl.

У влияния турбуленце нет входного изображения.

Идентификатором CLSID для этого действия является CLSID \_ D2D1Turbulence.

-   [Битовая карта вывода](#output-bitmap)

## <a name="example-image"></a>Пример изображения

![снимок экрана примера эффектов, показывающий выходные данные турбуленценого действия.](images/32-turbulence.png)

Турбуленценый результат вычисляет сумму одного или нескольких октав функции шума Perl. Шум Perl — это функция псевдо-Random, значение которой зависит от частоты, расположения и начального значения. Этот результат создает значения RGBA с помощью одного из этих уравнений.

При выборе \_ режима шума D2D1 турбуленце "сумма помех" \_ \_ \_ в результате используется это уравнение.

![Снимок экрана, на котором показана функция турбуленце, используемая для создания точечного рисунка.](images/turbulence-equation1.png)

Если выбрать \_ режим шума ТУРБУЛЕНЦЕ D2D1 \_ турбуленце \_ , то в результате будет использоваться это уравнение.

![Функция турбуленце, используемая для создания точечного рисунка.](images/turbulence-equation2.png)

> [!Note]  
> `PerlinNoise`Функция имеет диапазон \[ -1, 1 \] .

 

Этот результат выводит значения пикселей в предварительно умноженном альфа-канале.

## <a name="noise-modes"></a>Режимы шума



| Перечисление                           | Описание                                                                           |
|---------------------------------------|---------------------------------------------------------------------------------------|
| \_ \_ \_ Сумма фрактала шума D2D1 турбуленце \_ | Вычисляет сумму октав, сдвигя выходного диапазона от \[ -1, 1 \] до \[ 0, 1 \] . |
| D2D1 \_ турбуленце \_ шум \_ турбуленце   | Вычисляет сумму абсолютного значения каждого Октава.                                  |



 

> [!Note]  
> Ни один из режимов не содержит явный срез выходных значений.

Этот результат приводит к созданию точечного рисунка с логически бесконечным размером.
