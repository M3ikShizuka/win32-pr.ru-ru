# <a name="table-transfer-effect"></a>Действие перемещения таблицы

Используйте перенос таблицы, чтобы связать цвет интенситиес изображения с помощью функции передачи, созданной при интерполяции предоставленного списка значений.

Идентификатором CLSID для этого действия является CLSID \_ D2D1TableTransfer.

```C++
ComPtr<ID2D1Effect> tableTransferEffect;
m_d2dContext->CreateEffect(CLSID_D2D1TableTransfer, &tableTransferEffect);

tableTransferEffect->SetInput(0, bitmap);

float table[2] = {0.75f, 1.0f};
tableTransferEffect->SetValue(D2D1_TABLETRANSFER_PROP_BLUE_TABLE, table);

m_d2dContext->BeginDraw();
m_d2dContext->DrawImage(tableTransferEffect.Get());
m_d2dContext->EndDraw();
```



Функция перемещения основана на списке входных данных V = (v0, v1, v2, v3, V? , V<sub>n</sub>), где N — число элементов-1.

Интенсивность входных пикселей представлена как C. Интенсивность выходных пикселов, C может быть рассчитана с помощью уравнения.

Для значения C выберите значение k, например: k/N = C < (k + 1)/N

Выходные данные C рассчитываются с использованием следующего уравнения: C ' = V? + (C-k/N) \* N \* (V??? одного? -V?)

Этот результат работает с прямыми и предварительно умноженными альфа-образами. Результат выводит предварительно умноженные альфа-точечные рисунки.

Вот как выглядит график функции перемещения таблиц, если свойство Table имеет значение `[0.0, 0.25, 1.0]` .
