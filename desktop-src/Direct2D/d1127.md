# <a name="d1127-alpha-mode-mismatch"></a>D1127: несоответствие режима альфа-канала

Режим альфа, переданный в этот API, несовместим с форматом пикселей \[ *поверхности* \] .

## <a name="placeholders"></a>Заполнители

<dl> <dt>

<span id="surface"></span><span id="SURFACE"></span>*подоб*
</dt> <dd>

Поверхность Direct3D, не поддерживающая альфа-режим.

</dd> </dl> 



 

## <a name="possible-causes"></a>Возможные причины

Для таких операций, как ID2D1RenderTarget:: Креатебитмапфромвикбитмап или ID2D1RenderTarget:: Креатедксгисурфацерендертаржет, требуется как поверхность, так и указанный режим альфа-канала поверхности. Direct2D проверяет, совместим ли указанный альфа-режим с поверхностью. Если он несовместим, Direct2D выводит это сообщение.

## <a name="possible-fixes"></a>Возможные исправления

Измените приложение так, чтобы оно использовало альфа \_ -режим, соответствующий или совместимый с, режим альфа базовой поверхности. Дополнительные рекомендации по режимам альфа, предоставляемым Direct2D, см. в разделе [Поддерживаемые форматы пикселей и альфа-режимы](supported-pixel-formats-and-alpha-modes.md) .
