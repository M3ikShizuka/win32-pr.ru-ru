# <a name="d1132-dxgi-format-not-hwnd-or-dc-compatible"></a>D1132: формат DXGI не совместим с HWND или DC

Заданный формат DXGI недопустим для этого метода создания целевого объекта прорисовки.



|             |         |
|-------------|---------|
| Уровень ошибки | Предупреждение |



 

## <a name="possible-causes"></a>Возможные причины

Создание HWND или рендертаржетс контроллера домена с помощью ID2D1Factory:: Креатехвндрендертаржет или ID2D1Factory:: Креатедкрендертаржет предоставляет дополнительные ограничения на формат поверхности целевого объекта. В частности, эти целевые объекты должны иметь тип \_ \_ R8G8B8A8 \_ UNORM или DXGI \_ \_ B8G8R8A8 \_ UNORM в формате DXGI.

## <a name="possible-fixes"></a>Возможные исправления

Чтобы устранить эту проблему, убедитесь, что формат, указанный для HWND приложения или рендертаржет контроллера домена, является одним из указанных выше форматов. Если приложению не требуются эти типы целевых объектов, рассмотрите возможность подготовки к просмотру текстуры. параметры формата поверхности для обеспечения совместимости в качестве целевого объекта могут быть менее ограниченными.
