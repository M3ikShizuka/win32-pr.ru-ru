# <a name="d1140-stroke-transform-type-uses-default-dpi"></a>D1140: тип преобразования «Обводка» использует значение DPI по умолчанию

При расширяющем ID2D1Geometry с ФИКСИРОВАНной или волосяной линией Direct2D будет принимать значение DPI 96. При отрисовке расширяющейся геометрии ширина штриха может измениться.






 

## <a name="possible-causes"></a>Возможные причины

Когда Direct2D рисует геометрию в целевом объекте, она расширяет геометрию. Однако приложение также может создавать объекты ID2D1Geometry без целевого объекта и расширять их, используя методы ID2D1Geometry, такие как Widening, Строкеконтаинспоинт и Жетвиденедбаундс. Во время этих операций приложение может использовать D2D1 \_ Stroke преобразование типа "линия" \_ или " \_ \_ фиксированный". Обычно эти атрибуты учитывают DPI, но в отсутствие целевого объекта DPI не указывается. Это информационное сообщение предупреждает о том, что Direct2D будет учитывать значение DPI по умолчанию 96,0.

## <a name="possible-fixes"></a>Возможные исправления
