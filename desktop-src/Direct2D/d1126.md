# <a name="d1126-alpha-mode-unknown-not-allowed"></a>D1126: неизвестный режим альфа-канала не разрешен

\_Неизвестный \_ режим альфа-канала D2D1 \_ не допускается в качестве параметра для этого API.





 

## <a name="possible-causes"></a>Возможные причины

\_ \_ В некоторых контекстах разрешен Неизвестный режим альфа-канала, например описание формата для ID2D1RenderTarget:: креатекомпатиблерендертаржет. Однако, как и в \_ случае \_ неизвестного формата DXGI, Direct2D обычно требует полного указания форматов.

## <a name="possible-fixes"></a>Возможные исправления

При принятии решения в отношении описателя формата следует подумать, что альфа-канал использует режим альфа, отличный \_ от Альфа- \_ режима \_ Unknown. Используйте \_ режим альфа D2D1 Alpha \_ \_ , если на поверхности не используется альфа-канал. См. раздел Примечания в [**\_ \_ режиме альфа-D2D1 Alpha**](/windows/desktop/api/dcommon/ne-dcommon-d2d1_alpha_mode).
