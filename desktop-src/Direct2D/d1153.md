# <a name="d1153-closed-command-lists-cannot-be-set-as-targets"></a>D1153: списки закрытых команд не могут быть заданы в качестве целевых объектов

Была предпринята попытка установить список закрытых команд для целевого объекта.






 

## <a name="possible-causes"></a>Возможные причины

После закрытия списка команд приложение не должно пытаться установить его в качестве целевого объекта. Direct2D считает этот вызов ID2D1DeviceContext:: Сеттаржет ошибкой, так как содержимое списка команд не может измениться после его закрытия. Установка списка команд в качестве целевого объекта означает, что задача будет отображена.

## <a name="possible-fixes"></a>Возможные исправления

Чтобы устранить это сообщение, не вызывайте ID2D1DeviceContext:: Сеттаржет с закрытым списком команд. Если необходимо использовать список команд с новым, другим содержимым, создайте новый список команд.
