# <a name="d1184-createwicbitmaprendertarget-incompatible-with-hw-target-type"></a>D1184: Креатевикбитмапрендертаржет несовместим с типом целевого объекта HW

Не допускается передача флага D2D1 \_ визуализации \_ целевого \_ \_ устройства типа в креатевикбитмапрендертаржет.






 

## <a name="possible-causes"></a>Возможные причины

Когда приложение с именем ID2D1Factory:: Креатевикбитмапрендертаржет, поле Type в \_ \_ аргументе свойств целевого объекта прорисовки D2D1 \_ было D2D1 \_ D2D1 \_ визуализации \_ целевого \_ типа \_ Hardware.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что целевой тип, используемый при создании целевого объекта рендеринга WIC-интероппед, — это программное обеспечение или значение по умолчанию. Direct2D обычно резервирует АППАРАТную семантику для описания взаимодействия Direct3D.
