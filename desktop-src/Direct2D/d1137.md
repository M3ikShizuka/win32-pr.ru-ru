# <a name="d1137-unclosed-command-list"></a>D1137: незакрытый список команд

Ссылка на список команд не была заполнена или не была определена в контексте устройства.






 

## <a name="possible-causes"></a>Возможные причины

Это сообщение может появиться при попытке прорисовки или потоковой передачи списка команд перед вызовом операции ID2D1CommandList:: Close. Direct2D не учитывает список команд до тех пор, пока он не будет заполнен. использование метода Close указывает на заполнение списка команд. Если приложение будет использовать список команд для операции рисования, то используемый в нем контекст устройства не должен быть задан в качестве целевого объекта.

## <a name="possible-fixes"></a>Возможные исправления

Прежде чем пытаться использовать список команд, вызовите его метод ID2D1CommandList:: Close. Если список команд будет нарисован с помощью ID2D1DeviceContext::D Равимаже или в качестве источника кисти изображения, убедитесь, что список команд не является текущей целью соответствующего контекста устройства.
