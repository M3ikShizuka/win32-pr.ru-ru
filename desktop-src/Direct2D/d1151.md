# <a name="d1151-unsupported-array-selection"></a>D1151: неподдерживаемый выбор массива

Переданный IDXGISurface2 принадлежит массиву, но срезы массивов не поддерживаются на этом уровне устройства (D3D9).






 

## <a name="possible-causes"></a>Возможные причины

Текстуры Direct3D интероппед могут быть массивами. Direct2D будет обрабатываться только в одном срезе массива за раз. Другие срезы массивов родительской текстуры не будут затронуты или визуализированы с помощью Direct2D. Однако устройства уровня "функция 9" не поддерживают выбор срезов массивов и не могут использовать эту функцию взаимодействия.

## <a name="possible-fixes"></a>Возможные исправления

Для приложений, которые должны быть правильно запущены на устройствах уровня Feature 9, рассмотрите возможность изменения логики отрисовки, чтобы избежать необходимости в взаимодействии с массивными текстурами. По возможности используйте вместо него промежуточную. Чтобы избежать последствий, связанных с этим ограничением, приложение может выбрать его реализацию в качестве резервного пути, относящегося к уровню компонентов 9.
