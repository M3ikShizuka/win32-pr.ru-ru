# <a name="d1145-print-control-dpi-incorrect"></a>D1145: неправильная DPI для элемента управления печатью

Direct2D печать: \_ свойства элемента управления печатью \_ . значение растердпи недопустимо.






 

## <a name="possible-causes"></a>Возможные причины

При создании элемента управления печатью это сообщение означает, что в приложении указано недопустимое значение DPI; в частности, значение DPI меньше 1,0. Хотя Direct2D не учитывает это как ошибку, Direct2D печать будет работать так, как если бы приложение указало значение DPI 150,0.

## <a name="possible-fixes"></a>Возможные исправления

Чтобы устранить это сообщение, используйте значение DPI 1,0 или выше при создании элемента управления печатью.
