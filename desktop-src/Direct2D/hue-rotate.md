# <a name="hue-rotatation-effect"></a>Ротататионный тон, результат

Используйте параметр цветовой тон, чтобы изменить оттенок изображения, применив цветовую матрицу на основе угла поворота.

Идентификатором CLSID для этого действия является CLSID \_ D2D1HueRotation.

-   [Пример изображения](#example-image)
-   [Свойства эффектов](#effect-properties)
-   [Битовая карта вывода](#output-bitmap)
-   [Требования](#requirements)
-   [Связанные темы](#related-topics)

## <a name="example-image"></a>Пример изображения

В примере ниже показано, как повлиять на входные и выходные изображения оттенка, используя угол поворота в 270 градусов.



| До                                                       |
|--------------------------------------------------------------|
| ![изображение перед результатом.](images/default-before.jpg)   |
| После                                                        |
| ![изображение после преобразования.](images/17-huerotation.png) |



 


```C++
ComPtr<ID2D1Effect> hueRotationEffect;
m_d2dContext->CreateEffect(CLSID_D2D1HueRotation, &hueRotationEffect);

hueRotationEffect->SetInput(0, bitmap);
hueRotationEffect->SetValue(D2D1_HUEROTATION_PROP_ANGLE, 270.0f);

m_d2dContext->BeginDraw();
m_d2dContext->DrawImage(hueRotationEffect.Get());
m_d2dContext->EndDraw();
```



Этот результат вычисляет матрицу цветов на основе угла поворота (*?*), указанного с помощью \_ \_ \_ свойства Angle D2D1 хуеротатион. Ниже приведены матричные уравнения.

![Вычисление поворота оттенков](images/hue-formula.png)

Созданная матрица зависит только от угла поворота. Если требуется определенная матрица, можно использовать [цветовую матрицу](color-matrix.md) .

## <a name="effect-properties"></a>Свойства эффектов



| Отображаемое имя и перечисление индекса                         | Тип и значение по умолчанию           | Описание                              |
|------------------------------------------------------------|----------------------------------|------------------------------------------|
| Angle<br/> D2D1 \_ хуеротатион \_ , \_ угол<br/> | FLOAT<br/> указано<br/> | Угол поворота оттенка в градусах. |



 

## <a name="output-bitmap"></a>Битовая карта вывода

Размер выходного растрового изображения совпадает с размером входного растрового изображения.
