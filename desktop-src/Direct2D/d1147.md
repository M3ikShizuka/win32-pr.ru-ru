# <a name="d1147-print-control-closed-without-addpage"></a>D1147: элемент управления печатью закрыт без AddPage

Direct2D печать: Close вызывается без предыдущих вызовов AddPage






 

## <a name="possible-causes"></a>Возможные причины

Это сообщение может появиться, если [**ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) закрывается без предварительного вызова [**ID2D1PrintControl:: addPage**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-addpage). Direct2D Print считает, что элемент управления печатью не будет содержать страниц, и соответствующая команда [**ID2D1PrintControl:: Close**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1printcontrol-close) возвратит ошибку.

## <a name="possible-fixes"></a>Возможные исправления

Чтобы устранить это сообщение, добавьте страницу в элемент управления печатью перед закрытием. Если приложению не требуется печатать содержимое, не следует закрывать элемент управления печатью или создавать его вообще.
