# <a name="d1170-invalid-flags-with-dynamic-texture-bitmap"></a>D1170: недопустимые флаги с динамическим точечным рисунком текстуры

\_Параметры СОПОСТАВЛЕНИЯ D2D1 Write D2D1 (параметры карты \_ записи) \_ \| \_ \_ \_ при отображении битовой карты, созданной на основе динамической текстуры, требуется отменить.






 

## <a name="possible-causes"></a>Возможные причины

Direct2D может выводить это сообщение, если приложение вызывает ID2D1Bitmap1:: Map с текстурой, базовый ресурс Direct3D которой помечен как D3D11ное \_ использование \_ dynamic, но не указал оба \_ параметра D2D1 Map \_ Параметры \_ Write и D2D1 \_ Map \_ \_ flags Discard.

## <a name="possible-fixes"></a>Возможные исправления

Если приложению требуется доступ к точечному рисунку с помощью ID2D1Bitmap1:: Map и его базовый ресурс Direct3D с атрибутом \_ использования D3D11 \_ dynamic, укажите оба \_ \_ параметра: D2D1 Map параметры \_ Write и D2D1 \_ Map \_ \_ flags Discard. \_ \_ Описание параметров Map см. в разделе Параметры D2D1 Map.
