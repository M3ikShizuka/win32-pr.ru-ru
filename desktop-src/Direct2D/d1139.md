# <a name="d1139-unused-stroke-width"></a>D1139: неиспользуемая ширина штриха

Указан тип преобразования «линия», а ширина штриха не равна 1. Ширина штриха будет пропущена.






 

## <a name="possible-causes"></a>Возможные причины

Приложение нарисует примитив с помощью ID2D1StrokeStyle с D2D1 \_ обводкой \_ \_ типа преобразования \_ , но ширина штриха, используемая в этом вызове рисования, не равна 1. Когда задается атрибут «волосяный», он, в действительности, переопределяет ширину штриха, исправляя ее в 1,0 единиц, зависящих от устройства, независимо от преобразования масштаба или DPI.

## <a name="possible-fixes"></a>Возможные исправления

Когда для ID2D1StrokeStyle задается атрибут «волосяный», приложение должно использовать ширину штрихов 1,0 единиц.
