# <a name="opacity-effect"></a>Воздействие непрозрачности

Этот результат корректирует непрозрачность изображения, умножая альфа-канал входных данных на заданное значение прозрачности. Он имеет один вход.

Идентификатором CLSID для этого действия является CLSID \_ D2D1Opacity.

## <a name="effect-properties"></a>Свойства эффектов



| Отображаемое имя и перечисление индекса              | Тип и значение по умолчанию | Описание                                                                                                 |
|-------------------------------------------------|------------------------|-------------------------------------------------------------------------------------------------------------|
| Непрозрачность свойства непрозрачности \_ D2D1 \_ \_<br/> | FLOAT 1.0 f<br/>   | Множитель для альфа-канала входного изображения. Минимальное значение — 0,0 f, а максимальное значение — 1,0 f. |
