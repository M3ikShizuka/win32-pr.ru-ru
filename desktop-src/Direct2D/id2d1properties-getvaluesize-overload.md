Возвращает размер значения свойства в байтах с использованием индекса свойства.

| Метод                                                       | Описание                                                                                                     |
|:-------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------|
| [**Жетвалуесизе (U)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getvaluesize(u))     | Возвращает размер значения свойства в байтах с использованием индекса свойства. Это перегрузка шаблона.<br/> |
| [**Жетвалуесизе (UINT32)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getvaluesize(uint32)) | Возвращает размер значения свойства в байтах с использованием индекса свойства.<br/>                              |
