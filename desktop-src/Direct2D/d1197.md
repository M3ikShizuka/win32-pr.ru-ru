# <a name="d1197-wrong-d3d-device"></a>D1197: неправильное устройство D3D

Поверхность DXGI, переданная в этот API, связана с неправильным устройством.



|             |         |
|-------------|---------|
| Уровень ошибки | Предупреждение |



 

## <a name="possible-causes"></a>Возможные причины

Поверхность DXGI, переданная в этот API, связана с неправильным устройством.

## <a name="possible-fixes"></a>Возможные исправления

Убедитесь, что все поверхности DXGI имеют то же устройство D3D, что и контекст устройства D2D.
