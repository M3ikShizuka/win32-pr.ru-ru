# <a name="d1247-dynamic-copy-from-memory"></a>D1247: Динамическая копия из памяти

Копифроммемори не может скопировать память в растровое изображение, созданное из динамической текстуры.



|             |         |
|-------------|---------|
| Уровень ошибки | Предупреждение |



 

## <a name="possible-causes"></a>Возможные причины

Копифроммемори был вызван для точечного рисунка, созданного как \_ \_ Динамическое использование D3D11.

## <a name="possible-fixes"></a>Возможные исправления

Для этого типа точечного рисунка следует использовать сопоставление и сопоставление.
