# <a name="security-and-identity"></a>Безопасность и идентификация

Разрабатывайте более безопасные классические приложения с помощью интерфейсов API и служб Windows. Эти API предоставляют:

- Аутентификация
- Авторизация
- Шифрование
- Службы каталогов, удостоверений и доступа
- Родительский контроль
- Управление правами

В этом разделе также приводятся рекомендации и другие статьи по безопасности.
