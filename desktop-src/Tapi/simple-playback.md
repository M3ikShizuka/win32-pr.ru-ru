# <a name="simple-playback"></a>Простое воспроизведение

В следующем разделе описывается, как выполнить простое воспроизведение.

**Выполнение простого воспроизведения**

1.  Выберите терминал воспроизведения файлов на уровне вызовов.
2.  Создайте терминал воспроизведения в вызове TAPI.
3.  Задайте свойства — имя файла и тип хранилища.
4.  Вызовите метод [**Start**](/windows/desktop/api/tapi3if/nf-tapi3if-itmediacontrol-start) в терминале воспроизведения файлов, чтобы начать воспроизведение потоков.
