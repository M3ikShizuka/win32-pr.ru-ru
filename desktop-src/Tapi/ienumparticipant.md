# <a name="ienumparticipant-interface"></a>Интерфейс ИенумпартиЦипант

\[**ИенумпартиЦипант** недоступен для использования в Windows Vista, windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **иенумпартиЦипант** предоставляет стандартные методы перечисления com для интерфейса [**итпартиЦипант**](itparticipant.md) . Метод [**итпартиЦипантконтрол:: енумератепартиЦипантс**](itparticipantcontrol-enumerateparticipants.md) возвращает указатель на **иенумпартиЦипант**.

Интерфейс **иенумпартиЦипант** скрыт от Visual Basic и языков сценариев.

## <a name="members"></a>Элементы

Интерфейс **иенумпартиЦипант** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **ИенумпартиЦипант** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иенумпартиЦипант** содержит следующие методы.



| Метод                                  | Описание                                                                                        |
|:----------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**Клонировать**](ienumparticipant-clone.md) | Создает другой перечислитель с тем же состоянием перечисления, что и текущий.<br/> |
| [**Далее**](ienumparticipant-next.md)   | Возвращает указанное число следующих элементов в последовательности перечисления.<br/>                 |
| [**Перезапуск**](ienumparticipant-reset.md) | Выполняет сброс до начала последовательности перечисления.<br/>                                    |
| [**Пропустить**](ienumparticipant-skip.md)   | Пропускает следующее заданное число элементов в последовательности перечисления.<br/>           |
