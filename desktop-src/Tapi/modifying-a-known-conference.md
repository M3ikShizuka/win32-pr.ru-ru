# <a name="modifying-a-known-conference"></a>Изменение известной конференции

\[В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

В следующем фрагменте кода показано изменение интервала времени Конференции. Интервал времени по умолчанию составляет тридцати минут. В этом фрагменте интервал времени установлен в один год.

В этом фрагменте предполагается, что [Подключение к серверу ILS](connecting-to-an-ils-server.md) уже выполнено, и было выполнено [перечисление каталога конференций](enumerating-conference-directories.md) для получения записи каталога, которая будет изменена.

Этот фрагмент также предполагает, что это не многоадресная конференция. Для изменения времени на конференции многоадресной рассылки требуется уведомление о сервере распределения адресов многоадресной рассылки. Пример работы с адресацией многоадресной рассылки см. [в разделе Получение адреса многоадресной рассылки](acquiring-a-multicast-address.md) .
