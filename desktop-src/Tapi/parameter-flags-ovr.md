# <a name="parameter-flags"></a>Флаги параметров

Флаги параметров предоставляют сведения о различных флагах состояния для сеанса связи, например, следует ли блокировать идентификацию звонящего. Список флагов, определенных TAPI, см. в разделе [ \_ константы линекаллпарамфлагс](./linecallparamflags--constants.md) .

Не все поставщики служб поддерживают использование этих сведений.

**Интерфейс TAPI 2. x:** См. раздел [**линежеткаллинфо**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo) (**Двкаллпарамфлагс** Member of *лпкаллинфо*).

**Интерфейс TAPI 3. x:** См. раздел [**иткаллинфо:: Get \_ каллинфолонг**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfolong) (элемент **CIL \_ каллпарамсфлагс** [**каллинфо \_ Long**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_long)).
