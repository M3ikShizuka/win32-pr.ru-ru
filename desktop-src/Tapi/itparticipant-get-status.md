# <a name="itparticipantget_status-method"></a>Метод ИтпартиЦипант:: Get \_ status

\[**получить \_ состояние** недоступно для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **Get \_ Status** Возвращает тип Variant \_ bool, указывающий состояние участника.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_Status(
  [in]  ITStream     *pITStream,
  [out] VARIANT_BOOL *pStatus
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*питстреам* \[ окне\]
</dt> <dd>

Указатель на интерфейс [**итстреам**](/windows/win32/api/tapi3if/nn-tapi3if-itstream) .

</dd> <dt>

*пстатус* \[ заполняет\]
</dt> <dd>

ВАРИАНТ \_ true, если участник включен в потоке, вариант \_ false, если он отключен.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Код возврата                                                                                   | Описание                                                               |
|-----------------------------------------------------------------------------------------------|---------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Метод успешно выполнен.<br/>                                              |
| <dl> <dt>**E \_ нотимпл**</dt> </dl>     | Метод не реализован.<br/>                                        |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти для выполнения операции.<br/>           |
| <dl> <dt>**\_указатель E**</dt> </dl>     | Параметр *питстреам* не является допустимым указателем.<br/>              |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>  | Параметр *питстреам* не указывает на допустимый интерфейс.<br/> |



 

## <a name="remarks"></a>Remarks

Включение или отключение состояния участника в потоке позволяет приложению по сути отключать данного участника.
