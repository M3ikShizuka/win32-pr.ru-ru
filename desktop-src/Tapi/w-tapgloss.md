# <a name="w-telephony-api"></a>W (API телефонии)

[A](a-tapgloss.md) [B](b-tapgloss.md) [C](c-tapgloss.md) [D](d-tapgloss.md) [E](e-tapgloss.md) F G [р](h-tapgloss.md) [.](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [U](u-tapgloss.md) [V](v-tapgloss.md) W X Y Z

<dl> <dt>

<span id="tapi2.web_telephony_tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_TAPGLOSS"></span>**Веб-телефония**
</dt> <dd>

Платформа приложений, которая поддерживает веб-сайты с двойным доступом (доступные по телефону или Интернет-браузеру) и приложения IVR на основе HTML. Платформа включает в себя подсистему среды выполнения, интегрированную в сервер, интерфейс администрирования на основе COM для установки и настройки приложений веб-телефонии, а также доступ к объекту вызова TAPI 3. x и API сценариев для реализации операций телефонии.

</dd> <dt>

<span id="tapi2.web_telephony_engine_webte__tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_ENGINE_WEBTE__TAPGLOSS"></span>**Модуль веб-телефонии (Вте)**
</dt> <dd>

Обеспечивает поддержку приложений IVR на основе HTML и веб-сайтов с двойным доступом (веб-сайт, доступный по телефону, а также через графический браузер). См. раздел *веб-телефония*.

</dd> <dt>

<span id="tapi2.windows_open_services_architecture_wosa__tapgloss"></span><span id="TAPI2.WINDOWS_OPEN_SERVICES_ARCHITECTURE_WOSA__TAPGLOSS"></span>**Windows Архитектура открытых служб (ВОСА)**
</dt> <dd>

Архитектура программного обеспечения, обеспечивающая единый интерфейс системного уровня для подключения интерфейсных приложений к внутренним службам. Windows Телефонная связь является частью ВОСА.

</dd> </dl>
