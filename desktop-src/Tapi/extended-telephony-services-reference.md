# <a name="extended-telephony-services-reference"></a>Справочник по расширенным службам телефонии

Расширенные функции телефонии перечислены по категориям в следующих таблицах.

## <a name="extended-telephony-functions-for-line-devices"></a>Расширенные функции телефонии для устройств графики



| Функция                                                   | Описание                                                                                                  |
|------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**линенеготиатикстверсион**](/windows/desktop/api/Tapi/nf-tapi-linenegotiateextversion) | Позволяет приложению согласовывать версию расширения для использования с указанным устройством. Асинхронная. |
| [**линедевспеЦифик**](/windows/desktop/api/Tapi/nf-tapi-linedevspecific)                 | Предоставляет поставщикам услуг доступ к функциям для конкретных устройств, которые не предлагаются другими функциями TAPI. Синхронный. |
| [**линедевспеЦификфеатуре**](/windows/desktop/api/Tapi/nf-tapi-linedevspecificfeature)   | Отправляет функции коммутатора для конкретного устройства. Асинхронная.                                           |



 

## <a name="extended-telephony-functions-for-phone-devices"></a>Расширенные функции телефонии для телефонных устройств



| Функция                                                     | Описание                                                                                                  |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**фонедевспеЦифик**](/windows/desktop/api/Tapi/nf-tapi-phonedevspecific)                 | Функция экранирования для конкретного устройства, позволяющая использовать зависимые от поставщика расширения. Асинхронная.                          |
| [**фоненеготиатикстверсион**](/windows/desktop/api/Tapi/nf-tapi-phonenegotiateextversion) | Позволяет приложению согласовывать версию расширения для использования с указанным телефонным устройством. Синхронный. |
