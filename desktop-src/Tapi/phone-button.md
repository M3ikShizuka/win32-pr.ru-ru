## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Комментарии

Сообщение **\_ кнопки телефона** отправляется при каждом изменении состояния кнопки. Приложение гарантирует, что для каждого события нажатия кнопки в конечном итоге отправляется соответствующее событие кнопки up. Поставщик услуг, который не способен обнаружить фактическую кнопку, должен создать сообщение кнопки вверх вскоре после сообщения нажатия кнопки.
