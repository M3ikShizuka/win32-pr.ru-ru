# <a name="itqosapplicationid-interface"></a>Интерфейс Иткосаппликатионид

\[ Этот интерфейс недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **иткосаппликатионид** предоставляет метод, позволяющий приложению получить идентификатор качества обслуживания для текущего вызова.

Этот интерфейс реализуется с помощью [ИПКОНФ MSP](ipconf-msp.md) и предоставляется только в том случае, если вызов использует конференции IP.

## <a name="members"></a>Элементы

Интерфейс **иткосаппликатионид** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Иткосаппликатионид** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иткосаппликатионид** содержит следующие методы.



| Метод                                                                | Описание                         |
|:----------------------------------------------------------------------|:------------------------------------|
| [**сеткосаппликатионид**](itqosapplicationid-setqosapplicationid.md) | Задает идентификатор качества обслуживания.<br/> |
