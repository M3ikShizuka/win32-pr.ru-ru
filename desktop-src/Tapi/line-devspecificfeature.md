# <a name="line_devspecificfeature-message"></a>Строка \_ сообщения девспеЦификфеатуре

Сообщение TAPI **Line \_ девспеЦификфеатуре** отправляется для уведомления приложения о событиях конкретного устройства, происходящих в строке, адресе или вызове. Значение сообщения и интерпретация параметров зависят от устройства.


```C++
            
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хдевице* 
</dt> <dd>

Обработчик для линейного устройства или вызова. Это зависит от устройства.

</dd> <dt>

*двкаллбаккинстанце* 
</dt> <dd>

Экземпляр обратного вызова, указанный при открытии строки.

</dd> <dt>

*dwParam1* 
</dt> <dd>

Зависит от устройства.

</dd> <dt>

*dwParam2* 
</dt> <dd>

Зависит от устройства.

</dd> <dt>

*dwParam3* 
</dt> <dd>

Зависит от устройства.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Remarks

Сообщение **Line \_ девспеЦификфеатуре** используется поставщиком услуг в сочетании с функцией [**линедевспеЦификфеатуре**](/windows/desktop/api/Tapi/nf-tapi-linedevspecificfeature) . Его значение зависит от конкретного устройства.
