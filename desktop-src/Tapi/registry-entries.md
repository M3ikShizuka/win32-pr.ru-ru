# <a name="registry-entries-telephony-api"></a>Записи реестра (API телефонии)

Подключаемые терминалы классифицируются в подклассы терминала. Каждый суперкласс терминала имеет запись в реестре в следующем разделе:

**HKey \_ \_** \\ **программное обеспечение** локального компьютера \\ **Microsoft** \\ **Windows** \\ **CurrentVersion** \\ **телефон** \\ **терминалманажер**

Под ключом суперкласса терминала находятся записи для каждого подключаемого терминала в суперклассе терминала.

Запись для каждого суперкласса терминала определяется идентификатором CLSID суперкласса терминала. Это уникальный идентификатор для класса терминала. Класс Terminal имеет необязательный атрибут: Name, который является понятным именем для класса Terminal. Каждый подключаемый терминал определяется классом терминала CLSID. то есть GUID. Атрибуты для подключаемого терминала сохраняются в значениях подключаемого ключа терминала. Ниже приведены атрибуты для подключаемого терминала.



| Класс терминала | Имя ключа | Общедоступный идентификатор                                                                 |
|----------------|----------|-----------------------------------------------------------------------------------|
| Имя           | REG \_ SZ  | Понятное имя терминала                                                            |
| Company        | REG \_ SZ  | Название компании                                                                      |
| Версия        | REG \_ SZ  | Сведения о версии                                                               |
| CLSID          | REG \_ SZ  | CLSID терминала (используется в методе COM [**CoCreateInstance**](/windows/win32/api/combaseapi/nf-combaseapi-cocreateinstance) ) |
| Маршруты     | DWORD    | Направление терминала                                                                |
| MediaTypes     | DWORD    | Поддерживаемые типы носителей терминала                                                    |



 

Для класса терминала используются следующие атрибуты.



| CLSID | Имя ключа | Общедоступный идентификатор            |
|-------|----------|------------------------------|
| Имя  | REG \_ SZ  | Понятное имя класса терминала |
