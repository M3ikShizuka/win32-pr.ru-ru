# <a name="hdrvline"></a>хдрвлине

Тип данных **хдрвлине** представляет непрозрачный маркер поставщика услуг для структуры линейных данных. Поставщик услуг должен разрешить значение этого типа в ссылку на соответствующий экземпляр структуры данных.
