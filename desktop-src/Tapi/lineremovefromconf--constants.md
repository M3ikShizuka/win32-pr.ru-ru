# <a name="lineremovefromconf_-constants"></a>\_Константы линеремовефромконф

Скалярные константы **линеремовефромконф \_** описывают, как стороны, участвующие в конференц-вызове, можно удалить из вызова конференции.

<dl> <dt>

<span id="LINEREMOVEFROMCONF_ANY"></span><span id="lineremovefromconf_any"></span>**ЛИНЕРЕМОВЕФРОМКОНФ \_ любой**
</dt> <dd> <dl> <dt>



Любую участвующую в вызове конференцию можно удалить.


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_LAST"></span><span id="lineremovefromconf_last"></span>**ЛИНЕРЕМОВЕФРОМКОНФ, \_ последний**
</dt> <dd> <dl> <dt>



Из вызова конференции можно удалить только недавно добавленную вечеринку.


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_NONE"></span><span id="lineremovefromconf_none"></span>**ЛИНЕРЕМОВЕФРОМКОНФ \_ None**
</dt> <dd> <dl> <dt>



Невозможно удалить стороны из вызова конференции.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Комментарии

Без расширяемости. Все 32 бит зарезервированы.
