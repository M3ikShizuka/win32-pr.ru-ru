# <a name="itsubstream-mspi"></a>Итсубстреам (МСПИ)

Интерфейс [**итсубстреам**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstream) доступен, если MSP связан с вызовом и реализует подпотоки. Методы этого интерфейса позволяют приложению получать сведения о подпотоке; для запуска, приостановки или остановки подпотока; а для выбора или отменяя выбор терминалов в подпотоке.

Интерфейс [**итсубстреам**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstream) реализуется с помощью MSP. Дополнительные сведения об этом интерфейсе см. в разделе **итсубстреам** в интерфейсе MSP.
