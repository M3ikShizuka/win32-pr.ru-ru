# <a name="h323-tsp"></a>H323 TSP

\[H323 TSP недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

H323 TSP (H323. TSP), сопоставленный с протоколом [H323 MSP](./h323-msp.md) (H323msp.dll), позволяет приложению выполнять и принимать видеои звуковые вызовы с помощью протокола H. 323 для IP-телефонии.

Этот TSP поддерживает значения [**типа адреса**](./lineaddresstype--constants.md) линеаддресстипе \_ PHONENUMBER, ЛИНЕАДДРЕССТИПЕ \_ IPAddress и линеаддресстипе \_ имя_домена. ЛИНЕАДДРЕССТИПЕ \_ имя_домена ссылается на доменное имя компьютера.

Протокол H323 MSP был улучшен в Windows 2000 с пакетом обновления 1 (SP1) для обеспечения обмена данными между клиентами H323 и SDP.

H323 TSP устанавливается вместе с Windows 2000 и более поздними версиями.
