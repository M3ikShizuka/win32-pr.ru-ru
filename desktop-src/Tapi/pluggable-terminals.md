# <a name="pluggable-terminals"></a>Подключаемые терминалы

Подключаемые терминалы позволяют приложению создавать терминалы во время выполнения. Обычно терминалы создаются и контролируются парой поставщика услуг телефонии/поставщика услуг мультимедиа (TSP/MSP), и это остается самым универсальным и стабильным способом манипуляций с терминалами. Однако некоторые приложения должны использовать простые терминалы в ситуациях, когда соответствующая пара TSP/MSP недоступна. Использовать подключаемые терминалы могут только те разработчики приложений, которые хорошо знакомы с операциями TSP/MSP.

В TAPI версии 3,1 подключаемые терминалы добавляются в модель TAPI. Подключаемые терминалы позволяют сторонним разработчикам добавлять новый объект терминала, который может использовать любой MSP.

Дополнительные сведения см. в разделе [подключаемый модуль регистрации терминалов](pluggable-terminal-registration.md) и [запись подключаемого терминала](writing-a-pluggable-terminal.md) .
