# <a name="ittime-interface"></a>Интерфейс Иттиме

\[ В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **иттиме** представляет собой интерфейс, зависящий от поставщика, для объекта "объект большого двоичного объекта" в службе протокола дескрипторов сеансов (SDP). Он предоставляет доступ к набору времени активации для сеанса. Методы [**иенумтиме:: Next**](ienumtime-next.md) и [**Иттимеколлектион:: Create**](ittimecollection-create.md) создают интерфейс **иттиме** .

## <a name="members"></a>Элементы

Интерфейс **иттиме** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Иттиме** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иттиме** содержит следующие методы.



| Метод                                         | Описание                                                                 |
|:-----------------------------------------------|:----------------------------------------------------------------------------|
| [**получение \_ StartTime**](ittime-get-starttime.md) | Возвращает значение времени начала 32-разрядного NTP (сетевой протокол времени).<br/> |
| [**получить \_ StopTime**](ittime-get-stoptime.md)   | Возвращает значение времени окончания NTP.<br/>                                  |
| [**Добавление \_ StartTime**](ittime-put-starttime.md) | Задает 32-разрядное значение времени начала NTP.<br/>                         |
| [**разместить \_ StopTime**](ittime-put-stoptime.md)   | Задает значение времени окончания NTP.<br/>                                  |
