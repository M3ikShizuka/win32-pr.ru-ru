# <a name="origin"></a>Исходный домен

Источник — это общее описание того, где запущен сеанс, например, является ли он внешним или внутренним для корпоративной сети. Список источников, поддерживаемых TAPI, см. в разделе [ \_ константы линекаллоригин](./linecallorigin--constants.md) .

Не все поставщики служб поддерживают использование этих сведений.

**Интерфейс TAPI 2. x:** См. раздел [**линежеткаллинфо**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo) (**DwOrigin** Member of *лпкаллинфо*).

**Интерфейс TAPI 3. x:** См. раздел [**иткаллинфо:: Get \_ каллинфолонг**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfolong) (элемент **\_ источника CIL** из [**каллинфо \_ Long**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_long)).
