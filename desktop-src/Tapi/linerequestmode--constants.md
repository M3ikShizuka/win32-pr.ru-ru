# <a name="linerequestmode_-constants"></a>\_Константы линерекуестмоде

Константы битовых флагов **линерекуестмоде** описывают различные типы запросов телефонии, которые могут быть сделаны из одного приложения в другое.

<dl> <dt>

<span id="LINEREQUESTMODE_DROP"></span><span id="linerequestmode_drop"></span>**ЛИНЕРЕКУЕСТМОДЕ \_ DROP**
</dt> <dd> <dl> <dt>



Запрос [тапирекуестдроп](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop) .


</dt> </dl> </dd> <dt>

<span id="LINEREQUESTMODE_MAKECALL"></span><span id="linerequestmode_makecall"></span>**ЛИНЕРЕКУЕСТМОДЕ \_ MAKECALL**
</dt> <dd> <dl> <dt>



Запрос [тапирекуестмакекалл](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall) .


</dt> </dl> </dd> <dt>

<span id="LINEREQUESTMODE_MEDIACALL"></span><span id="linerequestmode_mediacall"></span>**ЛИНЕРЕКУЕСТМОДЕ \_ медиакалл**
</dt> <dd> <dl> <dt>



Эта константа устарела. Не используйте.


</dt> </dl> </dd> </dl>
