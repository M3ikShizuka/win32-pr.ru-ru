# <a name="itterminalevent-mspi"></a>Иттерминалевент (МСПИ)

Когда метод [**иттапиевентнотификатион:: Event**](/windows/desktop/api/Tapi3if/nf-tapi3if-ittapieventnotification-event) возвращает [**\_ событие TAPI**](/windows/desktop/api/Tapi3if/ne-tapi3if-tapi_event) , равное **\_ терминалу TE**, методы, предоставляемые интерфейсом **иттерминалевент** , можно использовать для получения сведений о произошедшем событии терминала, если MSP существует.

Интерфейс **иттерминалевент** реализуется MSP и недоступен, если с адресом не связан поставщик службы мультимедиа. Дополнительные сведения об этом интерфейсе см. в разделе **иттерминалевент** в интерфейсе MSP.
