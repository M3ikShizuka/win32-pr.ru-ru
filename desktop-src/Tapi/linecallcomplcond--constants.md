# <a name="linecallcomplcond_-constants"></a>\_Константы линекаллкомплконд

Константы побитового флага **линекаллкомплконд \_** описывают условия, при которых вызов может быть завершен.

<dl> <dt>

<span id="LINECALLCOMPLCOND_BUSY"></span><span id="linecallcomplcond_busy"></span>**ЛИНЕКАЛЛКОМПЛКОНД \_ занят**
</dt> <dd> <dl> <dt>



Завершение вызова в условиях занятости.


</dt> </dl> </dd> <dt>

<span id="LINECALLCOMPLCOND_NOANSWER"></span><span id="linecallcomplcond_noanswer"></span>**\_ответ линекаллкомплконд**
</dt> <dd> <dl> <dt>



Завершение вызова в разделе звонка без условий ответа.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Комментарии

Без расширяемости. Все 32 бит зарезервированы.
