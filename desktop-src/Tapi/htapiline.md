# <a name="htapiline"></a>хтапилине

Тип данных **хтапилине** представляет непрозрачный маркер TAPI для структуры линейных данных. TAPI должен разрешить значение этого типа в ссылку на соответствующий экземпляр структуры данных. Поставщик услуг не должен пытаться ссылаться на него так, как если бы он был указателем, делать предположения о его значениях или интерпретировать его представление любым способом, отличным от передачи его значения в TAPI в нужное время.
