# <a name="callhub-support"></a>Поддержка Каллхуб

Отслеживание Каллхуб — это новая функция в интерфейсе TAPI версии 3,0. Функции Каллхуб были добавлены в элементы программирования TAPI 2,1 с доставкой Windows 2000. *Каллхуб* представляет собой стороннее представление вызова, а дескрипторы вызовов TAPI представляют первое представление вызова. С помощью отслеживания Каллхуб поставщик услуг должен следовать за Каллхубс и предоставлять сведения о вызове в течение времени существования Каллхуб. По мере того, как новые стороны присоединяются и оставляют Каллхуб, поставщик услуг должен сообщить о TAPI.

Поставщику услуг не требуется реализовывать новые функции для поддержки Каллхуб. Он просто должен заполнить элемент **двкаллид** структуры [**линекаллинфо**](/windows/win32/api/tapi/ns-tapi-linecallinfo) . В TAPI 3 TAPI собирает все вызовы с одним и тем же **двкаллид** и создает обработчик каллхуб, который приложение может использовать для наблюдения за вызовом.
