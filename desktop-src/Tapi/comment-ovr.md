# <a name="comment"></a>Комментировать

Приложение может задать комментарий во время инициации сеанса. Комментарии часто используются в целях ведения журнала.

Не все поставщики служб поддерживают использование этих сведений.

**Интерфейс TAPI 2. x:** См. раздел [**линежеткаллинфо**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo) (**двкомментсизе** and **двкомментоффсет** Members of *лпкаллинфо*).

**Интерфейс TAPI 3. x:** См. раздел [**иткаллинфо:: Get \_ каллинфостринг**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfostring) (элемент **\_ комментария CIL** в [**\_ строке каллинфо**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_string)).
