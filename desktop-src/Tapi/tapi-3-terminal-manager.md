# <a name="tapi-3-terminal-manager"></a>Диспетчер терминалов TAPI 3

Диспетчер терминалов TAPI 3 создается Tapi3.dll по запросу MSP и используется MSP для таких задач, как создание и удаление динамических терминалов.

Точные сведения о реализации для динамических терминалов MSP выходят за рамки данной документации. Однако многие авторы могут использовать некоторые возможности терминала потоковой передачи мультимедиа (MST). этот терминал основан на DirectShow, а фильтры, относящиеся к конкретному протоколу, могут быть написаны и использованы вместе с MST.
