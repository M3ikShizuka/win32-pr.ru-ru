# <a name="phonehookswitchmode_-constants"></a>\_Константы фонехуксвитчмоде

Константы побитового флага **фонехуксвитчмоде \_** описывают компоненты микрофона и динамика устройства хуксвитч.

<dl> <dt>

<span id="PHONEHOOKSWITCHMODE_MIC"></span><span id="phonehookswitchmode_mic"></span>**\_микрофон фонехуксвитчмоде**
</dt> <dd> <dl> <dt>



Микрофон устройства активен, динамик выключен.


</dt> </dl> </dd> <dt>

<span id="PHONEHOOKSWITCHMODE_MICSPEAKER"></span><span id="phonehookswitchmode_micspeaker"></span>**ФОНЕХУКСВИТЧМОДЕ \_ микспеакер**
</dt> <dd> <dl> <dt>



Микрофон и динамик устройства активны.


</dt> </dl> </dd> <dt>

<span id="PHONEHOOKSWITCHMODE_ONHOOK"></span><span id="phonehookswitchmode_onhook"></span>**ФОНЕХУКСВИТЧМОДЕ \_ ONhook**
</dt> <dd> <dl> <dt>



Микрофон и динамик устройства являются onhook обработчиками.


</dt> </dl> </dd> <dt>

<span id="PHONEHOOKSWITCHMODE_SPEAKER"></span><span id="phonehookswitchmode_speaker"></span>**ФОНЕХУКСВИТЧМОДЕ \_ докладчик**
</dt> <dd> <dl> <dt>



Динамик устройства активен, микрофон выключен.


</dt> </dl> </dd> <dt>

<span id="PHONEHOOKSWITCHMODE_UNKNOWN"></span><span id="phonehookswitchmode_unknown"></span>**ФОНЕХУКСВИТЧМОДЕ \_ неизвестный**
</dt> <dd> <dl> <dt>



В настоящее время режим хуксвитча устройства неизвестен.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Комментарии

Без расширяемости. Все 32 бит зарезервированы.

Эти константы используются для предоставления индивидуального уровня контроля над компонентами микрофона и динамиков телефонного устройства.
