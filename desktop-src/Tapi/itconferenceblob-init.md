# <a name="itconferenceblobinit-method"></a>Метод Итконференцеблоб:: init

\[встречи и элементы управления встречными IP-телефонными соединениями недоступны для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **init** инициализирует конференцный BLOB-объект на основе текстовой строки. Если *пблоб* имеет **значение NULL**, используются значения по умолчанию.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Init(
  [in] BSTR               pName,
  [in] BLOB_CHARACTER_SET CharacterSet,
  [in] BSTR               pBlob
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*pName* \[ окне\]
</dt> <dd>

Указатель на представление имени Конференции в формате **BSTR** .

</dd> <dt>

*Кодировка* \[ окне\]
</dt> <dd>

[**Большой двоичный объект \_ Дескриптор \_ кодировки набора символов**](blob-character-set.md) .

</dd> <dt>

*пблоб* \[ окне\]
</dt> <dd>

Указатель на **строку BSTR** , содержащую большой двоичный объект конференции. Если **значение равно NULL**, то объект Conference BLOB инициализируется со значениями по умолчанию. В настоящее время значения конференций по умолчанию доступны, только если для параметра " *Кодировка* " задано значение " \_ ASCII".

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                         | Значение                                                                    |
|-----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Метод успешно выполнен.<br/>                                               |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>  | Недопустимый параметр *pName*, *character* или *пблоб* .<br/> |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти для выполнения операции.<br/>            |
| <dl> <dt>**\_Ошибка E**</dt> </dl>        | Незаданная ошибка.<br/>                                              |
| <dl> <dt>**E \_ нотимпл**</dt> </dl>     | Этот метод еще не реализован.<br/>                             |



 

## <a name="remarks"></a>Remarks

**Итконференцеблоб:: init** вернет ошибку, если большой двоичный объект уже инициализирован.

Приложение должно использовать [**сисаллокстринг**](/windows/win32/api/oleauto/nf-oleauto-sysallocstring) для выделения памяти для параметров *pName* и *пблоб* . Приложение должно использовать [**сисфристринг**](/windows/win32/api/oleauto/nf-oleauto-sysfreestring) для освобождения памяти, когда переменные больше не нужны.
