# <a name="tapi-levels-of-service"></a>Уровни обслуживания TAPI

Службы телефонии TAPI делятся на службы поддержки телефонии и службы, предоставляемые полным API телефонии. Как правило, полный API телефонии используется для реализации мощных телефонных приложений, а служба телефонии используется для добавления минимальных, но полезных функций телефонии в приложения, не связанные с телефонией.
