# <a name="tspi_linedropnoowner"></a>ТСПИ \_ линедропнувнер

Функция **тспи \_ линедропнувнер** устарела. TAPI больше не удаляет вызовы автоматически, если исходный владелец не может быть идентифицирован, чтобы распределять распределенный контроль вызовов между несколькими клиентскими компьютерами в среде клиент/сервер.
