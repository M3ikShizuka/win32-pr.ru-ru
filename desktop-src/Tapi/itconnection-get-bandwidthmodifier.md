# <a name="itconnectionget_bandwidthmodifier-method"></a>Метод Итконнектион:: Get \_ бандвидсмодифиер

\[ В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **Get \_ бандвидсмодифиер** получает модификатор пропускной способности, который представляет собой одно буквенно-цифровое слово, содержащее значение пропускной способности. Определены два модификатора: CT (общая конференция) и AS (максимальное число приложений).

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_BandwidthModifier(
  [out] BSTR *ppModifier
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ппмодифиер* \[ заполняет\]
</dt> <dd>

Указатель на **строку BSTR** , содержащую модификатор пропускной способности.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Код возврата                                                                                   | Описание                                                     |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Метод успешно выполнен.<br/>                                    |
| <dl> <dt>**\_указатель E**</dt> </dl>     | Параметр *ппмодифиер* не является допустимым указателем.<br/>   |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти для выполнения операции.<br/> |
| <dl> <dt>**\_Ошибка E**</dt> </dl>        | Незаданная ошибка.<br/>                                   |
| <dl> <dt>**E \_ нотимпл**</dt> </dl>     | Этот метод еще не реализован.<br/>                  |



 

## <a name="remarks"></a>Комментарии

Приложение должно использовать [**сисфристринг**](/windows/win32/api/oleauto/nf-oleauto-sysfreestring) для освобождения памяти, выделенной для параметра *ппмодифиер* .
