# <a name="cmspstream-pure-virtual-methods"></a>Чистые виртуальные методы Кмспстреам

Эти методы должны быть переопределены производными классами.



| Чистые виртуальные методы Кмспстреам        | Описание                                                                                                                                                                                               |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**получить \_ имя**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_name) | Получает имя потока. Класс производного потока должен предоставлять имя, которое позволяет пользователю принимать осмысленные различия между потоками, если это вызывается для в пользовательском интерфейсе приложения. |
