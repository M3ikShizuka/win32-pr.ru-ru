# <a name="ipconf-msp-interfaces"></a>Интерфейсы MSP Ипконф

\[ Интерфейсы MSP Ипконф недоступны для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

В MSP-конференциях для управления участниками реализовано несколько интерфейсов, зависящих от поставщика. Эти интерфейсы предоставляются на объекте Call, если этот MSP связан с вызовом.

Интерфейсы [**итпартиЦипантевент**](itparticipantevent.md) и [**иенумпартиЦипант**](ienumparticipant.md) являются изолированными объектами и перечисляются для удобства.



| Интерфейс                                            | Описание                                                                                                                                               |
|------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**имултикастконтрол**](imulticastcontrol.md)       | Позволяет приложению устанавливать и получать режим замыкания на себя ( [**\_ \_ режим многоадресной рассылки**](multicast-loopback-mode.md)) для объекта конференции многоадресной рассылки. |
| [**итпартиЦипант**](itparticipant.md)               | Позволяет приложению получать сведения о участниках Конференции и получать указатели на потоки, связанные с этими участниками.             |
| [**итпартиЦипантконтрол**](itparticipantcontrol.md) | Позволяет приложению извлекать указатели на участников Конференции.                                                                           |
| [**итпартиЦипантевент**](itparticipantevent.md)     | Позволяет приложению получать сведения о событиях, касающихся участника конференции.                                                                  |
| [**иенумпартиЦипант**](ienumparticipant.md)         | Перечисляет [**итпартиЦипант**](itparticipant.md).                                                                                                        |
