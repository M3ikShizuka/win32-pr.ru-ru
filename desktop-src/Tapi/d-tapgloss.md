<dl> <dt>

<span id="tapi2.d_channel_tapgloss"></span><span id="TAPI2.D_CHANNEL_TAPGLOSS"></span>**Канал D**
</dt> <dd>

Канал данных интерфейса ISDN. Это один из трех стандартных каналов в линии ISDN с обычной ставкой. На линии Анд-ISDN канал D может передавать сигнальную информацию и данные пакетов с низкой скоростью по частоте 16 000 бит/с. В линии на основе PRI-ISDN канал D выполняется с частотой 64 000 бит/с. См. раздел [*интерфейс базовой частоты (Анд-ISDN)*](b-tapgloss.md), [*B Channel*](b-tapgloss.md), [*интерфейс первичной ставки (PRI-ISDN)*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.device_class_tapgloss"></span><span id="TAPI2.DEVICE_CLASS_TAPGLOSS"></span>**класс устройства**
</dt> <dd>

Группа связанных физических устройств или драйверов устройств, с помощью которых приложения отправляют и получают данные или данные, которые составляют вызов. Список см. в разделе [классы устройств TAPI](tapi-device-classes.md).

</dd> <dt>

<span id="tapi2.did_tapgloss"></span><span id="TAPI2.DID_TAPGLOSS"></span>**СОЗДАЛ**
</dt> <dd>

См. раздел *прямой набор номеров*.

</dd> <dt>

<span id="tapi2.direct_inward_dialing_did__tapgloss"></span><span id="TAPI2.DIRECT_INWARD_DIALING_DID__TAPGLOSS"></span>**прямой набор номеров**
</dt> <dd>

Возможность, обеспечиваемая практически всеми современными УАТС, позволяющая внешнему вызывающему объекту обращаться к внутреннему расширению путем набора стандартного номера телефона из семи цифр без необходимости передачи через оператора или помощника. См. раздел [*Обмен частными ветвями (УАТС)*](p-tapgloss.md).

</dd> </dl>
