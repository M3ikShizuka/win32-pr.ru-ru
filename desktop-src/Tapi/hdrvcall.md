# <a name="hdrvcall"></a>хдрвкалл

Тип данных [**хдрвкалл**](hdrvline.md) представляет непрозрачный маркер поставщика услуг для структуры данных вызова. Поставщик услуг должен разрешить значение этого типа в ссылку на соответствующий экземпляр структуры данных.
