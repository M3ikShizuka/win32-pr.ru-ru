# <a name="track-controlled-record"></a>Запись Track-Controlled

В этом разделе приводится процедура для выполнения записи звуковых потоков, контролируемых системой мониторинга.

**Для выполнения записи звуковых потоков, контролируемых системой мониторинга**

1.  Выберите файловые терминалы на потоках.
2.  Создайте терминал записи файлов.
3.  Задайте имя файла.
4.  Перечисление потоков в вызове TAPI. Для выполнения этих действий см. следующую процедуру.
5.  Вызовите метод [**Start**](/windows/desktop/api/tapi3if/nf-tapi3if-itmediacontrol-start) в терминале записи файлов, чтобы начать запись потока.

**Перечисление потоков в вызове TAPI**

1.  Создайте дорожку с тем же типом и направлением звука, что и в потоке.
2.  Задайте свойства дорожки для звукового формата. Если не задано, формат будет таким же, как и формат потоков.
3.  Выберите запись в потоке.

Если в нескольких потоках выбрана одна запись, это подразумевает смешивание потоков.
