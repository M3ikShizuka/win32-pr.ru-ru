# <a name="ienummedia-interface"></a>Интерфейс Иенуммедиа

\[ В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **иенуммедиа** предоставляет стандартные методы перечисления com для интерфейса [**итмедиа**](itmedia.md) . Метод [**итмедиаколлектион:: Get \_ енумератиониф**](itmediacollection-get-enumerationif.md) возвращает указатель на **иенуммедиа**.

## <a name="members"></a>Элементы

Интерфейс **иенуммедиа** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иенуммедиа** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иенуммедиа** содержит следующие методы.



| Метод                            | Описание                                                                                        |
|:----------------------------------|:---------------------------------------------------------------------------------------------------|
| [**Клонировать**](ienummedia-clone.md) | Создает другой перечислитель с тем же состоянием перечисления, что и текущий.<br/> |
| [**Далее**](ienummedia-next.md)   | Возвращает указанное число следующих элементов в последовательности перечисления.<br/>                 |
| [**Перезапуск**](ienummedia-reset.md) | Выполняет сброс до начала последовательности перечисления.<br/>                                    |
| [**Пропустить**](ienummedia-skip.md)   | Пропускает следующее заданное число элементов в последовательности перечисления.<br/>           |
