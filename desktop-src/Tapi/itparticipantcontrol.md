# <a name="itparticipantcontrol-interface"></a>Интерфейс ИтпартиЦипантконтрол

\[**итпартиЦипантконтрол** недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **итпартиЦипантконтрол** реализуется с помощью MSP-ипконф. Этот интерфейс предоставляется для объекта вызова. Этот интерфейс позволяет приложению извлекать указатели на участников Конференции. Интерфейс **итпартиЦипантконтрол** создается путем вызова **QueryInterface** в [**иткаллинфо**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallinfo).

## <a name="members"></a>Элементы

Интерфейс **итпартиЦипантконтрол** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **ИтпартиЦипантконтрол** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **итпартиЦипантконтрол** содержит следующие методы.



| Метод                                                                      | Описание                                                                                                                                                                 |
|:----------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**енумератепартиЦипантс**](itparticipantcontrol-enumerateparticipants.md) | Перечисляет участников, которые в настоящее время участвуют в Конференции.<br/>                                                                                                    |
| [**получение \_ участников**](itparticipantcontrol-get-participants.md)          | Создает коллекцию участников, связанных с текущей Конференцией. Предоставляется для клиентских приложений автоматизации, например, написанных на Visual Basic.<br/> |
