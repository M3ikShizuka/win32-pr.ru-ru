# <a name="itterminalsupport-mspi"></a>Иттерминалсуппорт (МСПИ)

Интерфейс [**иттерминалсуппорт**](/windows/win32/api/tapi3if/nn-tapi3if-itterminalsupport) предоставляется для [объекта Address](address-object.md) , если существует MSP. Методы этого интерфейса позволяют приложению обнаруживать доступные терминалы и (или) создавать их и получать указатели на необходимые [объекты терминала](terminal-object.md).

Интерфейс [**иттерминалсуппорт**](/windows/win32/api/tapi3if/nn-tapi3if-itterminalsupport) реализуется MSP и недоступен, если с адресом не связан поставщик службы мультимедиа. Дополнительные сведения об этом интерфейсе см. в разделе **иттерминалсуппорт** в интерфейсе MSP.
