# <a name="displayable-address"></a>Отображаемый адрес

Отображаемый адрес — это строка, представляющая адрес в форме, которая подходит для отображения в пользовательском интерфейсе.

**Интерфейс TAPI 2. x:** См. раздел [**линежеткаллинфо**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo), **двдисплайаблеаддресссизе** and **двдисплайаблеаддрессоффсет** Members of [**линекаллинфо**](/windows/win32/api/tapi/ns-tapi-linecallinfo).

**Интерфейс TAPI 3. x:** См. раздел [**иткаллинфо:: Get \_ Каллинфостринг**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfostring) или [**иткаллинфо::p UT \_ каллинфостринг**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-put_callinfostring), который вызывается с помощью элемента **CI \_ дисплайаблеаддресс** в [**\_ строке каллинфо**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_string).
