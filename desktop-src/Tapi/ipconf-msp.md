# <a name="ipconf-msp"></a>Ипконф MSP

\[Ипконф MSP недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Ипконф MSP (Confmsp.dll), который связан со своим TSP (Ипконф. TSP), предоставляет доступ к конференциям в IP-сетях, таких как Интернет и многие корпоративные интрасети. В дополнение к стандартным методам МСПИ Этот MSP предоставляет интерфейсы, позволяющие управлять отдельными участниками Конференции.

Общие сведения о интерфейсах, предоставляемых этим MSP, см. в разделе [ИПКОНФ MSP interfaces](ipconf-msp-interfaces.md) .

Ипконф MSP усовершенствован в Windows 2000 с пакетом обновления 1 (SP1) для обеспечения обмена данными между клиентами H323 и SDP. Теперь он также предоставляет интерфейс [**имултикастконтрол**](imulticastcontrol.md) .

Ипконф MSP устанавливается вместе с Windows 2000.
