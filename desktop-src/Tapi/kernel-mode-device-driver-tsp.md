# <a name="kernel-mode-device-driver-tsp"></a>Kernel-Mode TSP драйвера устройства

TSP в режиме ядра (Кмддсп. TSP) позволяет приложению работать с устаревшими мини-портами глобальной сети (например, драйверами NDIS 4,0) как устройствами TAPI.

TSP в режиме ядра устанавливается вместе с операционными системами Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Millennium Edition, Windows 98 и Windows 95.
