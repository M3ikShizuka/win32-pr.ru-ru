<dl> <dt>

<span id="tapi2.t1_e1_tapgloss"></span><span id="TAPI2.T1_E1_TAPGLOSS"></span>**T1/E1**
</dt> <dd>

Канал цифровой передачи с емкостью 1 544 Мбит/с. В США эта ссылка называется *T1*. В Европейского союза это называется *E1*. Просмотреть [*число мегабит в секунду (Мбит/с)*](m-tapgloss.md).

</dd> <dt>

<span id="tapi2.tapi_tapgloss"></span><span id="TAPI2.TAPI_TAPGLOSS"></span>**TAPI**
</dt> <dd>

См. раздел *интерфейс программирования приложений телефонии (TAPI)*.

</dd> <dt>

<span id="tapi2.tapisrv_tapgloss"></span><span id="TAPI2.TAPISRV_TAPGLOSS"></span>**таписрв**
</dt> <dd>

Приложение службы поддержки, которое предоставляет контекст выполнения для библиотеки динамической компоновки TAPI для выполнения таких обязанностей, как выделение памяти или доступ к структурам данных, которые были переключены на диск.

</dd> <dt>

<span id="tapi2.tdd_tapgloss"></span><span id="TAPI2.TDD_TAPGLOSS"></span>**TDD**
</dt> <dd>

См. раздел *телекоммуникационные устройства для слуха (TDD)*.

</dd> <dt>

<span id="tapi2.telecommucations_devices_for_the_deaf_tdd__tapgloss"></span><span id="TAPI2.TELECOMMUCATIONS_DEVICES_FOR_THE_DEAF_TDD__TAPGLOSS"></span>**Телекоммукатионс устройства для слуха (TDD)**
</dt> <dd>

Устройство данных для передачи закодированных сигналов. Как правило, устройство — это простой компьютерный терминал, состоящий из клавиатуры, акустического кауплера и дисплея, который работает с частотой 300 бод.

</dd> <dt>

<span id="tapi2.telephony_tapgloss"></span><span id="TAPI2.TELEPHONY_TAPGLOSS"></span>**телефонии**
</dt> <dd>

Наука передачи голоса, данных, видео или изображений на расстоянии с помощью технологии, которая интегрирует компьютеры с телефонной сетью.

</dd> <dt>

<span id="tapi2.telephony_application_programming_interface_tapi__tapgloss"></span><span id="TAPI2.TELEPHONY_APPLICATION_PROGRAMMING_INTERFACE_TAPI__TAPGLOSS"></span>**Интерфейс программирования приложений телефонии (TAPI)**
</dt> <dd>

Набор функций, которые позволяют программировать устройства на основе телефонной линии аппаратно-независимым образом, предоставляя пользователям личную телефонию. TAPI поддерживает передачу речи и данных, позволяет использовать различные устройства терминала и поддерживает сложные типы подключения и методы управления вызовами, такие как звонки конференций, ожидание звонка и голосовая почта. TAPI позволяет управлять всеми элементами использования телефона — от простого вызова по электронной почте с помощью коммутации и обзвона в приложениях, разработанных для Microsoft Windows. См. [Обзор телефонии Майкрософт](./microsoft-telephony-overview.md)

</dd> <dt>

<span id="tapi2.telephony_service_provider_interface_tspi__tapgloss"></span><span id="TAPI2.TELEPHONY_SERVICE_PROVIDER_INTERFACE_TSPI__TAPGLOSS"></span>**Интерфейс поставщика услуг телефонии (ТСПИ)**
</dt> <dd>

Средство для создания поставщиков услуг для операционных систем Microsoft Windows. ТСПИ определяет, как сетевое совместное использование данных осуществляется с помощью телефонии Windows, который, в свою очередь, взаимодействует с API, который взаимодействует с приложениями телефонии Windows. Дополнительные сведения см. в разделе [интерфейс поставщика услуг телефонии (тспи)](./telephony-service-provider-interface-tspi-.md).

</dd> <dt>

<span id="tapi2.third_party_call_control_tapgloss"></span><span id="TAPI2.THIRD_PARTY_CALL_CONTROL_TAPGLOSS"></span>**Управление вызовами сторонних разработчиков**
</dt> <dd>

Модель управления вызовом субъекта, которая позволяет приложению устанавливать или отвечать на вызовы между любыми двумя сторонами.

</dd> <dt>

<span id="tapi2.time_to_live_ttl__tapgloss"></span><span id="TAPI2.TIME_TO_LIVE_TTL__TAPGLOSS"></span>**срок жизни (TTL)**
</dt> <dd>

Значение в диапазоне от 0 до 255, определяющее область, в которой пакеты многоадресной рассылки должны отправляться по сети по протоколу IP. Область определяется с точки зрения локального или удаленного назначения пакета. Каждый маршрутизатор уменьшает срок жизни на единицу. Когда значение достигает предопределенного нижнего предела, маршрутизатор порождает пакет. Текущие требования многоадресной магистрали (МБОНЕ), доступные на сайте faq.txt ftp://ftp.isi.edu/mbone/, определяют следующие стандартные области: локальная сеть, 1; локальный сайт, 15; регион, 63; мир, 127. Другие параметры могут иметь локальные значения; Например, 31 может указывать на все сайты в определенной организации.

</dd> <dt>

<span id="tapi2.tspi_tapgloss"></span><span id="TAPI2.TSPI_TAPGLOSS"></span>**тспи**
</dt> <dd>

См. раздел [интерфейс поставщика услуг телефонии (тспи)](./telephony-service-provider-interface-tspi-.md).

</dd> <dt>

<span id="tapi2.ttl_tapgloss"></span><span id="TAPI2.TTL_TAPGLOSS"></span>**ВЕЛИЧИН**
</dt> <dd>

См. *срок жизни (TTL)*.

</dd> <dt>

<span id="tapi2.twisted_pair_tapgloss"></span><span id="TAPI2.TWISTED_PAIR_TAPGLOSS"></span>**витая пара**
</dt> <dd>

Обычная телефонная связь. Каждая телефонная связь на самом деле является парой проводов.

</dd> </dl>
