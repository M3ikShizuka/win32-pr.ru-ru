# <a name="extended-phone-services"></a>Расширенные службы телефона

Расширенные службы телефонов (или телефонные службы для конкретных устройств) включают все расширения ТСПИ, определенные поставщиком услуг. ТСПИ определяет механизм, позволяющий поставщикам поставщиков услуг расширять ТСПИ с помощью расширений для конкретных устройств. ТСПИ только определяет механизм расширения, и это обеспечивает доступ к расширениям для конкретных устройств, поведение которых полностью определяется поставщиком услуг.

ТСПИ состоит из скалярных и битовых определений констант данных, структур данных, функций и сообщений обратного вызова. Определяются процедуры, позволяющие поставщику расширить большинство из них, как описано в подразделах этого раздела.
