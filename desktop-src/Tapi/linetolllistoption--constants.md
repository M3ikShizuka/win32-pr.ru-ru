# <a name="linetolllistoption_-constants"></a>\_Константы линетолллистоптион

Константы побитового флага **линетолллистоптион \_** описывают параметры для управления платным списком.

<dl> <dt>

<span id="LINETOLLLISTOPTION_ADD"></span><span id="linetolllistoption_add"></span>**ЛИНЕТОЛЛЛИСТОПТИОН \_ Добавить**
</dt> <dd> <dl> <dt>



Префикс будет добавлен в платный список.


</dt> </dl> </dd> <dt>

<span id="LINETOLLLISTOPTION_REMOVE"></span><span id="linetolllistoption_remove"></span>**ЛИНЕТОЛЛЛИСТОПТИОН \_ удалить**
</dt> <dd> <dl> <dt>



Префикс должен быть удален из списка платных плат.


</dt> </dl> </dd> </dl>

Без расширяемости. Все 32 бит зарезервированы.
