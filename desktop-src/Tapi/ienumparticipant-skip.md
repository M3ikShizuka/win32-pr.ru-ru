# <a name="ienumparticipantskip-method"></a>Метод ИенумпартиЦипант:: Skip

\[**Пропуск** недоступен для использования в Windows Vista, windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **Skip** пропускает следующее заданное число элементов в последовательности перечисления. Этот метод скрыт от Visual Basic и языков сценариев.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Skip(
  [in] ULONG celt
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*celt* \[ окне\]
</dt> <dd>

Число пропускаемых элементов.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Код возврата                                                                                   | Описание                                                     |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Число пропущенных элементов: *celt*.<br/>               |
| <dl> <dt>**S \_ false**</dt> </dl>       | Число пропущенных элементов не является *celt*.<br/>           |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти для выполнения операции.<br/> |
