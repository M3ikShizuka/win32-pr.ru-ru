# <a name="usage"></a>Использование

Существует два основных варианта использования интерфейса библиотеки DLL TSP пользовательского интерфейса: когда приложение вызывает функцию, предназначенную для создания пользовательского интерфейса, и когда поставщик услуг должен самостоятельно создавать пользовательский интерфейс в процессе обработки функции, которая не определена как создание пользовательского интерфейса.
