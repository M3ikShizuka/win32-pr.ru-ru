# <a name="linedevstatusflags_-constants"></a>\_Константы линедевстатусфлагс

Константы побитового флага **линедевстатусфлагс \_** описывают коллекцию элементов состояния логического устройства линии.

<dl> <dt>

<span id="LINEDEVSTATUSFLAGS_CONNECTED"></span><span id="linedevstatusflags_connected"></span>**ЛИНЕДЕВСТАТУСФЛАГС \_ подключен**
</dt> <dd> <dl> <dt>



Указывает, подключена ли линия к TAPI. Если **значение равно true**, линия подключена, а TAPI может использовать устройство линии. Если **значение равно false**, строка отсоединяется и приложение не может управлять устройством линии через TAPI.


</dt> </dl> </dd> <dt>

<span id="LINEDEVSTATUSFLAGS_INSERVICE"></span><span id="linedevstatusflags_inservice"></span>**ЛИНЕДЕВСТАТУСФЛАГС \_**
</dt> <dd> <dl> <dt>



Указывает, находится ли строка в службе. Если **значение равно true**, строка находится в службе; Если **значение равно false**, строка находится вне службы.


</dt> </dl> </dd> <dt>

<span id="LINEDEVSTATUSFLAGS_LOCKED"></span><span id="linedevstatusflags_locked"></span>**ЛИНЕДЕВСТАТУСФЛАГС \_ заблокирована**
</dt> <dd> <dl> <dt>



Указывает, заблокирована или разблокирована строка. Этот бит чаще всего используется с линейными устройствами, связанными с сотовыми телефонами. Многие Сотовые телефоны имеют механизм безопасности, требующий ввода пароля, чтобы телефон могли размещать вызовы. Этот бит можно использовать для обозначения приложений, заблокированных телефоном, и не может размещать вызовы до тех пор, пока пароль не будет указан в пользовательском интерфейсе телефона, чтобы приложение предприняло соответствующее предупреждение пользователю.


</dt> </dl> </dd> <dt>

<span id="LINEDEVSTATUSFLAGS_MSGWAIT"></span><span id="linedevstatusflags_msgwait"></span>**ЛИНЕДЕВСТАТУСФЛАГС \_ мсгваит**
</dt> <dd> <dl> <dt>



Указывает, ожидает ли строка сообщения. Если **значение равно true**, сообщение находится в состоянии ожидания; Если **значение равно false**, сообщение не ожидает выполнения.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Комментарии

Без расширяемости. Все 32 бит зарезервированы.

\_Константы линедевстатусфлагс используются в элементе **двдевстатусфлагс** структуры данных [**линедевстатус**](/windows/desktop/api/Tapi/ns-tapi-linedevstatus) .
