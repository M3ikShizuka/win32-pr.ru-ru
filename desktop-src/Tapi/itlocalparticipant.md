# <a name="itlocalparticipant-interface"></a>Интерфейс ИтлокалпартиЦипант

Интерфейс **итлокалпартиЦипант** предоставляется в объекте потока, если ипконф MSP поддерживает вызов. MSP реализует методы, позволяющие приложению получать и задавать сведения о участниках. Интерфейс **итлокалпартиЦипант** создается путем вызова **QueryInterface** в [**итстреам**](/windows/win32/api/tapi3if/nn-tapi3if-itstream).

## <a name="members"></a>Элементы

Интерфейс **итлокалпартиЦипант** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **ИтлокалпартиЦипант** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **итлокалпартиЦипант** содержит следующие методы.



| Метод                                                                                     | Описание                                                                            |
|:-------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------|
| [**получить \_ локалпартиЦипанттипединфо**](itlocalparticipant-get-localparticipanttypedinfo.md) | Возвращает сведения о участнике, такие как имя электронной почты или географическое расположение.<br/> |
| [**разместить \_ локалпартиЦипанттипединфо**](itlocalparticipant-put-localparticipanttypedinfo.md) | Задает сведения о участнике.<br/>                                               |
