# <a name="itaudiodevicecontrol-interface"></a>Интерфейс Итаудиодевицеконтрол

\[ Этот интерфейс недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **итаудиодевицеконтрол** предоставляет методы, позволяющие приложению получать и задавать параметры для управления звуковым устройством.

Этот интерфейс реализуется с помощью [ИПКОНФ MSP](ipconf-msp.md) и [H323 MSP](h323-msp.md). Он предоставляется, когда в вызове используются поставщики услуг или поставщик услуг стороннего поставщика, реализующий этот интерфейс. Чтобы получить указатель на интерфейс **итаудиодевицеконтрол** , вызовите метод **QueryInterface** в интерфейсе [**итстреам**](/windows/win32/api/tapi3if/nn-tapi3if-itstream) , который управляет звуковым устройством, соответствующим потоку. Для предоставления интерфейса **итаудиодевицеконтрол** тип мультимедиа интерфейса **итстреам** должен быть аудио.

## <a name="members"></a>Элементы

Интерфейс **итаудиодевицеконтрол** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Итаудиодевицеконтрол** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **итаудиодевицеконтрол** содержит следующие методы.



| Метод                                              | Описание                                                                                             |
|:----------------------------------------------------|:--------------------------------------------------------------------------------------------------------|
| [**Получить**](/windows/desktop/api/tapi3if/nf-tapi3if-itasrterminalevent-get_call)          | Возвращает значение заданного [Свойства звукового устройства](audiodeviceproperty.md).<br/>                  |
| [**GetRange**](/windows/desktop/api/tapi3if/nf-tapi3if-itasrterminalevent-get_terminal) | Возвращает диапазон допустимых значений для заданного [Свойства звукового устройства](audiodeviceproperty.md).<br/> |
| [**Параметр**](/windows/desktop/api/tapi3if/nf-tapi3if-itasrterminalevent-get_error)         | Задает значение заданного [Свойства звукового устройства](audiodeviceproperty.md).<br/>                  |
