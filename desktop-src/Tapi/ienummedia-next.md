# <a name="ienummedianext-method"></a>Метод Иенуммедиа:: Next

\[ В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

**Следующий** метод получает указанное число следующих элементов в последовательности перечисления.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Next(
  [in]  ULONG   celt,
  [out] ITMedia **pVal,
  [out] ULONG   *pceltFetched
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*celt* \[ окне\]
</dt> <dd>

Число запрошенных элементов.

</dd> <dt>

*Pval* \[ заполняет\]
</dt> <dd>

Указатель на интерфейс [**итмедиа**](itmedia.md) .

</dd> <dt>

*пцелтфетчед* \[ заполняет\]
</dt> <dd>

Указатель на число фактически предоставляемых элементов. Может иметь **значение NULL** , если значение *celt* равно единице.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                     | Значение                                                       |
|-------------------------------------------------------------------------------------------|---------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>      | Метод возвратил *celt* число элементов.<br/>         |
| <dl> <dt>**S \_ false**</dt> </dl>   | Количество оставшихся элементов меньше значения *celt*.<br/> |
| <dl> <dt>**\_указатель E**</dt> </dl> | Параметр *Pval* не является допустимым указателем.<br/>       |



 

## <a name="remarks"></a>Remarks

TAPI вызывает метод **AddRef** в интерфейсе [**итмедиа**](itmedia.md) , возвращенном методом **иенуммедиа:: Next**. Приложение должно вызвать **выпуск** в интерфейсе **итмедиа** , чтобы освободить ресурсы, связанные с ним.
