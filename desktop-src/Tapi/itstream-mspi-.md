# <a name="itstream-mspi"></a>Итстреам (МСПИ)

Интерфейс [**итстреам**](/windows/win32/api/tapi3if/nn-tapi3if-itstream) доступен, если MSP связан с вызовом. Методы этого интерфейса позволяют приложению получать сведения о потоке; для запуска, приостановки или остановки потока; а для выбора или отменяя выбор терминалов в потоке.

Интерфейс [**итстреам**](/windows/win32/api/tapi3if/nn-tapi3if-itstream) реализуется с помощью MSP. Дополнительные сведения об этом интерфейсе см. в разделе **итстреам** в интерфейсе MSP.
