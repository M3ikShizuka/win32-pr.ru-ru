# <a name="ienumtime-interface"></a>Интерфейс Иенумтиме

\[ В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **иенумтиме** предоставляет стандартные методы перечисления com для интерфейса [**иттиме**](ittime.md) . Метод [**иттимеколлектион:: Get \_ енумератиониф**](ittimecollection-get-enumerationif.md) возвращает указатель на **иенумтиме**.

## <a name="members"></a>Элементы

Интерфейс **иенумтиме** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иенумтиме** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иенумтиме** содержит следующие методы.



| Метод                           | Описание                                                                                        |
|:---------------------------------|:---------------------------------------------------------------------------------------------------|
| [**Клонировать**](ienumtime-clone.md) | Создает другой перечислитель с тем же состоянием перечисления, что и текущий.<br/> |
| [**Далее**](ienumtime-next.md)   | Возвращает указанное число следующих элементов в последовательности перечисления.<br/>                 |
| [**Перезапуск**](ienumtime-reset.md) | Выполняет сброс до начала последовательности перечисления.<br/>                                    |
| [**Пропустить**](ienumtime-skip.md)   | Пропускает следующее заданное число элементов в последовательности перечисления.<br/>           |
