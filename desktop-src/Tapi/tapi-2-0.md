# <a name="tapi-20"></a>TAPI 2,0

TAPI 2,0 предоставляет небольшое количество улучшений базовой функциональности TAPI 1,4. Однако были внесены значительные изменения в архитектуру, которые значительно улучшили стабильность. большинство изменений были фундаментальными изменениями, необходимыми для того, чтобы приложить TAPI к Windows NT 4,0 и воспользоваться преимуществами его возможностей (полной 32-разрядной поддержки, службами и юникодом). Однако эти изменения были внутренними в TAPI и не оказывают влияния на приложения TAPI.

Однако воздействие на разработчиков поставщиков услуг было существенным.
