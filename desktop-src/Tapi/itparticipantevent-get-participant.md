# <a name="itparticipanteventget_participant-method"></a>Метод ИтпартиЦипантевент:: Get \_ участника

\[**получить \_ Участник** недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **Get \_ участник** получает указатель на массив интерфейсов [**итпартиЦипант**](itparticipant.md) , представляющих участников, вовлеченных в событие.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_Participant(
  [out] ITParticipant **ppParticipant
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пппартиЦипант* \[ заполняет\]
</dt> <dd>

Указатель на массив интерфейсов [**итпартиЦипант**](itparticipant.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                           | Значение                                                          |
|-------------------------------------------------------------------------------------------------|------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>            | Метод успешно выполнен.<br/>                                     |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl>   | Недостаточно памяти для выполнения операции.<br/>  |
| <dl> <dt>**\_нежелательные \_ элементы TAPI**</dt> </dl> | Нет участников, связанных с событием.<br/>  |
| <dl> <dt>**\_указатель E**</dt> </dl>       | Параметр *пппартиЦипант* не является допустимым указателем.<br/> |
