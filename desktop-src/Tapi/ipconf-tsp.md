# <a name="ipconf-tsp"></a>Ипконф TSP

\[Ипконф TSP недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Ипконф TSP (Ипконф. TSP) при сопоставлении с MSP-протоколом [ипконф](./ipconf-msp.md) (Confmsp.dll) обеспечивает Конференц-доступ к сетям протокола Интернета, таким как Интернет и многие корпоративные интрасети.

Этот TSP поддерживает значение [**типа адреса**](./lineaddresstype--constants.md) линеаддресстипе \_ SDP.

Ипконф MSP был усовершенствован в Windows 2000 с пакетом обновления 1 (SP1) для обеспечения обмена данными между клиентами H323 и SDP.

Ипконф TSP устанавливается вместе с Windows 2000 и более поздними версиями.
