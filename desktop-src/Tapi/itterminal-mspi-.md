# <a name="itterminal-mspi"></a>Иттерминал (МСПИ)

Интерфейс [**иттерминал**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) является базовым интерфейсом для [объекта терминала](terminal-object.md). Он предоставляет методы для получения информации, такой как класс терминала и поддерживаемый мультимедиа.

Интерфейс [**иттерминал**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) реализуется MSP и недоступен, если с адресом не связан поставщик службы мультимедиа. Дополнительные сведения об этом интерфейсе см. в разделе **иттерминал** в интерфейсе MSP.
