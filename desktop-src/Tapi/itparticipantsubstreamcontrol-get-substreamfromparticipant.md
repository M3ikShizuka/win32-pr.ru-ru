# <a name="itparticipantsubstreamcontrolget_substreamfromparticipant-method"></a>Метод ИтпартиЦипантсубстреамконтрол:: Get \_ субстреамфромпартиЦипант

\[**получить \_ субстреамфромпартиЦипант** недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **Get \_ субстреамфромпартиЦипант** позволяет приложению определить, какие подпотоки связаны с данным участником.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_SubStreamFromParticipant(
  [in]  ITParticipant *pParticipant,
  [out] ITSubStream   **ppITSubStream
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ппартиЦипант* \[ окне\]
</dt> <dd>

Указатель на интерфейс [**итпартиЦипант**](itparticipant.md) .

</dd> <dt>

*ппитсубстреам* \[ заполняет\]
</dt> <dd>

Указатель на массив указателей интерфейса [**итпартиЦипант**](itparticipant.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Код возврата                                                                                   | Описание                                                                  |
|-----------------------------------------------------------------------------------------------|------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Метод успешно выполнен.<br/>                                                 |
| <dl> <dt>**\_указатель E**</dt> </dl>     | Параметр *ппитсубстреам* не является допустимым указателем.<br/>             |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>  | Параметр *ппартиЦипант* не указывает на допустимый интерфейс.<br/> |
| <dl> <dt>**\_непредвиденная E**</dt> </dl>  | Не удалось получить доступ к сведениям о участнике потока.<br/>       |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти для выполнения операции.<br/>              |
