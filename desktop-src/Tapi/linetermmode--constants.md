# <a name="linetermmode_-constants"></a>\_Константы линетерммоде

Константы побитового флага **линетерммоде \_** описывают различные типы событий в телефонной линии, которые могут направляться на устройство терминала.

<dl> <dt>

<span id="LINETERMMODE_BUTTONS"></span><span id="linetermmode_buttons"></span>**\_кнопки линетерммоде**
</dt> <dd> <dl> <dt>



Это события нажатия кнопки, отправленные с терминала на линию.


</dt> </dl> </dd> <dt>

<span id="LINETERMMODE_DISPLAY"></span><span id="linetermmode_display"></span>**ЛИНЕТЕРММОДЕ \_ дисплей**
</dt> <dd> <dl> <dt>



Это отображает сведения, отправляемые из строки в терминал.


</dt> </dl> </dd> <dt>

<span id="LINETERMMODE_HOOKSWITCH"></span><span id="linetermmode_hookswitch"></span>**ЛИНЕТЕРММОДЕ \_ хуксвитч**
</dt> <dd> <dl> <dt>



Это хуксвитч события, отправляемые из терминала в строку.


</dt> </dl> </dd> <dt>

<span id="LINETERMMODE_LAMPS"></span><span id="linetermmode_lamps"></span>**ЛИНЕТЕРММОДЕ \_ лампы**
</dt> <dd> <dl> <dt>



Это события лампы, отправляемые из линии в терминал.


</dt> </dl> </dd> <dt>

<span id="LINETERMMODE_MEDIABIDIRECT"></span><span id="linetermmode_mediabidirect"></span>**ЛИНЕТЕРММОДЕ \_ медиабидирект**
</dt> <dd> <dl> <dt>



Это двунаправленный поток мультимедиа, связанный с вызовом в строке и в терминале. Используйте это значение, если маршрутизация обоих однонаправленных каналов потока мультимедиа вызова не может контролироваться независимо друг от друга.


</dt> </dl> </dd> <dt>

<span id="LINETERMMODE_MEDIAFROMLINE"></span><span id="linetermmode_mediafromline"></span>**ЛИНЕТЕРММОДЕ \_ медиафромлине**
</dt> <dd> <dl> <dt>



Это однонаправленный поток мультимедиа из линии в терминал, связанный с вызовом в строке. Используйте это значение, если маршрутизация обоих однонаправленных каналов потока мультимедиа вызова может осуществляться независимо друг от друга.


</dt> </dl> </dd> <dt>

<span id="LINETERMMODE_MEDIATOLINE"></span><span id="linetermmode_mediatoline"></span>**ЛИНЕТЕРММОДЕ \_ медиатолине**
</dt> <dd> <dl> <dt>



Это однонаправленный поток мультимедиа от терминала до линии, связанной с вызовом в строке. Используйте это значение, если маршрутизация обоих однонаправленных каналов потока мультимедиа вызова может осуществляться независимо друг от друга.


</dt> </dl> </dd> <dt>

<span id="LINETERMMODE_RINGER"></span><span id="linetermmode_ringer"></span>**ЛИНЕТЕРММОДЕ \_ Ring**
</dt> <dd> <dl> <dt>



Это сведения об управлении звонками, отправляемые с коммутатора в терминал.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Remarks

Без расширяемости. Все 32 бит зарезервированы.

Эти константы описывают классы элементов управления и информационных потоков, которые могут направляться напрямую между линейным устройством и устройством терминала (например, набором телефонов).
