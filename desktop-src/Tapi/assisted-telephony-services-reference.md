# <a name="assisted-telephony-services-reference"></a>Справочник по службам телефонии

В следующей таблице перечислены функции службы, поддерживающие функцию телефонии. Функции телефонной связи позволяют любому приложению устанавливать голосовой вызов и вызовы мультимедиа. Общие сведения см. в обзоре службы [поддержки](./assisted-telephony-overview.md) .

## <a name="assisted-telephony-services-functions"></a>Функции служб телефонии службы поддержки



| Функция                                             | Описание                                                 |
|------------------------------------------------------|-------------------------------------------------------------|
| [**тапирекуестмакекалл**](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall)   | Отправляет запрос на размещение голоса.                    |
| [**тапирекуестмедиакалл**](tapirequestmediacall.md) | Является устаревшей. Не используйте.                                       |
| [**тапирекуестдроп**](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop)           | Является устаревшей. Не используйте.                                       |
| [**тапижетлокатионинфо**](/windows/desktop/api/Tapi/nf-tapi-tapigetlocationinfo)   | Возвращает коды, идентифицирующие страну/регион и город/область. |
