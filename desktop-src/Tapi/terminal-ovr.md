# <a name="terminal"></a>Терминал

Терминал — это источник или модуль подготовки отчетов для определенного адреса и типа носителя. Концепция терминала в основном используется в TAPI 3 и очень важна только для приложений TAPI 2. Дополнительные сведения см. в разделе Общие сведения об [объекте терминала](terminal-object.md) TAPI 3.
