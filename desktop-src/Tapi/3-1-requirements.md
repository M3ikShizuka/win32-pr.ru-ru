# <a name="tapi-31-requirements"></a>Требования к TAPI 3,1

Ниже перечислены требования к операционной системе для TAPI 3. x.

-   Для TAPI версии 3,0 требуется Windows 2000 или более поздняя версия.
-   Для TAPI версии 3,1 требуется операционная система Windows Server 2003 или Windows XP.

Обратите внимание, что для [управления конференцией IP-адресов](about-rendezvous-ip-telephony-conferencing.md) с помощью команды встреч требуется доступ хотя бы к одному серверу ILS или NTDS. Если сервер NTDS установлен, его можно использовать для нахождение сервера ILS. Если сервер NTDS не существует, конечный пользователь должен иметь имя сервера ILS.
