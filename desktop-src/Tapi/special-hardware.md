# <a name="special-hardware"></a>Специальное оборудование

Для некоторых дополнительных возможностей TAPI требуется, чтобы приложение могло, например, получать данные с телефонных телефонов. Большинство телефонов не могут напрямую подключаться к компьютерам для управления голосовыми вызовами, поэтому в настоящее время они не поддерживают функции телефонии за пределами пассивной роли, которую они играют в POTS. В будущем пользователи будут устанавливать и настраивать телефонные наборы, такие как другие периферийные устройства. Наборы будут сопровождаться картами, которые будут управлять потоком информации между компьютером и телефоном. Кроме того, будут доступны конфигурации клиента и сервера, позволяющие пользователям использовать преимущества телефонных служб путем подключения по локальной сети к серверу, на котором установлена такая плата и соответствующее программное обеспечение.
