# <a name="itaudiosettings-interface"></a>Интерфейс Итаудиосеттингс

\[ Этот интерфейс недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **итаудиосеттингс** предоставляет методы, позволяющие приложению получать и задавать параметры для управления звуковым устройством.

Этот интерфейс реализуется с помощью [ИПКОНФ MSP](ipconf-msp.md) и [H323 MSP](h323-msp.md). Он предоставляется только в том случае, если вызов использует эти поставщики служб.

## <a name="members"></a>Элементы

Интерфейс **итаудиосеттингс** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Итаудиосеттингс** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **итаудиосеттингс** содержит следующие методы.



| Метод                                              | Описание                                                                                                |
|:----------------------------------------------------|:-----------------------------------------------------------------------------------------------------------|
| [**Получить**](/windows/desktop/api/tapi3if/nf-tapi3if-itasrterminalevent-get_call)          | Возвращает значение заданного [Свойства звукового параметра](audiosettingsproperty.md).<br/>                  |
| [**GetRange**](/windows/desktop/api/tapi3if/nf-tapi3if-itasrterminalevent-get_terminal) | Возвращает диапазон допустимых значений для заданного [Свойства звукового параметра](audiosettingsproperty.md).<br/> |
| [**Параметр**](/windows/desktop/api/tapi3if/nf-tapi3if-itasrterminalevent-get_error)         | Задает значение заданного [Свойства звукового параметра](audiosettingsproperty.md).<br/>                  |
