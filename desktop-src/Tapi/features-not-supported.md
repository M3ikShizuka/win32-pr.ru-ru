# <a name="features-not-supported"></a>Функции не поддерживаются

-   Обработка динамического графа будет реализована с помощью производного MSP, если он решит это.
-   Если выбран этот параметр, подпотоки будут реализованы производным MSP.
-   Частные интерфейсы, такие как участники, элементы управления кодека и т. д., не будут реализованы базовыми классами.
-   Связь TSP-MSP будет реализована производным MSP.
