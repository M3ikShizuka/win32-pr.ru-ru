# <a name="accept"></a>Принять

В некоторых средах пользователь не предупреждает автоматически о вызове предложения, например, путем звонка или сообщения на экране компьютера. Операция Accept запускает процесс создания предупреждений (как правило, Звонок), и после этого выполняется операция [ответа](answer-ovr.md) . Приложение может [Удалить](drop-ovr.md) сеанс, [перенаправить](redirect-ovr.md) его или принять.

Если текущий поставщик услуг поддерживает его, приложение может отправить сведения о пользователе во время принятия. Нет никакой гарантии, что сеть будет предоставлять эту информацию вызывающей стороне.

Не все поставщики служб поддерживают использование этой операции.

**Интерфейс TAPI 2. x:** См. [**линеакцепт**](/windows/win32/api/tapi/nf-tapi-lineaccept).
