# <a name="cvideocaptureterminal"></a>квидеокаптуретерминал

Терминал **квидеокаптуретерминал** является производным от [ксинглефилтертерминал](csinglefilterterminal.md) и реализует статический терминал видеозаписи с помощью одного фильтра DirectShow. Он не поддерживает некоторые расширенные интерфейсы видеозаписи, которые применимы только к видеозаписи WDM. Большинству MSP не требуется переопределять реализацию этого терминала.

Определен в Мсптмвк. h.
