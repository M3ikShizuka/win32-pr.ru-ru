# <a name="tapi-3-terminal-manager-reference"></a>Справочник по диспетчеру терминалов TAPI 3



| Интерфейсы диспетчера терминалов                      | Описание                                                                                                                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**иттерминалконтрол**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalcontrol)   | Предоставляет методы для подключения и отключения терминала от графа фильтра, а также для запуска или завершения модуля подготовки отчетов.                                                                                 |
| [**иттерминалманажер**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager)   | Предоставляет методы для создания динамических терминалов.                                                                                                                                          |
| [**ITTerminalManager2**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager2) | Предоставляет методы для получения сведений о подключаемых классах терминалов, зарегистрированных в текущей системе. является производным от интерфейса [**иттерминалманажер**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager) . |
