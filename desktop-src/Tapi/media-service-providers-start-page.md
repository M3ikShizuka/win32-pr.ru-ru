# <a name="media-service-providers"></a>Поставщики служб мультимедиа

## <a name="purpose"></a>Назначение

Поставщик службы мультимедиа (MSP) позволяет приложению управлять носителями для конкретного транспортного механизма. MSP всегда сопряжен с поставщиком услуг телефонии (TSP).

Интерфейс поставщика службы мультимедиа (МСПИ) — это набор интерфейсов и методов, реализованных MSP, чтобы обеспечить управление приложениями через транспорт носителей во время сеанса связи. MSP обрабатывает механизмы для конкретных устройств и протоколов, необходимые для внедрения этих элементов управления, и взаимодействует со связанным TSP или приложением с помощью методов, предоставляемых в МСПИ.

## <a name="developer-audience"></a>Аудитория разработчиков

Требуется знание COM. Опыт разработки с помощью телекоммуникаций или других приложений телефонии полезен, но не нужен.

## <a name="run-time-requirements"></a>Требования к среде выполнения

МСПИ позволяет разрабатывать поставщики служб мультимедиа для определенных протоколов или устройств, работающих в операционных системах Windows Server 2003, Windows XP и Windows 2000.
