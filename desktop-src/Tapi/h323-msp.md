# <a name="h323-msp"></a>H323 MSP

\[он не доступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Протокол H323 MSP (H323msp.dll), если он связан с его TSP (H323. TSP), позволяет приложению создавать и принимать видеои звуковые вызовы с помощью протокола H. 323 для IP-телефонии.

протокол h323 MSP усовершенствован в Windows 2000 с пакетом обновления 1 (SP1) для обеспечения взаимодействия между клиентами H323 и SDP. Теперь он предоставляет интерфейсы [**IH323LineEx**](ih323lineex.md) и [**икэйфрамеконтрол**](ikeyframecontrol.md) .

файл_msp MSP устанавливается с Windows 2000.
