# <a name="h323-msp"></a>H323 MSP

\[В Windows Vista, Windows Server 2008 и последующих версиях операционной системы недоступно использование H323 MSP. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Протокол H323 MSP (H323msp.dll), если он связан с его TSP (H323. TSP), позволяет приложению создавать и принимать видеои звуковые вызовы с помощью протокола H. 323 для IP-телефонии.

Протокол H323 MSP был усовершенствован в Windows 2000 с пакетом обновления 1 (SP1) для обеспечения взаимодействия между клиентами H323 и SDP. Теперь он предоставляет интерфейсы [**IH323LineEx**](ih323lineex.md) и [**икэйфрамеконтрол**](ikeyframecontrol.md) .

Файл_MSP MSP устанавливается вместе с Windows 2000.
