# <a name="ittimecollection-interface"></a>Интерфейс Иттимеколлектион

\[встречи и элементы управления встречными IP-телефонными соединениями недоступны для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **иттимеколлектион** — это интерфейс, зависящий от поставщика, для объекта типа "объект" BLOB-объектов для Конференции по протоколу дескрипторов сеансов (SDP). Этот интерфейс обеспечивает доступ к [**иттиме**](ittime.md) интерфейсам по индексу, а также включает создание и удаление компонентов времени. Метод [**итсдп:: Get \_ тимеколлектион**](itsdp-get-timecollection.md) создает интерфейс **иттимеколлектион** .

## <a name="members"></a>Элементы

Интерфейс **иттимеколлектион** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Иттимеколлектион** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иттимеколлектион** содержит следующие методы.



| Метод                                                           | Описание                                                                                                           |
|:-----------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------|
| [**Создания**](ittimecollection-create.md)                        | Создает компонент [**иттиме**](ittime.md) .<br/>                                                             |
| [**Удален**](ittimecollection-delete.md)                        | Удаляет компонент [**иттиме**](ittime.md) .<br/>                                                             |
| [**получить \_ \_ NewEnum**](ittimecollection-get--newenum.md)          | Возвращает перечислитель для коллекции.<br/>                                                                  |
| [**получить \_ число**](ittimecollection-get-count.md)                 | Возвращает число элементов в коллекции.<br/>                                                                        |
| [**получить \_ енумератиониф**](ittimecollection-get-enumerationif.md) | Возвращает интерфейс перечисления [**иенумтиме**](ienumtime.md) , перечисляющий [**иттиме**](ittime.md).<br/> |
| [**получить \_ элемент**](ittimecollection-get-item.md)                   | При наличии индекса Возвращает элемент в коллекции.<br/>                                                         |
