# <a name="manual-terminal-selection"></a>Выбор терминала вручную

Если правила выбора по умолчанию не соответствуют потребностям приложения, приложение имеет возможность выбрать терминал, как всегда имеет значение. это значит, что он может использовать [**итстреам**](/windows/win32/api/tapi3if/nn-tapi3if-itstream) для перечисления потоков, имеющихся в вызове, и выбора терминала в соответствующих потоках (или, в случае с многодорожкными терминалами, создавать дорожки для потоков и выбрать созданные записи в потоках).
