# <a name="itparticipantsubstreamcontrol-interface"></a>Интерфейс ИтпартиЦипантсубстреамконтрол

\[**ИтпартиЦипантсубстреамконтрол** недоступен для использования в Windows Vista, windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **итпартиЦипантсубстреамконтрол** реализуется с помощью MSP-ипконф. Этот интерфейс предоставляется для объекта вызова. Этот интерфейс предоставляет методы, позволяющие приложению обнаруживать или контролировать сопоставление подпотока с участником. Интерфейс **итпартиЦипантсубстреамконтрол** создается путем вызова **QueryInterface** в [**иткаллинфо**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallinfo).

## <a name="members"></a>Элементы

Интерфейс **итпартиЦипантсубстреамконтрол** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **ИтпартиЦипантсубстреамконтрол** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **итпартиЦипантсубстреамконтрол** содержит следующие методы.



| Метод                                                                                              | Описание                                                     |
|:----------------------------------------------------------------------------------------------------|:----------------------------------------------------------------|
| [**получить \_ партиЦипантфромсубстреам**](itparticipantsubstreamcontrol-get-participantfromsubstream.md) | Возвращает участников, связанных с данным подпотоком.<br/> |
| [**получить \_ субстреамфромпартиЦипант**](itparticipantsubstreamcontrol-get-substreamfromparticipant.md) | Возвращает подпотоки, связанные с данным участником.<br/> |
| [**свитчтерминалтосубстреам**](itparticipantsubstreamcontrol-switchterminaltosubstream.md)        | Задает участника в подпотоке.<br/>                   |
