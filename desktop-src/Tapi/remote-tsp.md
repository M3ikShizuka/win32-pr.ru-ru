# <a name="remote-tsp"></a>Удаленный TSP

Удаленный TSP (Ремотесп. TSP) предоставляет доступ к ресурсам связи, не локальным по отношению к текущему компьютеру, например УАТС. Если удаленный TSP настроен для указания на указанный сервер, то ресурсы отображаются в локальном приложении, как если бы они находились на компьютере пользователя.

Tcmsetup — это программа командной строки, которая позволяет настраивать удаленный TSP. Дополнительные сведения об этой служебной программе см. в справке Windows или в комплекте Windows Resource Kit.

Удаленный TSP устанавливается вместе с операционными системами Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Me, Windows 98 и Windows 95.
