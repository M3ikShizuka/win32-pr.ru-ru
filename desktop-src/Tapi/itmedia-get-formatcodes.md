# <a name="itmediaget_formatcodes-method"></a>Метод Итмедиа:: Get \_ форматкодес

\[ В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **Get \_ форматкодес** получает список кодов формата полезных данных мультимедиа. Вариант возвращает SAFEARRAY для **BSTR** s. Каждая **BSTR** в этом массиве является строкой кода форматирования.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_FormatCodes(
  [out] VARIANT *pVal
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Pval* \[ заполняет\]
</dt> <dd>

Массив кодов формата полезных данных мультимедиа.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Код возврата                                                                                   | Описание                                                     |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Метод успешно выполнен.<br/>                                    |
| <dl> <dt>**\_указатель E**</dt> </dl>     | Параметр *Pval* не является допустимым указателем.<br/>         |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти для выполнения операции.<br/> |
| <dl> <dt>**\_Ошибка E**</dt> </dl>        | Незаданная ошибка.<br/>                                   |
| <dl> <dt>**E \_ нотимпл**</dt> </dl>     | Этот метод еще не реализован.<br/>                  |



 

## <a name="remarks"></a>Комментарии

При указании списка форматов полезных данных это означает, что все эти форматы могут использоваться в сеансе, но первый из этих форматов является форматом по умолчанию для сеанса. Если транспортным протоколом является RTP, то коды форматов являются типами полезных данных RTP.
