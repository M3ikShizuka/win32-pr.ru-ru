# <a name="media-streaming-terminal-mst"></a>Терминал потоковой передачи мультимедиа (MST)

терминал потоковой передачи мультимедиа (MST) — это динамический терминал, основанный на использовании фильтров DirectShow. Запись MSP, написанная с помощью [базовых классов MSP TAPI 3](tapi-3-msp-base-classes.md) , будет включать MST, но авторы MSP могут реализовать его без использования базовых классов.

Чтобы вызвать создание MST, приложение выполняет вызов [**иттерминалсуппорт:: креатетерминал**](/windows/win32/api/tapi3if/nf-tapi3if-itterminalsupport-createterminal) , используя *птерминалкласс* , для которого задано значение CLSID \_ медиастреамтерминал.

Затем интерфейсы [**итаммедиаформат**](/windows/win32/api/tapi3/nn-tapi3-itammediaformat) и [**италлокаторпропертиес**](/windows/win32/api/tapi3/nn-tapi3-itallocatorproperties) предоставляются в терминале, что позволяет приложению настраивать производительность потоковой передачи. Эти интерфейсы объявляются в Tapi3. h.

для реализации и использования MST необходимо знание фильтров DirectShow и управления графами фильтров. ознакомьтесь с DirectShow материалом в узле "графические и мультимедийные службы" пакета средств разработки программного обеспечения (SDK) платформы. Ссылка на потоковую передачу мультимедиа будет особенно полезна для авторов MSP.
