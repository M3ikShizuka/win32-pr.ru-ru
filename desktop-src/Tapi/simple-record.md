# <a name="simple-record"></a>Простая запись

Чтобы выполнить простую запись потоков, выполните следующие действия.

-   Выбор терминала записи файлов на уровне вызовов
-   Создайте терминал записи файлов.
-   Задайте свойства — имя файла и тип хранилища.
-   Выберите терминал записи файлов в вызове.
-   Вызовите метод [**Start**](/windows/desktop/api/tapi3if/nf-tapi3if-itmediacontrol-start) в терминале записи файлов, чтобы начать запись потоков.
