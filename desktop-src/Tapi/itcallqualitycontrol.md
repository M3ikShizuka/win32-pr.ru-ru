# <a name="itcallqualitycontrol-interface"></a>Интерфейс Иткаллкуалитиконтрол

\[этот интерфейс недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **иткаллкуалитиконтрол** предоставляет методы, позволяющие приложению получать и задавать параметры для контроля качества вызова.

Этот интерфейс реализуется с помощью [ИПКОНФ MSP](ipconf-msp.md) и [H323 MSP](h323-msp.md). Он предоставляется только в том случае, если вызов использует эти поставщики служб.

## <a name="members"></a>Элементы

Интерфейс **иткаллкуалитиконтрол** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иткаллкуалитиконтрол** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иткаллкуалитиконтрол** содержит следующие методы.



| Метод                                            | Описание                                                                                                     |
|:--------------------------------------------------|:----------------------------------------------------------------------------------------------------------------|
| [**Получить**](itcallqualitycontrol-get.md)           | Возвращает значение заданного [Свойства контроля качества вызова](callqualityproperty.md).<br/>                  |
| [**GetRange**](itcallqualitycontrol-getrange.md) | Возвращает диапазон допустимых значений для заданного [Свойства контроля качества вызова](callqualityproperty.md).<br/> |
| [**Параметр**](itcallqualitycontrol-set.md)           | Задает значение заданного [Свойства контроля качества вызова](callqualityproperty.md).<br/>                  |
