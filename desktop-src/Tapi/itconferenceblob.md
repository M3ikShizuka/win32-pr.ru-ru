# <a name="itconferenceblob-interface"></a>Интерфейс Итконференцеблоб

\[ В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **итконференцеблоб** управляет текстовым описанием Конференции. Интерфейс должен быть независимым от формата и синтаксиса описания Конференции. Чтобы управлять конкретными аспектами описания конференции, приложение должно запросить другой интерфейс. В настоящее время поддерживаются только описания конференций SDP. приложение должно использовать **QueryInterface** для [**итсдп**](itsdp.md) для доступа к различным полям описания конференции SDP. Интерфейс **итконференцеблоб** создается путем вызова **QueryInterface** в [**итдиректорйобжект**](/windows/desktop/api/Rend/nn-rend-itdirectoryobject).

## <a name="members"></a>Элементы

Интерфейс **итконференцеблоб** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Итконференцеблоб** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **итконференцеблоб** содержит следующие методы.



| Метод                                                             | Описание                                                                                                                                   |
|:-------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------|
| [**получить \_ кодировку**](itconferenceblob-get-characterset.md)     | Возвращает признак [**\_ \_ набора символов большого двоичного**](blob-character-set.md) объекта в кодировке ASCII, Unicode и т. д.<br/> |
| [**получить \_ конференцеблоб**](itconferenceblob-get-conferenceblob.md) | Получает указатель на большой двоичный объект конференции.<br/>                                                                                             |
| [**Ini**](itconferenceblob-init.md)                              | Инициализирует Конференц-большой двоичный объект.<br/>                                                                                                   |
| [**сетконференцеблоб**](itconferenceblob-setconferenceblob.md)    | Фиксирует изменения в большом двоичном объекте Конференции.<br/>                                                                                            |
