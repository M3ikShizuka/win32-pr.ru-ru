# <a name="itparticipantsubstreamcontrolget_participantfromsubstream-method"></a>Метод ИтпартиЦипантсубстреамконтрол:: Get \_ партиЦипантфромсубстреам

\[**получить \_ ПартиЦипантфромсубстреам** недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **Get \_ партиЦипантфромсубстреам** позволяет приложению обнаруживать, какие участники связаны с данным подпотоком.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_ParticipantFromSubStream(
  [in]  ITSubStream   *pITSubStream,
  [out] ITParticipant **ppParticipant
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*питсубстреам* \[ окне\]
</dt> <dd>

Указатель на интерфейс [**итсубстреам**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstream) .

</dd> <dt>

*пппартиЦипант* \[ заполняет\]
</dt> <dd>

Указатель на массив указателей интерфейса [**итпартиЦипант**](itparticipant.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Код возврата                                                                                     | Описание                                                                        |
|-------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>            | Метод успешно выполнен.<br/>                                                       |
| <dl> <dt>**\_указатель E**</dt> </dl>       | Параметр *питсубстреам* или *пппартиЦипант* не является допустимым указателем.<br/> |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>    | Параметр *питсубстреам* не указывает на допустимый интерфейс.<br/>         |
| <dl> <dt>**\_нежелательные \_ элементы TAPI**</dt> </dl> | Нет участников, связанных с подпотоком.<br/>                |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl>   | Недостаточно памяти для выполнения операции.<br/>                    |
