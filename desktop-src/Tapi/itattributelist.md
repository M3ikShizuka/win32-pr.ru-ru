# <a name="itattributelist-interface"></a>Интерфейс Итаттрибутелист

\[ В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **итаттрибутелист** предоставляет методы для получения и установки неинтерпретируемых атрибутов. По положению строк атрибута в пакете протокола дескрипторов сеанса (см. RFC 2327), методы предполагают, что все строки атрибутов существуют непосредственно перед тем, как были указаны атрибуты носителя, и после всех общих атрибутов. Интерфейс **итаттрибутелист** создается путем вызова **QueryInterface** в [**итдиректорйобжект**](/windows/desktop/api/Rend/nn-rend-itdirectoryobject).

## <a name="members"></a>Элементы

Интерфейс **итаттрибутелист** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Итаттрибутелист** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **итаттрибутелист** содержит следующие методы.



| Метод                                                          | Описание                                             |
|:----------------------------------------------------------------|:--------------------------------------------------------|
| [**Включить**](itattributelist-add.md)                              | Добавляет атрибут по указанному индексу.<br/>   |
| [**Удалить**](itattributelist-delete.md)                        | Удаляет атрибут по указанному индексу.<br/> |
| [**получить \_ аттрибутелист**](itattributelist-get-attributelist.md) | Возвращает список атрибутов.<br/>                 |
| [**получить \_ число**](itattributelist-get-count.md)                 | Возвращает количество атрибутов.<br/>               |
| [**получить \_ элемент**](itattributelist-get-item.md)                   | Возвращает атрибут, заданный индексом.<br/>   |
| [**разместить \_ аттрибутелист**](itattributelist-put-attributelist.md) | Задает список атрибутов.<br/>                 |
