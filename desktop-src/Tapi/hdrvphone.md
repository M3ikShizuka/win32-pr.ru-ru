# <a name="hdrvphone"></a>хдрвфоне

Тип данных **хдрвфоне** представляет непрозрачный маркер поставщика услуг для структуры данных телефона. Поставщик услуг должен разрешить значение этого типа в ссылку на соответствующий экземпляр структуры данных.
