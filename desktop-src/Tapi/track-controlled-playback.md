# <a name="track-controlled-playback"></a>Воспроизведение Track-Controlled

Для выполнения воспроизведения с контролем мониторинга выполните следующие действия.

-   Выберите Tracking терминалов на потоки.
-   Создайте терминал воспроизведения файлов.
-   Задайте свойства — имя файла и тип хранилища.
-   С помощью метода в терминале воспроизведения файлов перечислите терминалы записи файлов.
-   Перечисление потоков в вызове TAPI.
-   Выберите терминал файловых записей в потоке.
-   Вызовите метод [**Start**](/windows/desktop/api/tapi3if/nf-tapi3if-itmediacontrol-start) в терминале воспроизведения файлов, чтобы начать воспроизведение записанных данных в потоки TAPI.
