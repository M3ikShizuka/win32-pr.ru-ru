# <a name="linetonemode_-constants"></a>\_Константы линетонемоде

**\_ Константы линетонемоде** описывают различные варианты выбора, используемые при создании тонов линий.

<dl> <dt>

<span id="LINETONEMODE_BEEP"></span><span id="linetonemode_beep"></span>**\_звуковой сигнал линетонемоде**
</dt> <dd> <dl> <dt>



Гудок — это звуковой сигнал, который используется для объявления начала записи. Точное определение определяется поставщиком службы.


</dt> </dl> </dd> <dt>

<span id="LINETONEMODE_BILLING"></span><span id="linetonemode_billing"></span>**\_выставление счетов линетонемоде**
</dt> <dd> <dl> <dt>



Гудок — это гудок информации о выставлении счетов, например сигнал запроса кредитной карты. Точное определение определяется поставщиком службы.


</dt> </dl> </dd> <dt>

<span id="LINETONEMODE_BUSY"></span><span id="linetonemode_busy"></span>**ЛИНЕТОНЕМОДЕ \_ занят**
</dt> <dd> <dl> <dt>



Гудок — это занятый тон. Точное определение определяется поставщиком службы.


</dt> </dl> </dd> <dt>

<span id="LINETONEMODE_CUSTOM"></span><span id="linetonemode_custom"></span>**ЛИНЕТОНЕМОДЕ \_ настраиваемый**
</dt> <dd> <dl> <dt>



Тон — это пользовательский тон, определяемый по частоте его компонентов, типа [**линеженератетоне**](/windows/desktop/api/Tapi/ns-tapi-linegeneratetone).


</dt> </dl> </dd> <dt>

<span id="LINETONEMODE_RINGBACK"></span><span id="linetonemode_ringback"></span>**ЛИНЕТОНЕМОДЕ \_ звонка**
</dt> <dd> <dl> <dt>



Тон — звонка тон. Точное определение определяется поставщиком службы.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Remarks

Старшие 16 бит можно назначить для расширений для конкретных устройств. 16 младших разрядов зарезервированы.

Эти константы используются для определения тонов, которые будут создаваться в нестандартном виде при вызове удаленной стороны. Обратите внимание, что при обнаружении тона нестандартных тонов эти константы не используются.
