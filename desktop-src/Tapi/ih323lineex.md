# <a name="ih323lineex-interface"></a>Интерфейс IH323LineEx

\[**IH323LineEx** недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **IH323LineEx** реализуется с помощью интерфейса [H323 MSP](h323-msp.md) и доступен только для объектов Address в H. 323. Этот интерфейс предоставляет методы, обеспечивающие создание и обработку терминалов, которые могут взаимодействовать между клиентами H323 и SDP.

> [!Note]  
> Все методы в этом интерфейсе должны вызываться только после регистрации для входящих вызовов по этому адресу.

 

## <a name="members"></a>Элементы

Интерфейс **IH323LineEx** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **IH323LineEx** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **IH323LineEx** содержит следующие методы.



| Метод                                                                                 | Описание                                                              |
|:---------------------------------------------------------------------------------------|:-------------------------------------------------------------------------|
| [**сеталиас**](ih323lineex-setalias.md)                                               | Настраивает псевдоним H. 323 по умолчанию для адреса.<br/>             |
| [**сетдефаулткапабилитипреферренце**](ih323lineex-setdefaultcapabilitypreferrence.md) | Настройка веса предпочтений для возможностей по умолчанию.<br/> |
| [**SetExternalT120Address**](ih323lineex-setexternalt120address.md)                   | Настраивает внешний адрес T. 120 для обмена данными.<br/>       |
