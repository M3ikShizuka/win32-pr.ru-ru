# <a name="itsdpget_timecollection-method"></a>Метод Итсдп:: Get \_ тимеколлектион

\[встречи и элементы управления встречными IP-телефонными соединениями недоступны для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **Get \_ тимеколлектион** получает указатель на интерфейс [**иттимеколлектион**](ittimecollection.md) для Конференции.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_TimeCollection(
  [out] ITTimeCollection **ppTimeCollection
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пптимеколлектион* \[ заполняет\]
</dt> <dd>

Указатель на [**иттимеколлектион**](ittimecollection.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                                                           | Значение                                                                                     |
|---------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                                            | Метод успешно выполнен.<br/>                                                                |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>                                    | Параметр *пптимеколлектион* не является допустимым указателем.<br/>                         |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl>                                   | Недостаточно памяти для выполнения операции.<br/>                             |
| <dl> <dt>**\_Ошибка E**</dt> </dl>                                          | Незаданная ошибка.<br/>                                                               |
| <dl> <dt>**HRESULT \_ из \_ \_ кода ошибки (ошибка: \_ недопустимые \_ данные)**</dt> </dl> | Произошла внутренняя ошибка, обычно из-за сбоя предыдущего метода.<br/> |
| <dl> <dt>**E \_ нотимпл**</dt> </dl>                                       | Этот метод еще не реализован.<br/>                                              |



 

## <a name="remarks"></a>Remarks

[**Иттимеколлектион**](ittimecollection.md) указатель можно также получить с помощью **QueryInterface**.

TAPI вызывает метод **AddRef** в интерфейсе [**иттимеколлектион**](ittimecollection.md) , возвращенном методом **итсдп:: Get \_ тимеколлектион**. Приложение должно вызвать **выпуск** в интерфейсе **иттимеколлектион** , чтобы освободить ресурсы, связанные с ним.
