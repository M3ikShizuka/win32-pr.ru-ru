# <a name="acd-agent-client"></a>Клиент агента ACD

Агент центра обработки вызовов отвечает за получение и обработку входящих вызовов и, в то же время, выполнение исходящих вызовов клиентов или клиентов. Клиентское приложение агента ACD обеспечивает обмен данными с приложением [ACD proxy](acd-proxy.md) , предоставляя ему такие сведения, как доступность агента для вызовов и типы вызовов, которые может обработаться агентом. На следующей схеме показано, как объекты TAPI отображаются в агенте.
