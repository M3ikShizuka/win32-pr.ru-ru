# <a name="extended-telephony-phone-functions"></a>Функции расширенного телефонии телефонии

Расширенные службы телефонов (или службы Device-Specific Phone) включают все расширения для TAPI, определенного поставщиком услуг. TAPI определяет механизм, позволяющий поставщикам поставщиков услуг расширить TAPI, используя расширения для конкретных устройств. TAPI определяет только механизм расширения и обеспечивает доступ к модулям, специфичным для устройств. TAPI не определяет их поведение, которое полностью определяется поставщиком услуг.

TAPI состоит из скалярных и битовых определений констант данных, структур данных, функций и сообщений. Определяются процедуры, позволяющие поставщику расширить большинство из них, как описано в следующих разделах.

-   [Константы скалярных данных](scalar-data-constants.md)
-   [Константы данных флагов битов](bit-flag-data-constants.md)
-   [Функции и сообщения](functions-and-messages.md)
