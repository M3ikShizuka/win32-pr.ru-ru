# <a name="itmediacollectionget_enumerationif-method"></a>Метод Итмедиаколлектион:: Get \_ енумератиониф

\[ В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **Get \_ енумератиониф** получает указатель на интерфейс перечисления мультимедиа.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_EnumerationIf(
  [out] IEnumMedia **pVal
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Pval* \[ заполняет\]
</dt> <dd>

Указатель на интерфейс [**иенуммедиа**](ienummedia.md) для требуемого элемента.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                         | Значение                                                         |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Метод успешно выполнен.<br/>                                    |
| <dl> <dt>**\_указатель E**</dt> </dl>     | Параметр *Pval* не является допустимым указателем.<br/>         |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти для выполнения операции.<br/> |
| <dl> <dt>**\_Ошибка E**</dt> </dl>        | Незаданная ошибка.<br/>                                   |
| <dl> <dt>**E \_ нотимпл**</dt> </dl>     | Этот метод еще не реализован.<br/>                  |



 

## <a name="remarks"></a>Комментарии

Этот метод является взаимозаменяемым с [**Get \_ \_ NewEnum**](itmediacollection-get--newenum.md) , за исключением того, что он возвращает [**иенуммедиа**](ienummedia.md) вместо **IUnknown**.

TAPI вызывает метод **AddRef** в интерфейсе [**иенуммедиа**](ienummedia.md) , возвращенном методом **итмедиаколлектион:: Get \_ енумератионлф**. Приложение должно вызвать **выпуск** в интерфейсе **иенуммедиа** , чтобы освободить ресурсы, связанные с ним.
