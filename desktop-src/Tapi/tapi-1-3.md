# <a name="tapi-13"></a>TAPI 1,3

Интерфейс TAPI 1,3 (иногда называемый TAPI 1,03) был первой выпущенной версией спецификации TAPI. Системные двоичные файлы TAPI 1,3 (Tapi.dll и др.) находятся в пакете SDK для TAPI 1,0, который можно найти только в подписке Microsoft Developer Network (MSDN) уровня II. Системные двоичные файлы (которые находятся в каталоге Redist) являются распространяемыми, но предназначены для запуска только в Microsoft Windows 3,1 и Windows для рабочих групп 3,11. Они не поддерживаются и не будут работать в любой версии 32-разрядной ОС Windows.

Заголовки и библиотеки, используемые для разработки приложения TAPI 1,3, доступны только в пакете SDK для TAPI 1,0 и поддерживают только 16-разрядную разработку. На самом деле, библиотеки TAPI 1,3 являются единственными 16-разрядными библиотеками TAPI, которые доступны и должны использоваться для всех 16-разрядных приложений, независимо от целевой платформы или целевой версии.

Системные двоичные файлы TAPI 1,3 поддерживают только приложения Win16, TAPI 1,3 специалистами и приложений.
