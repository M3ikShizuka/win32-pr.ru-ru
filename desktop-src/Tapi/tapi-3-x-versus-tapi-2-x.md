# <a name="tapi-3x-vs-tapi-2x"></a>TAPI 3. x и TAPI 2. x

интерфейс API телефонии на основе COM, TAPI 3. x, доступен начиная с Windows XP. Интерфейс TAPI 3. x предоставляет значительно улучшенные средства разработки для современного программирования обмена данными, где "вызов" может быть видеопотоком в сети на основе IP-адресов, и в сеансе не участвует набор телефонных номеров. В целом, разработчики приложений могут найти TAPI 3. x обеспечивает лучшую поддержку для реализации гибких средств управления взаимодействием.

Тем не менее TAPI 2. x может быть верным выбором в некоторых ситуациях. Если для устаревшего приложения требуются незначительные изменения или существуют устаревшие платформы для поддержки, преобразование в API COM может быть неэкономичным.
