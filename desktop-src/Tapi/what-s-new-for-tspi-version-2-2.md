# <a name="whats-new-for-tspi-version-22"></a>Новые возможности для ТСПИ версии 2,2

Изменения ТСПИ версии 2,2 состоят в основном из нескольких структур данных, имеющих дополнительные элементы. Большая часть этих структур определена в TAPI. h и используется как приложениями TAPI, так и поставщиками услуг как средство передачи информации.

Функции и сообщения, которые были впервые или изменены для ТСПИ версии 2,2, следующие:

-   [**LINE_AGENTSESSIONSTATUS**](./line-agentsessionstatus.md)
-   [**LINE_QUEUESTATUS**](./line-queuestatus.md)
-   [**LINE_AGENTSTATUSEX**](./line-agentstatusex.md)
-   [**LINE_GROUPSTATUS**](./line-groupstatus.md)
-   [**LINE_PROXYSTATUS**](./line-proxystatus.md)

**LINEADDRCAPFLAGS_ACDGROUP** член [LINEADDRCAPFLAGS_ констант](./lineaddrcapflags--constants.md)

**LINEERR_INVALAGENTSESSIONSTATE** и **LINEERR_DISCONNECTED** члены [констант LINEERR_](./lineerr--constants.md).

-   [Константы LINEAGENTSTATEEX_](./lineagentstateex--constants.md)
-   [Константы LINEAGENTSTATUSEX_](./lineagentstatusex--constants.md)
-   [Константы LINEAGENTSESSIONSTATE_](./lineagentsessionstate--constants.md)
-   [Константы LINEAGENTSESSIONSTATUS_](./lineagentsessionstatus--constants.md)
-   [Константы LINEQUEUESTATUS_](./linequeuestatus--constants.md)
-   [Константы LINEGROUPSTATUS_](./linegroupstatus--constants.md)
-   [Константы LINEPROXYSTATUS_](./lineproxystatus--constants.md)
-   [Константы LINEPROXYREQUEST_](./lineproxyrequest--constants.md)
