# <a name="cmsparray"></a>кмспаррай

Шаблон Кмспаррай реализует интеллектуальный массив, который будет расти по запросу. Он предназначен для хранения небольших массивов с простыми типами данных. У него нет критической секции. Его единственными зависимостями являются **realloc** и **memmove**. Он используется внутренним образом для сохранения списков указателей объектов. Он похож на реализацию **ксимплеаррай** в ATL, но более эффективен для первоначального выделения.

Этот шаблон определен в Мспутилс. h.
