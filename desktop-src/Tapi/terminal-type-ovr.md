# <a name="terminal-type"></a>Тип терминала

**Тип терминала** определяет, является ли терминал статическим или динамическим. Например, модем является статическим терминалом, а окно видео на экране компьютера создается при необходимости и уничтожается, когда его полезность.

Эта концепция в основном используется приложениями TAPI 3.

**Интерфейс TAPI 3. x:** См. раздел [**иттерминал:: Get \_ терминалтипе**](/windows/win32/api/tapi3if/nf-tapi3if-itterminal-get_terminaltype), [**\_ тип терминала**](/windows/desktop/api/Tapi3if/ne-tapi3if-terminal_type).
