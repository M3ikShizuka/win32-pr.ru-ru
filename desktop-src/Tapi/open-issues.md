# <a name="open-issues"></a>Нерешенные проблемы

-   До окончательного выпуска Windows 2000 путь завершения вызова может немного измениться, чтобы упростить завершение работы вызова асинхронной операции (помогает защититься от взаимоблокировок для MSP, поддерживающих видео).
-   Методы и члены класса рабочего потока, базовых классов терминала и производных статических терминалов задокументированы только самими файлами заголовков.
