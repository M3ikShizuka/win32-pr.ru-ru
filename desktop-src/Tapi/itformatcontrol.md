# <a name="itformatcontrol-interface"></a>Интерфейс Итформатконтрол

\[этот интерфейс недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **итформатконтрол** предоставляет методы, позволяющие приложению получать сведения о формате приема или передачи для вызова.

Этот интерфейс реализуется с помощью интерфейса [H323 MSP](h323-msp.md) и предоставляется только в том случае, если вызов использует этого поставщика услуг.

## <a name="members"></a>Элементы

Интерфейс **итформатконтрол** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Итформатконтрол** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **итформатконтрол** содержит следующие методы.



| Метод                                                                     | Описание                                                                    |
|:---------------------------------------------------------------------------|:-------------------------------------------------------------------------------|
| [**жеткуррентформат**](itformatcontrol-getcurrentformat.md)               | Возвращает формат мультимедиа текущего потока.<br/>                        |
| [**жетнумберофкапабилитиес**](itformatcontrol-getnumberofcapabilities.md) | Возвращает число возможностей, связанных с текущим форматом.<br/> |
| [**жетстреамкапс**](itformatcontrol-getstreamcaps.md)                     | Возвращает возможности, связанные с данным индексом формата.<br/>         |
| [**релеасеформат**](itformatcontrol-releaseformat.md)                     | Освобождает структуру, связанную с форматом.<br/>                  |
| [**реордеркапабилитиес**](itformatcontrol-reordercapabilities.md)         | Задает новый порядок для возможностей форматирования.<br/>                           |
