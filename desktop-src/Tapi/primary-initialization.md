# <a name="primary-initialization"></a>Основная инициализация

Приложение TAPI должно вызывать операцию инициализации, которая предлагает ряд действий от TAPI и поставщиков услуг, которые настроили среду связи для приложения.

-   Инициализация является синхронной и не возвращает значение до завершения или сбоя операции.
-   Если ТАПИСРВ еще не запущен, TAPI запускает его.
-   TAPI настраивает подключение к процессу ТАПИСРВ.
-   ТАПИСВР загружает поставщики служб, указанные в реестре, и предлагает им инициализировать устройства, которые они поддерживают.

**Интерфейс TAPI 2. x:** Приложения выполняют инициализацию путем вызова [**линеинитиализикс**](/windows/win32/api/tapi/nf-tapi-lineinitializeexa).

**Интерфейс TAPI 3. x:** Приложения вызывают [**иттапи:: Initialize**](/windows/desktop/api/tapi3if/nf-tapi3if-ittapi-initialize).
