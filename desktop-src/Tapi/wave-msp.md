# <a name="wave-msp"></a>Звуковой MSP

Wave MSP (Wavemsp.dll) соответствует устаревшим специалистами, написанным до введения MSP в архитектуру потоковой передачи на основе протокола TAPI 3 в формате MSP. Некоторые устаревшие специалистами предоставляют идентификаторы устройств, которые были средством управления потоковой передачей в TAPI 2,1. Tapi3.dll использует звуковой MSP для любого TSP, который не имеет своего собственного MSP, но имеет звуковое устройство.

звуковой MSP устанавливается с Windows 2000.
