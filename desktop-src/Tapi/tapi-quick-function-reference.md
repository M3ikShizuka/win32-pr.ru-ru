# <a name="tapi-quick-function-reference"></a>Справочник по быстрой функции TAPI

Приведенная ниже сводка по функциям TAPI организована по категориям операций. Дополнительные сведения см. в обзоре [возможностей службы телефонии](./assisted-telephony-overview.md) и [уровня обслуживания](./tapi-levels-of-service.md).



| Раздел справки                                                                            | Описание                                                                                         |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [Справочник по базовым службам телефонии](basic-telephony-services-reference.md)                 | Функции, связанные с задачами телефонии на уровне POTS, например [**линемакекалл**](/windows/desktop/api/Tapi/nf-tapi-linemakecall). |
| [Справочник по дополнительным службам телефонии](supplementary-telephony-services-reference.md) | Функции с более сложными функциями, такими как переадресация вызовов.                                       |
| [Справочник по расширенным службам телефонии](extended-telephony-services-reference.md)           | Функции, которые обеспечивают доступ к функциям, зависящим от поставщика услуг, не определяются напрямую TAPI.   |
| [Справочник по службам телефонии](assisted-telephony-services-reference.md)           | Функции, устанавливающие голосовой вызов и вызовы мультимедиа, например, с помощью телефонного номеронабирателя.              |
