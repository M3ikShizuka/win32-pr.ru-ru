# <a name="a-telephony-api"></a>A (API телефонии)

<dl> <dt>

<span id="tapi2.analog_display_services_interface_adsi__tapgloss"></span><span id="TAPI2.ANALOG_DISPLAY_SERVICES_INTERFACE_ADSI__TAPGLOSS"></span>**Аналоговый интерфейс служб дисплеев (ADSI)**
</dt> <dd>

Беллкоре стандарт, определяющий протокол на потоке информации между коммутатором, сервером, системой электронной почты или бюро обслуживания, например, и телефоном, компьютером, терминалом данных или другим устройством, взаимодействующим с экраном. ADSI позволяет отображать текстовую версию диалога на экране во время телефонного подключения. Например, пользователь подключается к учетной записи голосовой почты и получает звуковое меню для функций голосовой почты, пока на экране отображается меню в текстовом виде.

</dd> <dt>

<span id="tapi2.assisted_telephony_tapgloss"></span><span id="TAPI2.ASSISTED_TELEPHONY_TAPGLOSS"></span>**Техническая поддержка телефонии**
</dt> <dd>

Набор функций, не только предназначенных для телефонной функциональности, но разработанных для обеспечения доступа к голосовым вызовам и вызовам мультимедиа для любого приложения. Дополнительные сведения см. в разделе [Техническая поддержка телефонии](./assisted-telephony-overview.md) в статье Обзор телефонии. См. раздел [*Базовая телефония*](b-tapgloss.md), [*Расширенная телефония*](e-tapgloss.md), [*добавочная телефония*](s-tapgloss.md).

</dd> <dt>

<span id="tapi2.asynchronous_tapgloss"></span><span id="TAPI2.ASYNCHRONOUS_TAPGLOSS"></span>**асинхронной**
</dt> <dd>

Метод передачи данных, позволяющий отправлять символы с неравномерными интервалами по строке, используя перед каждым символом начальную бит и заканчивая битом. См. раздел [*синхронность*](s-tapgloss.md).

</dd> </dl>
