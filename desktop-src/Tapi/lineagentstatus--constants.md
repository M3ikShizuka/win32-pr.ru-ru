# <a name="lineagentstatus_-constants"></a>\_Константы линеажентстатус

**\_ Константы линеажентстатус** перечисление состояния обновления элементов структуры [**линеажентстатус**](/windows/desktop/api/Tapi/ns-tapi-lineagentstatus) для агента.

<dl> <dt>

<span id="LINEAGENTSTATUS_ACTIVITY"></span><span id="lineagentstatus_activity"></span>**\_действие линеажентстатус**
</dt> <dd> <dl> <dt>



Элемент **двактивитид**, **двактивитисизе** или **двактивитйоффсет** в [**линеажентстатус**](/windows/desktop/api/Tapi/ns-tapi-lineagentstatus) обновлен.


</dt> </dl> </dd> <dt>

<span id="LINEAGENTSTATUS_ACTIVITYLIST"></span><span id="lineagentstatus_activitylist"></span>**ЛИНЕАЖЕНТСТАТУС \_ активитилист**
</dt> <dd> <dl> <dt>



[**Линеажентактивитилист**](/windows/desktop/api/Tapi/ns-tapi-lineagentactivitylist) обновлен. Приложение может вызвать [**линежетажентактивитилист**](/windows/desktop/api/Tapi/nf-tapi-linegetagentactivitylista) для получения обновленного списка.


</dt> </dl> </dd> <dt>

<span id="LINEAGENTSTATUS_CAPSCHANGE"></span><span id="lineagentstatus_capschange"></span>**ЛИНЕАЖЕНТСТАТУС \_ капсчанже**
</dt> <dd> <dl> <dt>



Возможности в [**линеаженткапс**](/windows/desktop/api/Tapi/ns-tapi-lineagentcaps) обновлены. Приложение может вызвать [**линежетаженткапс**](/windows/desktop/api/Tapi/nf-tapi-linegetagentcapsa) для получения обновленного списка.


</dt> </dl> </dd> <dt>

<span id="LINEAGENTSTATUS_GROUP"></span><span id="lineagentstatus_group"></span>**\_Группа линеажентстатус**
</dt> <dd> <dl> <dt>



[**Линеажентстатус**](/windows/desktop/api/Tapi/ns-tapi-lineagentstatus) обновлен.


</dt> </dl> </dd> <dt>

<span id="LINEAGENTSTATUS_GROUPLIST"></span><span id="lineagentstatus_grouplist"></span>**ЛИНЕАЖЕНТСТАТУС \_ грауплист**
</dt> <dd> <dl> <dt>



[**Линеажентграуплист**](/windows/desktop/api/Tapi/ns-tapi-lineagentgrouplist) обновлен. Приложение может вызвать [**линежетажентграуплист**](/windows/desktop/api/Tapi/nf-tapi-linegetagentgrouplista) для получения обновленного списка.


</dt> </dl> </dd> <dt>

<span id="LINEAGENTSTATUS_NEXTSTATE"></span><span id="lineagentstatus_nextstate"></span>**ЛИНЕАЖЕНТСТАТУС \_ NEXTSTATE**
</dt> <dd> <dl> <dt>



Элемент **двнекстстате** в [**линеажентстатус**](/windows/desktop/api/Tapi/ns-tapi-lineagentstatus) был обновлен.


</dt> </dl> </dd> <dt>

<span id="LINEAGENTSTATUS_STATE"></span><span id="lineagentstatus_state"></span>**\_состояние линеажентстатус**
</dt> <dd> <dl> <dt>



Элемент **двстате** в [**линеажентстатус**](/windows/desktop/api/Tapi/ns-tapi-lineagentstatus) был обновлен.


</dt> </dl> </dd> <dt>

<span id="LINEAGENTSTATUS_VALIDNEXTSTATES"></span><span id="lineagentstatus_validnextstates"></span>**ЛИНЕАЖЕНТСТАТУС \_ валиднекстстатес**
</dt> <dd> <dl> <dt>



Элемент **дввалиднекстстатес** в [**линеажентстатус**](/windows/desktop/api/Tapi/ns-tapi-lineagentstatus) был обновлен.


</dt> </dl> </dd> <dt>

<span id="LINEAGENTSTATUS_VALIDSTATES"></span><span id="lineagentstatus_validstates"></span>**ЛИНЕАЖЕНТСТАТУС \_ валидстатес**
</dt> <dd> <dl> <dt>



Элемент **дввалидстатес** в [**линеажентстатус**](/windows/desktop/api/Tapi/ns-tapi-lineagentstatus) был обновлен.


</dt> </dl> </dd> </dl>
