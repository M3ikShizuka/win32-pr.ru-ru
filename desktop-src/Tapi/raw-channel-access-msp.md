# <a name="raw-channel-access-msp"></a>Доступ к необработанному каналу MSP

MSP-канал необработанного доступа (Rcsmsp.dll) выполняет потоковую передачу через кспрокси в RCA, что позволяет выполнять потоковую передачу через устройства NDIS, ориентированные на подключения, такие как ATM и ISDN Card. Этот MSP-драйвер создается с помощью ндптсп, который является TSP, предоставленным для адаптеров глобальной сети NDIS. В выпуске Windows 2000 реализована только потоковая передача звука.

MSP для необработанного канала устанавливается вместе с Windows 2000.
