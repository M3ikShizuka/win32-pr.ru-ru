# <a name="trunk"></a>Trunk

Номер канала, через который направляется вызов. Этот элемент используется как для входящих, так и для исходящих вызовов.

Не все поставщики служб поддерживают использование этих сведений.

* * TAPI 2. x: * *[**линежеткаллинфо**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo), **двтрунк** [**линекаллинфо**](/windows/win32/api/tapi/ns-tapi-linecallinfo)

* * TAPI 3. x: * *[**иткаллинфо:: Get \_ каллинфолонг**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfolong), вызванный с помощью **\_ магистрального элемента CIL** из [**каллинфо \_ Long**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_long)
