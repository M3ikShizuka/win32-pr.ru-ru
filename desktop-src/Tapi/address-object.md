# <a name="address-object"></a>Объект Address

Объект Address представляет сущность, которая может принимать или принимать вызовы. Этот объект предоставляет интерфейсы и методы, позволяющие приложению выполнять ряд операций, например:

-   Определите, может ли указанный адрес поддерживать определенный набор потребностей типа носителя.
-   Перечислить вызовы, которые в настоящее время связаны с адресом.
-   Создание или переадресация вызовов.
-   Возвращает имя связанного поставщика услуг.
-   Если поставщик службы мультимедиа существует, получите указатели интерфейса, которые позволяют выполнять подробные операции терминала.
-   Получение и задание других сведений, например о том, ожидает ли сообщение.

Большинство адресов связаны с терминалом, но это не является единообразно. Например, удаленный TSP, предоставляющий доступ к оборудованию на сервере, будет иметь адрес на локальном компьютере, но не в терминале. Модем без динамиков также не имеет терминала, связанного с адресом.

Если у адреса нет связанного терминала, интерфейс [**иттерминалсуппорт**](/windows/win32/api/tapi3if/nn-tapi3if-itterminalsupport) не предоставляется в объекте.

В примере кода [Select a Address](select-an-address.md) показан базовый процесс получения указателя на объект адреса.

Список всех интерфейсов, связанных с объектом Address, см. в разделе [интерфейсы объекта адреса](address-object-interfaces.md) .
