# <a name="file-terminals"></a>Терминалы файловых терминалов

В TAPI 3,1 объект терминала расширен и включает файловый терминал. Файловые терминалы предоставляют простые средства для выполнения стандартных операций, например записи аудиопотока или потоков в файл или последующее воспроизведение записанного звука в поток.

Для работы с файловыми терминалами используются пять базовых объектов:

-   Терминал воспроизведения файлов
-   Терминал записи файлов
-   Запись о воспроизведении файлов
-   Запись файла
-   Событие файла

Объекты терминала файлов можно рассматривать как абстракцию для хранилища, который обычно является файлом. Дорожки соответствуют отдельным потокам звуковых данных в этом хранилище, например при записи голоса.

Дополнительные сведения о файловых терминалах см. в следующих статьях:

-   [Терминал воспроизведения файлов и терминал записи файлов](file-playback-terminal-and-file-recording-terminal.md)
-   [Мониторинг терминалов](track-terminals.md)

Дополнительные сведения и примеры кода и списки задач, связанные с файловыми терминалами, см. в разделе [Использование файловых терминалов](using-file-terminals.md).
