# <a name="call-and-media-controls-reference"></a>Справочник по вызовам и элементам управления мультимедиа

Справочные материалы по вызовам и управлению носителями содержат сведения о интерфейсах, методах, перечислителях и типах, используемых при настройке вызова и управлении.

Для получения сведений о DirectShow и модели COM можно также выполнить поиск в пакете SDK для платформы. TAPI 3. x использует фильтры DirectShow и диспетчеры фильтров для управления потоком мультимедиа после настройки подключения. Модель COM является базовой моделью программирования для TAPI 3. x, и в этом документе предполагается его знание.
