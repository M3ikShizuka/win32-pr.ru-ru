## <a name="remarks"></a>Комментарии

Версии **BSTR** константы протокола объявляются для использования Visual Basic приложений.

-   BSTR ТАПИПРОТОКОЛ \_ String \_ PSTN
-   BSTR ТАПИПРОТОКОЛ \_ String \_ H323
-   \_ \_ Многоадресная рассылка строк BSTR тапипротокол
