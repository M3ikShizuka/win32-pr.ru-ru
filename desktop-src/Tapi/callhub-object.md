# <a name="callhub-object"></a>Объект Каллхуб

Объект Каллхуб представляет стороннее представление многокомпонентного вызова. Связанные интерфейсы и методы получают и задают сведения о концентраторе, например, является ли он активным. С помощью объекта Каллхуб пользователь с необходимой защитой может обнаруживать и потенциально управлять другими участниками в вызове.

Объект Каллхуб не может быть создан напрямую приложением. Объект Каллхуб создается косвенно при получении входящего вызова через TAPI 3.

TAPI 3 использует поставщики услуг TAPI для предоставления необходимых сведений о вызовах для реализации с помощью объекта Каллхуб. Так как не все поставщики услуг предоставляют эти сведения, а не все оборудование поддерживает отслеживание концентратора вызовов, сведения о других пользователях в подключении могут быть ограничены или не существуют.

В примере [создания простого](create-a-simple-conference.md) примера кода конференции показано использование объекта каллхуб.

Список всех интерфейсов, связанных с объектом Каллхуб, см. в разделе [интерфейсы объекта каллхуб](callhub-object-interfaces.md) .
