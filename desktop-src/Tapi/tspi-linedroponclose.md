# <a name="tspi_linedroponclose"></a>ТСПИ \_ линедропонклосе

Функция **тспи \_ линедропонклосе** устарела. TAPI больше не удаляет вызовы автоматически, когда единственный владелец закрывает свой обработчик на устройстве линии, чтобы вместить распределенный контроль вызовов между несколькими клиентскими компьютерами в среде клиент/сервер.
