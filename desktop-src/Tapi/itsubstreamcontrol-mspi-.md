# <a name="itsubstreamcontrol-mspi"></a>Итсубстреамконтрол (МСПИ)

Интерфейс [**итсубстреамконтрол**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstreamcontrol) предоставляется в объекте потока, если имеется связанный MSP-объект и он реализует подпотоки. Методы этого интерфейса позволяют приложению создавать и удалять подпотоки.

Интерфейс [**итсубстреамконтрол**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstreamcontrol) реализуется с помощью MSP. Дополнительные сведения об этом интерфейсе см. в разделе **итсубстреамконтрол** в интерфейсе MSP.
