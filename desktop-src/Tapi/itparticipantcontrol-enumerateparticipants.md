# <a name="itparticipantcontrolenumerateparticipants-method"></a>Метод ИтпартиЦипантконтрол:: ЕнумератепартиЦипантс

\[**енумератепартиЦипантс** недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **енумератепартиЦипантс** перечисляет участников, связанных с текущей Конференцией. Этот метод предоставляется для приложений C и C++. клиентские приложения автоматизации, например написанные на Visual Basic, должны использовать метод [**get \_ участников**](itparticipantcontrol-get-participants.md) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT EnumerateParticipants(
  [out] IEnumParticipant **ppEnumParticipants
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ппенумпартиЦипантс* \[ заполняет\]
</dt> <dd>

Указатель на интерфейс [**иенумпартиЦипант**](ienumparticipant.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                         | Значение                                                               |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Метод успешно выполнен.<br/>                                          |
| <dl> <dt>**\_указатель E**</dt> </dl>     | Параметр *ппенумпартиЦипантс* не является допустимым указателем.<br/> |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти для выполнения операции.<br/>       |



 

## <a name="remarks"></a>Remarks

TAPI вызывает метод **AddRef** в интерфейсе [**иенумпартиЦипант**](ienumparticipant.md) , возвращенном **итпартиЦипантконтрол:: енумератепартиЦипантс**. Приложение должно вызвать **выпуск** в интерфейсе **иенумпартиЦипант** , чтобы освободить ресурсы, связанные с ним.
