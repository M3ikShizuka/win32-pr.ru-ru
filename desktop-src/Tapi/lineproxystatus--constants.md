# <a name="lineproxystatus_-constants"></a>\_Константы линепроксистатус

**\_ Константы линепроксистатус** указывают состояние прокси-сервера в строке, открытой в настоящее время приложением.

Список и описание всех возможных значений запроса прокси см. в разделе [**\_ константы линепроксирекуест**](lineproxyrequest--constants.md) .

<dl> <dt>

<span id="LINEPROXYSTATUS_ALLOPENFORACD"></span><span id="lineproxystatus_allopenforacd"></span>**ЛИНЕПРОКСИСТАТУС \_ аллопенфоракд**
</dt> <dd> <dl> <dt>



Теперь в этой строке открыты прокси для всех типов запросов прокси-сервера, необходимых для работы ACD с помощью TAPI версии 3,0 и выше.


</dt> </dl> </dd> <dt>

<span id="LINEPROXYSTATUS_CLOSE"></span><span id="lineproxystatus_close"></span>**ЛИНЕПРОКСИСТАТУС \_ Закрыть**
</dt> <dd> <dl> <dt>



Прокси-сервер закрыт.


</dt> </dl> </dd> <dt>

<span id="LINEPROXYSTATUS_OPEN"></span><span id="lineproxystatus_open"></span>**ЛИНЕПРОКСИСТАТУС \_ Открыть**
</dt> <dd> <dl> <dt>



Открыт новый прокси-сервер.


</dt> </dl> </dd> </dl>
