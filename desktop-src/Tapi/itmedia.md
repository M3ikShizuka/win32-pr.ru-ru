# <a name="itmedia-interface"></a>Интерфейс Итмедиа

\[ В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **итмедиа** представляет собой представление мультимедиа в протоколе дескриптора сеанса (см. RFC 2327). Этот интерфейс экспортирует методы для получения и задания основных свойств носителя, таких как тип. Методы [**итмедиаколлектион:: Get \_ Item**](itmediacollection-get-item.md) и [**итмедиаколлектион:: Create**](itmediacollection-create.md) создают интерфейс **итмедиа** .

## <a name="members"></a>Элементы

Интерфейс **итмедиа** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Итмедиа** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **итмедиа** содержит следующие методы.



| Метод                                                          | Описание                                                                                      |
|:----------------------------------------------------------------|:-------------------------------------------------------------------------------------------------|
| [**получить \_ форматкодес**](itmedia-get-formatcodes.md)             | Возвращает список кодов формата полезных данных мультимедиа.<br/>                                          |
| [**получить \_ MediaName**](itmedia-get-medianame.md)                 | Возвращает имя носителя.<br/>                                                                  |
| [**получить \_ медиатитле**](itmedia-get-mediatitle.md)               | Возвращает заголовок носителя.<br/>                                                                 |
| [**получить \_ нумпортс**](itmedia-get-numports.md)                   | Возвращает количество портов, необходимых для сеанса.<br/>                                      |
| [**получить \_ стартпорт**](itmedia-get-startport.md)                 | Возвращает 16-битное значение порта для первого порта.<br/>                                        |
| [**получить \_ транспортпротокол**](itmedia-get-transportprotocol.md) | Возвращает транспортный протокол.<br/>                                                          |
| [**разместить \_ форматкодес**](itmedia-put-formatcodes.md)             | Задает список кодов формата полезных данных мультимедиа.<br/>                                          |
| [**разместить \_ MediaName**](itmedia-put-medianame.md)                 | Задает имя носителя.<br/>                                                                  |
| [**разместить \_ медиатитле**](itmedia-put-mediatitle.md)               | Задает заголовок носителя.<br/>                                                                 |
| [**разместить \_ транспортпротокол**](itmedia-put-transportprotocol.md) | Задает транспортный протокол.<br/>                                                          |
| [**сетпортинфо**](itmedia-setportinfo.md)                      | Задает 16-битное значение порта для первого порта и число портов, необходимых для сеанса.<br/> |
