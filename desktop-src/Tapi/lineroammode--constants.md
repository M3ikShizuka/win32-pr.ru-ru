# <a name="lineroammode_-constants"></a>\_Константы линероаммоде

Константы побитового флага **линероаммоде \_** описывают состояние роуминга для линейного устройства.

<dl> <dt>

<span id="LINEROAMMODE_HOME"></span><span id="lineroammode_home"></span>**\_Домашняя страница линероаммоде**
</dt> <dd> <dl> <dt>



Линия подключена к узлу домашней сети.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_ROAMA"></span><span id="lineroammode_roama"></span>**ЛИНЕРОАММОДЕ \_ перемещаемый**
</dt> <dd> <dl> <dt>



Линия подключается к роумингу, и звонки начисляются соответствующим образом.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_ROAMB"></span><span id="lineroammode_roamb"></span>**ЛИНЕРОАММОДЕ \_ роамб**
</dt> <dd> <dl> <dt>



Линия подключена к перевозчику роуминга, а звонки начисляются соответствующим образом.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_UNAVAIL"></span><span id="lineroammode_unavail"></span>**ЛИНЕРОАММОДЕ \_ НЕсвободно**
</dt> <dd> <dl> <dt>



Режим перемещения недоступен и не будет известен.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_UNKNOWN"></span><span id="lineroammode_unknown"></span>**ЛИНЕРОАММОДЕ \_ неизвестный**
</dt> <dd> <dl> <dt>



Режим перемещения в настоящее время неизвестен, но может быть известен позже.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Remarks

Без расширяемости. Все 32 бит зарезервированы.
