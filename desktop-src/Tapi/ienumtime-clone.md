# <a name="ienumtimeclone-method"></a>Метод Иенумтиме:: Clone

\[ В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **clone** создает еще один перечислитель, который содержит то же состояние перечисления, что и текущий.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Clone(
  [out] IEnumTime **ppEnum
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ппенум* \[ заполняет\]
</dt> <dd>

Указатель на новый объект [**иенумтиме**](ienumtime.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                         | Значение                                                         |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Метод успешно выполнен.<br/>                                    |
| <dl> <dt>**\_указатель E**</dt> </dl>     | Параметр *ппенум* не является допустимым указателем.<br/>       |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти для выполнения операции.<br/> |
| <dl> <dt>**\_непредвиденная E**</dt> </dl>  | Сбой по неизвестным причинам.<br/>                          |



 

## <a name="remarks"></a>Комментарии

TAPI вызывает метод **AddRef** в интерфейсе [**иенумтиме**](ienumtime.md) , возвращенном **иенумтиме:: Clone**. Приложение должно вызвать **выпуск** в интерфейсе **иенумтиме** , чтобы освободить ресурсы, связанные с ним.
