# <a name="tapi-service-providers"></a>Поставщики услуг TAPI

Поставщики услуг TAPI — это уровень абстракции между приложениями TAPI и базовыми аппаратными и транспортными протоколами. [Поставщик услуг телефонии](telephony-service-providers-start-page.md) (TSP) предоставляет элементы управления звонками, а [поставщик услуг мультимедиа](./media-service-providers-start-page.md) (MSP) предоставляет подробный контроль над носителем вызова. Пара TSP/MSP может реализовывать возможности, необходимые для приложений, начиная от базовой телефонии до многоадресного мультимедиа-вещания.
