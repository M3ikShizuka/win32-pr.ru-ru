# <a name="c-telephony-api"></a>C (API телефонии)

[A](a-tapgloss.md) [B](b-tapgloss.md) C [D](d-tapgloss.md) [E](e-tapgloss.md) F G [р](h-tapgloss.md) [.](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N O [P](p-tapgloss.md) Q [R](r-tapgloss.md) [S](s-tapgloss.md) [T](t-tapgloss.md) [U](u-tapgloss.md) [V](v-tapgloss.md) [W](w-tapgloss.md) X Y Z

<dl> <dt>

<span id="tapi2.call_classification_tapgloss"></span><span id="TAPI2.CALL_CLASSIFICATION_TAPGLOSS"></span>**Классификация вызовов**
</dt> <dd>

Процесс TAPI, сообщающий об изменениях в типе потока мультимедиа (голоса, факса, модема данных и т. д.) для участвующих приложений.

</dd> <dt>

<span id="tapi2.call_progress_monitoring_tapgloss"></span><span id="TAPI2.CALL_PROGRESS_MONITORING_TAPGLOSS"></span>**мониторинг хода вызова**
</dt> <dd>

Процесс прослушивания звуковых тонов, таких как гудок, специальные информационные тона, сигналы занятости и звонка, для определения состояния звонка (его хода выполнения по сети).

</dd> <dt>

<span id="tapi2.call_redirection_tapgloss"></span><span id="TAPI2.CALL_REDIRECTION_TAPGLOSS"></span>**Перенаправление вызовов**
</dt> <dd>

Функция, которая позволяет приложению отклонения от вызова предложения к другому адресу без предварительного ответа на вызов. Перенаправление вызовов отличается от переадресации вызовов в том, что Пересылка вызовов выполняется с помощью параметра без участия приложения; перенаправление можно выполнить по требованию приложения, например, на основе сведений об ИДЕНТИФИКАТОРе вызывающего объекта. Он отличается от передачи вызова в том, что для передачи вызова необходимо сначала ответить на вызов. См. раздел *вызванный идентификатор*, *идентификатор звонящего*, [*Перенаправление идентификатора*](r-tapgloss.md), [*идентификатор перенаправления*](r-tapgloss.md).

</dd> <dt>

<span id="tapi2.called_id_tapgloss"></span><span id="TAPI2.CALLED_ID_TAPGLOSS"></span>**вызванный идентификатор**
</dt> <dd>

Определяет исходный адрес назначения вызова. См. раздел *Перенаправление вызова*.

</dd> <dt>

<span id="tapi2.caller_id_tapgloss"></span><span id="TAPI2.CALLER_ID_TAPGLOSS"></span>**Идентификатор звонящего**
</dt> <dd>

Определяет исходный адрес вызова. См. раздел *Перенаправление вызова*.

</dd> <dt>

<span id="tapi2.central_office_tapgloss"></span><span id="TAPI2.CENTRAL_OFFICE_TAPGLOSS"></span>**Центральный офис**
</dt> <dd>

Локальная компания телефонной компании, которая предоставляет телефонную службу в определенной области. Как правило, линии подписчиков присоединяются к переключаемому оборудованию, соединяющему других подписчиков друг с другом, локально и на длинном расстоянии. Также называется *Co*.

</dd> <dt>

<span id="tapi2.centrex_tapgloss"></span><span id="TAPI2.CENTREX_TAPGLOSS"></span>**центрекс**
</dt> <dd>

Служба рабочего телефона, предлагаемая местной телефонной компанией из местного центрального офиса.

</dd> <dt>

<span id="tapi2.computer_centric_connection_tapgloss"></span><span id="TAPI2.COMPUTER_CENTRIC_CONNECTION_TAPGLOSS"></span>**ориентированное на компьютер подключение**
</dt> <dd>

Подключение, которое использует карту надстройки компьютера или внешний ящик, подключенный к телефонной сети и набору телефонов. Сравните [*подключение, ориентированное на телефон*](p-tapgloss.md).

</dd> <dt>

<span id="tapi2.connected_id_tapgloss"></span><span id="TAPI2.CONNECTED_ID_TAPGLOSS"></span>**подключенный ИД**
</dt> <dd>

Идентификатор стороны, к которой вызывается фактический вызов. Это может отличаться от вызываемой стороны, если вызов был передан.

</dd> </dl>
