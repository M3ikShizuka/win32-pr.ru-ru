# <a name="multitrack-terminals"></a>Терминалы с многодорожкностью

В TAPI 3,1 введено понятие терминала с многодорожкностью, терминал, который, по сути, является коллекцией "Track" терминалов. Многодорожкные терминалы упрощают нагрузку программиста в ситуациях, когда в сеансе связи участвуют несколько потоков мультимедиа.

[Терминал записи файлов](file-playback-terminal-and-file-recording-terminal.md) является примером многодорожкного терминала. Он состоит из «дорожек», где каждая «дорожка» — это терминал, соответствующий потоку в записанном файле.

[Терминал транспортера](track-terminals.md) может быть другим терминалом с несколькими дорожками или терминалом с одной дорожкой. Терминал с одной дорожкой — это терминал, у которого нет терминалов по дорожкам. Терминал с одной дорожкой — это терминал в приложении TAPI 3.
