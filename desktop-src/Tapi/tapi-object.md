# <a name="tapi-object"></a>Объект TAPI

Объект TAPI представляет все ресурсы телефонии, к которым имеет доступ локальный компьютер, позволяя приложению перечислять все локальные и удаленные адреса. Приложение TAPI 3 должно создать экземпляр объекта TAPI, а затем инициализировать его.

В примере [инициализации кода TAPI](initialize-tapi.md) показан пример создания объекта TAPI.

Список всех интерфейсов, связанных с объектом TAPI, см. в разделе [интерфейсы объектов TAPI](tapi-object-interfaces.md) .
