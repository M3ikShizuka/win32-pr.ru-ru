# <a name="installation-telephony-api"></a>Установка (API телефонии)

Основные источники процедур установки — это набор ресурсов для целевой операционной системы, документация по приложению и инструкции поставщика услуг. Для поставщиков услуг Майкрософт соответствующий набор ресурсов содержит инструкции.

Процесс установки должен включать в себя список "Служба телефонии" в качестве зависимости.

Если приложение или поставщик услуг использует реестр для хранения постоянных закрытых данных, то эти сведения не должны быть помещены в раздел реестра TAPI. Формат этого раздела не гарантируется в будущих версиях.
