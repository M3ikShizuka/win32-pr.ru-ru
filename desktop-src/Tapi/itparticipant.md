# <a name="itparticipant-interface"></a>Интерфейс ИтпартиЦипант

\[**итпартиЦипант** недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Интерфейс **итпартиЦипант** реализуется с помощью [MSP-ипконф](ipconf-msp.md). Он позволяет приложению получать сведения о участниках Конференции и получать указатели на потоки, связанные с этими участниками.

Этот интерфейс предоставляется на объекте Call, когда вызов использует конференции IP. Указатель может быть получен путем вызова **QueryInterface** с помощью указателя [**иткаллинфо**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallinfo) .

## <a name="members"></a>Элементы

Интерфейс **итпартиЦипант** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **ИтпартиЦипант** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **итпартиЦипант** содержит следующие методы.



| Метод                                                                      | Описание                                                                                                                                                             |
|:----------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**енумератестреамс**](itparticipant-enumeratestreams.md)                  | Перечисляет потоки, связанные с текущим участником.<br/>                                                                                                  |
| [**получить \_ медиатипес**](itparticipant-get-mediatypes.md)                     | Возвращает [**типы носителей**](tapimediatype--constants.md) , связанные с участником.<br/>                                                                      |
| [**получить \_ партиЦипанттипединфо**](itparticipant-get-participanttypedinfo.md) | Возвращает представление BSTR требуемого типа информации, например Пти \_ EMAILADDRESS.<br/>                                                                     |
| [**получение \_ состояния**](itparticipant-get-status.md)                             | Возвращает состояние участника.<br/>                                                                                                                          |
| [**получить \_ Потоки**](itparticipant-get-streams.md)                           | Создает коллекцию потоков, связанных с текущим участником. Предоставляется для клиентских приложений автоматизации, например, написанных на Visual Basic.<br/> |
| [**\_состояние размещения**](itparticipant-put-status.md)                             | Указывает, включен ли поток, связанный с участником.<br/>                                                                                            |
