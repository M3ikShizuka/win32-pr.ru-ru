# <a name="ndis-proxy-tsp"></a>TSP прокси-сервера NDIS

Прокси-сервер NDIS — это компонент режима ядра, который предоставляет мини-порты NDIS как линии TAPI. Агент передачи прокси-сервера NDIS (Ндптсп. TSP) позволяет TAPI обмениваться данными с прокси-сервером NDIS.

Агент передачи прокси-сервера NDIS устанавливается вместе с операционными системами Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Millennium Edition, Windows 98 и Windows 95.
