# <a name="about-rendezvous-ip-telephony-conferencing"></a>О встречных конференциях IP-телефонии

\[ В Windows Vista, Windows Server 2008 и последующих версиях операционной системы нельзя использовать встречные средства и элементы управления Конференц-телефонией и интерфейсы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Встречные элементы управления TAPI 3 предоставляют механизмы для объявления и обнаружения многокомпонентных мультимедийных IP-конференций. Ниже описывается набор компонентов и интерфейсов модели COM для реализации конференций.

COM — это базовая модель программирования для TAPI 3, и ее знакомство с этим документом предполагается. Для получения сведений об COM выполните поиск по пакету SDK для платформы.

## <a name="features"></a>Компоненты

-   Предоставляет абстракцию каталога конференций для управления объявлениями об мультимедийных конференциях
-   Обеспечивает безопасность с помощью проверки подлинности, шифрования и уровня контроля доступа (ACL) для каждого объявления.
-   Предоставляет общую схему объявления конференции, позволяющую выполнять поиск по значениям атрибутов.
-   Разрешает расширения через атрибут, обслуживаемый поставщиком (объект Conference BLOB) в схеме
-   Предоставляет обертку COM для API распределения адресов многоадресной рассылки (MADCAP)

## <a name="architecture"></a>Архитектура

Следующие описания и схема иллюстрируют ключевые аспекты архитектуры системы.

-   Клиент может управлять Конференцией, хранящейся на динамическом каталоге ILS или сервере NTDS, с помощью встречных элементов управления. Протокол LDAP используется для взаимодействия с сервером ILS.
-   Встречные элементы управления предоставляют сдвоенные COM-интерфейсы для написания сценариев и программирования.
-   Сервер SAP с прямым доступом к Интернету прослушивает объявления конференций в Интернете и заполняет динамические серверы ILS сведениями о Конференции. Аналогичным образом также объявляется локально созданные конференции, область действия которых включает в себя Интернет. (Сервер SAP не планируется для Microsoft Windows 2000.)
