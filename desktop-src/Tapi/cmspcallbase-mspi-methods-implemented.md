# <a name="cmspcallbase-mspi-methods-implemented"></a>Реализованные методы МСПИ Кмспкаллбасе



| Вспомогательные методы Кмспкаллбасе МСПИ                             | Описание                                                                                                                                                                           |
|--------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**креатестреам**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-createstream)         | Создает поток мультимедиа для нового вызова. Получает блокировку в функции.                                                                                                           |
| [**енумератестреамс**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-enumeratestreams) | Перечисляет доступные в данный момент потоки. Получает блокировку в функции.                                                                                                            |
| [**получение \_ потоков**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-get_streams)          | Создает коллекцию потоков, доступных в данный момент. Предоставляется для клиентских приложений автоматизации, например, написанных на Microsoft Visual Basic. Получает блокировку в функции. |
