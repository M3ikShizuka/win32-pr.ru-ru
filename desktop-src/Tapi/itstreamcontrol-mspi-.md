# <a name="itstreamcontrol-mspi"></a>Итстреамконтрол (МСПИ)

Интерфейс [**итстреамконтрол**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol) предоставляется на объекте Call, если имеется связанный MSP. Методы этого интерфейса позволяют приложению создавать и удалять потоки.

Интерфейс [**итстреамконтрол**](/windows/win32/api/tapi3if/nn-tapi3if-itstreamcontrol) реализуется с помощью MSP. Дополнительные сведения об этом интерфейсе см. в разделе **итстреамконтрол** в интерфейсе MSP.
