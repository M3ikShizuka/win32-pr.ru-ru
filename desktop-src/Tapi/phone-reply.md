## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Комментарии

Функции, которые работают асинхронно, возвращают положительное значение идентификатора запроса приложению. Этот идентификатор запроса возвращается с ответным сообщением для идентификации завершенного запроса. Другой параметр для **\_ ответного сообщения Phone** содержит сведения об успешном или неудачном завершении. Возможные ошибки те же, что определены соответствующей функцией. Невозможно отключить это сообщение.
