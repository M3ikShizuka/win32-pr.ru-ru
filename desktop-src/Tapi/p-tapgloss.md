# <a name="p-telephony-api"></a>P (API телефонии)

<dl> <dt>

<span id="tapi2.phone_centric_connection_tapgloss"></span><span id="TAPI2.PHONE_CENTRIC_CONNECTION_TAPGLOSS"></span>**ориентированное на телефон подключение**
</dt> <dd>

Подключение, состоящее из одной POTS-линии, в которой компьютер подключен к коммутатору через набор настольных телефонов. Сравнение [*подключения, ориентированного на компьютеры*](c-tapgloss.md).

</dd> <dt>

<span id="tapi2.plain_old_telephone_service_pots__tapgloss"></span><span id="TAPI2.PLAIN_OLD_TELEPHONE_SERVICE_POTS__TAPGLOSS"></span>**Обычная старая телефонная служба (POTS)**
</dt> <dd>

Базовая служба телефонии, предоставляя стандартные однострочные телефоны, телефонные линии и доступ к сети с открытым переключением.

</dd> <dt>

<span id="tapi2.point_to_point_protocol_ppp__tapgloss"></span><span id="TAPI2.POINT_TO_POINT_PROTOCOL_PPP__TAPGLOSS"></span>**Протокол "точка — точка" (PPP)**
</dt> <dd>

Протокол связи, позволяющий компьютеру подключаться к другим компьютерам по стандартной телефонной линии с помощью высокоскоростного модема.

</dd> <dt>

<span id="tapi2.pots_tapgloss"></span><span id="TAPI2.POTS_TAPGLOSS"></span>**POTS**
</dt> <dd>

См. раздел *Обычная старая телефонная служба (POTS)*.

</dd> <dt>

<span id="tapi2.pri_isdn_tapgloss"></span><span id="TAPI2.PRI_ISDN_TAPGLOSS"></span>**PRI — ISDN**
</dt> <dd>

См. раздел *интерфейс основной частоты (PRI-ISDN)*.

</dd> <dt>

<span id="tapi2.primary_rate_interface_pri_isdn__tapgloss"></span><span id="TAPI2.PRIMARY_RATE_INTERFACE_PRI_ISDN__TAPGLOSS"></span>**Интерфейс основной частоты (PRI-ISDN)**
</dt> <dd>

Тип службы ISDN, предлагающий 23 B канала с частотой 64 кбит/с и один канал D в 64 кбит/с (23B + D). В Европейском профсоюзе PRI предоставляет 30 каналов и два канала D (30B + 2D). См. раздел [*базовый интерфейс Rate (Анд-ISDN)*](b-tapgloss.md).

</dd> <dt>

<span id="tapi2.private_branch_exchange_pbx__tapgloss"></span><span id="TAPI2.PRIVATE_BRANCH_EXCHANGE_PBX__TAPGLOSS"></span>**Обмен частными филиалами (УАТС)**
</dt> <dd>

Частная система коммутации по телефону, обычно расположенная в локальной среде клиента с помощью консоли помощника. УАТС подключена к общей группе строк из одного или нескольких центральных офисов, чтобы предоставить услуги нескольким телефонам, например в отеле, в бизнесе или в правительственном офисе. УАТС часто использует частные протоколы Digital Line, хотя некоторые из них являются аналоговыми.

</dd> <dt>

<span id="tapi2.provisioning_tapgloss"></span><span id="TAPI2.PROVISIONING_TAPGLOSS"></span>**подготовки**
</dt> <dd>

Сочетание параметров устройств и служб, образующих линию ISDN. При заказе линии ISDN телефонная компания подготавливает линию, то есть настраивает службу ISDN в соответствии с физическими возможностями коммутатора, а также выбранными параметрами.

</dd> <dt>

<span id="tapi2.public_switched_telephone_network_pstn__tapgloss"></span><span id="TAPI2.PUBLIC_SWITCHED_TELEPHONE_NETWORK_PSTN__TAPGLOSS"></span>**общедоступная коммутируемая телефонная сеть (PSTN)**
</dt> <dd>

Локальная аналоговая телефонная сеть, обеспечивающая качество обслуживания на основе голоса (3,1 кГц) (режим носителя).

</dd> </dl>
