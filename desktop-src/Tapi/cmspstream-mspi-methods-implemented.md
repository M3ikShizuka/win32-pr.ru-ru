# <a name="cmspstream-mspi-methods-implemented"></a>Реализованные методы МСПИ Кмспстреам



| Кмспстреам МСПИ методы                                   | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|-----------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**получить \_ mediaType**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_mediatype)          | Возвращает [**тип носителя**](tapimediatype--constants.md)потока.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**\_направление получения**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_direction)          | Возвращает [**направление терминала**](/windows/desktop/api/Tapi3if/ne-tapi3if-terminal_direction)потока.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**селекттерминал**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-selectterminal)         | Выбирает объект [**иттерминал**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) в потоке. Получает блокировку в функции. Этот метод базового класса знает только о поддержке списка терминалов. Предполагается, что производный класс переопределит этот метод, чтобы предоставить дополнительные функциональные возможности (такие как остановка и перезапуск потока при выборе терминала во время выполнения потока). Реализация базового класса проверяет, указывает ли указатель терминала на допустимый объем памяти, получает интерфейс [**иттерминалконтрол**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalcontrol) из терминала, извлекает маркер адреса из терминала и проверяет, соответствует ли он этому маркеру адреса MSP, проверяет, существует ли терминал в списке потоков, добавляет терминал в список терминалов потока и AddRef его. В случае сбоя любого из тестов возвращается интерфейс TAPI \_ E \_ инвалидтерминал. |
| [**унселекттерминал**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-unselectterminal)     | Отменяет выбор терминала из потока. Получает блокировку в функции. Находит терминал в списке терминалов потока, удаляет его из списка и освобождает. Если терминал не найден, он возвращает интерфейс TAPI \_ E \_ инвалидтерминал.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**енумератетерминалс**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-enumerateterminals) | Перечисляет терминалы, выбранные в потоке. На этом этапе это моментальный снимок терминалов. Получает блокировку в функции.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| [**получение \_ терминалов**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-get_terminals)          | Создает коллекцию терминалов, связанных с текущим потоком. Предоставляется для клиентских приложений автоматизации, например, написанных на Microsoft Visual Basic. На этом этапе это моментальный снимок терминалов. Получает блокировку в функции.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**стартстреам**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-startstream)               | Запускает поток. Вызывает **имедиаконтрол:: Run** на графе фильтра.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**паусестреам**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-pausestream)               | Приостанавливает поток. Вызывает **имедиаконтрол::P Аусе** на графе фильтра.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**стопстреам**](/windows/win32/api/tapi3if/nf-tapi3if-itstream-stopstream)                 | Останавливает поток. Вызывает **имедиаконтрол:: останавливаться** на графе фильтра.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
