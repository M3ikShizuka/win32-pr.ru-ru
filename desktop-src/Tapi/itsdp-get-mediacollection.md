# <a name="itsdpget_mediacollection-method"></a>Метод Итсдп:: Get \_ медиаколлектион

\[встречи и элементы управления встречными IP-телефонными соединениями недоступны для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **Get \_ медиаколлектион** получает указатель на интерфейс [**итмедиаколлектион**](itmediacollection.md) для Конференции.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_MediaCollection(
  [out] ITMediaCollection **ppMediaCollection
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ппмедиаколлектион* \[ заполняет\]
</dt> <dd>

Указатель на [**итмедиаколлектион**](itmediacollection.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                                                           | Значение                                                                                     |
|---------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                                            | Метод успешно выполнен.<br/>                                                                |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>                                    | Параметр *ппмедиаколлектион* не является допустимым указателем.<br/>                        |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl>                                   | Недостаточно памяти для выполнения операции.<br/>                             |
| <dl> <dt>**\_Ошибка E**</dt> </dl>                                          | Незаданная ошибка.<br/>                                                               |
| <dl> <dt>**HRESULT \_ из \_ \_ кода ошибки (ошибка: \_ недопустимые \_ данные)**</dt> </dl> | Произошла внутренняя ошибка, обычно из-за сбоя предыдущего метода.<br/> |
| <dl> <dt>**E \_ нотимпл**</dt> </dl>                                       | Этот метод еще не реализован.<br/>                                              |



 

## <a name="remarks"></a>Remarks

[**Итмедиаколлектион**](itmediacollection.md) указатель можно также получить с помощью **QueryInterface**.

TAPI вызывает метод **AddRef** в интерфейсе [**итмедиаколлектион**](itmediacollection.md) , возвращенном методом **итсдп:: Get \_ медиаколлектион**. Приложение должно вызвать **выпуск** в интерфейсе **итмедиаколлектион** , чтобы освободить ресурсы, связанные с ним.
