# <a name="third-party-msp"></a>MSP стороннего производителя

Специализированное коммуникационное оборудование обычно прибывает с одним или несколькими специалистами и MSP, характерными для устройства. Документация для устройства должна указывать возможности и предоставлять рекомендации по программированию.
