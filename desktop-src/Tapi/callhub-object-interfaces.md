# <a name="callhub-object-interfaces"></a>Интерфейсы объектов Каллхуб

[Объект каллхуб](callhub-object.md) предоставляет методы, которые извлекают сведения о участниках в вызове из нескольких сторон.

Интерфейсы [**иткаллхубевент**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhubevent) и [**иенумкаллхуб**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumcallhub) не предоставляются напрямую в объекте каллхуб, но тесно связаны с ним и перечислены здесь для удобства.



| Интерфейс                                | Описание                                                           |
|------------------------------------------|-----------------------------------------------------------------------|
| [**иткаллхуб**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhub)           | Предоставляет методы для получения информации, относящейся к объекту Каллхуб. |
| [**иткаллхубевент**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhubevent) | Используется для получения информации, относящейся к событиям Каллхуб.                |
| [**иенумкаллхуб**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumcallhub)     | Перечисляет [**иткаллхуб**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhub).                            |
