# <a name="tapi_audio_stream_config_caps-structure"></a>\_ \_ \_ Структура политик CAPS для настройки аудио потока TAPI \_

\[эта структура недоступна для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Структура **\_ \_ \_ \_ Caps config для настройки аудиопотока в интерфейсе** TAPI содержится в структуре " [**\_ \_ \_ заглушки" конфигурации потока TAPI**](tapi-stream-config-caps.md) , если для элемента **капстипе** задан элемент **аудиокап** объединения [**стреамконфигкапстипе**](streamconfigcapstype.md) .

```C++
} TAPI_AUDIO_STREAM_CONFIG_CAPS;
```
