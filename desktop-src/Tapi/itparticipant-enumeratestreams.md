# <a name="itparticipantenumeratestreams-method"></a>Метод ИтпартиЦипант:: Енумератестреамс

\[**енумератестреамс** недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **енумератестреамс** перечисляет потоки, в настоящее время с участниками. Этот метод предоставляется для приложений C и C++. клиентские приложения автоматизации, например написанные на Visual Basic, должны использовать метод [**get \_ Потоки**](itparticipant-get-streams.md) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT EnumerateStreams(
  [out] IEnumStream **ppEnumStream
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ппенумстреам* \[ заполняет\]
</dt> <dd>

Указатель на указатель интерфейса [**иенумстреам**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumstream) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Значение                                                                                     | Значение                                                         |
|-------------------------------------------------------------------------------------------|-----------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>      | Метод успешно выполнен.<br/>                                    |
| <dl> <dt>**\_указатель E**</dt> </dl> | Параметр *ппенумстреам* не является допустимым указателем.<br/> |



 

## <a name="remarks"></a>Remarks

TAPI вызывает метод **AddRef** в интерфейсе [**иенумстреам**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumstream) , возвращенном **итпартиЦипант:: енумератестреамс**. Приложение должно вызвать **выпуск** в интерфейсе **иенумстреам** , чтобы освободить ресурсы, связанные с ним.
