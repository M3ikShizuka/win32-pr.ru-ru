# <a name="country-or-region-code"></a>Код страны или региона

Код страны или региона важен только в том случае, если тип адреса — ЛИНЕАДДРЕССТИПЕ \_ PHONENUMBER. Он определяет область мира.

Все поставщики услуг, поддерживающие использование телефонных номеров, должны поддерживать использование этих сведений.

**Интерфейс TAPI 2. x:** См. раздел [**линежеткаллинфо**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo) (**Двкаунтрикоде** Member of *лпкаллинфо*).

**Интерфейс TAPI 3. x:** См. раздел [**иткаллинфо:: Get \_ каллинфолонг**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfolong) (элемент **CIL \_ каунтрикоде** [**каллинфо \_ Long**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_long)).
