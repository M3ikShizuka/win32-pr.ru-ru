## <a name="remarks"></a>Комментарии

Без расширяемости. Все 32 бит зарезервированы.

Когда в этой структуре данных происходят изменения, в приложение отправляется сообщение [**Line \_ каллинфо**](line-callinfo.md) . Параметры этого сообщения являются маркером вызова и указанием измененного информационного элемента. Структура данных [**линеаддресскапс**](/windows/desktop/api/Tapi/ns-tapi-lineaddresscaps) также указывает, какие из этих элементов сведений о вызовах являются допустимыми для каждого вызова в адресе.
