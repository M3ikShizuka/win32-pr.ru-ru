<dl> <dt>

<span id="tapi2.line_device_tapgloss"></span><span id="TAPI2.LINE_DEVICE_TAPGLOSS"></span>**устройство линии**
</dt> <dd>

Физический канал, состоящий из телефонной линии и оборудования для управления такими устройствами, как плата факса, модем или плата ISDN. Самый простой тип линейного устройства соответствует ровно одной телефонной линии.

</dd> <dt>

<span id="tapi2.line_device_class_tapgloss"></span><span id="TAPI2.LINE_DEVICE_CLASS_TAPGLOSS"></span>**класс линейного устройства**
</dt> <dd>

Аппаратно-независимое представление физического линейного устройства, например модема.

</dd> <dt>

<span id="tapi2.local_loop"></span><span id="TAPI2.LOCAL_LOOP"></span>**локальный цикл**
</dt> <dd>

Часть телефонной сети между телефоном и центральным коммутационным офисом телефонной компании. В этом цикле человеческий голос из домашнего телефона обычно передается в аналоговом формате, а цифровые данные с компьютера сначала должны быть преобразованы в аналоговые через модем. Цифровые сети постепенно заменяют аналоговые в локальном цикле.

</dd> </dl>
