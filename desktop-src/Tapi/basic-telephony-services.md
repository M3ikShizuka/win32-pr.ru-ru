# <a name="basic-telephony-services"></a>Базовые службы телефонии

Основные службы телефонии — это минимальное подмножество спецификаций телефонии. Поскольку все поставщики служб должны поддерживать функции основных служб телефонии, приложения, использующие только эти функции, будут работать с любым поставщиком услуг TAPI. Функции, содержащиеся в обычной телефонии, примерно соответствуют возможностям POTS.

Многие программисты будут использовать только те службы, которые предоставляет базовая телефония. Другие, например написание кода для телефонных систем УАТС, потребуют функции дополнительной телефонии.

Список функций базового телефонии см. в [справочнике по быстрой функции TAPI](./tapi-quick-function-reference.md).

Так как управление телефонными устройствами не предполагается предоставлять всем поставщикам услуг, службы телефонных устройств считаются необязательными. То есть они не являются частью обычной телефонии. Список служб телефонных устройств см. в разделе [Дополнительные службы телефонии](supplementary-telephony-services.md). Дополнительные сведения о телефонных устройствах см. в разделе [классы устройств TAPI](./tapi-device-classes.md).
