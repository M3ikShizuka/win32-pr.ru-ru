# <a name="tapi-22"></a>TAPI 2,2

TAPI 2,2 предоставляет специализированные функции управления центром обработки вызовов. В интерфейсе TAPI 2,1 и более ранних версиях центр обработки вызовов программного обеспечения требует тщательного сопоставления строк и адресов для вызова функции Center. Функции TAPI 2,1 координирует работу с возможностями центра обработки вызовов TAPI 3. x и значительно упрощают создание ACD Systems.

**Интерфейс TAPI 2. x:** Дополнительные сведения см. в разделе [Управление центром обработки вызовов](call-center-control.md).

**Интерфейс TAPI 3. x:** Дополнительные сведения см. в разделе [элементы управления центра обработки вызовов](./about-call-center-controls.md).
