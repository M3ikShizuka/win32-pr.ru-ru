# <a name="ih323lineexsetalias-method"></a>Метод IH323LineEx:: Сеталиас

\[**сеталиас** недоступен для использования в Windows Vista, Windows Server 2008 и последующих версиях операционной системы. API клиента RTC предоставляет аналогичные функциональные возможности.\]

Метод **сеталиас** настраивает псевдоним H. 323 по умолчанию для адреса. Этот псевдоним будет использоваться в обмене настройками для настройки вызова H. 323, чтобы другая сторона знала имя этой стороны.

При повторном вызове этого метода предыдущий псевдоним будет перезаписан.

Псевдоним, заданный для этого интерфейса, будет использоваться только в том случае, если для TSP не существует другого способа найти правильный псевдоним. В будущем, когда в TSP добавляется поддержка привратника, псевдоним, зарегистрированный в привратнике или назначенный привратником, будет переопределять все, что задается этим методом.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetAlias(
  [in] WCHAR *strAlias,
  [in] DWORD dwLength
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*стралиас* \[ окне\]
</dt> <dd>

Псевдоним, используемый в Юникоде. Завершение **null** не требуется.

</dd> <dt>

*двленгс* \[ окне\]
</dt> <dd>

Длина имени псевдонима в количестве символов, включая знак завершения **null** .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод может возвращать одно из этих значений.



| Код возврата                                                                               | Описание                                                                                                                      |
|-------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>      | Метод успешно выполнен.<br/>                                                                                                     |
| <dl> <dt>**\_указатель E**</dt> </dl> | Число символов, указанных в *двленгс* , превышает фактическое число символов в параметре *стралиас* .<br/> |
