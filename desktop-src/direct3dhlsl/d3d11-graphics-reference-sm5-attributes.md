# <a name="shader-model-5-attributes"></a>Атрибуты модели шейдеров 5

Модель шейдеров 5 реализует атрибуты из модели шейдеров 4 и ниже (см. пакет SDK для DirectX), а также следующие новые атрибуты:

-   [поддомен](sm5-attributes-domain.md)
-   [еарлидепсстенЦил](sm5-attributes-earlydepthstencil.md)
-   [вхождение](sm5-attributes-instance.md)
-   [макстессфактор](sm5-attributes-maxtessfactor.md)
-   [numthreads](sm5-attributes-numthreads.md)
-   [аутпутконтролпоинтс](sm5-attributes-outputcontrolpoints.md)
-   [аутпуттопологи](sm5-attributes-outputtopology.md)
-   [секционирования](sm5-attributes-partitioning.md)
-   [патчконстантфунк](sm5-attributes-patchconstantfunc.md)

Все эти атрибуты являются обязательными (за исключением атрибута **экземпляра** ). Невозможность их использования приведет к ошибкам компиляции.
