# <a name="nop---vs"></a>NOP — VS

Никакие операции не выполняются.

## <a name="syntax"></a>Синтаксис



| NOP |
|-----|



 

## <a name="remarks"></a>Remarks



| Версии шейдеров вершин | 1\_1 | 2 \_ 0 | 2 \_ x | 2 \_ SW | 3 \_ 0 | 3 \_ SW |
|------------------------|------|------|------|-------|------|-------|
| NOP                    | x    | x    | x    | x     | x    | x     |



 

Эта инструкция выполняет операцию No-Op или No. Для вызова этого метода используется следующий синтаксис:


```
nop
```
