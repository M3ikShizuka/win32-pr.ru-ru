# <a name="sampler-direct3d-9-asm-ps"></a>Образцы (Direct3D 9 ASM-PS)

Образец — это входная псевдо-регистрация для построителя текстуры, которая используется для указания этапа выборки. Регистрируются журналы этапов выборки шейдеров из 16 пикселей: S0 — S15. Поэтому для одного прохода шейдера можно считать до 16 поверхностей текстуры. В инструкциях, использующих регистр образцов, входят текслд и текслдп.

Перед использованием инструкции [дкл \_ самплертипе (SM2, SM3-PS ASM)](dcl-samplertype---ps.md) необходимо объявить образец.



| Версии шейдеров пикселей | 1\_1 | 1\_2 | 1 \_ 3 | 1\_4 | 2 \_ 0 | 2 \_ SW | 2 \_ x | 3 \_ 0 | 3 \_ SW |
|-----------------------|------|------|------|------|------|-------|------|------|-------|
| Образец               |      |      |      |      | x    | x     | x    | x    | x     |



 

Пробы являются псевдо-регистром, так как вы не можете напрямую считывать их или записывать.

Единица выборки соответствует этапу выборки текстуры с инкапсуляцией состояния, зависящего от выборки, предоставляемого [**сетсамплерстате**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9-setsamplerstate). Каждая выборка уникально определяет одну текстурную поверхность, которая устанавливается в соответствующий образец с помощью [**сеттекстуре**](/windows/desktop/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-settexture). Однако одну и ту же поверхность текстуры можно задать в нескольких пробах.

Во время рисования текстуру нельзя одновременно задать как цель рендеринга и текстуру на этапе.

Образец может отображаться в виде единственного аргумента в инструкции загрузки текстуры: [текслдл-PS](texldl---ps.md).

В PS \_ 3 \_ 0, если используется образец, его необходимо объявить в начале программы шейдера с помощью инструкции [дкл \_ самплертипе (SM2, SM3-PS ASM)](dcl-samplertype---ps.md) .

Выборка текстуры с более высоким измерением, чем имеется в координатах текстуры, недопустима. Выборка текстуры с меньшим измерением, чем есть в координатах текстуры, будет игнорировать дополнительные координаты текстуры.
