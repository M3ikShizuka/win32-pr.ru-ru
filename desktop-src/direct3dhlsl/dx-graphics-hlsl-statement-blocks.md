# <a name="statement-blocks"></a>Блоки инструкций

Блок инструкций содержит одну или несколько инструкций; Каждая инструкция состоит из [выражений](dx-graphics-hlsl-expressions.md) и [операторов](dx-graphics-hlsl-operators.md).


```
{
   statement 1;
   statement 2;
   ...
   statement n;
}
```



Блок инструкций также указывает на подобласть. Переменные, объявленные в блоке операторов, распознаются только в блоке.
