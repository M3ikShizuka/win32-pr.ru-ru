# <a name="floor"></a>floor

Возвращает максимальное целое число, которое меньше или равно указанному значению.



| *RET* -этаж (*x*) |
|------------------|



 

## <a name="parameters"></a>Параметры



| Элемент                                                   | Описание                            |
|--------------------------------------------------------|----------------------------------------|
| <span id="x"></span><span id="X"></span>*x*<br/> | \[в \] указанном значении.<br/> |



 

## <a name="return-value"></a>Возвращаемое значение

Максимальное целочисленное значение (возвращаемое как тип с плавающей запятой), которое меньше или равно значению параметра *x* .
