# <a name="texture2dgatherred-methods"></a>Методы Texture2D:: Гасерред

Возвращает красные компоненты четырех значений шаг текселя [**Texture2D**](sm5-object-texture2d.md), которые будут использоваться в операции линейной фильтрации.

| Метод                                                                                                  | Описание                                                                                                                                     |
|:--------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Гасерред (S, float, int)**](sm5-object-texture2d-gatherred.md)                                         | Возвращает красные компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации.<br/>                                 |
| [**Гасерред (S, float, int, uint)**](t2d-gatherred-s-float-int-uint-.md)                                   | Возвращает красные компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации, вместе с состоянием сопоставления плиток.<br/> |
| [**Гасерред (S, float, int2, int2, int2, int2)**](t2d-gatherred-s-float-int2-int2-int2-int2-.md)             | Возвращает красные компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации.<br/>                                 |
| [**Гасерред (S, float, int2, int2, int2, int2, uint)**](t2d-gathergreen-s-float-int2-int2-int2-int2-uint-.md) | Возвращает красные компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации, вместе с состоянием сопоставления плиток.<br/> |
