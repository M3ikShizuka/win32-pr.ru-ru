# <a name="texture2darraygathercmp-methods"></a>Методы Texture2DArray:: Гасеркмп

Для четырех шаг текселя значений [**Texture2DArray**](sm5-object-texture2darray.md) , которые будут использоваться в операции линейной фильтрации, возвращает их сравнение со значением сравнения.

Дополнительные сведения, описывающие базовую инструкцию DXBC, см. в документации по [gather4_c](./gather4-c--sm5---asm-.md) .

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                             | Описание                                                                                                      |
|:-----------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------|
| [**Гасеркмп (S, float, float, int)**](sm5-object-texture2darray-gathercmp.md)        | Выборка и сравнение текстуры и возврат всех четырех компонентов.<br/>                                       |
| [**Гасеркмп (S, float, float, int, uint)**](t2d-gathercmp-s-float-float-int-uint-.md) | Выбирает и сравнивает текстуру и возвращает все четыре компонента вместе с состоянием операции.<br/> |
