# <a name="processquadtessfactorsmax-function"></a>Функция Процесскуадтессфакторсмакс

Создает исправленные факторы тесселяции для четырех исправлений.

## <a name="syntax"></a>Синтаксис

``` syntax
void ProcessQuadTessFactorsMax(
  in  float4 RawEdgeFactors,
  in  float InsideScale,
  out float4 RoundedEdgeTessFactors,
  out float2 RoundedInsideTessFactors,
  out float2 UnroundedInsideTessFactors
);
```

## <a name="parameters"></a>Параметры

<dl> <dt>

*Раведжефакторс* \[ окне\]
</dt> <dd>

Тип: **float4**

Факторы тесселяции краев, передаваемые в стадию тесселяции.

</dd> <dt>

*Инсидескале* \[ окне\]
</dt> <dd>

Тип: **float**

Коэффициент масштабирования, применяемый к коэффициентам тесселяции UV, вычисленным на этапе тесселяции. Допустимый диапазон для Инсидескале — от 0,0 до 1,0.

</dd> <dt>

*Раундедеджетессфакторс* \[ заполняет\]
</dt> <dd>

Тип: **float4**

Скругленные границы — коэффициенты тесселяции, вычисленные на этапе тесселяции.

</dd> <dt>

*Раундединсидетессфакторс* \[ заполняет\]
</dt> <dd>

Тип: **float2**

Коэффициенты закругленной тесселяции, вычисленные на этапе тесселяции для внутренних границ.

</dd> <dt>

*Унраундединсидетессфакторс* \[ заполняет\]
</dt> <dd>

Тип: **float2**

Коэффициенты тесселяции, вычисленные на этапе тесселяции для внутренних границ.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Примечания

Формирует исправленные факторы тесселяции для четырех исправлений, вычисляя внутренние факторы тесселяции в качестве максимального значения факторов тесселяции краев. Внутренние факторы Тесс будут идентичными значениями, определяемыми максимальным значением всех четырех ребер, масштабируемых по Инсидескале. Затем результат округляется в зависимости от режима секционирования, но неокругленные результаты доступны с помощью параметра *унраундединсидетессфакторс* .
