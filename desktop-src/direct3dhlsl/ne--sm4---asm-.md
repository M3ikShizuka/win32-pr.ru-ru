# <a name="ne-sm4---asm"></a>NE (SM4-ASM)

Сравнение на уровне компонентов с плавающей запятой "не равно".



| Ne dest \[ . Mask \] , \[ - \] src0 \[ \_ ABS. \] \[ свиззле \] , \[ - \] src1 \[ \_ ABS \] \[ . свиззле\] |
|----------------------------------------------------------------------------------|



 



| Элемент                                                            | Описание                                            |
|-----------------------------------------------------------------|--------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[в \] результате операции.<br/>         |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[в \] компонентах, сравниваемых с *src1*.<br/> |
| <span id="src1"></span><span id="SRC1"></span>*src1*<br/> | \[в \] компонентах, сравниваемых с *src0*.<br/> |



 

## <a name="remarks"></a>Примечания

Эта инструкция выполняет сравнение с плавающей запятой (*src0* ! = *src1*) для каждого компонента и записывает результат в *dest*.

Если сравнение истинно, то для этого компонента возвращается значение 0xFFFFFFFF. В противном случае возвращается 0x0000000.

Перед сравнением с нетронутыми исходными регистрами источников выполняется очистка.

+ 0 равно-0.

Сравнение с NaN возвращает true.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
