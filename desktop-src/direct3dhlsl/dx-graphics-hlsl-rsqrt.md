# <a name="rsqrt"></a>рскрт

Возвращает обратную величину квадратного корня указанного значения.



| *RET* рскрт (*x*) |
|------------------|



 

## <a name="parameters"></a>Параметры



| Элемент                                                   | Описание                            |
|--------------------------------------------------------|----------------------------------------|
| <span id="x"></span><span id="X"></span>*x*<br/> | \[в \] указанном значении.<br/> |



 

## <a name="return-value"></a>Возвращаемое значение

Обратная часть квадратного корня параметра *x* .

## <a name="remarks"></a>Remarks

Эта функция использует следующую формулу: 1/ **sqrt**(*x*).
