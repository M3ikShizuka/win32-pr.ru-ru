# <a name="sampleinfo-sm41---asm"></a>самплеинфо (SM 4.1-ASM)

Запрашивает количество выборок в заданном представлении ресурсов шейдера или в средство программной прорисовки.



| \[\_uint \] dest \[ . Mask \] , сркресаурце \[ . свиззле\] |
|---------------------------------------------------|



 



| Элемент                                                                                                               | Описание                                                    |
|--------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/>                                                    | \[в \] адресе результатов операции.<br/> |
| <span id="srcResource"></span><span id="srcresource"></span><span id="SRCRESOURCE"></span>*сркресаурце*<br/> | \[в \] ресурсе шейдера.<br/>                         |



 

## <a name="remarks"></a>Remarks

Эта инструкция возвращает число выборок для данного ресурса или средства программной прорисовки. Он действителен только для ресурсов, которые могут быть загружены с помощью [**ld2dms**](ld2dms--sm4-1---asm-.md) , если средство программной прорисовки не задано как *сркресаурце*. *сркресаурце* может быть t \# -регистром (представление ресурсов шейдера) или регистром средства прорисовки.

Инструкция выполняет вычисление вектора (SampleCount, 0, 0, 0).

Свиззле On *сркресаурце* позволяет свиззлед возвращаемые значения произвольно, прежде чем они записываются в место назначения. Возвращаемое значение является плавающей запятой, если только \_ не используется модификатор uint. в этом случае возвращаемое значение является целым числом. Если нет ресурса, привязанного к указанному слоту, возвращается значение 0.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| X             | X               | x            |
