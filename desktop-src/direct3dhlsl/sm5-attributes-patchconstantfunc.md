# <a name="patchconstantfunc"></a>патчконстантфунк

Определяет функцию для вычисления данных констант исправлений.


```
patchconstantfunc("function_name")
```



## <a name="remarks"></a>Remarks

*\_ имя функции* — это имя отдельной функции, которая выводит данные константы исправления.

Этот атрибут поддерживается в следующих типах шейдеров:



| Вершина | Поверхности | Домен | Geometry | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
|        | x    |        |          |       |         |
