# <a name="point-size-register"></a>Регистр размера точки

Этот выходной регистр шейдера вершин содержит данные о размере точек вершин.



| Версии шейдеров вершин | 1\_1 | 2 \_ 0 | 2 \_ SW | 2 \_ x | 3 \_ 0 | 3 \_ SW |
|------------------------|------|------|-------|------|------|-------|
| Регистр размера точки    | x    | x    | x     | x    | x    | x     |



 

Регистр состоит из свойств, определяющих принцип работы каждой ККМ.



| Свойство        | Описание                                                                                     |
|-----------------|-------------------------------------------------------------------------------------------------|
| Имя            | Решает                                                                                            |
| Count           | 1 вектор, из которого можно использовать только один компонент, и должен быть задан маской компонента. |
| Разрешения ввода-вывода | Доступный только на запись.                                                                                     |



 

Используется только скалярный x-компонент размера точки.
