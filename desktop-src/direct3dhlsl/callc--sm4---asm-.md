# <a name="callc-sm4---asm"></a>каллк (SM4-ASM)

Условно вызывает подпрограмму, помеченную, где в программе отображается метка **l \#** .



| каллк { \_ z \|\_NZ} src0. Select \_ , компонент l\# |
|----------------------------------------------|



 



| Элемент                                                            | Описание                                                     |
|-----------------------------------------------------------------|-----------------------------------------------------------------|
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[в \] компоненте, для которого необходимо проверить условие.<br/> |
| <span id="l_"></span><span id="L_"></span>*н\#*<br/>      | \[в \] метке подпрограммы.<br/>                  |



 

## <a name="remarks"></a>Примечания

При обнаружении [RET](ret--sm4---asm-.md) верните выполнение инструкции после этого вызова.

Формат маркера содержит смещение соответствующей метки в шейдере для удобства.

В следующем примере показана инструкция Call.


```
                ...
                callc_z  r1.y, l3 // if all bits in r0.x are 0, call l3
                callc_nz r2.z, l3 // if any bit in r0.x is nonzero, call l3
                ...
                ret
                label l3
                    ...
                    retc_nz r0.x
                    ...
                ret

```



### <a name="restrictions"></a>Ограничения

-   Подпрограммы могут вкладывать 32 в глубину вложенности.
-   Стек адресов возврата управляется прозрачно в реализации.
-   Если в стеке адресов возврата уже 32 записей и выдается **вызов** , то вызов пропускается.
-   Автоматический стек параметров отсутствует. Приложение может использовать индексируемый временный массив регистров (x \# \[ \] ) для реализации стека вручную. Однако возвращаемые из вызова подпрограммы адреса не отображаются и являются ортогональными для любого ручного управления стеком, выполняемого приложением.
-   Индексирование параметра *l \#* не разрешено.
-   32-разрядный регистр, предоставляемый *src0* , тестируется на битовом уровне. Если любой бит не равен нулю, **каллк \_ NZ** выполняет вызов. Если все биты равны нулю, **каллк \_ z** выполнит вызов.
-   Рекурсия не разрешена.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
