# <a name="rwstructuredbuffer"></a>рвструктуредбуффер

Буфер чтения/записи, который может принимать тип T, являющийся структурой.



| Метод                                                                     | Описание                             |
|----------------------------------------------------------------------------|-----------------------------------------|
| [**декременткаунтер**](sm5-object-rwstructuredbuffer-decrementcounter.md) | Уменьшает значение скрытого счетчика объекта. |
| [**GetDimensions**](sm5-object-rwstructuredbuffer-getdimensions.md)       | Возвращает размеры ресурсов.           |
| [**инкременткаунтер**](sm5-object-rwstructuredbuffer-incrementcounter.md) | Увеличивает значение скрытого счетчика объекта. |
| [**Загрузить**](rwstructuredbuffer-load.md)                                    | Считывает данные буфера.                      |
| [**Оператор\[\]**](sm5-object-rwstructuredbuffer-operatorindex.md)        | Возвращает переменную ресурса.            |



 

Переменная ресурса также может быть передана в любую неупорядоченную или блокируемую операцию.

Для объектов Рвструктуредбуффер можно использовать префикс класса хранения **глобалликохерент**. Этот класс хранения вызывает барьеры памяти и выполняет синхронизацию для очистки данных во всем GPU, так что другие группы могут видеть операции записи. Без этого описателя барьер памяти или синхронизация будут сбрасывать только UAV в пределах текущей группы.

Формат UAV, привязанный к этому ресурсу, должен быть создан с \_ \_ неизвестным форматом DXGI.

Дополнительные сведения о [структурированных буферах](/windows/desktop/direct3d11/direct3d-11-advanced-stages-cs-resources)см. в обзорном материале.
