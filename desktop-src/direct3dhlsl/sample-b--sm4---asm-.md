## <a name="remarks"></a>Примечания

Исходные данные могут поступать из любого типа ресурсов, кроме буферов. Дополнительное смещение применяется к уровню детализации, вычисляемому в ходе выполнения инструкции.

Эта инструкция похожа на [образец](sample--sm4---asm-.md) инструкции с добавлением указанного значения *срклодбиас* к уровню детализации, вычисленному в рамках выполнения инструкции до выбора карт MIP. Значение *срклодбиас* добавляется в ВЫЧИСЛЕНную Лод для каждого пикселя вместе со значением «выборка миплодбиас» перед срезом Минлод и макслод.

### <a name="restrictions"></a>Ограничения

-   **пример \_ б** наследует те же ограничения, что и [образец](sample--sm4---asm-.md) инструкции, а также дополнительные ограничения для дополнительного параметра.
-   Диапазон *срклодбиас* имеет значение (-15.99 f). значения за пределами этого диапазона приведут к неопределенным результатам.
-   *срклодбиас* должен использовать один селектор компонента, если он не является скалярным.

Эта инструкция применяется к следующим этапам шейдера:

|---------------|-----------------|--------------|
|               |                 | x            |
