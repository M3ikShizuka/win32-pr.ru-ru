# <a name="shader-model-5-instruction-modifiers"></a>Модификаторы инструкций модели шейдеров 5

Модификаторы инструкций влияют на результат инструкции перед его записью в регистр. Шейдер Model 5 поддерживает следующие модификаторы инструкции.

## <a name="source-operand-modifiers"></a>Модификаторы исходного операнда

-   [Абсолютное значение](absolute-value.md)
-   [Negate](negate.md)

## <a name="instruction-result-modifiers"></a>Модификаторы результата инструкции

-   [Точный](precise.md)
-   [Saturate](saturate.md)
