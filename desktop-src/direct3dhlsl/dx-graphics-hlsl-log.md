# <a name="log"></a>log

Возвращает логарифм по базовому e указанного значения.



| Журнал *RET* (*x*) |
|----------------|



 

## <a name="parameters"></a>Параметры



| Элемент                                                   | Описание                            |
|--------------------------------------------------------|----------------------------------------|
| <span id="x"></span><span id="X"></span>*x*<br/> | \[в \] указанном значении.<br/> |



 

## <a name="return-value"></a>Возвращаемое значение

Логарифм по основанию e параметра *x* . Если параметр *x* отрицательный, эта функция возвращает неопределенное значение. Если параметр *x* равен 0, эта функция ВОЗВРАЩАЕТ-INF.

|-------|----------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------|--------------------------------|
