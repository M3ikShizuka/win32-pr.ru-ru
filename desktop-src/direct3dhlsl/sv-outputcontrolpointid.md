# <a name="sv_outputcontrolpointid"></a>ОКП \_ аутпутконтролпоинтид

Определяет индекс управляемого идентификатора контрольной точки при вызове главной точки входа шейдера поверхности.

## <a name="type"></a>Тип



| Тип     |
|------|
| uint |



 

## <a name="remarks"></a>Remarks

Это системное значение является необязательным.

Эта функция поддерживается в следующих типах шейдеров:



| Вершина | Поверхности | Домен | Geometry | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
|        | x    |        |          |       |         |
