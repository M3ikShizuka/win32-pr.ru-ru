# <a name="texturecubearraysamplecmplevelzero-methods"></a>Методы Текстурекубеаррай:: Самплекмплевелзеро

Производит выборку текстуры только на уровне mipmap 0, используя значение сравнения для отклонения выборок.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                              | Описание                                                                                                                            |
|:----------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплекмплевелзеро (S, float, float)**](dx-graphics-hlsl-to-samplecmplevelzero.md)                 | Производит выборку текстуры только на уровне mipmap 0 и сравнивает результат со значением сравнения.<br/>                                     |
| [**Самплекмплевелзеро (S, float, float, uint)**](tcubearray-samplecmplevelzero-s-float-float-uint-.md) | Производит выборку текстуры только на уровне mipmap 0 и сравнивает результат со значением сравнения. Возвращает состояние операции.<br/> |
