# <a name="mul"></a>mul

Умножает x и y с помощью матричных математических вычислений. Внутреннее измерение x-Columns и y-Rows должны быть равны.



| RET mul (x, y) |
|---------------|



 

## <a name="parameters"></a>Параметры



| Элемент                                                   | Описание                                                                           |
|--------------------------------------------------------|---------------------------------------------------------------------------------------|
| <span id="x"></span><span id="X"></span>*x*<br/> | \[в \] входном значении x. Если x является вектором, он обрабатывается как вектор строк.<br/>    |
| <span id="y"></span><span id="Y"></span>*&*<br/> | \[во \] входном значении y. Если y является вектором, он обрабатывается как вектор столбца.<br/> |



 

## <a name="return-value"></a>Возвращаемое значение

Результат x раз по оси y. Результат имеет измерение x-Rows x y-Columns.
