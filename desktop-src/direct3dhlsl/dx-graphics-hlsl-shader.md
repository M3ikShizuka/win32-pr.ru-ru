# <a name="shader-type"></a>Тип шейдера

Синтаксис объявления переменной шейдера в результате изменился с Direct3D 9 на Direct3D 10.

## <a name="shader-type-for-direct3d-10"></a>Тип шейдера для Direct3D 10

Объявите переменную шейдера в рамках этапа (в Direct3D 10) с помощью синтаксиса типа шейдера:



|                                                                                                                                                                                                     |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Сетпикселшадер** Compile ( **шадертаржет, шадерфунктион** ); **Сетжеометришадер** Compile ( **шадертаржет, шадерфунктион** ); **Сетвертексшадер** Compile ( **шадертаржет, шадерфунктион** ); |



 

### <a name="parameters"></a>Параметры



| Элемент                                                                                                                             | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|----------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="SetXXXShader"></span><span id="setxxxshader"></span><span id="SETXXXSHADER"></span>**сетксксксшадер**<br/>         | Вызов API Direct3D, который создает объект шейдера. Может быть либо: **сетпикселшадер** , либо **Сетжеометришадер** , либо **сетвертексшадер**.<br/>                                                                                                                                                                                                                                                                                                   |
| <span id="ShaderTarget"></span><span id="shadertarget"></span><span id="SHADERTARGET"></span>**шадертаржет**<br/>         | Модель шейдера для компиляции. Это допустимо для любого целевого объекта, включая все целевые объекты Direct3D 9, а также целевые объекты [модели шейдеров 4](dx-graphics-hlsl-sm4.md) : VS \_ 4 \_ 0, GS \_ 4 \_ 0 и PS \_ 4 \_ 0.<br/>                                                                                                                                                                                                                                          |
| <span id="ShaderFunction"></span><span id="shaderfunction"></span><span id="SHADERFUNCTION"></span>**шадерфунктион**<br/> | Строка ASCII, содержащая имя функции точки входа шейдера; Это функция, которая начинает выполнение при вызове шейдера. (...) Представляет аргументы шейдера; Это те же аргументы, которые передаются в API создания шейдера: [**вссетшадер**](/windows/desktop/api/d3d10/nf-d3d10-id3d10device-vssetshader) или [**гссетшадер**](/windows/desktop/api/d3d10/nf-d3d10-id3d10device-gssetshader) или [**PSSetShader**](/windows/desktop/api/d3d10/nf-d3d10-id3d10device-pssetshader).<br/> |



 

### <a name="example"></a>Пример

Ниже приведен пример создания шейдера вершин и объекта шейдера пикселей, скомпилированного для конкретной модели шейдера. В примере Direct3D 10 нет шейдера Geometry, поэтому указатель имеет значение **null**.


```
// Direct3D 10
technique10 Render
{
    pass P0
    {
        SetVertexShader( CompileShader( vs_4_0, VS() ) );
        SetGeometryShader( NULL );
        SetPixelShader( CompileShader( ps_4_0, PS() ) );
    }
}
```



## <a name="shader-type-for-direct3d-9"></a>Тип шейдера для Direct3D 9

Объявите переменную шейдера в рамках этапа (для Direct3D 9) с помощью синтаксиса типа шейдера:



|                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------------------------|
| **PixelShader** = компилировать **шадертаржет шадерфунктион (...); Вертексшадер** = компилировать **шадертаржет шадерфунктион (...);** |



 

### <a name="parameters"></a>Параметры



| Элемент                                                                                                                                                 | Описание                                                                                                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="XXXShader"></span><span id="xxxshader"></span><span id="XXXSHADER"></span>**ксксксшадер**<br/>                                         | Переменная шейдера, представляющая Скомпилированный шейдер. Может быть либо: **PixelShader** , либо **вертексшадер**.<br/>                                                                                                                                                                                                                                                                                           |
| <span id="ShaderTarget"></span><span id="shadertarget"></span><span id="SHADERTARGET"></span>**шадертаржет**<br/>                             | [Модель шейдера](dx-graphics-hlsl-models.md) для компиляции; зависит от типа переменной шейдера.<br/>                                                                                                                                                                                                                                                                                            |
| <span id="ShaderFunction_..._"></span><span id="shaderfunction_..._"></span><span id="SHADERFUNCTION_..._"></span>**Шадерфунктион (...)**<br/> | Строка ASCII, содержащая имя функции точки входа шейдера; Это функция, которая начинает выполнение при вызове шейдера. (...) Представляет аргументы шейдера; Это те же аргументы, которые передаются в API создания шейдера: [**сетвертексшадер**](/windows/desktop/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-setvertexshader) или [**сетпикселшадер**](/windows/desktop/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-setpixelshader).<br/> |



 

### <a name="example"></a>Пример

Ниже приведен пример шейдера вершин и объекта шейдера пикселей, скомпилированного для определенной модели шейдера.


```
// Direct3D 9
technique RenderSceneWithTexture1Light
{
    pass P0
    {          
        VertexShader = compile vs_2_0 RenderSceneVS( 1, true, true );
        PixelShader  = compile ps_2_0 RenderScenePS( true );
    }
}
```
