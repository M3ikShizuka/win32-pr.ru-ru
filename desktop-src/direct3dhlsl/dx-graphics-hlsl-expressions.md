# <a name="expressions"></a>Выражения

Выражение — это последовательность [переменных](dx-graphics-hlsl-variable-syntax.md) и литералов, прерывистого [операторами](dx-graphics-hlsl-statement-blocks.md). Литерал представляет собой явное значение данных, например 1 для целого числа или 2,1 для числа с плавающей запятой. Литералы часто используются для присвоения значения переменной.

Выражение, за которым следует точка с запятой (;) называется оператором. Инструкции могут варьироваться от простых выражений до блоков инструкций, которые выполняют последовательность действий. операторы заказа определяются операторами Flow управления.

Блок инструкций также указывает на подобласть. Переменные, объявленные в блоке операторов, распознаются только в блоке. Инструкции HLSL определяют порядок, в котором оцениваются выражения. Каждое выражение может быть одним из следующих.

-   Выражение
-   [Блок операторов](dx-graphics-hlsl-statement-blocks.md)
-   [Оператор return](dx-graphics-hlsl-return.md)
-   [Оператор управления потоком](dx-graphics-hlsl-flow-control.md)
