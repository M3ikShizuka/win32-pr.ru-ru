# <a name="asint-function"></a>Функция ASIN

Интерпретирует битовый шаблон входного значения как целое число. Дополнительные сведения о встроенной функции **ASIN** см. в разделе [**ASIN (DirectX HLSL)**](dx-graphics-hlsl-asint.md).

## <a name="syntax"></a>Синтаксис

``` syntax
int asint(
  in  value
);
```

## <a name="parameters"></a>Параметры

<dl> <dt>

*значение* \[ окне\]
</dt> <dd>

Входное значение.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **int**

Входные данные, интерпретируемые как целое число.
