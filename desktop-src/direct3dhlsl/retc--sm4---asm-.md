# <a name="retc-sm4---asm"></a>ретк (SM4-ASM)

Условное возвращение.



| ретк { \_ z \|\_NZ} src0. Выбор \_ компонента |
|----------------------------------------|



 



| Элемент                                                            | Описание                                                   |
|-----------------------------------------------------------------|---------------------------------------------------------------|
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[в \] регистре для проверки условия.<br/> |



 

## <a name="remarks"></a>Remarks

Если внутри подпрограммы эта инструкция возвращается к инструкции после вызова. Если не входит в подпрограмму, эта инструкция завершает выполнение программы.

В следующем примере показано, как использовать эту инструкцию.

``` syntax
           ...
           call l3
           ...
           ret
           label l3
               ...
               retc_nz r0.x  // If any bit in r0.x is nonzero, then return
               retc_z  r1.x  // If all bits in r0.x are zero, then return.
               ...
           ret
```

### <a name="restrictions"></a>Ограничения

-   **ретк** может появляться в любом месте программы, любое количество раз.
-   Последняя инструкция в основной программе или подпрограмме не может быть **ретк \_ z** или **ретк \_ NZ**. Вместо этого можно использовать безусловное значение [RET](ret--sm4---asm-.md) .
-   32-разрядный регистр, предоставляемый *src0* , тестируется на битовом уровне. Если любой бит не равен нулю, Return **\_ NZ** возвратит. Если все биты равны нулю, **ретк \_ z** возвратит значение.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
