# <a name="label-sm4---asm"></a>метка (SM4-ASM)

Указывает начало подпрограммы.



| Метка l\# |
|-----------|



 



| Элемент                                                       | Описание                         |
|------------------------------------------------------------|-------------------------------------|
| <span id="l_"></span><span id="L_"></span>*н\#*<br/> | \[в \] поле номер метки.<br/> |



 

## <a name="remarks"></a>Примечания

**Метка** может присутствовать непосредственно после инструкции [**RET**](ret--sm4---asm-.md) , которая не вложена ни в какие инструкции управления потоком.

Код, предшествующий первой **метке** в программе, является основной программой. Все подпрограммы отображаются в конце программы, обозначенные инструкциями **Label** .

В следующем примере показано, как использовать эту инструкцию.

``` syntax
 
               ...
                call l3
                ...
                ret
                label l3
                    ...
                    if_nz r0.x
                        ret
                    endif
                    ...
                ret
```

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
