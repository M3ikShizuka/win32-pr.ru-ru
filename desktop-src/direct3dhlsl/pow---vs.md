# <a name="pow---vs"></a>Pow — VS

<sup>Src1</sup>с полной точностью ABS (src0).

## <a name="syntax"></a>Синтаксис



| Pow DST, src0, src1 |
|---------------------|



 

где

-   DST — это регистр назначения.
-   src0 является регистром входного источника. Для исходного регистра требуется явное использование репликации свиззле, то есть только один из компонентов. x,. y,. z,. w свиззле (или. r,. g,. b,. a эквивалент) должен быть указан.
-   src1 является регистром входного источника. Для исходного регистра требуется явное использование репликации свиззле, то есть только один из компонентов. x,. y,. z,. w свиззле (или. r,. g,. b,. a эквивалент) должен быть указан.

## <a name="remarks"></a>Примечания



| Версии шейдеров вершин | 1\_1 | 2 \_ 0 | 2 \_ x | 2 \_ SW | 3 \_ 0 | 3 \_ SW |
|------------------------|------|------|------|-------|------|-------|
| pow                    |      | x    | x    | x     | x    | x     |



 

Эта инструкция работает, как показано здесь.


```
dest = pow(abs(src0), src1);
```



Это скалярная инструкция, поэтому исходные регистры должны иметь репликацию свиззлес, чтобы указать, какие каналы используются.

Скалярный результат реплицируется во все четыре выходных канала.

Эту инструкцию можно расширить как exp ( \* Журнал src1 (src0)).

Точность не меньше 15 бит.

Регистр приемника должен быть временным регистром и не должен совпадать с src1.
