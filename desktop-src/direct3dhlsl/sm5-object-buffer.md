# <a name="buffer"></a>Буфер

Тип буфера в том виде, в котором он существует в модели шейдеров 4, а также переменные ресурсов и сведения о буфере.



| Метод                                                   | Описание                         |
|----------------------------------------------------------|-------------------------------------|
| [**GetDimensions**](sm5-object-buffer-getdimensions.md) | Возвращает размеры ресурсов.       |
| [**Загрузить**](buffer-load.md)                              | Считывает данные буфера.                  |
| [**Оператор\[\]**](sm5-object-buffer-operatorindex.md)  | Возвращает переменную ресурса, доступную только для чтения. |



 

## <a name="minimum-shader-model"></a>Минимальная модель шейдера

Этот объект поддерживается в следующих моделях шейдеров.



| Модель шейдера                                                                | Поддерживается |
|-----------------------------------------------------------------------------|-----------|
| [Модели шейдера 5](d3d11-graphics-reference-sm5.md) и более поздних моделей шейдеров | Да       |



 

Этот объект поддерживается для следующих типов шейдеров:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
| x      | x    | x      | x        | x     | x       |
