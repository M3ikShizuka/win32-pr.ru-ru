# <a name="outputcontrolpoints"></a>аутпутконтролпоинтс

Определяет количество контрольных точек вывода (для каждого потока), которые будут созданы в шейдере поверхности.


```
outputcontrolpoints(X)
```



## <a name="remarks"></a>Примечания

Это будет количество раз, когда будет выполняться функция Main.

Этот атрибут поддерживается в следующих типах шейдеров:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
|        | x    |        |          |       |         |
