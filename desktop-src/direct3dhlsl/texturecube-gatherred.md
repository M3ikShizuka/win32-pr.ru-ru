# <a name="texturecubegatherred-methods"></a>Методы Текстурекубе:: Гасерред

Возвращает красные компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                           | Описание                                                                                                                                     |
|:-----------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Гасерред (S, float, uint)**](tcube-gatherred-s-float-uint-.md)  | Возвращает красные компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации, вместе с состоянием сопоставления плиток.<br/> |
