# <a name="asdouble-function"></a>Функция асдаубле

Повторно интерпретирует значение приведения (2 32-разрядные значения) в Double.

## <a name="syntax"></a>Синтаксис

``` syntax
double asdouble(
  in uint lowbits,
  in uint highbits
);
```

## <a name="parameters"></a>Параметры

<dl> <dt>

*ловбитс* \[ окне\]
</dt> <dd>

Тип: **uint**

Младший 32-разрядный шаблон входного значения.

</dd> <dt>

*хигхбитс* \[ окне\]
</dt> <dd>

Тип: **uint**

Старший 32-разрядный шаблон входного значения.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **Double**

Входные данные (2 32-разрядные значения) повторно приведены в виде Double.

## <a name="remarks"></a>Remarks

Также доступна следующая перегруженная версия:

``` syntax
double2 asdouble(uint2 lowbits, uint2 highbits);
```

Если входное значение является 2 32-разрядным компонентом, то возвращаемый тип будет содержать один Double. Если входное значение является 4 32-разрядным компонентом, возвращаемый тип будет содержать два значения типа Double. Если входное значение является 64-битным типом, то возвращаемое значение будет иметь то же количество компонентов, что и входное значение.
