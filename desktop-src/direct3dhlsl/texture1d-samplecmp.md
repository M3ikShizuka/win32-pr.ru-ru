# <a name="texture1dsamplecmp-methods"></a>Методы Texture1D:: Самплекмп

| Метод                                                                                         | Описание                                                                                                                                                                           |
|:-----------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплекмп (S, float, float, int)**](dx-graphics-hlsl-to-samplecmp.md)                          | Выбор текстуры с использованием значения сравнения для отклонения выборок.<br/>                                                                                                             |
| [**Самплекмп (S, float, float, int, float)**](t1d-samplecmp-s-float-float-int-float-.md)           | Выбор текстуры с использованием значения сравнения для отклонения выборок с необязательным значением для фиксации значений уровня детализации (Лод) в.<br/>                                     |
