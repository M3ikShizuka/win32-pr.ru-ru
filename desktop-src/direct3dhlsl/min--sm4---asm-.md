# <a name="min-sm4---asm"></a>min (SM4-ASM)

Минимальное количество перемещаемых компонентов.



| мин \[ \_ \] \[ . \] , \[ - \] src0 \[ \_ ABS. \] \[ свиззле \] , \[ - \] src1 \[ \_ ABS \] \[ . свиззле \] , |
|---------------------------------------------------------------------------------------------|



 



| Элемент                                                            | Описание                                                                                             |
|-----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[в \] результате операции.<br/> *конечный адрес*  =  *src0*  <  *src1* ? *src0* : *src1*<br/> |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[в \] компонентах, сравниваемых с *src1*.<br/>                                                  |
| <span id="src1"></span><span id="SRC1"></span>*src1*<br/> | \[в \] компонентах, сравниваемых с *src0*.<br/>                                                  |



 

## <a name="remarks"></a>Примечания

>= используется вместо >, поэтому если min (x, y) = x, то Max (x, y) = y.

NaN имеет особую обработку. Если один исходный операнд равен NaN, то возвращается другой исходный операнд и выбирается для каждого компонента. Если оба значения равны NaN, возвращается любое представление NaN. Это соответствует новым правилам IEEE 754R.

Перед сравнением выполняется сброс депоставлений с сохранением знака. Однако результат, записанный в *dest* , может быть или не может быть сброшен.

В следующей таблице показаны результаты, полученные при выполнении инструкции с различными классами чисел, предполагая, что ни переполнение, ни потеря точности не происходит. F означает конечное вещественное число.



|                    |          |              |          |         |
|--------------------|----------|--------------|----------|---------|
| **src0 src1 — >** | **-INF** | **Ж**        | **+ INF** | **Не число** |
| **-INF**           | -inf     | -inf         | -inf     | -inf    |
| **Ж**              | -inf     | src0 или src1 | src0     | src0    |
| **-INF**           | -inf     | src1         | +inf     | +inf    |
| **Не число**            | -inf     | src1         | +inf     | не число     |



 

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
