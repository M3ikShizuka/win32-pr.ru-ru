# <a name="dcl_input_control_point_count-sm5---asm"></a>число контрольных \_ точек ввода дкл \_ \_ \_ (SM5-ASM)

Объявите число точек управления входными данными шейдера поверхности в разделе объявления шейдера поверхности.



| \_ \_ число контрольных \_ точек ввода дкл \_ {1.. 32} |
|-------------------------------------------|



 



| Элемент                                                   | Описание                                      |
|--------------------------------------------------------|--------------------------------------------------|
| <span id="N"></span><span id="n"></span>*\N*<br/> | \[во \] счетчике контрольных точек ввода.<br/> |



 

## <a name="remarks"></a>Примечания

Требуется по крайней мере одна контрольная точка ввода. оно может быть пустым, если оно не требуется.

Эта инструкция применяется к следующим этапам шейдера:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
|        | X    |        |          |       |         |
