# <a name="lod-sm41---asm"></a>Лод (SM 4.1-ASM)

Возвращает уровень детализации (Лод), который будет использоваться для фильтрации текстур.



| Лод dest \[ . Mask \] , сркаддресс \[ . свиззле \] , сркресаурце \[ . свиззле \] , срксамплер |
|--------------------------------------------------------------------------------|



 



| Элемент                                                                                                               | Описание                                     |
|--------------------------------------------------------------------------------------------------------------------|-------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/>                                                    | \[в \] адресе результатов.<br/>   |
| <span id="srcAddress"></span><span id="srcaddress"></span><span id="SRCADDRESS"></span>*сркаддресс*<br/>     | \[в \] наборе координат текстуры.<br/> |
| <span id="srcResource"></span><span id="srcresource"></span><span id="SRCRESOURCE"></span>*сркресаурце*<br/> | \[в \] регистре текстуры.<br/>           |
| <span id="srcSampler"></span><span id="srcsampler"></span><span id="SRCSAMPLER"></span>*срксамплер*<br/>     | \[в \] регистре образца.<br/>           |



 

## <a name="remarks"></a>Примечания

Это ведет себя как [Пример](sample--sm4---asm-.md) инструкции, но отфильтрованный образец не создается. Инструкция рассчитывает следующий вектор (Клампедлод, Нонклампедлод, 0, 0). Нонклампедлод — это вычисленное значение Лод, которое игнорирует любые фиксации из образца или текстуры (IE: она может возвращать отрицательные значения). Клампедлод — это вычисленное значение Лод, которое будет использоваться фактическим **образцом** инструкции. Свиззле On *сркресаурце* позволяет свиззлед возвращаемые значения произвольно, прежде чем они записываются в место назначения.

Если нет ресурса, привязанного к указанному слоту, возвращается значение 0.

Если в образце используется анизотропная фильтрация, Лод должен соответствовать уровню MIP дробной части, основанной на меньшей оси эллиптического пространства.

Это допустимо для следующих типов текстур: Texture1D, Texture2D, Texture3D и Текстурекубе.

Инструкция **Лод** не определяется при использовании с образцом, в котором задана Фильтрация Point MIP, в частности, любое \_ перечисление D3D10 Filter, заканчивающееся на MIP \_ Point. (Пример такой: D3D10 \_ ФИЛЬТРОВАТЬ \_ min \_ MAG \_ MIP \_ Point.)

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
|               |                 | x            |
