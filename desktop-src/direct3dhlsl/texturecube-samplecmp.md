# <a name="texturecubesamplecmp-methods"></a>Методы Текстурекубе:: Самплекмп

Выбор текстуры с использованием значения сравнения для отклонения выборок.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                   | Описание                                                                                                                                                                           |
|:-----------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплекмп (S, float, float)**](dx-graphics-hlsl-to-samplecmp.md)                        | Выбор текстуры с использованием значения сравнения для отклонения выборок.<br/>                                                                                                             |
| [**Самплекмп (S, float, float, float)**](tcube-samplecmp-s-float-float-float-.md)           | Выбор текстуры с использованием значения сравнения для отклонения выборок с необязательным значением для фиксации значений уровня детализации (Лод) в.<br/>                                     |
| [**Самплекмп (S, float, float, float, uint)**](tcube-samplecmp-s-float-float-float-uint-.md) | Выбор текстуры с использованием значения сравнения для отклонения выборок с необязательным значением для фиксации значений уровня детализации (Лод) в. Возвращает состояние операции.<br/> |
