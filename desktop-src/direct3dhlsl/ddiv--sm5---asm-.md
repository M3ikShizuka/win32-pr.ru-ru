# <a name="ddiv-sm5---asm"></a>ддив (SM5-ASM)

Выполняет вычисление деления с двойной точностью на уровне компонентов.



| ддив \[ \_ \] \[ . Маска \] , \[ - \] src0 \[ \_ ABS \] \[ . свиззле \] \[ - \] src1 \[ \_ ABS \] \[ . свиззле\] |
|--------------------------------------------------------------------------------------------|



 



| Элемент                                                            | Описание                                                                                   |
|-----------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[в \] результате операции. Значение результата должно быть точным до 0,5 ULP. <br/> |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[в \] делимом.<br/>                                                               |
| <span id="src1"></span><span id="SRC1"></span>*src1*<br/> | \[в \] делителье.<br/>                                                                |



 

## <a name="remarks"></a>Remarks

Инструкция ДДИВ будет выдаваться компилятором HLSL каждый раз, когда оператор деления используется с Double. Точность этой инструкции должна быть 0,5 ULP.

Шейдеры, использующие эту инструкцию, будут помечены флагом построителя текстуры, который приведет к сбою привязки, если не выполняются все следующие условия.

-   Система поддерживает DirectX 11,1.
-   Система включает драйвер WDDM 1,2.
-   Драйвер сообщает о поддержке этой инструкции с помощью **\_ \_ параметров D3D11 Data Feature D3D11 \_ \_ . Для Екстендеддаублесшадеринструктионс** задано значение **true**.

В этой таблице F означает ограничение по настоящему вещественному числу.
