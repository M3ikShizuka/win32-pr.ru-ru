# <a name="texturecubearraygatherred-methods"></a>Методы Текстурекубеаррай:: Гасерред

Возвращает красные компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                | Описание                                                                                                                                     |
|:----------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Гасерред (S, float, uint)**](tcubearray-gatherred-s-float-uint-.md)  | Возвращает красные компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации, вместе с состоянием сопоставления плиток.<br/> |
