# <a name="bfi-sm5---asm"></a>БФИ (SM5-ASM)

Учитывая диапазон битов от ЛСБ числа, поместите это число битов в другое число в любом смещении.



| БФИ dest \[ . Mask \] , src0 \[ . свиззле \] , src1 \[ . свиззле \] , src2 \[ . свиззле \] , src3 \[ . свиззле\] |
|-------------------------------------------------------------------------------------------|



 



| Элемент                                                            | Описание                                                         |
|-----------------------------------------------------------------|---------------------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[в \] адресе результатов.<br/>                       |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[в \] битовой области для получения из *src2*.<br/>           |
| <span id="src1"></span><span id="SRC1"></span>*src1*<br/> | \[в \] битовом смещении для замены битов в *src3*.<br/> |
| <span id="src2"></span><span id="SRC2"></span>*src2*<br/> | \[\]число, из которого берутся биты. <br/>              |
| <span id="src3"></span><span id="SRC3"></span>*src3*<br/> | \[число, в \] котором заменяются биты.<br/>              |



 

## <a name="remarks"></a>Примечания

ЛСБ 5 разрядов *src0* предоставляют ширину битовой панели (0-31), которую следует взять из *src2*.

ЛСБ 5 бит *src1* предоставляет смещение битовых разрядов (0-31) для начала замены битов числа, считанного из *src3*.

``` syntax
Given width, offset:
                bitmask = (((1 << width)-1) << offset) & 0xffffffff
                dest = ((src2 << offset) & bitmask) | (src3 & ~bitmask)
```

Эта инструкция используется для упаковки целых чисел или флагов.

Эта инструкция применяется к следующим этапам шейдера:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
| X      | X    | X      | X        | X     | X       |
