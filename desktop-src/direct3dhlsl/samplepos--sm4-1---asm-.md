# <a name="samplepos-sm41---asm"></a>самплепос (SM 4.1-ASM)

Запрашивает расположение образца в заданном представлении ресурсов шейдера или в средство программной прорисовки.



| самплепос dest \[ . Mask \] , сркресаурце \[ . свиззле \] , sampleIndex (скалярный операнд) |
|--------------------------------------------------------------------------------|



 



| Элемент                                                                                                               | Описание                                                    |
|--------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/>                                                    | \[в \] адресе результатов операции.<br/> |
| <span id="srcResource"></span><span id="srcresource"></span><span id="SRCRESOURCE"></span>*сркресаурце*<br/> | \[в \] ресурсе шейдера.<br/>                         |
| <span id="sampleIndex"></span><span id="sampleindex"></span><span id="SAMPLEINDEX"></span>*sampleIndex*<br/> | \[в \] индексе образца.<br/>                     |



 

## <a name="remarks"></a>Remarks

Эта инструкция возвращает 2D-пример расположения образца *sampleIndex* для заданного ресурса. Он действителен только для ресурсов, которые могут быть загружены с помощью [**ld2dms**](ld2dms--sm4-1---asm-.md) , если средство программной прорисовки не задано как *сркресаурце*.

*сркресаурце* может быть t \# -регистром (представление ресурсов шейдера) или регистром средства прорисовки.

Инструкция выполняет вычисление вектора с плавающей запятой (Кспоситион, Ипоситион, 0, 0).

Свиззле On *сркресаурце* позволяет свиззлед возвращаемые значения произвольно, прежде чем они записываются в место назначения. Положение образца определяется относительно центра пикселя на основе системы координат пикселей.

Если *sampleIndex* выходит за пределы допустимого диапазона, возвращается нуль-вектор. Если нет ресурса, привязанного к указанному слоту, возвращается значение 0.

**самплепос** можно использовать для таких целей, как пользовательские разрешения в коде шейдера.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
|               |                 | x            |
