# <a name="texture1darraysamplecmplevelzero-methods"></a>Методы Texture1DArray:: Самплекмплевелзеро

Производит выборку текстуры только на уровне mipmap 0, используя значение сравнения для отклонения выборок.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                    | Описание                                                                                                                            |
|:----------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплекмплевелзеро (S, float, float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)                   | Производит выборку текстуры только на уровне mipmap 0 и сравнивает результат со значением сравнения.<br/>                                     |
| [**Самплекмплевелзеро (S, float, float, int, uint)**](t1darray-samplecmplevelzero-s-float-float-int-uint-.md) | Производит выборку текстуры только на уровне mipmap 0 и сравнивает результат со значением сравнения. Возвращает состояние операции.<br/> |
