# <a name="waveactivealltrue-function"></a>Функция Вавеактивеаллтруе

Возвращает значение true, если выражение имеет значение true во всех активных желобах в текущем Wave.

## <a name="syntax"></a>Синтаксис

``` syntax
bool WaveActiveAllTrue(
   bool expr
);
```

## <a name="parameters"></a>Параметры

<dl> <dt>

*expr* 
</dt> <dd>

Логическое выражение для вычисления.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Значение true, если выражение имеет значение true во всех дорожках.

## <a name="remarks"></a>Примечания

Эта функция поддерживается из модели шейдеров 6,0 на всех стадиях шейдера.
