# <a name="debugging-shaders-in-visual-studio"></a>Отладка шейдеров в Visual Studio

Новейшее средство для отладки шейдеров теперь поставляется как функция в Microsoft Visual Studio, которая называется отладчиком графики Visual Studio. Это новое средство заменяет средство [PIX](/windows/desktop/directx-sdk--august-2009-) for Windows. Графический отладчик Visual Studio значительно улучшил удобство использования, поддержку Windows 8 и Direct3D 11,1 и интеграцию с традиционными функциями Visual Studio, такими как стеки вызовов и отладка Windows для отладки [HLSL](dx-graphics-hlsl.md) . Дополнительные сведения об этой новой функции см. в разделе [Отладка графики DirectX](/visualstudio/debugger/visual-studio-graphics-diagnostics?view=vs-2015).
