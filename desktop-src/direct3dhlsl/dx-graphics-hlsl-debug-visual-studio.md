# <a name="debugging-shaders-in-visual-studio"></a>Отладка шейдеров в Visual Studio

новейшее средство для отладки шейдеров теперь поставляется как функция в Microsoft Visual Studio, называемая Visual Studio графическим отладчиком. это новое средство заменяет средство [PIX](/windows/desktop/directx-sdk--august-2009-) for Windows tool. Visual Studio графический отладчик значительно улучшил удобство использования, поддерживает Windows 8 и Direct3D 11,1 и интеграцию с традиционными Visual Studioными функциями, такими как стеки вызовов и отладка Windows для отладки [HLSL](dx-graphics-hlsl.md) . Дополнительные сведения об этой новой функции см. в разделе [Отладка графики DirectX](/visualstudio/debugger/visual-studio-graphics-diagnostics?view=vs-2015).
