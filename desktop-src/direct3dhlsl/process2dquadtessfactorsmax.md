# <a name="process2dquadtessfactorsmax-function"></a>Функция Process2DQuadTessFactorsMax

Создает исправленные факторы тесселяции для четырех исправлений.

## <a name="syntax"></a>Синтаксис

``` syntax
void Process2DQuadTessFactorsMax(
  in  float4 RawEdgeFactors,
  in  float2 InsideScale,
  out float4 RoundedEdgeTessFactors,
  out float2 RoundedInsideTessFactors,
  out float2 UnroundedInsideTessFactors
);
```

## <a name="parameters"></a>Параметры

<dl> <dt>

*Раведжефакторс* \[ окне\]
</dt> <dd>

Тип: **float4**

Факторы тесселяции краев, передаваемые в стадию тесселяции.

</dd> <dt>

*Инсидескале* \[ окне\]
</dt> <dd>

Тип: **float2**

Коэффициент масштабирования, применяемый к коэффициентам тесселяции UV, вычисленным на этапе тесселяции. Допустимый диапазон для Инсидескале — от 0,0 до 1,0.

</dd> <dt>

*Раундедеджетессфакторс* \[ заполняет\]
</dt> <dd>

Тип: **float4**

Скругленные границы — коэффициенты тесселяции, вычисленные на этапе тесселяции.

</dd> <dt>

*Раундединсидетессфакторс* \[ заполняет\]
</dt> <dd>

Тип: **float2**

Коэффициенты закругленной тесселяции, вычисленные на этапе тесселяции для внутренних границ.

</dd> <dt>

*Унраундединсидетессфакторс* \[ заполняет\]
</dt> <dd>

Тип: **float2**

Коэффициенты тесселяции, вычисленные на этапе тесселяции для внутренних границ.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Формирует исправленные факторы тесселяции для четырех исправлений, вычисляя внутренние факторы тесселяции в качестве максимального значения факторов тесселяции краев. Значения параметров "вы" и "V" в списке факторов тесселяции вычисляются независимо друг от друга с максимальным числом сторон в домене, а затем масштабируются по Инсидескале. Затем результат округляется в зависимости от режима секционирования, но неокругленные результаты доступны с помощью параметра Унраундединсидетессфакторс.
