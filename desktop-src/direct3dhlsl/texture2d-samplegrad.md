# <a name="texture2dsamplegrad-methods"></a>Методы Texture2D:: Самплеград

Примеры [**Texture2D**](sm5-object-texture2d.md) , использующие градиент, чтобы повлиять на способ вычисления расположения образца.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                   | Описание                                                                                                                                                                                                        |
|:---------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплеград (S, float, float, float, int)**](dx-graphics-hlsl-to-samplegrad.md)                            | Выбор текстуры с помощью градиента, влияющей на способ вычисления расположения образца.<br/>                                                                                                             |
| [**Самплеград (S, float, float, float, int, float)**](samplegrad-s-float-float-float-int-float-.md)           | Выбор текстуры с помощью градиента, влияющей на способ вычисления расположения выборки, с дополнительным значением для фиксации значений уровня детализации (Лод).<br/>                                     |
| [**Самплеград (S, float, float, float, int, float, uint)**](samplegrad-s-float-float-float-int-float-uint-.md) | Выбор текстуры с помощью градиента, влияющей на способ вычисления расположения выборки, с дополнительным значением для фиксации значений уровня детализации (Лод). Возвращает состояние операции.<br/> |
