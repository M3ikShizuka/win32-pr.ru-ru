# <a name="texturecubearraysamplegrad-methods"></a>Методы Текстурекубеаррай:: Самплеград

Выбор текстуры с помощью градиента, чтобы повлиять на способ вычисления расположения образца.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                      | Описание                                                                                                                                                                                                        |
|:------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплеград (S, float, float, float)**](dx-graphics-hlsl-to-samplegrad.md)                                   | Выбор текстуры с помощью градиента, влияющей на способ вычисления расположения образца.<br/>                                                                                                             |
| [**Самплеград (S, float, float, float, float)**](tcubearray-samplegrad-s-float-float-float-float-.md)           | Выбор текстуры с помощью градиента, влияющей на способ вычисления расположения выборки, с дополнительным значением для фиксации значений уровня детализации (Лод).<br/>                                     |
| [**Самплеград (S, float, float, float, float, uint)**](tcubearray-samplegrad-s-float-float-float-float-uint-.md) | Выбор текстуры с помощью градиента, влияющей на способ вычисления расположения выборки, с дополнительным значением для фиксации значений уровня детализации (Лод). Возвращает состояние операции.<br/> |
