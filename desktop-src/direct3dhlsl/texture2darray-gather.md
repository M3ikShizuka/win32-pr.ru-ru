# <a name="texture2darraygather-methods"></a>Методы Texture2DArray:: собрать

Возвращает четыре значения шаг текселя объекта [**Texture2DArray**](sm5-object-texture2darray.md) , которые будут использоваться в операции линейной фильтрации.

Дополнительные сведения, описывающие базовую инструкцию DXBC, см. в документации по [gather4](./gather4--sm5---asm-.md) .

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                | Описание                                                                                                               |
|:----------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------|
| [**Сбор данных (S, float, int)**](sm5-object-texture2darray-gather.md)        | Возвращает четыре значения шаг текселя, которые будут использоваться в операции линейной фильтрации.<br/>                                 |
| [**Сбор данных (S, float, int, uint)**](t2darray-gather-s-float-int-uint-.md)  | Возвращает четыре значения шаг текселя, которые будут использоваться в операции линейной фильтрации, вместе с состоянием сопоставления плиток.<br/> |
