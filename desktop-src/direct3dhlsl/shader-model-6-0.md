# <a name="shader-model-6"></a>Модель шейдеров 6

Все неоднородные встроенные функции звукового сигнала доступны на всех стадиях шейдера. Четырехъядерные встроенные функции доступны только в пиксельных и расчетных шейдерах.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                 | Описание                                                                                                                                                               |
|-----------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**куадреадакроссдиагонал**](quadreadacrossdiagonal.md)<br/> | Возвращает заданное локальное значение, которое считывается с диагонального противоположного полосы в данном Quad.<br/>                                                                |
| [**куадреадланеат**](quadreadlaneat.md)<br/>                   | Возвращает указанное исходное значение из полосы, определяемой ИДЕНТИФИКАТОРом полосы в текущем объекте Quad.<br/>                                                            |
| [**куадреадакросскс**](quadswapx.md)<br/>                      | Возвращает указанное локальное значение, считанное из другой полосы в данном Quad по оси X.<br/>                                                                    |
| [**куадреадакросси**](quadswapy.md)<br/>                      | Возвращает указанное исходное значение, считанное из другой полосы в данном Quad по оси Y.<br/>                                                                   |
| [**вавеактивеаллекуал**](waveactiveallequal.md)<br/>           | Возвращает значение true, если выражение одинаково для каждой активной полосы в текущем звуковом поле (и, таким образом, единообразно в нем).<br/>                                             |
| [**вавеактивебитанд**](waveallbitand.md)<br/>                  | Возвращает побитовое и для всех значений выражения для всех активных дорожек в текущей волновой передаче и реплицирует его обратно во все активные дорожки. <br/>           |
| [**вавеактивебитор**](waveallbitor.md)<br/>                    | Возвращает побитовое или для всех значений выражения для всех активных дорожек в текущей волновой передаче и реплицирует его обратно во все активные дорожки. <br/>            |
| [**вавеактивебитксор**](waveallbitxor.md)<br/>                  | Возвращает побитовое ИСКЛЮЧАЮЩее число всех значений выражения для всех активных дорожек в текущей волновой передаче и реплицирует его обратно во все активные дорожки. <br/>           |
| [**вавеактивекаунтбитс**](waveactivecountbits.md)<br/>         | Подсчитывает количество логических переменных, которые имеют значение true во всех активных желобах в текущей волновой области, и реплицирует результат во все желобы в волне.<br/> |
| [**вавеактивемакс**](waveallmax.md)<br/>                        | Возвращает максимальное значение выражения для всех активных дорожек в текущей волновой передаче и реплицирует его обратно во все активные дорожки. <br/>                           |
| [**вавеактивемин**](waveallmin.md)<br/>                        | Возвращает минимальное значение выражения для всех активных желобов в текущем звуковом процессе реплицирует его обратно во все активные дорожки. <br/>                               |
| [**вавеактивепродукт**](waveallproduct.md)<br/>                | Умножает значения выражения на все активные дорожки в текущей волне и реплицирует их обратно во все активные дорожки.<br/>                       |
| [**вавеактивесум**](waveallsum.md)<br/>                        | Суммирует значение выражения по всем активным желобам в текущей волновой записи и реплицирует его во все дорожки в текущей волн.<br/>                            |
| [**вавеактивеаллтруе**](wavealltrue.md)<br/>                   | Возвращает значение true, если выражение имеет значение true во всех активных желобах в текущем Wave.<br/>                                                                                |
| [**вавеактивеанитруе**](waveanytrue.md)<br/>                   | Возвращает значение true, если выражение имеет значение true в любом из активных желобов в текущей волны.<br/>                                                                         |
| [**вавеактивебаллот**](waveballot.md)<br/>                     | Возвращает 4-битовое целое число без знака для вычисления логического выражения для всех активных желобов в указанной волне. <br/>                              |
| [**вавежетланекаунт**](wavegetlanecount.md)<br/>               | Возвращает количество желобов в волне на этой архитектуре. <br/>                                                                                                   |
| [**вавежетланеиндекс**](wavegetlaneindex.md)<br/>               | Возвращает индекс текущей полосы в пределах текущей волны. <br/>                                                                                                |
| [**вавеисфирстлане**](waveisfirstlane.md)<br/>                 | Возвращает значение true только для активной полосы в текущем волне с наименьшим индексом. <br/>                                                                            |
| [**вавепрефикскаунтбитс**](waveprefixcountbytes.md)<br/>        | Возвращает сумму всех заданных переменных типа Boolean, для которых задано значение true для всех активных дорожек с индексами, меньшими, чем текущая полоса. <br/>                        |
| [**вавепрефикспродукт**](waveprefixproduct.md)<br/>             | Возвращает произведение всех значений в активных желобах в этой волновой области с индексами, меньшими, чем эта полоса.<br/>                                                    |
| [**вавепрефикссум**](waveprefixsum.md)<br/>                     | Возвращает сумму всех значений в активном желобе с меньшими индексами по сравнению с этим значением.<br/>                                                                   |
| [**вавереадланефирст**](wavereadfirstlane.md)<br/>             | Возвращает значение выражения для активной полосы текущего волна с наименьшим индексом. <br/>                                                          |
| [**вавереадланеат**](wavereadlaneat.md)<br/>                   | Возвращает значение выражения для заданного индекса полосы в заданной волны.<br/>                                                                        |
