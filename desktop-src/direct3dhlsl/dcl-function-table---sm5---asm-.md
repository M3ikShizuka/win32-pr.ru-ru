# <a name="dcl_function_table-sm5---asm"></a>\_Таблица функций дкл \_ (SM5-ASM)

Объявите таблицу функций.



| дкл \_ функция \_ таблицы ft \# = {FB \# , FB \# ,...} |
|-----------------------------------------------|



 



| Элемент                                                      | Описание                                   |
|-----------------------------------------------------------|-----------------------------------------------|
| <span id="ft"></span><span id="FT"></span>*ФТ*<br/> | \[в \] записях таблицы функций.<br/> |



 

## <a name="remarks"></a>Remarks

Эта функция объявляет таблицу функций как набор частей тела функции, объявленных ранее.

Это аналогично виртуальной таблицей с + #, за исключением наличия записи для каждого узла вызова интерфейса, а не для каждого метода.

В таблице функций не предусмотрено ограничений на количество основных частей функций.

\#В одной или нескольких таблицах функций допускается несколько ссылок на FB тела функции, что позволяет совместно использовать общий код.

Эта инструкция применяется к следующим этапам шейдера:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
| X      | X    | X      | X        | X     | X       |

|-----------------------------------------------------------|-----------|
