# <a name="processtritessfactorsmin-function"></a>Функция Процесстритессфакторсмин

Создает исправленные коэффициенты тесселяции для более трехуровневого обновления.

## <a name="syntax"></a>Синтаксис

``` syntax
void ProcessTriTessFactorsMin(
  in  float3 RawEdgeFactors,
  in  float InsideScale,
  out float3 RoundedEdgeTessFactors,
  out float RoundedInsideTessFactors,
  out float UnroundedInsideTessFactors
);
```

## <a name="parameters"></a>Параметры

<dl> <dt>

*Раведжефакторс* \[ окне\]
</dt> <dd>

Тип: **float3**

Факторы тесселяции краев, передаваемые в стадию тесселяции.

</dd> <dt>

*Инсидескале* \[ окне\]
</dt> <dd>

Тип: **float**

Коэффициент масштабирования, применяемый к коэффициентам тесселяции UV, вычисленным на этапе тесселяции. Допустимый диапазон для Инсидескале — от 0,0 до 1,0.

</dd> <dt>

*Раундедеджетессфакторс* \[ заполняет\]
</dt> <dd>

Тип: **float3**

Скругленные границы — коэффициенты тесселяции, вычисленные на этапе тесселяции.

</dd> <dt>

*Раундединсидетессфакторс* \[ заполняет\]
</dt> <dd>

Тип: **float**

Коэффициенты закругленной тесселяции, вычисленные на этапе тесселяции для внутренних границ.

</dd> <dt>

*Унраундединсидетессфакторс* \[ заполняет\]
</dt> <dd>

Тип: **float**

Коэффициенты тесселяции, вычисленные на этапе тесселяции для внутренних границ.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Примечания

Формирует исправленные коэффициенты тесселяции для трехуровневого обновления, вычисляя внутренний фактор тесселяции как минимум для факторов тесселяции краев, которые затем масштабируются по Инсидескале. Затем результат округляется в зависимости от режима секционирования, но неокругленные результаты доступны с помощью параметра Унраундединсидетессфактор.
