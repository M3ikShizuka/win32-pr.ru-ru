# <a name="texture1dsamplecmplevelzero-methods"></a>Методы Texture1D:: Самплекмплевелзеро

Пример [**Texture1D**](sm5-object-texture1d.md) только для mipmap уровня 0 и сравнивает результат со значением сравнения.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                               | Описание                                                                                                                            |
|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплекмплевелзеро (S, float, float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)              | Производит выборку текстуры только на уровне mipmap 0 и сравнивает результат со значением сравнения.<br/>                                     |
| [**Самплекмплевелзеро (S, float, float, int, uint)**](t1d-samplecmplevelzero-s-float-float-int-uint-.md) | Производит выборку текстуры только на уровне mipmap 0 и сравнивает результат со значением сравнения. Возвращает состояние операции.<br/> |



## <a name="remarks"></a>Примечания

Ссылается на [объект текстуры](dx-graphics-hlsl-to-type.md).
