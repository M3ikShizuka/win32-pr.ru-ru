# <a name="continue-sm4---asm"></a>продолжить (SM4-ASM)

Продолжение выполнения в начале текущего цикла.



| continue |
|----------|



 

## <a name="remarks"></a>Примечания

**Continue** можно использовать только внутри [цикла](loop--sm4---asm-.md) или [ендлуп](endloop--sm4---asm-.md).

В следующем примере показано, как использовать инструкцию **Continue** .


```
                loop
                    if_na r0.x
                        break
                    endif
                    if_z r1.x
                        ...
                        continue
                    endif
                    ...
                endloop
```



Формат маркера содержит смещение соответствующей инструкции Loop в шейдере для удобства.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
