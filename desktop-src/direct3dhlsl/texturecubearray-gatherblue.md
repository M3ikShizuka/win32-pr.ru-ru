# <a name="texturecubearraygatherblue-methods"></a>Методы Текстурекубеаррай:: Гасерблуе

Возвращает синие компоненты четырех шаг текселя значений, которые будут использоваться в операции линейной фильтрации.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                  | Описание                                                                                                                                      |
|:------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Гасерблуе (S, float, uint)**](tcubearray-gatherblue-s-float-uint-.md)  | Возвращает синие компоненты четырех шаг текселя значений, которые будут использоваться в операции линейной фильтрации, вместе с состоянием сопоставления плиток.<br/> |
