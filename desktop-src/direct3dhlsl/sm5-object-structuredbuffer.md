# <a name="structuredbuffer"></a>структуредбуффер

Буфер только для чтения, который может принимать тип T, являющийся структурой.



| Метод                                                             | Описание                            |
|--------------------------------------------------------------------|----------------------------------------|
| [**GetDimensions**](sm5-object-structuredbuffer-getdimensions.md) | Возвращает размеры ресурсов.          |
| [**Загрузить**](structuredbuffer-load.md)                              | Считывает данные буфера.                     |
| [**Оператор\[\]**](sm5-object-structuredbuffer-operatorindex.md)  | Возвращает переменную ресурса, доступную только для чтения. |



 

Формат UAV, привязанный к этому ресурсу, должен быть создан с \_ \_ неизвестным форматом DXGI.

Дополнительные сведения о [структурированных буферах](/windows/desktop/direct3d11/direct3d-11-advanced-stages-cs-resources)см. в обзорном материале.
