# <a name="cosh"></a>cosh

Возвращает гиперболический косинус указанного значения.



| *RET* cosh (*x*) |
|-----------------|



 

## <a name="parameters"></a>Параметры



| Элемент                                                   | Описание                                        |
|--------------------------------------------------------|----------------------------------------------------|
| <span id="x"></span><span id="X"></span>*x*<br/> | \[в \] заданном значении в радианах.<br/> |
