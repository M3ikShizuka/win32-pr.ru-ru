# <a name="fog-register"></a>Регистр тумана

Этот выходной регистр шейдера вершин содержит цвет тумана для вершины.

Регистр состоит из свойств, определяющих принцип работы каждой ККМ.



| Свойство        | Описание                                                                                     |
|-----------------|-------------------------------------------------------------------------------------------------|
| Имя            | офог                                                                                            |
| Count           | Один вектор, из которого можно использовать только один компонент и должен быть задан маской компонента |
| Разрешения ввода-вывода | Доступный только на запись.                                                                                     |



 

Регистры значений выходных тумана. Значение — это коэффициент тумана для интерполяции, который затем направляется в таблицу тумана. Используется только скалярный компонент x-компонента тумана.
