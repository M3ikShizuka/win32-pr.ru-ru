# <a name="texturecubearraygather-methods"></a>Методы Текстурекубеаррай:: собрать

Возвращает красные компоненты из четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации.

Дополнительные сведения, описывающие базовую инструкцию DXBC, см. в документации по [gather4](./gather4--sm5---asm-.md) .

### <a name="overload-list"></a>Список перегрузок



| Метод                                                          | Описание                                                                                                                                |
|:----------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------|
| [**Сбор данных (S, float)**](dx-graphics-hlsl-to-gather.md)           | Возвращает красные компоненты четырех значений шаг текселя (только красный компонент), которые будут использоваться в операции линейной фильтрации. <br/>           |
| [**Сбор данных (S, float, uint)**](tcubearray-gather-s-float-uint-.md) | Возвращает красные компоненты из четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации, вместе с состоянием сопоставления плиток.<br/> |
