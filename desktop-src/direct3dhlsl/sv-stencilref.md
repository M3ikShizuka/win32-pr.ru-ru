# <a name="sv_stencilref"></a>ОКП \_ стенЦилреф

ОКП \_ стенЦилреф представляет текущее значение ссылки на набор элементов для шейдера пикселей.

## <a name="type"></a>Тип



| Тип     |
|------|
| uint |
