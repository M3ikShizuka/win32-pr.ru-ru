# <a name="uaddc-sm5---asm"></a>уаддк (SM5-ASM)

Целое число без знака, добавленное с помощью макрокоманды.



| уаддк dest0 \[ . Mask \] , dest1 \[ . Mask \] , src0 \[ . свиззле \] , src1 \[ . свиззле\] |
|--------------------------------------------------------------------------|



 



| Элемент                                                               | Описание                                            |
|--------------------------------------------------------------------|--------------------------------------------------------|
| <span id="dest0"></span><span id="DEST0"></span>*dest0*<br/> | \[в \] поле Адрес результата.<br/>               |
| <span id="dest1"></span><span id="DEST1"></span>*dest1*<br/> | \[в \] 1, если создается. В противном случае 0.<br/> |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/>    | \[в \] 32-разрядный операнд для добавления.<br/>          |
| <span id="src1"></span><span id="SRC1"></span>*src1*<br/>    | \[в \] 32-разрядный операнд для добавления.<br/>          |



 

## <a name="remarks"></a>Remarks

*dest1* может иметь значение null, если не требуется выполнение.

Используйте эту инструкцию для арифметических операций с высокой точностью.

Эта инструкция применяется к следующим этапам шейдера:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
| X      | X    | X      | X        | X     | X       |



 

## <a name="minimum-shader-model"></a>Минимальная модель шейдера

Эта инструкция поддерживается в следующих моделях шейдеров:



| Модель шейдера                                              | Поддерживается |
|-----------------------------------------------------------|-----------|
| [Модель шейдера 5](d3d11-graphics-reference-sm5.md)        | Да       |
| [Модель шейдера 4,1](dx-graphics-hlsl-sm4.md)              | Нет        |
| [Модель шейдера 4](dx-graphics-hlsl-sm4.md)                | Нет        |
| [Модель шейдера 3 (DirectX HLSL)](dx-graphics-hlsl-sm3.md) | Нет        |
| [Модель шейдера 2 (DirectX HLSL)](dx-graphics-hlsl-sm2.md) | Нет        |
| [Модель шейдера 1 (DirectX HLSL)](dx-graphics-hlsl-sm1.md) | Нет        |
