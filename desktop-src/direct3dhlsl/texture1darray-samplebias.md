# <a name="texture1darraysamplebias-methods"></a>Методы Texture1DArray:: Самплебиас

Выбор текстуры после применения значения смещения к уровню mipmap.

| Метод                                                                                           | Описание                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплебиас (S, float, float, int)**](dx-graphics-hlsl-to-samplebias.md)                          | Выбор текстуры после применения значения смещения к уровню mipmap.<br/>                                                                                                             |
| [**Самплебиас (S, float, float, int, float)**](t1darray-samplebias-s-float-float-int-float-.md)      | Демонстрирует выбор текстуры после применения значения смещения к уровню mipmap с дополнительным значением для создания среза образцов значений уровня детализации (Лод).<br/>                                     |
| [**Самплебиас (S, float, float, int, float, uint)**](t1d-samplebias-s-float-float-int-float-uint-.md) | Демонстрирует выбор текстуры после применения значения смещения к уровню mipmap с дополнительным значением для создания среза образцов значений уровня детализации (Лод). Возвращает состояние операции.<br/> |
