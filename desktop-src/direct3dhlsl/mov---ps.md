# <a name="mov---ps"></a>MOV-PS

Перемещение данных между регистрами.

## <a name="syntax"></a>Синтаксис



| MOV DST, src |
|--------------|



 

where

-   DST — это регистр назначения.
-   src является исходным регистром.
