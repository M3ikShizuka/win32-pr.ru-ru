# <a name="dcl_output-omask-sm5---asm"></a>дкл \_ Output омаск (SM5-ASM)

Объявите выходной регистр, записываемый шейдером.



| дкл \_ выход o \# \[ . Mask\] |
|--------------------------|



 



| Элемент                                                   | Описание                            |
|--------------------------------------------------------|----------------------------------------|
| <span id="o"></span><span id="O"></span>*вывода*<br/> | \[в \] реестре выходных данных.<br/> |



 

## <a name="remarks"></a>Remarks


```
Example:
                dcl_output o[3].xyz
```



### <a name="restrictions"></a>Ограничения

-   Маска компонента может быть любым подмножеством \[ ксизв \] . Однако разрывы между компонентами не занимают место.
-   Допустимо объявлять надмножество маски компонента, объявленной для входных данных на следующем этапе. Однако взаимно исключающие маски не допускаются. Выводится O3. XY шейдера вершин, означающее, что построитель построителя текстуры помещает значение v3. z недопустимый, но вывод значений v3. x или v3. y или v3. XY является допустимым.

Эта инструкция применяется к следующим этапам шейдера:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
| X      | X    | X      | X        | X     |         |



 

## <a name="minimum-shader-model"></a>Минимальная модель шейдера

Эта инструкция поддерживается в следующих моделях шейдеров:



| Модель шейдера                                              | Поддерживается |
|-----------------------------------------------------------|-----------|
| [Модель шейдера 5](d3d11-graphics-reference-sm5.md)        | Да       |
| [Модель шейдера 4,1](dx-graphics-hlsl-sm4.md)              | Нет        |
| [Модель шейдера 4](dx-graphics-hlsl-sm4.md)                | Нет        |
| [Модель шейдера 3 (DirectX HLSL)](dx-graphics-hlsl-sm3.md) | Нет        |
| [Модель шейдера 2 (DirectX HLSL)](dx-graphics-hlsl-sm2.md) | Нет        |
| [Модель шейдера 1 (DirectX HLSL)](dx-graphics-hlsl-sm1.md) | Нет        |
