# <a name="utof-sm4---asm"></a>утоф (SM4-ASM)

Преобразование целого числа без знака в число с плавающей запятой.



| утоф dest \[ . Mask \] , src0 \[ . свиззле\] |
|--------------------------------------|



 



| Элемент                                                            | Описание                                                   |
|-----------------------------------------------------------------|---------------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[в \] адресе результата операции.<br/> |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[\]преобразуемые компоненты.<br/>                  |



 

## <a name="remarks"></a>Remarks

*src0* должен содержать неподписанный 32-разрядный целочисленный кортеж из 4 элементов. После выполнения инструкции *dest* будет содержать кортеж из четырех точек с плавающей запятой. Преобразование выполняется для каждого компонента.

Если целочисленное входное значение слишком велико для точного представления в формате с плавающей запятой, округление до ближайшего четного режима.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
