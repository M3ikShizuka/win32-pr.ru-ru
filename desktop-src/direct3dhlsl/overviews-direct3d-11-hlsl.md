# <a name="hlsl-shader-model-5"></a>Модель шейдера HLSL 5

В этом разделе содержится обзорный материал для языка High-Level шейдера, в частности новые функции в модели шейдеров 5, появившиеся в Microsoft Direct3D 11.

## <a name="in-this-section"></a>в этом разделе



| Элемент                                                                                                                                                                                                               | Описание                                                                                                                                                                                                                                                                                                                                                            |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="Dynamic_Linking"></span><span id="dynamic_linking"></span><span id="DYNAMIC_LINKING"></span>[Динамическая компоновка](overviews-direct3d-11-hlsl-dynamic-linking.md)<br/>                                 | Динамическое связывание позволяет среде выполнения принимать решение во время рисования (а не во время компиляции) о том, какой путь кода следует запустить. Это сокращает проблему распространения шейдера, вызванную появлением шейдеров практически идентичными входными сигнатурами.<br/>                                                                                                                         |
| <span id="Geometry_Shader_Features"></span><span id="geometry_shader_features"></span><span id="GEOMETRY_SHADER_FEATURES"></span>[Функции шейдера Geometry](overviews-direct3d-11-hlsl-gs-features.md)<br/> | Новые функции шейдера Geometry, в том числе: создание экземпляров, которое обеспечивает повышение производительности, когда порядок примитивов в потоке не имеет значения, и несколько потоков вывода, поэтому шейдер может выводить вершины в нескольких потоках.<br/>                                                                                                                |
| <span id="Tessellation"></span><span id="tessellation"></span><span id="TESSELLATION"></span>[Тесселяции](/windows/desktop/direct3d11/direct3d-11-advanced-stages-tessellation)<br/>                                        | Среда выполнения Direct3D 11 поддерживает три новых этапа, реализующих тесселяцию, которая преобразует поверхности подразделений с низким уровнем детализации в примитивы более высокого уровня для GPU. Тесселяции разбивает на плитки поверхности старшего порядка на структуры, подходящие для прорисовки. Три этапа тесселяции: поверхности-шейдер, тесселяция и поэтапный шейдер домена.<br/> |
