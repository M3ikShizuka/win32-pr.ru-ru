# <a name="relative-addressing-hlsl-ps-reference"></a>Относительная адресация (Справочник по HLSL PS)

\[ \] Синтаксис можно использовать только в типах регистров, которые могут быть относительно устранены в некоторых моделях шейдеров. Ниже перечислены поддерживаемые формы \[ \] синтаксиса.

Где:

-   "R" обозначает любой тип регистра, который может быть относительно адресован.
-   "A" обозначает любой регистр, который можно использовать в качестве индекса для относительно других регистров.
-   N0 — Ni, M0-MJ и k — целые числа >= 0.



| Синтаксис \[ \]                              | Эффективный индекс                       | Примеры                         |
|-------------------------------------------|---------------------------------------|----------------------------------|
| R \[ A + M0 +... + MJ \]                  | A + M0 +... + MJ                     | c \[ a0. x + 3 + 7 \]              |
| R \[ k \] (= ать)                         | k                                     | c \[ 10 \] (= C10)              |
| R \[ A \]                                  | Объект                                     | c \[ a0. y \]                      |
| Ать \[ N0 +... + Ni + A + M0 +... + MJ \] | A + k + N0 +... + Ni + M0 +... + MJ | C8 \[ 3 + 2 + a0. w + 5 + 6 + 1 \] |
| R \[ N0 +... + Ni + A + M0 +... + MJ \]  | A + N0 +... + Ni + M0 +... + MJ     | c \[ 2 + 1 + Al + 3 + 4 + 5 \]    |
| Ать \[ A \]                                 | A + k                                 | C12 \[ Al \] , C0 \[ a0. z \]        |
| Ать \[ A + M0 +... + MJ \]                 | A + k + M0 +... + MJ                 | v1 \[ Al + 4 + 8 \]               |
| R \[ N0 +... + Ni + A \]                  | A + N0 +... + Ni                     | o \[ 3 + 1 + Al \]                |
| Ать \[ N0 +... + Ni + A \]                 | A + k + N0 +... + Ni                 | O1 \[ 2 + 1 + 3 + Al \]           |



 

Регистры доступны в следующих версиях:



| Тип регистра                                                                                   | Версии шейдеров пикселей |
|-------------------------------------------------------------------------------------------------|-----------------------|
| [Счетчик цикла](dx9-graphics-reference-asm-ps-registers-loop-counter.md): Al во входных регистрах | PS \_ 3 \_ и выше   |
