# <a name="texturecubesamplegrad-methods"></a>Методы Текстурекубе:: Самплеград

Выбор текстуры с помощью градиента, чтобы повлиять на способ вычисления расположения образца.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                 | Описание                                                                                                                                                                                                        |
|:-------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплеград (S, float, float, float)**](dx-graphics-hlsl-to-samplegrad.md)                              | Выбор текстуры с помощью градиента, влияющей на способ вычисления расположения образца.<br/>                                                                                                             |
| [**Самплеград (S, float, float, float, float)**](tcube-samplegrad-s-float-float-float-float-.md)           | Выбор текстуры с помощью градиента, влияющей на способ вычисления расположения выборки, с дополнительным значением для фиксации значений уровня детализации (Лод).<br/>                                     |
| [**Самплеград (S, float, float, float, float, uint)**](tcube-samplegrad-s-float-float-float-float-uint-.md) | Выбор текстуры с помощью градиента, влияющей на способ вычисления расположения выборки, с дополнительным значением для фиксации значений уровня детализации (Лод). Возвращает состояние операции.<br/> |
