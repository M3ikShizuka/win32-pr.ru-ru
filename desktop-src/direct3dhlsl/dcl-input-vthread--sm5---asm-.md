# <a name="dcl_input-vthread-sm5---asm"></a>дкл \_ input всреад (SM5-ASM)

Объявите идентификаторы входных данных шейдера вычислений.



| дкл \_ Ввод {vThreadID.XYZ \|vThreadGroupID.xyz \| vThreadIDInGroup.xyz \|Всреадидинграупфлаттенед} |
|--------------------------------------------------------------------------------------------------|



 



| Элемент                                                                                                                                                                                                                                                                                                                                                                          | Описание                                                         |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------|
| <span id="vThreadID___vThreadGroupID___vThreadIDInGroup___vThreadIDInGroupFlattened"></span><span id="vthreadid___vthreadgroupid___vthreadidingroup___vthreadidingroupflattened"></span><span id="VTHREADID___VTHREADGROUPID___VTHREADIDINGROUP___VTHREADIDINGROUPFLATTENED"></span>*Всреадид \| всреадграупид \| всреадидинграуп \| всреадидинграупфлаттенед*<br/> | \[в \] 3-компонентном 32-разрядном целочисленном значении идентификатора.<br/> |



 

**\_ входные данные дкл** — это существующее объявление в других стадиях шейдера. Он используется в шейдере вычислений для объявления различных значений 3-разрядного целочисленного идентификатора из трех 32 компонентов, уникальных для шейдера вычислений. К ним относятся:

-   vThreadID.xyz
-   vGroupID.xyz
-   vThreadIDInGroup.xyz
-   Всреадидинграупфлаттенед (один компонент)

Эта инструкция применяется к следующим этапам шейдера:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
|        |      |        |          |       | X       |

| [Модель шейдера 4](dx-graphics-hlsl-sm4.md)                | Нет        |
| [Модель шейдера 3 (DirectX HLSL)](dx-graphics-hlsl-sm3.md) | Нет        |

| [Модель шейдера 1 (DirectX HLSL)](dx-graphics-hlsl-sm1.md) | Нет        |
