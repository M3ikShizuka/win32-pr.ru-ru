# <a name="texturecubearraygatheralpha-methods"></a>Методы Текстурекубеаррай:: Гасералфа

Возвращает альфа-компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                    | Описание                                                                                                                                      |
|:--------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Гасералфа (S, float, uint)**](tcubearray-gatheralpha-s-float-uint-.md) | Возвращает альфа-компоненты четырех значений шаг текселя, которые будут использоваться в операции линейной фильтрации, а также состояние сопоставления плиток.<br/> |
