# <a name="statements"></a>Операторы

HLSL операторы — это сочетания переменных и функций, составляющих выражения. как и в случае с предложением, оператор HLSL требует операторов, определяющих, как будет оцениваться выражение. Вы можете создать отдельные инструкции в шейдере или создать несколько операторов в группе, называемой блоком инструкций. Чтобы создать инструкции HLSL, необходимо знать синтаксис написания выражений и операторы, которые будут использоваться для объединения выражений.

-   [Выражения](dx-graphics-hlsl-expressions.md)
-   [Инструкции](dx-graphics-hlsl-operators.md)
-   [Блоки инструкций](dx-graphics-hlsl-statement-blocks.md)
