# <a name="discard-sm4---asm"></a>отменить (SM4-ASM)

Условно отметьте результаты построителя текстуры как отброшенные при достижении конца программы.



| отменить { \_ z \|\_NZ} src0. Выбор \_ компонента |
|-------------------------------------------|



 



| Элемент                                                            | Описание                                                                                       |
|-----------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[\]значение, определяющее, следует ли отбрасывать текущую обрабатываемую точку.<br/> |



 

## <a name="remarks"></a>Примечания

Эта инструкция Помечает текущий пиксель как завершенный, продолжая выполнение, чтобы другие Пиксели, выполняющиеся параллельно, могли получить производные при необходимости. Несмотря на то, что выполнение продолжится, все выходные данные шейдера пиксельных записываются до или после инструкции **Discard** отбрасываются.

Для **отмены \_ z**, если все биты в *src0. Select \_* равны нулю, пиксель отбрасывается.

Если в параметре " **отменить \_ NZ**" какие-либо биты в *src0. Select \_* являются ненулевыми, пиксель отбрасывается.

Кроме того, инструкция **Discard** может присутствовать внутри любой конструкции управления потоком.

В шейдере могут присутствовать несколько инструкций по **удалению** , и если они выполняются, точка завершается.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
|               |                 | x            |
