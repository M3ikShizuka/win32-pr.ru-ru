# <a name="texturecubearraysamplebias-methods"></a>Методы Текстурекубеаррай:: Самплебиас

Выбор текстуры после применения значения смещения к уровню mipmap.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                          | Описание                                                                                                                                                                                  |
|:------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплебиас (S, float, float)**](dx-graphics-hlsl-to-samplebias.md)                             | Выбор текстуры после применения значения смещения к уровню mipmap.<br/>                                                                                                             |
| [**Самплебиас (S, float, float, float)**](tcubearray-samplebias-s-float-float-float-.md)           | Демонстрирует выбор текстуры после применения значения смещения к уровню mipmap с дополнительным значением для создания среза образцов значений уровня детализации (Лод).<br/>                                     |
| [**Самплебиас (S, float, float, float, uint)**](tcubearray-samplebias-s-float-float-float-uint-.md) | Демонстрирует выбор текстуры после применения значения смещения к уровню mipmap с дополнительным значением для создания среза образцов значений уровня детализации (Лод). Возвращает состояние операции.<br/> |
