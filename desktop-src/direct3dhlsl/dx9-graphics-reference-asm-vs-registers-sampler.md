# <a name="sampler-direct3d-9-asm-vs"></a>Образцы (Direct3D 9 ASM-VS)

Образец — это входная псевдо-регистрация для шейдера вершин, которая используется для указания этапа выборки. Существует четыре пробных шейдера вершин: S0 — S3. Для одного прохода шейдера можно считать четыре поверхности текстуры.

Образцы (Direct3D 9 ASM-VS) s являются псевдо, так как вы не можете напрямую читать их или записывать.

Единица выборки соответствует этапу выборки текстуры с инкапсуляцией состояния, зависящего от выборки, предоставляемого [**сетсамплерстате**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9-setsamplerstate). Каждая выборка уникально определяет одну текстурную поверхность, которая устанавливается в соответствующий образец с помощью [**сеттекстуре**](/windows/desktop/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-settexture). Однако одну и ту же поверхность текстуры можно задать в нескольких пробах.

Во время рисования текстуру нельзя одновременно задать как цель рендеринга и текстуру на этапе.

Поскольку существует четыре пробы, можно считывать до четырех поверхностей текстуры из одного прохода шейдера. Образец может отображаться в виде единственного аргумента в инструкции загрузки текстуры: [текслдл-VS](texldl---vs.md).

В VS \_ 3 \_ 0, если используется образец, его необходимо объявить в начале программы шейдера с помощью инструкции [дкл \_ самплертипе (SM3-VS ASM)](dcl-samplertype---vs.md) .



| Версии шейдеров вершин | 1\_1 | 2 \_ 0 | 2 \_ SW | 2 \_ x | 3 \_ 0 | 3 \_ SW |
|------------------------|------|------|-------|------|------|-------|
| Образец                |      |      |       |      | x    | x     |
