# <a name="dcl_tgsm_structured-sm5---asm"></a>дкл \_ тгсм \_ Structured (SM5-ASM)

Объявите ссылку на область общей памяти, доступную для группы потоков COMPUTE Shader s. Память просматривается как массив структур.



| дкл \_ тгсм \_ Structured g \# , структбитестриде, структкаунт |
|----------------------------------------------------------|



 



| Элемент                                                                                                                                   | Описание                                                                                                   |
|----------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| <span id="g_"></span><span id="G_"></span>*модуле\#*<br/>                                                                             | \[в \] ссылке на блок общей памяти размером *структбитестриде* \* *структкаунт* байт. <br/> |
| <span id="structByteStride"></span><span id="structbytestride"></span><span id="STRUCTBYTESTRIDE"></span>*структбитестриде*<br/> | \[в \] структуре STRIDE. Это значение равно uint в байтах и должно быть кратно 4. <br/>           |
| <span id="structCount"></span><span id="structcount"></span><span id="STRUCTCOUNT"></span>*структкаунт*<br/>                     | \[в \] количестве структур.<br/>                                                                   |



 

## <a name="remarks"></a>Примечания

Общий объем хранилища для всех g \# должен быть <= объем общей памяти, доступной для каждой группы потоков, которая представляет собой 32 КБ или 8192 32-разрядные скаляры.

В экстремальном случае можно объявить 8192 с суммой g \# s, если каждый из них имеет *структбитестриде* 4 и *структкаунт* 1.

В противоположной крайней части можно объявить одну g \# с шагом структуры 32 КБ и числом структур, равным 1.

Эта инструкция применяется к следующим этапам шейдера:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
|        |      |        |          |       | X       |
