# <a name="d3dx_isnan-function"></a>D3DX \_ IsNaN, функция

Определяет, является ли значение NaN (не числом).

## <a name="syntax"></a>Синтаксис

``` syntax
bool D3DX_IsNan(
   FLOAT _V
);
```

## <a name="parameters"></a>Параметры

<dl> <dt>

*\_V* 
</dt> <dd>

Указанное значение.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

значение true, если значение NaN; в противном случае — false.
