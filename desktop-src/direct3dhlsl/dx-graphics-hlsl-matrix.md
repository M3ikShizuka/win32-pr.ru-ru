# <a name="matrix-type"></a>Тип матрицы

Матрица — это специальный тип данных, содержащий от одного до шестнадцати компонентов. Каждый компонент матрицы должен быть одного типа.



|                         |
|-------------------------|
| **Имя Типекомпонентс** |



 

## <a name="components"></a>Компоненты



| Элемент                                                                                                                             | Описание                                                                                                                                                                                                                                                                                       |
|----------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TypeComponents"></span><span id="typecomponents"></span><span id="TYPECOMPONENTS"></span>**типекомпонентс**<br/> | Одно имя, содержащее три части. Первая часть является одним из [скалярных](dx-graphics-hlsl-data-types.md) типов. Вторая часть — число строк. Третья часть — число столбцов. Число строк и столбцов является положительным целым числом от 1 до 4 включительно.<br/> |
| <span id="Name"></span><span id="name"></span><span id="NAME"></span>**Безымян**<br/>                                         | Строка ASCII, однозначно определяющая имя переменной.<br/>                                                                                                                                                                                                                            |



 

## <a name="examples"></a>Примеры

Ниже приводится несколько примеров.


```
int1x1    iMatrix;   // integer matrix with 1 row,  1 column
int4x1    iMatrix;   // integer matrix with 4 rows, 1 column
int1x4    iMatrix;   // integer matrix with 1 row, 4 columns
double3x3 dMatrix;   // double matrix with 3 rows, 3 columns

float2x2 fMatrix = { 0.0f, 0.1, // row 1
                     2.1f, 2.2f // row 2
                   };   
```



Матрицу можно объявить с помощью этого синтаксиса:


```
matrix <Type, Number> VariableName
```



Тип матрицы использует угловые скобки для указания типа, числа строк и числа столбцов. В этом примере создается матрица с плавающей точкой с двумя строками и двумя столбцами. Можно использовать любой скалярный тип данных.

Пример:


```
matrix <float, 2, 2> fMatrix = { 0.0f, 0.1, // row 1
                                 2.1f, 2.2f // row 2
                               };
```
