# <a name="absolute-value"></a>Абсолютное значение

Получение абсолютного значения исходного операнда, используемого в арифметической операции.



| \_просто |
|-------|



 

Этот модификатор используется только для одиночной и двойной точности с плавающей запятой и инструкций. Модификатор **\_ ABS** вызывает срабатывание знака чисел в исходном операнде, в том числе для значений INF.

Применение функции **\_ ABS** к NaN сохраняет значение NaN, хотя определенный битовый шаблон NaN не определен.

Если \_ ABS сочетается с модификатором [отрицания](negate.md) , это сочетание приводит к тому, что знак будет отрицательным, как если бы первым был применен модификатор **\_ ABS** , а затем — **отрицание**.
