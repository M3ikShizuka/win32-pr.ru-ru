### <a name="interpolation-notes"></a>Примечания к интерполяции

По умолчанию при выполнении многовыборочного сглаживания атрибуты вершин интерполируются из пикселного центра. Если пиксельный центр не охватывается, то перед интерполяцией атрибут выравниваются по пикселному центру.

Для пикселя, который не полностью покрывается, или атрибута, не охватывающего пиксельный центр, можно указать центроид выборку, которая вынуждает выборку будет выполняться где-то в охваченной области пикселя. Так как образец маски (если используется) применяется перед вычислением центроид, любое расположение выборки, маскированное с помощью образца маски, не может быть выбрано в качестве расположения центроид.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |



 

Чтобы указать входные данные как системное значение, [Используйте \_ входной дкл \_ ОКП (SM4-ASM)](dcl-input-sv.md).

Эта инструкция включена для облегчения отладки шейдера в сборке; нельзя создать шейдер на языке ассемблера с использованием модели шейдеров 4.
