# <a name="dcl_indexabletemp-sm4---asm"></a>дкл \_ индексаблетемп (SM4-ASM)

Объявляет индексируемый временный регистр.



| дкл \_ индексаблетемп x *N \[ \] , компоненткаунт* |
|-------------------------------------------------|



 



| Элемент                                                                                                                           | Описание                                                                                 |
|--------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| <span id="N"></span><span id="n"></span>*\N*<br/>                                                                         | \[\]целое число, обозначающее номер регистра.<br/>                              |
| <span id="_size_"></span><span id="_SIZE_"></span>*\[изменять\]*<br/>                                                        | \[в \] необязательном целочисленном значении. Число элементов в массиве регистров.<br/>  |
| <span id="ComponentCount"></span><span id="componentcount"></span><span id="COMPONENTCOUNT"></span>*компоненткаунт*<br/> | \[в \] необязательном целочисленном значении. Число компонентов в массиве регистров.<br/> |



 

Регистр содержит достаточно места для 32-разрядного значения из четырех компонентов; число элементов в массиве временных регистров (индексируемое и [неиндексируемое](dcl-temps.md)) не может превышать 4096.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |



 

Эта инструкция включена для облегчения отладки шейдера в сборке; нельзя создать шейдер на языке ассемблера с использованием модели шейдеров 4.

## <a name="example"></a>Например, .

Ниже приведено несколько примеров кода, созданного для индексируемых регистров.


```
dcl_indexableTemp x0[23], 2 ; // An indexable array of 23, 2-component, 32-bit elements
dcl_indexableTemp x1[16], 4 ; // An indexable array of 16, 4-component, 32-bit elements
```
