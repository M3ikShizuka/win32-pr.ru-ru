# <a name="processtritessfactorsmax-function"></a>Функция Процесстритессфакторсмакс

Создает исправленные коэффициенты тесселяции для более трехуровневого обновления.

## <a name="syntax"></a>Синтаксис

``` syntax
void ProcessTriTessFactorsMax(
  in  float3 RawEdgeFactors,
  in  float InsideScale,
  out float3 RoundedEdgeTessFactors,
  out float RoundedInsideTessFactor,
  out float UnroundedInsideTessFactor
);
```

## <a name="parameters"></a>Параметры

<dl> <dt>

*Раведжефакторс* \[ окне\]
</dt> <dd>

Тип: **float3**

Факторы тесселяции краев, передаваемые в стадию тесселяции.

</dd> <dt>

*Инсидескале* \[ окне\]
</dt> <dd>

Тип: **float**

Коэффициент масштабирования, применяемый к коэффициентам тесселяции UV, вычисленным на этапе тесселяции. Допустимый диапазон для Инсидескале — от 0,0 до 1,0.

</dd> <dt>

*Раундедеджетессфакторс* \[ заполняет\]
</dt> <dd>

Тип: **float3**

Скругленные границы — коэффициенты тесселяции, вычисленные на этапе тесселяции.

</dd> <dt>

*Раундединсидетессфактор* \[ заполняет\]
</dt> <dd>

Тип: **float**

Коэффициенты тесселяции, вычисляемые на этапе тесселяции, и округляются.

</dd> <dt>

*Унраундединсидетессфактор* \[ заполняет\]
</dt> <dd>

Тип: **float**

Первоначальные, нескругленные, UV, вычисляются на этапе тесселяции.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Примечания

Формирует исправленные коэффициенты тесселяции для трехуровневого обновления, вычисляя внутренний фактор тесселяции в качестве максимального значения факторов тесселяции краев, которые затем масштабируются по Инсидескале. Затем результат округляется в зависимости от режима секционирования, но неокругленные результаты доступны с помощью параметра Унраундединсидетессфактор.
