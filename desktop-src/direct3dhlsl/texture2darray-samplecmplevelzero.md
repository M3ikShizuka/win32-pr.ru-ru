# <a name="texture2darraysamplecmplevelzero-methods"></a>Методы Texture2DArray:: Самплекмплевелзеро

Пример [**Texture2DArray**](sm5-object-texture2darray.md) только для mipmap уровня 0 с использованием значения сравнения для отклонения выборок.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                                    | Описание                                                                                                                            |
|:----------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплекмплевелзеро (S, float, float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)                   | Производит выборку текстуры только на уровне mipmap 0 и сравнивает результат со значением сравнения.<br/>                                     |
| [**Самплекмплевелзеро (S, float, float, int, uint)**](t2darray-samplecmplevelzero-s-float-float-int-uint-.md) | Производит выборку текстуры только на уровне mipmap 0 и сравнивает результат со значением сравнения. Возвращает состояние операции.<br/> |
