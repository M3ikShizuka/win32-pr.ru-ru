# <a name="ult-sm4---asm"></a>метры (SM4-ASM)

Покомпонентное векторное целое число без знака, меньшее, чем сравнение.



| метры dest \[ . Mask \] , src0 \[ . свиззле \] , src1 \[ . свиззле\] |
|-------------------------------------------------------|



 



| Элемент                                                            | Описание                                                   |
|-----------------------------------------------------------------|---------------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[в \] адресе результата операции.<br/> |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[в \] значении, сравниваемом с *src1*.<br/>             |
| <span id="src1"></span><span id="SRC1"></span>*src1*<br/> | \[в \] значении, сравниваемом с *src1*.<br/>             |



 

## <a name="remarks"></a>Примечания

Выполняет сравнение целых чисел без знака (*src0*  <  *src1*) для каждого компонента и записывает результат в *dest*.

Если сравнение истинно, эта инструкция возвращает значение 0xFFFFFFFF для этого компонента. В противном случае возвращается 0x0000000.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
