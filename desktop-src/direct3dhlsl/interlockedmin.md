# <a name="interlockedmin-function-hlsl-reference"></a>Функция Интерлоккедмин (Справочник по HLSL)

Выполняет гарантированную атомарную минуту.

## <a name="syntax"></a>Синтаксис

``` syntax
void InterlockedMin(
  in  R dest,
  in  T value,
  out T original_value
);
```

## <a name="parameters"></a>Параметры

<dl> <dt>

*конечный адрес* \[ окне\]
</dt> <dd>

Тип: **R**

Адрес назначения.

</dd> <dt>

*значение* \[ окне\]
</dt> <dd>

Тип: **T**

Входное значение.

</dd> <dt>

*исходное \_ значение* \[ out\]
</dt> <dd>

Тип: **T**

Необязательный элемент. Исходное входное значение.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Remarks

Эта операция может выполняться только с типизированными ресурсами int и uint и переменными общей памяти. Существует два возможных использования этой функции. Первый — когда R является переменной общей памяти. В этом случае функция выполняет атомарный минимум значения в регистр общей памяти, на который ссылается dest. Второй сценарий заключается в том, что R является типом переменной ресурса. В этом сценарии функция выполняет атомарную минуту значения в расположении ресурса, на которое ссылается dest. Перегруженная функция имеет дополнительную выходную переменную, которой будет присвоено исходное значение dest. Эта перегруженная операция доступна только в том случае, если R доступен для чтения и записи.
