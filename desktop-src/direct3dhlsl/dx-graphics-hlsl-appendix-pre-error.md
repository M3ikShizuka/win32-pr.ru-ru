# <a name="error-directive"></a>\#Директива Error

Директива препроцессора, которая создает сообщения об ошибках во время компиляции.



| \#*маркер ошибки — строка* |
|------------------------|



 

## <a name="parameters"></a>Параметры



| Элемент                                                                                    | Описание                                                                                                                                                                    |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="token-string"></span><span id="TOKEN-STRING"></span>*Строка токена*<br/> | Сообщение об ошибке. Этот параметр состоит из ряда токенов, таких как ключевые слова, константы или полные инструкции. Строка токена подлежит раскрытию макросов. <br/> |



 

## <a name="remarks"></a>Remarks

\#директивы Error наиболее полезны для обнаружения несоответствий в программировании и нарушения ограничений во время предварительной обработки. При \# обнаружении директивы ошибки компиляция завершается.

## <a name="examples"></a>Примеры

В следующем примере демонстрируется обработка ошибок во время предварительной обработки.


```
#if !defined(__cplusplus)
  #error C++ compiler required.
#endif
```
