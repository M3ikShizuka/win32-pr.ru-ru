# <a name="firstbit-sm5---asm"></a>фирстбит (SM5-ASM)

Находит первый бит, заданный в числе, из ЛСБ или из самого начала.



| фирстбит { \_ Привет \|\_Lo|\_Ши} dest \[ . Mask \] , src0 \[ . свиззле\] |
|-------------------------------------------------------------|



 



| Элемент                                                            | Описание                                                                                                                           |
|-----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[в \] целочисленном положении первого бита в *src0* , начиная с ЛСБ для фирстбит в \_ фирстбит \_ Hi.<br/> |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[во \] входном целочисленном параметре.<br/>                                                                                                  |



 

## <a name="remarks"></a>Remarks

Эта операция возвращает целочисленное значение первого бита, заданного в 32-разрядном входном параметре, начиная с ЛСБ для фирстбит \_ или для фирстбит \_ Hi. Например, фирстбит \_ в 0x00000001 возвращает 0. фирстбит \_ Hi on 0x10000000 возвращает значение 3.

фирстбит \_ Ши (s for со знаком) возвращает первый 0 из нуля, если число отрицательное; в противном случае возвращается первый 1 из самого старшего.

Все варианты инструкции возвращают ~ 0 (0xFFFFFFFF в 32-разрядном регистре), если совпадений не найдено.

Используйте эту инструкцию, чтобы быстро перечислить биты набора битов или найти максимальную степень числа 2 в числе.

Эта инструкция применяется к следующим этапам шейдера:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Службы вычислений |
|--------|------|--------|----------|-------|---------|
| X      | X    | X      | X        | X     | X       |
