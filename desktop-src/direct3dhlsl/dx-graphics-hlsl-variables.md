# <a name="variables"></a>Переменные

Переменные HLSL похожи на переменные, определенные в языке программирования C. Подобно C, переменные имеют некоторые ограничения именования, имеют свойства области, зависящие от того, где они объявляются, и могут иметь к ним прикрепленные метаданные пользователя. Как и C, существует несколько стандартных типов данных. В отличие от языка C, существуют также дополнительные типы данных, определяемые HLSL, чтобы обеспечить максимальную производительность 4-компонентных векторов, использующих математические формулы для работы с данными трехмерной графики.
