# <a name="language-syntax"></a>Синтаксис языка

Шейдеры HLSL состоят из переменных, а функции, которые, в свою очередь, состоят из инструкций. Языковой синтаксис описывает определение и объявление переменных, добавление управления потоком, чтобы шейдер мог принимать решения во время выполнения на основе переменных и создавать пользовательские функции.
