# <a name="using-shader-linking"></a>Использование связывания шейдера

Мы покажем, как создавать предварительно скомпилированные функции HLSL, упаковывать их в библиотеки и связывать их с полными шейдерами во время выполнения. Связывание шейдера поддерживается начиная с Windows 8.1.

**Цель:** Узнайте, как использовать связывание с шейдером.

## <a name="prerequisites"></a>Предварительные требования

Предполагается, что вы знакомы с C++. Также вы должны быть знакомы с основными принципами программирования графики.

**Общее время выполнения:** 60 минут.

## <a name="where-to-go-from-here"></a>Дальнейшие действия

См. также [интерфейсы API компилятора HLSL](dx-graphics-d3dcompiler-reference.md).

Здесь показаны следующие действия:

-   Компиляция кода шейдера
-   Загрузка скомпилированного кода в библиотеку шейдера
-   Привязка ресурсов из исходных слотов к конечным слотам
-   Создание графов функций-связей (Флгс) для шейдеров
-   Связывание графов шейдера с библиотекой шейдеров для создания большого двоичного объекта шейдера, который может использоваться средой выполнения Direct3D

Далее мы создадим библиотеку шейдеров и связываем ресурсы из исходных слотов с конечными слотами.

[Упаковка библиотеки шейдеров](pachaging-a-shader-library.md)
