# <a name="d3dx_float_to_srgb-function"></a>D3DX \_ с плавающей запятой \_ к \_ функции sRGB

## <a name="syntax"></a>Синтаксис

``` syntax
FLOAT D3DX_FLOAT_to_SRGB(
   FLOAT val
);
```

## <a name="parameters"></a>Параметры

<dl> <dt>

*Val* 
</dt> <dd>

Преобразуемое значение FLOAT.

</dd> </dl>
