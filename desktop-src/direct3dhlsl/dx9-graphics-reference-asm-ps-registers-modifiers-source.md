# <a name="pixel-shader-source-register-modifiers"></a>Модификаторы исходных регистров исходного шейдера пикселей

Используйте модификаторы исходного регистра, чтобы изменить значение, считанное из регистра, перед выполнением инструкции. Содержимое регистра источника остается неизменным. Модификаторы полезны для настройки диапазона регистрируемых данных в процессе подготовки к использованию инструкции. Набор модификаторов, именуемых селекторами, копирует или реплицирует данные из одного канала (r, g, b, a) в другие каналы.

Модификаторы исходного регистра можно использовать только в арифметических инструкциях. Их нельзя использовать в инструкциях по адресам текстур. Исключением является модификатор масштабирования на [2](dx9-graphics-reference-asm-ps-registers-modifiers-scale-x2.md) . Для версии 1 \_ 1 со знаком Scale можно использовать исходный аргумент любой \* инструкции тексм. Для версии 1 \_ 2 или 1 \_ 3 со знаком Scale можно использовать в аргументе Source любой инструкции адреса текстуры.

Некоторые ограничения для определенных модификаторов:

-   Отрицание можно сочетать с модификатором смещения, подписанного масштабирования или scalex2. При объединении обратная работа выполняется в последнюю очередь.
-   Инвертировать нельзя использовать вместе с любым другим модификатором.
-   Инверсия, отрицания, смещение, масштабирование со знаком и scalex2 можно сочетать с любыми селекторами.
-   Модификаторы исходного регистра не следует использовать в регистрах констант, так как они приводят к неопределенным результатам. Для версии 1 \_ 4 модификаторы константы не допускаются и не проходят проверку.

-   Все константные регистры с плавающей запятой должны использовать модификатор ABS.
