# <a name="texture2dsamplecmplevelzero-methods"></a>Методы Texture2D:: Самплекмплевелзеро

Пример [**Texture2D**](sm5-object-texture2d.md) только для mipmap уровня 0 с использованием значения сравнения для отклонения выборок.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                                           | Описание                                                                                                                            |
|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**Самплекмплевелзеро (S, float, float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)          | Производит выборку текстуры только на уровне mipmap 0 и сравнивает результат со значением сравнения.<br/>                                     |
| [**Самплекмплевелзеро (S, float, float, int, uint)**](samplecmplevelzero-s-float-float-int-uint-.md) | Производит выборку текстуры только на уровне mipmap 0 и сравнивает результат со значением сравнения. Возвращает состояние операции.<br/> |
