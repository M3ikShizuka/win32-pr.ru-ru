## <a name="remarks"></a>Remarks

Эта инструкция выполняет 4 \* -разрядный элемент 32, записанный из *Src0* в *дстуав* по адресу в *дстаддресс*. *дстуав* — это типизированный UAV (u \# ).

Формат UAV определяет преобразование формата.

Число 32-разрядных беззнаковых целых чисел, полученных из адреса, определяется размерностью ресурса, объявленного на *дстуав*. Этот адрес находится в элементах.

Для адресации вне границ означает, что в память ничего не записывается.

*дстуав* всегда имеет маску записи. ксизв. Все компоненты должны быть записаны.

Он недопустим и не определен, чтобы использовать эту инструкцию для UAV, который не объявлен как типизированный. Это значит, что выполнение этого действия в структурированном или нетипизированном UAV недопустимо.

Эта инструкция применяется к следующим этапам шейдера:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
|        |      |        |          | X     | X       |



 

Так как Уавс доступны на всех стадиях шейдера для Direct3D 11,1, эта инструкция применяется ко всем этапам шейдера для среды выполнения Direct3D 11,1, которая доступна начиная с Windows 8.
