# <a name="dcl_function_body-sm5---asm"></a>\_тело функции дкл \_ (SM5-ASM)

Объявите тело функции.



| \_тело функции дкл \_ FB\# |
|--------------------------|



 



| Элемент                                                          | Описание                                                              |
|---------------------------------------------------------------|--------------------------------------------------------------------------|
| <span id="fb_"></span><span id="FB_"></span>*FB\#*<br/> | \[в \] метке места, где будет отображаться функция.<br/> |



 

## <a name="remarks"></a>Remarks

Эта инструкция объявляет уникальный текст функции, код которого будет отображаться позже в программе в метке FB \# .

Тела функций используются в объявлениях таблиц функций. Дополнительные сведения см. в [разделе \_ \_ Таблица функций дкл](dcl-function-table---sm5---asm-.md).

В шейдере поверхности и шейдере доменов, где существует несколько этапов (этап контрольной точки, фаза ветвления и фаза соединения), все тела функций (метка FB \# ) отображаются после всех этапов, а не группируются по этапам.

Не существует ограничения на количество существующих в теле функций.

Эта инструкция применяется к следующим этапам шейдера:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
| X      | X    | X      | X        | X     | X       |

Эта инструкция поддерживается в следующих моделях шейдеров:



| Модель шейдера                                              | Поддерживается |
|-----------------------------------------------------------|-----------|
| [Модель шейдера 5](d3d11-graphics-reference-sm5.md)        | Да       |
| [Модель шейдера 4,1](dx-graphics-hlsl-sm4.md)              | Нет        |
| [Модель шейдера 4](dx-graphics-hlsl-sm4.md)                | Нет        |
| [Модель шейдера 3 (DirectX HLSL)](dx-graphics-hlsl-sm3.md) | Нет        |
| [Модель шейдера 2 (DirectX HLSL)](dx-graphics-hlsl-sm2.md) | Нет        |
| [Модель шейдера 1 (DirectX HLSL)](dx-graphics-hlsl-sm1.md) | Нет        |
