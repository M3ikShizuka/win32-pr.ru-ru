# <a name="texreg2ar---ps"></a>texreg2ar-PS

Интерпретирует компоненты альфа-и красного цвета исходной регистрации как данные адреса текстуры (u, v) для выборки текстуры на этапе, соответствующей номеру регистра назначения. Результат сохраняется в реестре назначения.

## <a name="syntax"></a>Синтаксис



| texreg2ar DST, src |
|--------------------|



 

where

-   DST — это регистр назначения.
-   src является исходным регистром.

## <a name="remarks"></a>Remarks



| Версии шейдеров пикселей | 1\_1 | 1\_2 | 1 \_ 3 | 1\_4 | 2 \_ 0 | 2 \_ x | 2 \_ SW | 3 \_ 0 | 3 \_ SW |
|-----------------------|------|------|------|------|------|------|-------|------|-------|
| texreg2ar             | x    | x    | x    |      |      |      |       |      |       |



 

Эта инструкция полезна для операций повторного сопоставления цветового пространства.
