# <a name="round_pi-sm4---asm"></a>Round \_ Pi (SM4-ASM)

Округление числа с плавающей точкой до целого числа с плавающей запятой.



| округлить \_ Pi \[ \_ \] , конечный \[ . Mask \] , \[ - \] src0 \[ \_ ABS \] \[ . свиззле\] |
|------------------------------------------------------------------|



 



| Элемент                                                            | Описание                                                    |
|-----------------------------------------------------------------|----------------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[в \] адресе результатов операции.<br/> |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[в \] компонентах операции.<br/>             |



 

## <a name="remarks"></a>Примечания

Эта инструкция выполняет покомпонентное округление значений с плавающей запятой в *src0*, записывая целочисленные значения с плавающей запятой в *dest*.

**Округление числа \_ Пи** округляет в сторону + Infinity, обычно известное как ceil ().

В следующей таблице показаны результаты, полученные при выполнении инструкции с различными классами чисел.

F означает ограничение по настоящему вещественному числу.



|          |          |        |             |        |        |             |        |          |         |
|----------|----------|--------|-------------|--------|--------|-------------|--------|----------|---------|
| **src**  | **-INF** | **-F** | **— денорма** | **-0** | **+0** | **+ денорма** | **+ F** | **+ INF** | **Не число** |
| **dest** | -inf     | -F     | -0          | -0     | +0     | +0          | + F     | +inf     | не число     |



 

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
