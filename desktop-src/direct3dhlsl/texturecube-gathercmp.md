# <a name="texturecubegathercmp-methods"></a>Методы Текстурекубе:: Гасеркмп

Для четырех шаг текселя значений [**текстурекубе**](texturecube.md) , которые будут использоваться в операции линейной фильтрации, возвращает их сравнение со значением сравнения.

Дополнительные сведения, описывающие базовую инструкцию DXBC, см. в документации по [gather4_c](./gather4-c--sm5---asm-.md) .

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                       | Описание                                                                                                      |
|:-----------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------|
| [**Гасеркмп (S, float, float, uint)**](tcube-gathercmp-s-float-float-uint-.md) | Выбирает и сравнивает текстуру и возвращает все четыре компонента вместе с состоянием операции.<br/> |
