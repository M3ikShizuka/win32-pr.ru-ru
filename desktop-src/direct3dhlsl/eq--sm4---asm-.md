# <a name="eq-sm4---asm"></a>EQ (SM4-ASM)

Сравнение равенства с плавающей точкой на уровне компонентов.



| EQ dest \[ . Mask \] , \[ - \] src0 \[ \_ ABS. \] \[ свиззле \] , \[ - \] src1 \[ \_ ABS \] \[ . свиззле\] |
|----------------------------------------------------------------------------------|



 



| Элемент                                                            | Описание                                                   |
|-----------------------------------------------------------------|---------------------------------------------------------------|
| <span id="dest"></span><span id="DEST"></span>*dest*<br/> | \[в \] адресе результата операции.<br/> |
| <span id="src0"></span><span id="SRC0"></span>*src0*<br/> | \[в \] компоненте, комапре в *src1*.<br/>         |
| <span id="src1"></span><span id="SRC1"></span>*src1*<br/> | \[в \] компоненте, комапре в *src0*.<br/>         |



 

## <a name="remarks"></a>Remarks

Выполняет сравнение с плавающей запятой (*src0*  ==  *src1*) для каждого компонента и записывает результат в *dest*.

Если сравнение истинно, для этого компонента возвращается значение 0xFFFFFFFF. В противном случае возвращается 0x0000000.

Перед сравнением (исходные регистры с нетронутым исходным кодом) выполняется очистка. + 0 равно-0. Сравнение с NaN возвращает false.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
