# <a name="fcall-sm5---asm"></a>фкалл (SM5-ASM)

Вызов функции интерфейса.



| фкалл FP \# \[ arrayIndex \] \[ callSite\] |
|--------------------------------------|



 



| Элемент                                                                                                           | Описание                                                                                                                                                                                                                                                                                                |
|----------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="fp_"></span><span id="FP_"></span>*FP\#*<br/>                                                  | \[в \] указателе функции.<br/>                                                                                                                                                                                                                                                                    |
| <span id="arrayIndex"></span><span id="arrayindex"></span><span id="ARRAYINDEX"></span>*arrayIndex*<br/> | \[\]необязательно. Задает смещение в массиве указателей функции. Этот параметр должен быть целочисленным литералом без знака, если FP \# не объявлен как индексируемый. В противном случае arrayIndex может иметь форму "базовый литерал + смещение" из регистра шейдера. Например, фкалл FP1 \[ R1. w + 0 \] \[ 0 \] .<br/> |
| <span id="_callSite"></span><span id="_callsite"></span><span id="_CALLSITE"></span>*callSite*<br/>     | \[\]необязательно. Смещение целого числа без знака в выбранной таблице функции, выбор тела функции FB \# для выполнения. <br/>                                                                                                                                                                |



 

## <a name="remarks"></a>Примечания

*FP \#* \[  \] \[ arrayIndex \] разрешается в определенную таблицу функций, выбранную из API за пределами шейдера, из вариантов таблицы функций, перечисленных в объявлении *FP \#*.

Сумма \# в *\# FP* и *arrayIndex* выберите таблицу функций. Например, если интерфейс объявлен как fp4 \[ 4 \] \[ 3 \] (размер массива 4), следующие **фкаллs** эквивалентны: фкалл fp4 \[ 2 \] \[ 3 \] и FP5 \[ 1 \] \[ 3 \] , так как 4 + 2 = 5 + 1.

### <a name="restrictions"></a>Ограничения

-   Если *arrayIndex* использует динамическое индексирование, поведение не определено, если *arrayIndex* расходят на смежных вызовах шейдера, которые могут выполняться в локкстеп. Компилятор HLSL будет пытаться запретить этот случай.

    Смежные вызовы могут быть неактивными из-за управления потоком, так как они не прерывать выполнение локкстеп.

-   Если *FP \#*  +  *arrayIndex* указывает индекс вне допустимого диапазона, поведение не определено.
-   В неопределенных случаях, описанных здесь, это означает, что поведение текущего устройства D3D станет неопределенным, включая возможность потери устройства. Однако доступ к памяти за пределами текущего устройства D3D будет невозможен или выполнен в виде кода.

Эта инструкция применяется к следующим этапам шейдера:



| Вершина | Поверхности | Домен | Геометрия | Пиксель | Вычисления |
|--------|------|--------|----------|-------|---------|
| X      | X    | X      | X        | X     | X       |
