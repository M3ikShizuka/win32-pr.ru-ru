# <a name="endswitch-sm4---asm"></a>ендсвитч (SM4-ASM)

Завершает оператор [switch](switch--sm4---asm-.md) .



| ендсвитч |
|-----------|



 

## <a name="remarks"></a>Remarks

Формат маркера содержит смещение соответствующей инструкции switch в шейдере для удобства.

Эта инструкция применяется к следующим этапам шейдера:



| Вершинный построитель текстуры | Шейдер геометрии | Построитель текстуры |
|---------------|-----------------|--------------|
| x             | x               | x            |
