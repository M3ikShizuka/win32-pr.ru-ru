# <a name="utilitiesbinarytohex-method"></a>Метод Utilities. Бинаритохекс

\[Метод **бинаритохекс** доступен для использования в операционных системах, указанных в разделе требования.\]

Метод **бинаритохекс** преобразует упакованную двоичную строку в шестнадцатеричную строку.

## <a name="syntax"></a>Синтаксис


```VB
Utilities.BinaryToHex( _
  ByVal BinaryString _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Бинаристринг* \[ окне\]
</dt> <dd>

Двоичная упакованная строка, которая должна быть преобразована в шестнадцатеричную строку.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Шестнадцатеричная строка, эквивалентная заданной двоичной упакованной строке.
