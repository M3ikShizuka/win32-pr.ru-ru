# <a name="microsoft-strong-cryptographic-provider"></a>Microsoft Strong Cryptographic Provider

Поставщик строгой криптографии (Майкрософт) используется в качестве полного [*поставщика служб шифрования*](../secgloss/c-gly.md) RSA по умолчанию (CSP). Он поддерживает все алгоритмы расширенного поставщика служб шифрования Майкрософт и все те же значения [*длины ключей*](../secgloss/k-gly.md). Сведения о длине ключей по умолчанию и алгоритмах см. в разделе базовый поставщик служб шифрования Майкрософт. Дополнительные сведения о более длинных значениях длины ключей и доступных алгоритмах см. в таблице с [расширенным поставщиком служб шифрования Майкрософт](microsoft-enhanced-cryptographic-provider.md).



|                |                      |
|----------------|----------------------|
| Тип поставщика: | **PROV \_ RSA \_ Full**  |
| Имя поставщика: | **MS \_ strong \_ Prov** |
