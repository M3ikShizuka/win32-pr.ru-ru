# <a name="hasheddata-object"></a>Объект Хашеддата

\[CAPICOM — это 32-разрядный компонент, доступный для использования в следующих операционных системах: Windows Server 2008, Windows Vista и Windows XP. Вместо этого используйте [**класс HashAlgorithm**](/previous-versions/windows/) в пространстве имен [**System. Security. Cryptography**](/dotnet/api/system.security.cryptography?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

Объект **хашеддата** предоставляет функциональные возможности для хэширования строки.

## <a name="when-to-use"></a>Назначение

Объект **хашеддата** используется для выполнения следующих задач:

-   Укажите строку содержимого, содержащую данные для хэширования.
-   Примените указанный хэш-алгоритм к строке содержимого.

## <a name="members"></a>Элементы

Объект **хашеддата** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **хашеддата** содержит эти методы.



| Метод                          | Описание                                        |
|:--------------------------------|:---------------------------------------------------|
| [**Функции**](hasheddata-hash.md) | Создает хэш указанной строки.<br/> |



 

### <a name="properties"></a>Свойства

Объект **хашеддата** имеет следующие свойства.



| Свойство                                             | Тип доступа           | Описание                                                                                                                                                                          |
|:-----------------------------------------------------|:----------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Алгоритм**](hasheddata-algorithm.md)<br/> | Чтение/запись<br/> | Задает или получает тип используемого алгоритма хэширования.<br/>                                                                                                                     |
| [**Значение**](hasheddata-value.md)<br/>         | Только для чтения<br/>  | Извлекает хэшированные данные после успешного вызова метода [**хэширования**](hasheddata-hash.md) . Хэш возвращается в шестнадцатеричном формате. Это свойство по умолчанию.<br/> |



 

## <a name="remarks"></a>Комментарии

Чтобы создать хэш большого объема данных, вызовите метод [**хэширования**](hasheddata-hash.md) для каждого фрагмента данных. Хэш каждого фрагмента данных объединяется со свойством [**value**](hasheddata-value.md) до тех пор, пока свойство не будет считано. Содержимое свойства **value** сбрасывается при считывании свойства.

Объект **хашеддата** может быть создан и защищен для создания скриптов. ProgID для объекта **хашеддата** — "CAPICOM. Хашеддата. 1 ".
