# <a name="prov_dss_dh-provider-algorithms"></a>\_ \_ Алгоритмы поставщика DH Prov DSS

В следующей таблице перечислены алгоритмы, поддерживаемые поставщиком служб Microsoft Base DSS и Diffie-Hellman.



| Идентификатор алгоритма      | Описание                                 | Комментарии                                                                                                        |
|-------------------|---------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| \_MD5 калг         | Алгоритм хэширования MD5.                      | Предоставляется только для хэширования.                                                                                      |
| \_SHA калг         | Алгоритм хэширования SHA.                      | Необходимо использовать для подписей DSS.                                                                                |
| КАЛГ \_ SHA1        | То же, что и КАЛГ \_ SHA.                          | Без комментариев.                                                                                                     |
| \_знак DSS \_ калг   | Алгоритм подписи открытого и закрытого ключей DSS. | Длина ключа: может быть задано, 512 бит на 1 024 бит в 64 бит. Длина ключа по умолчанию: 1 024 бит.<br/> |
| КАЛГ \_ DH \_ SF      | Обмен ключами хранения и пересылки D-H.         | Длина ключа: может быть задано, 384 бит на 512 бит по 8 битам. Длина ключа по умолчанию: 512 бит.<br/>      |
| КАЛГ \_ DH \_ ефем   | Обмен ключами эфемерного D-H.                 | Длина ключа: может быть задано, 384 бит на 512 бит по 8 битам. Длина ключа по умолчанию: 512 бит.<br/>      |
| КАЛГ \_ цилинк \_ главный ключ шифрования | 40-разрядный вариант ключа DES.              | Длина ключа: 40 бит.                                                                                            |
