# <a name="prov_rsa_full"></a>PROV \_ RSA \_ Full

\_ \_ Тип поставщика Prov RSA Full поддерживает [*цифровые подписи*](../secgloss/d-gly.md) и шифрование данных. Он считается CSP общего назначения. Алгоритм открытого ключа RSA используется для всех операций открытого ключа.

## <a name="algorithms-supported"></a>Поддерживаемые алгоритмы

Описание каждого из этих алгоритмов см. в глоссарии.



| Назначение      | Поддерживаемые алгоритмы                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| Обмен ключами | [*RSA*](../secgloss/r-gly.md)                                                                       |
| Подпись    | RSA                                                                                                                               |
| Шифрование   | [*RC2*](../secgloss/r-gly.md)-[*RC4*](../secgloss/r-gly.md)<br/> |
| Хэширование      | [*MD5*](../secgloss/m-gly.md)[*SHA*](../secgloss/s-gly.md)<br/> |
