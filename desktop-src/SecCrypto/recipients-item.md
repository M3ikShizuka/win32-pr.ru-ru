# <a name="recipientsitem-property"></a>Свойство Recipients. Item

\[Свойство **Item** доступно для использования в операционных системах, указанных в разделе требования. Вместо этого используйте [**класс кмсреЦипиентколлектион**](/dotnet/api/system.security.cryptography.pkcs.cmsrecipientcollection?view=dotnet-plat-ext-3.1&preserve-view=true) в пространстве имен [**System. Security. Cryptography. PKCS**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

Свойство **Item** извлекает объект из коллекции [**получателей**](recipients.md) . Это свойство по умолчанию.

## <a name="syntax"></a>Синтаксис


```VB
Recipients.Item( _
  ByVal Index _
) As Variant
```



## <a name="property-value"></a>Значение свойства

Вариант, представляющий индексированный элемент в коллекции [**получателей**](recipients.md) .
