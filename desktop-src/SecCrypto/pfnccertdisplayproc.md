# <a name="pfnccertdisplayproc-callback-function"></a>Функция обратного вызова ПФНКЦЕРТДИСПЛАЙПРОК

Функция **пфнкцертдисплайпрок** — это определяемая пользователем функция обратного вызова, которая позволяет вызывающей стороне функции [**криптуидлгселектцертификате**](cryptuidlgselectcertificate.md) управлять отображением сертификатов, которые пользователь выбирает для просмотра.

## <a name="syntax"></a>Синтаксис


```C++
BOOL WINAPI * PFNCCERTDISPLAYPROC(
  _In_ PCCERT_CONTEXT pCertContext,
  _In_ HWND           hWndSelCertDlg,
  _In_ void           *pvCallbackData
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пцертконтекст* \[ окне\]
</dt> <dd>

Указатель на структуру [**\_ контекста сертификата**](/windows/desktop/api/Wincrypt/ns-wincrypt-cert_context) , представляющую сертификат для вывода.

</dd> <dt>

*хвндселцертдлг* \[ окне\]
</dt> <dd>

Описатель диалогового окна, из которого был выбран сертификат для просмотра.

</dd> <dt>

*пвкаллбаккдата* \[ окне\]
</dt> <dd>

Дополнительные данные, используемые функцией обратного вызова.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция возвращает **значение true** , чтобы указать, что она обрабатывает отображение сертификата и что диалоговое окно не должно отображать сертификат. Если эта функция возвращает **значение false**, то в диалоговом окне отображается сертификат.
