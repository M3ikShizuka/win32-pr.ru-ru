# <a name="qualifiers-object"></a>Квалификатор, объект

\[Объект **квалификаторов** доступен для использования в операционных системах, указанных в разделе требования. Вместо этого используйте [**класс X509Extension**](/dotnet/api/system.security.cryptography.x509certificates.x509extension?view=netcore-3.1) в пространстве имен [**System. Security. Cryptography. X509Certificates**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1) , вызвав конструктор, принимающий OID в качестве параметра, а затем использовать OID для политик сертификатов для обработки квалификаторов, которые являются частью сведений о политике в расширении политик сертификатов.\]

Объект **квалификаторов** представляет коллекцию квалификаторов.

## <a name="when-to-use"></a>Назначение

Объект **квалификаторов** используется для выполнения следующих задач:

-   Получение определенного расширенного свойства из коллекции.
-   Получение числа расширенных свойств в коллекции.
-   Выполните итерацию по коллекции.

## <a name="remarks"></a>Remarks

Не удается создать объект **квалификаторов** .

Свойство CAPICOM объекта [**полициинформатион. квалификаторы**](policyinformation-qualifiers.md) возвращает объект **квалификаторов** .
