# <a name="signerchain-property"></a>Свойство SignIn. Chain

\[Свойство **цепочки** доступно для использования в операционных системах, указанных в разделе требования. Вместо этого используйте [**класс кмссигнер**](/dotnet/api/system.security.cryptography.pkcs.cmssigner?view=dotnet-plat-ext-3.1&preserve-view=true) в пространстве имен [**System. Security. Cryptography. PKCS**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

Свойство **chain** извлекает цепочку подписывающих.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Signer.Chain As Chain
```



## <a name="property-value"></a>Значение свойства

Цепочка подписывающих.
