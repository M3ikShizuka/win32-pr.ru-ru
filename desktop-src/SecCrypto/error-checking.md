# <a name="error-checking-cryptography"></a>Проверка ошибок (криптография)

Для C++ возвращаемое значение всегда равно **HRESULT** и указывает, успешно ли завершился вызов метода. В случае сбоя вызова возвращаемое значение указывает на причину сбоя.
