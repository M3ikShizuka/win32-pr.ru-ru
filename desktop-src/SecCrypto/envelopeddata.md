# <a name="envelopeddata-object"></a>Объект Енвелопеддата

\[CAPICOM — это 32-разрядный компонент, доступный для использования в следующих операционных системах: Windows Server 2008, Windows Vista и Windows XP. Вместо этого используйте [**класс EnvelopedCms**](/dotnet/api/system.security.cryptography.pkcs.envelopedcms?view=dotnet-plat-ext-3.1&preserve-view=true) в пространстве имен [**System. Security. Cryptography. PKCS**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

Объект **енвелопеддата** предоставляет свойства и методы для учетом неизбежного увеличения данных о конфиденциальности с помощью шифрования. Для учетом неизбежного увеличения данных создается криптографический ключ сеанса. Затем этот [*ключ сеанса*](../secgloss/s-gly.md) шифруется для каждого предполагаемого получателя с помощью [*открытого ключа*](../secgloss/p-gly.md) этого получателя из сертификата получателя. Зашифрованные данные и набор зашифрованных ключей сеанса можно отправить всем получателям. Созданное сообщение находится в \# формате PKCS 7.

### <a name="methods"></a>Методы

Объект **енвелопеддата** содержит эти методы.



| Метод                                   | Описание                                                                                                 |
|:-----------------------------------------|:------------------------------------------------------------------------------------------------------------|
| [**расшифровка;**](envelopeddata-decrypt.md) | Расшифровывает содержимое, упакованное в оболочку.<br/>                                                                      |
| [**Шифрование**](envelopeddata-encrypt.md) | Шифрует содержимое, шифрует сеансовый ключ для каждого получателя и возвращает зашифрованный большой двоичный объект.<br/> |



 

### <a name="properties"></a>Свойства

Объект **енвелопеддата** имеет следующие свойства.



| Свойство                                                  | Тип доступа           | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|:----------------------------------------------------------|:----------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Алгоритм**](envelopeddata-algorithm.md)<br/>   | Чтение/запись<br/> | Алгоритм шифрования и [*Длина ключа*](../secgloss/k-gly.md).<br/>                                                                                                                                                                                                                                                                                                                              |
