# <a name="capicom_secret_type-enumeration"></a>\_Перечисление типов секретов CAPICOM \_

Перечисление **\_ \_ типа «CAPICOM Secret** » указывает тип секрета, используемый для получения ключа, используемого для шифрования и расшифровки данных.

## <a name="members"></a>Элементы



| Член                        | Описание                                                     | Значение |
|-------------------------------|-----------------------------------------------------------------|-------|
| **\_пароль для секрета CAPICOM \_** | Ключ шифрования должен быть производным от пароля.<br/> | 0     |
