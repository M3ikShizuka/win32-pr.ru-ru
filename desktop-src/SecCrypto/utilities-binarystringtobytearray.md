# <a name="utilitiesbinarystringtobytearray-method"></a>Метод Utilities. Бинаристрингтобитеаррай

\[Метод **бинаристрингтобитеаррай** доступен для использования в операционных системах, указанных в разделе требования.\]

Метод **бинаристрингтобитеаррай** преобразует упакованную двоичную строку в массив байтов.

## <a name="syntax"></a>Синтаксис


```VB
Utilities.BinaryStringToByteArray( _
  ByVal BinaryString _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Бинаристринг* \[ окне\]
</dt> <dd>

Двоичная упакованная строка для преобразования в массив байтов.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Массив, содержащий результат преобразования.
