# <a name="capicom_store_location-enumeration"></a>\_ \_ Перечисление расположения CAPICOM Store

Тип **перечисления \_ CAPICOM \_ Location** указывает расположение [*хранилища сертификатов*](../secgloss/c-gly.md).

## <a name="members"></a>Элементы



| Член                                      | Описание                                                                                                                                                                                                                                                                      | Значение |
|---------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------|
| **\_хранилище CAPICOM Memory \_**                  | Хранилище является хранилищем памяти. Любые изменения в содержимом хранилища не сохраняются.<br/>                                                                                                                                                                              | 0     |
| **хранилище CAPICOM на \_ локальном \_ компьютере \_**          | Хранилище является локальным хранилищем компьютера. Хранилища локальных компьютеров могут быть хранилищами для чтения и записи, только если у пользователя есть разрешения на чтение и запись. Если у пользователя есть разрешения на чтение и запись и если хранилище открыто для чтения и записи, то изменения в содержимом хранилища сохраняются.<br/> | 1     |
| **\_хранилище CAPICOM текущего \_ пользователя \_**           | Хранилище является хранилищем текущего пользователя. Хранилище текущего пользователя может быть хранилищем для чтения и записи. Если это так, изменения в содержимом хранилища сохраняются.<br/>                                                                                                                      | 2     |
| **CAPICOM \_ Active \_ Directory \_ пользовательское \_ хранилище** | Магазин является хранилищем Active Directory. Хранилища Active Directory могут быть открыты только в режиме только для чтения. Сертификаты нельзя добавлять в хранилища Active Directory или удалять из них.<br/>                                                                                        | 3     |
| **\_хранилище CAPICOM смарт- \_ карты \_ пользователя \_**       | Хранилища поддерживают хранилища сертификатов на основе смарт-карт. Магазин — это группа существующих смарт-карт. Представлено в CAPICOM 2,0.<br/>                                                                                                                                         | 4     |



## <a name="remarks"></a>Комментарии

Тип **перечисления \_ CAPICOM \_ Location** используется следующими методами.

-   [**PrivateKey. Open**](privatekey-open.md)
-   [**Store. Open**](store-open.md)
