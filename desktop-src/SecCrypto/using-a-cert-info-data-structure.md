# <a name="using-a-cert_info-data-structure"></a>Использование \_ структуры данных о сертификатах

[**Сертификат \_**](/windows/desktop/api/Wincrypt/ns-wincrypt-cert_context) Сведения о контексте и [**сертификате \_**](/windows/desktop/api/Wincrypt/ns-wincrypt-cert_info) — это основные структуры данных, содержащие информацию в сертификате. При использовании сертификата может потребоваться декодировать его. Дополнительные сведения см. [в разделе Декодирование \_ структуры сведений о сертификате](decoding-a-cert-info-structure.md). Сведения можно добавить в структуру декодированных **\_ сведений о сертификате** , после чего структура может быть перекодирована повторно. Дополнительные сведения см. [в разделе кодирование \_ структуры сведений о сертификате](encoding-a-cert-info-structure.md).
