# <a name="capicom_signed_data_verify_flag-enumeration"></a>\_ \_ \_ \_ Перечисление флагов проверки данных CAPICOM со знаком

Перечисление **\_ \_ \_ \_ флагов «CAPICOM подписанные данные** » указывает, что проверяется при проверке [*цифровой подписи*](../secgloss/d-gly.md) .

## <a name="members"></a>Элементы



| Член                                           | Описание                                                                                                 | Значение |
|--------------------------------------------------|-------------------------------------------------------------------------------------------------------------|-------|
| **CAPICOM \_ проверить \_ \_ только подпись**             | Проверяется только подпись.<br/>                                                                   | 0     |
| **CAPICOM \_ Проверка \_ подписи \_ и \_ сертификата** | Проверяется подпись и допустимость сертификата, используемого для создания подписи.<br/> | 1     |
