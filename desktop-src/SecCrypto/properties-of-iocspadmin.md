# <a name="properties-of-iocspadmin"></a>Свойства Иокспадмин

Следующие свойства определяются интерфейсом [**иокспадмин**](/windows/desktop/api/certadm/nn-certadm-iocspadmin) .



| Свойство                                                                                 | Описание                                                                                                                                                                                                                      |
|------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**окспкаконфигуратионколлектион**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspcaconfigurationcollection) | Возвращает экземпляр объекта [**окспкаконфигуратионколлектион**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspcaconfigurationcollection) . Этот объект представляет набор сертификатов, для которых служба респондента может выполнять запросы состояния. |
| [**окспсервицепропертиес**](/windows/desktop/api/Certadm/nf-certadm-iocspadmin-get_ocspserviceproperties)                 | Возвращает экземпляр объекта [**оксппропертиколлектион**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) . Этот объект представляет атрибуты службы респондента.                                                                            |
