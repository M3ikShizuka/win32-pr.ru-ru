# <a name="extending-cryptoapi-functionality"></a>Расширение функциональности CryptoAPI

Будущее [*криптографических*](../secgloss/c-gly.md) и защищенных коммуникаций легко прогнозировать. Могут возникать новые типы сертификатов, различные расширения сертификатов могут обнаружить распространенное использование, а новые типы сообщений могут быть введены. По этой причине расширяемость является частью структуры ключевых функций [*CryptoAPI*](../secgloss/c-gly.md) .
