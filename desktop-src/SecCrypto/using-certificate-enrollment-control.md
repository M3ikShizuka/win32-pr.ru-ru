# <a name="using-certificate-enrollment-control"></a>Использование управления регистрацией сертификатов

Управление регистрацией сертификатов — это один объект с множеством свойств и несколько методов, которые можно использовать для определения того, как Управление регистрацией сертификата обрабатывает сведения о пользователе, тип запрашиваемого сертификата, способ создания ключей, место хранения сертификата и связанные процессы. Существует два основных способа использования управления регистрацией сертификатов: для создания [*запроса на сертификат*](../secgloss/c-gly.md) (PKCS \# 10) и получения возвращенного сертификата и сохранения его в [*хранилище сертификатов*](../secgloss/c-gly.md). Дополнительные сведения и синтаксис для создания экземпляра объекта управления регистрацией сертификатов, задания его свойств и использования его методов см. в следующих разделах.
