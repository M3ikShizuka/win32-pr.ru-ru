# <a name="signer_cert_store_info-structure"></a>\_ \_ \_ Структура сведений о хранилище сертификатов подписывающих

**\_ \_ \_ Информационная структура хранилища сертификатов подписывания** определяет хранилище сертификатов, используемое для подписания документа.

> [!Note]  
> Эта структура не определена ни в одном файле заголовка. Чтобы использовать эту структуру, необходимо определить ее самостоятельно, как показано в этом разделе.

 

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _SIGNER_CERT_STORE_INFO {
  DWORD          cbSize;
  PCCERT_CONTEXT pSigningCert;
  DWORD          dwCertPolicy;
  HCERTSTORE     hCertStore;
} SIGNER_CERT_STORE_INFO, *PSIGNER_CERT_STORE_INFO;
```
