# <a name="signeddata-object"></a>Объект Сигнеддата

\[Объект **сигнеддата** доступен для использования в операционных системах, указанных в разделе требования. Вместо этого используйте [**класс SignedCms**](/dotnet/api/system.security.cryptography.pkcs.signedcms?view=dotnet-plat-ext-3.1&preserve-view=true) в пространстве имен [**System. Security. Cryptography. PKCS**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

Объект **сигнеддата** предоставляет свойства и методы для установки содержимого, подписанного [*цифровой подписью*](../secgloss/d-gly.md), для цифрового подписывания и цифровой подписи данных, а также для проверки цифровой сигнатуры подписанных данных. Подписанное сообщение находится в \# формате PKCS 7.

Подпись данных, если она проверена, подтверждает связь между подписавшим и данными и показывает, что данные не изменялись каким-либо образом после создания подписи.

## <a name="members"></a>Элементы

Объект **сигнеддата** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Объект **сигнеддата** содержит эти методы.



| Метод                              | Описание                                                         |
|:------------------------------------|:--------------------------------------------------------------------|
| [**Знак подписывания**](signeddata-cosign.md) | Сописывает уже подписанное сообщение.<br/>                       |
| [**Писать**](signeddata-sign.md)     | Создает цифровую подпись для подписанного содержимого.<br/> |
| [**Проверка**](signeddata-verify.md) | Определяет допустимость подписи или сигнатур.<br/>    |



 

### <a name="properties"></a>Свойства

Объект **сигнеддата** имеет следующие свойства.



| Свойство                                                   | Тип доступа           | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|:-----------------------------------------------------------|:----------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Сертификаты**](signeddata-certificates.md)<br/> | Только для чтения<br/>  | Извлекает коллекцию [**сертификатов**](certificates.md) подписанных данных.<br/>                                                                                                                                                                                                                                                                                                                                                          |
| [**Содержани**](signeddata-content.md)<br/>           | Чтение/запись<br/> | Данные для подписи. Это свойство должно быть инициализировано до вызова метода [**Sign**](signeddata-sign.md) .<br/> Когда значение этого свойства сбрасывается прямо или косвенно, все [*состояние*](../secgloss/s-gly.md) объекта сбрасывается, и все сигнатуры, связанные с объектом до изменения свойства, теряются.<br/> Это свойство по умолчанию.<br/> |
| [**Подписавшими**](signeddata-signers.md)<br/>           | Только для чтения<br/>  | Извлекает коллекцию [**подписывающих**](signers.md) , представляющую создатели подписей данных.<br/>                                                                                                                                                                                                                                                                                                                                    |
