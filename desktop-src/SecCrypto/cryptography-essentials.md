# <a name="cryptography-essentials"></a>Основы криптографии

Microsoft CryptoAPI, прикладной программный интерфейс (API) и CAPICOM предоставляют службы, позволяющие разработчикам добавлять безопасность на основе [*шифрования*](../secgloss/c-gly.md) для приложений. Интерфейс CryptoAPI включает функции для кодирования и декодирования из [*абстрактной нотации синтаксиса*](../secgloss/a-gly.md) (ASN. 1), шифрования и расшифровки данных, проверки подлинности с помощью [*цифровых сертификатов*](../secgloss/d-gly.md)и управления цифровыми сертификатами с помощью хранилищ сертификатов. CryptoAPI и CAPICOM поддерживают криптографию PKI и симметричного ключа.
