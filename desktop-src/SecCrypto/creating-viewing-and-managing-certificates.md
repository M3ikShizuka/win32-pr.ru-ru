# <a name="creating-viewing-and-managing-certificates"></a>Создание, просмотр и управление сертификатами

[*Сертификаты*](../secgloss/c-gly.md) являются структурами только для чтения. Содержимое сертификата можно просмотреть, но нельзя изменить. Сведения в самом сертификате включают даты допустимости сертификата, издателя, субъекта и открытого ключа. Однако сертификаты имеют связанные свойства, такие как отображаемое имя, описание и разрешенные варианты использования, которые можно просматривать и изменять.

В этом разделе демонстрируется создание тестовых сертификатов, Просмотр сертификатов и управление сертификатами и другими объектами безопасности. Сертификаты и [*сертификаты издателей программного обеспечения*](../secgloss/s-gly.md) (спкс), которые могут быть созданы с помощью Makecert, строго используются исключительно для тестирования. Тестовый [*корневой сертификат*](../secgloss/r-gly.md) и корневой [*закрытый ключ*](../secgloss/p-gly.md) предоставляются только для тестирования. Независимые поставщики программного обеспечения должны получить сертификат от ГТЕ, VeriSign, Inc. или другого [*центра сертификации*](../secgloss/c-gly.md) (CA), чтобы подписать код, который будет распространяться на общедоступный.
