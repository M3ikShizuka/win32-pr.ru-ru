# <a name="cryptography-cryptoapi-and-capicom"></a>Криптография, CryptoAPI и CAPICOM

CryptoAPI и CAPICOM предоставляют службы, позволяющие разработчикам приложений добавлять шифрование и расшифровку данных, проверку подлинности с помощью цифровых [*сертификатов*](../secgloss/c-gly.md), а также кодировать и декодировать ее из [*абстрактного синтаксиса*](../secgloss/a-gly.md) (ASN. 1) в приложения на основе Windows. Разработчики приложений могут использовать функции CryptoAPI, не зная сведений о базовой реализации, во многом так же, как они могут использовать графическую библиотеку, не зная никаких сведений о конфигурации графического оборудования. Интерфейс CryptoAPI работает с несколькими [поставщиками служб шифрования](microsoft-cryptographic-service-providers.md) , которые выполняют реальные функции шифрования.
