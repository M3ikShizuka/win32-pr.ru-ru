# <a name="required-libraries"></a>Требуемые библиотеки

Для связывания функций CryptoAPI требуется доступность Crypt32. lib. Программам, использующим некоторые функции, также может потребоваться advapi32. lib. При использовании компилятора Visual C++ эти файлы должны быть перечислены в модулях объектов и библиотек.
