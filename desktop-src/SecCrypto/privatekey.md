# <a name="privatekey-object"></a>Объект PrivateKey

\[Объект **PrivateKey** доступен для использования в операционных системах, указанных в разделе требования. Вместо этого используйте [**свойство X509Certificate2. PrivateKey**](/dotnet/api/system.security.cryptography.x509certificates.x509certificate2.privatekey?view=netcore-3.1) в пространстве имен [**System. Security. Cryptography. X509Certificates**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1) .\]

Объект **PrivateKey** представляет [*закрытый ключ*](../secgloss/p-gly.md) , связанный с сертификатом.

## <a name="when-to-use"></a>Назначение

Объект **PrivateKey** используется для выполнения следующих задач:

-   Получение сведений о закрытом ключе.
-   Откройте контейнер закрытого ключа.
-   Удалите закрытый ключ.

## <a name="members"></a>Элементы

Объект **PrivateKey** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **PrivateKey** содержит эти методы.



| Метод                                                  | Описание                                                                                                                                                          |
|:--------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Удален**](privatekey-delete.md)                     | Удаляет контейнер закрытого ключа, на который ссылается объект **PrivateKey** .<br/>                                                                                |
| [**Доступно**](privatekey-isaccessible.md)         | Получает логическое значение, указывающее, доступен ли закрытый ключ пользователю. Если значение — true, пользователь может получить доступ к закрытому ключу.<br/>                 |
| [**IsExportable**](privatekey-isexportable.md)         | Возвращает логическое значение, указывающее, можно ли экспортировать закрытый ключ. Если значение — true, закрытый ключ можно экспортировать.<br/>                               |
| [**ишардваредевице**](privatekey-ishardwaredevice.md) | Получает логическое значение, указывающее, хранится ли закрытый ключ на аппаратном устройстве. Если значение — true, закрытый ключ хранится на аппаратном устройстве.<br/> |
| [**исмачинекэйсет**](privatekey-ismachinekeyset.md)   | Возвращает логическое значение, указывающее, является ли закрытый ключ ключом компьютера. Если значение — true, закрытый ключ является ключом компьютера.<br/>                             |
| [**Защита**](privatekey-isprotected.md)           | Получает логическое значение, указывающее, защищен ли закрытый ключ. Если значение — true, закрытый ключ защищен.<br/>                                     |
| [**Съемная**](privatekey-isremovable.md)           | Получает логическое значение, указывающее, находится ли закрытый ключ на съемном устройстве. Если значение — true, закрытый ключ находится на съемном устройстве.<br/>             |
| [**Открыт**](privatekey-open.md)                         | Обращается к существующему контейнеру ключей.<br/>                                                                                                                       |



 

### <a name="properties"></a>Свойства

Объект **PrivateKey** имеет следующие свойства.



| Свойство                                                                 | Тип доступа          | Описание                                                                                               |
|:-------------------------------------------------------------------------|:---------------------|:----------------------------------------------------------------------------------------------------------|
| [**ContainerName**](privatekey-containername.md)<br/>             | Только для чтения<br/> | Извлекает строку, содержащую имя контейнера закрытого ключа. Это свойство по умолчанию.<br/> |
| [**KeySpec**](privatekey-keyspec.md)<br/>                         | Только для чтения<br/> | Извлекает спецификацию ключа.<br/>                                                               |
| [**ProviderName**](privatekey-providername.md)<br/>               | Только для чтения<br/> | Извлекает строку, содержащую имя CSP.<br/>                                          |
| [**ProviderType**](privatekey-providertype.md)<br/>               | Только для чтения<br/> | Возвращает значение перечисления, указывающее тип поставщика.<br/>                            |
| [**уникуеконтаинернаме**](privatekey-uniquecontainername.md)<br/> | Только для чтения<br/> | Извлекает строку, содержащую уникальное имя контейнера закрытого ключа.<br/>                        |
