# <a name="certificates-and-messages"></a>Сертификаты и сообщения

Базовые криптографические функции используют [*Сертификаты*](../secgloss/c-gly.md), а [*функции сообщений низкого уровня*](../secgloss/l-gly.md) и [*упрощенные функции сообщений*](../secgloss/s-gly.md) также используют их. Для отправки или получения сообщений, включающих сертификаты, используйте низкоуровневые функции сообщений.

Примеры использования сертификатов с функциями сообщений низкого уровня см. [в примере программы на языке c: кодирование и декодирование сообщения скрепляющая подпись](example-c-program-encoding-and-decoding-a-countersigned-message.md), [Пример программы на языке c: кодирование и декодирование хэшированного сообщения](example-c-program-encoding-and-decoding-a-hashed-message.md).

Примеры использования сертификатов с упрощенными функциями сообщений см. в разделе [Пример программы на языке C. Использование криптенкриптмессаже и криптдекриптмессаже](example-c-program-using-cryptencryptmessage-and-cryptdecryptmessage.md).

Дополнительные сведения о функциях сообщений низкого уровня см. в разделе [сообщения низкого уровня](low-level-messages.md).

Дополнительные сведения о упрощенных функциях сообщений см. в разделе [упрощенные сообщения](simplified-messages.md).
