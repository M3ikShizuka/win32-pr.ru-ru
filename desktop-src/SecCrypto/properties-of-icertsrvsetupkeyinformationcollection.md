# <a name="properties-of-icertsrvsetupkeyinformationcollection"></a>Свойства Ицертсрвсетупкэйинформатионколлектион

Следующие свойства определяются интерфейсом [**ицертсрвсетупкэйинформатионколлектион**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformationcollection) .



| Свойство                                                            | Описание                                                                                                              |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get__newenum) | Возвращает перечислитель для коллекции.                                                                                   |
| [**Расчета**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get_count)        | Возвращает число объектов [**ицертсрвсетупкэйинформатион**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformation) в коллекции.         |
| [**Элемент**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetupkeyinformationcollection-get_item)          | Возвращает объект [**ицертсрвсетупкэйинформатион**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetupkeyinformation) , идентифицируемый по индексу в коллекции. |
