# <a name="recipientscount-property"></a>Свойство Recipients. Count

\[Свойство **Count** доступно для использования в операционных системах, указанных в разделе требования. Вместо этого используйте [**класс кмсреЦипиентколлектион**](/dotnet/api/system.security.cryptography.pkcs.cmsrecipientcollection?view=dotnet-plat-ext-3.1&preserve-view=true) в пространстве имен [**System. Security. Cryptography. PKCS**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

Свойство **Count** извлекает количество объектов в коллекции [**получателей**](recipients.md) .

## <a name="syntax"></a>Синтаксис


```VB
Recipients.Count As Long
```



## <a name="property-value"></a>Значение свойства

Количество объектов в коллекции .
