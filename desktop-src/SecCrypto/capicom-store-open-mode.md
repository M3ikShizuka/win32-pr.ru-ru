# <a name="capicom_store_open_mode-enumeration"></a>\_ \_ Перечисление в открытом режиме CAPICOM Store \_

\_ \_ Тип перечисления CAPICOM Store Open \_ mode используется с методом [**Store. Open**](store-open.md) для указания способа открытия хранилища сертификатов.

## <a name="members"></a>Элементы



| Член                                      | Описание                                                                                                                                                              | Значение |
|---------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------|
| **CAPICOM \_ Store \_ открыт \_ только для чтения \_**        | Откройте хранилище в режиме только для чтения.<br/>                                                                                                                             | 0     |
| **CAPICOM \_ Store \_ открыть \_ чтение и \_ запись**       | Откройте хранилище в режиме чтения и записи.<br/>                                                                                                                            | 1     |
| **\_ \_ \_ Максимальное \_ допустимое число открытых в CAPICOM Store**  | Откройте хранилище в режиме чтения и записи, если у пользователя есть разрешения на чтение и запись. Если у пользователя нет разрешений на чтение и запись, откройте хранилище в режиме только для чтения.<br/> | 2     |
| **в \_ CAPICOM \_ Store \_ открыто \_ только существующее**    | Открывать только существующие магазины; не создавать новое хранилище. Представлено CAPICOM 2,0.<br/>                                                                              | 128   |
| **\_открыто хранилище CAPICOM, \_ \_ включая \_ архивированные** | Включать архивные сертификаты при использовании хранилища. Представлено CAPICOM 2,0.<br/>                                                                                | 256   |



## <a name="remarks"></a>Комментарии

При использовании \_ \_ \_ перечислений в режиме открытого хранилища CAPICOM можно использовать только один из следующих параметров.

-   CAPICOM \_ Store \_ открыт \_ только для чтения \_
-   CAPICOM \_ Store \_ открыть \_ чтение и \_ запись
-   \_ \_ \_ Максимальное \_ допустимое число открытых в CAPICOM Store
