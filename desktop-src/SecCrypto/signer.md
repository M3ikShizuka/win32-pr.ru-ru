# <a name="signer-object"></a>Объект SignIn

\[Объект- **подписавший** доступен для использования в операционных системах, указанных в разделе требования. Вместо этого используйте [**класс кмссигнер**](/dotnet/api/system.security.cryptography.pkcs.cmssigner?view=dotnet-plat-ext-3.1&preserve-view=true) в пространстве имен [**System. Security. Cryptography. PKCS**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

Объект- **подписывающий** устанавливает подписывающий объекта [**сигнеддата**](signeddata.md) или [**сигнедкоде**](signedcode.md) . [**Сертификат**](certificate.md) объекта " **подписавший** " должен иметь доступный закрытый ключ, который можно использовать для подписывания данных.

## <a name="members"></a>Элементы

Объект **SignIn** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

У объекта **подписавши** есть эти методы.



| Метод                      | Описание                                                       |
|:----------------------------|:------------------------------------------------------------------|
| [**Загрузить**](signer-load.md) | Загружает сертификат подписи из указанного PFX-файла.<br/> |



 

### <a name="properties"></a>Свойства

У объекта **подписавши** есть эти свойства.



| Свойство.                                                                     | Тип доступа           | Описание                                                                                                                                                                                                                                                                                                                                 |
|:-----------------------------------------------------------------------------|:----------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**аусентикатедаттрибутес**](signer-authenticatedattributes.md)<br/> | Только для чтения<br/>  | Коллекция атрибутов, прошедших проверку подлинности.<br/>                                                                                                                                                                                                                                                                                      |
| [**Сертификат**](signer-certificate.md)<br/>                         | Чтение/запись<br/> | Объект [**сертификата**](certificate.md) , представляющий сертификат подписи данных.<br/> Когда значение этого свойства сбрасывается прямо или косвенно, полное [*состояние*](../secgloss/s-gly.md) объекта сбрасывается.<br/> Это свойство по умолчанию.<br/> |
| [**Цепочк**](signer-chain.md)<br/>                                     | Только для чтения<br/>  | Цепочка подписи.<br/>                                                                                                                                                                                                                                                                                                         |
