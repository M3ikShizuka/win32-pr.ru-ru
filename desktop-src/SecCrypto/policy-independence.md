# <a name="policy-independence"></a>Независимость от политик

Сертификаты предоставляются в соответствии с политиками, которые определяют критерии, которым должны удовлетворять запросы для получения сертификата. Например, одной из политик может быть предоставление коммерческих сертификатов только в том случае, если кандидаты предоставляют свои идентификационные сведения лично. Другая политика может предоставлять [*учетные данные*](../secgloss/c-gly.md) на основе запросов электронной почты. Агентство, которое выдает кредитные карты, может обращаться к базе данных и отправлять запросы на телефонные звонки перед выдачей карты.

Политики реализуются в модулях политик, написанных на C++, C или Visual Basic. Функции служб сертификации изолированы от любых изменений в политике, которые может реализовать агентство. Изменения в политике можно полностью реализовать в коде модуля политики сервера. В [*центре сертификации*](../secgloss/c-gly.md) предприятия (ЦС) должен использоваться только модуль политики предприятия, предоставленный корпорацией Майкрософт.
