# <a name="error-handling-cryptography"></a>Обработка ошибок (криптография)

Во всех примерах C, сопровождающих документацию по функциям, пример функции [**михандлиррор**](myhandleerror.md) используется для вывода сообщения об ошибке и выхода из процедуры. В реальных ситуациях могут потребоваться более сложные процедуры обработки ошибок. Дополнительные сведения см. в разделе [примеры ограничений программы](example-program-limitations.md).
