# <a name="signer_spc_chain_info-structure"></a>\_ \_ Структура сведений о цепочке SPC в подписавшем \_

в структуре **\_ \_ \_ сведений о цепочке SPC в подписавшем** указан [*сертификат программного обеспечения Publisher*](../secgloss/s-gly.md) (SPC) и цепочку сертификатов, используемую для подписания документа.

> [!Note]  
> Эта структура не определена ни в одном файле заголовка. Чтобы использовать эту структуру, необходимо определить ее самостоятельно, как показано в этом разделе.

 

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _SIGNER_SPC_CHAIN_INFO {
  DWORD      cbSize;
  LPCWSTR    pwszSpcFile;
  DWORD      dwCertPolicy;
  HCERTSTORE hCertStore;
} SIGNER_SPC_CHAIN_INFO, *PSIGNER_SPC_CHAIN_INFO;
```
