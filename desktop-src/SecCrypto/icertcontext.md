# <a name="icertcontext-interface"></a>Интерфейс Ицертконтекст

\[CAPICOM — это 32-разрядный компонент, доступный для использования в следующих операционных системах: Windows Server 2008, Windows Vista и Windows XP.\]

Интерфейс **ицертконтекст** предоставляет доступ к контексту объекта [**сертификата**](certificate.md) CAPICOM X. 509v3. Этот контекст позволяет использовать CAPICOM Certificate в других производных классах CryptoAPI.

## <a name="when-to-use"></a>Назначение

Этот интерфейс следует использовать, если необходимо использовать объект CAPICOM [**Certificate**](certificate.md) в другом производном от CryptoAPI.

## <a name="members"></a>Элементы

Интерфейс **ицертконтекст** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **ицертконтекст** содержит следующие методы.



| Метод                                          | Описание                                                                                                          |
|:------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------|
| [**фриконтекст**](icertcontext-freecontext.md) | Освобождает контекст ПКЦЕРТ, \_ полученный через свойство [**цертконтекст**](icertcontext-certcontext.md) .<br/> |



 

### <a name="properties"></a>Свойства

Интерфейс **ицертконтекст** имеет следующие свойства.



| Свойство                                                   | Тип доступа           | Описание                                                        |
|:-----------------------------------------------------------|:----------------------|:-------------------------------------------------------------------|
| [**цертконтекст**](icertcontext-certcontext.md)<br/> | Чтение/запись<br/> | Задает или получает контекст ПКЦЕРТ \_ сертификата.<br/> |
