# <a name="utilitieshextobinary-method"></a>Метод Utilities. Хекстобинари

\[Метод **хекстобинари** доступен для использования в операционных системах, указанных в разделе требования.\]

Метод **хекстобинари** преобразует шестнадцатеричную строку в двоичную строку.

## <a name="syntax"></a>Синтаксис


```VB
Utilities.HexToBinary( _
  ByVal HexString _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Хексстринг* \[ окне\]
</dt> <dd>

Шестнадцатеричная строка, которую необходимо преобразовать в упакованную в двоичную строку.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Двоичная упакованная строка, эквивалентная заданной шестнадцатеричной строке.
