# <a name="prov_rsa_sig"></a>PROV \_ RSA \_ SIG

\_ \_ Тип поставщика SIG Prov RSA — это подмножество Prov \_ RSA \_ Full. Он поддерживает только те функции и алгоритмы, которые необходимы для [*хэширования*](../secgloss/h-gly.md) и [*цифровых подписей*](../secgloss/d-gly.md).

## <a name="algorithms-supported"></a>Поддерживаемые алгоритмы

Описание каждого из этих алгоритмов см. в глоссарии.



| Назначение      | Поддерживаемые алгоритмы                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| Обмен ключами | Нет                                                                                                                              |
| Подпись    | [*RSA*](../secgloss/r-gly.md)                                                                       |
| Encryption   | Нет                                                                                                                              |
| Хэширование      | [*MD5*](../secgloss/m-gly.md)[*SHA*](../secgloss/s-gly.md)<br/> |
