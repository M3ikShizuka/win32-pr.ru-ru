# <a name="rkeyopenkeyservice-function"></a>Функция Ркэйопенкэйсервице

Функция **ркэйопенкэйсервице** не поддерживается.

**Windows Server 2003:** Функция **ркэйопенкэйсервице** устанавливает соединение с удаленным компьютером и открывает маркер службы ключей. Маркер службы ключей может впоследствии использоваться функцией [**ркэйпфксинсталл**](rkeypfxinstall.md) . Обратите внимание, что это поведение изменилось в Windows Server 2003 с пакетом обновления 1 (SP1).

## <a name="syntax"></a>Синтаксис


```C++
ULONG RKeyOpenKeyService(
  _In_    LPSTR          pszMachineName,
  _In_    KEYSVC_TYPE    OwnerType,
  _In_    LPWSTR         pwszOwnerName,
  _In_    void           *pAuthentication,
  _Inout_ void           *pReserved,
  _Out_   KEYSVCC_HANDLE *phKeySvcCli
);
```

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена успешно, возвращается значение S \_ ОК.

Если функция завершается ошибкой, возвращается значение типа **ulong** , указывающее на ошибку.
