# <a name="properties-of-iocspcaconfigurationcollection"></a>Свойства Иокспкаконфигуратионколлектион

Следующие свойства определяются интерфейсом [**иокспкаконфигуратионколлектион**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) .



| Свойство                                                               | Описание                                                                                                                                                                              |
|------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get__newenum)    | Возвращает перечислитель для набора конфигурации.                                                                                                                                            |
| [**Расчета**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_count)           | Возвращает количество конфигураций [*центра сертификации*](../secgloss/c-gly.md) (ЦС) в наборе конфигурации. |
| [**Элемент**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_item)             | Возвращает конфигурацию ЦС, определенную по индексу в наборе конфигурации.                                                                                                                    |
| [**итембинаме**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_itembyname) | Возвращает конфигурацию ЦС, определенную по имени в наборе конфигурации.                                                                                                                     |
