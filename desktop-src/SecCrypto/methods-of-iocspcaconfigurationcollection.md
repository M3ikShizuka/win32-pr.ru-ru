# <a name="methods-of-iocspcaconfigurationcollection"></a>Методы Иокспкаконфигуратионколлектион

Следующие методы определяются интерфейсом [**иокспкаконфигуратионколлектион**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) . Методы доступа к свойствам не показаны здесь. Сведения о свойствах **иокспкаконфигуратионколлектион** см. в разделе [Свойства иокспкаконфигуратионколлектион](properties-of-iocspcaconfigurationcollection.md) .



| Метод                                                                                       | Описание                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**креатекаконфигуратион**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-createcaconfiguration) | Создает новую конфигурацию [*центра сертификации*](../secgloss/c-gly.md) (ЦС) и добавляет ее в набор конфигурации. |
| [**делетекаконфигуратион**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-deletecaconfiguration) | Удаляет именованную конфигурацию ЦС из набора конфигурации.                                                                                                                                   |
