# <a name="signeddatasigners-property"></a>Сигнеддата. Signs, свойство

\[Свойство **подписывающих** доступно для использования в операционных системах, указанных в разделе требования. Вместо этого используйте [**класс SignedCms**](/dotnet/api/system.security.cryptography.pkcs.signedcms?view=dotnet-plat-ext-3.1&preserve-view=true) в пространстве имен [**System. Security. Cryptography. PKCS**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

Свойство **подписавший** получает создатели подписей для данных.

## <a name="syntax"></a>Синтаксис


```VB
SignedData.Signers As Signers
```



## <a name="property-value"></a>Значение свойства

Коллекция [**подписывающих**](signers.md) , представляющая создатели подписей для данных.
