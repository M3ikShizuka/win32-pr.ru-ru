# <a name="maxuidlen"></a>максуидлен

МАКСУИДЛЕН — это числовая константа, указывающая максимальное число символов, которые будут использоваться некоторыми поставщиками служб шифрования Майкрософт при получении идентификатора пользователя. МАКСУИДЛЕН определяется в Винкрипт. h.



| Константа/значение                                                                                                                                                                                                  | Описание                                                                                                                                                                                                                                                                                                                                                                                                   |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="MAXUIDLEN"></span><span id="maxuidlen"></span><dl> <dt>**Максуидлен**</dt> <dt>64 (0x40)</dt> </dl> | Если параметр *псзконтаинер* функции [**CryptAcquireContext**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptacquirecontexta) имеет **значение NULL**, некоторые поставщики служб шифрования Майкрософт используют имя входа пользователя, вошедшего в систему, в качестве имени контейнера ключей. МАКСУИДЛЕН указывает максимальное число символов, включая завершающий символ **null** , который эти поставщики будут использовать для имени пользователя.<br/> |
