# <a name="transport-independence"></a>Независимость от транспорта

Сертификаты могут запрашиваться и распространяться с помощью любого транспортного механизма. Службы сертификатов принимают [*запросы на сертификаты*](../secgloss/c-gly.md) от кандидата по протоколу HTTP, DCOM, RPC, Disk File или пользовательским транспортом. Службы сертификатов относят сертификаты кандидату по протоколу HTTP, DCOM, RPC, дискового файла или пользовательскому транспорту.

Транспорты поддерживаются промежуточными приложениями и выходными библиотеками DLL модуля, обычно написанными на C/C++. Промежуточные приложения и модули выхода изолируют функции служб сертификации от взаимодействия с любым конкретным транспортом.
