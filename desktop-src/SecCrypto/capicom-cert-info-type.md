# <a name="capicom_cert_info_type-enumeration"></a>\_ \_ Перечисление типа сведений CAPICOM CERT \_

Тип **перечисления \_ CAPICOM \_ CERT \_ info** определяет, какие сведения должны быть запрошены из сертификата.

| **CAPICOM \_ CERT \_ info \_ \_ имя электронной почты поставщика \_**   | Возвращает адрес электронной почты издателя сертификата.<br/>                       | 3     |
