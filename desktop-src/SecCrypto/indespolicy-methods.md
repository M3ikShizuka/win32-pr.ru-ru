# <a name="indespolicy-methods"></a>Методы Индесполици

Интерфейс [**индесполици**](/windows/desktop/api/certpol/nn-certpol-indespolicy) поддерживает следующие методы.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                        | Описание                                                                                       |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**Метод Женератечалленже**](/windows/desktop/api/certpol/nf-certpol-indespolicy-generatechallenge)<br/> | Выполняет выбор политики, следует ли выдавать пароль вызова клиенту SCEP.<br/> |
| [**Метод Initialize**](/windows/desktop/api/certpol/nf-certpol-indespolicy-initialize)<br/>               | Инициализирует модуль политики NDES.<br/>                                                    |
| [**Notify, метод**](/windows/desktop/api/certpol/nf-certpol-indespolicy-notify)<br/>                       | Уведомляет подключаемый модуль о состоянии транзакции запроса на сертификат SCEP.<br/>        |
| [**Uninitialize - метод**](/windows/desktop/api/certpol/nf-certpol-indespolicy-uninitialize)<br/>           | Отменяет инициализацию модуля политики NDES.<br/>                                                  |
| [**Метод Верифирекуест**](/windows/desktop/api/certpol/nf-certpol-indespolicy-verifyrequest)<br/>         | Проверяет запрос сертификата NDES на отправку в ЦС.<br/>                        |
