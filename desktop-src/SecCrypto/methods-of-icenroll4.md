# <a name="methods-of-icenroll4"></a>Методы ICEnroll4

Методы доступа к свойствам не показаны здесь. Сведения о свойствах **ICEnroll4** см. в разделе [Свойства ICEnroll4](properties-of-icenroll4.md).
