# <a name="pvkprivatekeyacquirecontextfrommemory-function"></a>Функция Пвкприватекэйаккуиреконтекстфроммемори

> [!IMPORTANT]
> Это нерекомендуемый API. Корпорация Майкрософт может удалить этот API в будущих выпусках.

 

Функция **пвкприватекэйаккуиреконтекстфроммемори** создает временный контейнер в [*поставщике служб шифрования*](../secgloss/c-gly.md) (CSP) и загружает [*закрытый ключ*](../secgloss/p-gly.md) из памяти в контейнер.

> [!Note]  
> Эта функция не имеет связанного файла заголовка или библиотеки импорта. Чтобы вызвать эту функцию, необходимо создать файл заголовка, определяемый пользователем, и использовать функции [**LoadLibrary**](/windows/win32/api/libloaderapi/nf-libloaderapi-loadlibrarya) и [**GetProcAddress**](/windows/win32/api/libloaderapi/nf-libloaderapi-getprocaddress) для динамической привязки к Mssign32.dll.

 

## <a name="syntax"></a>Синтаксис


```C++
BOOL WINAPI PvkPrivateKeyAcquireContextFromMemory(
  _In_        LPCWSTR    pwszProvName,
  _In_        DWORD      dwProvType,
  _In_        BYTE       *pbData,
  _In_        DWORD      cbData,
  _In_        HWND       hwndOwner,
  _In_        LPCWSTR    pwszKeyName,
  _Inout_opt_ DWORD      *pdwKeySpec,
  _Out_       HCRYPTPROV *phCryptProv,
  _Out_       LPTSTR     *ppwszTmpContainer
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пвсзпровнаме* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая содержит имя CSP, тип которого запрашивается в *двпровтипе*.

</dd> <dt>

*двпровтипе* \[ окне\]
</dt> <dd>

Значение **DWORD** для типа CSP. Дополнительные сведения о типах CSP см. в разделе [типы поставщиков служб шифрования](cryptographic-provider-types.md).

</dd> <dt>

*pbData* \[ окне\]
</dt> <dd>

Указатель на буфер для получения данных контекста. Вызывающий объект должен предоставить этот ресурс.

</dd> <dt>

*cbData* \[ окне\]
</dt> <dd>

Значение **типа DWORD** , указывающее размер буфера *pbData* в байтах. Вызывающий объект должен предоставить это значение.

</dd> <dt>

*хвндовнер* \[ окне\]
</dt> <dd>

Если требуется пароль для расшифровки контекстных данных, на которые указывает параметр *pbData* , этот параметр является обработчиком родительского элемента диалогового окна. в противном случае он имеет **значение NULL**.

</dd> <dt>

*пвсзкэйнаме* \[ окне\]
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая содержит имя извлекаемого ключа.

</dd> <dt>

*пдвкэйспек* \[ в, out, необязательно\]
</dt> <dd>

Указатель на значение **DWORD** , указывающее тип ключа. Возможные значения: **в \_ Кэйексчанже** или **в \_ сигнатуре**.

</dd> <dt>

*фкриптпров* \[ заполняет\]
</dt> <dd>

Указатель на маркер для CSP.

</dd> <dt>

*ппвсзтмпконтаинер* \[ заполняет\]
</dt> <dd>

Адрес указателя на строку, завершающуюся нулем, для временного имени контейнера. Функция **пвкприватекэйаккуиреконтекстфроммемори** предоставляет буфер для этой строки и инициализирует ее. При вызове **пвкприватекэйаккуиреконтекстфроммемори** адрес должен указывать на значение **null** .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

При успешном выполнении эта функция возвращает **значение true**. Функция **пвкприватекэйаккуиреконтекстфроммемори** возвращает **значение false** в случае сбоя.
