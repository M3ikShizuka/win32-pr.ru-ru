# <a name="signed-data-additions"></a>Добавление подписанных данных

В возможности подписанных данных были внесены следующие изменения.

-   [*Внутреннее содержимое*](../secgloss/i-gly.md) , не связанное с данными, инкапсулируется в строку октета.
-   Для разных версий сообщений выполняется подготавливается.
-   Подписавшие можно идентифицировать по издателю и серийному номеру, как в PKCS \# 7 1,5 или с помощью идентификатора ключа субъекта.
-   Разрешено несколько подписывающих.
-   Допускается использование подписей RSA, DSA и ECDSA.
-   Для алгоритмов с шифрованием на основе эллиптических кривых (ECC) и AES требуется Windows Vista или более поздняя версия.
