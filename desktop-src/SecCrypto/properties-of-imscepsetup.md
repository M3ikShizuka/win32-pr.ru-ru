# <a name="properties-of-imscepsetup"></a>Свойства Имсцепсетуп

Следующие свойства определяются интерфейсом [**имсцепсетуп**](/windows/desktop/api/Casetup/nn-casetup-imscepsetup) .



| Свойство                                                 | Описание                                                                                                                                              |
|----------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**мсцеперрорид**](/windows/desktop/api/Casetup/nf-casetup-imscepsetup-get_msceperrorid)         | Возвращает идентификатор для дополнительных сведений об ошибке, связанных с неудачной спецификацией NDES. Любой вызов метода в родительском объекте сбрасывает это свойство.          |
| [**мсцеперрорстринг**](/windows/desktop/api/Casetup/nf-casetup-imscepsetup-get_msceperrorstring) | Возвращает строковые данные для получения дополнительных сведений об ошибке, связанной с неудачной спецификацией NDES. Любой вызов метода в родительском объекте сбрасывает это свойство. |
