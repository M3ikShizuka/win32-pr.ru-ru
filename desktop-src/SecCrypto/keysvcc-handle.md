# <a name="keysvcc_handle"></a>КЭЙСВКК, \_ обработчик

Тип **данных \_ Handle кэйсвкк** определяет маркер службы ключей. **Кэйсвкк \_ Handle** используется в функциях [**ркэйопенкэйсервице**](rkeyopenkeyservice.md) и [**ркэйклосекэйсервице**](rkeyclosekeyservice.md) .


```C++
typedef void* KEYSVCC_HANDLE;
```
