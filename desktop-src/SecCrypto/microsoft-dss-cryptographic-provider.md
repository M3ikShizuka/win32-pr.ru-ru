# <a name="microsoft-dss-cryptographic-provider"></a>Поставщик служб шифрования Microsoft DSS

Поставщик служб шифрования Microsoft DSS поддерживает хеширование, подписывание данных и проверку подписей с [*помощью алгоритмов*](../secgloss/s-gly.md) SHA и [*стандарта цифровой подписи*](../secgloss/d-gly.md) (DSS). Его можно экспортировать за пределы США и Канады.



|                   | Значение               |
|-------------------|---------------------|
| **Тип поставщика** | PROV \_ DSS           |
| **Имя поставщика** | \_Prov MS DEF \_ DSS \_  |
