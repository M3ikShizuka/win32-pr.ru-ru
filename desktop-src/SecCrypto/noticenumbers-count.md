# <a name="noticenumberscount-property"></a>Нотиценумберс. Count, свойство

\[Свойство **Count** доступно для использования в операционных системах, указанных в разделе требования. Дополнительные сведения см. в разделе [**квалификатор**](qualifier.md).\]

Свойство **Count** извлекает количество объектов [**расширения**](extension.md) в коллекции.

## <a name="syntax"></a>Синтаксис


```VB
NoticeNumbers.Count As Long
```



## <a name="property-value"></a>Значение свойства

Число объектов [**расширения**](extension.md) в коллекции.
