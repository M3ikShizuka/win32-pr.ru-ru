# <a name="protocol-flags"></a>Флаги протокола

Ниже приведены стандартные константы для протоколов шифрования. Эти значения определены в Винкрипт. h.



| Константа/значение                                                                                                                                                                                                                            | Описание                                                           |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------|
| <span id="CRYPT_FLAG_IPSEC"></span><span id="crypt_flag_ipsec"></span><dl> <dt>**Шифрования \_ ФЛАГ \_ IPSec**</dt> <dt>0x0010</dt> </dl>       | Протокол IPsec.<br/>               |
| <span id="CRYPT_FLAG_PCT1"></span><span id="crypt_flag_pct1"></span><dl> <dt>**Шифрования \_ ФЛАГ \_ PCT1**</dt> <dt>0x0001</dt> </dl>          | Протокол передачи частных соединений (PCT) версии 1.<br/> |
| <span id="CRYPT_FLAG_SIGNING"></span><span id="crypt_flag_signing"></span><dl> <dt>**Шифрования \_ ФЛАГ \_ подписи**</dt> <dt>0x0020</dt> </dl> | Протокол подписывания.<br/>                                          |
| <span id="CRYPT_FLAG_SSL2"></span><span id="crypt_flag_ssl2"></span><dl> <dt>**Шифрования \_ ФЛАГ \_ SSL2**</dt> <dt>0x0002</dt> </dl>          | Протокол SSL версии 2.<br/>             |
| <span id="CRYPT_FLAG_SSL3"></span><span id="crypt_flag_ssl3"></span><dl> <dt>**Шифрования \_ ФЛАГ \_ SSL3**</dt> <dt>0x0004</dt> </dl>          | Протокол SSL версии 3.<br/>                                    |
| <span id="CRYPT_FLAG_TLS1"></span><span id="crypt_flag_tls1"></span><dl> <dt>**Шифрования \_ ФЛАГ \_ TLS1**</dt> <dt>0x0008</dt> </dl>          | Протокол TLS версии 1.<br/>         |
