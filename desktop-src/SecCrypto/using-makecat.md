# <a name="using-makecat"></a>Использование Макекат

Средство [макекат](makecat.md) создает неподписанный файл каталога, который содержит [*хэши*](../secgloss/h-gly.md) набора файлов вместе со связанными атрибутами каждого файла в наборе. Файл каталога позволяет пользователю подписывать один файл (каталог) вместо подписывания множества отдельных файлов.

После подписания и передачи неподписанного файла каталога получатель может сравнить хэши исходных файлов с хэш-кодами, содержащимися в файле каталога, и убедиться, что эти файлы свободны от несанкционированного изменения.

Перед использованием средства [макекат](makecat.md) пользователь должен подготовить файл определения каталога (. CDF) с помощью любого текстового редактора. CDF-файл содержит список файлов и атрибуты файлов для каталогизации (спецификации перечислены ниже). Средство Макекат сканирует CDF-файл, проверяет список атрибутов каждого файла, добавляет указанные атрибуты в сам каталог, хэширует каждый из перечисленных файлов и сохраняет хэши каждого файла в файле каталога. Каждый файл имеет свой хэш и атрибуты, хранящиеся отдельно в каталоге. Этот файл каталога можно затем подписать и передать. Затем получатель может сравнить хэш каждого файла в каталоге с хэшом исходных файлов, чтобы доказать, что исходное содержимое не поддается иззаконному изменению. Макекат не изменяет файл. CDF.

В следующих примерах используются команды [макекат](makecat.md) .

-   Создайте файл каталога из файла Good. CDF.

    **Макекат-v Good. CDF**

Файл Good. CDF создает файл каталога Good.cat, анализируя файлы UnsignedPE.exe, UnsignedDOS.exe, Unsigned.cab, без знака. class и SignedPE.exe. Проанализированные файлы, а также хорошее. CDF и вновь созданные Good.cat, находятся в одном каталоге.

``` syntax
[CatalogHeader]
Name=Good.cat
ResultDir=.\
PublicVersion=0x00000001
EncodingType=
CATATTR1=0x10010001:Movie1:FirstMovie
CATATTR2=0x10010001:Movie2:SecondMovie
CATATTR3=0x10010001:Movie3:ThirdMovie

[CatalogFiles]
UnsignedPE=.\UnsignedPE.EXE
UnsignedDOS=.\UnsignedDOS.EXE
<HASH>UnsignedCAB=.\Unsigned.CAB
UnsignedClass=.\Unsigned.Class
SignedPE=.\SignedPE.EXE
```

> [!Note]  
> Последняя запись в файле. CDF должна всегда содержать явный символ новой строки в конце строки.
