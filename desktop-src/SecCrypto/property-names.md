# <a name="property-names"></a>Имена свойств

Имя свойства [*запроса на сертификат*](../secgloss/c-gly.md) может быть "понятным" именем одного из общих свойств, распознаваемых службами сертификации, или [*идентификатором объекта*](../secgloss/o-gly.md) (OID). Дополнительные сведения о именах свойств запросов см. в разделе [Свойства имени](name-properties.md).
