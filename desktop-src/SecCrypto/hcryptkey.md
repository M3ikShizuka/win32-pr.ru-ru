# <a name="hcryptkey"></a>хкрипткэй

Тип данных **хкрипткэй** используется для представления дескрипторов [*криптографических ключей*](../secgloss/c-gly.md). Эти дескрипторы используются для указания модулю CSP, какой ключ используется в конкретной операции. Модуль CSP не обеспечивает прямой доступ к значениям ключей. Вместо этого пользователь выполняет функции, используя значение ключа через маркер ключа.


```C++
typedef ULONG_PTR HCRYPTKEY;
```
