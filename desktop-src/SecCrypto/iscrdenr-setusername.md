# <a name="iscrdenrsetusername-method"></a>Метод Искрденр:: Сетусернаме

Метод **сетусернаме** указывает имя пользователя, от имени которого предназначена регистрация сертификата.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT setUserName(
  [in] DWORD dwFlags,
  [in] BSTR bstrUserName
);
```


```VB

SCrdEnr.setUserName( _
  ByVal dwFlags, _
  ByVal bstrUserName _
)
```





## <a name="parameters"></a>Параметры

<dl> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Это значение должно быть либо бросить \_ Регистрация \_ \_ имени участника-пользователя (определяется как 1), либо бросить с \_ именем, \_ \_ совместимым \_ с SAM (определяется как 2).

Присвойте этому параметру значение бросить \_ Регистрация \_ \_ имени участника-пользователя, если имя, указанное в *бструсернаме* , является универсальным именем участника, например " someone@example.com ". Имя участника-пользователя должно соответствовать существующему имени диспетчера доступа безопасности (SAM).

Присвойте этому параметру значение бросить \_ \_ \_ , совместимое с SAM \_ , если имя пользователя, указанное в *бструсернаме* , является именем SAM в формате " \\ пользователь домена".

</dd> <dt>

*бструсернаме* \[ окне\]
</dt> <dd>

Имя пользователя.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

### <a name="vb"></a>VB

Если метод завершается успешно, метод возвращает значение S \_ ОК.

Если метод завершается с ошибкой, возвращается значение **HRESULT** , указывающее на ошибку. Список распространенных кодов ошибок см. в разделе [Общие значения HRESULT](common-hresult-values.md).

## <a name="remarks"></a>Комментарии

Вызовите этот метод, чтобы указать имя пользователя для выдачи [*смарт-карты*](../secgloss/s-gly.md). Альтернативой для **сетусернаме** является [**Искрденр:: селектусернаме**](iscrdenr-selectusername.md).

После указания имени пользователя его значение можно получить, вызвав метод [**суперпользователя.**](iscrdenr-getusername.md)
