# <a name="certificatestatus-object"></a>Объект Цертификатестатус

\[CAPICOM — это 32-разрядный компонент, доступный для использования в следующих операционных системах: Windows Server 2008, Windows Vista и Windows XP. Вместо этого используйте [**структуру X509ChainStatus**](/dotnet/api/system.security.cryptography.x509certificates.x509chainstatus?view=netcore-3.1) в пространстве имен [**System. Security. Cryptography. X509Certificates**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1) .\]

Объект **цертификатестатус** содержит сведения о создании цепочки доверия сертификатов.

## <a name="members"></a>Элементы

Объект **цертификатестатус** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **цертификатестатус** содержит эти методы.



| Метод                                                               | Описание                                                                                                                                  |
|:---------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------|
| [**аппликатионполиЦиес**](certificatestatus-applicationpolicies.md) | Возвращает коллекцию [**OID**](oids.md) , представляющую идентификатор объекта (OID) политики приложения.<br/>                                           |
| [**цертификатеполиЦиес**](certificatestatus-certificatepolicies.md) | Возвращает коллекцию [**OID**](oids.md) , представляющую идентификатор объекта (OID) политики сертификата, используемый во время построения цепочки.<br/>                |
| [**EKU**](certificatestatus-eku.md)                                 | Возвращает объект [**EKU**](eku.md) , используемый для задания элементов EKU, которые должны быть возвращены при установке состояния срока действия сертификата.<br/> |



 

### <a name="properties"></a>Свойства

Объект **цертификатестатус** имеет следующие свойства.



| Свойство                                                                        | Тип доступа           | Описание                                                                                                                                                                                                                                                       |
|:--------------------------------------------------------------------------------|:----------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Сертификаты**](certificatestatus-certificates.md)<br/>               | Чтение/запись<br/> | Задает или получает коллекцию сертификатов, которые можно использовать для построения цепочки сертификатов.<br/> **CAPICOM 2.0.0.3 и более ранние версии:** Свойство [**Certificates**](certificatestatus-certificates.md) не поддерживается.<br/>                    |
| [**чеккфлаг**](certificatestatus-checkflag.md)<br/>                     | Чтение/запись<br/> | Флаг проверки достоверности. Значения можно объединить с помощью логического оператора **или** . Флаг проверки по умолчанию — [**CAPICOM \_ Check on \_ Online \_**](capicom-check-flag.md).<br/>                                                                                     |
| [**Результат**](certificatestatus-result.md)<br/>                           | Только для чтения<br/>  | Указывает, действителен ли сертификат. Срок действия сертификата проверяется с использованием текущего значения свойства [**чеккфлаг**](certificatestatus-checkflag.md) и параметров политики сертификата. Это свойство по умолчанию.<br/> |
| [**урлретриевалтимеаут**](certificatestatus-urlretrievaltimeout.md)<br/> | Чтение/запись<br/> | Задает или получает продолжительность времени до того, как URL-адрес будет определен как недоступный.<br/>                                                                                                                                                                     |
| [**верификатионтиме**](certificatestatus-verificationtime.md)<br/>       | Чтение/запись<br/> | Задает или получает время проверки сертификата.<br/>                                                                                                                                                                                          |



 

## <a name="remarks"></a>Комментарии

Не удается создать объект **цертификатестатус** .

Объект **цертификатестатус** возвращается методом [**Certificate. IsValid**](certificate-isvalid.md) .
