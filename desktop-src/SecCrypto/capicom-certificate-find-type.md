# <a name="capicom_certificate_find_type-enumeration"></a>\_ \_ Перечисление типов поиска для сертификата CAPICOM \_

Тип перечисления " **CAPICOM \_ Certificate \_ Find \_ Type** " определяет тип условий поиска, используемый для поиска конкретных сертификатов. Этот тип перечисления появился в CAPICOM 2,0.

## <a name="members"></a>Элементы



| Член                                                | Описание                                                                                                                                 | Значение |
|-------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|-------|
| **CAPICOM \_ Certificate \_ найти \_ \_ хэш SHA1**            | Возвращает сертификаты, соответствующие указанному хэшу SHA1.<br/>                                                                             | 0     |
| **CAPICOM \_ Certificate \_ найти \_ имя субъекта \_**         | Возвращает сертификаты, имена субъектов которых полностью или частично совпадают с указанным именем субъекта.<br/>                                   | 1     |
| **CAPICOM \_ Certificate \_ найти \_ \_ имя издателя**          | Возвращает сертификаты, имя издателя которых полностью или частично совпадает с указанным именем издателя.<br/>                                     | 2     |
| **\_ \_ \_ имя корневого узла поиска CAPICOM Certificate \_**            | Возвращает сертификаты, имя корневого субъекта которых полностью или частично совпадает с указанным корневым именем субъекта.<br/>                         | 3     |
| **\_ \_ \_ имя шаблона поиска CAPICOM \_ Certificate**        | Возвращает сертификаты, имя шаблона которых соответствует указанному имени шаблона.<br/>                                                      | 4     |
| **\_ \_ найти расширение CAPICOM \_ Certificate**             | Возвращает сертификаты, имеющие расширение, соответствующее указанному расширению.<br/>                                                  | 5     |
| **\_ \_ \_ Расширенное свойство поиска CAPICOM Certificate \_**    | Возвращает сертификаты с расширенным свойством, идентификатор свойства которого соответствует указанному идентификатору свойства.<br/>           | 6     |
| **CAPICOM \_ Certificate \_ найти \_ \_ политику приложения**   | Возвращает сертификаты в хранилище, которые имеют расширение или свойство расширенного использования ключа, Объединенные с идентификатором использования.<br/> | 7     |
| **CAPICOM \_ Certificate \_ Поиск \_ \_ политики сертификата**   | Возвращает сертификаты, содержащие указанный OID политики.<br/>                                                                          | 8     |
| **\_ \_ \_ допустимое время поиска CAPICOM Certificate \_**           | Возвращает сертификаты, время которых допустимо.<br/>                                                                                        | 9     |
| **\_ \_ \_ недействительное время \_ \_ \_ поиска CAPICOM Certificate** | Возвращает сертификаты, время которых еще не является допустимым.<br/>                                                                                | 10    |
| **\_время поиска CAPICOM Certificate истекло \_ \_ \_**         | Возвращает сертификаты, время которых истекло.<br/>                                                                                     | 11    |
| **\_Использование CAPICOM Certificate \_ Поиск \_ ключа \_**            | Возвращает сертификаты, содержащие ключ, который можно использовать указанным образом.<br/>                                                  | 12    |



## <a name="remarks"></a>Комментарии

Тип перечисления **CAPICOM \_ Certificate \_ Find \_ Type** используется методом [**Certificates. Find**](certificates-find.md) .
