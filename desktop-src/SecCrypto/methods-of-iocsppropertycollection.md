# <a name="methods-of-iocsppropertycollection"></a>Методы Иоксппропертиколлектион

Следующие методы определяются интерфейсом [**иоксппропертиколлектион**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) . Методы доступа к свойствам не показаны здесь. Сведения о свойствах **иоксппропертиколлектион** см. в разделе [Свойства иоксппропертиколлектион](properties-of-iocsppropertycollection.md) .



| Метод                                                                                      | Описание                                                                               |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**CreateProperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-createproperty)                     | Создает новое свойство и добавляет его в набор свойств.                                    |
| [**DeleteProperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-deleteproperty)                     | Удаляет именованное свойство из набора свойств.                                             |
| [**жеталлпропертиес**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-getallproperties)                 | Возвращает все свойства в наборе свойств.                                                    |
| [**инитиализефромпропертиес**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-initializefromproperties) | Создает набор свойств из свойств, содержащихся в существующей конфигурации сервера. |
