# <a name="hcryptprov_legacy"></a>ХКРИПТПРОВ, \_ устаревший

Тип данных **хкриптпров \_ Legacy** используется для замены типа данных [**хкриптпров**](hcryptprov.md) , в котором тип данных **хкриптпров** больше не используется. Параметры функции и структуры данных, имеющие тип **хкриптпров \_ Legacy** , должны иметь значение **null**.


```C++
typedef ULONG_PTR HCRYPTPROV_LEGACY;
```
