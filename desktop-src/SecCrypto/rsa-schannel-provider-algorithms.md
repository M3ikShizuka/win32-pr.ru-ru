# <a name="rsaschannel-provider-algorithms"></a>Алгоритмы поставщика RSA/SChannel

[](../secgloss/r-gly.md) / Поставщик криптографии [*SChannel*](../secgloss/s-gly.md) для Microsoft RSA может поддерживать следующие алгоритмы.



| Идентификатор алгоритма    | Описание                                                                                                                         | Комментарии                                                                                                        |
|-----------------|-------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| КАЛГ \_ RSA \_ KEYX | [ *Алгоритм обмена ключами* открытого ключа RSA](../secgloss/k-gly.md) | Длина ключа: может быть задано, 384 бит на 16 384 бит по 8 битам. Длина ключа по умолчанию: 1 024 бит.<br/> |
| \_MD5 калг       | Алгоритм хэширования MD5.                                                                                                              | Предоставляется только для хэширования.                                                                                      |
| \_SHA калг       | Алгоритм хэширования SHA.                                                                                                              | Необходимо использовать для подписей DSS.                                                                                |
| КАЛГ \_ RC2       | Алгоритм шифрования блока RC2.                                                                                                     | Длина ключа: 40 до 88 бит                                                                                       |
| КАЛГ \_ RC4       | Алгоритм шифрования потока RC4.                                                                                                    | Длина ключа: 40 до 88 бит                                                                                       |
