# <a name="application-uses"></a>Приложение использует

Службы сертификатов выпускают сертификаты, которые можно использовать в приложениях инфраструктуры открытых ключей (PKI). Например, сертификаты работают в приложениях SSL, S/MIME и IPSEC. Возможны ситуации, когда эти сертификаты могут использоваться для цифровой подписи и шифрования сообщений. Службы сертификатов предоставляют сертификаты, [*списки отзыва сертификатов*](../secgloss/c-gly.md) (CRL) и точки распространения списков отзыва сертификатов (CDP), используемые приложениями Майкрософт и сторонних производителей.
