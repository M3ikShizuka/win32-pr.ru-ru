# <a name="additional-documentation-on-cryptography"></a>Дополнительная документация по шифрованию

Дополнительная документация по [*шифрованию*](../secgloss/c-gly.md) и проверке подлинности, а также связанные вопросы безопасности включают следующие публикации:

-   Адамс, Карлисле и Стив Ллойд. *Основные сведения об PKI: основные понятия, стандарты и рекомендации по развертыванию*. 2D ED. Бостон: Addison-Wesley Professional, 2002.
-   Хауслэй, шоу расса и Тим полк. *Планирование PKI: рекомендации по развертыванию инфраструктуры открытых ключей*. Нью Йорк: Джон Wiley & сыновьями, Inc., 2001.
-   Комара, Брайан. *Microsoft Windows Server 2003 PKI и безопасность сертификатов*. Redmond, штат Вашингтон: Microsoft Press, 2004.
-   Менезес, Алфред J., Пол C. Van Урсчот и Скотт A. Ванстоне. *Руководство по примененной криптографии*. Контрольная сумма нажмите серию для дискретных математических операций и его приложений. Бока Ратон: CRC Press, 1996.
-   Раина, Капил. *Решения безопасности PKI для предприятия: решение закона HIPAA, электронной документации и другие проблемы соответствия*. Индианаполис: Wiley Publishing, Inc., 2003.
-   Шнайера, Брюс. *Примененная криптография: протоколы, алгоритмы и исходный код на языке C*. Нью Йорк: Джон Wiley & сыновьями, 1996.
-   Стинсон, D. R. *Криптография: теория и практика* (контрольная сумма нажимайте серию для дискретных математических операций и приложений). Бока Ратон: CRC Press, 1995.
