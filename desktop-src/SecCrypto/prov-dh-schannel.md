# <a name="prov_dh_schannel"></a>PROV \_ DH \_ SChannel

\_ \_ Тип поставщика SChannel Prov DH поддерживает оба протокола: Diffie-Hellman и [*SChannel*](../secgloss/s-gly.md) .

| Обмен ключами | DH (эфемерный)                                                                                                                                         |
| Подпись    | [*DSS*](../secgloss/d-gly.md)                                                                                            |
| Шифрование   | [*Des*](../secgloss/d-gly.md)[*Triple DES*](../secgloss/t-gly.md)<br/> |
