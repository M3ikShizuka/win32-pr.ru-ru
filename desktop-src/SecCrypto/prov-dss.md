# <a name="prov_dss"></a>PROV \_ DSS

\_Тип поставщика Prov DSS, например Prov \_ RSA \_ SIG, поддерживает только [*хэши*](../secgloss/h-gly.md) и [*цифровые подписи*](../secgloss/d-gly.md). Алгоритм подписи, заданный \_ типом поставщика Prov DSS, является [*алгоритмом цифровых подписей*](../secgloss/d-gly.md) (DSA).

## <a name="algorithms-supported"></a>Поддерживаемые алгоритмы

Описание каждого из этих алгоритмов см. в глоссарии.



| Назначение      | Поддерживаемые алгоритмы                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| Ключевые Exchange | Нет                                                                                                                              |
| Подпись    | [*DSS*](../secgloss/d-gly.md)                                                                       |
| Encryption   | Нет                                                                                                                              |
| Хэширование      | [*MD5*](../secgloss/m-gly.md)[*SHA*](../secgloss/s-gly.md)<br/> |
