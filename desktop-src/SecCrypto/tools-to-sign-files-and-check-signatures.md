# <a name="tools-to-sign-files-and-check-signatures"></a>Средства для подписывания файлов и проверки подписей

Средства CryptoAPI — это следующие средства, которые можно использовать для цифрового подписывания файлов, проверки подписей на файлы и создания файлов каталога.



| Средство                     | Комментарии                                                                                                                                                                                                                                                                            |
|--------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [макекат](makecat.md)   | Создает неподписанный файл каталога. Этот файл каталога впоследствии используется для сравнения [*хэшей*](../secgloss/h-gly.md) в файле каталога с хэшами отдельных файлов, что позволяет доказать, что исходные файлы не могут быть незаконными. |
| [SetReg](setreg.md)     | Задает разделы реестра, управляющие процессом проверки сертификатов.                                                                                                                                                                                                              |
| [SignTool](signtool.md) | Выполняет цифровую подпись файлов и подтверждает цифровые подписи файлов.                                                                                                                                                                                                                    |
