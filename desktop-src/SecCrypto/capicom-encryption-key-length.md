# <a name="capicom_encryption_key_length-enumeration"></a>\_ \_ Перечисление длины ключей шифрования CAPICOM \_

Тип перечисления " **\_ \_ \_ Длина ключа шифрования CAPICOM** " определяет [*длину ключа*](../secgloss/k-gly.md) , используемую при шифровании.

## <a name="members"></a>Элементы



| Член                                          | Описание                                                                               | Значение     |
|-------------------------------------------------|-------------------------------------------------------------------------------------------|-----------|
| **\_ \_ \_ Максимальная длина ключа шифрования \_ CAPICOM**   | Использует максимальную длину ключа, доступную в указанном алгоритме шифрования.<br/> | 0         |
| **\_ \_ Длина ключа шифрования \_ CAPICOM \_ 40 \_ бит**  | Использует 40-разрядные ключи.<br/>                                                              | 1         |
| **\_ \_ Длина ключа шифрования \_ CAPICOM \_ 56 \_ бит**  | Использует 56-разрядные ключи, если они доступны.<br/>                                                 | 2         |
| **\_ \_ Длина ключа шифрования \_ CAPICOM \_ 128 \_ бит** | Использует 128-разрядные ключи, если они доступны.<br/>                                                | 3         |
| **\_ \_ Длина ключа шифрования \_ CAPICOM \_ 192 \_ бит** | Использует 192-разрядные ключи. Эта длина ключа доступна только для алгоритма AES.<br/>                  | 4//v 2.0 |
| **\_ \_ Длина ключа шифрования \_ CAPICOM \_ 256 \_ бит** | Использует 256-разрядные ключи. Эта длина ключа доступна только для алгоритма AES.<br/>                  | 5//v 2.0 |



## <a name="remarks"></a>Комментарии

Тип перечисления " **CAPICOM" \_ \_ \_ длины ключа шифрования** используется свойством [**Algorithm. keylength**](algorithm-keylength.md) .
