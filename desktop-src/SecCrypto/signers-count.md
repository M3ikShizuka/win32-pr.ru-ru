# <a name="signerscount-property"></a>Свойство Signs. Count

\[Свойство **Count** доступно для использования в операционных системах, указанных в разделе требования. Вместо этого используйте коллекцию объектов Кмссигнер. Дополнительные сведения см. в описании [**класса кмссигнер**](/dotnet/api/system.security.cryptography.pkcs.cmssigner?view=dotnet-plat-ext-3.1&preserve-view=true) в пространстве имен [**System. Security. Cryptography. PKCS**](/dotnet/api/system.security.cryptography.pkcs?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

Свойство **Count** извлекает количество объектов [**подписывающих**](signer.md) в коллекции.

## <a name="syntax"></a>Синтаксис


```VB
Signers.Count As Long
```



## <a name="property-value"></a>Значение свойства

Число объектов [**подписывающих**](signer.md) в коллекции.
