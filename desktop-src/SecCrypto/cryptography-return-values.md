# <a name="cryptography-return-values"></a>Возвращаемые значения криптографии

Следующие возвращаемые значения используются криптографической функцией и определяются в Винкрипт. h. Возвращаемые криптографические значения классифицируются в соответствии со следующим образом.



| Возвращаемое значение                                                                       | Описание                                                                                                |
|------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [ASN. 1. возвращаемые значения кодирования/декодирования](asn-1-encoding-decoding-return-values.md) | Стандартные константы, возвращаемые функциями, которые выполняют ASN. 1 кодирование или декодирование.                    |
| [Возвращаемые значения сертификата и доверия](certificate-and-trust-return-values.md)     | Предопределенные константы, возвращаемые при определении срока действия и состояния доверия сертификата. |



 

Список возвращаемых значений CAPICOM см. в разделе [**\_ \_ код ошибки CAPICOM**](capicom-error-code.md).
