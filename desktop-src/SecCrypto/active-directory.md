# <a name="active-directory"></a>Active Directory

Службы сертификатов поддерживают использование Active Directory. В случае с [*центром сертификации*](../secgloss/c-gly.md) предприятия службы сертификации требуют использования Active Directory. ЦС предприятия публикует сертификаты пользователей и [*списки отзыва сертификатов*](../secgloss/c-gly.md) (CRL) в Active Directory. Изолированный ЦС можно администрировать для выполнения тех же действий, при условии, что Active Directory доступен автономному центру сертификации и используется модуль настраиваемой политики.
