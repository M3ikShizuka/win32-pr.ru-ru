# <a name="prov_rsa_schannel"></a>PROV \_ RSA \_ SChannel

\_ \_ Тип поставщика Prov RSA SChannel поддерживает протоколы RSA и [*SChannel*](../secgloss/s-gly.md) .

## <a name="algorithms-supported"></a>Поддерживаемые алгоритмы

Описание каждого из этих алгоритмов см. в глоссарии.



| Назначение      | Поддерживаемые алгоритмы                                                                                                                                                                                                                                                                                                          |
|--------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Обмен ключами | [*RSA*](../secgloss/r-gly.md)                                                                                                                                                                                                                                                                   |
| Подпись    | RSA                                                                                                                                                                                                                                                                                                                           |
| Шифрование   | Любой \_ CSP Prov RSA \_ SChannel должен реализовывать один или несколько из следующих алгоритмов: [ *RC4*](../secgloss/r-gly.md)<br/> [*DES*](../secgloss/d-gly.md)<br/> [*Triple DES*](../secgloss/t-gly.md)<br/> |
| Хэширование      | [*MD5*](../secgloss/m-gly.md)[*SHA*](../secgloss/s-gly.md)<br/>                                                                                                                                                                                             |
