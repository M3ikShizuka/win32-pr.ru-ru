# <a name="certificates-and-cryptoapi"></a>Сертификаты и CryptoAPI

[*CryptoAPI*](../secgloss/c-gly.md) поддерживает использование сертификатов X. 509, как определено в [IETF RFC 3280](https://www.ietf.org/rfc/rfc3280.txt). В этой документации предполагается использование [*цифрового сертификата*](../secgloss/c-gly.md)X. 509 или сравнимого.

Стандартный сертификат X. 509 содержит следующие сведения.



| Поле                | Описание                                                                                                                                                                                                                |
|----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Версия              | Номер версии сертификата.                                                                                                                                                                                         |
| Серийный номер        | Серийный номер сертификата.                                                                                                                                                                                          |
| Идентификатор алгоритма | Алгоритм подписи, используемый подписавшим сертификатом.                                                                                                                                                                        |
| Имя издателя          | Имя издателя сертификата.                                                                                                                                                                                     |
| До           | Дата, до которой сертификат является недействительным.                                                                                                                                                                            |
| Не позднее            | Дата, после которой сертификат является недействительным.                                                                                                                                                                             |
| Имя субъекта         | Имя пользователя или сущности, которым выдан сертификат.                                                                                                                                                      |
| Алгоритм            | Алгоритм, используемый для открытого ключа.                                                                                                                                                                                         |
| Открытый ключ субъекта   | Фактический открытый ключ (битовая строка).                                                                                                                                                                                          |
| Уникальный идентификатор издателя     | Необязательное поле. Если он указан, версия должна иметь версию 2.                                                                                                                                                                     |
| Уникальный идентификатор субъекта    | Необязательное поле. Если он указан, версия должна иметь версию 2.                                                                                                                                                                     |
| Модули           | Дополнительное поле. Представляет дополнительные данные, которые издатель может добавить к сертификату, например адрес электронной почты или авторизацию для выдачи сертификатов. Если имеются расширения, версия должна иметь версию 3.<br/> |
