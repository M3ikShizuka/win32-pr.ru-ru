# <a name="utilitiesbytearraytobinarystring-method"></a>Метод Utilities. Битеаррайтобинаристринг

\[Метод **битеаррайтобинаристринг** доступен для использования в операционных системах, указанных в разделе требования.\]

Метод **битеаррайтобинаристринг** преобразует массив байтов в двоичную строку.

## <a name="syntax"></a>Синтаксис


```VB
Utilities.ByteArrayToBinaryString( _
  ByVal varByteArray _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*варбитеаррай* \[ окне\]
</dt> <dd>

Массив байтов для преобразования в упакованную двоичную строку.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Двоичная упакованная строка, содержащая результат преобразования.
