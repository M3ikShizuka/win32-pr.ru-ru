# <a name="properties-of-iocsppropertycollection"></a>Свойства Иоксппропертиколлектион

Следующие свойства определяются интерфейсом [**иоксппропертиколлектион**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) .



| Свойство                                                        | Описание                                              |
|-----------------------------------------------------------------|----------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get__newenum)    | Возвращает перечислитель для набора свойств.                   |
| [**Count**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_count)           | Возвращает число свойств в наборе свойств.         |
| [**Компонент**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_item)             | Возвращает свойство, определяемое по индексу в наборе свойств. |
| [**итембинаме**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_itembyname) | Возвращает свойство, определяемое по имени в наборе свойств.  |
