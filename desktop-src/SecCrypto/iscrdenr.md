# <a name="iscrdenr-interface"></a>Интерфейс Искрденр

Интерфейс **искрденр** представляет Управление регистрацией смарт-карты. В основном это интерес для разработчиков, не использующих автоматизацию. сведения о программировании в Visual Basic или другом языке автоматизации см. в разделе объект [**ценролл**](/previous-versions/windows/desktop/legacy/aa376007(v=vs.85)) .

## <a name="members"></a>Элементы

Интерфейс **искрденр** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Искрденр** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Интерфейс **искрденр** содержит следующие методы.



| Метод                                                                    | Описание                                                                                                                                                                                                                                                                                                                                                                                                         |
|:--------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**enroll**](/previous-versions/windows/desktop/legacy/aa386564(v=vs.85))                                         | Запрашивает сертификат от имени пользователя и сохраняет полученный [*сертификат*](../secgloss/c-gly.md) на [*смарт-карте*](../secgloss/s-gly.md)пользователя.<br/>                                                                                                                                                |
| [**енумканаме**](iscrdenr-enumcaname.md)                                 | Перечисляет имена [*центров сертификации*](../secgloss/c-gly.md) (ЦС) для заданного имени шаблона сертификата.<br/>                                                                                                                                                                                                       |
| [**енумцерттемплатенаме**](iscrdenr-enumcerttemplatename.md)             | Перечисляет имена шаблонов сертификатов.<br/>                                                                                                                                                                                                                                                                                                                                                               |
| [**енумкспнаме**](iscrdenr-enumcspname.md)                               | Перечисляет имена доступных [*поставщиков служб шифрования*](../secgloss/c-gly.md) (CSP).<br/>                                                                                                                                                                                                               |
| [**жеткакаунт**](iscrdenr-getcacount.md)                                 | Возвращает число центров сертификации, желающих выдать сертификат на основе указанного шаблона сертификата.<br/>                                                                                                                                                                                                                                                                                                    |
| [**жетканаме**](iscrdenr-getcaname.md)                                   | Возвращает имя указанного ЦС для данного шаблона сертификата.<br/>                                                                                                                                                                                                                                                                                                                                 |
| [**жетцерттемплатекаунт**](iscrdenr-getcerttemplatecount.md)             | Извлекает количество шаблонов сертификатов.<br/>                                                                                                                                                                                                                                                                                                                                                           |
| [**жетцерттемплатенаме**](iscrdenr-getcerttemplatename.md)               | Извлекает имя шаблона сертификата.<br/>                                                                                                                                                                                                                                                                                                                                                          |
| [**жетцерттемплатесмиме**](iscrdenr-getcerttemplatesmime.md)             | Определите, содержит ли шаблон сертификата \_ \_ \_ Использование ключа защиты электронной почты сзоид пкикс \_ . Если это использование ключа является частью шаблона сертификата, шаблон сертификата поддерживает операции с [*безопасными и многоцелевыми расширениями электронной почты Интернета*](../secgloss/s-gly.md) (S/MIME).<br/> |
| [**жетенролледцертификатенаме**](iscrdenr-getenrolledcertificatename.md) | Возвращает имя сертификата, полученное в результате предыдущего успешного вызова [**искрденр::**](/previous-versions/windows/desktop/legacy/aa386564(v=vs.85))Recall. Этот метод также можно использовать для вывода сертификата в диалоговом окне.<br/>                                                                                                                                                                                                 |
| [**жетсигнингцертификатенаме**](iscrdenr-getsigningcertificatename.md)   | Извлекает имя субъекта из сертификата подписи. Этот метод также можно использовать для вывода сертификата в диалоговом окне. <br/>                                                                                                                                                                                                                                                                       |
| [**getUserName**](iscrdenr-getusername.md)                               | Возвращает имя пользователя, от имени которого предназначена регистрация сертификата.<br/>                                                                                                                                                                                                                                                                                                                   |
| [**ресетусер**](iscrdenr-resetuser.md)                                   | Удаляет имя пользователя из элемента управления смарт-картой.<br/>                                                                                                                                                                                                                                                                                                                                                        |

| [**селектусернаме**](iscrdenr-selectusername.md)                         | Отображает диалоговое окно **Выбор пользователя** , в котором можно выбрать имя пользователя. Имя пользователя применяется к пользователю, от имени которого предназначена регистрация сертификата.<br/>                                                                                                                                                                                                                                     |

| [**сетсигнингцертификате**](iscrdenr-setsigningcertificate.md)           | Указывает сертификат для подписи (также известный как *сертификат агента регистрации*).<br/>                                                                                                                                                                                                                                                                                                                      |
| [**сетусернаме**](iscrdenr-setusername.md)                               | Указывает имя пользователя, от имени которого предназначена регистрация сертификата.<br/>                                                                                                                                                                                                                                                                                                                   |
