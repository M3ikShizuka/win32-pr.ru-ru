# <a name="using-cert2spc"></a>Использование Cert2SPC

следующая команда заключает в оболочку сертификат [*X. 509*](../secgloss/x-gly.md) , *MyCert*. cer, в тестовый \# сертификат PKCS 7 [*Software Publisher certificate*](../secgloss/s-gly.md) (SPC), именуемый *MyCert*. SPC. Созданный SPC будет использоваться только в целях тестирования. SPC, используемый для фактического подписывания кода, который необходимо распространить на общедоступную службу, должен быть получен от ГТЕ, VeriSign, Inc. или другого доверенного [*центра сертификации*](../secgloss/c-gly.md) (CA).
