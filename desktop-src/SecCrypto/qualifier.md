# <a name="qualifier-object"></a>Объект квалификатора

\[Объект **квалификатора** доступен для использования в операционных системах, указанных в разделе требования. Вместо этого используйте [**класс X509Extension**](/dotnet/api/system.security.cryptography.x509certificates.x509extension?view=netcore-3.1) в пространстве имен [**System. Security. Cryptography. X509Certificates**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1) , вызвав конструктор, принимающий OID в качестве параметра, а затем использовать OID для политик сертификатов для обработки квалификаторов, которые являются частью сведений о политике в расширении политик сертификатов.\]

Объект **квалификатора** представляет собой указатель на инструкцию по сертификации (CPS) или квалификатор уведомления пользователя.

## <a name="when-to-use"></a>Назначение

Объект **квалификатора** используется для выполнения следующих задач:

-   Получите идентификатор объекта квалификатора.
-   Получите универсальный код ресурса (URI), указывающий на сервер публикаций Contribute, опубликованный [*центром сертификации*](../secgloss/c-gly.md) (ЦС).
-   Получите имя Организации, связанной с квалификатором.
-   Получение имени и содержимого уведомления пользователя.

| Свойство                                                          | Тип доступа          | Описание                                                                                                                         |

| [**кпспоинтер**](qualifier-cpspointer.md)<br/>             | Только для чтения<br/> | Извлекает строку, содержащую URI, указывающий на сервер публикаций Contribute, опубликованный центром сертификации.<br/>                                       |
| [**експлиЦиттекст**](qualifier-explicittext.md)<br/>         | Только для чтения<br/> | Извлекает строку, содержащую содержимое уведомления пользователя. Это свойство может быть пустым.<br/>                             |
