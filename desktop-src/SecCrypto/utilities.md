# <a name="utilities-object"></a>Служебный объект

\[Объект **служебных программ** доступен для использования в операционных системах, указанных в разделе требования.\]

Объект **Utilities** предоставляет функциональные возможности для генерации случайных чисел, преобразования, кодирования и декодирования из Base64.

## <a name="when-to-use"></a>Назначение

Объект **Utilities** используется для выполнения следующих задач:

-   Кодирует строку в кодировке Base64 или декодировать строку из Base64.
-   Преобразуйте двоичную упакованную строку в массив байтов и наоборот.
-   Преобразовать двоичную упакованную строку в шестнадцатеричную строку и наоборот.
-   Создание безопасного случайного числа.
-   Преобразование местного времени в время в формате UTC (время по Гринвичу) и наоборот.
