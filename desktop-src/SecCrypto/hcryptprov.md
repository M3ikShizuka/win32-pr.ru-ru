# <a name="hcryptprov"></a>хкриптпров

Тип данных **хкриптпров** используется для представления дескрипторов CSP. Эти дескрипторы используются для указания того, какой модуль CSP выполняет определенные операции.


```C++
typedef ULONG_PTR HCRYPTPROV;
```
