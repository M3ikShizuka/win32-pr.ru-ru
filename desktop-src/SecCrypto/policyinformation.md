# <a name="policyinformation-object"></a>Объект Полициинформатион

\[Объект **полициинформатион** доступен для использования в операционных системах, указанных в разделе требования. Вместо этого используйте [**класс X509Extension**](/dotnet/api/system.security.cryptography.x509certificates.x509extension?view=netcore-3.1) в пространстве имен [**System. Security. Cryptography. X509Certificates**](/dotnet/api/system.security.cryptography.x509certificates.publickey.-ctor?view=netcore-3.1) , вызвав конструктор, принимающий OID в качестве параметра, а затем использовать OID для политик сертификатов для обработки сведений о политике в расширении политик сертификатов.\]

Объект **полициинформатион** предоставляет доступ к сведениям о политике расширения.

## <a name="when-to-use"></a>Назначение

Объект **полициинформатион** используется для выполнения следующих задач:

-   Получите идентификатор объекта политики.
-   Получение коллекции квалификаторов политики.

## <a name="members"></a>Элементы

Объект **полициинформатион** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Элемент Property

Объект **полициинформатион** имеет следующие свойства.



| Свойство                                                      | Описание                                                                                                 |
|:--------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------|
| [**КОДА**](policyinformation-oid.md)<br/>               | Получает OID политики в виде объекта [**OID**](oid.md) . Это свойство по умолчанию.<br/>       |
| [**Квалификаторы**](policyinformation-qualifiers.md)<br/> | Извлекает коллекцию квалификаторов политики в виде объекта [**квалификаторов**](qualifiers.md) .<br/> |



 

## <a name="remarks"></a>Remarks

Не удается создать объект **полициинформатион** .
