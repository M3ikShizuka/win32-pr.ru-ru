# <a name="capicom_eku-enumeration"></a>\_Перечисление EKU в CAPICOM

Тип перечисления **CAPICOM \_ EKU** определяет имена расширенного использования ключа в зависимости от того, где их можно использовать.

## <a name="members"></a>Элементы



| Член                                     | Описание                                                                                                                                                 | Значение |
|--------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|-------|
| **\_другое расширенное EKU \_**                    | В сертификате используются определенные в локальной политике. Используется, если требуемое EKU не определено и значение OID должно быть задано приложением.<br/> | 0     |
| **\_ \_ \_ Проверка подлинности сервера CAPICOM EKU**             | Сертификат можно использовать для проверки подлинности сервера.<br/>                                                                                                | 1     |
| **\_ \_ \_ Проверка подлинности клиента CAPICOM EKU**             | Сертификат можно использовать для проверки подлинности клиента.<br/>                                                                                                | 2     |
| **\_Подписывание \_ кода CAPICOM EKU \_**            | Сертификат можно использовать для создания цифровой подписи.<br/>                                                                                           | 3     |
| **\_ \_ Защита электронной почты CAPICOM EKU \_**        | Сертификат можно использовать для защиты электронной почты.<br/>                                                                                                    | 4     |
| **\_Вход в \_ смарт-карту CAPICOM EKU \_**         | Сертификат можно использовать для входа с использованием смарт-карты. Представлено в CAPICOM 2,0.<br/>                                                                         | 5     |
| **\_ \_ \_ Файловая система с шифрованием CAPICOM EKU \_** | Сертификат можно использовать для EFS. Представлено в CAPICOM 2,0.<br/>                                                                                      | 6     |



## <a name="remarks"></a>Комментарии

Тип перечисления **CAPICOM \_ EKU** используется в [**EKU. Имя**](eku-name.md) свойства.
