# <a name="cryptography-constants"></a>Константы шифрования

Следующие константы используются функциями шифрования и определены в Винкрипт. h. Константы криптографии делятся на категории по использованию следующим образом.



| Константа                                                                                                           | Описание                                                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Константы для CryptEncodeObject и Криптдекодеобжект](constants-for-cryptencodeobject-and-cryptdecodeobject.md) | Предопределенные константы, используемые с [*нотацией абстрактного синтаксиса: одна*](../secgloss/a-gly.md) (ASN. 1) операции кодирования и декодирования. |
| [Константы для расширений Netscape](constants-for-netscape-extensions.md)                                         | Предопределенные константы, используемые для расширений сертификатов Netscape. В этом разделе приводятся подробные сведения об использовании этих расширений с операциями кодирования и декодирования ASN. 1.                                      |
| [Имена поставщиков служб шифрования](cryptographic-provider-names.md)                                                   | Стандартные строковые константы для имен распространенных [*поставщиков служб шифрования*](../secgloss/c-gly.md) (CSP).   |
| [**максуидлен**](maxuidlen.md)                                                                                     | Числовая константа, указывающая максимальное число символов, которые будут использоваться некоторыми поставщиками служб шифрования Майкрософт при получении идентификатора пользователя.                                                  |
| [**Константы кэширования закрытого ключа**](private-key-caching-constants.md)                                             | Константы, представляющие записи реестра, управляющие кэшированием [*закрытых ключей*](../secgloss/p-gly.md) CSP.                                                |
| [Флаги протокола](protocol-flags.md)                                                                               | Стандартные константы для протоколов шифрования.                                                                                                                                                            |
