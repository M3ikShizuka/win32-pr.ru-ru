# <a name="cryptxml"></a>криптксмл

Криптксмл предоставляет низкоуровневый набор собственных API-интерфейсов, которые реализуют создание и проверку цифровых подписей XML на основе синтаксиса подписи XML и обработки (второго выпуска) рекомендации W3C. Разработчики должны быть знакомы с этим стандартом для эффективного использования интерфейсов API цифровых подписей Криптксмл.

Криптксмл содержит следующие разделы:

-   [Общие сведения о цифровой подписи XML](xml-digital-signature-overview.md)
-   [Функциональность API цифровых подписей XML](xml-digital-signature-api-functionality.md)
-   [Алгоритмы шифрования цифровых подписей XML](xml-digital-signature-cryptographic-algorithms.md)
-   [Криптографические расширения цифровых подписей XML](xml-digital-signature-cryptographic-extensions.md)
-   [Вопросы безопасности цифровых подписей XML](xml-digital-signature-security-considerations.md)
