# <a name="hcrypthash"></a>хкрипсаш

Тип данных **хкрипсаш** используется для представления дескрипторов для [*хэш-объектов*](../secgloss/h-gly.md). Эти дескрипторы указывают модулю CSP, какой хэш используется в определенной операции. Модуль CSP не включает прямую обработку хэш-значений. Вместо этого пользователь управляет хэш-значениями с помощью хэш-маркера.


```C++
typedef ULONG_PTR HCRYPTHASH;
```
