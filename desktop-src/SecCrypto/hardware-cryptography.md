# <a name="hardware-cryptography"></a>Аппаратная криптография

Службы сертификатов поддерживают криптографию, связанную с оборудованием, например выдачу сертификатов, хранимых на [*смарт-картах*](../secgloss/s-gly.md). [*Центр сертификации*](../secgloss/c-gly.md) предприятия (ЦС) поддерживает выдачу сертификатов смарт-карт для входа в домен в операционной системе. Кроме того, ЦС предприятия и автономный ЦС поддерживают выдачу других типов сертификатов смарт-карт.

Дополнительные сведения см. в документации по продукту. Сведения для разработчиков о смарт-картах см. в статье [Проверка подлинности смарт-карт](../secauthn/smart-card-authentication.md). Сведения об элементе управления, который позволяет администратору запрашивать сертификаты смарт-карт от имени пользователей, см. в разделе [Управление регистрацией смарт-карт](/previous-versions/windows/desktop/secsmart/smart-card-enrollment-control-usage-scenario).
