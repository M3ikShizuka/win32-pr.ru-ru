# <a name="systemdevicesaepcontainersupportsnetworking"></a>System. Devices. Аепконтаинер. Суппортснетворкинг

Указывает, поддерживает ли контейнер конечной точки сопоставления устройств Суппортснетворкинг

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10, версия 1703, Windows 10, версия 1607

```
propertyDescription
   name = System.Devices.AepContainer.SupportsNetworking
   shellPKey = PKEY_Devices_AepContainer_SupportsNetworking
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
