# <a name="implementing-property-handlers"></a>Реализация обработчиков свойств

В Windows Vista и более поздних версиях метаданные стали центральным способом организации таких элементов, как файлы, электронная почта или контакты. Чтобы включить систему, в которой можно выполнять поиск элементов на основе их метаданных, а пользователи могут читать или записывать эти метаданные, в Windows Vista появилась новая система свойств. Метаданные в этой системе представлены расширяемым набором свойств. В этом наборе разделов описывается создание обработчика, который считывает и записывает свойства в файловый поток и из него. Эти обработчики называются обработчиками свойств, каждый из которых связан с заданными типами файлов, определяемыми расширением имени файла.
