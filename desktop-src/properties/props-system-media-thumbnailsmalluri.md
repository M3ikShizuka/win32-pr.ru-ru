# <a name="systemmediathumbnailsmalluri"></a>System. Media. Сумбнаилсмаллури

Универсальный код ресурса (URI) представления большого эскиза элемента мультимедиа.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.Media.ThumbnailSmallUri
   shellPKey = PKEY_Media_ThumbnailSmallUri
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 50
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
