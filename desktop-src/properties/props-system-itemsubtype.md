# <a name="systemitemsubtype"></a>System. Итемсубтипе

Описывает подтип элемента. Значение предназначено для вывода пользователю. В отличие от System. ItemType, который обычно используется для описания класса итемссат ALL, имеет одинаковый общий формат содержимого, System. Итемсубтипе может отличаться в зависимости от содержимого или назначения элемента. Например, это свойство может использоваться для обнаружения элемента с System. ItemType = "jpg" в виде System. Итемсубтипе = "Панорама" или System. Итемсубтипе = "Smart-снимок".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.ItemSubType
   shellPKey = PKEY_ItemSubType
   formatID = 28636AA6-953D-11D2-B5D6-00C04FD918D0
   propID = 37
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = false
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
