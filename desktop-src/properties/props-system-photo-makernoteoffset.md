# <a name="systemphotomakernoteoffset"></a>System. photo. Макернотеоффсет

Смещение заметки студии, указанной в [System. photo. макерноте](./props-system-photo-makernote.md). Дополнительные сведения см. в спецификации файла обмена изображениями (EXIF) 2,2. Это свойство не предназначено для отображения в оболочке, но доступно программно для приложений.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.MakerNoteOffset
   shellPKey = PKEY_Photo_MakerNoteOffset
   formatID = 813F4124-34E6-4D17-AB3E-6B1F3C2247A1
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
