# <a name="systemphotosubjectdistancenumerator"></a>System. photo. Субжектдистанценумератор

Числитель \_ субжектдистанценого фото в PKEY \_ .

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.SubjectDistanceNumerator
   shellPKey = PKEY_Photo_SubjectDistanceNumerator
   formatID = 8AF4961C-F526-43E5-AA81-DB768219178D
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
```
