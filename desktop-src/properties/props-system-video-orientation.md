# <a name="systemvideoorientation"></a>System. Video. Orientation

Это ориентация видео в градусах.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Video.Orientation
   shellPKey = PKEY_Video_Orientation
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 99
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
