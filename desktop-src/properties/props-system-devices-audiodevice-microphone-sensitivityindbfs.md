# <a name="systemdevicesaudiodevicemicrophonesensitivityindbfs"></a>System.Devices.AudioDevice.Microphone.SensitivityInDbfs

Сведения о конфиденциальности в dBFS для устройства с микрофоном.

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10, версия 1703, Windows 10, версия 1607

```
propertyDescription
   name = System.Devices.AudioDevice.Microphone.SensitivityInDbfs
   shellPKey = PKEY_Devices_AudioDevice_Microphone_SensitivityInDbfs
   formatID = 8943B373-388C-4395-B557-BC6DBAFFAFDB
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
