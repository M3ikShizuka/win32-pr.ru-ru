# <a name="systemstorageproviderfileversion"></a>System. Сторажепровидерфилеверсион

Версия файла поставщика хранилища для этого файла.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1

```
propertyDescription
   name = System.StorageProviderFileVersion
   shellPKey = PKEY_StorageProviderFileVersion
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
      IsInnate = false
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
