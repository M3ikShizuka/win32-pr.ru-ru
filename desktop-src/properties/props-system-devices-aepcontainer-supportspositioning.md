# <a name="systemdevicesaepcontainersupportspositioning"></a>System. Devices. Аепконтаинер. Суппортспоситионинг

Указывает, поддерживает ли контейнер конечной точки сопоставления устройств расположение

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10, версия 1703, Windows 10, версия 1607

```
propertyDescription
   name = System.Devices.AepContainer.SupportsPositioning
   shellPKey = PKEY_Devices_AepContainer_SupportsPositioning
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
