# <a name="systemdocumentdatecreated"></a>System. Document. DateCreated

Указывает дату и время создания документа. Эта информация хранится в документе, а не извлекается из файловой системы.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Document.DateCreated
   shellPKey = PKEY_Document_DateCreated
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Document.DateCreated
   shellPKey = PKEY_Document_DateCreated
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 12
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
