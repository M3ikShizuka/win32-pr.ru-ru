# <a name="systemdevicesdiscoverymethod"></a>System. Devices. Дисковеримесод

Указывает транспорт или физическое подключение, на котором обнаружено устройство.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Devices.DiscoveryMethod
   shellPKey = PKEY_Devices_DiscoveryMethod
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 52
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
