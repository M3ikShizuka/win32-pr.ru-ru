# <a name="systemimageparsingname"></a>System. Имажепарсингнаме

Имя синтаксического анализа образа.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1

```
propertyDescription
   name = System.ImageParsingName
   shellPKey = PKEY_ImageParsingName
   formatID = D7750EE0-C6A4-48EC-B53E-B87B52E6D073
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="windows-8-windows-7"></a>Windows 8, Windows 7

```
propertyDescription
   name = System.ImageParsingName
   shellPKey = PKEY_ImageParsingName
   formatID = D7750EE0-C6A4-48EC-B53E-B87B52E6D073
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
