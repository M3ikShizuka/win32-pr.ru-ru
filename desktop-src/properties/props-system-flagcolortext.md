# <a name="systemflagcolortext"></a>System. Флагколортекст

Понятная пользователю форма [System. флагколор](./props-system-flagcolor.md). Это значение не предназначено для синтаксического анализа программным способом.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.FlagColorText
   shellPKey = PKEY_FlagColorText
   formatID = 45EAE747-8E2A-40AE-8CBF-CA52ABA6152A
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
