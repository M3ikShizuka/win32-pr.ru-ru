# <a name="systemdeviceinterfacebluetoothdeviceaddress"></a>System. Девицеинтерфаце. Bluetooth. девицеаддресс

адрес устройства Bluetooth.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.DeviceInterface.Bluetooth.DeviceAddress
   shellPKey = PKEY_DeviceInterface_Bluetooth_DeviceAddress
   formatID = 2BD67D8B-8BEB-48D5-87E0-6CDA3428040A
   propID = 1
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
