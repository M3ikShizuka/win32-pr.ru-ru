# <a name="systemphotofocalplaneyresolution"></a>System. photo. Фокалпланэйресолутион

Указывает число пикселей в высоте изображения (направление по оси Y) на Фокалпланересолутионунит в плоскости, направляющей камеру. Вычислено на основе \_ \_ фокалпланэйресолутионнумератор и \_ фокалпланэйресолутионденоминатор фото на PKEY \_ .

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FocalPlaneYResolution
   shellPKey = PKEY_Photo_FocalPlaneYResolution
   formatID = 4FFFE4D0-914F-4AC4-8D6F-C9C61DE169B1
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
