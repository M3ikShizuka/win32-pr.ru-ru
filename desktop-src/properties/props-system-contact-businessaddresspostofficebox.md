# <a name="systemcontactbusinessaddresspostofficebox"></a>System. Contact. Бусинессаддресспостоффицебокс

Номер почтового ящика для контакта в поле "Рабочий адрес".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Contact.BusinessAddressPostOfficeBox
   shellPKey = PKEY_Contact_BusinessAddressPostOfficeBox
   formatID = BC4E71CE-17F9-48D5-BEE9-021DF0EA5409
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```
