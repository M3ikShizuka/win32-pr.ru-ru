# <a name="systemsearchqueryfocusedsummary"></a>System. Search. Куерифокуседсуммари

Сводка документа, ориентированная на запрос. Это автоматически создается системой поиска. Это свойство используется только при использовании значений в приложении, а не для предоставления значений обработчику свойства.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.QueryFocusedSummary
   shellPKey = PKEY_Search_QueryFocusedSummary
   formatID = 560C36C0-503A-11CF-BAA1-00004C752A9A
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
