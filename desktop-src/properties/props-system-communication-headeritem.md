# <a name="systemcommunicationheaderitem"></a>System. Communication. HeaderItem

Это свойство имеет значение true, если элемент является элементом заголовка, что означает, что элемент не был полностью скачан.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Communication.HeaderItem
   shellPKey = PKEY_Communication_HeaderItem
   formatID = C9C34F84-2241-4401-B607-BD20ED75AE7F
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
