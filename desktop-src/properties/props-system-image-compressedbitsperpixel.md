# <a name="systemimagecompressedbitsperpixel"></a>System. Image. Компресседбитсперпиксел

Указывает уровень сжатия изображения. . Вычислено из \_ образа PKEY \_ КОМПРЕССЕДБИТСПЕРПИКСЕЛНУМЕРАТОР и PKEY \_ image \_ компресседбитсперпикселденоминатор.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Image.CompressedBitsPerPixel
   shellPKey = PKEY_Image_CompressedBitsPerPixel
   formatID = 364B6FA9-37AB-482A-BE2B-AE02F60D4318
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
