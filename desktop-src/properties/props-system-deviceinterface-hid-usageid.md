# <a name="systemdeviceinterfacehidusageid"></a>System. Девицеинтерфаце. HID. Усажеид

Идентификатор использования HID устройства.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.Hid.UsageId
   shellPKey = PKEY_DeviceInterface_Hid_UsageId
   formatID = CBF38310-4A17-4310-A1EB-247F0B67593B
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
