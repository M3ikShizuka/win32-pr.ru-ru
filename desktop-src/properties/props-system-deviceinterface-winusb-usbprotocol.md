# <a name="systemdeviceinterfacewinusbusbprotocol"></a>System. Девицеинтерфаце. Винусб. Усбпротокол

Значение протокола из первого дескриптора интерфейса USB-устройства.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.WinUsb.UsbProtocol
   shellPKey = PKEY_DeviceInterface_WinUsb_UsbProtocol
   formatID = 95E127B5-79CC-4E83-9C9E-8422187B3E0E
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Byte
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
