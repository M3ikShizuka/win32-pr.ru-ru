# <a name="systemkindtext"></a>System. Киндтекст

Понятная пользователю форма [System. Kind](./props-system-kind.md). Это значение не предназначено для синтаксического анализа программным способом.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.KindText
   shellPKey = PKEY_KindText
   formatID = F04BEF95-C585-4197-A2B7-DF46FDC9EE6D
   propID = 100
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
