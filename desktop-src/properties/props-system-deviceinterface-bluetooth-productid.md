# <a name="systemdeviceinterfacebluetoothproductid"></a>System. Девицеинтерфаце. Bluetooth. ProductId

Идентификатор продукта устройства Bluetooth.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511

```
propertyDescription
   name = System.DeviceInterface.Bluetooth.ProductId
   shellPKey = PKEY_DeviceInterface_Bluetooth_ProductId
   formatID = 2BD67D8B-8BEB-48D5-87E0-6CDA3428040A
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="windows-10-version-1507-windows-81"></a>Windows 10, версия 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.Bluetooth.ProductId
   shellPKey = PKEY_DeviceInterface_Bluetooth_ProductId
   formatID = 2BD67D8B-8BEB-48D5-87E0-6CDA3428040A
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
