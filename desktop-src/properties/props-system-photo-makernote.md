# <a name="systemphotomakernote"></a>System. photo. Макерноте

Механизм расширяемости файла изображений (EXIF), позволяющий производителям камеры предоставлять пользовательские данные. Это свойство не предназначено для отображения в оболочке, но доступно программно для приложений.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.MakerNote
   shellPKey = PKEY_Photo_MakerNote
   formatID = FA303353-B659-4052-85E9-BCAC79549B84
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
