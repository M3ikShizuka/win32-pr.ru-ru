# <a name="systemisincomplete"></a>System. неполны

Указывает, было ли сообщение получено полностью. Это значение используется с некоторыми причинами ошибки.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.IsIncomplete
   shellPKey = PKEY_IsIncomplete
   formatID = 346C8BD1-2E6A-4C45-89A4-61B78E8E700F
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.IsIncomplete
   shellPKey = PKEY_IsIncomplete
   formatID = 346C8BD1-2E6A-4C45-89A4-61B78E8E700F
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = Boolean
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
