# <a name="systemstatusbarviewitemcount"></a>System. Статусбарвиевитемкаунт

Число элементов в представлении и предполагаемый общий размер.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.StatusBarViewItemCount
   shellPKey = PKEY_StatusBarViewItemCount
   formatID = 26DC287C-6E3D-4BD3-B2B0-6A26BA2E346D
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
