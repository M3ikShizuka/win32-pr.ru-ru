# <a name="systemmusicdiscnumber"></a>System. Music. Дискнумбер

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10, версия 1703, Windows 10, версия 1607

```
propertyDescription
   name = System.Music.DiscNumber
   shellPKey = PKEY_Music_DiscNumber
   formatID = 6AFE7437-9BCD-49C7-80FE-4A5C65FA5874
   propID = 104
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
