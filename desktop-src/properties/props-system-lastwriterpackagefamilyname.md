# <a name="systemlastwriterpackagefamilyname"></a>System. Ластвритерпаккажефамилинаме

Пометьте контейнер приложения. Имя семейства пакетов для последнего приложения, для которого нужно изменить содержимое файла.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.LastWriterPackageFamilyName
   shellPKey = PKEY_LastWriterPackageFamilyName
   formatID = 502CFEAB-47EB-459C-B960-E6D8728F7701
   propID = 101
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
