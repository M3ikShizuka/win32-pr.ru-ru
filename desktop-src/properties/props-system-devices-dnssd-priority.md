# <a name="systemdevicesdnssdpriority"></a>System. Devices. Днссд. Priority

Поле приоритета записи SRV. Обычно не используется.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.Devices.Dnssd.Priority
   shellPKey = PKEY_Devices_Dnssd_Priority
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
