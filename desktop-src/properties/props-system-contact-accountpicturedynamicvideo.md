# <a name="systemcontactaccountpicturedynamicvideo"></a>System. Contact. Аккаунтпиктурединамиквидео

Это поток, содержащий данные, необходимые для отображения изображения учетной записи динамического видео контакта.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Contact.AccountPictureDynamicVideo
   shellPKey = PKEY_Contact_AccountPictureDynamicVideo
   formatID = 0B8BB018-2725-4B44-92BA-7933AEB2DDE7
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Stream
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
