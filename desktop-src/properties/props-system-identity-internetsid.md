# <a name="systemidentityinternetsid"></a>System. Identity. Интернетсид

Идентификатор безопасности Интернета для удостоверения.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Identity.InternetSid
   shellPKey = PKEY_Identity_InternetSid
   formatID = 6D6D5D49-265D-4688-9F4E-1FDD33E7CC83
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
