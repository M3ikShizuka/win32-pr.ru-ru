# <a name="systemdeviceinterfaceserialusbproductid"></a>System. Девицеинтерфаце. Serial. Усбпродуктид

Код продукта USB последовательного устройства

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.DeviceInterface.Serial.UsbProductId
   shellPKey = PKEY_DeviceInterface_Serial_UsbProductId
   formatID = 4C6BF15C-4C03-4AAC-91F5-64C0F852BCF4
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
