# <a name="systemsearchrank"></a>System. Search. Rank

Ранг релевантности строки с диапазоном от 0-1000. Большее число означает лучшее совпадение. Только время выполнения запроса; не определено в схеме поиска. Это свойство может быть извлечено, но не доступно для поиска.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.Rank
   shellPKey = PKEY_Search_Rank
   formatID = 49691C90-7E17-101A-A91C-08002B2ECDA9
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
