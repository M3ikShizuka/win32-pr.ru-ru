# <a name="systemcreatoropenwithuioptions"></a>System. Креаторопенвисуиоптионс

Предоставляет параметры, влияющие на поведение элементов управления пользовательского интерфейса, запускающих файл с приложением, указанным в System. Креатораппид.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.CreatorOpenWithUIOptions
   shellPKey = PKEY_CreatorOpenWithUIOptions
   formatID = C2EA046E-033C-4E91-BD5B-D4942F6BBE49
   propID = 3
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
      IsInnate = false
      EnumeratedList
         UseValueForDefault = True
         enum
            name = Hidden
            value = 0
            text = Hidden
            defineToken = CREATOROPENWITHUIOPTION_HIDDEN
         enum
            name = Visible
            value = 1
            text = Visible
            defineToken = CREATOROPENWITHUIOPTION_VISIBLE
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
