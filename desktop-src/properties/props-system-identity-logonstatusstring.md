# <a name="systemidentitylogonstatusstring"></a>System. Identity. Логонстатусстринг

Строка, указывающая состояние входа пользователя в систему удостоверения.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Identity.LogonStatusString
   shellPKey = PKEY_Identity_LogonStatusString
   formatID = F18DEDF3-337F-42C0-9E03-CEE08708A8C3
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
