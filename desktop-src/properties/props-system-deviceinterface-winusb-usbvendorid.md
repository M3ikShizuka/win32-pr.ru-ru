# <a name="systemdeviceinterfacewinusbusbvendorid"></a>System. Девицеинтерфаце. Винусб. Усбвендорид

Идентификатор поставщика из дескриптора USB-устройства USB-устройства.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1

```
propertyDescription
   name = System.DeviceInterface.WinUsb.UsbVendorId
   shellPKey = PKEY_DeviceInterface_WinUsb_UsbVendorId
   formatID = 95E127B5-79CC-4E83-9C9E-8422187B3E0E
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
