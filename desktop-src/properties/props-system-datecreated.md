# <a name="systemdatecreated"></a>System. DateCreated

Дата и время создания элемента в файловой системе, где он находится в данный момент. Это свойство автоматически повышается с помощью файловой системы. Понятное имя службы индексирования — "создать".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.DateCreated
   shellPKey = PKEY_DateCreated
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 15
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.DateCreated
   shellPKey = PKEY_DateCreated
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 15
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
