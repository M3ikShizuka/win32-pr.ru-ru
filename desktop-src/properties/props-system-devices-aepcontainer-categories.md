# <a name="systemdevicesaepcontainercategories"></a>System.Devices.AepContainer.Categories

Категории, в которые входит устройство. Например, принтер, Камера и т. д.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.Devices.AepContainer.Categories
   shellPKey = PKEY_Devices_AepContainer_Categories
   formatID = 0BBA1EDE-7566-4F47-90EC-25FC567CED2A
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
