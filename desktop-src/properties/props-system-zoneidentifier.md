# <a name="systemzoneidentifier"></a>System. Зонеидентифиер

Пометьте веб-зону как значение перечисления УРЛЗОНЕ.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.ZoneIdentifier
   shellPKey = PKEY_ZoneIdentifier
   formatID = 502CFEAB-47EB-459C-B960-E6D8728F7701
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
