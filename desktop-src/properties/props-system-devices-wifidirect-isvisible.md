# <a name="systemdeviceswifidirectisvisible"></a>System. Devices. Вифидирект. Visible

Указывает, Wi-Fi ли текущая видимость прямого устройства

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.IsVisible
   shellPKey = PKEY_Devices_WiFiDirect_IsVisible
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
