# <a name="systemdevicesaepbluetoothleappearancesubcategory"></a>System. Devices. АЕП. Bluetooth. Le. Appearance. Подкатегория

Внешний вид устройства Bluetooth LE.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Le.Appearance.Subcategory
   shellPKey = PKEY_Devices_Aep_Bluetooth_Le_Appearance_Subcategory
   formatID = 995EF0B0-7EB3-4A8B-B9CE-068BB3F4AF69
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
