# <a name="systemphotofocalplanexresolution"></a>System. photo. Фокалпланексресолутион

Указывает число пикселей в области изображения (направление по оси X) на Фокалпланересолутионунит в плоскости, направляющей камеру. Вычислено на основе \_ \_ фокалпланексресолутионнумератор и \_ фокалпланексресолутионденоминатор фото на PKEY \_ .

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.FocalPlaneXResolution
   shellPKey = PKEY_Photo_FocalPlaneXResolution
   formatID = CFC08D97-C6F7-4484-89DD-EBEF4356FE76
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
