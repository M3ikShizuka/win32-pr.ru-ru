# <a name="systemmediumkeywords"></a>System. Медиумкэйвордс

Ключевые слова среднего уровня достоверности для элемента.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1

```
propertyDescription
   name = System.MediumKeywords
   shellPKey = PKEY_MediumKeywords
   formatID = F29F85E0-4FF9-1068-AB91-08002B27B3D9
   propID = 26
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
