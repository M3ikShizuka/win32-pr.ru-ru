# <a name="systemmusicgenre"></a>System. Music. Жанр

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Music.Genre
   shellPKey = PKEY_Music_Genre
   formatID = 56A3372E-CE9C-11D2-9F0E-006097C686F6
   propID = 11
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.

Обратите внимание, что некоторые форматы файлов, например файл .mp3, поддерживают только один жанр. Другие, такие как WMA-файлы, поддерживают несколько жанров. Если указать несколько жанров для формата файла с одним жанром, будет распознан только первый жанр в списке.
