# <a name="systemdevicesaepcontainersupportsaudio"></a>System.Devices.AepContainer.SupportsAudio

Указывает, поддерживает ли контейнер конечной точки сопоставления устройств приведение звука

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.Devices.AepContainer.SupportsAudio
   shellPKey = PKEY_Devices_AepContainer_SupportsAudio
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
