# <a name="systemdeviceslaunchdevicestagefromexplorer"></a>System. Devices. Лаунчдевицестажефромексплорер

Указывает, следует ли запустить этап устройства.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Devices.LaunchDeviceStageFromExplorer
   shellPKey = PKEY_Devices_LaunchDeviceStageFromExplorer
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 77
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
