# <a name="systemsupplementalalbumid"></a>System. дополнительное. Албумид

Содержит идентификаторы альбомов, членом которых является элемент. Можно использовать в сочетании с элементом альбома в интерфейсах API индексатора содержимого, чтобы уведомить другие приложения о фотоальбомах, созданных пользователем или уже созданными приложениями.

## <a name="windows-10-version-1703"></a>Windows 10 версии 1703

```
propertyDescription
   name = System.Supplemental.AlbumID
   shellPKey = PKEY_Supplemental_AlbumID
   formatID = 0C73B141-39D6-4653-A683-CAB291EAF95B
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
