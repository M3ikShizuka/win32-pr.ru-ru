# <a name="systemsecurityallowedenterprisedataprotectionidentities"></a>System. Security. Алловедентерприседатапротектионидентитиес

Параметры шифрования

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10, версия 1703, Windows 10, версия 1607

```
propertyDescription
   name = System.Security.AllowedEnterpriseDataProtectionIdentities
   shellPKey = PKEY_Security_AllowedEnterpriseDataProtectionIdentities
   formatID = 38D43380-D418-4830-84D5-46935A81C5C6
   propID = 32
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Multivalue String
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
