# <a name="systemlanguage"></a>System. Language

Основной язык файла, особенно если этот файл является документом.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1

```
propertyDescription
   name = System.Language
   shellPKey = PKEY_Language
   formatID = D5CDD502-2E9C-101B-9397-08002B2CF9AE
   propID = 28
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = String
```

## <a name="windows-8-windows-7-windows-vista"></a>Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Language
   shellPKey = PKEY_Language
   formatID = D5CDD502-2E9C-101B-9397-08002B2CF9AE
   propID = 28
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
