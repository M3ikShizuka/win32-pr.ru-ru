# <a name="systemdevicesclassguid"></a>System.Devices.ClassGuid

GUID класса устройства.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.Devices.ClassGuid
   shellPKey = PKEY_Devices_ClassGuid
   formatID = A45C254E-DF1C-4EFD-8020-67D146A850E0
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
