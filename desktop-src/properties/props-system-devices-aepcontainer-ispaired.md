# <a name="systemdevicesaepcontainerispaired"></a>System.Devices.AepContainer.IsPaired

Связан ли одна из конечных точек ассоциации дочерних устройств с системой или нет.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.Devices.AepContainer.IsPaired
   shellPKey = PKEY_Devices_AepContainer_IsPaired
   formatID = 0BBA1EDE-7566-4F47-90EC-25FC567CED2A
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
