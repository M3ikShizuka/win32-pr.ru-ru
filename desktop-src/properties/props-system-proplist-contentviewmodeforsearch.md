# <a name="systemproplistcontentviewmodeforsearch"></a>System. Проплист. Контентвиевмодефорсеарч

Определяет список свойств, отображаемых в режиме просмотра содержимого элемента в контексте поиска.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.PropList.ContentViewModeForSearch
   shellPKey = PKEY_PropList_ContentViewModeForSearch
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 14
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
