# <a name="systemphotofnumber"></a>System. photo. Фнумбер

Значение Фнумбер при создании фотографии, считанное из данных файла изображения (EXIF). Это свойство вычисляется из [System. photo. фнумбернумератор](./props-system-photo-fnumbernumerator.md) и [System. photo. фнумберденоминатор](./props-system-photo-fnumberdenominator.md).

Ниже приведены возможные значения, взятые из спецификации EXIF 2,2.

-   1
-   1.4
-   2
-   2.8
-   4
-   5.6
-   8
-   11
-   16
-   22
-   32

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Photo.FNumber
   shellPKey = PKEY_Photo_FNumber
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 33437
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Photo.FNumber
   shellPKey = PKEY_Photo_FNumber
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 33437
   SearchInfo
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
