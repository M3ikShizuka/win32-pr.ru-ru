# <a name="systemsupplementalresourceid"></a>System. дополнительный. ResourceId

Содержит идентификатор элемента в удаленной службе синхронизации. Используется для сравнения файла в системе с теми, которые доступны в облаке.

## <a name="windows-10-version-1703"></a>Windows 10 версии 1703

```
propertyDescription
   name = System.Supplemental.ResourceId
   shellPKey = PKEY_Supplemental_ResourceId
   formatID = 0C73B141-39D6-4653-A683-CAB291EAF95B
   propID = 3
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
