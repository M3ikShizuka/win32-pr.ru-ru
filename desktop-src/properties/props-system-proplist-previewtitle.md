# <a name="systemproplistpreviewtitle"></a>System. Проплист. PreviewTitle

Одно или два свойства, которые должны отображаться в разделе заголовка области просмотра. Необязательное второе свойство отображается в виде подзаголовка. Зарегистрируйтесь в регвалуе "PreviewTitle".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.PreviewTitle
   shellPKey = PKEY_PropList_PreviewTitle
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
