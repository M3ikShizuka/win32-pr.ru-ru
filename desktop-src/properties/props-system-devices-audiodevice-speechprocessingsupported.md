# <a name="systemdevicesaudiodevicespeechprocessingsupported"></a>System.Devices.AudioDevice.SpeechProcessingSupported

Поддержка речевого режима для звукового устройства. Если вариант \_ имеет значение true, устройство поддерживает речевой режим.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.Devices.AudioDevice.SpeechProcessingSupported
   shellPKey = PKEY_Devices_AudioDevice_SpeechProcessingSupported
   formatID = FB1DE864-E06D-47F4-82A6-8A0AEF44493C
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
