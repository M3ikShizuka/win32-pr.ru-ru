# <a name="systemidentityqualifiedusername"></a>System. Identity. Куалифиедусернаме

Полное имя пользователя удостоверения.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Identity.QualifiedUserName
   shellPKey = PKEY_Identity_QualifiedUserName
   formatID = DA520E51-F4E9-4739-AC82-02E0A95C9030
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
