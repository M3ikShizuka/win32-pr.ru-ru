# <a name="systemphotoexposureindex"></a>System. photo. Експосуреиндекс

Указывает индекс экспозиции, выбранный на камере или устройстве ввода на момент съемки. Вычислено на основе \_ \_ експосуреиндекснумератор и \_ експосуреиндексденоминатор фото на PKEY \_ .

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.ExposureIndex
   shellPKey = PKEY_Photo_ExposureIndex
   formatID = 967B5AF8-995A-46ED-9E11-35B3C5B9782D
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
