# <a name="systemdrmisprotected"></a>System. DRM. для защиты

Указывает, защищен ли файл с помощью цифрового Rights Management (DRM).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.DRM.IsProtected
   shellPKey = PKEY_DRM_IsProtected
   formatID = AEAC19E4-89AE-4508-B9B7-BB867ABEE2ED
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.DRM.IsProtected
   shellPKey = PKEY_DRM_IsProtected
   formatID = AEAC19E4-89AE-4508-B9B7-BB867ABEE2ED
   propID = 2
   SearchInfo
      IsColumn = true
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
