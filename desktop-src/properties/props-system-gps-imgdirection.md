# <a name="systemgpsimgdirection"></a>System. GPS. Имгдиректион

Указывает направление изображения при его записи. Вычисляется от PKEY \_ GPS \_ ИМГДИРЕКТИОННУМЕРАТОР и PKEY \_ GPS \_ имгдиректионденоминатор.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.GPS.ImgDirection
   shellPKey = PKEY_GPS_ImgDirection
   formatID = 16473C91-D017-4ED9-BA4D-B6BAA55DBCF8
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
