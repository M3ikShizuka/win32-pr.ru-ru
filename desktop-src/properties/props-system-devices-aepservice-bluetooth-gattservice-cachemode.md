# <a name="systemdevicesaepservicebluetoothgattservicecachemode"></a>System. Devices. Аепсервице. Bluetooth. Гаттсервице. CacheMode

задает Bluetooth режим кэша Gatt для запроса.

## <a name="windows-10-version-1511"></a>Windows 10 версии 1511

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.GattService.CacheMode
   shellPKey = PKEY_Devices_AepService_Bluetooth_GattService_CacheMode
   formatID = 9744311E-7951-4B2E-B6F0-ECB293CAC119
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt16
      IsInnate = true
```

## <a name="remarks"></a>Remarks

Значения PKEY определены в списке PKEY. h.
