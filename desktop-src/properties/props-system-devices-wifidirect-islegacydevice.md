# <a name="systemdeviceswifidirectislegacydevice"></a>System. Devices. Вифидирект. Ислегацидевице

Указывает, является ли устройство Wi-Fiным устройством устаревшей

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, версия 1703, Windows 10, версия 1607, Windows 10, версия 1511, Windows 10, версия 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.IsLegacyDevice
   shellPKey = PKEY_Devices_WiFiDirect_IsLegacyDevice
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 7
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Комментарии

Значения PKEY определены в списке PKEY. h.
