# <a name="directdraw"></a>Произведен

В этом разделе содержатся сведения о программировании с помощью компонента DirectDraw интерфейса прикладного программирования (API) DirectX.

DirectDraw больше не рекомендуется использовать. С выпуском Direct3D 9,0 вся двухмерная функциональность содержится в Direct3D, связанных вспомогательных функциях в D3DX и технологии DirectX 11 Direct2D. Однако справочная документация по DirectDraw по-прежнему доступна в этом разделе.

## <a name="in-this-section"></a>в этом разделе

Этот раздел содержит [справочную](directdraw-reference.md) документацию по DirectDraw.
