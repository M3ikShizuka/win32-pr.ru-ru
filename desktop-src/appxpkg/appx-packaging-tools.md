# <a name="packaging-and-deployment-tools-and-powershell-cmdlets"></a>Средства упаковки и развертывания и командлеты PowerShell

используйте эти средства и командлеты для создания, развертывания и инвентаризации пакетов приложений для Windows приложений.

## <a name="app-packaging-tools-and-guidance"></a>Средства и руководства по упаковке приложений

Узнайте, как упаковать приложение, создать тестовый сертификат и подписать приложение с помощью командной строки и PowerShell. См. также:

-   [Создание пакета приложений с помощью средства MakeAppx.exe](/windows/msix/package/create-app-package-with-makeappx-tool)
-   [Создание сертификата для подписания пакета](/windows/msix/package/create-certificate-package-signing)
-   [Подписание пакета приложения с помощью SignTool](/windows/msix/package/sign-app-package-using-signtool)

кроме того, можно использовать Visual Studio для создания и подписи пакета приложения. Дополнительные сведения см. в статье [Упаковка приложения UWP с помощью Visual Studio](/windows/msix/package/packaging-uwp-apps).

## <a name="powershell-cmdlets"></a>Командлеты PowerShell

используйте Windows PowerShell для установки, инвентаризации и удаления бизнес-Windows приложений, которые не подписаны на хранение. Дополнительные сведения см. в разделе [командлеты модуля appx](/powershell/module/appx/index?view=win10-ps).
