# <a name="about-windows-connect-now"></a>О Windows Connect Now

Windows Connect Now (WCN) предоставляет простой и безопасный механизм для точек доступа к сети и устройств (например, принтеров, камер и компьютеров) для подключения и настройки exchange. Этот API является Wi-Fi реализацией корпорацией Майкрософт протокола/Ви-фи Protected Configuration (WPS), которая была создана Wi-Fi Alliance в качестве решения для домашних сетей и малых предприятий. Эта технология не предназначена для корпоративных сценариев.

> [!Note]  
> Имя спецификации изменилось между версией 1.0 h и версией 2. Спецификация версии 1.0 h называлась Wi-Fi защищенная установка (WPS). Начиная с спецификации версии 2, спецификация называется Wi-Fi Simple Configuration (WSC). В нашей документации термины WPS и WSC используются взаимозаменяемыми, если не указано иное.

 

Windows Connect Now позволяет приложениям искать устройства с поддержкой WCN с помощью [API обнаружения функций](/previous-versions/windows/desktop/fundisc/fd-portal). Область поиска можно сократить до определенного идентификатора SSID, состояния, категории или даже расширить для включения всех устройств с поддержкой WCN. После обнаружения устройств API WCN позволяет обмениваться данными с устройством с поддержкой WCN для упрощения настройки или подключения.
