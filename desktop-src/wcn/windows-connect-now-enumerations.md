# <a name="windows-connect-now-enumerations"></a>Перечисления Windows Connect Now

Windows Connect теперь определяет следующие типы перечислений.



| Перечисление                                                                                             | Описание                                                                                                                   |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**\_тип атрибута \_ WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_attribute_type)                                                      | Определяет различные типы буферов атрибутов, определенные для Wi-Fi защищенной установки.<br/>                                      |
| [**\_тип пароля \_ WCN**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_password_type)                                                        | Определяет проверку подлинности, которая будет использоваться в сеансе WPS. <br/>                                                    |
| [**\_состояние сеанса \_ WCN**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_session_status)                                                      | Определяет состояние сеанса WPS.<br/>                                                                               |
| [**\_ \_ \_ состояние СОПОСТАВЛЕНИЯ типа значения \_ WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_association_state)                        | Определяет возможные состояния взаимосвязей беспроводной станции во время запроса на обнаружение.<br/>                          |
| [**\_тип значения \_ WCN \_ Boolean**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_boolean)                                             | Определяет значения, используемые для представления условий true/false, возникающих во время установки устройства и ассоциации.<br/>            |
| [**Тип \_ значения \_ \_ проверки подлинности типа WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_authentication_type)                    | Определяет методы проверки подлинности, поддерживаемые зарегистрированным объектом (точкой доступа или станцией).<br/>                            |
| [**\_ \_ \_ методы конфигурации типа значения \_ WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_config_methods)                              | Определяет методы конфигурации, поддерживаемые заявителем или регистратором.<br/>                                          |
| [**\_ \_ \_ Ошибка конфигурации типа значения \_ WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_configuration_error)                    | Определяет возможные значения ошибок, возвращаемые на устройство при попытке настроить его и связать с беспроводной локальной сети.<br/> |
| [**\_тип значения \_ WCN \_ \_ идентификатор пароля \_ устройства**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_device_password_id)                     | Определяет возможные значения пароля.<br/>                                                                              |
| [**\_тип значения для типа \_ \_ шифрования \_ WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_encryption_type)                            | Определяет поддерживаемые типы шифрования WLAN.<br/>                                                                       |
| [**\_ \_ радиочастотные типы значений \_ WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_rf_bands)                                          | Определяет частоту радиоволн, на которой подписчик отправляет запросы на обнаружение.<br/>                                    |
| [**\_ \_ версия типа значения \_ WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_version)                                             | Определяет поддерживаемую версию Wi-Fi Protected Setup (WPS).<br/>                                                      |
| [**\_тип значения WCN — \_ \_ \_ \_ защищенное \_ \_ состояние установки Wi Fi**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_wi_fi_protected_setup_state) | Определяет значения, которые указывают, настроено ли устройство.<br/>                                                            |
