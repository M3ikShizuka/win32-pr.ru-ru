# <a name="windows-connect-now-interfaces"></a>Windows Connect Now Интерфейс

[Windows Connect Now API](portal.md) (вкнапи) определяет и использует следующие интерфейсы.



| Интерфейс                                      | Описание                                                                                                |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**ивкнконнектнотифи**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcnconnectnotify) | получение уведомления об успешном или неуспешном выполнении при завершении сеанса Windows Connect Now Connect.<br/> |
| [**ивкндевице**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcndevice)               | Настраивает устройство и инициирует сеанс.<br/>                                                |
