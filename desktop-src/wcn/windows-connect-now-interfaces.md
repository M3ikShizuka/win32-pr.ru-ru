# <a name="windows-connect-now-interfaces"></a>Интерфейсы Windows Connect Now

[API Windows Connect Now](portal.md) (вкнапи) определяет и использует следующие интерфейсы.



| Интерфейс                                      | Описание                                                                                                |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**ивкнконнектнотифи**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcnconnectnotify) | Получение уведомления об успешном или неуспешном выполнении при завершении сеанса подключения Windows Connect.<br/> |
| [**ивкндевице**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcndevice)               | Настраивает устройство и инициирует сеанс.<br/>                                                |
