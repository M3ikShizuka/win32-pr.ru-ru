# <a name="strict-type-checking"></a>СТРОГОе проверка типов

Файл заголовка Windows. h содержит определения, макросы и структуры, помогающие писать исходный код, переносимый между версиями Windows. Некоторые из этих функций включаются при определении символа при создании приложения. В следующих разделах объясняются преимущества использования функциональных возможностей, а также их влияние на написание кода.

-   [Включение параметра](enabling-strict.md)
-   [Отключение](disabling-strict.md)
-   [ЖЕСТКОЕ соответствие](strict-compliance.md)
