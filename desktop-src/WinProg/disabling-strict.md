# <a name="disabling-strict"></a>Отключение

Чтобы отключить проверку **строгого** типа, определите имя символа **без \_ строгого** указания. В Visual C/C++ можно также указать это определение в командной строке или в файле makefile, указав **/дно \_** в качестве параметра компилятора.

чтобы **не задавать никаких \_ мер** для каждого файла, вставьте инструкцию **\# define** перед включением Windows. h:


```C++
#define NO_STRICT
#include <windows.h>
```



Для получения наилучших результатов следует также установить уровень предупреждений для сообщений об ошибках по меньшей мере в **/W3**. это всегда рекомендуется для приложений Windows, так как практика написания кода, вызывающая предупреждение (например, передача неверного количества параметров), обычно вызывает неустранимую ошибку во время выполнения, если она не исправлена.
