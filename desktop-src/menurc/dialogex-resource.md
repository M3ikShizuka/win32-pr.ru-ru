# <a name="dialogex-resource"></a>Ресурс ДИАЛОЖЕКС

Определяет диалоговое окно. Оператор определяет расположение и размеры диалогового окна на экране, а также стиль диалогового окна. Он также определяет следующее:

-   Идентификаторы справки в диалоговом окне, а также в элементах управления в диалоговом окне.
-   Используйте инструкцию [**операторе EXSTYLE**](exstyle-statement.md) для самого диалогового окна, а также для элементов управления в диалоговом окне.
-   Параметры веса шрифта и курсива для шрифта, используемого в диалоговом окне.
-   Данные, относящиеся к элементам управления, в диалоговом окне.
-   Использование предопределенных имен системных классов **бедит**, **иедит** и **хедит** .

``` syntax
nameID DIALOGEX x, y, width, height [ , helpID] [optional-statements]  {control-statements}
```

## <a name="remarks"></a>Remarks

Допустимые операции, которые могут содержаться в любом из числовых выражений в инструкциях **диаложекс** , приведены ниже.

-   Добавить ("+")
-   Вычитание ("-")
-   Унарный минус ("-")
-   Унарный оператор NOT (' ~ ')
-   И (' & ')
-   ИЛИ (' \| ')

Текст ресурса состоит из универсальных, статических операторов, кнопок и элементов управления Edit. Хотя каждая из этих семейств инструкций использует другой синтаксис для определения конкретных функций элементов управления, все они совместно используют общий синтаксис для определения положения, размера, расширенных стилей, идентификационного номера справки и данных, относящихся к конкретному элементу управления. Дополнительные сведения см. в разделе [Общие параметры управления](common-control-parameters.md).
