# <a name="wm_menugetobject-message"></a>\_Сообщение МЕНУЖЕТОБЖЕКТ WM

Посылается владельцу меню перетаскивания, когда курсор мыши попадает в пункт меню или перемещается от центра элемента к верхнему или нижнему краю элемента.


```C++
#define WM_MENUGETOBJECT                0x0124
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**менужетобжектинфо**](/windows/win32/api/winuser/ns-winuser-menugetobjectinfo) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Приложение должно вернуть одно из следующих значений.



| Возвращаемый код и значение                                                                                                                                                | Описание                                                                                                            |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**МНГО \_ Ошибка**</dt> <dt>0x00000001</dt> </dl>     | В элементе **пвобж** элемента [**менужетобжектинфо**](/windows/win32/api/winuser/ns-winuser-menugetobjectinfo) был возвращен указатель интерфейса.<br/> |
| <dl> <dt>**МНГО \_ Интерфейс неинтерфейса**</dt> <dt>0x00000000</dt> </dl> | Интерфейс не поддерживается.<br/>                                                                             |
