# <a name="displaying-progress-messages"></a>Отображение сообщений о ходе выполнения

По умолчанию версия-кандидат компилируется без отображения сообщений о ходе выполнения. Однако можно указать, что RC должен отображать сообщения о ходе выполнения. Для этого используйте параметр **/v** .

В следующем примере RC сообщает о ходе выполнения, так как компилирует файл определения ресурса Sample. RC и создает скомпилированный файл ресурсов Sample. res:

**Версия-кандидат/v Sample. RC**
