# <a name="acceltableentry-structure"></a>Структура АКЦЕЛТАБЛИНТРИ

Описывает данные в отдельном ресурсе таблицы сочетаний клавиш. Приведенное здесь определение структуры предназначено только для объяснения. он отсутствует в любом стандартном файле заголовка.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct {
  WORD fFlags;
  WORD wAnsi;
  WORD wId;
  WORD padding;
} ACCELTABLEENTRY;
```

## <a name="remarks"></a>Remarks

Структура **акцелтаблинтри** повторяется для всех записей таблицы сочетаний клавиш в ресурсе. Последняя запись в таблице помечается значением 0x0080.

Количество элементов в таблице можно вычислить, если длина ресурса делится на восемь. Затем приложение может случайно получить доступ к отдельным записям фиксированной длины.
