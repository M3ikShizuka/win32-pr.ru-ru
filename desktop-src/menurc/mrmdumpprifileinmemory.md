# <a name="mrmdumpprifileinmemory-function"></a>Функция Мрмдумпприфилеинмемори

\[Некоторые сведения относятся к предварительно выпущенному продукту, который может быть значительно изменен перед коммерческой выпуском. Майкрософт не дает никаких гарантий, явных или подразумеваемых, в отношении предоставленной здесь информации.\]

Выводит файл PRI (двоичный) в его эквивалент XML (как данные в памяти), чтобы сделать его более удобным для чтения. Функция выделяет память и возвращает указатель на эту память в *аутпутксмлдата*. Вызовите [**мрмфримемори**](mrmfreememory.md) с тем же указателем, чтобы освободить эту память. Дополнительные сведения и пошаговые руководства по использованию этих API см. в разделе [пакетные API индексирования ресурсов (PRI) и пользовательские системы сборки](/windows/uwp/app-resources/pri-apis-custom-build-systems).

## <a name="syntax"></a>Синтаксис


```C++
HRESULT HRESULT MrmDumpPriFileInMemory(
  _In_     PCWSTR      indexFileName,
  _In_opt_ PCWSTR      schemaPriFile,
  _In_     MrmDumpType dumpType,
  _Out_    BYTE        **outputXmlData,
  _Out_    ULONG       *outputXmlSize
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*индексфиленаме* \[ окне\]
</dt> <dd>

Тип: **пквстр**

Полный путь к файлу PRI. Это PRI-файл, который будет сохранен в XML.

</dd> <dt>

*счемаприфиле* \[ в необязательное\]
</dt> <dd>

Тип: **пквстр**

Необязательный полный путь к файлу схемы (или к PRI-файлу, представляющему схему; см. раздел Примечания).

</dd> <dt>

*думптипе* \[ окне\]
</dt> <dd>

Тип: **[ **мрмдумптипе**](mrmdumptype.md)**

Указывает, насколько подробным должен быть дамп XML, или следует ли создавать дамп схемы.

</dd> <dt>

*аутпутксмлдата* \[ заполняет\]
</dt> <dd>

Тип: **Byte \* \***

Адрес указателя на байт. Функция выделяет память и возвращает указатель на эту память в *аутпутксмлдата*. Вызовите [**мрмфримемори**](mrmfreememory.md) с помощью указателя на Byte, чтобы освободить эту память.

</dd> <dt>

*аутпутксмлсизе* \[ заполняет\]
</dt> <dd>

Тип: **ulong \** _

Адрес ULONG. В _outputXmlSize * функция возвращает размер выделенной памяти, на которую указывает *аутпутксмлдата*.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

\_ОК, если функция прошла успешно, в противном случае — другое значение. Чтобы определить успешность или сбой, используйте макросы Success () или FAILed () (определенные в файле Winerror. h).

## <a name="remarks"></a>Комментарии

Пакет ресурсов без схемы — это тот, который был создан с помощью аргумента [**мрмпаккагингоптионсомитсчемафромресаурцепаккс**](mrmpackagingoptions.md) , переданного в [**мрмкреатересаурцефиле**](mrmcreateresourcefile.md) или [**мрмкреатересаурцефилеинмемори**](mrmcreateresourcefileinmemory.md) (или с параметром *omitSchemaFromResourcePacks* в файле конфигурации PRI). Чтобы создать дамп пакета ресурсов без схемы, передайте путь к основным данным пакета PRI в качестве аргумента для параметра *счемаприфиле* .
