# <a name="preprocessor-operators"></a>Операторы препроцессора

Версия-кандидат поддерживает использование стандартных операторов предварительной обработки C в определениях макросов. Эти операторы описаны в следующей таблице.



| Оператор | Описание                                                 |
|----------|-------------------------------------------------------------|
| \#       | Заключает аргумент в кавычки.                            |
| \#@      | Заключает аргумент в одинарные кавычки.                     |
| \#\#     | Сцепляет токены, используемые в качестве аргументов, для формирования других маркеров. |
