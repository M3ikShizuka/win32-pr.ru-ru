# <a name="predefined-macros"></a>Предустановленный макрос

Версия-кандидат не поддерживает предварительно определенные макросы ANSI C (**\_ \_ Дата \_ \_**, **\_ \_ файл \_ \_**, **\_ \_ строка \_ \_**, **\_ \_ STDC \_ \_**, **\_ \_ время \_ , метка \_** **\_ \_ времени). \_ \_** Поэтому эти макросы нельзя включить в заголовочные файлы, которые будут включены в скрипт ресурсов.

Версия-кандидат определяет \_ вызов RC, что позволяет условно компилировать части файлов заголовков в зависимости от того, является ли компилятор компилятором C или компилятором RC. Это важно, поскольку компилятор RC поддерживает только подмножество инструкций, поддерживаемых компилятором C.

Чтобы условно скомпилировать код с помощью компилятора RC, заключите код, который компилятор RC не может скомпилировать с помощью \# метода IFNDEF RC \_ Invoked и **\# endif**.

Следующий пример взят из примеров пакета SDK. В нем показано, как создать файл заголовка, который может быть скомпилирован условно.

``` syntax
#ifndef RC_INVOKED
#pragma message("Including CntrOutl.H from " __FILE__)
#endif
```
