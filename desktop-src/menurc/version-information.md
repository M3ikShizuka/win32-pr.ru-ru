# <a name="version-information"></a>Сведения о версии

Сведения о версии облегчают приложениям правильную установку файлов и позволяют программам программы установки анализировать уже установленные файлы. Ресурс сведения о версии содержит номер версии файла, требуемую операционную систему и имя исходного файла.

### <a name="version-information-functions"></a>Функции сведений о версии



| Имя                                                         | Описание                                                                                                                                                                                                                                                                                                                                                           |
|--------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetFileVersionInfo**](/windows/desktop/api/Winver/nf-winver-getfileversioninfoa)             | Извлекает сведения о версии для указанного файла. <br/>                                                                                                                                                                                                                                                                                                     |
| [**жетфилеверсионинфоекс**](/windows/desktop/api/Winver/nf-winver-getfileversioninfoexa)         | Извлекает сведения о версии для указанного файла.<br/>                                                                                                                                                                                                                                                                                                      |
| [**Ошибка getfileversioninfosize**](/windows/desktop/api/Winver/nf-winver-getfileversioninfosizea)     | Определяет, может ли операционная система извлекать сведения о версии для указанного файла. Если доступна информация о версии, [**Ошибка getfileversioninfosize**](/windows/desktop/api/Winver/nf-winver-getfileversioninfosizea) возвращает размер этой информации в байтах. <br/>                                                                                                             |
| [**жетфилеверсионинфосизикс**](/windows/desktop/api/Winver/nf-winver-getfileversioninfosizeexa) | Определяет, может ли операционная система извлекать сведения о версии для указанного файла. Если доступна информация о версии, [**жетфилеверсионинфосизикс**](/windows/desktop/api/Winver/nf-winver-getfileversioninfosizeexa) возвращает размер этой информации в байтах.<br/>                                                                                                          |
| [**верфиндфиле**](/windows/desktop/api/Winver/nf-winver-verfindfilea)                           | Определяет место установки файла в зависимости от того, найдена ли в системе другая версия файла. Значения [**верфиндфиле**](/windows/desktop/api/Winver/nf-winver-verfindfilea) , возвращаемые в указанных буферах, используются при последующем вызове функции [**веринсталлфиле**](/windows/desktop/api/Winver/nf-winver-verinstallfilea) . <br/>                                                                          |
| [**веринсталлфиле**](/windows/desktop/api/Winver/nf-winver-verinstallfilea)                     | Устанавливает указанный файл на основе сведений, возвращаемых функцией [**верфиндфиле**](/windows/desktop/api/Winver/nf-winver-verfindfilea) . [**Веринсталлфиле**](/windows/desktop/api/Winver/nf-winver-verinstallfilea) распаковывает файл, при необходимости назначает уникальное имя файла и проверяет наличие ошибок, таких как устаревшие файлы. <br/>                                                                                   |
| [**верлангуаженаме**](/windows/desktop/api/Winver/nf-winver-verlanguagenamea)                   | Извлекает строку описания для языка, связанного с указанным двоичным идентификатором языка Майкрософт.<br/>                                                                                                                                                                                                                                          |
| [**VerQueryValue**](/windows/desktop/api/Winver/nf-winver-verqueryvaluea)                       | Извлекает указанные сведения о версии из указанного ресурса сведений о версии. Чтобы получить соответствующий ресурс, перед вызовом [**веркуеривалуе**](/windows/desktop/api/Winver/nf-winver-verqueryvaluea)необходимо сначала вызвать функцию [**Ошибка getfileversioninfosize**](/windows/desktop/api/Winver/nf-winver-getfileversioninfosizea) , а затем функцию [**GetFileVersionInfo**](/windows/desktop/api/Winver/nf-winver-getfileversioninfoa) . <br/> |



 

### <a name="version-information-structures"></a>Структуры сведений о версии



| Имя                                          | Описание                                                                                                                                                                                                                      |
|-----------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Строка**](string-str.md)                  | Описывает организацию данных в ресурсе файловой версии. Он содержит строку, описывающую конкретный аспект файла, например версию файла, уведомления об авторских правах или его товарные знаки.<br/>                |
| [**стрингфилеинфо**](stringfileinfo.md)      | Описывает организацию данных в ресурсе файловой версии. Он содержит сведения о версии, которые могут отображаться для определенного языка и кодовой страницы.<br/>                                                           |
| [**StringTable**](stringtable.md)            | Описывает организацию данных в ресурсе файловой версии. Он содержит сведения о языке и форматировании кодовой страницы для строк, указанных **дочерним** элементом. Кодовая страница — это упорядоченная кодировка.<br/> |
| [**Var**](var-str.md)                        | Описывает организацию данных в ресурсе файловой версии. Обычно он содержит список пар языков и идентификаторов кодовых страниц, поддерживаемых версией приложения или библиотеки DLL.<br/>                             |
| [**варфилеинфо**](varfileinfo.md)            | Описывает организацию данных в ресурсе файловой версии. Он содержит сведения о версии, не зависящие от конкретного языка и сочетания кодовой страницы.<br/>                                                        |
| [**VS \_ FIXEDFILEINFO**](/windows/win32/api/verrsrc/ns-verrsrc-vs_fixedfileinfo) | Содержит сведения о версии файла. Эта информация не зависит от языка и кодовой страницы. <br/>                                                                                                                   |
| [**VS \_ versionInfo**](vs-versioninfo.md)     | Описывает организацию данных в ресурсе файловой версии. Это корневая структура, которая содержит все остальные структуры сведений о версии файла.<br/>                                                                    |
