# <a name="icon-control"></a>ЗНАЧОК элемента управления

Определяет элемент управления "значок". Этот элемент управления является значком, отображаемым в диалоговом окне.

Оператор **Icon** , который можно использовать только в операторе [**диаложекс**](dialogex-resource.md) , определяет идентификатор значка ресурса, идентификатор элемента управления Icon, расположение и атрибуты элемента управления.

``` syntax
ICON text, id, x, y [, width, height, style [, extended-style]]
```

<dl> <dt>

<span id="text"></span><span id="TEXT"></span>*полнотекстовым*
</dt> <dd>

Имя значка (а не имени файла), определенного в файле ресурсов.

</dd> <dt>

<span id="width"></span><span id="WIDTH"></span>*Ширина*
</dt> <dd>

Это значение игнорируется и должно быть равно нулю.

</dd> <dt>

<span id="height"></span><span id="HEIGHT"></span>*равно*
</dt> <dd>

Это значение игнорируется и должно быть равно нулю.

</dd> <dt>

<span id="style"></span><span id="STYLE"></span>*стиль*
</dt> <dd>

Стиль элемента управления. Это необязательный параметр. Единственным значением, которое можно указать, является стиль **\_ значка SS** . Это стиль по умолчанию, определяющий, задан ли этот параметр.

</dd> </dl>

Дополнительные сведения об общем синтаксисе инструкции Control см. в разделе [Общие параметры управления](common-control-parameters.md).

## <a name="examples"></a>Примеры

В этом примере определяется элемент управления значком, идентификатором значка которого является 901, а имя — Микон:

``` syntax
ICON "myicon", 901, 30, 30
```
