# <a name="mrmcreateresourceindexerfrompreviouspridata-function"></a>Функция Мрмкреатересаурцеиндексерфромпревиауспридата

\[Некоторые сведения относятся к предварительно выпущенному продукту, который может быть значительно изменен перед коммерческой выпуском. Майкрософт не дает никаких гарантий, явных или подразумеваемых, в отношении предоставленной здесь информации.\]

Создает индексатор ресурсов из данных PRI, созданных при предыдущем вызове метода [**мрмкреатересаурцефилеинмемори**](mrmcreateresourcefileinmemory.md). Дополнительные сведения и пошаговые руководства по использованию этих API см. в разделе [пакетные API индексирования ресурсов (PRI) и пользовательские системы сборки](/windows/uwp/app-resources/pri-apis-custom-build-systems).

## <a name="syntax"></a>Синтаксис


```C++
HRESULT HRESULT MrmCreateResourceIndexerFromPreviousPriData (
  _In_     PCWSTR                   projectRoot,
  _In_     MrmPlatformVersion       platformVersion,
  _In_opt_ PCWSTR                   defaultQualifiers,
  _In_     BYTE                     *priData,
  _In_     ULONG                    priSize,
  _Inout_  MrmResourceIndexerHandle *indexer
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*прожектрут* \[ окне\]
</dt> <dd>

Тип: **пквстр**

Корневой каталог проекта приложения UWP, для которого будут создаваться файлы PRI. Иными словами, это путь к файлам ресурсов этого приложения. Это можно указать таким образом, чтобы можно было указать пути относительно корня в последующих вызовах API для того же индексатора ресурсов.

</dd> <dt>

*платформверсион* \[ окне\]
</dt> <dd>

Тип: **[ **мрмплатформверсион**](mrmplatformversion.md)**

Версия целевой платформы для индексатора ресурсов.

</dd> <dt>

*дефаулткуалифиерс* \[ в необязательное\]
</dt> <dd>

Тип: **пквстр**

Список квалификаторов ресурсов по умолчанию. Например, L "язык-EN-US \_ Scale-100 \_ контраст — Standard"

</dd> <dt>

*придата* \[ окне\]
</dt> <dd>

Тип: **Byte \** _

Указатель на данные PRI, созданные при предыдущем вызове к [_ *мрмкреатересаурцефилеинмемори* *](mrmcreateresourcefileinmemory.md). Не освобождайте *придата* до тех пор, пока не завершите использование индексатора ресурсов, созданного этой функцией.

</dd> <dt>

*присизе* \[ окне\]
</dt> <dd>

Тип: **ulong**

Размер данных, на которые указывает *придата*.

</dd> <dt>

*индексатор* \[ в, out\]
</dt> <dd>

Тип: **[**мрмресаурцеиндексерхандле**](mrmresourceindexerhandle.md) \** _

Указатель на обработчик индексатора ресурса.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: _ *HRESULT**

\_ОК, если функция прошла успешно, в противном случае — другое значение. Чтобы определить успешность или сбой, используйте макросы Success () или FAILed () (определенные в файле Winerror. h).

## <a name="remarks"></a>Комментарии

Не освобождайте *придата* до тех пор, пока не завершите использование индексатора ресурсов, созданного этой функцией.
