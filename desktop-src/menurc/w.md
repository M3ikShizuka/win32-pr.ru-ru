<dl> <dt>

<span id="tools.w_1_gly"></span><span id="TOOLS.W_1_GLY"></span>**Предупреждение: символ ASCII не соответствует виртуальному коду клавиши**
</dt> <dd>

В операторе [**ускорителей**](accelerators-resource.md) существует недопустимый код виртуального ключа. Значения ASCII для некоторых символов, таких как \* , ^ или &, не эквивалентны кодам виртуальных клавиш для соответствующих ключей. В случае звездочки ( \* ) код виртуального ключа эквивалентен значению ASCII для 8, числовому символу в том же ключе.

</dd> <dt>

<span id="tools.w_2_gly"></span><span id="TOOLS.W_2_GLY"></span>**Предупреждение: SHIFT или CONTROL, используемые без VIRTKEY**
</dt> <dd>

Параметры **ALT**, **SHIFT** и [**Control**](control-control.md) применяются только к виртуальным ключам в операторе [**ускорителей**](accelerators-resource.md) . Убедитесь, что параметр **VIRTKEY** используется с одним из этих вариантов.

</dd> <dt>

<span id="tools.w_3_gly"></span><span id="TOOLS.W_3_GLY"></span>**Запись ресаурцетипеимажеланг: лангуажесизе: size**
</dt> <dd>

(V) компилятор ресурсов Microsoft Windows (RC) записывает указанный ресурс. *Тип* — тип ресурса. Может быть числом или строкой. *Изображение* может быть числом или строкой. *Язык* — это национальный язык ресурса. *Размер* — это десятичный размер ресурса в байтах.

</dd> </dl>
