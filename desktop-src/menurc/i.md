# <a name="i-menus-and-other-resources"></a>I (меню и другие ресурсы)

<dl> <dt>

<span id="tools.i_1_gly"></span><span id="TOOLS.I_1_GLY"></span>**Недопустимое сочетание клавиш**
</dt> <dd>

Параметр *события* в операторе [**ускорителей**](accelerators-resource.md) не распознан или имеет длину более двух символов.

</dd> <dt>

<span id="tools.i_2_gly"></span><span id="TOOLS.I_2_GLY"></span>**Недопустимый тип ускорителя (ASCII или VIRTKEY)**
</dt> <dd>

Параметр *типа* в операторе [**ускорителей**](accelerators-resource.md) должен содержать значение **ASCII** или **VIRTKEY** .

</dd> <dt>

<span id="tools.i_3_gly"></span><span id="TOOLS.I_3_GLY"></span>**Недопустимый управляющий символ**
</dt> <dd>

Недопустимый управляющий символ в операторе [**ускорителей**](accelerators-resource.md) . Допустимый управляющий символ состоит из курсора (^), за которым следует одна буква.

</dd> <dt>

<span id="tools.i_4_gly"></span><span id="TOOLS.I_4_GLY"></span>**Недопустимый тип элемента управления**
</dt> <dd>

Оператор Control в инструкции [**DIALOG**](dialog-resource.md) должен быть одним из следующих: [**AUTO3STATE**](auto3state-control.md), [**автоcheckbox**](autocheckbox-control.md), [**автоradiobutton**](autoradiobutton-control.md), [**CheckBox**](checkbox-control.md), [**ComboBox**](combobox-control.md), [**Control**](control-control.md), [**столбец ctext**](ctext-control.md), [**дефпушбуттон**](defpushbutton-control.md), [**EditText**](edittext-control.md), [**GROUPBOX**](groupbox-control.md), [**Icon**](icon-control.md), [**ListBox**](listbox-control.md), [**лтекст**](ltext-control.md), [**пушбокс**](pushbox-control.md), [**кнопка**](pushbutton-control.md), [**RADIOBUTTON**](radiobutton-control.md), [**ртекст**](rtext-control.md), [**ScrollBar**](scrollbar-control.md), [**STATE3**](state3-control.md).

</dd> <dt>

<span id="tools.i_5_gly"></span><span id="TOOLS.I_5_GLY"></span>**Недопустимая директива в предварительно обработанном файле RC**
</dt> <dd>

Указанное имя файла содержит встроенный символ новой строки.

</dd> <dt>

<span id="tools.i_6_gly"></span><span id="TOOLS.I_6_GLY"></span>**Недопустимый. Файл EXE**
</dt> <dd>

Недопустимый исполняемый файл (. exe). Убедитесь, что компоновщик создал его правильно, и что файл существует.

</dd> <dt>

<span id="tools.i_7_gly"></span><span id="TOOLS.I_7_GLY"></span>**Недопустимый переключатель, параметр**
</dt> <dd>

Использован недопустимый параметр. Чтобы получить список параметров командной строки, используйте **версию-кандидат/?** .

</dd> <dt>

<span id="tools.i_8_gly"></span><span id="TOOLS.I_8_GLY"></span>**Недопустимый параметр, слишком много параметров-D**
</dt> <dd>

Указано слишком много параметров **-d** . Чтобы определить большое количество символов, используйте директиву [**\# include**](-include.md) , чтобы включить файл определений.

</dd> <dt>

<span id="tools.i_9_gly"></span><span id="TOOLS.I_9_GLY"></span>**Недопустимый тип**
</dt> <dd>

Тип ресурса не был среди типов, определенных в включаемом файле.

</dd> <dt>

<span id="tools.i_10_gly"></span><span id="TOOLS.I_10_GLY"></span>**Недопустимое использование. Использовать RC-? для справки**
</dt> <dd>

Убедитесь, что у вас есть по крайней мере одно имя файла для работы. Список параметров командной строки можно получить с помощью **RC-?** .

</dd> <dt>

<span id="tools.i_11_gly"></span><span id="TOOLS.I_11_GLY"></span>**Ошибка ввода-вывода при чтении файла**
</dt> <dd>

Сбой чтения. Так как это универсальная подпрограммы, не указывается ни одно конкретное имя файла.

</dd> <dt>

<span id="tools.i_12_gly"></span><span id="TOOLS.I_12_GLY"></span>**Поиск ошибок ввода-вывода в файле**
</dt> <dd>

Сбой поиска в файле. Так как это универсальная подпрограммы, не указывается ни одно конкретное имя файла.

</dd> <dt>

<span id="tools.i_13_gly"></span><span id="TOOLS.I_13_GLY"></span>**Ошибка ввода-вывода при записи файла**
</dt> <dd>

Ошибка записи. Так как это универсальная подпрограммы, не указывается ни одно конкретное имя файла.

</dd> </dl>
