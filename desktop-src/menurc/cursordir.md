# <a name="cursordir-structure"></a>Структура КУРСОРДИР

Содержит размеры отдельного изображения курсора в группе ресурсов. Приведенное здесь определение структуры предназначено только для объяснения. он отсутствует в любом стандартном файле заголовка.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct {
  WORD Width;
  WORD Height;
} CURSORDIR;
```



## <a name="members"></a>Члены

<dl> <dt>

**Width**
</dt> <dd>

Тип: **слово**

</dd> <dd>

Ширина курсора в пикселях. Допустимые значения: 16, 32 и 64.

</dd> <dt>

**Height**
</dt> <dd>

Тип: **слово**

</dd> <dd>

Высота курсора в пикселях. Допустимые значения: 16, 32 и 64.

</dd> </dl>

## <a name="remarks"></a>Remarks

Структура **курсордир** передается в структуру [**ресдир**](resdir.md) , если структура **ресдир** описывает курсор.
