# <a name="menu-statement"></a>Оператор меню

Определяет меню для диалогового окна. Если инструкция не указана, в диалоговом окне нет меню.

``` syntax
MENU menuname
```

<dl> <dt>

<span id="menuname"></span><span id="MENUNAME"></span>*menuname*
</dt> <dd>

Имя или идентификатор используемого меню.

</dd> </dl>

## <a name="examples"></a>Примеры

В следующем примере демонстрируется использование инструкции диалогового окна **меню** .

``` syntax
MENU errmenu
```
