# <a name="varfileinfo-block-statement"></a>Варфилеинфо BLOCK, инструкция

Определяет блок сведений о переменной.

``` syntax
BLOCK "VarFileInfo" { VALUE "Translation", langID, charsetID . . . }
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="langID"></span><span id="langid"></span><span id="LANGID"></span>*Надежно*
</dt> <dd>

Один из идентификаторов языка, указанных в разделе "Примечания".

</dd> <dt>

<span id="charsetID"></span><span id="charsetid"></span><span id="CHARSETID"></span>*чарсетид*
</dt> <dd>

Один из идентификаторов наборов символов, указанных в разделе "Примечания".

</dd> </dl>

## <a name="remarks"></a>Комментарии

Можно указать более одной пары идентификаторов, но каждую пару следует отделять от предшествующей пары запятыми.
