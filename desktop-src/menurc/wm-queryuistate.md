# <a name="wm_queryuistate-message"></a>\_Сообщение КУЕРЮИСТАТЕ WM

Приложение отправляет сообщение **WM \_ куерюистате** для получения состояния пользовательского интерфейса для окна.


```C++
#define WM_QUERYUISTATE                 0x0129
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется и должен быть равен 0.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется и должен быть равен 0.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение равно **null** , если индикаторы фокуса и сочетания клавиш видимы. В противном случае возвращаемое значение может быть одним или несколькими следующими значениями.



| Возвращаемый код и значение                                                                                                                                       | Описание                                                                 |
|---------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| <dl> <dt>**Уисф \_ Активный**</dt> <dt>0x4</dt> </dl>    | Элемент управления должен быть нарисован в стиле, используемом для активных элементов управления.<br/> |
| <dl> <dt>**Уисф \_ ХИДЕАКЦЕЛ**</dt> <dt>0x2</dt> </dl> | Сочетания клавиш скрыты.<br/>                                |
| <dl> <dt>**Уисф \_ ХИДЕФОКУС**</dt> <dt>0x1</dt> </dl> | Индикаторы фокуса скрыты.<br/>                                     |
