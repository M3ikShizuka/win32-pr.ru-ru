# <a name="direntry-structure"></a>Структура аренды

Содержит уникальный порядковый номер, определяющий отдельный шрифт в группе ресурсов Font. Приведенное здесь определение структуры предназначено только для объяснения. он отсутствует в любом стандартном файле заголовка.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct {
  WORD fontOrdinal;
} DIRENTRY;
```



## <a name="members"></a>Члены

<dl> <dt>

**фонтординал**
</dt> <dd>

Тип: **слово**

</dd> <dd>

Уникальный порядковый идентификатор для отдельного шрифта в группе ресурсов шрифтов.

</dd> </dl>

## <a name="remarks"></a>Remarks

Структура [**фонтдирентри**](fontdirentry.md) для указанного шрифта непосредственно соответствует структуре **аренды** для этого шрифта.
