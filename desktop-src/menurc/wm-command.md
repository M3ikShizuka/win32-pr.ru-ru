# <a name="wm_command-message"></a>\_Командное сообщение WM

Посылается, когда пользователь выбирает элемент команды из меню, когда элемент управления отправляет сообщение уведомления родительскому окну или при преобразовании нажатия клавиши быстрого вызова.


```C++
#define WM_COMMAND                      0x0111
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Описание этого параметра см. в разделе Примечания.

</dd> <dt>

*lParam* 
</dt> <dd>

Описание этого параметра см. в разделе Примечания.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если приложение обрабатывает это сообщение, оно должно вернуть ноль.

## <a name="example"></a>Пример

```c
BOOL AboutDlg (
    HWND hDlg, 
    UINT message, 
    WPARAM wParam, 
    LPARAM lParam)
{
    BOOL bRet = FALSE;
    
    switch (message) 
    {
        case WM_INITDIALOG:
            bRet = TRUE;
            break;

        case WM_COMMAND:
            if (wParam == IDOK ||
                wParam == IDCANCEL) 
            {
                EndDialog(hDlg, TRUE);
                bRet = TRUE;
            }
            break;
    }

    return bRet;
}
```
Пример, взятый из [классической выборки Windows](https://github.com/microsoft/Windows-classic-samples) на сайте GitHub.


## <a name="remarks"></a>Комментарии

Ниже приведены сводные сведения об использовании параметров *wParam* и *lParam* .



| Источник сообщения | wParam (высокое слово)                | wParam (низкое слово)                | lParam                       |
|----------------|-----------------------------------|----------------------------------|------------------------------|
| Меню           | 0                                 | Идентификатор меню (IDM \_ \* )        | 0                            |
| Accelerator    | 1                                 | Идентификатор ускорителя (IDM \_ \* ) | 0                            |
| Control        | Код уведомления, определяемый элементом управления | Идентификатор элемента управления               | Маркер окна управления |



 

### <a name="menus"></a>Меню

Если приложение включает разделитель меню, система отправляет **\_ командное сообщение WM** с младшим словом параметра *wParam* , установленным в ноль, когда пользователь выбирает разделитель.

Если меню определено со значением [**менуинфо. Двстиле**](/windows/win32/api/winuser/ns-winuser-menuinfo) **MNS \_ нотифибипос**, то вместо **\_ команды WM** отправляется [**WM \_ MENUCOMMAND**](wm-menucommand.md) .

### <a name="accelerators"></a>Клавиши быстрого доступа

Нажатия клавиш быстрого вызова, которые выбирают элементы из меню окно, преобразуются в сообщения [**WM \_ сискомманд**](wm-syscommand.md) .

Если нажатие клавиши быстрого вызова соответствует пункту меню, когда окно, которому принадлежит меню, является сведенным, то **\_ командное сообщение WM** не отправляется. Однако если нажатие клавиши быстрого вызова не соответствует ни одному из элементов в меню окна или меню окна, отправляется **\_ командное сообщение WM** , даже если окно является сведенным.
