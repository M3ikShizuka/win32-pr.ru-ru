# <a name="endif"></a>\#endif

Директива **\# endif** отмечает конец блока условной компиляции, определенного директивой **\# ifdef** . Для каждой директивы **\# If**, **\# ifdef** или **\# ifndef** требуется один **\# endif** .

``` syntax
#endif
```

Эта директива не имеет параметров.
