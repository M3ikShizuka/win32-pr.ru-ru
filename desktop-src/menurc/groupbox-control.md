# <a name="groupbox-control"></a>Элемент управления GROUPBOX

Определяет элемент управления "Группа". Элемент управления — это прямоугольник, который объединяет другие элементы управления. Элементы управления группируются путем рисования границы вокруг них и отображения заданного текста в левом верхнем углу.

Оператор **GROUPBOX** , который можно использовать только в инструкции [**диаложекс**](dialogex-resource.md) , определяет текст, идентификатор, измерения и атрибуты окна элемента управления.

``` syntax
GROUPBOX text, id, x, y, width, height [, style [, extended-style]]
```

<dl> <dt>

<span id="style"></span><span id="STYLE"></span>*стиль*
</dt> <dd>

Стили элементов управления. Это значение может представлять собой сочетание стиля класса кнопки "BS" и стилей **WS \_ TABSTOP** и **WS \_ disabled** . **\_**

Если стиль не указан, то по умолчанию используется стиль **BS \_ GROUPBOX**.

</dd> </dl>

Дополнительные сведения об общем синтаксисе инструкции Control см. в разделе [Общие параметры управления](common-control-parameters.md).

## <a name="remarks"></a>Комментарии

Если в стиле содержится элемент **WS \_ TABSTOP** или в тексте задан ускоритель, то фокус перемещается на первый элемент управления в группе.

## <a name="examples"></a>Примеры

В этом примере определяется элемент управления "Группа" с метками "Параметры":

``` syntax
GROUPBOX "Options", 101, 10, 10, 100, 100
```
