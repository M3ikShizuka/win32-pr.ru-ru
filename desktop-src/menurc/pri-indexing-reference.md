# <a name="package-resource-indexing-pri-reference"></a>Справочник по индексированию ресурсов пакетов (PRI)

Набор API-интерфейсов для работы с индексатором ресурсов. Индексатор ресурсов используется для создания файлов индекса ресурсов пакета (PRI) для приложения UWP. Дополнительные сведения и пошаговые руководства по использованию этих API см. в разделе [пакетные API индексирования ресурсов (PRI) и пользовательские системы сборки](/windows/uwp/app-resources/pri-apis-custom-build-systems).
