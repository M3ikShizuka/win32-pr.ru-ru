# <a name="wm_menudrag-message"></a>\_Сообщение МЕНУДРАГ WM

Посылается владельцу меню перетаскивания, когда пользователь перетаскивает элемент меню.


```C++
#define WM_MENUDRAG                     0x0123
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Позиция элемента, с которого началась операция перетаскивания.

</dd> <dt>

*lParam* 
</dt> <dd>

Маркер меню, содержащего элемент.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Приложение должно вернуть одно из следующих значений.



| Возвращаемый код и значение                                                                                                                                   | Описание                                                                           |
|-----------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------|
| <dl> <dt>**МНД \_ ПРОДОЛЖИТЬ**</dt> <dt>0</dt> </dl> | Меню должно оставаться активным. Если мышь освобождена, ее следует игнорировать.<br/> |
| <dl> <dt>**МНД \_ ЕНДМЕНУ**</dt> <dt>1</dt> </dl>  | Меню должно быть завершено.<br/>                                                      |



 

## <a name="remarks"></a>Комментарии

Приложение может вызвать функцию [**DoDragDrop**](/windows/win32/api/ole2/nf-ole2-dodragdrop) в ответ на это сообщение.

Чтобы создать меню перетаскивания, вызовите [**сетменуинфо**](/windows/desktop/api/Winuser/nf-winuser-setmenuinfo) с помощью **MNS \_ DRAGDROP**.
