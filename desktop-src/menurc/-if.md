# <a name="if"></a>\#if

Директива **\# If** управляет условной компиляцией файла ресурсов путем проверки указанного константного выражения. Значение, если константное выражение не равно нулю, **\# Если** указывает компилятору продолжить обработку операторов до следующей директивы **\# endif**, **\# else** или **\# elif** , а затем перейти к оператору после директивы **\# endif** . Значение, если константное выражение равно нулю, **\# Если** указывает компилятору перейти к следующей директиве **\# endif**, **\# else** или **\# elif** .

``` syntax
#if constant-expression
```

<dl> <dt>

<span id="constant-expression"></span><span id="CONSTANT-EXPRESSION"></span>*константное выражение*
</dt> <dd>

Выражение, которое должно быть проверено. Это значение представляет собой определенное имя, целочисленную константу или выражение, состоящее из имен, целых чисел, арифметических и реляционных операторов.

</dd> </dl>

## <a name="example"></a>Пример

В этом примере оператор [**Bitmap**](bitmap-resource.md) компилируется только в том случае, если значение назначенной версии меньше 3:

``` syntax
#if Version < 3
BITMAP 1 errbox.bmp
#endif
```
