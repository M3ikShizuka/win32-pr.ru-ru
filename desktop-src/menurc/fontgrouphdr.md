# <a name="fontgrouphdr-structure"></a>Структура ФОНТГРАУФДР

Содержит сведения, необходимые приложению для доступа к определенному шрифту. Приведенное здесь определение структуры предназначено только для объяснения. он отсутствует в любом стандартном файле заголовка.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct {
  WORD     NumberOfFonts;
  DIRENTRY DE;
} FONTGROUPHDR;
```



## <a name="members"></a>Члены

<dl> <dt>

**нумбероффонтс**
</dt> <dd>

Тип: **слово**

</dd> <dd>

Число отдельных шрифтов, связанных с этим ресурсом.

</dd> <dt>

**RU**
</dt> <dd>

Тип: **[ **Аренда**](direntry.md)**

</dd> <dd>

Структура, которая содержит уникальный порядковый идентификатор для каждого шрифта в ресурсе. Элемент **de** — это заполнитель для массива с переменной длиной из [**неарендных**](direntry.md) структур.

</dd> </dl>

## <a name="remarks"></a>Remarks

Структура **фонтграуфдр** соответствует данным для отдельных шрифтов в. RES — файл. Компилятор ресурсов автоматически добавляет структуру **фонтграуфдр** , как правило, в качестве последней записи в файле.
