# <a name="resource-compiler"></a>компилятор ресурсов

Компилятор ресурсов Microsoft Windows (RC) — это средство, используемое для создания приложений на основе Windows. В этом обзоре описывается создание файла определения ресурса (скрипта), компиляция ресурсов приложения и добавление скомпилированных ресурсов в приложение.

Это средство доступно в Visual Studio и пакете средств разработки программного обеспечения (SDK) для Microsoft Windows.

В следующих разделах описывается синтаксис файла ресурсов и Командная строка RC:

-   [Сведения о файлах ресурсов](about-resource-files.md)
-   [Использование компилятора ресурсов (командная строка RC)](using-rc-the-rc-command-line-.md)
