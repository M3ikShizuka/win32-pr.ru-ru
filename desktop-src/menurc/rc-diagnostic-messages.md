# <a name="rc-diagnostic-messages"></a>Диагностические сообщения RC

В этом разделе описываются диагностические сообщения, созданные версией-КАНДИДАТом. Многие из этих сообщений появляются, когда версии-КАНДИДАТу не удается правильно скомпилировать ресурсы. Описание приводит к уточнению причины каждого сообщения.

Прописная V в круглых скобках (V) в начале описания сообщения означает, что сообщение отображается, только если RC выполняется с параметром **/v** (verbose). Эти сообщения обычно являются информативными и не обязательно свидетельствуют об ошибках.
