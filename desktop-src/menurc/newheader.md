# <a name="newheader-structure"></a>Структура НЕВХЕАДЕР

Содержит число значков или компонентов курсора в группе ресурсов. Приведенное здесь определение структуры предназначено только для объяснения. он отсутствует в любом стандартном файле заголовка.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct {
  WORD Reserved;
  WORD ResType;
  WORD ResCount;
} NEWHEADER, *NEWHEADER;
```

<dl> <dt>

**Reserved**

Тип: **слово**

</dd> <dd>

</dt> <dd>

Тип: **слово**

</dd> <dd>

|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------|
| <span id="RES_CURSOR"></span><span id="res_cursor"></span><dl> <dt>**RES \_ КУРСОР**</dt> <dt>2</dt> </dl> | Тип ресурса курсора.<br/> |
| <span id="RES_ICON"></span><span id="res_icon"></span><dl> <dt>**RES \_ ЗНАЧОК**</dt> <dt>1</dt> </dl>       | Значок тип ресурса.<br/>   |

## <a name="remarks"></a>Remarks

Одна или несколько структур [**ресдир**](resdir.md) сразу же следуют структуре **невхеадер** в RES – файле. Элемент **рескаунт** указывает количество структур **ресдир** .
