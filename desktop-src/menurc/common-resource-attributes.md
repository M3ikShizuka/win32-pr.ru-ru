# <a name="common-resource-attributes"></a>Общие атрибуты ресурсов

Инструкции определения ресурсов, поддерживаемые в 16-разрядных Windows, включают параметр *Load-MEM* , указывающий характеристики загрузки и памяти ресурса. Эти атрибуты разрешены в скриптах ресурсов для обратной совместимости, но они не учитываются. Ресурсы Windows загружаются при загрузке соответствующего модуля и освобождаются при выгрузке модуля.

## <a name="load-attributes"></a>Загрузить атрибуты

Атрибуты нагрузки указывают время загрузки ресурса. Параметр Load должен быть одним из следующих атрибутов.



| attribute      | Описание                                                                  |
|----------------|------------------------------------------------------------------------------|
| **Установка**    | Не обрабатывается. В 16-разрядных Windows ресурс загружается с исполняемым файлом. |
| **лоадонкалл** | Не обрабатывается. В 16-разрядных Windows ресурс загружается при вызове.              |



 

## <a name="memory-attributes"></a>Атрибуты памяти

Атрибуты памяти указывают, является ли ресурс фиксированным или перемещаемым, является ли он недоступным и является ли он чистым. Параметр Memory может быть одним или несколькими следующими атрибутами.



| attribute       | Описание                                                                                                                               |
|-----------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| **FIXED**       | Не обрабатывается. В 16-разрядных Windows ресурс остается в фиксированном расположении в памяти.                                                              |
| **MOVEABLE**    | Не обрабатывается. В 16-разрядных Windows ресурс можно переместить, если это необходимо для сжатия памяти.                                                     |
| **ОТКЛОНЕНо** | Не обрабатывается. В 16-разрядных Windows ресурс может быть отклонен, если он больше не нужен.                                                            |
| **ЧИСТЫМИ**        | Не обрабатывается. Принимается на совместимость с существующими скриптами ресурсов.                                                                       |
| **НЕЧИСТЫМИ**      | Не обрабатывается. Принимается на совместимость с существующими скриптами ресурсов.                                                                       |
| **ИСПОЛЬЗУЕМЫЙ**      | Не обрабатывается. В 16-разрядных Windows для обычных модулей параметр SHARed не учитывается. Для ресурса из модуля ПЗУ Windows память является общей.        |
| **НЕ ЯВЛЯЮЩАЯСЯ общей**   | Не обрабатывается. В 16-разрядных Windows для обычных модулей игнорируется необщий доступ. Для ресурса из модуля ПЗУ Windows память не используется совместно. |
