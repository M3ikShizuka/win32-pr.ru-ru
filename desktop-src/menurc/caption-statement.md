# <a name="caption-statement"></a>ЗАГОЛОВОК, инструкция

Определяет заголовок для диалогового окна. Заголовок отображается в строке заголовка поля (если он имеется).

Заголовок по умолчанию пуст.

``` syntax
CAPTION "captiontext"
```

<dl> <dt>

<span id="captiontext"></span><span id="CAPTIONTEXT"></span>*каптионтекст*
</dt> <dd>

Символьная строка, заключенная в двойные кавычки (").

</dd> </dl>

## <a name="examples"></a>Примеры

В следующем примере демонстрируется использование инструкции **Caption** :

``` syntax
CAPTION "Error!" 
```
