# <a name="mrmpeekresourceindexermessages-function"></a>Функция Мрмпикресаурцеиндексермессажес

\[Некоторые сведения относятся к предварительно выпущенному продукту, который может быть значительно изменен перед коммерческой выпуском. Майкрософт не дает никаких гарантий, явных или подразумеваемых, в отношении предоставленной здесь информации.\]

Просмотр любых сообщений, созданных индексатором ресурсов. Дополнительные сведения и пошаговые руководства по использованию этих API см. в разделе [пакетные API индексирования ресурсов (PRI) и пользовательские системы сборки](/windows/uwp/app-resources/pri-apis-custom-build-systems).

## <a name="syntax"></a>Синтаксис


```C++
HRESULT HRESULT MrmPeekResourceIndexerMessages(
  _In_  MrmResourceIndexerHandle  indexer,
  _Out_ MrmResourceIndexerMessage **messages,
  _Out_ ULONG                     *numMsgs
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*индексатор* \[ окне\]
</dt> <dd>

Тип: **[ **мрмресаурцеиндексерхандле**](mrmresourceindexerhandle.md)**

Маркер, идентифицирующий индексатор ресурсов, сообщения которого вы хотите просмотреть.

</dd> <dt>

*сообщения* \[ заполняет\]
</dt> <dd>

Тип: **[ **мрмресаурцеиндексермессаже**](mrmresourceindexermessage.md)\*\***

Указатель на указатель на [**мрмресаурцеиндексермессаже**](mrmresourceindexermessage.md). Функция выделяет массив **мрмресаурцеиндексермессаже** и возвращает указатель на эту память в *сообщениях*. Не освобождайте указатель, адрес которого передается в *сообщения*.

</dd> <dt>

*нуммсгс* \[ заполняет\]
</dt> <dd>

Тип: **ulong \** _

Число сообщений, возвращенных в _messages *.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

\_ОК, если функция прошла успешно, в противном случае — другое значение. Чтобы определить успешность или сбой, используйте макросы Success () или FAILed () (определенные в файле Winerror. h).

## <a name="remarks"></a>Комментарии

Ваше приложение не владеет выделенным и возвращаемым в *сообщениях* памятью, поэтому не освобождайте его.
