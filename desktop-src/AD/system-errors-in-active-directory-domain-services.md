# <a name="system-errors-in-active-directory-domain-services"></a>Системные ошибки в службах домен Active Directory

В этом разделе содержатся значения системных ошибок, возвращаемые функциями служб домен Active Directory.



| Значение                             | Описание                                                           |
|-----------------------------------|-----------------------------------------------------------------------|
| **хрфилеклосе**<br/>        | Не удается закрыть файл DOS.<br/>                              |
| **храутофсреадс**<br/>     | Не удалось запустить поток из-за отсутствия доступных.<br/> |
| **хртуманио**<br/>        | Система занята, так как в ней слишком много операций ввода-вывода.<br/>        |
| **хрбфнотсинчронаус**<br/> | Страница буфера была удалена.<br/>                          |
| **хрбфпаженотфаунд**<br/>   | Не удалось найти страницу.<br/>                                   |
| **хрбфинусе**<br/>          | Не удалось сбросить буфер.<br/>                              |
