# <a name="determining-a-users-or-groups-membership-in-a-group"></a>Определение членства пользователя или группы в группе

Метод [**иадсграуп.**](/windows/desktop/api/iads/nf-iads-iadsgroup-ismember) GetObject можно использовать для определения того, является ли объект членом группы. Этот метод возвращает **значение true** , если указанный объект является прямым членом группы, то есть свойство элемента группы содержит указанный объект.

Группа может содержать другие группы. Метод [**иадсграуп. DataMember**](/windows/desktop/api/iads/nf-iads-iadsgroup-ismember) не рекурсивно проверяет членов групп в своем свойстве элемента, группирует в этих группах и т. д. Для рекурсивной проверки того, что объект является членом группы, перечислите группы в свойстве Member, проверьте члены этих групп, чтобы определить, является ли объект членом, и если эти группы содержат другие группы, проверьте их члены и т. д.

> [!Note]  
> Поскольку группы могут быть вложенными, членство в группе может иметь циклы. Любой скрипт, который перечисляет несколько групп, должен обеспечить выполнение внутреннего списка групп для завершения рекурсии, если группа уже была открыта.
