# <a name="using-standard-dialog-boxes-to-handle-objects-in-active-directory-domain-services"></a>Использование стандартных диалоговых окон для управления объектами в службах домен Active Directory Services

Windows предоставляет общую библиотеку диалоговых окон для распространенных операций, таких как открытие файла, обзор файлов и т. д. Начиная с Windows 2000, система предоставляет диалоговые окна, которые можно использовать для общих операций с пользовательскими интерфейсами в домен Active Directory Services.

Следующие диалоговые окна доступны для использования со службами домен Active Directory:

-   [Средство выбора объектов каталога](directory-object-picker.md)
-   [Браузер домена](domain-browser.md)
-   [Обозреватель контейнеров](container-browser.md)
-   [Вкладки свойств «пользователи и компьютеры Active Directory»](active-directory-users-and-computers-property-sheets.md)

Кроме того, мастера создания служб домен Active Directory также можно вызывать из приложения. Дополнительные сведения см. [в разделе Вызов мастеров создания из приложения](invoking-creation-wizards-from-your-application.md).
