# <a name="managing-users-on-member-servers-and-windows-2000-professional"></a>Управление пользователями на рядовых серверах и Windows 2000 Professional

На рядовых серверах и Windows 2000 Professional имеется локальная база данных безопасности. Локальная база данных безопасности может содержать собственные локальные пользователи, областью действия которых является только конкретный компьютер, на котором они созданы. При управлении этими типами пользователей на рядовых серверах и рабочих станциях используется поставщик WinNT.

При управлении пользователями в домене Windows 2000 с помощью ADSI используется поставщик LDAP. При управлении пользователями на рядовых серверах и рабочих станциях используется поставщик WinNT.
