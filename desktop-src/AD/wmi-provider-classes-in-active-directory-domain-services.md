# <a name="wmi-provider-classes-in-active-directory-domain-services"></a>Классы поставщиков WMI в службах домен Active Directory Services

Службы домен Active Directory Services имеют следующие классы для использования со сценариями WMI:

-   [**МСАДный \_ DomainController**](msad-domaincontroller.md)
-   [**МСАД \_ намингконтекст**](msad-namingcontext.md)
-   [**МСАД \_ реплкурсор**](msad-replcursor.md)
-   [**МСАД \_ реплнеигхбор**](msad-replneighbor.md)
-   [**МСАД \_ реплпендингоп**](msad-replpendingop.md)

Сам поставщик представлен классом [**ReplicationProvider1**](replicationprovider1.md) , экземпляр которого имеет имя «ReplProv1».
