# <a name="modifying-application-directory-partition-configuration"></a>Изменение конфигурации раздела каталога приложений

Раздел каталога приложения можно изменить, изменив определенные атрибуты объекта [**crossRef**](/windows/desktop/ADSchema/c-crossref) , представляющего раздел каталога приложений. Чтобы найти объект [**crossRef**](/windows/desktop/ADSchema/c-crossref) , представляющий конкретный раздел каталога приложений, выполните поиск объекта **crossRef** в контейнере секций, имеющего значение атрибута [**NCName**](/windows/desktop/ADSchema/a-ncname) , которое совпадает с различающимся именем раздела каталога приложений. Различающееся имя объекта **crossRef** можно использовать для привязки к объекту **crossRef** .

В следующей таблице перечислены атрибуты объекта [**crossRef**](/windows/desktop/ADSchema/c-crossref) , которые могут быть изменены.

| attribute                                                                                                    | Описание                                                                                                                                                                                                                                                                                                                                       |
|--------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**msDS-Replication-уведомлять-First-DSA — задержка**](/windows/desktop/ADSchema/a-msds-replication-notify-first-dsa-delay)           | Указывает задержку в секундах после изменения исходного объекта до уведомления первого партнера репликации. Дополнительные сведения см. в статье [Репликация разделов каталога приложений](application-directory-partition-replication.md).                                                                                                   |
| [**msDS-Replication-notify-последующий-DSA-Delay**](/windows/desktop/ADSchema/a-msds-replication-notify-subsequent-dsa-delay) | Указывает задержку (в секундах) между последовательными уведомлениями второго, третьего и т. д. для партнеров репликации. Дополнительные сведения см. в статье [Репликация разделов каталога приложений](application-directory-partition-replication.md).                                                                                                 |
| [**msDS-Сдреференцедомаин**](/windows/desktop/ADSchema/a-msds-sdreferencedomain)                                             | Определяет домен, используемый подсистемой безопасности для интерпретации ссылок на локальные домены в атрибутах [**нтсекуритидескриптор**](/windows/desktop/ADSchema/a-ntsecuritydescriptor) объектов в этом разделе каталога приложений. Дополнительные сведения см. в разделе [Безопасность раздела каталога приложений](application-directory-partition-security.md). |
| [**msDS-NC-Replica-Locations**](/windows/desktop/ADSchema/a-msds-nc-replica-locations)                                       | Содержит различающееся имя объекта [**нтдсдса**](/windows/desktop/ADSchema/c-ntdsdsa) для каждого контроллера домена, на котором размещена реплика раздела каталога приложений. Дополнительные сведения см. в разделе [Добавление или удаление реплики раздела каталога приложений](adding-or-deleting-an-application-directory-partition-replica.md).            |
