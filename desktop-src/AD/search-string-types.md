# <a name="initial-medial-and-final-search-strings"></a>Начальные, срединного и окончательные строки поиска

Существует три типа поиска с подстановочными знаками, упоминаемых в документации по созданию запросов к строкам поиска. Это поиск с подстановочными знаками: Initial-, срединного-и Final-Поиск строк. В этом разделе описываются эти операции поиска.

## <a name="initial-search-strings"></a>Начальные строки поиска

Начальные строки поиска соответствуют заданному набору символов в начале строки, за которым следует подстановочный знак. Например, строка начального поиска `Act*` будет соответствовать "Active Directory".

## <a name="medial-search-strings"></a>Строки Medial-Search

Срединного — строки поиска соответствуют заданному набору символов в середине строки с префиксом и/или с последующим символом-шаблоном. Строки срединного-Search `*Dir*` , `*ive*Dir*` и `*ve*Dir*tor*` будут соответствовать "Active Directory".

## <a name="final-search-strings"></a>Итоговые строки поиска

Конечные строки поиска соответствуют заданному набору символов в конце строки, перед которыми ставится символ-шаблон. Например, строка окончательного поиска `*ory` будет соответствовать "Active Directory".
