# <a name="administration-model"></a>Модель администрирования

Полномочные пользователи выполняют администрирование в домен Active Directory Services. Пользователь имеет право более высокого уровня полномочий на выполнение определенного набора действий с определенным набором объектов и классов объектов в определенном поддереве каталога. Это называется *делегированным администрированием*. Делегированное администрирование позволяет точно контролировать, кто может выполнять действия и делегирование полномочий без предоставления повышенных привилегий.
