# <a name="user-interface-mappings-in-active-directory-domain-services"></a>Сопоставления пользовательского интерфейса в службах домен Active Directory Services

Пользовательский интерфейс для служб домен Active Directory Services включает страницы свойств и другие пользовательские интерфейсы для установки и получения атрибутов часто используемых классов. В подразделах этого раздела содержатся таблицы, которые сопоставляют метки, отображаемые в пользовательских интерфейсах, с соответствующими атрибутами в схеме Active Directory.
