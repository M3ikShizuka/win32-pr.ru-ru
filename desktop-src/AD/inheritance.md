# <a name="inheritance-ad-ds"></a>Наследование (AD DS)

[*Наследование*](/previous-versions/windows/desktop/legacy/ms681910(v=vs.85)) позволяет распространить данный элемент ACE из контейнера, где он был применен ко всем дочерним элементам контейнера. Наследование можно сочетать с делегированием для предоставления административных прав всему поддереву каталога в одной операции.
