# <a name="deciding-which-attributes-to-retrieve-for-each-object-found"></a>Выбор извлекаемых атрибутов для каждого найденного объекта

При поиске объектов определенного класса кажется, что вы получаете атрибуты, которые действительно существуют в объектах этого класса. Следуйте тем же правилам, которые были рассмотрены в разделе [Поиск списка атрибутов для запроса](finding-a-list-of-attributes-to-query.md). Имейте в виду, что сконструированные атрибуты могут быть получены в запросе, но сконструированные атрибуты нельзя использовать в фильтрах.
