# <a name="security-in-active-directory-domain-services"></a>Безопасность в домен Active Directory Services

домен Active Directory Services являются частью Windows 2000 доверенных вычислительных систем и являются полным участником в инфраструктуре безопасности Windows 2000. Каждый объект на Active Directory сервере защищен собственным дескриптором безопасности. Система проверяет все попытки доступа к объекту или атрибуту на Active Directory сервере, проверяя права доступа, предоставленные дескриптором безопасности объекта.

В следующих разделах рассматриваются важные элементы безопасности служб домен Active Directory Services.

-   [Защита объектов и атрибутов](object-and-attribute-protection.md)
-   [Делегирование](delegation.md)
-   [Наследование](inheritance.md)

Дополнительные сведения и примеры кода см. [в разделе Управление доступом к объектам в домен Active Directory Services](controlling-access-to-objects-in-active-directory-domain-services.md).
