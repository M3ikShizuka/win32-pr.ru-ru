# <a name="characteristics-of-attributes"></a>Характеристики атрибутов

Каждый атрибут в объекте домен Active Directory Services определяется объектом [**attributeSchema**](/windows/desktop/ADSchema/c-attributeschema) в контейнере схемы. Свойства объекта **attributeSchema** определяют характеристики атрибута, такие как:

-   Идентификаторы атрибутов. Атрибуты имеют несколько идентификаторов, которые наиболее интересны с точки зрения программирования — [**lDAPDisplayName**](/windows/desktop/ADSchema/a-ldapdisplayname), который используется клиентами LDAP для чтения и записи атрибута, а также [**schemaIDGUID**](/windows/desktop/ADSchema/a-schemaidguid), который используется в дескрипторах безопасности для управления доступом к атрибуту.
-   Тип данных, содержащихся в экземплярах атрибута. Свойства синтаксиса атрибута определяют тип данных, например целочисленный, строковый или двоичный. Дополнительные свойства могут указывать диапазон значений, допустимых для атрибута, и определить, может ли экземпляр атрибута иметь несколько значений.
-   Включение атрибута в группы. Другие свойства помечают атрибут, включаемый в набор свойств, который является набором связанных свойств, или включается в набор атрибутов, которые реплицируются в глобальном каталоге или индексируются для оптимизации производительности поиска.

В следующей таблице перечислены [**lDAPDisplayName**](/windows/desktop/ADSchema/a-ldapdisplayname) и описание ключевых свойств объекта [**attributeSchema**](/windows/desktop/ADSchema/c-attributeschema) . Дополнительные сведения и полный список обязательных и необязательных свойств объекта **attributeSchema** см. в разделе [**класс Attribute-Schema**](/windows/desktop/ADSchema/c-attributeschema).

| [**ранжеловер**](/windows/desktop/ADSchema/a-rangelower), [ **ранжеуппер**](/windows/desktop/ADSchema/a-rangeupper)         | Пара целых чисел, задающих нижнюю и верхнюю границы диапазона значений для этого атрибута. Аналогично, если Ранжеуппер для строки имеет значение 2000, а Ранжеловер отсутствует, это означает, что ограничение длины не ограничено.<br/> |
