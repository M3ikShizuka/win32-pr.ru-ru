# <a name="finding-objects-by-name"></a>Поиск объектов по имени

Большинство объектов в службах домен Active Directory используют свойство **CN** в качестве атрибута именования. Однако некоторые объекты используют атрибут именования, отличный от **CN**. Например, контроллер домена использует свойство **домаинднс** для атрибута именования, а подразделение использует свойство **OrganizationalUnit** для атрибута именования. Чтобы не использовать другой атрибут именования для различных типов объектов, для поиска объектов по имени следует использовать свойство **Name** , которое содержит относительное различающееся имя объекта.

## <a name="examples"></a>Примеры

В следующем примере кода показаны различные строки запроса, которые можно использовать для поиска объектов по имени.

Следующая строка запроса находит все объекты с именем, которое начинается с «Джеффа».


```C++
(name=Jeff*)
```



Следующая строка запроса находит все объекты-компьютеры с именем, которое начинается с "арендовано" или "Corp".


```C++
(&(objectCategory=computer)(|(name=leased*)(name=corp*)))
```



Следующая строка запроса находит всех пользователей и с именем, которое начинается с "Карен" или "Джефф".


```C++
(&(&(objectClass=user)(objectCategory=person))(|(name=Karen*)(name=Jeff*)))
```
