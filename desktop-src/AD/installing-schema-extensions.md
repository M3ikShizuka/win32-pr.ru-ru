# <a name="installing-schema-extensions"></a>Установка расширений схемы

Обновления схемы будут строго контролироваться максимум на сайтах клиентов. Это рекомендуемая методика. Если службе требуются расширения схемы, расширения схемы должны быть установлены отдельно. Это позволяет клиентам выполнять обновление схемы отдельно и перед остальной частью установки.
