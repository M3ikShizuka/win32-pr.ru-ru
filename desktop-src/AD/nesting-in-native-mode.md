# <a name="nesting-in-native-mode"></a>Вложенность в собственном режиме

В следующем списке описывается, что может содержаться в группе, существующей в домене в основном режиме. Этот же список также применяется к группам распространения в доменах в смешанном режиме. Область группы определяет эти правила включения.

-   Универсальная группа может содержать другие универсальные группы, глобальные группы и учетные записи из любого домена в лесу, в котором находится эта универсальная группа.
-   Глобальная группа может содержать другие глобальные группы и учетные записи из того же домена, к которому принадлежит группа. Глобальная группа не может содержать универсальные группы или любые глобальные группы или учетные записи из другого домена.
-   Локальная группа домена может содержать универсальные группы, глобальные группы и учетные записи из любого домена или леса. Локальная группа домена также может содержать другие локальные группы домена из того же домена, к которому принадлежит эта группа. Локальная группа домена не может содержать другие локальные группы домена из любого другого домена или леса.
