# <a name="searching-in-active-directory-domain-services"></a>Поиск в домен Active Directory Services

Поиск является ключевой функцией служб домен Active Directory Services. Поиск позволяет находить объекты в каталоге на основе критериев выбора (запроса) и получать указанные свойства для найденных объектов.

Поиск в домен Active Directory Services — это вопрос поиска контроллера домена, привязки к объекту, где поиск должен начинаться в каталоге, отправка запроса и обработка результатов.
