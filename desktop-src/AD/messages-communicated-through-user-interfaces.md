-   [**\_выпуск ККПМ**](cqpm-release.md)

## <a name="miscellaneous-messages"></a>Прочие сообщения

В следующей таблице перечислены сообщения, которые может отправить Служба каталогов.



| Message                  | Значение                     | Описание                                                                                                                                                                                                                                   |
|--------------------------|---------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_сообщение дспроп аттрчанжед \_ | "Дспропаттрчанжед"       | Сообщение, отправленное для синхронизации страниц свойств и средств администрирования службы каталогов, объявленных в DSClient. h.                                                                                                                       |
| ДСКПМ \_ жеткласслист      | ККПМ \_ хандлерспеЦифик + 0   | Сообщение страницы, отправленное на страницы формы для получения списка классов для запроса, которое используется селектором полей и свойством для построения его списка классов вывода. wParam = Flags и lParam = ЛПЛПДСКУЕРИКЛАССЛИСТ, объявленные в dsquery. h. |
| ДСКПМ \_ хелптопикс        | (ККПМ \_ ХАНДЛЕРСПЕЦИФИК + 1) | Сообщение страницы, отправленное на страницы формы для обработки запроса "разделы справки". wParam = 0, lParam = Хвндпарент, объявленный в dsquery. h.                                                                                                         |
