# <a name="data-model"></a>Модель данных

Модель данных для служб домен Active Directory является производной от модели данных X. 500. Каталог содержит объекты, представляющие различные виды сортировки, описываемые атрибутами. Совокупность объектов, которые могут храниться в каталоге, определяется в схеме. Для каждого класса объектов схема определяет, какие атрибуты должен иметь экземпляр класса, какие дополнительные атрибуты он может иметь и какой класс объекта может быть родителем текущего класса Object.
