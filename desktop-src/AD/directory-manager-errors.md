# <a name="directory-manager-errors"></a>Ошибки диспетчера каталогов

В этом разделе содержатся возвращаемые значения для ошибок диспетчера каталогов из функций в домен Active Directory Services.



| Значение                    | Описание                                              |
|--------------------------|----------------------------------------------------------|
| **хрпмрекделетед**       | Запись была удалена.                                  |
| **хрремаинингверсионс**  | Осталось неактивная работа.                            |
| **хрфлдкэйтубиг**       | Ключ был усечен, так как он превышает 255 байт. |
| **хрфлдтуманисегментс** | Слишком много ключевых сегментов.                         |
| **хрфлднуллкэй**         | Ключ имеет **значение NULL**.                                     |
