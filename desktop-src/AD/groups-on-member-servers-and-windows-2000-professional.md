# <a name="groups-on-member-servers-and-windows-2000-professional"></a>группы на рядовых серверах и Windows 2000 Professional

на рядовых серверах и компьютерах, работающих на Windows 2000 Professional, существует локальная база данных безопасности. Эта локальная база данных безопасности может содержать собственные локальные пользователи и локальные группы, областью действия которых является только конкретный компьютер, на котором они созданы. при управлении этими типами пользователей и групп на рядовых серверах и компьютерах, работающих на Windows NT Workstation или Windows 2000 Professional, используйте поставщик WinNT.

если рядовой сервер или компьютер, работающий под Windows 2000 Professional или Windows 2000 Professional, является членом домена Windows 2000, группы или пользователи в домене можно использовать в локальной базе данных безопасности для предоставления прав на эту группу на этом конкретном компьютере.

при управлении группами в домене Windows 2000 с помощью ADSI используйте поставщик LDAP. при управлении группами на рядовых серверах и компьютерах, работающих на Windows NT Workstation или Windows 2000 Professional, используйте поставщик WinNT.

Необходимо выполнить привязку, по крайней мере, по одному экземпляру к каждому поставщику: 1) выполните привязку к поставщику LDAP, чтобы получить путь к группе или пользователю, который требуется добавить в группу в локальной базе данных, и 2) привяжите к поставщику WinNT, чтобы добавить этого пользователя или группу в локальную группу.
