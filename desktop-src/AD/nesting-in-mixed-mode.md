# <a name="nesting-in-mixed-mode"></a>Вложение в смешанном режиме

В этом разделе перечислены ограничения групп безопасности в домене смешанного режима.

Группы безопасности в домене в смешанном режиме имеют следующие ограничения.

-   Универсальные группы нельзя создавать в доменах в смешанном режиме, так как универсальная область поддерживается только в доменах Windows 2000 в основном режиме.
-   Глобальная группа может содержать учетные записи из того же домена, к которому принадлежит группа. Глобальная группа не может содержать универсальные группы, любые глобальные группы или учетные записи из другого домена.
-   Локальная группа домена может содержать глобальные группы и учетные записи из любого домена или леса. Локальная группа домена не может содержать другие локальные группы домена.

Имейте в виду, что группы распространения могут быть вложены в соответствии с правилами вложенности в основном режиме.
