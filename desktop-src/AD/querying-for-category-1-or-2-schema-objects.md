# <a name="querying-for-category-1-or-2-schema-objects"></a>Запрос для объектов схемы категории 1 или 2

Атрибут **systemFlags** объектов **attributeSchema** и **classSchema** представляет собой целочисленную битовую маску, которая содержит флаги, указывающие дополнительные системные характеристики атрибута или класса. Перечисление [**ADS \_ системфлаг \_ enum**](/windows/win32/api/iads/ne-iads-ads_systemflag_enum) содержит значения, которые соответствуют битам, которые можно задать в атрибуте **systemFlags** . Можно задать дополнительные биты **systemFlags** , например 0x10 бит, указывающий, относится ли атрибут или класс к категории 1 или категории 2. Бит 0x10 установлен для объектов Category 1, которые являются классами и атрибутами, включенными в основную схему, включенную в систему. Бит не задан для атрибутов и классов категории 2, которые являются расширениями схемы. Если для объекта **attributeSchema** или **classSchema** не существует свойства **systemFlags** , это категория 2.

**\_ \_ Бит правила сопоставления LDAP \_ \_ и** правило сопоставления можно использовать для поиска объектов, для которых в атрибуте **systemFlags** установлен флаг 0x10. Дополнительные сведения см. в разделе [синтаксис фильтра поиска](/windows/desktop/ADSI/search-filter-syntax).

## <a name="querying-for-category-1"></a>Запрос для категории 1

Следующая строка запроса ищет атрибуты категории 1 (объекты **attributeSchema** с битом 0x10, заданным в свойстве **systemFlags** ).


```C++
(&(objectCategory=attributeSchema)(systemFlags:1.2.840.113556.1.4.803:=16) )
```



Имейте в виду, что в приведенном выше примере синтаксис запроса LDAP требует наличия десятичных значений. Поэтому шестнадцатеричное значение флага должно быть преобразовано в тип Decimal. В этом случае категория 1 bit имеет значение 0x10, поэтому значением фильтра должно быть 16.

## <a name="querying-for-category-2"></a>Запрос для категории 2

Следующая строка запроса ищет атрибуты категории 2 (объекты **attributeSchema** , для которых в свойстве **systemFlags** не задан бит 0x10).


```C++
(&(objectCategory=attributeSchema)(!(systemFlags:1.2.840.113556.1.4.803:=16)))
```



Имейте в виду, что этот запрос также возвращает объекты **attributeSchema** , у которых нет свойства **systemFlags** , и, следовательно, неявным образом не задан заданный флаг.
