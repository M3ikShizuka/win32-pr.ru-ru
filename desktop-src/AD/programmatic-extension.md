# <a name="programmatic-extension"></a>Программное расширение

Преимуществом LDIF-файла является то, что администратор может проверить свою функцию. Однако схему также можно расширить программным способом. Программные расширения имеют несколько разрешений, но приложение непрозрачно; Администратор должен доверять документации, входящей в состав приложения установки. Использование программных расширений схемы может быть препятствием для развертывания приложений, которые их используют. Программное расширение является инвариантным; это Windows NT исполняемый файл. Двоичный файл не может быть изменен, в отличие от файла LDIF или .csv, который можно изменить случайно или злонамеренно.

Ниже перечислены преимущества LDIF-файла.

-   Приложения могут обнаруживать и восстанавливать ошибки, а также предоставлять отзывы пользователей.
-   Приложения могут работать в Юникоде без необходимости кодирования в кодировке Base64.
-   приложения могут использовать установщик Windows api установки.
-   Для установления подлинности приложения могут подписываться.
