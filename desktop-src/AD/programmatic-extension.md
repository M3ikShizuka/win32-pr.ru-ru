# <a name="programmatic-extension"></a>Программное расширение

Преимуществом LDIF-файла является то, что администратор может проверить свою функцию. Однако схему также можно расширить программным способом. Программные расширения имеют несколько разрешений, но приложение непрозрачно; Администратор должен доверять документации, входящей в состав приложения установки. Использование программных расширений схемы может быть препятствием для развертывания приложений, которые их используют. Программное расширение является инвариантным; Это исполняемый файл Windows NT. Двоичный файл не может быть изменен, в отличие от LDIF-или CSV-файла, который может быть изменен случайно или злонамеренно.

Ниже перечислены преимущества LDIF-файла.

-   Приложения могут обнаруживать и восстанавливать ошибки, а также предоставлять отзывы пользователей.
-   Приложения могут работать в Юникоде без необходимости кодирования в кодировке Base64.
-   Приложения могут использовать установщик Windows API установки.
-   Для установления подлинности приложения могут подписываться.
