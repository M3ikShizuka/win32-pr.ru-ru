# <a name="structural-abstract-and-auxiliary-classes"></a>Структурные, абстрактные и вспомогательные классы

Атрибут **objectClassCategory** объекта **classSchema** может иметь значение, как указано в следующей таблице, которое указывает, является ли класс структурным, абстрактным или вспомогательным.



| Значение | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|-------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1     | Структурный класс, который является единственным типом класса, который может иметь экземпляры в домен Active Directory Services. Структурный класс может быть подклассом абстрактного или структурного класса. Структурный класс может включать любое количество вспомогательных классов в его определении.                                                                                                                                                                                                                                           |
| 2     | Абстрактный класс, который является шаблоном, используемым для наследования новых абстрактных, вспомогательных и структурных классов. Абстрактный класс может быть только подклассом абстрактного класса. Не удается создать экземпляры абстрактных классов в службах домен Active Directory. Абстрактный класс может включать любое количество вспомогательных классов в его определении.                                                                                                                                                                                   |
| 3     | Вспомогательный класс, который может быть включен в определение структурного, абстрактного или вспомогательного класса, в этом случае значения **mustContain**, **системмустконтаин**, **mayContain** и **системмайконтаин** вспомогательного класса добавляются к элементам класса. Вспомогательный класс может быть подклассом абстрактного или вспомогательного класса. Экземпляры вспомогательных классов не могут быть созданы в службах домен Active Directory. Вспомогательный класс может включать любое количество вспомогательных классов в его определении. |



 

Не путайте **objectClassCategory** с [категорией объектов](object-class-and-object-category.md).
