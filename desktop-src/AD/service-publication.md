# <a name="service-publication"></a>Публикация службы

Службы объявляются с помощью объектов, хранящихся в службах домен Active Directory. Это называется *публикацией службы*. Клиенты запрашивают каталог для поиска служб. Это называется *встречей Client-Service*. В этом разделе обсуждаются типы объектов каталога, используемые для публикации службы, и объясняется, как они используются для встреч с клиентскими службами.
