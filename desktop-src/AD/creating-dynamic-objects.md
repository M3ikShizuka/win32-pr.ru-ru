# <a name="creating-dynamic-objects"></a>Создание динамических объектов

Укажите вспомогательный класс **DynamicObject** в качестве значения атрибута **objectClass** во время создания экземпляра объекта. В этом случае специальный вспомогательный класс будет отображаться в списке классов, который является значением атрибута **objectClass** для объекта. Если значение атрибута **ентриттл** не указано во время добавления специального вспомогательного класса, значение по умолчанию, как указано позже, будет предоставлено службами домен Active Directory Services.

Дополнительные сведения о добавлении вспомогательного класса в объект см. в разделе [динамические вспомогательные классы](dynamic-auxiliary-classes.md).
