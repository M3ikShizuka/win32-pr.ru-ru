# <a name="access-rights-for-active-directory-domain-services-objects"></a>Права доступа для объектов служб домен Active Directory Services

Все объекты каталога используют тот же набор предопределенных прав доступа, который соответствует общим операциям с каталогом. Эти права описаны в статье тип перечисления " [**\_ \_ права на рекламу**](/windows/win32/api/iads/ne-iads-ads_rights_enum) ".

Кроме того, можно определить закрытые права доступа (Управление правами доступа) для операций, которые выходят за рамки стандартных прав. См. раздел [Управление правами доступа](control-access-rights.md).
