# <a name="dsbackupgetbackuplogs-function"></a>Функция Дсбаккупжетбаккуплогс

\[Эта функция доступна для использования в операционных системах, указанных в разделе требования. В последующих версиях он может быть изменен или недоступен. начиная с Windows Vista используйте вместо этого [служба теневого копирования томов (VSS)](../vss/volume-shadow-copy-service-overview.md) .\]

Функция **дсбаккупжетбаккуплогс** получает список файлов журнала, для которых необходимо создать резервную копию для данного контекста резервного копирования.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT DsBackupGetBackupLogs(
  _In_  HBC     hbc,
  _Out_ LPTSTR  *pszBackupLogFiles,
  _Out_ LPDWORD pcbSize
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ХБК* \[ окне\]
</dt> <dd>

Содержит описатель контекста резервного копирования, полученный с помощью функции [**дсбаккуппрепаре**](dsbackupprepare.md) .

</dd> <dt>

*псзбаккуплогфилес* \[ заполняет\]
</dt> <dd>

Указатель на строку, получающий список имен файлов журналов в виде UNC-путей. Перед вызовом **дсбаккупжетбаккуплогс** инициализируйте это значение значением **null** .

Этот список получает список из одиночных строк, заканчивающихся нулем, заканчивающийся символом NULL.

Этот буфер выделяется функцией **дсбаккупжетбаккуплогс** и должен быть освобожден, если он больше не требуется путем вызова функции [**дсбаккупфри**](dsbackupfree.md) .

Первый символ каждого из имен файлов содержит одну из [**констант БФТ**](bft-constants.md) , определяющих тип имени.

</dd> <dt>

*пкбсизе* \[ заполняет\]
</dt> <dd>

Указатель на значение **DWORD** , которое получает размер буфера *псзбаккуплогфилес* в байтах.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение \_ ОК** , если функция выполнена успешно, или код ошибки Win32 или RPC в противном случае. В следующем списке перечислены другие возможные коды ошибок.

<dl> <dt>

**Ошибка \_ отказа в доступе \_**
</dt> <dd>

Вызывающий объект не имеет необходимых прав доступа для вызова этой функции. Функцию [**дссетаусидентити**](dssetauthidentity.md) можно использовать для задания учетных данных, используемых для функций резервного копирования и восстановления.

</dd> <dt>

**Ошибка \_ недопустимого \_ параметра**
</dt> <dd>

Недопустимый *ХБК*, *псзбаккуплогфилес* или *пкбсизе* .

</dd> <dt>

**Ошибка \_ не \_ хватает \_ памяти**
</dt> <dd>

Ошибка при выделении памяти.

</dd> </dl>

## <a name="remarks"></a>Remarks

Функция **дсбаккупжетбаккуплогс** предоставляет список файлов журналов, необходимых для резервного копирования. Полная резервная копия состоит из файлов базы данных, предоставляемых функцией [**дсбаккупжетдатабасенамес**](dsbackupgetdatabasenames.md) и файлами журнала. Добавочное резервное копирование Active Directory серверов не поддерживается.
