# <a name="class-icons"></a>Значки классов

Значок, используемый для представления объекта класса, можно указать в атрибуте **iconPath** контейнера дисплайспеЦифиерс. Более того, каждый класс может хранить несколько состояний значков. Например, класс папки может иметь значки для открытых, закрытых и отключенных состояний. Текущая реализация принимает максимум шестнадцати различных состояний значков для каждого класса.

Атрибут **iconPath** можно указать одним из двух способов.


```C++
<state>,<icon file name>
```



или диспетчер конфигурации служб


```C++
<state>,<module file name>,<resource ID>
```



В этих примерах " &lt; State &gt; " — это целое число со значением от 0 до 15. Значение 0 определяется по умолчанию или закрытому состоянию значка. Значение 1 определяется как открытое состояние значка. Значение 2 — отключенное состояние. Все остальные значения определяются приложением.

" &lt; Имя файла значка &gt; " — это путь и имя файла значка, содержащего изображение значка.

" &lt; Имя файла модуля &gt; " — это путь и имя файла модуля, например EXE или DLL, который содержит изображение значка в ресурсе. " &lt; Идентификатор ресурса &gt; " — это целое число, которое указывает идентификатор ресурса значка в модуле.

## <a name="adding-a-value-to-the-iconpath-attribute"></a>Добавление значения к атрибуту **iconPath**

Чтобы добавить значение в атрибут **iconPath** , выполните следующие действия.

1.  Определить, существует ли значение атрибута. Если значение должно быть заменено, сначала удалите существующее значение с помощью метода [**iAds::P утекс**](/windows/desktop/api/iads/nf-iads-iads-putex) с параметром *лнконтролкоде* , для которого установлено **свойство ADS \_ \_** , а для параметра *впроп* — значение, которое необходимо удалить. Не используйте свойство **ADS \_ \_ Очистка** или **\_ \_ Обновление свойства ADS** для *лнконтролкоде*.
2.  Создайте строку, представляющую данные значка атрибута. Пример см. в приведенном выше формате.
3.  Чтобы добавить новое значение, используйте метод [**iAds::P утекс**](/windows/desktop/api/iads/nf-iads-iads-putex) с параметром *лнконтролкоде* , для которого установлено **значение \_ AD \_ append**.
4.  Чтобы зафиксировать изменения в каталоге, вызовите функцию [**iAds:: сетинфо**](/windows/desktop/api/iads/nf-iads-iads-setinfo).
