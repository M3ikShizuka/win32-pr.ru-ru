# <a name="distributing-user-interface-components"></a>Распространение компонентов пользовательского интерфейса

Компоненты COM, приложения и (или) файлы, используемые расширением пользовательского интерфейса, можно распространять путем создания пакета установщик Windows и использования групповой политики для распространения пакета. Дополнительные сведения о создании пакета установщика см. в разделе [установщик Windows](/windows/desktop/Msi/windows-installer-portal). Дополнительные сведения о групповой политике см. в разделе [Групповая политика](/previous-versions/windows/desktop/Policy/group-policy-start-page).
