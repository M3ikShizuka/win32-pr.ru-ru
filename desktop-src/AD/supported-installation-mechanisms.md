# <a name="supported-installation-mechanisms"></a>Поддерживаемые механизмы установки

Службы домен Active Directory Services поддерживают четыре механизма расширения схемы Active Directory. Схему Active Directory можно расширить с помощью

-   [LDIF-сценарии](ldif-scripts.md) (предпочтительно)
-   [Скрипты значений с разделителями-запятыми (CSV)](comma-separated-value-csv-scripts.md)
-   [Программное расширение](programmatic-extension.md)
-   и путем [расширения пользовательского интерфейса для объектов каталога](extending-the-user-interface-for-directory-objects.md).
