# <a name="windows-machine-learning-interfaces"></a>Интерфейсы Машинное обучение Windows

В следующих разделах описываются COM-интерфейсы для пакета SDK для Windows Машинное обучение.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                             | Описание                                                                                                                                                                                |
|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ивинмлевалуатионконтекст**](/windows/win32/api/winml/nn-winml-iwinmlevaluationcontext)<br/> | Представляет контекст для привязки входных и выходных данных к модели WinML.<br/>                                                                                                             |
| [**ивинмлмодел**](/windows/win32/api/winml/nn-winml-iwinmlmodel)<br/>                         | Представляет модель Машинное обучение Windows с соответствующими метаданными; включает описания моделей (имя, автор, управление версиями и т. д.), а также ожидаемые входные и выходные данные. <br/> |
| [**ивинмлрунтиме**](/windows/win32/api/winml/nn-winml-iwinmlruntime)<br/>                     | Представляет среду выполнения для загрузки и вычисления модели WinML.<br/>                                                                                                                      |
| [**ивинмлрунтимефактори**](/windows/win32/api/winml/nn-winml-iwinmlruntimefactory)<br/>       | Представляет фабрику, которая создает среду выполнения WinML для загрузки и оценки модели.<br/>                                                                                         |
