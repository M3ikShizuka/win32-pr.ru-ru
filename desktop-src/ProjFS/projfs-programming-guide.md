# <a name="projected-file-system-projfs-programming-guide"></a>Руководством по программированию для проектируемой файловой системы (Прожфс)

Windows проектируемая файловая система (прожфс) позволяет приложению пользовательского режима, называемому "поставщиком", выполнять проецирование иерархических данных в файловую систему, что делает их отображаемыми в файловой системе как файлами и каталогами.
