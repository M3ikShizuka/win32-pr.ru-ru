# <a name="projfs-enumerations"></a>Перечисления Прожфс

Следующие перечисления объявляются в прожектедфслиб. h.

## <a name="in-this-section"></a>В этом разделе

| Раздел | Описание |
|-|-|
| [**PRJ_CALLBACK_DATA_FLAGS**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_callback_data_flags) | Флаги, управляющие тем, что возвращается в перечислении. |
| [**PRJ_COMPLETE_COMMAND_TYPE**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_complete_command_type) | Указывает типы команд. |
| [**PRJ_FILE_STATE**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_file_state) | Состояние элемента. |
| [**PRJ_NOTIFICATION**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_notification) | Значение уведомления, указанное при отправке уведомления в обратном вызове. |
| [**PRJ_NOTIFY_TYPES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_notify_types) | Типы уведомлений, описывающие изменение файла или папки. |
| [**PRJ_PLACEHOLDER_ID**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_placeholder_id) | Определяет длину идентификатора заполнителя. |
| [**PRJ_STARTVIRTUALIZING_FLAGS**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_startvirtualizing_flags) | Флаги, которые необходимо указать при запуске экземпляра виртуализации. |
| [**PRJ_UPDATE_FAILURE_CAUSES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_update_failure_causes) | Описание причины сбоя обновления. |
| [**PRJ_UPDATE_TYPES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_update_types) | Флаги, указывающие, разрешено ли обновления с учетом состояния файла или каталога на диске. |
