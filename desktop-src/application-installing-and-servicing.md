# <a name="application-installation-and-servicing"></a>Установка и обслуживание приложения

Использование доступных интерфейсов API и служб, предоставляемых Windows, для установки, управления и обслуживания классических приложений.

## <a name="in-this-section"></a>В этом разделе



| Раздел | Описание |
| --- | --- |
| [Обозреватель игр](/previous-versions/windows/desktop/legacy/hh437965(v=vs.85)) | Описывает использование обозревателя игр для отображения игр в Windows. |
| [Изолированные приложения и параллельные сборки](/windows/desktop/SbsCs/isolated-applications-and-side-by-side-assemblies-portal) | Изолированные приложения и параллельные сборки — это решение Microsoft Windows, уменьшающее конфликты управления версиями в клиентских приложениях Windows. |
| [Упаковка, развертывание и запрос приложений Windows](/windows/desktop/appxpkg/appx-portal) | Программное создание пакетов приложений Windows (в том числе UWP и классических приложений), а также установку, обновление, запрос и удаление приложений. |
| [Диспетчер перезапуска](/windows/desktop/RstMgr/restart-manager-portal) | API диспетчера перезапуска может сократить или уменьшить число перезапусков системы, необходимых для завершения установки или обновления. |
| [Обновление Orchestrator](./updateorchestrator/index.md) | API обновления Orchestrator планирует фоновое обновление программного обеспечения с учетом влияния на пользователя. | 
| [Установщик Windows](/windows/desktop/Msi/windows-installer-portal) | Microsoft установщик Windows — это служба установки и настройки, предоставляемая в Windows. Служба установщика позволяет клиентам обеспечить лучшее корпоративное развертывание и предоставляет стандартный формат для управления компонентами. Установщик также включает объявление приложений и функций в соответствии с операционной системой. |
| [программа установки Windows и миграция](/previous-versions/windows/desktop/wnf/windows-setup-and-migration-portal) | В этом разделе описываются уведомления, используемые для обнаружения и, возможно, восстановления приложения после завершения установки или миграции. Эти уведомления также можно использовать для приостановки операций во время временной настройки или миграции. |
