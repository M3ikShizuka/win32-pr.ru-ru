# <a name="description-wlanpolicy-element"></a>Элемент Description (Вланполици)

Элемент Description (Вланполици) содержит описание политики беспроводной локальной сети.

``` syntax
<xs:element name="description"
    type="nameType"
 />
```

Элемент **Description** определяется элементом [**вланполици**](wlan-policyschema-wlanpolicy-element.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows \[Только классические приложения Vista\]<br/>       |
| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
