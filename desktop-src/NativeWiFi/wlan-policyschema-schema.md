# <a name="wlan_policy-schema"></a>\_Схема политики WLAN

\_Схема политики WLAN определяет профиль политики WLAN, используемый собственной службой автонастройки WiFi.

Корневым элементом профиля политики WLAN является элемент [**вланполици**](wlan-policyschema-wlanpolicy-element.md) . Каждый профиль будет иметь только один корневой элемент. Все \_ элементы политики WLAN находятся в пространстве имен `https://www.microsoft.com/networking/WLAN/policy/v1` .

-   [\_Элементы ПОЛИЦИСЧЕМА WLAN](wlan-policyschema-elements.md)
-   [Простые типы беспроводных сетей \_ полицисчема](wlan-policyschema-simple-types.md)
-   [\_Сложные типы WLAN полицисчема](wlan-policyschema-complex-types.md)
