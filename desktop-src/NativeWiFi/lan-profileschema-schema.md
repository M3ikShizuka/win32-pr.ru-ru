# <a name="lan_profile-schema"></a>\_Схема профиля локальной сети

\_Схема профиля локальной сети определяет сетевой профиль проводной сети (LAN). Профиль локальной сети содержит сетевые параметры для интерфейса Ethernet.

Корневым элементом профиля локальной сети является элемент [**ланпрофиле**](lan-profileschema-lanprofile-element.md) . Каждый профиль будет иметь только один корневой элемент. Все \_ элементы профиля локальной сети находятся в пространстве имен `https://www.microsoft.com/networking/LAN/profile/v1` .

Чтобы просмотреть примеры профилей локальной сети, см. раздел [образцы проводных профилей](wired-profile-samples.md).

-   [\_Элементы схемы профиля локальной сети](lan-profileschema-elements.md)
