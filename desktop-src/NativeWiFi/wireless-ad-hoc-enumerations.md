# <a name="wireless-ad-hoc-enumerations"></a>Беспроводные нерегламентированные перечисления

Беспроводный нерегламентированный интерфейс определяет и использует следующие перечисления.

-   [**\_ \_ Алгоритм проверки подлинности прямого соединения DOT11 \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_auth_algorithm)
-   [**\_ \_ Алгоритм шифрования прямого соединения DOT11 \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_cipher_algorithm)
-   [**\_ \_ \_ Причина сбоя прямого подключения \_ DOT11**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_connect_fail_reason)
-   [**\_ \_ \_ Состояние подключения к сети прямого соединения DOT11 \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_network_connection_status)

> [!Note]  
> Нерегламентированный режим может быть недоступен в будущих версиях Windows. начиная с Windows 8.1 и Windows Server 2012 R2, используйте вместо этого [Wi-Fi Direct](about-the-wi-fi-direct-api.md) .
