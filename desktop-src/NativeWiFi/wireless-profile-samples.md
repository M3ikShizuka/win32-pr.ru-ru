# <a name="wireless-profile-samples"></a>Образцы профиля беспроводной связи

В следующих разделах приведены примеры XML-профилей для различных конфигураций беспроводных сетей. Эти профили XML используют [ \_ элементы схемы профиля WLAN](wlan-profileschema-elements.md) и [элементы схемы OneX](onexschema-elements.md).

Вы можете добавить профиль беспроводной сети в хранилище профилей программным путем, вызвав [**влансетпрофиле**](/windows/desktop/api/wlanapi/nf-wlanapi-wlansetprofile). Вы также можете добавить профиль в хранилище профилей с помощью служебной программы командной строки **netsh** . Дополнительные сведения см. в разделе [команды Netsh для беспроводной локальной сети (WLAN)](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc755301(v=ws.10)).
