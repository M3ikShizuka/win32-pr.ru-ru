# <a name="authmode-onex-element"></a>Элемент Аусмоде (OneX)

Элемент Аусмоде (OneX) указывает тип учетных данных, используемых для проверки подлинности. В следующей таблице описаны возможные значения.



| Значение         | Описание                                                                                                                                                             |
|---------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| мачинеорусер | Используйте учетные данные компьютера или пользователя. При входе пользователя в систему для проверки подлинности используются учетные данные пользователя. Если пользователь не вошел в систему, используются учетные данные компьютера. |
| компьютер       | Использовать только учетные данные компьютера.                                                                                                                                           |
| пользователь          | Используйте только учетные данные пользователя.                                                                                                                                              |
| guest         | Используйте только гостевые (пустые) учетные данные.                                                                                                                                     |



 

Этот элемент является необязательным. Если Аусмоде не указан в профиле, `machineOrUser` используется значение.

**Windows XP с SP3 и API беспроводной локальной сети для Windows XP с пакетом обновления 2 (SP2):** Этот элемент будет пропущен, если он есть в профиле.

``` syntax
<xs:element name="authMode">
    <xs:simpleType>
        <xs:restriction
            base="string"
        >
            <xs:enumeration
                value="machineOrUser"
             />
            <xs:enumeration
                value="machine"
             />
            <xs:enumeration
                value="user"
             />
            <xs:enumeration
                value="guest"
             />
        </xs:restriction>
    </xs:simpleType>
</xs:element>
```

Элемент **аусмоде** определяется элементом [**OneX**](onexschema-onex-element.md) .

## <a name="remarks"></a>Комментарии

Этот параметр можно задать в командной строке с помощью команды **netsh wlan set профилепараметер** . Дополнительные сведения см. в разделе [команды Netsh для беспроводной локальной сети (WLAN)](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc755301(v=ws.10)).
