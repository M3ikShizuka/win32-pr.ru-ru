# <a name="about-native-wifi"></a>О встроенных WiFi

Собственные функции интерфейса прикладного программирования (API) Wi-Fi имеют две цели: Управление профилями беспроводной сети и Управление беспроводными сетевыми подключениями. Элементы API предоставляются модулем автоматической настройки (ACM). Предоставленные элементы API соединения и отключения можно использовать для переопределения логики автоматической настройки.
