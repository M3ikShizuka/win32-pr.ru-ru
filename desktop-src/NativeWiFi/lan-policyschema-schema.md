# <a name="lan_policy-schema"></a>\_Схема политики локальной сети

\_Схема политики локальной сети определяет профиль политики проводной сети.

Корневым элементом профиля политики локальной сети является элемент [**ланполици**](lan-policyschema-lanpolicy-element.md) . Каждый профиль имеет только один корневой элемент. Все \_ элементы политики локальной сети находятся в пространстве имен `https://www.microsoft.com/networking/LAN/policy/v1` .

-   [\_Элементы полицисчема локальной сети](lan-policyschema-elements.md)
-   [\_Простые типы полицисчема локальной сети](lan-policyschema-simple-types.md)
