# <a name="network-blocklist-element"></a>Элемент Network (списка блокировки)

Элемент Network (списка блокировки) определяет Заблокированную сеть. Компьютеру не удается подключиться к заблокированной сети.

``` syntax
<xs:element name="network"
    type="networkItemType"
 />
```

Элемент **Network** определяется элементом [**списка блокировки**](wlan-policyschema-blocklist-networkfilter-element.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows \[Только классические приложения Vista\]<br/>       |
| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
