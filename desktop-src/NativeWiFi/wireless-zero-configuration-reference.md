в этом разделе содержатся сведения об интерфейсе программирования для службы настройки беспроводной связи в Windows XP и Windows Server 2003. Статьи:

-   [Функции настройки беспроводной связи нуль](wireless-zero-configuration-functions.md)
-   [Беспроводные структуры конфигурации](wireless-zero-configuration-structures.md)

беспроводная настройка — это служба Windows на Windows XP и Windows Server 2003, которая используется для настройки беспроводных сетевых подключений и управления ими на беспроводном адаптере. Имя службы для беспроводной нулевой конфигурации — ВЗКСВК. в Windows XP отображаемое имя для службы взксвк — беспроводная нулевая настройка. в Windows Server 2003 отображаемое имя для службы взксвк — конфигурация беспроводной связи.

Служба настройки беспроводной связи обычно запускается во время загрузки. Интерфейс программирования для службы настройки беспроводной связи можно использовать, только если запущена служба настройки беспроводной связи. Если служба настройки беспроводной связи не запущена, функции беспроводной настройки будут возвращать ошибку.

Чтобы включить службу настройки беспроводной связи, чтобы она автоматически запускалась, нажмите кнопку **Пуск** . выберите параметр **Параметры** , а затем выберите **панель управления**. если вы используете представление Windows XP, выберите категорию **производительность и обслуживание** , а затем щелкните **администрирование**. Если используется классический вид, выберите **Администрирование**. Щелкните значок **службы** в левой области. Щелкните значок беспроводной настройки на правой панели и измените **Тип запуска** Dropbox на **Автоматический**. Этот параметр позволяет настроить автоматический запуск службы во время загрузки. Затем нажмите кнопку **запустить** , чтобы запустить службу беспроводной нулевой настройки Wireless Zero и нажать кнопку **ОК** .

Беспроводную конфигурацию также можно запустить и остановить из командной строки. Чтобы запустить беспроводную конфигурацию, выполните следующую команду:

**NET start взксвк**

Чтобы отключить беспроводную конфигурацию, выполните следующую команду:

**NET взксвк**
