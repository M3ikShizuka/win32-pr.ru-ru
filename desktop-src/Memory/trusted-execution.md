# <a name="trusted-execution"></a>Доверенное выполнение

Енклавес используются для создания доверенных сред выполнения. Анклава — это изолированная область кода и данных в адресном пространстве для приложения. Только код, который выполняется в анклава, может обращаться к данным в одном анклава.
