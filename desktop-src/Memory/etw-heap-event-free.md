# <a name="etw_heap_event_free-event"></a>\_ \_ Событие освобождения события кучи ETW \_

Событие **\_ \_ \_ освобождения события кучи ETW** — это событие трассировки управления памятью для операции освобождения кучи.


```C++
typedef struct ETW_HEAP_EVENT_FREE
```

<dl> <dt>

*хеафандле* 
</dt> <dd>

</dd> <dt>

*Адрес* 
</dt> <dd>

Адрес освобожденной памяти.

</dd> <dt>

*Источник*

Источник памяти, используемый распределителем для выделения кучи.
