# <a name="about-memory-management"></a>Управление памятью

каждый процесс в 32-разрядной версии Microsoft Windows имеет собственное виртуальное адресное пространство, которое позволяет адресовать до 4 гигабайт памяти. каждый процесс на 64-разрядном Windows имеет виртуальное адресное пространство, равное 8 терабайтам. Все потоки процесса могут обращаться к своему виртуальному адресному пространству. Однако потоки не могут получить доступ к памяти, принадлежащей другому процессу, что защищает процесс от повреждения другим процессом.
