# <a name="fatal-application-exit"></a>Неустранимый выход из приложения

Функция **фаталаппексит** отображает окно сообщения и завершает работу приложения, когда пользователь закрывает окно сообщения. Эта функция должна использоваться только в качестве крайней функции, так как она может не освободить память или файлы, принадлежащие приложению.
