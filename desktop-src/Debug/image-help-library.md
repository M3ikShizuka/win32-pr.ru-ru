# <a name="image-help-library"></a>Библиотека справки по изображениям

В этом обзоре описывается набор функций, предоставляемых библиотекой DLL ImageHlp. Эти функции позволяют работать с переносимым исполняемым (PE) образом.

-   [О ImageHlp](about-imagehlp.md)
-   [Справочник по ImageHlp](imagehlp-reference.md)

Невозможно повторно распространить библиотеку DLL ImageHlp, входящую в состав операционной системы. Подмножество функций включено в [библиотеку справки по отладке](debug-help-library.md), которая является рестрибутабле.
