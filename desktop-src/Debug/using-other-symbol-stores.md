# <a name="using-other-symbol-stores"></a>Использование других хранилищ символов

Можно написать собственную программу создания хранилища символов, а не использовать SymStore.

Так как транзакции SymStore все зарегистрированы в текстовых файлах в формате CSV, можно использовать любые существующие файлы журнала SymStore для использования в собственной программе для работы с базами данных.

Если вы планируете использовать программу SymSrv, предоставляемую с инструментами отладки для Windows, рекомендуется также использовать SymStore. Обновления этих двух программ всегда будут выпущены вместе, поэтому их версии будут всегда совпадать.
