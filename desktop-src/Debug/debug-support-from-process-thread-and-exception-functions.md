# <a name="debug-support-from-process-thread-and-exception-functions"></a>Поддержка отладки из функций процессов, потоков и исключений

Некоторые функции, важные для отладки, фактически классифицируются как процессы, потоки или функции обработки исключений. Описание того, как запустить процесс, который будет отлажен, см. в разделе [функции обработки](process-functions-for-debugging.md). Сведения о том, как проверить контекст и выполнение потока, а также управлять им, см. в разделе [функции потока](thread-functions-for-debugging.md). Описание того, как отладчик должен обрабатывать исключения, см. в разделе [функции обработки исключений](exception-handling-functions-for-debugging.md).
