# <a name="decorated-symbol-names"></a>Декорированные имена символов

Декорированное имя символа содержит символы, которые определяют, как был объявлен открытый символ. Для \_ \_ функций STDCALL имена включают символ "@" и десятичное число, которое указывает число байтов в его параметрах функции. Например, декорированное имя функции [**LoadLibrary**](/windows/win32/api/libloaderapi/nf-libloaderapi-loadlibrarya) — LoadLibrary@4 . Для функций C++ декорирование имен является более сложным и отличается от компилятора к компилятору.

Чтобы получить недекорированное имя символа, используйте функцию [**ундекоратесимболнаме**](/windows/desktop/api/Dbghelp/nf-dbghelp-undecoratesymbolname) . Кроме того, можно вызвать функцию [**симсетоптионс**](/windows/desktop/api/Dbghelp/nf-dbghelp-symsetoptions) , чтобы запросить, чтобы обработчик символов всегда присутствовал символы с недекорированными именами. Этот параметр необходимо задать перед загрузкой символов, поскольку обработчик символов создает таблицы имен символов во время загрузки.
