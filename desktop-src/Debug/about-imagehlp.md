# <a name="about-imagehlp"></a>О ImageHlp

Функции ImageHlp в основном используются средствами программирования, служебными программами установки приложений и другими программами, которым требуется доступ к данным, содержащимся в образе PE. Все функции ImageHlp являются отдельными потоками. Таким образом, вызовы из более чем одного потока в эту функцию, скорее всего, приведут к непредвиденному поведению или повреждению памяти. Чтобы избежать этого, необходимо синхронизировать все одновременные вызовы из более чем одного потока в эту функцию.
