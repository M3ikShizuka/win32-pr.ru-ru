# <a name="error-handling-error-handling"></a>Обработка ошибок (обработка ошибок)

Хорошо написанные приложения включают код обработки ошибок, который позволяет им корректно восстанавливаться после непредвиденных ошибок. При возникновении ошибки приложению может потребоваться запросить вмешательство пользователя или выполнить восстановление самостоятельно. В чрезвычайных случаях приложение может выйти из системы или завершить его работу.

-   [Об обработке ошибок](about-error-handling.md)
-   [Использование обработки ошибок](using-error-handling.md)
-   [Справочник по обработке ошибок](error-handling-reference.md)

Дополнительные сведения об обработке исключений см. в разделе [структурированная обработка исключений](structured-exception-handling.md).
