# <a name="avx-functions"></a>Функции AVX

Ниже перечислены функции Intel AVX.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                   | Описание                                                                                                                    |
|-------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| [**копиконтекст**](/windows/desktop/api/WinBase/nf-winbase-copycontext)<br/>                           | Копирует исходную структуру контекста (включая любые Ксстате) в инициализированную структуру контекста назначения.<br/>         |
| [**жетенабледксстатефеатурес**](/windows/desktop/api/WinBase/nf-winbase-getenabledxstatefeatures)<br/> | Возвращает маску включенных функций Ксстате на процессорах x86 или x64.<br/>                                                    |
| [**жетксстатефеатуресмаск**](/windows/desktop/api/WinBase/nf-winbase-getxstatefeaturesmask)<br/>       | Возвращает маску функций Ксстате, установленных в структуре [**контекста**](/windows/desktop/api/WinNT/ns-winnt-wow64_context) .<br/>                        |
| [**инитиализеконтекст**](/windows/desktop/api/WinBase/nf-winbase-initializecontext)<br/>               | Инициализирует структуру [**контекста**](/windows/desktop/api/WinNT/ns-winnt-arm64_nt_context) в буфере с требуемым размером и выравниванием.<br/>       |
| [**локатексстатефеатуре**](/windows/desktop/api/WinBase/nf-winbase-locatexstatefeature)<br/>           | Получает указатель на состояние процессора для функции Ксстате в структуре [**контекста**](/windows/desktop/api/WinNT/ns-winnt-arm64_nt_context) .<br/> |
| [**сетксстатефеатуресмаск**](/windows/desktop/api/WinBase/nf-winbase-setxstatefeaturesmask)<br/>       | Задает маску функций Ксстате, установленных в структуре [**контекста**](/windows/desktop/api/WinNT/ns-winnt-arm64_nt_context) .<br/>                             |
