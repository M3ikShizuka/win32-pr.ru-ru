# <a name="creating-a-basic-debugger"></a>Создание базового отладчика

[Отладка выполняющегося процесса](debugging-a-running-process.md) описывает присоединение отладчика к выполняющемуся процессу. При [написании главного цикла отладчика](writing-the-debugger-s-main-loop.md) описываются функции, используемые в главном цикле выполнения отладчика.

Хотя большинство функций отладки используются для создания отладчика, некоторые функции предназначены для использования в отлаживаемом процессе. Дополнительные сведения см. [в разделе взаимодействие с отладчиком](communicating-with-the-debugger.md).
