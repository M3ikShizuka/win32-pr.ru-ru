# <a name="notifying-the-user"></a>Уведомление пользователя

Чтобы уведомить пользователя о том, что возникла какая-либо ошибка, многие приложения просто создают звук с помощью функции [**мессажебип**](/windows/desktop/api/WinUser/nf-winuser-messagebeep) или Flash-окна, используя функцию [**FlashWindow интерфейса**](/windows/desktop/api/Winuser/nf-winuser-flashwindow) или [**флашвиндовекс**](/windows/desktop/api/Winuser/nf-winuser-flashwindowex) . Приложение также может использовать эти функции, чтобы привлечь внимание к ошибке, а затем отобразить окно сообщения или сообщение об ошибке, содержащее сведения об ошибке.
