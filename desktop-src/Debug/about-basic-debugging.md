# <a name="about-basic-debugging"></a>Основные сведения об отладке

[Функции отладки](debugging-functions.md) можно использовать для создания базового отладчика, управляемого событиями. На *основе событий* означает, что отладчик уведомляется каждый раз при возникновении определенных событий в отлаживаемом процессе. Уведомление позволяет отладчику предпринять соответствующие действия в ответ на события.

Общие сведения об отладке терминов см. в разделе [Отладка терминологии](debugging-terminology.md).

[Поддержка отладки из функций процессов, потоков и исключений](debug-support-from-process-thread-and-exception-functions.md) описывает особенности отладки определенных функций процесса, потока и обработки исключений.

[Создание базового отладчика](creating-a-basic-debugger.md) описывает использование базовых функций отладки для создания простого отладчика. Эти функции позволяют приложению ожидать события отладки, вызывать исключения точек останова, передавать управление выполнением отладчику и т. д.

[Отладка событий](debugging-events.md) описывает механизм событий отладки. События отладки приводят к тому, что система уведомляет отладчик.
