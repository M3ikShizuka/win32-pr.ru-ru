# <a name="dds_header_dxt10-structure"></a>\_Структура DXT10 заголовков DDS \_

Расширение заголовков DDS для обработки массивов ресурсов, форматов пикселей DXGI, которые не соответствуют устаревшим структурам формата пикселей Microsoft DirectDraw, и дополнительным метаданным.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct {
  DXGI_FORMAT              dxgiFormat;
  D3D10_RESOURCE_DIMENSION resourceDimension;
  UINT                     miscFlag;
  UINT                     arraySize;
  UINT                     miscFlags2;
} DDS_HEADER_DXT10;
```

## <a name="remarks"></a>Комментарии

Используйте эту структуру вместе с [**\_ заголовком DDS**](dds-header.md) для хранения массива ресурсов в файле DDS. Дополнительные сведения см. в разделе [массивы текстур](dx-graphics-dds-pguide.md).

Этот заголовок отображается, если элемент **двфауркк** структуры [**DDS \_ PIXELFORMAT**](dds-pixelformat.md) имеет значение "содержимым DX10".
