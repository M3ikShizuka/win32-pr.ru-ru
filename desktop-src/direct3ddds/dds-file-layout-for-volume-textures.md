# <a name="dds-volume-texture-example"></a>Пример текстуры тома DDS

Для текстуры тома используйте ДДСКАПС \_ Complex, DDSCAPS2 \_ том, ддсд \_ Depth, flags, Set и двдепс. Текстура тома — это расширение стандартной текстуры для Direct3D 9; текстуру тома можно определить с помощью или без MIP-карты.

Для томов без MIP-карты каждый срез глубины записывается в файл по порядку. Если включены MIP-карты, все срезы глубины для данного уровня mipmap записываются вместе, при этом каждый уровень содержит половину столько срезов, сколько на предыдущем уровне, как минимум 1.

Например, Схема тома 64-by-64-на-4 с использованием формата пикселей R8G8B8 (3 байта на пиксель) со всеми уровнями mipmap будет содержать следующее:



| Компоненты DDS                      | \# Байт    |
|-------------------------------------|-------------|
| заголовок                              | 128 байт   |
| 64-by-64 срез 1 из 4 основного образа.   | 12288 байт |
| 64-by-64 срез 2 из 4 основного образа.   | 12288 байт |
| 64-by-64 срез 3 из 4 основного образа.   | 12288 байт |
| 64-by-64 срез 4 из 4 основного образа.   | 12288 байт |
| 32-by-32 срез 1 из 2 mipmapного образа. | 3072 байт  |
| 32-by-32 срез 2 из 2 mipmap образа. | 3072 байт  |
| 16 на 16 срез 1 из 1 mipmap изображения. | 768 байт   |
| 8 на 8 срез 1 из 1 mipmap изображения.   | 192 байт   |
| 4-по 4 срез 1 из 1 mipmap образа.   | 48 байт    |
| 2-by-2 срез 1 из 1 mipmapного образа.   | 12 байт    |
| 1-by-1 срез 1 из 1 mipmap образа.   | 3 байта     |



 

Обратите внимание, что минимальный уровень mipmap составляет всего 3 байта, поскольку биткаунт имеет значение 24, а на этом уровне нет дополнительного сжатия.

Поддержка текстур томов была добавлена в DirectX 8.
