# <a name="about-the-winsock-spi"></a>Сведения об использовании Winsock SPI

Winsock предоставляет интерфейс поставщика услуг для создания служб Winsock, обычно называемый Winsock SPI. Существует два типа поставщиков услуг: поставщики транспорта и поставщики пространств имен. Примеры поставщиков транспорта включают в себя стеки протоколов, такие как TCP/IP или IPX/SPX, а пример поставщика пространства имен — интерфейс системы доменных имен (DNS) в Интернете. Отдельные разделы спецификации интерфейса поставщика услуг относятся к каждому типу поставщика услуг.

Поставщики услуг [транспорта](transport-service-providers-2.md) и [пространств имен](name-space-service-providers-2.md) должны быть зарегистрированы в \_32.dll Ws2 во время их установки. Эту регистрацию необходимо выполнять только один раз для каждого поставщика, так как необходимые сведения сохраняются в постоянном хранилище.
