# <a name="pointers-windows-sockets-2"></a>Указатели (сокеты Windows 2)

Все указатели, используемые приложениями с сокетами Windows, должны быть далеко, хотя это касается только 16-разрядных приложений и не имеет смысла в 32-бит. Чтобы упростить это, предоставляются определения типов данных, такие как **лфостент** .
