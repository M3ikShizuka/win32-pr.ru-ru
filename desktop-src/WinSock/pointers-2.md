# <a name="pointers-windows-sockets-2"></a>указатели (сокеты Windows 2)

все указатели, используемые приложениями с сокетами Windows, должны быть далеко, хотя это относится только к 16-разрядным приложениям и не имеет смысла в 32-бит. Чтобы упростить это, предоставляются определения типов данных, такие как **лфостент** .
