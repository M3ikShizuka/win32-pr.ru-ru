# <a name="multicast"></a>Многоадресная рассылка

Универсальные функции Winsock MultiPoint поддерживают многоадресную рассылку IP-адресов. Однако поставщики транспорта TCP/IP, поддерживающие многоадресную рассылку, должны также предоставлять поддержку Berkeley Software Distribution (BSD), поддерживающую соответствующие параметры многоадресной рассылки. это упрощает перенос существующих многоадресных приложений в Windows сокеты 2.
