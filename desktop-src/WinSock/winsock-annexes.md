# <a name="winsock-annexes"></a>Библиотеки Winsock

Для приложений Winsock предоставляются сведения о реализации распространенных наборов протоколов Winsock и обсуждается использование каждого протокола с Winsock.

Каждый протокол в этом приложении имеет соглашения, поведения или специальные функции, которые не предоставляются для универсальной реализации. В этом разделе описываются сведения, которые необходимо учитывать разработчикам при реализации или использовании описанных протоколов.

В число этих подразделов входят следующие:

-   [Приложение Winsock ATM](winsock-atm-annex.md)
-   [Приложение Winsock IPX/SPX](winsock-ipx-spx-annex.md)
-   [Приложение Winsock TCP/IP](winsock-tcp-ip-annex.md)
