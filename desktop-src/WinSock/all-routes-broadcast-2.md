# <a name="all-routes-broadcast"></a>Широковещательная рассылка всех маршрутов

Общая широковещательная рассылка через Интернет достигается путем установки полей **SA \_ нетнум** и **SA \_ ноденум** в двоичные файлы (1). Поставщик услуг преобразует этот запрос в пакет типа 20, который распознает и пересылает IPX-маршрутизаторы. Пакет посещает все подсети и может дублироваться несколько раз. Приемники должны поддерживать несколько повторяющихся копий датаграммы.

Использование этого типа вещания не является популярным среди сетевых администраторов, поэтому его использование должно быть чрезвычайно ограниченным. Многие маршрутизаторы отключают такой тип вещания, в результате чего части подсети невидимы для пакета.
