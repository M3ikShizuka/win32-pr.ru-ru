# <a name="summary-of-namespace-provider-functions"></a>Общие сведения о функциях поставщика пространства имен

Функции поставщика службы пространства имен можно сгруппировать в пять категорий:

-   Конфигурация поставщика пространства имен (и установка)
-   Инициализация поставщика
-   Установка службы
-   Клиентские запросы
-   Вспомогательные функции (и макросы)

В следующих разделах определяются функции в каждой категории и кратко описывается их предполагаемое использование. Также описаны основные структуры данных.

-   [Инициализация и очистка поставщика пространства имен](name-space-provider-initialization-and-cleanup-2.md)
-   [Установка службы в списке SPI Windows Sockets 2](service-installation-in-the-windows-sockets-2-spi-2.md)
-   [Запрос на обслуживание](service-query-2.md)
