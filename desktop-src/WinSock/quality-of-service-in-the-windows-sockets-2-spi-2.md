# <a name="quality-of-service-in-the-windows-sockets-2-spi"></a>качество обслуживания в Windows сокетах 2 SPI

качество обслуживания (qos) реализовано в Windows с помощью различных компонентов QoS, поддерживаемых Windows Server 2003, Windows XP и Windows 2000. подробное описание качества обслуживания и его компонентов см. в разделе, посвященном качеству обслуживания, который находится в пакете Microsoft Windows Software Development Kit (SDK). Каждый компонент QoS и роль, которую он играет в реализации QoS, описаны в этом разделе, в котором содержатся дополнительные рекомендации по реализации приложений и служб, поддерживающих QoS. более подробные сведения см. в разделе [качество обслуживания (QoS)](/previous-versions/windows/desktop/qos/qos-start-page) Windows SDK.

В этом разделе описываются возможности качества обслуживания, доступные для разработчиков Winsock. В следующем списке перечислены подразделы этого раздела.

-   [Модель использования](usage-model-2.md)
-   [Обновления QoS](qos-updates-2.md)
-   [Значения по умолчанию в SPI](default-values-in-the-spi-2.md)
-   [Шаблоны качества обслуживания в SPI](qos-templates-in-the-spi-2.md)
