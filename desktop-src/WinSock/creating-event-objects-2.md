# <a name="creating-event-objects"></a>Создание объектов событий

Ws2 \_32.dll предоставляет средства для создания объектов событий как для приложений, так и для поставщиков служб, хотя в большинстве случаев объекты событий будут создаваться приложениями. службы объектов событий доступны для поставщиков служб Windows sockets через [**впукреативент**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpucreateevent) просто в качестве удобного механизма для любой внутренней обработки, которая может оказаться полезной. Обратите внимание, что обработчик объекта события допустим только в контексте вызывающего процесса. в средах Windows реализация объектов событий осуществляется через собственные службы событий, предоставляемые операционной системой.
