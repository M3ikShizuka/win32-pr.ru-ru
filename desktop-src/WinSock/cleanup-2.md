# <a name="cleanup"></a>Очистка

32.dll Ws2 \_ (и многоуровневые протоколы) будут вызывать [**вспклеануп**](/previous-versions/windows/hardware/network/ff566270(v=vs.85)) один раз для каждого вызова [**вспстартуп**](/windows/desktop/api/Ws2spi/nf-ws2spi-wspstartup). При каждом вызове **вспклеануп** должен уменьшить счетчик ссылок на процесс, а когда счетчик достигнет нуля, поставщик услуг должен подготовиться к выгрузке из памяти. Первый способ — завершить передачу неотправленных данных на сокеты, настроенные для корректного закрытия. После этого все ресурсы, удерживаемые поставщиком, будут освобождены. Поставщик услуг должен оставаться в состоянии, где его можно немедленно повторно инициализировать с помощью вызова **вспстартуп**.
