# <a name="winsock-ipxspx-annex"></a>Приложение Winsock IPX/SPX

В этом разделе описываются расширения Winsock, относящиеся к обмену пакетами с Exchange/виртуализированными пакетами (IPX/SPX). В нем также описываются аспекты базовых функций Winsock, которые либо потребовали особого внимания, либо могут демонстрировать уникальное поведение.



| Элемент          | Описание                                                                                                                                     |
|------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| Имена протоколов | IPX, SPX                                                                                                                                        |
| Описание      | Предоставляет транспортные службы на уровне сети IPX: IPX для ненадежных датаграмм, SPX для надежных потоков сообщений, ориентированных на подключение. |
| Семейство адресов   | AF \_ IPX                                                                                                                                         |
| Файл заголовка      | Всипкс. h                                                                                                                                         |



 

В этом разделе описывается использование Winsock с семейством протоколов IPX, что позволяет использовать традиционные приложения IPX для переноса в Winsock. Сети IPX работают так же, как и IP-сети, поэтому при использовании IPX/SPX следует учитывать такие различия.
