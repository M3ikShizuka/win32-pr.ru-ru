# <a name="using-sendto-while-connected"></a>Использование функции SendTo при подключении

[**Вспсендто**](/previous-versions/windows/desktop/legacy/ms742291(v=vs.85)) всегда доставляет данные по указанному адресу, несмотря на то, что в [**вспконнект**](/previous-versions/windows/hardware/network/ff566275(v=vs.85))был установлен назначенный кэширующий узел для отправляющего сокета.
