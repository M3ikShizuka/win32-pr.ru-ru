# <a name="ipv6-automatic-tunneling-and-6to4"></a>Автоматическое туннелирование IPv6 и 6to4

Автоматическое туннелирование с IPv4-совместимыми адресами и 6to4 работают через маршрут для префикса, который находится по ссылке на интерфейс \# 2. Биты 32, следующие за префиксом, извлекаются и используются в качестве адреса назначения IPv4 для инкапсулированного пакета.

При автоматическом туннелировании используется префикс::/96, который включен по умолчанию. Его можно отключить, удалив маршрут для::/96.

6to4 использует префикс 2002::/16, который не включен по умолчанию.
