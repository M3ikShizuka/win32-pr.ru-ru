# <a name="appendix-b-ip-version-agnostic-source-code"></a>Приложение б. Независимый от IP-версии исходный код

В этом приложении показана перезаписанная версия примеров приложений Симплек. c и Simple. c, которые корректно обрабатывают IPv4 или IPv6.

-   [Код клиента с поддержкой IPv6](ipv6-enabled-client-code-2.md)
-   [Серверный код с поддержкой IPv6](ipv6-enabled-server-code-2.md)

Этот код обладает рекомендации, установленные в этом руководстве по IPv6, и включает в себя, чтобы предоставить исходный код, который был успешно изменен для добавления поддержки IPv6. Этот пример намеренно прост, но предоставляет практический пример для изучения и просмотра. Только IPv4-версия этого исходного кода предоставляется в [приложении а: исходный код только для IPv4](appendix-a-ipv4-only-source-code-2.md).

Сравнивая исходный код в приложении A (только IPv4) и приложение б (независимо от версии IP), вы получаете представление об изменениях, необходимых для изменения приложения Windows Sockets для добавления поддержки IPv6.
