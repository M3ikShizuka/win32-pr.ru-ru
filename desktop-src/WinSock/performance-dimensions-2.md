# <a name="performance-dimensions"></a>Измерения производительности

Производительность приложения зависит от перспективы. Для пользователя идеальное приложение обеспечивает высокую производительность и скорость реагирования. Для администратора идеальным приложением будет использоваться минимальный объем сетевых ресурсов. Производительность также отличается в зависимости от типа приложения. Производительность приложения для выполнения операций с массовыми файлами может отличаться от производительности транзакционного приложения.

В следующих разделах рассматриваются различные измерения производительности и типы приложений, а также влияние сетевых сред на производительность.

-   [Требования к производительности: пользователи и администраторы](performance-needs-users-and-administrators-2.md)
-   [Транзакционные и потоковые приложения](transactional-versus-streaming-applications-2.md)
-   [Разные сетевые среды](different-network-environments-2.md)
