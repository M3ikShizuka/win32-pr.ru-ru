# <a name="directed-broadcast"></a>Направленное вещание

Как правило, это больше, чем широковещательная рассылка ALL-Routes, а направленная широковещательная рассылка ограничивается указанной вами локальной сетью. Введите **SA \_ нетнум** с номером целевой сети и **SA \_ ноденум** с двоичными данными. Маршрутизаторы перенаправляют этот запрос в целевую сеть, где он преобразуется в локальную трансляцию. Промежуточные сети не видят этот пакет в качестве вещания.
