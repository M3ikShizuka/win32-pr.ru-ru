# <a name="appendix-a-ipv4-only-source-code"></a>Приложение а. исходный код только для IPv4

В этом приложении представлен исходный код для примеров программного обеспечения платформы (SDK) Симплек. c и Simple. c, которые используются в этом руководствах по IPv6 в качестве примеров:

-   [Код клиента только для IPv4](ipv4-only-client-code-2.md)
-   [Серверный код только для IPv4](ipv4-only-server-code-2.md)

> [!WARNING]
>
> Этот код включен в это руководством, чтобы сравнить независимый от IP-версии код, предоставленный в приложении б. Этот пример кода предоставляется для сравнения и предоставляет четкие и легко доступные справочные материалы во время чтения этого руководств по IPv6. Версия этого исходного кода, не зависящая от версии IP, предоставляется в [приложении б: независимый от IP-версии исходный код](appendix-b-ip-version-agnostic-source-code-2.md).

 

> [!Note]  
> Сравнивая исходный код в приложении A (только IPv4) и приложении б (независимо от версии IP), вы получаете представление об изменениях, необходимых для изменения приложения Windows Sockets для добавления поддержки IPv6.
