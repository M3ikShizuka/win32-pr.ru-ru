# <a name="winsock-tcpip-structures"></a>Структуры Winsock TCP/IP

В следующем списке приведены краткие описания каждой структуры Winsock TCP/IP. Для получения дополнительных сведений о любой структуре щелкните имя структуры.



| Структура TCP/IP                                 | Описание                                                                                                                              |
|--------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_сведения об интерфейсе**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info)      | Используется с командой IOCTL для получения сведений об IP-адресе интерфейса.                                                         |
| [**\_сведения об интерфейсе \_ ex**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info_ex) | Используется с командой **\_ ioctl получить \_ \_ список интерфейсов** для получения сведений об IP-адресе интерфейса, включая адреса IPv6. |
| [**SOCKADDR \_ Gen**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-sockaddr_gen)            | Предоставляет сведения об универсальном адресе сокета. Используется со структурой [**\_ сведений об интерфейсе**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info) .                        |
