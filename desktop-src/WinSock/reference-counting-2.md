# <a name="reference-counting"></a>Подсчет ссылок

Процесс может вызвать [**вспклосесоккет**](/previous-versions/windows/hardware/network/ff566273(v=vs.85)) на повторяющемся сокете, и дескриптор будет освобожден. Однако базовый сокет остается открытым до тех пор, пока не будет вызван **вспклосесоккет** для последнего оставшегося дескриптора.
