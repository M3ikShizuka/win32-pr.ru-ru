# <a name="ipv6-site-prefixes"></a>Префиксы сайта IPv6

Команда IPv6 рту позволяет публиковать префиксы по ссылке для настройки с длиной префикса сайта. Если задано значение, длина префикса сайта помещается в параметр сведений о префиксе в объявлениях маршрутизатора. Пример:

``` syntax
ipv6 rtu 2002:836b:9820:2::/64 4 pub life 1800 spl 48
```

Указывает префикс, который находится по ссылке на интерфейс \# 4. Префикс опубликован, то есть он включается в объявления маршрутизатора, если интерфейс \# 4 является интерфейсом объявления. Время жизни в параметре "сведения о префиксе" составляет 1800 секунд (30 минут). Длина префикса сайта в параметре сведений о префиксе равна 48.

Параметр получение сведений о префиксе, указывающий префикс сайта, приводит к созданию записи в таблице префиксов сайта, которую можно отобразить с помощью команды IPv6 СПТ. Таблица префиксов сайтов используется для удаления недопустимых адресов локального сайта, возвращенных [**функцию getaddrinfo**](/windows/desktop/api/Ws2tcpip/nf-ws2tcpip-getaddrinfo) и связанными функциями.
