# <a name="broadcast-to-local-network"></a>Широковещательная рассылка в локальную сеть

Вещание можно установить в локально подключенную сеть, установив для **SA \_ нетнум** значение двоичные нули (0), а **SA \_ ноденум** — двоичные (1). Это вещание может быть отправлено в основную сеть для устройства или во все локально подключенные сети при выборе поставщика услуг. Широковещательные передачи в локальную сеть не распространяются маршрутизаторами.
