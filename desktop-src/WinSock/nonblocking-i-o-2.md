# <a name="nonblocking-inputoutput"></a>Неблокируемые входные и выходные данные

Если сокет находится в неблокирующем режиме, любая операция ввода-вывода должна либо завершиться немедленно, либо вернуть код ошибки ВСАЕВАУЛДБЛОКК, указывающий, что операция не может быть завершена сразу. В последнем случае необходимо использовать механизм, чтобы определить, когда может потребоваться повторить операцию с ожиданием, что операция будет выполнена. Для этой цели определен набор сетевых событий. Эти события можно опрашивать или ожидать с помощью [**вспселект**](/previous-versions/windows/desktop/legacy/ms742289(v=vs.85))или можно зарегистрировать для асинхронной доставки, вызвав [**вспасинкселект**](/previous-versions/windows/desktop/legacy/ms742267(v=vs.85)) или [**вспевентселект**](/previous-versions/windows/hardware/network/ff566287(v=vs.85)). Дополнительные сведения см. в разделе [уведомление о сетевых событиях](notification-of-network-events-2.md) .
