| [**\_данные адреса \_ IPX**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_address_data)             | Предоставляет сведения о конкретном адаптере, к которому привязан протокол IPX. |

| [**\_данные СПКСКОННСТАТУС \_ IPX**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_spxconnstatus_data) | Предоставляет сведения о подключенном соединителе SPX.                   |
