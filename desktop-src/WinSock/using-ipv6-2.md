# <a name="using-ipv6"></a>Использование IPv6

в Windows XP позже для настройки IPv4 и управления им предоставляется новое средство командной строки. Это средство использует команду netsh interface IP для настройки протокола IPv4 и управления им.

в Windows XP с пакетом обновления 1 (SP1) и более поздних версиях это новое средство командной строки было усовершенствовано для поддержки настройки и управления IPv6. Это усовершенствованное средство — команда netsh interface IPv6. Изменения конфигурации, выполненные с помощью команд Netsh.exe, являются постоянными и не теряются при перезапуске компьютера или протокола IPv6.

следующая команда доступна в Windows XP с пакетом обновления 1 (SP1) и более поздних версий для запроса и настройки IPv6 на локальном компьютере:

-   [Netsh.exe](netsh-exe.md)

до Windows XP с пакетом обновления 1 (SP1), конфигурация ipv6 и управление использовали несколько старых программ командной строки для настройки ipv6 и управления им. При использовании этих старых средств изменения IPv6 не являются постоянными и теряются при перезапуске компьютера или протокола IPv6.

в Windows XP доступны следующие более старые команды

-   [Net.exe](net-exe-2.md)
-   [Ipv6.exe](ipv6-exe-2.md)
-   [Ipsec6.exe](ipsec6-exe-2.md)

эти старые средства были также предоставлены в предварительной версии технологии IPv6 для Windows 2000

Изменения конфигурации, использующие эти старые средства, можно сохранить, разместив их в виде командных строк в файле командного сценария (. cmd), который выполняется после перезагрузки компьютера или протокола IPv6. чтобы автоматически восстановить изменения конфигурации после перезапуска, можно использовать Windows запланированные задачи для запуска cmd-файла при запуске компьютера.

эти старые средства не предоставляются на Windows Server 2003 и более поздних версий. средство "netsh interface ipv6" предназначено для настройки ipv6 и управления им из командной строки на Windows Server 2003 и более поздних версиях.
