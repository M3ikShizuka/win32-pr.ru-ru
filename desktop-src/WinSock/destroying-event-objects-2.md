# <a name="destroying-event-objects"></a>Уничтожение объектов событий

Сущность, которая создает объект события (приложение или поставщик услуг), несет ответственность за его уничтожение, когда оно больше не требуется. Поставщики услуг выполняют это через **впуклосивент**.
