# <a name="ipv6-multicast-destination-addresses"></a>IPv6-адреса назначения для многоадресной рассылки

При отправке на адрес назначения для многоадресной рассылки протокол IPv6 обычно требует, чтобы в приложении был указан исходящий интерфейс. Это можно сделать с помощью **параметра \_ многоадресная рассылка IPv6 \_ при** использовании сокета или путем привязки сокета к определенному исходному адресу.

Можно также указать интерфейс по умолчанию для конкретного адреса многоадресной рассылки, всю многоадресную область адресов или все адреса многоадресной рассылки. Это делается с помощью маршрута, который помещает префикс многоадресной рассылки в ссылку на нужный исходящий интерфейс. В следующих примерах показано, как это можно указать:

``` syntax
ipv6 rtu ff02::5/128 3
ipv6 rtu ff0e::/16 3
ipv6 rtu ff00::/8 3
```
