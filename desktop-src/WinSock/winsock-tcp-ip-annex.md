# <a name="winsock-tcpip-annex"></a>Приложение Winsock TCP/IP

В этом разделе описываются функции протокола управления передачей/протокола Интернета (TCP/IP), структуры данных и элементы управления.



| Элемент          | Описание                                                                                                                                 |
|------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| Имена протоколов | TCP, UDP                                                                                                                                    |
| Описание      | Предоставляет транспортные службы на уровне IP-сети: UDP для ненадежных датаграмм, TCP для надежных потоков байтов, ориентированных на подключение. |
| Семейство адресов   | **AF \_ INET**, **AF \_ INET6**                                                                                                                 |
| Файл заголовка      | *Ws2tcpip. h*                                                                                                                                |



 

Предлагаются два основных типа транспортных служб: ненадежные датаграммы (UDP) и надежные ориентированные на подключение потоки (TCP). Кроме того, необработанный сокет необязательно поддерживается. Необработанные сокеты позволяют приложению взаимодействовать через протоколы, отличные от TCP и UDP, такие как ICMP. Необработанные сокеты поддерживаются в семействах адресов **AF \_ inet** и **AF \_ INET6** с некоторыми ограничениями.

В этом разделе описываются расширения для Winsock, характерные для протоколов TCP/IP. Здесь также описываются аспекты базовых функций Winsock, требующих особого внимания или которые могут привести к уникальному поведению при использовании TCP/IP.
