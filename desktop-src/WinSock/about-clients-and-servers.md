# <a name="about-servers-and-clients"></a>О серверах и клиентах

Существует два разных типа сетевых приложений сокетов: сервер и клиент.

Серверы и клиенты имеют разные поведения. Поэтому процесс их создания отличается. Ниже приведена общая модель создания потокового сервера TCP/IP и клиента.

## <a name="server"></a>Сервер

1.  Инициализируйте Winsock.
2.  Создайте сокет.
3.  Привяжите сокет.
4.  Прослушивание сокета для клиента.
5.  Принимает соединение от клиента.
6.  Получение и отправка данных.
7.  Производить отключение.

## <a name="client"></a>Клиент

1.  Инициализируйте Winsock.
2.  Создайте сокет.
3.  Подключение на сервер.
4.  Отправка и получение данных.
5.  Производить отключение.

> [!Note]  
> Некоторые шаги одинаковы для клиента и сервера. Эти действия реализуются практически одинаково. Некоторые действия, описанные в этом руководстве, относятся к типу создаваемого приложения.
