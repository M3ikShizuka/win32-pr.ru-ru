# <a name="installation-and-configuration-functions"></a>Функции установки и настройки

следующие функции реализуются в \_32.dll Ws2 и предназначены для использования приложениями, которые устанавливают Windows сокеты и поставщики служб пространства имен на компьютере. Эти функции не влияют на выполняющиеся в настоящее время приложения и не являются изменениями, внесенными этими функциями, видимыми для выполняющихся в настоящее время приложений.

Для всех поставщиков идентификатором поставщика является идентификатор GUID, создаваемый разработчиком поставщика (с помощью служебной программы Uuidgen.exe) и предоставляемый для Ws2 \_32.dll.



| Функция                                                                      | Описание                                                                                                                                              |
|-------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**вскдеинсталлпровидер**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscdeinstallprovider)                        | Удаляет поставщик транспортной службы из реестра.                                                                                                  |
| [**WSCDeinstallProvider32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscdeinstallprovider32)                      | Удаляет указанный поставщик 32-разрядной службы транспорта из реестра на 64-разрядной платформе.                                                          |
| [**всценумпротоколс**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumprotocols)                                | Извлекает сведения о доступных транспортных протоколах.                                                                                               |
| [**WSCEnumProtocols32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumprotocols32)                              | Извлекает сведения о доступных транспортных протоколах в 32-битном каталоге на 64-разрядных платформах.                                                     |
| [**всЦинсталлпровидер**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscinstallprovider)                            | Регистрирует нового поставщика транспортной службы.                                                                                                              |
| [**WSCInstallProvider64 \_ 32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscinstallprovider64_32)                   | Регистрирует нового поставщика транспортной службы в базах данных конфигурации 32-bit и 64-bit на 64-разрядной платформе.                               |
| [**всЦинсталлпровидерандчаинс**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscinstallproviderandchains)            | Регистрирует новый 32-разрядный поставщик транспортных служб, а также его конкретные цепочки протоколов в базе данных конфигурации системы на 32-разрядной платформе.   |
| [**WSCInstallProviderAndChains64 \_ 32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscinstallproviderandchains64_32) | Регистрирует новый поставщик транспорта и его конкретные цепочки протоколов в базах данных конфигурации системы 32-bit и 64-bit на 64-разрядной платформе. |
