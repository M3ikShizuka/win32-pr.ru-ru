# <a name="pseudo-blocking-and-true-blocking"></a>Псевдо и истинная блокировка

в 16 Windows средах значение true block не поддерживается операционной системой; Таким образом, операция блокирования, которая не может быть выполнена немедленно, обрабатывается следующим образом:

-   поставщик услуг инициирует операцию, а затем входит в цикл, в течение которого он отправляет все Windows сообщения (при необходимости передавая процессор другому потоку).
-   затем он проверяет завершение функции Windowsных сокетов.
-   Если функция завершена или при вызове [**вспканцелблоккингкалл**](/previous-versions/windows/desktop/legacy/ms742269(v=vs.85)) был вызван, цикл завершается, а функция блокировки завершается с соответствующим результатом.

Это относится к термину «псевдо-Block», а цикл, упомянутый выше, называется блокирующим обработчиком по умолчанию.
