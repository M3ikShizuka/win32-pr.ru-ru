# <a name="iobjectids-methods"></a>Методы Иобжектидс

Интерфейс [**иобжектидс**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectids) предоставляет следующие методы.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                            | Описание                                                                                     |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**Добавить метод**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-add)<br/>           | Добавляет объект [**иобжектид**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) в коллекцию.<br/>                     |
| [**Метод AddRange**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-addrange)<br/> | Добавляет диапазон объектов [**иобжектид**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) в коллекцию.<br/>            |
| [**Метод Clear**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-clear)<br/>       | Удаляет все объекты [**иобжектид**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) из коллекции.<br/>              |
| [**Удалить метод**](/windows/desktop/api/CertEnroll/nf-certenroll-iobjectids-remove)<br/>     | Удаляет объект [**иобжектид**](/windows/desktop/api/CertEnroll/nn-certenroll-iobjectid) из коллекции по значению индекса.<br/> |
