# <a name="trust-models"></a>Модели доверия

По мере увеличения числа организаций, реализующих инфраструктуру открытых ключей (PKI), и количества конечных пользователей и компьютеров в этих организациях увеличивается и снижается логистическая проблема, связанная с сертификацией удостоверений и установкой доверия. Для устранения этих проблем были внесены две основные модели доверия, иерархии сертификатов и перекрестная сертификация. Встроенные в обе модели являются концепцией цепочки сертификатов. Дополнительные сведения см. в следующих разделах:

-   [Иерархия сертификатов](about-certificate-hierarchy.md)
-   [Перекрестная сертификация](about-cross-certification.md)
-   [Цепочка сертификатов](about-certificate-chain.md)
