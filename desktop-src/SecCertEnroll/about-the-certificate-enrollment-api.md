Последние сведения см. в статье Библиотека Microsoft Developer Network (MSDN) ( [https://msdn.microsoft.com/library/default.asp](/documentation/) ).

-   [Кодирование запроса сертификата](about-certificate-request-encoding.md)
-   [Запросы сертификатов](certificate-requests.md)
