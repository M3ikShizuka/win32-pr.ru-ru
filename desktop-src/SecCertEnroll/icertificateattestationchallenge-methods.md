# <a name="icertificateattestationchallenge-methods"></a>Методы Ицертификатеаттестатиончалленже

Интерфейс [**ицертификатеаттестатиончалленже**](/windows/desktop/api/Certenroll/nn-certenroll-icertificateattestationchallenge) поддерживает следующие методы.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                           | Описание                                                                                                                                                                                                                                                    |
|-------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Метод Декриптчалленже**](/windows/desktop/api/Certenroll/nf-certenroll-icertificateattestationchallenge-decryptchallenge)<br/> | Расшифровывает запрос, полученный от [управления сертификатами, через CMS](/windows/desktop/SecGloss/c-gly) (CMC) и создает повторно зашифрованный ответ для отправки в ЦС.<br/> |
| [**Метод Initialize**](/windows/desktop/api/Certenroll/nf-certenroll-icertificateattestationchallenge-initialize)<br/>             | Выполните инициализацию, используя полное [Управление сертификатами через CMS](/windows/desktop/SecGloss/c-gly) (CMC), возвращенное из центра сертификации.<br/>                                        |
