# <a name="boolean-certificate-enrollment-api"></a>Логическое значение (API регистрации сертификатов)

Логическое значение может быть **true** или **false**. **Значение false** кодируется как TLV-Triplet, в котором поле **значения** равно нулю (0). Значение **true** кодируется как ненулевое значение. **Тег** для **логического** типа данных — 0x01.
