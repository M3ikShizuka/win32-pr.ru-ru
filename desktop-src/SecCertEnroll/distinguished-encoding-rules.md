# <a name="distinguished-encoding-rules"></a>Distinguished Encoding Rules

Аннотация абстрактного синтаксиса One (ASN. 1) определяет следующие наборы правил, которые управляют кодированием и декодированием структур данных, передаваемых между компьютерами.

-   Базовые правила кодирования (ЛИЧЕСТВО)
-   Правила канонической кодировки (CER)
-   Distinguished Encoding Rules (DER)
-   Правила упакованной кодировки (на)

Исходный набор правил был определен спецификацией ЛИЧЕСТВО. CER и DER были разработаны позже как специализированные подмножества ЛИЧЕСТВО. В было создано в ответ на критику о объеме пропускной способности, необходимой для передачи данных с помощью ЛИЧЕСТВО или его разновидностей. ДЛЯ обеспечивает значительную экономию.

Протокол DER был создан для удовлетворения требований спецификации X. 509 для безопасной пересылки данных. API регистрации сертификатов использует исключительно DER. Дополнительные сведения см. в следующих разделах:

-   [Синтаксис преобразования DER](about-der-transfer-syntax.md)
-   [Кодировка DER для типов ASN. 1](about-der-encoding-of-asn-1-types.md)
