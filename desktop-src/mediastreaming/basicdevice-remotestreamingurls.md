# <a name="basicdeviceremotestreamingurls-property"></a>Басикдевице. Ремотестреамингурлс, свойство

Возвращает вектор URL-адресов удаленной потоковой передачи.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_RemoteStreamingUrls(
  [out] IVector< HSTRING > **value
);
```



## <a name="property-value"></a>Значение свойства

Перечисляемая коллекция указателей на URL-адреса удаленной потоковой передачи.
