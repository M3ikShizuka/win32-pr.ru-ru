# <a name="basicdevicediscoveredoncurrentnetwork-property"></a>Басикдевице. Дисковередонкуррентнетворк, свойство

Возвращает значение, указывающее, находится ли устройство в текущей сети.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_DiscoveredOnCurrentNetwork(
  [out] boolean *value
);
```



## <a name="property-value"></a>Значение свойства

Указатель на логическое значение, равное **true** , если устройство находится в текущей сети.
