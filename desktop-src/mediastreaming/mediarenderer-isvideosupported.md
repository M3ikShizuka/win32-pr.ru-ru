# <a name="mediarendererisvideosupported-property"></a>Медиарендерер. Исвидеосуппортед, свойство

Возвращает значение, указывающее, способен ли ДМР воспроизводить видео-содержимое.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_IsVideoSupported(
  [out] boolean *value
);
```



## <a name="property-value"></a>Значение свойства

Логическое значение, равное **true** , если ДМР способен воспроизводить видеосодержимое, и **false** в противном случае.
