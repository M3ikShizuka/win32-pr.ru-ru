# <a name="basicdeviceuniquedevicename-property"></a>Басикдевице. Уникуедевиценаме, свойство

Возвращает уникальное имя устройства (УДН).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_UniqueDeviceName(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Значение свойства

Указатель на модель устройства УДН.
