# <a name="playbackoperation-class"></a>Класс Плайбаккоператион

Регистрирует обработчик событий, который вызывается, когда завершается асинхронная операция, запущенная одним из методов воспроизведения [**медиарендерер**](mediarenderer.md) , и предоставляет метод, возвращающий результаты операции.

**Плайбаккоператион** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Класс **плайбаккоператион** содержит следующие методы.



| Метод                                             | Описание                                                                                                                                |
|:---------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](playbackoperation-getresults.md) | Возвращает результаты асинхронной операции, запущенной одним из методов воспроизведения [**медиарендерер**](mediarenderer.md) .<br/> |



 

### <a name="properties"></a>Элемент Property

Класс **плайбаккоператион** имеет следующие свойства.



| Свойство.                                                    | Тип доступа           | Описание                                                                                                                                                                           |
|:------------------------------------------------------------|:----------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Завершено**](playbackoperation-completed.md)<br/> | Чтение/запись<br/> | Возвращает или задает обработчик событий, вызываемый при завершении асинхронной операции, запущенной одним из методов воспроизведения [**медиарендерер**](mediarenderer.md) . <br/> |
