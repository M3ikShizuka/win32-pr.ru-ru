# <a name="basicdevicemodelname-property"></a>Басикдевице. ModelName, свойство

Возвращает имя модели устройства.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_ModelName(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Значение свойства

Указатель на имя модели устройства.
