# <a name="mediarendererisaudiosupported-property"></a>Медиарендерер. Исаудиосуппортед, свойство

Возвращает значение, указывающее, способен ли ДМР воспроизведение звукового содержимого.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_IsAudioSupported(
  [out] boolean *value
);
```



## <a name="property-value"></a>Значение свойства

Логическое значение, равное **true** , если ДМР способен воспроизводить аудио-содержимое, и **false** в противном случае.
