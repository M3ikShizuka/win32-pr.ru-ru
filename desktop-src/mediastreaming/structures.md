# <a name="media-streaming-structures"></a>Структуры потоковой передачи мультимедиа

[API потоковой передачи мультимедиа](media-streaming-api-portal.md) предоставляет следующие структуры.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                           | Описание                                                                                                                                        |
|-----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**плайспид**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-playspeed)<br/>                       | Представляет скорость воспроизведения в виде рационального числа. <br/>                                                                                      |
| [**поситионинформатион**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-positioninformation)<br/>   | Содержит текущие значения сведений о положении воспроизведения мультимедиа, полученных из ДМР.<br/>                                               |
| [**рендерингпараметерс**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-renderingparameters)<br/>   | Содержит текущие значения параметров отрисовки в ДМР.<br/>                                                                         |
| [**траккинформатион**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-trackinformation)<br/>         | Содержит номер текущей записи и длительность в составе [**поситионинформатион**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-positioninformation) , полученной из ДМР.<br/> |
| [**транспортинформатион**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-transportinformation)<br/> | Содержит текущие значения данных, связанных с транспортом воспроизведения мультимедиа, полученных из ДМР.<br/>                                      |
