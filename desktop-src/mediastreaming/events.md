# <a name="events-media-streaming-api"></a>События (API потоковой передачи мультимедиа)

[API потоковой передачи мультимедиа](media-streaming-api-portal.md) создает следующие события.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                           | Описание                                                                                                                                              |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Событие Коннектионстатусчанжед**](connectionstatuschanged.md)<br/>     | Происходит при изменении состояния сетевого подключения устройства s.<br/>                                                                                   |
| [**Событие Девицеарривал**](devicearrival.md)<br/>                         | Происходит при добавлении нового устройства в список устройств, возвращенных методом [**качеддевицес**](idevicecontroller-cacheddevices.md) .<br/>     |
| [**Событие Девицедепартуре**](devicedeparture.md)<br/>                     | Происходит при удалении нового устройства из списка устройств, возвращенных методом [**качеддевицес**](idevicecontroller-cacheddevices.md) .<br/> |
| [**Событие Рендерингпараметерсупдате**](renderingparametersupdate.md)<br/> | Возникает при обновлении любого из набора параметров элемента управления отрисовки в ДМР.<br/>                                                          |
| [**Событие Транспортпараметерсупдате**](transportparametersupdate.md)<br/> | Происходит при каждом обновлении любого из наборов параметров транспорта в ДМР.<br/>                                                                  |
