# <a name="createmediarendereroperationgetresults-method"></a>Креатемедиарендерероператион:: Results, метод

Возвращает результаты асинхронной операции.

## <a name="syntax"></a>Синтаксис


```C++
MediaRenderer* GetResults();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Результаты операции. Указатель на только что созданный объект [**медиарендерер**](mediarenderer.md) .
