# <a name="getstreampropertiesoperation-class"></a>Класс Жетстреампропертиесоператион

Регистрирует обработчик событий, вызываемый при завершении асинхронной операции, запущенной [**жетстреампропертиесасинк**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)) , и предоставляет метод, возвращающий результаты операции.

**Жетстреампропертиесоператион** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#getstreampropertiesoperation-class)

### <a name="methods"></a>Методы

Класс **жетстреампропертиесоператион** содержит следующие методы.



| Метод                                                        | Описание                                                                                                                                          |
|:--------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](getstreampropertiesoperation-getresults.md) | Возвращает результаты асинхронной операции, запущенной [**жетстреампропертиесасинк**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)).<br/> |



 

### <a name="properties"></a>Свойства

Класс **жетстреампропертиесоператион** имеет следующие свойства.



| Свойство.                                                               | Тип доступа           | Описание                                                                                                                                                                                   |
|:-----------------------------------------------------------------------|:----------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Завершено**](getstreampropertiesoperation-completed.md)<br/> | Чтение/запись<br/> | Возвращает или задает обработчик событий, который вызывается при завершении асинхронной операции, запущенной [**жетстреампропертиесасинк**](/previous-versions/windows/desktop/legacy/hh829001(v=vs.85)) .<br/> |
