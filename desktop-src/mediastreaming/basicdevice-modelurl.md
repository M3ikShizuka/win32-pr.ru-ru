# <a name="basicdevicemodelurl-property"></a>Басикдевице. Моделурл, свойство

Возвращает URL-адрес модели для устройства.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_ModelUrl(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Значение свойства

Указатель на URL-адрес модели устройства.
