# <a name="basicdevicecanwakedevices-property"></a>Басикдевице. Канвакедевицес, свойство

Возвращает значение, указывающее, может ли устройство пробудиться.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_CanWakeDevices(
  [out] boolean *value
);
```



## <a name="property-value"></a>Значение свойства

Указатель на логическое значение, равное **true** , если устройство может выходить из спящего режима.
