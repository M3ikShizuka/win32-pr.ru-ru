# <a name="mediarendereractioninformation-property"></a>Медиарендерер. Актионинформатион, свойство

Возвращает сведения о том, какие методы в настоящее время можно вызывать для ДМР.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT get_ActionInformation(
  [out] IMediaRendererActionInformation **value
);
```



## <a name="property-value"></a>Значение свойства

Ссылка на интерфейс [**имедиарендерерактионинформатион**](/previous-versions/windows/desktop/api/windows.media.streaming/nn-windows-media-streaming-imediarendereractioninformation) .
