# <a name="transportstate-enumeration"></a>Перечисление Транспортстате

Определяет доступные состояния транспорта, определенные правилами UPnP.

## <a name="syntax"></a>Синтаксис


```C++
typedef enum _TransportState { 
  Unknown         = 0,
  Stopped         = 1,
  Playing         = 2,
  Transitioning   = 3,
  Paused          = 4,
  Recording       = 5,
  NoMediaPresent  = 6,
  Last            = 7
} TransportState;
```
