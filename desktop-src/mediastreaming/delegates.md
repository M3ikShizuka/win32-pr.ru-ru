# <a name="delegates"></a>Делегаты

[API потоковой передачи мультимедиа](media-streaming-api-portal.md) предоставляет следующие функции обработчика событий.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                   | Описание                                                                                                                                                                                                            |
|-----------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**коннектионстатушандлер**](/previous-versions/windows/desktop/legacy/hh828836(v=vs.85))<br/>                   | Представляет функцию, которая будет выполнять обработку события [**коннектионстатусчанжед**](connectionstatuschanged.md) , которое уведомляет клиента об изменениях в состоянии сетевого подключения устройства s.<br/>               |
| [**девицеконтроллерфиндерхандлер**](/previous-versions/windows/desktop/legacy/hh828843(v=vs.85))<br/>       | Представляет функцию, которая будет выполнять обработку событий [**девицеарривал**](devicearrival.md) и [**девицедепартуре**](devicedeparture.md) , которые уведомляют клиента об изменениях в списке кэшированных устройств.<br/> |
| [**рендерингпараметерсупдатехандлер**](/previous-versions/windows/desktop/legacy/hh828994(v=vs.85))<br/> | Представляет функцию, которая будет выполнять обработку события [**рендерингпараметерсупдате**](renderingparametersupdate.md) , которое уведомляет клиента об обновлении с любыми параметрами элемента управления отрисовки ДМР.<br/>  |
| [**транспортпараметерсупдатехандлер**](/previous-versions/windows/desktop/legacy/hh829007(v=vs.85))<br/> | Представляет функцию, которая будет выполнять обработку события [**транспортпараметерсупдате**](transportparametersupdate.md) , которое уведомляет клиента об обновлении с любыми транспортными параметрами ДМР s.<br/>          |
