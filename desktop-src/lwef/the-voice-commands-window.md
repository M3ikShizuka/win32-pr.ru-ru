# <a name="the-voice-commands-window"></a>Окно "Voice Commands"

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Если установлен совместимый речевой механизм, то Microsoft Agent предоставляет специальное окно, называемое **окном голосовых команд** , которое отображает команды, для которых включено распознавание речи. **Окно голосовых команд** служит визуальным запросом о том, что можно проголосовать в качестве входных данных (команды нельзя выбирать с помощью мыши).

![диалоговое окно "Voice Commands"](images/f2voice.gif)

Это окно появляется, когда пользователь выбирает команду **открытия окна "открыть голосовые команды** ", выбирая команду или щелкнув символ правой кнопкой мыши и выбирая команду из всплывающего меню символа. Однако если пользователь отключает речевой ввод, **окно голосовых команд** недоступно.

В **окне "Voice Commands** " команды с поддержкой голоса отображаются в виде дерева. Если текущее размещенное приложение предоставляет команды Voice, они отображаются в верхней части окна. Записи также отображаются для других приложений, использующих этот символ. В окне также содержатся глобальные речевые команды, предоставляемые агентом Microsoft Agent. Если в текущем размещенном приложении нет голосовых команд, в верхней части окна отобразятся глобальные речевые команды.

Пользователь может изменять размер и перемещать **окно "Voice Commands**". Microsoft Agent запоминает последнее расположение окна и повторно отображает его в этом расположении, если пользователь закрывает и снова открывает окно. Если записи в окне превышают текущий отображаемый размер окна, появляются полосы прокрутки.
