# <a name="text-to-speech-engines-text-normalization"></a>Нормализация текста для модулей преобразования текста в речь

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Нормализация — это процесс идентификации чисел, сокращений, аббревиатур и идиоматикс, а также преобразования их в полный текст по мере необходимости, обычно на основе контекста предложения.

Например, с помощью обработчика "L&H Трувоице американский английский TTS", предложение:

Короля Георгия, умер, 1952.

будет считаться в **нормальном контексте** как:

   Короля 19 52 в 1 февраля, умер.

Но в **контексте сообщения электронной почты** она будет считаться:

   Короля Георгия — шестая часть Англия, умер в шестой 19 52 февраля.

Сведения об использовании **контекстного** речевого тега см. в документации по программированию агента [речевого вывода для агента Microsoft Agent](microsoft-agent-speech-output-tags.md).
