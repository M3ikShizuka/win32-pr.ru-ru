# <a name="creating-a-new-sound-file"></a>Создание нового звукового файла

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

При первом запуске редактора можно создать новый звуковой файл, выбрав **запись** в меню **аудио** или нажав кнопку **запись** на панели инструментов редактора звука, а затем нажимая на микрофон, подключенный к системе. Нажмите кнопку " **Закрыть** " на панели инструментов, чтобы прерывать запись. Вы можете выбрать команду **воспроизвести** в меню **аудио** или на панели инструментов, чтобы увидеть, как агент Майкрософт обрабатывает звуковой файл без лингвистического расширения. Чтобы создать другой файл, выберите пункт Создать в меню **файл** или на панели инструментов.
