# <a name="mpupdatecontrol-function"></a>Функция Мпупдатеконтрол

Позволяет управлять операцией обновления сигнатур, которая была асинхронно инициирована через [**мпупдатестарт**](mpupdatestart.md). Для вызова этой функции требуются права администратора, так как она допускает отмену обновления сигнатур на уровне системы.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT WINAPI MpUpdateControl(
  _In_ MPHANDLE  hUpdateHandle,
  _In_ MPCONTROL UpdateControl
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хупдатехандле* \[ окне\]
</dt> <dd>

Тип: **мфандле**

Обработчик асинхронной операции обновления сигнатур. Этот маркер возвращается функцией [**мпсканстарт**](mpscanstart.md) .

</dd> <dt>

*Упдатеконтрол* \[ окне\]
</dt> <dd>

Тип: **файл mpcontrol**

Задает параметр управления обновлением сигнатур. Это должно быть следующее значение:



| Значение                                                                                                                                                               | Значение                                          |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------|
| <span id="MPCONTROL_ABORT"></span><span id="mpcontrol_abort"></span><dl> <dt>**прервать файл MPCONTROL \_**</dt> </dl> | Прервать операцию обновления сигнатур.<br/> |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

Если функция выполнена успешно, возвращается значение **S \_**.

Если функция завершается ошибкой, возвращаемое значение является неудачным кодом **HRESULT** . Вызывающий объект может использовать функцию [**мперрормессажеформат**](mperrormessageformat.md) для получения общего описания сообщения об ошибке.
