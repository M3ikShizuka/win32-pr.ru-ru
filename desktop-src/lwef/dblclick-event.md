# <a name="dblclick-event"></a>Двойное событие

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Происходит при двойном щелчке символа пользователем.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

 *Подагент * * * \_ Двойное* Нажатие *  **(ByVal** *чарактерид*, **ByVal** , **ByVal** *SHIFT*, **ByVal** *X*, **ByVal** *Y ** * *)*



| Отделение          | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
|---------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *чарактерид* | Возвращает идентификатор символа двойного щелчка в виде строки.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| *Кнопка*      | Возвращает целое число, определяющее кнопку, которая была нажата и была вызвана для события. Аргумент Button — это битовое битовое значение, соответствующее левой кнопке (бит 0), правой кнопке (бит 1) и средней кнопке (бит 2). Эти биты соответствуют значениям 1, 2 и 4 соответственно. Задан только один из битов, указывающий кнопку, вызвавшую событие. Если символ содержит значок панели задач, то если также задан бит 13, щелчок на значке панели задач.                                                                                                                                                                       |
| *Сдвиг*       | Возвращает целое число, соответствующее состоянию клавиш SHIFT, CTRL и ALT при нажатии или отпускании кнопки, заданной в аргументе Button. Если ключ не работает, устанавливается бит. Аргумент сдвига является битовым битом с наименьшими значащими битами, соответствующими клавише SHIFT (бит 0), клавишей CTRL (бит 1) и клавишей ALT (бит 2). Эти биты соответствуют значениям 1, 2 и 4 соответственно. Аргумент shift указывает состояние этих ключей. Можно установить некоторые, все или ни одного бита, указывая, что не нажаты некоторые, все или ни один из этих ключей. Например, при нажатии клавиш CTRL и ALT значение Shift будет равно 6. |
| *X, Y*         | Возвращает целое число, указывающее текущее положение указателя мыши. </dd> </dl>

### <a name="remarks"></a>Комментарии

Это событие отправляется только клиенту ввода-активного клиента символа. Когда пользователь дважды щелкает символ или его значок на панели задач без клиента input-Active, сервер отправляет событие последнему входному клиенту. Если символ является видимым ([видимым](visible-property.md)  =  ), он также устанавливает активный клиент как текущий клиент ввода-активного, отправляя событие [**активатеинпут**](activateinput-event.md) этому клиенту, а затем отправляя событие **DblClick** . Если символ является скрытым (видимым = **false**) и пользователь дважды щелкает значок на панели задач с помощью кнопки 1, он также автоматически отображает этот символ.
