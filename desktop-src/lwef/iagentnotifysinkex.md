# <a name="iagentnotifysinkex"></a>иажентнотифисинкекс

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

**Иажентнотифисинкекс** является производным от интерфейса [**иажентнотифисинк**](https://www.bing.com/search?q=**IAgentNotifySink**) . Он включает все методы **иажентнотифисинк** и предоставляет доступ к дополнительным функциям.

**Методы в порядке таблицы Vtable**



| Методы Иажентнотифисинкекс                                                   | Описание                                                        |
|------------------------------------------------------------------------------|--------------------------------------------------------------------|
| [**хелпкомплете**](iagentnotifysinkex--helpcomplete.md)                     | Происходит, когда пользователь выбирает меню или символ в режиме справки. |
| [**листенингстате**](iagentnotifysinkex--listeningstate.md)                 | Происходит при изменении состояния прослушивания символа.               |
| [**дефаултчарактерчанже**](iagentnotifysinkex--defaultcharacterchange.md) | Происходит, когда пользователь изменяет символ по умолчанию.                |
| [**ажентпропертичанже**](iagentnotifysinkex--agentpropertychange.md)       | Происходит, когда пользователь изменяет параметр свойства агента.            |
| [**активеклиентчанже**](iagentnotifysinkex--activeclientchange.md)         | Происходит при изменении активного клиента символа.              |
