# <a name="iresultsviewercontents-property"></a>Свойство Иресултсвиевер:: Content

> [!NOTE]
> Windows Desktop Search 2. x — это устаревшая технология, которая изначально была доступна в качестве надстройки для Windows XP и Windows Server 2003. В более поздних выпусках используйте [API поиска Windows](../search/-search-reference-entry-page.md) . 

Это свойство отслеживает тип содержимого, отображаемого в представлении результатов.

## <a name="syntax"></a>Синтаксис

## <a name="property-value"></a>Значение свойства

Возвращает указатель на тип содержимого, отображаемого в представлении результатов.
