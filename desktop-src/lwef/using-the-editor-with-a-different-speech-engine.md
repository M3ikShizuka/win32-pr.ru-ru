# <a name="using-the-editor-with-a-different-speech-engine"></a>Использование редактора с другим модулем распознавания речи

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Хотя редактор звука устанавливает модуль распознавания речи Майкрософт (4,0), он может использоваться с другим модулем распознавания речи, если этот механизм поддерживает необходимые интерфейсы, описанные в документе требования к речевому подсистеме. Прежде чем пытаться использовать редактор с другим подсистемой, уточните у поставщика, что они соответствуют этим требованиям.

Чтобы использовать редактор звука с другим модулем распознавания речи, выберите команду "обработчик речи" в меню " **Правка** ". Откроется диалоговое окно, в котором отображается текущее используемое ядро. Чтобы выбрать другой обработчик, отобразите список модулей и нажмите кнопку **ОК**. Если в списке нет других модулей, другие совместимые модули не включены.
