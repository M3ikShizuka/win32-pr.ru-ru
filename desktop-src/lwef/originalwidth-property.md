# <a name="originalwidth-property"></a>Оригиналвидс, свойство

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает целочисленное значение, указывающее исходную ширину символа.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Субагент. ***Символы ("*** чарактерид * * *"). оригиналвидс**

</dd> </dl>

## <a name="remarks"></a>Комментарии

Это свойство возвращает ширину кадра символа, созданную с помощью редактора символов Microsoft Agent.
