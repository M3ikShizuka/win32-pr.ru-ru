# <a name="endpoint-data-loss-prevention-structures"></a>Структуры защиты от потери данных конечной точки

Структуры, используемые функцией предотвращения потери данных в конечной точке.



| Структуры                                                       | Описание                                                           |
|-------------------------------------------------------------------|-----------------------------------------------------------------------|
| [DLP_APP_OP_ENLIGHTENED_LEVEL](endpointdlp-dlp_app_op_enlightened_level.md)                       | Связывает действие защиты от потери данных (DLP) конечной точки с уровнем применения.                                  |
| [DLP_DOCUMENT_INFO](endpointdlp-dlp_document_info.md)                       | Указывает сведения о документе, связанном с операцией DLP конечной точки.                                  |
| [DLP_POSTOP_STATUS](enpointdlp-dlp_postop_status.md)                         | Указывает сведения о состоянии операции DLP конечной точки.  |
| [DLP_PRINT_INFO](endpointdlp-dlp_print_info.md)                       | Указывает сведения о документе, связанном с операцией печати в конечной точке.                                  |
