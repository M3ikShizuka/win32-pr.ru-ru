## <a name="remarks"></a>Remarks

Этот метод можно использовать для доступа к методам и свойствам [**символьного**](/windows/desktop/lwef/the-characters-object) объекта.

> [!Note]  
> Этот метод может потребоваться для некоторых языков программирования, которые не поддерживают коллекции. Он не требуется для VBScript или Visual Basic. Дополнительные сведения об указании методов [**символов**](/windows/desktop/lwef/the-characters-object) см. в разделе символьные методы объекта.
