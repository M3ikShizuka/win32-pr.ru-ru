# <a name="windows-defender-reference"></a>Справочник по Защитнику Windows

Защитник Windows помогает защитить компьютеры от всплывающих окон, снижения производительности и угроз безопасности, вызванных шпионским и другим нежелательным программным обеспечением. Приложения могут запрашивать такие сведения, как работоспособность компьютера, конкретные сведения о вредоносных программах и журнал вредоносных программ, а также запускать проверки по требованию.
