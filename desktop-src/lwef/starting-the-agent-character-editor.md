# <a name="starting-the-agent-character-editor"></a>Запуск редактора символов агента

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

чтобы запустить редактор символов агента, выберите параметр **редактор символов агента** в меню " **пуск** " Windows панели задач или дважды щелкните значок **редактора символов Microsoft Agent** на рабочем столе. Откроется окно редактора, в котором отображаются меню, панель инструментов с часто используемыми командами, дерево со списком компонентов, составляющих определение символа, и набор страниц с вкладками, которые изменяются в зависимости от выбора в дереве компонентов.

![Снимок экрана, на котором показана страница "Свойства" символа — редактор символов Microsoft Agent.](images/f1chared.gif)

Для доступа к полям в окне с клавиатуры можно использовать клавишу TAB и SHIFT + TAB для перемещения между элементами управления или использовать клавишу доступа (ALT +*подчеркнутая буква*) для перехода к конкретному элементу управления. После завершения работы редактора можно приступить к созданию нового определения символа или загрузке существующего определения символа.

В строке состояния отображаются сведения о командах или кнопках на панели инструментов при наведении на них указателя мыши. В нем также отображаются сводные сведения о данных анимации символов и сведения о состоянии при создании символа.
