# <a name="iresulttype-interface"></a>Интерфейс Иресулттипе

> [!NOTE]
> Windows Desktop Search 2. x — это устаревшая технология, которая изначально была доступна в качестве надстройки для Windows XP и Windows Server 2003. В более поздних выпусках используйте [API поиска Windows](../search/-search-reference-entry-page.md) . 

Предоставляет сведения о типе свойства.

## <a name="members"></a>Элементы

Интерфейс **иресулттипе** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иресулттипе** также имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Интерфейс **иресулттипе** имеет следующие свойства.



| Свойство                                                                 | Тип доступа           | Описание                                                                           |
|:-------------------------------------------------------------------------|:----------------------|:--------------------------------------------------------------------------------------|
| [**DisplayName**](-search-2x-iresulttype-displayname.md)<br/>     | Только для чтения<br/>  | Локализованное отображаемое имя типа:<br/>                                        |
| [**GetProperty**](-search-2x-iresulttype-getproperty.md)<br/>     | Чтение/запись<br/> | Это свойство содержит указанные сведения о свойстве. <br/>                    |
| [**прецеиведтипе**](-search-2x-iresulttype-preceivedtype.md)<br/> | Только для чтения<br/>  | Это свойство содержит строку, используемую для задания типа в индексе. <br/> |
| [**пропертикаунт**](-search-2x-iresulttype-propertycount.md)<br/> | Только для чтения<br/>  | Это свойство содержит количество свойств, предоставляемых типом. <br/>      |
| [**ИД пользователя**](-search-2x-iresulttype-uid.md)<br/>                     | Только для чтения<br/>  | Это свойство содержит уникальный идентификатор для типа. <br/>                |



 

## <a name="remarks"></a>Комментарии

Эти методы предоставляют типы возвращаемых сведений.
