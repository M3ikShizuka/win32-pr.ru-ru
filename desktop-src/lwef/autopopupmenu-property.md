# <a name="autopopupmenu-property"></a>Аутопопупмену, свойство

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает или задает значение, указывающее, отображается ли всплывающее меню символа при щелчке символа правой кнопкой мыши или его значка на панели задач.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*агент. ***Символы * * * * ("*** чарактерид * * *"). Аутопопупмену* *  \[  =  *Boolean*\]



| Отделение      | Описание                                                                                                                                                                                                                                           |
|-----------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *boolean* | Логическое выражение, указывающее, будет ли сервер автоматически отображать всплывающее меню символа щелчком правой кнопкой мыши. **True** (по умолчанию) — отображение меню щелчком правой кнопкой мыши. <br/> **Значение false** Не отображает меню щелчком правой кнопкой мыши.<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Комментарии

Если задать для этого свойства **значение false**, можно создать собственное поведение при обработке меню. Чтобы отобразить меню после присвоения этому свойству значения **false**, используйте метод [**шовпопупмену**](showpopupmenu-method.md) .

Это свойство применяется только к символу, используемому вашим клиентским приложением. Этот параметр не влияет на другие клиенты символов или других символов клиентского приложения.
