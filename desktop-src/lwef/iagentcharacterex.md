# <a name="iagentcharacterex"></a>иажентчарактерекс

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

**Иажентчарактерекс** является производным от интерфейса [**иажентчарактер**](iagentcharacter.md) . Он включает все методы **иажентчарактер** , а также предоставляет доступ к дополнительным функциям.

**Методы в порядке таблицы Vtable**



| Методы Иажентчарактерекс                                         | Описание                                                                    |
|-------------------------------------------------------------------|--------------------------------------------------------------------------------|
| [**шовпопупмену**](iagentcharacterex--showpopupmenu.md)         | Отображает всплывающее меню для символа.                                    |
| [**сетаутопопупмену**](iagentcharacterex--setautopopupmenu.md)   | Указывает, будет ли сервер автоматически отображать всплывающее меню символа.    |
| [**жетаутопопупмену**](iagentcharacterex--getautopopupmenu.md)   | Возвращает значение, указывающее, будет ли сервер автоматически отображать всплывающее меню символа. |
| [**жеселпфиленаме**](iagentcharacterex--gethelpfilename.md)     | Возвращает имя файла справки для символа.                                   |
| [**сеселпфиленаме**](iagentcharacterex--sethelpfilename.md)     | Задает имя файла справки для символа.                                      |
| [**сеселпмодеон**](iagentcharacterex--sethelpmodeon.md)         | Устанавливает режим справки в.                                                             |
| [**жеселпмодеон**](iagentcharacterex--gethelpmodeon.md)         | Возвращает значение, указывающее, включен ли режим справки.                                               |
| [**сеселпконтекстид**](iagentcharacterex--sethelpcontextid.md)   | Задает Хелпконтекстид для символа.                                      |
| [**жеселпконтекстид**](iagentcharacterex--gethelpcontextid.md)   | Возвращает Хелпконтекстид для символа.                                   |
| [**Очень активная**](iagentcharacterex--getactive.md)                 | Возвращает активное состояние для символа.                                    |
| [**Прослушивание**](iagentcharacterex--listen.md)                       | Задает состояние прослушивания для символа.                                    |
| [**сетлангуажеид**](iagentcharacterex--setlanguageid.md)         | Задает идентификатор языка для символа.                                        |
| [**жетлангуажеид**](iagentcharacterex--getlanguageid.md)         | Возвращает идентификатор языка для символа.                                     |
| [**жетттсмодеид**](iagentcharacterex--getttsmodeid.md)           | Возвращает идентификатор режима TTS, заданный для символа.                                 |
| [**сетттсмодеид**](iagentcharacterex--setttsmodeid.md)           | Задает идентификатор режима TTS для символа.                                        |
| [**жетсрмодеид**](iagentcharacterex--getsrmodeid.md)             | Возвращает идентификатор текущего режима модуля распознавания речи.                       |
| [**сетсрмодеид**](iagentcharacterex--setsrmodeid.md)             | Задает модуль распознавания речи.                                            |
| [**GetGUID**](iagentcharacterex--getguid.md)                     | Возвращает идентификатор символа.                                            |
| [**жеторигиналсизе**](iagentcharacterex--getoriginalsize.md)     | Возвращает исходный размер рамки символов.                              |
| [**Ситуацию**](iagentcharacterex--think.md)                         | Отображает указанный текст в подсказке "мысли" символа.              |
| [**GetVersion**](iagentcharacterex--getversion.md)               | Возвращает версию символа.                                          |
| [**жетаниматионнамес**](iagentcharacterex--getanimationnames.md) | Возвращает имена анимаций для символа.                         |
| [**жетсрстатус**](iagentcharacterex--getsrstatus.md)             | Возвращает условия, необходимые для поддержки речевого ввода.                      |
