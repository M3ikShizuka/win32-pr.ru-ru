# <a name="using-the-microsoft-agent-character-editor"></a>Использование редактора символов Microsoft Agent

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Редактор символов Microsoft Agent позволяет компилировать анимацию символов для использования с Microsoft Agent. Анимацию можно определить путем импорта растровых изображений Windows, настройки их длительности и, при необходимости, включения ветвления, звуковых эффектов и наложения речи. Сведения о проектировании анимации символов см. в разделе [Разработка символов для Microsoft Agent](designing-characters-for-microsoft-agent.md).

С помощью редактора символов агента можно определить имя и описание символа, а также параметры вывода, включая преобразование текста в речь (TTS), синтезированный голосовый вывод, поддержку всплывающего меню и отображение всплывающего слова символа.

В этом документе описывается редактор символов Microsoft Agent для Microsoft Agent, который можно скачать с веб <https://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=c21be0ec-1712-427d-a43d-94c122ef8eb6> .
