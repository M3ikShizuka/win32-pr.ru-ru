# <a name="iresultsviewerisupdateneeded-property"></a>Свойство Иресултсвиевер:: Исупдатенидед

> [!NOTE]
> Windows Desktop Search 2. x — это устаревшая технология, которая изначально была доступна в качестве надстройки для Windows XP и Windows Server 2003. В более поздних выпусках используйте [API поиска Windows](../search/-search-reference-entry-page.md) . 

Это возвращает значение TRUE, если запрос views был изменен и требует обновления.

## <a name="syntax"></a>Синтаксис

## <a name="property-value"></a>Значение свойства

При вызове этого метода возвращается указатель на флаг, указывающий, изменился ли запрос.
