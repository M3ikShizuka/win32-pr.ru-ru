# <a name="animation-sound-effects"></a>Звуковые эффекты анимации

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Кроме того, Microsoft Agent позволяет включать звуковые эффекты как часть анимации символа. с помощью редактора символов Microsoft Agent можно указать имя файла стандартного Windowsного звука (. Файлы WAV), которые нужно активировать для данного кадра. Обратите внимание, что Microsoft Agent не использует звуковые эффекты и речевые выходные данные, поэтому речь не начнется до завершения звукового эффекта. Таким образом, старайтесь не использовать длинный или циклический звуковой эффект как часть анимации символа.
