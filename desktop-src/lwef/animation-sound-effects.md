# <a name="animation-sound-effects"></a>Звуковые эффекты анимации

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Кроме того, Microsoft Agent позволяет включать звуковые эффекты как часть анимации символа. С помощью редактора символов Microsoft Agent можно указать имя файла стандартного звука Windows (. Файлы WAV), которые нужно активировать для данного кадра. Обратите внимание, что Microsoft Agent не использует звуковые эффекты и речевые выходные данные, поэтому речь не начнется до завершения звукового эффекта. Таким образом, старайтесь не использовать длинный или циклический звуковой эффект как часть анимации символа.
