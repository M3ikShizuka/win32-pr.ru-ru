# <a name="show-event"></a>Отобразить событие

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Происходит при отображении символа.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

 *Подагент * * * \_ Показывать (ByVal* *  *чарактерид*, **ByVal** , *Причина * * *)**



| Часть          | Описание                                                                                                                                                                                                                                                                 |
|---------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *чарактерид* | Возвращает идентификатор символа, показанного в виде строки.                                                                                                                                                                                                                          |
| *Причина*       | Возвращает значение, указывающее, что привело к отображению символа. 2 пользователь показал символ (с помощью меню или команды Voice).<br/> 4 клиентское приложение показало этот символ.<br/> 6 другое клиентское приложение показало этот символ.<br/> |



 

</dd> </dl>

### <a name="remarks"></a>Remarks

Сервер отправляет это событие всем клиентам символа. Чтобы запросить текущее состояние символа, используйте свойство [**Visible**](visible-property.md) .
