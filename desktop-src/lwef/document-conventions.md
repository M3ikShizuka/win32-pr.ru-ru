# <a name="document-conventions"></a>Соглашения о документе

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

В этой документации используются следующие типографские соглашения:



| Обозначение                       | Описание                                                                    |
|----------------------------------|--------------------------------------------------------------------------------|
| **Под, видимый, заголовок**        | Слова полужирным шрифтом, начинающиеся с прописной буквы, обозначают ключевые слова.               |
| *агент, строка, Now*             | Курсивом обозначаются заполнители для предоставленной информации.                 |
| ВВОД, F1                        | Слова в прописных буквах обозначают имена файлов, имена ключей и последовательности ключей. |
| `Agent1.Commands.Enabled = True` | Текст в этом шрифте указывает примеры кода.                                       |
| `' This is a comment`            | Апостроф ( `'` ) указывает на комментарий к коду.                                  |
| `Agent1.Commands.Add "Test1", _` | Пробел и символ подчеркивания ( \_ ) продолжат строку кода.                       |
| `[words or expression]`          | Элементы внутри квадратных скобок необязательны.                                     |
| `This | That`                    | Вертикальная черта указывает на возможность выбора между двумя или более элементами.                   |
| *Субагент*                          | Слово «Agent» в курсиве представляет имя используемого элемента управления агента.  |



 

Описания программных интерфейсов в этом документе соответствуют соглашениям для Microsoft VBScript. Однако они должны также применяться и к другим языкам.
