# <a name="width-property-commandswindow-object"></a>Свойство Width (объект Коммандсвиндов)

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает целочисленное значение, указывающее ширину (в пикселях) окна "Voice Commands".

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Syntax** 
</dt> <dd>

*агент * * *. Коммандсвиндов. Width**

</dd> </dl>

### <a name="remarks"></a>Remarks

Сервер отображает окно "Voice Commands" в зависимости от расположения и размера, заданного пользователем.
