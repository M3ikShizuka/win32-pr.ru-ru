# <a name="loading-a-character"></a>Загрузка символа

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Чтобы анимировать символ, необходимо сначала загрузить символ. Используйте метод [**Load**](load-method.md) для загрузки данных символа. Microsoft Agent поддерживает два формата для данных символов и анимации: один структурированный файл и отдельные файлы. Как правило, используется формат одного файла (. ACS), когда данные хранятся локально. Формат нескольких файлов (. ACF,. АКА) лучше подходит, если требуется загружать анимацию по отдельности, например при доступе к анимациям с HTTP-сервера.

Клиентское приложение может загружать только один экземпляр одного и того же символа. Любая попытка загрузить один и тот же символ несколько раз завершится ошибкой. Однако в приложении может быть загружено несколько экземпляров одного и того же символа путем предоставления отдельных подключений к Microsoft Agent. Например, приложение может загрузить один и тот же символ из двух копий элемента управления Microsoft Agent.

Можно также определить собственные символы для использования с Microsoft Agent. вы можете использовать любое средство визуализации, которое предпочитает создавать изображения, при условии, что в итоге Windows файлы формата растровых изображений. Чтобы собрать и скомпилировать изображения символов в анимации для использования с Microsoft Agent, используйте редактор символов Microsoft Agent. Это средство позволяет определить свойства символа по умолчанию, а также определить анимацию для символа. Редактор символов Microsoft Agent также позволяет выбрать соответствующий формат файла при создании символа.
