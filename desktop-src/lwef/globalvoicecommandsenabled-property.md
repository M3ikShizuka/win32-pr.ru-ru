# <a name="globalvoicecommandsenabled-property"></a>Глобалвоицекоммандсенаблед, свойство

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает или задает значение, указывающее, включена ли поддержка голоса для глобальных команд агента.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*символы Agent. ***("**_чарактерид_*_"). Commands. Глобалвоицекоммандсенаблед_*

\[ = *логическая*\]



| Часть      | Описание                                                                                                                                                                                                            |
|-----------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *boolean* | Логическое выражение, указывающее, включены ли параметры голоса для глобальных команд агента. **True** (по умолчанию) параметры голоса включены. <br/> **Значение false** Параметры голоса отключены.<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Remarks

Microsoft Agent автоматически добавляет параметры голоса (грамматики) для открытия и закрытия окна "Voice Commands", а также для отображения и скрытия символа. Если для **глобалвоицекоммандсенаблед** задано **значение false**, то агент отключает любые параметры голоса для этих команд, а также параметры голоса для [**заголовка**](caption-property.md) объектов [**команд**](/windows/desktop/lwef/the-commands-collection-object) других клиентов. Это позволяет исключить их из текущей активной грамматики клиента. Однако, так как это потенциально блокирует голосовой доступ к другим клиентам, сбросьте это свойство в **значение true** после обработки речевого ввода пользователя.

Отключение свойства не влияет на всплывающее меню символа. Глобальные команды, добавленные сервером, по-прежнему будут отображаться. Их нельзя удалить из всплывающего меню.
