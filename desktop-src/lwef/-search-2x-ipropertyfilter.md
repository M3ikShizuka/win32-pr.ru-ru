# <a name="ipropertyfilter-interface"></a>Интерфейс Ипропертифилтер

> [!NOTE]
> Windows Desktop Search 2. x — это устаревшая технология, которая изначально была доступна в качестве надстройки для Windows XP и Windows Server 2003. В более поздних выпусках используйте [API поиска Windows](../search/-search-reference-entry-page.md) . 

Отображает свойства, используемые для фильтрации запроса.

## <a name="members"></a>Элементы

Интерфейс **ипропертифилтер** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Ипропертифилтер** также имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Интерфейс **ипропертифилтер** имеет следующие свойства.



| Свойство                                                                                      | Тип доступа           | Описание                                                   |
|:----------------------------------------------------------------------------------------------|:----------------------|:--------------------------------------------------------------|
| [**Ипропертифилтер:: Индексколумн**](-search-2x-ipropertyfilter-indexcolumn.md)<br/>     | Чтение/запись<br/> | Имя индексируемого столбца свойства, по которому производится фильтрация. <br/> |
| [**Ипропертифилтер:: Логикоператор**](-search-2x-ipropertyfilter-logicoperator.md)<br/> | Чтение/запись<br/> | Логический оператор, используемый при применении фильтра. <br/>       |
| [**Ипропертифилтер:: Нестингдепс**](-search-2x-ipropertyfilter-nestingdepth.md)<br/>   | Чтение/запись<br/> | Отфильтровывает глубину внутри вложенного набора круглых скобок. <br/> |
| [**Ипропертифилтер:: Text**](-search-2x-ipropertyfilter-text.md)<br/>                   | Чтение/запись<br/> | Текст фильтра. <br/>                               |
| [**Ипропертифилтер:: UID**](-search-2x-ipropertyfilter-uid.md)<br/>                     | Чтение/запись<br/> | UID свойства, по которому необходимо выполнить фильтрацию. <br/>                 |



 

## <a name="remarks"></a>Комментарии

Эти свойства используются для фильтрации запроса.
