# <a name="spd-tag"></a>Тег SPD

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Задает среднее значение скорости общения для речевого вывода.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

**\\SPD =***номер***\\**



| Отделение     | Описание                                          |
|----------|------------------------------------------------------|
| *number* | Базовая средняя скорость беседы в словах в минуту. |



 

</dd> </dl>

### <a name="remarks"></a>Комментарии

Этот тег поддерживается только для выходных данных, созданных системой TTS. Диапазон значений параметра может отличаться в зависимости от установленного обработчика TTS.
