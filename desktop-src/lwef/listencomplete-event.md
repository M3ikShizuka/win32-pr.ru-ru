# <a name="listencomplete-event"></a>Событие Листенкомплете

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Происходит при завершении режима прослушивания (распознавание речи).

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

 *Подагент. * * * листенкомплете (ByVal* *  *чарактерид*, **ByVal** , *Причина * * *)**



| Часть          | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|---------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *чарактерид* | Возвращает идентификатор прослушивающего символа в виде строки.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| *Причина*       | Возвращает причину завершения события как целое число, которое может быть одним из следующих: 1 режим прослушивания был отключен программным кодом.<br/> истекло время ожидания 2 режима прослушивания (включенного программным кодом).<br/> истекло время ожидания 3 режима прослушивания (включенного с помощью ключа прослушивания).<br/> 4 режим прослушивания отключен, так как пользователь освободил ключ прослушивания.<br/> 5 режим прослушивания завершен, так как пользователь завершил говорить.<br/> 6 режим прослушивания завершен, так как клиент ввода был деактивирован.<br/> 7 режим прослушивания завершен, так как был изменен символ по умолчанию.<br/> 8 режим прослушивания завершен, так как пользователь отключил речевой ввод.<br/> |



 

</dd> </dl>

### <a name="remarks"></a>Remarks

Это событие отправляется всем клиентам, когда заканчивается время ожидания в режиме прослушивания, после освобождения пользователем ключа прослушивания, когда входной активный клиент вызывает метод [**Listen**](listen-method.md) со значением **false** или пользователь завершил говорить. С помощью этого события можно определить, когда следует возобновлять речевой вывод (звук).

Если включить режим прослушивания с помощью метода [**Listen**](listen-method.md) , а затем пользователь нажмет клавишу прослушивания, режим прослушивания сбрасывается и сохраняется до завершения тайм-аута ключа прослушивания, отменяется ключ прослушивания или пользователь завершает диктовку, в зависимости от того, что происходит позже. В этом случае вы не получите событие **листенкомплете** , пока не завершится режим прослушивания ключа.

Событие возвращает символ для клиентов, которые в настоящий момент загрузили этот символ. Все остальные клиенты получают символ null (пустая строка).
