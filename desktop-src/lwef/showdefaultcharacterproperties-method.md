# <a name="showdefaultcharacterproperties-method"></a>Метод Шовдефаултчарактерпропертиес

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Отображает свойства символа по умолчанию.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*агент * * *. Шовдефаултчарактерпропертиес* *  \[ *X* , *Y*\]



| Часть | Описание                                                                                                                                                |
|------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *X*  | Необязательный элемент. Короткое целое значение, указывающее координату горизонтального (*X* ) экрана для отображения окна. Эта координата должна быть указана в пикселях. |
| *да*  | Необязательный элемент. Короткое целое значение, указывающее координату вертикального экрана (*Y*) для отображения окна. Эта координата должна быть указана в пикселях.    |



 

</dd> </dl>

## <a name="remarks"></a>Remarks

При вызове этого метода отображается окно «Свойства символа по умолчанию» (не вкладка свойств Microsoft Agent). Если не указать координаты X и Y, окно будет отображаться в последнем расположении.
