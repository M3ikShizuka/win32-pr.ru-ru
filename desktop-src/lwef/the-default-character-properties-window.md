# <a name="the-default-character-properties-window"></a>Окно "Свойства символа по умолчанию"

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Кроме того, чтобы разрешить приложениям загружать определенные символы, приложения могут загружать символ, который является общим ресурсом для пользователя, называемый *символом по умолчанию*. Символ по умолчанию доступен из любого приложения, но этот символ может выбирать только пользователь. Чтобы облегчить выбор этого символа, агент предоставляет окно, предоставляющее доступ к выбору этого символа, который называется окном «Свойства символа по умолчанию». Доступ к этому окну поддерживается через API агента.

![диалоговое окно «Свойства символа Genie»](images/f8dpwin.gif)

Окно «Свойства символа по умолчанию» не может использоваться для предоставления выбора символов, кроме символов по умолчанию.
