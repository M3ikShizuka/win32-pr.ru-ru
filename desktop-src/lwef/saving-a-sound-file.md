# <a name="saving-a-sound-file"></a>Сохранение звукового файла

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Когда вы будете готовы сохранить звуковой файл, выберите команду **сохранить** в меню **файл** или на панели инструментов редактора. Редактор отображает диалоговое окно **Сохранить как** и предлагает имя и тип файла по умолчанию в зависимости от того, были ли созданы лингвистические сведения для файла. Если сохранить файл в виде звукового файла (. wav), редактор сохранит только звуковые данные. Если сохранить сведения о файле в виде лингвистического звукового файла (. лвв), то сведения о Word и phoneme автоматически включаются в состав измененного звукового файла. После подтверждения или изменения имени, расположения, типа файла и формата нажмите кнопку **сохранить** .

![Снимок экрана, на котором показано диалоговое окно "Сохранить как" с файлом "имя файла", "Сохранить как тип", "формат" и выбран параметр "атрибуты".](images/f7listsave.gif)

Если вы хотите сохранить звуковой файл с новым именем, другим расположением или другим форматом, выберите команду **Сохранить как** в меню **файл** . Когда появится диалоговое окно **Сохранить как** , введите новое имя файла и нажмите кнопку **сохранить** .

Можно также сохранить часть звукового файла. Например, может потребоваться сохранить файл без чрезмерного тишины в начале или в конце. В представлении аудио выберите часть файла, которую необходимо сохранить, и выберите пункт **Сохранить выделенный фрагмент как** в меню **файл** . Команда доступна, только если в **звуковом представлении** выбрано выделение.
