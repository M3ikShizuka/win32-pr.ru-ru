# <a name="iagentnotifysinkexagentpropertychange"></a>Иажентнотифисинкекс:: Ажентпропертичанже

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

``` syntax
HRESULT AgentPropertyChange(
);
```

Уведомляет клиентское приложение, когда пользователь изменяет параметр свойства Microsoft Agent.

-   Нет возвращаемого значения.

Когда пользователь изменяет параметр свойства Microsoft Agent на странице свойств Microsoft Agent, сервер отправляет это событие всем клиентам, если сервер в данный момент не приостановлен.
