## <a name="remarks"></a>Комментарии

При загрузке символа с помощью протокола HTTP (. ACF File) необходимо использовать метод **Get** для получения данных анимации, прежде чем можно будет воспроизвести анимацию. Этот метод не используется, если вы загрузили символ с помощью протокола UNC (. Файл ACS). Вы также не можете получить данные HTTP для символа с помощью **Get** , если вы загрузили этот символ с помощью протокола UNC (. Файл символов ACS).

Если объявляется ссылка на объект и устанавливается этот метод, он возвращает объект [**запроса**](/windows/desktop/lwef/the-request-object) . Если не удается загрузить связанную анимацию, сервер устанавливает для свойства [**Status**](status-property.md) объекта **запроса** значение "Failed" (ошибка) с соответствующим номером ошибки. Чтобы проверить состояние и определить, какое действие следует предпринять, можно использовать событие [**рекуесткомплете**](requestcomplete-event.md) .

Анимация или звуковые данные, полученные с помощью метода **Get** , хранятся в кэше браузера. Последующие вызовы проверяют кэш и, если данные анимации уже есть, элемент управления загружает данные непосредственно из кэша. После загрузки анимации или звуковые данные можно воспроизвести с помощью методов [**Play**](play-method.md) или [**говорите**](speak-method.md) .
