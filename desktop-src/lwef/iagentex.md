# <a name="iagentex"></a>иажентекс

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

**Иажентекс** является производным от интерфейса [**иажент**](iagent.md) . Он включает все методы **иажент** , а также предоставляет доступ к дополнительным функциям.

**Методы в порядке таблицы Vtable**



| Методы Иажентекс                                                                   | Описание                                              |
|------------------------------------------------------------------------------------|----------------------------------------------------------|
| [**шовдефаултчарактерпропертиес**](iagentex--showdefaultcharacterproperties.md) | Отображает свойства символов по умолчанию.               |
| [**GetVersion**](iagentex--getversion.md)                                         | Возвращает номер версии агента Microsoft Agent (сервер). |
