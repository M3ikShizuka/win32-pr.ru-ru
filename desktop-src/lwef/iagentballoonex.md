# <a name="iagentballoonex"></a>иажентбаллунекс

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

[**Иажентбаллунекс**](/windows/desktop/lwef/iagentballoon) является производным от интерфейса **иажентбаллун** . Он включает все методы **иажентбаллун** и предоставляет доступ к дополнительным функциям.

**Методы в порядке таблицы Vtable**



| Методы Иажентбаллунекс                                  | Описание                                                        |
|----------------------------------------------------------|--------------------------------------------------------------------|
| [**Стиль**](https://www.bing.com/search?q=**GetStyle**)                     | Возвращает стиль вывода всплывающей подсказки в слове.                           |
| [**SetStyle**](https://www.bing.com/search?q=**SetStyle**)                     | Задает стиль вывода всплывающей подсказки в слове.                              |
| [**сетнумлинес**](https://www.bing.com/search?q=**SetNumLines**)               | Задает числовые строки в тексте всплывающего окна.                  |
| [**сетнумчарсперлине**](https://www.bing.com/search?q=**SetNumCharsPerLine**) | Задает количество символов в строке в выноске. |
