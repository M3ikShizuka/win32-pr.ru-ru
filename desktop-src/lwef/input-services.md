# <a name="input-services"></a>Входные службы

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Клиентское приложение предоставляет основной пользовательский интерфейс для взаимодействия с символом. Можно программировать символ, чтобы реагировать на любые формы ввода, от нажатий кнопок до ввода введенного текста. Кроме того, Microsoft Agent предоставляет события, которые позволяют программировать, что происходит при нажатии пользователем, двойном щелчке или перетаскивании символа. Сервер передает координаты указателя и все состояние ключа модификатора для этих событий.
