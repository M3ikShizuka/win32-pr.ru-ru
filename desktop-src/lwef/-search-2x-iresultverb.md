# <a name="iresultverb-interface"></a>Интерфейс Иресултверб

> [!NOTE]
> Windows настольный поиск 2. x — это устаревшая технология, которая изначально была доступна в качестве надстройки для Windows XP и Windows Server 2003. в более поздних выпусках используйте вместо этого [API Windows поиска](../search/-search-reference-entry-page.md) . 

Предоставляет свойства команды.

## <a name="members"></a>Элементы

Интерфейс **иресултверб** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иресултверб** также имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Элемент Property

Интерфейс **иресултверб** имеет следующие свойства.



| Свойство                                                             | Тип доступа           | Описание                                                                                                       |
|:---------------------------------------------------------------------|:----------------------|:------------------------------------------------------------------------------------------------------------------|
| [**DisplayName**](-search-2x-iresultverb-displayname.md)<br/> | Только для чтения<br/>  | Это свойство возвращает указатель на локализованное отображаемое имя для команды. <br/>                           |
| [**доит**](-search-2x-iresultverb-doit.md)<br/>               | Только на запись<br/> | Выполняет команду. <br/>                                                                                    |
| [**Включен**](-search-2x-iresultverb-enabled.md)<br/>         | Только для чтения<br/>  | Это свойство возвращает значение TRUE, если команда включена. <br/>                                                    |
| [**HelpText**](-search-2x-iresultverb-helptext.md)<br/>       | Только для чтения<br/>  | Это свойство возвращает указатель на локализованную строку справки для команды. <br/>                            |
| [**Значок**](-search-2x-iresultverb-icon.md)<br/>               | Только для чтения<br/>  | Это свойство возвращает указатель на маркер необязательного значка, связанного с командой. <br/>              |
| [**Имя**](-search-2x-iresultverb-name.md)<br/>               | Только для чтения<br/>  | Это свойство возвращает указатель на имя кононикал для команды, например Print, Open и т. д. <br/> |



 

## <a name="remarks"></a>Remarks

Эти методы предоставляют свойства и действия, применимые к командам.
