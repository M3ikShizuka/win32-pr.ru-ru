# <a name="mpfreememory-function"></a>Функция Мпфримемори

Освобождает память для диспетчера защиты от вредоносных программ. Все буферы, выделенные и возвращаемые функциями защиты от вредоносных программ, должны быть освобождены вызывающей стороной с помощью этой функции.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI MpFreeMemory(
  _In_ PVOID pMemory
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пмемори* \[ окне\]
</dt> <dd>

Тип: **PVOID**

Указатель на память, выделенную функцией защиты от вредоносных программ.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Чтобы упростить управление памятью для клиентов, диспетчер защиты от вредоносных программ также определяет макросы для освобождения памяти, связанной с различными структурами, возвращаемыми функциями защиты от вредоносных программ. В файле заголовка мпмемфри. h определены следующие макросы:



| Имя                            | Описание                                                                      |
|---------------------------------|----------------------------------------------------------------------------------|
| \_Бесплатная информация о мпресаурце \_          | Освобождает выделенные [**\_ сведения о мпресаурце**](mpresource-info.md).                  |
| \_Бесплатная информация о мпсреат \_            | Освобождает выделенные [**\_ сведения о мпсреат**](mpthreat-info.md).                      |
| \_Бесплатные локализованные \_ сведения мпсреат \_ | Освобождает выделенные [**\_ локализованные \_ сведения мпсреат**](mpthreat-localized-info.md). |
