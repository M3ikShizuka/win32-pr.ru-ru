# <a name="ttsmodeid-property"></a>Ттсмодеид, свойство

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает или задает режим модуля TTS, используемый для символа.

</dd> <dt>

## <a name="remarks"></a>Комментарии

Это свойство определяет идентификатор режима обработчика TTS (преобразование текста в речь) для речевого вывода символа. Идентификатор режима для модуля TTS — это отформатированная строка, определяемая поставщиком речи, которая однозначно определяет режим работы механизма. Дополнительные сведения см. [в разделе доступ к модулю распознавания речи в коде](accessing-a-speech-engine-in-your-code.md).

Задание этого свойства переопределяет попытку сервера загрузить модуль на основе параметра TTS, скомпилированного символом, и текущего параметра [**LanguageID**](languageid-property.md) символа. Однако если указать идентификатор режима для ядра, который не установлен, или если пользователь отключил речевой вывод на странице свойств Microsoft Agent (**аудиуутпут. Enabled = false**), сервер выдаст ошибку.

Если вы не установили для символа идентификатор режима TTS (или не были успешно установлены), сервер проверяет, соответствует ли параметру режима TTS, заданному для символа, параметру [**LanguageID**](languageid-property.md) , и что соответствующий обработчик TTS установлен. Если да, то режим TTS, используемый символом для выходного выхода, и это свойство возвращает идентификатор этого режима. В противном случае сервер запрашивает совместимый речевой обработчик SAPI, соответствующий **LanguageID** символа, а также пол и Age, заданные для идентификатора скомпилированного режима символа. Если не задан параметр **LanguageID** символа, его **LanguageID** является текущим языком пользователя. Если соответствующий механизм не найден, запрос этого свойства возвращает пустую строку для идентификатора режима ядра. Аналогичным образом, если вы запрашиваете это свойство, когда пользователь отключил речевой вывод на странице свойств Microsoft Agent (**аудиуутпут. Enabled = false**), значение будет пустой строкой.

При запросе или задании этого свойства будет загружен соответствующий обработчик (если он еще не загружен). Однако, если подсистема, указанная в параметре «скомпилированный символ», установлена и соответствует параметру идентификатора языка символа, подсистема будет загружена при загрузке символа.

Это свойство применяется только к символу, используемому вашим клиентским приложением. Этот параметр не влияет на другие клиенты символов или других символов клиентского приложения.

Требования к речевому подсистему агента Microsoft Agent основаны на Microsoft Speech API. Модули, поддерживающие требования к SAPI Microsoft Agent, можно установить и использовать с агентом.

> [!Note]  
> Это свойство также возвращает пустую строку, если в системе не установлена совместимая поддержка звука.

 

> [!Note]  
> Установка **ттсмодеид** может завершиться ошибкой, если не установлен Speech.dll и указанный модуль не соответствует параметру режима TTS, скомпилированному для символа.

 

> [!Note]  
> Запрос этого свойства обычно не возвращает ошибку. Тем не менее, если при загрузке обработчика речи занимает слишком много времени, может возникнуть ошибка, указывающая, что время ожидания запроса истекло.
