# <a name="agent-control-events"></a>События элемента управления агента

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Элемент управления Microsoft Agent предоставляет несколько событий, которые позволяют клиентскому приложению отслеживанию состояния сервера:

-   [**активатеинпут**](activateinput-event.md)
-   [**активеклиентчанже**](activeclientchange-event.md)
-   [**ажентпропертичанже**](agentpropertychange-event.md)
-   [**баллунхиде**](balloonhide-event.md)
-   [**баллуншов**](balloonshow-event.md)
-   [**Закладка**](bookmark-event.md)
-   [**Щелкните**](click-event.md)
-   [**Get-Help**](command-event.md)
-   [**Кнопки**](dblclick-event.md)
-   [**деактиватеинпут**](deactivateinput-event.md)
-   [**дефаултчарактерчанже**](defaultcharacterchange-event.md)
-   [**драгкомплете**](dragcomplete-event.md)
-   [**драгстарт**](dragstart-event.md)
-   [**хелпкомплете**](helpcomplete-event.md)
-   [**Скрыть**](hide-event.md)
-   [**идлекомплете**](idlecomplete-event.md)
-   [**идлестарт**](idlestart-event.md)
-   [**листенкомплете**](listencomplete-event.md)
-   [**листенстарт**](listenstart-event.md)
-   [**Переместить**](move-event.md)
-   [**рекуесткомплете**](requestcomplete-event.md)
-   [**RequestStart**](requeststart-event.md)
-   [**Казывающи**](show-event.md)
-   [**Размер**](size-event.md)

Сервер больше не отправляет эти события перезагрузки и завершения работы.
