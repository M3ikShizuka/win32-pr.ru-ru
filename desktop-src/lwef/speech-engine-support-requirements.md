# <a name="speech-engine-support-requirements"></a>Требования поддержки обработчика речи

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Microsoft Agent использует интерфейс прикладного программирования (SAPI) Microsoft Speech для поддержки речевого ввода (распознавания речи, SR) и речевого вывода (преобразования текста в речь или TTS). Благодаря поддержке этого стандарта службы распознавания речи Microsoft Agent могут поддерживаться другими обработчиками речи. В этом документе описываются необходимые интерфейсы SAPI, используемые агентом Microsoft Agent. Дополнительные сведения о SAPI см. на веб-сайте группы [речи Майкрософт](https://msdn.microsoft.com/library/ee705648.aspx) .

-   [Требования для модулей распознавания речи](requirements-for-speech-recognition-engines.md)
-   [Требования для поддержки средства редактирования лингвистических сведений Майкрософт](requirements-for-supporting-the-microsoft-linguistic-information-sound-editing-tool.md)
