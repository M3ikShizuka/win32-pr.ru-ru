# <a name="iagentpropertysheet"></a>иажентпропертишит

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

**Иажентпропертишит** определяет интерфейс, позволяющий приложениям задавать и запрашивать свойства для страницы свойств Microsoft Agent (окно).

**Методы в порядке таблицы Vtable**



| Методы Иажентпропертишит                             | Описание                                                                                  |
|---------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**Во всех видимых**](iagentpropertysheet--getvisible.md)   | Возвращает значение, указывающее, является ли страница свойств Microsoft Agent видимой.                               |
| [**сетвисибле**](iagentpropertysheet--setvisible.md)   | Задает свойство [**Visible**](visible-property.md) страницы свойств Microsoft Agent. |
| [**Методом Disposition**](iagentpropertysheet--getposition.md) | Возвращает расположение страницы свойств Microsoft Agent.                                  |
| [**GetSize**](iagentpropertysheet--getsize.md)         | Возвращает размер листа свойств Microsoft Agent.                                      |
| [**GetPage.**](iagentpropertysheet--getpage.md)         | Возвращает текущую страницу для страницы свойств Microsoft Agent.                             |
| [**сетпаже**](iagentpropertysheet--setpage.md)         | Задает текущую страницу для страницы свойств Microsoft Agent.                                |
