# <a name="installing-the-sound-editor"></a>Установка звукового редактора

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Рекомендуемой конфигурацией системы для использования звукового редактора является ПК с процессором Pentium 166, не менее 48 МБ ОЗУ и совместимой с Windows звуковой платой. Если вы хотите записать речевой ввод с помощью средства, потребуется также совместимый микрофон.

Чтобы установить средство Microsoft лингвистического редактирования звука, откройте его самораспаковывающийся файл установки. При этом будут автоматически установлены соответствующие файлы в системе. Если вы скачиваете редактор звука с веб-сайта Microsoft Agent, можно выбрать установку редактора после загрузки или сохранения на диске для последующего открытия и установки. Средство установки предложит установить себя в подкаталог Tools агента Microsoft Agent. Мы рекомендуем использовать это расположение.

Команда Майкрософт и управление модулем распознавания речи (версия. 4,0) также необходимо установить, прежде чем можно будет использовать редактор звука. Обычно он устанавливается вместе с звуковым редактором, но если впоследствии он был удален, его можно переустановить из пакета SDK для платформы Майкрософт или с веб-сайта Microsoft Agent по адресу <https://www.microsoft.com/downloads/details.aspx?FamilyID=914fef6b-d764-4eca-88dc-d49d8912625c> . Редактор звука может формировать только лингвистические сведения на основе языка, поддерживаемого модулем распознавания речи. Для создания сведений для других языков необходимо установить совместимую подсистему распознавания речи для этого языка. Обратитесь к поставщику речевого модуля, чтобы определить, поддерживают ли они Microsoft лингвистическое средство редактирования звука.
