# <a name="dlpnotifypredragdrop-function"></a>Функция Длпнотифипредрагдроп

Предоставляет системе сведения о документе до инициирования операции перетаскивания.

## <a name="syntax"></a>Синтаксис


```C++
void WINAPI DlpNotifyPreDragDrop(_In_ const PDLP_DOCUMENT_INFO DocumentInfo);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Документинфо* \[ окне\]
</dt> <dd>

Указатель на структуру [PDLP_DOCUMENT_INFO](endpointdlp-dlp_document_info.md) , содержащую сведения о документе, связанном с операцией перетаскивания.

</dd> </dl>


## <a name="return-value"></a>Возвращаемое значение

Возвращает значение void.
