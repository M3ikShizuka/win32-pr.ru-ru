# <a name="iagentballoon"></a>иажентбаллун

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

**Иажентбаллун** определяет интерфейс, позволяющий приложениям запрашивать свойства всплывающей программы Microsoft Agent. Эти функции также доступны из [**иажентбаллунекс**](https://www.bing.com/search?q=**IAgentBalloonEx**).

Начальные значения по умолчанию для всплывающего слова символа задаются в редакторе символов Microsoft Agent, но после запуска приложения пользователь может переопределить свойства " [**включено**](enabled-property.md) " и " [Шрифт](fontname-property.md) ". Если пользователь изменяет свойства всплывающего окна, это изменение затрагивает все символы. Свойства объекта [**иажентбаллун**](/windows/desktop/lwef/iagentballoon) также применяются к выходным данным с помощью метода [**обработки**](think-method.md) .
