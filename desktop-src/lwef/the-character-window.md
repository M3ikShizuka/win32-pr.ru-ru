# <a name="the-character-window"></a>Окно символов

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Microsoft Agent отображает анимированные символы в собственных окнах, которые всегда отображаются в верхней части z-порядка (то есть всегда поверх остальных окон). Пользователь может переместить окно символа, перетащив его с помощью левой кнопки мыши. Изображение символа перемещается с указателем. Кроме того, приложение может перемещать символ с помощью метода [**MoveTo**](moveto-method.md) .

Когда пользователь щелкает правой кнопкой мыши символ, появляется всплывающее меню, в котором отображаются следующие команды:

Открыть \| окно "Close <span class="underline">V</span>лосовой Commands"

<span class="underline">H</span>IDE

----------------------------…

Команда\*


*осерхостингаппликатионкаптион\*\**

\*Перечисленные команды основаны на клиенте input-Active. Дополнительные сведения об определении команд, отображаемых во всплывающем меню, см. в разделе Общие сведения о интерфейсе программирования агента Майкрософт.

\*\*Перечисленные записи — это все остальные приложения, в которых в настоящее время размещается символ. Дополнительные сведения об определении этой записи см. в статье Обзор программного интерфейса агента Майкрософт.

Команда открыть \| окно закрытия речи управляет отображением окна команд текущего активного символа. Если службы распознавания речи отключены, эта команда отключена. Если службы распознавания речи не установлены, эта команда не отображается.

Команда Hide скрывает символ. Анимация, назначенная **скрытому** состоянию символа, воспроизводит и скрывает символ. Буква "H" в разделе "скрыть" — это клавиша доступа команды (назначенная).

Команды для приложений, на которых в данный момент размещается символ, следуют за командой Hide, перед которой стоит разделитель. Имена других приложений, использующих этот символ, также предшествуют разделителю.
