# <a name="toolbar-buttons-microsoft-agent-character-editor"></a>Кнопки панели инструментов (редактор символов Microsoft Agent)

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

:::image type="icon" source="images/f9charnew.gif":::

<dl> <dt>

<span id="New_Custom_Character"></span><span id="new_custom_character"></span><span id="NEW_CUSTOM_CHARACTER"></span>**Новый настраиваемый символ**
</dt> <dd>

Сбрасывает редактор для создания нового пользовательского определения символа. Если существующий символ загружается и имеет несохраненные изменения в файлах, редактор отображает сообщение, чтобы определить, следует ли сохранять или удалять несохраненные изменения.

</dd> </dl>

:::image type="icon" source="images/f10charopen.gif":::

<dl> <dt>

<span id="Open_Character_Definition"></span><span id="open_character_definition"></span><span id="OPEN_CHARACTER_DEFINITION"></span>**Открыть определение символа**
</dt> <dd>

Открывает диалоговое окно Открытие файла, позволяющее открыть существующий файл определения символа для редактирования. Если существующий символ загружается и имеет несохраненные изменения в файлах, редактор отображает сообщение, чтобы определить, следует ли сохранять или удалять несохраненные изменения.

</dd> </dl>

:::image type="icon" source="images/f11charsave.gif":::

<dl> <dt>

<span id="Save_Character_Definition"></span><span id="save_character_definition"></span><span id="SAVE_CHARACTER_DEFINITION"></span>**Сохранить определение символа**
</dt> <dd>

Сохраняет определение символа. Если определение символа не существует (не было именовано), редактор отображает диалоговое окно Сохранить как для ввода имени файла.

</dd> </dl>

:::image type="icon" source="images/f12charbuild.gif":::

<dl> <dt>

<span id="Build_Character"></span><span id="build_character"></span><span id="BUILD_CHARACTER"></span>**Символ сборки**
</dt> <dd>

Создает символ Microsoft Agent из определения символа.

</dd> </dl>

:::image type="icon" source="images/f13charprint.gif":::

<dl> <dt>

<span id="Print_Character_Definition"></span><span id="print_character_definition"></span><span id="PRINT_CHARACTER_DEFINITION"></span>**Печать определения символа**
</dt> <dd>

Печать текущего файла определения символа, открытого в редакторе.

</dd> </dl>

:::image type="icon" source="images/f14charcut.gif":::

<dl> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Бавьте**
</dt> <dd>

Удаляет выбранный элемент в редакторе и помещает его в буфер обмена Windows.

</dd> </dl>

:::image type="icon" source="images/f15charcopy.gif":::

<dl> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Копии**
</dt> <dd>

Копирует выбранный элемент из редактора в буфер обмена Windows.

</dd> </dl>

:::image type="icon" source="images/f16charpaste.gif":::

<dl> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Авить**
</dt> <dd>

Копирует данные из текущего буфера обмена Windows в выбранное расположение.

</dd> </dl>

:::image type="icon" source="images/f17chardel.gif":::

<dl> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Удален**
</dt> <dd>

Удаляет выбранный элемент из редактора.

</dd> </dl>

:::image type="icon" source="images/f18charundo.gif":::

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Отмен**
</dt> <dd>

Удаляет изменения, внесенные в редакторе.

</dd> </dl>

:::image type="icon" source="images/f19charredo.gif":::

<dl> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Redo**
</dt> <dd>

Изменяет действие отмены в редакторе на противоположное.

</dd> </dl>

:::image type="icon" source="images/f20charaanim.gif":::

<dl> <dt>

<span id="New_Animation"></span><span id="new_animation"></span><span id="NEW_ANIMATION"></span>**Создать анимацию**
</dt> <dd>

Создает новый объект анимации в редакторе.

</dd> </dl>

:::image type="icon" source="images/f21charanfr.gif":::

<dl> <dt>

<span id="New_Animation_Frame"></span><span id="new_animation_frame"></span><span id="NEW_ANIMATION_FRAME"></span>**Создать кадр анимации**
</dt> <dd>

Создает новый кадр для анимации.

</dd> </dl>

:::image type="icon" source="images/f22charprev.gif":::

<dl> <dt>

<span id="Preview"></span><span id="preview"></span><span id="PREVIEW"></span>**Предварительный просмотр**
</dt> <dd>

Воспроизводит анимацию, начиная с ее выбранного кадра.

</dd> </dl>

:::image type="icon" source="images/f23charprevex.gif":::

<dl> <dt>

<span id="Preview_Exit_Branching"></span><span id="preview_exit_branching"></span><span id="PREVIEW_EXIT_BRANCHING"></span>**Просмотр ветвления выхода**
</dt> <dd>

Воспроизводит ветвь выхода анимации, начиная с выбранного кадра.

</dd> </dl>

:::image type="icon" source="images/f24charstop.gif":::

<dl> <dt>

<span id="Stop_Preview"></span><span id="stop_preview"></span><span id="STOP_PREVIEW"></span>**Прерывать предварительную версию**
</dt> <dd>

Останавливает воспроизведение предварительной версии анимации.

</dd> </dl>

:::image type="icon" source="images/f25charadd.gif":::

<dl> <dt>

<span id="Add_Image_File"></span><span id="add_image_file"></span><span id="ADD_IMAGE_FILE"></span>**Добавить файл изображения**
</dt> <dd>

Отображает диалоговое окно Выбор файла изображения. Выбранные изображения добавляются в список.

</dd> </dl>

:::image type="icon" source="images/f26charmvup.gif":::

<dl> <dt>

<span id="Move_Up"></span><span id="move_up"></span><span id="MOVE_UP"></span>**Вверх**
</dt> <dd>

Перемещает изображение вверх в упорядоченном (z-порядке) списке. В списке изображений фрейма изображение перемещается вверх в визуальном z-порядке.

</dd> </dl>

:::image type="icon" source="images/f27charmvdwn.gif":::

<dl> <dt>

<span id="Move_Down"></span><span id="move_down"></span><span id="MOVE_DOWN"></span>**Переместить вниз**
</dt> <dd>

Перемещает изображение вниз в упорядоченном (z-упорядоченном) списке. В списке изображений фрейма изображение перемещается вниз в визуальном z-порядке.

</dd> </dl>
