# <a name="advanced-character-options-window-voice-commands-window"></a>Окно "Дополнительные параметры символов" (окно "Voice Commands")

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

В окне Дополнительные параметры символов можно настроить параметры взаимодействия пользователей со всеми символами. Например, пользователи могут отключить речевой ввод или изменить входные параметры. Пользователи также могут изменять параметры вывода для всплывающего окна Word. Эти параметры переопределяют любой набор, заданный клиентским приложением или заданный в качестве части определения символа. Приложение не может изменить или отключить эти параметры, так как они применяются к общим предпочтениям пользователя для работы со всеми символами. Однако сервер будет уведомлять ваше приложение ([**дефаултчарактерчанже**](defaultcharacterchange-event.md)), когда пользователь изменит и применит параметр. Можно также отобразить или закрыть окно с помощью свойства [**Visible**](visible-property.md) окна и получить доступ к его расположению через его свойства [**Top**](top-property.md) и [**Left**](left-property.md) .

Помимо поддержки анимации символа, Microsoft Agent поддерживает звуковые выходные данные для символа. Сюда входят выходные и звуковые эффекты. Для выходного выхода сервер автоматически выполняет синхронизацию определенных для символа изображений рот с выходными данными. Можно выбрать синтез текста в речь, записанные аудио-файлы или только текст всплывающей подсказки.
