# <a name="agent-control-properties"></a>Свойства элемента управления агента

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

К следующим свойствам напрямую обращаются из элемента управления агента:

-   [**Подключен**](connected-property.md)
-   [**Имя**](name-property-a.md)
-   [**раисерекуестеррорс**](raiserequesterrors-property.md)

Кроме того, некоторые среды программирования могут назначать дополнительные свойства времени разработки или времени выполнения. Например, Visual Basic добавляет свойства Left, index, Tag и Top, определяющие расположение элемента управления в форме, даже если элемент управления не отображается на странице формы во время выполнения.

Свойство suspended по-прежнему поддерживается для обратной совместимости, но всегда возвращает **значение false** , так как сервер больше не поддерживает состояние SUSPENDED.
