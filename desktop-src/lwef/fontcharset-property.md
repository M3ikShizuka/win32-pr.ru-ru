# <a name="fontcharset-property"></a>Фонтчарсет, свойство

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает или задает кодировку для шрифта, отображаемого в тексте всплывающего окна указанного символа.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Агент ***. Символы ("**_чарактерид_*_"). Значение всплывающего окна. фонтчарсет_ *  \[  =  \]



| Часть    | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
|---------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *value* | Целочисленное значение, указывающее кодировку, используемую шрифтом. ниже приведены некоторые общие параметры для значения: 0 стандартных Windows символов (ANSI).<br/> 1 набор символов по умолчанию.<br/> 2. кодировка символов.<br/> 128. двухбайтовая кодировка (DBCS), уникальная для японской версии Windows.<br/> 129. двухбайтовая кодировка (DBCS), уникальная для корейской версии Windows.<br/> 134. двухбайтовая кодировка (DBCS), уникальная для упрощенной версии Windows.<br/> 136. кодировка двухбайтовых символов (DBCS) уникальна для традиционного китайского версии Windows.<br/> 255. Расширенные символы обычно отображаются приложениями Microsoft MS-DOS.<br/> Дополнительные сведения о других значениях наборов символов см. в документации по пакету SDK для платформы.<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Remarks

Значение по умолчанию для набора символов всплывающего слова символа задается в редакторе символов Microsoft Agent. Кроме того, пользователь может переопределить параметры набора символов для всех символов на странице свойств Microsoft Agent.

Это свойство применяется только к символу, используемому вашим клиентским приложением. Этот параметр не влияет на другие клиенты символов или других символов клиентского приложения.

> [!Note]  
> Если вы используете некомпилированный символ, проверьте свойства [**FontName**](fontname-property.md) и **фонтчарсет** для символа, чтобы определить, подходит ли он для вашего языкового стандарта. Возможно, потребуется задать эти значения перед использованием метода [**говорите**](speak-method.md) , чтобы обеспечить отображение текста в выноске слова.
