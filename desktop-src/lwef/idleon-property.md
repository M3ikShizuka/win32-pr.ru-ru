# <a name="idleon-property"></a>Идлеон, свойство

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает или задает логическое значение, определяющее, управляет ли сервер **состояние простоя** анимацией состояния для указанного символа.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Агент ***. Символы ("**_чарактерид_*_"). Идлеон_ *  \[  =  *Boolean*\]

</dd> </dl> 

| Часть      | Описание                                                                                                                                                                                                             |
|-----------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *boolean* | Логическое выражение, указывающее, управляет ли сервер режимом простоя. **True** (по умолчанию) Обработка состояния простоя сервера включена. <br/> **Значение false** Обработка состояния простоя сервера отключена. <br/> |



 

## <a name="remarks"></a>Remarks

Сервер автоматически устанавливает время ожидания после последней анимации для символа. По завершении этого интервала времени сервер начинает **состояние простоя** состояние символа, **состояние простоя** его анимацию с регулярными интервалами. Если вы хотите отключить сервер от автоматического воспроизведения анимации состояния **состояние простоя** , установите свойство в **значение false** и Воспроизведите анимацию или вызовите метод [**остановите**](stop-method.md) . Установка этого значения не влияет на текущее состояние анимации символа.

Это свойство применяется только к символу, используемому вашим клиентским приложением. Этот параметр не влияет на другие клиенты символов или других символов клиентского приложения.
