# <a name="width-property-characters-object"></a>Свойство Width (объект characters)

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает или задает ширину кадра для указанного символа.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Syntax** 
</dt> <dd>

*Агент ***. Символы ("**_чарактерид_*_")._ *  \[ =  *Значение* ширины\]



| Часть    | Описание                                                |
|---------|------------------------------------------------------------|
| *value* | Длинное целое число, указывающее ширину кадра символа. |



 

</dd> </dl>

## <a name="remarks"></a>Remarks

Свойство [**Width**](width-property.md) всегда выражается в пикселях. Параметр этого свойства применяется ко всем клиентам символа.

Несмотря на то, что символ отображается в окне области неправильной формы, расположение символа основано на внешних измерениях прямоугольного кадра анимации, используемого при компиляции символа с помощью редактора символов Microsoft Agent.
