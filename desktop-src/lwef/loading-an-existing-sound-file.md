# <a name="loading-an-existing-sound-file"></a>Загрузка существующего звукового файла

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

можно также загрузить существующий звуковой файл Windows (. wav) или лингвистическо улучшенный звуковой файл (. лвв), выбрав команду **открыть** в меню **файл** или на панели инструментов. Откроется диалоговое окно **Открыть** . Выберите файл и нажмите кнопку **Открыть** , чтобы загрузить файл в редактор.
