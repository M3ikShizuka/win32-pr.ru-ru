# <a name="ipropertyfiltercollection-interface"></a>Интерфейс Ипропертифилтерколлектион

> [!NOTE]
> Windows Desktop Search 2. x — это устаревшая технология, которая изначально была доступна в качестве надстройки для Windows XP и Windows Server 2003. В более поздних выпусках используйте [API поиска Windows](../search/-search-reference-entry-page.md) . 

Обеспечивает доступ к свойствам возвращенной коллекции на основе отправленного запроса.

## <a name="members"></a>Элементы

Интерфейс **ипропертифилтерколлектион** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Ипропертифилтерколлектион** также имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Интерфейс **ипропертифилтерколлектион** имеет следующие свойства.



| Свойство                                                                         | Тип доступа           | Описание                                                  |
|:---------------------------------------------------------------------------------|:----------------------|:-------------------------------------------------------------|
| [**AddItem**](-search-2x-ipropertyfiltercollection-additem.md)<br/>       | Только для чтения<br/>  | Добавляет новый фильтр в коллекцию. <br/>             |
| [**открытым**](-search-2x-ipropertyfiltercollection-clear.md)<br/>           | Только на запись<br/> | Очищает коллекцию. <br/>                           |
| [**Расчета**](-search-2x-ipropertyfiltercollection-count.md)<br/>           | Только для чтения<br/>  | Число фильтров в коллекции. <br/>             |
| [**GetITem**](-search-2x-ipropertyfiltercollection-getitem.md)<br/>       | Чтение/запись<br/> | Возвращает конкретный фильтр в коллекции. <br/> |
| [**RemoveItem**](-search-2x-ipropertyfiltercollection-removeitem.md)<br/> | Только на запись<br/> | Удаляет конкретный фильтр в коллекции. <br/>     |



 

## <a name="remarks"></a>Комментарии

Эти свойства используются для фильтрации коллекции, возвращаемой запросом.
