# <a name="idlecomplete-event"></a>Событие Идлекомплете

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Происходит, когда сервер завершает состояние **состояние простоя** символа.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

 *Подагент * * * \_ идлекомплете* *  **(ByVal** *чарактерид * *)**



| Часть          | Описание                                         |
|---------------|-----------------------------------------------------|
| *чарактерид* | Возвращает идентификатор состояние простоя символа в виде строки. |



 

</dd> </dl>

### <a name="remarks"></a>Remarks

Сервер отправляет это событие всем клиентам символа.
