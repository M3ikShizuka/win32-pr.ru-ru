# <a name="iagentballoonsetvisible"></a>Иажентбаллун:: Сетвисибле

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

``` syntax
HRESULT SetVisible(
   long bVisible  // word balloon Visible setting 
);
```

Задает свойство [**Visible**](visible-property.md) для всплывающего окна Word.

-   Возвращает значение S \_ , чтобы указать, что операция выполнена успешно.

<dl> <dt>

<span id="bVisible"></span><span id="bvisible"></span><span id="BVISIBLE"></span>*бвисибле*
</dt> <dd>

Параметр видимого свойства. Значение **true** показывает всплывающую подсказку слова; значение **false** скрывает его.

</dd> </dl>
