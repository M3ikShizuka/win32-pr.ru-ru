# <a name="hide-event"></a>Скрыть событие

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Происходит при скрытии символа.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

 *Подагент * * * \_ Hide (* *  **ByVal** *чарактерид*, **ByVal** , *Причина * * *)**



| Отделение          | Описание                                                                                                                                                                                                                                                                                                                                                                                                       |
|---------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *чарактерид* | Возвращает идентификатор скрытого символа в виде строки.                                                                                                                                                                                                                                                                                                                                                               |
| *Причина*       | Возвращает значение, указывающее, что привело к скрытию символа. 1 пользователь скрыл символ, выбирая команду во всплывающем меню значка символа на панели задач или используя речевой ввод.<br/> 3 клиентское приложение скрыло символ.<br/> 5 другое клиентское приложение скрыло символ.<br/> 7. пользователь скрыл символ, выбрав команду во всплывающем меню символа.<br/> |



 

</dd> </dl>

### <a name="remarks"></a>Комментарии

Сервер отправляет это событие всем клиентам символа. Чтобы запросить текущее состояние символа, используйте свойство [**Visible**](visible-property.md) .
