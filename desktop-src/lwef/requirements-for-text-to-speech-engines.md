# <a name="requirements-for-text-to-speech-engines"></a>Требования для модулей преобразования текста в речь

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Подсистема должна быть полностью совместимой с SAPI 4,0. Кроме того, подсистема также должна поддерживать следующие интерфейсы SAPI для помеченных текста и уведомлений закладок. Эти интерфейсы позволяют Microsoft Agent быстро выводить текст на всплывающую подсказку и пакет LIP — синхронизировать рот символа (или его эквивалент) с произнесенными словами.

-   [иттсцентралв](ittscentralw.md)
-   [иттснотифисинкв](ittsnotifysinkw.md)
-   [иттсбуфнотифисинкв](ittsbufnotifysinkw.md)
-   [иттсаттрибутесв](ittsattributesw.md)
