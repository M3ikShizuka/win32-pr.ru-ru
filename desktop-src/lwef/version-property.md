# <a name="version-property"></a>Свойство Version

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает версию символа.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*символы Agent. ***("**_чарактерид_*_"). Версия_*

</dd> </dl>

## <a name="remarks"></a>Remarks

Свойство **Version** возвращает строку, соответствующую версии стандартного определения набора анимации, для которого был скомпилирован символ. Номер версии символа автоматически задается при сборке с помощью редактора символов Microsoft Agent.
