# <a name="move-event"></a>Переместить событие

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Происходит при перемещении символа.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

Перемещение агента в **подсистеме**  \_ **(ByVal** *чарактерид*, **ByVal** *X*, **ByVal** *Y*, **ByVal** , *Причина * * *)**



| Часть          | Описание                                                                                                                                                                                                                                                                                                                                   |
|---------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *чарактерид* | Возвращает идентификатор перемещаемого символа.                                                                                                                                                                                                                                                                                                   |
| *X*           | Возвращает координату по оси x (в пикселях) верхнего края нового расположения символьного кадра в виде целого числа.                                                                                                                                                                                                                                         |
| *да*           | Возвращает координату по оси y (в пикселях) левого края нового расположения символьного кадра в виде целого числа.                                                                                                                                                                                                                                        |
| *Причина*       | Возвращает значение, указывающее, что привело к перемещению символа. 1 Пользователь переместил символ.<br/> 2 клиентское приложение переместило символ.<br/> 3 другое клиентское приложение переместило этот символ.<br/> 4 сервер агента переместил символ, чтобы он оставался на экране после изменения разрешения экрана.<br/> |



 

</dd> </dl>

### <a name="remarks"></a>Remarks

Это событие возникает, когда пользователь или приложение изменяет расположение символа. Координаты имеют отношение к левому верхнему углу экрана. Это событие отправляется только клиентам символа (приложения, которые загрузили символ).
