# <a name="showpopupmenu-method"></a>Метод Шовпопупмену

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Отображает всплывающее меню символа в указанном расположении.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Символы Agent. ***("**_чарактерид_*_"). Шовпопупмену_ * _x, y_



| Часть | Описание                                                                                                                                          |
|------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| *x*  | Обязательный. Целочисленное значение, указывающее горизонтальную (*x*) координату экрана для отображения меню. Эти координаты должны быть указаны в пикселях. |
| *y*  | Обязательный. Целочисленное значение, указывающее координату вертикального экрана (*y*) для отображения меню. Эти координаты должны быть указаны в пикселях.   |



 

</dd> </dl>

## <a name="remarks"></a>Remarks

Агент автоматически отображает всплывающее меню символа, когда пользователь щелкает правой кнопкой мыши символ. Если для [**аутопопупмену**](autopopupmenu-property.md) задано **значение false**, можно использовать этот метод для вывода меню.

Меню остается отображаемым до тех пор, пока пользователь не выберет команду или не отобразит другое меню. В каждый момент времени может отображаться только одно всплывающее меню; Поэтому вызовы этого метода будут отменяться (удалить) из предыдущего меню.

Этот метод следует вызывать только в том случае, если клиентское приложение является активным клиентом символа. в противном случае произойдет сбой. Чтобы определить успешность этого метода, можно вызвать его как функцию, и она возвратит логическое значение, указывающее, успешно ли выполнен метод.


```
   If Genie.ShowPopupMenu (10,10) = True Then
      ' The menu will be displayed

   Else 
      ' The menu will not be displayed

   End If
```
