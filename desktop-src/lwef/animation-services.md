# <a name="animation-services"></a>Службы анимации

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Службы анимации Microsoft Agent управляют анимацией и перемещением изображения символа в отдельном окне на экране. Анимация определяется как последовательность времени и при необходимости разветвления кадров, состоящая из одного или нескольких изображений.

-   [Загрузка символа](loading-a-character.md)
-   [Загрузка символа по умолчанию](loading-the-default-character.md)
-   [Анимация символа](animating-a-character.md)
