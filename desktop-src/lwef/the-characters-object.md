# <a name="the-characters-object"></a>Объект characters

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Клиентское приложение может поддерживать один или несколько символов. Кроме того, можно совместно использовать один символ в нескольких приложениях. Microsoft Agent определяет объект [**characters**](/windows/desktop/lwef/the-characters-object) как набор символов. Чтобы получить доступ к символу, загрузите данные символов в коллекцию [**символов**](/windows/desktop/lwef/the-characters-object) и укажите, что элемент в коллекции использует методы и свойства, поддерживаемые для этого символа.

-   [Методы объекта character](characters-object-methods.md)
-   [Методы объекта character](character-object-methods.md)
-   [Свойства объекта Characters](character-object-properties.md)
