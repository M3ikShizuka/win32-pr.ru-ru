# <a name="designing-characters-for-microsoft-office"></a>Разработка символов для Microsoft Office

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

В этом документе содержатся сведения, которые могут помочь при проектировании и разработке символа Microsoft Agent для использования в Microsoft Office 2000 (символы Microsoft Agent не работают с Microsoft Office 97). В этом документе предполагается, что у вас есть опыт работы с редактором символов Microsoft Agent и сведения о создании символов Microsoft Agent. Дополнительные сведения о редакторе символов агентов и проектировании и разработке символов см. в [редакторе символов Microsoft Agent](using-the-microsoft-agent-character-editor.md). Дополнительные сведения о создании символов для Microsoft Agent предоставляются при [проектировании символов для Microsoft Agent](designing-characters-for-microsoft-agent.md).
