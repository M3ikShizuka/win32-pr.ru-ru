# <a name="fontitalic-property"></a>Фонтиталик, свойство

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает стиль шрифта для указанного символа, который в настоящий момент отображается в окне всплывающего окна Word.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Агент ***. Символы ("**_чарактерид_*_"). Всплывающее сообщение. Фонтиталик_*



| Значение     | Описание                     |
|-----------|---------------------------------|
| **True**  | Выноски выделены курсивом.     |
| **False** | Всплывающий шрифт не является курсивом. |



 

</dd> </dl>

## <a name="remarks"></a>Remarks

Значение по умолчанию для параметров шрифта всплывающей подсказки для символа задается в редакторе символов Microsoft Agent. Кроме того, пользователь может переопределить параметры шрифта для всех символов на странице свойств Microsoft Agent.
