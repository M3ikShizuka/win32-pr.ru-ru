# <a name="enabled-property-speech-input-object"></a>Свойство Enabled (объект речевого ввода)

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает логическое значение, указывающее, включен ли речевой ввод.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*агент * * *. Спичинпут. Enabled**



| Значение     | Описание               |
|-----------|---------------------------|
| **True**  | Речевой ввод включен.  |
| **False** | Речевой ввод отключен. |



 

</dd> </dl>

## <a name="remarks"></a>Remarks

Свойство [**Enabled**](enabled-property.md) отражает параметр прослушивать входные данные на странице "речевое вход" страницы свойств агента (дополнительные параметры символов). Параметр свойства влияет на все символы агента и доступен только для чтения; только пользователь может изменить это свойство.
