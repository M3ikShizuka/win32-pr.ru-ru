# <a name="animation-design"></a>Разработка анимации

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

### <a name="image-design"></a>Разработка образа

палитра Microsoft Office используется при проектировании символов, чтобы максимально сокращать возможные проблемы при реализации палитры. Избегайте выбора цвета прозрачности, аналогичного цвету, используемому в документе.

### <a name="sounds"></a>Звуки

Microsoft Agent позволяет воспроизводить звуки в анимации. Мы рекомендуем не включать звуки для **неактивных** анимаций. Это значит, что задержка в середине анимации не будет задерживаться, если агент должен загрузить системную БИБЛИОТЕКУ мультимедиа.

### <a name="frame-size"></a>Размер кадра

типичные помощники Office 123 x 93 пикселей. Хотя можно создавать символы других размеров, они будут масштабироваться до 123 x 93 в галерее помощников.

### <a name="frame-transition"></a>Переход кадров

Все **анимации, за** исключением конца, **приветствия**, **показа** и **скрытия** , должны начинаться и заканчиваться анимацией рестпосе. Microsoft Office не воспроизводит явные анимации **возврата** , поэтому их не следует определять. Все анимации также должны иметь ветвь Exit. Выход из ветви позволяет нам "Спешите вверх и завершить" текущую анимацию перед вызовом следующей анимации. Если ветвь Exit не указана, переход между анимациями может быть жерки.

### <a name="character-properties"></a>Свойства символов

Microsoft Agent позволяет задать свойства [**Name**](name-property.md), [**Description**](description-property.md) и [**екстрадата**](extradata-property.md) символа. Microsoft Office использует поле **екстрадата** для хранения одной или нескольких вводной фраз и фраз напоминаний. Microsoft Office выбрать другие фразы для введения в выноску в галерее помощника. Мы используем фразы напоминания при получении напоминания от Outlook.

Поле [**екстрадата**](extradata-property.md) имеет следующий формат:

IntroPhrase1~~IntroPhrase2~~IntroPhrase3 ^ ^ ReminderPhrase1~~ReminderPhrase2~~ReminderPhrase3

Вводные фразы разделяются парой символов тильды (~), за которой следуют фразы с напоминаниями. Эти фразы-напоминания также разделяются парой символов тильды. Два набора фраз разделяются двумя знаками вставки (^^). Количество таких фраз не ограничено, за исключением того, что по крайней мере один из них должен иметь значение.
