# <a name="iagentcharacterstopall"></a>Иажентчарактер:: Стопалл

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

``` syntax
HRESULT StopAll();
   long lType,  // request type
```

Останавливает все анимации (запросы) и удаляет их из очереди анимации символа.

<dl> <dt>

<span id="lType"></span><span id="ltype"></span><span id="LTYPE"></span>*лтипе*
</dt> <dd>

Битовое поле, указывающее типы запросов, которые нужно приостанавливаться (и которые удаляются из очереди символов), состоящие из следующих элементов:



|                                                                                   |                                                                                                          |
|-----------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| Тип **беззнаковой неподписанного типа Long без знака** **\_ \_ ALL = 0xFFFFFFFF;**<br/>              | Останавливает все запросы анимации, включая запросы на [**подготовку**](iagentcharacter--prepare.md) , не связанные с очередями. |
| Тип **постоянного неподписанного длинного** **\_ типа " \_ воспроизвести = 0x00000001;** "<br/>             | Останавливает все запросы на воспроизведение.                                                                                 |
| **неподписанный длинный** **тип останавливаемого типа беззнакового \_ \_ перемещения = 0x00000002;**<br/>             | Останавливает все запросы на [**Перемещение**](https://www.bing.com/search?q=**Move**) .                                               |
| Тип **неподписанного длинного типа Long без знака** **\_ \_ : выговаривать = 0x00000004;**<br/>            | Останавливает все запросы на [**Распознавание речи**](iagentcharacter--speak.md) .                                              |
| **неподписанный длинный тип беззнакового** **\_ типа завершения \_ Prepare = 0x00000008;**<br/>          | Останавливает все отложенные запросы на [**подготовку**](iagentcharacter--prepare.md) .                                   |
| Тип **однобайтового типа const беззнаковых** **\_ типов \_ нонкуеуедпрепаре = 0x00000010;**<br/> | Останавливает все запросы на [**подготовку**](iagentcharacter--prepare.md) , не относящиеся к очереди.                               |
| Невидимый тип **беззнакового неподписанного** **\_ типа long \_ = 0x00000020;**<br/>          | Останавливает все запросы на [**Скрытие**](iagentcharacter--hide.md) или [**Отображение**](iagentcharacter--show.md) .       |



 

</dd> </dl>
