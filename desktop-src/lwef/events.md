# <a name="iagentnotifysink"></a>иажентнотифисинк

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Иажентнотифисинк уведомляет клиентов при возникновении определенных изменений состояния. Эти функции также доступны из [иажентнотифисинкекс](iagentnotifysinkex.md).

**Методы в порядке таблицы Vtable**



| иажентнотифисинк                                                      | Описание                                                                              |
|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| [**Кнопки**](command-method.md)                                     | Происходит, когда сервер обрабатывает команду, определяемую клиентом.                               |
| [**активатеинпутстате**](iagentnotifysink--activateinputstate.md)    | Происходит, когда символ попадает в активное значение или перестает быть активным.                            |
| [**баллунвисиблестате**](iagentnotifysink---balloonvisiblestate.md) | Происходит при изменении **видимого** состояния символа.                                   |
| [**Событие "Щелчок"**](click-event.md)                                    | Происходит при щелчке символа.                                                      |
| [**Двойное событие**](dblclick-event.md)                              | Происходит при двойном щелчке символа.                                               |
| [**драгстарт**](/windows/desktop/lwef/dragstart-event)                                | Происходит, когда пользователь начинает перетаскивать символ.                                          |
| [**драгкомплете**](https://www.bing.com/search?q=**DragComplete**)                          | Происходит, когда пользователь останавливает перетаскивание символа.                                           |
| [**RequestStart**](iagentnotifysink--requeststart.md)                | Происходит, когда сервер начинает обработку объекта [**запроса**](/windows/desktop/lwef/the-request-object) .    |
| [**рекуесткомплете**](iagentnotifysink--requestcomplete.md)          | Происходит, когда сервер завершает обработку объекта [**запроса**](/windows/desktop/lwef/the-request-object) . |
| [**Закладка**](iagentnotifysink--bookmark.md)                        | Происходит, когда сервер обрабатывает закладку.                                             |
| [**Выключен**](iagentnotifysink--idle.md)                                | Происходит, когда сервер запускает или завершает обработку бездействия.                                   |
| [**Переместить**](iagentnotifysink--move.md)                                | Происходит при перемещении символа.                                                  |
| [**Размер**](iagentnotifysink---size.md)                               | Происходит при изменении размера символа.                                                |
| [**баллунвисиблестате**](iagentnotifysink---balloonvisiblestate.md) | Происходит при изменении состояния видимости для всплывающего слова символа.                  |



 

События Иажентнотифисинк:: Restart и Иажентнотифисинк:: Shutdown, поддерживаемые в более ранних версиях Microsoft Agent, теперь устарели. Так как поддерживается для обратной совместимости, сервер больше не отправляет эти события.
