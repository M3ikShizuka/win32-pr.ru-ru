# <a name="the-listening-tip"></a>Совет прослушивания

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Совет прослушивания — это еще одна служба речевого ввода, предоставляемая агентом Майкрософт. При установке речевого ввода агент включает специальное окно подсказки, которое появляется, когда пользователь нажимает клавишу прослушивания или вызывает метод Listen. Совет прослушивания отображается только в том случае, если доступны службы распознавания речи. Если ни один клиент не создал голосовую команду или не загружает модуль распознавания речи, то TIP прослушивания не отображается. Кроме того, для отображения Совета должен быть включен параметр "речевое вход" и "Отображать подсказки для прослушивания" в дополнительных параметрах символов.

Время ожидания Совета прослушивания истекает автоматически. Если время ожидания текста "Херд" завершается, а пользователь по-прежнему удерживает нажатую клавишу, Tip возвращается к тексту "прослушивание", если сервер не получает другой соответствующий utterance. В этом случае подсказка отображает новый текст «Херд» и начинает истечение времени ожидания для этого текста подсказки. Если пользователь отпускает нажатую клавишу, а сервер отображает текст «Херд», время ожидания истекает, а окно TIP прослушивания скрывается по истечении интервала времени ожидания.

Если сервер еще не пытался загрузить модуль распознавания речи, подсказка прослушивания не отображается. Аналогичным образом, если пользователь отключил отображение Совета прослушивания или отключил речевой ввод в дополнительных параметрах символов, подсказка прослушивания не будет отображаться.

Совет прослушивания не отображается, если указатель находится над значком на панели задач символа. Вместо этого отображается стандартное окно подсказок по уведомлениям, в котором отображается имя символа.

Клиентские приложения не могут выполнять запись непосредственно в Tip прослушивания, но можно указать альтернативный текст, отображаемый сервером при распознавании соответствующей Voice-команды. Для этого задайте свойство [**достоверности**](confidence-property.md) и новое свойство [**конфиденцетекст**](confidencetext-property.md)для команды. Если речевое входное значение соответствует команде, но наилучшее соответствие не превышает степень достоверности, сервер использует текст, заданный в свойстве **конфиденцетекст** в окне подсказки. Если клиент не предоставил это значение, сервер отображает текст (грамматика), которому он соответствует.

Текст подсказки прослушивания отображается на языке, основанном на параметре идентификатор языка символов ввода активного клиента, независимо от того, доступен ли модуль распознавания речи, совместимый с языком.
