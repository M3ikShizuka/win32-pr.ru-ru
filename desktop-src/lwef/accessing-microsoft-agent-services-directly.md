# <a name="accessing-microsoft-agent-services-directly"></a>Прямой доступ к службам Microsoft Agent Services

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

если вы используете C, C++ или Java, вы можете получить доступ к серверу Microsoft Agent напрямую с помощью интерфейсов ActiveX (OLE). Дополнительные сведения об этих интерфейсах см. в разделе [программирование интерфейса сервера агента Microsoft Agent](programming-the-microsoft-agent-server-interface.md).
