# <a name="status-property"></a>Свойство Status

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает состояние канала вывода звука.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*агент * * *. Аудиуутпут. status**



| Значение | Описание                                                                                                       |
|-------|-------------------------------------------------------------------------------------------------------------------|
| 0     | Канал звукового вывода доступен (не занят).                                                                 |
| 1     | Звуковые выходные данные не поддерживаются; Например, из-за отсутствия звуковой платы.                                |
| 2     | Не удается открыть канал вывода звука (занят); Например, так как другое приложение воспроизводит звук. |
| 3     | Канал вывода звука занят, так как сервер обрабатывает речевой ввод пользователя.                              |
| 4     | Канал вывода звука занят, так как в настоящий момент символ говорит.                                       |
| 5     | Канал вывода звука не занят, но ожидает ввода речи пользователем.                                    |
| 6     | Возникла другая (неизвестная) проблема при попытке доступа к каналу вывода звука.                          |



 

</dd> </dl>

## <a name="remarks"></a>Комментарии

Этот параметр позволяет клиентскому приложению запрашивать канал вывода звука, возвращая целочисленное значение, которое указывает состояние канала вывода звука. Его можно использовать, чтобы определить, подходит ли символ для диктовки или нужно ли попытаться включить режим прослушивания (с помощью метода [**Listen**](listen-method.md) ).
