# <a name="use-natural-variation"></a>Использовать естественный вариант

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Согласованность представления в обычном интерфейсе приложения, например меню и диалоговых окнах, делает интерфейс более предсказуемым, изменяя анимацию и речевые выходные данные в интерфейсе символа. Соответствующим образом менять ответы на символы предоставляют более естественный интерфейс. Если символ всегда обращается к пользователю точно таким же образом; Например, всегда говорят о одних и тех же словах, пользователь, скорее всего, будет считать символ скучным, дисинтерестед или даже грубым. Взаимодействие с человеком редко подразумевает точное повторение. Даже при повторении чего-то в подобной ситуации мы можем изменить слова, жесты или выражение лица.

Microsoft Agent позволяет создавать отдельные варианты для символа. При определении анимации символа можно использовать вероятности ветвления в любом кадре анимации, чтобы изменить анимацию при ее воспроизведении. Можно также назначить несколько анимаций для каждого состояния. Microsoft Agent случайным образом выбирает одну из назначенных анимаций при каждом запуске состояния. Для речевого вывода можно также включить в выходной текст символы вертикальной черты, чтобы автоматически изменять текст. Например, Microsoft Agent случайным образом выберет одну из следующих инструкций при обработке этого текста как части метода [**говорите**](speak-method.md) :

«Я могу сказать это. \| Я могу сказать, что. \| Я могу сказать что-то другое».
