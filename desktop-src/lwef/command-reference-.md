# <a name="command-reference-microsoft-agent-character-editor"></a>Справочник по командам (редактор символов Microsoft Agent)

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

## <a name="the-file-menu"></a>Меню «файл»

<dl> <dt>

<span id="New"></span><span id="new"></span><span id="NEW"></span>**Новые функции**
</dt> <dd>

Сбрасывает редактор символов агента для создания нового определения символа. Если существующий символ загружается и имеет несохраненные изменения, редактор отображает сообщение, чтобы определить, следует ли сохранять или удалять несохраненные изменения.

</dd> <dt>

<span id="Open"></span><span id="open"></span><span id="OPEN"></span>**Открыт**
</dt> <dd>

Открывает диалоговое окно Открытие файла, позволяющее открыть существующий файл определения символа для редактирования. Если существующий символ загружается и имеет несохраненные изменения в файле, то редактор отображает сообщение, чтобы определить, следует ли сохранять или удалять несохраненные изменения.

</dd> <dt>

<span id="Save"></span><span id="save"></span><span id="SAVE"></span>**Автоматически**
</dt> <dd>

Сохраняет определение символа. Если определение символа не существует (не было именовано), редактор отображает диалоговое окно Сохранить как для ввода имени файла.

</dd> <dt>

<span id="Save_As"></span><span id="save_as"></span><span id="SAVE_AS"></span>**Сохранить как**
</dt> <dd>

Отображает диалоговое окно Сохранить как, в котором можно ввести новое имя для файла определения символа.

</dd> <dt>

<span id="Print"></span><span id="print"></span><span id="PRINT"></span>**Печатал**
</dt> <dd>

Отображает диалоговое окно Печать, позволяющее выбрать режим печати и напечатать файл определения символа.

</dd> <dt>

<span id="Build_Character"></span><span id="build_character"></span><span id="BUILD_CHARACTER"></span>**Символ сборки**
</dt> <dd>

Отображает диалоговое окно символ сборки, включающее параметры для определения способа построения файлов данных и анимации символов для использования с Microsoft Agent.

</dd> <dt>

<span id="Page_Setup"></span><span id="page_setup"></span><span id="PAGE_SETUP"></span>**Параметры страницы**
</dt> <dd>

Отображает диалоговое окно Параметры страницы, позволяющее задать параметры печати для файла определения символа.

</dd> <dt>

<span id="Most_Recently_Open_Files"></span><span id="most_recently_open_files"></span><span id="MOST_RECENTLY_OPEN_FILES"></span>**Последние открытые файлы**
</dt> <dd>

Отслеживает последние открытые файлы определения символов. При выборе файла он автоматически открывается для редактирования. Если существующий символ загружается и имеет несохраненные изменения в файле, то редактор отображает сообщение, чтобы определить, следует ли сохранять или удалять несохраненные изменения.

</dd> <dt>

<span id="Exit"></span><span id="exit"></span><span id="EXIT"></span>**Выполняет**
</dt> <dd>

Завершает работу редактора символов агента. Если существующий символ загружается и имеет несохраненные изменения в файле, то редактор отображает сообщение, чтобы определить, следует ли сохранять или удалять несохраненные изменения.

</dd> </dl>

## <a name="the-edit-menu"></a>Меню «Правка»

<dl> <dt>

<span id="Undo"></span><span id="undo"></span><span id="UNDO"></span>**Отмен**
</dt> <dd>

Удаляет изменения, внесенные в редакторе.

</dd> <dt>

<span id="Redo"></span><span id="redo"></span><span id="REDO"></span>**Redo**
</dt> <dd>

Изменяет действие отмены в редакторе на противоположное.

</dd> <dt>

<span id="Cut"></span><span id="cut"></span><span id="CUT"></span>**Бавьте**
</dt> <dd>

Удаляет выбранный элемент из редактора и помещает его в буфер обмена Windows.

</dd> <dt>

<span id="Copy"></span><span id="copy"></span><span id="COPY"></span>**Копии**
</dt> <dd>

Копирует выбранный элемент из редактора в буфер обмена Windows.

</dd> <dt>

<span id="Paste"></span><span id="paste"></span><span id="PASTE"></span>**Авить**
</dt> <dd>

Копирует данные из текущего буфера обмена Windows в выбранное расположение.

</dd> <dt>

<span id="Delete"></span><span id="delete"></span><span id="DELETE"></span>**Удален**
</dt> <dd>

Удаляет выбранный элемент из редактора.

</dd> <dt>

<span id="New_Animation"></span><span id="new_animation"></span><span id="NEW_ANIMATION"></span>**Создать анимацию**
</dt> <dd>

Создает новый объект анимации в редакторе.

</dd> <dt>

<span id="New_Frame"></span><span id="new_frame"></span><span id="NEW_FRAME"></span>**Создать рамку**
</dt> <dd>

Создает новый кадр для анимации.

</dd> <dt>

<span id="New_Frames_from_Files"></span><span id="new_frames_from_files"></span><span id="NEW_FRAMES_FROM_FILES"></span>**Новые кадры из файлов**
</dt> <dd>

Отображает диалоговое окно Выбор файлов изображений и создает фреймы, используя выбранные файлы.

</dd> <dt>

<span id="Open_Frame_Window"></span><span id="open_frame_window"></span><span id="OPEN_FRAME_WINDOW"></span>**Открыть окно фрейма**
</dt> <dd>

Отображает текущий кадр и его изображения в отдельном окне без масштабирования изображений, загруженных в кадр.

</dd> <dt>

<span id="Preview___Stop_Preview"></span><span id="preview___stop_preview"></span><span id="PREVIEW___STOP_PREVIEW"></span>**Предварительный просмотр \| предварительной версии**
</dt> <dd>

Воспроизводит (или останавливает воспроизведение) анимацию, начиная с ее выбранного кадра.

</dd> <dt>

<span id="Preview___Stop_Preview_Exit_Branching"></span><span id="preview___stop_preview_exit_branching"></span><span id="PREVIEW___STOP_PREVIEW_EXIT_BRANCHING"></span>**Предварительная версия \| Отмена предварительной версии выйти из ветви**
</dt> <dd>

Воспроизводит (или останавливает воспроизведение) ветвление анимации, начиная с выбранного кадра.

</dd> </dl>
