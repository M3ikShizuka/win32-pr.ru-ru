# <a name="programming-the-microsoft-agent-control"></a>Программирование элемента управления Microsoft Agent

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Хотя приложения могут выполнять запись непосредственно в службы агента Майкрософт с помощью своих интерфейсов сервера автоматизации, Microsoft Agent также включает элемент управления ActiveX (OLE). Этот элемент управления поддерживает простое программирование с использованием языка сценариев, такого как Microsoft Visual Basic Scripting Edition (VBScript) или других языков, поддерживающих интерфейс элемента управления ActiveX.
