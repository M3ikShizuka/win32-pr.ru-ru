# <a name="command-method"></a>Метод Command

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает объект [**Command**](/windows/desktop/lwef/the-command-object) в коллекции [**Commands**](/windows/desktop/lwef/the-commands-collection-object) .

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**

</dd> </dl>

## <a name="remarks"></a>Комментарии

Этот метод можно использовать для доступа к свойствам объекта [**команды**](/windows/desktop/lwef/the-command-object) .

> [!Note]  
> Этот метод может потребоваться для некоторых языков программирования. Он не требуется для VBScript или Visual Basic. Дополнительные сведения об указании методов **команд** см. в разделе Свойства объекта команды.
