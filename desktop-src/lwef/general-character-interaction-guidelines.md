\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Принципы хорошего дизайна пользовательского интерфейса также применяются к проектированию интерфейса с помощью Microsoft Agent. Необходимо понимать пользователей и их цели, а также следовать хорошим процессам разработки пользовательского интерфейса. Рекомендации по интерфейсу Windows для проектирования программного обеспечения предоставляют отличный обзор общих принципов и методологии разработки.
