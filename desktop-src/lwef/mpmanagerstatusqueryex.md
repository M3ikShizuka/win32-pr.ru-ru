# <a name="mpmanagerstatusqueryex-function"></a>Функция Мпманажерстатускуерекс

Возвращает сведения о состоянии различных компонентов диспетчера защиты от вредоносных программ.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT WINAPI MpManagerStatusQueryEx(
  _In_  MPHANDLE       hMpHandle,
  _In_  DWORD          dwFlag,
  _Out_ PMPSTATUS_INFO pStatusInfo
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хмфандле* \[ окне\]
</dt> <dd>

Тип: **мфандле**

Обработчик для интерфейса диспетчера защиты от вредоносных программ. Этот маркер возвращается функцией [**мпманажеропен**](mpmanageropen.md) .

</dd> <dt>

*dwFlag* \[ окне\]
</dt> <dd>

Тип: **DWORD**

Определяет, какие сведения о запросах возвращаются. Некоторые сведения являются дорогостоящими и могут не потребоваться.



| Значение                                                                                                                                                                                                         | Значение                                        |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------|
| <span id="MP_STATUS_QUERY_FLAGS_NONE"></span><span id="mp_status_query_flags_none"></span><dl> <dt>**\_ \_ флаги запроса состояния пакета управления \_ \_ нет**</dt> </dl>       | По умолчанию.<br/>                            |
| <span id="MP_STATUS_QUERY_FLAG_NOSTATE"></span><span id="mp_status_query_flag_nostate"></span><dl> <dt>**\_флаг запроса состояния пакета управления " \_ \_ \_ несостояние"**</dt> </dl> | Не запрашивать сведения о состоянии.<br/> |



 

</dd> <dt>

*пстатусинфо* \[ заполняет\]
</dt> <dd>

Тип: **пмпстатус \_ info** .

Указатель на структуру, которая возвращает сведения о состоянии для последних просмотров, активных угроз и состояния различных компонентов. См [**. \_ сведения о мпстатус**](mpstatus-info.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

Если функция выполнена успешно, возвращается значение **S \_**. Вызов этой функции гарантированно будет выполнен, даже если служба защиты от вредоносных программ не запущена.

Если функция завершается ошибкой, возвращаемое значение является неудачным кодом **HRESULT** . Вызывающий объект может использовать функцию [**мперрормессажеформат**](mperrormessageformat.md) для получения общего описания сообщения об ошибке.
