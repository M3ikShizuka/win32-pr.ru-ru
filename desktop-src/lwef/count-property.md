# <a name="count-property"></a>Count, свойство

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает длинное целое число (свойство только для чтения), которое указывает количество объектов [**Command**](/windows/desktop/lwef/the-command-object) в коллекции [**Commands**](/windows/desktop/lwef/the-commands-collection-object) .

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Агент ***. Символы ("**_чарактерид_*_"). Команды. Count_*

</dd> </dl>

## <a name="remarks"></a>Remarks

**Count** включает только число объектов [**Command**](/windows/desktop/lwef/the-command-object) , определенных в коллекции [**команд**](/windows/desktop/lwef/the-commands-collection-object) . Сервер или другие записи клиента не включены.
