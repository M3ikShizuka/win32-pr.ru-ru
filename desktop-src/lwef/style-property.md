# <a name="style-property"></a>Свойство Style

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает или задает стиль вывода слова для символа.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Символы Agent. ***("**_чарактерид_*_"). Стиль всплывающего_ *  \[  =   окна\]



| Часть    | Описание                                                                                                                                                                                                                                                                       |
|---------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *Стиль* | Целое число, представляющее стиль вывода всплывающего окна. Параметр стиля — это битовое значение битов, соответствующих: всплывающее сообщение (бит 0), размер к тексту (бит 1), автоматическое скрытие (бит 2), автоматическая скорость (бит 3), число символов в строках (бит 16-23) и число строк (бит 24-31). |



 

</dd> </dl>

## <a name="remarks"></a>Remarks

Если в качестве всплывающего элемента стиля задано значение 1, то при использовании метода [**говорите**](https://www.bing.com/search?q=**Speak**) или [**мышления**](think-method.md) отображается всплывающая подсказка, если пользователь не переопределит этот параметр на странице свойств Microsoft Agent. Если задано значение 0, всплывающее сообщение не отображается.

Если для бита стиля размера в тексте задано значение 1, то всплывающая подсказка автоматически изменяет высоту всплывающего окна на текущий размер текста для оператора [**говорите**](https://www.bing.com/search?q=**Speak**) или « [**подумать**](think-method.md) ». Если задано значение 0, Высота всплывающего окна зависит от значения свойства [**нумберофлинес**](numberoflines-property.md) . Если этот бит стиля имеет значение 1 и вы пытаетесь задать свойство **нумберофлинес** , агент выдает ошибку.

Если бит стиля автоматического скрытия имеет значение 1, то при завершении речевого вывода текст всплывающего окна автоматически скрывается. Если задано значение 0, всплывающее сообщение остается видимым до следующего [**произношения**](https://www.bing.com/search?q=**Speak**) [**или вызова**](think-method.md) , если символ скрыт, либо пользователь щелкает или перетаскивает этот символ.

Если для бита автоматической установки задано значение 1, то всплывающая подсказка выводится на основе текущей скорости вывода, например по одному слову за раз. Когда вывод превышает размер всплывающего окна, первый текст автоматически прокручивается. Если задано значение 0, весь текст, включенный в оператор « [**говорите**](https://www.bing.com/search?q=**Speak**) » или « [**подумать**](think-method.md) », отображается одновременно.

Значение, чтобы извлечь только четыре младших бита **и** значение, возвращаемое **Style** с 255. Для задания битового значения **или** значения, возвращаемого со значением битов, которые вы хотите задать. Чтобы отключить бит **и** значение, возвращаемое с дополнением к биту, выполните одно из следующих действий.


```
   Const BalloonOn = 1

   ' Turn the word balloon off
   Genie.Balloon.Style = Genie.Balloon.Style And (Not BalloonOn)
   Genie.Speak "No balloon"

   ' Turn the word balloon on
   Genie.Balloon.Style = Genie.Balloon.Style Or BalloonOn
   Genie.Speak "Balloon"
```



Свойство **Style** также возвращает количество символов в строке нижнего байта верхнего слова и число строк в верхнем байте верхнего слова. Хотя это может быть проще в чтении с помощью свойств [**чарсперлине**](charsperline-property.md) и [**Нумберофлинес**](numberoflines-property.md), свойство **Style** также позволяет задавать эти значения.

Например, чтобы изменить число строк, очистите биты 24 – 31 с логической операцией **и** , прежде чем устанавливать новое значение в качестве произведения нового значения, равного 2 ^ 24, добавленному к существующему значению свойства **Style** .


```
   ' Set the number of lines to 4
   Genie.Balloon.Style = (Genie.Balloon.Style <b>AND</b> &amp;H00FFFFFF) + (4*(2^24))
```



Чтобы задать число символов в строке, очистите биты 16 – 23 с логической операцией **и** , прежде чем устанавливать новое значение в качестве произведения нового значения со значением 2 ^ 16, добавленного к существующему значению свойства Style.


```
   ' Set the number of characters per line to 16
   Genie.Balloon.Style = (Genie.Balloon.Style AND &amp;HFF00FFFF) + (16*(2^16))
```



Свойство **Style** можно задать, даже если пользователь отключил отображение всплывающих окон с помощью вкладки свойств Microsoft Agent. Однако значения для числа строк должны быть в диапазоне от 1 до 128, а число символов в строке должно находиться в диапазоне от 8 до 255. Если указать недопустимое значение для свойства **Style** , то агент выдаст ошибку.

Это свойство применяется только к символу, используемому вашим клиентским приложением. Этот параметр не влияет на другие клиенты символов или других символов клиентского приложения.

Значения по умолчанию для этих битов стиля основываются на параметрах, когда символ компилируется с помощью редактора символов Microsoft Agent.
