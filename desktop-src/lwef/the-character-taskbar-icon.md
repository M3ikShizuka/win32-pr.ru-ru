# <a name="the-character-taskbar-icon"></a>Значок панели задач "символ"

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Если символ был создан для включения значка, этот значок отображается в области уведомлений на панели задач при запуске агента Майкрософт. Этот значок предоставляет доступ к всплывающему меню символа, которое предоставляет доступ к глобальным командам агента, таким как скрытые и отображаемые символы.

![область уведомлений с часами и значком](images/f1tbicon.gif)

При перемещении указателя над значком панели задач отображается окно подсказки, которое отражает имя символа (на текущем языке системы). Одним щелчком значка на панели задач символа отображается символ. Действие, связанное с двойным щелчком значка, зависит от текущего приложения, управляющего этим символом.

При щелчке значка правой кнопкой мыши отображается всплывающее меню. Когда символ отображается, во всплывающем меню отображаются те же команды, которые отображаются при щелчке символа правой кнопкой мыши. Если этот символ скрыт, отображаются только окна Открытие (или закрытие) с голосовыми командами и отображение команд.
