# <a name="add-method"></a>Добавить метод.

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Добавляет объект [Command](the-command-object.md) в коллекцию [Commands](the-commands-collection-object.md) .

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Агент ***. Символы ("**_чарактерид_*_"). Команды. Добавление_ *  *имени*, *заголовка*, *голоса*, *включенного*, *видимого*



| Часть      | Описание                                                                                                                                                                                                                                                                                                         |
|-----------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *Name*    | Обязательный. Строковое значение, соответствующее ИДЕНТИФИКАТОРу, присваиваемому команде.                                                                                                                                                                                                                                        |
| *Caption* | Необязательный элемент. Строковое значение, соответствующее имени, которое будет отображаться во всплывающем меню символа и в окне команд, если клиентское приложение является входным. Дополнительные сведения см. в разделе свойство [Caption](caption-property.md) объекта [Command](the-command-object.md) .                       |
| *Голосовая связь*   | Необязательный элемент. Строковое значение, соответствующее словам или фразе, используемым модулем распознавания речи для распознавания этой команды. Дополнительные сведения о способах форматирования для строки см. в разделе свойство [Voice](voice-property.md) объекта [команды](the-command-object.md) .                                |
| *Включен* | Необязательный элемент. Логическое значение, указывающее, включена ли команда. По умолчанию используется значение **True**. Дополнительные сведения см. в описании свойства [Enabled](enabled-property.md) объекта [команды](the-command-object.md) .                                                                                              |
| *Visible* | Необязательный элемент. Логическое значение, указывающее, отображается ли команда во всплывающем меню символа, если клиентское приложение является входным. По умолчанию используется значение **True**. Дополнительные сведения см. в разделе свойство [Visible](visible-property.md) объекта [Command](the-command-object.md) . |



 

</dd> </dl>

## <a name="remarks"></a>Remarks

Значение свойства [имени](name-property.md) [командного](the-command-object.md) объекта должно быть уникальным в пределах коллекции [Commands](the-commands-collection-object.md) . Перед созданием новой команды с тем же значением свойства Name необходимо удалить команду. Попытка создать команду с уже существующим свойством Name вызывает ошибку.

Этот метод также возвращает объект [Command](the-command-object.md) . Это позволяет объявить объект и назначить ему команду при вызове addMethod.


```
   Dim Cmd1 as IAgentCtlCommandEx
   Set Cmd1 = Genie.Commands.Add ("my first command", "Test", "Test", True, True)
   Cmd1.VoiceCaption = "this is a test"
```
