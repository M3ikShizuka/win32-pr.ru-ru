# <a name="name-property-agent-control"></a>Свойство Name (элемент управления агента)

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает имя, используемое в коде для обнаружения элемента управления. Это свойство доступно только для чтения во время выполнения.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*агент * * *. Безымян**

</dd> </dl>

## <a name="remarks"></a>Комментарии

в некоторых средах программирования, таких как Visual Basic, при добавлении элемента управления автоматически создается имя элемента управления по умолчанию, которое можно изменить во время разработки. В скриптах HTML имя можно определить в &lt; &gt; теге объекта. При определении имени следуйте правилам языка программирования для определения имен объектов.
