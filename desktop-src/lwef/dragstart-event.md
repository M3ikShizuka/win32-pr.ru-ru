# <a name="dragstart-event"></a>Событие Драгстарт

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Происходит, когда пользователь начинает перетаскивать символ.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

 *Подагент * * * \_ драгстарт* *  **(ByVal** *чарактерид*, **(** *кнопка* ByVal, **(ByVal** *SHIFT*, **(** ByVal *X*, **(ByVal** *Y * *)**



| Часть          | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
|---------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *чарактерид* | Возвращает идентификатор нажатого символа в виде строки.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| *Кнопка*      | Возвращает целое число, определяющее кнопку, которая была нажата и была вызвана для события. Аргумент Button — это битовое битовое значение, соответствующее левой кнопке (бит 0), правой кнопке (бит 1) и средней кнопке (бит 2). Эти биты соответствуют значениям 1, 2 и 4 соответственно. Задан только один из битов, указывающий кнопку, вызвавшую событие.                                                                                                                                                                                                                                                                                |
| *Сдвиг*       | Возвращает целое число, соответствующее состоянию клавиш SHIFT, CTRL и ALT при нажатии или отпускании кнопки, заданной в аргументе Button. Если ключ не работает, устанавливается бит. Аргумент сдвига является битовым битом с наименьшими значащими битами, соответствующими клавише SHIFT (бит 0), клавишей CTRL (бит 1) и клавишей ALT (бит 2). Эти биты соответствуют значениям 1, 2 и 4 соответственно. Аргумент shift указывает состояние этих ключей. Можно установить некоторые, все или ни одного бита, указывая, что не нажаты некоторые, все или ни один из этих ключей. Например, при нажатии клавиш CTRL и ALT значение Shift будет равно 6. |
| *X, Y*         | Возвращает целое число, указывающее текущее положение указателя мыши. Значения X и Y всегда выражаются в пикселях относительно левого верхнего угла экрана.                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |



 

</dd> </dl>

### <a name="remarks"></a>Remarks

Это событие отправляется только клиенту ввода-активного клиента символа. Когда пользователь перетаскивает символ без ввода активного клиента, сервер устанавливает последний входной и активный клиент как текущий клиент ввода-активности, отправляя событие [**активатеинпут**](activateinput-event.md) этому клиенту, а затем отправляя событие **драгстарт** .
