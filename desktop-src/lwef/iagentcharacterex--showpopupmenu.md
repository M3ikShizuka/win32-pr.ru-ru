# <a name="iagentcharacterexshowpopupmenu"></a>Иажентчарактерекс:: Шовпопупмену

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

``` syntax
HRESULT ShowPopupMenu(
   short x,  // x-coordinate of pop-up menu
   short y   // y-coordinate of pop-up menu
);
```

Отображает всплывающее меню для символа.

-   Возвращает значение S \_ , чтобы указать, что операция выполнена успешно.

<dl> <dt>

<span id="x"></span><span id="X"></span>*x*
</dt> <dd>

Координата x всплывающего меню символа в пикселях относительно начала координат экрана (в левом верхнем углу).

</dd> <dt>

<span id="y"></span><span id="Y"></span>*&*
</dt> <dd>

Координата y всплывающего меню символа (в пикселях) относительно начала координат экрана (вверху слева).

</dd> </dl>

При установке [**иажентчарактерекс:: сетаутопопупмену**](iagentcharacterex--setautopopupmenu.md) в **значение false** сервер больше не будет автоматически отображать меню при щелчке символа или его значка на панели задач. Этот метод можно использовать для вывода меню.

Меню отображается до тех пор, пока пользователь не выберет команду или не отобразит другое меню. В каждый момент времени может отображаться только одно всплывающее меню; Поэтому вызовы этого метода будут отменяться (удалить) из предыдущего меню.

Этот метод следует вызывать только в том случае, если клиентское приложение является активным клиентом символа. в противном случае произойдет сбой.
