# <a name="windows-defender"></a>Защитник Windows

Защитник Windows помогает защитить компьютеры от всплывающих окон, снижения производительности и угроз безопасности, вызванных шпионскими и другими нежелательными программами. Приложения могут запрашивать такие сведения, как работоспособность компьютера, конкретные сведения о вредоносных программах и журнал вредоносных программ, а также запускать проверки по требованию.
