# <a name="extradata-property"></a>Екстрадата, свойство

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает строку, указывающую дополнительные данные, хранящиеся как часть символа.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Агент*. **Символы ("***чарактерид***"). Екстрадата**

</dd> </dl>

## <a name="remarks"></a>Комментарии

Значение по умолчанию для свойства **екстрадата** для символа определяется при компиляции символа с помощью редактора символов Microsoft Agent. Он не может быть изменен или указан во время выполнения.

> [!Note]  
> Параметр свойства **екстрадата** является необязательным и не может быть указан для всех символов.
