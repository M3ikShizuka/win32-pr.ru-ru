# <a name="hotkey-property"></a>Свойство горячих клавиш

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает строку, которая указывает текущий параметр пользователя для ключа прослушивания.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Syntax**

## <a name="remarks"></a>Комментарии

Свойство **горячая** установка отражает текущее значение ключа прослушивания на странице речевого ввода страницы свойств агента (дополнительные параметры символов). Свойство доступно только для чтения; только пользователь может изменить этот параметр.
