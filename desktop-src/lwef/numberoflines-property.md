# <a name="numberoflines-property"></a>Нумберофлинес, свойство

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает число строк, поддерживаемое для всплывающей подсказки для указанного символа.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Агент ***. Символы ("**_чарактерид_*_"). Всплывающее сообщение. Нумберофлинес_*

</dd> </dl>

## <a name="remarks"></a>Remarks

Свойство **нумберофлинес** возвращает число строк текста в виде длинного целочисленного значения.
