# <a name="visibilitycause-property"></a>Висибилитикаусе, свойство

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Возвращает целочисленное значение, указывающее, что привело к видимому состоянию символа.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Агент*. **Символы ("**_чарактерид_*_"). Висибилитикаусе_*



| Значение | Описание                                                                                                  |
|-------|--------------------------------------------------------------------------------------------------------------|
| 0     | Символ не был показан.                                                                            |
| 1     | Пользователь скрыл символ с помощью команды во всплывающем меню значка символа на панели задач или с помощью речевого ввода. |
| 2     | Пользователь показал символ.                                                                               |
| 3     | Приложение скрыло символ.                                                                          |
| 4     | Приложение показало этот символ.                                                                       |
| 5     | Другое клиентское приложение скрыло символ.                                                                |
| 6     | Этот символ был показан в другом клиентском приложении.                                                             |
| 7     | Пользователь скрыл символ, используя команду во всплывающем меню символа.                                 |



 

</dd> </dl>

## <a name="remarks"></a>Remarks

Это свойство можно использовать для определения того, что привело к перемещению символа, когда несколько приложений совместно используют один и тот же символ. Эти значения совпадают со значениями, возвращаемыми событиями [**Показать**](show-event.md) и [**Скрыть**](hide-event.md) .
