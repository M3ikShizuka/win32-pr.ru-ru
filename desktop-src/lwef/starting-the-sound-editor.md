# <a name="starting-the-sound-editor"></a>Запуск редактора звука

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Чтобы запустить средство редактирования лингвистических сведений Майкрософт, выберите его в меню " **Пуск** " или дважды щелкните значок звукового редактора. Откроется окно звукового редактора, в котором отображаются меню, панель инструментов для часто используемых команд, текстовое поле для ввода слов, используемых редактором для обработки звуковых файлов, и область отображения для просмотра и редактирования звуковых и лингвистических данных.

![Снимок экрана, на котором показана начальная страница средства редактирования лингвистических сведений Майкрософт.](images/f1listliq.gif)

После запуска звукового редактора можно начать запись нового звукового файла или загрузить существующий звуковой файл.
