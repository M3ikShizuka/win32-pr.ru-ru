# <a name="introduction-to-microsoft-agent"></a>Введение в Microsoft Agent

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Microsoft Agent — это набор программируемых программных служб, поддерживающих представление интерактивных анимированных символов. Разработчики могут использовать символы в качестве интерактивных помощников, чтобы познакомиться с веб-страницами или приложениями, а также использовать их в дополнение к обычным средствам использования окон, меню и элементов управления.

Microsoft Agent позволяет разработчикам программного обеспечения и авторам веб-авторов внедрять новые формы взаимодействия с пользователем, известные как диалоговые интерфейсы, которые используют естественные аспекты общения в социальных сетях. Помимо ввода мыши и клавиатуры, Microsoft Agent включает дополнительную поддержку для распознавания речи, чтобы приложения могли отвечать на голосовые команды. Символы могут отвечать с помощью синтезированной речи, записанного звука или текста в виде всплывающей формулировки.

Подход к интерфейсу взаимодействия, который упрощается службами агента Майкрософт, не заменяет обычный дизайн графического пользовательского интерфейса. Вместо этого можно легко смешивать символы, используя традиционные компоненты интерфейса, такие как окна, меню и элементы управления, для расширения и расширения интерфейса приложения.

Программные интерфейсы Microsoft Agent упрощают анимацию символа для реагирования на вводимые пользователем данные. Анимированные символы отображаются в отдельном окне, что обеспечивает максимальную гибкость для того места, где они могут отображаться на экране. Microsoft Agent включает элемент управления ActiveX, который делает его службы доступными для языков программирования, поддерживающих ActiveX, включая языки веб-сценариев, такие как Visual Basic scripting Edition (VBScript). Это означает, что взаимодействие с символами можно программировать даже на HTML-страницах с помощью &lt; &gt; тега object.
