# <a name="advanced-character-options-window-microsoft-agent-user-interface"></a>Диалоговое окно "Дополнительные параметры символов" (пользовательский интерфейс агента Microsoft)

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

Microsoft Agent поддерживает определенные глобальные параметры, позволяющие пользователю управлять взаимодействием со всеми символами. В окне Дополнительные параметры символов отображаются эти параметры и их текущие параметры, которые могут быть показаны любым приложением размещения с помощью программного интерфейса агента Майкрософт. Microsoft Agent запоминает последнюю просмотренную страницу и отображает эту страницу при появлении страницы свойств.

-   [Страница «выходные данные»](the-output-page.md)
-   [Страница речевого ввода](the-speech-input-page.md)
-   [Страница "авторские права"](the-copyright-page.md)
