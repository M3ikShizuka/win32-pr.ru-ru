# <a name="iresultsviewer-interface"></a>Интерфейс Иресултсвиевер

> [!NOTE]
> Windows настольный поиск 2. x — это устаревшая технология, которая изначально была доступна в качестве надстройки для Windows XP и Windows Server 2003. в более поздних выпусках используйте вместо этого [API Windows поиска](../search/-search-reference-entry-page.md) . 

Предоставляет методы и свойства для представления результатов.

## <a name="members"></a>Элементы

Интерфейс **иресултсвиевер** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иресултсвиевер** также имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

| [**GoBack**](-search-2x-iresultsviewer-goback.md)       | Не реализован.<br/>                                                            |
| [**GoForward**](-search-2x-iresultsviewer-goforward.md) | Не реализован.<br/>                                                            |
| [**Обновляется**](-search-2x-iresultsviewer-refresh.md)     | Не реализован.<br/>                                                            |
| [**Позиции**](-search-2x-iresultsviewer-stop.md)           | Не реализован.<br/>                                                            |
| [**Update**](-search-2x-iresultsviewer-update.md)       | Применяет любые изменения запроса и перемещает представление к новому набору результатов.<br/> |
