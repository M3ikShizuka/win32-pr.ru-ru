# <a name="stopall-method"></a>Метод Стопалл

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Nописание**
</dt> <dd>

Останавливает все запросы анимации или заданные типы запросов для указанного символа.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Syntax**
</dt> <dd>

*Агент ***. Символы ("*** чарактерид * * *").* *  \[ *Тип* стопалл\]



| Отделение   | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|--------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *Type* | Необязательный элемент. Чтобы использовать этот параметр, можно использовать любое из следующих значений. Можно также указать несколько типов, разделяя их запятыми. <br/> "**Get**" <br/> Для отмены всех запросов [**Get**](get-method.md) , помещенных в очередь.<br/> "**Нонкуеуеджет**" <br/> Для отмены всех запросов [**Get**](get-method.md) , не относящихся к очереди (метод **Get** с параметром **Queue** имеет значение **false**).<br/> "**Переместить**" <br/> Для отмены всех запросов [**MoveTo**](moveto-method.md) , помещенных в очередь.<br/> "**Воспроизвести**" <br/> Для отмены всех запросов на [**Воспроизведение**](play-method.md) в очереди.<br/> «**Говорите**» <br/> Чтобы прерывать все запросы, находящиеся в очереди, [**обговаривает**](speak-method.md) .<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Комментарии

Если параметр **типа** не задан, сервер останавливает все анимации для символа, включая очереди запросов на [**Получение**](get-method.md) , не являющиеся в очереди, и очищает ее очередь анимации. Он также прекращает воспроизводить анимацию или скрытие символа.

Этот метод не создаст объект [**запроса**](/windows/desktop/lwef/the-request-object) .
