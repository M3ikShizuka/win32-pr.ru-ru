# <a name="mpmanageropen-function"></a>Функция Мпманажеропен

Устанавливает подключение к диспетчеру защиты от вредоносных программ на локальном компьютере.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT WINAPI MpManagerOpen(
  _In_  DWORD     dwReserved,
  _Out_ PMPHANDLE phMpHandle
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*двресервед* \[ окне\]
</dt> <dd>

Тип: **DWORD**

Зарезервировано для последующего использования. Должен иметь значение 0.

</dd> <dt>

*фмфандле* \[ заполняет\]
</dt> <dd>

Тип: **пмфандле**

Обработчик для интерфейса диспетчера защиты от вредоносных программ. Этот обработчик должен быть закрыт функцией [**мфандлеклосе**](mphandleclose.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

Если функция выполнена успешно, возвращается значение **S \_**. Вызов этой функции гарантированно будет выполнен, даже если служба защиты от вредоносных программ не запущена.

Если функция завершается ошибкой, возвращаемое значение является неудачным кодом **HRESULT** . Вызывающий объект может использовать функцию [**мперрормессажеформат**](mperrormessageformat.md) для получения общего описания сообщения об ошибке.
