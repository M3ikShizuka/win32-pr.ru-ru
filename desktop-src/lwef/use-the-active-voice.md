# <a name="use-the-active-voice"></a>Использование активного голоса

\[Microsoft Agent является устаревшим в Windows 7 и может быть недоступен в последующих версиях Windows.\]

При использовании речевого вывода для предоставления сведений о директиве или для получения ответа от пользователя используйте активный голос и четко укажите ожидаемое действие пользователя. В следующем примере показаны различия.



| Директива                            |  Оценка                   |
|--------------------------------------|-------------------------------|
| Позвольте мне повторить свой номер.           | Нет действий пользователя                |
| Число будет повторяться.         | Пассивный Voice, без действий пользователя |
| Прослушивать, пока число повторяется. | Пассивный Voice                 |
| Прослушивать повторение.            | Лучший вариант                   |



 

Кроме того, создайте выходные данные, чтобы unfold ключевую информацию в конце фразы, как показано в следующих примерах:



| Нерекомендуемый способ                                                 | Используйте это                                           |
|--------------------------------------------------|--------------------------------------------|
| "— Три следующие цифры?"           | "Является ли следующая цифра 3?"            |
| "Нажмите кнопку ОК, чтобы начать."                 | "Чтобы начать, нажмите кнопку" ОК ".                 |
| "Скажите, что" Готово "для завершения заказа". | "Чтобы завершить заказ, скажите" Готово ". |
