# <a name="use-appropriate-timing-and-emphasis"></a>Используйте соответствующее время и выделение

\[Microsoft Agent является устаревшим по отношению к Windows 7 и может быть недоступен в последующих версиях Windows.\]

Как и все отзывы, эффективность речевого вывода зависит от времени и ударения. Хорошая информация может быть связана с темпом, объемом и шагом, используемыми при произнесении какого-либо сообщения. Если в качестве голоса используется модуль преобразования текста в речь, большинство механизмов позволяют задать скорость, приостанавливать, отменять и акцентировать слова или фразы. Эти атрибуты можно использовать для указания интересующих вас интересов и понимания символа, а также для направления внимания пользователя или указания намерения персонажа. Дополнительные сведения о настройке атрибутов речи см. в разделе [теги речевого вывода агента Microsoft Agent](microsoft-agent-speech-output-tags.md). Если вы используете звуковые файлы в качестве выходных данных символа, учитывайте эти факторы и на записанном аудио.
