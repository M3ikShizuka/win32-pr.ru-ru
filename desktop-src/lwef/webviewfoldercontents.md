# <a name="webviewfoldercontents-object"></a>Объект Вебвиевфолдерконтентс

Реализуется оболочкой для использования внутри *WebView*. **Вебвиевфолдерконтентс** автоматически инициализирует себя в текущей папке WebView. Он создает представление папки оболочки, которое отображает содержимое папки в одном из пяти форматов: мелкий значок, крупный значок, список, сведения или эскиз. Он не является допустимым за пределами WebView.

Методы и свойства, предоставляемые **вебвиевфолдерконтентс** , идентичны параметрам объекта [**шеллфолдервиев**](/windows/desktop/shell/shellfolderview) .

## <a name="members"></a>Элементы

Объект **вебвиевфолдерконтентс** имеет следующие типы членов:

-   [События](#events)
-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="events"></a>События

Объект **вебвиевфолдерконтентс** содержит эти события.



| Событие                                                              | Описание                                                                              |
|:-------------------------------------------------------------------|:-----------------------------------------------------------------------------------------|
| [**SelectionChanged**](webviewfoldercontents-selectionchanged.md) | Происходит при изменении состояния выбора любого элемента или элементов в представлении.<br/> |



 

### <a name="methods"></a>Методы

Объект **вебвиевфолдерконтентс** содержит эти методы.



| Метод                                                       | Описание                                                                                                          |
|:-------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------|
| [**попупитеммену**](webviewfoldercontents-popupitemmenu.md) | Создает контекстное меню для указанного элемента и возвращает выбранную командную строку.<br/>                   |
| [**SelectedItems**](webviewfoldercontents-selecteditems.md) | Возвращает объект [**фолдеритемс**](../shell/folderitems.md) , представляющий все выбранные элементы в представлении.<br/> |
| [**SelectItem**](webviewfoldercontents-selectitem.md)       | Задает состояние выбора элемента в представлении.<br/>                                                          |



 

### <a name="properties"></a>Свойства

Объект **вебвиевфолдерконтентс** имеет следующие свойства.



| Свойство                                                            | Тип доступа          | Описание                                                                                                                              |
|:--------------------------------------------------------------------|:---------------------|:-----------------------------------------------------------------------------------------------------------------------------------------|
| [**Приложение**](webviewfoldercontents-application.md)<br/> | Только для чтения<br/> | Не реализован.<br/>                                                                                                              |
| [**фокуседитем**](webviewfoldercontents-focuseditem.md)<br/> | Только для чтения<br/> | Возвращает объект [**FolderItem**](../shell/folderitem.md) , представляющий элемент, имеющий фокус ввода.<br/>                           |
| [**Папка**](webviewfoldercontents-folder.md)<br/>           | Только для чтения<br/> | Возвращает объект [**Folder**](../shell/folder.md) , представляющий представление.<br/>                                                            |
| [**Parent**](webviewfoldercontents-parent.md)<br/>           | Только для чтения<br/> | Не реализован.<br/>                                                                                                              |
| [**Сценарий**](webviewfoldercontents-script.md)<br/>           | Только для чтения<br/> | Возвращает объект скрипта для представления.<br/>                                                                                       |
| [**виевоптионс**](webviewfoldercontents-viewoptions.md)<br/> | Только для чтения<br/> | Возвращает набор флагов [**шеллфолдервиевоптионс**](/windows/desktop/api/shldisp/ne-shldisp-shellfolderviewoptions) , которые указывают текущие параметры представления.<br/> |
