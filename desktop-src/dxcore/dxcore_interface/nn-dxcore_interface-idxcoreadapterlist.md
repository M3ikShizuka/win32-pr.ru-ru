# <a name="idxcoreadapterlist-interface"></a>Интерфейс Идкскореадаптерлист

Интерфейс **идкскореадаптерлист** реализует методы для получения элементов адаптера из созданного списка, а также сведения о списке. **Идкскореадаптерлист** наследует от интерфейса [IUnknown](/windows/win32/api/unknwn/nn-unknwn-iunknown) . Инструкции по программированию и примеры кода см. [в разделе Использование дкскоре для перечисления адаптеров](../dxcore-enum-adapters.md).
