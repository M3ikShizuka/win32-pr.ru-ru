# <a name="idxcoreadapterfactory-interface"></a>Интерфейс Идкскореадаптерфактори

Интерфейс **идкскореадаптерфактори**   реализует методы для создания объектов перечисления адаптера дкскоре, а также для получения сведений о них. **Идкскореадаптерфактори** наследует от интерфейса [IUnknown](/windows/win32/api/unknwn/nn-unknwn-iunknown) . Инструкции по программированию и примеры кода см. [в разделе Использование дкскоре для перечисления адаптеров](../dxcore-enum-adapters.md).
