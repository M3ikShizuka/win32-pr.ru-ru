# <a name="idxcoreadapterlistgetadaptercount-method"></a>Метод Идкскореадаптерлист:: Жетадаптеркаунт

Возвращает количество адаптеров в объекте списка адаптеров Дкскоре. Инструкции по программированию и примеры кода см. [в разделе Использование дкскоре для перечисления адаптеров](../dxcore-enum-adapters.md).

## <a name="syntax"></a>Синтаксис

```cpp
virtual uint32_t STDMETHODCALLTYPE GetAdapterCount() = 0;
```

## <a name="returns"></a>Возвращаемое значение

Тип: **uint32_t**

Возвращает число элементов адаптера в списке.
