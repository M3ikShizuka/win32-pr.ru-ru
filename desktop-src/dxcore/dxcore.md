# <a name="dxcore"></a>DXCore

Дкскоре — это API перечисления адаптеров для графических и расчетных устройств, поэтому некоторые его средства перекрываются с функциями [DXGI](../direct3ddxgi/dx-graphics-dxgi.md). Дкскоре используется Direct3D 12.

Этот набор документации содержит сведения о программировании с помощью Дкскоре, а также справочник по Дкскоре.

## <a name="requirements"></a>Requirements (Требования)

| &nbsp; | &nbsp; |
| ---- |:---- |
| **Поддерживаемые среды выполнения** | Windows/C + + |
| **Рекомендуемые языки программирования** | C++ |
| **Минимальная версия клиента** | Windows 10, версия 2004 (10,0; Сборка 19041) |
| **Header** | дкскоре. h и dxcore_interface. h |
| **Библиотека** | дкскоре. lib |
| **КОМПОНОВКИ** | dxcore.dll |
