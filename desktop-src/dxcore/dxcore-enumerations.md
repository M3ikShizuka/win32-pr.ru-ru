# <a name="dxcore-enumerations"></a>Перечисления DXCore

Следующие перечисления объявляются в dxcore_interface. h.

## <a name="in-this-section"></a>Содержание раздела

| Раздел и описание |
|-|
| [**Дкскореадаптерпреференце**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterpreference.md). Определяет константы, указывающие параметры адаптера Дкскоре для использования в качестве критерия сортировки списка. |
| [**Дкскореадаптерпроперти**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterproperty.md). Определяет константы, указывающие свойства адаптера Дкскоре. |
| [**Дкскореадаптерстате**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterstate.md). Определяет константы, указывающие виды состояний адаптера Дкскоре. |
| [**Дкскоренотификатионтипе**](./dxcore_interface/ne-dxcore_interface-dxcorenotificationtype.md). Определяет константы, указывающие типы уведомлений, создаваемых объектами [идкскореадаптер](./dxcore_interface/nn-dxcore_interface-idxcoreadapter.md) или [идкскореадаптерлист](./dxcore_interface/nn-dxcore_interface-idxcoreadapterlist.md) . |
| [**Дкскоресегментграуп**](./dxcore_interface/ne-dxcore_interface-dxcoresegmentgroup.md). Определяет константы, указывающие группирование сегмента памяти адаптера. |
