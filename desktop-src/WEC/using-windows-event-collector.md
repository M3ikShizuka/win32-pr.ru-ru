# <a name="using-windows-event-collector"></a>Использование сборщика событий Windows

В этом разделе перечислены разделы, объясняющие задачи, которые можно выполнить с помощью пакета SDK сборщика событий Windows. Примеры кода и объяснения для всех задач включены в следующие разделы.

## <a name="in-this-section"></a>в этом разделе

-   [Создание подписки, инициированной сборщиком](creating-an-event-collector-subscription.md)

    Содержит сведения и пример кода C++ для создания подписки, инициированной сборщиком. Подписка, инициированная сборщиком, позволяет принимать события на локальном компьютере (сборщике событий), пересылаемые с удаленного компьютера (источника событий).

-   [Настройка исходной подписки, инициированной источником](setting-up-a-source-initiated-subscription.md)

    Содержит сведения и пример кода C++ для создания подписки, инициированной исходным кодом. Подписка, инициированная источником, позволяет принимать события на локальном компьютере (сборщике событий), пересылаемые с удаленного компьютера (источника событий), не указывая источники событий в подписке.

-   [Добавление источника событий к подписке, инициированной сборщиком](adding-an-event-source-to-an-event-collector-subscription.md)

    Содержит сведения и пример кода C++ для добавления источника события (локального компьютера или удаленного компьютера) к подписке, инициированной сборщиком. Подписка, инициированная сборщиком, не может начать сбор событий, пока в подписку не будет добавлен источник событий.

-   [Создание подписок на события оборудования](creating-hardware-event-subscriptions.md)

    Содержит сведения о создании подписки на события для получения событий оборудования с компьютера, на котором установлен контроллер управления основной платой (BMC).

-   [Удаление подписки сборщика событий](deleting-an-event-collector-subscription.md)

    Содержит сведения и пример кода C++ для удаления подписки сборщика событий с локального компьютера.

-   [Отображение свойств подписки сборщика событий](displaying-the-properties-of-an-event-collector-subscription.md)

    Содержит сведения и пример кода C++ для отображения значений свойств подписки сборщика событий. Значения свойств могут предоставлять полезную информацию, такую как адреса источников событий, состояние подписки и источников событий, а также сведения о доставке событий.

-   [Отображение состояния подписки сборщика событий](displaying-the-status-of-an-event-collector-subscription.md)

    Содержит сведения и пример кода C++ для отображения состояния времени выполнения источников событий, связанных с подпиской сборщика событий. Это позволяет просматривать сообщения об ошибках, которые могут помочь в решении проблем, препятствующих пересылке событий источником событий.

-   [Список подписок сборщика событий](listing-event-collector-subscriptions.md)

    Содержит сведения и пример кода C++ для перечисления подписок, существующих на локальном компьютере. Вы можете использовать имена подписок, полученные с помощью этого примера, для удаления подписки, добавления источника событий в подписку, получения свойств подписки или просмотра состояния подписки.

-   [Удаление источника событий из подписки, инициированной сборщиком](removing-an-event-source-from-an-event-collector-subscription.md)

    Содержит сведения и пример кода C++ для удаления источника событий из подписки, инициированной сборщиком. Источник событий можно удалить из подписки, инициированной сборщиком, без отключения подписки.

-   [Повторная попытка подписки сборщика событий](retrying-an-event-collector-subscription.md)

    Содержит сведения и пример кода C++ для повторной попытки подписки сборщика событий в случае сбоя одного или нескольких источников событий. Вы можете повторить попытку одного источника событий или повторить всю подписку.
