# <a name="windows-event-collector-enumerations"></a>Перечисления сборщика событий Windows

В следующей таблице перечислены перечисления пакета SDK сборщика событий Windows.



| Перечисление                                                                                               | Описание                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ режим конфигурации подписки \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_configuration_mode)                       | Задает различные режимы конфигурации, которые изменяют параметры по умолчанию для подписки.                                            |
| [**\_ \_ Формат содержимого подписки \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_content_format)                               | Указывает способ подготовки событий к просмотру на компьютере, который отправляет события, прежде чем события будут отправлены на компьютер сборщика событий. |
| [**\_ \_ тип учетных данных подписки EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_credentials_type)                           | Указывает тип учетных данных для использования при взаимодействии с источниками событий.                                                         |
| [**\_ \_ режим доставки подписки \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_delivery_mode)                                 | Указывает способ доставки событий через подписку на события (с помощью принудительной или опрашивающей модели).                                          |
| [**\_ \_ идентификатор свойства подписки \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_property_id)                                     | Определяет значения для определения свойств подписки на события, используемых для конфигурации подписки.                                           |
| [**\_ \_ \_ \_ активное \_ состояние времени выполнения подписки EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_active_status) | Указывает состояние подписки или источника событий в отношении подписки.                                               |
| [**\_ \_ \_ идентификатор сведений о состоянии среды выполнения \_ подписки EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_info_id)             | Задает значение, определяющее свойство состояния среды выполнения для источника события или подписки.                                |
| [**\_Тип подписки \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_type)                                                    | Указывает тип используемой подписки (инициированная источником или подписка, инициированная сборщиком).                                     |
| [**\_тип варианта \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_variant_type)                                                              | Определяет значения, указывающие типы данных, которые используются в функциях сборщика событий Windows.                                  |
