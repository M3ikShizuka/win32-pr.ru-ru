# <a name="windows-event-collector-functions"></a>Функции сборщика событий Windows

В следующем списке кратко описаны функции, которые используются в сборщике событий Windows.

## <a name="in-this-section"></a>Содержание раздела

<dl> <dt>

[**екклосе**](/windows/desktop/api/Evcoll/nf-evcoll-ecclose)
</dt> <dd>

Закрывает маркер, полученный от других функций сборщика событий.

</dd> <dt>

[**екделетесубскриптион**](/windows/desktop/api/Evcoll/nf-evcoll-ecdeletesubscription)
</dt> <dd>

Удаляет существующую подписку.

</dd> <dt>

[**еценумнекстсубскриптион**](/windows/desktop/api/Evcoll/nf-evcoll-ecenumnextsubscription)
</dt> <dd>

Возобновляет перечисление подписок, зарегистрированных на локальном компьютере.

</dd> <dt>

[**екжетобжектаррайпроперти**](/windows/desktop/api/Evcoll/nf-evcoll-ecgetobjectarrayproperty)
</dt> <dd>

Извлекает значения свойств для источников событий подписки.

</dd> <dt>

[**екжетобжектаррайсизе**](/windows/desktop/api/Evcoll/nf-evcoll-ecgetobjectarraysize)
</dt> <dd>

Возвращает количество индексов массива значений свойств для источников событий подписки.

</dd> <dt>

[**екжетсубскриптионпроперти**](/windows/desktop/api/Evcoll/nf-evcoll-ecgetsubscriptionproperty)
</dt> <dd>

Извлекает значение свойства из объекта подписки.

</dd> <dt>

[**екжетсубскриптионрунтиместатус**](/windows/desktop/api/Evcoll/nf-evcoll-ecgetsubscriptionruntimestatus)
</dt> <dd>

Извлекает сведения о состоянии времени выполнения для источника события подписки или самой подписки.

</dd> <dt>

[**еЦинсертобжектаррайелемент**](/windows/desktop/api/Evcoll/nf-evcoll-ecinsertobjectarrayelement)
</dt> <dd>

Вставляет пустой объект в массив значений свойств для источников событий подписки.

</dd> <dt>

[**екопенсубскриптионенум**](/windows/desktop/api/Evcoll/nf-evcoll-ecopensubscriptionenum)
</dt> <dd>

Создает перечислитель подписки для перечисления всех зарегистрированных подписок на локальном компьютере.

</dd> <dt>

[**екопенсубскриптион**](/windows/desktop/api/Evcoll/nf-evcoll-ecopensubscription)
</dt> <dd>

Открывает существующую подписку или создает новую.

</dd> <dt>

[**ексавесубскриптион**](/windows/desktop/api/Evcoll/nf-evcoll-ecsavesubscription)
</dt> <dd>

Сохраняет сведения о конфигурации подписки.

</dd> <dt>

[**ексетобжектаррайпроперти**](/windows/desktop/api/Evcoll/nf-evcoll-ecsetobjectarrayproperty)
</dt> <dd>

Задает значение свойства в массиве значений свойств для источников событий подписки.

</dd> <dt>

[**ексетсубскриптионпроперти**](/windows/desktop/api/Evcoll/nf-evcoll-ecsetsubscriptionproperty)
</dt> <dd>

Устанавливает новые значения или обновляет существующие значения подписки.

</dd> <dt>

[**екремовеобжектаррайелемент**](/windows/desktop/api/Evcoll/nf-evcoll-ecremoveobjectarrayelement)
</dt> <dd>

Удаляет элемент из массива объектов, который содержит значения свойств для источников событий подписки.

</dd> <dt>

[**екретрисубскриптион**](/windows/desktop/api/Evcoll/nf-evcoll-ecretrysubscription)
</dt> <dd>

Повторяет попытку подключения к источнику событий подписки, которая не подключена.

</dd> </dl>
