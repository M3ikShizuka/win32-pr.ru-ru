Текущая Долгота (в градусах). Долгота находится в диапазоне от-180 до 180, где Восток — положительное значение.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
Longitude = LocationDisp.DispLatLongReport.Longitude
```



## <a name="property-value"></a>Значение свойства

Это свойство доступно только для чтения ( **число** с плавающей запятой двойной точности).
