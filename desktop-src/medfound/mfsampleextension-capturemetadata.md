# <a name="mfsampleextension_capturemetadata-attribute"></a>Мфсампликстенсион \_ каптуреметадата, атрибут

Хранилище [**имфаттрибутес**](/windows/desktop/api/mfobjects/nn-mfobjects-imfattributes) для всех метаданных, связанных с конвейером захвата.

## <a name="data-type"></a>Тип данных

**[**Имфаттрибутес**](/windows/desktop/api/mfobjects/nn-mfobjects-imfattributes)** хранится в виде **IUnknown** 

## <a name="remarks"></a>Комментарии

Этот атрибут может присутствовать в любой [**имфсампле**](/windows/desktop/api/mfobjects/nn-mfobjects-imfsample).

Этот атрибут доступен только для чтения.
