# <a name="sensor-profile-guids"></a>Идентификаторы GUID профиля датчика

Следующие идентификаторы GUID определяют различные типы профилей датчиков. Эти идентификаторы GUID чаще всего используются в структуре [**сенсорпрофилеид**](/windows/win32/api/mfidl/ns-mfidl-sensorprofileid) .

<dl> <dt>

<span id="_KSCAMERAPROFILE_HighFrameRate"></span><span id="_kscameraprofile_highframerate"></span><span id="_KSCAMERAPROFILE_HIGHFRAMERATE"></span>**Кскамерапрофиле \_ Хигхфрамерате**
</dt> <dd> <dl> <dt>



Высокая частота кадров.


</dt> </dl> </dd> <dt>

<span id="KSCAMERAPROFILE_HDRWithWCGVideo"></span><span id="kscameraprofile_hdrwithwcgvideo"></span><span id="KSCAMERAPROFILE_HDRWITHWCGVIDEO"></span>**КСКАМЕРАПРОФИЛЕ \_ хдрвисвкгвидео**
</dt> <dd> <dl> <dt>



HDR с ВКГ видео.


</dt> </dl> </dd> <dt>

<span id="KSCAMERAPROFILE_HDRWithWCGPhoto"></span><span id="kscameraprofile_hdrwithwcgphoto"></span><span id="KSCAMERAPROFILE_HDRWITHWCGPHOTO"></span>**КСКАМЕРАПРОФИЛЕ \_ хдрвисвкгфото**
</dt> <dd> <dl> <dt>



HDR с ВКГ фото.


</dt> </dl> </dd> <dt>

<span id="KSCAMERAPROFILE_VariablePhotoSequence"></span><span id="kscameraprofile_variablephotosequence"></span><span id="KSCAMERAPROFILE_VARIABLEPHOTOSEQUENCE"></span>**КСКАМЕРАПРОФИЛЕ \_ вариаблефотосекуенце**
</dt> <dd> <dl> <dt>



Переменная последовательность фотографий.


</dt> </dl> </dd> <dt>

<span id="KSCAMERAPROFILE_VideoHDR8"></span><span id="kscameraprofile_videohdr8"></span><span id="KSCAMERAPROFILE_VIDEOHDR8"></span>**КСКАМЕРАПРОФИЛЕ \_ VideoHDR8**
</dt> <dd> <dl> <dt>



Видео HDR8.


</dt> </dl> </dd> </dl>

<span id="KSCAMERAPROFILE_CompressedCamera"></span><span id="kscameraprofile_compressedcamera"></span><span id="KSCAMERAPROFILE_COMPRESSEDCAMERA"></span>**КСКАМЕРАПРОФИЛЕ \_ компресседкамера**

Профиль поддерживает сжатые потоки камеры, чтобы обеспечить сохранение сжатых потоков непосредственно в сжатом формате без затрат на распаковку и повторное сжатие потока.


</dt> </dl> </dd> </dl>
