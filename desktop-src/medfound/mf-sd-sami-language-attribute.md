# <a name="mf_sd_sami_language-attribute"></a>\_ \_ Атрибут языка MF SD Sami \_

Содержит имя языка для синхронизированного доступного обмена мультимедиа (SAMI), определенное для потока.

Этот атрибут имеется в дескрипторах потока, возвращаемых источником носителя SAMI.

## <a name="data-type"></a>Тип данных

Строка расширенных символов

## <a name="remarks"></a>Комментарии

Имя языка SAMI указывается в файле SAMI.

Константа GUID для этого атрибута экспортируется из мфууид. lib.
