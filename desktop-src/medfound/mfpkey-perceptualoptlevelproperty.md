# <a name="mfpkey_perceptualoptlevel-property"></a>МФПКЭЙ \_ перцептуалоптлевел, свойство

Указывает, должен ли кодек использовать консервативную оптимизацию искусственного при кодировании.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкперцептуалоптлевел

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

0

## <a name="remarks"></a>Remarks

Консервативная оптимизация искусственного — это процесс, с помощью которого кодек пытается выделить "важные" и "неважные" регионы в видеокадре. Определив эти регионы кадра, кодек предоставит более высокий приоритет для важных регионов, за счет чего качество неважных регионов.

Искусственного оптимизация позволяет подчеркнуть, что изображение отображается правильно для человеческого глаза, а не накладывается на более математическую точность.

Результаты оптимизации будут зависеть от типа закодированного видео. Эта функция может быть хорошо подходит для кодирования с низкими и низким уровнем разрешений (например, веб-потоковой передачи), но ее, возможно, следует избегать, когда надежде для архивирования качества видео.
