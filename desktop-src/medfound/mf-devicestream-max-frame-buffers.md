# <a name="mf_devicestream_max_frame_buffers-attribute"></a>\_ \_ \_ Атрибут числа \_ буферов кадров MF девицестреам

Указывает максимальное число кадров, которое будет буферизовано источником видеозаписи для этого потока.

## <a name="data-type"></a>Тип данных

**UINT32**
