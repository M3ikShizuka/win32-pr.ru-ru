# <a name="mfpkey_stat_bmax-property"></a>МФПКЭЙ \_ stat \_ Бмакс, свойство

Указывает максимальное окно буфера (в миллисекундах) закодированного потока. Только для чтения.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

**g \_ всзвмвкмаксбитрате**

## <a name="data-type"></a>Тип данных

**VT \_ I4**

## <a name="remarks"></a>Remarks

Чтобы определить максимальное число окон буфера закодированного потока, прочитайте это свойство в конце кодирования.
