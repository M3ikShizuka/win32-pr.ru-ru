# <a name="mft_enum_video_renderer_extension_profile-attribute"></a>\_ \_ \_ \_ Атрибут профиля расширения модуля подготовки \_ отчетов MFT Enum

Содержит список всех записей **видеорендеререкстенсионпрофиле** в теге **видеорендеререкстенсионпрофилес** файла манифеста приложения UWP. [**Мфтенумекс**](/windows/desktop/api/mfapi/nf-mfapi-mftenumex) сохраняет это в хранилище атрибутов объекта [**Имфактивате**](/windows/desktop/api/mfobjects/nn-mfobjects-imfactivate) , который **мфтенумекс** создает для МФТС, имеющего связанный манифест UWP, содержащий тег **видеорендеререкстенсионпрофилес** .
