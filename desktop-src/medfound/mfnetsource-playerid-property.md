# <a name="mfnetsource_playerid-property"></a>МФНЕТСАУРЦЕ \_ плайерид, свойство

Значение поля "c-плайерид", используемое сетевым источником для ведения журнала. Приложения могут задать для этого свойства любое строковое значение.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

Строка расширенных символов (**WCHAR** \* )

VT \_ LPWSTR

**пвсзвал**



## <a name="remarks"></a>Комментарии

Константа **мфнетсаурце \_ плайерид** определяет идентификатор GUID для этого ключа свойства. Идентификатор свойства (PID) равен нулю.

Приложения могут использовать это свойство для настройки сетевого источника. Чтобы задать свойство, передайте указатель **ипропертисторе** в сопоставитель источника. Дополнительные сведения см. [в разделе Настройка источника мультимедиа](configuring-a-media-source.md).
