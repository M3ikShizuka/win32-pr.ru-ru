# <a name="mfsampleextension_repeatfirstfield-attribute"></a>Мфсампликстенсион \_ репеатфирстфиелд, атрибут

Указывает, следует ли повторять первое поле в чередовании кадров. Этот атрибут применяется к примерам мультимедиа.

## <a name="data-type"></a>Тип данных

**Bool** , сохраненный как **UINT32**

## <a name="getset"></a>Get/Set

Чтобы получить этот атрибут, вызовите метод [**имфаттрибутес:: UInt32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-getuint32).

Чтобы задать этот атрибут, вызовите [**имфаттрибутес:: SetUINT32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-setuint32).

## <a name="applies-to"></a>Применяется к

[**имфсампле**](/windows/desktop/api/mfobjects/nn-mfobjects-imfsample)

## <a name="remarks"></a>Комментарии

Если значение равно **false** или атрибут не задан, первое поле не повторяется. Если значение равно **true**, первое поле повторяется. Значение **true** допустимо только в том случае, если выполняются следующие условия.

-   Тип мультимедиа смешанный и прогрессивный. (Атрибут атрибута « [**\_ \_ \_ режим с чередованием» MF MT**](mf-mt-interlace-mode-attribute.md) для типа носителя — **мфвидеоинтерлаце \_ миксединтерлацеорпрогрессиве**.)
-   Кадр является прогрессивным, а атрибут [**мфсампликстенсион \_ чередования**](mfsampleextension-interlaced-attribute.md) в образце имеет **значение true**.
-   В образце задан атрибут [**мфсампликстенсион \_ боттомфиелдфирст**](mfsampleextension-bottomfieldfirst-attribute.md) . Значение может быть **true** или **false**. Порядок полей определяется этим атрибутом.

Этот атрибут используется для выпадающих 3:2. В следующей таблице показан порядок отображения полей.



| Мфсампликстенсион \_ репеатфирстфиелд | Мфсампликстенсион \_ боттомфиелдфирст | Порядок полей         |
|-------------------------------------|-------------------------------------|---------------------|
| **TRUE**                            | **TRUE**                            | Нижние, верхние и нижние |
| **TRUE**                            | **FALSE**                           | Верхний, нижний, верхний |
| **FALSE**                           | **TRUE**                            | Нижний, верхний        |
| **FALSE**                           | **FALSE**                           | Верхний и нижний        |



 

Константа GUID для этого атрибута экспортируется из мфууид. lib.
