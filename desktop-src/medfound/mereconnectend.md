# <a name="mereconnectend-event"></a>Событие Мереконнектенд

Сигнализирует о том, что источник мультимедиа завершил попытку повторного подключения к серверу.

Вызвано источником мультимедиа в конце попытки повторного подключения. Источник сети создает это событие, если оно повторно подключается к серверу. Сеанс мультимедиа перенаправляет это событие в приложение.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |
