# <a name="mfnetsource_proxybypassforlocal-property"></a>МФНЕТСАУРЦЕ \_ проксибипассфорлокал, свойство

Указывает, должен ли локатор прокси использовать прокси-сервер для локальных адресов.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

Логическое значение (**Long**)

VT \_ I4

**лвал**



## <a name="remarks"></a>Комментарии

Константа **мфнетсаурце \_ проксибипассфорлокал** определяет идентификатор GUID для этого ключа свойства. Идентификатор свойства (PID) равен нулю.

Приложения могут использовать это свойство для настройки локатора прокси при создании прокси-объекта-локатора. Чтобы задать свойство, передайте указатель **ипропертисторе** в параметре *ппроксиконфиг* функции [**мфкреатепроксилокатор**](/windows/desktop/api/mfidl/nf-mfidl-mfcreateproxylocator) . Если прокси-сервер должен использоваться для локальных адресов, необходимо задать значение 0. в противном случае ненулевое значение настраивает локатор прокси-сервера по умолчанию, чтобы пропустить прокси-сервер для локальных адресов.
