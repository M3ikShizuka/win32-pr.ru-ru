# <a name="opm_set_acp_and_cgmsa_signaling"></a>ОПМ \_ Установка \_ \_ сигнала ACP и \_ кгмса \_

Указывает сведения о видеосигнале, отличном от уровня защиты.



| Требование | Значение |
|--------------|---------------------------------------------------------------------------------------------------------------------|
| Идентификатор GUID команды | ОПМ \_ Установка \_ \_ сигнала ACP и \_ кгмса \_                                                                                |
| Входные данные   | Структура [**\_ \_ \_ \_ \_ \_ параметров сигнализации ОПМ Set ACP и кгмса**](/windows/desktop/api/opmapi/ns-opmapi-opm_set_acp_and_cgmsa_signaling_parameters) |



 

## <a name="remarks"></a>Remarks

Эта команда эквивалентна \_ команде дксва коппсетсигналинг, используемой в протоколе сертифицированной защиты выходных данных (Копп).

В случае с CGMS-A для некоторых стандартов защиты требуется, чтобы ТВ-сигнал содержал информацию в тех же пакетах ВБИ звукозаписи, что и для битов с ЗАЩИТой от имени A. Помимо прочего, эта информация включает в себя пропорции изображения. Телевизоры могут плохо отображаться, если сведения о пропорции не соответствуют потоку видео. Приложение может использовать эту команду, чтобы задать пропорции, чтобы графический драйвер мог формировать правильные пакеты ВБИ.
