# <a name="mesessioncapabilitieschanged-event"></a>Событие Месессионкапабилитиесчанжед

Вызывается сеансом мультимедиа при изменении возможностей сеанса.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |



## <a name="remarks"></a>Комментарии

Событие содержит следующие атрибуты.



| attribute                                                                     | Описание                                      |
|-------------------------------------------------------------------------------|--------------------------------------------------|
| [**\_сессионкапс событий \_ MF**](mf-event-sessioncaps-attribute.md)              | Новые флаги возможностей сеанса.                  |
| [**\_Дельта событий MF \_ сессионкапс \_**](mf-event-sessioncaps-delta-attribute.md) | Указывает, какие флаги возможностей изменились. |
