# <a name="mf_source_reader_enable_video_processing-attribute"></a>MF \_ source \_ Reader \_ Включение \_ \_ атрибута обработки видео

Включает обработку видео [модулем чтения исходного кода](source-reader.md).

## <a name="data-type"></a>Тип данных

**UINT32**



| Значение                                                                                                                                                                | Значение                                        |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------|
| <span id="Nonzero"></span><span id="nonzero"></span><span id="NONZERO"></span><dl> <dt>**Отличные**</dt> </dl> | Включить обработку видео.<br/>            |
| <span id="Zero"></span><span id="zero"></span><span id="ZERO"></span><dl> <dt>**Ноль**</dt> </dl>             | Отключить обработку видео. (по умолчанию)<br/> |



 

## <a name="getset"></a>Get/Set

Чтобы получить этот атрибут, вызовите метод [**имфаттрибутес:: UInt32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-getuint32).

Чтобы задать этот атрибут, вызовите [**имфаттрибутес:: SetUINT32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-setuint32).

## <a name="remarks"></a>Комментарии

Если этот атрибут имеет **значение true** (отличное от нуля), модуль чтения исходного кода может выполнять следующую ограниченную обработку видео для несжатых видеокадров:

-   Преобразование из YUV в RGB-32.
-   Деинтерлейсинга.

Эти операции выполняются в программном обеспечении и не оптимизируются для воспроизведения. Эта функция предназначена для приложений, которые обрабатывают небольшое количество кадров, например для создания эскиза видео, или для приложений, которые не декодируются кадры в режиме реального времени. Операция чередования выполняет интерполяцию данных из одного поля, поэтому они теряются.

Избегайте этого параметра, если вы используете Direct3D для показа видеокадров, так как GPU обычно предоставляет лучшие возможности обработки видео.

Если этот атрибут имеет **значение true**, следующие атрибуты должны иметь **значение false**:

-   [\_ \_ \_ Диспетчер D3D чтения источника \_ MF](mf-source-reader-d3d-manager.md)
-   [\_ \_ запретить \_ конвертеры MF ReadWrite](mf-readwrite-disable-converters.md)
