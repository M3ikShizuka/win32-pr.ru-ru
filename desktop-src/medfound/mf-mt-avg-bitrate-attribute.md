# <a name="mf_mt_avg_bitrate-attribute"></a>\_Атрибут с \_ средним скоростью MF MT \_

Приблизительная скорость передачи данных в потоке видео (в битах в секунду) для типа видео.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

Этот атрибут соответствует элементу **двбитрате** в структурах [**видеоинфохеадер**](/previous-versions/windows/desktop/api/amvideo/ns-amvideo-videoinfoheader) и [**VIDEOINFOHEADER2**](/previous-versions/windows/desktop/api/dvdmedia/ns-dvdmedia-videoinfoheader2) .

Константа GUID для этого атрибута экспортируется из мфууид. lib.
