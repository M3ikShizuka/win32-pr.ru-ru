# <a name="mfnetsource_browserwebpage-property"></a>МФНЕТСАУРЦЕ \_ бровсервебпаже, свойство

Значение поля "CS (ссылка)", используемое сетевым источником для ведения журнала. Приложения могут задать для этого свойства URL-адрес веб-страницы, в которой было внедрено приложение.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

Строка расширенных символов (**WCHAR** \* )

VT \_ LPWSTR

**пвсзвал**



## <a name="remarks"></a>Remarks

Константа **мфнетсаурце \_ бровсервебпаже** определяет идентификатор GUID для этого ключа свойства. Идентификатор свойства (PID) равен нулю.

Приложения могут использовать это свойство для настройки сетевого источника. Чтобы задать свойство, передайте указатель **ипропертисторе** в сопоставитель источника. Дополнительные сведения см. [в разделе Настройка источника мультимедиа](configuring-a-media-source.md).
