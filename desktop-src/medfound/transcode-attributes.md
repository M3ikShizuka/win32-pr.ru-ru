# <a name="transcode-attributes"></a>Атрибуты перекодирования

С API перекодирования используются следующие атрибуты.



| attribute                                                                          | Описание                                                                                  |
|------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [\_ \_ Настройка профиля для ПЕРЕкодирования MF \_](mf-transcode-adjust-profile.md)                  | Флаги профиля, определяющие параметры потока для топологии перекодирования.                    |
| [MF \_ \_ контаинертипе](mf-transcode-containertype.md)                     | Указывает тип контейнера закодированного файла.                                             |
| [\_ \_ не разграниченияая \_ Вставка \_ кодировщика MF](mf-transcode-donot-insert-encoder.md)     | Указывает, должен ли кодировщик быть включен в топологию перекодировки.                     |
| [MF \_ \_ енкодингпрофиле](mf-transcode-encodingprofile.md)                 | Указывает профиль соответствия устройства для кодирования файлов формата ASF. |
| [MF \_ \_ куалитивсспид](mf-transcode-qualityvsspeed.md)                   | Задает компромисс между качеством кодирования и скоростью кодирования.                          |
| [\_код MF \_ пропустить \_ перенос метаданных \_](mf-transcode-skip-metadata-transfer.md) | Указывает, записываются ли метаданные в перекодированный файл.                                |
| [**MF \_ \_ топологимоде**](/windows/desktop/api/mfidl/ne-mfidl-mf_transcode_topologymode_flags)             | Указывает, будет ли загрузчик топологии загружать преобразования на основе оборудования.                   |
