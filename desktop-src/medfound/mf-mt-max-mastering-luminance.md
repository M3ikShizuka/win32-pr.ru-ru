# <a name="mf_mt_max_mastering_luminance-attribute"></a>\_ \_ \_ \_ Атрибут освещенности для максимального размера MF MT

Задает максимальную светимость дисплея, на котором было создано содержимое, в НИТС. Этот атрибут имеет ту же семантику, что и **максимальный уровень \_ \_ \_ яркости при отображении** , как определено в стандарте CEA-861,3.

## <a name="data-type"></a>Тип данных

**UINT32**
