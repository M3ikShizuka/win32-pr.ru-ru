# <a name="stream-attributes"></a>Атрибуты потока

Следующие атрибуты применяются к потокам мультимедиа. Чтобы получить атрибуты из примера носителя, используйте метод [**имфмедиасаурцеекс:: жетстреаматтрибутес**](/windows/desktop/api/mfidl/nf-mfidl-imfmediasourceex-getstreamattributes) .



| attribute                                                                                   | Описание                                   |
|---------------------------------------------------------------------------------------------|-----------------------------------------------|
| [Мфстреамекстенсион \_ камераекстринсикс](mfstreamextension-cameraextrinsics.md)               | Внешние камеры для потока.         |
| [Мфстреамекстенсион \_ пинхолекамераинтринсикс](mfstreamextension-pinholecameraintrinsics.md) | Встроенные функции камеры пинхоле для потока. |



 

Не каждый поток мультимедиа содержит все перечисленные здесь атрибуты. В некоторых случаях атрибут применяется только к определенным типам данных.
