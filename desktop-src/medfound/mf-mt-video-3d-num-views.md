# <a name="mf_mt_video_3d_num_views-attribute"></a>\_ \_ \_ Атрибут 3D-представления трехмерного видео MF MT \_ \_

Число представлений в трехмерной видеопоследовательности.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

Типичная трехмерная последовательность видео содержит два представления: слева и справа. В настоящее время значение этого атрибута должно быть либо 1, либо 2.
