# <a name="codecapi_avdecsoftwaredynamicformatchange-property"></a>КОДЕКАПИ \_ авдексофтварединамикформатчанже, свойство

Указывает, следует ли использовать динамическое изменение формата программного обеспечения для внутреннего изменения размера.

## <a name="data-type"></a>Тип данных

**ЛОГИЧЕСКОМ**

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авдексофтварединамикформатчанже**

## <a name="property-value"></a>Значение свойства

Если задано **значение true**, то для внутреннего изменения размера используется динамическое изменение формата программного обеспечения.
