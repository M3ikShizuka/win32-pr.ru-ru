# <a name="mf_mt_pad_control_flags-attribute"></a>\_ \_ \_ Атрибут флагов элемента управления MF MT Pad \_

Задает пропорции выходного прямоугольника для типа видеоклипа.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Комментарии

Значение этого атрибута является членом перечисления [**мфвидеопадфлагс**](/windows/desktop/api/mfapi/ne-mfapi-mfvideopadflags) .

Константа GUID для этого атрибута экспортируется из мфууид. lib.
