# <a name="mestreamsinkprerolled-event"></a>Событие Местреамсинкпрероллед

Вызывается приемником потока, когда поток получил достаточное количество предварительных данных для начала подготовки к просмотру. Это событие вызывается приемниками носителей, поддерживающими интерфейс [**имфмедиасинкпреролл**](/windows/desktop/api/mfidl/nn-mfidl-imfmediasinkpreroll) .

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |
