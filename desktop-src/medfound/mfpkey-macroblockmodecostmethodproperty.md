# <a name="mfpkey_macroblockmodecostmethod-property"></a>МФПКЭЙ \_ макроблоккмодекостмесод, свойство

Указывает метод стоимости, используемый кодеком для определения используемого режима макроблокк.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкмакроблоккмодекостмесод

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

0

## <a name="remarks"></a>Remarks

Для этого свойства можно задать одно из следующих значений.



| Значение | Используемый метод                                                                                            |
|-------|--------------------------------------------------------------------------------------------------------|
| 0     | SAD/Хадамард. Этот параметр настраивает кодек для использования только искажений при вычислении затрат.             |
| 1     | Стоимость удаленных рабочих столов. Этот параметр настраивает кодек для учета скорости и искажения при вычислении затрат. |
