# <a name="mf_mt_dv_aaux_ctrl_pack_1-attribute"></a>% \_ MF \_ DV \_ ааукс \_ CTRL \_ , \_ атрибут Pack 1

Пакет управления "Audio-вспомогательный" (ААУКС) для второго звукового блока в цифровом видеоролике (DV).

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

этот атрибут соответствует элементу **dwDVAAuxCtl1** структуры DirectShow [**двинфо**](/windows/win32/api/strmif/ns-strmif-dvinfo) .

Константа GUID для этого атрибута экспортируется из мфууид. lib.
