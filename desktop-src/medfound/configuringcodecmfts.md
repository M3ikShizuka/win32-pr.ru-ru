# <a name="configuring-codec-mfts"></a>Настройка кодека МФТС

В этом разделе описывается процесс настройки кодека МФТС. Каждый кодек имеет определенные процедуры, но сведения, общие для всех, описаны здесь.

## <a name="configuring-mft-inputs-and-outputs"></a>Настройка входных и выходных файлов MFT

Каждый файл MFT поддерживает определенные типы входных и выходных данных. Поддерживаемые входные типы можно получить, повторно вызвав [**имфтрансформ:: жетинпутаваилаблетипе**](/windows/desktop/api/mftransform/nf-mftransform-imftransform-getinputavailabletype), увеличивая индекс типа с каждым вызовом. При обнаружении соответствующего типа задайте тип входных данных, вызвав [**имфтрансформ:: сетинпуттипе**](/windows/desktop/api/mftransform/nf-mftransform-imftransform-setinputtype). Затем можно повторить процесс для типа выходных данных с помощью вызовов [**имфтрансформ:: жетаутпутаваилаблетипе**](/windows/desktop/api/mftransform/nf-mftransform-imftransform-getoutputavailabletype) и [**Имфтрансформ:: сетаутпуттипе**](/windows/desktop/api/mftransform/nf-mftransform-imftransform-setoutputtype). Необходимо запрашивать или задавать доступные выходные типы только после установки входного типа.

## <a name="configuring-the-codec-mfts-for-encoding"></a>Настройка кодека МФТС для кодирования

все видеокодеки Windows Media Audio и Video поддерживают различные функции кодирования. Эти функции обычно настраиваются путем установки свойств в MFT с помощью методов интерфейса **ипропертисторе** . Некоторые свойства настраиваются с помощью специализированных интерфейсов кодека. Эти интерфейсы перечислены для каждого кодека в разделе [объекты кодека](codecobjects.md).

Ниже приведен общий порядок операций для настройки кодировки MFT.

1.  Настройте необходимые функции кодека с помощью методов **ипропертисторе**.
2.  При необходимости используйте интерфейсы кодека MFT для настройки дополнительных функций.
3.  Настройте типы входных и выходных данных. Порядок, в котором должны настраиваться типы, зависит от отдельных кодеков. Дополнительные сведения см. в разделе [Работа с аудио](workingwithaudio.md) и [Работа с видео](workingwithvideo.md).

## <a name="configuring-the-codec-mfts-for-decoding"></a>Настройка кодека МФТС для декодирования

Декодирование проще, чем кодирование, так как поддерживается меньше возможностей декодера.

Ниже приведен общий порядок операций для настройки декодирования MFT.

1.  Настройте необходимые функции декодера с помощью методов **ипропертисторе**.
2.  Установите тип входных данных в тип, используемый для выходных данных кодировщика.
3.  Настройте тип выходных данных. Поддерживаемые типы вывода различаются для различных входных данных.

> [!Note]  
> Важно использовать тот же тип носителя для ввода декодера, что использовалось для выходных данных кодировщика. это связано с тем, что видеокодеки Windows media аудио и видео используют форматы мультимедиа с дополнительными данными. Без данных расширенного формата декодировать сжатое содержимое нельзя.
