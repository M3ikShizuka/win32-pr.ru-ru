# <a name="mf_pd_total_file_size-attribute"></a>\_ \_ Общий \_ \_ атрибут размера файла MF PD

Задает общий размер исходного файла в байтах. Этот атрибут применяется к дескрипторам представления. При необходимости источник мультимедиа может задавать этот атрибут.

## <a name="data-type"></a>Тип данных

**UINT64**

## <a name="remarks"></a>Remarks

Константа GUID для этого атрибута экспортируется из мфууид. lib.
