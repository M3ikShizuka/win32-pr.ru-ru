# <a name="mfnetsource_playerversion-property"></a>МФНЕТСАУРЦЕ \_ плайерверсион, свойство

Значение поля "c-плайерверсион", используемое сетевым источником для ведения журнала. Приложения могут задать для этого свойства номер версии приложения.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

**лонглонг**

VT \_ i8

**хвал**



## <a name="remarks"></a>Remarks

Константа **мфнетсаурце \_ плайерверсион** определяет идентификатор GUID для этого ключа свойства. Идентификатор свойства (PID) равен нулю.

Приложения могут использовать это свойство для настройки сетевого источника. Чтобы задать свойство, передайте указатель **ипропертисторе** в сопоставитель источника. Дополнительные сведения см. в разделе [сопоставитель источников](source-resolver.md).
