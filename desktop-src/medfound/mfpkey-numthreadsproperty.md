# <a name="mfpkey_numthreads-property"></a>МФПКЭЙ \_ NUMTHREADS, свойство

Указывает число потоков, которые будет использовать кодировщик.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкнумсреадс

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

1

## <a name="remarks"></a>Remarks

Это значение предназначено для разделения кодирования на несколько потоков, чтобы использовать преимущества компьютеров с несколькими процессорами. Разделение задач кодирования на несколько потоков может привести к незначительному снижению качества по сравнению с одним потоком.

для видео-кодировщика (wmvencod.dll), выпущенного с Windows XP и Windows Vista, этому свойству должно быть присвоено значение 1, 2 или 4. Другие значения будут округляться вниз.

для видеокодировщика (wmvencod.dll), выпущенного с Windows 7, этому свойству должно быть присвоено значение 1, 2, 4 или 8. Другие значения будут округляться вниз.
