# <a name="mfpkey_wmadec_drcmode-property"></a>МФПКЭЙ \_ вмадек \_ Дркмоде, свойство

Задает динамический режим управления диапазоном, который будет использоваться декодером аудио.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмакдрксеттинг

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

0

## <a name="remarks"></a>Комментарии

Это целое число в диапазоне от 0 до 2. Чем выше значение, тем меньше динамический диапазон распакованных звуковых выходных данных. Значение 0 означает, что кодек не должен выполнять динамический контроль диапазонов, то есть кодек не изменит собственный динамический диапазон закодированного содержимого.

Если это значение равно 1 или 2, то кодек будет использовать следующие свойства для определения необходимого динамического элемента управления диапазоном:

-   [МФПКЭЙ \_ вмадрк \_ авгреф](mfpkey-wmadrc-avgrefproperty.md)
-   [МФПКЭЙ \_ вмадрк \_ пеакреф](mfpkey-wmadrc-peakrefproperty.md)
-   [МФПКЭЙ \_ вмадрк \_ авгтаржет](mfpkey-wmadrc-avgtargetproperty.md)
-   [МФПКЭЙ \_ вмадрк \_ пеактаржет](mfpkey-wmadrc-peaktargetproperty.md)

Если целевые значения не предоставлены, кодек предоставит свой собственный.
