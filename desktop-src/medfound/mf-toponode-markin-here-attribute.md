# <a name="mf_toponode_markin_here-attribute"></a>\_Атрибут MF топоноде \_ Маркин \_ здесь

Указывает, применяет ли конвейер отметку на этом узле. Пометка — это точка, с которой начинается презентация. Если компоненты конвейера формируют данные перед точкой отметки, данные не подготавливаются к просмотру.

## <a name="data-type"></a>Тип данных

**UINT32**

Рассматривать как логическое значение.

## <a name="remarks"></a>Remarks

> [!Note]  
> Большинству приложений не требуется использовать этот атрибут. [Сеанс мультимедиа](media-session.md) автоматически задает этот атрибут при необходимости.

 

Этот атрибут применяется ко всем типам узлов. Если атрибут имеет **значение true**, то конвейер Media Foundation обрезает выходные примеры из этого узла в соответствии со временем начала презентации. Загрузчик топологии задает этот атрибут при разрешении топологии.

Рекомендуется, чтобы только один узел в каждой ветви топологии имел атрибут со значением **true**. Ветвь топологии определяется в качестве пути от исходного узла до выходного узла. В ветви на [ \_ \_ \_ ](mf-toponode-markout-here-attribute.md) \_ \_ \_ одном узле ветви должны быть установлены атрибуты MF топоноде и MF топоноде Маркин. Они не могут быть установлены на разных узлах в одной ветви.

Константа GUID для этого атрибута экспортируется из мфууид. lib.
