# <a name="mft_enum_transcode_only_attribute-attribute"></a>\_Атрибут атрибута с перечислением \_ только для ПЕРЕкодирования MFT \_ \_

Указывает, оптимизирован ли декодер для перекодирования, а не для воспроизведения.

В настоящее время этот атрибут применяется только к аппаратным декодерам, использующим мини-драйвер Австреам. Атрибут хранится в реестре со сведениями о возможностях декодера. Дополнительные сведения см. в документации по [**ижеткапабилитиескэй**](/windows/win32/api/strmif/nn-strmif-igetcapabilitieskey).

Приложения обычно не используют этот атрибут.

## <a name="data-type"></a>Тип данных

**REG \_ DWORD**

## <a name="remarks"></a>Remarks

Если эта запись реестра имеется и равна 1, функция [**мфтенумекс**](/windows/desktop/api/mfapi/nf-mfapi-mftenumex) пропускает декодер, если только вызывающий объект не указал **флаг \_ перечисления MFT enum флага \_ \_ \_ только** в **мфтенумекс**.

Константа GUID для этого атрибута экспортируется из мфууид. lib.
