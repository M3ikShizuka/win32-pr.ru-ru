# <a name="topoedit-toolbar"></a>Панель инструментов Топоедит

По умолчанию панель инструментов отображается в верхней части окна приложения и под строкой меню. Панель инструментов обеспечивает быстрый доступ к пунктам [меню элементов управления](controls-menu.md) с помощью мыши. Кнопки панели инструментов становятся доступными только после успешного разрешения топологии.

На следующем снимке экрана показана панель инструментов Топоедит.

![снимок экрана, показывающий панель инструментов топоедит](images/410257c5-ae5c-454c-a9a8-ba05fc5235b5.jpg)
