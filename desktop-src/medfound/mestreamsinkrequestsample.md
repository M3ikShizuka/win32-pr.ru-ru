# <a name="mestreamsinkrequestsample-event"></a>Событие Местреамсинкрекуестсампле

Вызывается приемником потока для запроса нового примера носителя из конвейера. Для каждого события Местреамсинкрекуестсампле конвейер запрашивает данные от следующего вышестоящего компонента.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |
