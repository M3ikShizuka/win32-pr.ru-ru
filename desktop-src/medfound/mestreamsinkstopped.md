# <a name="mestreamsinkstopped-event"></a>Событие Местреамсинкстоппед

Вызывается приемником потока при завершении перехода в остановленное состояние. Переход на остановленный происходит при вызове метода [**имфпресентатионклокк:: остановки**](/windows/desktop/api/mfidl/nf-mfidl-imfpresentationclock-stop) в часах представления приемника.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |
