Media Foundation определяет несколько базовых типов объектов, используемых во всех Media Foundation API.



| Раздел                                       | Описание                                                                         |
|---------------------------------------------|-------------------------------------------------------------------------------------|
| [Атрибуты](attributes-and-properties.md) | Атрибуты и свойства — это пары "ключ-значение", хранящиеся в объекте.                  |
| [Типы носителей](media-types.md)              | Тип носителя описывает формат потока цифрового мультимедиа.                        |
| [Буферы мультимедиа](media-buffers.md)          | Буфер мультимедиа управляет блоком памяти, чтобы он мог совместно использоваться объектами. |
