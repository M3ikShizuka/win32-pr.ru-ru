# <a name="why-does-the-audio-encoder-report-no-supported-input-types"></a>Почему в отчете Audio Encoder нет поддерживаемых типов входных данных?

Кодировщику аудио не удается определить Поддерживаемые входные типы, пока не задан тип выходных данных. Перечислите типы выходных данных и выберите тот, который соответствует вашим требованиям. После установки типа выходных данных объект будет возвращать Поддерживаемые входные типы.
