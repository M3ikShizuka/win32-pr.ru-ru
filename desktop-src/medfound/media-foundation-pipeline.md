# <a name="media-foundation-pipeline"></a>Конвейер Media Foundation

Уровень *конвейера* в Microsoft Media Foundation — это уровень архитектуры, который непосредственно создает или обрабатывает данные мультимедиа.

Большинство приложений не вызывают методы непосредственно в компонентах конвейера, хотя это можно сделать. Прочтите эти разделы при написании пользовательского компонента конвейера.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                     | Описание                                                                                                                           |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Источники мультимедиа](media-sources.md)<br/>                             | Источники мультимедиа создают данные мультимедиа, обычно из файла, устройства записи или сетевого потока. <br/>                              |
| [Преобразования Media Foundation](media-foundation-transforms.md)<br/> | Media Foundation преобразования (МФТС) обработка данных мультимедиа. Например, кодеки в Media Foundation реализуются как МФТС. <br/>   |
| [Приемники носителей](media-sinks.md)<br/>                                 | Приемники носителей используют данные мультимедиа. Например, приемник мультимедиа может записать данные в файл или отправить данные по сети.<br/> |
