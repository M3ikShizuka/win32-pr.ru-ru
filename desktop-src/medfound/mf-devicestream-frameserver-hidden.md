# <a name="mf_devicestream_frameserver_hidden-attribute"></a>\_ \_ \_ Скрытый атрибут девицестреам фрамесервер

Этот атрибут, если задан в потоке, помечает поток как скрытый от клиента.

## <a name="data-type"></a>Тип данных

**UINT32**

| Требование | Значение |
|-------------------------------------|------------------------------------------------------------------------------------|
| Минимальная версия клиента<br/> | \[Только для настольных приложений Windows 10 версии 1703\]<br/>                          |
| Минимальная версия сервера<br/> | Ни одна версия не поддерживается<br/>                                                          |
