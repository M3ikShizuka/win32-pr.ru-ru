# <a name="mf_source_reader_enable_transcode_only_transforms-attribute"></a>MF \_ исходный \_ считыватель \_ включить \_ \_ \_ атрибут преобразования "только преобразование кода"

Позволяет [модулю чтения исходного кода](source-reader.md) использовать Media Foundation преобразований (МФТС), оптимизированных для перекодирования.

## <a name="data-type"></a>Тип данных

**UINT32**

Рассматривать как логическое значение.

## <a name="remarks"></a>Комментарии

Некоторые МФТС, особенно декодеры, оптимизированы для перекодирования, а не для воспроизведения. По умолчанию модуль чтения исходного кода не будет загружать такие преобразования. Установите для этого атрибута **значение true** , если вы хотите использовать перекодирование МФТС с модулем чтения исходного кода.

Приложение может установить этот атрибут, если он не обрабатывает данные в режиме реального времени (для перекодирования или аналогичных сценариев). В противном случае для воспроизведения в режиме реального времени используйте поведение по умолчанию.

На внутреннем уровне этот атрибут заставляет модуль чтения исходного кода включить **флаг \_ перечисления MFT \_ флага Enum \_ \_ только** при вызове [**мфтенумекс**](/windows/desktop/api/mfapi/nf-mfapi-mftenumex).
