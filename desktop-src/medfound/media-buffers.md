# <a name="media-buffers"></a>Буферы мультимедиа

Буфер мультимедиа — это COM-объект, который управляет блоком памяти, обычно для хранения данных мультимедиа. Буферы мультимедиа используются для перемещения данных из одного компонента конвейера в следующий. Большинство приложений не используют буферы мультимедиа напрямую, так как сеанс мультимедиа обрабатывает весь поток данных между объектами конвейера. Буферы мультимедиа необходимо использовать при написании собственного компонента конвейера или при использовании компонента конвейера напрямую без сеанса мультимедиа.

Буферы мультимедиа предоставляют интерфейс [**имфмедиабуффер**](/windows/desktop/api/mfobjects/nn-mfobjects-imfmediabuffer) . Этот интерфейс предназначен для чтения или записи данных любого типа. Для несжатых видеокадров требуется специальная обработка, так как они могут храниться в областях Direct3D, расположенных в видеопамяти.
