# <a name="mfnetsource_credential_manager-property"></a>\_Свойство диспетчера учетных данных мфнетсаурце \_

Содержит указатель на интерфейс [**имфнеткредентиалманажер**](/windows/desktop/api/mfidl/nn-mfidl-imfnetcredentialmanager) . Используйте это свойство, чтобы передать реализацию приложения [**имфнеткредентиалманажер**](/windows/desktop/api/mfidl/nn-mfidl-imfnetcredentialmanager) в сетевой источник.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

Указатель **IUnknown**

VT \_ Unknown

**пунквал**



## <a name="remarks"></a>Remarks

Константа **мфнетсаурце \_ Credential \_ Manager** определяет **идентификатор GUID** для ключа свойства. Идентификатор свойства (PID) равен нулю.
