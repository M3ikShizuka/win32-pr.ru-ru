# <a name="mfnetsource_browseruseragent-property"></a>МФНЕТСАУРЦЕ \_ бровсерусеражент, свойство

Значение первой части поля "CS (User-Agent)", используемое сетевым источником для ведения журнала. Приложения могут задать для этого свойства любое строковое значение.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

Строка расширенных символов (**WCHAR** \* )

VT \_ LPWSTR

**пвсзвал**



## <a name="remarks"></a>Комментарии

Константа **мфнетсаурце \_ бровсерусеражент** определяет идентификатор GUID для этого ключа свойства. Идентификатор свойства (PID) равен нулю.

Приложения могут использовать это свойство для настройки сетевого источника. Чтобы задать свойство, передайте указатель **ипропертисторе** в сопоставитель источника. Дополнительные сведения см. [в разделе Настройка источника мультимедиа](configuring-a-media-source.md).
