# <a name="mfpkey_bmax-property"></a>МФПКЭЙ \_ бмакс, свойство

Указывает окно буфера (в миллисекундах) для потока с переменным ограничением переменной скорости (VBR) с пиковой скоростью (заданный параметром [мфпкэй \_ рмакс](mfpkey-rmaxproperty.md)).

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкбмакс

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

Значение по умолчанию отсутствует.

## <a name="remarks"></a>Комментарии

Необходимо задать это значение для кодирования с пиковой интенсивностью VBR. После начала обработки образцов вы не должны запрашивать это значение до завершения кодирования потока. Кодек интерпретирует запрос этого значения как сигнал, над которым находится сеанс кодирования. Следующий образец, который вы обрабатываете, рассматривается как начало нового сеанса.
