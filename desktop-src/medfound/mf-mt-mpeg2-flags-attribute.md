# <a name="mf_mt_mpeg2_flags-attribute"></a>\_ \_ \_ Атрибут флагов MF MT MPEG2

Содержит различные флаги для типа мультимедиа MPEG-2.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Комментарии

Этот атрибут соответствует элементу **dwFlags** структуры [**MPEG2VIDEOINFO**](/previous-versions/windows/desktop/api/dvdmedia/ns-dvdmedia-mpeg2videoinfo) . Список допустимых флагов см. в документации по **MPEG2VIDEOINFO**.

Константа GUID для этого атрибута экспортируется из мфууид. lib.
