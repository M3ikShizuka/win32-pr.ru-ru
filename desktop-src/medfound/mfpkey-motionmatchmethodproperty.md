# <a name="mfpkey_motionmatchmethod-property"></a>МФПКЭЙ \_ мотионматчмесод, свойство

Указывает метод, используемый для сопоставления движения.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкмотионматчмесод

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

0

## <a name="remarks"></a>Remarks

Для этого свойства можно задать одно из следующих значений.



| Значение | Используемый метод                       |
|-------|-----------------------------------|
| 0     | Сумма абсолютных различий (SAD) |
| 1     | Hadamard                          |
| –1    | Макроблокк — адаптивный.              |



 

Сумма абсолютных различий (SAD) — это более быстрый, но менее точный метод, чем преобразование Хадамард. Преобразование Хадамард является более точным, но трудоемким. Адаптивный режим макроблокк обеспечивает адекватный компромисс между двумя методами путем динамического выбора между двумя преобразованиями, выбирая преобразование Хадамард только при необходимости.
