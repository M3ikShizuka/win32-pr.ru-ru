# <a name="media-type-helper-functions"></a>Вспомогательные функции типа мультимедиа

Следующие функции относятся к объектам типа мультимедиа.



| Функция                                                                     | Описание                                                           |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| [**мфаверажетимеперфраметофрамерате**](/windows/desktop/api/mfapi/nf-mfapi-mfaveragetimeperframetoframerate) | Вычисляет частоту кадров от средней длительности видеокадра. |
| [**мфкалкулатеимажесизе**](/windows/desktop/api/mfapi/nf-mfapi-mfcalculateimagesize)                         | Получает размер изображения для несжатого видеоформата.            |
| [**мфкомпарефуллтопартиалмедиатипе**](/windows/desktop/api/mfapi/nf-mfapi-mfcomparefulltopartialmediatype)   | Сравнение полного типа носителя с частичным типом носителя.                   |
| [**мфкреатемедиатипе**](/windows/desktop/api/mfapi/nf-mfapi-mfcreatemediatype)                               | Создает пустой тип носителя.                                          |
| [**мффрамератетоаверажетимеперфраме**](/windows/desktop/api/mfapi/nf-mfapi-mfframeratetoaveragetimeperframe) | Преобразует частоту кадров видео в Длительность кадра.                    |
| [**мфжетстридефорбитмапинфохеадер**](/windows/desktop/api/mfapi/nf-mfapi-mfgetstrideforbitmapinfoheader)     | Возвращает минимальный шаг поверхности для формата видео.              |
| [**мфисформатюв**](/windows/desktop/api/mfapi/nf-mfapi-mfisformatyuv)                                       | Запрашивает, является ли код FOURCC или значение **D3DFORMAT** форматом YUV. |
| [**мфвалидатемедиатипесизе**](/windows/desktop/api/mfapi/nf-mfapi-mfvalidatemediatypesize)                   | Проверяет размер буфера для блока видеоформата.              |
| [**мфврапмедиатипе**](/windows/desktop/api/mfapi/nf-mfapi-mfwrapmediatype)                                   | Создает тип носителя, который служит оболочкой для другого типа мультимедиа.                   |
