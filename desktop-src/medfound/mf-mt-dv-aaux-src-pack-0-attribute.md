# <a name="mf_mt_dv_aaux_src_pack_0-attribute"></a>% \_ MF \_ DV \_ ааукс \_ src \_ , \_ атрибут Pack 0

Исходный пакет Audio вспомогательный (ААУКС) для первого звукового блока в цифровом видеоролике (DV).

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

этот атрибут соответствует элементу **двдвааукссрк** структуры DirectShow [**двинфо**](/windows/win32/api/strmif/ns-strmif-dvinfo) .

Константа GUID для этого атрибута экспортируется из мфууид. lib.
