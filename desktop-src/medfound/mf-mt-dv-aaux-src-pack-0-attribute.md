# <a name="mf_mt_dv_aaux_src_pack_0-attribute"></a>% \_ MF \_ DV \_ ааукс \_ src \_ , \_ атрибут Pack 0

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Комментарии

Этот атрибут соответствует элементу **двдвааукссрк** структуры [**двинфо**](/windows/win32/api/strmif/ns-strmif-dvinfo) DirectShow.

Константа GUID для этого атрибута экспортируется из мфууид. lib.
