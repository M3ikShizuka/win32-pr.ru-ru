# <a name="overview-of-table-of-contents-parser"></a>Общие сведения о средстве синтаксического анализа оглавления

средство синтаксического анализа оглавления (средство синтаксического анализа TOC) — это Windows компонент, позволяющий хранить дополнительные данные индекса в файлах мультимедиа. Примерами расширенных сведений об индексах являются таблицы содержимого, семантические индексы и доски истории.

С помощью средства синтаксического анализа ОГЛАВЛЕНИя можно написать программное обеспечение, позволяющее пользователям находить нужные сцены в видеофайл. Например, можно предоставить пользователю оглавление, чтобы он мог найти основные разделы видео, а затем просмотреть конкретный раздел. Можно также предоставить пользователю индекс, чтобы он мог найти список сцен, связанных с определенным словом или фразой.

генератор оглавления (генератор toc) — это Windows компонент, который работает вместе с анализатором оглавления. Генератор ОГЛАВЛЕНИя может обнаруживать сцены и кластеры сцен в файле видео, а затем формировать оглавление, которое можно использовать в качестве отправной точки в программном обеспечении.

В следующем списке приводятся некоторые сценарии, в которых можно использовать службы, предоставляемые средством синтаксического анализа или создания ОГЛАВЛЕНИя.

-   Создайте оглавление с нуля и внедрите его в видеофайл.
-   Чтение оглавления, уже внедренного в видеофайл.
-   Изменение оглавления, уже внедренного в видеофайл.
-   Используйте генератор ОГЛАВЛЕНИя для создания оглавления на основе сцен и кластеров, обнаруженных в видеофайле. Измените созданное оглавление и внедрите его в видеофайл.
