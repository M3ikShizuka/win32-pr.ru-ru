# <a name="mfpkey_deltamvrangeindex-property"></a>МФПКЭЙ \_ делтамвранжеиндекс, свойство

Указывает метод, используемый для кодирования сведений о векторе движения.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкделтамвранжеиндекс

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

0

## <a name="remarks"></a>Комментарии

Если это свойство задано, кодировщик увеличивает диапазон разностных векторов движения в полях. Сведения о векторе движения допустимы только для полей с чередованием.

Для этого свойства можно задать одно из следующих значений:



| Значение | Описание                                                                          |
|-------|--------------------------------------------------------------------------------------|
| 0     | Использовать существующий диапазон разностных векторов движения.                                          |
| 1     | Дважды диапазон разностного вектора движения в горизонтальном направлении.                    |
| 2     | Дважды диапазон разностного вектора движения в вертикальном направлении.                      |
| 3     | Дважды диапазон разностного вектора движения в горизонтальном и вертикальном направлениях. |
