# <a name="mfpkey_keydist-property"></a>МФПКЭЙ \_ кэйдист, свойство

Указывает максимальное время в миллисекундах между опорными кадрами в выходных коде.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвккэйфрамедистанце

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

Значение по умолчанию зависит от того, какая версия Windows выполняется, как показано в следующей таблице.



| Операционная система | Значение по умолчанию |
|------------------|---------------|
| Windows XP       | 8000          |
| Windows Vista    | 8000          |
| Windows 7        | 2000          |



 

## <a name="remarks"></a>Комментарии

Внутренняя логика кодека определяет фактическое расположение каждого ключевого кадра. Расстояние между любыми двумя ключевыми кадрами может быть меньше значения этого свойства, однако оно никогда не будет больше.
