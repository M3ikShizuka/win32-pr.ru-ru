# <a name="mfnetsource_udp_port_range-property"></a>МФНЕТСАУРЦЕ \_ UDP \_ - \_ свойство диапазона портов

Диапазон допустимых UDP-портов, которые источник сети может использовать для получения потокового содержимого. Значение этого свойства представляет собой строку в формате " *m*–*n* ", где *m* и *n* — номера портов.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

Строка расширенных символов (**WCHAR** \* )

VT \_ LPWSTR

**пвсзвал**



## <a name="remarks"></a>Комментарии

**\_ \_ \_ Диапазон UDP-портов Constant мфнетсаурце** определяет идентификатор GUID для этого ключа свойства. Идентификатор свойства (PID) равен нулю.

Приложения могут использовать это свойство для настройки сетевого источника. Чтобы задать свойство, передайте указатель **ипропертисторе** в сопоставитель источника. Дополнительные сведения см. [в разделе Настройка источника мультимедиа](configuring-a-media-source.md).
