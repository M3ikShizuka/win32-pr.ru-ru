# <a name="mf_mt_mpeg2_content_packet-attribute"></a>\_ \_ \_ Атрибут пакета содержимого MF MT MPEG2 \_

Для типа мультимедиа, описывающего транспортный поток MPEG-2, указывает, содержат ли транспортные пакеты заголовки пакетов содержимого.

## <a name="data-type"></a>Тип данных

**UINT32**



| Значение                                                                                                | Значение                                                                                                                                                                                                            |
|------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="0"></span><dl> <dt>**0**</dt> </dl> | Заголовки пакетов содержимого не добавляются.<br/>                                                                                                                                                                    |
| <span id="1"></span><dl> <dt>**1**</dt> </dl> | При 200 – 1000 миллисекундах в начало транспортного пакета добавляется 14-байтовый заголовок пакета содержимого, который определяется ассоциацией радиоотраслей и бизнес-АРИБ.<br/> |
