# <a name="mfpkey_bdeltaqp-property"></a>МФПКЭЙ \_ бделтакп, свойство

Указывает Дельта-увеличение между рисунком куантизер фрейма привязки и изображением куантизер в B-кадре.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкбделтакп

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

0

## <a name="remarks"></a>Комментарии

Picture куантизер (QP) — это мера сжатия кадра. Более высокие значения представляют более высокие коэффициенты сжатия. QP можно задать с шагом в половину точек. Как правило, B-кадр сжимается в обработчике запросов так же, как или 0,5 выше, чем обработчик QP маркера привязки. При указании разностного пакета запросов в виде «B-Frame» выше 0 можно сжимать B-кадры при увеличении коэффициента сжатия.

Разностный обработчик запросов "B-Frame" можно задать только в приращениях с целыми точками. Этому свойству должно быть присвоено целое значение от 0 до 31. Рекомендуемое значение — 1.
