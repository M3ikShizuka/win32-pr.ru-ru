# <a name="mf_source_reader_disable_camera_plugins-attribute"></a>\_Чтение источника \_ MF \_ Отключить \_ \_ атрибут подключаемых модулей камеры

Отключает использование подключаемых модулей камеры, выполняемых после обработки, модулем [чтения исходного кода](source-reader.md).

## <a name="data-type"></a>Тип данных

**Bool** , сохраненный как **UINT32**

## <a name="remarks"></a>Remarks

Этот атрибут применяется, когда модуль чтения исходного кода используется с источником видеозаписи. Если этот атрибут имеет **значение true**, то модуль чтения исходного кода не сможет загружать какие-либо подключаемые модули последующей обработки, которые может предоставить камера. В противном случае по умолчанию средство чтения источника загрузит их.

Значение этого атрибута по умолчанию равно **false**. Задайте атрибут при создании модуля чтения исходного кода.
