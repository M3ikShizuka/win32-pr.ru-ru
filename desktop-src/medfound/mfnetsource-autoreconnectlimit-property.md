# <a name="mfnetsource_autoreconnectlimit-property"></a>МФНЕТСАУРЦЕ \_ аутореконнектлимит, свойство

Количество попыток повторного подключения сетевого источника к серверу и возобновления потоковой передачи при потере соединения.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

**DWORD** (хранится в формате **Long**)

VT \_ I4

**лвал**



## <a name="remarks"></a>Remarks

Константа **мфнетсаурце \_ аутореконнектлимит** определяет идентификатор GUID для этого ключа свойства. Идентификатор свойства (PID) равен нулю.

Приложения могут использовать это свойство для настройки сетевого источника. Чтобы задать свойство, передайте объект **ипропертисторе** в сопоставитель источника. Дополнительные сведения см. [в разделе Настройка источника мультимедиа](configuring-a-media-source.md).
