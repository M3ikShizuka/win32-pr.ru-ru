# <a name="mfpkey_wmaaecma_featr_micarr_preproc-property"></a>МФПКЭЙ \_ вмааекма \_ феатр \_ микарр, \_ свойство

Указывает, выполняет ли система DSP для записи голоса предварительную обработку массива микрофонов.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

Доступно только с помощью [**ипропертисторе**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Тип данных

Логическое значение VT \_

## <a name="default-value"></a>Значение по умолчанию

ВАРИАНТ \_ true

## <a name="applies-to"></a>Применяется к

-   [DSP для записи речи](voicecapturedmo.md)

## <a name="remarks"></a>Remarks

Предварительная обработка может удалить стационарные звуки, мешающие обработке, например тон с фиксированным шагом.

Это свойство может иметь следующие значения.



| Значение          | Описание            |
|----------------|------------------------|
| ВАРИАНТ \_ false | Отключить предварительную обработку. |
| ВАРИАНТ \_ true  | Включить предварительную обработку.  |



 

Значением этого свойства по умолчанию является вариант \_ true (включено). Перед установкой этого свойства необходимо присвоить свойству [ \_ \_ \_ режима компонента мфпкэй вмааекма](mfpkey-wmaaecma-feature-modeproperty.md) значение Variant \_ true.

DSP использует это свойство только в том случае, если включена обработка массива микрофона.
