# <a name="mfpkey_dyn_bandtrunc_qfloor-property"></a>МФПКЭЙ \_ Дин \_ бандтрунк \_ кфлур, свойство

Определяет качество, при котором разрешена максимальная пропускная способность. По качеству, которое лучше или равно значению этого свойства, Допускается использование максимальной закодированной пропускной способности.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

Доступно только с помощью [**ипропертисторе**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Тип данных

**VT \_ R4**
