# <a name="metransformmarker-event"></a>Событие Метрансформмаркер

Отправляется асинхронным Media Foundation преобразованием (MFT) в ответ на сообщение **\_ \_ \_ маркера команды сообщения MFT** .

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание               |
|----------------------|---------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> |



## <a name="attributes"></a>Атрибуты

Для этого события определены следующие атрибуты.



| attribute                                                      | Описание                                                                                                                |
|----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [\_ \_ контекст MFT события \_ MF](mf-event-mft-context.md)<br/> | Значение параметра *улпарам* из сообщения **\_ \_ \_ метки команды сообщения MFT** .<br/>*Необходимости*<br/> |



## <a name="remarks"></a>Комментарии

Асинхронный МФТС отправляет это событие через интерфейс [**имфмедиаевентженератор**](/windows/desktop/api/mfobjects/nn-mfobjects-imfmediaeventgenerator) . Синхронное МФТС никогда не отправляет это событие.

Клиент асинхронной таблицы MFT может поместить маркер в поток, вызвав [**имфтрансформ::P роцессмессаже**](/windows/desktop/api/mftransform/nf-mftransform-imftransform-processmessage) с сообщением **\_ \_ \_ маркера команды сообщения MFT** . Параметр *улпарам* содержит данные, определяемые приложением.

Когда MFT заканчивает обработку всех входных данных, которые были доступны во время вызова [**ProcessMessage**](/windows/desktop/api/mftransform/nf-mftransform-imftransform-processmessage) , MFT ставит в очередь событие метрансформмаркер. Атрибут [ \_ \_ \_ context события MF в MFT](mf-event-mft-context.md) события содержит значение параметра *улпарам* . Дополнительные сведения см. в разделе [асинхронная МФТС](asynchronous-mfts.md).
