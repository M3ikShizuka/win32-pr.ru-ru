# <a name="writing-an-asf-header-object-for-a-new-file"></a>Запись объекта заголовка ASF для нового файла

Объект ASF Контентинфо сохраняет сведения об объекте заголовка ASF для файла. При создании или изменении ASF-файла необходимо создать объект заголовка. Для этого приложение должно предоставить профиль кодирования содержимого объекту Контентинфо, чтобы он знал о характеристиках создаваемого файла мультимедиа.

Для записи нового файла можно использовать объект Контентинфо для:

-   Собирайте данные заголовка из объекта профиля создаваемого файла,
-   Заполните различные объекты заголовков в библиотеке ASF, которые поддерживаются внутренне Media Foundation,
-   Инициализируйте [мультиплексор ASF](asf-multiplexer.md) для создания пакета данных ASF и
-   Создайте объект верхнего уровня заголовка в двоичном формате, который может быть записан в файл.

Дополнительные сведения о профилях см. в разделе [профиль ASF](asf-profile.md).
