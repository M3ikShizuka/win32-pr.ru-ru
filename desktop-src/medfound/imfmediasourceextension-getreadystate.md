# <a name="imfmediasourceextensiongetreadystate-method"></a>Метод Имфмедиасаурцеекстенсион:: Жетреадистате

Возвращает состояние готовности источника мультимедиа.

## <a name="syntax"></a>Синтаксис


```C++
MF_MSE_READY GetReadyState();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Состояние готовности источника мультимедиа.
