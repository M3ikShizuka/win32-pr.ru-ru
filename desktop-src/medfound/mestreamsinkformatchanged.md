# <a name="mestreamsinkformatchanged-event"></a>Событие Местреамсинкформатчанжед

Вызывается приемником потока, когда тип носителя приемника больше не действителен.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |



## <a name="remarks"></a>Remarks

Приемник потока может вызвать это, даже если что-то произошло, что сделает недействительным тип носителя приемника потока. Например, расширенный обработчик видео (Евр) отправляет это событие при изменении отображения.

Значение **HRESULT** , полученное с помощью [**имфмедиаевент::-Status**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getstatus) , может означать, почему тип мультимедиа больше не действителен.
