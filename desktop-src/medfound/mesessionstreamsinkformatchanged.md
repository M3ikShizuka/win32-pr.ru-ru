# <a name="mesessionstreamsinkformatchanged-event"></a>Событие Месессионстреамсинкформатчанжед

Вызывается сеансом мультимедиа при изменении формата в приемнике мультимедиа.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |



## <a name="attributes"></a>Атрибуты

Для этого события определены следующие атрибуты.



| attribute                                                                    | Описание                                                                                  |
|------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**\_ \_ узел вывода событий \_ MF**](mf-event-output-node-attribute.md)<br/> | Определяет узел топологии для приемника мультимедиа, формат которого изменился.<br/> <br/> |



## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|----------------------------------------------------------------------------------------------------------|
| Минимальная версия клиента<br/> | Только для \[ классических приложений Windows Vista\]<br/>                                                           |
| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2008\]<br/>                                                     |
