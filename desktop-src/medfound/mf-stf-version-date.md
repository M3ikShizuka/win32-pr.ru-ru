# <a name="mf_stf_version_date-attribute"></a>\_ \_ Атрибут даты версии MF СТФ \_

Этот атрибут хранит значение [fileTime](/windows/desktop/api/minwinbase/ns-minwinbase-filetime) , представляющее метку даты и времени фабрики преобразования датчика.

## <a name="data-type"></a>Тип данных

**UINT64**

|-------------------------------------|------------------------------------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows 10, только для \[ настольных приложений версии 1703\]<br/>                          |
| Минимальная версия сервера<br/> | Ни одна версия не поддерживается<br/>                                                          |
