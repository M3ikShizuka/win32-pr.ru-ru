# <a name="mfnetsource_logurl-property"></a>МФНЕТСАУРЦЕ \_ логурл, свойство

Список URL-адресов, на которые сетевой источник будет передавать данные журнала.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

Массив строк расширенных символов (**калпвстр**)

VT \_ Векторная \| \_ LPWSTR

**калпвстр**



## <a name="remarks"></a>Remarks

Константа **мфнетсаурце \_ логурл** определяет идентификатор GUID для этого ключа свойства. Идентификатор свойства (PID) равен нулю.

Приложения могут использовать это свойство для настройки сетевого источника. Чтобы задать свойство, передайте указатель **ипропертисторе** в сопоставитель источника. Дополнительные сведения см. [в разделе Настройка источника мультимедиа](configuring-a-media-source.md).
