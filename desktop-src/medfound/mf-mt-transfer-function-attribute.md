# <a name="mf_mt_transfer_function-attribute"></a>\_ \_ Атрибут функции перемещения MF MT \_

Указывает функцию преобразования из RGB в Р'Г'Б "для типа видеоклипа.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Комментарии

Значение этого атрибута является членом перечисления [**мфвидеотрансферфунктион**](/windows/desktop/api/mfobjects/ne-mfobjects-mfvideotransferfunction) .
