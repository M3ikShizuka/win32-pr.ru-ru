# <a name="mf_mt_video_3d_left_is_base-attribute"></a>\_Полевой \_ \_ 3D-ролик MF \_ \_ — \_ базовый атрибут

Для формата 3D-видео указывает, какое представление является базовым представлением.

## <a name="data-type"></a>Тип данных

**Bool** , сохраненный как **UINT32**

## <a name="remarks"></a>Комментарии

По умолчанию левое представление в трехмерной видеопоследовательности является базовым представлением. Если в правильном представлении используется базовое представление, присвойте этому атрибуту **значение false**.

Чтобы преобразовать видео стереоскопик в 2D, следует сохранить базовое представление и отменить другое представление.
