# <a name="mfcaptured3d-sample"></a>Пример MFCaptureD3D

Демонстрация просмотра видео с видеокамеры с помощью Direct3D для визуализации видео.

## <a name="apis-demonstrated"></a>Продемонстрированные API

В этом образце демонстрируются следующие API-интерфейсы.

-   [**имфмедиасаурце**](/windows/desktop/api/mfidl/nn-mfidl-imfmediasource)
-   [**имфсаурцереадер**](/windows/desktop/api/mfreadwrite/nn-mfreadwrite-imfsourcereader)
-   [**мфенумдевицесаурцес**](/windows/desktop/api/mfidl/nf-mfidl-mfenumdevicesources)
