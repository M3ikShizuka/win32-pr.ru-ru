# <a name="mf_devicestream_frameserver_shared-attribute"></a>\_ \_ Общий атрибут MF девицестреам фрамесервер \_

Этот атрибут, если он задан в потоке, явно помечает поток как общий для сервера Frame.

## <a name="data-type"></a>Тип данных

**UINT32**

| Требование | Значение |
|-------------------------------------|------------------------------------------------------------------------------------|
| Минимальная версия клиента<br/> | \[Только для настольных приложений Windows 10 версии 1703\]<br/>                          |
| Минимальная версия сервера<br/> | Ни одна версия не поддерживается<br/>                                                          |
