# <a name="mf_mt_audio_avg_bytes_per_second-attribute"></a>\_ \_ \_ Среднее \_ число байт \_ в секунду для \_ звука MF MT

Среднее число байтов в секунду для типа звукового носителя.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

Этот атрибут соответствует элементу **навгбитесперсек** структуры [**вавеформатекс**](/previous-versions/dd757713(v=vs.85)) .

Константа GUID для этого атрибута экспортируется из мфууид. lib.
