# <a name="major-media-types"></a>Основные типы носителей

В типе мультимедиа *основной тип* описывает общую категорию данных, например аудио или видео. *Подтип*, если он есть, дополнительно уточните основной тип. Например, если основной тип — Video, подтип может быть 32-разрядным видео RGB. Подтипы также отличают закодированные форматы, такие как видео H. 264, от несжатых форматов.

Основной тип и подтип идентифицируются по идентификаторам GUID и хранятся в следующих атрибутах:



| attribute                                             | Описание |
|-------------------------------------------------------|-------------|
| [\_ \_ основной тип MF \_ MT](mf-mt-major-type-attribute.md) | Основной тип. |
| [подтип MF \_ MT \_](mf-mt-subtype-attribute.md)        | Подтип.    |



 

Определены следующие основные типы.



| Основной тип                    | Описание                                                                                                                                                | Подтипы                                             |
|-------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------|
| **Мфмедиатипе \_ аудио**        | Аудиосигнал.                                                                                                                                                     | [Идентификаторы GUID для звуковых подтипов](audio-subtype-guids.md).      |
| **\_Двоичный файл мфмедиатипе**       | Двоичный поток.                                                                                                                                             | Нет.                                                |
| **Мфмедиатипе \_ филетрансфер** | Поток, содержащий файлы данных.                                                                                                                         | Нет.                                                |
| **Мфмедиатипе \_ HTML**         | Поток HTML.                                                                                                                                               | Нет.                                                |
| **\_Изображение мфмедиатипе**        | Поток изображений по-прежнему.                                                                                                                                        | [Идентификаторы GUID и CLSID WIC](../wic/-wic-guids-clsids.md).       |
| **Мфмедиатипе с \_ защитой**    | Защищенный носитель.                                                                                                                                           | Подтип определяет схему защиты содержимого. |
| **\_Восприятие мфмедиатипе**   | Потоки из датчика камеры или единицы обработки, которые являются причиной и распознающи необработанные видеоданные и позволяют понять окружающую среду и людей. | Нет.                                                |
| **Мфмедиатипе \_ Sami**         | Синхронизированные доступные субтитры обмена мультимедиа (SAMI).                                                                                                 | Нет.                                                |
| **\_Сценарий мфмедиатипе**       | Поток скрипта.                                                                                                                                             | Нет.                                                |
| **\_Поток мфмедиатипе**       | Мультиплексированный поток или простейший поток.                                                                                                                   | [Идентификаторы GUID подтипов потока](stream-subtype-guids.md)     |
| **\_Видео мфмедиатипе**        | Видео.                                                                                                                                                     | [Идентификаторы GUID для подтипов видео](video-subtype-guids.md).      |



 

Сторонние компоненты могут определять новые основные типы и новые подтипы.
