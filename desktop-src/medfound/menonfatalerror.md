# <a name="menonfatalerror-event"></a>Событие Менонфаталеррор

Во время потоковой передачи произошла устранимая ошибка. Любой компонент Media Foundation может отправить это событие в любое время.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE            | Описание                                                        |
|--------------------|--------------------------------------------------------------------|
| VT \_ UI4<br/> | Значение **HRESULT** , описывающее ошибку.<br/> <br/> |



## <a name="attributes"></a>Атрибуты

Для этого события не определены атрибуты.

## <a name="remarks"></a>Комментарии

Это событие сигнализирует о непредвиденной, но устранимой ошибке во время потоковой передачи. Например, источник мультимедиа может отправить это событие при удалении пакета.

Не отправляйте это событие, чтобы сообщить о сбое асинхронного метода. В случае сбоя асинхронного метода код ошибки возвращается в виде обычного события для этого метода.

Если возникает неустранимая ошибка потоковой передачи, отправьте событие [Миррор](meerror.md) .
