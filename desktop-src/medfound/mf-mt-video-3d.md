# <a name="mf_mt_video_3d-attribute"></a>\_ \_ \_ Трехмерный видео-атрибут MF MT

Указывает, содержит ли поток видео трехмерное содержимое.

## <a name="data-type"></a>Тип данных

**Bool** , сохраненный как **UINT32**

## <a name="remarks"></a>Remarks

Этот атрибут применяется к типам видеоклипов. Если этот атрибут имеет **значение true**, поток видео содержит трехмерное содержимое. Значение по умолчанию — **FALSE**.
