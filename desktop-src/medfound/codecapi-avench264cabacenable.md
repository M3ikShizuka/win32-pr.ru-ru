# <a name="codecapi_avench264cabacenable-property"></a>КОДЕКАПИ \_ AVEncH264CABACEnable, свойство

Включает или отключает кабак (Адаптивное двоичное арифметическое кодирование) для H. 264-кода энтропии.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (логическое значение VT \_ )

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ AVEncH264CABACEnable**

## <a name="property-value"></a>Значение свойства

Если значение является **вариантным \_ true**, Кабак включен.
