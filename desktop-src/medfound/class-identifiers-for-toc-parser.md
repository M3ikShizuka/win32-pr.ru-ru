# <a name="class-identifiers-for-table-of-contents-parser"></a>Идентификаторы классов для средства синтаксического анализа оглавления

Следующие классы объявляются и связываются с идентификаторами классов (CLSID) в вмкодекдсп. h.



| Имя класса       | Понятное имя объекта |
|------------------|----------------------|
| ктоцентри        | Запись ОГЛАВЛЕНИя            |
| ктоцентрилист    | Список записей ОГЛАВЛЕНИя       |
| кток             | ОГЛАВЛЕНИЕ                  |
| ктокколлектион   | Коллекция TOC       |
| ктокпарсер       | Средство синтаксического анализа ОГЛАВЛЕНИя           |
| ктокженератордмо | Генератор содержания        |



 

В приведенной выше таблице приводятся понятные имена объектов для каждого класса. В этой документации эти понятные имена используются для ссылки на экземпляры классов. Например, документация ссылается на экземпляр класса Ктоцентри в качестве объекта записи ОГЛАВЛЕНИя.

В коде можно использовать **\_ \_ ууидоф** для ссылки на идентификаторы CLSID. Например, можно использовать следующий код для ссылки на идентификатор CLSID для Ктокженератордмо.


```C++
__uuidof(CTocGeneratorDmo)
```



### <a name="clsid-constants-defined-in-wmcodecdsph"></a>Константы CLSID, определенные в Вмкодекдсп. h

В качестве альтернативы использованию **\_ \_ ууидоф** можно использовать константы для ссылки на идентификаторы CLSID. В вмкодекдсп. h определены следующие константы.



| Имя класса     | Константа CLSID        |
|----------------|-----------------------|
| ктоцентри      | \_КТОЦЕНТРИ CLSID      |
| ктоцентрилист  | \_КТОЦЕНТРИЛИСТ CLSID  |
| кток           | \_КТОК CLSID           |
| ктокколлектион | \_КТОККОЛЛЕКТИОН CLSID |
| ктокпарсер     | \_КТОКПАРСЕР CLSID     |



 

### <a name="clsid-constants-defined-in-wmcodecdspuuidlib"></a>Константы CLSID, определенные в Вмкодекдспууид. lib

Следующая константа CLSID объявлена, но не определена в вмкодекдсп. h. Чтобы использовать его в коде, необходимо выполнить компоновку к вмкодекдспууид. lib.



| Имя класса       | Константа CLSID          |
|------------------|-------------------------|
| ктокженератордмо | \_КТОКЖЕНЕРАТОРДМО CLSID |
