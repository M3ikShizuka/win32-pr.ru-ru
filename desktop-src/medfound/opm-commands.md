# <a name="opm-commands"></a>Команды ОПМ

В этом разделе перечислены доступные команды для [Output Protection Manager](output-protection-manager.md) (ОПМ).

Чтобы отправить команду ОПМ, вызовите [**иопмвидеуутпут:: Configure**](/windows/desktop/api/opmapi/nf-opmapi-iopmvideooutput-configure). Для каждой команды перечислены следующие сведения.



| Значение             | Описание                                                                                                                                                            |
|--------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Идентификатор GUID команды | Идентифицирует команду. Задайте для элемента **гуидсеттинг** структуры [**\_ \_ параметров ОПМ**](/windows/desktop/api/opmapi/ns-opmapi-opm_configure_parameters) значение, равное этому значению. |
| Входные данные   | Указывает способ интерпретации массива **абпараметерс** в структуре [**параметров ОПМ \_ Configure \_**](/windows/desktop/api/opmapi/ns-opmapi-opm_configure_parameters) .                      |



 

Определены следующие команды:



| Команда                                                                                                       | Описание                                                                                         |
|---------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**ОПМ \_ Установка \_ \_ сигнала ACP и \_ кгмса \_**](opm-set-acp-and-cgmsa-signaling.md)                               | Указывает сведения о видеосигнале, отличном от уровня защиты.                      |
| [**ОПМ \_ Set \_ HDCP \_ СРМ**](opm-set-hdcp-srm.md)                                                               | Обновляет сообщение о возобновлении системы (СРМ) для High-Bandwidth Digital Защита содержимого (HDCP). |
| [**ОПМ \_ установить \_ \_ уровень защиты**](opm-set-protection-level.md)                                               | Задает уровень защиты для механизма защиты выходных данных.                                       |
| [**ОПМ \_ установить \_ \_ уровень защиты \_ в \_ соответствии \_ с \_ DVD-диском CSS**](opm-set-protection-level-according-to-css-dvd.md) | Задает уровень защиты HDCP для воспроизведения DVD-дисков, следующие правила системы расшифровки содержимого DVD-диска (CSS). |
