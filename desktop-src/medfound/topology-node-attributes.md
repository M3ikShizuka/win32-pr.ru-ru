# <a name="topology-node-attributes"></a>Атрибуты узла топологии

Следующие атрибуты применяются к узлам топологии.

## <a name="general-topology-node-attributes"></a>Общие атрибуты узла топологии



| attribute                                                                       | Описание                                                                                                                                              |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ метод подключения MF \_ топоноде**](mf-toponode-connect-method-attribute.md)   | Указывает, как загрузчик топологии подключает этот узел топологии и является ли этот узел необязательным.                                                        |
| [**\_декодер MF топоноде \_**](mf-toponode-decoder-attribute.md)                  | Указывает, является ли объект узла топологии декодером.                                                                                                  |
| [**\_ \_ расшифровка MF топоноде**](mf-toponode-decryptor-attribute.md)              | Указывает, является ли базовый объект узла топологии расшифровывается.                                                                                     |
| [**\_ТОПОНОДЕ MF \_**](mf-toponode-discardable-attribute.md)          | Указывает, может ли конвейер удалять образцы из узла топологии.                                                                                    |
| [**MF \_ топоноде \_ Ошибка \_ мажортипе**](mf-toponode-error-majortype-attribute.md) | Содержит основной тип носителя для узла топологии. Этот атрибут задается, когда не удается загрузить топологию, так как не удалось найти правильный декодер. |
| [**\_ \_ ПОДТИП ошибки MF \_ топоноде**](mf-toponode-error-subtype-attribute.md)     | Содержит подтип носителя для узла топологии. Этот атрибут задается, когда не удается загрузить топологию, так как не удалось найти правильный декодер.    |
| [**\_ \_ код ошибки MF топоноде**](mf-toponode-errorcode-attribute.md)              | Содержит код ошибки последней неудачной попытки подключения для данного узла топологии.                                                                  |
| [**MF \_ топоноде \_ заблокировано**](mf-toponode-locked-attribute.md)                    | Указывает, можно ли изменить типы мультимедиа в этом узле топологии.                                                                                  |
| [**MF \_ топоноде \_ Маркин \_**](mf-toponode-markin-here-attribute.md)         | Указывает, применяет ли конвейер отметку на этом узле.                                                                                             |
| [**Разметка MF \_ топоноде \_ \_**](mf-toponode-markout-here-attribute.md)       | Указывает, применяется ли к этому узлу отметка для конвейера.                                                                                            |



 

## <a name="source-node-attributes"></a>Атрибуты исходного узла



| attribute                                                                                       | Описание                                                                                                       |
|-------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [**MF \_ топоноде \_ медиастарт**](mf-toponode-mediastart-attribute.md)                            | Указывает время начала презентации относительно запуска исходного файла мультимедиа в единицах измерения в 100 нс. |
| [**MF \_ топоноде \_ медиастоп**](mf-toponode-mediastop-attribute.md)                              | Указывает время окончания презентации относительно запуска исходного файла мультимедиа в единицах 100-наносекундных.  |
| [**\_ \_ дескриптор представления MF \_ топоноде**](mf-toponode-presentation-descriptor-attribute.md) | Содержит указатель на дескриптор представления для источника мультимедиа.                                           |
| [**MF \_ топоноде \_ Sequence, пакет \_ ELEMENTID**](mf-toponode-sequence-elementid-attribute.md)           | Указывает элемент, содержащий исходный узел.                                                                |
| [**\_источник MF топоноде \_**](mf-toponode-source-attribute.md)                                    | Содержит указатель на источник мультимедиа, связанный с узлом топологии.                                           |
| [**\_ \_ дескриптор потока MF \_ топоноде**](mf-toponode-stream-descriptor-attribute.md)             | Содержит указатель на дескриптор потока для источника мультимедиа.                                                 |
| [**\_ \_ ИД ворккуеуе ТОПОНОДЕ для MF \_**](mf-toponode-workqueue-id-attribute.md)                       | Указывает рабочую очередь для узла топологии.                                                                       |
| [**\_ \_ \_ класс MMCSS ТОПОНОДЕ ворккуеуе \_ (MF)**](mf-toponode-workqueue-mmcss-class-attribute.md)    | Указывает задачу службы планировщика классов мультимедиа (MMCSS) для узла топологии.                                  |
| [**MF \_ топоноде \_ ворккуеуе \_ MMCSS \_ TASKID**](mf-toponode-workqueue-mmcss-taskid-attribute.md)  | Указывает идентификатор задачи MMCSS для узла топологии.                                                            |



 

## <a name="transform-node-attributes"></a>Преобразование атрибутов узла



| attribute                                                                             | Описание                                                                                                |
|---------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**MF \_ топоноде \_ D3DAWARE**](mf-toponode-d3daware-attribute.md)                      | Указывает, поддерживает ли преобразование, связанное с узлом топологии, поддержку ускорения видео DirectX (ДКСВА) |
| [**\_сток ТОПОНОДЕ \_ MF**](mf-toponode-drain-attribute.md)                            | Указывает, когда преобразуются преобразования.                                                                     |
| [**\_топоноде \_ запись на диск MF**](mf-toponode-flush-attribute.md)                            | Указывает, когда преобразование очищается.                                                                     |
| [**\_ИД объекта топоноде для \_ преобразования MF \_**](mf-toponode-transform-objectid-attribute.md) | Идентификатор класса (CLSID) преобразования, связанного с этим узлом топологии.                          |



 

## <a name="output-node-attributes"></a>Атрибуты выходного узла



| attribute                                                                                  | Описание                                                                                                      |
|--------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**MF \_ топоноде \_ Отключить \_ преднакат**](mf-toponode-disable-preroll-attribute.md)            | Указывает, использует ли сеанс мультимедиа преднакат приемника носителя, представленного этим узлом топологии.            |
| [**MF \_ ТОПОНОДЕ \_ Shutdown \_ On \_ Remove**](mf-toponode-noshutdown-on-remove-attribute.md) | Указывает, завершает ли сеанс мультимедиа приемник мультимедиа при удалении выходного узла из топологии. |
| [**\_топоноде со \_ скоростью MF**](mf-toponode-rateless-attribute.md)                           | Указывает, является ли приемник носителя, связанный с этим узлом топологии, недоступным.                                 |
| [**MF \_ топоноде \_ STREAMID**](mf-toponode-streamid-attribute.md)                           | Идентификатор потока приемника потока, связанного с этим узлом топологии.                                     |
