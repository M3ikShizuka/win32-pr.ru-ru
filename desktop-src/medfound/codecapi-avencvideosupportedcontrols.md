# <a name="codecapi_avencvideosupportedcontrols-property"></a>КОДЕКАПИ \_ авенквидеосуппортедконтролс, свойство

Задает поддерживаемые элементы управления для кодировщика видео.

## <a name="data-type"></a>Тип данных

**UINT64 (VT \_ UI8)**

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеосуппортедконтролс**

## <a name="remarks"></a>Комментарии

Это свойство используется только с [кодировщиками камер H. 264 увк 1,5](camera-encoder-h264-uvc-1-5.md).
