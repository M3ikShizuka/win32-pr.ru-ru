# <a name="mfsampleextension_bottomfieldfirst-attribute"></a>Мфсампликстенсион \_ боттомфиелдфирст, атрибут

Задает поле превосходство для чередующихся видеокадров. Этот атрибут применяется к примерам мультимедиа.

## <a name="data-type"></a>Тип данных

**Bool** , сохраненный как **UINT32**

## <a name="getset"></a>Get/Set

Чтобы получить этот атрибут, вызовите метод [**имфаттрибутес:: UInt32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-getuint32).

Чтобы задать этот атрибут, вызовите [**имфаттрибутес:: SetUINT32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-setuint32).

## <a name="applies-to"></a>Применяется к

[**имфсампле**](/windows/desktop/api/mfobjects/nn-mfobjects-imfsample)

## <a name="remarks"></a>Комментарии

Если видеокадр имеет чередование, а образец содержит два поля с чередованием, этот атрибут указывает, какое поле отображается первым. Если **значение равно true**, то нижнее поле является первым по времени. Если **значение равно false**, верхнее поле является первым.

Если кадр является чередованием, а образец содержит одно поле, этот атрибут указывает, какое поле в образце содержит. Если **значение — true**, образец содержит нижнее поле. Если **значение равно false**, образец содержит верхнее поле.

Если кадр является прогрессивным, этот атрибут описывает порядок упорядочивания полей при чередовании выходных данных. Если **значение равно true**, то нижнее поле должно выводиться первыми. Если **значение равно false**, верхнее поле должно быть выведено первым.

Если этот атрибут не задан, тип носителя описывает поле превосходство.

Константа GUID для этого атрибута экспортируется из мфууид. lib.
