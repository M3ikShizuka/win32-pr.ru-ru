# <a name="mfnetsource_ppbandwidth-property"></a>МФНЕТСАУРЦЕ \_ ппбандвидс, свойство

Указывает пропускную способность для пары пакетов и полосу пропускания во время выполнения, обнаруженную сетевым источником. Значение свойства представляет собой массив из двух значений:

-   Пропускная способность пары пакетов в битах в секунду.
-   Пропускная способность во время выполнения в битах в секунду.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

Массив значений типа **ulong** (**Каул**)

VT \_ вектор \| VT \_ UI4

**каул**



## <a name="remarks"></a>Remarks

Константа **мфнетсаурце \_ ппбандвидс** определяет идентификатор GUID для этого ключа свойства. Идентификатор свойства (PID) равен нулю.

Это свойство доступно только для чтения. Чтобы получить это свойство, выполните запрос к источнику сети для интерфейса **ипропертисторе** и вызовите метод **Ипропертисторе:: GetValue**.
