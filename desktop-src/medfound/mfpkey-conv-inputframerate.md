# <a name="mfpkey_conv_inputframerate-property"></a>МФПКЭЙ \_ , \_ Инпутфрамерате, свойство

Указывает частоту входного кадра в виде коэффициента.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

Доступно только с помощью [**ипропертисторе**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Тип данных

VT \_ UI8

## <a name="applies-to"></a>Применение

-   [Преобразователь частоты кадров](framerateconverter.md)

## <a name="remarks"></a>Комментарии

Сохраните числитель в более высоких 4 байтах и знаменатель в младших 4 байтах.
