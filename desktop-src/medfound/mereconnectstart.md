# <a name="mereconnectstart-event"></a>Событие Мереконнектстарт

Сигнализирует, что источник мультимедиа пытается повторно подключиться к серверу.

Вызвано медиа-источником в начале попытки повторного подключения. Источник сети создает это событие, если пытается повторно подключиться к серверу. Сеанс мультимедиа перенаправляет это событие в приложение.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |
