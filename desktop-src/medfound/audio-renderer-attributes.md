# <a name="audio-renderer-attributes"></a>Атрибуты модуля подготовки звука

Для настройки модуля подготовки звука можно использовать следующие атрибуты.



| attribute                                                                                                     | Описание                                                                                                 |
|---------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**\_ \_ \_ \_ идентификатор конечной точки атрибута модуля подготовки звука MF \_**](mf-audio-renderer-attribute-endpoint-id-attribute.md)     | Указывает идентификатор для устройства конечной точки аудио.                                                     |
| [**\_ \_ \_ \_ роль конечной точки атрибута модуля подготовки звука MF \_**](mf-audio-renderer-attribute-endpoint-role-attribute.md) | Указывает роль конечной точки аудио для модуля подготовки звука.                                                   |
| [**\_ \_ \_ Флаги атрибута модуля подготовки \_ звука MF**](mf-audio-renderer-attribute-flags-attribute.md)                  | Содержит флаги для настройки модуля подготовки звука.                                                             |
| [**\_ \_ \_ \_ ИД сеанса атрибута модуля подготовки звука \_ MF**](mf-audio-renderer-attribute-session-id-attribute.md)       | Указывает класс политики аудио для модуля подготовки звука.                                                    |
| [\_ \_ \_ \_ Категория потока АТРИБУТОВ модуля подготовки звука \_ MF](mf-audio-renderer-attribute-stream-category.md)           | Указывает категорию звукового потока для модуля [подготовки потоковой передачи звука](streaming-audio-renderer.md) (САР). |
