# <a name="mfpkey_videowindow-property"></a>МФПКЭЙ \_ видеовиндов, свойство

Указывает объем содержимого в миллисекундах, который может поместиться в буфер модели.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвквидеовиндов

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

3000

## <a name="remarks"></a>Комментарии

Окно буфера — это один из параметров модели «сегмент утечки», используемый для управления скоростью кодека.
