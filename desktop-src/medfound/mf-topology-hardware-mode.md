# <a name="mf_topology_hardware_mode-attribute"></a>\_ \_ Атрибут режима оборудования топологии MF \_

Указывает, следует ли загружать в топологии Microsoft Media Foundation преобразования на основе оборудования (МФТС).

## <a name="data-type"></a>Тип данных

**[**Мфтопологи \_ \_Режим оборудования**](/windows/desktop/api/mfidl/ne-mfidl-mftopology_hardware_mode)** хранится как **UINT32**

## <a name="getset"></a>Get/Set

Чтобы получить этот атрибут, вызовите метод [**имфаттрибутес:: UInt32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-getuint32).

Чтобы задать этот атрибут, вызовите [**имфаттрибутес:: SetUINT32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-setuint32).

## <a name="applies-to"></a>Применяется к

[**имфтопологи**](/windows/desktop/api/mfidl/nn-mfidl-imftopology)

## <a name="remarks"></a>Remarks

Этот атрибут является необязательным. Задайте атрибут перед разрешением топологии.



| Значение                                  | Описание                                                                                                                                                                                                                                                                       |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **МФТОПОЛОГИ \_ хвмоде \_ использование \_ оборудования**  | Загрузчик топологии будет загружать аппаратные МФТС, такие как аппаратные декодеры, если они доступны.<br/> Загрузчик топологии автоматически возвращается к программному декодированию, если аппаратный декодер не найден или если аппаратный декодер не может подключиться по какой-либо причине.<br/> |
| **\_ \_ только программное обеспечение мфтопологи хвмоде \_** | Загрузчик топологии будет загружать только МФТС программного обеспечения, включая программные декодеры.                                                                                                                                                                                                    |



 

Значение по умолчанию — **мфтопологи \_ хвмоде \_ Software \_ только** для совместимости с существующими приложениями. Рекомендуемое значение — **мфтопологи \_ хвмоде \_ использовать \_ оборудование**.

Если загрузчик топологии вставляет в топологию основную таблицу оборудования, в узле Topology задается атрибут [ \_ \_ \_ \_ атрибута «URL-адрес перечисления MFT](mft-enum-hardware-url-attribute.md) ». Чтобы проверить наличие в файле MFT оборудования, перечислите узлы в разрешенной топологии и проверьте наличие этого атрибута.

Константа GUID для этого атрибута экспортируется из мфууид. lib.
