# <a name="mf_mt_h264_max_mb_per_sec-attribute"></a>\_ \_ \_ Атрибут с максимальным числом \_ МБ \_ в \_ секунду для H264 Single bitrate MF MT

Указывает максимальную скорость обработки макроблокк для потока видео H. 264.

## <a name="data-type"></a>Тип данных

**\[ UINT32 \]** сохраняется как **UINT8 \[ \]**

## <a name="applies-to"></a>Применяется к

[**имфмедиатипе**](/windows/desktop/api/mfobjects/nn-mfobjects-imfmediatype)

## <a name="remarks"></a>Комментарии

Этот атрибут применяется к типам носителей для потоков H. 264, переданных по USB. Значение атрибута является массивом значений UINT32, которые соответствуют следующим полям в дескрипторе формата видео УВК 1,5 H. 264.

| Элемент массива | Поле дескриптора                                           |
|---------------|------------------------------------------------------------|
| 0             | **двмаксмбперсеконересолутионноскалабилити**                |
| 1             | **двмаксмбперсектворесолутионсноскалабилити**               |
| 2             | **двмаксмбперсексриресолутионсноскалабилити**             |
| 3             | **двмаксмбперсекфаурресолутионсноскалабилити**              |
| 4             | **двмаксмбперсеконересолутионтемпоралскалабилити**          |
| 5             | **двмаксмбперсектворесолутионстемпоралскалаблилити**        |
| 6             | **двмаксмбперсексриресолутионстемпоралскалабилити**       |
| 7             | **двмаксмбперсекфаурресолутионстемпоралскалабилити**        |
| 8             | **двмаксмбперсеконересолутионтемпоралкуалитискалабилити**   |
| 9             | **двмаксмбперсектворесолутионстемпоралкуалитискалабилити**  |
| 10            | **двмаксмбперсексриресолутионстемпоралкуалитискалаблити** |
| 11            | **двмаксмбперсекфаурресолутионстемпоралкуалитискалабилити** |
| 12            | **двмаксмбперсеконересолутионфуллскалабилити**              |
| 13            | **двмаксмбперсектворесолутионсфуллскалабилити**             |
| 14            | **двмаксмбперсексриресолутионсфуллскалабилити**           |
| 15            | **двмаксмбперсекфаурресолутионсфуллскалабилити**            |



 

Этот атрибут также используется с [кодировщиками камер H. 264 увк 1,5](camera-encoder-h264-uvc-1-5.md).
