# <a name="connecting-and-disconnecting-topology-nodes"></a>Подключение и отключение узлов топологии

Чтобы топология была функциональной, необходимо подключить исходный узел и выходной узел. Чтобы подключить два узла топологии, перетащите выходные данные узла одного узла в входные данные узла другого узла. Топоедит отображает соединение узла в виде черной линии. Это эквивалентно соединению узлов топологии путем вызова метода [**имфтопологиноде:: коннектаутпут**](/windows/desktop/api/mfidl/nf-mfidl-imftopologynode-connectoutput) .

Топологию можно разрешить только в том случае, если соединение с узлом является допустимым. то есть, если типы носителей двух узлов совместимы. Сведения об устранении топологий см. в разделе [разрешение топологии с помощью топоедит](resolving-a-topology-with-topoedit.md).

При попытке установить соединение с уже подключенным узлом подключение нового узла заменяет старое соединение с узлом.

Чтобы удалить подключение, выберите соединение с узлом и нажмите клавишу DELETE или выберите **Удалить** в меню **топология** .
