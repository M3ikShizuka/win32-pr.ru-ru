# <a name="adding-source-nodes-with-topoedit"></a>Добавление исходных узлов с помощью Топоедит

Исходный узел представляет поток в файле мультимедиа. Чтобы создать исходный узел в Топоедит, укажите файл мультимедиа. Топоедит перечисляет потоки в файле и создает соответствующие исходные узлы.

Сведения о добавлении исходных узлов программным путем с помощью Media Foundation API-интерфейсов см. в разделе [Создание исходных узлов](creating-source-nodes.md).

## <a name="to-add-a-source-node-to-a-topology"></a>Добавление исходного узла в топологию

1.  В меню **топология** выберите команду **Добавить источник**.

    Откроется диалоговое окно **Выбор источника носителя** .

2.  Перейдите к папке, в которой находится файл мультимедиа.

3.  В поле **имя файла:** введите имя файла.

4.  Нажмите кнопку **Открыть**.

Топоедит создает исходные узлы для потоков. На **панели топология** отображается исходный узел, содержащийся в сером поле, в котором отображается имя файла мультимедиа. Исходный узел показывает тип потока узла.
