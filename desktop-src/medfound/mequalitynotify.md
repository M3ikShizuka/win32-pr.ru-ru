# <a name="mequalitynotify-event"></a>Событие Мекуалитинотифи

Предоставляет отзыв диспетчеру качества о качестве качества воспроизведения.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE           | Описание                         |
|-------------------|-------------------------------------|
| VT \_ i8<br/> | См. заметки.<br/> <br/> |



## <a name="remarks"></a>Remarks

Это событие вызывается некоторыми компонентами конвейера. Сеанс мультимедиа перенаправляет событие диспетчеру качества, вызывая метод [**имфкуалитиманажер:: нотификуалитевент**](/windows/desktop/api/mfidl/nf-mfidl-imfqualitymanager-notifyqualityevent) .

Расширенный тип события указывает значение данных события.



| Расширенный тип                            | Данные событий                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_ \_ уведомление о \_ задержке при обработке уведомления MF \_ | Приблизительная задержка обработки, представленная компонентом, в единицах 100-наносекундных.<br/> Задержка обработки — это количество задержки, которое компонент вводит в конвейер путем обработки образца. В некоторых случаях задержка не может быть получена просто путем просмотра вызовов [**имфкуалитиманажер:: нотифипроцессинпут**](/windows/desktop/api/mfidl/nf-mfidl-imfqualitymanager-notifyprocessinput) и [**Имфкуалитиманажер:: нотифипроцессаутпут**](/windows/desktop/api/mfidl/nf-mfidl-imfqualitymanager-notifyprocessoutput). Например, может не быть однозначного соответствия между входными и выходными примерами. В этом случае компонент может отправить событие Мекуалитинотифи с задержкой обработки. При изменении задержки обработки компонент может отправить новое событие в любое время во время потоковой передачи.<br/> |
| \_ \_ \_ Задержка ВЫБОРки уведомлений \_ MF         | Время запаздывания для примера в единицах измерения 100-наносекундных. Если значение положительное, пример был последним. Если значение отрицательное, пример был ранним.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |



 

Чтобы получить расширенный тип, вызовите метод [**имфмедиаевент:: жетекстендедтипе**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getextendedtype).

Компоненты конвейера не являются обязательными для отправки этого события.
