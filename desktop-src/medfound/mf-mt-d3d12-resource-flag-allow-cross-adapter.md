# <a name="mf_mt_d3d12_resource_flag_allow_cross_adapter-attribute"></a>\_ \_ Флаг ресурса MF MT D3D12 \_ \_ \_ Разрешить \_ атрибут перекрестного \_ адаптера

Указывает, могут ли ресурсы в потоке использоваться для данных перекрестных адаптеров.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Комментарии

Значение этого атрибута соответствует элементу D3D12_RESOURCE_FLAG_ALLOW_CROSS_ADAPTER перечисления [D3D12_RESOURCE_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_resource_flags) .
