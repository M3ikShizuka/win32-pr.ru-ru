# <a name="table-of-contents-parser-interfaces"></a>Интерфейсы средства синтаксического анализа оглавления

Следующие интерфейсы доказали доступ к службам средства синтаксического анализа оглавления.



| Интерфейс                                | Описание                                                                                                           |
|------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**иток**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itoc)                     | Представляет отдельное оглавление.                                                                           |
| [**итокколлектион**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itoccollection) | Представляет коллекцию таблиц содержимого.                                                                        |
| [**итоцентри**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocentry)           | Представляет отдельную запись в содержании.                                                                |
| [**итоцентрилист**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocentrylist)   | Представляет список записей в содержании.                                                                  |
| [**итокпарсер**](/windows/desktop/api/wmcodecdsp/nn-wmcodecdsp-itocparser)         | Предоставляет методы для внедрения оглавления в видеофайл и чтения оглавления из видеофайла. |
