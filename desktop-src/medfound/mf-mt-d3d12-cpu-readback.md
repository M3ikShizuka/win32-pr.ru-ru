# <a name="mf_mt_d3d12_cpu_readback-attribute"></a>\_ \_ \_ Атрибут реадбакк процессора MF MT D3D12 \_

Указывает, требуется ли доступ к ЦП для связанных ресурсов Direct3D. 

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Комментарии

Значение этого атрибута по умолчанию равно 0 (FALSE).
