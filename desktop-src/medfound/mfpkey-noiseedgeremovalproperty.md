# <a name="mfpkey_noiseedgeremoval-property"></a>МФПКЭЙ \_ ноисиджеремовал, свойство

Указывает, должен ли кодек попытаться обнаружить шумы в кадрах и удалить их.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкноисиджеремовал

## <a name="data-type"></a>Тип данных

Логическое значение VT \_

## <a name="default-value"></a>Значение по умолчанию

FALSE

## <a name="remarks"></a>Remarks

Шумы в кадре обычно являются вертикальными пустыми интервалами (ВБИ) из кадра телевизионных телепередач. ВБИ — это первые 21 строки просмотра кадра телевизора. Эти строки просмотра не содержат данных видео — они содержат данные о широковещательной рассылке. Когда на карте записи записывается ТВ-сигнал, ВБИ обычно удаляется из рамки. Функция обнаружения и исправления шумового периметра, выполняемая кодеком, может исправить только ребро с тремя или менее строками шума. Если записанное видео содержит более трех бесшумных линий, возникает проблема с оборудованием, используемым для записи видео.

Если кодек настроен на удаление шума, он дублирует строки, смежные с шумным ребром, для заполнения рамки.
