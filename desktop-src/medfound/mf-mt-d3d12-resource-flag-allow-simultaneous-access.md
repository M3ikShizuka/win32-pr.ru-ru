# <a name="mf_mt_d3d12_resource_flag_allow_simultaneous_access-attribute"></a>\_ \_ Флаг ресурса MF MT D3D12 \_ \_ \_ Разрешить \_ одновременный \_ доступ к атрибуту

Указывает, могут ли ресурсы Direct3D в потоке быть доступны одновременно нескольким различным командным очередям.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

Значение этого атрибута соответствует элементу D3D12_RESOURCE_FLAG_ALLOW_SIMULTANEOUS_ACCESS перечисления [D3D12_RESOURCE_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_resource_flags) .
