# <a name="mestreamsinkscrubsamplecomplete-event"></a>Событие Местреамсинкскрубсамплекомплете

Вызывается приемником потока при завершении запроса на очистку.

Очистка происходит, когда скорость воспроизведения равна нулю и часы презентации запускаются с указанным сруббинг временем. Если приемник мультимедиа поддерживает очистку, каждый поток в приемнике вызывает это событие при вызове метода [**имфклоккстатесинк:: онклоккстарт**](/windows/desktop/api/mfidl/nf-mfidl-imfclockstatesink-onclockstart) , когда скорость воспроизведения равна нулю.

Если поток визуализирует данные при очистке, он отправляет событие сразу после подготовки данных. Если поток не отображает данные, он отправляет событие сразу после вызова [**онклоккстарт**](/windows/desktop/api/mfidl/nf-mfidl-imfclockstatesink-onclockstart) .

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |



## <a name="attributes"></a>Атрибуты

Для этого события определены следующие атрибуты.



| attribute                                                                              | Описание                                                                                                                                                   |
|----------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ время СКРУБСАМПЛЕ события \_ MF**](mf-event-scrubsample-time-attribute.md)<br/> | Время презентации, для которой были подготовлены данные. Если приемник мультимедиа не отображает данные при очистке, он не устанавливает этот атрибут.<br/> <br/> |
