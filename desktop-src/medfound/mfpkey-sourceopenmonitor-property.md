# <a name="mfpkey_sourceopenmonitor-property"></a>МФПКЭЙ \_ саурцеопенмонитор, свойство

Содержит указатель на интерфейс [**имфсаурцеопенмонитор**](/windows/desktop/api/mfidl/nn-mfidl-imfsourceopenmonitor) приложения.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

Указатель **IUnknown**

VT \_ Unknown

**пунквал**



## <a name="remarks"></a>Remarks

Приложения могут передавать это свойство в сопоставитель источника для получения уведомлений о событиях из источника сети.
