# <a name="mepolicyerror-event"></a>Событие Меполициеррор

Вызывается надежным выходом при возникновении ошибки при принудительном применении политики вывода.

Если сеанс мультимедиа получает это событие, он останавливает воспроизведение и перенаправляет событие в приложение.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |



## <a name="remarks"></a>Remarks

Возможные значения, получаемые из [**имфмедиаевент:: Status**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getstatus) , включают следующее.



| Значение                      | Описание                                                    |
|----------------------------|----------------------------------------------------------------|
| \_Политика MF \_ E \_ не поддерживается | Доверенный выход не поддерживает текущую политику вывода. |
