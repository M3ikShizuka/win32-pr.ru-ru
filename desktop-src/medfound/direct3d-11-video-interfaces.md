# <a name="direct3d-11-video-interfaces"></a>Интерфейсы видео Direct3D 11

В этом разделе содержатся справочные сведения по интерфейсам видео Microsoft Direct3D 11.

| Раздел                                                                                 | Описание                                                                                              |
|---------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [**ID3D11AuthenticatedChannel**](/windows/desktop/api/d3d11/nn-d3d11-id3d11authenticatedchannel)<br/>           | Предоставляет коммуникационный канал с графическим драйвером или средой выполнения Microsoft Direct3D. <br/> |
| [**ID3D11CryptoSession**](/windows/desktop/api/d3d11/nn-d3d11-id3d11cryptosession)<br/>                         | Представляет сеанс шифрования. <br/>                                                          |
| [**ID3D11VideoContext**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videocontext)<br/>                           | Предоставляет функции видео устройства Direct3D 11. <br/>                                    |
| [**ID3D11VideoContext1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videocontext1)<br/>                         | Предоставляет функции видео устройства Direct3D 11.<br/>                                     |
| [**ID3D11VideoContext2**](/windows/win32/api/d3d11_4/nn-d3d11_4-id3d11videocontext2)<br/>                         | Предоставляет функции видео устройства Direct3D 11. <br/>                                    |
| [**ID3D11VideoDecoder**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodecoder)<br/>                           | Представляет декодер видео с аппаратным ускорением для Direct3D 11.<br/>                              |
| [**ID3D11VideoDecoderOutputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodecoderoutputview)<br/>       | Определяет выходные поверхности, к которым можно получить доступ во время декодирования видео.<br/>                    |
| [**ID3D11VideoDevice**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videodevice)<br/>                             | Предоставляет возможности декодирования видео и обработки видео на устройстве Direct3D 11. <br/>       |
| [**ID3D11VideoDevice1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videodevice1)<br/>                           | Предоставляет возможности декодирования видео и обработки видео на устройстве Direct3D 11. <br/>       |
| [**ID3D11VideoProcessor**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessor)<br/>                       | Представляет обработчик видео для Direct3D 11.<br/>                                                 |
| [**ID3D11VideoProcessorEnumerator**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessorenumerator)<br/>   | Перечисляет возможности видеопроцессора устройства Direct3D 11. <br/>                         |
| [**ID3D11VideoProcessorEnumerator1**](/windows/desktop/api/d3d11_1/nn-d3d11_1-id3d11videoprocessorenumerator1)<br/> | Перечисляет возможности видеопроцессора устройства Direct3D 11.<br/>                          |
| [**ID3D11VideoProcessorInputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessorinputview)<br/>     | Определяет поверхности ввода, к которым можно получить доступ во время обработки видео.<br/>                   |
| [**ID3D11VideoProcessorOutputView**](/windows/desktop/api/d3d11/nn-d3d11-id3d11videoprocessoroutputview)<br/>   | Определяет выходные поверхности, к которым можно получить доступ во время обработки видео.<br/>                  |
