# <a name="mesourcecharacteristicschanged-event"></a>Событие Месаурцечарактеристиксчанжед

Вызывается источником мультимедиа при изменении характеристик источника.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |



## <a name="attributes"></a>Атрибуты

Для этого события определены следующие атрибуты.



| attribute                                                                                                   | Описание                                                          |
|-------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------|
| [**\_ \_ характеристики источника события \_ MF**](mf-event-source-characteristics-attribute.md)<br/>          | Новые характеристики источника мультимедиа.<br/> <br/>      |
| [**\_ \_ \_ старые характеристики источника события \_ MF**](mf-event-source-characteristics-old-attribute.md)<br/> | Предыдущие характеристики источника мультимедиа.<br/> <br/> |
