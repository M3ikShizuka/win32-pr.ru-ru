# <a name="file-menu"></a>Меню «Файл»

Меню **файл** содержит следующие команды:



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Get-Help</th>
<th>Описание</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Новые функции...</strong></td>
<td>Очищает существующую топологию и удаляет узлы топологии с <strong>панели топология</strong>.<br/>
<ul>
<li>Если топология в данный момент открыта, предлагает пользователю сохранить топологию. Сведения о сохранении топологии см. в разделе <a href="saving-and-loading-topologies.md">Сохранение и загрузка топологий</a>.<br/></li>
<li>Создает пустую топологию. Пользователь может добавить узлы топологии в новую топологию. Дополнительные сведения см. в разделе <a href="building-topologies-by-using-topoedit.md">Создание топологий с помощью топоедит</a>.<br/></li>
</ul></td>
</tr>
<tr class="even">
<td><strong>Файл отрисовки...</strong></td>
<td>Создает топологию воспроизведения для файла мультимедиа. Топоедит автоматически создает и подключает узлы в топологии. Дополнительные сведения см. в статье <a href="building-a-playback-topology-with-topoedit.md">Создание топологии воспроизведения с помощью топоедит</a>.</td>
</tr>
<tr class="odd">
<td><strong>Обработка преобразования кода...</strong></td>
<td>Создает топологию перекодирования. Дополнительные сведения см. в разделе <a href="building-a-transcode-topology-with-topoedit.md">Создание топологии перекодирования с помощью топоедит</a>.<br/>
<blockquote>
[!Note]<br />
Для этого компонента требуется Windows 7.
</blockquote>
<br/></td>
</tr>
<tr class="even">
<td><strong>Открыть...</strong></td>
<td>Загружает существующую топологию, сохраненную в виде XML-файла. См. раздел <a href="saving-and-loading-topologies.md">Сохранение и загрузка топологий</a>.</td>
</tr>
<tr class="odd">
<td><strong>Сохранить...</strong></td>
<td>Сохраняет открытую в данный момент топологию в виде XML-файла. См. раздел <a href="saving-and-loading-topologies.md">Сохранение и загрузка топологий</a>.</td>
</tr>
<tr class="even">
<td><strong>Выход</strong></td>
<td>Закрывает приложение Топоедит.</td>
</tr>
</tbody>
</table>
