# <a name="file-menu"></a>Меню «Файл»

Меню **файл** содержит следующие команды:




| Команда | Описание | 
|---------|-------------|
| <strong>Новые функции...</strong> | Очищает существующую топологию и удаляет узлы топологии с <strong>панели топология</strong>.<br /><ul><li>Если топология в данный момент открыта, предлагает пользователю сохранить топологию. Сведения о сохранении топологии см. в разделе <a href="saving-and-loading-topologies.md">Сохранение и загрузка топологий</a>.<br /></li><li>Создает пустую топологию. Пользователь может добавить узлы топологии в новую топологию. Дополнительные сведения см. в разделе <a href="building-topologies-by-using-topoedit.md">Создание топологий с помощью топоедит</a>.<br /></li></ul> | 
| <strong>Файл отрисовки...</strong> | Создает топологию воспроизведения для файла мультимедиа. Топоедит автоматически создает и подключает узлы в топологии. Дополнительные сведения см. в статье <a href="building-a-playback-topology-with-topoedit.md">Создание топологии воспроизведения с помощью топоедит</a>. | 
| <strong>Обработка преобразования кода...</strong> | Создает топологию перекодирования. Дополнительные сведения см. в разделе <a href="building-a-transcode-topology-with-topoedit.md">Создание топологии перекодирования с помощью топоедит</a>.<br /><blockquote>[!Note]<br />для этой функции требуется Windows 7.</blockquote><br /> | 
| <strong>Открыть...</strong> | Загружает существующую топологию, сохраненную в виде XML-файла. См. раздел <a href="saving-and-loading-topologies.md">Сохранение и загрузка топологий</a>. | 
| <strong>Сохранить...</strong> | Сохраняет открытую в данный момент топологию в виде XML-файла. См. раздел <a href="saving-and-loading-topologies.md">Сохранение и загрузка топологий</a>. | 
| <strong>Выход</strong> | Закрывает приложение Топоедит. |
