# <a name="media-types-media-foundation"></a>Типы носителей (Media Foundation)

*Тип мультимедиа* — это способ описания формата потока мультимедиа. В Media Foundation типы мультимедиа представлены интерфейсом [**имфмедиатипе**](/windows/desktop/api/mfobjects/nn-mfobjects-imfmediatype) . Приложения используют типы мультимедиа для обнаружения формата файла мультимедиа или потока мультимедиа. Объекты в конвейере Media Foundation используют типы мультимедиа для согласования форматов, которые они будут доставлять или получать.
