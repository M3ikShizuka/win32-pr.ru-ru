# <a name="mf_mt_d3d12_resource_flag_allow_unordered_access-attribute"></a>\_ \_ Флаг ресурса MF MT D3D12 \_ \_ \_ Разрешить \_ неупорядоченный \_ атрибут доступа

Указывает, можно ли создать неупорядоченное представление доступа для ресурсов Direct3D в потоке, связанном с типом носителя. 

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

Значение этого атрибута соответствует элементу D3D12_RESOURCE_FLAG_ALLOW_UNORDERED_ACCESS перечисления [D3D12_RESOURCE_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_resource_flags) .
