# <a name="mepolicyreport-event"></a>Событие Меполицирепорт

Вызывается доверенным выходом для отправки сведений о состоянии применения политики вывода.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |



## <a name="remarks"></a>Remarks

Атрибуты и коды состояния для этого события зависят от конкретной системы защиты содержимого, которую обеспечивает доверенный выход.
