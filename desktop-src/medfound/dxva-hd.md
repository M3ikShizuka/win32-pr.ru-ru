# <a name="dxva-hd"></a>ДКСВА-HD

High Definition (ДКСВА-HD) ускорение видео Microsoft DirectX — это API для обработки видео с аппаратным ускорением. ДКСВА-HD использует GPU для выполнения таких функций, как декомпозиция, композиция и преобразование цветового пространства.

ДКСВА-HD аналогичен [дксваной обработке видео](dxva-video-processing.md) (ДКСВА-президент), но предлагает расширенные возможности и более простую модель обработки. Предоставляя более гибкую модель композиции, ДКСВА-HD обеспечивает поддержку нового поколения HD оптических форматов и стандартов вещания.

API ДКСВА-HD требует наличия драйвера экрана WDDM, поддерживающего интерфейс драйвера устройства ДКСВА-HD (DDI) или программного процессора подключаемого модуля.

## <a name="improvements-over-dxva-vp"></a>Улучшения по сравнению с ДКСВА — вице-президент

ДКСВА-HD расширяет набор функций, предоставляемых ДКСВА-президента. К усовершенствованиям относятся:

-   Смешение RGB и YUV. Любой поток может быть либо RGB, либо YUV. Между основным потоком и подпотоками больше нет различий.
-   Разчередование нескольких потоков. Любой поток может быть либо прогрессивным, либо чередованием. Кроме того, ритмичность и частота кадров могут меняться от одного потока ввода к другому.
-   Цвета фона RGB. Ранее поддерживались только цвета фона YUV.
-   Ключ яркости. Если ключ яркости включен, значения яркости, которые попадают в указанный диапазон, становятся прозрачными.
-   Динамическое переключение между режимами с разчередованием.

ДКСВА-HD также определяет некоторые дополнительные функции, которые могут поддерживаться драйверами. Однако приложения не должны рассчитывать, что все драйверы будут поддерживать эти функции. К дополнительным функциям относятся:

-   Обратный чересстрочности (например, 60i в 24p).
-   Преобразование ставок кадров (например, 24p в 120p).
-   Режимы заливки альфа-канала.
-   Фильтрация шумов и расширения границы.
-   Анаморфик нелинейное масштабирование.
-   Расширенная Икбкр (Ксвикк).
