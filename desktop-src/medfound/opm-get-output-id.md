# <a name="opm_get_output_id"></a>ОПМ \_ получить \_ \_ идентификатор выхода

Возвращает уникальный идентификатор монитора, связанного с этим выходным видео.



| Требование | Значение |
|--------------|------------------------------------------------------------------|
| GUID запроса | ОПМ \_ получить \_ \_ идентификатор выхода                                             |
| Входные данные   | None                                                             |
| Возвращать данные  | Структура [**\_ \_ \_ данных идентификатора выхода ОПМ**](/windows/desktop/api/opmapi/ns-opmapi-opm_output_id_data) |



 

## <a name="remarks"></a>Remarks

Драйвер видеоадаптера назначает уникальный идентификатор каждому монитору. Этот запрос возвращает идентификатор монитора, связанного с текущим указателем [**иопмвидеуутпут**](/windows/desktop/api/opmapi/nn-opmapi-iopmvideooutput) .
