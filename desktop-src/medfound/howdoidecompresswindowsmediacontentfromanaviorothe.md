без сведений о форматировании, включая расширенные данные формата, нельзя распаковать данные, закодированные с помощью кодека Windows Media. Кодеки созданы для создания потоков данных для использования с контейнером файлов ASF. В разделе заголовка файла ASF хранятся сведения о формате для всех потоков в файле. при использовании содержимого, закодированного с помощью одного из кодеков Windows Media Audio и Video, за пределами ASF-файла, необходимо убедиться, что сведения о формате доступны.
