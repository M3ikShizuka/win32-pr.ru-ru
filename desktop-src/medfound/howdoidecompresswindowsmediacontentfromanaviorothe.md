# <a name="how-do-i-decompress-streams-from-an-avi-file-or-other-file-about-which-i-have-no-format-information"></a>Разделы справки распаковать потоки из файла AVI или другого файла, о котором у меня нет сведений о форматировании?

Без сведений о форматировании, включая Расширенные данные формата, нельзя распаковать данные, закодированные с помощью кодека Windows Media. Кодеки созданы для создания потоков данных для использования с контейнером файлов ASF. В разделе заголовка файла ASF хранятся сведения о формате для всех потоков в файле. При использовании содержимого, закодированного с помощью одного из кодеков аудио и видео Windows Media, за пределами ASF-файла, необходимо убедиться, что сведения о формате доступны. Способ сделать это может быть разным в разных контейнерах.
