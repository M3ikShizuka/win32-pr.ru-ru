# <a name="decoding-to-spdif-microsoft-media-foundation"></a>Декодирование в S/PDIF (Microsoft Media Foundation)

Идентификатор класса для объекта декодера, который преобразует содержимое Windows Media Audio 9,1 Professional для доставки по формату Digital Interconnect (S/PDIF) в формате "Sony/Philips", — это CLSID \_ квмаудиоспдтксдмо. Этот объект используется так же, как любой другой декодер. Он поддерживает те же свойства, что и декодер Windows Media Audio 9. Типы выходных данных, перечисленные этим декодером, являются форматами S/PDIF.

Для полной поддержки выходных данных S/PDIF на клиентском компьютере должен быть установлен специальный аппаратный компьютер.
