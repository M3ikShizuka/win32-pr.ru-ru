# <a name="simplecapture-sample"></a>Пример Симплекаптуре

\[Не рекомендуется. API Мфплай может быть удален из будущих выпусков Windows. Приложения должны использовать [модуль чтения исходного кода](source-reader.md) для записи видео.\]

Демонстрация предварительного просмотра видео с устройства видеозаписи с помощью API Мфплай.

## <a name="apis-demonstrated"></a>Продемонстрированные API

В этом образце демонстрируются следующие интерфейсы Microsoft Media Foundation:

-   [**имфмедиасаурце**](/windows/desktop/api/mfidl/nn-mfidl-imfmediasource)
-   [**имфпмедиаитем**](/windows/desktop/api/mfplay/nn-mfplay-imfpmediaitem)
-   [**имфпмедиаплайер**](/windows/desktop/api/mfplay/nn-mfplay-imfpmediaplayer)
-   [**имфпмедиаплайеркаллбакк**](/windows/desktop/api/mfplay/nn-mfplay-imfpmediaplayercallback)
