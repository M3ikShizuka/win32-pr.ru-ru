# <a name="saving-and-loading-topologies"></a>Сохранение и загрузка топологий

С помощью Топоедит можно сохранить топологию в виде XML-файла и загрузить ее позже. После загрузки полной топологии ее необходимо устранить снова. Сведения об этой процедуре см. [в разделе Разрешение топологии с помощью топоедит](resolving-a-topology-with-topoedit.md).

## <a name="to-save-a-topology"></a>Сохранение топологии

1.  В меню **File** (Файл) выберите пункт **Save** (Сохранить).

    Откроется диалоговое окно **сохранить файл** .

2.  Перейдите в папку, в которой должен быть сохранен файл топологии.

3.  В поле **имя файла:** введите имя файла.

4.  Выберите команду **Сохранить**.

Файл топологии будет сохранен в виде XML-файла.

## <a name="to-load-a-saved-topology"></a>Загрузка сохраненной топологии

1.  В меню **Файл** выберите **Открыть**.

    Откроется диалоговое окно **Открытие файла** .

2.  Перейдите в папку, где расположен XML-файл топологии.

3.  Укажите имя файла в поле **имя файла:** или выберите файл из списка.

4.  Нажмите кнопку **Открыть**.

Файл топологии загружается на **панели топология**.
