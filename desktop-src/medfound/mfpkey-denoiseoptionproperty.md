# <a name="mfpkey_denoiseoption-property"></a>МФПКЭЙ \_ деноисеоптион, свойство

Указывает, будет ли кодек использовать фильтр шума при кодировании.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкденоисеоптион

## <a name="data-type"></a>Тип данных

Логическое значение VT \_

## <a name="default-value"></a>Значение по умолчанию

FALSE

## <a name="remarks"></a>Remarks

Фильтр шума может улучшить качество звуковых источников, таких как фильм, содержащий много граней или снимка видео, на низком свете. Этот фильтр можно использовать в сценариях кодирования в реальном времени, когда внешняя Предварительная обработка не является вариантом.

Этот фильтр должен быть отключен для чистых видеороликов, так как он может уменьшить качество изображения, когда качество подходит для начала.
