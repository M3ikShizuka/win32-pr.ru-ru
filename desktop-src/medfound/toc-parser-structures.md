Синтаксический анализатор содержимого использует следующие структуры.



| Структура                                              | Описание                                                        |
|--------------------------------------------------------|--------------------------------------------------------------------|
| [**Дескриптор ОГЛАВЛЕНИя \_**](/windows/desktop/api/wmcodecdsp/ns-wmcodecdsp-toc_descriptor)              | Содержит описательные сведения для оглавления.             |
| [**\_дескриптор записи оглавления \_**](/windows/desktop/api/wmcodecdsp/ns-wmcodecdsp-toc_entry_descriptor) | Содержит описательные сведения для записи в содержании. |
