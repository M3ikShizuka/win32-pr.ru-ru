# <a name="content-protection-commands"></a>Защита содержимого команды

Содержит список команд для метода [**IDirect3DAuthenticatedChannel9:: Configure**](/windows/desktop/api/d3d9/nf-d3d9-idirect3dauthenticatedchannel9-configure) .



| Команда                                                                                                           | Описание                                                                                                           |
|-------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**\_Инициализация D3DAUTHENTICATEDCONFIGURE**](d3dauthenticatedconfigure-initialize.md)                             | Инициализирует аутентифицированный канал.                                                                                |
| [**D3DAUTHENTICATEDCONFIGURE \_ CRYPTOSESSION**](d3dauthenticatedconfigure-cryptosession.md)                       | Связывает сеанс шифрования с устройством декодера DirectX Video Acceleration 2 (ДКСВА-2) и устройством Direct3D. |
| [**D3DAUTHENTICATEDCONFIGURE \_ енкриптионвхенакцессибле**](d3dauthenticatedconfigure-encryptionwhenaccessible.md) | Задает уровень шифрования, который выполняется перед тем, как защищенное содержимое становится доступным для ЦП или шины.         |
| [**\_Защита D3DAUTHENTICATEDCONFIGURE**](d3dauthenticatedconfigure-protection.md)                             | Включает или отключает защиту устройства.                                                                        |
| [**D3DAUTHENTICATEDCONFIGURE \_ шаредресаурце**](d3dauthenticatedconfigure-sharedresource.md)                     | Позволяет процессу открывать общий ресурс или отключать процесс от открытия общих ресурсов.                     |
