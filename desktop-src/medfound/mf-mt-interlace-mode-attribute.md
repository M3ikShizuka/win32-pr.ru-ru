# <a name="mf_mt_interlace_mode-attribute"></a>\_ \_ Атрибут режима с чередованием для MF MT \_

Описывает, как чередуются кадры в видеороликах.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Комментарии

Значение этого атрибута является членом перечисления [**мфвидеоинтерлацемоде**](/windows/desktop/api/mfobjects/ne-mfobjects-mfvideointerlacemode) .

Константа GUID для этого атрибута экспортируется из мфууид. lib.
