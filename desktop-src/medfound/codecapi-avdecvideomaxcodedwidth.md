# <a name="codecapi_avdecvideomaxcodedwidth-property"></a>КОДЕКАПИ \_ авдеквидеомакскодедвидс, свойство

Задает максимальную ширину изображения, которую видеодекодер будет принимать в качестве входного типа.

## <a name="data-type"></a>Тип данных

**Long** (**VT \_ I4**)

## <a name="property-guid"></a>GUID свойства

КОДЕКАПИ \_ авдеквидеомакскодедвидс
