# <a name="mestreamseeked-event"></a>Событие Местреамсикед

Вызывается потоком мультимедиа после вызова [**имфмедиасаурце:: Start**](/windows/desktop/api/mfidl/nf-mfidl-imfmediasource-start) , чтобы выполнить поиск в потоке. Поток мультимедиа вызывает это событие, когда источник мультимедиа создает событие [месаурцесикед](mesourceseeked.md) .

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE           | Описание                                                        |
|-------------------|--------------------------------------------------------------------|
| VT \_ i8<br/> | Новое время начала, в 100-наносекундных единицах.<br/> <br/> |
