# <a name="mf_toponode_markout_here-attribute"></a>\_Атрибут MF топоноде \_ метка \_

Указывает, применяется ли к этому узлу отметка для конвейера. Отметка — это точка, в которой заканчивается презентация. Если компоненты конвейера формируют данные после точки отметки, данные не подготавливаются к просмотру.

## <a name="data-type"></a>Тип данных

**UINT32**

Рассматривать как логическое значение.

## <a name="remarks"></a>Комментарии

Этот атрибут применяется ко всем типам узлов.

Если этот атрибут имеет **значение true**, то конвейер Media Foundation обрезает выходные примеры из этого узла в соответствии со временем окончания презентации. Загрузчик топологии задает этот атрибут при разрешении топологии. Большинству приложений не требуется устанавливать или извлекать этот атрибут.

Рекомендуется, чтобы только один узел в каждой ветви топологии имел атрибут со значением **true**. Ветвь топологии определяется в качестве пути от исходного узла до выходного узла. В ветви на \_ \_ \_ одном узле ветви должны быть установлены атрибуты MF топоноде и [MF \_ топоноде \_ Маркин \_ ](mf-toponode-markin-here-attribute.md) . Они не могут быть установлены на разных узлах в одной ветви.

Константа GUID для этого атрибута экспортируется из мфууид. lib.
