# <a name="mf_mt_max_frame_average_luminance_level-attribute"></a>\_ \_ \_ \_ \_ Атрибут среднего уровня освещенности \_ для сервера MF MT Max

Указывает максимальный средний уровень яркости каждого кадра для содержимого в НИТС. Этот атрибут имеет ту же семантику, что и **максфалл** , как определено в стандарте CEA-861,3.

## <a name="data-type"></a>Тип данных

**UINT32**
