# <a name="opm_set_protection_level"></a>ОПМ \_ установить \_ \_ уровень защиты

Задает уровень защиты для механизма защиты выходных данных.



| Требование | Значение |
|--------------|-----------------------------------------------------------------------------------------------------|
| Идентификатор GUID команды | ОПМ \_ установить \_ \_ уровень защиты                                                                         |
| Входные данные   | Структура [**\_ \_ \_ \_ параметров уровня защиты ОПМ Set**](/windows/desktop/api/opmapi/ns-opmapi-opm_set_protection_level_parameters) |



 

## <a name="remarks"></a>Комментарии

Некоторые соединители могут поддерживать несколько механизмов защиты. С помощью этого типа соединителя можно применить несколько механизмов защиты к одному и тому же соединителю с разными настройками для каждого из них.
