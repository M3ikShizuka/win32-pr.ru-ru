# <a name="mestreamsinkformatinvalidated-event"></a>Событие Местреамсинкформатинвалидатед

Посылается приемником потока, когда нисходящий формат становится недействительным, и его необходимо повторно согласовать.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |



## <a name="remarks"></a>Комментарии

Необходимо повторно отправить данные, помещенные в очередь на приемник, после текущей позиции воспроизведения.
