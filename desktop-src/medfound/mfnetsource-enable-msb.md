# <a name="mfnetsource_enable_msb-property"></a>МФНЕТСАУРЦЕ включить свойство в виде нестаршего \_ \_ значения

Указывает, включен ли в сетевом источнике протокол многоадресной рассылки вещания потока мультимедиа (в виде пересылки).



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

**LONG**

VT \_ I4

**лвал**



## <a name="remarks"></a>Комментарии

Константа **\_ включения мфнетсаурце \_** для этого ключа свойства определяет идентификатор GUID. Идентификатор свойства (PID) равен нулю.

Приложения могут использовать это свойство для настройки сетевого источника. Чтобы задать свойство, передайте указатель **ипропертисторе** в сопоставитель источника. Дополнительные сведения см. [в разделе Настройка источника мультимедиа](configuring-a-media-source.md).
