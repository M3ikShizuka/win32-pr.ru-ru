# <a name="mf_sa_d3d12_heap_flags-attribute"></a>\_ \_ \_ \_ Атрибут флагов D3D12 кучи MF SA

Содержит значение, указывающее параметры кучи, используемые для ресурсов Direct3D в потоке.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Комментарии

Значение этого атрибута является побитовым или сочетанием значений перечисления [D3D12_HEAP_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_heap_flags) .
