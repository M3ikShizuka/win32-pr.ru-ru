# <a name="mfpkey_pixelaspectratio-property"></a>МФПКЭЙ \_ пикселаспектратио, свойство

Задает пропорцию в пикселях для выходного типа носителя.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

Доступно только с помощью [**ипропертисторе**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Тип данных

VT \_ UI8

## <a name="applies-to"></a>Применяется к

-   [DSP изменения видеоконтроллеров](videoresizer.md)

## <a name="remarks"></a>Remarks

Сохраняет числитель в старших 4 байтах и знаменатель в младших 4 байтах.
