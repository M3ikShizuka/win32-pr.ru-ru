# <a name="table-of-contents-parser"></a>Средство синтаксического анализа оглавления

средство синтаксического анализа оглавления (средство синтаксического анализа TOC) — это Windows компонент, позволяющий хранить дополнительные сведения об индексах в файлах расширенного формата системы (ASF).

> [!Note]  
> Средство синтаксического анализа ОГЛАВЛЕНИя не связано напрямую с интерфейсами API Microsoft Media Foundation, хотя его можно использовать в сочетании с Media Foundation.
