# <a name="mfsampleextension_derivedfromtopfield-attribute"></a>Мфсампликстенсион \_ дериведфромтопфиелд, атрибут

Указывает, был ли извлеченный видеокадр получен из верхнего или нижнего поля. Этот атрибут применяется к примерам мультимедиа.

## <a name="data-type"></a>Тип данных

**Bool** , сохраненный как **UINT32**

## <a name="getset"></a>Get/Set

Чтобы получить этот атрибут, вызовите метод [**имфаттрибутес:: UInt32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-getuint32).

Чтобы задать этот атрибут, вызовите [**имфаттрибутес:: SetUINT32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-setuint32).

## <a name="applies-to"></a>Применяется к

[**имфсампле**](/windows/desktop/api/mfobjects/nn-mfobjects-imfsample)

## <a name="remarks"></a>Комментарии

Этот атрибут допустим только для образцов с нечередованием. Установите этот атрибут, если фрейм был раззагружен путем интерполяции одного из полей.

Если значение равно **true**, то нижнее поле было интерполяции из верхнего поля. Если значение равно **false**, верхнее поле было интерполяция от нижнего поля.

Если атрибут не задан, кадр не был развернут. Кадр является либо истинным прогрессивным кадром, либо чередованием кадров.

Этот атрибут является информационным. Этот атрибут может быть установлен программным обеспечением. Если этот атрибут задан, он предоставляет указание, что можно восстановить исходное поле, удалив строки с интерполяцией. Например, если атрибут имеет **значение true**, можно восстановить исходное верхнее поле, удалив нижнее поле с интерполяцией.

Константа GUID для этого атрибута экспортируется из мфууид. lib.
