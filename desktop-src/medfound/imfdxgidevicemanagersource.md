# <a name="imfdxgidevicemanagersource-interface"></a>Интерфейс Имфдксгидевицеманажерсаурце

Предоставляет функциональные возможности для получения [**имфдксгидевицеманажер**](/windows/desktop/api/mfobjects/nn-mfobjects-imfdxgidevicemanager) из приемника отрисовки Microsoft Media Foundation видео.

## <a name="members"></a>Элементы

Интерфейс **имфдксгидевицеманажерсаурце** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Имфдксгидевицеманажерсаурце** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **имфдксгидевицеманажерсаурце** содержит следующие методы.



| Метод                                                      | Описание                                                                                                              |
|:------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------|
| [**Управление**](imfdxgidevicemanagersource-getmanager.md) | Возвращает [**имфдксгидевицеманажер**](/windows/desktop/api/mfobjects/nn-mfobjects-imfdxgidevicemanager) из приемника отрисовки Media Foundation видео.<br/> |
