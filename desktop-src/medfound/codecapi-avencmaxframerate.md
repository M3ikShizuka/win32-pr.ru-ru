# <a name="codecapi_avencmaxframerate-property"></a>КОДЕКАПИ \_ авенкмаксфрамерате, свойство

Задает максимальную скорость ввода видеокадров, передаваемых кодировщику в режиме реального времени.

## <a name="data-type"></a>Тип данных

**Улонгулонг** (VT \_ UI8)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмаксфрамерате**

## <a name="remarks"></a>Remarks

Это свойство позволяет вызывающему объекту указать максимальную скорость ввода видеокадров, передаваемых кодировщику в реальном времени. Это дает кодировщику указание скорости, необходимой для обработки входящих кадров. Это полезно для кодировщиков, которые задают себя в определенной конфигурации состояния электропитания в зависимости от разрешения и частоты кадров исходного видео.

Частота кадров выражается в виде коэффициента. Верхний 32 бит значения атрибута содержит числитель, а младшие 32 бита содержат знаменатель. Например, если частота кадров составляет 30 кадров в секунду (кадров/с), то соотношение составляет 30/1. Если частота кадров составляет 29,97 кадров/с, то коэффициент равен 30000/1001.
