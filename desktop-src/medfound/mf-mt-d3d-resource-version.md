# <a name="mf_mt_d3d_resource_version-attribute"></a>\_ \_ \_ Атрибут версии ресурса D3D \_ MT

Указывает версию Direct3D ресурсов, хранящихся в потоке данных, связанном с типом носителя. 

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Комментарии

Значение этого атрибута соответствует члену перечисления [MF_MT_D3D_RESOURCE_VERSION_ENUM](/windows/win32/api/mfd3d12/ne-mfd3d12-mf_mt_d3d_resource_version_enum) .
