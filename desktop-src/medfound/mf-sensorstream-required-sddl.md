# <a name="mf_sensorstream_required_sddl-attribute"></a>\_ \_ Требуемый \_ атрибут SDDL сенсорстреам MF

Этот атрибут используется для указания на потоке языка определения дескрипторов безопасности (SDDL) для указания детальных прав доступа для данного датчика.

## <a name="data-type"></a>Тип данных

**WCHAR\***
