# <a name="mfpkey_asfstreamsink_corrected_leakybucket-property"></a>МФПКЭЙ \_ асфстреамсинк, \_ исправленное \_ свойство леакибуккет

Задает параметры "сегмент утечки" (см. примечания) для потока в приемнике ASF Media.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

Массив значений **типа DWORD** (**Каул**)

VT \_ вектор \| VT \_ UI4

**каул**



## <a name="remarks"></a>Комментарии

Приложение может задать это свойство в потоке приемника мультимедиа ASF после согласования типа носителя для потока. Используйте это свойство, чтобы указать фактическое окно буфера, которое будет использоваться кодеком Windows Media. Эти сведения можно получить из кодека с помощью интерфейса **ивмкодеклеакибуккет** , который описан в пакете SDK для формата Windows Media.

Значение этого свойства представляет собой массив из трех значений **типа DWORD** в следующем порядке:

-   Скорость в битах в секунду.
-   Размер буфера в байтах.
-   Заполнение начальной буферизации в байтах.
