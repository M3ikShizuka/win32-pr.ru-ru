# <a name="mf_sensorstream_required_capabilities-attribute"></a>\_ \_ Атрибут требуемых \_ возможностей MF сенсорстреам

Этот атрибут содержит список строк возможностей, разделенных точкой с запятой, которые указывают возможности, необходимые для конкретного потока. Список строк возможностей, которые могут быть добавлены в этот атрибут, см. в разделе [девицекапабилити](/uwp/schemas/appxpackage/appxmanifestschema/element-devicecapability).

## <a name="data-type"></a>Тип данных

**WCHAR\***
