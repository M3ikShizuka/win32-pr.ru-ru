# <a name="meerror-event"></a>Событие Миррор

Сигнализирует о серьезной ошибке. Любой компонент Media Foundation может отправить это событие в любое время. Вызовите [**имфмедиаевент:: Status**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getstatus) , чтобы получить код ошибки операции, которая завершилась ошибкой.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |



## <a name="remarks"></a>Комментарии

Это событие следует использовать только для непредвиденных ошибок. Не отправляйте это событие, чтобы сообщить о сбое асинхронного метода. В случае сбоя асинхронного метода код ошибки возвращается в виде обычного события для этого метода.

Если во время потоковой передачи возникает устранимая ошибка, отправьте событие [менонфаталеррор](menonfatalerror.md) .
