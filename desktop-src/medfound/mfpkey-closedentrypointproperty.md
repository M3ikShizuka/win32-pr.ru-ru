# <a name="mfpkey_closedentrypoint-property"></a>МФПКЭЙ \_ клоседентрипоинт, свойство

Указывает шаблон кодировки, используемый кодировщиком в начале группы изображений.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

**g \_ всзвмвкклоседентрипоинт**

## <a name="data-type"></a>Тип данных

**Логическое значение VT \_**

## <a name="default-value"></a>Значение по умолчанию

**ВАРИАНТ \_ false**

## <a name="remarks"></a>Комментарии

Если для этого свойства задать значение **\_ true**, шаблон кодировки — ббиббп. Если для этого свойства задано значение " **Variant \_ false**", шаблон кодировки — иббпббп.
