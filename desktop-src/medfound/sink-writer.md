# <a name="sink-writer"></a>Модуль записи приемников

Модуль записи приемника — это компонент для кодирования аудио-или видеофайлов.

На приведенной ниже схеме на высоком уровне показано, как приложение использует модуль записи приемника для кодирования и аудио-или видеофайла.

![Схема, на которой показан модуль записи приемника.](images/encoding09.png)

Модуль записи приемника содержит приемник мультимедиа и, при необходимости, один или несколько кодировщиков. Кодировщики преобразуют несжатые аудио или видео данные в закодированные битстреамс. Приемник мультимедиа выводит битстреамс в файл. Модуль записи приемников выполняет следующие задачи:

-   Загружает приемник мультимедиа.
-   Находит и загружает кодировщики.
-   Управляет потоком данных для кодировщиков и приемника мультимедиа.

Приложение передает аудио-и видеоданные в модуль записи приемника в качестве входных данных. Не имеет значения, как приложение получает или создает входные данные. Один из вариантов — использовать [средство чтения исходного кода](source-reader.md), как показано на следующей схеме. Однако модуль записи приемника не требует использования модуля чтения исходного кода. Эти два компонента являются независимыми.

![Схема, на которой показан модуль чтения исходного кода и модуль записи приемника.](images/encoding02.png)
