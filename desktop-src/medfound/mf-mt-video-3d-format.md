# <a name="mf_mt_video_3d_format-attribute"></a>\_ \_ \_ Атрибут трехмерного формата видео MF MT \_

Для типа видеоматериала указывает, как трехмерные видеокадры хранятся в памяти.

## <a name="data-type"></a>Тип данных

**MFVideo3DFormat** хранится как **UINT32**

## <a name="remarks"></a>Remarks

Значение этого атрибута является членом перечисления [**MFVideo3DFormat**](/windows/desktop/api/mfapi/ne-mfapi-mfvideo3dformat) . Атрибут применяется только в том случае, если атрибут [ \_ \_ \_ 3D-видео MF](mf-mt-video-3d.md) имеет **значение true**.

Этот атрибут необходим для несжатых трехмерных видеоформатов. Это необязательно для сжатого трехмерного видео. Источник мультимедиа, который доставляет закодированные кадры, может иметь возможность установить атрибут на основе информации в контейнере файлов. В противном случае атрибут должен быть задан декодером в выходном типе носителя декодера.
