# <a name="mfpkey_vbrquality-property"></a>МФПКЭЙ \_ вбркуалити, свойство

Задает реальный уровень качества для кодирования с переменной скоростью (1 проход) с учетом качества (с частотой VBR).

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвквбркуалити, g \_ всзвмкпаудиовбркуалити

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="remarks"></a>Комментарии

Не все значения в диапазоне имеют уникальное значение. Значения, представляющие этап качества, начиная с предыдущего уровня: 1, 4, 8, 11, 15, 18, 22, 25, 29, 33, 36, 40, 43, 47, 50, 54, 58, 61, 65, 68, 72, 75, 79, 83, 86, 90, 93, 97 и 100.

Для объектов звукового кодировщика режимы качества предоставляются в структуре [**вавеформатекс**](/previous-versions/dd757713(v=vs.85)) типов выходных данных, получаемых с помощью [**Имедиаобжект:: жетаутпуттипе**](/previous-versions/windows/desktop/api/mediaobj/nf-mediaobj-imediaobject-getoutputtype).
