# <a name="mfpkey_decodercomplexityprofile-property"></a>МФПКЭЙ \_ декодеркомплекситипрофиле, свойство

Указывает профиль сложности закодированного содержимого.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкдекодеркомплекситипрофиле

## <a name="data-type"></a>Тип данных

**VT \_ BSTR**

## <a name="remarks"></a>Remarks

Это значение можно прочитать только после завершения кодирования.

Для звука это свойство имеет одно из следующих значений.



| Значение | Описание                                                                                    |
|-------|------------------------------------------------------------------------------------------------|
| L1  | Стандартная кодировка с частотой выборки 44,1 кГц и максимальной скоростью в 161 кбит/с.         |
| L2  | Стандартная кодировка с частотой выборки до 48 кГц и максимальной скоростью в 161 кбит/с.         |
| Индекс  | Стандартная кодировка с частотой выборки до 48 кГц и максимальной скоростью в 385 кбит/с.         |
| "M0a" | Professional кодирование с частотой дискретизации 48 кгц и скоростью между 48 и 192 кбит/с.  |
| "M0b" | Professional кодирование с частотой выборки до 48 кгц и максимальной скоростью в 192 кбит/с.      |
| M1  | Professional кодирование с частотой выборки до 48 кгц и максимальной скоростью в 384 кбит/с.      |
| M2  | Professional кодирование с частотой выборки до 96 кгц и максимальной скоростью в 768 кбит/с.      |
| M3  | Professional кодирование с частотой выборки до 96 кгц и максимальной скоростью 1,5 мбит/с.      |
| N1  | Кодирование без потерь с частотой выборки до 48 кГц и максимум 2 канала.                |
| N2  | Кодирование без потерь с частотой выборки до 96 кГц и максимум 6 каналов (5,1 вокруг). |
| N3  | Кодирование без потерь с частотой выборки до 96 кГц и максимум 8 каналов (7,1 вокруг). |



 

Для видео это свойство имеет одно из следующих значений:



| Значение | Описание                                                                       |
|-------|-----------------------------------------------------------------------------------|
| ПРОФИЛЯ  | расширенный профиль (доступен только в кодеке расширенного профиля Windows Media видео 9) |
| CP  | больше не поддерживается                                                               |
| МВ  | Основной профиль                                                                      |
| ПОРТОВ  | простой профиль                                                                    |



 

Для видеоматериала можно запросить уровень профиля, установив [мфпкэй \_ декодеркомплекситирекуестед](mfpkey-decodercomplexityrequestedproperty.md) перед началом кодирования. Кодек попытается выполнить кодирование в параметрах запрошенного уровня сложности, но другие настроенные параметры будут иметь приоритет. Следует всегда проверять фактическое значение профиля сложности после кодирования в случае, если запрос не может быть обработан.
