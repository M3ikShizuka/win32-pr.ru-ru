# <a name="imfsourcebufferlistgetlength-method"></a>Метод Имфсаурцебуфферлист:: DATALENGTH

Возвращает число объектов [**имфсаурцебуффер**](/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfsourcebuffer) в списке.

## <a name="syntax"></a>Синтаксис


```C++
DWORD GetLength();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Число исходных буферов в списке.
