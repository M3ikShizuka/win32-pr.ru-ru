# <a name="processing-codec-dmo-input-and-output"></a>обработка кодека DMO входных и выходных данных

после настройки типа входных данных и типа вывода для DMO можно начать обработку образцов данных. вы передаете образцы в DMO для обработки с помощью метода **имедиаобжект::P роцессинпут** , а затем получаете обработанный пример путем вызова **имедиаобжект::P роцессаутпут**. Необходимо задать точные метки времени и длительности для всех пройденных выборок. Метки времени не являются строго необходимыми, но помогают поддерживать синхронизацию аудио и видео. Если у вас нет меток времени для выборки, лучше оставить их, не используя неопределенные значения.
