# <a name="mestreamsinkdevicechanged-event"></a>Событие Местреамсинкдевицечанжед

Вызывается приемниками потоков расширенного модуля подготовки видео (Евр) при изменении видеоустройства. Например, ЕВР вызывает это событие при получении события [**\_ \_ изменения дисплея EC**](../directshow/ec-display-changed.md) .

Конвейер Media Foundation отвечает на это событие, повторно отправляя все примеры запросов, которые завершились сбоем во время изменения устройства.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |
