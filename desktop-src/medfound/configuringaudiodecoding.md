# <a name="configuring-audio-decoding-microsoft-media-foundation"></a>Настройка декодирования звука (Microsoft Media Foundation)

декодирование Windows мультимедийного содержимого намного проще, чем его кодирование. После создания объекта звукового декодера задайте тип входных данных с помощью метода **имедиаобжект:: сетинпуттипе** или **Имфтрансформ:: сетинпуттипе** . Тип носителя, используемый для входных данных декодера, должен соответствовать типу выходных данных, который использовался при кодировании содержимого. Сюда входят данные расширенного формата, добавленные в структуру **вавеформатекс** . Необходимо убедиться, что эти данные верны, так как декодер не может обрабатывать примеры без него.

После настройки типа входных данных можно настроить любые функции декодера, которые вы хотите использовать. Функции декодера, такие как используемые для кодирования, задаются с помощью методов **ипропертибаг** или **ипропертисторе**.

После установки типа входных данных и настройки всех функций декодера можно перечислить типы выходных данных, поддерживаемые декодером, выполнив вызовы **имедиаобжект:: жетаутпуттипе** или **Имфтрансформ:: жетаутпуттипе**.
