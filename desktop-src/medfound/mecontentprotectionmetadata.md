# <a name="mecontentprotectionmetadata-event"></a>Событие Меконтентпротектионметадата

Поток мультимедиа использует это событие для отправки в декодер метаданных, относящихся к системе защиты.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |



## <a name="attributes"></a>Атрибуты

Для этого события определены следующие атрибуты.



| attribute                                                                                              | Описание                                                                                         |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [\_ \_ \_ KEYDATA МЕТАДАННЫХ потока событий \_ MF](mf-event-stream-metadata-keydata.md)<br/>                | Этот атрибут является необязательным. Данные, относящиеся к системе защиты.<br/> <br/>              |
| [\_ \_ \_ \_ кэйидс содержимого МЕТАДАННЫХ для потока событий MF \_](mf-event-stream-metadata-content-keyids.md)<br/> | Идентификаторы ключей содержимого, с которыми связано событие.<br/> <br/>                          |
| [\_ \_ \_ SYSTEMID метаданные потока событий \_ MF](mf-event-stream-metadata-systemid.md)<br/>              | Этот атрибут является необязательным. Идентификатор системы, для которого предназначены ключевые данные.<br/> <br/> |



## <a name="remarks"></a>Комментарии

Это событие используется, например, для взаимодействия с событием смены ключа. В этом случае его следует отправить как можно раньше, чтобы обеспечить время декодера для подготовки перед выполнением примеров, зашифрованных с помощью нового идентификатора ключа.
