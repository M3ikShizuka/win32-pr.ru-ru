# <a name="asf-mutual-exclusion-type-guids"></a>Идентификаторы GUID типов взаимного исключения ASF

Следующие идентификаторы GUID определяют типы для объектов взаимного исключения для потоков ASF.



| Константа                                                                                                                                                                                                                                              | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="MFASFMutexType_Language"></span><span id="mfasfmutextype_language"></span><span id="MFASFMUTEXTYPE_LANGUAGE"></span><dl> <dt>**\_Язык мфасфмутекстипе**</dt> </dl>                 | Потоки являются взаимоисключающими по отношению к языку. Этот тип взаимного исключения похож на звуковые дорожки на DVD-диске.<br/>                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="MFASFMutexType_Bitrate"></span><span id="mfasfmutextype_bitrate"></span><span id="MFASFMUTEXTYPE_BITRATE"></span><dl> <dt>**\_Скорость мфасфмутекстипе**</dt> </dl>                     | Потоки являются взаимоисключающими по скорости. Этот тип взаимного исключения также называется исключением с несколькими битовыми частотами (MBR).<br/>                                                                                                                                                                                                                                                                                                                                                        |
| <span id="MFASFMutexType_Presentation"></span><span id="mfasfmutextype_presentation"></span><span id="MFASFMUTEXTYPE_PRESENTATION"></span><dl> <dt>**Мфасфмутекстипе \_ презентацию**</dt> </dl> | Потоки являются взаимоисключающими по представлениям. Этот тип можно использовать во многих сценариях, но его следует использовать только в том случае, если содержимое совпадает, но принимает другую форму. Например, два потока, содержащие одно и то же видео, один из которых отформатировано для заполнения экрана, а другой, обслуживающий исходные Широкоэкранные пропорции, должны быть взаимоисключающими с помощью этого типа. Еще один пример — потоки, содержащие видео одного снимка сцены из разных углов.<br/> |
| <span id="MFASFMutexType_Unknown"></span><span id="mfasfmutextype_unknown"></span><span id="MFASFMUTEXTYPE_UNKNOWN"></span><dl> <dt>**Мфасфмутекстипе \_ неизвестный**</dt> </dl>                     | Потоки являются взаимоисключающими на основе настраиваемых критериев.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                           |
