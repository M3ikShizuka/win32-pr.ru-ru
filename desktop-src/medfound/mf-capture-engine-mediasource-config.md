# <a name="mf_capture_engine_mediasource_config-attribute"></a>\_ \_ \_ Атрибут конфигурации МЕДИАСАУРЦЕ модуля записи MF \_

Содержит свойства конфигурации для источника записи.

## <a name="data-type"></a>Тип данных

**Ипропертисторе \* *_ хранится как _* IUnknown\***

## <a name="remarks"></a>Remarks

В настоящее время этот атрибут не учитывается. В будущем он может использоваться для передачи свойств конфигурации источнику мультимедиа.
