# <a name="mfnetsource_hostexe-property"></a>МФНЕТСАУРЦЕ \_ хостексе, свойство

Значение поля "c-хостексе", используемое сетевым источником для ведения журнала. Приложения могут задать для этого свойства имя ведущего приложения. Например, значение может быть "iexplore.exe", если приложение размещено на веб-странице.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

Строка расширенных символов (**WCHAR** \* )

VT \_ LPWSTR

**пвсзвал**



## <a name="remarks"></a>Remarks

Константа **мфнетсаурце \_ хостексе** определяет идентификатор GUID для этого ключа свойства. Идентификатор свойства (PID) равен нулю.

Приложения могут использовать это свойство для настройки сетевого источника. Чтобы задать свойство, передайте указатель **ипропертисторе** в сопоставитель источника. Дополнительные сведения см. [в разделе Настройка источника мультимедиа](configuring-a-media-source.md).
