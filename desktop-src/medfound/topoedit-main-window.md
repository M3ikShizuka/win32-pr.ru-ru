# <a name="topoedit-main-window"></a>Главное окно Топоедит

После запуска Топоедит появится окно, показанное на следующем снимке экрана.

![снимок экрана, показывающий главное окно топоедит](images/809bce6e-43b7-4399-a41b-deff894ba151.jpg)

Элементы главного окна Топоедит:

-   **Панель топологии**. Отображается в левой части окна. Эта область используется для добавления узлов топологии, просмотра топологии и подключения узлов топологии. Сведения об использовании этой панели см. в разделе [Создание топологий с помощью топоедит](building-topologies-by-using-topoedit.md).
-   **Панель «атрибуты»**. Отображается в правой части окна. В этой области отображаются атрибуты выбранного узла топологии. Сведения об атрибутах, отображаемых на этой панели, см. в разделе [Просмотр сведений о топологии](viewing-topology-information.md).
-   Ярлык. См. раздел [Топоедит Menus](topoedit-menus.md).
-   Панель инструментов. См. [панель инструментов топоедит](topoedit-toolbar.md).
