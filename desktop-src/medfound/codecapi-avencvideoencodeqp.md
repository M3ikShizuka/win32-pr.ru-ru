# <a name="codecapi_avencvideoencodeqp-property"></a>КОДЕКАПИ \_ авенквидеоенкодекп, свойство

Указывает параметр дискретизация (QP) для кодирования видео.

## <a name="data-type"></a>Тип данных

**Улонглонг** (VT \_ UI8)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеоенкодекп**

## <a name="property-value"></a>Значение свойства

Набор из 4 16 битовых полей, используемых для указания кадра QPs в кодировке fixed-QP.

Поля:

-   BITS 0-15: служба QP по умолчанию
-   BITS 16-31: обработчики запросов, используемые для кадров I
-   BITS 32-47: служба QP, используемая для кадров P
-   BITS 48-63: служба QP, используемая для кадров B

## <a name="remarks"></a>Комментарии

Это свойство также используется с [кодировщиками камер H. 264 увк 1,5](camera-encoder-h264-uvc-1-5.md).

Чтобы обеспечить согласованное использование в разных кодировщиках, необходимо предположить, что кодировщики будут рассматривать только обработчики запросов по умолчанию и могут игнорировать значения QP для изображений I/P/B.
