# <a name="direct3d-9-video-interfaces"></a>Интерфейсы видео Direct3D 9

Описание интерфейсов видео Microsoft Direct3D 9.



| Интерфейс                                                                | Описание                                                                                                |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IDirect3D9ExOverlayExtension**](/windows/desktop/api/d3d9/nn-d3d9-idirect3d9exoverlayextension)     | Запрашивает аппаратные возможности наложения устройства Direct3D.                                            |
| [**IDirect3DAuthenticatedChannel9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dauthenticatedchannel9) | Предоставляет канал связи с графическим драйвером или с исполняющей средой Direct3D.                        |
| [**IDirect3DCryptoSession9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dcryptosession9)               | Представляет сеанс шифрования.                                                                        |
| [**IDirect3DDevice9Video**](/windows/desktop/api/d3d9/nn-d3d9-idirect3ddevice9video)                   | Позволяет приложению использовать защиту содержимого и службы шифрования, реализованные графическим драйвером. |
| [**IDirect3DDXVADevice9**](idirect3ddxvadevice9.md)                     | Представляет аппаратный ускоритель для ускорения видео DirectX (ДКСВА) 1,0.                               |
| [**IDirect3DVideoDevice9**](idirect3dvideodevice9.md)                   | Включает аппаратное ускорение декодирования с устройства Direct3D с помощью ДКСВА версии 1,0.                      |
