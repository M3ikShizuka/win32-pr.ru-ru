# <a name="mf_mt_timestamp_can_be_dts-attribute"></a>\_ \_ Метка времени MF \_ MT \_ может быть \_ атрибутом DTS

Указывает, может ли декодер использовать метки времени декодирования (DTS) при установке меток времени.

## <a name="data-type"></a>Тип данных

**Bool** , сохраненный как **UINT32**
