# <a name="mfpkey_loopfilter-property"></a>МФПКЭЙ \_ лупфилтер, свойство

Указывает, должен ли кодек использовать фильтр для деблокировки в процессе кодирования.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвклупфилтер

## <a name="data-type"></a>Тип данных

Логическое значение VT \_

## <a name="remarks"></a>Комментарии

Фильтр в цикле является разблокирующим методом, который применяется во время кодирования и декодирования, чтобы сократить количество блокирующих артефактов во время кодирования ("в цикле"), чтобы будущие кадры P-и B-Frame не перенаправлялись.

Результатом применения фильтра в цикле является то, что края блоков макросов в выходных видеокадрах менее заметны. В то же время изображение может стать более мягким в внешнем виде.

Хотя фильтр в цикле может привести к уменьшению детализации изображения в некоторых кадрах, общее качество видео будет существенно выгодно. Самым большим недостатком использования фильтрации в цикле являются дополнительные затраты на производительность декодирования.
