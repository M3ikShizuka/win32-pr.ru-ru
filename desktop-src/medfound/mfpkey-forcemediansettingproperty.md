# <a name="mfpkey_forcemediansetting-property"></a>МФПКЭЙ \_ форцемедиансеттинг, свойство

Указывает, должен ли кодек использовать медианную фильтрацию во время кодирования.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкфорцемедиансеттинг

## <a name="data-type"></a>Тип данных

Логическое значение VT \_

## <a name="default-value"></a>Значение по умолчанию

FALSE

## <a name="remarks"></a>Remarks

Функция медианы указывает, что кодек игнорирует шум и зернистость при вычислении движения между кадрами. Это может улучшить качество очень шумных видеороликов, но может привести к тому, что в случае чистого видео в исходном видеоролике могут возникнуть помехи (также известные как конечные артефакты).

По умолчанию кодек использует внутреннюю логику, чтобы определить, следует ли использовать медианную фильтрацию. Если задать это свойство, эта логика будет переопределена.

> [!Note]  
> Этот фильтр отличается от фильтров нечеткого размытия, обнаруженных во многих приложениях для редактирования и последующей обработки видео.
