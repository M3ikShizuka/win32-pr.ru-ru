# <a name="how-to-seek-during-playback"></a>Поиск во время воспроизведения

\[Мфплай доступен для использования в операционных системах, указанных в разделе требования. В последующих версиях он может быть изменен или недоступен. \]

В этом разделе описывается поиск во время воспроизведения с помощью Мфплай.

**Поиск во время воспроизведения**

1.  Инициализируйте **пропвариант** для хранения времени поиска в единицах 100-наносекундных в виде **большого \_ целого** типа (**VT \_ i8**).
2.  Вызовите [**имфпмедиаплайер:: SetPosition**](/windows/desktop/api/mfplay/nf-mfplay-imfpmediaplayer-setposition). Укажите **МФП \_ поситионтипе \_ 100 НС** для первого параметра и передайте **пропвариант** для второго параметра.

## <a name="requirements"></a>Требования

Для Мфплай требуется Windows 7.
