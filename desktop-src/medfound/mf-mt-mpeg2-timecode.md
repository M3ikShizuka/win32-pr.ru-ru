# <a name="mf_mt_mpeg2_timecode-attribute"></a>Атрибут времени с кодом MF \_ MT \_ MPEG2 \_

Для типа мультимедиа, описывающего транспортный поток MPEG-2, указывает, что транспортные пакеты содержат 4-байтовый код времени.

## <a name="data-type"></a>Тип данных

**UINT32**



| Значение                                                                                                | Значение                                                                                                                                        |
|------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="0"></span><dl> <dt>**0,0**</dt> </dl> | Код времени не добавляется.<br/>                                                                                                              |
| <span id="1"></span><dl> <dt>**1**</dt> </dl> | В начале каждого транспортного пакета добавляется 4-байтовый код времени. Этот код времени необходим для некоторых стандартов цифрового телевидения.<br/> |
