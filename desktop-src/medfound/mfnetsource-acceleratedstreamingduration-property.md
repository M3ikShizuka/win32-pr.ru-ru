# <a name="mfnetsource_acceleratedstreamingduration-property"></a>МФНЕТСАУРЦЕ \_ акцелератедстреамингдуратион, свойство

Длительность ускорения потоковой передачи для сетевого источника в миллисекундах.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

**DWORD** (хранится в формате **Long**)

VT \_ I4

**лвал**



## <a name="remarks"></a>Комментарии

Константа **мфнетсаурце \_ акцелератедстреамингдуратион** определяет идентификатор GUID для этого ключа свойства. Идентификатор свойства (PID) равен нулю.

Приложения могут использовать это свойство для настройки сетевого источника. Чтобы задать свойство, передайте объект **ипропертисторе** в сопоставитель источника. Дополнительные сведения см. [в разделе Настройка источника мультимедиа](configuring-a-media-source.md).

Это свойство применяется к функции быстрого запуска служб Windows Media, которая быстро воспроизводит содержимое, не дожидаясь длительной первоначальной буферизации. При использовании быстрого запуска сервер, на котором работают службы Windows Media, отправит некоторые данные в начале содержимого быстрее, чем указано в скорости потока содержимого.

Значение этого свойства по умолчанию равно 10 000 миллисекундам.
