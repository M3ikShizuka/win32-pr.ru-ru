# <a name="codecapi_avencvideoselectlayer-property"></a>КОДЕКАПИ \_ авенквидеоселектлайер, свойство

Задает слой видеоизображения для видеокодировщика.

## <a name="data-type"></a>Тип данных

**ULONG (VT \_ UI4)**

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеоселектлайер**

## <a name="remarks"></a>Комментарии

Это свойство также используется с [кодировщиками камер H. 264 увк 1,5](camera-encoder-h264-uvc-1-5.md).
