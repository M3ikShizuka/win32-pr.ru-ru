# <a name="mesourcemetadatachanged-event"></a>Событие Месаурцеметадатачанжед

Вызывается источником мультимедиа при обновлении его метаданных.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |



## <a name="remarks"></a>Комментарии

Если источник мультимедиа не может предоставить все метаданные при первоначальном создании источника, он должен вызвать это событие после того, как метаданные будут доступны.

Источник мультимедиа должен создать новый дескриптор представления и скопировать все атрибуты из дескриптора презентации (PD) в объект события. Приложение может использовать объект события для перечисления новых атрибутов PD. В частности, значения для файла [с \_ \_ продолжительностью MF PD](mf-pd-duration-attribute.md) и [MF \_ PD \_ общего \_ \_ размера](mf-pd-total-file-size-attribute.md) могут быть неизвестны до тех пор, пока файл не будет полностью скачан.
