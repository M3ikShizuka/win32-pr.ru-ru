# <a name="mf_mt_spatial_audio_max_dynamic_objects-attribute"></a>\_ \_ \_ Атрибут пространственного звука ( \_ Максимальное \_ число \_ объектов) MF MT

Указывает максимальное число динамических звуковых объектов, которые могут быть отображены в конечной точке аудио симулатанеаусли.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

[**Имфспатиалаудиосампле**](/windows/desktop/api/mfspatialaudio/nn-mfspatialaudio-imfspatialaudiosample) может содержать меньше выборок, чем число, указанное этим атрибутом. Если образец содержит больше звуковых объектов, чем указано этим атрибутом, поведение не определено.
