# <a name="mesessiontopologystatus-event"></a>Событие Месессионтопологистатус

Вызывается сеансом мультимедиа при изменении состояния топологии.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE                | Описание                                                                                         |
|------------------------|-----------------------------------------------------------------------------------------------------|
| VT \_ Unknown<br/> | Указатель на интерфейс [**имфтопологи**](/windows/desktop/api/mfidl/nn-mfidl-imftopology) топологии.<br/> <br/> |



## <a name="attributes"></a>Атрибуты

Для этого события определены следующие атрибуты.



| attribute                                                                            | Описание                                                      |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------|
| [**\_ \_ состояние ТОПОЛОГИИ событий \_ MF**](mf-event-topology-status-attribute.md)<br/> | Указывает новое состояние топологии.<br/> <br/> |



## <a name="remarks"></a>Remarks

Пример кода, извлекающий указатель [**имфтопологи**](/windows/desktop/api/mfidl/nn-mfidl-imftopology) из события, см. в разделе событие [месессионтопологисет](mesessiontopologyset.md) .
