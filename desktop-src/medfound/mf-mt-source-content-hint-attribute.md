# <a name="mf_mt_source_content_hint-attribute"></a>\_Атрибут указания \_ \_ содержимого источника MF MT \_

Описывает предполагаемые пропорции для типа видеоклипа.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

Значение этого атрибута является членом перечисления [**мфвидеосркконтенсинтфлагс**](/windows/desktop/api/mfapi/ne-mfapi-mfvideosrccontenthintflags) .

Константа GUID для этого атрибута экспортируется из мфууид. lib.
