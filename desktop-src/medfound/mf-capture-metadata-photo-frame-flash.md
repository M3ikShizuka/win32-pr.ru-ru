# <a name="mf_capture_metadata_photo_frame_flash-attribute"></a>\_ \_ \_ \_ \_ Атрибут флеш-кадра фото захвата метаданных записи MF

Указывает, активирована ли флэш-память для захваченного кадра.

## <a name="data-type"></a>Тип данных

**UINT32**



| Значение                                                                               | Значение                                                                                                                                               |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>0</dt> </dl>        | Флэш-память не активирована в этом кадре.<br/>                                                                                                   |
| <dl> <dt>ненулевое значение</dt> </dl> | В этом кадре активирована флэш-память.<br/>                                                                                                       |
| <dl> <dt>1</dt> </dl>        | Зарезервировано<br/> Не следует явно проверять значение 1. Приложения должны проверять наличие! = 0, чтобы указать, была ли запущена флэш-память.<br/> |
