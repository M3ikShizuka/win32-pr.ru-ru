# <a name="mf_user_data_payload-attribute"></a>\_ \_ Атрибут полезных данных пользователя MF \_

Определяет, включать ли полезные данные пользователя в образец Output.

## <a name="data-type"></a>Тип данных

**ОБЪЕКТЕ**

## <a name="remarks"></a>Remarks

Пользовательские данные могут содержать данные, такие как данные о закрытых заголовках или линейчатые данные.

Данные пользователя передаются без декодирования.
