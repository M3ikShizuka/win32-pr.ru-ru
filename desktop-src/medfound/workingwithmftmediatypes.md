# <a name="working-with-mft-media-types"></a>Работа с типами файлов MFT

Тип мультимедиа — это способ описания формата потока мультимедиа. В Media Foundation типы мультимедиа представлены интерфейсом **имфмедиатипе** . Этот интерфейс наследует интерфейс **имфаттрибутес** . Сведения о типе мультимедиа указываются в виде атрибутов.

Чтобы создать новый тип мультимедиа, вызовите функцию **мфкреатемедиатипе** . Эта функция возвращает указатель на интерфейс **имфмедиатипе** . Тип носителя изначально не имеет атрибутов.

Пакет SDK для Media Foundation предоставляет несколько вспомогательных функций для инициализации типов мультимедиа из структур форматов. Например, функция **мфинитмедиатипефромвидеоинфохеадер** инициализирует тип видео из структуры **видеоинфохеадер** , а функция **мфинитмедиатипефромвавеформатекс** инициализирует тип видео из структуры **WAVEFORMATEX** или **WAVEFORMATEXTENSIBLE** .

Типы форматов, используемые кодеками, обычно ограничиваются теми, которые описаны структурами **видеоинфохеадер** и **вавеформатекс** .

Дополнительные сведения о создании Media Foundation типов носителей и доступе к ним можно найти в документации по Media Foundation SDK.
