# <a name="mfpkey_bavg-property"></a>МФПКЭЙ \_ бавг, свойство

Указывает окно буфера (в миллисекундах) для потока с переменным числом переменных с переменной скоростью (VBR) с средней скоростью (заданным параметром [мфпкэй \_ равг](mfpkey-ravgproperty.md)).

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкбавг

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

Значение по умолчанию отсутствует, но кодек вычислит соответствующее значение на основе других параметров с ограничением VBR.

## <a name="remarks"></a>Комментарии

Это значение высчитывается кодеком после окончательного прохода кодирования. Не запрашивать это значение до завершения кодирования. Кодек интерпретирует запрос этого значения как сигнал, над которым находится сеанс кодирования. Следующий образец, который вы обрабатываете, рассматривается как начало нового сеанса.
