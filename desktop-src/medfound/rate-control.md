# <a name="rate-control"></a>Управление скоростью

[Сеанс мультимедиа](media-session.md) поддерживает изменение скорости воспроизведения, которое приложение может использовать для реализации таких функций воспроизведения, как перемотка вперед и назад. В этом разделе описывается, как приложения могут изменять скорость воспроизведения при использовании сеанса мультимедиа.

Сведения о поддержке управления частотой в собственных компонентах конвейера см. в разделе [Реализация управления скоростью](implementing-rate-control.md).
