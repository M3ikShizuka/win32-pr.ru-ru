# <a name="mfpkey_postprocessmode-property"></a>МФПКЭЙ \_ постпроцессмоде, свойство

Задает режим обработки, выполняемый для декодера.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

**g \_ всзвмвкфорцепостпроцессмоде**

## <a name="data-type"></a>Тип данных

**VT \_ I4**

## <a name="remarks"></a>Комментарии

Присвойте этому свойству одно из следующих значений.



| Значение | Значение                                                                                |
|-------|----------------------------------------------------------------------------------------|
| -1    | Декодер устанавливает режим последующей обработки на основе доступных ресурсов ЦП. |
| 0     | Декодер не выполняет обработку после обработки.                                               |
| 1     | декодер фсе выполняет быстрое деблокирование.                                                 |
| 2     | Декодер выполняет полное деблокирование.                                                  |
| 3     | Декодер выполняет быстрое деблокировку и отдачу.                                    |
| 4     | Декодер выполняет полное деблокирование и выдачу.                                    |



 

Так как значение этого свойства переходит от 0 до 4, сложность декодирования, использование ресурсов ЦП и увеличение качества изображения.
