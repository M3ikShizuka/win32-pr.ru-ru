# <a name="mf_mt_d3d12_resource_flag_deny_shader_resource-attribute"></a>\_ \_ Флаг ресурса MF D3D12, \_ \_ \_ отклонить \_ атрибут ресурса шейдера \_

Указывает, запрещено ли создание представления ресурсов шейдера для ресурсов Direct3D в потоке, связанном с типом носителя. 

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

Значение этого атрибута соответствует элементу D3D12_RESOURCE_FLAG_DENY_SHADER_RESOURCE перечисления [D3D12_RESOURCE_FLAGS](/windows/win32/api/d3d12/ne-d3d12-d3d12_resource_flags) .
