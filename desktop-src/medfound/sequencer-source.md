# <a name="sequencer-source"></a>Источник Sequencer

Источник Sequencer позволяет приложению последовательно воспроизводить коллекцию источников мультимедиа с плавным переходом между источниками. Его можно использовать для создания списков воспроизведения или одновременного воспроизведения потоков из нескольких источников.
