# <a name="codecapi_avencvideotemporallayercount-property"></a>КОДЕКАПИ \_ авенквидеотемпораллайеркаунт, свойство

Задает число временных слоев видео для кодировщика видео.

## <a name="data-type"></a>Тип данных

**ULONG (VT \_ UI4)**

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеотемпораллайеркаунт**

## <a name="remarks"></a>Remarks

Это свойство также используется с [кодировщиками камер H. 264 увк 1,5](camera-encoder-h264-uvc-1-5.md).

КОДЕКАПИ \_ авенквидеотемпораллайеркаунт, [кодекапи \_ авенквидеаусаже](codecapi-avencvideousage.md)и [кодекапи \_ авенккоммонратеконтролмоде](/windows/desktop/DirectShow/avenccommonratecontrolmode-property) являются статическими свойствами кодировщика. После установки они вступят в силу только после того, как на закрепление выходных данных камеры будет вызван тип набора носителей.
