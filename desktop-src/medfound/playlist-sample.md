# <a name="playlist-sample"></a>Пример списка воспроизведения

Показывает, как использовать Microsoft Media Foundation для воспроизведения последовательности звуковых файлов в списке воспроизведения. В примере используется [источник Sequencer](sequencer-source.md) для создания списка воспроизведения и управления им.

> [!Note]  
> Этот пример больше не включен в пакет SDK.

 

## <a name="apis-demonstrated"></a>Продемонстрированные API

В этом образце демонстрируются следующие интерфейсы Media Foundation:

-   [**имфмедиасаурцетопологипровидер**](/windows/desktop/api/mfidl/nn-mfidl-imfmediasourcetopologyprovider)
-   [**имфмедиасессион**](/windows/desktop/api/mfidl/nn-mfidl-imfmediasession)
-   [**имфсекуенцерсаурце**](/windows/desktop/api/mfidl/nn-mfidl-imfsequencersource)

## <a name="usage"></a>Использование

пример списка воспроизведения создает приложение Windows.

-   Чтобы создать новый список воспроизведения, выберите в меню **файл** пункт **Добавить в список воспроизведения** . В диалоговом окне **открытия файла** выберите один или несколько звуковых файлов. Файлы добавляются в список воспроизведения.
-   Чтобы воспроизвести последовательность, нажмите кнопку **воспроизвести**.
-   Чтобы приостановить текущий сегмент, нажмите кнопку **приостановить**.
-   Чтобы прерывать текущий сегмент, нажмите кнопку " **Закрыть**".
-   Чтобы перейти к файлу, дважды щелкните его в списке воспроизведения.
-   Чтобы удалить файл из списка воспроизведения, выберите его в списке воспроизведения. Затем в меню **файл** выберите команду **удалить из списка воспроизведения** .
