# <a name="codecapi_avencvideoroienabled-property"></a>КОДЕКАПИ \_ авенквидеороиенаблед, свойство

Указывает, будет ли учитываться атрибут [мфсампликстенсион \_ роиректангле](mfsampleextension-roirectangle.md) , заданный для входного образца.

## <a name="data-type"></a>Тип данных

**Ulong** (VT \_ UI4)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеороиенаблед**

## <a name="remarks"></a>Remarks

По умолчанию установлено значение 0.

Если кодировка MFT кодировщика принимает ненулевое значение, предполагается, что кодировщик будет учитывать атрибут [мфсампликстенсион \_ роиректангле](mfsampleextension-roirectangle.md) , заданный во входном примере.
