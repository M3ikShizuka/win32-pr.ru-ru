# <a name="mf_sa_required_sample_count_progressive-attribute"></a>\_ \_ \_ \_ Прогрессивный атрибут подсчета ВЫБОРки для сопоставления MF \_

Указывает число выборок, которые необходимо выделить для преобразования Microsoft Media Foundation (MFT) для последовательного содержимого.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

Это значение используется, если следующий узел имеет [**имфвидеосамплеаллокатор**](/windows/desktop/api/mfidl/nn-mfidl-imfvideosampleallocator).
