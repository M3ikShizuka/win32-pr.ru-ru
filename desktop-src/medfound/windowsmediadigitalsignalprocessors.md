# <a name="digital-signal-processors"></a>Обработчики цифровых сигналов

В этом разделе описываются объекты обработчика цифровых сигналов (DSP), предоставляемые Windows.

Корпорация Майкрософт использует термин " *обработчик цифровых сигналов* " для обозначения набора COM-объектов, выполняющих преобразования для несжатых аудио-и видеоданных. Поставщики DSP, описанные в этом пакете SDK, преобразуют аудио и видео в различных форматах без сжатия.

Поставщики DSP могут использоваться самостоятельно или в сочетании с аудиокодеками и видеокодеками. За исключением схемы DSP для записи речи, каждый из перечисленных здесь DSP реализует два отдельных, но схожих интерфейса.



| Интерфейс                              | Описание                                 |
|----------------------------------------|---------------------------------------------|
| [**имфтрансформ**](/windows/desktop/api/mftransform/nn-mftransform-imftransform)   | Совместимо с Microsoft Media Foundation. |
| [**имедиаобжект**](/previous-versions/windows/desktop/api/mediaobj/nn-mediaobj-imediaobject) | Совместимо с DirectShow.                 |



 

Поставщики DSP можно настроить с помощью интерфейса [**ипропертисторе**](/windows/win32/api/propsys/nn-propsys-ipropertystore) , чтобы задать свойства. Некоторые поставщики DSP имеют дополнительные интерфейсы, устанавливающие свойства. Чтобы использовать эти интерфейсы, вызовите метод [**QueryInterface**](/windows/win32/api/unknwn/nf-unknwn-iunknown-queryinterface(q)) любого другого интерфейса DSP. В справочной статье по каждому DSP содержится список поддерживаемых свойств, интерфейсов и других функций.

Этот раздел содержит следующие подразделы.



| DSP                                                      | Описание                                          |
|----------------------------------------------------------|------------------------------------------------------|
| [DSP по интерполяции аудио](audioresampler.md)                | Преобразует частоту выборки звукового потока.       |
| [Преобразование элемента управления цветом DSP](colorcontroltransform.md) | Регулирует цветовые характеристики потока видео. |
| [DSP цветового преобразователя](colorconverter.md)                | Преобразует видеопоток между цветовыми форматами.       |
| [Средство преобразования частоты кадров DSP](framerateconverter.md)       | Изменяет частоту кадров потока видео.            |
| [DSP изменения видеоконтроллеров](videoresizer.md)                    | Изменяет размер видеопотока.                              |
| [DSP для записи речи](voicecapturedmo.md)                 | Инкапсулирует несколько DSP, связанных с захватом голоса.  |
