# <a name="mf_capture_metadata_digitalwindow-attribute"></a>\_ \_ Атрибут ДИГИТАЛВИНДОВ записи MF метаданных \_

Значение, представляющее параметры границ цифрового окна для захвата видео.

## <a name="data-type"></a>Тип данных

**ОБЪЕКТЕ**

## <a name="remarks"></a>Комментарии

Значение **большого двоичного объекта** этого свойства является структурой [дигиталвиндовсеттинг](/windows/win32/api/mfapi/ns-mfapi-digitalwindowsetting) .
