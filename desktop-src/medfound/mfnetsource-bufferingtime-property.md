# <a name="mfnetsource_bufferingtime-property"></a>МФНЕТСАУРЦЕ \_ буфферингтиме, свойство

Количество секунд, в течение которых данные сетевых источников при запуске заменяются.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

**DWORD** (хранится в формате **Long**)

VT \_ I4

**лвал**



## <a name="remarks"></a>Комментарии

Константа **мфнетсаурце \_ буфферингтиме** определяет идентификатор GUID для этого ключа свойства. Идентификатор свойства (PID) равен нулю.

Приложения могут использовать это свойство для настройки сетевого источника. Чтобы задать свойство, передайте указатель **ипропертисторе** в сопоставитель источника. Дополнительные сведения см. [в разделе Настройка источника мультимедиа](configuring-a-media-source.md).

Значение этого свойства по умолчанию равно 5 секундам.
