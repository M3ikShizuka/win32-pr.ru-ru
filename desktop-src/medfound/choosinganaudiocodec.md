# <a name="choosing-an-audio-codec-microsoft-media-foundation"></a>Выбор аудиокодека (Microsoft Media Foundation)

[**Кодировщик Windows Media Audio**](windowsmediaaudioencoder.md) предоставляет три категории кодирования звука, как показано в следующей таблице.



| Категория                         | Назначение                                                    |
|----------------------------------|------------------------------------------------------------|
| Windows Media Audio Standard     | Универсальное сжатие звука.                      |
| Windows Media Audio Professional | Сжатие звука с несколькими каналами и высокой четкости.       |
| Windows Media Audio без потерь     | Сжатие звука без потери исходных данных. |



 

Стандартная категория обеспечивает кодирование звука общего назначения, пригодное для различных сценариев воспроизведения. Например, он может сжимать звук с низкой скоростью для потоковой передачи по сети с ограниченной пропускной способностью или для отрисовки на портативных устройствах. На другом конце шкалы он может создать сжатое аудио содержимое для высокого качества воспроизведения. Выделение стандартных алгоритмов кодирования — музыка, но они подходят и для других сложных звуковых материалов. Стандартная категория должна иметь значение по умолчанию для звукового содержимого. Для удовлетворения конкретных потребностей следует использовать категории "профессиональная" и "без потерь".

Отдельный кодировщик, [**голосовой кодировщик Windows Media**](windowsmediaaudiovoiceencoder.md), обеспечивает сжатие речи.
