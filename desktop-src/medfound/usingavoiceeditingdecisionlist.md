# <a name="using-an-editing-decision-list-for-encoding-voice"></a>Использование списка решений для изменения кодировки голоса

Изменение списка решений (ЕДЛ) — это данные, предоставленные кодеку, который предоставляет сведения о том, как закодировать определенные части содержимого. голосовой кодек Windows Media Audio 9 поддерживает простой едл, в котором можно указать части содержимого, содержащие музыку. По умолчанию кодек обнаруживает фрагменты музыки при настройке для кодирования смешанного содержимого. ЕДЛ следует использовать только в том случае, если кодек неправильно обнаруживает типы содержимого.

Чтобы использовать ЕДЛ, необходимо установить для голоса кодировщика смешанное содержимое. Настройте режим работы кодека "смешанный", задав для свойства [мфпкэй \_ вмавоице \_ ENC \_ мусикспичклассмоде](mfpkey-wmavoice-enc-musicspeechclassmodeproperty.md) значение 2. Задайте ЕДЛ с помощью свойства [мфпкэй \_ вмавоице \_ ENC \_ ЕДЛ](mfpkey-wmavoice-enc-edlproperty.md) . Значением этого свойства является строка, содержащая разделенный запятыми список диапазонов времени в содержимом, которые должны быть закодированы в виде музыки. Первый элемент в списке — это версия ЕДЛ, которая всегда имеет значение 1. Вторым элементом является количество разделов музыки, описанных в списке. После второго элемента находится несколько пар значений, равных второму элементу. Каждая пара значений описывает начальную и конечную точки музыкального воспроизведения в содержимом, в миллисекундах.

Например, строка ЕДЛ "1, 4, 1000, 2000, 5000, 6000, 9000, 10000, 13000, 14000" указывает четыре типа музыкальных данных, каждый из которых имеет одну секунду. Если сведения в строке ЕДЛ недопустимы, они игнорируются.
