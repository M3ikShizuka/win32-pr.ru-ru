# <a name="mfsampleextension_multiplexed_manager-attribute"></a>\_Атрибут мфсампликстенсион мультиплексированного \_ диспетчера

Предоставляет экземпляр [**имфмуксстреамсамплеманажер**](/windows/desktop/api/mfobjects/nn-mfobjects-imfmuxstreamsamplemanager) , который используется для доступа к коллекции образцов из подпотоков мультиплексированного источника мультимедиа.

## <a name="data-type"></a>Тип данных

**[**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown)**

## <a name="remarks"></a>Remarks

Передайте это значение в [**имфаттрибутес:: ununknown**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-getunknown) , чтобы получить экземпляр [**имфмуксстреамсамплеманажер**](/windows/desktop/api/mfobjects/nn-mfobjects-imfmuxstreamsamplemanager).
