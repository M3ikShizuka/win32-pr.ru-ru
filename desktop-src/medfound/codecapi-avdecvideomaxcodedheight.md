# <a name="codecapi_avdecvideomaxcodedheight-property"></a>КОДЕКАПИ \_ авдеквидеомакскодедхеигхт, свойство

Задает максимальную высоту изображения, которую видеодекодер будет принимать в качестве входного типа.

## <a name="data-type"></a>Тип данных

**Long** (**VT \_ I4**)

## <a name="property-guid"></a>GUID свойства

КОДЕКАПИ \_ авдеквидеомакскодедхеигхт
