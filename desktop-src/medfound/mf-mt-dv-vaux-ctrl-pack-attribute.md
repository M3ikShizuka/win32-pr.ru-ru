# <a name="mf_mt_dv_vaux_ctrl_pack-attribute"></a>\_ \_ \_ Атрибут пакета MF ваукс \_ CTRL dv Book \_

Video вспомогательный (ВАУКС) пакет управления версиями в цифровом видеоролике (DV).

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

этот атрибут соответствует элементу **двдввауксктл** структуры DirectShow [**двинфо**](/windows/win32/api/strmif/ns-strmif-dvinfo) .

Константа GUID для этого атрибута экспортируется из мфууид. lib.
