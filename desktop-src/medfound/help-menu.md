# <a name="help-menu"></a>Меню «Справка»

Меню **Справка** предоставляет следующие команды:



| Команда      | Описание                                                                      |
|--------------|----------------------------------------------------------------------------------|
| **Справка...**  | Открывает справку Топоедит на веб-сайте MSDN.                                         |
| **О программе...** | Открывает диалоговое окно **About** , в котором отображаются сведения о версии топоедит. |
