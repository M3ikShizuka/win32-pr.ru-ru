# <a name="mf_audio_renderer_attribute_flags-attribute"></a>\_ \_ \_ Атрибут флагов атрибутов модуля подготовки звука \_ MF

Содержит флаги для настройки модуля подготовки звука.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

Значение этого атрибута является побитовым оператором **или** для следующих флагов.



| Значение                                                   | Описание                                                                                                                                                                                                                                                                                                                       |
|---------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ Флаги атрибута модуля подготовки звука MF \_ \_ \_ Кросспроцесс** | Модуль подготовки звука использует многопроцессный сеанс аудио. Этот флаг позволяет модулям подготовки звука в нескольких процессах использовать один и тот же сеанс звука вместе с соответствующими элементами управления томами и политиками.<br/> Если этот флаг не установлен, то сеанс звука не может совместно использоваться модулями подготовки звука в других процессах.<br/> |
| **\_ \_ Флаги атрибута модуля подготовки звука MF \_ \_ \_ не сохранены**    | API Windows audio session (васапи) не сохранит свойства для этого сеанса аудио, например том сеанса.<br/> Если этот флаг не установлен, ВАСАПИ сохранит свойства сеанса аудио.<br/>                                                                                                       |



 

Этот атрибут можно использовать для настройки модуля подготовки звука. Использование зависит от того, какая функция вызывается для создания модуля подготовки звука:

-   [**Мфкреатеаудиорендерер**](/windows/desktop/api/mfidl/nf-mfidl-mfcreateaudiorenderer): Задайте этот атрибут с помощью указателя интерфейса [**имфаттрибутес**](/windows/desktop/api/mfobjects/nn-mfobjects-imfattributes) , указанного в параметре *паудиоаттрибутес* .
-   [**Мфкреатеаудиорендерерактивате**](/windows/desktop/api/mfidl/nf-mfidl-mfcreateaudiorendereractivate): Задайте этот атрибут с помощью указателя интерфейса [**имфактивате**](/windows/desktop/api/mfobjects/nn-mfobjects-imfactivate) , полученного в параметре *ппактивате* . Задайте атрибут перед вызовом [**имфактивате:: активатеобжект**](/windows/desktop/api/mfobjects/nf-mfobjects-imfactivate-activateobject).

Константа GUID для этого атрибута экспортируется из мфууид. lib.
