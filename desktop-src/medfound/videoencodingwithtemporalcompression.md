# <a name="video-encoding-with-temporal-compression-microsoft-media-foundation"></a>Кодирование видео с временным сжатием (Microsoft Media Foundation)

для достижения максимально возможного коэффициента сжатия Windows видеокодеки мультимедиа используют *временное сжатие*. Временное сжатие — это способ уменьшения размера сжатого видео без кодирования каждого кадра в виде полного образа. Полностью закодированные кадры (как и статическое изображение) называются *опорными кадрами*. Все остальные кадры в видео представлены данными, указывающими изменение с момента последнего кадра. Эти кадры называются *разностными кадрами*.

Объем сжатия, который можно достичь с помощью временного сжатия, зависит от содержимого. Если видео статичное, без большого количества движений или других изменений в изображении, для каждого ключевого кадра можно создать множество разностных кадров. Чем больше использованных разностных кадров, тем меньше закодированного видеопотока. Однако если видео является динамическим, с большим количеством движений или изменением цветов, преимущество использования временного сжатия меньше.
