# <a name="mebufferingstopped-event"></a>Событие Мебуфферингстоппед

Сигнализирует, что источник мультимедиа прекратил буферизацию данных.

Источник мультимедиа отправляет его, когда прекращает буферизацию данных после отправки события [мебуфферингстартед](mebufferingstarted.md) .

Потоковые потоки, которые реализуют интерфейс [**имфбитестреамбуфферинг**](/windows/desktop/api/mfidl/nn-mfidl-imfbytestreambuffering) , также отправляют это событие.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |



## <a name="remarks"></a>Комментарии

Когда сеанс мультимедиа получает это событие, он перезапускает часы презентации. Сеанс мультимедиа также перенаправляет событие в приложение.
