# <a name="mfpkey_dyn_bandtrunc_qceil-property"></a>МФПКЭЙ \_ Дин \_ бандтрунк \_ кцеил, свойство

Определяет качество, при котором разрешена Минимальная пропускная способность. С качеством, хуже или равной значению этого свойства, Допускается Минимальная пропускная способность.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

Доступно только с помощью [**ипропертисторе**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Тип данных

**VT \_ R4**
