# <a name="mf_mt_d3d12_texture_layout-attribute"></a>\_ \_ \_ Атрибут макета для текстуры MF MT D3D12 \_

Указывает параметры макета текстуры, которые использовались для создания связанных ресурсов Direct3D.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="remarks"></a>Remarks

Значение этого атрибута соответствует члену перечисления [D3D12_TEXTURE_LAYOUT](/windows/win32/api/d3d12/ne-d3d12-d3d12_texture_layout) .
