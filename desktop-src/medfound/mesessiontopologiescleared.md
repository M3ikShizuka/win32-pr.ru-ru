# <a name="mesessiontopologiescleared-event"></a>Событие Месессионтопологиесклеаред

Вызывается сеансом мультимедиа, когда метод [**имфмедиасессион:: клеартопологиес**](/windows/desktop/api/mfidl/nf-mfidl-imfmediasession-cleartopologies) завершается асинхронно.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |
