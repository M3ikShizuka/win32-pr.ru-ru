# <a name="tools-menu"></a>Меню «Сервис»

Меню **Сервис** предоставляет следующие команды:




| Команда | Описание | 
|---------|-------------|
| <strong>Выбранный узел Spy</strong> | Записывает все вызовы методов для выбранного узла преобразования или выходного узла и сохраняет результаты в текстовый файл. Дополнительные сведения см. в разделе <a href="logging-topology-node-activities.md">Регистрация действий узла топологии</a> . | 
| <strong>Пользовательский Тополоадер...</strong> | Позволяет пользователю предоставить пользовательский загрузчик топологии.<br /><blockquote>[!Note]<br />этот компонент добавлен для Windows 7.</blockquote><br /> |
