# <a name="mfpkey_rmax-property"></a>МФПКЭЙ \_ рмакс, свойство

Указывает пиковую скорость потока (в битах в секунду), используемую для воспроизведения с ограничением в 2 прохода с переменной скоростью (VBR).

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкмаксбитрате

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

Значение по умолчанию отсутствует.

## <a name="remarks"></a>Remarks

Это значение представляет пиковую скорость воспроизведения. Значение параметра [мфпкэй \_ бмакс](mfpkey-bmaxproperty.md) используется для описания буфера с точки зрения этой скорости. фактически, ограниченная скорость действия аналогична постоянной скорости потока (CBR) с использованием этого значения в качестве скорости. Однако поток с ограничением VBR можно воспроизводить с более низкой скоростью, пока буфер увеличивается.

Это значение необходимо задать для многоэтапной кодировки VBR с ограничением в максимальной нагрузке. После начала обработки образцов вы не должны запрашивать это значение до завершения кодирования потока. Кодировщик интерпретирует запрос этого значения как сигнал, над которым находится сеанс кодирования. Следующий образец, который вы обрабатываете, рассматривается как начало нового сеанса.

Обычно это значение составляет от двух до трех раз больше значения параметра [мфпкэй \_ равг](mfpkey-ravgproperty.md).
