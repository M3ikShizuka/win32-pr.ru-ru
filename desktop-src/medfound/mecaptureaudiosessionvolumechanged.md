# <a name="mecaptureaudiosessionvolumechanged-event"></a>Событие Мекаптуреаудиосессионволумечанжед

Отправляется источником аудиозаписи при изменении тома.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE               | Описание                           |
|-----------------------|---------------------------------------|
| VT \_ пуст <br/> | Нет данных события.<br/> <br/> |



## <a name="remarks"></a>Remarks

Это событие отправляется потоком мультимедиа источника записи звука.

Источник записи звука отправляет это событие при изменении тома внешним действием, например, если пользователь изменяет том с помощью панели управления. Источник записи не отправляет событие, если приложение изменяет том непосредственно в источнике.
