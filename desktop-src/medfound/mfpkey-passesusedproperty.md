# <a name="mfpkey_passesused-property"></a>МФПКЭЙ \_ пассесусед, свойство

Указывает число проходов, которые кодировщик будет использовать для кодирования содержимого. Чтение и запись.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвкпассесусед

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

1

## <a name="remarks"></a>Remarks

Это значение не может превышать значение [мфпкэй \_ пассесрекоммендед](mfpkey-passesrecommendedproperty.md).
