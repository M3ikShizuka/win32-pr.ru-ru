# <a name="codec-objects"></a>Объекты кодека

В этом разделе описываются кодировщики и декодеры, поддерживаемые в Microsoft Media Foundation. Этот раздел содержит следующие подразделы.




| Раздел | Описание | 
|-------|-------------|
| <a href="aac-decoder.md"><strong>Декодер AAC</strong></a> | Звуковой декодер, который декодирует следующие расширенные профили (AAC) и высокоэффективные AAC (HE-AAC):<ul><li>Профиль низкой сложности MPEG-2 AAC (LC) (многоканальный).</li><li>MPEG-4 HE-AAC v1 (многоканальный) с AAC-LC Core.</li><li>MPEG-4 HE-AAC v2 (стерео) с AAC-LC Core.</li></ul> | 
| <a href="aac-encoder.md"><strong>Кодировщик AAC</strong></a> | Звуковой кодировщик, который кодирует расширенный профиль с низким уровнем сложности (AAC). | 
| <a href="dv-video-decoder.md"><strong>Видеодекодер DV</strong></a> | Видеодекодер, который декодирует видео в DV-коде. | 
| <a href="h-264-video-decoder.md"><strong>Декодер видео H. 264</strong></a> | Видеодекодер, декодированный в видео H. 264. Он поддерживает декодирование базовых, главных и высококачественных профилей вплоть до уровня 5,1. | 
| <a href="h-264-video-encoder.md"><strong>Кодировщик видео H. 264</strong></a> | Видеокодировщик, который кодирует видео H. 264. Он поддерживает следующие профили H. 264:<ul><li>Базовый профиль</li><li>Профиль Main</li></ul> | 
| <a href="h-264-video-decoder.md"><strong>Декодер видео H. 265/HEVC</strong></a> | Видеодекодер, который поддерживает декодирование содержимого H. 265/HEVC в формате приложение б и может использоваться при воспроизведении файлов MP4 и M2TS. Он поддерживает следующие профили H. 264:<ul><li>Профиль Main</li><li>Главный профиль 10</li><li>Неподвижный основной рисунок</li></ul> | 
| <a href="h-264-video-encoder.md"><strong>H. 265/HEVC видео Encoder</strong></a> | Видеокодировщик, который кодирует формат H. 265/HEVC. Он поддерживает следующие профили H. 265:<ul><li>Профиль Main</li></ul> | 
| <a href="mpeg4part2videodecoder.md"><strong>Видеодекодер (часть 2) памяти MPEG4</strong></a> | Видеодекодер MPEG-4 Part 2. | 
| <a href="windowsmediaaudiodecoder.md"><strong>Windows Декодер мультимедиа</strong></a> | декодер, который декодирует потоки, закодированные кодировщиком Windows Media Audio. | 
| <a href="windowsmediaaudioencoder.md">Windows Media Audio Encoder</a> | Звуковой кодировщик, поддерживающий три категории закодированных выходных данных:<ul><li>Standard</li><li>Professional</li><li>Lossless</li></ul>Стандартная категория предназначена для кодирования звука общего назначения. категория Professional предназначена для кодирования звука с несколькими каналами или высокой четкости. Категория без потерь предназначена для сжатия звука без потери исходных данных. | 
| <a href="windowsmediaaudiovoicedecoder.md"><strong>Windows Декодер мультимедиа аудио</strong></a> | декодер, который декодирует потоки, закодированные с помощью Windows Media Audio Voice. | 
| <a href="windowsmediaaudiovoiceencoder.md">Windows Кодировщик Media Audio Voice</a> | Кодировщик для кодирования звука, содержащего в основном речь. | 
| <a href="windows-media-mp3-decoder.md"><strong>Windows Декодер MP3 Media</strong></a> | Звуковой декодер, декодированный в следующих звуковых форматах.<ul><li>ISO/IEC 11172-3 (аудио MPEG-1), уровень 3</li><li>Стандарт ISO/IEC 13818-3 (MPEG-2 Audio), низкое расширение частоты выборки</li></ul> | 
| <a href="windowsmediampeg4decoder.md"><strong>Windows Декодер Media MPEG4 v1/v2</strong></a> | Декодер, который реализует декодирование видео MPEG-4 v1/v2. | 
| <a href="windows-media-video-7-and-8-encoders.md"><strong>Windows Кодировщик Media Video 7/8</strong></a> | видеокодировщик, реализующий предыдущие версии кодировщика видео Windows Media. | 
| <a href="windowsmediavideo9decoder.md">Windows Декодер Media Video 9</a> | видеодекодер, который декодирует потоки, закодированные кодировщиком Windows Media video 9. | 
| <a href="windowsmediavideo9encoder.md">Windows Кодировщик Media Video 9</a> | Видеокодировщик, поддерживающий три категории выходных данных ендодед:<ul><li>Windows Media Video 9</li><li>Windows Расширенный профиль Media Video 9</li><li>Windows Изображение Media Video 9,1</li></ul>категория Windows Media Video 9 предназначена для кодирования видео общего назначения. Категория расширенный профиль предназначена для кодирования видео или видео с высокой степенью четкости, которое соответствует стандарту VC-1 Advanced Profile SMPTE Standard. Категория изображений предназначена для преобразования растровых изображений в динамические видео. | 
| <a href="windowsmediavideo9screendecoder.md"><strong>Windows Экранный декодер видео 9</strong></a> | декодер, который декодирует потоки, закодированные с помощью экранного кодировщика Windows Media видео 9. | 
| <a href="windowsmediavideo9screenencoder.md"><strong>Windows Кодировщик экрана видео в Media 9</strong></a> | Кодировщик для кодирования компьютера — видео приложения (снимок экрана) или другое очень статическое видео. |
