# <a name="media-session-attributes"></a>Атрибуты сеанса мультимедиа

Для настройки сеанса мультимедиа используются следующие атрибуты.



| attribute                                                                                            | Описание                                                                                                 |
|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**\_ \_ \_ диспетчер защиты содержимого сеанса \_ MF**](mf-session-content-protection-manager-attribute.md) | Предоставляет интерфейс обратного вызова для приложения, получающего объект включения содержимого из сеанса PMP. |
| [**\_ \_ глобальное время сеанса \_ MF**](mf-session-global-time-attribute.md)                                | Указывает, будут ли топологии иметь глобальное время начала и окончания.                                        |
| [**\_ \_ Диспетчер качества сеансов \_ MF**](mf-session-quality-manager-attribute.md)                        | Содержит идентификатор CLSID диспетчера качества для сеанса мультимедиа.                                              |
| [**\_ \_ Удаленный \_ исходный режим сеанса \_ MF**](mf-session-remote-source-mode-attribute.md)                 | Указывает, что источник мультимедиа выполняется в удаленном процессе.                                             |
| [**\_ \_ контекст сервера сеансов \_ MF**](mf-session-server-context-attribute.md)                          | Позволяет двум экземплярам сеанса мультимедиа совместно использовать один и тот же процесс PMP.                                   |
| [**\_тополоадер сеанс \_ MF**](mf-session-topoloader-attribute.md)                                   | Содержит идентификатор CLSID для загрузчика топологии для сеанса мультимедиа.                                              |
