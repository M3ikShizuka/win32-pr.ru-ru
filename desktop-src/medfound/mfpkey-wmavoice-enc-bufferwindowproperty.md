# <a name="mfpkey_wmavoice_enc_bufferwindow-property"></a>МФПКЭЙ \_ вмавоице \_ ENC, \_ свойство буффервиндов

Указывает окно буфера (в миллисекундах), которое будет использоваться для речевого кодека.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмаквоицебуффер

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

1500 для 4, 5, 8, 12 и 16 кбит/с

2177 в течение 10 и 20 кбит/с
