# <a name="mesourceseeked-event"></a>Событие Месаурцесикед

Возникает, когда источник мультимедиа выполняет поиск в новой должности. Источник мультимедиа вызывает это событие, если источник запущен или приостановлен, а приложение вызывает [**имфмедиасаурце:: Start**](/windows/desktop/api/mfidl/nf-mfidl-imfmediasource-start) с временем начала, которое не равно текущей позицией.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE           | Описание                                                                |
|-------------------|----------------------------------------------------------------------------|
| VT \_ i8<br/> | Новая начальная позицией в единицах измерения 100-наносекундных.<br/> <br/> |
