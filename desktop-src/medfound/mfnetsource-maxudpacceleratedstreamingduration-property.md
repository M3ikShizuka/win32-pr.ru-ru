# <a name="mfnetsource_maxudpacceleratedstreamingduration-property"></a>МФНЕТСАУРЦЕ \_ максудпакцелератедстреамингдуратион, свойство

Максимальная продолжительность ускоренной потоковой передачи (в миллисекундах), когда сетевой источник использует транспорт UDP.



Тип данных

Тип ПРОПВАРИАНТ (VT)

ПРОПВАРИАНТ, элемент

**DWORD** (хранится в формате **Long**)

VT \_ I4

**лвал**



## <a name="remarks"></a>Комментарии

Константа **мфнетсаурце \_ максудпакцелератедстреамингдуратион** определяет идентификатор GUID для этого ключа свойства. Идентификатор свойства (PID) равен нулю.

Для транспорта UDP это свойство переопределяет свойство [**мфнетсаурце \_ акцелератедстреамингдуратион**](mfnetsource-acceleratedstreamingduration-property.md) , если значение этого свойства меньше.

Приложения могут использовать это свойство для настройки сетевого источника. Чтобы задать свойство, передайте указатель **ипропертисторе** в сопоставитель источника. Дополнительные сведения см. [в разделе Настройка источника мультимедиа](configuring-a-media-source.md).

Значение этого свойства по умолчанию равно 8 000 миллисекундам.
