# <a name="meendofpresentationsegment-event"></a>Событие Миндофпресентатионсегмент

Вызывается источником Sequencer при завершении сегмента, за которым следует другой сегмент. По завершении последнего сегмента источник Sequencer вызывает событие [миндофпресентатион](meendofpresentation.md) .

Сеанс мультимедиа перенаправляет это событие в приложение.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE              | Описание                           |
|----------------------|---------------------------------------|
| VT \_ пуст<br/> | Нет данных события.<br/> <br/> |



## <a name="attributes"></a>Атрибуты

Для этого события определены следующие атрибуты.



| attribute                                                                                               | Описание                                                                          |
|---------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**\_ \_ топология источника событий MF \_ \_ отменена**](mf-event-source-topology-canceled-attribute.md)<br/> | Указывает, отменил ли источник Sequencer этот сегмент.<br/> <br/> |
