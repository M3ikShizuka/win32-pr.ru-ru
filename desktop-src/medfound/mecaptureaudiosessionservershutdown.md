# <a name="mecaptureaudiosessionservershutdown-event"></a>Событие Мекаптуреаудиосессионсервершутдовн

Отправляется источником записи звука, когда сеанс записи звука отключается из-за завершения работы сервера аудио.

## <a name="event-values"></a>Значения событий

Возможные значения, полученные из [**имфмедиаевент:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) , включают следующее.



| VARTYPE               | Описание                           |
|-----------------------|---------------------------------------|
| VT \_ пуст <br/> | Нет данных события.<br/> <br/> |
