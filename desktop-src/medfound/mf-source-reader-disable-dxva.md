# <a name="mf_source_reader_disable_dxva-attribute"></a>\_Чтение источника \_ MF \_ Отключить \_ атрибут дксва

Указывает, включает ли [модуль чтения исходного](source-reader.md) экрана ускорение видео DirectX (дксва) для видеодекодера.

## <a name="data-type"></a>Тип данных

**UINT32**

## <a name="getset"></a>Get/Set

Чтобы получить этот атрибут, вызовите метод [**имфаттрибутес:: UInt32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-getuint32).

Чтобы задать этот атрибут, вызовите [**имфаттрибутес:: SetUINT32**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-setuint32).

## <a name="remarks"></a>Комментарии

Этот атрибут применяется, если выполняются следующие условия.

-   Модуль чтения исходного кода декодирует поток видео.
-   Видеодекодер поддерживает декодирование ДКСВА.
-   Приложение использует атрибут [ \_ \_ \_ \_ диспетчера D3D источника MF](mf-source-reader-d3d-manager.md) для установки [Диспетчер устройств Direct3D](direct3d-device-manager.md) в модуле чтения исходного кода.

Этот атрибут позволяет приложению отключить ДКСВА, продолжая декодироваться на поверхности Direct3D.

По умолчанию модуль чтения исходного кода использует [Диспетчер устройств Direct3D](direct3d-device-manager.md) в двух целях:

-   Включение декодирования ДКСВА в видеодекодере.
-   Для выделения поверхностей Direct3D для видеороликов.

Если значение \_ \_ атрибута дксва чтения источника MF \_ \_ равно **true**, модуль чтения исходного кода отключает декодирование дксва, хотя он по-прежнему использует [Диспетчер устройств Direct3D](direct3d-device-manager.md) для выделения поверхностей Direct3D.

Если атрибут [ \_ \_ \_ \_ диспетчера D3D источника данных MF](mf-source-reader-d3d-manager.md) не ЗАДАН, то \_ модуль чтения источника MF \_ \_ disable \_ дксва не учитывается.

Значение этого атрибута по умолчанию равно **false**, означающее, что при наличии включен декодирование дксва.
