# <a name="mf_mt_video_rotation-attribute"></a>\_ \_ Атрибут поворота видео MF MT \_

Задает поворот видеокадра в направлении против часовой стрелки.

## <a name="data-type"></a>Тип данных

**Мфвидеоротатионформат** хранится как **UINT32**

## <a name="remarks"></a>Remarks

Видео с карманного устройства, например мобильного телефона, часто поворачивается на 90, 180 или 270 градусов. Если камера сохраняет ориентацию в виде метаданных в видеофайл, образ можно настроить во время воспроизведения.

Если этому атрибуту присвоено значение **мфвидеоротатионформат \_ 90**, это означает, что содержимое повернуто на 90 градусов в направлении по часовой стрелке. Если содержимое было повернуто на 90 градусов по часовой стрелке, значение атрибута будет равно **мфвидеоротатионформат \_ 270**.

Поддерживаемые значения для этого атрибута перечислены в [**мфвидеоротатионформат**](/windows/desktop/api/mfapi/ne-mfapi-mfvideorotationformat).

Константа GUID для этого атрибута экспортируется из мфууид. lib.
