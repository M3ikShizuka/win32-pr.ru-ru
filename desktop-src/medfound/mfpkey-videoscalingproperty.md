# <a name="mfpkey_videoscaling-property"></a>МФПКЭЙ \_ видеоскалинг, свойство

Указывает, будет ли кодек использовать оптимизацию масштабирования видео.

## <a name="constant-for-ipropertybag"></a>Константа для Ипропертибаг

g \_ всзвмвквидеоскалинг

## <a name="data-type"></a>Тип данных

VT \_ I4

## <a name="default-value"></a>Значение по умолчанию

0

## <a name="remarks"></a>Комментарии

Масштабирование видео — это тип оптимизации искусственного, который может улучшить качество визуального изображения в кодировке с низкой скоростью. Оптимизация масштабирования видео сокращает произносит производимые артефакты, но может пожертвовать деталями изображения. Эта оптимизация влияет на диапазон яркости в кодированном видео, как описано ниже, но не влияет на физическое разрешение выходного видео.

Для этого свойства можно задать одно из следующих значений.



| Значение | Описание                                                                                                           |
|-------|-----------------------------------------------------------------------------------------------------------------------|
| 0     | Масштабирование видео не будет использоваться.                                                                                       |
| 1     | Кодировщик будет использовать консервативную оптимизацию масштабирования видео. Диапазон яркости будет масштабироваться от 0... 255 до 26... 229. |
| 2     | Кодировщик будет использовать агрессивную оптимизацию масштабирования видео. Диапазон яркости будет масштабироваться от 0... 255 до 31... 224.   |
