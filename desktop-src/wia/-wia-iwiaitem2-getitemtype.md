# <a name="iwiaitem2getitemtype-method"></a>Метод IWiaItem2:: ItemType

Возвращает сведения о типе элемента.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetItemType(
  [out] LONG *pItemType
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*питемтипе* \[ заполняет\]
</dt> <dd>

Тип: **Long \** _

Получает указатель на _ *Long**, содержащий сочетание флагов типа. См. раздел [**Флаги типа элемента WIA**](-wia-wia-item-type-flags.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

Если этот метод завершается успешно, возвращается значение **S \_ ОК**. В противном случае возвращается код ошибки **HRESULT** .

## <a name="remarks"></a>Комментарии

Каждый объект [**IWiaItem2**](-wia-iwiaitem2.md) в иерархическом дереве объектов, связанных с аппаратным устройством получения образа Windows (WIA) 2,0, имеет конкретный тип данных. Объекты элементов представляют собой папки и файлы. Папки содержат объекты File. Объекты файлов содержат данные, полученные устройством, например изображениями и звуками. Этот метод позволяет приложениям обнаруживать тип любого элемента в иерархическом дереве объектов элементов на устройстве.

Элемент может иметь более одного типа. Например, элемент, представляющий звуковой файл, будет иметь атрибуты Type [**виаитемтипеаудио**](-wia-wia-item-type-flags.md) \| **виаитемтипефиле**.
