# <a name="wiadeviceinfopropertyid"></a>виадевицеинфопропертид

Свойства сведений об устройстве — это свойства, описывающие установку и установку устройства. Эти свойства доступны через интерфейс [**ивиадевмгр**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiadevmgr) , а также через корневой элемент.

свойства сведений об устройстве начинаются с префикса "WIA \_ DIP \_ " (свойство сведений об устройстве) при использовании в C/C++ и предоставляются Windows получения образа (WIA). Для сценариев эти константы используют префикс "DeviceInfo" и являются частью перечислимого типа Виадевицеинфопропертид. Дополнительные сведения см. в разделе [**константы свойств сведений об устройстве**](-wia-wiadeviceinfoprop.md) .
