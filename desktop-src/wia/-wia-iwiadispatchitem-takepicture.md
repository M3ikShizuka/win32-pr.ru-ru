# <a name="itemtakepicture-method"></a>Item. Такепиктуре, метод

Метод **такепиктуре** объекта [**Item**](-wia-item.md) заставляет устройство цифровой камеры взять изображение и возвратить объект **элемента** , представляющий полученный образ. Этот метод применим только к устройствам цифровой камеры.

## <a name="syntax"></a>Синтаксис


```JScript
retVal = Item.TakePicture()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **ивиадиспатчитем**

Объект [**Item**](-wia-item.md) , представляющий изображение, которое создает этот метод.
