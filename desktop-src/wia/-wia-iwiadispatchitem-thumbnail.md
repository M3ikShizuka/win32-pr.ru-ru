# <a name="itemthumbnail-property"></a>Свойство Item. эскиза

Путь и имя файла эскиза изображения. Это свойство имеет **значение NULL** , если элемент не поддерживает эскизы, или если не удается построить путь. Только для чтения.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Item.Thumbnail
```



## <a name="property-value"></a>Значение свойства

Строка, которая получает путь и имя файла.
