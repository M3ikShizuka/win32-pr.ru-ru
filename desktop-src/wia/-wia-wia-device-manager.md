# <a name="wia-device-manager"></a>диспетчер устройств WIA

Диспетчер устройств получения образа Windows (WIA) является расширением монитора событий по прежнему образу (сти). Диспетчер устройств WIA предоставляет объекты, методы и интерфейсы для следующих объектов:

-   Установка устройств
-   Перечисление устройств
-   Запрос свойств установленных устройств
-   Создание объектов устройств
-   Наблюдение за событиями устройства
-   Получение образов
-   Регистрация конечных приложений.
