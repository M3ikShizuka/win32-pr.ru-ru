# <a name="item-object"></a>Объект Item

Windows Аппаратные устройства для получения изображений (WIA) представлены в виде иерархических деревьев объектов **элементов** . Корневой элемент в этом дереве представляет само устройство, а дочерние элементы — изображения, папки или сканирование кроватях.

Используйте объект **Item** для перемещения данных в файл, перехода по дереву элементов для конкретного устройства или для получения сведений об образе или устройстве.

## <a name="members"></a>Элементы

Объект **Item** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)
