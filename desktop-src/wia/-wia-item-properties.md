# <a name="item-properties"></a>Свойства элементов

Свойства элемента — это свойства, управляющие функцией элемента "получение образа Windows" (WIA). Доступ к этим свойствам осуществляется через любой элемент дерева, представляющего устройство. Существует три типа свойств элементов: Common, сканер и Camera.

Справочные сведения об этих свойствах см. в следующих статьях:

-   [**Стандартные константы свойств элементов WIA**](-wia-wiaitempropcommonitem.md)
-   [**Константы свойства элемента WIA камеры**](-wia-wiaitempropcameraitem.md)
-   [**Константы свойства элемента сканера WIA**](-wia-wiaitempropscanneritem.md)
