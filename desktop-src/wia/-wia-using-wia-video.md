# <a name="using-wia-video"></a>Использование видео WIA

WIA предоставляет интерфейс [**ивиавидео**](/windows/desktop/api/Wiavideo/nn-wiavideo-iwiavideo) , позволяющий приложениям отображать потоковые видео в окне, записывать изображения из видеопотока и сохранять эти изображения как JPEG-файлы.

в следующих разделах объясняется, как использовать возможности видео о получении изображений (WIA) Windows.

-   [Предварительный просмотр видео](-wia-previewing-video.md)
-   [Запись изображения с изображением по-прежнему из потока видео](-wia-capturing-a-still-image-from-a-video-stream.md)

> [!Note]  
> WIA не поддерживает видео устройства в Windows Server 2003, Windows Vista или более поздней версии. для этих версий Windows используйте [DirectShow](/previous-versions//ms783323(v=vs.85)) для получения изображений из видео.
