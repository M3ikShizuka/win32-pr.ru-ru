# <a name="wia_raw_header-structure"></a>\_Структура заголовка необработанного WIA \_

структура **\_ \_ заголовков WIA RAW** определяет изображение в формате необработанных данных устройства и позволяет приложениям использовать формат raw в Windows передачи изображений (WIA).

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _WIA_RAW_HEADER {
  DWORD Tag;
  DWORD Version;
  DWORD HeaderSize;
  DWORD XRes;
  DWORD YRes;
  DWORD XExtent;
  DWORD YExtent;
  DWORD BytesPerLine;
  DWORD BitsPerPixel;
  DWORD ChannelsPerPixel;
  DWORD DataType;
  BYTE  BitsPerChannel[8];
  DWORD Compression;
  DWORD PhotometricInterp;
  DWORD LineOrder;
  DWORD RawDataOffset;
  DWORD RawDataSize;
  DWORD PaletteOffset;
  DWORD PaletteSize;
} WIA_RAW_HEADER;
```

## <a name="remarks"></a>Remarks

Так как это не формат файла, используйте пустую строку для \_ \_ Свойства расширения файла WIA IPA \_ .

Палитра и данные могут находиться в любом порядке.

**Равдатасизе** не содержит ни заголовок, ни палитру. Используйте это поле для проверки успешности перемещения образа.

**Палеттесизе** — байт, а не число записей в палитре.
