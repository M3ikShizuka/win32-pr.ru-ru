# <a name="iwiaitem2-interface"></a>Интерфейс IWiaItem2

Интерфейс **IWiaItem2** предоставляет приложения с теми же функциями, что и интерфейс [**ивиаитем**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) (возможность запрашивать устройства для обнаружения их возможностей, доступа к интерфейсам и свойствам элементов, а также для управления устройством). он также предоставляет приложению возможность динамически создавать и использовать фильтры обработки изображений, которые могут быть расширениями драйверов устройств Windowsного получения изображений (WIA) 2,0, предоставляемых в Windows Vista.

## <a name="members"></a>Элементы

## <a name="remarks"></a>Remarks

Дерево элементов WIA 2,0, которое может видеть приложение, отличается от дерева, созданного и обслуживаемого WIA 2,0 минидривер. Когда минидривер создает дерево элементов, служба WIA 2,0 использует это дерево элементов WIA 2,0 в качестве инструкции по созданию идентичных копий, которые можно просматривать с помощью приложений для работы с образами. Элементы в скопированном дереве называются элементами приложения. Элементы в дереве, созданном минидривер, называются элементами драйвера. в Windows Vista деревья элементов WIA 2,0 построены на основе объектов **IWiaItem2** , каждый из которых реализует интерфейс **IWiaItem2** .

Интерфейс **IWiaItem2** , как и все интерфейсы модели компонентных объектов (com), наследует методы интерфейса [IUnknown](/windows/win32/api/unknwn/nn-unknwn-iunknown) .



| Методы IUnknown                                        | Описание                               |
|---------------------------------------------------------|-------------------------------------------|
| [IUnknown:: QueryInterface](/windows/win32/api/unknwn/nf-unknwn-iunknown-queryinterface(q)) | Возвращает указатели на поддерживаемые интерфейсы. |
| [IUnknown:: AddRef](/windows/win32/api/unknwn/nf-unknwn-iunknown-addref)                 | Увеличивает значение счетчика ссылок.               |
| [IUnknown:: Release](/windows/win32/api/unknwn/nf-unknwn-iunknown-release)               | Уменьшает значение счетчика ссылок.               |
