# <a name="iwiaapperrorhandlerreportstatus-method"></a>Метод Ивиаапперрорхандлер:: Репортстатус

Обрабатывает состояние устройства и сообщения об ошибках во время передачи данных изображения и отображает сообщения для пользователя.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT ReportStatus(
  [in] LONG      lFlags,
  [in] IWiaItem2 *pWiaItem2,
  [in] HRESULT   hrStatus,
  [in] LONG      lPercentComplete
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

</dt> <dd>

Тип: **Long**

Не используется. Задайте значение 0.

</dd> <dt>

*pWiaItem2* \[ окне\]
</dt> <dd>

Тип: **[**IWiaItem2**](-wia-iwiaitem2.md) \** _

Указатель на передаваемый элемент.

</dd> <dt>

_hrStatus * \[ в\]
</dt> <dd>

Тип: **HRESULT**

Код состояния устройства.

</dd> <dt>

</dt> <dd>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

Возвращает *хрстатус* , если восстановление из ошибки невозможно. В противном случае возвращается одно из следующих значений.



| Код возврата                                                                                              | Описание                                                                                                                                                                                                                                        |
|----------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                     | Если *хрстатус* является ошибкой, для исправления ошибки предпринимается соответствующее действие, а перемещение может быть продолжено. Если *хрстатус* является информационным, пользователь получил уведомление о немодальном диалоговом окне и отказался от отмены перемещения.<br/> |
| <dl> <dt>**S \_ false**</dt> </dl>                  | Пользователь отменил перемещение из немодального диалогового окна обработчика ошибок. Это значение может возвращаться в любой момент, независимо от того, что *хрстатус* . <br/>                                                                                      |
| <dl> <dt>**\_состояние WIA \_ не \_ обработано**</dt> </dl> | Никаких действий не выполнено; Это значит, что пользователю не было предоставлено диалоговое окно. Будет вызван следующий обработчик ошибок. Порядок обработчиков ошибок: приложение, драйвер и система по умолчанию.<br/>                                                 |



 

## <a name="remarks"></a>Комментарии

Параметр *лперценткомплете* позволяет отображать ход выполнения в окне обработчика ошибок. Например, драйвер может предоставить оценку того, сколько времени занимает "прогрев". Параметр *лперценткомплете* , передаваемый в **Ивиаапперрорхандлер:: репортстатус** , имеет то же значение, что и **лперценткомплете** , который драйвер задает в структуре [**виатрансферпарамс**](-wia-wiatransferparams.md) .

Обработчик ошибок может использовать макросы Success и FAILed, чтобы выяснить, имеет ли *хрстатус* ошибку серьезности \_ или степень серьезности \_ .

Если *хрстатус* имеет серьезность \_ успеха, пользователю следует разрешить отменять перемещение.

Если *хрстатус* является ошибкой серьезности \_ , обработчик ошибок должен отобразить модальное диалоговое окно, принадлежащее родительскому окну приложения.
