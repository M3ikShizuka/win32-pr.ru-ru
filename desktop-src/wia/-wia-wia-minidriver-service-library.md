# <a name="wia-minidriver-service-library"></a>Библиотека WIA Минидривер Service

чтобы максимально упростить реализацию драйвера, Windows получения изображений (WIA) предоставляет библиотеку минидривер Service Library, которая выполняет многие задачи администрирования, необходимые для архитектуры WIA. Библиотека служб предоставляет вспомогательные функции для обслуживания дерева объектов [интерфейса ивиадрвитем](https://msdn.microsoft.com/library/ms793976.aspx) устройства.

Основные функции библиотеки службы выполняют вспомогательные функции, которые в противном случае должны быть реализованы большинством драйверов устройств. Эти функции считывают и записывают свойства. Они также создают объекты элементов драйвера и копируют свойства сведений об устройстве.
