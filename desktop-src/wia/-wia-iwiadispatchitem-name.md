# <a name="itemname-property"></a>Item.Name, свойство

Имя элемента, отображаемое в пользовательском интерфейсе. Только для чтения.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Item.Name
```



## <a name="property-value"></a>Значение свойства

Строка, которая получает имя.
