# <a name="itemthumbwidth-property"></a>Свойство Item. Сумбвидс

Ширина (в пикселях) эскиза изображения. Это свойство возвращает значение-1, если этот элемент не поддерживает эскизы. Только для чтения.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Item.ThumbWidth
```



## <a name="property-value"></a>Значение свойства

Переменная, которая получает ширину.
