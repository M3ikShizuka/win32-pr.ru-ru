# <a name="image-intent-constants"></a>Константы намерения образа

Константы намерения образа определяют тип данных, которые должно представлять изображение. Свойство [**сканирования \_ IP-адресов с \_ \_ намерением "WIA**](-wia-wiaitempropscanneritem.md) " использует эти флаги. Чтобы обеспечить намерение, объедините предустановленный флаг типа изображения с нужным флагом размера или качества с помощью оператора или. \_Цель WIA \_ None не должна сочетаться с другими флагами. Обратите внимание, что изображение не может одновременно иметь оттенки серого и цвета.

Ниже приведены допустимые константы намерения образа.



| Предназначенные флаги типа образа           | Описание                                  |
|-------------------------------------|----------------------------------------------|
| \_Цель WIA \_ None                   | Значение по умолчанию. Не предустановки никаких свойств. |
| \_ \_ \_ Цвет типа изображения с намерением WIA \_     | Предустановленные свойства для цветового содержимого.         |
| \_ \_ тип изображения с НАМЕРЕНием WIA \_ \_ | Предустановленные свойства для содержимого в градациях серого.     |
| \_ \_ \_ текст типа изображения с намерением WIA \_      | Предустановленные свойства для текстового содержимого.          |
| \_ \_ \_ маска типа изображения с намерением WIA \_      | Маска для всех флагов типа изображения.        |



 



| Предполагаемый размер изображения/флаги качества | Описание                                  |
|-----------------------------------|----------------------------------------------|
| Цель WIA — \_ \_ Минимальный \_ Размер       | Предустановленные свойства для сворачивания размера изображения.    |
| \_развертывание с намерением WIA \_ \_    | Предустановленные свойства для повышения качества изображения. |
| \_Маска размера намерений WIA \_ \_           | Маска для всех флагов размера и качества.      |
| \_ \_ лучшее \_ предварительное ознакомление с намерением WIA        | Задает наилучшее качество предварительной версии.          |
