# <a name="installwiadevice-function"></a>Функция Инсталлвиадевице

Функция **инсталлвиадевице** устанавливает устройство для получения образа Windows (WIA) в качестве устройства с корневым перечислением. Это может привести к появлении предупреждения системы безопасности, если любой устанавливаемый файл или установщик не имеют цифровой подписи и не являются доверенными.

## <a name="syntax"></a>Синтаксис


```C++
DWORD WINAPI InstallWiaDevice(
  _In_ PWIADEVICEINSTALL *pWiaDeviceInstall
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пвиадевицеинсталл* \[ окне\]
</dt> <dd>

Тип: **пвиадевицеинсталл \** _

Указатель на структуру ВИАДЕВИЦЕИНСТАЛЛ. Для элемента _szFriendlyName * структуры необходимо задать фактическое значение FriendlyName для устройства.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **DWORD**

Если функция завершается успешно, возвращается значение ошибки \_ Success.

Если функция завершается ошибкой, возвращается код ошибки Win32.
