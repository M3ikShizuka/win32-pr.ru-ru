# <a name="syntax-conventions"></a>Синтаксические обозначения

В следующей таблице перечислены термины, используемые в инструкциях синтаксиса для Справочника по модели сценариев для получения образа Windows (WIA). 

| Термин                      | Описание                                                    |
|---------------------------|----------------------------------------------------------------|
| *Имя файла*                | Путь и имя файла.                                   |
| *обждевицеинфо*           | Объект [**DeviceInfo**](-wia-deviceinfo.md) .                |
| *обждевицеинфоколлектион* | Коллекция объектов [**DeviceInfo**](-wia-deviceinfo.md) . |
| *обжитем*                 | Объект [**Item**](-wia-item.md) .                           |
| *обжитемколлектион*       | Коллекция объектов [**Item**](-wia-item.md) .             |
| *обжвиа*                  | Объект [**WIA**](-wia-wia.md) .                              |
| *PropID*                  | Константа свойства WIA.                                       |
| *var*                     | Значение **типа Variant** .                                           |
