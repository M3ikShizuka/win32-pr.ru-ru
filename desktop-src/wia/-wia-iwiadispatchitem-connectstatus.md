# <a name="itemconnectstatus-property"></a>Свойство Item. Коннектстатус

Возвращает состояние подключения устройства. Это свойство применяется только к элементам типа "устройство" (корневые элементы). Возможные значения: "Connected", "disconnected" или **null** (если это свойство не применяется к элементу). Только для чтения.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Item.ConnectStatus
```



## <a name="property-value"></a>Значение свойства

Строка, которая получает состояние подключения.
