# <a name="deviceinfouiclsid-property"></a>DeviceInfo. Уиклсид, свойство

Получает идентификатор класса пользовательского интерфейса, предоставленного производителем для этого аппаратного устройства получения образа Windows (WIA). Значение представляет собой строковое представление GUID.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = DeviceInfo.UIClsid
```



## <a name="property-value"></a>Значение свойства

Строка для получения идентификатора класса.
