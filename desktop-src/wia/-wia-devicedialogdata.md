# <a name="devicedialogdata-structure"></a>Структура ДЕВИЦЕДИАЛОГДАТА

Определяет данные, необходимые для вызова диалогового окна устройства.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct {
  DWORD    cbSize;
  HWND     hwndParent;
  IWiaItem *pIWiaItemRoot;
  DWORD    dwFlags;
  LONG     lIntent;
  LONG     lItemCount;
  IWiaItem **ppWiaItem;
} DEVICEDIALOGDATA;
```
