# <a name="deviceinfoport-property"></a>DeviceInfo. Port, свойство

Извлекает порт, к которому подключено устройство для получения образа Windows (WIA).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = DeviceInfo.Port
```



## <a name="property-value"></a>Значение свойства

Строка, получающая порт.
