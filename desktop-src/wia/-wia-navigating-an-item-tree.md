# <a name="navigating-an-item-tree"></a>Навигация по дереву элементов

Приложение переходит к дереву элементов устройства для получения изображений (WIA), чтобы найти и выбрать образы на этом устройстве. С помощью этого метода приложение выбирает и получает изображение, не выполняя показ пользователю диалогового окна.

Устройство WIA представляется как корневой элемент в дереве объектов [**ивиаитем**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) или [**IWiaItem2**](-wia-iwiaitem2.md) . Дочерние элементы в дереве представляют изображения для камеры или сканирования кроватях для сканера.

После выбора устройства WIA приложение вызывает метод [**ивиаитем:: енумчилдитемс**](/windows/desktop/api/wia_xp/nf-wia_xp-iwiaitem-enumchilditems) интерфейса [**ивиаитем**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) или [**IWiaItem2**](-wia-iwiaitem2.md) устройства, чтобы перечислить дочерние элементы (Images или Scan кроватях) устройства. Инструкции см. [в разделе Выбор устройства](-wia-selecting-a-device.md).

Метод [**ивиаитем:: енумчилдитемс**](/windows/desktop/api/wia_xp/nf-wia_xp-iwiaitem-enumchilditems) создает объект перечисления для дочерних элементов устройства и возвращает указатель на интерфейс [**иенумвиаитем**](/windows/desktop/api/wia_xp/nn-wia_xp-ienumwiaitem) этого объекта. Затем приложение может использовать методы интерфейса **иенумвиаитем** для получения указателей на элементы в дереве элементов устройства.
