# <a name="property-attributes-wia"></a>Атрибуты свойств (WIA)

Все объекты элементов имеют свойства. Свойства имеют атрибуты. Например, атрибуты свойства указывают, считывается ли свойство из, записывается в или удаляется. Они также указывают допустимые значения свойств. Следующие константы являются допустимыми атрибутами свойств: 

| Атрибут свойства        | Значение                                                                                                  |
|---------------------------|----------------------------------------------------------------------------------------------------------|
| \_ \_ кэширование свойства WIA      | Устройство может кэшировать значение свойства.                                                               |
| \_флаг prop для WIA \_           | Свойство содержит список допустимых значений флагов. Значения флагов объединяются с помощью побитовой операции **или** . |
| \_список свойств \_ WIA           | Свойство содержит список допустимых значений.                                                                 |
| WIA \_ prop \_ None           | С этим свойством не связаны допустимые значения.                                          |
| \_диапазон prop для WIA \_          | Свойство имеет диапазон допустимых значений.                                                                |
| чтение свойства "WIA" \_ \_           | Приложение может считывать значение свойства.                                                           |
| WIA \_ prop \_ RW             | Приложение может считывать и записывать значение свойства.                                                 |
| \_ \_ требуется синхронизация свойства "WIA" \_ | Не используйте.                                                                                              |
| запись свойства "WIA \_ prop" \_          | Приложение может записать значение свойства.                                                          |
