# <a name="deviceinfoname-property"></a>DeviceInfo.Name, свойство

имя аппаратного устройства для Windowsного получения изображений (WIA), которое отображается в пользовательском интерфейсе.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = DeviceInfo.Name
```



## <a name="property-value"></a>Значение свойства

Строка, которая получает имя.
