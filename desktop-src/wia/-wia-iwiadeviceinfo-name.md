# <a name="deviceinfoname-property"></a>DeviceInfo.Name, свойство

Имя аппаратного устройства для получения образа Windows (WIA), отображаемое в пользовательском интерфейсе.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = DeviceInfo.Name
```



## <a name="property-value"></a>Значение свойства

Строка, которая получает имя.

|-------------------------------------|---------------------------------------------------------------------------------------------------------------|
| Минимальная версия клиента<br/> | Windows 2000 Professional, только для \[ классических приложений Windows XP\]<br/>                                        |
| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2003\]<br/>                                                          |
