# <a name="wia-20-item-category-guids"></a>Идентификаторы GUID категории элементов WIA 2,0

Windows Элементы для получения изображений (WIA) 2,0 группируются в категории, определяющие, как будет обрабатываться или использовать [**IWiaItem2**](-wia-iwiaitem2.md) . Например, если элемент представляет устройство подачи, то приложение должно содержать необходимые свойства устройства подачи документов и действовать как устройство подачи документов. Если элемент представляет готовый файл, то приложение WIA 2,0 должно обрабатывать его таким образом, предполагая, что данные статичны и расположены на устройстве. (Правила для каждого элемента будут определены в отдельных документах спецификации.) Каждая категория имеет константу типа GUID.



| Константа                                                                                                                                                                                               | Описание                                                                                                                                                                    |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WIA_CATEGORY_AUTO"></span><span id="wia_category_auto"></span><dl> <dt>**\_Автоматическая категория \_ WIA**</dt> </dl>                             | Элемент, представляющий параметры автоматической настройки для устройства со сканером WIA 2,0, поддерживающего автоматическую настройку сканирования.<br/>                                   |
| <span id="WIA_CATEGORY_FEEDER"></span><span id="wia_category_feeder"></span><dl> <dt>**\_податчик категории WIA \_**</dt> </dl>                       | Элемент веб-канала, который является программируемым источником данных и соответствует стандартным правилам и наборам свойств, необходимым для поддержки податчиков документов.<br/>                             |
| <span id="WIA_CATEGORY_FEEDER_BACK"></span><span id="wia_category_feeder_back"></span><dl> <dt>**\_ \_ обратный подачи категории WIA \_**</dt> </dl>       | Программируемый источник данных, описывающий источник данных обратной стороны для дуплексного элемента податчика.<br/>                                                                       |
| <span id="WIA_CATEGORY_FEEDER_FRONT"></span><span id="wia_category_feeder_front"></span><dl> <dt>**\_ \_ Передняя сторона подачи категории \_ WIA**</dt> </dl>    | Программируемый источник данных, описывающий источник данных переднего плана для дуплексного элемента податчика.<br/>                                                                      |

| <span id="WIA_CATEGORY_ROOT"></span><span id="wia_category_root"></span><dl> <dt>**\_корень категории \_ WIA**</dt> </dl>                             | Корневой элемент для устройства. <br/>                                                                                                                                      |
