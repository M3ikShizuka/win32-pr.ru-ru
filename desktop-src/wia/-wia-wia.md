# <a name="wia-object"></a>WIA, объект

объект **Wia** является точкой входа для всех функций создания скриптов Windows изображений (Wia). Любое приложение, использующее модель скриптов WIA, должно создавать объект [**сафевиа**](-wia-safewia.md) или **WIA** . Используйте этот объект для перечисления и создания устройств и получения уведомлений о событиях оборудования.

> [!Note]  
> Этот объект не является надежным для скриптов. Версию этого объекта, которая является надежной для скриптов, см. в разделе [**сафевиа**](-wia-safewia.md).
