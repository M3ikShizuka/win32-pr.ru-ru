# <a name="itemfirmwareversion-property"></a>Свойство Item. Фирмвареверсион

Извлекает версию встроенного по устройства. Это свойство применяется только к элементам типа "устройство" (корневые элементы). Только для чтения.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Item.FirmwareVersion
```



## <a name="property-value"></a>Значение свойства

Строка, которая получает версию встроенного по.
