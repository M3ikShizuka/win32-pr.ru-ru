# <a name="itempictureheight-property"></a>Свойство Item. Пиктурехеигхт

Высота изображений, созданных этой цифровой камерой (в пикселях). Применяется только к цифровым камерам. Только для чтения.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Item.PictureHeight
```



## <a name="property-value"></a>Значение свойства

Переменная, которая получает высоту.
