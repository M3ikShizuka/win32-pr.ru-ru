# <a name="itemwidth-property"></a>Свойство Item. Width

Ширина (в пикселях) элемента. Только для чтения.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Item.Width
```



## <a name="property-value"></a>Значение свойства

Переменная, которая получает ширину.

## <a name="remarks"></a>Комментарии

Если элемент является цифровой камерой, свойство **Width** представляет ширину изображений, формируемых камерой. Если элемент является сканером, это свойство представляет ширину средства сканирования. Если элемент является изображением, это свойство представляет фактическую ширину изображения.
