# <a name="itemthumbheight-property"></a>Свойство Item. Сумбхеигхт

Высота эскиза изображения в пикселях. Это свойство возвращает значение-1, если этот элемент не поддерживает эскизы. Только для чтения.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Item.ThumbHeight
```



## <a name="property-value"></a>Значение свойства

Переменная, которая получает высоту эскиза.
