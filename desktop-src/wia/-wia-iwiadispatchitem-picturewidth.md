# <a name="itempicturewidth-property"></a>Свойство Item. Пиктуревидс

Ширина (в пикселях) изображений, созданных этой цифровой камерой. Применяется только к цифровым камерам. Только для чтения.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Item.PictureWidth
```



## <a name="property-value"></a>Значение свойства

Переменная, которая получает ширину.
