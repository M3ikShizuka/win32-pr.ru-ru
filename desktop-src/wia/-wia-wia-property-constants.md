# <a name="wia-property-constants"></a>Константы свойств WIA

Константы свойств для свойства "получение образа Windows" (WIA) включают следующие группы констант, поддерживаемые Windows Millennium Edition (Windows Me), Windows XP и более поздних версий.

-   [**Константы свойств сведений об устройстве**](-wia-wiadeviceinfoprop.md)
-   [**Общие константы свойств устройства**](-wia-wiaitempropcommondevice.md)
-   [**Константы свойств устройства сканера**](-wia-wiaitempropscannerdevice.md)
-   [**Стандартные константы свойств элементов WIA**](-wia-wiaitempropcommonitem.md)
-   [**Константы свойства элемента сканера WIA**](-wia-wiaitempropscanneritem.md)

В Windows Me, Windows XP и Windows Server 2003 поддерживаются следующие группы констант. но не поддерживаются в Windows Vista и более поздних версиях.

-   [**Константы свойств устройства камеры**](-wia-wiaitempropcameradevice.md)
-   [**Константы свойства элемента WIA камеры**](-wia-wiaitempropcameraitem.md)

В Windows Me и Windows XP поддерживается следующая группа констант. но не поддерживается в Windows Server 2003 и более поздних версиях.

-   [**Константы свойств устройства WIA видео**](-wia-wiaitempropvideodevice.md)
