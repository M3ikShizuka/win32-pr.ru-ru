# <a name="wia-device-type-specifiers"></a>Описатели типа устройства WIA

Windows Спецификаторы типов устройств для получения изображений (WIA) — это константы, указывающие тип устройства, подключенного к компьютеру пользователя. Используйте макрос GET \_ стидевице \_ Type для получения этих значений из свойства типа устройства WIA. Ниже приведены допустимые спецификаторы типа устройства WIA. 

| Тип устройства                 | Значение                                                                                                                     |
|-----------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| стидевицетипедефаулт        | Универсальное устройство WIA. Во время перечислений устройств Эта константа используется для перечисления всех устройств WIA.                         |
| стидевицетипедигиталкамера  | Устройство является камерой. *этот описатель не поддерживается* Windows Vista *и более поздних версий.*                                     |
| стидевицетипесканнер        | Устройство является сканером.                                                                                                    |
| стидевицетипестреамингвидео | Устройство содержит потоковую передачу видео. *этот описатель не поддерживается* Windows Server 2003 *,* Windows Vista *или более поздней версии.* |
