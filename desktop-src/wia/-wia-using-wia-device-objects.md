# <a name="using-wia-device-objects"></a>Использование объектов устройств WIA

Устройство обработки изображений представляется в виде иерархического дерева объектов элементов WIA (интерфейсов [**ивиаитем**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) ). Как правило, корень этого дерева элементов представляет само устройство, а другие элементы в дереве представляют изображения, для камеры или для сканирования регионов для сканера.

Приложения используют диспетчер устройств WIA для создания и перечисления устройств WIA. В следующих разделах объясняется, как создавать и использовать устройства WIA:

-   [Выбор устройства](-wia-selecting-a-device.md)
-   [Устройства камеры WIA](-wia-wia-camera-devices.md)
-   [Устройства сканера WIA](-wia-wia-scanner-devices.md)
