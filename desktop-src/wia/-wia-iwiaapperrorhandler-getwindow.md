# <a name="iwiaapperrorhandlergetwindow-method"></a>Метод Ивиаапперрорхандлер:: onwindow

Получает обработчик для диалогового окна, отображающего сообщения об ошибках, и предоставляет одну или несколько кнопок для продолжения, отмены или прерывания работы приложения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetWindow(
  [out] HWND *phwnd
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ФВНД* \[ заполняет\]
</dt> <dd>

Тип: **HWND\***

HWND, используемый обработчиком ошибок приложения, обработчиком ошибок драйвера и обработчиком ошибок по умолчанию для диалоговых окон сообщений об устройствах (ошибки и информационные сообщения). Выходное значение может быть **равно NULL**.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

Если этот метод завершается успешно, возвращается значение **S \_ ОК**. В противном случае возвращается код ошибки **HRESULT** .

## <a name="remarks"></a>Remarks

*фвнд* указывает на окно, передаваемое в [**репортстатус**](-wia-iwiaerrorhandler-reportstatus.md) с помощью прокси-сервера Windowsного образа (WIA) 2,0. Это окно должно оставаться действительным на время пересылки данных.
