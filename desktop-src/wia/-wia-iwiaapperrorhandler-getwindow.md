# <a name="iwiaapperrorhandlergetwindow-method"></a>Метод Ивиаапперрорхандлер:: onwindow

Получает обработчик для диалогового окна, отображающего сообщения об ошибках, и предоставляет одну или несколько кнопок для продолжения, отмены или прерывания работы приложения.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetWindow(
  [out] HWND *phwnd
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ФВНД* \[ заполняет\]
</dt> <dd>

Тип: **HWND \** _

HWND, используемый обработчиком ошибок приложения, обработчиком ошибок драйвера и обработчиком ошибок по умолчанию для диалоговых окон сообщений об устройствах (ошибки и информационные сообщения). В качестве выходного значения может быть задано значение _ * NULL * *.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

Если этот метод завершается успешно, возвращается значение **S \_ ОК**. В противном случае возвращается код ошибки **HRESULT** .

## <a name="remarks"></a>Комментарии

*ФВНД* указывает на окно, передаваемое в [**Репортстатус**](-wia-iwiaerrorhandler-reportstatus.md) с помощью прокси-сервера получения образа Windows (WIA) 2,0. Это окно должно оставаться действительным на время пересылки данных.
