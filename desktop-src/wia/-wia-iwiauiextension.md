# <a name="iwiauiextension-interface"></a>Интерфейс Ивиауиекстенсион

Интерфейс **ивиауиекстенсион** предоставляет методы, которые заменяют системный интерфейс по умолчанию, предоставляют пользовательский логотип точечного рисунка устройства и предоставляют пользовательский значок устройства.

## <a name="members"></a>Элементы

Интерфейс **ивиауиекстенсион** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Ивиауиекстенсион** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивиауиекстенсион** содержит следующие методы.



| Метод                                                                  | Описание                                                                                  |
|:------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------|
| [**девицедиалог**](-wia-iwiauiextension-devicedialog.md)               | Предоставляет настраиваемый пользовательский интерфейс, который заменяет стандартный системный интерфейс пользователя.<br/> |
| [**жетдевицебитмаплого**](-wia-iwiauiextension-getdevicebitmaplogo.md) | Возвращает пользовательский логотип точечного рисунка для устройства.<br/>                                         |
| [**жетдевицеикон**](-wia-iwiauiextension-getdeviceicon.md)             | Возвращает пользовательский значок устройства.<br/>                                                        |
