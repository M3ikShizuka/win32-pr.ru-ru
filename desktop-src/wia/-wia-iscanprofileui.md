# <a name="iscanprofileui-interface"></a>Интерфейс Исканпрофилеуи

Интерфейс **исканпрофилеуи** позволяет приложениям отображать диалоговое окно, чтобы пользователи могли создавать, изменять и удалять профили сканирования.

## <a name="members"></a>Элементы

Интерфейс **исканпрофилеуи** наследует от интерфейса [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) . **Исканпрофилеуи** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **исканпрофилеуи** содержит следующие методы.



| Метод                                                             | Описание                                                                                      |
|:-------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------|
| [**сканпрофиледиалог**](-wia-iscanprofileui-scanprofiledialog.md) | Отображает диалоговое окно, позволяющее пользователям создавать, изменять и удалять профили сканирования.<br/> |



 

## <a name="remarks"></a>Remarks

Диалоговое окно является модальным; пользователь не может переключиться на родительское окно, пока диалоговое окно не закроется.
