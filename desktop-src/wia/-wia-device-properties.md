# <a name="device-properties-wia"></a>Свойства устройства (WIA)

Свойства устройства — это свойства, управляющие функцией всего устройства. Доступ к этим свойствам возможен только через корневой элемент дерева, представляющего устройство. Существует три типа свойств устройства: Common, сканер и камера.

Справочные сведения см. в следующих статьях:

-   [**Общие константы свойств устройства**](-wia-wiaitempropcommondevice.md)
-   [**Константы свойств устройства камеры**](-wia-wiaitempropcameradevice.md)
-   [**Константы свойств устройства сканера**](-wia-wiaitempropscannerdevice.md)
