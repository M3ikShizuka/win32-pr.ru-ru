# <a name="commands-design-basics"></a>Команды (основы разработки)

> [!NOTE]
> Это руководство по проектированию было создано для Windows 7 и не Обновлено для более новых версий Windows. Многие рекомендации по-прежнему применяются в принципе, но презентация и примеры не соответствуют нашим [текущим руководствам по проектированию](/windows/uwp/design/).

Команды — это действия, которые могут выполнять пользователи при использовании приложения. Ознакомьтесь с рекомендациями по добавлению команд в меню, ленты и панели инструментов приложения.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                   | Описание                                                                                                                                                                                                                        |
|-----------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Меню](cmd-menus.md)<br/>       | Меню представляют собой иерархические списки команд или параметров, доступных пользователям в текущем контексте.<br/>                                                                                                                          |
| [Ленты](cmd-ribbons.md)<br/>   | Ленты — это современный способ, позволяющий пользователям быстро находить, анализировать и использовать команды с минимальным количеством щелчков, но при этом не нужно обращаться к справке.<br/> |
| [Панели инструментов](cmd-toolbars.md)<br/> | Панели инструментов — это способ группировки команд для эффективного доступа.<br/>                                                                                                                                                              |
