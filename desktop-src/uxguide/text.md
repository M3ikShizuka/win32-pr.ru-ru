# <a name="text-design-basics"></a>Текст (основы разработки)

> [!NOTE]
> это руководство по проектированию было создано для Windows 7 и не обновлялось для более новых версий Windows. Многие рекомендации по-прежнему применяются в принципе, но презентация и примеры не соответствуют нашим [текущим руководствам по проектированию](/windows/uwp/design/).

Текст включает в себя любой текст, который пользователи могут видеть в вашем приложении. Ознакомьтесь с этими рекомендациями по использованию текста, стиля и тона пользовательского интерфейса для приложений.

Тон предназначен для создания конкретного ответа или распознавания эмоций в модуле чтения; Он создает индивидуального пользователя, который может привлекать или стоек пользователей.<br/> |
