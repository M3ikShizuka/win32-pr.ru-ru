# <a name="windows-design-basics"></a>Windows (основы проектирования)

> [!NOTE]
> это руководство по проектированию было создано для Windows 7 и не обновлялось для более новых версий Windows. Многие рекомендации по-прежнему применяются в принципе, но презентация и примеры не соответствуют нашим [текущим руководствам по проектированию](/windows/uwp/design/).

Windows являются основными областями "холсты" или пользовательского интерфейса для классического приложения, включая основные окна и всплывающие окна, диалоги и мастера. При принятии решения о том, какую поверхность использовать и как лучше ее использовать, следуйте этим рекомендациям.
