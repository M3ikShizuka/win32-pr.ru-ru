# <a name="next-hop-flags"></a>Флаги следующего прыжка

## <a name="next-hop-state-flags"></a>Флаги состояния следующего прыжка



| Константа                     | Значение | Описание                              |
|------------------------------|-------|------------------------------------------|
| RTM \_ — \_ состояние " \_ создано" | 0     | Указывает, что был создан следующий прыжок. |
| \_состояние RTM \_ \_ удалено | 1     | Указывает, что следующий прыжок был удален. |



 

## <a name="next-hop-flags"></a>Флаги следующего прыжка



| Константа                    | Значение  | Описание                                                                                                                                           |
|-----------------------------|--------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| RTM \_ — \_ Флаги \_ удаленных | 0x0001 | Следующий прыжок указывает на удаленное назначение, которое недоступно напрямую. Чтобы получить полный путь, клиент должен выполнить рекурсивный Уточняющий запрос. |
| RTM \_ , \_ Флаги NEXTHOP \_   | 0x0002 | Этот флаг зарезервирован для использования в будущем.                                                                                                                 |

| Константа                  | Значение | Описание                 |
|---------------------------|-------|-----------------------------|
| RTM \_ , \_ изменение \_ новой версии | 0x01  | Создан новый следующий прыжок. |
