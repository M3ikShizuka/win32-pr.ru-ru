# <a name="client"></a>Клиент

Клиент RTMv2 является либо экземпляром протокола маршрутизации, либо компонентом управления, который регистрируется и взаимодействует с диспетчером таблиц маршрутизации с помощью API RTMv2. Если не указано иное, любая ссылка на клиент диспетчера таблиц маршрутизации ссылается на клиента RTMv2.

Клиент должен выполняться в том же процессе, что и диспетчер маршрутизаторов.
