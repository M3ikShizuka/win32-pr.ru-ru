# <a name="about-routing-protocol-interface"></a>О интерфейсе протокола маршрутизации

В следующей документации описывается интеграция протоколов маршрутизации сторонних производителей в службу маршрутизации и удаленного доступа (RRAS). RRAS — это функция Windows, которая выступает в качестве маршрутизатора с несколькими протоколами. RRAS определяет интерфейс между диспетчером маршрутизаторов и протоколом маршрутизации. Протокол маршрутизации реализуется в библиотеке динамической компоновки (DLL).

Используйте этот интерфейс для реализации протоколов маршрутизации, таких как ИГРП, НЛСП и BGP, в качестве библиотек DLL пользовательского режима, работающих с RRAS.
