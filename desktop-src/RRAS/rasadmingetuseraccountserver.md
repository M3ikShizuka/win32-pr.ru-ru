# <a name="rasadmingetuseraccountserver-function"></a>Функция Расадминжетусераккаунтсервер

\[эта функция предоставляется только для обеспечения обратной совместимости с Windows NT Server 4,0. он возвращает \_ вызов ошибки \_ \_ , не реализованный на сервере Windows 2003. Приложения должны использовать функцию [**мпрадминжетпдксервер**](/windows/desktop/api/Mprapi/nf-mprapi-mpradmingetpdcserver) .\]

Функция **расадминжетусераккаунтсервер** извлекает имя сервера, на котором находится база данных учетных записей пользователей. Используйте возвращенное имя сервера в функциях [**расадминусержетинфо**](rasadminusergetinfo.md) и [**расадминусерсетинфо**](rasadminusersetinfo.md) , чтобы получить или задать сведения об указанном пользователе.

## <a name="syntax"></a>Синтаксис


```C++
DWORD RasAdminGetUserAccountServer(
  _In_  const WCHAR  *lpszDomain,
  _In_  const WCHAR  *lpszServer,
  _Out_       LPWSTR lpszUserAccountServer
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*лпсздомаин* \[ окне\]
</dt> <dd>

Указатель на строку в Юникоде, заканчивающуюся **нулем,** которая указывает имя домена, к которому принадлежит RAS-сервер. Этот параметр имеет **значение NULL** для приложений администрирования RAS, выполняемых на рабочих станциях или серверах, которые не являются членами домена. Если этот параметр имеет **значение NULL**, параметр *Лпсзсервер* должен иметь значение, отличное от **null**.

</dd> <dt>

*лпсзсервер* \[ окне\]
</dt> <dd>

Указатель на строку **в Юникоде**, заканчивающуюся нулем и указывающую имя сервера RAS. Укажите имя с начальными \\ \\ символами в формате: \\ \\ *ServerName*. Этот параметр может иметь **значение NULL** , если параметр *лпсздомаин* имеет **значение**, отличное от NULL.

</dd> <dt>

*лпсзусераккаунтсервер* \[ заполняет\]
</dt> <dd>

Указатель на буфер, который получает строку в Юникоде, заканчивающуюся **нулем**, которая указывает имя контроллера домена с базой данных учетных записей пользователей. Буфер должен быть достаточно большим, чтобы вместить имя сервера (УНКЛЕН + 1). Функция добавляет префикс к возвращенному имени сервера с ведущими \\ \\ символами "" в формате \\ \\ *ServerName*.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается успешно, возвращается значение ошибки \_ Success.

Если функция завершается ошибкой, возвращаемое значение может быть следующим кодом ошибки.



| Значение                                                                                                    | Значение                                                     |
|----------------------------------------------------------------------------------------------------------|-------------------------------------------------------------|
| <dl> <dt>**Ошибка \_ недопустимого \_ параметра**</dt> </dl> | *Лпсздомаин* и *Лпсзсервер* имеют **значение NULL**.<br/> |



 

Расширенные сведения об ошибке для этой функции отсутствуют. не вызывайте [**GetLastError**](/windows/desktop/api/errhandlingapi/nf-errhandlingapi-getlasterror).

## <a name="remarks"></a>Remarks

Функция **расадминжетусераккаунтсервер** получает имя сервера с базой данных учетных записей пользователей. Для этой функции требуется имя сервера RAS или имя домена, в котором находится сервер удаленного доступа.

Параметр *лпсздомаин* должен указывать допустимое имя домена. Этот параметр имеет **значение NULL** для приложений администрирования RAS, работающих на серверах, которые не являются членами домена (например, сервер находится в собственной рабочей группе). В этом случае в параметре *лпсзсервер* должно быть указано имя сервера. Чтобы получить имя сервера, вызовите функцию [**GetComputerName**](/windows/win32/api/winbase/nf-winbase-getcomputernamea) . Обязательно добавьте в имя сервера \\ \\ символы "".

Если имя сервера, указанное параметром *лпсзсервер* , является изолированным сервером (то есть сервер или рабочая станция не является членом домена), то имя сервера возвращается в буфер *лпсзусераккаунтсервер* .

Затем используйте имя сервера учетных записей пользователей в вызове функции [**неткуеридисплайинформатион**](/windows/win32/api/lmaccess/nf-lmaccess-netquerydisplayinformation) , чтобы перечислить пользователей в базе данных учетных записей пользователей.
