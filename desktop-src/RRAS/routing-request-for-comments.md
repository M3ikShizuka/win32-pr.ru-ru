# <a name="routing-request-for-comments"></a>Запрос на маршрутизацию для комментариев

Запросы на комментарии (RFC) — это непрерывная серия документов, которые обсуждают технологии Интернета и Интернета. Конкретные документы RFC идентифицируются по числу. Следующие RFC приведены в разделе Служба удаленного доступа (RAS) пакета SDK платформы. Дополнительные сведения о документах RFC и о том, как найти копии определенного документа RFC, см. в разделе [https://www.rfc-editor.org/rfc.html](https://www.rfc-editor.org/rfc.html) .

-   [RFC 791](https://www.ietf.org/rfc/rfc791.txt) – Internet Protocol, спецификация протокола Интернет-программы дарпа, 1981 сентября
-   [RFC 2863](https://www.ietf.org/rfc/rfc2863.txt) — группа интерфейсов MIB, 2000 июня
-   [RFC 1354](https://www.ietf.org/rfc/rfc1354.txt) — таблица MIB таблицы пересылки IP, июль, 1992
