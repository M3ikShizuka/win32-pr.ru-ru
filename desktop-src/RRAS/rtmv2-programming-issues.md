# <a name="rtmv2-programming-issues"></a>Проблемы программирования RTMv2

Функции RTMv2 написаны с помощью следующих допущений.

-   Функции RTMv2 не распределяют память для клиента. Клиент всегда должен выделить память.
-   При отмене регистрации клиента он должен выполнять операции очистки, такие как освобождение всей выделенной памяти.
-   Клиенты должны правильно освобождать дескрипторы; утечки памяти могут возникать, если клиент не следит за этой практикой.
