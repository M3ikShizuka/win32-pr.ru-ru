# <a name="ras-interfaces"></a>Интерфейсы RAS

Интерфейс представляет сеть, к которой можно подключиться через адаптер ЛВС или WAN. Каждый интерфейс имеет уникальный идентификатор маршрутизатора. Активные интерфейсы имеют адаптер, обеспечивающий подключение к сети, которую они представляют. Неактивные интерфейсы не имеют адаптера, предоставляющего возможность подключения, если только администратор не отключил интерфейс после того, как он уже имел адаптер.

Маршрутизация пакета в сеть, представленная интерфейсом, приведет к тому, что маршрутизатор выделит адаптер для этого интерфейса и установит подключение к удаленной сети через ГЛОБАЛЬную сеть. Выделение адаптера для интерфейса называется *привязкой*.

Интерфейсы являются управляемыми объектами. Каждый интерфейс отображается в виде строки в таблице Interface соответствующей SNMP MIB.
