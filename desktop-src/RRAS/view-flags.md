# <a name="view-flags"></a>Просмотреть флаги

Используйте флаги представления для управления представлениями таблицы маршрутизации.



| Константа                 | Значение      | Описание                                                      |
|--------------------------|------------|------------------------------------------------------------------|
| \_Максимальная \_ длина адреса \_ RTM | 16         | Максимальный размер адреса для семейства адресов.                          |
| \_Максимальное число \_ просмотров RTM          | 32         | Максимальное число представлений, которые могут быть активными в таблице маршрутизации. |
| \_идентификатор представления \_ RTM \_ укаст     | 0          | Указывает представление одноадресной рассылки.                                        |
| \_идентификатор представления \_ RTM \_ мкаст     | 1          | Указывает представление многоадресной рассылки.                                      |
| \_ \_ размер маски представления \_ RTM    | 0x20       | Указывает максимальное число представлений, которые можно настроить.    |
| \_Маска представления RTM — \_ \_ нет    | 0x00000000 | Возвращает сведения независимо от представления.                      |
| \_Маска представления \_ RTM \_ любая     | 0x00000000 | Возвращает назначения из всех представлений. Это значение по умолчанию.  |
| \_ \_ укаст маски представления \_ RTM   | 0x00000001 | Возвращает назначения из представления одноадресной рассылки.                      |
| \_ \_ мкаст маски представления \_ RTM   | 0x00000002 | Возвращает назначения из представления многоадресной рассылки.                    |
| ОКОНЧАТЕЛЬНая \_ \_ Маска \_ представления     | 0xFFFFFFFF | Возвращает сведения из всех представлений.                              |
