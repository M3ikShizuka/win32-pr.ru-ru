# <a name="destinations"></a>Места назначения

Назначение в таблице маршрутизации — это сетевая запись, представленная сетевым адресом и маской сети.

Конечная запись в таблице маршрутизации включает:

-   Адрес, выраженный в виде сетевого адреса и маски сети.
-   Список маршрутов к назначению.
-   Список областей непрозрачных указателей.
-   Представления, в которых это назначение является допустимым. Назначение содержит структуру для каждого представления, которое содержит следующие сведения:
    -   Идентификатор представления.
    -   Указатель на оптимальный маршрут к назначению в этом представлении.
    -   Владелец лучшего маршрута в этом представлении.
    -   Флаги, связанные с лучшим маршрутом в этом представлении.
    -   Обработчик для всех маршрутов, которые находятся в состоянии удержания в этом представлении.
