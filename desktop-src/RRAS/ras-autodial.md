# <a name="ras-autodial"></a>Автонабор номера RAS

В системе Windows XP был добавлен компонент «Подключение по умолчанию». Если попытка подключения не удалась, система проверяет явное совпадение (для имени Winsock или имени общего ресурса) в базе данных, если это не так, при наличии подключения по умолчанию.

**Windows 2000:** Если попытка подключения к сетевому адресу завершается сбоем из-за недоступности узла, функция автоматического набора номера может автоматически запустить операцию подключения удаленного доступа. Для этого необходимо выполнить автоматический автонабор номеров для поиска в базе данных сетевых адресов, чтобы найти запись телефонной книги, которую можно использовать для установки подключения.

**Windows NT 4,0:  ** Служба RAS сохраняет базу данных имен Winsock и (или) общих имен, которые были успешно достигнуты при активном подключении удаленного доступа или VPN-подключения. Позже, когда попытка подключения к Winsock или общей папке завершается неудачно, система проверяет эту базу данных и предлагает установить сохраненное подключение.
