# <a name="ras-server-administration"></a>Администрирование сервера удаленного доступа

Windows NT Server 4,0 предоставляет набор функций для администрирования разрешений пользователей и портов на серверах RAS Windows NT/Windows 2000. Windows 95 не поддерживает эти функции. С помощью этих функций можно разработать приложение удаленного администрирования сервера для выполнения следующих задач.

-   Перечисление пользователей, имеющих указанный набор разрешений RAS
-   Назначение или Отмена разрешений RAS для указанного пользователя
-   Перечисление настроенных портов на сервере удаленного доступа
-   Получение сведений и статистики об указанном порте на сервере удаленного доступа
-   Сброс счетчиков статистики для указанного порта
-   Отключение указанного порта

Кроме того, можно установить библиотеку DLL администрирования сервера RAS для аудита подключений пользователей и назначения IP-адресов пользователям, входящим в их набор. Библиотека DLL экспортирует набор функций, которые сервер удаленного доступа вызывает каждый раз, когда пользователь пытается подключиться или отключиться.
