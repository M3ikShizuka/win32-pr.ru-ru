# <a name="support-functions-reference"></a>Справочник по функциям поддержки

В диспетчере маршрутизаторов предоставляются следующие функции для маршрутизации протоколов. Когда диспетчер маршрутизатора вызывает функцию [**стартпротокол**](/windows/desktop/api/Routprot/nc-routprot-pstart_protocol) (реализованную протоколом маршрутизации), диспетчер маршрутизатора передает в протокол маршрутизации структуру [**\_ функций поддержки**](/windows/desktop/api/Routprot/ns-routprot-support_functions_50) , содержащую указатели на эти функции:

[**деманддиалрекуест**](/previous-versions/windows/desktop/legacy/aa373924(v=vs.85))

[**мибентрикреате**](/previous-versions/windows/desktop/legacy/aa374538(v=vs.85))

[**мибентриделете**](/previous-versions/windows/desktop/legacy/aa374539(v=vs.85))

[**мибентрижет**](/previous-versions/windows/desktop/legacy/aa374540(v=vs.85))

[**мибентрижетфирст**](/previous-versions/windows/desktop/legacy/aa374541(v=vs.85))

[**мибентрижетнекст**](/previous-versions/windows/desktop/legacy/aa374542(v=vs.85))

[**мибентрисет**](/previous-versions/windows/desktop/legacy/aa374543(v=vs.85))

[**сетинтерфацерецеиветипе**](/previous-versions/windows/desktop/legacy/aa382181(v=vs.85))

[**валидатерауте**](/previous-versions/windows/desktop/legacy/aa382342(v=vs.85))
