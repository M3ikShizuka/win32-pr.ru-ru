# <a name="router-rras"></a>Маршрутизатор (RRAS)

В этой документации маршрутизатор — это сервер Windows NT или Windows 2000, на котором запущена служба маршрутизации и удаленного доступа (RRAS) и диспетчер таблиц маршрутизации.
