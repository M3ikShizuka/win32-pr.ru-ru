# <a name="routing-table-query-flags"></a>Флаги запроса таблицы маршрутизации

Используйте эти константы для запросов к таблицам маршрутизатора.



| Константа              | Значение      | Описание                                                                |
|-----------------------|------------|----------------------------------------------------------------------------|
| \_нет соответствия \_ RTM      | 0x00000000 | Соответствует ни одному из критериев; возвращаются все маршруты для назначения. |
| \_владелец соответствия \_ RTM     | 0x00000001 | Сопоставление маршрутов с одним и тем же владельцем.                                            |
| \_сосед по СООТВЕТСТВИЮ RTM \_ | 0x00000002 | Сопоставление маршрутов с тем же соседом.                                     |
| \_преф совпадений RTM \_      | 0x00000004 | Соответствует маршрутам с одинаковым предпочтением.                              |
| \_следующий прыжок для соответствия RTM \_   | 0x00000008 | Совпадает с маршрутами, которые имеют одинаковый следующий прыжок.                                |
| \_интерфейс соответствия \_ RTM | 0x00000010 | Соответствует маршрутам, которые находятся в одном интерфейсе.                             |
| \_полное совпадение RTM \_      | 0x0000FFFF | Сопоставление маршрутов со всеми критериями.                                          |
| \_оптимальный \_ протокол RTM   | 0          | Возвращает маршрут независимо от того, какой протокол является его владельцем.                      |
| RTM \_ этот \_ протокол   | ~0         | Возвращает оптимальный маршрут для вызывающего протокола.                           |
