# <a name="remote-access-request-for-comments"></a>Запрос на удаленный доступ для комментариев

Запросы на комментарии (RFC) — это непрерывная серия документов, которые обсуждают технологии Интернета и Интернета. Конкретные документы RFC идентифицируются по числу. Следующие RFC приведены в разделе Служба удаленного доступа (RAS) пакета SDK платформы. Дополнительные сведения о документах RFC и о том, как найти копии определенного документа RFC, см. в разделе [https://www.rfc-editor.org/rfc.html](https://www.rfc-editor.org/rfc.html) .

-   [RFC 791](https://www.ietf.org/rfc/rfc791.txt) – Internet Protocol, спецификация протокола Интернет-программы дарпа, 1981 сентября
-   [RFC 1171](https://www.ietf.org/rfc/rfc1171.txt) — протокол PPP, 1990 июля
-   [RFC 1172](https://www.ietf.org/rfc/rfc1172.txt) — параметры начальной настройки PPP, июль, 1990
-   [RFC 1570](https://www.ietf.org/rfc/rfc1570.txt) — расширения PPP LCP, январь, 1994
-   [RFC 1962](https://www.ietf.org/rfc/rfc1962.txt) — протокол PPP, 1996 июня
-   [RFC 1974](https://www.ietf.org/rfc/rfc1974.txt) – протокол PPP Стак ЛЗС compression, август 1996
-   [RFC 1990](https://www.ietf.org/rfc/rfc1990.txt) — протокол многоканального протокола PPP (MP), август 1996
-   [RFC 2118](https://www.ietf.org/rfc/rfc2118.txt) — протокол Microsoft "точка-точка — сжатие (МППК)", 1997 марта
-   [RFC 2284](https://www.ietf.org/rfc/rfc2284.txt) – протокол расширенной проверки подлинности PPP, 1998 марта
-   [RFC 2433](https://www.ietf.org/rfc/rfc2433.txt) — расширения Microsoft PPP CHAP, 1998 октября
-   [RFC 2484](https://www.ietf.org/rfc/rfc2484.txt) — параметр конфигурации PPP LCP для международного языка, Январь 1999
-   [RFC 2548](https://www.ietf.org/rfc/rfc2848.txt) — атрибуты RADIUS, зависящие от поставщика (Майкрософт)
-   [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt) — удаленная проверка подлинности в службе пользователя (RADIUS), июнь 2000
-   [RFC 2866](https://www.ietf.org/rfc/rfc2866.txt) — учет Radius, 2000 июня
-   [RFC 2868](https://www.ietf.org/rfc/rfc2868.txt) — атрибуты RADIUS для поддержки протокола туннелирования, июнь 2000
-   [RFC 2869](https://www.ietf.org/rfc/rfc2869.txt) — расширения RADIUS, 2000 июня
-   [Черновик](https://www.freeradius.org/rfc/draft-ietf-radius-mschap-attr-01.txt) протоколов MSCHAP для RADIUS с атрибутом 01 — атрибуты RADIUS для поддержки MS-CHAP, ноябрь, 1997
