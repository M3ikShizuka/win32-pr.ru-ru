# <a name="ras-common-dialog-boxes"></a>Общие диалоговые окна службы RAS

Windows предоставляет набор функций, отображающих диалоговые окна RAS, предоставляемые системой. Эти функции облегчают приложениям отображение привычного пользовательского интерфейса, чтобы пользователи могли выполнять задачи RAS. Например, пользователи могут устанавливать и отслеживать подключения, а также работать с записями в телефонной книге. Windows 95 в настоящее время не поддерживает эти функции.

Функция [**расфонебукдлг**](/windows/desktop/api/Rasdlg/nf-rasdlg-rasphonebookdlga) отображает диалоговое окно основного **удаленного доступа к сети** . В этом диалоговом окне пользователь может набрать, изменить или удалить выбранную запись телефонной книги, создать новую запись телефонной книги или указать параметры пользователя. Функция **расфонебукдлг** использует структуру [**распбдлг**](/previous-versions/windows/desktop/legacy/aa377607(v=vs.85)) для указания дополнительных входных и выходных параметров. Например, можно задать элементы структуры, чтобы управлять положением диалогового окна на экране. Структуру **распбдлг** можно использовать для указания функции обратного вызова [**распбдлгфунк**](/windows/desktop/api/Rasdlg/nc-rasdlg-raspbdlgfunca) , которая получает уведомления об активности пользователей, когда диалоговое окно открыто. Например, служба RAS вызывает функцию **распбдлгфунк** , если пользователь набирает, редактирует, создает или удаляет запись телефонной книги.

Функцию [**расдиалдлг**](/windows/desktop/api/Rasdlg/nf-rasdlg-rasdialdlga) можно использовать для запуска операции подключения RAS без отображения диалогового окна «основные подключения **к сети** ». С помощью **расдиалдлг** вы указываете номер телефона или запись телефонной книги для вызова. Функция отображает поток диалоговых окон, указывающий состояние операции соединения. Функция **расдиалдлг** использует структуру [**расдиалдлг**](/previous-versions/windows/desktop/legacy/aa377023(v=vs.85)) для указания дополнительных входных и выходных параметров, таких как расположение диалогового окна и подзаписи телефонной книги для вызова.

Чтобы отобразить страницу свойств **Удаленный доступ к сети** , вызовите функцию [**расмонитордлг**](/previous-versions/windows/desktop/legacy/aa377584(v=vs.85)) . Это диалоговое окно позволяет пользователю отслеживать состояние существующих подключений. Функция **расмонитордлг** использует структуру [**расмонитордлг**](/previous-versions/windows/desktop/legacy/aa377591(v=vs.85)) для указания дополнительных входных и выходных параметров, таких как расположение диалогового окна и страница страницы свойств, отображаемая поверх.

Вы можете вызвать функцию [**расентридлг**](/windows/desktop/api/Rasdlg/nf-rasdlg-rasentrydlga) , чтобы отобразить страницу свойств для создания, изменения или копирования записи телефонной книги. Функция **расентридлг** использует структуру [**расентридлг**](/previous-versions/windows/desktop/legacy/aa377260(v=vs.85)) для указания дополнительных входных и выходных параметров, таких как расположение диалогового окна и тип операции телефонной книги.
