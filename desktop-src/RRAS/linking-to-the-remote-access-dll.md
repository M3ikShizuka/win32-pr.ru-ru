# <a name="linking-to-the-remote-access-dll"></a>Связывание с библиотекой DLL удаленного доступа

Если приложение связывается статически с DLL-библиотекой RASAPI32, приложение не сможет загрузиться, если служба удаленного доступа не установлена. Приложение RAS может загружаться, когда служба удаленного доступа не установлена с помощью [**LoadLibrary**](/windows/win32/api/libloaderapi/nf-libloaderapi-loadlibrarya) для загрузки библиотеки DLL, а [**GetProcAddress**](/windows/win32/api/libloaderapi/nf-libloaderapi-getprocaddress) — для получения указателей на функции RAS.

Функции RAS находятся в RASAPI32.DLL. Библиотека импорта для этих функций — RASAPI32. LIB. Чтобы использовать функции RAS, программы должны включать следующие файлы:



| Файл       | Описание                                                                 |
|------------|-----------------------------------------------------------------------------|
| Соединяющ. Высоты      | Содержит прототипы функций RAS, константы и определения структур. |
| РАСЕРРОР. Высоты | Содержит коды ошибок RAS.                                               |
