# <a name="s-g--g-and---pairs"></a>(s, g), ( \* , g) и ( \* , \* ) пары

Нотация (s, g) представляет конкретный источник и группу. Запрашиваются только данные из источника, s и группы g.

Нотация ( \* , g) представляет источник с подстановочным знаком и определенную группу. Запрошены все данные из любого источника в группу g.

Нотация ( \* , \* ) представляет источник с подстановочным знаком и группу подстановочных знаков. Запрошены все данные из всех источников и всех групп.

Эти обозначения используются для описания добавления и удаления членства в группах.
