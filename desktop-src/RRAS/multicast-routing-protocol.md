# <a name="multicast-routing-protocol"></a>Протокол многоадресной маршрутизации

Протокол многоадресной маршрутизации — это поставщик услуг одного типа, который выступает в качестве клиента в инфраструктуре архитектуры маршрутизатора. Архитектура маршрутизации разработана для расширения такими клиентскими модулями маршрутизатора.

Протокол многоадресной маршрутизации управляет членством в группах и управляет путем, которым данные многоадресной рассылки принимаются по сети. Ниже приведены примеры протоколов многоадресной маршрутизации: независимый протокол (PIM), Открытый многоадресный путь (МОСПФ) и протокол многоадресной маршрутизации (ДВМРП) для вектора расстояний. Протокол IGMP — это специальный протокол многоадресной маршрутизации, который выступает в качестве посредника между узлами и маршрутизаторами.
