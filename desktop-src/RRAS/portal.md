# <a name="routing-and-remote-access-service"></a>Служба маршрутизации и удаленного доступа

## <a name="purpose"></a>Назначение

Службу удаленного доступа (RAS) можно использовать для создания клиентских приложений. В этих приложениях отображаются общие диалоговые окна службы RAS, Управление подключениями и устройствами удаленного доступа и работа с записями в телефонной книге.

Интерфейсы API маршрутизации позволяют создавать приложения для администрирования возможностей маршрутизации операционной системы.

Разработчики могут использовать интерфейсы API протокола маршрутизации для реализации протоколов маршрутизации.

## <a name="developer-audience"></a>Аудитория разработчиков

API-интерфейсы службы маршрутизации и удаленного доступа предназначены для использования программистами C/C++. Программисты также должны быть знакомы с основными понятиями сети.
