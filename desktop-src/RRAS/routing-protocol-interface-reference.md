# <a name="routing-protocol-interface-reference"></a>Справочник по интерфейсу протокола маршрутизации

В этой документации описываются функции и структуры, используемые для реализации протокола маршрутизации в виде библиотеки DLL пользовательского режима.

MPR50 должен быть определен в файле make для протокола маршрутизации.

Макрос **\_ - \_ Привязка sizeof (x)** определяет размер в байтах структуры [**\_ \_ \_ сведений о привязке IP-адаптера**](/windows/desktop/api/Routprot/ns-routprot-ip_adapter_binding_info) , которая содержит количество IP-адресов.

-   [Структуры интерфейса протокола маршрутизации](routing-protocol-interface-structures.md)
