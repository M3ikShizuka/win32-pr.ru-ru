# <a name="finding-specific-information-in-the-routing-table"></a>Поиск конкретных сведений в таблице маршрутизации

Клиенты должны иметь возможность размещать в таблице маршрутизации определенные сведения, а не требовать для перечисления всех данных. API RTMv2 предоставляет возможность поиска определенных назначений, маршрутов и последующих прыжков на основе определенных критериев.
