# <a name="routing-table-manager-instance"></a>Экземпляр диспетчера таблиц маршрутизации

Экземпляр — это отдельная таблица, используемая диспетчером таблиц маршрутизации для обслуживания сведений маршрутизации о подмножестве интерфейсов. Экземпляры обычно используются, чтобы позволить физическому маршрутизатору действовать как набор виртуальных маршрутизаторов — по одному виртуальному маршрутизатору на логическую сеть.

В настоящее время диспетчер таблиц маршрутизации поддерживает только один экземпляр (по умолчанию равен нулю). Клиент может зарегистрироваться в других экземплярах, но ни один из виртуальных маршрутизаторов, кроме значения по умолчанию, распознается или используется диспетчером маршрутизаторов.
