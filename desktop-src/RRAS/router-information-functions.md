# <a name="router-information-functions"></a>Функции информации маршрутизатора

Используйте следующие функции для работы с заголовками и блоками информации маршрутизатора. Заголовок информации состоит из частных метаданных и информационных блоков. Информационные блоки — это массивы [информационных структур](router-information-structures.md) различных [типов](router-information-enumerations.md).

Следующие функции обрабатывают заголовки данных:

-   [**мпринфокреате**](/windows/desktop/api/Mprapi/nf-mprapi-mprinfocreate)
-   [**мпринфоделете**](/windows/desktop/api/Mprapi/nf-mprapi-mprinfodelete)
-   [**мпринфодупликате**](/windows/desktop/api/Mprapi/nf-mprapi-mprinfoduplicate)
-   [**мпринфоремовеалл**](/windows/desktop/api/Mprapi/nf-mprapi-mprinforemoveall)

Следующие функции управляют информационными блоками в заголовке данных:

-   [**мпринфоблоккадд**](/windows/desktop/api/Mprapi/nf-mprapi-mprinfoblockadd)
-   [**мпринфоблоккфинд**](/windows/desktop/api/Mprapi/nf-mprapi-mprinfoblockfind)
-   [**мпринфоблокккуерисизе**](/windows/desktop/api/Mprapi/nf-mprapi-mprinfoblockquerysize)
-   [**мпринфоблоккремове**](/windows/desktop/api/Mprapi/nf-mprapi-mprinfoblockremove)
-   [**мпринфоблокксет**](/windows/desktop/api/Mprapi/nf-mprapi-mprinfoblockset)

Многие [функции администрирования и настройки маршрутизатора](understanding-mprinfo-functions-and-information-headers.md) используют заголовки данных.
