# <a name="forwarder"></a>Сервер пересылки

Сервер пересылки является компонентом режима ядра маршрутизатора, который отвечает за пересылку данных из одного интерфейса маршрутизатора в другие. Сервер пересылки также определяет, предназначен ли пакет для локальной доставки, является ли он направленным из другого интерфейса или обоими.

Существует два сервера пересылки в режиме ядра: одноадресная рассылка и многоадресная рассылка.

Диспетчер маршрутизаторов получает лучшие маршруты ко всем местам назначения из диспетчера таблиц маршрутизации. Эти маршруты передаются в одноадресный перенаправитель. Одноадресный перенаправитель использует эти маршруты для выполнения фактической пересылки данных одноадресной рассылки. Таким образом, одноадресный перенаправитель поддерживает кэш лучших маршрутов в представлении одноадресной рассылки таблицы маршрутизации.

Диспетчер групп многоадресной рассылки использует сведения из представления многоадресной рассылки таблицы маршрутизации для добавления записей многоадресной рассылки в многоадресный сервер пересылки.
