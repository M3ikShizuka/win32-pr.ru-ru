# <a name="multicast-group"></a>Группа многоадресной рассылки

Группа многоадресной рассылки — это IP-адрес класса D (то есть в диапазоне от 224.0.0.0 до 239.255.255.255). Данные многоадресной рассылки, отправляемые на адрес в этом диапазоне, не предназначены для одного целевого объекта. Вместо этого эти данные могут быть получены любым узлом, который выполняет запрос на присоединение к определенной группе. Группа многоадресной рассылки называется либо символом G, либо g.
