# <a name="change-notification-flags"></a>Изменение флагов уведомления



| Константа                         | Значение      | Описание                                                                                                                                                           |
|----------------------------------|------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_ \_ типы изменения номера \_ RTM          | 3          | Указывает число типов изменений, которые в настоящее время используются диспетчером таблиц маршрутизации.                                                                            |
| \_тип изменения \_ RTM \_           | 0x0001     | Уведомляет клиента о любом изменении назначения.                                                                                                                   |
| \_ \_ оптимальный тип изменения RTM \_          | 0x0002     | Уведомляет клиента об изменениях в оптимальном маршруте или при изменении наилучшего маршрута.                                                                                     |
| \_Пересылка \_ типов изменений в RTM \_    | 0x0004     | Уведомляет клиента о любых наиболее оптимальных изменениях маршрута, влияющих на перенаправление (например, изменения следующего прыжка).                                                                      |
| RTM \_ уведомлять \_ только \_ отмеченные \_ DESTS | 0x00010000 | Уведомляет клиента об изменениях назначений, помеченных клиентом. Если этот флаг не указан, отправляются сообщения уведомления об изменении для всех назначений. |
