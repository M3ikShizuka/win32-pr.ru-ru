# <a name="enumeration-flags"></a>Флаги перечисления



| Константа               | Значение      | Описание                                                                                                                                               |
|------------------------|------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_Начало ПЕРЕЧИСЛЕНИЯ \_ RTM       | 0x00000000 | Перечисление маршрутов или назначений, начиная с 0/0.                                                                                                         |
| \_Далее перечисление RTM \_        | 0x00000001 | Перечисление маршрутов или назначений, начиная с указанного адреса или длины маски (например, 10/8). Перечисление переходит к концу таблицы маршрутизации. |
| \_диапазон ПЕРЕЧИСЛЕНИЯ \_ RTM       | 0x00000002 | Перечисляет маршруты или назначения в указанном поддереве, заданном параметром длина адреса или маски (например, 10/8).                                            |
| RTM, \_ Перечисление \_ всех \_ DESTS  | 0x00000000 | Возвращает все назначения.                                                                                                                                  |
| RTM — \_ собственное перечисление \_ \_ DESTS  | 0x01000000 | Возвращать только те назначения, которыми владеет клиент.                                                                                                      |
| RTM, \_ Перечисление \_ всех \_ маршрутов | 0x00000000 | Возвращает все маршруты.                                                                                                                                        |
| Окончательная версия \_ Перечисление \_ собственных \_ маршрутов | 0x00010000 | Возвращать только те маршруты, которыми владеет клиент.                                                                                                            |
