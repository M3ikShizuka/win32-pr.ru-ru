# <a name="ipx-service-table-management"></a>Управление таблицами службы IPX

Протокол IPX-маршрутизации, который регистрируется для обработки службы, поддерживает таблицу привязки службы, определенную в спецификациях протокола SAP для службы Novell. Протокол маршрутизации также предоставляет диспетчеру маршрутизаторов доступ к этой таблице через следующие функции и структуры.

-   [Функции управления таблицей службы](service-table-management-functions.md)
-   [Структуры управления Service Table](service-table-management-structures.md)

Дополнительные сведения о регистрации протокола маршрутизации для обработки службы см. в описании функции [**регистерпротокол**](/windows/desktop/api/Routprot/nc-routprot-pregister_protocol) .
