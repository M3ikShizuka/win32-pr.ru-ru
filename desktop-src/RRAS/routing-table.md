# <a name="routing-table"></a>Таблица маршрутизации

Диспетчер таблиц маршрутизации обслуживает отдельную таблицу маршрутизации для каждого семейства адресов (например, IPX и IPv4). Семейство адресов IPX поддерживается API-интерфейсом RTMv1. Протокол IPv4 поддерживается API RTMv2.

Таблица маршрутизации состоит из назначений, маршрутов и следующих прыжков. Эти записи определяют маршрут к целевой сети.
