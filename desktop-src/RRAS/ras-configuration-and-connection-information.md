# <a name="ras-configuration-and-connection-information"></a>Сведения о конфигурации RAS и подключении

приложения, работающие на Windows NT 4,0 и более поздних версий и Windows 95, могут использовать функцию [**расенумконнектионс**](/windows/desktop/api/Ras/nf-ras-rasenumconnectionsa) для получения сведений о существующих подключениях на локальном компьютере. Сведения для каждого подключения включают в себя маркер подключения и имя записи телефонной книги, используемой для установления соединения. Вы можете использовать маркер соединения в вызове функции [**расжетконнектстатус**](/windows/desktop/api/Ras/nf-ras-rasgetconnectstatusa) , чтобы получить текущее состояние соединения.

Windows NT Сервер 4,0 и более поздние версии предоставляют две новые функции для получения сведений об RAS. Windows 95does не поддерживают эти функции.

Функция [**расенумдевицес**](/windows/desktop/api/Ras/nf-ras-rasenumdevicesa) возвращает имя и тип устройств, ПОДДЕРЖИВАЮЩИХ службу RAS, настроенных на локальном компьютере.

Функция [**расконнектионнотификатион**](/windows/desktop/api/Ras/nf-ras-rasconnectionnotificationa) указывает объект события, который система сообщает при создании или завершении подключения RAS.
