# <a name="ras-phone-books"></a>Телефонные книги RAS

Телефонные книги предоставляют стандартный способ для получения и указания сведений, необходимых диспетчеру подключений удаленного доступа для установления удаленного подключения. Телефонные книги связывают имена записей со сведениями, такими как номера телефонов, COM-порты и параметры модема. Каждая запись телефонной книги содержит сведения, необходимые для установления подключения RAS.

Телефонные книги хранятся в файлах телефонной книги, которые являются текстовыми файлами, содержащими имена записей и связанные с ними сведения. Служба RAS создает файл телефонной книги с именем RASPHONE. PBK. Пользователь может использовать диалоговое окно основного **удаленного доступа к сети** для создания личных файлов телефонной книги. API RAS в настоящее время не поддерживает создание файла телефонной книги. Некоторые функции RAS, например функция [**rasdial**](/windows/desktop/api/Ras/nf-ras-rasdiala) , имеют параметр, задающий файл телефонной книги. Если вызывающий объект не указывает файл телефонной книги, функция использует файл телефонной книги по умолчанию, выбранный пользователем на странице свойств **предпочтения пользователей** диалогового окна **коммутируемая сеть** .

Windows NT 4,0 предоставляет функции [**расфонебукдлг**](/windows/desktop/api/Rasdlg/nf-rasdlg-rasphonebookdlga) и [**расентридлг**](/windows/desktop/api/Rasdlg/nf-rasdlg-rasentrydlga) , которые ОТОБРАЖАЮТ встроенный пользовательский интерфейс RAS, позволяющий пользователям работать с телефонными книгами и записями в телефонной книге.

**Windows 95:** Удаленный доступ к сети хранит записи телефонной книги в реестре, а не в файле телефонной книги. Windows 95 не поддерживает личные телефонные книги или функции, которые отображают встроенные диалоговые окна RAS.
