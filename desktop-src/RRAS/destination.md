# <a name="destination"></a>Назначение

Назначение — это узел, который присоединился к группе многоадресной рассылки. Такой узел уведомил локальный маршрутизатор с использованием протокола IGMP, что он заинтересован в получении данных, отправляемых в одну или несколько групп многоадресной рассылки.
