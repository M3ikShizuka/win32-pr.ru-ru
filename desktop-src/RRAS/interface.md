# <a name="interface-rras"></a>Интерфейс (RRAS)

Интерфейс — это логическое подключение к сети. Каждый интерфейс идентифицируется по уникальному индексу. Протоколы многоадресной маршрутизации (например, МОСПФ) работают со всеми типами интерфейсов аналогичным образом.

В случае с интерфейсом локальной сети интерфейс соответствует фактическому физическому устройству на компьютере (адаптер локальной сети). В случае с интерфейсом глобальной сети интерфейс сопоставляется с портом при установлении соединения. Интерфейсы глобальной сети могут основываться на туннелях, а порт может быть портом виртуальной частной сети (VPN).

Операционная система Windows 2000 и более поздних версий поддерживает интерфейс "точка — MultiPoint". Этот тип интерфейса можно просмотреть в виде коллекции ссылок типа "точка-точка", совместно использующих одну точку завершения.

Чтобы однозначно идентифицировать точную ссылку в интерфейсе типа "точка — MultiPoint", API МГМ использует адрес следующего прыжка идентификатора интерфейса. Для поддержки этой идентификации API МГМ использует идентификатор расширенного интерфейса, который включает адрес [следующего прыжка](next-hop.md) .
