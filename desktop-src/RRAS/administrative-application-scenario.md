# <a name="administrative-application-scenario"></a>Сценарий административного приложения

Административные приложения вызывают подмножество функций МГМ, связанных с перечислением записей многоадресной пересылки (мфес) и статистикой МФЕ. Приложению не требуется регистрироваться в диспетчере групп многоадресной рассылки для использования этих функций (то есть приложению не требуется маркер).
