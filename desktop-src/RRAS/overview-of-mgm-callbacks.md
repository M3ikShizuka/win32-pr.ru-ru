# <a name="overview-of-mgm-callbacks"></a>Общие сведения о обратных вызовах МГМ

В API МГМ существует два типа обратных вызовов:

-   [Обратные вызовы протокола маршрутизации](routing-protocol-callbacks.md)
-   [Включение и отключение обратных вызовов IGMP](igmp-enable-and-disable-callbacks.md)

Эти обратные вызовы в сочетании с интерфейсами API МГМ обеспечивают непрерывную связь между протоколами маршрутизации, IGMP и диспетчером групп многоадресной рассылки.
