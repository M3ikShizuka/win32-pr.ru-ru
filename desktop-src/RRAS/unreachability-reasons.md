# <a name="unreachability-reasons"></a>Причины недостижимости

В следующей таблице перечислены постоянные значения, указывающие, почему интерфейс недоступен.



| Значение                                       | Значение                                                                                        |
|---------------------------------------------|------------------------------------------------------------------------------------------------|
| \_Администратор интерфейса \_ MPR \_ отключен             | Администратор отключил интерфейс.                                                  |
| \_ \_ сбой подключения к ИНТЕРФЕЙСу MPR \_         | Предыдущая попытка соединения завершилась сбоем. Взгляните на элемент **двластеррор** для кода ошибки. |
| \_ограничение на \_ количество подключений \_ \_ многопротокольного интерфейса | Исходящие звонки в настоящее время не разрешены.                                                   |
| не \_ \_ хватает \_ \_ ресурсов многопротокольного интерфейса          | Нет доступных портов или устройств для использования.                                                     |
| \_служба интерфейса \_ MPR \_ приостановлена             | Служба приостановлена.                                                                         |
| многопротокольный \_ интерфейс \_ без \_ \_ смысла            | Сетевой кабель отключен от сетевой карты.                                       |
| \_интерфейс MPR \_ без \_ устройства                  | Сетевая карта удалена с компьютера.                                            |
