# <a name="next-hop"></a>Следующий прыжок

Следующий прыжок — следующий маршрутизатор по пути к назначению. Пакеты из источника перенаправляются в место назначения на основе сквозного прыжка.

Адрес маршрутизатора, который является следующим прыжком в маршруте, используется для уникальной идентификации ссылок в интерфейсе типа "точка — MultiPoint", где все ссылки совместно используют один и тот же индекс интерфейса.
