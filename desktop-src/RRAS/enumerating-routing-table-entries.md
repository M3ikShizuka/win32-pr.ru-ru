# <a name="enumerating-routing-table-entries"></a>Перечисление записей таблицы маршрутизации

Функции перечисления позволяют клиенту получать сведения о конкретном типе объекта таблицы маршрутизации (маршруты, назначения и следующие прыжки). Как протоколы маршрутизации, так и программы администрирования могут использовать эти функции для нахождение конкретных данных.

Пример кода, демонстрирующий использование этих функций, см. в разделе [перечисление всех назначений](enumerate-all-destinations.md) и [перечисление всех маршрутов](enumerate-all-routes.md).
