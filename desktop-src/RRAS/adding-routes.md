# <a name="adding-routes"></a>Добавление маршрутов

После обнаружения клиентом маршрута клиент может добавить этот маршрут в таблицу маршрутизации, вызвав функцию [**ртмадднекссоп**](/windows/desktop/api/Rtmv2/nf-rtmv2-rtmaddnexthop) .

Клиент может удалить маршруты из таблицы маршрутизации, вызвав функцию [**ртмделетераутетодест**](/windows/desktop/api/Rtmv2/nf-rtmv2-rtmdeleteroutetodest) .

Пример кода, демонстрирующий использование этих функций, см. в разделе [Добавление и обновление маршрутов с помощью ртмаддраутетодест](add-and-update-routes-using-rtmaddroutetodest.md).
