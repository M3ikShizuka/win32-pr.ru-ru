Используйте следующие функции для управления удаленными пользователями:

-   [**мпрадминжетпдксервер**](/windows/desktop/api/Mprapi/nf-mprapi-mpradmingetpdcserver)
-   [**мпрадминсендусермессаже**](/windows/desktop/api/Mprapi/nf-mprapi-mpradminsendusermessage)
-   [**мпрадминусержетинфо**](/windows/desktop/api/Mprapi/nf-mprapi-mpradminusergetinfo)
-   [**мпрадминусерсетинфо**](/windows/desktop/api/Mprapi/nf-mprapi-mpradminusersetinfo)
