# <a name="adapters"></a>Адаптеры

Адаптер представляет оборудование, которое подключается к определенному сегменту сети. Одним из примеров адаптера является связанная ЛОКАЛЬная карта. Аналогично, компьютер с двумя модемами, каждый из которых способен подключаться к удаленной сети, имеет два адаптера, один для представления каждого модема.
