# <a name="view"></a>Представление

Представление является подмножеством таблицы маршрутизации и содержит группу связанных маршрутов (например, многоадресные маршруты). В настоящее время поддерживаются только одноадресные и многоадресные представления. Представления иногда называются базами данных маршрутизации (Рибс).
