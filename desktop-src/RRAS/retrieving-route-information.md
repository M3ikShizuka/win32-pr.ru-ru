# <a name="retrieving-route-information"></a>Получение сведений о маршруте

Для получения сведений о маршруте от диспетчера таблиц маршрутизации используются три метода.

-   Перечисление маршрутов (описанных в разделе [Перечисление записей таблицы маршрутизации](enumerating-routing-table-entries.md)).
-   Поиск конкретных маршрутов (см. описание в разделе [Поиск конкретных сведений в таблице маршрутизации](finding-specific-information-in-the-routing-table.md)).
-   Получение измененных назначений (описанных в разделе [Получение уведомлений об изменениях](receiving-notification-of-changes.md)).
