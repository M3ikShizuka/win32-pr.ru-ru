# <a name="maintaining-client-specific-lists"></a>Обслуживание списков Client-Specific

RTMv2 предоставляет функции, позволяющие клиентам создавать частные списки маршрутов, хранимых в таблице маршрутизации. Использование этого списка может оказаться более эффективным, чем перечисление маршрутов из таблицы маршрутизации.

Пример кода, демонстрирующий использование этих функций, см. [в разделе использование Client-Specific списка маршрутов](use-a-client-specific-route-list.md).
