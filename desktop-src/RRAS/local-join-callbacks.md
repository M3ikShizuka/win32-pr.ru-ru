# <a name="local-join-callbacks"></a>Обратные вызовы локального объединения

Когда диспетчер групп многоадресной рассылки получает уведомления IGMP о наличии новых получателей для группы в интерфейсе, Диспетчер групп многоадресной рассылки вызывает обратный вызов [**пмгм \_ локального \_ Присоединение \_**](/windows/desktop/api/Mgm/nc-mgm-pmgm_local_join_callback) к протоколу маршрутизации, владеющему интерфейсом, если он существует. Этот обратный вызов уведомляет протокол маршрутизации изменения.

Этот обратный вызов и обратный вызов [**\_ \_ \_ обратного вызова пмгм локального выхода**](/windows/desktop/api/Mgm/nc-mgm-pmgm_local_leave_callback) используются для синхронизации перенаправления между протоколами IGMP и маршрутизацией.
