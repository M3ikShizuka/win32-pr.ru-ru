# <a name="sysinfo-command"></a>Команда сисинфо

Команда сисинфо извлекает сведения о системе MCI. Команда сисинфо является системной командой MCI; он интерпретируется непосредственно MCI.

Чтобы отправить эту команду, вызовите функцию [**mciSendString**](/previous-versions//dd757161(v=vs.85)) с заданным параметром *лпсзкомманд* , как показано ниже.

``` syntax
_stprintf_s(
  lpszCommand, 
  TEXT("sysinfo %s %s %s"), 
  lpszDeviceID, 
  lpszRequest, 
  lpszFlags
);
```

## <a name="parameters"></a>Параметры

<dl> <dt>

*лпсздевицеид* 
</dt> <dd>

Идентификатор устройства MCI или типа устройства. Если указан тип устройства, это должно быть стандартное имя типа устройства MCI, указанное в справочном материале для команды [**возможности**](capability.md) . Можно указать «ALL», если флаг, указанный в *лпсзрекуест* , допускает эту возможность.

</dd> <dt>

## <a name="examples"></a>Примеры

Следующая команда возвращает число открытых устройств аудио-сигнала.

``` syntax
sysinfo waveaudio quantity open
```

Следующая команда возвращает имя (псевдоним устройства) первого открытого устройства аудио-сигнала.

``` syntax
sysinfo waveaudio name 1 open
```
