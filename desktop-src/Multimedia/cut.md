# <a name="cut-command"></a>Команда "вырезать"

Команда Cut удаляет данные из рабочей области и копирует их в буфер обмена. Устройство Digital-Video распознает эту команду.

Чтобы отправить эту команду, вызовите функцию [**mciSendString**](/previous-versions//dd757161(v=vs.85)) с заданным параметром *лпсзкомманд* , как показано ниже.

``` syntax
_stprintf_s(
  lpszCommand, 
  TEXT("cut %s %s %s"), 
  lpszDeviceID, 
  lpszItem, 
  lpszFlags
); 
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="lpszDeviceID"></span><span id="lpszdeviceid"></span><span id="LPSZDEVICEID"></span>*лпсздевицеид*
</dt> <dd>

Идентификатор устройства MCI. Этот идентификатор или псевдоним назначается при открытии устройства.

</dd> <dt>

<span id="lpszItem"></span><span id="lpszitem"></span><span id="LPSZITEM"></span>*лпсзитем*
</dt> <dd>

Один из следующих флагов, определяющих элемент для вырезания.



| Значение                 | Значение                                                                                                                                                                                                                               |
|-----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| в *прямоугольнике*        | Задает часть каждого кадра, которая вырезается. Если этот параметр опущен, по умолчанию используется весь кадр. Если этот элемент указан, рамки не удаляются. Вместо этого область внутри прямоугольника становится черным.                                       |
| *поток* аудиопотока | Указывает аудиопоток в рабочей области, затронутой командой. Если вы используете этот флаг, а также хотите вырезать видео, необходимо также использовать флаг "поток видео". (Если ни один из флагов не указан, все потоки аудио и видео будут обрезаны.) |
| от *расположения*       | Указывает начало диапазона для вырезания. Если этот параметр опущен, по умолчанию используется текущая заданная позицией.                                                                                                                                                |
| в *Расположение*         | Задает конец диапазона для вырезания. Данные аудио и видео выводятся исключительно из этой должности. Если этот параметр опущен, по умолчанию используется конец рабочей области.                                                                                  |
| *поток* видео поток | Указывает поток видео в рабочей области, на который влияет команда. Если вы используете этот флаг, а также хотите вырезать звук, необходимо также использовать флаг "Audio Stream". (Если ни один из флагов не указан, все потоки аудио и видео будут обрезаны.) |



 

</dd> <dt>

<span id="lpszFlags"></span><span id="lpszflags"></span><span id="LPSZFLAGS"></span>*лпсзфлагс*
</dt> <dd>

Может принимать значение "Wait", "notify", "Test" или их сочетание. Дополнительные сведения об этих флагах см. [в разделе Флаги ожидания, уведомления и проверки](the-wait-notify-and-test-flags.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает нуль в случае успеха или ошибку в противном случае.

## <a name="remarks"></a>Комментарии

Изменение становится постоянным только при явном сохранении данных. Однако воспроизведение действует так, как если бы данные были удалены.
