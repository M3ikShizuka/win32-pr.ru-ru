# <a name="delete-command"></a>удалить команду

Команда DELETE удаляет сегмент данных из файла. Эта команда распознает цифровые видеоролики и звуковые устройства аудио-видео.

Чтобы отправить эту команду, вызовите функцию [**mciSendString**](/previous-versions//dd757161(v=vs.85)) с заданным параметром *лпсзкомманд* , как показано ниже.

``` syntax
_stprintf_s(
  lpszCommand, 
  TEXT("delete %s %s %s"), 
  lpszDeviceID, 
  lpszPosition, 
  lpszFlags
); 
```

| вавеаудио | от <em>расположения</em> | в <em>Расположение</em> | 




 

В следующей таблице перечислены флаги, которые могут быть указаны в параметре *лпсзпоситион* , и их значения.



| Значение                 | Значение                                                                                                                                                                                                                                      |
|-----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| в *прямоугольнике*        | Указывает часть каждого кадра, удаленную. Если этот параметр опущен, по умолчанию используется весь кадр. Если этот элемент указан, рамки не удаляются. Вместо этого область внутри прямоугольника становится черным.                                          |

| от *расположения*       | Указывает расположение, с которого начинается удаление. Если этот флаг опущен, удаление начинается с текущей позиции.                                                                                                                       |
| в *Расположение*         | Указывает расположение, в котором заканчивается удаление. Если этот флаг опущен, удаление продолжится до конца содержимого или рабочей области.                                                                                                       |
| *поток* видео поток | Указывает поток видео в рабочей области, на который влияет команда. Если вы используете этот флаг, а также хотите удалить аудио, необходимо также использовать флаг "Audio Stream". (Если ни один из флагов не указан, все потоки аудио и видео будут удалены.)     |

## <a name="return-value"></a>Возвращаемое значение

Возвращает нуль в случае успеха или ошибку в противном случае.

## <a name="remarks"></a>Remarks

Перед выполнением команд, использующих значения позиций, необходимо задать требуемый формат времени с помощью команды [Set](set.md) .

## <a name="examples"></a>Примеры

Следующая команда удаляет данные звуковой волны с 1 миллисекунда до 900 миллисекунд (при условии, что для формата времени задано значение миллисекунд).

``` syntax
delete mysound from 1 to 900
```
