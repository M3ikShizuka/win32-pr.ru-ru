# <a name="multimedia-pc-specifications"></a>Спецификации мультимедийных ПК

Совет по маркетингу мультимедийных ПК (MPC) разработал две спецификации, которые позволят внедрить мультимедийные возможности. Спецификация уровня 1, разработанная в 1990, содержит базовое определение мультимедийных вычислений в функциональных возможностях, аппаратных компонентах и программных компонентах. Спецификация уровня 2, выданная в 1993, основана на первой спецификации и фокусируется на улучшенных мультимедийных функциях с момента выпуска первой спецификации.

В этом приложении перечислены спецификации.

-   [Спецификация уровня 1](level-1-specification.md)
-   [Спецификация уровня 2](level-2-specification.md)
