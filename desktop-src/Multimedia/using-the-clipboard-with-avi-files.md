# <a name="using-the-clipboard-with-avi-files"></a>Использование буфера обмена с файлами AVI

Буфер обмена предоставляет удобное, временное хранилище, которое приложение может использовать для копирования или перемещения AVI. Авифиле включает функции буфера обмена, которые можно использовать с дисками или файлами памяти.

Файл можно скопировать в буфер обмена с помощью функции [**авипутфилеонклипбоард**](/windows/desktop/api/Vfw/nf-vfw-aviputfileonclipboard) . Чтобы записать файл из буфера обмена в память или на диск, используйте функцию [**авижетфромклипбоард**](/windows/desktop/api/Vfw/nf-vfw-avigetfromclipboard) .

Вы можете очистить файл из буфера обмена с помощью функции [**авиклеарклипбоард**](/windows/desktop/api/Vfw/nf-vfw-aviclearclipboard) . Эта функция не удаляет другие типы данных, например текст, из буфера обмена. Если в приложении используются функции буфера обмена, очистите буфер обмена с **авиклеарклипбоард** перед закрытием приложения или закрытием файла в буфере обмена. Вы можете вызвать **авиклеарклипбоард** в приложении, независимо от того, использовался ли **авипутфилеонклипбоард** .

> [!Note]  
> Если приложение копирует файл в буфер обмена, а файл содержит данные потока, которые могут измениться, можно создать файл памяти клонированных потоков с помощью функции [**авимакефилефромстреамс**](/windows/desktop/api/Vfw/nf-vfw-avimakefilefromstreams) . Затем можно поместить файл в буфер обмена, а затем освободить исходный файл без его аннулирования.
