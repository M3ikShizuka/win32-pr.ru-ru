# <a name="drv_install-message"></a>\_Сообщение об установке DRV

Уведомляет устанавливаемый драйвер. Драйвер должен создать и инициализировать все необходимые разделы и значения реестра, а также убедиться, что вспомогательные драйверы и оборудование установлены и правильно настроены.

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="dwDriverId"></span><span id="dwdriverid"></span><span id="DWDRIVERID"></span>*двдриверид*
</dt> <dd>

Идентификатор устанавливаемого драйвера. Это то же значение, которое ранее было возвращено драйвером [**из \_ открытого сообщения DRV**](drv-open.md) .

</dd> <dt>

<span id="hdrvr"></span><span id="HDRVR"></span>*хдрвр*
</dt> <dd>

Описатель устанавливаемого экземпляра драйвера.

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Адрес структуры [**дрвконфигинфо**](/windows/win32/api/mmiscapi/ns-mmiscapi-drvconfiginfo) или **значение NULL**. Если задана структура, она содержит имена раздела реестра и значения, связанные с драйвером.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из следующих значений:



| Требование | Значение |
|-----------------|--------------------------------------------------------------------------------------------|
| ДРВКНФ \_ ОК      | Установка выполнена успешно; дальнейшие действия не требуются.                             |
| ДРВКНФ \_ Отмена  | Сбой установки..                                                                  |
| перезапустить ДРВКНФ \_ | Установка выполнена успешно, но она вступит в силу только после перезапуска системы. |



 

## <a name="remarks"></a>Remarks

Параметр *lParam1* не используется.

Некоторые устанавливаемые драйверы добавляют сведения о конфигурации к значению, назначенному значению реестра, связанному с драйвером.
