# <a name="wm_cap_set_scale-message"></a>\_Сообщение с \_ заданным \_ масштабом WM Cap

В сообщении о **\_ \_ \_ масштабировании WM Set Scale** можно включить или отключить масштабирование для видеороликов предварительного просмотра. Если масштабирование включено, захваченный видеокадр растягивается по размерам окна Capture. Это сообщение можно отправить явно или с помощью макроса [**каппревиевскале**](/windows/desktop/api/Vfw/nf-vfw-cappreviewscale) .


```C++
WM_CAP_SET_SCALE 
wParam = (WPARAM) (BOOL)f; 
lParam = 0L; 
```



## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="f"></span><span id="F"></span>*ж*
</dt> <dd>

Флаг масштабирования для предварительного просмотра. Укажите **значение true** для параметра растяжение кадров предварительного просмотра до размера окна записи или **значение false** , чтобы отобразить их в натуральном размере.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** в случае успеха или **false** в противном случае.

## <a name="remarks"></a>Remarks

Изображения с предварительным просмотром — это немедленное представление захваченных кадров в окне Capture. Он не влияет на размер кадров, сохраняемых в файле.

Масштабирование не оказывает никакого влияния при использовании оверлея для показа видео в буфере кадров.
