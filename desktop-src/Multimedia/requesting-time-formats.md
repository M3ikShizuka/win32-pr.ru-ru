# <a name="requesting-time-formats"></a>Запрос форматов времени

Windows использует структуру [**ммтиме**](/previous-versions//dd757347(v=vs.85)) для представления времени в одном или нескольких различных форматах, включая миллисекунды, примеры, SMPTE и форматы указателей MIDI song. Элемент **wType** указывает формат времени.

Функция [**мидистреампоситион**](/windows/win32/api/mmeapi/nf-mmeapi-midistreamposition) использует структуру **ммтиме** . Перед вызовом этой функции необходимо задать элемент **wType** , чтобы указать запрошенный формат времени. Чтобы узнать, поддерживается ли запрошенный формат времени, проверьте **wType** после вызова. Если запрошенный формат времени не поддерживается, время указывается в альтернативном формате времени, выбранном драйвером устройства, а элемент **wType** изменяется для указания выбранного формата времени.

Дополнительные сведения о структуре **ммтиме** см. в разделе [мультимедийные таймеры](multimedia-timers.md).
