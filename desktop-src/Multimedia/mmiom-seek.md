# <a name="mmiom_seek-message"></a>\_Сообщение ммиом Seek

Сообщение **ммиом \_ Seek** отправляется в процедуру ввода-вывода функцией [**ммиосик**](/windows/win32/api/mmiscapi/nf-mmiscapi-mmioseek) , чтобы запросить перемещение текущего расположения файла.


```C++
MMIOM_SEEK 
lParam1 = (LPARAM) lNewFilePos 
lParam2 = (LPARAM) lChangeFlag 
```



## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="lNewFilePos"></span><span id="lnewfilepos"></span><span id="LNEWFILEPOS"></span>*лневфилепос*
</dt> <dd>

Новое расположение файла. Значение этого параметра зависит от флага, указанного в **лчанжефлаг**.

</dd> <dt>

<span id="lChangeFlag"></span><span id="lchangeflag"></span><span id="LCHANGEFLAG"></span>*лчанжефлаг*
</dt> <dd>

Флаг, указывающий, как изменяется расположение файла. Определены следующие значения:



| Требование | Значение |
|-----------|------------------------------------------------------------------------------------------------------------------------|
| Поиск \_ вал. | Переместить расположение файла, чтобы *лневфилепос* байты из текущей положения. *лневфилепос* может быть положительным или отрицательным. |
| \_конец поиска | Переместите положение файла в *лневфилепос* байт с конца файла.                                             |
| Поиск \_ набора | Переместите положение файла в *лневфилепос* байт с начала файла.                                       |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает новое расположение файла. Если возникает ошибка, возвращается значение 1.

## <a name="remarks"></a>Комментарии

Процедура ввода-вывода отвечает за поддержание текущего расположения файла в элементе **лдискоффсет** структуры [**ммиоинфо**](/previous-versions//dd757322(v=vs.85)) .
