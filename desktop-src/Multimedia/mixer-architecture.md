# <a name="mixer-architecture"></a>Архитектура микшера

Основной элемент архитектуры микшера — это *звуковая линия*. Звуковая строка состоит из одного или нескольких каналов данных, исходящих из одного источника или системного ресурса. Например, в стереосистеме есть два канала данных, но они рассматриваются как одна звуковая строка, так как она происходит из одного источника.

Архитектура микшера предоставляет службы маршрутизации для управления звуковыми линиями на компьютере. Службы маршрутизации можно использовать, если установлены соответствующие аппаратные устройства и драйверы программного обеспечения. Архитектура микшера позволяет нескольким строкам аудио источника сопоставляться с одной конечной строкой назначения.

С каждой звуковой линией могут быть связаны элементы управления микшера. Элемент управления микшера может выполнять любое количество функций (например, управляющий том) в зависимости от характеристик соответствующей звуковой линии.
