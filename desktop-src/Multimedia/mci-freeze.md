# <a name="mci_freeze-command"></a>\_Команда заморозить MCI

Команда « \_ замораживание» MCI закрепляет движение на экране. Эта команда распознает цифровые видеоролики, наложение видео и устройства ВИДЕОМАГНИТОФОНА.

Чтобы отправить эту команду, вызовите функцию [**мЦисендкомманд**](/previous-versions//dd757160(v=vs.85)) со следующими параметрами.


```C++
MCIERROR mciSendCommand(
  MCIDEVICEID wDeviceID, 
  MCI_FREEZE, 
  DWORD dwFlags, 
  (DWORD) (LPMCI_GENERIC_PARMS) lpFreeze
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="wDeviceID"></span><span id="wdeviceid"></span><span id="WDEVICEID"></span>*вдевицеид*
</dt> <dd>

Идентификатор устройства MCI, который должен получить командное сообщение.

</dd> <dt>

<span id="dwFlags"></span><span id="dwflags"></span><span id="DWFLAGS"></span>*dwFlags*
</dt> <dd>

\_Уведомление MCI, \_ Ожидание MCI или, для устройств цифрового видео и видеомагнитофона, тест MCI \_ . Дополнительные сведения об этих флагах см. [в разделе Флаги ожидания, уведомления и проверки](the-wait-notify-and-test-flags.md).

</dd> <dt>

<span id="lpFreeze"></span><span id="lpfreeze"></span><span id="LPFREEZE"></span>*лпфризе*
</dt> <dd>

Указатель на [**общую структуру \_ \_ пармс MCI**](mci-generic-parms.md) . (Устройства с дополнительными параметрами могут заменить эту структуру структурой, зависящей от устройства.)

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает нуль в случае успеха или ошибку в противном случае.

## <a name="remarks"></a>Remarks

Тип устройства **дигиталвидео** использует следующие дополнительные флаги:

<dl> <dt>

<span id="MCI_DGV_FREEZE_AT"></span><span id="mci_dgv_freeze_at"></span>\_ДГВ MCI \_ \_
</dt> <dd>

Элемент **RC** структуры, идентифицируемой *лпфризе* , содержит допустимый прямоугольник. Прямоугольник задает область внутри буфера кадров, для которой включен бит маски блокировки для каждого пикселя. Указанные пиксели не будут обновлены до тех пор, пока не будет снят бит маски блокировки. Если этот флаг не указан, прямоугольником по умолчанию выделяется весь буфер кадров. Этот флаг поддерживается только в том случае, если команда [MCI \_ Жетдевкапс](mci-getdevcaps.md) возвращает **значение true** для параметра ДГВ MCI, \_ \_ жетдевкапс \_ может \_ блокировать флаг.

</dd> <dt>

<span id="MCI_DGV_FREEZE_OUTSIDE"></span><span id="mci_dgv_freeze_outside"></span>MCI \_ ДГВ \_ заморозить \_ снаружи
</dt> <dd>

Область за пределами региона, указанного для \_ \_ флага MCI ДГВ Freeze On \_ , зафиксирована.

</dd> </dl>

Для устройств с цифровыми видео параметр *лпфризе* указывает на структуру [**MCI \_ ДГВ \_ Freeze \_ пармс**](/windows/desktop/api/Digitalv/ns-digitalv-mci_dgv_rect_parms) .

Тип устройства **видеомагнитофона** использует следующие дополнительные флаги:

<dl> <dt>

<span id="MCI_VCR_FREEZE_FIELD"></span><span id="mci_vcr_freeze_field"></span>\_ \_ поле замораживания видеомагнитофона MCI \_
</dt> <dd>

Заморозить только один член текущего кадра.

</dd> <dt>

<span id="MCI_VCR_FREEZE_FRAME"></span><span id="mci_vcr_freeze_frame"></span>\_ \_ кадр замораживания видеомагнитофона MCI \_
</dt> <dd>

Закрепить оба поля текущего кадра.

</dd> <dt>

<span id="MCI_VCR_FREEZE_INPUT"></span><span id="mci_vcr_freeze_input"></span>\_ \_ Ввод данных о замораживании видеомагнитофона MCI \_
</dt> <dd>

Закрепление текущего кадра на экране (используется для записи).

</dd> <dt>

<span id="MCI_VCR_FREEZE_OUTPUT"></span><span id="mci_vcr_freeze_output"></span>\_закрепление \_ \_ выходных данных MCI видеомагнитофона
</dt> <dd>

Закрепление текущего кадра из ВИДЕОМАГНИТОФОНА (используется с записью кадров).

</dd> </dl>

Для устройств ВИДЕОМАГНИТОФОНА параметр *лпфризе* указывает на [**общую структуру \_ \_ пармс MCI**](mci-generic-parms.md) .

Для типа устройства **оверлея** используется следующий дополнительный флаг:

<dl> <dt>

<span id="MCI_OVLY_RECT"></span><span id="mci_ovly_rect"></span>MCI \_ овли \_ Rect
</dt> <dd>

Элемент **RC** структуры, идентифицируемой *лпфризе* , содержит допустимый прямоугольник. Если этот флаг не указан, драйвер устройства будет закреплять весь кадр.

</dd> </dl>

Для устройств с наложением параметр *лпфризе* указывает на структуру [**MCI \_ овли \_ Rect \_ пармс**](mci-ovly-rect-parms.md) .
