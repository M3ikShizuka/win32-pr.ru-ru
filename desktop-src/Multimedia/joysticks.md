# <a name="joysticks"></a>Джойстики

В этом обзоре описываются функции и сообщения, поддерживающие джойстики, а также другие вспомогательные устройства ввода, которые отслеживание позиций в абсолютной системе координат, например сенсорный экран, планшет с поддержкой планшетов и светлое перо. Расширенные возможности также обеспечивают поддержку педалей через руль, штурвалов и других устройств, использующих до шести осей перемещения, точки зрения крышки и 32 кнопок.

-   [О джойстиках](about-joysticks.md)
-   [Использование джойстиков](using-joysticks.md)
-   [Справочник по джойстику](joystick-reference.md)

API джойстика мультимедиа заменен Директинпут, который обладает дополнительными возможностями для современных устройств ввода и обеспечения обратной связи. Дополнительные сведения см. в документации DirectX.
