# <a name="vcm-services"></a>ВКМ Services

Как правило, приложение использует ВКМ для выполнения следующих задач:

-   Находите, откройте или установите программу сжатия или распаковки.
-   Настройте или получите сведения о конфигурации сжатия или распаковки.
-   Используйте ряд функций для сжатия, распаковки или рисования данных.

Функции и макросы библиотеки Дравдиб выполняют эти задачи неявно и могут предоставлять наиболее удобный способ использования ВКМ. Дополнительные сведения о библиотеке Дравдиб см. в разделе [дравдиб](drawdib.md).
