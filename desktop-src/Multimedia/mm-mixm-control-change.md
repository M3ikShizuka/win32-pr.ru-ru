# <a name="mm_mixm_control_change-message"></a>\_ \_ Сообщение об изменении элемента управления миксм mm \_

Устройство-микшер отправляет сообщение об **\_ \_ \_ изменении элемента управления миксм mm** , чтобы уведомить приложение о том, что изменилось состояние элемента управления, связанного с звуковой линией. Приложение должно обновить отображаемые и кэшированные значения для указанного элемента управления.


```C++
MM_MIXM_CONTROL_CHANGE 
wParam = (WPARAM) hMixer 
lParam = (LPARAM) dwControlID 
```



## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="hMixer"></span><span id="hmixer"></span><span id="HMIXER"></span>*хмиксер*
</dt> <dd>

Маркер устройства микшера (**хмиксер**), отправившего сообщение уведомления.

</dd> <dt>

<span id="dwControlID"></span><span id="dwcontrolid"></span><span id="DWCONTROLID"></span>*двконтролид*
</dt> <dd>

Идентификатор элемента управления микшера, состояние которого изменилось. Этот идентификатор совпадает с элементом **двконтролид** структуры **миксерконтрол**, возвращенной функцией **миксержетлинеконтролс**.

</dd> </dl>

## <a name="remarks"></a>Remarks

Приложение должно открыть микшер устройства и указать окно обратного вызова для получения сообщения об **\_ \_ \_ изменении элемента управления mm миксм** .
