# <a name="retrieving-the-sequence-division-type"></a>Получение типа деления последовательности

*Тип деления* для последовательности MIDI определяет промежуток времени между событиями MIDI в последовательности. Чтобы определить тип деления последовательности, используйте команду [**MCI \_ Status**](mci-status.md) и присвойте элементу **двитем** [**\_ состояния MCI \_ пармс**](mci-status-parms.md) структуры значение MCI \_ Seq \_ Status \_ дивтипе.

Если команда **" \_ состояние MCI** " выполнена успешно, элемент **двретурн** структуры **\_ \_ пармс "состояние MCI** " будет содержать одно из следующих значений для указания типа деления.



| Значение                        | Тип деления                     |
|------------------------------|-----------------------------------|
| MCI \_ Seq \_ div \_ ппкн          | ППКН (штук на четверть заметки)     |
| MCI \_ Seq \_ div \_ SMPTE \_ 24     | SMPTE, 24 кадра кадров/с (кадров в секунду) |
| MCI \_ Seq \_ div \_ SMPTE \_ 25     | SMPTE, 25 кадров/с                     |
| MCI \_ Seq \_ div \_ SMPTE \_ 30     | SMPTE, 30 кадров/с                     |
| MCI \_ Seq \_ div \_ SMPTE \_ 30DROP | SMPTE, 30 кадров с раскрывающимся кадром          |



 

Необходимо иметь представление о типе деления последовательности для изменения или запроса его темп. Невозможно изменить тип деления последовательности с помощью средства Sequencer MCI.
