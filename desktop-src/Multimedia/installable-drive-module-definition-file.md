# <a name="installable-drive-module-definition-file"></a>Установочный файл Module-Definition диска

Определение модуля (. DEF). файл устанавливаемого драйвера содержит имя драйвера, экспортирует функцию [дриверпрок](/windows/win32/api/mmiscapi/nc-mmiscapi-driverproc) и определяет описание драйвера. В следующем примере показан типичный файл определения модуля для устанавливаемого драйвера:


```C++
LIBRARY OSCI 
DESCRIPTION 'FREQ,AMPL:Oscilloscope frequency and amplitude drivers.'
EXPORTS
    DriverProc
 
```



Некоторые приложения установки могут открыть драйвер и получить строку описания для использования при установке драйвера. Чтобы сохранить совместимость с этими приложениями установки, строка описания должна иметь следующую форму:

*Псевдоним* описания \[ ,*псевдоним* \] ... **:**_текст_  

*Псевдоним* задает уникальное имя драйвера, который приложения могут использовать для открытия драйвера. Псевдоним также служит в качестве имени значения, связанного с драйвером в реестре. Несколько псевдонимов разделяются запятыми. *Текст* описывает назначение драйвера.
