# <a name="avifile-functions-and-macros"></a>Функции и макросы Авифиле

Функции и макросы Авифиле предоставляют доступ к файлам на основе времени, использующим формат файла сведений о ресурсах (Metallica), например файлы аудио-и аудио-видео с чередованием (AVI). Эти функции и макросы управляют Metallica-файлами, что делает их ненужными для управления и навигации по архитектуре Metallica.
