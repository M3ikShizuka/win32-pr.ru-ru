# <a name="about-audio-mixers"></a>О программе Audio Миксерс

Службы микшера звука управляют маршрутизацией звуковых линий на целевое устройство для воспроизведения или записи. Эти службы также могут управлять Томом и другими эффектами. Многие из методов, необходимых для использования этих служб, похожи на звуковые устройства, обсуждаемые в других обзорах мультимедиа.
