# <a name="multimedia-input"></a>Входные данные мультимедиа

Windows предоставляет функции, позволяющие приложению принимать входные данные мультимедиа. Функции ввода мультимедиа были введены в 16-разрядной Windows. Многие их функции были заменены компонентами DirectX. Дополнительные сведения см. в документации DirectX.

Следующие разделы обсуждаются в документации по входным мультимедиа.

-   [Файловый ввод-вывод мультимедиа](multimedia-file-i-o.md)
-   [Джойстики](joysticks.md)
-   [Таймеры мультимедиа](multimedia-timers.md)
