# <a name="mci-device-types"></a>Типы устройств MCI

Следующие значения обозначают устройства в сообщениях и структурах MCI:



| Значение                         | Значение                       |
|-------------------------------|-------------------------------|
| \_ \_ идентификатор устройства mci \_          | Любое устройство                    |
| \_анимация MCI девтипе \_       | Анимация — устройство воспроизведения     |
| \_ \_ аудио компакт-диск MCI девтипе \_       | Устройство CD Audio               |
| MCI \_ девтипе \_ dat             | Ленточное устройство Digital Audio     |
| \_устройство MCI девтипе \_ Digital \_ Video  | Устройство воспроизведения цифрового видео |
| \_ДЕВТИПЕ MCI \_           | Неопределенное устройство              |
| наложение MCI \_ девтипе \_         | Устройство наложения видео          |
| \_сканер ДЕВТИПЕ \_ MCI         | Устройство сканера                |
| Программа MCI \_ девтипе \_ Sequencer       | Устройство MIDI Sequencer         |
| \_видеомагнитофон ДЕВТИПЕ \_ MCI             | Устройство записи видеокассет       |
| \_ВИДЕОДИСК MCI девтипе \_       | Проигрыватель видеодиск              |
| \_ \_ звуковая волна MCI девтипе \_ | Звуковая волна-устройство         |
