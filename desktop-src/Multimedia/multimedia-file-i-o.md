# <a name="multimedia-file-io"></a>Файловый ввод-вывод мультимедиа

Службы файлового ввода-вывода предоставляют больше функций, чем стандартные службы операционной системы, включая поддержку буферизованного ввода-вывода, файлы формата файлов обмена ресурсами (Metallica), файлы памяти и пользовательские системы хранения. Кроме того, службы файлового ввода-вывода оптимизированы для приложений, зависящих от производительности.

-   [О файловых операциях ввода-вывода мультимедиа](about-multimedia-file-i-o.md)
-   [Использование файлового ввода-вывода мультимедиа](using-multimedia-file-i-o.md)
-   [Справочник по операциям файлового ввода-вывода мультимедиа](multimedia-file-i-o-reference.md)
