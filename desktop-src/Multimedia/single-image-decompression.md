# <a name="single-image-decompression"></a>Распаковка Single-Image

Для распаковки одного изображения можно использовать функцию [**иЦимажедекомпресс**](/windows/desktop/api/Vfw/nf-vfw-icimagedecompress) . Эта функция возвращает маркер распакованного DIB. Распакованный DIB-формат хранится в формате "CF \_ DIB".
