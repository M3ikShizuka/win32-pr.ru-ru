# <a name="midi-input-data-types"></a>Типы входных данных MIDI

Windows определяет следующие типы данных для входных функций MIDI.



| Значение                            | Значение                                                                                                                                                                                     |
|----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **хмидиин**                      | Работа устройства ввода MIDI.                                                                                                                                                              |
| [**мидихдр**](/windows/win32/api/mmeapi/ns-mmeapi-midihdr)       | Заголовок для буфера потока или блок данных, исключающих систему MIDI. Для входных приложений эта структура записывает только данные, не зависящие от системы (потоковая передача не поддерживается для ввода MIDI). |
| [**мидиинкапс**](/windows/win32/api/mmeapi/ns-mmeapi-midiincaps) | Структура, используемая для получения сведений о возможностях входного устройства MIDI.                                                                                                                    |
