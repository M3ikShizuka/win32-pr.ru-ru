# <a name="multimedia-recording"></a>Запись мультимедиа

Вы можете реализовать возможности записи в приложении с помощью пользовательского интерфейса, встроенного в МЦивнд. Вы можете использовать функцию [**мЦивндкреате**](/windows/desktop/api/Vfw/nf-vfw-mciwndcreatea) и макрос [**мЦивнднев**](/windows/desktop/api/Vfw/nf-vfw-mciwndnew) для предоставления элементов управления для запуска и остановки записи и для сохранения записанных данных. С помощью **мЦивндкреате** можно указать стили окна для вывода окна мЦивнд и включения кнопки **запись** на панель инструментов. С помощью **мЦивнднев** можно указать записываемый тип устройства и указать, что данные должны быть записаны в новый файл.

Если приложению требуется более изощренная Настройка, можно автоматизировать и настроить запись с помощью макроса [**мЦивндрекорд**](/windows/desktop/api/Vfw/nf-vfw-mciwndrecord) . Дополнительные сведения см. [в разделе Настройка процесса записи](customizing-the-recording-process.md).

> [!Note]  
> Некоторые устройства, такие как CD Audio и МЦИАВИ, используются только для воспроизведения. Для записи можно использовать другие устройства, например устройства аудио-сигнала. Если указать устройство, которое не может записать, МЦивнд пропускает кнопку **записи** на панели инструментов.
