# <a name="preloading-patches-with-internal-midi-synthesizers"></a>Предварительная загрузка исправлений с помощью внутренних синтезаторов MIDI

Некоторые внутренние устройства MIDI синтезатора не могут одновременно загружать все свои исправления. Эти устройства должны предварительно загружать свои данные исправлений.

Windows предоставляет следующие функции, чтобы запросить предварительную загрузку и кэширование указанных исправлений для синтезатора.



| Значение                                                      | Значение                                                                                                     |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**мидиауткачепатчес**](/windows/win32/api/mmeapi/nf-mmeapi-midioutcachepatches)         | Запрашивает, что внутренняя Предварительная загрузка устройства MIDI синтезатора и кэширование указанных исправлений мелодик.              |
| [**мидиауткачедрумпатчес**](/windows/win32/api/mmeapi/nf-mmeapi-midioutcachedrumpatches) | Запрашивает предварительную загрузку внутреннего устройства MIDI и кэширование заданных исправлений, основанных на ключах. |



 

Сведения о том, как определить, поддерживает ли определенное устройство предварительную загрузку исправлений, см. в статье [запросы к выходным устройствам MIDI](querying-midi-output-devices.md).
