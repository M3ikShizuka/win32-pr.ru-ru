# <a name="using-joysticks"></a>Использование джойстиков

В этом разделе содержатся примеры, демонстрирующие выполнение следующих задач.

-   [Получение возможностей драйвера](getting-the-driver-capabilities.md)

-   [Обработка сообщений джойстика](processing-joystick-messages.md)

Примеры взяты из простого джойстика, который извлекает сведения о положении и состоянии кнопок из джойстика, воспроизводит звуковые ресурсы и выводит маркеры на экране при нажатии пользователем кнопок джойстика.
