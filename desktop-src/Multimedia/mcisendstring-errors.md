# <a name="mcisendstring-errors"></a>Ошибки mciSendString

Функция [**mciSendString**](/previous-versions//dd757161(v=vs.85)) возвращает следующие ошибки, но не [**мЦисендкомманд**](/previous-versions//dd757160(v=vs.85)):



| Значение                             | Значение                                           |
|-----------------------------------|---------------------------------------------------|
| МЦИЕРР \_ неверная \_ константа             | Значение, указанное для параметра, неизвестно.   |
| МЦИЕРР \_ неверное \_ целое число              | Целое число в команде недопустимо или отсутствует. |
| МЦИЕРР \_ повторяющихся \_ флагов          | Флаг или значение указано дважды.              |
| МЦИЕРР \_ отсутствует \_ \_ Строка команды  | Команда не указана.                         |
| МЦИЕРР \_ отсутствует \_ \_ имя устройства     | Не указано имя устройства.                     |
| МЦИЕРР \_ отсутствует \_ строковый \_ аргумент | Строковое значение отсутствовало в команде.      |
| для МЦИЕРР \_ New \_ требуется \_ псевдоним      | Псевдоним должен использоваться с именем устройства "New". |
| МЦИЕРР \_ без \_ закрывающей \_ кавычки        | Отсутствует закрывающая кавычка.              |
| МЦИЕРР \_ уведомлять \_ при \_ автоматическом \_ открытии    | Флаг "notify" недопустим при автоматическом открытии.      |
| \_ \_ переполнение параметра мЦиерр           | Выходная строка недостаточно длинна.            |
| \_внутреннее средство синтаксического анализа мЦиерр \_          | Произошла внутренняя ошибка средства синтаксического анализа.                |
| МЦИЕРР \_ НЕраспознанное \_ ключевое слово     | Указан неизвестный параметр команды.       |
