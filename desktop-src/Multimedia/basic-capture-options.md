# <a name="basic-capture-options"></a>Основные параметры записи

Изменяя один или несколько параметров записи, определенных в структуре [**каптурепармс**](/windows/win32/api/vfw/ns-vfw-captureparms) , можно выполнять следующие задачи:

-   Изменение частоты захвата кадров.
-   Укажите клавиатуру или элемент управления мышью для завершения сеанса записи.
-   Укажите длительность сеанса записи.
