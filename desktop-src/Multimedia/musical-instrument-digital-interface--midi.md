# <a name="musical-instrument-digital-interface-midi"></a>Цифровой интерфейс музыкальных инструментов (MIDI)

MIDI — это протокол и набор команд для хранения и передачи информации о музыке. Устройства с выходом MIDI преобразуют эти сведения и используют их для синтезирования музыки.
