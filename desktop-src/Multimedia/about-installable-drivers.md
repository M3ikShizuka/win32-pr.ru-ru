# <a name="about-installable-drivers"></a>Сведения об устанавливаемых драйверах

Устанавливаемый драйвер — это библиотека динамической компоновки (DLL), которая предоставляет стандартный интерфейс, с помощью которого приложения и библиотеки DLL Windows взаимодействуют с драйвером и управляют им. Устанавливаемые драйверы чаще всего используются для драйверов мультимедийных устройств, но устанавливаемые драйверы также могут использоваться в других целях и особенно полезны в ситуациях, требующих стандартного интерфейса и управления несколькими экземплярами.
