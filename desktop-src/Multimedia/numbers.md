# <a name="numbers-windows-multimedia"></a>числа (Windows мультимедиа)

Элементы управления "число" позволяют пользователю вводить числовые данные, связанные со строкой. Числовые данные выражаются как целые числа со знаком, целые числа без знака или целочисленные шкала значения. Эти элементы управления используют [**\_ неподписанные**](/previous-versions//dd757298(v=vs.85)) структуры [**миксерконтролдетаилс \_**](/previous-versions//dd757297(v=vs.85)) и миксерконтролдетаилс для получения и задания свойств элементов управления. В следующей таблице описаны типы элементов управления "число".



| Control  | Описание                                                                                                                         |
|----------|-------------------------------------------------------------------------------------------------------------------------------------|
| Со знаком   | Позволяет указать целочисленные значения в диапазоне от – 231 до (231 – 1).                                                               |
| Без знака | Позволяет указать целочисленные значения в диапазоне от 0 до (232 – 1).                                                                   |
| Шкала  | Позволяет указать целочисленные шкала значения в десятых долях децибел. Диапазон значений для этого элемента управления — от – 32 768 до 32 767. |
| Процент  | Позволяет указать значения в виде процентов.                                                                                         |
