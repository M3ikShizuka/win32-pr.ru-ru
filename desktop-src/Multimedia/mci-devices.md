# <a name="mci-devices"></a>Устройства MCI

Каждое устройство мультимедиа MCI поддерживает основной набор команд MCI таким образом, который имеет смысл для этого устройства. Например, команда [**воспроизвести**](play.md) ([**MCI \_ Play**](mci-play.md)) заставляет открытое устройство воспроизводить файл или Track независимо от типа данных, с которыми работает устройство. В следующих разделах рассматриваются устройства MCI и способы их реагирования на стандартные команды MCI.

-   [Типы устройств](device-types.md)
-   [Имена устройств](device-names.md)
-   [Поддержка драйверов для команд MCI](driver-support-for-mci-commands.md)
-   [Поведение драйверов по умолчанию](default-behavior-of-drivers.md)
