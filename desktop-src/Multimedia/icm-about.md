# <a name="icm_about-message"></a>ICM \_ о сообщении

**ICM \_ о** сообщении уведомляет драйвер сжатия видео о необходимости отобразить диалоговое окно о программе или запрашивает драйвер сжатия видео, чтобы определить, имеется ли в нем диалоговое окно About. Это сообщение можно отправить явно или с помощью макроса [**икабаут**](/windows/desktop/api/Vfw/nf-vfw-icabout) .


```C++
ICM_ABOUT 
wParam = (DWORD_PTR) (UINT_PTR) hwnd; 
lParam = 0; 
```



## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="hwnd"></span><span id="HWND"></span>*HWND*
</dt> <dd>

Обработано с родительским окном отображаемого диалогового окна. Можно также определить, содержит ли драйвер диалоговое окно о программе, указав в этом параметре значение-1, как в макросе [**иккуерябаут**](/windows/desktop/api/Vfw/nf-vfw-icqueryabout) . Драйвер возвращает ИЦЕРР \_ ОК, если в нем есть диалоговое окно About или ицерр не \_ поддерживается в противном случае.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ИЦЕРР \_ ОК, если драйвер поддерживает это сообщение или ИЦЕРР \_ не поддерживается в противном случае.
