# <a name="level-2-specification"></a>Спецификация уровня 2

Спецификация уровня 2 была разработана, чтобы способствовать внедрению улучшенных мультимедийных возможностей. Эта спецификация основана на требованиях, заданных в спецификации уровня 1, и является надмножеством его. Спецификация уровня 2 определяет минимальные системные функции для улучшенных возможностей мультимедиа. Это не рекомендация для конфигурации системы.
