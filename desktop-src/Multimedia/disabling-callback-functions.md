# <a name="disabling-callback-functions"></a>Отключение функций обратного вызова

Вы можете постоянно или временно отключить любую функцию обратного вызова, указав **значение NULL** вместо функции обратного вызова при отправке соответствующего сообщения для установки функции обратного вызова.
