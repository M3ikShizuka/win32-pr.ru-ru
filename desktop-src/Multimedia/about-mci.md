# <a name="about-mci"></a>О MCI

MCI предоставляет приложения с возможностями, не зависящими от устройства, для управления аудио-и визуальными устройствами. Приложение может использовать MCI для управления любым поддерживаемым устройством мультимедиа, в том числе устройствами аудио-и видеоадаптеров MIDI, устройствами CD Audio и цифровыми видео (воспроизведением видео).
