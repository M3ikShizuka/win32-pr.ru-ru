# <a name="info-command"></a>INFO, команда

Команда info Извлекает описание оборудования с устройства. Все устройства MCI распознают эту команду.

Чтобы отправить эту команду, вызовите функцию [**mciSendString**](/previous-versions//dd757161(v=vs.85)) с заданным параметром *лпсзкомманд* , как показано ниже.

``` syntax
_stprintf_s(
  lpszCommand, 
  TEXT("info %s %s %s"), 
  lpszDeviceID, 
  lpszInfoType, 
  lpszFlags
); 
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает нуль в случае успеха или ошибку в противном случае.

## <a name="examples"></a>Примеры

Следующая команда извлекает описание оборудования, связанного с устройством "мисаунд".

``` syntax
info mysound product
```
