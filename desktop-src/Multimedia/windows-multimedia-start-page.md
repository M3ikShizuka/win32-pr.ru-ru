# <a name="windows-multimedia"></a>Мультимедиа Windows

## <a name="purpose"></a>Назначение

Поддержка мультимедиа в Microsoft Windows позволяет приложениям использовать звук и видео.

## <a name="where-applicable"></a>Где применимо

Мультимедиа Windows можно использовать во всех приложениях на базе Windows.

## <a name="developer-audience"></a>Аудитория разработчиков

Windows мультимедиа предназначена для использования программистами C/C++. Знание графического пользовательского интерфейса Windows и архитектуры, управляемой сообщениями, является обязательным.
