# <a name="adding-user-controlled-playback"></a>Добавление User-Controlled воспроизведения

Можно добавить управляемое пользователем воспроизведение в существующее приложение, вызвав функцию [**мЦивндкреате**](/windows/desktop/api/Vfw/nf-vfw-mciwndcreatea) следующим образом:


```C++
MCIWndCreate(hwndParent, hInstModule, NULL, "filename.typ"); 
```



Параметры [**мЦивндкреате**](/windows/desktop/api/Vfw/nf-vfw-mciwndcreatea) указывают дескрипторы родительского окна и экземпляра модуля, связанного с окном мЦивнд. Они также указывают стили окна и имя файла (или имя устройства), связываемые с окном МЦивнд.

**МЦивндкреате** автоматически выполняет следующие действия, которые можно реализовать в коде для других классов окон:

1.  Зарегистрируйте класс окна МЦивнд.
2.  Создайте окно МЦивнд.
3.  Загрузить указанное содержимое.
4.  Установите текущую позиции воспроизведения в начале содержимого.
5.  Отображение элемента управления устройством.
6.  При необходимости Отобразите область воспроизведения окна.
