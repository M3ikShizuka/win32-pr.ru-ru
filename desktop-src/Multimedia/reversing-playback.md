# <a name="reversing-playback"></a>Обратное воспроизведение

Некоторые устройства поддерживают воспроизведение в обратном направлении. Содержимое такого устройства можно воспроизвести в обратном направлении с помощью макроса [**мЦивндплайреверсе**](/windows/desktop/api/Vfw/nf-vfw-mciwndplayreverse) . Этот макрос определяет область воспроизведения от текущей позицией воспроизведения до начала содержимого. Устройство цифрового видео МЦИАВИ. DRV, может воспроизводиться назад. Устройства, которые не могут воспроизводиться назад, например аудио компакт-диск, могут выдавать сообщение об ошибке при вызове **мЦивндплайреверсе** .
