# <a name="drawdib"></a>дравдиб

Функции Дравдиб предоставляют высокопроизводительные возможности рисования изображений для независимых от устройства точечных рисунков (DIB). Функции Дравдиб поддерживают рисунки размером 8-разрядных, 16-разрядных, 24-битных и 32-битных глубин изображения.

Функции Дравдиб записываются непосредственно в видеопамять. Они не полагаются на функции интерфейса графических устройств (GDI).
