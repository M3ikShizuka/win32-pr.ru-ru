# <a name="mixer-device-queries"></a>Запросы устройства микшера

Службы микшера предоставляют функции для определения количества устройств микшера, имеющихся в системе, и возможностей устройств. Для определения идентификатора устройства микшера можно также использовать функцию служб микшера.

Функцию [**миксержетнумдевс**](/windows/win32/api/mmeapi/nf-mmeapi-mixergetnumdevs) можно использовать для определения количества устройств микшера, имеющихся в системе. Устройства микшера идентифицируются по идентификатору устройства. Идентификаторы устройств определяются неявно на количестве устройств, имеющихся в данной системе. Они находятся в диапазоне от нуля до числа имеющихся устройств.

Перед использованием микшера устройства необходимо определить его возможности. Звуковые возможности могут меняться от одного компьютера мультимедиа к другому, поэтому приложения должны работать с различными звуковыми устройствами.

Для определения возможностей микшерных устройств можно использовать функцию [**миксержетдевкапс**](/windows/win32/api/mmeapi/nf-mmeapi-mixergetdevcaps) . Эта функция заполняет структуру [**миксеркапс**](/windows/win32/api/mmeapi/ns-mmeapi-mixercaps) данными о возможностях указанного устройства.

Функция [**миксержетид**](/windows/win32/api/mmeapi/nf-mmeapi-mixergetid) извлекает идентификатор устройства микшера звука, связанный с указанным маркером устройства. Например, эту функцию можно использовать для получения идентификатора устройства для звукового микшера, а затем использовать идентификатор устройства для настройки тома или для показа другого элемента управления.
