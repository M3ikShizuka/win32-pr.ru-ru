# <a name="mciwnd-window-user-interface"></a>Пользовательский интерфейс окна МЦивнд

МЦивнд предоставляет дополнительные возможности для настройки вида окна МЦивнд, настройки поведения приложения и настройки производительности воспроизведения. В окно МЦивнд включены следующие функции:

-   Панель инструментов с кнопками **воспроизведения**, **окончания**, **записи** и **меню**
-   Значение TrackBar, которое управляет размещением в содержимом воспроизведения
-   Всплывающее меню с общими командами
-   Область воспроизведения видео-и других устройств, отображающих изображения

На следующем рисунке показано начальное состояние воспроизведения видео, управляемого пользователем. Используемый пример файла — CLOCK.AVI.

![Изображение clock.avi](images/mciwnd1.gif)

Окно МЦивнд содержит область воспроизведения видео и других устройств, отображающих изображения во время воспроизведения. МЦивнд пропускает область воспроизведения на устройствах аудио-Audio, в модулях MIDI и других устройствах, которые не записываются на экран. На следующем рисунке показана область воспроизведения аудио-звука.

![изображение окна мЦивнд](images/mciwnd4.gif)

Кнопка **воспроизвести** находится в левом нижнем углу окна мЦивнд. Он отображается при остановке содержимого. Пользователь может воспроизвести содержимое следующими способами:

-   Чтобы воспроизвести содержимое из текущей позицией воспроизведения, нажмите кнопку **воспроизвести** .
-   Чтобы воспроизвести содержимое в полноэкранном режиме с текущей позицией воспроизведения, нажмите кнопку **Воспроизведение** , удерживая нажатой клавишу CTRL.
-   Чтобы воспроизвести содержимое в обратном направлении от текущей позиции воспроизведения, нажмите кнопку **Воспроизведение** , удерживая нажатой клавишу Shift.

Кнопка **меню** , расположенная рядом с кнопкой **Воспроизведение** , активирует меню, позволяющее пользователю открывать и закрывать AVI-файлы, а также настраивать размер изображения, скорость воспроизведения и громкость. (Пользователь также может активировать меню, щелкая правой кнопкой мыши каждый раз, когда курсор находится в клиентской области окна.) Меню также содержит команды для изменения конфигурации текущего устройства, копирования содержимого воспроизведения в буфер обмена и для выполнения команд MCI.

Значение TrackBar справа от кнопки **меню** представляет продолжительность воспроизведения (или записи) содержимого. Ползунок на TrackBar представляет текущую положение воспроизведения внутри содержимого. Когда ползунок располагается в левом конце TrackBar, текущая позиция воспроизведения является началом содержимого. Пользователь может переместиться в другое место в содержимом, перетащив ползунок на TrackBar. Кнопка " **Закрыть** " находится в левом нижнем углу окна мЦивнд. Он отображается при воспроизведении содержимого. На следующем рисунке показано, как выполняется воспроизведение видео.

![изображение воспроизведения видео](images/mciwnd2.gif)

Элементы управления МЦивнд также могут включать кнопку **записи** для устройств, которые могут записываться. Кнопка **записи** помечается красным кружком и отображается только в том случае, если устройство поддерживает запись.

> [!Note]  
> Окно воспроизведения должно быть высвоено на границе с четырьмя пикселями для наилучшей производительности воспроизведения видео. как правило, Windows автоматически выстраивает окно при его создании. Если пользователь перемещает или растягивает окно от первоначального положения, скорость воспроизведения видео может снизиться на половину.
