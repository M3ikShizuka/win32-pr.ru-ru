# <a name="looping-playback"></a>Циклическое воспроизведение

Циклический перебор звука контролируется членами **двлупс** и **dwFlags** в структурах [**вавехдр**](/windows/win32/api/mmeapi/ns-mmeapi-wavehdr) , передаваемых на устройство с помощью функции [**вавеаутврите**](/windows/win32/api/mmeapi/nf-mmeapi-waveoutwrite) . Используйте флаги **вхдр \_ Бегинлуп** и **Вхдр \_ ендлуп** в элементе **dwFlags** , чтобы указать начальный и конечный блоки данных для циклов.

Чтобы перебрать один блок данных, укажите оба флага для одного и того же блока. Чтобы указать число циклов, используйте элемент **двлупс** в структуре [**вавехдр**](/windows/win32/api/mmeapi/ns-mmeapi-wavehdr) для первого блока в цикле.

Для завершения циклического звука можно вызвать функцию [**вавеаутбреаклуп**](/windows/win32/api/mmeapi/nf-mmeapi-waveoutbreakloop) .
