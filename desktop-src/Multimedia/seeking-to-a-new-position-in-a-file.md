# <a name="seeking-to-a-new-position-in-a-file"></a>Поиск новой позицией в файле

В следующем примере показано перемещение в начало открытого файла с помощью функции [**ммиосик**](/windows/win32/api/mmiscapi/nf-mmiscapi-mmioseek) .


```C++
mmioSeek(hFile, 0L, SEEK_SET); 
```



В следующем примере перемещается в конец открытого файла.


```C++
mmioSeek(hFile, 0L, SEEK_END); 
```



В следующем примере перемещается в точку с 10 байтами из конца открытого файла.


```C++
mmioSeek(hFile, -10L, SEEK_END); 

```
