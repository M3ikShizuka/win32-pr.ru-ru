# <a name="mci"></a>MCI

Интерфейс управления мультимедиа (MCI) предоставляет стандартные команды для воспроизведения мультимедийных устройств и записи файлов ресурсов мультимедиа. Эти команды являются универсальным интерфейсом почти всех типов устройств мультимедиа.
