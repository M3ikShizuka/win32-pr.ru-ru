# <a name="drv_load-message"></a>\_Сообщение о загрузке DRV

Уведомляет драйвер о том, что он был загружен. Драйвер должен убедиться в наличии оборудования и вспомогательных драйверов, необходимых для правильной работы.

## <a name="parameters"></a>Параметры

Параметр *хдрвр* всегда равен нулю. Параметры *двдриверид*, *lParam1* и *lParam2* не используются.

## <a name="return-value"></a>Возвращаемое значение

Возвращает ненулевое значение в случае успеха или ноль в противном случае.

## <a name="remarks"></a>Remarks

Сообщение **о \_ загрузке DRV** всегда является первым сообщением, которое получает драйвер устройства.
