# <a name="video-for-windows"></a>Видео для Windows

видео для Windows (VFW) предоставляет функции, позволяющие приложению обрабатывать данные видео. VFW появился в 16-разрядной Windows. Многие его функции были заменены компонентами DirectX. Дополнительные сведения см. в документации DirectX.
