# <a name="video-for-windows"></a>Видео для Windows

Видео для Windows (VFW) предоставляет функции, позволяющие приложению обрабатывать данные видео. VFW появился в 16-разрядной версии Windows. Многие его функции были заменены компонентами DirectX. Дополнительные сведения см. в документации DirectX.
