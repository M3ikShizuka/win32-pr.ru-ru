# <a name="device-types"></a>Типы устройств

MCI распознает базовый набор *типов устройств*. Тип устройства — это набор драйверов MCI, использующих общий набор команд и используемый для управления аналогичными устройствами мультимедиа или файлами данных. Многие команды MCI, например [**Open**](open.md) ([**MCI \_ Open**](mci-open.md)), потребовали указать тип устройства.

В следующей таблице перечислены определенные типы устройств. Текущая реализация MCI включает наборы команд для подмножества этих устройств.



| Тип устройства      | Константа                      | Описание                                      |
|------------------|-------------------------------|--------------------------------------------------|
| **кдаудио**      | \_ \_ аудио компакт-диск MCI девтипе \_       | Проигрыватель CD Audio                                  |
| **включая**          | MCI \_ девтипе \_ dat             | Ленточный проигрыватель для цифрового аудио                        |
| **дигиталвидео** | \_устройство MCI девтипе \_ Digital \_ Video  | Цифровое видео в окне (не на основе GDI)        |
| **иной**        | \_ДЕВТИПЕ MCI \_           | Неопределенное устройство MCI                             |
| **объединения**      | наложение MCI \_ девтипе \_         | Наложение устройства (аналоговый видеоролик в окне)        |
| **scanner**      | \_сканер ДЕВТИПЕ \_ MCI         | Сканер изображений                                    |
| **sequencer**    | Программа MCI \_ девтипе \_ Sequencer       | Секвенсор MIDI                                   |
| **видеомагнитофон**          | \_видеомагнитофон ДЕВТИПЕ \_ MCI             | Устройство записи видеокассет или проигрыватель                |
| **видеодиск**    | \_ВИДЕОДИСК MCI девтипе \_       | Проигрыватель видеодиск                                 |
| **вавеаудио**    | \_ \_ звуковая волна MCI девтипе \_ | Звуковое устройство, воспроизводящее оцифрованные файлы звукозаписи |



 

В этом документе имена типов устройств выделены полужирным шрифтом. Имена типов устройств используются с интерфейсом командной строки. Константы типа устройства используются с интерфейсом командной строки.
