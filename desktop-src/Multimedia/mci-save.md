# <a name="mci_save-command"></a>Команда "Сохранить" MCI \_

Команда MCI \_ Save сохраняет текущий файл. Устройства, которые изменяют файлы, не должны уничтожить исходную копию до получения сообщения о сохранении. Эта команда распознает видео-наложение и аудио-изображения. Несмотря на то, что Цифровые видеоустройства и средства нумерации MIDI также распознают эту команду, драйверы МЦИАВИ и МЦИСЕК не реализуют их.

Чтобы отправить эту команду, вызовите функцию [**мЦисендкомманд**](/previous-versions//dd757160(v=vs.85)) со следующими параметрами.


```C++
MCIERROR mciSendCommand(
  MCIDEVICEID wDeviceID, 
  MCI_SAVE, 
  DWORD dwFlags, 
  (DWORD) (LPMCI_SAVE_PARMS ) lpSave
);
```

</dd> <dt>

</dd> <dt>

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает нуль в случае успеха или ошибку в противном случае.

## <a name="remarks"></a>Remarks

Эта команда поддерживается устройствами, которые возвращают **значение true** при вызове команды [MCI \_ ЖЕТДЕВКАПС](mci-getdevcaps.md) с \_ \_ флагом MCI жетдевкапс \_ .

Следующий дополнительный флаг применяется ко всем устройствам, поддерживающим [MCI \_ Save](/windows):

<dl> <dt>

<span id="MCI_SAVE_FILE"></span><span id="mci_save_file"></span>\_файл для сохранения MCI \_
</dt> <dd>

Элемент **лпфиленаме** структуры, идентифицируемой *лпсаве* , содержит адрес буфера, содержащего имя файла назначения.

</dd> </dl>

С типом устройства **дигиталвидео** используются следующие дополнительные флаги:

<dl> <dt>

<span id="MCI_DGV_RECT"></span><span id="mci_dgv_rect"></span>MCI \_ ДГВ \_ Rect
</dt> <dd>

Элемент **RC** структуры, идентифицируемой *лпсаве* , содержит допустимый прямоугольник. Прямоугольник указывает область буфера фрейма, которая будет сохранена в указанном файле. Первая пара координат задает верхний левый угол прямоугольника. Вторая пара указывает ширину и высоту. Устройства с цифровыми видео должны использовать команду [ \_ захвата MCI](mci-capture.md) для записи содержимого буфера кадров. (Устройства с наложением видео также должны использовать MCI \_ ЗАХВАТИТЬ.) Этот флаг предназначен для обеспечения совместимости с существующим набором команд оверлея видео MCI.

</dd> <dt>

<span id="MCI_DGV_SAVE_ABORT"></span><span id="mci_dgv_save_abort"></span>\_ \_ прерывание при СОХРАНЕНИИ ДГВ MCI \_
</dt> <dd>

Останавливает операцию сохранения. Это должен быть единственный флаг.

</dd> <dt>

<span id="MCI_DGV_SAVE_KEEPRESERVE"></span><span id="mci_dgv_save_keepreserve"></span>MCI \_ ДГВ \_ Сохранить \_ кипресерве
</dt> <dd>

Неиспользуемое место на диске, оставшееся до первоначальной команды, [ \_ зарезервированной MCI](mci-reserve.md) , не освобождается.

</dd> </dl>

Для устройств с цифровыми видео параметр *лпсаве* указывает на структуру [**MCI \_ ДГВ \_ Save \_ пармс**](/windows/desktop/api/Digitalv/ns-digitalv-mci_dgv_save_parmsa) .

Для типа устройства **оверлея** используется следующий дополнительный флаг:

<dl> <dt>

<span id="MCI_OVLY_RECT"></span><span id="mci_ovly_rect"></span>MCI \_ овли \_ Rect
</dt> <dd>

Элемент **RC** структуры, идентифицируемой *лпсаве* , содержит допустимый отображаемый прямоугольник, указывающий область видеобуфера для сохранения.

</dd> </dl>

Для устройств с наложением параметр *лпсаве* указывает на структуру [**MCI \_ овли \_ Save \_ пармс**](/previous-versions//dd743447(v=vs.85)) .
