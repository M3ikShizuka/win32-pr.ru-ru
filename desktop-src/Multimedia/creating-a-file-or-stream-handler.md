# <a name="creating-a-file-or-stream-handler"></a>Создание файла или обработчика потока

В приложении, написанном на языке C, обработчик файлов или потоков обычно создает функцию для каждого метода. Приложение обращается к этим функциям через массив указателей функций, создаваемых обработчиком потока. Структура **иавистреамвтбл** содержит массив указателей функций. Обработчик потока может назначить любое имя, которое он хочет создать для методов. Расположение указателя функции в структуре подразумевает соответствие функции методу. Например, первый указатель функции в структуре соответствует методу **QueryInterface** . Обработчик потоков должен предоставлять все функции интерфейса.
