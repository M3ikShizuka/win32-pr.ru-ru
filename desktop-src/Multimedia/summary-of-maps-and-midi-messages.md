# <a name="summary-of-maps-and-midi-messages"></a>Сводка по картам и сообщениям MIDI

Ниже приведены байты состояния для сообщений MIDI и типы карт, влияющие на каждое сообщение.



| Состояние MIDI | Описание               | Типы карт                |
|-------------|---------------------------|--------------------------|
| 0x80-0x8F   | Заметка отключена                  | Карты каналов, карты ключей   |
| 0x90-0x9F   | Примечания по                    | Карты каналов, карты ключей   |
| 0X82 — 0xAF   | Полифоник — ключ афтертауч | Карты каналов, карты ключей   |
| 0xB0-0xBF   | Изменение элемента управления            | Карты каналов, карты исправлений |
| 0xC0-0xCF   | Изменение программы            | Карты каналов, карты исправлений |
| 0xD0 — 0xDF   | Канал афтертауч        | Карты каналов             |
| 0xE0 — 0xEF   | Шаг — изменение изгиба         | Карты каналов             |
| 0xF0 — 0xFF   | Система                    | Не сопоставлена               |



 

-   Старшие четыре бита представляют значение состояния. Младшие четыре бита представляют сведения о канале.
-   Карты исправлений влияют только на контроллер 7 (основной том).
-   Системные сообщения отправляются на все устройства, перечисленные в карте каналов.
