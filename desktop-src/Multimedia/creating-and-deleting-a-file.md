# <a name="creating-and-deleting-a-file"></a>Создание и удаление файла

Чтобы создать файл, присвойте параметру *двопенфлагс* функции [**ммиупен**](/windows/win32/api/mmiscapi/nf-mmiscapi-mmioopen) значение MMIO \_ Create. В следующем примере создается файл, который открывается для чтения и записи.


```C++
HMMIO hFile; 

hFile = mmioOpen("NEWFILE.TXT", NULL, MMIO_CREATE | MMIO_READWRITE); 
if (hFile != NULL) 
    // File created successfully. 
else 
    // File cannot be created. 
```



Если создаваемый файл уже существует, он будет обрезан до нулевой длины.

Чтобы удалить файл, присвойте параметру *двопенфлагс* функции **ммиупен** значение MMIO \_ Delete. После удаления файла его нельзя восстановить с помощью стандартных средств. Если приложение удаляет файл по запросу пользователя, запросите пользователя перед удалением файла, чтобы убедиться, что пользователь хочет его удалить.
