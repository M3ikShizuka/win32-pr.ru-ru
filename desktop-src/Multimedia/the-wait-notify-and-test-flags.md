# <a name="the-wait-notify-and-test-flags"></a>Флаги ожидания, уведомления и проверки

Большинство команд MCI включают флаги, которые изменяют команду. Флаги «Wait» ( \_ Wait MCI) и «notify notify» (уведомления MCI \_ ) являются общими для каждой команды. Флаг "тест" ( \_ тест MCI) доступен для устройств цифрового видео и видеомагнитофона. В следующих разделах описывается использование этих флагов.

-   [Флаг ожидания](the-wait-flag.md)
-   [Флаг уведомления](the-notify-flag.md)
-   [Флаг теста](the-test-flag.md)
