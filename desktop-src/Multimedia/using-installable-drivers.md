# <a name="using-installable-drivers"></a>Использование устанавливаемых драйверов

Для предоставления приложениям или библиотекам DLL стандартного способа доступа к устройству или набору полезных подпрограмм используются устанавливаемые драйверы. В следующих разделах показано, как создать устанавливаемый драйвер с помощью функции [дриверпрок](/windows/win32/api/mmiscapi/nc-mmiscapi-driverproc) , а также как открыть устанавливаемый драйвер и направить его на выполнение полезных задач.
