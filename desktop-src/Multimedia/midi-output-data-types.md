# <a name="midi-output-data-types"></a>Типы выходных данных MIDI

Windows определяет следующие типы данных для выходных функций MIDI.



| Значение                              | Значение                                                                              |
|------------------------------------|--------------------------------------------------------------------------------------|
| **хмидиаут**                       | Маркер устройства вывода MIDI.                                                      |
| [**мидихдр**](/windows/win32/api/mmeapi/ns-mmeapi-midihdr)         | Заголовок для блока безсистемных или потоковых данных MIDI.                          |
| [**мидиауткапс**](/windows/win32/api/mmeapi/ns-mmeapi-midioutcaps) | Структура, используемая для получения сведений о возможностях конкретного выходного устройства MIDI. |
