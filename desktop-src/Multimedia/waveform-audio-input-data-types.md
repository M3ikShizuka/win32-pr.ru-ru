# <a name="waveform-audio-input-data-types"></a>Типы входных данных Waveform-Audio

Для входных функций аудио-сигнала определены следующие типы данных.



| Тип                                 | Описание                                                                                                                                                     |
|--------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **хвавеин**                          | Маркер открытого устройства ввода аудио-сигнала.                                                                                                                  |
| [**вавеформатекс**](/windows/win32/api/mmeapi/ns-mmeapi-waveformatex) | Структура, указывающая форматы данных, поддерживаемые устройством ввода аудио-сигнала. Эта структура также используется для устройств вывода аудио-звука. |
| [**вавехдр**](/windows/win32/api/mmeapi/ns-mmeapi-wavehdr)           | Структура, используемая в качестве заголовка для блока входных данных аудио-сигнала. Эта структура также используется для устройств вывода аудио-звука.                             |
| [**вавеинкапс**](/windows/win32/api/mmeapi/ns-mmeapi-waveincaps)     | Структура, используемая для получения сведений о возможностях конкретного входного устройства аудио.                                                                   |
