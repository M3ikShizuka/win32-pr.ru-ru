# <a name="sharing-an-io-procedure-with-other-applications"></a>Совместное использование процедуры ввода-вывода с другими приложениями

Если вы хотите совместно использовать процедуру ввода-вывода с другими приложениями, необходимо написать библиотеку динамической компоновки (DLL) для приложения. Вы можете совместно использовать процедуру ввода-вывода одним из следующих способов:

-   Включите код для процедуры ввода-вывода в библиотеку DLL.
-   Создайте функцию в библиотеке DLL, которая вызывает функцию [**ммиоинсталлиопрок**](/windows/win32/api/mmiscapi/nf-mmiscapi-mmioinstallioproc) для установки процедуры ввода-вывода.
-   Экспортируйте эту функцию в файл определений модуля библиотеки DLL.

Чтобы использовать общую процедуру ввода-вывода, приложение должно сначала вызвать функцию в библиотеке DLL, чтобы установить процедуру ввода-вывода.
