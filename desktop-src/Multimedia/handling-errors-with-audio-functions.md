# <a name="handling-errors-with-audio-functions"></a>Обработка ошибок с помощью звуковых функций

Функции аудио-и вспомогательного аудио при возникновении ошибки возвращают ненулевое значение. Windows предоставляет функции, преобразующие эти значения ошибок в текстовые описания ошибок. Приложение по-прежнему должно проверять значения ошибок, чтобы определить, как продолжать работу, но текстовые описания ошибок можно использовать в диалоговых окнах, описывающих ошибки для пользователей.

Для получения текстовых описаний значений ошибок звука можно использовать следующие функции:



| Функция                                           | Описание                                                                 |
|----------------------------------------------------|-----------------------------------------------------------------------------|
| [**вавеинжетеррортекст**](/windows/win32/api/mmeapi/nf-mmeapi-waveingeterrortext)   | Получает текстовое описание указанной ошибки ввода аудио-сигнала.  |
| [**вавеаутжетеррортекст**](/windows/win32/api/mmeapi/nf-mmeapi-waveoutgeterrortext) | Получает текстовое описание указанной ошибки вывода звуковой волны. |



 

Единственными звуковыми функциями, не возвращающими значения ошибок, являются [**ауксжетнумдевс**](/windows/win32/api/mmeapi/nf-mmeapi-auxgetnumdevs), [**вавеинжетнумдевс**](/windows/win32/api/mmeapi/nf-mmeapi-waveingetnumdevs)и [**вавеаутжетнумдевс**](/windows/win32/api/mmeapi/nf-mmeapi-waveoutgetnumdevs). Эти функции возвращают нуль, если в системе нет устройств или если они столкнулись с ошибками.
