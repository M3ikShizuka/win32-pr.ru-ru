# <a name="mciwnd-window-class"></a>Класс окна МЦивнд

МЦивнд — это класс окна для управления устройствами мультимедиа. Библиотека функций, сообщений и макросов, связанных с МЦивнд, предоставляет простой метод для добавления возможностей воспроизведения или записи мультимедийных данных в приложения.
