# <a name="max-macro"></a>max - макрос

Макрос **Max** сравнивает два значения и возвращает значение большего. Тип данных может быть любым числовым типом данных, подписанным или неподписанным. Тип данных аргументов и возвращаемое значение одинаковы.

## <a name="syntax"></a>Синтаксис


```C++
 max(
    value1,
    value2
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*value1* 
</dt> <dd>

Задает первое из двух значений.

</dd> <dt>

*value2* 
</dt> <dd>

Задает второе из двух значений.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение — это большее из двух указанных значений.

## <a name="remarks"></a>Комментарии

Макрос **Max** определяется следующим образом:


```C++
#define max(a, b)  (((a) > (b)) ? (a) : (b)) 
```
