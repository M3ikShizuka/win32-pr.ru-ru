# <a name="lists"></a>Списки

Элементы управления "список" предоставляют одновыборные или множественные состояния для сложных звуковых линий. Эти элементы управления используют [**\_ логическую структуру миксерконтролдетаилс**](/previous-versions//dd757295(v=vs.85)) для извлечения и установки свойств элементов управления. Структура [**миксерконтролдетаилс \_ листтекст**](/previous-versions//dd757296(v=vs.85)) также используется для получения всех текстовых описаний элемента управления с несколькими элементами. В следующей таблице описаны типы элементов управления "список".



| Control           | Описание                                                                                                                                                                                                                                                                      |
|-------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Один выбор     | Разрешает выбор элемента управления только по одному элементу за раз. В отличие от элемента управления мультиплексирования, этот элемент управления можно использовать для управления более чем строками исходного кода. Например, этот элемент управления можно использовать для выбора фильтра с низким уровнем передачи из списка фильтров, поддерживаемых устройством микшера. |
| Мультиплексор (МУЛЬТИПЛЕКСОР) | Ограничит выбор строки до одной строки исходного кода за раз.                                                                                                                                                                                                                       |
| Множественный выбор   | Позволяет пользователю выбрать несколько элементов из списка одновременно. В отличие от элемента управления микшера, элемент управления множественного выбора можно использовать для управления более чем строками исходного кода.                                                                                                  |
| Громк             | Позволяет пользователю выбирать исходные строки из списка одновременно.                                                                                                                                                                                                               |
