# <a name="joystick-notifications"></a>Уведомления джойстика

Можно записать прямые джойстики для отправки в функцию с помощью функции [**жойсеткаптуре**](/windows/win32/api/joystickapi/nf-joystickapi-joysetcapture) . Только одно приложение за раз может записывать сообщения с джойстика, но вы можете запросить джойстик из другого приложения с помощью функции [**жойжетпос**](/windows/win32/api/joystickapi/nf-joystickapi-joygetpos) или [**жойжетпосекс**](/windows/win32/api/joystickapi/nf-joystickapi-joygetposex) .

> [!Note]  
> Возможно, джойстику не удается связаться с приложением, которое записало джойстик, если второе приложение использует **жойжетпос** или **жойжетпосекс** для отправки запросов к джойстику примерно в то же время, когда отправляется сообщение. В этом случае второе приложение может перехватить сообщение.

 

Если вы хотите записывать сообщения из двух джойстиков, подключенных к системе, используйте [**жойсеткаптуре**](/windows/win32/api/joystickapi/nf-joystickapi-joysetcapture) дважды, по одному разу для каждого джойстика. Окно получает отдельные и отдельные сообщения для каждого устройства.

Захваченный джойстик можно освободить с помощью функции [**жойрелеасекаптуре**](/windows/win32/api/joystickapi/nf-joystickapi-joyreleasecapture) . Если приложение не освобождает джойстик перед завершением, джойстик автоматически выпустится вскоре после уничтожения окна сбора данных.

Невозможно захватить неподключенный джойстик. Функция **жойсеткаптуре** возвращает жойерр \_ , не подключенную, если указанное устройство не подключено.
