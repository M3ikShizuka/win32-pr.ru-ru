# <a name="mm_acm_filterchoose-message"></a>\_Сообщение mm ACM \_ филтерчусе

Сообщение **mm \_ ACM \_ филтерчусе** уведомляет о функции-обработчике диалогового окна [**акмфилтерчусе**](/windows/desktop/api/Msacm/nf-msacm-acmfilterchoose) перед добавлением элемента в один из трех раскрывающихся списков. Это сообщение позволяет приложению настроить параметры, доступные в пользовательском интерфейсе.


```C++
        MM_ACM_FILTERCHOOSE
        wParam = (WPARAM) wDropDown
        lParam = (LONG) lCustom
      
```



## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="wDropDown"></span><span id="wdropdown"></span><span id="WDROPDOWN"></span>*вдропдовн*
</dt> <dd>

Инициализируется раскрывающийся список и операция проверки или добавления.



| Требование | Значение |

| \_Пользовательская \_ Проверка филтерчусе    | Параметр *lParam* является указателем на структуру [**вавефилтер**](/windows/desktop/api/Mmreg/ns-mmreg-wavefilter) , которую необходимо добавить в раскрывающийся список настраиваемое имя.                                                                                                   |
| \_Добавление фильтра \_ филтерчусе       | Параметр *lParam* является указателем на буфер, который принимает структуру [**вавефилтер**](/windows/desktop/api/Mmreg/ns-mmreg-wavefilter) , которая будет добавлена в раскрывающийся список фильтра. Приложение должно скопировать структуру фильтра для добавления в этот буфер. |
| \_Проверка фильтра \_ филтерчусе    | Параметр *lParam* является указателем на структуру [**вавефилтер**](/windows/desktop/api/Mmreg/ns-mmreg-wavefilter) , которую необходимо добавить в раскрывающийся список фильтр.                                                                                                        |
| \_Добавление филтерчусе филтертаг \_    | Параметр *lParam* является указателем на **DWORD** , который принимает тег фильтра звуковой волны, который будет добавлен в раскрывающийся список тег фильтра.                                                                                        |

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если приложение обрабатывает это сообщение или **значение false** в противном случае.

## <a name="remarks"></a>Комментарии

Если приложение обрабатывает \_ операцию добавления фильтра филтерчусе \_ , размер буфера памяти, указанный в параметре *lParam* , будет определен из функции [**акмметрикс**](/windows/desktop/api/Msacm/nf-msacm-acmmetrics) .

Если приложение обрабатывает операцию проверки, приложение должно предшествовать возвращаемому значению с параметром **SetWindowLong** (HWND, DWL \_ МСГРЕСУЛТ, (Long) **false**), чтобы диалоговое окно не выходило из списка этого выбора или с **SetWindowLong** (HWND, DWL \_ мсгресулт, (Long)**true**), чтобы разрешить диалоговому окну Вывод этого выбора. При обработке операции добавления приложение должно предшествовать возврату с помощью **SetWindowLong** (HWND, DWL \_ МСГРЕСУЛТ, (Long)**false**), чтобы указать, что больше нет необходимых дополнений или с **SetWindowLong** (HWND, DWL \_ мсгресулт, (Long)**true**), если требуются дополнительные дополнения.
