# <a name="the-midi-mapper-architecture"></a>Архитектура сопоставителя MIDI

Средство сопоставления MIDI использует карту установки MIDI для определения способа преобразования и перенаправления получаемых сообщений. Карта установки MIDI состоит из следующих типов карт.

-   [Схема канала](the-channel-map.md)
-   [Карты исправлений](patch-maps.md)
-   [ключевые Карты](key-maps.md)
