# <a name="looping-playback-for-mciwnd"></a>Циклическое воспроизведение для МЦивнд

МЦивнд поддерживает воспроизведение в виде непрерывного цикла. Вы можете воспроизвести содержимое файла или устройства несколько раз в цикле с помощью макроса [**мЦивндсетрепеат**](/windows/desktop/api/Vfw/nf-vfw-mciwndsetrepeat) в сочетании с кнопкой **Воспроизведение** на панели инструментов. Устройство воспроизведения видео, МЦИАВИ, поддерживает циклы воспроизведения. Чтобы определить, активировано ли непрерывное воспроизведение, используйте макрос [**мЦивнджетрепеат**](/windows/desktop/api/Vfw/nf-vfw-mciwndgetrepeat) .
