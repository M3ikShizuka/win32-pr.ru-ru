# <a name="creating-temporary-streams"></a>создание временных Потоки

Временные потоки могут быть полезными несколькими способами. Можно использовать временный поток в качестве рабочего потока, например, при изменении формата потока. Или можно создать временный поток для хранения скопированных частей других потоков.

В памяти можно создать поток, не связанный с каким-либо файлом, с помощью функции [**авистреамкреате**](/windows/desktop/api/Vfw/nf-vfw-avistreamcreate) . Эта функция возвращает адрес интерфейса в новом потоке в указанном расположении и используется внутри другими функциями, которые создают временные потоки.

Вы можете создать сжатый поток из несжатого потока с помощью функции [**авимакекомпресседстреам**](/windows/desktop/api/Vfw/nf-vfw-avimakecompressedstream) . Вы определяете поток для сжатия, метод сжатия и параметры сжатия, а также обработчик для нового потока.

Когда вы завершите использование потока, созданного с помощью **авистреамкреате** или **авимакекомпресседстреам**, закройте поток, используя функцию [**авистреамрелеасе**](/windows/desktop/api/Vfw/nf-vfw-avistreamrelease) . **Авистреамрелеасе** освобождает ресурсы, используемые временным потоком.
