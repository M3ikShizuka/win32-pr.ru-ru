# <a name="about-avifile-functions-and-macros"></a>О функциях и макросах Авифиле

Функции и макросы Авифиле обрабатывали информацию в файлах на основе времени как один или несколько *потоков данных* вместо блоков данных, называемых фрагментами. Потоки данных ссылаются на компоненты файла на основе времени. Файл AVI может содержать несколько различных типов данных, таких как последовательность видео, звуковая дорожка на английском языке и французская звуковая дорожка. С помощью Авифиле приложение может обращаться к каждому из этих компонентов отдельно.

> [!Note]  
> Несмотря на то что функции и макросы Авифиле работают с любым файлом Metallica, в этом обзоре показано, как использовать его только с файлами AVI. AVI-файлы обычно представляют собой временные файлы, используемые в макросах и функциях Авифиле.

 

Функции и макросы Авифиле содержатся в библиотеке динамической компоновки. Чтобы инициализировать библиотеку, используйте функцию [**авифилеинит**](/windows/desktop/api/Vfw/nf-vfw-avifileinit) . После инициализации библиотеки можно использовать любую из функций или макросов Авифиле. Чтобы освободить библиотеку, используйте функцию [**авифиликсит**](/windows/desktop/api/Vfw/nf-vfw-avifileexit) . Авифиле поддерживает подсчет ссылок на приложения, использующие библиотеку, но не на тех, для которых она была выпущена. Приложения должны сбалансировать каждое использование **авифилеинит** с помощью вызова **авифиликсит** , чтобы полностью освободить библиотеку после того, как каждое приложение завершит его использование.
