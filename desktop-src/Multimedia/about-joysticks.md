# <a name="about-joysticks"></a>О джойстиках

Джойстик является вспомогательным устройством ввода для приложений, которые предоставляют альтернативы использованию клавиатуры и мыши. Джойстик предоставляет сведения о позиционировании в системе координат с абсолютным максимальным и минимальным значениями на каждой оси перемещения.

Службы джойстика загружаются при запуске операционной системы. Службы джойстика могут одновременно отслеживать два джойстика, каждый из которых имеет перемещение с двумя или тремя осями. Каждый джойстик может иметь до четырех кнопок. Функции джойстика можно использовать для определения возможностей драйвера джойстиков и джойстика. Кроме того, можно обработать сведения о положении и кнопке джойстика, запросив джойстик напрямую или записывая джойстик и обрабатывая сообщения из него. Второй метод проще, так как приложению не нужно вручную запрашивать джойстик или записывать время создания запросов с регулярными интервалами.
