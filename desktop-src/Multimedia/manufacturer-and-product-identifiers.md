# <a name="manufacturer-and-product-identifiers"></a>Идентификаторы производителя и продукта

В этом приложении документируется изготовитель и идентификаторы продуктов, определенные для мультимедийных приложений. Эти идентификаторы используются, когда приложение выдает запрос об установленных устройствах на компьютере. Чтобы использовать эти идентификаторы в приложении, включите ММРЕГ. H файл.

-   [Идентификаторы производителей](manufacturer-identifiers.md)
-   [Идентификаторы продуктов корпорации Майкрософт](microsoft-corporation-product-identifiers.md)
-   [Драйверы звуковых систем Microsoft Windows](microsoft-windows-sound-system-drivers.md)
-   [Идентификаторы продуктов](product-identifiers.md)

> [!Note]  
> Корпорация Майкрософт больше не поддерживает Центральный реестр для идентификаторов продуктов.
