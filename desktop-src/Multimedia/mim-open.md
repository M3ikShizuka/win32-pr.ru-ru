# <a name="mim_open-message"></a>MIM \_ ОТКРЫТЬ сообщение

**MIM \_ открытое** сообщение отправляется в функцию обратного вызова midi при открытии устройства ввода midi.


```C++
MIM_OPEN 
dwParam1 = reserved 
dwParam2 = reserved 
```
