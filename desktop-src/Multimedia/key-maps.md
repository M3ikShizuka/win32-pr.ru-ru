# <a name="key-maps"></a>Карты ключей

Каждая запись в таблице преобразования с картой исправлений может иметь связанную карту ключей. Карты ключей влияют на сообщения о заметках, заметок и полифоник-Key-афтертауч. В карте ключей имеется таблица перевода с записью для каждого значения ключа MIDI 128. Например, если запись для значения ключа 60 имеет значение 72, средство сопоставления MIDI изменяет сообщения в формате MIDI, как показано на следующем рисунке.

![изображение сопоставителя MIDI](images/mmap-a06.gif)

Карты ключей полезны для синтезаторов, имеющих на основе ключей инструменты, имеющие определенный звук, назначенный каждому ключу. Карты ключей обычно назначаются первому исправлению в картах исправлений на каналах с помощью последствий (10 и 16).
