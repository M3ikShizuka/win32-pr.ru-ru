# <a name="function-data-types-and-return-values"></a>Типы данных и возвращаемые значения функции

Функции и макросы Авифиле используют обработчики файлов и потоков, реализованные с помощью технологии OLE. Стандартный тип данных функции OLE — **стдапи**, а функция возвращает значение **HRESULT** (ноль для успешного выполнения или ошибка в противном случае). Если функция возвращает значение, отличное от **HRESULT**, тип прототипа функции немного отличается от синтаксиса, который внедряет тип возвращаемого значения в круглые скобки после "стдапи \_ ". Например, функция, возвращающая значение **типа Long** Data, использует **стдапи \_ (Long)** в инструкции prototype.
