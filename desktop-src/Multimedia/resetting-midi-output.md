# <a name="resetting-midi-output"></a>Сброс выходных данных MIDI

Функция [**мидиаутресет**](/windows/win32/api/mmeapi/nf-mmeapi-midioutreset) отключает все заметки на всех каналах MIDI для УКАЗАННОГО устройства MIDI. Затем функция помечает все ожидающие системные буферы как выполненные и возвращает их приложению. Эта функция может быть полезной в приложении, которое использует выходные данные MIDI, чтобы предоставить пользователю возможность сбрасывать выходные данные MIDI.

> [!Note]  
> Завершение безсистемного сообщения без отправки ЕОКС (исключающий) байт может привести к проблемам с принимающим устройством. Функция **мидиаутресет** не ОТПРАВЛЯЕТ байт еокс, когда он завершает сообщение, исключающий систему, потому что приложения отвечают за это.
