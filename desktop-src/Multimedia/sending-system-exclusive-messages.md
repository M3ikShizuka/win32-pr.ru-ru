# <a name="sending-system-exclusive-messages"></a>Отправка сообщений System-Exclusive

Сообщения, исключающие систему MIDI — это единственные сообщения MIDI, которые не помещаются в одно значение даублеворд. Сообщения, исключающие из системы, могут иметь любую длину. Windows предоставляет функцию [**мидиаутлонгмсг**](/windows/win32/api/mmeapi/nf-mmeapi-midioutlongmsg) для отправки безсистемных сообщений на устройства вывода MIDI. Для указания блоков данных, исключающих систему MIDI, используйте структуру [**мидихдр**](/windows/win32/api/mmeapi/ns-mmeapi-midihdr) .

После отправки безсистемного блока данных с помощью **мидиаутлонгмсг** необходимо дождаться завершения работы драйвера устройства с блоком данных перед его освобождением. При отправке нескольких блоков данных необходимо отслеживать завершение каждого блока данных, чтобы вы могли получить сведения о том, когда отправлять дополнительные блоки. Сведения о различных методах мониторинга завершения блоков данных см. в разделе [Управление блоками данных MIDI](managing-midi-data-blocks.md).

> [!Note]  
> Любой байт состояния MIDI, отличный от сообщения системы в режиме реального времени, прерывает системное сообщение. При использовании нескольких блоков данных для отправки одного сообщения, эксклюзивного от системы, не отправляйте сообщения MIDI, отличные от системных сообщений в режиме реального времени между блоками данных.
