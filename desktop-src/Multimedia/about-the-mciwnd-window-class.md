# <a name="about-the-mciwnd-window-class"></a>О классе окна МЦивнд

Класс окна МЦивнд прост в использовании. С помощью одной функции [**мЦивндкреате**](/windows/desktop/api/Vfw/nf-vfw-mciwndcreatea) приложение может создать элемент управления, который воспроизводит любое устройство, использующее интерфейс управления мультимедиа (MCI). К таким устройствам относятся аудио CD, аудио-и видеоустройства, MIDI и видео.

Автоматизировать воспроизведение также можно быстро и легко. Используя только одну функцию и два макроса, приложение может создать окно МЦивнд с соответствующим устройством мультимедиа, воспроизвести устройство и закрыть как устройство, так и окно после завершения воспроизведения содержимого.

> [!Note]  
> На некоторых устройствах воспроизводится содержимое, хранящееся в файлах. Другие устройства, например устройства с компакт-дисков, воспроизводят содержимое, которое хранится на другом носителе. В целях ясности в этом обзоре используются оба обстоятельства: "воспроизведение устройства".
