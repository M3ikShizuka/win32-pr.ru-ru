# <a name="video-compression-manager"></a>Диспетчер сжатия видео

Диспетчер сжатия видео (ВКМ) предоставляет доступ к интерфейсу, используемому устанавливаемыми в процессе сжатия, для управления данными в режиме реального времени. Приложения могут использовать устанавливаемые программы сжатия для выполнения следующих задач:

-   Сжатие и распаковка данных видео.
-   Отправляйте данные, сжатые с помощью модуля подготовки отчетов, и нарисуйте их на экране.
-   Сжатие, распаковка и Рисование данных с помощью модулей подготовки отчетов, определяемых приложением.
-   Используйте модули подготовки отчетов для обработки текста и пользовательских данных.
