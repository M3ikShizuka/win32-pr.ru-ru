# <a name="sliders-windows-multimedia"></a>Ползунки (мультимедиа Windows)

Элементы управления "ползунок" обычно являются горизонтальными элементами управления, которые могут быть скорректированы влево или вправо. Эти элементы управления используют [**\_ подписанную структуру миксерконтролдетаилс**](/previous-versions//dd757297(v=vs.85)) для извлечения и установки свойств элементов управления. В следующей таблице описаны типы ползунков.



| Control    | Описание                                                                                                               |
|------------|---------------------------------------------------------------------------------------------------------------------------|
| Ползунок     | Имеет диапазон от – 32 768 до 32 767. Драйвер микшера определяет ограничения этого элемента управления.                               |
| Сдвиг        | Имеет диапазон от – 32 768 до 32 767. Драйвер микшера определяет ограничения для этого элемента управления, значение 0 в качестве значения среднего. |
| Ксаунд Pan | Обеспечивает расширенное управление звуком с помощью Ксаунд. Этот элемент управления имеет диапазон от – 15 до 15.                               |
