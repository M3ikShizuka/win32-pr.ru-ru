# <a name="about-waveform-audio"></a>О звуковой волны

Существует несколько способов добавления звука в приложение с помощью звуковой волны. Простейший метод, описанный в этом разделе, заключается в использовании функции [**PlaySound**](/previous-versions//dd743680(v=vs.85)) . Большая часть других элементов API аудио-сигнала имеет относительно низкий уровень.

Дополнительные сведения см. в следующих разделах:

-   [Простое воспроизведение звука](simple-audio-playback.md)
-   [Звуковая волна-интерфейс](waveform-audio-interface.md)
-   [Вспомогательный звуковой интерфейс](auxiliary-audio-interface.md)
-   [Звуковые блоки данных](audio-data-blocks.md)
-   [Обработка ошибок с помощью звуковых функций](handling-errors-with-audio-functions.md)

В дополнение к описанным здесь интерфейсам API для воспроизведения аудио-файлов и других мультимедийных данных, таких как звуки MIDI и видео, можно использовать интерфейс управления мультимедиа. Дополнительные сведения см. в разделе [MCI](mci.md).
