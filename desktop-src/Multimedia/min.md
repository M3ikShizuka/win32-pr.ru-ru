# <a name="min-macro"></a>min - макрос

Макрос **min** сравнивает два значения и возвращает меньшее из них. Тип данных может быть любым числовым типом данных, подписанным или неподписанным. Тип данных аргументов и возвращаемое значение одинаковы.

## <a name="syntax"></a>Синтаксис


```C++
 min(
    value1,
    value2
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*value1* 
</dt> <dd>

Задает первое из двух значений.

</dd> <dt>

*value2* 
</dt> <dd>

Задает второе из двух значений.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение представляет собой меньшее из двух указанных значений.

## <a name="remarks"></a>Комментарии

Макрос **min** определяется следующим образом:


```C++
#define min(a, b)  (((a) < (b)) ? (a) : (b)) 
```
