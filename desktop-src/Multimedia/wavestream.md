# <a name="wavestream"></a>вавестреам

Приложения используют функции обратного вызова вавестреам во время потоковой записи для обработки звуковых буферов. Окно Capture вызывает функцию обратного вызова вавестреам перед записью каждого звукового буфера на диск.
