# <a name="settuner-command"></a>Команда сеттунер

Команда сеттунер изменяет текущий тюнер или параметр канала текущего тюнера. Устройства ВИДЕОМАГНИТОФОНА распознают эту команду.

Чтобы отправить эту команду, вызовите функцию [**mciSendString**](/previous-versions//dd757161(v=vs.85)) с заданным параметром *лпсзкомманд* , как показано ниже.

``` syntax
_stprintf_s(
  lpszCommand, 
  TEXT("settuner %s %s %s"), 
  lpszDeviceID, 
  lpszTuner, 
  lpszFlags
); 
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="lpszDeviceID"></span><span id="lpszdeviceid"></span><span id="LPSZDEVICEID"></span>*лпсздевицеид*
</dt> <dd>

Идентификатор устройства MCI. Этот идентификатор или псевдоним назначается при открытии устройства.

</dd> <dt>

<span id="lpszTuner"></span><span id="lpsztuner"></span><span id="LPSZTUNER"></span>*лпсзтунер*
</dt> <dd>

Один из следующих флагов.



| Значение                            | Значение                                                                                                                                                                                                                                                                                                     |
|----------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *канал* канала                | Задает для тюнера значение *канал*. Возможно, вы не сможете изменить канал во время записи, в зависимости от ВИДЕОМАГНИТОФОНА. Канал, превышающий максимальный, задает для тюнера максимальный канал.                                                                                                                    |
| Поиск канала с переходом по каналу | Выполняет поиск следующего канала с допустимым сигналом. "Поиск" — увеличивает номер канала в поиске. "Поиск вниз" уменьшает номер канала в поиске. При превышении максимального номера канала тюнер переносится на канал 1. Аналогичным образом, при поиске, тюнер переносится к максимальному каналу. |
| канал исключается канала           | Увеличивает или уменьшает канал тюнера. Возможно, вы не сможете изменить канал во время записи, в зависимости от ВИДЕОМАГНИТОФОНА. При превышении максимального номера канала тюнер переносится на канал 1. Аналогичным образом, при поиске, тюнер переносится к максимальному каналу.                              |
| номер                   | Указывает тюнер, затронутый командой сеттунер. Если аргумент *Number* не задан, то предполагается значение по умолчанию, равное 1.                                                                                                                                                                                    |



 

</dd> <dt>

<span id="lpszFlags"></span><span id="lpszflags"></span><span id="LPSZFLAGS"></span>*лпсзфлагс*
</dt> <dd>

Может принимать значение "Wait", "notify", "Test" или их сочетание. Дополнительные сведения об этих флагах см. [в разделе Флаги ожидания, уведомления и проверки](the-wait-notify-and-test-flags.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает нуль в случае успеха или ошибку в противном случае.
