# <a name="about-video-capture"></a>Сведения о записи видео

Авикап поддерживает потоковую передачу видеозаписи и захват одного кадра в режиме реального времени. Кроме того, Авикап предоставляет контроль над видеоисточниками, которые являются устройствами управления мультимедийными интерфейсами (MCI), чтобы пользователь мог контролировать (с помощью приложения) начальные и останавливаемые позиции источника видео, а также дополнить операцию записи, включив запись кадра шагов.

Окна, создаваемые с помощью класса окна Авикап, могут выполнять следующие задачи:

-   Запись аудио-и видеопотоков в файл AVI с чередованием Аудо.
-   динамически Подключение и отключать видео и звуковые устройства ввода.
-   Просмотр входящего видеосигнала в реальном времени с помощью наложения или предварительных версий.
-   Укажите файл, который будет использоваться при записи и копировании содержимого файла записи в другой файл.
-   Задайте скорость записи.
-   Отображение диалоговых окон, управляющих источником и форматом видео.
-   Создание, сохранение и загрузка палитр.
-   Копирование изображений и палитр в буфер обмена.
-   Запись и сохранение одного образа как аппаратно-независимого точечного рисунка (DIB).
