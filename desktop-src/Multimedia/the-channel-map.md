# <a name="the-channel-map"></a>Схема канала

Схема канала влияет на все сообщения канала MIDI. Сообщения каналов MIDI включают в себя заметку, заметку, полифоник-Key-афтертауч, управление изменениями, программу-изменение, Channel-афтертауч и шаг с изгибом — изменения. Средство сопоставления MIDI использует одну карту каналов с записью для каждого 16 каналов MIDI. Каждая запись схемы канала указывает следующее:

-   Канал назначения для сообщения MIDI
-   Целевое устройство вывода для сообщения MIDI
-   Необязательная схема исправлений с указанием других возможных изменений сообщения MIDI

Для целевого канала задан один из 16 каналов MIDI. Сообщения MIDI изменяются для отражения каждого нового назначения канала. Например, если в записи конечного канала для MIDI Channel 4 задано значение 6, все сообщения MIDI, отправленные на канал 4, будут сопоставлены с каналом 6, как показано на следующем рисунке.

![сопоставленный образ MIDI](images/mmap-a05.gif)

В этом примере 0x93 байт состояния MIDI сопоставляется с 0x95. Низкий порядок байтов состояния MIDI указывает номер канала. Для исходных каналов задано значение активно или неактивно. Сообщения, отправленные в неактивные исходные каналы, игнорируются, поэтому неактивный канал в силе или отключен.

Целевое устройство вывода настроено на одно из доступных выходных устройств MIDI. Устройство вывода MIDI может быть внутренним синтезатором или физическим портом вывода MIDI.

Системные сообщения MIDI — это сообщения MIDI (с состоянием bytes) от 0xF0 до 0xFF. Нет канала, связанного с системными сообщениями MIDI, поэтому они не могут быть сопоставлены. Системные сообщения MIDI отправляются на все устройства вывода MIDI, перечисленные в карте каналов.
