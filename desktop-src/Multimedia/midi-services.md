# <a name="midi-services"></a>Службы MIDI

Большинство приложений смогут использовать секвенсор MIDI MCI или буферы потока (и функцию [**мидистреамаут**](/windows/win32/api/mmeapi/nf-mmeapi-midistreamout) ) для реализации всех необходимых функций MIDI. Серьезные разработчики MIDI, которые создают средства создания и виртуализации MIDI, могут использовать сочетание возможностей потока и службы MIDI или использовать только службы MIDI. В следующих разделах приводятся общие сведения об использовании служб MIDI.

-   [Запрос устройств MIDI](querying-midi-devices.md)
-   [Открытие и закрытие драйверов устройств](opening-and-closing-device-drivers.md)
-   [Выделение и подготовка блоков данных MIDI](allocating-and-preparing-midi-data-blocks.md)
-   [Управление блоками данных MIDI](managing-midi-data-blocks.md)
-   [Запрос форматов времени](requesting-time-formats.md)
-   [Обработка ошибок с помощью функций MIDI](handling-errors-with-midi-functions.md)
