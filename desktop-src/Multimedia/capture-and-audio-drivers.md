# <a name="capture-and-audio-drivers"></a>Драйверы записи и аудио

Драйвер записи и базовое оборудование могут определять несколько аспектов видеозаписи, включая допустимые источники видео, параметры вывода, форматы и параметры сжатия. Звуковой драйвер определяет аудио формат и необязательный параметр сжатия, используемый для захваченных звуковых данных.
