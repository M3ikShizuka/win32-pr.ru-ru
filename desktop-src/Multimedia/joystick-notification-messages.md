# <a name="joystick-notification-messages"></a>Сообщения уведомления джойстика

Джойстик сообщений уведомляет приложение о том, что джойстик изменил расположение или что одна из ее кнопок изменила состояние. Сообщения, начинающиеся с MM \_ JOY1, отправляются в функцию, если приложение запрашивает входные данные из джойстика с помощью идентификатора JOYSTICKID1, а mm \_ JOY2 отправляет сообщения, если приложение запрашивает входные данные из джойстика с помощью идентификатора JOYSTICKID2.

Сообщения в следующей таблице указывают состояние кнопок джойстика:



| Сообщение                                         | Описание                                                     |
|-------------------------------------------------|-----------------------------------------------------------------|
| [**\_JOY1BUTTONDOWN мм**](mm-joy1buttondown.md) | Была нажата кнопка на джойстике JOYSTICKID1.              |
| [**\_JOY1BUTTONUP мм**](mm-joy1buttonup.md)     | Была снята кнопка на джойстике JOYSTICKID1.             |
| [**\_JOY1MOVE мм**](mm-joy1move.md)             | Джойстик JOYSTICKID1 измененное расположение в направлении x или y. |
| [**\_JOY1ZMOVE мм**](mm-joy1zmove.md)           | Джойстик JOYSTICKID1 измененное расположение в z-направлении.       |
| [**\_JOY2BUTTONDOWN мм**](mm-joy2buttondown.md) | Была нажата кнопка на джойстике JOYSTICKID2.              |
| [**\_JOY2BUTTONUP мм**](mm-joy2buttonup.md)     | Была снята кнопка на джойстике JOYSTICKID2.             |
| [**\_JOY2MOVE мм**](mm-joy2move.md)             | Расположение джойстика JOYSTICKID2 изменено в направлении x или y  |
| [**\_JOY2ZMOVE мм**](mm-joy2zmove.md)           | Джойстик JOYSTICKID2 измененное расположение в z-направлении.       |



 

Все сообщения сообщают о несуществующих кнопках как освобожденных.
