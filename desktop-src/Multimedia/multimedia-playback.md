# <a name="multimedia-playback"></a>Воспроизведение мультимедиа

Функция [**мЦивндкреате**](/windows/desktop/api/Vfw/nf-vfw-mciwndcreatea) предоставляет средства для управления окном мЦивнд и связанным с ним устройством. В общем случае эта функция регистрирует класс окна МЦивнд и создает окно МЦивнд для использования служб MCI. В этом разделе описывается использование **мЦивндкреате** для выполнения следующих задач.

-   [Добавление User-Controlled воспроизведения](adding-user-controlled-playback.md)
-   [Автоматизация воспроизведения](automating-playback.md)
-   [Использование стилей окна для изменения окна МЦивнд](using-window-styles-to-change-the-mciwnd-window.md)
-   [Разрешение пользователю указывать устройства и файлы](allowing-the-user-to-specify-devices-and-files.md)
