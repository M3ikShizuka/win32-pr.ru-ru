# <a name="specifying-time-formats"></a>Указание форматов времени

Типы данных мультимедиа обычно могут использовать время для обнаружения значимых позиций в содержимом. Распространенные форматы времени: миллисекунды, дорожки и кадры; также существуют и другие менее распространенные форматы времени, такие как SMPTE (общество изображений и телевизионных специалистов) 24. Время — это формат и справочная система для звуковых данных аудио, MIDI и CD. Видео поддерживает время, даже если оно записано как последовательность кадров (поток), которая обычно воспроизводится с определенной скоростью. Для обозначения формата времени доступны несколько макросов.

Вы можете получить текущий формат времени для файла или устройства с помощью макроса [**мЦивнджеттимеформат**](/windows/desktop/api/Vfw/nf-vfw-mciwndgettimeformat) . Можно изменить текущий формат времени на любой другой формат времени, поддерживаемый устройством, с помощью макроса [**мЦивндсеттимеформат**](/windows/desktop/api/Vfw/nf-vfw-mciwndsettimeformat) . Или можно задать формат времени равным миллисекундам или кадрам с помощью макросов [**мЦивндусетиме**](/windows/desktop/api/Vfw/nf-vfw-mciwndusetime) или [**мЦивндусефрамес**](/windows/desktop/api/Vfw/nf-vfw-mciwnduseframes) .

> [!Note]  
> Непостоянное форматирование, например дорожные и SMPTE, может вызвать нестабильное поведение панели инструментов. Для этих форматов времени может потребоваться отключить панель инструментов, указав \_ стиль окна мЦивндф ноплайбар при создании окна мЦивнд.
