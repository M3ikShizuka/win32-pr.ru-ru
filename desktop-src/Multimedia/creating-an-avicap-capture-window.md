# <a name="creating-an-avicap-capture-window"></a>Создание окна Авикап Capture

Окно Capture класса Авикап Window можно создать с помощью функции [**капкреатекаптуревиндов**](/windows/desktop/api/Vfw/nf-vfw-capcreatecapturewindowa) . Эта функция возвращает маркер окна, который определяет окно записи и используется приложением для отправки последующих сообщений в окно.

Можно создать одно или несколько окон захвата в приложении и подключить каждое окно захвата к другому устройству записи.
