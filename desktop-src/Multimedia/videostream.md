# <a name="videostream"></a>Videostream

Приложения могут использовать функции обратного вызова videostream во время потоковой записи для обработки захваченного кадра видео. Окно Capture вызывает функцию обратного вызова videostream перед записью каждого захваченного кадра на диск.
