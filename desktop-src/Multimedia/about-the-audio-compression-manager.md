# <a name="about-the-audio-compression-manager"></a>О диспетчере аудиосжатия

Диспетчер аудиосжатия добавляет поддержку на уровне системы для следующих служб:

-   Прозрачное сжатие и распаковка звука во время выполнения
-   Волна-выбор формата звуковых данных
-   Волна-выбор фильтра данных звука
-   Волна-преобразование формата данных аудио
-   Звуковая волна — фильтрация данных
