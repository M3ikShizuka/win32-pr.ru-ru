# <a name="single-image-compression"></a>Сжатие Single-Image

Для сжатия одного изображения можно использовать функцию [**иЦимажекомпресс**](/windows/desktop/api/Vfw/nf-vfw-icimagecompress) . Эта функция возвращает маркер сжатого аппаратно-независимого точечного рисунка (DIB). Сжатый DIB упакован с использованием \_ формата CF DIB.
