# <a name="multimedia-audio"></a>Мультимедийные аудио

Windows предоставляет функции, позволяющие приложению добавлять службы аудио. Функции аудио мультимедиа были введены в 16-разрядной Windows. Многие их функции были заменены компонентами DirectX. Дополнительные сведения см. в документации DirectX.

Следующие разделы обсуждаются в документации мультимедиа аудио.

-   [Диспетчер аудиосжатия](audio-compression-manager.md)
-   [Аудио Миксерс](audio-mixers.md)
-   [Цифровой интерфейс музыкальных инструментов (MIDI)](musical-instrument-digital-interface--midi.md)
-   [Звуковая волна](waveform-audio.md)

Звук также можно воспроизвести с помощью интерфейса управления носителями. Дополнительные сведения см. в разделе [MCI](mci.md).
