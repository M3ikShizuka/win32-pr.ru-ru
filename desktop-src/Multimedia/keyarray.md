# <a name="keyarray"></a>кэйаррай

КЭЙАРРАЙ указывает тип, используемый для определения массива ключей.


```C++
typedef WORD KEYARRAY[MIDIPATCHSIZE];
```



<dl> <dt>

**КЭЙАРРАЙ \[ мидипатчсизе\]**
</dt> <dd>

Каждый элемент массива соответствует исправлению, основанному на ключах, с каждым из 16 битов, представляющим один из 16 каналов MIDI. Биты задаются для каждого канала, который использует это конкретное исправление. Например, если исправление для раздела номер 60 используется физическими каналами MIDI 9 и 15, то элементу 60 массива должно быть присвоено значение 0x8200.

</dd> </dl>
