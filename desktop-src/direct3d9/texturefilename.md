Позволяет указать имя файла текстуры, применяемой к сетке или грани. Этот шаблон должен отображаться в объекте Material.

``` syntax
template TextureFilename 
{ 
    < A42790E1-7810-11cf-8F52-0040333594A3 > 
    string filename; 
} 
```
