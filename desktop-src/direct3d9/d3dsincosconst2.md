# <a name="d3dsincosconst2"></a>D3DSINCOSCONST2

Создает вторую константу (в регистре register src3) для инструкции шейдера пикселей [синкос-PS](../direct3dhlsl/sincos---ps.md) .

``` syntax
#define D3DSINCOSCONST2 -0.020833334f, -0.12500000f, 1.0f, 0.50000000f
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает вторую константу, необходимую [синкос-PS](../direct3dhlsl/sincos---ps.md).
