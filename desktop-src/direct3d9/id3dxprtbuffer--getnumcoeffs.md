# <a name="id3dxprtbuffergetnumcoeffs-method"></a>Метод ID3DXPRTBuffer:: Жетнумкоеффс

Возвращает число скаляров на цветовой канал, используемый в памяти для хранения выборок.

## <a name="syntax"></a>Синтаксис


```C++
UINT GetNumCoeffs();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **uint**](../winprog/windows-data-types.md)**

Возвращает число скаляров на цветовой канал, используемое в памяти для хранения выборок.
