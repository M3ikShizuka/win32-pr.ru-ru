# <a name="fire-flares-and-explosions-direct3d-9"></a>Пожар, Фларес и взрывные развертывания (Direct3D 9)

Вы можете использовать Microsoft Direct3D для имитации естественного явленийа, включающего выпуски энергии. Например, приложение может создать впечатление пожара, применяя пламени-подобные текстуры к набору объявлений. Это особенно эффективно, если приложение использует последовательность пожарных текстур для анимации огонь на каждом рекламе в пожаре. Изменение скорости воспроизведения анимации с рекламного объявления на рекламу увеличивает вид реальной огонь. Сембланце запутанной 3D огонь можно достичь, разуровневый объявления и текстуры на объявлениях.

Фларес и Flash можно имитировать, применяя последовательное освещение для всех примитивов сцены. Несмотря на то что это высокопроизводительный метод, он позволяет приложению имитировать локализованные блики или Flash. Это значит, что часть сцены, в которой происходят блики или Flash, может быть ярче.

Другой способ — разместить объявление перед сценой, чтобы покрыть всю область рендеринга. Приложение применяет поочередно белые текстуры к рекламному стенду и уменьшает прозрачность с течением времени. Все сцены постепенно поступают в белый цвет по мере прохождения времени. Это метод создания блика с низкой нагрузкой. Однако при использовании этого метода может быть трудно создать впечатление светлой флэш-памяти от источника освещения с одной точкой.

Взрывы могут отображаться в виде трехмерной сцены, аналогичной той, которая используется для пожара, вспышки и фларес. Например, ваше приложение может использовать объявление для показа волновой волны и увеличения плумеи состояния, когда происходит развертывание. В то же время приложение может использовать набор объявлений для имитации огонь. Кроме того, он может разместить один рекламный стенд перед сценой, чтобы добавить Блик освещения к всей сцене.

Energy беамс можно имитировать с помощью объявлений. Приложение также может отображать их с помощью примитивов, определенных в виде линейных списков или линейных полос. Дополнительные сведения см. в [статье строки](line-lists.md) и [полосы строк](line-strips.md).

Приложение может создавать принудительные поля с помощью объявлений или примитивов, определенных как списки треугольников. Чтобы создать принудительное поле из списков треугольников, определите набор необъединяемых треугольников в списке треугольников по поверх региона, охваченного полем Force. Зазоры между треугольниками позволяют пользователю видеть сцену позади треугольников, как это может показаться при просмотре поля Force. Примените текстуру к списку треугольников, которая дает треугольникам вид свечения с энергиями. Дополнительные сведения см. в разделе [списки треугольников](triangle-lists.md).
