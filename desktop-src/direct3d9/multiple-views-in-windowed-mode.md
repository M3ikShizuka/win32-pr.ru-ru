# <a name="multiple-views-in-windowed-mode-direct3d-9"></a>Несколько представлений в оконном режиме (Direct3D 9)

В дополнение к цепочке подкачки, которая владеет и управляется через объект Direct3DDevice, приложение может использовать метод [**IDirect3DDevice9:: креатеаддитионалсвапчаин**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-createadditionalswapchain) для создания дополнительных цепочек подкачки для предоставления нескольких представлений из одного устройства.

Как правило, приложение создает одну цепочку подкачки для каждого представления и связывает каждую цепочку подкачки с определенным представлением. Приложение визуализирует изображения в задних буферах каждой цепочки буферов, а затем использует метод [**IDirect3DDevice9::P**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-present) повторной передачи, чтобы предоставить их по отдельности. Обратите внимание, что только одна цепочка подкачки одновременно может быть полноэкранной на каждом адаптере.
