# <a name="d3dxvec3transformcoordarray-function-d3dx9mathh"></a>Функция D3DXVec3TransformCoordArray (D3dx9math. h)

Преобразует массив (x, y, z, 1) по заданной матрице и проецирует результат обратно в w = 1.

## <a name="syntax"></a>Синтаксис


```C++
D3DXVECTOR3* D3DXVec3TransformCoordArray(
  _Inout_       D3DXVECTOR3 *pOut,
  _In_          UINT        OutStride,
  _In_    const D3DXVECTOR3 *pV,
  _In_          UINT        VStride,
  _In_    const D3DXMATRIX  *pM,
  _In_          UINT        n
);
```

## <a name="remarks"></a>Remarks

Эта функция преобразует массив *ПС (* x, y, z, 1) на матрицу *PM*, проецирование результата обратно в w = 1.

Возвращаемое значение для этой функции совпадает со значением, возвращаемым в параметре *тоска* . Таким образом, функция [**D3DXVec3TransformCoord**](d3dxvec3transformcoord.md) может использоваться в качестве параметра для другой функции.
