# <a name="idirectxfiledatareference-interface"></a>Интерфейс Идиректксфиледатареференце

Приложения используют методы интерфейса Идиректксфиледатареференце для поддержки объектов ссылок на данные. Объект ссылки на данные ссылается на объект данных, определенный ранее в файле. Это позволяет использовать один и тот же объект несколько раз без повторения в файле. Не рекомендуется.

## <a name="members"></a>Элементы

Интерфейс **идиректксфиледатареференце** наследует от [**идиректксфилеобжект**](idirectxfileobject.md). **Идиректксфиледатареференце** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **идиректксфиледатареференце** содержит следующие методы.



| Метод                                                | Описание                                      |
|:------------------------------------------------------|:-------------------------------------------------|
| [**Разрешить**](idirectxfiledatareference--resolve.md) | Разрешает ссылки на данные. Не рекомендуется.<br/> |



 

## <a name="remarks"></a>Remarks

После определения того, что объект является объектом ссылки на данные, используйте метод [**идиректксфиледатареференце:: Resolve**](idirectxfiledatareference--resolve.md) для получения упоминаемого ранее объекта в файле. Сведения о том, как определить объект ссылки на данные, см. в разделе интерфейс [**идиректксфиледата**](idirectxfiledata.md) .

Идентификатор GUID для интерфейса Идиректксфиледатареференце — IID \_ идиректксфиледатареференце.

Тип Лпдиректксфиледатареференце определяется как указатель на этот интерфейс.


```
typedef interface IDirectXFileDataReference *LPDIRECTXFILEDATAREFERENCE;
```
