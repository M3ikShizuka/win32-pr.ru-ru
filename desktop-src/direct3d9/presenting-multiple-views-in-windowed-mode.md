# <a name="presenting-multiple-views-in-windowed-mode-direct3d-9"></a>Представление нескольких представлений в оконном режиме (Direct3D 9)

В дополнение к цепочке подкачки, которая владеет и управляется через интерфейс [**IDirect3DDevice9**](/windows/desktop/api) , приложение может создавать дополнительные цепочки подкачки для представления нескольких представлений с одного устройства. Приложение обычно создает одну цепочку подкачки на представление с помощью метода [**IDirect3DDevice9:: креатеаддитионалсвапчаин**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-createadditionalswapchain) и связывает каждую цепочку подкачки с определенным окном. Приложение визуализирует изображения в задние буферы каждой цепочки буферов, а затем представляет их по отдельности.

Только одна цепочка подкачки одновременно может быть полноэкранной на каждом адаптере.
