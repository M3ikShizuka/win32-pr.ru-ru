# <a name="d3dxf"></a>D3DXF

Параметры формата файла X, загрузки и сохранения.

## <a name="file-formats"></a>Форматы файлов

В следующей таблице указаны типы файловых данных.



| \#Определяет формат файла     | Значение | Описание                                                                                    |
|-------------------------------|-------|------------------------------------------------------------------------------------------------|
| \_ \_ Двоичный FILEFORMAT D3DXF     | 0     | Двоичный файл в старом формате. См. раздел [Справочник по X (Legacy)](dx9-graphics-reference-x-file.md). |
| D3DXF \_ FILEFORMAT \_ текст       | 1     | Текстовый файл.                                                                                     |
| \_Сжатый D3DXF FILEFORMAT \_ | 2     | Сжатый файл. С этим флагом необходимо также указать двоичный формат или текстовый формат.   |



 

## <a name="file-load-options"></a>Параметры загрузки файлов

В следующей таблице указаны параметры загрузки файлов с помощью x:



| \#определенно                      | Значение | Описание                |
|-------------------------------|-------|----------------------------|
| D3DXF \_ филелоад \_ фромфиле     | 0     | Загрузка данных из файла.     |
| D3DXF \_ филелоад \_ фромвфиле    | 1     | Загрузка данных из файла.     |
| D3DXF \_ филелоад \_ фромресаурце | 2     | Загрузка данных из ресурса. |
| D3DXF \_ филелоад \_ фроммемори   | 3     | Загрузка данных из памяти.     |



 

## <a name="file-save-options"></a>Параметры сохранения файла

В следующей таблице указаны параметры сохранения и загрузки файлов с расширением x.



| \#определенно                 | Значение | Описание                    |
|--------------------------|-------|--------------------------------|
| D3DXF \_ филесаве \_ TOFILE  | 0     | Сохранить в файл.                |
| D3DXF \_ филесаве \_ товфиле | 1     | Сохранить в файл расширенных символов. |
