# <a name="id3dxlinegetpattern-method"></a>Метод ID3DXLine:: методом Pattern

Возвращает шаблон линии стиппле.

## <a name="syntax"></a>Синтаксис


```C++
DWORD GetPattern();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **DWORD**](../winprog/windows-data-types.md)**

Возвращает шаблон строки стиппле: 1 является непрозрачным, 0 является прозрачным.
