# <a name="d3dxf_fileloadmemory-structure"></a>\_Структура ФИЛЕЛОАДМЕМОРИ D3DXF

Определяет данные памяти.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct D3DXF_FILELOADMEMORY {
  LPCVOID lpMemory;
  SIZE_T  dSize;
} D3DXF_FILELOADMEMORY, *LPD3DXF_FILELOADMEMORY;
```
