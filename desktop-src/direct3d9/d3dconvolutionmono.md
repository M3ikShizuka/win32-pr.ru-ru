# <a name="d3dconvolutionmono"></a>D3DCONVOLUTIONMONO

Задает максимальные размеры фильтров свертки текстур.



| \#определенно                      | Значение | Описание                                                |
|-------------------------------|-------|------------------------------------------------------------|
| D3DCONVOLUTIONMONO \_ MAXHEIGHT | 7     | Максимальная высота фильтра свертки монохромных текстур. |
| D3DCONVOLUTIONMONO \_ MAXWIDTH  | 7     | Максимальная ширина фильтра свертки текстур монохромной текстуры.  |
