# <a name="programming-guide-for-direct3d-9"></a>Руководством по программированию для Direct3D 9

В этом руководством содержится описание графического конвейера, реализованного в Microsoft Direct3D. Это рекомендации для разработчиков, которые реализуют графические функции Direct3D в своих приложениях. В этом руководством содержатся описания архитектуры, функциональные схемы и описания стандартных блоков в конвейере, а также фрагменты кода и примеры приложений. Эти сведения делятся на следующие разделы:

-   [Начало работы](getting-started.md) — этот раздел содержит общие сведения о конвейере и учебных курсах, которые помогут вам получить простое приложение для работы с графикой в течение нескольких минут.
-   [Эффекты](effects.md) . в этом разделе описываются эффекты и файлы эффектов для создания приложений, которые могут работать на различных аппаратных платформах.
-   [Дополнительные разделы](advanced-topics.md) . Этот раздел содержит примеры различных типов специальных эффектов, которые можно реализовать. Такие темы, как сопоставление окружения и выпуклости, сглаживание, смешение вершин, tween-анимация, высокая динамический диапазон (HDR) и предварительно вычисленное радианце-перемещение (PRT), показывают, как применять к приложению специальные спецэффекты.
-   [Советы программирования](programming-tips.md) . в этом разделе содержатся сведения, помогающие эффективно разрабатывать графические приложения Direct3D.

Дополнительные сведения о конкретных методах API см. на страницах [справочника](dx9-graphics-reference.md) .
