# <a name="antialiasing-direct3d-9"></a>Сглаживание (Direct3D 9)

Сглаживание — это метод, который можно использовать для уменьшения внешнего вида пикселов лестницы при рисовании любой линии, не полностью горизонтальной или вертикальной. В трехмерных сценах этот артефакт наиболее заметен в границах между многоугольниками различных цветов. Сглаживание позволяет эффективно смешивать пикселы на этих границах, чтобы получить более естественный вид сцены.
