# <a name="environment-mapping-direct3d-9"></a>Сопоставление окружения (Direct3D 9)

Сопоставление окружения — это методика, которая имитирует высокоотражающие поверхности без использования трассировки лучей. На практике сопоставление среды применяет специальную текстурную карту, содержащую изображение сцены, окружающей объект с самим объектом. Результат приближен к появлению отраженной поверхности, достаточно близко к тому, чтобы обмануть глаза, без каких-либо сложных вычислений, участвующих в трассировке лучей.

На следующем рисунке показан тип сопоставления среды, именуемого сопоставлением сферического окружения. Дополнительные сведения см. в статье [сопоставление сферического окружения](spherical-environment-mapping.md).

![Иллюстрация чайник с примененной текстурой, отражающей окружающую окружение](images/spheremapped-teapot.png)

Чайник на этом изображении отражает свое окружение. на самом деле, к объекту применяется текстура. Поскольку сопоставление окружения использует текстуру вместе с специально вычисленными координатами текстуры, его можно выполнить в режиме реального времени.

В этом разделе содержатся сведения о выполнении двух распространенных типов сопоставления среды с Direct3D. Существует множество типов сопоставления среды, используемых в графической отрасли, но в следующих разделах рассматриваются две наиболее распространенные формы: сопоставление кубических сред и сферическое сопоставление среды.

-   [Сопоставление кубических сред](cubic-environment-mapping.md)
-   [Сопоставление сферического окружения](spherical-environment-mapping.md)
