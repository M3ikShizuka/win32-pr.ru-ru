# <a name="cloning-and-sharing-direct3d-9"></a>Клонирование и совместное использование (Direct3D 9)

## <a name="cloning-parameters"></a>Клонирование параметров

Клонирование имеет следующие ограничения.

-   Клоны наследуют пул исходного воздействия. См. раздел Параметры совместного использования.
-   Клоны наследуют методы исходного воздействия, передают, параметры и заметки (включая все заметки, добавленные с помощью [**ID3DXEffect**](id3dxeffect.md)).
-   Клоны наследуют заметки, динамически добавляемые к исходному результату.
-   Клонирование на новое устройство завершится ошибкой, если пул исходных эффектов не **равен null** и исходный результат содержал общий зависимый от устройства параметр (например, текстуру или шейдер).

## <a name="sharing-parameters"></a>Совместное использование параметров

Пул — это буфер, который предоставляет общий доступ к параметрам различных эффектов. Чтобы добавить параметры в пул, укажите общее использование при создании этого результата.

У пула есть следующие ограничения.

-   Параметр добавляется в пул при первом добавлении в пул результата, содержащего этот параметр (Shared).
-   Пул получает начальные значения из первого общего параметра; Общие параметры последующего получения значений из пула.
-   Параметр удаляется из пула, когда освобождаются все ссылки на общий параметр.
-   Все эффекты в пуле, которые содержат одинаковый (общий) зависимый от устройства параметр, должны иметь одно и то же устройство.

**Null** можно использовать для указания отсутствия пула, в этом случае параметры не являются общими. Это почти эквивалентно указанию уникального пула только для этого результата. Единственное отличие состоит в том, что при клонировании результата клон не будет совместно использовать его общие параметры с исходным.
