# <a name="effectparamdword"></a>еффектпарамдворд

Определяет действие DWORD. Этот шаблон заменил шаблон [**еффектдворд**](effectdword.md) .

``` syntax
template EffectParamDWord
{
    < E13963BC-AE51-4c5d-B00F-CFA3A9D97CE5 >
    STRING ParamName;
    DWORD Value;
} 
```

Где:

-   ParamName — имя параметра.
-   Value — значение параметра.
