# <a name="fog-parameters-direct3d-9"></a>Параметры тумана (Direct3D 9)

Управление параметрами тумана осуществляется с помощью состояний отрисовки устройства. Типы пиксельных и вершинных туманов поддерживают все формулы тумана, появившиеся в [формулах тумана (Direct3D 9)](fog-formulas.md). Перечисляемый тип [**D3DFOGMODE**](./d3dfogmode.md) определяет константы, которые можно использовать для определения формулы тумана, которую должен использовать Microsoft Direct3D. \_Состояние отображения D3DRS фогтаблемоде управляет режимом тумана, который Direct3D использует для пиксельных туманов, а \_ состояние отображения D3DRS фогвертексмоде — режим для вершинного тумана.

При использовании линейной формулы тумана начальное и конечное расстояния задаются с помощью D3DRS \_ фогстарт и D3DRS \_ фоженд Rendering States. Сведения о том, как система интерпретирует эти значения, зависит от типа тумана, используемого приложением-пиксельным или вершинным туманом, и при использовании пиксельного тумана, если используется глубина, основанная на z или w. В следующей таблице перечислены типы туманов и их начальные и конечные единицы.



| Тип тумана  | Единицы начала и окончания тумана      |
|-----------|--------------------------|
| Пиксель (Z) | Пространство устройства \[ 0,0, 1,0\] |
| Пиксель (W) | Место на камере             |
| Вершина    | Место на камере             |



 

\_Состояние отрисовки D3DRS фогденсити управляет плотностью тумана, применяемой при включенной формуле экспоненциального тумана. Плотность тумана по сути является весовым фактором, от 0,0 до 1,0 (включительно), которое масштабирует значение расстояния в экспоненте.

Цвет, используемый системой для смешения тумана, управляется через \_ состояние рендеринга устройства D3DRS фогколор. Дополнительные сведения см. в разделе [Цвет тумана (Direct3D 9)](fog-color.md) и [смешение тумана (Direct3D 9)](fog-blending.md).
