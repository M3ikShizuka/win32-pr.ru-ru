# <a name="indexedcolor"></a>индекседколор

Состоит из параметра индекса и цвета RGBA и используется для определения цветов вершин сетки. Индекс определяет вершину, к которой применяется цвет.

``` syntax
template IndexedColor
{
    < 1630B820-7842-11cf-8F52-0040333594A3 >
    DWORD index;
    ColorRGBA indexColor;
} 
```

Где:

-   index — DWORD. См. также описание поля выше.
-   Индексколор — шаблон Колорргба. См. [**колорргба**](colorrgba.md).
