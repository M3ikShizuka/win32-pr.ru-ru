# <a name="id3dxkeyframedanimationsetgetnumrotationkeys-method"></a>Метод ID3DXKeyframedAnimationSet:: Жетнумротатионкэйс

Возвращает число ключей вращения в указанной анимации по опорным кадрам.

## <a name="syntax"></a>Синтаксис


```C++
UINT GetNumRotationKeys(
  [in] UINT Animation
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Анимация* \[ окне\]
</dt> <dd>

Тип: **[ **uint**](../winprog/windows-data-types.md)**

Индекс анимации.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **uint**](../winprog/windows-data-types.md)**

Количество ключей вращения в указанной анимации опорных кадров.
