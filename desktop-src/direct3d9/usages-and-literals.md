# <a name="usages-and-literals-direct3d-9"></a>Использование и литералы (Direct3D 9)

Использование аналогично области действия параметра, так как оно определяет область, в которой параметр является допустимым.



| Значение  | Описание                                                                                                                                                                                                                                                                         |
|--------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| const  | Параметр будет постоянным в области всех функций. (Обратите внимание, что такие параметры по-прежнему можно записать с помощью [**ID3DXEffect**](id3dxeffect.md) или [**ID3DXEffectCompiler**](id3dxeffectcompiler.md), так как это происходит вне области действия всех функций.) |
| общие | Параметр будет совместно использоваться в пуле эффектов.                                                                                                                                                                                                                                    |
| static | Параметр будет невидимым для приложения, то есть к нему нельзя получить доступ из [**ID3DXEffect**](id3dxeffect.md) или [**ID3DXEffectCompiler**](id3dxeffectcompiler.md).                                                                                                  |



 

Пометка параметра как литерала означает, что его значение никогда не изменится. Это позволяет компилятору влияния выполнить дополнительные оптимизации.

Только необщие параметры верхнего уровня могут быть помечены как литералы. Параметры могут быть помечены только как литерал с помощью [**ID3DXEffectCompiler**](id3dxeffectcompiler.md). Литеральные значения не могут быть заданы с помощью [**ID3DXEffect**](id3dxeffect.md).
