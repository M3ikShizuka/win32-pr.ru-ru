# <a name="windowed-vs-full-screen-mode-direct3d-9"></a>Режим с окнами и Full-Screenом (Direct3D 9)

Приложения Direct3D могут работать в одном из двух режимов: в полноэкранном режиме или в окне.

Полноэкранный режим означает, что окно, в котором выполняется приложение, охватывает весь рабочий стол, скрывая все работающие приложения (включая среду разработки). Игры обычно по умолчанию запускаются в полноэкранном режиме и скрывают все другие работающие приложения, чтобы дать пользователю возможность полностью погрузиться в игру. Приложение, работающее в режиме с оконным режимом, использует рабочий стол совместно со всеми работающими приложениями. Различия в коде между полноэкранным и оконным режимом минимальны.

Поскольку приложение, запущенное в полноэкранном экране, занимает весь экран, для отладки этого приложения требуется отдельный монитор или удаленный отладчик. Используйте [Панель управления DirectX](https://msdn.microsoft.com/library/Ee416791(v=VS.85).aspx) для включения отладки с несколькими мониторами. Одним из преимуществ приложения в оконном режиме является то, что код можно выполнять по шагам в отладчике без дополнительных мониторов или удаленного отладчика.
