# <a name="working-with-device-windows-direct3d-9"></a>Работа с окнами устройств (Direct3D 9)

В этом разделе перечислены проблемы, которые могут возникнуть при работе с окнами устройств в приложениях Direct3D.

-   Direct3D подключается только к окнам фокуса, а не к окну устройства с помощью функции обработки сообщений Direct3D и обрабатывает только сообщения окна фокуса. Таким образом, окно фокуса должно быть родительским для любого окна устройства.
