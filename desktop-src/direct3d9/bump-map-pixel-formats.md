# <a name="bump-map-pixel-formats-direct3d-9"></a>Форматы пикселей выпуклого распределения (Direct3D 9)

Схема выпуклости — это объект [**IDirect3DTexture9**](/windows/desktop/api) , который использует специализированный формат пикселей. Вместо того чтобы хранить компоненты красного, зеленого и синего цветов, каждый пиксель на карте выпуклости сохраняет значения Дельта для вас и v (D<sub>U</sub> <sub>и D)</sub>и иногда является компонентом светимости L. Эти значения применяются системой, как описано в разделе [формулы сопоставления выпуклости (Direct3D 9)](bump-mapping-formulas.md) .

Можно указать формат пикселей гиперкарты, задав один из следующих форматов: D3DFMT \_ CxV8U8, D3DFMT \_ V8U8, D3DFMT \_ L6V5U5, D3DFMT \_ X8L8V8U8, D3DFMT \_ Q8W8V8U8 или D3DFMT \_ V16U16. Описание см. в разделе [D3DFORMAT](d3dformat.md).

Компоненты точки<sub>d и</sub> d<sub>V</sub> имеют значения со знаком в диапазоне от-1,0 до + 1,0. Компонент светимости, при использовании, является целочисленным значением без знака в диапазоне от 0 до 255.

> [!Note]  
> Прежде чем выбрать формат пикселей гиперкарты, проверьте, поддерживается ли определенный формат. Дополнительные сведения см. [в разделе Использование сопоставления выпуклости (Direct3D 9)](using-bump-mapping.md).
