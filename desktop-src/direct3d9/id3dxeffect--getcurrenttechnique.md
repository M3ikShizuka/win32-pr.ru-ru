# <a name="id3dxeffectgetcurrenttechnique-method"></a>Метод ID3DXEffect:: Жеткурренттечникуе

Возвращает текущий метод.

## <a name="syntax"></a>Синтаксис


```C++
D3DXHANDLE GetCurrentTechnique();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[D3DXHANDLE](dx9-graphics-reference-effects-constants.md)**

Уникальный идентификатор для текущего метода. См. раздел [Handles (Direct3D 9)](handles.md).
