# <a name="d3dx_parameter"></a>D3DX, \_ параметр

Эти флаги предоставляют дополнительные сведения о параметрах эффектов.

Константы параметров эффектов используются в [**D3DXPARAMETER \_ DESC**](d3dxparameter-desc.md).



| Константа                                                                                                                                                                                           | Описание                                                                                                                                                                                                                                                                                                                             |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="D3DX_PARAMETER_ANNOTATION"></span><span id="d3dx_parameter_annotation"></span><dl> <dt>**\_Аннотация параметра \_ D3DX**</dt> </dl> | Этот параметр помечен как аннотация. См. раздел [Добавление сведений, чтобы параметры вступили в действие с заметками](using-an-effect.md).<br/>                                                                                                                                                                                                 |
| <span id="D3DX_PARAMETER_LITERAL"></span><span id="d3dx_parameter_literal"></span><dl> <dt>**\_Литерал параметра \_ D3DX**</dt> </dl>          | Этот параметр помечен как литеральное значение. Параметры литерала не могут быть изменены после компиляции, что позволяет компилятору оптимизировать их использование. Общие параметры не могут быть помечены как литералы. См. раздел [использование и литералы (Direct3D 9)](usages-and-literals.md). <br/>                                                               |
| <span id="D3DX_PARAMETER_SHARED"></span><span id="d3dx_parameter_shared"></span><dl> <dt>**\_Общий параметр \_ D3DX**</dt> </dl>             | Значение параметра будет совместно использоваться всеми эффектами в том же пространстве имен. Изменение значения в одном эффекте приведет к его изменению во всех общих эффектах. См. раздел [Общие параметры](cloning-and-sharing.md). **D3DX \_ \_Общий параметр** не может быть объединен с аннотацией параметра **D3DX \_ \_** или **\_ \_ заметкой параметра D3DX**.<br/> |
