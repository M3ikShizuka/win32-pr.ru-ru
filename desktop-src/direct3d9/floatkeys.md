# <a name="floatkeys"></a>флоаткэйс

Определяет массив чисел с плавающей запятой (float) и число элементов float в этом массиве. Используется для определения наборов клавиш анимации.

``` syntax
template FloatKeys
{
    < 10DD46A9-775B-11cf-8F52-0040333594A3 >
    DWORD nValues;
    array float values[nValues];
} 
```

-   Nзначения — число элементов с плавающей запятой.
-   значения \[ nзначения \] — массив значений float.
