# <a name="antialiasing-state-direct3d-9"></a>Состояние сглаживания (Direct3D 9)

[Сглаживание (Direct3D 9)](antialiasing.md) — это метод, который делает линии и границы более гладкими на экране. По умолчанию Direct3D не выполняет сглаживание. Чтобы включить полноэкранное сглаживание, установите \_ для состояния отрисовки D3DRS Мултисамплеантиалиас **значение true**. Чтобы отключить его, присвойте параметру D3DRS \_ мултисамплеантиалиас значение **false**.
