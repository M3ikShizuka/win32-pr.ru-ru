# <a name="meshfacewraps"></a>мешфацеврапс

Используется для определения топологии текстуры каждой грани в оболочке. Значение элемента Нфацеврапвалуес должно быть равно числу граней в сети.

``` syntax
template MeshFaceWraps
{
    < ED1EC5C0-C0A8-11D0-941C-0080C80CFA7B >
    DWORD nFaceWrapValues;
    array Boolean2d faceWrapValues[nFaceWrapValues];
} 
```

Этот шаблон больше не используется.
