# <a name="working-with-earlier-drivers-direct3d-9"></a>Работа с более ранними версиями драйверов (Direct3D 9)

В этом разделе перечислены проблемы, которые могут возникнуть при работе с Direct3D 9 на драйверах, написанных для версий Direct3D, предшествующих Direct3D 9.

-   При работе с устройством HAL T&L интенсивность тумана вычислена, но операция абсолютного значения не применяется к этому значению. Вместо этого значение остается отрицательным, если это то, что является вычисленным. Лучший способ избежать этой ситуации — настроить преобразования соответствующим образом. Менее предпочтительный метод заключается в том, чтобы значения, заданные при запуске тумана и тумана, были отрицательными.

Чтобы проверить драйвер Direct3D 9, найдите ненулевое значение для D3DDEVCAPS2 \_ стреамоффсет в DevCaps2 члене [**D3DCAPS9**](/windows/desktop/api/D3D9Caps/ns-d3d9caps-d3dcaps9).
