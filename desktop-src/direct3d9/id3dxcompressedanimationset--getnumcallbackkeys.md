# <a name="id3dxcompressedanimationsetgetnumcallbackkeys-method"></a>Метод ID3DXCompressedAnimationSet:: Жетнумкаллбакккэйс

Возвращает число ключей обратного вызова в наборе анимации.

## <a name="syntax"></a>Синтаксис


```C++
UINT GetNumCallbackKeys();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **uint**](../winprog/windows-data-types.md)**

Число ключей обратного вызова в наборе анимации.
