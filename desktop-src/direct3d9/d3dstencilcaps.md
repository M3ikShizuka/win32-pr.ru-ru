# <a name="d3dstencilcaps"></a>D3DSTENCILCAPS

Флаги возможностей набора элементов драйвера.



|                          |             |                                                                                                       |
|--------------------------|-------------|-------------------------------------------------------------------------------------------------------|
| \#определенно                 | Значение       | Описание                                                                                           |
| D3DSTENCILCAPS \_     | 0x00000001L | Не обновляйте запись в буфере шаблона. Это значение по умолчанию.                             |
| D3DSTENCILCAPS \_ 0     | 0x00000002L | Задайте для записи в буфере набора элементов значение 0.                                                                    |
| D3DSTENCILCAPS \_ заменить  | 0x00000004L | Замените запись в буфере набора элементов значением ссылки.                                                |
| D3DSTENCILCAPS \_ инкрсат  | 0x00000008L | Увеличение записи буфера набора элементов с фиксацией до максимального значения.                                    |
| D3DSTENCILCAPS \_ декрсат  | 0x00000010L | Уменьшение записи в буфере набора элементов с фиксацией в ноль.                                                 |
| D3DSTENCILCAPS \_ инверсия   | 0x00000020L | Инвертировать биты в записи в буфере набора элементов.                                                          |
| D3DSTENCILCAPS \_ INCR     | 0x00000040L | Увеличение записи буфера набора элементов с переносом в ноль, если новое значение превышает максимальное значение.      |
| D3DSTENCILCAPS \_ декр     | 0x00000080L | Уменьшите запись в буфере набора элементов, заключив в нее максимальное значение, если новое значение меньше нуля. |
| D3DSTENCILCAPS \_ твосидед | 0x00000100L | Устройство поддерживает набор элементов с двумя сторонами.                                                                |



 

Элементы в буфере набора элементов — это целые значения в диапазоне от 0 до 2 ⁿ-1, где n — битовая глубина буфера шаблона.

Эти константы используются членом СтенЦилкапс объекта [**D3DCAPS9**](/windows/desktop/api/D3D9Caps/ns-d3d9caps-d3dcaps9).
