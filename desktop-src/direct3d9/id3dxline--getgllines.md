# <a name="id3dxlinegetgllines-method"></a>Метод ID3DXLine:: Жетгллинес

Возвращает режим отображения линий в стиле OpenGL.

## <a name="syntax"></a>Синтаксис


```C++
BOOL GetGLLines();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **bool** .](../winprog/windows-data-types.md)**

Возвращает **значение true** , если линии в стиле OpenGL включены, и **значение false** , если линии в стиле Direct3D включены.
