# <a name="d3dtss_tci"></a>D3DTSS \_ тЦи

Флаги возможностей координат текстуры в драйвере.



| \#определенно                                 | Значение       | Описание                                                                                                                                                                                                          |
|------------------------------------------|-------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| D3DTSS \_ тЦи \_ пропускания                    | 0x00000000 | Используйте указанные координаты текстуры, содержащиеся в формате вершины. Это значение разрешается в ноль.                                                                                                               |
| D3DTSS \_ тЦи \_ камераспаценормал           | 0x00010000L | Используйте нормальную вершину, преобразованную в пространство камеры, в качестве координат входной текстуры для преобразования текстур этого этапа.                                                                                        |
| D3DTSS \_ тЦи \_ камераспацепоситион         | 0x00020000L | Используйте расположение вершины, преобразованное в область камеры, в качестве координат входной текстуры для преобразования текстур этого этапа.                                                                                      |
| D3DTSS \_ тЦи \_ камераспацерефлектионвектор | 0x00030000L | Используйте Вектор отражения, преобразованный в пространство камеры, как входную координату текстуры для преобразования текстур этого этапа. Вектор отражения вычисляются на основе расположения входной вершины и обычного вектора. |
| D3DTSS \_ тЦи \_ сферемап                   | 0x00040000L | Используйте указанные координаты текстуры для сопоставления сфер.                                                                                                                                                            |



 

Эти константы используются D3DTSS \_ текскурдиндекс.
