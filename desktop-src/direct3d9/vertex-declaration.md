# <a name="vertex-declaration-direct3d-9"></a>Объявление вершины (Direct3D 9)

Объявление вершины определяет макет буфера вершин и программирует подсистему тесселяции. Начиная с Direct3D 9, потоки вершин выражаются в виде массива структур [**D3DVERTEXELEMENT9**](d3dvertexelement9.md) (вместо использования кодов гибкого формата ВЕРШИН (фвф)). Каждый элемент массива описывает каждый тип данных каждой вершины. Преобразование кодов> ФВФ в новый формат рассматривается в следующих разделах:

-   [Сопоставление кодов ФВФ с объявлением Direct3D 9 (Direct3D 9)](mapping-fvf-codes-to-a-directx-9-declaration.md)
-   [Фиксированные коды ФВФ функций (Direct3D 9)](fixed-function-fvf-codes.md)
-   [Сопоставление между объявлением Direct3D и кодами ФВФ (Direct3D 9)](mapping-between-a-directx-9-declaration-and-fvf-codes.md)
-   [Сопоставление между объявлением Direct3D 9 и объявлением Direct3D 8 (Direct3D 9)](mapping-between-a-directx-9-declaration-and-directx-8.md)
-   [программирование одного или нескольких Потоки (Direct3D 9)](programming-one-or-more-streams.md)
