# <a name="d3dadapter_default"></a>D3DADAPTER \_ по умолчанию

Указывает основной адаптер дисплея.



| \#определенно            | Значение | Описание                                  |
|---------------------|-------|----------------------------------------------|
| D3DADAPTER \_ по умолчанию | 0     | Используется для указания основного видеоадаптера. |
