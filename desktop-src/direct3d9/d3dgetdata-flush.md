# <a name="d3dgetdata_flush"></a>D3DGETDATA \_ запись на диск

Этот макрос создает значение, передаваемое в [**GetData**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3dquery9-getdata) для сброса данных запроса.

``` syntax
#define D3DGETDATA_FLUSH (1 << 0)
```
