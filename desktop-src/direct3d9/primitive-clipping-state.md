# <a name="primitive-clipping-state-direct3d-9"></a>Отсеченное состояние примитива (Direct3D 9)

Примитивы, которые находятся частично внутри (или полностью вне) [представления фрустум](viewports-and-clipping.md) , можно обрезать таким образом, чтобы отображалась только видимая часть примитива. Обрезка сокращает объем работы, который выполняется только для тех примитивов или частей примитива, которые будут видимы.

Чтобы использовать конвейер для обрезки, установите для \_ состояния прорисовки D3DRS обрезки значение **true** (по умолчанию), чтобы включить обрезку, или **false** , чтобы отключить вырезание Direct3D.
