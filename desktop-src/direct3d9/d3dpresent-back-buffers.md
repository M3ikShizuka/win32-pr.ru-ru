# <a name="d3dpresent_back_buffers"></a>D3DPRESENT \_ задние \_ буферы

Указывает количество задних буферов, которые может использовать устройство.



| \#определенно                           | Значение | Описание                                               |
|------------------------------------|-------|-----------------------------------------------------------|
| \_ \_ Максимальное число буферов ЗАДНего D3DPRESENT \_     | 3     | Максимальное число задних буферов, поддерживаемое Direct3D 9.   |
| \_ \_ Максимальное число D3DPRESENT буферов обратной передачи \_ \_ | 30    | Максимальное число задних буферов, поддерживаемых в Direct3D 9Ex. |
