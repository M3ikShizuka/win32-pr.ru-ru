# <a name="idirectxfileenumobject-interface"></a>Интерфейс Идиректксфилинумобжект

Приложения используют методы интерфейса Идиректксфилинумобжект для прохода по объектам данных в файле и для получения объекта данных по его глобальному уникальному идентификатору (GUID) или по имени. Не рекомендуется.

## <a name="members"></a>Элементы

Интерфейс **идиректксфилинумобжект** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Идиректксфилинумобжект** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **идиректксфилинумобжект** содержит следующие методы.



| Метод                                                                     | Описание                                                                     |
|:---------------------------------------------------------------------------|:--------------------------------------------------------------------------------|
| [**жетдатаобжектбид**](idirectxfileenumobject--getdataobjectbyid.md)     | Извлекает объект данных с указанным GUID. Не рекомендуется.<br/>   |
| [**жетдатаобжектбинаме**](idirectxfileenumobject--getdataobjectbyname.md) | Извлекает объект данных с указанным именем. Не рекомендуется.<br/>   |
| [**жетнекстдатаобжект**](idirectxfileenumobject--getnextdataobject.md)     | Извлекает следующий объект верхнего уровня в файле DirectX. Не рекомендуется.<br/> |



 

## <a name="remarks"></a>Remarks

Идентификатор GUID для интерфейса Идиректксфилинумобжект — IID \_ идиректксфилинумобжект.

Тип ЛПДИРЕКТКСФИЛИНУМОБЖЕКТ определяется как указатель на этот интерфейс.


```
typedef interface IDirectXFileEnumObject *LPDIRECTXFILEENUMOBJECT;
```
