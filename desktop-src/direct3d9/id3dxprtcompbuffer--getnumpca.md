# <a name="id3dxprtcompbuffergetnumpca-method"></a>Метод ID3DXPRTCompBuffer:: Жетнумпка

Извлекает количество основных векторов анализа компонентов (PCA) для использования в каждом кластере.

## <a name="syntax"></a>Синтаксис


```C++
UINT GetNumPCA();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **uint**](../winprog/windows-data-types.md)**

Возвращает количество векторов базового анализа основных компонентов (PCA) для использования в каждом кластере.
