# <a name="id3dxkeyframedanimationsetgetsourcetickspersecond-method"></a>Метод ID3DXKeyframedAnimationSet:: Жетсаурцетикксперсеконд

Возвращает число тактов кадра анимации, произошедших в секунду.

## <a name="syntax"></a>Синтаксис


```C++
DOUBLE GetSourceTicksPerSecond();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **Double**](../winprog/windows-data-types.md)**

Число тактов кадра анимации, которые находятся в секунду.
