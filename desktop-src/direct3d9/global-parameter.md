# <a name="global-parameter"></a>Глобальный параметр

Каждый ДКССАС, соответствующий требованиям, должен определить как минимум один параметр глобального действия с глобальной семантикой. Глобальный параметр также может использовать одну или несколько необязательных заметок. Синтаксис выглядит следующим образом:


```
int VariableName : SasGlobal
<
    SasVersion 
    [OptionalAnnotations]
>;
```



где:

-   VariableName — это заданное пользователем имя переменной текстовой строки в формате ASCII.
-   Сасглобал — это семантическое ключевое слово, которое определяет этот параметр как глобальный параметр ДКССАС.
-   Сасверсион является единственной обязательной заметкой.
-   Оптионаланнотатионс может включать следующее:
    -   [сасеффектаусор](#saseffectauthor)
    -   [сасеффектаусорингсофтваре](#saseffectauthoringsoftware)
    -   [сасеффекткатегори](#saseffectcategory)
    -   [сасеффекткомпани](#saseffectcompany)
    -   [сасеффектдескриптион](#saseffectdescription)
    -   [сасеффекселп](#saseffecthelp)
    -   [сасеффектревисион](#saseffectrevision)

## <a name="sasversion"></a>сасверсион

Единственная обязательная Аннотация — Сасверсион. Он объявляется следующим образом:


```
int3 SasVersion = { major, minor, revision };
```



где:

-   Основная указывает на основной выпуск ДКССАС. Основные выпуски ДКССАС могут содержать изменения в наборе семантики и заданных заметок. Семантика и заметки могут добавляться и удаляться, и в целом обратная совместимость с предыдущими выпусками не гарантируется.
-   дополнительный указывает дополнительный выпуск SAS. Вспомогательные выпуски ДКССАС могут включать в себя добавление новой семантики или заметок. Кроме того, семантика и аннотации могут быть помечены как устаревшие в стандарте ДКССАС. Устаревшие семантика и заметки по-прежнему должны поддерживаться ведущими приложениями, но при этом могут выдавать предупреждения диагностики при использовании такой семантики или аннотации. Вспомогательные версии обратно совместимы с предыдущими выпусками.
-   Редакция указывает редакцию ДКССАС. Редакции ДКССАС существуют только как средства для исправления ошибок, удаления неоднозначности и уточнения стандарта. Изменения в стандарте обратно совместимы с предыдущими выпусками.

Текущая версия — 1.0.0. Для этой заметки значение по умолчанию отсутствует.

## <a name="saseffectauthor"></a>сасеффектаусор

Здесь объявляется лицо, создавшее этот результат. Он объявляется следующим образом:


```
string SasEffectAuthor = "value";
```



где value — это строка, идентифицирующая автора действия. Значение по умолчанию - пустая строка.

## <a name="saseffectauthoringsoftware"></a>сасеффектаусорингсофтваре

Он объявляет программное обеспечение, на котором был создан этот результат. Он объявляется следующим образом:


```
string SasEffectAuthoringSoftware = "value";
```



где value — это строка, идентифицирующая программное обеспечение для создания эффектов. Значение по умолчанию - пустая строка.

## <a name="saseffectcategory"></a>сасеффекткатегори

Это объявляет категорию влияния. Он объявляется следующим образом:


```
string SasEffectCategory = "value";
```



где value — это строка, идентифицирующая категорию эффектов. Значение по умолчанию - пустая строка. Категория выражается с помощью значения, похожего на путь, с использованием косой черты в качестве разделителя. Эффекты могут принадлежать только одной категории, так как для выражения включения в нескольких путях в одном значении Сасеффекткатегори нет синтаксиса. Значение этой заметки не обрабатывается ведущим приложением с учетом регистра.

## <a name="saseffectcompany"></a>сасеффекткомпани

Это объявляет компанию, которая создала этот результат. Он объявляется следующим образом:


```
string SasEffectCompany = "value";
```



где value — это строка, идентифицирующая имя компании, которой принадлежит этот результат. Значение по умолчанию - пустая строка.

## <a name="saseffectdescription"></a>сасеффектдескриптион

Это описание действия. Он объявляется следующим образом:


```
string SasEffectDescription = "value";
```



где value — это строка, описывающая результат. Значение по умолчанию - пустая строка.

## <a name="saseffecthelp"></a>сасеффекселп

Это строка справки, которая может отображаться для пользователя при запросе справки по связанному с ним результату. Он объявляется следующим образом:


```
string SasEffectHelp = "value";
```



где value — это строка, которая может отображаться, если пользователь запрашивает справку. Значение по умолчанию - пустая строка.

## <a name="saseffectrevision"></a>сасеффектревисион

Эта заметка позволяет средствам и пользователям записывать номер редакции связанного файла эффектов. Например, пользователи могут вставить соответствующие ключевые слова в значение этой заметки, чтобы вызвать подстановку ключевых слов в своем любимом программном обеспечении управления версиями. Он объявляется следующим образом:


```
string SasEffectRevision = "value";
```



где value — это строка, идентифицирующая редакцию действия. Значение по умолчанию - пустая строка.

## <a name="examples"></a>Примеры

Ниже приведен пример, в котором используется только единственная обязательная Аннотация:


```
int gp : SasGlobal
<
  int3 SasVersion = {1,0,0};
>;
```



Ниже приведен пример, в котором используется обязательная Аннотация и несколько необязательных заметок.


```
int gp : SasGlobal
<
  int3 SasVersion = {1,0,0};
  string SasEffectAuthor = "Mike's Shader";
  string SasEffectAuthoringSoftware = "fxe 2.5.4";
  string SasEffectCategory = "/surface/procedural/wood";
  string SasEffectCompany = "Microsoft Corporation";
  string SasEffectDescription = "Renders an irridescent surface.";
  string SasEffectHelp = "For more information, see https://somelocation/skin.htm";    
  string SasEffectRevision = "$Revision$";  
>;
```
