# <a name="animtickspersecond"></a>анимтикксперсеконд

Описывает коэффициент масштабирования анимации.

``` syntax
template AnimTicksPerSecond
{
    < 9E415A43-7BA6-4a73-8743-B73D47E88476 >
    DWORD AnimTicksPerSecond;
} 
```

Где:

-   Анимтикксперсеконд — коэффициент масштабирования для преобразования штампов времени анимации в глобальное время. Его значение делит время анимации на 30, 60, 24 или 25.
