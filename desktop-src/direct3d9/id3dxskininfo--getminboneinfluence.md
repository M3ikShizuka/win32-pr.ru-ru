# <a name="id3dxskininfogetminboneinfluence-method"></a>Метод ID3DXSkinInfo:: Жетминбонеинфлуенце

Возвращает минимальную влияние на кость. Влияние значений меньше этого значения не учитывается.

## <a name="syntax"></a>Синтаксис


```C++
FLOAT GetMinBoneInfluence();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **float**](../winprog/windows-data-types.md)**

Возвращает минимальное значение влияния на кость.
