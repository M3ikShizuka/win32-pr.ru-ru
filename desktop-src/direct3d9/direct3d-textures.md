# <a name="direct3d-textures-direct3d-9"></a>Текстуры Direct3D (Direct3D 9)

Текстуры — это очень мощное средство для обеспечения реалистичности созданных на компьютере трехмерных изображений. Direct3D поддерживает широкий набор функций для работы с текстурами, предоставляя разработчикам удобный доступ к дополнительным методам обработки текстур.

В этом разделе вы узнаете, как приступить к работе с текстурами.

-   [Основные понятия текстурирования (Direct3D 9)](basic-texturing-concepts.md)
-   [Координаты текстуры (Direct3D 9)](texture-coordinates.md)
-   [Фильтрация текстур (Direct3D 9)](texture-filtering.md)
-   [Ресурсы текстуры (Direct3D 9)](texture-resources.md)
-   [Перенос текстур (Direct3D 9)](texture-wrapping.md)
-   [Смешение текстур (Direct3D 9)](texture-blending.md)

Эти разделы содержат более подробные сведения о дополнительных функциях текстурирования.

-   [Автоматическое создание MIP-карты (Direct3D 9)](automatic-generation-of-mipmaps.md)
-   [Автоматическое управление текстурами (Direct3D 9)](automatic-texture-management.md)
-   [Сжатые ресурсы текстуры (Direct3D 9)](compressed-texture-resources.md)
-   [Рекомендации по оборудованию для текстурирования (Direct3D 9)](hardware-considerations-for-texturing.md)
-   [Ресурсы текстуры тома (Direct3D 9)](volume-texture-resources.md)

Для повышения производительности рассмотрите возможность использования динамических текстур. Динамические текстуры можно блокировать, производить в них запись и разблокировать для каждого кадра. См. раздел [Использование динамических текстур](performance-optimizations.md).
