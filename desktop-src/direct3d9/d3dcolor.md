# <a name="d3dcolor"></a>D3DCOLOR

Определяет основной тип цвета Direct3D.


```C++
typedef DWORD D3DCOLOR;
```



## <a name="remarks"></a>Remarks

4-байтовое значение, которое обычно представляет альфа-, красный, зеленый и синий компоненты цвета. Он также может представлять светимость и яркость.

Тип **D3DCOLOR** можно задать с помощью одного из следующих макросов.

-   [**D3DCOLOR \_ ARGB**](d3dcolor-argb.md)
-   [**D3DCOLOR \_ айув**](d3dcolor-ayuv.md)
-   [**D3DCOLOR \_ колорвалуе**](d3dcolor-colorvalue.md)
-   [**D3DCOLOR \_ RGBA**](d3dcolor-rgba.md)
-   [**D3DCOLOR \_ ксргб**](d3dcolor-xrgb.md)
-   [**D3DCOLOR \_ ксюв**](d3dcolor-xyuv.md)
