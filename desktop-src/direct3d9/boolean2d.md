# <a name="boolean2d"></a>Boolean2d

Определяет набор из двух логических значений, используемых в шаблоне [**мешфацеврапс**](meshfacewraps.md) для определения топологии текстуры отдельного лица.

``` syntax
template Boolean2d
{
    < 4885AE63-78E8-11cf-8F52-0040333594A3 >
    Boolean u;
    Boolean v;
} 
```

Где:

-   u — логическое значение. См. раздел [**Boolean**](boolean.md).
-   v — логическое значение. См. раздел [**Boolean**](boolean.md).
