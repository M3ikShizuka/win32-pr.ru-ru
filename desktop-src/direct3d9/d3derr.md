## <a name="remarks"></a>Remarks

Ошибки представлены отрицательными значениями и не могут быть объединены.

[ \_ Имеются](device-state-return-codes.md)и другие коды возврата.

Код устройства \_ FACD3D используется для создания кодов ошибок, как в следующих макросах.


```
// Direct3D Errors
#define D3DERR_UNSUPPORTEDFACTORVALUE  MAKE_D3DHRESULT(2079)
#define D3DERR_WASSTILLDRAWING         MAKE_D3DHRESULT(540)
    
#define _FACD3D                  0x876   // Direct3D facility code
#define MAKE_D3DHRESULT( code )  MAKE_HRESULT( 1, _FACD3D, code )
#define MAKE_D3DSTATUS( code )   MAKE_HRESULT( 0, _FACD3D, code )
```
