# <a name="patch"></a>Обновление

Определяет исправление элемента управления Безье. Массив определяет контрольные точки для исправления.

``` syntax
template Patch
{
    < A3EB5D44-FC22-429D-9AFB-3221CB9719A6 >
    DWORD nControlIndices;
    array DWORD controlIndices[nControlIndices];
} 
```

Где:

-   Нконтролиндицес — число индексов контрольных точек.
-   массив DWORD Контролиндицес \[ нконтролиндицес \] -массив индексов контрольных точек.

Тип исправления определяется количеством контрольных точек, как показано в следующей таблице.



| Число контрольных точек | Тип                              |
|--------------------------|-----------------------------------|
| 10                       | Треугольное исправление Безье третьего порядка     |
| 15                       | Треугольное исправление для Безье   |
| 16                       | Исправление для четырехъядерного прямоугольника Безье третьего порядка |



 

Порядок контрольных точек задается в шаблоне «спираль», как показано на следующих диаграммах для треугольных и прямоугольных исправлений.
