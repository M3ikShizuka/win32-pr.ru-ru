# <a name="xskinmeshheader"></a>ксскинмешхеадер

Этот шаблон создается на основе отдельной сетки только в тех сетках, которые содержат экспортированные сведения о обложках. Этот шаблон предназначен для предоставления сведений о характере экспортированной информации о обложках.

``` syntax
template XSkinMeshHeader 
{ 
    < 3CF169CE-FF7C-44ab-93C0-F78F62D172E2 >  
    WORD nMaxSkinWeightsPerVertex; 
    WORD nMaxSkinWeightsPerFace; 
    WORD nBones; 
}
```

Где:

-   Нмаксскинвеигхтспервертекс — максимальное число преобразований, влияющих на вершину сетки.
-   Нмаксскинвеигхтсперфаце — максимальное число уникальных преобразований, влияющих на три вершины любой грани.
-   Нбонес — число костей, влияющих на вершины в этой сетке.
