# <a name="outline-and-fill-state-direct3d-9"></a>Состояние структуры и заливки (Direct3D 9)

Примитивы, не имеющие текстур, визуализируются с цветом, указанным в материалах, или с цветами, заданными для вершин, если таковые имеются. Можно выбрать метод для заполнения этих данных, указав значение, определенное перечисляемым типом [**D3DFILLMODE**](./d3dfillmode.md) для \_ состояния отрисовки D3DRS филлмоде.

Чтобы включить дизеринг, приложение должно передать \_ перечисленное значение D3DRS дисеренабле в качестве первого параметра в [**IDirect3DDevice9:: сетрендерстате**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-setrenderstate). Необходимо установить для второго параметра **значение true** , чтобы включить дизеринг, и **false** , чтобы отключить его.

Иногда Рисование последнего пикселя в строке может привести к невозможности перекрытия окружающих примитивов. Это можно контролировать с помощью \_ перечислимого значения D3DRS ластпиксел. Однако не изменяйте этот параметр без некоторых обдумывания. В некоторых случаях подавление отрисовки последнего пикселя может привести к некачественному зазору между примитивами.

Контуры объектов можно нарисовать, установив соответствующий шаблон рисования линии. Состояние рисования линии по умолчанию — Рисование сплошных линий. Дополнительные сведения см. [в разделе Поддержка рисования линий в D3DX (Direct3D 9)](line-drawing-support-in-d3dx.md) Render State.
