# <a name="manipulating-the-depth-buffer-direct3d-9"></a>Управление буфером глубины (Direct3D 9)

Буферы глубины связаны с устройством. Приложения должны перемещать буферы глубины при установке целевых объектов рендеринга. Методы [**IDirect3DDevice9:: жетдепсстенЦилсурфаце**](/windows/desktop/api) и [**IDirect3DDevice9:: сетдепсстенЦилсурфаце**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-setdepthstencilsurface) используются для обработки буферов глубины.
