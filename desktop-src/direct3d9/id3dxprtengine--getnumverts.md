# <a name="id3dxprtenginegetnumverts-method"></a>Метод ID3DXPRTEngine:: Жетнумвертс

Извлекает количество вершин в сетке, включая все новые вершины, добавленные в результате адаптивной пространственной выборки.

## <a name="syntax"></a>Синтаксис


```C++
UINT GetNumVerts();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **uint**](../winprog/windows-data-types.md)**

Количество вершин в сетке, включая все новые вершины, добавленные в результате адаптивной выборки.
