# <a name="programmable-vertex-processing-direct3d-9"></a>Программируемая обработка вершин (Direct3D 9)

При использовании запрограммированного шейдера вершин элементы, обновленные в целевом буфере вершин, контролируются программой-функцией шейдера. Когда приложение записывает данные в один из конечных реестров вершин, обновляется соответствующий элемент в каждой вершине целевого буфера вершин. Элементы в целевом буфере вершин, которые не записываются приложением, не изменяются. [**IDirect3DDevice9::P роцессвертицес**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-processvertices) завершится ошибкой, если приложение записывает данные в элементы, отсутствующие в целевом буфере вершин.
