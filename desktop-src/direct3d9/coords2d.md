# <a name="coords2d"></a>Coords2d

Определяет двухмерный вектор, используемый для определения координат текстуры (u, v) сетки.

``` syntax
template Coords2d
{
    < F6F23F44-7686-11cf-8F52-0040333594A3 >
    float u;
    float v;
} 
```

-   значение координаты u-u.
-   значение координаты v-v.
