# <a name="other-d3dx-constants"></a>Другие константы D3DX

К другим константам D3DX относятся следующие.

## <a name="16-bit-floating-point-numbers"></a>16-разрядные числа с плавающей запятой



|                         |                                         |
|-------------------------|-----------------------------------------|
| \#определенно                | Описание                             |
| D3DX \_ 16F \_          | Число десятичных разрядов точности   |
| D3DX \_ 16F \_ Эпсилон      | Наименьший такой, что 1,0 + Эпсилон! = 1,0 |
| D3DX \_ 16F \_ Мант \_    | Число бит в мантиссе              |
| D3DX \_ 16F \_ Max          | Максимальное значение                           |
| D3DX \_ 16F \_ Max \_ 10 \_ exp | Максимальный показатель десятичной степени                |
| D3DX \_ 16F \_ Max \_ exp     | Максимальный показатель двоичной степени                 |
| D3DX \_ 16F \_ мин          | Минимальное положительное значение                  |
| D3DX \_ 16F \_ min \_ 10 \_ exp | Минимальный показатель десятичной степени                |
| D3DX \_ 16F \_ min \_ exp     | Минимальный показатель двоичной степени                 |
| D3DX \_ 16F \_ основание системы счисления        | Основание системы счисления                          |
| D3DX \_ 16F \_ Округляет       | Округление при сложении: near                 |
| D3DX \_ 1BYPI             | 1/pi                                    |
| D3DX \_ Pi                | pi                                      |
| D3DX \_ по умолчанию — \_ float    | Максимальное значение float                     |



 

Эти \# определения объявляются в d3dx9. h и d3dx9math. h.

## <a name="prt-constants"></a>Константы PRT



|                  |                                            |
|------------------|--------------------------------------------|
| \#определенно         | Описание                                |
| D3DXSH \_ минордер | Наименьший допустимый порядок моделирования.  |
| D3DXSH \_ максордер | Наибольший допустимый порядок моделирования. |



 

Эти \# определения объявляются в d3dx9math. h. Дополнительные сведения о PRT см. в разделе [предварительно вычисленный перенос радианце (Direct3D 9)](precomputed-radiance-transfer.md).

## <a name="texture-constants"></a>Константы текстуры



|                        |                                                                    |
|------------------------|--------------------------------------------------------------------|
| \#определенно               | Описание                                                        |
| D3DFMT \_ из \_ файла     | Возьмем формат в точности из файла.                               |
| D3DX \_ по умолчанию          | Значение по умолчанию.                                                   |
| D3DX \_ по умолчанию \_ NONPOW2 | Не округляйте числа, например ширину или высоту, в степень числа двух. |
| D3DX \_ из \_ файла       | Возьмите измерения текстуры точно из файла.                   |



 

Эти \# определения объявляются в d3dx9. h.

## <a name="other-d3dx-constants"></a>Другие константы D3DX

Константы версий UNUSED16 и D3DX, \_ определенные в D3dx9mesh. h и D3dx9core. h, используются внутренне. Не используйте эти константы.
