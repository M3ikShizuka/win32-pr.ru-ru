# <a name="binary-encoding"></a>Двоичное кодирование

В этом разделе описывается двоичная версия формата файла DirectX (. x), представленная в выпуске DirectX 3,0.

Двоичный формат представляет собой лексическое представление текстового формата. Маркеры могут быть изолированными или сопровождаться простыми записями данных. Изолированные токены предоставляют грамматические структуры, а токены для чтения и записи предоставляют необходимые данные.

Обратите внимание, что все данные хранятся в формате с прямым порядком байтов. Допустимый поток двоичных данных состоит из заголовка, за которым следуют шаблоны и (или) объекты данных.

В этом разделе обсуждаются следующие компоненты формата двоичного файла и приводится пример шаблона и двоичного объекта данных.

-   [Header](header.md)
-   [Лексемы](tokens.md)
-   [Записи маркеров](token-records.md)
-   [Шаблоны](dx9-graphics-reference-x-file-binaryencoding-templates.md)
-   [Data](dx9-graphics-reference-x-file-binaryencoding-data.md)
-   [Примеры](examples.md)
