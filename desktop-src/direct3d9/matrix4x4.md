# <a name="matrix4x4"></a>Matrix4x4

Определяет матрицу 4 x 4. Используется в качестве матрицы преобразования кадров.

``` syntax
template Matrix4x4
{
    < F6F23F45-7686-11cf-8F52-0040333594A3 >
    array float matrix[16];
} 
```

Где:

-   массив с плавающей запятой массива \[ 16 \] — 16 массивов с плавающей запятой.
