# <a name="id3dxprtbufferistexture-method"></a>Метод ID3DXPRTBuffer:: «текстурирование»

Указывает, содержит ли буфер текстуру.

## <a name="syntax"></a>Синтаксис


```C++
BOOL IsTexture();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **bool** .](../winprog/windows-data-types.md)**

Возвращает **значение true** , если буфер содержит текстуру; в противном случае возвращает **false** .
