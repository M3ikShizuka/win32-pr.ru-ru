# <a name="d3dcursorcaps"></a>D3DCURSORCAPS

Флаги возможностей курсора драйвера.



| \#определенно              | Значение       | Описание                                                                                     |
|-----------------------|-------------|-------------------------------------------------------------------------------------------------|
| \_Цвет D3DCURSORCAPS  | 0x00000001L | Драйвер поддерживает аппаратный цветовой курсор в режимах с высоким разрешением (высота >= 400). |
| D3DCURSORCAPS \_ Ловрес | 0x00000002L | Драйвер поддерживает аппаратный цветовой курсор в режимах с низким разрешением (высота < 400).            |

|--------------------------|------------|
| Заголовок                   | d3d9caps. h |
| Минимальная операционная система | Windows 98 |
