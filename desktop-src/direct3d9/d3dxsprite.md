# <a name="d3dxsprite"></a>D3DXSPRITE

Следующие флаги используются для указания параметров отрисовки спрайта в параметре flags метода [**Begin**](id3dxsprite--begin.md) :



| \#определенно                             | Описание                                                                                                                                                                                                                                                                                                                                                                                                                       |
|--------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| D3DXSPRITE \_ донотсавестате           | Состояние устройства не должно сохраняться или восстанавливаться при вызове [**Begin**](id3dxsprite--begin.md) или [**End**](id3dxsprite--end.md) .                                                                                                                                                                                                                                                                                            |
| D3DXSPRITE \_ донотмодифи \_ рендерстате | Состояние отрисовки устройства не должно изменяться при вызове [**Begin**](id3dxsprite--begin.md) . Предполагается, что устройство находится в допустимом состоянии для рисования вершин, содержащих Усажеиндекс = 0 в D3DDECLUSAGE \_ положении, D3DDECLUSAGE \_ ТЕКСКУРД и D3DDECLUSAGE \_ Color Data.                                                                                                                                                     |
| D3DXSPRITE \_ обжектспаце              | Преобразования мира, представления и проекции не изменяются. Преобразования, настроенные для устройства, используются для трансформации спрайтов при прорисовке пакетных спрайтов (при вызове [**flush**](id3dxsprite--flush.md) или [**End**](id3dxsprite--end.md) ). Если этот флаг не указан, изменения в мире, представлении и проекции будут изменены таким образом, чтобы спрайты рисулись в координатах экранного пространства.              |
| D3DXSPRITEный \_ стенд                | Каждый спрайт будет повернут вокруг своего центра, чтобы он был подключен к средству просмотра. Сначала необходимо вызвать [**сетворлдвиевлх**](id3dxsprite--setworldviewlh.md) или [**сетворлдвиеврх**](id3dxsprite--setworldviewrh.md) .                                                                                                                                                                                                                |
| D3DXSPRITE \_ алфабленд               | Включает альфа-смешение с параметром D3DRS \_ алфатестенабле, имеющим значение **true** (для ненулевой альфа-составляющей). D3DBLEND \_ сркалфа будет являться источником состояния смешения, а D3DBLEND \_ инвсркалфа — целевым состоянием смешения в вызовах [**сетрендерстате**](/windows/desktop/api). См. раздел [состояние альфа-смешения (Direct3D 9)](alpha-blending-state.md). [**ID3DXFont**](id3dxfont.md) ждет, когда этот флаг будет установлен при рисовании текста. |
| \_ \_ Текстура сортировки D3DXSPRITE            | Сортировка спрайтов по текстуре перед рисованием. Это может повысить производительность при рисовании неперекрывающихся спрайтов с равномерным глубиной. Можно также сочетать D3DXSPRITEную \_ \_ текстуру сортировки с D3DXSPRITE \_ \_ глубины сортировки \_ фронттобакк или D3DXSPRITE \_ \_ глубиной сортировки \_ бакктофронт. После этого список спрайтов будет отсортирован по глубине, а второй — по текстуре.<br/>                                                                           |
| D3DXSPRITE \_ Сортировка \_ глубины \_ фронттобакк | Перед рисованием спрайты сортируются по глубине в порядке «перед задним планом». Эту процедуру рекомендуется использовать при рисовании непрозрачных спрайтов с различной глубиной. Вы можете объединить \_ \_ глубину сортировки D3DXSPRITE \_ фронттобакк с \_ D3DXSPRITE \_ текстурой сортировки, чтобы сортировать сначала по глубине, а затем по текстуре.<br/>                                                                                                                                   |
| D3DXSPRITE \_ Сортировка \_ глубины \_ бакктофронт | Перед рисованием спрайты сортируются по глубине в обратном порядке. Эту процедуру рекомендуется использовать при рисовании прозрачных спрайтов с различной глубиной. Вы можете объединить \_ \_ глубину сортировки D3DXSPRITE \_ бакктофронт с \_ D3DXSPRITE \_ текстурой сортировки, чтобы сортировать сначала по глубине, а затем по текстуре.<br/>                                                                                                                              |
| D3DXSPRITE \_ не \_ использовать \_ ADDREF \_ | Отключает вызов AddRef () для каждой прорисовки и Release () на Flush () для повышения производительности.                                                                                                                                                                                                                                                                                                                                         |
