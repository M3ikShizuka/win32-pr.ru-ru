# <a name="id3dxfontgetdc-method"></a>Метод ID3DXFont:: GetDC

Возвращает маркер для контекста дисплея устройства (DC) с набором шрифтов.

## <a name="syntax"></a>Синтаксис


```C++
HDC GetDC();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **HDC**](../winprog/windows-data-types.md)**

Обработайте с КОНТРОЛЛЕРом экрана.
