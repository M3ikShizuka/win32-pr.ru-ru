# <a name="id3dxfont-interface"></a>Интерфейс ID3DXFont

Интерфейс ID3DXFont инкапсулирует текстуры и ресурсы, необходимые для отображения определенного шрифта на определенном устройстве.

## <a name="members"></a>Элементы

Интерфейс **ID3DXFont** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **ID3DXFont** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ID3DXFont** содержит следующие методы.



| Метод                                                    | Описание                                                                                                                                                                                                                                   |
|:----------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DrawText**](id3dxfont--drawtext.md)                   | Рисует форматированный текст. Этот метод поддерживает строки ANSI и Юникод.<br/>                                                                                                                                                               |
| [**GetDC**](id3dxfont--getdc.md)                         | Возвращает маркер для контекста дисплея устройства (DC) с набором шрифтов.<br/>                                                                                                                                                           |
| [**GetDesc**](id3dxfont--getdesc.md)                     | Возвращает описание текущего объекта Font. Жетдескв и DESC идентичны этому методу, за исключением того, что указатель возвращается в структуру [**D3DXFONT \_ Дескв**](d3dxfont-desc.md) или **D3DXFONT \_ DESC** соответственно.<br/> |
| [**GetDevice**](id3dxfont--getdevice.md)                 | Извлекает устройство Direct3D, связанное с объектом Font.<br/>                                                                                                                                                                     |
| [**жетглифдата**](id3dxfont--getglyphdata.md)           | Возвращает сведения о размещении и ориентации глифа в ячейке символов.<br/>                                                                                                                                            |
| [**жеттекстметрикс**](id3dxfont--gettextmetrics.md)       | Получает характеристики шрифта, определенные в структуре [**текстметрик**](/windows/win32/api/wingdi/ns-wingdi-textmetrica) . Этот метод поддерживает параметры компилятора ANSI и Unicode.<br/>                                                                       |
| [**онлостдевице**](id3dxfont--onlostdevice.md)           | Используйте этот метод, чтобы освободить все ссылки на ресурсы видеопамяти и удалить все статеблоккс. Этот метод должен вызываться при каждом потере устройства или перед сбросом устройства.<br/>                                              |
| [**онресетдевице**](id3dxfont--onresetdevice.md)         | Этот метод используется для повторного получения ресурсов и сохранения начального состояния.<br/>                                                                                                                                                                    |
| [**прелоадчарактерс**](id3dxfont--preloadcharacters.md) | Загружает ряд символов в видеопамять для повышения эффективности отрисовки на устройстве.<br/>                                                                                                                               |
| [**прелоадглифс**](id3dxfont--preloadglyphs.md)         | Загружает ряд глифов в видеопамять для повышения эффективности отрисовки на устройстве.<br/>                                                                                                                                   |
| [**прелоадтекст**](id3dxfont--preloadtext.md)             | Загружает форматированный текст в видеопамять для повышения эффективности отрисовки на устройстве. Этот метод поддерживает строки ANSI и Юникод.<br/>                                                                                        |



 

## <a name="remarks"></a>Remarks

Интерфейс **ID3DXFont** получается путем вызова [**D3DXCreateFont**](d3dxcreatefont.md) или [**D3DXCreateFontIndirect**](d3dxcreatefontindirect.md).

Тип LPD3DXFONT определяется как указатель на интерфейс **ID3DXFont** .


```
typedef interface ID3DXFont ID3DXFont;
typedef interface ID3DXFont *LPD3DXFONT;
```
