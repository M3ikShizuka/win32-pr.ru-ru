# <a name="d3dvtxpcaps"></a>D3DVTXPCAPS

Сочетание одного или нескольких флагов, управляющих поведением при создании устройства.



| \#определенно                                | Описание                                                                                                                                                                                        |
|-----------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| D3DVTXPCAPS \_ директионаллигхтс          | Устройство может выполнять направленный свет.                                                                                                                                                                  |
| D3DVTXPCAPS \_ локалвиевер                | Устройство может выполнять локальное средство просмотра.                                                                                                                                                                        |
| D3DVTXPCAPS \_ MATERIALSOURCE7            | Если эта политика установлена, устройство поддерживает цветовые материальные состояния: D3DRS \_ амбиентматериалсаурце, D3DRS \_ ДИФФУСЕМАТЕРИАЛСАУРЦЕ, D3DRS \_ емиссивематериалсаурце и D3DRS \_ спекуларматериалсаурце. |
| D3DVTXPCAPS \_ No \_ тексжен \_ нонлокалвиевер | Устройство не поддерживает создание текстур в режиме нелокального просмотра.                                                                                                                               |
| D3DVTXPCAPS \_ поситионаллигхтс           | Устройство может выполнять позиционированные источники света (включая точку и место).                                                                                                                                         |
| D3DVTXPCAPS \_ тексжен                     | Устройство может выполнять тексжен.                                                                                                                                                                              |
| D3DVTXPCAPS \_ тексжен \_ сферемап          | Устройство поддерживает D3DTSS \_ тЦи \_ сферемап.                                                                                                                                                            |
| Tween — D3DVTXPCAPS \_                   | Устройство может выполнять анимацию вершин.                                                                                                                                                                     |
