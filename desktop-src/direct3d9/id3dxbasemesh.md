# <a name="id3dxbasemesh-interface"></a>Интерфейс ID3DXBaseMesh

Приложения используют методы интерфейса **ID3DXBaseMesh** для управления и запроса сетки и прогрессивных сетчатых объектов.

## <a name="members"></a>Элементы

Интерфейс **ID3DXBaseMesh** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **ID3DXBaseMesh** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ID3DXBaseMesh** содержит следующие методы.



| Метод                                                                            | Описание                                                                                                                                                                                                           |
|:----------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**клонемеш**](id3dxbasemesh--clonemesh.md)                                     | Создает точную копию сетки с помощью декларатора.<br/>                                                                                                                                                                          |
| [**клонемешфвф**](id3dxbasemesh--clonemeshfvf.md)                               | Создает точную копию сетки, используя код гибкого формата вершин (ФВФ).<br/>                                                                                                                                                   |
| [**конвертаджаценцитопоинтрепс**](id3dxbasemesh--convertadjacencytopointreps.md) | Преобразует сведения о смежности сетки в массив точек зрения.<br/>                                                                                                                                  |
| [**конвертпоинтрепстоаджаценци**](id3dxbasemesh--convertpointrepstoadjacency.md) | Преобразует данные репрезентативного указания в сведения о смежности сетки.<br/>                                                                                                                                          |
| [**дравсубсет**](id3dxbasemesh--drawsubset.md)                                   | Рисует подмножество сетки.<br/>                                                                                                                                                                                  |
| [**женератеаджаценци**](id3dxbasemesh--generateadjacency.md)                     | Создание списка границ сетки, а также списка лиц, совместно использующих каждое ребро.<br/>                                                                                                                            |
| [**жетаттрибутетабле**](id3dxbasemesh--getattributetable.md)                     | Извлекает либо таблицу атрибутов для сетки, либо число записей, хранящихся в таблице атрибутов сетки.<br/>                                                                                          |
| [**Объявление о выходе**](id3dxbasemesh--getdeclaration.md)                           | Извлекает объявление, описывающее вершины в сетке.<br/>                                                                                                                                               |
| [**GetDevice**](id3dxbasemesh--getdevice.md)                                     | Извлекает устройство, связанное с сеткой.<br/>                                                                                                                                                             |
| [**жетфвф**](id3dxbasemesh--getfvf.md)                                           | Возвращает значение фиксированной вершины функции.<br/>                                                                                                                                                                      |
| [**жетиндексбуффер**](id3dxbasemesh--getindexbuffer.md)                           | Получает данные в буфере индекса.<br/>                                                                                                                                                                     |
| [**жетнумбитеспервертекс**](id3dxbasemesh--getnumbytespervertex.md)               | Возвращает число байтов на вершину.<br/>                                                                                                                                                                       |
| [**жетнумфацес**](id3dxbasemesh--getnumfaces.md)                                 | Извлекает количество лиц в сетке.<br/>                                                                                                                                                                 |
| [**жетнумвертицес**](id3dxbasemesh--getnumvertices.md)                           | Возвращает количество вершин в сетке.<br/>                                                                                                                                                              |
| [**Параметры команды**](id3dxbasemesh--getoptions.md)                                   | Извлекает параметры сетки, включенные для этой сетки во время создания.<br/>                                                                                                                                         |
| [**жетвертексбуффер**](id3dxbasemesh--getvertexbuffer.md)                         | Извлекает буфер вершин, связанный с сеткой.<br/>                                                                                                                                                      |
| [**локкиндексбуффер**](id3dxbasemesh--lockindexbuffer.md)                         | Блокирует буфер индексов и получает указатель на буфер памяти буфера индекса.<br/>                                                                                                                                    |
| [**локквертексбуффер**](id3dxbasemesh--lockvertexbuffer.md)                       | Блокирует буфер вершин и получает указатель на память буфера вершин.<br/>                                                                                                                                   |
| [**унлоккиндексбуффер**](id3dxbasemesh--unlockindexbuffer.md)                     | Разблокирует буфер индексов.<br/>                                                                                                                                                                                   |
| [**унлокквертексбуффер**](id3dxbasemesh--unlockvertexbuffer.md)                   | Разблокирует буфер вершин.<br/>                                                                                                                                                                                   |
| [**упдатесемантикс**](id3dxbasemesh--updatesemantics.md)                         | Этот метод позволяет пользователю изменить объявление сетки, не изменяя макет данных в буфере вершин. Вызов допустим только в том случае, если форматы старого и нового объявлений имеют одинаковый размер вершины.<br/> |



 

## <a name="remarks"></a>Remarks

Сетка — это объект, состоящие из набора многоугольных граней. Сетка определяет набор вершин и набор лиц (грани определяются в виде вершин и нормалей сетки).

Тип LPD3DXBASEMESH определяется как указатель на интерфейс **ID3DXBaseMesh** .


```
typedef struct ID3DXBaseMesh *LPD3DXBASEMESH;
```
