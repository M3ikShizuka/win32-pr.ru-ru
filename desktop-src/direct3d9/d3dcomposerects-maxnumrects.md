# <a name="d3dcomposerects_maxnumrects"></a>D3DCOMPOSERECTS \_ макснумректс

Указывает максимальное число прямоугольников, используемых при составлении глифов в [**компосеректс**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9ex-composerects).



|                              |                                                                                                                                      |
|------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| \#определенно                     | Описание                                                                                                                          |
| D3DCOMPOSERECTS \_ макснумректс | Максимальное число глифов прямоугольников для объединения в текстовой строке. См. [**компосеректс**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9ex-composerects). |
