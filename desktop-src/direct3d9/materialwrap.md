# <a name="materialwrap"></a>материалврап

Определяет набор из двух логических значений, используемых в шаблоне [**мешфацеврапс**](meshfacewraps.md) для определения топологии текстуры отдельного лица. Используйте вместо [**Boolean2d**](boolean2d.md) , если координаты текстуры (u, v) охватывают диапазон от 0 до 2 вместо 0 до 1.

``` syntax
template MaterialWrap
{
    < 4885ae60-78e8-11cf-8f52-0040333594a3 >
    Boolean u;
    Boolean v;
} 
```

Где:

-   u — логическое значение. См. раздел [**Boolean**](boolean.md).
-   v — логическое значение. См. раздел [**Boolean**](boolean.md).

> [!Note]  
> Шаблон [**мешфацеврапс**](meshfacewraps.md) больше не используется.
