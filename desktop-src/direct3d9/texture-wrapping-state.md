# <a name="texture-wrapping-state-direct3d-9"></a>Состояние переноса текстур (Direct3D 9)

D3DRS \_ WRAP0 через D3DRS \_ WRAP7 Rendering States включает и отключает перенос u и v для различных текстур в многотекстурном каскадном соединении. Для этих состояний отрисовки можно задать сочетание \_ флагов D3DWRAPCOORD 0, D3DWRAPCOORD \_ 1, D3DWRAPCOORD \_ 2 и D3DWRAPCOORD 3, \_ чтобы включить перенос в первую, второй, третьем и четвертом направлениях текстуры. Чтобы полностью отключить заключение, используйте нулевое значение. По умолчанию перенос текстур отключен во всех направлениях для всех стадий текстуры.

Общие сведения см. в разделе [Перенос текстур (Direct3D 9)](texture-wrapping.md).
