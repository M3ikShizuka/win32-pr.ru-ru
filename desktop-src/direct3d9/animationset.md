# <a name="animationset"></a>Анимация

Содержит один или несколько объектов [**анимации**](animation.md) . Каждая анимация в наборе анимации имеет одно и то же время в любой заданной точке. Увеличение времени набора анимации увеличивает время для всех содержащихся в ней анимаций.

``` syntax
template AnimationSet
{
    < 3D82AB50-62DA-11cf-AB39-0020AF71E433 >
    [ Animation < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 > ]
} 
```

Где:

-   \[ Анимация < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 > \] — необязательный шаблон анимации.
