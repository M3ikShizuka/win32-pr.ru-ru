# <a name="id3dxpatchmeshgetcontrolverticesperpatch-method"></a>Метод ID3DXPatchMesh:: Жетконтролвертицесперпатч

Возвращает число вершин управления на каждое исправление.

## <a name="syntax"></a>Синтаксис


```C++
DWORD GetControlVerticesPerPatch();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **DWORD**](../winprog/windows-data-types.md)**

Число вершин управления для каждого исправления.
