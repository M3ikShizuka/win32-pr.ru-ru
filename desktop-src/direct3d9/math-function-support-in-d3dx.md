# <a name="math-function-support-in-d3dx-direct3d-9"></a>Поддержка математических функций в D3DX (Direct3D 9)

D3DX — это служебная Библиотека, которая предоставляет вспомогательные службы. Это уровень выше компонента Direct3D.

## <a name="math"></a>Математический

Математические функции, содержащиеся в наборе функций, предоставляются для:

-   Вычисления цветов
-   Плоскост
-   Обработка матриц
-   Кватернионы
-   Двумерные векторы
-   Трехмерные векторы
-   векторы 4D

Обратите внимание, что при сочетании с перегрузками C++ поддержка базовых трехмерных математических типов является обширной.

Дополнительные сведения об этих функциях см. в разделе [функции D3DX](dx9-graphics-reference-d3dx-functions.md). Чтобы найти необходимую функцию, они разбиваются на несколько папок.

## <a name="float16"></a>FLOAT16

При использовании типа данных FLOAT16 не забудьте ограничить значения максимальным значением D3DX \_ 16F \_ Max. Любое значение FLOAT16, превышающее это, приведет к неопределенному поведению в конвейере. См. [другие константы D3DX](other-d3dx-constants.md).
