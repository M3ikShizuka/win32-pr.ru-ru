# <a name="dxfile-constants"></a>Константы ДКСФИЛЕ

Следующие флаги используются для указания каналов текстуры для обработки. Не рекомендуется.

## <a name="dxfileformat"></a>дксфилеформат



| \#определенно                 | Значение | Описание      |
|--------------------------|-------|------------------|
| \_двоичный файл дксфилеформат     | 0     | Двоичный файл.     |
| ДКСФИЛЕФОРМАТ \_ текст       | 1     | Текстовый файл.       |
| \_сжатый дксфилеформат | 2     | Сжатый файл. |



 

Эти \# определения объявляются в дксфиле. h.

## <a name="dxfileloadoptions"></a>дксфилелоадоптионс



| \#определенно                 | Значение | Описание                  |
|--------------------------|-------|------------------------------|
| ДКСФИЛЕЛОАД \_ фромфиле     | 0x00 | Загрузить файл из файла.     |
| ДКСФИЛЕЛОАД \_ фромресаурце | 0x01L | Загрузка файла из ресурса. |
| ДКСФИЛЕЛОАД \_ фроммемори   | 0x02L | Загрузка файла из памяти.     |
| ДКСФИЛЕЛОАД \_ фромстреам   | 0x04L | Загрузить файл из потока.   |
| ДКСФИЛЕЛОАД \_ фромурл      | 0x08L | Загрузка файла из URL-адреса.      |



 

Эти \# определения объявляются в дксфиле. h.
