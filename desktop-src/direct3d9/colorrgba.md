# <a name="colorrgba"></a>колорргба

Определяет объект Color с альфа-компонентом. Он используется для цвета лицевой стороны в определении шаблона материала.

``` syntax
template ColorRGBA
{
    < 35FF44E0-6C7C-11cf-8F52-0040333594A3 >
    float red;
    float green;
    float blue;
    float alpha;
} 
```
