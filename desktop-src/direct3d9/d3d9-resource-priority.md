# <a name="d3d9_resource_priority"></a>\_Приоритет ресурса \_ d3d9

Константы, используемые для задания приоритета ресурса в [**SetPriority**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3dresource9-setpriority).



| Константа/значение                                                                                                                                                                                                                                                                     | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="D3D9_RESOURCE_PRIORITY_MINIMUM"></span><span id="d3d9_resource_priority_minimum"></span><dl> <dt>**D3d9 \_ \_ \_ Минимальный приоритет ресурса**</dt> <dt>0x28000000</dt> </dl> | Ресурс имеет самый низкий приоритет. Эта константа помечает ресурс как неиспользуемый и для вытеснения. Ресурс должен быть исключен, как только другой ресурс требует объем памяти, занимаемый ресурсом.<br/>                                                                                                                                                                                                              |
| <span id="D3D9_RESOURCE_PRIORITY_LOW"></span><span id="d3d9_resource_priority_low"></span><dl> <dt>**D3d9 \_ \_Приоритет ресурсов \_ низкая**</dt> <dt>0x50000000</dt> </dl>             | Ресурс запланирован с низким приоритетом. Размещение ресурса не является критическим, и операционная система выполняет минимальную работу, чтобы найти расположение ресурса. Пометка ресурса как низкого приоритета позволяет другим более важным ресурсам занимать более быструю память.<br/>                                                                                                                                                      |
| <span id="D3D9_RESOURCE_PRIORITY_NORMAL"></span><span id="d3d9_resource_priority_normal"></span><dl> <dt>**D3d9 \_ Приоритет ресурса — \_ \_ Обычная**</dt> <dt>0x78000000</dt> </dl>    | Ресурс запланирован с нормальным приоритетом. Расположение ресурса важно для повышения производительности, но это не очень важно. Операционная система должна попытаться поместить ресурс, помеченный как нормальный, в предпочтительное расположение ресурса вместо ресурса с низким приоритетом. Как правило, текстуры помечаются как обычные.<br/>                                                                                                     |
| <span id="D3D9_RESOURCE_PRIORITY_HIGH"></span><span id="d3d9_resource_priority_high"></span><dl> <dt>**D3d9 \_ Приоритет ресурсов — \_ \_ высокий**</dt> <dt>0xA0000000</dt> </dl>          | Ресурс запланирован с высоким приоритетом. Размещение ресурса является критически важным для производительности. Операционная система всегда пытается поместить ресурс, помеченный как высокий, в предпочтительное расположение ресурса вместо ресурса с низким приоритетом или с обычным приоритетом. Как правило, целевые объекты рендеринга помечены как высокие.<br/>                                                                                                         |
| <span id="D3D9_RESOURCE_PRIORITY_MAXIMUM"></span><span id="d3d9_resource_priority_maximum"></span><dl> <dt>**D3d9 \_ \_ \_ Максимальный приоритет ресурса**</dt> <dt>0xc8000000</dt> </dl> | Ресурс имеет максимально возможный приоритет. Эта константа помечает приоритет ресурса как Soft-закрепленный. Кратковременно закрепленный ресурс удаляется из памяти, только если отсутствует другой способ разрешения памяти буфера DMA. Операционная система пытается разделить буфер DMA до минимального размера и исключить все другие ресурсы, которые не закреплены и не закреплены, прежде чем будет удалено программно закрепленный ресурс. <br/> |



## <a name="remarks"></a>Комментарии

Значения, отличные **от \_ d3d9 \_ \_ минимального приоритета ресурса** и **d3d9 \_ \_ приоритета \_ ресурсов** , рассматриваются планировщиком как указания.

Можно использовать уровни приоритета, отличные от значений, определенных ранее в этом разделе. Например, пометка ресурса с уровнем приоритета 0x78000001 указывает, что приоритет ресурса немного выше среднего.
