# <a name="id3dxconstanttablegetbufferpointer-method"></a>Метод ID3DXConstantTable:: Жетбуфферпоинтер

Возвращает указатель на буфер, содержащий таблицу констант.

## <a name="syntax"></a>Синтаксис


```C++
LPVOID GetBufferPointer();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **лпвоид**](../winprog/windows-data-types.md)**

Возвращает указатель на буфер.
