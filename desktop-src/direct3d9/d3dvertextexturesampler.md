# <a name="d3dvertextexturesampler"></a>D3DVERTEXTEXTURESAMPLER

Константы образца текстуры для вершины.

Эти константы определяют выборки текстур, используемые шейдерами вершин.



| \#определенно                 | Описание        |
|--------------------------|--------------------|
| D3DVERTEXTEXTURESAMPLER0 | (D3DDMAPSAMPLER + 1) |
| D3DVERTEXTEXTURESAMPLER1 | (D3DDMAPSAMPLER + 2) |
| D3DVERTEXTEXTURESAMPLER2 | (D3DDMAPSAMPLER + 3) |
| D3DVERTEXTEXTURESAMPLER3 | (D3DDMAPSAMPLER + 4) |



 

## <a name="d3ddmapsampler"></a>D3DDMAPSAMPLER

Это внутренняя константа, используемая образцом текстур в тесселяции.



| \#определенно       | Описание                                           |
|----------------|-------------------------------------------------------|
| D3DDMAPSAMPLER | 256, то есть максимальное число образцов текстуры. |



 

## <a name="constant-information"></a>Сведения о константе



| Требование                         | Значение            |
|--------------------------|-------------|
| Заголовок                   | d3d9types. h |
| Минимальная операционная система | Windows 98  |
