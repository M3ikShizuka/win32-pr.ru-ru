# <a name="d3ddtcaps"></a>D3DDTCAPS

Константы, описывающие типы данных вершин, поддерживаемые устройством.



| \#определенно              | Значение       | Описание                                                                                                                   |
|-----------------------|-------------|-------------------------------------------------------------------------------------------------------------------------------|
| D3DDTCAPS \_ UBYTE4     | 0x00000001L | байт без знака 4D.                                                                                                             |
| D3DDTCAPS \_ UBYTE4N    | 0x00000002L | Нормализованный байт без знака 4D. Каждый из четырех байтов нормализован путем деления на 255,0.                                      |
| D3DDTCAPS \_ SHORT2N    | 0x00000004L | Нормализованная, коротко знакомая с двухмерной подписью, расширенная до (первый байт/32767.0, второй байт/32767.0, 0, 1).                                     |
| D3DDTCAPS \_ SHORT4N    | 0x00000008L | Нормализовано, короткое время со знаком, расширено до (первый байт/32767.0, второй байт/32767.0, третий байт/32767.0, четвертый байт/32767.0).  |
| D3DDTCAPS \_ USHORT2N   | 0x00000010L | Нормализованная, двухмерная без знака, расширенная до (первый байт/65535.0, второй байт/65535.0, 0, 1).                                   |
| D3DDTCAPS \_ USHORT4N   | 0x00000020L | Нормализованное значение неподписанного формата 4D без знака, расширено до (первый байт/65535.0, второй байт/65535.0, третий байт/65535.0, четвертый байт/65535.0). |
| D3DDTCAPS \_ UDEC3      | 0x00000040L | Трехмерный формат без знака 10 10 10, развернутый до (значение, значение, значение, 1).                                                             |
| D3DDTCAPS \_ DEC3N      | 0x00000080L | формат трехмерной подписи 10 10 10 нормализован и расширен до (v \[ 0 \] /511,0, v \[ 1 \] /511,0, v \[ 2 \] /511,0, 1).                           |
| D3DDTCAPS \_ FLOAT16 \_ 2 | 0x00000100L | Двумерные 16-битные числа с плавающей запятой.                                                                                             |
| D3DDTCAPS \_ FLOAT16 \_ 4 | 0x00000200L | 4D 16-разрядные числа с плавающей запятой.                                                                                             |



 

Эти константы используются членом Деклтипес объекта [**D3DCAPS9**](/windows/desktop/api/D3D9Caps/ns-d3d9caps-d3dcaps9).
