# <a name="clouds-smoke-and-vapor-trails-direct3d-9"></a>Облака, состояния и Вапор (Direct3D 9)

Все облака, журналы состояния и Вапор могут быть созданы с помощью расширения технологии объявления. См. раздел [объявления (Direct3D 9)](billboarding.md). При повороте объявления на двух осях, а не на одной, приложение может позволить пользователю просматривать рекламу с любого угла. Как правило, приложение поворачивает объявление на горизонтальной и вертикальной оси.

Чтобы создать простое облако, приложение может повернуть прямоугольный примитив на одной или двух осях, чтобы примитив отгранен пользователю. Затем текстуру, подобную облачной, можно применить к примитиву с прозрачностью. Дополнительные сведения о применении прозрачных текстур к примитивам см. в разделе [смешение текстур (Direct3D 9)](texture-blending.md). Вы можете анимировать облако, применяя ряд текстур с течением времени.

Приложение может создавать более сложные облака, формируя их из группы примитивов. Каждая часть облака является прямоугольным примитивом. Примитивы можно перемещать независимо от времени, чтобы получить внешний вид динамической тумана. Эта концепция показана на следующем рисунке.

![Иллюстрация примитивов, образующих более сложные облака](images/cloud.png)

Внешний вид состояния отображается так же, как и облака. Обычно требуется несколько объявлений, например сложных облаков. Как правило, билловс и растет с течением времени, поэтому объявления, составляющие Плуме, должны перемещаться соответствующим образом. Может потребоваться добавить дополнительные объявления, так как Плуме растет и разбросаны.

Вапорный след — это Плуме, который не растет. Однако, как и в случае с плумеом, он разбросается с течением времени. На следующем рисунке показана методика использования объявлений для имитации вапорного следа.

![Иллюстрация объявлений, имитирующих вапорный след](images/vapor.png)
