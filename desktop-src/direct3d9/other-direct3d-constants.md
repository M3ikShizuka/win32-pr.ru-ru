# <a name="other-direct3d-constants"></a>Другие константы Direct3D

## <a name="sdk-version"></a>Версия пакета SDK



|                     |
|---------------------|
| \#определенно            |
| \_Версия пакета SDK для D3D \_   |
| \_Версия пакета SDK D3D9b \_ |



 

Эти \# определения объявляются в d3d9. h.

## <a name="other-constants"></a>Другие константы

В следующей таблице перечислены константы, которые используются внутри:



| \#определенно                              | Значение                                             | Описание                                                        |
|---------------------------------------|---------------------------------------------------|--------------------------------------------------------------------|
| \_Максимальное число \_ одновременных \_ рендертаржетс D3D | 4                                                 | Максимальное число рендертаржетс.                               |
| D3DDMAPSAMPLER                        | 256                                               | Максимальное число выборок карт смещения.                    |
| D3DDP \_ макстекскурд                    | 8                                                 | Максимальное число координат текстуры.                         |
| MAXD3DDECLLENGTH                      | 64 (не включает элемент вершины маркера "End") | Максимальное число элементов в объявлении вершины.                |
| MAXD3DDECLUSAGEINDEX                  | 15                                                | Максимальный индекс (0-15), который можно использовать в объявлении вершины. |



 

Эти \# определения объявляются в d3d9types. h.
