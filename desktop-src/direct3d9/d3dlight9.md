# <a name="d3dlight9-structure"></a>Структура D3DLIGHT9

Определяет набор свойств освещения.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct D3DLIGHT9 {
  D3DLIGHTTYPE  Type;
  D3DCOLORVALUE Diffuse;
  D3DCOLORVALUE Specular;
  D3DCOLORVALUE Ambient;
  D3DVECTOR     Position;
  D3DVECTOR     Direction;
  float         Range;
  float         Falloff;
  float         Attenuation0;
  float         Attenuation1;
  float         Attenuation2;
  float         Theta;
  float         Phi;
} D3DLIGHT9, *LPD3DLIGHT;
```



## <a name="members"></a>Члены

<dl> <dt>

**Тип**
</dt> <dd>

Тип: **[ **D3DLIGHTTYPE**](./d3dlighttype.md)**

</dd> <dd>

Тип источника освещения. Это значение является одним из элементов перечисляемого типа [**D3DLIGHTTYPE**](./d3dlighttype.md) .

</dd> <dt>

**Диффузное**
</dt> <dd>

Тип: **[ **D3DCOLORVALUE**](d3dcolorvalue.md)**

</dd> <dd>

Рассеянный цвет, выдаваемый источником света. Этот элемент является структурой [**D3DCOLORVALUE**](d3dcolorvalue.md) .

</dd> <dt>

**Отражающее**
</dt> <dd>

Тип: **[ **D3DCOLORVALUE**](d3dcolorvalue.md)**

</dd> <dd>

Отражающий цвет, порожденный светом. Этот элемент является структурой [**D3DCOLORVALUE**](d3dcolorvalue.md) .

</dd> <dt>

**Окружающее**
</dt> <dd>

Тип: **[ **D3DCOLORVALUE**](d3dcolorvalue.md)**

</dd> <dd>

Окружающий цвет, выдаваемый источником освещения. Этот элемент является структурой [**D3DCOLORVALUE**](d3dcolorvalue.md) .

</dd> <dt>

**Позиция**
</dt> <dd>

Тип: **[ **D3DVECTOR**](d3dvector.md)**

</dd> <dd>

Расположение света в мировом пространстве, определяемое структурой [**D3DVECTOR**](d3dvector.md) . Этот член не имеет смысла для направленного освещения и в этом случае игнорируется.

</dd> <dt>

**Направление**
</dt> <dd>

Тип: **[ **D3DVECTOR**](d3dvector.md)**

</dd> <dd>

Направление, указывающее, что источник света указывает на мировое пространство, заданное структурой [**D3DVECTOR**](d3dvector.md) . Этот элемент имеет значение только для направленного и прожектора. Этот вектор не требует нормализации, но он должен иметь ненулевую длину.

</dd> <dt>

**Диапазон**
</dt> <dd>

Тип: **float**

</dd> <dd>

Расстояние, за пределами которого освещение не имеет силы. Максимальное допустимое значение для этого элемента — квадратный корень из ФЛТ \_ Max. Этот элемент не влияет на направленный свет.

</dd> <dt>

**Переходят**
</dt> <dd>

Тип: **float**

</dd> <dd>

Уменьшите освещения между внутренним конусом прожектора (угол, заданный параметром тета) и внешним краями внешнего конуса (угол, заданный параметром фи).

Воздействие спада на освещение незаметно. Более того, при формировании кривой рассеивания начисляются небольшие потери производительности. По этим причинам большинство разработчиков устанавливают это значение равным 1,0.

</dd> <dt>

**Attenuation0**
</dt> <dd>

Тип: **float**

</dd> <dd>

Значение, указывающее, как интенсивность освещения изменяется по расстоянию. Значения затухания игнорируются для направленного освещения. Этот элемент представляет константу затухания. Сведения о ослаблении см. в разделе [Свойства освещения (Direct3D 9)](light-properties.md). Допустимые значения для этого элемента в диапазоне от 0,0 до бесконечности. Для ненаправленного освещения все три значения затухания не должны быть установлены равными 0,0 одновременно.

</dd> <dt>

**Attenuation1**
</dt> <dd>

Тип: **float**

</dd> <dd>

Значение, указывающее, как интенсивность освещения изменяется по расстоянию. Значения затухания игнорируются для направленного освещения. Этот элемент представляет константу затухания. Сведения о ослаблении см. в разделе [Свойства освещения (Direct3D 9)](light-properties.md). Допустимые значения для этого элемента в диапазоне от 0,0 до бесконечности. Для ненаправленного освещения все три значения затухания не должны быть установлены равными 0,0 одновременно.

</dd> <dt>

**Attenuation2**
</dt> <dd>

Тип: **float**

</dd> <dd>

Значение, указывающее, как интенсивность освещения изменяется по расстоянию. Значения затухания игнорируются для направленного освещения. Этот элемент представляет константу затухания. Сведения о ослаблении см. в разделе [Свойства освещения (Direct3D 9)](light-properties.md). Допустимые значения для этого элемента в диапазоне от 0,0 до бесконечности. Для ненаправленного освещения все три значения затухания не должны быть установлены равными 0,0 одновременно.

</dd> <dt>

**Тета-**
</dt> <dd>

Тип: **float**

</dd> <dd>

Угол в радианах для внутреннего конуса прожектора, то есть полностью освещенный прожектор. Это значение должно находиться в диапазоне от 0 до значения, заданного параметром фи.

</dd> <dt>

**Фи**
</dt> <dd>

Тип: **float**

</dd> <dd>

Угол в радианах, определяющий внешнее ребро внешнего конуса прожектора. Точки за пределами этой конуса не освещены в центре внимания. Это значение должно находиться в диапазоне от 0 до PI.

</dd> </dl>
