# <a name="states-direct3d-9"></a>Состояния (Direct3D 9)

Устройство Direct3D использует состояния для настройки всей конвейерной обработки. Состояния образцов управляют настройкой образцов текстур путем определения адресации текстуры и режимов фильтрации. Состояние текстуры. Настройка многотекстурного смешения, определяющая, как текстуры объединяются и смешиваются друг с другом. Состояния отрисовки управляют большей частью состояния конвейера.

Блоки состояний — это удобный механизм для сохранения и восстановления состояния. См. раздел [State Blocks Save and Restore State (Direct3D 9)](state-blocks-save-and-restore-state.md)
