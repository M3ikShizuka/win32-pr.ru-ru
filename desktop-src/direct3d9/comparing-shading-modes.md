# <a name="comparing-shading-modes-direct3d-9"></a>Сравнение режимов заливки (Direct3D 9)

В режиме плоской заливки Следующая пирамида отображается с четким ребром между соседними лицами. Однако в режиме заливки Гуро значения заливки интерполируются на границе, а окончательный вид — на изогнутой поверхности.

![Иллюстрация пирамиды с четкими краями и стрелками, указывающими на нормальную поверхность](images/shade2.png)

Заливка по методу Гуро охватывает плоские поверхности более реалистично, чем Плоская заливка. Грань в режиме плоской заливки — это однородный цвет, но заливка по методу Гуро позволяет более правильно попадать в сторону. Этот результат особенно заметен, если имеется источник точки поблизости.

Заливка по методу Гуро сглаживает четкие края многоугольников, которые видны с плоской заливкой. Однако это может привести к созданию диапазонов машинного типа, которые являются полосами цветов или освещения, которые не смешиваются плавно в смежных многоугольниках. Приложение может уменьшить внешний вид диапазонов, увеличивая количество многоугольников в объекте, увеличивая разрешение экрана или увеличивая глубину цвета приложения.

Заливка по методу Гуро может пропускать некоторые сведения. Например, на следующем рисунке фокус полностью содержится в многоугольнике.

![Иллюстрация прожектора в пределах многоугольника](images/gouraud.png)

В этом случае заливка по методу Гуро, которая выполняет интерполяцию между вершинами, полностью пропустила прожектор; грань будет отображаться, как будто прожектор не существовал.
