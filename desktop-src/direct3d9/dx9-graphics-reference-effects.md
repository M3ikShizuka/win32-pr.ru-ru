# <a name="effect-reference-direct3d-9-graphics"></a>Справочник по действиям (графика Direct3D 9)

Эффекты представляют собой абстракцию более высокого уровня, которая использует преимущества шейдеров вершин и пикселей для инкапсуляции эффектов отрисовки. Эффекты могут быть написаны на высоком уровне или с помощью языка ассемблера шейдера.

Ниже приведены статьи по API.

-   [Интерфейсы эффектов](dx9-graphics-reference-effects-interfaces.md)
-   [Функции эффектов](dx9-graphics-reference-effects-functions.md)
-   [Структуры эффектов](dx9-graphics-reference-effects-structures.md)
-   [Константы эффектов](dx9-graphics-reference-effects-constants.md)
-   [Формат эффектов (Direct3D 9)](dx9-graphics-reference-effects-file-format.md)
-   [Справочник по стандарту DirectX для заметок и семантик](dx9-graphics-reference-effects-dxsas.md)

В качестве альтернативы использованию API для создания эффектов можно также использовать средство командной строки Compile. См. раздел [средство компилятора Effect](../direct3dtools/fxc.md).
