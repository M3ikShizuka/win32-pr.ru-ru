# <a name="fog-types-direct3d-9"></a>Типы тумана (Direct3D 9)

Существует два способа добавления тумана в сцену: с пиксельным туманом и (или) вершиной тумана. В следующих разделах показаны формулы, используемые в уравнениях тумана, а также реализация вершин и пиксельных туманов. Приложение может реализовать туман с шейдером вершин и одновременно при необходимости пиксельный туман.

-   [Формулы тумана](fog-formulas.md)
-   [Параметры тумана](fog-parameters.md)
-   [Смешение тумана](fog-blending.md)
-   [Цвет тумана](fog-color.md)
-   [Вершинный туман](vertex-fog.md)
-   [Пиксельный туман](pixel-fog.md)

Смешение тумана управляется состояниями отрисовки; Он не является частью программируемого конвейера пикселей.
