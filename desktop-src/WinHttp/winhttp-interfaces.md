# <a name="winhttp-interfaces"></a>Интерфейсы WinHTTP

[Службы Microsoft Windows HTTP Services (WinHTTP)](about-winhttp.md) предоставляют следующие два интерфейса:

-   [**Ивинхттпрекуест**](iwinhttprequest-interface.md) реализует методы WinHTTP, не затрагивающие события.
-   [**Ивинхттпрекуестевентс**](iwinhttprequestevents-interface.md) реализует события WinHTTP и методы на основе событий.
