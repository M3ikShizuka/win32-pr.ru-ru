# <a name="using-the-winhttp-cc-api"></a>Использование API WinHTTP C/C++

В этом разделе содержатся сведения, относящиеся к использованию интерфейса C/C++ WinHTTP.

-   [Сеансы WinHTTP](winhttp-sessions-overview.md) описывают базовый поток приложения WinHTTP, включая порядок вызова функций.
-   [Дескрипторы хинтернет в WinHTTP](hinternet-handles-in-winhttp.md) описывают различные типы дескрипторов, соглашения об именовании для этих дескрипторов и их иерархическую структуру.
-   [Универсальные указатели ресурсов (URL-адреса) в WinHTTP](uniform-resource-locators--urls--in-winhttp.md) описывают, как WinHTTP использует URL-адреса, и предоставляет пример кода для демонстрации использования предоставленных функций.
-   [Проверка подлинности в WinHTTP](authentication-in-winhttp.md) описывает поддержку WinHTTP для проверки подлинности сеансов HTTP на сервере и прокси-сервер.
-   Проверка подлинности [Passport в WinHTTP](passport-authentication-in-winhttp.md) описывает поддержку WinHTTP для проверки подлинности Passport 1,4.
-   Протокол [SSL в WinHTTP](ssl-in-winhttp.md) описывает поддержку WinHTTP для протокола SSL (SSL).
-   [При использовании WinHTTP в качестве параллельной сборки](using-winhttp-as-a-side-by-side-assembly.md) отображается пример манифеста приложения, который можно использовать для связи с WinHTTP.
-   [Обработка файлов cookie в WinHTTP](cookie-handling-in-winhttp.md) описывает несколько параметров, которые клиентские приложения WinHTTP могут использовать для обработки файлов cookie.
-   [Обработка ошибок в WinHTTP](error-handling-in-winhttp.md) описывает, как настроить параметры WinHTTP программным способом из приложения C/C++.
-   [Поддержка автопрокси WinHTTP](winhttp-autoproxy-support.md) описывает различные способы, с помощью которых функции API WinHTTP сообщают об ошибках.
