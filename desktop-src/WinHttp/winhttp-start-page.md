# <a name="windows-http-services"></a>Windows Службы HTTP

## <a name="purpose"></a>Назначение

службы Microsoft Windows HTTP (WinHTTP) предоставляют разработчикам интерфейс прикладного программирования (API) http для отправки запросов по протоколу http другим серверам http.

## <a name="where-applicable"></a>Где применимо

WinHTTP поддерживает настольные клиентские приложения, Windowsные службы и Windows серверные приложения.

дополнительные сведения об использовании WinHTTP для приложений, построенных на платформа .NET Framework майкрософт, см. в разделе [API винхттфандлер](/dotnet/api/system.net.http?view=netframework-4.8) .

## <a name="developer-audience"></a>Аудитория разработчиков

WinHTTP предлагает как интерфейс прикладного программирования (API) C/C++, так и компонент автоматизации модели COM, который подходит для использования в приложениях на основе Active Server страниц (ASP).

Основное понимание протокола HTTP важно для использования любого интерфейса.

## <a name="run-time-requirements"></a>Требования к среде выполнения

WinHTTP 5,1 предлагает улучшения по сравнению с версией 5,0. Он включен в операционную систему. дополнительные сведения о новых возможностях см. в статьях [что нового в WinHTTP 5,1](what-s-new-in-winhttp-5-1.md) и [что нового в Windows Server 2008 и Windows Vista](what-s-new-in-windows-longhorn.md).

> [!IMPORTANT]
> Скачивание WinHTTP 5,0 больше не доступно. Начиная с 1 октября 2004 г. Корпорация Майкрософт удалила пакет SDK для WinHTTP 5,0 с сайта MSDN и прекратил поддержку продукта для версии 5,0.
