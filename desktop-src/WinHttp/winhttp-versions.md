# <a name="winhttp-versions"></a>Версии WinHTTP

Пользователи служб Microsoft Windows HTTP Services (WinHTTP) должны использовать последнюю версию технологии версии 5,1. Версия 5,0 больше не поддерживается.

## <a name="version-51"></a>Версия 5.1

WinHTTP 5,1 предлагает улучшения по сравнению с версией 5,0; Дополнительные сведения о новых возможностях см. в статье новые возможности [в WinHTTP 5,1](what-s-new-in-winhttp-5-1.md).

В версии 5,1 служба WinHTTP является компонентом операционной системы следующих операционных систем:

-   Windows 2000, пакет обновления 3 (SP3) и более поздние версии (кроме сервера Datacenter)
-   Windows XP с пакетом обновления 1 (SP1) и более поздние версии
-   Windows Server 2003 с пакетом обновления 1 (SP1) и более поздние версии

Для Windows Server 2003 с пакетом обновления 1 (SP1) WinHTTP — это системная параллельная сборка. Дополнительные сведения см. в разделе [Параллельные сборки](/windows/desktop/SbsCs/about-side-by-side-assemblies-).

Повторное распространение WinHTTP 5,1 за пределы этих операционных систем не поддерживается.

## <a name="version-50"></a>Версия 5.0

Распространяемый компонент WinHTTP 5,0, "winhttp5.dll", работает в Windows 2000 или более поздней версии или Windows NT 4,0 с Microsoft Internet Explorer 5,01 или более поздней версии.

> [!IMPORTANT]
> Скачивание WinHTTP 5,0 больше не доступно. Начиная с 1 октября 2004 г. Корпорация Майкрософт удалила пакет SDK для WinHTTP 5,0 с сайта MSDN и прекратил поддержку продукта для версии 5,0.
