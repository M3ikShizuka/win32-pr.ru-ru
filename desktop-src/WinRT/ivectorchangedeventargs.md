# <a name="ivectorchangedeventargs-interface"></a>Интерфейс Ивекторчанжедевентаргс

Предоставляет данные для события [**векторчанжед**](/uwp/api/windows.foundation.collections.iobservablevector-1?view=winrt-19041) .

## <a name="members"></a>Элементы

Интерфейс **ивекторчанжедевентаргс** наследует от **IInspectable**. **Ивекторчанжедевентаргс** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивекторчанжедевентаргс** содержит следующие методы.



| Метод                                                                        | Описание                                                           |
|:------------------------------------------------------------------------------|:----------------------------------------------------------------------|
| [**получить \_ коллектиончанже**](ivectorchangedeventargs-get-collectionchange.md) | Возвращает тип изменения, произошедшего в векторе.<br/>       |
| [**получить \_ индекс**](ivectorchangedeventargs-get-index.md)                       | Возвращает позицию в векторе, где произошло изменение.<br/> |
