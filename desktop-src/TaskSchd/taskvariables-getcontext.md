# <a name="taskvariablesgetcontext-method"></a>Тасквариаблес. метод Context

Для создания скриптов используется для совместного использования контекста разными шагами и задачами, которые находятся в одном и том же экземпляре задания. Этот метод не реализован.

## <a name="syntax"></a>Синтаксис


```VB
TaskVariables.GetContext( _
  ByRef context _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*контекст* \[ заполняет\]
</dt> <dd>

Контекст, используемый для совместного использования контекста разными шагами и задачами, которые находятся в одном и том же экземпляре задания.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
