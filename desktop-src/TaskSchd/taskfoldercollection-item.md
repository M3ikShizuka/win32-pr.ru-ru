# <a name="taskfoldercollectionitem-property"></a>Таскфолдерколлектион. Item, свойство

Для сценариев возвращает указанную папку из коллекции.

## <a name="syntax"></a>Синтаксис


```VB
TaskFolderCollection.Item( _
  ByVal Index _
) As TaskFolder
```



## <a name="property-value"></a>Значение свойства

Объект [**таскфолдер**](taskfolder.md) , представляющий запрашиваемую папку.
