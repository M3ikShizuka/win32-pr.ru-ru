# <a name="actioncollectioncontext-property"></a>ActionCollection. Context, свойство

Для создания скриптов Возвращает или задает идентификатор участника для задачи.

## <a name="syntax"></a>Синтаксис


```VB
ActionCollection.Context As String
```



## <a name="property-value"></a>Значение свойства

Идентификатор участника для задачи. Указанный здесь идентификатор должен соответствовать идентификатору, указанному в свойстве ID интерфейса IPrincipal, определяющего участника.
