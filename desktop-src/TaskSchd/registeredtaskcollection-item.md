# <a name="registeredtaskcollectionitem-property"></a>Регистередтаскколлектион. Item, свойство

Для создания скриптов возвращает указанную зарегистрированную задачу из коллекции.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
RegisteredTaskCollection.Item( _
  ByVal Index _
) As RegisteredTask
```



## <a name="property-value"></a>Значение свойства

Объект [**регистередтаск**](registeredtask.md) , содержащий зарегистрированную задачу.
