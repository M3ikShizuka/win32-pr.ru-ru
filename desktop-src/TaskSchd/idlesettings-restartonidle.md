# <a name="idlesettingsrestartonidle-property"></a>Идлесеттингс. Рестартонидле, свойство

Для создания скриптов Возвращает или задает логическое значение, указывающее, перезапускается ли задача, когда компьютер циклически переключается в состояние простоя.

## <a name="syntax"></a>Синтаксис


```VB
IdleSettings.RestartOnIdle As Boolean
```



## <a name="property-value"></a>Значение свойства

Логическое значение, указывающее, должна ли задача перезапускаться, когда компьютер циклически перейдет в состояние простоя. Значение по умолчанию — False.

## <a name="remarks"></a>Remarks

Это свойство используется только в том случае, если свойство [**идлесеттингс. стопонидлинд**](idlesettings-stoponidleend.md) имеет значение true.

При чтении или записи XML для задачи этот параметр указывается в элементе [**рестартонидле**](taskschedulerschema-restartonidle-idlesettingstype-element.md) схемы планировщик задач.
