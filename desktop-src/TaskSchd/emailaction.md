# <a name="emailaction-object"></a>Объект EmailAction

\[Этот объект больше не поддерживается. Используйте Иексекактион с командлетом PowerShell [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) в качестве обходного пути.\]

## <a name="remarks"></a>Remarks

Действие электронной почты должно иметь допустимое значение для свойств [**сервера**](emailaction-server.md), [**из**](emailaction-from.md), и [**to**](emailaction-to.md) или [**CC**](emailaction-cc.md) , чтобы задача была зарегистрирована и выполнена правильно.

При чтении или записи собственного XML-кода для задачи действие электронной почты задается с помощью элемента [**SendEmail**](taskschedulerschema-sendemail-actiongroup-element.md) схемы планировщик задач.
