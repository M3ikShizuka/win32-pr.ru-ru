# <a name="registeredtaskgetinstances-method"></a>Метод Регистередтаск. WebMethod

Для сценариев возвращает все текущие выполняющиеся экземпляры зарегистрированной задачи.

> [!Note]  
> **Регистередтаск.** -экземпляры будут возвращать только экземпляры текущей зарегистрированной задачи, которые выполняются в контексте безопасности пользователя или ниже. Например, для членов группы "Администраторы" будут возвращены все экземпляры текущей зарегистрированной задачи, но члены группы "Пользователи **" будут** возвращать только экземпляры текущей зарегистрированной **задачи, которые** выполняются в контексте безопасности группы "Пользователи".

 

## <a name="syntax"></a>Синтаксис


```VB
RegisteredTask.GetInstances( _
  ByVal flags, _
  ByRef runningTasks _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*flags* 
</dt> <dd>

Этот параметр зарезервирован для будущего использования и должен иметь значение 0.

</dd> <dt>

*руннингтаскс* \[ заполняет\]
</dt> <dd>

Объект [**руннингтаскколлектион**](runningtaskcollection.md) , который содержит все текущие выполняющиеся экземпляры задачи.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
