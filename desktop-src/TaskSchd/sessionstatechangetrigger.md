# <a name="sessionstatechangetrigger-object"></a>Объект Сессионстатечанжетригжер

Объект скрипта, который активирует задачи для подключения консоли или отключения, удаленного подключения или отключения, а также для блокировки или разблокировки рабочей станцией.

## <a name="members"></a>Элементы

Объект **сессионстатечанжетригжер** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Объект **сессионстатечанжетригжер** имеет следующие свойства.



| Свойство                                                                | Тип доступа           | Описание                                                                                                                                                                                               |
|:------------------------------------------------------------------------|:----------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Задержка**](sessionstatechangetrigger-delay.md)<br/>             | Чтение/запись<br/> | Возвращает или задает значение, указывающее, как долго происходит задержка перед запуском задачи после обнаружения изменения состояния сеанса сервера терминалов.<br/>                                         |
| [**Активировано**](/windows/desktop/api/taskschd/nf-taskschd-itrigger-get_enabled)<br/>                          | Чтение/запись<br/> | Наследуется от объекта [**триггера**](trigger.md) . Возвращает или задает логическое значение, указывающее, включен ли триггер.<br/>                                                              |
| [**ендбаундари**](/windows/desktop/api/taskschd/nf-taskschd-itrigger-get_endboundary)<br/>                  | Чтение/запись<br/> | Наследуется от объекта [**триггера**](trigger.md) . Возвращает или задает дату и время деактивации триггера. Триггер не может запустить задачу после ее деактивации.<br/>               |
| [**ексекутионтимелимит**](/windows/desktop/api/taskschd/nf-taskschd-itrigger-get_executiontimelimit)<br/>    | Чтение/запись<br/> | Наследуется от объекта [**триггера**](trigger.md) . Возвращает или задает максимальное время, в течение которого задача может запускаться триггером.<br/>                                                 |
| [**Удостоверения**](/windows/desktop/api/taskschd/nf-taskschd-itrigger-get_id)<br/>                                    | Чтение/запись<br/> | Наследуется от объекта [**триггера**](trigger.md) . Возвращает или задает идентификатор триггера.<br/>                                                                                             |
| [**Повторение**](/windows/desktop/api/taskschd/nf-taskschd-itrigger-get_repetition)<br/>                    | Чтение/запись<br/> | Наследуется от объекта [**триггера**](trigger.md) . Возвращает или задает значение, указывающее, как часто выполняется задача и как долго шаблон повторения повторяется после запуска задачи.<br/> |
| [**стартбаундари**](/windows/desktop/api/taskschd/nf-taskschd-itrigger-get_startboundary)<br/>              | Чтение/запись<br/> | Наследуется от объекта [**триггера**](trigger.md) . Возвращает или задает дату и время активации триггера.<br/>                                                                            |
| [**StateChange**](sessionstatechangetrigger-statechange.md)<br/> | Чтение/запись<br/> | Возвращает или задает тип изменения сеанса сервера терминалов, запускающего запуск задачи.<br/>                                                                                                      |
| [**Тип**](/windows/desktop/api/taskschd/nf-taskschd-itrigger-get_type)<br/>                                | Только для чтения<br/>  | Наследуется от объекта [**триггера**](trigger.md) . Возвращает тип триггера.<br/>                                                                                                            |
| [**UserId**](sessionstatechangetrigger-userid.md)<br/>           | Чтение/запись<br/> | Возвращает или задает пользователя для сеанса сервера терминалов. При обнаружении изменения состояния сеанса для этого пользователя запускается задача.<br/>                                                               |



 

## <a name="remarks"></a>Комментарии

При чтении или записи собственного XML-кода для задачи триггер изменения состояния сеанса задается с помощью элемента [**сессионстатечанжетригжер**](taskschedulerschema-sessionstatechangetrigger-triggergroup-element.md) схемы планировщик задач.
