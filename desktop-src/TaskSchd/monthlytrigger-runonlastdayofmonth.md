# <a name="monthlytriggerrunonlastdayofmonth-property"></a>Монслитригжер. Рунонластдайофмонс, свойство

Для создания скриптов Возвращает или задает логическое значение, указывающее, что задача выполняется в последний день месяца.

## <a name="syntax"></a>Синтаксис


```VB
MonthlyTrigger.RunOnLastDayOfMonth As Boolean
```



## <a name="property-value"></a>Значение свойства

Значение true указывает, что задача выполняется в последний день месяца независимо от фактической даты этого дня; в противном случае — значение false.
