# <a name="showmessageaction-object"></a>Объект Шовмессажеактион

\[Этот объект больше не поддерживается. для отображения сообщения в сеансе пользователя можно использовать иексекактион с функцией Windows scripting [**MsgBox**](/previous-versions/sfw6660x(v=vs.80)) .\]

Для сценариев представляет действие, которое отображает окно сообщения при активации задачи.

## <a name="members"></a>Элементы

Объект **шовмессажеактион** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Элемент Property

Объект **шовмессажеактион** имеет следующие свойства.



| Свойство                                                        | Тип доступа           | Описание                                                                                               |
|:----------------------------------------------------------------|:----------------------|:----------------------------------------------------------------------------------------------------------|
| [**Удостоверения**](action-id.md)<br/>                              | Чтение/запись<br/> | Наследуется от объекта [**Action**](action.md) . Возвращает или задает идентификатор действия.<br/> |
| [**MessageBody**](showmessageaction-messagebody.md)<br/> | Чтение/запись<br/> | Возвращает или задает текст сообщения, отображаемого в тексте окна сообщения.<br/>                |
| [**Название**](showmessageaction-title.md)<br/>             | Чтение/запись<br/> | Возвращает или задает заголовок окна сообщения.<br/>                                                     |
| [**Тип**](action-type.md)<br/>                          | Только для чтения<br/>  | Наследуется от объекта [**Action**](action.md) . Возвращает тип действия.<br/>               |



 

## <a name="remarks"></a>Remarks

Для задачи, которая содержит действие «окно сообщения», появится окно сообщения, если задача активирована, а задача имеет интерактивный тип входа в систему. Чтобы установить для параметра Тип входа в интерактивный режим, укажите значение 3 (**\_ \_ Интерактивный \_ маркер входа в систему**) или 4 (**\_ \_ Группа входа** в систему) в свойстве [**LogonType**](principal-logontype.md) участника задачи или в параметре *LogonType* [**таскфолдер. регистертаск**](taskfolder-registertask.md) или [**таскфолдер. регистертаскдефинитион**](taskfolder-registertaskdefinition.md).

При чтении или записи собственного XML-кода для задачи действие окна сообщения указывается с помощью элемента [**ShowMessage**](taskschedulerschema-showmessage-actiongroup-element.md) схемы планировщик задач.
