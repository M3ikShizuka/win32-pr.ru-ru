# <a name="tasknamedvaluecollectioncount-property"></a>Таскнамедвалуеколлектион. Count, свойство

Для сценариев возвращает число пар "имя-значение" в коллекции.

## <a name="syntax"></a>Синтаксис


```VB
TaskNamedValueCollection.Count As long
```



## <a name="property-value"></a>Значение свойства

Число пар "имя-значение" в коллекции.
