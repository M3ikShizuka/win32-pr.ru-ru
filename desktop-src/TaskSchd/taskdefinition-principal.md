# <a name="taskdefinitionprincipal-property"></a>Таскдефинитион. Principal, свойство

Для создания скриптов Возвращает или задает участника задачи, предоставляющей учетные данные безопасности для задачи.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
TaskDefinition.Principal As Principal
```



## <a name="property-value"></a>Значение свойства

Участник задачи, предоставляющий учетные данные безопасности для задачи.
