# <a name="runningtaskcollection-object"></a>Объект Руннингтаскколлектион

Объект скрипта, предоставляющий коллекцию, которая используется для управления выполняемыми задачами.

## <a name="members"></a>Элементы

Объект **руннингтаскколлектион** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Элемент Property

Объект **руннингтаскколлектион** имеет следующие свойства.



| Свойство.                                                | Описание                                                    |
|:--------------------------------------------------------|:---------------------------------------------------------------|
| [**Count**](runningtaskcollection-count.md)<br/> | Возвращает число выполняемых задач в коллекции.<br/> |
| [**Элемент**](runningtaskcollection-item.md)<br/>   | Возвращает указанную задачу из коллекции. <br/>       |
