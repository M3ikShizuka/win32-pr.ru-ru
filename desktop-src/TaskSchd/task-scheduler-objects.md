# <a name="task-scheduler-scripting-objects"></a>планировщик задач объектов скрипта

Объекты сценариев, описанные в следующих разделах, предоставляют программный доступ к функциональным возможностям, доступным в планировщик задач для Visual Basic и Visual Basic разработчиков скриптов.


В планировщик задач 2,0 представлены следующие объекты.



| Объект                                                         | Описание                                                                                                                                                                                                                                           |
|----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Действие**](action.md)                                       | Предоставляет общие свойства, наследуемые всеми объектами действий.                                                                                                                                                                              |
| [**ActionCollection**](actioncollection.md)                   | Содержит действия, выполняемые задачей.                                                                                                                                                                                                           |
| [**буттригжер**](boottrigger.md)                             | Представляет триггер, который запускает задачу при загрузке системы.                                                                                                                                                                                    |
| [**комхандлерактион**](comhandleraction.md)                   | Представляет действие, которое вызывает обработчик.                                                                                                                                                                                                            |
| [**даилитригжер**](dailytrigger.md)                           | Представляет триггер, который запускает задачу на основе ежедневного расписания.                                                                                                                                                                                    |
| [**емаилактион**](emailaction.md)                             | Представляет действие, которое отправляет сообщение электронной почты.                                                                                                                                                                                                     |
| [**EventTrigger**](eventtrigger.md)                           | Представляет триггер, который запускает задачу при возникновении системного события.                                                                                                                                                                                   |
| [**ексекактион**](execaction.md)                               | Представляет действие, выполняющее операцию командной строки.                                                                                                                                                                                          |
| [**идлесеттингс**](idlesettings.md)                           | Указывает, как планировщик задач выполняет задачи, когда компьютер находится в состоянии простоя.                                                                                                                                                            |
| [**идлетригжер**](idletrigger.md)                             | Представляет триггер, который запускает задачу при возникновении условия простоя.                                                                                                                                                                                |
| [**логонтригжер**](logontrigger.md)                           | Представляет триггер, который запускает задачу при входе пользователя в систему.                                                                                                                                                                                          |
| [**монслидовтригжер**](monthlydowtrigger.md)                 | Представляет триггер, который запускает задачу в расписании ежемесячного дня недели.                                                                                                                                                                            |
| [**монслитригжер**](monthlytrigger.md)                       | Представляет триггер, который запускает задачу на основе ежемесячного расписания.                                                                                                                                                                                  |
| [**NetworkSettings**](networksettings.md)                     | Предоставляет параметры, которые служба планировщик задач использует для получения сетевого профиля.                                                                                                                                                               |
| [**Основного**](principal.md)                                 | Предоставляет учетные данные безопасности для участника.                                                                                                                                                                                                    |
| [**регистередтаск**](registeredtask.md)                       | Предоставляет методы, которые используются для немедленного выполнения задачи, получения всех запущенных экземпляров задачи, получения или задания учетных данных, используемых для регистрации задачи, а также свойств, описывающих задачу.                                      |
| [**регистередтаскколлектион**](registeredtaskcollection.md)   | Содержит все зарегистрированные задачи.                                                                                                                                                                                                           |
| [**регистратионинфо**](registrationinfo.md)                   | Предоставляет административные сведения, которые можно использовать для описания задачи. Эти сведения включают в себя такие сведения, как описание задачи, автора задачи, Дата регистрации задачи и дескриптор безопасности задачи. |
| [**регистратионтригжер**](registrationtrigger.md)             | Представляет триггер, который запускает задачу при регистрации задачи.                                                                                                                                                                                  |
| [**репетитионпаттерн**](repetitionpattern.md)                 | Определяет, как часто выполняется задача и как долго шаблон повторения повторяется после запуска задачи.                                                                                                                                          |
| [**руннингтаск**](runningtask.md)                             | Предоставляет методы для получения информации из выполняемой задачи и управления ей.                                                                                                                                                                              |
| [**руннингтаскколлектион**](runningtaskcollection.md)         | Используется для получения выполняемой задачи.                                                                                                                                                                                                                      |
| [**сессионстатечанжетригжер**](sessionstatechangetrigger.md) | Используется для активации задач подключения консоли или отключения, удаленного подключения или отключения, а также для блокировки или разблокировки рабочей станцией.                                                                                                                   |
| [**шовмессажеактион**](showmessageaction.md)                 | Представляет действие, которое отображает окно сообщения при активации задачи.                                                                                                                                                                               |
| [**таскдефинитион**](taskdefinition.md)                       | Определяет все компоненты задачи, такие как параметры задачи, триггеры, действия и сведения о регистрации.                                                                                                                                     |
| [**таскфолдер**](taskfolder.md)                               | Предоставляет методы, используемые для регистрации (создания) задач в папке, удаления задач из папки и создания или удаления вложенных папок из папки.                                                                                           |
| [**таскфолдерколлектион**](taskfoldercollection.md)           | Подсчитывает количество папок в коллекции и извлекает указанную папку из коллекции.                                                                                                                                                   |
| [**таскнамедвалуепаир**](tasknamedvaluepair.md)               | Создает пару "имя-значение", в которой имя связано со значением.                                                                                                                                                                             |
| [**таскнамедвалуеколлектион**](tasknamedvaluecollection.md)   | Содержит коллекцию пар "имя-значение" для объекта [**таскнамедвалуепаир**](tasknamedvaluepair.md) .                                                                                                                                                    |
| [**TaskService**](taskservice.md)                             | Предоставляет доступ к службе планировщик задач для управления зарегистрированными задачами.                                                                                                                                                                          |
| [**тасксеттингс**](tasksettings.md)                           | Предоставляет параметры, используемые службами планировщик задач для выполнения задачи.                                                                                                                                                                      |
| [**тасквариаблес**](taskvariables.md)                         | Определяет переменные задачи, которые могут быть переданы в качестве параметров обработчикам задач и внешним исполняемым файлам, запускаемым задачами.                                                                                                                         |
| [**TimeTrigger**](timetrigger.md)                             | Представляет триггер, который запускает задачу при активации триггера.                                                                                                                                                                                |
| [**Необходимость**](trigger.md)                                     | Предоставляет общие свойства, наследуемые всеми объектами триггера.                                                                                                                                                                             |
| [**тригжерколлектион**](triggercollection.md)                 | Используется для добавления, удаления из и получения триггеров задачи.                                                                                                                                                                                     |
| [**виклитригжер**](weeklytrigger.md)                         | Представляет триггер, который запускает задачу на основе еженедельного расписания.                                                                                                                                                                                   |
