# <a name="tasksettingscompatibility-property"></a>Тасксеттингс. Compatibility, свойство

Для создания скриптов Возвращает или задает целочисленное значение, указывающее версию планировщик задач, с которой совместима задача.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
TaskSettings.Compatibility As Integer
```



## <a name="property-value"></a>Значение свойства



| Значение                                                                                                                                                                                                                                         | Значение                                                    |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------|
| <span id="TASK_COMPATIBILITY_AT"></span><span id="task_compatibility_at"></span><dl> <dt>**Задача \_ СОВМЕСТИМОСТЬ \_ с**</dt> <dt>0</dt> </dl> | Задача совместима с командой AT.<br/>     |
| <span id="TASK_COMPATIBILITY_V1"></span><span id="task_compatibility_v1"></span><dl> <dt>**Задача \_ СОВМЕСТИМОСТЬ \_ v1**</dt> <dt>1</dt> </dl> | Задача совместима с планировщик задач 1,0.<br/> |
| <span id="TASK_COMPATIBILITY_V2"></span><span id="task_compatibility_v2"></span><dl> <dt>**Задача \_ СОВМЕСТИМОСТЬ \_ v2**</dt> <dt>2</dt> </dl> | Задача совместима с планировщик задач 2,0.<br/> |



 

## <a name="remarks"></a>Комментарии

Совместимость задач, которая задается с помощью свойства [**Compatibility**](/windows/desktop/api/taskschd/nf-taskschd-itasksettings-get_compatibility) , должна быть установлена только для \_ обеспечения совместимости задач, если требуется \_ доступ к задаче или ее изменение с компьютера под управлением Windows XP, Windows Server 2003 или Windows 2000. В противном случае рекомендуется использовать совместимость с планировщик задач 2,0, так как задача будет иметь больше функций.

Задачи, совместимые с командой AT, могут иметь только один триггер времени.

Задачи, совместимые с планировщик задач 1,0, могут иметь только триггер времени, триггер входа или загрузочный триггер, и задача может иметь только исполняемое действие.

Дополнительные сведения о совместимости задач см. [в разделе новые возможности планировщик задач](what-s-new-in-task-scheduler.md) и [задач](tasks.md).
