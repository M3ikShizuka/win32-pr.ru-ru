# <a name="boottrigger-object"></a>Объект Буттригжер

Объект **буттригжер** имеет следующие типы членов:

-   [Свойства](#properties)

## <a name="remarks"></a>Комментарии

Служба планировщик задач запускается при загрузке операционной системы, а задачи триггера загрузки настроены на запуск при запуске службы планировщик задач.

Только член группы "Администраторы" может создать задачу с триггером загрузки.

При создании собственного XML-кода для задачи триггер загрузки указывается с помощью элемента [**буттригжер**](taskschedulerschema-boottrigger-triggergroup-element.md) схемы планировщик задач.
