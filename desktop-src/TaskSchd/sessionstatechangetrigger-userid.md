# <a name="sessionstatechangetriggeruserid-property"></a>Сессионстатечанжетригжер. UserId, свойство

Для сценариев Возвращает или задает пользователя для сеанса сервера терминалов. При обнаружении изменения состояния сеанса для этого пользователя запускается задача.

## <a name="syntax"></a>Синтаксис


```VB
SessionStateChangeTrigger.UserId As String
```



## <a name="property-value"></a>Значение свойства

Пользователь для сеанса сервера терминалов.
