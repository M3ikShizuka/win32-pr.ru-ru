# <a name="networksettingsname-property"></a>NetworkSettings.Name, свойство

Для создания скриптов Возвращает или задает имя сетевого профиля. Имя используется в целях показа.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
NetworkSettings.Name As String
```



## <a name="property-value"></a>Значение свойства

Имя сетевого профиля.
