# <a name="task-scheduler-interfaces"></a>Интерфейсы планировщик задач

Интерфейсы, описанные в следующих разделах, предоставляют программный доступ к функциональным возможностям, доступным в планировщик задач.

В этих разделах содержится описание интерфейса, список свойств и методов, определенных интерфейсом, и примечания о специальных обстоятельствах, которые следует отметить при использовании интерфейса.


В планировщик задач 2,0, который используется в операционной системе Windows Vista, представлены следующие интерфейсы.



| Интерфейс                                                        | Описание                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAction**](/windows/desktop/api/taskschd/nn-taskschd-iaction)                                       | Предоставляет общие свойства, наследуемые всеми объектами действий.                                                                                                                                                                                                                          |
| [**IActionCollection**](/windows/desktop/api/taskschd/nn-taskschd-iactioncollection)                   | Содержит действия, выполняемые задачей. Его методы можно использовать для добавления, удаления из и получения действий задачи.                                                                                                                                                          |
| [**ибуттригжер**](/windows/desktop/api/taskschd/nn-taskschd-iboottrigger)                             | Представляет триггер, который запускает задачу при запуске системы.                                                                                                                                                                                                                               |
| [**икомхандлерактион**](/windows/desktop/api/taskschd/nn-taskschd-icomhandleraction)                   | Представляет действие, которое вызывает обработчик.                                                                                                                                                                                                                                                        |
| [**идаилитригжер**](/windows/desktop/api/taskschd/nn-taskschd-idailytrigger)                           | Представляет триггер, который запускает задачу на основе ежедневного расписания.                                                                                                                                                                                                                                |
| [**иемаилактион**](/windows/desktop/api/taskschd/nn-taskschd-iemailaction)                             | Представляет действие, которое отправляет сообщение электронной почты.                                                                                                                                                                                                                                                 |
| [**иевенттригжер**](/windows/desktop/api/taskschd/nn-taskschd-ieventtrigger)                           | Представляет триггер, который запускает задачу при возникновении системного события.                                                                                                                                                                                                                               |
| [**иексекактион**](/windows/desktop/api/taskschd/nn-taskschd-iexecaction)                               | Представляет действие, выполняющее операцию командной строки.                                                                                                                                                                                                                                      |
| [**иидлесеттингс**](/windows/desktop/api/taskschd/nn-taskschd-iidlesettings)                           | Указывает, как планировщик задач выполняет задачи, когда компьютер находится в состоянии простоя.                                                                                                                                                                                                        |
| [**иидлетригжер**](/windows/win32/api/taskschd/nn-taskschd-iidletrigger)                             | Представляет триггер, который запускает задачу при переходе компьютера в состояние простоя.                                                                                                                                                                                                                |
| [**илогонтригжер**](/windows/desktop/api/taskschd/nn-taskschd-ilogontrigger)                           | Представляет триггер, который запускает задачу при входе пользователя в систему.                                                                                                                                                                                                                                      |
| [**имаинтенанцесеттингс**](/windows/desktop/api/taskschd/nn-taskschd-imaintenancesettings)             | Предоставляет параметры, используемые планировщик задач для выполнения задач во время автоматического обслуживания.                                                                                                                                                                                                  |
| [**имонслидовтригжер**](/windows/desktop/api/taskschd/nn-taskschd-imonthlydowtrigger)                 | Представляет триггер, который запускает задачу в расписании ежемесячного дня недели.                                                                                                                                                                                                                        |
| [**имонслитригжер**](/windows/desktop/api/taskschd/nn-taskschd-imonthlytrigger)                       | Представляет триггер, который запускает задачу на основе ежемесячного расписания.                                                                                                                                                                                                                              |
| [**инетворксеттингс**](/windows/desktop/api/taskschd/nn-taskschd-inetworksettings)                     | Предоставляет параметры, которые служба планировщик задач использует для получения сетевого профиля.                                                                                                                                                                                                           |
| [**IPrincipal**](/windows/desktop/api/taskschd/nn-taskschd-iprincipal)                                 | Предоставляет учетные данные безопасности для участника.                                                                                                                                                                                                                                                |
| [**IPrincipal2**](/windows/desktop/api/taskschd/nn-taskschd-iprincipal2)                               | Предоставляет дополнительные учетные данные безопасности для участника.                                                                                                                                                                                                                                         |
| [**ирегистередтаск**](/windows/desktop/api/taskschd/nn-taskschd-iregisteredtask)                       | Предоставляет методы, которые используются для немедленного выполнения задачи, получения всех запущенных экземпляров задачи, получения или задания учетных данных, используемых для регистрации задачи, а также свойств, описывающих задачу.                                                                                  |
| [**ирегистередтаскколлектион**](/windows/desktop/api/taskschd/nn-taskschd-iregisteredtaskcollection)   | Содержит все зарегистрированные задачи.                                                                                                                                                                                                                                                       |
| [**ирегистратионинфо**](/windows/desktop/api/taskschd/nn-taskschd-iregistrationinfo)                   | Предоставляет административные сведения, которые можно использовать для описания задачи. Эти сведения включают в себя такие сведения, как описание задачи, автора задачи, Дата регистрации задачи и дескриптор безопасности задачи.                                             |
| [**ирегистратионтригжер**](/windows/desktop/api/taskschd/nn-taskschd-iregistrationtrigger)             | Представляет триггер, который запускает задачу при регистрации задачи.                                                                                                                                                                                                                              |
| [**ирепетитионпаттерн**](/windows/desktop/api/taskschd/nn-taskschd-irepetitionpattern)                 | Определяет, как часто выполняется задача и как долго шаблон повторения повторяется после запуска задачи.                                                                                                                                                                                      |
| [**ируннингтаск**](/windows/desktop/api/taskschd/nn-taskschd-irunningtask)                             | Предоставляет методы для получения информации из выполняемой задачи и управления ей.                                                                                                                                                                                                                          |
| [**ируннингтаскколлектион**](/windows/desktop/api/taskschd/nn-taskschd-irunningtaskcollection)         | Предоставляет коллекцию, которая используется для управления выполняемыми задачами.                                                                                                                                                                                                                                      |
| [**исессионстатечанжетригжер**](/windows/desktop/api/taskschd/nn-taskschd-isessionstatechangetrigger) | Запускает задачи для подключения консоли или отключения, удаленного подключения или отключения, а также для блокировки или разблокировки рабочей станцией.                                                                                                                                                                      |
| [**ишовмессажеактион**](/windows/desktop/api/taskschd/nn-taskschd-ishowmessageaction)                 | Представляет действие, которое отображает окно сообщения при активации задачи.                                                                                                                                                                                                                           |
| [**итаскдефинитион**](/windows/desktop/api/taskschd/nn-taskschd-itaskdefinition)                       | Определяет все компоненты задачи, такие как параметры задачи, триггеры, действия и сведения о регистрации.                                                                                                                                                                                 |
| [**ITaskFolder**](/windows/desktop/api/taskschd/nn-taskschd-itaskfolder)                               | Предоставляет методы, используемые для регистрации (создания) задач в папке, удаления задач из папки и создания или удаления вложенных папок из папки.                                                                                                                                       |
| [**итаскфолдерколлектион**](/windows/desktop/api/taskschd/nn-taskschd-itaskfoldercollection)           | Предоставляет сведения и контроль над коллекцией папок, содержащих задачи.                                                                                                                                                                                                                  |
| [**ITaskHandler**](/windows/desktop/api/taskschd/nn-taskschd-itaskhandler)                             | Определяет методы, вызываемые службой планировщик задач для управления обработчиком COM.                                                                                                                                                                                                        |
| [**ITaskHandlerStatus**](/windows/desktop/api/taskschd/nn-taskschd-itaskhandlerstatus)                 | Предоставляет методы, используемые обработчиками COM для уведомления планировщик задач о состоянии обработчика.                                                                                                                                                                                  |
| [**итаскнамедвалуепаир**](/windows/desktop/api/taskschd/nn-taskschd-itasknamedvaluepair)               | Создает пару "имя-значение", в которой имя связано со значением.                                                                                                                                                                                                                         |
| [**итаскнамедвалуеколлектион**](/windows/desktop/api/taskschd/nn-taskschd-itasknamedvaluecollection)   | Содержит коллекцию пар "имя-значение" для интерфейса [**итаскнамедвалуепаир**](/windows/desktop/api/taskschd/nn-taskschd-itasknamedvaluepair) .                                                                                                                                                                                           |
| [**GetFolder интерфейса ITaskService**](/windows/desktop/api/taskschd/nn-taskschd-itaskservice)                             | Предоставляет доступ к службе планировщик задач для управления зарегистрированными задачами.                                                                                                                                                                                                                      |
| [**итасксеттингс**](/windows/desktop/api/taskschd/nn-taskschd-itasksettings)                           | Предоставляет параметры, которые служба планировщик задач использует для выполнения задачи.                                                                                                                                                                                                                   |
| [**ITaskSettings2**](/windows/desktop/api/taskschd/nn-taskschd-itasksettings2)                         | Предоставляет дополнительные параметры, которые планировщик задач использует для выполнения задачи.                                                                                                                                                                                                                    |
| [**итасквариаблес**](/windows/desktop/api/taskschd/nn-taskschd-itaskvariables)                         | Определяет переменные задачи, которые могут быть переданы в качестве параметров обработчикам задач и внешним исполняемым файлам, запускаемым задачами. Обработчики задач, которым необходимо вводить и выводить данные в переменные задания, должны выполнить интерфейс запроса в указателе на службы для [**итасквариаблес**](/windows/desktop/api/taskschd/nn-taskschd-itaskvariables). |
| [**итиметригжер**](/windows/desktop/api/taskschd/nn-taskschd-itimetrigger)                             | Представляет триггер, который запускает задачу при активации триггера.                                                                                                                                                                                                                            |
| [**итригжер**](/windows/desktop/api/taskschd/nn-taskschd-itrigger)                                     | Предоставляет общие свойства, наследуемые всеми интерфейсами триггеров.                                                                                                                                                                                                                      |
| [**ITriggerCollection**](/windows/desktop/api/taskschd/nn-taskschd-itriggercollection)                 | Предоставляет методы, используемые для добавления триггеров задачи и их удаления из них.                                                                                                                                                                                                        |
| [**ивиклитригжер**](/windows/desktop/api/taskschd/nn-taskschd-iweeklytrigger)                         | Представляет триггер, который запускает задачу на основе еженедельного расписания.                                                                                                                                                                                                                               |



 


> [!WARNING]
> Интерфейсы планировщик задач 1,0 доступны только в операционных системах Windows 2000, Windows XP и Windows Server 2003. Они являются устаревшими в Windows Vista и могут быть полностью удалены в будущем. Вместо этого используйте перечисленные выше интерфейсы планировщик задач 2,0.
