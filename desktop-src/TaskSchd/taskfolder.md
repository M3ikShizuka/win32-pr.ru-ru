# <a name="taskfolder-object"></a>Объект Таскфолдер

Объект скрипта, предоставляющий методы, используемые для регистрации (создания) задач в папке, удаления задач из папки и создания или удаления вложенных папок из папки.

## <a name="members"></a>Элементы

Объект **таскфолдер** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **таскфолдер** содержит эти методы.



| Метод                                                              | Описание                                                                                                                                    |
|:--------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateFolder**](taskfolder-createfolder.md)                     | Создает папку для связанных задач.<br/>                                                                                                 |
| [**DeleteFolder**](taskfolder-deletefolder.md)                     | Удаляет вложенную папку из родительской папки.<br/>                                                                                         |
| [**делететаск**](taskfolder-deletetask.md)                         | Удаляет задачу из папки.<br/>                                                                                                     |
| [**GetFolder**](taskfolder-getfolder.md)                           | Возвращает папку, содержащую задачи в указанном расположении.<br/>                                                                          |
| [**GetFolders**](taskfolder-getfolders.md)                         | Возвращает все вложенные папки в папке.<br/>                                                                                              |
| [**жетсекуритидескриптор**](taskfolder-getsecuritydescriptor.md)   | Возвращает дескриптор безопасности для папки.<br/>                                                                                        |
| [**Задача задания**](taskfolder-gettask.md)                               | Возвращает задачу в указанном месте в папке.<br/>                                                                                    |
| [**Задачи**](taskfolder-gettasks.md)                             | Возвращает все задачи в папке.<br/>                                                                                                   |
| [**регистертаск**](taskfolder-registertask.md)                     | Регистрирует (создает) новую задачу в папке с помощью XML для определения задачи.<br/>                                                          |
| [**регистертаскдефинитион**](taskfolder-registertaskdefinition.md) | Регистрирует (создает) задачу в указанном расположении с помощью интерфейса [**итаскдефинитион**](/windows/desktop/api/taskschd/nn-taskschd-itaskdefinition) для определения задачи.<br/> |
| [**сетсекуритидескриптор**](taskfolder-setsecuritydescriptor.md)   | Задает дескриптор безопасности для папки.<br/>                                                                                        |



 

### <a name="properties"></a>Свойства

Объект **таскфолдер** имеет следующие свойства.



| Свойство                                   | Тип доступа          | Описание                                                                        |
|:-------------------------------------------|:---------------------|:-----------------------------------------------------------------------------------|
| [**Имя**](taskfolder-name.md)<br/> | Только для чтения<br/> | Возвращает имя, используемое для задания папки, содержащей задачу.<br/> |
| [**Путь**](taskfolder-path.md)<br/> | Только для чтения<br/> | Возвращает путь к месту хранения папки.<br/>                            |
