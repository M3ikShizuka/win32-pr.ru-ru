# <a name="taskdefinitiondata-property"></a>Таскдефинитион. Data, свойство

Для создания скриптов Возвращает или задает данные, связанные с задачей. Эти данные пропускаются службой планировщик задач, но используются сторонними производителями, которые хотят расширить формат задачи.

## <a name="syntax"></a>Синтаксис


```VB
TaskDefinition.Data As String
```



## <a name="property-value"></a>Значение свойства

Данные, связанные с задачей.
