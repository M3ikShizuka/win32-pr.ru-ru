# <a name="tasknamedvaluecollectioncreate-method"></a>Таскнамедвалуеколлектион. Create, метод

Для создания скриптов создает пару "имя-значение" в коллекции.

## <a name="syntax"></a>Синтаксис


```VB
TaskNamedValueCollection.Create( _
  ByVal name, _
  ByVal value, _
  ByRef pair _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*имя* \[ окне\]
</dt> <dd>

Имя, связанное со значением в паре «имя-значение».

</dd> <dt>

*значение* \[ окне\]
</dt> <dd>

Значение, связанное с именем в паре «имя-значение».

</dd> <dt>

*пара* \[ заполняет\]
</dt> <dd>

Пара "имя-значение", созданная в коллекции.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
