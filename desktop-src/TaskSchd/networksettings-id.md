# <a name="networksettingsid-property"></a>NetworkSettings.Id, свойство

Для создания скриптов Возвращает или задает значение идентификатора GUID, определяющее сетевой профиль.

## <a name="syntax"></a>Синтаксис


```VB
NetworkSettings.Id As String
```



## <a name="property-value"></a>Значение свойства

Значение идентификатора GUID, определяющее сетевой профиль.
