# <a name="taskvariables-object"></a>Объект Тасквариаблес

Объект скрипта, который определяет переменные задачи, которые могут передаваться в качестве параметров обработчикам задач и внешним исполняемым файлам, запускаемым задачами.

## <a name="members"></a>Элементы

Объект **тасквариаблес** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Объект **тасквариаблес** содержит эти методы.



| Метод                                         | Описание                                                                                               |
|:-----------------------------------------------|:----------------------------------------------------------------------------------------------------------|
| [**GetContext**](taskvariables-getcontext.md) | Используется для совместного использования контекста разными шагами и задачами, которые находятся в одном и том же экземпляре задания.<br/> |
| [**GetInput**](taskvariables-getinput.md)     | Возвращает входные переменные для задачи.<br/>                                                           |
| [**сетаутпут**](taskvariables-setoutput.md)   | Задает выходные переменные для задачи.<br/>                                                          |
