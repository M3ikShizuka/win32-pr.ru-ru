# <a name="emailactioncc-property"></a>EmailAction.Cc, свойство

\[Этот объект больше не поддерживается. Используйте Иексекактион с командлетом PowerShell [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) в качестве обходного пути.\]

Для создания скриптов Возвращает или задает адрес электронной почты или адреса, которые нужно отправить в сообщении электронной почты.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
EmailAction.Cc As String
```



## <a name="property-value"></a>Значение свойства

Адрес электронной почты или адреса, которые вы хотите отправить в сообщении электронной почты.
