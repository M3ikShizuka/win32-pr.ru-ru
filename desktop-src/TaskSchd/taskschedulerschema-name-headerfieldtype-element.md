# <a name="name-headerfieldtype-element"></a>Элемент Name (Хеадерфиелдтипе)

Содержит имя поля заголовка в сообщении электронной почты.

``` syntax
<xs:element name="Name"
    type="nonEmptyString"
 />
```

Элемент **Name** определяется сложным типом [**хеадерфиелдтипе**](taskschedulerschema-headerfieldtype-complextype.md) .

## <a name="parent-element"></a>Родительский элемент



| Элемент                                                                         | Унаследован от                                                               | Описание                                                    |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------|----------------------------------------------------------------|
| [**хеадерфиелд**](taskschedulerschema-headerfield-headerfieldstype-element.md) | [**хеадерфиелдтипе**](taskschedulerschema-headerfieldtype-complextype.md) | Содержит поле для заголовка в сообщении электронной почты. <br/> |
