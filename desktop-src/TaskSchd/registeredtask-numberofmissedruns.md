# <a name="registeredtasknumberofmissedruns-property"></a>Регистередтаск. Нумберофмисседрунс, свойство

Для сценариев возвращает количество случаев, когда зарегистрированная задача пропустила запланированное выполнение.

## <a name="syntax"></a>Синтаксис


```VB
RegisteredTask.NumberOfMissedRuns As Integer
```



## <a name="property-value"></a>Значение свойства

Количество случаев, когда зарегистрированная задача пропустила запланированное выполнение.
