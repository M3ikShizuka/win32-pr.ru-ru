# <a name="tasknamedvaluecollectionremove-method"></a>Таскнамедвалуеколлектион. Remove, метод

Для создания скриптов удаляет выбранную пару "имя-значение" из коллекции.

## <a name="syntax"></a>Синтаксис


```VB
TaskNamedValueCollection.Remove( _
  ByVal index _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*индекс* \[ окне\]
</dt> <dd>

Индекс удаляемой пары "имя-значение".

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
