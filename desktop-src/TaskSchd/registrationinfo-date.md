# <a name="registrationinfodate-property"></a>Регистратионинфо. Date, свойство

Для создания скриптов Возвращает или задает дату и время регистрации задачи.

## <a name="syntax"></a>Синтаксис


```VB
RegistrationInfo.Date As String
```



## <a name="property-value"></a>Значение свойства

Дата регистрации задачи.

## <a name="remarks"></a>Remarks

При чтении или записи XML для задачи Дата регистрации указывается с помощью элемента [**Date**](taskschedulerschema-date-registrationinfotype-element.md) схемы планировщик задач.
