# <a name="taskdefinitionsettings-property"></a>Таскдефинитион. Settings, свойство

Для создания скриптов Возвращает или задает параметры, определяющие, как служба планировщик задач выполняет задачу.

## <a name="syntax"></a>Синтаксис


```VB
TaskDefinition.Settings As TaskSettings
```



## <a name="property-value"></a>Значение свойства

Параметры, определяющие, как служба планировщик задач выполняет задачу.
