# <a name="taskserviceconnecteduser-property"></a>TaskService. Коннектедусер, свойство

Для создания скриптов возвращает имя пользователя, подключенного к службе планировщик задач.

## <a name="syntax"></a>Синтаксис


```VB
TaskService.ConnectedUser As String
```



## <a name="property-value"></a>Значение свойства

Имя пользователя, подключенного к службе планировщик задач.
