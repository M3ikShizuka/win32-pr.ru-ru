# <a name="actioncollectioncount-property"></a>ActionCollection. Count, свойство

Для сценариев возвращает количество действий в коллекции.

## <a name="syntax"></a>Синтаксис


```VB
ActionCollection.Count As long
```



## <a name="property-value"></a>Значение свойства

Число действий в коллекции. Коллекция может содержать до 32 действий.
