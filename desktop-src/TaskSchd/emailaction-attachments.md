# <a name="emailactionattachments-property"></a>Емаилактион. вложения, свойство

\[Этот объект больше не поддерживается. Используйте Иексекактион с командлетом PowerShell [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) в качестве обходного пути.\]

Для сценариев Возвращает или задает массив вложений, отправляемых с сообщением электронной почты.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
EmailAction.Attachments As String
```



## <a name="property-value"></a>Значение свойства

Массив вложений, отправляемых с сообщением электронной почты.

## <a name="remarks"></a>Комментарии

В массиве вложений может быть не более восьми вложений.
