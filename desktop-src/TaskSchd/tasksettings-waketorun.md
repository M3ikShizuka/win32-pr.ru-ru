# <a name="tasksettingswaketorun-property"></a>Тасксеттингс. Вакеторун, свойство

Для создания скриптов Возвращает или задает логическое значение, указывающее, что планировщик задач выводит компьютер из спящего режима во время выполнения задачи.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
TaskSettings.WakeToRun As Boolean
```



## <a name="property-value"></a>Значение свойства

Если значение — true, свойство указывает, что планировщик задач будет выводить компьютер из спящего режима во время выполнения задачи.

## <a name="remarks"></a>Remarks

Когда служба планировщик задач пробуждает компьютер для выполнения задачи, экран может остаться недоступным, даже если компьютер больше не находится в спящем или спящем режиме. экран будет включен, когда Windows Vista обнаружит, что пользователь вернулся на использование компьютера.

При чтении или записи XML для задачи этот параметр указывается в элементе [**вакеторун**](taskschedulerschema-waketorun-settingstype-element.md) схемы планировщик задач.
