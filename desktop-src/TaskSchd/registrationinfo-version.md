# <a name="registrationinfoversion-property"></a>Регистратионинфо. Version, свойство

Для сценариев Возвращает или задает номер версии задачи.

## <a name="syntax"></a>Синтаксис


```VB
RegistrationInfo.Version As String
```



## <a name="property-value"></a>Значение свойства

Номер версии задачи.

## <a name="remarks"></a>Комментарии

При чтении или записи XML для задачи номер версии задачи указывается с помощью элемента [**Version**](taskschedulerschema-version-registrationinfotype-element.md) схемы планировщик задач.
