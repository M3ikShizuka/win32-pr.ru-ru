# <a name="taskdefinitionregistrationinfo-property"></a>Таскдефинитион. Регистратионинфо, свойство

Для создания скриптов Возвращает или задает сведения о регистрации, используемые для описания задачи, например описание задачи, автора задачи и дату регистрации задачи.

## <a name="syntax"></a>Синтаксис


```VB
TaskDefinition.RegistrationInfo As RegistrationInfo
```



## <a name="property-value"></a>Значение свойства

Сведения о регистрации, которые используются для описания задачи, например описание задачи, автора задачи и Дата регистрации задачи.
