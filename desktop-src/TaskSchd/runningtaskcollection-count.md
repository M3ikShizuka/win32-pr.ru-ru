# <a name="runningtaskcollectioncount-property"></a>Руннингтаскколлектион. Count, свойство

Для сценариев возвращает количество выполняемых задач в коллекции.

## <a name="syntax"></a>Синтаксис


```VB
RunningTaskCollection.Count As long
```



## <a name="property-value"></a>Значение свойства

Количество выполняемых задач в коллекции.
