# <a name="registeredtaskpath-property"></a>Регистередтаск. Path, свойство

Для создания скриптов Возвращает путь к месту хранения зарегистрированной задачи.

## <a name="syntax"></a>Синтаксис


```VB
RegisteredTask.Path As String
```



## <a name="property-value"></a>Значение свойства

Путь к месту хранения зарегистрированной задачи.
