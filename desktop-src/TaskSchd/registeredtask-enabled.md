# <a name="registeredtaskenabled-property"></a>Регистередтаск. Enabled, свойство

Для создания скриптов Возвращает или задает логическое значение, указывающее, включена ли зарегистрированная задача.

## <a name="syntax"></a>Синтаксис


```VB
RegisteredTask.Enabled As Boolean
```



## <a name="property-value"></a>Значение свойства

Логическое значение, указывающее, включена ли зарегистрированная задача.
