# <a name="taskdefinitionxmltext-property"></a>TaskDefinition.Xml, свойство текста

Для создания скриптов Возвращает или задает определение задачи в формате XML.

## <a name="syntax"></a>Синтаксис


```VB
TaskDefinition.XmlText As String
```



## <a name="property-value"></a>Значение свойства

Определение задачи в формате XML.

## <a name="remarks"></a>Комментарии

XML-код для задачи определяется [схемой планировщик задач](task-scheduler-schema.md). Пример XML-кода задачи см. в разделе [Пример ежедневного триггера (XML)](daily-trigger-example--xml-.md).
