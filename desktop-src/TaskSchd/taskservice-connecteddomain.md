# <a name="taskserviceconnecteddomain-property"></a>TaskService. Коннектеддомаин, свойство

Для создания скриптов возвращает имя домена, к которому подключен компьютер [**TargetServer**](taskservice-targetserver.md) .

## <a name="syntax"></a>Синтаксис


```VB
TaskService.ConnectedDomain As String
```



## <a name="property-value"></a>Значение свойства

Домен, к которому подключен компьютер [**TargetServer**](taskservice-targetserver.md) .
