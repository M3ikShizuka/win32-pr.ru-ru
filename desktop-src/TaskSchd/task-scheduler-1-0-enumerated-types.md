# <a name="task-scheduler-10-enumerated-types"></a>Перечислимые типы планировщик задач 1,0

\[\[Этот API может быть изменен или недоступен в последующих версиях операционной системы или продукта. Вместо них используйте [планировщик задач интерфейсы 2,0](task-scheduler-2-0-interfaces.md) или [перечислимые типы планировщик задач 2,0](task-scheduler-2-0-enumerated-types.md) . \]\]

Следующие перечислимые типы введены планировщик задач 1,0.



| Имя                                             | Описание                                                                        |
|--------------------------------------------------|------------------------------------------------------------------------------------|
| [**\_тип триггера задачи \_**](/windows/desktop/api/Mstask/ne-mstask-task_trigger_type) | Определяет типы [*триггеров*](t.md) , связанных с задачей. |
| [**таскпаже**](/windows/desktop/api/Mstask/ne-mstask-taskpage)                     | Определяет тип извлекаемой страницы задачи.                                     |
