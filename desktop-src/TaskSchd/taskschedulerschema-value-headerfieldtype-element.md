# <a name="value-headerfieldtype-element"></a>Элемент value (Хеадерфиелдтипе)

Содержит значение поля заголовка в сообщении электронной почты.

``` syntax
<xs:element name="Value"
    type="string"
 />
```

Элемент **value** определяется сложным типом [**хеадерфиелдтипе**](taskschedulerschema-headerfieldtype-complextype.md) .

## <a name="parent-element"></a>Родительский элемент



| Элемент                                                                         | Унаследован от                                                               | Описание                                                    |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------|----------------------------------------------------------------|
| [**хеадерфиелд**](taskschedulerschema-headerfield-headerfieldstype-element.md) | [**хеадерфиелдтипе**](taskschedulerschema-headerfieldtype-complextype.md) | Содержит поле для заголовка в сообщении электронной почты. <br/> |
