# <a name="actioncollectionxmltext-property"></a>Свойство ActionCollection:: XmlText

Для создания скриптов Возвращает или задает версию коллекции в формате XML.

## <a name="syntax"></a>Синтаксис


```VB
.XmlText As String
```



## <a name="property-value"></a>Значение свойства

Версия коллекции в формате XML.
