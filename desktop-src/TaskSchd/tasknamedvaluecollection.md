# <a name="tasknamedvaluecollection-object"></a>Объект Таскнамедвалуеколлектион

Объект скрипта, содержащий коллекцию пар "имя-значение" для объекта [**таскнамедвалуепаир**](tasknamedvaluepair.md) .

## <a name="members"></a>Элементы

Объект **таскнамедвалуеколлектион** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **таскнамедвалуеколлектион** содержит эти методы.



| Метод                                            | Описание                                                        |
|:--------------------------------------------------|:-------------------------------------------------------------------|
| [**Clear**](tasknamedvaluecollection-clear.md)   | Очищает всю коллекцию пар "имя-значение".<br/>       |
| [**Создание**](tasknamedvaluecollection-create.md) | Создает пару "имя-значение" в коллекции.<br/>            |
| [**Отменит**](tasknamedvaluecollection-remove.md) | Удаляет выбранную пару "имя-значение" из коллекции.<br/> |



 

### <a name="properties"></a>Свойства

Объект **таскнамедвалуеколлектион** имеет следующие свойства.



| Свойство                                                   | Описание                                                        |
|:-----------------------------------------------------------|:-------------------------------------------------------------------|
| [**Count**](tasknamedvaluecollection-count.md)<br/> | Возвращает число пар "имя-значение" в коллекции.<br/>  |
| [**Элемент**](tasknamedvaluecollection-item.md)<br/>   | Возвращает указанную пару "имя-значение" из коллекции.<br/> |
