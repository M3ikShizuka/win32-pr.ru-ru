# <a name="registeredtaskgetruntimes-method"></a>Регистередтаск. среды выполнения, метод

Для сценариев Возвращает время, которое запланировано для выполнения зарегистрированной задачи в течение заданного времени.

## <a name="syntax"></a>Синтаксис


```VB
RegisteredTask.GetRunTimes( _
  ByVal begin, _
  ByVal end, _
  ByRef count, _
  ByRef rgstTaskTimes _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Начало* \[ окне\]
</dt> <dd>

Время начала запроса.

</dd> <dt>

*конец* \[ окне\]
</dt> <dd>

Время окончания запроса.

</dd> <dt>

*количество* \[ заполняет\]
</dt> <dd>

Количество запланированных запусков задачи.

</dd> <dt>

*ргсттасктимес* \[ заполняет\]
</dt> <dd>

Запланированное время выполнения задачи.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Если зарегистрированная задача содержит триггеры, которые по отдельности отключены, эти триггеры будут влиять на следующее запланированное время выполнения, которое возвращается, даже если они отключены.
