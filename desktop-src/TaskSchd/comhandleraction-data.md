# <a name="comhandleractiondata-property"></a>Комхандлерактион. Data, свойство

Для создания скриптов Возвращает или задает дополнительные данные, связанные с обработчиком.

## <a name="syntax"></a>Синтаксис


```VB
ComHandlerAction.Data As String
```



## <a name="property-value"></a>Значение свойства

Аргументы, необходимые обработчику.

## <a name="remarks"></a>Remarks

При чтении или записи XML данные обработчика COM-данных указываются в элементе [**Data**](taskschedulerschema-data-comhandlertype-element.md) схемы планировщик задач.
