# <a name="starting-an-executable-weekly"></a>Запуск исполняемого файла еженедельно

Создание задачи, запускающей исполняемый файл еженедельно, осуществляется путем определения еженедельного триггера и исполняемого действия.

## <a name="weekly-triggers"></a>Еженедельные триггеры

Еженедельные триггеры используют начальную границу для активации триггера и для указания времени суток, в течение которого выполняется задача. При активации триггера он также указывает, когда задача выполняется в течение недели, и если задача выполняется каждую неделю, каждую неделю, каждую третью неделю и т. д.
