# <a name="actionid-property"></a>Action.Id, свойство

Для создания скриптов Возвращает или задает идентификатор действия.

## <a name="syntax"></a>Синтаксис


```VB
Action.Id As String
```



## <a name="property-value"></a>Значение свойства

Определяемый пользователем идентификатор действия. Этот идентификатор используется планировщик задач в целях ведения журнала.

## <a name="remarks"></a>Remarks

Сведения о совместной работе действий и задач см. в разделе [действия задач](task-actions.md).
