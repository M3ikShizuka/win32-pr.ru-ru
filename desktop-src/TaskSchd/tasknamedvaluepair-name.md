# <a name="tasknamedvaluepairname-property"></a>TaskNamedValuePair.Name, свойство

Для создания скриптов Возвращает или задает имя, связанное со значением в паре «имя-значение».

## <a name="syntax"></a>Синтаксис


```VB
TaskNamedValuePair.Name As String
```



## <a name="property-value"></a>Значение свойства

Имя, связанное со значением в паре «имя-значение».
