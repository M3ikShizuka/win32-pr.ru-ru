# <a name="taskdefinition-object"></a>Объект Таскдефинитион

Объект скрипта, который определяет все компоненты задачи, такие как параметры задачи, триггеры, действия и сведения о регистрации.

## <a name="members"></a>Элементы

Объект **таскдефинитион** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Объект **таскдефинитион** имеет следующие свойства.



| Свойство                                                               | Тип доступа           | Описание                                                                                                                                                                             |
|:-----------------------------------------------------------------------|:----------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Действия**](taskdefinition-actions.md)<br/>                   | Чтение/запись<br/> | Возвращает или задает коллекцию действий, выполняемых задачей.<br/>                                                                                                         |
| [**Data**](taskdefinition-data.md)<br/>                         | Чтение/запись<br/> | Возвращает или задает данные, связанные с задачей. Эти данные пропускаются службой планировщик задач, но используются сторонними производителями, которые хотят расширить формат задачи.<br/> |
| [**Основного**](taskdefinition-principal.md)<br/>               | Чтение/запись<br/> | Возвращает или задает участника задачи, предоставляющей учетные данные безопасности для задачи.<br/>                                                                                 |
| [**регистратионинфо**](taskdefinition-registrationinfo.md)<br/> | Чтение/запись<br/> | Возвращает или задает сведения о регистрации, используемые для описания задачи, например описание задачи, автора задачи и дату регистрации задачи.<br/> |
| [**Параметры**](taskdefinition-settings.md)<br/>                 | Чтение/запись<br/> | Возвращает или задает параметры, определяющие, как служба планировщик задач выполняет задачу.<br/>                                                                                      |
| [**Триггеры**](taskdefinition-triggers.md)<br/>                 | Чтение/запись<br/> | Возвращает или задает коллекцию триггеров, которые используются для запуска задачи.<br/>                                                                                                         |
| [**XmlText**](taskdefinition-xmltext.md)<br/>                   | Чтение/запись<br/> | Возвращает или задает определение задачи в формате XML.<br/>                                                                                                                       |



 

## <a name="remarks"></a>Комментарии

При чтении или записи собственного XML-кода для задачи определение задачи указывается с помощью элемента [**Task**](taskschedulerschema-task-element.md) схемы планировщик задач.
