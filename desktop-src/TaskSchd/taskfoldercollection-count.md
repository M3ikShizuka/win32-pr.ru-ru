# <a name="taskfoldercollectioncount-property"></a>Таскфолдерколлектион. Count, свойство

Для сценариев возвращает количество папок в коллекции.

## <a name="syntax"></a>Синтаксис


```VB
TaskFolderCollection.Count As long
```



## <a name="property-value"></a>Значение свойства

Число триггеров в коллекции.
