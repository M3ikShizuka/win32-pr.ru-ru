# <a name="taskserviceconnected-property"></a>TaskService. Connected, свойство

Для создания скриптов возвращает логическое значение, указывающее, установлено ли соединение со службой планировщик задач.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
TaskService.Connected
```



## <a name="property-value"></a>Значение свойства

Логическое значение, указывающее, установлено ли соединение со службой планировщик задач.
