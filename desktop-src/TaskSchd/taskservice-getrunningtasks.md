# <a name="taskservicegetrunningtasks-method"></a>TaskService. Жетруннингтаскс, метод

Для создания скриптов Возвращает коллекцию выполняющихся задач.

> [!Note]  
> **TaskService. жетруннингтаскс** будет возвращать только коллекцию выполняющихся задач, выполняемых в контексте безопасности пользователя или ниже. Например, для членов группы "Администраторы" **жетруннингтаскс** будет возвращать коллекцию всех выполняемых задач, но для членов группы "Пользователи" **жетруннингтаскс** будет возвращать только коллекцию задач, которые выполняются в контексте безопасности группы "Пользователи".

 

## <a name="syntax"></a>Синтаксис


```VB
TaskService.GetRunningTasks( _
  ByVal flags _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Флаги* \[ окне\]
</dt> <dd>

Передайте значение 1, чтобы вернуть все выполняемые задачи, включая скрытые задачи. Передайте значение 0, чтобы получить коллекцию выполняющихся задач, которые не являются скрытыми.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Объект [**руннингтаскколлектион**](runningtaskcollection.md) , содержащий выполняемые в данный момент задачи.
