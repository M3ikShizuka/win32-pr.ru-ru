# <a name="tasknamedvaluepairvalue-property"></a>Таскнамедвалуепаир. Value, свойство

Для создания скриптов Возвращает или задает значение, связанное с именем в паре «имя-значение».

## <a name="syntax"></a>Синтаксис


```VB
TaskNamedValuePair.Value As String
```



## <a name="property-value"></a>Значение свойства

Значение, связанное с именем в паре «имя-значение».
