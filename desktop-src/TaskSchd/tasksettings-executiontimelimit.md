# <a name="tasksettingsexecutiontimelimit-property"></a>Тасксеттингс. Ексекутионтимелимит, свойство

Для создания скриптов Возвращает или задает период времени, в течение которого может завершиться задача. По умолчанию задача будет остановлена 72 часов после начала ее выполнения. Это можно изменить, изменив этот параметр.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
TaskSettings.ExecutionTimeLimit As String
```



## <a name="property-value"></a>Значение свойства

Время, в течение которого разрешено выполнение задачи. Формат этой строки — Пнинмндтнхнмнс, где Нью-то число лет, nM — это количество месяцев, а — число дней, равное, т. е. в качестве разделителя даты и времени, nH — это количество часов, а в качестве значения nS — количество секунд (например, PT5M указывает 5 минут, а P1M4DT2H5M — один месяц, четыре дня, два часа и пять минут). Значение PT0S позволит задаче выполняться неограниченное время. Если для этого параметра задано значение Nothing, то ограничение времени выполнения будет бесконечным.

## <a name="remarks"></a>Комментарии

При чтении или записи XML для задачи этот параметр указывается в элементе [**ексекутионтимелимит**](taskschedulerschema-executiontimelimit-settingstype-element.md) схемы планировщик задач.
