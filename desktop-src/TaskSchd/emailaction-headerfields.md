# <a name="emailactionheaderfields-property"></a>Емаилактион. Хеадерфиелдс, свойство

\[Этот объект больше не поддерживается. Используйте Иексекактион с командлетом PowerShell [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) в качестве обходного пути.\]

Для создания скриптов Возвращает или задает сведения о заголовке электронного письма, которое необходимо отправить.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
EmailAction.HeaderFields As TaskNamedValueCollection
```



## <a name="property-value"></a>Значение свойства

Сведения заголовка в сообщении электронной почты, которое необходимо отправить.
