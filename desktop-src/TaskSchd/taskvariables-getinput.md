# <a name="taskvariablesgetinput-method"></a>Метод Тасквариаблес. input

Для создания скриптов получает входные переменные для задачи. Этот метод не реализован.

## <a name="syntax"></a>Синтаксис


```VB
TaskVariables.GetInput( _
  ByRef input _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*входные данные* \[ заполняет\]
</dt> <dd>

Входные переменные для задачи.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
