# <a name="taskfoldername-property"></a>TaskFolder.Name, свойство

Для сценария возвращает имя, которое используется для задания папки, содержащей задачу.

## <a name="syntax"></a>Синтаксис


```VB
TaskFolder.Name As String
```



## <a name="property-value"></a>Значение свойства

Имя, используемое для задания папки.
