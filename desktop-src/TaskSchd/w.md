<dl> <dt>

<span id="_msb_work_items_gly"></span><span id="_MSB_WORK_ITEMS_GLY"></span>**рабочие элементы**
</dt> <dd>

Элемент, который можно запланировать с помощью службы планировщик задач. Это может быть любой элемент, выполняемый службой планировщик задач в момент, определенный триггерами элемента.

В настоящее время задачи являются единственным допустимым типом рабочего элемента.

</dd> <dt>

<span id="_msb_working_directory_gly"></span><span id="_MSB_WORKING_DIRECTORY_GLY"></span>**Рабочий каталог**
</dt> <dd>

Каталог, в котором планировщик задач начнет выполнение задачи. Если рабочий каталог не указан, планировщик задач будет выполнять задачу в каталоге% WINDIR% \\ System32.

</dd> </dl>
