# <a name="runningtaskcurrentaction-property"></a>Руннингтаск. Куррентактион, свойство

Для сценария возвращает имя текущего действия, выполняемого выполняющейся задачей.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
RunningTask.CurrentAction As String
```



## <a name="property-value"></a>Значение свойства

Имя текущего действия, выполняемого выполняющейся задачей.
