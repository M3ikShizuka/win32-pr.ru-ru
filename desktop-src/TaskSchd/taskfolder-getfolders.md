# <a name="taskfoldergetfolders-property"></a>Таскфолдер.

Для сценариев получает все вложенные папки в папке.

## <a name="syntax"></a>Синтаксис


```VB
TaskFolder.GetFolders( _
  ByVal flags, _
  ByRef folders _
)
```
