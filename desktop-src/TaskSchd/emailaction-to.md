# <a name="emailactionto-property"></a>EmailAction.To, свойство

\[Этот объект больше не поддерживается. Используйте Иексекактион с командлетом PowerShell [**Send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) в качестве обходного пути.\]

Для создания скриптов Возвращает или задает адрес электронной почты или адреса, по которым необходимо отправить сообщение.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
EmailAction.To As String
```



## <a name="property-value"></a>Значение свойства

Адрес электронной почты или адреса, по которым вы хотите отправить сообщение.
