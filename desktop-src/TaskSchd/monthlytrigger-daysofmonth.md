# <a name="monthlytriggerdaysofmonth-property"></a>Монслитригжер. DaysOfMonth, свойство

Для сценариев Возвращает или задает дни месяца, в которые выполняется задача.

## <a name="syntax"></a>Синтаксис


```VB
MonthlyTrigger.DaysOfMonth As long
```



## <a name="property-value"></a>Значение свойства

Битовая маска, указывающая дни месяца, в течение которых выполняется задача.
