# <a name="runningtask-object"></a>Объект Руннингтаск

Объект скрипта, предоставляющий методы для получения информации из выполняемой задачи и управления ей.

## <a name="members"></a>Элементы

Объект **руннингтаск** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **руннингтаск** содержит эти методы.



| Метод                                 | Описание                                                           |
|:---------------------------------------|:----------------------------------------------------------------------|
| [**Обновить**](runningtask-refresh.md) | Обновляет все локальные переменные экземпляра задачи.<br/> |
| [**Позиции**](runningtask-stop.md)       | Останавливает этот экземпляр задачи.<br/>                           |



 

### <a name="properties"></a>Свойства

Объект **руннингтаск** имеет следующие свойства.



| Свойство                                                      | Тип доступа          | Описание                                                                         |
|:--------------------------------------------------------------|:---------------------|:------------------------------------------------------------------------------------|
| [**куррентактион**](runningtask-currentaction.md)<br/> | Только для чтения<br/> | Возвращает имя текущего действия, выполняемого выполняющейся задачей.<br/> |
| [**енгинепид**](runningtask-enginepid.md)<br/>         | Только для чтения<br/> | Возвращает идентификатор процесса для подсистемы (процесса), выполняющей задачу.<br/>  |
| [**инстанцегуид**](runningtask-instanceguid.md)<br/>   | Только для чтения<br/> | Возвращает идентификатор GUID для этого экземпляра задачи.<br/>                  |
| [**Имя**](runningtask-name.md)<br/>                   | Только для чтения<br/> | Возвращает имя задачи.<br/>                                               |
| [**Путь**](runningtask-path.md)<br/>                   | Только для чтения<br/> | Возвращает путь, по которому хранится задача.<br/>                               |
| [**Состояние**](runningtask-state.md)<br/>                 | Только для чтения<br/> | Возвращает состояние выполняющейся задачи. <br/>                                     |
