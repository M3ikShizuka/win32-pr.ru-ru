# <a name="timetrigger-object"></a>Объект Тиметригжер

Объект скрипта, представляющий триггер, который запускает задачу в заданную дату и время.

## <a name="members"></a>Элементы

Объект **тиметригжер** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Элемент Property

Объект **тиметригжер** имеет следующие свойства.



| Свойство                                                            | Тип доступа           | Описание                                                                                                                                                                      |
|:--------------------------------------------------------------------|:----------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Активировано**](trigger-enabled.md)<br/>                       | Чтение/запись<br/> | Наследуется от [**триггера**](trigger.md). Возвращает или задает логическое значение, указывающее, включен ли триггер.<br/>                                                |
| [**ендбаундари**](trigger-endboundary.md)<br/>               | Чтение/запись<br/> | Наследуется от [**триггера**](trigger.md). Возвращает или задает дату и время деактивации триггера. Триггер не может запустить задачу после ее деактивации.<br/> |
| [**ексекутионтимелимит**](trigger-executiontimelimit.md)<br/> | Чтение/запись<br/> | Наследуется от [**триггера**](trigger.md). Возвращает или задает максимальное время, в течение которого разрешено выполнение задачи, запущенной триггером.<br/>                           |
| [**Удостоверения**](/windows/desktop/api/taskschd/nf-taskschd-itrigger-get_id)<br/>                                | Чтение/запись<br/> | Наследуется от [**триггера**](trigger.md). Возвращает или задает идентификатор триггера.<br/>                                                                               |
| [**рандомделай**](dailytrigger-randomdelay.md)<br/>          | Чтение/запись<br/> | Возвращает или задает время задержки, которое добавляется случайным образом к времени начала триггера.<br/>                                                                                    |
| [**Повторение**](trigger-repetition.md)<br/>                 | Чтение/запись<br/> | Наследуется от [**триггера**](trigger.md). Возвращает или задает частоту выполнения задачи и период повторения шаблона повторения после запуска задачи.<br/>          |
| [**стартбаундари**](trigger-startboundary.md)<br/>           | Чтение/запись<br/> | Наследуется от [**триггера**](trigger.md). Возвращает или задает дату и время активации триггера. Этот элемент является обязательным.<br/>                                    |
| [**Тип**](/windows/desktop/api/taskschd/nf-taskschd-itrigger-get_type)<br/>                            | Только для чтения<br/>  | Наследуется от [**триггера**](trigger.md). Возвращает тип триггера.<br/>                                                                                              |



 

## <a name="remarks"></a>Remarks

Элемент [**стартбаундари**](taskschedulerschema-startboundary-triggerbasetype-element.md) является обязательным элементом для триггеров времени и календаря ([**тиметригжер**](taskschedulerschema-timetrigger-triggergroup-element.md) и [**календартригжер**](taskschedulerschema-calendartrigger-triggergroup-element.md)).

При чтении или записи XML-кода для задачи триггер простоя указывается с помощью элемента [**тиметригжер**](taskschedulerschema-timetrigger-triggergroup-element.md) схемы планировщик задач.
