# <a name="runningtaskname-property"></a>RunningTask.Name, свойство

Для сценариев возвращает имя задачи.

## <a name="syntax"></a>Синтаксис


```VB
RunningTask.Name As string
```



## <a name="property-value"></a>Значение свойства

Имя данной задачи.
