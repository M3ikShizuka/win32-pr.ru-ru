# <a name="taskservice-object"></a>Объект TaskService

Для сценариев предоставляет доступ к службе планировщик задач для управления зарегистрированными задачами.

перед вызовом любого из других методов **TaskService** следует вызвать метод [**TaskService. Подключение**](taskservice-connect.md) .

## <a name="members"></a>Элементы

Объект **TaskService** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **TaskService** содержит эти методы.



| Метод                                                 | Описание                                                                                                                                                                                                          |
|:-------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Подключение**](taskservice-connect.md)                 | Подключается к удаленному компьютеру и связывает все последующие вызовы в этом интерфейсе с удаленным сеансом.<br/>                                                                                                 |
| [**GetFolder**](taskservice-getfolder.md)             | Возвращает путь к папке зарегистрированных задач.<br/>                                                                                                                                                            |
| [**жетруннингтаскс**](taskservice-getrunningtasks.md) | Возвращает коллекцию выполняющихся задач.<br/>                                                                                                                                                                       |
| [**NewTask**](taskservice-newtask.md)                 | Возвращает пустой объект определения задачи, который заполняется параметрами и свойствами, а затем регистрируется с помощью метода [**таскфолдер. регистертаскдефинитион**](taskfolder-registertaskdefinition.md) .<br/> |



 

### <a name="properties"></a>Свойства

Объект **TaskService** имеет следующие свойства.



| Свойство                                                          | Описание                                                                                                                 |
|:------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------|
| [**Подключен**](taskservice-connected.md)<br/>             | Возвращает логическое значение, указывающее, установлено ли соединение со службой планировщик задач.<br/>                          |
| [**коннектеддомаин**](taskservice-connecteddomain.md)<br/> | Возвращает имя домена, к которому подключен компьютер [**TargetServer**](taskservice-targetserver.md) .<br/> |
| [**коннектедусер**](taskservice-connecteduser.md)<br/>     | Возвращает имя пользователя, подключенного к службе планировщик задач.<br/>                                       |
| хигхестверсион<br/>                                         | Возвращает самую новую версию планировщик задач, поддерживаемую компьютером.<br/>                                             |
| [**TargetServer**](taskservice-targetserver.md)<br/>       | Возвращает имя компьютера, на котором запущена служба планировщик задач, к которой подключен пользователь.<br/>          |
