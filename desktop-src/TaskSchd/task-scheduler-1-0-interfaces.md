# <a name="task-scheduler-10-interfaces"></a>Интерфейсы планировщик задач 1,0

\[\[Этот API может быть изменен или недоступен в последующих версиях операционной системы или продукта. Вместо них используйте [планировщик задач интерфейсы 2,0](task-scheduler-2-0-interfaces.md) или [перечислимые типы планировщик задач 2,0](task-scheduler-2-0-enumerated-types.md) . \]\]

Интерфейсы, описанные в следующих разделах, предоставляют программный доступ к функциональным возможностям, доступным в планировщик задач, используемом в операционных системах Windows 2000, Windows XP и Windows Server 2003.

В этих разделах содержится описание интерфейса, список свойств и методов, определенных интерфейсом, и примечания о специальных обстоятельствах, которые следует отметить при использовании интерфейса.


Следующие интерфейсы представлены планировщик задач 1,0.



| Интерфейс                                        | Описание                                                                                                                                                                                                                           |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**иенумворкитемс**](/windows/desktop/api/Mstask/nn-mstask-ienumworkitems)         | Предоставляет методы для перечисления задач в [*папке «Назначенные задания*](s.md)».                                                                                                              |
| [**ипровидетаскпаже**](/windows/desktop/api/Mstask/nn-mstask-iprovidetaskpage)     | Предоставляет методы для доступа к параметрам страницы свойств задачи.                                                                                                                                                                 |
| [**исчедуледворкитем**](/windows/desktop/api/Mstask/nn-mstask-ischeduledworkitem) | Предоставляет методы для управления конкретными [*рабочими элементами*](w.md).                                                                                                                                                 |
| [**ITask**](/windows/desktop/api/Mstask/nn-mstask-itask)                           | Предоставляет методы для выполнения задач, получения или задания сведений о задачах, а также для завершения задач. Он является производным от интерфейса [**исчедуледворкитем**](/windows/desktop/api/Mstask/nn-mstask-ischeduledworkitem) и наследует все методы этого интерфейса. |
| [**итасксчедулер**](/windows/desktop/api/Mstask/nn-mstask-itaskscheduler)         | Предоставляет методы для планирования задач.                                                                                                                                                                                            |
| [**итасктригжер**](/windows/desktop/api/Mstask/nn-mstask-itasktrigger)             | Предоставляет методы для доступа и настройки триггеров для задачи. Триггеры указывают время начала задачи, критерии повторения и другие параметры, управляющие запуском задачи.                                                     |
