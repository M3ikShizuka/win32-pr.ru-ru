# <a name="taskfoldergettasks-property"></a>Таскфолдер. Tasks, свойство

Для создания скриптов возвращает все задачи в папке.

## <a name="syntax"></a>Синтаксис


```VB
TaskFolder.GetTasks( _
  ByVal flags, _
  ByRef ppTasks _
)
```
