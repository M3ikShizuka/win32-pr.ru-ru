# <a name="sessionstatechangetriggerdelay-property"></a>Сессионстатечанжетригжер. Delay, свойство

Для создания скриптов Возвращает или задает значение, которое указывает, как долго происходит задержка перед запуском задачи после обнаружения изменения состояния сеанса сервера терминалов. Формат этой строки — Пнинмндтнхнмнс, где Нью-то число лет, nM — это количество месяцев, а — число дней, равное, т. е. в качестве разделителя даты и времени, nH — это количество часов, а в качестве значения nS — количество секунд (например, PT5M указывает 5 минут, а P1M4DT2H5M — один месяц, четыре дня, два часа и пять минут).

## <a name="syntax"></a>Синтаксис


```VB
SessionStateChangeTrigger.Delay As String
```



## <a name="property-value"></a>Значение свойства

Задержка, выполняемая перед запуском задачи после обнаружения изменения состояния сеанса сервера терминалов.
