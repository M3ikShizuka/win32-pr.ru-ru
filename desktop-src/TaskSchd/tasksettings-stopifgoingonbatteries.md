# <a name="tasksettingsstopifgoingonbatteries-property"></a>Тасксеттингс. Стопифгоингонбаттериес, свойство

Для создания скриптов Возвращает или задает логическое значение, указывающее, что задача будет остановлена, если компьютер перейдет в аккумулятор.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
TaskSettings.StopIfGoingOnBatteries As Boolean
```



## <a name="property-value"></a>Значение свойства

Логическое значение, указывающее, что задача будет остановлена при переходе компьютера на питание. Если значение — true, свойство указывает, что задача будет остановлена при переходе компьютера на питание. Если значение равно false, свойство указывает, что задача не будет остановлена, если компьютер переходит в аккумулятор. Значение по умолчанию равно True. Дополнительные сведения см. в разделе Примечания.

## <a name="remarks"></a>Комментарии

При чтении или записи XML для задачи этот параметр указывается в элементе [**стопифгоингонбаттериес**](taskschedulerschema-stopifgoingonbatteries-settingstype-element.md) схемы планировщик задач.
