# <a name="tasksettings-object"></a>Объект Тасксеттингс

Объект скрипта, предоставляющий параметры, которые служба планировщик задач использует для выполнения задачи.

## <a name="members"></a>Элементы

Объект **тасксеттингс** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

## <a name="remarks"></a>Комментарии

По умолчанию задача будет остановлена 72 часов после начала ее выполнения. Это можно изменить, изменив параметр [**ексекутионтимелимит**](tasksettings-executiontimelimit.md) .

При чтении или записи XML-кода для задачи параметры задачи определяются в элементе [**Settings**](taskschedulerschema-settings-tasktype-element.md) схемы планировщик задач.
