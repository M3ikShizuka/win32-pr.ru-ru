# <a name="registeredtask-object"></a>Объект Регистередтаск

Объект скрипта, предоставляющий методы, которые используются для немедленного выполнения задачи, получают все выполняющиеся экземпляры задачи, получают или задают учетные данные, используемые для регистрации задачи, а также свойства, описывающие задачу.

## <a name="members"></a>Элементы

Объект **регистередтаск** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **регистередтаск** содержит эти методы.



| Метод                                                                | Описание                                                                                     |
|:----------------------------------------------------------------------|:------------------------------------------------------------------------------------------------|
| [**Экземпляры экземпляров**](registeredtask-getinstances.md)                   | Возвращает все экземпляры зарегистрированной задачи, которые выполняются в данный момент.<br/>             |
| [**Среды выполнения**](/windows/desktop/api/taskschd/nf-taskschd-iregisteredtask-getruntimes)                    | Возвращает время, которое запланировано для выполнения зарегистрированной задачи в течение заданного времени.<br/> |
| [**жетсекуритидескриптор**](registeredtask-getsecuritydescriptor.md) | Возвращает дескриптор безопасности, который используется в качестве учетных данных для зарегистрированной задачи.<br/>    |
| [**Выполнить**](registeredtask-run.md)                                     | Немедленно запускает зарегистрированную задачу.<br/>                                                |
| [**рунекс**](registeredtask-runex.md)                                 | Выполняет зарегистрированную задачу сразу же с помощью указанных флагов и идентификатора сеанса.<br/> |
| [**сетсекуритидескриптор**](registeredtask-setsecuritydescriptor.md) | Задает дескриптор безопасности, используемый в качестве учетных данных для зарегистрированной задачи.<br/>    |
| [**Stop**](registeredtask-stop.md)                                   | Немедленно останавливает зарегистрированную задачу.<br/>                                               |



 

### <a name="properties"></a>Свойства

Объект **регистередтаск** имеет следующие свойства.



| Свойство                                                                   | Тип доступа           | Описание                                                                               |
|:---------------------------------------------------------------------------|:----------------------|:------------------------------------------------------------------------------------------|
| [**Определение**](registeredtask-definition.md)<br/>                 | Только для чтения<br/>  | Возвращает определение задачи.<br/>                                               |
| [**Активировано**](registeredtask-enabled.md)<br/>                       | Чтение/запись<br/> | Возвращает или задает логическое значение, указывающее, включена ли зарегистрированная задача.<br/>  |
| [**ластрунтиме**](registeredtask-lastruntime.md)<br/>               | Только для чтения<br/>  | Возвращает время последнего запуска зарегистрированной задачи.<br/>                                |
| [**ласттаскресулт**](registeredtask-lasttaskresult.md)<br/>         | Только для чтения<br/>  | Возвращает результаты, которые были возвращены при последнем запуске зарегистрированной задачи.<br/> |
| [**Имя**](registeredtask-name.md)<br/>                             | Только для чтения<br/>  | Возвращает имя зарегистрированной задачи.<br/>                                          |
| [**Расчетное nextruntime**](registeredtask-nextruntime.md)<br/>               | Только для чтения<br/>  | Возвращает время следующего запланированного выполнения зарегистрированной задачи.<br/>               |
| [**нумберофмисседрунс**](registeredtask-numberofmissedruns.md)<br/> | Только для чтения<br/>  | Возвращает количество случаев, когда зарегистрированная задача пропустила запланированное выполнение.<br/>       |
| [**Путь**](registeredtask-path.md)<br/>                             | Только для чтения<br/>  | Возвращает путь, по которому хранится зарегистрированная задача.<br/>                          |
| [**Состояние**](registeredtask-state.md)<br/>                           | Только для чтения<br/>  | Возвращает операционное состояние зарегистрированной задачи.<br/>                             |
| [**XML**](registeredtask-xml.md)<br/>                               | Только для чтения<br/>  | Возвращает сведения о регистрации в формате XML для зарегистрированной задачи.<br/>       |
