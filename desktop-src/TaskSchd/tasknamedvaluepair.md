# <a name="tasknamedvaluepair-object"></a>Объект Таскнамедвалуепаир

Объект скрипта, который используется для создания пары "имя-значение", в которой имя связано со значением.

## <a name="members"></a>Элементы

Объект **таскнамедвалуепаир** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Объект **таскнамедвалуепаир** имеет следующие свойства.



| Свойство                                             | Описание                                                                            |
|:-----------------------------------------------------|:---------------------------------------------------------------------------------------|
| [**Имя**](tasknamedvaluepair-name.md)<br/>   | Возвращает или задает имя, связанное со значением в паре "имя-значение".<br/> |
| [**Значение**](tasknamedvaluepair-value.md)<br/> | Возвращает или задает значение, связанное с именем в паре "имя-значение".<br/> |



 

## <a name="remarks"></a>Комментарии

При чтении или записи собственного XML-кода для задачи пара "имя-значение" указывается с помощью элемента [**валуекуериес**](taskschedulerschema-valuequeries-eventtriggertype-element.md) схемы планировщик задач.
