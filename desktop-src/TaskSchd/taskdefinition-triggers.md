# <a name="taskdefinitiontriggers-property"></a>Таскдефинитион. Triggers, свойство

Для создания скриптов Возвращает или задает коллекцию триггеров, которые используются для запуска задачи.

## <a name="syntax"></a>Синтаксис


```VB
TaskDefinition.Triggers As TriggerCollection
```



## <a name="property-value"></a>Значение свойства

Коллекция триггеров, которые используются для запуска задачи.
