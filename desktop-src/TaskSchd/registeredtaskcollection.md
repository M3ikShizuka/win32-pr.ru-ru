# <a name="registeredtaskcollection-object"></a>Объект Регистередтаскколлектион

Объект скрипта, содержащий все зарегистрированные задачи.

## <a name="members"></a>Элементы

Объект **регистередтаскколлектион** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Объект **регистередтаскколлектион** имеет следующие свойства.



| Свойство                                                   | Тип доступа          | Описание                                                        |
|:-----------------------------------------------------------|:---------------------|:-------------------------------------------------------------------|
| [**Расчета**](registeredtaskcollection-count.md)<br/> | Только для чтения<br/> | Возвращает число зарегистрированных задач в коллекции.<br/>  |
| [**Элемент**](registeredtaskcollection-item.md)<br/>   | Только для чтения<br/> | Возвращает указанную зарегистрированную задачу из коллекции.<br/> |
