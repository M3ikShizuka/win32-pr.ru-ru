# <a name="taskdefinitionactions-property"></a>Таскдефинитион. Actions, свойство

Для создания скриптов Возвращает или задает коллекцию действий, выполняемых задачей.

## <a name="syntax"></a>Синтаксис


```VB
TaskDefinition.Actions As ActionCollection
```



## <a name="property-value"></a>Значение свойства

Коллекция действий, выполняемых задачей.
