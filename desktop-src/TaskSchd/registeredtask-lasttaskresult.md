# <a name="registeredtasklasttaskresult-property"></a>Регистередтаск. Ласттаскресулт, свойство

Для создания скриптов возвращает результаты, которые были возвращены при последнем запуске зарегистрированной задачи.

## <a name="syntax"></a>Синтаксис


```VB
RegisteredTask.LastTaskResult As String
```



## <a name="property-value"></a>Значение свойства

Результаты, которые были возвращены при последнем запуске зарегистрированной задачи.
