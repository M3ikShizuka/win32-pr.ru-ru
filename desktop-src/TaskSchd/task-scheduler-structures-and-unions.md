# <a name="task-scheduler-structures-and-unions"></a>планировщик задач структур и объединений

В этом разделе описываются структуры и объединения, используемые интерфейсами API планировщик задач.

Все структуры и объединения, приведенные ниже, используются планировщик задач 1,0.



| Имя                                               | Описание                                                                                                                     |
|----------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**ОБНОВЛЯЕТСЯ**](/windows/desktop/api/Mstask/ns-mstask-daily)                             | Определяет интервал в днях, в течение которого выполняется задача.                                                                          |
| [**монслидате**](/windows/desktop/api/Mstask/ns-mstask-monthlydate)                 | Определяет день месяца, в который будет выполняться задача.                                                                                 |
| [**монслидов**](/windows/desktop/api/Mstask/ns-mstask-monthlydow)                   | Определяет даты выполнения задачи по месяцам, неделям и дням недели.                                                     |
| [**\_триггер задачи**](/windows/desktop/api/Mstask/ns-mstask-task_trigger)              | Определяет время выполнения запланированного [*рабочего элемента*](w.md).                                                  |
| [**\_объединение типов триггеров \_**](/windows/desktop/api/Mstask/ns-mstask-trigger_type_union) | Определяет расписание вызова триггера в пределах члена **типа** структуры [**\_ триггера задачи**](/windows/desktop/api/Mstask/ns-mstask-task_trigger) . |
| [**Запуск**](/windows/desktop/api/Mstask/ns-mstask-weekly)                           | Определяет интервал (в неделях) между вызовами задачи.                                                                  |
