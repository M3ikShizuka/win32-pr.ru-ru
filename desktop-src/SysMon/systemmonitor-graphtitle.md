# <a name="systemmonitorgraphtitle-property"></a>Системмонитор. Графтитле, свойство

Возвращает или задает заголовок диаграммы.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property GraphTitle As String
```



## <a name="property-value"></a>Значение свойства

Заголовок диаграммы. Максимальная длина заголовка — 128 символов. Если заголовок превышает 128 символов, заголовок усекается.

**Windows XP и windows 2000:** Длина заголовка не ограничена.
