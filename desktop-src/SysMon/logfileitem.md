# <a name="logfileitem-object"></a>Объект Логфилеитем

Этот класс используется для получения пути к файлу журнала, к которому обращается элемент управления системного монитора.

Чтобы получить этот объект, вызовите [**файл файл_журнала. Item**](systemmonitor-logfiles-item.md).

## <a name="members"></a>Элементы

Объект **логфилеитем** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Объект **логфилеитем** имеет следующие свойства.



| Свойство                                    | Описание                                    |
|:--------------------------------------------|:-----------------------------------------------|
| [**Путь**](logfileitem-path.md)<br/> | Возвращает путь к файлу журнала.<br/> |
