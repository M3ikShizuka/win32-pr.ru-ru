# <a name="systemmonitorshowscalelabels-property"></a>Системмонитор. Шовскалелабелс, свойство

Возвращает или задает значение, определяющее, отображаются ли метки шкалы на вертикальной оси графа.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property ShowScaleLabels As Boolean
```



## <a name="property-value"></a>Значение свойства

Значение true указывает, что метки шкалы отображаются на вертикальной оси диаграммы. в противном случае — значение false. Значение этого свойства по умолчанию — true.
