# <a name="systemmonitorbackcolorctl-property"></a>Системмонитор. Баккколорктл, свойство

Возвращает или задает цвет фона элемента управления.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property BackColorCtl As stdole.OLE_COLOR
```



## <a name="property-value"></a>Значение свойства

Цвет фона элемента управления.
