# <a name="logfilescount-property"></a>Файл_журнала. Count, свойство

Число экземпляров [**логфилеитем**](logfileitem.md) в коллекции.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property Count As Long
```



## <a name="property-value"></a>Значение свойства

Число файлов журнала в коллекции.
