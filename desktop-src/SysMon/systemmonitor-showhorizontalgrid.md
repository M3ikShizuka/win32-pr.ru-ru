# <a name="systemmonitorshowhorizontalgrid-property"></a>Системмонитор. Шовхоризонталгрид, свойство

Возвращает или задает значение, определяющее, отображаются ли на диаграмме горизонтальные линии сетки.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property ShowHorizontalGrid As Boolean
```



## <a name="property-value"></a>Значение свойства

Значение true указывает, что горизонтальные линии сетки отображаются на графе; в противном случае — значение false. Значение этого свойства по умолчанию — false.
