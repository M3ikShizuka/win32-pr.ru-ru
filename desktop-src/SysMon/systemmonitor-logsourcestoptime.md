# <a name="systemmonitorlogsourcestoptime-property"></a>Системмонитор. Логсаурцестоптиме, свойство

Извлекает метку времени последнего значения счетчика из счетчика в коллекции счетчиков, которая была зарегистрирована в файлах журнала.

## <a name="syntax"></a>Синтаксис


```VB
Property LogSourceStopTime As Date
```



## <a name="property-value"></a>Значение свойства

Метка времени последнего значения счетчика из счетчика в коллекции счетчиков, которая была зарегистрирована в файлах журнала.
