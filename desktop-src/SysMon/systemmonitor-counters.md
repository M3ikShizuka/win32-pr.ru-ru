# <a name="systemmonitorcounters-property"></a>Системмонитор. Counters, свойство

Извлекает **счетчики** , содержащие коллекцию объектов [**каунтеритем**](counteritem.md) .

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property Counters As ICounters
```



## <a name="property-value"></a>Значение свойства

Объект **счетчиков** , содержащий коллекцию объектов [**каунтеритем**](counteritem.md) .

## <a name="remarks"></a>Комментарии

Это свойство по умолчанию объекта [**системмонитор**](systemmonitor.md) .
