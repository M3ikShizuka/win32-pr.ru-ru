# <a name="counteritemvisible-property"></a>Каунтеритем. Visible, свойство

Возвращает или задает значение, указывающее, отображается ли счетчик в графе.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
Property Visible As Boolean
```



## <a name="property-value"></a>Значение свойства

Значение true, если счетчик включен в график; в противном случае — значение false.
