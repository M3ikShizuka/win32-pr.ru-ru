# <a name="systemmonitorlogfilename-property"></a>Системмонитор. LogFileName, свойство

Возвращает или задает имя файла журнала, используемого в качестве источника значений счетчиков, отображаемых в системном мониторе.

> [!Note]  
> Это свойство стало устаревшим в свойстве [**файл_журнала**](systemmonitor-logfiles.md) .

 

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property LogFileName As String
```



## <a name="property-value"></a>Значение свойства

Путь к файлу журнала. Можно указать абсолютный, относительный или UNC-путь. Имя файла журнала должно иметь расширение CSV, TSV или BLG.

## <a name="exceptions"></a>Исключения



| Тип исключения                                  | Условие                                                              |
|-------------------------------------------------|------------------------------------------------------------------------|
| **System. Runtime. InteropServices. COMException** | Не удалось найти указанный файл. Значение Err. Number — 0xC0000BD1. |
| **System. ArgumentException**                    | Нельзя задавать пустую строку.                                    |



 

## <a name="remarks"></a>Комментарии

Это свойство возвращает имя файла журнала из первого члена коллекции файлов [**журнала**](systemmonitor-logfiles.md) .

Установка этого свойства завершится ошибкой, если коллекция файлов [**журнала**](systemmonitor-logfiles.md) содержит один или несколько элементов.

Для создания файлов журналов, добавляемых в эту коллекцию, необходимо использовать средство Logman.exe или оснастку MMC Perfmon. msc. Для Perfmon. msc журналы счетчиков находятся в папке **журналы и оповещения производительности**. Дополнительные сведения об использовании Logman.exe или Perfmon. msc см. в подокне **центра справки и поддержки**, которое можно найти по программе Logman.
