# <a name="systemmonitorreset-method"></a>Метод Системмонитор:: Reset

Удаляет все объекты [**каунтеритем**](counteritem.md) из объекта коллекции **Counters** и очищает графы и представления отчетов.

## <a name="syntax"></a>Синтаксис


```VB
Sub Reset()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
