# <a name="systemmonitoronsamplecollected-event"></a>Системмонитор. Онсамплеколлектед, событие

Уведомляет вас при сборе значений выборки для счетчиков.

## <a name="syntax"></a>Синтаксис


```VB
SystemMonitor.OnSampleCollected()
```



## <a name="parameters"></a>Параметры

У этого события нет параметров.

## <a name="return-value"></a>Возвращаемое значение

Это событие не возвращает значение.
