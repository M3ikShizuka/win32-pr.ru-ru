# <a name="systemmonitorpaste-method"></a>Системмонитор: метод:P вставить

Добавляет список счетчиков, скопированных в буфер обмена, в текущую коллекцию счетчиков.

## <a name="syntax"></a>Синтаксис


```VB
Sub Paste()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
