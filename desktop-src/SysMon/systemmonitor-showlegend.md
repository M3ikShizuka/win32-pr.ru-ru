# <a name="systemmonitorshowlegend-property"></a>Системмонитор. Шовлеженд, свойство

Возвращает или задает значение, определяющее, отображаются ли условные обозначения. В условных обозначениях перечислены счетчики и показан цвет, используемый для графика каждого счетчика.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property ShowLegend As Boolean
```



## <a name="property-value"></a>Значение свойства

Значение true указывает, что условные обозначения отображаются в элементе управления; в противном случае — значение false. Значение этого свойства по умолчанию — true.
