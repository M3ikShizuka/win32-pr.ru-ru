# <a name="systemmonitorappearance-property"></a>Системмонитор. Appearance, свойство

Извлекает или задает внешний вид элемента управления для включения или исключения трехмерных эффектов отображения.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property Appearance As Long
```



## <a name="property-value"></a>Значение свойства

Значение вида, определяющее, закрашивается ли элемент управления трехмерными эффектами.

Для этого свойства можно задать одно из следующих значений.



| Значение                                                                        | Значение                                                                                  |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| <dl> <dt>0</dt> </dl> | Закрашивает элемент управления без визуальных эффектов.<br/>                                    |
| <dl> <dt>1</dt> </dl> | Закрашивает элемент управления с помощью трехмерных эффектов. Это значение по умолчанию.<br/> |



 

## <a name="exceptions"></a>Исключения



| Тип исключения               | Условие                                    |
|------------------------------|----------------------------------------------|
| **System. ArgumentException** | Указано недопустимое значение внешнего вида. |



 

## <a name="remarks"></a>Комментарии

Это внешнее свойство. Значение этого свойства определяется контейнером. Установка значения этого свойства может повлиять на иллюзию элемента управления и контейнера на одно приложение.
