# <a name="systemmonitorminimumscale-property"></a>Системмонитор. Минимумскале, свойство

Возвращает или задает минимальное значение вертикальной оси (Y) диаграммы.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property MinimumScale As Long
```



## <a name="property-value"></a>Значение свойства

Положительное минимальное значение вертикальной оси графа. По умолчанию минимальное значение вертикального масштаба равно 0. Наибольшее значение, которое можно установить как минимальное значение, равное значению меньше [**максимумскале**](systemmonitor-maximumscale.md) .

## <a name="remarks"></a>Комментарии

Элемент управления автоматически корректирует расположение чисел шкалы, отображаемых на вертикальном масштабировании, в соответствии с размером отображаемого элемента управления.
