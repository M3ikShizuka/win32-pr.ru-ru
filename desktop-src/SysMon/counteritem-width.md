# <a name="counteritemwidth-property"></a>Каунтеритем. Width, свойство

Возвращает или задает толщину линии, используемую для отображения значения счетчика.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
Property Width As Long
```



## <a name="property-value"></a>Значение свойства

Толщина линии, используемой в линейном графике. Допустимые значения находятся в диапазоне от 1 до 3, где 1 (значение по умолчанию) является самой короткой шириной.

**до Windows Vista:** Допустимы значения в диапазоне от 1 до 9. если приложение будет выполняться в Windows Vista, не используйте значение ширины больше 3.

## <a name="exceptions"></a>Исключения



| Тип исключения               | Условие                              |
|------------------------------|----------------------------------------|
| **System.ArgumentException** | Указана недопустимая толщина линии. |



 

## <a name="remarks"></a>Комментарии

Ширина линии по умолчанию для первых 16 добавляемых счетчиков — 1. Ширина линии по умолчанию для следующих 16 счетчиков (счетчиков 17-32), которые вы добавили, равна 2. Ширина линии по умолчанию для следующих 16 счетчиков (счетчики 33 – 48), которые вы добавили, равна 3. После этого СИСМОН случайным образом выбирает толщину линии. Дополнительные сведения см. в разделе [**каунтеритем. Color**](counteritem-color.md).

Чтобы задать [**стиль линии**](counteritem-linestyle.md) , отличный от сплошной, ширина должна быть равна 1. Если указать значение больше 1 и указать несплошной стиль линии, СИСМОН игнорирует указанную толщину линии.
