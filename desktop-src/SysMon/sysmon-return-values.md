# <a name="sysmon-return-values"></a>Возвращаемые значения СИСМОН

Ниже приведен список возвращаемых значений системного монитора, определенных в Смонмсг. h.



| Возвращаемое значение                                       | Описание                                                                                                                                                                                                                                  |
|----------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_ \_ \_ Путь к счетчику состояния смон дупл \_ (0xC0001388)     | Коллекция счетчиков уже содержит указанный счетчик.                                                                                                                                                                               |
| \_Состояние смон \_ нет \_ \_ объекта сисмон (0xC0001389)      | Параметры не содержат никаких полных объектов HTML системного монитора.                                                                                                                                                                        |
| \_Состояние смон \_ слишком \_ мало \_ выборок (0xC000138A)       | Указанный файл журнала содержит менее двух выборок данных.                                                                                                                                                                                 |
| \_ \_ \_ Предельный размер файла журнала состояния смон \_ \_ (0xC000138B)  | Указанный файл журнала превышает ограничения на размер элемента управления системного монитора. Если в данный момент выбран файл журнала, выберите текущее действие в качестве источника данных, чтобы выгрузить текущий файл журнала, а затем повторно выбрать указанный файл журнала. |
| \_ \_ \_ Источник данных файла журнала состояния смон \_ \_ (0xC000138C) | Чтобы добавить новый файл журнала в источник данных, необходимо изменить тип источника данных на тип, отличный от Сисмонлогфилес.                                                                                                                          |
| СМОН \_ состояния \_ дупл \_ \_ \_ путь к файлу журнала (0xC000138D)   | Коллекция файлов журнала уже содержит указанный файл журнала.                                                                                                                                                                             |



 

Описание дополнительных возвращаемых значений, возвращаемых системным монитором, см. в разделе [коды системных ошибок](/windows/desktop/Debug/system-error-codes) и [вспомогательные коды ошибок данных производительности](/windows/desktop/PerfCtrs/checking-pdh-interface-return-values).
