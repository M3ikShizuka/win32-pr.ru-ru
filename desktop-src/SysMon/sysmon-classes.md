# <a name="sysmon-classes"></a>Классы СИСМОН

Для настройки элемента управления "системный монитор (СИСМОН)" можно использовать следующие классы.



| Имя класса                             | Описание                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**каунтеритем**](counteritem.md)     | Предоставляет сведения о свойстве элемента счетчика.                                                                  |
| [**Counters**](counters.md)           | Предоставляет методы для управления коллекцией объектов [**каунтеритем**](counteritem.md) .                |
| [**логфилеитем**](logfileitem.md)     | Предоставляет свойство для получения пути к одному файлу журнала, к которому обращается элемент управления системного монитора.     |
| [**LogFiles**](logfiles.md)           | Предоставляет методы и свойства для управления коллекцией объектов [**логфилеитем**](logfileitem.md) . |
| [**системмонитор**](systemmonitor.md) | Предоставляет методы и свойства, используемые для настройки элемента управления системного монитора.                            |
