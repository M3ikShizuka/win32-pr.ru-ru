# <a name="systemmonitormanualupdate-property"></a>Системмонитор. Мануалупдате, свойство

Возвращает или задает значение, указывающее, будет ли содержимое системного монитора обновляться вручную или автоматически с указанными интервалами.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property ManualUpdate As Boolean
```



## <a name="property-value"></a>Значение свойства

Значение true указывает, что граф обновляется вручную. Значение false указывает, что граф обновляется автоматически на основе интервала обновления, указанного в [**системмонитор. упдатеинтервал**](systemmonitor-updateinterval.md).

## <a name="remarks"></a>Комментарии

По умолчанию граф обновляется автоматически.

Чтобы обновить граф вручную, вызовите метод [**системмонитор. коллектсампле**](systemmonitor-collectsample.md) .
