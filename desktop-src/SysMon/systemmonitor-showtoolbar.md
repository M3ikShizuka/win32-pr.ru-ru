# <a name="systemmonitorshowtoolbar-property"></a>Системмонитор. Шовтулбар, свойство

Возвращает или задает значение, определяющее, отображается ли панель инструментов в элементе управления.

## <a name="syntax"></a>Синтаксис


```VB
Property ShowToolbar As Boolean
```



## <a name="property-value"></a>Значение свойства

Значение true указывает, что панель инструментов отображается на элементе управления; в противном случае — значение false. Значение этого свойства по умолчанию — true.

## <a name="remarks"></a>Комментарии

Панель инструментов позволяет пользователю добавлять или удалять счетчики, выбирать тип графа, используемого для отображения данных счетчиков, и изменять другие свойства элемента управления.
