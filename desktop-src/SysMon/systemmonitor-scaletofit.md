# <a name="systemmonitorscaletofit-method"></a>Системмонитор. Скалетофит, метод

Масштабирование значений счетчиков в соответствии с графиком.

## <a name="syntax"></a>Синтаксис


```VB
SystemMonitor.ScaleToFit( _
  ByVal selectedCountersOnly As Boolean _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*селектедкаунтерсонли* \[ окне\]
</dt> <dd>

Значение true, чтобы масштабировать только выбранные счетчики; в противном случае значение false для масштабирования всех счетчиков.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Этот метод очистит представление графа. Затем СИСМОН использует коэффициент масштабирования, заданный для каждого счетчика, для перерисовки графа, если источник данных является файлом журнала, или для построения диаграммы новых значений счетчиков, если источник данных имеет активность в реальном времени.
