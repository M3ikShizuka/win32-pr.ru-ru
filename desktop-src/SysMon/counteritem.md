# <a name="counteritem-object"></a>Объект Каунтеритем

Этот класс используется для извлечения пути и значения счетчика, а также для указания цвета, используемого для отображения счетчика.

Чтобы получить этот объект, вызовите [**Counters. Item**](counters-item.md).

## <a name="members"></a>Элементы

Объект **каунтеритем** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **каунтеритем** содержит эти методы.



| Метод                                             | Описание                                                                    |
|:---------------------------------------------------|:-------------------------------------------------------------------------------|
| [**жетдатаат**](counteritem-getdataat.md)         | Извлекает данные счетчиков из определенной точки линейного графика.<br/> |
| [**Статистика**](counteritem-getstatistics.md) | Возвращает среднее, максимальное и минимальное значения счетчика.<br/> |
| [**GetValue**](counteritem-getvalue.md)           | Извлекает Последнее значение счетчика.<br/>                            |



 

### <a name="properties"></a>Свойства

Объект **каунтеритем** имеет следующие свойства.



| Свойство                                                  | Описание                                                                     |
|:----------------------------------------------------------|:--------------------------------------------------------------------------------|
| [**Цвет**](counteritem-color.md)<br/>             | Возвращает или задает цвет, используемый для отображения значения счетчика.<br/>         |
| [**линестиле**](counteritem-linestyle.md)<br/>     | Возвращает или задает стиль линии, используемый для отображения значения счетчика.<br/>    |
| [**Путь**](counteritem-path.md)<br/>               | Получает путь к счетчику.<br/>                                   |
| [**скалефактор**](counteritem-scalefactor.md)<br/> | Возвращает или задает коэффициент масштабирования, используемый для отображения значения счетчика.<br/>  |
| [**Выбрано**](counteritem-selected.md)<br/>       | Возвращает или задает значение, указывающее, выбран ли счетчик.<br/> |
| [**Значение**](counteritem-value.md)<br/>             | Извлекает текущее значение счетчика.<br/>                          |
| [**Visible**](counteritem-visible.md)<br/>         | Возвращает или задает значение, указывающее, отображается ли счетчик в графе.<br/>  |
| [**Ширина**](counteritem-width.md)<br/>             | Возвращает или задает толщину линии, используемую для отображения значения счетчика.<br/>    |



 

## <a name="remarks"></a>Комментарии

[**Значение**](counteritem-value.md) является свойством по умолчанию **каунтеритем**.
