# <a name="counters-collection"></a>Коллекция Counters

Этот класс используется для управления коллекцией объектов [**каунтеритем**](counteritem.md) .

Чтобы получить этот объект, вызовите [**системмонитор. Counters**](systemmonitor-counters.md).

## <a name="members"></a>Элементы

Коллекция **Counters** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Коллекция **Counters** содержит эти методы.



| Метод                            | Описание                                                                           |
|:----------------------------------|:--------------------------------------------------------------------------------------|
| [**Включить**](counters-add.md)       | Добавляет экземпляр [**каунтеритем**](counteritem.md) в коллекцию.<br/>      |
| [**Отменит**](counters-remove.md) | Удаляет экземпляр [**каунтеритем**](counteritem.md) из коллекции.<br/> |



 

### <a name="properties"></a>Свойства

Коллекция **Counters** содержит эти свойства.



| Свойство.                                   | Описание                                                                                         |
|:-------------------------------------------|:----------------------------------------------------------------------------------------------------|
| [**Count**](counters-count.md)<br/> | Возвращает количество экземпляров [**каунтеритем**](counteritem.md) в коллекции.<br/>  |
| [**Элемент**](counters-item.md)<br/>   | Извлекает указанный экземпляр [**каунтеритем**](counteritem.md) из коллекции.<br/> |



 

## <a name="remarks"></a>Remarks

Объект **Counters** является свойством по умолчанию объекта [**системмонитор**](systemmonitor.md) .

Добавьте в эту коллекцию счетчики, которые необходимо отобразить. СИСМОН извлекает значения счетчиков либо из системы, либо из файла журнала в зависимости от указанного [**источника данных**](systemmonitor-datasourcetype.md) .
