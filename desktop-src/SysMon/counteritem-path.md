# <a name="counteritempath-property"></a>Каунтеритем. Path, свойство

Получает путь к счетчику.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property Path As String
```



## <a name="property-value"></a>Значение свойства

Путь к счетчику, например "процессор ( \_ всего) \\ % загруженности процессора".
