# <a name="systemmonitorenabletooltips-property"></a>Системмонитор. Енаблетултипс, свойство

Возвращает или задает значение, определяющее, отображается ли всплывающая подсказка при наведении указателя мыши на счетчик в одном из представлений графа (не поддерживается для представления отчета).

## <a name="syntax"></a>Синтаксис


```VB
Property EnableToolTips As Boolean
```



## <a name="property-value"></a>Значение свойства

Значение true показывает всплывающую подсказку со сведениями о счетчике; в противном случае — значение false.

## <a name="remarks"></a>Комментарии

Для линейных графиков подсказка привязана к точке данных, расположенной ближе всего к мыши. Для линейчатых диаграмм и гистограмм подсказка представляет общее значение данных линейки, а не точку внутри линии.

Всплывающая подсказка содержит различные сведения, основанные на источнике данных. Для файлов журнала подсказка содержит путь к счетчику, значение счетчика, отметку времени, количество выборок данных, включаемых в точку данных, и минимальное и максимальное значения данных.

Для действий в реальном времени подсказка содержит путь к счетчику, значение счетчика и отметку времени.
