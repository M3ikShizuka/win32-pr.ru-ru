# <a name="systemmonitorlogsourcestarttime-property"></a>Системмонитор. Логсаурцестарттиме, свойство

Возвращает метку времени самого раннего значения счетчика из счетчика в коллекции счетчиков, которая была зарегистрирована в файлах журнала.

## <a name="syntax"></a>Синтаксис


```VB
Property LogSourceStartTime As Date
```



## <a name="property-value"></a>Значение свойства

Метка времени самого раннего значения счетчика из счетчика в коллекции счетчиков, которая была зарегистрирована в файлах журнала.
