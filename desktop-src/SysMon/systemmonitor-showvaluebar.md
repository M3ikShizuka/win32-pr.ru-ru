# <a name="systemmonitorshowvaluebar-property"></a>Системмонитор. Шоввалуебар, свойство

Возвращает или задает значение, определяющее, отображается ли в элементе управления Строка значений (набор статистических значений ниже диаграммы).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property ShowValueBar As Boolean
```



## <a name="property-value"></a>Значение свойства

Значение true указывает, что панель значений отображается в элементе управления; в противном случае — false. Значение этого свойства по умолчанию — true.

## <a name="remarks"></a>Комментарии

Отображаемая статистика включает Последнее значение, среднее значение и минимальное и максимальное значения текущего выбранного счетчика за отображаемый интервал времени. Чтобы выбрать отображаемые значения счетчика, пользователь должен выбрать счетчик на панели условных обозначений элемента управления.
