# <a name="whats-new-in-system-monitor"></a>Новые возможности системного монитора

В следующей таблице показано, что нового в каждом выпуске системного монитора (СИСМОН).



| Версия     | Описание функций                                                                                                                                                                                                                                                                                                                              |
|-------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Версия 3,7 | Эта версия добавляет новые типы графов, возможность выбора нескольких счетчиков, получения значений счетчиков из точки на диаграмме, сохранения значений счетчиков в виде графа в файле журнала, а также возможность непрерывной прокрутки линейного графа в окне графа, а не на саму себя. Входит в Windows Server 2008 и Windows Vista.<br/> |
