# <a name="counteritemcolor-property"></a>Каунтеритем. Color, свойство

Возвращает или задает цвет, используемый для отображения значения счетчика.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
Property Color As stdole.OLE_COLOR
```



## <a name="property-value"></a>Значение свойства

Цвет, используемый для отображения значения счетчика.

## <a name="remarks"></a>Комментарии

Если не указать используемый цвет, СИСМОН выбирает цвета для первых шестнадцати счетчиков. Если указать более шестнадцати счетчиков, СИСМОН будет использовать те же цвета, что и первые шестнадцати счетчиков. Чтобы отличать счетчики друг от друга, СИСМОН изменяет [**стиль линии**](counteritem-linestyle.md) , используемый для каждой кратности 16 счетчиков до первых счетчиков 80. Например, первые шестнадцати счетчиков используют сплошной стиль линии, следующие шестнадцать используют тип штриховой линии и т. д. Затем СИСМОН устанавливает [**толщину линии**](counteritem-width.md) равным 2 для счетчиков 81-96 и 3 для счетчиков 96-112. Если коллекция содержит более 112 счетчиков, счетчики будут содержать одинаковые значения цвета, стиля линии и ширины.

**До Windows Vista:** Если не указать используемый цвет, СИСМОН выбирает цвета для первых шестнадцати счетчиков. Если указать более шестнадцати счетчиков, СИСМОН будет использовать те же цвета, что и первые шестнадцати счетчиков. Чтобы отличать счетчики друг от друга, СИСМОН увеличивает [**ширину линии**](counteritem-width.md) первых трех счетчиков, которые получают одно и то же назначение цвета. Если более трех счетчиков используют один и тот же цвет, СИСМОН случайным образом выбирает толщину линии, используемую для счетчика.
