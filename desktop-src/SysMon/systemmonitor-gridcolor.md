# <a name="systemmonitorgridcolor-property"></a>Системмонитор. Гридколор, свойство

Возвращает или задает цвет линий сетки, используемых в графе.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property GridColor As stdole.OLE_COLOR
```



## <a name="property-value"></a>Значение свойства

Цвет горизонтальных и вертикальных линий сетки, используемых в графе, если выбраны свойства [**шовхоризонталгрид**](systemmonitor-showhorizontalgrid.md) или [**шоввертикалгрид**](systemmonitor-showverticalgrid.md) .
