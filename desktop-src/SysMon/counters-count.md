# <a name="counterscount-property"></a>Свойство Counters. Count

Возвращает количество экземпляров [**каунтеритем**](counteritem.md) в коллекции.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
Property Count As Long
```



## <a name="property-value"></a>Значение свойства

Число счетчиков в коллекции.
