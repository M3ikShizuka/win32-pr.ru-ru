# <a name="windows-libraries"></a>Библиотеки Windows

в Windows 7 представлены библиотеки, которые предоставляют пользователям единое, согласованное представление файлов, даже если эти файлы хранятся в разных расположениях. Библиотеки могут быть настроены и упорядочены пользователем, а библиотеки могут содержать папки, находящиеся на компьютере пользователя, а также папки, к которым предоставлен общий доступ по сети. Библиотеки представляют собой более простое представление базовой системы хранения, так как для пользователя файлы и папки в библиотеке отображаются в одном представлении независимо от того, где они физически хранятся.

разработчики, пишущие новые программы в Windows 7, должны использовать библиотеки как средства, с помощью которых пользователи взаимодействуют с файлами, используемыми программой. использование библиотек в программе предоставит пользователям более понятную, более простую и эффективную работу в Windows 7.

Разработчики также должны проанализировать существующие программы и при необходимости обновить их для работы с библиотеками. Поскольку библиотеки не являются частью файловой системы, API-интерфейсы на основе файловой системы не будут иметь доступа к библиотекам, которые могли быть настроены пользователем.

Программы, которые в настоящее время разрешают пользователям хранить свое содержимое в разных папках или на разных компьютерах, будут использовать наиболее выгодные преимущества при добавлении поддержки библиотек. Библиотеки упрощают управление содержимым в различных местах хранения для разработчика и пользователя.

В этом руководство описываются дополнительные сведения о библиотеках, о том, как программы могут работать с библиотеками, а также некоторые способы использования библиотек для улучшения работы пользователей.
