# <a name="ishelldispatch3-object"></a>Объект IShellDispatch3

Расширяет объект [**IShellDispatch2**](ishelldispatch2-object.md) . **IShellDispatch3** поддерживает один новый метод в дополнение к свойствам и методам, поддерживаемым **IShellDispatch2**.

> [!Note]  
> **IShellDispatch3** реализован и доступен через объект [**Shell**](shell.md) .

 

## <a name="members"></a>Элементы

Объект **IShellDispatch3** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Объект **IShellDispatch3** содержит эти методы.



| Метод                                             | Описание                                                  |
|:---------------------------------------------------|:-------------------------------------------------------------|
| [**аддторецент**](ishelldispatch3-addtorecent.md) | Добавляет файл в список последних использованных файлов (MRU).<br/> |
