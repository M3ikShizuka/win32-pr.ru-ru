# <a name="tabthumbnails-sample"></a>Пример TabThumbnails

Демонстрируется, как приложение может предоставить несколько целевых объектов переключения (как для вкладок) в таскбанд и как предоставить их эскизы.

В этом разделе содержатся следующие подразделы.

-   [Requirements](#requirements)
-   [Загрузка образца](#downloading-the-sample)
-   [Создание примера](#building-the-sample)
-   [Запуск примера](#running-the-sample)

## <a name="requirements"></a>Требования



| Продукт                                | Минимальная версия продукта |
|----------------------------------------|-------------------------|
| Windows                                | Windows 7               |
| Windows SDK | 7.0                     |

## <a name="building-the-sample"></a>Построение образца

Чтобы построить образец из командной строки, выполните следующую команду:

1.  Откройте окно командной строки и перейдите в каталог проекта **табсумбнаилс** .
2.  Введите `msbuild TabThumbnails.sln`.

чтобы создать пример с использованием Microsoft Visual Studio (предпочтительно):

1.  откройте обозреватель Windows и перейдите в каталог проекта **табсумбнаилс** .
2.  Дважды щелкните значок файла Табсумбнаилс. sln, чтобы открыть проект в Visual Studio.
3.  В меню **Построение** выберите пункт **Построить решение**.

## <a name="running-the-sample"></a>Запуск примера

1.  перейдите в каталог, содержащий новый исполняемый файл, используя командную строку или обозреватель Windows.
2.  В командной строке введите `TabThumbnails.exe` . кроме того, в обозревателе Windows дважды щелкните значок TabThumbnails.exe.
