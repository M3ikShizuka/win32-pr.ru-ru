# <a name="iuseridentitygetidentityfolder-method"></a>Метод Иусеридентити:: Жетидентитифолдер

\[**Жетидентитифолдер** не поддерживается и может быть изменен или недоступен в будущем. Вместо этого используйте [учетные записи пользователей с быстрым переключением пользователей и удаленный рабочий стол](fastuserswitching.md).\]

Возвращает файловую папку, связанную с этим удостоверением.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetIdentityFolder(
  [in] DWORD dwFlags,
  [in] WCHAR *pszPath,
  [in] ULONG ulBuffSize
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Тип: **DWORD**

Обязательный. Значение типа, указывающее, является ли папка, связанная с этим удостоверением, перемещаемой. Необходимо установить одно из следующих значений.

<dt>



 (GIF \_ ПЕРЕМЕЩАЕМая \_ Папка)


</dt> <dd>

Папка является перемещаемой.

</dd> <dt>



 (GIF \_ НЕ \_ перемещаемая \_ Папка)


</dt> <dd>

Папка является локальной.

</dd> </dl> </dd> <dt>

*псзпас* \[ окне\]
</dt> <dd>

Тип: **WCHAR \***

Указатель на строку расширенных символов, которая получает путь к папке.

</dd> <dt>

*улбуффсизе* \[ окне\]
</dt> <dd>

Тип: **ulong**

Значение типа, указывающее размер *псзпас*.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

Если этот метод завершается успешно, возвращается значение **S \_ ОК**. В противном случае возвращается код ошибки **HRESULT** .
