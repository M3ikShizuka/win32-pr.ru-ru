# <a name="folderitemverbs_newenum-method"></a>Фолдеритемвербс. \_ Метод NewEnum

Создает и возвращает новый объект [**фолдеритемвербс**](folderitemverbs.md) , являющийся копией этого объекта фолдеритемвербс.

## <a name="syntax"></a>Синтаксис


```JScript
FolderItemVerbs._NewEnum()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
