# <a name="application-scenarios"></a>Сценарии приложения

К приложениям и службам, которые могут использовать диспетчер синхронизации, относятся следующие:

-   Microsoft Office приложения, которым требуется подготовить файлы для автономного использования.
-   Кэширование на стороне клиента, позволяющее кэшировать файлы локально.
-   Браузеры, которые могут локально кэшировать HTML-страницы.
-   Программы электронной почты, которые могут кэшировать сообщения электронной почты локально.
-   Базы данных, которые могут хранить данные локально.
