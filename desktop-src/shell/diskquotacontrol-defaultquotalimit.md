# <a name="diskquotacontroldefaultquotalimit-property"></a>Дисккуотаконтрол. Дефаулткуоталимит, свойство

Задает или возвращает предельную квоту по умолчанию.

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```JScript
iDefaultQuotaLimit = DiskQuotaControl.DefaultQuotaLimit
DiskQuotaControl.DefaultQuotaLimit = iDefaultQuotaLimit
```



## <a name="property-value"></a>Значение свойства

**Целочисленное** значение, указывающее или получающее предельную квоту по умолчанию для новых пользователей в байтах.
