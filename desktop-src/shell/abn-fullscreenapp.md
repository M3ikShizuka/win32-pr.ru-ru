# <a name="abn_fullscreenapp-message"></a>\_Сообщение АБН фуллскринапп

Уведомляет панель приложений, когда полноэкранное приложение открывается или закрывается. Это уведомление отправляется в виде определяемого приложением сообщения, которое задается новым сообщением [**АБМ \_**](abm-new.md) .


```C++

ABN_FULLSCREENAPP 

    fOpen = (BOOL) lParam; 

            
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*fOpen* 
</dt> <dd>

Флаг, указывающий, будет ли полноэкранное приложение открываться или закрываться. Этот параметр имеет **значение true** , если приложение открывается, или **значение false** , если оно закрывается.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Примечания

При открытии полноэкранного приложения панель приложений должен быть направлен в нижнюю часть z-порядка. Когда он закрывается, панель приложений должен восстановить свою координату z-порядка.
