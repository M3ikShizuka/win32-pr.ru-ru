# <a name="iuseridentityopenidentityregkey-method"></a>Метод Иусеридентити:: Опенидентитирегкэй

\[Метод **иусеридентити:: опенидентитирегкэй** можно использовать в Windows 2000. В Windows XP эта функция заменена [учетными записями пользователей с быстрым переключением пользователей и удаленный рабочий стол](fastuserswitching.md)и может быть изменена или недоступна в последующих версиях.\]

Не рекомендуется. Извлекает ключ в реестре, соответствующий этому удостоверению пользователя.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT OpenIdentityRegKey(
  [in]  DWORD dwDesiredAccess,
  [out] HKEY  *phKey
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*двдесиредакцесс* \[ окне\]
</dt> <dd>

Тип: **DWORD**

Значение, описывающее уровень доступа, запрошенный для раздела реестра.

</dd> <dt>

*фкэй* \[ заполняет\]
</dt> <dd>

Тип: **hKey \** _

Указатель, получающий раздел реестра.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: _ *HRESULT**

Результат запроса реестра. В случае успеха возвращается значение **S \_ ОК**. В противном случае будет возвращен один из следующих кодов ошибок.



| Код возврата                                                                                            | Описание                                                        |
|--------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------|
| <dl> <dt>**\_ \_ не найдено удостоверение E \_**</dt> </dl> | С этим удостоверением не связан раздел реестра.<br/> |
| <dl> <dt>**\_Ошибка E**</dt> </dl>                 | Не удалось получить доступ к разделу реестра.<br/>             |



 

## <a name="remarks"></a>Примечания

Параметр *двдесиредакцесс* — это стандартный дескриптор безопасности доступа к реестру, описывающий доступ к разделу реестра. Дополнительные сведения о безопасности реестра, включая список допустимых значений для этого параметра, см. в разделе [раздел реестра Security and Access Rights](../sysinfo/registry-key-security-and-access-rights.md).
