# <a name="ishelldispatch6-object"></a>Объект IShellDispatch6

Расширяет объект [**IShellDispatch5**](ishelldispatch5.md) . В дополнение к свойствам и методам, поддерживаемым **IShellDispatch5**, **IShellDispatch6** добавляет метод, отображающий панель поиска приложений.

> [!Note]  
> **IShellDispatch6** реализован и доступен через объект [**Shell**](shell.md) .

 

## <a name="members"></a>Элементы

Объект **IShellDispatch6** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Объект **IShellDispatch6** содержит эти методы.



| Метод                                                 | Описание                                                                                                                  |
|:-------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------|
| [**сеарчкомманд**](ishelldispatch6-searchcommand.md) | Отображает панель поиска приложений, которая обычно появляется при вводе условия поиска с начального экрана.<br/> |
