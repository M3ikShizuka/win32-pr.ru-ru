# <a name="folder2-object"></a>Объект Folder2

Расширяет объект [**Folder**](folder.md) для поддержки автономных папок.

## <a name="members"></a>Элементы

Объект **folder2** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **folder2** содержит эти методы.



| Метод                                                                 | Описание                                                                          |
|:-----------------------------------------------------------------------|:-------------------------------------------------------------------------------------|
| [**дисмисседвебвиевбаррикаде**](folder2-dismissedwebviewbarricade.md) | Вызывается в ответ на веб-представление баррикаде, закрытое пользователем.<br/> |
| [**Полнит**](folder2-synchronize.md)                             | Синхронизирует все автономные файлы в папке.<br/>                             |



 

### <a name="properties"></a>Свойства

Объект **folder2** имеет следующие свойства.



| Свойство                                                                            | Тип доступа          | Описание                                                               |
|:------------------------------------------------------------------------------------|:---------------------|:--------------------------------------------------------------------------|
| [**хаветошоввебвиевбаррикаде**](folder2-havetoshowwebviewbarricade.md)<br/> | Только для чтения<br/> | В настоящее время не поддерживается.<br/>                                       |
| [**оффлинестатус**](folder2-offlinestatus.md)<br/>                           | Только для чтения<br/> | Содержит автономное состояние папки.<br/>                     |
| [**Самообслуживания**](folder2-self.md)<br/>                                             | Только для чтения<br/> | Содержит объект [**FolderItem**](folderitem.md) папки.<br/> |
