# <a name="shelllinkobject-object"></a>Объект Шелллинкобжект

Управляет связями оболочки. Этот объект делает большую часть функциональных возможностей интерфейса [**ишелллинк**](/windows/desktop/api/Shobjidl_core/nn-shobjidl_core-ishelllinka) , доступного для сценариев приложений.

## <a name="members"></a>Элементы

Объект **шелллинкобжект** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **шелллинкобжект** содержит эти методы.



| Метод                                                     | Описание                                                                                    |
|:-----------------------------------------------------------|:-----------------------------------------------------------------------------------------------|
| [**жетиконлокатион**](shelllinkobject-geticonlocation.md) | Возвращает расположение значка, назначенного ссылке.<br/>                                 |
| [**Проблемы**](shelllinkobject-resolve.md)                 | Выполняет поиск целевого объекта для ссылки на оболочку, даже если целевой объект был перемещен или переименован.<br/> |
| [**Сохранить**](shelllinkobject-save.md)                       | Сохраняет все изменения в ссылке.<br/>                                                      |
| [**сетиконлокатион**](shelllinkobject-seticonlocation.md) | Задает расположение значка, назначенного ссылке.<br/>                                 |



 

### <a name="properties"></a>Свойства

Объект **шелллинкобжект** имеет следующие свойства.



| Свойство                                                                | Тип доступа           | Описание                                                                                               |
|:------------------------------------------------------------------------|:----------------------|:----------------------------------------------------------------------------------------------------------|
| [**Аргументы**](shelllinkobject-arguments.md)<br/>               | Чтение/запись<br/> | Содержит аргументы ссылки.<br/>                                                                   |
| [**Описание**](shelllinkobject-description.md)<br/>           | Чтение/запись<br/> | Возвращает или задает описание ссылки.<br/>                                                      |
| [**Сочетание клавиш**](shelllinkobject-hotkey.md)<br/>                     | Чтение/запись<br/> | Возвращает или задает сочетание клавиш для ссылки.<br/>                                               |
| [**Путь**](shelllinkobject-path.md)<br/>                         | Чтение/запись<br/> | Возвращает или задает путь к объекту ссылки.<br/>                                                      |
| [**Возвращающий showcommand**](shelllinkobject-showcommand.md)<br/>           | Чтение/запись<br/> | Возвращает или задает начальное состояние отображения команды ссылки (с указанием размера, сворачивания или развертывания).<br/> |
| [**WorkingDirectory**](shelllinkobject-workingdirectory.md)<br/> | Чтение/запись<br/> | Возвращает или задает рабочий каталог, указанный в ссылке.<br/>                                      |
