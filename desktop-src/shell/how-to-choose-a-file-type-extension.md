# <a name="how-to-choose-a-file-type-extension"></a>Выбор расширения типа файлов

Типы файлов идентифицируются по расширению имени файла. Перед определением нового типа файлов необходимо сначала выбрать расширение имени файла.

## <a name="instructions"></a>Инструкции

### <a name="step-1"></a>Шаг 1.

Выберите длинное расширение имени файла, которое будет использоваться для задания типа файла. Дополнительные сведения см. в разделе [обработчики типов файлов](fa-file-extensions.md).

### <a name="step-2"></a>Шаг 2.

Проверьте базу данных MIME [(IANA)](https://www.iana.org/assignments/media-types) , чтобы определить, связано ли расширение имени файла с другим приложением.

### <a name="step-3"></a>Шаг 3.

Определите тип MIME при определении нового типа общедоступного файла и зарегистрируйте его с помощью IANA.
