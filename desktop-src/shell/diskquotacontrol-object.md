# <a name="diskquotacontrol-object"></a>Объект Дисккуотаконтрол

Позволяет администратору управлять свойствами дисковой квоты тома. Файловая система NTFS позволяет администратору управлять использованием дискового пространства на общем томе, выделяя заданный объем места на диске или *предельную квоту* для каждого пользователя. Этот объект можно использовать для задания предельной квоты по умолчанию, которая будет автоматически назначена всем новым пользователям.

## <a name="remarks"></a>Remarks

Администратор может использовать объект **дисккуотаконтрол** для выполнения ряда задач, включая следующие:

-   Включение и отключение системы дисковой квоты тома.
-   Получение состояния системы квот на томе.
-   Запрет места на диске для пользователей, превышающих предельную квоту.
-   Указание значения порога предупреждения по умолчанию и предельной квоты, которые будут назначены новым пользователям.
-   Добавление и удаление пользователей.

Объект **дисккуотаконтрол** позволяет задать глобальные значения по умолчанию для тома для таких свойств, как ограничение квоты. Однако каждый пользователь представлен объектом [**дидисккуотаусер**](didiskquotauser-object.md) , который можно использовать для указания отдельных параметров квоты.

Существует несколько способов получить объект [**дидисккуотаусер**](didiskquotauser-object.md) пользователя:

-   Объекты [**дидисккуотаусер**](didiskquotauser-object.md) для всех пользователей с квотами на томе предоставляются в виде коллекции, и их можно перечислить. Обсуждение того, как перечислить объекты **дидисккуотаусер** , см. в разделе **перечисление пользователей дисковой квоты** раздела "Примечания" в **дидисккуотаусер**.
-   При добавлении нового пользователя метод [**adduser**](diskquotacontrol-adduser.md) возвращает объект пользователя [**дидисккуотаусер**](didiskquotauser-object.md) .
-   Если у вас есть имя пользователя, метод [**финдусер**](diskquotacontrol-finduser.md) возвращает объект [**дидисккуотаусер**](didiskquotauser-object.md) пользователя.

этот объект делает необходимые функции интерфейса идисккуотаконтрол доступными для создания сценариев и приложений на основе Microsoft Visual Basic.
