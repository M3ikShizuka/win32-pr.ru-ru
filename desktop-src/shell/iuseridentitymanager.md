# <a name="iuseridentitymanager-interface"></a>Интерфейс Иусеридентитиманажер

\[**Иусеридентитиманажер** не поддерживается и может быть изменен или недоступен в будущем. Вместо этого используйте [учетные записи пользователей с быстрым переключением пользователей и удаленный рабочий стол](fastuserswitching.md).\]

Предоставляет методы для идентификации, перечисления и управления удостоверениями пользователей в системе.

## <a name="members"></a>Элементы

Интерфейс **иусеридентитиманажер** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Иусеридентитиманажер** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иусеридентитиманажер** содержит следующие методы.



| Метод                                                                  | Описание                                                                                                                                                             |
|:------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**енумидентитиес**](iuseridentitymanager-enumidentities.md)           | Не рекомендуется. Возвращает объект [**иенумусеридентити**](ienumuseridentity.md) , который можно использовать для перечисления удостоверений пользователей, имеющихся в системе.<br/> |
| [**жетидентитибикукие**](iuseridentitymanager-getidentitybycookie.md) | Не рекомендуется. Возвращает конкретное удостоверение пользователя по файлу cookie, используемому для уникальной идентификации.<br/>                                                                        |
| [**Выполнен**](iuseridentitymanager-logoff.md)                           | Не рекомендуется. Выйдите из текущего удостоверения.<br/>                                                                                                                    |
| [**Входа**](iuseridentitymanager-logon.md)                             | Не рекомендуется. Отображает пользовательский интерфейс, позволяющий пользователю выбрать удостоверение пользователя. В случае успешного выполнения удостоверение пользователя будет зарегистрировано и извлечено.<br/>        |
| [**манажеидентитиес**](iuseridentitymanager-manageidentities.md)       | Не рекомендуется. Отображает пользовательский интерфейс, позволяющий пользователю управлять удостоверениями пользователей.<br/>                                                                          |
