# <a name="didiskquotauserid-property"></a>DIDiskQuotaUser.ID, свойство

Возвращает идентификатор, который однозначно идентифицирует пользователя.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
iID = DIDiskQuotaUser.ID
```



## <a name="property-value"></a>Значение свойства

**Целочисленное** значение, уникально идентифицирующее объект [**дидисккуотаусер**](didiskquotauser-object.md) пользователя в рамках определенного процесса [**дисккуотаконтрол**](diskquotacontrol-object.md) .

## <a name="remarks"></a>Remarks

Это свойство предназначено для использования в таких языках, как Microsoft Visual Basic, которые не поддерживают указатели.
