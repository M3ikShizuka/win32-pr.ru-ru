# <a name="iuseridentitymanagerlogon-method"></a>Метод Иусеридентитиманажер:: logon

\[**Иусеридентитиманажер:: logon** не поддерживается и может быть изменен или недоступен в будущем. Вместо этого используйте [учетные записи пользователей с быстрым переключением пользователей и удаленный рабочий стол](fastuserswitching.md).\]

Отображает пользовательский интерфейс, позволяющий пользователю выбрать удостоверение пользователя. В случае успешного выполнения удостоверение пользователя будет зарегистрировано и извлечено.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Logon(
  [in]  HWND          hwndParent,
  [in]  DWORD         dwFlags,
  [out] IUserIdentity **ppIdentity
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хвндпарент* \[ окне\]
</dt> <dd>

Тип: **HWND**

Значение **HWND** , определяющее окно, которое будет перенесено на передний план после закрытия пользовательского интерфейса входа.

</dd> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Тип: **DWORD**

Необязательные флаги для определения того, как будет вести себя пользовательский интерфейс. Задайте значение Уил \_ Force \_ UI, чтобы заставить пользовательский интерфейс отображаться, даже если удостоверение уже выбрано.

</dd> <dt>

*ппидентити* \[ заполняет\]
</dt> <dd>

Тип: **[ **иусеридентити**](iuseridentity.md)\*\***

Адрес указателя, который получает выбранное удостоверение пользователя.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

Результат операции входа в систему. В случае успеха возвращается значение S \_ ОК. В противном случае будет возвращен один из следующих кодов ошибок.



| Код возврата                                                                                            | Описание                                                                                 |
|--------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ \_ отменено пользователем**</dt> </dl>      | Пользователь отменил операцию входа из пользовательского интерфейса.<br/>                               |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl>          | Не удалось создать удостоверение пользователя.<br/>                                          |
| <dl> <dt>**\_непредвиденная E**</dt> </dl>           | Непредвиденный сбой операции.<br/>                                               |
| <dl> <dt>**Электронные \_ удостоверения \_ отключены**</dt> </dl> | Управление удостоверениями отключено в системе.<br/>                                   |
| <dl> <dt>**\_удостоверения \_ отключены**</dt> </dl> | Управление удостоверениями отключено в системе.<br/>                                   |
| <dl> <dt>**\_Изменение идентификатора \_ E**</dt> </dl>   | Система в настоящее время переключает удостоверения и не может выполнить операцию.<br/> |
