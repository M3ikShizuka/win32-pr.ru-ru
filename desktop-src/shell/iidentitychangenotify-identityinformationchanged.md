# <a name="iidentitychangenotifyidentityinformationchanged-method"></a>Метод Иидентитичанженотифи:: Идентитинформатиончанжед

\[**Идентитинформатиончанжед** не поддерживается и может быть изменен или недоступен в будущем. Вместо этого используйте [учетные записи пользователей с быстрым переключением пользователей и удаленный рабочий стол](fastuserswitching.md).\]

Вызывается при изменении сведений об удостоверении пользователя или при добавлении или удалении удостоверения пользователя.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT IdentityInformationChanged(
   DWORD dwType
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*двтипе* 
</dt> <dd>

Тип: **DWORD**

Значение типа, указывающее тип измененной информации или операцию, выполненную с удостоверением. Может быть сочетанием следующих значений.

<dt>



 (ИИК \_ ТЕКУЩЕЕ \_ удостоверение \_ изменено)


</dt> <dd>

Текущее удостоверение было изменено.

</dd> <dt>



 (ИИК \_ Идентификатор \_ изменен)


</dt> <dd>

Удостоверение было изменено.

</dd> <dt>



 (ИИК \_ УДОСТОВЕРЕНие \_ удалено)


</dt> <dd>

Удостоверение удалено.

</dd> <dt>



 (ИИК \_ УДОСТОВЕРЕНие \_ добавлено)


</dt> <dd>

Добавлено новое удостоверение.

</dd> </dl> </dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

Если этот метод завершается успешно, возвращается значение **S \_ ОК**. В противном случае возвращается код ошибки **HRESULT** .

## <a name="remarks"></a>Remarks

Этот метод можно реализовать, чтобы обеспечить пользовательское поведение приложения при изменении списка удостоверений пользователей в системе.
