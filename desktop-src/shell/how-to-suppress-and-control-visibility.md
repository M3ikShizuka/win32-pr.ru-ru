# <a name="how-to-suppress-and-control-verb-visibility"></a>Как подавлять и контролировать видимость команд

В этом разделе рассказывается, как управлять видимостью команд.

## <a name="instructions"></a>Инструкции


для управления видимостью команды можно использовать параметры политики Windows. Команды можно подавлять с помощью параметров политики, добавив подраздел **суппрессионполици** или значение **суппрессионполициекс** GUID в подраздел реестра команды. Присвойте параметру подраздела **СУППРЕССИОНПОЛИЦИ** идентификатор политики. Если политика включена, команда и связанная с ней запись контекстного меню подавляются. Возможные значения идентификатора политики см. в описании перечисления [**ограничений**](/windows/desktop/api/shlobj_core/ne-shlobj_core-restrictions) .
