# <a name="ienumuseridentity-interface"></a>Интерфейс Иенумусеридентити

\[**Иенумусеридентити** не поддерживается и может быть изменен или недоступен в будущем. Вместо этого используйте [учетные записи пользователей с быстрым переключением пользователей и удаленный рабочий стол](fastuserswitching.md).\]

Предоставляет перечисление всех удостоверений пользователей, имеющихся в системе.

## <a name="members"></a>Элементы

Интерфейс **иенумусеридентити** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Иенумусеридентити** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иенумусеридентити** содержит следующие методы.



| Метод                                         | Описание                                                                                                                  |
|:-----------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------|
| [**Клонировать**](ienumuseridentity-clone.md)       | Не рекомендуется. Возвращает копию текущего перечисления.<br/>                                                               |
| [**GetCount**](ienumuseridentity-getcount.md) | Не рекомендуется. Возвращает число удостоверений пользователей, находящихся в системе в настоящее время.<br/>                                            |
| [**Далее**](ienumuseridentity-next.md)         | Не рекомендуется. Извлекает из перечисления массив интерфейсов удостоверения пользователя.<br/>                                  |
| [**Reset**](ienumuseridentity-reset.md)       | Не рекомендуется. Сбрасывает внутреннее число извлеченных интерфейсов в перечислении.<br/>                                 |
| [**Сразу**](ienumuseridentity-skip.md)         | Не рекомендуется. Пропускает заданное число интерфейсов удостоверения пользователя в перечислении. Используется при извлечении интерфейсов.<br/> |



 

## <a name="remarks"></a>Примечания

Для получения сведений об удостоверениях пользователей, имеющихся в системе, можно использовать этот интерфейс. Чтобы получить этот интерфейс, из интерфейса [**иусеридентитиманажер**](iuseridentitymanager.md) можно вызвать [**Иусеридентитиманажер:: енумидентитиес**](iuseridentitymanager-enumidentities.md) .
