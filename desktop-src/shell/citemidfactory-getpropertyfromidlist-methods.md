# <a name="citemidfactorygetpropertyfromidlist-methods"></a>Методы Цитемидфактори:: Жетпропертифромидлист

Возвращает свойство из [**ипропертисторе**](/windows/win32/api/propsys/nn-propsys-ipropertystore) в IDList.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                        | Описание                                                                                                                                   |
|:------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------|
| [**жетпропертифромидлист**](/windows/win32/api/shidfact/nf-shidfact-citemidfactory-getpropertyfromidlist(pcuidlist_relative_pcwstr_propvariant))     | Возвращает свойство из [**ипропертисторе**](/windows/win32/api/propsys/nn-propsys-ipropertystore) в IDList как Variant с помощью ключа.<br/>            |
| [**жетпропертифромидлист**](/windows/win32/api/shidfact/nf-shidfact-citemidfactory-getpropertyfromidlist(pcuidlist_relative_pcwstr_propvariant))    | Возвращает свойство из [**ипропертисторе**](/windows/win32/api/propsys/nn-propsys-ipropertystore) в IDList как Variant с использованием именованного свойства.<br/> |
| [**Жетпропертифромидлист**]/windows/win32/api/shidfact/nf-shidfact-citemidfactory-getpropertyfromidlist (pcuidlist_relative_pcwstr_propvariant))  | Возвращает свойство из [**ипропертисторе**](/windows/win32/api/propsys/nn-propsys-ipropertystore) в IDList с помощью ключа.<br/>                         |
| [**жетпропертифромидлист**](/windows/win32/api/shidfact/nf-shidfact-citemidfactory-getpropertyfromidlist(pcuidlist_relative_pcwstr_propvariant)) | Возвращает свойство из [**ипропертисторе**](/windows/win32/api/propsys/nn-propsys-ipropertystore) в IDList с помощью именованного свойства.<br/>              |
