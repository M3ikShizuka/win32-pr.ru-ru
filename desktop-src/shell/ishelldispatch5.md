# <a name="ishelldispatch5-object"></a>Объект IShellDispatch5

Расширяет объект [**IShellDispatch4**](ishelldispatch4.md) . В дополнение к свойствам и методам, поддерживаемым **IShellDispatch4**, **IShellDispatch5** добавляет метод, который отображает открытые окна в трехмерном стеке.

> [!Note]  
> **IShellDispatch5** реализован и доступен через объект [**Shell**](shell.md) .

 

## <a name="members"></a>Элементы

Объект **IShellDispatch5** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Объект **IShellDispatch5** содержит эти методы.



| Метод                                                   | Описание                                                                    |
|:---------------------------------------------------------|:-------------------------------------------------------------------------------|
| [**виндовсвитчер**](ishelldispatch5-windowswitcher.md) | Отображает открытые окна в трехмерном стеке, который можно переворачивать.<br/> |
