# <a name="imultimonitordockingsite-interface"></a>Интерфейс Имултимонитордоккингсите

Реализуется браузером. Предоставляет методы, управляющие монитором, содержащим панель задач Windows в системе с несколькими мониторами.

## <a name="members"></a>Элементы

Интерфейс **имултимонитордоккингсите** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Имултимонитордоккингсите** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **имултимонитордоккингсите** содержит следующие методы.



| Метод                                                            | Описание                                                                                |
|:------------------------------------------------------------------|:-------------------------------------------------------------------------------------------|
| [**Монитор**](imultimonitordockingsite-getmonitor.md)         | Возвращает текущий монитор по умолчанию.<br/>                                               |
| [**рекуестмонитор**](imultimonitordockingsite-requestmonitor.md) | Проверяет, что монитор активен и доступен.<br/>                              |
| [**сетмонитор**](imultimonitordockingsite-setmonitor.md)         | Изменения, которые монитор используется для закрепленных панелей инструментов в системе с несколькими мониторами.<br/> |



 

## <a name="remarks"></a>Примечания

Обычно интерфейс **имултимонитордоккингсите** не реализуется. Браузер оболочки реализует этот интерфейс для поддержки нескольких мониторов.
