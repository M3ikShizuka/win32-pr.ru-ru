# <a name="abm_querypos-message"></a>\_Сообщение АБМ куерипос

Запрашивает размер и расположение экрана для панель приложений. При выполнении запроса в сообщении предлагается ребро экрана и ограничивающий прямоугольник для панель приложений. система корректирует ограничивающий прямоугольник таким образом, чтобы панель приложений не влиял на Windowsную панель задач или другие appbar.


```C++
SHAppBarMessage(ABM_QUERYPOS, pabd); 
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пабд* 
</dt> <dd>

Указатель на структуру [**аппбардата**](/windows/desktop/api/Shellapi/ns-shellapi-appbardata) . Элемент **уедже** задает ребро экрана, а член **RC** содержит предложенный ограничивающий прямоугольник. Когда функция [**шаппбармессаже**](/windows/desktop/api/Shellapi/nf-shellapi-shappbarmessage) возвращает, **RC** содержит утвержденный ограничивающий прямоугольник. При отправке этого сообщения необходимо указать члены **кбсизе**, **HWND**, **уедже** и **RC** . все остальные элементы игнорируются.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Всегда возвращает **значение true**.

## <a name="remarks"></a>Remarks

Объект панель приложений должен отправить это сообщение перед отправкой сообщения [**\_ сетпос АБМ**](abm-setpos.md) .
