# <a name="folderitems2-object"></a>Объект FolderItems2

Расширяет объект [**фолдеритемс**](folderitems.md) . Он поддерживает один дополнительный метод.

## <a name="members"></a>Элементы

Объект **FolderItems2** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Объект **FolderItems2** содержит эти методы.



| Метод                                            | Описание                                                                                                                                                                                                                                         |
|:--------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**инвокевербекс**](folderitems2-invokeverbex.md) | Выполняет команду для коллекции объектов [**FolderItem**](folderitem.md) . Этот метод является расширением метода [**инвокеверб**](folderitem-invokeverb.md) , что позволяет дополнительно управлять операцией с помощью набора флагов.<br/> |
