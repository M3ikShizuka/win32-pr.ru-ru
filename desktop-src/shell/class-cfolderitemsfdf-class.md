# <a name="cfolderitemsfdf-class"></a>Класс Кфолдеритемсфдф

**Кфолдеритемсфдф** — это коллекция [**фолдеритемс**](folderitems.md). Он реализует интерфейс [**иинсертитем**](/windows/desktop/api/Shobjidl/nn-shobjidl-iinsertitem) .

## <a name="members"></a>Элементы

Класс **кфолдеритемсфдф** наследует от [**кфолдеритемс**](class-cfolderitems-class.md) , но не содержит дополнительных элементов.

## <a name="remarks"></a>Remarks

FDF означает *из разных папок*.

Класс [**кфолдеритемс**](class-cfolderitems-class.md) перечисляет элементы из нескольких папок и использует интерфейс [**иинсертитем**](/windows/desktop/api/Shobjidl/nn-shobjidl-iinsertitem) для вставки элементов в коллекцию.
