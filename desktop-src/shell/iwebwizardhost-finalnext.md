# <a name="webwizardhostfinalnext-method"></a>Вебвизардхост. Финалнекст, метод

Переход к странице мастера на стороне клиента, расположенной после страницы, на которой размещены HTML-страницы на стороне сервера, или завершение работы мастера при отсутствии дополнительных страниц на стороне клиента.

## <a name="syntax"></a>Синтаксис


```JScript
iRetVal = WebWizardHost.FinalNext()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="remarks"></a>Примечания

Когда мастер отображает последнюю HTML-страницу на стороне сервера и пользователь нажимает кнопку " **Далее** " или **"Готово** ", сервер вызывает **финалнекст** в обработчике событий этой кнопки.
