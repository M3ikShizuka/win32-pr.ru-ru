# <a name="didiskquotauseraccountstatus-property"></a>Дидисккуотаусер. Аккаунтстатус, свойство

Возвращает состояние учетной записи пользователя.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
iAccountStatus = DIDiskQuotaUser.AccountStatus
```



## <a name="property-value"></a>Значение свойства

Это свойство может принимать одно из следующих значений.



| Состояние            | Значение | Описание                         |
|-------------------|-------|-------------------------------------|
| дкакктресолвед    | 0     | Сведения об учетной записи разрешены.    |
| дкакктунаваилабле | 1     | Сведения об учетной записи недоступны. |
| дкакктделетед     | 2     | Учетная запись удалена.           |
| дкакктинвалид     | 3     | Недопустимая учетная запись.                 |
| дкакктункновн     | 4     | Не удается найти учетную запись.            |
| дкакктунресолвед  | 5     | Сведения об учетной записи не разрешены.  |
