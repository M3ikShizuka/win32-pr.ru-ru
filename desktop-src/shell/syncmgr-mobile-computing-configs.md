# <a name="mobile-computing-configurations"></a>Конфигурации мобильных вычислений

Диспетчер синхронизации полезен для компьютеров, настроенных следующим образом.

-   Мобильный компьютер, используемый на стыковочный станции в сети с высокой пропускной способностью, но иногда используемый через коммутируемое соединение.
-   Мобильный компьютер используется в основном через коммутируемое подключение.
-   Настольный компьютер используется строго через коммутируемое подключение, например компьютер, используемый удаленным пользователем.
-   Настольный компьютер используется строго через сеть с высокой пропускной способностью.

Несмотря на то, что последний вариант не является типичным сценарием для мобильных устройств, проблемы с доступом к сети могут быть удобными для локального кэширования ресурсов. Во всех этих конфигурациях приложение может использовать Синкмгр, чтобы хранить файлы и другие ресурсы в локальном кэше и синхронизировать их между Интернетом и автономным использованием.
