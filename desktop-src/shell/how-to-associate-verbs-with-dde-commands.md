# <a name="how-to-associate-verbs-with-dde-commands"></a>Связывание команд с командами DDE

При вызове глагола обычно запускается приложение, указанное в подразделе команд команды. тем не менее, если приложение поддерживает платформа динамических данных Exchange (DDE), можно запустить оболочку сеанса dde.

Чтобы указать, что вызов команды должен инициировать сеанс DDE, выполните следующие действия.

## <a name="instructions"></a>Instructions

### <a name="step-1"></a>Шаг 1.

Добавьте подраздел **ддиксек** в ключ команды.

### <a name="step-2"></a>Шаг 2.

Установите значение по умолчанию **ддиксек** в строку команды DDE.

## <a name="remarks"></a>Комментарии

Ключ **ддиксек** содержит три дополнительных подраздела, которые обеспечивают некоторый контроль над процессом DDE:

-   **приложение**. В качестве значения по умолчанию для этого подраздела задайте имя приложения, которое будет использоваться для установки диалога DDE. Если подраздел **приложения** отсутствует, в качестве имени приложения используется значение по умолчанию для подраздела **Command команды** .
-   **раздел**. В качестве значения по умолчанию для этого подраздела укажите имя раздела диалога DDE. Если подраздел **раздела** отсутствует, в качестве имени раздела используется система.
-   **ифексек**. Задайте для этого подраздела по умолчанию команду DDE, которая будет использоваться, если невозможно запустить сеанс DDE. При сбое инициации запускается приложение, указанное значением по умолчанию для подраздела **команд** команды. Если ключ **ифексек** существует, его значение по умолчанию будет использоваться в качестве команды DDE. Если подраздел **ифексек** отсутствует, в качестве команды DDE будет использоваться значение по умолчанию ключа **ддиксек** .

В следующем примере указывается, что вызов команды Open для MyProgram. 1 инициирует сеанс DDE с помощью команды DDE Open ("%1") и имени приложения MyProgram.

```
HKEY_CLASSES_ROOT
   MyProgram.1
      (Default) = MyProgram Application
      Shell
         (Default) = doit
         open
            command
               (Default) = C:\MyDir\MyProgram.exe "%1"
            ddeexec
               (Default) = Open("%1")
               application
                  (Default) = MyProgram
```
