# <a name="fm_getfocus-message"></a>Сообщение FM- \_ фокуса

Отправляется расширением диспетчера файлов для получения типа окна диспетчера файлов, имеющего фокус ввода.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает тип окна диспетчера файлов, имеющего фокус ввода. Может быть одним из указанных далее.



| Код возврата                                                                                    | Описание                                         |
|------------------------------------------------------------------------------------------------|-----------------------------------------------------|
| <dl> <dt>**ФМФОКУС \_ dir**</dt> </dl>    | Часть каталога в окне каталога.<br/> |
| <dl> <dt>**\_дерево фмфокус**</dt> </dl>   | Древовидная часть окна каталога.<br/>      |
| <dl> <dt>**\_диски фмфокус**</dt> </dl> | Панель диска в окне каталога.<br/>         |
| <dl> <dt>**\_Поиск фмфокус**</dt> </dl> | Окно результатов поиска.<br/>                   |
