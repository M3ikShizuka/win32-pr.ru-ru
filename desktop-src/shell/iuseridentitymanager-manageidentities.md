# <a name="iuseridentitymanagermanageidentities-method"></a>Метод Иусеридентитиманажер:: Манажеидентитиес

\[**Иусеридентитиманажер:: манажеидентитиес** не поддерживается и может быть изменен или недоступен в будущем. Вместо этого используйте [учетные записи пользователей с быстрым переключением пользователей и удаленный рабочий стол](fastuserswitching.md).\]

Отображает пользовательский интерфейс, позволяющий пользователю управлять удостоверениями пользователей.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT ManageIdentities(
  [in] HWND  hwndParent,
  [in] DWORD dwFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хвндпарент* \[ окне\]
</dt> <dd>

Тип: **HWND**

Значение **HWND** , определяющее окно, которое будет перенесено на передний план после закрытия пользовательского интерфейса.

</dd> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Тип: **DWORD**

Необязательные флаги для определения того, как будет вести себя пользовательский интерфейс. Задайте значение УИМИ \_ создать \_ новое \_ удостоверение, чтобы предложить пользователю создать новое удостоверение.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

Результат операции управления. В случае успеха возвращается значение S \_ ОК. В противном случае будет возвращен один из следующих кодов ошибок.



| Код возврата                                                                                            | Описание                                               |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------|
| <dl> <dt>**Электронные \_ удостоверения \_ отключены**</dt> </dl> | Управление удостоверениями отключено в системе.<br/> |
| <dl> <dt>**\_ \_ отменено пользователем**</dt> </dl>      | Пользователь отменил диалоговое окно.<br/>                  |
