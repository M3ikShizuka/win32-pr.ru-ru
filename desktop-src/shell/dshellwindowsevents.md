# <a name="dshellwindowsevents-object"></a>Объект Дшеллвиндовсевентс

Получает события регистрации окна [**ишеллвиндовс**](/windows/desktop/api/Exdisp/nn-exdisp-ishellwindows) .

## <a name="members"></a>Элементы

Объект **дшеллвиндовсевентс** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Объект **дшеллвиндовсевентс** содержит эти методы.



| Метод                                                           | Описание                                                       |
|:-----------------------------------------------------------------|:------------------------------------------------------------------|
| [**виндоврегистеред**](dshellwindowsevents-windowregistered.md) | Вызывается, когда окно регистрируется как окно оболочки. <br/> |
| [**виндовревокед**](dshellwindowsevents-windowrevoked.md)       | Вызывается при отмене регистрации окна оболочки. <br/> |



 

## <a name="remarks"></a>Примечания

Используйте **дшеллвиндовсевентс** для отслеживания регистрации или отзыва окна оболочки. Дополнительные сведения см. в разделе [**ишеллвиндовс**](/windows/desktop/api/Exdisp/nn-exdisp-ishellwindows).
