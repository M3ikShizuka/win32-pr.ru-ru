# <a name="shell-color-palette-handling-functions"></a>Функции обработки цветовой палитры оболочки

в этом разделе описываются функции обработки цветовой палитры Windows оболочки. Программные элементы, описанные в этой документации, экспортируются с помощью Shlwapi.dll и определяются в Shlwapi. h и Shlwapi. lib.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                           | Описание                                                                           |
|-----------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**колораджустлума**](/windows/desktop/api/Shlwapi/nf-shlwapi-coloradjustluma)<br/>           | Изменяет светимость значения RGB. Оттенок и насыщенность не затрагиваются.<br/> |
| [**колорхлсторгб**](/windows/desktop/api/Shlwapi/nf-shlwapi-colorhlstorgb)<br/>               | Преобразует цвета из оттенка-яркости-насыщенности (HLS) в формат RGB.<br/>         |
| [**колорргбтохлс**](/windows/desktop/api/Shlwapi/nf-shlwapi-colorrgbtohls)<br/>               | Преобразует цвета из формата RGB в формат "цветовой тон-светимость" (HLS).<br/>         |
| [**шкреатешеллпалетте**](/windows/desktop/api/Shlwapi/nf-shlwapi-shcreateshellpalette)<br/> | Создает палитру полутонов для указанного контекста устройства.<br/>               |
