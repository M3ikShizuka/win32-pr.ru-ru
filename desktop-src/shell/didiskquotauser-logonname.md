# <a name="didiskquotauserlogonname-property"></a>Дидисккуотаусер. Логоннаме, свойство

Возвращает имя учетной записи входа пользователя.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
LogonName = DIDiskQuotaUser.LogonName
```



## <a name="property-value"></a>Значение свойства

Строковое значение, заданное для имени входа учетной записи пользователя.
