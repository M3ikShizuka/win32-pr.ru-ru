# <a name="sfvm_updatestatusbar-message"></a>\_Сообщение сфвм упдатестатусбар

Уведомляет объект обратного вызова о том, что строка состояния обновляется. Используется [**ишеллфолдервиевкб:: мессажесфвкб**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb).


```C++
SFVM_UPDATESTATUSBAR 

    wParam = (WPARAM)(BOOL) fInitialize;

            
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*финитиализе* \[ окне\]
</dt> <dd>

**Логическое** значение, которое устанавливается равным **true** , если строка состояния инициализируется.

</dd> </dl>

## <a name="remarks"></a>Remarks

При получении этого уведомления:

-   \_Если обновление будет обработано, возвращайте ОК.
-   Возвращает \_ значение HRESULT ( \_ УСПЕШНОе серьезность, 0, 1), чтобы объект представления системной папки установил текст строки состояния.
-   Возврат E \_ не удается, чтобы объект представления системной папки обрабатывал строку состояния.

Текст строки состояния по умолчанию выглядит следующим образом.



| Состояние                      | Текст строки состояния                         |
|-----------------------------|-----------------------------------------|
| Элементы не выбраны           | " \# \# Objects" (в папке)          |
| Выбран один элемент           | Всплывающая подсказка для элемента, если она доступна. |
| Выбрано более одного элемента | " \# \# объекты выбраны"                 |
