# <a name="newwdevents-object"></a>Объект Неввдевентс

Расширяет объект [**вебвизардхост**](webwizardhost.md) , позволяя страницам на стороне сервера, размещенным в мастере, проверять, что пользователь прошел проверку подлинности с помощью учетная запись Майкрософт.

## <a name="members"></a>Элементы

Объект **неввдевентс** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Объект **неввдевентс** содержит эти методы.



| Метод                                                            | Описание                                                                                                                         |
|:------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------|
| [**пасспортаусентикате**](inewwdevents-passportauthenticate.md) | Позволяет страницам на стороне сервера, размещенным в мастере, проверить, что пользователь прошел проверку подлинности с помощью учетная запись Майкрософт.<br/> |
