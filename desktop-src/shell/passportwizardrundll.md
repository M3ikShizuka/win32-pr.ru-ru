# <a name="passportwizardrundll-function"></a>Функция Пасспортвизардрундлл

\[эта функция доступна в Windows XP с пакетом обновления 2 (sp2) и Windows Server 2003. Он может быть изменен или недоступен в последующих версиях Windows.\]

Запускает мастер паспорта при использовании с Rundll32.exe.

## <a name="syntax"></a>Синтаксис


```C++
void PassportWizardRunDll(
  _In_ HWND      hwndStub,
  _In_ HINSTANCE hAppInstance,
  _In_ LPTSTR    lpszCmdLine,
  _In_ int       nCmdShow
);
```

<dl> <dt>

</dt> <dd>

Тип: **HWND**

Маркер окна-владельца. Обычно этот параметр имеет значение **null**.

</dd> <dt>

</dt> <dd>

Тип: **HINSTANCE**

Обработчик файла библиотеки, полученный в виде возвращаемого значения из [**LoadLibrary**](/windows/win32/api/libloaderapi/nf-libloaderapi-loadlibrarya)("нетплвиз").

## <a name="return-value"></a>Возвращаемое значение

Нет.

## <a name="remarks"></a>Remarks

Мастер паспорта используется для получения паспорта по умолчанию для Windows. Паспорт предоставляет пользователю персонализированный доступ ко всем веб-сайтам MSN и другим сайтам с поддержкой Passport, используя адрес электронной почты пользователя. Использование **пасспортвизардрундлл** в качестве точки входа в файл Netplwiz.dll с помощью команды rundll32 позволяет запускать мастер паспорта из командной строки, как будто это исполняемый файл.

**Пасспортвизардрундлл** используется исключительно в контексте Rundll32.exe команды следующим образом:

**rundll32.exe netplwiz.dll, Пасспортвизардрундлл**

Использование функции точки входа с Rundll32.exe не похоже на вызов обычной функции. Имя функции и имя файла .dll, где она хранится, используются только в качестве параметров командной строки. Определение функции, показанное в разделе синтаксис, является только стандартным прототипом для всех функций, которые можно вызывать с помощью rundll32. Определенные значения для *хвндстуб*, *хаппинстанце* и *нкмдшов* не предоставляются пользователем, но обрабатываются в фоновом режиме с помощью rundll32. **Пасспортвизардрундлл** не использует значение *лпсзкмдлине* , поэтому дополнительные данные не требуются.
