# <a name="roaming-user-profiles"></a>Проверка перемещаемых профилей пользователей

Если компьютер работает под Windows 2000 Server или более поздней версии в сети, пользователи могут хранить свои профили на сервере. Эти профили называются *перемещаемыми профилями пользователей*.

Перемещаемые профили пользователей имеют следующие преимущества.

-   Автоматическая доступность ресурсов. Уникальный профиль пользователя автоматически становится доступным при входе на любой компьютер в сети. Пользователям не нужно создавать профиль на каждом компьютере, который используется в сети.
-   Упрощенная замена и резервное копирование компьютера. При необходимости замены компьютера пользователя его можно легко заменить, так как все сведения о профиле пользователя хранятся отдельно в сети независимо от отдельного компьютера. Когда пользователь впервые входит в систему на новом компьютере, копия профиля пользователя копируется на новый компьютер.

Профиль пользователя не загружается автоматически при входе пользователя в систему с помощью функции [**LogonUser**](/windows/win32/api/winbase/nf-winbase-logonusera) . Чтобы программно загрузить перемещаемый профиль пользователя, используйте функцию [**LoadUserProfile**](/windows/desktop/api/Userenv/nf-userenv-loaduserprofilea) . Чтобы выгрузить перемещаемый профиль пользователя, загруженный **LoadUserProfile**, вызовите функцию [**унлоадусерпрофиле**](/windows/desktop/api/Userenv/nf-userenv-unloaduserprofile) .
