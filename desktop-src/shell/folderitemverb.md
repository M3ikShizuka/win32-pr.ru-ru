# <a name="folderitemverb-object"></a>Объект Фолдеритемверб

Представляет единственную команду, доступную для элемента. Этот объект содержит свойства и методы, позволяющие получить сведения о команде.

## <a name="members"></a>Элементы

Объект **фолдеритемверб** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **фолдеритемверб** содержит эти методы.



| Метод                              | Описание                                                                                  |
|:------------------------------------|:---------------------------------------------------------------------------------------------|
| [**доит**](folderitemverb-doit.md) | Выполняет команду для [**FolderItem**](folderitem.md) , связанной с командой.<br/> |



 

### <a name="properties"></a>Свойства

Объект **фолдеритемверб** имеет следующие свойства.



| Свойство                                                     | Тип доступа          | Описание                          |
|:-------------------------------------------------------------|:---------------------|:-------------------------------------|
| [**Приложение**](folderitemverb-application.md)<br/> | Только для чтения<br/> | Не реализован.<br/>          |
| [**Имя**](folderitemverb-name.md)<br/>               | Только для чтения<br/> | Содержит имя команды.<br/> |
| [**Parent**](folderitemverb-parent.md)<br/>           | Только для чтения<br/> | Не реализован.<br/>          |
