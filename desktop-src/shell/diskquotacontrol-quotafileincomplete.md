# <a name="diskquotacontrolquotafileincomplete-property"></a>Дисккуотаконтрол. Куотафилеинкомплете, свойство

Возвращает логическое значение, указывающее, завершен ли файл квоты для тома.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
bQuotaFileIncomplete = DiskQuotaControl.QuotaFileIncomplete
```



## <a name="property-value"></a>Значение свойства

Это свойство имеет значение **true** , если файл квоты является неполным, или **false** в противном случае.
