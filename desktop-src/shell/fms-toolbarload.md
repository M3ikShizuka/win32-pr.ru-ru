# <a name="fms_toolbarload-structure"></a>\_Структура FMS тулбарлоад

Содержит сведения о настраиваемых кнопках, добавляемых на панель инструментов диспетчера файлов. Кнопки предоставляются БИБЛИОТЕКой расширения диспетчера файлов.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _FMS_TOOLBARLOAD {
  DWORD        dwSize;
  LPEXT_BUTTON lpButtons;
  WORD         cButtons;
  WORD         cBitmaps;
  WORD         idBitmap;
  HBITMAP      hBitmap;
} FMS_TOOLBARLOAD;
```



## <a name="members"></a>Члены

<dl> <dt>

**двсизе**
</dt> <dd>

Тип: **DWORD**

</dd> <dd>

Размер структуры в байтах. Диспетчер файлов задает размер перед вызовом расширения и проверяет размер после возврата процедуры расширения.

</dd> <dt>

**лпбуттонс**
</dt> <dd>

Тип: **\_ кнопка лпекст**

</dd> <dd>

Адрес массива структур [**\_ кнопок ext**](ext-button.md) .

</dd> <dt>

**кбуттонс**
</dt> <dd>

Тип: **слово**

</dd> <dd>

Количество структур [**\_ кнопок ext**](ext-button.md) в массиве, на которое указывает элемент **лпбуттонс** . Это число равно количеству кнопок и разделителей, добавляемых на панель инструментов.

</dd> <dt>

**кбитмапс**
</dt> <dd>

Тип: **слово**

</dd> <dd>

Количество кнопок, представленных заданным растровым изображением.

</dd> <dt>

**идбитмап**
</dt> <dd>

Тип: **слово**

</dd> <dd>

Идентификатор ресурса точечного рисунка в исполняемом файле для библиотеки DLL расширения. Ресурс точечного рисунка содержит изображения для количества кнопок, заданных параметром **кбитмапс**. Диспетчер файлов загружает ресурс точечного рисунка, а затем использует его для вывода кнопок.

</dd> <dt>

**хбитмап**
</dt> <dd>

Тип: **хбитмап**

</dd> <dd>

Маркер точечного рисунка, который диспетчер файлов будет использовать для получения и просмотра изображений кнопок, если **идбитмап** имеет значение 0.

</dd> </dl>
