# <a name="sfvm_size-message"></a>\_Сообщение о размере сфвм

Уведомляет объект обратного вызова о том, что размер представления папки изменился. Используется [**ишеллфолдервиевкб:: мессажесфвкб**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb).


```C++
SFVM_SIZE
```



## <a name="parameters"></a>Параметры

Это сообщение не содержит параметров.
