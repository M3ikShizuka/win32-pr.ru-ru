# <a name="customizing-icons"></a>Настройка значков

Windows предоставляет значки по умолчанию для каждого элемента, отображаемого на рабочем столе и в обозревателе Windows. Запоминающие устройства, например дисковые накопители, также назначаются значками по умолчанию. Однако эти значки часто предоставляют пользователю небольшое представление о содержимом файла или о том, какая программа связана с ней. Можно назначить пользовательский значок для папки файловой системы, создав файл [Desktop.ini](how-to-customize-folders-with-desktop-ini.md) . Ознакомьтесь со следующими документами, чтобы узнать, как с помощью реестра связать пользовательские значки с типами файлов и буквами дисков.

-   [Назначение пользовательского значка типу файла](how-to-assign-a-custom-icon-to-a-file-type.md)
-   [Назначить букве диска пользовательский значок и метку](how-to-assign-a-custom-icon-and-label-to-a-drive-letter.md)
