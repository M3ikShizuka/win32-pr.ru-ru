# <a name="abm_new-message"></a>АБМ \_ новое сообщение

Регистрирует новый панель приложений и указывает идентификатор сообщения, который система должна использовать для отправки сообщений уведомления. Панель приложений должен отправить это сообщение перед отправкой любых других сообщений панель приложений.


```C++
fRegistered = (BOOL) SHAppBarMessage(ABM_NEW, pabd); 
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пабд* 
</dt> <dd>

Указатель на структуру [**аппбардата**](/windows/desktop/api/Shellapi/ns-shellapi-appbardata) , которая содержит маркер окна нового панель приложений и идентификатор сообщения. При отправке этого сообщения необходимо указать члены **кбсизе**, **HWND** и **укаллбаккмессаже** ; все остальные элементы игнорируются.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** в случае успеха или **значение false** , если возникла ошибка, или если панель приложений уже зарегистрирован.
