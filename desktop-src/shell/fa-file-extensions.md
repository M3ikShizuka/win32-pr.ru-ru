# <a name="file-type-handlers"></a>Обработчики типов файлов

[Регистрация типа файла](fa-how-work.md) — это первый шаг в создании сопоставления файлов, что делает этот тип файлов "известным" оболочке. Однако без обработчиков типов файлов оболочка не может предоставить пользователю сведения о файле и о нем.

Этот раздел организован следующим образом:

-   [Создание типа файла, известного оболочке](#make-a-file-type-known-to-shell)
-   [Описания обработчика типов файлов](#file-type-handler-descriptions)
-   [Связанные темы](#related-topics)

## <a name="make-a-file-type-known-to-shell"></a>Создание типа файла, известного оболочке

на следующем снимке экрана Windows Explorer изображение файла изображения будет отображаться в библиотеке **изображений** оболочки и связано только с приложением Paint.

![снимок экрана с браузером открытие изображения без типа файлов](images/file-assoc/fileassoc-filetypehandler.png)

На предыдущем снимке экрана отсутствуют следующие функциональные возможности, включенные обработчиком типов файлов:

-   Эскиз или предварительный просмотр
-   Команды, относящиеся к образу, в контекстном меню, например:
    -   Повернуть предварительный просмотр
    -   Задать как фон рабочего стола
    -   Печать
-   Свойства, относящиеся к конкретному образу, в области **сведений** , например:
    -   Дата съемки
    -   Теги
    -   Рейтинг
-   Индексирование текста файла

На следующем снимке экрана один и тот же файл («во. известный») имеет расширение .jpg, которое является зарегистрированным типом файлов, связанным с обработчиками типов файлов, поэтому отображаются эскизы и другие свойства.

![изображение с зарегистрированным типом файла и обработчиком типов связанных файлов](images/file-assoc/fileassoc-filetypehandler-2ndex.png)

## <a name="file-type-handler-descriptions"></a>Описания обработчика типов файлов

Функциональные возможности, предоставляемые каждым обработчиком типов файлов, перечислены в следующей таблице.



| Обработчик                                                      | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
|--------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Контекстное меню](context-menu-handlers.md)                   | Обработчик контекстного меню, иногда называемый обработчиком контекстного меню, является обработчиком типа файлов, добавляющим команды в существующее контекстное меню. Эти обработчики связаны с определенным типом файла и вызываются каждый раз, когда отображается контекстное меню для элемента типа файла.                                                                                                                                                                                                                                                                           |
| [Thumbnail](thumbnail-providers.md)                         | Обработчик, предоставляющий изображение для представления элемента оболочки.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [Свойство](../properties/building-property-handlers-properties.md) | обработчик свойства, который предоставляет доступ к свойствам элемента для Windows поиска, обозревателю Windows и другим приложениям, которым требуется доступ к свойствам.                                                                                                                                                                                                                                                                                                                                                                                                              |
| [Предварительный просмотр](preview-handlers.md)                              | обработчик, который быстро создает доступное только для чтения упрощенное представление элемента, отображаемого на панели предварительного просмотра Windows Explorer.                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [Фильтры](../search/-search-3x-wds-extidx-filters.md)              | Фильтр, реализация интерфейса [**IFilter**](/windows/win32/api/filter/nn-filter-ifilter) , который сканирует документы для текста и свойств (также называемых атрибутами). Он извлекает фрагменты текста из этих документов, отфильтровывая внедренное форматирование и сохраняет информацию о положении текста. Он также извлекает фрагменты значений, которые являются свойствами всего документа или строго определенных частей документа. **IFilter** предоставляет основу для создания приложений более высокого уровня, таких как индексаторы документов и средства просмотра, независимые от приложений. |
