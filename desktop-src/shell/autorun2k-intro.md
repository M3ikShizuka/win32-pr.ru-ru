# <a name="using-hardware-autoplay"></a>Использование автоматического запуска оборудования

Автозапуск — это новая функция, интродуктед в Windows XP, которая определяет содержимое, например изображения, музыку или видеофайлы на съемных носителях и съемных устройствах. Затем Автозапуск автоматически запускает приложения для воспроизведения или показа этого содержимого. Это упрощает использование специализированных периферийных устройств, таких как проигрыватели MPEG-Layer 3 (MP3) и Цифровые средства чтения фотографий. Кроме того, это упрощает для пользователей, которые не знакомы с программным обеспечением, необходимым для доступа к различным типам содержимого.

Расширенный набор значков, включая фирменные значки для основных производителей, обеспечивает улучшенное визуальное различие дисков и других подключенных устройств в **Мой компьютер**. Значки для устройств с съемными носителями, за исключением дисководов гибких дисков, предоставляются как в готовых, так и в неготовых версиях для указания текущего состояния устройства. Это подтверждает, что устройство, подключенное к системе, готово к использованию и помогает избежать сообщения "устройство не готово" с устройств без вставки носителя.

> [!Note]  
> Автозапуск не следует путать с AutoRun. Функция автозапуска позволяет компакт-диску автоматически запускать функцию, например установщик или игру, при помещении на устройство чтения с устройства. Это достигается за счет использования файла autorun. INF в корневом каталоге компакт-диска. Дополнительные сведения см. [в разделе Создание приложения для чтения компакт-дисков с поддержкой автозапуска](autoplay.md).
