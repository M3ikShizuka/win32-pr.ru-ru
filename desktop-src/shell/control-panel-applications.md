# <a name="implementing-control-panel-items"></a>Реализация элементов панели управления

Элементы панели управления — это библиотеки DLL или исполняемые файлы (exe), позволяющие пользователям настраивать среду Windows. Доступ к ним обычно осуществляется щелчком значка на панели управления.

В этом разделе описываются элементы панели управления и объясняется, как создавать и регистрировать их, чтобы они правильно отображались на панели управления. В Windows Vista содержатся сведения о том, как добавлять ссылки на задачи, которые отображаются в элементе панели управления и в результатах поиска панели управления.
