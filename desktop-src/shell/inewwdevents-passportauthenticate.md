# <a name="newwdeventspassportauthenticate-method"></a>Неввдевентс. Пасспортаусентикате, метод

Позволяет страницам на стороне сервера, размещенным в мастере, проверить, что пользователь прошел проверку подлинности с помощью учетная запись Майкрософт.

## <a name="syntax"></a>Синтаксис


```JScript
bRetVal = NewWDEvents.PassportAuthenticate(
  bstrSignInUrl
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*бстрсигнинурл* \[ окне\]
</dt> <dd>

Тип: **[ **BSTR**](/previous-versions/windows/desktop/automat/bstr)**

Строка, содержащая URL-адрес веб-страницы, которая перенаправляется на учетная запись Майкрософт Пользовательский интерфейс входа.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **Boolean**

Задайте значение **true** , если проверка подлинности выполнена, и **false** в противном случае.

## <a name="remarks"></a>Remarks

Этот метод может быть вызван, даже если пользователь уже вошел в учетная запись Майкрософт. В этом случае метод возвращает **значение true** без отображения учетная запись Майкрософт пользовательского интерфейса.
