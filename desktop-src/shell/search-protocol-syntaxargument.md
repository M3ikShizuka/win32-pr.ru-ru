# <a name="syntax-argument-the-windows-shell"></a>Аргумент СИНТАКСИСа (оболочка Windows)

`syntax`Аргумент позволяет задать синтаксис запроса. Допустимые значения — это либо синтаксис естественного запроса (НКС), либо расширенный синтаксис запросов (АКС). АКС является значением по умолчанию и предполагается, что он должен быть проанализирован и поддерживаться.

## <a name="example"></a>Например, .

``` syntax
search:query=from:paolo&syntax=AQS
```

### <a name="argument-information"></a>Сведения о аргументе



|                          |                                         |
|--------------------------|-----------------------------------------|
| Минимальная операционная система | Windows Vista с пакетом обновления 1 (SP1) |
