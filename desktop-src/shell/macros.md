# <a name="shell-macros"></a>Макросы оболочки

в этом разделе описываются макросы служебной программы Windows Shell.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                  | Описание                                                                                                                                                                                                                                                     |
|------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ОПРЕДЕЛЕНИЕ \_ PROPERTYKEY**](/windows/desktop/api/Propkeydef/nf-propkeydef-define_propertykey)<br/>           | Используется для упаковки идентификатора формата (FMTID) и идентификатора свойства (PID) в структуру [**PROPERTYKEY**](/windows/win32/api/wtypes/ns-wtypes-propertykey) , представляющую ключ свойства.<br/>                                                                                    |
| [**IID \_ ППВ \_ args**](/windows/win32/api/combaseapi/nf-combaseapi-iid_ppv_args)<br/>                      | Используется для получения указателя на интерфейс, который автоматически предоставляет значение IID запрошенного интерфейса на основе типа используемого указателя интерфейса. Это позволяет избежать распространенной ошибки кодирования, проверяя тип значения, передаваемого во время компиляции.<br/> |
| [**исекуалпропертикэй**](/windows/desktop/api/Propkeydef/nf-propkeydef-isequalpropertykey)<br/>            | Сравнивает элементы двух структур [**PROPERTYKEY**](/windows/win32/api/wtypes/ns-wtypes-propertykey) и возвращает, равны ли они.<br/>                                                                                                                                 |
| [**македллверулл**](/windows/desktop/api/Shlwapi/nf-shlwapi-makedllverull)<br/>                      | Используется для упаковки сведений о версии библиотеки DLL в значение УЛОНГЛОНГ.<br/>                                                                                                                                                                                         |
| [**Нетаддр \_ дисплайеррортип**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_displayerrortip)<br/> | Отображает сообщение об ошибке в всплывающей подсказке, связанной с элементом управления "сетевой адрес".<br/>                                                                                                                                                            |
| [**Нетаддр \_**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_getaddress)<br/>           | Указывает, соответствует ли сетевой адрес указанному типу и формату.<br/>                                                                                                                                                                         |
| [**Нетаддр \_ жеталловтипе**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_getallowtype)<br/>       | Извлекает типы сетевых адресов, принимаемые указанным элементом управления сетью.<br/>                                                                                                                                                                |
| [**Нетаддр \_ сеталловтипе**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_setallowtype)<br/>       | Задает типы сетевых адресов, принимаемые указанным элементом управления для сетевых адресов.<br/>                                                                                                                                                                     |
