# <a name="iuseridentitymanagergetidentitybycookie-method"></a>Метод Иусеридентитиманажер:: Жетидентитибикукие

\[**Жетидентитибикукие** не поддерживается и может быть изменен или недоступен в будущем. Вместо этого используйте [учетные записи пользователей с быстрым переключением пользователей и удаленный рабочий стол](fastuserswitching.md).\]

Возвращает конкретное удостоверение пользователя по файлу cookie, используемому для уникальной идентификации.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetIdentityByCookie(
  [in]  GUID          *uidCookie,
  [out] IUserIdentity **ppIdentity
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*уидкукие* \[ окне\]
</dt> <dd>

Тип: **GUID \** _

Адрес значения _ *GUID**, представляющего файл cookie для удостоверения, которое необходимо получить.

</dd> <dt>

*ппидентити* \[ заполняет\]
</dt> <dd>

Тип: **[ **иусеридентити**](iuseridentity.md)\*\***

Адрес указателя, который получит объект удостоверения пользователя.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **HRESULT**

Результат запроса получения. В случае успеха возвращается значение S \_ ОК. В противном случае будет возвращен один из следующих кодов ошибок.



| Код возврата                                                                                            | Описание                                               |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------|
| <dl> <dt>**\_ \_ не найдено удостоверение E \_**</dt> </dl> | Не удалось найти запрошенное удостоверение.<br/>     |
| <dl> <dt>**Электронные \_ удостоверения \_ отключены**</dt> </dl> | Управление удостоверениями отключено в системе.<br/> |
