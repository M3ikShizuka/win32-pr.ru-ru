# <a name="abn_statechange-message"></a>\_Сообщение АБН STATECHANGE

Уведомляет панель приложений о том, что состояние автоскрытия или Always on панели задач было изменено, т. е. пользователь выбрал или снял флажок "Always On Top" или "автоматически скрывать" на странице свойств панели задач.


```C++
ABN_STATECHANGE 
```



## <a name="parameters"></a>Параметры

Это сообщение не содержит параметров.

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Remarks

Панель приложений может использовать это сообщение уведомления, чтобы установить его состояние в соответствии с соответствующими настройками панели задач (при необходимости).
