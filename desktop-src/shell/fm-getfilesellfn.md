# <a name="fm_getfilesellfn-message"></a>\_Сообщение FM жетфилеселлфн

Отправляется расширением диспетчера файлов для получения сведений о выбранном файле из активного окна диспетчера файлов (в окне каталога или в окне результатов поиска). Выбранный файл может иметь длинное имя файла.

## <a name="parameters"></a>Параметры

<dl> <dt>

*index* 
</dt> <dd>

Отсчитываемый от нуля индекс выбранного файла для извлечения.

</dd> <dt>

*лпфмсгфс* 
</dt> <dd>

Адрес структуры [**FMS \_ жетфилесел**](fms-getfilesel.md) , которая получает сведения о выделенном фрагменте.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает отсчитываемый от нуля индекс выбранного файла, который был получен.

## <a name="remarks"></a>Remarks

Только расширения, поддерживающие длинные имена файлов (например, сетевые расширения), должны использовать это сообщение.

Расширение может использовать сообщение [**FM \_ жетселкаунтлфн**](fm-getselcountlfn.md) для получения числа выбранных файлов.
