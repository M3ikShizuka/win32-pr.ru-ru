# <a name="shellfolderviewoc-object"></a>Объект Шеллфолдервиевок

Перенаправляет события, инициируемые указанным объектом [**шеллфолдервиев**](shellfolderview.md) , в соответствующие обработчики событий **шеллфолдервиевок** .

## <a name="members"></a>Элементы

Объект **шеллфолдервиевок** имеет следующие типы членов:

-   [События](#events)
-   [Методы](#methods)

### <a name="events"></a>События

Объект **шеллфолдервиевок** содержит эти события.



| Событие                                                          | Описание                                                                                                                     |
|:---------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------|
| [**енумдоне**](shellfolderviewoc-enumdone.md)                 | Указывает, что объект [**шеллфолдервиев**](shellfolderview.md) завершил перечисление содержимого папки.<br/> |
| [**SelectionChanged**](shellfolderviewoc-selectionchanged.md) | Указывает, что состояние выбора одного или нескольких элементов в представлении изменилось.<br/>                                     |



 

### <a name="methods"></a>Методы

Объект **шеллфолдервиевок** содержит эти методы.



| Метод                                                   | Описание                                                                                                                                                 |
|:---------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**сетфолдервиев**](shellfolderviewoc-setfolderview.md) | Перенаправляет события указанного объекта [**шеллфолдервиев**](shellfolderview.md) в соответствующий обработчик событий **шеллфолдервиевок** .<br/> |



 

## <a name="remarks"></a>Remarks

Объект [**шеллфолдервиев**](shellfolderview.md) запускает два события, [**енумдоне**](shellfolderviewoc-enumdone.md) и [**SelectionChanged**](shellfolderviewoc-selectionchanged.md), которые обычно обрабатываются приложениями. Однако некоторые приложения должны выполнять обработку событий из ряда объектов **шеллфолдервиев** . Например, в приложении может размещаться элемент управления WebBrowser, позволяющий пользователям перемещаться по ряду папок. Каждая папка имеет собственный объект **шеллфолдервиев** со связанными событиями. Обработка этих событий может быть сложной.

Объект **шеллфолдервиевок** упрощает обработку событий для таких сценариев. Он позволяет приложениям выполнять обработку событий для всех объектов [**шеллфолдервиев**](shellfolderview.md) с помощью одной пары обработчиков событий **шеллфолдервиевок** . Каждый раз, когда пользователь переходит к новой папке, приложение передает связанный объект **шеллфолдервиев** объекту **шеллфолдервиевок** , вызывая [**сетфолдервиев**](shellfolderviewoc-setfolderview.md). Затем при срабатывании события [**енумдоне**](shellfolderviewoc-enumdone.md) или [**SelectionChanged**](shellfolderviewoc-selectionchanged.md) объект **шеллфолдервиевок** перенаправляет событие в собственный обработчик для обработки.
