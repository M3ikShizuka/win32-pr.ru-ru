# <a name="folder-object"></a>Объект Folder

Представляет папку оболочки. Этот объект содержит свойства и методы, позволяющие получить сведения о папке.

## <a name="members"></a>Элементы

Объект **Folder** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **Folder** содержит следующие методы.



| Метод                                      | Описание                                                                                                                |
|:--------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------|
| [**копихере**](folder-copyhere.md)         | Копирует элемент или элементы в папку.<br/>                                                                            |
| [**жетдетаилсоф**](folder-getdetailsof.md) | Извлекает сведения об элементе в папке. Например, его размер, тип или время последнего изменения.<br/> |
| [**Элементы**](folder-items.md)               | Извлекает объект [**фолдеритемс**](folderitems.md) , представляющий коллекцию элементов в папке.<br/>    |
| [**мовехере**](folder-movehere.md)         | Перемещает элемент или элементы в эту папку.<br/>                                                                          |
| [**NewFolder**](folder-newfolder.md)       | Создает новую папку.<br/>                                                                                           |
| [**ParseName**](folder-parsename.md)       | Создает и возвращает объект [**FolderItem**](folderitem.md) , представляющий указанный элемент.<br/>                 |



 

### <a name="properties"></a>Элемент Property

Объект **Folder** имеет эти свойства.



| Свойство.                                               | Тип доступа          | Описание                                          |
|:-------------------------------------------------------|:---------------------|:-----------------------------------------------------|
| [**Приложение**](folder-application.md)<br/>   | Только для чтения<br/> | Содержит объект приложения папки.<br/> |
| [**Parent**](folder-parent.md)<br/>             | Только для чтения<br/> | Не реализован.<br/>                          |
| [**парентфолдер**](folder-parentfolder.md)<br/> | Только для чтения<br/> | Содержит объект родительской **папки** .<br/>    |
| [**Титуль**](folder-title.md)<br/>               | Только для чтения<br/> | Содержит заголовок папки.<br/>         |



 

## <a name="remarks"></a>Remarks

> [!Note]  
> Не все методы реализуются для всех папок. Например, метод [**PARSENAME**](folder-parsename.md) не реализован для папки панели управления ( \_ элементы управления CSID). При попытке вызвать нереализованный метод возникнет ошибка 0x800A01BD (Decimal 445).
