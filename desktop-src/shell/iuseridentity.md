# <a name="iuseridentity-interface"></a>Интерфейс Иусеридентити

\[**Иусеридентити** не поддерживается и может быть изменен или недоступен в будущем. Вместо этого используйте [учетные записи пользователей с быстрым переключением пользователей и удаленный рабочий стол](fastuserswitching.md).\]

Предоставляет методы, предоставляющие данные для идентификации, реестра и папки для конкретного удостоверения пользователя.

## <a name="members"></a>Элементы

Интерфейс **иусеридентити** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Иусеридентити** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иусеридентити** содержит следующие методы.



| Метод                                                         | Описание                                                                                      |
|:---------------------------------------------------------------|:-------------------------------------------------------------------------------------------------|
| [**GetCookie**](iuseridentity-getcookie.md)                   | Не рекомендуется. Возвращает файл cookie, используемый для уникальной идентификации этого удостоверения пользователя.<br/>             |
| [**жетидентитифолдер**](iuseridentity-getidentityfolder.md)   | Не рекомендуется. Возвращает файловую папку, связанную с этим удостоверением.<br/>                       |
| [**GetName**](iuseridentity-getname.md)                       | Не рекомендуется. Возвращает имя, связанное с этим удостоверением пользователя.<br/>                         |
| [**опенидентитирегкэй**](iuseridentity-openidentityregkey.md) | Не рекомендуется. Извлекает ключ в реестре, соответствующий этому удостоверению пользователя.<br/> |



 

## <a name="remarks"></a>Примечания

Этот интерфейс предоставляет сведения, соответствующие конкретному удостоверению пользователя, присутствующему в системе. Вы можете получить доступ к папке удостоверений этого пользователя, ключу реестра и файлу cookie для идентификации, а также получить имя, связанное с удостоверением пользователя.
