# <a name="stackedby-argument-the-windows-shell"></a>Аргумент СТАККЕДБИ (оболочка Windows)

`stackedby`Аргумент указывает столбец свойств, по которому должны быть упорядочены результаты стека. Можно выполнить стек по любому допустимому свойству из системы свойств.

## <a name="example"></a>Например, .


```
search:query=vacation&stackedby=System.Author
```



### <a name="argument-information"></a>Сведения о аргументе



|                          |                                         |
|--------------------------|-----------------------------------------|
| Минимальная операционная система | Windows Vista с пакетом обновления 1 (SP1) |
