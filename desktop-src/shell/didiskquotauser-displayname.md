# <a name="didiskquotauserdisplayname-property"></a>Дидисккуотаусер. DisplayName, свойство

Возвращает отображаемое имя пользователя.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
DisplayName = DIDiskQuotaUser.DisplayName
```



## <a name="property-value"></a>Значение свойства

Строковое значение, заданное отображаемым именем пользователя.

## <a name="remarks"></a>Remarks

Это свойство содержит понятное имя пользователя. Его значение не обязательно определено.
