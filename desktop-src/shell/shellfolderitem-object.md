# <a name="shellfolderitem-object"></a>Объект Шеллфолдеритем

Расширяет объект [**FolderItem**](folderitem.md) . В дополнение к свойствам и методам, поддерживаемым **FolderItem**, **шеллфолдеритем** имеет два дополнительных метода.

## <a name="members"></a>Элементы

Объект **шеллфолдеритем** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Объект **шеллфолдеритем** содержит эти методы.



| Метод                                                       | Описание                                                                                                                                                                                         |
|:-------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ExtendedProperty**](shellfolderitem-extendedproperty.md) | Возвращает значение свойства из набора свойств элемента. Свойство может быть задано либо по имени, либо идентификатором формата набора свойств (FMTID) и идентификатором свойства (PID).<br/> |
| [**инвокевербекс**](invokeverbex.md)                         | Выполняет команду для элемента оболочки.<br/>                                                                                                                                                         |
