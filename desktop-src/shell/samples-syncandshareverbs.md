## <a name="running-the-sample"></a>Запуск примера

Чтобы запустить пример (Sync), выполните следующие действия.

1.  Перейдите к каталогу, содержащему `sync.reg` файл.
2.  Введите `sync.reg ` в командной строке или дважды щелкните значок для `sync.reg` его регистрации.
3.  откройте обозреватель Windows и выберите файл.
4.  Щелкните параметр **Синхронизация** на панели команд и выберите подпараметр, например **Paint**.

Чтобы запустить пример (Share), сделайте следующее:

1.  Перейдите в каталог, содержащий `share.reg` файл.
2.  Введите `share.reg` в командной строке или дважды щелкните значок для `share.reg` его регистрации.
3.  откройте обозреватель Windows и выберите файл. Выберите параметр **общий доступ** на панели команд.
4.  Щелкните параметр **поделиться с** на панели команд и выберите подпараметр, например **Paint**.

## <a name="removing-the-sample"></a>Удаление образца

Чтобы удалить пример (Sync), выполните следующие действия.

1.  Перейдите в каталог, содержащий файл Унинсталлсинк. reg.
2.  Введите `uninstallsync.reg` в командной строке или дважды щелкните значок для `uninstallsync.reg` .

Чтобы удалить пример (Share), сделайте следующее:

1.  Перейдите в каталог, содержащий файл Унинсталлшаре. reg.
2.  Введите `uninstallshare.reg` в командной строке или дважды щелкните значок для `uninstallshare.reg.`
