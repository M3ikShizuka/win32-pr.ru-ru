-   [Функции обработки пути оболочки](shlwapi-path.md)
-   [Функции обработки реестра оболочки](shlwapi-reg.md)
-   [Функции обработки строк оболочки](shlwapi-string.md)
-   [Различные служебные функции оболочки](shlwapi-misc.md)
