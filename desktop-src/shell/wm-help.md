# <a name="wm_help-message"></a>\_Справочное сообщение WM

Указывает, что пользователь нажал клавишу F1. Если при нажатии клавиши F1 меню активно, **\_ Справка WM** отправляется в окно, связанное с меню; в противном **случае \_ Справка WM** отправляется в окно, имеющее фокус клавиатуры. Если ни один из окон не имеет фокуса клавиатуры, **\_ Справка WM** отправляется в текущее активное окно.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Должен равняться нулю.

</dd> <dt>

*лфи* 
</dt> <dd>

Адрес структуры [**HELPINFO**](/windows/win32/api/winuser/ns-winuser-helpinfo) , содержащей сведения о пункте меню, элементе управления, диалоговом окне или окне, для которого запрашивается справка.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true**.

## <a name="remarks"></a>Remarks

Функция [**дефвиндовпрок**](/windows/desktop/api/winuser/nf-winuser-defwindowproca) передает **WM \_ справку** родительскому окну дочернего окна или владельцу окна верхнего уровня.
