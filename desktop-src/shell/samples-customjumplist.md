## <a name="building-the-sample"></a>Построение образца

Чтобы построить образец из командной строки, выполните следующую команду:

1.  Откройте окно командной строки и перейдите в каталог проекта **кустомжумплист** .
2.  Введите `msbuild CustomJumpListSample.sln`.

чтобы создать пример с использованием Microsoft Visual Studio (предпочтительно):

1.  откройте обозреватель Windows и перейдите в каталог проекта **кустомжумплист** .
2.  Дважды щелкните значок файла Кустомжумплистсампле. sln, чтобы открыть проект в Visual Studio.
3.  В меню **Построение** выберите пункт **Построить решение**.

## <a name="running-the-sample"></a>Запуск примера

1.  перейдите в каталог, содержащий новый исполняемый файл, используя командную строку или обозреватель Windows.
2.  В командной строке введите `CustomJumpListSample.exe` . кроме того, в обозревателе Windows дважды щелкните значок CustomJumpListSample.exe.
3.  Этот пример должен запускаться от имени администратора при первом запуске, чтобы он мог установить необходимые регистрации типов файлов. После регистрации типов файлов пример можно запустить от имени обычного пользователя.
4.  Выберите параметры в меню в примере приложения, чтобы увидеть, как они влияют на список переходов приложения на панели задач.
