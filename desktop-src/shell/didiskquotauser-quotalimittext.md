# <a name="didiskquotauserquotalimittext-property"></a>Дидисккуотаусер. Куоталимиттекст, свойство

Возвращает текущую [**предельную квоту**](diskquotacontrol-object.md) пользователя в виде текстовой строки.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
QuotaLimitText = DIDiskQuotaUser.QuotaLimitText
```



## <a name="property-value"></a>Значение свойства

Строковое значение, содержащее текущую квоту пользователя.
