# <a name="parsefield-function"></a>Функция Парсефиелд

\[В настоящее время функция **парсефиелд** должна быть доступна для использования в следующей версии операционной системы Microsoft Windows. Он может быть изменен или недоступен в последующих версиях.\]

Считывает строку из файла Setup. INF и извлекает указанное поле из строки.

## <a name="syntax"></a>Синтаксис


```C++
bool ParseField(
  _In_ LPCTSTR *szData,
  _In_ int     n,
  _In_ LPTSTR  *szBuf,
  _In_ int     iBufLen
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*сздата* \[ окне\]
</dt> <dd>

Тип: **LPCTSTR \** _

Указатель на строку из файла Setup. INF.

</dd> <dt>

_n * \[ в\]
</dt> <dd>

Тип: **int**

**Целое** число, указывающее поле для извлечения.

<dt>



  (0)


</dt> <dd>

Указывает поле перед знаком равенства (=).

</dd> <dt>



 (1)


</dt> <dd>

</dd> </dl> </dd> <dt>

*сзбуф* \[ окне\]
</dt> <dd>

Указатель на буфер, который получает извлеченное поле.

</dd> <dt>

_iBufLen * \[ в\]
</dt> <dd>

Тип: **int**

**Целое** число, получающее размер буфера, получающего извлеченное поле.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **bool** .

Возвращает **значение true** , если функция выполнена успешно, и **значение false** в случае сбоя.

## <a name="remarks"></a>Примечания

Поля в строке должны быть разделены запятыми.

Начальные и конечные пробелы удаляются.
