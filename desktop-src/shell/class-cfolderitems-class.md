# <a name="cfolderitems-class"></a>Класс Кфолдеритемс

**Кфолдеритемс** — это коллекция [**фолдеритемс**](folderitems.md). Он реализует следующие интерфейсы: [**иперсистфолдер**](/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ipersistfolder), [**IObjectWithSite**](/windows/win32/api/ocidl/nn-ocidl-iobjectwithsite), [**FolderItems3**](folderitems3-object.md), [**иобжектсафети**](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa768224(v=vs.85)), [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch).

## <a name="remarks"></a>Remarks

Класс **кфолдеритемс** перечисляет только элементы из определенной папки.
