## <a name="description"></a>Описание

В этом примере показано, как реализовать команду оболочки с помощью метода Дроптаржет. этот метод является предпочтительным для реализаций команд, которые должны работать в Windows XP. В этом примере реализуется изолированный объект модели COM, но предполагается, что реализация глагола будет интегрирована в существующие приложения. Для этого основной объект приложения регистрирует фабрику класса для себя. Этот объект реализует [**интерфейс IDropTarget**](/windows/win32/api/oleidl/nn-oleidl-idroptarget) для команд приложения. Обратите внимание, что COM запускает приложение, если оно еще не запущено, но подключается к работающему экземпляру приложения, если оно существует.

## <a name="building-the-sample"></a>Построение образца

Чтобы построить образец из командной строки, выполните следующую команду:

1.  Откройте окно командной строки и перейдите в каталог проекта **дроптаржетверб** .
2.  Введите `msbuild DropTargetVerb.sln`.

чтобы создать пример с использованием Microsoft Visual Studio (предпочтительно):

1.  откройте обозреватель Windows и перейдите в каталог проекта **дроптаржетверб** .
2.  Дважды щелкните значок файла Дроптаржетверб. sln, чтобы открыть проект в Visual Studio.
3.  В меню **Построение** выберите пункт **Построить решение**.

## <a name="running-the-sample"></a>Запуск примера

1.  перейдите в каталог, содержащий новый исполняемый файл, используя командную строку или обозреватель Windows.
2.  В командной строке введите `DropTargetVerb.exe` . кроме того, в обозревателе Windows дважды щелкните значок DropTargetVerb.exe.
3.  Следуйте инструкциям в отображаемом диалоговом окне.
