# <a name="createuserprofileex-function"></a>Функция Креатеусерпрофиликс

\[Эта функция недоступна в Windows Vista.\]

Создает профиль пользователя для указанного пользователя.

## <a name="syntax"></a>Синтаксис


```C++
BOOL WINAPI CreateUserProfileEx(
  _In_      PSID    pSid,
  _In_      LPCTSTR lpUserName,
  _In_opt_  LPCTSTR lpUserHive,
  _Out_opt_ LPTSTR  lpProfileDir,
  _In_      DWORD   dwDirSize,
  _In_      BOOL    bWin9xUpg
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пустой pSid* \[ окне\]
</dt> <dd>

Тип: **пустой pSid**

Идентификатор безопасности нового пользователя.

</dd> <dt>

*лпусернаме* \[ окне\]
</dt> <dd>

Тип: **LPCTSTR**

Указатель на буфер, содержащий имя пользователя нового пользователя.

</dd> <dt>

*лпусерхиве* \[ в необязательное\]
</dt> <dd>

Тип: **LPCTSTR**

Указатель на буфер, содержащий [куст реестра](../sysinfo/registry-hives.md) для использования. Этот параметр может принимать значение **NULL**.

</dd> <dt>

*лппрофиледир* \[ out, необязательно\]
</dt> <dd>

Тип: **LPTSTR**

Указатель на буфер, который при возврате этой функции получает путь к каталогу профиля пользователя. Этот параметр может принимать значение **NULL**.

</dd> <dt>

*двдирсизе* \[ окне\]
</dt> <dd>

Тип: **DWORD**

Размер буфера, заданного параметром *лппрофиледир*, в тчарс.

</dd> <dt>

*bWin9xUpg* \[ окне\]
</dt> <dd>

Тип: **bool** .

**Значение true** , если профиль пользователя создается в ходе миграции профиля из Windows 9x; в противном случае — **значение false**.

Если **значение равно true**, профиль пользователя настраивается в каталоге профиля по умолчанию — обычно C: \\ Documents and Settings \\ *username*. Если этот каталог уже существует, он используется. В противном случае он будет создан.

Если задано **значение false**, каталог профиля по умолчанию создается, если он не существует. Если каталог профиля по умолчанию уже существует, для этого профиля пользователя создается новый каталог.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Тип: **bool** .

Возвращает **значение true** , если новый профиль пользователя успешно создан; в противном случае — **значение false**.

## <a name="remarks"></a>Примечания

Эта функция не объявлена в заголовках пакета средств разработки программного обеспечения и не имеет связанной библиотеки импорта. Для связи с Userenv.dll необходимо использовать функции [**LoadLibrary**](/windows/win32/api/libloaderapi/nf-libloaderapi-loadlibrarya) и [**GetProcAddress**](/windows/win32/api/libloaderapi/nf-libloaderapi-getprocaddress) . Версия ANSI функции **креатеусерпрофиликса** упоминается из Userenv.dll как порядковый номер 153. Версия Юникода, **креатеусерпрофиликсв** , упоминается как порядковый номер 154.
