# <a name="folderitems3-object"></a>Объект FolderItems3

Расширяет объект [**FolderItems2**](folderitems2-object.md) . Этот объект поддерживает дополнительный метод и свойство.

## <a name="members"></a>Элементы

Объект **FolderItems3** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **FolderItems3** содержит эти методы.



| Метод                                | Описание                                                       |
|:--------------------------------------|:------------------------------------------------------------------|
| [**Filter**](folderitems3-filter.md) | Задает фильтр с подстановочными знаками, применяемый к возвращаемым элементам.<br/> |



 

### <a name="properties"></a>Свойства

Объект **FolderItems3** имеет следующие свойства.



| Свойство                                       | Тип доступа          | Описание                                                       |
|:-----------------------------------------------|:---------------------|:------------------------------------------------------------------|
| [**Команды**](folderitems3-verbs.md)<br/> | Только для чтения<br/> | Возвращает список команд, общих для всех элементов папки.<br/> |
