# <a name="shellwindows-object"></a>Объект Шеллвиндовс

Представляет коллекцию открытых окон, принадлежащих оболочке. Методы, связанные с этими объектами, могут управлять и выполнять команды в оболочке и получать другие объекты, связанные с оболочкой.

## <a name="members"></a>Элементы

Объект **шеллвиндовс** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **шеллвиндовс** содержит эти методы.



| Метод                                                 | Описание                                                                                                         |
|:-------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](shellwindows--newenum-method-7ral.md) | Создает и возвращает новый объект **шеллвиндовс** , являющийся копией этого объекта **шеллвиндовс** .<br/>        |
| [**Элемент**](shellwindows-item.md)                      | Извлекает объект [**InternetExplorer**](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa752084(v=vs.85)) , представляющий окно оболочки.<br/> |



 

### <a name="properties"></a>Свойства

Объект **шеллвиндовс** имеет следующие свойства.



| Свойство                                       | Тип доступа          | Описание                                                |
|:-----------------------------------------------|:---------------------|:-----------------------------------------------------------|
| [**Расчета**](shellwindows-count.md)<br/> | Только для чтения<br/> | Содержит число элементов в коллекции.<br/> |
