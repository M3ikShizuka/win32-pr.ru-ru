# <a name="data-access-and-storage"></a>Доступ к данным и хранилище

В Windows есть API, компоненты и службы, поддерживающие классические приложения в доступе к данным и хранилище. Они предоставляют:

-   Управление файлами и файловой системой.
-   Доступ к базе данных.
-   Поддержка перемещения, синхронизации и репликации данных.
-   Доступ к файлам XML, пакетам и журналам.
-   Основное изображение.
-   Поддержка резервного копирования.
