# <a name="process-status-api"></a>API состояния процесса

Программный интерфейс "состояние процесса" (PSAPI) — это вспомогательная библиотека, которая упрощает получение сведений о процессах и драйверах устройств. Дополнительные сведения см. в следующих разделах:

-   [О PSAPI](about-psapi.md)
-   [Использование PSAPI](using-psapi.md)
-   [Справочник по PSAPI](psapi-reference.md)

Эти функции доступны в Psapi.dll.

Одни и те же сведения доступны через данные о производительности в реестре. Дополнительные сведения см. в статье [Performance Counters](/windows/desktop/PerfCtrs/performance-counters-portal).
