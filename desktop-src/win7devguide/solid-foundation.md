# <a name="solid-foundation"></a>Твердая основа

Windows 7 предоставляет высокопроизводительную платформу и средства разработки, которые обеспечивают основные принципы работы операционной системы. Windows 7 построены на платформе Windows Vista, предоставляя разработчикам возможность создавать приложения, совместимые с обеими платформами, одновременно ограничивая проблемы совместимости приложений. Windows 7 значительно улучшает производительность и управление питанием, позволяя создавать приложения, которые оптимизируют работу мобильных приложений. В результате приложения будут визуально привлекательными, проще в создании и поддерживают более широкий спектр международных стандартов. Усовершенствования также были сделаны в глобальной поддержке, доступности и развертывании приложений.
