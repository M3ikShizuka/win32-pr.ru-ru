# <a name="devices-windows-7-developer-guide"></a>устройства (руководством для разработчиков Windows 7)

устройства являются фундаментальной частью работы пк, а Windows 7 — новые возможности для разработчиков приложений, взаимодействующих с устройствами. Платформа взаимодействия с устройствами позволяет связать приложения и службы с определенным устройством, чтобы пользователи могли получить максимальную пользу их периферийных устройств сразу после подключения. Платформа датчика предоставляет набор API-интерфейсов для обнаружения и обмена данными с устройствами датчиков, которые обеспечивают новое поколение приложений, которые знают о своих средах. Платформа расположения предоставляет новые API-интерфейсы для использования данных о расположении от приемника GPS или других служб, которые обеспечивают поведение приложений для мобильных пользователей, зависящих от местоположения. (См. раздел [Основные сведения об устройстве — обзор](https://www.microsoft.com/whdc/device/default.mspx).)

## <a name="device-experience-platform"></a>Платформа для работы с устройствами

Windows 7 сочетает программное обеспечение и службы для создания новых возможностей для мобильных телефонов, портативных проигрывателей, камер и принтеров. Windows 7 упрощает использование этих устройств непосредственно из Windows рабочего стола. кроме того, он предоставляет изготовителям устройств четкое размещение на рабочем столе Windows, с возможностями фирменной символики и простым интерфейсом для представления функциональных возможностей и служб, поддерживаемых устройством.

с помощью платформы взаимодействия с устройствами каждый сеанс Windows превращается в портал, чтобы клиенты могли получить больше ценности с устройств. Платформа взаимодействия с устройствами позволяет пользователям подключаться с производителем устройства, находить и использовать связанные службы, а также изучать аксессуары. Так как устройство подключено к веб-службам Майкрософт, компании устройств могут обновлять работу даже после доставки устройств потребителям. платформа взаимодействия с устройствами может создать интерфейс, похожий на приложение, для устройств Windows *Logo* , таких как мобильные телефоны.

платформа взаимодействия с устройствами предоставляет приложениям доступ к таким устройствам, как мобильные телефоны и проигрыватели мультимедиа, которые реализуют службы через *протокол передачи мультимедиа (MTP)* или модель драйверов [Windowsных портативных устройств](https://www.bing.com/search?q=Windows+Portable+Devices) .

Чтобы обеспечить синхронизацию персональных данных между компьютером и устройством, платформа взаимодействия с устройствами размещает новую платформу синхронизации для подключенных устройств и предоставляет пользовательский интерфейс для выбора целевых приложений для синхронизации данных, таких как **Контакты**, **Календарь** и **задачи**. (см. раздел [Windows Device Experience](https://www.microsoft.com/whdc/device/DeviceExperience/default.mspx).)

## <a name="windows-biometric-framework"></a>Биометрическая платформа Windows

биометрическая платформа Windows (вбф) предоставляет API, позволяющий приложениям использовать устройства отпечатков пальцев для регистрации, идентификации и проверки удостоверений пользователей без получения прямого доступа к любому биометрному оборудованию или образцам. вы можете использовать вбф с устройствами отпечатка пальца, которые имеют драйверы интерфейса биометрического устройства (вбди) Windows. ВБФ расширяется с помощью подключаемых адаптеров, которые управляют взаимодействием датчиков, биометрического сопоставления и хранилищем шаблонов. Это гарантирует, что ВБФ можно будет использовать с широким спектром датчиков отпечатков пальцев. в Windows 7 устройства чтения отпечатков пальцев могут использовать вбф для проверки подлинности во время *контроля учетных записей* и Windows входа. (см. [биометрическая платформа Windows API](../secbiomet/biometric-service-api-portal.md).)
