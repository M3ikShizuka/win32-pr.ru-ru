# <a name="devices-windows-7-developer-guide"></a>Устройства (руководством для разработчиков Windows 7)

Устройства являются фундаментальной частью работы с ПК, и Windows 7 предоставляет новые возможности для разработчиков приложений, взаимодействующих с устройствами. Платформа взаимодействия с устройствами позволяет связать приложения и службы с определенным устройством, чтобы пользователи могли получить максимальную пользу их периферийных устройств сразу после подключения. Платформа датчика предоставляет набор API-интерфейсов для обнаружения и обмена данными с устройствами датчиков, которые обеспечивают новое поколение приложений, которые знают о своих средах. Платформа расположения предоставляет новые API-интерфейсы для использования данных о расположении от приемника GPS или других служб, которые обеспечивают поведение приложений для мобильных пользователей, зависящих от местоположения. (См. раздел [Основные сведения об устройстве — обзор](https://www.microsoft.com/whdc/device/default.mspx).)

## <a name="device-experience-platform"></a>Платформа для работы с устройствами

Windows 7 сочетает в себе программное обеспечение и службы для создания новых возможностей для мобильных телефонов, портативных проигрывателей, камер и принтеров. Windows 7 упрощает использование этих устройств непосредственно с рабочего стола Windows. Кроме того, он предоставляет изготовителям устройств четкое размещение на рабочем столе Windows, с возможностями фирменной символики и простым интерфейсом для представления функциональных возможностей и служб, поддерживаемых устройством.

С помощью платформы взаимодействия с устройствами каждый сеанс Windows станет порталом, чтобы клиенты могли получить больше ценности с устройств. Платформа взаимодействия с устройствами позволяет пользователям подключаться с производителем устройства, находить и использовать связанные службы, а также изучать аксессуары. Так как устройство подключено к веб-службам Майкрософт, компании устройств могут обновлять работу даже после доставки устройств потребителям. Платформа взаимодействия с устройствами может создать интерфейс, аналогичный приложению, для устройств с *эмблемой* Windows, таких как мобильные телефоны.

Платформа взаимодействия с устройствами предоставляет приложениям доступ к таким устройствам, как мобильные телефоны и проигрыватели мультимедиа, которые реализуют службы через *протокол передачи мультимедиа (MTP)* или модель драйверов [переносных устройств Windows](https://www.bing.com/search?q=Windows+Portable+Devices) .

Чтобы обеспечить синхронизацию персональных данных между компьютером и устройством, платформа взаимодействия с устройствами размещает новую платформу синхронизации для подключенных устройств и предоставляет пользовательский интерфейс для выбора целевых приложений для синхронизации данных, таких как **Контакты**, **Календарь** и **задачи**. (См. раздел [взаимодействие с устройствами Windows](https://www.microsoft.com/whdc/device/DeviceExperience/default.mspx).)

## <a name="windows-biometric-framework"></a>Биометрическая платформа Windows

Биометрическая платформа Windows (ВБФ) предоставляет API, позволяющий приложениям использовать устройства отпечатков пальцев для регистрации, идентификации и проверки удостоверений пользователей без получения прямого доступа к любому биометрному оборудованию или образцам. ВБФ можно использовать с устройствами отпечатка пальца, которые имеют драйверы интерфейса биометрического устройства Windows (ВБДИ). ВБФ расширяется с помощью подключаемых адаптеров, которые управляют взаимодействием датчиков, биометрического сопоставления и хранилищем шаблонов. Это гарантирует, что ВБФ можно будет использовать с широким спектром датчиков отпечатков пальцев. В Windows 7 читатели отпечатков пальцев могут использовать ВБФ для проверки подлинности во время *контроля учетных записей* и входа в Windows. (См. [БИОМЕТРИЧЕСКАЯ платформа Windows API](../secbiomet/biometric-service-api-portal.md).)
