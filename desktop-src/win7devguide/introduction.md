# <a name="introduction-windows-7-developer-guide"></a>Введение (руководством для разработчиков Windows 7)

Разработка программного обеспечения еще не была интересна. Мир никогда не был подключен к Интернету, интерактивным и мобильным. Взаимодействие с пользователем и содержимое никогда не имели более высокой точности. Усовершенствования в связи, мобильность, естественные интерфейсы, графика и мультимедиа позволяют использовать новые сценарии и возможности для разработчиков. В то же время ожидание никогда не было выше для повышения производительности, гибкости и взаимодействия. Платформа для разработчиков мирового класса должна обеспечивать гибкую и полную основу для разработчиков и их решений, а также создавать приложения, устанавливающие их отдельно от конкуренции и увеличивая их инвестиции.

Операционная система Windows 7 является важнейшей платформой для разработчиков. Прослушивая наших клиентов, просматривая сценарии завтрашнего дня и создавая возможности Windows Vista, корпорация Майкрософт предлагает разработчикам широкий спектр вариантов выбора и возможностей, предоставляемых платформой для разработчиков Windows, а также предоставляет разработчикам возможность создавать творческие решения, которые поддерживают мобильные и подключенные, высококачественные и обеспечивают интуитивно понятный пользовательский интерфейс. Важнее всего то, что основные понятия платформы разработки, такие как безопасность, производительность и совместимость, являются важнейшими приоритетами в Windows 7.

В этом руководство обобщены основные принципы разработки для каждого из следующих трех областей:

[Твердая основа](solid-foundation.md)

Удобство работы пользователей начинается с надежного фундамента. Windows 7 предоставляет надежную платформу разработки и инновационные средства, обеспечивающие дополнительные возможности, повышенную совместимость приложений, лучшую производительность и сложную поддержку документов. Это приводит к простой, более надежной среде для ваших приложений и привычному интуитивному пользователю.

Мы сохранили единообразный, предсказуемый и простой в использовании интерфейс пользователя, добавляя поддержку нескольких касаний, прокручивать и другие интуитивно понятные функции взаимодействия, которые используют преимущества новейших возможностей ПК и мобильных устройств. С Windows 7 разработчики могут создавать правильные пользовательские интерфейсы для своих приложений. (См. [руководство по взаимодействию с пользователем Windows](/windows/apps/desktop/).)

[Более широкие возможности приложений](richer-application-experiences.md)

Windows 7 позволяет разработчикам создавать приложения уже сегодня, которые будут выполняться на компьютерах завтрашнего дня. Программное обеспечение и устройства с поддержкой служб, такие как мобильные телефоны, портативные проигрыватели мультимедиа и Цифровые камеры, постоянная связь по запросу и расширенные приложения. Windows 7 предоставляет платформу для обеих платформ, что упрощает разработчикам использование функций и функций оборудования следующего поколения, в то же время гарантируя, что пользователи всегда включены и всегда подключены. (См. раздел [Центр разработчиков оборудования для Windows](https://www.microsoft.com/whdc/default.mspx).)

[Лучшее из Windows и Интернета](the-best-of-windows-and-the-web.md)

Windows 7 позволяет легко подключиться и оставаться подключенными. Сетевые подключения Windows предоставляют разработчикам возможности для улучшения кэширования и совместного использования данных для повышения производительности сети. Диагностика сети в Windows 7 предоставляют разработчикам актуальные сведения для мониторинга сетевых проблем. Windows Internet Explorer 8 — это основная платформа для создания быстрых веб-интерфейсов. Если целью является подключение устройств, подключение к сети или подключение к *Интернету*, Windows 7 предоставляет улучшенную платформу. (См. раздел [Разработка веб-приложений](https://msdn.microsoft.com/library/aa155073.aspx).)
