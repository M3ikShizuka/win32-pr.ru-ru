# <a name="federated-search"></a>Федеративный поиск

Windows 7 поддерживает поиск документов за пределами компьютера пользователя. Разработчики и специалисты по ИТ могут включать поисковые системы, репозитории документов, веб-приложения и собственные хранилища данных для поиска в Windows 7 без необходимости писать и развертывать клиентский код. Это позволяет конечным пользователям выполнять поиск в корпоративной интрасети или в Интернете так же легко, как и при поиске локальных файлов в одном привычном интерфейсе Windows.

Поддержка *OpenSearch* , совместимого с Windows 7, разработчикам и специалистам по ИТ можно добавить в любое существующее веб-приложение с возможностью поиска, добавив выходные данные *RSS* или *Atom* , как определено в общедоступном стандарте [OpenSearch](http://www.opensearch.org/) . Затем файлы описания *OpenSearch* могут быть созданы разработчиками и опытными пользователями для определения параметров подключения, позволяющих настольному клиенту создавать соединитель поиска для службы.

Сервер SharePoint Search Server также может запрашивать эти совместимые службы *OpenSearch* . Это обеспечивает также доступ к службе поиска из веб-интерфейса центра поиска *SharePoint* .

Windows 7 поддерживает поиск документов за пределами собственного ПК пользователя, а также предварительный просмотр результатов поиска
