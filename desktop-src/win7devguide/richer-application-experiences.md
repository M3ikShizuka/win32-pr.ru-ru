# <a name="richer-application-experiences"></a>Более широкие возможности приложений

Windows 7 позволяет разработчикам создавать отличительные и интуитивно понятные приложения, которые значительно улучшают возможности обнаружения, удобства использования и полноценному. новые методы интеграции с настольными системами помещают функциональные возможности приложения прямо под рукой, а Windows Explorer и *библиотеки* предоставляют простой доступ к информации, которая обеспечивает высокую ценность. элемент управления ленты Windows и Windowsная структура анимации упрощают создание интерактивных и привлекательных пользовательских интерфейсов. Новые сенсорные API позволяют естественным взаимодействиям с помощью многофункционального и сенсорного панорамирования, а интерфейсы API манипуляций и инерции позволяют использовать впечатляющие визуальные эффекты.

Быстрые усовершенствования в технологии оборудования и программного обеспечения также приводят к более высокому пользовательскому интерфейсу. в Windows 7 эти усовершенствования посвящены управлению разработчиками с помощью новых и гибких интерфейсов api, которые используют все преимущества технологии, а также облегчают разработку привлекательных приложений.
