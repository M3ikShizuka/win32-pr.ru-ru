# <a name="richer-application-experiences"></a>Более широкие возможности приложений

Windows 7 позволяет разработчикам создавать отличительные и интуитивно понятные приложения, которые значительно улучшают возможности обнаружения, удобства использования и полноценному. Новые методы интеграции настольных систем помещают функциональные возможности приложения непосредственно под рукой, а Проводник и *библиотеки* Windows предоставляют легкий доступ к ценным сведениям. Элемент управления ленты Windows и платформа Windows Animation Framework упрощают создание интерактивных и привлекательных пользовательских интерфейсов. Новые сенсорные API позволяют естественным взаимодействиям с помощью многофункционального и сенсорного панорамирования, а интерфейсы API манипуляций и инерции позволяют использовать впечатляющие визуальные эффекты.

Быстрые усовершенствования в технологии оборудования и программного обеспечения также приводят к более высокому пользовательскому интерфейсу. В Windows 7 эти усовершенствования посвящены управлению разработчиками с помощью новых и гибких интерфейсов API, которые используют все преимущества технологии, а также упрощают разработку привлекательных приложений.
