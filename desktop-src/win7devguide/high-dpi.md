# <a name="high-dpi"></a>Высокое разрешение

По мере продвижения технологий отображения, производители увеличили число пикселей, поддерживаемое их отображением. В то время как текст, изображения и элементы пользовательского интерфейса выглядят гораздо более четкими и удобочитаемыми при отображении высокого разрешения, операционная система должна масштабироваться для поддержки визуального интерфейса; в противном случае все просто выглядит меньше.

Windows 7 поддерживает дисплеи высокого *dpi* . На рынке данных предполагается, что развертывание экранов с высоким *разрешением* (120-144 точек на дюйм) увеличится в течение периода времени Windows 7. При выполнении собственных разрешений на этих экранах многие приложения отображаются очень мелкими, если они не используют *высокое разрешение dpi*. Некоторые приложения (например, Windows Internet Explorer) имеют функции масштабирования шрифтов, позволяющие пользователям изменять масштаб, но многие приложения не имеют. Функция *высокого DPI* в Windows 7:

-   Обеспечивает оптимальное взаимодействие Windows и приложений на стандартном оборудовании (параметры *dpi* оптимизированы в соответствии с возможностями оборудования).
-   Позволяет оболочке Windows и другим приложениям на основе Windows выглядеть хорошо с различными параметрами *dpi* .
-   Учитывает параметры *dpi* по умолчанию, основанные на спецификациях оборудования и возможностях.
-   Позволяет пользователям персонализировать параметры *dpi* без перезагрузки.
-   Гарантирует, что экран всегда будет иметь собственное разрешение.

Функция масштабирования *dpi* в Windows масштабирует шрифты и элементы пользовательского интерфейса (например, кнопки, значки и поля ввода) на вычисленный процент, как указано в параметре *dpi* . Это отличается от масштабирования, которое происходит при понижении разрешения экрана. В случае масштабирования *dpi* Windows предоставляет шрифты и элементы пользовательского интерфейса, которые рисуются более чем в пикселях, что приводит к увеличению и более высокой точности, а также более четкому интерфейсу Windows. Сторонние приложения для Windows могут использовать параметры *высокого DPI* и соответствующим образом настроить пользовательский интерфейс, объявив данные *высокого DPI* . Разработчикам приложений больше не следует рассчитывать на то, что 96 dpi является идеальным решением для всех приложений.

Дополнительные сведения о *высоком уровне dpi* и о создании приложений с поддержкой *высокого DPI* см. в разделе [высокое разрешение dpi](../hidpi/high-dpi-desktop-application-development-on-windows.md).
