# <a name="support-in-windows-7-and-windows-vista"></a>поддержка в Windows 7 и Windows Vista

для всех номеров sku, кроме N Windows 7 и Windows Vista, среда выполнения WPD входит в состав операционной системы.

### <a name="suported-api-elements"></a>Элементы API поддерживаемые

в следующей таблице приведены функции, поддерживаемые для Windows 7, Windows Vista и Windows vista с обновлением платформы.



| Компонент                        | Windows 7 | Windows Vista | Windows Vista с обновлением платформы |
|--------------------------------|-----------|---------------|----------------------------------------|
| MTP по USB                   | Да       | Да           | Да                                    |
| MTP по IP                    | Да       | Да           | Да                                    |
| MTP поверх Bluetooth             | Да       | нет            | Да                                    |
| Службы устройств WPD и MTP    | Да       | нет            | Да                                    |
| Автоматизация WPD                 | Да       | нет            | нет                                     |
| Множественная функция/множественное транспортировка | Да       | нет            | нет                                     |
| Этап устройства                   | Да       | нет            | нет                                     |
| Платформа синхронизации устройств           | Да       | Нет            | нет                                     |



 

### <a name="the-platform-update-for-windows-vista"></a>обновление платформы для Windows Vista

обновление платформы для Windows Vista поддерживает расширения Windows 7 для платформы Windowsных портативных устройств (WPD).

### <a name="windows-vista-n-and-kn-versions-require-installation-of-windows-media-format-11-sdk"></a>Windows для версий Vista N и KN требуется установка пакета SDK для Windows Media Format 11

-   для версий Windows 7 N и KN необходимо установить Windows Media Format 11 SDK, чтобы включить функции WPD.
-   для версий Windows Vista N и KN необходимо установить Windows Media Format 11 SDK, чтобы включить функции WPD.
