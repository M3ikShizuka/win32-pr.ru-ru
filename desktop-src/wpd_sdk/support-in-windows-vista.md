# <a name="support-in-windows-7-and-windows-vista"></a>Поддержка в Windows 7 и Windows Vista

Для всех, кроме номеров SKU Windows 7 и Windows Vista, среда выполнения WPD входит в состав операционной системы.

### <a name="suported-api-elements"></a>Элементы API поддерживаемые

В следующей таблице приведены функции, поддерживаемые в Windows 7, Windows Vista и Windows Vista с обновлением платформы.



| Функция                        | Windows 7 | Windows Vista | Windows Vista с обновлением платформы |
|--------------------------------|-----------|---------------|----------------------------------------|
| MTP по USB                   | Да       | Да           | Да                                    |
| MTP по IP                    | Да       | Да           | Да                                    |
| MTP по Bluetooth             | Да       | Нет            | Да                                    |
| Службы устройств WPD и MTP    | Да       | Нет            | Да                                    |
| Автоматизация WPD                 | Да       | Нет            | Нет                                     |
| Множественная функция/множественное транспортировка | Да       | Нет            | Нет                                     |
| Этап устройства                   | Да       | Нет            | Нет                                     |
| Платформа синхронизации устройств           | Да       | Нет            | Нет                                     |



 

### <a name="the-platform-update-for-windows-vista"></a>Обновление платформы для Windows Vista

Обновление платформы для Windows Vista поддерживает расширения Windows 7 для платформы портативных устройств Windows (WPD).

### <a name="windows-vista-n-and-kn-versions-require-installation-of-windows-media-format-11-sdk"></a>Для версий N и KN Windows Vista требуется установка пакета SDK Windows Media Format 11

-   Для версий Windows 7 N и KN необходимо установить пакет SDK для Windows Media Format 11, чтобы включить функции WPD.
-   Для версий N и KN Windows Vista необходимо установить пакет SDK для Windows Media Format 11, чтобы включить функции WPD.
