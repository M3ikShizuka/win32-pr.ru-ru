# <a name="iportabledevicevaluesgetbuffervalue-method"></a>Метод Ипортабледевицевалуес:: Жетбуффервалуе

Метод **жетбуффервалуе** извлекает значение **массива БАЙТОВ** (тип VT \_ vector \| VT \_ UI1), заданный ключом.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetBufferValue(
  [in]  REFPROPERTYKEY key,
  [out] BYTE           **ppValue,
  [out] DWORD          *pcbValue
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ключ* \[ окне\]
</dt> <dd>

Ключ **рефпропертикэй** , указывающий извлекаемый элемент.

</dd> <dt>

*ппвалуе* \[ заполняет\]
</dt> <dd>

Указатель на полученное **\* *значение Byte _. Вызывающий объект отвечает за освобождение памяти путем вызова _* CoTaskMemFree**.

</dd> <dt>

*пкбвалуе* \[ заполняет\]
</dt> <dd>

Указатель на размер *ппвалуе* в байтах.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                                                            | Описание                                                          |
|------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                                   | Метод выполнен успешно.<br/>                                     |
| <dl> <dt>**\_вытипемисматч E \_**</dt> </dl>                   | Свойство, указанное *ключом* , не является типом **Byte** \* .<br/> |
| <dl> <dt>**HRESULT \_ из \_ Win32 (ошибка \_ не \_ найдена)**</dt> </dl> | Свойство, указанное *ключом* , отсутствует в коллекции.<br/> |



 

## <a name="remarks"></a>Remarks

Получение буфера **со значением NULL** или буфера нулевого размера не поддерживается.
