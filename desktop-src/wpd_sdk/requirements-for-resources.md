# <a name="requirements-for-resources"></a>Требования к ресурсам

Портативные устройства Windows определяют следующие категории ресурсов в качестве значений **PROPERTYKEY** . Эти значения используются для описания отдельных ресурсов в объекте. Элемент *PID* ключа свойства может отличаться для описания различных ресурсов одного типа для всех типов, за исключением **\_ ресурсов WPD \_ по умолчанию**, которые могут описывать только один ресурс. На странице «связанное описание» для каждого типа ресурсов перечислены атрибуты, поддерживаемые этим ресурсом.



| PROPERTYKEY ресурсов                                                | Описание                                                                                                    |
|---------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------|
| [**\_ресурс \_ по умолчанию для WPD**](wpd-resource-default.md)              | Задает весь файл за объектом. Это единственный необходимый ресурс для любого объекта с ресурсом. |
| [**\_ \_ Обложка ресурсов \_ WPD**](wpd-resource-album-art.md)         | Задает изображение, представляющее иллюстрацию альбома для объекта.                                           |
| [**\_ \_ звуковой ролик ресурса \_ WPD**](wpd-resource-audio-clip.md)       | Указывает аудиоклип для объекта.                                                                        |
| [**\_фотография контакта по ресурсу WPD \_ \_**](wpd-resource-contact-photo.md) | Задает изображение, представляющее фотографию лица, на которое ссылается объект Contact.                |
| [**\_универсальный ресурс \_ WPD**](wpd-resource-generic.md)              | Универсальный ресурс неопределенного типа данных. Он должен рассматриваться как массив байтов.                             |
| [**\_значок ресурса \_ WPD**](wpd-resource-icon.md)                    | Значок.                                                                                                       |
| [**\_эскиз ресурса \_ WPD**](wpd-resource-thumbnail.md)          | Эскиз изображения.                                                                                             |
