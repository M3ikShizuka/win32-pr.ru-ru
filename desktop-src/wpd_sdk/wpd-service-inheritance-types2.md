# <a name="wpd_service_inheritance_types-enumeration"></a>\_ \_ Перечисление типов наследования службы WPD \_

Тип **перечисления \_ \_ \_ типов наследования службы WPD** указывает отношение наследования для службы.

## <a name="syntax"></a>Синтаксис


```C++
typedef enum tagWPD_SERVICE_INHERITANCE_TYPES { 
  WPD_SERVICE_INHERITANCE_IMPLEMENTATION  = 0
} WPD_SERVICE_INHERITANCE_TYPES;
```
