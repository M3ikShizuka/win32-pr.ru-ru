# <a name="wpd-programming-guide"></a>Программное руководством по программированию WPD

основной источник для этого руководством по программированию — это пример, поставляемый с пакетом sdk для Windowsных мобильных устройств (WPD). Этот пример с именем WpdApiSample.exe состоит из семи модулей. cpp. Шесть из этих модулей содержат код, демонстрирующий 26 задач, которые приложение может выполнить с помощью программного интерфейса (API) WPD.

Исключения, приведенные выше, являются разделами, описывающими: задачи службы устройств, контекстное меню и расширения страниц свойств. Эти разделы не были взяты из Впдаписампле. Задачи службы устройств основаны на образце с именем Впдсервицеаписампле. расширения контекстного меню и страницы свойств — это фрагменты кода, взятые из приложений, которые не поставляются с Windows SDK.

В следующих разделах основное внимание уделяется задачам, выполняемым этими примерами, и объясняется, как каждый из них выполняется с помощью интерфейсов WPD и ряда методов, найденных в этих интерфейсах.
