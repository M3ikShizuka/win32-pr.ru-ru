# <a name="client-properties"></a>Свойства клиента

Портативные устройства Windows поддерживают следующие свойства клиента.



| Свойство                                          | VarType                   | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|---------------------------------------------------|---------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ требуемый доступ клиента WPD \_**                  | **VT \_ UI4**               | Значение, указывающее, что клиенту требуются уникальные разрешения доступа, такие как универсальный доступ \_ только для чтения. Если это свойство не задано, то для WPD предполагается универсальный \_ \| общий доступ для чтения \_ , который является рекомендуемым по умолчанию.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **\_ \_ основной номер версии клиента WPD \_**                   | **VT \_ UI4**               | Число, указывающее основной номер версии клиента.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **\_ \_ дополнительный номер версии клиента WPD \_**                   | **VT \_ UI4**               | Число, указывающее дополнительный номер версии клиента.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **\_имя клиента \_ WPD**                             | **VT \_ LPWSTR**            | Строка имени для клиентского приложения.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **\_версия клиента \_ WPD**                         | **VT \_ UI4**               | Число, указывающее номер редакции (или номер сборки) клиента.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **\_ \_ \_ качество \_ \_ обслуживания клиента WPD**   | **VT \_ UI4**               | Значение типа, указывающее качество службы безопасности для подключения к драйверу. Это связано с флагами качества службы безопасности для **CreateFile**.<br/> Например, эти флаги позволяют или запрещают драйверу олицетворять клиента. \_Если это значение не указано, предполагается идентификатор безопасности. Как правило, приложения должны присвоить этому параметру \_ олицетворение безопасности, так как для работы некоторых драйверов потребуется олицетворение. Однако приложения, выполняющиеся в рамках требований Strict Security с известным подмножеством устройств, могут ограничить способность драйвера определять или олицетворять вызывающего. Обратите внимание, что уменьшение уровня олицетворения на что-то меньше, чем ОЛИЦЕТВОРЕНие безопасности, может привести к тому, что \_ некоторые драйверы будут иметь ограниченную функциональность.<br/>                                         |
| **\_режим клиентской \_ общей папки WPD \_**                      | **VT \_ UI4**               | Значение типа, указывающее режим совместного использования, запрашиваемый клиентом из драйвера. Возможны такие же значения, как и для **CreateFile**, например \_ чтение общей папки \_ , запись в общую \_ папку \_ и т. д.<br/> \_ \_ \| \_ \_ Если клиент не указал этот параметр, предполагается запись общего файлового ресурса для чтения. Это рекомендуемое значение по умолчанию. Тем не менее, если требуется другой режим общего доступа, например, ФАЙЛовый \_ ресурс \_ читается для запроса только для чтения, этот параметр является обязательным.<br/>                                                                                                                                                                                                                                                                                                                                                          |
| **\_ \_ \_ \_ закрытый ключ клиентского приложения WPD с цифровыми правами \_** | **VT \_ вектор \_ \| VT \_ UI1** | Значение, указывающее закрытый ключ приложения Windows Media DRM для клиента.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **\_сертификат приложения WPD клиента для \_ WMDRM \_ \_**  | **VT \_ вектор \_ \| VT \_ UI1** | Значение, указывающее сертификат Windows Media DRM клиента.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **\_куки- \_ файл событий клиента \_ WPD**                    | **VT \_ LPWSTR**            | Предоставленный клиентом файл cookie, возвращенный драйвером в событиях, опубликованных как прямой результат операций, выданных этим клиентом. Это необязательное свойство, которое драйвер может указать в качестве параметра события. Клиенты обычно используют это значение для оптимизации обработки событий. Например, если клиент удаляет объект, его обработчик событий может игнорировать результирующее событие, зная, что оно уже заняло соответствующее действие (например, обновило представление в потоке, который выдал удаление).                                                                                                                                                                                                                                                                                                                                                                |
| **\_ \_ Минимальный \_ \_ Размер буфера для результатов клиента WPD \_**   | **VT \_ UI4**               | Указывает минимальный размер буфера (в байтах) для отправки команд драйверу. Клиенты могут использовать этот параметр, чтобы рекомендовать минимальный размер внутреннего буфера, используемого для отправки команд драйверу. Это может привести к уменьшению объема памяти для клиентов, отправляющих только команды с мелкими размерами данных, например получение или задание свойств. Драйвер может переопределить этот размер буфера, установив **портабледевицересултсбуффермин** в разделе параметров устройства. Размер буфера должен быть 1024 или выше. Если ни клиент, ни драйвер не устанавливают размер буфера, по умолчанию используется значение 1 МБ.                                                                                                                                                                                                                                                  |
| **\_клиент WPD \_ вручную \_ закрывается \_ при \_ отключении**    | **Логическое значение VT \_**              | Дополнительный параметр для клиентов, которые вручную вызывают **ипортабледевице:: Close** или **Ипортабледевицесервице:: Close** для каждого отключенного объекта, вместо того чтобы полагаться на API для этого. Значение FALSE принимается, если клиент не указал этот параметр. Это рекомендуемое значение по умолчанию, если клиент не прослушивает сообщения **WM \_ девицечанже** . Клиенты, указывающие этот параметр, обычно прослушивают сообщения **WM \_ девицечанже** для устройства, а затем вручную закрывают экземпляры **ипортабледевице** и **ипортабледевицесервице** при отключении устройства. Клиенты, которые выбирают этот параметр, должны тщательно управлять временем существования каждого экземпляра **ипортабледевице** и **ипортабледевицесервице** , освобождая эти объекты при отключении устройства. Несоблюдение этого действия приведет к утечке в процессах WUDFHost.exe для драйвера. |
