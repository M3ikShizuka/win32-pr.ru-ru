# <a name="wpd_section_data_units_values-enumeration"></a>\_ \_ \_ Перечисление значений единиц данных раздела WPD \_

В **\_ разделе WPD \_ Data \_ Units \_** enumeration перечисление Values указываются единицы для раздела данных, на который указывает ссылка.

## <a name="syntax"></a>Синтаксис


```C++
typedef enum WPD_SECTION_DATA_UNITS_VALUES { 
  WPD_SECTION_DATA_UNITS_BYTES         = 0,
  WPD_SECTION_DATA_UNITS_MILLISECONDS  = 1
} ;
```
