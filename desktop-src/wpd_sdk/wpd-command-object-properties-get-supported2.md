# <a name="wpd_command_object_properties_get_supported-command"></a>\_ \_ \_ \_ Поддерживаемые команды в СВОЙСТВАх объекта команды WPD \_

Команда **WPD \_ : \_ свойства объекта команды " \_ \_ получить \_ поддерживаемый** " возвращает свойства, поддерживаемые объектом.

## <a name="command-category"></a>Категория команды

**\_ \_ свойства объекта категории \_ WPD**

## <a name="parameters"></a>Параметры

Драйверу необходим следующий параметр.



| Параметр                                         | VarType        | Описание                                                            |
|---------------------------------------------------|----------------|------------------------------------------------------------------------|
| **\_ \_ \_ \_ идентификатор объекта свойств объекта свойства \_ WPD** | **VT \_ LPWSTR** | Обязательный. Идентификатор объекта, который содержит запрошенные свойства. |



 

## <a name="return-value"></a>Возвращаемое значение

Драйвер должен вернуть следующие результаты.



| Результат                                                | VarType         | Описание                                                                                                                                                                                                                                                                                                                                            |
|-------------------------------------------------------|-----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ свойства объекта свойства \_ WPD \_ \_ ключи свойств** | **VT \_ Unknown** | Обязательный. Интерфейс [**ипортабледевицекэйколлектион**](iportabledevicekeycollection.md) , указывающий все поддерживаемые свойства.                                                                                                                                                                                                            |
| **Свойство WPD, \_ \_ Общее \_ значение HRESULT**                    | **\_Ошибка VT**   | Обязательный. Значение **HRESULT** , указывающее общее число успешных или неудачных попыток. возможные значения результатов включают [Windows коды ошибок переносных устройств](error-constants.md). Если вызывающий объект делает недопустимый запрос, драйвер должен вернуть **HRESULT \_ из \_ Win32 (ошибка \_ не \_ поддерживается)** , но в противном случае не требуется возвращать какое-либо другое результирующее значение. |
| **\_ \_ \_ код ошибки общего драйвера \_ для свойства WPD \_**        | **VT \_ UI4**     | Необязательный элемент. Код ошибки, относящийся к драйверу. Обычно это используется только для тестирования драйверов или, если все драйверы, устройства и клиенты разработаны вместе.                                                                                                                                                                                                |
