# <a name="appointment-properties"></a>Свойства встречи

Портативные устройства Windows поддерживают следующие свойства встречи.



| Свойство                                   | VarType        | Описание                                                                          |
|--------------------------------------------|----------------|--------------------------------------------------------------------------------------|
| **\_встреча, \_ принятая \_ участником WPD**  | **VT \_ LPWSTR** | Разделенный точками с запятой список участников, которые приняли встречу.             |
| **\_встречи с \_ неотклоненной ВСТРЕЧей WPD \_**  | **VT \_ LPWSTR** | Разделенный точками с запятой список участников, которые отклонили встречу.             |
| **\_Расположение встречи \_ WPD**             | VT \_ LPWSTR     | Удобное для чтения место встречи, например "здание 5, комната 7".    |
| **\_ \_ Дополнительные участники встречи \_ WPD**  | **VT \_ LPWSTR** | Разделенный точками с запятой список необязательных участников встречи.                  |
| **\_Участники, \_ необходимые для встречи с WPD \_**  | **VT \_ LPWSTR** | Разделенный точками с запятой список обязательных участников встречи.                  |
| **\_ресурсы для встреч WPD \_**            | **VT \_ LPWSTR** | Разделенный точками с запятой список ресурсов, необходимых для встречи.                  |
| **\_встречи с неоднократными \_ \_ участниками** | **VT \_ LPWSTR** | Разделенный точками с запятой список участников, которые предварительно приняли встречу. |
| **\_тип встречи \_ WPD**                 | **VT \_ LPWSTR** | Тип встречи, например "Личное", "Бизнес" и т. д.              |
