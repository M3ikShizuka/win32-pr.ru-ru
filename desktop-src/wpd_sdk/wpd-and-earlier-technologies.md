# <a name="wpd-and-earlier-technologies"></a>WPD и более ранние технологии

WPD содержит множество улучшений для цифровых камер и портативных проигрывателей Digital Media, и является предпочтительным способом доступа к этим устройствам в новых версиях Windows. существующие api Windowsного получения изображений (WIA) и Windows Media диспетчер устройств (вмдм) в Windows продолжат поддерживаться, и совместимость с WPD обеспечивается уровнями совместимости. Эти уровни совместимости гарантируют, что устройства WPD могут быть прозрачно доступны для существующих приложений WIA и ВМДМ как устройства WIA или поставщики услуг ВМДМ соответственно. Однако эти приложения могут не иметь тех же самых возможностей, что и новый API-интерфейс WPD и новые устройства, основанные на MTP.

Драйвер WPD MTP также поддерживает устройства, поддерживающие протокол обмена изображениями (PTP). Устройства PTP, поддерживаемые в WIA, продолжат работать в приложениях WIA через уровень совместимости. корпорация майкрософт сохраняет обязательства по обеспечению совместимости приложений и цифровых камер в следующей версии Windows, и эти новые технологии разработаны для дополнения и улучшения цифровой работы камеры в Windows.
