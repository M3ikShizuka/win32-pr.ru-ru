# <a name="selecting-a-device-wpd-api"></a>Выбор устройства (API-интерфейс WPD)

После того как приложение перечислит подключенные устройства, может потребоваться предоставить пользователю возможность выбрать определенное устройство и установить подключение для дальнейшей работы. В примере приложения `ChooseDevice` функция в модуле девицеенумератион. cpp выполняет эти задачи.

`ChooseDevice`Функция использует код перечисления устройств, который мы уже видели для перечисления доступных устройств. (Дополнительные сведения см. в разделе [Перечисление устройств](enumerating-devices.md) .)
