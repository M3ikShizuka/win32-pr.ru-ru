# <a name="collection-interfaces"></a>Интерфейсы коллекции

Следующие интерфейсы используются для хранения групп значений различных типов, включая значения **PROPERTYKEY** , значения **GUID** или другие интерфейсы коллекции. Эти интерфейсы используются как клиентскими приложениями, так и драйверами для хранения данных для параметров метода.



| Интерфейс                                                                            | Описание                                                            |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------|
| [**ипортабледевицекэйколлектион**](iportabledevicekeycollection.md)                 | Содержит коллекцию значений **PROPERTYKEY** .                          |
| [**ипортабледевицепропвариантколлектион**](iportabledevicepropvariantcollection.md) | Содержит коллекцию значений **пропвариант** одного объекта VarType.      |
| [**ипортабледевицевалуес**](iportabledevicevalues.md)                               | Содержит коллекцию пар **PROPERTYKEY** / **пропвариант** .           |
| [**ипортабледевицевалуесколлектион**](iportabledevicevaluescollection.md)           | Содержит коллекцию **ипортабледевицевалуес** интерфейсов с индексацией от нуля. |
