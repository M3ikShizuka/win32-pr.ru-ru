# <a name="iwpdserializerwriteiportabledevicevaluestobuffer-method"></a>Метод Ивпдсериализер:: Вритеипортабледевицевалуестобуффер

Метод **вритеипортабледевицевалуестобуффер** сериализует интерфейс **ипортабледевицевалуес** в выделенный вызывающим объектом массив байтов.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT WriteIPortableDeviceValuesToBuffer(
  [in]  DWORD                 dwOutputBufferLength,
  [in]  IPortableDeviceValues *pResults,
  [out] BYTE                  *pBuffer,
  [out] DWORD                 *pdwBytesWritten
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*дваутпутбуфферленгс* \[ окне\]
</dt> <dd>

**Параметр DWORD** , указывающий размер *pBuffer* в байтах.

</dd> <dt>

*пресултс* \[ окне\]
</dt> <dd>

Указатель на интерфейс [**ипортабледевицевалуес**](iportabledevicevalues.md) для сериализации.

</dd> <dt>

*pBuffer* \[ заполняет\]
</dt> <dd>

Указатель на выделенный вызывающим объектом буфер. Чтобы узнать размер требуемого буфера, вызовите **жетсериализедсизе**.

</dd> <dt>

*пдвбитесвриттен* \[ заполняет\]
</dt> <dd>

Указатель на **DWORD** , указывающий число байтов, фактически записанных в выделенный вызывающим объектом буфер.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                                   | Описание                                               |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Метод выполнен успешно.<br/>                          |
| <dl> <dt>**\_указатель E**</dt> </dl>     | Обязательный аргумент-указатель имеет **значение NULL**.<br/>      |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Предоставленный вызывающим объектом буфер недостаточно велик.<br/> |



 

## <a name="remarks"></a>Комментарии

Этот метод копирует интерфейс **ипортабледевицевалуес** в существующий буфер. Если необходимо выделить новый буфер, используйте [**жетбуфферфромипортабледевицевалуес**](iwpdserializer-getbufferfromiportabledevicevalues.md).
