# <a name="mtpbluetooth-connection-interfaces"></a>Интерфейсы подключения MTP и Bluetooth

Следующие интерфейсы позволяют приложениям перечислять и подключаться только к устройствам, поддерживающим протокол передачи мультимедиа (MTP) через Bluetooth (MTP/Bluetooth). Интерфейсы драйвера, коллекции и клиента WPD не ограничиваются протоколом MTP/Bluetooth.



| Интерфейс                                                              | Описание                                                                                                         |
|------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**иконнектионрекуесткаллбакк**](iconnectionrequestcallback.md)       | Определяет один метод обратного вызова, используемый приложениями для получения уведомлений о завершенных и отмененных запросах. |
| [**иенумпортабледевицеконнекторс**](ienumportabledeviceconnectors.md) | Перечисляет интерфейсы **ипортабледевицеконнектор** .                                                                 |
| [**ипортабледевицеконнектор**](/windows/desktop/api/portabledeviceconnectapi/nn-portabledeviceconnectapi-iportabledeviceconnector)           | Поддерживает методы, которые вызываются приложениями для установки подключений к устройствам MTP Bluetooth.                          |
