# <a name="email-properties"></a>Свойства электронной почты

Windows Портативные устройства поддерживают следующие свойства электронной почты.



| Свойство                         | VarType        | Описание                                                                                                                               |
|----------------------------------|----------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ строка скрытия сообщения электронной почты WPD \_**        | **VT \_ LPWSTR** | Получатели скрытой копии сообщения электронной почты. Получатели СКРЫТой копии получают копию сообщения, но их имена не отображаются как получатели.   |
| **\_Электронная почта WPD — \_ \_ строка CC**         | **VT \_ LPWSTR** | Получатели копии сообщения электронной почты. Получатели копии получают копию сообщения электронной почты, и их имена отображаются как получатели. |
| **\_сообщения электронной почты WPD \_ с \_ вложениями** | **Логическое значение VT \_**   | Логическое значение, указывающее, есть ли у этого сообщения электронной почты вложения.                                                               |
| **\_Прочитано электронное письмо WPD \_ \_ \_**  | **Логическое значение VT \_**   | Логическое значение, указывающее, было ли Прочитано это сообщение электронной почты.                                                                 |
| **\_время получения электронной почты WPD \_ \_**   | **\_Дата VT**   | Значение, указывающее, когда было получено сообщение электронной почты.                                                                              |
| **\_ \_ адрес отправителя электронной почты WPD \_**  | **VT \_ LPWSTR** | Адрес электронной почты отправителя.                                                                                                         |
| **\_сообщение электронной почты WPD \_ по \_ строке**         | **VT \_ LPWSTR** | Основные получатели сообщения электронной почты.                                                                                                |
