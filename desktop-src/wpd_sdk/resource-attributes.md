# <a name="resource-attributes"></a>Атрибуты ресурсов

Портативные устройства Windows поддерживают следующие атрибуты ресурсов. Эти атрибуты возвращаются следующими методами.

-   [**Ипортабледевицересаурцес:: Жетресаурцеаттрибутес**](/windows/desktop/api/PortableDeviceApi/nf-portabledeviceapi-iportabledevicecapabilities-getfixedpropertyattributes)



| attribute                                                  | VarType      | Описание                                                                                                                                 |
|------------------------------------------------------------|--------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| **\_атрибут ресурса \_ WPD \_ может быть \_ удален**                  | **Логическое значение VT \_** | Логическое значение, указывающее, имеет ли клиент разрешение на удаление ресурса. Если он отсутствует, предполагается, что он имеет значение false.                |
| **\_атрибут ресурса \_ WPD \_ может \_ читать**                    | **Логическое значение VT \_** | Логическое значение, указывающее, имеет ли клиент разрешение на открытие ресурса для доступа на чтение. Если он отсутствует, предполагается, что он имеет значение false.  |
| **\_атрибут ресурса \_ WPD \_ может \_ записывать**                   | **Логическое значение VT \_** | Логическое значение, указывающее, имеет ли клиент разрешение на открытие ресурса для доступа на запись. Если он отсутствует, предполагается, что он имеет значение false. |
| **\_ \_ \_ Размер неоптимального \_ \_ буфера чтения \_ для атрибута ресурса WPD**  | **VT \_ UI4**  | Рекомендуемый размер буфера, который вызывающий объект должен использовать для буферизованного чтения из ресурса.                                                  |
| **\_атрибут ресурса WPD — \_ \_ оптимальный \_ \_ Размер буфера записи \_** | **VT \_ UI4**  | Рекомендуемый размер буфера, который вызывающий объект должен использовать для буферизованной записи ресурса.                                                   |
| **\_ \_ \_ Общий \_ Размер атрибута ресурса WPD**                  | **VT \_ UI8**  | Общий размер данных ресурса в байтах.                                                                                              |
