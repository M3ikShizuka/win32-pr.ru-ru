# <a name="wpd-samples"></a>Примеры WPD

Windows SDK предоставляет два примера: [впдаписампле](wpdapisample-sample-application.md) и [впдсервицеаписампле](wpdapisample-sample-service-application.md).

Впдаписампле — это классическое приложение командной строки, которое позволяет пользователю выполнять несколько функций: перечисление подключенных устройств, просмотр объектов на этих устройствах и перечисление свойств объектов.

Впдсервицеаписампле — это классическое приложение командной строки, которое поддерживает службу контактов и позволяет пользователю выполнять такие функции, как перечисление подключенных служб устройств, получение свойств и форматов для данной службы и вызов методов для этой службы.
