# <a name="wpd_command_mtp_ext_read_data-command"></a>Команда WPD, \_ \_ \_ команда MTP ext \_ Read \_ Data

Команда **WPD команды " \_ \_ MTP \_ ext \_ Read \_ data** " извлекает данные с устройства после выполнения **команды \_ WPD \_ MTP \_ ext \_ Execute \_ \_ с командой \_ Data \_ to \_ Read** .

## <a name="command-category"></a>Категория команды

**\_операции категории WPD для \_ \_ поставщика MTP ext \_ \_**

## <a name="parameters"></a>Параметры

Драйверу требуются следующие параметры.



| Параметр                                                   | VarType             | Описание                                                                            |
|-------------------------------------------------------------|---------------------|----------------------------------------------------------------------------------------|
| **\_свойство WPD \_ для \_ \_ контекста перемещения \_ MTP**              | VT \_ LPWSTR          | Обязательный. Определяет контекст, возвращенный предыдущим вызовом к устройству. |
| **\_свойство WPD \_ . \_ \_ \_ число \_ байтов \_ для чтения (в байтах \_ ) MTP** | VT \_ UI4             | Обязательный. Указывает число байтов для чтения.                                       |
| **Свойство WPD. Дополнительные \_ \_ данные о \_ \_ передаваемых \_ данных MTP**                 | VT \_ вектор \| VT \_ UI1 | Обязательный. Определяет буфер, в который копируются данные устройства.                  |



 

## <a name="return-value"></a>Возвращаемое значение

Драйвер возвращает следующие результаты.



| Результат                                                  | VarType             | Описание                                                       |
|---------------------------------------------------------|---------------------|-------------------------------------------------------------------|
| **\_ \_ \_ \_ \_ число \_ \_ прочитанных байт в свойстве WPD MTP** | VT \_ UI4             | Обязательный. Указывает число байтов, полученных от устройства. |
| **Свойство WPD. Дополнительные \_ \_ данные о \_ \_ передаваемых \_ данных MTP**             | VT \_ вектор \| VT \_ UI1 | Обязательный. Буфер, содержащий данные устройства.               |



 

## <a name="calling-methods"></a>Вызов методов
