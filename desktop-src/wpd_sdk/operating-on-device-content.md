# <a name="operating-on-device-content"></a>Работа с содержимым устройства

API-интерфейс WPD поддерживает ряд операций на устройствах. К этим операциям относятся:

-   Передача содержимого с компьютера на подключенное устройство.
-   Передача содержимого с подключенного устройства на компьютер.
-   Перемещение содержимого на подключенном устройстве.
-   Удаление содержимого с подключенного устройства.
-   Добавление ресурса в объект на подключенном устройстве.

Эти операции описаны в следующих разделах.
