# <a name="support-in-windows-xp"></a>Поддержка в Windows XP

Чтобы приложение выполнялось в Windows XP, пользователю потребуется копия среды выполнения формата Windows Media на своем компьютере. Эту среду выполнения можно повторно распространить с помощью приложения WPD. (Среда выполнения находится в папке распространяемых компонентов в пакете SDK формата Windows Media.)
