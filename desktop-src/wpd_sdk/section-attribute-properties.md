# <a name="section-attribute-properties"></a>Свойства атрибута раздела

Windows Портативные устройства поддерживают следующие свойства данных раздела.



| Свойство                                             | VarType         | Описание                                                                                                                                                                                                                                                                                                                                 |
|------------------------------------------------------|-----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ Длина данных раздела \_ WPD**                       | **VT \_ UI8**     | Задает длину упоминаемого объекта.                                                                                                                                                                                                                                                                                              |
| **\_ \_ смещение данных раздела \_ WPD**                       | **VT \_ UI8**     | Задает отсчитываемое от нуля смещение данных для упоминаемого объекта.                                                                                                                                                                                                                                                                      |
| **\_ \_ \_ \_ объектный ресурс ссылок на данные раздела WPD \_** | **VT \_ Unknown** | Объект [**ипортабледевицекэйколлектион**](iportabledevicekeycollection.md) , содержащий одно значение, указывающее ключ для данного ресурса. Этот ресурс представляет собой объект, на который \_ ссылается \_ смещение данных раздела WPD \_ и \_ \_ Длина данных раздела WPD \_ .<br/> Если это свойство не существует, \_ \_ предполагается значение по умолчанию для ресурса WPD.<br/> |
| **\_ \_ единицы данных раздела \_ WPD**                        | **VT \_ UI4**     | Указывает единицы, используемые для смещения, например байты, миллисекунды и т. д. Возможные значения этого свойства определяются в перечислении **\_ \_ \_ \_ значений единиц измерения данных раздела WPD** в файле портабледевице. h.<br/> Если единицы не указаны, предполагается использование байтов.<br/>                                          |
