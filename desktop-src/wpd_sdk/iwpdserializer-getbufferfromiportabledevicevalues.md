# <a name="iwpdserializergetbufferfromiportabledevicevalues-method"></a>Метод Ивпдсериализер:: Жетбуфферфромипортабледевицевалуес

Метод **жетбуфферфромипортабледевицевалуес** сериализует отправленный интерфейс **ипортабледевицевалуес** в выделенный массив байтов. Возвращаемый массив байтов выделяется для вызывающего объекта и должен быть освобожден вызывающим объектом с помощью **CoTaskMemFree**.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetBufferFromIPortableDeviceValues(
  [in]  IPortableDeviceValues *pSource,
  [out] BYTE                  **ppBuffer,
  [out] DWORD                 *pdwBufferSize
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*псаурце* \[ окне\]
</dt> <dd>

Указатель на интерфейс [**ипортабледевицевалуес**](iportabledevicevalues.md) для сериализации.

</dd> <dt>

*ппбуффер* \[ заполняет\]
</dt> <dd>

Указатель на **байт \* *_, содержащий сериализованные данные. Портативные устройства Windows распределяют эту память; вызывающий объект должен освободить его, вызвав _* CoTaskMemFree**.

</dd> <dt>

*пдвбуфферсизе* \[ заполняет\]
</dt> <dd>

Указатель на **DWORD** , указывающий размер выделенного буфера в байтах.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                                   | Описание                                                            |
|-----------------------------------------------------------------------------------------------|------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Метод выполнен успешно.<br/>                                       |
| <dl> <dt>**\_указатель E**</dt> </dl>     | Обязательный аргумент-указатель имеет **значение NULL**.<br/>                   |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти для создания буфера.<br/> |
