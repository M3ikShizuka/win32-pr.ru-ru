# <a name="wpd-constants"></a>Константы WPD

Windows Портативные устройства определяют следующие константные типы.



| Тип константы                                              | Описание                                                                                                                                                                                                                               |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Общие константы](general-constants.md)                 | Общие константы использования для устройств и приложений.                                                                                                                                                                                       |
| [Константы ошибок](error-constants.md)                     | Значения ошибок, возвращаемые методами.                                                                                                                                                                                                         |
| [Константы событий](event-constants.md)                     | типы событий, определяемые Windows портативными устройствами. Типы событий включают добавление объекта, удаление объекта, обновление объекта, перемещение объекта, копирование объекта, запрос на пересылку объектов, сброс устройства, удаление устройства и обновление возможностей устройства. |
| [Идентификаторы GUID типов объектов](object-type-guids.md)                 | типы объектов, определяемые Windows портативными устройствами. К типам объектов относятся аудио, электронная почта и объекты изображений.                                                                                                                                  |
| [Идентификаторы GUID формата объектов](object-format-guids.md)             | типы форматов файлов, определенные Windows портативных устройств. К типам форматов относятся файлы WMA, WMV и JPEG.                                                                                                                                           |
| [Идентификаторы GUID типов ресурсов](resource-type-guids.md)             | типы ресурсов, определяемые Windows портативными устройствами. К типам ресурсов относятся изображения аудио и эскиза.                                                                                                                                |
| [Свойства и атрибуты](properties-and-attributes.md) | свойства и атрибуты, определенные Windows портативных устройств.                                                                                                                                                                            |
