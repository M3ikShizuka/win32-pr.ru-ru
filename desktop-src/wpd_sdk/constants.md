# <a name="wpd-constants"></a>Константы WPD

Портативные устройства Windows определяют следующие константные типы.



| Тип константы                                              | Описание                                                                                                                                                                                                                               |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Общие константы](general-constants.md)                 | Общие константы использования для устройств и приложений.                                                                                                                                                                                       |
| [Константы ошибок](error-constants.md)                     | Значения ошибок, возвращаемые методами.                                                                                                                                                                                                         |
| [Константы событий](event-constants.md)                     | Типы событий, определяемые переносными устройствами Windows. Типы событий включают добавление объекта, удаление объекта, обновление объекта, перемещение объекта, копирование объекта, запрос на пересылку объектов, сброс устройства, удаление устройства и обновление возможностей устройства. |
| [Идентификаторы GUID типов объектов](object-type-guids.md)                 | Типы объектов, определяемые портативными устройствами Windows. К типам объектов относятся аудио, электронная почта и объекты изображений.                                                                                                                                  |
| [Идентификаторы GUID формата объектов](object-format-guids.md)             | Типы форматов файлов, определенные портативными устройствами Windows. К типам форматов относятся файлы WMA, WMV и JPEG.                                                                                                                                           |
| [Идентификаторы GUID типов ресурсов](resource-type-guids.md)             | Типы ресурсов, определяемые портативными устройствами Windows. К типам ресурсов относятся изображения аудио и эскиза.                                                                                                                                |
| [Свойства и атрибуты](properties-and-attributes.md) | Свойства и атрибуты, определенные портативными устройствами Windows.                                                                                                                                                                            |
