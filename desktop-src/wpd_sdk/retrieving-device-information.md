# <a name="retrieving-device-information"></a>Получение сведений об устройстве

API-интерфейс WPD предоставляет широкий набор интерфейсов и методов, которые можно использовать для получения сведений для конкретного устройства. Как правило, данные устройства можно разделить на три категории, как описано в следующей таблице.



| Данные устройства                                                                             | Описание                                                                               |
|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [Функциональные категории](retrieving-the-functional-categories-supported-by-a-device.md) | Функциональные категории описывают операции, которые может выполнять устройство. |
| [Типы содержимого](retrieving-the-content-types-supported-by-a-device.md)                 | Типы содержимого обозначают типы содержимого, которое может быть отображено устройством.         |
| [События](retrieving-the-events-supported-by-a-device.md)                               | Некоторые устройства способны инициировать события, которые приложение может отреагировать на. |
