# <a name="common-information-properties"></a>Общие сведения о свойствах

Портативные устройства Windows поддерживают следующие общие свойства информации.



| Свойство                                      | VarType        | Описание                                                                                              |
|-----------------------------------------------|----------------|----------------------------------------------------------------------------------------------------------|
| **\_Общие \_ сведения о \_ WPD**           | **VT \_ LPWSTR** | Для встреч, задач и аналогичных объектов это свойство содержит заметки для данного объекта.     |
| **\_субъект общей \_ информации о \_ WPD**         | **VT \_ LPWSTR** | Значение типа, указывающее поле темы данного объекта.                                                 |
| **\_ \_ \_ основной текст общей информации о WPD \_**      | **VT \_ LPWSTR** | Это свойство содержит текст тела объекта в формате обычного текста или HTML.                          |
| **\_стандартный \_ приоритет данных \_ WPD**        | **VT \_ UI4**    | Значение, указывающее приоритет данного объекта. 0 означает наивысший приоритет.                    |
| **\_ \_ \_ \_ Дата и время начала общей информации для WPD** | **\_Дата VT**   | Значение типа, указывающее дату и время запланированного запуска встречи, задачи или аналогичного объекта. |
| **\_ \_ \_ \_ Дата и время окончания общей информации о WPD**   | **\_Дата VT**   | Значение типа, указывающее дату и время, когда запланировано завершение встречи, задачи или аналогичного объекта.   |
