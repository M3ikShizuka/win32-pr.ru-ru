# <a name="task-properties"></a>Свойства задачи

Windows Портативные устройства поддерживают следующие свойства задачи.



| Свойство                         | VarType        | Описание                                                                                 |
|----------------------------------|----------------|---------------------------------------------------------------------------------------------|
| **\_владелец задачи \_ WPD**             | **VT \_ LPWSTR** | Владелец задачи.                                                                      |
| **\_ \_ процент выполнения задачи \_ WPD** | **VT \_ UI4**    | Число от 0 до 100, указывающее, как завершается задача.                         |
| **\_ \_ Дата напоминания о ЗАДАЧе WPD \_**    | **\_Дата VT**   | Значение, указывающее, когда следует отправить напоминание для выполнения задачи, если оно не завершено. |
| **\_состояние задачи \_ WPD**            | **VT \_ LPWSTR** | Строковое описание состояния задачи, например "выполняется".                 |
