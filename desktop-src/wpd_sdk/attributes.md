# <a name="property-attributes-portabledeviceh"></a>Атрибуты свойств (Портабледевице. h)

Windows Портативные устройства поддерживают следующие атрибуты свойств. Эти атрибуты возвращаются следующими методами.

-   [**Ипортабледевицекапабилитиес:: Жетфикседпропертяттрибутес**](/windows/desktop/api/PortableDeviceApi/nf-portabledeviceapi-iportabledevicecapabilities-getfixedpropertyattributes)
-   [**Ипортабледевицепропертиес:: Жетпропертяттрибутес**](/windows/desktop/api/PortableDeviceApi/nf-portabledeviceapi-iportabledeviceproperties-getpropertyattributes)
-   [**Ипортабледевицесервицекапабилитиес:: Жетформатпропертяттрибутес**](/windows/desktop/api/PortableDeviceAPI/nf-portabledeviceapi-iportabledeviceservicecapabilities-getformatpropertyattributes)



| attribute                                           | VarType         | Описание                                                                                                                                                                                                                                                                                                                                                                                    |
|-----------------------------------------------------|-----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_атрибут свойства \_ WPD \_ может быть \_ удален**           | **Логическое значение VT \_**    | Логическое значение, указывающее, может ли клиент удалить свойство. Чтобы удалить свойство, присвойте ему значение VT \_ Empty.                                                                                                                                                                                                                                                                   |
| **\_атрибут свойства \_ WPD \_ может \_ считывать**             | **Логическое значение VT \_**    | Логическое значение, указывающее, может ли клиент считывать свойство.                                                                                                                                                                                                                                                                                                                       |
| **\_атрибут свойства \_ WPD \_ может \_ записывать**            | **Логическое значение VT \_**    | Логическое значение, указывающее, может ли клиент изменять свойство.                                                                                                                                                                                                                                                                                                                     |
| **\_ \_ значение атрибута свойства WPD \_ по умолчанию \_**        | VT \_ *XXXX*      | Значение, определяемое устройством, которое указывает значение свойства по умолчанию. Это относится только к записываемым свойствам.                                                                                                                                                                                                                                                               |
| **\_ \_ \_ элементы ПЕРЕЧИСЛЕНИЯ атрибута свойства \_ WPD** | **VT \_ Unknown** | Интерфейс [**ипортабледевицепропвариантколлектион**](iportabledevicepropvariantcollection.md) , который содержит коллекцию значений для свойства, чей атрибут **\_ \_ \_ формы свойства WPD** имеет значение **\_ \_ \_ \_ перечисление формы атрибута типа WPD**. Тип данных зависит от запрашиваемого свойства.                                                                              |
| **\_свойство WPD \_ атрибута \_ FAST \_ свойства**        | **Логическое значение VT \_**    | Если значение — true, это свойство принадлежит группе *быстрых свойств* . Это свойства, которые можно быстро получить с устройства.                                                                                                                                                                                                                                                        |
| **\_ \_ форма атрибута свойства \_ WPD**                  | **VT \_ UI4**     | Перечислимое значение [**впдаттрибутеформ**](wpdattributeform.md) , указывающее форму допустимых значений, допустимых для этого свойства.                                                                                                                                                                                                                                                         |
| **\_ \_ имя атрибута для свойства WPD \_**                  | **VT \_ LPWSTR**  | Строка, указывающая понятное для сценария имя свойства. Допустимые символы: алфавитные буквы \[ a-zA-Z0-9 \] и " \_ ".                                                                                                                                                                                                                                                                    |
| **\_ \_ \_ максимальный диапазон атрибута свойства \_ WPD**            | VT \_ *XXXX*      | Максимальное значение свойства, атрибутом **\_ \_ \_ формы свойства WPD** которого является **\_ \_ \_ \_ диапазон форм атрибутов типа WPD**. Тип данных может быть любым из числовых типов.                                                                                                                                                                                                               |
| **\_ \_ \_ min диапазона атрибута свойства \_ WPD**            | VT \_ *XXXX*      | Минимальное значение свойства, атрибутом **\_ \_ \_ формы свойства WPD** которого является **\_ \_ \_ \_ диапазон форм атрибутов типа WPD**. Тип данных может быть любым из числовых типов.                                                                                                                                                                                                               |
| **\_ \_ шаг атрибута свойства \_ WPD \_**           | VT \_ *XXXX*      | Значение шага для свойства, атрибутом которого является атрибут **\_ свойства \_ \_** WPD, **является \_ \_ \_ \_ диапазон формы атрибута WPD**. Шаг определяет, сколько должно измениться свойство диапазона. Например, свойство с минимальным значением 10, максимальным значением 20, а шаг 5 может иметь следующие значения: **10**, **15**, **20**. Тип данных может быть любым из числовых типов. |
| **\_ \_ \_ регулярное выражение атрибута свойства WPD \_**   | **VT \_ LPWSTR**  | Строка регулярного выражения, указывающая допустимые значения для свойств, форма которых является **\_ \_ \_ \_ регулярным \_ выражением в форме атрибута свойства WPD**.                                                                                                                                                                                                                                             |
| **\_свойство WPD \_ атрибут \_ VarType**               | **VT \_ UI4**     | Целое число, указывающее VARTYPE свойства, например **VT \_ bool**.                                                                                                                                                                                                                                                                                                              |
| **\_ \_ \_ максимальный \_ Размер атрибута свойства WPD**             | **VT \_ UI8**     | Значение, указывающее максимальный размер значения этого свойства в байтах.                                                                                                                                                                                                                                                                                                              |
