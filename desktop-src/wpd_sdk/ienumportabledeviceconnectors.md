# <a name="ienumportabledeviceconnectors-interface"></a>Интерфейс Иенумпортабледевицеконнекторс

Интерфейс **иенумпортабледевицеконнекторс** поддерживает перечисление интерфейсов [**ипортабледевицеконнектор**](/windows/desktop/api/portabledeviceconnectapi/nn-portabledeviceconnectapi-iportabledeviceconnector) , представляющих устройства MTP и Bluetooth, которые были парны с компьютером. Обратите внимание, что при этом будут получены все ранее связанные устройства, включая устройства, которые связаны, но отключены. Чтобы определить, подключено ли устройство, запросите **свойство \_ девпкэй \_ мтпбс** для этого устройства.

## <a name="members"></a>Элементы

Интерфейс **иенумпортабледевицеконнекторс** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иенумпортабледевицеконнекторс** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иенумпортабледевицеконнекторс** содержит следующие методы.



| Метод                                               | Описание                                                                                                                                 |
|:-----------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------|
| [**Клонировать**](ienumportabledeviceconnectors-clone.md) | Создает копию текущего интерфейса **иенумпортабледевицеконнекторс** .<br/>                                                       |
| [**Далее**](ienumportabledeviceconnectors-next.md)   | Извлекает следующий один или несколько объектов [**ипортабледевицеконнектор**](/windows/desktop/api/portabledeviceconnectapi/nn-portabledeviceconnectapi-iportabledeviceconnector) в последовательности перечисления.<br/> |
| [**Перезапуск**](ienumportabledeviceconnectors-reset.md) | Сбрасывает последовательность перечисления в начало.<br/>                                                                                |
| [**Пропустить**](ienumportabledeviceconnectors-skip.md)   | Пропускает указанное число устройств в последовательности перечисления.<br/>                                                               |
