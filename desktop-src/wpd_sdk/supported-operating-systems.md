# <a name="supported-operating-systems"></a>Поддерживаемые операционные системы

Портативные устройства Windows поддерживаются в операционных системах Windows 7, Windows Vista и Windows XP. Приложение будет компилироваться и ссылаться на эти операционные системы с помощью указанных заголовков и библиотек.
