# <a name="network-association-properties"></a>Свойства сетевой ассоциации

Windows Портативные устройства поддерживают следующие свойства ассоциации сети.



| Свойство                                                  | VarType                   | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|-----------------------------------------------------------|---------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ \_ \_ идентификаторы сети узла сетевых ассоциаций WPD \_** | **VT \_ вектор \| VT \_ UI1** | Список идентификаторов узлов EUI-64, допустимых для этой ассоциации. Это массив байтов, содержащий целое число физических сетевых адресов EUI-64. Эти значения EUI-64 — это физические сетевые адреса, доступные на узле во время сетевой ассоциации. Если узел имеет физические сетевые адреса MAC-48 (типичные сети IPv4), каждый адрес MAC-48 будет кодироваться в адрес EUI-64, как два половины адреса MAC-48, разделенные FF-FF.<br/> |
| **\_X509V3SEQUENCE сетевая \_ Ассоциация WPD \_**             | **VT \_ вектор \| VT \_ UI1** | Последовательность сертификатов X. 509 v3, предоставляемых для проверки подлинности сервера TLS.                                                                                                                                                                                                                                                                                                                                                                                                      |
