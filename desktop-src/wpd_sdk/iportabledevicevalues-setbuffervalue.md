# <a name="iportabledevicevaluessetbuffervalue-method"></a>Метод Ипортабледевицевалуес:: Сетбуффервалуе

Метод **сетбуффервалуе** добавляет новое значение типа **Byte** \* (тип VT \_ vector \| VT \_ UI1) или перезаписывает существующий.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetBufferValue(
  [in] REFPROPERTYKEY key,
  [in] BYTE           *pValue,
  [in] DWORD          cbValue
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ключ* \[ окне\]
</dt> <dd>

Объект **рефпропертикэй** , указывающий элемент для создания или перезаписи.

</dd> <dt>

*pValue* \[ окне\]
</dt> <dd>

**Байт \*** , содержащий данные для записи в элемент. Отправленные данные буфера копируются в интерфейс, поэтому вызывающий объект может освободить этот буфер после выполнения этого вызова.

</dd> <dt>

*кбвалуе* \[ окне\]
</dt> <dd>

Размер значения, на которое указывает *pValue*, в байтах.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |



 

## <a name="remarks"></a>Remarks

Если существующее значение имеет тот же ключ, что и параметр *Key* , он перезаписывает существующее значение без предупреждения. Существующий ключ памяти освобождается соответствующим образом.

Установка буфера, **имеющего значение NULL** или нулевого размера, не поддерживается.
