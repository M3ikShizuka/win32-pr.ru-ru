# <a name="wpd_command_sms_send-command"></a>Команда WPD команда \_ \_ \_ Send SMS

Команда **WPD \_ \_ SMS \_ Send** инициирует отправку SMS-сообщения от функционального объекта SMS.

## <a name="command-category"></a>Категория команды

**\_Категория WPD \_ SMS**

## <a name="parameters"></a>Параметры

Драйверу требуются следующие параметры.



| Параметр                              | VarType             | Описание                                                                                                                                                                                                                          |
|----------------------------------------|---------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_ \_ \_ Цель общей команды \_ для свойства WPD | VT \_ LPWSTR          | Обязательный. Идентификатор объекта для функционального объекта SMS, который должен отправить сообщение. Различные функциональные объекты SMS могут иметь разные параметры.                                                                                     |
| \_ \_ получатель SMS свойства \_ WPD          | VT \_ LPWSTR          | Обязательный. Универсальный код ресурса (URI) получателя.                                                                                                                                                                                                  |
| \_ \_ \_ тип сообщения SMS свойства \_ WPD      | VT \_ UI4             | Обязательный. Перечислитель [**\_ \_ типов SMS-сообщений**](sms-message-types.md) , указывающий тип сообщения (текстовый или двоичный).                                                                                                        |
| \_SMS, \_ \_ ТЕКСТОВОЕ \_ сообщение свойства WPD      | VT \_ LPWSTR          | Необязательный элемент. Если **\_ \_ \_ \_ тип сообщения SMS свойства WPD** указывает текстовое сообщение, это строка сообщения; в противном случае этот параметр не включается.                                                                                  |
| \_ \_ \_ двоичное сообщение свойства WPD SMS \_    | VT \_ вектор \| VT \_ UI1 | Необязательный элемент. Если **\_ \_ \_ \_ тип сообщения SMS свойства WPD** указывает на двоичное сообщение, это указатель на массив байтов; в противном случае этот параметр не включается. Первый DWORD значения — это длина массива в байтах. |



 

## <a name="return-value"></a>Возвращаемое значение

Драйвер должен вернуть следующие результаты.



| Результат                                         | VarType   | Описание                                                                                                                                                                                                                                                                                                                                                                            |
|------------------------------------------------|-----------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Свойство WPD, \_ \_ Общее \_ значение HRESULT**             | \_Ошибка VT | Обязательный. **Значение HRESULT** , указывающее на успешное выполнение или неудачу выполнения команды. Если вызывающий объект делает недопустимый запрос, драйвер должен вернуть **значение HRESULT \_ из \_ Win32 (ошибка \_ не \_ поддерживается)** и не требуется возвращать какие-либо другие результирующие значения. Коды ошибок включают в себя [коды ошибок переносных устройств Windows](error-constants.md) или любые другие соответствующие коды ошибок. |
| **\_ \_ \_ код ошибки общего драйвера \_ для свойства WPD \_** | VT \_ UI4   | Необязательный элемент. Код ошибки, относящийся к драйверу. Обычно это используется только для тестирования драйверов или, если все драйверы, устройства и клиенты разработаны вместе.                                                                                                                                                                                                                                |



 

## <a name="calling-methods"></a>Вызов методов
