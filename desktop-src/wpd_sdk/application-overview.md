# <a name="application-overview"></a>Обзор приложений

Windows Портативные устройства (WPD) предоставляют инфраструктуру, которая обеспечивает доступ ПК к многофункциональным устройствам, в которых хранятся различные носители и мультимедийные данные. Эта инфраструктура обеспечивает расхождение устройств и изначально сосредоточена на интеллектуальных запоминающих устройствах, таких как портативные проигрыватели мультимедиа, цифровые камеры и мобильные телефоны.

WPD состоит из интерфейса драйвера устройства (DDI) и набора прикладных программных интерфейсов (API). в этом обзоре описывается, как можно воспользоваться преимуществами этих api в приложении Windows.

Инфраструктура WPD предлагает высоко расширяемые механизмы представления и управления устройствами. Это позволяет поставщикам решений по вертикали использовать API-интерфейс WPD для создания улучшенных пользовательских интерфейсов, которые выходят за пределы стандартизированного набора WPD. К таким примерам относится предоставляемое поставщиком программное обеспечение, которое выполняет обновления встроенного по или диагностику устройств (для удаленной поддержки).
