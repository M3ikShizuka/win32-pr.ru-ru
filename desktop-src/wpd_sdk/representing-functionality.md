# <a name="representing-functionality"></a>Представление функциональных возможностей

Существуют функциональные объекты, которые позволяют вычислить или логически сгруппировать возможности устройства. Например, приложение может видеть, что устройство поддерживает SMS, выполняя поиск функционального объекта SMS. Аналогичным образом приложение может видеть, что устройство имеет возможности камеры, выполняя поиск функционального объекта захвата образа.

Это гибкое представление объекта помогает обеспечить простую поддержку устройств с несколькими функциями. Драйверы могут просто предоставлять любые функциональные объекты, представляющие их устройства, что более детально, чем использование традиционных классов устройств. Представление объекта также помогает изолировать перекрывающиеся функциональные элементы. Например, некоторые телефоны могут иметь две камеры или четыре хранилища.

в операционной системе Windows 7 службы расширяют функциональные объекты путем предоставления обширных запросов возможностей и абстрактного группирования содержимого. Приложения могут использовать службы для обнаружения возможностей устройства и более эффективного взаимодействия с содержимым. Например, приложение может видеть, что устройство поддерживает возможности синхронизации контактов путем поиска объекта службы Contacts и может найти все контакты как дочерние объекты объекта службы без рекурсивного поиска в иерархии хранилища.

Службы также позволяют приложениям обнаруживать и вызывать пользовательское поведение на устройстве.
