# <a name="music-properties"></a>Свойства музыки

Windows Портативные устройства поддерживают следующие свойства музыки.



| Свойство               | VarType        | Описание                                |
|------------------------|----------------|--------------------------------------------|
| **\_музыкальный \_ альбом WPD**  | **VT \_ LPWSTR** | Альбом, к которому относится данная запись.            |
| **\_музыкальные слова для музыкальных файлов WPD \_** | **VT \_ LPWSTR** | Слова песни для музыки.                  |
| **\_музыкальное \_ настроение WPD**   | **VT \_ LPWSTR** | Строка, описывающая настроение музыки. |
| **\_музыкальная \_ ДОРОЖКа WPD**  | **VT \_ UI4**    | Номер направления записи в альбоме. |
