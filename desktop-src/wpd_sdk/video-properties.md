# <a name="video-properties"></a>Свойства видео

Портативные устройства Windows поддерживают следующие свойства видео.



| Свойство                                         | VarType        | Описание                                                                                                                                                                                                                                             |
|--------------------------------------------------|----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ создатель видео WPD**                           | **VT \_ LPWSTR** | Автор видеофайла.                                                                                                                                                                                                                           |
| **\_скорость видео \_ WPD**                          | **VT \_ UI4**    | Скорость воспроизведения видеофайла.                                                                                                                                                                                                                         |
| **\_ \_ Размер буфера видео \_ WPD**                     | **VT \_ UI4**    | Значение, указывающее размер требуемого видеобуфера для подготовки к просмотру этого файла.                                                                                                                                                                              |
| **\_кредиты видео о WPD \_**                          | **VT \_ LPWSTR** | Кредиты, в которых перечислены приведения и сотрудников для видео.                                                                                                                                                                                                    |
| **\_код Video \_ FourCC для WPD \_**                     | **VT \_ DWORD**  | Зарегистрированный код FourCC, указывающий кодек, который использовался для файла видео. Список форматов FourCC см. в статье [зарегистрированные коды FourCC и форматы Wave](https://msdn2.microsoft.com/library/ms867195.aspx) на веб-сайте MSDN. |
| **\_Видеочастота видео WPD \_**                        | **VT \_ UI4**    | Частота кадров видеофайла в кадрах/сек x 1 000. Например, частота кадров 29,97 представлена как 29970.                                                                                                                                 |
| **\_Жанр видео \_ WPD**                            | **VT \_ LPWSTR** | Жанр этого видеофайла. Нет стандартных определений жанра.                                                                                                                                                                                  |
| **\_ \_ \_ расстояние между кадрами ВИДЕОключа WPD \_**             | **VT \_ UI4**    | Интервал между опорными кадрами в миллисекундах.                                                                                                                                                                                                       |
| **\_ \_ Настройка качества видео \_ WPD**                 | **VT \_ UI4**    | Параметр качества для файла видео. Это зависит от кодека.                                                                                                                                                                                        |
| **\_ \_ \_ номер канала рекордедтв видео \_ WPD**      | **VT \_ UI4**    | Номер канала телевидения, из которого записано видео.                                                                                                                                                                                              |
| **\_ \_ \_ Описание программы WPD Video \_ рекордедтв** | **VT \_ LPWSTR** | Описание записанной телепрограммы.                                                                                                                                                                                                       |
| **\_ \_ Повторная РЕКОРДЕДТВ видео WPD \_**               | **Логическое значение VT \_**   | Логическое значение, указывающее, была ли Телепрограмма показана повтор.                                                                                                                                                                     |
| **\_ \_ \_ имя станции рекордедтв ВИДЕОролика WPD \_**        | **VT \_ LPWSTR** | Телевизионная станция, из которой записано видео.                                                                                                                                                                                                |
| **\_Тип сканирования видеороликов WPD \_ \_**                       | **VT \_ UI4**    | Перечислитель [**\_ \_ \_ типов сканирования видео WPD**](wpd-video-scan-types.md) , указывающий чересстрочную развертку видеофайла.                                                                                                                                  |
