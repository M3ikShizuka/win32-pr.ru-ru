# <a name="wpd_content_type_functional_object"></a>\_ \_ \_ функциональный объект типа содержимого WPD \_

Объект, описывающий его тип как \_ \_ функциональный объект содержимого WPD \_ , представляет функциональный объект, инкапсулирующий функциональность устройства.

Все функциональные объекты, независимо от типа, поддерживают следующие свойства. (При определении настраиваемого функционального объекта он также должен поддерживать эти свойства.)



| Имя свойства                                                                                                         | Обязательный или необязательный                                                                  |
|-----------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [\_идентификатор объекта \_ WPD](object-properties.md)                                                                | Обязательно, только для чтения. Клиент не может задать это свойство даже во время создания.        |
| [\_ \_ идентификатор родительского объекта \_ WPD](object-properties.md)                                                 | Обязательный.                                                                             |
| [\_имя объекта \_ WPD](object-properties.md)                                                            | Обязательный.                                                                             |
| [\_ \_ Постоянный \_ уникальный идентификатор объекта \_ WPD](object-properties.md)                          | Обязательно, только для чтения. Клиент не может задать это свойство даже во время создания.        |
| [\_Формат объекта \_ WPD](object-properties.md)                                                        | Обязательный.                                                                             |
| [\_ \_ тип содержимого объекта \_ WPD](object-properties.md)                                           | Обязательный.                                                                             |
| [\_объект WPD \_](object-properties.md)                                                    | Требуется, если объект скрыт.                                                     |
| [\_система объектов \_ WPD](object-properties.md)                                                    | Требуется, если объект является системным объектом (представляет системный файл).                 |
| [\_размер объекта \_ WPD](object-properties.md)                                                            | Требуется, если у объекта есть по крайней мере один ресурс.                                     |
| [\_ \_ \_ имя ИСХОДНОГО файла \_ объекта WPD](object-properties.md)                              | Требуется, если объект представляет файл.                                             |
| [\_объект WPD \_ не \_ поднимается](object-properties.md)                                       | Рекомендуется, если объект не предназначен для использования устройством.                 |

| [\_ \_ Ключевые слова объекта WPD](object-properties.md)                                                    | Необязательный элемент.                                                                             |
| [\_ \_ идентификатор синхронизации объекта \_ WPD](object-properties.md)                                                     | Необязательный элемент.                                                                             |

| [\_ \_ Дата создания объекта \_ WPD](object-properties.md)                                           | Необязательный элемент.                                                                             |
| [\_ \_ Дата изменения объекта \_ WPD](object-properties.md)                                         | (рекомендуется).                                                                          |
| [\_ \_ Дата создания объекта \_ WPD](object-properties.md)                                         | Необязательный элемент.                                                                             |
| [\_ \_ обратные ссылки объекта WPD \_](object-properties.md)                                                                | Рекомендуется, если на объект ссылается другой объект.                            |

## <a name="typical-resources"></a>Типичные ресурсы

Обычно эти объекты не размещают ресурсы.

## <a name="functional-object-categories"></a>Функциональные категории функционального объекта

Функциональные объекты могут быть сгруппированы по категориям в зависимости от их функций. Категория описывается \_ свойством "Категория функционального \_ объекта WPD" \_ (значение идентификатора GUID). Категория определяет, какие дополнительные свойства поддерживаются.

В следующей таблице описаны категории, определяемые переносными устройствами Windows. Сведения о дополнительных свойствах и ресурсах, поддерживаемых объектом, см. в описании категории.



| Функциональная категория                                                                                        | Описание                                                                                                                                                                                               |
|------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_функциональная \_ Категория WPD \_ все**](wpd-functional-category-all.md)                                      | Эта функциональная Категория допустима только в качестве параметра для определенных функций запроса (чтобы указать, что все типы функциональных объектов приемлемы) и не являются зарегистрированной функциональной категорией драйвера. |
| [**Функция \_ \_ \_ отслеживания звука для ФУНКЦИОНАЛЬНой категории WPD \_**](wpd-functional-category-audio-capture.md)                 | Объект инкапсулирует функцию захвата звука на устройстве, например средство записи голоса или другой компонент записи звука.                                                                      |
| [**\_устройство функциональной \_ категории WPD \_**](wpd-functional-category-device.md)                                | Объект инкапсулирует устройство (то есть самый верхний объект устройства).                                                                                                                          |
| [**\_Конфигурация сети для функциональной \_ категории WPD \_ \_**](wpd-functional-category-network-configuration.md) | Объект инкапсулирует функции сетевой конфигурации для устройства, например профили Wi-Fi или партнерство.                                                                                   |
| [**\_ \_ \_ сведения о визуализации ФУНКЦИОНАЛЬНой категории WPD \_**](wpd-functional-category-rendering-information.md) | Объект описывает типы файлов мультимедиа, которые устройство может воспроизводить.                                                                                                                            |
| [**\_функциональная \_ Категория WPD \_ SMS**](wpd-functional-category-sms.md)                                      | Объект инкапсулирует функцию коротких сообщений (обычно называется "Text Messaging") на устройстве.                                                                                             |
| [**\_функциональная \_ Категория WPD \_ все \_ еще \_ отслеживает образы**](wpd-functional-category-still-image-capture.md)    | Объект инкапсулирует функцию захвата изображений на устройстве, например на камере или в качестве вложения камеры.                                                                                              |
| [**\_хранилище функциональной \_ категории WPD \_**](wpd-functional-category-storage.md)                              | Объект инкапсулирует физическое хранилище файлов на устройстве.                                                                                                                                              |
| [**\_запись видео о функциональной \_ категории WPD \_ \_**](wpd-functional-category-video-capture.md)                 | Объект инкапсулирует функцию видеозаписи на устройстве, например компонент записи видео. Приложение использует этот объект для получения программного управления.                                 |
