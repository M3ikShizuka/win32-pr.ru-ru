# <a name="iportabledevicevaluescollectiongetat-method"></a>Метод Ипортабледевицевалуесколлектион:: GetAt

Метод **GetAt** извлекает элемент из коллекции по индексу, начинающимся с нуля.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetAt(
  [in]  const DWORD                 dwIndex,
  [out]       IPortableDeviceValues **ppValues
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*двиндекс* \[ окне\]
</dt> <dd>

Значение **типа DWORD** , указывающее Отсчитываемый от нуля индекс в коллекции.

</dd> <dt>

*ппвалуес* \[ заполняет\]
</dt> <dd>

Адрес переменной, которая получает указатель на интерфейс [**ипортабледевицевалуес**](iportabledevicevalues.md) из коллекции. Вызывающий объект отвечает за вызов **Release** на этом интерфейсе, когда он выполняется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                                  | Описание                                                                      |
|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Метод выполнен успешно.<br/>                                                 |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl> | Отсчитываемый от нуля индекс, который был передан, был вне допустимого диапазона.<br/>             |
| <dl> <dt>**\_указатель E**</dt> </dl>    | Обязательный аргумент-указатель имеет **значение NULL**.<br/>                             |
| <dl> <dt>**\_непредвиденная E**</dt> </dl> | Коллекция содержит указатель ипортабледевицевалуес **со значением NULL**  .<br/> |



 

## <a name="remarks"></a>Комментарии

Все изменения, внесенные в значения в полученном интерфейсе, будут внесены в версию, хранящуюся в коллекции.
