# <a name="class-extension-properties"></a>Свойства расширения класса

Windows Портативные устройства поддерживают следующие свойства расширения класса.



| Свойство                                                                      | VarType         | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|-------------------------------------------------------------------------------|-----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ Параметры расширения класса \_ WPD \_ поддерживаемые \_ \_ типы содержимого**                 | **VT \_ Unknown** | Значение, указывающее список (надмножество) типов содержимого, поддерживаемых драйвером (аналогично вызову \_ возможностей команды WPD \_ \_ получить \_ поддерживаемые \_ \_ типы содержимого в **\_ функциональной \_ категории WPD \_ ALL**).                                                                                                                                                                                                                                                                                                                                             |
| **\_ \_ Параметры расширения класса \_ WPD \_ не \_ зарегистрировать \_ \_ адаптер устройства \_ WPD**    | **Логическое значение VT \_**    | Значение типа, указывающее, требуется ли вызывающему объекту библиотеку расширений класса WPD для регистрации интерфейса класса устройства WPD. Если это значение равно true, вызывающий объект принимает ответственность за регистрацию.<br/> Если это значение равно false, то это означает, что вызывающий объект ждет, что библиотека расширений классов будет выполнять регистрацию.<br/>Большинство драйверов должны позволить библиотеке расширений классов выполнить регистрацию, за исключением случаев, когда регистрация интерфейса класса устройства WPD библиотекой расширений классов может привести к неблагоприятным последствиям. |
| **\_ \_ Параметры расширения класса \_ WPD \_ зарегистрировать \_ \_ частный \_ интерфейс устройства \_ WPD** | **Логическое значение VT \_**    | Указывает, что вызывающему объекту требуется библиотека расширений класса WPD для регистрации интерфейса закрытого класса устройства WPD. Это не рекомендуется для большинства драйверов. Его следует использовать только в тех случаях, когда регистрация интерфейса класса устройства WPD с помощью библиотеки расширений классов приведет к неблагоприятным последствиям. Этот параметр обычно используется в сочетании с **\_ \_ параметрами расширения класса WPD \_ \_ не \_ зарегистрировать \_ WPD \_ Device \_ Interface** со значением **true** .                                                                                          |
| **\_ \_ Параметры расширения класса \_ WPD \_ \_ значения идентификаторов устройств \_**            | **VT \_ Unknown** | Это [ипортабледевицевалуес](iportabledevicevalues.md) , которая содержит идентификационные значения устройства (**\_ \_ производитель устройства** WPD, **\_ \_ модель устройства WPD**, **\_ \_ \_ Версия встроенного по устройства WPD** и **\_ \_ идентификатор функционального \_ уникального \_ идентификатора устройства WPD**). Включить это с другими параметрами расширения класса при инициализации                                                                                                                                                                                                                               |
| **\_ \_ Параметры расширения класса \_ WPD \_ \_ пропускная способность транспорта**                      | **VT \_ UI4**     | Указывает теоретическую максимальную пропускную способность транспорта в килобитах в секунду                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **\_ \_ Параметры расширения класса \_ WPD \_ \_ значения идентификаторов устройств \_**            | **VT \_ Unknown** | Это [ипортабледевицевалуес](iportabledevicevalues.md) , которая содержит идентификационные значения устройства (**\_ \_ производитель устройства** WPD, **\_ \_ модель устройства WPD**, **\_ \_ \_ Версия встроенного по устройства WPD** и **\_ \_ идентификатор функционального \_ уникального \_ идентификатора устройства WPD**). Включите это с другими параметрами расширения класса при инициализации.                                                                                                                                                                                                                              |
