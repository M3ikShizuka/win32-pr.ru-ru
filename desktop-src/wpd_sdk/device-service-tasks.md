# <a name="device-service-tasks"></a>Задачи службы устройств

API-интерфейс WPD поддерживает ряд задач служб устройств. В том числе можно делать следующее.

-   Перечисление служб устройств.
-   Открытие службы устройства.
-   Перечисление содержимого в службе устройства.
-   Получение поддерживаемых событий службы.
-   Получение поддерживаемых форматов службы.
-   Получение поддерживаемых методов службы.
-   Получение поддерживаемых абстрактных служб.
-   Получение свойств объекта содержимого.
-   Запись свойств содержимого объекта.
-   Синхронный вызов методов службы.
-   Асинхронный вызов методов службы.

Эти операции описаны в следующих разделах и демонстрируются в приложении Впдсервицесаписампле. Кроме того, API-интерфейс WPD также поддерживает передачу содержимого, перемещение содержимого и удаление содержимого из службы устройства.
