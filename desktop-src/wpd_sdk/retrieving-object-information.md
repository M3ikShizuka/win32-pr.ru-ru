# <a name="retrieving-object-information"></a>Получение сведений об объекте

Помимо получения сведений об устройстве, API-интерфейс WPD предоставляет набор интерфейсов и методов, которые можно использовать для получения сведений для заданного объекта. Как правило, извлечение сведений об объектах можно разделить на две категории, как описано в следующей таблице.



| Данные устройства                                                                                    | Описание                                                                               |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [Идентификатор объекта](retrieving-an-object-identifier-from-a-persistent-unique-identifier.md)   | Можно получить идентификатор объекта, сопоставляемый с заданным постоянным уникальным идентификатором. |
| [Функциональные идентификаторы объектов](retrieving-the-functional-object-identifiers-for-a-device.md) | Каждая функциональная Категория поддерживает один или несколько функциональных объектов.                         |
