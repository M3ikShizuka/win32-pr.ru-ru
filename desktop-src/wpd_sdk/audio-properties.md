# <a name="audio-properties"></a>Свойства звука

Портативные устройства Windows поддерживают следующие свойства аудио.



| Свойство                         | VarType     | Описание                                                                                                                                                                                                        |
|----------------------------------|-------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ Битовая глубина звука \_ WPD**       | **VT \_ UI4** | Битовая глубина звука.                                                                                                                                                                                        |
| **\_звуковая \_ скорость WPD**          | **VT \_ UI4** | Битовая скорость звука в битах в секунду.                                                                                                                                                                     |
| **\_Выравнивание аудио \_ блока \_ WPD** | **VT \_ UI4** | Выравнивание блока для звукового файла в байтах.                                                                                                                                                                   |
| **\_число звуковых \_ каналов WPD \_**   | **VT \_ R4**  | Число каналов в этом звуковом файле, например 1, 2 или 5,1.                                                                                                                                              |
| **\_код звукового \_ формата WPD \_**     | **VT \_ UI4** | Номер зарегистрированного кода формата WAVE. Список зарегистрированных форматов WAVE см. в статье [зарегистрированные коды FourCC и форматы Wave](https://msdn2.microsoft.com/library/ms867195.aspx) на веб-сайте MSDN. |
