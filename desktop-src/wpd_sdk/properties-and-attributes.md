# <a name="properties-and-attributes"></a>Свойства и атрибуты

Свойства описывают объекты, свойства, ресурсы и устройства. Атрибуты — это особый тип свойства, который описывает другие свойства или ресурсы.

Свойства и атрибуты определяются как структуры **PROPERTYKEY** с двумя частями: GUID категории и уникальный идентификатор для этой категории. Свойства, описывающие связанные объекты, имеют общий идентификатор GUID. Этот идентификатор GUID является категорией свойств. Категория может описывать одно или несколько свойств, каждый из которых имеет уникальный идентификатор. Корпорация Майкрософт определяет набор категорий свойств. их можно использовать для собственных свойств или самостоятельно создавать новые категории.
