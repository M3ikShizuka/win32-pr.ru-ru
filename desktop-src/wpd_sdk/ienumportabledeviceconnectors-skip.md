# <a name="ienumportabledeviceconnectorsskip-method"></a>Метод Иенумпортабледевицеконнекторс:: Skip

Метод **Skip** пропускает указанное число устройств в последовательности перечисления.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Skip(
  [in] UINT32 cConnectors
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*кконнекторс* \[ окне\]
</dt> <dd>

Число пропускаемых устройств.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                             | Описание                                                                                                                                                                               |
|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>    | Метод выполнен успешно.<br/>                                                                                                                                                          |
| <dl> <dt>**S \_ false**</dt> </dl> | Не удалось пропускать указанное число устройств. Одна из возможных причин: параметр *кконнекторс* указывает больше устройств, чем фактически остается в последовательности перечисления.<br/> |
