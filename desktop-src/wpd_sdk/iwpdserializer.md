# <a name="iwpdserializer-interface"></a>Интерфейс Ивпдсериализер

Интерфейс **ивпдсериализер** используется драйвером устройства для сериализации интерфейсов [**ипортабледевицевалуес**](iportabledevicevalues.md) в необработанные буферы данных, используемые для взаимодействия с приложением.

Приложениям не требуется использовать этот интерфейс, так как данные сериализуются и десериализуются автоматически при вызове [**ипортабледевице:: сендкомманд**](/windows/desktop/api/PortableDeviceApi/nf-portabledeviceapi-iportabledevice-sendcommand).

Чтобы получить этот интерфейс, вызовите **CoCreateInstance** и передайте **IID \_ ивпдсериализер**.

## <a name="members"></a>Элементы

Интерфейс **ивпдсериализер** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Ивпдсериализер** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивпдсериализер** содержит следующие методы.



| Метод                                                                                          | Описание                                                                                                      |
|:------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------|
| [**жетбуфферфромипортабледевицевалуес**](iwpdserializer-getbufferfromiportabledevicevalues.md) | Сериализует отправленный интерфейс **ипортабледевицевалуес** в выделенный массив байтов.<br/>                |
| [**жетипортабледевицевалуесфромбуффер**](iwpdserializer-getiportabledevicevaluesfrombuffer.md) | Десериализует массив байтов в интерфейс **ипортабледевицевалуес** .<br/>                                  |
| [**жетсериализедсизе**](iwpdserializer-getserializedsize.md)                                   | Вычисляет размер буфера, необходимого для хранения сериализованного интерфейса **ипортабледевицевалуес** .<br/> |
| [**вритеипортабледевицевалуестобуффер**](iwpdserializer-writeiportabledevicevaluestobuffer.md) | Сериализует интерфейс **ипортабледевицевалуес** в выделенный вызывающим объектом массив байтов.<br/>                   |
