# <a name="mtp-extension-events"></a>События расширения MTP

События уведомляют приложение о том, что изменения произошли на устройстве или с. Например, приложение может зарегистрироваться для получения нотфикатионс об удалении устройства.

## <a name="vendor-extended-event-codes"></a>Коды расширенных событий поставщика

Когда производитель устройства поддерживает событие, расширенное поставщиком, его драйвер объединяет код события поставщика (UINT16) с старшим 16 разрядами идентификатора GUID **\_ \_ \_ \_ расширенных \_ событий поставщика событий WPD MTP** .

Например, если код, расширенный поставщиком, — 0xC001, то результирующий идентификатор GUID будет выглядеть так, как показано в следующем примере:


```C++
{C0010000-5738-4ff2-8445-BE3126691059}
```



## <a name="vendor-extended-event-parameters"></a>Параметры расширенных событий поставщика

Параметры для расширенного события поставщика передаются по **\_ \_ \_ \_ идентификатору события "параметр события WPD** " и **свойству "WPD" \_ события " \_ MTP \_ ext \_ \_**", которое представляет собой коллекцию **пропвариантс**. Эти **пропвариантс** соответствуют параметрам события. Если параметры отсутствуют, эта коллекция пуста.


```C++
{C0010000-5738-4ff2-8445-BE3126691059}
```
