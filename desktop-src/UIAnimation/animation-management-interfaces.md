# <a name="animation-management-interfaces"></a>Интерфейсы управления анимацией

в этом разделе содержатся справочные спецификации для интерфейсов диспетчера анимации Windows, поддерживающих управление анимациями.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                   | Описание                                                                                                                                                              |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**иуианиматионманажер**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager)<br/>                           | Определяет диспетчер анимации, предоставляющий центральный интерфейс для создания и управления анимациями.<br/>                                                       |
| [**IUIAnimationManager2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager2)<br/>                         | Определяет [**диспетчер анимации**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager2), предоставляющий центральный интерфейс для создания и управления анимациями в нескольких измерениях.<br/> |
| [**иуианиматионманажеревенсандлер**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanagereventhandler)<br/>   | Определяет метод обработки обновлений состояния для диспетчера анимации.<br/>                                                                                         |
| [**IUIAnimationManagerEventHandler2**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanagereventhandler2)<br/> | Определяет метод обработки обновлений для [**диспетчера анимации**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationmanager2).<br/>                                                                |
