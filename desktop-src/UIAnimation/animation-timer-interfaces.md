# <a name="animation-timer-interfaces"></a>Интерфейсы таймера анимации

В этом разделе содержатся справочные спецификации для интерфейсов диспетчера анимации Windows, которые поддерживают таймеры анимации.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                         | Описание                                                                                   |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**иуианиматионтимер**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimer)<br/>                                     | Определяет таймер анимации, который предоставляет службы для управления временем анимации.<br/> |
| [**иуианиматионтимерклиентевенсандлер**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerclienteventhandler)<br/> | Определяет метод обработки событий, связанных с изменениями в состоянии клиента таймера.<br/>    |
| [**иуианиматионтимеревенсандлер**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimereventhandler)<br/>             | Определяет методы для обработки временных событий.<br/>                                        |
| [**иуианиматионтимерупдатехандлер**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerupdatehandler)<br/>           | Определяет методы для обработки событий обновления времени.<br/>                                 |
