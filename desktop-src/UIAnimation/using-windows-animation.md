# <a name="windows-animation-tasks"></a>Задачи анимации Windows

Разделы, содержащиеся в этом разделе, описывают основные задачи, необходимые для приложений, использующих Windows Animation Manager.

Эти задачи по порядку включают:

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                                | Описание                                                                                                                                                                                                                                                                                           |
|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Создание основных объектов анимации](adding-animation-to-an-application.md)<br/>               | Чтобы использовать анимацию Windows в приложении, первым шагом является создание небольшого набора основных объектов анимации.<br/>                                                                                                                                                                           |
| [Создание переменных анимации](create-animation-variables.md)<br/>                              | Приложение должно создать переменную анимации для каждой визуальной характеристики, которая должна быть анимирована с помощью анимации Windows.<br/>                                                                                                                                                            |
| [Обновление диспетчера анимации и рисование кадров](introducing-windows-animation-manager.md)<br/> | Каждый раз, когда приложение планирует раскадровку, приложение должно передать текущее время в диспетчер анимации. Текущее время также требуется, если диспетчеру анимации нужно обновить свое состояние и задать для всех переменных анимации соответствующие значения интерполяции.<br/> |
| [Чтение значений переменных анимации](updating---application-driven-animation.md)<br/>         | Каждый раз, когда приложение рисуется, оно должно считывать текущие значения переменных анимации, представляющих визуальные характеристики для анимации.<br/>                                                                                                                                  |
| [Создание раскадровки и добавление переходов](updating---timer-driven-animation.md)<br/>          | Чтобы создать анимацию, приложение должно создать раскадровку.<br/>                                                                                                                                                                                                                        |
| [Планирование раскадровки](scheduling-a-storyboard.md)<br/>                                      | После создания раскадровки она планируется диспетчером анимации.<br/>                                                                                                                                                                                                                   |
