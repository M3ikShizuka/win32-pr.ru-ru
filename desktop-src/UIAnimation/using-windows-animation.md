в разделах, содержащихся в этом разделе, описываются основные задачи, необходимые для приложений, использующих Windows диспетчера анимации.

Эти задачи по порядку включают:

| Раздел                                                                                                | Описание                                                                                                                                                                                                                                                                                           |
|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Создание основных объектов анимации](adding-animation-to-an-application.md)<br/>               | чтобы использовать анимацию Windows в приложении, первым шагом является создание небольшого набора основных объектов анимации.<br/>                                                                                                                                                                           |

| [Обновление диспетчера анимации и рисование кадров](introducing-windows-animation-manager.md)<br/> | Каждый раз, когда приложение планирует раскадровку, приложение должно передать текущее время в диспетчер анимации. | [Создание раскадровки и добавление переходов](updating---timer-driven-animation.md)<br/>          | Чтобы создать анимацию, приложение должно создать раскадровку.<br/>                                                                                                                                                                                                                        |
