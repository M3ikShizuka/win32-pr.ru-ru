# <a name="systemgpsaltitude-photo-metadata-policy"></a>Политика метаданных фото System. GPS. Высота

Политика метаданных фотографии для свойства [System. GPS. Высота](../properties/props-system-gps-altitude.md) .

### <a name="pkey"></a>PKEY

По \_ высоте GPS на PKEY \_

### <a name="description"></a>Описание

Высота указывается в виде абсолютного значения, на которое ссылается измерительные единицы.

### <a name="containers"></a>Контейнеры

JPEG, TIFF

### <a name="read-only"></a>Только для чтения

Да

### <a name="output-propvariant-type"></a>Тип выходного ПРОПВАРИАНТ

VT \_ R8

### <a name="input-propvariant-type"></a>Тип входного ПРОПВАРИАНТ

VT \_ R8

### <a name="conflict-resolution-policy"></a>Политика разрешения конфликтов

Это значение можно записать, записав в System. GPS. Высота. числитель и System. GPS. Высота. знаменатель. Он не может быть записан напрямую. Пути записи в следующих таблицах указывают, где значение может быть сохранено при его создании, а не когда оно записывается напрямую. При считывании значения из разных схем выверятся.
