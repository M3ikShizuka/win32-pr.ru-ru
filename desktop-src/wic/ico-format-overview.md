# <a name="ico-format-overview"></a>Общие сведения о формате ICO

в этом разделе содержатся сведения о кодека машинного кода ICO, доступного через компонент Windows imaging (WIC).

## <a name="codec-identity"></a>Удостоверение кодека

В следующей таблице приведены сведения об идентификации кодека.



|   Компонент            | Описание       |
|------------------------|-------------------|
| Формальные имена         | Формат значка (ICO) |
| Расширения имени файла | ICO               |
| тип MIME              | изображение или ICO         |
| Поддержка спецификаций  |                   |



 

В следующей таблице перечислены идентификаторы GUID, используемые для поиска компонентов кодека машинного кода ICO.



| Компонент        | Понятное имя            | GUID                                |
|------------------|--------------------------|-------------------------------------|
| Формат контейнера | GUID \_ контаинерформатико | a3a860c4-338f-4c17-919afba4b5628f21 |
| Показан          | \_ВИЦИКОДЕКОДЕР CLSID     | c61bfcdf-2e0f-4aad-a8d7e06bafebcdfe |
| Кодировщик          | НЕДОСТУПНО            | НЕДОСТУПНО                       |



 

## <a name="encoding"></a>Кодирование

Компонент WIC не предоставляет собственный кодировщик изображений ICO.

## <a name="decoding"></a>Декодирование

Интерфейс API декодирования WIC разработан как независимый от кодека, а декодирование изображений для кодеков, поддерживающих WIC, по сути является одинаковым. Дополнительные сведения о декодировании изображений см. в разделе [Общие сведения о декодировании](-wic-creating-decoder.md). Дополнительные сведения об использовании декодированных данных изображения см. в разделе [Обзор источников точечных рисунков](-wic-bitmapsources.md).
