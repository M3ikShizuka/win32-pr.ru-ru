# <a name="systemphotoprogrammode-photo-metadata-policy"></a>Политика метаданных фото для System. photo. Программоде

### <a name="pkey"></a>PKEY

\_Программоде фото на PKEY \_

### <a name="containers"></a>Контейнеры

JPEG, TIFF

### <a name="read-only"></a>Только для чтения

Нет

### <a name="output-propvariant-type"></a>Тип выходного ПРОПВАРИАНТ

### <a name="input-type"></a>Тип входных данных
