# <a name="codec-availability-and-platform-support"></a>Доступность кодеков и поддержка платформ

корпорация майкрософт рекомендует, чтобы поставщики камер включали обновленные кодеки необработанных Windows изображений (WIC) в дистрибутиве программного обеспечения с новыми камерами и что обновленный кодек устанавливается вместе с установкой по по умолчанию для поставщиков Windows 7, Windows Vista и Windows XP. Производители камер также должны предоставить последний кодек WIC как загружаемый с веб-узлов поддержки.

## <a name="codec-discovery"></a>Обнаружение кодеков

Корпорация Майкрософт выполняет следующие действия, чтобы помочь пользователям в обращении к веб-сайтам поставщиков для загрузки кодеков:

-   в обозревателе Windows, если пользователь дважды щелкает файл, который не распознается (вариант по умолчанию, когда необработанный файл находится на компьютере, но кодек еще не установлен), появится диалоговое окно с сообщением о том, что файл не распознан и спрашивает, требуется ли пользователю найти программу, которая понимает файл. Корпорация Майкрософт поддерживает существующую службу перенаправления для перенаправления пользователей на веб-сайты поставщиков, которые могут предоставить программе сведения о файле. это средство существует в Windows XP и в Windows Vista.
-   Windows фотоальбом, Фотоальбом Windows Live и средство просмотра фотографий Windows 7 распознают набор расширений файлов как необработанные файлы. Если файлы из этого набора находятся в папках, которые просматриваются каким-либо из этих приложений, а кодек для этого файла еще не установлен, появится диалоговое окно, как описано в предыдущем абзаце.

Дополнительные сведения об обнаружении кодеков см. в разделе Доступность кодеков и поддержка платформ.

## <a name="windows-xp-platform-support"></a>Windows Поддержка платформы XP

корпорация майкрософт выпустила Windows XP SP3 с WIC и автономный установщик WIC для Windows XP с пакетом обновления 2 (SP2). Они используют кодеки WIC для включения поддержки эскизов и предварительного просмотра на этих платформах. поэтому важно, чтобы загрузка и установка кодеков поддерживалась и тестировалась на Windows 7, Windows Vista и Windows XP.
