# <a name="iwmdrmsecuritygetmachinecertificate-method"></a>Метод Ивмдрмсекурити:: Жетмачинецертификате

Метод **жетмачинецертификате** извлекает сертификат компьютера подсистемы DRM на клиентском компьютере.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetMachineCertificate(
  [in]      DWORD dwCertificateType,
  [out]     BYTE  rgbVersion[4],
  [out]     BYTE  **ppbCertificate,
  [in, out] DWORD *pcbCertificate
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*двцертификатетипе* \[ окне\]
</dt> <dd>

Тип получаемого сертификата. Задайте одно из значений, приведенных в следующей таблице.



| Значение                        | Описание                                                                           |
|------------------------------|---------------------------------------------------------------------------------------|
| \_Тип сертификата \_ WMDRM \_ v1 | Сертификат будет извлечен в формате, используемом компонентами предыдущих версий.            |
| \_Тип сертификата \_ WMDRM \_ v2 | сертификат будет извлечен в формате, используемом компонентами Windows Vista. |



 

</dd> <dt>

*ргбверсион \[ 4 \]* \[ out\]
</dt> <dd>

Массив из четырех байтов, указывающих версию подсистемы DRM на клиентском компьютере.

</dd> <dt>

*ппбцертификате* \[ заполняет\]
</dt> <dd>

Адрес переменной, которая получает указатель на данные сертификата. Задайте для параметра **значение NULL** , чтобы метод предоставил размер буфера, необходимый для хранения сертификата в *пкбцертификате*.

</dd> <dt>

*пкбцертификате* \[ в, out\]
</dt> <dd>

Размер сертификата в байтах. Если *ппбцертификате* имеет **значение NULL**, для этого значения будет задан размер сертификата. Если *ппбцертификате* не равно **null**, это значение должно быть равно размеру буфера.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |
