# <a name="transcoding-files-qasf"></a>Перекодирование файлов (КАСФ)

Вы можете создать граф фильтра для перекодирования файлов с помощью [модуля записи WM ASF](wm-asf-writer-filter.md) различными способами. Самым простым способом является совместное создание, Настройка и Добавление модуля записи WM ASF в граф фильтра, а затем использование метода **играфбуилдер:: renderFile** для автоматической сборки графа.

Другой способ — добавить каждый фильтр на диаграмму вручную и соединить эти контакты. После добавления модуля записи WM ASF настройте его с помощью методов **иконфигасфвритер** , если профиль по умолчанию не подходит, и подключите входные ПИН-коды модуля записи WM ASF к соответствующим выходным контактам в вышестоящем фильтре.
