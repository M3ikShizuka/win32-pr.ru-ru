# <a name="wmt_videoimage_transition_cross_fade"></a>\_ \_ \_ перекрестное выцветание перехода ВМТ видеоимаже \_

Переход с перекрестной выцветанием не имеет специального эффекта; характеристики исчезания (растворение) определяются членами **фпревблендкоеф** и **фкуррблендкоеф** структуры [**ВМТ \_ видеоимаже \_ SAMPLE2**](/previous-versions/windows/desktop/api/Wmsdkidl/ns-wmsdkidl-wmt_videoimage_sample2) .

## <a name="parameters"></a>Параметры

Этот переход не принимает параметров.

## <a name="remarks"></a>Комментарии

Настройка этого перехода аналогична установке элемента **двеффекттипе** структуры **ВМТ \_ видеоимаже \_ SAMPLE2** в значение 0.
