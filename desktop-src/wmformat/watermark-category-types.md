# <a name="watermark-category-types"></a>Типы категорий водяных знаков

Для получения сведений о водяных знаках дмос, установленных на компьютере, можно использовать функцию перечисления DMO **дмоенум**. Пакет SDK для формата Windows Media определяет идентификаторы GUID, которые определяют категории подложек DMO. Дополнительные сведения о **дмоенум** см. в документации по DirectShow на сайте MSDN.

В следующей таблице перечислены идентификаторы для категорий водяных знаков.



| Константа категории DMO с водяными знаками | Код GUID                                 |
|------------------------------------|--------------------------------------|
| ВМТ \_ дмокатегори \_ аудио \_ водяной знак | 65221c5a-fa75-4b39-b50c-06c336b6a3ef |
| ВМТ \_ дмокатегори \_ видео \_ | 187cc922-8efc-4404-9daf-63f4830df1bc |
