# <a name="bandwidth-sharing-types"></a>Типы совместного использования пропускной способности

Можно использовать типы совместного использования пропускной способности, чтобы указать природу объекта совместного использования пропускной способности в профиле. Типы совместного использования пропускной способности используются в качестве параметров для [**ивмбандвидсшаринг:: GetType**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmbandwidthsharing-gettype) и [**Ивмбандвидсшаринг:: сеттипе**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmbandwidthsharing-settype).

В следующей таблице перечислены идентификаторы для типов, совместно использующих пропускную способность.



| Константа типа совместного использования пропускной способности      | GUID                                 |
|--------------------------------------|--------------------------------------|
| CLSID \_ вмбандвидсшаринг \_ монопольно | af6060aa-5197-11d2-b6af-00c04fd908e9 |
| \_ВМБАНДВИДСШАРИНГ CLSID \_ частично   | af6060ab-5197-11d2-b6af-00c04fd908e9 |
