# <a name="iwmdrmsecuritygetsecurityversion-method"></a>Метод Ивмдрмсекурити:: Жетсекуритиверсион

Метод **жетсекуритиверсион** извлекает версию подсистемы DRM на клиентском компьютере.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetSecurityVersion(
  [out] BSTR *pbstrVersion
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пбстрверсион* \[ заполняет\]
</dt> <dd>

Адрес переменной, которая получает номер версии подсистемы DRM.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |



 

## <a name="remarks"></a>Комментарии

Номер версии выражается в виде строки, состоящей из четырех чисел, разделенных точками. Первое число — основной номер версии, который обычно имеет значение 2. Второе число — это дополнительный номер версии в диапазоне от 2 до 10. Третье число всегда равно 0. Четвертое число устанавливается в 0 или 1, а является логическим значением, указывающим, является ли подсистема индивидуальной. Например, номер версии "2.4.0.1" указывает на отдельную четвертую младшую версию второй основной версии.
