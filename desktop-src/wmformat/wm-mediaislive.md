# <a name="wmmediaislive"></a>WM/Медиаисливе

Атрибут **WM/медиаисливе** указывает, является ли содержимое широковещательным.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиаисливе

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут используется для трансляции телепередач.
