# <a name="drm-individualization"></a>Индивидуальное управление цифровыми правами

владельцам защищенного содержимого могут потребоваться обновления некоторых компонентов управления цифровыми правами (DRM), включенных в пакет SDK Windows Media Format, перед обращением к их содержимому. Если пользователь принимает обновление, отдельная версия файла безопасности (уникально для компьютера) будет скачана с веб-сайта корпорации Майкрософт. Если пользователь отклоняет обновление, он не сможет получить доступ к содержимому. Тем не менее они по-прежнему смогут обращаться к незащищенному содержимому и защищенному содержимому, которое не требует обновления. Выполнение обновления системы безопасности позволяет повысить уровень защиты, предлагаемый DRM.

Индивидуальная настройка может выполняться либо во время установки, либо когда приложение впервые встречает защищенный ASF-файл, требующий этого. Поскольку этот процесс изменяет системные компоненты пользователя, приложение должно уведомить пользователя и получить информированное согласие перед выполнением обновления. в Microsoft проигрыватель Windows Media, например, отображается диалоговое окно со следующим текстом:

**Требуется обновление системы безопасности**

**Владелец защищенного содержимого, к которому вы пытаетесь получить доступ, требует сначала обновить некоторые компоненты Microsoft Digital Rights Management (DRM) на вашем компьютере.**

**Нажмите кнопку ОК, чтобы обновить компоненты DRM.**

**Дополнительно**

**После нажатия кнопки ОК в службу Microsoft в Интернете отправляются уникальный идентификатор и файл безопасности DRM. Файл заменяется настроенной версией, содержащей уникальный идентификатор. Это повышает уровень защиты, обеспечиваемый DRM.**

Любое приложение, поддерживающее индивидуальную установку, должно использовать те же или похожие слова. Дополнительные сведения о политике конфиденциальности корпорации Майкрософт см. в разделе заявление [о конфиденциальности](privacy-statement.md) этой документации.

> [!Note]  
> DRM не поддерживает версию этого пакета SDK на базе x64.
