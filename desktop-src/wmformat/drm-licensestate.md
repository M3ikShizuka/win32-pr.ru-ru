# <a name="drm_licensestate"></a>\_ЛИЦЕНСЕСТАТЕ DRM

Свойство **\_ лиценсестате DRM** не используется. В действительности это не свойство, а просто корневая строка для других идентификаторов строк свойств состояния лицензии, таких как **DRM \_ лиценсестате \_ воспроизведения**.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдрм \_ лиценсестате

## <a name="data-type"></a>Тип данных

**\_ \_ двоичный тип ВМТ**

## <a name="remarks"></a>Комментарии

Это свойство не возвращает никаких сведений.
