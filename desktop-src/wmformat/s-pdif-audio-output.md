# <a name="spdif-audio-output"></a>Выходные данные S/PDIF Audio

Некоторые звуковые устройства используют формат Digital Interconnect (S/PDIF) Sony/Philips для передачи звуковых данных. Теперь вы можете декодировать аудио в S/PDIF из аудио, закодированного с помощью кодека Windows Media Professional.
