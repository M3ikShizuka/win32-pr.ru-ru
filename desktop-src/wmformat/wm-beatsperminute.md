# <a name="wmbeatsperminute"></a>WM/Беатсперминуте

Атрибут **WM/беатсперминуте** содержит ритм содержимого в минуту.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмбеатсперминуте

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Значением атрибута **беатсперминуте** является целое число, хранящееся в виде строки.

### <a name="example"></a>Пример



| Тип файла | Пример значения |
|-----------|---------------|
| звук;     | "120"         |
