# <a name="reading-streams-with-non-square-pixels"></a>Чтение потоков с неквадратными пикселями

Приложения для чтения, которым необходимо правильно обменять неквадратные Пиксели, должны проверять атрибуты уровня потока в заголовке ASF и расширения модулей данных в каждом примере. Существует два варианта, когда выходной прямоугольник должен быть скорректирован для компенсации неквадратных пикселей.

Когда исходное содержимое было захвачено с устройства, например с помощью цифровой видеокамеры с неквадратными пикселями в его ККД, приложение-считыватель должно изменить его прямоугольник вывода, чтобы изображение правильно отображалось на мониторе компьютера с квадратными пикселями.

Когда приложение для чтения выводит видео, которое будет воспроизводиться на телевизоре NTSC, например через подключение S-Video на видеоадаптере, необходимо настроить прямоугольник видео таким образом, чтобы изображение правильно отображалось на неквадратных пикселях телевизора.
