# <a name="bannerimagedata"></a>баннеримажедата

Атрибут **баннеримажедата** содержит изображение баннера для фирменной символики. Тип изображения, который содержит этот атрибут, описан атрибутом [**баннеримажетипе**](bannerimagetype.md) .

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмбаннеримажедата

## <a name="data-type"></a>Тип данных

**\_ \_ двоичный тип ВМТ**

## <a name="remarks"></a>Комментарии

Изображение, хранящееся в этом атрибуте, может иметь формат точечного рисунка, JPEG или GIF.
