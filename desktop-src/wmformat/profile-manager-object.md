# <a name="profile-manager-object"></a>Объект диспетчера профилей

Профиль — это набор параметров носителя, используемых для создания файла ASF. Объект диспетчера профилей создает объекты профиля для редактирования. Объекты профиля могут создаваться без каких бы то ни было данных или построены из существующих данных профиля. Объект диспетчера профилей также предоставляет методы для перечисления поддерживаемых кодеков и запроса информации для этих кодеков.

Объект диспетчера профилей создается функцией [**вмкреатепрофилеманажер**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-wmcreateprofilemanager) , которая устанавливает указатель на интерфейс [**ивмпрофилеманажер**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmprofilemanager) . Другие интерфейсы объекта диспетчера профилей можно получить, вызвав метод **QueryInterface** .

Объект диспетчера профилей поддерживает следующие интерфейсы.



| Интерфейс                                                      | Описание                                                                                                                                  |
|----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------|
| [**ивмкодеЦинфо**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmcodecinfo)                           | Извлекает сведения о поддерживаемых кодеках и их форматах.                                                                              |
| [**IWMCodecInfo2**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmcodecinfo2)                         | Извлекает имена поддерживаемых кодеков и описания их форматов. Наследует все методы **ивмкодеЦинфо**.          |
| [**IWMCodecInfo3**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmcodecinfo3)                         | Извлекает свойства кодека и запросы для поддерживаемых функций. Наследует все методы **ивмкодеЦинфо** и **IWMCodecInfo2**. |
| [**ивмпрофилеманажер**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmprofilemanager)                 | Создает новые профили, загружает существующие профили и сохраняет пользовательские профили.                                                                    |
| [**IWMProfileManager2**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmprofilemanager2)               | Управляет версией системных профилей, перечисленных диспетчером профилей. Наследует все методы **ивмпрофилеманажер**.             |
| [**ивмпрофилеманажерлангуаже**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmprofilemanagerlanguage) | Управляет языком системных профилей, проанализированных диспетчером профилей.                                                                  |



 

## <a name="remarks"></a>Remarks

При создании объекта диспетчера профилей он анализирует все системные профили, что может занять несколько секунд. Создание и выпуск диспетчера профилей каждый раз, когда потребуется его использовать, может негативно сказаться на производительности. Вы должны создать диспетчер профилей один раз в своем приложении и освободить его только тогда, когда вам больше не понадобится его использовать.
