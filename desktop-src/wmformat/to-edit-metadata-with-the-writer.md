# <a name="to-edit-metadata-with-the-writer"></a>Изменение метаданных с помощью модуля записи

Вы можете обращаться непосредственно к модулю записи, а также к метаданным, которые будут переключаться в заголовок файла. Вызовите метод **QueryInterface** любого интерфейса в объекте модуля записи, чтобы получить указатель на интерфейс [**ивмхеадеринфо**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmheaderinfo) или [**IWMHeaderInfo2**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmheaderinfo2) . После создания указателя на соответствующий интерфейс можно манипулировать метаданными точно так же, как при загрузке файла в объект редактора метаданных. Дополнительные сведения об изменении метаданных см. [в разделе Работа с метаданными](working-with-metadata.md).

Перед вызовом [**ивмвритер:: бегинвритинг**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmwriter-beginwriting)необходимо внести все изменения в метаданные.

> [!Note]  
> Если задать метаданные для файла, записать файл, а затем подготовиться к записи нового файла без освобождения модуля записи, то некоторые метаданные, заданные для первого файла, останутся установленными и будут включаться в последующие файлы. При записи нескольких файлов с одним и тем же экземпляром объекта модуля записи у вас есть два варианта: проверять все метаданные перед записью каждого файла или записывать только метаданные модуля записи, которые применяются ко всем записываемым файлам.
