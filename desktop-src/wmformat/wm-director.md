# <a name="wmdirector"></a>WM/Director

Атрибут **WM/Director** содержит имя режиссера.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдиректор

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Обычно видеофайлы имеют директоры, звуковые файлы, скорее всего, используют [**WM/Producer**](wm-producer.md).
