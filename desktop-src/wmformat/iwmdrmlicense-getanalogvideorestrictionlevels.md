# <a name="iwmdrmlicensegetanalogvideorestrictionlevels-method"></a>Метод Ивмдрмлиценсе:: Жетаналогвидеорестриктионлевелс

Метод **жетаналогвидеорестриктионлевелс** извлекает все аналоговые видеоограничения, установленные для текущей лицензии.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetAnalogVideoRestrictionLevels(
  [out]     WMDRM_ANALOG_VIDEO_RESTRICTIONS rgAnalogVideoRestrictions[],
  [in, out] DWORD                           *pcRestrictions
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*\[ рганалогвидеорестриктионс \]* \[исходящий трафик\]
</dt> <dd>

Получает массив структур [**\_ ограничений аналогового \_ видео \_ WMDRM**](wmdrm-analog-video-restrictions.md) . Задайте значение **null** , чтобы получить количество элементов в массиве в *пкресктриктионс*.

</dd> <dt>

*пкрестриктионс* \[ в, out\]
</dt> <dd>

Число элементов в массиве ограничений. Если *рганалогвидеорестриктионс* имеет значение **null**, для этого параметра задается количество элементов, необходимых в массиве.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |



 

## <a name="remarks"></a>Remarks

Необходимо выделить память для массива ограничений. Чтобы сделать это, сначала вызовите метод с параметром *рганалогвидеорестриктионс* , которому присвоено значение **null**, которое задаст в параметре *пкрестриктионс* число обязательных элементов.
