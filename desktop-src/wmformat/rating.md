# <a name="rating"></a>Рейтинг

Атрибут рейтинга содержит рейтинг содержимого.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмратинг

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Это атрибут уровня файла.

Для отдельных реализаций в существующих файлах используются разные масштабы и нотации. Поэтому следует избегать использования этого атрибута. стандартизация для разработки единообразных пользовательских интерфейсов отсутствует.
