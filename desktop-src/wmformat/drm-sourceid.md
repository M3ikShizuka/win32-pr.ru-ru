# <a name="drm_sourceid"></a>Идентификатор \_ SOURCEID DRM

Атрибут **\_ SourceID DRM** содержит идентификатор, связывающий защищенное содержимое с источником.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдрм \_ SourceID

## <a name="data-type"></a>Тип данных

**ВМТ, \_ тип \_ DWORD**

## <a name="remarks"></a>Remarks

идентификаторы источников используются для различения содержимого, защищенного нестандартным способом, от содержимого, защищенного с помощью стандартного Windows Media DRM.
