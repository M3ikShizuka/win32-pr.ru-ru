# <a name="image-streams"></a>Потоки изображений

Поток изображений — это специальный тип потока, который содержит изображения, назначенные времени представления. Приложение может отображать изображения в потоке изображений по желанию, но Типичная реализация — отображение каждого изображения до тех пор, пока не будет доставлено следующее изображение, например слайд-шоу. Как правило, поток изображений кодируется в файл с аудио-потоком для создания простого представления изображений, синхронизированных с музыкой или речью.

Потоки изображений подобны видеопотокам в том, что они создаются из несжатых образцов, сжатых как часть процесса записи файлов, но они также подобны произвольным потокам, так как необходимо назначить для содержимого скорость потока и окно буфера, не полагаясь на свойства, назначенные кодеком.
