# <a name="copying-data-from-one-file-to-another"></a>Копирование данных из одного файла в другой

На самом базовом уровне копирование потока из одного файла ASF в другой довольно просто. Однако существуют проблемы, которые следует учитывать при работе с потоками из нескольких входных файлов или при копировании потоков, которые были первоначально распакованы и повторно кодируются.

В следующих разделах описывается копирование потоков.



| Section                                                                                              | дескрипитон                                                                                    |
|------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------|
| [копирование Потоки без распаковки данных](copying-streams-without-decompressing-the-data.md) | Описывает копирование потоков с помощью сжатых образцов для сохранения качества содержимого. |
| [копирование Потоки с использованием распакованных примеров](copying-streams-using-decompressed-samples.md)         | Описывает проблемы при копировании потоков с помощью распакованных образцов.                      |
