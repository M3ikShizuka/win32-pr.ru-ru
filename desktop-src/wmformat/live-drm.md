# <a name="live-drm"></a>Live DRM

Начиная с серии Windows Media 9, пакет SDK для формата Windows Media можно использовать для записи файлов, защищенных с помощью DRM. Это позволяет использовать такие сценарии, как Live DRM, в которых защита DRM применяется к файлу или потоку при кодировании, а не на этапе обработки после завершения кодирования. Live DRM позволяет владельцам содержимого защищать прямые широковещательные трансляции, такие как концертов или спортивные мероприятия. Live DRM также представляет упрощенный или упрощенный способ создания защищенных файлов. Для применения любой версии защиты DRM можно использовать Live DRM. Защищенный файл, созданный с помощью Live DRM, точно такой же, как и файл, защищенный как этап последующей обработки.

**Примечание** . DRM не поддерживает версию этого пакета SDK на базе x64.
