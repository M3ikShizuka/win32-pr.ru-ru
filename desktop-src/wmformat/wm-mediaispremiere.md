# <a name="wmmediaispremiere"></a>WM/Медиаиспремиере

Атрибут **WM/медиаиспремиере** указывает, является ли содержимое первым из рядов.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиаиспремиере

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут используется для трансляции телепередач.
