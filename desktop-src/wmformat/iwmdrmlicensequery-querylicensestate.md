# <a name="iwmdrmlicensequeryquerylicensestate-method"></a>Метод Ивмдрмлиценсекуери:: Куерилиценсестате

Метод **куерилиценсестате** запрашивает в локальном хранилище лицензий сведения о лицензиях, которые применяются к идентификатору ключа для одного или нескольких конкретных прав.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT QueryLicenseState(
  [in]  BSTR                   bstrKID,
  [in]  DWORD                  cActionsToQuery,
  [in]  BSTR                   rgbstrActionsToQuery[],
  [out] DRM_LICENSE_STATE_DATA rgResultStateData[]
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*бстркид* \[ окне\]
</dt> <dd>

Идентификатор ключа, для которого запрашиваются запросы. Будут оцениваться только лицензии, применимые к этому ИДЕНТИФИКАТОРу ключа.

</dd> <dt>

*кактионстокуери* \[ окне\]
</dt> <dd>

Число действий, для которых запрашиваются запросы. Это значение должно быть равно количеству элементов в массивах, переданных для параметров *ргбстрактионстокуери* и *ргресултстатедата* .

</dd> <dt>

*\[ ргбстрактионстокуери \]* \[в\]
</dt> <dd>

Массив из одного или нескольких прав, для которых запрашиваются запросы. Этот массив должен содержать столько элементов, сколько указано параметром *кактионстокуери*. Каждому элементу должно быть присвоено одно из следующих констант.



| Константа                                        | Описание                                                                                                                                        |
|-------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| g \_ всзвмдрм \_ лиценсестате \_ BACKUP               | Включите, чтобы запросить сведения о правильном резервном копировании и восстановлении лицензии.                                                               |
| g \_ всзвмдрм \_ лиценсестате \_ коллаборативеплай    | Включите, чтобы запросить сведения о правильном доступе к содержимому с группой пользователей в рамках сценария совместного воспроизведения.          |
| g \_ всзвмдрм \_ лиценсестате \_ Copy                 | Включить в запрос, чтобы получить сведения о правильном копировании содержимого на внешние устройства или носитель.                                                 |
| g \_ всзвмдрм \_ лиценсестате \_ копитокд             | Включить в запрос, чтобы получить сведения о правильном копировании содержимого на компакт-диск.                                                                        |
| g \_ всзвмдрм \_ лиценсестате \_ копитононсдмидевице  | Включить в запрос, чтобы получить сведения о правильном копировании содержимого на устройство, которое не поддерживает инициативу по защите цифровых мультимедийных данных (SDMI). |
| g \_ всзвмдрм \_ лиценсестате \_ копитосдмидевице     | Включите, чтобы запросить сведения о правильном копировании содержимого на устройство, которое поддерживает SDMI.                                           |
| g \_ всзвмдрм \_ лиценсестате \_ креатесумбнаилимаже | Включить в запрос для получения подробных сведений о правильном создании эскиза из содержимого.                                                     |
| g \_ всзвмдрм \_ лиценсестате, \_ Воспроизведение             | Включить в запрос для получения подробных сведений о правом воспроизведения содержимого.                                                                              |
| g \_ всзвмдрм \_ лиценсестате \_ плайлистбурн         | Включить в запрос, чтобы получить сведения о правильном копировании содержимого на компакт-диск в составе списка воспроизведения.                                                  |



 

</dd> <dt>

*\[ ргресултстатедата \]* \[исходящий трафик\]
</dt> <dd>

Массив из одной или нескольких [**структур \_ \_ \_ данных состояния лицензии DRM**](drmdrm-license-state-data.md) , которые получают сведения о состоянии лицензии, которые относятся к правому в соответствующем элементе параметра *ргбстрактионстокуери* .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |



 

## <a name="remarks"></a>Remarks

Будет выполнен поиск и оценка всех лицензий, применимых к указанному ИДЕНТИФИКАТОРу ключа. Результаты суммируются, поэтому каждая структура **\_ \_ \_ данных состояния лицензии DRM** может содержать данные из нескольких лицензий.
