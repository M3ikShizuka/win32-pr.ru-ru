# <a name="using-system-profiles"></a>Использование системных профилей

Системные профили, включенные в пакет SDK для Windows Media Format, были разработаны для удовлетворения основных потребностей цифрового мультимедийного содержимого, состоящего из аудио и видео. Из-за широкого спектра доступных функций, а также улучшений в ситуации с цифровыми носителями, системные профили были удалены. Последний набор системных профилей использует кодеки Windows Media версии 8, которые были выпущены в Windows XP. Несмотря на то, что поддержка последних версий кодеков не поддерживается, системные профили делают отличную отправную точку для создания профилей в соответствии с вашими потребностями.
