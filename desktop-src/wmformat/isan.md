# <a name="isan"></a>исан

Атрибут **Исан** содержит номер Международной стандартной АУДИОВИЗУАЛЬНОГО (Исан), определяющий содержимое.

## <a name="global-constant"></a>Глобальная константа

g \_ всзисан

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

ИСАН — это Международный стандарт для идентификации аудиовизуального Works. Сведения о ИСАН см. на [веб-сайте](https://www.isan.org/)Standard.

Объект редактора метаданных проверяет строку ИСАН при задании этого атрибута. Допустимое форматирование строк, как описано в разделе 4,1 спецификации ИСАН, состоит из 16 шестнадцатеричных цифр, которые дополнительно разделяются пробелами или дефисами в 4-значные сегменты. Необходимо соблюдать форматированный номер, который также может быть отделен пробелом или дефисом, по допустимому символу Check. Символом проверки может быть десятичная цифра или заглавная буква. Сведения о том, как получить номер проверки, см. в приложении A из руководства пользователя ИСАН.

После строки стандартной ИСАН может следовать информация о версии. При наличии сведений о версии она состоит из восьми шестнадцатеричных цифр, за которыми следует номер чека. Форматирование этих символов следует тем же правилам, что и базовая строка.

## <a name="examples"></a>Примеры

Ниже приведены три приемлемых отформатированных строки для примера ИСАН.


```
00003BAB93520000G
0000 3BAB 9352 0000 G
0000-3BAB-9352-0000-G
```



Следующая строка показывает формат ИСАН с информацией о версии.


```C++
0000-3BAB-9352-0000-G-0000-0000-Q
```
