# <a name="drm_copy_opl-structure"></a>\_Структура копирования \_ ОПЛ DRM

Структура **\_ копирования \_ ОПЛ DRM** содержит сведения об уровнях защиты выходных данных (оплс), указанных в лицензии для действий копирования.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct DRM_COPY_OPL {
  WORD               wMinimumCopyLevel;
  DRM_OPL_OUTPUT_IDS oplIdIncludes;
  DRM_OPL_OUTPUT_IDS oplIdExcludes;
} ;
```
