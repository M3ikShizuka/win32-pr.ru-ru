# <a name="wmisrc"></a>WM/ИСРК

Атрибут **WM/ИСРК** содержит Международный стандартный код записи (ИСРК).

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмисрк

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

ИСРК используется в отрасли записи для задания записи. Он не зависит от МКДИ, который определяет компактный Дикс по содержанию.
