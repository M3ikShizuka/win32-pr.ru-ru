# <a name="drm_drmheader_subscriptioncontentid"></a>\_Дрмхеадер \_ субскриптионконтентид DRM

Атрибут **DRM \_ дрмхеадер \_ субскриптионконтентид** содержит идентификатор содержимого подписки.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдрм \_ дрмхеадер \_ субскриптионконтентид

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут имеется только в содержимом DRM версии 7. ИДЕНТИФИКАТОР содержимого подписки является необязательным и определяется исключительно создателем содержимого. Объект модуля записи не выполняет никаких действий с этим атрибутом. Его можно задать с помощью [**ивмдрмвритер:: сетдрматтрибуте**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) , и его можно получить при помощи [**Ивмдрмреадер:: жетдрмпроперти**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
