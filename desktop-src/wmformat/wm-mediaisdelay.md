# <a name="wmmediaisdelay"></a>WM/Медиаисделай

Атрибут **WM/медиаисделай** указывает, является ли содержимое широковещательной задержкой ленты.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиаисделай

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут используется для трансляции телепередач.
