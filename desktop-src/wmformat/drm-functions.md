# <a name="microsoft-windows-media-drm-client-functions"></a>Функции клиента DRM Microsoft Windows Media

В состав расширенных API клиента DRM Microsoft Windows Media реализуются следующие функции.



| Функция                                                             | Описание                                                                                                                                                                                     |
|----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**вмдрмкреатепровидер**](wmdrmcreateprovider.md)                   | Создает фабрику класса, которая может создавать другие объекты DRM. Эта функция не требует наличия библиотеки-заглушки от корпорации Майкрософт и создает объекты, которые не поддерживают защищенные функции DRM. |
| [**вмдрмкреатепротектедпровидер**](wmdrmcreateprotectedprovider.md) | Создает фабрику класса, которая может создавать другие объекты DRM. Эта функция требует наличия библиотеки-заглушки от корпорации Майкрософт и создает объекты, поддерживающие защищенные функции DRM.                |
| [**вмдрмшутдовн**](wmdrmshutdown.md)                               | Высвобождает ресурсы, используемые интерфейсами API.                                                                                                                                                            |
| [**вмдрмстартуп**](wmdrmstartup.md)                                 | Инициализирует ресурсы, используемые интерфейсами API.                                                                                                                                                         |
