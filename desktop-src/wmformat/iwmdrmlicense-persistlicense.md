# <a name="iwmdrmlicensepersistlicense-method"></a>Ивмдрмлиценсе: метод:P Ерсистлиценсе

Метод **персистлиценсе** сохраняет текущую лицензию из временного хранилища в постоянное локальное хранилище лицензий.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT PersistLicense();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |



 

## <a name="remarks"></a>Комментарии

Если текущая лицензия не находится во временном хранилище, этот метод завершится ошибкой.

Можно вызвать [**канперсист**](iwmdrmlicense-canpersist.md) , чтобы запросить, разрешено ли сохранение лицензии.
