# <a name="ireferenceclock-interface"></a>Интерфейс Иреференцеклокк

Интерфейс **иреференцеклокк** предоставляет доступ к внешним часам. Этот интерфейс обеспечивает синхронизацию всех подпрограмм подготовки к одному и тому же часам.

Этот интерфейс можно получить из объекта модуля чтения.

## <a name="members"></a>Элементы

Интерфейс **иреференцеклокк** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Иреференцеклокк** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **иреференцеклокк** содержит следующие методы.



| Метод                                                   | Описание                                                               |
|:---------------------------------------------------------|:--------------------------------------------------------------------------|
| [**адвисепериодик**](ireferenceclock-adviseperiodic.md) | Не реализовано в этом пакете SDK.<br/>                                   |
| [**адвисетиме**](ireferenceclock-advisetime.md)         | Запрашивает асинхронное уведомление об истечении времени.<br/> |
| [**GetTime**](ireferenceclock-gettime.md)               | Извлекает текущее время ссылки.<br/>                          |
| [**Unadvise**](ireferenceclock-unadvise.md)             | Отменяет запрос на уведомление.<br/>                                |
