# <a name="drm_rights"></a>\_Права DRM

Свойство **\_ Rights DRM** определяет права, которые будут необходимы приложению при следующей попытке открыть защищенный файл.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдрм \_ права

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Это свойство, доступное для чтения и записи, которое извлекается с помощью [**ивмдрмреадер:: жетдрмпроперти**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty) и задается с помощью [**Ивмдрмреадер:: сетдрмпроперти**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-setdrmproperty) или [**ивмдрмвритер:: сетдрматтрибуте**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute).
