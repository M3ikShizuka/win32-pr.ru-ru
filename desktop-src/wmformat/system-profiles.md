# <a name="system-profiles"></a>Системные профили

В следующей таблице содержится полный список поддерживаемых системных профилей. Каждый из перечисленных профилей имеет имя и идентификатор профиля. Идентификатор профиля — это константа, заданная в качестве значения GUID, назначенного системному профилю. Чтобы использовать идентификаторы системного профиля в коде, необходимо включить вмсиспрф. h в приложение. Примеры, демонстрирующие загрузку системного профиля, см. [в разделе Загрузка системного профиля](to-load-a-system-profile.md).

> [!IMPORTANT]
> Перечисленные ниже профили используют кодеки Windows Media Audio и Windows Media Video. Стандартные системные профили, которые используют кодеки Windows Media 9 Series, отсутствуют. Вы можете создать собственный профиль серии Windows Media 9 с помощью профиля версии 8 в качестве отправной точки. Дополнительные сведения см. в разделе [повторное использование конфигураций потоков](reusing-stream-configurations.md).
