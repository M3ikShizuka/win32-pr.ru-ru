# <a name="wmmediaisrepeat"></a>WM/Медиаисрепеат

Атрибут **WM/медиаисрепеат** указывает, является ли содержимое повторяющимся эпизодом ряда.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиаисрепеат

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут используется для трансляции телепередач.
