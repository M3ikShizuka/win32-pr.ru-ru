# <a name="wmmediastationname"></a>WM/Медиастатионнаме

Атрибут **WM/медиастатионнаме** содержит имя станции, которая передает содержимое.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиастатионнаме

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут используется для трансляции телепередач.
