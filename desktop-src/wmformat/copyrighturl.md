# <a name="copyrighturl"></a>копиригхтурл

Атрибут **копиригхтурл** содержит веб-адрес страницы авторского права, связанной с фирменной символикой.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмкопиригхтурл

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Этот атрибут применяется ко всему файлу.
