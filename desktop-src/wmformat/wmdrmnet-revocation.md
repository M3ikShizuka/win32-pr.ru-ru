# <a name="wmdrmnet_revocation"></a>\_Отзыв вмдрмнет

Свойство **\_ отзыва вмдрмнет** извлекает список отзыва для Windows Media DRM 10 для сетевых устройств.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдрмнет, \_ отзыв

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Список отзыва содержит сертификаты отозванных устройств.
