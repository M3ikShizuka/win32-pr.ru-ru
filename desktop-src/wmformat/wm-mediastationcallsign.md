# <a name="wmmediastationcallsign"></a>WM/Медиастатионкаллсигн

Атрибут **WM/медиастатионкаллсигн** содержит знак вызова станции, который передает содержимое.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиастатионкаллсигн

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут используется для трансляции телепередач и радио.
