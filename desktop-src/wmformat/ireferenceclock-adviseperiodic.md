# <a name="ireferenceclockadviseperiodic-method"></a>Метод Иреференцеклокк:: Адвисепериодик

Этот метод не реализован.

в других реализациях интерфейса **иреференцеклокк** , например DirectShow в компоненте® Microsoft® DirectX®, метод **адвисепериодик** используется для создания периодического запроса рекомендаций, который будет сообщать об указанном объекте события каждый раз, когда истечет указанный интервал времени. Этот пакет SDK не реализует функциональность этого метода, и все вызовы, выполненные в нем, будут возвращать значение E \_ нотимпл.

## <a name="syntax"></a>Синтаксис


```C++
 AdvisePeriodic();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.
