# <a name="iwmdrmlicensemanagementdeletelicense-method"></a>Ивмдрмлиценсеманажемент: метод:D Елетелиценсе

Метод **делетелиценсе** удаляет лицензию из временного локального хранилища лицензий.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT DeleteLicense(
  [in] BSTR  bstrKID,
  [in] DWORD dwFlags
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*бстркид* \[ окне\]
</dt> <dd>

Идентификатор ключа (Детская) удаляемой лицензии.

</dd> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Флаги параметра удаления лицензии. Задайте одно из значений, приведенных в следующей таблице.



| Значение                                    | Описание                                                                                                                                                                                           |
|------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_Лицензия на удаление WMDRM \_ \_ немедленно      | Указывает, что лицензия должна быть немедленно удалена из хранилища.                                                                                                                              |
| \_ \_ лицензионный маркер удаления WMDRM \_ \_ для \_ очистки | Указывает, что лицензия должна быть помечена для удаления, но не должна удаляться из хранилища до тех пор, пока не будет вызван метод [**клеанлиценсесторе**](iwmdrmlicensemanagement-cleanlicensestore.md) . |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                                            | Описание                                                                                                         |
|--------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                   | Метод выполнен успешно.<br/>                                                                                    |
| <dl> <dt>**DRM \_ E \_ лиценсенотфаунд**</dt> </dl> | Указанная лицензия не существует в хранилище.<br/> ИЛИ<br/> Хранилище не найдено.<br/> |



 

## <a name="remarks"></a>Комментарии

Чтобы удалить лицензии из постоянного локального хранилища лицензий, необходимо использовать отзыв лицензий.
