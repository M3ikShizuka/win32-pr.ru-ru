# <a name="drm_baselicenseacqurl"></a>\_БАСЕЛИЦЕНСЕАККУРЛ DRM

Атрибут **\_ баселиценсеаккурл DRM** содержит частичный базовый URL-адрес, по которому приложение проигрывателя должно получить лицензию на содержимое.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдрм \_ баселиценсеаккурл

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Это необязательное свойство для чтения и записи, которое задается и извлекается с помощью [**ивмдрмреадер:: жетдрмпроперти**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty). Он предоставляется, чтобы разрешить системам распространения лицензий использовать относительные пути в свойствах URL-адреса для получения лицензий.

После установки этого свойства все URL-адреса частичного приобретения лицензий в заголовках содержимого будут содержать базовый URL-адрес, добавляемый в начало частичного URL-адреса, чтобы сформировать полный URL-адрес для перехода к приложению проигрывателя. Вызов **ивмдрмреадер:: жетдрмпроперти** для **DRM \_ баселиценсеаккурл** будет работать только в том же сеансе, в котором она была задана. Свойство не хранится в заголовке содержимого; Он является динамическим, и в содержимом сохраняется только относительный URL-адрес.
