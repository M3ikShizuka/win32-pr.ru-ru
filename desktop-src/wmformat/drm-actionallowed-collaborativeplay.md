# <a name="drm_actionallowed_collaborativeplay"></a>\_Актионалловед \_ коллаборативеплай DRM

Атрибут **DRM \_ актионалловед \_ коллаборативеплай** указывает, разрешает ли лицензия возможность совместного воспроизведения содержимого.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдрм \_ актионалловед \_ коллаборативеплай

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Право совместного воспроизведения применяется для воспроизведения защищенного содержимого в рамках сеанса совместной работы с помощью одноранговых служб. Например, пользователи MSN Messenger 2004 могут воспроизводить защищенное содержимое в сеансе Мусикмикс. Это право можно использовать только для того, чтобы защищенный файл участвовал в одном сеансе одновременно, и все пользователи, участвующие в сеансе, должны быть в сети для отображения содержимого.

Это свойство только для чтения, которое извлекается с помощью [**ивмдрмреадер:: жетдрмпроперти**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
