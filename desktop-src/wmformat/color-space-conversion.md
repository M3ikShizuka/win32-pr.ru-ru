# <a name="color-space-conversion"></a>Преобразование цветового пространства

Часто существует разница между глубиной цвета сжатого видеоформата в профиле и входным форматом. В этом случае исходное видео должно быть преобразовано, чтобы соответствовать цветовому пространству назначения. Модуль записи обрабатывает этот процесс, с внутренним преобразователем цветового пространства.

Читатель также взаимодействует с преобразователем цветового пространства для согласования любых различий между сжатым форматом и выходным форматом.

Как и при выполнении всех преобразований данных, преобразование между глубиной цвета может уменьшить качество выходных данных. По возможности следует использовать форматы входных и выходных данных с той же глубиной цвета, что и в сжатом формате.
