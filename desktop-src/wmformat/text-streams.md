# <a name="text-streams"></a>Текстовые потоки

Текстовые потоки содержат простой текст, хранящийся в примерах. Приложение может использовать текст из текстового потока любым выбранным способом. Например, может потребоваться создать приложение для учебного языка, которое вводит слова на иностранном языке. Каждое слово можно включить как образец текстового потока в режиме презентации, который соответствует произнесенному слову в звуковом потоке. Затем приложение может отобразить текст при доставке образца.
