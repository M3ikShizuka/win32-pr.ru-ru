# <a name="drm_v1licenseacqurl"></a>\_V1LICENSEACQURL DRM

Атрибут **\_ V1LicenseAcqURL DRM** содержит адрес веб-сайта, на котором пользователь может получить лицензию DRM версии 1.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдрм \_ V1LicenseAcqURL

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут может быть задан с помощью [**ивмдрмвритер:: сетдрматтрибуте**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) , и его можно получить при помощи [**Ивмдрмреадер:: жетдрмпроперти**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
