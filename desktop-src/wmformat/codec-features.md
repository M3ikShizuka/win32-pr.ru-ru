# <a name="codec-features"></a>Функции кодеков

пакет SDK для Windows Media Format поставляется с несколькими аудиокодеками аудио и видео. Вы можете использовать кодеки, предоставляемые для сжатия и распаковки содержимого в соответствии с различными потребностями. Кодек, используемый модулем записи для сжатия данных, задается сведениями о конфигурации потока в профиле. Сведения из профиля сохраняются в заголовке файла, созданного модулем записи. Затем, когда файл открывается модулем чтения или синхронным модулем чтения, сведения о профиле в заголовке определяют кодек, необходимый для распаковки данных.
