# <a name="protected-files"></a>Защищенные файлы

если файл защищен с помощью Windows Media DRM, раздел данных файла шифруется. Процесс защиты файла также называется упаковкой файла. расширенные api клиента DRM для Windows мультимедиа не позволяют упаковывать файлы. вы можете упаковать файлы с помощью объектов пакета sdk Windows media Rights Manager или пакета sdk Windows media Format.

Пользователь не может получить доступ к данным в защищенном файле без предварительного получения лицензии на этот файл.
