# <a name="protected-files"></a>Защищенные файлы

Если файл защищен с помощью Windows Media DRM, раздел данных файла шифруется. Процесс защиты файла также называется упаковкой файла. Расширенные API клиента DRM Windows Media не позволяют упаковывать файлы. Вы можете упаковать файлы с помощью объектов пакета SDK Windows Media Rights Manager или пакета SDK Windows Media Format.

Пользователь не может получить доступ к данным в защищенном файле без предварительного получения лицензии на этот файл.
