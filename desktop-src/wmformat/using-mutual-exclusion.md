# <a name="using-mutual-exclusion"></a>Использование взаимного исключения

Можно использовать взаимоисключающие объекты для указания групп потоков, только один из которых будет доставляться за раз. Взаимное исключение можно использовать для работы с различными сценариями — как собственными, так и пользовательскими.

В следующих разделах описывается работа с различными типами взаимного исключения.



| Section                                                                                  | Описание                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [Использование множественного взаимного исключения скорости](using-multiple-bit-rate-mutual-exclusion.md) | Описывает, как использовать взаимное исключение для кодирования одного входа в несколько потоков с разными настройками скорости. |
| [Использование настраиваемых типов взаимного исключения](using-custom-mutual-exclusion-types.md)           | Обсуждается пользовательское взаимное исключение.                                                                                |
