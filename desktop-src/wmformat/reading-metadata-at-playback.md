# <a name="reading-metadata-at-playback"></a>Считывание метаданных при воспроизведении

И асинхронный, и синхронный объекты чтения могут считывать метаданные из заголовка загруженного ASF-файла. При чтении файлов атрибуты метаданных доступны только для чтения. Оба объекта читатель могут быть запрошены для интерфейсов [**ивмхеадеринфо**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmheaderinfo) и [**IWMHeaderInfo2**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmheaderinfo2) .

Дополнительные сведения о доступе к метаданным см. [в разделе Работа с метаданными](working-with-metadata.md).
