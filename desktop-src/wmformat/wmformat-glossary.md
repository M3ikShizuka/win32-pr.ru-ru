# <a name="wmformat-glossary"></a>Глоссарий Вмформат

Чтобы найти термин в глоссарии, щелкните букву алфавита, который является первой буквой в термине, который требуется найти.

Вы также можете читать термины глоссария в тексте справки, щелкнув ссылки термина "подчеркнутый Глоссарий". После щелчка ссылки глоссария термин и определение отображаются во всплывающем окне. Чтобы закрыть окно, щелкните в любом месте экрана.

<dl> <dt>

<span id="wmformat.bandwidth"></span><span id="WMFORMAT.BANDWIDTH"></span>**пропускная способность**
</dt> <dd>

Емкость передачи данных для носителя.

</dd> <dt>

<span id="wmformat.bit_rate"></span><span id="WMFORMAT.BIT_RATE"></span>**скорость потока**
</dt> <dd>

Число передаваемых битов в единицу времени, обычно выраженное в битах в секунду.

</dd> <dt>

<span id="wmformat.broadband"></span><span id="WMFORMAT.BROADBAND"></span>**широкополосный** доступ
</dt> <dd>

Носитель передачи, предназначенный для высокоскоростной передачи данных на больших расстояниях. Примерами широкополосных сетей являются службы кабельных модемов и DSL.

</dd> <dt>

<span id="wmformat.cleanpoint"></span><span id="WMFORMAT.CLEANPOINT"></span>**клеанпоинт**
</dt> <dd>

Сжатый пример, который не зависит от других примеров для распаковки. Допускается только поиск в клеанпоинт.

</dd> <dt>

<span id="wmformat.clear_content"></span><span id="WMFORMAT.CLEAR_CONTENT"></span>**очистить содержимое**
</dt> <dd>

Windows незашифрованный файл мультимедиа.

</dd> <dt>

<span id="wmformat.codec"></span><span id="WMFORMAT.CODEC"></span>**кодек**
</dt> <dd>

Аббревиатура для сжатия и декомпрессора. Программное или аппаратное обеспечение, используемое для сжатия и распаковки цифровых носителей.

</dd> <dt>

<span id="wmformat.content"></span><span id="WMFORMAT.CONTENT"></span>**содержимое**
</dt> <dd>

Аудио, видео, изображения, текст или любая другая информация, содержащаяся в цифровом файле мультимедиа или потоке.

</dd> <dt>

<span id="wmformat.content_header"></span><span id="WMFORMAT.CONTENT_HEADER"></span>**заголовок содержимого**
</dt> <dd>

часть файловой структуры файла мультимедиа Windows, которая содержит сведения, необходимые для распаковки и отображения данных содержимого на клиентском компьютере. В упакованном файле существует дополнительный заголовок содержимого, содержащий идентификатор ключа, идентификатор содержимого и URL-адрес для получения лицензии. Этот заголовок содержимого может также включать обязательный номер версии и атрибуты, определенные поставщиком содержимого.

</dd> <dt>

<span id="wmformat.content_owner"></span><span id="WMFORMAT.CONTENT_OWNER"></span>**владелец содержимого**
</dt> <dd>

пользователь или организация, которые создают Windows файлы мультимедиа, например метку записи, movie studio или исполнитель.

</dd> <dt>

<span id="wmformat.cookie"></span><span id="WMFORMAT.COOKIE"></span>**файл cookie**
</dt> <dd>

Небольшой файл данных, хранящийся на локальном компьютере пользователя и содержащий сведения о пользователе, который имеет отношение к веб-сайту, например предпочтения пользователя.

</dd> <dt>

<span id="wmformat.custom_profile"></span><span id="WMFORMAT.CUSTOM_PROFILE"></span>**Настраиваемый профиль**
</dt> <dd>

Группа параметров, определяемая конечным пользователем для настройки выходных данных кодировщика.

</dd> <dt>

<span id="wmformat.decrypt"></span><span id="WMFORMAT.DECRYPT"></span>**расшифровка**
</dt> <dd>

Для преобразования зашифрованного содержимого обратно в исходную форму.

</dd> <dt>

<span id="wmformat.deinterlace"></span><span id="WMFORMAT.DEINTERLACE"></span>**разчередование**
</dt> <dd>

Чтобы объединить поля с чередованием в видеокадре таким образом, чтобы во время воспроизведения линии видеокадра нарисованы последовательно.

</dd> <dt>

<span id="wmformat.delta_frame"></span><span id="WMFORMAT.DELTA_FRAME"></span>**разностный кадр**
</dt> <dd>

Видеокадр, который содержит только изменения из предыдущего кадра. В отличие от этого, ключевой кадр содержит все данные, необходимые для создания этого кадра.

</dd> <dt>

<span id="wmformat.digital_rights_management__drm_"></span><span id="WMFORMAT.DIGITAL_RIGHTS_MANAGEMENT__DRM_"></span>**Управление цифровыми правами (DRM)**
</dt> <dd>

Любая технология, используемая для защиты от интересов владельцев цифрового содержимого и служб (например, владельцев авторских прав). Как правило, полномочные получатели или пользователи должны получить лицензию для использования содержимого.

</dd> <dt>

<span id="wmformat.digital_rights_management__drm__component"></span><span id="WMFORMAT.DIGITAL_RIGHTS_MANAGEMENT__DRM__COMPONENT"></span>**компонент управления цифровыми правами (DRM)**
</dt> <dd>

Компонент проигрывателя, который обрабатывает все функции управления цифровыми правами, такие как расшифровка упакованных файлов или инициирование приобретения лицензий.

</dd> <dt>

<span id="wmformat.directx_media_object__dmo_"></span><span id="WMFORMAT.DIRECTX_MEDIA_OBJECT__DMO_"></span>**Объект мультимедиа DirectX (DMO)**
</dt> <dd>

COM-объект, обрабатывающий потоки мультимедийных данных из буферов, выделенных клиентом.

</dd> <dt>

<span id="wmformat.dmo"></span><span id="WMFORMAT.DMO"></span>**DMO**
</dt> <dd>

См. определение для: [ *объект мультимедиа DirectX (DMO)*](/windows)

</dd> <dt>

<span id="wmformat.drm"></span><span id="WMFORMAT.DRM"></span>**DRM**
</dt> <dd>

См. определение для: [ *Управление цифровыми правами (DRM)* .](/windows)

</dd> <dt>

<span id="wmformat.encode"></span><span id="WMFORMAT.ENCODE"></span>**кодирование**
</dt> <dd>

Для преобразования аудио-и видеосодержимого в указанный цифровой формат, обычно включающий технологию сжатия или технологию шифрования.

</dd> <dt>

<span id="wmformat.encrypt"></span><span id="WMFORMAT.ENCRYPT"></span>**Шифрование**
</dt> <dd>

Для программной маскировки содержимого с целью скрытия его вещества.

</dd> <dt>

<span id="wmformat.frame"></span><span id="WMFORMAT.FRAME"></span>**кадр**
</dt> <dd>

Одно из нескольких последовательных изображений, образующих видео.

</dd> <dt>

<span id="wmformat.frame_rate"></span><span id="WMFORMAT.FRAME_RATE"></span>**Частота кадров**
</dt> <dd>

Число кадров видео, отображаемых в секунду. Более высокие частоты кадров обычно создают более плавные движения на изображении.

</dd> <dt>

<span id="wmformat.id3"></span><span id="WMFORMAT.ID3"></span>**ID3**
</dt> <dd>

Стандарт для добавления метаданных в MP3-файлы.

</dd> <dt>

<span id="wmformat.index"></span><span id="WMFORMAT.INDEX"></span>**индекс**
</dt> <dd>

маркер, созданный Windows пакетом средств разработки программного обеспечения (SDK) для поиска в файле видео.

</dd> <dt>

<span id="wmformat.individualization"></span><span id="WMFORMAT.INDIVIDUALIZATION"></span>**Индивидуальная**
</dt> <dd>

Процесс создания компонента управления цифровыми правами (DRM) в уникальном проигрывателе потребителя. Этот процесс повышает безопасность, усложняя повреждение более чем одного игрока. Этот процесс также называется получением обновления системы безопасности.

</dd> <dt>

<span id="wmformat.interlace"></span><span id="WMFORMAT.INTERLACE"></span>**чересстрочную развертку**
</dt> <dd>

Отображение видеокадра в двух полях. Одно поле содержит четные строки рамки, другое поле содержит нечетные линии. Во время воспроизведения строки в одном поле отображаются первыми, а затем отображаются строки во втором поле.

</dd> <dt>

<span id="wmformat.inverse_telecine"></span><span id="WMFORMAT.INVERSE_TELECINE"></span>**обратная чересстрочности**
</dt> <dd>

Процесс, удаляющий кадры, которые были добавлены при преобразовании пленки с 24 кадрами/с, в видео с 30 кадрами/с.

</dd> <dt>

<span id="wmformat.key"></span><span id="WMFORMAT.KEY"></span>**ключ**
</dt> <dd>

фрагмент данных, необходимый для разблокировки упакованного Windows файла мультимедиа. Этот ключ включен в отдельную лицензию.

</dd> <dt>

<span id="wmformat.key_frame"></span><span id="WMFORMAT.KEY_FRAME"></span>**ключевой кадр**
</dt> <dd>

Видеокадр, содержащий все данные, необходимые для создания изображения без ссылки на предыдущие фреймы.

</dd> <dt>

<span id="wmformat.key_id"></span><span id="WMFORMAT.KEY_ID"></span>**идентификатор ключа**
</dt> <dd>

значение, идентифицирующее ключ для защищенного Windowsного файла мультимедиа.

</dd> <dt>

<span id="wmformat.license"></span><span id="WMFORMAT.LICENSE"></span>**Лицензия**
</dt> <dd>

Данные, присоединенные к защищенному содержимому, описывающие способ использования содержимого.

</dd> <dt>

<span id="wmformat.license_acquisition"></span><span id="WMFORMAT.LICENSE_ACQUISITION"></span>**приобретение лицензий**
</dt> <dd>

процесс получения лицензии на воспроизведение упакованного Windows файла мультимедиа. проигрыватель пытается получить лицензию с URL-адреса получения лицензии, который указан в Windowsном файле мультимедиа.

</dd> <dt>

<span id="wmformat.license_management_service"></span><span id="WMFORMAT.LICENSE_MANAGEMENT_SERVICE"></span>**Служба управления лицензиями**
</dt> <dd>

служба майкрософт, которая выполняет процесс управления лицензиями в Windows пакете sdk для диспетчера прав на носитель.

</dd> <dt>

<span id="wmformat.license_revocation"></span><span id="WMFORMAT.LICENSE_REVOCATION"></span>**отзыв лицензий**
</dt> <dd>

Процесс удаления лицензий с компьютера пользователя.

</dd> <dt>

<span id="wmformat.license_revocation_acknowledgement"></span><span id="WMFORMAT.LICENSE_REVOCATION_ACKNOWLEDGEMENT"></span>**Подтверждение отзыва лицензии**
</dt> <dd>

Подписанное подтверждение от клиентского компьютера на сервер лицензирования, указывающий на то, что лицензии были отозваны.

</dd> <dt>

<span id="wmformat.license_revocation_challenge"></span><span id="WMFORMAT.LICENSE_REVOCATION_CHALLENGE"></span>**запрос отзыва лицензии**
</dt> <dd>

Запрос от клиентского компьютера к серверу лицензирования для отзыва лицензий.

</dd> <dt>

<span id="wmformat.license_revocation_response"></span><span id="WMFORMAT.LICENSE_REVOCATION_RESPONSE"></span>**ответ на отзыв лицензии**
</dt> <dd>

Ответ на запрос отзыва лицензии, указывающий, какие лицензии необходимо удалить с компьютера пользователя.

</dd> <dt>

<span id="wmformat.licensing_server"></span><span id="WMFORMAT.LICENSING_SERVER"></span>**сервер лицензирования**
</dt> <dd>

компьютер, на котором выполняется Windows службы лицензий носителей и выдает лицензии.

</dd> <dt>

<span id="wmformat.marker"></span><span id="WMFORMAT.MARKER"></span>**маркер**
</dt> <dd>

текстовая строка, связанная с назначенным временем в Windows содержимом на основе носителя. Маркеры часто обозначают удобные точки для начала воспроизведения, например начало новой сцены.

</dd> <dt>

<span id="wmformat.mbr"></span><span id="WMFORMAT.MBR"></span>**MBR-диск**
</dt> <dd>

См. определение для: [ *несколько битовых частот (MBR)*](/windows)

</dd> <dt>

<span id="wmformat.mime"></span><span id="WMFORMAT.MIME"></span>**MIME**
</dt> <dd>

См. определение для: [ *многоцелевое расширение электронной почты в Интернете (MIME)*](/windows)

</dd> <dt>

<span id="wmformat.multiple_bit_rate__mbr_"></span><span id="WMFORMAT.MULTIPLE_BIT_RATE__MBR_"></span>**несколько битов (MBR)**
</dt> <dd>

Характеристика потока данных, в которой одно и то же содержимое кодируется в несколько различных битовых ставок для оптимизации доставки содержимого.

</dd> <dt>

<span id="wmformat.multipurpose_internet_mail_extension__mime_"></span><span id="WMFORMAT.MULTIPURPOSE_INTERNET_MAIL_EXTENSION__MIME_"></span>**Многоцелевое расширение электронной почты в Интернете (MIME)**
</dt> <dd>

Стандарт, расширяющий протокол SMTP для кодирования файлов данных, не использующих ASCII, таких как видео, звуковые и двоичные файлы, для вложения в Интернет-почту.

</dd> <dt>

<span id="wmformat.output_protection_level"></span><span id="WMFORMAT.OUTPUT_PROTECTION_LEVEL"></span>**уровень защиты выходных данных**
</dt> <dd>

Параметр в лицензии, указывающий, какие технологии можно использовать для воспроизведения или копирования защищенного цифрового содержимого мультимедиа.

</dd> <dt>

<span id="wmformat.packaged_windows_media_file"></span><span id="WMFORMAT.PACKAGED_WINDOWS_MEDIA_FILE"></span>**упакованный Windows файл мультимедиа**
</dt> <dd>

Windowsный файл мультимедиа, зашифрованный с помощью ключа, который пользователи не могут воспроизвести, если у них нет ключа, предоставленного лицензией. упакованный Windows файл мультимедиа создается и защищается с помощью реализации управления цифровыми правами с помощью Windows пакета sdk или программы на основе пакета sdk для Microsoft Windows media Format.

</dd> <dt>

<span id="wmformat.packaging"></span><span id="WMFORMAT.PACKAGING"></span>**Упаковка**
</dt> <dd>

процесс, который защищает и подписывает Windowsный файл мультимедиа, создавая упакованный Windows файл мультимедиа. процесс упаковки включает в себя создание или указание ключа, создание и подпись заголовка содержимого, а также шифрование файла мультимедиа Windows с помощью этих сведений.

</dd> <dt>

<span id="wmformat.packet"></span><span id="WMFORMAT.PACKET"></span>**Пакетная**
</dt> <dd>

Единица данных, передаваемая по сети от одного компьютера или устройства к другому.

</dd> <dt>

<span id="wmformat.plug_in"></span><span id="WMFORMAT.PLUG_IN"></span>**подключаемый модуль**
</dt> <dd>

Дополнительный программный компонент, расширяющий или расширяющий возможности другого программного обеспечения.

</dd> <dt>

<span id="wmformat.profile"></span><span id="WMFORMAT.PROFILE"></span>**профиль**
</dt> <dd>

Группа параметров, которые соответствуют типу содержимого и скорости в битах с соответствующими аудиокодеками и видеокодеками.

</dd> <dt>

<span id="wmformat.registration"></span><span id="WMFORMAT.REGISTRATION"></span>**Регистрация**
</dt> <dd>

Процесс, в ходе которого потребитель вводит информацию для получения лицензии, например адрес электронной почты.

</dd> <dt>

<span id="wmformat.revocation_list"></span><span id="WMFORMAT.REVOCATION_LIST"></span>**список отзыва**
</dt> <dd>

Список, содержащий все сертификаты приложений для этих приложений проигрывателя, которые известны как поврежденные или поврежденные. Этот список включен в лицензии, а затем хранится на компьютерах потребителей компонентом управления цифровыми правами (DRM) приложения проигрывателя.

</dd> <dt>

<span id="wmformat.rgb"></span><span id="WMFORMAT.RGB"></span>**RGB**
</dt> <dd>

Цветовая модель, описывающая сведения о цвете в терминах красного (R), зеленого (G) и синего (B) интенситиес, составляющих цвет.

</dd> <dt>

<span id="wmformat.sdmi"></span><span id="WMFORMAT.SDMI"></span>**SDMI**
</dt> <dd>

См. определение для: [ *инициатива для защиты цифровых музыкальных файлов (SDMI)*](/windows)

</dd> <dt>

<span id="wmformat.secure_digital_music_initiative__sdmi_"></span><span id="WMFORMAT.SECURE_DIGITAL_MUSIC_INITIATIVE__SDMI_"></span>**Инициатива по защите цифровой музыки (SDMI)**
</dt> <dd>

Организация, устанавливающая стандарты безопасной цифровой музыки. Одной из основных целей SDMI является создание платформы для безопасного воспроизведения, хранения и распространения цифровой музыки.

</dd> <dt>

<span id="wmformat.security_upgrade"></span><span id="WMFORMAT.SECURITY_UPGRADE"></span>**Обновление системы безопасности**
</dt> <dd>

Процесс создания компонента управления цифровыми правами (DRM) в уникальном проигрывателе потребителя. Этот процесс повышает безопасность, усложняя повреждение более чем одного игрока. Этот процесс также известен как отдельная.

</dd> <dt>

<span id="wmformat.streaming"></span><span id="WMFORMAT.STREAMING"></span>**потоковая передача**
</dt> <dd>

Метод доставки цифровых носителей по сети в непрерывном потоке. Цифровой носитель воспроизводится клиентским программным обеспечением по мере его получения. Как правило, потоковая передача не требуется, чтобы пользователи загружали файл перед его воспроизведением.

</dd> <dt>

<span id="wmformat.system_profile"></span><span id="WMFORMAT.SYSTEM_PROFILE"></span>**системный профиль**
</dt> <dd>

Стандартные шаблоны по умолчанию, содержащие необходимые технические сведения для кодирования определенного фрагмента содержимого.

</dd> <dt>

<span id="wmformat.system_profiles"></span><span id="WMFORMAT.SYSTEM_PROFILES"></span>**системные профили**
</dt> <dd>

Нет доступных определений.

</dd> <dt>

<span id="wmformat.target_bandwidth"></span><span id="WMFORMAT.TARGET_BANDWIDTH"></span>**Целевая пропускная способность**
</dt> <dd>

Число битов, которые передаются пользователям в зависимости от скорости их сетевого подключения.

</dd> <dt>

<span id="wmformat.telecine"></span><span id="WMFORMAT.TELECINE"></span>**чересстрочности**
