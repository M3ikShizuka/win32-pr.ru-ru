# <a name="configuring-arbitrary-stream-types"></a>Настройка произвольных типов потоков

Для большинства произвольных типов потоков требуется только скорость потока, окно буфера и соответствующий основной тип носителя в структуре [**\_ \_ типа мультимедиа WM**](/previous-versions/windows/desktop/api/wmsdkidl/ns-wmsdkidl-wm_media_type) . Однако для некоторых произвольных типов требуется дополнительная настройка.

Если при настройке потока возникли проблемы, см. пример приложения с именем Женпрофиле, которое входит в состав этого пакета SDK. Библиотека, определенная в Женпрофиле, содержит код для включения всех типов потоков. Дополнительные сведения о Женпрофиле и других примерах см. в разделе [примеры приложений](sample-applications.md).
