# <a name="wmmediaisfinale"></a>WM/Медиаисфинале

Атрибут **WM/медиаисфинале** указывает, является ли содержимое последним из рядов.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиаисфинале

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут используется для трансляции телепередач.
