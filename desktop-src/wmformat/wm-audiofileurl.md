# <a name="wmaudiofileurl"></a>WM/Аудиофилеурл

Атрибут **WM/аудиофилеурл** содержит адрес официальной веб-страницы со сведениями о файле, в котором отображается этот атрибут. Например, песня может иметь обратную ссылку на страницу альбома на веб-сайте исполнителя.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмаудиофилеурл

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**
