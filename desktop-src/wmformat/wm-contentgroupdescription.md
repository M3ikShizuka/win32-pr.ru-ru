# <a name="wmcontentgroupdescription"></a>WM/Контентграупдескриптион

Атрибут **WM/контентграупдескриптион** содержит описание группы содержимого.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмконтентграупдескриптион

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Группы содержимого — это коллекции носителей, к которым могут принадлежать отдельные файлы. Например, звуковой файл может быть музыкальным файлом, который можно дорожить на одном компакт-диске, входящем в упакованный набор. Аналогичным образом видеоклип может быть частью ряда.
