# <a name="wmvideoclosedcaptioning"></a>WM/Видеоклоседкаптионинг

Атрибут **WM/видеоклоседкаптионинг** указывает, включает ли содержимое скрытые субтитры.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмвидеоклоседкаптионинг

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут используется для широковещательного телевидения и другого видеоматериала.
