# <a name="writing-image-streams"></a>запись образа Потоки

Входные данные для потока изображений должны быть битовыми изображениями в формате RGB. Модуль записи координирует сжатие входных образцов изображения, используя формат JPEG. Перед началом записи файла, содержащего поток изображений, необходимо задать качество изображения для входных данных с помощью \_ параметра g всзжпегкомпрессионкуалити. Используйте [**IWMWriterAdvanced2:: сетинпутсеттинг**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmwriteradvanced2-setinputsetting) , чтобы задать для свойства Quality значение **DWORD** в диапазоне от 1 до 100. Низкие значения представляют собой коэффициент высокой степени сжатия за счет высокого качества, а высокие значения дают высококачественные изображения, требующие больше пространства.

Потокам изображений часто требуются большие окна буфера, чем обычные видеопотоки. Точный требуемый размер зависит от типа изображения и качества изображения, среди прочих факторов. Используйте пробную версию и ошибку, чтобы определить подходящий размер для образов, которые планируется обработать.
