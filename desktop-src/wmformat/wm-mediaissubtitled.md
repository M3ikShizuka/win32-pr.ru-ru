# <a name="wmmediaissubtitled"></a>WM/Медиаиссубтитлед

Атрибут **WM/медиаиссубтитлед** указывает, содержит ли содержимое субтитры.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиаиссубтитлед

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут используется для трансляции телепередач.
