# <a name="viewing-drm-attributes-in-the-metadata-editor"></a>Просмотр атрибутов DRM в редакторе метаданных

Редактор метаданных предоставляет интерфейс Ивмдрмедитор, который позволяет редактировать приложения для проверки определенных атрибутов заголовка DRM в защищенном ASF-файле и атрибутов в лицензии содержимого, если таковая имеется, даже если в приложении нет статической библиотеки DRM (вмстубдрм. lib), необходимой для полностью включенных приложений для проигрывателя DRM и модуля записи.

> [!Note]  
> DRM не поддерживает версию этого пакета SDK на базе x64.
