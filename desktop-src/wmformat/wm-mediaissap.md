# <a name="wmmediaissap"></a>WM/Медиаиссап

Атрибут **WM/медиаиссап** указывает, включает ли содержимое вторую звуковую программу (SAP).

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиаиссап

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут используется для трансляции телепередач.

SAP обычно определяет телевизионные программы, которые включают в себя альтернативную звуковую дорожку на другом языке.
