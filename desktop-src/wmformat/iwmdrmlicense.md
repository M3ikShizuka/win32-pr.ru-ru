# <a name="iwmdrmlicense-interface"></a>Интерфейс Ивмдрмлиценсе

Интерфейс **ивмдрмлиценсе** представляет список из одной или нескольких лицензий.

## <a name="members"></a>Элементы

Интерфейс **ивмдрмлиценсе** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Ивмдрмлиценсе** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмдрмлиценсе** содержит следующие методы.



| Метод                                                                                   | Описание                                                                                                                                                                                                                        |
|:-----------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**канперсист**](iwmdrmlicense-canpersist.md)                                           | Запрашивает возможность сохранения лицензии в локальном хранилище лицензий.<br/>                                                                                                                                               |
| [**CreateDecryptor**](iwmdrmlicense-createdecryptor.md)                                 | Создает объект дешифратора, используя параметры текущей лицензии.<br/>                                                                                                                                                   |
| [**креатинкриптор**](iwmdrmlicense-createencryptor.md)                                 | Создает объект шифратора, используя параметры текущей лицензии.<br/>                                                                                                                                                  |
| [**креатесекуредекриптор**](iwmdrmlicense-createsecuredecryptor.md)                     | Создает объект безопасного дешифратора. Безопасный дешифратор отличается от обычного дешифратора в том, что расшифровывает содержимое, а затем повторно шифрует его в соответствии с параметрами, заданными в параметрах этого метода.<br/> |
| [**жетаналогвидеорестриктионлевелс**](iwmdrmlicense-getanalogvideorestrictionlevels.md) | Возвращает все установленные для текущей лицензии ограничения аналогового видео.<br/>                                                                                                                                                     |
| [**жетинклусионлист**](iwmdrmlicense-getinclusionlist.md)                               | Извлекает весь список включения для текущей лицензии или цепочки лицензий.<br/>                                                                                                                                           |
| [**Лицензия**](iwmdrmlicense-getlicense.md)                                           | Извлекает лицензию как данные язык XML (XML) или Extensible Media Rights (КСМР).<br/>                                                                                                                        |
| [**жетлиценсепроперти**](iwmdrmlicense-getlicenseproperty.md)                           | Извлекает свойство из текущей лицензии.<br/>                                                                                                                                                                          |
| [**GetNext**](iwmdrmlicense-getnext.md)                                                 | Загружает сведения о следующей лицензии в списке.<br/>                                                                                                                                                               |
| [**жетаутпутпротектионлевелс**](iwmdrmlicense-getoutputprotectionlevels.md)             | Извлекает сведения обо всех уровнях защиты выходных данных (Оплс), назначенных лицензии.<br/>                                                                                                                                |
| [**персистлиценсе**](iwmdrmlicense-persistlicense.md)                                   | Сохраняет текущую лицензию из временного хранилища в постоянное локальное хранилище лицензий.<br/>                                                                                                                              |
| [**ресетенумератион**](iwmdrmlicense-resetenumeration.md)                               | Сбрасывает список лицензий в исходное состояние.<br/>                                                                                                                                                                          |
