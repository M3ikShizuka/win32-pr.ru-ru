# <a name="software-redistribution"></a>Повторное распространение программного обеспечения

включение программы Windowsного формата мультимедиа в программу установки приложения называется перераспределением. в пакет SDK для Windows Media Format входит пакет установки, который можно добавить в программу установки приложения. Пакет установки — это исполняемый файл с именем wmfdist95.exe. при установке пакета SDK для Windows Media Format пакет установки копируется в \\ папку redist каталога установки (c: \\ вмсдк \\ вмфсдк по умолчанию).
