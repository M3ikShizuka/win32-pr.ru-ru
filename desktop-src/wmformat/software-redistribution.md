# <a name="software-redistribution"></a>Повторное распространение программного обеспечения

Включение программного обеспечения формата Windows Media в программу установки приложения называется перераспределением. Пакет SDK для формата Windows Media включает установочные пакеты, которые можно добавить в программу установки приложения. Пакет установки — это исполняемый файл с именем wmfdist95.exe. При установке пакета SDK для формата Windows Media пакет установки копируется в \\ папку Redist каталога установки (c: \\ вмсдк \\ вмфсдк по умолчанию).

В следующих разделах приводятся процедуры и другие сведения о настройке распространения программного обеспечения.

| [Создание настройки распространения](to-create-a-redistribution-setup.md)           | Описание процедуры создания программы установки приложения.                                                                                                                       |
| [Определение состояния установки](detecting-setup-status.md)                               | Содержит пример кода, который проверяет наличие состояния установки в реестре, чтобы определить, нужно ли установить пакет распространения.                                    |
