# <a name="bannerimagetype"></a>баннеримажетипе

Атрибут **баннеримажетипе** задает тип изображения, хранящегося в атрибуте [**баннеримажедата**](bannerimagedata.md) . Этот атрибут является значением из типа перечисления [**ВМТ \_ image \_ Type**](/previous-versions/windows/desktop/api/Wmsdkidl/ne-wmsdkidl-wmt_image_type) .

g \_ всзвмбаннеримажетипе

**ВМТ, \_ тип \_ DWORD**
