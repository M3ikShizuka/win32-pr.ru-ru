# <a name="multicast-station-attributes"></a>Атрибуты многоадресной станции

При потоковой передаче файла с многоадресной станции станция может наложить некоторые атрибуты на файл. Эти атрибуты, перечисленные в следующей таблице, фактически не хранятся в файле и доступны только в том случае, если файл является потоковой передачей. Они содержат сведения о станции и, как правило, идентичны для всего содержимого с станции.



| Атрибут многоадресной станции                 | Глобальный идентификатор      | Тип данных             |
|---------------------------------------------|------------------------|-----------------------|
| [**NSC \_ адрес**](nsc-address.md)         | g \_ всзвмнскаддресс     | **\_Строка типа \_ ВМТ** |
| [**\_Описание NSC**](nsc-description.md) | g \_ всзвмнскдескриптион | **\_Строка типа \_ ВМТ** |
| [**NSC \_ адрес электронной почты**](nsc-email.md)             | g \_ всзвмнсцемаил       | **\_Строка типа \_ ВМТ** |
| [**\_Имя NSC**](nsc-name.md)               | g \_ всзвмнскнаме        | **\_Строка типа \_ ВМТ** |
| [**NSC \_ Телефон**](nsc-phone.md)             | g \_ всзвмнскфоне       | **\_Строка типа \_ ВМТ** |
