# <a name="iwmdrmdecrypt-interface"></a>Интерфейс Ивмдрмдекрипт

Интерфейс **ивмдрмдекрипт** предоставляет метод для расшифровки содержимого.

## <a name="members"></a>Элементы

Интерфейс **ивмдрмдекрипт** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Ивмдрмдекрипт** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмдрмдекрипт** содержит следующие методы.



| Метод                                   | Описание                                 |
|:-----------------------------------------|:--------------------------------------------|
| [**расшифровка;**](iwmdrmdecrypt-decrypt.md) | Расшифровывает буфер данных на месте.<br/> |
