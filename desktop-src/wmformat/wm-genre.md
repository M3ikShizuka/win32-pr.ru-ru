# <a name="wmgenre"></a>WM/жанр

Атрибут **WM/жанр** содержит жанр содержимого.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмженре

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Это предпочтительный атрибут для указания жанра содержимого.

Если изменить файл **WM/жанр** или **WM/GENREID** в файле MP3, то другой атрибут будет изменен на Match.

### <a name="example"></a>Пример



| Тип файла | Пример значения |
|-----------|---------------|
| звук;     | Рок        |
| видео;     | Драма       |
