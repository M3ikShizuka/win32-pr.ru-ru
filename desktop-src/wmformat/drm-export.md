# <a name="drm-export"></a>Экспорт DRM

Windows функция экспорта DRM для мультимедиа позволяет создавать лицензированные приложения, которые расшифровывать содержимое, защищенное Windows Media drm, и повторно зашифровать в выходной схеме защиты содержимого (CPS), явно указанной соответствующим издателем лицензии. Издатель лицензии явно разрешает экспорт содержимого в конкретный CPS с помощью списка включения.

> [!Note]  
> Чтобы получить доступ к функциям экспорта, необходимо применить к [сертификату приложения для экспорта](export-application-certificate.md) из корпорации Майкрософт.

 

Windows Функции экспорта DRM для мультимедиа описаны в следующих разделах.

|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [Экспорт сертификата приложения](export-application-certificate.md)                         | Описывает сертификат приложения экспорта.                                                        |
| [Явные списки авторизации и включений](explicit-authorization-and-inclusion-lists.md) | Описание процесса явной авторизации и работы списков включения.                      |
| [Экспорт сжатого содержимого](exporting-compressed-content.md)                             | в этой статье описывается, как экспортировать сжатое содержимое из аудио или видео, защищенного с использованием Windows Media DRM. |
