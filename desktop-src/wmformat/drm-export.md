# <a name="drm-export"></a>Экспорт DRM

Функция экспорта DRM в Windows Media позволяет создавать лицензированные приложения, которые расшифровывать содержимое, защищенное с помощью Windows Media DRM, и повторно зашифровать в выходной схеме защиты содержимого (CPS), явно указанной соответствующим издателем лицензии. Издатель лицензии явно разрешает экспорт содержимого в конкретный CPS с помощью списка включения.

> [!Note]  
> Чтобы получить доступ к функциям экспорта, необходимо применить к [сертификату приложения для экспорта](export-application-certificate.md) из корпорации Майкрософт.

 

Функции экспорта DRM в Windows Media описаны в следующих разделах.



| Section                                                                                      | Описание                                                                                         |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [Экспорт сертификата приложения](export-application-certificate.md)                         | Описывает сертификат приложения экспорта.                                                        |
| [Явные списки авторизации и включений](explicit-authorization-and-inclusion-lists.md) | Описание процесса явной авторизации и работы списков включения.                      |
| [Экспорт сжатого содержимого](exporting-compressed-content.md)                             | Описывает, как экспортировать сжатое содержимое из защищенного аудио или видео содержимого Windows Media DRM. |
