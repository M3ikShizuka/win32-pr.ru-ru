# <a name="using-the-callback-methods"></a>Использование методов обратного вызова

Несколько интерфейсов в пакете SDK формата Windows Media содержат методы, которые вызываются асинхронно. Большинство из этих методов используют функции обратного вызова для передачи информации в управляющее приложение.

В следующих разделах описаны некоторые общие проблемы, связанные с использованием методов обратного вызова в пакете SDK формата Windows Media.



| Section                                                                          | Описание                                                                                                                                                                                          |
|----------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Использование обратного вызова OnStatus](using-the-onstatus-callback.md)                   | Описывает, как реализовать метод обратного вызова [**ивмстатускаллбакк:: OnStatus**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmstatuscallback-onstatus) , который используется несколькими объектами для уведомления приложений о ходе выполнения операции пакета SDK. |
| [Использование событий с асинхронными вызовами](using-events-with-asynchronous-calls.md) | Описывает распространенный способ работы с асинхронными вызовами методов в приложении.                                                                                                                  |
| [Использование параметра context](using-the-context-parameter.md)                   | Представляет параметр *пвконтекст* , совместно используемый несколькими обратными вызовами и их вызывающими методами, и объясняет, как его использовать.                                                                             |
