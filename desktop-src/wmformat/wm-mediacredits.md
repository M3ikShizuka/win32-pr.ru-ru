# <a name="wmmediacredits"></a>WM/Медиакредитс

Атрибут **WM/медиакредитс** содержит кредиты для вещания мультимедиа.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиакредитс

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут используется для трансляции телепередач или радио.
