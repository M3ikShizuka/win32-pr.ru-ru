# <a name="audio-resampling"></a>Ресамплинг звука

Каждый сжатый формат аудиокодека имеет определенную частоту выборки и размер выборки. Они не должны соответствовать параметрам формата входных данных или выходного формата. Если формат входных данных отличается от параметров сжатого формата, описанного в профиле, модуль записи выполнит повторную выборку звука в процессе кодирования, чтобы соответствовать сжатому формату. Только определенные форматы принимаются модулем записи в качестве входных данных. При перечислении входных форматов для сжатого аудиопотока все полученные форматы можно повторно вычислить в соответствии с форматом в профиле.

При чтении сжатого звука средство чтения повторно выполнит выборку содержимого в соответствии с форматом выходных данных. Необходимо использовать один из выходных форматов, перечисленных модулем чтения, поэтому вы гарантируете, что звук можно будет перевыборить в параметры формата вывода.

Каждая перевыборка может влиять на качество звука. По возможности следует использовать форматы входных и выходных данных с параметрами, соответствующими сжатому формату.
