# <a name="iwmdrmlicensemanagementrestorelicenses-method"></a>Метод Ивмдрмлиценсеманажемент:: Ресторелиценсес

Метод **ресторелиценсес** восстанавливает лицензии из резервной копии лицензий, созданной путем вызова метода [**баккуплиценсес**](iwmdrmlicensemanagement-backuplicenses.md) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT RestoreLicenses(
  [in]  BSTR     bstrBackupDirectory,
  [in]  DWORD    dwFlags,
  [out] IUnknown **ppunkCancelationCookie
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*бстрбаккупдиректори* \[ окне\]
</dt> <dd>

UNC-путь к расположению, из которого будут восстановлены лицензии.

</dd> <dt>

*dwFlags* \[ окне\]
</dt> <dd>

Флаги, указывающие используемые параметры восстановления. В настоящее время поддерживается только флаг \_ \_ индивидуализируйте для восстановления WMDRM, который настраивает метод для выполнения индивидуализации в процессе восстановления, если это необходимо.

</dd> <dt>

*ппункканцелатионкукие* \[ заполняет\]
</dt> <dd>

Указатель, получающий указатель на интерфейс **IUnknown** объекта, определяющий этот асинхронный вызов. Этот указатель интерфейса можно использовать для отмены асинхронного вызова путем вызова метода [**ивмдрмевентженератор:: канцеласинкоператион**](iwmdrmeventgenerator-cancelasyncoperation.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |



 

## <a name="remarks"></a>Комментарии

Этот метод выполняется асинхронно. Он возвращает сразу после вызова, а затем создает ряд событий **мевмдрмлиценсересторепрогресс** , за которыми следует событие **мевмдрмлиценсересторекомплетед** после завершения обработки. Значение каждого из событий **мевмдрмлиценсересторепрогресс** , полученных путем вызова **Имфмедиаевент:: GetValue** , является указателем **IUnknown** . Чтобы получить экземпляр интерфейса [**ивмдрмлиценсебаккупресторестатус**](iwmdrmlicensebackuprestorestatus.md) , можно вызвать метод **QueryInterface** полученного интерфейса **IUnknown** .

Дополнительные сведения об использовании асинхронных методов расширенных API-интерфейсов клиента DRM Windows Media см. [в разделе Использование модели событий Media Foundation](using-the-media-foundation-model.md).

Резервное копирование может выполняться с локального компьютера или с другого компьютера.
