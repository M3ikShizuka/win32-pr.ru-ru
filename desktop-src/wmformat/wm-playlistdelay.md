# <a name="wmplaylistdelay"></a>WM/Плайлистделай

Атрибут **WM/плайлистделай** содержит время задержки в миллисекундах, которое должно предшествовать воспроизведению файла в списке воспроизведения.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмплайлистделай

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Не все проигрыватели мультимедиа учитывают задержку списка воспроизведения.
