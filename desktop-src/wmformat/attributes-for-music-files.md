# <a name="attributes-for-music-files"></a>Атрибуты для музыкальных файлов

В этом разделе перечислены атрибуты, обычно используемые для звуковых файлов, содержащих музыку. Рекомендуется устанавливать атрибуты для файлов в соответствии с этими списками, чтобы обеспечить полную совместимость файлов с самыми разнообразными приложениями воспроизведения. Атрибуты в этом разделе перечислены в трех категориях: первичный, вторичный и третичный.

Основные атрибуты содержат самые основные сведения о файле. При создании звуковых файлов для распространения это минимальный набор атрибутов, которые следует использовать.

Вторичные атрибуты содержат общие метаданные, которые важны, но не являются универсальными для всех звуковых файлов.

При необходимости необходимо добавить третичные атрибуты, но не обязательно описывать файл.
