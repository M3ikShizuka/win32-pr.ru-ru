# <a name="drm_actionallowed_copytosdmidevice"></a>\_Актионалловед \_ копитосдмидевице DRM

Атрибут **DRM \_ актионалловед \_ копитосдмидевице** указывает, разрешено ли копирование содержимого на устройство SDMI.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдрм \_ актионалловед \_ копитосдмидевице

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Windows Лицензии Media DRM 10 используют действие копирования для ограничения копирования на устройства. Чтобы определить, разрешено ли копирование, проверьте свойство [**\_ Актионалловед \_ Copy DRM**](drm-actionallowed-copy.md) .

Это свойство только для чтения, которое извлекается с помощью [**ивмдрмреадер:: жетдрмпроперти**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
