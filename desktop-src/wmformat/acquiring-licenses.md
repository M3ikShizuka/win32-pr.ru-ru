# <a name="acquiring-licenses"></a>Получение лицензий

распространенный сценарий для получения лицензий — когда пользователь имеет защищенный Windowsный файл мультимедиа и пытается получить доступ к содержимому в первый раз. так как клиентские api-интерфейсы Windows Media DRM отделены от подпрограмм обработки файлов ASF, базовый сценарий приобретения лицензий, поддерживаемый, требует больше вызовов API, чем использование пакета sdk для Windows Media Format и пакета sdk для Media Foundation.

в этом разделе описывается использование расширенных api-интерфейсов клиента DRM Windows Media для получения лицензий, которые хранятся в локальном хранилище лицензий на клиентском компьютере. Он содержит следующие разделы.



| Раздел                                                                | Описание                                                                                                                                |
|----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [Автоматическое получение лицензий](silent-license-acquisition.md)         | Описывает, как получать лицензии без вмешательства пользователя.                                                                               |
| [Получение лицензий без уведомления](non-silent-license-acquisition.md) | Описывает, как получить лицензии, если требуется вмешательство пользователя (например, оплата содержимого).                                     |
| [Предварительная Поставка лицензий](license-pre-delivery.md)                     | Описывает, как получать лицензии с сервера лицензий на клиентский компьютер.                                                                 |
| [Создание лицензий локально](creating-licenses-locally.md)           | Описывает создание собственных лицензий без их загрузки с сервера лицензирования и хранение их в локальном хранилище лицензий. |
