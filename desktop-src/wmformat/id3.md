# <a name="id3-support"></a>Поддержка ID3

ID3 — это организация, которая определила набор стандартов для включения метаданных в звуковые файлы MPEG Layer-3 (MP3). объекты пакета SDK для Windowsного формата мультимедиа поддерживают атрибуты, совместимые с ID3. Поддерживаются три различные версии ID3: ID3v1. x, ID3v 2.2 и ID3v 2.3/v2, 4. Список атрибутов, которые эквивалентны кадрам ID3, см. в разделе [Поддержка тегов ID3](id3-tag-support.md).

Если не указано иное, объекты этого пакета SDK не выполняют проверку данных кадра ID3. Например, атрибут метаданных [**WM/песен \_ синхронизированы**](wm-lyrics-synchronised.md) сохраняет песни песни с соответствующими штампами времени. При записи атрибута **WM/песен \_ синхронизированы** объекты этого пакета SDK не будут проверять, что отметки времени находятся в хронологическом порядке или выполняют проверку любого вида.
