# <a name="iwmdrmlicensebackuprestorestatus-interface"></a>Интерфейс Ивмдрмлиценсебаккупресторестатус

Интерфейс **ивмдрмлиценсебаккупресторестатус** предоставляет метод для получения подробных сведений о состоянии операции резервного копирования или восстановления лицензий.

Этот интерфейс доставляется с событиями хода выполнения, созданными во время операций резервного копирования и восстановления лицензий. Во время резервного копирования лицензий создаются несколько событий **мевмдрмлиценсебаккуппрогресс** , каждый из которых имеет сопутствующий экземпляр этого интерфейса. То же самое относится к событиям **мевмдрмлиценсересторепрогресс** , которые создаются во время восстановления лицензии.

Чтобы получить указатель на экземпляр интерфейса **ивмдрмлиценсебаккупресторестатус** , необходимо сначала вызвать метод **Имфмедиаевент:: GetValue** события Progress. Значение, получаемое из события, является указателем на интерфейс **IUnknown** объекта, реализующего интерфейс **ивмдрмлиценсебаккупресторестатус** .

## <a name="members"></a>Элементы

Интерфейс **ивмдрмлиценсебаккупресторестатус** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Ивмдрмлиценсебаккупресторестатус** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмдрмлиценсебаккупресторестатус** содержит следующие методы.



| Метод                                                          | Описание                                                                                   |
|:----------------------------------------------------------------|:----------------------------------------------------------------------------------------------|
| [**GetStatus**](iwmdrmlicensebackuprestorestatus-getstatus.md) | Получает подробные сведения о состоянии операции резервного копирования или восстановления лицензий.<br/> |
