# <a name="drm_isdrmcached"></a>\_ИСДРМКАЧЕД DRM

Свойство **DRM \_ исдрмкачед** указывает, хранятся ли на локальном компьютере сведения о лицензии DRM версии 1.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдрм \_ исдрмкачед

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Это свойство только для чтения, которое извлекается с помощью [**ивмдрмреадер:: жетдрмпроперти**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty). Это **справедливо** , если URL-адрес получения лицензии совпадает с одним из двух известных URL-адресов, используемых для приобретения локальной лицензии в DRM версии 1.
