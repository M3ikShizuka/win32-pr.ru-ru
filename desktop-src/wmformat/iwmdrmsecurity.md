# <a name="iwmdrmsecurity-interface"></a>Интерфейс Ивмдрмсекурити

Интерфейс **ивмдрмсекурити** управляет различными сведениями, относящимися к безопасности, для клиентских компьютеров и подсистемы управления цифровыми правами (DRM).

Чтобы получить экземпляр этого интерфейса, вызовите метод [**ивмдрмпровидер:: CreateObject**](iwmdrmprovider-createobject.md). Передайте **IID \_ ивмдрмсекурити** в качестве параметра *riid* .

## <a name="members"></a>Элементы

Интерфейс **ивмдрмсекурити** наследует от [**ивмдрмевентженератор**](iwmdrmeventgenerator.md). **Ивмдрмсекурити** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмдрмсекурити** содержит следующие методы.



| Метод                                                                                      | Описание                                                                                                      |
|:--------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------|
| [**чеккцертфорревокатион**](iwmdrmsecurity-checkcertforrevocation.md)                     | Определяет, был ли отозван сертификат.<br/>                                                    |
| [**жетконтентенаблерсфорревокатионс**](iwmdrmsecurity-getcontentenablersforrevocations.md) | Получает интерфейсы включения содержимого, которые обеспечивают возобновление компонентов на основе отозванных сертификатов.<br/> |
| [**жетконтентенаблерсфромхашес**](iwmdrmsecurity-getcontentenablersfromhashes.md)         | Получает интерфейсы включения содержимого, которые обеспечивают возобновление компонентов на основе хэшированных сертификатов.<br/>  |
| [**жетмачинецертификате**](iwmdrmsecurity-getmachinecertificate.md)                       | Получает сертификат компьютера подсистемы DRM на клиентском компьютере.<br/>                        |
| [**жетревокатиондата**](iwmdrmsecurity-getrevocationdata.md)                               | Извлекает список отзыва сертификатов из локального хранилища.<br/>                                         |
| [**жетревокатиондатаверсион**](iwmdrmsecurity-getrevocationdataversion.md)                 | Возвращает номер версии списка отзыва сертификатов в локальном хранилище.<br/>                     |
| [**жетсекуритиверсион**](iwmdrmsecurity-getsecurityversion.md)                             | Извлекает версию подсистемы DRM на клиентском компьютере.<br/>                                    |
| [**перформсекуритюпдате**](iwmdrmsecurity-performsecurityupdate.md)                       | Инициирует обновление безопасности для подсистемы DRM на клиентском компьютере.<br/>                              |
| [**сетревокатиондата**](iwmdrmsecurity-setrevocationdata.md)                               | Задает список отзыва сертификатов в локальном хранилище.<br/>                                                |
