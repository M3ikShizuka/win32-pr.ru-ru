# <a name="iwmdrmlicensemanagementstorelicense-method"></a>Метод Ивмдрмлиценсеманажемент:: Сторелиценсе

Метод **сторелиценсе** включает лицензию, созданную за пределами локальной подсистемы DRM в локальном хранилище лицензий.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT StoreLicense(
  [in] BSTR bstrLicenseResponse
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*бстрлиценсереспонсе* \[ окне\]
</dt> <dd>

Строка ответа лицензии для декодирования и добавления в локальное хранилище лицензий.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |



 

## <a name="remarks"></a>Комментарии

Этот метод можно использовать для добавления лицензий КСМР, созданных в локальном хранилище лицензий.
