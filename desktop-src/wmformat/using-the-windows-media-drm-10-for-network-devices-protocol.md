# <a name="using-the-windows-media-drm-10-for-network-devices-protocol"></a>Использование протокола Windows Media DRM 10 для сетевых устройств

Пакет Windows Media Format SDK предоставляет некоторые функции для поддержки протокола защищенных сетевых устройств, Windows Media DRM 10 для сетевых устройств. Этот протокол определяет сообщения, которые передаются между устройством воспроизведения цифрового носителя, например видеопроигрывателем Set-Top, и компьютером, который обслуживает данные на устройстве. Данные мультимедиа, передаваемые между сервером и устройством, шифруются для защиты от перехвата.

Взаимодействие между приложением и устройствами, поддерживающими Windows Media DRM 10 для сетевых устройств, может использовать любой предпочтительный сетевой протокол. Пакет SDK для формата Windows Media не предоставляет никаких объектов, которые выполняют сетевую связь. Вместо этого объекты этого пакета SDK обрабатывают некоторые сообщения Windows Media DRM 10 для сетевых устройств после их получения приложением.

Функции, поддерживающие Windows Media DRM 10 для сетевых устройств, — это регистрация устройств, обнаружение с учетом расположения и преобразование файлов, защищенных с помощью DRM. Эти функции описаны в следующих разделах.
