# <a name="using-the-windows-media-drm-10-for-network-devices-protocol"></a>использование протокола Windows Media DRM 10 для сетевых устройств

пакет SDK для Windows media Format предоставляет некоторые функции для поддержки протокола защищенных сетевых устройств, Windows Media DRM 10 для сетевых устройств. Этот протокол определяет сообщения, которые передаются между устройством воспроизведения цифрового носителя, например видеопроигрывателем Set-Top, и компьютером, который обслуживает данные на устройстве. Данные мультимедиа, передаваемые между сервером и устройством, шифруются для защиты от перехвата.
