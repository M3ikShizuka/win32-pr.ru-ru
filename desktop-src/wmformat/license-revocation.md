# <a name="license-revocation-windows-media-format-sdk"></a>Отзыв лицензий (пакет SDK для формата Windows Media)

Пакет Windows Media Format SDK предоставляет методы, с помощью которых приложение может взаимодействовать с веб-узлом поставщика лицензий для отзыва лицензии на содержимое.

Типичный сценарий отзыва лицензии — когда пользователь передает старый компьютер другому пользователю. В этом случае у исходного пользователя может быть несколько лицензий на защищенное содержимое, сохраненное на компьютере, даже если этот пользователь больше не использует этот компьютер. Отзыв лицензии позволяет исходному пользователю удалить лицензии с компьютера.

Процесс отзыва лицензий включает передачу сообщений между клиентским приложением и веб-приложение, созданное издателем лицензии.

**Примечание** . DRM не поддерживает версию этого пакета SDK на базе x64.
