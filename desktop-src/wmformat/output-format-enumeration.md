# <a name="output-format-enumeration"></a>Перечисление формата вывода

Объект Reader и синхронный объект Reader взаимодействуют с кодеками для перечисления возможных форматов вывода для сжатых потоков. при чтении файла, содержащего содержимое, сжатое с помощью одного или нескольких кодеков Windows мультимедиа, можно просмотреть возможные форматы вывода, чтобы выбрать тот, который наилучшим образом соответствует вашим потребностям. Для удобства каждый кодек имеет формат вывода по умолчанию, для которого задан наиболее часто используемый формат. Дополнительные сведения о перечислении Output см. [в разделе Работа с выходами](working-with-outputs.md).

Вы можете внести определенные изменения в выходной формат в зависимости от типа носителя. Например, для видеопотоков можно изменить размер и глубину цвета. Объекты чтения поддерживают интерфейс для тестирования изменений, внесенных в формат вывода, именуемый [**ивмреадертипенеготиатион**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmreadertypenegotiation).
