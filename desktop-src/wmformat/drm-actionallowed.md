# <a name="drm_actionallowed"></a>\_АКТИОНАЛЛОВЕД DRM

Свойство **\_ актионалловед DRM** является базовой строкой для других строковых констант свойства актионалловед, таких как DRM \_ актионалловед \_ копитокд, поскольку свойство "действие разрешено" не является расширяемым, эта базовая строка не используется для приложений.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдрм \_ актионалловед

## <a name="data-type"></a>Тип данных

Не тип данных.

## <a name="remarks"></a>Комментарии

Свойства "Актионалловед" в настоящее время не расширяемы.
