# <a name="wmt_videoimage_transition_iris"></a>ВМТ \_ видеоимаже \_ 1-2-3 \_

Переход IRI раскрывает новый образ вдоль оси x и оси y. Визуальный результат этого перехода заключается в том, что новый образ отображается в расширяющемся перекрестном переходе на все стороны фрейма.

## <a name="parameters"></a>Параметры

В следующей таблице описаны параметры, используемые этим переходом, и перечислены члены структуры [**ВМТ \_ видеоимаже \_ SAMPLE2**](/previous-versions/windows/desktop/api/Wmsdkidl/ns-wmsdkidl-wmt_videoimage_sample2) , которой они назначены.




| Параметр | Член структуры | Описание | 
|-----------|------------------|-------------|
| Центр по оси X | <strong>fEffectPara0</strong> | Координата по оси X относительно видеокадра центра результатов диафрагмы. | 
| Центр по оси Y | <strong>fEffectPara1</strong> | Координата по оси Y относительно видеокадра центра результатов диафрагмы. | 
| Ширина | <strong>fEffectPara2</strong> | Ширина вертикальной линии, в пикселях которой отображается новое изображение. | 
| Высота | <strong>fEffectPara3</strong> | Высота горизонтальной линии, которая отображает новое изображение (в пикселях). | 
| Композиция | <strong>fEffectPara4</strong> | Задайте одно из следующих значений:<ul><li>0 — задает нормальную композицию, в которой предыдущее изображение является фоном, а текущее изображение — передним планом.</li><li>1 — указывает обратную композицию, в которой текущий рисунок является фоновым изображением, а предыдущее изображение — передний план.</li></ul> |
