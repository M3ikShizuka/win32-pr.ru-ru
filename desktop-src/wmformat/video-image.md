# <a name="video-image"></a>Видеоизображение

кодек версии 2 для Windows media video 9 позволяет создавать Windows видеофайлов мультимедиа, содержимое которых состоит из ряда по-прежнему изображений, таких как цифровые фотографии. Изображения в потоке видеоизображения можно обрабатывать для создания панорамы и масштабирования, а также для ряда переходов. Конечные пользователи могут использовать эту возможность для создания и совместного использования простых слайдов, которые также могут содержать звуковую дорожку музыки или речевое звуковое сопровождение. благодаря расширенному сжатию кодека мультимедийного изображения Windows, файл, содержащий несколько фотографий, по-прежнему достаточно мал для удобного обмена электронной почтой через интернет.

Поток видеоизображения имеет собственный тип мультимедиа и специализированная структура, определяющая различные переходы, доступные при переходе от одного изображения к другому. Вы можете сдвигать и изменять представление изображений, одновременно объединяя их. затем полученный файл можно воспроизвести как любой другой файл Windows Media video.
