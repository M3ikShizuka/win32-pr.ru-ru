# <a name="copyright"></a>Авторские права

Атрибут **Copyright** содержит сообщение об авторских правах для содержимого.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмкопиригхт

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="example"></a>Пример



| Тип файла | Пример значения                          |
|-----------|----------------------------------------|
| Любой       | "© Корпорация Майкрософт (Microsoft Corporation)". |



 

## <a name="remarks"></a>Комментарии

Текст сообщения об авторских правах не соответствует ни одному стандарту. Отдельные компании могут использовать любой подходящий формат.

Этот атрибут применяется ко всему файлу.
