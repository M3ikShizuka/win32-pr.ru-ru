# <a name="wmproviderstyle"></a>WM/Провидерстиле

Атрибут **WM/провидерстиле** содержит стиль файла, назначенный поставщиком содержимого метаданных.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмпровидерстиле

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Этот атрибут по сути является вторым дизайном-жанром. Поскольку различные классификаторы имеют различные понятия, связанные с содержимым, стиль поставщика может отличаться от жанра, назначенного автором содержимого.
