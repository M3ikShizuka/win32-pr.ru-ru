# <a name="wmmediaistape"></a>WM/Медиаистапе

Атрибут **WM/медиаистапе** указывает, было ли содержимое отправлено с ранее записанной ленты.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиаистапе

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Этот атрибут используется для трансляции телепередач.
