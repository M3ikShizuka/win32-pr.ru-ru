# <a name="features"></a>Компоненты

Пакет Windows Media Format SDK предоставляет множество возможностей для работы с файлами ASF и другими файлами в расширенном системном формате. Многие из этих функций непросто классифицируются в одной из основных функций пакета SDK. В этом разделе описываются возможности пакета Windows Media Format SDK с точки зрения программирования. В каждом разделе приводятся общие сведения о функции и ссылки на соответствующие разделы в этой документации.



| Section                                                                      | Описание                                                                                                                                                      |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Функции файлов ASF](asf-file-features.md)                                   | Описание функций, встроенных в расширенный системный формат (ASF). Эти функции обычно реализуются с помощью профиля, используемого для создания файла. |
| [Функции кодеков](codec-features.md)                                         | Описание функций кодеков Windows Media, которые непосредственно поддерживаются объектами пакета SDK формата Windows Media.                                    |
| [Функции записи файлов](file-writing-features.md)                           | Описание функций, поддерживающих запись файлов.                                                                                                                |
| [Функции чтения файлов](file-reading-features.md)                           | Описание функций, поддерживающих чтение файлов.                                                                                                                |
| [Компоненты метаданных](metadata-features.md)                                   | Описание функций, поддерживающих метаданные, которые представляют собой сведения о файле и его содержимом.                                                               |
| [Функции цифровых Rights Management](digital-rights-management-features.md) | Описание функций управления цифровыми правами, интегрированных в пакет SDK Windows Media Format.                                                      |
