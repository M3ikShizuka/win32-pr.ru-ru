# <a name="error-codes-windows-media-format-11-sdk"></a>коды ошибок (пакет SDK для Windows Media Format 11)

корпорация майкрософт определила следующие коды ошибок и успешности для использования в пакете SDK для Windows Media Format. эти коды являются подмножеством более крупной коллекции, которое можно просмотреть в файлах заголовков асферр. h (коды ASF) и нсеррор. h (другие коды), которые включены в пакет SDK для Windows Media Format. Дополнительные распространенные коды ошибок можно просмотреть в файле Winerror. h.
