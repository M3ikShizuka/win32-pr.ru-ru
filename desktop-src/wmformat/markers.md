# <a name="markers"></a>Метки

Маркеры — это именованные места на временной шкале файла ASF. Каждому маркеру присваивается имя и время презентации. Можно указать столько маркеров, сколько требуется для файла.

Маркеры полезны для разбиения больших файлов ASF на логические части. Приложение, использующее объект Reader для воспроизведения файлов ASF, может предоставить пользователю возможность перехода от маркера к маркеру, что упрощает навигацию по цифровым носителям. Например, если вы выводите файл ASF из презентации, вы можете разместить маркеры на временной шкале для каждой из основных обсуждаемых разделов. При воспроизведении, вместо того чтобы получать одну длинную временную шкалу и выгадать место для поиска, пользователь может просто выбрать раздел из списка и перейти непосредственно к соответствующей части файла.

Маркеры управляются объектом редактора метаданных. Можно добавлять, удалять и просматривать маркеры в файле в любое время после его написания.
