# <a name="iwmdrmlicensemanagementprocesslicenserevocationresponse-method"></a>Ивмдрмлиценсеманажемент: метод:P Роцесслиценсеревокатионреспонсе

Метод **процесслиценсеревокатионреспонсе** отменяет лицензии из локального хранилища лицензий. Этот метод использует большой двоичный объект отзыва лицензий (ЛРБ), полученный от сервера отзыва лицензий, для обнаружения лицензий, которые нужно отозвать.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT ProcessLicenseRevocationResponse(
  [in]  BYTE  *pbSignedLRB,
  [in]  DWORD cbSignedLRB,
  [out] BYTE  **ppbSignedACK,
  [out] DWORD *pcbSignedACK
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пбсигнедлрб* \[ окне\]
</dt> <dd>

BLOB-объект отзыва лицензий (ЛРБ), полученный от сервера отзыва лицензий в ответ на запрос, созданный путем вызова метода [**креателиценсеревокатиончалленже**](iwmdrmlicensemanagement-createlicenserevocationchallenge.md) .

</dd> <dt>

*кбсигнедлрб* \[ окне\]
</dt> <dd>

Размер ЛРБ в байтах.

</dd> <dt>

*ппбсигнедакк* \[ заполняет\]
</dt> <dd>

Адрес указателя, получающего адрес подтверждения отзыва лицензии. Подтверждение должно быть отправлено службе отзыва лицензий. После завершения работы с этими данными необходимо освободить память, вызвав **CoTaskMemFree**.

</dd> <dt>

*пкбсигнедакк* \[ заполняет\]
</dt> <dd>

Адрес переменной, которая получает размер подтверждения в байтах.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |
