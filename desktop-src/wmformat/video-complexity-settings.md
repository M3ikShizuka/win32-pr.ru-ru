# <a name="video-complexity-settings"></a>Параметры сложности видео

Некоторые видеокодеки поддерживают несколько *уровней сложности*. Уровень сложности, используемый для кодирования потока, не влияет напрямую на скорость потока, но может повлиять на качество. Если скорость потока — мера размера сжатых образцов в ASF-файле, уровень сложности — это мера вычислительной мощности, необходимой для восстановления сжатых данных. Таким образом, уровень сложности обозначает вычислительную мощность воспроизводимой платформы, чем доступная пропускная способность.

Дополнительные сведения о работе с уровнями сложности см. в разделе [Настройка потоков видео](configuring-video-streams.md).
