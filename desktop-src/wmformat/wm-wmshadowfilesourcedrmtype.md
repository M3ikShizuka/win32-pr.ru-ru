# <a name="wmwmshadowfilesourcedrmtype-windows-media-format-11-sdk"></a>WM/вмшадовфилесаурцедрмтипе (пакет SDK для Windows Media Format 11)

Атрибут **WM/вмшадовфилесаурцедрмтипе** содержит тип Rights Management, который используется для защиты исходного файла, от которого ПОЛУЧЕН файл ASF.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмвмшадовфилесаурцедрмтипе

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

содержимое, которое существует в формате, отличном от ASF, и защищено с помощью некоторых видов управления правами, может быть преобразовано в Windowsный файл мультимедиа, защищенный Windows media DRM через процесс, называемый импортом лицензий. Такой файл ASF должен содержать этот атрибут, а также WM/Вмшадовфилесаурцефилетипе для мониторинга места, откуда поступило содержимое.
