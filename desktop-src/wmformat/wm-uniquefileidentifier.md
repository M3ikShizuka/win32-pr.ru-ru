# <a name="wmuniquefileidentifier"></a>WM/Уникуефилеидентифиер

Атрибут **WM/уникуефилеидентифиер** содержит уникальный идентификатор файла для содержимого.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмуникуефилеидентифиер

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ WM**

## <a name="remarks"></a>Комментарии

Уникальный идентификатор файла — это универсальная строка, которая может использоваться приложениями и службами для уникальной идентификации файла. Строка содержит произвольные значения, разделенные точкой с запятой. Не следует очищать этот атрибут. Можно добавлять значения и удалять собственные значения, но все остальные должны остаться без изменений.
