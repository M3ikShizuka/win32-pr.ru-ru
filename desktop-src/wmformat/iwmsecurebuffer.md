# <a name="iwmsecurebuffer-interface"></a>Интерфейс Ивмсекуребуффер

Интерфейс **ивмсекуребуффер** шифрует и расшифровывает указатели данных.

## <a name="members"></a>Элементы

Интерфейс **ивмсекуребуффер** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Ивмсекуребуффер** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмсекуребуффер** содержит следующие методы.



| Метод                                     | Описание                         |
|:-------------------------------------------|:------------------------------------|
| [**расшифровка;**](iwmsecurebuffer-decrypt.md) | Расшифровывает указатель данных.<br/> |
| [**Шифрование**](iwmsecurebuffer-encrypt.md) | Шифрует указатель данных.<br/> |
