# <a name="wmmediaclasssecondaryid"></a>WM/Медиакласссекондарид

Атрибут **WM/медиакласссекондарид** содержит идентификатор GUID вторичного класса мультимедиа.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиакласссекондарид

## <a name="data-type"></a>Тип данных

**\_GUID типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Для этого атрибута следует задать одно из значений, приведенных в следующей таблице.



| Идентификатор GUID вторичного класса                   | Описание                                                                                                                                                                             |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| "E0236BEB-C281-4EDE-A36D-7AF76A3D45B5" | Используется для файлов звуковой книги.                                                                                                                                                               |
| "3A172A13-2BD9-4831-835B-114F6A95943F" | Используется для звуковых файлов, содержащих произнесенные слова, но не для звуковых книг. Например, подпрограммы комедия.                                                                           |
| "6677DB9B-E5A0-4063-A1AD-ACEB52840CF1" | Используется для звуковых файлов, связанных с новостями.                                                                                                                                                    |
| "1B824A67-3F80-4E3E-9CDE-F7361B0F5F1B" | Используйте для звуковых файлов с разговором Показать содержимое.                                                                                                                                             |
| "1FE2E091-4E1E-40CE-B22D-348C732E0B10" | Используется для видеофайлов, связанных с новостями.                                                                                                                                                    |
| "D6DE1D88-C77C-4593-БФБК-9C61E8C373E3" | Используйте для видеофайлов, содержащих веб-экраны, короткие мультипликация, трейлеры фильмов и т. д. Это общий идентификатор для развлекательного видео, который не помещается в другую категорию. |
| "00033368-5009-4AC3-A820-5D2D09A4E7C1" | Используется для звуковых файлов, содержащих звуковые клипы из игр.                                                                                                                                  |
| "F24FF731-96FC-4D0F-A2F5-5A3483682B1A" | Используется для звуковых файлов, содержащих полные песни из звуковых дорожек игры. Если в файле кодируется только часть песни, используйте вместо этого идентификатор для звуковых клипов игры.                   |
| "E3E689E2-BA8C-4330-96DF-A0EEEFFA6876" | Используйте для видеофайлов с музыкальными видео.                                                                                                                                            |
| "B76628F4-300D-443D-9CB5-01C285109DAF" | Используйте для видеофайлов, содержащих общие сведения о домашнем видео.                                                                                                                                      |
| "A9B87FC9-BD47-4BF0-AC4F-655B89F7D868" | Используется для видеофайлов с функцией мультипликация.                                                                                                                                           |
| "BA7F258A-62F7-47A9-B21F-4651C42A000E" | Используется для видеофайлов, содержащих телевизионные передачи. Для веб-показов используйте более универсальный идентификатор.                                                                                  |
| "44051B5B-B103-4B5C-92AB-93060A9463F0" | Используется для видеофайлов с корпоративным видео. Например, записанные встречи или обучающие видеоматериалы.                                                                                      |
| "0B710218-8C0C-475E-AF73-4C41C0C8F8CE" | Используется для видеофайлов с домашним видео, сделанным из фотографий.                                                                                                                        |



 

При указании идентификатора вторичного класса файл также должен содержать атрибут идентификатора первичного класса.
