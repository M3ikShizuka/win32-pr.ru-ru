# <a name="iwmdrmencryptscatter-interface"></a>Интерфейс Ивмдрменкриптскаттер

Интерфейс **ивмдрменкриптскаттер** выполняет расшифровку и шифрование данных.

## <a name="members"></a>Элементы

Интерфейс **ивмдрменкриптскаттер** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Ивмдрменкриптскаттер** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмдрменкриптскаттер** содержит следующие методы.



| Метод                                                                | Описание                                                            |
|:----------------------------------------------------------------------|:-----------------------------------------------------------------------|
| [**енкриптскаттер**](iwmdrmencryptscatter-encryptscatter.md)         | Выполняет расшифровку и шифрование данных.<br/>                              |
| [**инитенкриптскаттер**](iwmdrmencryptscatter-initencryptscatter.md) | Инициализирует интерфейс **ивмдрменкриптскаттер** для использования.<br/> |
