# <a name="device-conformance-templates"></a>Шаблоны соответствия устройств

кодек Windows Media 9 Series поддерживает шаблоны соответствия устройств, которые задают диапазоны параметров конфигурации потока и алгоритмов кодеков. Каждый шаблон определяет диапазоны значений, соответствующих определенным устройствам.

Раньше производители оборудования, которые предоставили устройствам, способным воспроизводить файлы ASF, работали со своими стандартами. Это привело к широкому разнородному спектру возможностей на схожих устройствах, которые в настоящее время существуют.

с шаблонами соответствия устройств Windows кодеки мультимедиа устанавливают общий земля для аналогичных устройств. Производители оборудования могут указать, к каким шаблонам соответствуют устройства, а создателям содержимого — более уверенно нацелить их файлы на чтение устройств. Кроме того, приложения-проигрыватели легче определить, не подходит ли файл для устройства, прежде чем пытаться его воспроизвести.

Шаблон соответствия устройств определяется строкой, которая хранится в виде атрибута метаданных, связанного с потоком, к которому применяется шаблон. Список шаблонов и их строк и параметров см. в разделе [Параметры шаблона соответствия устройств](device-conformance-template-parameters.md).

шаблоны согласованности устройств поддерживаются для всех кодеков Windows media 9 и более поздних версий, за исключением видеокодека Windows Media Video 9 и кодека Windows media Audio 9 без потерь.
