# <a name="local-license-store"></a>Локальное хранилище лицензий

При доставке лицензии DRM на клиентский компьютер она помещается в локальное хранилище лицензий. Это защищенный файл на жестком диске, содержащий все лицензии, выданные пользователю вместе с другими данными DRM.

вы можете управлять данными в локальном хранилище лицензий несколькими ограниченными способами с помощью расширенных api-интерфейсов Windows Media DRM.

Помимо локального хранилища лицензий, некоторые объекты используют временное хранилище лицензий. Лицензия во временном хранилище существует только в течение короткого периода времени. Однако некоторые лицензии, которые начинаются во временном хранилище, можно сохранить в обычном локальном хранилище лицензий.
