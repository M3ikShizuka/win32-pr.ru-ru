# <a name="advanced-systems-format-asf-files"></a>Файлы в формате ASF

Все объекты пакета SDK для Windows Media Format работают с файлами, которые соответствуют структуре файлов в формате ASF. Эти файлы называются файлами ASF в этой документации. Дополнительные сведения о структуре файлов ASF см. [в разделе Обзор формата ASF](overview-of-the-asf-format.md).

Иногда термин *файл Windows Media* используется для описания файлов, созданных с помощью этого пакета SDK или других технологий Windows Media. Неточно вызывать файл ASF в файл Windows Media, если он не содержит только содержимое, закодированное с помощью кодеков Windows Media. Дополнительные сведения см. в руководстве по использованию расширений файлов WMA и WMV в разделе [рекомендации по расширению имен файлов](file-name-extension-guidelines.md) .
