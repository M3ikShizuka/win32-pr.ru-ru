# <a name="windows-media-drm-10-for-network-devices"></a>Windows Media DRM 10 для сетевых устройств

Приложение может передавать защищенное содержимое в цифровые приемники мультимедиа в сети с помощью Windows Media DRM 10 для сетевых устройств. Эта функция является протоколом для безопасной связи с сетевыми устройствами.

Пакет SDK Windows Media Format поддерживает Windows Media DRM 10 для сетевых устройств путем обслуживания базы данных сетевых устройств, проверки сетевого сходства этих устройств и преобразования содержимого, защищенного с помощью DRM, в защищенный формат транспорта.

> [!Note]  
> DRM не поддерживает версию этого пакета SDK на базе x64.
