# <a name="wmcategory"></a>WM/Category

Атрибут **WM/Category** содержит категорию содержимого. Нет спецификаций для категорий. Пользователям рекомендуется вводить собственные категории для организации содержимого.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмкатегори

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="example"></a>Пример



| Тип файла | Пример значения   |
|-----------|-----------------|
| звук;     | "Движущие музыку" |
| видео;     | "Домашние видео"   |
