# <a name="drm_crypto_type-enumeration"></a>\_ \_ Перечисление типов шифрования DRM

Перечисление **\_ \_ типа шифрования DRM** определяет типы криптографических алгоритмов, которые могут использоваться для шифрования содержимого.

## <a name="syntax"></a>Синтаксис


```C++
typedef enum DRM_CRYPTO_TYPE { 
  CRYPTO_TYPE_MCE  = 0
} ;
```



## <a name="constants"></a>Константы

<dl> <dt>

<span id="CRYPTO_TYPE_MCE"></span><span id="crypto_type_mce"></span>**\_тип шифрования \_ MCE**
</dt> <dd>

Задает 128-разрядное шифрование в режиме счетчика AES (AES).

</dd> </dl>
