# <a name="output-media-properties-object"></a>Объект свойств выходного носителя

Объект свойств выходного носителя используется для извлечения и задания выходного свойства. Объекты свойств выходных данных мультимедиа создаются для поддерживаемых выходных форматов потоков в файле, который загружается в объект модуля чтения. Для сжатых потоков свойства вывода определяются возможными выходными данными для десжатого кодека.

Объект свойств выходного носителя создается с помощью [**ивмреадер:: жетаутпутпропс**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmreader-getoutputprops) этот метод создает выходной объект свойств носителя, который содержит свойства выходного формата по умолчанию. Для выходных данных могут поддерживаться и другие форматы. Чтобы получить дополнительные выходные форматы, можно вызвать [**ивмреадер:: жетаутпутформаткаунт**](/previous-versions/windows/desktop/api/wmsdkidl/nf-wmsdkidl-iwmreader-getoutputformatcount) , чтобы получить количество поддерживаемых выходных форматов, а затем прокручивать их с помощью вызовов [**Ивмреадер:: жетаутпутформат**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmreader-getoutputformat). **Жетаутпутформат** создает выходной объект свойств мультимедиа, заполненный данными для выбранного формата выходных данных.

Объекты свойств выходных данных мультимедиа также можно создать с помощью синхронного модуля чтения. Все имена методов идентичны параметрам в модуле чтения, и все они предоставляются интерфейсом [**ивмсинкреадер**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmsyncreader) .

**Жетаутпутпропс** и **жетаутпутформат** устанавливают указатель на интерфейс [**ивмаутпутмедиапропс**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmoutputmediaprops) . Другие интерфейсы объекта выходных свойств мультимедиа можно получить, вызвав метод **QueryInterface** .

Каждый объект свойств выходного носителя поддерживает следующие интерфейсы.



| Интерфейс                                          | Описание                                                                                                |
|----------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**ивммедиапропс**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmmediaprops)             | Используется в качестве базового интерфейса для других интерфейсов свойств мультимедиа (входные, выходные и видео).             |
| [**ивмаутпутмедиапропс**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmoutputmediaprops) | Извлекает свойства выходных данных.                                                                     |
| [**ивмвидеомедиапропс**](/previous-versions/windows/desktop/api/Wmsdkidl/nn-wmsdkidl-iwmvideomediaprops)   | Управляет свойствами потока видео. Это необязательный интерфейс, доступный только для потоков видео. |
