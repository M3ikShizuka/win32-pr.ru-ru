# <a name="iwmdrmencrypt-interface"></a>Интерфейс Ивмдрменкрипт

Интерфейс **ивмдрменкрипт** предоставляет метод для шифрования данных на месте.

## <a name="members"></a>Элементы

Интерфейс **ивмдрменкрипт** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Ивмдрменкрипт** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмдрменкрипт** содержит следующие методы.



| Метод                                   | Описание                                 |
|:-----------------------------------------|:--------------------------------------------|
| [**Шифрование**](iwmdrmencrypt-encrypt.md) | Шифрует буфер данных на месте.<br/> |
