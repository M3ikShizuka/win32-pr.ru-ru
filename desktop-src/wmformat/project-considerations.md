# <a name="project-considerations"></a>Project Следует

необходимо убедиться, что конечный пользователь может правильно запускать приложения, созданные с помощью пакета SDK для Windows Media Format. В этом разделе описаны две вещи, которые необходимо учитывать при распространении приложений, расширений имен файлов и распространения программного обеспечения.

Необходимо использовать правильные расширения имен файлов для всех файлов, созданных в приложениях. Использование правильных расширений имен файлов упрощает распознавание файлов ASF и предотвращает путаницу при декодировании файлов.

Необходимо предоставить все распространяемые компоненты, необходимые для запуска приложений. Без правильных распространяемых пакетов пользователи не смогут использовать ваши приложения.

В следующих разделах подробно описаны расширения имен файлов и повторное распространение программного обеспечения.
