# <a name="wmparentalratingreason"></a>WM/Паренталратингреасон

Атрибут **WM/паренталратингреасон** содержит причины, по которым Родительский рейтинг назначается содержимому.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмпаренталратингреасон

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут используется для широковещательного телевидения и другого видеоматериала, которому назначена оценка родительского объекта.
