# <a name="wmpartofset"></a>WM/Партофсет

Атрибут **WM/партофсет** содержит номер части и общее количество частей в наборе, которому принадлежит файл. Два значения разделяются косой чертой.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмпартофсет

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="example"></a>Пример



| Тип файла | Пример значения |
|-----------|---------------|
| Все       | "2/3"         |
