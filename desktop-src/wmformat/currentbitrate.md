# <a name="currentbitrate"></a>куррентбитрате

Атрибут Куррентбитрате содержит текущую общую скорость потока активных потоков в файле.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмкуррентбитрате

## <a name="data-type"></a>Тип данных

**ВМТ, \_ тип \_ DWORD**

## <a name="remarks"></a>Remarks

Это закодированный атрибут.

Значение, полученное для **куррентбитрате** , отличается в зависимости от используемого объекта. В объекте чтения или синхронном модуле чтения значение представляет собой реальную скорость потока во время вызова. В объекте редактора метаданных значение является средней скоростью потока файла.

Фактическая скорость потока в файле равна скорости потока всех активных потоков плюс некоторые издержки. это значение, которое, например, отображается при воспроизведении файла с проигрыватель Windows Media.

Этот атрибут не может дублироваться на уровне файла. если этот атрибут используется для отдельного потока, он будет рассматриваться как пользовательские метаданные и не будет передавать свое нормальное значение объектам из пакета SDK Windows Media Format.
