# <a name="attributes-for-video-files"></a>Атрибуты для видеофайлов

В этом разделе перечислены атрибуты, обычно используемые для видеофайлов. Рекомендуется устанавливать атрибуты для файлов в соответствии с этими списками, чтобы обеспечить полную совместимость файлов с самыми разнообразными приложениями воспроизведения. Атрибуты в этом разделе перечислены в трех категориях: первичный, вторичный и третичный.

Основные атрибуты содержат самые основные сведения о файле. При создании видеофайлов для распространения это минимальный набор атрибутов, которые следует использовать.

Вторичные атрибуты содержат общие метаданные, которые важны, но не являются универсальными для всех видеофайлов.

При необходимости необходимо добавить третичные атрибуты, но не обязательно описывать файл.
