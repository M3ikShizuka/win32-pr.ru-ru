# <a name="actions-and-rights"></a>Действия и права

Если файл защищен с помощью Windows Media DRM, его использование полностью ограничено. Лицензии могут быть выпущены, чтобы пользователи могли использовать содержимое в конкретных целях. Каждый способ, которым пользователь может использовать содержимое, описывается действием. Например, действием является воспроизведение файла на компьютере.

Считается, что лицензия, позволяющая заданное действие, предоставляет право. Например, лицензия может предоставить право на воспроизведение содержимого на компьютере.

Действия и права ссылаются на те же способы использования содержимого. Разница заключается в том, что действие относится к использованию, а право указывает на разрешение. Несмотря на это различие, слова Action и right часто взаимозаменяемы в этой документации и других документах, описывающих DRM Windows Media.

Действия, управляемые правами DRM в Windows Media, перечислены в разделе [константы прав](rights-constants.md) этой документации.

Некоторые методы расширенных API клиента DRM Windows Media используют разные константы для ссылки на базовые права DRM. Например, метод [**ивмдрмлиценсекуери:: куерилиценсестате**](iwmdrmlicensequery-querylicensestate.md) использует набор строк, характерных для состояний лицензии. Хотя эти константы определяют разные строки, чем базовые константы прав, они ссылаются на те же права в лицензии.
