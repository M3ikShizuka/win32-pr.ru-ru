# <a name="sdk-versions"></a>Версии пакета SDK

в этой документации описывается пакет sdk для Windows media format 7 с помощью пакета sdk для Windows media format 11. Отдельные методы, структуры, функции и перечисления включают требования к версии в справочных разделах.

пакет SDK для Windows Media Format 11 поддерживается только в Windows Vista и Windows XP.

пакет SDK для Windows Media Format 9,5 поддерживается только в Windows XP. если вы пишете приложение на основе пакета SDK для Windows Media Format 9,5, ваше приложение не будет работать в других операционных системах.

для некоторых функций пакета SDK серии Windows Media Format 9 также требуется Windows XP. Отдельные справочные разделы, относящиеся к этим функциям, вызывают это требование.
