# <a name="sdk-versions"></a>Версии пакета SDK

В этой документации описывается пакет SDK для Windows Media Format 7 с помощью пакета SDK для Windows Media Format 11. Отдельные методы, структуры, функции и перечисления включают требования к версии в справочных разделах.

Пакет SDK Windows Media Format 11 поддерживается только в Windows Vista и Windows XP.

Пакет SDK Windows Media Format 9,5 поддерживается только в Windows XP. Если вы напишете приложение на основе пакета SDK Windows Media Format 9,5, ваше приложение не будет работать в других операционных системах.

Для некоторых функций пакета SDK для Windows Media Format 9 также требуется Windows XP. Отдельные справочные разделы, относящиеся к этим функциям, вызывают это требование.
