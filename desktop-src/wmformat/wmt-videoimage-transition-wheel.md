# <a name="wmt_videoimage_transition_wheel"></a>\_ \_ колесо перехода ВМТ \_ видеоимаже

Переход колесика показывает новый образ, поворачивая четыре линии вокруг общей точки вращения, например звезды колеса.

## <a name="parameters"></a>Параметры

В следующей таблице описаны параметры, используемые этим переходом, и перечислены члены структуры [**ВМТ \_ видеоимаже \_ SAMPLE2**](/previous-versions/windows/desktop/api/Wmsdkidl/ns-wmsdkidl-wmt_videoimage_sample2) , которой они назначены.




| Параметр | Член структуры | Описание | 
|-----------|------------------|-------------|
| Центр по оси X | <strong>fEffectPara0</strong> | Координата по оси X относительно видеокадра центра воздействия колесика. | 
| Центр по оси Y | <strong>fEffectPara1</strong> | Координата по оси Y относительно видеокадра центра воздействия колесика. | 
| Angle | <strong>fEffectPara2</strong> | Угол поворота (в градусах) периферийных элементов этого угла. Значение 0,0 указывает на старое изображение без выявила новый образ. Значение 90,0 указывает, что новый образ полностью отображен. Перемещение с 0,0 на 90,0 отображается с поворотом по часовой стрелке.<br /> | 
| Композиция | <strong>fEffectPara3</strong> | Задайте одно из следующих значений:<ul><li>0 — задает нормальную композицию, в которой предыдущее изображение является фоном, а текущее изображение — передним планом.</li><li>1 — указывает обратную композицию, в которой текущий рисунок является фоновым изображением, а предыдущее изображение — передний план.</li></ul> |
