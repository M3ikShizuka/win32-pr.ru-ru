# <a name="wmtoolversion"></a>WM/Тулверсион

Атрибут **WM/тулверсион** содержит номер версии приложения, используемого для создания файла. При использовании этого атрибута необходимо также включить имя приложения с помощью [**WM/ToolName**](wm-toolname.md).

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмтулверсион

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**
