# <a name="file-streams"></a>Файловые потоки

Файловый поток содержит один или несколько файлов данных. При передаче сведений о файле в виде потока необходимо прикрепить имя файла к каждому образцу в потоке. Сведения о присоединении связанных сведений к образцам см. в разделе [модули данных](data-unit-extensions.md).
