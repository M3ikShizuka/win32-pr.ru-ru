# <a name="creating-licenses-locally"></a>Создание лицензий локально

В определенных обстоятельствах, например при [импорте DRM](drm-import.md), можно создавать собственные лицензии. Windows Лицензии DRM для мультимедиа могут быть написаны несколькими разными способами, но для создания собственной лицензии необходимо использовать двоичную схему Extensible Media Rights (КСМР). Дополнительные сведения см. в разделе [Создание лицензии КСМР](building-an-xmr-license.md).

При создании лицензии ее можно добавить в локальное хранилище лицензий, вызвав метод [**ивмдрмлиценсеманажемент:: сторелиценсе**](iwmdrmlicensemanagement-storelicense.md) .
