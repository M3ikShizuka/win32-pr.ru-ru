# <a name="playlist-burning"></a>Запись списка воспроизведения

Windows Лицензии Media DRM 10 могут поддерживать копирование списков воспроизведения защищенных файлов на компакт-диск Red Book. Издатели лицензий могут ограничивать количество раз, когда содержимое может быть скопировано на компакт-диск в списке воспроизведения по общему числу раз и по числу раз, когда может быть скопирован конкретный список воспроизведения. дополнительные сведения о правах на запись списка воспроизведения в лицензии см. в документации по пакету SDK Windows Media rights Manager 10.

пакет SDK Windows Media Format позволяет быстро проверить все файлы в списке воспроизведения, чтобы убедиться, что они лицензированы для записи списка воспроизведения.

**Примечание** . DRM не поддерживает версию этого пакета SDK на базе x64.
