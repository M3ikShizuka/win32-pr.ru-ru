# <a name="wmmediaoriginalchannel"></a>WM/Медиаоригиналчаннел

Атрибут **WM/медиаоригиналчаннел** содержит канал, на котором было вещание содержимого.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиаоригиналчаннел

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Этот атрибут используется для трансляции телепередач.
