# <a name="is_trusted"></a>Является \_ доверенным

Атрибут **является \_ доверенным** атрибутом уровня файла, который указывает, является ли URL-адрес для получения лицензий в файле доверенным.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмтрустед

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Это закодированный атрибут.

Перед переходом к URL-адресу получения лицензии, содержащемуся в файле, защищенном DRM, приложение должно сначала проверить, имеет ли это свойство значение true. Если значение равно false, приложение должно уведомлять пользователя о том, что URL-адрес, возможно, был незаконным.

Этот атрибут не может дублироваться на уровне файла. если этот атрибут используется для отдельного потока, он будет рассматриваться как пользовательские метаданные и не будет передавать свое нормальное значение объектам из пакета SDK Windows Media Format.
