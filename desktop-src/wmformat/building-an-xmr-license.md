# <a name="building-an-xmr-license"></a>Создание лицензии КСМР

чтобы создать лицензию для обработки Windows Media DRM, необходимо использовать двоичную схему с правами расширяемого носителя (ксмр). КСМР — это схема для передачи прав на использование мультимедиа и ограничений, которые должны быть лицензированы отдельно.

важный материал в лицензии шифруется с помощью открытого ключа в коллекции сертификатов drm в Windows media, поэтому он виден только в подсистеме расширенного API клиента drm для Windows media. .

Вы обязаны убедиться, что структура лицензии и параметры политики действительны и согласованы с намерением издателя лицензии и соответствуют правилам соответствия.

чтобы получить полный набор объектов ксмр, которые должны присутствовать в лицензии, ознакомьтесь с разWindows правил соответствия импорту mac DRM.

Чтобы передать лицензию КСМР в подсистему DRM, вызовите метод [**ивмдрмлиценсеманажемент:: сторелиценсе**](iwmdrmlicensemanagement-storelicense.md) . Используйте следующий формат для передачи лицензии в параметре *бстрлиценсереспонсе* :


```C++
<LICENSERESPONSE>
    <LICENSE version="3.0.0.0">insert base64-encoded XMR license here</LICENSE>
</LICENSERESPONSE>
```



Эта строка должна быть в формате Юникода (UTF-16).
