# <a name="building-an-xmr-license"></a>Создание лицензии КСМР

Чтобы создать лицензию для обработки Windows Media DRM, необходимо использовать двоичную схему КСМР (Extensible Media Rights). КСМР — это схема для передачи прав на использование мультимедиа и ограничений, которые должны быть лицензированы отдельно.

Важный материал в лицензии шифруется с помощью открытого ключа в коллекции сертификатов DRM Windows Media, поэтому он доступен только для подсистемы расширенного API клиента Windows Media DRM. .

Вы обязаны убедиться, что структура лицензии и параметры политики действительны и согласованы с намерением издателя лицензии и соответствуют правилам соответствия.

Чтобы узнать полный набор объектов КСМР, которые должны присутствовать в лицензии, ознакомьтесь с разрядом правил соответствия импорта DRM в Windows Media.

Чтобы передать лицензию КСМР в подсистему DRM, вызовите метод [**ивмдрмлиценсеманажемент:: сторелиценсе**](iwmdrmlicensemanagement-storelicense.md) . Используйте следующий формат для передачи лицензии в параметре *бстрлиценсереспонсе* :


```C++
<LICENSERESPONSE>
    <LICENSE version="3.0.0.0">insert base64-encoded XMR license here</LICENSE>
</LICENSERESPONSE>
```



Эта строка должна быть в формате Юникода (UTF-16).
