# <a name="idrmstatuscallback-interface"></a>Интерфейс Идрмстатускаллбакк

Интерфейс **идрмстатускаллбакк** предоставляет метод для получения состояния асинхронных вызовов DRM.

## <a name="members"></a>Элементы

Интерфейс **идрмстатускаллбакк** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Идрмстатускаллбакк** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **идрмстатускаллбакк** содержит следующие методы.



| Метод                                          | Описание                                                      |
|:------------------------------------------------|:-----------------------------------------------------------------|
| [**OnStatus**](idrmstatuscallback-onstatus.md) | Получает сообщения о состоянии от асинхронных процессов.<br/> |
