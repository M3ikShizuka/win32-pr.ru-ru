# <a name="revocation-type-identifiers"></a>Идентификаторы типов отзыва

Константы, перечисленные в следующей таблице, определяют значения GUID, которые определяют различные типы отзыва.



| Константа                         | Описание                                                                                         |
|----------------------------------|-----------------------------------------------------------------------------------------------------|
| \_ \_ приложение типа отзыва WMDRM \_     | Определяет сведения об отзыве, относящиеся к сертификатам приложений.                           |
| \_ \_ устройство типа отзыва WMDRM \_  | Определяет сведения об отзыве, относящиеся к сертификатам устройств.                                |
| \_тип ОТЗЫВА WMDRM \_ \_ кардеа  | Определяет сведения об отзыве, относящиеся к сертификатам Windows Media DRM для сетевых устройств. |
| \_тип ОТЗЫВА WMDRM \_ \_ ревинфо | Не документировано в этом выпуске.                                                                     |
| \_тип ОТЗЫВА WMDRM \_ — \_ GRL     | Не документировано в этом выпуске.                                                                     |
| \_тип ОТЗЫВА WMDRM \_ \_ HDCP    | Не документировано в этом выпуске.                                                                     |
