# <a name="drm_lasignaturerootcert"></a>\_ЛАСИГНАТУРЕРУТЦЕРТ DRM

Атрибут **\_ ласигнатурерутцерт DRM** содержит корневой сертификат в цепочке сертификатов, который используется для проверки подлинности сертификата, содержащегося в [**\_ ласигнатуреликсрвцерт DRM**](drm-lasignaturelicsrvcert.md).

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдрм \_ ласигнатурерутцерт

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Это свойство можно задать с помощью метода [**ивмдрмвритер:: сетдрматтрибуте**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) . Он недоступен для объекта Reader.
