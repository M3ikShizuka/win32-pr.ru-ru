# <a name="high-resolution-audio-support"></a>Поддержка High-Resolution Audio

кодек Windows media audio 9 Professional и кодек Windows media audio 9 без потерь поддерживают звуковые файлы с высоким разрешением. Звук в высоком разрешении определяется как звуковой формат с более чем 16 битами на выборку. Количество битов, используемых для отдельного звукового образца, влияет на точность закодированной звуковой волны.

поскольку форматы звука в кодировке PCM используются в Windows операционных системах в течение длительного времени, некоторые из Windows функций устарели и не могут обрабатывать звук PCM с высоким разрешением. В результате форматы PCM с высоким разрешением представлены с использованием разных структур, чем стандартный звук. Дополнительные сведения см. [в разделе Работа с аудио в High-Resolution PCM](working-with-high-resolution-pcm-audio.md).
