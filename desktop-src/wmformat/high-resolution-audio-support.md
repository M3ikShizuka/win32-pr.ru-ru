# <a name="high-resolution-audio-support"></a>Поддержка High-Resolution Audio

Как кодек Windows Media Audio 9 Professional, так и кодек Windows Media Audio 9 без потерь поддерживают звуковые файлы высокого разрешения. Звук в высоком разрешении определяется как звуковой формат с более чем 16 битами на выборку. Количество битов, используемых для отдельного звукового образца, влияет на точность закодированной звуковой волны.

Поскольку форматы звука, закодированные в PCM, использовались в операционных системах Windows в течение длительного времени, некоторые функции Windows устарели и не могут обрабатывать звук PCM с высоким разрешением. В результате форматы PCM с высоким разрешением представлены с использованием разных структур, чем стандартный звук. Дополнительные сведения см. [в разделе Работа с аудио в High-Resolution PCM](working-with-high-resolution-pcm-audio.md).
