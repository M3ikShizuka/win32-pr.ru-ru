# <a name="importing-license-and-key-material"></a>Импорт лицензии и материала ключа

если у вас есть мультимедийное содержимое, зашифрованное с помощью сторонней системы защиты содержимого, и вы хотите импортировать лицензию и материал ключа в Windows media DRM, выполните следующие действия:

1.  получите Windows коллекцию сертификатов виртуальной машины DRM Media, вызвав метод [**ивмдрмсекурити:: жетмачинецертификате**](iwmdrmsecurity-getmachinecertificate.md) .
2.  Проанализируйте коллекцию сертификатов, убедившись, что она подписана правильно и проверена на известный корневой открытый ключ Майкрософт. Коллекция сертификатов соответствует схеме КСМР. Дополнительные сведения см. в разделе [Создание лицензии КСМР](building-an-xmr-license.md).
3.  Необязательно. Извлеките список отзыва, вызвав метод [**ивмдрмсекурити:: жетревокатиондата**](iwmdrmsecurity-getrevocationdata.md) .
4.  Необязательно. Убедитесь, что сертификат в коллекции не отозван. Дополнительные сведения см. в разделе [Проверка отзыва сертификата](checking-certificate-revocation.md).
5.  создайте лицензию в формате ксмр, который представляет требования политики для импортируемого содержимого и содержит соответствующий материал ключа DRM для Windows мультимедиа. Дополнительные сведения см. в разделе [Создание лицензии КСМР](building-an-xmr-license.md).
6.  передайте лицензию ксмр в систему DRM Windows Media для обработки с помощью метода [**ивмдрмлиценсеманажемент:: сторелиценсе**](iwmdrmlicensemanagement-storelicense.md) .

> [!Note]  
> сведения о схеме ксмр будут предоставлены вам при лицензировании Windows Media DRM.
