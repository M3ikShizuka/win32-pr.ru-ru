# <a name="bit-rate"></a>Скорость

Скорость потока — это объем данных в секунду, доставляемый из ASF-файла. Показатели скорости в битах в секунду (бит/с) или килобитах в секунду (кбит/с). Скорость потока часто путают с пропускной способностью, которая представляет собой измерение мощности передачи данных в сети. Пропускная способность также измеряется в бит/с и кбит/с.

Пакет Windows Media Format SDK можно использовать для создания приложений, обеспечивающих потоковую передачу содержимого на основе Windows Media из расположений в Интернете или интрасети. При потоковой передаче данных по сети или через Интернет скорость работы конечных пользователей важна. Если пропускная способность, доступная в сети, меньше, чем скорость файла ASF, воспроизведение файла будет каким-то образом прервано. Как правило, недостаточная пропускная способность приведет к пропуску любого из этих примеров или приостановки воспроизведения в процессе буферизации данных.

Каждому файлу ASF во время создания назначается значение скорости, основанное на типе и количестве потоков, включенных в используемый профиль. Отдельные потоки имеют собственные скорости. Скорость потока может быть постоянной (исходные данные сжимаются таким образом, чтобы поддерживать постоянный поток данных с одинаковой частотой) или переменной (исходные данные сжимаются таким образом, чтобы поддерживать одинаковое качество в течение всего времени, даже если это может означать неравномерное выполнение потока данных). Различные типы скорости разрядов могут применяться к разным потокам в одном файле.

Вы можете кодировать одно и то же содержимое в несколько различных потоков, каждый из которых имеет разную скорость. Затем можно настроить потоки, чтобы они были взаимоисключающими. Это позволяет создать один файл, который можно передать пользователям с различной пропускной способностью. Эта функция называется многоразрядной скоростью или MBR.
