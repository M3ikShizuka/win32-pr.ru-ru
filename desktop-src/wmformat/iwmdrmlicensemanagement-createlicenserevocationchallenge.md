# <a name="iwmdrmlicensemanagementcreatelicenserevocationchallenge-method"></a>Метод Ивмдрмлиценсеманажемент:: Креателиценсеревокатиончалленже

Метод **креателиценсеревокатиончалленже** создает запрос отзыва лицензии.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT CreateLicenseRevocationChallenge(
  [in]  BYTE  *pbMachineID,
  [in]  DWORD cbMachineID,
  [in]  BYTE  *pbChallenge,
  [in]  DWORD cbChallenge,
  [out] BYTE  **ppbChallengeOutput,
  [out] DWORD *pcbChallengeOutput
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пбмачинеид* \[ окне\]
</dt> <dd>

Идентификатор компьютера, указанный пользователем. Это значение используется для запроса лицензии на сервере и должно соответствовать любому формату, используемому сервером лицензирования.

</dd> <dt>

*кбмачинеид* \[ окне\]
</dt> <dd>

Размер (в байтах) идентификатора компьютера.

</dd> <dt>

*пбчалленже* \[ окне\]
</dt> <dd>

Определяемые пользователем данные запроса. Эти данные, помимо идентификатора компьютера, используются для запроса сервера лицензирования на отзыв лицензий.

</dd> <dt>

*кбчалленже* \[ окне\]
</dt> <dd>

Размер (в байтах) данных запроса.

</dd> <dt>

*ппбчалленжеаутпут* \[ заполняет\]
</dt> <dd>

Адрес указателя, получающего адрес вывода запроса. Этот буфер представляет собой данные, отправляемые в службу отзыва лицензий. После завершения работы с этими данными необходимо освободить память, вызвав **CoTaskMemFree**.

</dd> <dt>

*пкбчалленжеаутпут* \[ заполняет\]
</dt> <dd>

Адрес переменной, получающей размер выделенных выходных данных запроса в байтах.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Метод возвращает **значение HRESULT**. Допустимые значения включают, но не ограничиваются, значения, приведенные в следующей таблице.



| Код возврата                                                                          | Описание                      |
|--------------------------------------------------------------------------------------|----------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl> | Метод выполнен успешно.<br/> |
