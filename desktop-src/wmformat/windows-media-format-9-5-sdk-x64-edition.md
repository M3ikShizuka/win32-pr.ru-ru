# <a name="windows-media-format-95-sdk-x64-edition"></a>Windows Media Format 9,5 SDK x64 Edition

эта документация относится к новому Windows Media format 9,5 sdk x64 Edition, а также к предыдущей, 32-разрядной версии пакета sdk Windows Media format.

В следующем списке приведены изменения пакета SDK для 64-разрядной версии.

-   DRM не поддерживается. Файлы, связанные с DRM (включая примеры), были удалены из установки пакета SDK.
-   Windows Диспетчер устройств носителя не является частью установки пакета SDK на основе x64.
-   примеры и заголовки, поддерживающие интерфейсы кодеков Windows Media Audio и Video, не являются частью установки пакета SDK на основе x64.
-   Локализованные системные профили не являются частью установки пакета SDK на основе x64.

примечания в разделе справки по программированию этой документации приводят к функциям, которые не поддерживаются в Windows Media Format 9,5 SDK x64 Edition.
