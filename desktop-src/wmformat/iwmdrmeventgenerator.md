# <a name="iwmdrmeventgenerator-interface"></a>Интерфейс Ивмдрмевентженератор

Интерфейс **ивмдрмевентженератор** является расширением интерфейса **имфмедиаевентженератор** , который предоставляет метод для отмены асинхронных операций.

## <a name="members"></a>Элементы

Интерфейс **ивмдрмевентженератор** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Ивмдрмевентженератор** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивмдрмевентженератор** содержит следующие методы.



| Метод                                                                    | Описание                                   |
|:--------------------------------------------------------------------------|:----------------------------------------------|
| [**канцеласинкоператион**](iwmdrmeventgenerator-cancelasyncoperation.md) | Отменяет асинхронную операцию.<br/> |
