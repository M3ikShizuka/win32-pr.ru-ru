# <a name="implementing-network-functionality"></a>Реализация функций сети

в этом разделе описывается, как использовать сетевые функции, доступные в пакете SDK для Windows Media Format. приложение может использовать эти функции для чтения потоков asf из сети, вещания потоков asf в сеть или отправки asf-потоков в точку публикации на сервере, на котором работает службы Windows Media.

В следующих разделах описываются сетевые возможности этого пакета SDK.



| Section                                                                    | Описание                                                                                                          |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [Обзор сетевых интерфейсов](overview-of-networking-interfaces.md) | Имена и описание каждого интерфейса, поддерживающего сетевые методы.                                                 |
| [Чтение данных ASF по сети](reading-asf-data-over-a-network.md)     | Описывает, как воспроизводить файлы из источника сети и как настроить параметры сети для объекта Reader.    |
| [Отправка данных ASF по сети](sending-asf-data-over-a-network.md)     | описывает, как рассылать данные asf по сети или передавать данные asf в точку публикации на Windows сервере мультимедиа. |
| [сетевые Параметры по умолчанию](default-networking-settings.md)             | Содержит краткий справочник по параметрам по умолчанию, используемым пакетом SDK.                                                 |
