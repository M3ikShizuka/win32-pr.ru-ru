# <a name="implementing-network-functionality"></a>Реализация функций сети

В этом разделе описывается, как использовать сетевые функции, доступные в пакете SDK формата Windows Media. Приложение может использовать эти функции для чтения потоков ASF из сети, вещания потоков ASF в сеть или отправки ASF-потоков в точку публикации на сервере, на котором работают службы Windows Media.

В следующих разделах описываются сетевые возможности этого пакета SDK.



| Section                                                                    | Описание                                                                                                          |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [Обзор сетевых интерфейсов](overview-of-networking-interfaces.md) | Имена и описание каждого интерфейса, поддерживающего сетевые методы.                                                 |
| [Чтение данных ASF по сети](reading-asf-data-over-a-network.md)     | Описывает, как воспроизводить файлы из источника сети и как настроить параметры сети для объекта Reader.    |
| [Отправка данных ASF по сети](sending-asf-data-over-a-network.md)     | Описывает, как рассылать данные ASF по сети или передавать данные ASF в точку публикации на сервере Windows Media. |
| [Параметры сети по умолчанию](default-networking-settings.md)             | Содержит краткий справочник по параметрам по умолчанию, используемым пакетом SDK.                                                 |
