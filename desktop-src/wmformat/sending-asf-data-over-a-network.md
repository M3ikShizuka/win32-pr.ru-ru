# <a name="sending-asf-data-over-a-network"></a>Отправка данных ASF по сети

пакет SDK для Windows Media Format поддерживает два разных способа отправки данных ASF по сети:

-   Приложение может транслировать данные ASF по протоколу HTTP.
-   приложение может передать данные ASF в точку публикации на Windows сервере мультимедиа. Затем сервер распространяет данные.

Эти функции описаны в следующих разделах:

-   [Трансляция данных ASF](broadcasting-asf-data.md)
-   [Отправка данных ASF в точку публикации](sending-asf-data-to-a-publishing-point.md)
