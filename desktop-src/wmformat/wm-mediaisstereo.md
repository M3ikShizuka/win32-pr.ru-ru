# <a name="wmmediaisstereo"></a>WM/Медиаисстерео

Атрибут **WM/медиаисстерео** указывает, является ли содержимое широковещательным (стерео).

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиаисстерео

## <a name="data-type"></a>Тип данных

**\_bool типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Этот атрибут используется для трансляции телепередач.
