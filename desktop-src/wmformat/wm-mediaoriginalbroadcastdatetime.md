# <a name="wmmediaoriginalbroadcastdatetime"></a>WM/Медиаоригиналброадкастдатетиме

Атрибут **WM/медиаоригиналброадкастдатетиме** содержит дату и время вещания содержимого.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиаоригиналброадкастдатетиме

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Этот атрибут используется для трансляции телепередач или радио.
