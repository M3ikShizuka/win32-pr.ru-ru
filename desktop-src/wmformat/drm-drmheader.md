# <a name="drm_drmheader"></a>\_ДРМХЕАДЕР DRM

**DRM \_ Дрмхеадер** — это базовая строка для атрибутов дрмхеадер, таких как DRM \_ хеадеркэйид.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмдрм \_ DRM \_ дрмхеадер

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Remarks

Эту строку можно использовать для создания пользовательских свойств Дрмхеадер. Он не является атрибутом самого себя.
