# <a name="wmprovidercopyright"></a>WM/Провидеркопиригхт

Атрибут **WM/провидеркопиригхт** содержит заявление об авторских правах поставщика мультимедиа, который доставляет содержимое.

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

Нет.
