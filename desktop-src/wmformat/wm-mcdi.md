# <a name="wmmcdi"></a>WM/МКДИ

Атрибут **WM/мкди** содержит идентификатор музыкального компакт-диска. Это двоичный дамп оглавления с компакт-диска, который используется для уникальной идентификации компакт-диска.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммкди

## <a name="data-type"></a>Тип данных

**\_ \_ двоичный тип ВМТ**

## <a name="remarks"></a>Remarks

Этот атрибут совместим с кадром ID3, МКДИ. Для спецификации ID3 для кадра МКДИ требуется, чтобы для каждого файла существовала только одна такая рамка, а также существовала допустимая ТРКК рамка. Редактор метаданных не выполняет никаких проверок для этих правил. Кроме того, размер WM/МКДИ не ограничен 804 байтами, как и кадр МКДИ ID3.
