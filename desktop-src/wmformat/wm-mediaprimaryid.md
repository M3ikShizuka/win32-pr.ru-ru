# <a name="wmmediaclassprimaryid"></a>WM/Медиакласспримарид

Атрибут **WM/медиакласспримарид** содержит идентификатор GUID основного класса мультимедиа.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвммедиакласспримарид

## <a name="data-type"></a>Тип данных

**\_GUID типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Для этого атрибута следует задать одно из значений, приведенных в следующей таблице.



| GUID первичного класса                     | Описание                                                  |
|----------------------------------------|--------------------------------------------------------------|
| "D1607DBC-E323-4BE2-86A1-48A42A28441E" | Используется для музыкальных файлов. Не используйте для звука, который не является музыкальным. |
| "DB9830BD-3AB3-4FAB-8A37-1A995F7FF74B" | Используется для видеофайлов.                                         |
| "01CD0F29-DA4E-4157-897B-6275D50C4F11" | Используется для звуковых файлов, которые не являются музыкой.                      |
| "FCF24A76-9A57-4036-990D-E35DD8B244E1" | Используется для файлов, не являющихся ни аудио, ни видео.               |



 

При указании идентификатора первичного класса необходимо также задать идентификатор дополнительного класса, чтобы уточнить тип содержимого в файле.
