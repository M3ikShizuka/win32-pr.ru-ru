# <a name="wmauthorurl"></a>WM/Аусорурл

Атрибут **WM/аусорурл** содержит адрес веб-сайта автора.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвмаусорурл

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Адрес, указанный этим атрибутом, должен соответствовать исполнителю или автору в атрибуте [**Author**](author.md) в файле.
