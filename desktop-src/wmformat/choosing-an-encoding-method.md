# <a name="choosing-an-encoding-method"></a>Выбор метода кодирования

Некоторые кодеки, например кодек Windows Media Video 9, поддерживают несколько методов кодирования. Метод кодирования, выбранный для потока, будет зависеть от способа доставки потока. В следующей таблице описано, когда использовать различные методы кодирования.



| Метод Encoding                | Описание                                                                                                                                                                                       |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 — постоянная скорость передачи (CBR) | Единственный параметр для потоковой передачи в реальном времени. Кодируется в прогнозируемую скорость и обеспечивает самое низкое качество всех методов кодирования.                                                                    |
| 2. Передача CBR                     | Используйте для файлов, которые будут передаваться по сети клиентскому модулю чтения, но не будут рассылаться из источника в реальном времени. Кодируется в прогнозируемую скорость, но имеет более высокое качество, чем 1-проход CBR. |
| 1 — скорость передачи переменной (VBR) | Используется, если необходимо указать качество закодированных выходных данных. Обеспечивает наиболее стабильное качество всех методов кодирования. Используйте только для локальных файлов или для загрузки.                        |
| 2-проходные с неограниченными     | Используйте, если необходимо указать пропускную способность, но колебания, окружающие указанную пропускную способность, приемлемы. Для локальных файлов или только для загрузки.                                                    |
| 2 — пропускная мощность — ограниченная       | Используйте в тех же условиях, что и без ограничений, но если необходимо указать максимальную скорость в течение некоторого времени. Для локальных файлов или только для загрузки.                                                |
