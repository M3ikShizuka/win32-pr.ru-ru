# <a name="asf-file-features"></a>Функции файлов ASF

основным предназначением пакета SDK для Windows Media format является предоставление поддержки инкапсуляции цифровых мультимедийных данных в файлах ASF и доставки мультимедиа в клиентское приложение. Сценарии доставки могут сильно различаться в зависимости от приложения, но все они используют структуру файлов ASF между созданием и доставкой. Файлы ASF соответствуют хорошо определенной, но очень гибкой структуре. Дополнительные сведения о структуре файлов ASF см. в разделе [Обзор формата ASF](overview-of-the-asf-format.md).

Подробные сведения о данных в файле ASF приведены в спецификации ASF, которую можно загрузить с [веб-сайта корпорации Майкрософт](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).

пакет SDK для Windows Media Format обеспечивает поддержку функций спецификации ASF в основном через профиль, используемый для создания файла. Дополнительные сведения о профилях см. в разделе [Профили](profiles.md).
