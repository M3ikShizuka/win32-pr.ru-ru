# <a name="wmencodingsettings"></a>WM/Енкодингсеттингс

Атрибут **WM/енкодингсеттингс** содержит параметры, используемые для кодирования содержимого.

## <a name="global-constant"></a>Глобальная константа

g \_ всзвменкодингсеттингс

## <a name="data-type"></a>Тип данных

**\_Строка типа \_ ВМТ**

## <a name="remarks"></a>Комментарии

Нет стандартного формата для указания параметров кодировки. Однако предпочтительнее описать параметры кодировки на английском языке, а не включать дамп параметров.
