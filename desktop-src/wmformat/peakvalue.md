# <a name="peakvalue"></a>пеаквалуе

Атрибут **пеаквалуе** содержит 16-битное значение амплитуды, обозначающее пиковый уровень громкости звукового содержимого. При использовании [**аверажелевел**](averagelevel.md)этот атрибут используется для нормализации. Нормализация — это процесс настройки уровня громкости воспроизведения звуковых файлов таким образом, чтобы громкость файлов воспроизведена на одном и том же уровне, а средний объем для каждого из них одинаков.

## <a name="global-constant"></a>Глобальная константа

g \_ всзпеаквалуе

## <a name="data-type"></a>Тип данных

**ВМТ, \_ тип \_ DWORD**

## <a name="remarks"></a>Комментарии

Этот атрибут задается объектом модуля записи на основе информации из кодека. Значение автоматически задается только для потоков, сжатых с помощью одного из аудиокодеков Windows Media Audio.

**Пеаквалуе** не предназначен только для чтения. Однако если файл будет воспроизводиться проигрывателем Windows Media, это значение изменять не следует. Проигрыватель Windows Media использует его для нормализации уровней файлов в списке воспроизведения.
