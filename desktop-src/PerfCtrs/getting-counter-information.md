# <a name="getting-counter-information"></a>Получение сведений о счетчике

Чтобы получить сведения о счетчике, например тип счетчика, текст справки и путь, вызовите функцию [**пдхжеткаунтеринфо**](/windows/desktop/api/Pdh/nf-pdh-pdhgetcounterinfoa) . Функция возвращает сведения о счетчике в виде [**\_ \_ информационной структуры счетчика PDH**](/windows/desktop/api/Pdh/ns-pdh-pdh_counter_info_a) .
