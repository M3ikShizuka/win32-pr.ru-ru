# <a name="getting-the-size-of-a-log-file"></a>Получение размера файла журнала

[**Пдхжетлогфилесизе**](/windows/desktop/api/Pdh/nf-pdh-pdhgetlogfilesize) возвращает размер указанного файла журнала в байтах. Эта функция используется, если необходимо выделить память для буфера, который будет принимать содержимое файла журнала.
