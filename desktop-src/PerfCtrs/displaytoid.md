# <a name="displaytoid"></a>дисплайтоид

Таблица **дисплайтоид** связывает удобную для пользователя строку, отображаемую системным монитором, с идентификатором GUID, хранящимся в других таблицах.

В таблице **дисплайтоид** определены следующие поля:

-   **Идентификатор GUID:** Уникальный идентификатор, созданный для журнала. Это поле является первичным ключом этой таблицы.
-   **RunID:** Зарезервировано для внутреннего использования.
-   **DisplayString:** Имя файла журнала, отображаемое в системном мониторе.
-   **Логстарттиме:** Время начала процесса ведения журнала в формате гггг-мм-дд чч: мм: СС: nnn.
-   **Логстоптиме:** Время остановки процесса ведения журнала в формате гггг-мм-дд чч: мм: СС: nnn. Несколько файлов журнала с одинаковым значением **DisplayString** можно различать с помощью значений в полях и **логстарттиме** . Значения в полях **логстарттиме** и **логстоптиме** также позволяют быстро получить доступ к общему времени сбора.
-   **Нумберофрекордс:** Количество выборок, сохраненных в таблице для каждой коллекции журналов.
-   **Минутестаутк:** Значение, используемое для преобразования данных строки, хранящихся в формате UTC, в местное время.
-   **Тимезоненаме:** Имя часового пояса, в который были собраны данные. Если вы собираете или записали данные из файла, собранного на системах в собственном часовом поясе, это поле будет содержать расположение.

**Примечание**  .  До Windows Vista Наборы сборщиков данных хранились в реестре по адресу

**\_ \_ \\ \\ \\ \\ Запросы журнала SysmonLog CURRENTCONTROLSET Services System \\ для локального компьютера hKey**

. Указанные выше поля не соответствуют значениям в реестре. В Windows Vista Наборы сборщиков данных не хранятся в реестре.
