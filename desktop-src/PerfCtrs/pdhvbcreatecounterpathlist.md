# <a name="pdhvbcreatecounterpathlist-function"></a>Функция Пдхвбкреатекаунтерпаслист

Функция **пдхвбкреатекаунтерпаслист** отображает диалоговое окно обзора счетчиков производительности, позволяющее пользователю выбрать несколько счетчиков производительности. После этого каждый выбранный путь счетчика должен быть считан с помощью функции [**пдхвбжеткаунтерпасфромлист**](pdhvbgetcounterpathfromlist.md) .

> [!IMPORTANT]
> Функция, описанная в этом разделе, может быть изменена или недоступна в будущем. Вместо этого корпорация Майкрософт рекомендует использовать функции, описанные в разделе [функции счетчиков производительности](performance-counters-functions.md).

Функция Пдхвбкреатекаунтерпаслист ( \_ ByVal Детаиллевел в качестве типа Long, \_ ByVal Каптионстринг в качестве строки \_ )

## <a name="parameters"></a>Параметры

<dl> <dt>

*детаиллевел* 
</dt> <dd>

Типы счетчиков, отображаемых в диалоговом окне. Используйте одно из следующих значений.



| Значение                                                                                                                                                                               | Значение                                                                                                                                                 |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="PERF_DETAIL_ADVANCED"></span><span id="perf_detail_advanced"></span><dl> <dt>**\_Дополнительные сведения о производительности \_**</dt> </dl> | Счетчики, которые могут быть понятны расширенному пользователю, а также счетчики новичков и пользователей.<br/>                                            |
| <span id="PERF_DETAIL_EXPERT"></span><span id="perf_detail_expert"></span><dl> <dt>**\_эксперт подробных сведений о производительности \_**</dt> </dl>       | Счетчики, которые, скорее всего, будут понятны эксперту и разработчику программного обеспечения, а также счетчики для новичков и опытных пользователей.<br/> |
| <span id="PERF_DETAIL_NOVICE"></span><span id="perf_detail_novice"></span><dl> <dt>**\_сведения о производительности — \_ новичок**</dt> </dl>       | Только счетчики, которые может понять пользователь новичка.<br/>                                                                                  |
| <span id="PERF_DETAIL_WIZARD"></span><span id="perf_detail_wizard"></span><dl> <dt>**\_Мастер сведений о производительности \_**</dt> </dl>       | Все счетчики в системе.<br/>                                                                                                                  |



 

</dd> <dt>

*каптионстринг* 
</dt> <dd>

Строковая переменная, содержащая текст, который будет отображаться в строке заголовка диалогового окна.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Функция возвращает число выбранных пользователем путей счетчиков.
