# <a name="pdhvbgetonecounterpath-function"></a>Функция Пдхвбжетонекаунтерпас

Функция **пдхвбжетонекаунтерпас** отображает диалоговое окно, позволяющее пользователю просматривать доступные счетчики производительности и выбирать один счетчик. Выбранный счетчик возвращается в переменной *PathString* . Переменная *PathString* должна быть распределена и инициализирована перед вызовом этой функции, а размер измерения должен быть указан переменной *PathLength* .

> [!IMPORTANT]
> Функция, описанная в этом разделе, может быть изменена или недоступна в будущем. Вместо этого корпорация Майкрософт рекомендует использовать функции, описанные в разделе [функции счетчиков производительности](performance-counters-functions.md).

Функция Пдхвбжетонекаунтерпас ( \_ ByVal PathString в качестве строки, \_ ByVal PathLength как Long, ByVal \_ детаиллевел как Long, \_ ByVal каптионстринг как String \_ ) как long

## <a name="parameters"></a>Параметры

<dl> <dt>

*PathString* 
</dt> <dd>

Инициализированная строковая переменная, используемая для получения пути счетчика, выбранного пользователем.

</dd> <dt>

*PathLength* 
</dt> <dd>

Длина инициализированного PathString.

</dd> <dt>

*детаиллевел* 
</dt> <dd>

Типы счетчиков, отображаемых в диалоговом окне. Этот параметр может принимать одно из указанных ниже значений.



| Значение                                                                                                                                                                               | Значение                                                                                                                                                 |

| <span id="PERF_DETAIL_ADVANCED"></span><span id="perf_detail_advanced"></span><dl> <dt>**\_Дополнительные сведения о производительности \_**</dt> </dl> | Счетчики, которые могут быть понятны расширенному пользователю, а также счетчики новичков и пользователей.<br/>                                            |
| <span id="PERF_DETAIL_EXPERT"></span><span id="perf_detail_expert"></span><dl> <dt>**\_эксперт подробных сведений о производительности \_**</dt> </dl>       | Счетчики, которые, скорее всего, будут понятны эксперту и разработчику программного обеспечения, а также счетчики для новичков и опытных пользователей.<br/> |

*каптионстринг*

## <a name="return-value"></a>Возвращаемое значение

Функция возвращает число символов, записанных в буфер *PathString* .
