# <a name="determining-whether-a-counter-is-located-on-a-computer"></a>Определение того, находится ли счетчик на компьютере

Чтобы определить, установлен ли счетчик на определенном компьютере, вызовите [**пдхвалидатепас**](/windows/desktop/api/Pdh/nf-pdh-pdhvalidatepatha) с полным путем к счетчику. Функция возвращает значение "ошибка", \_ Если счетчик находится на указанном компьютере.

[**Пдхвалидатепас**](/windows/desktop/api/Pdh/nf-pdh-pdhvalidatepatha) проверяет весь путь; Таким образом, если какой либо объект, экземпляр или счетчик в пути не существует, возвращаемое значение указывает на это. **Пдхвалидатепас** проверяет указанный путь счетчика в следующем порядке: компьютер, объект производительности, экземпляр и счетчик.
