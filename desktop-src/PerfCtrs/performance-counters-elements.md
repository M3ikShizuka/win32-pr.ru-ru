# <a name="performance-counters-elements"></a>Элементы счетчиков производительности

Схема счетчиков производительности определяет следующие элементы:

-   [**Counter (CounterSet)**](performance-counters-counter--counterset--element.md)
-   [**Каунтераттрибуте (Каунтераттрибутес)**](performance-counters-counterattribute--counterattributes--element.md)
-   [**Каунтераттрибутес (счетчик)**](performance-counters-counterattributes--counter--element.md)
-   [**Счетчики**](performance-counters-counters-element.md)
-   [**CounterSet (поставщик)**](performance-counters-counterset--provider--element.md)
-   [**поставщик (счетчики)**](performance-counters-provider--counters--element.md)
-   [**Структура (структуры)**](performance-counters-struct--structs--element.md)
-   [**структуры (CounterSet)**](performance-counters-structs--counterset--element.md)

В этом разделе содержатся имена элементов, используемых в манифесте. Однако, чтобы получить подробные сведения для каждого элемента, см. сложный тип, содержащий элемент. Корневым элементом раздела счетчиков производительности в манифесте является элемент [**Counters**](performance-counters-counters-element.md) .
