# <a name="struct-structs-element"></a>Элемент структуры (структуры)

Определяет структуру, содержащую значения счетчиков в родительском наборе счетчиков.

``` syntax
<xs:element name="struct"
    type="man:struct"
 />
```

Элемент **struct** определяется сложным типом [**structs**](performance-counters-structs-complex-type.md) .

## <a name="requirements"></a>Требования

|-------------------------------------|------------------------------------------------------|

| Минимальная версия сервера<br/> | Windows Только для \[ настольных приложений сервера 2008\]<br/> |
