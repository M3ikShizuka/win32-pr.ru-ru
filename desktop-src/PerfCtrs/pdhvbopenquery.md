# <a name="pdhvbopenquery-function"></a>Функция Пдхвбопенкуери

Функция **пдхвбопенкуери** создает и инициализирует уникальную структуру запроса, которая используется для управления сбором данных о производительности.

> [!IMPORTANT]
> Функция, описанная в этом разделе, может быть изменена или недоступна в будущем. Вместо этого корпорация Майкрософт рекомендует использовать функции, описанные в разделе [функции счетчиков производительности](performance-counters-functions.md).

Функция Пдхвбопенкуери ( \_ ByVal Куерихандле как long \_ )

## <a name="parameters"></a>Параметры

<dl> <dt>

*куерихандле* 
</dt> <dd>

Переменная, которая была удалена (равна 0) перед вызовом функции и, если функция выполнена успешно, содержит уникальный идентификатор созданного и открытого запроса. Этот маркер используется при последующих вызовах других функций PDH для обнаружения запроса.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция завершается успешно, она возвращает **длинное** целое число, равное \_ успешному выполнению, и новый маркер в переменной *куерихандле* .

Если функция завершается ошибкой, возвращаемое значение является [кодом системной ошибки](/windows/desktop/Debug/system-error-codes) или [кодом ошибки PDH](pdh-error-codes.md). Ниже приведены возможные значения.



| Код возврата                                                                                                     | Описание                                                  |
|-----------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------|
| <dl> <dt>**\_Недопустимый \_ аргумент PDH**</dt> </dl>           | Аргумент является недопустимым или неверным.<br/>             |
| <dl> <dt>**\_ \_ сбой выделения памяти \_ PDH**</dt> </dl> | Не удалось выделить временный буфер памяти.<br/> |
