# <a name="performance-counters"></a>Счетчики производительности

Windows Счетчики производительности обеспечивают высокоуровневый уровень абстракции, обеспечивающий единообразный интерфейс для сбора различных типов системных данных, таких как ЦП, память и использование диска. Системные администраторы часто используют счетчики производительности для мониторинга систем на наличие проблем с производительностью или поведением. Разработчики программного обеспечения часто используют счетчики производительности для проверки использования ресурсов их программ.
