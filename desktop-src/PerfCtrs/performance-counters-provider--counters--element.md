# <a name="provider-counters-element"></a>Элемент Provider (Counters)

Определяет поставщик и счетчики, которые он предоставляет.

``` syntax
<xs:element name="provider"
    type="man:provider"
 />
```

Элемент **provider** определяется сложным типом [**счетчиков**](performance-counters-counters-complex-type.md) .

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Только для \[ классических приложений Windows Vista\]<br/>       |
| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2008\]<br/> |
