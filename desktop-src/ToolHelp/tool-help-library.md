# <a name="tool-help-library"></a>Библиотека справки по инструментам

Функции, предоставляемые библиотекой справки по инструментам, упрощают получение сведений о выполняющихся в данный момент приложениях. Эти функции предназначены для упрощения создания средств, особенно отладчиков.
