# <a name="flush-method-of-the-control-class"></a>Метод Flush класса Control

Очищает перенаправляемые буферы для потока и записывает в файл конфигурации все буферизованные данные.

## <a name="syntax"></a>Синтаксис


```mof
void Flush();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
