# <a name="boot-event-collector-wmi-provider"></a>Поставщик WMI сборщика событий загрузки

## <a name="purpose"></a>Назначение

поставщик WMI сборщика событий загрузки предоставляет доступ к сведениям о подключении и конфигурации для функции сбора событий установки и загрузки на сервере Windows. Это позволяет просматривать список текущих подключений и журнал подключений между сервером сборщика и его конечными компьютерами. Кроме того, этот поставщик позволяет управлять конфигурацией сервера сборщика.

> [!Note]  
> Поставщик WMI сборщика событий загрузки реализуется в BEvtCol.exe.
