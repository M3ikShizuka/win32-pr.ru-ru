# <a name="setconfiguration-method-of-the-control-class"></a>Метод Сетконфигуратион класса Control

Задайте новую активную конфигурацию сборщика.

## <a name="syntax"></a>Синтаксис


```mof
Uint32 SetConfiguration(
  [in]  string Config,
  [in]  Uint32 OldTimestampLow,
  [in]  Uint32 OldTimestampHigh,
  [out] Uint32 NewTimestampLow,
  [out] Uint32 NewTimestampHigh,
  [out] string ErrorString,
  [out] string WarningString,
  [out] string InfoString,
  [out] uint32 ErrorType
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Конфигурация* \[ окне\]
</dt> <dd>

Конфигурация для активации.

</dd> <dt>

*Олдтиместамплов* \[ окне\]
</dt> <dd>

Младшие биты метки времени, указывающие, когда была задана текущая активная конфигурация. Проверка атомарности включена, если для этого свойства не задано значение 0.

</dd> <dt>

*Олдтиместамфигх* \[ окне\]
</dt> <dd>

Старшие биты метки времени, указывающие, когда была задана текущая активная конфигурация. Проверка атомарности включена, если для этого свойства не задано значение 0.

</dd> <dt>

*Невтиместамплов* \[ заполняет\]
</dt> <dd>

При возврате из этого метода этот параметр содержит младшие биты метки времени, указывающие, когда была задана новая конфигурация. Проверка атомарности включена, если для этого свойства не задано значение 0.

</dd> <dt>

*Невтиместамфигх* \[ заполняет\]
</dt> <dd>

При возврате из этого метода этот параметр содержит старшие биты метки времени, указывающие, когда была задана новая конфигурация. Проверка атомарности включена, если для этого свойства не задано значение 0.

</dd> <dt>

*ErrorString* \[ заполняет\]
</dt> <dd>

Когда этот метод возвращает значение, если произошла ошибка, этот параметр содержит описание ошибки.

</dd> <dt>

*Варнингстринг* \[ заполняет\]
</dt> <dd>

При возврате из этого метода этот параметр содержит все сообщения о предупреждениях для операции.

</dd> <dt>

*Инфостринг* \[ заполняет\]
</dt> <dd>

При возврате из этого метода этот параметр содержит сведения о новой активной конфигурации.

</dd> <dt>

*ErrorType* \[ заполняет\]
</dt> <dd>

Когда этот метод возвращает значение, если произошла ошибка, этот параметр указывает тип ошибки.

<dt>

0
</dt> <dd>

Отсутствует новая конфигурация.

</dd> <dt>

1
</dt> <dd>

Недопустимый формат новой конфигурации.

</dd> <dt>

2
</dt> <dd>

Недопустимая новая конфигурация.

</dd> <dt>

3
</dt> <dd>

Произошла ошибка, вызванная открытием сокета.

</dd> <dt>

4
</dt> <dd>

Произошла ошибка записи в файл.

</dd> <dt>

5
</dt> <dd>

Произошла ошибка атомарности.

</dd> </dl> </dd> </dl>
