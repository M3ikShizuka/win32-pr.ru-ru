# <a name="control-class"></a>Control - класс

Управление экземпляром сборщика. Требуются права администратора (BA).

Следующий пример синтаксиса — упрощенный MOF-код, который включает все наследуемые свойства.

## <a name="syntax"></a>Синтаксис

``` syntax
[Provider("BootEventCollectorWmiProvider"), AMENDMENT]
class Control
{
};
```

## <a name="members"></a>Члены

Класс **Control** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Класс **Control** имеет следующие методы.



| Метод                                                         | Описание                                                                                                                                                                                                                                                                                                                                                               |
|:---------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Контрольная точка**](control-checkpoint.md)                       | Если текущая конфигурация является результатом операции отмены/повтора или восстановления, помечает ее как, если она задана явным образом, чтобы журнал сохранял время, когда оно было задано, и файл резервной копии будет создан для него при следующем изменении конфигурации. Если текущая конфигурация уже задана явно, не оказывает никакого влияния. Возвращает 1 при успешном выполнении, 0 при ошибке.<br/> |
| [**думпдиагностикс**](control-dumpdiagnostics.md)             | Выводит данные диагностики в журнал.<br/>                                                                                                                                                                                                                                                                                                                  |
| [**фастшутдовн**](control-fastshutdown.md)                   | Быстро останавливает работу сборщика, отменяя все данные в очереди.<br/>                                                                                                                                                                                                                                                                                                    |
| [**Очистка**](control-flush.md)                                 | Очистите буферы сервера пересылки.<br/>                                                                                                                                                                                                                                                                                                                                   |
| [**GetConfiguration**](control-getconfiguration.md)           | Считывает активную конфигурацию сборщика.<br/>                                                                                                                                                                                                                                                                                                                |
| [**исконфигуратионекуал**](control-isconfigurationequal.md)   | Сравните аргумент с активной конфигурацией сборщика. Возвращает 1, если они совпадают, 0 — если нет.<br/>                                                                                                                                                                                                                                                 |
| [**листбаккупс**](control-listbackups.md)                     | Возврат списка сохраненных файлов конфигурации резервной копии, которые можно восстановить.<br/>                                                                                                                                                                                                                                                                                  |
| [**Повторить**](control-redo.md)                                   | Сброс активной конфигурации сборщика из более позднего файла резервной копии (определяется путем перехода от текущей исходной метки времени). Если конфигурация отменена, это означает, что изменение отменено. Возвращает 1 при успешном выполнении, 0 при ошибке.<br/>                                                                                                    |
| [**RestoreFile**](control-restorefile.md)                     | Восстановите активную конфигурацию сборщика из файла резервной копии. Возвращает 1 при успешном выполнении, 0 при ошибке.<br/>                                                                                                                                                                                                                                                        |
| [**ресторефромтиме**](control-restorefromtime.md)             | Восстановление активной конфигурации сборщика из файла резервной копии, выбранного по метке времени. Возвращает 1 при успешном выполнении, 0 при ошибке.<br/>                                                                                                                                                                                                                               |
| [**SetConfiguration**](control-setconfiguration.md)           | Задайте новую активную конфигурацию сборщика. Возвращает 1 при успешном выполнении, 0 при ошибке.<br/>                                                                                                                                                                                                                                                                           |
| [**Закрытия**](control-shutdown.md)                           | Останавливает сборщик. Если сборщик работает в качестве службы, то остановка службы является лучшим подходом.<br/>                                                                                                                                                                                                                                                     |
| [**Отменить**](control-undo.md)                                   | Восстановите активную конфигурацию сборщика из предыдущего файла резервной копии (определяется путем возврата из текущей исходной метки времени). Если конфигурация была только что установлена, это означает отмену этого изменения. Последовательные вызовы будут отменяться до более ранних и более ранних конфигураций. Возвращает 1 при успешном выполнении, 0 при ошибке.<br/>                           |
| [**валидатеконфигуратион**](control-validateconfiguration.md) | Проверьте правильность текста конфигурации, не настроив его как активный. Возвращает 1 при успешном выполнении, 0 при ошибке.<br/>                                                                                                                                                                                                                                                     |
