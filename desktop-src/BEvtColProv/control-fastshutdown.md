# <a name="fastshutdown-method-of-the-control-class"></a>Метод Фастшутдовн класса Control

Быстро останавливает работу сборщика, отменяя все данные в очереди.

## <a name="syntax"></a>Синтаксис


```mof
void FastShutdown();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
