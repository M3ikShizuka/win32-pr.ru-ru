# <a name="checkpoint-method-of-the-control-class"></a>Метод Checkpoint класса Control

Если текущая конфигурация является результатом операции отмены/повтора или восстановления, помечает ее как, если она задана явным образом, чтобы журнал сохранял время, когда оно было задано, и файл резервной копии будет создан для него при следующем изменении конфигурации. Если текущая конфигурация уже задана явно, не оказывает никакого влияния.

## <a name="syntax"></a>Синтаксис


```mof
Uint32 Checkpoint(
  [in]  Uint32 OldTimestampLow,
  [in]  Uint32 OldTimestampHigh,
  [out] string ErrorString,
  [out] string WarningString,
  [out] string InfoString,
  [out] uint32 ErrorType
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Олдтиместамплов* \[ окне\]
</dt> <dd>

Метка времени, когда была задана текущая конфигурация. Это Младшая часть [**fileTime**](/windows/desktop/api/minwinbase/ns-minwinbase-filetime).
