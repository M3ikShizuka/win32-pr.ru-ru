# <a name="shutdown-method-of-the-control-class"></a>Метод Shutdown класса Control

Останавливает сборщик. Если сборщик работает в качестве службы, то остановка службы является лучшим подходом.

## <a name="syntax"></a>Синтаксис


```mof
void Shutdown();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
