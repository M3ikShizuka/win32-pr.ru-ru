# <a name="functions-programming-reference"></a>Функции (Справочник по программированию)

Доступные функции Ксинпут.

## <a name="in-this-section"></a>В этом разделе

| Раздел                                                                                 | Описание                                                                                                                                             |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ксинпутенабле**](/windows/desktop/api/XInput/nf-xinput-xinputenable)<br/>                                       | Задает состояние отчетов Ксинпут.<br/>                                                                                                          |
| [**ксинпутжетаудиодевицеидс**](/windows/desktop/api/XInput/nf-xinput-xinputgetaudiodeviceids)<br/>                 | Получает идентификаторы звукового устройства для отображения звука и записи звука, связанные с головным телефоном, подключенным к указанному контроллеру.<br/> |
| [**ксинпутжетбаттеринформатион**](/windows/desktop/api/XInput/nf-xinput-xinputgetbatteryinformation)<br/>         | Извлекает тип батареи и состояние оплаты беспроводного контроллера.<br/>                                                                       |
| [**ксинпутжеткапабилитиес**](/windows/desktop/api/XInput/nf-xinput-xinputgetcapabilities)<br/>                     | Получает возможности и функции подключенного контроллера.<br/>                                                                           |
| [**ксинпутжетдсаундаудиодевицегуидс**](/windows/desktop/api/XInput/nf-xinput-xinputgetdsoundaudiodeviceguids)<br/> | Получает идентификаторы GUID устройства отображения звука и записи звука, связанные с гарнитурой, подключенной к указанному контроллеру.<br/>          |
| [**ксинпутжеткэйстроке**](/windows/desktop/api/XInput/nf-xinput-xinputgetkeystroke)<br/>                           | Получает событие ввода игрового планшета.<br/>                                                                                                             |
| [**ксинпутжетстате**](/windows/desktop/api/XInput/nf-xinput-xinputgetstate)<br/>                                   | Извлекает текущее состояние указанного контроллера.<br/>                                                                                     |
| [**ксинпутсетстате**](/windows/desktop/api/XInput/nf-xinput-xinputsetstate)<br/>                                   | Отправляет данные на подключенный контроллер. Эта функция используется для активации функции вибрации контроллера.<br/>                              |
