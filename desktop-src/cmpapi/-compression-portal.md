# <a name="compression-api"></a>API сжатия

## <a name="purpose"></a>Назначение

API сжатия предоставляет алгоритмы сжатия Windows MSZIP, XPRESS, XPRESS \_ Хаффа и лзмс. Это позволяет разработчикам приложений Windows управлять версиями, службами и расширять предоставляемые алгоритмы сжатия.

## <a name="developer-audience"></a>Аудитория разработчиков

API сжатия предназначен для использования профессиональными разработчиками C/C++ приложений Windows. API сжатия предоставляет алгоритмы сжатия без потерь, используемые Windows при использовании общедоступного интерфейса и API пользовательского режима. API сжатия — это рекомендуемый способ управления алгоритмами сжатия для разработчиков Windows. Эта функция 64-бит на 64-разрядной версии Windows и 32-bit в 32-bit Windows.

## <a name="run-time-requirements"></a>Требования к среде выполнения

API сжатия доступен, начиная с Windows 8 или Windows Server 2012.
