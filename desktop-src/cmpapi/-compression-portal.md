# <a name="compression-api"></a>API сжатия

## <a name="purpose"></a>Назначение

API сжатия предоставляет алгоритмы сжатия Windows MSZIP, xpress, xpress \_ хаффа и лзмс. это позволяет разработчикам Windows приложений управлять версиями, службами и расширять предоставляемые алгоритмы сжатия.

## <a name="developer-audience"></a>Аудитория разработчиков

API сжатия предназначен для использования профессиональными разработчиками C/C++ Windows приложений. api сжатия предоставляет алгоритмы сжатия без потерь, используемые Windows хотя открытый интерфейс и API пользовательского режима. API сжатия — это рекомендуемый метод для Windows разработчикам управлять алгоритмами сжатия. эта функция 64-бит на 64-бит Windows и 32-bit на Windows 32-bit.

## <a name="run-time-requirements"></a>Требования к среде выполнения

API сжатия доступен, начиная с Windows 8 или Windows Server 2012.
