# <a name="id3dx11effectstringvariable-interface"></a>Интерфейс ID3DX11EffectStringVariable

Интерфейс строковых переменных обращается к строковой переменной.

## <a name="members"></a>Элементы

Интерфейс **ID3DX11EffectStringVariable** наследует от [**ID3DX11EffectVariable**](id3dx11effectvariable.md). **ID3DX11EffectStringVariable** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ID3DX11EffectStringVariable** содержит следующие методы.



| Метод                                                               | Описание                         |
|:---------------------------------------------------------------------|:------------------------------------|
| [**GetString**](id3dx11effectstringvariable-getstring.md)           | Получите строку.<br/>          |
| [**жетстрингаррай**](id3dx11effectstringvariable-getstringarray.md) | Получение массива строк.<br/> |



 

## <a name="remarks"></a>Примечания

> [!Note]  
> Пакет SDK для DirectX не предоставляет никаких скомпилированных двоичных файлов для эффектов. Для создания приложения типа Effects необходимо использовать исходный текст Effects 11. Дополнительные сведения об использовании источника Effects 11 см. в разделе [различия между эффектами 10 и эффекты 11](d3d11-graphics-programming-guide-effects-differences.md).
