# <a name="tiled-resources"></a>Мозаичные ресурсы

Мозаичные ресурсы можно рассматривать как крупные логические ресурсы, использующие небольшие объемы физической памяти.

В этом разделе описывается, зачем требуются мозаичные ресурсы и как создаются и используются мозаичные ресурсы.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                   | Описание                                                                                                                                                                                                                                         |
|-----------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Зачем нужны мозаичные ресурсы?](why-are-tiled-resources-needed-.md)<br/>       | Требуется выполнить мозаичное выделение ресурсов, так что память с графическим процессором не тратится на хранение областей поверхностей, к которым известно приложение, и оборудование может понять, как выполнять фильтрацию по смежным плиткам.<br/>     |
| [Создание мозаичных ресурсов](creating-tiled-resources.md)<br/>                     | Ресурсы мозаичного заполнения создаются путем указания флага D3D11 ресурса с назначением [**\_ \_ \_ мозаики**](/windows/desktop/api/D3D11/ne-d3d11-d3d11_resource_misc_flag) при создании ресурса.<br/>                                                                                          |
| [API-интерфейсы мозаичного ресурса](tiled-resource-apis.md)<br/>                               | Интерфейсы API, описанные в этом разделе, работают с мозаичными ресурсами и пулом плиток.<br/>                                                                                                                                                              |
| [Доступ конвейера к мозаичным ресурсам](pipeline-access-to-tiled-resources.md)<br/> | Ресурсы мозаичного заполнения можно использовать в представлениях ресурсов шейдера (SRV), представлениях целевого объекта отрисовки (РТВ), представления трафарета глубины (DSV) и неупорядоченных представлениях доступа (UAV), а также в некоторых точках привязки, где не используются представления, например привязки буфера вершин. <br/> |
| [Мозаичные ресурсы. уровни компонентов](tiled-resources-features-tiers.md)<br/>         | Direct3D 11,2 предоставляет поддержку мозаичных ресурсов на двух уровнях с использованием [**значений \_ \_ \_ уровня D3D11 мозаичных ресурсов**](/windows/desktop/api/D3D11/ne-d3d11-d3d11_tiled_resources_tier) . <br/>                                                                                         |
