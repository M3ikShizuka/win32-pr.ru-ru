# <a name="shader-reference-direct3d-11-graphics"></a>Справочник по шейдеру (графика Direct3D 11)

API Direct3D определяет несколько элементов API, которые помогают создавать программируемые шейдеры и управлять ими. Шейдеры — это исполняемые программы, которые запрограммированы исключительно с помощью HLSL.
