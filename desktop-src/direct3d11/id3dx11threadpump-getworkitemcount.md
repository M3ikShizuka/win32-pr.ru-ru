# <a name="id3dx11threadpumpgetworkitemcount-method"></a>Метод ID3DX11ThreadPump:: Жетворкитемкаунт

> [!Note]  
> библиотека служебной программы D3DX (D3DX 9, D3DX 10 и D3DX 11) является устаревшей для Windows 8 и не поддерживается для приложений магазина Windows.

 

Возвращает количество рабочих элементов в конвейере потока.

## <a name="syntax"></a>Синтаксис


```C++
UINT GetWorkItemCount();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Тип: **[ **uint**](/windows/desktop/WinProg/windows-data-types)**

Количество рабочих элементов, поставленных в конвейер потоков.
