# <a name="d3d11_rect"></a>D3D11 \_ Rect

D3D11 \_ Rect объявляется следующим образом:

``` syntax
typedef RECT D3D11_RECT;
```

Дополнительные сведения о структуре этого прямоугольника GDI см. в разделе [**Rect**](/previous-versions//dd162897(v=vs.85)).

## <a name="remarks"></a>Примечания

Эта структура используется для прямоугольных прямоугольников [**ссылку ID3D11DeviceContext:: рсжетсЦиссорректс**](/windows/desktop/api/D3D11/nf-d3d11-id3d11devicecontext-rsgetscissorrects) и [**ссылку ID3D11DeviceContext:: рссетсЦиссорректс**](/windows/desktop/api/D3D11/nf-d3d11-id3d11devicecontext-rssetscissorrects).
