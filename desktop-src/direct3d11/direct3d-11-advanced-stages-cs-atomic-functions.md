# <a name="atomic-functions"></a>Атомарные функции

Чтобы получить доступ к новому типу ресурса или общей памяти, используйте блокируемую встроенную функцию. Блокируемые функции гарантированно работают в атомарном режиме. То есть они гарантированно выполняются в запрограммированном порядке. В этом разделе перечислены атомарные функции.

-   [**интерлоккедадд**](/windows/desktop/direct3dhlsl/interlockedadd)
-   [**интерлоккедмин**](/windows/desktop/direct3dhlsl/interlockedmin)
-   [**интерлоккедмакс**](/windows/desktop/direct3dhlsl/interlockedmax)
-   [**Взаимоблокировка**](/windows/desktop/direct3dhlsl/interlockedor)
-   [**интерлоккеданд**](/windows/desktop/direct3dhlsl/interlockedand)
-   [**интерлоккедксор**](/windows/desktop/direct3dhlsl/interlockedxor)
-   [**интерлоккедкомпаресторе**](/windows/desktop/direct3dhlsl/interlockedcomparestore)
-   [**интерлоккедкомпариксчанже**](/windows/desktop/direct3dhlsl/interlockedcompareexchange)
-   [**интерлоккедексчанже**](/windows/desktop/direct3dhlsl/interlockedexchange)
