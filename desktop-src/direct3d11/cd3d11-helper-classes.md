# <a name="cd3d11-helper-structures"></a>Вспомогательные структуры CD3D11

Direct3D 11 определяет несколько вспомогательных структур, которые можно использовать для создания структур Direct3D. Эти вспомогательные структуры ведут себя как классы C++.

| [**\_Поле CD3D11**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_box)<br/>                                                  | Представляет поле и предоставляет удобные методы для создания рамок.<br/>                                                    |
| [**CD3D11 \_ \_ трафарета \_ глубины**](/windows/win32/api/d3d11/ns-d3d11-cd3d11_depth_stencil_desc)<br/>                  | Представляет структуру состояния с набором элементов глубины и предоставляет удобные методы для создания структур-шаблонов глубины.<br/> |
