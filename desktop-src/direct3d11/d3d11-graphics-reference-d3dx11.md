# <a name="d3dx-11-reference"></a>Справочник по D3DX 11

В этом разделе описывается API D3DX 11.

> [!NOTE]  
> Библиотека служебной программы D3DX (D3DX 9, D3DX 10 и D3DX 11) является устаревшей для Windows 8 и не поддерживается для Microsoft Store приложений. Дополнительные сведения о рекомендуемых заменах для D3DX11 см. в разделе " [жить без D3DX](https://walbourn.github.io/living-without-d3dx/)". Для более старых проектов можно использовать пакет NuGet [Microsoft. дкссдк. D3DX](https://www.nuget.org/packages/Microsoft.DXSDK.D3DX) , не полагаясь на УСТАРЕВШИЙ пакет SDK DirectX или директсетуп.
