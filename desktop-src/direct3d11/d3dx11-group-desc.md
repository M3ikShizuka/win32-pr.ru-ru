# <a name="d3dx11_group_desc-structure"></a>\_ \_ Структура DESC группы D3DX11

Описывает группу эффектов.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct _D3DX11_GROUP_DESC {
  LPCSTR Name;
  UINT   Techniques;
  UINT   Annotations;
} D3DX11_GROUP_DESC;
```
