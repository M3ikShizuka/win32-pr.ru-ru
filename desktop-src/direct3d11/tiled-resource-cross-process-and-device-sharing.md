# <a name="tiled-resource-cross-process-and-device-sharing"></a>Мозаичная обработка ресурсов и совместное использование устройств

Пулы могут совместно использоваться разными процессами — точно так же как традиционные ресурсы. Мозаичные ресурсы, ссылающиеся на пулы плиток, нельзя совместно использовать на устройствах и процессах. Но отдельные процессы могут создавать собственные мозаичные ресурсы, которые сопоставляются с пулами мозаичных элементов, которые являются общими для этих ресурсов.

Размер общих пулов плиток изменить нельзя.
