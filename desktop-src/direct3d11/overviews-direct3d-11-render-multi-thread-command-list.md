# <a name="command-list"></a>Список команд

Список команд — это последовательность команд GPU, которые можно записать и воспроизвести. Список команд может повысить производительность, уменьшая объем издержек, создаваемых средой выполнения.

Используйте список команд в следующих сценариях.

-   В пределах одного кадра отрисовывает часть сцены в одном потоке, записывая другую часть сцены во втором потоке. В конце кадра воспроизводите список записанных команд в первом потоке. Этот подход используется для масштабирования сложных задач отрисовки в нескольких потоках или ядрах.
-   Передайте список команд перед его отображением (например, во время загрузки уровня) и эффективно воспроизводите его позже в сцене. Эта оптимизация хорошо работает, когда вам нужно будет визуализировать что-то часто.

Список команд является неизменяемым и предназначен для записи и воспроизведения в ходе отдельного выполнения приложения. Список команд не предназначен для предварительной записи выполнения игры и загрузки с носителя, так как нет способа сохранить список.

Список команд должен быть записан отложенным контекстом, но его можно воспроизводить только в непосредственных контекстах. Отложенные контексты могут создавать списки команд одновременно.

-   Сведения о записи списка команд см. в разделе [как записать список команд](overviews-direct3d-11-render-multi-thread-command-list-record.md).
-   Чтобы воспроизвести список команд, см. раздел [как воспроизвести список команд](overviews-direct3d-11-render-multi-thread-command-list-play.md).
-   При использовании списка команд производительность зависит от объема поддержки, реализованной в драйвере. Сведения о проверке поддержки драйверов см. [в разделе как проверить наличие поддержки драйверов](overviews-direct3d-11-render-multi-thread-support.md).
