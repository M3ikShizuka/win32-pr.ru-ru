# <a name="unified-memory-architecture"></a>Архитектура единой памяти

Запрос на то, поддерживает ли архитектура унифицированной памяти (верхнюю память), может помочь определить способ обработки некоторых ресурсов.

Логическое значение, заданное драйвером, может быть считано из структуры [**D3D11 \_ \_ Data \_ D3D11 \_ OPTIONS2**](/windows/desktop/api/D3D11/ns-d3d11-d3d11_feature_data_d3d11_options2) , чтобы определить, поддерживает ли оборудование верхнюю область.

В приложениях, работающих в самой операционной системе, может потребоваться больше ресурсов с включенным доступом к ЦП, чем если он недоступен. Область памяти позволяет приложениям избегать копирования данных ресурсов, а не только эффективно для графических адаптеров, не использующих верхнюю часть.
