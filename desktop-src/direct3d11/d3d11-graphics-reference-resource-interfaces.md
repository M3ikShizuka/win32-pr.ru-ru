# <a name="resource-interfaces-direct3d-11-graphics"></a>Интерфейсы ресурсов (графика Direct3D 11)

Существует ряд интерфейсов для двух основных типов ресурсов: буферов и текстур. Существуют также интерфейсы для представлений ресурсов.

Приложение использует представление для привязки ресурса к этапу конвейера. Представление определяет, как к ресурсу можно получить доступ во время подготовки к просмотру. В этом разделе описываются интерфейсы представления.


## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                       | Описание                                                                                                                                                                                            |
|-----------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ID3D11Buffer**](/windows/desktop/api/D3D11/nn-d3d11-id3d11buffer)<br/>                             | Интерфейс буфера обращается к ресурсу буфера, который является неструктурированной памятью. Буферы обычно хранят данные вершин или индексов.<br/>                                                                  |
| [**ID3D11DepthStencilView**](/windows/desktop/api/D3D11/nn-d3d11-id3d11depthstencilview)<br/>         | Интерфейс представления глубины-трафарета обращается к ресурсу текстуры во время тестирования шаблона глубины.<br/>                                                                                                    |
| [**ID3D11RenderTargetView**](/windows/desktop/api/D3D11/nn-d3d11-id3d11rendertargetview)<br/>         | Интерфейс представления рендеринга определяет целевые подресурсы подготовки к просмотру, к которым можно получить доступ во время подготовки к просмотру.<br/>                                                                             |
| [**ID3D11RenderTargetView1**](/windows/desktop/api/D3D11_3/nn-d3d11_3-id3d11rendertargetview1)<br/>       | Интерфейс представления рендеринга представляет собой подресурсы подготовки к просмотру, к которым можно получить доступ во время подготовки к просмотру.<br/>                                                                             |
| [**ID3D11Resource**](/windows/desktop/api/D3D11/nn-d3d11-id3d11resource)<br/>                         | Интерфейс ресурсов предоставляет общие действия для всех ресурсов.<br/>                                                                                                                              |
| [**ID3D11ShaderResourceView**](/windows/desktop/api/D3D11/nn-d3d11-id3d11shaderresourceview)<br/>     | Интерфейс отображения шейдера-ресурсов определяет подресурсы, к которым может получить доступ шейдер во время подготовки к просмотру. Примерами ресурсов шейдера являются константный буфер, буфер текстуры и текстура.<br/>  |
| [**ID3D11ShaderResourceView1**](/windows/desktop/api/D3D11_3/nn-d3d11_3-id3d11shaderresourceview1)<br/>   | Интерфейс шейдера-представления ресурсов представляет подресурсы, к которым может получить доступ шейдер во время подготовки к просмотру. Примерами ресурсов шейдера являются константный буфер, буфер текстуры и текстура.<br/> |
| [**ID3D11Texture1D**](/windows/desktop/api/D3D11/nn-d3d11-id3d11texture1d)<br/>                       | Интерфейс текстуры 1D обращается к данным шаг текселя, которые представляют собой структурированную память.<br/>                                                                                                                     |
| [**ID3D11Texture2D**](/windows/desktop/api/D3D11/nn-d3d11-id3d11texture2d)<br/>                       | Интерфейс двухмерной текстуры управляет шаг текселя данными, которые представляют собой структурированную память.<br/>                                                                                                                      |
| [**ID3D11Texture2D1**](/windows/desktop/api/D3D11_3/nn-d3d11_3-id3d11texture2d1)<br/>                     | Интерфейс двухмерной текстуры представляет шаг текселя данные, которые представляют собой структурированную память.<br/>                                                                                                                   |
| [**ID3D11Texture3D**](/windows/desktop/api/D3D11/nn-d3d11-id3d11texture3d)<br/>                       | Интерфейс трехмерной текстуры обращается к данным шаг текселя, которые представляют собой структурированную память.<br/>                                                                                                                     |
| [**ID3D11Texture3D1**](/windows/desktop/api/D3D11_3/nn-d3d11_3-id3d11texture3d1)<br/>                     | Интерфейс трехмерной текстуры представляет шаг текселя данные, которые представляют собой структурированную память.<br/>                                                                                                                   |
| [**ID3D11UnorderedAccessView**](/windows/desktop/api/D3D11/nn-d3d11-id3d11unorderedaccessview)<br/>   | Интерфейс представления указывает части ресурса, к которым конвейеру может получить доступ во время подготовки к просмотру.<br/>                                                                                                |
| [**ID3D11UnorderedAccessView1**](/windows/desktop/api/D3D11_3/nn-d3d11_3-id3d11unorderedaccessview1)<br/> | Интерфейс представления неупорядоченного доступа представляет части ресурса, к которым конвейеру может получить доступ во время подготовки к просмотру.<br/>                                                                             |
| [**ID3D11View**](/windows/desktop/api/D3D11/nn-d3d11-id3d11view)<br/>                                 | Интерфейс представления указывает части ресурса, к которым конвейеру может получить доступ во время подготовки к просмотру.<br/>                                                                                                |
