# <a name="multithreading"></a>Многопоточности

В Direct3D 11 реализована поддержка создания и отрисовки объектов с помощью нескольких потоков.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                                                   | Описание                                                                                                                                                                                                                          |
|-------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Вводные сведения о многопоточности в Direct3D 11](overviews-direct3d-11-render-multi-thread-intro.md)<br/>         | Многопоточность разработана для повышения производительности за счет выполнения работы с использованием одного или нескольких потоков одновременно. <br/>                                                                                                         |
| [Создание объектов с использованием многопоточности](overviews-direct3d-11-render-multi-thread-create.md)<br/>                  | Используйте интерфейс [**ID3D11Device**](/windows/desktop/api/D3D11/nn-d3d11-id3d11device) для создания ресурсов и объектов, используйте [**ссылку ID3D11DeviceContext**](/windows/desktop/api/D3D11/nn-d3d11-id3d11devicecontext) для [отрисовки](overviews-direct3d-11-render-multi-thread-render.md).<br/> |
| [Немедленное и отложенное отображение](overviews-direct3d-11-render-multi-thread-render.md)<br/>                     | Direct3D 11 поддерживает два типа отрисовки: немедленное и отложенное. Оба метода реализуются с помощью интерфейса [**ссылку ID3D11DeviceContext**](/windows/desktop/api/D3D11/nn-d3d11-id3d11devicecontext) .<br/>                                                      |
| [Список команд](overviews-direct3d-11-render-multi-thread-command-list.md)<br/>                                   | Список команд — это последовательность команд GPU, которые можно записать и воспроизвести. Список команд может повысить производительность, уменьшая объем издержек, создаваемых средой выполнения.<br/>                                    |
| [Различия в потоках между версиями Direct3D](overviews-direct3d-11-render-multi-thread-differences.md)<br/> | Многие многопоточные модели программирования используют примитивы синхронизации (такие как мьютексы) для создания критических разделов и предотвращения доступа к коду более чем одним потоком за раз.<br/>                       |
