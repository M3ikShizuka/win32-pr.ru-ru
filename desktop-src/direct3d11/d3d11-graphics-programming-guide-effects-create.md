# <a name="create-an-effect-direct3d-11"></a>Создание влияния (Direct3D 11)

Эффект создается путем загрузки байт-кода скомпилированного эффекта в платформу Effects. В отличие от эффектов 10, результат должен быть скомпилирован перед созданием эффекта. Эффекты, загруженные в память, можно создать путем вызова [**D3DX11CreateEffectFromMemory**](d3dx11createeffectfrommemory.md).
