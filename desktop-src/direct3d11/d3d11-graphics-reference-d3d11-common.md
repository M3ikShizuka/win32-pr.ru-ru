# <a name="common-version-reference"></a>Ссылка на стандартную версию

API Direct3D определяет несколько элементов API, общих для Direct3D 12, Direct3D 11, Direct3D 10 и Direct3D 10,1. Эти элементы API можно использовать в коде для любой из этих версий Direct3D. Эти элементы API называются независимыми от версии.
