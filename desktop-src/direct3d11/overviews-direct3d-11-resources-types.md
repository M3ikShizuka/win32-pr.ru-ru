# <a name="types-of-resources"></a>Типы ресурсов

Ресурсы — это области памяти, к которым можно получить доступ с помощью конвейера Direct3D, они являются стандартными блоками сцены. Ресурсы содержат много типов данных, таких как геометрические объекты, текстуры или данные шейдера, используемые Direct3D для заполнения и визуализации сцены. В этом разделе описываются типы ресурсов Direct3D 10, а также новые типы в Direct3D 11, включая структурированные буферы и доступные для записи текстуры и буферы.
