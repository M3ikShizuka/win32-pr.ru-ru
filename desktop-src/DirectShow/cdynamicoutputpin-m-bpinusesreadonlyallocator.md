# <a name="cdynamicoutputpinm_bpinusesreadonlyallocator-member"></a>Элемент Кдинамикаутпутпин:: m \_ бпинусесреадонляллокатор

Флаг, указывающий, доступны ли образцы из распределителя закрепления только для чтения. Если значение равно **true**, пример доступен только для чтения. Значение по умолчанию — **false**.

## <a name="syntax"></a>Синтаксис


```C++
BOOL m_bPinUsesReadOnlyAllocator;
```
