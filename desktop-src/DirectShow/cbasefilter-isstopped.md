# <a name="cbasefilterisstopped-method"></a>Кбасефилтер. метод Stopped

`IsStopped`Метод определяет, остановлен ли фильтр.

## <a name="syntax"></a>Синтаксис


```C++
BOOL IsStopped();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если фильтр остановлен, или **false** в противном случае.
