# <a name="directshow-interfaces"></a>интерфейсы DirectShow

в этом разделе содержатся справочные сведения по интерфейсам Microsoft® DirectShow® core. Для обеспечения широкой поддержки различных типов оборудования, форматов файлов и потоковых операций реализуется большое количество интерфейсов. Некоторые из них используются приложениями; другие используются для внутренних целей фильтрами и другими DirectShow компонентами.

в следующем разделе содержится алфавитный список всех интерфейсов DirectShow кода.

-   [алфавитный список интерфейсов DirectShow](alphabetical-list-of-directshow-interfaces.md)
