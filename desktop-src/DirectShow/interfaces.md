# <a name="directshow-interfaces"></a>Интерфейсы DirectShow

В этом разделе содержатся справочные сведения о базовых интерфейсах® Microsoft® DirectShow. Для обеспечения широкой поддержки различных типов оборудования, форматов файлов и потоковых операций реализуется большое количество интерфейсов. Некоторые из них используются приложениями; другие используются для внутренних целей фильтрами и другими компонентами DirectShow.

В следующем разделе содержится алфавитный список всех интерфейсов DirectShow кода.

-   [Алфавитный список интерфейсов DirectShow](alphabetical-list-of-directshow-interfaces.md)
