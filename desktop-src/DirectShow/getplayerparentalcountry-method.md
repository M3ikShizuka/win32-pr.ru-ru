# <a name="getplayerparentalcountry-method"></a>Метод Жетплайерпаренталкаунтри

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`GetPlayerParentalCountry`Метод извлекает текущую страну или регион, как задано в объекте **мсвебдвд** .

``` syntax
[ iCountryCode = ] MSWebDVD.GetPlayerParentalCountry()
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение с кодом страны или региона.
