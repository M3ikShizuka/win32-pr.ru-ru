# <a name="avencvideocbrmotiontradeoff-property"></a>Авенквидеокбрмотионтрадеофф, свойство

Указывает компромисс между движениями и по-прежнему образами. Это свойство применяется только к режиму управления постоянной скоростью (CBR).

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеокбрмотионтрадеофф**

## <a name="property-value"></a>Значение свойства

Значение этого свойства имеет следующий диапазон.



| Значение | Описание               |
|-------|---------------------------|
| 0     | Оптимизировать для изображений по-прежнему |
| 100   | Оптимизировать для движения.      |
