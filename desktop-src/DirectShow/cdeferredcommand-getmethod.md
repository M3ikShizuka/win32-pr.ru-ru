# <a name="cdeferredcommandgetmethod-method"></a>Кдеферредкомманд. метод метода

`GetMethod`Метод получает идентификатор диспетчеризации (DISPID) выполняемого метода.

## <a name="syntax"></a>Синтаксис


```C++
long GetMethod();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает идентификатор DISPID метода для выполнения.
