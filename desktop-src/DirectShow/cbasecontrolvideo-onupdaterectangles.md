# <a name="cbasecontrolvideoonupdaterectangles-method"></a>Кбасеконтролвидео. Онупдатеректанглес, метод

`OnUpdateRectangles`Метод вызывается при изменении исходного или конечного прямоугольника.

## <a name="syntax"></a>Синтаксис


```C++
virtual HRESULT OnUpdateRectangles();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение "ошибка".
