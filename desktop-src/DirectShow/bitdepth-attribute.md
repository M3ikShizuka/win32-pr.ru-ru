# <a name="bitdepth-attribute"></a>Атрибут битдепс

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`bitdepth`Атрибут задает битовую глубину выходного видео в битах на пиксель.

## <a name="possible-values"></a>Возможные значения

Должно быть 16, 24 или 32. Значение по умолчанию — 16.

## <a name="applies-to"></a>Применяется к

[**сгруппировать**](group-element.md)

## <a name="remarks"></a>Remarks

Устанавливайте этот атрибут только в том случае, если атрибут Type имеет значение `video` .
