# <a name="alpha-setter-effect"></a>Результат установки альфа-канала

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Этот параметр задает альфа-биты на видеоизображении.

Идентификатор класса (CLSID): {506D89AE-909A-44f7-9444-ABD575896E35}

Имя переменной CLSID: CLSID \_ дксталфасеттер

Понятное имя: "Дксталфасеттер"

### <a name="properties"></a>Свойства



| Свойство  | Тип   | По умолчанию | Описание                                                 |
|-----------|--------|---------|-------------------------------------------------------------|
| Коэффициент альфа     | BYTE   | -1      | Задает альфа-канал для всего изображения.                        |
| алфарамп | double | -1.0    | Задает значение альфа в процентах от исходного альфа-значения. |



 

## <a name="remarks"></a>Комментарии

Свойство с отрицательным значением игнорируется. Только одно свойство может иметь неотрицательное значение. Свойство alpha задает константное альфа-значение для каждого пикселя в изображении. Это свойство перезаписывает альфа-значения из исходного изображения. Свойство Алфарамп задает альфа-значение каждого пикселя в процентах от исходного значения пикселя от 0,0 до 1,0.
