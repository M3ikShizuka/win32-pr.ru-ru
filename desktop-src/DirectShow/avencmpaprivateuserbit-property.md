# <a name="avencmpaprivateuserbit-property"></a>Авенкмпаприватеусербит, свойство

Задает значение закрытого бита пользователя в потоке MPEG-1. Это свойство применяется к кодировщикам аудио в формате MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпаприватеусербит**

## <a name="property-value"></a>Значение свойства

Это свойство может иметь значение 0 или 1.
