# <a name="titlesavailable-property"></a>Титлесаваилабле, свойство

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`TitlesAvailable`Свойство получает количество заголовков, доступных на диске.

``` syntax
[ iTitles = ] MSWebDVD.TitlesAvailable
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает количество доступных заголовков в виде целого значения от 1 до 99.

## <a name="remarks"></a>Remarks

Это свойство доступно только для чтения и не имеет значения по умолчанию.
