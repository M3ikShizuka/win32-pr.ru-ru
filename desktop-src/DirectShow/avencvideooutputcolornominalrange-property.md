# <a name="avencvideooutputcolornominalrange-property"></a>Авенквидеуутпутколорноминалранже, свойство

Указывает Номинальный диапазон для закодированного видео.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеуутпутколорноминалранже**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенквидеоколорноминалранже**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolornominalrange) .
