# <a name="enablefx-attribute"></a>Атрибут енаблефкс

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`enablefx`Атрибут указывает, включены ли эффекты. Если значение равно **true**, механизм визуализации визуализирует эффекты. В противном случае это условие не выполняется. Значение по умолчанию — **true**.

## <a name="possible-values"></a>Возможные значения

Следующие значения определены как TRUE: y, Y, t, T, 1. Следующие значения определены как FALSE: n, N, f, F, 0 (ноль).

## <a name="applies-to"></a>Применяется к

[**графика**](timeline-element.md)
