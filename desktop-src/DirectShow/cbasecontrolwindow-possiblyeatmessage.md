# <a name="cbasecontrolwindowpossiblyeatmessage-method"></a>Кбасеконтролвиндов. Поссиблеатмессаже, метод

`PossiblyEatMessage`Метод пересылает сообщения клавиатуры и мыши в окно сток сообщений.

## <a name="syntax"></a>Синтаксис


```C++
BOOL PossiblyEatMessage(
   UINT   uMsg,
   WPARAM wParam,
   LPARAM lParam
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Uiacp* 
</dt> <dd>

Окно сообщения.

</dd> <dt>

*wParam* 
</dt> <dd>

Первый параметр сообщения.

</dd> <dt>

*lParam* 
</dt> <dd>

Второй параметр сообщения.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если сообщение было перенаправлено в окно, или **значение false** в противном случае.

## <a name="remarks"></a>Remarks

Окно сток сообщений — это окно, предназначенное для получения определенных сообщений мыши и клавиатуры. Изначально окно имеет **значение NULL**; его можно задать, вызвав [**кбасеконтролвиндов::p UT \_ мессажедраин**](cbasecontrolwindow-put-messagedrain.md).

Если окно стока сообщений не имеет **значение NULL**, `PossiblyEatMessage` в окно отправляется следующее сообщение:

-   WM \_ char
-   WM \_ деадчар
-   WM \_ KeyDown
-   WM \_ KEYUP
-   WM \_ лбуттондблклк
-   WM \_ лбуттондовн
-   WM \_ лбуттонуп
-   WM \_ мбуттондблклк
-   WM \_ мбуттондовн
-   WM \_ мбуттонуп
-   WM \_ маусеактивате
-   WM \_ MOUSEMOVE
-   WM \_ нклбуттондблклк
-   WM \_ нклбуттондовн
-   WM \_ нклбуттонуп
-   WM \_ нкмбуттондблклк
-   WM \_ нкмбуттондовн
-   WM \_ нкмбуттонуп
-   WM \_ нкмаусемове
-   WM \_ нкрбуттондблклк
-   WM \_ нкрбуттондовн
-   WM \_ нкрбуттонуп
-   WM \_ рбуттондблклк
-   WM \_ рбуттондовн
-   WM \_ рбуттонуп
-   WM \_ сисчар
-   WM \_ сисдеадчар
-   WM \_ сискэйдовн
-   WM \_ сискэйуп

Другие сообщения игнорируются. Если окно сток сообщений имеет **значение NULL**, метод игнорирует все сообщения окна. Метод возвращает **значение true** , если отправляет сообщение, или **значение false** в противном случае. Класс **кбасевиндов** вызывает этот метод при получении сообщения окна.
