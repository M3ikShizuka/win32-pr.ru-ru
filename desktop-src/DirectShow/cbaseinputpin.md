# <a name="cbaseinputpin-class"></a>Класс Кбасеинпутпин

![Иерархия классов кбасеинпутпин](images/filter07.png)

`CBaseInputPin`Класс является абстрактным базовым классом для реализации входных ПИН-кодов. Этот класс добавляет поддержку интерфейса [**имеминпутпин**](/windows/desktop/api/Strmif/nn-strmif-imeminputpin) в дополнение к поддержке интерфейса [**Ипин**](/windows/desktop/api/Strmif/nn-strmif-ipin) , предоставляемой [**кбасепин**](cbasepin.md).

Чтобы использовать этот класс, необходимо создать новый класс и переопределить по крайней мере следующие методы:

-   [**Кбасеинпутпин:: Бегинфлуш**](cbaseinputpin-beginflush.md)
-   [**Кбасеинпутпин:: Ендфлуш**](cbaseinputpin-endflush.md)
-   [**Кбасеинпутпин:: Receive**](cbaseinputpin-receive.md)
-   [**Кбасепин:: Чеккмедиатипе**](cbasepin-checkmediatype.md)
-   [**Кбасепин:: Жетмедиатипе**](cbasepin-getmediatype.md)

В зависимости от функции ПИН-кода может потребоваться переопределить дополнительные методы в `CBaseInputPin` или **кбасепин**.
