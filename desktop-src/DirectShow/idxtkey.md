# <a name="idxtkey-interface"></a>Интерфейс Идксткэй

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`IDxtKey`Интерфейс задает свойства для перехода по [ключу](key-transition.md) .

этот интерфейс используется внутренне DirectShow служб редактирования (DES) при отрисовке ключа. Приложениям DES не требуется использовать этот интерфейс. Чтобы задать свойства для перехода в DES, используйте интерфейс [**ипропертисеттер**](ipropertysetter.md) .

## <a name="members"></a>Элементы

Интерфейс **идксткэй** наследует от **идксеффект**. **Идксткэй** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **идксткэй** содержит следующие методы.



| Метод                                            | Описание                                                                            |
|:--------------------------------------------------|:---------------------------------------------------------------------------------------|
| [**получить \_ оттенок**](idxtkey-get-hue.md)               | Возвращает значение оттенка, по которому следует подключаться.<br/>                                    |
| [**получить \_ инверсию**](idxtkey-get-invert.md)         | Получает логическое значение, указывающее, выполняется ли обратная операция с ключом.<br/> |
| [**получить \_ KeyType**](idxtkey-get-keytype.md)       | Возвращает тип ключа.<br/>                                                  |
| [**получить \_ светимость**](idxtkey-get-luminance.md)   | Извлекает значение светимости для ключа.<br/>                              |
| [**получить \_ RGB**](idxtkey-get-rgb.md)               | Извлекает цвет RGB, по которому следует подключаться.<br/>                                    |
| [**получить \_ подобие**](idxtkey-get-similarity.md) | Извлекает диапазон цветовых данных, которые становятся прозрачными.<br/>                 |
| [**вставить \_ оттенок**](idxtkey-put-hue.md)               | Задает значение оттенка ключа.<br/>                                    |
| [**вернуть \_ инверсию**](idxtkey-put-invert.md)         | Указывает, выполняется ли обратная операция с ключом.<br/>                            |
| [**вставить \_ KeyType**](idxtkey-put-keytype.md)       | Указывает тип ключа.<br/>                                                  |
| [**Поставьте \_ светимость**](idxtkey-put-luminance.md)   | Задает значение светимости для ключа.<br/>                              |
| [**перевести \_ RGB**](idxtkey-put-rgb.md)               | Задает цвет RGB для ключа.<br/>                                    |
| [**разместить \_ подобие**](idxtkey-put-similarity.md) | Указывает диапазон цветовых данных, которые становятся прозрачными.<br/>                 |
