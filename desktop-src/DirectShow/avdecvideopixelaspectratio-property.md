# <a name="avdecvideopixelaspectratio-property"></a>Авдеквидеопикселаспектратио, свойство

Задает пропорцию пикселя декодированного видеопотока.

Это свойство доступно только для чтения.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авдеквидеопикселаспектратио**

## <a name="remarks"></a>Комментарии

Верхние 16 разрядов значения содержат ширину, а младшие 16 бит содержат высоту.
