# <a name="csourceseekingm_plock-member"></a>Элемент Ксаурцесикинг:: m \_ плокк

Эта переменная инициализируется в методе-конструкторе. см. раздел [**ксаурцесикинг:: ксаурцесикинг**](csourceseeking-csourceseeking.md).
