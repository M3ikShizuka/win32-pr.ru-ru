# <a name="currentangle-property"></a>Куррентангле, свойство

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`CurrentAngle`Свойство задает или получает текущий угол в блоке угла.

``` syntax
[ iCurAngle = ] MSWebDVD.CurrentAngle
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение, представляющее угол. Значение должно быть в значениях от 1 до 9.

## <a name="remarks"></a>Remarks

Это свойство доступно для чтения и записи и не имеет значения по умолчанию.
