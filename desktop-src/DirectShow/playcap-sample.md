# <a name="playcap-sample"></a>Пример Плайкап

## <a name="description"></a>Описание

Простое приложение для записи.

Это приложение создает окно предварительного просмотра для первого устройства видеозаписи, которое оно обнаруживает в системе пользователя (если оно есть). В нем демонстрируется простой пример использования интерфейсов [**ICaptureGraphBuilder2**](/windows/desktop/api/Strmif/nn-strmif-icapturegraphbuilder2) и [**икреатедевенум**](/windows/desktop/api/Strmif/nn-strmif-icreatedevenum) для создания диаграммы захвата.

Пример приложения для записи, которое поддерживает дополнительные функции, см. в разделе [Амкап Sample](amcap-sample.md).

## <a name="downloading-the-sample"></a>Загрузка образца

Чтобы скачать примеры пакета SDK для DirectShow, установите последнюю версию [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Этот пример устанавливается по следующему пути: *\[ \] корневые примеры SDK* \\ \\ мультимедиа \\ DirectShow \\ Capture \\ плайкап.
