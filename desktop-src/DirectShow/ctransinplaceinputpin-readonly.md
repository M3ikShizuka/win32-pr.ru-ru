# <a name="ctransinplaceinputpinreadonly-method"></a>Ктрансинплацеинпутпин. ReadOnly, метод

`ReadOnly`Метод указывает, доступен ли входной поток только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
const BOOL ReadOnly();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение переменной члена [**m \_ бреадонли**](ctransinplaceinputpin-m-breadonly.md) .
