# <a name="idxtalphasetter-interface"></a>Интерфейс Идксталфасеттер

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`IDxtAlphaSetter`Интерфейс задает свойства для действия [альфа-установки](alpha-setter-effect.md) .

Этот интерфейс внутренне используется службами редактирования DirectShow (DES) при визуализации воздействия на альфа-метод задания. Приложениям DES не требуется использовать этот интерфейс. Чтобы задать свойства для перехода в DES, используйте интерфейс [**ипропертисеттер**](ipropertysetter.md) .

## <a name="members"></a>Элементы

Интерфейс **идксталфасеттер** наследует от **идксеффект**. **Идксталфасеттер** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **идксталфасеттер** содержит следующие методы.



| Метод                                                  | Описание                                                |
|:--------------------------------------------------------|:-----------------------------------------------------------|
| [**получение \_ альфа-канала**](idxtalphasetter-get-alpha.md)         | Извлекает альфа-значение для всего изображения.<br/> |
| [**получить \_ алфарамп**](idxtalphasetter-get-alpharamp.md) | Извлекает свойство альфа-пандуса.<br/>              |
| [**разместить \_ альфа-канал**](idxtalphasetter-put-alpha.md)         | Задает альфа-значение для всего изображения.<br/> |
| [**разместить \_ алфарамп**](idxtalphasetter-put-alpharamp.md) | Указывает свойство пандус альфа.<br/>              |
