# <a name="cbaseinputpinisflushing-method"></a>Метод Кбасеинпутпин. Flush

`IsFlushing`Метод запрашивает, выполняется ли в данный момент фильтр на сброс.

## <a name="syntax"></a>Синтаксис


```C++
BOOL IsFlushing();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает переменную члена [**кбасеинпутпин:: m \_ бфлушинг**](cbaseinputpin-m-bflushing.md) .
