# <a name="avencddsurrounddownmixlevel-property"></a>Авенкддсурраунддовнмикслевел, свойство

Задает уровень окружающей довнмикси в децибел (dB) в потоке цифрового звука Dolby. Это свойство применяется к кодировщикам цифрового аудио Dolby.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкддсурраунддовнмикслевел**
