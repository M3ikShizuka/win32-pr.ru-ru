# <a name="savebookmark-method"></a>Метод Савебукмарк

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`SaveBookmark`Метод сохраняет текущую позицию на диске и состояние объекта **мсвебдвд** , чтобы пользователь мог вернуться к тому же месту позже.

``` syntax
MSWebDVD.SaveBookmark()
```

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Комментарии

Закладка является моментальным снимком текущего состояния DVD-навигатора. Сюда входят такие сведения, как место воспроизведения на диске и выбор потоков аудио-и звуковых изображений. Сохранив закладку, пользователь может закрыть приложение, завершить работу компьютера и вернуться позже, чтобы продолжить просмотр диска справа от того, где он или он остался, и все параметры, как и ранее. В любое заданное время можно сохранить только одну закладку. При вызове `SaveBookmark` старая закладка перезаписывается.
