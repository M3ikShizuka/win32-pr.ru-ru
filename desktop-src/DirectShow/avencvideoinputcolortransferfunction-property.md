# <a name="avencvideoinputcolortransferfunction-property"></a>Авенквидеоинпутколортрансферфунктион, свойство

Указывает функцию преобразования из RGB в Р'Г'Б "для входного видео.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеоинпутколортрансферфунктион**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенквидеоколортрансферфунктион**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolortransferfunction) .
