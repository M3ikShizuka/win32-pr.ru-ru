# <a name="avenccommonstreamendhandling-property"></a>Авенккоммонстреамендхандлинг, свойство

Указывает, отбрасывает ли кодировщик частичные группы изображений (группы GOP) в конце потока.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенккоммонстреамендхандлинг**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенккоммонстреамендхандлинг**](/windows/win32/api/codecapi/ne-codecapi-eavenccommonstreamendhandling) .
