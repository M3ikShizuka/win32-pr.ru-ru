# <a name="cmediatypeistemporalcompressed-method"></a>Кмедиатипе. Истемпоралкомпрессед, метод

`IsTemporalCompressed`Метод определяет, использует ли поток временное сжатие.

## <a name="syntax"></a>Синтаксис


```C++
BOOL IsTemporalCompressed() const;
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение элемента **бтемпоралкомпрессион** .
