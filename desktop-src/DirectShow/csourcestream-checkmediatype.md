# <a name="csourcestreamcheckmediatype-method"></a>Ксаурцестреам. Чеккмедиатипе, метод

`CheckMediaType`Метод определяет, принимает ли ПИН-код конкретный тип носителя. Этот метод реализует чистый виртуальный метод [**кбасепин:: чеккмедиатипе**](cbasepin-checkmediatype.md) .

## <a name="syntax"></a>Синтаксис


```C++
virtual HRESULT CheckMediaType(
   const CMediaType *pMediaType
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пмедиатипе* 
</dt> <dd>

Указатель на объект [**кмедиатипе**](cmediatype.md) , который содержит предложенный тип мультимедиа.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из значений **HRESULT** , приведенных в следующей таблице.



| Код возврата                                                                            | Описание                                          |
|----------------------------------------------------------------------------------------|------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>   | Этот ПИН-код поддерживает этот тип носителя.<br/>        |
| <dl> <dt>**\_Ошибка E**</dt> </dl> | ПИН-код не поддерживает этот тип носителя.<br/> |



 

## <a name="remarks"></a>Remarks

По умолчанию ПИН-код поддерживает один тип носителя. Этот метод извлекает поддерживаемый тип, вызывая версию с одним параметром метода [**ксаурцестреам:: жетмедиатипе**](csourcestream-getmediatype.md) и сравнивая его с предложенным типом. Если ПИН-код поддерживает более одного типа носителей, Переопределите этот метод.
