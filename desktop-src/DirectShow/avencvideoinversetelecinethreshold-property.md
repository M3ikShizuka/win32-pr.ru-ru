# <a name="avencvideoinversetelecinethreshold-property"></a>АвенквидеоинверсетелеЦинесрешолд, свойство

Задает пороговое значение, при котором кодировщик считает поле видео избыточным.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеоинверсетелеЦинесрешолд**

## <a name="property-value"></a>Значение свойства

Значение этого свойства имеет следующий диапазон.



| Значение | Описание                                                    |
|-------|----------------------------------------------------------------|
| 0     | Менее вероятно, что кодировщику следует считать поля видео избыточными. |
| 100   | Более вероятно, что кодировщик может считать поля видео избыточными. |



 

## <a name="remarks"></a>Remarks

Задайте это свойство, если кодировщик выполняет обратную чересстрочности с аналоговым источником видео.
