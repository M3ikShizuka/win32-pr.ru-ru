# <a name="avencaudiointervaltoskip-property"></a>Авенкаудиоинтервалтоскип, свойство

Указывает количество звуковых образцов для пропущенного кодировщика.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT64** (**VT \_ UI8**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкаудиоинтервалтоскип**
