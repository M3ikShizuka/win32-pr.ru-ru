# <a name="french-overseas-territories"></a>Заморские территории Франции

Описание следующих таблиц см. в разделе [таблицы и назначения](tables-and-assignments.md).

Индексы частоты в таблице широковещательной рассылки для французских загранок могут использоваться в качестве номеров каналов. индексы частоты кабелей не подходят для использования в качестве номеров каналов.

### <a name="french-overseas-territories-broadcast-assignments"></a>Рассылку назначений для загранных территорий Франции


```C++
         4L,            // Lowest channel
         9L,            // Highest channel

       175250000L,      // 4
       183250000L,      // 5
       191250000L,      // 6
       199250000L,      // 7
       207250000L,      // 8
       215250000L,      // 9
```



### <a name="french-overseas-territories-cable-assignments"></a>Назначения кабелей для зафиксированных территорий Франции


```C++
         4L,            // Lowest channel
         9L,            // Highest channel

       175250000L,      // 4
       183250000L,      // 5
       191250000L,      // 6
       199250000L,      // 7
       207250000L,      // 8
       215250000L,      // 9
```
