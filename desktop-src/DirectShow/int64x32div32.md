# <a name="int64x32div32-function"></a>Функция Int64x32Div32

`Int64x32Div32`Функция реализует формулу, `((a*b)+rnd)/c` где *a* — это 64-разрядное значение, а *b*, *c*, а *Rnd* — 32-разрядные значения.

## <a name="syntax"></a>Синтаксис


```C++
LONGLONG WINAPI Int64x32Div32(
   LONGLONG a,
   LONG     b,
   LONG     c,
   LONG     rnd
);
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает либо `(a * b + rnd)/c` вычисление, либо одно из следующих значений.



| Код возврата                                                                                       | Описание                                                              |
|---------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------|
| <dl> <dt>**0x7FFFFFFFFFFFFFFF**</dt> </dl> | Произошло переполнение, так как результат слишком велик (положительный).<br/> |
| <dl> <dt>**0x8000000000000000**</dt> </dl> | Произошло переполнение, так как результат слишком большой (отрицательный).<br/> |



 

## <a name="remarks"></a>Remarks

Округление на деление в сторону нуля. Деление на ноль считается условием переполнения.

Метки времени и время поиска представляют собой 64-битные значения, поэтому эта функция полезна для выполнения преобразований в 32-разрядных системах. Например, в MPEG-1 ссылка на системный таймер составляет 90 кГц или 90 000 тактов в секунду. Формула для преобразования этой формулы в ссылочное время (100-наносекундных единиц) —


```C++
(timestamp * 1000) / 9
```



который можно вычислить как `Int64x32Div32(timestamp, 1000, 9, 0)` . Используйте параметр *Rnd* в качестве фактора округления.
