# <a name="crendererinputpininactive-method"></a>Крендереринпутпин. Inactive, метод

`Inactive`Метод уведомляет ПИН-код о том, что фильтр больше не активен. Этот метод переопределяет метод [**кбасеинпутпин:: Inactive**](cbaseinputpin-inactive.md) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Inactive();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** .
