# <a name="crendererinputpin-class"></a>Класс Крендереринпутпин

![крендереринпут закрепить иерархию классов](images/rbase01.png)

Класс **кбасерендереринпутпин** реализует входной ПИН-код для класса [**кбасерендерер**](cbaserenderer.md) . За исключением указанных случаев, методы в этом классе делегируют соответствующие методы класса **кбасерендерер** .



| Защищенные переменные членов                                       | Описание                                                                            |
|------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**m \_ прендерер**](crendererinputpin-m-prenderer.md)            | Указатель на фильтр.                                                                 |
| Открытые методы                                                   | Описание                                                                            |
| [**крендереринпутпин**](crendererinputpin-crendererinputpin.md) | Метод конструктора.                                                                    |
| [**бреакконнект**](crendererinputpin-breakconnect.md)           | Добавляет настраиваемый код при разрыве соединения.                                       |
| [**комплетеконнект**](crendererinputpin-completeconnect.md)     | Завершает соединение.                                                              |
| [**чеккмедиатипе**](crendererinputpin-checkmediatype.md)       | Определяет, может ли ПИН-код поддерживать конкретный тип носителя.                               |
| [**Активен**](crendererinputpin-active.md)                       | Переключает ПИН-код в активный (приостановленный или запущенный) режим.                               |
| [**Неактивно**](crendererinputpin-inactive.md)                   | Переключает ПИН-код в неактивное состояние и освобождает память распределителя.        |
| [**сетмедиатипе**](crendererinputpin-setmediatype.md)           | Задает тип носителя для ПИН-кода.                                                        |
| [**Выделен**](crendererinputpin-allocator.md)                 | Извлекает указатель на распределитель памяти по умолчанию.                                   |
| Методы Ипин                                                     | Описание                                                                            |
| [**QueryId**](crendererinputpin-queryid.md)                     | Возвращает идентификатор для ПИН-кода.                                                   |
| [**EndOfStream**](crendererinputpin-endofstream.md)             | Информирует ПИН-код о том, что дополнительные данные не ожидаются до тех пор, пока не будет выполнена новая команда запуска. |
| [**бегинфлуш**](crendererinputpin-beginflush.md)               | Информирует ПИН-код, чтобы начать операцию очистки.                                            |
| [**ендфлуш**](crendererinputpin-endflush.md)                   | Сообщает ПИН-коду, чтобы завершить операцию очистки.                                              |
| Методы Имеминпутпин                                             | Описание                                                                            |
| [**Получать**](crendererinputpin-receive.md)                     | Извлекает следующий блок данных из потока.                                      |
