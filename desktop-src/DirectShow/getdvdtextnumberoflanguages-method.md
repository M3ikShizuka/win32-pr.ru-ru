# <a name="getdvdtextnumberoflanguages-method"></a>Метод Жетдвдтекстнумберофлангуажес

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`GetDVDTextNumberOfLanguages`Метод извлекает количество языков текста, доступных на диске.

``` syntax
[ iLanguages = ] MSWebDVD.GetDVDTextNumberOfLanguages()
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение, указывающее количество доступных языков текста.
