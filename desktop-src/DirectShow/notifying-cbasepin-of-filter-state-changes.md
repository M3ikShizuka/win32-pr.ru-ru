# <a name="notifying-cbasepin-of-filter-state-changes"></a>Уведомление Кбасепин об изменениях состояния фильтра

Класс **кбасепин** уведомляется всякий раз, когда изменяется состояние фильтра владельца. Для каждого перехода состояния фильтр вызывает соответствующий метод для ПИН-кода, как показано в следующей таблице.



| Новое состояние фильтра | Метод Кбасепин                                 |
|------------------|-------------------------------------------------|
| Остановлена          | [**Кбасепин:: Inactive**](cbasepin-inactive.md) |
| Пауза           | [**Кбасепин:: Active**](cbasepin-active.md)     |
| Выполнение          | [**Кбасепин:: Run**](cbasepin-run.md)           |



 

Производный класс должен переопределять эти методы для реагирования на изменение состояния. В зависимости от фильтра ПИН-код может запустить рабочий поток, который доставляет примеры, зафиксировать или отменять выделение памяти и т. д.
