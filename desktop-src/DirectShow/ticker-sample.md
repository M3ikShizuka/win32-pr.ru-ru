# <a name="ticker-sample"></a>Пример использования Tick

## <a name="description"></a>Описание

В этом примере для смешения видео и текста используется модуль подготовки видео с смешением. Он использует интерфейс [**IVMRMixerBitmap9**](/previous-versions/windows/desktop/api/Vmr9/nn-vmr9-ivmrmixerbitmap9) для смешения текста с нижней частью окна видео.

Приложение создает точечный рисунок с текстовой строкой по умолчанию и прокручивает его в нижней части экрана. Чтобы изменить текст, выберите команду **задать текстовую строку...** в меню " **деление** ". Чтобы задать шрифт, выберите пункт **задать шрифт...** в меню " **деление** ". При выборе параметра **использовать статическое изображение** из меню **Ticks** приложение переключается с динамической текстовой строки на статическое растровое изображение.
