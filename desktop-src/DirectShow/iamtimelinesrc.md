# <a name="iamtimelinesrc-interface"></a>Интерфейс Иамтимелинесрк

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`IAMTimelineSrc`интерфейс предоставляет методы для управления и установки свойств исходных объектов в [службах DirectShow editing Services](directshow-editing-services.md) (DES). Исходный объект представляет один поток из источника мультимедиа.

Можно использовать часть данных в исходном файле, задав время начала и окончания воспроизведения носителя. Эти значения указывают начало и конец исходного объекта относительно исходного источника мультимедиа. Время работы с носителями может отличаться от времени начала и окончания работы объекта на временной шкале, что позволяет быстро или медленно воспроизводить воспроизведение. (С источниками звука происходит сдвиг высоты.)

Чтобы создать исходный объект, вызовите [**иамтимелине:: креатимптиноде**](iamtimeline-createemptynode.md) со значением \_ источник основного типа временной шкалы \_ \_ . Вы можете запросить возвращенный указатель [**иамтимелинеобж**](iamtimelineobj.md) для интерфейса **иамтимелинесрк** . Дополнительные сведения см. в статьях [Создание временной шкалы](constructing-a-timeline.md) и [Работа с источниками](working-with-sources.md).

### <a name="methods"></a>Методы

Интерфейс **иамтимелинесрк** содержит следующие методы.



| Метод                                                    | Описание                                                                                              |

| [**FixMediaTimes2**](iamtimelinesrc-fixmediatimes2.md)   | Округляет указанные значения времени, заданные как значения **рефтиме** , до ближайшей границы фрейма.<br/> |

| [**жетмедиаленгс**](iamtimelinesrc-getmedialength.md)   | Возвращает длину носителя этого исходного объекта.<br/>                                             |
| [**GetMediaLength2**](iamtimelinesrc-getmedialength2.md) | Получает длину носителя этого исходного объекта в виде значения **рефтиме** .<br/>                     |
| [**жетмедианаме**](iamtimelinesrc-getmedianame.md)       | Извлекает имя исходного файла, представленного этим исходным объектом.<br/>                      |
| [**жетмедиатимес**](iamtimelinesrc-getmediatimes.md)     | Получает время начала и окончания воспроизведения носителя.<br/>                                                     |
| [**GetMediaTimes2**](iamtimelinesrc-getmediatimes2.md)   | Получает время начала и окончания воспроизведения в виде значений **рефтиме** .<br/>                              |
| [**жетстреамнумбер**](iamtimelinesrc-getstreamnumber.md) | Возвращает текущий номер потока для исходного объекта.<br/>                                    |
| [**жетстретчмоде**](iamtimelinesrc-getstretchmode.md)   | Извлекает режим растяжения для источника видео.<br/>                                                 |
| [**иснормалрате**](iamtimelinesrc-isnormalrate.md)       | Указывает, будет ли воспроизводиться воспроизведение клипа с нормальным темпом воспроизведения.<br/>                             |
| [**модифистоптиме**](iamtimelinesrc-modifystoptime.md)   | Задает время окончания относительно временной шкалы.<br/>                                                 |
| [**ModifyStopTime2**](iamtimelinesrc-modifystoptime2.md) | Задает время окончания в виде значения **рефтиме** .<br/>                                                   |
| [**сетдефаултфпс**](iamtimelinesrc-setdefaultfps.md)     | Задает частоту кадров исходного объекта по умолчанию.<br/>                                                  |
| [**сетмедиаленгс**](iamtimelinesrc-setmedialength.md)   | Указывает длительность исходного файла.<br/>                                                    |
| [**SetMediaLength2**](iamtimelinesrc-setmedialength2.md) | Задает продолжительность исходного файла в виде значения **рефтиме** .<br/>                            |

| [**сплицевиснекст**](iamtimelinesrc-splicewithnext.md)   | Присоединяет этот исходный объект к другому исходному объекту.<br/>                                            |
