# <a name="cbasevideorenderergetstddev-method"></a>Кбасевидеорендерер. Жетстддев, метод

`GetStdDev`Метод оценивает стандартное отклонение в миллисекундах между сроком выполнения каждого кадра и его фактической отрисовкой для статистики по кадрам.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetStdDev(
   int      nSamples,
   int      *piResult,
   LONGLONG llSumSq,
   LONGLONG iTot
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*нсамплес* 
</dt> <dd>

Целочисленное значение, содержащее число видеороликов, полученных модулем подготовки видео.

</dd> <dt>

*пиресулт* 
</dt> <dd>

Указатель на целочисленное значение, которое будет содержать стандартное отклонение.

</dd> <dt>

*ллсумск* 
</dt> <dd>

Значение, представляющее стандартное отклонение (в миллисекундах) всех визуализированных видеороликов. Чем меньше значение, тем более последовательна отрисовка.

</dd> <dt>

*итот* 
</dt> <dd>

Значение, представляющее среднее значение в миллисекундах между отметкой времени и временем визуализации для всех визуализированных видеороликов.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение "ошибка".
