# <a name="camthreadthreadexists-method"></a>Камсреад. Среадексистс, метод

`ThreadExists`Метод запрашивает, существует ли поток.

## <a name="syntax"></a>Синтаксис


```C++
BOOL ThreadExists();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если поток существует, или **значение false** , если поток не существует.
