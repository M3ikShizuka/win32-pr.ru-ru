# <a name="ctransformfilterfindpin-method"></a>Ктрансформфилтер. Финдпин, метод

Метод **финдпин** получает ПИН-код с указанным идентификатором. Этот метод реализует метод [**ибасефилтер:: финдпин**](/windows/desktop/api/Strmif/nf-strmif-ibasefilter-findpin) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT FindPin(
   LPCWSTR Id,
   IPin    **ppPin
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Id* 
</dt> <dd>

Строка расширенных символов, содержащая идентификатор ПИН-кода.

</dd> <dt>

*пппин* 
</dt> <dd>

Получает указатель на интерфейс [**Ипин**](/windows/desktop/api/Strmif/nn-strmif-ipin) ПИН-кода. Если метод завершается с ошибкой, `*ppPin` устанавливается в **значение NULL**.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из значений **HRESULT** , приведенных в следующей таблице.



| Код возврата                                                                                       | Описание                                           |
|---------------------------------------------------------------------------------------------------|-------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>              | Успешно.<br/>                                   |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl>     | Недостаточно памяти.<br/>                       |
| <dl> <dt>**\_указатель E**</dt> </dl>         | **Пустой** аргумент указателя.<br/>                 |
| <dl> <dt>**VFW \_ E \_ не \_ найден**</dt> </dl> | Не удалось найти ПИН-код с этим идентификатором.<br/> |



 

## <a name="remarks"></a>Remarks

> [!IMPORTANT]
> Реализация этого метода не вызывает [**Ипин:: QueryId**](/windows/desktop/api/Strmif/nf-strmif-ipin-queryid) для сопоставления с идентификатором ПИН-кода. Вместо этого метод предполагает, что входной ПИН-код имеет имя in, а выходной ПИН-код имеет имя out. Если используется другой набор ПИН-кодов, Переопределите этот метод.
