# <a name="cmediasamplem_lactual-member"></a>Элемент Кмедиасампле:: m \_ лактуал

Длина допустимых данных в буфере в байтах. Значение должно быть меньше или равно размеру буфера, заданному переменной члена [**кмедиасампле:: m \_ кббуффер**](cmediasample-m-cbbuffer.md) .

## <a name="syntax"></a>Синтаксис


```C++
LONG m_lActual;
```
