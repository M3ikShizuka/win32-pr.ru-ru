# <a name="vmrplayer-sample"></a>Пример Вмрплайер

## <a name="description"></a>Описание

В этом примере фильтр с изображением прорисовки 9 (VMR-9) используется для альфа Blend одного или двух работающих видео и статического изображения.

## <a name="usage"></a>Использование

Чтобы открыть первое видео, выберите в меню **файл** пункт **Открыть основной поток** . Чтобы открыть второе видео, выберите **Открыть вторичный поток** в меню **файл** (сначала необходимо открыть основной поток). Чтобы воспроизвести видео, нажмите кнопку **Воспроизведение** .

Можно задать расположение, размер и альфа-значения видео, выбрав **основной поток** или **поток Секондард** в меню **Свойства VMR** .

Чтобы добавить статическое растровое изображение в видео, выберите **статический образ приложения** в меню **Свойства VMR** и щелкните поле **отображаемое изображение приложения** . Вы можете использовать то же диалоговое окно для управления положением, размером и альфа-значением точечного рисунка.

Чтобы записать изображение в смешанном видео, выберите пункт **записать растровое изображение** в меню **Свойства VMR** .

Кроме того, можно указать основной поток образа из командной строки:

**Вмрплайер** **/p** *имя файла*
