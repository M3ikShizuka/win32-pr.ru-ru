# <a name="cmediatypeisfixedsize-method"></a>Кмедиатипе. Исфикседсизе, метод

`IsFixedSize`Метод определяет, имеют ли образцы фиксированный размер или переменный размер.

## <a name="syntax"></a>Синтаксис


```C++
BOOL IsFixedSize() const;
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение элемента **бфикседсизесамплес** .
