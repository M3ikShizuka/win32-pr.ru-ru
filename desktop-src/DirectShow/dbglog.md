# <a name="dbglog-macro"></a>Макрос Дбглог

Макрос **дбглог** отправляет строку в расположение выходных данных отладки, если ведение журнала включено для указанного типа и уровня. Этот макрос не учитывается в розничных сборках.

## <a name="syntax"></a>Синтаксис


```C++
void DbgLog(
         DWORD Types,
         DWORD Level,
   const TCHAR *pFormat,
               ...
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Типы* 
</dt> <dd>

Побитовое сочетание одного или нескольких типов сообщений.

</dd> <dt>

*Уровень* 
</dt> <dd>

Уровень ведения журнала для этого сообщения.

</dd> <dt>

*пформат* 
</dt> <dd>

Строка формата в стиле **printf** .

</dd> <dt>

*...* 
</dt> <dd>

Дополнительные аргументы для строки формата.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот макрос не возвращает значение.

## <a name="remarks"></a>Комментарии

Если для журнала отладки любого из типов сообщений задан указанный уровень или выше, этот макрос отправляет отформатированную строку в расположение выходных данных отладки.

Макрос автоматически добавляет символ новой строки в выходную строку.

> [!Note]  
> Параметры макроса должны заключаться в дополнительный набор круглых скобок:

 


```C++
DbgLog((LOG_TRACE, 3, TEXT("Connected input pin %d"), nPinNumber));
```
