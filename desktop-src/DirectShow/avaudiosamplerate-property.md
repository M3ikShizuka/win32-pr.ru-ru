# <a name="avaudiosamplerate-property"></a>Аваудиосамплерате, свойство

Возвращает частоту дискретизации потока звуковых битов в примерах в секунду.

Это свойство доступно только для чтения.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ аваудиосамплерате**

## <a name="property-value"></a>Значение свойства

Набор возможных значений зависит от реализации декодера и потока звукового бита.
