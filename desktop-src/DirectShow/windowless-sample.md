# <a name="windowless-sample"></a>Пример без окон

## <a name="description"></a>Описание

В этом примере демонстрируется режим без окон в фильтре визуализации 9 (VMR-9).

Вы можете записать изображение из окна видео, щелкнув его правой кнопкой мыши или выбрав пункт **записать текущий кадр** в меню **изображение** .

## <a name="downloading-the-sample"></a>Загрузка образца

чтобы скачать примеры пакета SDK для DirectShow, установите последнюю версию [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).
