# <a name="timeline-element"></a>Элемент Timeline

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`timeline`Элемент определяет временную шкалу. Этот элемент является корневым узлом в XML-файле.

## <a name="attributes"></a>Атрибуты

[**дефаултфкс**](defaultfx-attribute.md), [**дефаулттранс**](defaulttrans-attribute.md), [**енаблефкс**](enablefx-attribute.md), [**енаблетранс**](enabletrans-attribute.md), [**Частота кадров**](framerate-attribute.md)

## <a name="parentchild-information"></a>Сведения о родителе и дочернем элементе



| Метка | Значение |
|----------|-------------------------------------------------------|
| Parent   | Отсутствует. Этот элемент должен быть корневым узлом в файле. |
| Дети | [**сгруппировать**](group-element.md)                        |



 

## <a name="examples"></a>Примеры


```XML
<timeline> </timeline>
```
