# <a name="ccmdqueuenew-method"></a>Ккмдкуеуе. New, метод

`New`Метод инициализирует команду для выполнения и возвращает новый объект [**кдеферредкомманд**](cdeferredcommand.md) .

## <a name="syntax"></a>Синтаксис


```C++
virtual HRESULT New(
   CDeferredCommand **ppCmd,
   LPUNKNOWN        pUnk,
   REFTIME          time,
   GUID             *iid,
   long             dispidMethod,
   short            wFlags,
   long             cArgs,
   VARIANT          *pDispParams,
   VARIANT          *pvarResult,
   short            *puArgErr,
   BOOL             bStream
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ppCmd* 
</dt> <dd>

Адрес указателя на объект [**кдеферредкомманд**](cdeferredcommand.md) , по которому приложение может отменить команду, установить новое время презентации или получить сведения об оценке.

</dd> <dt>

*pUnk* 
</dt> <dd>

Указатель на объект, который будет выполнять команду.

</dd> <dt>

*time* 
</dt> <dd>

Время выполнения команды в очереди или команд.

</dd> <dt>

*IID* 
</dt> <dd>

Указатель на глобальный уникальный идентификатор (**GUID**) вызываемого интерфейса.

</dd> <dt>

*диспидмесод* 
</dt> <dd>

Вызываемый метод интерфейса.

</dd> <dt>

*вфлагс* 
</dt> <dd>

Флаги, описывающие контекст вызова. Этот параметр поддерживает те же флаги, что и метод **IDispatch:: Invoke** .

</dd> <dt>

*каргс*

Число переданных аргументов.

## <a name="return-value"></a>Возвращаемое значение

\_При успешном выполнении возвращает значение ОК. Возвращает E \_ OUTOFMEMORY, если *ппкмд* возвращает из создания нового объекта [**кдеферредкомманд**](cdeferredcommand.md) со значением **null**. В противном случае возвращает **значение HRESULT** , указывающее на ошибку при попытке создать новый объект **кдеферредкомманд** . Если возникает ошибка, объект не помещается в очередь.

## <a name="remarks"></a>Remarks

Новый объект [**кдеферредкомманд**](cdeferredcommand.md) будет инициализирован с параметрами и будет добавлен в очередь во время создания. Этот метод аналогичен методу **IDispatch:: Invoke** .

Значения параметра *вфлагс* включают следующее:



| Значение                    | Описание                                                                                                                                                          |
|--------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_метод Dispatch         | Элемент выполняется как метод. Если свойство имеет такое же имя, \_ можно задать и флаг диспетчеризации пропертижет.                                       |
| \_ПРОПЕРТИЖЕТ диспетчеризации    | Элемент извлекается как свойство или элемент данных.                                                                                                          |
| \_ПРОПЕРТИПУТ диспетчеризации    | Элемент изменяется как свойство или элемент данных.                                                                                                            |
| \_ПРОПЕРТИПУТРЕФ диспетчеризации | Элемент изменяется с помощью ссылочного назначения, а не назначения значения. Это значение допустимо, только если свойство принимает ссылку на объект. |
