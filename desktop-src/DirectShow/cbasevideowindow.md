# <a name="cbasevideowindow-class"></a>Класс Кбасевидеовиндов

![Иерархия классов кбасевидеовиндов](images/wctrl01.png)

`CBaseVideoWindow`Класс реализует методы **IDispatch** интерфейса [**ивидеовиндов**](/windows/desktop/api/Control/nn-control-ivideowindow) . Оставшиеся методы **ивидеовиндов** оставлены чисто виртуальными методами и должны быть реализованы производным классом.

Методы **IDispatch** в этом классе являются стандартными реализациями и подробно описаны здесь.
