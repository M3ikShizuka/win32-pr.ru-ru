# <a name="y-property"></a>y, свойство

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`y`Свойство задает или получает координаты y верхнего левого угла прямоугольника.

``` syntax
[ iYPos = ] DVDRect.y
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает точку по оси Y левого верхнего угла прямоугольника в виде целого числа.

## <a name="remarks"></a>Комментарии

Это свойство доступно для чтения и записи и не имеет значения по умолчанию.
