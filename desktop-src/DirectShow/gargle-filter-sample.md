# <a name="gargle-filter-sample"></a>Образец фильтра гаргле

## <a name="description"></a>Описание

Фильтр Гаргле является фильтром преобразования звуковых эффектов.

Фильтр Гаргле модуляцию аудио-сигнала, передавая его, путем умножения волнового сигнала на другой волн, который создается в фильтре математически. Модулатинг волна по умолчанию — треугольная волна. Страница свойств также предлагает альтернативу квадратной волны. Частоту модулатинг Wave можно задать с помощью страницы свойств фильтра. При низкой частоте модуляции этот результат звучит как тремоло. При более высоких частоте модуляции он звучит как искажение, добавляя дополнительные частоты выше и ниже первоначального некачественного звука.
