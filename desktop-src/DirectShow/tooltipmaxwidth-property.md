# <a name="tooltipmaxwidth-property"></a>Тултипмаксвидс, свойство

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`ToolTipMaxWidth`Свойство задает или получает максимальную ширину всплывающей подсказки, связанной с объектом **мсвебдвд** .

``` syntax
[ iMaxWidth = ] MSWebDVD.ToolTipMaxWidth
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает максимальную ширину всплывающей подсказки приложения в виде целого числа.

## <a name="remarks"></a>Комментарии

Это свойство доступно для чтения и записи и имеет значение по умолчанию 200 пикселей.
