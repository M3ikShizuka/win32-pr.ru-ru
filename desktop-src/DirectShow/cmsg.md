# <a name="cmsg-class"></a>Класс КМСГ

Класс [**кмсгсреад**](cmsgthread.md) обеспечивает поддержку для рабочего потока, в котором запросы могут быть отправлены асинхронно, а не напрямую. Класс [**камсреад**](camthread.md) предоставляет рабочий поток, к которому могут отправляться отдельные запросы. Только один клиент может выполнить запрос за раз, и клиент блокируется до тех пор, пока рабочий поток не завершит запрос. В отличие от этого класс **кмсгсреад** предоставляет рабочий поток, в который может быть отправлено любое количество запросов. Запросы (в форме объекта) помещаются `CMsg` в очередь и выполняются в асинхронном режиме. Ответ или возвращаемое значение не получено.



| Элементы данных              | Описание                                                                                                                                                                       |
|---------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| dwFlags                   | Параметр Flag для кода запроса.                                                                                                                                               |
| лппарам                   | Данные, необходимые рабочему потоку в качестве параметра или возвращаемых значений. Эти данные не должны основываться на стеке, так как на них будет ссылка в течение некоторого времени после завершения операции очереди. |
| певент                    | Объект события, который может сообщить рабочему потоку для указания на завершение операции.                                                                                         |
| Uiacp                      | Код запроса, определяемый клиентом класса потока и понятный переопределенной функцией рабочего потока.                                                           |
| Функции элементов          | Описание                                                                                                                                                                       |
