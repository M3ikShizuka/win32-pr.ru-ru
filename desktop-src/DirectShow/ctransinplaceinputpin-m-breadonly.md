# <a name="ctransinplaceinputpinm_breadonly-member"></a>Элемент Ктрансинплацеинпутпин:: m \_ бреадонли

Флаг, указывающий, доступен ли входной поток только для чтения. Вышестоящий фильтр задает эти сведения при вызове метода [**нотифяллокатор**](ctransinplaceinputpin-notifyallocator.md) . По умолчанию используется значение **false**.

## <a name="syntax"></a>Синтаксис


```C++
BOOL m_bReadOnly;
```
