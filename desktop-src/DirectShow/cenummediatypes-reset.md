# <a name="cenummediatypesreset-method"></a>Ценуммедиатипес. Reset, метод

`Reset`Метод сбрасывает последовательность перечисления до начала. Этот метод реализует метод [**иенуммедиатипес:: Reset**](/windows/desktop/api/Strmif/nf-strmif-ienummediatypes-reset) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Reset();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение S \_ ОК.
