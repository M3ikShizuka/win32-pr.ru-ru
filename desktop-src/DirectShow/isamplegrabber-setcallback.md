# <a name="isamplegrabbersetcallback-method"></a>Метод Исамплеграббер:: Сеткаллбакк

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Метод **сеткаллбакк** задает метод обратного вызова для вызова входящих образцов.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetCallback(
   ISampleGrabberCB *pCallback,
   long             WhichMethodToCallback
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*pCallback* 
</dt> <dd>

Указатель на интерфейс [**исамплеграбберкб**](isamplegrabbercb.md) , содержащий метод обратного вызова, или **значение NULL** для отмены обратного вызова.

</dd> <dt>

*вхичмесодтокаллбакк* 
</dt> <dd>

Индекс, указывающий метод обратного вызова. Должно иметь одно из следующих значений.



| Значение | Описание                                                                                                                                                                                     |
|-------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0     | Образец фильтра захвата вызывает метод [**исамплеграбберкб:: самплекб**](isamplegrabbercb-samplecb.md) . Этот метод получает указатель [**имедиасампле**](/windows/desktop/api/Strmif/nn-strmif-imediasample) .               |
| 1     | Образец фильтра захвата вызывает метод [**исамплеграбберкб:: буфферкб**](isamplegrabbercb-buffercb.md) . Этот метод получает указатель на буфер, содержащийся в примере носителя. |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если этот метод завершается успешно, возвращается значение **S \_ ОК**. В противном случае возвращается код ошибки **HRESULT** .

## <a name="remarks"></a>Комментарии

Поток обработки данных блокируется до тех пор, пока метод обратного вызова не вернет значение. Если обратный вызов не возвращает быстрое значение, он может помешать воспроизведению.

Фильтр не вызывает функцию обратного вызова для предварительных образцов или для образцов, в которых элемент **двстреамид** структуры [**\_ \_ свойств SAMPLE2**](/windows/win32/api/strmif/ns-strmif-am_sample2_properties) не является \_ потоком \_ мультимедиа.
