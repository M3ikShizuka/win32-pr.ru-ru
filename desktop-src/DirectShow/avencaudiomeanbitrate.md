# <a name="avencaudiomeanbitrate-property"></a>Авенкаудиомеанбитрате, свойство

Указывает среднюю скорость потока закодированного аудиопотока в битах в секунду. Это свойство применяется только к режимам кодирования с постоянным битом (CBR) и переменной скоростью с переменным битом (VBR).

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенккоммонмеанбитрате**

## <a name="property-value"></a>Значение свойства

Кодировщики могут реализовывать это свойство как перечислимый набор или как линейный Диапазон.
