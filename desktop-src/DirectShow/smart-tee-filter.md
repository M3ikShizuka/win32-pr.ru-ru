# <a name="smart-tee-filter"></a>Интеллектуальный фильтр Tee

Фильтр Smart tee используется в графах видеозаписи для разбиения видеопотока в поток предварительного просмотра и поток записи. Это выполняется без дополнительных копирований данных. Выходные контакты поддерживают любые типы мультимедиа, поддерживаемые в нисходящем соединении.

Фильтр Smart tee полезен, если фильтр видеозаписи не предоставляет отдельные ПИН-коды для записи и предварительного просмотра. Фильтр Smart tee доставляет данные предварительного просмотра только в том случае, если это не повредит производительность. Он также удаляет метки времени из предварительной версии потока. При необходимости построитель диаграмм для записи автоматически вставляет фильтр Smart Tee. Дополнительные сведения см. в разделе [объединение видеозаписей и предварительной версии](combining-video-capture-and-preview.md).

ПИН-код записи является выходным закреплением 0, а предварительный ПИН-код — выходным закреплением 1.
