# <a name="defaulttrans-attribute"></a>Атрибут дефаулттранс

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`defaulttrans`Атрибут задает идентификатор класса (CLSID) перехода по умолчанию для временной шкалы.

## <a name="possible-values"></a>Возможные значения

Значение должно быть строкой в формате {00000000-0000-0000-0000-000000000000} .

## <a name="applies-to"></a>Применение

[**графика**](timeline-element.md)
