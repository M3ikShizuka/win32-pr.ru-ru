# <a name="avdecaacdownmixmode-property"></a>Авдекаакдовнмиксмоде, свойство

Указывает, использует ли декодер AAC стандартные уравнения MPEG-2/MPEG-4 стерео довнмикс или использует нестандартный довнмикс.

Примером нестандартного довнмикс является тот, который определен документом АРИБ STD-B21 версии 4,4.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авдекаакдовнмиксмоде**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавдекаакдовнмиксмоде**](/windows/win32/api/codecapi/ne-codecapi-eavdecaacdownmixmode) .
