# <a name="bda-media-types"></a>Типы файлов BDA

В следующих таблицах описываются типы носителей, используемые для фильтров цифрового телевидения BDA.

## <a name="antenna"></a>Антенн

Этот тип мультимедиа используется для подключения ПИН-кода между фильтром поставщика сети и фильтром тюнера.



| Метка | Значение |
|-------------|----------------------------------|
| Основной тип  | КСДАТАФОРМАТ \_ тип \_ BDA \_ антенна |
| Subtype     | МЕДИАСУБТИПЕ \_ None               |
| Тип формата | Отформатировать \_ нет                     |



 

## <a name="intermediate-frequency-if-signal"></a>Сигнал промежуточной частоты (IF)

Этот тип мультимедиа представляет сигнал, который отправляется из тюнера в демодулятор.



| Метка | Значение |
|-------------|-------------------------------------|
| Основной тип  | КСДАТАФОРМАТ \_ введите \_ BDA, \_ Если \_ сигнал |
| Subtype     | МЕДИАСУБТИПЕ \_ None                  |
| Тип формата | Отформатировать \_ нет                        |
