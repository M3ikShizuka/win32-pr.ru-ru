# <a name="dmo-demo-sample"></a>Демонстрационный пример DMO

## <a name="description"></a>Описание

В этом примере приложения демонстрируется использование [объектов мультимедиа DirectX](directx-media-objects.md) (DMO) в приложении. Он передает звуковые данные из WAV-файла с помощью звукового сигнала DMO в буфер DirectSound.

## <a name="usage"></a>Использование

Нажмите кнопку **звукового файла** , чтобы загрузить звуковой файл. Используйте раскрывающееся поле со списком, чтобы выбрать звуковой эффекты DMO. Нажмите кнопку **Воспроизведение** , чтобы услышать звуковой файл, воспроизводимый с звуковым действием.

## <a name="downloading-the-sample"></a>Загрузка образца

Чтобы скачать примеры пакета SDK для DirectShow, установите последнюю версию [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Этот пример устанавливается по следующему пути: *\[ \] корневые примеры SDK* \\ \\ мультимедиа \\ DirectShow \\ DMO \\ дмодемо.
