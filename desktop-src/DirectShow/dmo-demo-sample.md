# <a name="dmo-demo-sample"></a>DMO Демонстрационный пример

## <a name="description"></a>Описание

в этом примере приложения демонстрируется использование [объектов мультимедиа DirectX](directx-media-objects.md) (DMO) в приложении. он передает звуковые данные из WAV-файла с помощью звукового DMO в буфер DirectSound.

## <a name="usage"></a>Использование

Нажмите кнопку **звукового файла** , чтобы загрузить звуковой файл. Используйте раскрывающееся поле со списком, чтобы выбрать DMO звукового эффекты. Нажмите кнопку **Воспроизведение** , чтобы услышать звуковой файл, воспроизводимый с звуковым действием.

## <a name="downloading-the-sample"></a>Загрузка образца

чтобы скачать примеры пакета SDK для DirectShow, установите последнюю версию [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

этот пример устанавливается по следующему пути: *\[ \] корневые примеры SDK* \\ \\ мультимедиа \\ DirectShow \\ DMO \\ дмодемо.
