# <a name="type-1-vs-type-2-dv-avi-files"></a>Файлы видео AVI типа-1 и Type-2

Видеокамеры создают чередование аудио-видео; в каждом кадре видео также содержатся звуковые данные. При сохранении DV-данных в файл AVI можно выбрать следующее:

-   Храните данные с чередованием в виде одного потока в файле AVI. Это называется файлом типа 1.
-   Разделите чередующиеся данные на отдельные потоки аудио и видео. Это называется файлом типа 2.

Для видеозаписи, где максимальная пропускная способность очень важна, лучше использовать файл типа 1, так как файлы типа 2 содержат избыточные звуковые данные. (В потоке видео по-прежнему есть звуковые данные. Звук просто скрыт, помечая поток как видео.) Кроме того, при написании файла типа 2 требуется дополнительное процессорное время для разделения потока с чередованием.

С другой стороны, файлы типа-1 менее эффективны для редактирования в режиме реального времени. Приложение должно извлекать звук из потока с чередованием, вносить изменения и переносить данные в промежутке. кроме того, формат type-1 несовместим с Microsoft® Video для Windows® (VFW). DirectShow могут работать с файлами обоих типов.

Файл типа 2 можно преобразовать в тип-1 с помощью фильтра [DV мультиплексор](dv-muxer-filter.md) . Файл типа 1 можно преобразовать в тип-2 с помощью фильтра [разделителя DV](dv-splitter-filter.md) . На следующей схеме показано различие между двумя форматами.
