# <a name="sources-of-dvd-region-information"></a>Источники сведений о регионе DVD

Существует три источника информации о регионе DVD, которые позволяют определить регион для воспроизведения DVD на компьютере под управлением Windows.

-   Название DVD-диска. Большинство заголовков DVD-дисков отмечены для определенного региона. Некоторые заголовки можно воспроизводить только в одном регионе. Некоторые из них можно воспроизвести в нескольких регионах, а другие могут играть все регионы. Регионы с 1 по 6 были определены в исходной спецификации DVD-Video. Регион 7 в настоящее время не определен. Регион 8 был добавлен в 1999 для специальных мест, например в самолете. Диски DVD-ROM (без видеозоны) не должны содержать код региона.
-   Дисковод DVD-дисков. Существует два типа дисководов DVD-дисков: RPC Phase 1 (RPC1) и RPC Phase 2 (RPC2). Диски RPC1 не поддерживают встроенную аппаратную поддержку для управления регионами. Для этих дисков Windows хранит сведения о количестве изменений в регионе, и регион можно изменить только один раз. RPC2 диски сохраняют сведения о количестве изменений в регионе на оборудовании, а общий регион таких дисков можно изменить до 5 раз.
-   DVD-декодер. Некоторые декодеры DVD, оборудование или программное обеспечение задаются для определенного региона. Как правило, пользователь не может изменить регион декодера.
