# <a name="avencddcentredownmixlevel-property"></a>Авенкддцентредовнмикслевел, свойство

Указывает уровень центра довнмикс в децибел (dB) в потоке цифрового звука Dolby. Это свойство применяется к кодировщикам цифрового аудио Dolby.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкддцентредовнмикслевел**
