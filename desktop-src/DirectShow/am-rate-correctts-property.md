# <a name="am_rate_correctts-property"></a>\_ \_ Свойство корректтс Rate

Навигатор DVD использует это свойство для информирования декодера о том, что навигатор устанавливает правильные отметки времени для образцов, которые он передает в декодер.



| Метка | Значение |
|-------------------|-------------------------------|
| Идентификатор GUID набора свойств | \_Кспропсетид \_ тсратечанже |
| Идентификатор свойства       | \_Ставка \_ корректтс           |
| Тип данных         | **LONG**                      |



 

## <a name="remarks"></a>Remarks

В более ранних версиях DVD-навигатора не были заданы правильные метки времени, когда скорость воспроизведения отличается от 1,0. Многие декодеры обдействуют эту проблему, игнорируя отметки времени во время перемотки назад или вперед и оценивая правильное время презентации.

Эти проблемы были исправлены в текущей версии навигатора по DVD. Чтобы обеспечить обратную совместимость с существующими декодерами, Навигатор DVD указывает, что штампы времени верны, установив \_ \_ свойство корректтс Rate в декодере со значением **true**. Если это свойство задано, декодер должен использовать фактические штампы времени вместо оценки времени представления.
