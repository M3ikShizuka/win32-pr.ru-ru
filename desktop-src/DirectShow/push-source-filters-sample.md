# <a name="push-source-filters-sample"></a>Пример фильтров источника push-уведомлений

## <a name="description"></a>Описание

Этот пример состоит из трех исходных фильтров, которые предоставляют следующие исходные данные в качестве видеопотока:

-   Кпушсаурцебитмап: одно битовое изображение (загруженное из текущего каталога)
-   Кпушсаурцебитмапсет: набор точечных рисунков (загруженных из текущего каталога)
-   Кпушсаурцедесктоп: копия текущего образа рабочего стола (только GDI)

## <a name="usage"></a>Использование

Чтобы использовать фильтр, загрузите его в Графедит и выводите его выходной ПИН-код. При этом будет подключен модуль подготовки видео (и, возможно, фильтр преобразователя цветового пространства), и вы сможете отобразить выходные данные. Если вы хотите вывести выходные данные в AVI-файл, загрузите его, загрузите фильтр записи файлов, введите имя выходных данных в модуль записи файлов и отфильтруйте выходной код фильтра Пушсаурце. Можно также загрузить и подключить видео компрессоры, видеоэффекты и т. д.

> [!Note]  
> Фильтр записи рабочего стола не поддерживает наложение оборудования, поэтому он не будет записывать видео, отображаемое в области наложения или курсоров, отображаемых через наложение оборудования. Он использует GDI для преобразования текущего образа рабочего стола в точечный рисунок, который передается в выходной закрепление в качестве примера носителя.

Этот пример устанавливается по следующему пути: *\[ \] корневые примеры SDK* \\ \\ мультимедиа \\ DirectShow \\ фильтры \\ пушсаурце.
