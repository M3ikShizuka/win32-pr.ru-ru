# <a name="avdecdddynamicrangescalehigh-property"></a>Авдекдддинамикранжескалехигх, свойство

Задает высокоуровневая вырезание, когда декодер выполняет динамическое управление диапазонами в потоке звука Dolby AC-3.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авдекдддинамикранжескалехигх**

## <a name="property-value"></a>Значение свойства

Значение этого свойства имеет следующий диапазон.



| Значение | Описание                                                    |
|-------|----------------------------------------------------------------|
| 0     | Отсутствует динамическое сжатие диапазона. Сохраните полный динамический диапазон. |
| 100   | Применить полное сжатие динамического диапазона.                          |



 

## <a name="remarks"></a>Комментарии

Это свойство применяется только в том случае, если свойство [**авдекддоператионалмоде**](avdecddoperationalmode-property.md) имеет значение еавдекддоператионалмоде \_ line.
