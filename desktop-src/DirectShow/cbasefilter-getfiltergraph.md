# <a name="cbasefiltergetfiltergraph-method"></a>Кбасефилтер. Жетфилтерграф, метод

`GetFilterGraph`Метод получает указатель на диспетчер графа фильтра.

## <a name="syntax"></a>Синтаксис


```C++
IFilterGraph* GetFilterGraph();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение [**кбасефилтер:: m \_ пграф**](cbasefilter-m-pgraph.md).
