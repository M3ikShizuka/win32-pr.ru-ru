# <a name="cbasewindowgetmemoryhdc-method"></a>Кбасевиндов. Жетмеморихдк, метод

`GetMemoryHDC`Метод извлекает маркер контекста устройства памяти (DC).

## <a name="syntax"></a>Синтаксис


```C++
virtual HDC GetMemoryHDC();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает маркер контроллера памяти.
