# <a name="cbaserendererm_pqsink-member"></a>Элемент Кбасерендерер:: m \_ пксинк

Указатель на объект, который получает сообщения контроля качества.

## <a name="syntax"></a>Синтаксис


```C++
IQualityControl *m_pQSink;
```



## <a name="remarks"></a>Remarks

Базовый класс не реализует контроль качества. Эта переменная члена по умолчанию **имеет значение NULL**.
