# <a name="ikspropertysetset-method"></a>Метод Икспропертисет:: Set

`Set`Метод задает свойство, определяемое идентификатором GUID набора свойств и идентификатором свойства.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Set(
  [in] REFGUID guidPropSet,
  [in] DWORD   dwPropID,
  [in] LPVOID  pInstanceData,
  [in] DWORD   cbInstanceData,
  [in] LPVOID  pPropData,
  [in] DWORD   cbPropData
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*гуидпропсет* \[ окне\]
</dt> <dd>

Идентификатор GUID набора свойств.

</dd> <dt>

*dwPropID* \[ окне\]
</dt> <dd>

Идентификатор свойства в наборе свойств.

</dd> <dt>

*пинстанцедата* \[ окне\]
</dt> <dd>

Указатель на буфер, содержащий данные экземпляра для свойства.

</dd> <dt>

*кбинстанцедата* \[ окне\]
</dt> <dd>

Размер буфера *пинстанцедата* в байтах.

</dd> <dt>

*ппропдата* \[ окне\]
</dt> <dd>

Указатель на буфер, содержащий значение свойства.

</dd> <dt>

*кбпропдата* \[ окне\]
</dt> <dd>

Сисе буфера *ппропдата* в байтах.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** . Ниже приведены возможные значения.



| Код возврата                                                                                              | Описание                                                                 |
|----------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                     | Успешно.<br/>                                                         |
| <dl> <dt>**\_набор свойств \_ E \_ не поддерживается**</dt> </dl> | Набор свойств не поддерживается.<br/>                               |
| <dl> <dt>**идентификатор "E \_ prop" \_ \_ не поддерживается**</dt> </dl>  | Идентификатор свойства не поддерживается для указанного набора свойств.<br/> |



 

## <a name="remarks"></a>Комментарии

> [!Note]  
> Другой интерфейс с таким именем существует в файле заголовка дсаунд. h. Два интерфейса несовместимы. Интерфейс **иксконтрол** , документированный в пакете DDK по DirectShow, теперь является рекомендуемым интерфейсом для передачи наборов свойств между драйверами WDM и компонентами пользовательского режима.

 

Перед Кспрокси. h необходимо включить KS. h.
