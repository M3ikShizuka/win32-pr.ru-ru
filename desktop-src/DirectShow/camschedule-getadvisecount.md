# <a name="camschedulegetadvisecount-method"></a>Камсчедуле. Жетадвисекаунт, метод

`GetAdviseCount`Метод извлекает количество ожидающих запросов рекомендаций.

## <a name="syntax"></a>Синтаксис


```C++
DWORD GetAdviseCount();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает число ожидающих запросов рекомендаций.
