# <a name="mspid"></a>мспид

> [!Note]  
> Это нерекомендуемый API. Новые приложения не должны использовать его.

 

Тип данных **мспид** определяет назначение Steam мультимедиа.


```C++
typedef GUID MSPID;
typedef REFGUID REFMSPID;
```



## <a name="remarks"></a>Комментарии

**Мспид** — это просто typedef для идентификатора GUID. Определены следующие Мспидс.



| Значение               | Описание           |
|---------------------|-----------------------|
| МСПИД \_ примаривидео | Основной видеопоток. |
| МСПИД \_ примаряудио | Основной звуковой поток. |



 

Тип **рефмспид** определяет ссылку на **мспид**.
