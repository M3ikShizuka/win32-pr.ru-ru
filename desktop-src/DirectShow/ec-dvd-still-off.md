# <a name="ec_dvd_still_off"></a>\_неподвижный DVD-диск EC \_ \_

Сигнализирует об окончании всех по-прежнему (PGC, Cell или ВОБУ).

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

Ноль.

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Ноль.

</dd> </dl>

## <a name="remarks"></a>Remarks

Это событие указывает, что все активные в настоящее время события по-прежнему освобождены.

Это событие возникает во всех доменах.
