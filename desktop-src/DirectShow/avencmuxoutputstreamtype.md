# <a name="avencmuxoutputstreamtype-property"></a>Авенкмуксаутпутстреамтипе, свойство

Указывает тип выходного потока, созданного мультиплексором.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмуксаутпутстреамтипе**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенкмуксаутпут**](/windows/win32/api/codecapi/ne-codecapi-eavencmuxoutput) .
