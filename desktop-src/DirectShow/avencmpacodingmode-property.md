# <a name="avencmpacodingmode-property"></a>Авенкмпакодингмоде, свойство

Указывает режим кодирования звука MPEG-1/2. Это свойство применяется к кодировщикам аудио в формате MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпакодингмоде**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенкмпакодингмоде**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpacodingmode) .
