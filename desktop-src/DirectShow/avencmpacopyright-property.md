# <a name="avencmpacopyright-property"></a>Авенкмпакопиригхт, свойство

Задает значение по умолчанию для бита авторского права в аудиопотока MPEG-1. Это свойство применяется к кодировщикам аудио в формате MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпакопиригхт**

## <a name="property-value"></a>Значение свойства

Это свойство может иметь следующие значения.



| Значение          | Описание           |
|----------------|-----------------------|
| ВАРИАНТ \_ false | Бит авторских прав отключен. |
| ВАРИАНТ \_ true  | Бит авторского права включен.  |



 

## <a name="remarks"></a>Remarks

Кодировщик может переопределить этот параметр на основе содержимого входного потока.
