# <a name="dmo-enumerated-types"></a>Перечислимые типы DMO

В этом разделе описываются перечислимые типы объектов мультимедиа Microsoft DirectX (DMO).



| Перечисляемый тип                                                            | Описание                                                                  |
|----------------------------------------------------------------------------|------------------------------------------------------------------------------|
| [**\_ \_ флаги перечисления DMO**](/previous-versions/windows/desktop/api/Dmoreg/ne-dmoreg-dmo_enum_flags)                                 | Определяет флаги, определяющие условия поиска при перечислении дмос.            |
| [**\_ \_ \_ Флаги буфера ВХОДных данных DMO \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_input_data_buffer_flags)     | Определяет флаги, описывающие входной буфер.                                 |
| [**\_ \_ \_ Флаги сведений о входном потоке DMO \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_input_stream_info_flags)     | Определяет флаги, описывающие входной поток.                                 |
| [**\_ \_ \_ Флаги буфера выходных данных DMO \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_output_data_buffer_flags)   | Определяет флаги, описывающие выходной буфер.                                |
| [**\_ \_ \_ Флаги сведений о ПОТОКе вывода DMO \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_output_stream_info_flags)   | Определяет флаги, описывающие выходной поток.                                |
| [**\_ \_ Флаги вывода процесса DMO \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_process_output_flags)            | Определяет флаги, указывающие запросы на обработку выходных данных.                       |
| [**\_ \_ Флаги регистра DMO**](/previous-versions/windows/desktop/api/Dmoreg/ne-dmoreg-dmo_register_flags)                         | Определяет флаги, указывающие сведения реестра для DMO.                   |
| [**\_ \_ Флаги установки типа DMO \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_set_type_flags)                        | Определяет флаги для установки типа мультимедиа в потоке.                        |
| [**\_ \_ \_ Флаги видеопотока вывода DMO \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_video_output_stream_flags) | Определяет флаги, описывающие запрошенные функции для оптимизации видео.      |
| [**\_Тип кривой \_ MP**](/previous-versions/windows/desktop/api/Medparam/ne-medparam-mp_curve_type)                                   | Определяет кривую, которую параметр мультимедиа следует за сегментом конверта. |
| [**тип пакета управления \_**](/previous-versions/windows/desktop/api/Medparam/ne-medparam-mp_type)                                                | Указывает тип данных для параметра.                                     |
