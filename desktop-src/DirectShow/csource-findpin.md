# <a name="csourcefindpin-method"></a>Ксаурце. Финдпин, метод

`FindPin`Метод извлекает ПИН-код с указанным идентификатором. Этот метод реализует метод [**ибасефилтер:: финдпин**](/windows/desktop/api/Strmif/nf-strmif-ibasefilter-findpin) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT FindPin(
   LPCWSTR Id,
   IPin    **ppPin
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Id* 
</dt> <dd>

Указатель на строку, завершающуюся нулем, которая определяет ПИН-код.

</dd> <dt>

*пппин* 
</dt> <dd>

Получает указатель на интерфейс [**Ипин**](/windows/desktop/api/Strmif/nn-strmif-ipin) ПИН-кода. Если метод завершается с ошибкой, \* *пппин* имеет значение **null** .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из значений **HRESULT** , приведенных в следующей таблице.



| Код возврата                                                                                       | Описание                                           |
|---------------------------------------------------------------------------------------------------|-------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>              | Успешно.<br/>                                   |
| <dl> <dt>**\_указатель E**</dt> </dl>         | **Пустой** аргумент указателя.<br/>                 |
| <dl> <dt>**VFW \_ E \_ не \_ найден**</dt> </dl> | Не удалось найти ПИН-код с этим идентификатором.<br/> |



 

## <a name="remarks"></a>Комментарии

Первый ПИН-код всегда имеет имя "1"; второй ПИН-код называется "2"; и т. д. Дополнительные сведения см. в разделе [**ксаурцестреам:: QueryId**](csourcestream-queryid.md).
