# <a name="playattimeintitle-method"></a>Метод Плайаттимеинтитле

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`PlayAtTimeInTitle`Метод запускает воспроизведение в указанное время в заданном заголовке.

``` syntax
MSWebDVD.PlayAtTimeInTitle(sTime, iTitle)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="sTime"></span><span id="stime"></span><span id="STIME"></span>*стиме*
</dt> <dd>

Указывает время начала воспроизведения в виде строки. Строка должна иметь формат "чч: мм: СС: FF" (указание часов, минут, секунд, кадров).

</dd> <dt>

<span id="iTitle"></span><span id="ititle"></span><span id="ITITLE"></span>*ититле*
</dt> <dd>

Задает индекс заголовка в виде целого числа.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.
