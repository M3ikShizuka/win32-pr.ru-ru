# <a name="imediadetwritebitmapbits-method"></a>Метод Имедиадет:: Вритебитмапбитс

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`WriteBitmapBits`Метод извлекает кадр видео в указанное время мультимедиа и записывает его в файл. Кадр видео всегда находится в 24-разрядном формате RGB.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT WriteBitmapBits(
   double StreamTime,
   long   Width,
   long   Height,
   BSTR   Filename
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*стреамтиме* 
</dt> <dd>

Время извлечения кадра видео.

</dd> <dt>

*Width* 
</dt> <dd>

Ширина изображения в пикселях.

</dd> <dt>

*Height* 
</dt> <dd>

Высота изображения в пикселях.

</dd> <dt>

*Имя файла* 
</dt> <dd>

Путь к файлу, в котором нужно сохранить точечный рисунок. Если файл уже существует, этот метод перезаписывает его.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает \_ ОК успешно. В противном случае возвращает значение **HRESULT** , указывающее причину ошибки. Ниже перечислены возможные коды ошибок.



| Код возврата                                                                                             | Описание                                                                                       |
|---------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_интерфейс E**</dt> </dl>           | Не удалось добавить [**образец фильтра захвата**](sample-grabber-filter.md) к диаграмме.<br/> |
| <dl> <dt>**\_Ошибка E**</dt> </dl>                  | Ошибка.<br/>                                                                               |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl>           | Недостаточно памяти.<br/>                                                                   |
| <dl> <dt>**\_непредвиденная E**</dt> </dl>            | Непредвиденная ошибка.<br/>                                                                      |
| <dl> <dt>**STG \_ E \_ ACCESSDENIED**</dt> </dl>     | Не удается перезаписать файл.<br/>                                                                 |
| <dl> <dt>**VFW \_ E \_ инвалидмедиатипе**</dt> </dl> | Недопустимый тип носителя.<br/>                                                                    |



 

## <a name="remarks"></a>Комментарии

Перед вызовом этого метода задайте имя файла и поток, вызвав [**имедиадет::p UT \_ filename**](imediadet-put-filename.md) и [**имедиадет::p UT \_ куррентстреам**](imediadet-put-currentstream.md).

Этот метод помещает средство обнаружения мультимедиа в режим захвата точечного рисунка. После вызова этого метода различные методы потоковой информации в **имедиадет** не работают, если не создать новый экземпляр детектора мультимедиа.

> [!Note]  
> Файл заголовка Кедит. h несовместим с заголовками Direct3D позднее версии 7.

 

> [!Note]  
> Чтобы получить Кедит. h, скачайте [обновление Microsoft Windows SDK для Windows Vista и платформа .NET Framework 3,0](https://msdn.microsoft.com/windowsvista/bb980924.aspx). Кедит. h недоступен в Microsoft Windows SDK для Windows 7 и платформа .NET Framework 3,5 с пакетом обновления 1 (SP1).
