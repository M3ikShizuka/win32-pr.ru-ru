# <a name="avencvideooutputcolorprimaries-property"></a>Авенквидеуутпутколорпримариес, свойство

Задает цветовые цвета для закодированного видео.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеуутпутколорпримариес**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенквидеоколорпримариес**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolorprimaries) .
