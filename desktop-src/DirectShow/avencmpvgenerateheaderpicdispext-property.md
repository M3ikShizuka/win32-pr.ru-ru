# <a name="avencmpvgenerateheaderpicdispext-property"></a>Авенкмпвженератехеадерпикдиспекст, свойство

Указывает, создает ли кодировщик заголовки расширения изображения. Это свойство применяется к кодировщикам видео MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпвженератехеадерпикдиспекст**

## <a name="remarks"></a>Remarks

Если значение является **вариантным \_ true**, кодировщик создает заголовки расширений изображений.
