# <a name="ctransformoutputpincurrentmediatype-method"></a>Ктрансформаутпутпин. Куррентмедиатипе, метод

`CurrentMediaType`Метод извлекает тип носителя для текущего подключения ПИН-кода.

## <a name="syntax"></a>Синтаксис


```C++
CMediaType& CurrentMediaType();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает ссылку на переменную-член [**кбасепин:: m \_ MT**](cbasepin-m-mt.md) .
