# <a name="avddsurroundmode-property"></a>Авддсурраундмоде, свойство

Указывает, кодируется ли звук в окружающее окружение Dolby.

Это свойство доступно только для чтения.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авддсурраундмоде**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавддсурраундмоде**](/windows/desktop/api/codecapi/ne-codecapi-eavddsurroundmode) .
