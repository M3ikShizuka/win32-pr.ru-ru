# <a name="gettitleparentallevels-method"></a>Метод Жеттитлепаренталлевелс

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`GetTitleParentalLevels`Метод получает уровни родительского управления для указанного заголовка.

``` syntax
[ iLevels = ] MSWebDVD.GetTitleParentalLevels(iTitle)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="iTitle"></span><span id="ititle"></span><span id="ITITLE"></span>*ититле*
</dt> <dd>

Задает заголовок в виде целого числа.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение, отдельные биты которого указывают, какие уровни родительского управления (ПМЛ) установлены в указанном заголовке.

## <a name="remarks"></a>Комментарии

Заголовок может содержать главы или даже короткие сегменты, имеющие ПМЛ, отличающиеся от общего ПМЛ для заголовка. Используйте этот метод, чтобы определить все Пмлс, которые будут обнаружены при воспроизведении указанного заголовка. Возвращаемое целое число — это набор битовых флагов, которые определены, как показано в таблице ниже. Выполните побитовую операцию и для *илевелс* и каждого возможного значения. Если операция возвращает **значение true**, это означает, что в этом заголовке будет обнаружено ПМЛ.



| Значение  | Описание          |
|--------|----------------------|
| 0x100  | Родительский уровень 1 DVD |
| 0x200  | Родительский уровень DVD 2 |
| 0x400  | Родительский уровень DVD 3 |
| 0x800  | Родительский уровень DVD 4 |
| 0x1000 | Родительский уровень диска DVD 5 |
| 0x2000 | Родительский уровень DVD 6 |
| 0x4000 | Родительский уровень DVD 7 |
| 0x8000 | Родительский уровень DVD 8 |
