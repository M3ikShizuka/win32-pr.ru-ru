# <a name="dvduniqueid-property"></a>Двдуникуеид, свойство

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`DVDUniqueID`Свойство получает созданное системой число, уникально идентифицирующее текущий диск.

``` syntax
[ iDiscID = ] MSWebDVD.DVDUniqueID
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение, уникально идентифицирующее текущий диск.

## <a name="remarks"></a>Remarks

Это свойство доступно только для чтения и не имеет значения по умолчанию. Идентификатор идентификатора (ID) не уникален, но он уникален для всех практических целей. Этот идентификатор применяется ко всем реплицируемым копиям диска. Иными словами, все копии указанного фильма имеют одинаковый идентификатор. ИДЕНТИФИКАТОР основан на размерах файлов, датах и других данных, а не на значении БКА.
