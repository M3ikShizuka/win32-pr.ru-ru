# <a name="tape-location-search"></a>Поиск расположения на ленте

Хотя DV-устройства поддерживают команду для поиска по коду времени или по номеру абсолютной записи (АТН), драйвер МСДВ не имеет стандартного, независимого от устройства способа доступа к этим функциям. Единственным вариантом является отправка необработанной команды AV/C драйверу, как описано в разделе, [поставляющем необработанные команды AV/c](issuing-raw-av-c-commands.md).

Драйвер УВК поддерживает набор свойств для поиска по расположению ленты. Чтобы отправить команду поиска, используйте интерфейс **иксконтрол** и задайте \_ свойство пропсетид ext \_ Transport. Использование набора свойств является более безопасным, чем отправка необработанных команд AV/C на устройство, так как команды AV/C обходят фильтр и непосредственно переходят к драйверу устройства.
