# <a name="cbaseinputpinm_breadonly-member"></a>Элемент Кбасеинпутпин:: m \_ бреадонли

Флаг, указывающий, создает ли распределитель образцы мультимедиа только для чтения. Если **значение — true**, распределитель создает образцы, которые доступны только для чтения.

## <a name="syntax"></a>Синтаксис


```C++
BYTE m_bReadOnly;
```
