# <a name="iamtimelineobjsetmuted-method"></a>Метод Иамтимелинеобж:: Сетмутед

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`SetMuted`Метод задает состояние отключения объекта. Отключенный объект не готовится к просмотру, но остается на временной шкале.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetMuted(
   BOOL newVal
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*неввал* 
</dt> <dd>

Флаг, указывающий состояние отключения. Если **значение — true**, объект и все его дочерние элементы будут отключены.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если этот метод завершается успешно, возвращается значение **S \_ ОК**. В противном случае возвращается код ошибки **HRESULT** .

## <a name="remarks"></a>Remarks

При отключении звука объект также выключает все дочерние элементы, содержащиеся в объекте. Например, если выключить трассировку, то переходы, источники и эффекты в этой дорожке также будут отключены. После того как объект больше не будет отключен, его дочерние элементы возвращаются к предыдущему состоянию, которое может быть отключено или отключено.
