# <a name="track"></a>Track

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Объект Track содержит исходные объекты. Чтобы создать этот объект, вызовите метод [**иамтимелине:: креатимптиноде**](iamtimeline-createemptynode.md) .

Объект Track предоставляет следующие интерфейсы:

-   [**иамтимелиниффектабле**](iamtimelineeffectable.md)
-   [**иамтимелинеобж**](iamtimelineobj.md)
-   [**иамтимелинесплиттабле**](iamtimelinesplittable.md)
-   [**иамтимелинетракк**](iamtimelinetrack.md)
-   [**иамтимелинетрансабле**](iamtimelinetransable.md)
-   [**иамтимелиневиртуалтракк**](iamtimelinevirtualtrack.md)
