# <a name="karaokeaudiopresentationmode-property"></a>Караокеаудиопресентатионмоде, свойство

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`KaraokeAudioPresentationMode`Свойство задает или получает правый левый набор динамиков для вспомогательных каналов караоке.

``` syntax
[iMode ] = MSWebDVD.KaraokeAudioPresentationMode
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение, содержащее набор битовых флагов, указывающих, как вспомогательные каналы караоке довнмиксед на левый и правый динамики.

## <a name="remarks"></a>Комментарии

Это свойство доступно для чтения и записи и имеет нулевое значение по умолчанию.

Звуковые каналы отсчитываются от нуля, поэтому каналы 0 и 1 обычно представляют правый и левый каналы динамиков, а каналы 2 – 4 — три вспомогательных канала караоке. Когда объект Мсвебдвд переходит в режим Караоке, он автоматически выключает каналы 2 и выше. Используйте побитовые операции **или** , чтобы задать соответствующий бит для отправки вспомогательного канала на левый динамик, правый докладчик, оба динамика (оба бита на) или отсутствие динамиков (оба бита выключены). По умолчанию эти биты отключены каждый раз, когда Навигатор DVD переходит в режим караоке. Значения битов и соответствующие им действия приведены в следующей таблице.



| Значение  | Описание                            |
|--------|----------------------------------------|
| 0x0004 | Довнмикс канал 2 на левый динамик  |
| 0x0008 | Довнмикс Channel 3 к левому докладчику  |
| 0x0010 | Канал довнмикс 4 на левый динамик  |
| 0x0400 | Довнмикс канал 2 к правому докладчику |
| 0x0800 | Довнмикс Channel 3 к правому докладчику |
| 0x1000 | Довнмикс Channel 4 к правому докладчику |
