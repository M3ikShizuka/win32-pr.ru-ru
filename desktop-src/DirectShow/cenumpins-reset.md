# <a name="cenumpinsreset-method"></a>Ценумпинс. Reset, метод

`Reset`Метод сбрасывает последовательность перечисления до начала. Этот метод реализует метод [**иенумпинс:: Reset**](/windows/desktop/api/Strmif/nf-strmif-ienumpins-reset) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Reset();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение S \_ ОК.
