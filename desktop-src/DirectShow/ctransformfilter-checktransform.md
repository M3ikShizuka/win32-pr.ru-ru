# <a name="ctransformfilterchecktransform-method"></a>Ктрансформфилтер. Чекктрансформ, метод

`CheckTransform`Метод проверяет, совместим ли входной тип носителя с выходным типом носителя.

## <a name="syntax"></a>Синтаксис


```C++
virtual HRESULT CheckTransform(
   const CMediaType *mtIn,
   const CMediaType *mtOut
) = 0;
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*мтин* 
</dt> <dd>

Указатель на объект [**кмедиатипе**](cmediatype.md) , указывающий тип входных данных.

</dd> <dt>

*мтаут* 
</dt> <dd>

Указатель на объект [**кмедиатипе**](cmediatype.md) , указывающий тип выходных данных.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** . Возможные значения включают в себя, показанные в следующей таблице.



| Код возврата                                                                                                | Описание                                    |
|------------------------------------------------------------------------------------------------------------|------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                       | Типы мультимедиа совместимы.<br/>     |
| <dl> <dt>**\_тип VFW \_ E \_ не \_ принят**</dt> </dl> | Типы мультимедиа несовместимы.<br/> |



 

## <a name="remarks"></a>Комментарии

Производный класс должен реализовывать этот метод. Возврат \_ ОК, если фильтр может принимать оба указанных типа мультимедиа или код ошибки в противном случае.
