# <a name="eject-method"></a>Метод reeject

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`Eject`Метод извлекает или вставляет диск из DVD-дисковода.

``` syntax
        MSWebDVD.Eject()
```

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Комментарии

На некоторых DVD-дисководах этот метод действует как переключатель, извлечение и вставка при альтернативных вызовах. Это зависит от оборудования.
