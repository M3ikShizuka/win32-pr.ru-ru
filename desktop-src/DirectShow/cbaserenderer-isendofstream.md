# <a name="cbaserendererisendofstream-method"></a>Кбасерендерер. Исендофстреам, метод

`IsEndOfStream`Метод запрашивает, было ли получено уведомление о завершении потока.

## <a name="syntax"></a>Синтаксис


```C++
BOOL IsEndOfStream();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает флаг [**\_ беос кбасерендерер:: m**](cbaserenderer-m-beos.md) .
