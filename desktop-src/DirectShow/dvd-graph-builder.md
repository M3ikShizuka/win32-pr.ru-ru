# <a name="dvd-graph-builder"></a>построитель DVD-Graph

построитель dvd-Graph — это вспомогательный объект для создания графов воспроизведения dvd-дисков. Приложения могут использовать этот компонент для создания графов фильтров для воспроизведения DVD-дисков и навигации. Создайте этот объект, вызвав **CoCreateInstance**.



| Метка | Значение |
|------------------|----------------------------------------------|
| Идентификатор класса | \_ДВДГРАФБУИЛДЕР CLSID                       |
| Интерфейсы       | [**идвдграфбуилдер**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) |
