# <a name="ctransformfilterm_bsampleskipped-member"></a>Элемент Ктрансформфилтер:: m \_ бсамплескиппед

Флаг, указывающий, был ли удален последний пример. Если метод [**Receive**](ctransformfilter-receive.md) удаляет пример, он устанавливает значение **true**.

## <a name="syntax"></a>Синтаксис


```C++
BOOL m_bSampleSkipped;
```
