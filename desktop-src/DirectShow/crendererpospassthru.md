# <a name="crendererpospassthru-class"></a>Класс Крендерерпоспасссру

![Иерархия классов крендерерпоспасссру](images/cutil14.png)

`CRendererPosPassThru`Класс обрабатывает команды Seek для фильтров модуля подготовки отчетов, передавая их в следующий фильтр.

Этот класс является производным от класса [**кпоспасссру**](cpospassthru.md) . Он добавляет поддержку кэширования меток времени из примеров по мере их поступления. Используйте этот класс так же, как класс **кпоспасссру** . Дополнительные сведения см. в документации по **кпоспасссру** .

Фильтр модуля подготовки отчетов должен обновить `CRendererPosPassThru` кэшированные временные метки объекта следующим образом:

-   Для каждого образца, полученного фильтром, вызовите метод [**крендерерпоспасссру:: регистермедиатиме**](crendererpospassthru-registermediatime.md) .
-   Когда фильтр останавливается или получает вызов **ендфлуш** , вызовите метод [**Крендерерпоспасссру:: ресетмедиатиме**](crendererpospassthru-resetmediatime.md) .
-   Когда фильтр получает уведомление об окончании потока, вызовите метод [**крендерерпоспасссру:: EOS**](crendererpospassthru-eos.md) .

Пример использования этого класса см. в исходном коде [**кбасерендерер**](cbaserenderer.md) .



| Открытые методы                                                            | Описание                                                         |
|---------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**крендерерпоспасссру**](crendererpospassthru-crendererpospassthru.md) | Метод конструктора.                                                 |
| [**жетмедиатиме**](crendererpospassthru-getmediatime.md)                 | Извлекает метки времени для текущего образца.                    |
| [**регистермедиатиме**](crendererpospassthru-registermediatime.md)       | Кэширует метки времени из текущего образца.                     |
| [**ресетмедиатиме**](crendererpospassthru-resetmediatime.md)             | Сбрасывает кэшированные отметки времени в ноль.                              |
| [**EOS**](crendererpospassthru-eos.md)                                   | Обновляет кэшированные метки времени после уведомления о завершении потока. |
