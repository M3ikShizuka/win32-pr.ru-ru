# <a name="am_rate_maxfulldatarate-property"></a>\_ \_ Свойство максфуллдатарате Rate

Возвращает максимальную скорость, с которой декодер может обрабатывать данные. Значением этого свойства является максимальная скорость воспроизведения декодера x 10000. Например, если максимальная скорость пересылки равна 2,0, значение этого свойства равно 20000.

Тип данных для этого свойства — **\_ максфуллдатарате**, то есть `typedef` для типа **Long**.

Это свойство доступно только для чтения.



| Метка | Значение |
|-------------------|------------------------------------|
| Идентификатор GUID набора свойств | \_Кспропсетид \_ тсратечанже      |
| Идентификатор свойства       | \_Ставка \_ максфуллдатарате          |
| Тип данных         | **AM \_ Максфуллдатарате** (**длинный**) |
