# <a name="list-of-multimedia-streaming-interfaces"></a>Список интерфейсов потоковой передачи мультимедиа

> [!Note]  
> Эти API-интерфейсы являются устаревшими. приложения должны использовать [**образец фильтра захвата**](sample-grabber-filter.md) или реализовать настраиваемый фильтр для получения данных из графа фильтра DirectShow.

 

Потоковая передача мультимедиа состоит из нескольких интерфейсов. в этой статье приводится краткое описание каждого интерфейса. Дополнительные сведения о потоковой передаче мультимедиа см. [в разделе сведения об архитектуре потоковой передачи мультимедиа](about-the-multimedia-streaming-architecture.md).

Эта статья состоит из следующих разделов:

-   [Базовые интерфейсы потоковой передачи мультимедиа](base-multimedia-streaming-interfaces.md)
-   [Интерфейсы потоковой передачи DirectDraw](directdraw-streaming-interfaces.md)
-   [Интерфейсы потоковой передачи аудио](audio-streaming-interfaces.md)
