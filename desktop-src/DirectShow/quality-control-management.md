# <a name="quality-control-management"></a>Управление Quality-Control

Контроль качества — это механизм для настройки скорости потока данных с помощью графа фильтра в ответ на производительность во время выполнения. Если фильтр модуля подготовки отчетов получает слишком много данных или слишком мало данных, он может отправить *качественное сообщение*. В сообщении о контроле запрашивается коррекция скорости передачи данных. По умолчанию сообщения о контроле качества передаются из модуля подготовки отчетов, пока не достигают фильтра, который может ответить (если таковой имеется). Приложение также может реализовать пользовательский диспетчер качества. В этом случае модуль подготовки отчетов передает качественные сообщения непосредственно менеджеру по качеству приложения.
