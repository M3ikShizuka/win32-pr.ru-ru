# <a name="defaultsubpicturelcid-property"></a>ДефаултсубпиктурелЦид, свойство

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`DVDAdm.DefaultSubpictureLCID`Свойство задает или получает параметр реестра для указанного пользователем кода языка по умолчанию для потока субтитров.

``` syntax
[ iSubpictureLCID = ] DVD.DVDAdm.DefaultSubpictureLCID
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение, представляющее указанный пользователем код языка по умолчанию, который хранится в параметрах реестра для приложения DVD. Это значение не обязательно соответствует потоку субтитров по умолчанию, созданному на DVD-диске. Диапазон допустимых кодов языка см. в документации по Win32 в пакете Platform SDK.

## <a name="remarks"></a>Remarks

Это свойство доступно для чтения и записи и не имеет значения по умолчанию. Если не указан LCID по умолчанию, объект Мсдвдадм будет воспроизводить поток субтитров, помеченный как поток по умолчанию на диске.
