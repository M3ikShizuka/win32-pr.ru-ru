# <a name="src-attribute"></a>Атрибут src

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`src`Атрибут указывает путь к файлу мультимедиа. Файл может быть любым типом, поддерживаемым DirectShow, или по-прежнему образом. Поддерживаемые форматы для изображений по-прежнему являются точечными рисунками, независимыми от устройства точечными рисунками, GIF и JPEG.

## <a name="possible-values"></a>Возможные значения

Значение должно быть строкой, указывающей имя пути.

## <a name="applies-to"></a>Применение

[**clip**](clip-element.md)
