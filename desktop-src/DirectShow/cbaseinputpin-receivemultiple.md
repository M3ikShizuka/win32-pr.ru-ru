# <a name="cbaseinputpinreceivemultiple-method"></a>Кбасеинпутпин. Рецеивемултипле, метод

`ReceiveMultiple`Метод получает массив выборок. Этот метод реализует метод [**имеминпутпин:: рецеивемултипле**](/windows/desktop/api/Strmif/nf-strmif-imeminputpin-receivemultiple) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT ReceiveMultiple(
   IMediaSample **pSamples,
   long         nSamples,
   long         *nSamplesProcessed
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*псамплес* 
</dt> <dd>

Адрес массива указателей [**имедиасампле**](/windows/desktop/api/Strmif/nn-strmif-imediasample) , размером *нсамплес*.

</dd> <dt>

*нсамплес* 
</dt> <dd>

Количество выборок для обработки.

</dd> <dt>

*нсамплеспроцессед* 
</dt> <dd>

Указатель на переменную, которая получает количество обработанных выборок.

</dd> </dl>

| Код возврата                                                                                             | Описание                                                |
|---------------------------------------------------------------------------------------------------------|------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                    | Успешно.<br/>                                        |

| <dl> <dt>**\_указатель E**</dt> </dl>               | **Пустой** аргумент указателя.<br/>                      |
| <dl> <dt>**VFW \_ E \_ инвалидмедиатипе**</dt> </dl> | Недопустимый тип носителя.<br/>                             |
| <dl> <dt>**\_ \_ ошибка среды выполнения VFW E \_**</dt> </dl>   | Произошла ошибка времени выполнения.<br/>                      |

## <a name="remarks"></a>Комментарии

Этот метод ведет себя как метод [**кбасеинпутпин:: Receive**](cbaseinputpin-receive.md) , но получает массив выборок. В базовом классе метод циклически перебирает массив и вызывает **Receive** с каждым образцом. Переопределите эту функцию, если фильтр может обрабатывать пакеты образцов более эффективно, чем по одному за раз.
