# <a name="cmediasamplem_pmediatype-member"></a>Элемент Кмедиасампле:: m \_ пмедиатипе

Указатель на тип мультимедиа, если тип был изменен из предыдущего примера в потоке данных. В противном случае эта переменная-член имеет **значение NULL**.

## <a name="syntax"></a>Синтаксис


```C++
AM_MEDIA_TYPE *m_pMediaType;
```
