# <a name="video-mixing-renderer-video-subtypes"></a>Подтипы видео модуля подготовки видео для микширования

Следующие подтипы определены для использования с фильтром формирователя видео (VMR).

## <a name="vmr-7-subtypes"></a>Подтипы VMR-7



| Subtype                              | Описание                                      |
|--------------------------------------|--------------------------------------------------|
| МЕДИАСУБТИПЕ \_ RGB32 \_ D3D \_ DX7 \_ RT    | 32-разрядный целевой объект рендеринга RGB.                        |
| МЕДИАСУБТИПЕ \_ RGB16 \_ D3D \_ DX7 \_ RT    | 16-разрядный целевой объект рендеринга RGB.                        |
| МЕДИАСУБТИПЕ \_ ARGB32 \_ D3D \_ DX7 \_ RT   | Целевой объект прорисовки ARGB 32.                       |
| МЕДИАСУБТИПЕ \_ ARGB4444 \_ D3D \_ DX7 \_ RT | Целевой объект прорисовки ARGB4444. Для графических изображений. |
| МЕДИАСУБТИПЕ \_ ARGB1555 \_ D3D \_ DX7 \_ RT | Целевой объект прорисовки ARGB1555. Для графических изображений. |
