# <a name="cbasewindowgetdefaultrect-method"></a>Кбасевиндов. Жетдефаултрект, метод

`GetDefaultRect`Метод получает размер клиентской области по умолчанию.

## <a name="syntax"></a>Синтаксис


```C++
virtual RECT GetDefaultRect();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает прямоугольник по умолчанию.

## <a name="remarks"></a>Remarks

Когда окно активируется, объект вызывает этот метод, чтобы определить, насколько крупнее он должен сделать клиентскую область окна. В базовом классе этот метод возвращает прямоугольник, высота и ширина которого являются определенными константами ДЕФХЕИГХТ и ДЕФВИДС. Производный класс должен переопределять этот метод. Для модуля обработки видео производный класс обычно возвращает размер собственного видеообраза.
