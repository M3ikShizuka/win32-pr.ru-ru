# <a name="cdynamicoutputpinsynchronousblockoutputpin-method"></a>Кдинамикаутпутпин. Синчронаусблоккаутпутпин, метод

`SynchronousBlockOutputPin`Метод блокирует ПИН-код; не возвращает, пока ПИН-код не будет заблокирован.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SynchronousBlockOutputPin();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** . Возможные значения включают в себя, показанные в следующей таблице.



| Код возврата                                                                                                                    | Описание                                              |
|--------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                                           | Успешно.<br/>                                      |
| <dl> <dt>**\_ \_ ПИН-код VFW E \_ уже \_ заблокирован**</dt> </dl>                   | ПИН-код уже заблокирован в другом потоке.<br/>     |
| <dl> <dt>**\_ \_ ПИН-код VFW E \_ уже \_ заблокирован \_ в \_ этом \_ потоке**</dt> </dl> | ПИН-код уже заблокирован в вызывающем потоке.<br/> |



 

## <a name="remarks"></a>Комментарии

Не вызывайте этот метод из потока потоковой передачи.
