# <a name="mpeg-2-kernel-media-types"></a>Типы носителей ядра MPEG-2

Для нескольких идентификаторов GUID типа носителя MPEG-2 в Windows DDK определены имена, отличные от имен, используемых в DirectShow. В следующей таблице приведены имена DirectShow (определенные в Ксууидс. h) и соответствующие имена в режиме ядра (определенные в Ксмедиа. h).



| Имя в Ксууидс. h              | Имя в Ксмедиа. h                          |
|--------------------------------|--------------------------------------------|
| ФОРМАТ \_ вавеформатекс           | \_вавеформатекс ОПИСАТЕЛЬ ксдатаформат \_      |
| ФОРМАТ \_ MPEG2Video             | \_Видео о спецификаторе ксдатаформат \_ MPEG2 \_      |
| МЕДИАСУБТИПЕ \_ ATSC \_ Si         | \_ПОДТИП ксдатаформат \_ ATSC \_ Si            |
| МЕДИАСУБТИПЕ \_ Dolby \_ AC3       | КСДАТАФОРМАТ \_ подтип \_ AC3 \_ Audio          |
| МЕДИАСУБТИПЕ \_ DVB \_ Si          | \_ПОДТИП ксдатаформат \_ DVB \_ Si             |
| МЕДИАСУБТИПЕ \_ DVD \_ лпкм \_ Audio | КСДАТАФОРМАТ \_ подтип \_ лпкм \_ Audio         |
| МЕДИАСУБТИПЕ \_ MPEG2 \_ Audio     | КСДАТАФОРМАТ \_ подтипа \_ MPEG2 \_ Audio        |
| \_Программа медиасубтипе MPEG2 \_   | \_Программа статического ксдатаформат \_ типа \_ MPEG2 \_ |
| \_Транспорт медиасубтипе MPEG2 \_ | \_Транспорт ксдатаформат типа \_ MPEG2 \_       |
| \_Видео медиасубтипе MPEG2 \_     | \_Видео о ПОДТИПЕ ксдатаформат \_ MPEG2 \_        |
| \_Разделы MEDIATYPE MPEG2 \_     | \_Разделы ксдатаформат типа \_ MPEG2 \_        |
| MEDIATYPE \_ Audio               | \_тип ксдатаформат \_ Audio                  |
| MEDIATYPE \_ MPEG2 \_ PES          | КСДАТАФОРМАТ \_ тип \_ MPEG2 \_ PES             |
| \_Видео MEDIATYPE               | \_видео типа \_ ксдатаформат                  |
