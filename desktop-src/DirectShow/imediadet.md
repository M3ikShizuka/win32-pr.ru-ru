# <a name="imediadet-interface"></a>Интерфейс Имедиадет

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`IMediaDet`Интерфейс получает сведения о файле мультимедиа, например число потоков, тип носителя, длительность и частоту кадров каждого потока. Он также содержит методы для извлечения отдельных кадров из потока видео. Объект [детектора мультимедиа (медиадет)](media-detector--mediadet.md) предоставляет этот интерфейс.

Чтобы получить сведения о файле с помощью этого интерфейса, выполните следующие действия.

1.  Создайте экземпляр объекта Медиадет, вызвав **CoCreateInstance**. Идентификатор класса — CLSID \_ медиадет.
2.  Вызовите [**имедиадет::p UT \_ filename**](imediadet-put-filename.md) , чтобы указать имя исходного файла.
3.  Вызовите метод [**имедиадет:: Get \_ аутпутстреамс**](imediadet-get-outputstreams.md) , чтобы получить число потоков вывода в источнике.
4.  Вызовите [**имедиадет::p UT \_ куррентстреам**](imediadet-put-currentstream.md) , чтобы указать конкретный поток.
5.  Вызовите любой из следующих методов:
    -   [**Имедиадет:: получение \_ частоты кадров**](imediadet-get-framerate.md)
    -   [**Имедиадет:: Get \_ стреамленгс**](imediadet-get-streamlength.md)
    -   [**Имедиадет:: Get \_ стреаммедиатипе**](imediadet-get-streammediatype.md)
    -   [**Имедиадет:: Get \_ стреамтипе**](imediadet-get-streamtype.md)

Чтобы получить кадр видео, вызовите метод [**имедиадет:: жетбитмапбитс**](imediadet-getbitmapbits.md) или [**Имедиадет:: вритебитмапбитс**](imediadet-writebitmapbits.md). Возвращаемый кадр всегда находится в 24-разрядном формате RGB.

> [!Note]  
> Не используйте один и тот же объект Медиадет с несколькими файлами. Чтобы получить информацию или видеокадры из нескольких файлов, используйте отдельные экземпляры Медиадет.

 

Интерфейс **имедиадет** не поддерживает форматы [**VIDEOINFOHEADER2**](/previous-versions/windows/desktop/api/dvdmedia/ns-dvdmedia-videoinfoheader2) , поэтому этот интерфейс нельзя использовать для получения полей с чередованием или сведений о чересстрочную развертке. Кроме того, если вышестоящий декодер поддерживает только **VIDEOINFOHEADER2**, нельзя использовать `IMediaDet` . Это может быть, например, с помощью декодера MPEG-2. Кроме того, `IMediaDet` интерфейс игнорирует любые потоки в файле, которые не являются видео или аудио. Например, если файл содержит аудио-поток, поток данных и видеопоток, метод **Get \_ аутпутстреамс** будет сообщать только о двух потоках (аудио и видео).
