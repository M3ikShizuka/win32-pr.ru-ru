# <a name="cbasefilterfindpin-method"></a>Кбасефилтер. Финдпин, метод

`FindPin`Метод извлекает ПИН-код с указанным идентификатором. Этот метод реализует метод [**ибасефилтер:: финдпин**](/windows/desktop/api/Strmif/nf-strmif-ibasefilter-findpin) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT FindPin(
   LPCWSTR Id,
   IPin    **ppPin
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Id* 
</dt> <dd>

Указатель на константу, заканчивающуюся нулем строкой Юникода, которая идентифицирует ПИН-код.

</dd> <dt>

*пппин* 
</dt> <dd>

Адрес переменной, которая получает указатель на интерфейс [**Ипин**](/windows/desktop/api/Strmif/nn-strmif-ipin) ПИН-кода.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из следующих значений **HRESULT** .



| Код возврата                                                                                       | Описание                               |
|---------------------------------------------------------------------------------------------------|-------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>              | Успешно.<br/>                       |
| <dl> <dt>**\_указатель E**</dt> </dl>         | **Пустой** аргумент указателя.<br/>     |
| <dl> <dt>**VFW \_ E \_ не \_ найден**</dt> </dl> | Не удалось найти соответствующий ПИН-код.<br/> |



 

## <a name="remarks"></a>Remarks

Этот метод вызывает метод [**кбасепин:: Name**](cbasepin-name.md) для сравнения имени каждого контакта со строкой, заданной параметром *ID* .

Если метод выполнен успешно, то интерфейс **Ипин** имеет необработанный счетчик ссылок. Не забудьте освободить его по завершении.
