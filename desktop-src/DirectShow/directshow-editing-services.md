# <a name="directshow-editing-services"></a>DirectShow Изменение служб

\[Этот API не поддерживается и может быть изменен или недоступен в будущем.\]

DirectShow Службы редактирования (DES) — это прикладной программный интерфейс (API), который значительно упрощает задачи, связанные с редактированием видео. алгоритм DES основан на архитектуре ядра DirectShow. он абстрагирует большую часть сложности DirectShow и предоставляет набор интерфейсов, предназначенных специально для управления проектами редактирования видео. как разработчик приложения вы получаете преимущества DirectShow внутри платформы, гораздо более подходящей для создания приложений для редактирования видео.
