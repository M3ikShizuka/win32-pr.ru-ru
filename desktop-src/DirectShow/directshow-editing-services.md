# <a name="directshow-editing-services"></a>Службы редактирования DirectShow

\[Этот API не поддерживается и может быть изменен или недоступен в будущем.\]

Службы редактирования данных DirectShow (DES) — это прикладной программный интерфейс (API), который значительно упрощает задачи, связанные с редактированием видео. Алгоритм DES построен на основе основной архитектуры DirectShow. Он абстрагирует большую часть сложности DirectShow и предоставляет набор интерфейсов, предназначенных специально для управления проектами редактирования видео. Как разработчик приложения, вы получаете преимущества DirectShow внутри платформы, что гораздо лучше подходит для создания приложений для редактирования видео.
