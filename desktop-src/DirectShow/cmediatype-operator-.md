# <a name="cmediatypecmediatypeoperator-method-mtypeh---cmtype-parameter"></a>Кмедиатипе. Кмедиатипе:: operator = метод (Мтипе. h) — параметр кмтипе

Этот оператор перегружает оператор присваивания для копирования типа мультимедиа.

## <a name="syntax"></a>Синтаксис


```C++
CMediaType& CMediaType::operator=(
  [ref] const CMediaType &cmtype
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*кмтипе* \[ ЗНАЧ\]
</dt> <dd>

Ссылка на объект **кмедиатипе** .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ссылку на объект.
