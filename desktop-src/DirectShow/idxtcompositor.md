# <a name="idxtcompositor-interface"></a>Интерфейс Идксткомпоситор

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`IDxtCompositor`Интерфейс задает свойства для перехода [компоновщика](compositor-transition.md) .

этот интерфейс используется внутренне DirectShow служб редактирования (DES) при визуализации перехода компоновщика. Приложениям DES не требуется использовать этот интерфейс. Чтобы задать свойства для перехода в DES, используйте интерфейс [**ипропертисеттер**](ipropertysetter.md) .

Переход с помощью компоновщика комбинирует изображение переднего плана на фоновое изображение. *Исходный прямоугольник* определяет раздел составного изображения. *Прямоугольник назначения* определяет раздел фонового изображения, получающего изображение переднего плана. Эти прямоугольники показаны на следующей схеме.

![Свойства перехода для компоновщика](images/compmeasure.png)
