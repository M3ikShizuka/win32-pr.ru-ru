# <a name="capture-method"></a>Метод Capture

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`Capture`Метод захватывает изображение по-прежнему из кадра видео, когда объект мсвебдвд находится в режиме без окон.

``` syntax
MSWebDVD.Capture()
```

## <a name="remarks"></a>Remarks

Этот метод захватывает текущий кадр из изображения DVD-Video и вставляет его в окно, из которого пользователь может сохранить или изменить изображение. Чтобы этот метод был выполнен, объект Мсвебдвд должен находиться в режиме без окон.
