# <a name="analog-television"></a>Аналоговые телевизоры

Аналоговый телевизор отличается от других сценариев записи видео несколькими способами:

-   Плата тюнера подстраивается к аналоговому сигналу, который затем поддается цифре.
-   Звук переносится в аналоговый сигнал. То, как звук достигает звуковой карты, зависит от оборудования.
-   Сигнал может содержать дополнительные данные в вертикальном интервале (ВБИ), например скрытые субтитры (CC), Стандартный телетекст (WST) и расширенные службы данных (XDS).

На следующей диаграмме показан типичный граф фильтра для просмотра телепередач.

![Граф аналогового телевидения](images/vidcap06.png)

-   Фильтр [ТВ-тюнера](tv-tuner-filter.md) управляет настройкой.
-   Фильтр [телевизионных аудио](tv-audio-filter.md) управляет обкодированием звука.
-   Если в карточке тюнера имеется более одного физического ввода, то фильтр [аналогового микшера видео](analog-video-crossbar-filter.md) позволяет приложению выбрать, какие входные данные декодированы и готовы к просмотру.
-   Фильтр [записи видео WDM](wdm-video-capture-filter.md) предоставляет оцифрованный поток видео.

построитель Graph отслеживания автоматически вставляет все фильтры, необходимые для вышестоящего потока, из фильтра записи.
