# <a name="pointer-validation-macros"></a>Макросы проверки указателя

Microsoft DirectShow предоставляет несколько макросов для проверки указателей.



| Макрос                                                | Описание                                                                   |
|------------------------------------------------------|-------------------------------------------------------------------------------|
| [**Контрольная точка**](checkpointer.md)                 | Проверяет, имеет ли указатель **значение NULL**.                                         |
| [**валидатереадптр**](validatereadptr.md)           | Проверяет, имеет ли вызывающий процесс доступ на чтение к блоку памяти.          |
| [**валидатереадвритептр**](validatereadwriteptr.md) | Проверяет, что вызывающий процесс имеет доступ на чтение и запись к блоку памяти.    |
| [**валидатестрингптр**](validatestringptr.md)       | Проверяет, имеет ли вызывающий процесс доступ на чтение к строке.                |
| [**валидатестрингптра**](validatestringptra.md)     | Проверяет, имеет ли вызывающий процесс доступ на чтение к строке ANSI.          |
| [**валидатестрингптрв**](validatestringptrw.md)     | Проверяет, имеет ли вызывающий процесс доступ на чтение к строке расширенных символов. |
| [**валидатевритептр**](validatewriteptr.md)         | Проверяет, имеет ли вызывающий процесс доступ на запись к блоку памяти.         |
