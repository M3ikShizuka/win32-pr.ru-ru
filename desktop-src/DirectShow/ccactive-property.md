# <a name="ccactive-property"></a>Ккактиве, свойство

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`CCActive`Свойство задает или получает текущее состояние скрытых субтитров.

``` syntax
[ bCCActive = ] MSWebDVD.CCActive
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает логическое значение, указывающее, включен ли режим субтитров.

## <a name="remarks"></a>Remarks

Это свойство доступно для чтения и записи со значением по умолчанию false.
