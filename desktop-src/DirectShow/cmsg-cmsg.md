# <a name="cmsgcmsg-constructor"></a>КМСГ. КМСГ, конструктор

Конструирует объект [**КМСГ**](cmsg.md) .

## <a name="syntax"></a>Синтаксис


```C++
CMsg(
   UINT     u,
   DWORD    dw,
   LPVOID   lp,
   CAMEvent *pEvent
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*u* 
</dt> <dd>

Код запроса, определяемый клиентом класса потока и понятный переопределенной функцией рабочего потока.

</dd> <dt>

*dw* 
</dt> <dd>

Параметр Flag для кода запроса.

</dd> <dt>

*LP* 
</dt> <dd>

Указатель на данные, необходимые рабочему потоку в качестве параметра или возвращаемых значений. Эти данные не должны основываться на стеке, так как на них будет ссылка в течение некоторого времени после завершения операции очереди.

</dd> <dt>

*певент* 
</dt> <dd>

Указатель на объект события, который может сообщить рабочему потоку для указания на завершение операции.

</dd> </dl>

## <a name="remarks"></a>Комментарии

Эта функция-член содержит запрос на выполнение рабочего потока [**кмсгсреад**](cmsgthread.md) . Все параметры передаются в функцию рабочего потока в качестве параметров при обработке этого сообщения. Значения параметров определяются клиентской функцией, которая вызывает рабочий поток и производный класс, который предоставляет функцию выполнения рабочего потока.
