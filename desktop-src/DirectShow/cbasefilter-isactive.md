# <a name="cbasefilterisactive-method"></a>Кбасефилтер. on, метод

`IsActive`Метод определяет, активен ли фильтр (выполняется или приостановлен).

## <a name="syntax"></a>Синтаксис


```C++
BOOL IsActive();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если фильтр приостановлен или выполняется, или **значение false** , если он остановлен.
