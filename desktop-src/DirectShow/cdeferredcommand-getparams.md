# <a name="cdeferredcommandgetparams-method"></a>Кдеферредкомманд. params, метод

`GetParams`Метод извлекает список аргументов **DISPPARAMS** в метод.

## <a name="syntax"></a>Синтаксис


```C++
DISPPARAMS* GetParams();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает список аргументов **DISPPARAMS** .
