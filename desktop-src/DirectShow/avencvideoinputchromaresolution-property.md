# <a name="avencvideoinputchromaresolution-property"></a>Авенквидеоинпутчромаресолутион, свойство

Задает разрешение чрома для входного видео.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеоинпутчромаресолутион**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенквидеочромаресолутион**](/windows/win32/api/codecapi/ne-codecapi-eavencvideochromaresolution) .
