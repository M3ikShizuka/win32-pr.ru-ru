# <a name="avencmpaenableredundancyprotection-property"></a>Авенкмпаенаблередунданципротектион, свойство

Указывает, следует ли добавить в заголовок кадра циклическую проверку избыточности (CRC). Это свойство применяется к кодировщикам аудио в формате MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпаенаблередунданципротектион**

## <a name="property-value"></a>Значение свойства

Это свойство может иметь следующие значения.



| Значение          | Описание                |
|----------------|----------------------------|
| ВАРИАНТ \_ false | Не добавляйте контрольную сумму CRC. |
| ВАРИАНТ \_ true  | Добавление контрольной суммы CRC.        |
