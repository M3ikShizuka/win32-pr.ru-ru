# <a name="system-device-enumerator"></a>Перечислитель системных устройств

Перечислитель системных устройств перечисляет фильтры и аппаратные устройства, установленные в системе. Приложения могут использовать этот компонент для нахождение фильтров и устройств в определенной категории. Создайте этот объект, вызвав **CoCreateInstance**.



| Метка | Значение |
|------------------|------------------------------------------|
| Идентификатор класса | \_СИСТЕМДЕВИЦЕЕНУМ CLSID                  |
| Интерфейсы       | [**икреатедевенум**](/windows/desktop/api/Strmif/nn-strmif-icreatedevenum) |
