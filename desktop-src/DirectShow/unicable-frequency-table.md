# <a name="unicable-frequency-table"></a>Таблица частоты уникабле

Из-за его длины таблица Уникабле не включается здесь. Цель таблицы Уникабле заключается в том, чтобы включить в одну таблицу комплексный набор частот (по сути, для всех остальных таблиц, а также некоторые реальные наблюдения), чтобы упростить автоматическое сканирование каналов, не прибегая к линейному поиску всего спектра частот. Для тех частот, которые не включены в эту таблицу, потребителю потребуется ввести частоту через пользовательский интерфейс, что приведет к записи переопределения частоты в реестре.
