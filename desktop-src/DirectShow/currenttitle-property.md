# <a name="currenttitle-property"></a>Курренттитле, свойство

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`CurrentTitle`Свойство получает число воспроизводимых в данный момент названий.

``` syntax
[ iCurTitle = ] MSWebDVD.CurrentTitle
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение от 1 до 99, представляющее текущее название диска.

## <a name="remarks"></a>Remarks

Это свойство доступно только для чтения и не имеет значения по умолчанию.
