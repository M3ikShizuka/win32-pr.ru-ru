# <a name="disablescreensaver-property"></a>Дисаблескринсавер, свойство

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`DVDAdm.DisableScreenSaver`Свойство включает или выключает экранную заставку.

``` syntax
[ bDisabled = ] DVD.DVDAdm.DisableScreenSaver
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает логическое значение, указывающее, отключены ли параметры экранной заставки для приложения проигрывателя DVD. значение true означает, что параметры отключены.

## <a name="remarks"></a>Комментарии

Это свойство доступно для чтения и записи со значением по умолчанию true. При просмотре DVD-Videoного диска пользователь обычно не использует мышь или клавиатуру в течение продолжительного периода времени. Таким образом, элемент управления ActiveX® Мсвебдвд по умолчанию отключает экранную заставку. Объект
