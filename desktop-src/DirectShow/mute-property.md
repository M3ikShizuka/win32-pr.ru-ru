# <a name="mute-property"></a>Выключить свойство

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`Mute`Свойство включает или выключает выходные данные звукового потока.

``` syntax
[ bMute = ] MSWebDVD.Mute
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает логическое значение, указывающее, следует ли отключить звуковый поток.

## <a name="remarks"></a>Комментарии

Это свойство доступно для чтения и записи со значением по умолчанию false.
