# <a name="enableresetonstop-property"></a>Енаблересетонстоп, свойство

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`EnableResetOnStop`Свойство задает или получает значение, определяющее, как возобновляется воспроизведение, когда граф фильтра выполняет переход из остановленного состояния.

``` syntax
[ bEnableReset = ] MSWebDVD.EnableResetOnStop
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает логическое значение, указывающее, где будет начато воспроизведение объекта Мсвебдвд после остановки графа фильтра.

## <a name="remarks"></a>Комментарии

Это свойство доступно для чтения и записи.

Возможные значения этого свойства: со значением по умолчанию true.



| Значение | Описание                                                                                       |
|-------|---------------------------------------------------------------------------------------------------|
| true  | DVD-навигатор будет сброшен и начнет играть с начала диска. Это значение по умолчанию |
| false | Воспроизведение DVD-навигатора продолжится с того места, на котором она была прервана.                                                 |
