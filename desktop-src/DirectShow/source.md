# <a name="source"></a>Источник

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Исходный объект представляет исходный ролик на временной шкале. Чтобы создать этот объект, вызовите метод [**иамтимелине:: креатимптиноде**](iamtimeline-createemptynode.md) .

Исходный объект предоставляет следующие интерфейсы:

-   [**иамтимелиниффектабле**](iamtimelineeffectable.md)
-   [**иамтимелинеобж**](iamtimelineobj.md)
-   [**иамтимелинесрк**](iamtimelinesrc.md)
-   [**иамтимелинесплиттабле**](iamtimelinesplittable.md)
