# <a name="iamtimelinecomp-interface"></a>Интерфейс Иамтимелинекомп

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Интерфейс **иамтимелинекомп** вставляет или извлекает Виртуальные дорожки в композиции в [службах редактирования DirectShow](directshow-editing-services.md) (DES).

Композиция — это коллекция слоев, которая выступает в качестве отдельной составной *записи*. Например, композиция, которая содержит две дорожки с переходом между ними, действует как единая дорожка с предкомпозициым переходом. Композиция должна содержать носители только одного типа (например, аудио или видео), но это ограничение не применяется. *Виртуальная дорожка* — это любой объект, который может находиться в композиции, включая записи и другие композиции.

Самые верхние узлы на временной шкале являются *группами*. Группы реализуют `IAMTimelineComp` интерфейс и интерфейс [**иамтимелинеграуп**](iamtimelinegroup.md) .

Чтобы создать объект композиции, вызовите [**иамтимелине:: креатимптиноде**](iamtimeline-createemptynode.md) со значением, \_ составляющим основной тип временной шкалы \_ \_ . Вы можете запросить возвращенный указатель [**иамтимелинеобж**](iamtimelineobj.md) для `IAMTimelineComp` интерфейса. Дополнительные сведения см. [в разделе Модель временной шкалы](the-timeline-model.md) и [Построение временной шкалы](constructing-a-timeline.md).

### <a name="methods"></a>Методы

Интерфейс **иамтимелинекомп** содержит следующие методы.



| Метод                                                                       | Описание                                                                                                                                               |
|:-----------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**жеткаунтофтипе**](iamtimelinecomp-getcountoftype.md)                     | Извлекает количество объектов заданного типа, содержащихся в этой композиции, и всех ее виртуальных дорожек рекурсивно.<br/>                      |
| [**жетнекствтракк**](iamtimelinecomp-getnextvtrack.md)                       | Извлекает следующую виртуальную запись после указанной виртуальной записи.<br/>                                                                              |
| [**жетрекурсивелайерофтипе**](iamtimelinecomp-getrecursivelayeroftype.md)   | Выполняет упорядочение по глубине виртуальных дорожек, содержащихся в этой композиции, и извлекает из этого упорядочения *n*-й виртуальный объект Track.<br/> |
| [**жетрекурсивелайерофтипеи**](iamtimelinecomp-getrecursivelayeroftypei.md) | Не поддерживается.<br/>                                                                                                                                 |
| [**жетвтракк**](iamtimelinecomp-getvtrack.md)                               | Извлекает виртуальную дорожку по указанному приоритету.<br/>                                                                                         |
| [**втраккжеткаунт**](iamtimelinecomp-vtrackgetcount.md)                     | Возвращает число виртуальных дорожек, содержащихся в композиции.<br/>                                                                           |

| [**втракксвапприоритиес**](iamtimelinecomp-vtrackswappriorities.md)         | Переключает уровни приоритета двух дорожек.<br/>                                                                                                    |
