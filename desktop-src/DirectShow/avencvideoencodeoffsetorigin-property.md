# <a name="avencvideoencodeoffsetorigin-property"></a>Авенквидеоенкодеоффсеторигин, свойство

Задает левый и верхний углы прямоугольника обрезки, если видео обрезано.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеоенкодеоффсеторигин**

## <a name="property-value"></a>Значение свойства

Верхние 16 разрядов значения содержат смещение от левого края входного кадра в пикселях. Младшие 16 бит содержат смещение от верхнего края входного кадра (в пикселях).

## <a name="remarks"></a>Remarks

Приложения могут задать это свойство, чтобы обрезать входное видео. Используйте свойство [**авенквидеоенкодедименсион**](avencvideoencodedimension-property.md) , чтобы указать ширину и высоту прямоугольника обрезки.
