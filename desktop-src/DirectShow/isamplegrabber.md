# <a name="isamplegrabber-interface"></a>Интерфейс Исамплеграббер

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Интерфейс **исамплеграббер** предоставляется [**образцом фильтра захвата**](sample-grabber-filter.md) . Он позволяет приложению извлекать отдельные примеры мультимедиа по мере их перемещения через граф фильтра.

## <a name="members"></a>Элементы

Интерфейс **исамплеграббер** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Исамплеграббер** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **исамплеграббер** содержит следующие методы.



| Метод                                                                | Описание                                                                                                                       |
|:----------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------|
| [**жетконнектедмедиатипе**](isamplegrabber-getconnectedmediatype.md) | Извлекает тип носителя для соединения со входным закреплением образца захвата.<br/>                                        |
| [**жеткуррентбуффер**](isamplegrabber-getcurrentbuffer.md)           | Извлекает копию образца, который фильтр получил последним.<br/>                                                 |
| [**жеткуррентсампле**](isamplegrabber-getcurrentsample.md)           | Не реализован.<br/>                                                                                                       |
| [**сетбуфферсамплес**](isamplegrabber-setbuffersamples.md)           | Указывает, следует ли копировать демонстрационные данные в буфер, так как он проходит через фильтр.<br/>                                     |
| [**SetCallback**](isamplegrabber-setcallback.md)                     | Указывает метод обратного вызова для вызова в входящих примерах.<br/>                                                               |
| [**сетмедиатипе**](isamplegrabber-setmediatype.md)                   | Указывает тип носителя для соединения во входном закрепление образца области захвата.<br/>                                    |
| [**сетонешот**](isamplegrabber-setoneshot.md)                       | Указывает, останавливается ли фильтр [**захвата выборки**](sample-grabber-filter.md) после того, как фильтр получит образец.<br/> |
