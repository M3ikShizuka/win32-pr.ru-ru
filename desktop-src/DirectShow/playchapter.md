# <a name="playchapter"></a>плайчаптер

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`PlayChapter`Событие отправляется при `PlayChapter` включении или отключении команды.

``` syntax
PlayChapter(bEnabled)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="bEnabled"></span><span id="benabled"></span><span id="BENABLED"></span>*бенаблед*
</dt> <dd>

Указывает, включена или отключена операция в качестве логического значения.

</dd> </dl>
