# <a name="cbasecontrolwindowput_visible-method"></a>Кбасеконтролвиндов. размещение \_ видимого метода

`put_Visible`Метод делает отображение или скрытие окна.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT put_Visible(
   long Visible
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Visible* 
</dt> <dd>

Логический флаг автоматизации (0 означает, что окно скрыто, 1 означает, что окно отображается).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** .
