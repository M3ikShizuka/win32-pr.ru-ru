# <a name="coutputqueuem_bsendanyway-member"></a>Элемент Каутпуткуеуе:: m \_ бсенданивай

Флаг для переопределения пакетной обработки. Установка этого флага в **значение true** переопределяет флаг [**каутпуткуеуе:: m \_ ббатчексакт**](coutputqueue-m-bbatchexact.md) и доставляет все ожидающие выборки.

## <a name="syntax"></a>Синтаксис


```C++
BOOL m_bSendAnyway;
```
