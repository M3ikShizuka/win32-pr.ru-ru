# <a name="avencmpvscanpattern-property"></a>Авенкмпвсканпаттерн, свойство

Указывает шаблон сканирования макроблокк. Это свойство применяется к кодировщикам видео MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпвсканпаттерн**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенкмпвсканпаттерн**](/windows/win32/api/codecapi/ne-codecapi-eavencmpvscanpattern) .
