# <a name="cenummediatypes-class"></a>Класс Ценуммедиатипес

![Иерархия классов ценуммедиатипес](images/filter04.png)

`CEnumMediaTypes`Класс реализует перечислитель для предпочтительных типов мультимедиа.

Этот класс реализует интерфейс [**иенуммедиатипес**](/windows/desktop/api/Strmif/nn-strmif-ienummediatypes) . Он вызывает следующие методы [**кбасепин**](cbasepin.md) :

-   [**Кбасепин:: жетмедиатипе**](cbasepin-getmediatype.md): Извлекает тип мультимедиа, на который ссылается индекс, начинающийся с нуля.
-   [**Кбасепин:: жетмедиатипеверсион**](cbasepin-getmediatypeversion.md): определяет, изменился ли набор предпочтительных типов.

Каждый раз, когда ПИН-код изменяет свой список предпочтительных типов мультимедиа, этот ПИН-код увеличивает номер версии типа носителя. В этом случае объект перечислителя больше не синхронизируется с закреплением, а методы класса возвращают VFW \_ E \_ Enum \_ из \_ \_ Sync. Вызовите метод [**ценуммедиатипес:: Reset**](cenummediatypes-reset.md) для повторной синхронизации перечислителя.



| Открытые методы                                               | Описание                                                     |

| [**ценуммедиатипес**](cenummediatypes-cenummediatypes.md)   | Метод конструктора.                                             |

| [**Далее**](cenummediatypes-next.md)                         | Извлекает указанное число типов мультимедиа.                    |
| [**Перезапуск**](cenummediatypes-reset.md)                       | Сбрасывает последовательность перечисления в начало.               |
