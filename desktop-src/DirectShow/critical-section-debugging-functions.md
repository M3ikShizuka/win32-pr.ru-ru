# <a name="critical-section-debugging-functions"></a>Функции отладки критического раздела

Эти функции помогают отлаживать критически важные разделы в коде, что может упростить поиск причины взаимоблокировки. Эти функции используют вспомогательный класс [**ккритсек**](ccritsec.md) .



| Функция                             | Описание                                                                  |
|--------------------------------------|------------------------------------------------------------------------------|
| [**критчеккин**](critcheckin.md)   | Возвращает **значение true** , если текущий поток владеет заданной критической секцией.  |
| [**критчеккаут**](critcheckout.md) | Возвращает **значение false** , если текущий поток владеет заданной критической секцией. |
| [**дбглокктраце**](dbglocktrace.md) | Включает или отключает ведение журнала отладки для заданной критической секции.              |
