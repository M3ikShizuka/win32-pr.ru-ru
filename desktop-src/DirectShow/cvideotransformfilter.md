# <a name="cvideotransformfilter-class"></a>Класс Квидеотрансформфилтер

![Иерархия классов квидеотрансформфилтер](images/vtsip01.png)

`CVideoTransformFilter`Класс разработан в основном как базовый класс для фильтров распаковки AVI. Этот класс добавляет поддержку контроля качества к классу [**ктрансформфилтер**](ctransformfilter.md) . Метод **Receive** фильтра может принимать решение о перетаскивании кадров на основе сообщений об качествах от модуля подготовки отчетов и измерений производительности, собираемых фильтром во время потоковой передачи.

Если фильтр удаляет кадр, кадры по-своему удаляются, пока не достигнет следующего ключевого кадра. Для потоков MPEG фильтр не различает фреймы B и кадры P.
