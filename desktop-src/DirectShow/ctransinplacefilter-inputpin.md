# <a name="ctransinplacefilterinputpin-method"></a>Ктрансинплацефилтер. InputPin, метод

`InputPin`Метод получает указатель на входной ПИН-код фильтра.

## <a name="syntax"></a>Синтаксис


```C++
CTransInPlaceInputPin* InputPin();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает переменную члена [**ктрансформфилтер:: m \_ пинпут**](ctransformfilter-m-pinput.md) .
