# <a name="cpullpinseek-method"></a>Кпуллпин. Seek, метод

`Seek`Метод задает начальные и останавливаемые позиции потока.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Seek(
   REFERENCE_TIME tStart,
   REFERENCE_TIME tStop
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*тстарт* 
</dt> <dd>

Указывает начальную точку в байтах, умноженную на 10 000 000.

</dd> <dt>

*тстоп* 
</dt> <dd>

Указывает позиция окончания в байтах, умноженная на 10 000 000.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение \_ ОК, если метод завершается успешно, или код ошибки в противном случае.

## <a name="remarks"></a>Remarks

Если рабочий поток запущен, метод приостанавливает поток, очищает граф фильтра и возобновляет поток. Поток начинает извлечение данных из новой начальной должности. В противном случае новые значения позиций вступают в силу при каждом запуске потока.

Позиции задаются относительно начала исходного источника. Умножьте требуемые смещения в байтах на константные единицы, которые определены в библиотеке базовых классов как 10 000 000.

Когда ПИН-код сначала соединяется, начальная и начальная позиции по умолчанию задаются в начале и в конце потока.
