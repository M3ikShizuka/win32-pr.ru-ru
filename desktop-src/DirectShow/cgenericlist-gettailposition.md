# <a name="cgenericlistgettailposition-method"></a>Кженериклист. Жеттаилпоситион, метод

`GetTailPosition`Метод получает позицию последнего элемента списка.

## <a name="syntax"></a>Синтаксис


```C++
POSITION GetTailPosition();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение расположения или **значение NULL** , если список пуст.
