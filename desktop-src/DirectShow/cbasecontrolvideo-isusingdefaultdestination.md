# <a name="cbasecontrolvideoisusingdefaultdestination-method"></a>Кбасеконтролвидео. Исусингдефаултдестинатион, метод

`IsUsingDefaultDestination`Метод определяет, использует ли модуль подготовки отчетов окно назначения по умолчанию.

## <a name="syntax"></a>Синтаксис


```C++
virtual HRESULT IsUsingDefaultDestination() = 0;
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

\_При использовании назначения по умолчанию возвращает значение s ОК; в противном случае возвращает \_ false.
