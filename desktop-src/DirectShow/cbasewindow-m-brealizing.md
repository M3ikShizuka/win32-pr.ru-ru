# <a name="cbasewindowm_brealizing-member"></a>Элемент Кбасевиндов:: m \_ бреализинг

Флаг, указывающий, реализована ли новая палитра. В отладочных сборках значение равно **true** , когда палитра изменяется, и **false** в противном случае. Эта переменная не используется в розничных сборках.

## <a name="syntax"></a>Синтаксис


```C++
BYTE m_bRealizing;
```
