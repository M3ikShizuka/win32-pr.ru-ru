# <a name="enumerating-devices-and-filters"></a>Перечисление устройств и фильтров

Иногда приложению необходимо выбрать определенный фильтр в системе пользователя. Например, приложение для записи видео может отображать список доступных устройств записи. Так как DirectShow использует архитектуру на основе компонентов, вы не сможете во время разработки использовать фильтры, установленные в системе пользователя. Это особенно справедливо для фильтров, представляющих аппаратные устройства. DirectShow предоставляет два компонента, которые находят зарегистрированные фильтры:

-   [Перечислитель системных устройств](system-device-enumerator.md) находит фильтры по их категориям.
-   Средство [сопоставления фильтров](filter-mapper.md) находит фильтры в соответствии с критериями поиска, предоставленными приложением.

Перечислители, обсуждаемые в этом разделе, следуют стандартной форме, используемой интерфейсами перечисления COM. Дополнительные сведения см. в разделе «Иенумкскскскс» пакета средств разработки программного обеспечения (SDK) платформы Майкрософт.

В этом разделе рассматриваются следующие вопросы.

-   [Использование перечислителя системных устройств](using-the-system-device-enumerator.md)
-   [Использование средства сопоставления фильтров](using-the-filter-mapper.md)
