# <a name="installing-the-windows-region-change-support-components"></a>установка компонентов поддержки изменения региона Windows

разработчики приложений могут использовать поддержку смены региона dvd, встроенной в Windows, чтобы изменить регион в приложениях воспроизведения DVD.

в Windows 2000 и более поздних версиях компонент, предоставляющий поддержку изменения региона DVD, находится в storprop.dll DLL (доступно в папке% windir% \\ system32). Эта библиотека DLL устанавливается по умолчанию и не требует дополнительных действий для обеспечения доступности этой библиотеки DLL.
