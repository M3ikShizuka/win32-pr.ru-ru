# <a name="avencvideoinversetelecineenable-property"></a>АвенквидеоинверсетелеЦининабле, свойство

Указывает, выполняет ли кодировщик обратную чересстрочности.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеоинверсетелеЦининабле**

## <a name="remarks"></a>Remarks

Если значение является **вариантным \_ true**, кодировщик выполняет обратную чересстрочности. Если обратный чересстрочности не требуется, задайте для этого свойства значение **Variant \_ false**. Чтобы выполнить инверсный чересстрочности за пределами кодировщика, присвойте этому свойству значение **Variant \_ false** и установите для свойства [**авенквидеуутпутскантипе**](avencvideooutputscantype-property.md) значение еавенквидеуутпутскан \_ самеасинпут.
