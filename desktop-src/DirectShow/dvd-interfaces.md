# <a name="dvd-interfaces"></a>Интерфейсы DVD

Эти интерфейсы поддерживают воспроизведение DVD-дисков и навигацию.



| Интерфейс                                    | Описание                                                                                            |
|----------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**иамдекодеркапс**](/windows/desktop/api/Strmif/nn-strmif-iamdecodercaps)     | Реализованы декодерами для указания различных возможностей.                                              |
| [**идвдкмд**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | Блокировать [Фильтр навигации по DVD](dvd-navigator-filter.md) , пока команда не будет запущена или завершена.        |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | Поиск и воспроизведение DVD-Video названий, включая заголовки караоке.                                          |
| [**идвдграфбуилдер**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | Создание графа фильтра для DVD-Video воспроизведения из доступного программного обеспечения и оборудования в системе.    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | Запрос различных атрибутов DVD-диска или DVD-навигатора.                               |
| [**идвдстате**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | Сохраните состояние сеанса пользователя на диск, включая расположение воспроизведения, родительский уровень и регион. |
