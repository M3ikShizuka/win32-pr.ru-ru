# <a name="media-locator"></a>Указатель мультимедиа

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Объект локатора носителей находит файлы, связанные с исходными объектами. Используйте этот объект для проверки источников на временной шкале. Чтобы создать этот объект, вызовите **CoCreateInstance**. Идентификатор класса — CLSID \_ медиалокатор.

Объект локатора мультимедиа предоставляет следующий интерфейс:

-   [**имедиалокатор**](imedialocator.md)
