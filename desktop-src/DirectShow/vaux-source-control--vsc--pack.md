## <a name="remarks"></a>Комментарии

Ниже перечислены коды полей, которые представляют интерес:

-   **CGMS**: система управления поколением копий. 0 = копирование разрешено без ограничений.

    Фактические пакеты VSC в DV-потоке могут содержать различные значения.

<!-- -->

-   **Режим "REC**": режим записи. 1 = исходный.
-   Отображение **: режим выбора дисплея.** 000 = 4:3 пропорции, полный формат; 010 = 16:9 пропорции.
-   **Бксис**: Широковещательная система. Это поле определяет тип сведений о широких экранных сигналах.
    -   0 = тип 0 (см. IEC 61880)
    -   1 = тип 1 (см. ЕТСИ EN 300 294)
