# <a name="dmo-interfaces"></a>Интерфейсы DMO

В этом разделе содержатся справочные сведения обо всех COM-интерфейсах объектов мультимедиа Microsoft DirectX (DMO) и их методах.



| Интерфейс                                                            | Описание                                                              |
|----------------------------------------------------------------------|--------------------------------------------------------------------------|
| [**идмокуалитиконтрол**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-idmoqualitycontrol)                     | Поддерживает контроль качества для DMO.                                       |
| [**идмовидеуутпутоптимизатионс**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-idmovideooutputoptimizations) | Поддерживает оптимизацию видео на DMO.                                   |
| [**иенумдмо**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-ienumdmo)                                         | Предоставляет методы для перечисления дмос.                                   |
| [**имедиабуффер**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-imediabuffer)                                 | Предоставляет методы для управления буфером данных.                         |
| [**имедиаобжект**](/previous-versions/windows/desktop/api/Mediaobj/nn-mediaobj-imediaobject)                                 | Предоставляет методы для управления DMO.                                 |
| [**имедиаобжектинплаце**](/previous-versions/windows/desktop/api/mediaobj/nn-mediaobj-imediaobjectinplace)                   | Предоставляет методы для обработки данных на месте.                           |
| [**имедиапараминфо**](/previous-versions/windows/desktop/api/Medparam/nn-medparam-imediaparaminfo)                           | Извлекает сведения о параметрах мультимедиа, поддерживаемых объектом. |
| [**имедиапарамс**](/previous-versions/windows/desktop/api/Medparam/nn-medparam-imediaparams)                                 | Задает и получает параметры мультимедиа для объекта.                        |
