# <a name="time-format-guids"></a>Идентификаторы GUID формата времени

Следующие глобальные уникальные идентификаторы (GUID) определяют различные форматы времени.



| Идентификатор GUID                                                                                                                                                                                       | Описание                                       |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------|
| <span id="TIME_FORMAT_NONE"></span><span id="time_format_none"></span><dl> <dt>**формат времени — \_ \_ нет**</dt> </dl>                    | Нет формата.<br/>                             |
| <span id="TIME_FORMAT_FRAME"></span><span id="time_format_frame"></span><dl> <dt>**\_кадр формата \_ времени**</dt> </dl>                 | Видеокадры.<br/>                          |
| <span id="TIME_FORMAT_SAMPLE"></span><span id="time_format_sample"></span><dl> <dt>**\_пример формата \_ времени**</dt> </dl>              | Примеры в потоке.<br/>                 |
| <span id="TIME_FORMAT_FIELD"></span><span id="time_format_field"></span><dl> <dt>**\_поле формата \_ времени**</dt> </dl>                 | Чередующиеся поля видео.<br/>               |
| <span id="TIME_FORMAT_BYTE"></span><span id="time_format_byte"></span><dl> <dt>**\_байт формата \_ времени**</dt> </dl>                    | Смещение в пределах потока в байтах.<br/>         |
| <span id="TIME_FORMAT_MEDIA_TIME"></span><span id="time_format_media_time"></span><dl> <dt>**время \_ формата времени в формате \_ мультимедиа \_**</dt> </dl> | Справочное время (100-наносекундных единиц).<br/> |
