# <a name="crendererinputpinbreakconnect-method"></a>Крендереринпутпин. Бреакконнект, метод

Метод **бреакконнект** освобождает ПИН-код из соединения. Этот метод переопределяет метод [**кбасеинпутпин:: бреакконнект**](cbaseinputpin-breakconnect.md) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT BreakConnect();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** .
