# <a name="directshow-data-types"></a>Типы данных DirectShow

DirectShow предоставляет следующие определения типов.



| Тип данных                                 | Описание                                                                                       |
|-------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**регистрация DVD-диска \_**](dvd-register.md)     | Содержит значение для реестра общих параметров DVD-диска (ГПРМ) или реестра системных параметров (СПРМ). |
| [**гпрмаррай**](gprmarray.md)            | Содержит массив значений ГПРМ.                                                                 |
| [**оаевент**](oaevent.md)                | Определяет обработчик событий, совместимый с автоматизацией.                                                    |
| [**оафилтерстате**](oafilterstate.md)    | Определяет состояние графа фильтра.                                                            |
| [**оахвнд**](oahwnd.md)                  | Определяет совместимый с автоматизацией обработчик окна.                                                   |
| [**\_время ссылки**](reference-time.md) | Определяет тип данных для времени ссылки.                                                         |
| [**рефтиме**](reftime.md)                | Определяет тип данных, используемый для выражения времени в секундах.                                            |
| [**спрмаррай**](sprmarray.md)            | Содержит массив значений СПРМ.                                                                 |
