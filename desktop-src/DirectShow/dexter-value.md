# <a name="dexter_value-structure"></a>\_Структура значения Декстер

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Определяет свойство, которое должно быть задано для перехода или воздействия, а также количество различных значений, принимаемых свойством на протяжении перехода или воздействия.

## <a name="syntax"></a>Синтаксис


```C++
typedef struct {
  VARIANT        v;
  REFERENCE_TIME rt;
  DWORD          dwInterp;
} DEXTER_VALUE;
```
