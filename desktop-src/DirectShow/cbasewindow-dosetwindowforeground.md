# <a name="cbasewindowdosetwindowforeground-method"></a>Кбасевиндов. Досетвиндовфореграунд, метод

`DoSetWindowForeground`Метод переводит окно на передний план.

## <a name="syntax"></a>Синтаксис


```C++
void DoSetWindowForeground(
   BOOL bFocus
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*бфокус* 
</dt> <dd>

Логическое значение, указывающее, следует ли передавать фокус окна. Если значение равно **true**, окно получает фокус.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
