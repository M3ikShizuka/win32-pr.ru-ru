# <a name="cpullpinconnect-method"></a>Кпуллпин. Connect, метод

`Connect`Метод завершает соединение с выходным закреплением.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Connect(
   IUnknown      *pUnk,
   IMemAllocator *pAlloc,
   BOOL          bSync
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*pUnk* 
</dt> <dd>

Указатель на интерфейс **IUnknown** выходного ПИН-кода.

</dd> <dt>

*паллок* 
</dt> <dd>

Указатель на интерфейс [**имемаллокатор**](/windows/desktop/api/Strmif/nn-strmif-imemallocator) предпочтительного распределителя входного контакта или **значение NULL**.

</dd> <dt>

*бсинк* 
</dt> <dd>

Логическое значение, указывающее, следует ли использовать синхронные операции чтения. Если **значение — true**, ПИН-код выполняет синхронные операции чтения для выходного ПИН-кода. Если **значение равно false**, то ПИН-код выполняет асинхронные запросы на чтение.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение HRESULT**. Ниже приведены возможные значения.



| Код возврата                                                                                               | Описание                                                                     |
|-----------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                      | Успешно.<br/>                                                             |
| <dl> <dt>**VFW \_ E \_ уже \_ подключен**</dt> </dl> | Входной ПИН-код уже подключен.<br/>                                  |
| <dl> <dt>**\_интерфейс E**</dt> </dl>             | Закрепление вывода не предоставляет [**иасинкреадер**](/windows/desktop/api/Strmif/nn-strmif-iasyncreader).<br/> |



 

## <a name="remarks"></a>Комментарии

Вызывайте этот метод во время подключения входного контакта. Если метод завершается сбоем, ПИН-код не должен подключиться.

Этот метод запрашивает выходной ПИН-код для интерфейса **иасинкреадер** . В случае успеха вызывается [**кпуллпин::D еЦидеаллокатор**](cpullpin-decideallocator.md) для согласования распределителя для подключения. Если входной ПИН-код имеет предпочтительный распределитель, укажите его в параметре *паллок* . метод **деЦидеаллокатор** передает этот указатель в метод [**Иасинкреадер:: рекуесталлокатор**](/windows/desktop/api/Strmif/nf-strmif-iasyncreader-requestallocator) выходного контакта. В противном случае установите для *Паллок* **значение NULL**.

Если значение *бсинк* равно **true**, то объект **кпуллпин** выполняет синхронные запросы чтения путем вызова [**иасинкреадер:: синкреадалигнед**](/windows/desktop/api/Strmif/nf-strmif-iasyncreader-syncreadaligned)выходного контакта. В противном случае вызывается метод [**иасинкреадер:: request**](/windows/desktop/api/Strmif/nf-strmif-iasyncreader-request) для выполнения перекрывающихся запросов на чтение.
