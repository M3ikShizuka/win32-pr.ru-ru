# <a name="cmediacontrolgetidsofnames-method"></a>Кмедиаконтрол. GetIDsOfNames, метод

Карты одну функцию-член и необязательный набор параметров в соответствующий набор идентификаторов диспетчеризации целых чисел (dispid), который может использоваться при последующих вызовах функции-члена [**кмедиаконтрол:: Invoke**](cmediacontrol-invoke.md) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetIDsOfNames(
   REFIID  riid,
   OLECHAR **rgszNames,
   UINT    cNames,
   LCID    lcid,
   DISPID  *rgdispid
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*riid* 
</dt> <dd>

Идентификатор ссылки. Зарезервировано для последующего использования. Должен иметь **значение NULL**.

</dd> <dt>

*ргсзнамес* 
</dt> <dd>

Адрес указателя на переданный массив сопоставляемых имен.

</dd> <dt>

*Записи CNAME* 
</dt> <dd>

Количество сопоставляемых имен.

</dd> <dt>

*lcid* 
</dt> <dd>

Контекст языкового стандарта, в котором следует интерпретировать имена.

</dd> <dt>

*ргдиспид* 
</dt> <dd>

Указатель на выделенный вызывающим объектом массив, каждый элемент которого содержит идентификатор, соответствующий одному из имен, переданных в массиве *ргсзнамес* . Первый элемент представляет имя элемента; последующие элементы представляют каждый из параметров элемента.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из следующих значений.



| Код возврата                                                                                            | Описание                                                                                                                                          |
|--------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**из \_ \_ неизвестного \_ идентификатора CLSID**</dt> </dl> | Идентификатор CLSID не распознан.<br/>                                                                                                             |
| <dl> <dt>**\_выункновннаме E \_**</dt> </dl>    | Одно или несколько имен не были известны. Возвращаемые идентификаторы DISPID содержат \_ неизвестный идентификатор DISPID для каждой записи, соответствующей неизвестному имени.<br/> |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl>          | Недостаточно памяти.<br/>                                                                                                                            |
| <dl> <dt>**\_ОК**</dt> </dl>                   | Успешно.<br/>                                                                                                                                  |
