# <a name="avencmpvqscaletype-property"></a>Авенкмпвкскалетипе, свойство

Указывает, является ли шкала куантизер линейной или не линейной. Это свойство применяется к кодировщикам видео MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпвкскалетипе**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенкмпвкскалетипе**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpvqscaletype) .
