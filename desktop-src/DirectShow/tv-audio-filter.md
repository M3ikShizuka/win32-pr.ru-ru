# <a name="tv-audio-filter"></a>Фильтр ТВ-звука

| Типы носителей входных закрепления                    | MEDIATYPE \_ аналогаудио                                                                                                         |
| Интерфейсы входных закрепления                     | **икспропертисет**                                                                                                             |
| Типы носителей для выходного ПИН-кода                   | MEDIATYPE \_ аналогаудио                                                                                                         |
| Интерфейсы выходного ПИН-кода                    | **икспропертисет**                                                                                                             |
| Фильтровать CLSID                             | \_ТВАУДИОФИЛТЕР CLSID                                                                                                           |
