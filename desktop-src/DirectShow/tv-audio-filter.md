# <a name="tv-audio-filter"></a>Фильтр ТВ-звука

Фильтр ТВ-звука обеспечивает управление телевизионным декодированием, стерео или моноаурал, а также выбором дополнительного аудио программы (SAP).



| Метка | Значение |
|------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| Интерфейсы фильтра                        | [**Иамтваудио**](/windows/desktop/api/Strmif/nn-strmif-iamtvaudio), **IPersistPropertyBag**, **испеЦифипропертипажес**, [**икспропертисет**](ikspropertyset.md) |
| Типы носителей входных закрепления                    | MEDIATYPE \_ аналогаудио                                                                                                         |
| Интерфейсы входных закрепления                     | **икспропертисет**                                                                                                             |
| Типы носителей для выходного ПИН-кода                   | MEDIATYPE \_ аналогаудио                                                                                                         |
| Интерфейсы выходного ПИН-кода                    | **икспропертисет**                                                                                                             |
| Фильтровать CLSID                             | \_ТВАУДИОФИЛТЕР CLSID                                                                                                           |
| CLSID страницы свойств                      | \_кскатегори \_ тваудио                                                                                                        |
| [Заслуживают](merit.md)                       | \_ \_ не \_ использовать                                                                                                            |
