# <a name="getsprm-method"></a>Метод Жетспрм

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`GetSPRM`Метод извлекает указанный регистр системного параметра.

``` syntax
[ iSPRM = ] MSWebDVD.GetSPRM(iIndex)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="iIndex"></span><span id="iindex"></span><span id="IINDEX"></span>*ииндекс*
</dt> <dd>

Указывает регистр, значение которого необходимо получить в виде целого числа. Целое число должно находиться в диапазоне от 0 до 23.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение, представляющее содержимое указанного регистра.

## <a name="remarks"></a>Комментарии

Диск управляет регистрами системных параметров (Спрмс). Приложению проигрывателя не требуется доступ к этим регистрам для использования стандартных функций навигации. Спрмс представляет состояние проигрывателя. Каждый из них имеет смысл, задается пользовательскими предпочтениями, командами диска и другими случаями, когда приложение не имеет прямого контроля. Приложение может считывать эти регистры, но не может выполнять запись в них. Чтобы эффективно использовать эти регистры, вам, возможно, потребуется более подробная информация о командах перехода DVD, чем указано в этой документации. В следующей таблице показано содержимое каждого регистра. Более подробные сведения о содержимом регистров см. в разделе [ **IDvdInfo2:: жеталлспрмс**](/windows/desktop/api/Strmif/nf-strmif-idvdinfo2-getallsprms)



| Регистрация | Содержимое                        |
|----------|---------------------------------|
| 0        | Код языка меню              |
| 1        | Номер аудиопотока             |
| 2        | Номер потока субтитров        |
| 3        | Текущий номер угла            |
| 4        | Номер текущего заголовка            |
| 5        | Номер заголовка                    |
| 6        | Номер PGC                      |
| 7        | Текущий номер главы (PTT)    |
| 8        | Номер выделенной кнопки       |
| 9        | Таймер навигации                |
| 10       | Переход PGC для навигации. Таймер         |
| 11       | Режим звуковой презентации Караоке |
| 12       | Код страны или региона ПМЛ         |
| 13       | пмл                             |
| 14       | Параметр видео                   |
| 15       | Звуковые возможности                |
| 16       | Язык звука                  |
| 17       | Расширение языка звука        |
| 18       | Язык субтитров             |
| 19       | Расширение языка субтитров   |
| 20       | Код региона проигрывателя              |
| 21       | Зарезервировано                        |
| 22       | Зарезервировано                        |
| 23       | Зарезервировано                        |
