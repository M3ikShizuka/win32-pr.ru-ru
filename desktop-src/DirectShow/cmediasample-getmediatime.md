# <a name="cmediasamplegetmediatime-method"></a>Кмедиасампле. Жетмедиатиме, метод

`GetMediaTime`Метод извлекает значения времени носителя для этого образца. Этот метод реализует метод [**имедиасампле:: жетмедиатиме**](/windows/desktop/api/Strmif/nf-strmif-imediasample-getmediatime) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetMediaTime(
   LONGLONG *pStart,
   LONGLONG *pEnd
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пстарт* 
</dt> <dd>

Указатель на переменную, которая получает время начала воспроизведения.

</dd> <dt>

*Ожидаем* 
</dt> <dd>

Указатель на переменную, которая получает время окончания воспроизведения носителя.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из значений **HRESULT** , приведенных в следующей таблице.



| Код возврата                                                                                                  | Описание                                         |
|--------------------------------------------------------------------------------------------------------------|-----------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                         | Успешно.<br/>                                 |
| <dl> <dt>**\_ \_ \_ \_ не задано время в \_ подключении к VFW E Media**</dt> </dl> | Для этого образца не заданы времена носителя.<br/> |



 

## <a name="remarks"></a>Remarks

Переменная-член [**кмедиасампле:: m \_ медиаенд**](cmediasample-m-mediaend.md) указывает смещение от [**кмедиасампле:: m \_ медиастарт**](cmediasample-m-mediastart.md), но значение, полученное параметром *"пересчитать* ", является абсолютным временем носителя, вычисленным как **m \_ медиастарт**  +  **m \_ медиаенд**.

Дополнительные сведения о времени мультимедиа см. [в разделе время и часы в DirectShow](time-and-clocks-in-directshow.md).
