# <a name="cqueue-class"></a>Класс Ккуеуе

Шаблон класса **ккуеуе** реализует простую, статично изменяемую очередь.



| Открытые методы                                  | Описание                             |
|-------------------------------------------------|-----------------------------------------|
| [**ккуеуе**](cqueue-cqueue.md)                 | Метод конструктора.                     |
| [**~ Ккуеуе**](cqueue--cqueue.md)               | Метод деструктора.                      |
| [**жеткуеуеобжект**](cqueue-getqueueobject.md) | Извлекает следующий элемент из очереди. |
| [**путкуеуеобжект**](cqueue-putqueueobject.md) | Помещает элемент в очередь.            |



 

## <a name="remarks"></a>Remarks

Конструктор класса задает размер очереди. Используйте [**ккуеуе::P уткуеуеобжект**](cqueue-putqueueobject.md) , чтобы поместить элемент в очередь, и метод [**Ккуеуе:: жеткуеуеобжект**](cqueue-getqueueobject.md) для вывода элемента в очередь. Если очередь заполнена, метод **путкуеуеобжект** блокируется до тех пор, пока не будет выведен элемент из очереди. Если очередь пуста, **жеткуеуеобжект** блокируется до постановки элемента в очередь. Параметр шаблона задает тип элемента. Например:


```
CQueue<int> number_queue;
number_queue.PutQueueObject(7);
```



Класс использует два семафора для управления операциями очереди, семафором Get и семафором "разместить". Метод [**жеткуеуеобжект**](cqueue-getqueueobject.md) ожидает семафор "Get" (с помощью функции **WaitForSingleObject** ) и освобождает семафор "помещаемие" (с помощью функции **ReleaseSemaphore** ). Метод [**путкуеуеобжект**](cqueue-putqueueobject.md) ожидает семафора "помещает" и освобождает семафор "Get". Класс использует критическую секцию для сериализации операций очереди между несколькими потоками.
