# <a name="cbasecontrolwindowhidecursor-method"></a>Кбасеконтролвиндов. Хидекурсор, метод

`HideCursor`Метод скрывает или отображает курсор.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT HideCursor(
   long HideCursor
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*хидекурсор* 
</dt> <dd>

Значение, указывающее, следует ли отображать курсор. Задайте значение ОАТРУЕ, чтобы скрыть курсор, или ОАФАЛСЕ для отображения курсора.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** .
