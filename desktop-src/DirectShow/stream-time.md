# <a name="stream_time"></a>\_время потока

> [!Note]  
> Это нерекомендуемый API. Новые приложения не должны использовать его.

 

\_Тип данных потокового времени используется для выражения времени ссылки в потоковой передаче мультимедиа Microsoft DirectShow. Единицы измерения — 100 наносекунд.


```C++
typedef LONGLONG STREAM_TIME;
```



## <a name="remarks"></a>Комментарии

Этот тип данных эквивалентен типу данных [**ссылочного \_ времени**](reference-time.md) DirectShow.
