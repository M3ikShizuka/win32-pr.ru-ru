# <a name="coareftimeoperator-double-method"></a>Коарефтиме. operator, двойной метод

Этот оператор преобразует время ссылки в значение **типа Double** .

## <a name="syntax"></a>Синтаксис


```C++
double operator double();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает время ссылки в секундах.
