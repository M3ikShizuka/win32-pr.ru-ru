# <a name="camscheduleadvise-method"></a>Камсчедуле. Advise, метод

`Advise`Метод отправляет все запросы, запланированные на указанное время или ранее.

## <a name="syntax"></a>Синтаксис


```C++
REFERENCE_TIME Advise(
  [ref] const REFERENCE_TIME &rtTime
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*рттиме* \[ ЗНАЧ\]
</dt> <dd>

Значение, указывающее текущее время ссылки.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает время ссылки следующего запланированного запроса на выполе или максимальное \_ время, если не осталось ничего.

## <a name="remarks"></a>Комментарии

Когда часы вызывают этот метод, он указывает текущее время ссылки. Планировщик определяет, какие запросы на выступающий истек срок действия, если они есть, и отправляет их. Если срок действия одноразового запроса истекает, Планировщик удаляет его. Если срок действия периодического запроса истекает, планировщик повторно планирует его для следующего времени. Метод возвращает время следующего ожидающего запроса.

Чтобы отправить запрос advise, планировщик сигнализирует о событии или семафоре, указанном в параметре *хнотифи* метода [**Камсчедуле:: аддадвисепаккет**](camschedule-addadvisepacket.md) .
