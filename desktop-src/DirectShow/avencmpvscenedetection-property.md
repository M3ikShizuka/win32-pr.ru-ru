# <a name="avencmpvscenedetection-property"></a>Авенкмпвсценедетектион, свойство

Указывает, как кодировщик ведет себя при обнаружении новой сцены. Это свойство применяется к кодировщикам видео MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпвсценедетектион**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенкмпвсценедетектион**](/windows/win32/api/codecapi/ne-codecapi-eavencmpvscenedetection) .
