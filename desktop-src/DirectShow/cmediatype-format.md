# <a name="cmediatypeformat-method"></a>Кмедиатипе. Format, метод

`Format`Метод получает указатель на блок формата.

## <a name="syntax"></a>Синтаксис


```C++
BYTE* Format() const;
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает элемент **пбформат** .
