# <a name="avencvideopixelaspectratio-property"></a>Авенквидеопикселаспектратио, свойство

Задает пропорцию в пикселях.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеопикселаспектратио**

## <a name="property-value"></a>Значение свойства

Верхние 16 разрядов значения содержат ширину, а младшие 16 бит содержат высоту.
