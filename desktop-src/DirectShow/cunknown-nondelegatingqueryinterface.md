# <a name="cunknownnondelegatingqueryinterface-method"></a>Кункновн. Нонделегатингкуеринтерфаце, метод

Извлекает указатель интерфейса и увеличивает счетчик ссылок. Этот метод реализует метод **инонделегатингункновн:: нонделегатингкуеринтерфаце** .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT NonDelegatingQueryInterface(
   REFIID riid,
   void   **ppv
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*riid* 
</dt> <dd>

Идентификатор интерфейса.

</dd> <dt>

*ппв* 
</dt> <dd>

Адрес указателя для получения интерфейса.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из значений **HRESULT** , приведенных в следующей таблице.



| Код возврата                                                                                   | Описание                                        |
|-----------------------------------------------------------------------------------------------|----------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Успешно.<br/>                                |
| <dl> <dt>**\_интерфейс E**</dt> </dl> | Объект не поддерживает этот интерфейс.<br/> |
| <dl> <dt>**\_указатель E**</dt> </dl>     | **Пустой** аргумент указателя.<br/>              |



 

## <a name="remarks"></a>Remarks

Класс **кункновн** предоставляет только интерфейс **иукновн** . Переопределите этот метод, чтобы предоставить дополнительные интерфейсы. Сведения о том, как переопределить этот метод, см. [в разделе Реализация IUnknown](how-to-implement-iunknown.md).
