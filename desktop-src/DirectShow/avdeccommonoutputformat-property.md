# <a name="avdeccommonoutputformat-property"></a>Авдеккоммонаутпутформат, свойство

Задает формат выходных данных для декодера.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**BSTR** (**VT \_ BSTR**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авдеккоммонаутпутформат**

## <a name="property-value"></a>Значение свойства



| Идентификатор GUID                                                               | Описание                                                                                                                                                                                                         |
|--------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| КОДЕКАПИ \_ GUID \_ авдекаудиуутпутформат \_ PCM                        | Звук PCM с использованием любого числа каналов                                                                                                                                                                             |
| КОДЕКАПИ \_ GUID \_ авдекаудиуутпутформат \_ PCM для \_ наушников            | Аудио PCM стерео, с использованием только левой или правой (в наличии/Стро) довнмикс                                                                                                                                                        |
| \_Автоматический стерео кодекапи GUID \_ авдекаудиуутпутформат \_ PCM \_ \_          | Звук стерео PCM с использованием автоматического выбора режима стерео довнмикс (с конца/ro или lt/RT). Это значение можно использовать для звуковых форматов, в которых входной поток определяет предпочтительный режим довнмикс, например Dolby AC-3. |
| КОДЕКАПИ \_ GUID \_ авдекаудиуутпутформат \_ PCM \_ стерео \_ матриксенкодед | Аудио PCM стерео, использование стерео довнмикс (lt/RT) в кодировке Matrix                                                                                                                                                       |
| КОДЕКАПИ \_ GUID \_ авдекаудиуутпутформат \_ SPDIF \_ битовый поток           | S/PDIF (формат цифрового интерфейса Sony/Philips) сжатый битовый поток, как определено в IEC-60958                                                                                                                        |
| КОДЕКАПИ \_ GUID \_ авдекаудиуутпутформат \_ SPDIF, \_ PCM                 | Стерео PCM S/PDIF, как определено в IEC-60958                                                                                                                                                                          |
