# <a name="ctransinplacefiltertypesmatch-method"></a>Ктрансинплацефилтер. Типесматч, метод

`TypesMatch`Метод определяет, соответствует ли входной тип носителя выходному типу носителя.

## <a name="syntax"></a>Синтаксис


```C++
BOOL TypesMatch();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если типы входных и выходных носителей совпадают, или **false** в противном случае.
