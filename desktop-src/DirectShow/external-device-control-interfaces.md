# <a name="external-device-control-interfaces"></a>Интерфейсы управления внешними устройствами

Эти интерфейсы поддерживают управление приложениями на внешних устройствах, таких как видеокамеры.



| Интерфейс                                            | Описание                                                                             |
|------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [**иамкамераконтрол**](/windows/desktop/api/Strmif/nn-strmif-iamcameracontrol)         | Управление камерой.                                                                       |
| [**иамекстдевице**](/windows/desktop/api/Strmif/nn-strmif-iamextdevice)                 | Управление внешним устройством, например ВИДЕОМАГНИТОФОНА или видеомагнитофоном.                                  |
| [**иамексттранспорт**](/windows/desktop/api/Strmif/nn-strmif-iamexttransport)           | Управление функциями передачи ВИДЕОМАГНИТОФОНА, такими как воспроизведение, пауза, запись, быстрая переадресация и перемотка назад. |
| [**иамтимекодедисплай**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodedisplay)     | Задайте свойства на устройстве, которое отображает временной код SMPTE.                                |
| [**иамтимекодеженератор**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodegenerator) | Задайте свойства на устройстве, которое создает код времени SMPTE.                               |
| [**иамтимекодереадер**](/windows/desktop/api/Strmif/nn-strmif-iamtimecodereader)       | Считывает временной код SMPTE с внешнего устройства.                                            |
