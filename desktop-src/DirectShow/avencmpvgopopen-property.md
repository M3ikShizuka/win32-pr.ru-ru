# <a name="avencmpvgopopen-property"></a>Авенкмпвгопопен, свойство

Указывает, создает ли кодировщик открытые группы изображений (группы GOP) или закрытый группы GOP. Это свойство применяется к кодировщикам видео MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпвгопопен**

## <a name="property-value"></a>Значение свойства



| Значение          | Описание |
|----------------|-------------|
| ВАРИАНТ \_ false | Закрытые группы GOP |
| ВАРИАНТ \_ true  | Открыть группы GOP   |



 

## <a name="remarks"></a>Комментарии

Открытый GOP содержит разностные кадры, которые ссылаются на кадры из предыдущего GOP. Закрытый GOP не содержит никаких разностных кадров, ссылающихся на предыдущий GOP.
