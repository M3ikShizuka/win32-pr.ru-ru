# <a name="ctransformfilteralterquality-method"></a>Ктрансформфилтер. Алтеркуалити, метод

`AlterQuality`Метод уведомляет фильтр о запросе изменения качества.

## <a name="syntax"></a>Синтаксис


```C++
virtual HRESULT AlterQuality(
   Quality q
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*формате* 
</dt> <dd>

Структура [**качества**](/windows/win32/api/strmif/ns-strmif-quality) , содержащая сообщение контроля качества.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** . Возможные значения включают в себя, показанные в следующей таблице.



| Код возврата                                                                             | Описание                                                                           |
|-----------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------|
| <dl> <dt>**S \_ false**</dt> </dl> | Не обрабатывает сообщение качества. Сообщение должно быть передано в поток.<br/> |
| <dl> <dt>**\_ОК**</dt> </dl>    | Обработано сообщение качества. Никаких дополнительных действий не требуется.<br/>               |



 

## <a name="remarks"></a>Remarks

Переопределите этот метод, если фильтр может выполнять контроль качества. Дополнительные сведения см. в разделе [Управление качеством](quality-control-management.md).
