# <a name="cdrawimagem_bstretch-member"></a>Элемент Кдравимаже:: m \_ бстретч

`m_bStretch`Переменная члена указывает, нужно ли растянуть изображение видео в соответствии с конечным окном. Значение равно **false** , если исходный прямоугольник имеет тот же размер, что и конечный прямоугольник, или **true** в противном случае.

## <a name="syntax"></a>Синтаксис


```C++
BOOL m_bStretch;
```
