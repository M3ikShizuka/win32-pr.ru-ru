# <a name="crenderedinputpinactive-method"></a>Крендерединпутпин. активный метод

`Active`Метод уведомляет ПИН-код о том, что фильтр активен. Этот метод переопределяет метод [**кбасепин:: Active**](cbasepin-active.md) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Active();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает \_ ОК, если успешно, или код ошибки в противном случае.
