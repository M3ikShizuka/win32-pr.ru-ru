# <a name="cbasepinm_bcanreconnectwhenactive-member"></a>Элемент Кбасепин:: m \_ бканреконнектвхенактиве

Флаг, указывающий, поддерживает ли ПИН-код динамическое повторное подключение.

## <a name="syntax"></a>Синтаксис


```C++
bool m_bCanReconnectWhenActive;
```



## <a name="remarks"></a>Remarks

По умолчанию этот флаг **имеет значение false**.
