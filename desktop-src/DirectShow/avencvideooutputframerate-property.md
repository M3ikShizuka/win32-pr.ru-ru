# <a name="avencvideooutputframerate-property"></a>Авенквидеуутпутфрамерате, свойство

Указывает частоту кадров в потоке вывода кодировщика, в кадрах в секунду.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT64** (**VT \_ UI8**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеуутпутфрамерате**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является отношением, определяющим частоту кадров. Верхние 32 бита содержат числитель, а младшие 32 биты содержат знаменатель. В следующей таблице показаны распространенные частоты кадров в кадрах за секунду.



| Частота кадров | Соотношение      |
|------------|------------|
| 23,97      | 24000/1001 |
| 24         | 24/1       |
| 25         | 25/1       |
| 29,97      | 30000/1001 |
| 30         | 30/1       |
| 50         | 50/1       |
| 59,94      | 60000/1001 |
| 60         | 60/1       |



 

## <a name="remarks"></a>Комментарии

Приложения могут установить это свойство, чтобы указать частоту выходных кадров. Кодировщики также могут возвращать это свойство как возможность. В зависимости от кодировщика значение может быть ограничено частотой входного кадра. В противном случае кодировщик может преобразовать частоту кадров внутри. См. раздел [**свойство авенквидеуутпутфрамератеконверсион**](avencvideooutputframerateconversion-property.md).

Это свойство также используется с [кодировщиками камер H. 264 увк 1,5](/windows/desktop/medfound/camera-encoder-h264-uvc-1-5).
