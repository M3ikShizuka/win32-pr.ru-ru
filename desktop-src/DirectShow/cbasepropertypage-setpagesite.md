# <a name="cbasepropertypagesetpagesite-method"></a>Кбасепропертипаже. Сетпажесите, метод

`SetPageSite`Метод инициализирует страницу свойств и предоставляет указатель на интерфейс **ипропертипажесите** рамки свойства. Этот метод реализует метод **ипропертипаже:: сетпажесите** .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetPageSite(
   IPropertyPageSite *pPageSite
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ппажесите* 
</dt> <dd>

Указатель на интерфейс **ипропертипажесите** .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** . Ниже приведены возможные значения.



| Код возврата                                                                                  | Описание                    |
|----------------------------------------------------------------------------------------------|--------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>         | Успешно.<br/>            |
| <dl> <dt>**\_непредвиденная E**</dt> </dl> | Непредвиденный сбой.<br/> |



 

## <a name="remarks"></a>Remarks

Метод сохраняет указатель *ппажесите* в переменной члена [**кбасепропертипаже:: m \_ ппажесите**](cbasepropertypage-m-ppagesite.md) .
