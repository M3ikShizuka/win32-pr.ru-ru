# <a name="cmediapositioninvoke-method"></a>Кмедиапоситион. Invoke, метод

`Invoke`Метод предоставляет доступ к свойствам и методам, предоставляемым объектом.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Invoke(
   DISPID     dispidMember,
   REFIID     riid,
   LCID       lcid,
   WORD       wFlags,
   DISPPARAMS *pdispparams,
   VARIANT    *pvarResult,
   EXCEPINFO  *pexcepinfo,
   UINT       *puArgErr
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*диспидмембер* 
</dt> <dd>

Идентификатор элемента. Чтобы получить идентификатор диспетчеризации, используйте [**кмедиапоситион:: GetIdsOfNames**](cmediaposition-getidsofnames.md) .

</dd> <dt>

*riid* 
</dt> <dd>

Зарезервировано для последующего использования. Должен иметь \_ значение IID null.

</dd> <dt>

*lcid* 
</dt> <dd>

Контекст языкового стандарта, в котором следует интерпретировать аргументы.

</dd> <dt>

*вфлагс* 
</dt> <dd>

Флаги, описывающие контекст вызова.

</dd> <dt>

*пдисппарамс* 
</dt> <dd>

Указатель на структуру **диппарамс** , содержащую аргументы.

</dd> <dt>

*пварресулт* 
</dt> <dd>

Указатель на **вариант** , который получает результат, или **значение NULL** , если вызывающий объект не ждет результата.

</dd> <dt>

*пексцепинфо* 
</dt> <dd>

Указатель на структуру, которая получает сведения об исключении.

</dd> <dt>

*puArgErr* 
</dt> <dd>

Указатель на переменную, которая получает индекс первого аргумента, который вызывает ошибку.

</dd> </dl>
