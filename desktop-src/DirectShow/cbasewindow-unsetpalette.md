# <a name="cbasewindowunsetpalette-method"></a>Кбасевиндов. Унсетпалетте, метод

`UnsetPalette`Метод удаляет текущую палитру окна и восстанавливает системную палитру по умолчанию.

## <a name="syntax"></a>Синтаксис


```C++
void UnsetPalette();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
