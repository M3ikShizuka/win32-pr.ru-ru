# <a name="cbasecontrolvideogetcurrentimage-method"></a>Кбасеконтролвидео. Жеткуррентимаже, метод

`GetCurrentImage`Метод извлекает копию текущего изображения в модуле подготовки отчетов.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetCurrentImage(
   long *pBufferSize,
   long *pVideoImage
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пбуфферсизе* 
</dt> <dd>

Указатель на размер выходного буфера.

</dd> <dt>

*пвидеоимаже* 
</dt> <dd>

Указатель на выходной буфер для изображения.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** , которое зависит от реализации. может принимать одно из следующих значений или другие значения, отсутствующие в списке.



| Код возврата                                                                                        | Описание                                                                       |
|----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| <dl> <dt>**\_Ошибка E**</dt> </dl>             | Ошибка.<br/>                                                               |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>       | Недопустимый аргумент.<br/>                                                      |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl>      | Недостаточно памяти. Возвращается, если параметр *пвидеоинфо* имеет **значение NULL**.<br/>   |
| <dl> <dt>**NOERROR**</dt> </dl>             | Успешно.<br/>                                                               |
| <dl> <dt>**VFW \_ E \_ не \_ приостановлен**</dt> </dl> | Не удалось выполнить операцию, поскольку фильтр не приостановлен.<br/> |



 

## <a name="remarks"></a>Комментарии

Эта функция члена извлекает изображение из примера и копирует его в выходной буфер. Раздел видео, скопированный в выходной буфер, отражает исходный прямоугольник, заданный через интерфейс [**ибасиквидео**](/windows/desktop/api/Control/nn-control-ibasicvideo) . Он не отражает конечный прямоугольник.
