# <a name="avencmpvlevel-property"></a>Авенкмпвлевел, свойство

Задает уровень MPEG-2. Это свойство применяется к кодировщикам видео MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпвлевел**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенкмпвлевел**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpvlevel) .
