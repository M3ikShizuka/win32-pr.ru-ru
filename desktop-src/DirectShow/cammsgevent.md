# <a name="cammsgevent-class"></a>Класс Каммсжевент

![Иерархия классов каммсжевент](images/util06.png)

`CAMMsgEvent`Класс является оболочкой для объектов событий, выполняющих обработку сообщений.

Этот класс является производным от объекта [**камевент**](camevent.md) . Он добавляет один метод [**каммсжевент:: ваитмсг**](cammsgevent-waitmsg.md), который отправляет входящие сообщения в ожидании.



| Открытые методы                                 | Описание                                                          |
|------------------------------------------------|----------------------------------------------------------------------|
| [**каммсжевент**](cammsgevent-cammsgevent.md) | Конструктор.                                                         |
| [**ваитмсг**](cammsgevent-waitmsg.md)         | Ожидает получения сигнала о событии, а также при диспетчеризации отправленных сообщений. |
