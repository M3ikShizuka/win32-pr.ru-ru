# <a name="cdrawimagescalesourcerect-method"></a>Кдравимаже. Скалесаурцерект, метод

`ScaleSourceRect`Метод масштабирует прямоугольник, если существует разница между собственным размером видео и форматом типа мультимедиа.

## <a name="syntax"></a>Синтаксис


```C++
virtual RECT ScaleSourceRect(
   const RECT *pSource
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*псаурце* 
</dt> <dd>

Указатель на немасштабируемый прямоугольник.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает масштабируемый прямоугольник.

## <a name="remarks"></a>Комментарии

В классе **кдравимаже** этот метод возвращает *псаурце* без каких бы то ни было изменений. Этот метод можно переопределить, если фильтр растягивает входящее изображение видео. Например, размер собственного видео может быть 320 240, но тип носителя для входного ПИН-кода может быть 640 480. В этом случае фильтру потребуется масштабировать исходный прямоугольник с коэффициентом 2.
