# <a name="countryregion-assignments"></a>Назначения стран и регионов

Коды вызова стандарта международной телекоммуникационных союзов (ITU) передаются в метод [**иамтунер::p UT \_ каунтрикоде**](/windows/desktop/api/Strmif/nf-strmif-iamtuner-put_countrycode) для выбора сопоставлений частот телевизора по умолчанию.

Например, передача 20 кода, вызывающего код ITU-T для Египет, выберет частоту телевизора по умолчанию для Египет.

Список кодов вызова страны см. в текущей рекомендации ITU-T по назначенным кодам стран.
