# <a name="directshow-structures"></a>Структуры DirectShow

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Службы редактирования DirectShow определяют следующие структуры.



| Структура                             | Описание                                          |
|---------------------------------------|------------------------------------------------------|
| [**Декстер, \_ параметр**](dexter-param.md) | Указывает свойство для перехода или воздействия.      |
| [**\_значение Декстер**](dexter-value.md) | Задает значение свойства.                   |
| [**SCompFmt0**](scompfmt0.md)        | Содержит сведения о форматировании для интеллектуального повторного сжатия. |
