# <a name="currenttime-property"></a>CurrentTime, свойство

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`CurrentTime`Свойство получает текущее время воспроизведения.

``` syntax
[ sCurTime = ] MSWebDVD.CurrentTime
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает **строку** из 11 символов, представляющую текущее время воспроизведения в формате "*чч: мм: СС: FF*" (часы, минуты, секунды, кадры).

## <a name="remarks"></a>Комментарии

Это свойство доступно только для чтения и не имеет значения по умолчанию.
