# <a name="copp-command-reference"></a>Справочник по командам Копп

В этом разделе описываются команды сертифицированного протокола выходной защиты (Копп). Определены следующие команды.



| Команда              | GUID                             |
|----------------------|----------------------------------|
| Задать уровень защиты | **ДКСВА \_ коппсетпротектионлевел** |
| Настройка сигнализации        | **ДКСВА \_ коппсетсигналинг**       |



 

Команда "задать уровень защиты"

Задает уровень защиты для указанного механизма защиты выходных данных. В зависимости от соединителя можно применить несколько механизмов защиты для одного соединителя с разными параметрами для каждого механизма.

**GUID**: дксва \_ коппсетпротектионлевел

**Входные данные**: [**Структура \_ коппсетпротектионлевелкмддата дксва**](/windows/desktop/api/dxva9typ/ns-dxva9typ-dxva_coppsetprotectionlevelcmddata) .

Команда "задать сигнализацию"

Указывает сведения о видеосигнале, отличном от уровня защиты.

Для некоторых стандартов защиты с поддержкой CGMS-A требуется, чтобы сигнал телевсион содержал информацию о пропорции и другую информацию в тех же ВБИах, что и биты CGMS-A. Телевизоры могут плохо отображаться, если сведения о пропорции не соответствуют потоку видео. Приложение может использовать эту команду, чтобы задать пропорции, чтобы графический драйвер мог формировать правильные пакеты ВБИ.

Эта команда также предназначена для расширения, если в будущих стандартах требуются дополнительные сведения о сигналах.

**GUID**: дксва \_ коппсетсигналинг

**Входные данные**: [**Структура \_ коппсетсигналингкмддата дксва**](/windows/desktop/api/dxva9typ/ns-dxva9typ-dxva_coppsetsignalingcmddata) .
