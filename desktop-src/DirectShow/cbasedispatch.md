# <a name="cbasedispatch-class"></a>Класс Кбаседиспатч

![Иерархия классов кбаседиспатч](images/cutil01.png)

класс **кбаседиспатч** является базовым классом для реализации интерфейса **IDispatch** в фильтре DirectShow.

этот класс ограничен для поддержки интерфейсов, совместимых с автоматизацией, экспортируемых DirectShow библиотекой типов, куартзтипелиб. Например, классы [**кмедиаконтрол**](cmediacontrol.md) и [**кмедиапоситион**](cmediaposition.md) используют **Кбаседиспатч** для реализации [**имедиаконтрол**](/windows/desktop/api/Control/nn-control-imediacontrol) и [**имедиапоситион**](/windows/desktop/api/Control/nn-control-imediaposition)соответственно. В связи с этим ограничением, вероятно, нет причин использовать **кбаседиспатч** непосредственно в собственных фильтрах.

Чтобы использовать этот класс, выполните следующие действия.

-   Объявите новый класс, поддерживающий **IDispatch**.
-   Присвойте новому классу закрытую переменную-член типа **кбаседиспатч**.
-   Реализуйте методы **IDispatch** .
-   В методах **IDispatch** вызовите методы **кбаседиспатч** .

Дополнительные сведения см. в исходном коде любого из примеров классов, объявленных в Ктлутил. h.



| Открытые методы                                             | Описание                                                                                                         |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**кбаседиспатч**](cbasedispatch-cbasedispatch.md)       | Метод конструктора.                                                                                                 |
| [**~ Кбаседиспатч**](cbasedispatch--cbasedispatch.md)     | Метод деструктора.                                                                                                  |
| [**GetIDsOfNames**](cbasedispatch-getidsofnames.md)       | Карты набор имен в соответствующий набор идентификаторов dispid.                                                              |
| [**GetTypeInfo**](cbasedispatch-gettypeinfo.md)           | Извлекает сведения о типе для объекта, который затем можно использовать для получения сведений о типе для интерфейса. |
| [**жеттипеинфокаунт**](cbasedispatch-gettypeinfocount.md) | Возвращает число интерфейсов сведений о типе, предоставляемых объектом.                                            |
