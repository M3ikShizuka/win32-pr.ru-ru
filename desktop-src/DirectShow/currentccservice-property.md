# <a name="currentccservice-property"></a>Курренткксервице, свойство

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`CurrentCCService`Свойство задает или получает текущую службу закрытого субтитров.

``` syntax
[ iService = ] MSWebDVD.CurrentCCService
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение, указывающее тип включенной службы субтитров.

## <a name="remarks"></a>Remarks

Возможные значения этого свойства:



| Значение | Описание                                                          |
|-------|----------------------------------------------------------------------|
| 0x0   | Нет текущей службы. Это значение по умолчанию.                       |
| 0x1   | Истинное название, первое поле. Служба по умолчанию.                       |
| 0x2   | Субтитры для вторичного языка, второе поле. Обычно не используется. |



 

Это свойство доступно для чтения и записи.
