# <a name="ctransinplacefilteroutputpin-method"></a>Ктрансинплацефилтер. OutputPin, метод

`OutputPin`Метод получает указатель на выходной ПИН-код фильтра.

## <a name="syntax"></a>Синтаксис


```C++
CTransInPlaceOutputPin* OutputPin();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает переменную члена [**ктрансформфилтер:: m \_ паутпут**](ctransformfilter-m-poutput.md) .
