# <a name="x-property"></a>x, свойство

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`x`Свойство задает или получает x-координату верхнего левого угла прямоугольника.

``` syntax
[ iXPos = ] DVDRect.x
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает координату X верхнего левого угла прямоугольника в виде целого числа.

## <a name="remarks"></a>Комментарии

Это свойство доступно для чтения и записи и не имеет значения по умолчанию.
