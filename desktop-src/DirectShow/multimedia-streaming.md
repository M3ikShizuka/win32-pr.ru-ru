# <a name="multimedia-streaming"></a>Потоковая передача мультимедиа

> [!Note]  
> Эти API-интерфейсы являются устаревшими. Приложения должны использовать [**образец фильтра захвата**](sample-grabber-filter.md) или реализовать настраиваемый фильтр для получения данных из графа фильтра DirectShow.

 

Потоковая передача мультимедиа — это набор API-интерфейсов, позволяющих приложению получать аудио-и видеоданные из графа фильтра DirectShow без написания пользовательского фильтра DirectShow.
