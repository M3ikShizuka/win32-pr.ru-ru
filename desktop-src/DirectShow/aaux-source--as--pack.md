## <a name="remarks"></a>Комментарии

Ниже перечислены коды полей, которые представляют интерес:

-   LF: флаг заблокированного режима. Указывает, заблокирован ли звук.
    -   0 = заблокировано
    -   1 = разблокировано
-   AF размер: размер звуковых кадров. Указывает количество звуковых образцов на кадр.

    Определение IEC 61834:

    -   00:1111 = 1068 образцов на кадр
    -   01:0000 = 1280 образцов на кадр

    Определение 314M SMPTE:

    -   01:0110 = 1602 образцов на кадр
    -   01:1000 = 1920 образцов на кадр

    В зависимости от частоты кадров точное количество выборок в кадре может отличаться. Например, NTSC — 30000/1001 кадров в секунду (29,97 кадров/с). При наличии звука 32 кГц в кадре около 1067,73 звуковых образцов. Таким же номинальной ставкой является 1068, но фактическое число зависит от каждого кадра. Кроме того, при использовании незаблокированного звука количество звуковых образцов на кадр может различаться в пределах определенного диапазона с течением времени.

<!-- -->

-   SM: стерео режим.
    -   0 = стерео
    -   1 = моно
-   ЧН: количество звуковых каналов на звуковой блок.
    -   0 = один канал на звуковой блок;
    -   1 = два канала на звуковой блок
-   ЗВУКОВОЙ режим: указывает содержимое звукового сигнала для каждого канала. Интерпретация этого поля зависит от того, какие значения помещаются в поля SM и ЧН. Приведенные ниже определения относятся к значениям, используемым МСДВ; Дополнительные сведения см. в спецификациях.

    Определение IEC 61834:

    -   0000 = ch a/c/e/g является левым каналом, ch b/d/f/h является верным каналом
    -   1111 = нет звуковых данных

    Определение 314M SMPTE:

    -   0000 = CH1 (CH3)
    -   0001 = CH2 (CH4)

-   50/60: число полей.
    -   0 = 60 полей
    -   1 = 50 полей
-   СТИПЕ: системный тип.

    Определение IEC 61834:

    -   00000 = 525-60 или 625-50, двсд
    -   00001 = 525-60 или 625-50, двсл (см. IEC 61883-5)

    Определение SMPTE 314M/СПМТЕ 370:

    -   00000 = 2 звуковых блока на кадр видео
    -   00010 = 4 звуковых блока на кадр видео
    -   00011 = 8 звуковых блоков на кадр видео

-   SMP: частота выборки.
    -   000 = 48 кГц
    -   010 = 32 кГц
-   QU: дискретизация.
    -   0 = линейное 16 бит
    -   1 = 12 бит без линейного
