# <a name="ezrgb24-filter-sample"></a>Образец фильтра EZRGB24

## <a name="description"></a>Описание

Фильтр EZRGB24 является фильтром преобразования обработки изображений. Этот фильтр выполняет ряд индивидуальных видеоэффектов, включая красный, зеленый, синий, затемненный, XOR, размытие, серый и эффект тиснения.

В этом примере показано, как использовать несколько базовых классов DirectShow, включая [**ктрансформфилтер**](ctransformfilter.md), [**кперсистстреам**](cpersiststream.md)и [**кбасепропертипаже**](cbasepropertypage.md).

## <a name="downloading-the-sample"></a>Загрузка образца

Чтобы скачать примеры пакета SDK для DirectShow, установите последнюю версию [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Этот пример устанавливается по следующему пути: *\[ \] корневые примеры SDK* \\ \\ мультимедиа \\ DirectShow \\ фильтры \\ EZRGB24.
