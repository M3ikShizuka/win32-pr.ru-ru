# <a name="getnumberofchapters-method"></a>Метод Жетнумберофчаптерс

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`GetNumberOfChapters`Метод извлекает количество глав в указанном заголовке.

``` syntax
[iChapters = ] MSWebDVD.GetNumberOfChapters(iTitle)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="iTitle"></span><span id="ititle"></span><span id="ITITLE"></span>*ититле*
</dt> <dd>

Задает заголовок в виде целого числа.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение от 1 до 999, указывающее количество глав.
