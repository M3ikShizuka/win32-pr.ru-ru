# <a name="mapper-sample"></a>Пример сопоставителя

## <a name="description"></a>Описание

В этом примере демонстрируется использование метода [**IFilterMapper2:: енумматчингфилтерс**](/windows/desktop/api/Strmif/nf-strmif-ifiltermapper2-enummatchingfilters) для нахождение фильтров в реестре.

Пользовательский интерфейс примера приложения содержит элементы управления, соответствующие параметрам метода [**енумматчингфилтерс**](/windows/desktop/api/Strmif/nf-strmif-ifiltermapper2-enummatchingfilters) , такие как минимальное значение, категории закрепления и типы мультимедиа. Выберите нужные параметры поиска, а затем нажмите кнопку **Поиск** , чтобы перечислить фильтры, соответствующие этим параметрам.

## <a name="downloading-the-sample"></a>Загрузка образца

чтобы скачать примеры пакета SDK для DirectShow, установите последнюю версию [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

этот пример устанавливается по следующему пути: \[ корневые примеры SDK \] \\ \\ мультимедиа \\ DirectShow прочие \\ средства \\ сопоставления.
