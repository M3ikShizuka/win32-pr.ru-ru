# <a name="cenummediatypesnext-method"></a>Ценуммедиатипес. Next, метод

`Next`Метод извлекает указанное число типов мультимедиа. Этот метод реализует метод [**иенуммедиатипес:: Next**](/windows/desktop/api/Strmif/nf-strmif-ienummediatypes-next) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT Next(
   ULONG         cMediaTypes,
   AM_MEDIA_TYPE **ppMediaTypes,
   ULONG         *pcFetched
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*кмедиатипес* 
</dt> <dd>

Число извлекаемых типов мультимедиа.

</dd> <dt>

*ппмедиатипес* 
</dt> <dd>

Массив указателей на [**структуры \_ \_ типов мультимедиа**](/windows/win32/api/strmif/ns-strmif-am_media_type) *кпинс* размера.

</dd> <dt>

*пкфетчед* 
</dt> <dd>

Указатель на переменную, которая получает число типов мультимедиа, возвращаемых методом. Может иметь **значение NULL** , если *кмедиатипес* равен 1.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из значений **HRESULT** , приведенных в следующей таблице.



| Код возврата                                                                                                | Описание                                                                         |
|------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| <dl> <dt>**S \_ false**</dt> </dl>                    | Не удалось получить столько типов носителей, сколько запрошено.<br/>                       |
| <dl> <dt>**\_ОК**</dt> </dl>                       | Успешно.<br/>                                                                 |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>               | Недопустимый аргумент.<br/>                                                        |
| <dl> <dt>**\_указатель E**</dt> </dl>                  | **Пустой** аргумент указателя.<br/>                                               |
| <dl> <dt>**VFW \_ E \_ перечисление не \_ \_ \_ синхронизировано**</dt> </dl> | Состояние ПИН-кода изменилось и теперь не согласуется с перечислителем.<br/> |



 

## <a name="remarks"></a>Комментарии

Если метод выполнен, массив, заданный параметром *ппмедиатипес* , содержит указатели \_ на \_ структуры типа мультимедиа. Число структур равно *\* пкфетчед*. Освободите каждый тип мультимедиа, вызвав функцию [**делетемедиатипе**](deletemediatype.md) .

Этот метод вызывает метод [**кбасепин:: жетмедиатипе**](cbasepin-getmediatype.md) ПИН-кода для получения типов мультимедиа.
