# <a name="about-the-audio-capture-filter"></a>О фильтре записи звука

DirectShow включает захват аналоговых входных данных на звуковой карте с помощью [фильтра записи звука](audio-capture-filter.md). Этот фильтр использует API-интерфейсы Wave *xxx* для управления любым устройством, драйвером которого поддерживаются эти API. Каждая карта в системе представляется отдельным экземпляром фильтра.

Фильтр записи звука предоставляет все входные данные на карточке, например микрофон или вход MIDI, в качестве ПИН-кода входа. Входной ПИН-код представляет то, что драйвер предоставляет в виде линий источника звука. однако данные не передаются через эти входные контакты и не подключаются к другим фильтрам DirectShow. Они просто предоставляют приложению способ управления входными данными. Приложение может использовать входной ПИН-код для включения или отключения входных данных, а также для установки смешанных свойств, таких как наподобие басов, высоких частот, панорамы и т. д. Доступное количество элементов управления зависит от драйвера. Чтобы полностью понять и использовать возможности определенной звуковой платы, вам потребуется документация от производителя карты.

> [!Note]  
> Вы можете записать CD-Audio входные данные, но этот поток звука уже пропустил преобразование "цифровой-аналоговый", поэтому качество звука будет утеряно с исходного компакт-диска.
