# <a name="setdelaytime"></a>сетделайтиме

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`SetDelayTime`Метод задает время задержки для подсказки, связанной с объектом **мсвебдвд** .

``` syntax
MSWebDVD.SetDelayTime(iDelayType, iNewVal)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="iDelayType"></span><span id="idelaytype"></span><span id="IDELAYTYPE"></span>*иделайтипе*
</dt> <dd>

Указывает тип задержки в виде целого числа.



| Значение | Описание                                                                                                                                                                                                                                      |
|-------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| -1    | Чтобы сбросить время задержки аутопоп до значения по умолчанию, установите *иневвал* в значение-1.                                                                                                                                                                       |
| 0     | Задайте время, в течение которого окно подсказки остается видимым, если указатель мыши находится в ограничивающем прямоугольнике инструмента.                                                                                                                         |
| 1     | Задайте период времени, в течение которого указатель должен оставаться свободным внутри ограничивающего прямоугольника инструмента перед отображением окна подсказки.                                                                                                                    |
| 2     | Задайте время, необходимое для отображения последующих окон всплывающей подсказки при переходе указателя с одного инструмента на другое.                                                                                                                          |
| 3     | Установите все времена задержки в пропорции по умолчанию. Аутопоп время в десять раз больше начального времени, а время перепоказа — одно-пятое начальное время. Если этот флаг установлен, используйте положительное значение Иневвал, чтобы указать начальное время в миллисекундах. |



 

</dd> <dt>

<span id="iNewVal"></span><span id="inewval"></span><span id="INEWVAL"></span>*иневвал*
</dt> <dd>

Указывает задержку в миллисекундах в виде целого числа.



| Значение                    | Описание                                                   |
|--------------------------|---------------------------------------------------------------|
| -1                       | Устанавливает для задержки, указанной в *иделайтипе* , значение по умолчанию |
| любое другое отрицательное значение | Задает для всех типов задержки значение по умолчанию.                  |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.
