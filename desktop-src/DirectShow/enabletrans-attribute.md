# <a name="enabletrans-attribute"></a>Атрибут енаблетранс

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`enabletrans`Атрибут указывает, включены ли переходы. Если значение равно **true**, механизм визуализации полностью визуализирует переходы. В противном случае он отображает их как «вырезание». Значение по умолчанию — **true**.

## <a name="possible-values"></a>Возможные значения

Следующие значения определены как TRUE: y, Y, t, T, 1. Следующие значения определены как FALSE: n, N, f, F, 0 (ноль).

## <a name="applies-to"></a>Применяется к

[**графика**](timeline-element.md)
