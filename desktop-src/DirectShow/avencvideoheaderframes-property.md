# <a name="avencvideoheaderframes-property"></a>Авенквидеохеадерфрамес, свойство

Указывает начальный номер кадра в заголовке Group of Pictures (GOP).

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеохеадерфрамес**

## <a name="property-value"></a>Значение свойства

Допустимый диапазон этого свойства зависит от частоты кадров.
