# <a name="avencddsurround90degreeephaseshift-property"></a>AVEncDDSurround90DegreeePhaseShift, свойство

Указывает, применяется ли сдвиг этапа 90 градусов к окружающим каналам в потоке цифрового аудио Dolby. Это свойство применяется к кодировщикам цифрового аудио Dolby.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ AVEncDDSurround90DegreeePhaseShift**
