# <a name="creftimeoperator-method"></a>Крефтиме. operator + = метод

## <a name="syntax"></a>Синтаксис


```C++
CRefTime& operator+=(
  [ref] const CRefTime &rt
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*RT* \[ ЗНАЧ\]
