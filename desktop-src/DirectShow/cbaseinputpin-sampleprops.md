# <a name="cbaseinputpinsampleprops-method"></a>Кбасеинпутпин. Самплепропс, метод

`SampleProps`Метод получает свойства последнего примера в структуре [**\_ \_ свойств SAMPLE2**](/windows/win32/api/strmif/ns-strmif-am_sample2_properties) .

## <a name="syntax"></a>Синтаксис


```C++
AM_SAMPLE2_PROPERTIES* SampleProps();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает адрес переменной члена [**кбасеинпутпин:: m \_ самплепропс**](cbaseinputpin-m-sampleprops.md) .
