# <a name="crendererinputpinbeginflush-method"></a>Крендереринпутпин. Бегинфлуш, метод

`BeginFlush`Метод начинает операцию записи на диск. Этот метод переопределяет метод [**кбасеинпутпин:: бегинфлуш**](cbaseinputpin-beginflush.md) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT BeginFlush();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** .
