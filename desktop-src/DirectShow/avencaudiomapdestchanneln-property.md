# <a name="avencaudiomapdestchanneln-property"></a>Авенкаудиомапдестчаннелн, свойство

Указывает, какой канал звука сопоставлен с каналом *N* в закодированном звуковом потоке.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**См. раздел Примечания.**

## <a name="remarks"></a>Комментарии

В этом справочном разделе описывается набор связанных свойств. Каждое свойство сопоставляет входной канал с указанным выходным каналом. Определены следующие свойства.



| GUID свойства                            | Описание             |
|------------------------------------------|-------------------------|
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel0**  | Целевой канал 0.  |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel1**  | Целевой канал 1.  |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel2**  | Целевой канал 2.  |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel3**  | Целевой канал 3.  |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel4**  | Целевой канал 4.  |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel5**  | Целевой канал 5.  |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel6**  | Целевой канал 6.  |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel7**  | Целевой канал 7.  |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel8**  | Целевой канал 8.  |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel9**  | Целевой канал 9.  |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel10** | Канал назначения 10. |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel11** | Канал назначения 11. |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel12** | Канал назначения 12. |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel13** | Канал назначения 13. |
| **КОДЕКАПИ \_ AVEncAudioMapDestChannel14** | Канал назначения 14. |
