# <a name="whql-tests"></a>Тесты WHQL

Тестирование различных сочетаний образов Панскан/Леттербокс и 704/720 для широкого спектра:

-   видео 16x9 (704 или 720), 16x9 (всего 720)
-   16x9 видео (704 или 720) Леттербокс, 4x3 подизображение
-   16x9 видео (704 или 720) Панскан, 4x3 подизображение
-   видео 4x3 (704 или 720), подизображение 4x3
