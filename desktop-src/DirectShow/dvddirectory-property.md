# <a name="dvddirectory-property"></a>Двддиректори, свойство

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`DVDDirectory`Свойство задает или получает текущий каталог текущего тома DVD.

``` syntax
[ sDirPath = ] MSWebDVD.DVDDirectory
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает строковое значение, указывающее путь к корневому каталогу DVD-диска.

## <a name="remarks"></a>Комментарии

Это свойство доступно для чтения и записи и не имеет значения по умолчанию. Используйте этот метод, чтобы задать корневой путь при наличии нескольких DVD-дисководов в системе. Если в системе только один диск и буква диска выше "C", объект Мсвебдвд находит его автоматически. Для стандартного DVD-Video диска корневой путь должен включать в себя \_ Каталог видео служб терминалов:


```VB
MSWebDVD.DVDDirectory = "e:\\video_ts"
```



Некоторые DVD-тома могут иметь видео в каталоге, отличном от "Video \_ TS". Общая идея состоит в том, что дополнительный "том DVD" (набор. ИФО. VOB и. Файлы БУП, которые обычно хранятся в \_ каталоге Video TS), можно поместить в подкаталог на диске. Если изменить корень так, чтобы он указывал на этот каталог, Мсвебдвд будет выполнять действия на этом отдельном томе DVD. Будет доступен новый набор меню, заголовков и т. д., не зависящий от заголовков в \_ корневом каталоге служб Video TS, который больше не будет доступен. Такие каталоги называются "скрытыми каталогами". В следующем примере показано, как задать скрытый каталог в качестве корневого, где "Hidden" — это заполнитель для любого имени, которое авторы диска задали в каталоге.


```VB
MSWebDVD.DVDDirectory = "d:\\webdvd\\hidden"
```
