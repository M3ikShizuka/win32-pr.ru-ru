# <a name="avencvideoheaderminutes-property"></a>Авенквидеохеадерминутес, свойство

Задает номер начальной минуты в заголовке Group of Pictures (GOP).

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеохеадерминутес**

## <a name="property-value"></a>Значение свойства

Допустимый диапазон этого свойства — от 0 до 59 включительно.
