# <a name="lock-attribute"></a>заблокировать атрибут

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`lock`Атрибут указывает, следует ли изменять объект. Если значение равно **true**, приложение должно считать объект заблокированным и запретить изменения в этом объекте или его дочерних элементах. Значение по умолчанию — **FALSE**.

## <a name="possible-values"></a>Возможные значения

Возможные значения. следующие значения определены как TRUE: y, Y, t, T, 1. Следующие значения определены как FALSE: n, N, f, F, 0 (ноль).

## <a name="applies-to"></a>Применяется к

[**клип**](clip-element.md), [**составной**](composite-element.md), [**эффекты**](effect-element.md), [**Группа**](group-element.md), [**временная шкала**](timeline-element.md), [**Переход**](transition-element.md)
