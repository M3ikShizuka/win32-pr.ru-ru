# <a name="coutputqueue-class"></a>Класс Каутпуткуеуе

![каутпуткуеуе](images/oput01.png)

`COutputQueue`Класс реализует очередь для доставки образцов мультимедиа.

Этот класс включает выходной ПИН-код для предоставления примеров в асинхронном режиме. Образцы помещаются в очередь, а рабочий поток доставляет их во входной ПИН-код. Очередь также может содержать управляющие сообщения, указывающие на новый сегмент, уведомление конца потока или операцию очистки.

Чтобы использовать этот класс, создайте объект **каутпуткуеуе** для каждого выходного контакта в фильтре. В методе конструктора укажите входной ПИН-код, подключенный к этому выходному ПИН-коду. При использовании этого класса закрепление вывода не вызывает методы непосредственно во входном ПИН-коде. Вместо этого он вызывает соответствующие методы в `COutputQueue` , как показано в следующей таблице.

Объект также может определить, следует ли использовать рабочий поток в зависимости от характеристик входного ПИН-кода.
