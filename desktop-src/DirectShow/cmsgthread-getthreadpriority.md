# <a name="cmsgthreadgetthreadpriority-method"></a>Кмсгсреад. Жетсреадприорити, метод

Использует функцию Microsoft Win32 **жетсреадприорити** для получения приоритета текущего рабочего потока.

## <a name="syntax"></a>Синтаксис


```C++
int GetThreadPriority();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает приоритет потока в виде целого числа.
