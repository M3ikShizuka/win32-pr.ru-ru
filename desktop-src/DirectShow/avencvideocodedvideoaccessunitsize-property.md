# <a name="avencvideocodedvideoaccessunitsize-property"></a>Авенквидеокодедвидеоакцессунитсизе, свойство

Указывает размер единиц доступа к видео в байтах. Это свойство применяется только к режимам управления с переменной скоростью (VBR).

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеокодедвидеоакцессунитсизе**

## <a name="property-value"></a>Значение свойства

Это свойство возвращается в виде диапазона значений. Чтобы получить поддерживаемый диапазон, вызовите метод [**икодекапи:: жетпараметерранже**](/windows/desktop/api/Strmif/nf-strmif-icodecapi-getparameterrange).
