# <a name="avencddchannelbwlowpassfilter-property"></a>Авенкддчаннелбвловпассфилтер, свойство

Указывает, применяется ли фильтр низкого прохода к основным каналам ввода в потоке цифрового аудио Dolby. Это свойство применяется к кодировщикам цифрового аудио Dolby.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкддчаннелбвловпассфилтер**
