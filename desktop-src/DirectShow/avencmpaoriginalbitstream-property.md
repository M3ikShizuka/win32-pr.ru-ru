# <a name="avencmpaoriginalbitstream-property"></a>Авенкмпаоригиналбитстреам, свойство

Задает значение по умолчанию для исходного бита в аудиопотока MPEG-1. Это свойство применяется к кодировщикам аудио в формате MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпаоригиналбитстреам**

## <a name="property-value"></a>Значение свойства

Это свойство может иметь следующие значения.



| Значение          | Описание          |
|----------------|----------------------|
| ВАРИАНТ \_ false | Исходный бит отключен. |
| ВАРИАНТ \_ true  | Исходный бит включен.  |



 

## <a name="remarks"></a>Remarks

Кодировщик может переопределить этот параметр на основе содержимого входного потока.
