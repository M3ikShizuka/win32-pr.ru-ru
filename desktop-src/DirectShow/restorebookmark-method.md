# <a name="restorebookmark-method"></a>Метод Ресторебукмарк

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`RestoreBookmark`Метод перемещает объект **мсвебдвд** в точку на диске, как указано в текущей закладке, с восстановленными параметрами аудио, видео и субтитров.

``` syntax
MSWebDVD.RestoreBookmark()
```

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.
