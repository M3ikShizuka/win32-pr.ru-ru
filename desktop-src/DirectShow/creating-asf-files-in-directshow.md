# <a name="creating-asf-files-in-directshow-directshow"></a>Создание файлов ASF в DirectShow (DirectShow)

в этом разделе описывается использование фильтра [модуля записи DirectShow WM ASF](wm-asf-writer-filter.md) для создания файлов в расширенном системном формате (ASF). ASF — это формат контейнера, который может содержать любой тип сжатых или несжатых данных. Windows файлы формата мультимедиа — это файлы ASF, которые используют определенные кодеки, как указано в Windows Media format sdk (пакет средств разработки программного обеспечения). эти файлы используют расширения wma для Windows мультимедийных файлов мультимедиа и wmv для Windows видеофайлов мультимедиа.

-   [Создание графов фильтра для записи файлов ASF](building-filter-graphs-to-write-asf-files.md)
