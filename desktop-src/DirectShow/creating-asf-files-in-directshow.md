# <a name="creating-asf-files-in-directshow-directshow"></a>Создание файлов ASF в DirectShow (DirectShow)

В этом разделе описывается, как использовать фильтр [модуля записи DirectShow WM ASF](wm-asf-writer-filter.md) для создания файлов в расширенном системном формате (ASF). ASF — это формат контейнера, который может содержать любой тип сжатых или несжатых данных. Файлы формата Windows Media — это файлы ASF, которые используют определенные кодеки, как указано в пакете средств разработки программного обеспечения (SDK) Windows Media Format. Эти файлы используют расширения WMA для звуковых файлов Windows Media и WMV для видеофайлов Windows Media.
