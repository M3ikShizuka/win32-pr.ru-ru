# <a name="avencdddchighpassfilter-property"></a>Авенкдддчигхпассфилтер, свойство

Указывает, применяется ли к потоковому потоку Dolby Digital Audio фильтр с блокированием контроллеров домена. Это свойство применяется к кодировщикам цифрового аудио Dolby.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкдддчигхпассфилтер**
