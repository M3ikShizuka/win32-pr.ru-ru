# <a name="avencvideooutputcolorlighting-property"></a>Авенквидеуутпутколорлигхтинг, свойство

Указывает предполагаемые условия освещения для просмотра кодированного видео.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеуутпутколорлигхтинг**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенквидеоколорлигхтинг**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolorlighting) .
