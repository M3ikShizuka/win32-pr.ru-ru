# <a name="avdsploudnessequalization-property"></a>Авдсплауднессекуализатион, свойство

Включает или отключает обравенство громкости в аудио-декодере или в поставщике цифровых сигналов (DSP).

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авдсплауднессекуализатион**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавдсплауднессекуализатион**](/windows/desktop/api/codecapi/ne-codecapi-eavdsploudnessequalization) .

## <a name="remarks"></a>Комментарии

Обравенство громкости — это процесс DSP, который поддерживает последовательный уровень громкости при изменении аудиопотока.
