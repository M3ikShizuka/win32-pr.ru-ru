# <a name="miscellaneous-helper-functions"></a>Различные вспомогательные функции

Ниже приведены различные вспомогательные функции.



| Функция                                                   | Описание                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**екуалпинс**](equalpins.md)                             | Проверяет, находятся ли два контакта на одном объекте.                                                                    |
| [**Int64x32Div32**](int64x32div32.md)                     | Реализует формулу, `((a*b)+rnd)/c` где `a` — это 64-разрядное значение `b` , и `c` `rnd` — 32-разрядные значения. |
| [**исекуалобжект**](isequalobject.md)                     | Проверяет, находятся ли два интерфейса в одном объекте.                                                              |
| [**ллмулдив**](llmuldiv.md)                               | Реализует формулу, `((a*b)+rnd)/c` в которой каждый термин является 64-битным значением.                                     |
| [**ваитдиспатчингмессажес**](waitdispatchingmessages.md) | Ожидает сигнала объекта, во время диспетчеризации сообщений окна.                                        |
