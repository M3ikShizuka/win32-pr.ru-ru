# <a name="returnfromsubmenu-method"></a>Метод Ретурнфромсубмену

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`ReturnFromSubmenu`Метод возвращает отображение из подменю в родительское меню или к текущему заголовку, если меню является меню верхнего уровня.

``` syntax
MSWebDVD.ReturnFromSubmenu()
```

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.
