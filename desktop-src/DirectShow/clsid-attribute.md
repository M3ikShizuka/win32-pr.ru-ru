# <a name="clsid-attribute"></a>Атрибут CLSID

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`clsid`Атрибут задает идентификатор класса (CLSID) для объекта.

## <a name="possible-values"></a>Возможные значения

Значение должно быть строкой в формате {00000000-0000-0000-0000-000000000000} .

[**в**](at-element.md), [**воздействие**](effect-element.md), [**Переход**](transition-element.md)
