# <a name="cbaseinputpincheckstreaming-method"></a>Кбасеинпутпин. Чеккстреаминг, метод

Определяет, может ли ПИН-код принимать выборки.

## <a name="syntax"></a>Синтаксис


```C++
virtual HRESULT CheckStreaming();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из значений **HRESULT** , перечисленных в следующей таблице.



| Код возврата                                                                                           | Описание                           |
|-------------------------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                  | Успешно.<br/>                   |
| <dl> <dt>**S \_ false**</dt> </dl>               | ПИН-код в данный момент очищается.<br/> |
| <dl> <dt>**\_ \_ ошибка среды выполнения VFW E \_**</dt> </dl> | Произошла ошибка времени выполнения.<br/> |
| <dl> <dt>**VFW \_ E — \_ неправильное \_ состояние**</dt> </dl>   | ПИН-код остановлен.<br/>        |



 

## <a name="remarks"></a>Комментарии

Производный класс может переопределить этот метод для выполнения дальнейших проверок. В переопределяющем методе также вызывается реализация базового класса.

Метод [**кбасеинпутпин:: Receive**](cbaseinputpin-receive.md) вызывает этот метод. Для вызова этого метода следует переопределить метод [**кбасепин:: EndOfStream**](cbasepin-endofstream.md) .
