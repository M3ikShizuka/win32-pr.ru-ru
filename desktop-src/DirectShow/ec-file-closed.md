# <a name="ec_file_closed"></a>\_файл EC \_ закрыт

Исходный файл был закрыт из-за непредвиденного события. Например, работа сетевого сервера была завершена.

Ноль.

</dd> </dl>

## <a name="default-action"></a>Действие по умолчанию

Нет.

## <a name="remarks"></a>Remarks

Устаревший фильтр источника Windows Media отправляет это событие. Новые фильтры не должны отсылать это событие.
