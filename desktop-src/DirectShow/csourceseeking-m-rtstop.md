# <a name="csourceseekingm_rtstop-member"></a>Элемент Ксаурцесикинг:: m \_ ртстоп

Время окончания. По умолчанию для этого параметра задано очень большое число. Производный класс может сбрасывать значение в его конструкторе или при инициализации фильтра.

## <a name="syntax"></a>Синтаксис


```C++
CRefTime m_rtStop;
```



## <a name="remarks"></a>Remarks

Чтобы получить доступ к этой переменной, удерживайте критический раздел **m \_ плокк** .
