# <a name="avaudiochannelcount-property"></a>Аваудиочаннелкаунт, свойство

Возвращает количество каналов в потоке звукового бита.

Это свойство доступно только для чтения.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ аваудиочаннелкаунт**

## <a name="remarks"></a>Комментарии

Число каналов включает канал с низкой частотой (НИЗКОЧАСТОТный), если он присутствует.
