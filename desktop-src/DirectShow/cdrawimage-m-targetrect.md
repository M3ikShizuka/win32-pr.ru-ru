# <a name="cdrawimagem_targetrect-member"></a>Элемент Кдравимаже:: m \_ таржетрект

`m_TargetRect`Переменная члена задает целевой прямоугольник для рисования. Начальное значение — это пустой прямоугольник, указывающий, что изображение должно отображаться во всем окне назначения.

## <a name="syntax"></a>Синтаксис


```C++
RECT m_TargetRect;
```
