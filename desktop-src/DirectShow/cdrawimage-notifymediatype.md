# <a name="cdrawimagenotifymediatype-method"></a>Кдравимаже. Нотифимедиатипе, метод

`NotifyMediaType`Метод уведомляет объект **кдравимаже** текущего типа мультимедиа.

## <a name="syntax"></a>Синтаксис


```C++
void NotifyMediaType(
   CMediaType *pMediaType
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пмедиатипе* 
</dt> <dd>

Указатель на объект [**кмедиатипе**](cmediatype.md) или **значение NULL** для очистки типа мультимедиа.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Фильтр-владелец должен вызывать этот метод при каждом изменении типа мультимедиа. Обычно это происходит, когда ПИН-код сначала соединяется и после динамического изменения формата.

Объект **кдравимаже** сохраняет указатель *пмедиатипе* в переменной элемента **m \_ пмедиатипе** . Таким образом, если вызывающему объекту нужно освободить объект **кмедиатипе** , он должен обновить объект **кдравимаже** , вызвав этот метод еще раз, либо с новым указателем, либо со значением **null** . В противном случае ошибка может возникнуть, когда объект **кдравимаже** пытается сослаться на старый указатель.
