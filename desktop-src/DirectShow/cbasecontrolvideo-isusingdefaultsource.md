# <a name="cbasecontrolvideoisusingdefaultsource-method"></a>Кбасеконтролвидео. Исусингдефаултсаурце, метод

`IsUsingDefaultSource`Метод определяет, использует ли модуль подготовки отчетов окно источника по умолчанию.

## <a name="syntax"></a>Синтаксис


```C++
virtual HRESULT IsUsingDefaultSource() = 0;
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает \_ ОК, если модуль подготовки отчетов использует источник по умолчанию; в противном случае возвращает \_ значение false.
