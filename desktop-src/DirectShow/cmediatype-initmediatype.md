# <a name="cmediatypeinitmediatype-method"></a>CMediaType.Iniметод Тмедиатипе

`InitMediaType`Метод инициализирует тип мультимедиа.

## <a name="syntax"></a>Синтаксис


```C++
void InitMediaType();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Remarks

Этот метод обнуляет память объекта, устанавливает для свойства fixed-Sample-size **значение true** и задает размер выборки равным 1.
