# <a name="wst-codec-filter"></a>Фильтр кодека WST

> [!IMPORTANT]
> этот компонент был удален из Windows Vista и более поздних операционных систем. он доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003.

 

начиная с Windows Vista этот фильтр заменяется фильтром вбикодек, который описан в документации по технологиям Microsoft TV.

Стандартный телетекст (WST) — это Европейский стандарт для передачи данных с помощью ВБИ для аналоговых телевизионных сигналов PAL. В этом стандарте предоставляются как субтитры, так и службы данных. Кодек WST — это фильтр режима ядра, который получает необработанные образцы ВБИ и, при необходимости, декодированные примеры телетекста из фильтра записи через фильтр [преобразователя tee/Sink-to-Sink](tee-sink-to-sink-converter.md) . Этот кодек декодирует и (или) дублирует некодированные и исправленные данные телетекста с ошибками для фильтра [WST](wst-decoder-filter.md) . Кодек WST соответствует фильтру декодера CC для передач NTSC. Декодер WST соответствует [строке 21 декодера](line-21-decoder-filter.md) для NTSC; эти фильтры создают точечные рисунки, которые отправляются на наложение Mixer или в модуль подготовки видео для микширования.

Этот фильтр имеет два входных ПИН-кода: ВБИ и ХВКК. ПИН-код ВБИ используется для необработанных данных ВБИ, а ПИН-код ХВКК используется в том случае, когда в оборудовании, используемом фильтром записи, выполняется декодирование ВБИ. Когда данные поступают на ХВКК ПИН-код, кодек WST работает в режиме сквозного подключения и отправляет данные непосредственно в декодер WST, не обрабатывая его каким-либо образом. Если фильтр записи предоставляет ПИН-код ХВКК, он должен быть напрямую подключен к соответствующему ПИН-коду WST.

Фильтр "WST кодек" отображается в категории фильтра "ВБИ кодеки WDM Streaming"**( \_ кскатегори \_ вбикодек**).

Так как это фильтр режима ядра, приложения не могут создать его напрямую с помощью **CoCreateInstance**. Вместо этого используйте [перечислитель системных устройств](system-device-enumerator.md). Дополнительные сведения см. в разделе [Создание фильтров Kernel-Mode](creating-kernel-mode-filters.md).
