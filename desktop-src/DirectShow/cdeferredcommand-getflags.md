# <a name="cdeferredcommandgetflags-method"></a>Кдеферредкомманд... flags, метод

`GetFlags`Метод получает флаги контекста, связанные с отложенной командой.

## <a name="syntax"></a>Синтаксис


```C++
short GetFlags();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из следующих значений.



| Код возврата                                                                                             | Описание                                                                                                                                                                    |
|---------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**\_метод Dispatch**</dt> </dl>         | Запустите член как метод. Если свойство имеет такое же имя, \_ можно задать и флаг диспетчеризации пропертижет.<br/>                                               |
| <dl> <dt>**\_ПРОПЕРТИЖЕТ диспетчеризации**</dt> </dl>    | Элемент извлекается как свойство или элемент данных.<br/>                                                                                                         |
| <dl> <dt>**\_ПРОПЕРТИПУТ диспетчеризации**</dt> </dl>    | Элемент изменяется как свойство или элемент данных.<br/>                                                                                                           |
| <dl> <dt>**\_ПРОПЕРТИПУТРЕФ диспетчеризации**</dt> </dl> | Элемент изменяется с помощью ссылочного назначения, а не назначения значения. Этот флаг допустим, только если свойство принимает ссылку на объект.<br/> |
