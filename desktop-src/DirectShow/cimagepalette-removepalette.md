# <a name="cimagepaletteremovepalette-method"></a>Цимажепалетте. Ремовепалетте, метод

`RemovePalette`Метод удаляет существующую логическую палитру и вызывает [**кбасевиндов:: унсетпалетте**](cbasewindow-unsetpalette.md) для объекта **кбасевиндов** .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT RemovePalette();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение S \_ ОК.
