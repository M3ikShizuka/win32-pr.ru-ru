# <a name="property-setter"></a>Метод задания свойства

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Объект метода задания свойств задает свойства **IDispatch** для объекта временной шкалы. Чтобы создать этот объект, вызовите **CoCreateInstance**. Идентификатор класса — CLSID \_ пропертисеттер.

Объект метода задания свойств предоставляет следующие интерфейсы:

-   [**иамсетеррорлог**](iamseterrorlog.md)
-   [**ипропертисеттер**](ipropertysetter.md)
