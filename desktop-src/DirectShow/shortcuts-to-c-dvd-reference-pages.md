# <a name="shortcuts-to-c-dvd-reference-pages"></a>Ярлыки на страницы справочника по C++ DVD

Ниже перечислены интерфейсы, которые обычно используются при написании DVD-приложения на C++.



| Интерфейс                                    | Назначение                                                                                          |
|----------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**IAMLine21Decoder**](/previous-versions/windows/desktop/api/il21dec/nn-il21dec-iamline21decoder) | Управляет отображением скрытых субтитров.                                                              |
| [**идвдкмд**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | Используется в расширенных методах синхронизации команд для управления объектом Command.                 |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | Выдает команды фильтру [DVD Navigator](dvd-navigator-filter.md) (методы "Set").     |
| [**идвдграфбуилдер**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | Создает граф фильтра DVD.                                                                    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | Запрашивает сведения о состоянии и сведения на диске в навигаторе DVD-диска (методы "Get"). |
| [**идвдстате**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | Сохраняет сведения, например закладки, на диск.                                                   |
| [**ивидеофраместеп**](/windows/desktop/api/Strmif/nn-strmif-ivideoframestep)   | Управляет пошаговым выполнением кадра, если декодер поддерживает его.                                            |
