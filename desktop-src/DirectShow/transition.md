# <a name="transition"></a>Переход

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Объект перехода создает видеопереход между дорожками или композициями. Чтобы создать этот объект, вызовите метод [**иамтимелине:: креатимптиноде**](iamtimeline-createemptynode.md) .

Объект перехода предоставляет следующие интерфейсы:

-   [**иамтимелинеобж**](iamtimelineobj.md)
-   [**иамтимелинесплиттабле**](iamtimelinesplittable.md)
-   [**иамтимелинетранс**](iamtimelinetrans.md)
