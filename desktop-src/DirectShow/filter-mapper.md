# <a name="filter-mapper"></a>Средство сопоставления фильтров

Средство сопоставления фильтров выполняет поиск зарегистрированных фильтров в реестре. фильтр Graph Manager использует этот компонент для создания графов фильтра. Приложения могут использовать его для поиска фильтров, соответствующих указанным условиям поиска. Создайте этот объект, вызвав **CoCreateInstance**.




| Метка | Применение |
|--------|-------|
| Идентификатор класса | CLSID_FilterMapper2 | 
| Интерфейсы | <ul><li><a href="/windows/desktop/api/Strmif/nn-strmif-ifiltermapper2"><strong>IFilterMapper2</strong></a></li><li><a href="iamfilterdata.md"><strong>Иамфилтердата</strong></a> (не рекомендуется)</li><li><a href="/windows/desktop/api/Strmif/nn-strmif-ifiltermapper3"><strong>IFilterMapper3</strong></a> (не рекомендуется)</li></ul> |
