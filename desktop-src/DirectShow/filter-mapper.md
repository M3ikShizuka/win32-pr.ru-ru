# <a name="filter-mapper"></a>Средство сопоставления фильтров

Средство сопоставления фильтров выполняет поиск зарегистрированных фильтров в реестре. Диспетчер графов фильтров использует этот компонент для создания графов фильтра. Приложения могут использовать его для поиска фильтров, соответствующих указанным условиям поиска. Создайте этот объект, вызвав **CoCreateInstance**.
