# <a name="overview-of-event-notification"></a>Обзор уведомления о событиях

Фильтр Уведомляет диспетчер графа фильтра о событии, публикуя уведомление о событии. Событие может быть ожидаемым, например концом потока, или сообщением об ошибке, например при сбое визуализации потока. Диспетчер графов фильтров обрабатывает некоторые события фильтра по отдельности и оставляет другим приложениям возможность обработки. Если диспетчер графов фильтров не обрабатывает событие фильтра, уведомление о событии помещается в очередь. Граф фильтра также может ставить в очередь собственные уведомления о событиях для приложения.

Приложение получает события из очереди и реагирует на них в зависимости от типа события. Уведомление о событии в DirectShow, таким образом, аналогично схеме очереди сообщений Microsoft Windows. Приложение также может отменить поведение диспетчера графа фильтра по умолчанию для данного типа событий. Затем Диспетчер графа фильтров помещает эти события непосредственно в очередь, чтобы приложение было обработано.

Этот механизм позволяет

-   Диспетчер графа фильтров для взаимодействия с приложением.
-   Фильтры для взаимодействия с приложением и диспетчером графа фильтров.
-   Приложение для определения степени участия в обработке событий.
