# <a name="coareftimeoperator-method"></a>Коарефтиме. operator/-метод

Этот оператор делит время ссылки на значение.

## <a name="syntax"></a>Синтаксис


```C++
COARefTime operator/(
   LONG l
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*l* 
</dt> <dd>

Равн.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает новый объект **коарефтиме** , равный частям данного объекта и **l**.
