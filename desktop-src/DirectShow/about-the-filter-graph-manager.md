# <a name="about-the-filter-graph-manager"></a>о диспетчере Graph фильтра

[фильтр Graph Manager](filter-graph-manager.md) — это COM-объект, управляющий фильтрами в графе фильтра. Он выполняет множество функций, включая следующие:

-   Согласование изменений состояния между фильтрами.
-   Установка ссылочного времени.
-   Передача событий обратно в приложение.
-   Предоставление методов для приложений для построения графа фильтра.

Каждая из этих функций описана вкратце. Подробные сведения можно найти в другой части документации.

**Изменения состояния**. Изменения состояния в фильтрах должны выполняться в определенном порядке. Таким образом, приложение не выполняет команды изменения состояния непосредственно в фильтры. вместо этого он предоставляет единственную команду для фильтра Graph Manager, который распространяет команду на каждый из фильтров. поиск работает аналогичным образом: приложение предоставляет команду поиска для фильтра Graph Manager, который распределяет его по фильтрам.

**Ссылочное время**. Все фильтры в графе используют одинаковые часы, называемые *ссылочным временем*. Ссылочное время гарантирует, что все потоки будут синхронизированы. Время, когда необходимо визуализировать видеокадр или образец звука, называется *временем презентации*. Время презентации измеряется относительно ссылочного времени. фильтр Graph Manager выбирает эталонный таймер, обычно либо часы звуковой карты, либо системные часы.

**события Graph**. фильтр Graph Manager использует очередь событий для информирования приложения о событиях, происходящих в графе фильтра. этот механизм аналогичен Windowsному циклу обработки сообщений.

**Graph методы создания**. фильтр Graph Manager предоставляет приложению методы для добавления фильтров к диаграмме, подключения фильтров к другим фильтрам и фильтров отключения.

> [!Note]  
> фильтры всегда находятся в свободной потоке, находятся в том же процессе, что и фильтр Graph Manager, и загружаются с внутрипроцессного сервера. таким образом, вызовы методов не маршалируются между фильтрами или между фильтрами и фильтром Graph Manager.
