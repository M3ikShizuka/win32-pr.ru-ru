# <a name="avencvideonooffieldstoskip-property"></a>Авенквидеонуффиелдстоскип, свойство

Указывает число полей для пропуска во время кодирования.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT64** (**VT \_ UI8**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеонуффиелдстоскип**

## <a name="remarks"></a>Remarks

Для последовательного видео задайте для этого свойства значение вдвое больше числа кадров для пропуска.
