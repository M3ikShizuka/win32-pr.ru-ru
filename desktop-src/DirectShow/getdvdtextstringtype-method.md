# <a name="getdvdtextstringtype-method"></a>Метод Жетдвдтекстстрингтипе

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`GetDVDTextStringType`Метод получает значение, указывающее тип сведений, содержащихся в указанной текстовой строке DVD.

``` syntax
[ iStringType = ] MSWebDVD.GetDVDTextStringType(iLangIndex, iStringIndex)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="iLangIndex"></span><span id="ilangindex"></span><span id="ILANGINDEX"></span>*илангиндекс*
</dt> <dd>

Задает язык текстовой строки в виде целого числа.

</dd> <dt>

<span id="iStringIndex"></span><span id="istringindex"></span><span id="ISTRINGINDEX"></span>*истрингиндекс*
</dt> <dd>

Задает номер индекса текстовой строки в виде целого числа.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение, указывающее тип сведений, содержащихся в строке.
