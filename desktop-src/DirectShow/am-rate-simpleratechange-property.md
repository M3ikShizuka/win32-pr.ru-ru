# <a name="am_rate_simpleratechange-property"></a>\_ \_ Свойство симплератечанже Rate

Это свойство используется для отправки изменений скорости в декодер. Типом данных для этого свойства является структура **AM \_ симплератечанже** , которая дает новый темп воспроизведения и время презентации, когда вступает в силу новая ставка.



|                   |                                                     |
|-------------------|-----------------------------------------------------|
| Идентификатор GUID набора свойств | \_Кспропсетид \_ тсратечанже                       |
| Идентификатор свойства       | \_Ставка \_ симплератечанже                          |
| Тип данных         | [**\_Симплератечанже**](/previous-versions/windows/desktop/api/Dvdmedia/ns-dvdmedia-am_simpleratechange) |
