# <a name="csourceposition-class"></a>Класс Ксаурцепоситион

`CSourcePosition` является абстрактным классом для реализации интерфейса [**имедиапоситион**](/windows/desktop/api/Control/nn-control-imediaposition) в фильтре источника.

Этот класс устарел. Если фильтр источников доступен для поиска, он должен предоставлять интерфейс [**имедиасикинг**](/windows/desktop/api/Strmif/nn-strmif-imediaseeking) вместо **имедиапоситион**. Для этой цели можно использовать базовый класс [**ксаурцесикинг**](csourceseeking.md) .
