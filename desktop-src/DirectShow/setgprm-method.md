# <a name="setgprm-method"></a>Метод Сетгпрм

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`SetGPRM`Метод задает указанное значение для регистра общих параметров.

``` syntax
MSWebDVD.SetGPRM(iIndex, nValue)
```

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="iIndex"></span><span id="iindex"></span><span id="IINDEX"></span>*ииндекс*
</dt> <dd>

Задает регистр общего параметра для задания в качестве целого числа. Целочисленное значение должно находиться в диапазоне от 0 до 15.

</dd> <dt>

<span id="nValue"></span><span id="nvalue"></span><span id="NVALUE"></span>*Nзначение*
</dt> <dd>

Задает новое значение для регистра в виде целого числа.

</dd> </dl>

## <a name="remarks"></a>Remarks

Общие регистры параметров, или Гпрмс, представляют собой 16-битные регистры, которые каждый диск может использовать уникальным образом для временного хранения данных. Приложению проигрывателя не требуется доступ к этим регистрам для любого стандартного элемента управления воспроизведением или навигации с помощью объекта **мсвебдвд** . Этот метод предоставляется для приложений проигрывателя, реализующих расширенные функциональные возможности. Не пытайтесь изменить Гпрмс напрямую, если вы не имеете исчерпывающих знаний о спецификации DVD и способы, с помощью которых Гпрмс используется на конкретном диске для воспроизведения. Изменение этих значений может привести к непредсказуемому поведению.
