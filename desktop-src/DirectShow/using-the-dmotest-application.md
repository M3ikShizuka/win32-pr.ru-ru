# <a name="using-the-dmotest-application"></a>Использование приложения Дмотест

> [!Note]  
> Приложение Дмотест больше не доступно в пакете SDK.

 

Служебная программа Дмотест может помочь проверить соответствие DMO спецификациям DMO. При разработке DMO следует включить тестовое приложение DMO в ходе тестирования.

Дмотест устанавливается в \\ каталог bin дксутилс в корневом каталоге пакета SDK. Чтобы использовать Дмотест, необходимо сначала создать тестовые данные для DMO. Пакет SDK включает фильтр DirectShow для этой цели. Сведения о фильтре и приложении Дмотест можно найти в справке по приложению Дмотест. В приложении выберите **содержимое** из меню **Справка** . Справка содержит пошаговые инструкции по созданию тестовых данных и выполнению тестов.
