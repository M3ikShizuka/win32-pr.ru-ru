> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Интерфейс `IGrfCache` не поддерживается.

Интерфейс **игрфкаче** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) , но не содержит дополнительных элементов.
