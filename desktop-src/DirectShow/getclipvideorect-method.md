# <a name="getclipvideorect-method"></a>Метод Жетклипвидеорект

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`GetClipVideoRect`Метод извлекает прямоугольник обрезки, который в настоящее время определен для экрана видео.

``` syntax
[ oClipRect = ] MSWebDVD.GetClipVideoRect()
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает объект [двдрект](dvdrect-object.md) , определяющий прямоугольник.
