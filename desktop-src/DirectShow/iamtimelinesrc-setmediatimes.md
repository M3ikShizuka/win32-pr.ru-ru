# <a name="iamtimelinesrcsetmediatimes-method"></a>Метод Иамтимелинесрк:: Сетмедиатимес

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`SetMediaTimes`Метод задает время начала и окончания воспроизведения носителя.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SetMediaTimes(
   REFERENCE_TIME Start,
   REFERENCE_TIME Stop
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Запуск* 
</dt> <dd>

Время начала носителя (в 100-наносекундных единицах).

</dd> <dt>

*Остановить* 
</dt> <dd>

Время окончания носителя в единицах с 100 нс.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если этот метод завершается успешно, возвращается значение **S \_ ОК**. В противном случае возвращается код ошибки **HRESULT** .

## <a name="remarks"></a>Комментарии

Время работы со носителями — это время начала и окончания работы относительно исходного файла носителя. Всегда устанавливайте время носителя при добавлении источника видео или звука на временную шкалу. В противном случае могут возникнуть проблемы отрисовки. Время окончания должно быть больше времени начала.

Чтобы использовать кадр по-прежнему из источника видео, задайте для параметра время окончания значение меньше времени начала, например 100 наносекунд. Установка одного и того же значения приводит к ошибке отрисовки.

Если длительность временной шкалы не соответствует длительности носителя, источник растягивается или сжимается соответствующим образом. Это приводит к тому, что клип будет воспроизводиться медленнее или быстрее, чем созданная скорость. (Смещение шага произойдет в источнике звука.) дополнительные сведения см. [в разделе время в DirectShow редактирования служб](time-in-directshow-editing-services.md).

Вы можете указать длительность исходного файла, вызвав метод [**сетмедиаленгс**](iamtimelinesrc-setmedialength.md) . Если задать время окончания воспроизведения, превышающее длительность, DES усекает время окончания.
