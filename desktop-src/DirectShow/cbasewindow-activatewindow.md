# <a name="cbasewindowactivatewindow-method"></a>Кбасевиндов. Активатевиндов, метод

`ActivateWindow`Метод изменяет размер окна в соответствии с требованиями производного класса.

## <a name="syntax"></a>Синтаксис


```C++
virtual HRESULT ActivateWindow();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из значений **HRESULT** , приведенных в следующей таблице.



| Код возврата                                                                             | Описание                              |
|-----------------------------------------------------------------------------------------|------------------------------------------|
| <dl> <dt>**S \_ false**</dt> </dl> | Окно уже активировано.<br/> |
| <dl> <dt>**\_ОК**</dt> </dl>    | Успешно.<br/>                      |



 

## <a name="remarks"></a>Комментарии

Этот метод вызывает метод [**кбасевиндов:: жетдефаултрект**](cbasewindow-getdefaultrect.md) для определения размера окна. Производный класс должен переопределять **жетдефаултрект** для получения размера отображаемых изображений.

Если окно уже активно, вызов `ActivateWindow` перемещает окно в начало Z-порядка, но не изменяет размер окна. То же самое верно, если у окна есть родительский элемент.
