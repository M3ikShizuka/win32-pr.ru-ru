# <a name="cbasepinname-method"></a>Метод CBasePin.Name

`Name`Метод получает идентификатор ПИН-кода.

## <a name="syntax"></a>Синтаксис


```C++
LPWSTR Name();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение переменной члена [**кбасепин:: m \_ pName**](cbasepin-m-pname.md) .
