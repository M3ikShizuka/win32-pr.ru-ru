# <a name="getdelaytime-method"></a>Метод Жетделайтиме

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`GetDelayTime`Метод получает время задержки для подсказки, связанной с объектом **мсвебдвд** .

``` syntax
[ iDelay = ] MSWebDVD.GetDelayTime()
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение, указывающее время задержки всплывающей подсказки в миллисекундах.
