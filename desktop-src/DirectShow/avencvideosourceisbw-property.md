# <a name="avencvideosourceisbw-property"></a>Авенквидеосаурцеисбв, свойство

Указывает, является ли видео монохромным (черно-белым) или содержит цвет.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеосаурцеисбв**

## <a name="property-value"></a>Значение свойства

Если значение является **вариантным \_ true**, видео является монохромным. В противном случае видео содержит цвет.
