# <a name="avencmpvgopsinseq-property"></a>Авенкмпвгопсинсек, свойство

Указывает количество групп изображений (группы GOP) между заголовками последовательности. Это свойство применяется к кодировщикам видео MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпвгопсинсек**

## <a name="remarks"></a>Комментарии

Это свойство доступно для чтения и записи.
