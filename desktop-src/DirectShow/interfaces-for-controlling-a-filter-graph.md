# <a name="interfaces-for-controlling-a-filter-graph"></a>Интерфейсы для управления графом фильтра

Эти интерфейсы предоставляют методы для управления графом фильтра.



| Интерфейс                                  | Описание                                                                                               |
|--------------------------------------------|-----------------------------------------------------------------------------------------------------------|
| [**иамклоккаджуст**](/windows/desktop/api/Strmif/nn-strmif-iamclockadjust)   | Настройте часы графа.                                                                                   |
| [**иамграфстреамс**](/windows/desktop/api/Strmif/nn-strmif-iamgraphstreams) | Синхронизировать динамические потоки в графе фильтра.                                                               |
| [**ифилтерчаин**](/windows/desktop/api/Strmif/nn-strmif-ifilterchain)       | Управление цепочками фильтров.                                                                                |
| [**имедиаконтрол**](/windows/desktop/api/Control/nn-control-imediacontrol)     | Выполнение, приостановка и остановка графа фильтра. (Также предоставляет совместимые с автоматизацией методы для построения диаграмм.) |
| [**имедиаевентекс**](/windows/desktop/api/Control/nn-control-imediaeventex)     | Реагирование на события в графе.                                                                           |
| [**имедиасикинг**](/windows/desktop/api/Strmif/nn-strmif-imediaseeking)     | Поиск в файле.                                                                                       |
| [**икуеуекомманд**](/windows/desktop/api/Control/nn-control-iqueuecommand)     | Команды очереди для запуска в более позднее время.                                                                    |
| [**ивидеофраместеп**](/windows/desktop/api/Strmif/nn-strmif-ivideoframestep) | Кадр — пошаговое воспроизведение видеопотока.                                                                        |
