# <a name="cimagesample-class"></a>Класс Цимажесампле

![Иерархия классов Цимажесампле](images/wutil03.png)

`CImageSample`Класс реализует пример носителя, который управляет независимым от устройства точечным рисунком GDI (DIB). Этот класс является производным от класса [**кмедиасампле**](cmediasample.md) . Он предназначен для использования с классом [**Цимажеаллокатор**](cimageallocator.md) . Класс **Цимажеаллокатор** предоставляет распределитель, создающий `CImageSample` объекты.



| Защищенные переменные членов                        | Описание                                                       |
|---------------------------------------------------|-------------------------------------------------------------------|
| [**m \_ дибдата**](cimagesample-m-dibdata.md)      | Содержит сведения о соотношении DIB, которым управляет этот объект.  |
| [**m \_ Бинит**](cimagesample-m-binit.md)          | Указывает, инициализирован ли объект.                |
| Открытые методы                                    | Описание                                                       |
| [**Цимажесампле**](cimagesample-cimagesample.md) | Метод конструктора.                                               |
| [**жетдибдата**](cimagesample-getdibdata.md)     | Извлекает сведения о соотношении DIB, которым управляет этот объект. |
| [**сетдибдата**](cimagesample-setdibdata.md)     | Задает сведения о соотношении DIB, которым управляет этот объект.      |
