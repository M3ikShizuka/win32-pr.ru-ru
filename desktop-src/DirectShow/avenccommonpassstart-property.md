# <a name="avenccommonpassstart-property"></a>Авенккоммонпассстарт, свойство

Запускает первый проход кодирования.

Это свойство доступно только на запись.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенккоммонпассстарт**

## <a name="property-value"></a>Значение свойства

Значение этого свойства — это этап начала кодирования. Чтобы получить диапазон поддерживаемых значений, запросите свойство [**авенккоммонмултипассмоде**](avenccommonmultipassmode-property.md) кодировщика.
