# <a name="property-page-helper-functions"></a>Вспомогательные функции страницы свойств

заголовочный файл видеоктл. h в базовых классах Microsoft® DirectShow® предоставляет функции, помогающие в реализации страниц свойств.



| Функция                                                 | Описание                                                                            |
|----------------------------------------------------------|----------------------------------------------------------------------------------------|
| [**жетдиалогсизе**](getdialogsize.md)                   | Извлекает размер диалогового окна ресурса в пикселях экрана.                          |
| [**стрингфромресаурце**](stringfromresource.md)         | Загружает строку из файла ресурсов с заданным идентификатором ресурса.                |
| [**видестрингфромресаурце**](widestringfromresource.md) | Загружает строку расширенных символов из файла ресурсов с заданным идентификатором ресурса. |
