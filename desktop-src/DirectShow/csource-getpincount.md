# <a name="csourcegetpincount-method"></a>Ксаурце. Жетпинкаунт, метод

`GetPinCount`Метод извлекает количество ПИН-кодов в фильтре. Этот метод реализует чистый виртуальный метод [**кбасефилтер:: жетпинкаунт**](cbasefilter-getpincount.md) .

## <a name="syntax"></a>Синтаксис


```C++
int GetPinCount();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает количество ПИН-кодов в данном фильтре.
