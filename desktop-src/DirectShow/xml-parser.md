# <a name="xml-parser"></a>Синтаксический анализатор XML

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Объект XML Parser преобразуется между файлами проекта XML и временными шкалами. Чтобы создать этот объект, вызовите **CoCreateInstance**.

Объект синтаксического анализа XML предоставляет следующий интерфейс:

-   [**IXml2Dex**](ixml2dex.md)

**Идентификатор класса:** \_XML2DEX CLSID
