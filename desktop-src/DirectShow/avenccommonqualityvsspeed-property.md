# <a name="avenccommonqualityvsspeed-property"></a>Авенккоммонкуалитивсспид, свойство

Задает компромисс между качеством и скоростью кодирования. Это свойство допустимо во всех режимах управления скоростью.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенккоммонкуалитивсспид**

## <a name="property-value"></a>Значение свойства

Значение этого свойства имеет следующий диапазон.



| Значение | Описание:                      |
|-------|----------------------------------|
| 0     | Более низкое качество, более быстрая кодировка.  |
| 100   | Более высокое качество, медленная кодировка. |
