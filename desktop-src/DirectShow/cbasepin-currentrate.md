# <a name="cbasepincurrentrate-method"></a>Кбасепин. Куррентрате, метод

`CurrentRate`Метод получает частоту сегментов, заданную методом [**кбасепин:: невсегмент**](cbasepin-newsegment.md) .

## <a name="syntax"></a>Синтаксис


```C++
double CurrentRate();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение [**кбасепин:: m \_ драте**](cbasepin-m-drate.md).
