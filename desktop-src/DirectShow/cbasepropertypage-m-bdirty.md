# <a name="cbasepropertypagem_bdirty-member"></a>Элемент Кбасепропертипаже:: m \_ бдирти

Флаг **m \_ бдирти** указывает, изменились ли какие либо свойства. Установите для этого флага **значение true** , если пользователь изменяет свойства через диалоговые элементы управления.

## <a name="syntax"></a>Синтаксис


```C++
BOOL m_bDirty;
```
