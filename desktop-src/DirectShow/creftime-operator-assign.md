# <a name="creftimeoperator-method-reftimeh"></a>Крефтиме. operator =-метод (Рефтиме. h)

## <a name="syntax"></a>Синтаксис


```C++
CRefTime& operator=(
  [ref] const CRefTime &rt
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*RT* \[ ЗНАЧ\]
</dt> <dd>

Ссылка на объект **крефтиме** , указывающий новое время ссылки.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение
