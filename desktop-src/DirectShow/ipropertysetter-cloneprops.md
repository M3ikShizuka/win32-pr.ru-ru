# <a name="ipropertysettercloneprops-method"></a>Метод Ипропертисеттер:: Клонепропс

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`CloneProps`Метод копирует набор свойств из этого метода задания свойств и добавляет их в новый метод задания свойств.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT CloneProps(
  [out] IPropertySetter **ppSetter,
  [in]  REFERENCE_TIME  rtStart,
  [in]  REFERENCE_TIME  rtStop
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ппсеттер* \[ заполняет\]
</dt> <dd>

Получает указатель на интерфейс **ипропертисеттер** нового метода задания свойства.

</dd> <dt>

*ртстарт* \[ окне\]
</dt> <dd>

Время начала диапазона значений для клонирования в единицах измерения 100-наносекундных.

</dd> <dt>

*ртстоп* \[ окне\]
</dt> <dd>

Зарезервировано.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если этот метод завершается успешно, возвращается значение **S \_ ОК**. В противном случае возвращается код ошибки **HRESULT** .

## <a name="remarks"></a>Комментарии

Клонированием будут только значения, попадающие после указанного времени начала. Время на клонированных значениях затем корректируется относительно времени начала. Например, если *ртстарт* равен 20000000 (2 секунды), то значение в момент времени 30000000 (3 секунды) клонируется с временем 10000000 (1 секунда). Наконец, каждому клонированному свойству присваивается начальное значение, равное значению исходного свойства в момент начала (при необходимости в правильной интерполяции). Фактически данные свойства разбиваются на указанное время начала.

Если метод завершается успешно, интерфейс [**ипропертисеттер**](ipropertysetter.md) , который он возвращает, имеет необработанный счетчик ссылок. Не забудьте освободить интерфейс по завершении его использования.
