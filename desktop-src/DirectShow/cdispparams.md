# <a name="cdispparams-class"></a>Класс Кдисппарамс

![Иерархия классов кдисппарамс](images/cutil11.png)

`CDispParams`Класс реализует структуру **DISPPARAMS** , используемую в автоматизации в качестве базового класса C++. Метод **IDispatch:: Invoke** использует структуру OLE **DISPPARAMS** для хранения аргументов, передаваемых в любой метод или свойство.

|------------------------------------------------|--------------------------------------|
| [**кдисппарамс**](cdispparams-cdispparams.md) | Конструирует объект **кдисппарамс** . |
