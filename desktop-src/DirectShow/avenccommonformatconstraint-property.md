# <a name="avenccommonformatconstraint-property"></a>Авенккоммонформатконстраинт, свойство

Задает формат целевого объекта для кодировщика.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**BSTR** (**VT \_ BSTR**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенккоммонформатконстраинт**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является строкой **BSTR** , содержащей строковое представление идентификатора GUID. Определены следующие идентификаторы GUID.



| Идентификатор GUID                                         | Описание                     |
|----------------------------------------------|---------------------------------|
| КОДЕКАПИ \_ GUID \_ авенккоммонформататск        | Кабель ATSC.          |
| КОДЕКАПИ \_ GUID \_ авенккоммонформатдвб         | Кабель DVB.           |
| КОДЕКАПИ \_ GUID \_ авенккоммонформатдвд \_ дашвр | ДИСК DVD-VR                          |
| КОДЕКАПИ \_ GUID \_ авенккоммонформатдвд \_ плусвр | DVD + VR                          |
| КОДЕКАПИ \_ GUID \_ авенккоммонформатдвд \_ V      | DVD-Video                       |
| КОДЕКАПИ \_ GUID \_ авенккоммонформасигхмат     | HighMAT                         |
| КОДЕКАПИ \_ GUID \_ авенккоммонформасигхмпв     | Не документировано в этом выпуске. |
| КОДЕКАПИ \_ GUID \_ AVEncCommonFormatMP3         | Аудио-слой MPEG — 3 (MP3)        |
| КОДЕКАПИ \_ GUID \_ авенккоммонформатсвкд        | Компакт-диск Super Video (СВКД)           |
| КОДЕКАПИ \_ GUID \_ авенккоммонформатунспеЦифиед | Неуказанный формат.             |
| КОДЕКАПИ \_ GUID \_ авенккоммонформатвкд         | Компакт-диск с видео (VCD)                  |



 

## <a name="remarks"></a>Remarks

Приложения могут задать для этого свойства значение, определяющее целевой формат. Кодировщики также могут возвращать это свойство как возможность.
