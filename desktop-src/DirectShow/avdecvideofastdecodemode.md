# <a name="avdecvideofastdecodemode-property"></a>Авдеквидеофастдекодемоде, свойство

Возвращает или задает скорость декодирования видео.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авдеквидеофастдекодемоде**

## <a name="property-value"></a>Значение свойства

Значение этого свойства может находиться в диапазоне от 0 до 32, где 0 означает нормальную декодирование, а 32 — самый быстрый режим декодирования. Точное поведение зависит от реализации декодера. Не все значения в диапазоне от 1 до 32 обязательно определяются в разных режимах.

Перечисление [**еавфастдекодемоде**](/windows/desktop/api/codecapi/ne-codecapi-eavfastdecodemode) содержит некоторые предопределенные режимы для этого свойства.
