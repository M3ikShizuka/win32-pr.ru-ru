# <a name="multimedia-streaming-data-types"></a>Типы данных потоковой передачи мультимедиа

> [!Note]  
> Это нерекомендуемый API. Новые приложения не должны использовать его.

 

Интерфейсы потоковой передачи мультимедиа Microsoft DirectShow используют следующие типы данных.



| Тип данных                           | Описание                                            |
|-------------------------------------|--------------------------------------------------------|
| [**мспид**](mspid.md)              | Определяет назначение потока мультимедиа.              |
| [**\_время потока**](stream-time.md) | Определяет тип данных, используемый для выражения времени ссылок. |
