# <a name="cdeferredcommandisstreamtime-method"></a>Кдеферредкомманд. Исстреамтиме, метод

`IsStreamTime`Метод указывает, должна ли команда выполняться во время потока или во время презентации.

## <a name="syntax"></a>Синтаксис


```C++
BOOL IsStreamTime();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если задано потоковое время. в противном случае возвращает **значение false**.
