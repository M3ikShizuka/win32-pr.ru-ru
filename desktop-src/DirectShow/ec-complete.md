# <a name="ec_complete"></a>EC \_ завершено

Все данные из определенного потока были подготовлены к просмотру.

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

(**HRESULT**) Состояние потока по завершении. Если во время воспроизведения ошибок не возникло, значение равно S \_ .

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

(**IUnknown** \* ) Ноль или указатель на интерфейс [**ибасефилтер**](/windows/desktop/api/Strmif/nn-strmif-ibasefilter) визуализатора.

</dd> </dl>

## <a name="default-action"></a>Действие по умолчанию

По умолчанию диспетчер графов фильтров не перенаправляет это событие в приложение. Однако после того, как все потоки в отчете Graph **EC \_ завершены**, диспетчер графов фильтров публикует в приложении отдельное событие **EC \_ Complete** .

Если для этого события отключено действие по умолчанию, приложение получает все события **EC \_ Complete** из модулей подготовки отчетов.

## <a name="remarks"></a>Remarks

Фильтр модуля подготовки отчетов отправляет это событие при получении уведомления конца потока. (В конце потока сообщается метод [**Ипин:: EndOfStream**](/windows/desktop/api/Strmif/nf-strmif-ipin-endofstream) .) Фильтр отправляет ровно одно событие **\_ завершения EC** для каждого потока. Перед отправкой события фильтр должен обработать все ожидающие выборки. Остановка модуля подготовки отчетов сбрасывает все кэшированное состояние конца потока.

Если модуль подготовки отчетов приостановлен, он не отправляет **EC \_ Complete** до тех пор, пока не будет вызван метод [**имедиафилтер:: Run**](/windows/desktop/api/Strmif/nf-strmif-imediafilter-run) . Более того, он по-своему отправляет события **EC \_ Complete** для каждого перехода от паузы до запуска, пока фильтр не будет остановлен или сброшен.

Фильтры присвойте параметру *lParam2* указатель [**ибасефилтер**](/windows/desktop/api/Strmif/nn-strmif-ibasefilter) . Если включено действие по умолчанию, диспетчер графа фильтра устанавливает этот параметр в ноль.
