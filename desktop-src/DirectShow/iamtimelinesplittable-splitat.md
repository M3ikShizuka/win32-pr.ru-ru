# <a name="iamtimelinesplittablesplitat-method"></a>Метод Иамтимелинесплиттабле:: Сплитат

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`SplitAt`Метод разделяет объект в указанное время.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT SplitAt(
   REFERENCE_TIME Time
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Time* 
</dt> <dd>

Время разделения объекта относительно начала временной шкалы в единицах с частотой 100 нс.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** . Возможные значения:



| Код возврата                                                                                   | Описание                                        |
|-----------------------------------------------------------------------------------------------|----------------------------------------------------|
| <dl> <dt>**S \_ false**</dt> </dl>       | Нет для разбиения.<br/>                       |
| <dl> <dt>**\_ОК**</dt> </dl>          | Успешно.<br/>                                |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>  | В данный момент объект не существует.<br/> |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти.<br/>                    |



 

## <a name="remarks"></a>Комментарии

При разделении источника, воздействия или перехода этот метод создает второй объект того же типа. Исходный объект усекается в указанное время разбиения, а новый объект заменяет обрезанную часть. Новый объект наследует все те же свойства. В исходном объекте метод также разделяет любые эффекты, которые попадают в время разбиения.

Вызов этого метода в дорожке разделяет все источники, эффекты и переходы, содержащиеся в дорожке, в указанное время разбиения. Вторая запись не создается. (Запись начинается в момент нулевого значения и расширяется на бесконечность.)

Для трассировки, если время разбиения позже, чем все в дорожке, метод возвращает \_ значение false. Для любого другого объекта, если время разбиения не попадает в объект, метод возвращает E \_ INVALIDARG.
