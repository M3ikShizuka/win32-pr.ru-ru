# <a name="dbgdumpobjectregister-function"></a>Функция Дбгдумпобжектрегистер

`DbgDumpObjectRegister`Функция отображает сведения об активных объектах. Игнорируется в розничных сборках.

## <a name="syntax"></a>Синтаксис


```C++
void DbgDumpObjectRegister(void);
```



## <a name="parameters"></a>Параметры

У этой функции нет параметров.

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

В отладочных сборках библиотека отладки поддерживает список активных объектов. Список включает все объекты, производные от [**кбасеобжект**](cbaseobject.md), которые были созданы текущим модулем и не были уничтожены. Конструктор **кбасеобжект** и методы деструктора обновляют список.

Эта функция создает несколько \_ сообщений в памяти журнала. При уровне ведения журнала 1 функция отображает только общее число объектов. При уровне ведения журнала 2 или более поздней версии отображается список объектов.
