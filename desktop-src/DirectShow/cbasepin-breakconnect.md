# <a name="cbasepinbreakconnect-method"></a>Кбасепин. Бреакконнект, метод

`BreakConnect`Метод освобождает ПИН-код из соединения.

## <a name="syntax"></a>Синтаксис


```C++
virtual HRESULT BreakConnect();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение S \_ ОК.

## <a name="remarks"></a>Комментарии

Этот метод вызывается при отключении ПИН-кода методом [**кбасепин::D метода соединения**](cbasepin-disconnect.md) . Он также вызывается во время попытки соединения, если метод [**кбасепин:: чеккконнект**](cbasepin-checkconnect.md) завершается с ошибкой.

Этот метод должен освобождать все ресурсы, полученные методом **чеккконнект** . Например, если **чеккконнект** выделяет память, `BreakConnect` освободите память. Если **чеккконнект** запрашивает соединительный ПИН-код для интерфейса, `BreakConnect` должен освободить интерфейс.

Обратите внимание, что `BreakConnect` может быть вызван без соответствующего вызова **комплетеконнект**. Поэтому нельзя предположить, что **комплетеконнект** был вызван ранее.
