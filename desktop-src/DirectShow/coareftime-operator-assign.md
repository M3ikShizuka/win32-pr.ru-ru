# <a name="coareftimeoperator-method-ctlutilh"></a>Коарефтиме. operator =-метод (Ктлутил. h)

Этот оператор назначает новое время ссылки.

## <a name="syntax"></a>Синтаксис


```C++
COARefTime& operator=(
  [ref] const double &rd
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*RD* \[ ЗНАЧ\]
</dt> <dd>

Ссылка на значение **Double** , указывающее новое время ссылки в секундах.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ссылку на объект.
