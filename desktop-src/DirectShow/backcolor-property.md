# <a name="backcolor-property-directshow"></a>Свойство BackColor (DirectShow)

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`BackColor`Свойство задает или получает цвет полос, отображаемых вокруг краев прямоугольника видео, если пропорции изображения в машинном коде не совпадают с областью отображения объекта.

``` syntax
[ iBackColor = ] MSWebDVD.BackColor
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение, представляющее RGB значения цвета фона.

## <a name="remarks"></a>Remarks

Это свойство доступно для чтения и записи со значением по умолчанию "Выкл-черный" (0x100010).
