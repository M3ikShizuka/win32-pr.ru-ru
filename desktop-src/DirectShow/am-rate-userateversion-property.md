# <a name="am_rate_userateversion-property"></a>\_ \_ Свойство усератеверсион Rate

Это свойство используется для сигнализации о версии свойства изменения скорости, которую должен использовать декодер. Значение является типом **слова** . Байт высокого порядка содержит дополнительный номер версии, а младший байт содержит младший байт. Таким же значением версии 1,1 будет сигнальное значение 0x0101.

Если декодер не поддерживает указанную версию, он должен вызвать сбой вызова [**икспропертисет:: Set**](ikspropertyset-set.md) и Return E без \_ интерфейса. Если в фильтре источника не задана версия, декодеру следует по умолчанию установить версию 1,0.



|                   |                               |
|-------------------|-------------------------------|
| Идентификатор GUID набора свойств | \_Кспропсетид \_ тсратечанже |
| Идентификатор свойства       | \_Ставка \_ усератеверсион      |
| Тип данных         | **WORD**                      |
