# <a name="cbaseallocatorm_fenablereleasecallback-member"></a>Элемент Кбасеаллокатор:: m \_ фенаблерелеасекаллбакк

## <a name="syntax"></a>Синтаксис


```C++
BOOL m_fEnableReleaseCallback;
```
