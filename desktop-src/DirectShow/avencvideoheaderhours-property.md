# <a name="avencvideoheaderhours-property"></a>Авенквидеохеадерхаурс, свойство

Задает номер часа начала в заголовке Group of Pictures (GOP).

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеохеадерхаурс**

## <a name="property-value"></a>Значение свойства

Допустимый диапазон значений этого свойства — от 0 до 23 включительно.
