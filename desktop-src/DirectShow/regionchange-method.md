# <a name="regionchange-method"></a>Метод Региончанже

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`RegionChange`Метод отображает системное диалоговое окно, позволяющее пользователю изменить регион, связанный с DVD-дисководом.

``` syntax
MSWebDVD.RegionChange()
```

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Remarks

Регион для DVD-дисковода можно изменить только пять раз.
