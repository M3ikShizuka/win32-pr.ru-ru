# <a name="avencddsurroundexmode-property"></a>Авенкддсурраундексмоде, свойство

Указывает, кодируется ли поток цифрового звука Dolby в цифровом окружении Dolby, например. Это свойство применяется к кодировщикам цифрового аудио Dolby.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкддсурраундексмоде**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенкддсурраундексмоде**](/windows/desktop/api/codecapi/ne-codecapi-eavencddsurroundexmode) .

## <a name="remarks"></a>Комментарии

Это свойство доступно для чтения и записи.
