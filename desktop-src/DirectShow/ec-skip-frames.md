# <a name="ec_skip_frames"></a>EC \_ пропустить \_ кадры

Отправляется навигатором DVD, когда необходимо пропускать кадры.

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

Число пропускаемых кадров.

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Зарезервировано.

</dd> </dl>

## <a name="default-action"></a>Действие по умолчанию

Это событие не отправляется в приложение. Приложения не могут переопределить действие по умолчанию для этого события.
