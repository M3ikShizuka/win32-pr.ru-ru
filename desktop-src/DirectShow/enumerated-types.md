> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Службы редактирования DirectShow определяют следующие перечислимые типы.



| Перечисление                                                         | Описание                                                                  |
|---------------------------------------------------------------------|------------------------------------------------------------------------------|
| [**\_ \_ Флаги поиска декстерф Track \_**](dexterf-track-search-flags.md) | Задает ограничивающие условия для поиска объекта на временной шкале. |
| [**\_основной \_ тип временной шкалы**](timeline-major-type.md)                | Указывает основной тип объекта временной шкалы.                               |
