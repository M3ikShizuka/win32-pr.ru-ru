# <a name="displaytype-function"></a>Функция Дисплайтипе

`DisplayType`Функция отправляет сведения о типе мультимедиа в расположение выходных данных отладки. Игнорируется в розничных сборках.

## <a name="syntax"></a>Синтаксис


```C++
void DisplayType(
         LPTSTR        label,
   const AM_MEDIA_TYPE *pmtIn
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*label* 
</dt> <dd>

Строка, содержащая сообщение, отображаемое со сведениями о типе носителя.

</dd> <dt>

*пмтин* 
</dt> <dd>

оинтер к структуре [**\_ \_ типа мультимедиа AM**](/windows/win32/api/strmif/ns-strmif-am_media_type) , которая содержит тип носителя.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Эта функция не возвращает значение.

## <a name="remarks"></a>Комментарии

Эта функция создает несколько \_ сообщений трассировки журнала. При уровне ведения журнала 2 или выше функция отображает основной тип, подтип и тип формата, а также данные из блока формата. При уровне ведения журнала 5 или выше функция отображает дополнительные сведения, такие как исходный и целевой прямоугольники для типов видео.
