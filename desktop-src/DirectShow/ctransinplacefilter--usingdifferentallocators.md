# <a name="ctransinplacefilterusingdifferentallocators-method"></a>Ктрансинплацефилтер. Усингдифференталлокаторс, метод

`UsingDifferentAllocators`Метод определяет, используют ли контакты ввода и вывода различные распределителя.

## <a name="syntax"></a>Синтаксис


```C++
BOOL UsingDifferentAllocators() const;
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если крепления ввода и вывода используют разные распределителя, или **false** в противном случае.
