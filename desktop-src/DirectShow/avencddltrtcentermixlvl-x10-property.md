# <a name="avencddltrtcentermixlvl_x10-property"></a>Авенкддлтртцентермикслвл \_ X10, свойство

Задает сдвиг уровня, применяемый к центральному каналу для lt/RT довнмиксинг. Это свойство применяется к кодировщикам цифрового аудио Dolby.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкддлтртцентермикслвл \_ x10**

## <a name="property-value"></a>Значение свойства

Значение указывается как децибел (dB) x 10.

## <a name="remarks"></a>Комментарии

Это свойство доступно для чтения и записи.
