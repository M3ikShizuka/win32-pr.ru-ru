# <a name="imedialocator-interface"></a>Интерфейс Имедиалокатор

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`IMediaLocator`интерфейс предоставляет методы для проверки имен файлов в [службах DirectShow editing Services](directshow-editing-services.md) (DES). Используйте этот интерфейс, чтобы убедиться, что заданное имя файла и путь соответствуют существующему файлу. Этот интерфейс также позволяет искать файл в других местах и отображать **открытое** диалоговое окно, чтобы пользователь мог найти файл.

Указатель мультимедиа реализует этот интерфейс. Временная шкала и модуль визуализации также поддерживают проверку имени файла с помощью следующих методов:

-   [**Иамтимелине:: валидатесаурценамес**](iamtimeline-validatesourcenames.md): проверяет и обновляет имена файлов на временной шкале.
-   [**Ирендеренгине:: сетсаурценамевалидатион**](irenderengine-setsourcenamevalidation.md): указывает, как подсистема подготовки отчетов проверяет имена файлов во время отрисовки.

Как правило, приложение DES вызывает эти методы, а не создает экземпляр указателя мультимедиа напрямую. Дополнительные сведения см. [в разделе Использование указателя мультимедиа](using-the-media-locator.md).

|:-----------------------------------------------------------|:-----------------------------------------------------------------------------------|
