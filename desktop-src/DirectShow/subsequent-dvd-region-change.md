# <a name="subsequent-dvd-region-change"></a>Изменение следующего региона DVD

в Windows 2000 и более поздних версий навигатор dvd вызывает страницу свойств устройства dvd-rom в диспетчер устройств. Эта страница свойств также выдается приложением DVDPlay.exe, когда необходимо изменить регион. Пользовательский интерфейс этой страницы свойств очень похож на DVDRgn.exe и является региональным в операционной системе.

для дисков RPC1 Windows компоненты операционной системы управляют изменениями в регионах. RPC2 диски поддерживают настройку региона на своем оборудовании. Если количество допустимых изменений на RPC2 диске исчерпано, диск не сможет вызвать изменение региона, а компонент изменения региона в операционной системе сообщит пользователю.
