# <a name="cbaserendererhavecurrentsample-method"></a>Кбасерендерер. Хавекуррентсампле, метод

`HaveCurrentSample`Метод определяет, имеет ли фильтр образец.

## <a name="syntax"></a>Синтаксис


```C++
virtual BOOL HaveCurrentSample();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если фильтр имеет образец, или **false** в противном случае.
