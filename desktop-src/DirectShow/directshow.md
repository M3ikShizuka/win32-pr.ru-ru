# <a name="directshow"></a>DirectShow

Интерфейс прикладного программирования (API) Microsoft DirectShow — это архитектура потоковой передачи мультимедиа для Microsoft Windows. С помощью DirectShow ваши приложения могут выполнять высококачественное воспроизведение видео и звука или захват.

Заголовки, библиотеки, средства SDK и образцы DirectShow доступны в [Windows SDK](https://msdn.microsoft.com/windows/aa904949.aspx).

> [!Note]  
> Предыдущие версии пакета SDK для DirectShow были добавлены в пакет SDK для DirectX. Последняя версия пакета SDK DirectX для включения DirectShow — это дополнение DirectX 9,0 SDK Update-(февраль 2005). После этой версии DirectShow была перемещена в Windows SDK. Чтобы получить последнюю версию заголовков, библиотек и образцов DirectShow, Скачайте Windows SDK.
