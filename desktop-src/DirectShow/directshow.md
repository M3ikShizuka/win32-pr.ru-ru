# <a name="directshow"></a>DirectShow

интерфейс прикладного программирования (API) microsoft DirectShow — это архитектура потоковой передачи мультимедиа для Microsoft Windows. используя DirectShow, ваши приложения могут выполнять высококачественное воспроизведение видео и звука или захват.

в [Windows SDK](https://msdn.microsoft.com/windows/aa904949.aspx)доступны заголовки DirectShow, библиотеки, средства SDK и примеры.

> [!Note]  
> предыдущие версии пакета sdk для DirectShow были добавлены в пакет sdk для DirectX. последняя версия пакета sdk directx для включения DirectShow была дополнена directx 9,0 sdk Update-(февраль 2005). после этой версии DirectShow был перемещен в Windows SDK. чтобы получить последнюю версию DirectShow заголовков, библиотек и примеров, скачайте Windows SDK.
