# <a name="composition"></a>Композиция

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Объект композиции — это контейнер для дорожек. Он также может содержать другие композиции (для вложения дорожек), эффекты и переходы. Чтобы создать этот объект, вызовите метод [**иамтимелине:: креатимптиноде**](iamtimeline-createemptynode.md) .

Объект композиции предоставляет следующие интерфейсы:

-   [**иамтимелинекомп**](iamtimelinecomp.md)
-   [**иамтимелиниффектабле**](iamtimelineeffectable.md)
-   [**иамтимелинеобж**](iamtimelineobj.md)
-   [**иамтимелинетрансабле**](iamtimelinetransable.md)
-   [**иамтимелиневиртуалтракк**](iamtimelinevirtualtrack.md)
