# <a name="group-directshow"></a>Группа (DirectShow)

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Объект Group представляет композицию верхнего уровня. Каждая группа определяет один видеофайл или звуковой поток в окончательном проекте. Чтобы создать этот объект, вызовите метод [**иамтимелине:: креатимптиноде**](iamtimeline-createemptynode.md) .

Объект Group предоставляет следующие интерфейсы:

-   [**иамтимелинекомп**](iamtimelinecomp.md)
-   [**иамтимелинеграуп**](iamtimelinegroup.md)
-   [**иамтимелинеобж**](iamtimelineobj.md)
