# <a name="cdeferredcommandgetiid-method"></a>Кдеферредкомманд. Жетиид, метод

`GetIID`Метод получает идентификатор интерфейса (IID) интерфейса, на котором будет выполняться метод.

## <a name="syntax"></a>Синтаксис


```C++
REFIID GetIID();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает идентификатор IID интерфейса, на котором будет выполняться метод.
