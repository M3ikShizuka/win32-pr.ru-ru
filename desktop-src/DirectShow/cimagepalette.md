# <a name="cimagepalette-class"></a>Класс Цимажепалетте

`CImagePalette`Класс управляет палитрами для модулей подготовки видео. Его можно использовать для создания логической палитры из формата видео. Этот класс предназначен для использования с классами [**кбасевиндов**](cbasewindow.md) и [**кдравимаже**](cdrawimage.md) , поэтому он является довольно специализированным.



| Защищенные переменные членов                                       | Описание                                                                                    |
|------------------------------------------------------------------|------------------------------------------------------------------------------------------------|
| [**m \_ хпалетте**](cimagepalette-m-hpalette.md)                  | Маркер логической палитры, которой управляет этот объект.                                        |
| [**m \_ пбасевиндов**](cimagepalette-m-pbasewindow.md)            | Указатель на объект **кбасевиндов** , управляющий окном.                                 |
| [**m \_ пдравимаже**](cimagepalette-m-pdrawimage.md)              | Указатель на объект **кдравимаже** , который рисует изображение видео.                               |
| [**m \_ пфилтер**](cimagepalette-m-pfilter.md)                    | Указатель на фильтр-владелец.                                                                  |
| Открытые методы                                                   | Описание                                                                                    |
| [**Цимажепалетте**](cimagepalette-cimagepalette.md)             | Метод конструктора.                                                                            |
| [**копипалетте**](cimagepalette-copypalette.md)                 | Копирует палитру из любой структуры **видеоинфо** в любую структуру палеттизед **видеоинфо** . |
| [**макеидентитипалетте**](cimagepalette-makeidentitypalette.md) | Пытается сделать палитру, которая сопоставляется непосредственно с палитрой, выбранной в устройстве вывода.   |
| [**макепалетте**](cimagepalette-makepalette.md)                 | Создает логическую палитру из таблицы цветов в видеоформате.                              |
| [**препарепалетте**](cimagepalette-preparepalette.md)           | Настраивает палитру на основе типа носителя из фильтра-владельца.                               |
| [**ремовепалетте**](cimagepalette-removepalette.md)             | Удаляет существующую логическую палитру.                                                          |
