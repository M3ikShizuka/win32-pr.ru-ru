# <a name="cbasedispatchgettypeinfo-method"></a>Кбаседиспатч. GetTypeInfo, метод

`GetTypeInfo`Метод получает сведения о типе для объекта, который затем можно использовать для получения сведений о типе для интерфейса.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetTypeInfo(
   REFIID    riid,
   UINT      itinfo,
   LCID      lcid,
   ITypeInfo **pptinfo
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*riid* 
</dt> <dd>

Ссылка на идентификатор интерфейса (IID), указывающий интерфейс.

</dd> <dt>

*итинфо* 
</dt> <dd>

Возвращаемые сведения о типе. Должен равняться нулю.

</dd> <dt>

*lcid* 
</dt> <dd>

Идентификатор языкового стандарта.

</dd> <dt>

*пптинфо* 
</dt> <dd>

Адрес переменной, получающей указатель **ITypeInfo** .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** . Ниже приведены возможные значения.



| Код возврата                                                                                             | Описание                                    |
|---------------------------------------------------------------------------------------------------------|------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                    | Успешно.<br/>                            |
| <dl> <dt>**\_указатель E**</dt> </dl>               | **Пустой** аргумент указателя.<br/>          |
| <dl> <dt>**Введите \_ E \_ елементнотфаунд**</dt> </dl> | Параметр *итинфо* не равен нулю.<br/> |



 

## <a name="remarks"></a>Remarks

Этот метод ведет себя подобно методу **IDispatch:: GetTypeInfo** . Однако он включает дополнительный параметр *riid*, указывающий интерфейс, для которого требуется получить сведения о типе.
