# <a name="name-attribute"></a>Атрибут name

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`name`Атрибут задает имя объекта.

## <a name="possible-values"></a>Возможные значения

Может быть любым значением.

## <a name="applies-to"></a>Применяется к

[**Группа**](group-element.md), [ **параметр**](param-element.md)

## <a name="remarks"></a>Remarks

Для элемента **Group** этот атрибут указывает определенное приложением имя для группы.

Для элемента **param** этот атрибут указывает имя свойства.
