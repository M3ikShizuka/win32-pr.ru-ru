# <a name="windowlessactivation-property"></a>Виндовлессактиватион, свойство

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`WindowlessActivation`Свойство Инициализирует объект **мсвебдвд** во время разработки для оконного или безоконного режима.

``` syntax
[ bWindowless = ] MSWebDVD.WindowlessActivation
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение, указывающее, находится ли объект в оконном режиме в виде логического значения.

## <a name="remarks"></a>Remarks

Это свойство доступно для чтения и записи со значением по умолчанию true. Поэтому это свойство необходимо задавать, только если объект **мсвебдвд** выполняется в оконном контейнере. При его наличии на веб-странице в Microsoft® Internet Explorer, **мсвебдвд** всегда находится в режиме без окон, и вам не нужно задавать это свойство.
