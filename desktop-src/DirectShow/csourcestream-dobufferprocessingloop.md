# <a name="csourcestreamdobufferprocessingloop-method"></a>Ксаурцестреам. Добуфферпроцессинглуп, метод

`DoBufferProcessingLoop`Метод создает данные мультимедиа и доставляет их в последующий входной ПИН-код.

## <a name="syntax"></a>Синтаксис


```C++
virtual HRESULT DoBufferProcessingLoop();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** . Возможные значения включают в себя, показанные в следующей таблице.



| Код возврата                                                                             | Описание                                                             |
|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------|
| <dl> <dt>**S \_ false**</dt> </dl> | Поток получил запрос на завершение.<br/>                              |
| <dl> <dt>**\_ОК**</dt> </dl>    | Поток завершился, или нисходящий фильтр не принимает выборки.<br/> |



 

## <a name="remarks"></a>Remarks

Этот метод реализует главный цикл, который обрабатывает данные и доставляет их нисходящим. Каждый раз через цикл метод получает пустой пример носителя из распределителя. Он передает пример методу [**ксаурцестреам:: филлбуффер**](csourcestream-fillbuffer.md) . Метод **филлбуффер** , который должен реализовываться производным классом, создает данные мультимедиа и помещает их в буфер выборки.

Цикл завершается, когда происходит одно из следующих событий:

-   Метод [**имеминпутпин:: Receive**](/windows/desktop/api/Strmif/nf-strmif-imeminputpin-receive) входного контакта отклоняет выборку.
-   Метод **филлбуффер** возвращает \_ значение false, указывающее конец потока, или возвращает код ошибки.
-   Поток получает запрос [**ксаурцестреам:: останавливаться**](csourcestream-stop.md) .

`DoBufferProcessingLoop`Метод обрабатывает уведомление о завершении потока. При возникновении ошибки оно отправляет событие [**EC \_ еррораборт**](ec-errorabort.md) в Диспетчер графа фильтров.
