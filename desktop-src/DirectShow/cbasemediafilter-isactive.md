# <a name="cbasemediafilterisactive-method"></a>Кбасемедиафилтер. on, метод

`IsActive`Метод определяет, является ли объект активным (запущен или приостановлен).

## <a name="syntax"></a>Синтаксис


```C++
BOOL IsActive();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если объект приостановлен или запущен, или **значение false** , если он остановлен.
