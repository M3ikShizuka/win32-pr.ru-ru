# <a name="cbaserenderergetrenderevent-method"></a>Кбасерендерер. Жетрендеревент, метод

`GetRenderEvent`Метод получает событие, которое планирует подготовку к просмотру.

## <a name="syntax"></a>Синтаксис


```C++
CAMEvent* GetRenderEvent();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает указатель на событие [**кбасерендерер:: m \_ рендеревент**](cbaserenderer-m-renderevent.md) .
