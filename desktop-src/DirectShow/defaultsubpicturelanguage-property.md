# <a name="defaultsubpicturelanguage-property"></a>Дефаултсубпиктурелангуаже, свойство

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`DefaultSubpictureLanguage`Свойство получает язык подизображения по умолчанию.

``` syntax
[ iLang = ] MSWebDVD.DefaultSubpictureLanguage
```

## <a name="return-value"></a>Возвращаемое значение

Ретуррн значение LCID, содержащее основной идентификатор языка для аудио языка по умолчанию.

## <a name="remarks"></a>Комментарии

Это свойство доступно только для чтения и не имеет значения по умолчанию.
