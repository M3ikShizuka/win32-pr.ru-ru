# <a name="cimagedisplay-class"></a>Класс Цимажедисплай

![Цимажедисплайкласшиерарчи](images/wutil06.png)

`CImageDisplay`Класс является вспомогательным классом для модулей подготовки видео GDI для управления форматом отображения. Объект хранит структуру [**видеоинфо**](/previous-versions/windows/desktop/api/amvideo/ns-amvideo-videoinfo) , описывающую текущий режим просмотра, который инициализируется в методе конструктора объекта. Метод **чеккмедиатипе** объекта проверяет, можно ли эффективно визуализировать предложенный тип мультимедиа с помощью GDI.
