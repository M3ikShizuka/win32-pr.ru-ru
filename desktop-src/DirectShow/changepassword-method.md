# <a name="changepassword-method"></a>Метод ChangePassword

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`DVDAdm.ChangePassword`Метод сохраняет новый пароль приложения в реестре.

``` syntax
        DVD.DVDAdm.ChangePassword(sUserName, sOld, sNew)
```

Нет возвращаемого значения.

## <a name="remarks"></a>Комментарии

В настоящее время параметр *сусернаме* игнорируется на этом и всех связанных методах. Это означает, что кто угодно знает, что пароль может установить родительский уровень. Для приложения существует только один пароль и один родительский уровень. Не поддерживается отдельное имя пользователя для входа в систему или управление паролями. Чтобы применить уровни родительского управления, родительские элементы должны задать пароль, а затем задать уровень родительского уровня, соответствующий младшим членам группы родственников. Когда родители хотят просмотреть диск с содержимым для взрослых, он может изменить уровень, а затем снова изменить его после завершения просмотра. Пока дочерние элементы не узнают пароль, они могут просматривать только содержимое на уровне, установленном для них.
