# <a name="avencmpalayer-property"></a>Авенкмпалайер, свойство

Задает звуковой слой MPEG. Это свойство применяется к кодировщикам аудио в формате MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпалайер**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенкмпалайер**](/windows/win32/api/codecapi/ne-codecapi-eavencmpalayer) .
