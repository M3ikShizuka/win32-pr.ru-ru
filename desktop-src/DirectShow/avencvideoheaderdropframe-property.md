# <a name="avencvideoheaderdropframe-property"></a>Авенквидеохеадердропфраме, свойство

Задает значение флага удаления кадра в заголовке группы рисунков (GOP).

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеохеадердропфраме**

## <a name="property-value"></a>Значение свойства

Значение этого свойства может быть равно 0 или 1.

## <a name="remarks"></a>Комментарии

Режим перетаскивания кадров используется в видео NTSC для исправления расхождения между видео, которое составляет 29,97 кадров в секунду, и отображение кода времени — 30 кадров в секунду. В режиме переноса кадров номера кадров 00 и 01 пропускаются в начале каждой минуты, за исключением минут, которые еще кратны десяти (00, 10, 20 и т. д.). Удаляются только номера кадров в коде времени; фактические кадры не удаляются.
