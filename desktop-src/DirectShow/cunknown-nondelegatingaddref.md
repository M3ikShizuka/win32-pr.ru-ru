# <a name="cunknownnondelegatingaddref-method"></a>Кункновн. Нонделегатингаддреф, метод

`NonDelegatingAddRef`Метод увеличивает значение счетчика ссылок на объект. Этот метод реализует метод **инонделегатингункновн:: нонделегатингаддреф** .

## <a name="syntax"></a>Синтаксис


```C++
ULONG NonDelegatingAddRef();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает счетчик ссылок.
