# <a name="ksproperty_extxport_timecode_search"></a>Поиск по времени КСПРОПЕРТИ \_ ексткспорт \_ \_

Это свойство отправляет на устройство команду для поиска указанного кода времени. Драйвер устройства УВК поддерживает это свойство.



| Метка | Значение |
|-------------------|----------------------------------------|
| Идентификатор GUID набора свойств | ПРОПСЕТИД \_ ext \_ Transport              |
| Идентификатор свойства       | Поиск по времени КСПРОПЕРТИ \_ ексткспорт \_ \_ |
| Тип данных         | **Кспроперти \_ Структура \_ ексткспорт**  |



 

## <a name="remarks"></a>Remarks

Заполните **элемент времени в** структуре **кспроперти \_ ексткспорт \_ S** с нужным кадром, вторым, минутным и часовым. структура **кспроперти \_ ексткспорт \_** описана в пакете DDK по Windows.
