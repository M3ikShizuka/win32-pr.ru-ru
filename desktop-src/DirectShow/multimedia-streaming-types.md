В этом разделе перечислены типы перечислений, определенные для мультимедиа Microsoft DirectShow.



| Тип перечисления                      | Описание                                      |
|---------------------------------------|--------------------------------------------------|
| [**\_состояние потока**](/previous-versions/windows/desktop/api/mmstream/ne-mmstream-stream_state) | Описывает состояние потока.                 |
| [**\_тип потока**](/previous-versions/windows/desktop/api/mmstream/ne-mmstream-stream_type)   | Определяет направление потока данных для потока. |
