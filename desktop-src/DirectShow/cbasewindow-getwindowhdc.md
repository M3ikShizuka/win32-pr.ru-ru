# <a name="cbasewindowgetwindowhdc-method"></a>Кбасевиндов. Жетвиндовхдк, метод

`GetWindowHDC`Метод получает маркер для контекста устройства (DC) окна.

## <a name="syntax"></a>Синтаксис


```C++
HDC GetWindowHDC();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает маркер для контроллера домена.
