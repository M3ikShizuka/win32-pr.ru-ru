# <a name="avencddsurround3dbattenuation-property"></a>AVEncDDSurround3dBAttenuation, свойство

Указывает, размещаются ли объемные каналы в потоке Dolby Digital Audio в 3 децибел (dB) перед кодированием. Это свойство применяется к кодировщикам цифрового аудио Dolby.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ AVEncDDSurround3dBAttenuation**
