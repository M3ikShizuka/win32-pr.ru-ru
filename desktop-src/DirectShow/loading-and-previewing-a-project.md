# <a name="loading-and-previewing-a-project"></a>Загрузка и предварительный просмотр Project

\[Этот API не поддерживается и может быть изменен или недоступен в будущем.\]

в этом разделе описывается загрузка и предварительный просмотр файла проекта с помощью [служб DirectShow editing Services](directshow-editing-services.md) (DES). В нем представлен пример приложения, которое загружает файл XML-проекта в пустую временную шкалу и готовит к просмотру временную шкалу. Вы можете изменить пример приложения для работы с любым примером XML-файла проекта, который поставляется с этим пакетом SDK.
