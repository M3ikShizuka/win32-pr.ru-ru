# <a name="reference_time"></a>\_время ссылки

**Ссылочный \_** тип данных определяет единицы времени ссылки в DirectShow. Каждая единица времени ссылки составляет 100 наносекунд.


```C++
typedef LONGLONG REFERENCE_TIME;
```



## <a name="remarks"></a>Комментарии

Тип данных **ссылочного \_ времени** — это 64-разрядное целое число.

Время ссылки обычно измеряется в одном из следующих базовых показателей:

-   Абсолютное время. В этом случае базовый уровень будет зависеть от реализации часов. Дополнительные сведения см. в разделе [эталонные часы](reference-clocks.md).
-   Относительно начала воспроизведения.

Дополнительные сведения о времени ссылки см. [в разделе время и часы в DirectShow](time-and-clocks-in-directshow.md).
