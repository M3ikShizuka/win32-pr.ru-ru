# <a name="avencddservice-property"></a>Авенкддсервице, свойство

Указывает службу аудио, которая содержится в потоке цифрового звука Dolby. Это свойство применяется к кодировщикам цифрового аудио Dolby.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкддсервице**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенкддсервице**](/windows/desktop/api/codecapi/ne-codecapi-eavencddservice) .

## <a name="remarks"></a>Комментарии

Это свойство доступно для чтения и записи.
