# <a name="avencstataudiopeakpcmvalue-property"></a>Авенкстатаудиопеакпкмвалуе, свойство

Возвращает наивысший уровень громкости, имеющийся в аудио содержимом.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкстатаудиопеакпкмвалуе**

## <a name="remarks"></a>Комментарии

Это свойство доступно после завершения кодирования.

Это свойство применимо только к кодированию с переменной скоростью с переменным качеством (VBR)
