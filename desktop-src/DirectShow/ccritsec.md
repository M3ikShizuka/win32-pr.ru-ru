# <a name="ccritsec-class"></a>Класс Ккритсек

Класс **ккритсек** обеспечивает блокировку потока.

этот класс является тонкой оболочкой для Windows **критического объекта \_ раздела** . Вы можете заблокировать и разблокировать поток, вызвав методы [**ккритсек:: Lock**](ccritsec-lock.md) и [**Ккритсек:: Unlock**](ccritsec-unlock.md) . Однако безопаснее использовать этот класс в сочетании с классом [**каутолокк**](cautolock.md) . Когда класс **каутолокк** выходит из области действия, он автоматически разблокирует объект **ккритсек** . Более того, он компилируется в эффективный встроенный код.



| Открытые переменные члена                            | Описание                                              |
|----------------------------------------------------|----------------------------------------------------------|
| [**m \_ куррентовнер**](ccritsec-m-currentowner.md) | Идентификатор потока владеющего потока.                  |
| [**m \_ локккаунт**](ccritsec-m-lockcount.md)       | Количество невыполненных блокировок для этого объекта.              |
| [**m \_ фтраце**](ccritsec-m-ftrace.md)             | Логическое значение, указывающее, следует ли отслеживать эту блокировку. |
| Открытые методы                                     | Описание                                              |
| [**ккритсек**](ccritsec-ccritsec.md)              | Метод конструктора.                                      |
| [**~ Ккритсек**](ccritsec--ccritsec.md)            | Метод деструктора.                                       |
| [**Блокировка**](ccritsec-lock.md)                      | Блокирует объект критической секции.                       |
| [**Блокирован**](ccritsec-unlock.md)                  | Разблокирует объект критической секции.                     |
