# <a name="cbasicaudio-class"></a>Класс Кбасикаудио

![Иерархия классов кбасикаудио](images/cutil07.png)

`CBasicAudio`Класс реализует методы **IDispatch** интерфейса [**ибасикаудио**](/windows/desktop/api/Control/nn-control-ibasicaudio) . Оставшиеся методы **ибасикаудио** оставлены чисто виртуальными методами и должны быть реализованы производным классом.

Методы **IDispatch** в этом классе являются стандартными реализациями и подробно описаны здесь.
