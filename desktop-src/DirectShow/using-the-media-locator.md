# <a name="using-the-media-locator"></a>Использование указателя мультимедиа

\[Этот API не поддерживается и может быть изменен или недоступен в будущем.\]

Указатель мультимедиа — это вспомогательный объект, который проверяет имена файлов и ищет недостающие файлы в локальных или сетевых каталогах. Средство обнаружения мультимедиа сохраняет кэш путей к каталогам, в которых файлы успешно найдены в прошлых поисках. Чтобы найти файл, он выполняет поиск в каталогах в кэше. В этом случае средство обнаружения мультимедиа может отобразить диалоговое окно открытия файла для того, чтобы пользователь мог выбрать файл вручную. Предполагая, что пользователь найдет файл, указатель носителя добавляет новый каталог в свой кэш. Указатель мультимедиа предоставляет интерфейс [**имедиалокатор**](imedialocator.md) .

Как правило, приложение не создает экземпляр указателя мультимедиа напрямую. Вместо этого временная шкала и подсистема визуализации предоставляют следующие методы проверки имен файлов с помощью средства обнаружения мультимедиа.

-   Чтобы проверить имена файлов на временной шкале, вызовите метод [**иамтимелине:: валидатесаурценамес**](iamtimeline-validatesourcenames.md). При необходимости этот метод также обновляет исходные объекты с использованием правильных имен файлов.
-   Чтобы проверить имена файлов при подготовке проекта к просмотру, вызовите метод [**ирендеренгине:: сетсаурценамевалидатион**](irenderengine-setsourcenamevalidation.md).

Оба метода принимают флаги, управляющие поведением указателя мультимедиа. Например, можно ограничить поиск локальными каталогами.
