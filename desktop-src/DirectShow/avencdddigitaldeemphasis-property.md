# <a name="avencdddigitaldeemphasis-property"></a>Авенкдддигиталдимфасис, свойство

Указывает, включено ли Цифровое выделение в потоке цифрового звука Dolby. Это свойство применяется к кодировщикам цифрового аудио Dolby.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкдддигиталдимфасис**
