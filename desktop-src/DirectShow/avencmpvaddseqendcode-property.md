# <a name="avencmpvaddseqendcode-property"></a>Авенкмпваддсекендкоде, свойство

Указывает, добавляет ли кодировщик конец последовательности в конец потока. Это свойство применяется к кодировщикам видео MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпваддсекендкоде**

## <a name="remarks"></a>Remarks

Если значение равно **\_ true**, кодировщик добавляет код конца последовательности в конец потока.
