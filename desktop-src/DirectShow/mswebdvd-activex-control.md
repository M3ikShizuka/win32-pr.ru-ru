# <a name="mswebdvd-activex-control"></a>Элемент управления ActiveX Мсвебдвд

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

> [!Note]  
> Этот компонент является устаревшим. Сведения о воспроизведении DVD-дисков и навигации в DirectShow см. в разделе [DVD-приложения](dvd-applications.md).

 

Элемент управления® ActiveX Microsoft® Мсвебдвд можно использовать для создания приложения DVD, основанного на сценариях. Он предоставляет приложению следующие объекты и события.

-   [мсвебдвд](mswebdvd-object.md)
-   [мсдвдадм](msdvdadm-object.md)
-   [двдрект](dvdrect-object.md)
-   [События Мсвебдвд](mswebdvd-events.md)
