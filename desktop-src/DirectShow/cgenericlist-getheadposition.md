# <a name="cgenericlistgetheadposition-method"></a>Кженериклист. Жесеадпоситион, метод

`GetHeadPosition`Метод получает позицию первого элемента в списке.

## <a name="syntax"></a>Синтаксис


```C++
POSITION GetHeadPosition();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение расположения или **значение NULL** , если список пуст.
