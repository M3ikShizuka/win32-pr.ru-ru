# <a name="avdecddoperationalmode-property"></a>Авдекддоператионалмоде, свойство

Задает режим управления сжатием для звукового потока Dolby AC-3.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авдекддоператионалмоде**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавдекддоператионалмоде**](/windows/desktop/api/codecapi/ne-codecapi-eavdecddoperationalmode) .
