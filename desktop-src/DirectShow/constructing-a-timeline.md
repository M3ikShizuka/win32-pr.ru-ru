# <a name="constructing-a-timeline"></a>Создание временной шкалы

\[Этот API не поддерживается и может быть изменен или недоступен в будущем.\]

в этой статье описывается создание временной шкалы в [службах DirectShow editing Services](directshow-editing-services.md) (DES). В нем представлен пример консольного приложения, которое создает временную шкалу и готовит ее к просмотру. Временная шкала является минимальной, состоящей из одной видеогруппы с одним исходным клипом, но она демонстрирует большинство концепций, необходимых для создания более сложных временных шкал.

-   [Настройка типа носителя группы](setting-the-group-media-type.md)
-   [Добавление источника](adding-a-source.md)
-   [Создание временной шкалы: пример кода](creating-a-timeline--example-code.md)
