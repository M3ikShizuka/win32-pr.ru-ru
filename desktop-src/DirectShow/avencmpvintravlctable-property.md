# <a name="avencmpvintravlctable-property"></a>Авенкмпвинтравлктабле, свойство

Указывает, какую таблицу с кодировкой переменной длины (Влк) следует использовать для написания кода энтропии. Это свойство применяется к кодировщикам видео MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпвинтравлктабле**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавенкмпвинтравлктабле**](/windows/desktop/api/codecapi/ne-codecapi-eavencmpvintravlctable) .
