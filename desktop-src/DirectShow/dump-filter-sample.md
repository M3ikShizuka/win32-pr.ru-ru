# <a name="dump-filter-sample"></a>Образец фильтра дампа

## <a name="description"></a>Описание

Фильтр дампа — это фильтр модуля подготовки отчетов, который записывает полученные примеры мультимедиа в текстовый файл.

В этом примере показано, как использовать базовый класс фильтра [**кбасефилтер**](cbasefilter.md) и отображаемый класс входных ПИН-кодов [**крендерединпутпин**](crenderedinputpin.md). В нем также показано, как реализовать интерфейс [**ифилесинкфилтер**](/windows/desktop/api/Strmif/nn-strmif-ifilesinkfilter) . Фильтр дампа имеет один входной ПИН-код, который записывает каждый пример, который он получает непосредственно в файл.

## <a name="usage"></a>Использование

Этот фильтр является полезным средством отладки. Например, можно проверить, поразрядить результаты фильтра преобразования. Граф можно построить вручную с помощью Графедит и подключить фильтр дампа к выходным данным фильтра преобразования или любого другого выходного контакта. Можно также подключить фильтр tee и разместить фильтр дампа в одной части фильтра tee, а типичные выходные данные — в другом, чтобы отслеживать результаты в сценарии в режиме реального времени.
