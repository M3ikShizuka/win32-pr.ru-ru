# <a name="cbaserenderergetsampletimes-method"></a>Кбасерендерер. Жетсамплетимес, метод

`GetSampleTimes`Метод извлекает метки времени из образца.

## <a name="syntax"></a>Синтаксис


```C++
virtual HRESULT GetSampleTimes(
   IMediaSample   *pMediaSample,
   REFERENCE_TIME *pStartTime,
   REFERENCE_TIME *pEndTime
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пмедиасампле* 
</dt> <dd>

Указатель на интерфейс [**имедиасампле**](/windows/desktop/api/Strmif/nn-strmif-imediasample) образца.

</dd> <dt>

*пстарттиме* 
</dt> <dd>

Указатель на переменную, которая получает время начала.

</dd> <dt>

*пендтиме* 
</dt> <dd>

Указатель на переменную, которая получает время окончания.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** . Возможные значения включают в себя, показанные в следующей таблице.



| Код возврата                                                                                                    | Описание                                                                        |
|----------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                           | Пример должен быть визуализирован немедленно.<br/>                              |
| <dl> <dt>**S \_ false**</dt> </dl>                        | Пример должен быть запланирован для подготовки к просмотру на основе меток времени.<br/> |
| <dl> <dt>**\_Ошибка E**</dt> </dl>                         | Не выводите этот пример.<br/>                                              |
| <dl> <dt>**время начала работы с VFW \_ E \_ \_ \_ после \_ окончания**</dt> </dl> | Неправильная отметка времени: время окончания предшествует времени начала.<br/>            |



 

## <a name="remarks"></a>Комментарии

Фильтр вызывает этот метод, чтобы определить, как он должен работать с примером. Если возвращаемое значение равно S \_ , то фильтр немедленно выводит пример. Если возвращаемое значение равно S \_ , то фильтр планирует подготовку к просмотру на основе меток времени. Если возвращаемое значение является кодом ошибки, то фильтр отклоняет выборку.

Этот метод возвращает значение \_ ОК, если в образце нет штампов времени или если фильтр не имеет контрольного времени. В противном случае возвращается значение метода [**кбасерендерер:: шаулддравсампленов**](cbaserenderer-shoulddrawsamplenow.md) . В базовом классе **шаулддравсампленов** всегда возвращает значение S \_ false. Производный класс может переопределить это поведение. Например, если производный класс реализует управление качеством, он может вернуть «E \_ не удалось удалить образец».
