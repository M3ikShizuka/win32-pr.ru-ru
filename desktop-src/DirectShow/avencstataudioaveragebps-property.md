# <a name="avencstataudioaveragebps-property"></a>Авенкстатаудиоаверажебпс, свойство

Возвращает среднее число бит в секунду закодированного звука.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкстатаудиоаверажебпс**

## <a name="remarks"></a>Remarks

Это свойство доступно после завершения кодирования.

Это свойство применимо только к кодированию с переменной скоростью с переменным качеством (VBR)
