# <a name="coareftime-class"></a>Класс Коарефтиме

![Иерархия классов коарефтиме](images/cutil05.png)

`COARefTime`Класс преобразует время ссылки между секундами и 100-наносекундных единицами.

Этот класс преобразует время между ссылками, совместимыми со службой автоматизации, и временем ссылки, совместимыми с C/C++. Совместимые с автоматизацией интерфейсы используют значения **Double** для представления времени в секундах. Другие интерфейсы используют 64-битные значения **лонглонг** для представления времени в единицах 100-наносекундных. Для этих значений определены следующие типы:

``` syntax
typedef LONGLONG  REFERENCE_TIME;
typedef double    REFTIME;
```

Фильтры могут использовать `COARefTime` класс для преобразования между двумя форматами. Этот класс является производным от класса [**крефтиме**](creftime.md) .
