# <a name="transitions-and-effects"></a>Переходы и эффекты

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

В этом разделе описываются эффекты и переходы, предоставляемые службами редактирования DirectShow.



| Переход                              | Описание                                                                            |
|-----------------------------------------|----------------------------------------------------------------------------------------|
| [Compositor](compositor-transition.md) | Растягивает передний план до указанного прямоугольника, не изменяя фон. |
| [Ключ](key-transition.md)               | Выполняет ввод ключей на основе чрома, альфа-тона, оттенка или светимости.                             |
| [Очистка SMPTE](smpte-wipe-transition.md) | Выполняет стандартные очистки SMPTE.                                                     |



 



| Действие                                        | Описание                           |
|-----------------------------------------------|---------------------------------------|
| [Альфа-метод задания](alpha-setter-effect.md)       | Задает альфа-биты на видеоизображении. |
| [Конверт тома](volume-envelope-effect.md) | Задает том в потоке аудио.   |



 



| Исходный объект                                | Описание                                        |
|----------------------------------------------|----------------------------------------------------|
| [Источник цвета видео](video-color-source.md) | Создает непрерывное видеоизображение сплошного цвета. |
