# <a name="avencvideofieldswap-property"></a>Авенквидеофиелдсвап, свойство

Меняет порядок чередующихся полей в исходном видео.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенквидеофиелдсвап**

## <a name="remarks"></a>Remarks

Если значение является **вариантным \_ true**, кодировщик меняет порядок полей.
