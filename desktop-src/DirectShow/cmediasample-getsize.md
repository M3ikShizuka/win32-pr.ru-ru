# <a name="cmediasamplegetsize-method"></a>Кмедиасампле. DataSize, метод

`GetSize`Метод получает размер буфера. Этот метод реализует метод [**имедиасампле:: resize**](/windows/desktop/api/Strmif/nf-strmif-imediasample-getsize) .

## <a name="syntax"></a>Синтаксис


```C++
LONG GetSize();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает размер буфера в байтах.
