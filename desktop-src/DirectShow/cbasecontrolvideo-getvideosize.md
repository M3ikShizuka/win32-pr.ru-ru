# <a name="cbasecontrolvideogetvideosize-method"></a>Кбасеконтролвидео. Жетвидеосизе, метод

`GetVideoSize`Метод извлекает ширину и высоту собственного видео.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT GetVideoSize(
   long *pWidth,
   long *pHeight
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пвидс* 
</dt> <dd>

Указатель на ширину видео.

</dd> <dt>

*феигхт* 
</dt> <dd>

Указатель на высоту видео.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** .
