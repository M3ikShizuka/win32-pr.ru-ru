# <a name="enumerating-objects-in-a-filter-graph"></a>Перечисление объектов в фильтре Graph

Приложению может потребоваться выбрать определенный фильтр в графе фильтра или даже определенный ПИН-код для фильтра. Например, он может использовать интерфейс, предоставляемый определенным фильтром. Кроме того, он может создать специализированный граф фильтра и для подключения фильтров необходимо вызвать методы на отдельных контактах. для этой цели DirectShow предоставляет несколько методов перечисления объектов в графе фильтра.

Перечислители, обсуждаемые в этом разделе, следуют стандартной форме, используемой интерфейсами перечисления COM. Дополнительные сведения см. в разделе "Иенумкскскскс" в пакете Platform SDK. Сведения о перечислении фильтров, зарегистрированных на компьютере пользователя, но еще не находящимся на графе фильтра, см. в разделе [Перечисление устройств и фильтров](enumerating-devices-and-filters.md).

Эта статья содержит следующие разделы:

-   [Перечисление фильтров](enumerating-filters.md)
-   [Перечисление ПИН-кодов](enumerating-pins.md)
-   [Перечисление типов мультимедиа](enumerating-media-types.md)
