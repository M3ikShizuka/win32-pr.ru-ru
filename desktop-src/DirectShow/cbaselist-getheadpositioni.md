# <a name="cbaselistgetheadpositioni-method"></a>Кбаселист. Жесеадпоситиони, метод

`GetHeadPositionI`Метод получает позицию первого элемента в списке.

## <a name="syntax"></a>Синтаксис


```C++
POSITION GetHeadPositionI();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение расположения или **значение NULL** , если список пуст.
