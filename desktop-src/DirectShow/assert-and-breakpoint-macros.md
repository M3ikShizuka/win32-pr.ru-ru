# <a name="assert-and-breakpoint-macros"></a>Макросы Assert и точка останова

[DirectShow базовые классы](directshow-base-classes.md) предоставляют несколько макросов, выполняющих утверждения или вызывающие точки останова.



| Макрос                                        | Описание                                                                                                                        |
|----------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**УТВЕРЖДАЮЩЕ**](assert.md)                     | Вычисляет выражение и отображает диагностическое сообщение, если выражение имеет **значение false**.                                         |
| [**дбгассерталигнед**](dbgassertaligned.md) | Проверяет, соответствует ли указатель заданной границе.                                                                        |
| [**дбгбреак**](dbgbreak.md)                 | Отображает окно сообщения с указанной строкой, именем исходного файла и номером строки.                                |
| [**ВЫПОЛНИТЬ \_ Assert**](execute-assert.md)    | Вычисляет выражение в сборках Debug и Retail. В отладочных сборках отображает диагностическое сообщение, если выражение имеет **значение false**. |
| [**кассерт**](kassert.md)                   | Вычисляет выражение и вызывает исключение точки останова, если выражение имеет **значение false**.                                         |
| [**кдбгбреак**](kdbgbreak.md)               | Вызывает исключение точки останова и записывает в журнал указанную строку.                                                                      |
