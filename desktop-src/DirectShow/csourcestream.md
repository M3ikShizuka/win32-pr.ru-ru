# <a name="csourcestream-class"></a>Класс Ксаурцестреам

![Иерархия классов ксаурцестреам](images/source02.png)

Класс **ксаурцестреам** предоставляет выходной ПИН-код для класса фильтра [**ксаурце**](csource.md) .

Сведения об использовании этого класса см. в разделе [**ксаурце**](csource.md). Этот класс наследует класс [**камсреад**](camthread.md) , который предоставляет рабочий поток для потоковой передачи данных из ПИН-кода. Класс **ксаурцестреам** реализует следующие вспомогательные методы для отправки запросов в поток:

-   [**Ксаурцестреам:: Exit**](csourcestream-exit.md)
-   [**Ксаурцестреам:: init**](csourcestream-init.md)
-   [**Ксаурцестреам::P Аусе**](csourcestream-pause.md)
-   [**Ксаурцестреам:: Run**](csourcestream-run.md)
-   [**Ксаурцестреам:: останавливаться**](csourcestream-stop.md)

Первый запрос к потоку должен иметь значение [**init**](csourcestream-init.md). Запрос на [**выход**](csourcestream-exit.md) завершает поток. На практике необязательно вызывать ни один из этих методов напрямую, так как методы закрепления [**ксаурцестреам:: Active**](csourcestream-active.md) и [**Ксаурцестреам:: Inactive**](csourcestream-inactive.md) вызывают их по мере необходимости.

Класс также предоставляет несколько методов "handler":

-   [**Ксаурцестреам:: Онсреадкреате**](csourcestream-onthreadcreate.md)
-   [**Ксаурцестреам:: Онсреаддестрой**](csourcestream-onthreaddestroy.md)
-   [**Ксаурцестреам:: Онсреадстартплай**](csourcestream-onthreadstartplay.md)

Они не выполняют никаких действий в базовом классе, но производный класс может переопределять их.
