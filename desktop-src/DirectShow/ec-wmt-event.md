# <a name="ec_wmt_event-dshowh"></a>EC_WMT_EVENT (DShow. h)

Отправляется фильтром [модуля чтения WM ASF](wm-asf-reader-filter.md) при чтении файлов ASF, защищенных с помощью управления цифровыми правами (DRM).

## <a name="parameters"></a>Параметры

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

Одно из следующих значений **\_ состояния ВМТ** :



| Значение                         | Описание                                                                                                       |
|-------------------------------|-------------------------------------------------------------------------------------------------------------------|
| ВМТ \_ получить \_ лицензию         | Посылается, когда компонент DRM завершил процесс приобретения лицензии успешно или неудачно. |
| ВМТ \_ индивидуализируйте            | Процесс обновления безопасности завершен успешно или неудачно.                                |
| ВМТ \_ нуждается в \_ индивидуализации | Для выполнения запрошенного действия в файле необходимо, чтобы приложение получало обновление системы безопасности.          |
| ВМТ \_ без \_ прав               | Приложение не имеет прав на выполнение запрошенного действия с файлом, защищенным DRM версии 1.                |
| ВМТ \_ без \_ прав \_           | Приложение не имеет прав на выполнение запрошенного действия с файлом, защищенным DRM версии 7.                |



 

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Указатель на структуру [**\_ \_ \_ данных события ВМТ**](/previous-versions/windows/desktop/api/evcode/ns-evcode-am_wmt_event_data) , которая содержит сведения о событии, или **значение NULL**. Элемент **pData** этой структуры указывает на дополнительные данные, тип которых зависит от значения **lParam1**, как показано в следующей таблице.



| lParam1                       | \_ВМТ \_ события \_ Data. pdata                                                                                                                       |
|-------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| ВМТ \_ получить \_ лицензию         | Указатель на структуру [**\_ \_ \_ данных лицензии WM Get**](/windows/desktop/wmformat/wm-get-license-data) . Эта структура описана в пакете SDK для формата Windows Media. |
| ВМТ \_ индивидуализируйте            | Указатель на структуру [**\_ \_ состояния WM индивидуализируйте**](/windows/desktop/wmformat/wm-individualize-status) .                                                        |
| ВМТ \_ нуждается в \_ индивидуализации | **Значение NULL**.                                                                                                                                        |
| ВМТ \_ без \_ прав               | Указатель на строку расширенных символов, содержащую URL-адрес запроса.                                                                                   |
| ВМТ \_ без \_ прав \_           | Указатель на структуру [**\_ \_ \_ данных лицензии WM Get**](/windows/desktop/wmformat/wm-get-license-data) .                                                               |



 

Значение *lParam2* может быть **равно NULL**. Проверьте значение перед разыменованием указателя.

</dd> </dl>
