# <a name="ipropertysetter-interface"></a>Интерфейс Ипропертисеттер

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`IPropertySetter`Интерфейс задает свойства влияния или перехода в [службах редактирования DirectShow](directshow-editing-services.md) (DES).

Чтобы использовать этот интерфейс, создайте экземпляр объекта метода задания свойств (CLSID \_ пропертисеттер) и свяжите его с результатом или переходом, вызвав метод [**иамтимелинеобж:: сетпропертисеттер**](iamtimelineobj-setpropertysetter.md) . Дополнительные сведения см. в разделе [Работа с эффектами и переходами](working-with-effects-and-transitions.md).

Обычно приложению необходимо вызвать метод [**ипропертисеттер:: клеарпропс**](ipropertysetter-clearprops.md) , чтобы очистить существующие свойства, и метод [**Ипропертисеттер:: аддпроп**](ipropertysetter-addprop.md) для добавления новых свойств. Другие методы этого интерфейса вызываются другими компонентами DES.

## <a name="members"></a>Элементы

Интерфейс **ипропертисеттер** наследует от интерфейса [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **Ипропертисеттер** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ипропертисеттер** содержит следующие методы.



| Метод                                               | Описание                                                                                                                                   |
|:-----------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------|
| [**аддпроп**](ipropertysetter-addprop.md)           | Добавляет свойство в метод задания свойства с массивом пар «время-значение», определяющим значение свойства за определенный период времени.<br/> |
| [**клеарпропс**](ipropertysetter-clearprops.md)     | Удаляет все данные свойств из метода задания свойств.<br/>                                                                                 |
| [**клонепропс**](ipropertysetter-cloneprops.md)     | Копирует набор свойств из этого метода задания свойств и добавляет их в новый метод задания свойств.<br/>                                       |
| [**фрипропс**](ipropertysetter-freeprops.md)       | Освобождает ресурсы, выделенные методом [**ипропертисеттер:: "PROPS**](ipropertysetter-getprops.md) ".<br/>                             |
| [**Props**](ipropertysetter-getprops.md)         | Извлекает свойства, заданные для этого объекта, с соответствующими значениями.<br/>                                                      |
| [**лоадфромблоб**](ipropertysetter-loadfromblob.md) | Загружает данные свойств из формата сохраняемости.<br/>                                                                                     |
| [**LoadXML**](ipropertysetter-loadxml.md)           | Загружает данные свойства, выраженные в язык XML (XML).<br/>                                                                 |
| [**принтксмл**](ipropertysetter-printxml.md)         | Преобразует данные свойств в XML-строку.<br/>                                                                                         |
| [**саветоблоб**](ipropertysetter-savetoblob.md)     | Сохраняет данные свойства в формате сохраняемости.<br/>                                                                                   |
| [**сетпропс**](ipropertysetter-setprops.md)         | Задает для свойств целевого объекта соответствующее состояние в течение указанного времени.<br/>                                          |
