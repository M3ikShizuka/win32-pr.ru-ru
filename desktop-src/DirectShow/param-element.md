# <a name="param-element"></a>Param, элемент

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`param`Элемент указывает значение свойства для перехода, воздействия или другого подобъекта.

## <a name="attributes"></a>Атрибуты

[**имя**](name-attribute.md), [ **значение**](value-attribute.md)

## <a name="parentchild-information"></a>Сведения о родителе и дочернем элементе



|          |                                                                                                          |
|----------|----------------------------------------------------------------------------------------------------------|
| Parent   | [**клип**](clip-element.md), [**результат**](effect-element.md), [**Переход**](transition-element.md) |
| Дети | [**в**](at-element.md), [ **Линейная**](linear-element.md)                                               |



 

## <a name="remarks"></a>Комментарии

Атрибут **value** задает значение свойства в начале перехода или воздействия. Для указания изменяющихся значений используйте элемент **at** или **линейный** . Если элемент **param** **не содержит ни одного** или **линейного** элемента, значение остается постоянным в течение действия или перехода.

> [!Note]  
> Элемент **param** внутри элемента **Clip** не **может содержать** **линейные** элементы или.

 

Многие переходы поддерживают стандартное свойство **Progress** , которое находится в диапазоне от 0 до 1,0, указывая, какой процент перехода отражается в выходных данных. **Выполняется** = 0,0, переход находится в начале последовательности (всего лишь первое изображение видео). Время **выполнения** = 0,5, переход на половину завершен. (Например, при очистке, **выполняется** = 0,5 граница перехода находится в центре изображения). **Выполняется** = 1,0, переход выполнен (целиком — второе изображение). По умолчанию переходы переходят в **состояние Progress** = 0,0 в начале перехода на **Progress** = 1,0 в конце.

Другие свойства обычно относятся к одному конкретному переходу или результату. Например, переход от очистки поддерживает свойство **градиентсизе** , которое управляет шириной области переходов.

Чтобы выполнить переход назад, задайте для свойства **Progress** значение 1,0 в начале перехода и используйте **линейный** элемент для изменения значения на 0,0, как показано в следующем примере:


```
<transition clsid="{af279b30-86eb-11d1-81bf-0000f87557db}" start="0" stop="6">
    <param name="progress" value="1.0">
        <linear time="6" value="0.0" />
    </param>
</transition>
```



## <a name="examples"></a>Примеры


```XML
<param name="progress" value="1.0" />
```
