# <a name="overview-of-the-timeline-components"></a>Общие сведения о компонентах временной шкалы

\[Этот API не поддерживается и может быть изменен или недоступен в будущем.\]

Временная шкала в [службах редактирования DirectShow](directshow-editing-services.md) (DES) является COM-компонентом. Объекты внутри временной шкалы также являются компонентами COM. В этой статье представлен обзор интерфейсов, предоставляемых различными объектами.

Эта статья содержит следующие разделы:

-   [Временная шкала](the-timeline.md)
-   [Объекты временной шкалы](timeline-objects.md)
-   [Подобъекты](subobjects.md)

Дополнительные сведения о временных шкалах см. [в разделе Модель временной шкалы](the-timeline-model.md). Список компонентов в DES см. в разделе [службы редактирования DirectShow объекты служб](directshow-editing-services-objects.md).
