# <a name="cbasebasicvideo-class"></a>Класс Кбасебасиквидео

![Иерархия классов кбасебасиквидео](images/wctrl02.png)

`CBaseBasicVideo`Класс реализует методы **IDispatch** интерфейса [**ибасиквидео**](/windows/desktop/api/Control/nn-control-ibasicvideo) . Оставшиеся методы **ибасиквидео** оставлены чисто виртуальными методами и должны быть реализованы производным классом.

Методы **IDispatch** в этом классе являются стандартными реализациями и подробно описаны здесь.
