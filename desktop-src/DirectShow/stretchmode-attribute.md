# <a name="stretchmode-attribute"></a>Атрибут стретчмоде

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`stretchmode`Атрибут указывает, как растянуть изображение, которое не соответствует выходным измерениям.

## <a name="possible-values"></a>Возможные значения

Должно иметь одно из следующих значений.

-   "Stretch": изображение растягивается в соответствии с размером выходного кадра без сохранения пропорций. (по умолчанию)
-   "обрезать": изображение обрезается в соответствии с размером выходного кадра.
-   "пресервеаспектратио": исходные пропорции сохраняются, а черная леттербокс — в более коротком измерении.
-   "пресервеаспектратионолеттербокс": Размер изображения изменяется для заполнения всего целевого кадра с сохранением пропорций.

## <a name="applies-to"></a>Применяется к

[**clip**](clip-element.md)
