## <a name="remarks"></a><span data-ttu-id="cb671-130">Комментарии</span><span class="sxs-lookup"><span data-stu-id="cb671-130">Remarks</span></span>

<span data-ttu-id="cb671-131">В контексте этого набора свойств Rate измеряет скорость, с которой отметки времени отменяются относительно эталонного времени.</span><span class="sxs-lookup"><span data-stu-id="cb671-131">In the context of this property set, rate measures the rate at which time stamps advance relative to the reference clock.</span></span> <span data-ttu-id="cb671-132">Оцените скорость воспроизведения.</span><span class="sxs-lookup"><span data-stu-id="cb671-132">Rate the inverse of playback speed.</span></span> <span data-ttu-id="cb671-133">Например, если скорость воспроизведения составляет 2x, то метки времени должны увеличиваться в 1/2.</span><span class="sxs-lookup"><span data-stu-id="cb671-133">For example, if the playback speed is 2x, the time stamps must increase at 1/2 the normal rate.</span></span> <span data-ttu-id="cb671-134">Это преобразуется в более быструю скорость воспроизведения, так как образцы отображаются раньше, чем обычные.</span><span class="sxs-lookup"><span data-stu-id="cb671-134">This translates to a faster playback speed, because samples are rendered earlier than normal.</span></span>

<span data-ttu-id="cb671-135">Образцы отправляются в декодер со штампом времени, равным времени презентации по ставке 1x.</span><span class="sxs-lookup"><span data-stu-id="cb671-135">Samples are sent to the decoder with a time stamp equal to the presentation time at 1x rate.</span></span> <span data-ttu-id="cb671-136">Декодер должен масштабировать метки времени на выходных образцах до правильного времени презентации для текущей ставки.</span><span class="sxs-lookup"><span data-stu-id="cb671-136">The decoder must scale the time stamps on the output samples to the correct presentation time for the current rate.</span></span> <span data-ttu-id="cb671-137">Например, если ставка 1/2 (то есть скорость воспроизведения — 2x), декодер должен масштабировать отметки времени на 1/2.</span><span class="sxs-lookup"><span data-stu-id="cb671-137">For example, if the rate is 1/2 (meaning the playback speed is 2x), the decoder must scale the time stamps by 1/2.</span></span> <span data-ttu-id="cb671-138">Как правило, метки времени имеют только кадры.</span><span class="sxs-lookup"><span data-stu-id="cb671-138">Generally, only I frames have time stamps.</span></span> <span data-ttu-id="cb671-139">Декодеру необходимо интерполяцию меток времени для кадров B и P.</span><span class="sxs-lookup"><span data-stu-id="cb671-139">The decoder must interpolate the time stamps for the B and P frames.</span></span> <span data-ttu-id="cb671-140">Обратите внимание, что при обратном воспроизведении метки времени продолжают увеличиваться — отметки времени никогда не переходят назад.</span><span class="sxs-lookup"><span data-stu-id="cb671-140">Note that during reverse playback, time stamps continue to increase — time stamps never go backward.</span></span>

<span data-ttu-id="cb671-141">Определены две версии набора свойств изменения скорости: версия 1,0 и версия 1,1.</span><span class="sxs-lookup"><span data-stu-id="cb671-141">Two versions of the Rate Change property set are defined, version 1.0 and version 1.1.</span></span> <span data-ttu-id="cb671-142">Поведение по умолчанию предоставляется версией 1,0.</span><span class="sxs-lookup"><span data-stu-id="cb671-142">The default behavior is given by version 1.0.</span></span> <span data-ttu-id="cb671-143">Разработчикам декодера рекомендуется поддерживать версию 1,1, так как она обеспечивает более гладкую работу по воспроизведению.</span><span class="sxs-lookup"><span data-stu-id="cb671-143">Decoder vendors are encouraged to support version 1.1, because it provides a smoother playback experience.</span></span> <span data-ttu-id="cb671-144">В настоящее время Навигатор DVD использует версию 1,0.</span><span class="sxs-lookup"><span data-stu-id="cb671-144">The DVD Navigator currently uses version 1.0.</span></span> <span data-ttu-id="cb671-145">Обработчик буфера потока использует версию 1,1.</span><span class="sxs-lookup"><span data-stu-id="cb671-145">The Stream Buffer Engine uses version 1.1.</span></span>
