# <a name="effect"></a>Действие

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

Объект Effect создает видео или звуковой результат. Чтобы создать этот объект, вызовите метод [**иамтимелине:: креатимптиноде**](iamtimeline-createemptynode.md) .

Объект Effect предоставляет следующие интерфейсы:

-   [**иамтимелиниффект**](iamtimelineeffect.md)
-   [**иамтимелинеобж**](iamtimelineobj.md)
-   [**иамтимелинесплиттабле**](iamtimelinesplittable.md)
-   **ипропертипаже**
