# <a name="currentdiscside-property"></a>Куррентдисксиде, свойство

> [!Note]  
> Этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`CurrentDiscSide`Свойство получает текущую сторону диска.

``` syntax
[ iCurSide = ] MSWebDVD.CurrentDiscSide
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает целочисленное значение, представляющее текущую сторону; Сторона A = 1 и сторона B = 2.

## <a name="remarks"></a>Комментарии

Это свойство доступно только для чтения и не имеет значения по умолчанию.
