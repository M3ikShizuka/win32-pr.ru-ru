# <a name="avencmpvuseconcealmentmotionvectors-property"></a>Авенкмпвусеконцеалментмотионвекторс, свойство

Указывает, использует ли кодировщик векторы движения для скрытия. Это свойство применяется к кодировщикам видео MPEG.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**Вариант \_ BOOL** (**\_ логическое значение VT**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкмпвусеконцеалментмотионвекторс**

## <a name="remarks"></a>Remarks

Если значение является **вариантным \_ true**, кодировщик использует векторы движения скрытия.
