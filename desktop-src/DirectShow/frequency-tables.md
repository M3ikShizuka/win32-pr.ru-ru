# <a name="frequency-tables"></a>Частоты таблиц

Фильтр ТВ-тюнера (kstvtune.ax) содержит внутренний список таблиц с частотой. Каждая таблица Frequency содержит список частот и соответствует частоте вещания или кабелей для определенной страны или региона. Приложение подстраивается к определенной частоте, вызывая метод [**иамтунер::p UT \_ Channel**](/windows/desktop/api/Strmif/nf-strmif-iamtuner-put_channel) с индексом нужной частоты.

Для некоторых стран и регионов номера индексов таблиц частоты сопоставляются непосредственно с номерами каналов. Однако фиксированные номера каналов не подходят для всех рынков. Например, европейские номера каналов на самом деле не используются потребителями. Вместо этого потребители хотят выбирать и назначать собственные номера каналов для частот, используемых операторами вещания или кабелей в их области. В этих странах и регионах приложения не должны предоставлять номера индексов непосредственно пользователю. Инстреад, приложение должно удерживать внутреннее сопоставление между номерами каналов (представленными пользователю) и индексами частоты (для настройки).

Большинство операторов кабелей, не связанных с США, можно рассылать по частоте их выбора, часто смешивание частот из разных стандартов в одном и том же канале. Таким образом, таблица частоты "Уникабле" используется для любой страны или региона, где отсутствует стандартный центр стандартов кабельного канала. Кроме того, предоставляется механизм переопределения отдельных частот в таблицах частоты. Этот механизм описан в следующем разделе: [переопределение частоты](frequency-overrides.md).
