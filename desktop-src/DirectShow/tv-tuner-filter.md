# <a name="tv-tuner-filter"></a>Фильтр ТВ-тюнера

Фильтр ТВ-тюнера выбирает аналоговый канал вещания или кабель для просмотра. Один выходной поток — это аппаратный путь для аналогового видео басебанд. Эти выходные данные должны быть подключены к фильтру аналогового микшера видео. Входные контакты включают входные данные для кабеля и входной антенны.
