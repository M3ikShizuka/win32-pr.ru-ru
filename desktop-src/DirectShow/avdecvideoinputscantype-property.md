# <a name="avdecvideoinputscantype-property"></a>Авдеквидеоинпутскантипе, свойство

Указывает способ чередования декодированного видео-потока.

Это свойство доступно только для чтения.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авдеквидеоинпутскантипе**

## <a name="property-value"></a>Значение свойства

Значение этого свойства является членом перечисления [**еавдеквидеоинпутскантипе**](/windows/win32/api/codecapi/ne-codecapi-eavdecvideoinputscantype) .

## <a name="remarks"></a>Комментарии

Верхние 16 разрядов значения содержат ширину, а младшие 16 бит содержат высоту.
