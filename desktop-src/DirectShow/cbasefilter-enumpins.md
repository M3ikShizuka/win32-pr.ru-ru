# <a name="cbasefilterenumpins-method"></a>Кбасефилтер. Енумпинс, метод

`EnumPins`Метод перечисляет ПИН-коды в этом фильтре. Этот метод реализует метод [**ибасефилтер:: енумпинс**](/windows/desktop/api/Strmif/nf-strmif-ibasefilter-enumpins) .

## <a name="syntax"></a>Синтаксис


```C++
HRESULT EnumPins(
   IEnumPins **ppEnum
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ппенум* 
</dt> <dd>

Адрес переменной, которая получает указатель на интерфейс [**иенумпинс**](/windows/desktop/api/Strmif/nn-strmif-ienumpins) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из следующих значений **HRESULT** .



| Код возврата                                                                                   | Описание                          |
|-----------------------------------------------------------------------------------------------|--------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>          | Success<br/>                   |
| <dl> <dt>**E \_ OUTOFMEMORY**</dt> </dl> | Недостаточно памяти<br/>       |
| <dl> <dt>**\_указатель E**</dt> </dl>     | **Пустой** аргумент указателя<br/> |



 

## <a name="remarks"></a>Remarks

Этот метод создает экземпляр базового класса [**ценумпинс**](cenumpins.md) и возвращает указатель на этот объект типа **иенумпинс**. Класс **ценумпинс** вызывает метод [**Кбасефилтер:: жетпин**](cbasefilter-getpin.md) фильтра для перечисления ПИН-кодов в фильтре.

Если этот метод завершился успешно, то интерфейс **иенумпинс** имеет необработанный счетчик ссылок. Вызывающий объект должен освободить интерфейс.
