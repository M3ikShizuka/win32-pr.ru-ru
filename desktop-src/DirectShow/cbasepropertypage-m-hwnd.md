# <a name="cbasepropertypagem_hwnd-member"></a>Элемент HWND Кбасепропертипаже:: m \_

`m_hwnd`Переменная-член содержит маркер диалогового окна. Эта переменная-член инициализируется после того, как объект создает диалоговое окно, когда функция **креатедиалогпарам** возвращает.

## <a name="syntax"></a>Синтаксис


```C++
HWND m_hwnd;
```
