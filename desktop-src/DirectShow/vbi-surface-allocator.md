# <a name="vbi-surface-allocator"></a>Распределитель поверхности ВБИ

Распределитель поверхности ВБИ управляет выделением буферов ВБИ в аналоговых телевизионных диаграммах с помощью аппаратных сценариев записи видеопортов. При использовании таких устройств, как декодер Bt829, буфер кадров может содержать несколько буферов записи ВБИ, доступ к которым осуществляется через собственный механизм на основе оборудования, известный как универсальный порт видео. Фильтр распределителя ВБИ Surface соединяется дальше от фильтра записи и не имеет выходного ПИН-кода. Фильтр тесно работает с [микшером наложения](overlay-mixer-filter.md) (с помощью DirectDraw) для выполнения согласованных операций с аппаратным портом оборудования, используя тот же ограниченный буфер памяти проигрывателя SVGA.
