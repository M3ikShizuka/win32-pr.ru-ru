# <a name="cdrawimagem_sourcerect-member"></a>Элемент Кдравимаже:: m \_ саурцерект

Переменная члена **m \_ саурцерект** указывает исходный прямоугольник для рисования. Начальное значение является пустым прямоугольником, указывающим, что изображение должно быть выведено целиком.

## <a name="syntax"></a>Синтаксис


```C++
RECT m_SourceRect;
```
