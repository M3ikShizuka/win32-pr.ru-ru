# <a name="irenderengineconnectfrontend-method"></a>Метод Ирендеренгине:: Коннектфронтенд

> [!Note]  
> \[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

 

`ConnectFrontEnd`Метод создает внешний интерфейс графа фильтра из текущей временной шкалы.

## <a name="syntax"></a>Синтаксис


```C++
HRESULT ConnectFrontEnd();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **HRESULT** . Возможные возвращаемые значения включают следующее.



| Код возврата                                                                                                  | Описание                                                                    |
|--------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------|
| <dl> <dt>**\_ОК**</dt> </dl>                         | Успешно.<br/>                                                            |
| <dl> <dt>**\_предупреждать \_ аутпутресет**</dt> </dl>          | Часть графа отрисовки была удалена.<br/>                         |
| <dl> <dt>**E \_ INVALIDARG**</dt> </dl>                 | Для этого модуля визуализации не задана временная шкала.<br/>                             |
| <dl> <dt>**E \_ должен \_ инициализировать модуль \_ подготовки отчетов**</dt> </dl>       | Не удалось инициализировать модуль визуализации.<br/>                                 |
| <dl> <dt>**\_модуль подготовки \_ отчетов \_ не \_ работает**</dt> </dl> | Не удалось выполнить операцию, так как проект не был успешно визуализирован.<br/> |
| <dl> <dt>**\_непредвиденная E**</dt> </dl>                 | Непредвиденная ошибка.<br/>                                                   |
| <dl> <dt>**VFW \_ E \_ инвалидмедиатипе**</dt> </dl>      | Недопустимый тип носителя.<br/>                                                 |



 

## <a name="remarks"></a>Remarks

Этот метод не создает часть отрисовки графа фильтра. Приложение должно подключить выходные контакты на внешнем интерфейсе к нужным фильтрам отрисовки:

-   Для предварительного просмотра вызовите метод [**ирендеренгине:: рендераутпутпинс**](irenderengine-renderoutputpins.md) .
-   Чтобы вывести файл, вызовите [**ирендеренгине:: жетграупаутпутпин**](irenderengine-getgroupoutputpin.md) , чтобы получить выходной ПИН-код для каждой группы, а затем соедините контакты с фильтром мультиплексора.

Если используется базовый механизм визуализации, то выходные сигналы на внешнем интерфейсе создают несжатые данные. Если используется интеллектуальный механизм визуализации, то выходные данные создают сжатые.

При изменении временной шкалы после построения графа фильтра необходимо вызвать `ConnectFrontEnd` повторный вызов для перестроения внешнего интерфейса. По возможности метод сохраняет часть графа отрисовки. Тем не менее, если добавить или удалить группу или изменить порядок групп, будет `ConnectFrontEnd` удалена визуализация, а приложение должно перестроиться. Если метод удаляет часть отрисовки, возвращается \_ предупреждение \_ аутпутресет.
