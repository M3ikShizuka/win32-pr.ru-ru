# <a name="ksproperty_extxport_atn_search"></a>КСПРОПЕРТИ \_ ексткспорт \_ АТН \_ Search

Это свойство отправляет на устройство команду для поиска абсолютного номера записи (АТН). Драйвер устройства УВК поддерживает это свойство.



| Метка | Значение |
|-------------------|---------------------------------------|
| Идентификатор GUID набора свойств | ПРОПСЕТИД \_ ext \_ Transport             |
| Идентификатор свойства       | КСПРОПЕРТИ \_ ексткспорт \_ АТН \_ Search     |
| Тип данных         | **Кспроперти \_ Структура \_ ексткспорт** |



 

## <a name="remarks"></a>Remarks

Задайте для элемента **двабстраккнумбер** структуры **кспроперти \_ ексткспорт \_** следующее значение:


```
(absolute track number << 1) + continuity bit
```



Спецификация УВК не определяет, как устройство использует бит непрерывности. структура **кспроперти \_ ексткспорт \_** описана в пакете DDK по Windows.
