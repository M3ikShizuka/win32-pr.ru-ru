# <a name="coutputqueuefreesamples-method"></a>Каутпуткуеуе. Фрисамплес, метод

`FreeSamples`Метод освобождает все ожидающие выборки.

## <a name="syntax"></a>Синтаксис


```C++
void FreeSamples();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Этот метод удаляет все ожидающие выборки из очереди и из образца массива.
