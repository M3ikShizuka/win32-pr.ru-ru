# <a name="cimageallocatornotifymediatype-method"></a>Цимажеаллокатор. Нотифимедиатипе, метод

`NotifyMediaType`Метод информирует объект о текущем типе мультимедиа.

## <a name="syntax"></a>Синтаксис


```C++
void NotifyMediaType(
   CMediaType *pMediaType
);
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*пмедиатипе* 
</dt> <dd>

Указатель на объект [**кмедиатипе**](cmediatype.md) или **значение NULL** для очистки типа мультимедиа.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Remarks

Фильтр-владелец должен вызывать этот метод при каждом изменении типа мультимедиа. Обычно это происходит, когда ПИН-код сначала соединяется и после динамического изменения формата. Распределитель использует тип носителя для проверки предложенных свойств распределителя, а также при создании образцов носителя.

Объект **Цимажеаллокатор** сохраняет указатель *пмедиатипе* в переменной элемента **m \_ пмедиатипе** . Таким образом, если вызывающему объекту нужно освободить объект **кмедиатипе** , он должен обновить распределитель путем повторного вызова этого метода либо с новым указателем, либо со значением **null** . В противном случае при попытке распределителя ссылаться на старый указатель может возникнуть ошибка.
