# <a name="avencddlorosurroundmixlvl_x10-property"></a>Авенкддлоросурраундмикслвл \_ X10, свойство

Задает сдвиг уровня, применяемый к окружающим каналам для довнмиксинг и ro. Это свойство применяется к кодировщикам цифрового аудио Dolby.

Это свойство доступно для чтения и записи.

## <a name="data-type"></a>Тип данных

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID свойства

**КОДЕКАПИ \_ авенкддлоросурраундмикслвл \_ x10**

## <a name="property-value"></a>Значение свойства

Значение указывается как децибел (dB) x 10.
