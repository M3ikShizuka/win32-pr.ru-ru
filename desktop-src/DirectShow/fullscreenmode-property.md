# <a name="fullscreenmode-property"></a>Фуллскринмоде, свойство

> [!Note]  
> этот компонент доступен для использования в операционных системах Microsoft Windows 2000, Windows XP и Windows Server 2003. В последующих версиях он может быть изменен или недоступен.

 

`FullScreenMode`Свойство задает или получает значение, указывающее, находится ли дисплей в полноэкранном режиме.

``` syntax
[ bFullScreen = ] MSWebDVD.FullScreenMode
```

## <a name="return-value"></a>Возвращаемое значение

Возвращает логическое значение, указывающее, следует ли включать или отключать полноэкранное воспроизведение.

## <a name="remarks"></a>Remarks

Это свойство доступно для чтения и записи со значением по умолчанию false.



| Значение | Описание                                            |
|-------|--------------------------------------------------------|
| Да  | Включить воспроизведение в полноэкранном режиме. Это значение по умолчанию. |
| false | Отключить воспроизведение в полноэкранном режиме.                          |



 

Полноэкранный режим доступен, только если элемент управления Мсвебдвд работает в оконном режиме.
