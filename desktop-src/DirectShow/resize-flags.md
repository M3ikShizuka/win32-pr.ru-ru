# <a name="resize-flags"></a>Изменение размера флагов

\[Не рекомендуется. Этот API может быть удален из будущих выпусков Windows.\]

Эти флаги определяют, как отображается источник видео, если его размер не соответствует выходным измерениям.



| Константа/значение                                                                                                                                                                                                                                                                                      | Описание                                                                                                                                                                                                                        |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="RESIZEF_STRETCH"></span><span id="resizef_stretch"></span><dl> <dt>**Ресизеф \_ РАСТЯЖЕНИе**</dt> <dt>0</dt> </dl>                                                                          | Изображение растягивается в соответствии с размером целевого кадра в обоих измерениях без сохранения пропорций.<br/>                                                                                                            |
| <span id="RESIZEF_CROP"></span><span id="resizef_crop"></span><dl> <dt>**Ресизеф \_ Обрезать**</dt> <dt>1</dt> </dl>                                                                                   | Размер изображения не изменяется. Если изображение меньше целевого кадра, окружающая область будет черным. Если изображение больше целевого кадра, изображение обрезается.<br/>                                             |
| <span id="RESIZEF_PRESERVEASPECTRATIO"></span><span id="resizef_preserveaspectratio"></span><dl> <dt>**Ресизеф \_ ПРЕСЕРВЕАСПЕКТРАТИО**</dt> <dt>2</dt> </dl>                                      | Размер изображения изменяется в соответствии с целевым кадром по одному измерению с сохранением пропорций. Если отношение ширины к высоте в изображении не соответствует соотношениям в целевом кадре, создается леттербокс.<br/> |
| <span id="RESIZEF_PRESERVEASPECTRATIO_NOLETTERBOX"></span><span id="resizef_preserveaspectratio_noletterbox"></span><dl> <dt>**Ресизеф \_ ПРЕСЕРВЕАСПЕКТРАТИО \_ нолеттербокс**</dt> <dt>3</dt> </dl> | Размер изображения изменяется для заполнения всего целевого фрейма с сохранением пропорций. Вместо создания леттербокс в этом режиме изображение обрезается по сторонам или вдоль верхней и нижней части.<br/>                 |
