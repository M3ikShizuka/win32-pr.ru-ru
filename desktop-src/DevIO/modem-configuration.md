# <a name="modem-configuration"></a>Конфигурация модема

Функции настройки модема позволяют настроить модем перед подключением. Приложение может устанавливать параметры модема и определять возможности модема без использования команд, относящихся к любому модемному устройству. Ниже приведены общие функции, которые приложение может установить перед вызовом:

-   Основной режим работы (синхронный, асинхронный, а также включено ли Управление ошибками).
-   V. 42. Управление ошибками (определено рекомендацией CCITT V. 42), включая определенные параметры. CCITT означает Международный телеграфа и Консултативеный Комитет.
-   V. 42bis (определяется в соответствии с рекомендацией CCITT V. 42bis) и MNP5 Data Compression.
-   Параметры времени ожидания, включая настройку вызова, неактивность и доставку буферизованных данных.

Перед настройкой конфигурации модема приложение должно определить возможности модемного устройства с помощью функции [**жеткоммпропертиес**](/windows/desktop/api/Winbase/nf-winbase-getcommproperties) . Эта функция заполняет структуру [**коммпроп**](/windows/desktop/api/WinBase/ns-winbase-commprop) . Эта структура содержит как общую часть, которая применяется ко всем устройствам связи, так и часть, относящуюся к каждому подтипу поставщика. Для модемных устройств часть структуры **коммпроп** , относящаяся к конкретному поставщику, является структурой [**модемдевкапс**](/windows/desktop/api/Mcx/ns-mcx-modemdevcaps) .

Приложение может получить и задать текущую конфигурацию модема с помощью функций [**жеткоммконфиг**](/windows/desktop/api/Winbase/nf-winbase-getcommconfig) и [**сеткоммконфиг**](/windows/desktop/api/Winbase/nf-winbase-setcommconfig) , в которых используется структура [**коммконфиг**](/windows/desktop/api/Winbase/ns-winbase-commconfig) . Эта структура содержит как общую часть, которая применяется ко всем устройствам связи, так и часть, относящуюся к каждому подтипу поставщика. Для модемных устройств часть структуры **коммконфиг** , относящаяся к конкретному поставщику, является структурой [**модемсеттингс**](/windows/desktop/api/Mcx/ns-mcx-modemsettings) .

После настройки модема приложение может использовать интерфейс программирования приложений телефонии (TAPI) для фактического установления соединения.

Функции настройки модема не предоставляют долгосрочного управления и обслуживания модема. Для этой цели поставщики услуг модема должны предоставлять диалоговые окна настройки модема.
