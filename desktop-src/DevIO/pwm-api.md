Модулятор по ширине импульсов (широты) — это способ создания прямоугольной импульсной волновой волны с заданной толщиной импульса, которая приводит к получению варианта среднего значения звукового сигнала. Наиболее распространенные применения включают в себя серво моторы, индикаторы DIMM или другие связанные функции. Модулятор предназначен для использования в основном для Интернет вещей сценариев.

## <a name="about-pulse-width-modulation"></a>О модуляции для импульсной ширины

С помощью форм-волнового ИМПУЛЬСа можно разделить два параметра:

-   Период волны (T)

-   Цикл пошлины, где частота сигнала (f) является обратной точкой f = 1/T

Цикл пошлины описывает пропорции  / **активного** времени по отношению к обычному интервалу или **периоду** времени. Цикл с низкой нагрузкой соответствует среднему времени низкой выходной мощности, так как в большинстве случаев выключение питания отключено. Цикл пошлины выражается в процентах. Полностью включено — 100%. Полностью отключено — 0%. **Активная** половина времени — 50%.
