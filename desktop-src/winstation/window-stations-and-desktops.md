# <a name="window-stations-and-desktops"></a>Оконные станции и рабочие столы

Windows предоставляет три основные категории объектов: пользовательский интерфейс, интерфейс графических устройств (GDI) и ядро. [Объекты ядра](/windows/desktop/SysInfo/kernel-objects) являются защищаемыми, тогда как [объекты пользователя](/windows/desktop/SysInfo/user-objects) и [объекты GDI](/windows/desktop/SysInfo/gdi-objects) — нет. Таким образом, для обеспечения дополнительной безопасности объекты пользовательского интерфейса управляются с помощью оконных станций и настольных компьютеров, которые сами являются защищаемыми объектами.
