# <a name="window-stations"></a>Оконные станции

*Оконная станция* содержит буфер обмена, таблицу Atom и один или несколько объектов [рабочего стола](desktops.md) . Каждый объект "станция окна" является защищаемым объектом. При создании станции Windows она связывается с вызывающим процессом и назначается текущему сеансу.

*Интерактивная станция окна* — единственная Оконная станция, которая может отображать пользовательский интерфейс или принимать вводимые пользователем данные. Он назначается сеансу входа интерактивного пользователя и содержит клавиатуру, мышь и устройство вывода. Всегда имеет имя «WinSta0». Все остальные оконные станции являются неинтерактивными. Это означает, что они не могут отображать пользовательский интерфейс или принимать входные данные пользователя.

Когда пользователь входит в систему с помощью службы удаленных рабочих столов, для пользователя запускается сеанс. Каждый сеанс связан со своей собственной интерактивной рабочей станцией с именем «WinSta0». Дополнительные сведения см. в разделе [Удаленный рабочий стол Sessions](/windows/desktop/TermServ/terminal-services-sessions).
