Представляет сведения о журнале пикселей.

## <a name="syntax"></a>Синтаксис


```C++
} PixelHistoryOperation;
```

## <a name="members"></a>Участники

**Аль**  
Идентификатор события графики, связанного с этой операцией.

**PCP**  
Упакованные вызовы, связанные с этой операцией.

**рендертаржетптр**  
Целевой объект прорисовки, изначально связанный (внутри захваченного приложения) с этой операцией.

**иприм**  
Индекс фактического примитива, связанного с его операцией.

**нумпримс**  
Общее число примитивов, связанных с этой операцией.

**нумвертсперприм**  
Количество вершин на примитив.

**иинстанце**  
При отрисовке экземпляров — номер экземпляра фактического экземпляра, связанного с этой операцией.

**иинстанцекаунт**  
При отрисовке экземпляров — общее количество экземпляров, связанных с этой операцией.

**бассемблерстажеженератесинстанцеид**  
значение true, если входной ассемблер создает идентификаторы экземпляров; в противном случае — false.

**flags**  
Сочетание значений ПИКСЕЛХИСТОРИФЛАГС. Дополнительные сведения см. в описании перечисления ПИКСЕЛХИСТОРИФЛАГС.

**пвсфиле**  
ФИЛЕПТР для потока байтов построителя текстуры. Это значение передается обратно для отладки.

**пгсфиле**  
ФИЛЕПТР для потока байтов шейдера Geometry. Это значение передается обратно для отладки.

**ппсфиле**  
ФИЛЕПТР для потока байтов построителя текстуры. Это значение передается обратно для отладки.

**фсфиле**  
ФИЛЕПТР для потока байтов шейдера поверхности. Это значение передается обратно для отладки.

**пдсфиле**  
ФИЛЕПТР для потока байтов шейдера домена. Это значение передается обратно для отладки.

**пксфиле**  
ФИЛЕПТР для потока байтов вычислений COMPUTE Shader. Это значение передается обратно для отладки.

**вертексшадерфиле**  
Строка COM, содержащая путь к файлу исходного файла шейдера вершин.

**пикселшадерфиле**  
Строка COM, содержащая путь к файлу исходного кода шейдера пикселей.

**жеометришадерфиле**  
Строка COM, содержащая путь к файлу файла исходного шейдера Geometry.

**хуллшадерфиле**  
Строка COM, содержащая путь к файлу исходного файла шейдера поверхности.

**домаиншадерфиле**  
Строка COM, содержащая путь к файлу исходного файла шейдера домена.

**псред**  
Выходные данные шейдера пикселей: значение красного цвета компонента.

**псгрин**  
Выходные данные шейдера пикселей: значение зеленого компонента цвета.

**псблуе**  
Выходные данные шейдера пикселей: значение синего компонента цвета

**псалфа**  
Выходные данные шейдера пикселей: значение компонента цвета Alpha

**лабелпсред**  
Строка COM, содержащая имя метки, связанной с компонентом красного цвета в выходных данных шейдера пикселей.

**лабелпсгрин**  
Строка COM, содержащая имя метки, связанной с компонентом зеленого цвета в выходных данных шейдера пикселей.

**лабелпсблуе**  
Строка COM, содержащая имя метки, связанной с компонентом синего цвета выходных данных шейдера пикселей.

**лабелпсалфа**  
Строка COM, содержащая имя метки, связанной с компонентом цвета альфа-канала выходных данных шейдера пикселей.

**пикселкиллреасон**  
Выходные данные шейдера пикселей: Причина завершения вывода пикселя.

**пикселокклудед**  
true, если пиксель имеет значение перекрыто; в противном случае — значение false.

**фбред**  
Буфера кадров: значение красного цвета компонента буфера кадров перед слиянием выходных данных шейдера пикселей.

**фбгрин**  
Буфера кадров: значение зеленого компонента цвета буфера кадров перед слиянием выходных данных шейдера пикселей.

**фбблуе**  
Буфера кадров: значение синего компонента цвета буфера кадров перед слиянием выходных данных шейдера пикселей.

**фбалфа**  
Буфера кадров: значение альфа-компонента цвета буфера кадров перед слиянием выходных данных шейдера пикселей.

**лабелфбред**  
Строка COM, содержащая имя метки, связанной с компонентом красного цвета буфера кадров.

**лабелфбгрин**  
Строка COM, содержащая имя метки, связанной с компонентом зеленого цвета буфера кадров.

**лабелфбблуе**  
Строка COM, содержащая имя метки, связанной с компонентом синего цвета буфера кадров.

**лабелфбалфа**  
Строка COM, содержащая имя метки, связанной с компонентом цвета альфа-канала буфера кадров.

**Topology**  
Топология вершин вызовов Draw (список треугольников, полоска треугольника и т. д.).

**вершины**  
Строка COM, содержащая буфер вершин, начиная с этого примитива. Буфер вершин соответствует формату входного макета, указанному на этапе конвейера.

**вертекссизе**  
Размер одной вершины в байтах.

**инпутлайаут**  
Строка COM, содержащая последовательность структур Инпутлайаутструкт, связанных с вызовом Draw.

**HResult**  
Значение HRESULT для DirectX. В случае проблемы это может быть использовано для вывода ошибки.
