# <a name="effect-compiler-tool"></a>Инструмент Effect-Compiler

FXC (fxc.exe) — это автономное средство для компиляции шейдеров [HLSL](/windows/desktop/direct3dhlsl/dx-graphics-hlsl) для всех версий Direct3D. Это средство находится по адресу:

(*Корень пакета SDK*) \\ Служебная программа \\ bin \\ x86\\
