# <a name="span-idvspixenginepixenginetextureformatdescspanpixenginetextureformatdesc-structure"></a><span id="vspixengine.pixenginetextureformatdesc"></span>Структура Пиксенгинетекстуреформатдеск

Представляет описание формата текстуры.

## <a name="syntax"></a>Синтаксис


```C++
} PixEngineTextureFormatDesc;
```

## <a name="members"></a>Члены

**дксгиформат**  
Формат DXGI текстуры.

**Формат.**  
Формат данных текстуры.

**блоккбитес**  
Число байтов в блоке данных.

**блоккхеигхт**  
Высота (числовые примеры на оси Y) блока.

**блокквидс**  
Ширина (число выборок на оси X) блока.

**чаннелкс**  
Описывает свойства канала X. Дополнительные сведения см. в разделе Структура Пиксенгинечаннелдескриптион.

**канал**  
Описывает свойства канала Y. Дополнительные сведения см. в разделе Структура Пиксенгинечаннелдескриптион.

**чаннелз**  
Описывает свойства канала Z. Дополнительные сведения см. в разделе Структура Пиксенгинечаннелдескриптион.

**чаннелв**  
Описание свойств канала W. Дополнительные сведения см. в разделе Структура Пиксенгинечаннелдескриптион.

**свиззле**  
Описывает свиззле (обмен каналами), примененный к примеру.
