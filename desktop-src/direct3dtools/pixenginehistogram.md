## <a name="members"></a>Члены

**хоризонталмин**  
Минимальное значение для каждого из компонентов X, Y, Z и W в горизонтальной оси (домене) гистограммы.

**хоризонталмакс**  
Максимальное значение для каждого из компонентов X, Y, Z и W в горизонтальной оси (домене) гистограммы.

**вертикалмин**  
Минимальное значение для каждого из компонентов X, Y, Z и W в вертикальной оси (диапазон) гистограммы.

**вертикалмакс**  
Максимальное значение для каждого из компонентов X, Y, Z и W в вертикальной оси (диапазон) гистограммы.

**dataLength**  
Число выборок, рассматриваемых в гистограмме.
