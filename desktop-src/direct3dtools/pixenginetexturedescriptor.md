Представляет описание ресурса текстуры.

## <a name="syntax"></a>Синтаксис


```C++
} PixEngineTextureDescriptor;
```

## <a name="members"></a>Участники

**форматдеск**  
Описывает формат текстуры.

**mipmapCount**  
Число MIP-карты, связанных с текстурой.

**arrayCount**  
Если текстура является массивом текстуры, число текстур в массиве.

**Длина**  

**sampleCount**  
Число выборок, связанных с текстурой.
