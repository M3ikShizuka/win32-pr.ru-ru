# <a name="monitor-configuration-monitor-configuration"></a>Мониторинг конфигурации (конфигурация монитора)

Функции конфигурации монитора можно использовать для получения сведений с монитора и для изменения параметров монитора. Эти функции можно использовать для:

-   Изменение параметров геометрических объектов монитора, таких как ширина и высота дисплея.
-   Изменение параметров изображения, таких как яркость и контрастность.
-   Сброс параметров монитора до заводских настроек по умолчанию.
-   Дегаусс монитор.

На внутреннем уровне функции конфигурации монитора используют интерфейс командной строки отображения данных (DDC/CI) для отправки команд монитору.

Функции конфигурации монитора описаны в следующих разделах:

-   [Сведения о конфигурации монитора](about-monitor-configuration.md)
-   [Использование конфигурации монитора](using-monitor-configuration.md)
-   [Справочник по конфигурации монитора](monitor-configuration-reference.md)
