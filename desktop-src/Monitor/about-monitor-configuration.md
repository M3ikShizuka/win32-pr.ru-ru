# <a name="about-monitor-configuration"></a>Сведения о конфигурации монитора

Для функций конфигурации монитора используются следующие функции.

-   Калибровка цветов. Правильно откалиброванный монитор правильно воспроизводит цвета. Как правило, приложение калибровки цветов отображает шаблон тестирования и содержит элементы управления для изменения параметра монитора. Пользователь изменяет параметр до тех пор, пока не будет выполнено условие, и повторяет этот процесс для каждого параметра монитора до калибровки монитора.
-   Настройка области отображения монитора. Функции конфигурации монитора можно использовать для изменения размера и расположения области отображения монитора. Например, когда ЖК-монитор подключен к компьютеру с помощью аналогового соединителя, наилучшее качество изображения получается при наличии однозначного сопоставления между пикселями в буфере кадров видеокарты и пикселами на ЖК-мониторе.
-   Сохранение и восстановление параметров монитора. Некоторым пользователям требуется несколько наборов параметров монитора, так как для разных сценариев требуются разные параметры монитора. Например, параметры монитора, оптимальные для настольных приложений, не подходят для просмотра телепередач.
-   Использование функций монитора, зависящих от поставщика. С помощью функций конфигурации монитора изготовители могут создавать приложения, использующие функции монитора, относящиеся к конкретному поставщику.

Функции конфигурации монитора делятся на функции высокого уровня и низкого уровня. Функции высокого уровня проще в использовании, чем низкоуровневые функции. Чтобы использовать функции высокого уровня, не нужно ничего знать о интерфейсе командной строки интерфейса дисплея (DDC/CI). Однако функции высокого уровня не предоставляют доступ к функциям данного монитора, относящимся к поставщику.

Низкоуровневые функции — это тонкая оболочка для команд DDC/CI. Для использования низкоуровневых функций необходимо быть знакомым со стандартом DDC/CI, а также со стандартом Set монитора VESA (MCCS). Как правило, следует использовать функции высокого уровня, если не требуется использовать функции, доступные только в низкоуровневых функциях.

Функции конфигурации монитора высокого уровня спроектированы таким образом, что приложение не может перевести монитор в неработоспособное состояние. Низкоуровневые функции можно использовать для отправки кодов в монитор. Тем не менее имейте в виду, что некоторые коды кодов восмогут отключать важные функциональные возможности или перевести монитор в состояние, в котором пользователь не может видеть отображаемое изображение. Дополнительные сведения см. в разделе набор команд управления монитора VESA (MCCS) Standard.
