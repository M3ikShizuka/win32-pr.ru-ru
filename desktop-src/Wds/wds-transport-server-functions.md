# <a name="wds-transport-server-functions"></a>Функции транспортного сервера WDS

Функции, предоставляемые сервером многоадресной рассылки, который взаимодействует с поставщиками содержимого.



| Функция                                                                         | Описание                                              |
|----------------------------------------------------------------------------------|----------------------------------------------------------|
| [**вдстранспортсервераллокатебуффер**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverallocatebuffer)     | Выделяет буфер в памяти.                            |
| [**вдстранспортсерверкомплетереад**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservercompleteread)         | Предоставляет состояние операции чтения.                       |
| [**вдстранспортсерверфрибуффер**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverfreebuffer)             | Освобождает память, используемую буфером.                        |
| [**вдстранспортсерверрегистеркаллбакк**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportserverregistercallback) | Регистрирует обратный вызов поставщика на сервере многоадресной рассылки. |
| [**вдстранспортсервертраце**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservertrace)                       | Отправляет сообщение об отладке.                               |
| [**вдстранспортсервертрацев**](/windows/desktop/api/Wdstpdi/nf-wdstpdi-wdstransportservertracev)                     | Отправляет сообщение об отладке.                               |
