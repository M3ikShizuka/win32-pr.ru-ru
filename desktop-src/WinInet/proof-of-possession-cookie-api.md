# <a name="proof-of-possession-cookie-api"></a>API-интерфейс подтверждения принадлежности к файлу

Позволяет создавать файлы cookie для подтверждения владения, которые поддерживают единый вход для веб-приложений, прошедших проверку подлинности с помощью протокола проверки подлинности в Интернете OAuth. Вы можете использовать сведения, возвращаемые этим API, для реализации единого входа в форму учетных записей Майкрософт.
