# <a name="vds-data-types"></a>Типы данных VDS

\[начиная с Windows 8 и Windows Server 2012, интерфейс COM [службы виртуальных дисков](virtual-disk-service-portal.md) заменяется [API Windows служба хранилища управления](/previous-versions/windows/desktop/stormgmt/windows-storage-management-api-portal).\]

Типы данных, поддерживаемые службой VDS, возвращают возвращаемые функцией значения, параметры функций и члены структуры. Типы данных определяют размер и смысл этих элементов.



| Тип данных                                                                                      | Описание                                                                                                                                                                       |
|------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="VDS_OBJECT_ID"></span><span id="vds_object_id"></span>**\_ИД объекта \_ VDS**<br/> | Идентификатор объекта VDS. Эти значения не обязательно должны быть уникальными во всех перезагрузках. <br/> Этот тип объявлен в службе VDS. h (Вдшвпрв. h для поставщиков оборудования) как GUID. <br/> |
| <span id="VDS_PATH_ID"></span><span id="vds_path_id"></span>**\_ИД пути к службе VDS \_**<br/>       | ИДЕНТИФИКАТОР пути службы VDS для Multipath I/O (MPIO). <br/> Этот тип объявлен в службе VDS. h (Вдшвпрв. h для поставщиков оборудования) как UINT64.<br/>                                      |
