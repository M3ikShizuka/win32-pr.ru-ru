# <a name="system-restore-api"></a>API восстановления системы

API восстановления системы позволяет приложению управлять точками восстановления. Существует также доступный в сценарии интерфейс с помощью [инструментарий управления Windows (WMI)](/windows/desktop/WmiSdk/wmi-start-page) (WMI).

Дополнительные сведения об API восстановления системы см. в следующих разделах:

-   [Вызов Срсетресторепоинт](calling-srsetrestorepoint.md)
-   [Текст описания точки восстановления](restore-point-description-text.md)
-   [Вложенные вызовы Срсетресторепоинт](nested-calls-to-srsetrestorepoint.md)

Дополнительные сведения о классах WMI для восстановления системы см. в разделе [Настройка восстановления системы](configuring-system-restore.md).
