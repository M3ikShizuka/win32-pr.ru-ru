# <a name="system-restore-functions"></a>Функции восстановления системы

При восстановлении системы используются следующие функции.



| Функция                                             | Описание                                                                                                   |
|------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**срремовересторепоинт**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srremoverestorepoint) | Удаляет указанную точку восстановления.                                                                          |
| [**срсетресторепоинт**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa)       | Задает начало и конец набора изменений, чтобы система восстановления системы могла создать точку восстановления. |
