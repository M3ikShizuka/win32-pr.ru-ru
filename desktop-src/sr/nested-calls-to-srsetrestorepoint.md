# <a name="nested-calls-to-srsetrestorepoint"></a>Вложенные вызовы Срсетресторепоинт

В этом разделе описывается поддержка вложенных вызовов [**срсетресторепоинт**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa) с помощью \_ типов событий Begin Nested \_ System \_ Change и End \_ Nested \_ System \_ Change.

Приложения могут безопасно вызывать [**срсетресторепоинт**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa) при использовании этих типов событий. Первый вызов функции создает точку восстановления. Последующие вложенные вызовы функции не создают точки восстановления. Например, предположим, что приложение выполняет следующие вызовы **срсетресторепоинт**:<dl> Для точки восстановления A с Двевенттипе = начало \_ вложенного \_ \_ изменения системы  
Для точки восстановления B с Двевенттипе = начало \_ вложенного \_ \_ изменения системы  
Для точки восстановления B с Двевенттипе = конец \_ вложенного \_ \_ изменения системы  
Для точки восстановления A с Двевенттипе = конец \_ вложенного \_ \_ изменения системы  
</dl>

Второй вызов не создает новую точку восстановления, так как вызов является вложенным.
