# <a name="configuring-system-restore"></a>Настройка восстановления системы

средство восстановления системы использует [инструментарий управления Windows (WMI)](/windows/desktop/WmiSdk/wmi-start-page) (WMI) для предоставления сценариям возможности настройки и использования его функциональных возможностей. Он предоставляет два класса, [**SystemRestore**](systemrestore.md) и [**системрестореконфиг**](systemrestoreconfig.md), в корневом \\ пространстве имен по умолчанию.

Класс [**SystemRestore**](systemrestore.md) предоставляет методы для следующих задач:

-   Отключение и включение мониторинга
-   Перечисление доступных точек восстановления
-   Запуск восстановления в локальной системе
-   Получение состояния последнего восстановления системы

Класс [**системрестореконфиг**](systemrestoreconfig.md) предоставляет свойства для управления частотой создания точек восстановления по расписанию и объемом дискового пространства, занятого восстановлением системы на каждом диске.

Доступ к обоим классам можно получить удаленно с помощью стандартных методов WMI. полное описание инструментария WMI см. в разделе [инструментарий управления Windows (WMI)](/windows/desktop/WmiSdk/wmi-start-page).
