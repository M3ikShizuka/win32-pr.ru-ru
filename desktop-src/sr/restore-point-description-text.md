# <a name="restore-point-description-text"></a>Текст описания точки восстановления

Когда приложение вызывает функцию [**срсетресторепоинт**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa) , оно может предоставить любой текст в качестве описания для точки восстановления. Однако в следующей таблице показан рекомендуемый текст описания.



| Изменение системы                            | Тип точки восстановления      | Описание            |
|------------------------------------------------|-------------------------|------------------------|
| Установка приложения                       | \_Установка приложения    | Установленное *AppName*    |
| Изменение приложения (Добавление и удаление компонентов) | ИЗМЕНИТЬ \_ Параметры        | Настроено *AppName*   |
| Удаление приложения                            | \_Удаление приложения  | Удалено *AppName*      |
| Установка драйвера устройства                     | \_Установка драйвера \_ устройства | Установленный *имя_драйвера* |



 

Установщики, такие как установщик Windows и InstallShield, используют следующие соглашения для текста описания:

-   Название продукта соответствует команде. Например, установлено *AppName*.
-   Имя продукта можно использовать отдельно (*AppName*), а название продукта и название компании могут быть использованы (*MyCompany AppName*).
