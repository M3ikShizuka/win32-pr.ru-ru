# <a name="basewindow-sample"></a>Пример Басевиндов

В этом примере приложения показано, как передавать данные о состоянии приложения в сообщение [**\_ нккреате WM**](/windows/desktop/winmsg/wm-nccreate) .

## <a name="description"></a>Описание

Пример приложения Басевиндов является разновидностью [образца Windows Hello World](windows-hello-world-sample.md). Он использует сообщение [**WM \_ нккреате**](/windows/desktop/winmsg/wm-nccreate) для передачи данных приложения в процедуру окна. Этот пример рассматривается в разделе [Управление состоянием приложения](managing-application-state-.md).
