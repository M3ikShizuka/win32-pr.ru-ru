# <a name="get-started-with-win32-and-c"></a>Get Started with Win32 and C++ (Приступая к работе с Win32 и C++)

Эта серия приступит к работе, чтобы научиться писать настольную программу на C++ с помощью API Win32 и COM.

В первом модуле вы узнаете пошаговое руководство по созданию и отображению окна. В последующих модулях будут представлены элементы модели COM, графики и текста, а также введенные пользователем данные.

Для этой серии предполагается, что у вас есть хорошее знание программирования на C++. Нет опыта работы с программированием Windows. Если вы не знакомы с C++, вы можете найти учебные материалы в [центре разработчиков Visual C++](https://msdn.microsoft.com/vstudio//default.aspx). (Этот ресурс может быть недоступен в некоторых языках и странах.)
