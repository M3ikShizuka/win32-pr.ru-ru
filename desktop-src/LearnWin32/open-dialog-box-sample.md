В этом примере приложения показано, как инициализировать библиотеку модели COM и использовать API на основе COM в программе Windows.

## <a name="description"></a>Описание

Образец приложения открыть диалоговое окно открывает диалоговое окно **Открыть** , как показано на снимке экрана ниже. В примере демонстрируется вызов COM-объекта в программе Windows. Этот пример обсуждается в [модуле 2: использование COM в программе Windows](module-2--using-com-in-your-windows-program.md).
