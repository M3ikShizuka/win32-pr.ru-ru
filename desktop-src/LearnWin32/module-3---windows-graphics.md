# <a name="module-3-windows-graphics"></a>Модуль 3. Windows Рисунка

В [модуле 1](your-first-windows-program.md) этой серии показано, как создать пустое окно. [модуль 2](module-2--using-com-in-your-windows-program.md) занял небольшое изучение модели COM, которая является основой для многих современных Windows api. Теперь пришло время добавить графику в пустое окно, созданное в модуле 1.
