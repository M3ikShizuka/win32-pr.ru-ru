# <a name="module-3-windows-graphics"></a>Модуль 3. Графика Windows

В [модуле 1](your-first-windows-program.md) этой серии показано, как создать пустое окно. [Модуль 2](module-2--using-com-in-your-windows-program.md) занял небольшое изучение модели COM, что является основой для многих современных API Windows. Теперь пришло время добавить графику в пустое окно, созданное в модуле 1.

Этот модуль начинается с высокоуровневого обзора архитектуры графики Windows. Затем мы рассмотрим Direct2D — мощный графический API, который появился в Windows 7.
