# <a name="pbm_setstep-message"></a>\_Сообщение СЕТСТЕП PBM

Задает шаг приращения для индикатора выполнения. Шаг приращения — это величина, на которую индикатор выполнения увеличивает текущее местоположение при получении сообщения [**\_ степит PBM**](pbm-stepit.md) . По умолчанию шаг с шагом равен 10.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Шаг приращения нового шага.

</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает шаг предыдущего шага.
