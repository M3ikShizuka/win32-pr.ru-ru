# <a name="sb_getborders-message"></a>Сообщение SB с \_ ВЫграничными границами

Извлекает текущие значения ширины горизонтальных и вертикальных границ окна состояния.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на целочисленный массив с тремя элементами. Первый элемент получает ширину горизонтальной границы, второй получает ширину вертикальной границы, а третья — ширину границы между прямоугольниками.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если успешно, или **false** в противном случае.

## <a name="remarks"></a>Комментарии

Границы определяют интервал между внешним краями окна и прямоугольниками в окне, содержащими текст. Кроме того, границы определяют интервал между прямоугольниками.
