# <a name="lb_getcursel-message"></a>\_Сообщение с НЕрекурсивной балансировкой нагрузки

Возвращает индекс выбранного в данный момент элемента, если он имеется, в списке с единственным выбором.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

В списке с единственным выбором возвращаемое значение — это Отсчитываемый от нуля индекс текущего выбранного элемента. Если выбор отсутствует, возвращается значение фунтов \_ Err.

## <a name="remarks"></a>Remarks

Чтобы получить индексы выбранных элементов в списке с множественным выбором, используйте сообщение [**\_ жетселитемс балансировки нагрузки**](lb-getselitems.md) . Чтобы определить, выбран ли элемент, имеющий прямоугольник фокуса в списке множественного выбора, используйте сообщение [**\_ жетсел балансировки нагрузки**](lb-getsel.md) .

При отправке в список с множественным выбором метод **балансировки нагрузки \_** возвращает индекс элемента, имеющего прямоугольник фокуса. Если элементы не выбраны, возвращается ноль.
