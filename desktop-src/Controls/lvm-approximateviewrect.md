# <a name="lvm_approximateviewrect-message"></a>\_Сообщение LVM аппроксиматевиеврект

Вычисляет приблизительную ширину и высоту, необходимые для вывода заданного числа элементов. Это сообщение можно отправить явным образом или использовать макрос [**\_ аппроксиматевиеврект ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_approximateviewrect) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Число элементов, отображаемых в элементе управления. Если этот параметр имеет значение-1, то в сообщении используется общее число элементов в элементе управления.

</dd> <dt>

*lParam* 
</dt> <dd>

[**Ловорд**](/previous-versions/windows/desktop/legacy/ms632659(v=vs.85)) — это предлагаемое по оси x измерение элемента управления в пикселях. Этот параметр можно установить в значение-1, чтобы разрешить сообщению использовать текущее значение ширины.

[**HIWORD**](/previous-versions/windows/desktop/legacy/ms632657(v=vs.85)) — это предлагаемое по оси y измерение элемента управления (в пикселях). Этот параметр можно установить в значение-1, чтобы разрешить сообщению использовать текущее значение высоты.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **типа DWORD** , которое содержит приблизительную ширину (в [**ловорд**](/previous-versions/windows/desktop/legacy/ms632659(v=vs.85))) и высоту (в [**HIWORD**](/previous-versions/windows/desktop/legacy/ms632657(v=vs.85))), необходимую для вывода элементов в пикселях.

## <a name="remarks"></a>Remarks

Установка размера элемента управления "представление списка" на основе измерений, предоставляемых этим сообщением, может оптимизировать перерисовку и уменьшить мерцание.
