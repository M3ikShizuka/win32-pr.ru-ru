# <a name="mcm_hittest-message"></a>\_Сообщение MCM HITTEST

Определяет, какая часть элемента управления "календарь месяца" находится в заданной точке экрана. Это сообщение можно отправить явным образом или с помощью макроса [**монскал \_ HitTest**](/windows/desktop/api/Commctrl/nf-commctrl-monthcal_hittest) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**мчиттестинфо**](/windows/desktop/api/Commctrl/ns-commctrl-mchittestinfo) . При отправке сообщения элементу **кбсизе** должен быть присвоен размер структуры **мчиттестинфо** , а в параметре **PT** — точка, которую нужно проверить.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Задает значения в элементах



| Код возврата                                                                                           | Описание                                                                                                                                                                                                                                                               |
|-------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**МЧТ \_ Календарь**</dt> </dl>         | Заданная точка находилась в календаре.<br/>                                                                                                                                                                                                                       |
| <dl> <dt>**МЧТ \_ календарбк**</dt> </dl>       | Заданная точка находилась в фоновом режиме календаря.<br/>                                                                                                                                                                                                              |
| <dl> <dt>**МЧТ \_ календардате**</dt> </dl>     | Заданная точка находилась в определенной дате в календаре. В структуре [**SYSTEMTIME**](/windows/desktop/api/minwinbase/ns-minwinbase-systemtime) в отношении *lParam->St* задается Дата в заданной точке.<br/>                                                                                    |
| <dl> <dt>**МЧТ \_ календардатенекст**</dt> </dl> | Заданная точка была поверх даты следующего месяца (частично отображается в конце отображаемого в данный момент месяца). Если пользователь щелкнет здесь, то месячный календарь прокручивает свой экран к следующему месяцу или набору месяцев.<br/>                                 |
| <dl> <dt>**МЧТ \_ календардатепрев**</dt> </dl> | Заданная точка была на дату за предыдущий месяц (частично отображается в конце отображаемого в данный момент месяца). Если пользователь щелкнет здесь, то месячный календарь прокручивает свой экран к предыдущему месяцу или набору месяцев.<br/>                         |
| <dl> <dt>**МЧТ \_ CALENDARDAY**</dt> </dl>      | Заданная точка была в более чем день (например, «пятница»). Для структуры [**SYSTEMTIME**](/windows/desktop/api/minwinbase/ns-minwinbase-systemtime) в параметре *lParam->St* задается соответствующая дата в верхней строке.<br/>                                                                      |
| <dl> <dt>**МЧТ \_ календарвикнум**</dt> </dl>  | Заданная точка находилась на номер недели (только для стиля [**\_ викнумберс MCS**](month-calendar-control-styles.md) ). Для структуры [**SYSTEMTIME**](/windows/desktop/api/minwinbase/ns-minwinbase-systemtime) в параметре *lParam->St* задается соответствующая дата в крайнем левом столбце.<br/> |
| <dl> <dt>**МЧТ \_ Далее**</dt> </dl>             | Заданная точка находится в области, которая вызывает прокрутку календаря в месяц до следующего месяца или набора месяцев. Этот флаг используется для изменения других флагов проверки нажатия.<br/>                                                                                   |
| <dl> <dt>**МЧТ \_ нигде**</dt> </dl>          | Заданная точка не находилась в элементе управления ' календарь на месяц ' или в неактивной части элемента управления.<br/>                                                                                                                                                        |
| <dl> <dt>**МЧТ \_ назад**</dt> </dl>             | Заданная точка находится в области, которая приведет к прокрутке календаря в течение месяца до предыдущего месяца или набора месяцев. Этот флаг используется для изменения других флагов проверки нажатия.<br/>                                                                               |
| <dl> <dt>**\_заголовок МЧТ**</dt> </dl>            | Заданная точка находилась в названии месяца.<br/>                                                                                                                                                                                                                      |
| <dl> <dt>**МЧТ \_ титлебк**</dt> </dl>          | Заданная точка находилась на фоне названия месяца.<br/>                                                                                                                                                                                                    |
| <dl> <dt>**МЧТ \_ титлебтннекст**</dt> </dl>     | Заданная точка находилась над кнопкой в правом верхнем углу элемента управления. Если пользователь щелкнет здесь, то месячный календарь прокручивает свой экран к следующему месяцу или набору месяцев. <br/>                                                                           |
| <dl> <dt>**МЧТ \_ титлебтнпрев**</dt> </dl>     | Заданная точка находилась над кнопкой в левом верхнем углу элемента управления. Если пользователь щелкнет здесь, то месячный календарь прокручивает свой экран к предыдущему месяцу или набору месяцев. <br/>                                                                        |
| <dl> <dt>**МЧТ \_ титлемонс**</dt> </dl>       | Заданная точка находилась в строке названия месяца, а не в названии месяца.<br/>                                                                                                                                                                                                 |
| <dl> <dt>**МЧТ \_ титлэйеар**</dt> </dl>        | Заданная точка находилась в строке заголовка месяца, над значением года.<br/>                                                                                                                                                                                               |
| <dl> <dt>**МЧТ \_ тодайлинк**</dt> </dl>        | Заданная точка находилась на ссылке "сегодня" в нижней части элемента управления "месячный календарь".<br/> Элемент **ухит** структуры [**мчиттестинфо**](/windows/desktop/api/Commctrl/ns-commctrl-mchittestinfo) в параметре *lParam* будет равен возвращаемому значению. <br/>                                          |
