# <a name="list-view-item-states"></a>Состояния элементов List-View

Значение состояния элемента состоит из состояния элемента, необязательного индекса маски оверлея и необязательного индекса маски изображения состояния.

Состояние элемента определяет его внешний вид и функциональность. Состояние может быть равно нулю или иметь одно или несколько из следующих значений:



| Константа                                                                                                                                                                        | Описание                                                                                                                                                          |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="LVIS_ACTIVATING"></span><span id="lvis_activating"></span><dl> <dt>**\_Активация лвис**</dt> </dl>             | В настоящее время не поддерживается.<br/>                                                                                                                                  |
| <span id="LVIS_CUT"></span><span id="lvis_cut"></span><dl> <dt>**ЛВИС \_ Вырезать**</dt> </dl>                                  | Элемент помечен для операции вырезания и вставки.<br/>                                                                                                         |
| <span id="LVIS_DROPHILITED"></span><span id="lvis_drophilited"></span><dl> <dt>**ЛВИС \_ дрофилитед**</dt> </dl>          | Элемент выделяется как цель перетаскивания.<br/>                                                                                                        |
| <span id="LVIS_FOCUSED"></span><span id="lvis_focused"></span><dl> <dt>**ЛВИС \_**</dt> </dl>                      | Элемент имеет фокус, поэтому он окружается стандартным прямоугольником фокуса. Хотя можно выбрать несколько элементов, фокус может быть только одного элемента.<br/> |
| <span id="LVIS_OVERLAYMASK"></span><span id="lvis_overlaymask"></span><dl> <dt>**ЛВИС \_ оверлаймаск**</dt> </dl>          | Индекс изображения оверлея элемента извлекается маской.<br/>                                                                                                    |
| <span id="LVIS_SELECTED"></span><span id="lvis_selected"></span><dl> <dt>**ЛВИС \_ выбрано**</dt> </dl>                   | Элемент выбран. Внешний вид выбранного элемента зависит от того, имеет ли он фокус, а также от системных цветов, используемых для выбора.<br/>             |
| <span id="LVIS_STATEIMAGEMASK"></span><span id="lvis_stateimagemask"></span><dl> <dt>**ЛВИС \_ статеимажемаск**</dt> </dl> | Индекс изображения состояния элемента извлекается маской.<br/>                                                                                                      |



## <a name="remarks"></a>Комментарии

С помощью маски **\_ оверлаймаск лвис** можно изолировать Отсчитываемый от единицы индекс изображения оверлея. С помощью маски **\_ статеимажемаск лвис** можно изолировать Отсчитываемый от единицы индекс изображения состояния.
