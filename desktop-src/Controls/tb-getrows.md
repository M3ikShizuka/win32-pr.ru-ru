# <a name="tb_getrows-message"></a>Сообщение о GETROWSх (ТБ) \_

Извлекает количество строк кнопок на панели инструментов с [**тбстиле \_**](toolbar-control-and-button-styles.md) стилем.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает число строк.
