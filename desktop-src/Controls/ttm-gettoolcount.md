# <a name="ttm_gettoolcount-message"></a>\_Сообщение ТТМ жеттулкаунт

Извлекает число инструментов, поддерживаемых элементом управления ToolTip.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает число инструментов.
