# <a name="tvn_selchanged-notification-code"></a>\_Код уведомления ТВН селчанжед

Сообщает родительскому окну элемента управления древовидного представления о том, что выделение изменилось с одного элемента на другой. Этот код уведомления отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
TVN_SELCHANGED 

    pnmtv = (LPNMTREEVIEW) lParam 
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмтривиев**](/windows/win32/api/commctrl/ns-commctrl-nmtreeviewa) . Элементы **итемолд** и **итемнев** структуры **нмтривиев** представляют собой [**твитем**](/windows/win32/api/commctrl/ns-commctrl-tvitema) структуры, содержащие сведения о ранее выбранном элементе и вновь выбранном элементе. Допустимы только члены **масок**, **хитем**, **State** и **lParam** этих структур. Элементы **статемаск** структур **твитем** , заданные с помощью **итемолд** и **итемнев** , не определены на входе. Элемент **Action** структуры **нмтривиев** указывает тип действия, вызвавшего изменение выбора. Может иметь одно из следующих значений.



| Требование | Значение |
|-----------------|-------------------|
| ТВК \_ бикэйбоард | Нажатием клавиши.   |
| ТВК \_ бимаусе    | Щелчком мыши. |
| ТВК \_ неизвестный    | Неизвестно.          |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение игнорируется.
