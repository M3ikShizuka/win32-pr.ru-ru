# <a name="lvm_enablegroupview-message"></a>\_Сообщение LVM енаблеграупвиев

Включает или отключает отображение элементов в элементе управления "представление списка" в виде группы.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>**Логическое** значение, указывающее, следует ли включить элемент управления "представление списка" для группирования отображаемых элементов. Чтобы включить группирование, используйте **значение true** , а для отключения — **значение false** . </dd> <dt>

*lParam* 
</dt> <dd>Должен иметь **значение NULL**.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из следующих значений.



| Код возврата                                                                       | Описание                                                                                  |
|-----------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| <dl> <dt>**0**</dt> </dl>  | Возможность отображения элементов списка в виде группы уже включена или отключена.<br/> |
| <dl> <dt>**1**</dt> </dl>  | Состояние элемента управления успешно изменено.<br/>                                |
| <dl> <dt>**-1**</dt> </dl> | Операция не удалась.<br/>                                                             |



 

## <a name="remarks"></a>Комментарии

**LVM \_ ЕНАБЛЕГРАУПВИЕВ** не поддерживается в стиле [**LVS \_ овнердата**](list-view-window-styles.md) .

> [!Note]  
> Чтобы использовать это сообщение, необходимо указать манифест, указывающий Comclt32.dll версии 6,0. Дополнительные сведения о манифестах см. в разделе [Включение визуальных стилей](cookbook-overview.md).
