# <a name="dl_dragging-notification-code"></a>\_Код уведомления для перетаскивания списка рассылки

Сигнализирует, что пользователь переместил указатель мыши при перетаскивании элемента. \_Перетаскивание DL также периодически отправляется во время перетаскивания, даже если мышь не перемещена. Поле со списком перетаскивания отправляет этот код уведомления в его родительское окно в виде сообщения списка перетаскивания. Дополнительные сведения см. в статье [перетаскивание сообщений в поле со списком](about-list-boxes.md).


```C++
DL_DRAGGING

    pDragInfo = (LPARAM)(LPDRAGLISTINFO) lParam; 
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Идентификатор элемента управления поля со списком перетаскивания.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**драглистинфо**](/windows/win32/api/commctrl/ns-commctrl-draglistinfo) , содержащую \_ код уведомления для перетаскивания списка рассылки, маркер в поле со списком перетаскивания и позицию курсора.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение определяет тип курсора мыши, который должен быть установлен в списке перетаскивания. Это может быть значение списка рассылки \_ стопкурсор, DL \_ КОПИКУРСОР или DL \_ мовекурсор. Если возвращается любое другое значение, курсор не изменяется.

## <a name="remarks"></a>Remarks

Как правило, процедура окна обрабатывает \_ код уведомления для перетаскивания списка рассылки, определяя элемент под курсором и вырисуя значок вставки. Чтобы получить элемент под курсором, используйте функцию [**лбитемфромпт**](/windows/desktop/api/Commctrl/nf-commctrl-lbitemfrompt) , указав **значение true** для параметра *баутоскролл* . Этот параметр приводит к периодической прокрутке окна списка перетаскивания, если курсор находится выше или ниже области клиента. Чтобы нарисовать значок вставки, используйте функцию [**дравинсерт**](/windows/desktop/api/Commctrl/nf-commctrl-drawinsert) .
