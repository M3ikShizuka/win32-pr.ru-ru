# <a name="pbm_deltapos-message"></a>\_Сообщение ДЕЛТАПОС PBM

Изменяет текущую позицию индикатора выполнения на заданное приращение и перерисовывает линию, чтобы отразить новую позицию.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Величина, на которую должно быть перемещено расположение.

</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает предыдущее расположение.

## <a name="remarks"></a>Комментарии

Если приращение приводит к значению, расположенному за пределами диапазона элемента управления, то устанавливается равным ближайшей границе.

Поведение этого сообщения не определено, если оно отправляется элементу управления с стилем [**\_ области PBS**](progress-bar-control-styles.md) .
