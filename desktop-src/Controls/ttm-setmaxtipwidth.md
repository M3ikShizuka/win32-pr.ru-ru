# <a name="ttm_setmaxtipwidth-message"></a>\_Сообщение ТТМ сетмакстипвидс

Задает максимальную ширину окна подсказки.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>

Максимальная ширина окна подсказки, или значение-1, чтобы разрешить любую ширину.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает предыдущую максимальную ширину всплывающей подсказки.

## <a name="remarks"></a>Remarks

Значение максимальной ширины не указывает фактическую ширину окна подсказки. Вместо этого, если строка подсказки превышает максимальную ширину, элемент управления разбивает текст на несколько строк, используя пробелы для определения разрывов строк. Если текст не может быть разбит на несколько строк, он отображается в одной строке, что может превысить максимальную ширину подсказки.
