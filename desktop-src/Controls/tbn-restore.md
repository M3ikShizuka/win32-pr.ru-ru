# <a name="tbn_restore-notification-code"></a>\_Код уведомления о восстановлении ТБН

Сообщает родительскому окну панели инструментов, что панель инструментов находится в процессе восстановления. Этот код уведомления отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
TBN_RESTORE 

    lpnmtb = (LPNMTBRESTORE) lParam; 
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмтбресторе**](/windows/win32/api/commctrl/ns-commctrl-nmtbrestore) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Приложение должно вернуть ноль в ответ на первый ТБН код уведомления о **\_ восстановлении** , полученный в начале процесса восстановления, чтобы продолжить восстановление сведений о кнопке. Если приложение возвращает ненулевое значение, процесс восстановления отменяется.

## <a name="remarks"></a>Remarks

Приложение получит этот код уведомления один раз в начале процесса восстановления и один раз для каждой кнопки. Этот код уведомления позволяет извлекать сведения из потока данных, сохраненного ранее. Если вы не сохранили какую бы то ни было информации, пропустите код уведомления. Дополнительные сведения об обработке **ТБН \_ RESTORE** см. в статье [Настройка панелей инструментов](toolbar-controls-overview.md) .
