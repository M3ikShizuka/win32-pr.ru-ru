# <a name="sb_setparts-message"></a>\_Сообщение SB сетпартс

Задает количество частей в окне состояния и координату правого края каждой части.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Число устанавливаемых частей (не может быть больше 256).

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на целочисленный массив. Число элементов указывается в параметре *wParam*. Каждый элемент задает расположение в клиентских координатах правого края соответствующей части. Если элемент имеет значение-1, правый край соответствующей части расширяется на границу окна.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если успешно, или **false** в противном случае.
