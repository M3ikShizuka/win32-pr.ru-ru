# <a name="lvm_getview-message"></a>\_Сообщение LVM

Извлекает текущее представление элемента управления "представление списка".

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **типа DWORD** , указывающее текущее представление.

## <a name="remarks"></a>Комментарии

Ниже приведены значения для представлений.

-   \_ \_ сведения о представлении LV
-   \_значок представления "Латвия" \_
-   \_список ПРЕДСТАВЛЕНИЙ \_ LV
-   \_представление LV \_ маленькие значки
-   \_плитка представления "Латвия" \_

> [!Note]  
> Чтобы использовать это сообщение, необходимо указать манифест, указывающий Comclt32.dll версии 6,0. Дополнительные сведения о манифестах см. в разделе [Включение визуальных стилей](cookbook-overview.md).
