# <a name="em_setlangoptions-message"></a>\_Сообщение СЕТЛАНГОПТИОНС EM

Задает параметры для редактора методов ввода (IME) и поддержки восточно-азиатских языков в элементе управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется; оно должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Задает параметры языка. Список возможных значений см. в разделе [**EM \_ жетлангоптионс**](em-getlangoptions.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Это сообщение возвращает значение 1.

## <a name="remarks"></a>Комментарии

Сообщение **\_ сетлангоптионс EM** управляет следующими элементами:

-   Автоматическая привязка шрифтов.
-   Автоматическое переключение клавиатуры.
-   Автоматическая настройка размера шрифта.
-   Использование шрифтов по умолчанию пользовательского интерфейса вместо шрифтов по умолчанию для документов.
-   Уведомления для клиента во время составления IME.
-   Как редактор IME прерывает режим композиции.
-   Проверка орфографии, Автозамена и прогнозирование сенсорной клавиатуры.

В этом сообщении задаются значения всех флагов параметров языка. Чтобы изменить подмножество флагов, отправьте сообщение [**\_ жетлангоптионс EM**](em-getlangoptions.md) для получения текущих флагов параметров, измените флаги, которые необходимо изменить, а затем отправьте сообщение **EM \_ сетлангоптионс** с результатом.
