# <a name="em_linefromchar-message"></a>\_Сообщение ЛИНЕФРОМЧАР EM

Возвращает индекс строки, содержащей указанный индекс символа в многострочном элементе управления Edit. Индекс символа — это Отсчитываемый от нуля индекс символа, начиная с начала элемента управления "поле ввода". Это сообщение можно отправить либо в элемент управления "поле ввода", либо в элемент управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Индекс символа, содержащегося в строке, номер которой должен быть получен. Если этот параметр имеет значение-1, **EM \_ линефромчар** получает номер строки текущей строки (строка, содержащая курсор) или, если имеется выделенный фрагмент, номер строки, содержащей начало выделения.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение — номер строки, начинающейся с нуля, с индексом символа, указанным параметром *wParam*.

## <a name="remarks"></a>Комментарии

**Расширенное редактирование:** Поддерживается в Microsoft Rich Edit 1,0 и более поздних версиях. Если индекс символа больше 64 000, используйте сообщение [**EM \_ екслинефромчар**](em-exlinefromchar.md) . Дополнительные сведения о совместимости расширенных версий редактирования с различными версиями системы см. в разделе [Общие сведения об элементах управления редактированием](about-rich-edit-controls.md).
