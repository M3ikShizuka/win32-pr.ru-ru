# <a name="em_setwordbreakproc-message"></a>\_Сообщение СЕТВОРДБРЕАКПРОК EM

Заменяет функцию переносить по умолчанию элемента управления "Изменить" на определяемую приложением функцию переноса слов. Это сообщение можно отправить либо в элемент управления "поле ввода", либо в элемент управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется.

</dd> <dt>

*lParam* 
</dt> <dd>

Адрес определяемой приложением функции WordWrap. Дополнительные сведения о разбиении строк см. в описании функции обратного вызова [*едитвордбреакпрок*](/windows/win32/api/winuser/nc-winuser-editwordbreakproca) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Это сообщение не возвращает значение.

## <a name="remarks"></a>Remarks

Функция WordWrap сканирует текстовый буфер, который содержит текст, отправляемый на экран, и ищет первое слово, не помещается на текущей линии экрана. Функция WordWrap помещает это слово в начало следующей строки на экране.

Функция WordWrap определяет точку, в которой система должна разбивать строку текста для элементов управления многострочного редактирования, обычно в виде символа пробела, разделяющего два слова. Однострочный или однострочный элемент управления "поле ввода" может вызывать эту функцию, когда пользователь нажимает клавиши со стрелками в сочетании с клавишей CTRL для перемещения курсора к следующему слову или предыдущему слову. Функция WordWrap по умолчанию разбивает строку текста по символу пробела. Определяемая приложением функция может определять переносить дефис или символ, отличный от символа пробела.

**Расширенное редактирование:** Поддерживается в Microsoft Rich Edit 1,0 и более поздних версиях. Дополнительные сведения о совместимости расширенных версий редактирования с различными версиями системы см. в разделе [Общие сведения об элементах управления редактированием](about-rich-edit-controls.md).
