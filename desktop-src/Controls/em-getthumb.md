# <a name="em_getthumb-message"></a>\_Сообщение EM

Возвращает расположение бегунка (Thumb) в вертикальной полосе прокрутки для многострочного элемента управления "поле ввода". Это сообщение можно отправить либо в элемент управления "поле ввода", либо в элемент управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение является положением ползунка.

## <a name="remarks"></a>Remarks

**Расширенное редактирование:** Поддерживается в Microsoft Rich Edit 2,0 и более поздних версиях. Дополнительные сведения о совместимости расширенных версий редактирования с различными версиями системы см. в разделе [Общие сведения об элементах управления редактированием](about-rich-edit-controls.md).
