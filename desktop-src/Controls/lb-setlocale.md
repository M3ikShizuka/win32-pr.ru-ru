# <a name="lb_setlocale-message"></a>Сообщение ФУНТОВой \_ SETLOCALE

Задает текущий языковой стандарт для списка. Языковой стандарт можно использовать для определения правильного порядка сортировки отображаемого текста (для списков с стилем [**\_ сортировки фунтов**](list-box-styles.md) ) и текста, добавленного сообщением [**\_ ADDSTRING балансировки нагрузки**](lb-addstring.md) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Указывает код локали, который будет использоваться списком для сортировки при добавлении текста.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение — это предыдущий идентификатор локали. Если параметр *wParam* задает языковой стандарт, который не установлен в системе, возвращается значение фунтов \_ Err, а текущий языковой стандарт списка не изменяется.

## <a name="remarks"></a>Remarks

Используйте макрос [**макелЦид**](/windows/desktop/api/winnt/nf-winnt-makelcid) для создания идентификатора локали.
