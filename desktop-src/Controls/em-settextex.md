# <a name="em_settextex-message"></a>\_Сообщение СЕТТЕКСТЕКС EM

Объединяет функциональные возможности сообщений [**WM \_ SETTEXT**](/windows/desktop/winmsg/wm-settext) и [**EM \_ реплацесел**](em-replacesel.md) и добавляет возможность задать текст с помощью кодовой страницы и использовать форматированный текст или обычный текст.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Указатель на структуру [**сеттекстекс**](/windows/desktop/api/Richedit/ns-richedit-settextex) , которая указывает флаги и дополнительную кодовую страницу для использования при преобразовании в Юникод.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на текст, заканчивающийся нулем, для вставки. Этот текст является строкой ANSI, если только кодовая страница не 1200 (Unicode). Если параметр *lParam* начинается с допустимой ASCII-последовательности RTF (например, "{ \\ RTF" или "{уртф"), текст считывается в с помощью средства чтения RTF.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если операция устанавливает весь текст и выполняется удачно, возвращается значение 1.

Если операция устанавливает выделение и выполняется, возвращаемое значение представляет собой число скопированных байтов или символов.

Если операция завершается ошибкой, возвращаемое значение равно нулю.
