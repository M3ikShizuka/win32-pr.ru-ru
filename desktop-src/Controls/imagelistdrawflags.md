# <a name="imagelistdrawflags"></a>имажелистдравфлагс

Передается в метод [**IImageList::D RAW**](/windows/desktop/api/CommonControls/nf-commoncontrols-iimagelist-draw) в элементе **фстиле** элемента [**имажелистдравпарамс**](/windows/win32/api/commctrl/ns-commctrl-imagelistdrawparams).



| Константа/значение                                                                                                                                                                                                                            | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="ILD_NORMAL"></span><span id="ild_normal"></span><dl> <dt>**Ку \_ Обычная**</dt> <dt>0x00000000</dt> </dl>                      | Рисует изображение, используя цвет фона для списка изображений. Если цвет фона равен \_ значению CLR None, изображение прозрачно рисуется с помощью маски.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="ILD_TRANSPARENT"></span><span id="ild_transparent"></span><dl> <dt>**Ку \_ ПРОЗРАЧное**</dt> <dt>0x00000001</dt> </dl>       | Рисует изображение прозрачно с помощью маски независимо от цвета фона. Это значение не действует, если список изображений не содержит маску.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="ILD_BLEND25"></span><span id="ild_blend25"></span><dl> <dt>**Ку \_ BLEND25**</dt> <dt>0x00000002</dt> </dl>                   | Рисует изображение, смешивание 25% с цветом смешения, заданным в **ргбфг**. Это значение не действует, если список изображений не содержит маску.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="ILD_FOCUS"></span><span id="ild_focus"></span><dl> <dt>**Ку \_ Фокусировка**</dt> <dt>0x00000002</dt> </dl>                         | То же, что и **ку \_ BLEND25**.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="ILD_BLEND50"></span><span id="ild_blend50"></span><dl> <dt>**Ку \_ BLEND50**</dt> <dt>0x00000004</dt> </dl>                   | Рисует изображение, наложение 50 процента на цвет смешения, заданный параметром **ргбфг**. Это значение не действует, если список изображений не содержит маску.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="ILD_SELECTED"></span><span id="ild_selected"></span><dl> <dt>**Ку \_ ВЫБРАНо**</dt> <dt>0x00000004</dt> </dl>                | То же, что и **ку \_ BLEND50**.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="ILD_BLEND"></span><span id="ild_blend"></span><dl> <dt>**Ку \_ BLEND**</dt> <dt>0x00000004</dt> </dl>                         | То же, что и **ку \_ BLEND50**.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="ILD_MASK"></span><span id="ild_mask"></span><dl> <dt>**Ку \_ Маска**</dt> <dt>0x00000010</dt> </dl>                            | Рисует маску.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="ILD_IMAGE"></span><span id="ild_image"></span><dl> <dt>**Ку \_ ОБРАЗ**</dt> <dt>0x00000020</dt> </dl>                         | Если наложение не требует прорисовки маски, установите этот флаг.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| <span id="ILD_ROP"></span><span id="ild_rop"></span><dl> <dt>**Ку \_ ВЕРХНЕМ**</dt> <dt>0x00000040</dt> </dl>                               | Рисует изображение с помощью растрового кода операции, заданного элементом **двроп** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| <span id="ILD_OVERLAYMASK"></span><span id="ild_overlaymask"></span><dl> <dt>**Ку \_ ОВЕРЛАЙМАСК**</dt> <dt>0x00000F00</dt> </dl>       | Чтобы извлечь изображение оверлея из члена **фстиле** , используйте логическое и, чтобы объединить **Фстиле** со значением **ку \_ оверлаймаск** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="ILD_PRESERVEALPHA"></span><span id="ild_preservealpha"></span><dl> <dt>**Ку \_ ПРЕСЕРВЕАЛФА**</dt> <dt>0x00001000</dt> </dl> | Сохраняет альфа-канал в месте назначения.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="ILD_SCALE"></span><span id="ild_scale"></span><dl> <dt>**Ку \_ МАСШТАБИРОВАНие**</dt> <dt>0x00002000</dt> </dl>                         | Приводит к тому, что изображение будет масштабироваться до версии CX, CY вместо усечения.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| <span id="ILD_DPISCALE"></span><span id="ild_dpiscale"></span><dl> <dt>**Ку \_ ДПИСКАЛЕ**</dt> <dt>0x00004000</dt> </dl>                | Масштабирует изображение до текущего масштаба отображения.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="ILD_ASYNC"></span><span id="ild_async"></span><dl> <dt>**Ку \_ АСИНХРОНный**</dt> <dt>0x00008000</dt> </dl>                         | **Windows Vista и более поздних версий.** Нарисуйте изображение, если оно доступно в кэше. Не извлекайте его автоматически. Вызываемый метод Draw возвращает «E» в \_ ожидании вызывающему компоненту, который затем должен выполнить альтернативное действие, например, предоставить другое изображение и поставить в очередь фоновую задачу, чтобы принудительно загрузить изображение через [**форцеимажепресент**](/windows/desktop/api/Commoncontrols/nf-commoncontrols-iimagelist2-forceimagepresent) с помощью \_ флага илфип Always. \_Затем флаг Async ку предотвращает блокировку текущего потока операцией извлечения и особенно важна, если метод Draw вызывается из потока пользовательского интерфейса.<br/> |
