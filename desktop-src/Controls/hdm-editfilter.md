# <a name="hdm_editfilter-message"></a>\_Сообщение ЕДИТФИЛТЕР HDM

Перемещает фокус ввода в поле ввода, когда фокус находится на кнопке фильтра.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Значение типа, указывающее Изменяемый столбец.

</dd> <dt>

*lParam* 
</dt> <dd>

Флаг, указывающий способ управления изменениями пользователя. Используйте этот флаг, чтобы указать, что делать, если пользователь находится в процессе редактирования фильтра при отправке сообщения.



| Значение                                                                                                                                      | Значение                                           |
|--------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------|
| <dl> <dt></dt><dt> **Значение true**</dt> </dl>  | Отмена изменений, внесенных пользователем. <br/> |
| <dl> <dt></dt><dt> **Значение false**</dt> </dl> | Примите изменения, внесенные пользователем. <br/>  |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает целое число. **LResult** преобразуется в целое число, которое указывает **true**(1) или **false**(0).
