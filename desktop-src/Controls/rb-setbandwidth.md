# <a name="rb_setbandwidth-message"></a>\_Сообщение СЕТБАНДВИДС RB

Задает ширину закрепленной полосы.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Индекс полосы.

</dd> <dt>

*lParam* 
</dt> <dd>

Новая ширина в пикселях.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **true** , если значение было задано, и **false** в противном случае.
