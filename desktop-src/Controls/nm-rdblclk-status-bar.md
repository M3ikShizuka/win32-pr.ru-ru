# <a name="nm_rdblclk-status-bar-notification-code"></a>\_Код уведомления NM рдблклк (строка состояния)

Уведомляет родительские окна элемента управления "строка состояния", что пользователь дважды щелкнул правой кнопкой мыши внутри элемента управления. Этот код уведомления отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
NM_RDBLCLK

    lpnm = (LPNMMOUSE) lParam;
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нммаусе**](/windows/win32/api/commctrl/ns-commctrl-nmmouse) , содержащую дополнительные сведения об этом уведомлении. Элемент **двитемспек** содержит индекс раздела, который был выбран.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , чтобы указать, что нажатие кнопки мыши было обработано, и подавить обработку по умолчанию системой. Возвращает **значение false** , чтобы разрешить обработку по умолчанию щелчка.
