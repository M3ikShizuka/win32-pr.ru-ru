# <a name="rebar-control-styles"></a>Стили элемента управления "Главная панель"

Элементы управления "Главная панель" поддерживают разнообразные стили элементов управления в дополнение к стандартным стилям окна.



| Константа                                                                                                                                                                        | Описание                                                                                                                                                                                                                                                                                                                                                 |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="RBS_AUTOSIZE"></span><span id="rbs_autosize"></span><dl> <dt>**Автоподбор RBS \_**</dt> </dl>                      | [Версия 4,71](common-control-versions.md). Элемент управления "Главная панель" будет автоматически изменять макет полос при изменении размера или расположения элемента управления. В этом случае будет отправлено уведомление [ \_ AUTOSIZE РБН](rbn-autosize.md) .<br/>                                                                                              |
| <span id="RBS_BANDBORDERS"></span><span id="rbs_bandborders"></span><dl> <dt>**\_БАНДБОРДЕРС RBS**</dt> </dl>             | [Версия 4,71](common-control-versions.md). Элемент управления "Главная панель" отображает узкие строки для разделения смежных полос.<br/>                                                                                                                                                                                                                                 |
| <span id="RBS_DBLCLKTOGGLE"></span><span id="rbs_dblclktoggle"></span><dl> <dt>**\_ДБЛКЛКТОГГЛЕ RBS**</dt> </dl>          | [Версия 4,71](common-control-versions.md). Полоса главных панелей будет переключаться в развернутое или уменьшенное состояние, когда пользователь дважды щелкает полосу. Без этого стиля состояние развернутого или минимального состояния переключается, когда пользователь выполняет однократный щелчок на полосе.<br/>                                                                                          |
| <span id="RBS_FIXEDORDER"></span><span id="rbs_fixedorder"></span><dl> <dt>**\_ФИКСЕДОРДЕР RBS**</dt> </dl>                | [Версия 4,70](common-control-versions.md). Элемент управления "Главная панель" всегда отображает полосы в том же порядке. Можно перемещать полосы в разные строки, но порядок полос является статическим.<br/>                                                                                                                                                                      |
| <span id="RBS_REGISTERDROP"></span><span id="rbs_registerdrop"></span><dl> <dt>**\_РЕГИСТЕРДРОП RBS**</dt> </dl>          | [Версия 4,71](common-control-versions.md). Элемент управления "Главная панель" создает коды уведомлений [РБН \_ GetObject](rbn-getobject.md) при перетаскивании объекта на полосу в элементе управления. Чтобы получать уведомления РБН \_ GetObject, инициализируйте OLE с помощью вызова [**Олеинитиализе**](/windows/desktop/api/ole2/nf-ole2-oleinitialize) или [**CoInitialize**](/windows/desktop/api/objbase/nf-objbase-coinitialize).<br/> |
| <span id="RBS_TOOLTIPS"></span><span id="rbs_tooltips"></span><dl> <dt>**\_всплывающие подсказки RBS**</dt> </dl>                      | [Версия 4,71](common-control-versions.md). Пока не поддерживается.<br/>                                                                                                                                                                                                                                                                                  |
| <span id="RBS_VARHEIGHT"></span><span id="rbs_varheight"></span><dl> <dt>**\_ВАРХЕИГХТ RBS**</dt> </dl>                   | [Версия 4,71](common-control-versions.md). Элемент управления "Главная панель" отображает полосы с минимально необходимой высотой, если это возможно. Без этого стиля элемент управления "Главная панель" отображает все полосы с одинаковой высотой, используя высоту отображаемой полосы с максимальной точностью для определения высоты других делений.<br/>                                                   |
| <span id="RBS_VERTICALGRIPPER"></span><span id="rbs_verticalgripper"></span><dl> <dt>**\_ВЕРТИКАЛГРИППЕР RBS**</dt> </dl> | [Версия 4,71](common-control-versions.md). Захват размера будет отображаться вертикально, а не горизонтально в вертикальной панели элементов управления. Этот стиль игнорируется для элементов управления "Главная панель", не имеющих стиля " [**CCS по \_ вертикали**](common-control-styles.md) ".<br/>                                                                            |
