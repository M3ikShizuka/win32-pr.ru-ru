# <a name="lb_setcolumnwidth-message"></a>Сообщение сетколумнвидс балансировки нагрузки \_

Задает ширину (в пикселях) всех столбцов в списке с несколькими столбцами.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Задает ширину (в пикселях) всех столбцов.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Это сообщение не возвращает значение.
