# <a name="sb_gettext-message"></a>\_SMS SB

Извлекает текст из указанной части окна состояния.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Отсчитываемый от нуля индекс части, из которой требуется получить текст.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на буфер, который получает текст в виде строки, завершающейся нулем. Чтобы определить необходимый размер буфера, используйте сообщение [**SB \_ жеттекстленгс**](sb-gettextlength.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает 32-разрядное значение, состоящее из 2 16-разрядных значений. Нижнее слово указывает длину текста в символах. В верхнем слове указывается тип операции, используемой для рисования текста. Тип может иметь одно из следующих значений.



| Код возврата                                                                                    | Описание                                                                               |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| <dl> <dt>**0**</dt> </dl>               | Текст отображается с границей, которая отображается ниже плоскости окна.<br/>  |
| <dl> <dt>**SBT \_ НЕграницы**</dt> </dl>  | Текст отображается без границ.<br/>                                             |
| <dl> <dt>**SBT \_ POPOUT**</dt> </dl>     | Текст отображается с границей, которая отображается выше плоскости окна.<br/> |
| <dl> <dt>**SBT \_ RTLREADING**</dt> </dl> | Текст отображается в противоположном направлении текста в родительском окне.<br/>  |



 

## <a name="remarks"></a>Remarks

**Предупреждение системы безопасности:** Неправильное использование этого сообщения может нарушить безопасность программы. Это сообщение не позволяет получить сведения о размере буфера. Если вы используете это сообщение, сначала вызовите [**SB \_ жеттекстленгс**](sb-gettextlength.md) , чтобы получить необходимое количество символов, а затем вызовите сообщение, чтобы получить строку. Если вы ждете до **вызова \_ SB** GetText, текст может измениться, что сделает недействительной возвращаемое значение **SB \_ жеттекстленгс**. прежде чем продолжить, ознакомьтесь с [вопросами безопасности: Microsoft Windows](sec-comctls.md) .

Это сообщение возвращает не более 65 535 символов. Если длина текстовой строки превышает это значение, она усекается.

Если текст имеет \_ Тип рисования SBT овнердрав, то это сообщение возвращает 32-разрядное значение, связанное с текстом, а не тип операции и длину.

Нормальный отображаемый текст Windows слева направо (LTR). Windows могут быть *отражены* для отображения таких языков, как иврит или арабский, которые читаются справа налево. Если \_ задано значение SBT RTLREADING, строка *lParam* считывает в обратном направлении от текста в родительском окне.
