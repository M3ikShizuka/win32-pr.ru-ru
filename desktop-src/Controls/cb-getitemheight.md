# <a name="cb_getitemheight-message"></a>\_Сообщение GETITEMHEIGHT CB

Определяет высоту элементов списка или поля выбора в поле со списком.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Компонент поля со списком, высота которого необходимо получить. Этот параметр должен иметь значение-1, чтобы получить высоту поля выбора. Чтобы получить высоту элементов списка, значение должно быть равно нулю, если только поле со списком не имеет стиль [**CBS \_ овнердраввариабле**](combo-box-styles.md) . В этом случае параметр *wParam* — это Отсчитываемый от нуля индекс определенного элемента списка.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение — это высота (в пикселях) элементов списка в поле со списком. Если поле со списком имеет стиль [**CBS \_ овнердраввариабле**](combo-box-styles.md) , это высота элемента, заданного параметром *wParam* . Если параметр *wParam* равен-1, то возвращаемое значение является высотой элемента управления редактирования (или статического текста) в поле со списком. Если возникает ошибка, возвращается значение CB \_ Err.
