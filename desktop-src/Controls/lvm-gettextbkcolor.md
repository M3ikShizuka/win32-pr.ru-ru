# <a name="lvm_gettextbkcolor-message"></a>\_Сообщение LVM жеттекстбкколор

Извлекает цвет фона текста для элемента управления "представление списка". Это сообщение можно отправить явно или с помощью макроса [**\_ жеттекстбкколор ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_gettextbkcolor) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает цвет фона текста.
