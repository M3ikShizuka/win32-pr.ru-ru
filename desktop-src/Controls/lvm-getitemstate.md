# <a name="lvm_getitemstate-message"></a>\_Сообщение LVM жетитемстате

Возвращает состояние элемента представления списка. Это сообщение можно отправить явно или с помощью макроса [**\_ жетитемстате ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_getitemstate) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Индекс элемента представления списка.

</dd> <dt>

*lParam* 
</dt> <dd>

Сведения о состоянии для извлечения. Этот параметр может быть сочетанием следующих значений:



| Значение                                                                                                                                                                           | Значение                                                                                                                                                              |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="LVIS_CUT"></span><span id="lvis_cut"></span><dl> <dt>**ЛВИС \_ Вырезать**</dt> </dl>                                  | Элемент помечен для операции вырезания и вставки.<br/>                                                                                                         |
| <span id="LVIS_DROPHILITED"></span><span id="lvis_drophilited"></span><dl> <dt>**ЛВИС \_ дрофилитед**</dt> </dl>          | Элемент выделяется как цель перетаскивания.<br/>                                                                                                        |
| <span id="LVIS_FOCUSED"></span><span id="lvis_focused"></span><dl> <dt>**ЛВИС \_**</dt> </dl>                      | Элемент имеет фокус, поэтому он окружается стандартным прямоугольником фокуса. Хотя можно выбрать несколько элементов, фокус может быть только одного элемента.<br/> |
| <span id="LVIS_SELECTED"></span><span id="lvis_selected"></span><dl> <dt>**ЛВИС \_ выбрано**</dt> </dl>                   | Элемент выбран. Внешний вид выбранного элемента зависит от того, имеет ли он фокус, а также от системных цветов, используемых для выбора.<br/>             |
| <span id="LVIS_OVERLAYMASK"></span><span id="lvis_overlaymask"></span><dl> <dt>**ЛВИС \_ оверлаймаск**</dt> </dl>          | Используйте эту маску для получения индекса изображения оверлея элемента.<br/>                                                                                                 |
| <span id="LVIS_STATEIMAGEMASK"></span><span id="lvis_stateimagemask"></span><dl> <dt>**ЛВИС \_ статеимажемаск**</dt> </dl> | Используйте эту маску для получения индекса изображения состояния элемента.<br/>                                                                                                   |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает текущее состояние указанного элемента. Единственными допустимыми битами в возвращаемом значении являются те, которые соответствуют битам, заданным в параметре *lParam* .

## <a name="remarks"></a>Комментарии

Сведения о состоянии элемента включают набор битовых флагов, а также индексы списка изображений, которые указывают изображение состояния элемента и изображение оверлея.
