# <a name="cb_setcursel-message"></a>\_Сообщение СЕТКУРСЕЛ CB

Приложение отправляет сообщение **\_ сеткурсел CB** , чтобы выбрать строку в списке поля со списком. При необходимости список Прокручивает строку в представление. Текст в элементе управления "поле со списком" изменяется в соответствии с новым выделением, и все предыдущие выделения в списке удаляются.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Указывает отсчитываемый от нуля индекс строки для выбора. Если этот параметр имеет значение-1, все текущие выделенные элементы списка удаляются, а элемент управления "поле ввода" удаляется.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если сообщение прошло успешно, возвращается значение индекса выбранного элемента. Если параметр *wParam* больше числа элементов в списке или параметр *wParam* равен-1, возвращается значение CB \_ Err, а выделение снимается.
