# <a name="em_getlinecount-message-winuserh"></a>Сообщение EM_GETLINECOUNT (Winuser. h)

Возвращает число строк в многострочном элементе управления "поле ввода". Это сообщение можно отправить либо в элемент управления "поле ввода", либо в элемент управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение — это целое число, указывающее общее количество текстовых строк в элементе управления "поле ввода" или форматированном элементе управления. Если элемент управления не содержит текста, возвращается значение 1. Возвращаемое значение никогда не будет меньше 1.

## <a name="remarks"></a>Комментарии

Сообщение **EM \_ жетлинекаунт** извлекает общее количество текстовых строк, а не только число видимых в данный момент строк.

Если функция переноса слов включена, число строк может измениться при изменении размеров окна редактирования.

**Расширенное редактирование:** Поддерживается в Microsoft Rich Edit 1,0 и более поздних версиях. Дополнительные сведения о совместимости расширенных версий редактирования с различными версиями системы см. в разделе [Общие сведения об элементах управления редактированием](about-rich-edit-controls.md).
