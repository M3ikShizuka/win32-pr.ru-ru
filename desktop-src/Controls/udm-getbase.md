# <a name="udm_getbase-message"></a>\_Сообщение POLYBASE UDM

Извлекает текущий базовый основание системы счисления (то есть базовый 10 или 16) для элемента управления "вверх/вниз".

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение является текущим базовым значением.
