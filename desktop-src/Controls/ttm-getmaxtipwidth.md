# <a name="ttm_getmaxtipwidth-message"></a>\_Сообщение ТТМ жетмакстипвидс

Возвращает максимальную ширину окна подсказки.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **типа int** , представляющее максимальную ширину подсказок в пикселях. Если максимальная ширина не была задана ранее, сообщение возвращает значение-1.

## <a name="remarks"></a>Remarks

Максимальное значение ширины подсказки не указывает фактическую ширину окна подсказки. Вместо этого, если строка подсказки превышает максимальную ширину, элемент управления разбивает текст на несколько строк, используя пробелы для определения разрывов строк. Если текст не может быть разбит на несколько строк, он будет отображаться в одной строке. Длина этой строки может превысить максимальную ширину подсказки.
