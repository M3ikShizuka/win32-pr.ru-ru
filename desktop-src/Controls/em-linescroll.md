# <a name="em_linescroll-message"></a>\_Сообщение ЛИНЕСКРОЛЛ EM

Прокручивает текст в многострочном элементе управления Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

**Изменить элементы управления:** Число символов для горизонтальной прокрутки.

**Элементы управления Rich Edit:** Этот параметр не используется; оно должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Число строк для вертикальной прокрутки.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если сообщение отправляется в многострочный элемент управления Edit, возвращается значение **true**.

Если сообщение отправляется в однострочный элемент управления Edit, возвращается значение **false**.

## <a name="remarks"></a>Комментарии

Элемент управления не выполняет вертикальную прокрутку за последней строкой текста в элементе управления "поле ввода". Если текущая строка плюс число строк, заданное параметром *lParam* , превышает общее число строк в элементе управления "поле ввода", значение корректируется таким образом, что последняя строка элемента управления "поле ввода" прокручивается до верхней границы окна "Правка-элемент управления".

**Изменить элементы управления:** Сообщение **EM \_ линескролл** прокручивает текст по вертикали или горизонтали в многострочном элементе управления "поле ввода". Сообщение **EM \_ линескролл** можно использовать для прокрутки по горизонтали за последним символом любой строки.

**Расширенное редактирование:** Поддерживается в Microsoft Rich Edit 1,0 и более поздних версиях. Сообщение **EM \_ линескролл** прокручивает текст по вертикали в многострочном элементе управления Edit. Дополнительные сведения о совместимости расширенных версий редактирования с различными версиями системы см. в разделе [Общие сведения об элементах управления редактированием](about-rich-edit-controls.md).
