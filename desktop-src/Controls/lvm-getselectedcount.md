# <a name="lvm_getselectedcount-message"></a>\_Сообщение LVM жетселектедкаунт

Определяет число выбранных элементов в элементе управления "представление списка". Это сообщение можно отправить явно или с помощью макроса [**\_ жетселектедкаунт ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_getselectedcount) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает число выбранных элементов.
