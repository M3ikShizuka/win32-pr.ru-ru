# <a name="list-box-styles"></a>Стили списков

Чтобы создать список с помощью функции [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) или [**CreateWindowEx**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) , используйте класс ListBox, соответствующие константы стиля окна и следующие константы стиля для определения списка. После создания элемента управления эти стили нельзя изменить, за исключением указанных.



| Константа                                                                                                                                                                              | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="LBS_COMBOBOX"></span><span id="lbs_combobox"></span><dl> <dt>**\_поле со СПИСКОМ фунтов**</dt> </dl>                            | Уведомляет список о том, что он является частью поля со списком. Это позволяет координации между двумя элементами управления, чтобы они представлю единый пользовательский интерфейс. В поле со списком должен быть задан этот стиль. Если стиль задается любым, кроме поля со списком, то список будет независимо от того, является ли он дочерним элементом поля со списком, и в результате чего произойдет сбой.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_DISABLENOSCROLL"></span><span id="lbs_disablenoscroll"></span><dl> <dt>**ФУНТ \_ дисабленоскролл**</dt> </dl>       | Показывает отключенную горизонтальную или вертикальную полосу прокрутки, если окно списка не содержит достаточно элементов для прокрутки. Если не указать этот стиль, полоса прокрутки будет скрыта, если список не содержит достаточно элементов. Этот стиль следует использовать с стилем [**WS \_ VSCROLL**](/windows/desktop/winmsg/window-styles) или [**WS \_ HSCROLL**](/windows/desktop/winmsg/window-styles) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_EXTENDEDSEL"></span><span id="lbs_extendedsel"></span><dl> <dt>**ФУНТ \_ екстендедсел**</dt> </dl>                   | Позволяет выбирать несколько элементов с помощью клавиши SHIFT, а также сочетаний мыши или специальных клавиш.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_HASSTRINGS"></span><span id="lbs_hasstrings"></span><dl> <dt>**ФУНТ \_ хасстрингс**</dt> </dl>                      | Указывает, что список содержит элементы, состоящие из строк. В списке хранятся память и адреса строк, чтобы приложение может использовать сообщение [**фунтовой нагрузки \_**](lb-gettext.md) для получения текста для определенного элемента. По умолчанию этот стиль имеет все списки, за исключением оконных списков, созданных владельцем. Можно создать список, нарисованный владельцем, либо с этим стилем, либо без него.<br/> Для списков, рисуемых владельцем, без этого стиля, [**сообщение \_ балансировки нагрузки**](lb-gettext.md) получает значение, связанное с элементом (данными элемента). <br/>                                                                                                                                                                                                                                                                                                                                    |
| <span id="LBS_MULTICOLUMN"></span><span id="lbs_multicolumn"></span><dl> <dt>**ФУНТ из \_ МНОГОстолбцовой**</dt> </dl>                   | Указывает список с несколькими столбцами, который прокручивается по горизонтали. В списке автоматически вычисляется ширина столбцов, или приложение может задать ширину с помощью сообщения [**\_ сетколумнвидс балансировки нагрузки**](lb-setcolumnwidth.md) . Если список имеет стиль **фунта \_ овнердравфиксед** , приложение может задать ширину, когда в списке будет отправлено сообщение [**WM \_ меасуреитем**](wm-measureitem.md) . <br/> Список с **\_ многостолбцовым стилем фунта** не может прокручиваться по вертикали, он игнорирует все получаемые сообщения [**\_ VSCROLL WM**](wm-vscroll.md) . <br/> Стили **фунтов с \_ многостолбцовыми** и **фунтами \_ овнердраввариабле** не могут быть объединены. Если указаны оба значения, **фунт \_ овнердраввариабле** игнорируется.<br/>                                                                                                                                                                                |
| <span id="LBS_MULTIPLESEL"></span><span id="lbs_multiplesel"></span><dl> <dt>**ФУНТ \_ мултиплесел**</dt> </dl>                   | Включает или выключает выбор строки каждый раз, когда пользователь щелкает или дважды щелкает строку в списке. Пользователь может выбрать любое количество строк.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NODATA"></span><span id="lbs_nodata"></span><dl> <dt>**некоторая \_ фунтов**</dt> </dl>                                  | Указывает список без данных. Укажите этот стиль, если количество элементов в списке превысит 1000. Список без данных также должен иметь стиль **фунта \_ овнердравфиксед** , но не должен иметь стиль **\_ хасстрингс** **фунта \_** или фунта. <br/> Список без данных напоминает список, нарисованный владельцем, за исключением того, что он не содержит строк или растровых данных для элемента. Команды для добавления, вставки или удаления элемента всегда игнорируют все заданные элементы. запросы на поиск строки в списке всегда завершаются ошибкой. Система отправляет сообщение [**WM \_ DRAWITEM**](wm-drawitem.md) в окно владельца, когда должен быть нарисован элемент. Элемент **ItemId** структуры [**дравитемструкт**](/windows/win32/api/winuser/ns-winuser-drawitemstruct) , переданный с сообщением **WM \_ DRAWITEM** , указывает номер строки рисуемого элемента. Список без данных не отправляет сообщение [**\_ DELETEITEM WM**](wm-deleteitem.md) .<br/> |
| <span id="LBS_NOINTEGRALHEIGHT"></span><span id="lbs_nointegralheight"></span><dl> <dt>**ФУНТ \_ ноинтегралхеигхт**</dt> </dl>    | Указывает, что размер списка в точности соответствует размеру, указанному приложением при создании списка. Как правило, система изменяет размер окна списка таким образом, чтобы в списке не отображались частичные элементы.<br/> Для списков, имеющих стиль **фунта \_ овнердраввариабле** , всегда применяется стиль **фунта \_ ноинтегралхеигхт** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="LBS_NOREDRAW"></span><span id="lbs_noredraw"></span><dl> <dt>**ФУНТ \_ норедрав**</dt> </dl>                            | Указывает, что при внесении изменений внешний вид списка не обновляется.<br/> Чтобы изменить состояние перерисовки элемента управления, используйте сообщение [**WM \_ сетредрав**](/windows/desktop/gdi/wm-setredraw) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="LBS_NOSEL"></span><span id="lbs_nosel"></span><dl> <dt>**ФУНТ на \_ нос**</dt> </dl>                                     | Указывает, что список содержит элементы, которые могут быть просмотрены, но не выбраны. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NOTIFY"></span><span id="lbs_notify"></span><dl> <dt>**\_уведомление о фунтах**</dt> </dl>                                  | Заставляет список отправить код уведомления родительскому окну каждый раз, когда пользователь щелкает элемент списка ([ЛБН \_ селчанже](lbn-selchange.md)), дважды щелкает элемент ([ЛБН \_ дблклк](lbn-dblclk.md)) или отменяет выбор ([ЛБН \_ селканцел](lbn-selcancel.md)). <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="LBS_OWNERDRAWFIXED"></span><span id="lbs_ownerdrawfixed"></span><dl> <dt>**ФУНТ \_ овнердравфиксед**</dt> </dl>          | Указывает, что владелец списка отвечает за рисование его содержимого и то, что элементы в списке имеют одинаковую высоту. Окно-владелец получает сообщение [**WM \_ меасуреитем**](wm-measureitem.md) при создании списка и сообщение [**WM \_ DRAWITEM**](wm-drawitem.md) при изменении визуального аспекта списка.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="LBS_OWNERDRAWVARIABLE"></span><span id="lbs_ownerdrawvariable"></span><dl> <dt>**ФУНТ \_ овнердраввариабле**</dt> </dl> | Указывает, что владелец списка отвечает за рисование его содержимого и то, что элементы в списке имеют переменную высоту. Окно-владелец получает сообщение [**WM \_ меасуреитем**](wm-measureitem.md) для каждого элемента в поле, когда создается список, и сообщение [**WM \_ DRAWITEM**](wm-drawitem.md) при изменении визуального аспекта списка.<br/> Этот стиль приводит к включению стиля **фунта \_ ноинтегралхеигхт** .<br/> Этот стиль пропускается, если указан **\_ многостолбцовый стиль фунта** . <br/>                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="LBS_SORT"></span><span id="lbs_sort"></span><dl> <dt>**Сортировка по ФУНТам \_**</dt> </dl>                                        | Сортирует строки в списке в алфавитном порядке.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="LBS_STANDARD"></span><span id="lbs_standard"></span><dl> <dt>**ФУНТ ( \_ стандартный)**</dt> </dl>                            | Сортирует строки в списке в алфавитном порядке. Родительское окно получает код уведомления каждый раз, когда пользователь щелкает элемент списка, дважды щелкает элемент или отменяет выбор. Список имеет вертикальную полосу прокрутки и имеет границы на всех сторонах. Этот стиль сочетает стили **границ \_ фунта**, **фунтов \_ сортировки**, [**WS \_ VSCROLL**](/windows/desktop/winmsg/window-styles)и [**WS \_**](/windows/desktop/winmsg/window-styles) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="LBS_USETABSTOPS"></span><span id="lbs_usetabstops"></span><dl> <dt>**ФУНТ \_ усетабстопс**</dt> </dl>                   | Позволяет списку распознавать и расширять символы табуляции при рисовании строк. Можно использовать сообщение [**\_ сеттабстопс балансировки нагрузки**](lb-settabstops.md) , чтобы указать позиции табуляции. Позиции табуляции по умолчанию разбиваются на 32 единиц шаблона диалогового окна. Единицы шаблонов диалоговых окон — это аппаратно-независимые единицы, используемые в шаблонах диалоговых окон. Чтобы преобразовать измерения из шаблона диалогового окна в единицы экрана (в пикселях), используйте функцию [**мапдиалогрект**](/windows/desktop/api/winuser/nf-winuser-mapdialogrect) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_WANTKEYBOARDINPUT"></span><span id="lbs_wantkeyboardinput"></span><dl> <dt>**ФУНТ \_ ванткэйбоардинпут**</dt> </dl> | Указывает, что владелец списка получает сообщения [**WM \_ вкэйтоитем**](wm-vkeytoitem.md) всякий раз, когда пользователь нажимает клавишу, а поле со списком имеет фокус ввода. Это позволяет приложению выполнять специальную обработку ввода с клавиатуры.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
