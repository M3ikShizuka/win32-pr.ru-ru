# <a name="hit-test-options"></a>Параметры проверки нажатия

В этом разделе перечислены значения параметров, которые используются с параметром *двоптионс* функции [**хиттестсемебаккграунд**](/windows/desktop/api/Uxtheme/nf-uxtheme-hittestthemebackground) . Список значений, возвращаемых при указании одного из этих параметров, см. в разделе [возвращаемые значения проверки попадания](theme-hit-test-retval.md).

В следующей таблице перечислены параметры проверки нажатия.

|------------------------------|--------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| ХТТБ \_ баккграундсег          | 0x00000000                                                                                                               | Параметр проверки нажатия сегмента фона темы.                                                                                                                                           |
| ХТТБ \_ фикседбордер            | 0x00000002                                                                                                               | Параметр проверки нажатия фиксированной границы.                                                                                                                                                       |
| \_заголовок хттб                | 0x00000004                                                                                                               | Параметр проверки нажатия подписи.                                                                                                                                                            |
| ХТТБ \_ ресизингбордер         | (ХТТБ \_ РЕСИЗИНГБОРДЕР \_ Left \| хттб \_ ресизингбордер \_ Top \| хттб \_ ресизингбордер \_ right \| хттб \_ ресизингбордер \_ Bottom) | Изменение размера параметров проверки нажатия границы.                                                                                                                                                   |
