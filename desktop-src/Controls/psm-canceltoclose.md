# <a name="psm_canceltoclose-message"></a>\_Сообщение ПСМ канцелтоклосе

Отправляется приложением при выполнении изменений с момента последнего уведомления об [ \_ использовании PSN](psn-apply.md) , которое не может быть отменено. Это сообщение можно отправить явным образом или с помощью макроса [**пропшит \_ канцелтоклосе**](/windows/desktop/api/Prsht/nf-prsht-propsheet_canceltoclose) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Должен равняться нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Должен равняться нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Комментарии

**ПСМ \_ КАНЦЕЛТОКЛОСЕ** отключает кнопку **"Отмена** " и изменяет текст кнопки " **ОК** " на "Закрыть".

Большинство страниц свойств ожидают необратимых изменений, пока не будет \_ получено уведомление о применении PSN. Однако в некоторых обстоятельствах страница свойств может вносить необратимые изменения за пределами стандартной \_ последовательности PSN Apply/PSN \_ Reset. Одним из примеров является страница свойств, содержащая кнопку « **изменить** », которая используется для вывода поддиалогового окна для редактирования свойства. Когда пользователь нажимает кнопку **ОК** , чтобы отправить изменение, страница вкладки свойств имеет несколько параметров.

-   Он может записывать изменения, но дожидаться получения PSN \_ уведомления об использовании для их применения. Это предпочтительный подход.
-   Изменения можно применить сразу после выхода из поддиалогового окна, но запомнить исходные параметры. Эти параметры можно использовать для восстановления исходного состояния при \_ получении уведомления о сбросе PSN.
-   Он может применить изменения немедленно и не пытаться восстановить исходные параметры при получении уведомления о [ \_ сбросе PSN](psn-reset.md) . Этот подход не рекомендуется, но может быть необходим, если изменения слишком далеко не позволяют сделать другие два варианта практичными.

В третьем варианте приложения должны отправить сообщение **ПСМ \_ канцелтоклосе** на страницу свойств. Указывает пользователю, что изменения, внесенные с помощью поддиалогового окна, нельзя отменить, нажав кнопку **Отмена** .

> [!Note]  
> Это сообщение не поддерживается при использовании стиля мастера Aero ([**командном PSH \_ аеровизард**](/windows/desktop/api/Prsht/ns-prsht-propsheetheadera_v2)).
