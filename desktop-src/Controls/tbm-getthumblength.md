# <a name="tbm_getthumblength-message"></a>\_Сообщение ТБМ жетсумбленгс

Получает длину ползунка в TrackBar.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>
