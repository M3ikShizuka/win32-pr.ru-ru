# <a name="tvm_getisearchstring-message"></a>\_Сообщение TVM жетисеарчстринг

Извлекает строку добавочного поиска для элемента управления "дерево-представление". Элемент управления "представление дерева" использует строку добавочного поиска для выбора элемента на основе символов, введенных пользователем. Это сообщение можно отправить явно или с помощью макроса [**\_ жетисеарчстринг TreeView**](/windows/desktop/api/Commctrl/nf-commctrl-treeview_getisearchstring) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на буфер, который получает строку добавочного поиска.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает количество символов в строке добавочного поиска.

## <a name="remarks"></a>Комментарии

**Предупреждение системы безопасности:** Неправильное использование этого сообщения может нарушить безопасность программы. Необходимо выделить достаточно большой буфер для хранения строки. Сначала необходимо вызвать сообщение, передав **значение NULL** в *lParam*. Это возвращает количество символов, которые являются обязательными **, и исключает их**. Затем вызовите сообщение еще раз, чтобы получить строку. Прежде чем продолжить, ознакомьтесь с [вопросами безопасности: Microsoft Windows Controls](sec-comctls.md) .

Если элемент управления "дерево-представление" не находится в режиме добавочного поиска, возвращаемое значение равно нулю.
