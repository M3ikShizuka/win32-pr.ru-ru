# <a name="dtm_getmccolor-message"></a>\_Сообщение DTM жетмкколор

Возвращает цвет для заданной части месячного календаря в элементе управления "Выбор даты и времени" (DTP). Это сообщение можно отправить явным образом или использовать макрос [**\_ жетмонскалколор DateTime**](/windows/desktop/api/Commctrl/nf-commctrl-datetime_getmonthcalcolor) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Значение типа **int** , указывающее, какой цвет календаря месяца получить. Значение может быть одним из следующих.



| Значение                                                                                                                                                                     | Значение                                                                                                                                                                                                 |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="MCSC_BACKGROUND"></span><span id="mcsc_background"></span><dl> <dt>**\_фон MCSC**</dt> </dl>       | Получение цвета фона, отображаемого в интервале между месяцами.<br/>                                                                                                                                      |
| <span id="MCSC_MONTHBK"></span><span id="mcsc_monthbk"></span><dl> <dt>**MCSC \_ монсбк**</dt> </dl>                | Получение цвета фона, отображаемого в течение месяца.<br/>                                                                                                                                    |
| <span id="MCSC_TEXT"></span><span id="mcsc_text"></span><dl> <dt>**MCSC \_ текст**</dt> </dl>                         | Получение цвета, используемого для показа текста в течение месяца.<br/>                                                                                                                                      |
| <span id="MCSC_TITLEBK"></span><span id="mcsc_titlebk"></span><dl> <dt>**MCSC \_ титлебк**</dt> </dl>                | Получение цвета фона, отображаемого в заголовке календаря.<br/>                                                                                                                             |
| <span id="MCSC_TITLETEXT"></span><span id="mcsc_titletext"></span><dl> <dt>**MCSC \_ титлетекст**</dt> </dl>          | Получение цвета, используемого для показа текста в заголовке календаря.<br/>                                                                                                                         |
| <span id="MCSC_TRAILINGTEXT"></span><span id="mcsc_trailingtext"></span><dl> <dt>**MCSC \_ траилингтекст**</dt> </dl> | Получение цвета, используемого для вывода текста заголовка и конца дня. </dd> <dt>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **COLORREF** , представляющее параметр цвета для указанной части элемента управления "месячный календарь" в случае успеха. В случае неудачи сообщение возвращает значение-1.
