# <a name="cbem_getextendedstyle-message"></a>\_Сообщение кбем жетекстендедстиле

Возвращает расширенные стили, используемые для элемента управления ComboBoxEx.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение типа DWORD, которое содержит [Расширенные стили элемента управления ComboBoxEx](comboboxex-control-extended-styles.md) , используемые для элемента управления.
