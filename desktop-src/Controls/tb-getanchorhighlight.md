# <a name="tb_getanchorhighlight-message"></a>\_Сообщение ЖЕТАНЧОРХИГХЛИГХТ ТБ

Получает параметр выделения закреплений для панели инструментов.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает логическое значение, указывающее, задано ли выделение закреплений. Если это значение не равно нулю, выделение закреплений включено. Если это значение равно нулю, то оно отключено.
