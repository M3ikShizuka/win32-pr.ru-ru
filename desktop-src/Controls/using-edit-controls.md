# <a name="using-edit-controls"></a>Использование элементов управления Edit

В этом разделе содержатся сведения и примеры кода для реализации элементов управления "поле ввода".

Элементы управления "поле ввода" обычно используются в диалоговых окнах, но их также можно использовать в клиентской области стандартного окна. Однострочные элементы управления "поле ввода" полезны для получения одной строки от пользователя. Многострочные элементы управления "поле ввода" позволяют реализовать большинство функций простого текстового процессора в приложении.
