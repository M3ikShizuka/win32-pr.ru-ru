# <a name="dl_canceldrag-notification-code"></a>\_Код уведомления канцелдраг списка рассылки

Сигнализирует, что пользователь отменил операцию перетаскивания, щелкнув правой кнопкой мыши или нажав клавишу ESC. Поле со списком перетаскивания отправляет этот код уведомления в его родительское окно в виде сообщения списка перетаскивания. Дополнительные сведения см. в статье [перетаскивание сообщений в поле со списком](about-list-boxes.md).


```C++
DL_CANCELDRAG

    pDragInfo = (LPARAM)(LPDRAGLISTINFO) lParam;
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**драглистинфо**](/windows/win32/api/commctrl/ns-commctrl-draglistinfo) , содержащую \_ код уведомления DL канцелдраг, маркер для списка перетаскивания и позицию курсора.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Remarks

Обрабатывая \_ код уведомления канцелдраг списка рассылки, приложение может сбросить свое внутреннее состояние, чтобы указать, что перетаскивание не действует.
