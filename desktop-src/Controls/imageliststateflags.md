# <a name="imageliststateflags"></a>имажелистстатефлагс

Следующие флаги передаются в метод [**IImageList::D RAW**](/windows/desktop/api/CommonControls/nf-commoncontrols-iimagelist-draw) в элементе **фстате** элемента [**имажелистдравпарамс**](/windows/win32/api/commctrl/ns-commctrl-imagelistdrawparams).



| Константа/значение                                                                                                                                                                                                             | Описание                                                                                                                                                                                                                                                                                                                                                                                   |
|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="ILS_NORMAL"></span><span id="ils_normal"></span><dl> <dt>**ILS \_ Обычная**</dt> <dt>0x00000000</dt> </dl>       | Состояние изображения не изменяется.<br/>                                                                                                                                                                                                                                                                                                                                                   |
| <span id="ILS_GLOW"></span><span id="ils_glow"></span><dl> <dt>**ILS \_ СВЕЧЕНИе**</dt> <dt>0x00000001</dt> </dl>             | Не поддерживается. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| <span id="ILS_SHADOW"></span><span id="ils_shadow"></span><dl> <dt>**ILS \_ ТЕНЕВая**</dt> версия <dt>0x00000002</dt> </dl>       | Не поддерживается. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| <span id="ILS_SATURATE"></span><span id="ils_saturate"></span><dl> <dt>**ILS \_ НАСЫЩЕНность**</dt> <dt>0x00000004</dt> </dl> | Уменьшает насыщенность цвета значка до оттенков серого. Это относится только к образам 32 бита. <br/>                                                                                                                                                                                                                                                                                            |
| <span id="ILS_ALPHA"></span><span id="ils_alpha"></span><dl> <dt>**ILS \_ Альфа**</dt> - <dt>0x00000008</dt> </dl>          | Альфа-смешение значка. Альфа-смешение определяет уровень прозрачности значка в соответствии со значением его альфа-канала. Значение альфа-канала обозначается элементом **Frame** в методе [**имажелистдравпарамс**](/windows/win32/api/commctrl/ns-commctrl-imagelistdrawparams) . Альфа-канал может принимать значение от 0 до 255, при этом 0 становится полностью прозрачным, а 255 — полностью непрозрачным.<br/> |
