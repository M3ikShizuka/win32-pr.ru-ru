# <a name="rb_getbarheight-message"></a>\_Сообщение ЖЕТБАРХЕИГХТ RB

Извлекает высоту элемента управления главной панели.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **uint** , представляющее высоту элемента управления в пикселях.
