# <a name="udm_setrange-message"></a>\_Сообщение SETRANGE UDM

Задает минимальное и максимальное положение (диапазон) для элемента управления "вверх/вниз".

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>

[**Ловорд**](/previous-versions/windows/desktop/legacy/ms632659(v=vs.85)) — это **короткое** значение, указывающее максимальное значение для элемента управления "вверх/вниз", а [**HIWORD**](/previous-versions/windows/desktop/legacy/ms632657(v=vs.85)) — **короткое** значение, указывающее минимальную длину. Ни одна из позиций не может быть больше \_ значения обновления максвал или меньше значения обновления \_ минвал. Кроме того, разница между двумя позициями не может превышать обновления \_ максвал.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Комментарии

Максимальная длина может быть меньше, чем минимальная. При нажатии кнопки со стрелкой вверх текущее расположение перемещается ближе к максимальному положению, а нажатие кнопки со стрелкой вниз перемещается к минимальному положению.
