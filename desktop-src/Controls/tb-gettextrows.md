# <a name="tb_gettextrows-message"></a>\_Сообщение ЖЕТТЕКСТРОВС ТБ

Возвращает максимальное количество текстовых строк, которые могут отображаться на кнопке панели инструментов.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение типа INT, представляющее максимальное количество текстовых строк, которое будет отображаться элементом управления для кнопки.
