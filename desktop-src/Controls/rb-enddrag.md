# <a name="rb_enddrag-message"></a>\_Сообщение ЕНДДРАГ RB

Прерывает операцию перетаскивания элемента управления главной панели. Это сообщение не приводит к отправке уведомления [РБН \_ енддраг](rbn-enddrag.md) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение для этого сообщения не используется.
