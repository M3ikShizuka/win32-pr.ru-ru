# <a name="em_getlimittext-message"></a>\_Сообщение ЖЕТЛИМИТТЕКСТ EM

Возвращает текущее ограничение текста для элемента управления "поле ввода". Это сообщение можно отправить либо в элемент управления "поле ввода", либо в элемент управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение — это ограничение текста.

## <a name="remarks"></a>Комментарии

**Изменение элементов управления, форматированное изменение 2,0 и более поздние версии:** Ограничение текста — это максимальный объем текста в **файле TCHAR**, который может содержать элемент управления. Для текста ANSI это число байтов; для текста в Юникоде это число символов. Два документа с одинаковым ограничением на количество символов выдают одинаковый размер текста, даже если один из них имеет значение ANSI, а другой — в Юникоде.

**Расширенное редактирование 1,0:** Ограничение текста — это максимальный объем текста в байтах, который может содержаться в элементе управления Rich Edit.

**Расширенное редактирование:** Поддерживается в Microsoft Rich Edit 1,0 и более поздних версиях. Дополнительные сведения о совместимости расширенных версий редактирования с различными версиями системы см. в разделе [Общие сведения об элементах управления редактированием](about-rich-edit-controls.md).
