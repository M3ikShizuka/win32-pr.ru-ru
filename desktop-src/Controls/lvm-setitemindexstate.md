# <a name="lvm_setitemindexstate-message"></a>\_Сообщение LVM сетитеминдексстате

Задает состояние элемента представления списка. Отправьте это сообщение явным образом или с помощью макроса [**\_ сетитеминдексстате ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_setitemindexstate) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* \[ окне\]
</dt> <dd>

Указатель на структуру [**лвитеминдекс**](/windows/win32/api/commctrl/ns-commctrl-lvitemindex) для элемента. Вызывающий процесс отвечает за выделение этой структуры и задание элементов.

</dd> <dt>

*lParam* \[ окне\]
</dt> <dd>

Указатель на структуру [**лвитем**](/windows/win32/api/commctrl/ns-commctrl-lvitema) . Вызывающий процесс отвечает за выделение памяти для структуры. Задайте для элемента **состояния** одно или несколько (как побитовое сочетание) флагов [состояния элементов списка](list-view-item-states.md) . Установите элемент **статемаск** структуры, чтобы указать допустимые биты элемента **State** . Дополнительные сведения см. в описании элемента **статемаск** структуры **лвитем** .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает одно из следующих значений типа **HRESULT**.



| Код возврата                                                                                  | Описание                                                       |
|----------------------------------------------------------------------------------------------|-------------------------------------------------------------------|
| <dl> <dt>**\_Ошибка E**</dt> </dl>       | Не удалось задать состояние.<br/>                            |
| <dl> <dt>**\_непредвиденная E**</dt> </dl> | Элемент управления "представление списка" не готов к работе.<br/> |
| <dl> <dt>**\_ОК**</dt> </dl>         | Операция выполнена успешно.<br/>                          |
