# <a name="em_getfileline-message-commctrlh"></a>Сообщение EM_GETFILELINE (Коммктрл. h)

Копирует строку текста из элемента управления "поле ввода" независимо от того, как линии отображаются на экране, и помещаются в указанный буфер.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Отсчитываемый от нуля индекс строки, извлекаемой из многострочного элемента управления "поле ввода". Нулевое значение задает верхнюю строку. Этот параметр игнорируется с помощью однострочного элемента управления Edit.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на буфер, который получает копию строки. Перед отправкой сообщения задайте для первого слова этого буфера размер буфера в **файле TCHAR**. Для текста ANSI это число байтов; для текста в Юникоде это число символов. Размер в первом слове перезаписывается скопированной строкой.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение — это число скопированных данных **TCHAR**. Возвращаемое значение равно нулю, если номер строки, указанный параметром *wParam* , больше числа строк в элементе управления "поле ввода".

## <a name="remarks"></a>Комментарии

Скопированная строка не содержит завершающего нуль-символа.
