# <a name="cbem_getimagelist-message"></a>Сообщение КБЕМ/ \_ ImageList

Возвращает маркер для списка изображений, назначенного элементу управления ComboBoxEx.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает маркер в список изображений, назначенный элементу управления в случае успеха, или **значение NULL** в противном случае.
