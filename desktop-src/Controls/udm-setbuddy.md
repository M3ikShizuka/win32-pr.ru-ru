# <a name="udm_setbuddy-message"></a>\_Сообщение СЕТБУДДИ UDM

Устанавливает окно собеседника для элемента управления "вверх/вниз".

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Обработайте с новым дружественным окном.

</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение — это маркер предыдущего дружественного окна.
