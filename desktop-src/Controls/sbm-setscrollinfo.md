# <a name="sbm_setscrollinfo-message"></a>\_Сообщение СБМ сетскроллинфо

Сообщение **СБМ \_ сетскроллинфо** отправляется для установки параметров полосы прокрутки.

Приложения не должны отсылать это сообщение напрямую. Вместо этого они должны использовать функцию [**сетскроллинфо**](/windows/desktop/api/Winuser/nf-winuser-setscrollinfo) . Окно получает это сообщение через функцию [*WindowProc*](/previous-versions/windows/desktop/legacy/ms633573(v=vs.85)) . Приложения, которые реализуют пользовательский элемент управления "полоса прокрутки", должны отвечать на эти сообщения, чтобы функция **сетскроллинфо** работала правильно.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Указывает, перерисовывается ли полоса прокрутки для отражения новой позицией ползунка. Если этот параметр имеет **значение true**, полоса прокрутки перерисовывается. Если значение равно **false**, полоса прокрутки не перерисовывается.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**скроллинфо**](/windows/win32/api/winuser/ns-winuser-scrollinfo) . Перед вызовом [**сетскроллинфо**](/windows/desktop/api/Winuser/nf-winuser-setscrollinfo)задайте для элемента **кбсизе** структуры значение **sizeof**(**скроллинфо**), установите элемент **фмаск** , чтобы указать заданные параметры, и укажите новые значения параметров в соответствующих элементах.

Элемент **фмаск** может иметь одно или несколько из следующих значений.



| Значение                                                                                                                                                                           | Значение                                                                                                                        |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| <span id="SIF_DISABLENOSCROLL"></span><span id="sif_disablenoscroll"></span><dl> <dt>**SIF \_ дисабленоскролл**</dt> </dl> | Отключает полосу прокрутки вместо ее удаления, если новые параметры полосы прокрутки сделают полосу прокрутки ненужной.<br/> |
| <span id="SIF_PAGE"></span><span id="sif_page"></span><dl> <dt>**\_страница SIF**</dt> </dl>                                  | Задает для страницы прокрутки значение, указанное в элементе **нпаже** .<br/>                                                |
| <span id="SIF_POS"></span><span id="sif_pos"></span><dl> <dt>**SIF \_ POS**</dt> </dl>                                     | Задает в качестве расположения прокрутки значение, указанное в элементе **npos** . <br/>                                            |
| <span id="SIF_RANGE"></span><span id="sif_range"></span><dl> <dt>**\_диапазон SIF**</dt> </dl>                               | Задает для диапазона прокрутки значение, указанное в элементах **nмин.** и **nмакс.** . <br/>                                 |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение является текущей позицией ползунка.

## <a name="remarks"></a>Remarks

Сообщения, указывающие расположение полосы прокрутки, [**WM \_ HSCROLL**](wm-hscroll.md) и [**WM \_ VSCROLL**](wm-vscroll.md), содержат только 16 бит данных о положении. Однако структура [**скроллинфо**](/windows/win32/api/winuser/ns-winuser-scrollinfo) , используемая [**СБМ \_ жетскроллинфо**](sbm-getscrollinfo.md), **СБМ \_ сетскроллинфо**, [**жетскроллинфо**](/windows/desktop/api/Winuser/nf-winuser-getscrollinfo)и [**сетскроллинфо**](/windows/desktop/api/Winuser/nf-winuser-setscrollinfo) , предоставляет 32 бит для расположения данных в полосе прокрутки. Эти сообщения и функции можно использовать при обработке сообщений **WM \_ HSCROLL** или **WM \_ VSCROLL** для получения данных о положении полосы прокрутки 32 бит.
