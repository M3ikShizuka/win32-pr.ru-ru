# <a name="dtn_userstring-notification-code"></a>\_Код уведомления ДТН усерстринг

Отправляется элементом управления "Выбор даты и времени" (DTP), когда пользователь заканчивает редактирование строки в элементе управления. Этот код уведомления отправляется только элементами управления DTP, для которых задан стиль [**\_ Аппканпарсе служб DTS**](date-and-time-picker-control-styles.md) . Этот код уведомления отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
DTN_USERSTRING

    lpDTstring = (LPNMDATETIMESTRING) lParam;
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмдатетиместринг**](/windows/win32/api/commctrl/ns-commctrl-nmdatetimestringa) , содержащую сведения об экземпляре кода уведомления.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Владелец элемента управления должен возвращать ноль.

## <a name="remarks"></a>Remarks

Обработка этого кода уведомления позволяет владельцу предоставлять пользовательские ответы на строки, которые пользователь вводит в элемент управления. Владелец должен быть подготовлен для анализа входной строки и при необходимости предпринимать необходимые действия.
