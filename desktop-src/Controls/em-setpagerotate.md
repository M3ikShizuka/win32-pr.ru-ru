# <a name="em_setpagerotate-message"></a>\_Сообщение СЕТПАЖЕРОТАТЕ EM

\[**EM \_ СЕТПАЖЕРОТАТЕ** доступен для использования в операционных системах, указанных в разделе требования. В последующих версиях он может быть изменен или недоступен.\]

Задает макет текста для элемента управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Значение макета текста. Это может быть одно из следующих значений.



| Значение                                                                                                                                       | Значение                                                                                       |
|---------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| <span id="EPR_0"></span><span id="epr_0"></span><dl> <dt>**EPR \_ 0**</dt> </dl>       | Текст перетекает слева направо и сверху вниз.<br/>                              |
| <span id="EPR_90"></span><span id="epr_90"></span><dl> <dt>**EPR \_ 90**</dt> </dl>    | Текст перетекает снизу вверх и слева направо.<br/>                              |
| <span id="EPR_180"></span><span id="epr_180"></span><dl> <dt>**EPR \_ 180**</dt> </dl> | Текст перетекает справа налево и снизу вверх.<br/>                              |
| <span id="EPR_270"></span><span id="epr_270"></span><dl> <dt>**EPR \_ 270**</dt> </dl> | Текст перетекает сверху вниз и справа налево.<br/>                              |
| <span id="EPR_SE"></span><span id="epr_se"></span><dl> <dt>**EPR \_ SE**</dt> </dl>    | **Windows 8:** Направление текста сверху вниз и слева направо (разметка текста в формате монгольский).<br/> |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение — новое значение макета текста.

## <a name="remarks"></a>Комментарии

Это сообщение задает макет текста для всего документа. Однако внедренное содержимое не поворачивается и должно быть повернуто отдельно приложением.
