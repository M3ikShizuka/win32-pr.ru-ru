# <a name="tbm_settipside-message"></a>\_Сообщение ТБМ сеттипсиде

Размещает элемент управления ToolTip, используемый элементом управления TrackBar. Элементы управления TrackBar, использующие всплывающие подсказки в стиле [**\_ подсказок TBS**](trackbar-control-styles.md) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Значение, представляющее расположение, в котором отображается элемент управления ToolTip. Значение может быть одним из следующих.



| Значение                                                                                                                                                   | Значение                                                                                                                       |
|---------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| <span id="TBTS_TOP"></span><span id="tbts_top"></span><dl> <dt>**ТБТС \_ сверху**</dt> </dl>          | Элемент управления ToolTip будет расположен над TrackBar. Этот флаг предназначен для использования с горизонтальными TrackBar.<br/>         |
| <span id="TBTS_LEFT"></span><span id="tbts_left"></span><dl> <dt>**ТБТС \_ слева**</dt> </dl>       | Элемент управления ToolTip будет расположен слева от TrackBar. Этот флаг предназначен для использования с вертикальными TrackBar.<br/>  |
| <span id="TBTS_BOTTOM"></span><span id="tbts_bottom"></span><dl> <dt>**ТБТС \_ снизу**</dt> </dl> | Элемент управления ToolTip будет расположен под TrackBar. Этот флаг предназначен для использования с горизонтальными TrackBar.<br/>         |
| <span id="TBTS_RIGHT"></span><span id="tbts_right"></span><dl> <dt>**ТБТС \_ вправо**</dt> </dl>    | Элемент управления ToolTip будет расположен справа от TrackBar. Этот флаг предназначен для использования с вертикальными TrackBar.<br/> |



 

</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение, представляющее предыдущее расположение элемента управления ToolTip. Возвращаемое значение равно одному из возможных значений параметра *wParam*.
