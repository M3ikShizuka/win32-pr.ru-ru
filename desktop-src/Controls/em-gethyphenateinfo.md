# <a name="em_gethyphenateinfo-message"></a>\_Сообщение ЖЕСИФЕНАТЕИНФО EM

Получает сведения о расстановке переносов для элемента управления форматированного ввода (Майкрософт).

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Структура [**хифенатеинфо**](/windows/win32/api/richedit/ns-richedit-hyphenateinfo) .

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>
