# <a name="em_setzoom-message"></a>\_Сообщение СЕТЗУМ EM

Задает коэффициент масштабирования для многострочного элемента управления "поле ввода" или элемента управления Rich Edit. Соотношение должно быть числом в диапазоне от 1/64 до 64. В элементе управления "поле ввода" должен быть установлен расширенный набор стилей **ES \_ ex \_ масштабируемый** , чтобы это сообщение имело значение. см. раздел [Правка Управление расширенными стилями](edit-control-window-extended-styles.md).

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Числитель коэффициента масштабирования.

</dd> <dt>

*lParam* 
</dt> <dd>

Знаменатель коэффициента масштабирования. Эти параметры могут иметь следующие значения.



| Значение                                                                                                                                                                                                                                                              | Значение                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="Both_0"></span><span id="both_0"></span><span id="BOTH_0"></span><dl> <dt>**Оба значения 0**</dt> </dl>                                                                                                   | Отключает масштаб с помощью сообщения **EM \_ сетзум** (по-прежнему может произойти с помощью [**тксжетекстент**](/windows/desktop/api/Textserv/nf-textserv-itexthost-txgetextent)).<br/> |
| <span id="1_64____wParam___lParam____64"></span><span id="1_64____wparam___lparam____64"></span><span id="1_64____WPARAM___LPARAM____64"></span><dl> <dt>**1/64 < (wParam/lParam) < 64**</dt> </dl> | Отображение масштаба с помощью числителя или знаменателя коэффициента масштабирования<br/>                                                                                |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если новый параметр масштабирования принимается, возвращается значение **true**.

Если новое значение масштаба не принято, возвращается значение **false**.

## <a name="remarks"></a>Remarks

**Изменить:** поддерживается в Windows 10 1809 и более поздних версиях. В элементе управления "поле ввода" должен быть установлен расширенный набор стилей **ES \_ ex \_ масштабируемый** , чтобы это сообщение имело значение. см. раздел [Правка Управление расширенными стилями](edit-control-window-extended-styles.md). Дополнительные сведения об элементе управления "поле ввода" см. в разделе [Edit Controls](about-edit-controls.md).
