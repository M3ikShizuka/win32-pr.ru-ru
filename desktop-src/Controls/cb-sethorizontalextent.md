# <a name="cb_sethorizontalextent-message"></a>\_Сообщение СЕСОРИЗОНТАЛЕКСТЕНТ CB

Приложение отправляет сообщение **\_ сесоризонталекстент CB** , чтобы задать ширину (в пикселях), на которую окно списка можно прокручивать горизонтально (прокручиваемую ширину). Если ширина списка меньше этого значения, горизонтальная полоса прокрутки горизонтально прокручивает элементы списка. Если ширина окна списка равна или больше этого значения, горизонтальная полоса прокрутки скрыта или, если поле со списком имеет стиль [**\_ дисабленоскролл CBS**](combo-box-styles.md) , отключено.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Задает ширину прокручиваемого окна списка в пикселях.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>
