# <a name="em_settabstops-message"></a>\_Сообщение СЕТТАБСТОПС EM

Сообщение **EM \_ сеттабстопс** задает позицию табуляции в многострочном элементе управления Edit. При копировании текста в элемент управления любой символ табуляции в тексте приводит к тому, что пространство будет создано до следующей позиции табуляции.

Это сообщение обрабатывается только многострочными элементами управления Edit. Это сообщение можно отправить либо в элемент управления "поле ввода", либо в элемент управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Количество единиц табуляции, содержащихся в массиве. Если этот параметр равен нулю, параметр *lParam* игнорируется, а табуляция по умолчанию задается каждые 32 единиц шаблона диалогового окна. Если этот параметр равен 1, то позиции табуляции задаются для каждых *n* единиц шаблона диалогового окна, где *n* — расстояние, на которое указывает параметр *lParam* . Если этот параметр больше 1, *lParam* является указателем на массив единиц табуляции.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на массив целых чисел без знака, указывающий позиции табуляции, в единицах шаблона диалогового окна. Если параметр *wParam* равен 1, этот параметр является указателем на целое число без знака, которое содержит расстояние между всеми позициями табуляции, в единицах шаблона диалогового окна.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если заданы все вкладки, возвращается значение **true**.

Если все вкладки не заданы, возвращается значение **false**.

## <a name="remarks"></a>Комментарии

Сообщение **EM \_ сеттабстопс** не перерисовывает окно элемента управления редактированием автоматически. Если приложение изменяет позиции табуляции для текста, уже нарисованного в элементе управления "поле ввода", он должен вызвать функцию [**инвалидатерект**](/windows/desktop/api/winuser/nf-winuser-invalidaterect) для перерисовки окна управления редактированием.

Значения, указанные в массиве, находятся в единицах шаблонов диалоговых окон, которые являются аппаратно-независимыми устройствами, используемыми в шаблонах диалоговых окон. Чтобы преобразовать измерения из шаблона диалогового окна в единицы экрана (в пикселях), используйте функцию [**мапдиалогрект**](/windows/desktop/api/winuser/nf-winuser-mapdialogrect) .

**Расширенное редактирование:** Поддерживается в Microsoft Rich Edit 3,0 и более поздних версиях. Элемент управления Rich Edit может иметь максимальное количество заданных вкладок, заданное в параметре MAX \_ Tab Stop \_ . Дополнительные сведения о совместимости расширенных версий редактирования с различными версиями системы см. в разделе [Общие сведения об элементах управления редактированием](about-rich-edit-controls.md).
