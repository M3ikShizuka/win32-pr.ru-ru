# <a name="dtn_wmkeydown-notification-code"></a>\_Код уведомления ДТН вмкэйдовн

Отправляется элементом управления "Выбор даты и времени" (DTP), когда пользователь вводит в поле обратного вызова. Этот код уведомления отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
DTN_WMKEYDOWN

    lpDTKeystroke = (LPNMDATETIMEWMKEYDOWN)lParam;
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмдатетимевмкэйдовн**](/windows/win32/api/commctrl/ns-commctrl-nmdatetimewmkeydowna) , содержащую сведения об этом экземпляре кода уведомления. Структура включает сведения о ключе, введенном пользователем, подстроке, определяющей поле обратного вызова, а также о текущей системной дате и времени.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Владелец элемента управления должен возвращать ноль.

## <a name="remarks"></a>Комментарии

Обработка этого кода уведомления позволяет владельцу элемента управления предоставлять определенные ответы на нажатия клавиш в полях обратного вызова элемента управления.
