# <a name="sb_issimple-message"></a>\_Непростое сообщение SB

Проверяет, находится ли элемент управления "строка состояния" в простом режиме.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ненулевое значение, если элемент управления "строка состояния" находится в простом режиме, или ноль в противном случае.
