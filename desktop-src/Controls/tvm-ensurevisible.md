# <a name="tvm_ensurevisible-message"></a>\_Сообщение TVM енсуревисибле

Обеспечивает видимость элемента представления в виде дерева, расширяя родительский элемент или прокручивает элемент управления представлением дерева, если это необходимо. Это сообщение можно отправить явно или с помощью макроса [**\_ енсуревисибле TreeView**](/windows/desktop/api/Commctrl/nf-commctrl-treeview_ensurevisible) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>

Маркер для элемента.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ненулевое значение, если система прокручивает элементы в элементе управления "дерево" и никакие элементы не были развернуты. В противном случае сообщение возвращает ноль.

## <a name="remarks"></a>Remarks

Если сообщение TVM \_ енсуревисибле расширяет родительский элемент, родительское окно получает коды уведомлений [ТВН \_ итемекспандинг](tvn-itemexpanding.md) и [ТВН \_ итемекспандед](tvn-itemexpanded.md) .
