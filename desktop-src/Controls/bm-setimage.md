# <a name="bm_setimage-message"></a>\_Сообщение BM сетимаже

Связывает новый образ (значок или растровое изображение) с кнопкой.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Тип изображения, связываемого с кнопкой. Этот параметр может принимать одно из следующих значений:

-   \_точечный рисунок изображения
-   \_значок изображения

</dd> <dt>

*lParam* 
</dt> <dd>

Маркер (**Хикон** или **хбитмап**) к изображению, связываемый с кнопкой.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение является обработчиком изображения, ранее связанного с кнопкой, если таковая имеется; в противном случае он имеет **значение NULL**.

## <a name="remarks"></a>Remarks

Внешний вид текста, значка или обоих элементов в элементе управления "Кнопка" зависит от [**\_ значка BS**](button-styles.md) и стилей [**\_ растрового изображения BS**](button-styles.md) , а также от того, вызывается ли сообщение **BM \_ сетимаже** . Возможны следующие результаты.



| \_Значок BS или \_ набор битовых карт BS? | \_Вызвана СЕТИМАЖЕ BM? | Результат              |
|-----------------------------|----------------------|---------------------|
| Да                         | Да                  | Отображать только значок.     |
| Нет                          | Да                  | Отображение значка и текста. |
| Да                         | Нет                   | Показывать только текст.     |
| Нет                          | Нет                   | Показывать только текст      |
