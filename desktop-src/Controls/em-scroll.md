# <a name="em_scroll-message"></a>\_Сообщение с прокруткой EM

Прокручивает текст по вертикали в многострочном элементе управления Edit. Это сообщение эквивалентно отправке сообщения [**WM \_ VSCROLL**](wm-vscroll.md) в элемент управления Edit. Это сообщение можно отправить либо в элемент управления "поле ввода", либо в элемент управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Действие, которое займет полоса прокрутки. Этот параметр может принимать одно из указанных ниже значений.



| Значение                                                                                                                                                   | Значение                           |
|---------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------|
| <span id="SB_LINEDOWN"></span><span id="sb_linedown"></span><dl> <dt>**SB \_ линедовн**</dt> </dl> | Выполняет прокрутку вниз на одну строку.<br/> |
| <span id="SB_LINEUP"></span><span id="sb_lineup"></span><dl> <dt>**построителя \_**</dt> </dl>       | Выполняет прокрутку вверх на одну строку.<br/>   |
| <span id="SB_PAGEDOWN"></span><span id="sb_pagedown"></span><dl> <dt>**SB \_ PageDown**</dt> </dl> | Выполняет прокрутку вниз на одну страницу.<br/> |
| <span id="SB_PAGEUP"></span><span id="sb_pageup"></span><dl> <dt>**SB \_ PageUp**</dt> </dl>       | Выполняет прокрутку вверх на одну страницу.<br/>   |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если сообщение прошло успешно, [**HIWORD**](/previous-versions/windows/desktop/legacy/ms632657(v=vs.85)) возвращает значение **true**, а [**ловорд**](/previous-versions/windows/desktop/legacy/ms632659(v=vs.85)) — число строк, прокручиваемых командой. Возвращаемое число может не совпадать с фактическим числом строк, прокручеясь, если прокрутка перемещается в начало или конец текста. Если параметр *wParam* задает недопустимое значение, возвращается значение **false**.

## <a name="remarks"></a>Комментарии

Для прокрутки до определенной позиции строки или символа используйте сообщение [**EM \_ линескролл**](em-linescroll.md) . Чтобы прокрутить курсор на представление, используйте сообщение [**EM \_ скроллкарет**](em-scrollcaret.md) .

**Расширенное редактирование:** Поддерживается в Microsoft Rich Edit 1,0 и более поздних версиях. Дополнительные сведения о совместимости расширенных версий редактирования с различными версиями системы см. в разделе [Общие сведения об элементах управления редактированием](about-rich-edit-controls.md).
