# <a name="em_settypographyoptions-message"></a>\_Сообщение СЕТТИПОГРАФЙОПТИОНС EM

Задает текущее состояние типографских параметров для элемента управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Задает одно или оба из следующих значений.



| Значение                                                                                                                                                                                    | Значение                                                                                 |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| <span id="TO_ADVANCEDTYPOGRAPHY_"></span><span id="to_advancedtypography_"></span><dl> <dt>**Для \_ АДВАНЦЕДТИПОГРАФИ**</dt> </dl> | Включено дополнительное разбиение строк и форматирование строк. <br/>                    |
| <span id="TO_SIMPLELINEBREAK"></span><span id="to_simplelinebreak"></span><dl> <dt>**\_симплелинебреак**</dt> </dl>             | Более быстрое разбиение строк для простого текста (требуется **\_ адванцедтипографи**). <br/> |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Маска, состоящая из одного или нескольких флагов в параметре *wParam*. Будут установлены или сняты только флаги, установленные в этой маске. Это позволяет устанавливать или снимать один флаг без считывания текущих состояний флагов.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если *wParam* является допустимым; в противном случае — **значение false**.

## <a name="remarks"></a>Remarks

При необходимости расширенное разбиение строк включается автоматически элементом управления Rich Editing, например для обработки сложных наборов символов, таких как арабский и иврит, а также для математики. Он также необходим для выравнивания абзацев, расстановки переносов и других типографских функций.
