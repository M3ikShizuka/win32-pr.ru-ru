# <a name="lvm_getcallbackmask-message"></a>\_Сообщение LVM жеткаллбаккмаск

Возвращает маску обратного вызова для элемента управления "представление списка". Это сообщение можно отправить явно или с помощью макроса [**\_ жеткаллбаккмаск ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_getcallbackmask) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает маску обратного вызова.
