# <a name="tb_getimagelistcount-message"></a>\_Сообщение ЖЕТИМАЖЕЛИСТКАУНТ ТБ

Возвращает число списков изображений, связанных с панелью инструментов.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает число списков изображений.
