# <a name="cb_limittext-message"></a>\_Сообщение ЛИМИТТЕКСТ CB

Ограничивает длину текста, который пользователь может ввести в поле ввода поля со списком.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Максимальное число **тчарс** , которое может ввести пользователь, не включая завершающий символ null. Если этот параметр равен нулю, длина текста ограничена 0x7FFFFFFE символами.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение всегда равно **true**.

## <a name="remarks"></a>Remarks

Если поле со списком не имеет стиля [**CBS \_ аутохскролл**](combo-box-styles.md) , Установка предельного размера текста больше, чем размер элемента управления "поле ввода", не оказывает никакого влияния.

Сообщение **\_ лимиттекст CB** ограничивает только текст, который может ввести пользователь. Он не влияет на любой текст, уже находящиеся в элементе управления "поле ввода" при отправке сообщения, и на длину текста, скопированного в элемент управления "поле ввода", при выборе строки в списке.

Ограничение по умолчанию для текста, которое пользователь может ввести в поле ввода, — 30 000 **тчарс**.
