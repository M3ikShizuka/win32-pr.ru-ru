# <a name="em_getwordbreakproc-message"></a>\_Сообщение ЖЕТВОРДБРЕАКПРОК EM

Возвращает адрес текущей функции WordWrap. Это сообщение можно отправить либо в элемент управления "поле ввода", либо в элемент управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение указывает адрес функции WordWrap, определяемой приложением. Если функция WordWrap не существует, возвращается значение **null** .

## <a name="remarks"></a>Remarks

Функция WordWrap сканирует текстовый буфер, содержащий текст, который должен быть отправлен на экран, с поиском первого слова, которое не умещается на текущей линии вывода. Функция WordWrap помещает это слово в начало следующей строки на экране. Функция WordWrap определяет точку, в которой система должна разбивать строку текста для элементов управления многострочного редактирования, обычно в виде символа пробела, разделяющего два слова.

**Расширенное редактирование:** Поддерживается в Microsoft Rich Edit 1,0 и более поздних версиях. Дополнительные сведения о совместимости расширенных версий редактирования с различными версиями системы см. в разделе [Общие сведения об элементах управления редактированием](about-rich-edit-controls.md).
