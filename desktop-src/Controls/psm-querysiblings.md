# <a name="psm_querysiblings-message"></a>\_Сообщение ПСМ куерисиблингс

Отправляется на страницу свойств, которая затем пересылает сообщение каждой из страниц. Это сообщение можно отправить явным образом или с помощью макроса [**пропшит \_ куерисиблингс**](/windows/desktop/api/Prsht/nf-prsht-propsheet_querysiblings) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Первый определяемый приложением параметр.

</dd> <dt>

*lParam* 
</dt> <dd>

Второй параметр, определяемый приложением.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ненулевое значение со страницы на странице свойств или нуль, если ни одна страница не возвращает ненулевое значение.

## <a name="remarks"></a>Remarks

Если страница возвращает ненулевое значение, вкладка свойств не отправляет сообщение на последующие страницы.
