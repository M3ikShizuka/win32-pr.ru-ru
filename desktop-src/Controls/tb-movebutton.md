# <a name="tb_movebutton-message"></a>\_Сообщение МОВЕБУТТОН ТБ

Перемещает кнопку из одного индекса в другой.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Отсчитываемый от нуля индекс перемещаемой кнопки.

</dd> <dt>

*lParam* 
</dt> <dd>

Отсчитываемый от нуля индекс, по которому будет перемещена кнопка.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ненулевое значение в случае успеха или ноль в противном случае.
