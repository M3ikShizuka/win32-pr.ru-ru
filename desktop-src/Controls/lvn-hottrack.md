# <a name="lvn_hottrack-notification-code"></a>\_Код уведомления ЛВН хоттракк

Посылается элементом управления "представление списка", когда пользователь перемещает указатель мыши на элемент. Этот код уведомления отправляется только элементами управления "представление списка", имеющим расширенный стиль представления списка [**LVS \_ ex \_ траккселект**](extended-list-view-styles.md) . Он отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
LVN_HOTTRACK

    lpnmlv = (LPNMLISTVIEW) lParam;
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмлиствиев**](/windows/win32/api/commctrl/ns-commctrl-nmlistview) , содержащую сведения об этом коде уведомления. Члены этой структуры **Член iItem**, **iSubItem** и **птактион** содержат сведения об элементе. Принимающее приложение может изменить элемент **Член iItem** , чтобы указать элемент, который будет выбран. Если для **Член iItem** задано значение-1, элемент не будет выбран.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ноль, чтобы позволить представлению списка выполнять нормальную обработку выбора. Если приложение возвращает ненулевое значение, элемент не будет выбран.
