# <a name="rb_moveband-message"></a>\_Сообщение МОВЕБАНД RB

Перемещает диапазон из одного индекса в другой.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Отсчитываемый от нуля индекс перемещаемого диапазона.

</dd> <dt>

*lParam* 
</dt> <dd>

Отсчитываемый от нуля индекс новой позиции полосы.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ненулевое значение в случае успеха или ноль в противном случае.

## <a name="remarks"></a>Remarks

Скорее всего, это сообщение изменит индекс других полос в элементе управления "Главная панель". Если полоса перемещена из индекса 6 в индекс 0, то все полосы в между будут иметь индекс, увеличивающийся на единицу.

значение *lParam* никогда не должно превышать число полос минус единица. Количество полос можно получить с помощью сообщения [**RB \_ жетбандкаунт**](rb-getbandcount.md) .
