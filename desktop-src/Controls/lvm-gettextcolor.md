# <a name="lvm_gettextcolor-message"></a>\_Сообщение LVM жеттекстколор

Извлекает цвет текста элемента управления "представление списка". Это сообщение можно отправить явно или с помощью макроса [**\_ жеттекстколор ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_gettextcolor) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>
