# <a name="em_filelinefromchar-message"></a>\_Сообщение ФИЛЕЛИНЕФРОМЧАР EM

Возвращает индекс строки, содержащей указанный индекс символа в многострочном элементе управления Edit, независимо от того, как линии отображаются на экране. Индекс символа — это Отсчитываемый от нуля индекс символа, начиная с начала элемента управления "поле ввода".

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Индекс символа, содержащегося в строке, номер которой должен быть получен. Если этот параметр имеет значение-1, **EM \_ филелинефромчар** получает номер строки текущей строки (строка, содержащая курсор) или, если имеется выделенный фрагмент, номер строки, содержащей начало выделения.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение — это номер строки (начинающийся с нуля) строки, содержащей индекс символа, заданный параметром *wParam*, независимо от того, как линии отображаются на экране.
