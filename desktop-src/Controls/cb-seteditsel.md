# <a name="cb_seteditsel-message"></a>\_Сообщение СЕТЕДИТСЕЛ CB

Приложение отправляет сообщение **\_ сетедитсел CB** для выбора символов в элементе управления "поле ввода" поля со списком.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется.

</dd> <dt>

*lParam* \[ окне\]
</dt> <dd>

[**Ловорд**](/previous-versions/windows/desktop/legacy/ms632659(v=vs.85)) параметра *lParam* указывает начальную точку. Если **ловорд** имеет значение-1, то выбор, если он есть, будет удален.

[**HIWORD**](/previous-versions/windows/desktop/legacy/ms632657(v=vs.85)) параметра *lParam* задает конечную точку. Если **HIWORD** имеет значение-1, то выбирается весь текст из начальной позиции к последнему символу в элементе управления "поле ввода".

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если сообщение прошло удачно, возвращается значение **true**. Если сообщение отправляется в поле со списком с помощью стиля [**\_ DROPDOWNLIST**](combo-box-styles.md) , то это значение — CB \_ Err.

## <a name="remarks"></a>Комментарии

Позиции отсчитываются от нуля. Первый символ элемента управления "поле ввода" находится в нулевой позиции. Первый символ после последнего выбранного символа находится в конечной позиции. Например, чтобы выбрать первые четыре символа элемента управления "поле ввода", используйте начальную точку 0 и конечную точку 4.
