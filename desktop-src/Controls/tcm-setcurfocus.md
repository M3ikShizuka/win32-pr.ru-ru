# <a name="tcm_setcurfocus-message"></a>\_Сообщение СЕТКУРФОКУС TCM

Устанавливает фокус на указанную вкладку в элементе управления "Вкладка". Это сообщение можно отправить явным образом или с помощью макроса [**табктрл \_ сеткурфокус**](/windows/desktop/api/Commctrl/nf-commctrl-tabctrl_setcurfocus) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Индекс вкладки, получающей фокус.

</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Комментарии

Если элемент управления "Вкладка" имеет стиль [**\_ кнопок TCS**](tab-control-styles.md) (режим кнопки), вкладка с фокусом может отличаться от выбранной вкладки. Например, если выбрана вкладка, пользователь может нажать клавиши со стрелками, чтобы установить фокус на другую вкладку, не изменяя выбранную вкладку. В режиме кнопки **TCM \_ сеткурфокус** устанавливает фокус ввода на кнопку, связанную с указанной вкладкой, но не изменяет выбранную вкладку.

Если в элементе управления "Вкладка" отсутствует [**стиль \_ кнопок TCS**](tab-control-styles.md) , изменение фокуса также приведет к изменению выбранной вкладки. В этом случае элемент управления "Вкладка" отправляет коды уведомлений [ТКН \_ Селчангинг](tcn-selchanging.md) и [ТКН \_ селчанже](tcn-selchange.md) в родительское окно.
