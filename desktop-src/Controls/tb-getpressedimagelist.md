# <a name="tb_getpressedimagelist-message"></a>\_Сообщение ЖЕТПРЕССЕДИМАЖЕЛИСТ ТБ

Возвращает список изображений, используемый элементом управления Toolbar для отображения кнопок в нажатом состоянии.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Должен равняться нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Должен равняться нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает маркер в список изображений или **значение NULL** , если список изображений не задан.
