# <a name="pgm_setborder-message"></a>\_Сообщение СЕТБОРДЕР PGM

Задает текущий размер границы для элемента управления страничного навигатора. Это сообщение можно отправить явным образом или использовать макрос [**\_ сетбордер пейджера**](/windows/desktop/api/Commctrl/nf-commctrl-pager_setborder) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>

Новый размер границы в пикселях. Это значение не должно быть больше, чем кнопка страничного навигатора, или меньше нуля. Если значение *lParam* слишком велико, граница будет иметь тот же размер, что и кнопка. Если параметр *lParam* имеет отрицательное значение, то размер границы будет обнулен.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение типа INT, которое содержит предыдущий размер границы (в пикселях).
