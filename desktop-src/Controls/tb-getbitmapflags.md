# <a name="tb_getbitmapflags-message"></a>\_Сообщение ЖЕТБИТМАПФЛАГС ТБ

Получает флаги, описывающие тип используемого точечного рисунка.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение типа **DWORD** , описывающее тип точечного рисунка, который следует использовать. Если для этого возвращаемого значения установлен \_ большой флаг ТББФ, приложения должны использовать крупные точечные рисунки (24 x 24); в противном случае приложения должны использовать небольшие точечные рисунки (16 x 16). Все остальные биты зарезервированы.

## <a name="remarks"></a>Remarks

Значение, возвращаемое **ТБ \_ жетбитмапфлагс** , — это только рекомендации. Элемент управления ToolBar рекомендует крупные или небольшие растровые изображения в зависимости от того, выбраны ли для пользователя крупные или малые шрифты.
