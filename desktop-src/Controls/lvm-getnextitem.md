# <a name="lvm_getnextitem-message"></a>\_Сообщение LVM жетнекститем

Выполняет поиск элемента представления списка с указанными свойствами и несет указанную связь с указанным элементом. Это сообщение можно отправить явно или с помощью макроса [**\_ жетнекститем ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_getnextitem) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Индекс элемента, с которого начинается поиск, или значение-1, чтобы найти первый элемент, соответствующий указанным флагам. Сам указанный элемент исключен из поиска.

</dd> <dt>

*lParam* 
</dt> <dd>

Задает связь с элементом, указанным в параметре *wParam*. Это может быть одно или несколько из следующих значений:

## <a name="return-value"></a>Возвращаемое значение

Возвращает индекс следующего элемента, если он был успешным, или значение-1 в противном случае.

## <a name="remarks"></a>Комментарии

Обратите внимание, что следующие флаги, предназначенные для использования только с Windows Vista, являются взаимоисключающими с любыми другими используемыми флагами: ЛВНИ \_ висиблеонли, лвни \_ САМЕГРАУПОНЛИ, лвни \_ висиблеордер, ЛВНИ \_ директионмаск и лвни \_ статемаск.
