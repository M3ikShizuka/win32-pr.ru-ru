В этом разделе описывается пример кода свойства. Он содержит следующие подразделы:

-   [Описание](#description)

-   [Загрузка образца](#downloading-the-sample)

## <a name="description"></a>Описание

В примере свойства показано, как реализовать три различных стиля страниц свойств: модальный, немодальный и мастер. В нем также показано, как с помощью функции [*пропшитпрок*](/windows/desktop/api/Prsht/nc-prsht-pfnpropsheetcallback) изменить диалоговое окно страницы свойств до или после его создания.

## <a name="minimum-requirements"></a>Минимальные требования



| Продукт          | Version                              |
|------------------|--------------------------------------|
| DLL              | comctl32.dll версии 4,0             |
| Операционная система | Windows 95, Microsoft Windows NT 3,1 |



 

## <a name="downloading-the-sample"></a>Загрузка образца

Пример свойства устанавливается как часть [пакета средств разработки программного обеспечения (SDK) для Windows](https://msdn.microsoft.com/windows/bb980924.aspx) и доступен в следующем расположении.



| Расположение    | Путь или URL-адрес                                                                                                 |
|-------------|----------------------------------------------------------------------------------------------------------|
| Пакет Windows SDK | % Program Files% \\ Microsoft \\ Windows SDK \\ \[ номер версии \] \\ примеры \\ винуи \\ элементы управления \\ Общие \\ Свойства |



 

## <a name="building-the-sample"></a>Построение образца

Чтобы построить пример с помощью командной строки, выполните следующую команду:

1.  Откройте окно командной строки и перейдите в каталог проекта.
2.  Введите `msbuild [project file]`.

Чтобы построить пример с помощью Visual Studio, сделайте следующее:

1.  Откройте проводник Windows и перейдите в каталог проекта.
2.  Дважды щелкните значок VCPROJ-файла, чтобы открыть проект в Visual Studio.
3.  В меню **Сборка** выберите пункт **построить решение** , чтобы выполнить сборку решения.
