# <a name="tree-view-control-item-states"></a>Состояния элементов элемента управления Tree-View

В этом разделе перечислены флаги состояния элемента, используемые для указания состояния элемента в элементе управления "дерево-представление".



| Константа                                                                                                                                                                        | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TVIS_BOLD"></span><span id="tvis_bold"></span><dl> <dt>**ТВИС \_ Bold**</dt> </dl>                               | Элемент выделен полужирным шрифтом.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="TVIS_CUT"></span><span id="tvis_cut"></span><dl> <dt>**ТВИС \_ Вырезать**</dt> </dl>                                  | Элемент выбирается как часть операции вырезания и вставки. <br/>                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="TVIS_DROPHILITED"></span><span id="tvis_drophilited"></span><dl> <dt>**ТВИС \_ дрофилитед**</dt> </dl>          | Элемент выбран как цель перетаскивания.<br/>                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="TVIS_EXPANDED"></span><span id="tvis_expanded"></span><dl> <dt>**\_развернутый твис**</dt> </dl>                   | Список дочерних элементов элемента в настоящее время развернут; Это значит, что дочерние элементы видимы. Это значение применяется только к родительским элементам.<br/>                                                                                                                                                                                                                                                                                                                  |
| <span id="TVIS_EXPANDEDONCE"></span><span id="tvis_expandedonce"></span><dl> <dt>**ТВИС \_ експандедонце**</dt> </dl>       | Список дочерних элементов элемента был расширен по крайней мере один раз. Коды уведомлений [ТВН \_ Итемекспандинг](tvn-itemexpanding.md) и [ТВН \_ итемекспандед](tvn-itemexpanded.md) не формируются для родительских элементов, для которых это состояние задано в ответ на [**сообщение \_ расширения TVM**](tvm-expand.md) . Использование тве \_ сворачивания и тве \_ Коллапсересет с **TVM \_ expand** приведет к сбросу этого состояния. Это значение применяется только к родительским элементам. <br/> |
| <span id="TVIS_EXPANDPARTIAL"></span><span id="tvis_expandpartial"></span><dl> <dt>**ТВИС \_ експандпартиал**</dt> </dl>    | **Версия 4,70**. Частично развернутый элемент древовидного представления. В этом состоянии видимыми являются только некоторые, но не все дочерние элементы, и отображается символ «плюс» родительского элемента. <br/>                                                                                                                                                                                                                                                                              |
| <span id="TVIS_SELECTED"></span><span id="tvis_selected"></span><dl> <dt>**ТВИС \_ выбрано**</dt> </dl>                   | Элемент выбран. Его внешний вид зависит от того, имеет ли он фокус. Элемент будет отображаться с использованием системных цветов для выбора. <br/>                                                                                                                                                                                                                                                                                                              |
| <span id="TVIS_OVERLAYMASK"></span><span id="tvis_overlaymask"></span><dl> <dt>**ТВИС \_ оверлаймаск**</dt> </dl>          | Маска для битов, используемых для указания индекса изображения оверлея элемента.<br/>                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="TVIS_STATEIMAGEMASK"></span><span id="tvis_stateimagemask"></span><dl> <dt>**ТВИС \_ статеимажемаск**</dt> </dl> | Маска для битов, используемых для указания индекса изображения состояния элемента.<br/>                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="TVIS_USERMASK"></span><span id="tvis_usermask"></span><dl> <dt>**ТВИС \_ усермаск**</dt> </dl>                   | То же, что и **ТВИС \_ статеимажемаск**.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                     |



## <a name="remarks"></a>Remarks

При установке или извлечении индекса изображения наложения элемента или индекса изображения состояния необходимо указать следующие маски в элементе **статемаск** структуры [**твитем**](/windows/win32/api/commctrl/ns-commctrl-tvitema) :

-   **ТВИС \_ оверлаймаск**
-   **ТВИС \_ статеимажемаск**
-   **ТВИС \_ усермаск**

Эти значения также можно использовать для маскирования битов состояния, которые не представляют интереса.
