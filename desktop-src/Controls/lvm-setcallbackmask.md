# <a name="lvm_setcallbackmask-message"></a>\_Сообщение LVM сеткаллбаккмаск

Изменяет маску обратного вызова для элемента управления "представление списка". Это сообщение можно отправить явно или с помощью макроса [**\_ сеткаллбаккмаск ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_setcallbackmask) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

| Значение                                                                                                                                                                           | Значение                                                                                            |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| <span id="LVIS_CUT"></span><span id="lvis_cut"></span><dl> <dt>**ЛВИС \_ Вырезать**</dt> </dl>                                  | Элемент помечен для операции вырезания и вставки.<br/>                                       |
| <span id="LVIS_DROPHILITED"></span><span id="lvis_drophilited"></span><dl> <dt>**ЛВИС \_ дрофилитед**</dt> </dl>          | Элемент выделяется как цель перетаскивания.<br/>                                      |
| <span id="LVIS_FOCUSED"></span><span id="lvis_focused"></span><dl> <dt>**ЛВИС \_**</dt> </dl>                      | Элемент находится в фокусе.<br/>                                                                 |

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если успешно, или **false** в противном случае.

## <a name="remarks"></a>Комментарии

*Маска обратного вызова* элемента управления "представление списка" представляет собой набор битовых флагов, указывающих состояния элементов, для которых приложение, а не элемент управления, сохраняет текущие данные. Маска обратного вызова применяется ко всем элементам элемента управления, в отличие от конструкции элемента обратного вызова, которая применяется к конкретному элементу. По умолчанию маска обратного вызова равна нулю, то есть элемент управления "список" хранит все сведения о состоянии элемента. После создания элемента управления "представление списка" и инициализации его элементов можно отправить сообщение **LVM \_ сеткаллбаккмаск** , чтобы изменить маску обратного вызова. Чтобы получить текущую маску обратного вызова, отправьте сообщение [**LVM \_ жеткаллбаккмаск**](lvm-getcallbackmask.md) .

Дополнительные сведения о изображениях наложения и изображениях состояний см. в разделе [добавление List-View списков изображений](using-list-view-controls.md).

Дополнительные сведения о обратных вызовах в представлении списка см. [в разделе элементы обратного вызова и маска обратного вызова](list-view-controls-overview.md).
