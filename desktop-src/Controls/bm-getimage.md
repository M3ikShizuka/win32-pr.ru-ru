# <a name="bm_getimage-message"></a>BM, \_ сообщение о СОобразе

Извлекает маркер изображения (значок или растровое изображение), связанного с кнопкой.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Тип изображения, связываемого с кнопкой. Этот параметр может принимать одно из указанных ниже значений.



| Значение                                                                                                                                                      | Значение              |
|------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------|
| <span id="IMAGE_BITMAP"></span><span id="image_bitmap"></span><dl> <dt>**\_точечный рисунок изображения**</dt> </dl> | Точечный рисунок.<br/> |
| <span id="IMAGE_ICON"></span><span id="image_icon"></span><dl> <dt>**\_значок изображения**</dt> </dl>       | Значок.<br/>  |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение — это обработчик изображения, если таковой имеется; в противном случае он имеет **значение NULL**.
