# <a name="lvm_getitemrect-message"></a>\_Сообщение LVM жетитемрект

Извлекает ограничивающий прямоугольник для всего элемента или его части в текущем представлении. Это сообщение можно отправить явно или с помощью макроса [**\_ жетитемрект ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_getitemrect) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* \[ окне\]
</dt> <dd>

Индекс элемента представления списка.

</dd> <dt>

*lParam* \[ в, out\]
</dt> <dd>

Указатель на структуру [**Rect**](/previous-versions//dd162897(v=vs.85)) , которая получает ограничивающий прямоугольник. При отправке сообщения **левый** элемент этой структуры используется для указания части элемента списка, из которой извлекается ограничивающий прямоугольник. Необходимо задать одно из следующих значений:



| Значение                                                                                                                                                                     | Значение                                                                                                         |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| <span id="LVIR_BOUNDS"></span><span id="lvir_bounds"></span><dl> <dt>**\_границы лвир**</dt> </dl>                   | Возвращает ограничивающий прямоугольник для всего элемента, включая значок и метку.<br/>                     |
| <span id="LVIR_ICON"></span><span id="lvir_icon"></span><dl> <dt>**\_значок лвир**</dt> </dl>                         | Возвращает ограничивающий прямоугольник значка или маленький значок.<br/>                                            |
| <span id="LVIR_LABEL"></span><span id="lvir_label"></span><dl> <dt>**\_Метка лвир**</dt> </dl>                      | Возвращает ограничивающий прямоугольник текста элемента.<br/>                                                     |
| <span id="LVIR_SELECTBOUNDS"></span><span id="lvir_selectbounds"></span><dl> <dt>**ЛВИР \_ селектбаундс**</dt> </dl> | Возвращает объединение ЛВИР \_ значка и \_ прямоугольников меток лвир, но исключает столбцы в представлении отчета.<br/> |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если успешно, или **false** в противном случае.
