# <a name="tcm_adjustrect-message"></a>\_Сообщение АДЖУСТРЕКТ TCM

Вычисляет область просмотра элемента управления вкладки по заданному прямоугольнику окна или вычисляет прямоугольник окна, который соответствует заданной отображаемой области. Это сообщение можно отправить явным образом или с помощью макроса [**табктрл \_ аджустрект**](/windows/desktop/api/Commctrl/nf-commctrl-tabctrl_adjustrect) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Выполняемая операция. Если этот параметр имеет **значение true**, *lParam* задает отображаемый прямоугольник и получает соответствующий прямоугольник окна. Если этот параметр имеет **значение false**, *lParam* задает прямоугольник окна и получает соответствующую область экрана.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**Rect**](/previous-versions//dd162897(v=vs.85)) , которая указывает заданный прямоугольник и получает вычисляемый прямоугольник.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Комментарии

Это сообщение относится только к элементам управления "Вкладка", расположенным вверху. Он не применяется к элементам управления "Вкладка", находящиеся на сторонах или внизу.
