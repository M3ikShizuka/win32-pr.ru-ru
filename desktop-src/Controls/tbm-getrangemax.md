# <a name="tbm_getrangemax-message"></a>\_Сообщение ТБМ жетранжемакс

Получает максимальную положение ползунка в TrackBar.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает 32-разрядное значение, указывающее максимальную позицию ползунка в диапазоне от минимума до максимального значения.
