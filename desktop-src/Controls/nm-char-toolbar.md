# <a name="nm_char-toolbar-notification-code"></a>\_Код уведомления (панель инструментов) NM

Отправляется панелью инструментов при получении сообщения [**WM \_ char**](/windows/desktop/inputdev/wm-char) . Этот код уведомления отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
NM_CHAR

    lpnmc = (LPNMCHAR) lParam;
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмчар**](/windows/win32/api/commctrl/ns-commctrl-nmchar) , которая содержит дополнительные сведения о символе, вызвавшем код уведомления. Элемент **двитемпрев** этой структуры содержит идентификатор команды, которая находится в активном виде, или значение-1, если ни один элемент не является активным. Элемент **двитемнекст** этой структуры содержит идентификатор команды, который будет активным, или-1, если ключ не соответствует сочетанию клавиш любого элемента.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , указывающее, что панель инструментов не должна обрабатывать символ, и **false** , чтобы панель инструментов обрабатывала символ.
