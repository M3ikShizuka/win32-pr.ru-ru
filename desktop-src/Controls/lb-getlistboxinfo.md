# <a name="lb_getlistboxinfo-message"></a>Сообщение жетлистбоксинфо балансировки нагрузки \_

Возвращает количество элементов на столбец в указанном списке.

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется; оно должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется; оно должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение — это количество элементов на столбец.
