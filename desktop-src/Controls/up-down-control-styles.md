# <a name="up-down-control-styles"></a>Стили элементов управления Up-Down

В этом разделе перечислены стили, используемые при создании элементов управления "вверх/вниз".



| Константа                                                                                                                                                            | Описание                                                                                                                                                                                                                                                                                                                                                                                               |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="UDS_ALIGNLEFT"></span><span id="uds_alignleft"></span><dl> <dt>**ДОМЕНОВ обновления \_ алигнлефт**</dt> </dl>       | Размещает элемент управления "вверх/вниз" рядом с левым ребром окна собеседника. Окно собеседника перемещается вправо, и его ширина уменьшается в соответствии с шириной элемента управления "вверх-вниз".<br/>                                                                                                                                                                                                   |
| <span id="UDS_ALIGNRIGHT"></span><span id="uds_alignright"></span><dl> <dt>**ДОМЕНОВ обновления \_ алигнригхт**</dt> </dl>    | Размещает элемент управления "вверх/вниз" рядом с правой границей окна собеседника. Ширина окна собеседника уменьшается в соответствии с шириной элемента управления "вверх-вниз".<br/>                                                                                                                                                                                                                          |
| <span id="UDS_ARROWKEYS"></span><span id="uds_arrowkeys"></span><dl> <dt>**ДОМЕНОВ обновления \_ арровкэйс**</dt> </dl>       | Заставляет элемент управления "вверх/вниз" увеличивать и уменьшать расположение при нажатии клавиш со СТРЕЛКАми вверх и вниз.<br/>                                                                                                                                                                                                                                                                          |
| <span id="UDS_AUTOBUDDY"></span><span id="uds_autobuddy"></span><dl> <dt>**ДОМЕНОВ обновления \_**</dt> </dl>       | Автоматически выбирает предыдущее окно в z-порядке в качестве дружественного окна элемента управления "вверх/вниз".<br/>                                                                                                                                                                                                                                                                                                |
| <span id="UDS_HORZ"></span><span id="uds_horz"></span><dl> <dt>**ДОМЕНОВ обновления \_ горизонтали**</dt> </dl>                      | Приводит к тому, что стрелки вверх-вниз элемента управления указываются слева и справа, а не вверх и вниз.<br/>                                                                                                                                                                                                                                                                                                            |
| <span id="UDS_HOTTRACK"></span><span id="uds_hottrack"></span><dl> <dt>**ДОМЕНОВ обновления \_ хоттракк**</dt> </dl>          | Заставляет элемент управления работать с "горячим отслеживанием". Это значит, что в элементе управления выделяется стрелка вверх и стрелка вниз, так как указатель проходит над ними. Для этого стиля требуется Windows 98 или Windows 2000. Если система работает под управлением Windows 95 или Windows NT 4,0, флаг игнорируется. Чтобы проверить, включено ли оперативное отслеживание, вызовите [**системпараметерсинфо**](/windows/desktop/api/winuser/nf-winuser-systemparametersinfoa). <br/> |
| <span id="UDS_NOTHOUSANDS"></span><span id="uds_nothousands"></span><dl> <dt>**ДОМЕНОВ обновления \_ тысячи**</dt> </dl> | Не вставляет разделитель тысяч между всеми тремя десятичными цифрами. <br/>                                                                                                                                                                                                                                                                                                                     |
| <span id="UDS_SETBUDDYINT"></span><span id="uds_setbuddyint"></span><dl> <dt>**ДОМЕНОВ обновления \_ сетбуддинт**</dt> </dl> | Заставляет элемент управления "вверх/вниз" задать текст собеседника (с помощью сообщения [**WM \_ SETTEXT**](/windows/desktop/winmsg/wm-settext) ) при изменении расположения. Текст состоит из расположения в формате десятичной или шестнадцатеричной строки.<br/>                                                                                                                                                             |
| <span id="UDS_WRAP"></span><span id="uds_wrap"></span><dl> <dt>**ДОМЕНОВ обновления \_ Перенос**</dt> </dl>                      | Приводит к тому, что позиция «переносится», если она увеличивается или уменьшается за концом диапазона.<br/>                                                                                                                                                                                                                                                                                 |
