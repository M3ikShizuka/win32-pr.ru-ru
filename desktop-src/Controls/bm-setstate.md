# <a name="bm_setstate-message"></a>\_Сообщение BM SETSTATE

Задает состояние выделения для кнопки. Состояние выделения указывает, будет ли кнопка выделена, как будто пользователь отправил ее. Это сообщение можно отправить явным образом или воспользоваться макросом [**кнопки \_ SetState**](/windows/desktop/api/Windowsx/nf-windowsx-button_setstate) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

**Логическое** значение, указывающее, выделяется ли кнопка. Значение **true** выделяет кнопку. Значение **false** удаляет все выделения.

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Это сообщение всегда возвращает ноль.

## <a name="remarks"></a>Комментарии

Выделение влияет только на внешний вид кнопки. Он не влияет на состояние флажка переключателя или на флажок.

Кнопка автоматически выделяется, когда пользователь наводит на нее курсор и нажимает и удерживает левую кнопку мыши. Выделение удаляется, когда пользователь отпускает кнопку мыши.
