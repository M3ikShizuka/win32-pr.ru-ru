# <a name="tb_setstyle-message"></a>\_Сообщение SETSTYLE ТБ

Задает стиль для элемента управления ToolBar.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>

Значение, указывающее стили, которые должны быть заданы для элемента управления. Это значение может быть сочетанием [стилей элементов управления панели инструментов](toolbar-control-and-button-styles.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.
