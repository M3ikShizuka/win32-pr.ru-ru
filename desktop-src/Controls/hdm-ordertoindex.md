# <a name="hdm_ordertoindex-message"></a>\_Сообщение ОРДЕРТОИНДЕКС HDM

Извлекает значение индекса для элемента на основе его порядка в элементе управления "заголовок". Это сообщение можно отправить явным образом или воспользоваться [**заголовком макроса \_ ордертоиндекс**](/windows/desktop/api/Commctrl/nf-commctrl-header_ordertoindex) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Порядок, в котором элемент отображается внутри элемента управления "заголовок" слева направо. Например, значение индекса элемента в дальнем левом столбце должно быть равно 0. Для следующего элемента справа будет использоваться значение 1 и т. д.

</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение типа INT, указывающее индекс элемента. Если параметр *wParam* является недопустимым (отрицательный или слишком большой), то возвращаемое значение равно *wParam*.
