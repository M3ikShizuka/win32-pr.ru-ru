# <a name="hdn_begintrack-notification-code"></a>\_Код уведомления ХДН бегинтракк

Сообщает родительскому окну элемента управления заголовка, что пользователь начал перетаскивание разделителя в элементе управления (т. е. пользователь нажал левую кнопку мыши, когда курсор мыши находится на разделителе в элементе управления "заголовок"). Этот код уведомления отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
HDN_BEGINTRACK

    pNMHeader = (LPNMHEADER) lParam; 
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмхеадер**](/windows/win32/api/commctrl/ns-commctrl-nmheadera) , содержащую сведения о элементе управления заголовка и элементе, разделитель которого нужно перетащить.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение false** , чтобы разрешить отслеживание разделителя, или **значение true** , чтобы предотвратить отслеживание.
