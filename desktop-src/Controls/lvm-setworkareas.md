# <a name="lvm_setworkareas-message"></a>\_Сообщение LVM сетворкареас

Задает рабочие области в элементе управления "представление списка". Это сообщение можно отправить явным образом или использовать макрос [**\_ сетворкареас ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_setworkareas) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Число структур в массиве по адресу *ЛПРК*. Максимально допустимое количество рабочих областей определяется \_ \_ ЗНАЧЕНИЕМ параметра LV Max воркареас.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на массив структур [**Rect**](/previous-versions//dd162897(v=vs.85)) , содержащих новые рабочие области элемента управления "представление списка". Значения в этих структурах находятся в клиентских координатах. Если этот параметр имеет **значение NULL**, то Рабочая область будет задаваться в клиентской области элемента управления. *wParam* задает количество структур в этом массиве.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение для этого сообщения не используется.
