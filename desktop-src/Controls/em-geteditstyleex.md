# <a name="em_geteditstyleex-message"></a>\_Сообщение ЖЕТЕДИТСТИЛИКС EM

Извлекает текущие расширенные флаги стиля правки.


```C++
#define EM_GETEDITSTYLEEX       (WM_USER + 276)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает расширенные флаги стиля редактирования, которые могут включать одно или несколько из следующих значений.



| Код возврата                                                                                                | Описание                                                                                                                                                                                                                                                  |
|------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**SES \_ ex \_ хандлефриендлюрл**</dt> </dl>  | Отображать ссылки на дружественные имена с одним и тем же цветом текста и подчеркиванием в качестве автоматических связей при условии, что временное форматирование не используется или использует автоцвет текста (по умолчанию: 0).<br/>                                                                       |
| <dl> <dt>**SES \_ ex \_ Мультисенсорная**</dt> </dl>         | Включите поддержку сенсорного ввода в расширенном редактировании. Это включает выделение, размещение курсора и вызов контекстного меню. Если этот флаг не установлен, касание эмулируется командами мыши, которые не принимают особенности сенсорного режима (по умолчанию: 0). <br/>      |
| <dl> <dt>**SES \_ ex \_ ноацетатеселектион**</dt> </dl> | Отображать выделенный текст, используя классический текст и цвета фона, а не фоновый цвет ацетате (по умолчанию: 0). <br/>                                                                                                               |
| <dl> <dt>**SES \_ EX ( \_ нематематический)**</dt> </dl>             | Отключить вставку математических зон (по умолчанию: 1). Чтобы включить редактирование и отображение математических выражений, отправьте сообщение [**EM \_ сетедитстиликс**](em-seteditstyleex.md) с параметром *wParam* Set в значение 0, а для параметра *lParam* — значение **SES \_ ex \_ Math**. <br/>                              |
| <dl> <dt>**\_важные SES \_**</dt> </dl>            | Отключить вставку таблиц. Сообщение [**EM \_ инсерттабле**](em-inserttable.md) возвращает значение **E \_ Fail** , а таблицы RTF пропускаются (значение по умолчанию: 0). <br/>                                                                                                  |
| <dl> <dt>**SES \_ ex \_ усесинглелине**</dt> </dl>      | Включение многострочного элемента управления в качестве однострочного элемента управления с возможностью прокрутки по вертикали, если высота одной строки превышает высоту окна (значение по умолчанию: 0). <br/>                                                                   |
| <dl> <dt>**SES \_ хидетемпформат**</dt> </dl>         | Скрытие временного форматирования, созданного при вызове [**итекстфонт. Reset**](/windows/desktop/api/Tom/nf-tom-itextfont-reset) с **томапплитмп**. Например, такое форматирование используется в средствах проверки орфографии для вывода волнистой подчеркивания в словах с ошибками.<br/> |
| <dl> <dt>**SES \_ ex \_ усемаусевпарам**</dt> </dl>     | Используйте параметр *wParam* при обработке сообщения [**WM \_ MOUSEMOVE**](/windows/desktop/inputdev/wm-mousemove) и не вызывайте [**жетасинккэйстате**](/windows/desktop/api/winuser/nf-winuser-getasynckeystate).<br/>                                                                                              |
