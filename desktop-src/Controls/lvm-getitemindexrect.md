# <a name="lvm_getitemindexrect-message"></a>\_Сообщение LVM жетитеминдексрект

Извлекает ограничивающий прямоугольник для всей или части подэлемента в текущем представлении элемента управления "представление списка". Отправьте это сообщение явным образом или с помощью макроса [**\_ жетитеминдексрект ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_getitemindexrect) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* \[ окне\]
</dt> <dd>

Указатель на структуру [**лвитеминдекс**](/windows/win32/api/commctrl/ns-commctrl-lvitemindex) для родительского элемента подэлемента. Вызывающий процесс отвечает за выделение этой структуры и задание ее членов. *wParam* не может иметь **значение NULL**.

</dd> <dt>

*lParam* \[ в, out\]
</dt> <dd>

Указатель на структуру [**Rect**](/previous-versions//dd162897(v=vs.85)) для получения координат. Вызывающий процесс отвечает за выделение этой структуры. Значение *lParam* не должно быть **null**. Установите **верхний** элемент в качестве индекса подэлемента. Установите для **левого** элемента одно из следующих значений, указывающее часть подэлемента, для которой должен быть получен ограничивающий прямоугольник.



| Значение                                                                                                                                                   | Значение                                                                                        |
|---------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------|
| <span id="LVIR_BOUNDS"></span><span id="lvir_bounds"></span><dl> <dt>**\_границы лвир**</dt> </dl> | Возвращает ограничивающий прямоугольник для всего вложенного элемента, включая значок и метку.<br/> |
| <span id="LVIR_ICON"></span><span id="lvir_icon"></span><dl> <dt>**\_значок лвир**</dt> </dl>       | Возвращает ограничивающий прямоугольник значка или маленький значок подэлемента.<br/>            |
| <span id="LVIR_LABEL"></span><span id="lvir_label"></span><dl> <dt>**\_Метка лвир**</dt> </dl>    | Возвращает ограничивающий прямоугольник для текста подэлемента.<br/>                                 |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если успешно, или **false** в противном случае.
