# <a name="mcm_getcurrentview-message"></a>\_Сообщение MCM жеткуррентвиев

Возвращает текущее представление календаря. Это сообщение можно отправить явным образом или с помощью макроса [**монскал \_ жеткуррентвиев**](/windows/desktop/api/Commctrl/nf-commctrl-monthcal_getcurrentview) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Должен равняться нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Должен равняться нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Текущее представление. Одно из следующих значений.



| Код возврата                                                                                  | Описание              |
|----------------------------------------------------------------------------------------------|--------------------------|
| <dl> <dt>**МКМВ \_ месяц**</dt> </dl>   | Представление за месяц.<br/> |
| <dl> <dt>**МКМВ \_ год**</dt> </dl>    | Ежегодное представление.<br/>  |
| <dl> <dt>**МКМВное \_ десятилетие**</dt> </dl>  | Представление десятилетия.<br/>  |
| <dl> <dt>**МКМВ \_ век**</dt> </dl> | Представление "век".<br/> |
