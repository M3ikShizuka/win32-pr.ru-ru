# <a name="tbm_gettic-message"></a>\_Сообщение ТБМ жеттик

Извлекает логическую точку деления в TrackBar. Логическое положение может быть любым из целочисленных значений в диапазоне от минимума до максимальной позиции ползунка.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Отсчитываемый от нуля индекс, указывающий деление. Допустимые индексы находятся в диапазоне от нуля до двух меньше, чем число тактов, возвращенное сообщением [**ТБМ \_ жетнумтикс**](tbm-getnumtics.md) .

</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает логическую точку указанного деления или-1, если параметр *wParam* не указывает допустимый индекс.
