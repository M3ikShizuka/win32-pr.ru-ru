# <a name="em_setimemodebias-message"></a>\_Сообщение СЕТИМЕМОДЕБИАС EM

Задайте смещение режима редактора метода ввода (IME) для элемента управления расширенного редактирования.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Значение смещения режима IME. Может быть одним из следующих.



| Значение                                                                                                                                                                                        | Значение                                    |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------|
| <span id="IMF_SMODE_PLAURALCLAUSE"></span><span id="imf_smode_plauralclause"></span><dl> <dt>**интеллектуальный фильтр сообщений \_ смоде \_ плауралклаусе**</dt> </dl> | Задает для смещения режима IME значение Name.<br/> |
| <span id="IMF_SMODE_NONE"></span><span id="imf_smode_none"></span><dl> <dt>**интеллектуальный фильтр сообщений \_ смоде \_ None**</dt> </dl>                            | Без смещения.<br/>                        |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Это должно быть то же значение, что и параметр *wParam*.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Это сообщение возвращает новый параметр смещения режима IME.

## <a name="remarks"></a>Комментарии

Когда редактор IME создает список альтернативных символов, это сообщение задает условия, по которым некоторые из вариантов будут отображаться в верхней части списка.

Чтобы задать уровень смещения в режиме платформы текстовых служб (TSF), используйте [**EM \_ сетктфмодебиас**](em-setctfmodebias.md).

Перед вызовом этой функции приложение должно вызвать [**EM \_ исиме**](em-isime.md) .
