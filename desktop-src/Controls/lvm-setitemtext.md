# <a name="lvm_setitemtext-message"></a>\_Сообщение LVM сетитемтекст

Изменяет текст элемента представления списка или подэлемента. Это сообщение можно отправить явно или с помощью макроса [**\_ сетитемтекст ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_setitemtext) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Отсчитываемый от нуля индекс элемента списка представления.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**лвитем**](/windows/win32/api/commctrl/ns-commctrl-lvitema) . Элемент **iSubItem** является индексом подэлемента или может быть равен нулю, чтобы задать метку элемента. Элемент **псзтекст** является адресом строки, завершающейся нулем, содержащей новый текст. Он также может иметь **значение NULL**. Элемент **псзтекст** также может быть LPSTR \_ тексткаллбакк для указания элемента обратного вызова, для которого в родительском окне сохраняется текст. В этом случае элемент управления "представление списка" отправляет родительский код уведомления [**ЛВН \_ жетдиспинфо**](lvn-getdispinfo.md) , когда он нуждается в тексте.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если сообщение отправляется явным образом, оно возвращает **true** в случае успеха или **false** в противном случае.
