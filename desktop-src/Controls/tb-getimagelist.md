# <a name="tb_getimagelist-message"></a>\_Сообщение ТБ

Извлекает список изображений, используемый элементом управления Toolbar для отображения кнопок в состоянии по умолчанию. Элемент управления ToolBar использует этот список изображений для вывода кнопок, если они не являются неактивными или отключенными.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Должен равняться нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Должен равняться нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает маркер в список изображений или **значение NULL** , если список изображений не задан.
