# <a name="cbem_getcombocontrol-message"></a>\_Сообщение кбем жеткомбоконтрол

Возвращает маркер для элемента управления дочернего поля со списком.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает маркер элемента управления "поле со списком" в элементе управления ComboBoxEx.
