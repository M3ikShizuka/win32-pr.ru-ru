# <a name="em_selectiontype-message"></a>\_Сообщение СЕЛЕКТИОНТИПЕ EM

Определяет тип выбора для элемента управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если выделение пустое, возвращаемое значение будет SEL \_ пустым.

Если выделение не пустое, возвращаемое значение представляет собой набор флагов, содержащих одно или несколько из следующих значений.



| Код возврата                                                                                     | Описание                                 |
|-------------------------------------------------------------------------------------------------|---------------------------------------------|
| <dl> <dt>**\_текст SEL**</dt> </dl>        | Текст.<br/>                            |
| <dl> <dt>**\_объект SEL**</dt> </dl>      | По крайней мере один COM-объект.<br/>         |
| <dl> <dt>**SEL \_ (многосимвольный)**</dt> </dl>   | Более одного символа текста.<br/> |
| <dl> <dt>**\_объект SEL**</dt> </dl> | Более одного COM-объекта.<br/>        |



 

## <a name="remarks"></a>Комментарии

Это сообщение полезно при обработке [**\_ размера WM**](/windows/desktop/winmsg/wm-size) для родителя элемента управления неформатированного редактирования самого нижнего уровня.
