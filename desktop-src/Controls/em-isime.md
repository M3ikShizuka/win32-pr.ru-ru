# <a name="em_isime-message"></a>\_Сообщение ИСИМЕ EM

Определение с использованием текущего языкового стандарта ввода элемента управления редактирования является восточно-азиатским языковым стандартом.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если это языковой стандарт Восточной Азии. В противном случае возвращается **значение false**.
