# <a name="em_setcharformat-message"></a>\_Сообщение СЕТЧАРФОРМАТ EM

Задает форматирование символов в элементе управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Форматирование символов, которое применяется к элементу управления. Если этот параметр равен нулю, задается символьный формат по умолчанию. В противном случае может быть одним из следующих значений.



| Значение                                                                                                                                                                           | Значение                                                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="SCF_ALL"></span><span id="scf_all"></span><dl> <dt>**SCF \_ все**</dt> </dl>                                     | Применяет форматирование ко всему тексту в элементе управления. Недопустимо при **\_ выборе SCF** или **в \_ слове SCF**.<br/>                                                                                                                                                                                         |
| <span id="SCF_ASSOCIATEFONT"></span><span id="scf_associatefont"></span><dl> <dt>**\_АССОЦИАТЕФОНТ SCF**</dt> </dl>       | **RichEdit 4,1:** Связывает шрифт с заданным сценарием, тем самым изменяя шрифт по умолчанию для этого скрипта. Чтобы указать шрифт, используйте следующие члены [**CHARFORMAT2**](/windows/desktop/api/Richedit/ns-richedit-charformat2a): **ихеигхт**, **бчарсет**, **бпитчандфамили**, **сзфаценаме** и **LCID**.<br/>                     |
| <span id="SCF_ASSOCIATEFONT2"></span><span id="scf_associatefont2"></span><dl> <dt>**\_ASSOCIATEFONT2 SCF**</dt> </dl>    | **RichEdit 4,1:** Связывает шрифт суррогата (плоскость 2) с заданным сценарием, тем самым изменяя шрифт по умолчанию для этого скрипта. Чтобы указать шрифт, используйте следующие члены [**CHARFORMAT2**](/windows/desktop/api/Richedit/ns-richedit-charformat2a): **ихеигхт**, **бчарсет**, **бпитчандфамили**, **сзфаценаме** и **LCID**.<br/> |
| <span id="SCF_CHARREPFROMLCID"></span><span id="scf_charrepfromlcid"></span><dl> <dt>**\_ЧАРРЕПФРОМЛЦИД SCF**</dt> </dl> | Возвращает символ поработать из LCID.<br/>                                                                                                                                                                                                                                                   |
| <span id="SCF_DEFAULT"></span><span id="scf_default"></span><dl> <dt>**\_по умолчанию — SCF**</dt> </dl>                         | **RichEdit 4,1:** Задает шрифт по умолчанию для элемента управления. <br/>                                                                                                                                                                                                                                      |
| <span id="SPF_DONTSETDEFAULT"></span><span id="spf_dontsetdefault"></span><dl> <dt>**SPF \_ донтсетдефаулт**</dt> </dl>    | Предотвращает задание формата абзаца по умолчанию, если элемент управления Rich Edit пуст.<br/>                                                                                                                                                                                                             |
| <span id="SCF_NOKBUPDATE"></span><span id="scf_nokbupdate"></span><dl> <dt>**\_НОКБУПДАТЕ SCF**</dt> </dl>                | **RichEdit 4,1:** Предотвращает переключение клавиатуры для соответствия шрифту. Например, если установлен арабский шрифт, то обычно функция автоматической клавиатуры для языков с двунаправленным письмом изменяет клавиатуру на арабскую клавиатуру.<br/>                                                                                 |
| <span id="SCF_SELECTION"></span><span id="scf_selection"></span><dl> <dt>**Выбор параметров SCF \_**</dt> </dl>                   | Применяет форматирование к текущему выделенному фрагменту. Если выделение пусто, то к точке вставки применяется форматирование символов, а новый символьный формат действует только до тех пор, пока точка вставки не изменится.<br/>                                                                      |
| <span id="SPF_SETDEFAULT"></span><span id="spf_setdefault"></span><dl> <dt>**SPF \_ сетдефаулт**</dt> </dl>                | Задает атрибуты форматирования абзаца по умолчанию.<br/>                                                                                                                                                                                                                                              |

| <span id="SCF_USEUIRULES"></span><span id="scf_useuirules"></span><dl> <dt>**\_УСЕУИРУЛЕС SCF**</dt> </dl>                | **RichEdit 4,1:** Используется с **\_ выбором SCF**. Указывает, что формат получен из панели инструментов или другого средства пользовательского интерфейса, поэтому правила форматирования пользовательского интерфейса следует использовать вместо форматирования литерала.<br/>                                                                                                               |
| <span id="SCF_WORD"></span><span id="scf_word"></span><dl> <dt>**\_слово SCF**</dt> </dl>                                  | Применение форматирования к выбранному слову или словам. Если выделение пустое, но точка вставки находится внутри слова, форматирование применяется к слову. Значение **\_ слова SCF** должно использоваться в сочетании с значением выбора параметра **SCF \_** .<br/>                                        |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**чарформат**](/windows/win32/api/richedit/ns-richedit-charformata) , определяющую используемое форматирование символов. Изменяются только атрибуты форматирования, заданные элементом **двмаск** .

Microsoft Rich Edit 2,0 и более поздние версии: этот параметр может быть указателем на структуру [**CHARFORMAT2**](/windows/desktop/api/Richedit/ns-richedit-charformat2a) , которая является расширением структуры [**чарформат**](/windows/win32/api/richedit/ns-richedit-charformata) . Перед отправкой сообщения **EM \_ сетчарформат** задайте для элемента **кбсизе** структуры значение `sizeof(CHARFORMAT)` или, чтобы `sizeof(CHARFORMAT2)` указать, какая версия структуры используется.

Члены **сзфаценаме** и **бчарсет** могут быть перевышены, если они недопустимы для символов, например Arial для символов кандзи.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если операция завершилась удачно, возвращаемое значение будет ненулевым.

Если операция завершается ошибкой, возвращаемое значение равно нулю.

## <a name="remarks"></a>Комментарии

Если это сообщение отправляется несколько раз с теми же параметрами, то его воздействие на текст переключается. То есть Отправка сообщения после того, как выдает результат, отправляя сообщение дважды, отменяет результат и т. д.
