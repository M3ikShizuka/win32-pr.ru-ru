# <a name="ccm_setversion-message"></a>\_Сообщение СЕТВЕРСИОН CCM

Это сообщение используется для информирования элемента управления о том, что ожидается поведение, связанное с определенной версией.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Номер версии.

</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает версию, указанную в предыдущем сообщении **CCM \_ сетверсион** . Если параметру *wParam* задано значение, большее, чем текущая версия DLL, возвращается-1.

## <a name="remarks"></a>Комментарии

В некоторых случаях элемент управления может вести себя по-разному в зависимости от версии. В основном это относится к ошибкам, исправленным в более поздних версиях. Сообщение **CCM \_ сетверсион** позволяет информировать элемент управления о ожидаемом поведении. Вы можете определить, какую версию вы указали, отправив сообщение [**CCM- \_ версии**](ccm-getversion.md) . Пример использования этого сообщения см. в разделе [пользовательский вывод с помощью List-View и Tree-View элементов управления](custom-draw.md).

Если установлена версия ComCtl32.dll 6, независимо от значения, заданного в параметре *wParam*, сообщение **CCM \_ сетверсион** Возвращает версию 6.

> [!Note]  
> Это сообщение задает только номер версии для элемента управления, в который он отправляется.
