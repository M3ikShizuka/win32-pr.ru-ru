# <a name="ttm_getmargin-message"></a>Сообщение ТТМи о \_ прибыли

Извлекает верхнее, левое, нижнее и правое поля, заданные для окна подсказки. Поле — это расстояние (в пикселях) между границей окна подсказки и текстом, содержащимся в окне подсказки.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**Rect**](/previous-versions//dd162897(v=vs.85)) , которая будет принимать сведения о полях. Элементы структуры **Rect** не определяют ограничивающий прямоугольник. Для этого сообщения члены структуры интерпретируется следующим образом:



| Значение                                                                                                                                   | Значение                                                                            |
|-----------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------|
| <span id="top"></span><span id="TOP"></span><dl> <dt>**Вверх**</dt> </dl>          | Расстояние между верхней границей и верхней частью текста подсказки в пикселях.<br/>         |
| <span id="left"></span><span id="LEFT"></span><dl> <dt>**слева**</dt> </dl>       | Расстояние между левой границей и левым концом текста подсказки в пикселях.<br/>   |
| <span id="bottom"></span><span id="BOTTOM"></span><dl> <dt>**Нижний**</dt> </dl> | Расстояние между нижней границей и нижней частью текста подсказки в пикселях.<br/>   |
| <span id="right"></span><span id="RIGHT"></span><dl> <dt>**Правильно**</dt> </dl>    | Расстояние между правой границей и правым концом текста подсказки в пикселях.<br/> |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение для этого сообщения не используется.

## <a name="remarks"></a>Комментарии

При создании элемента управления ToolTip все четыре поля по умолчанию имеют нулевое значение.
