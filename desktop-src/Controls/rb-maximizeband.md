# <a name="rb_maximizeband-message"></a>\_Сообщение МАКСИМИЗЕБАНД RB

Изменяет размер полосы в элементе управления главной панели до его идеального или самого крупного размера.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Отсчитываемый от нуля индекс диапазона, который необходимо развернуть.

</dd> <dt>

*lParam* 
</dt> <dd>

Указывает, следует ли использовать идеальную ширину полосы, если полоса развернута. Если это значение не равно нулю, будет использоваться идеальная ширина. Если это значение равно нулю, то полоса будет максимально велика.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение не используется.
