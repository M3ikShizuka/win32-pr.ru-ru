# <a name="em_getzoom-message"></a>\_Сообщение о СОХРАНЕНИИ EM

Возвращает текущий коэффициент масштабирования для многострочного элемента управления "поле ввода" или элемента управления Rich Edit. Ъявлению масштаба всегда находится в диапазоне от 1/64 до 64.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Получает числитель коэффициента масштабирования.

</dd> <dt>

*lParam* 
</dt> <dd>

Получает знаменатель коэффициента масштабирования.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Сообщение возвращает **значение true** , если обрабатывается сообщение, которое будет иметь значение, если параметр *wParam* и *lParam* не равны **null**.

## <a name="remarks"></a>Remarks

**Изменить:** поддерживается в Windows 10 1809 и более поздних версиях. В элементе управления "поле ввода" должен быть установлен расширенный набор стилей **ES \_ ex \_ масштабируемый** , чтобы это сообщение имело значение. см. раздел [Правка Управление расширенными стилями](edit-control-window-extended-styles.md). Дополнительные сведения об элементе управления "поле ввода" см. в разделе [Edit Controls](edit-controls.md).
