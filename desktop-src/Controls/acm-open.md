# <a name="acm_open-message"></a>ACM \_ открытое сообщение

Открывает ролик AVI и отображает его первый кадр в элементе управления анимации. Это сообщение можно отправить явным образом или воспользоваться макросом [**анимации \_ Open**](/windows/desktop/api/Commctrl/nf-commctrl-animate_open) или [**анимировать \_ опенекс**](/windows/desktop/api/Commctrl/nf-commctrl-animate_openex) . Мы рекомендуем использовать версию этого сообщения в Юникоде, ACM \_ опенв.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

[Версия 4,71](common-control-versions.md) и более поздние версии. Экземпляр обработчика для модуля, из которого должен быть загружен ресурс. Присвойте этому параметру значение **null** , чтобы элемент управления использовал значение HINSTANCE, используемое для создания окна. Обратите внимание, что если окно создается библиотекой DLL, значение параметра *wParam* по умолчанию — это значение HINSTANCE библиотеки DLL, а не приложение, ВЫЗЫВАЮЩЕЕ библиотеку DLL.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на буфер, содержащий путь к AVI-файлу или имя ресурса AVI. Кроме того, этот параметр может состоять из идентификатора ресурса AVI в [**ловорд**](/previous-versions/windows/desktop/legacy/ms632659(v=vs.85)) и нуль в [**HIWORD**](/previous-versions/windows/desktop/legacy/ms632657(v=vs.85)). Чтобы создать это значение, используйте макрос [**макеинтресаурце**](/windows/desktop/api/winuser/nf-winuser-makeintresourcea) . Элемент управления загружает ресурс AVI из модуля, указанного в обработчике экземпляра, переданном в функцию [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) , в макрос [**анимации \_ CREATE**](/windows/desktop/api/Commctrl/nf-commctrl-animate_create) или в функцию создания диалогового окна, которая создала элемент управления. В [версии 4,71](common-control-versions.md) и более поздних ресурс загружается из модуля, указанного параметром *wParam*. Ресурс AVI должен иметь тип AVI. Если этот параметр имеет **значение NULL**, система ЗАкрывает AVI-файл, который ранее был открыт для указанного элемента управления анимации, если таковой имеется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ненулевое значение в случае успеха или ноль в противном случае.

## <a name="remarks"></a>Remarks

Файл AVI или ресурс, указанный параметром *лпсзнаме* , не должен содержать звук.

Мы рекомендуем использовать версию этого сообщения в Юникоде, ACM \_ опенв.

Вы можете открывать только оповещения AVI. ACM \_ Open и [**анимировать \_ Open**](/windows/desktop/api/Commctrl/nf-commctrl-animate_open) Fail, если свойство *lParam* задает ролик AVI, содержащий звук.

Для закрытия файла AVI или ресурса AVI, который был ранее открыт для указанного элемента управления анимацией, можно использовать параметр [**анимировать \_ Close**](/windows/desktop/api/Commctrl/nf-commctrl-animate_close) .
