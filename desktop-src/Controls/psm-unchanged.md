# <a name="psm_unchanged-message"></a>ПСМ \_ НЕизмененное сообщение

Информирует страницу свойств о том, что данные на странице возвращены к ранее сохраненному состоянию. Это сообщение можно отправить явно или с помощью [**\_ неизмененного**](/windows/desktop/api/Prsht/nf-prsht-propsheet_unchanged) макроса пропшит.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Обработчик для страницы, которая возвращена в ранее сохраненное состояние.

</dd> <dt>

*lParam* 
</dt> <dd>

Должен равняться нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Комментарии

Страница свойств отключает кнопку **Применить** , если на странице свойств не зарегистрированы изменения других страниц.

> [!Note]  
> Это сообщение не поддерживается при использовании стиля мастера Aero ([**командном PSH \_ аеровизард**](/windows/desktop/api/Prsht/ns-prsht-propsheetheadera_v2)).
