# <a name="em_getlangoptions-message"></a>\_Сообщение ЖЕТЛАНГОПТИОНС EM

Возвращает параметры расширенного элемента управления редактированием для редактора методов ввода (IME) и поддержки азиатских языков.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает параметры IME и восточно-азиатского языка, которые могут принимать следующие значения: ноль или более.



| Код возврата                                                                                                     | Описание                                                                                                                                                                                                                                                                                                                                                                                                                   |
|-----------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>**Автофильтр интеллектуального фильтра сообщений \_**</dt> </dl>                    | Если этот флаг установлен, элемент управления автоматически изменяет шрифты при явном изменении пользователем на другую раскладку клавиатуры. Рекомендуется отключить **\_ Автошрифт интеллектуального фильтра** для универсальных шрифтов Юникода. Этот параметр включен по умолчанию (1).<br/>                                                                                                                                                                     |
| <dl> <dt>**интеллектуальный фильтр сообщений \_ аутофонтсизеаджуст**</dt> </dl>          | Если этот флаг установлен, элемент управления масштабирует размеры шрифта, привязанные к шрифтам, от размера точки вставки в соответствии со скриптом. Например, восточноазиатские шрифты немного больше, чем западных. Этот параметр включен по умолчанию (1).<br/>                                                                                                                                                                                              |
| <dl> <dt>**автоклавиатура интеллектуального фильтра сообщений \_**</dt> </dl>                | Если этот флаг установлен, элемент управления автоматически изменяет раскладку клавиатуры, когда пользователь явно изменяет другой шрифт, или когда пользователь явно изменяет точку вставки на новое место в тексте. Будет автоматически включен для элементов управления для двунаправленного письма. Для всех остальных элементов управления он по умолчанию отключен. По умолчанию этот параметр отключен (0).<br/>                                 |
| <dl> <dt>**интеллектуальный фильтр сообщений \_ дисаблеаутобидиаутокэйбоард**</dt> </dl> | **Windows 8**. Если этот флаг установлен, элемент управления использует логику, не зависящую от языка, для автоматического переключения клавиатуры. По умолчанию этот параметр отключен (0).<br/>                                                                                                                                                                                                                                                            |
| <dl> <dt>**интеллектуальный фильтр сообщений \_ дуалфонт**</dt> </dl>                    | Если этот флаг установлен, элемент управления использует режим с двумя шрифтами. Используется для поддержки азиатских языков. Элемент управления использует английский шрифт для текста ASCII и азиатский шрифт для азиатского текста. Этот параметр включен по умолчанию (1).<br/>                                                                                                                                                                                                   |
| <dl> <dt>**интеллектуальный фильтр сообщений \_ имеалвайссенднотифи**</dt> </dl>         | Этот флаг управляет тем, как элемент управления Rich Edit уведомляет клиента во время составления IME: <br/> 0: нет уведомлений о [коротком \_ изменении](en-change.md) или [en \_ селчанже](en-selchange.md) в течение неопределенного состояния. Отправить уведомление, когда поступает последняя строка. Это значение по умолчанию.<br/> 1: отправка событий " [en \_ Change](en-change.md) " и " [en \_ селчанже](en-selchange.md) " в неопределенном состоянии.<br/> |
| <dl> <dt>**интеллектуальный фильтр сообщений \_ имеканцелкомплете**</dt> </dl>           | Этот флаг определяет, как элемент управления использует строку композиции IME, если пользователь отменяет его. Если этот флаг установлен, элемент управления отменяет строку композиции. Если этот флаг не установлен, элемент управления использует строку композиции в качестве строки результата. По умолчанию этот параметр отключен (0).<br/>                                                                                                              |
| <dl> <dt>**интеллектуальный фильтр сообщений \_ ноимплиЦитланг**</dt> </dl>              | **Windows 8**. Если этот флаг установлен, отключите клавиатурный ввод клавиатуры с помощью языка клавиатуры и убедитесь, что идсс языка, отличного от восточноазиатских, совместимы с символом поработать. По умолчанию этот параметр отключен (0). <br/>                                                                                                                                                                             |
| <dl> <dt>**интеллектуальный фильтр сообщений \_ нокбдлидфиксуп**</dt> </dl>               | **Windows 8**. Если этот флаг установлен, то элемент управления "форматированный" не отключает язык клавиатуры с отметкой на пустом элементе управления. По умолчанию этот параметр отключен (0).<br/>                                                                                                                                                                                                                                                       |
| <dl> <dt>**Проверка \_ орфографии интеллектуального фильтра**</dt> </dl>               | **Windows 8**. Если этот флаг установлен, элемент управления Rich Edit включает проверку орфографии. По умолчанию этот параметр отключен (0). <br/>                                                                                                                                                                                                                                                                                      |
| <dl> <dt>**интеллектуальный фильтр сообщений \_ ткбаутокорректион**</dt> </dl>           | **Windows 8**. Если этот флаг установлен, включите автозамену сенсорной клавиатуры. По умолчанию этот параметр отключен (0). <br/>                                                                                                                                                                                                                                                                                                  |
| <dl> <dt>**интеллектуальный фильтр сообщений \_ ткбпредиктион**</dt> </dl>               | **Windows 10**: игнорируется.<br/> **Windows 8**. Если этот флаг установлен, элемент управления "форматированный" позволяет выбрать сенсорную клавиатуру. По умолчанию этот параметр отключен (0). <br/>                                                                                                                                                                                                                                        |
| <dl> <dt>**интеллектуальный фильтр сообщений \_ уифонтс**</dt> </dl>                     | Использовать шрифты по умолчанию пользовательского интерфейса. По умолчанию этот параметр отключен (0).<br/>                                                                                                                                                                                                                                                                                                                                        |
