| <span id="TVS_CHECKBOXES"></span><span id="tvs_checkboxes"></span><dl> <dt>**\_флажки телевизоров**</dt> </dl>                | [Версия 4,70](common-control-versions.md). Включает флажки для элементов в элементе управления "представление в виде дерева". Флажок отображается только в том случае, если изображение связано с элементом. При задании этого стиля элемент управления эффективно использует [**дравфрамеконтрол**](/windows/desktop/api/winuser/nf-winuser-drawframecontrol) для создания и установки списка изображений состояния, содержащего два изображения. Изображение состояния 1 является снятым, а состояние image 2 — установленным флажком. Если задать для изображения состояния нулевое значение, флажок не будет полностью удален. Дополнительные сведения см. в разделе [Работа с индексами изображений состояний](tree-view-controls.md).<br/> [Версия 5,80](common-control-versions.md). Отображает флажок, даже если с элементом не связано ни одного изображения.<br/> После создания элемента управления "дерево-представление" с этим стилем стиль удалить нельзя. Вместо этого необходимо уничтожить элемент управления и создать новый на его месте. При уничтожении элемента управления представления в виде дерева не уничтожается список изображений состояния флажков. Его необходимо уничтожить явным образом. Получите маркер в списке изображений состояния, отправив элемент управления "дерево-представление" для сообщения [**TVM- \_ ImageList**](tvm-getimagelist.md) . Затем уничтожить список изображений с помощью [**\_ уничтожения ImageList**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_destroy).<br/> Если вы хотите использовать этот стиль, необходимо задать для ТЕЛЕВИЗОРов \_ стиль с [**SetWindowLong**](/windows/desktop/api/winuser/nf-winuser-setwindowlonga) после создания элемента управления TreeView и перед заполнением дерева. В противном случае флажки в зависимости от временных проблем могут оказаться неустановленными.<br/> |
