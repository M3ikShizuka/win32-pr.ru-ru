# <a name="mcn_select-notification-code"></a>МКН \_ Выбор кода уведомления

Посылается элементом управления "месячный календарь", когда пользователь делает явный выбор даты в элементе управления "месячный календарь". Этот код уведомления отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
MCN_SELECT

    lpNMSelChange = (LPNMSELCHANGE) lParam;
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмселчанже**](/windows/win32/api/commctrl/ns-commctrl-nmselchange) , содержащую сведения о выбранном в данный момент диапазоне дат.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Remarks

Код **МКН \_ SELECT** уведомления аналогичен [**МКН \_ селчанже**](mcn-selchange.md), но **МКН \_ SELECT** отправляется только в ответ на явно выбранные даты пользователя. **МКН \_ СЕЛЧАНЖЕ** применяется к любому изменению выбора.
