# <a name="em_seteditstyle-message"></a>\_Сообщение СЕТЕДИТСТИЛЕ EM

Задает текущие флаги изменения стиля для элемента управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Задает один или несколько флагов редактирования стиля. Список возможных значений см. в разделе [**EM \_ жетедитстиле**](em-geteditstyle.md).

</dd> <dt>

*lParam* 
</dt> <dd>

Маска, состоящая из одного или нескольких значений *wParam* . Будут установлены или сняты только значения, указанные в этой маске. Это позволяет устанавливать или снимать один флаг без считывания текущих состояний флагов.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение является состоянием флагов стиля правки после того, как элемент управления Rich Edit попытается применить изменения стиля редактирования. Флаги редактирования стиля представляют собой набор флагов, указывающих текущий стиль редактирования.
