# <a name="tvm_getvisiblecount-message"></a>\_Сообщение TVM жетвисиблекаунт

Получает количество элементов, которые могут быть полностью видны в клиентском окне элемента управления "дерево-представление". Это сообщение можно отправить явно или с помощью макроса [**\_ жетвисиблекаунт TreeView**](/windows/desktop/api/Commctrl/nf-commctrl-treeview_getvisiblecount) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает число элементов, которые могут быть полностью видны в клиентском окне элемента управления "дерево-представление".

## <a name="remarks"></a>Remarks

Количество элементов, которые могут быть полностью видимыми, может быть больше числа элементов в элементе управления. Элемент управления вычисляет это значение, разделив высоту клиентского окна по высоте элемента.

Обратите внимание, что возвращаемое значение — это количество элементов, которые могут быть *полностью* видимы. Если вы видите все 20 элементов и часть одного элемента, возвращаемое значение равно 20.
