# <a name="dtm_setmccolor-message"></a>\_Сообщение DTM сетмкколор

Задает цвет для заданной части месячного календаря в элементе управления "Выбор даты и времени" (DTP). Это сообщение можно отправить явным образом или использовать макрос [**\_ сетмонскалколор DateTime**](/windows/desktop/api/Commctrl/nf-commctrl-datetime_setmonthcalcolor) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Значение типа **int** , указывающее, какой цвет календаря месяца нужно задать. Значение может быть одним из следующих.



| Значение                                                                                                                                                                     | Значение                                                                                                                                                                                            |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="MCSC_BACKGROUND"></span><span id="mcsc_background"></span><dl> <dt>**\_фон MCSC**</dt> </dl>       | Задает цвет фона, отображаемый в интервале между месяцами.<br/>                                                                                                                                      |
| <span id="MCSC_MONTHBK"></span><span id="mcsc_monthbk"></span><dl> <dt>**MCSC \_ монсбк**</dt> </dl>                | Задайте цвет фона, отображаемый в пределах месяца.<br/>                                                                                                                                    |
| <span id="MCSC_TEXT"></span><span id="mcsc_text"></span><dl> <dt>**MCSC \_ текст**</dt> </dl>                         | Задайте цвет, используемый для показа текста в течение месяца.<br/>                                                                                                                                      |
| <span id="MCSC_TITLEBK"></span><span id="mcsc_titlebk"></span><dl> <dt>**MCSC \_ титлебк**</dt> </dl>                | Задайте цвет фона, отображаемый в заголовке календаря.<br/>                                                                                                                             |
| <span id="MCSC_TITLETEXT"></span><span id="mcsc_titletext"></span><dl> <dt>**MCSC \_ титлетекст**</dt> </dl>          | Задайте цвет, используемый для показа текста в заголовке календаря.<br/>                                                                                                                         |
| <span id="MCSC_TRAILINGTEXT"></span><span id="mcsc_trailingtext"></span><dl> <dt>**MCSC \_ траилингтекст**</dt> </dl> | Задайте цвет, используемый для вывода текста заголовка и дня конца. Дни заголовков и конечных дней — это дни предыдущего и следующего месяцев, которые отображаются в календаре текущего месяца.<br/> |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Значение **COLORREF** , представляющее цвет, который будет задан для указанной области в месячном календаре.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **COLORREF** , представляющее предыдущий цвет для указанной части элемента управления "календарь месяца" в случае успеха. В противном случае сообщение возвращает значение-1.

## <a name="remarks"></a>Комментарии

Если стили оформления включены, это сообщение не действует, кроме случаев, когда *wParam* имеет MCSC \_ Background.
