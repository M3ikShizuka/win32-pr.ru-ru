# <a name="unicode-support-for-common-controls"></a>Поддержка Юникода для стандартных элементов управления

В этом разделе описывается поддержка Юникода для общих управляющих уведомлений.


для версий 5,80 и более поздних comctl32.dll уведомления общих элементов управления поддерживают формат ANSI и юникод в системах Windows 95 или более поздней версии. Система определяет, какой формат следует использовать, отправляя окно сообщение [**\_ нотифиформат WM**](wm-notifyformat.md) . Чтобы указать формат, вернитесь в \_ ANSI для уведомлений ANSI или NFR \_ Unicode для уведомлений в Юникоде. Если это сообщение не обрабатывается, система вызывает [**исвиндовуникоде**](/windows/desktop/api/winuser/nf-winuser-iswindowunicode) для определения формата. поскольку Windows 95 и Windows 98 всегда возвращают **значение FALSE** для вызова этой функции, они по умолчанию используют уведомления ANSI.
