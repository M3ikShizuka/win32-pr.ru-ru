# <a name="window-classes-commctrlh"></a>Классы окон (Коммктрл. h)

В этом разделе перечислены имена классов окон, предоставляемые общей библиотекой элементов управления.



| Константа                                                                                                                                                                     | Описание                                                                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="ANIMATE_CLASS"></span><span id="animate_class"></span><dl> <dt>**\_класс анимации**</dt> </dl>                | Создает элементы управления анимации. Эти элементы управления автоматически отображают видеоклип с чередованием видео (AVI).<br/>                                                                                                                                                  |
| <span id="DATETIMEPICK_CLASS"></span><span id="datetimepick_class"></span><dl> <dt>**\_класс датетимепикк**</dt> </dl> | Создает элементы управления средства выбора даты и времени. Эти элементы управления предоставляют простой интерфейс для обмена данными о дате и времени с пользователем. <br/>                                                                                                                |
| <span id="HOTKEY_CLASS"></span><span id="hotkey_class"></span><dl> <dt>**класс горячих клавиш \_**</dt> </dl>                   | Создает элементы управления горячими ключами. Эти элементы управления облегчают пользователю определение горячих клавиш. <br/>                                                                                                                                                             |
| <span id="LINK_CLASS"></span><span id="link_class"></span><dl> <dt>**\_класс ссылки**</dt> </dl>                         | Создает элементы управления Syslink. Эти элементы управления используются со ссылками. <br/>                                                                                                                                                                                      |
| <span id="MONTHCAL_CLASS"></span><span id="monthcal_class"></span><dl> <dt>**\_класс монскал**</dt> </dl>             | Создает элементы управления "месячный календарь". Эти элементы управления предоставляют пользователю простой способ выбора даты из привычного интерфейса. <br/>                                                                                                                        |
| <span id="NATIVEFNTCTL_CLASS"></span><span id="nativefntctl_class"></span><dl> <dt>**\_класс нативефнтктл**</dt> </dl> | Создает собственные элементы управления "Шрифт". Эти элементы управления используются со встроенными шрифтами. <br/>                                                                                                                                                                           |
| <span id="PROGRESS_CLASS"></span><span id="progress_class"></span><dl> <dt>**\_класс Progress**</dt> </dl>             | Создает индикаторы выполнения. Эти элементы управления указывают ход выполнения длительной операции.<br/>                                                                                                                                                                 |
| <span id="REBARCLASSNAME"></span><span id="rebarclassname"></span><dl> <dt>**ребаркласснаме**</dt> </dl>              | Создает элементы управления главной панели. Эти элементы управления действуют как контейнер для дочерних окон. <br/>                                                                                                                                                                       |
| <span id="STANDARD_CLASSES"></span><span id="standard_classes"></span><dl> <dt>**СТАНДАРТНЫЕ \_ классы**</dt> </dl>       | Создает внутренние элементы управления User32, такие как кнопка, Правка и полоса прокрутки.<br/>                                                                                                                                                                            |
| <span id="STATUSCLASSNAME"></span><span id="statusclassname"></span><dl> <dt>**статускласснаме**</dt> </dl>           | Создает окна состояния. Эти элементы управления отображают сведения о состоянии в горизонтальном окне.<br/>                                                                                                                                                           |
| <span id="TOOLBARCLASSNAME"></span><span id="toolbarclassname"></span><dl> <dt>**тулбаркласснаме**</dt> </dl>        | Создает панели инструментов. Эти элементы управления содержат кнопки, выполняющие команды меню. <br/>                                                                                                                                                                     |
| <span id="TOOLTIPS_CLASS"></span><span id="tooltips_class"></span><dl> <dt>**класс всплывающих подсказок \_**</dt> </dl>             | Создает элементы управления ToolTip. Эти элементы управления отображают небольшое всплывающее окно, содержащее строку текста, описывающую Назначение инструмента в приложении.<br/>                                                                                            |
| <span id="TRACKBAR_CLASS"></span><span id="trackbar_class"></span><dl> <dt>**\_класс TrackBar**</dt> </dl>             | Создает ползунки. Эти элементы управления позволяют пользователю выбирать из диапазона значений, перемещая ползунок. <br/>                                                                                                                                                   |
| <span id="UPDOWN_CLASS"></span><span id="updown_class"></span><dl> <dt>**класс "исключающий" \_**</dt> </dl>                   | Создает элементы управления "вверх-вниз". Эти элементы управления объединяют пару стрелок с элементом управления "Кнопка ввода". Щелкая стрелки, вы увеличиваете или уменьшаете значение в элементе управления "поле ввода". <br/>                                                                                |
| <span id="WC_BUTTON"></span><span id="wc_button"></span><dl> <dt>**\_Кнопка WC**</dt> </dl>                            | Создает элементы управления "Кнопка". Эти элементы управления позволяют пользователю инициировать действие.<br/>                                                                                                                                                                        |
| <span id="WC_COMBOBOX"></span><span id="wc_combobox"></span><dl> <dt>**\_поле со СПИСКОМ WC**</dt> </dl>                      | Создает элементы управления ComboBox. Эти элементы управления содержат раскрывающийся список, в котором можно выбрать один элемент. <br/>                                                                                                                                       |
| <span id="WC_COMBOBOXEX"></span><span id="wc_comboboxex"></span><dl> <dt>**WC \_ COMBOBOXEX**</dt> </dl>                | Создает элементы управления ComboBoxEx. Эти элементы управления предоставляют расширение элемента управления "поле со списком", которое обеспечивает встроенную поддержку изображений элементов. <br/>                                                                                                            |
| <span id="WC_EDIT"></span><span id="wc_edit"></span><dl> <dt>**WC \_ изменить**</dt> </dl>                                  | Создает элементы управления "Правка". Эти элементы управления содержат редактируемый текст. <br/>                                                                                                                                                                                       |
| <span id="WC_HEADER"></span><span id="wc_header"></span><dl> <dt>**\_заголовок WC**</dt> </dl>                            | Создает элементы управления "заголовок". Эти элементы управления отображают заголовки в верхней части столбцов данных и позволяют пользователю сортировать информацию, щелкая заголовки. <br/>                                                                                      |
| <span id="WC_LISTBOX"></span><span id="wc_listbox"></span><dl> <dt>**\_список WC**</dt> </dl>                         | Создает элементы управления ListBox. Эти элементы управления содержат списки элементов. <br/>                                                                                                                                                                                   |
| <span id="WC_IPADDRESS"></span><span id="wc_ipaddress"></span><dl> <dt>**\_IP-адрес WC**</dt> </dl>                   | Создает элементы управления IP-адресами. Эти элементы управления похожи на элемент управления Edit, но позволяют вводить числовой адрес в формате протокола Интернета (IP). <br/>                                                                                       |
| <span id="WC_LINK"></span><span id="wc_link"></span><dl> <dt>**Ссылка на WC \_**</dt> </dl>                                  | Создает элементы управления SysLink. Эти элементы управления содержат ссылки гипертекста.<br/>                                                                                                                                                                                   |
| <span id="WC_LISTVIEW"></span><span id="wc_listview"></span><dl> <dt>**\_LISTVIEW WC**</dt> </dl>                      | Создает элементы управления "представление списка". Эти элементы управления отображают коллекцию элементов, каждая из которых состоит из значка и метки, и предоставляет несколько способов упорядочения элементов.<br/>                                                                                    |
| <span id="WC_NATIVEFONTCTL"></span><span id="wc_nativefontctl"></span><dl> <dt>**WC \_ нативефонтктл**</dt> </dl>       | Создает собственные элементы управления "Шрифт". Это невидимые элементы управления, работающие в фоновом режиме для включения предопределенных элементов управления диалогового окна для вывода текущего языка системы.<br/>                                                                         |
| <span id="WC_PAGESCROLLER"></span><span id="wc_pagescroller"></span><dl> <dt>**WC \_ пажескроллер**</dt> </dl>          | Создает элементы управления страничного навигатора. Эти элементы управления используются для хранения и прокрутки другого окна. <br/>                                                                                                                                                              |
| <span id="WC_SCROLLBAR"></span><span id="wc_scrollbar"></span><dl> <dt>**WC \_ полоса прокрутки**</dt> </dl>                   | Создает элементы управления "полоса прокрутки". Эти элементы управления позволяют пользователю прокручивать содержимое окна.<br/>                                                                                                                                                     |
| <span id="WC_STATIC"></span><span id="wc_static"></span><dl> <dt>**WC \_ статический**</dt> </dl>                            | Создает статические элементы управления. Эти элементы управления содержат нередактируемый текст. <br/>                                                                                                                                                                                  |
| <span id="WC_TABCONTROL"></span><span id="wc_tabcontrol"></span><dl> <dt>**WC \_ TABCONTROL**</dt> </dl>                | Создает элементы управления "Вкладка". Эти элементы управления определяют несколько страниц для одной области окна или диалогового окна. Каждая страница состоит из набора сведений или группы элементов управления, которые отображаются приложением, когда пользователь выбирает соответствующую вкладку.<br/> |
| <span id="WC_TREEVIEW"></span><span id="wc_treeview"></span><dl> <dt>**WC \_ TREEVIEW**</dt> </dl>                      | Создает элементы управления "дерево-представление". Эти элементы управления отображают иерархический список элементов. Каждый элемент состоит из метки и необязательного точечного рисунка.<br/>                                                                                                              |
