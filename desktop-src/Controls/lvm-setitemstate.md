# <a name="lvm_setitemstate-message"></a>\_Сообщение LVM сетитемстате

Изменяет состояние элемента в элементе управления "представление списка". Это сообщение можно отправить явно или с помощью макроса [**\_ сетитемстате ListView**](/windows/desktop/api/Commctrl/nf-commctrl-listview_setitemstate) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Индекс элемента представления списка. Если этот параметр имеет значение-1, то изменение состояния применяется ко всем элементам.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**лвитем**](/windows/win32/api/commctrl/ns-commctrl-lvitema) . Элемент **статемаск** указывает, какие биты состояния следует изменить, а элемент **State** содержит новые значения для этих битов. Остальные элементы игнорируются.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если сообщение отправляется явным образом, оно возвращает **true** в случае успеха или **false** в противном случае.

## <a name="remarks"></a>Комментарии

Значение состояния элемента включает набор битовых флагов, указывающих на состояние элемента. Значение состояния может также включать индексы списка изображений, указывающие изображение состояния элемента и изображение оверлея.
