# <a name="tb_changebitmap-message"></a>\_Сообщение ЧАНЖЕБИТМАП ТБ

Изменяет точечный рисунок для кнопки на панели инструментов.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Идентификатор команды для получения нового точечного рисунка.

</dd> <dt>

*lParam* 
</dt> <dd>

Отсчитываемый от нуля индекс изображения в списке изображений на панели инструментов. Система отображает указанное изображение в кнопке. Задайте для этого параметра значение I \_ имажекаллбакк, после чего панель инструментов отправит уведомление [**\_ жетдиспинфо ТБН**](tbn-getdispinfo.md) , чтобы получить индекс изображения, когда он понадобится.

[Версия 5,81](common-control-versions.md). Установите этот параметр в значение I \_ имаженоне, чтобы указать, что кнопка не имеет изображения. Макет кнопки не будет содержать пробелы для точечного рисунка, а только для текста.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если успешно, или **false** в противном случае.
