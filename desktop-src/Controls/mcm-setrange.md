# <a name="mcm_setrange-message"></a>\_Сообщение SETRANGE MCM

Задает минимальную и максимальную допустимые даты для элемента управления "месячный календарь". Это сообщение можно отправить явно или с помощью макроса [**монскал \_ SetRange**](/windows/desktop/api/Commctrl/nf-commctrl-monthcal_setrange) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Значения флагов, указывающие, какие ограничения дат задаются. Это значение должно быть одним из следующих:



| Значение                                                                                                                                          | Значение                                                                                                                                                          |
|------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="GDTR_MAX"></span><span id="gdtr_max"></span><dl> <dt>**ГДТР \_ Max**</dt> </dl> | Устанавливается максимально допустимая дата. Структура [**SYSTEMTIME**](/windows/desktop/api/minwinbase/ns-minwinbase-systemtime) в *лпсистимеаррай* \[ 1 \] должна содержать сведения о датах. <br/> |
| <span id="GDTR_MIN"></span><span id="gdtr_min"></span><dl> <dt>**ГДТР \_ мин**</dt> </dl> | Устанавливается минимальная допустимая дата. Структура [**SYSTEMTIME**](/windows/desktop/api/minwinbase/ns-minwinbase-systemtime) в *лпсистимеаррай* \[ 0 \] должна содержать сведения о датах. <br/> |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на массив, сокоторый из двух элементов, содержит структуры [**SYSTEMTIME**](/windows/desktop/api/minwinbase/ns-minwinbase-systemtime) , которые содержат ограничения по датам. Максимальный предел должен быть в *лпсистимеаррай* \[ 1 \] , если указан параметр Гдтр \_ Max, а *лпсистимеаррай* \[ 0 \] должен содержать минимальное ограничение, если \_ задано значение гдтр min.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ненулевое значение в случае успеха или ноль в противном случае.
