# <a name="lvm_setview-message"></a>\_Сообщение LVM сетвиев

Задает представление элемента управления "представление списка".

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Значение **типа DWORD** , указывающее представление.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает 1, если успешно, или-1 в противном случае. Например, если представление недопустимо, возвращается значение-1.

## <a name="remarks"></a>Remarks

Ниже приведены значения для представлений.

-   \_ \_ сведения о представлении LV
-   \_значок представления "Латвия" \_
-   \_список ПРЕДСТАВЛЕНИЙ \_ LV
-   \_представление LV \_ маленькие значки
-   \_плитка представления "Латвия" \_

> [!Note]  
> Чтобы использовать это сообщение, необходимо указать манифест, указывающий Comctl32.dll версии 6,0. Дополнительные сведения о манифестах см. в разделе [Включение визуальных стилей](cookbook-overview.md).
