# <a name="date-and-time-picker-control-styles"></a>Стили элемента управления выбора даты и времени

Перечисленные здесь стили окон относятся к элементам управления выбора даты и времени.



| Константа                                                                                                                                                                                             | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="DTS_APPCANPARSE"></span><span id="dts_appcanparse"></span><dl> <dt>**аппканпарсе служб DTS \_**</dt> </dl>                                  | Позволяет владельцу анализировать входные данные пользователя и предпринимать необходимые действия. Он позволяет пользователям редактировать в клиентской области элемента управления при нажатии клавиши F2. Элемент управления отправляет коды уведомлений [ДТН \_ усерстринг](dtn-userstring.md) при завершении работы пользователей.<br/>                                                                                                                                                                                                                                                                    |
| <span id="DTS_LONGDATEFORMAT"></span><span id="dts_longdateformat"></span><dl> <dt>**лонгдатеформат служб DTS \_**</dt> </dl>                         | Отображает дату в длинном формате. Строка форматирования по умолчанию для этого стиля определяется ЛОКАЛЬным \_ слонгдатеформат, которая выдает выходные данные, например "Пятница, 19 апреля, 1996". При использовании этого стиля кнопка раскрывающегося списка не отображает значок.<br/>                                                                                                                                                                                                                                                                                     |
| <span id="DTS_RIGHTALIGN"></span><span id="dts_rightalign"></span><dl> <dt>**RIGHTALIGN служб DTS \_**</dt> </dl>                                     | Раскрывающийся календарь в виде месяца будет выставляться по правому краю элемента управления, а не по левому краю, который является значением по умолчанию. <br/>                                                                                                                                                                                                                                                                                                                                                                                                               |
| <span id="DTS_SHOWNONE"></span><span id="dts_shownone"></span><dl> <dt>**шовноне служб DTS \_**</dt> </dl>                                           | В элементе управления невозможно выбрать дату, которая сейчас выбрана. В этом стиле элемент управления отображает флажок, который автоматически выбирается при каждом выборе или вводе даты. Если флажок не выбран, приложение не сможет извлечь дату из элемента управления, поскольку элемент управления не имеет даты. Состояние флажка можно задать с помощью сообщения [**DTM \_ сетсистемтиме**](dtm-setsystemtime.md) или запроса с сообщением [**DTM \_ GETSYSTEMTIME**](dtm-getsystemtime.md) .<br/> |
| <span id="DTS_SHORTDATEFORMAT"></span><span id="dts_shortdateformat"></span><dl> <dt>**шортдатеформат служб DTS \_**</dt> </dl>                      | Отображает дату в кратком формате. Строка форматирования по умолчанию для этого стиля определяется с помощью языкового стандарта \_ сшортдате, который создает выходные данные, например "4/19/96".<br/>                                                                                                                                                                                                                                                                                                                                                                               |
| <span id="DTS_SHORTDATECENTURYFORMAT"></span><span id="dts_shortdatecenturyformat"></span><dl> <dt>**шортдатецентуриформат служб DTS \_**</dt> </dl> | **Версия 5,80.** Аналогично стилю **\_ Шортдатеформат служб DTS** , за исключением года, является поле из четырех цифр. Строка форматирования по умолчанию для этого стиля основана на языке \_ сшортдате. Выходные данные выглядят следующим образом: "4/19/1996".<br/>                                                                                                                                                                                                                                                                                                              |
| <span id="DTS_TIMEFORMAT"></span><span id="dts_timeformat"></span><dl> <dt>**TIMEFORMAT служб DTS \_**</dt> </dl>                                     | Отображает время. Строка форматирования по умолчанию для этого стиля определяется стандартом LOCALE \_ стимеформат, который создает выходные данные, например "5:31:42 PM".<br/>                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DTS_UPDOWN"></span><span id="dts_updown"></span><dl> <dt>**истечение работы служб DTS \_**</dt> </dl>                                                 | Размещает элемент управления "вверх-вниз" справа от элемента управления DTP, чтобы изменить значения даты и времени. Этот стиль можно использовать вместо календаря с раскрывающимся списком, который является стилем по умолчанию.<br/>                                                                                                                                                                                                                                                                                                                                                |



## <a name="remarks"></a>Remarks

\_Стили КСКСКСФОРМАТ DTS, определяющие формат вывода, не могут быть объединены. Если ни один из стилей форматирования не подходит, используйте сообщение [**DTM \_ сетформат**](dtm-setformat.md) для определения пользовательского формата.
