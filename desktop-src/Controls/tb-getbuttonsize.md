# <a name="tb_getbuttonsize-message"></a>\_Сообщение ЖЕТБУТТОНСИЗЕ ТБ

Извлекает текущую ширину и высоту кнопок панели инструментов в пикселях.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **типа DWORD** , содержащее значения ширины и высоты в нижнем слове и верхнем слове соответственно.
