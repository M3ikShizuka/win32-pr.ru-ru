# <a name="sb_settiptext-message"></a>\_Сообщение SB сеттиптекст

Задает текст подсказки для части в строке состояния. Чтобы включить подсказки, строка состояния должна быть создана с помощью стиля [**\_ подсказок SBT**](status-bar-styles.md) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Отсчитываемый от нуля индекс части, которая будет принимать текст подсказки.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на символьный буфер, содержащий новый текст подсказки.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение не используется.

## <a name="remarks"></a>Remarks

Этот текст подсказки отображается в двух ситуациях:

-   Если соответствующая панель в строке состояния содержит только значок.
-   Когда соответствующая панель в строке состояния содержит текст, усеченный из-за размера панели.
