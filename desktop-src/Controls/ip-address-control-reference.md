# <a name="ip-address-control"></a>Управление IP-адресами

В этом разделе содержатся сведения об элементах программирования, используемых с элементами управления IP-адресами.

### <a name="overviews"></a>Разделы общих сведений



| Раздел                                          | Содержимое                                                                                                                    |
|------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [Элементы управления IP-адресами](ip-address-controls.md) | Элемент управления IP-адреса позволяет пользователю ввести IP-адрес в удобном для понимания формате.<br/> |



 

### <a name="macros"></a>Макросы



| Раздел                                         | Содержимое                                                                                                                              |
|-----------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [**ПЕРВЫЙ \_ IP-адрес**](/windows/desktop/api/Commctrl/nf-commctrl-first_ipaddress)   | Извлекает значение поля 0 из упакованного IP-адреса, полученного с [**\_ сообщением IPM**](ipm-getaddress.md) . <br/> |
| [**Четвертый \_ IPAddress**](/windows/desktop/api/Commctrl/nf-commctrl-fourth_ipaddress) | Извлекает значение поля 3 из упакованного IP-адреса, полученного с [**\_ сообщением IPM**](ipm-getaddress.md) . <br/> |
| [**макеипаддресс**](/windows/desktop/api/Commctrl/nf-commctrl-makeipaddress)        | Упаковывает четыре значения типа Byte в одно значение LPARAM, которое подходит для использования с сообщением [**IPM \_ сетаддресс**](ipm-setaddress.md) . <br/>  |
| [**макеипранже**](/windows/desktop/api/Commctrl/nf-commctrl-makeiprange)            | Упаковывает два байтовых значения в одно значение LPARAM, которое подходит для использования с сообщением [**IPM \_ SETRANGE**](ipm-setrange.md) . <br/>       |
| [**ВТОРОЙ \_ IP-адрес**](/windows/desktop/api/Commctrl/nf-commctrl-second_ipaddress) | Извлекает значение поля 1 из упакованного IP-адреса, полученного с [**\_ сообщением IPM**](ipm-getaddress.md) . <br/> |
| [**Третий \_ IPAddress**](/windows/desktop/api/Commctrl/nf-commctrl-third_ipaddress)   | Извлекает значение поля 2 из упакованного IP-адреса, полученного с [**\_ сообщением IPM**](ipm-getaddress.md) . <br/> |



 

### <a name="messages"></a>Сообщения



| Раздел                                         | Содержимое                                                                                                                              |
|-----------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [**\_КЛЕАРАДДРЕСС IPM**](ipm-clearaddress.md) | Очищает содержимое элемента управления "IP-адрес". <br/>                                                                            |
| [**\_адрес IPM**](ipm-getaddress.md)     | Возвращает значения адреса для всех четырех полей в элементе управления IP-адреса. <br/>                                                    |
| [**IPM \_**](ipm-isblank.md)           | Определяет, являются ли все поля в элементе управления "IP-адрес" пустыми. <br/>                                                             |
| [**\_СЕТАДДРЕСС IPM**](ipm-setaddress.md)     | Задает значения адреса для всех четырех полей в элементе управления "IP-адрес". <br/>                                                    |
| [**IPM \_ SETFOCUS**](ipm-setfocus.md)         | Устанавливает фокус клавиатуры на указанное поле в элементе управления IP-адреса. Будет выбран весь текст в этом поле. <br/> |
| [**IPM \_ SETRANGE**](ipm-setrange.md)         | Задает допустимый диапазон для указанного поля в элементе управления IP-адреса. <br/>                                                   |
