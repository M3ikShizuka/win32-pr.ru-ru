# <a name="udn_deltapos-notification-code"></a>\_Код уведомления УДН делтапос

Отправляется операционной системой в родительское окно элемента управления "вверх/вниз", когда изменяется расположение элемента управления. Это происходит, когда пользователь запрашивает изменение значения, нажимая клавишу со стрелкой вверх или вниз элемента управления. Этот код уведомления отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
UDN_DELTAPOS 

    lpnmud = (LPNMUPDOWN) lParam;
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмупдовн**](/windows/win32/api/commctrl/ns-commctrl-nmupdown) , содержащую сведения об изменении позиции. Элемент **ИПОС** этой структуры содержит текущую точку элемента управления. Элемент **класса idelta** структуры представляет собой целое число со знаком, содержащее предлагаемое изменение в положении. Если пользователь щелкнул кнопку Up (вверх), это положительное значение. Если пользователь щелкнул кнопку вниз, это отрицательное значение.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ненулевое значение, чтобы предотвратить изменение в положении элемента управления, или нуль, чтобы разрешить изменение.

## <a name="remarks"></a>Комментарии

\_Код уведомления УДН делтапос отправляется перед сообщением [**WM \_ VSCROLL**](wm-vscroll.md) или [**WM \_ HSCROLL**](wm-hscroll.md) , которое фактически изменяет расположение элемента управления. Это позволяет проверять, разрешать, изменять или запрещать изменение.
