# <a name="rb_setparent-message"></a>\_Сообщение СЕТПАРЕНТ RB

Задает родительское окно элемента управления главной панели.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Обрабатываемое родительское окно.

</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает маркер предыдущего родительского окна или **значение NULL** , если предыдущий родительский элемент отсутствует.

## <a name="remarks"></a>Комментарии

Элемент управления "Главная панель" отправляет сообщения уведомления в окно, указанное этим сообщением. Это сообщение фактически не изменяет родительский элемент управления главной панели.
