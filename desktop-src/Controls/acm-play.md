# <a name="acm_play-message"></a>ACM \_ воспроизведение сообщения

Воспроизводит ролик AVI в элементе управления Animation. Элемент управления воспроизводит клип в фоновом режиме, в то время как поток продолжит выполнение. Это сообщение можно отправить явно или с помощью макроса [**анимировать \_ Play**](/windows/desktop/api/Commctrl/nf-commctrl-animate_play) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Значение **uint** , указывающее количество повторов воспроизведения фрагмента AVI. Значение-1 означает воспроизведение клипа в течение неограниченного времени.

</dd> <dt>

*lParam* 
</dt> <dd>

[**Ловорд**](/previous-versions/windows/desktop/legacy/ms632659(v=vs.85)) — это Отсчитываемый от нуля индекс кадра, с которого начинается воспроизведение. Значение должно быть меньше 65 536. Нулевое значение означает, что начинается с первого кадра в ролике AVI.

[**HIWORD**](/previous-versions/windows/desktop/legacy/ms632657(v=vs.85)) — это Отсчитываемый от нуля индекс кадра, в котором завершается воспроизведение. Значение должно быть меньше 65 536. Значение-1 означает, что заканчивается последним кадром в ролике AVI.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ненулевое значение в случае успеха или ноль в противном случае.

## <a name="remarks"></a>Remarks

С помощью [**анимации можно \_**](/windows/desktop/api/Commctrl/nf-commctrl-animate_seek) направить элемент управления анимации для показа определенного кадра клипа AVI.
