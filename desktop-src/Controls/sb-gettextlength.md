# <a name="sb_gettextlength-message"></a>\_Сообщение SB жеттекстленгс

Извлекает длину текста в символах из указанной части окна состояния.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Отсчитываемый от нуля индекс части, из которой требуется получить текст.

</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает 32-разрядное значение, состоящее из 2 16-разрядных значений. Нижнее слово указывает длину текста в символах. В верхнем слове указывается тип операции, используемой для рисования текста. Тип может иметь одно из следующих значений:



| Код возврата                                                                                    | Описание                                                                                       |
|------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| <dl> <dt>**0**</dt> </dl>               | Текст отображается с границей, которая отображается ниже плоскости окна.<br/>          |
| <dl> <dt>**SBT \_ НЕграницы**</dt> </dl>  | Текст отображается без границ.<br/>                                                     |
| <dl> <dt>**SBT \_ овнердрав**</dt> </dl>  | Текст рисуется родительским окном.<br/>                                                |
| <dl> <dt>**SBT \_ POPOUT**</dt> </dl>     | Текст отображается с границей, которая отображается выше плоскости окна.<br/>         |
| <dl> <dt>**SBT \_ RTLREADING**</dt> </dl> | Текст будет отображаться в обратном направлении к тексту в родительском окне.<br/> |



 

## <a name="remarks"></a>Remarks

Нормальный отображаемый текст Windows слева направо (LTR). Windows могут быть *отражены* для отображения таких языков, как иврит или арабский, которые читаются справа налево. Если \_ задано значение SBT RTLREADING, то указанный текст окна состояния будет читаться в обратном направлении от текста в родительском окне.

Это сообщение возвращает максимальную длину строки в 65 535 символов. Если фактическая длина текстовой строки превышает это значение, сообщение [**SB \_ gettext**](sb-gettext.md) усекает его.
