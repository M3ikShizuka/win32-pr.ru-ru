# <a name="tvm_setborder-message"></a>\_Сообщение TVM сетбордер

**Предназначен для внутреннего использования; не рекомендуется для использования в приложениях.**

Задает размер границы для элементов в элементе управления "представление дерева". Сообщение можно отправить явно или с помощью макроса [**\_ сетбордер TreeView**](/windows/desktop/api/Commctrl/nf-commctrl-treeview_setborder) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Флаги действия. Этот параметр может принимать одно или несколько из следующих значений:



| Значение                                                                                                                                                         | Значение                                                                                               |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| <span id="TVSBF_XBORDER"></span><span id="tvsbf_xborder"></span><dl> <dt>**ТВСБФ \_ ксбордер**</dt> </dl> | Применяет указанный размер границы к левой части элементов в элементе управления "дерево-представление". <br/> |
| <span id="TVSBF_YBORDER"></span><span id="tvsbf_yborder"></span><dl> <dt>**ТВСБФ \_ ибордер**</dt> </dl> | Применяет указанный размер границы к верхней части элементов в элементе управления "дерево-представление".<br/>        |



 

</dd> <dt>

*lParam* 
</dt> <dd>

[**Ловорд**](/previous-versions/windows/desktop/legacy/ms632659(v=vs.85)) — это **короткое** значение, указывающее размер левой границы в пикселях. [**HIWORD**](/previous-versions/windows/desktop/legacy/ms632657(v=vs.85)) — это **короткое** значение, указывающее размер верхней границы (в пикселях).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение **типа Long** , содержащее предыдущий размер границы (в пикселях). [**Ловорд**](/previous-versions/windows/desktop/legacy/ms632659(v=vs.85)) содержит предыдущий размер горизонтальной границы, а [**HIWORD**](/previous-versions/windows/desktop/legacy/ms632657(v=vs.85)) содержит предыдущий размер вертикальной границы.

## <a name="remarks"></a>Remarks

**Предупреждение системы безопасности:** Использование этого сообщения может нарушить безопасность программы.

Граница элемента задается только для целей разделения. Успешный параметр активирует повторное вычисление полос прокрутки.

Это сообщение может не поддерживаться в будущих версиях Comctl32.dll. Кроме того, это сообщение не определено в коммктрл. h. Добавьте следующие определения в исходные файлы приложения для использования сообщения:

``` syntax
#define TVM_SETBORDER (TV_FIRST + 35)
#define TVSBF_XBORDER 0x00000001
#define TVSBF_YBORDER 0x00000002 
```
