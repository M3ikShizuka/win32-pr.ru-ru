# <a name="rb_hittest-message"></a>\_Сообщение HITTEST RB

Определяет, какая часть полосы элементов управления находится в заданной точке экрана, если в этой точке существует полоса главной панели.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**рбхиттестинфо**](/windows/win32/api/commctrl/ns-commctrl-rbhittestinfo) . Перед отправкой сообщения элемент **PT** этой структуры должен быть инициализирован в точке, которая будет проверена на попадание, в клиентских координатах.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает отсчитываемый от нуля индекс полосы в заданной точке или значение-1, если полоса элементов не находилась в точке.
