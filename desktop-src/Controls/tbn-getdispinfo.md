# <a name="tbn_getdispinfo-notification-code"></a>\_Код уведомления ТБН жетдиспинфо

Извлекает отображаемые сведения об элементе панели инструментов. Этот код уведомления отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
TBN_GETDISPINFO 

    lptbdi = (LPNMTBDISPINFO) lParam; 
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмтбдиспинфо**](/windows/desktop/api/Commctrl/ns-commctrl-nmtbdispinfoa) . Элемент **идкомманд** указывает идентификатор команды элемента, член **lParam** содержит определяемые приложением данные элемента, а член **двмаск** указывает, какие сведения запрашиваются.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение игнорируется элементом управления.
