# <a name="em_setwordwrapmode-message"></a>\_Сообщение СЕТВОРДВРАПМОДЕ EM

Задает параметры разбиения по словам и переноса слов для элемента управления расширенного редактирования.

> [!Note]  
> Это сообщение поддерживается только в версиях Microsoft Rich Edit 1,0, поддерживающих азиатские языки. Он не поддерживается в каких-либо более поздних версиях.

 

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Задает одно или несколько из следующих значений.



| Значение                                                                                                                                                         | Значение                                                                                                               |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| <span id="WBF_WORDWRAP"></span><span id="wbf_wordwrap"></span><dl> <dt>**ВБФ переносить по \_ словам**</dt> </dl>    | Включает операции переноса по словам для восточноазиатских языков, например кинсоку на японском языке. <br/>                                 |
| <span id="WBF_WORDBREAK"></span><span id="wbf_wordbreak"></span><dl> <dt>**ВБФ \_ специализированные**</dt> </dl> | Включает операции разбиения по словам на английском языке в японском и китайском языках. Включает Ханжеул операцию разбиения на слова.<br/> |
| <span id="WBF_OVERFLOW"></span><span id="wbf_overflow"></span><dl> <dt>**\_переполнение ВБФ**</dt> </dl>    | Распознает знаки пунктуации переполнения. (В настоящее время не поддерживается.)<br/>                                                |

## <a name="return-value"></a>Возвращаемое значение

Это сообщение возвращает текущие параметры переноса слов и разбиения на слова.

## <a name="remarks"></a>Remarks

Это сообщение не должно отправляться приложением, заданным процедурой разбиения по словам.
