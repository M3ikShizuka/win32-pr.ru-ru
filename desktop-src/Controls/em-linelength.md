# <a name="em_linelength-message"></a>\_Сообщение ЛИНЕЛЕНГС EM

Извлекает длину строки в символах в элементе управления "поле ввода". Это сообщение можно отправить либо в элемент управления "поле ввода", либо в элемент управления Rich Edit.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Индекс символа в строке, длина которого должна быть получена. Если этот параметр больше числа символов в элементе управления, возвращаемое значение равно нулю.

Этот параметр может быть равен-1. В этом случае сообщение возвращает число невыбранных символов в строках, содержащих выбранные символы. Например, если выделение, расширенное от четвертого символа одной строки до восьмого символа от конца следующей строки, возвращаемое значение будет равно 10 (три символа в первой строке и семь на следующем).

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Для элементов управления многострочного редактирования возвращаемое значение является длиной в **TCHAR** s строки, заданной параметром *wParam* . Для текста ANSI это число байтов; для текста в Юникоде это число символов. Он не включает символ возврата каретки в конце строки.

Для однострочных элементов управления редактирования возвращаемое значение является длиной в **TCHAR** s текста в элементе управления "поле ввода".

Если параметр *wParam* больше числа символов в элементе управления, возвращаемое значение равно нулю.

## <a name="remarks"></a>Комментарии

Используйте сообщение [**EM \_ линеиндекс**](em-lineindex.md) , чтобы получить индекс символа для заданного номера строки в многострочном элементе управления Edit.

**Расширенное редактирование:** Поддерживается в Microsoft Rich Edit 1,0 и более поздних версиях. Дополнительные сведения о совместимости расширенных версий редактирования с различными версиями системы см. в разделе [Общие сведения об элементах управления редактированием](about-rich-edit-controls.md).
