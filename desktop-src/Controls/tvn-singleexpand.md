# <a name="tvn_singleexpand-notification-code"></a>\_Код уведомления ТВН сингликспанд

Посылается элементом управления "дерево" с помощью стиля " [**телевизоры \_ сингликспанд**](tree-view-control-window-styles.md) ", когда пользователь открывает или закрывает элемент дерева одним щелчком мыши. Этот код уведомления отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
TVN_SINGLEEXPAND

    lpnmtv = (LPNMTREEVIEW)lParam;
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмтривиев**](/windows/win32/api/commctrl/ns-commctrl-nmtreeviewa) , содержащую сведения об этом коде уведомления.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ТВНРЕТ \_ по умолчанию, чтобы разрешить выполнение поведения по умолчанию. Чтобы изменить поведение по умолчанию, возвратите:



| Код возврата                                                                                    | Описание                                                      |
|------------------------------------------------------------------------------------------------|------------------------------------------------------------------|
| <dl> <dt>**ТВНРЕТ \_ скиполд**</dt> </dl> | Пропустить обработку по умолчанию для элемента, для которого выполняется отмена выбора.<br/> |
| <dl> <dt>**ТВНРЕТ \_ скипнев**</dt> </dl> | Пропустить обработку выбранного элемента по умолчанию.<br/>   |



 

## <a name="remarks"></a>Комментарии

Чтобы пропустить обработку выбранных и невыбранных элементов по умолчанию, возвратите ТВНРЕТ \_ скиполд и твнрет \_ скипнев, объединив их с логическим или.

Этот код уведомления отправляется только элементами управления "дерево", которые имеют [**стиль \_ Сингликспанд "телевизоры**](tree-view-control-window-styles.md) ".
