# <a name="mcn_getdaystate-notification-code"></a>\_Код уведомления МКН жетдайстате

Посылается элементом управления "месячный календарь" для запроса сведений о том, как должны отображаться отдельные дни. Этот код уведомления отправляется только элементам управления "месячный календарь", использующим стиль [**MCS \_ дайстате**](month-calendar-control-styles.md) , и отправляется в виде сообщения [**\_ уведомления WM**](wm-notify.md) .


```C++
MCN_GETDAYSTATE

    lpNMDayState = (LPNMDAYSTATE) lParam;
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмдайстате**](/windows/win32/api/commctrl/ns-commctrl-nmdaystate) . Структура содержит сведения о кадре времени, для которого элемент управления нуждается в сведениях, и получает адрес массива, предоставляющего эти данные.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Нет возвращаемого значения.

## <a name="remarks"></a>Remarks

Обработка этого кода уведомления позволяет приложению настраивать его отображение, указывая, что определенные дни отображаются полужирным шрифтом.
