# <a name="dtm_setsystemtime-message"></a>\_Сообщение DTM сетсистемтиме

Задает время в элементе управления "Выбор даты и времени" (DTP). Это сообщение можно отправить явным образом или использовать макрос [**\_ сетсистемтиме DateTime**](/windows/desktop/api/Commctrl/nf-commctrl-datetime_setsystemtime) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Значение типа, указывающее действие, которое должно быть выполнено. Для этого значения необходимо задать одно из следующих значений:



| Значение                                                                                                                                             | Значение                                                                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="GDT_VALID"></span><span id="gdt_valid"></span><dl> <dt>**ГДТ \_ допустимое**</dt> </dl> | Установите элемент управления DTP в соответствии с данными в структуре, на которую указывает *lParam* . <br/>                                                                                                                                                                 |
| <span id="GDT_NONE"></span><span id="gdt_none"></span><dl> <dt>**ГДТ \_ None**</dt> </dl>    | Установите для элемента управления DTP значение "без даты" и снимите его флажок. Если этот флаг указан, параметр *lParam* игнорируется. Этот флаг применяется только к элементам управления DTP, для которых задан [**стиль \_ Шовноне служб DTS**](date-and-time-picker-control-styles.md) . <br/> |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**SYSTEMTIME**](/windows/desktop/api/minwinbase/ns-minwinbase-systemtime) , содержащую системное время, используемое для задания элемента управления DTP.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает ненулевое значение в случае успеха или ноль в противном случае.
