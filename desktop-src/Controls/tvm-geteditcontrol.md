# <a name="tvm_geteditcontrol-message"></a>\_Сообщение TVM жетедитконтрол

Получает маркер элемента управления "поле ввода", используемый для изменения текста элемента представления дерева. Это сообщение можно отправить явно или с помощью макроса [**\_ жетедитконтрол TreeView**](/windows/desktop/api/Commctrl/nf-commctrl-treeview_geteditcontrol) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает маркер в элемент управления "поле ввода", если он успешно, или **значение NULL** в противном случае.

## <a name="remarks"></a>Remarks

Когда начинается редактирование метки, элемент управления "Правка" создается, но не размещается или не отображается. Перед отображением элемент управления иерархического представления отправляет родительскому окну код уведомления [ТВН \_ бегинлабеледит](tvn-beginlabeledit.md) .

Чтобы настроить редактирование меток, реализуйте обработчик для [ТВН \_ бегинлабеледит](tvn-beginlabeledit.md) и отправьте сообщение **TVM \_ жетедитконтрол** в элемент управления "дерево-представление". Если метка редактируется, возвращаемое значение будет обработано элементом управления "поле ввода". Используйте этот маркер для настройки элемента управления "поле ввода", отправляя обычные сообщения **EM \_ xxx** .
