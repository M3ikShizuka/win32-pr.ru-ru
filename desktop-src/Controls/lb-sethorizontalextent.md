# <a name="lb_sethorizontalextent-message"></a>Сообщение сесоризонталекстент балансировки нагрузки \_

Задает ширину (в пикселях), на которую окно списка можно прокручивать горизонтально (прокручиваемую ширину). Если ширина списка меньше этого значения, горизонтальная полоса прокрутки горизонтально прокручивает элементы списка. Если ширина окна списка равна или больше этого значения, то горизонтальная полоса прокрутки скрыта.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Указывает число пикселей, на которое можно прокручивать список.

Windows 95/Windows 98/Windows Millennium Edition (Windows Me): параметр *wParam* ограничен 16-разрядными значениями.

</dd> <dt>

*lParam* 
</dt> <dd>

Этот параметр не используется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Это сообщение не возвращает значение.

## <a name="remarks"></a>Комментарии

Чтобы ответить на сообщение **\_ сесоризонталекстент балансировки нагрузки** , список должен быть определен с помощью стиля [**WS \_ HSCROLL**](/windows/desktop/winmsg/window-styles) .

Обратите внимание, что окно списка не обновляет горизонтальный экстент динамически.

Это сообщение не влияет на список с несколькими столбцами.
