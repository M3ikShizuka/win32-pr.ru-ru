# <a name="em_setimestatus-message"></a>\_Сообщение СЕТИМЕСТАТУС EM

Задает флаги состояния, определяющие взаимодействие элемента управления редактирования с редактором метода ввода (IME).

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Тип состояния, которое необходимо задать. Этот параметр может иметь следующее значение.



| Значение                                                                                                                                                                                       | Значение                                                       |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------|
| <span id="EMSIS_COMPOSITIONSTRING"></span><span id="emsis_compositionstring"></span><dl> <dt>**ЕМСИС \_ компоситионстринг**</dt> </dl> | Задает поведение для обработки строки композиции.<br/> |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Данные, относящиеся к типу состояния. Если параметр *wParam* имеет значение **емсис \_ компоситионстринг**, это может быть одно или несколько из следующих значений.



| Значение                                                                                                                                                                                                            | Значение                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="EIMES_GETCOMPSTRATONCE"></span><span id="eimes_getcompstratonce"></span><dl> <dt>**ЕИМЕС \_ жеткомпстратонце**</dt> </dl>                         | Если этот флаг установлен, элемент управления "поле ввода" ловит сообщение [**\_ \_ композиции редактора IME WM**](/windows/desktop/Intl/wm-ime-composition) с параметром *lParam* , имеющим значение "GC ресултстр", \_ и возвращает результирующую строку немедленно. Если этот флаг не установлен, элемент управления "поле ввода" передает сообщение **\_ \_ композиции редактора WM IME** в процедуру окна по умолчанию и обрабатывает результирующую строку из сообщения [**WM \_ char**](/windows/desktop/inputdev/wm-char) . это поведение элемента управления "поле ввода" по умолчанию.<br/> |
| <span id="EIMES_CANCELCOMPSTRINFOCUS"></span><span id="eimes_cancelcompstrinfocus"></span><dl> <dt>**ЕИМЕС \_ канцелкомпстринфокус**</dt> </dl>             | Если этот флаг установлен, элемент управления "поле ввода" отменяет строку композиции при получении сообщения [**WM \_ SETFOCUS**](/windows/desktop/inputdev/wm-setfocus) . Если этот флаг не установлен, элемент управления "поле ввода" не отменяет строку композиции; Это поведение элемента управления "поле ввода" по умолчанию.<br/>                                                                                                                                                                     |
| <span id="EIMES_COMPLETECOMPSTRKILLFOCUS"></span><span id="eimes_completecompstrkillfocus"></span><dl> <dt>**ЕИМЕС \_ комплетекомпстркиллфокус**</dt> </dl> | Если этот флаг установлен, элемент управления "поле ввода" завершает строку композиции после получения сообщения [**WM \_ киллфокус**](/windows/desktop/inputdev/wm-killfocus) . Если этот флаг не установлен, то элемент управления "поле ввода" не завершает строку композиции. Это поведение элемента управления "поле ввода" по умолчанию.<br/>                                                                                                                                                                 |



 

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает предыдущее значение параметра *lParam* .

## <a name="remarks"></a>Комментарии

**Расширенное редактирование:** Сообщение **EM \_ сетиместатус** не поддерживается.
