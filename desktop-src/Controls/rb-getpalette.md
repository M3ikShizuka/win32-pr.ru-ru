# <a name="rb_getpalette-message"></a>Сообщение RB в виде \_ палитры

Извлекает текущую палитру элемента управления "Главная панель".

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **хпалетте** , указывающий текущую палитру элемента управления главной панели.
