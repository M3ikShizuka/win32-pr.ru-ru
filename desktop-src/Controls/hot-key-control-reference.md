В этом разделе содержатся сведения об элементах программирования, используемых с элементами управления горячими ключами.

### <a name="overviews"></a>Разделы общих сведений



| Раздел                                    | Содержимое                                                                                                                     |
|------------------------------------------|------------------------------------------------------------------------------------------------------------------------------|
| [Элементы управления горячими ключами](hot-key-controls.md) | Ключевой элемент управления — это окно, которое позволяет пользователю ввести сочетание клавиш, которое будет использоваться в качестве сочетания клавиш.<br/> |

| [**ХКМ \_ сесоткэй**](hkm-sethotkey.md) | Задает сочетание горячих клавиш для элемента управления "горячий ключ".<br/>                                           |
| [**ХКМ \_ сетрулес**](hkm-setrules.md)   | Определяет недопустимые сочетания и сочетание модификатора по умолчанию для элемента управления горячего ключа. <br/> |
