# <a name="lvm_insertgroup-message"></a>\_Сообщение LVM инсертграуп

Вставляет группу в элемент управления "представление списка".

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Индекс, по которому добавляется группа. Если это значение равно-1, группа добавляется в конец списка.</dd> <dt>

*lParam* 
</dt> <dd>Указатель на структуру <a href="/windows/win32/api/commctrl/ns-commctrl-lvgroup">**лвграуп**</a> , содержащую добавляемую группу.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает индекс элемента, к которому была добавлена группа, или значение-1, если операция завершилась ошибкой.

## <a name="remarks"></a>Remarks

Чтобы включить режим группы, вызовите [**LVM \_ Енаблеграупвиев**](lvm-enablegroupview.md) или [**ListView \_ енаблеграупвиев**](/windows/desktop/api/Commctrl/nf-commctrl-listview_enablegroupview).

Невозможно вставить группу в пустой элемент управления "список-представление".

Не забудьте задать **играупид** в элементах, в которые была добавлена группа. В противном случае после обработки сообщения [**LVM \_ енаблеграупвиев**](lvm-enablegroupview.md) со значением **true** элемент управления ListView не будет отображать ни одного элемента.

> [!Note]  
> Чтобы использовать это сообщение, необходимо предоставить манифест с указанием Comclt32 версии 6,0. Дополнительные сведения о манифестах см. в разделе [Включение визуальных стилей](cookbook-overview.md).
