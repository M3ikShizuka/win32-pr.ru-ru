# <a name="em_getimeproperty-message"></a>\_Сообщение ЖЕТИМЕПРОПЕРТИ EM

Извлекает свойство и возможности редактора метода ввода (IME), связанного с текущим языковым стандартом ввода.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Указывает тип извлекаемых сведений о свойствах. Этот параметр может принимать одно из указанных ниже значений.



| Значение                                                                                                                                                                     | Значение                                                                                  |
|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------|
| <span id="IGP_PROPERTY"></span><span id="igp_property"></span><dl> <dt>**ИГП, \_ свойство**</dt> </dl>                | Сведения о свойстве.<br/>                                                         |
| <span id="IGP_CONVERSION"></span><span id="igp_conversion"></span><dl> <dt>**\_Преобразование ИГП**</dt> </dl>          | Возможности преобразования. <br/>                                                     |
| <span id="IGP_SENTENCE"></span><span id="igp_sentence"></span><dl> <dt>**ИГП \_ предложение**</dt> </dl>                | Возможности режима предложений. <br/>                                                  |
| <span id="IGP_UI"></span><span id="igp_ui"></span><dl> <dt>**\_Пользовательский интерфейс ИГП**</dt> </dl>                                  | Возможности пользовательского интерфейса. <br/>                                                 |
| <span id="IGP_SETCOMPSTR"></span><span id="igp_setcompstr"></span><dl> <dt>**ИГП \_ сеткомпстр**</dt> </dl>          | Возможности строк композиции. <br/>                                             |
| <span id="IGP_SELECT"></span><span id="igp_select"></span><dl> <dt>**ИГП \_ SELECT**</dt> </dl>                      | Возможности наследования выбора. <br/>                                          |
| <span id="IGP_GETIMEVERSION"></span><span id="igp_getimeversion"></span><dl> <dt>**ИГП \_ жетимеверсион**</dt> </dl> | Возвращает номер версии системы, для которой был создан указанный редактор IME. <br/> |



 

</dd> <dt>

*lParam* 
</dt> <dd>

Не используется; должно быть равно нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение свойства или возможности в зависимости от значения параметра *lParam* . Дополнительные сведения см. в разделе «Примечания».

## <a name="remarks"></a>Remarks

Если свойство *wParam* имеет значение ИГП \_ , оно возвращает одно или несколько из следующих значений.



| Требование | Значение |
|-------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_prop в редакторе IME \_ на \_ курсоре                | Если задано, окно преобразования находится в позиции курсора. Если флажок снят, окно находится ближе к позиции курсора.                                                                                                                                                                  |
| \_ \_ Специальный \_ Пользовательский интерфейс редактора IME              | Если задано, IME имеет нестандартный пользовательский интерфейс. Приложение не должно рисоваться в окне IME.                                                                                                                                                                  |
| Кандлист свойства IME, \_ \_ \_ начиная \_ с \_ 1 | Если задано значение, строки в списке кандидатов нумеруются, начиная с 1. Если флажок снят, строки начинаются с нуля.                                                                                                                                                                |
| редактор IME, \_ \_ кодировка Юникод                  | Если этот параметр задан, редактор IME просматривается как Уникодеиме. Система и редактор IME будут взаимодействовать через интерфейс Уникодеиме. Если флажок снят, IME будет использовать интерфейс ANSI для взаимодействия с системой.                                                                    |
| не \_ \_ выбрано Автозаполнение IME \_ \_   | Если задано, окно преобразования находится в позиции курсора. Если флажок снят, окно находится ближе к позиции курсора.                                                                                                                                                                  |
| режим IME " \_ \_ принять \_ широкий \_ ключ"       | Если этот параметр задан, редактор IME обрабатывает вставленный символ Юникода, который поступил из функции [**SendInput**](/windows/desktop/api/winuser/nf-winuser-sendinput) с помощью \_ пакета VK. Если это не так, редактор IME может не обработать введенный символ Юникода, а вставленный в него символ Unicode может быть отправлен непосредственно приложению. |



 

Если параметр *wParam* имеет значение ИГП \_ UI, он возвращает одно или несколько из следующих значений.



| Требование | Значение |
|-----------------|-------------------------------------------------------------------------------------------------------|
| Ограничение пользовательского интерфейса \_ \_ 2700   | Поддерживает значения режима escape-последовательности 0 или 2700. Дополнительные сведения см. в разделе **лфескапемент**.             |
| \_ROT90 Cap пользовательского интерфейса \_  | Поддерживает значения 0, 900, 1800 и 2700 для текста. Дополнительные сведения см. в разделе **лфескапемент**. |
| \_ротани Cap пользовательского интерфейса \_ | Поддерживает любое значение escape-последовательности текста. Дополнительные сведения см. в разделе **лфескапемент**.                       |



 

Если параметр *wParam* имеет значение ИГП \_ сеткомпстр, он возвращает одно или несколько из следующих значений.



| Требование | Значение |
|------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_КОМПСТР SCS Cap \_            | Может создать строку композиции, вызвав функцию [**иммсеткомпоситионстринг**](/windows/desktop/api/imm/nf-imm-immsetcompositionstringa) с параметром SCS \_ сетстр.                                                      |
| \_МАКЕРЕАД SCS Cap \_           | Может создать строку чтения из соответствующей строки композиции при использовании функции [**иммсеткомпоситионстринг**](/windows/desktop/api/imm/nf-imm-immsetcompositionstringa) с SCS \_ сетстр и без установки *лпреад*. |
| \_СЕТРЕКОНВЕРТСТРИНГ SCS Cap \_ | Этот редактор IME может поддерживать повторное преобразование. Используйте [**иммсеткомпоситионстринг**](/windows/desktop/api/imm/nf-imm-immsetcompositionstringa) для повторного преобразования.                                                                             |



 

Если параметр *wParam* имеет значение ИГП \_ SELECT, то он возвращает одно или несколько из следующих значений.



| Требование | Значение |
|-----------------------|------------------------------------------------------|
| Выбор \_ ограничения \_ конвмоде | Наследует режим преобразования, если выбран новый редактор IME. |
| Выбор \_ \_ предложения крепления | Наследует режим предложения при выборе нового редактора IME.   |



 

Если параметр *wParam* имеет значение ИГП \_ жетимеверсион, он возвращает одно или несколько из следующих значений.



| Требование | Значение |
|--------------|---------------------------------------------|
| Непостоянное \_ 0310 | редактор IME был создан для Windows 3,1.        |
| Непостоянное \_ 0400 | редактор IME был создан для Windows 95 или более поздней версии. |



 

Это сообщение похоже на [**иммжетпроперти**](/windows/desktop/api/imm/nf-imm-immgetproperty), за исключением того, что он использует текущий языковой стандарт ввода. Перед вызовом этой функции приложение должно вызвать [**EM \_ исиме**](em-isime.md) .
