# <a name="about-pager-controls"></a>Сведения об элементах управления страничного навигатора

*Элемент управления страничного навигатора* — это контейнер окон, который используется с окном, у которого недостаточно отображаемой области для отображения всего содержимого. Элемент управления страничный навигатор позволяет пользователю выполнить прокрутку до области окна, которая в настоящий момент не находится в представлении.

-   [Сведения об элементах управления страничного навигатора](#about-pager-controls)

## <a name="about-pager-controls"></a>Сведения об элементах управления страничного навигатора

В Microsoft Internet Explorer версии 4,0 (commctrl.dll версии 4,71) представлен элемент управления страничного навигатора. Этот элемент управления полезен в ситуациях, когда окно не имеет достаточной области для вывода дочернего окна. Например, если приложение имеет недостаточную панель инструментов для отображения всех ее элементов, можно назначить панель инструментов элементу управления страничного навигатора, а пользователи смогут прокручиваться влево или вправо для доступа ко всем элементам. Можно также создавать элементы управления страничного навигатора, которые прокручиваются вертикально.

Окно, назначенное элементу управления страничного навигатора, называется *автономным окном*.

На следующем снимке экрана показана панель инструментов, содержащаяся внутри элемента управления страничного навигатора. Элемент управления страничный навигатор отображается красным цветом, чтобы показать, какие области элемента управления видимы.

> [!Note]  
> Элемент управления страничного навигатора реализован в версии 4,71 и более поздних Comctl32.dll.
