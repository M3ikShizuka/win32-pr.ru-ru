# <a name="about-common-controls"></a>Общие сведения об элементах управления

В этом разделе обсуждаются общие элементы управления, набор окон, реализованный в общей библиотеке элементов управления, Comctl32.dll, который является библиотекой DLL, входящей в состав операционной системы Windows. Как и другие окна управления, обычный элемент управления — это дочернее окно, которое приложение использует в сочетании с другим окном для включения взаимодействия с пользователем.
