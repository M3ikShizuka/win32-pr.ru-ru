# <a name="em_displayband-message"></a>\_Сообщение ДИСПЛАЙБАНД EM

Отображает часть содержимого элемента управления Rich Edit, как было ранее отформатировано для устройства с помощью сообщения [**EM \_ форматранже**](em-formatrange.md) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется; оно должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Структура [**Rect**](/previous-versions//dd162897(v=vs.85)) , указывающая область дисплея устройства.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если операция выполнена удачно, возвращается значение **true**.

Если операция завершается неудачно, возвращается значение **false**.

## <a name="remarks"></a>Комментарии

Объекты текстовых и компонентных объектов модели (COM) обрезаются прямоугольником. Приложению не нужно задавать область отсечения.

Чередование — это процесс, с помощью которого одна страница выходных данных создается с использованием одного или нескольких отдельных прямоугольников или делений. Когда на странице помещаются все полосы, происходит полное изображение. Этот подход часто используется для растровых принтеров, которые не имеют достаточного объема памяти или могут одновременно создавать изображения на полной странице. К устройствам с чередованием относятся большинство матричных принтеров, а также некоторые лазерные принтеры.
