# <a name="tdm_set_progress_bar_state-message"></a>TDM \_ задать \_ \_ сообщение о \_ состоянии индикатора выполнения

Задает состояние индикатора выполнения в диалоговом окне задачи.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* \[ окне\]
</dt> <dd>

Значение **типа int** , указывающее состояние индикатора выполнения. Этот параметр может принимать одно из указанных ниже значений.



| Значение                                                                                                                                                   | Значение                                               |
|---------------------------------------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------|
| <span id="PBST_NORMAL"></span><span id="pbst_normal"></span><dl> <dt>**ПБСТ, \_ Обычная**</dt> </dl> | Задает для индикатора выполнения нормальное состояние.<br/> |
| <span id="PBST_PAUSED"></span><span id="pbst_paused"></span><dl> <dt>**ПБСТ \_ приостановлена**</dt> </dl>    | Задает состояние "приостановлено" для индикатора выполнения.<br/> |
| <span id="PBST_ERROR"></span><span id="pbst_error"></span><dl> <dt>**ПБСТ, \_ Ошибка**</dt> </dl>    | Задайте для индикатора выполнения состояние ошибки.<br/>   |



 

</dd> <dt>

*lParam* \[ окне\]
</dt> <dd>

Должен равняться нулю.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если функция выполнена, возвращаемое значение не равно нулю.

Если функция выполняется неудачно, возвращается нулевое значение. Для получения расширенной информации об ошибке вызова GetLastError.
