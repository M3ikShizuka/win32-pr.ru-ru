# <a name="lb_setsel-message"></a>Сообщение сетсел балансировки нагрузки \_

Выбирает элемент в списке с множественным выбором и при необходимости прокручивает его до представления.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Указывает, как задать выбор. Если этот параметр имеет **значение true**, элемент выделяется и выделяется; Если значение равно **false**, выделение удаляется и элемент больше не выбирается.

</dd> <dt>

*lParam* 
</dt> <dd>

Задает отсчитываемый от нуля индекс элемента, который необходимо задать. Если этот параметр имеет значение-1, выбор добавляется или удаляется из всех элементов, в зависимости от значения *wParam*, и прокрутка не выполняется.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если возникает ошибка, возвращается значение фунтов \_ Err.

## <a name="remarks"></a>Remarks

Это сообщение следует использовать только в списках с множественным выбором.
