# <a name="cb_setitemheight-message"></a>\_Сообщение СЕТИТЕМХЕИГХТ CB

Приложение отправляет сообщение **\_ сетитемхеигхт CB** , чтобы задать высоту элементов списка или поле выбора в поле со списком.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Указывает компонент поля со списком, для которого задается высота.

Этот параметр должен иметь значение 1, чтобы задать высоту поля выбора. Значение высоты элементов списка должно быть равно нулю, если только поле со списком не имеет стиль [**CBS \_ овнердраввариабле**](combo-box-styles.md) . В этом случае параметр *wParam* — это Отсчитываемый от нуля индекс определенного элемента списка.

</dd> <dt>

*lParam* 
</dt> <dd>

Задает высоту (в пикселях) компонента поля со списком, идентифицируемого параметром *wParam*.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Если индекс или высота являются недопустимыми, возвращается значение CB \_ Err.

## <a name="remarks"></a>Комментарии

Высота поля выбора в поле со списком задается независимо от высоты элементов списка. Приложение должно гарантировать, что высота поля выбора не меньше высоты определенного элемента списка.
