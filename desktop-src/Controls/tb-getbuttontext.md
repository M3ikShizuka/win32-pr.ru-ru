# <a name="tb_getbuttontext-message"></a>\_Сообщение ЖЕТБУТТОНТЕКСТ ТБ

Получает отображаемый текст кнопки на панели инструментов.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Идентификатор команды для кнопки, текст которой необходимо получить.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на буфер, который получает текст кнопки.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает длину строки в символах, на которую указывает *lParam*. Длина не включает завершающего нуль символа. В случае неудачи возвращаемое значение равно-1.

## <a name="remarks"></a>Комментарии

**Предупреждение системы безопасности:** Неправильное использование этого сообщения может нарушить безопасность программы. Это сообщение не позволяет получить сведения о размере буфера. Если вы используете это сообщение, сначала вызовите сообщение, передав ему **значение NULL** в параметре *lParam*, после чего будет возвращено необходимое число символов, исключая **null** . Затем вызовите сообщение еще раз, чтобы получить строку. Прежде чем продолжить, ознакомьтесь с [вопросами безопасности: элементы управления Microsoft Windows](sec-comctls.md) .

Возвращаемая строка соответствует тексту, который в данный момент отображается кнопкой.
