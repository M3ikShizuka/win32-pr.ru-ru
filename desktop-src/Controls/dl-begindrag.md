# <a name="dl_begindrag-notification-code"></a>\_Код уведомления бегиндраг списка рассылки

Сообщает родительскому окну поля списка перетаскивания, что пользователь щелкнул левую кнопку мыши на элементе. Поле со списком перетаскивания отправляет этот код уведомления в виде сообщения списка перетаскивания. Дополнительные сведения см. в статье [перетаскивание сообщений в поле со списком](about-list-boxes.md).


```C++
DL_BEGINDRAG

    pDragInfo = (LPARAM)(LPDRAGLISTINFO) lParam; 
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**драглистинфо**](/windows/win32/api/commctrl/ns-commctrl-draglistinfo) , содержащую \_ код уведомления DL бегиндраг, маркер для списка перетаскивания и позицию курсора.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , чтобы начать операцию перетаскивания, или **false** , чтобы предотвратить операцию перетаскивания.

## <a name="remarks"></a>Remarks

При обработке этого кода уведомления процедура окна обычно определяет элемент списка в указанной позиции курсора с помощью функции [**лбитемфромпт**](/windows/desktop/api/Commctrl/nf-commctrl-lbitemfrompt) . Затем он возвращает **значение true** или **false** в зависимости от того, следует ли перетаскивать элемент. Перед возвращением значения **true** процедура Window должна сохранить индекс элемента списка, чтобы приложение знало, какой элемент нужно переместить или скопировать после завершения операции перетаскивания.
