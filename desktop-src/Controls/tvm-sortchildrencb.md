# <a name="tvm_sortchildrencb-message"></a>\_Сообщение TVM сортчилдренкб

Сортирует элементы древовидного представления с помощью определяемой приложением функции обратного вызова, которая сравнивает элементы. Это сообщение можно отправить явно или с помощью макроса [**\_ сортчилдренкб TreeView**](/windows/desktop/api/Commctrl/nf-commctrl-treeview_sortchildrencb) .

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Зарезервировано. Должен равняться нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**твсорткб**](/windows/win32/api/commctrl/ns-commctrl-tvsortcb) . Элемент **лпфнкомпаре** является адресом определяемой приложением функции обратного вызова, которая вызывается во время операции сортировки каждый раз, когда необходимо сравнивать относительный порядок двух элементов списка. Дополнительные сведения о функции обратного вызова см. в описании **твсорткб**.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , если успешно, или **false** в противном случае.
