# <a name="sb_getparts-message"></a>\_Сообщение SB

Извлекает количество частей в окне состояния. Сообщение также получает координату правого края указанного числа частей.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Число частей, для которых необходимо получить координаты. Если этот параметр больше числа частей в окне, сообщение получает координаты только для существующих частей.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на целочисленный массив с тем же количеством элементов, что и части, заданные параметром *wParam*. Каждый элемент массива получает клиентскую координату правого края соответствующей части. Если элемент имеет значение-1, то расположение правого края для этой части расширяется до правого края окна. Чтобы получить текущее количество частей, присвойте этому параметру значение 0.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает количество частей в окне.

## <a name="remarks"></a>Комментарии

Это сообщение всегда возвращает количество частей в строке состояния.
