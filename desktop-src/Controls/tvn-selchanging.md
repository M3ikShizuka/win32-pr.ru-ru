# <a name="tvn_selchanging-notification-code"></a>\_Код уведомления ТВН селчангинг

Сообщает родительскому окну элемента управления древовидного представления о том, что выбор собирается измениться с одного элемента на другой. Этот код уведомления отправляется в виде сообщения [**WM \_ Notify**](wm-notify.md) .


```C++
TVN_SELCHANGING 

    pnmtv = (LPNMTREEVIEW) lParam 
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**нмтривиев**](/windows/win32/api/commctrl/ns-commctrl-nmtreeviewa) . Члены **итемолд** и **итемнев** содержат действительные сведения о текущем выбранном элементе и вновь выбранном элементе. Член **действия** указывает, вызывает ли действие мыши или клавиатуры изменение выбора. Список возможных значений см. в описании кода уведомления [ТВН \_ селчанжед](tvn-selchanged.md) .

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает **значение true** , чтобы предотвратить изменение выбора.

## <a name="remarks"></a>Комментарии

При ответе на этот код уведомления приложения не должны удалять элементы, которые получают или теряют выбор.
