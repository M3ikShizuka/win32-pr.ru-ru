# <a name="tb_getinsertmarkcolor-message"></a>\_Сообщение ЖЕТИНСЕРТМАРККОЛОР ТБ

Получает цвет, используемый для рисования метки вставки панели инструментов.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>Должен равняться нулю.</dd> <dt>

*lParam* 
</dt> <dd>Должен равняться нулю.</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращает значение [**COLORREF**](/windows/desktop/gdi/colorref) , содержащее текущий цвет отметки вставки.
