# <a name="visual-styles"></a>Стили оформления

## <a name="purpose"></a>Назначение

*Визуальные стили* изменяют внешний вид общих элементов управления на основе темы, выбранной пользователем. До Windows 8 необходимо специально настроить приложение для использования стилей оформления; в противном случае стандартные элементы управления приложения всегда подготавливаются к просмотру в стиле, связанном с классической темой Windows, независимо от текущей выбранной темы. В Windows 8 стили оформления не могут быть отключены, классический режим Windows больше не существует, и режим высокой контрастности был изменен для работы с визуальными стилями.

В этом разделе приводится обзор визуальных стилей и объясняется, как настроить приложение для их использования.

## <a name="developer-audience"></a>Аудитория разработчиков

Визуальные стили предназначены для разработчиков C/C++ и конструкторов пользовательского интерфейса. Как правило, разработчикам нужен умеренный уровень понимания концепций программирования пользовательского интерфейса, программирования Windows API и Юникода.

## <a name="run-time-requirements"></a>Требования к среде выполнения

Windows Vista и более поздние операционные системы.

> [!Note]  
> Визуальные стили не поддерживаются в режиме 256-цвета.
