# <a name="tdm_set_button_elevation_required_state-message"></a>\_Сообщение о \_ \_ состоянии "требуется повышение прав" для кнопки TDM \_ \_

Указывает, должен ли данная кнопка диалогового окна задачи или ссылка на команду иметь значок щита контроля учетных записей (UAC). то есть, требуется ли повышение прав для действия, вызываемого кнопкой.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* \[ окне\]
</dt> <dd>

Идентификатор обновляемой кнопки или ссылки на команду.

</dd> <dt>

*lParam* \[ окне\]
</dt> <dd>

Задайте значение 0, чтобы указать, что действие, вызываемое кнопкой, не требует повышения прав. Задайте значение ненулевой, чтобы обозначить, что действие требует повышения прав.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Возвращаемое значение игнорируется.
