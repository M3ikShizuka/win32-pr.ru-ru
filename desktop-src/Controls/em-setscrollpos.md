# <a name="em_setscrollpos-message"></a>\_Сообщение СЕТСКРОЛЛПОС EM

Прокручивает содержимое элемента управления Rich Edit до указанной точки.

## <a name="parameters"></a>Параметры

<dl> <dt>

*wParam* 
</dt> <dd>

Этот параметр не используется; оно должно быть равно нулю.

</dd> <dt>

*lParam* 
</dt> <dd>

Указатель на структуру [**точек**](/previous-versions//dd162805(v=vs.85)) , указывающую точку в виртуальном текстовом пространстве документа, выраженную в пикселях. Документ будет прокручиваться до тех пор, пока эта точка не будет расположена в левом верхнем углу окна редактирования элемента управления. Если необходимо изменить представление так, чтобы левый верхний угол представления был не больше двух строк, а затем один символ из левого края. Необходимо передать точку (7, 22).

Форматируемый элемент управления "поле ввода" проверяет координаты x и y и корректирует их при необходимости, чтобы в верхней части отображалась полная строка. Это также гарантирует, что текст не будет полностью прокручиваться за пределы прямоугольника представления.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Это сообщение всегда возвращает значение 1.
