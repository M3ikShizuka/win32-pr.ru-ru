# <a name="levels-of-snmp-support"></a>Уровни поддержки SNMP

Реализация Microsoft WinSNMP обеспечивает поддержку SNMP-связи уровня 2. Уровень 2 поддерживает стандартный протокол SNMPv2, определенный в RFC 1902-1908. Она также поддерживает оболочку сообщений на основе сообщества, как определено в RFC 1901 (SNMPv2C).

Поддержка связи уровня 2 включает в себя кодирование сообщений и службы декодирования, ранее именуемые поддержкой связи уровня 0 в WinSNMP версии 1.1 a. Уровень 2 также поддерживает все операции протокола в версиях, которые ранее назывались поддержкой связи уровня 1 в WinSNMP версии 1.1 a.

Реализация Возвращает максимальный уровень SNMP-взаимодействия, который она поддерживает, в ответ на вызов функции WinSNMP приложением в функцию [**снмпстартуп**](/windows/desktop/api/Winsnmp/nf-winsnmp-snmpstartup) .

Если приложение WinSNMP использует реализацию для кодирования сообщений SNMP и только для декодирования, приложение должно выполнить необходимые преобразования, которые будут выполнены в реализации. Это включает преобразование ловушек в неограниченном числе, возвращенных вызовом функции [**снмпреквмсг**](/windows/desktop/api/Winsnmp/nf-winsnmp-snmprecvmsg) для SNMPv2c ловушек. Он также включает преобразование типов PDU, определенных в стандарте, в соответствующий тип PDU, определенный в SNMPv2C, в соответствии с RFC 1908.
