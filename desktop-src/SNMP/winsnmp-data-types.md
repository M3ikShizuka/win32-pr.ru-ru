# <a name="winsnmp-data-types"></a>Типы данных WinSNMP

\[Протокол SNMP доступен для использования в операционных системах, указанных в разделе требования. В последующих версиях он может быть изменен или недоступен. Вместо этого используйте [Служба удаленного управления Windows](/windows/desktop/WinRM/portal), который является реализацией Microsoft WS-Man.\]

Стандартные типы данных WinSNMP определяются в WINSNMP. H файл.

Обратите внимание, что параметру WinSNMP указываются некоторые параметры с типом длинного целого числа со знаком **смиинт**. Это позволяет параметру WinSNMP соответствовать элементам данных, особенно компонентам PDU, определенным в соответствующих документах RFC.
