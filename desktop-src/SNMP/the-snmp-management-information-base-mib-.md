# <a name="the-snmp-management-information-base-mib"></a>База данных управления SNMP (MIB)

База данных управления (MIB) описывает набор управляемых объектов. Консольное приложение управления SNMP может управлять объектами на определенном компьютере, если в службе SNMP есть библиотека DLL агента расширения, поддерживающая MIB.

Каждый управляемый объект в MIB имеет уникальный идентификатор. Идентификатор включает тип объекта (например, счетчик, строка, датчик или адрес), уровень доступа объекта (например, чтение или чтение/запись), ограничения размера и сведения о диапазоне.

В следующей таблице содержится частичный список MIB, поставляемых с системой. Они устанавливаются вместе со службой SNMP в каталоге% systemroot% \\ System32. Полный список MIB см. в разделе Windows Resource Kit.



| MIB         | Описание                                                                                                                                      |
|-------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| Служба. MIB    | Определяемая корпорацией Майкрософт MIB, которая содержит типы объектов для мониторинга сетевого трафика между удаленными узлами и DHCP-серверами.                        |
| ХОСТМИБ. MIB | Содержит типы объектов для мониторинга и управления ресурсами узла                                                                                 |
| LMMIB2. MIB  | Описание служб рабочей станции и сервера                                                                                                           |
| MIB \_ II. MIB | Содержит базу управляющих данных (MIB-II), которая предоставляет простую, работоспособную архитектуру и систему для управления Интернет-клиентами на основе TCP/IP. |
| Адрес. MIB    | Определяемая корпорацией Майкрософт MIB для службы Windows Internet Name Service (WINS)                                                                               |



 

**Windows NT:** Как правило, можно скопировать MIB из агента расширения SNMP, который поддерживает определенную MIB. Некоторые дополнительные MIB доступны в комплекте Windows NT 4,0 Resource Kit.

Библиотеки DLL агента расширения для MIB-II, диспетчера LAN MIB-II и MIB ресурсов узла устанавливаются вместе со службой SNMP. Библиотеки DLL агента расширения для других MIB устанавливаются при установке соответствующих служб. Во время запуска службы SNMP загружает все библиотеки DLL агента расширения, перечисленные в реестре.

Пользователи могут добавлять другие библиотеки DLL агента расширения, которые реализуют дополнительные MIB. Для этого необходимо добавить запись реестра для новой библиотеки DLL в службе SNMP. Кроме того, они должны зарегистрировать новую MIB с помощью консольного приложения управления SNMP. Дополнительные сведения см. в документации, входящей в состав консольного приложения управления.

Дополнительные сведения см. в разделе [MIB Name Tree](mib-name-tree.md).
