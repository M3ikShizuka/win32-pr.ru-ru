# <a name="configuring-snmp-security"></a>Настройка безопасности SNMP

Функции безопасности SNMP позволяют указать сообщества и узлы, на которых компьютер принимает запросы, а также тип операций, которые должны приниматься от компьютеров, принадлежащих сообществом. Протокол SNMP также позволяет указать, следует ли отсылать ловушки проверки подлинности, когда несанкционированный сообщество или узел запрашивают информацию.

Дополнительные сведения см. в разделе Windows Resource Kit и в электронной документации по службе SNMP.

**Windows Server 2003:** По умолчанию имена сообществ не настроены, а служба SNMP настроена на прием запросов SNMP только с имени узла localhost.
