# <a name="working-with-variable-binding-lists"></a>Работа с списками привязок переменных

*Переменная Binding* — это связывание имени экземпляра объекта SNMP со связанным значением. *Список привязок переменных* представляет собой последовательность записей привязки переменных. В WinSNMP единица данных протокола (PDU) включает список переменных привязки.

Сведения о структуре списка привязок переменных ограничиваются реализацией Microsoft WinSNMP. WinSNMP-приложение может обращаться к списку привязок переменных с помощью маркера типа **хснмп \_ вбл**. Дополнительные сведения см. в разделе [объекты обработчика ресурсов](resource-handle-objects.md).

В приложении WinSNMP можно создавать списки привязок переменных и управлять ими, а также включать их в устройства PDU. Для выполнения этих операций приложение использует [функции привязки переменной](winsnmp-functions.md)WinSNMP. Дополнительные сведения о работе с таблицей WinSNMP и переменными привязками см. в разделах, перечисленных в следующей таблице.

| [Управление списком привязок переменных](managing-a-variable-binding-list.md) | Описывает, как управлять списком привязок переменных. |



 

Дополнительные сведения о привязках переменных и списках привязок переменных см. в разделе [RFC1905](https://www.ietf.org/rfc/rfc1905.txt), "операции протокола для версии 2 протокола простого сетевого управления (SNMPv2)" и [функции привязки переменной](winsnmp-functions.md)WinSNMP.
