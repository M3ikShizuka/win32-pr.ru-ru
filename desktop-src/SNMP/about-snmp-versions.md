# <a name="about-snmp-versions"></a>О версиях SNMP

Исходная Стандартная инфраструктура управления сетью по стандарту Internet Standard, описанная в RFC 1155, 1157 и 1213, называется платформой SNMP версии 1. Соответствующие части предлагаемой платформы для версии 2C протокола простого сетевого управления (SNMPv2C) описаны в документах RFC 1901 – 1908.

API-интерфейс WinSNMP поддерживает функции протокола SNMP, описанные в соответствующих документах RFC в Интернете. При использовании WinSNMP приложения не накладывает ограничений на использование SNMPv2C.

Сущность управления может поддерживать другую версию протокола SNMP, чем та, которую поддерживает приложение WinSNMP. Реализация Microsoft WinSNMP выполняет соответствующие преобразования из SNMPv2C в соответствии с соответствующей RFC.
