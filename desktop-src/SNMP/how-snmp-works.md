# <a name="how-snmp-works"></a>Как работает SNMP

Ниже показано, как приложение консоли управления SNMP от стороннего производителя Возвращает сведения из службы SNMP:

1.  Консольное приложение управления SNMP формирует SNMP-сообщение на основе входных данных пользователя. Сообщение включает блок данных протокола (PDU) и сведения о проверке подлинности. Для выполнения этого шага приложение консоли управления может использовать библиотеку API управления Microsoft SNMP (MGMTAPI.DLL) или библиотеку Microsoft [WINSNMP API](winsnmp-api.md) (WSNMP32.DLL).
2.  Консольное приложение управления SNMP отправляет SNMP-сообщение службе SNMP, используя библиотеки службы SNMP.
3.  Служба SNMP получает запрос. Он проверяет данные проверки подлинности и исходный IP-адрес.
4.  Служба SNMP выбирает соответствующий агент расширения и запрашивает получение агентом запрошенной информации.
5.  Служба SNMP отправляет ответ в консольное приложение управления SNMP.
