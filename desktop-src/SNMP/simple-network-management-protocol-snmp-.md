# <a name="simple-network-management-protocol-snmp"></a>Протокол SNMP

\[Протокол SNMP доступен для использования в операционных системах, указанных в разделе требования. В последующих версиях он может быть изменен или недоступен. вместо этого используйте [служба удаленного управления Windows](/windows/desktop/WinRM/portal), который является реализацией Microsoft WS-Man.\]

Протокол SNMP — это стандартный протокол Интернета для обмена данными управления между приложениями консоли управления, такими как HP OpenView, Novell узлов, IBM Нетвиев или Sun NET Manager, и управляемыми объектами. Управляемые объекты могут включать узлы, маршрутизаторы, мосты и концентраторы.
