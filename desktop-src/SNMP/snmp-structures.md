# <a name="snmp-structures"></a>Структуры SNMP

\[Протокол SNMP доступен для использования в операционных системах, указанных в разделе требования. В последующих версиях он может быть изменен или недоступен. Вместо этого используйте [Служба удаленного управления Windows](/windows/desktop/WinRM/portal), который является реализацией Microsoft WS-Man.\]

В следующей таблице перечислены структуры, используемые с SNMP.



| Структура SNMP                                         | Описание                                                                                             |
|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [**аснани**](/windows/desktop/api/Snmp/ns-snmp-asnany)                           | Описывает тип и значение указанной переменной SNMP.                                              |
| [**AsnCounter64**](/previous-versions/windows/desktop/legacy/aa377953(v=vs.85))               | Содержит 64-разрядный счетчик, заданный двумя значениями, описывающими его младшие и старшие разряды. |
| [**аснобжектидентифиер**](/windows/desktop/api/Snmp/ns-snmp-asnobjectidentifier) | Описывает идентификатор объекта (OID).                                                                   |
| [**асноктетстринг**](/windows/desktop/api/Snmp/ns-snmp-asnoctetstring)           | Представляет строку октетов, обычно в байтовых значениях.                                                  |
| [**снмпварбинд**](/windows/desktop/api/Snmp/ns-snmp-snmpvarbind)                 | Описывает привязку переменной SNMP.                                                                     |
| [**снмпварбиндлист**](/windows/desktop/api/Snmp/ns-snmp-snmpvarbindlist)         | Содержит список привязок к переменным SNMP.                                                              |
