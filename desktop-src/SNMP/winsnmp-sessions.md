# <a name="winsnmp-sessions"></a>Сеансы WinSNMP

*Сеанс* — это основная единица управления ресурсами и связью между вызывающим приложением WinSNMP и реализацией Microsoft WinSNMP. Рекомендуется, чтобы приложение использовало сеансы для организации своих операций и снижения спроса на ресурсы реализации.

Дополнительные сведения см. [в разделе Открытие и закрытие сеанса WinSNMP](opening-and-closing-a-winsnmp-session.md).
