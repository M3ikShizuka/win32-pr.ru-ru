# <a name="about-the-winsnmp-20-addendum"></a>Об дополнении WinSNMP 2,0

Дополнение версии 2,0 — это обновление спецификации API WinSNMP Manager версии 1.1 a. (В настоящее время спецификация не определяет версию 2,0 для API WinSNMP.) Это дополнение было совместно разработано с учетом взаимодействия, ввода и поддержки нескольких компаний, ассоциаций и отдельных пользователей.

В спецификации WinSNMP 1.1 определяется прикладной программный интерфейс WinSNMP Manager для разработки приложений управления сетью на основе SNMP. В дополнении WinSNMP версии 2,0 интерфейс переименовывается как API-интерфейс WinSNMP. Это вызвано тем, что Ошибка WinSNMP 2,0 поддерживает SNMP-операции агента и диспетчера. При программировании в среде WinSNMP вы должны быть знакомы с обоими документами.

Реализация Microsoft WinSNMP совместима с WinSNMP 2,0. Он поддерживает все функции и операции, определенные как в спецификации WinSNMP 1.1, так и в дополнении WinSNMP 2,0.
