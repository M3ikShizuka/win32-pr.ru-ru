# <a name="system-files-for-snmp"></a>Системные файлы для SNMP

В следующей таблице описаны основные файлы, относящиеся к службе SNMP.



| Имя файла     | Описание                                                                                                                                                                                                                         |
|--------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| DHCPMIB.DLL  | Библиотека DLL агента расширения, которая реализует интерфейс MIB DHCP, определяемый корпорацией Майкрософт. Устанавливается только на DHCP-серверах.                                                                                                                                 |
| EVNTAGNT.DLL | DLL-библиотека SNMP, которая преобразует журналы событий в ловушки SNMP; также известен как транслятор событий SNMP.                                                                                                                                       |
| HOSTMIB.DLL  | Библиотека DLL агента расширения, которая реализует MIB ресурсов узла.                                                                                                                                                                         |
| LMMIB2.DLL   | Библиотека DLL агента расширения, которая реализует MIB LAN Manager-II.                                                                                                                                                                             |
| MGMTAPI.DLL  | Библиотека API управления Microsoft SNMP. Этот API позволяет приложениям диспетчера SNMP "прослушивать" запросы SNMP Manager, отправлять запросы и получать ответы от агентов SNMP.                                                |
| MIB. ЯЧЕЙКИ      | Скомпилированные данные MIB, используемые MGMTAPI.DLL.                                                                                                                                                                                       |
| SNMP.EXE     | Служба SNMP. Это главный агент, который получает SNMP-запросы и доставляет их в соответствующую библиотеку DLL агента расширения.                                                                                                        |
| SNMPAPI.DLL  | Библиотека DLL служебных программ SNMP, используемая библиотеками DLL агента расширения SNMP и приложениями диспетчера. Эта библиотека DLL содержит платформу для разработки библиотек DLL агента расширения.                                                                                   |
| SNMPSNAP.DLL | Приложение конфигурации SNMP, которое является компонентом оснастки консоли управления (MMC). Оснастка добавляет несколько страниц в страницу свойств службы SNMP. Дополнительные сведения см. в справке службы SNMP в Интернете. |
| SNMPTRAP.EXE | Служба ловушек SNMP. Получает SNMP-ловушки и пересылает их в приложения SNMP Manager.                                                                                                                                              |
| WINSMIB.DLL  | Библиотека DLL агента расширения, которая реализует базу MIB WINS, определенную корпорацией Майкрософт. Устанавливается только на WINS-серверы.                                                                                                                                 |
| WSNMP32.DLL  | Библиотека Microsoft [WINSNMP API](winsnmp-api.md) . Этот API позволяет приложениям диспетчера SNMP "прослушивать" запросы SNMP Manager, отправлять запросы и получать ответы от агентов SNMP.                                     |



 

Дополнительные сведения см. [в статье база данных управления SNMP (MIB)](the-snmp-management-information-base-mib-.md). (также можно обратиться к комплекту Windows 2000 resource Kit.)
