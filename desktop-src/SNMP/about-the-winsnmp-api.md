# <a name="about-the-winsnmp-api"></a>Сведения об API WinSNMP

\[Протокол SNMP доступен для использования в операционных системах, указанных в разделе требования. В последующих версиях он может быть изменен или недоступен. вместо этого используйте [служба удаленного управления Windows](/windows/desktop/WinRM/portal), который является реализацией Microsoft WS-Man.\]

WinSNMP API — это интерфейс для разработки приложений управления сетью с поддержкой SNMP.

API-интерфейс WinSNMP предоставляет следующие возможности.

-   Технология SNMP-включения для приложений управления сетью
-   Поддержка SNMP версии 1 и SNMP версии 2C (SNMPv2C)

Помимо операций SNMP Manager, функция WinSNMP API версии 2,0 также поддерживает операции агента SNMP.

API WinSNMP поддерживает 32-разрядные приложения и выполняется в однои многопоточных средах. поддержка API WinSNMP доступна для приложений, которые выполняются в операционной среде Windows 2000 или более поздней версии.
