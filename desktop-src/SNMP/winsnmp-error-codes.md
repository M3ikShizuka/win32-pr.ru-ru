# <a name="winsnmp-error-codes"></a>Коды ошибок WinSNMP

\[Протокол SNMP доступен для использования в операционных системах, указанных в разделе требования. В последующих версиях он может быть изменен или недоступен. вместо этого используйте [служба удаленного управления Windows](/windows/desktop/WinRM/portal), который является реализацией Microsoft WS-Man.\]

> [!Note]  
> Ошибки, описанные в этом разделе, отличаются от кодов ошибок SNMP, определенных соответствующими RFC.

 

Все функции WinSNMP возвращают значение **снмпапи \_** в случае сбоя функции. Чтобы получить расширенные сведения об ошибке при сбое функции WinSNMP, приложение WinSNMP должно немедленно вызвать функцию [**снмпжетластеррор**](/windows/desktop/api/Winsnmp/nf-winsnmp-snmpgetlasterror) . В следующей таблице перечислены разделы, в которых обсуждаются коды расширенных ошибок, возвращаемые функциями WinSNMP.



| Раздел                                                        | Описание                                             |
|--------------------------------------------------------------|---------------------------------------------------------|
| [Общие коды ошибок WinSNMP](winsnmp-common-error-codes.md) | Описывает распространенные коды ошибок для API WinSNMP.       |
| [Ошибки сетевого транспорта](network-transport-errors.md)     | Описание ошибок сетевого транспорта для API WinSNMP. |



 

Ошибки WinSNMP, которые содержат сведения, относящиеся к контексту, содержатся на странице справки по функциям.
