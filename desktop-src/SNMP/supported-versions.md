# <a name="supported-versions"></a>Поддерживаемые версии

\[Протокол SNMP доступен для использования в операционных системах, указанных в разделе требования. В последующих версиях он может быть изменен или недоступен. Вместо этого используйте [Служба удаленного управления Windows](/windows/desktop/WinRM/portal), который является реализацией Microsoft WS-Man.\]

Windows Vista и Windows Server 2008 поддерживают поддержку в SNMPv2c. В этом разделе содержатся основные сведения о каждой из этих версий SNMP.

Протокол простого управления сетью версии 1 (SP1) является самой ранней версией протокола. Спецификация "стандарт" определяется RFC 1157, RFC 1155 и RFC 1212. В системе безопасности в соответствии со строками сообщества.

Протоколы версии 2C (SNMPv2c) определяются стандартом RFC 1901, RFC 1905 и RFC 1906. SNMPv2c обновляет операции протокола и типы данных, определенные в SNMPv2p. SNMPv2c, известный как «SNMPv2 на основе строк сообщества», использует безопасность на основе сообщества.

SNMPv2x поддерживается на платформах клиентских и серверных операционных систем Windows 2000 и более поздних версий, включая Windows XP и Windows Server 2003.
