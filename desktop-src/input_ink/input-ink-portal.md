Узнайте о различных интерфейсах API рукописного ввода для приложений Windows.

| Раздел | Описание |
|---|---|
| [Показ рукописных данных](ink-presenter.md)<br/> | Интерфейсы API-интерфейсов рукописного ввода позволяют приложениям Microsoft Win32 управлять вводом, обработкой и отрисовкой рукописного ввода (Standard и Modified) с помощью объекта [**InkPresenter**](/uwp/api/Windows.UI.Input.Inking.InkPresenter) , вставленного в визуальное дерево [DirectComposition](../directcomp/directcomposition-portal.md) приложения.<br/> |
