# <a name="encoding-windows-web-services"></a>Кодировка (веб-службы Windows)

Кодирование — это преобразование между XML и линейным форматом сети. Декодирование является обратным преобразованием.


Кодировка определяется с помощью константы [**\_ кодировки WS**](/windows/desktop/api/WebServices/ne-webservices-ws_encoding) .
