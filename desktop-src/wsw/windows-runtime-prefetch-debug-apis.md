# <a name="prefetch-debug-apis-for-windows-store-apps"></a>Предзагрузка API отладки для приложений Магазина Windows

Следующие API используются для отладки и тестирования поведения при выборке содержимого в приложении для Магазина Windows, которое реализует класс [**контентпрефетчер**](/uwp/api/Windows.Networking.BackgroundTransfer.ContentPrefetcher) .



| Интерфейс                                                              | Описание                                                                                                                                                                      |
|------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**иконтентпрефетчертасктригжер**](/windows/desktop/api/IContentPrefetcherTaskTrigger/nn-icontentprefetchertasktrigger-icontentprefetchertasktrigger) | Используйте этот интерфейс, чтобы проверить, зарегистрирован ли установленный пакет приложения для Магазина Windows для этой фоновой задачи, и вручную активировать его операции предварительного выборки содержимого.<br/> |
