# <a name="prefetch-debug-apis-for-windows-store-apps"></a>предзагрузка api отладки для приложений магазина Windows

следующие api используются для отладки и тестирования поведения при выборке содержимого в приложении Windows Store, которое реализует класс [**контентпрефетчер**](/uwp/api/Windows.Networking.BackgroundTransfer.ContentPrefetcher) .



| Интерфейс                                                              | Описание                                                                                                                                                                      |
|------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**иконтентпрефетчертасктригжер**](/windows/desktop/api/IContentPrefetcherTaskTrigger/nn-icontentprefetchertasktrigger-icontentprefetchertasktrigger) | используйте этот интерфейс, чтобы убедиться, что установленный пакет приложения для магазина Windows зарегистрирован для этой фоновой задачи и вручную активирует его операции предварительного выборки содержимого.<br/> |
