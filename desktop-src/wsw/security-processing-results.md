# <a name="security-processing-results"></a>Результаты обработки безопасности

В защищенном канале в приложение доставляются только те сообщения, которые успешно прошли проверку безопасности. Для этих сообщений некоторые результаты проверки безопасности присоединяются в виде свойств сообщения, и приложение может извлекать и исследовать эти свойства для выполнения дополнительных действий, таких как проверка авторизации.


Функцию [**всжетмессажепроперти**](/windows/desktop/api/WebServices/nf-webservices-wsgetmessageproperty) можно использовать для получения любых свойств, связанных с безопасностью, определенных в [**\_ \_ \_ идентификаторе свойства сообщения WS**](/windows/desktop/api/WebServices/ne-webservices-ws_message_property_id). **Всжетмессажепроперти** возвращает ошибку для запросов, которые запрашивают свойства безопасности, неприменимые к типу безопасности, используемому в канале. Сообщение остается владельцем свойств, возвращаемых функцией запроса.

С результатами обработки безопасности используются следующие элементы API.

| Перечисление                                                                | Описание                                                                                 |
|----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| [**\_ \_ идентификатор свойства токена безопасности WS \_ \_**](/windows/desktop/api/WebServices/ne-webservices-ws_security_token_property_id) | Определяет ключи для полей и свойств, которые могут быть извлечены из маркера безопасности. |



 



| Функция                                                         | Описание                                           |
|------------------------------------------------------------------|-------------------------------------------------------|
| [**всжетсекурититокенпроперти**](/windows/desktop/api/WebServices/nf-webservices-wsgetsecuritytokenproperty) | Извлекает из маркера безопасности поле или свойство. |



 



| Дескриптор                                       | Описание                                     |
|----------------------------------------------|-------------------------------------------------|
| [\_токен безопасности \_ WS](ws-security-token.md) | Непрозрачный маркер, представляющий маркер безопасности. |
