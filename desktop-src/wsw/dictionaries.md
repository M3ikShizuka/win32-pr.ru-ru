# <a name="dictionaries"></a>Словари

Словари позволяют разрешать разметку общих данных узла, таких как имена элементов или атрибутов, пространства имен и общий текст в двоичном XML-формате.


[**Словарь>**](/windows/desktop/api/WebServices/ns-webservices-ws_xml_dictionary) может быть указан как часть типа [**\_ \_ строки WS XML**](/windows/desktop/api/WebServices/ns-webservices-ws_xml_string) . Все строки в одном **словаре>** должны быть уникальными.

Идентификатор словаря поддерживается для всех кодировок (а не только для двоичного формата XML).

В словарях используется следующая функция:

-   [**всжетдиктионари**](/windows/desktop/api/WebServices/nf-webservices-wsgetdictionary)
