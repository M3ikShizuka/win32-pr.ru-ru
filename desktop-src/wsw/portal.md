# <a name="windows-web-services-api"></a>Windows API веб-служб

## <a name="purpose"></a>Назначение

ВВСАПИ — это реализация SOAP в машинном коде, которая обеспечивает основные функциональные возможности сетевого взаимодействия путем поддержки набора WS- \* и. \* Семейство протоколов NET. ВВСАПИ предназначен для использования компонентами и приложениями, которые относятся к одной из следующих категорий:

-   Требование к машинному коду
-   Требовать минимальных зависимостей
-   Требовать минимальное время запуска
-   Среды с ограниченным объемом памяти

## <a name="developer-audience"></a>Аудитория разработчиков

Windows API веб-служб (ВВСАПИ) предоставляет программный интерфейс (API) C/C++ для создания на них веб-служб и клиентов на основе SOAP. Основное понимание веб-служб и связанных с ними протоколов важно для использования этого API.

## <a name="run-time-requirements"></a>Требования к среде выполнения

Windows API веб-служб (ввсапи) — это компонент операционной системы Windows 7 и Windows Server 2008 R2 или более поздних версий Microsoft Windows.
