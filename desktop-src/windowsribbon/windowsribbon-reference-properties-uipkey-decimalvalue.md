# <a name="ui_pkey_decimalvalue"></a>UI \_ PKEY \_ деЦималвалуе

Определяет свойство UI \_ PKEY \_ деЦималвалуе.

```
propertyDescription
   name = UI_PKEY_DecimalValue
   shellPKey = UI_PKEY_DecimalValue
   formatID = 00000201-7363-696e-8441798acf5aebb7
   propID = 201
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Remarks

UI \_ PKEY \_ деЦималвалуе используется приложением для запроса значения в поле редактирования элемента управления " [Счетчик](windowsribbon-controls-spinner.md) ".
