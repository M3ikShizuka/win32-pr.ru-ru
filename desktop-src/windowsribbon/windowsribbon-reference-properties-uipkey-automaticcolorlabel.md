# <a name="ui_pkey_automaticcolorlabel"></a>UI \_ PKEY \_ аутоматикколорлабел

Определяет свойство UI \_ PKEY \_ аутоматикколорлабел.

```
propertyDescription
   name = UI_PKEY_AutomaticColorLabel
   shellPKey = UI_PKEY_AutomaticColorLabel
   formatID = 00000406-7363-696e-8441798acf5aebb7
   propID = 406
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Remarks

UI \_ PKEY \_ аутоматикколорлабел используется приложением для запроса значения метки, связанной с **автоматическим** нажатием кнопки [**дропдовнколорпиккер**](windowsribbon-element-dropdowncolorpicker.md). Эта кнопка используется приложением для указания цвета по умолчанию по мере необходимости.

\_АУТОМАТИККОЛОРЛАБЕЛ UI PKEY \_ допустим только для [**дропдовнколорпиккер**](windowsribbon-element-dropdowncolorpicker.md) , где или указывается в `ThemeColors` `StandardColors` качестве значения атрибута **колортемплате** .
