# <a name="ui_pkey_minvalue"></a>UI \_ PKEY \_ MinValue

Определяет \_ \_ Свойство MinValue UI PKEY.

```
propertyDescription
   name = UI_PKEY_MinValue
   shellPKey = UI_PKEY_MinValue
   formatID = 00000204-7363-696e-8441798acf5aebb7
   propID = 204
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Комментарии

UI \_ PKEY \_ класса MinValue используется приложением для запроса минимального значения элемента управления " [Счетчик](windowsribbon-controls-spinner.md) ".
