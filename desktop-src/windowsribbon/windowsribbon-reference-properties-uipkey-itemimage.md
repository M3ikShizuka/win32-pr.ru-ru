# <a name="ui_pkey_itemimage"></a>UI \_ PKEY \_ итемимаже

Определяет свойство UI \_ PKEY \_ итемимаже.

```
propertyDescription
   name = UI_PKEY_ItemImage
   shellPKey = UI_PKEY_ItemImage
   formatID = 00000106-7363-696e-8441798acf5aebb7
   propID = 106
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Remarks

UI \_ PKEY \_ итемимаже используется приложением для запроса изображения, связанного с элементом или командой, в элементе управления галереи.

Значение свойства является объектом [**иуиимаже**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) .
