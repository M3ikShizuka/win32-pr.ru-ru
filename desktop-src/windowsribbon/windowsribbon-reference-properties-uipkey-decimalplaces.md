# <a name="ui_pkey_decimalplaces"></a>Число \_ десятичных элементов пользовательского интерфейса \_

Определяет свойство "число \_ десятичных элементов пользовательского интерфейса" \_ .

```
propertyDescription
   name = UI_PKEY_DecimalPlaces
   shellPKey = UI_PKEY_DecimalPlaces
   formatID = 00000206-7363-696e-8441798acf5aebb7
   propID = 206
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Remarks

\_Число знаков \_ подсистемы пользовательского интерфейса, которое используется приложением для запроса количества цифр, отображаемых после десятичной запятой в поле редактирования элемента управления " [Счетчик](windowsribbon-controls-spinner.md) ".
