# <a name="ui_pkey_increment"></a>\_ \_ ПРИРАЩЕНИЕ пользовательского интерфейса

Определяет \_ \_ свойство ИНКРЕМЕНТА пользовательского интерфейса.

```
propertyDescription
   name = UI_PKEY_Increment
   shellPKey = UI_PKEY_Increment
   formatID = 00000205-7363-696e-8441798acf5aebb7
   propID = 205
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Комментарии

Приращение пользовательского интерфейса в пользовательском интерфейсе \_ \_ используется приложением для запроса того, сколько значение в поле редактирования элемента управления " [Счетчик](windowsribbon-controls-spinner.md) " увеличивается или уменьшается при нажатии кнопки увеличения или уменьшения.
