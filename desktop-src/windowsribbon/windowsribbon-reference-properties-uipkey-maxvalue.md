# <a name="ui_pkey_maxvalue"></a>UI \_ PKEY, \_ MaxValue

Определяет свойство " \_ MaxValue" UI PKEY \_ .

```
propertyDescription
   name = UI_PKEY_MaxValue
   shellPKey = UI_PKEY_MaxValue
   formatID = 00000203-7363-696e-8441798acf5aebb7
   propID = 203
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Remarks

UI \_ PKEY \_ MaxValue используется приложением для запроса максимального значения элемента управления " [Счетчик](windowsribbon-controls-spinner.md) ".
