# <a name="ui_pkey_commandtype"></a>\_COMMANDTYPE UI PKEY \_

Определяет \_ свойство CommandType PKEY пользовательского интерфейса \_ .

```
propertyDescription
   name = UI_PKEY_CommandType
   shellPKey = UI_PKEY_CommandType
   formatID = 00000105-7363-696e-8441798acf5aebb7
   propID = 105
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Remarks

Параметр "CommandType" пользовательского интерфейса \_ PKEY \_ используется приложением для запроса типов команд в элементе управления "Коллекция".

Значение свойства находится в перечислении [**\_ COMMANDTYPE пользовательского интерфейса**](/windows/desktop/api/uiribbon/ne-uiribbon-ui_commandtype) .
