# <a name="ui_pkey_color"></a>Цвет пользовательского интерфейса \_ PKEY \_

Определяет \_ \_ свойство цвета UI PKEY.

```
propertyDescription
   name = UI_PKEY_Color
   shellPKey = UI_PKEY_Color
   formatID = 00000400-7363-696e-8441798acf5aebb7
   propID = 400
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Комментарии

Цвет пользовательского интерфейса \_ PKEY \_ используется приложением для запроса значения цвета элемента управления [**дропдовнколорпиккер**](windowsribbon-element-dropdowncolorpicker.md) .

Значение свойства является [COLORREF](../gdi/colorref.md) значением.

Старший байт значения [COLORREF](../gdi/colorref.md) игнорируется.
