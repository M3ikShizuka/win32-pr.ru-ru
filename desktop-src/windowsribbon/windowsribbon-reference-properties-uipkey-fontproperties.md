# <a name="ui_pkey_fontproperties"></a>UI \_ PKEY \_ фонтпропертиес

Определяет свойство UI \_ PKEY \_ фонтпропертиес.

```
propertyDescription
   name = UI_PKEY_FontProperties
   shellPKey = UI_PKEY_FontProperties
   formatID = 00000300-7363-696e-8441798acf5aebb7
   propID = 300
   typeInfo
      type = IPropertyStore
```

## <a name="remarks"></a>Комментарии

UI \_ PKEY \_ фонтпропертиес — это хранилище свойств, используемое приложением для запроса всех свойств шрифта.
