# <a name="ui_pkey_enabled"></a>Пользовательский интерфейс \_ PKEY \_ включен

Определяет свойство UI \_ PKEY \_ Enabled.

```
propertyDescription
   name = UI_PKEY_Enabled
   shellPKey = UI_PKEY_Enabled
   formatID = 00000004-7363-696e-8441798acf5aebb7
   propID = 1
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Комментарии

Пользовательский интерфейс " \_ PKEY" \_ используется приложением для запроса того, включен ли элемент управления и может реагировать на взаимодействие с пользователем.

> [!Note]  
> При изменении режима приложения значение не устанавливается автоматически, а видимость команды переключается на 0.
