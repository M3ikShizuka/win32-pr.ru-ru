# <a name="ui_pkey_colormode"></a>UI \_ PKEY \_ колормоде

Определяет свойство UI \_ PKEY \_ колормоде.

```
propertyDescription
   name = UI_PKEY_ColorMode
   shellPKey = UI_PKEY_ColorMode
   formatID = 00000402-7363-696e-8441798acf5aebb7
   propID = 402
   typeInfo
      type = UI_SWATCHCOLORMODE
```

## <a name="remarks"></a>Remarks

Значение свойства находится в перечислении [**\_ Сватчколормоде пользовательского интерфейса**](/windows/desktop/api/Uiribbon/ne-uiribbon-ui_swatchcolormode) .
