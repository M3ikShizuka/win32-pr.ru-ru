# <a name="ui_pkey_globalhighlightcolor"></a>UI \_ PKEY \_ глобалхигхлигхтколор

Определяет свойство UI \_ PKEY \_ глобалхигхлигхтколор.

```
propertyDescription
   name = UI_PKEY_GlobalHighlightColor
   shellPKey = UI_PKEY_GlobalHighlightColor
   formatID = 00002001-7363-696e-8441798acf5aebb7
   propID = 2001
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Комментарии

UI \_ PKEY \_ глобалхигхлигхтколор используется приложением для запроса глобального цвета выделения для настроенной ленты.

**Введено в Windows 8:**   \_ГЛОБАЛХИГХЛИГХТКОЛОР UI PKEY \_ не может быть задан независимо от [UI \_ PKEY \_ глобалбаккграундколор](windowsribbon-reference-properties-uipkey-globalbackgroundcolor.md).
