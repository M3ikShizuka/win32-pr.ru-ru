# <a name="ui_pkey_themecolorscategorylabel"></a>UI \_ PKEY \_ семеколорскатегорилабел

Определяет свойство UI \_ PKEY \_ семеколорскатегорилабел.

```
propertyDescription
   name = UI_PKEY_ThemeColorsCategoryLabel
   shellPKey = UI_PKEY_ThemeColorsCategoryLabel
   formatID = 00000403-7363-696e-8441798acf5aebb7
   propID = 403
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Remarks

UI \_ PKEY \_ семеколорскатегорилабел используется приложением для запроса значения метки, связанной с категорией **цветов темы** [**дропдовнколорпиккер**](windowsribbon-element-dropdowncolorpicker.md).

UI \_ PKEY \_ семеколорскатегорилабел допустим только для [**дропдовнколорпиккер**](windowsribbon-element-dropdowncolorpicker.md) , где `ThemeColors` указывается как значение атрибута **колортемплате** (это единственный шаблон, содержащий помеченные категории).

На следующем снимке экрана показан `ThemeColors` [**дропдовнколорпиккер**](windowsribbon-element-dropdowncolorpicker.md).

![снимок экрана элемента дропдовнколорпиккер с атрибутом колортемплате, для которого задано значение семеколорс.](images/markup/colortemplate.themedcolors.1.png)
