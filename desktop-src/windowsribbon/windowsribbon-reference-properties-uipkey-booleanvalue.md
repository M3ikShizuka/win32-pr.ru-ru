# <a name="ui_pkey_booleanvalue"></a>UI \_ PKEY \_ булеанвалуе

Определяет свойство UI \_ PKEY \_ булеанвалуе.

```
propertyDescription
   name = UI_PKEY_BooleanValue
   shellPKey = UI_PKEY_BooleanValue
   formatID = 00000200-7363-696e-8441798acf5aebb7
   propID = 200
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Remarks

UI \_ PKEY \_ булеанвалуе используется приложением для запроса переключения состояния [флажка](windowsribbon-controls-checkbox.md), выключателя и элемента управления кнопки [**сплитбуттонгаллери**](windowsribbon-element-splitbuttongallery.md). [](windowsribbon-controls-togglebutton.md)
