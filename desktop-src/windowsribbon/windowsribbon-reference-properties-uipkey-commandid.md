# <a name="ui_pkey_commandid"></a>UI \_ PKEY \_ CommandId

Определяет свойство UI \_ PKEY \_ CommandId.

```
propertyDescription
   name = UI_PKEY_CommandId
   shellPKey = UI_PKEY_CommandId
   formatID = 00000100-7363-696e-8441798acf5aebb7
   propID = 100
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Комментарии

UI \_ PKEY \_ CommandId используется приложением для запроса идентификаторов команд в элементе управления галереи.

Значение свойства является ИДЕНТИФИКАТОРом команды.
