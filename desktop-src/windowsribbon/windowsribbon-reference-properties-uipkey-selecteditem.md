# <a name="ui_pkey_selecteditem"></a>UI \_ PKEY \_ SelectedItem

Определяет свойство UI \_ PKEY \_ SelectedItem.

```
propertyDescription
   name = UI_PKEY_SelectedItem
   shellPKey = UI_PKEY_SelectedItem
   formatID = 00000104-7363-696e-8441798acf5aebb7
   propID = 104
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Remarks

UI \_ PKEY \_ SelectedItem используется приложением для запроса выбранного элемента в элементе управления галереи.

Значение свойства — это индекс выбранного элемента в коллекции.
