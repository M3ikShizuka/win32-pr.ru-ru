# <a name="ui_pkey_formatstring"></a>Язык интерфейса пользователя (FormatString) UI \_ \_

Определяет \_ \_ свойство свойства FormatString пользовательского интерфейса.

```
propertyDescription
   name = UI_PKEY_FormatString
   shellPKey = UI_PKEY_FormatString
   formatID = 00000207-7363-696e-8441798acf5aebb7
   propID = 207
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Remarks

Язык интерфейса пользователя \_ \_ (FormatString) UI используется приложением для запроса формата (или единицы измерения) для значения элемента управления " [Счетчик](windowsribbon-controls-spinner.md) ".
