# <a name="ui_pkey_standardcolorstooltips"></a>UI \_ PKEY \_ стандардколорстултипс

Определяет свойство UI \_ PKEY \_ стандардколорстултипс.

```
propertyDescription
   name = UI_PKEY_StandardColorsTooltips
   shellPKey = UI_PKEY_StandardColorsTooltips
   formatID = 00000412-7363-696e-8441798acf5aebb7
   propID = 412
   typeInfo
      type = VT_VECTOR | VT_LPWSTR
```

## <a name="remarks"></a>Remarks

UI \_ PKEY \_ стандардколорстултипс используется приложением для запроса всплывающих подсказок палитры цветов [**дропдовнколорпиккер**](windowsribbon-element-dropdowncolorpicker.md).

Значение свойства представляет собой массив строковых значений.

Каждое строковое значение соответствует подсказке для цветовой палитры в [**дропдовнколорпиккер**](windowsribbon-element-dropdowncolorpicker.md) независимо от значения, указанного для атрибута **колортемплате** .
