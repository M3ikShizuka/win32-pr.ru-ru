# <a name="ui_pkey_morecolorslabel"></a>UI \_ PKEY \_ мореколорслабел

Определяет свойство UI \_ PKEY \_ мореколорслабел.

```
propertyDescription
   name = UI_PKEY_MoreColorsLabel
   shellPKey = UI_PKEY_MoreColorsLabel
   formatID = 00000408-7363-696e-8441798acf5aebb7
   propID = 408
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Remarks

UI \_ PKEY \_ мореколорслабел используется приложением для запроса значения метки, связанной с кнопкой " **другие цвета** " объекта [**дропдовнколорпиккер**](windowsribbon-element-dropdowncolorpicker.md).

\_МОРЕКОЛОРСЛАБЕЛ UI PKEY \_ допустим только для [**дропдовнколорпиккер**](windowsribbon-element-dropdowncolorpicker.md) , где или указывается в `ThemeColors` `StandardColors` качестве значения атрибута **колортемплате** .
