# <a name="ui_pkey_globalbackgroundcolor"></a>UI \_ PKEY \_ глобалбаккграундколор

Определяет свойство UI \_ PKEY \_ глобалбаккграундколор.

```
propertyDescription
   name = UI_PKEY_GlobalBackgroundColor
   shellPKey = UI_PKEY_GlobalBackgroundColor
   formatID = 00002000-7363-696e-8441798acf5aebb7
   propID = 2000
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Remarks

UI \_ PKEY \_ глобалбаккграундколор используется приложением для запроса глобального цвета фона для настроенной ленты.

**введено в Windows 8:** [ui \_ pkey \_ глобалхигхлигхтколор](windowsribbon-reference-properties-uipkey-globalhighlightcolor.md) нельзя задать независимо от ui \_ pkey \_ глобалбаккграундколор.
