# <a name="ui_pkey_representativestring"></a>UI \_ PKEY \_ репресентативестринг

Определяет свойство UI \_ PKEY \_ репресентативестринг.

```
propertyDescription
   name = UI_PKEY_RepresentativeString
   shellPKey = UI_PKEY_RepresentativeString
   formatID = 00000208-7363-696e-8441798acf5aebb7
   propID = 208
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Комментарии

UI \_ PKEY \_ репресентативестринг используется приложением для запроса длины поля редактирования для элемента управления " [Счетчик](windowsribbon-controls-spinner.md) ".

> [!Note]  
> Значение UI \_ PKEY \_ репресентативестринг используется платформой для вычисления ширины элемента управления " [Счетчик](windowsribbon-controls-spinner.md) " и никогда не отображается.
