# <a name="ui_pkey_nocolorlabel"></a>UI \_ PKEY \_ ноколорлабел

Определяет свойство UI \_ PKEY \_ ноколорлабел.

```
propertyDescription
   name = UI_PKEY_NoColorLabel
   shellPKey = UI_PKEY_NoColorLabel
   formatID = 00000407-7363-696e-8441798acf5aebb7
   propID = 407
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Комментарии

UI \_ PKEY \_ ноколорлабел используется приложением для запроса значения метки, связанной с кнопкой " **Нет цвета** " в [**дропдовнколорпиккер**](windowsribbon-element-dropdowncolorpicker.md).

Значение свойства — это строка, представляющая метку, связанную с кнопкой **без цвета** [**дропдовнколорпиккер**](windowsribbon-element-dropdowncolorpicker.md).

\_НОКОЛОРЛАБЕЛ UI PKEY \_ является допустимым для всех значений атрибутов **колортемплате** .
