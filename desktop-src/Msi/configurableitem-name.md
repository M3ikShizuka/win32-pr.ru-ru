# <a name="configurableitemname-property"></a>ConfigurableItem.Name, свойство

Свойство **Name** объекта [**конфигураблеитем**](configurableitem-object.md) возвращает значение из столбца Name таблицы модулеконфигуратион.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = ConfigurableItem.Name
```



## <a name="property-value"></a>Значение свойства

## <a name="c"></a>C++

См. раздел [**Получение \_ имени**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_name) функции.
