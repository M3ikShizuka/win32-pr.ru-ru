# <a name="position-control-attribute"></a>Атрибут элемента управления положением

Этот атрибут определяет расположение элемента управления в диалоговом окне. Ниже приведены четыре поля атрибута: левое, верхнее, ширина и высота. Введите ширину элемента управления, высоту и координаты левого угла элемента управления в столбцах ширина, высота, X и Y [таблицы управления](control-table.md). Используйте [единицы установщика](installer-units.md) для единиц длины и расстояния.

## <a name="valid-controls"></a>Допустимые элементы управления

Все активные элементы управления.

## <a name="associated-bit-flags"></a>Связанные битовые флаги

Этот атрибут не использует битовые флаги.

## <a name="remarks"></a>Remarks

См. раздел [атрибуты элементов управления](control-attributes.md) и элемент управления, которые необходимо создать в элементах [управления](controls.md).

Единица установщика равна одной двенадцатой высоте размера шрифта с размером в 10 пунктов MS Sans Serif.
