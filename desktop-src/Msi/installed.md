# <a name="installed-property"></a>Установленное свойство

**Установленное** свойство задается только в том случае, если продукт установлен для каждого компьютера или для текущего пользователя. Это свойство не задано, если продукт установлен для другого пользователя.

**Установленное** свойство может использоваться в условных выражениях для определения того, установлен ли продукт для каждого компьютера или для текущего пользователя. Например, свойство может использоваться в условном столбце таблицы последовательностей или для различения первой установки и установки обслуживания.

Чтобы определить, установлен ли продукт для другого пользователя, проверьте свойство [**продуктстате**](productstate.md) .

Значение свойства [**ProductVersion**](productversion.md) — это версия продукта в строковом формате.
