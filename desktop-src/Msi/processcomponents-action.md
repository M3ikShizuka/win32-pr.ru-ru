# <a name="processcomponents-action"></a>Действие Процесскомпонентс

Действие Процесскомпонентс регистрирует и отменяет регистрацию компонентов, их путей их ключей и клиентов компонентов. Действие Процесскомпонентс запрашивает ключевой столбец [таблицы Component](component-table.md) для определения кэйпасс. Эта регистрация используется [**мсижеткомпонентпас**](/windows/desktop/api/Msi/nf-msi-msigetcomponentpatha) для возврата пути к компоненту для клиента продукта.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие Процесскомпонентс должно следовать после действия [инсталлинитиализе](installinitialize-action.md) .

## <a name="actiondata-messages"></a>Сообщения Актиондата

Для каждого регистрируемого компонента.



| Поле | Описание данных действия      |
|-------|---------------------------------|
| \[1\] | ProductId                       |
| \[2\] | ComponentId                     |
| \[3\] | Путь к ключу для компонента. |



 

Для каждого компонента выполняется отмена регистрации.



| Поле | Описание данных действия |
|-------|----------------------------|
| \[1\] | ProductId                  |
| \[2\] | ComponentId                |
