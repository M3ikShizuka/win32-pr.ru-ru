# <a name="windows-installer-features"></a>установщик Windows функции

Функция — это часть общей функциональности приложения, которую пользователь распознает и может принимать решение об установке независимо. Например, функция может быть средством проверки орфографии, тезаурусом или набором картинок. Иерархические отношения родительской и дочерней функций обычно существуют таким образом, что если дочерняя функция установлена, родительский компонент также устанавливается автоматически.
