# <a name="installerpatchinfo-property"></a>Свойство Installer. Патчинфо

Свойство **патчинфо** только для чтения объекта [**установщика**](installer-object.md) возвращает сведения об исправлении.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Installer.PatchInfo
```



## <a name="property-value"></a>Значение свойства
