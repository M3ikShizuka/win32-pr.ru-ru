# <a name="propertyname-control-attribute"></a>PropertyName, атрибут элемента управления

Это имя свойства, связанного с этим элементом управления. Если [непрямое](indirect-control-attribute.md) значение бита не задано, элемент управления отображает или изменяет значение свойства с таким именем. Этот атрибут задается в столбце свойств [таблицы Control](control-table.md). Этот атрибут возвращает значение NULL для элементов управления инерт.

## <a name="valid-controls"></a>Допустимые элементы управления

Все элементы управления.

## <a name="associated-bit-flags"></a>Связанные битовые флаги

Этот атрибут не использует битовые флаги.

## <a name="remarks"></a>Комментарии

См. раздел [атрибуты элементов управления](control-attributes.md) и элемент управления, которые необходимо создать в элементах [управления](controls.md).
