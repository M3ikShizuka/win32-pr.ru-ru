# <a name="msipatcholdassemblyfile-table"></a>Таблица Мсипатчолдассемблифиле

Таблица Мсипатчолдассемблифиле связывает файл в [таблице File](file-table.md) с именем сборки в [таблице мсипатчолдассемблинаме](msipatcholdassemblyname-table.md). Несколько старых имен сборок могут быть связаны с одним файлом.

Таблица Мсипатчолдассемблифиле содержит следующие столбцы.



| Столбец     | Type                         | Ключ | Допускает значения NULL |
|------------|------------------------------|-----|----------|
| File\_     | [Идентификатор](identifier.md) | Д   | Нет        |
| Сборка\_ | [Идентификатор](identifier.md) | Д   | Нет        |



 

## <a name="columns"></a>Столбцы

<dl> <dt>

<span id="File_"></span><span id="file_"></span><span id="FILE_"></span>File\_
</dt> <dd>

Внешний ключ к [таблице файлов](file-table.md) , указывающий сборку для исправления. Этот столбец является частью первичного ключа.

</dd> <dt>

<span id="Assembly_"></span><span id="assembly_"></span><span id="ASSEMBLY_"></span>Сборок\_
</dt> <dd>

Внешний ключ к [таблице мсипатчолдассемблинаме](msipatcholdassemblyname-table.md) , который определяет одно из старых имен сборок для сборки. Этот столбец является частью первичного ключа.

</dd> </dl>

## <a name="remarks"></a>Remarks

Windows При установке исправлений сборок, установленных в глобальный кэш сборок, установщик использует таблицу Мсипатчолдассемблифиле и таблицу [мсипатчолдассемблинаме](msipatcholdassemblyname-table.md) . При освобождении более новой версии сборки будет изменено строгое имя сборки. Эти две таблицы вместе указывают старое имя сборки для обновленной сборки. Это позволяет установщику использовать старое имя сборки для поиска исходного файла в глобальном кэше сборок и применения двоичного исправления. Без этих сведений установщику может потребоваться доступ к исходному источнику установки для исправления сборки, установленной в глобальном кэше сборок.

Таблица Мсипатчолдассемблифиле и [Таблица мсипатчолдассемблинаме](msipatcholdassemblyname-table.md) не создаются автоматически [патчвиз](patchwiz-dll.md). Пакет обновления, указанный в [таблице упградедимажес](upgradedimages-table-patchwiz-dll-.md) , должен содержать эти таблицы, чтобы исправление имело эти данные.
