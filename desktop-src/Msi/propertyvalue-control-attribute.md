# <a name="propertyvalue-control-attribute"></a>Атрибут элемента управления PropertyValue

Этот атрибут является текущим значением свойства, отображаемого или изменяемого этим элементом управления. Если [непрямое](indirect-control-attribute.md) значение бита не задано, это значение PropertyName. Если задан бит непрямого атрибута, это значение равно Индиректпропертинаме. Если атрибут изменяется, элемент управления отображает новое значение в виде строки или целого числа в зависимости от [целочисленного](integer-control-attribute.md) атрибута элемента управления.

## <a name="valid-controls"></a>Допустимые элементы управления

Все активные элементы управления.

## <a name="associated-bit-flags"></a>Связанные битовые флаги

Этот атрибут не использует битовые флаги.

## <a name="remarks"></a>Комментарии

См. раздел [атрибуты элементов управления](control-attributes.md) и элемент управления, которые необходимо создать в элементах [управления](controls.md).
