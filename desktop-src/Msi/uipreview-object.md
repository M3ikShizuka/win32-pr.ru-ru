# <a name="uipreview-object"></a>Объект Уипревиев

Объект **уипревиев** используется для просмотра диалоговых окон пользовательского интерфейса и объявлений во время разработки. Этот объект создается методом [**енаблеуипревиев**](database-enableuipreview.md) объекта [**Database**](database-object.md) .

## <a name="members"></a>Элементы

Объект **уипревиев** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **уипревиев** содержит эти методы.



| Метод                                           | Описание                                                                                             |
|:-------------------------------------------------|:--------------------------------------------------------------------------------------------------------|
| [**виевбиллбоард**](uipreview-viewbillboard.md) | Отображает созданное объявление, используя элемент управления ведущего приложения в отображаемом в данный момент диалоговом окне.<br/> |
| [**виевдиалог**](uipreview-viewdialog.md)       | Отображает диалоговое окно созданного пользовательского интерфейса, сохраненное в текущей базе данных.<br/>                           |



 

### <a name="properties"></a>Свойства

Объект **уипревиев** имеет следующие свойства.



| Свойство                                          | Тип доступа           | Описание                                                                                                                                                                       |
|:--------------------------------------------------|:----------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Свойство**](uipreview-property.md)<br/> | Чтение/запись<br/> | Представляет строковое значение именованного свойства установщика или значение типа системной переменной среды для текущего процесса, если предваряется знаком процента (%).<br/> |
