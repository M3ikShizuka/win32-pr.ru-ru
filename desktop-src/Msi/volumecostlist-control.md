# <a name="volumecostlist-control"></a>Элемент управления Волумекостлист

Элемент управления Волумекостлист предоставляет сведения о затратах, связанных с выбором на различных томах. Элемент управления показывает все тома, участвующие в текущей установке, а также все тома, имеющие типы, указанные в битах атрибута. Если необходимое место на диске превышает объем, доступный на каком-либо диске, том выделяется в таблице. При щелчке заголовка столбца выполняется сортировка томов в соответствии с выбранным столбцом.

Следующие строки используются для заголовков столбцов в элементе управления и должны быть указаны в [таблице уитекст](uitext-table.md):

-   **волумекоставаилабле**
-   **волумекостдифференце**
-   **волумекострекуиред**
-   **волумекостсизе**
-   **волумекостволуме**

Для форматирования размера файла используются следующие четыре локализованные строки:

-   **Байт**
-   **Статья базы знаний**
-   **МБАЙТА**
-   **ГБ**

Авторы могут задать ширину столбцов в элементе управления Волумекостлист, добавив ширину столбцов после любых характеристик шрифта. Ширина столбцов указывается как последовательность положительных целых чисел, заключенных в фигурные скобки. Пустые фигурные скобки или {0} скрытие столбца. Отрицательное число или строка, которая не может быть преобразована в положительное целое, является недопустимой шириной столбца. При обнаружении недопустимой ширины столбца в ряде оставшиеся столбцы скрываются. Можно указать не более пяти значений ширины столбцов.

Содержимое текстового поля [таблицы элемента управления](control-table.md) никогда не отображается элементом управления волумекостлист. Вместо этого поле определяет стиль текста, отображаемого элементом управления, и содержит описание элемента управления, используемого программами проверки экрана. Чтобы задать шрифт и начертание текстовой строки, перед строкой отображаемых символов укажите { \\ Style} или {&Style}. Где Style — это идентификатор, указанный в столбце система создала шрифт [таблицы система создала шрифт](textstyle-table.md). Если ни один из этих элементов отсутствует, но свойство [**дефаултуифонт**](defaultuifont.md) определено как допустимый стиль текста, будет использоваться этот шрифт. Сведения, приведенные ниже, считываются с помощью служебных программ проверки экрана в качестве описания элемента управления. См. раздел [Специальные возможности](accessibility.md).

Обратите внимание, что установщик не обновляет содержимое Волумекостконтрол, когда пользователь вводит другой путь в [элемент управления паседит](pathedit-control.md), [диалоговое окно обзора](browse-dialog.md), [элемент управления директорилист](directorylist-control.md)или [директорикомбо](directorycombo-control.md) , поскольку отсутствует свойство, связанное с элементом управления волумекостлист.
