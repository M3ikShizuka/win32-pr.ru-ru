# <a name="language"></a>Язык

Тип данных Language — это текстовая строка, содержащая один или несколько допустимых числовых идентификаторов языка. При наличии двух или более идентификаторов языков их необходимо разделять запятыми.

Тип данных Language — это 16-разрядное значение, которое представляет собой сочетание числовых идентификаторов первичного и языкового языка. Первичный LANGID состоит из битов от 0 до 9, а идентификатор подязыка — от 10 до 15. Список числовых идентификаторов основного и вспомогательного языков см. в разделе [константы и строки идентификатора языка](../intl/language-identifier-constants-and-strings.md) .

Для основных идентификаторов языка значение Range 0x200 to 0x3ff определяется пользователем. Диапазон 0x000 to 0x1ff зарезервирован для использования системой. Для идентификаторов языков, диапазон 0x20 — 0x3F, определяемый пользователем. Диапазон 0x00 до 0x1F зарезервирован для использования системой.
