# <a name="viewcolumninfo-property"></a>Свойство View. ColumnInfo

Свойство **ColumnInfo** объекта [**View**](view-object.md) возвращает объект [**Record**](record-object.md) , содержащий запрошенные сведения о каждом столбце результирующего набора. Могут быть запрошены либо имена столбцов, либо определения столбцов. Константы, предоставленные в списке выбора, не имеют имен.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = View.ColumnInfo
```



## <a name="property-value"></a>Значение свойства

Необходимые сведения для каждого столбца.



| Имя параметра                                                                                                                                                                                                                                                          | Значение                                                                |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------|
| <span id="msiColumnInfoNames"></span><span id="msicolumninfonames"></span><span id="MSICOLUMNINFONAMES"></span><dl> <dt>**мсиколумнинфонамес**</dt> <dt>0</dt> </dl> | Возвращает имена всех неконстантных столбцов в результирующем наборе.<br/> |
| <span id="msiColumnInfoTypes"></span><span id="msicolumninfotypes"></span><span id="MSICOLUMNINFOTYPES"></span><dl> <dt>**мсиколумнинфотипес**</dt> <dt>1</dt> </dl> | Возвращает типы всех неконстантных столбцов в результирующем наборе.<br/> |



 

## <a name="remarks"></a>Комментарии

Описания столбцов, возвращаемые свойством **ColumnInfo** , имеют формат, описанный в разделе [Формат определения столбца](column-definition-format.md). Каждый столбец описывается строкой в соответствующем поле записи. Строка определения состоит из одной буквы, представляющей тип данных, а затем ширины столбца (в символах, если применимо или в других байтах). Нулевая ширина означает неограниченную ширину (длинные текстовые поля и потоки). Прописная буква означает, что в столбце разрешены значения NULL.
