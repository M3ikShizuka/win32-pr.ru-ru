# <a name="transparent-control-attribute"></a>Прозрачный атрибут элемента управления

Если прозрачный контрольный бит установлен для текстового элемента управления, элемент управления прозрачно отображается в фоновом режиме с помощью элемента управления, где нет символов. Если этот бит не установлен, элемент управления "текст" является непрозрачным.

## <a name="valid-controls"></a>Допустимые элементы управления

[Text](text-control.md)

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                              |
|---------|-------------|---------------------------------------|
| 65536   | 0x00010000  | **мсидбконтролаттрибутестранспарент** |
