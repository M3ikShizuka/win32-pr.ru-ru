# <a name="_rowstate-column"></a>\_RowState, столбец

Зарезервированное имя столбца \_ RowState представляет непостоянное состояние, связанное с каждой строкой таблицы в таблице базы данных установщика. Псевдо-столбец имеет тип [Integer](integer.md), а значение состоит из набора битовых флагов. Все биты доступны для чтения, но можно задать только UserInfo и временные биты. Эти данные доступны только при условии, что таблицы заблокированы или используются (то есть пока существует представление, содержащее таблицы). В следующей таблице показаны атрибуты, которые может иметь строка.



| Имя           | Значение | Значение                                                        |
|----------------|-------|----------------------------------------------------------------|
| ираусеринфо    | 0x01  | Атрибут предназначен для внешнего использования. Это значение можно обновить.  |
| иратемпорари   | 0x02  | Строка не является постоянной. Это значение можно обновить.          |
| ирамодифиед    | 0x04  | Строка была обновлена.                                        |
| ираинсертед    | 0x08  | Строка была вставлена.                                       |
| ирамержефаилед | 0x0F  | Предпринята попытка выполнить слияние с неидентичными неключевыми данными. |



 

Биты с 6 по 8 зарезервированы.
