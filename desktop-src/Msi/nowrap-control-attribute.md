# <a name="nowrap-control-attribute"></a>Атрибут управления "без переноса"

Если этот бит установлен, текст в элементе управления отображается в одной строке. Если текст выходит за пределы полей элемента управления, он усекается, а в конце вставляется многоточие (...), чтобы указать усечение. Если этот бит не задан, текст переносится.

## <a name="valid-controls"></a>Допустимые элементы управления

[Text](text-control.md)

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                         |
|---------|-------------|----------------------------------|
| 262144  | 0x00040000  | **мсидбконтролаттрибутесноврап** |



 

## <a name="remarks"></a>Remarks

Чтобы задать этот атрибут для элемента управления, включите бит перетекания в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

См. раздел [Управление атрибутами](control-attributes.md) и [элементами управления](controls.md).
