# <a name="arphelptelephone-property"></a>АРФЕЛПТЕЛЕФОНЕ, свойство

Свойство **арфелптелефоне** — номера телефонов технической поддержки. Это значение отображается в приложении "обслуживание продукта".

Windows установщик 4,0 или установщик Windows 4,5 на Windows Server 2008 или Windows Vista.
