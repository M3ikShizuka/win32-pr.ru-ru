# <a name="windows-installer-and-logo-requirements"></a>Требования к установщик Windows и логотипам

Логотип сертифицирован для Microsoft Windows определяет продукты, которые были проверены за счет независимого тестирования в соответствии со спецификацией приложения для Windows. Эта спецификация была разработана корпорацией Майкрософт совместно с клиентами и другими разработчиками для создания надежных и управляемых приложений. Поставщики программного обеспечения, которые соответствуют спецификации для логотипа сертифицировано для Microsoft Windows, а затем могут лицензировать логотип для использования на упаковке, рекламе, материалах и других маркетинговых материалах.

Дополнительные сведения о сертификации Windows Vista, Windows XP и Windows Server 2008 для приложений см. в [Microsoft Partner Network]( https://partner.microsoft.com).

Дополнительные сведения о сертификации Windows Server 2003 и Windows 2000 для приложений см. в разделе [https://msdn.microsoft.com/certification](https://www.microsoft.com/windowsserver2003/partners/isvs/cfw.mspx) .

Если вы создаете пакет установки для своего приложения, вы можете воспользоваться возможностями установщик Windows Майкрософт, чтобы удовлетворить некоторые требования к сертификации. В следующей таблице приведены ссылки на сведения, относящиеся к установщик Windows документации, относящейся к некоторым требованиям к сертификации. Обратите внимание, что не все требования к сертификации, обсуждаемые в спецификации приложения, зависят от установщик Windows.
