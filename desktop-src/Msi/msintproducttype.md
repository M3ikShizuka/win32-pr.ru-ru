# <a name="msintproducttype-property"></a>Мсинтпродукттипе, свойство

установщик задает свойство **мсинтпродукттипе** для Windows NT, Windows 2000 и более поздних операционных систем. это свойство указывает тип продукта Windows.

для операционных систем Windows 2000 и более поздних версий установщик устанавливает следующие значения. Обратите внимание, что значения равны значению поля **впродукттипе** структуры [**осверсионинфоекс**](/windows/win32/api/winnt/ns-winnt-osversioninfoexa) .



| Значение | Значение                                  |
|-------|------------------------------------------|
| 1     | Windows 2000 Professional и более поздних версий      |
| 2     | контроллер домена Windows 2000 и более поздней версии |
| 3     | Windows 2000 Server и более поздние версии            |



 

для операционных систем, предшествующих Windows 2000, установщик устанавливает следующие значения.



| Значение | Значение                      |
|-------|------------------------------|
| 1     | Windows NT Рабочая       |
| 2     | Windows NT контроллер домена |
| 3     | Windows NT Сервером            |
