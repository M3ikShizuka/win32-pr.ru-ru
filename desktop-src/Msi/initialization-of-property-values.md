# <a name="initialization-of-property-values"></a>Инициализация значений свойств

Рекомендуется, чтобы все свойства, используемые при установке, были указаны в [таблице свойств](property-table.md) с начальным значением. Установщик задает эти значения для свойств при запуске установки. Свойства, для которых пустое значение является допустимым значением, и свойства, встроенные в установщик, не требуют инициализации.
