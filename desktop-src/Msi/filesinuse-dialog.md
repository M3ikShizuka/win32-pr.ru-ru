# <a name="filesinuse-dialog"></a>Диалоговое окно Филесинусе

Диалоговое окно Филесинусе предупреждает пользователя о том, что в настоящий момент выполняются файлы, которые должны быть перезаписаны или удалены при установке. Это дает пользователю возможность завершить работу этих процессов и избежать перезагрузки компьютера, чтобы завершить замену или удаление этих файлов. См. раздел [перезагрузки системы](system-reboots.md).

Это диалоговое окно будет создано по мере необходимости в [действии инсталлвалидате](installvalidate-action.md).
