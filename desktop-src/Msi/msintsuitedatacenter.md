# <a name="msintsuitedatacenter-property"></a>Мсинтсуитедатацентер, свойство

в Windows 2000 и более поздних операционных системах установщик устанавливает для свойства **мсинтсуитедатацентер** значение 1, если установлен Windows 2000 datacenter Server. Установщик задает для этого свойства значение 1, только если в \_ \_ структуре [**осверсионинфоекс**](/windows/win32/api/winnt/ns-winnt-osversioninfoexa) установлен флаг центра обработки данных ver Suite. В противном случае установщик не устанавливает это свойство.
