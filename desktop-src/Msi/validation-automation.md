# <a name="validation-automation"></a>Автоматизация проверки

Evalcom2.dll можно использовать для реализации операций проверки для пакетов установки и модулей слияния.

В следующих разделах этой документации содержатся сведения об использовании установщик Windows проверки в средствах разработки.

-   [Использование Evalcom2](using-evalcom2.md)
-   [Интерфейсы EvalCom2](evalcom2-interfaces.md)

Рекомендуется устанавливать Evalcom2.dll с помощью установщик Windows. Пакет Orca.msi, поставляемый в составе пакета средств разработки программного обеспечения Microsoft Windows (SDK) для Windows XP с пакетом обновления 2 (SP2) или более поздней версии, Evalcom2.dll. Идентификатор компонента компонента, содержащего COM-интерфейс, — {53315473-B8D6-470A-9951 – D9AE5C11FC91}.

Сведения о получении Windows SDK см. в разделе [Windows SDK Components for установщик Windows Developers](platform-sdk-components-for-windows-installer-developers.md).
