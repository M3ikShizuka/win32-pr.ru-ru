# <a name="featureinfotitle-property"></a>Феатуреинфо. Title, свойство

Свойство **Title** является свойством, предназначенным только для чтения и возвращающим заголовок компонента в столбце Title [таблицы Feature](feature-table.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = FeatureInfo.Title
```
