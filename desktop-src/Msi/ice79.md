# <a name="ice79"></a>ICE79

ICE79 проверяет ссылки на компоненты и компоненты, введенные в поля базы данных с помощью типа данных [Condition](condition.md) .

## <a name="result"></a>Результат

ICE79 отправляет два предупреждения.



| Предупреждение ICE79                                                                      | Описание                                                      |
|------------------------------------------------------------------------------------|------------------------------------------------------------------|
| В базе данных отсутствует \_ Таблица проверки. Не удалось полностью проверить имена свойств. | В базе данных отсутствует [ \_ Таблица проверки](-validation-table.md). |
| Ошибка при получении значений из столбца \[ 2 \] в таблице \[ 1 \] . Пропуск столбца.         | Ошибка при извлечении значения.                                          |



 

ICE79 отправляет две ошибки.



| ICE79, ошибка                                                          | Описание                               |
|----------------------------------------------------------------------|-------------------------------------------|
| Компонент "% ls", указанный в столбце "% s". % s "строки% s является недопустимым. | Обнаружена недопустимая ссылка на компонент. |
| Компонент "% ls", указанный в столбце "% s". % s "строки% s является недопустимым.   | Обнаружена недопустимая ссылка на компонент    |



 

## <a name="example"></a>Пример

ICE79 сообщает о следующих ошибках в примере:

``` syntax
Component 'NoSuchComponent' referenced in column 
'InstallExecuteSequence'.'Condition' of row Custom2 is invalid.
Feature 'NoSuchFeature' referenced in column 
'InstallExecuteSequence'.'Condition' of row Custom1 is invalid.
```

В этом примере Носучкомпонент отсутствует в [таблице Component](component-table.md) , а носучфеатуре отсутствует в [таблице Feature](feature-table.md).

[Таблица инсталлексекутесекуенце](installexecutesequence-table.md) (частичная)



| Действие  | Условие                                |
|---------|------------------------------------------|
| Custom1 | ТЕСТАКТИОН = 1046 и &Носучфеатуре>2  |
| Custom2 | ТЕСТАКТИОН = 146 и $NoSuchComponent>2 |



 

Чтобы устранить эти ошибки, введите допустимые записи для Носучфеатуре и Носучкомпонент в таблицах компонентов и компонентов.
