# <a name="ice22"></a>ICE22

ICE22 использует [таблицу феатурекомпонентс](featurecomponents-table.md) для проверки сопоставления компонентов и компонентов, упоминаемых в [таблице публишкомпонент](publishcomponent-table.md).

## <a name="result"></a>Результат

ICE22 отправляет сообщение об ошибке, если компоненты и компоненты сопоставлены неправильно в [таблице публишкомпонент](publishcomponent-table.md).

## <a name="example"></a>Пример

В следующем примере ICE22 отправляет ошибку, которая {00000003-0004-0000-0000-624474732465} не имеет правильного сопоставления для \_ столбцов компонента и компонента \_ .

[Таблица публишкомпонент](publishcomponent-table.md) (частичная)



| ComponentId                            | Функция\_ | Компонент\_ |
|----------------------------------------|-----------|-------------|
| {00000002-0003-0000-0000-624474736554} | Feat1     | Comp1       |
| {00000003-0004-0000-0000-624474732465} | Feat1     | Comp2       |



 

[Таблица феатурекомпонентс](featurecomponents-table.md) (частичная)



| Функция\_ | Компонент\_ |
|-----------|-------------|
| Feat1     | Comp1       |
