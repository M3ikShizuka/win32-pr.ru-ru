# <a name="msipatcholdassemblyname-table"></a>Таблица Мсипатчолдассемблинаме

В таблице Мсипатчолдассемблинаме указывается старое имя сборки.

Таблица Мсипатчолдассемблинаме содержит следующие столбцы.



| Столбец   | Type                         | Ключ | Допускает значения NULL |
|----------|------------------------------|-----|----------|
| Сборка | [Идентификатор](identifier.md) | Да   | Нет        |
| Имя     | [Text](text.md)             | Да   | Нет        |
| Значение    | [Text](text.md)             | Нет   | Нет        |



 

## <a name="columns"></a>Столбцы

<dl> <dt>

<span id="Assembly"></span><span id="assembly"></span><span id="ASSEMBLY"></span>Сборок
</dt> <dd>

Уникальный идентификатор для старого имени сборки. Этот ключ используется в качестве сопоставления между этой и [таблицей мсипатчолдассемблифиле](msipatcholdassemblyfile-table.md).

</dd> <dt>

<span id="Name"></span><span id="name"></span><span id="NAME"></span>Безымян

<span id="Value"></span><span id="value"></span><span id="VALUE"></span>Значений

</dd> </dl>

## <a name="remarks"></a>Remarks

Windows При установке исправлений сборок, установленных в глобальный кэш сборок, установщик использует [таблицу мсипатчолдассемблифиле](msipatcholdassemblyfile-table.md) и таблицу мсипатчолдассемблинаме. При освобождении более новой версии сборки будет изменено строгое имя сборки. Эти две таблицы вместе указывают старое имя сборки для обновленной сборки. Это позволяет установщику использовать старое имя сборки для поиска исходного файла в глобальном кэше сборок и применения двоичного исправления. Без этих сведений установщику может потребоваться доступ к исходному источнику установки для исправления сборки, установленной в глобальном кэше сборок.

[Таблица мсипатчолдассемблифиле](msipatcholdassemblyfile-table.md) и таблица мсипатчолдассемблинаме не создаются автоматически [патчвиз](patchwiz-dll.md). Пакет обновления, указанный в [таблице упградедимажес](upgradedimages-table-patchwiz-dll-.md) , должен содержать эти таблицы, чтобы исправление имело эти данные.
