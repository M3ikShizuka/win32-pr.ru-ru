# <a name="installer-database-reference"></a>Ссылка на базу данных установщика

Чтобы добавить функции установщика в приложение, можно использовать таблицы установщик Windows базы данных и функции базы данных установщика. В подразделах этого раздела описывается каждая из функций базы данных и таблиц базы данных.

Список всех таблиц базы данных установки см. в разделе [таблицы базы данных](database-tables.md).

Список функций, используемых для работы с данными в базах данных установки, см. в разделе [функции базы данных](database-functions.md).
