# <a name="custom-actions"></a>Особые действия

установщик Windows предоставляет множество встроенных действий для выполнения процесса установки. Список этих действий см. в [справочнике по стандартным действиям](standard-actions-reference.md).

Стандартные действия достаточно для выполнения установки в большинстве случаев. Однако существуют ситуации, когда разработчик пакета установки находит необходимость в написании настраиваемого действия. Например:

-   Необходимо запустить исполняемый файл во время установки, который устанавливается на компьютере пользователя или устанавливается вместе с приложением.
-   Необходимо вызывать специальные функции во время установки, определенные в библиотеке динамической компоновки (DLL).
-   вы хотите использовать функции, написанные на языках разработки Microsoft Visual Basic scripting Edition или microsoft JScript литеральный текст скрипта во время установки.
-   Необходимо отложить выполнение некоторых действий до момента выполнения скрипта установки.
-   Необходимо добавить сведения о времени и ходе выполнения в элемент управления ProgressBar и элемент управления Text TimeRemaining.

В следующих разделах описываются пользовательские действия и способы их включения в пакет установки.
