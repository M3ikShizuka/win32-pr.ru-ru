# <a name="msirestartmanagercontrol-property"></a>МСИРЕСТАРТМАНАЖЕРКОНТРОЛ, свойство

свойство **мсирестартманажерконтрол** указывает, использует ли пакет установщик Windows функцию [диалогового окна](filesinuse-dialog.md) [диспетчер перезапуска](../rstmgr/restart-manager-portal.md) или филесинусе.

## <a name="value"></a>Значение



| Значение                                                                                        | Значение                                                                                                                                                                                                                                                                                                                                                                |
|----------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>0</dt> </dl>                 | Это значение по умолчанию, если свойство не задано. Windows установщик всегда пытается использовать [диспетчер перезапуска](../rstmgr/restart-manager-portal.md) в Windows Vista.<br/>                                                                                                                                                                                                       |
| <dl> <dt>Включен</dt> </dl>         | Отключает взаимодействие пакета с [диспетчером перезапуска](../rstmgr/restart-manager-portal.md). Windows Установщик использует [диалоговое окно филесинусе](filesinuse-dialog.md). <br/>                                                                                                                                                                                                      |
| <dl> <dt>"Дисаблешутдовн"</dt> </dl> | Windows Установщик использует [диалоговое окно филесинусе](filesinuse-dialog.md). этот параметр отключает попытки [диспетчера](../rstmgr/restart-manager-portal.md) перезапуска по устранению перезапусков при установке пакета установщик Windows, который не был создан для использования диспетчера перезапуска. Установщик по-прежнему использует диспетчер перезапуска для обнаружения файлов, используемых приложениями. <br/> |



 

## <a name="remarks"></a>Remarks

Свойство **мсирестартманажерконтрол** игнорируется, если [Диспетчер перезапуска](../rstmgr/restart-manager-portal.md) недоступен или отключен.

Значение этого свойства можно изменить с помощью преобразований настройки или обновлений. Изменение значения этого свойства с настраиваемых действий не оказывает никакого влияния.
