# <a name="sessionlanguage-property"></a>Свойство Session. Language

Свойство **Language** объекта [**Session**](session-object.md) является свойством, предназначенным только для чтения и представляющим числовой идентификатор языка, используемый текущим сеансом установки.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Session.Language
```



## <a name="property-value"></a>Значение свойства
