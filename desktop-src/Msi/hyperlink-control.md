# <a name="hyperlink-control"></a>Элемент управления HyperLink

Элемент управления HyperLink отображает ссылку HTML на адрес, который открывается в браузере по умолчанию для данного компьютера. Ссылки не поддерживаются для протоколов, отличных от HTML.

**[установщик Windows 4,5 или более ранней версии](not-supported-in-windows-installer-4-5.md):** Не поддерживается. этот элемент управления доступен, начиная с установщик Windows 5,0.

Текстовое значение элемента управления HyperLink использует <a> тег привязки и значение атрибута href для указания URL-адреса и отображаемого текста ссылки.

``` syntax
[Blue Yonder Airlines](https://www.blueyonderairlines.com)
```

## <a name="remarks"></a>Remarks

Этот элемент управления можно создать из \_ класса Link WC с помощью функции [**CreateWindowEx**](/windows/win32/api/winuser/nf-winuser-createwindowexa) . У него есть \_ стили WS Child, WS \_ TABSTOP и WS \_ .

Не размещайте [элементы управления](text-control.md) прозрачного текста поверх цветных растровых изображений. Текст может не отображаться, если пользователь изменяет цветовую схему отображения. Например, текст может стать невидимым, если пользователь устанавливает параметр высокой контрастности в целях специальных возможностей.

Если текст в элементе управления длиннее, чем ширина элемента управления, то текст переносится или усекается в зависимости от того, достаточно ли высоты для размещения текста в оболочке.
