# <a name="removal-of-assemblies-from-the-global-assembly-cache"></a>Удаление сборок из глобального кэша сборок

Установщик Windows определяет, разрешено ли удаление сборки среды CLR на основе списка клиентов, независимо от сборки. Установщик Windows сохраняет один бит ПИН-кода для представления клиентов сборки установщик Windows. Установщик закрепляет сборку для первого клиента установщик Windows и открепляет его при удалении последнего установщик Windows клиента. Сборка поддерживает бит ПИН-кода для каждого клиента сборки.

Установщик Windows напрямую не несет ответственности за физическое удаление сборок среды CLR с компьютера. Установщик отменяет закрепление сборки при удалении последнего установщик Windows клиента. Если установщик Windows является последним клиентом сборки, среда CLR предоставляет возможность принудительной выполнения синхронной очистки сборки. Процесс очистки является атомарным, и на этом этапе не выполняется "откат". Все незакрепленные сборки среды CLR должны быть выполнены после того, как пользователь сможет отменить установку или удаление.
