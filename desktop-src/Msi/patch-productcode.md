# <a name="patchproductcode-property"></a>Свойство patch. ProductCode

Свойство **ProductCode** объекта [**Patch**](patch-object.md) возвращает идентификатор [**ProductCode**](productcode.md) идентификатора продукта.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Patch.ProductCode
```



## <a name="property-value"></a>Значение свойства
