# <a name="sequence-action"></a>Действие последовательности

Действие SEQUENCE сортирует и выполняет в последовательности действия в таблице.

Свойство [**Sequence**](sequence.md) определяет таблицу, используемую этим действием. Таблица должна иметь ту же схему, что и [Таблица инсталлексекутесекуенце](installexecutesequence-table.md). Таблица должна содержать действие, условие и столбец Sequence.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Ограничения последовательности отсутствуют.

## <a name="actiondata-messages"></a>Сообщения Актиондата

Нет сообщений Актиондата.
