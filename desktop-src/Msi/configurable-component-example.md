# <a name="configurable-component-example"></a>Пример настраиваемого компонента

В этом примере следующие таблицы Модулеконфигуратион и Модулесубститутион позволяют потребителю модуля независимо настраивать атрибут Password для элемента управления Edit, атрибут CHECKSUM для файла и сжатый атрибут для одного и того же файла.

[Таблица Модулесубститутион](modulesubstitution-table.md)



| Таблица   | Строка           | Столбец     | Значение                        |
|---------|---------------|------------|------------------------------|
| Control | Dialog1; Edit1 | Атрибуты | \[= Пароль\]                |
| Файл    | Файл1         | Атрибуты | \[= CHECKSUM \] \[ = сжатый\] |



 

[Таблица Модулеконфигуратион](moduleconfiguration-table.md)



| Имя       | Формат   | Тип | контекстдата                              | DefaultValue | Атрибуты | DisplayName | Описание |
|------------|----------|------|------------------------------------------|--------------|------------|-------------|-------------|
| Пароль   | Битами |      | 2097152; True = 2097152; False = 0             | 0            | 0          |             |             |
| Контрольная сумма   | Битами |      | 1024; CHECKSUM = 1024; Без контрольной суммы = 0         | 0            | 0          |             |             |
| Compressed | Битами |      | 24576; Сжатый = 16384; Без сжатия = 8192 | 8192         | 0          |             |             |
