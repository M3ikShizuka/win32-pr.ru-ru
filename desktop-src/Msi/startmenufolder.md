# <a name="startmenufolder-property"></a>Programmenufolder, свойство

Установщик задает для свойства **programmenufolder** полный путь к папке меню "Пуск". По умолчанию это свойство имеет значение Папка для текущего пользователя. Если существует профиль "все пользователи" и задано свойство [**ALLUSERS**](allusers.md) , то для этого свойства задается папка в профиле "все пользователи".
