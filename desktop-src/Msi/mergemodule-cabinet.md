# <a name="mergemodulecabinet"></a>Установочного модуля. cab

Каждый файл, доставленный модулем слияния, должен храниться в CAB-файле, внедренном в виде потока в структурированном хранилище модуля слияния. В стандартном модуле слияния имя этого ящика всегда равно: установочного модуля. cab. Дополнительные сведения см. в разделе [Создание CAB-файлов установочного модуля. cab](generating-mergemodule-cabinet-cabinet-files.md).
