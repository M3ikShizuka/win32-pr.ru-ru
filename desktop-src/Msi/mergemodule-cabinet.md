# <a name="mergemodulecabinet"></a>MergeModule.CABinet

Каждый файл, доставленный модулем слияния, должен храниться в CAB-файле, внедренном в виде потока в структурированном хранилище модуля слияния. В стандартном модуле слияния имя этого CAB-файла всегда равно: MergeModule.CABinet. Дополнительные сведения см. в разделе [Создание CAB-файлов MergeModule.CABinet](generating-mergemodule-cabinet-cabinet-files.md).
