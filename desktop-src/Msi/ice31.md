# <a name="ice31"></a>ICE31

ICE31 проверяет все стандартные стили шрифтов, используемые в [элементах управления](controls.md) , отображающих текст. Также проверяется, что свойство [**дефаултуифонт**](defaultuifont.md) ссылается на допустимый стиль шрифта.

Элементы управления могут иметь стандартный стиль шрифта, как описано в разделе [Добавление элементов управления и текста](adding-controls-and-text.md). Чтобы задать шрифт и начертание текстовой строки, перед строкой отображаемых символов укажите { \\ Style} или {&*Style*}. Где Style — это идентификатор, указанный в столбце система создала шрифт [таблицы система создала шрифт](textstyle-table.md). Если ни один из этих элементов отсутствует, но свойство [**дефаултуифонт**](defaultuifont.md) определено как допустимый стиль текста, будет использоваться этот шрифт.

ICE31 проверяет текстовый столбец для каждого элемента управления в [таблице управления](control-table.md) , чтобы проверить наличие допустимой записи в [таблице система создала шрифт](textstyle-table.md).

ICE31 игнорирует [элемент управления скроллаблетекст](scrollabletext-control.md).

## <a name="results"></a>Результаты

ICE31 отправляет сообщение об ошибке для неопределенных стилей, слишком длинных имен стилей, отсутствующих таблиц система создала шрифт и тегов Style без закрывающих скобок.

ICE31 отправляет предупреждение, если тег Style находится не в начале строки, или если элемент управления имеет несколько тегов Style.

## <a name="example"></a>Пример

ICE31 отправляет следующие ошибки для приведенного примера:

-   Элемент управления Диалогб. Control1 использует неопределенный система создала шрифт Бадстиле.
-   Элемент управления Диалогб. Control2 использует неопределенный система создала шрифт Бадстиле.
-   Для элемента управления Диалогб. Control6 в текстовом стиле отсутствует закрывающая фигурная скобка.
-   Control Диалогб. Control3 указывает текстовый стиль, который является слишком длинным, чтобы быть допустимым.

ICE31 отправляет следующее предупреждение для приведенного примера:

-   Тег стиля текста в Диалогб. Control4 не оказывает влияния. Вы действительно хотите, чтобы он отображался как текст?
