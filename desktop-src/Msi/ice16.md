# <a name="ice16"></a>ICE16

ICE16 проверяет, что длина значения свойства [**ProductName**](productname.md) в [таблице свойств](property-table.md) не превышает 63 символов. Ограничение на длину раздела реестра для DisplayName не существует.

## <a name="result"></a>Результат

ICE16 отправляет сообщение об ошибке, если значение ProductName, заданное в таблице свойств, длиннее 63 символов.
