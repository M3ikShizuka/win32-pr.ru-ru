# <a name="authoring-the-user-interface-for-password-input"></a>Создание пользовательского интерфейса для ввода пароля

Для каждого пароля, который должен быть задан пользователем, добавьте элемент управления Edit в диалоговое окно, в котором значение пароля сохраняется в свойстве. Этот [элемент управления](edit-control.md) "поле ввода" должен иметь [атрибут управления паролями](password-control-attribute.md). Это указывает, что указанное свойство является паролем и не позволяет установщику записывать свойство в файл журнала.

[Атрибуты](control-attributes.md) элемента управления для элемента управления "поле ввода пароля": Мсидбконтролаттрибутесвисибле, Мсидбконтролаттрибутесенаблед и мсидбконтролаттрибутеспассвординпут (1 + 2 + 2097152). Значения X, Y, ширины, высоты и элемента управления \_ Далее зависят от макета элемента управления в диалоговом окне.
