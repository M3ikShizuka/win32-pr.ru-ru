# <a name="table-names"></a>Имена таблиц

Имена таблиц ограничены 31 символами. Если имя таблицы содержит точку (это разрешено, но не рекомендуется), имя должно быть заключено в знаки ударения \` (значение ASCII 96), например \` filename. ext \` . этот escape-синтаксис предотвращает конфликт между именем таблицы и зарезервированными словами, такими как ключевые слова SQL. дополнительные сведения см. в разделе [синтаксис SQL](sql-syntax.md).
