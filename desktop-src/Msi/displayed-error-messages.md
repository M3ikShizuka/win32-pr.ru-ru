# <a name="displayed-error-messages"></a>Отображаемые сообщения об ошибках

[Функция установщика](installer-function-reference.md) может выводить сообщение об ошибке, возвращая одну из следующих ошибок. Окно ошибки отображается только в том случае, если уровень пользовательского интерфейса находится на уровне Full, Reduced или Basic. Дополнительные сведения см. в разделе [уровни пользовательского интерфейса](user-interface-levels.md).

Функции установщика отображают только сообщения об ошибках, перечисленные в следующей таблице. Для ошибок, не приводимых в этом списке, вызывающая функция отвечает за обработку сообщений об ошибках.



| Код ошибки               | Error                        |
|--------------------------|------------------------------|
| Ошибка при \_ установке \_ приостановки  | Установка была приостановлена.   |
| Ошибка при \_ установке \_ усерексит | Пользователь завершил установку. |
| Ошибка \_ \_ при установке  | Сбой установки.              |
