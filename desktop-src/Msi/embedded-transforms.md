# <a name="embedded-transforms"></a>Встроенные преобразования

Встроенные преобразования хранятся в файле .msi пакета. Это гарантирует, что преобразование всегда доступно для пользователей, если пакет установки доступен. Кроме того, преобразования могут предоставляться пользователям в виде автономных файлов MST.

Чтобы добавить встроенное преобразование в список преобразования, добавьте двоеточие (:) в качестве префикса к имени файла. Поскольку установщик всегда может получить преобразование из хранилища в .msi файле, встроенные преобразования не кэшируются на компьютере пользователя. Встроенные преобразования могут использоваться в сочетании с [защищенными преобразованиями](secured-transforms.md) или [незащищенными](unsecured-transforms.md)преобразованиями. Дополнительные сведения см. в разделе [применение преобразований](applying-transforms.md).
