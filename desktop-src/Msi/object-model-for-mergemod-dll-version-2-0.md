# <a name="object-model-for-mergemoddll-version-20"></a>Объектная модель для Mergemod.dll версии 2,0

Объектная модель для Mergemod.dll версии 2,0 организована следующим образом.

![Объектная модель для mergemod.dll версии 2,0](images/mergobj2.png)

[Объект Merge](merge-object.md) является основным объектом модели. [**Объект GetObjects**](getfiles-object.md) является вторичным объектом. Зависимости представляют собой коллекции [**объектов зависимостей**](dependency-object.md). Ошибки представляют собой коллекции [**объектов ошибок**](error-object.md). Конфигураблеитемс — это коллекции [**объектов конфигураблеитем**](configurableitem-object.md).
