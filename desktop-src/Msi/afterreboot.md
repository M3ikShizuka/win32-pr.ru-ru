# <a name="afterreboot-property"></a>АФТЕРРЕБУТ, свойство

Установщик устанавливает свойство **афтерребут** в значение 1 после перезагрузки, вызванной [действием форцеребут](forcereboot-action.md). Установщик добавляет АФТЕРРЕБУТ = 1 в командную строку, выполняемую сразу после перезагрузки.

## <a name="remarks"></a>Remarks

[Действие форцеребут](forcereboot-action.md) всегда должно использоваться с условным оператором, так что установщик запускает перезагрузку только при необходимости. Если определенный файл был заменен или был установлен конкретный компонент, может потребоваться перезагрузка. Регистр для каждого продукта различается, и для определения необходимости перезагрузки может потребоваться пользовательское действие. Условие в действии Форцеребут обычно использует свойство **афтерребут** .
