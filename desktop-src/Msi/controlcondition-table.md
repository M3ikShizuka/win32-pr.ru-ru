# <a name="controlcondition-table"></a>Таблица Таблица controlcondition

Таблица Таблица controlcondition позволяет автору указать специальные действия, применяемые к элементам управления в зависимости от результата условного оператора. Например, с помощью этой таблицы автор может скрыть элемент управления на основе свойства [**версионнт**](versionnt.md) .

Таблица Таблица controlcondition содержит следующие столбцы.



| Столбец    | Type                         | Ключ | Допускает значения NULL |
|-----------|------------------------------|-----|----------|
| Диалог\_  | [Идентификатор](identifier.md) | Да   | Нет        |
| элементом управления\_ | [Идентификатор](identifier.md) | Да   | Нет        |
| Действие    | [Text](text.md)             | Да   | Нет        |
| Условие | [Condition](condition.md)   | Да   | Нет        |



 

## <a name="columns"></a>Столбцы

<dl> <dt>

<span id="Dialog_"></span><span id="dialog_"></span><span id="DIALOG_"></span>Откроется\_
</dt> <dd>

Внешний ключ к первому столбцу [диалоговой таблицы](dialog-table.md). Объединение этого поля с полем элемента управления \_ определяет уникальный элемент управления.

</dd> <dt>

<span id="Control_"></span><span id="control_"></span><span id="CONTROL_"></span>Элемента\_
</dt> <dd>

Внешний ключ ко второму столбцу [управляющей таблицы](control-table.md). При объединении этого поля поле диалогового окна \_ определяет уникальный элемент управления.

</dd> <dt>

<span id="Action"></span><span id="action"></span><span id="ACTION"></span>Поддержки
</dt> <dd>

Действие, выполняемое над элементом управления. Возможные действия показаны в следующей таблице.



| Значение   | Значение                     |
|---------|-----------------------------|
| Значение по умолчанию | Установите для элемента управления значение по умолчанию. |
| Отключить | Отключить элемент управления.        |
| Включить  | Включение элемента управления.         |
| Скрыть    | Скрыть элемент управления.           |
| Показать    | Отображение элемента управления.        |



 

</dd> <dt>

<span id="Condition"></span><span id="condition"></span><span id="CONDITION"></span>Выполняет
</dt> <dd>

Условный оператор, указывающий, при каких условиях должно запускаться действие. Этот столбец нельзя оставлять пустым. Если эта инструкция не возвращает значение TRUE, действие не выполняется. Если задано значение 1, действие применяется всегда. Дополнительные сведения о синтаксисе условных операторов см. в разделе [Синтаксис условных](conditional-statement-syntax.md)инструкций.

</dd> </dl>

## <a name="remarks"></a>Комментарии

Если требуется скрыть и отключить [элемент управления "кнопка](pushbutton-control.md) " или ["флажок"](checkbox-control.md) на основе условного оператора в поле Условие таблицы таблица controlcondition, следует использовать четыре записи для каждого элемента управления, чтобы отключить его, а также скрыть элемент управления. Элементы управления "Кнопка" или "флажок", которые были скрыты, по-прежнему могут быть доступны с помощью сочетаний клавиш.

Например, следующие записи скрывают и отключают элемент управления в диалоговом окне а при установке продукта. Элемент управления будет видимым и включенным, если продукт не установлен.



| Диалог  | Control  | Действие  | Условие                      |
|---------|----------|---------|--------------------------------|
| Диалоговое окно | Элемент управления | Скрыть    | [**Установлено**](installed.md) |
| Диалоговое окно | Элемент управления | Отключить | Установлено                      |
| Диалоговое окно | Элемент управления | Показать    | НЕ установлено                  |
| Диалоговое окно | Элемент управления | Включить  | НЕ установлено                  |
