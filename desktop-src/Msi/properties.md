Свойства являются глобальными переменными, которые установщик Windows использовать во время установки. В следующих разделах описываются свойства, используемые установщиком.

-   [Сведения о свойствах](about-properties.md)
-   [Использование свойств](using-properties.md)
-   [Справочник по свойствам](property-reference.md)

Термин "свойство" также ссылается на атрибут объекта автоматизации. См. раздел [интерфейс автоматизации](automation-interface.md).
