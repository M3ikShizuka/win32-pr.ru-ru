# <a name="nocompanyname-property"></a>НОКОМПАНИНАМЕ, свойство

Присвойте свойству **нокомпанинаме** значение 1, чтобы запретить автоматическую настройку свойства [**COMPANYNAME**](companyname.md) установщиком. Это свойство используется приложениями, которым необходимо получить название компании при первом запуске.

## <a name="default-value"></a>Значение по умолчанию

Не задано. Установщик автоматически устанавливает свойство [**COMPANYNAME**](companyname.md) , используя значения из реестра.
