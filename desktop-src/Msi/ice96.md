# <a name="ice96"></a>ICE96

ICE96 проверяет, что [действие публишфеатурес](publishfeatures-action.md) и [действие публишпродукт](publishproduct-action.md) указаны в [таблице адвтексекутесекуенце](advtexecutesequence-table.md). Невозможно объявить пакет без этих действий.

## <a name="result"></a>Результат

ICE96 отправляет следующие предупреждения.



| Предупреждение ICE96                                                            | Описание                                                                                                  |
|--------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| Действие Публишфеатурес требуется в таблице Адвтексекутесекуенце. | Невозможно объявить пакет без действия Публишфеатурес, входящего в таблицу Адвтексекутесекуенце. |
| Действие Публишпродукт требуется в таблице Адвтексекутесекуенце.  | Невозможно объявить пакет без действия Публишпродукт, входящего в таблицу Адвтексекутесекуенце.  |
