# <a name="last-printed-summary-property"></a>Свойство "Сводка последнего вывода"

В пакете установки для свойства **Последняя распечатанная сводка** может быть задана дата и время, когда [Административная установка](administrative-installation.md) записывается при создании административного образа. Для неадминистративных установок это свойство совпадает со свойством [**создания сводки времени или даты**](create-time-date-summary.md) .

В преобразовании это свойство должно иметь значение null.

В пакете исправлений это свойство должно иметь значение null.
