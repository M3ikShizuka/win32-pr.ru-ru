# <a name="upgradedimages-table-patchwizdll"></a>Таблица Упградедимажес (Patchwiz.dll)

Таблица Упградедимажес содержит сведения об обновленных образах продукта. Обновленный образ должен представлять собой полностью несжатый образ установки последней версии продукта, например административный образ или образ установки без сжатия с компакт-диска. пакет обновления установщик Windows обновляет целевой образ в обновленном образе. Таблица Упградедимажес является обязательной в базе данных создания исправлений (файл. PCP) и используется [уикреатепатчпаккажеекс](uicreatepatchpackageex--patchwiz-dll-.md).

Таблица Упградедимажес, содержащая хотя бы одну запись, необходима в каждой базе данных создания исправлений (файл. PCP). Эта таблица используется [уикреатепатчпаккажеекс](uicreatepatchpackageex--patchwiz-dll-.md).

Таблица Упградедимажес содержит следующие столбцы.



| Столбец       | Type | Ключ | Допускает значения NULL |
|--------------|------|-----|----------|
| Обновлено     | текст | Да   | Нет        |
| мсипас      | текст |     | Нет        |
| патчмсипас | текст |     | Да        |
| симболпасс  | текст |     | Да        |
| Семейство       | текст |     | Нет        |



 

## <a name="columns"></a>Столбцы

<dl> <dt>

<span id="Upgraded"></span><span id="upgraded"></span><span id="UPGRADED"></span>Обновит
</dt> <dd>

Обновленное поле — это произвольный идентификатор для подключения целевых образов к обновленному образу продукта.

</dd> <dt>

<span id="MsiPath"></span><span id="msipath"></span><span id="MSIPATH"></span>мсипас
</dt> <dd>

В этом поле указывается полный путь (включая имя файла) к расположению файла .msi для обновленного образа. Это расположение исходных файлов для обновленного образа.

</dd> <dt>

<span id="PatchMsiPath"></span><span id="patchmsipath"></span><span id="PATCHMSIPATH"></span>патчмсипас
</dt> <dd>

Необязательный Патчмсипас указывает на измененную копию обновленной базы данных установки, которая содержит дополнительную разработку, относящуюся к процессу установки исправления. Например, в свойстве [**Patch**](patch.md) были внесены дополнительные диалоговые окна или настраиваемые действия.

</dd> <dt>

<span id="SymbolPaths"></span><span id="symbolpaths"></span><span id="SYMBOLPATHS"></span>симболпасс
</dt> <dd>

Разделенный точками с запятой список папок для поиска файлов символов, которые могут использоваться для оптимизации создания двоичного исправления. Обратите внимание, что поиск в подкаталогах папок, указанных в этом поле, не выполняется. Оптимизированное двоичное обновление может быть небольшим. Visual C++ должны быть установлены на компьютере, создающем исправление, и использоваться для создания файлов символов. Это поле является необязательным, и установщик создает двоичное исправление, даже если файлы символов не указаны или если файлы символов становятся недоступными для Patchwiz.dll.

</dd> <dt>

<span id="Family"></span><span id="family"></span><span id="FAMILY"></span>Семьи
</dt> <dd>

Внешний ключ в [таблицу имажефамилиес](imagefamilies-table-patchwiz-dll-.md). Каждый обновленный образ должен принадлежать только одному семейству.

</dd> </dl>
