# <a name="searchorder"></a>сеарчордер

Установка этой [системной политики](system-policy.md) для отдельных пользователей определяет порядок, в котором установщик выполняет поиск трех типов источников. Ниже приведены типы источников.

"n" — сеть

"m" — носитель (компакт-диск или DVD-диск)

"u" — универсальный указатель ресурсов (URL-адрес)

Например, для поиска сетевых источников в первую очередь, источники мультимедиа Second и URL-адреса последними устанавливают для этой политики значение "НМУ". Чтобы пропустить Поиск конкретного типа источника, оставьте соответствующую букву из значения.

Если Сеарчордер не задан, то по умолчанию используется порядок поиска Network, Media, а затем URL-адрес.

## <a name="registry-key"></a>Ключ реестра

**HKey \_ Текущие \_ политики пользовательского** \\ **программного обеспечения** \\  \\  \\  \\ **установщик** Microsoft Windows

## <a name="data-type"></a>Тип данных

**REG \_ SZ**
