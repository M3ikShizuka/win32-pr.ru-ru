# <a name="record-object"></a>Объект Record

Объект **Record** — это контейнер для хранения и передачи переменного количества значений. Поля в записи индексируются с числовым индексом и могут содержать строки, целые числа, объекты и значения NULL. Поля, превышающие размер выделенной записи, обрабатываются как непостоянное значение null. Поле с номером 0 зарезервировано.

## <a name="members"></a>Элементы

Объект **Record** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **Record** содержит эти методы.



| Метод                                  | Описание                                                                                          |
|:----------------------------------------|:-----------------------------------------------------------------------------------------------------|
| [**клеардата**](record-cleardata.md)   | Очищает данные во всех полях, присвоив им значение null.<br/>                                      |
| [**форматтекст**](record-formattext.md) | Форматирует поля в соответствии с шаблоном в поле 0.<br/>                                      |
| [**реадстреам**](record-readstream.md) | Считывает указанное число байтов из поля записи, содержащего потоковые данные.<br/>                |
| [**сетстреам**](record-setstream.md)   | Копирует содержимое указанного файла в указанное поле записи в виде потоковых данных.<br/> |



 

### <a name="properties"></a>Свойства

Объект **Record** имеет эти свойства.



| Свойство                                             | Тип доступа           | Описание                                                                                   |
|:-----------------------------------------------------|:----------------------|:----------------------------------------------------------------------------------------------|
| [**DataSize**](record-datasize.md)<br/>       |                       | Возвращает размер данных для указанного поля.<br/>                             |
| [**FieldCount**](record-fieldcount.md)<br/>   |                       | Возвращает число полей в записи.<br/>                                        |
| [**IntegerData**](record-integerdata.md)<br/> | Чтение/запись<br/> | Передает 32-разрядные целочисленные данные в указанное поле в записи или из него.<br/> |
| [**IsNull**](record-isnull.md)<br/>           |                       | Возвращает значение true, если указанное поле имеет значение null, и false, если поле содержит данные.<br/>  |
| [**StringData**](record-stringdata.md)<br/>   | Чтение/запись<br/> | Передает строковые данные в указанное поле в записи или из него.<br/>         |
