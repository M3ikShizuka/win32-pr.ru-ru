# <a name="returning-values-from-an-external-user-interface-handler"></a>Возвращение значений из внешнего обработчика пользовательского интерфейса

Внешний обработчик пользовательского интерфейса может возвращать любое количество значений, чтобы установщик Windows в зависимости от типа кнопки, предоставленной в параметре типа сообщения, который установщик передает в обработчик.

Внешний обработчик пользовательского интерфейса может возвращать значения – 1 и 0 в любое время, так как они не связаны с типами кнопок. Возвращаемое значение – 1 указывает, что во внешнем обработчике пользовательского интерфейса произошла внутренняя ошибка. Возвращаемое значение 0 указывает, что внешний обработчик пользовательского интерфейса не обработал сообщение установщика, и установщик должен обработать сообщение.

Для сообщений, не содержащих тип кнопки, например ИНСТАЛЛМЕССАЖЕ \_ актиондата и инсталлмессаже \_ , возврат идканцел отменяет установку. Возврат ИДОК уведомляет установщик о том, что сообщение было обработано внешним обработчиком пользовательского интерфейса.

Оставшиеся возвращаемые значения, как описано ниже, непосредственно связаны с типами кнопок, которые включены в тип сообщений.



| Возвращаемое значение внешнего пользовательского интерфейса | Значение                                                                                                |
|--------------------------|--------------------------------------------------------------------------------------------------------|
| IDOK                     | Пользователь нажал кнопку **ОК** . Сведения о сообщении были понятны.                     |
| IDCANCEL                 | Была нажата кнопка **"Отмена"** . Отмените установку.                                            |
| IDABORT                  | Была нажата кнопка **прервать** . Прервать установку.                                              |
| IDRETRY                  | Была нажата кнопка **"повторить** ". Повторите действие.                                                |
| IDIGNORE                 | Нажата кнопка " **пропустить** ". Игнорировать ошибку и продолжить.                                      |
| IDYES                    | Нажата кнопка **Да** . Ответ голосами подтверждающими, продолжение текущей последовательности событий..   |
| IDNO                     | Была нажата кнопка " **нет** ". Отрицательный ответ — не продолжать текущую последовательность событий. |



 

Например, если внешний обработчик пользовательского интерфейса отправляет сообщение с \_ флагом "МБ абортретригноре сообщений", внешний обработчик пользовательского интерфейса может возвращать одно из следующих значений:

-   – 1 (ошибка во внешнем обработчике пользовательского интерфейса)
-   0 (не предпринимать никаких действий во внешнем обработчике пользовательского интерфейса, разрешить установщик Windows его обработку)
-   ИДАБОРТ (нажата кнопка **прерывания** )
-   ИДРЕТРИ (нажата кнопка **"повторить** ")
-   ИДИГНОРЕ (нажата кнопка "**пропустить** ")
