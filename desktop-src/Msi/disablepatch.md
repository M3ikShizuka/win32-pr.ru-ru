# <a name="disablepatch"></a>дисаблепатч

Если для этой [системной политики](system-policy.md) на уровне компьютера задано значение "1", установщик не будет применять исправления. Эту политику можно использовать для обеспечения безопасности в средах, в которых необходимо ограничить установку исправлений.

## <a name="registry-key"></a>Ключ реестра

**HKey \_ \_** \\ **политики программного обеспечения** локального компьютера \\  \\  \\  \\ **установщик** Microsoft Windows

## <a name="data-type"></a>Тип данных

**REG \_ DWORD**
