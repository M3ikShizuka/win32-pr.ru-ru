# <a name="versionnt64-property"></a>VersionNT64, свойство

Установщик задает для свойства **VersionNT64** номер версии операционной системы, только если система работает на 64-разрядном компьютере. Если операционная система не является 64-разрядной, свойство не определено.

Значение является целым числом: MajorVersion \* 100 + minorversion.

По соображениям совместимости свойство также не определено, если свойство [**сводки шаблона**](template-summary.md) указывает, что пакет предназначен для 32-разрядных систем Intel, а операционная система является 64-разрядной архитектурой, не Intel64 или x64 (например, ARM64).

## <a name="remarks"></a>Комментарии

Проверка условных выражений на 64-разрядную версию Windows осуществляется просто с помощью имени свойства или путем проверки версии с помощью оператора сравнения.
