# <a name="versionnt64-property"></a>VersionNT64, свойство

Установщик задает для свойства **VersionNT64** номер версии операционной системы, только если система работает на 64-разрядном компьютере. Если операционная система не является 64-разрядной, свойство не определено.

Значение является целым числом: MajorVersion \* 100 + minorversion.

по соображениям совместимости свойство также не определено, если свойство [**сводки шаблона**](template-summary.md) указывает, что пакет предназначен для 32-разрядных систем intel (x86), а операционная система не может запустить 64-разрядный код intel (x64), например Windows 10 на ARM (ARM64).

> [!NOTE]
> начиная с Windows 10 сборка 21277, ARM64 сборки в канале разработки Windows Insider Preview программы имеют поддержку для 64-разрядных приложений. Для этих сборок ARM64 свойство VersionNT64 определено для пакетов x86.

## <a name="remarks"></a>Remarks

условные выражения проверяются на 64 бит Windows просто с помощью имени свойства или путем проверки версии с помощью оператора сравнения.
