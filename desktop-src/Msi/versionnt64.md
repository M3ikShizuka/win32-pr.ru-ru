# <a name="versionnt64-property"></a>VersionNT64, свойство

Установщик задает для свойства **VersionNT64** номер версии операционной системы, только если система работает на 64-разрядном компьютере. Если операционная система не является 64-разрядной, свойство не определено.

Значение является целым числом: MajorVersion \* 100 + minorversion.

По соображениям совместимости свойство также не определено, если свойство [**сводки шаблона**](template-summary.md) указывает, что пакет предназначен для 32-разрядных систем Intel (x86), а операционная система не может запустить 64-разрядный код Intel (x64), например Windows 10 в ARM (ARM64).

> [!NOTE]
> Начиная с Windows 10 Build 21277, сборки ARM64 в канале разработки программы предварительной оценки Windows имеют поддержку для 64-разрядных приложений. Для этих сборок ARM64 свойство VersionNT64 определено для пакетов x86.

## <a name="remarks"></a>Remarks

Проверка условных выражений на 64-разрядную версию Windows осуществляется просто с помощью имени свойства или путем проверки версии с помощью оператора сравнения.
