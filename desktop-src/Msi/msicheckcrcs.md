# <a name="msicheckcrcs-property"></a>МСИЧЕКККРКС, свойство

циклическая проверка избыточности (CRC) доступна в установщик Windows версии 2,0 и более поздних. Установщик выполняет контрольную сумму для файлов только в том случае, если задано свойство **мсичекккркс** . Дополнительные сведения см. в разделе [Проверка CRC во время установки](crc-checking-during-an-installation.md).
