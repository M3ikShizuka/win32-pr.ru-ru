# <a name="error-dialog"></a>Диалоговое окно ошибки

Диалоговое окно ошибки представляет собой модальное диалоговое окно, в котором отображается сообщение об ошибке. В каждой установке может существовать более одного диалогового окна ошибки.

Необходимо задать свойство Еррордиалог, которое указывает, какое диалоговое окно следует использовать. Если это свойство не задано или не указывает на допустимое диалоговое окно ошибки, то сообщения об ошибках отображаться не будут. В этом случае ошибка заносится в журнал только с предупреждением об отсутствующем диалоговом окне.

В диалоговом окне ошибки должен быть установлен [бит стиля диалогового окна](error-dialog-style-bit.md) ошибки. Диалоговое окно должно содержать [элемент управления "текст](text-control.md) " с именем ErrorText. Запись для диалогового окна ошибки в [таблице диалоговых окон](dialog-table.md) должна содержать элемент управления ErrorText, введенный в \_ первое поле элемента управления.

Диалоговое окно должно содержать семь [кнопок](pushbutton-control.md). Все эти кнопки указывают таблице ControlEvent событие [EndDialog](enddialog-controlevent.md) в [таблице таблице ControlEvent событие](controlevent-table.md). Каждая кнопка задает один из следующих атрибутов: **еррораборт**, **еррорканцел**, **ерроригноре**, **еррорно**, **ерророк**, **еррорретри**, **еррорес**.

> [!Note]  
> Фокус этих элементов управления не должен быть связан с использованием элемента управления \_ следующий столбец в [таблице управления](control-table.md).

 

Эти кнопки должны размещаться примерно в том же положении в диалоговом окне, поскольку при их создании создается только подмножество этих семи кнопок в зависимости от сообщения. Координата X кнопок изменяется таким образом, чтобы отображаемые кнопки были пробелом. Координаты Y, высота и ширина кнопок не меняются. Так как кнопки располагаются горизонтально, другие элементы управления не могут быть помещены в одну и ту же горизонтальную область диалогового окна.

В диалоговом окне ошибки поля элементов управления \_ по умолчанию и отмены элемента управления \_ в [таблице](dialog-table.md) не учитываются. В \_ первом поле элемента управления для диалогового окна ошибки должен быть указан элемент управления ErrorText.

если в этом диалоговом окне включен [элемент управления значком](icon-control.md) с именем еррорикон, отображаются следующие стандартные значки Windows:

-   Иди \_ Ошибка в ответ на сообщения имтфаталексит.
-   Иди \_ предупреждение в ответ на сообщения имтеррор и имтварнинг.
-   Иди \_ сведения в ответ на сообщения имтаутофдискспаце.

элемент управления еррорикон должен быть создан с помощью [атрибута элемента управления FixedSize](fixedsize-control-attribute.md) , чтобы избежать неправильного определения размера стандартных значков Windows.
