# <a name="version-windows-installer"></a>Версия (установщик Windows)

Тип данных Version — это текстовая строка, содержащая допустимую строку версии. Строка версии имеет формат

``` syntax
xxxxx.xxxxx.xxxxx.xxxxx
```

где *x* — цифра.

Максимальная допустимая строка версии — 65535.65535.65535.65535.

Ниже приведены примеры допустимых строк версии.

-   1
-   1,0
-   1.00
-   10,00
-   1.00.1
-   1.0.1
-   1.00.10
-   1.00.100
-   1.0.1000.0
