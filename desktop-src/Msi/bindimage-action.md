# <a name="bindimage-action"></a>Действие BindImage

Действие BindImage привязывает каждый исполняемый файл или библиотеку DLL, которые должны быть привязаны к импортируемым им библиотекам DLL. Действие BindImage действует для каждого файла в таблице [BindImage](bindimage-table.md) , но только для тех, которые должны быть установлены локально. Установщик рассчитывает виртуальный адрес каждой функции, импортированной из всех библиотек DLL, а затем сохраняет вычисленный виртуальный адрес в [*таблице адресов импорта*](i-gly.md) импортируемого образа (IAT).

действие BindImage внутренне вызывает API Windows **биндимажеекс** .

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие BindImage должно следовать после действия [инсталлфилес](installfiles-action.md) .
