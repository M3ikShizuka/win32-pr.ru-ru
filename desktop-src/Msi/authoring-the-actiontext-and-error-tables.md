# <a name="authoring-the-actiontext-and-error-tables"></a>Создание Актионтекст и таблиц ошибок

Примеры спецификаций включают отправку сообщений Актиондата, когда пользовательское действие создает или удаляет учетную запись пользователя, и сообщает об ошибке, если учетная запись не может быть создана.

Введите следующие записи в таблицу Актионтекст, чтобы указать сведения, используемые сообщениями Актионтекст.

[Таблица Актионтекст](actiontext-table.md)



| Действие            | Описание                                                       | Шаблон                          |
|-------------------|-------------------------------------------------------------------|-----------------------------------|
| процессаккаунтс   | Создание действий для создания учетных записей пользователей на локальном компьютере. | Учетная запись: \[ 1 \] , атрибуты: \[ 2\] |
| креатеаккаунт     | Создание учетной записи пользователя на локальном компьютере.                      | Учетная запись: \[ 1\]                    |
| RemoveAccount     | Удаление учетной записи пользователя с локального компьютера.                    | Учетная запись: \[ 1\]                    |
| роллбаккаккаунт   | Откат создания учетных записей пользователей на локальном компьютере. | Учетная запись: \[ 1\]                    |
| унинсталлаккаунтс | Создание действий для удаления учетных записей пользователей на локальном компьютере. | Учетная запись: \[ 1\]                    |



 

Введите следующие записи в таблицу ошибок, чтобы указать сведения, используемые при создании отчетов об ошибках.

[Таблица ошибок](error-table.md)



| Error | Сообщение                                                                        |
|-------|--------------------------------------------------------------------------------|
| 25001 | Не удалось создать учетную запись пользователя " \[ 2 \] " на локальном компьютере. Код ошибки: \[ 3 \] . |
| 25002 | Учетная запись пользователя " \[ 2 \] " уже существует на локальном компьютере.                      |
| 25003 | Не удалось удалить учетную запись пользователя " \[ 2 \] " на локальном компьютере. Код ошибки: \[ 3 \] . |
| 25004 | Учетная запись пользователя " \[ 2 \] " не существует на локальном компьютере.                      |



 

Продолжайте [Создание таблицы инсталлексекутесекуенце](authoring-the-installexecutesequence-table.md).
