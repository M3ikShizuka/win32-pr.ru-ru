# <a name="product-object"></a>Объект Product

Объект **Product** представляет уникальный экземпляр продукта, который объявлен, установлен или неизвестен.

Экземпляр объекта можно создать с помощью свойства **Product** в виде "Виндовсинсталлер. Installer. Product (*ProductCode* *, с* учетом *контекста*)". Для контекста на компьютер значение "со значением *" должно быть* равно null. *Для указанного* текущего пользователя значение «со значением» может быть равно null, если контекст не относится к компьютеру. Требуются параметры *ProductCode* и *context* .

### <a name="methods"></a>Методы

Объект **Product** содержит эти методы.



| Метод                                                                 | Описание                                                                                                        |
|:-----------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------|
| [**саурцелистаддмедиадиск**](product-sourcelistaddmediadisk.md)       | Добавьте диск в набор зарегистрированных дисков.<br/>                                                              |
| [**саурцелистаддсаурце**](product-sourcelistaddsource.md)             | Добавьте сеть или URL-адрес в исходный список.<br/>                                                         |
| [**саурцелистклеаралл**](product-sourcelistclearall.md)               | Очищает полный исходный список указанного типа источников.<br/>                                       |
| [**саурцелистклеармедиадиск**](product-sourcelistclearmediadisk.md)   | Удалить диск из набора зарегистрированных дисков из списка источников.<br/>                                    |
| [**саурцелистклеарсаурце**](product-sourcelistclearsource.md)         | Удаление сети или источника URL-адреса из списка источников.<br/>                                                    |
| [**саурцелистфорцересолутион**](product-sourcelistforceresolution.md) | Очищает последний использованный источник. Это приводит к тому, что разрешение списка источников будет вычисляться в следующий раз, когда требуется источник.<br/> |
