# <a name="componentinfocomponentcode-property"></a>Компонентинфо. Компоненткоде, свойство

Код компонента рассматриваемого компонента. Это свойство вызывает функцию [**мсижеткомпонентпасекс**](/windows/desktop/api/Msi/nf-msi-msigetcomponentpathexa) .

**[установщик Windows 4,5 или более ранней версии](not-supported-in-windows-installer-4-5.md):** Не поддерживается. это свойство доступно, начиная с установщик Windows 5,0.
