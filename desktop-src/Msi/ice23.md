# <a name="ice23"></a>ICE23

ICE23 проверяет порядок табуляции элементов управления для каждого диалогового окна.

ICE23 проверяет следующее в [диалоговом окне](dialog-table.md) и таблице [элементов управления](control-table.md):

-   Каждая запись в диалоговой таблице указывает элемент управления в \_ первом столбце элемента управления, который существует в диалоговом окне, указанном в столбце диалогового окна.
-   Каждая запись в таблице элементов управления указывает элемент управления в \_ следующем столбце, который находится в том же диалоговом окне, что и элемент управления, указанный в столбце Control, или элемент управления \_ Next содержит значение null.
-   После управления следующими \_ записями элемента управления в таблице элементов управления создается один замкнутый цикл, который возвращается к первоначальному элементу управления. Не каждый элемент управления должен находиться в цикле, но цикл должен проходить через каждый элемент управления, имеющий запись в \_ следующем столбце элемента управления.

## <a name="result"></a>Результат

ICE23 отправляет сообщение об ошибке, если порядок табуляции элементов управления не формирует в диалоговом окне один замкнутый цикл.

## <a name="example"></a>Пример

ICE23 выводит следующие сообщения об ошибках для приведенного примера.

-   Dialog1 не имеет контроля в \_ первую очередь.
-   \_Первым элементом диалогового окна Dialog2 называется несуществующий элемент управления контролкс.
-   Dialog3 имеет незавершенный порядок табуляции в элементе управления Контролб.
-   Dialog4 имеет неправильный порядок табуляции в элементе управления Контролк
-   Dialog5 имеет неправильный порядок табуляции в элементе управления Контролк.
-   Управление \_ следующим элементом управления Dialog6. контролк ссылается на неизвестный элемент управления.

Чтобы устранить эти ошибки, обратите внимание на следующее в приведенных выше таблицах и внесите указанные изменения.

Не каждая строка в таблице диалога содержит элемент управления, указанный в \_ первом столбце элемента управления. Измените \_ первый столбец элемента управления в записи Dialog1 в диалоговой таблице на элемент управления, существующий в Dialog1.

Не каждая строка в диалоговой таблице содержит элемент управления, указанный в \_ первом столбце элемента управления, который существует в диалоговом окне. Измените \_ первый столбец элемента управления Dialog2 на элемент управления, существующий в Dialog2.

После управления следующими \_ записями в таблице элементов управления из элемента управления в элемент управления в каждом случае не создается замкнутый цикл. Измените \_ следующий столбец элемента управления для контролб в Dialog3 на элемент управления.

После управления следующими \_ записями в таблице элементов управления от Control к Control не происходит возврат к первоначальному элементу управления в каждом случае. Измените \_ следующий столбец элемента управления для контролк в Dialog4, чтобы он ссылался на элемент управления.

После управления следующими \_ записями в таблице элементов управления от Control к Control не передается каждый элемент управления в диалоговом окне, имеющий запись в \_ следующем столбце. Измените значение элемента управления \_ следующий столбец для контролк в Dialog5 на управление.

Элемент управления \_ Next не ссылается на допустимый элемент управления, который находится в том же диалоговом окне, что и элемент управления, указанный в столбце Control. Измените \_ следующий столбец элемента управления для контролк в Dialog6, чтобы он ссылался на элемент управления.
