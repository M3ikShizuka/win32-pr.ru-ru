# <a name="clientcomponentcode-property"></a>Свойство Client. Компоненткоде

Код компонента рассматриваемого компонента. Это свойство вызывает функцию [**мсиенумклиентсекс**](/windows/desktop/api/Msi/nf-msi-msienumclientsexa) .

**[Установщик Windows 4,5 или более ранней версии](not-supported-in-windows-installer-4-5.md):** Не поддерживается. Это свойство доступно, начиная с установщик Windows 5,0.
