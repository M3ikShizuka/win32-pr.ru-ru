# <a name="componentinfopath-property"></a>Компонентинфо. Path, свойство

Путь к компоненту. Это свойство вызывает функцию [**мсижеткомпонентпасекс**](/windows/desktop/api/Msi/nf-msi-msigetcomponentpathexa) .

**[установщик Windows 4,5 или более ранней версии](not-supported-in-windows-installer-4-5.md):** Не поддерживается. это свойство доступно, начиная с установщик Windows 5,0.
