# <a name="restrictions-on-property-names"></a>Ограничения по именам свойств

Все имена свойств должны соответствовать этим ограничениям.

-   Имя свойства — это [идентификатор](identifier.md), который представляет собой текстовую строку, которая должна начинаться с буквы или символа подчеркивания. Идентификаторы и имена свойств могут содержать буквы, цифры, символы подчеркивания или точки; но не может начинаться с цифры или точки.
-   Имена [общедоступных свойств](public-properties.md) не могут содержать строчные буквы.
-   Имена [частных свойств](private-properties.md) должны содержать несколько строчных букв.
-   Имена свойств с префиксом% представляют переменные среды системы и пользователя. Они никогда не вносятся в [таблицу свойств](property-table.md). Постоянные параметры переменных среды можно изменить только с помощью [таблицы Environment](environment-table.md).
