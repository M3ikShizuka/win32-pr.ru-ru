# <a name="userexit-dialog"></a>Диалоговое окно Усерексит

Это модальное диалоговое окно отображается в конце установки, которая была завершена пользователем. Обычно это диалоговое окно содержит только некоторый текст и кнопку **ОК** . Имя этого диалогового окна должно появиться в [таблице админуисекуенце](adminuisequence-table.md) и [таблице инсталлуисекуенце](installuisequence-table.md) с параметром-2 в качестве порядкового номера.
