# <a name="ice94"></a>ICE94

ICE94 проверяет [таблицу ярлыков](shortcut-table.md), [таблицу компонентов](feature-table.md)и [таблицу мсиассембли](msiassembly-table.md) и отправляет предупреждение, если есть необъявленные ярлыки, указывающие на файл сборки в глобальном кэше сборок. Если запись в целевом поле в таблице ярлыков не является функцией в таблице Feature, это сочетание не объявляется. Если запись в \_ поле Component Таблицы ярлыков также указана в таблице мсиассембли, ярлык указывает на файл сборки. Если запись в поле файлового \_ приложения в таблице мсиассембли пуста, файл сборки находится в глобальном кэше сборок.

## <a name="result"></a>Результат

ICE94 отправляет следующее предупреждение.



| Предупреждение ICE94                                                                                | Описание                                                                            |
|----------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------|
| Необъявленный ярлык " \[ 2 \] " указывает на файл сборки в глобальном кэше сборок. | Необъявленный ярлык указывает на файл сборки в глобальном кэше сборок. |

ICE94 сообщает о следующей ошибке в примере:

``` syntax
The non-advertised shortcut 'shortcut1' points to an assembly file in the global assembly cache.
```
