# <a name="sign-setupexe-and-mysetupmsi"></a>Подписывание Setup.exe и MySetup.msi

Перед размещением Setup.exe и MySetup.msi на веб-сервере следует подписать файлы цифровым сертификатом и закрытым ключом MyCert. cer и MyCert. PVK с помощью программы [SignTool](../seccrypto/signtool.md) . Дополнительные сведения об использовании средства SignTool см. в [справочнике по инструментам CryptoAPI](../seccrypto/cryptoapi-tools-reference.md) в пакете средств разработки программного обеспечения (SDK) для Microsoft Windows.
