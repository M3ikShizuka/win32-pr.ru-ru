# <a name="ice53"></a>ICE53

ICE53 проверяет наличие записей в таблице реестра, которые записывают данные частного установщика или значения политики в системный реестр.

## <a name="result"></a>Результат

ICE53 отправляет предупреждение, если в таблице реестра указано, что в реестре записывается внутренняя информация или сведения о политике.

## <a name="example"></a>Пример

ICE53 отправляет следующее предупреждение для приведенного примера.

``` syntax
Registry Key 'Registry1' writes Installer internal or policy information.
```

[Таблица реестра](registry-table.md) (частичная)



| Реестр             | Root         | Ключ                                                                                                   |
|----------------------|--------------|-------------------------------------------------------------------------------------------------------|
| Registry1<br/> | 1<br/> | **Программное обеспечение** \\ **Политики** \\ **Microsoft** \\ **Windows** \\ **Установщик** \\ **Дисаблероллбакк**<br/> |



 

Строка таблицы реестра "Registry1" записывает в реестр значение системной политики, влияющее на установку всех пакетов. В зависимости от пакета можно отключить откат только для этого пакета, задав свойство [**дисаблероллбакк**](-disablerollback.md) в [таблице свойств](property-table.md). См. раздел [ROLLBACK Installation](rollback-installation.md).
