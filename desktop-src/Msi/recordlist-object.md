# <a name="recordlist-object"></a>Объект Рекордлист

Объект **рекордлист** представляет собой коллекцию объектов [**Record**](record-object.md) . Перед обращением к его свойствам необходимо убедиться, что объект **рекордлист** существует и не пуст.

## <a name="members"></a>Элементы

Объект **рекордлист** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Элемент Property

Объект **рекордлист** имеет следующие свойства.



| Свойство                                     | Описание                                                          |
|:---------------------------------------------|:---------------------------------------------------------------------|
| [**Расчета**](recordlist-count.md)<br/> | Возвращает количество элементов в объекте **рекордлист** .<br/> |
| [**Элемент**](recordlist-item.md)<br/>   | Возвращает запись в коллекции объектов **рекордлист** .<br/>   |
