# <a name="about-the-installer-database"></a>Сведения о базе данных установщика

база данных установщик Windows состоит из множества взаимосвязанных таблиц, которые вместе образуют реляционную базу данных, необходимую для установки группы приложений. Поскольку база данных является реляционной, таблицы связываются по данным в значениях первичного и внешнего ключей. Это обеспечивает эффективный способ изменения процесса установки и означает, что пользователи могут легко настраивать большое приложение или группу приложений. Таблицы базы данных соответствуют общему макету всей группы приложений, включая:

-   Доступные функции
-   Компоненты
-   Отношение между компонентами и компонентами
-   Необходимые параметры реестра
-   Пользовательский интерфейс для приложения

Чтобы создать базу данных установки, необходимо заполнить таблицы всеми сведениями о приложениях и процессе установки. Ручная разработка всех этих таблиц превращается в большую задачу, даже при установке умеренного размера. Поэтому для создания базы данных установщика доступны некоторые сторонние средства.
