# <a name="msiarpsettingsidentifier-property"></a>МСИАРПСЕТТИНГСИДЕНТИФИЕР, свойство

Свойство **мсиарпсеттингсидентифиер** содержит разделенный точками с запятой список расположений реестра, в которых приложение хранит параметры и предпочтения пользователя. Во время обновления операционной системы программа установки может использовать эти сведения для улучшения работы пользователей, выполняющих миграцию приложений.

Значением свойства **мсиарпсеттингсидентифиер** является литеральный текст, содержащий список расположений реестра, в которых приложение хранит свои параметры. Значение может указывать несколько возможных расположений в реестре. Разделяйте несколько расположений реестра с помощью точки с запятой. Параметры приложения обычно хранятся в кустах реестра **HKCU** или **HKLM** в формате " *\\ \\ версия приложения компании*". В следующем примере показано возможное значение для свойства **мсиарпсеттингсидентифиер** .

*MyCompany \\ аппсуите \\ 1,0; MyCompany \\ App1 \\ 1,0; MyCompany \\ \\ 1,0*

Это свойство является необязательным и может быть задано в таблице [свойств](property-table.md) . Если это свойство отображается в таблице свойств, значение не должно быть равно **null**.
