# <a name="template-summary-property"></a>Свойство сводки шаблона

Для пакета установки свойство сводка по **шаблону** указывает на версии платформы и языка, совместимые с этой базой данных установки. Ниже приведен синтаксис сведений о свойстве **сводки шаблона** для базы данных установки.

\[*свойство платформы* \] ; \[ *идентификатор языка* \] \[ ,*идентификатор языка* \] \[ ,... \] .

Следующие примеры являются допустимыми значениями для свойства **сводки шаблона** :

-   x64; 1033
-   Intel; 1033
-   Intel64; 1033
-   ; 1033
-   Intel; 1033, 2046
-   Intel64; 1033, 2046
-   Intel; 0
-   ARM; 1033, 2046
-   ARM; 0
-   Arm64; 1033, 2046
-   Arm64; 0

Свойство **Сводка шаблона** преобразования указывает версии платформы и языка, совместимые с преобразованием. В файле преобразования может быть указан только один язык. Указанная платформа и язык определяют, можно ли применить преобразование к определенной базе данных. Свойство Platform и свойство Language можно оставить пустыми, если для проверки преобразования не используются ограничения преобразования.

Свойство **Сводка шаблона** пакета исправлений представляет собой разделенный точками с запятой список кодов продуктов, которые могут принять исправление. Если для создания пакета обновления используется [Msimsp.exe](msimsp-exe.md) и [Patchwiz.dll](patchwiz-dll.md) , этот список получается из [таблицы таржетимажес](targetimages-table-patchwiz-dll-.md) файла создания исправления.

Это свойство сводки является обязательным.

## <a name="remarks"></a>Комментарии

Если текущая платформа не соответствует одной из платформ, указанных в свойстве **сводки шаблона** , установщик не обрабатывает пакет.

Если спецификация платформы отсутствует в значении свойства **сводки шаблона** , установщик предполагает, что архитектура Intel.

Если это 64-разрядный пакет установщик Windows выполняется на платформе Intel64 (Itanium), введите Intel64 в свойстве **сводки шаблона** .

Если это 64-разрядный пакет установщик Windows выполняется на платформе x64, введите x64 в свойстве **сводки шаблона** .

Если это 64-разрядный установщик Windowsный пакет, который выполняется на Arm64 платформе, введите Arm64 в свойстве **сводки шаблона** .

Пакет установщик Windows не может быть помечен как поддерживающий как Intel64, так и x64; Например, значение свойства " **Сводка" шаблона** Intel64, x64 является недопустимым.

Пакет установщик Windows не может быть помечен как поддерживающий как 32-разрядную, так и 64-разрядную платформу. Например, значения свойств **сводки шаблона** , такие как Intel, x64 или Intel, Intel64 являются недопустимыми.

Если ввести 0 (ноль) в поле Идентификатор языка свойства **Сводка по шаблону** или оставьте это поле пустым, это означает, что пакет не является нейтральным к языку.

Если это установщик Windows пакет, который выполняется в Windows RT, введите значение ARM в свойстве **Сводка шаблона** .

Модули слияния — это единственные пакеты, которые могут иметь несколько языков. В исходной базе данных установщика можно указать только один язык. Дополнительные сведения см. в разделе [модули слияния с несколькими языками](multiple-language-merge-modules.md).

Платформа Alpha не поддерживается установщик Windows.

**Установщик Windows:** Следующий синтаксис не поддерживается:

\[*свойство Platform* \] \[ , \] \[ ,... \] свойства \[ платформы *идентификатор языка* \] \[ ,*идентификатор языка* \] \[ ,... \] .

Следующие примеры не являются допустимыми значениями для свойства **сводки шаблона** :

-   Альфа, Intel; 1033
-   Intel, Альфа; 1033
-   Альфа; 1033
-   Альфа; 1033, 2046
