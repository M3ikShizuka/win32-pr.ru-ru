# <a name="removefile-table"></a>Таблица Ремовефиле

Таблица Ремовефиле содержит список файлов, которые будут удалены [действием ремовефилес](removefiles-action.md). Установка значения NULL для столбца FileName в этой таблице поддерживает удаление пустых папок.

Таблица Ремовефиле содержит следующие столбцы.



| Столбец      | Type                                     | Ключ | Допускает значения NULL |
|-------------|------------------------------------------|-----|----------|
| филекэй     | [Идентификатор](identifier.md)             | Да   | Нет        |
| Компонент\_ | [Идентификатор](identifier.md)             | Нет   | Нет        |
| FileName    | [вилдкардфиленаме](wildcardfilename.md) | Нет   | Да        |
| дирпроперти | [Идентификатор](identifier.md)             | Нет   | Нет        |
| InstallMode | [Integer](integer.md)                   | Нет   | Нет        |



 

## <a name="columns"></a>Столбцы

<dl> <dt>

<span id="FileKey"></span><span id="filekey"></span><span id="FILEKEY"></span>филекэй
</dt> <dd>

Первичный ключ, используемый для распознавания этой конкретной записи таблицы.

</dd> <dt>

<span id="Component_"></span><span id="component_"></span><span id="COMPONENT_"></span>См\_
</dt> <dd>

Внешний ключ первый столбец [таблицы Component](component-table.md). Это поле ссылается на компонент, который управляет удаляемым файлом.

</dd> <dt>

<span id="FileName"></span><span id="filename"></span><span id="FILENAME"></span>Файлов
</dt> <dd>

Этот столбец содержит Локализуемое имя удаляемого файла. Если этот столбец имеет значение null, то указанная папка будет удалена, если она пуста. Все файлы, соответствующие подстановочному знаку, будут удалены из указанного каталога.

</dd> <dt>

<span id="DirProperty"></span><span id="dirproperty"></span><span id="DIRPROPERTY"></span>дирпроперти
</dt> <dd>

Имя свойства, значение которого предполагается преобразовать в полный путь к папке удаляемого файла. Свойство может быть именем каталога в [таблице Directory](directory-table.md), свойством, заданным [таблицей аппсеарч](appsearch-table.md), или любым другим свойством, представляющим полный путь.

</dd> <dt>

<span id="InstallMode"></span><span id="installmode"></span><span id="INSTALLMODE"></span>InstallMode
</dt> <dd>

Должно иметь одно из следующих значений.



| Константа                                | Шестнадцатеричный | Decimal | Описание                                                                                                   |
|-----------------------------------------|-------------|---------|---------------------------------------------------------------------------------------------------------------|
| **мсидбремовефилеинсталлмодеонинсталл** | 0x001       | 1       | Удаляйте только при установке связанного компонента (Мсиинсталлстателокал или Мсиинсталлстатесаурце). |
| **мсидбремовефилеинсталлмодеонремове**  | 0x002       | 2       | Удаляется только при удалении связанного компонента (Мсиинсталлстатеабсент).                           |
| **мсидбремовефилеинсталлмодеонбос**    | 0x003       | 3       | Удалите один из указанных выше вариантов.                                                                          |



 

</dd> </dl>
