# <a name="dependency-object"></a>Объект зависимости

Объект **зависимости** возвращает модуль, от которого зависит текущий модуль, который еще не был объединен с текущей базой данных установки.

## <a name="members"></a>Элементы

Объект **зависимости** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Объект **зависимости** имеет эти свойства.



| Свойство                                           | Описание                                             |
|:---------------------------------------------------|:--------------------------------------------------------|
| [**Язык**](dependency-language.md)<br/> | Возвращает язык модуля.<br/>           |
| [**Модуль**](dependency-module.md)<br/>     | Возвращает идентификатор модуля требуемого модуля.<br/> |
| [**Версия**](dependency-version.md)<br/>   | Возврат версии требуемого модуля.<br/>   |
