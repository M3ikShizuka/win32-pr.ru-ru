# <a name="wildcardfilename"></a>вилдкардфиленаме

Тип данных Вилдкардфиленаме — это [имя файла](filename.md) , которое может также содержать подстановочные знаки "?" для любого отдельного символа или " \* " для вхождения любого символа в ноль или более.
