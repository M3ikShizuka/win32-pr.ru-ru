# <a name="clientusersid-property"></a>Свойство Client., то есть

Идентификатор безопасности пользователя для компонента. Это свойство вызывает функцию [**мсиенумклиентсекс**](/windows/desktop/api/Msi/nf-msi-msienumclientsexa) .

**[установщик Windows 4,5 или более ранней версии](not-supported-in-windows-installer-4-5.md):** Не поддерживается. это свойство доступно, начиная с установщик Windows 5,0.
