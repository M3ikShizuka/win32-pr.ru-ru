# <a name="remotevolume-control-attribute"></a>Атрибут элемента управления Ремотеволуме

Если этот бит задан, в элементе управления отображаются все тома, вовлеченные в текущую установку, а также все удаленные тома. Если этот бит не задан, элемент управления перечисляет тома в текущей установке.

## <a name="valid-controls"></a>Допустимые элементы управления

[директорикомбо](directorycombo-control.md)

[волумекостлист](volumecostlist-control.md)

[волумеселекткомбо](volumeselectcombo-control.md)

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                               |
|---------|-------------|----------------------------------------|
| 262144  | 0x00040000  | **мсидбконтролаттрибутесремотеволуме** |



 

## <a name="remarks"></a>Комментарии

Чтобы задать этот атрибут для элемента управления, включите бит Ремотеволуме в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

См. раздел [Управление атрибутами](control-attributes.md) и [элементами управления](controls.md).
