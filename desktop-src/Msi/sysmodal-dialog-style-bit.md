# <a name="sysmodal-dialog-style-bit"></a>Бит стиля диалогового окна Сисмодал

Если этот бит стиля задан, то диалоговое окно останавливает все другие приложения, и другие приложения не могут перестать сосредоточиться на этом. Это состояние остается до закрытия диалогового окна Сисмодал.

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                          |
|---------|-------------|-----------------------------------|
| 8       | 0x00000008  | **мсидбдиалогаттрибутессисмодал** |
