# <a name="admintoolsfolder-property"></a>Админтулсфолдер, свойство

Свойство **админтулсфолдер** содержит полный путь к каталогу файловой системы, в котором хранятся средства администрирования. Консоль управления (MMC) сохраняет настраиваемые консоли в этом каталоге и выполняет роуминг с пользователем. Если задано свойство [**ALLUSERS**](allusers.md) , это свойство указывает на каталог файловой системы, содержащий средства администрирования для всех пользователей компьютера.
