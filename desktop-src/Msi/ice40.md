# <a name="ice40"></a>ICE40

ICE40 выполняет различные проверки.

## <a name="result"></a>Результат

ICE40 отправляет предупреждения по следующим адресам:

-   Свойство [**REINSTALLMODE**](reinstallmode.md) было переопределено.
-   [Таблица ремовеинифиле](removeinifile-table.md) содержит запись тега DELETE без значения.
-   В файле .msi отсутствует [Таблица ошибок](error-table.md) , а свойство " [**Сводка по количеству страниц**](page-count-summary.md) " меньше или равно 100. это предупреждение компилятора устарело, так как установщик Windows не требует, чтобы пакет имел таблицу ошибок. Сообщения об ошибках можно получить с помощью Msimsg.dll.
