# <a name="removeshortcuts-action"></a>Действие Ремовешорткутс

Действие Ремовешорткутс управляет удалением объявленного ярлыка, компонент которого выбран для удаления, или необъявленный ярлык, компонент которого выбран для удаления. Дополнительные сведения см. в [таблице ярлыков](shortcut-table.md).

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие Ремовешорткутс должно следовать перед действием [креатешорткутс](createshortcuts-action.md) .

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия      |
|-------|---------------------------------|
| \[1\] | Идентификатор удаленного ярлыка. |
