# <a name="character-count-summary-property"></a>Свойство сводки количества символов

Свойство " **Сводка числа символов** " используется только в преобразованиях. Эта часть информационного потока сводки делится на 2 16-разрядные слова. В верхнем слове содержатся [*Флаги проверки преобразования*](t-gly.md). В нижнем слове содержатся [*флаги состояния ошибки преобразования*](t-gly.md).

Это свойство должно иметь значение NULL в пакете установки или пакете исправлений.
