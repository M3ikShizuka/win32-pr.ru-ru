# <a name="template"></a>Шаблон

Тип данных шаблона — это текстовая строка, которая может содержать свойства, заключенные в квадратные скобки \[ \] . Тип шаблона допускает все форматы [форматированного](formatted.md) типа, а \[ *1* , \] где *1* — число.
