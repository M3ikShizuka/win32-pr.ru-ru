# <a name="msilogging-property"></a>Мсилоггинг, свойство

свойство **мсилоггинг** задает режим ведения журнала по умолчанию для пакета установщик Windows. Если это необязательное свойство имеется в [таблице свойств](property-table.md), установщик создает файл журнала с именем MSI \* . Журналь. Полный путь к файлу журнала предоставляется значением свойства [**мсилогфилелокатион**](msilogfilelocation.md) .

## <a name="value"></a>Значение

Значение этого свойства должно быть строкой следующих символов, определяющих режим ведения журнала по умолчанию.



| Значение                                                                        | Значение                                                                        |
|------------------------------------------------------------------------------|--------------------------------------------------------------------------------|
| <dl> <dt>I</dt> </dl> | Сообщения о состоянии.<br/>                                                    |
| <dl> <dt>w</dt> </dl> | Некритические предупреждения.<br/>                                                  |
| <dl> <dt>e</dt> </dl> | Все сообщения об ошибках.<br/>                                                 |
| <dl> <dt>конкретного</dt> </dl> | Запуск действий.<br/>                                                |
| <dl> <dt>r</dt> </dl> | Записи, относящиеся к действию.<br/>                                            |
| <dl> <dt>u</dt> </dl> | Запросы пользователей.<br/>                                                      |
| <dl> <dt>c</dt> </dl> | Начальные параметры пользовательского интерфейса.<br/>                                              |
| <dl> <dt>m</dt> </dl> | Сведения о нехватке памяти или аварийном завершении.<br/>                            |
| <dl> <dt>o</dt> </dl> | Сообщения о нехватке места на диске.<br/>                                         |
| <dl> <dt>ш</dt> </dl> | Свойства терминала.<br/>                                                |
| <dl> <dt>3,3</dt> </dl> | Подробный вывод.<br/>                                                     |
| <dl> <dt>x</dt> </dl> | Дополнительные отладочные сведения. доступно только на Windows Server 2003.<br/> |
| <dl> <dt>!</dt> </dl> | Запись каждой строки в журнал.<br/>                                         |



 

## <a name="remarks"></a>Remarks

это свойство доступно начиная с установщик Windows 4,0.

Нельзя использовать значения "+" и " \* " параметра/l в значении свойства **мсилоггинг** .

Режим ведения журнала можно задать с помощью политики, параметра командной строки или программно. Это переопределяет режим ведения журнала по умолчанию. дополнительные сведения о всех методах, доступных для настройки режима ведения журнала, см. в разделе " [нормальное ведение журнала](normal-logging.md) " раздела [установщик Windows logging](windows-installer-logging.md) .

Если свойство **мсилоггинг** имеется в [таблице свойств](property-table.md), то режим ведения журнала по умолчанию для пакета можно изменить, изменив значение этого свойства с помощью [преобразования базы данных](database-transforms.md). Режим ведения журнала по умолчанию нельзя изменить с помощью [пакета исправлений](patch-packages.md) (MSP-файла).

Если свойство **мсилоггинг** задано в [таблице свойств](property-table.md), для всех пользователей компьютера можно указать режим ведения журнала по умолчанию, задав политику [дисаблелоггингфромпаккаже](disableloggingfrompackage.md) и политику [ведения журнала](logging.md) . Настройка политики Дисаблелоггингфромпаккаже и политики ведения журнала приведет к переопределению свойства **мсилоггинг** для всех пакетов.
