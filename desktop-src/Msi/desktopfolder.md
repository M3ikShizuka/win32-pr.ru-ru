# <a name="desktopfolder-property"></a>DesktopFolder, свойство

Установщик задает для свойства **DesktopFolder** полный путь к папке рабочего стола текущего пользователя. Если существует профиль "все пользователи" и задано свойство [**ALLUSERS**](allusers.md) , то для этого свойства задается папка в профиле "все пользователи".

Windows установщик 4,0 или установщик Windows 4,5 на Windows Server 2008 или Windows Vista.
