# <a name="recordcleardata-method"></a>Метод Record. Клеардата

Метод **клеардата** объекта [**Record**](record-object.md) очищает данные во всех полях, присвоив им значение null. Освобождаются все объекты, хранящиеся в полях.

## <a name="syntax"></a>Синтаксис


```JScript
Record.ClearData()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
