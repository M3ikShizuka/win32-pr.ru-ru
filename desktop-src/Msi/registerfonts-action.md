# <a name="registerfonts-action"></a>Действие Регистерфонтс

Действие Регистерфонтс регистрирует установленные шрифты в системе. Это действие сопоставляет имя шрифта в столбце Фонттитле [таблицы Font](font-table.md) с путем к установленному файлу шрифта.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие [инсталлфилес](installfiles-action.md) должно быть вызвано до вызова действия регистерфонтс.

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия |
|-------|----------------------------|
| \[1\] | Файл шрифта.                 |



 

## <a name="remarks"></a>Комментарии

Действие Регистерфонтс выполняется, если файл, указанный в \_ столбце file [таблицы Font](font-table.md) , относится к устанавливаемому компоненту.
