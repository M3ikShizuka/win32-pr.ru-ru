# <a name="logging-windows-installer"></a>ведение журнала (установщик Windows)

Эта [Системная политика](system-policy.md) для компьютера используется, только если ведение журнала не включено с помощью параметра командной строки "/l" или [**мсиенаблелог**](/windows/desktop/api/Msi/nf-msi-msienableloga). Если в этом случае политика задана, установщик создает файл журнала в папке% TEMP% с случайным именем: MSI \* . Журналь. Укажите режим ведения журнала, задав в качестве значения политики строку символов. Используйте те же символы, чтобы указать политику режима ведения журнала, используемую [параметром командной строки](command-line-options.md)"/l". Обратите внимание, что для политики нельзя использовать "+" и " \* ".

Режим ведения журнала можно задать с помощью политики, параметра командной строки или программно. дополнительные сведения о всех методах, доступных для настройки режима ведения журнала, см. в разделе " [нормальное ведение журнала](normal-logging.md) " раздела [установщик Windows logging](windows-installer-logging.md) .

Можно предотвратить ввод конфиденциальных сведений, например паролей, в файл журнала и сделать их видимыми. Дополнительные сведения см. в разделе [предотвращение записи конфиденциальной информации в файл журнала](preventing-confidential-information-from-being-written-into-the-log-file.md) .
