# <a name="productinstallproperty-method"></a>Метод Product. Инсталлпроперти

Свойство **инсталлпроперти** является значением свойства для экземпляра данного продукта.

Это свойство вызывает функцию [**мсижетпродуктинфоекс**](/windows/desktop/api/Msi/nf-msi-msigetproductinfoexa) с *ProductCode* *, а* также *контекстом* объекта [**Product**](product-object.md) и запрошенным свойством в качестве параметра.

## <a name="syntax"></a>Синтаксис


```JScript
Product.InstallProperty(
  property
)
```

## <a name="return-value"></a>Возвращаемое значение

Если вызов будет выполнен, свойство будет содержать значение в виде строки.
