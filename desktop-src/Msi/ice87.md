# <a name="ice87"></a>ICE87

ICE87 проверяет, что следующие свойства не были созданы в [таблице свойств](property-table.md). Вместо этого эти свойства следует задавать в командной строке.

-   [**Свойство ADDLOCAL**](addlocal.md)
-   [**REMOVE, свойство**](remove.md)
-   [**АДДСАУРЦЕ, свойство**](addsource.md)
-   [**АДДДЕФАУЛТ, свойство**](adddefault.md)
-   [**Переустановить свойство**](reinstall.md)
-   [**ОБЪЯВЛЕНИЕ свойства**](advertise.md)
-   [**КОМПАДДЛОКАЛ, свойство**](compaddlocal.md)
-   [**КОМПАДДСАУРЦЕ, свойство**](compaddsource.md)
-   [**ФИЛЕАДДЛОКАЛ, свойство**](fileaddlocal.md)
-   [**ФИЛЕАДДСАУРЦЕ, свойство**](fileaddsource.md)
-   [**ФИЛЕАДДДЕФАУЛТ, свойство**](fileadddefault.md)

## <a name="result"></a>Результат

ICE87 отправляет следующее предупреждение.



| Предупреждение ICE87                                                                                                                        | Описание                                                                                                                       |
|--------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| Свойство ' \[ 1 \] ' не должно быть создано в таблице свойств. Это может привести к неправильному удалению продукта | Указанное свойство не должно быть задано в [таблице свойств](property-table.md). Вместо этого задайте свойство в командной строке. |
