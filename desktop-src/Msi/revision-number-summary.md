# <a name="revision-number-summary-property"></a>Свойство сводки номеров редакций

Для пакета установки свойство **Сводка номера редакции** содержит [*код пакета*](p-gly.md) для пакета установщика. Дополнительные сведения о кодах пакетов см. в разделе [коды пакетов](package-codes.md).

Для преобразования в свойстве **Сводка номера редакции** перечислены идентификаторы GUID и версия новых и исходных продуктов, а также идентификатор GUID кода обновления. Список отделяется точкой с запятой следующим образом.

Исходный продукт исходного кода — версия продукта; New-Product код New — версия продукта; Обновление кода "

Для пакета исправлений свойство **Сводка номера редакции** указывает код исправления GUID для исправления. За этим может следовать список идентификаторов GUID кода исправления для устаревших исправлений, которые будут удалены после применения этого исправления. Коды исправлений объединяются без разделителей, разделяющих идентификаторы GUID в списке.

* * Установщик Windows 3,0: * * Если в [таблице мсипатчсекуенце](msipatchsequence-table.md)содержится информация о последовательности, установщик Windows 3,0 использует сведения о последовательности в таблице и игнорирует список устаревших исправлений, включенных в свойство **сводки номеров редакций** . Установщик Windows 3,0 по-прежнему может использовать устаревшие сведения об исправлениях в свойстве **сводки номеров редакций** , если пакет не содержит таблицу мсипатчсекуенце.

* * Установщик Windows 2,0: * * [Таблица мсипатчсекуенце](msipatchsequence-table.md) не поддерживается. Установщик Windows 2,0 по-прежнему может использовать устаревшие сведения об исправлениях в свойстве **сводки номеров редакций** , если пакет не содержит таблицу мсипатчсекуенце.

Это свойство сводки является обязательным.
