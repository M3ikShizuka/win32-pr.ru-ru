# <a name="removeduplicatefiles-action"></a>Действие Ремоведупликатефилес

Действие Ремоведупликатефилес удаляет файлы, установленные действием [дупликатефилес](duplicatefiles-action.md) .

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие Ремоведупликатефилес должно быть выполнено после действия [инсталлвалидате](installvalidate-action.md) и перед действием [инсталлфилес](installfiles-action.md) .

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия                    |
|-------|-----------------------------------------------|
| \[1\] | Идентификатор удаленного файла.                   |
| \[9\] | Идентификатор каталога, содержащего удаленный файл. |



 

## <a name="remarks"></a>Remarks

Чтобы удалить файл, дублирующийся с помощью действия [дупликатефилес](duplicatefiles-action.md) , используя действие ремоведупликатефилес, необходимо выбрать компонент, связанный с записью файла в таблице дупликатефиле, для удаления.

Используйте столбец DestFolder [таблицы дупликатефиле](duplicatefile-table.md) , чтобы указать имя свойства, значение которого определяет целевую папку.
