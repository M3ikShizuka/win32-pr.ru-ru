# <a name="registercomplus-action"></a>Действие Регистеркомплус

Действие Регистеркомплус регистрирует приложения COM+.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие Регистеркомплус должно следовать за [действием инсталлфилес](installfiles-action.md) и [действием унрегистеркомплус](unregistercomplus-action.md).

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия              |
|-------|-----------------------------------------|
| \[1\] | Идентификатор приложения COM+. |
