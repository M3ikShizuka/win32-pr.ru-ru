# <a name="installerproductstate-property-method"></a>Метод свойства Installer. Продуктстате

**Свойство продуктстате** является свойством только для чтения, которое возвращает сведения о состоянии установки для продукта.

## <a name="syntax"></a>Синтаксис


```JScript
Installer.ProductState Property(
  Product
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Продукт* 
</dt> <dd>

Указывает код продукта.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="remarks"></a>Комментарии

Возвращает одно из значений, приведенных в следующей таблице.



| Состояние установки        | Описание                                      |
|---------------------------|--------------------------------------------------|
| мсиинсталлстатеабсент     | Продукт устанавливается для другого пользователя.   |
| мсиинсталлстатедефаулт    | Продукт установлен для текущего пользователя.   |
| мсиинсталлстатеадвертисед | Продукт объявлен, но не установлен.     |
| мсиинсталлстатеинвалидарг | Функции передан недопустимый параметр. |
| мсиинсталлстатеункновн    | Продукт не объявлен и не установлен. |
| мсиинсталлстатебадконфиг  | Данные конфигурации повреждены.               |
