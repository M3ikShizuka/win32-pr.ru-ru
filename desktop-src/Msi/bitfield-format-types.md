# <a name="bitfield-format-types"></a>Типы битового формата

Типы битового формата для настраиваемых данных могут использоваться в полях базы данных Integer. Пользователь должен выбрать значение из указанного подмножества. Это относится только к правилам и не применяется в Mergemod.dll. Атрибут Мсмконфигитемноннуллабле является неявным в этом формате данных и не требует установки. Значение NULL никогда не является допустимым значением для этого типа.

Существует следующий тип формата битов.

[Произвольный тип битового поля](arbitrary-bitfield-type.md)

Настраиваемые элементы типа формата битовых значений используются в целочисленных столбцах, и в целом их можно заменить любым целым числом. Пользователь должен выбрать значение из указанного подмножества. Это относится только к правилам, но не применяется Mergemod.dll. Автор должен подготовить модуль к обработке любого значения.
