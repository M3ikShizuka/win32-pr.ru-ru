# <a name="companyname-property"></a>COMPANYNAME, свойство

Свойство **COMPANYNAME** — это организация или компания пользователя, выполняющего установку.

## <a name="default-value"></a>Значение по умолчанию

Значение по умолчанию может быть размещено в [таблице свойств](property-table.md). Если свойство **COMPANYNAME** не задано, установщик настраивает его автоматически с помощью значений из реестра.

## <a name="remarks"></a>Remarks

Задайте свойство [**нокомпанинаме**](nocompanyname.md) , чтобы отключить автоматический параметр **COMPANYNAME**.
