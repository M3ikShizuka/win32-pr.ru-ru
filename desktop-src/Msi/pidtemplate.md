# <a name="pidtemplate-property"></a>Пидтемплате, свойство

Свойство **пидтемплате** содержит строку, используемую в качестве шаблона для свойства [**PIDKEY**](pidkey.md) . Синтаксис, используемый в шаблоне, см. в разделе Тип элемента управления [MaskedEdit](maskededit-control.md) . [Действие валидатепродуктид](validateproductid-action.md) использует это значение для проверки идентификатора продукта.
