# <a name="selectionpath-controlevent"></a>Селектионпас таблице ControlEvent событие

[Элемент управления селектионтри](selectiontree-control.md) использует событие селектионпас для публикации пути к выделенному элементу. Если элемент выбран для выполнения из источника, то это путь к источнику. Если элемент выбран для отсутствия, то строка является строкой **абсентпас** из [таблицы уитекст](uitext-table.md). Это событие должно быть создано в [таблице таблица eventmapping](eventmapping-table.md).

Для этого таблице ControlEvent событие требуется, чтобы пользовательский интерфейс выполнялся на [*полном уровне пользовательского интерфейса*](f-gly.md) . Это событие не будет работать с [*ограниченным пользовательским*](r-gly.md) интерфейсом или [*базовым интерфейсом*](b-gly.md). Дополнительные сведения см. в разделе [уровни пользовательского интерфейса](user-interface-levels.md).

Это событие может влиять только на элементы управления, находящиеся в том же диалоговом окне, что и элемент управления Селектионтри.

## <a name="published-by"></a>Кем опубликовано

[селектионтри](selectiontree-control.md)

## <a name="argument"></a>Аргумент

Нет.

## <a name="action-on-subscribers"></a>Действие на подписчиках

Нет.

## <a name="typical-use"></a>Типичные случаи использования

Элемент управления " [текст](text-control.md) " в том же модальном диалоговом окне, что и селектионтри, должен подписываться на событие через [таблицу таблица eventmapping](eventmapping-table.md). Элемент управления "текст" отображает путь выделенного элемента.
