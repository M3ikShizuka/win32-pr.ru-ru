# <a name="using-windows-installer-with-wmi"></a>использование установщик Windows с WMI

поставщик установщик Windows позволяет инструментарий управления Windows (WMI) (WMI) получать доступ к информации, собранной из приложений установщик Windows.

[поставщик установщик Windows](/previous-versions/windows/desktop/msiprov/windows-installer-provider) является необязательным Windows компонентом. необязательная установка поставщика установщик Windows обеспечивает обратную совместимость, но не указывает на доступность поставщика в будущем. дополнительные сведения о доступности поставщика установщик Windows см. в разделе [доступность операционной системы компонентов WMI](../wmisdk/operating-system-availability-of-wmi-components.md) в документации по [инструментарий управления Windows (WMI) (WMI)](../wmisdk/wmi-start-page.md) .

текущее описание этого поставщика WMI см. в разделе [поставщик установщик Windows](/previous-versions/windows/desktop/msiprov/windows-installer-provider) в основной документации по [инструментарий управления Windows (WMI) (wmi)](../wmisdk/wmi-start-page.md) . список классов WMI, поддерживаемых установщик Windows, см. в разделе [установленные классы приложений](/previous-versions//aa390887(v=vs.85)).
