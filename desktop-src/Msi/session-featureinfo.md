# <a name="sessionfeatureinfo-method"></a>Session. Феатуреинфо, метод

Метод **феатуреинфо** объекта [**Session**](session-object.md) возвращает объект **феатуреинфо** , содержащий описательные сведения для указанного компонента.

## <a name="syntax"></a>Синтаксис


```JScript
Session.FeatureInfo(
  Feature
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Компонент* 
</dt> <dd>

Определяет интересующую функцию.

</dd> </dl>
