# <a name="creating-application-summary-property"></a>Создание свойства сводки приложения

Свойство « **Создание сводки приложения» передает сведения о** том, какое приложение создало базу данных установщика. В общем случае значение для этого свойства Summary является именем программного обеспечения, которое используется для создания этой базы данных.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Версия<br/> | Windows установщик 5,0 на Windows Server 2012, Windows 8, Windows Server 2008 R2 или Windows 7. Windows установщик 4,0 или установщик Windows 4,5 на Windows Server 2008 или Windows Vista. Windows установщик на Windows Server 2003 или Windows XP<br/> |
