# <a name="summaryinfo-object"></a>Объект Суммаринфо

Объект **суммаринфо** используется для чтения, создания и обновления свойств документа из [сводного информационного потока](summary-information-stream.md) объекта хранилища.

## <a name="members"></a>Элементы

Объект **суммаринфо** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **суммаринфо** содержит эти методы.



| Метод                                 | Описание                                                                                                                                    |
|:---------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------|
| [**Сохранении**](summaryinfo-persist.md) | Форматирует и записывает ранее сохраненные свойства в стандартный [поток сводных данных](summary-information-stream.md).<br/> |



 

### <a name="properties"></a>Свойства

Объект **суммаринфо** имеет следующие свойства.



| Свойство                                                                             | Описание                                                                                     |
|:-------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------|
| [**Свойство Property (объект Суммаринфо)**](summaryinfo-summaryinfo.md)<br/> | Возвращает или задает значение для указанного свойства в информационном потоке сводки.<br/> |
| [**пропертикаунт**](summaryinfo-propertycount.md)<br/>                        | Указывает текущее количество значений свойств в объекте сводных данных.<br/>   |
