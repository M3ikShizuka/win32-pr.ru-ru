# <a name="msinetassemblysupport-property"></a>Мсинетассемблисуппорт, свойство

Свойство **мсинетассемблисуппорт** указывает, поддерживает ли компьютер общеязыковые сборки времени выполнения. В системах, поддерживающих общеязыковые сборки времени выполнения, установщик устанавливает значение **мсинетассемблисуппорт** в качестве версии файла Fusion.dll. Установщик не устанавливает это свойство, если операционная система не поддерживает общеязыковые сборки времени выполнения. Если несколько версий Fusion.dll установлены параллельно на компьютере пользователя, это свойство устанавливается в последнюю версию файла Fusion.dll. например, если платформа .NET Framework версии 1.0.3705 (Fusion.dll версии 1.0.3705.15) и платформа .NET Framework версии 1.1.4322 (Fusion.dll версии 1.1.4322.314) устанавливаются параллельно, свойство **мсинетассемблисуппорт** имеет значение 1.1.4322.314. Дополнительные сведения см. в разделе [сборки](assemblies.md).
