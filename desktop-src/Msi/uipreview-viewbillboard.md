# <a name="uipreviewviewbillboard-method"></a>Уипревиев. Виевбиллбоард, метод

Метод **виевбиллбоард** объекта [**уипревиев**](uipreview-object.md) отображает созданное объявление, используя элемент управления ведущего приложения в отображаемом в данный момент диалоговом окне.

## <a name="syntax"></a>Синтаксис


```JScript
UIPreview.ViewBillboard(
  control,
  billboard
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*control* 
</dt> <dd>

Обязательное имя элемента управления, в котором размещается объявление, с учетом регистра, вместе с диалоговым окном и первичными ключами таблицы управляющей базы данных.

</dd> <dt>

*печат* 
</dt> <dd>

Обязательное имя объявления, которое будет отображаться с использованием указанного элемента управления и текущего диалогового окна, и первичного ключа таблицы базы данных объявления.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
