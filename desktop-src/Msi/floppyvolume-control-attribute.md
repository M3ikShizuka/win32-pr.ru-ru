# <a name="floppyvolume-control-attribute"></a>Атрибут элемента управления Флоппиволуме

Если задан бит управления Флоппиволуме, в элементе управления отображаются все тома, вовлеченные в текущую установку, а также все гибкие тома.

Если этот бит не задан, элемент управления перечисляет тома в текущей установке.

## <a name="valid-controls"></a>Допустимые элементы управления

[директорикомбо](directorycombo-control.md)

[волумекостлист](volumecostlist-control.md)

[волумеселекткомбо](volumeselectcombo-control.md)

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                               |
|---------|-------------|----------------------------------------|
| 2097152 | 0x00200000  | **мсидбконтролаттрибутесфлоппиволуме** |



 

## <a name="remarks"></a>Remarks

Чтобы задать этот атрибут для элемента управления, включите бит Флоппиволуме в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

См. раздел [атрибуты элементов управления](control-attributes.md) и элемент управления, которые необходимо создать в элементах [управления](controls.md).
