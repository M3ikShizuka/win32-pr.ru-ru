# <a name="adding-registry-information"></a>Добавление данных реестра

В [таблице реестра](registry-table.md)и связанных таблицах базы данных установки содержатся сведения о реестре, необходимые приложению, написанному в системном реестре. См. [раздел «таблицы реестра](registry-tables-group.md)». В этом разделе вы добавите сведения, которые должны быть зарегистрированы на компьютере пользователя, с помощью примера блокнота.

Используйте редактор базы данных, чтобы открыть MNP2000.msi и ввести следующие данные в пустую таблицу реестра.
