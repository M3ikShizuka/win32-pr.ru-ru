# <a name="removeodbc-action"></a>Действие Ремовеодбк

Действие Ремовеодбк удаляет источники данных, переводчики и драйверы, перечисленные для удаления во время установки. Это действие запрашивает таблицу [одбкдатасаурце](odbcdatasource-table.md), [таблицу Одбктранслатор](odbctranslator-table.md)и [таблицу одбкдривер](odbcdriver-table.md) для каждого источника данных, переводчика или драйвера, запланированного для удаления.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Ограничения последовательности отсутствуют.

## <a name="actiondata-messages"></a>Сообщения Актиондата

Для каждого установленного драйвера.



| Поле | Описание данных действия               |
|-------|------------------------------------------|
| \[1\] | Описание драйвера. Ключ драйвера ODBC. |
| \[2\] | ComponentId                              |



 

Для каждого установленного транслятора.



| Поле | Описание данных действия               |
|-------|------------------------------------------|
| \[1\] | Описание драйвера. Ключ драйвера ODBC. |
| \[2\] | ComponentId                              |



 

Для каждого установленного источника данных.



| Поле | Описание данных действия                              |
|-------|---------------------------------------------------------|
| \[1\] | Описание драйвера. Ключ драйвера ODBC.                |
| \[2\] | ComponentId                                             |
| \[3\] | регистрация: SQL \_ удалить \_ dsn или SQL \_ удалить \_ SYS \_ dsn |



 

## <a name="remarks"></a>Remarks

Если счетчик использования ODBC и счетчик использования файлов становятся нулями, файл удаляется. Регистрация зависит от количества использований ODBC, и удаление файлов основано на счетчиках общего числа ссылок на библиотеки DLL.
