# <a name="installercomponentpath-property"></a>Свойство Installer. Компонентпас

Свойство **компонентпас** является свойством только для чтения, которое возвращает полный путь к установленному компоненту. Если путь к разделу для компонента является ключом реестра, то возвращается раздел реестра.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Installer.ComponentPath
```



## <a name="property-value"></a>Значение свойства

## <a name="remarks"></a>Remarks

Если компонент является разделом реестра, то корни реестра представлены в числовом виде. Например, путь реестра "HKEY \_ Current \_ user \\ Software \\ Microsoft" будет возвращен как "01: \\ Software \\ Майкрософт". Возвращаемые корни реестра определяются следующим образом.



| Root                    | Возвращаемое значение |
|-------------------------|----------------|
| \_корневой узел классов hKey \_     | 00             |
| HKEY \_ текущего \_ пользователя     | 01             |
| HKEY \_ локальный \_ компьютер    | 02             |
| \_Пользователи hKey             | 03             |
| \_данные о производительности hKey \_ | 04             |
