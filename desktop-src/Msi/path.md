# <a name="path"></a>Путь

Тип данных Path — это текстовая строка, содержащая допустимый полный путь.

Строка может также содержать имя свойства, заключенное в квадратные скобки \[ \] . В этом случае имя свойства, включая квадратные скобки, заменяется в строке значением свойства.

Примеры:

-   UNC-путь: \\ \\ \\ Общая папка сервера
-   Локальный диск: c: \\ TEMP
-   С именем свойства: \[ диск \] \\ TEMP
