# <a name="ice44"></a>ICE44

ICE44 проверяет, что [невдиалог](newdialog-controlevent.md), [спавндиалог](spawndialog-controlevent.md)и [спавнваитдиалог](spawnwaitdialog-controlevent.md) контролевентс ссылаются на допустимые диалоговые окна в [таблице диалоговых](dialog-table.md)окон.

## <a name="result"></a>Результат

ICE44 отправляет сообщение об ошибке, если имеется событие элемента управления диалогового окна, которое не ссылается на диалоговое окно, указанное в таблице диалоговых окон.

## <a name="example"></a>Пример

ICE44 сообщит о следующих ошибках в приведенном примере.



| ICE44, ошибка                                                                                                                               | Описание                                                                                                                                                                                                                  |
|-------------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Событие элемента управления для элемента управления "Dialog1". Control1 имеет тип Спавндиалог, но его аргумент Dialog2 не является допустимой записью в диалоговой таблице. | Существуют действия Спавндиалог, Спавнваитдиалог или Невдиалог с аргументом, который не ссылается на диалоговое окно в таблице диалоговых окон. Чтобы устранить эту ошибку, добавьте аргумент, который является ключом в таблице диалоговых окон.<br/> |

|---------|-------|
