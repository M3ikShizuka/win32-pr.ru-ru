# <a name="date-property"></a>Date, свойство

Свойство **Date** — это текущий месяц, день и год в виде строки литерального текста в формате mm/дд/гггг. Например, дата 22 июня 2005 может быть представлена в виде 06/22/2005. Формат значения зависит от языкового стандарта пользователя и имеет формат, полученный с помощью [**жетдатеформат**](/windows/desktop/api/datetimeapi/nf-datetimeapi-getdateformata) с \_ параметром Date шортдате. значение этого свойства задается установщик Windows, а не автором пакета.

## <a name="remarks"></a>Remarks

Поскольку это текстовая строка, ее нельзя использовать в условных выражениях.
