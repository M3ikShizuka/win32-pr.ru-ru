# <a name="shortcut"></a>Сочетание клавиш

Тип данных shortcut обычно используется в целевом столбце в [таблице ярлыков](shortcut-table.md). Если он содержит квадратные скобки ( \[ \] ), цель ярлыка вычисляется как отформатированная строка. В противном случае сочетание клавиш вычисляется как [идентификатор](identifier.md) и должно быть допустимым внешним ключом в [таблице Feature](feature-table.md).
