# <a name="oleadvtsupport-property"></a>Олеадвтсуппорт, свойство

Установщик устанавливает значение true для свойства **олеадвтсуппорт** , если система текущего пользователя была обновлена для работы с установкой по запросу через COM. Обратите внимание, что для того, чтобы установщик зарегистрировал класс COM и включил установку по запросу, класс должен быть указан в таблицах [классов](class-table.md) и [ProgID](progid-table.md) .

Windows установщик на Windows Server 2003 или Windows XP. сведения о минимальном Windows пакета обновления, который требуется для установщик Windows версии, см. в [установщик Windows требования к Run-Time](windows-installer-portal.md) .<br/> |
