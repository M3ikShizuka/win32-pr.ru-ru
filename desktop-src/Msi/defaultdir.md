# <a name="defaultdir"></a>дефаултдир

Тип данных Дефаултдир — это текстовая строка, содержащая либо допустимое [имя файла](filename.md) , либо допустимый [идентификатор](identifier.md). Он используется только в [таблице Directory](directory-table.md). Он должен быть идентификатором, если каталог является корневым каталогом. Если каталог не является корневым, это значение должно быть парой имя файла или имя файла: имя файла. Обратите внимание, что "." допускается в качестве имени файла и имеет специальное значение в таблице Directory.
