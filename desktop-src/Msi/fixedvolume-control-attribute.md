# <a name="fixedvolume-control-attribute"></a>Атрибут элемента управления Фикседволуме

Если задан бит управления Фикседволуме, элемент управления отображает все тома, вовлеченные в текущую установку, а также все фиксированные внутренние жесткие диски.

Если этот бит не задан, элемент управления выводит список томов в текущей установке.

## <a name="valid-controls"></a>Допустимые элементы управления

[директорикомбо](directorycombo-control.md)

 

[волумекостлист](volumecostlist-control.md)

 

[волумеселекткомбо](volumeselectcombo-control.md)



| Decimal | Шестнадцатеричный | Константа                              |
|---------|-------------|---------------------------------------|
| 131072  | 0x00020000  | **мсидбконтролаттрибутесфикседволуме** |



 

## <a name="remarks"></a>Remarks

Чтобы задать этот атрибут для элемента управления, включите бит Фикседволуме в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

См. раздел [атрибуты элементов управления](control-attributes.md) и элемент управления, которые необходимо создать в элементах [управления](controls.md).
