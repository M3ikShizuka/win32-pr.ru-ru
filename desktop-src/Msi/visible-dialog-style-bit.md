# <a name="visible-dialog-style-bit"></a>Бит видимого стиля диалогового окна

Если этот бит установлен, то диалоговое окно изначально создано как видимое, в противном случае оно скрыто.

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                         |
|---------|-------------|----------------------------------|
| 1       | 0x00000001  | **мсидбдиалогаттрибутесвисибле** |
