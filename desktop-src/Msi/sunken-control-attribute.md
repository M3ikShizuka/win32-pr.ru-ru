# <a name="sunken-control-attribute"></a>Атрибут утопленного элемента управления

Если этот бит установлен, то элемент управления отображается с утопленным, трехмерным видом. Результат этого бита стиля отличается для разных элементов управления и версий Windows. На некоторых элементах управления он не имеет видимого результата. Если система не поддерживает атрибут утопленного элемента управления, элемент управления отображается в визуальном стиле по умолчанию. Если этот бит не задан, элемент управления отображается с визуальным стилем по умолчанию.

## <a name="valid-controls"></a>Допустимые элементы управления

Все элементы управления.

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                         |
|---------|-------------|----------------------------------|
| 4       | 0x00000004  | **мсидбконтролаттрибутессункен** |



 

## <a name="remarks"></a>Комментарии

Чтобы задать этот атрибут для элемента управления, включите Утопленный бит в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

См. раздел [Управление атрибутами](control-attributes.md) и [элементами управления](controls.md).
