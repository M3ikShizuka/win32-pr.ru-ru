# <a name="installer-units"></a>Единицы установщика

установщик Windowsая единица пользовательского интерфейса приблизительно равна одной двенадцатой (1/12) высотой размера шрифта в 10-пт MS Sans Serif. Относительный размер диалоговых окон, элементов управления и шрифтов может изменяться в зависимости от размера шрифта, используемого в пользовательском интерфейсе. Чтобы обеспечить правильное отображение пользовательского интерфейса, разработчики программы установки должны всегда тестировать свои приложения, используя фактические размеры шрифтов.
