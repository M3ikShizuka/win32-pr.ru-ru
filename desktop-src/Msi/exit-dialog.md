# <a name="exit-dialog"></a>Выйти из диалогового окна

После успешной установки появится диалоговое окно выход. Обычно это диалоговое окно содержит только некоторый текст и кнопку **ОК** . Имя этого диалогового окна должно появиться в [таблице админуисекуенце](adminuisequence-table.md) и [таблице инсталлуисекуенце](installuisequence-table.md) с параметром-1 в качестве порядкового номера. Это модальное диалоговое окно.
