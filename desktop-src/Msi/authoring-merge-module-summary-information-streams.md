# <a name="authoring-merge-module-summary-information-streams"></a>создание сводных данных о модуле слияния Потоки

При создании модуля слияния требуется информационный поток сводки, чтобы включить эти необходимые свойства сводных данных.

Каждый модуль слияния должен иметь сводный информационный поток для модуля слияния со следующими свойствами сводных данных:

-   [**Сводка по шаблону**](template-summary.md)
-   [**Сводка по номеру редакции**](revision-number-summary.md)
-   [**Сводка по количеству страниц**](page-count-summary.md)
-   [**Сводка по количеству слов**](word-count-summary.md)

Полный список свойств сводных данных в модулях слияния см. в разделе [ссылка на сводный поток информации модуля слияния](merge-module-summary-information-stream-reference.md).
