# <a name="patch-object"></a>Объект Patch

Объект **Patch** представляет уникальный экземпляр зарегистрированного или примененного исправления.

Экземпляр объекта можно создать с помощью свойства **Patch** как «Виндовсинсталлер. Installer. patch (*патчкоде*, *ProductCode*,) ».  Для контекста компьютера параметром « *значение» должен* быть пустой строкой. Значение *ProductCode* может быть пустой строкой для исправлений, которые зарегистрированы только и еще не применены к какому либо продукту. Значение *ProductCode* может быть пустой строкой при чтении или обновлении сведений списка источников исправления.

## <a name="members"></a>Элементы

Объект **Patch** имеет следующие типы членов:

### <a name="methods"></a>Методы

Объект **Patch** содержит следующие методы.



| Метод                                                               | Описание                                                                                                                             |
|:---------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------|
| [**саурцелистаддмедиадиск**](patch-sourcelistaddmediadisk.md)       | Добавьте диск в набор зарегистрированных дисков.<br/>                                                                                   |
| [**саурцелистаддсаурце**](patch-sourcelistaddsource.md)             | Добавьте сеть или URL-адрес в исходный список. <br/>                                                                             |
| [**саурцелистклеаралл**](patch-sourcelistclearall.md)               | Очищает полный исходный список указанного типа источников.<br/>                                                            |
| [**саурцелистклеармедиадиск**](patch-sourcelistclearmediadisk.md)   | Удалить диск из набора зарегистрированных дисков из списка источников. <br/>                                                        |
| [**саурцелистклеарсаурце**](patch-sourcelistclearsource.md)         | Удаление сети или источника URL-адреса из списка источников.<br/>                                                                         |
| [**саурцелистфорцересолутион**](patch-sourcelistforceresolution.md) | Удаляет последний использованный источник из исходного списка. Это приводит к тому, что разрешение списка источников будет вычисляться в следующий раз, когда требуется источник.<br/> |

| Свойство                                                  | Описание                                                                                                |
|:----------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------|
| [**Контекст**](patch-context.md)<br/>               | Контекстом этого экземпляра Patch является значение МСИИНСТАЛЛКОНТЕКСТ.<br/>                                   |
| [**медиадискс**](patch-mediadisks.md)<br/>         | Перечисление всех дисков мультимедиа для данного экземпляра исправления.<br/>                                         |

| [**саурцелистинфо**](patch-sourcelistinfo.md)<br/> | Возвращает и задает свойства сведений об источнике. Это свойство для чтения или записи.<br/>              |
