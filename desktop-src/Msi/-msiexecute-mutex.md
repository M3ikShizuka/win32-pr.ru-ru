Поскольку две установки не могут быть запущены в одном процессе, попытка вызова программного интерфейса (API) установщика возвращает сообщение об ошибке \_ Installed \_ \_ (1618) в двух случаях:

-   Пока \_ задан мьютекс мсиексекуте.
-   Пока текущий процесс обрабатывает [таблицу инсталлуисекуенце](installuisequence-table.md) или [таблицу админуисекуенце](adminuisequence-table.md).
