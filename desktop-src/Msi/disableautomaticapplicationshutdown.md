# <a name="disableautomaticapplicationshutdown"></a>дисаблеаутоматикаппликатионшутдовн

если для этой системной политики на уровне компьютера задано значение 1 (один), установщик Windows не взаимодействует с [диспетчером перезапуска](/windows/desktop/RstMgr/restart-manager-portal), но будет использовать [диалоговое окно филесинусе](filesinuse-dialog.md).

если для этой системной политики на компьютере задано значение 2 (два), установщик Windows использует [диалоговое окно филесинусе](filesinuse-dialog.md). этот параметр отключает попытки [диспетчера](/windows/desktop/RstMgr/restart-manager-portal) перезапуска по устранению перезапусков при установке пакета установщик Windows, который не был создан для использования диспетчера перезапуска. Установщик по-прежнему использует диспетчер перезапуска для обнаружения файлов, используемых приложениями.

политика дисаблеаутоматикаппликатионшутдовн доступна начиная с версии установщик Windows 4,0 на Windows Vista.

## <a name="registry-key"></a>Ключ реестра

**HKey \_ \_** \\ **политики программного обеспечения** локального компьютера \\  \\  \\  \\ **установщик** Microsoft Windows

## <a name="data-type"></a>Тип данных

**REG \_ DWORD**
