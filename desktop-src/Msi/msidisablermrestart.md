# <a name="msidisablermrestart-property"></a>МСИДИСАБЛЕРМРЕСТАРТ, свойство

Свойство **мсидисаблермрестарт** определяет, каким способом приложения или службы, использующие файлы, затрагиваемые обновлением, должны быть выключены и перезапущены, чтобы обеспечить установку обновления.

## <a name="value"></a>Значение



| Значение                                                                        | Значение                                                                                                                                                                                      |
|------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <dl> <dt>0</dt> </dl> | Все системные службы, которые были отключены для установки обновления, перезапускаются. Все приложения, зарегистрированные в [диспетчере перезапуска](../rstmgr/restart-manager-portal.md) , перезапускаются.<br/> |
| <dl> <dt>1</dt> </dl> | Процессы, которые были закрыты с помощью [диспетчера перезапуска](../rstmgr/restart-manager-portal.md) во время установки обновления, не будут перезапущены после применения обновления.<br/>                           |



 

## <a name="remarks"></a>Remarks

Свойство **мсидисаблермрестарт** игнорируется, если [Диспетчер перезапуска](../rstmgr/restart-manager-portal.md) недоступен или отключен.
