# <a name="sessioncomponentcosts-property"></a>Свойство Session. Компоненткостс

Свойство Компоненткостс объекта [**Session**](session-object.md) возвращает объект [**рекордлист**](recordlist-object.md) , перечисляющий место на диске, необходимое для установки компонента. Эти сведения используются в пользовательском интерфейсе для вывода места на диске, необходимого для всех дисков. Объем возвращаемого места на диске составляет несколько 512 байт.

Свойство Компоненткостс должно использоваться только после того, как установщик завершит [Расчет стоимости файлов](file-costing.md) и после [действия костфинализе](costfinalize-action.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Session.ComponentCosts
```



## <a name="property-value"></a>Значение свойства

## <a name="remarks"></a>Remarks

Чтобы получить общую стоимость, добавьте затраты для всех компонентов, а также стоимость подсистемы установщика (компонент = "").

Компоненткостс возвращает [**объект рекордлист**](recordlist-object.md). Каждая запись в возвращенном объекте Рекордлист имеет следующие поля:



| Поле | Описание                                          |
|-------|------------------------------------------------------|
| 1     | Имя тома или диска                                    |
| 2     | Окончательная стоимость места на диске (кратная 512 байт).     |
| 3     | Затраты на временное место на диске, кратные 512 байт. |
