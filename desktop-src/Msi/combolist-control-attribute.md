# <a name="combolist-control-attribute"></a>Атрибут элемента управления Комболист

Если для поля со списком задан бит управления Комболист, поле редактирования заменяется статическим текстовым полем. Это предотвращает ввод пользователем нового значения и требует, чтобы пользователь выбрал только одно из предопределенных значений.

Если этот бит не задан, поле со списком содержит поле редактирования.

## <a name="valid-controls"></a>Допустимые элементы управления

[ComboBox](combobox-control.md)

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Описание                     |
|---------|-------------|---------------------------------|
| 131072  | 0x00020000  | мсидбконтролаттрибутескомболист |



 

## <a name="remarks"></a>Remarks

Чтобы задать этот атрибут для элемента управления, включите бит Комболист в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

См. раздел [атрибуты элементов управления](control-attributes.md) и элемент управления, которые необходимо создать в элементах [управления](controls.md).
