# <a name="pushlike-control-attribute"></a>Атрибут элемента управления Пушлике

Если этот бит установлен для флажка или группы переключателей, то кнопка рисуется с помощью кнопки «кнопка», но ее логика остается прежней. Если бит не задан, элементы управления отображаются в обычном стиле.

## <a name="valid-controls"></a>Допустимые элементы управления

[CheckBox](checkbox-control.md)[радиобуттонграуп](radiobuttongroup-control.md)

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                           |
|---------|-------------|------------------------------------|
| 131072  | 0x00020000  | **мсидбконтролаттрибутеспушлике** |



 

## <a name="remarks"></a>Remarks

Чтобы задать этот атрибут для элемента управления, включите бит Пушлике в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

См. раздел [Управление атрибутами](control-attributes.md) и [элементами управления](controls.md).
