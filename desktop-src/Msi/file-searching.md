# <a name="file-searching"></a>Поиск файлов

Во время установки часто приходится искать конкретный файл. Для этой цели в установщике существует четыре стандартных действия поиска. [Действие аппсеарч](appsearch-action.md) использует [таблицу аппсеарч](appsearch-table.md) , чтобы определить, существует ли файл в системе пользователя. Во время установки обновления [действие ккпсеарч](ccpsearch-action.md) и [действие рмккпсеарч](rmccpsearch-action.md) могут использовать соответствующие таблицы для проверки системы пользователя на наличие предыдущей версии продукта.
