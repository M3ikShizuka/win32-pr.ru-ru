# <a name="reinstallmode-property"></a>REINSTALLMODE, свойство

Свойство **REINSTALLMODE** — это строка, которая содержит буквы, указывающие тип выполняемой переустановки. Параметры не учитывают регистр и не зависят от порядка. Обычно это свойство всегда используется вместе со свойством [**REINSTALL**](reinstall.md) . Однако это свойство можно также использовать во время установки, а не только для повторной установки.

> [!Note]  
> Установщик Windows игнорирует свойство **REINSTALLMODE** во время [административной установки](administrative-installation.md).

Если свойство **REINSTALLMODE** определено без определения свойства [**REINSTALL**](reinstall.md) , то указанные режимы "обнаружения" по-прежнему применяются и указывают режим "Перезапись" для обычной установки. Свойство **REINSTALLMODE** влияет только на те функции, которые выбраны обычным образом для установки. Наличие свойства **REINSTALLMODE** не переустанавливает компоненты. Для переустановки компонентов требуется наличие свойства **переустановки** .

Коды параметров для этого свойства соответствуют [параметру командной строки](command-line-options.md) "/f". Параметр командной строки имеет значение по умолчанию "пекмс".

> [!Note]  
> Все файлы, содержащие сведения о контрольной сумме, проверяются и восстанавливаются. Флаг REINSTALLMODE \_ филеверифи (ккоде выше) исправляет только те файлы, которые имеют мсидбфилеаттрибутесчекксум в столбце Attributes [таблицы File](file-table.md).
