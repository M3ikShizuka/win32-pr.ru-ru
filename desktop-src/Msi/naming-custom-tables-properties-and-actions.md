# <a name="naming-custom-tables-properties-and-actions"></a>Именование настраиваемых таблиц, свойств и действий

авторы пакетов должны гарантировать, что имена любых настраиваемых таблиц, свойств или действий, определенных в их пакете, не будут конфликтовать с именами стандартных таблиц, свойств или действий, которые являются собственными для установщик Windows.

Авторы пакетов должны следовать следующим правилам для настраиваемых имен таблиц, свойств и действий.

-   Создание имен для пользовательских таблиц, свойств или действий с префиксом, идентифицирующим приложение.
-   Никогда не используйте имя с MSI в качестве префикса. начиная с установщик Windows 2,0, всем новым свойствам, действиям и таблицам в машинном установщик Windows присваиваются имена с префиксом Msi. Этот префикс не чувствителен к регистру, например Мсиневинтерналтабле. Поскольку префикс не чувствителен к регистру, авторы должны избегать всех вариантов префикса MSI.

Дополнительные сведения см. в разделе [имена таблиц](table-names.md).
