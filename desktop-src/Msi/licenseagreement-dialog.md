# <a name="licenseagreement-dialog"></a>Диалоговое окно Лиценсеагримент

В этом модальном диалоговом окне отображается лицензионное соглашение для пользователя. Обычно диалоговое окно отображает текст соглашения с помощью [элемента управления скроллаблетекст](scrollabletext-control.md) и пары переключателей, которые позволяют пользователю принять или отклонить соглашение. По юридическим причинам рекомендуется, чтобы ни одна кнопка не была представлена пользователю, как уже выбрана по умолчанию. Это заставляет пользователя выбрать активный вариант. Авторы обычно используют для этой цели [элемент управления радиобуттонграуп](radiobuttongroup-control.md) . Обратите внимание, что фокус на диалоговом окне не перемещается в элемент управления Радиобуттонграуп до тех пор, пока не будет выбрана одна из кнопок в группе.
