# <a name="progress-control-attribute"></a>Атрибут элемента управления progress

Этот атрибут измеряет, насколько далеко заполняется индикатор индикатора хода выполнения. Запись содержит два целых числа и строку. Первое число — это ход выполнения, второе число — диапазон (максимально возможное число для хода выполнения). При значении целые числа могут быть указаны в виде целочисленных полей или строк, содержащих целые числа. Если второе число отсутствует, предполагается, что используется значение по умолчанию (1024). Если ход выполнения больше, чем диапазон, он изменяется на диапазон. Третье поле — это строка, имя действия, для которого отображается ход выполнения.

Дополнительные сведения см. в статьях [Создание элемента управления ProgressBar](authoring-a-progressbar-control.md)и [Добавление настраиваемых действий в ProgressBar](adding-custom-actions-to-the-progressbar.md).

## <a name="valid-controls"></a>Допустимые элементы управления

[ProgressBar](progressbar-control.md)

## <a name="associated-bit-flags"></a>Связанные битовые флаги

Этот атрибут не использует битовые флаги.
