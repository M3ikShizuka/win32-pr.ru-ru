# <a name="ice15"></a>ICE15

ICE15 проверяет, что тип содержимого и ссылки на расширения в таблицах [MIME](mime-table.md) и [расширений](extension-table.md) являются обратными. Таблица MIME должна ссылаться на тип содержимого на расширение, которое таблица расширений ссылается на тот же тип содержимого.

Несколько расширений могут ссылаться на один и тот же тип MIME, если тип MIME ссылается на одно из расширений. Несколько типов MIME могут ссылаться на одно и то же расширение, если расширение ссылается на один из типов MIME.

Обратите внимание, что всякий раз, когда MIME ссылается на расширение, это расширение не может иметь \_ столбец MIME в таблице Extension со значением NULL.

## <a name="result"></a>Результат

ICE15 отправляет сообщение об ошибке, если тип содержимого и ссылки на расширение не являются обратными.

## <a name="example"></a>Пример

ICE15 отправляет два сообщения об ошибке для приведенного примера:

-   Типы содержимого Test/x-закрыли в таблице MIME ссылаются на расширение TST, но расширение TST в таблице расширений ссылается на закрылки/x-колыханий. Это не является обратным.
-   Закрылки типа содержимого ссылается на расширение FLP, но это расширение содержит запись NULL в \_ СТОЛБЦЕ MIME таблицы Extension.

[Таблица MIME](mime-table.md) (частичная)



| ContentType   | Расширение\_ |
|---------------|-------------|
| тест/x-тест   | TST         |
| закрылки/x | FLP         |



 

[Таблица расширения](extension-table.md) (частичная)



| Расширение | MIME\_        |
|-----------|---------------|
| TST       | закрылки/x |
| FLP       | NULL          |
