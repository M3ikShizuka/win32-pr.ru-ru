# <a name="featureinfodescription-property"></a>Феатуреинфо. Description, свойство

Свойство **Description** является свойством только для чтения, которое возвращает описание функции в столбце Описание [таблицы Feature](feature-table.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = FeatureInfo.Description
```



## <a name="property-value"></a>Значение свойства
