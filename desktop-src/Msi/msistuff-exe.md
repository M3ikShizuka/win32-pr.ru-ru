# <a name="msistuffexe"></a>Msistuff.exe

Msistuff.exe — это служебная программа командной строки, которая может использоваться для просмотра или настройки ресурсов в исполняемом файле начальной загрузки Setup.exe.

это средство доступно только в [компонентах Windows SDK для разработчиков установщик Windows](platform-sdk-components-for-windows-installer-developers.md).

## <a name="syntax"></a>Синтаксис

**мсистуфф setup.exe** *параметр {value}*

Если после параметра не указаны данные, этот ресурс удаляется.

## <a name="command-line-options"></a>Параметры командной строки

В Msistuff.exe используются следующие параметры командной строки без учета регистра. Вместо тире можно использовать разделитель с косой чертой. Если параметр указан несколько раз, используется только последнее вхождение.



| Параметр               | Идентификатор ресурса                  | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|----------------------|------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| параметры не указаны |                              | Отображение настраиваемых ресурсов в Setup.exe.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **-u**               | ИСЕТУППРОПНАМЕ \_ BASEURL      | Задайте BaseURL в качестве базового URL-адреса Setup.exe. Если значение не указано, расположением Setup.exe по умолчанию является съемный носитель. Только установки на основе URL-адресов подчиняются проверке с помощью [**WinVerifyTrust**](/windows/win32/api/wintrust/nf-wintrust-winverifytrust). Замыкающая косая черта URL-адреса является необязательной. Этот параметр можно опустить.                                                                                                                                                                                                                     |
| **-d**               | \_база данных исетуппропнаме     | Задайте MSI, имя файла .msi. Это относительный путь к .msi файлу относительно расположения программы Setup.exe. Этот параметр является обязательным, если не указан параметр-m. Параметры-d и-m являются взаимоисключающими. Они не могут быть указаны одновременно.                                                                                                                                                                                                                                                    |
| **-m**               | \_исправление исетуппропнаме        | В поле MSP укажите имя файла MSP. Это относительный путь к MSP-файлу относительно расположения программы Setup.exe. Этот параметр является обязательным, если не указан параметр-d. Параметры-m и-d являются взаимоисключающими. Они не могут быть указаны одновременно.                                                                                                                                                                                                                                                    |
| **-n**               | ИСЕТУППРОПНАМЕ \_ ProductName  | Укажите название продукта, имя продукта. Это имя, которое используется в тексте баннера для скачанного пользовательского интерфейса. Этот параметр можно опустить. Если этот параметр опущен, по умолчанию используется значение Product.                                                                                                                                                                                                                                                                                                                            |
| **-o**               | \_Операция исетуппропнаме    | Укажите тип выполняемой операции. Допустимые значения: INSTALL, МИНПАТЧ, МАЖПАТЧ и ИНСТАЛЛУПД. Дополнительные сведения об этих параметрах см. в разделе [начальная загрузка Интернета](internet-download-bootstrapping.md).                                                                                                                                                                                                                                                                                           |
| **-v**               | \_Минимальный исетуппропнаме \_ MSI | установите минимальную версию Msi, минимальную версию установщик Windows требуется на компьютере. если на компьютере не установлена минимальная версия установщик Windows, для обновления установщик Windows устанавливается соответствующий [Instmsi.exe](instmsi-exe.md) . Значение этого свойства имеет тот же формат, что и значение PID \_ PAGECOUNT. См. [**сводное свойство количества страниц**](page-count-summary.md) . значение должно быть не меньше 200, значение для установщик Windows версии 2,0. Этот параметр является обязательным. |
| **-i**               | ИСЕТУППРОПНАМЕ \_ инстлокатион | задайте расположение url-адреса инстмси, базовый url-адрес для исполняемых файлов обновления установщик Windows. Если это значение отсутствует, расположение исполняемых файлов обновления по умолчанию устанавливается в расположение Setup.exe. Этот параметр можно опустить.                                                                                                                                                                                                                                                                                                |
| **-a**               | ИСЕТУППРОПНАМЕ \_ инстмсиа     | задайте инстмсиа, имя версии ANSI для исполняемого файла обновления установщик Windows. Это относительный путь к версии ANSI Instmsi.exe относительно расположения, указанного параметром ИСЕТУППРОПНАМЕ \_ инстлокатион. Этот параметр является обязательным.                                                                                                                                                                                                                                                                                   |
| **-w**               | ИСЕТУППРОПНАМЕ \_ инстмсив     | задайте инстмсив, имя версии в юникоде для исполняемого файла обновления установщик Windows. Это относительный путь к версии Юникода Instmsi.exe относительно расположения, указанного параметром ИСЕТУППРОПНАМЕ \_ инстлокатион. Этот параметр является обязательным.                                                                                                                                                                                                                                                                             |
| **-p**               | \_Свойства исетуппропнаме   | Задайте строки свойства = значение. Это пары "свойство = значение" для включения в командную строку. Этот параметр можно опустить. Этот параметр не может быть указан несколько раз и должен быть указан в последнюю очередь в командной строке. Вся Командная строка, следующая за-p, считается частью {value}.                                                                                                                                                                                                                      |
