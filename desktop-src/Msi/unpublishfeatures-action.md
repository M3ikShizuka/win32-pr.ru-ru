# <a name="unpublishfeatures-action"></a>Действие Унпублишфеатурес

При удалении всего продукта действие Унпублишфеатурес удаляет из системного реестра сведения о сопоставлении состояния выбора и компонента компонента. Это действие запрашивает [таблицу феатурекомпонентс](featurecomponents-table.md).

## <a name="sequence-restrictions"></a>Ограничения последовательности

Ограничения последовательности отсутствуют.
