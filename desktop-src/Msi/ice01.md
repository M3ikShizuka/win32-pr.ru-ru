# <a name="ice01"></a>ICE01

ICE01 проверяет работоспособность механизма ICE. В этом ICE для получения времени используется свойство [**time**](time.md) , которое возвращает либо системное время, либо значение None.

## <a name="result"></a>Результат

ICE01 отправляет сообщение, предоставляющее время, которое программа установки вызвала ICE01. Эта ICE никогда не должна возвращать ошибку.
