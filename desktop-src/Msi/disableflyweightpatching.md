# <a name="disableflyweightpatching"></a>дисаблефливеигхтпатчинг

Если для этой системной политики на уровне компьютера задано значение 1 (одна), все параметры [оптимизации исправлений](patch-optimization.md) будут отключены во время установки.

Параметры [оптимизации исправлений](patch-optimization.md) и политика дисаблефливеигхтпатчинг доступны начиная с версии установщик Windows 3,0.

## <a name="registry-key"></a>Ключ реестра

**HKey \_ \_** \\ **Политики программного обеспечения** локального компьютера \\  \\  \\  \\ **установщик** Microsoft Windows

## <a name="data-type"></a>Тип данных

**REG \_ DWORD**
