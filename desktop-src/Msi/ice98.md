# <a name="ice98"></a>ICE98

ICE98 проверяет поле Description [таблицы одбкдатасаурце](odbcdatasource-table.md) для источника данных ODBC. Она использует функцию **склвалиддсн** , чтобы убедиться, что используются только допустимые символы, а описание не превышает максимально допустимой длины.

## <a name="result"></a>Результат

ICE98 отправляет следующие предупреждения.



| Предупреждение ICE98                    | Описание                                                                                                                                                                                                 |
|----------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Недопустимое имя источника данных. | значение в столбце Description [таблицы одбкдатасаурце](odbcdatasource-table.md) либо содержит недопустимые символы, либо имеет слишком большую длину, что означает, что он превышает SQL \_ максимальную \_ длину DSN, \_ равную 32. |



 

## <a name="example"></a>Пример

ICE98 сообщает о следующих предупреждениях для примера:

``` syntax
The data source name is invalid: !
The data source name is invalid: <String of length > 32>
```

[Таблица одбкдатасаурце](odbcdatasource-table.md) (частичная)



| DataSource | Описание                      |
|------------|----------------------------------|
| бадчар    | !                                |
| тулонг    | <String of length > 32> |
