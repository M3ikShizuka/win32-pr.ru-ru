# <a name="listview-table"></a>Таблица ListView

Строки ListView не рассматриваются как отдельные элементы управления, но они являются частью ListView, который функционирует как элемент управления. Таблица ListView определяет значения для всех элементов ListView.

Таблица ListView содержит следующие столбцы.



| Столбец   | Type                         | Ключ | Допускает значения NULL |
|----------|------------------------------|-----|----------|
| Свойство | [Идентификатор](identifier.md) | Д   | Нет        |
| Номер    | [Integer](integer.md)       | Д   | Нет        |
| Значение    | [Формате](formatted.md)   | Нет   | Нет        |
| Текст     | [Формате](formatted.md)   | Нет   | Д        |
| Двоичный\_ | [Идентификатор](identifier.md) | Нет   | Д        |



 

## <a name="columns"></a>Столбцы

<dl> <dt>

<span id="Property"></span><span id="property"></span><span id="PROPERTY"></span>Свойства
</dt> <dd>

Именованное свойство, которое должно быть привязано к этому элементу. Все элементы, привязанные к одному и тому же свойству, становятся частью одного и того же ListView.

</dd> <dt>

<span id="Order"></span><span id="order"></span><span id="ORDER"></span>Порядок
</dt> <dd>

Положительное целое число, используемое для определения порядка элементов, отображаемых в одном списке ListView. Целые числа не должны быть последовательными. Если ListView определен как упорядоченный, то все элементы должны иметь значение упорядочения. Если ListView определен как неупорядоченный, этот столбец игнорируется.

</dd> <dt>

<span id="Value"></span><span id="value"></span><span id="VALUE"></span>Значений
</dt> <dd>

Строка значения, связанная с этим элементом. Если выбрать строку, связанному свойству присваивается это значение.

</dd> <dt>

<span id="Text"></span><span id="text"></span><span id="TEXT"></span>Полнотекстовым
</dt> <dd>

Видимый, локализуемый текст, назначаемый элементу. Если эта запись или весь столбец отсутствуют, по умолчанию используется соответствующая запись в поле значение.

</dd> <dt>

<span id="Binary_"></span><span id="binary_"></span><span id="BINARY_"></span>Двоичный\_
</dt> <dd>

Данные изображения для значка. Это внешний ключ для [двоичной таблицы](binary-table.md).

</dd> </dl>

## <a name="remarks"></a>Remarks

Содержимое полей Value и Text форматируется функцией [**мсиформатрекорд**](/windows/desktop/api/Msiquery/nf-msiquery-msiformatrecorda) при создании элемента управления, поэтому они могут содержать любое выражение, которое может интерпретировать функция мсиформатрекорд. Форматирование происходит только при создании элемента управления и не обновляется, если свойство, вовлеченное в выражение, изменяется в течение жизненного цикла элемента управления.
