# <a name="importing-original-installation-database"></a>Импорт исходной базы данных установки

Начните разработку пакета обновления, скопировав установочный пакет и дерево исходного каталога исходного продукта. Инструкции по созданию пакета установки для MNP2000 приведены в [примере установки](an-installation-example.md). Необходимо скопировать содержимое следующих папок:

В. \\ пример \\MNP2000.msi

В. \\ пример \\ блокнота\\

В. \\ пример \\ двоичного файла\\

C: \\ \\ значок образца\\

Обновите содержимое папки блокнота, чтобы они соответствовали источнику, описанному в разделе [планирование основного обновления](planning-a-major-upgrade.md). Удалите все устаревшие исходные файлы, например Baseball.txt, и замените их обновленными файлами, такими как Baseba01.txt. Добавьте дополнительные новые файлы, предоставляемые обновлением, например Opera01.txt.

Переименуйте MNP2000.msi в MNP2001.msi. В последующих шагах будет использоваться редактор таблиц для изменения этой базы данных в MSI-файл для обновления. Таблицы базы данных, которые не были явно изменены в последующих разделах, идентичны таблицам в базе данных исходного продукта MNP2000.msi.
