# <a name="ice93"></a>ICE93

ICE93 выдает предупреждение, если пользовательское действие использует то же имя, что и стандартное действие. Список всех стандартных имен действий см. в разделе [Справочник по стандартным действиям](standard-actions-reference.md).

## <a name="result"></a>Результат

ICE93 отправляет следующее предупреждение.



| Предупреждение ICE93                                                      | Описание                                                                                                                    |
|--------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| В пользовательском действии " \[ 1 \] " используется то же имя, что и для стандартного действия. | В столбце Action таблицы CustomAction имеется идентификатор, который является именем установщик Windows стандартного действия. |
