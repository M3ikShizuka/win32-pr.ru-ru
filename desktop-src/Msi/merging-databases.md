# <a name="merging-databases"></a>Слияние баз данных

С помощью установщика можно добавить данные в одну базу данных в другую базу данных, выполнив слияние. Операции [слияния и преобразования](merges-and-transforms.md) выполняются на всей базе данных, а слияние объединяет две базы данных в одну. Слияние полезно для групп разработчиков, так как они позволяют разделить базу данных установки крупного приложения на небольшие части и затем объединить их в полную базу данных установки позже.

**Объединение нескольких баз данных компонентов в одну полную базу данных**

1.  Разрабатывайте частичные базы данных компонентов отдельно.
2.  Объедините каждую базу данных компонента в основную базу данных продукта, вызвав функцию [**мсидатабасемерже**](/windows/desktop/api/Msiquery/nf-msiquery-msidatabasemergea) .
