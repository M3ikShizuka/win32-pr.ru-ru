# <a name="ice42"></a>ICE42

ICE42 проверяет, что внутренние серверы не связаны с EXE – файлами в [таблице классов](class-table.md). Также проверяется, что только классы Локалсервер и LocalServer32 имеют аргументы и значения Дефинпрок.

## <a name="result"></a>Результат

ICE42 отправляет сообщение об ошибке, если в таблице классов есть внутренние серверы, связанные с ИСПОЛНЯЕМыми файлами.

## <a name="example"></a>Пример

ICE42 сообщит о следующих ошибках в приведенном примере.



| ICE42, ошибка                                                                                                                             | Описание                                                                                                                                                                                                               |
|-----------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CLSID "{GUID1}" является сервером InProc, но реализующий компонент "Component1" содержит EXE-файл ("test.exe") в качестве ключа.                | В качестве сервера INPROC указан исполняемый файл. Файлы EXE не могут быть серверами INPROC.                                                                                                                            |
| CLSID "{GUID1}" в контексте "InProcServer32" содержит аргумент. Аргументы могут иметь только контексты Локалсервер.                              | Чтобы устранить эту ошибку, удалите аргумент.                                                                                                                                                                                   |
| CLSID "{GUID1}" в контексте "InProcServer32" указывает значение InProc по умолчанию. Только контексты Локалсервер могут иметь значения INPROC по умолчанию. | Существует объект со значением INPROC по умолчанию, которое не является объектом, работающим в контекстах Локалсервер или LocalServer32. Чтобы устранить эту ошибку, удалите значение Дефлнпрок или измените контекст класса.<br/> |
