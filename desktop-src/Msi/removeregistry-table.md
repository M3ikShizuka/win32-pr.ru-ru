# <a name="removeregistry-table"></a>Таблица Ремоверегистри

Таблица Ремоверегистри содержит сведения реестра, которые приложению необходимо удалить из системного реестра.

Таблица Ремоверегистри содержит следующие столбцы.



| Столбец         | Type                         | Ключ | Допускает значения NULL |
|----------------|------------------------------|-----|----------|
| ремоверегистри | [Идентификатор](identifier.md) | Д   | Нет        |
| Root           | [Integer](integer.md)       | Нет   | Нет        |
| Ключ            | [регпас](regpath.md)       | Нет   | Нет        |
| Имя           | [Формате](formatted.md)   | Нет   | Д        |
| Компонент\_    | [Идентификатор](identifier.md) | Нет   | Нет        |

## <a name="remarks"></a>Remarks

Сведения из реестра удаляются из системного реестра, если соответствующий компонент выбран для локальной установки или запуска из источника.

Эта таблица упоминается при выполнении [действия ремоверегистривалуес](removeregistryvalues-action.md) .
