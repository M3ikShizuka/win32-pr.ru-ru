# <a name="componentcomponentcode-property"></a>Свойство Component. Компоненткоде

Код компонента рассматриваемого компонента. Это свойство вызывает функцию [**мсиенумкомпонентсекс**](/windows/desktop/api/Msi/nf-msi-msienumcomponentsexa) .

**[Установщик Windows 4,5 или более ранней версии](not-supported-in-windows-installer-4-5.md):** Не поддерживается. Это свойство доступно, начиная с установщик Windows 5,0.
