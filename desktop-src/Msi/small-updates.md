# <a name="small-updates"></a>Небольшие обновления

Небольшое обновление вносит изменения в один или несколько файлов приложения, которые являются слишком небольшими, чтобы гарантировать изменение кода продукта. Небольшое обновление также часто называют обновлением для быстрого устранения исправлений (QFE). Небольшое обновление не допускают повторной организации дерева компонентов компонентов.

Типичное небольшое обновление изменяет только один или два файла или раздел реестра. Так как небольшое обновление изменяет сведения в файле .msi, необходимо изменить код пакета установки. Код пакета хранится в свойстве [**Сводка номера редакции**](revision-number-summary.md) [информационного потока сводки](summary-information-stream.md).

Код продукта никогда не изменяется при небольшом обновлении, поэтому все изменения, представленные небольшим обновлением, должны соответствовать рекомендациям, описанным в разделе [изменение кода продукта](changing-the-product-code.md). Обновление требует [значительных](major-upgrades.md) обновлений для изменения [**ProductCode**](productcode.md). Если необходимо различать продукты без изменения кода продукта, используйте [незначительное обновление](minor-upgrades.md).
