# <a name="uilevel-property"></a>Duilevel, свойство

Установщик задает для свойства **duilevel** уровень пользовательского интерфейса. Внутренний пользовательский интерфейс включается и задается параметром [**мсисетинтерналуи**](/windows/desktop/api/Msi/nf-msi-msisetinternalui). Для этого свойства задан один из следующих типов данных ИНСТАЛЛУИЛЕВЕЛ.



| инсталлуилевел          | Числовое значение | Уровень пользовательского интерфейса                        |
|-------------------------|---------------|---------------------------------------------|
| ИНСТАЛЛУИЛЕВЕЛ \_ None    | 2             | Полностью автоматическая установка.             |
| ИНСТАЛЛУИЛЕВЕЛ \_ Basic   | 3             | Простой ход выполнения и обработка ошибок.         |
| ИНСТАЛЛУИЛЕВЕЛ \_ уменьшилось | 4             | Разработанный пользовательский интерфейс, диалоговые окна мастера подавляются.     |
| ИНСТАЛЛУИЛЕВЕЛ \_ Full    | 5             | Создан пользовательский интерфейс с мастерами, ходом выполнения, ошибками. |
