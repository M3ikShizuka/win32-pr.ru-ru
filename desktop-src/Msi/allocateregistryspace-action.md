# <a name="allocateregistryspace-action"></a>Действие Аллокатерегистриспаце

Действие Аллокатерегистриспаце гарантирует, что объем свободного места в реестре, заданный свойством [**аваилаблефрирег**](availablefreereg.md) , существует в реестре.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие Аллокатерегистриспаце должно быть вызвано после [инсталлинитиализе](installinitialize-action.md). Рекомендуется запланировать Аллокатерегистриспаце перед всеми остальными действиями, чтобы убедиться, что для продолжения достаточно места в реестре.

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия     |
|-------|--------------------------------|
| \[1\] | Пространство реестра, требуемое в КБ. |
