# <a name="installer-database"></a>База данных установщика

Этот материал предназначен главным образом для разработчиков, желающих создать новые пакеты установщика и получить подробные сведения о реляционной базе данных установщика. Полные пакеты установщика применяются разработчиками приложений в качестве эффективных средств для управления установкой приложений. Знание базы данных установки также полезно для разработчиков и администраторов приложений, желающих изменить существующие пакеты установщика и процессы установки с помощью настраиваемых действий. Общие сведения о установщик Windows и технологии управления компонентами см. в разделе [About установщик Windows](about-windows-installer.md).

База данных установщика содержит все необходимые сведения для установки приложения. Поэтому разработчикам средств для создания пакетов установки и разработчиков, создающих пакеты установки без использования таких средств, необходимо понимать базу данных установщика. Основные области функционирования базы данных установщика представлены в следующих разделах:

[Сведения о базе данных установщика](about-the-installer-database.md)

[Использование базы данных установщика](using-the-installer-database.md)

[Ссылка на базу данных установщика](installer-database-reference.md)
