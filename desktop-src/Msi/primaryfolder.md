# <a name="primaryfolder-property"></a>ПРИМАРИФОЛДЕР, свойство

**Примарифолдер** — это глобальное свойство, позволяющее автору назначить первичную папку для установки. Значение этого свойства должно быть именем ключа каталога, который существует в [таблице Directory](directory-table.md).

Установщик использует разрешенный путь к первичной папке, чтобы определить, какой том следует использовать при задании значений свойства [**примариволумепас**](primaryvolumepath.md) , свойства [**Примариволумеспацеаваилабле**](primaryvolumespaceavailable.md) , свойства [**примариволумеспацерекуиред**](primaryvolumespacerequired.md) и свойства [**примариволумеспацеремаининг**](primaryvolumespaceremaining.md) . Установщик предоставляет значения этих свойств для пользователей в пользовательском интерфейсе, помогая им управлять дисковым пространством. Обычно Авторы пакетов могут выбрать самую крупную папку в качестве основной папки.

Значение этого свойства должно быть именем ключа записи каталога, найденной в [таблице Directory](directory-table.md).
