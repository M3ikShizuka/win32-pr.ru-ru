# <a name="program-information-tables-group"></a>Группа "таблицы сведений о программах"

Группа "таблицы сведений о программе" содержит важную информацию, используемую во время установки.

-   [Таблица свойств](property-table.md) предоставляет средства для указания всех [свойств](properties.md) установки.
-   [Двоичная таблица](binary-table.md) содержит двоичные данные для таких элементов, как точечные рисунки, анимации и значки. Двоичная таблица также используется для хранения данных настраиваемых действий. Эта таблица может содержать точечный рисунок для объявления, значок программы или исполняемую форму настраиваемого действия.
-   [Таблица ошибок](error-table.md) используется для поиска шаблонов форматирования сообщений об ошибках при обработке ошибок с набором кодов ошибок. Это происходит в обычном случае, когда шаблон форматирования не задан. В установщике имеется собственный механизм обработки ошибок. Ошибки передаются как записи.
-   [Таблица ярлыков](shortcut-table.md) таблица ярлыков содержит сведения, необходимые приложению для создания ярлыков на компьютере пользователя.
-   [Таблица ресервекост](reservecost-table.md) содержит место на диске, необходимое для правильной работы каждого компонента.
