# <a name="integer-control-attribute"></a>Целочисленный атрибут элемента управления

Если этот бит задан для элемента управления, связанное с ним свойство, указанное в столбце Property [таблицы Control](control-table.md) , является целым числом. Если этот бит не задан, свойство является строковым значением.

## <a name="valid-controls"></a>Допустимые элементы управления

[CheckBox](checkbox-control.md)

[ComboBox](combobox-control.md)

[Правка](edit-control.md)

[ListBox](listbox-control.md)

[радиобуттонграуп](radiobuttongroup-control.md)

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                          |
|---------|-------------|-----------------------------------|
| 16      | 0x00000010  | **мсидбконтролаттрибутесинтежер** |



 

## <a name="remarks"></a>Remarks

Чтобы задать этот атрибут для элемента управления, включите целочисленный бит в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

См. раздел [Управление атрибутами](control-attributes.md) и [элементами управления](controls.md).
