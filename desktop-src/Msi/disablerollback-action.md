# <a name="disablerollback-action"></a>Действие Дисаблероллбакк

Действие Дисаблероллбакк отключает [откат](rollback-installation.md) для оставшейся части установки. Откат отключен только для действий, которые упорядочены после действия Дисаблероллбакк в таблицах последовательности. Откат будет отключен для всей установки, если действие Дисаблероллбакк является виртуализированным до [действия инсталлинитиализе](installinitialize-action.md).

## <a name="sequence-restrictions"></a>Ограничения последовательности

Ограничения последовательности отсутствуют.

## <a name="actiondata-message"></a>Сообщение Актиондата

Нет сообщений Актиондата.
