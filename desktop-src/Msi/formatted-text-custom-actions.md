# <a name="formatted-text-custom-actions"></a>Настраиваемые действия форматированного текста

Этот тип настраиваемого действия задает свойство или каталог установки из форматированной текстовой строки. Параметрами в этой строке могут быть свойства, переменные среды, пути к файлам или пути к каталогам компонентов. Это полезно для передачи сведений в последующие настраиваемые действия.

Следующие базовые типы настраиваемых действий задают свойство или каталог. Дополнительные сведения см. в описании конкретного типа настраиваемого действия.



| Тип настраиваемого действия                                 | Описание                                    |
|----------------------------------------------------|------------------------------------------------|
| [Тип настраиваемого действия 51](custom-action-type-51.md) | Задает свойство с форматированной текстовой строкой.  |
| [Тип настраиваемого действия 35](custom-action-type-35.md) | Задает каталог с форматированной текстовой строкой. |



 

Сведения о всех типах настраиваемых действий и их кодировании в [таблицу CustomAction](customaction-table.md)см. в разделе [сводный список всех типов пользовательских действий](summary-list-of-all-custom-action-types.md) .
