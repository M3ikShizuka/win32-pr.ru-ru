# <a name="using-isolated-components"></a>Использование изолированных компонентов

С помощью [изолированных компонентов](isolated-components.md)Авторы пакетов установки могут указать, что установщик копирует общие файлы приложения непосредственно в папку приложения, а не в общее расположение. Этот частный набор файлов затем используется исключительно приложением.

Дополнительные сведения см. в статьях [Установка COM-компонента в частном расположении](installing-a-com-component-to-a-private-location.md), [Установка компонента, не ЯВЛЯЮЩЕГОСЯ компонентом COM, в частном расположении](installing-a-non-com-component-to-a-private-location.md), [Создание COM-компонента в существующем пакете](make-a-com-component-in-an-existing-package-private.md) или [Создание частного компонента, не являющегося компонентом COM, в существующем пакете](make-a-non-com-component-in-an-existing-package-private.md).
