# <a name="components-and-features"></a>Компоненты и компоненты

Установщик Windows организует установку вокруг концепций компонентов и функций.

Функция — это часть общей функциональности приложения, которую пользователь может решить установить независимо. Дополнительные сведения см. в разделе [установщик Windows Features](windows-installer-features.md).

Компонент — это часть приложения или продукта для установки. Установщик всегда устанавливает или удаляет компонент с компьютера пользователя в качестве согласованного элемента. Компоненты обычно скрываются от пользователя. Когда пользователь выбирает компонент для установки, установщик определяет, какие компоненты должны быть установлены для предоставления этой функции. Дополнительные сведения см. в разделе [установщик Windows Components](windows-installer-components.md).

Авторы пакетов установки должны решить, как разделить приложение на компоненты и компоненты. Выбор компонентов в основном определяется функциями приложения с точки зрения пользователя. Поскольку компоненты обычно должны совместно использоваться несколькими приложениями или даже в разных продуктах, рекомендуется, чтобы авторы были стандартным методом выбора компонентов. Дополнительные сведения см. [в разделе Упорядочение приложений по компонентам](organizing-applications-into-components.md).
