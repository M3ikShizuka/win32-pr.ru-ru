# <a name="applying-a-patch-package-to-an-administrative-installation"></a>Применение пакета исправлений к административной установке

Вы можете применить небольшое обновление к исходному административному образу MNP2000.msi, установив образец исправления MNP2000. MSP, созданный при [создании пакета исправлений](generating-a-patch-package.md). Затем обновление может быть распространено для пользователей путем запроса на переустановку приложения из нового административного источника.

Администратор может использовать следующую командную строку для обновления исходного образа, расположенного по адресу//сервер/MNP2000.msi, на новый исходный образ, который будет создан административной установкой с полностью обновленного компакт-диска.

**msiexec/a//сервер/MNP2000.msi/p MNP2000. MSP**

Члены Рабочей группы, использующие MNP2000, должны переустановить приложение из нового административного исходного образа, чтобы получить обновление.

Чтобы полностью переустановить приложения и кэшировать обновленный MSI файл на компьютере пользователя, пользователи могут ввести одну из следующих команд.

**msiexec/фвомус//сервер/MNP2000.msi**

**msiexec/I//сервер/MNP2000.msi REINSTALL = ALL REINSTALLMODE = vomus REINSTALL**

Чтобы переустановить только обновленную функцию концерта и кэшировать обновленный MSI файл на компьютере пользователя, пользователи могут ввести следующую команду.

**msiexec/I//сервер/MNP2000.msi REINSTALL = концерт REINSTALLMODE = vomus REINSTALL**
