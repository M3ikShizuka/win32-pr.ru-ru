# <a name="condition"></a>Условие

Тип данных Condition — это текстовая строка, содержащая допустимый условный оператор, который можно вычислить как true или false. Дополнительные сведения о синтаксисе условных операторов см. в разделе [Синтаксис условных](conditional-statement-syntax.md)инструкций.
