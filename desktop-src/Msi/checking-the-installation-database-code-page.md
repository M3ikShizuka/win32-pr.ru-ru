# <a name="checking-the-installation-database-code-page"></a>Проверка кодовой страницы базы данных установки

*Кодовая страница* — это внутренняя таблица, используемая операционной системой для отображения символов (букв, цифр и знаков препинания) в номере символа. Различные кодовые страницы обеспечивают поддержку наборов символов, используемых в разных странах или регионах. Кодовые страницы упоминаются по числу; Например, кодовая страница 932 представляет собой японскую кодировку, а кодовая страница 950 — одну из китайских кодировок. Список кодовых страниц ASCII см. [в разделе Локализация таблиц ошибок и актионтекст](localizing-the-error-and-actiontext-tables.md) .

Та же кодовая страница ASCII, 1252, может использоваться для английского и французского языков. Поэтому кодовую страницу MNP2000.msi базы данных (на английском языке) не нужно сбрасывать, чтобы изменить установку на французский.

дополнительные сведения о настройке кодовой страницы см. в разделе [обработка кодовых страниц (установщик Windows)](code-page-handling-windows-installer-.md).
