# <a name="pidkey-property"></a>PIDKEY, свойство

Свойство **PIDKEY** содержит часть идентификатора продукта, вводимых пользователем. [Действие валидатепродуктид](validateproductid-action.md) проверяет идентификатор продукта, указанный путем сравнения свойства **PIDKEY** со свойством [**пидтемплате**](pidtemplate.md) .

Свойство **PIDKEY** можно задать в командной строке или с помощью пользовательского интерфейса. Это свойство содержит только часть идентификатора продукта, вводимых пользователем.
