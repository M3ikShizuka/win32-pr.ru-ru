# <a name="windows-sdk-components-for-windows-installer-developers"></a>Windows компоненты пакета SDK для установщик Windows разработчиков

компоненты комплекта установщик Windows software development kit включены в пакет средств разработки Microsoft Windows software development kit (SDK). Windows SDK включает в себя распространяемые компоненты, документацию, средство проверки базы данных установщика, редактор таблиц базы данных, схему базы данных, средства разработки, Visual Basic scripting Edition (VBScript), образец продукта и примеры кода.

страницу загрузки Windows SDK см. в разделе [пакет средств разработки программного обеспечения Microsoft Windows](https://developer.microsoft.com/windows/downloads/windows-10-sdk).
