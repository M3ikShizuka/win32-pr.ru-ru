# <a name="windows-sdk-components-for-windows-installer-developers"></a>Windows SDK компонентов для разработчиков установщик Windows

Компоненты комплекта установщик Windows Software Development Kit включены в пакет средств разработки программного обеспечения (SDK) для Microsoft Windows. Windows SDK включает в себя распространяемые компоненты, документацию, средство проверки базы данных установщика, редактор таблиц базы данных, схему базы данных, средства разработки, Visual Basic Scripting Edition (VBScript), образец продукта и примеры кода.

На странице загрузки Windows SDK см. раздел пакет [средств разработки программного обеспечения (SDK) для Microsoft Windows](https://developer.microsoft.com/windows/downloads/windows-10-sdk).
