# <a name="recordfieldcount-property"></a>Свойство Record. FieldCount

Свойство **FieldCount** объекта [**Record**](record-object.md) является свойством, предназначенным только для чтения и возвращающим количество полей в записи. Доступ для чтения к полям за пределами этого количества возвращает значения NULL. Ошибка доступа для записи.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Record.FieldCount
```
