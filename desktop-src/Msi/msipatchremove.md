# <a name="msipatchremove-property"></a>МСИПАТЧРЕМОВЕ, свойство

Свойство **мсипатчремове** указывает список исправлений, удаляемых во время установки. Отдельные исправления в списке разделяются точками с запятой. их можно представить с помощью GUID кода исправления или полного пути к исправлению. Функция [**мсиремовепатчес**](/windows/desktop/api/Msi/nf-msi-msiremovepatchesa) и метод [**ремовепатчес**](installer-removepatches.md) объекта [Installer](installer-object.md) задают свойство **мсипатчремове** .

Свойство **мсипатчремове** может быть задано в командной строке следующим образом для удаления исправления.

**msiexec/i A: \\Example.msi мсипатчремове = c: \\ патчи \\ qfe1. MSP; { 0BBB87F1-3186-409C-8CDD-C88AA2A4A7E0}; {A86B443B-E3BF-4009-ADED-F716FC735858}/QB**
