# <a name="a-small-update-patching-example"></a>Пример небольшого обновления исправлений

В этом примере показано, как создать [пакет исправлений](patch-packages.md) , который применяет [небольшое обновление](small-updates.md) к образцу пакета установки, рассматриваемого в [примере установки](an-installation-example.md). Небольшое обновление вносит изменения в один или несколько файлов приложения, которые считаются слишком небольшими, чтобы гарантировать изменение кода продукта. Дополнительные сведения см. в разделе [Установка исправлений и обновления](patching-and-upgrades.md).

в этом примере показано, как создать пакет исправлений установщик Windows, который обновляет функцию концерта гипотетического продукта MNP2000 для исправления ошибки в исходном продукте. В примере пакета исправлений применяется небольшое обновление продукта, которое не требует изменения кода продукта. Дополнительные сведения о основных обновлениях см. в разделе [основные обновления](major-upgrades.md) в разделе об [исправлениях и обновлениях](patching-and-upgrades.md) .

Образец пакета обновления имеет следующие характеристики.

-   Она корректирует незначительную ошибку в расписании концерта, отображаемом функцией концерта.
-   Он обновляет код пакета в соответствии с измененным пакетом установки.
-   Небольшое обновление можно применить путем исправления локальной установки продукта.
