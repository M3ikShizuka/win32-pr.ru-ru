# <a name="arpsystemcomponent-property"></a>АРПСИСТЕМКОМПОНЕНТ, свойство

Если задать для свойства **арпсистемкомпонент** значение 1 с помощью командной строки или преобразования, приложение не будет отображаться в списке **Установка и удаление программ** **панели управления**.
