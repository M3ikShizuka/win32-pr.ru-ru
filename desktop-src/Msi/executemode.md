# <a name="executemode-property"></a>ЕКСЕКУТЕМОДЕ, свойство

Свойство **ексекутемоде** указывает, как установщик выполняет обновления системы. Это свойство может быть полезно, если необходимо запустить установку без фактического изменения системы. Свойству можно присвоить значение None, чтобы отключить операции обновления, такие как установка файлов и значений реестра.

Это свойство может иметь одно из следующих двух значений. Установщик проверяет только первую букву значения и не учитывает регистр.

<dl> <dt>

<span id="None"></span><span id="none"></span><span id="NONE"></span>None
</dt> <dd>

В систему не вносятся изменения. Установщик запускает пользовательский интерфейс, а затем запрашивает базу данных.

</dd> <dt>

<span id="Script"></span><span id="script"></span><span id="SCRIPT"></span>Индекса
</dt> <dd>

Все изменения в системе выполняются с помощью выполнения скриптов. Это режим по умолчанию.

</dd> </dl>

## <a name="default-value"></a>Значение по умолчанию

Если это свойство не определено, режим выполнения по умолчанию имеет значение script.
