# <a name="leftunit-property"></a>Лефтунит, свойство

Если задано свойство **лефтунит** , то единица размещения размещается слева от числа вместо обычной правой стороны. Это соответствует языкам, в которых этот формат принят. В пользовательском интерфейсе это свойство используется для вывода размера файлов и доступного пространства на разных томах.
