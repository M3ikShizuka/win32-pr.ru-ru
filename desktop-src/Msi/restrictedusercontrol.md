# <a name="restrictedusercontrol-property"></a>Рестриктедусерконтрол, свойство

Свойству **рестриктедусерконтрол** должно быть присвоено значение 1, если ограниченные [общедоступные свойства](public-properties.md) могут быть отправлены на серверную часть. Авторы могут использовать это свойство в условном операторе, чтобы обеспечить выполнение диалоговых окон или действий в зависимости от наличия ограничений на общие свойства.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Версия<br/> | Windows установщик 5,0 на Windows Server 2012, Windows 8, Windows Server 2008 R2 или Windows 7. Windows установщик 4,0 или установщик Windows 4,5 на Windows Server 2008 или Windows Vista. Windows установщик на Windows Server 2003 или Windows XP. сведения о минимальном Windows пакета обновления, который требуется для установщик Windows версии, см. в [установщик Windows требования к Run-Time](windows-installer-portal.md) .<br/> |
