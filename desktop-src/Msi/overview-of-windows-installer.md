# <a name="overview-of-windows-installer"></a>Общие сведения о установщик Windows

Установщик Windows сокращает совокупную стоимость владения для клиентов, позволяя им эффективно устанавливать и настраивать ваши продукты и приложения. Установщик также может предоставить вашему продукту новые возможности для объявления функций без их установки, установки продуктов по требованию и добавления пользовательских настроек.
