# <a name="disablerollback"></a>дисаблероллбакк

Если для этой [системной политики](system-policy.md) задано значение 1, установщик не сохраняет файлы отката во время установки и отключает откат установки. По умолчанию откат включен. Администраторам рекомендуется не использовать эту политику, если она совершенно необязательна. Дополнительные сведения см. в разделе [ROLLBACK Installation](rollback-installation.md).

## <a name="registry-key"></a>Ключ реестра

Чтобы отключить откат для установки на уровне пользователя, задайте для **дисаблероллбакк** значение 1 в следующем разделе реестра:

**HKey \_ Текущие \_ политики пользовательского** \\ **программного обеспечения** \\  \\  \\  \\ **установщик** Microsoft Windows

Чтобы отключить откат для установки на компьютере, установите **дисаблероллбакк** в значение 1 в следующем разделе реестра.

**HKey \_ \_** \\ **Политики программного обеспечения** локального компьютера \\  \\  \\  \\ **установщик** Microsoft Windows

## <a name="data-type"></a>Тип данных

**REG \_ DWORD**
