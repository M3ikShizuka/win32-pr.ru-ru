# <a name="anypath"></a>анипас

Тип данных Анипас — это текстовая строка, содержащая либо полный путь, либо относительный путь. При указании относительного пути можно добавить длинное имя файла с коротким именем файла, разделив короткие и длинные имена с помощью вертикальной черты ( \| ). Обратите внимание, что таким образом нельзя указать несколько уровней каталога или полных путей. Путь может содержать свойства, заключенные в квадратные скобки ( \[ \] ).

Примеры допустимых данных Анипас:

-   \\\\\\Общая папка сервера \\ TEMP
-   c: \\ TEMP
-   \\Каталог
-   пакета ispac ~ 1 \| Project состояние

Примеры недопустимых данных Анипас:

-   c: \\ temp \\ пакета ispac ~ 1 \| c: \\ temp one \\ Project Status
-   \\временное \\ пакета ispac ~ 1 \| \\ временное \\ состояние Project
