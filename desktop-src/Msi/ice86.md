# <a name="ice86"></a>ICE86

ICE86 выдает предупреждение, если пакет использует свойство [**AdminUser**](adminuser.md) в столбце базы данных типа [Condition](condition.md) . Авторы пакетов должны использовать [**привилегированное**](privileged.md) свойство в условных инструкциях.

## <a name="result"></a>Результат

ICE86 отправляет следующее предупреждение.



| Предупреждение ICE86                                                                                               | Описание                                                            |
|-------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------|
| Свойство \` % s \` Найдено в столбце \` % s \` . \` % s \` в строке% s. \`\`Часто более подходящим является привилегированное свойство. | В поле условия было использовано свойство [**AdminUser**](adminuser.md) . |
