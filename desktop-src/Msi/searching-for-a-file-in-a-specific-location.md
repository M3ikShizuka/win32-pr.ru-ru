# <a name="searching-for-a-file-in-a-specific-location"></a>Поиск файла в указанном расположении

**Поиск файла в определенном месте в пользовательской системе**

1.  Вывод подписи файла и имени в [таблице сигнатур](signature-table.md). Оставшиеся поля в этой записи могут иметь значение NULL для поиска любой версии MyApp.exe.

    [Таблица сигнатур](signature-table.md)

    

    | Подпись          | Имя файла            |
    |--------------------|----------------------|
    | аппфиле<br/> | MyApp.exe<br/> |

    

     

2.  Введите свойство, устанавливаемое установщиком, если MyApp.exe установлен.

    [Таблица аппсеарч](appsearch-table.md) (частичная)

    

    | Свойство         | Подпись          |
    |------------------|--------------------|
    | MYAPP<br/> | аппфиле<br/> |

    

     

3.  Используйте [таблицу дрлокатор](drlocator-table.md). Введите полный путь к файлу в системе пользователя в поле путь. Введите значение 0 в столбец Depth для поиска в папке bin.

    [Таблица Дрлокатор](drlocator-table.md)

    

    | Подпись          | Parent | Путь                                                    | Глубина        |
    |--------------------|--------|---------------------------------------------------------|--------------|
    | аппфиле<br/> |        | C: \\ Program Files \\ мипродуктс \\ Projects \\ bin<br/> | 0<br/> |

    

     

4.  Включите действие Аппсеарч в последовательность действий. Если MyApp.exe установлен в папку C: \\ Program Files \\ мипродуктс \\ Projects \\ , УСТАНОВЩИК устанавливает для свойства MyApp значение Location.
