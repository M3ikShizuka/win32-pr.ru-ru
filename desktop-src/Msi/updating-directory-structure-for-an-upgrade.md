# <a name="updating-directory-structure-for-an-upgrade"></a>Обновление структуры каталогов для обновления

Используйте редактор базы данных Orca, поставляемый вместе с пакетом SDK, или другой редактор, чтобы открыть [таблицу Directory](directory-table.md) в MNP2001.msi. Структура каталогов образца обновления аналогична структуре каталогов исходного продукта и не требует внесения изменений в исходную таблицу каталога, описанную в разделе [Указание структуры каталогов](specifying-directory-structure.md).
