# <a name="componentusersid-property"></a>Свойство Component., то есть

Идентификатор безопасности пользователя для перечисленного компонента. Это свойство вызывает функцию [**мсиенумкомпонентсекс**](/windows/desktop/api/Msi/nf-msi-msienumcomponentsexa) .

**[установщик Windows 4,5 или более ранней версии](not-supported-in-windows-installer-4-5.md):** Не поддерживается. это свойство доступно, начиная с установщик Windows 5,0.
