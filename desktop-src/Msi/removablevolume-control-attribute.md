# <a name="removablevolume-control-attribute"></a>Атрибут элемента управления Ремоваблеволуме

Если этот бит задан, элемент управления отображает все тома, вовлеченные в текущую установку, а также все съемные тома. Если этот бит не задан, элемент управления перечисляет тома в текущей установке.

## <a name="valid-controls"></a>Допустимые элементы управления

[директорикомбо](directorycombo-control.md)

 

[волумекостлист](volumecostlist-control.md)

 

[волумеселекткомбо](volumeselectcombo-control.md)

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                                  |
|---------|-------------|-------------------------------------------|
| 65536   | 0x00010000  | **мсидбконтролаттрибутесремоваблеволуме** |



 

## <a name="remarks"></a>Remarks

Чтобы задать этот атрибут для элемента управления, включите бит Ремоваблеволуме в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

См. раздел [Управление атрибутами](control-attributes.md) и [элементами управления](controls.md).
