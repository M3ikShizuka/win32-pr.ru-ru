# <a name="sessiondatabase-property"></a>Свойство Session. Database

Свойство **базы данных** объекта [**Session**](session-object.md) является свойством, предназначенным только для чтения и возвращающим базу данных для текущего сеанса установки в качестве объекта [**базы данных**](database-object.md) .

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Session.Database
```
