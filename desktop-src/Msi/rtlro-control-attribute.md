# <a name="rtlro-control-attribute"></a>Атрибут элемента управления РТЛРО

Если этот бит стиля установлен, текст в элементе управления отображается в порядке чтения справа налево.

[скроллаблетекст](scrollabletext-control.md)

[волумекостлист](volumecostlist-control.md)

 

[CheckBox](checkbox-control.md)

 

[ComboBox](combobox-control.md)

 

[директорилист](directorylist-control.md)

 

[директорикомбо](directorycombo-control.md)

[селектионтри](selectiontree-control.md)

 

[волумеселекткомбо](volumeselectcombo-control.md)

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                        |
|---------|-------------|---------------------------------|
| 32      | 0x00000020  | **мсидбконтролаттрибутесртлро** |



 

## <a name="remarks"></a>Remarks

Чтобы задать этот атрибут для элемента управления, включите бит РТЛРО в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

См. раздел [Управление атрибутами](control-attributes.md) и [элементами управления](controls.md).
