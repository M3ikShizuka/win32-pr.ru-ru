# <a name="rollbackdisabled-property"></a>Роллбаккдисаблед, свойство

Установщик задает свойство **роллбаккдисаблед** , когда откат был отключен. **Роллбаккдисаблед** используется авторами пакетов, которым необходимо убедиться, что в установщике не отключен откат. Свойство **роллбаккдисаблед** может использоваться в условном выражении, которое фактически отказывается продолжать установку, если задано свойство **роллбаккдисаблед** .

## <a name="default-value"></a>Значение по умолчанию

По умолчанию откат включен.

## <a name="remarks"></a>Remarks

Поскольку [откат](rollback-custom-actions.md) и [фиксация](commit-custom-actions.md) не выполняются при отключенном откате, установщик не может правильно установить пакет, использующий эти типы настраиваемых действий в транзакции во время установки. В этом случае автор пакета должен включить условие с помощью Дисаблероллбакк, которое предотвращает продолжение установки, если откат отключен.

Значение политики Дисаблероллбакк может быть задано администратором в рамках назначения системной политики. Администраторам рекомендуется не отключать откат, если это не требуется. Дополнительные сведения о значении политики Дисаблероллбакк см. в разделе [Системная политика](system-policy.md).
