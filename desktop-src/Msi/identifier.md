# <a name="identifier"></a>Идентификатор

Тип данных identifier — это текстовая строка. Идентификаторы могут содержать символы ASCII A – Z (a – z), цифры, символы подчеркивания ( \_ ) или точки (.). Однако каждый идентификатор должен начинаться с буквы или символа подчеркивания.
