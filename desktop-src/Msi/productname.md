# <a name="productname-property"></a>Свойство ProductName

Свойство **ProductName** содержит имя устанавливаемого приложения. Используется только в целях показа. Объявляется как свойство продукта.

Это свойство является обязательным.

Это свойство можно изменить с помощью преобразования.

## <a name="remarks"></a>Remarks

Длина свойства **ProductName** не может превышать 63 символов. Ограничение на длину раздела реестра для DisplayName не существует.
