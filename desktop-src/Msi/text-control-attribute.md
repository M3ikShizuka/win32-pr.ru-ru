# <a name="text-control-attribute"></a>Атрибут элемента управления Text

Этот атрибут является текстовой строкой, отображаемой в элементе управления. При установке параметра, если поле 0 записи не равно null, запись форматируется с помощью Форматтекст. Если поле 0 равно null, то текст определяется в первом поле записи. При получении значения всегда возвращается в первом поле. Для некоторых элементов управления этот текст может быть невидимым. в Windows сочетание клавиш для элемента управления определяется путем помещения "&" перед нужным символом в этой строке.

## <a name="valid-controls"></a>Допустимые элементы управления

Все элементы управления, за исключением [элемента управления Line](line-control.md).

## <a name="associated-bit-flags"></a>Связанные битовые флаги

Нет.
