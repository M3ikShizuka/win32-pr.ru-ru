# <a name="msintsuitepersonal-property"></a>Мсинтсуитеперсонал, свойство

В операционных системах Windows XP и более поздних версий установщик устанавливает для свойства **мсинтсуитеперсонал** значение 1, если операционная система является Windows XP Home Edition. Установщик задает для этого свойства значение 1 только в том случае, если в \_ \_ структуре [**осверсионинфоекс**](/windows/win32/api/winnt/ns-winnt-osversioninfoexa) ЗАДАН флаг Personal набора ver. В противном случае установщик не устанавливает это свойство.

Установщик Windows 4,0 или установщик Windows 4,5 на Windows Server 2008 или Windows Vista. Установщик Windows в Windows Server 2003 или Windows XP.
