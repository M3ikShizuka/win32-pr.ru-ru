# <a name="msintsuitepersonal-property"></a>Мсинтсуитеперсонал, свойство

в операционных системах Windows xp и более поздних версий установщик устанавливает для свойства **мсинтсуитеперсонал** значение 1, если операционная система Windows XP Home Edition. Установщик задает для этого свойства значение 1 только в том случае, если в \_ \_ структуре [**осверсионинфоекс**](/windows/win32/api/winnt/ns-winnt-osversioninfoexa) ЗАДАН флаг Personal набора ver. В противном случае установщик не устанавливает это свойство.
