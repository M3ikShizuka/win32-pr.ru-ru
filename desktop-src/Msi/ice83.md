# <a name="ice83"></a>ICE83

ICE83 проверяет [таблицу мсиассембли](msiassembly-table.md). Это настраиваемое действие ICE отправляет ошибку, если путь к ключу для компонента, содержащего сборку Win32, установлен в файл манифеста. Явная ошибка отправляется, если значение, указанное в поле ключевого пути [таблицы Component](component-table.md) , равно значению, указанному в \_ поле файл манифеста таблицы мсиассембли. Это настраиваемое действие ICE отправляет ошибку, если в таблице Мсиассембли есть хотя бы одна запись, а [Таблица инсталлексекутесекуенце](installexecutesequence-table.md) не содержит как [действие мсипублишассемблиес](msipublishassemblies-action.md) , так и [действие мсиунпублишассемблиес](msiunpublishassemblies-action.md).

## <a name="result"></a>Результат

ICE83 отправляет следующие ошибки.



| ICE83, ошибка                                                                                                   | Описание                                                                                                                                                                                                    |
|---------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Путь к ключу для сборки Win32 SxS (компонент \_ = \[ 1 \] ) не должен быть файлом манифеста                       | ICE83 отправляет эту ошибку, если поле ключевого пути для сборки Win32 имеет значение файла манифеста (Component. ключевой = = Мсиассембли. file \_ manifest). \[1 \] — ключевой путь в таблице Component.                          |
| В таблице Инсталлексекутесекуенце должны присутствовать действия Мсипублишассемблиес и Мсиунпублишассемблиес. | ICE83 отправляет эту ошибку, если в таблице Мсиассембли есть хотя бы одна запись, но таблица Инсталлексекутесекуенце не содержит как действие Мсиассемблипублиш, так и действие Мсиассемблюнпублиш. |
