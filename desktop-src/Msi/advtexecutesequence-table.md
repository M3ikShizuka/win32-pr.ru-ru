# <a name="advtexecutesequence-table"></a>Таблица Адвтексекутесекуенце

В таблице Адвтексекутесекуенце перечислены действия, которые вызывает установщик при выполнении [действия объявления](advertise-action.md) верхнего уровня.

В таблице Адвтексекутесекуенце можно использовать только следующие действия. В этой таблице нельзя использовать настраиваемые действия.

[костфинализе](costfinalize-action.md)

[костинитиализе](costinitialize-action.md)

[креатешорткутс](createshortcuts-action.md)

[Функции InstallFinalize](installfinalize-action.md)

[инсталлинитиализе](installinitialize-action.md)

[инсталлвалидате](installvalidate-action.md)

[мсипублишассемблиес](msipublishassemblies-action.md)

[публишкомпонентс](publishcomponents-action.md)

[публишфеатурес](publishfeatures-action.md)

[публишпродукт](publishproduct-action.md)

[регистерклассинфо](registerclassinfo-action.md)

[регистерекстенсионинфо](registerextensioninfo-action.md)

[регистермимеинфо](registermimeinfo-action.md)

[регистерпрогидинфо](registerprogidinfo-action.md)

Столбцы идентичны столбцам [таблицы инсталлексекутесекуенце](installexecutesequence-table.md). Таблица Адвтексекутесекуенце содержит следующие столбцы.



| Столбец    | Type                         | Ключ | Допускает значения NULL |
|-----------|------------------------------|-----|----------|
| Действие    | [Идентификатор](identifier.md) | Да   | Нет        |
| Условие | [Condition](condition.md)   | Нет   | Да        |
| Последовательность  | [Integer](integer.md)       | Нет   | Да        |



 

## <a name="columns"></a>Столбцы

<dl> <dt>

<span id="Action"></span><span id="action"></span><span id="ACTION"></span>Поддержки
</dt> <dd>

Имя [стандартного действия](standard-actions.md) , которое должен выполнить установщик. Это первичный ключ таблицы.

</dd> <dt>

<span id="Condition"></span><span id="condition"></span><span id="CONDITION"></span>Выполняет
</dt> <dd>

Логическое выражение. Если результат вычисления выражения равен false, действие пропускается. Если синтаксис выражения недопустим, последовательность завершается, возвращая Иесбадактиондата. Дополнительные сведения о синтаксисе условных операторов см. в разделе [Синтаксис условных](conditional-statement-syntax.md)инструкций.

</dd> <dt>

<span id="Sequence"></span><span id="sequence"></span><span id="SEQUENCE"></span>Виртуализирован
</dt> <dd>

Положительное значение указывает на расположение последовательности действия. Следующие отрицательные значения указывают, что действие вызывается, если установщик возвращает флаг завершения. Каждый флаг завершения (отрицательное значение) можно использовать без более чем одного действия. Несколько действий могут иметь флаги завершения, но они должны быть разными флагами. Флаги завершения (отрицательные значения) обычно используются с [диалоговыми окнами](dialog-boxes.md).



| Флаг завершения          | Значение | Описание                                                                          |
|---------------------------|-------|--------------------------------------------------------------------------------------|
| мсидоактионстатуссукцесс  | -1    | Успешное завершение. Используется с диалоговыми окнами [выхода](exit-dialog.md) .               |
| мсидоактионстатусусерексит | -2    | Пользователь завершает установку. Используется с диалоговыми окнами [усерексит](userexit-dialog.md) .     |
| мсидоактионстатусфаилуре  | –3    | Неустранимое завершение работы. Используется с диалоговыми окнами [FatalError](fatalerror-dialog.md) . |
| мсидоактионстатуссуспенд  | –4    | Установка приостановлена.                                                                |



 

Ноль, все остальные отрицательные числа или значение NULL указывают на то, что действие никогда не вызывается.

</dd> </dl>
