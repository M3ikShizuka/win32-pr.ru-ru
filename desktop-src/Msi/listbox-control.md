# <a name="listbox-control"></a>Элемент управления ListBox

Элемент управления ListBox — это обычный список, позволяющий пользователю выбрать один вариант из списка предварительно заданных значений. Возможные значения считываются из [таблицы ListBox](listbox-table.md). Можно связать свойство строки или целого числа, введя имя свойства в столбце свойств [таблицы управления](control-table.md).

## <a name="control-attributes"></a>Атрибуты элемента управления

Чтобы изменить значение атрибута с помощью события, подпишите элемент управления на таблице ControlEvent событие в [таблице таблица eventmapping](eventmapping-table.md) и перечислите идентификатор атрибута в столбце Attribute. ## <a name="remarks"></a>Комментарии

Этот элемент управления можно создать из класса LISTBOX с помощью функции [**CreateWindowEx**](/windows/win32/api/winuser/nf-winuser-createwindowexa) . У него есть стили **WS \_ TABSTOP**, **WS \_ Group** и **WS \_** . Если бит стиля элемента управления имеет значение ON, элемент управления создается с использованием стилей «фунт», « **WS \_ VSCROLL**» и « **WS \_** », в противном случае элемент управления создается с использованием **\_ стандартного стиля фунта** . **\_**
