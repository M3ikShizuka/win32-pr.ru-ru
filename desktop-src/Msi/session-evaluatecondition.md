# <a name="sessionevaluatecondition-method"></a>Session. Евалуатекондитион, метод

Метод **евалуатекондитион** объекта [**Session**](session-object.md) вычисляет логическое выражение, содержащее символы и значения. Этот метод использует функцию [**мсиевалуатекондитион**](/windows/desktop/api/Msiquery/nf-msiquery-msievaluateconditiona) .

## <a name="syntax"></a>Синтаксис


```JScript
Session.EvaluateCondition(
  condition
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*выполняет* 
</dt> <dd>

Обязательная строка, содержащая логическое выражение. Дополнительные сведения см. в разделе [Синтаксис условных инструкций](conditional-statement-syntax.md).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод возвращает целое число, указывающее на вычисление условия.



| Константа                  | Значение | Описание                               |
|---------------------------|-------|-------------------------------------------|
| мсиевалуатекондитионфалсе | 0     | Условие принимает значение false.         |
| мсиевалуатекондитионтруе  | 1     | Условие принимает значение true.          |
| мсиевалуатекондитионноне  | 2     | Условное выражение не указано. |
| мсиевалуатекондитионеррор | 3     | Условие содержит синтаксическую ошибку.    |



 

## <a name="remarks"></a>Комментарии

Условные выражения можно использовать для сравнения состояний компонентов и компонентов. В следующей таблице показаны компоненты и состояния компонентов, используемые методом Евалуатекондитион.



| Состояние                 | Значение | Описание                                              |
|-----------------------|-------|----------------------------------------------------------|
| NULL                  | NULL  | Для компонента или компонента не предпринято никаких действий.                 |
| мсиинсталлстатеабсент | 2     | Отсутствует компонент или компонент.                     |
| мсиинсталлстателокал  | 3     | Компонент или компонент установлен на локальном компьютере. |
| мсиинсталлстатесаурце | 4     | Компонент или компонент установлен для запуска из источника.    |



 

> [!Note]  
> Состояния не задаются, пока не будет вызван метод [**сетинсталллевел**](session-setinstalllevel.md) , либо напрямую, либо с помощью [действия костфинализе](costfinalize-action.md). Поэтому проверка состояния полезна только в условных выражениях в таблице последовательности действий.
