# <a name="mediasourcedir-property"></a>Медиасаурцедир, свойство

Установщик задает для свойства **медиасаурцедир** значение 1, если при установке используется источник, расположенный на носителе, например компакт-диск. Это свойство не задано, если при установке используется источник, расположенный в сетевом расположении. Например, [элемент управления селектионтри](selectiontree-control.md) использует **медиасаурцедир** для определения того, выполняется ли установка из источника или выполняется из сетевого расположения.
