# <a name="executable-files"></a>Исполняемые файлы

Пользовательское действие может запустить исполняемый файл, хранящийся на компьютере пользователя или содержащийся в пакете установки.

Следующие типы пользовательского действия запускают исполняемый файл.



| Тип настраиваемого действия                                 | Описание                                         |
|----------------------------------------------------|-----------------------------------------------------|
| [Тип настраиваемого действия 2](custom-action-type-2.md)   | EXE-файл, хранящийся в потоке двоичных таблиц.           |
| [Тип настраиваемого действия 18](custom-action-type-18.md) | EXE-файл, установленный с продуктом.                  |
| [Тип настраиваемого действия 50](custom-action-type-50.md) | EXE-файл с путем, указанным значением свойства. |
| [Тип настраиваемого действия 34](custom-action-type-34.md) | EXE-файл с путем, ссылающимся на каталог.       |



 

Сведения о всех типах настраиваемых действий и их кодировании в [таблицу CustomAction](customaction-table.md)см. в разделе [сводный список всех типов пользовательских действий](summary-list-of-all-custom-action-types.md) .
