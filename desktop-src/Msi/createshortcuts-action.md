# <a name="createshortcuts-action"></a>Действие Креатешорткутс

Действие Креатешорткутс управляет созданием ярлыков.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие Креатешорткутс должно следовать после действия [инсталлфилес](installfiles-action.md) и действия [ремовешорткутс](removeshortcuts-action.md) .

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия      |
|-------|---------------------------------|
| \[1\] | Идентификатор созданного ярлыка. |



 

## <a name="remarks"></a>Комментарии

Действие Креатешорткутс создает ярлыки файлов ключей компонентов компонентов, выбранных для установки или объявления.
