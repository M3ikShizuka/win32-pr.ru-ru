# <a name="summary-information-stream"></a>Сводный поток информации

материал в этом разделе предназначен для разработчиков, создающих собственные программы установки, которые хотят узнать больше о установщик Windows таблицах базы данных. общие сведения см. в разделе [About установщик Windows](about-windows-installer.md).

Программа установки использует поток сводных данных для двух целей. во-первых, он содержит сведения о пакете, который можно просмотреть с помощью Microsoft Windows Explorer. Эти сведения доступны через интерфейс **IStream** . Во вторых, он содержит свойства, которые используются установщиком для установки пакета. Следующие разделы содержат сведения об использовании информационного потока сводки с установщиком.

-   [Сведения о потоке сводных данных](about-the-summary-information-stream.md)
-   [Использование потока сводных данных](using-the-summary-information-stream.md)
-   [Справочник по информационному потоку сводных данных](summary-information-stream-reference.md)
