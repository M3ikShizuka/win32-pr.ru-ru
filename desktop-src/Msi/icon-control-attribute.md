# <a name="icon-control-attribute"></a>Атрибут элемента управления Icon

Если этот бит задан, текст заменяется изображением значка, а текстовый столбец в [таблице управления](control-table.md) является внешним ключом в [двоичную таблицу](binary-table.md).

Если этот бит не задан, текст в элементе управления указывается в столбце text [таблицы Control](control-table.md).

## <a name="valid-controls"></a>Допустимые элементы управления

[CheckBox](checkbox-control.md)

[PushButton](pushbutton-control.md)

[радиобуттонграуп](radiobuttongroup-control.md)

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                       |
|---------|-------------|--------------------------------|
| 5242288 | 0x00080000  | **мсидбконтролаттрибутесикон** |



 

## <a name="remarks"></a>Комментарии

Чтобы задать этот атрибут для элемента управления, включите биты значков в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

Текстовый столбец в таблице управления используется в качестве внешнего ключа для [двоичной таблицы](binary-table.md), поэтому элемент управления не может содержать как изображение значка, так и текст.

Не задавайте биты значка и [точечного рисунка](bitmap-control-attribute.md) .

См. раздел [атрибуты элементов управления](control-attributes.md) и элемент управления, которые необходимо создать в элементах [управления](controls.md).
