# <a name="maintenance-installation"></a>Установка обслуживания

После успешной установки приложения пользователю может потребоваться вернуться к установке, чтобы добавить или удалить компоненты. Это называется установкой обслуживания.

[Преобразования](transforms.md) , которые были применены к установке, должны быть доступны во время последующих установок обслуживания.
