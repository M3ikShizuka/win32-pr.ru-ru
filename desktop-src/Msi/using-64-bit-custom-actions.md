# <a name="using-64-bit-custom-actions"></a>Использование 64-разрядных настраиваемых действий

в 64-разрядных операционных системах установщик Windows может вызывать пользовательские действия, компилируемые для 32-разрядных или 64-разрядных систем. 64-разрядное пользовательское действие на основе [скриптов](scripts.md) должно быть явно помечено как 64-разрядное настраиваемое действие путем добавления бита **msidbCustomActionType64BitScript** к числовому типу настраиваемого действия в столбце Type [таблицы CustomAction](customaction-table.md). Пользовательские действия, основанные на [исполняемых файлах](executable-files.md) или [библиотеках динамической компоновки](dynamic-link-libraries.md) , которые соответствуют 64-разрядным операционным системам, не нуждаются в включении этого дополнительного бита в столбец Type таблицы CustomAction.

Дополнительные сведения см. в статье [64-разрядные пользовательские действия](64-bit-custom-actions.md).
