# <a name="word-count-summary-property"></a>Свойство сводки количества слов

В сводных данных пакета установки свойство « **Сводка слов** » указывает тип образа исходного файла. Если это свойство отсутствует, по умолчанию оно равно нулю (0).

Это свойство является битовым полем. Новые биты могут быть добавлены в будущем. В настоящее время доступны следующие биты.



| bit   | Значение          | Описание                                                                                                                                                                                                                                      |
|-------|----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Бит 0 | 0 1<br/> | Длинные имена файлов. Короткие имена файлов.<br/>                                                                                                                                                                                                    |
| Бит 1 | 0 2<br/> | Источник не сжат. Источник сжат.<br/>                                                                                                                                                                                         |
| Бит 2 | 0 4<br/> | Источником является исходный носитель. Источником является административный образ, созданный административной установкой.<br/>                                                                                                                                 |
| Бит 3 | 0 8<br/> | Для установки этого пакета могут потребоваться повышенные привилегии. Для установки этого пакета повышенные привилегии не требуются.<br/> Доступно начиная с установщик Windows версии 4,0 и Windows Vista или Windows Server 2008.<br/> |



 

Они объединяются для предоставления свойству **Сводка по количеству слов** одного из следующих значений, указывающих тип исходного изображения файла.



| Значение | Тип                                                                                                                                                                                                                                                                                            |
|-------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0     | Исходный источник с длинными именами файлов. Соответствует дереву в [таблице каталогов](directory-table.md). Для установки этого пакета могут потребоваться повышенные привилегии.                                                                                                                                     |
| 1     | Исходный источник, использующий короткие имена файлов. Соответствует дереву в [таблице каталогов](directory-table.md). Для установки этого пакета могут потребоваться повышенные привилегии.                                                                                                                                    |
| 2     | Сжатые исходные файлы, использующие длинные имена файлов. Совпадает с CAB и файлами в [таблице Media](media-table.md). Для установки этого пакета могут потребоваться повышенные привилегии.                                                                                                                   |
| 3     | Сжатые исходные файлы с использованием коротких имен файлов. Совпадает с CAB и файлами в [таблице Media](media-table.md). Для установки этого пакета могут потребоваться повышенные привилегии.                                                                                                                  |
| 4     | Административный образ, использующий длинные имена файлов. Соответствует дереву в [таблице каталогов](directory-table.md). Для установки этого пакета могут потребоваться повышенные привилегии.                                                                                                                                |
| 5     | Административный образ, используя короткие имена файлов. Соответствует дереву в [таблице каталогов](directory-table.md). Для установки этого пакета могут потребоваться повышенные привилегии.                                                                                                                               |
| 8     | Для установки этого пакета повышенные привилегии не требуются. Используйте это значение при [создании пакетов без диалогового окна UAC](authoring-packages-without-the-uac-dialog-box.md). Доступно начиная с установщик Windows версии 4,0 и Windows Vista или Windows Server 2008.<br/> |



 

Обратите внимание, что если пакет помечен как сжатый (установлен бит 1), установщик Windows устанавливает только файлы, расположенные в корне источника. В этом случае даже файлы, помеченные как несжатые в [таблице файлов](file-table.md) , должны находиться в корневой папке для установки. Чтобы указать исходный образ, в котором есть как CAB-файл (сжатые файлы), так и несжатые файлы, соответствующие дереву в [таблице каталогов](directory-table.md), пометьте пакет как несжатый, отменив бит 1 неопределенное (значение = 0) в свойстве " **Сводка** ", и задайте **мсидбфилеаттрибутескомпрессед** (значение = 16384) в столбце Attributes [таблицы File](file-table.md) для каждого файла в CAB.

При преобразовании свойство " **Сводка слов** " должно иметь значение null.

В потоке сводных данных пакета исправлений свойство " **Сводка слов** " содержит минимальную версию установщик Windows, необходимую для установки исправления.



| Значение | Значение                                                                                                                                                                              |
|-------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1     | Значение по умолчанию, указывающее, что для создания исправления использовался МСПАТЧ.                                                                                                        |
| 2     | Для применения исправления требуется не менее установщик Windows 1,2. Исправление с числом слов "2" не удается сразу же при использовании с установщик Windowsной версией более 1,2. |
| 3     | Для применения исправления требуется не менее установщик Windows 2,0. Исправление с числом слов "3" не удается сразу же при использовании с установщик Windowsной версией более 2,0. |
| 4     | Для применения исправления требуется не менее установщик Windows 3,0. Исправление с числом слов "4" завершается ошибкой, если используется с установщик Windowsной версией более 3,0.             |
| 5     | Для применения исправления требуется не менее установщик Windows 3,1.                                                                                                               |



 

Это свойство сводки является обязательным.
