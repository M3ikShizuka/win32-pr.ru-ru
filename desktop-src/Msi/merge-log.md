# <a name="mergelog-method"></a>Метод Merge. log

Метод **log** объекта [**Merge**](merge-object.md) записывает текстовую строку в текущий открытый файл журнала.

## <a name="syntax"></a>Синтаксис


```JScript
Merge.Log(
  Message
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Message* 
</dt> <dd>

Текстовая строка для записи в файл журнала.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
