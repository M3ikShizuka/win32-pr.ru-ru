# <a name="ccp_drive-property"></a>\_Свойство диска CCP

Свойство **\_ Drive CCP** устанавливается в корневой путь к съемному тому, для которого выполняется поиск по адресу [рмккпсеарч](rmccpsearch-action.md). Действие Рмккпсеарч использует подписи файлов для проверки того, что соответствующие продукты установлены в системе перед установкой обновления.

Это свойство обычно задается с помощью пользовательского интерфейса или настраиваемого действия. Это свойство должно быть задано до выполнения действия [рмккпсеарч](rmccpsearch-action.md) .
