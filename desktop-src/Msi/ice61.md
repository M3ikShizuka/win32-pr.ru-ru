# <a name="ice61"></a>ICE61

ICE61 проверяет таблицу обновления, чтобы убедиться в соблюдении следующих условий.

-   Все свойства Актионпроперти не были предварительно созданы в таблице свойств.
-   Все свойства Актионпроперти являются общими свойствами.
-   Все свойства Актионпроперти включены в свойство [**секурекустомпропертиес**](securecustomproperties.md) .
-   Все свойства Актионпроперти уникальны для каждой записи в таблице обновления.
-   Все значения Версионмакс не меньше соответствующих значений Версионмин.
-   Значения Версионмин и Версионмакс являются допустимыми версиями продукта. Допустимый формат версии продукта см. в свойстве [**ProductVersion**](productversion.md) .
-   Попытка удалить более новую или равную версию текущего продукта не выполняется.

Ошибка исправления предупреждения или ошибки, о которой сообщает ICE61, обычно приводит к проблемам при обновлении приложения. В зависимости от точной ошибки это может быть любая из файлов из более старой версии, удаление файлов из старой версии, несмотря на то, что они необходимы для нового приложения, или завершении сбоя обновления.

## <a name="result"></a>Результат

ICE61 отправляет предупреждение или ошибку, если любое из указанных выше условий имеет значение true.

## <a name="example"></a>Пример

ICE61 сообщает о следующих ошибках и предупреждении для приведенных примеров.

``` syntax
This product should remove only older versions of itself. The Maximum version is not less than the current product. (2.01.0000 2.01.0000)
```

В этом случае в первой строке будет предпринята попытка удалить продукт той же версии. (Столбец Версионмакс равен версии продукта в таблице свойств).

Чтобы устранить эту ошибку, используйте версию в столбце Версионмакс ниже, чем текущая версия, указанная в таблице свойств. Удалите бит **мсидбупградеаттрибутесверсионмаксинклусиве** из столбца Attributes, если значение версионмакс равно текущей версии. Если намерение обнаруживает только продукт и не удаляет его, добавление бита **мсидбупградеаттрибутесонлидетект** к столбцу Attributes также приведет к устранению этой ошибки.

``` syntax
Upgrade.ActionProperty EnglishAPPFOUND must be added to the SecureCustomProperties property.
```

Если свойство не указано в свойстве [**секурекустомпропертиес**](securecustomproperties.md) , оно не передается на серверную часть установки при обнаружении свойства.

Чтобы устранить эту ошибку, добавьте свойство в [**секурекустомпропертиес**](securecustomproperties.md).

``` syntax
Upgrade.ActionProperty EnglishAPPFOUND must not contain lowercase letters.
```

Свойства обновления должны быть общедоступными свойствами для передачи результата на серверную часть установки.

Чтобы устранить эту ошибку, используйте все прописные буквы в имени свойства.

``` syntax
Upgrade.ActionProperty OLDAPPFOUND may be used in only one record of the Upgrade table.
```

Свойство может использоваться только в одной строке таблицы обновления.

Чтобы устранить эту ошибку, используйте другое свойство для второй строки.

``` syntax
Upgrade.VersionMax cannot be less than Upgrade.VersionMin. (OLDAPPFOUND)
```

Минимальная версия должна быть меньше максимальной версии.

Чтобы устранить эту ошибку, проверьте номера версий на наличие опечаток. Если они верны и вы хотите использовать диапазон между двумя версиями, переключите их так, чтобы Версионмин меньше Версионмакс.
