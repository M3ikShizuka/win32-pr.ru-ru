# <a name="replacedinusefiles-property"></a>Реплацединусефилес, свойство

Свойство **реплацединусефилес** задается, если программа установки записывает файл, который используется. Это свойство используется настраиваемыми действиями для обнаружения необходимости перезагрузки для завершения установки.

Задается во время действий [инсталлексекуте](installexecute-action.md) и [функции InstallFinalize](installfinalize-action.md) .
