# <a name="comments-summary-property"></a>Свойство сводки комментариев

Свойство **сводки комментариев** передает общее назначение пакета установки, преобразования или пакета исправлений.

Автор пакета установки, преобразования или пакета исправлений должен установить в качестве значения свойства **Summary Comment** одно из следующих значений:

<dl> "Эта база данных установщика содержит логику и данные, необходимые для установки <> *продукта* ".  
"Это преобразование содержит логику и данные, необходимые для установки <> *продукта* ".  
"Это исправление содержит логику и данные, необходимые для установки <> *продукта* ".  
</dl>

где <*product*> — имя продукта.
