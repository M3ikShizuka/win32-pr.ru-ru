# <a name="diskcost-dialog"></a>Диалоговое окно Дисккост

Диалоговое окно Дисккост обычно появляется, когда задано свойство [**аутофдискспаце**](outofdiskspace.md) . Это означает, что недостаточно места на диске для выбранной установки.

Это диалоговое окно можно также активировать из [диалогового окна выбора](selection-dialog.md) , чтобы предоставить пользователю обзор дискового пространства, необходимого для установки.

Диалоговое окно Дисккост является модальным полем диалогового окна. Обычно он содержит управляющую [кнопка](pushbutton-control.md) , которая возвращает пользователю предыдущее диалоговое окно и [элемент управления волумекостлист](volumecostlist-control.md).
