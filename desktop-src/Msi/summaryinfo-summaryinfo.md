# <a name="summaryinfoproperty-property"></a>Суммаринфо. Property, свойство

Свойство **Property** объекта [**суммаринфо**](summaryinfo-object.md) задает или получает значение для указанного свойства в информационном потоке сводки. Свойства считываются при создании объекта [**суммаринфо**](summaryinfo-object.md) , но не записываются, пока не будет вызван метод [**PERSISTED**](summaryinfo-persist.md) . Установка свойства в значение Empty приводит к его удалению; Аналогично, при запросе несуществующего свойства возвращается пустое значение. В противном случае значения могут передаваться как строки, целые числа или типы даты и времени.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = SummaryInfo.Property
```



## <a name="property-value"></a>Значение свойства

Обязательный идентификатор свойства одного из свойств сводки.
