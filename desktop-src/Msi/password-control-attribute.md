# <a name="password-control-attribute"></a>Атрибут элемента управления Password

Этот атрибут создает элемент управления изменением для ввода паролей. Элемент управления отображает каждый символ в виде звездочки ( \* ), так как они введены в элемент управления.

Установка атрибута элемента управления Password не даст установщику записать свойство, связанное с элементом управления "поле ввода", в файл журнала. Дополнительные сведения см. в разделе [предотвращение записи конфиденциальной информации в файл журнала](preventing-confidential-information-from-being-written-into-the-log-file.md).

## <a name="valid-controls"></a>Допустимые элементы управления

[Элемент управления "Поле ввода"](edit-control.md)

## <a name="associated-bit-flags"></a>Связанные битовые флаги

Этот атрибут не использует битовые флаги.
