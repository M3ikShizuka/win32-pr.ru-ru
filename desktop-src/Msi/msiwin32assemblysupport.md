# <a name="msiwin32assemblysupport-property"></a>MsiWin32AssemblySupport, свойство

Свойство **MsiWin32AssemblySupport** указывает, поддерживает ли компьютер сборки Win32. В системах, поддерживающих сборки Win32, установщик устанавливает значение **MsiWin32AssemblySupport** в качестве версии файла sxs.dll. Установщик не устанавливает это свойство, если операционная система не поддерживает сборки Win32. Дополнительные сведения см. в разделе [сборки](assemblies.md).
