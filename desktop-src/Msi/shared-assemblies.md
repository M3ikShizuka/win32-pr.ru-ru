# <a name="shared-assemblies"></a>Общие сборки

Сборка Win32 может быть установлена как общая сборка и доступна для использования несколькими приложениями на компьютере. Общие сборки должны устанавливаться пакетом установщик Windows, который используется для установки или обновления приложения.

Общие сборки устанавливаются как [Параллельные сборки](side-by-side-assemblies.md). Установщик Windows устанавливает общие параллельные сборки в папку WinSxS. Сборки не регистрируются глобально в системе, но они глобально доступны для любого приложения, которое указывает зависимость от сборки в файле манифеста. Дополнительные сведения см. в разделе [изолированные приложения и параллельные сборки](../sbscs/isolated-applications-and-side-by-side-assemblies-portal.md).

В операционных системах, предшествующих Windows XP, общие сборки обычно устанавливаются в системную папку Windows и регистрируются глобально. Последняя установленная версия доступна для любого приложения, которое выполняет привязку к ней.

Сведения о создании пакета установщик Windows для установки общей сборки см. в следующих статьях:

-   [Установка сборок Win32 для параллельного совместного использования в Windows XP](installing-win32-assemblies-for-side-by-side-sharing-on-windows-xp.md)
-   [Установка сборок Win32 для закрытого использования приложения в Windows XP](installing-win32-assemblies-for-the-private-use-of-an-application-on-windows-xp.md)
