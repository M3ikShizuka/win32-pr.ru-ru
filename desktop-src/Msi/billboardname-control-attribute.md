# <a name="billboardname-control-attribute"></a>Атрибут элемента управления Биллбоарднаме

Этот атрибут возвращает имя текущего активного объявления или задает и отображает объявление по имени. Этот атрибут используется в режиме предварительного просмотра.

## <a name="valid-controls"></a>Допустимые элементы управления

[Печат](billboard-control.md)

## <a name="associated-bit-flags"></a>Связанные битовые флаги

Этот атрибут не использует битовые флаги.

## <a name="remarks"></a>Комментарии

См. раздел [атрибуты элементов управления](control-attributes.md) и элемент управления, которые необходимо создать в элементах [управления](controls.md).
