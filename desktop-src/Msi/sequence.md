# <a name="sequence-property"></a>SEQUENCE, свойство

Свойство **Sequence** указывает таблицу, имеющую ту же схему, что и таблица инсталлексекутесекуенце, то есть столбцы действия, условия и последовательности. Это свойство используется [действием Sequence](sequence-action.md).

## <a name="remarks"></a>Remarks

Действие SEQUENCE выполняет действия в таблице в порядке, указанном в столбце Sequence таблицы.
