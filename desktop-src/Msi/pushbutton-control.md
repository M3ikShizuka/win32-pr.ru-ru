# <a name="pushbutton-control"></a>Кнопка элемента управления

Элемент управления "Кнопка" отображает базовую кнопку "Отправить", также известную как кнопка команды.

Элементы управления "Кнопка" не могут использоваться для публикации [Игноречанже таблице ControlEvent событие](ignorechange-controlevent.md), [селектиондескриптион таблице ControlEvent событие](selectiondescription-controlevent.md), [селектионсизе таблице ControlEvent событие](selectionsize-controlevent.md), [Селектионпас таблице ControlEvent событие](selectionpath-controlevent.md), [селектионпасон таблице ControlEvent событие](selectionpathon-controlevent.md), [селектионактион таблице ControlEvent событие](selectionaction-controlevent.md), [SelectionNoItems таблице ControlEvent событие](selectionnoitems-controlevent.md), [ActionText таблице ControlEvent событие](actiontext-controlevent.md), [ActionData таблице ControlEvent событие](actiondata-controlevent.md), [SetProgress таблице ControlEvent событие](setprogress-controlevent.md)или [TimeRemaining таблице ControlEvent событие](timeremaining-controlevent.md).

С этим элементом управления можно использовать следующие атрибуты. Чтобы изменить значение атрибута с помощью события, подпишите элемент управления на таблице ControlEvent событие в [таблице таблица eventmapping](eventmapping-table.md) и перечислите идентификатор атрибута в столбце Attribute.
