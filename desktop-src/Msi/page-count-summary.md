# <a name="page-count-summary-property"></a>Свойство сводки количества страниц

Свойство " **Сводка по количеству страниц** " содержит минимальную версию установщика, необходимую для пакета установки. Как минимум установщик Windows 2,0, этому свойству должно быть присвоено целое число 200. Как минимум установщик Windows 3,0, этому свойству должно быть присвоено целое число 300. Для минимума установщик Windows 3,1 это свойство должно иметь значение 301. Для минимума установщик Windows 4,5 это свойство должно иметь значение 405. Для минимума установщик Windows 5,0 это свойство должно иметь значение 500.

Для [64-разрядных пакетов установщик Windows](64-bit-windows-installer-packages.md)этому свойству должно быть присвоено целое число 200 или выше. Для 64-разрядных пакетов установщик Windows на платформе Arm64 этому свойству должно быть присвоено целое число 500 или выше.

Для пакета преобразования свойство **Сводка количества страниц** содержит минимальную версию установщика, необходимую для обработки преобразования. Задает большее из двух значений свойств **сводки количества страниц** , принадлежащих к базам данных, используемым для создания преобразования.

Для пакета исправлений свойство сводка по **количеству страниц** имеет значение null.

Это свойство сводки является обязательным.

Это свойство можно использовать для создания пакета, который может быть установлен только в указанной минимальной или более поздней версии установщик Windows.
