# <a name="enabled-control-attribute"></a>Включенный атрибут элемента управления

Этот атрибут указывает, включен или отключен данный элемент управления. Большинство элементов управления отображаются серым цветом при отключении.

Если этот бит задан, элемент управления создается как включенный.

Если этот бит не задан, элемент управления создается как отключенный.

## <a name="valid-controls"></a>Допустимые элементы управления

Все элементы управления. Установка этого атрибута не влияет на внешний вид некоторых элементов управления, которые не связаны со свойством, например точечными рисунками и значками.

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                          |
|---------|-------------|-----------------------------------|
| 2       | 0x00000002  | **мсидбконтролаттрибутесенаблед** |



 

## <a name="remarks"></a>Remarks

Можно также использовать [таблицу таблица controlcondition](controlcondition-table.md) для отключения или включения элемента управления в соответствии со значением свойства или условного оператора.

Можно также включить или отключить элемент управления, подписавший элемент управления на [таблице ControlEvent событие](control-events.md). Введите идентификатор атрибута в столбце атрибут и идентификатор события в столбце событие [таблицы таблица eventmapping](eventmapping-table.md).

См. раздел [атрибуты элементов управления](control-attributes.md) и элемент управления, которые необходимо создать в элементах [управления](controls.md).
