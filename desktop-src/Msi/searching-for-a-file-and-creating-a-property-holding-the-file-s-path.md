# <a name="searching-for-a-file-and-creating-a-property-holding-the-files-path"></a>Поиск файла и создание свойства, содержащего путь к файлу

**Поиск файла и создание свойства, содержащего путь к этому файлу**

1.  Сначала выполните поиск файла, указав подпись файла и имя в [таблице сигнатур](signature-table.md).

    Оставшиеся поля этой записи можно оставить пустыми, чтобы указать поиск любой версии MyApp.exe.

    [Таблица сигнатур](signature-table.md) (частичная)

    

    | Подпись          | Имя файла            |
    |--------------------|----------------------|
    | аппфиле<br/> | MyApp.exe<br/> |

    

     

2.  Затем укажите путь к файлу, в котором выполняется поиск, в [таблице дрлокатор](drlocator-table.md).

    Поскольку Аппфолдер не перечислен в [таблице сигнатур](signature-table.md), установщик определяет, что аппфолдер является папкой, а не файлом.

    [Таблица Дрлокатор](drlocator-table.md)

    

    | Подпись            | Parent             | Путь | Глубина |
    |----------------------|--------------------|------|-------|
    | аппфиле<br/>   |                    |      |       |
    | аппфолдер<br/> | аппфиле<br/> |      |       |

    

     

3.  Наконец, заполните [таблицу аппсеарч](appsearch-table.md) , чтобы [действие аппсеарч](appsearch-action.md) возвращало путь аппфолдер.

    После того как установщик выполнит действие Аппсеарч, значение MYFOLDER является полным путем к Аппфолдер.

    [Таблица аппсеарч](appsearch-table.md) (частичная)

    

    | Свойство            | Подпись            |
    |---------------------|----------------------|
    | MYFOLDER<br/> | аппфолдер<br/> |
