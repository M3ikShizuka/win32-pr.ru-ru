# <a name="standard-actions"></a>Стандартные действия

Действие инкапсулирует типичную функцию, выполняемую во время установки или обслуживания приложения. Установщик Windows имеет множество встроенных стандартных действий. Разработчики пакетов установки могут также создавать собственные [пользовательские действия](custom-actions.md).

Примеры стандартных действий установщика включают следующие.

-   [Действие креатешорткутс](createshortcuts-action.md): управляет созданием ярлыков для файлов, установленных с текущим приложением. Это действие использует [таблицу ярлыков](shortcut-table.md) для своих данных.
-   [Действие инсталлфилес](installfiles-action.md): копирует выбранные файлы из исходного каталога в целевой каталог. Это действие использует [таблицу File](file-table.md) для своих данных.
-   [Действие вритерегистривалуес](writeregistryvalues-action.md): настраивает сведения реестра, необходимые приложению в реестре. Это действие использует [таблицу реестра](registry-table.md) для своих данных.
