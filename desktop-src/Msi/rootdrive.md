# <a name="rootdrive-property"></a>РУТДРИВЕ, свойство

Свойство **рутдриве** указывает диск по умолчанию для целевого каталога установки. Если столбец каталога в [таблице Directory](directory-table.md) указывает корневой каталог назначения по имени свойства, которое не определено, установщик использует значение свойства **рутдриве** для разрешения пути к целевому каталогу.

Если **рутдриве** не задан в командной строке или создан в [таблице свойств](property-table.md), установщик устанавливает это свойство. Во время [административной установки](administrative-installation.md) установщик устанавливает **рутдриве** в первый подключенный сетевой диск, который может быть записан в. Если это не административная установка или если установщик не может найти сетевые диски, установщик устанавливает **рутдриве** на локальный диск, на который может быть помещено наибольшее свободное пространство.

Значение этого свойства должно заканчиваться на " \\ ".
