# <a name="indirectpropertyname-control-attribute"></a>Атрибут элемента управления Индиректпропертинаме

Этот атрибут является именем косвенного свойства, связанного с элементом управления. Если задан [непрямой](indirect-control-attribute.md) бит, элемент управления отображает или изменяет значение свойства с этим именем, а это значение является значением свойства, перечисленного в столбце свойств [таблицы Control](control-table.md).

Для элементов управления инерт этот атрибут возвращает значение null.

## <a name="valid-controls"></a>Допустимые элементы управления

Все активные элементы управления.

## <a name="associated-bit-flags"></a>Связанные битовые флаги

Этот атрибут не использует битовые флаги.

## <a name="remarks"></a>Remarks

См. раздел [атрибуты элементов управления](control-attributes.md) и элемент управления, которые необходимо создать в элементах [управления](controls.md).
