# <a name="patching-customized-applications"></a>Исправление настроенных приложений

При установке исправления и одного или нескольких пользовательских преобразований в приложение сначала устанавливается исправление, а затем преобразования настройки. По проекту это исправление не нарушается последующей установкой настройки. Однако сначала установка преобразований, а затем исправление, может нарушить настройку.

Например, перерыв в настройке может произойти, когда исправление используется для обновления продукта с версии 1 до версии 2, а преобразование настройки, которое работает для версии 1, не работает в версии 2. В этом случае обновление версии не может быть применено к настроенному продукту без предварительного удаления и повторной установки исходного продукта.
