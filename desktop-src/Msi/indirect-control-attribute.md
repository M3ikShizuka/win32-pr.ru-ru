# <a name="indirect-control-attribute"></a>Косвенный атрибут элемента управления

Атрибут косвенного элемента управления указывает, имеется ли ссылка косвенно на значение, отображаемое или изменяемое этим элементом управления. Если этот бит задан, элемент управления отображает или изменяет значение свойства с идентификатором, указанным в столбце свойства [таблицы Control](control-table.md). Если этот бит не задан, элемент управления отображает или изменяет значение свойства в столбце свойств таблицы управления.

## <a name="valid-controls"></a>Допустимые элементы управления

Все активные элементы управления.

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                           |
|---------|-------------|------------------------------------|
| 8       | 0x00000008  | **мсидбконтролаттрибутесиндирект** |

См. раздел [атрибуты элементов управления](control-attributes.md) и элемент управления, которые необходимо создать в элементах [управления](controls.md).
