# <a name="error-object"></a>Объект ошибки

Объект **Error** возвращает сведения об одной ошибке слияния.

## <a name="members"></a>Элементы

Объект **Error** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Элемент Property

Объект **Error** имеет следующие свойства.



| Свойство                                                | Описание                                                                                 |
|:--------------------------------------------------------|:--------------------------------------------------------------------------------------------|
| [**датабасекэйс**](error-databasekeys.md)<br/>   | Возвращает первичные ключи строки в таблице базы данных, вызвавшей ошибку.<br/> |
| [**датабасетабле**](error-databasetable.md)<br/> | Возвращает имя таблицы в базе данных, вызвавшей ошибку.<br/>           |
| [**Язык**](error-language.md)<br/>           | Возврат языка ошибки.<br/>                                                |
| [**модулекэйс**](error-modulekeys.md)<br/>       | Возвращает первичные ключи строки в таблице Module, вызвавшие ошибку.<br/>   |
| [**модулетабле**](error-moduletable.md)<br/>     | Возвращает имя таблицы в модуле, вызвавшем ошибку.<br/>             |
| [**Путь**](error-path.md)<br/>                   | Возврат пути к файлу или каталогу, вызвавшему ошибку. В настоящее время не используется.<br/>    |
| [**Тип**](error-type.md)<br/>                   | Возврат типа ошибки.<br/>                                                        |
