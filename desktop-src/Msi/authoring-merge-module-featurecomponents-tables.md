# <a name="authoring-merge-module-featurecomponents-tables"></a>Создание таблиц Феатурекомпонентс для модуля слияния

В каждом модуле слияния требуется пустая [Таблица феатурекомпонентс](featurecomponents-table.md) . Эта пустая таблица содержит схему для средства слияния, если в целевом MSI-файле отсутствует собственная таблица Феатурекомпонентс.

Если, и только если, в целевом MSI-файле отсутствует таблица Феатурекомпонентс, то средство слияния использует пустую таблицу, указанную в модуле. В этом случае средство слияния создает повторяющуюся таблицу в файле Target. msi и добавляет строки, связывающие новые компоненты в модуле слияния с компонентами пакета установки приложения.
