# <a name="authoring-merge-module-featurecomponents-tables"></a>Создание таблиц Феатурекомпонентс для модуля слияния

В каждом модуле слияния требуется пустая [Таблица феатурекомпонентс](featurecomponents-table.md) . Эта пустая таблица содержит схему для средства слияния, если целевой файл .msi не имеет собственной таблицы Феатурекомпонентс.

Если, и только в случае, в целевом .msi нет таблицы Феатурекомпонентс, в которой средство слияния использует пустую таблицу, указанную в модуле. В этом случае средство слияния создает повторяющуюся таблицу в целевом .msi файле и добавляет строки, связывающие новые компоненты в модуле слияния с компонентами пакета установки приложения.
