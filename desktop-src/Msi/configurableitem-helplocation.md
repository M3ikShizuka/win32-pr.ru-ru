# <a name="configurableitemhelplocation-property"></a>Конфигураблеитем. HelpLocation, свойство

Свойство **HelpLocation** возвращает значение из столбца HelpLocation [таблицы модулеконфигуратион](moduleconfiguration-table.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = ConfigurableItem.HelpLocation
```



## <a name="property-value"></a>Значение свойства

## <a name="c"></a>C++

См. раздел [**Получение функции \_ HelpLocation**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_helplocation) .
