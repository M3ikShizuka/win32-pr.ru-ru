# <a name="installing-permanent-components-files-fonts-registry-keys"></a>Установка постоянных компонентов, файлов, шрифтов, разделов реестра

Чтобы установить файл, шрифт или раздел реестра, чтобы он не был удален при удалении продукта, необходимо сделать все компоненты, содержащие файл, шрифт или раздел реестра, постоянными. Чтобы сделать компонент постоянным, установите **мсидбкомпонентаттрибутесперманент** в столбце Attributes [таблицы Component](component-table.md).

Обратите внимание, что установщик удаляет раздел реестра после удаления последнего значения или подраздела раздела. Чтобы предотвратить удаление пустого раздела реестра при удалении, напишите фиктивное значение под ключом, который необходимо сохранить, и введите + в столбце имя [таблицы реестра](registry-table.md).
