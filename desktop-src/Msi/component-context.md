# <a name="componentcontext-property"></a>Свойство Component. Context

Контекст, который был определен как применимый к рассматриваемому компоненту. Это свойство вызывает функцию [**мсиенумкомпонентсекс**](/windows/desktop/api/Msi/nf-msi-msienumcomponentsexa) .

**[Установщик Windows 4,5 или более ранней версии](not-supported-in-windows-installer-4-5.md):** Не поддерживается. Это свойство доступно, начиная с установщик Windows 5,0.
