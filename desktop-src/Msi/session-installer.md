# <a name="sessioninstaller-property"></a>Свойство Session. Installer

Свойство **установщика** только для чтения объекта [**Session**](session-object.md) Возвращает активный объект **установщика** .

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Session.Installer
```
