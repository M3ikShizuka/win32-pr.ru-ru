# <a name="remoteadmints-property"></a>Ремотеадминтс, свойство

установщик задает свойство **ремотеадминтс** , когда система настроена для удаленного администрирования с помощью служб Терминал Windows. Обратите внимание, что в этом случае установщик не устанавливает свойство [**терминалсервер**](terminalserver.md) .

## <a name="default-value"></a>Значение по умолчанию

Отсутствует. Свойство не определено, если не работает на сервере удаленного администрирования со службами терминалов.
