# <a name="remoteadmints-property"></a>Ремотеадминтс, свойство

Установщик задает свойство **ремотеадминтс** , когда система настроена для удаленного администрирования с помощью служб терминалов Windows. Обратите внимание, что в этом случае установщик не устанавливает свойство [**терминалсервер**](terminalserver.md) .

## <a name="default-value"></a>Значение по умолчанию

Нет. Свойство не определено, если не работает на сервере удаленного администрирования со службами терминалов.

Установщик Windows 4,0 или установщик Windows 4,5 на Windows Server 2008 или Windows Vista. Установщик Windows в Windows Server 2003 или Windows XP. Сведения о минимальном пакете обновления Windows, который требуется для установщик Windows версии, см. в [установщик Windows Run-Time требования](windows-installer-portal.md) .<br/> |
