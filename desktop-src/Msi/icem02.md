# <a name="icem02"></a>ICEM02

ICEM02 проверяет, что все зависимости и исключения модуля связаны с текущим модулем.

Модуль слияния ICEs хранится в файле слияния Module. CUB, который называется Мержемод. CUB, а не в файле. CUB, содержащем значение ICEs, используемое для проверки пакета.

## <a name="result"></a>Результат

ICEM02 отправляет сообщения об ошибках, если база данных модуля пытается указать зависимости или исключения, которые не связаны с текущим модулем. ICEM02 отправляет сообщение об ошибке, если база данных модуля пытается указать текущий модуль как зависимый или исключенный самим.

## <a name="example"></a>Пример

ICEM02 будет размещать следующие сообщения об ошибках для модуля, содержащего записи базы данных, показанные ниже.

``` syntax
The dependency OtherModule.GUID2.1033.OtherModule.GUID3.0 in the 
ModuleDependency table creates a dependency for an unrelated module. A 
module can only define dependencies for itself

This module is listed as depending on itself!

The exclusion OtherModule.GUID2.1033.OtherModule.GUID3.0 in the 
ModuleExclusion table creates an excluded module for an unrelated 
module. A module can only define exclusions for itself.

This module excludes itself from the target database!
```

[Таблица ModuleSignature](modulesignature-table.md)



| ModuleID         | Язык | Версия |
|------------------|----------|---------|
| MyModule. *GUID1* | 1033     | 1.0     |



 

[Таблица Модуледепенденци](moduledependency-table.md)



| ModuleID            | модулелангуаже | рекуиредид          | рекуиредлангуаже | RequiredVersion |
|---------------------|----------------|---------------------|------------------|-----------------|
| Осермодуле. *GUID2* | 1033           | Осермодуле. *GUID3* | 0                | 1.0             |
| MyModule. *GUID1*    | 1033           | MyModule. *GUID1*    | 1033             | 1.2             |



 

[Таблица модуликсклусион](moduleexclusion-table.md) (частичная)



| ModuleID            | модулелангуаже | ексклудедид          | ексклудедлангуаже |
|---------------------|----------------|---------------------|------------------|
| Осермодуле. *GUID2* | 1033           | Осермодуле. *GUID3* | 0                |
| MyModule. *GUID1*    | 1033           | MyModule. *GUID1*    | 1033             |



 

Модуль слияния ICE публикует первую ошибку из-за первой строки в таблице Модуледепенденци, которая не указывает требуемую зависимость для текущего модуля, указанного в таблице ModuleSignature. Зависимости модуля можно указывать только в собственной таблице Модуледепенденци. Если Осермодуле. *GUID3* является обязательным для текущего модуля, замените первые два столбца строки данными из таблицы ModuleSignature. Если Осермодуле. *GUID3* не требуется для этого модуля, удалите эту строку.

Модуль слияния ICE отправляет вторую ошибку, так как модуль не может задать зависимость от самого себя.

Модуль слияния ICE публикует третью ошибку из-за первой строки в таблице Модуликсклусион, которая не указывает требуемое исключение для текущего модуля, указанного в таблице ModuleSignature. Исключения модуля могут быть указаны только в собственной таблице Модуликсклусион. Если текущий модуль исключает Осермодуле. *GUID3* замените первые два столбца строки данными из таблицы ModuleSignature. Если текущий модуль не исключает Осермодуле. *GUID3*, удалите эту строку.

Модуль слияния ICE отправляет четвертую ошибку, так как модуль не может указать, что он исключается.
