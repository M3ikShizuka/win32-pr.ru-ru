# <a name="summaryinfopropertycount-property"></a>Суммаринфо. Пропертикаунт, свойство

Свойство **пропертикаунт** объекта [**суммаринфо**](summaryinfo-object.md) является свойством только для чтения, указывающим текущее количество значений свойств в объекте сводных данных. Он учитывает свойства учетной записи, которые были добавлены, удалены или заменены.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = SummaryInfo.PropertyCount
```
