# <a name="nousername-property"></a>Свойство USERNAME

Присвойте свойству " **username** " значение 1, чтобы запретить установщику установку свойства [**username**](username.md) . Это свойство используется приложениями, которым необходимо получить имя пользователя при первом запуске.

## <a name="default-value"></a>Значение по умолчанию

Не задано. Установщик устанавливает свойство [**username**](username.md) автоматически, используя значения из реестра.
