# <a name="error-message-custom-actions"></a>Пользовательские действия в сообщении об ошибке

Этот тип настраиваемого действия отображает указанное сообщение об ошибке и возвращает ошибку, завершая установку. Отображаемое сообщение об ошибке может быть представлено в виде строки или индекса в [таблице ошибок](error-table.md).



| Тип настраиваемого действия                                 | Описание                                                |
|----------------------------------------------------|------------------------------------------------------------|
| [Тип настраиваемого действия 19](custom-action-type-19.md) | Завершает установку и отображает сообщение об ошибке. |



 

Сведения о том, как кодировать каждый из типов пользовательских действий в [таблицу CustomAction](customaction-table.md), см. в разделе [сводный список всех типов пользовательских действий](summary-list-of-all-custom-action-types.md) .
