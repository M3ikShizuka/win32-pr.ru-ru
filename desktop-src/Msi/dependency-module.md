# <a name="dependencymodule-property"></a>Свойство зависимости. Module

Свойство **модуля** только для чтения [**объекта зависимости**](dependency-object.md) возвращает ModuleID модуля, который необходим текущей строке в форме **BSTR**. Этот ModuleID является той же формой, которая используется в [таблице ModuleSignature](modulesignature-table.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Dependency.Module
```
