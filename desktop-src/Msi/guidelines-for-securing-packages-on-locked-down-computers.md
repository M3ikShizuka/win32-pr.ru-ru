# <a name="securing-packages-on-locked-down-computers"></a>Защита пакетов на заблокированных компьютерах

Соблюдение следующих рекомендаций при создании установщик Windows пакета, который будет использоваться на заблокированных компьютерах, помогает поддерживать безопасную среду во время установки:

-   Проверьте пакет на совместимость с [политикой системы](system-policy.md)установщик Windows машины.
-   Убедитесь, что пакет выполняется на всех [уровнях пользовательского интерфейса](user-interface-levels.md): None, Basic, Limited и Full.
-   Протестируйте пакет в разделах NTFS с [*повышенными*](e-gly.md) правами и без повышенных привилегий.
-   Начиная с установщик Windows 3,0, [исправление контроля учетных записей пользователей (UAC)](user-account-control--uac--patching.md) позволяет пользователям без прав администратора устанавливать исправления в приложениях, установленных в контексте на компьютере. Протестируйте пакет исправлений в Windows Vista и Windows XP для установки пользователями с доступом администратора и пользователями, не являющимися администраторами.
