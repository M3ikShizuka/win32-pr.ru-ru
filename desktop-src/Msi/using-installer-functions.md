# <a name="using-installer-functions"></a>Использование функций установщика

Чтобы добавить функциональные возможности установщик Windows в приложение, используйте функции службы установщика. Поскольку установщик является программой установки и системой управления компонентами, приложение, использующее установщик, должно включать функции службы установщика в рамках нормальной работы.

-   [Переустановка компонента или приложения](reinstalling-a-feature-or-application.md)
