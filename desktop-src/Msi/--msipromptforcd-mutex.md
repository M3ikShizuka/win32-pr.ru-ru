# <a name="__msipromptforcd-mutex"></a>\_\_Мьютекс Мсипромптфоркд

\_ \_ Мьютекс мсипромптфоркд существует, когда установщик предлагает пользователю вставить компакт-диск. Программы автозапуска должны проверять, \_ \_ не задан ли в данный момент мьютекс мсипромптфоркд перед запуском. Обратите внимание, что можно выполнить запрос к дисководу компакт-дисков перед тем, как будет существовать ключ выполнения или \_ мьютекс мсиексекуте.
