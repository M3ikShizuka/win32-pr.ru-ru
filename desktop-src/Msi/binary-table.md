# <a name="binary-table"></a>Двоичная таблица

Двоичная таблица содержит двоичные данные для таких элементов, как точечные рисунки, анимации и значки. Двоичная таблица также используется для хранения данных настраиваемых действий. См. раздел [ограничения OLE для потоков](ole-limitations-on-streams.md).

Двоичная таблица содержит следующие столбцы.



| Столбец | Type                         | Ключ | Допускает значения NULL |
|--------|------------------------------|-----|----------|
| Имя   | [Идентификатор](identifier.md) | Да   | Нет        |
| Данные   | [Двоичный](binary.md)         | Нет   | Нет        |



 

## <a name="columns"></a>Столбцы

<dl> <dt>

<span id="Name"></span><span id="name"></span><span id="NAME"></span>Безымян
</dt> <dd>

Уникальный ключ, идентифицирующий определенные двоичные данные. Если двоичные данные относятся к элементу управления, ключ отображается в столбце text связанного элемента управления в [таблице управления](control-table.md). Этот ключ должен быть уникальным для всех элементов управления, для которых требуются двоичные данные.

</dd> <dt>

<span id="Data"></span><span id="data"></span><span id="DATA"></span>Data
</dt> <dd>

Неформатированные двоичные данные.

</dd> </dl>
