# <a name="applying-a-configurable-merge-module-with-customizations"></a>Применение настраиваемого модуля слияния с настройками

Следуйте общим рекомендациям по применению модулей слияния, описанным в разделе [применение модулей слияния](applying-merge-modules.md). Кроме того, потребители модулей слияния должны выполнить следующие действия, чтобы настроить модуль слияния во время установки.

-   Используйте модуль слияния, который создан с учетом настраиваемых параметров. Дополнительные сведения см. [в разделе Создание модуля слияния, который может быть настроен конечным пользователем](creating-a-merge-module-that-can-be-configured-by-the-end-user.md) .
-   Используйте средство слияния и настройки, которое вызывает Mergemod.dll версии 2,0 или более поздней. Более ранние версии Mergmod.dll не могут настраивать параметры модуля слияния. Авторы должны создавать модули слияния, предоставляющие значения по умолчанию и совместимые с более ранними версиями Mergmod.dll.
-   При появлении запроса средства клиента конфигурации укажите сведения о настройке. Авторы должны создавать модули слияния, которые используют значения по умолчанию, когда пользователь отклоняет предоставление информации.
