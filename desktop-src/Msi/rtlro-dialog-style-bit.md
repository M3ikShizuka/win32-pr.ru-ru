# <a name="rtlro-dialog-style-bit"></a>Бит стиля диалогового окна РТЛРО

Если этот бит стиля установлен, текст в диалоговом окне отображается в порядке чтения справа налево.

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                       |
|---------|-------------|--------------------------------|
| 128     | 0x00000080  | **мсидбдиалогаттрибутесртлро** |
