# <a name="productcode-property"></a>ProductCode, свойство

Свойство **ProductCode** — это уникальный идентификатор для конкретного выпуска продукта, представленный в виде строкового [GUID](guid.md), например " {12345678-1234-1234-1234-123456789012} ". Буквы, используемые в этом GUID, должны быть прописными буквами. Этот идентификатор должен различаться для разных версий и языков.

Обновление продукта, которое обновляет продукт в совершенно новом продукте, также должно изменить код продукта. В 32-разрядных и 64-разрядных версиях пакета приложения должны быть назначены разные коды продуктов.

**Код продукта** объявляется как свойство продукта, и является основным методом указания продукта. Это свойство является обязательным.
