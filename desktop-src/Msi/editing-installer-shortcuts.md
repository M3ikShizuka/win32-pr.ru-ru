# <a name="editing-installer-shortcuts"></a>Изменение ярлыков установщика

В [таблице ярлыков](shortcut-table.md) содержатся сведения, необходимые приложению для создания ярлыков установщика на компьютере пользователя. Ярлыки установщика поддерживают [установку по запросу](installation-on-demand.md) и [объявления](advertisement.md). Обратите внимание, что вы не можете изменять свойства ярлыков установщика, как и в случае с обычными сочетаниями клавиш.

Установка свойства [**дисаблеадвтшорткутс**](disableadvtshortcuts.md) отключает создание ярлыков установщика и указывает, что эти сочетания клавиш следует заменить обычными сочетаниями клавиш.
