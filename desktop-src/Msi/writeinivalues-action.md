# <a name="writeinivalues-action"></a>Действие Вритеинивалуес

Действие Вритеинивалуес записывает сведения о файле .ini, которые должны быть записаны приложением в свои .ini файлы. Запись этих сведений наследуется [таблицей Component](component-table.md). Значение .ini записывается, если соответствующий компонент настроен для установки локально или выполняется из источника.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие Инсталлвалидате должно следовать перед действием Вритеинивалуес. Если в последовательности есть [действие ремовеинивалуес](removeinivalues-action.md) , оно должно быть перед действием вритеинивалуес.

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия              |
|-------|-----------------------------------------|
| \[1\] | Идентификатор файла .ini.                |
| \[2\] | .ini ключ файла в следующем разделе. |
| \[3\] | Элемент удален из файла .ini.            |
| \[4\] | Значение, удаленное из файла .ini.           |
