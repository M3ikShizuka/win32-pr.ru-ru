# <a name="writeinivalues-action"></a>Действие Вритеинивалуес

Действие Вритеинивалуес записывает сведения о ini-файле, которые должны быть записаны приложением в свои ini-файлы. Запись этих сведений наследуется [таблицей Component](component-table.md). Значение ini-файлов записывается, если соответствующий компонент настроен для установки локально или выполняется из источника.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие Инсталлвалидате должно следовать перед действием Вритеинивалуес. Если в последовательности есть [действие ремовеинивалуес](removeinivalues-action.md) , оно должно быть перед действием вритеинивалуес.

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия              |
|-------|-----------------------------------------|
| \[1\] | Идентификатор ini-файла.                |
| \[2\] | INI-файл в следующем разделе. |
| \[3\] | Элемент удален из ini-файла.            |
| \[4\] | Значение, удаленное из ini-файла.           |
