# <a name="installerpatchtransforms-property"></a>Свойство Installer. Патчтрансформс

Свойство **патчтрансформс** только для чтения возвращает список преобразований, разделенных точкой с запятой, которые находятся в указанном пакете исправлений и применяются к указанному продукту.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Installer.PatchTransforms
```



## <a name="property-value"></a>Значение свойства
