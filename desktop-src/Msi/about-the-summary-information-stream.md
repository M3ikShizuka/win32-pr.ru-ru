# <a name="about-the-summary-information-stream"></a>Сведения о потоке сводных данных

информационный поток сводки содержит сведения о пакете, который можно просмотреть с помощью Microsoft Windows Explorer. Эти сведения доступны через интерфейс **IStream** . Автор должен предоставить значения для каждого из этих свойств.

Информационный поток сводки использует COM для создания структурированного хранилища баз данных. Модель COM поддерживает понятие структурированного хранилища, доступного через интерфейс **IStream** . Структурированное хранилище, в свою очередь, поддерживает концепцию наборов свойств в качестве гибкого метода сериализации практически любой информации. спецификация COM определяет один стандартный набор свойств, сводную информацию, которая используется для заполнения страниц свойств, видимых в Windows Explorer. Таким образом, данные, хранящиеся в потоке сводных данных, могут быть просмотрены пользователями, когда они правой кнопкой мыши щелкнули базу данных установщика или преобразовывают и выбрали пункт [Свойства](about-properties.md).

Список набора свойств сводки см. в разделе [набор свойств потока сводных данных](summary-information-stream-property-set.md).

Краткие описания свойств сводных данных, используемых в базах, преобразованиях и исправлениях, см. в разделе [описания свойств сводки](summary-property-descriptions.md).
