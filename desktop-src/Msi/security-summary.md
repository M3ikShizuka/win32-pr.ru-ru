# <a name="security-summary-property"></a>Свойство сводки безопасности

Свойство **сводки безопасности передает сведения о** том, следует ли открывать пакет только для чтения. Средство редактирования базы данных не должно изменять принудительно доступную только для чтения базу данных и выдавать предупреждение при попытке изменить базу данных, доступную только для чтения. Следующие значения этого свойства применимы к установщик Windows файлам.



| Значение | Описание           |
|-------|-----------------------|
| 0     | Без ограничений        |
| 2     | Рекомендуется только для чтения |
| 4     | Принудительно применяется только для чтения    |



 

Это свойство должно иметь значение рекомендуется только для чтения (2) для базы данных установки и принудительно использовать только для чтения (4) для преобразования или исправления.
