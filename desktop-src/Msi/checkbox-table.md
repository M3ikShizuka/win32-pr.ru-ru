# <a name="checkbox-table"></a>Таблица флажков

В таблице CheckBox перечислены значения флажков.

Таблица CheckBox содержит следующие столбцы.



| Столбец   | Type                         | Ключ | Допускает значения NULL |
|----------|------------------------------|-----|----------|
| Свойство | [Идентификатор](identifier.md) | Да   | Нет        |
| Значение    | [Формате](formatted.md)   | Нет   | Да        |



 

## <a name="columns"></a>Столбцы

<dl> <dt>

<span id="Property"></span><span id="property"></span><span id="PROPERTY"></span>Свойства
</dt> <dd>

Именованное свойство, которое должно быть привязано к этому элементу.

</dd> <dt>

<span id="Value"></span><span id="value"></span><span id="VALUE"></span>Значений
</dt> <dd>

Строка значения, связанная с этим элементом.

</dd> </dl>

## <a name="remarks"></a>Комментарии

Если установлен флажок, то соответствующее свойство устанавливается в указанное значение. Если значение не указано или таблица не существует, свойство устанавливается в исходное значение, если установлен флажок. Если исходное значение равно null, свойство имеет значение 1.

Содержимое столбца значений форматируется функцией [**мсиформатрекорд**](/windows/desktop/api/Msiquery/nf-msiquery-msiformatrecorda) при создании элемента управления. Поэтому он может содержать любое выражение, которое может интерпретировать функция **мсиформатрекорд** . Столбец значение форматируется только при создании элемента управления и не обновляется, если свойство, вовлеченное в выражение, изменяется в течение жизненного цикла элемента управления.
