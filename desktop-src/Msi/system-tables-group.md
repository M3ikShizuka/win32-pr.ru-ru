# <a name="system-tables-group"></a>Группа системных таблиц

Таблицы в группе системных таблиц следят за таблицами и столбцами базы данных установки.

-   [ \_ Таблица](-tables-table.md) Tables отслеживает все таблицы в базе данных. Сюда входят таблицы, которые могли быть созданы для собственных настраиваемых действий. Запросите эту таблицу, чтобы узнать, существует ли таблица.
-   [ \_ Таблица Columns](-columns-table.md) отслеживает столбцы в базе данных установки. Временные столбцы в настоящее время не записываются этой таблицей. Запросите эту таблицу, чтобы узнать, существует ли данный столбец.
-   в [ \_ Потоки таблице](-streams-table.md) перечислены внедренные потоки данных OLE.
-   В [ \_ таблице хранения](-storages-table.md) перечислены внедренные хранилища данных OLE.
-   [ \_ Таблица проверки](-validation-table.md). \_Таблица проверки отслеживает типы и допустимые диапазоны каждого столбца в базе данных. \_Таблица проверки используется в процессе проверки базы данных, чтобы убедиться в том, что все столбцы включены в учет и имеют правильные значения. Эта таблица не поставляется с базой данных установщика.
