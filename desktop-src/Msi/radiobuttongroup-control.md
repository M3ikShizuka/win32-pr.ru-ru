# <a name="radiobuttongroup-control"></a>Элемент управления Радиобуттонграуп

Элемент управления Радиобуттонграуп — это группа переключателей. Он позволяет пользователю выбрать строковое или целочисленное значение для свойства из предварительно определенного списка значений. Этот элемент управления можно связать со свойством, введя имя свойства в столбец Property [таблицы Control](control-table.md). Укажите возможные значения для выбора в столбце значение [таблицы RadioButton](radiobutton-table.md). Обратите внимание, что отображаемая строка не обязательно соответствует значению, выбранному пользователем.

Каждый элемент управления Радиобуттонграуп связан со свойством. Значение по умолчанию для этого свойства должно быть инициализировано в [таблице свойств](property-table.md). В каждом Радиобуттонграуп, указанном в [таблице RadioButton](radiobutton-table.md), может быть один переключатель со значением в поле значение, которое соответствует значению по умолчанию для этого свойства. Это кнопка по умолчанию для элемента управления Радиобуттонграуп. Кнопка **по умолчанию** изначально отображается как выбранная в элементе управления.

Обратите внимание, что фокус на диалоговом окне не может перемещаться в элемент управления Радиобуттонграуп до тех пор, пока не будет выбрана одна из кнопок в группе. Чтобы фокус переместится в эту группу кнопок, укажите одну из кнопок в качестве кнопки по умолчанию для группы.

Элементы управления Радиобуттонграуп задают только значения свойств и не могут использоваться для отправки [таблице ControlEvent событие](control-events.md).

Реализация рассматривает всю группу как один элемент управления, поэтому невозможно скрыть или отключить отдельные кнопки в группе. Аналогично, все кнопки должны иметь один и тот же стиль, то есть все они имеют текст или все они имеют точечные рисунки (или другие аналогичные функции). Положения кнопок создаются относительно группы. Таким образом можно перемещать всю группу, изменяя только координаты группы без изменения отдельных кнопок. При создании элемента управления проверяется, что отдельные кнопки не выходят за границы группы.

## <a name="remarks"></a>Remarks

поскольку Windows рисует рамку, между верхней границей окна элемента управления и видимым кадром возникает разрыв, даже если заголовок отсутствует.

Этот элемент управления можно создать из класса BUTTON с помощью функции [**CreateWindowEx**](/windows/win32/api/winuser/nf-winuser-createwindowexa) . Если задан бит [хасбордер](hasborder-control-attribute.md) , он имеет стиль **\_ GROUPBOX** , в противном случае он имеет стиль **BS \_ овнердрав** .

Элемент управления Радиобуттонграуп не должен перекрывать другие элементы управления, а другие элементы управления не должны перекрываться с Радиобуттонграуп. Перекрытие этого элемента управления и другого может привести к неправильному функционированию или отображению элементов управления. Чтобы программы чтения с экрана, содержащие дополнительный описательный текст о Радиобуттонграуп элементе управления, были доступны в примере, приведенном в статье [Добавление дополнительного текста к переключателям](adding-extra-text-to-radio-buttons.md).
