# <a name="sessionproductproperty-property"></a>Свойство Session. Продуктпроперти

Свойство **продуктпроперти** объекта [**Session**](session-object.md) является свойством, предназначенным только для чтения и получающим свойства продукта из базы данных продукта.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Session.ProductProperty
```



## <a name="property-value"></a>Значение свойства

Обязательное имя свойства с учетом регистра.
