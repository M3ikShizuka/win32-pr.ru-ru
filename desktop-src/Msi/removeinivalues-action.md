# <a name="removeinivalues-action"></a>Действие Ремовеинивалуес

Действие Ремовеинивалуес удаляет .ini сведения о файле, указанные для удаления в [таблице ремовеинифиле](removeinifile-table.md) , если компонент настроен для установки локально или запуска от исходного кода. Действие Ремовеинивалуес удаляет сведения о файле .ini, которые были связаны с компонентом в [таблице инифиле](inifile-table.md). Это действие также удаляет сведения о файле .ini, если данные были записаны [действием вритеинивалуес](writeinivalues-action.md) и запланировано удаление компонента.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие [инсталлвалидате](installvalidate-action.md) должно быть вызвано перед действием ремовеинивалуес. Если в последовательности используется действие [вритеинивалуес](writeinivalues-action.md) , оно должно появиться после ремовеинивалуес.

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия    |
|-------|-------------------------------|
| \[1\] | Идентификатор файла .ini.      |
| \[2\] | Раздел .iniного ключа файла.     |
| \[3\] | Элемент удален из файла .ini.  |
| \[4\] | Значение, удаленное из файла .ini. |
