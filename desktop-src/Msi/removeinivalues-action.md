# <a name="removeinivalues-action"></a>Действие Ремовеинивалуес

Действие Ремовеинивалуес удаляет сведения из ini-файла, указанные для удаления в [таблице ремовеинифиле](removeinifile-table.md) , если компонент настроен на локальную установку или запуск из исходного кода. Действие Ремовеинивалуес удаляет данные ini-файла, связанные с компонентом в [таблице инифиле](inifile-table.md). Это действие также удаляет сведения из ini-файла, если данные были записаны [действием вритеинивалуес](writeinivalues-action.md) и запланировано удаление компонента.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие [инсталлвалидате](installvalidate-action.md) должно быть вызвано перед действием ремовеинивалуес. Если в последовательности используется действие [вритеинивалуес](writeinivalues-action.md) , оно должно появиться после ремовеинивалуес.

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия    |
|-------|-------------------------------|
| \[1\] | Идентификатор ini-файла.      |
| \[2\] | Раздел Key. ini-файла.     |
| \[3\] | Элемент удален из ini-файла.  |
| \[4\] | Значение, удаленное из ini-файла. |
