# <a name="searching-for-a-directory-and-a-file-in-the-directory"></a>Поиск каталога и файла в каталоге

**Поиск каталога и файла в этом каталоге**

1.  Сначала найдите каталог.

    AppDir должен быть определен как действительная подпись каталога. Если AppDir не определен как действительная сигнатура, Аппсеарч не имеет места для поиска файла, например, если поиск выполняется в c: \\ MyDir \\MyApp.exe, AppDir должен быть определен как c: \\ MyDir. AppDir может быть определен путем включения записи в [таблицу дрлокатор](drlocator-table.md)или другим методом. Ни одна запись не включена в [таблицу сигнатур](signature-table.md) для поиска в каталоге. Для поиска файлов перечислите подпись файла и имя в таблице сигнатур. Оставшиеся поля в этой записи могут иметь значение NULL для поиска любой версии MyApp.exe.

    [Таблица сигнатур](signature-table.md) (частичная)

    

    | Подпись          | Имя файла            |
    |--------------------|----------------------|
    | аппфиле<br/> | MyApp.exe<br/> |

    

     

2.  Используйте [таблицу аппсеарч](appsearch-table.md).

    Введите свойство, устанавливаемое установщиком при установке каталога с подписью AppDir. Если установщик обнаруживает, что этот каталог установлен, он задает для MYDIR путь к каталогу. Введите свойство, устанавливаемое установщиком, если MyApp.exe установлен.

    [Таблица аппсеарч](appsearch-table.md) (частичная)

    

    | Свойство         | Подпись          |
    |------------------|--------------------|
    | MYDIR<br/> | AppDir<br/>  |
    | MYAPP<br/> | аппфиле<br/> |

    

     

3.  Используйте [таблицу дрлокатор](drlocator-table.md).

    Введите в родительском столбце сигнатуру AppDir, которая определена как путь к каталогу. Укажите в столбце Depth число уровней подкаталогов для поиска в этом каталоге. AppDir должен быть определен как подпись каталога. AppDir можно определить, включив запись, как показано ниже, или с помощью другого метода.

    [Таблица Дрлокатор](drlocator-table.md)

    

    | Подпись | Parent | Путь      | Глубина |
    |-----------|--------|-----------|-------|
    | AppDir    |        | C: \\ MyDir | 0     |
    | аппфиле   | AppDir |           | 0     |

    

     

4.  Включите действие Аппсеарч в последовательность действий.

    Если MyApp.exe найдена для установки в AppDir, установщик задает для свойства MYAPP расположение файла.
