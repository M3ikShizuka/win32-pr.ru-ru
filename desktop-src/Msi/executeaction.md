# <a name="executeaction-property"></a>EXECUTEACTION, свойство

Свойство **EXECUTEACTION** установлено, чтобы определить, какое действие верхнего уровня инициирует [действие EXECUTEACTION](executeaction-action.md) .

## <a name="remarks"></a>Remarks

Действия верхнего уровня, такие как [действие установки](install-action.md), [объявление действия](advertise-action.md)и [действия администратора](admin-action.md) , присвойте свойству **EXECUTEACTION** имя действия верхнего уровня.

Windows установщик на Windows Server 2003 или Windows XP. сведения о минимальном Windows пакета обновления, который требуется для установщик Windows версии, см. в [установщик Windows требования к Run-Time](windows-installer-portal.md) .<br/> |
