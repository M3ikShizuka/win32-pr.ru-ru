# <a name="executeaction-property"></a>EXECUTEACTION, свойство

Свойство **EXECUTEACTION** установлено, чтобы определить, какое действие верхнего уровня инициирует [действие EXECUTEACTION](executeaction-action.md) .

## <a name="remarks"></a>Комментарии

Действия верхнего уровня, такие как [действие установки](install-action.md), [объявление действия](advertise-action.md)и [действия администратора](admin-action.md) , присвойте свойству **EXECUTEACTION** имя действия верхнего уровня.

## <a name="requirements"></a>Требования



| Требование | Значение |
|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Версия<br/> | Установщик Windows 5,0 в Windows Server 2012, Windows 8, Windows Server 2008 R2 или Windows 7. Установщик Windows 4,0 или установщик Windows 4,5 на Windows Server 2008 или Windows Vista. Установщик Windows в Windows Server 2003 или Windows XP. Сведения о минимальном пакете обновления Windows, который требуется для установщик Windows версии, см. в [установщик Windows Run-Time требования](windows-installer-portal.md) .<br/> |
