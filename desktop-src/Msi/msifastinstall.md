# <a name="msifastinstall-property"></a>МСИФАСТИНСТАЛЛ, свойство

Свойство **мсифастинсталл** можно использовать для сокращения времени, необходимого для установки большого пакета установщик Windows. Свойство может быть задано в командной строке или в таблице [свойств](property-table.md) для настройки операций, определяемых пользователем или разработчиком, которые не являются обязательными для установки.

Значение свойства **мсифастинсталл** может быть сочетанием следующих значений.



| Значение | Значение                                                                      |
|-------|------------------------------------------------------------------------------|
| 0     | Значение по умолчанию                                                                |
| 1     | Для этой установки нет сохраненных точек восстановления системы.                      |
| 2     | Выполнять только [Расчет стоимости файлов](file-costing.md) и пропустить проверку других затрат. |
| 4     | Сократите частоту сообщений о ходе выполнения.                                   |



 

**[Установщик Windows 4,5 или более ранней версии](not-supported-in-windows-installer-4-5.md):** Не поддерживается. Это свойство доступно, начиная с установщик Windows 5,0.
