# <a name="entity-relationship-diagram-legend"></a>Условные обозначения диаграммы отношений сущностей

Черный круг, ведущий к открытому ромбу, указывает на связь «один ко многим» между первичным ключом первой таблицы и внешним ключом второй таблицы. Пример такого вида см. на рисунке для [группы «таблицы реестра](registry-tables-group.md)». Extension — это первичный ключ таблицы расширения и внешний ключ таблицы команд. Таким образом, можно зарегистрировать одно расширение, чтобы иметь несколько глаголов, но конкретную команду можно связать только с одним расширением.
