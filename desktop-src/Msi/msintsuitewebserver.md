# <a name="msintsuitewebserver-property"></a>Мсинтсуитевебсервер, свойство

в Windows 2000 и более поздних операционных системах установщик устанавливает для свойства **мсинтсуитевебсервер** значение 1, если установщик работает в выпуске Windows Server 2003. Установщик задает для этого свойства значение 1, только если в \_ \_ структуре [**осверсионинфоекс**](/windows/win32/api/winnt/ns-winnt-osversioninfoexa) установлен флаг колонки ver Suite. В противном случае установщик не устанавливает это свойство.

## <a name="remarks"></a>Remarks

это свойство доступно только в выпуске установщика Windows Server 2003.
