# <a name="modal-dialog-style-bit"></a>Бит модального стиля диалогового окна

Если этот бит задан, диалоговое окно является модальным, другие диалоговые окна того же приложения не могут быть поверх него, а диалоговое окно сохраняет элемент управления во время его работы. Если этот бит не задан, диалоговое окно немодальное, и другие диалоговые окна этого же приложения могут быть перемещены поверх него. После создания и отображения немодального диалогового окна Пользовательский интерфейс возвращает управление установщику. Затем установщик периодически вызывает пользовательский интерфейс, чтобы обновить диалоговое окно и дать ему возможность обработать сообщения. Как только это будет сделано, элемент управления возвращается в установщик.

> [!Note]  
> В последовательности мастера не должно быть безрежимных диалоговых окон, так как при этом будет возвращено управление установщиком, преждевременно Завершая последовательность мастера.
