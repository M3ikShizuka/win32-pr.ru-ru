# <a name="msisfcbypass-table"></a>Таблица Мсисфкбипасс

таблица мсисфкбипасс содержит список файлов, которые должны обходить Windows защиту файлов при установке файлов в Microsoft Windows Me.

Таблица Мсисфкбипасс содержит следующий столбец.



| Столбец | Type                         | Ключ | Допускает значения NULL |
|--------|------------------------------|-----|----------|
| File\_ | [Идентификатор](identifier.md) | Д   | Нет        |



 

## <a name="columns"></a>Столбцы

<dl> <dt>

<span id="File_"></span><span id="file_"></span><span id="FILE_"></span>File\_
</dt> <dd>

внешний ключ к [таблице файлов](file-table.md) , указывающий файл, который должен обходить Windows защиту файлов при установке файла на Windows Me.

</dd> </dl>
