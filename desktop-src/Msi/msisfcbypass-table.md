# <a name="msisfcbypass-table"></a>Таблица Мсисфкбипасс

Таблица Мсисфкбипасс содержит список файлов, которые должны обходить защиту файлов Windows при установке файлов в Microsoft Windows Me.

Таблица Мсисфкбипасс содержит следующий столбец.



| Столбец | Type                         | Ключ | Допускает значения NULL |
|--------|------------------------------|-----|----------|
| Файл\_ | [Идентификатор](identifier.md) | Да   | Нет        |



 

## <a name="columns"></a>Столбцы

<dl> <dt>

<span id="File_"></span><span id="file_"></span><span id="FILE_"></span>File\_
</dt> <dd>

Внешний ключ к [таблице файлов](file-table.md) , указывающий файл, который должен обходить защиту файлов Windows при установке файла в Windows Me.

</dd> </dl>
