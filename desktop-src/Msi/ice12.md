# <a name="ice12"></a>ICE12

ICE12 запрашивает таблицы [CustomAction](customaction-table.md), [Directory](directory-table.md), [админексекутесекуенце](adminexecutesequence-table.md), [админуисекуенце](adminuisequence-table.md), [адвтексекутесекуенце](advtexecutesequence-table.md), [инсталлексекутесекуенце](installexecutesequence-table.md)и [InstallUISequence](installuisequence-table.md) , чтобы проверить следующее:

-   Это [действие костфинализе](costfinalize-action.md) выполняется во всех таблицах последовательностей, содержащих действия типа [пользовательское действие 35](custom-action-type-35.md) или [тип настраиваемого действия 51](custom-action-type-51.md).
-   Каждый [тип настраиваемого действия 35](custom-action-type-35.md) поступает после [действия костфинализе](costfinalize-action.md). в таблицах последовательности.
-   Каждый [тип настраиваемого действия 51](custom-action-type-51.md) , который имеет внешний ключ к таблице каталога в исходном столбце таблицы CustomAction, предшествует [действию костфинализе](costfinalize-action.md) в таблицах последовательности.

Обратите внимание, что ICE12 не проверяет форматированный текст в целевом столбце таблицы CustomAction.

## <a name="result"></a>Результат

ICE12 отправляет сообщение об ошибке, если проверка настраиваемых действий, заданных в свойстве каталога, завершается ошибкой.

## <a name="example"></a>Пример

ICE12 будет публиковать три ошибки для показанного примера.

-   Папка "MyFolder" для CA1 не найдена в таблице Directory
-   Для CA2 последовательность "80" предшествует Костфинализе в таблице Инсталлексекутесекуенце. Он должен следовать после ( CF@100 )
-   Для CA3 последовательность "125" поступает после Костфинализе в таблице Инсталлексекутесекуенце. Он должен предшествовать ( CF@100 )

[Таблица CustomAction](customaction-table.md) (частичная)



| Действие | Тип | Источник        |
|--------|------|---------------|
| CA1    | 35   | MyFolder      |
| CA2    | 35   | виндовсфолдер |
| CA3    | 51   | виндовсфолдер |



 

[Таблица каталога](directory-table.md)



| Каталог     | \_Родительский каталог | дефаултдир    |
|---------------|-------------------|---------------|
| TARGETDIR     |                   | SourceDir     |
| виндовсфолдер | TARGETDIR         | виндовсфолдер |



 

[Таблица инсталлексекутесекуенце](installexecutesequence-table.md) (частичная)



| Действие       | Последовательность |
|--------------|----------|
| костфинализе | 100      |
| CA2          | 80       |
| CA3          | 125      |



 

Чтобы исправить ошибку для CA1, измените запись в ее исходном столбце в таблице CustomAction на существующую запись в таблице Directory или добавьте MyFolder в таблицу Directory.

Чтобы исправить ошибку для CA2, измените ее последовательность в таблице Инсталлексекутесекуенце таким, чтобы она появилась после действия Костфинализе.

Чтобы исправить ошибку для CA3, измените ее последовательность в таблице Инсталлексекутесекуенце таким, чтобы она наступила перед действием Костфинализе.
