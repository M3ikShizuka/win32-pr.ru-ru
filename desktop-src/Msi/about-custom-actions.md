# <a name="about-custom-actions"></a>Сведения о настраиваемых действиях

Хотя стандартные действия достаточно для выполнения установки в большинстве случаев, настраиваемые действия позволяют автору пакета установки расширить возможности стандартных действий, включая исполняемые файлы, библиотеки динамической компоновки и скрипты. Настраиваемые действия нельзя использовать в таблицах последовательности, используемых для объявления, [таблицы адвтуисекуенце](advtuisequence-table.md) и [таблицы адвтексекутесекуенце](advtexecutesequence-table.md).
