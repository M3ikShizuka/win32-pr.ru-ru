# <a name="configuremodule-object"></a>Объект Конфигуремодуле

**Объект конфигуремодуле** — это интерфейс, реализуемый клиентом. Mergemod.dllвызывает методы в интерфейсе **имсмконфигуремодуле** , чтобы запросить предоставление сведений о конфигурации клиентским средством. Модуль настраивается на основе ответов клиента на вызовы этого интерфейса.

## <a name="members"></a>Элементы

Объект **конфигуремодуле** имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Объект **конфигуремодуле** содержит эти методы.



| Метод                                                           | Описание                                                                        |
|:-----------------------------------------------------------------|:-----------------------------------------------------------------------------------|
| [**провидеинтежердата**](configuremodule-provideintegerdata.md) | Вызывается методом Mergemod.dll для получения целых чисел, используемых для настройки модуля.<br/> |
| [**провидетекстдата**](configuremodule-providetextdata.md)       | Вызывается Mergemod.dll для получения текста, используемого для настройки модуля.<br/>     |
