# <a name="msintsuitesmallbusinessrestricted-property"></a>Мсинтсуитесмаллбусинессрестриктед, свойство

в Windows 2000 и более поздних операционных системах установщик устанавливает для свойства **мсинтсуитесмаллбусинессрестриктед** значение 1, если Microsoft Small Business Server установлен с ограниченным клиентской лицензией принудительно. Установщик задает для этого свойства значение 1, только если в \_ \_ \_ структуре [**осверсионинфоекс**](/windows/win32/api/winnt/ns-winnt-osversioninfoexa) установлен флаг смаллбусинесс Suite Restricted. В противном случае установщик не устанавливает это свойство.
