# <a name="disableloggingfrompackage"></a>дисаблелоггингфромпаккаже

Для этой [системной политики](system-policy.md) на уровне компьютера задано значение "1", чтобы отключить ведение журнала, указанное для пакета, с помощью свойства [**мсилоггинг**](msilogging.md) для всех пользователей компьютера.

для свойства [**мсилоггинг**](msilogging.md) и политики дисаблелоггингфромпаккаже требуется установщик Windows 4,0.

## <a name="registry-key"></a>Ключ реестра

**HKey \_ \_** \\ **политики программного обеспечения** локального компьютера \\  \\  \\  \\ **установщик** Microsoft Windows

## <a name="data-type"></a>Тип данных

**REG \_ DWORD**
