# <a name="applying-major-upgrades-by-installing-the-product"></a>Применение основных обновлений путем установки продукта

Основное обновление можно применить, установив новый установочный пакет для обновленного продукта. Поскольку основные обновления получают код продукта, отличный от исходного продукта, установка обновления должна рассматриваться как установка нового продукта. Обновление можно просто установить как другой продукт. Новый пакет установки может выполнять удаление старого продукта, включая [таблицу обновления](upgrade-table.md) , а также действие [Финдрелатедпродуктс](findrelatedproducts-action.md) и [действие RemoveExistingProducts](removeexistingproducts-action.md).

**Распространение крупного обновления до текущих пользователей из командной строки**

-   В командной строке используйте команду: **msiexec/i \[** _путь к обновленному файлу MSI_ .*_\]_*

**Распространение крупного обновления до текущих пользователей из программы**

-   В программе вызовите [**мсиинсталлпродукт**](/windows/desktop/api/Msi/nf-msi-msiinstallproducta) и укажите путь к обновленному MSI-файлу.
