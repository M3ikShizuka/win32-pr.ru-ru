# <a name="commonfiles64folder-property"></a>CommonFiles64Folder, свойство

Установщик задает для свойства **CommonFiles64Folder** полный путь к стандартной папке общих файлов 64-bit. Для существующего свойства [**коммонфилесфолдер**](commonfilesfolder.md) задается соответствующая 32-разрядная папка.

## <a name="remarks"></a>Комментарии

Установщик задает это свойство для 64-разрядной версии Windows. Это свойство не используется в 32-разрядной версии Windows. При использовании 64-разрядной версии Windows значение может быть "C: \\ Program Files \\ Common Files".
