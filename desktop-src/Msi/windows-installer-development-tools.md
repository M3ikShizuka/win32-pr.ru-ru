# <a name="windows-installer-development-tools"></a>Средства разработки установщик Windows

Следующие средства доступны только в [компонентах Windows SDK для разработчиков установщик Windows](platform-sdk-components-for-windows-installer-developers.md).



| Служебная программа                          | Описание                                                                                                                                                                                                                                      |
|----------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Instmsi.exe](instmsi-exe.md)   | Распространяемый пакет для установки установщик Windows в операционных системах Windows, предшествующих Windows Me.                                                                                                                               |
| [Msicert.exe](msicert-exe.md)   | Заполняет таблицу Мсидигиталсигнатуре и таблицу Мсидигиталцертификате данными цифровой подписи, принадлежащими внешним CAB-файлам в таблице Media.                                                                           |
| [Msidb.exe](msidb-exe.md)       | Импортирует и экспортирует таблицы и потоки базы данных, объединяет базы данных и применяет преобразования.                                                                                                                                                       |
| [Msifiler.exe](msifiler-exe.md) | Заполняет таблицу файлов версиями, языками и размерами на основе исходного каталога. Также можно обновить таблицу Мсифилехаш с помощью хэшей файлов.                                                                                      |
| [Msiinfo.exe](msiinfo-exe.md)   | Изменяет или отображает [информационный поток сводки](summary-information-stream.md).                                                                                                                                                                  |
| [Msimerg.exe](msimerg-exe.md)   | Выполняет слияние одной базы данных с другой.                                                                                                                                                                                                                |
| [Msimsp.exe](msimsp-exe.md)     | Средство создания исправлений. Рекомендуемым методом для создания пакета исправлений является использование средства создания исправлений, например Msimsp.exe с PATCHWIZ.DLL.                                                                                                 |
| [Msistuff.exe](msistuff-exe.md) | Отображает или настраивает ресурсы в исполняемом файле начальной загрузки Setup.exe.                                                                                                                                                                      |
| [Мситул. MAK](msitool-mak.md)   | Файл makefile, который можно использовать для создания средств и настраиваемых действий.                                                                                                                                                                                      |
| [Msitran.exe](msitran-exe.md)   | Создает преобразование или применяет файл преобразования к базе данных.                                                                                                                                                                                 |
| [Msival2.exe](msival2-exe.md)   | Запускает один или набор внутренних средств [оценки согласованности — ICEs](internal-consistency-evaluators-ices.md).                                                                                                                                       |
| [Msizap.exe](msizap-exe.md)     | Удаляет установщик Windows сведения о продукте или обо всех продуктах, установленных на компьютере.                                                                                                                                                      |
| [Orca.exe](orca-exe.md)         | Редактор базы данных. Создает и редактирует MSI файлы и модули слияния.                                                                                                                                                                                 |
| [PATCHWIZ.DLL](patchwiz-dll.md) | Создает [пакет исправлений](patch-packages.md) установщик Windows из файла свойств создания исправления (файл. PCP). Рекомендуемым методом для создания пакета исправлений является использование средства создания исправлений, например Msimsp.exe с PATCHWIZ.DLL. |
| [Wilogutl.exe](wilogutl-exe.md) | Помогает анализировать файлы журналов из установщик Windows установки и отображает предлагаемые решения об ошибках.                                                                                                                              |
