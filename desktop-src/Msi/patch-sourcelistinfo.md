# <a name="patchsourcelistinfo-property"></a>Свойство patch. Саурцелистинфо

Свойство **саурцелистинфо** объекта [**Patch**](patch-object.md) возвращает и задает свойства исходных сведений для исправления. Это свойство вызывает [**мсисаурцелистжетинфо**](/windows/desktop/api/Msi/nf-msi-msisourcelistgetinfoa) или [**мсисаурцелистсетинфо**](/windows/desktop/api/Msi/nf-msi-msisourcelistsetinfoa). Это свойство для чтения или записи.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Patch.SourceListInfo
```



## <a name="property-value"></a>Значение свойства

Имя свойств сведений об источнике, для которого запрашивается или задается исправление. Возможные значения см. в подразделе «Примечания» этого раздела.

## <a name="remarks"></a>Remarks

Можно задать не все свойства, которые могут быть получены. Параметр *сзпроперти* может принимать одно из следующих значений.



| Свойство.         | Можно ли установить? | Значение                                                                                                                                                                                                                                                           |
|------------------|----------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| медиапаккажепас | Да        | Путь относительно корня установочного носителя.                                                                                                                                                                                                          |
| DiskPrompt       | Да        | Шаблон подсказки, используемый при запросе пользователя к установочному носителю.                                                                                                                                                                                          |
| ластуседсаурце   | Да        | Последнее использованное исходное расположение исправления. При задании этого свойства укажите для расположения источника значение "n;" для источника сети или "u;" для типа URL-адреса. Например, используйте "n; \\ \\ \\Тест "Рабочая Рабочая область \\ " или "u; https://microsoft.com/Patches/Office/SP1 ". |
| ластуседтипе     | Нет        | "n", если последний источник является сетевым расположением. "u", если последний использованный источник был URL-адресом. "m", если последний использованный источник был носителем. Пустая строка (""), если нет последнего использованного источника.                                                                      |
| PackageName      | Да        | имя пакета установщик Windows или пакета исправлений в источнике.                                                                                                                                                                                         |
