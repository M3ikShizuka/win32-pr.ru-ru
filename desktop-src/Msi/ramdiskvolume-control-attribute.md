# <a name="ramdiskvolume-control-attribute"></a>Атрибут элемента управления Рамдискволуме

Если этот бит задан, в элементе управления отображаются все тома, участвующие в текущей установке, а также все тома RAM-дисков. Если этот бит не задан, в списке элементов управления отображаются тома текущей установки.

## <a name="valid-controls"></a>Допустимые элементы управления

[директорикомбо](directorycombo-control.md)

 

[волумекостлист](volumecostlist-control.md)

 

[волумеселекткомбо](volumeselectcombo-control.md)

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                                |
|---------|-------------|-----------------------------------------|
| 1048567 | 0x00100000  | **мсидбконтролаттрибутесрамдискволуме** |



 

## <a name="remarks"></a>Remarks

Чтобы задать этот атрибут для элемента управления, включите бит Рамдискволуме в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

См. раздел [Управление атрибутами](control-attributes.md) и [элементами управления](controls.md).
