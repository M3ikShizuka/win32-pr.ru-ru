# <a name="released-versions-tools-and-redistributables"></a>Выпущенные версии, средства и распространяемые пакеты

в следующих разделах содержатся дополнительные сведения о выпущенных версиях, средствах и распространяемых компонентах установщик Windows.

-   [Windows SDK компоненты для установщик Windows разработчиков](platform-sdk-components-for-windows-installer-developers.md) объясняет, где можно получить распространяемые компоненты установщика, документацию, средство проверки базы данных, редактор таблиц базы данных, схему базы данных, средства разработки, средства VBScript, пример продукта и примеры кода.
-   [установщик Windows распространяемые пакеты](windows-installer-redistributables.md) содержат сведения о том, где можно получить распространяемые компоненты установщика.
-   [выпущенные версии установщик Windows](released-versions-of-windows-installer.md) перечислены выпущенные версии установщика, которые поставлялись с разными операционными системами.
-   [средства разработки установщик Windows](windows-installer-development-tools.md) содержат справочные страницы по использованию средств разработки установщика.
-   [Instmsi.exe](instmsi-exe.md) описывает, какие инстмси совместимы с операционной системой. Инстмси — это распространяемый пакет для установки установщика.
