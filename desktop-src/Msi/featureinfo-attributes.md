# <a name="featureinfoattributes-property"></a>Феатуреинфо. Attributes, свойство

Свойство **Attributes** — это свойство, доступное для чтения и записи, которое возвращает значение для функции в столбце Attributes [таблицы Feature](feature-table.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = FeatureInfo.Attributes
FeatureInfo.Attributes = propVal 
```
