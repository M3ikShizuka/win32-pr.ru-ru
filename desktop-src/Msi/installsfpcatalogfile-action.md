# <a name="installsfpcatalogfile-action"></a>Действие Инсталлсфпкаталогфиле

действие инсталлсфпкаталогфиле устанавливает каталоги, используемые Windows Me для защиты Windows файлов. Инсталлсфпкаталогфиле запрашивает [таблицу компонентов](component-table.md), [таблицу файлов](file-table.md), таблицу [филесфпкаталог](filesfpcatalog-table.md) и [таблицу сфпкаталог](sfpcatalog-table.md). Каталоги устанавливаются, если они связаны с файлом в компоненте, настроенном для локальной установки, или если они связаны с восстанавливаемым файлом в локально установленном компоненте.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие Инсталлсфпкаталогфиле должно быть упорядочено до [инсталлфилес](installfiles-action.md) и после [костфинализе](costfinalize-action.md).

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия                                    |
|-------|---------------------------------------------------------------|
| \[1\] | Имя устанавливаемого каталога.                          |
| \[2\] | Имя каталога, от которого зависит Установка этого каталога. |



 

## <a name="remarks"></a>Remarks

Каталог, который зависит от другого каталога, установленного после родительского каталога.
