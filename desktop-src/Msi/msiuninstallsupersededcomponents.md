# <a name="msiuninstallsupersededcomponents-property"></a>МСИУНИНСТАЛЛСУПЕРСЕДЕДКОМПОНЕНТС, свойство

Присвойте свойству **мсиунинсталлсуперседедкомпонентс** значение 1 в [таблице свойств](property-table.md) или в командной строке. Если этому свойству присвоено значение 1, установщик определяет, избыточны ли компоненты в любых заменяемых исправлениях. Установщик может отменить регистрацию и удаление избыточных компонентов, чтобы предотвратить появление устаревших компонентов на компьютере.

Установка этого свойства влияет на компоненты во всех заменяемых исправлениях. Чтобы включить эту функцию для отдельных компонентов, используйте атрибут **мсидбкомпонентаттрибутесунинсталлонсуперседенце** в [таблице Component](component-table.md).

**[Установщик Windows 4,0 и более ранних версий](not-supported-in-windows-installer-4-0.md):** Не поддерживается. Версии, предшествующие установщик Windows 4,5, игнорируют свойство **мсиунинсталлсуперседедкомпонентс** .
