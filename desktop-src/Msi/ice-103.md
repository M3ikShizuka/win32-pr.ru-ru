# <a name="ice-103"></a>ICE 103

ICE 103 проверяет события управления [мсипринт](msiprint-controlevent.md) и [мсилаунчапп](msilaunchapp-controlevent.md) .

ICE 103 проверяет, что событие элемента управления [мсипринт](msiprint-controlevent.md) используется только в диалоговом окне с элементом управления [скроллаблетекст](scrollabletext-control.md) .

**[Установщик Windows 4,5 или более ранней версии](not-supported-in-windows-installer-4-5.md):** Не поддерживается. Эта ICE доступна начиная с установщик Windows 5,0.
