# <a name="secure-at-source-transforms"></a>Преобразования с защитой по источнику

На преобразованиях с защитой по источнику должен находиться источник, расположенный в корне источника пакета. При установке или объявлении пакета установщик сохраняет преобразования в кэше, где в защищенной файловой системе пользователь не имеет доступа на запись. Если локальная копия преобразования становится недоступной, установщик выполняет поиск источника для восстановления кэша. Метод аналогичен поиску в исходном списке для MSI-файла. Дополнительные сведения см. в разделе [устойчивость источника](source-resiliency.md). При удалении продукта любым пользователем все защищенные преобразования для этого продукта удаляются с компьютера пользователя.

Чтобы применить преобразования с защитой по источнику при установке пакета, установите [политику трансформссекуре](transformssecure-policy.md) или свойство [**трансформссекуре**](transformssecure.md) или сделайте @ первый символ, переданный в список преобразования. Каждое преобразование должно быть обозначается именем файла и должно находиться в корне источника пакета. Если какой либо из преобразований не находится в корне источника пакета, установка завершается сбоем. Дополнительные сведения см. в разделе [применение преобразований](applying-transforms.md).
