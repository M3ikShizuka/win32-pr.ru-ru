# <a name="personalfolder-property"></a>Персоналфолдер, свойство

Установщик задает для свойства **персоналфолдер** полный путь к личной папке для текущего пользователя.

|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
