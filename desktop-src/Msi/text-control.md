# <a name="text-control"></a>Text Control

Элемент управления "текст" отображает статический текст, который может использовать предопределенный стиль.

Рекомендуемый метод отображения текста с заданными разрывами строк заключается в использовании нескольких однострочных текстовых элементов управления, расположенных друг за другом. Последовательности символов \\ n, \\ r \\ n или \\ n \\ r в текстовом поле для элемента управления не отображаются в виде разрыва строки. Эти последовательности символов буквально отображаются элементом управления.

## <a name="remarks"></a>Комментарии

Этот элемент управления можно создать из статического класса с помощью функции [**CreateWindowEx**](/windows/win32/api/winuser/nf-winuser-createwindowexa) . У него есть стили **SS \_ Left**, **WS \_ дочернего элемента** и **\_ группы WS** .

Не размещайте элементы управления прозрачного текста поверх цветных растровых изображений. Текст может не отображаться, если пользователь изменяет цветовую схему отображения. Например, текст может стать невидимым, если пользователь устанавливает параметр высокой контрастности в целях специальных возможностей.
