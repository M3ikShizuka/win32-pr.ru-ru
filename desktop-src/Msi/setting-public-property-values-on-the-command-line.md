# <a name="setting-public-property-values-on-the-command-line"></a>Установка значений открытых свойств в командной строке

Чтобы задать для открытого свойства значение литеральной строки, включите строку в кавычки.

СВОЙСТВО = "строка"

Кавычки требуются только в том случае, если строка содержит пробелы. Чтобы очистить открытое свойство в командной строке (присвоив ему значение null), задайте в качестве значения свойства пустую строку. В этом случае требуется ввести кавычки.

СВОЙСТВО = "".
