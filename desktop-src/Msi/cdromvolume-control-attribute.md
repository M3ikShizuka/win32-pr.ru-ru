# <a name="cdromvolume-control-attribute"></a>Атрибут элемента управления Кдромволуме

Если установлен бит управления Кдромволуме, элемент управления отображает все тома в текущей установке, а также все тома компакт-дисков.

Если этот бит не задан, элемент управления отображает все тома в текущей установке.

## <a name="valid-controls"></a>Допустимые элементы управления

[директорикомбо](directorycombo-control.md)

 

[волумекостлист](volumecostlist-control.md)

 

[волумеселекткомбо](volumeselectcombo-control.md)

## <a name="value"></a>Значение



| Decimal | Шестнадцатеричный | Константа                              |
|---------|-------------|---------------------------------------|
| 524288  | 0x00080000  | **мсидбконтролаттрибутескдромволуме** |



 

## <a name="remarks"></a>Remarks

Чтобы задать этот атрибут для элемента управления, включите бит Кдромволуме в столбец Attributes записи элемента управления в [таблице Control](control-table.md).

См. раздел [атрибуты элементов управления](control-attributes.md) и элемент управления, которые необходимо создать в элементах [управления](controls.md).
