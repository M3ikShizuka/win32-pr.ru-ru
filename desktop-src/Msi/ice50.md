# <a name="ice50"></a>ICE50

ICE50 проверяет, что значки ярлыков отображаются правильно и соответствуют их расширению целевого файла.

## <a name="result"></a>Результат

ICE50 отправляет сообщение об ошибке, если расширение файла значка и целевого объекта не совпадает. ICE50 отправляет предупреждение, если значки хранятся в файлах, не имеющих расширения .exe или. ico.

## <a name="example"></a>Пример

ICE50 сообщает об ошибке в приведенном примере.



| Ошибка или предупреждение ICE50                                                                                                              | Описание                                                                                                                                                                                                                                    |
|-------------------------------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Расширение значка "Icon2. dat" для ярлыка "Shortcut2" не соответствует расширению файла ключа для компонента "Component2". | Если расширения значка и целевого файла не совпадают, при объявлении компонента ярлык не будет иметь правильное контекстное меню. Чтобы устранить эту ошибку, переименуйте значок в соответствии с расширением целевого файла.<br/> |
| Расширение значка "Icon1.bat" для ярлыка "Shortcut1" не является "exe" или "ICO". Значок будет отображаться неправильно.         | Не все версии оболочки правильно отображают значки, хранящиеся в файлах, у которых нет расширений "exe" или "ICO". Чтобы устранить это предупреждение, переименуйте значок с расширением "exe" или "ICO".<br/>                                      |
