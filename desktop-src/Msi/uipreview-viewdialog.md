# <a name="uipreviewviewdialog-method"></a>Уипревиев. Виевдиалог, метод

Метод **виевдиалог** объекта [**уипревиев**](uipreview-object.md) отображает диалоговое окно созданного пользовательского интерфейса, хранящееся в текущей базе данных.

## <a name="syntax"></a>Синтаксис


```JScript
UIPreview.ViewDialog(
  dialog
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*диалог* 
</dt> <dd>

Обязательное имя диалогового окна, с учетом регистра и первичный ключ таблицы базы данных диалоговых окон.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
