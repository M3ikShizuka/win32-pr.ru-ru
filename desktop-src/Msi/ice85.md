# <a name="ice85"></a>ICE85

ICE85 проверяет, что столбец SourceName [таблицы MoveFile](movefile-table.md) является допустимым длинным именем файла. Это поле может содержать подстановочные знаки ( \* и?).

## <a name="result"></a>Результат

ICE85 отправляет следующие ошибки.



| ICE85, ошибка                                                      | Описание                                                              |
|------------------------------------------------------------------|--------------------------------------------------------------------------|
| Параметр SourceName " \[ 2 \] ", обнаруженный в таблице MoveFile, имеет неправильный формат. | Поле SourceName в таблице MoveFile не является допустимым длинным именем файла |
