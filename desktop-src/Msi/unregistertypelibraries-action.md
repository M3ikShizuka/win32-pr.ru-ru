# <a name="unregistertypelibraries-action"></a>Действие Унрегистертипелибрариес

Действие Унрегистертипелибрариес отменяет регистрацию библиотек типов в системе. Это действие применяется к каждому файлу, указанному в таблице [typelib](typelib-table.md) , запускаемой для удаления.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Действие Унрегистертипелибрариес должно следовать перед действием [ремовефилес](removefiles-action.md) .

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия                        |
|-------|---------------------------------------------------|
| \[1\] | [Идентификатор GUID](guid.md) незарегистрированной библиотеки типов. |
