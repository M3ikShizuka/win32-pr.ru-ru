# <a name="user-policies"></a>Политики пользователя

Следующие пользовательские политики можно настроить в разделе

**HKey \_ текущие \_ политики пользовательского** \\ **программного обеспечения** \\  \\  \\  \\ **установщик** Microsoft Windows



| Имя значения                                                            | Типы данных значений          | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
|-----------------------------------------------------------------------|---------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [AlwaysInstallElevated](alwaysinstallelevated.md)<br/>         | **REG \_ DWORD**<br/> | Если для этого параметра задано значение "1", а также задано соответствующее значение компьютера, установщик всегда устанавливается с повышенными привилегиями.<br/> В противном случае установщик использует повышенные привилегии для установки управляемых приложений и использует уровень привилегий текущего пользователя для неуправляемых приложений.<br/>                                                                                                                                                                                                      |
| [дисаблемедиа](disablemedia.md)<br/>                           | **REG \_ DWORD**<br/> | Если для политики [дисаблемедиа](disablemedia.md) задано значение "1", то пользователи и администраторы, выполняющие установку одного продукта, не смогут использовать диалоговое окно "Обзор" для просмотра источников мультимедиа, таких как CD-ROM, для источников других устанавливаемых продуктов. Просмотр других продуктов не зависит от того, имеет ли установка более высокий уровень привилегий. Пользователь по-прежнему может переустановить продукт с носителя, если у пользователя есть правильно помеченный источник мультимедиа.<br/> |
| [дисаблероллбакк](disablerollback.md)<br/>                     | **REG \_ DWORD**<br/> | Если это значение равно 1, установщик не будет хранить файлы отката во время установки, отключив откат установки. По умолчанию откат включен. Администраторам рекомендуется не использовать эту политику, если она совершенно необязательна.<br/>                                                                                                                                                                                                                                                             |
| [сеарчордер](searchorder.md)<br/>                             | **REG \_ SZ**<br/>    | Порядок, в котором установщик выполняет поиск трех различных типов источников:<br/> "n" — сеть<br/> "m" — носитель (компакт-диск или DVD-диск)<br/> "u" — URL-адрес (универсальный указатель ресурсов)<br/> Например, значение "НМУ" указывает установщику сначала искать сетевые источники, источники мультимедиа во второй и URL-источники. При выходе из этой буквы соответствующий тип тома удаляется из поиска. По умолчанию при отсутствии этого значения используется сеть, затем носитель, за которым следует URL-адрес.<br/>          |
| [Политика Трансформсатсаурце](transformsatsource-policy.md)<br/> | **REG \_ DWORD**<br/> | Если это значение существует и имеет значение "1"; Установщик выполняет поиск файлов преобразования в корне всех сетевых источников в расходе для [**продукта**](sourcelist.md) . По умолчанию преобразования хранятся в папке Application Data профиля пользователя.<br/>                                                                                                                                                                                                                                             |
