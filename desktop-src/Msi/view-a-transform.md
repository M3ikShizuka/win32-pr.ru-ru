# <a name="view-a-transform"></a>Просмотр преобразования

Файл WiLstXfm.vbs VBScript предоставляется в [Windows SDK компонентов для установщик Windows разработчиков](platform-sdk-components-for-windows-installer-developers.md). Этот пример скрипта можно использовать для просмотра файла преобразования.

В примере демонстрируется использование:

-   [\_Таблица переformview](-transformview-table.md)
-   [**Метод Опендатабасе (объект установщика)**](installer-opendatabase.md)
-   [**Метод ластерроррекорд**](installer-lasterrorrecord.md) [ **объекта Installer**](installer-object.md)
-   [**Метод Апплитрансформ**](database-applytransform.md)
-   [**Метод OpenView**](database-openview.md)
-   [**Метод Commit**](database-commit.md) [ **объекта Database**](database-object.md)
-   [**Свойство IsNull**](record-isnull.md)
-   [**Свойство StringData**](record-stringdata.md) [ **объекта Record**](record-object.md)

Для использования этого примера требуется CScript.exe версия сервера сценариев Windows. Чтобы использовать CScript.exe для запуска этого образца, введите команду в командной строке, используя следующий синтаксис. Если первый аргумент имеет значение/?, отображается справка. или, если указано слишком мало аргументов. Чтобы перенаправить выходные данные в файл, завершите командную строку с помощью VBS > \[ *путь к файлу* \] . Пример возвращает значение 0 для успешного выполнения, 1, если вызывается Справка, и 2 в случае сбоя сценария.

**cscript WiLstXfm.vbs** *\[ путь для ссылки на \] \[ путь к параметру базы данных \] \[ для \] просмотра преобразования*

Укажите путь к справочной базе данных установщик Windows. Укажите список путей для преобразования просматриваемых файлов. Каждому пути в списке может предшествовать необязательное числовое значение. Это значение указывает набор ошибочных условий, которые следует подавлять. Эти значения можно добавить вместе, чтобы отключить несколько условий. Если параметр Numeric не указан, все условия ошибки подавляются. Аргументы в этом списке выполняются в порядке слева направо, в котором они отображаются в командной строке.



| Значение | Условие ошибки для подавления                   |
|-------|-----------------------------------------------|
| 1     | Добавление уже существующей строки.             |
| 2     | Удаление несуществующей строки.           |
| 4     | Добавление уже существующей таблицы.           |
| 8     | Удаление несуществующей таблицы.         |
| 16    | Обновление несуществующей строки.           |
| 256   | Несоответствие кодовых страниц базы данных и преобразования. |
