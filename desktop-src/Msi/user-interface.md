# <a name="user-interface-windows-installer"></a>пользовательский интерфейс (установщик Windows)

Windows Установщик предоставляет полный пользовательский интерфейс для установки приложения или продукта. Пользовательский интерфейс предоставляет пользователю параметры, доступные для настройки установки, и получает от пользователя сведения о ожидающем процессе установки.

[О пользовательском интерфейсе](about-the-user-interface.md) описаны функциональные возможности пользовательского интерфейса установщика. [Использование пользовательского интерфейса](using-the-user-interface.md) описывает использование внутреннего пользовательского интерфейса установщика.

Справочные сведения о внутреннем диалоговом окне установщика, а также стили и параметры элемента управления представлены в [справочнике по пользовательскому интерфейсу](user-interface-reference.md).
