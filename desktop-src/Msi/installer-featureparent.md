# <a name="installerfeatureparent-property"></a>Свойство Installer. Феатурепарент

Свойство **феатурепарент** указывает родительский компонент функции. Пустая строка для этого свойства указывает, что эта функция не имеет родителя и является корневым элементом.

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```JScript
propVal = Installer.FeatureParent
```
