# <a name="a-localization-example"></a>Пример локализации

В этом примере показано, как локализовать пакет установщик Windows, описанный в [примере установки](an-installation-example.md). Исходный пакет установки изменен с английского на французскую языковую версию.

Пример локализации имеет следующие характеристики.

-   Пользовательский интерфейс установки MNP2000 приложения следует изменить с английского на французский.
-   Французская версия MNP2000 включает Readme.txt и Help.txt файлы, написанные на французском языке.
-   Версия на французском языке включает список агентов путешествий в Франции, которые могут предоставлять билеты в красный парк. Этот список (представленный в виде нового файла, Fre.txt) не является частью английской версии.

Общая процедура локализации установки описана в разделе [локализация пакета установщик Windows](localizing-a-windows-installer-package.md).

Скопируйте английскую версию MNP2000.msi и все исходные файлы, описанные в разделе [Планирование установки](planning-the-installation.md) в новую папку. Измените имя MNP2000.msi в папке на MNPFren.msi. В следующих разделах эта копия будет изменена для локализации приложения на французский.
