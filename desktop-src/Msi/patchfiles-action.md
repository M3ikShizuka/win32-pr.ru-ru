# <a name="patchfiles-action"></a>Действие Патчфилес

Действие Патчфилес запрашивает [таблицу исправлений](patch-table.md) , чтобы определить, какие исправления должны быть применены. Действие Патчфилес также выполняет обновление файлов по байтам.

## <a name="sequence-restrictions"></a>Ограничения последовательности

Если файл, который должен быть исправлен, не установлен, действие Патчфилес должно следовать после [действия инсталлфилес](installfiles-action.md) , устанавливающего файл. Ранее установленные файлы могут быть исправлены в любой точке последовательности действий. [Действие дупликатефилес](duplicatefiles-action.md) должно следовать после действия патчфилес, чтобы предотвратить дублирование неисправленной версии файла.

## <a name="actiondata-messages"></a>Сообщения Актиондата



| Поле | Описание данных действия                    |
|-------|-----------------------------------------------|
| \[1\] | Идентификатор исправленного файла.                   |
| \[2\] | Идентификатор каталога, содержащего исправленный файл. |
| \[3\] | Размер исправления в байтах.                       |
