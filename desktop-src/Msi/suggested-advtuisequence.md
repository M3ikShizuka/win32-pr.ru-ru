# <a name="suggested-advtuisequence"></a>Предложенные Адвтуисекуенце

Не создавайте таблицу Адвтуисекуенце. Установщик не использует эту таблицу. Таблица Адвтуисекуенце не должна существовать в базе данных установки, или ее необходимо оставить пустой. Имя приложения, которое необходимо объявить, должно быть передано в установщик во время инициализации.
