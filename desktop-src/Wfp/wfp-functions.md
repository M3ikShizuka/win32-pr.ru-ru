# <a name="wrp-functions"></a>Функции WRP

Следующие функции позволяют перечислять защищенные ресурсы.



| Функция                                         | Описание                                                 |
|--------------------------------------------------|-------------------------------------------------------------|
| [**сфЦисфилепротектед**](/windows/desktop/api/Sfc/nf-sfc-sfcisfileprotected) | Определяет, защищен ли указанный файл.         |
| [**сфЦискэйпротектед**](/windows/desktop/api/Sfc/nf-sfc-sfciskeyprotected)   | Определяет, защищен ли указанный раздел реестра. |



 

Если это возможно для операционной системы, следует использовать функции в этой таблице, а не устаревшие функции: [**сфкжетнекстпротектедфиле**](/windows/desktop/api/Sfc/nf-sfc-sfcgetnextprotectedfile) и [**сфкжетфилес**](sfcgetfiles.md).
