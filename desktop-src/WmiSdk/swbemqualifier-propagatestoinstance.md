# <a name="swbemqualifierpropagatestoinstance-property"></a>Свбемкуалифиер. Пропагатестоинстанце, свойство

Свойство **пропагатестоинстанце** объекта [**свбемкуалифиер**](swbemqualifier.md) является логическим значением, указывающим, может ли этот квалификатор распространяться на экземпляр.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
SWbemQualifier.PropagatesToInstance As Boolean
```
