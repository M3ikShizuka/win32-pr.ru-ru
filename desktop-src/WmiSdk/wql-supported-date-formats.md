# <a name="wql-supported-date-formats"></a>WQL-Supported форматы даты

Помимо формата **DateTime** WMI, для использования в предложении **WHERE** WQL поддерживаются следующие форматы даты.

Формат отображения времени различается для разных языков. Сценарии, подключающиеся к удаленным компьютерам, должны указывать время в формате, соответствующем языковому стандарту конечного компьютера.

Например, США формат даты и времени — MM-ДД-ГГГГ. Если сценарий на компьютере с США подключается к целевому компьютеру в формате дд-мм-гггг, запросы обрабатываются на целевом компьютере как дд-мм-гггг. Чтобы избежать различных результатов между языками, используйте формат [**DateTime**](datetime.md) . Дополнительные сведения см. в разделе [Формат даты и времени](date-and-time-format.md).
