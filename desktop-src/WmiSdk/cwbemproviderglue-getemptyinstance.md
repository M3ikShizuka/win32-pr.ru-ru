# <a name="cwbemprovidergluegetemptyinstance-methods"></a>Методы Квбемпровидерглуе:: Жетемптинстанце

\[Класс [**квбемпровидерглуе**](/windows/win32/api/wbemglue/nl-wbemglue-cwbemproviderglue) является частью платформы поставщика WMI, которая теперь считается окончательным состоянием, и дальнейшая разработка, улучшения и обновления не будут доступны для проблем, связанных с безопасностью, затрагивающих эти библиотеки. [API MI](/previous-versions/windows/desktop/wmi_v2/windows-management-infrastructure) следует использовать для всех новых разработок.\]

Метод **жетемптинстанце** извлекает один незаполненный экземпляр указанного класса.
