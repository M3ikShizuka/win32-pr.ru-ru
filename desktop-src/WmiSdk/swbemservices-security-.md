# <a name="swbemservicessecurity_-property"></a>SWbemServices. Security, \_ свойство

Свойство **безопасности \_** объекта [**SwbemServices**](swbemservices.md) используется для получения или задания параметров безопасности для объекта **SwbemServices** . Это свойство является объектом [**свбемсекурити**](swbemsecurity.md) .

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemServices.Security_ As Object
```
