# <a name="chstringchstring-constructors"></a>Конструкторы Чстринг:: Чстринг

\[Класс [**чстринг**](chstring.md) является частью платформы поставщика WMI, которая теперь считается окончательным состоянием, и дальнейшая разработка, улучшения и обновления не будут доступны для проблем, связанных с безопасностью, затрагивающих эти библиотеки. [API MI](/previous-versions/windows/desktop/wmi_v2/windows-management-infrastructure) следует использовать для всех новых разработок.\]

Каждый из следующих конструкторов Инициализирует новый объект [**чстринг**](chstring.md) с указанными данными.

### <a name="overload-list"></a>Список перегрузок



| Конструктор                                                                        | Описание                                                  |
|:-----------------------------------------------------------------------------------|:-------------------------------------------------------------|
| [**Чстринг ()**](/windows/win32/api/rrascfg/nn-rrascfg-ieapproviderconfig)                                          | Создает пустой объект Чстринг.<br/>                 |
| [**Чстринг (LPCSTR)**](/windows/win32/api/chstring/nf-chstring-chstring-chstring(lpcstr))                              | Инициализирует со значением LPCSTR.<br/>                |
| [**Чстринг (ЛПКВСТР)**](/windows/win32/api/chstring/nf-chstring-chstring-chstring(lpcwstr))                            | Инициализирует со значением ЛПКВСТР.<br/>               |
| [**Чстринг (WCHAR, int)**](/windows/win32/api/chstring/nf-chstring-chstring-chstring(wchar_int))                        | Инициализирует целочисленные копии значения WCHAR.<br/>   |
| [**Чстринг (ЛПКВСТР, int)**](/windows/win32/api/chstring/nf-chstring-chstring-chstring(lpcwstr_int))                    | Инициализирует целочисленные копии значения ЛПКВСТР.<br/> |
| [**Чстринг (const Чстринг&)**](/windows/win32/api/chstring/nf-chstring-chstring-chstring(constchstring_))            | Реплицирует параметр Чстринг.<br/>                |
| [**Чстринг (const без знака \* )**](/windows/win32/api/chstring/nf-chstring-chstring-chstring(constunsignedchar)) | Инициализирует со значением, на которое указывает char \* .<br/>  |
