# <a name="swbempropertyname-property"></a>SWbemProperty.Name, свойство

Свойство **Name** объекта [**SWbemProperty**](swbemproperty.md) ВОЗВРАЩАЕТ имя этого свойства WMI. Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemProperty.Name As String
```
