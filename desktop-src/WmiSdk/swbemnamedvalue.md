# <a name="swbemnamedvalue-object"></a>Объект Свбемнамедвалуе

Объект **свбемнамедвалуе** представляет одно именованное значение, которое принадлежит объекту коллекции [**свбемнамедвалуесет**](swbemnamedvalueset.md) . Не удается создать этот объект с [**помощью вызова функции**](creating-an-object-using-vbscript.md) VBScript.

## <a name="members"></a>Элементы

Объект **свбемнамедвалуе** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Свойства

Объект **свбемнамедвалуе** имеет следующие свойства.



| Свойство                                          | Тип доступа           | Описание                                                                                    |
|:--------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------|
| [**Имя**](swbemnamedvalue-name.md)<br/>   | Только для чтения<br/>  | Имя элемента **свбемнамедвалуе** .<br/>                                               |
| [**Значение**](swbemnamedvalue-value.md)<br/> | Чтение/запись<br/> | Значение элемента **свбемнамедвалуе** . Это свойство данного объекта по умолчанию.<br/> |
