# <a name="swbemeventsourcesecurity_-property"></a>Свбемевентсаурце. Security, \_ свойство

Свойство **Security \_** объекта [**свбемевентсаурце**](swbemeventsource.md) Возвращает или задает параметры безопасности объекта **свбемевентсаурце** . Это свойство является объектом [**свбемсекурити**](swbemsecurity.md) .

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemEventSource.Security_ As Object
```
