# <a name="swbemprivilege-object"></a>Объект Свбемпривилеже

Объект **свбемпривилеже** представляет одну привилегию. Не удается создать этот объект с **помощью вызова функции** VBScript.

## <a name="members"></a>Элементы

Объект **свбемпривилеже** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Элемент Property

Объект **свбемпривилеже** имеет следующие свойства.



| Свойство                                                     | Тип доступа           | Описание                                                                      |
|:-------------------------------------------------------------|:----------------------|:---------------------------------------------------------------------------------|
| [**DisplayName**](swbemprivilege-displayname.md)<br/> | Только для чтения<br/>  | Отображаемое имя этой привилегии.<br/>                                       |
| [**Идентификатор**](swbemprivilege-identifier.md)<br/>   | Чтение/запись<br/> | Целое число, представляющее устанавливаемое или получаемое право доступа.<br/> |
| [**IsEnabled**](swbemprivilege-isenabled.md)<br/>     | Чтение/запись<br/> | Логическое значение, указывающее, включена ли эта привилегия.<br/>            |
| [**Имя**](swbemprivilege-name.md)<br/>               | Только для чтения<br/>  | Имя этой привилегии.<br/>                                               |
