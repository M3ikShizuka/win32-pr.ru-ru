# <a name="operating-system-availability-of-wmi-components"></a>Доступность компонентов WMI в операционной системе

инструментарий WMI устанавливается по умолчанию на всех Windows настольных и серверных платформах. Тем не менее некоторые поставщики WMI могут быть установлены в зависимости от выпуска и конфигурации ОС. например, поставщик SNMP не включен по умолчанию, а поставщик установщик Windows (поставщик MSI) по умолчанию не устанавливается в 64-разрядных операционных системах. Дополнительные сведения о доступности конкретного поставщика см. в разделе [поставщики WMI](/windows/desktop/SrvNodes/wmi-mi-omi-providers).
