# <a name="swbemobjectpathissingleton-property"></a>Свбемобжектпас. Singleton, свойство

Свойство «- **Singleton** » объекта [**Свбемобжектпас**](swbemobjectpath.md) — это логическое значение, указывающее, представляет ли этот путь одноэлементный экземпляр. Одноэлементный экземпляр — это экземпляр класса, который не может иметь более одного экземпляра. Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemObjectPath.IsSingleton As Boolean
```
