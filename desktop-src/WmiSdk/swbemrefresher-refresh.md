# <a name="swbemrefresherrefresh-method"></a>Свбемрефрешер. Refresh, метод

Метод **свбемрефрешер. Refresh** обновляет все элементы, содержащиеся в объекте [**свбемрефрешер**](swbemrefresher.md) .

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

## <a name="syntax"></a>Синтаксис


```VB
SWbemRefresher.Refresh( _
  [ ByVal iFlags ] _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*ифлагс* \[ используемых\]
</dt> <dd>

Для флагов необходимо задать значение 0 (ноль).

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
