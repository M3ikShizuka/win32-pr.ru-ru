# <a name="__event-class"></a>\_\_Класс событий

Класс системы **\_ \_ событий** является абстрактным базовым классом, который служит родительским классом для всех внутренних и внешних событий. Все новые типы событий должны в конечном итоге быть производными от **\_ \_ события**. Внутренние события наследуются непосредственно от этого класса. Определяемые пользователем внешние события являются производными от подкласса этого класса с именем [**\_ \_ екстринсицевент**](--extrinsicevent.md).

Приведенный ниже синтаксис является упрощенной версией кода MOF и включает все унаследованные свойства. Свойства перечислены в алфавитном порядке, а не в MOF.

## <a name="syntax"></a>Синтаксис

``` syntax
[abstract]
class __Event : __IndicationRelated
{
  uint8  SECURITY_DESCRIPTOR[];
  uint64 TIME_CREATED;
};
```

### <a name="properties"></a>Элемент Property

Класс **\_ \_ событий** имеет следующие свойства.

<dl> <dt>

**\_дескриптор безопасности**
</dt> <dd> <dl> <dt>

Тип данных: массив **Uint8**
</dt> <dt>

Дескриптор, используемый поставщиком событий для определения того, какие пользователи могут получить событие. Дополнительные сведения см. в разделе [константы безопасности WMI](wmi-security-constants.md).

</dd> <dt>

**ВРЕМЯ \_ создания**
</dt> <dd> <dl> <dt>

Тип данных: **UInt64**
</dt> <dt>

Тип доступа: только для чтения
</dt> </dl>

Уникальное значение, указывающее время создания события. Это 64-разрядное значение, представляющее число 100-наносекундных интервалов после 1 января 1601 г. Эти сведения находятся в формате всеобщего скоординированного времени (UTC).

Дополнительные сведения об использовании значений **UInt64** в скриптах см. [в разделе Создание сценариев в WMI](/windows/desktop/WmiSdk/creating-a-wmi-script).

## <a name="remarks"></a>Remarks

Класс **\_ \_ событий** является производным от [**\_ \_ индикатионрелатед**](--indicationrelated.md), у которого нет свойств.
