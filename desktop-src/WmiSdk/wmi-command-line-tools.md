# <a name="wmi-command-line-tools"></a>Средства командной строки инструментария WMI

Инструментарий WMI имеет несколько средств командной строки, используемых администраторами и модулями записи поставщиков WMI.



| Section                      | Описание                                                                                                                                                         |
|------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**mofcomp**](mofcomp.md)   | Компилирует код MOF-файл (MOF) в двоичную форму, хранящуюся в репозитории WMI. Используется при создании или изменении MOF-файла для поставщика WMI.<br/> |
| [**smi2smir**](smi2smir.md) | Принимает один модуль SNMP Information в качестве входных данных и принимает все дополнительные модули, необходимые для разрешения внешних ссылок.<br/>                               |
| [**инициирует**](winmgmt.md)   | Запускает службу WMI. начиная с Windows Vista были добавлены несколько новых параметров.<br/>                                                                |
| [**wmiadap**](wmiadap.md)   | Запускает программу командной строки ADAP для анализа и перемещения библиотек производительности в репозиторий WMI.<br/>                                                       |
| [**wmic**](wmic.md)         | Выполняет операции WMI в командной строке.<br/>                                                                                                               |
