# <a name="__namespacecreationevent-class"></a>\_\_Класс Намеспацекреатионевент

Класс System **\_ \_ намеспацекреатионевент** сообщает о событии создания пространства имен, которое представляет собой тип [встроенного события](determining-the-type-of-event-to-receive.md) , создаваемого при добавлении нового пространства имен к текущему пространству имен.

Приведенный ниже синтаксис является упрощенной версией кода MOF и включает все унаследованные свойства. Свойства перечислены в алфавитном порядке, а не в MOF.

## <a name="syntax"></a>Синтаксис

``` syntax
class __NamespaceCreationEvent : __NamespaceOperationEvent
{
  uint8       SECURITY_DESCRIPTOR[];
  __Namespace TargetNamespace;
  uint64      TIME_CREATED;
};
```

### <a name="properties"></a>Элемент Property

Класс **\_ \_ намеспацекреатионевент** имеет следующие свойства.

**\_дескриптор безопасности**

</dd> <dt>

**Имен**
</dt> <dd> <dl> <dt>

Тип данных: **\_ \_ пространство имен**
</dt> <dt>

Тип доступа: только для чтения

Копия созданного экземпляра [**\_ \_ пространства имен**](--namespace.md) . Свойство **Name** экземпляра **\_ \_ пространства имен** указывает, какое пространство имен было создано. Это свойство наследуется от [**\_ \_ намеспацеоператионевент**](--namespaceoperationevent.md).

</dd> <dt>

**ВРЕМЯ \_ создания**
</dt> <dd> <dl> <dt>

Тип данных: **UInt64**

Дополнительные сведения об использовании значений **UInt64** в скриптах см. [в разделе Создание сценариев в WMI](/windows/desktop/WmiSdk/creating-a-wmi-script).
