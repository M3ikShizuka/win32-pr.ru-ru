# <a name="wmi-qualifiers"></a>Квалификаторы WMI

Инструментарий WMI имеет несколько типов [*квалификаторов*](gloss-q.md)класса и свойств. Квалификаторы также могут изменять [*разновидности*](gloss-f.md). В инструментарии WMI используются следующие типы квалификаторов и разновидностей.

Имя каждого квалификатора отображается с типом данных и индикатор того, можно ли применить квалификатор к классу, экземпляру, свойству или методу. Для таких квалификаторов, как **Association** (рассматривается в разделе [meta квалификаторы](meta-qualifiers.md)), существует подразумеваемое правило использования, которое также должен присутствовать квалификатор meta. Например, неявное правило использования для квалификаторов **агрегатов** заключается в том, что квалификатор **ассоциации** также должен присутствовать.



| Тип квалификатора                              | Описание                                                                                                                           |
|---------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Метаконфигурации](meta-qualifiers.md)                 | Уточнение определения мета-конструкций путем уточнения фактического использования объявления класса или свойства.                          |
| [Необязательный](optional-qualifiers.md)         | Рассматриваются ситуации, которые не являются общими для всех реализаций, совместимых с CIM.                                                                 |
| [Разновидности квалификаторов](qualifier-flavors.md)  | Предоставляет дополнительные сведения об квалификаторе, например, может ли производный класс или экземпляр переопределить исходное значение квалификатора. |
| [Стандартный](standard-qualifiers.md)         | Поддерживает описания, которые должны быть обработаны всеми CLS-совместимыми реализациями.                                                         |
| [Зависит от инструментария WMI](wmi-specific-qualifiers.md) | Описывает квалификаторы, относящиеся к WMI, такие как квалификаторы класса счетчика производительности.                                                   |



 

Дополнительные сведения о применении квалификаторов к классам WMI см. в разделе [Добавление квалификатора](adding-a-qualifier.md). Чтобы узнать, как проверить квалификаторы существующих классов WMI, см. Приведенный ниже пример кода.

## <a name="example"></a>Пример

Следующий код PowerShell, взятый из [коллекции TechNet](https://Gallery.TechNet.Microsoft.Com/Get-WMI-Class-qualifiers-239970e7), описывает, как получить квалификаторы из класса WMI.


```PowerShell
Function Get-WMIClassesWithQualifiers 
{ 
 Param([string]$qualifier = "dynamic", 
  [string]$namespace = "root\cimv2") 
 $classes = Gwmi -list -namespace $namespace 
 foreach($class in $classes) 
 { 
  $query = "select * from meta_class where __this isa ""$($class.name)"" " 
  $a = gwmi -Query $query -Namespace $namespace |  
  select -Property __class, qualifiers 
   if($a.qualifiers | % { $_ | ? { $_.name -match "$qualifier" }}) 
    { $a.__class } 
  } #end foreach $class 
} 
```
