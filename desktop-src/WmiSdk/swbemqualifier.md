# <a name="swbemqualifier-object"></a>Объект Свбемкуалифиер

Свойства объекта **свбемкуалифиер** можно использовать для представления одного квалификатора класса, экземпляра, свойства или параметра метода WMI. Не удается создать этот объект с [помощью вызова функции](creating-an-object-using-vbscript.md) VBScript.

## <a name="members"></a>Элементы

Объект **свбемкуалифиер** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Элемент Property

Объект **свбемкуалифиер** имеет следующие свойства.



| Свойство.                                                                       | Тип доступа           | Описание                                                                                           |
|:-------------------------------------------------------------------------------|:----------------------|:------------------------------------------------------------------------------------------------------|
| [**Изменено**](swbemqualifier-isamended.md)<br/>                       | Только для чтения<br/>  | Логическое значение, указывающее, был ли этот квалификатор локализован с помощью операции MERGE.<br/> |
| [**Локальный**](swbemqualifier-islocal.md)<br/>                           | Только для чтения<br/>  | Логическое значение, указывающее, является ли этот квалификатор локальным.<br/>                                   |
| [**Допускающий переопределение**](swbemqualifier-isoverridable.md)<br/>               | Чтение/запись<br/> | Логическое значение, указывающее, можно ли переопределить этот квалификатор при распространении.<br/>          |
| [**Безымян**](swbemqualifier-name.md)<br/>                                 | Только для чтения<br/>  | Имя этого квалификатора.<br/>                                                                    |
| [**пропагатестоинстанце**](swbemqualifier-propagatestoinstance.md)<br/> | Чтение/запись<br/> | Логическое значение, указывающее, может ли этот квалификатор распространяться на экземпляр.<br/>           |
| [**пропагатестосубкласс**](swbemqualifier-propagatestosubclass.md)<br/> | Только для чтения<br/>  | Логическое значение, указывающее, можно ли распространить этот квалификатор в подкласс.<br/>            |
| [**Значение**](swbemqualifier-value.md)<br/>                               | Чтение/запись<br/> | Значение Variant этого квалификатора. Это свойство данного объекта по умолчанию.<br/>              |
