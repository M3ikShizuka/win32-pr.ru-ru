# <a name="meta-qualifiers"></a>Квалификаторы Meta

Квалификаторы meta уточняют определение мета-конструкций в модели CIM путем уточнения фактического использования объявления класса или свойства в синтаксисе MOF.

<dt>

<span id="Association"></span><span id="association"></span><span id="ASSOCIATION"></span>**Связке**
</dt> <dd>

Тип данных: **логический**

Область применения: классы

Указывает, является ли класс классом ассоциации, используемым для описания связи между двумя другими классами. Отсутствие этого квалификатора указывает на то, что класс не является классом ассоциации. Этот квалификатор является взаимоисключающим с **указанием**. Дополнительные сведения см. в разделе [объявление класса ассоциации](declaring-an-association-class.md).

</dd> <dt>

<span id="Indication"></span><span id="indication"></span><span id="INDICATION"></span>**Указание**
</dt> <dd>

Тип данных: **логический**

Область применения: классы

Указывает, определяет ли класс объекта указание. Этот квалификатор является взаимоисключающим с **Ассоциацией**. Значение по умолчанию — **false**.

</dd> </dl>
