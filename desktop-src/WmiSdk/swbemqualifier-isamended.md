# <a name="swbemqualifierisamended-property"></a>Свбемкуалифиер. свойство

Свойство **с параметром «** [**свбемкуалифиер**](swbemqualifier.md) » объекта «объекты» является логическим значением, указывающим, был ли этот квалификатор локализован с помощью операции MERGE.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemQualifier.IsAmended As Boolean
```
