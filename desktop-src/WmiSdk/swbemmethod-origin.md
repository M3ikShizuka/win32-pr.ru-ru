# <a name="swbemmethodorigin-property"></a>Свбеммесод. Origin, свойство

Свойство **Origin** объекта [**свбеммесод**](swbemmethod.md) возвращает имя класса, в котором был представлен этот метод. Для классов с иерархиями с глубоким наследованием часто желательно понять, какие методы были объявлены в каких классах. Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemMethod.Origin As String
```
