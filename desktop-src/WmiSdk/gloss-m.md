# <a name="m-wmi"></a>M (WMI)

[A](gloss-a.md) B [C](gloss-c.md) [D](gloss-d.md) [E](gloss-e.md) [F](gloss-f.md) G [р](gloss-h.md) [.](gloss-i.md) J [K](gloss-k.md) [L](gloss-l.md) M [N](gloss-n.md) [O](gloss-o.md) [P](gloss-p.md) [Q](gloss-q.md) [R](gloss-r.md) [S](gloss-s.md) [T](gloss-t.md) U V [W](gloss-w.md) X Y Z

<dl> <dt>

<span id="wmi.gloss_managed_object"></span><span id="WMI.GLOSS_MANAGED_OBJECT"></span>**управляемый объект**
</dt> <dd>

Аппаратный или программный компонент системы, представленный в виде экземпляра класса WMI.

</dd> <dt>

<span id="wmi.gloss_managed_object_format"></span><span id="WMI.GLOSS_MANAGED_OBJECT_FORMAT"></span>**MOF-файл (MOF)**
</dt> <dd>

Язык, основанный на языке определения интерфейсов (IDL), который описывает сведения об управлении.

</dd> <dt>

<span id="wmi.gloss_management_information_base"></span><span id="WMI.GLOSS_MANAGEMENT_INFORMATION_BASE"></span>**база данных управления (MIB)**
</dt> <dd>

База данных, в которой хранится коллекция представлений сетевых объектов.

</dd> <dt>

<span id="wmi.gloss_metamodel"></span><span id="WMI.GLOSS_METAMODEL"></span>**MetaModel**
</dt> <dd>

Компонент CIM, описывающий сущности и связи, представляющие *управляемые объекты*. Например, классы, экземпляры и ассоциации включены в MetaModel.

</dd> <dt>

<span id="wmi.gloss_metaschema"></span><span id="WMI.GLOSS_METASCHEMA"></span>**метасчема**
</dt> <dd>

[*Схема*](gloss-s.md) *MetaModel*.

</dd> <dt>

<span id="wmi.gloss_method_provider"></span><span id="WMI.GLOSS_METHOD_PROVIDER"></span>**Поставщик метода**
</dt> <dd>

Поставщик, который обеспечивает доступ WMI к методам класса. См. также раздел [написание поставщика методов](writing-a-method-provider.md).

</dd> <dt>

<span id="wmi.gloss_mib"></span><span id="WMI.GLOSS_MIB"></span>**MIB**
</dt> <dd>

См. статью *база данных управления (MIB)*.

</dd> <dt>

<span id="wmi.gloss_mof"></span><span id="WMI.GLOSS_MOF"></span>**MOF**
</dt> <dd>

См. *MOF-файл (MOF)*.

</dd> <dt>

<span id="wmi.gloss_mof_file"></span><span id="WMI.GLOSS_MOF_FILE"></span>**MOF-файл**
</dt> <dd>

Текстовый файл, содержащий определения классов и экземпляров, заданных с помощью языка *MOF-файл (MOF)* .

</dd> </dl>
