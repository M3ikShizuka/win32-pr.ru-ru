# <a name="__classmodificationevent-class"></a>\_\_Класс Классмодификатионевент

Класс System **\_ \_ классмодификатионевент** представляет событие изменения класса, которое представляет собой тип [встроенного события](determining-the-type-of-event-to-receive.md) , создаваемого при изменении класса в пространстве имен.

Приведенный ниже синтаксис является упрощенной версией кода MOF и включает все унаследованные свойства. Свойства перечислены в алфавитном порядке, а не в MOF.

## <a name="syntax"></a>Синтаксис

``` syntax
class __ClassModificationEvent : __ClassOperationEvent
{
  object PreviousClass;
  uint8  SECURITY_DESCRIPTOR[];
  object TargetClass;
  uint64 TIME_CREATED;
};
```

## <a name="remarks"></a>Remarks

Класс **\_ \_ классмодификатионевент** является производным от [**\_ \_ классоператионевент**](--classoperationevent.md).

Событие сообщает о новой и старой версиях определения класса.
