# <a name="swbemobjectpathclass-property"></a>Свбемобжектпас. class, свойство

Свойство **Class** объекта [**свбемобжектпас**](swbemobjectpath.md) — это имя класса, который является частью пути к объекту.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
SWbemObjectPath.Class As String
```
