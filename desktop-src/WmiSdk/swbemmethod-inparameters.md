# <a name="swbemmethodinparameters-property"></a>Свбеммесод. out, свойство

Свойство " **Parameters** " объекта [**Свбеммесод**](swbemmethod.md) является объектом [**SWbemObject**](swbemobject.md) , свойства которого определяют входные параметры для этого метода. Это свойство доступно только для чтения. Обратите внимание, что любые изменения, внесенные в этот объект, не отражаются в определении базового метода.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemMethod.InParameters As Object
```
