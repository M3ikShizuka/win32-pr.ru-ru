# <a name="swbemobjectpathdisplayname-property"></a>Свбемобжектпас. DisplayName, свойство

Свойство **DisplayName** объекта [**свбемобжектпас**](swbemobjectpath.md) представляет собой строку, содержащую путь в форме, которую можно использовать в качестве отображаемого имени моникера.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
SWbemObjectPath.DisplayName As String
```
