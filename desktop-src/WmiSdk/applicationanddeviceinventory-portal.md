# <a name="application-inventory-classes"></a>Классы инвентаризации приложений

## <a name="purpose"></a>Назначение

классы WMI для инвентаризации приложений позволяют обнаруживать установленные приложения Win32 и приложения Windows store в системе Windows.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                                  | Описание                                                                                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_InstalledWin32Program Win32**](win32-installedwin32program.md)<br/>         | Класс [**Win32 \_ InstalledWin32Program**](win32-installedwin32program.md) представляет установленное приложение Win32.<br/>                                                                                                                            |
| [**\_Инсталледсторепрограм Win32**](win32-installedstoreprogram.md)<br/>         | класс [**Win32 \_ инсталледсторепрограм**](win32-installedstoreprogram.md) представляет установленное приложение Windows store.<br/>                                                                                                                    |
| [**\_Инсталледпрограмфрамеворк Win32**](win32-installedprogramframework.md)<br/> | Класс [**Win32 \_ инсталледпрограмфрамеворк**](win32-installedprogramframework.md) представляет платформу технологии, которая компилируется в экземпляре [**Win32 \_ InstalledWin32Program**](win32-installedwin32program.md) или используется во время выполнения.<br/> |



 

## <a name="developer-audience"></a>Аудитория разработчиков

классы WMI для инвентаризации приложений предназначены для использования разработчиками решений Enterprise управления, а также средствами локального или удаленного администрирования, которые должны создавать инвентаризацию приложений в системе.
