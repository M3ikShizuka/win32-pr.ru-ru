# <a name="creating-a-sibling-namespace-with-mof-code"></a>Создание родственного пространства имен с помощью MOF-кода

Другим способом создания пространства имен является использование MOF-файл (MOF) кода для создания пространства имен одного уровня. Пространство имен одного уровня — это пространство имен, которое не существует в качестве дочернего для текущего пространства имен.

В следующей процедуре описывается создание пространства имен с общим пространством с помощью MOF-кода.

**Создание пространства имен с общим пространством с помощью MOF-кода**

1.  Перед объявлением пространства имен вставьте команду [**\# pragma Namespace**](pragma-namespace.md) в свой MOF-код.

    Команда [**\# pragma Namespace**](pragma-namespace.md) указывает инструментарию WMI, где создавать экземпляры после директивы.

2.  Создайте экземпляр класса [**\_ \_ Namespace**](--namespace.md) .
3.  Скомпилируйте код с помощью программы [mofcomp](mofcomp.md) или интерфейса [**имофкомпилер**](/windows/desktop/api/Wbemcli/nn-wbemcli-imofcompiler) .

    Дополнительные сведения см. в разделе [Компиляция MOF-файлов](compiling-mof-files.md).

В следующем примере MOF-кода показано, как создать пространство имен в качестве одноуровневого элемента для \\ пространства имен root CIMv2.

``` syntax
#pragma namespace("\\\\.\\Root")

instance of __Namespace 
{
    Name = "MyNamespace";
};
```
