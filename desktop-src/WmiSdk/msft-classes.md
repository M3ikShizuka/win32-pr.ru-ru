# <a name="msft-classes"></a>Классы MSFT

классы MSFT относятся к реализации Windows [*модель CIM*](gloss-c.md) (CIM).

Объекты можно разделить на несколько основных категорий:

-   [Классы поставщиков политик](policy-provider-classes.md)
-   [Классы устранения неполадок WMI](wmi-troubleshooting-classes.md)

Следующие классы MSFT определены в System. mof, но не реализуются инструментарием WMI.

-   MSFT \_ вмикоривент
-   MSFT \_ вмикорелогоффевент
-   MSFT \_ вмикорелогоневент
-   MSFT \_ вмикореобжект
-   MSFT \_ вмикорестатус
-   MSFT \_ вмикореусер
-   MSFT \_ вмитаскфаилуре
