# <a name="describing-the-ldap-namespace"></a>Описание пространства имен LDAP

При удаленном подключении к Active Directory серверу, отличному от того, который используется в текущем домене, необходимо указать имя компьютера, который уже является членом домена, принадлежащего нужному Active Directory.

Чтобы удаленно подключиться к компьютеру, который уже является членом домена, принадлежащего серверу Active Directory, введите следующую команду.

**\\\\\\LDAP корневого \\ каталога ремотекомпутер \\**

В этом примере показано, что состоит из двух частей, составляющих полное имя пространства имен. Первая часть ( \\ \\ ремотекомпутер) представляет компьютер, который уже является членом домена, принадлежащего нужному серверу Active Directory. Вторая часть ( \\ LDAP корневого \\ каталога \\ ) представляет схему Active Directory в поставщике служб каталогов.

> [!Note]  
> Дополнительные сведения о поддержке и установке этого компонента в определенной операционной системе см. в статье [доступность компонентов WMI в операционной системе](operating-system-availability-of-wmi-components.md).
