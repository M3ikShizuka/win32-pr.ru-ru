# <a name="swbemrefreshableitemrefresher-property"></a>Свбемрефрешаблеитем. Обновитель, свойство

Свойство **свбемрефрешаблеитем. Обновитель** представляет родительский объект [**свбемрефрешер**](swbemrefresher.md) , содержащий объект [**свбемрефрешаблеитем**](swbemrefreshableitem.md) .

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemRefreshableItem.Refresher As Object
```
