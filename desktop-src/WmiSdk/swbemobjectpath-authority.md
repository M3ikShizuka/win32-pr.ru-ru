# <a name="swbemobjectpathauthority-property"></a>Свбемобжектпас. Authority, свойство

Свойство **Authority** объекта [**свбемобжектпас**](swbemobjectpath.md) содержит строку, определяющую компонент-центр пути к объекту. Дополнительные сведения см. в описании параметра *страусорити* метода [**SWbemLocator. коннектсервер**](swbemlocator-connectserver.md) и [настройке параметров безопасности для IWbemServices и других прокси-серверов](setting-the-security-on-iwbemservices-and-other-proxies.md).

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
SWbemObjectPath.Authority As String
```
