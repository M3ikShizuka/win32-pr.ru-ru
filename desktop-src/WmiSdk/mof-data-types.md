# <a name="mof-data-types"></a>Типы данных MOF

Язык MOF-файл (MOF) поддерживает множество типов данных, которые используются для присвоения значений свойствам, квалификаторам и параметрам методов.

Язык MOF поддерживает следующие типы данных и категории типов данных:

-   [Точечный рисунок и Битвалуес](bitmap-and-bitvalues.md)
-   [boolean](boolean.md)
-   [DATETIME](date-and-time-format.md)
-   [Массивы MOF](mof-arrays.md)
-   [Показатели](numbers.md)
-   [ОБЪЕКТАМИ](object.md)
-   [Справочные материалы](references.md)
-   [Строки](strings.md)
-   [Схема значения](value-map.md)
-   [VOID](void.md)

Дополнительные сведения о командах см. в разделе [команды препроцессора](preprocessor-commands.md).
