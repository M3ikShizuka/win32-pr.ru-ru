# <a name="swbemobjectpathparentnamespace-property"></a>Свбемобжектпас. Парентнамеспаце, свойство

Свойство **парентнамеспаце** объекта [**свбемобжектпас**](swbemobjectpath.md) содержит имя родительского пространства имен, которое является частью пути к объекту. Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemObjectPath.ParentNamespace As String
```
