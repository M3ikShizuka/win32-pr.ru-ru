# <a name="swbemqualifiername-property"></a>SWbemQualifier.Name, свойство

Свойство **Name** объекта [**свбемкуалифиер**](swbemqualifier.md) определяет имя этого квалификатора. Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemQualifier.Name As String
```
