# <a name="swbemobjectpathserver-property"></a>Свбемобжектпас. Server, свойство

Свойство **Server** объекта [**свбемобжектпас**](swbemobjectpath.md) содержит имя сервера, который является частью пути к объекту.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
SWbemObjectPath.Server As String
```
