# <a name="swbemmethodset-object"></a>Объект Свбеммесодсет

Объект **свбеммесодсет** — это коллекция объектов [**свбеммесод**](swbemmethod.md) . Элементы извлекаются с помощью метода [**Item**](swbemmethodset-item.md) . Дополнительные сведения см. [в разделе доступ к коллекции](accessing-a-collection.md). Не удается создать этот объект с **помощью вызова функции** VBScript.

> [!Note]  
> В этой версии API доступ на запись к сведениям о методе не поддерживается. Если необходимо определить методы или изменить существующие определения методов, можно определить изменения метода в MOF-файле и отправить изменения с помощью компилятора MOF. Кроме того, можно использовать API WMI COM.

| Метод                              | Описание                                                                                                                                  |
