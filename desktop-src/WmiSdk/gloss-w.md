# <a name="w-wmi"></a>W (WMI)

[A](gloss-a.md) B [C](gloss-c.md) [D](gloss-d.md) [E](gloss-e.md) [F](gloss-f.md) G [р](gloss-h.md) [.](gloss-i.md) J [K](gloss-k.md) [L](gloss-l.md) [M](gloss-m.md) [N](gloss-n.md) [O](gloss-o.md) [P](gloss-p.md) [Q](gloss-q.md) [R](gloss-r.md) [S](gloss-s.md) [T](gloss-t.md) U V W X Y Z

<dl> <dt>

<span id="wmi.gloss_win32_schema"></span><span id="WMI.GLOSS_WIN32_SCHEMA"></span>**Схема Win32**
</dt> <dd>

расширение майкрософт для [*схемы CIM*](gloss-c.md) , которое содержит определения классов и экземпляров для представления [*управляемых объектов*](gloss-m.md) , существующих в типичной Windows среде.

</dd> <dt>

<span id="wmi.gloss_wmi_class"></span><span id="WMI.GLOSS_WMI_CLASS"></span>**Класс WMI**
</dt> <dd>

Шаблон для типа [*управляемого объекта*](gloss-m.md). Классы WMI определяют базовые единицы управления.

</dd> <dt>

<span id="wmi.gloss_wmi_class_provider"></span><span id="WMI.GLOSS_WMI_CLASS_PROVIDER"></span>**Поставщик классов WMI**
</dt> <dd>

COM-сервер, предоставляющий определения классов. Поставщики классов реализуют методы в интерфейсе [**IWbemServices**](/windows/desktop/api/WbemCli/nn-wbemcli-iwbemservices) для поддержки извлечения, изменения, удаления, перечисления и обработки запросов данных.

</dd> <dt>

<span id="wmi.gloss_wmi_control"></span><span id="WMI.GLOSS_WMI_CONTROL"></span>**Элемент управления WMI**
</dt> <dd>

Оснастка консоли управления (MMC), которая позволяет пользователю настраивать параметры WMI на локальном или удаленном компьютере.

</dd> <dt>

<span id="wmi.gloss_wmi_for_windows_driver_model"></span><span id="WMI.GLOSS_WMI_FOR_WINDOWS_DRIVER_MODEL"></span>**инструментарий WMI для WDM (WDM)**
</dt> <dd>

Компонент низкого уровня, обеспечивающий доступ между WMI и аппаратными устройствами.

</dd> <dt>

<span id="wmi.gloss_wmi_odbc_adapter"></span><span id="WMI.GLOSS_WMI_ODBC_ADAPTER"></span>**ODBC адаптер WMI**
</dt> <dd>

интерфейс, который позволяет приложению получать информацию из управления Windows с помощью *язык запросов WMI*. Это предоставляет стандартный API, позволяющий приложениям на основе ODBC использовать данные в [*репозитории CIM*](gloss-c.md) , как если бы они были базой данных.

</dd> <dt>

<span id="wmi.gloss_wmi_query_language"></span><span id="WMI.GLOSS_WMI_QUERY_LANGUAGE"></span>**Язык запросов WMI (WQL)**
</dt> <dd>

подмножество языка структурированных запросов (SQL) с расширениями для поддержки уведомлений о событиях инструментария wmi и других компонентов, относящихся к инструментарию wmi.

</dd> <dt>

<span id="wmi.gloss_wmi_method"></span><span id="WMI.GLOSS_WMI_METHOD"></span>**Метод WMI**
</dt> <dd>

Метод, реализованный API WMI [com](com-api-for-wmi.md) или [API скриптов](scripting-api-for-wmi.md). Два примера представляют собой COM-метод [**IWbemServices:: ExecMethod**](/windows/desktop/api/WbemCli/nf-wbemcli-iwbemservices-execmethod) и аналогичный метод API создания скриптов [**SWbemServices.Exeкмесод**](swbemservices-execmethod.md).

</dd> <dt>

<span id="wmi.gloss_wmi_repository"></span><span id="WMI.GLOSS_WMI_REPOSITORY"></span>**Репозиторий WMI**
</dt> <dd>

Центральная область хранения, управляемая [*Диспетчер объектов CIM*](gloss-c.md).

</dd> <dt>

<span id="wmi.gloss_wql"></span><span id="WMI.GLOSS_WQL"></span>**ВОДИТ**
</dt> <dd>

См. *язык запросов WMI (WQL)*.

</dd> </dl>
