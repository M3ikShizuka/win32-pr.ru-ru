# <a name="__classoperationevent-class"></a>\_\_Класс Классоператионевент

Класс System **\_ \_ классоператионевент** является базовым классом для всех внутренних событий, связанных с классом.

Приведенный ниже синтаксис является упрощенной версией кода MOF и включает все унаследованные свойства. Свойства перечислены в алфавитном порядке, а не в MOF.

## <a name="syntax"></a>Синтаксис

``` syntax
class __ClassOperationEvent : __Event
{
  uint8  SECURITY_DESCRIPTOR[];
  object TargetClass;
  uint64 TIME_CREATED;
};
```

### <a name="properties"></a>Элемент Property

Класс **\_ \_ классоператионевент** имеет следующие свойства.

<dl> <dt>

**\_дескриптор безопасности**
</dt> <dd> <dl> <dt>

Тип данных: массив **Uint8**
</dt> <dt>

Тип доступа: только для чтения
</dt> </dl>

Дескриптор, используемый поставщиком событий для определения того, какие пользователи могут получить событие. Это свойство наследуется от [**\_ \_ события**](--event.md).

</dd> <dt>

**TargetClass**
</dt> <dd> <dl> <dt>

Тип данных: **объект**
</dt> <dt>

Тип доступа: только для чтения
</dt> </dl>

Класс, затронутый событием. Для событий создания это только что созданный класс. Для событий изменения это новая версия измененного класса. Для событий удаления это удаленный класс.

</dd> <dt>

**ВРЕМЯ \_ создания**
</dt> <dd> <dl> <dt>

Тип данных: **UInt64**
</dt> <dt>

Тип доступа: только для чтения
</dt> </dl>

Уникальное значение, указывающее время создания события. Это 64-разрядное значение, представляющее число 100-наносекундных интервалов после 1 января 1601 г. Эти сведения находятся в формате всеобщего скоординированного времени (UTC). Это свойство наследуется от [**\_ \_ события**](--event.md).

Дополнительные сведения об использовании значений **UInt64** в скриптах см. [в разделе Создание сценариев в WMI](/windows/desktop/WmiSdk/creating-a-wmi-script).

</dd> </dl>

## <a name="remarks"></a>Remarks

Класс **\_ \_ классоператионевент** является производным от [**\_ \_ события**](--event.md).

Экземпляры **\_ \_ классоператионевент** не создаются; создаются только экземпляры его подклассов. Следующие классы являются производными от **\_ \_ классоператионевент**:

[**\_\_класскреатионевент**](--classcreationevent.md)

[**\_\_классмодификатионевент**](--classmodificationevent.md)

[**\_\_классделетионевент**](--classdeletionevent.md)
