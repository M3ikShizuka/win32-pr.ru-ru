# <a name="swbemnamedvaluesetdeleteall-method"></a>Свбемнамедвалуесет. DeleteAll, метод

Метод **DeleteAll** объекта [**свбемнамедвалуесет**](swbemnamedvalueset.md) удаляет все именованные значения из коллекции, тем самым удаляя их.

## <a name="syntax"></a>Синтаксис


```VB
SWbemNamedValueSet.DeleteAll()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

После завершения метода **DeleteAll** объект **Err** может содержать приведенный ниже код ошибки.

<dl> <dt>

**вбемеррфаилед** -2147749889 (0x80041001)
</dt> <dd>

Незаданная ошибка.

</dd> </dl>
