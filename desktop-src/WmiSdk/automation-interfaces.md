# <a name="automation-interfaces"></a>Интерфейсы автоматизации

API автоматизации делает интерфейсы API скриптов для WMI доступными программистам C/C++ вместо версий COM API.

В следующей таблице перечислены объекты WMI и их использование в API автоматизации. Ссылка на перекрестные ссылки на документацию по интерфейсу для API скриптов для эквивалентов WMI.



| Объект автоматизации                                 | Описание                                                                                                                 |
|---------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [**исвбемевентсаурце**](swbemeventsource.md)     | Получает события в сочетании с [**ISWbemServices.Exeкнотификатионкуери**](swbemservices-execnotificationquery.md).   |
| [**исвбемластеррор**](swbemlasterror.md)         | Предоставляет расширенные сведения об ошибке при возникновении ошибки.                                                                   |
| [**исвбемлокатор**](swbemlocator.md)             | Получает ссылку на объект [**исвбемсервицес**](swbemservices.md) , который может получать доступ к инструментарию WMI на определенном компьютере узла. |
| [**исвбеммесод**](swbemmethod.md)               | Содержит определение одного метода.                                                                                        |
| [**исвбеммесодсет**](swbemmethodset.md)         | Получает доступ к коллекции объектов [**исвбеммесод**](swbemmethod.md) .                                                 |
| [**исвбемнамедвалуе**](swbemnamedvalue.md)       | Содержит одно именованное значение.                                                                                              |
| [**исвбемнамедвалуесет**](swbemnamedvalueset.md) | Получает доступ к коллекции объектов [**исвбемнамедвалуе**](swbemnamedvalue.md) .                                         |
| [**исвбемобжект**](swbemobject.md)               | Содержит класс или экземпляр объекта модель CIM (CIM) и манипулирует им.                                  |
| [**исвбемобжектпас**](swbemobjectpath.md)       | Создает и проверяет путь к объекту.                                                                                     |
| [**исвбемобжектсет**](swbemobjectset.md)         | Получает доступ к коллекции объектов [**исвбемобжект**](swbemobject.md) .                                                 |
| [**исвбемпривилеже**](swbemprivilege.md)         | Задает или очищает привилегию.                                                                                                 |
| [**исвбемпривилежесет**](swbemprivilegeset.md)   | Получает доступ к коллекции объектов [**исвбемпривилеже**](swbemprivilege.md) .                                           |
| [**исвбемпроперти**](swbemproperty.md)           | Используется для хранения одного свойства CIM.                                                                                      |
| [**исвбемпропертисет**](swbempropertyset.md)     | Получение доступа к коллекции объектов [**исвбемпроперти**](swbemproperty.md) .                                              |
| [**исвбемкуалифиер**](swbemqualifier.md)         | Содержит квалификатор одного класса.                                                                                          |
| [**исвбемкуалифиерсет**](swbemqualifierset.md)   | Получает доступ к коллекции объектов [**исвбемкуалифиер**](swbemqualifier.md) .                                           |
| [**исвбемсекурити**](swbemsecurity.md)           | Управляет параметрами безопасности, такими как привилегия модели COM, Аусентикатионлевел и Имперсонатионлевел.      |
| [**исвбемсервицес**](swbemservices.md)           | Создает, обновляет и извлекает экземпляры или классы.                                                                       |
| [**исвбемсинк**](swbemsink.md)                   | Получает результаты асинхронных операций клиентского приложения и уведомлений о событиях.                                 |
