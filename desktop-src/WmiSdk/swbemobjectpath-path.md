# <a name="swbemobjectpathpath-property"></a>Свбемобжектпас. Path, свойство

Свойство **path** объекта [**свбемобжектпас**](swbemobjectpath.md) содержит абсолютный путь. Это то же самое, что и свойство [ \_ \_ path](wmi-system-properties.md) в COM-API. Это свойство данного объекта по умолчанию.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
SWbemObjectPath.Path As String
```
