# <a name="swbemqualifierisoverridable-property"></a>Свойство Свбемкуалифиер. Overridable

Свойство **Overridable** объекта [**свбемкуалифиер**](swbemqualifier.md) является логическим значением, указывающим, может ли этот квалификатор быть переопределен при распространении.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
SWbemQualifier.IsOverridable As Boolean
```
