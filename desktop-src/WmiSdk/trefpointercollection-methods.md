# <a name="trefpointercollection-methods"></a>Методы Трефпоинтерколлектион

\[Класс [**трефпоинтерколлектион**](/windows/desktop/api/RefPtrCo/nl-refptrco-trefpointercollection) является частью платформы поставщика WMI, которая теперь считается окончательным состоянием, и дальнейшая разработка, улучшения и обновления не будут доступны для проблем, связанных с безопасностью, затрагивающих эти библиотеки. [API MI](/previous-versions/windows/desktop/wmi_v2/windows-management-infrastructure) следует использовать для всех новых разработок.\]
