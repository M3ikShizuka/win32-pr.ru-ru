# <a name="chstringformatmessagew-methods"></a>Методы Чстринг:: Форматмессажев

\[Класс [**чстринг**](chstring.md) является частью платформы поставщика WMI, которая теперь считается окончательным состоянием, и дальнейшая разработка, улучшения и обновления не будут доступны для проблем, связанных с безопасностью, затрагивающих эти библиотеки. [API MI](/previous-versions/windows/desktop/wmi_v2/windows-management-infrastructure) следует использовать для всех новых разработок.\]

Перегруженный метод **форматмессажев** форматирует строку сообщения.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                              | Описание                                                                                     |
|:--------------------------------------------------------------------|:------------------------------------------------------------------------------------------------|
| [**Форматмессажев (UINT)**](/previous-versions/windows/desktop/legacy/aa385519(v=vs.85))       | Форматирует это сообщение в соответствии с идентификатором ресурса, указанным в параметре **uint**.<br/> |
| [**Форматмессажев (ЛПКВСТР)**](/windows/win32/api/chstring/nf-chstring-chstring-formatmessagew(lpcwstr_---)) | Форматирует эту строку сообщения в соответствии с форматом, заданным параметром **лпквстр**.<br/>    |
