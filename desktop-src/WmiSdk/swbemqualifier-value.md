# <a name="swbemqualifiervalue-property"></a>Свбемкуалифиер. Value, свойство

Свойство **value** объекта [**свбемкуалифиер**](swbemqualifier.md) содержит значение Variant этого квалификатора. Это свойство данного объекта по умолчанию.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
SWbemQualifier.Value As Variant
```
