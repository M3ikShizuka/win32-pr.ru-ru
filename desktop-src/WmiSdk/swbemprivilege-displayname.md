# <a name="swbemprivilegedisplayname-property"></a>Свбемпривилеже. DisplayName, свойство

Свойство **DisplayName** объекта [**свбемпривилеже**](swbemprivilege.md) представляет собой строку, в которой отображается описание объекта **свбемпривилеже** . Например, привилегия, определяющая, может ли пользователь выполнить метод завершения работы объекта, имеет строку «завершение работы системы» в свойстве **свбемпривилеже. DisplayName** . Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemPrivilege.DisplayName As 
```
