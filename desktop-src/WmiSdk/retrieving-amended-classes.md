# <a name="retrieving-amended-classes"></a>Извлечение измененных классов

Можно получить и изменить измененный класс, указав языковой стандарт и установив флаг, указывающий, что требуется локализованная информация. Процедура, используемая для получения определений локализованных классов, зависит от способа подключения к инструментарию WMI. Дополнительные сведения см. в статьях [получение исправленных классов с помощью API скриптов для WMI](retrieving-amended-classes-using-the-scripting-api-for-wmi.md) и [извлечение измененных классов с помощью API COM для WMI](retrieving-amended-classes-using-the-com-api-for-wmi.md).
