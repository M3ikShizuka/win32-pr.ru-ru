# <a name="swbemmethodsetcount-property"></a>Свбеммесодсет. Count, свойство

Свойство **Count** объекта [**свбеммесодсет**](swbemmethodset.md) определяет количество элементов в коллекции. Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemMethodSet.Count As Integer
```
