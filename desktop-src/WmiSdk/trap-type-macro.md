# <a name="trap-type-macro"></a>Макрос типа ЛОВУШКи

Для единообразия каждый экземпляр макроса типа "ЛОВУШКа" сначала сопоставляется с экземпляром макроса [типа уведомления](notification-type-macro.md) с помощью сопоставления, определенного в [RFC 1908](https://www.ietf.org/rfc/rfc1908.txt).

> [!Note]  
> Дополнительные сведения об установке поставщика см. в разделе [Настройка среды SNMP WMI](setting-up-the-wmi-snmp-environment.md).
