# <a name="swbemprivilegesetdeleteall-method"></a>Свбемпривилежесет. DeleteAll, метод

Метод **DeleteAll** объекта [**свбемпривилежесет**](swbemprivilegeset.md) удаляет все привилегии из коллекции, тем самым удаляя его.

## <a name="syntax"></a>Синтаксис


```VB
SWbemPrivilegeSet.DeleteAll()
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.

## <a name="error-codes"></a>Коды ошибок

После завершения метода **DeleteAll** объект **Err** может содержать приведенный ниже код ошибки.

<dl> <dt>

**вбемеррфаилед** -2147749889 (0x80041001)
</dt> <dd>

Незаданная ошибка.

</dd> </dl>
