# <a name="preprocessor-commands"></a>Команды препроцессора

Команда препроцессора — это строка кода, помещенная в MOF-файл, управляющая способом компиляции кода компилятором MOF. При компиляции MOF-файлов используются две команды препроцессора:

-   [\#относится](-include.md)
-   [\#pragma](-pragma.md)
