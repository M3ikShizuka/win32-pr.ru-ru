# <a name="swbemnamedvaluevalue-property"></a>Свбемнамедвалуе. Value, свойство

Свойство **value** объекта [**свбемнамедвалуе**](swbemnamedvalue.md) возвращает значение типа Variant элемента **свбемнамедвалуе** . Это свойство по умолчанию для объектов **свбемнамедвалуе** . Изменения, вносимые в свойство Value, автоматически отражаются в коллекции [**свбемнамедвалуесет**](swbemnamedvalueset.md) , к которой принадлежит объект **свбемнамедвалуе** .

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
SWbemNamedValue.Value As Variant
```
