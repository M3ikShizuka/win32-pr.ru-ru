# <a name="access-and-max-access-clauses"></a>Предложения доступа и MAX-ACCESS

Каждое определение объекта MIB содержит предложение доступа (в формате CLR) или MAX-ACCESS (SNMPv2C), которое определяет права доступа к объекту.

> [!Note]  
> Дополнительные сведения об установке поставщика см. в разделе [Настройка среды SNMP WMI](setting-up-the-wmi-snmp-environment.md).

 

В следующей таблице приведено сопоставление предложения доступа в некотором списке.



| Предложение доступа MIB | Квалификатор CIM       |
|-------------------|---------------------|
| Только для чтения         | **Чтение**            |
| чтение и запись        | **Чтение**, **запись** |
| только запись        | **Запись**           |
| недоступно    | **\_Недоступно** |



 

В следующей таблице перечислены сопоставления предложения MAX-ACCESS SNMPv2C.



| MIB — предложение ACCESS     | Квалификатор CIM       |
|-----------------------|---------------------|
| Только для чтения             | **Чтение**            |
| чтение и запись            | **Чтение**, **запись** |
| для чтения и создания           | **Чтение**            |
| доступный — для — уведомление | **\_Недоступно** |
| недоступно        | **\_Недоступно** |



 

Если объект MIB сопоставлен недоступному и не является свойством с ключом класса, сопоставление самого объекта MIB отсутствует.
