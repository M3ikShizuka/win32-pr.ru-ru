# <a name="qualifiers-specific-to-the-snmp-provider"></a>Квалификаторы, характерные для поставщика SNMP

Квалификаторы содержат зависящую от реализации контекстную информацию и свойства транспорта, определяющие, как поставщик SNMP обращается к агенту SNMP. Ниже перечислены квалификаторы, уникальные для поставщика SNMP.

<dt>

<span id="AgentAddress_"></span><span id="agentaddress_"></span><span id="AGENTADDRESS_"></span>**ажентаддресс** 
</dt> <dd>

Тип данных: **\_ строка CIM**

Адрес транспорта, связанный с агентом SNMP, например IP-адрес или DNS-имя. Необходимо задать для **ажентаддресс** допустимый адрес узла одноадресной рассылки или имя узла домена, которое можно разрешить с помощью процесса разрешения имен доменов операционной системы. **Ажентаддресс** не имеет значения по умолчанию.

</dd> <dt>

<span id="AgentFlowControlWindowSize_"></span><span id="agentflowcontrolwindowsize_"></span><span id="AGENTFLOWCONTROLWINDOWSIZE_"></span>**ажентфловконтролвиндовсизе** 
</dt> <dd>

Тип данных: **CIM \_ SINT32**

Максимальное количество параллельно необработанных SNMP-запросов, которые могут быть отправлены этому агенту, пока ответ еще не получен. Запрос SNMP считается ожидающим, пока не будет получен ответ PDU или истекло время его ожидания. Большой размер окна обычно повышает производительность, но некоторые агенты могут не работать при большой нагрузке. **Ажентфловконтролвиндовсизе** может иметь значение 0, чтобы указать неограниченный размер окна или целое число от 1 до 2 ^ 32-1. Значение по умолчанию — 10. Этот квалификатор является необязательным.

</dd> <dt>

<span id="AgentReadCommunityName_"></span><span id="agentreadcommunityname_"></span><span id="AGENTREADCOMMUNITYNAME_"></span>**ажентреадкоммунитинаме** 
</dt> <dd>

Тип данных: **\_ строка CIM**

Строка октета переменной длины, используемая агентом SNMP для проверки подлинности блока данных протокола SNMP во время операции чтения (следующие запросы SNMP GET и GET). Имя сообщества должно быть сопоставлено с строкой Юникода и содержать только буквенно-цифровые символы. Значение по умолчанию — Public. Этот квалификатор является необязательным.

</dd> <dt>

<span id="AgentRetryCount_"></span><span id="agentretrycount_"></span><span id="AGENTRETRYCOUNT_"></span>**ажентретрикаунт** 
</dt> <dd>

Тип данных: **CIM \_ SINT32**

Количество повторных попыток одного SNMP-запроса при отсутствии ответа от агента SNMP до того, как запрос будет считаться неудачным. Для **ажентретрикаунт** должно быть задано целое число от 0 до 2 ^ 32-1. Значение по умолчанию — 1. Этот квалификатор является необязательным.

</dd> <dt>

<span id="AgentRetryTimeout_"></span><span id="agentretrytimeout_"></span><span id="AGENTRETRYTIMEOUT_"></span>**ажентретритимеаут** 
</dt> <dd>

Тип данных: **CIM \_ SINT32**

Время в миллисекундах, по истечении которого будет считаться, что единица данных протокола SNMP была удалена. Число миллисекунд, в течение которых ожидается ответ на SNMP-запрос, отправленный агенту SNMP. Для **ажентретритимеаут** должно быть задано целое число от 0 до 2 ^ 32-1. Значение по умолчанию — 500. Этот квалификатор является необязательным.

</dd> <dt>

<span id="AgentSNMPVersion_"></span><span id="agentsnmpversion_"></span><span id="AGENTSNMPVERSION_"></span>**ажентснмпверсион** 
</dt> <dd>

Тип данных: **\_ строка CIM**

Версия протокола SNMP, используемая при взаимодействии с агентом SNMP. В настоящее время допустимыми значениями являются 1 (для CLR) и 2C (для SNMPv2C). Значение по умолчанию — 1. Этот квалификатор является необязательным.

</dd> <dt>

<span id="AgentTransport_"></span><span id="agenttransport_"></span><span id="AGENTTRANSPORT_"></span>**аженттранспорт** 
</dt> <dd>

Тип данных: **\_ строка CIM**

Транспортный протокол, используемый для взаимодействия с агентом SNMP. в настоящее время единственными допустимыми значениями являются протоколы ip и internet Packet Exchange (IPX). Значение по умолчанию — IP. Этот квалификатор является необязательным.

</dd> <dt>

<span id="AgentVarBindsPerPdu_"></span><span id="agentvarbindsperpdu_"></span><span id="AGENTVARBINDSPERPDU_"></span>**ажентварбиндсперпду** 
</dt> <dd>

Тип данных: **CIM \_ SINT32**

Максимальное число привязок переменных, содержащихся в одном PDU SNMP. Каждый SNMP-запрос, отправленный агенту SNMP, содержит одну или несколько переменных SNMP. Этот параметр задает максимальное число переменных, которые могут быть добавлены в один запрос. **Ажентварбиндсперпду** может иметь значение 0 (ноль), чтобы реализация могла определить оптимальные привязки переменных или целое число от 1 до 2 ^ 32-1. Обратите внимание, что хотя увеличение этого значения может повысить производительность, некоторые агенты и сети не могут работать с полученным запросом. Значение по умолчанию — 10. Этот квалификатор является необязательным.

</dd> <dt>

<span id="AgentWriteCommunityName_"></span><span id="agentwritecommunityname_"></span><span id="AGENTWRITECOMMUNITYNAME_"></span>**ажентвритекоммунитинаме** 
</dt> <dd>

Тип данных: **\_ строка CIM**

Строка октета переменной длины, используемая агентом SNMP для проверки подлинности блока данных протокола SNMP во время операции записи (запрос SNMP SET). Имя сообщества должно быть сопоставлено с строкой Юникода и содержать только буквенно-цифровые символы. Значение по умолчанию — Public. Этот квалификатор является необязательным.

</dd> </dl>
