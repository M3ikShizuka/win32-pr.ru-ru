# <a name="swbemrefreshableitemindex-property"></a>Свбемрефрешаблеитем. index, свойство

Свойство **свбемрефрешаблеитем. index** представляет индекс элемента в родительском объекте [**свбемрефрешер**](swbemrefresher.md) .

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
SWbemRefreshableItem.Index As Long
```
