# <a name="cim-class-qualifiers-for-mib-objects"></a>Квалификаторы класса CIM для объектов MIB

Поставщик SNMP использует следующие квалификаторы класса CIM при сопоставлении определений объектов MIB с определениями классов CIM. Чтобы поставщик SNMP полностью разрешил объект группы, должен присутствовать обязательный квалификатор. Ошибка при определении обязательного квалификатора возвращает ошибку. Указание недопустимого значения квалификатора также возвращает ошибку.

> [!Note]  
> Дополнительные сведения об установке поставщика см. в разделе [Настройка среды SNMP WMI](setting-up-the-wmi-snmp-environment.md).

 



| Квалификатор           | Описание                                                                                                                             |
|---------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| **Описание**     | **строка** Используемых<br/> Описание класса.<br/>                                                                      |
| **\_ObjectID группы** | **строка** Является обязательным, если класс предназначен для поставщика класса SNMP.<br/> Идентификатор объекта собранной коллекции.<br/> |
| **\_Импорты модулей** | **строка** Используемых<br/> Разделенный запятыми список имен модулей MIB, используемых для разрешения операций импорта.<br/>                              |
| **\_Имя модуля**    | **строка** Используемых<br/> Имя удостоверения модуля MIB.<br/>                                                                 |
| **Ссылки**       | **строка** Используемых<br/> Ссылается на другой документ, содержащий дополнительные сведения о классе.<br/>                     |
| **Одноэлементный**       | **Bool (логическое** ) Обязателен для классов, сопоставленных из скалярных коллекций.<br/> Указывает класс, который может иметь только один экземпляр.<br/>  |
