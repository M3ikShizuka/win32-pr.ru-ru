# <a name="swbempropertyvalue-property"></a>SWbemProperty. Value, свойство

Свойство **value** объекта [**SWbemProperty**](swbemproperty.md) определяет значение Variant свойства WMI. Это свойство автоматизации данного объекта по умолчанию.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
SWbemProperty.Value As Variant
```
