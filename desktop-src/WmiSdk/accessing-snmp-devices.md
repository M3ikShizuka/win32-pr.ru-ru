# <a name="accessing-snmp-devices"></a>Доступ к SNMP-устройствам

Поставщик SNMP позволяет клиентским приложениям получать доступ к информации SNMP через WMI. [Поставщик SNMP](snmp-provider.md) выступает в качестве шлюза для систем и устройств, использующих SNMP для управления. Только на компьютере управления или мониторинга должен быть установлен инструментарий WMI, так как он может выполнять чтение с любого устройства с поддержкой SNMP.

> [!Note]  
> Дополнительные сведения об установке поставщика см. в разделе [Настройка среды SNMP WMI](setting-up-the-wmi-snmp-environment.md).

 

Переменные объектов MIB могут быть считаны и записаны в, а SNMP-ловушки могут быть автоматически сопоставлены с событиями WMI, которые могут быть определены для любых произвольных операций создания, удаления или обновления данных за пределами ловушек, определенных MIB. Эта функция инструментария WMI работает в качестве расширения стандартных возможностей SNMP. Дополнительные сведения см. в разделе [наблюдение за событиями](monitoring-events.md).
