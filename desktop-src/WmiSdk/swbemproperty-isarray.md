# <a name="swbempropertyisarray-property"></a>SWbemProperty. IsArray, свойство

Свойство **IsArray** объекта [**SWbemProperty**](swbemproperty.md) является логическим значением, которое можно использовать, чтобы определить, имеет ли это свойство тип массива. Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemProperty.IsArray As Boolean
```
