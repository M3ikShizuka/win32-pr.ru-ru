# <a name="swbemprivilegeidentifier-property"></a>Свбемпривилеже. identifier, свойство

Свойство **идентификатора** объекта [**Свбемпривилеже**](swbemprivilege.md) представляет собой [вбемпривилежеенум](/windows/desktop/api/Wbemdisp/ne-wbemdisp-wbemprivilegeenum) целое число, представляющее заданную или получаемую привилегию. Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно для чтения и записи.

## <a name="syntax"></a>Синтаксис


```VB
SWbemPrivilege.Identifier As Integer
```
