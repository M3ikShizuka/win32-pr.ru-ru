# <a name="cinstancesetwbemint64-methods"></a>Методы CInstance:: SetWBEMINT64

\[Класс [**CInstance**](/windows/win32/api/instance/nl-instance-cinstance) является частью платформы поставщика WMI, которая теперь считается окончательным состоянием, и дальнейшая разработка, улучшения и обновления не будут доступны для проблем, связанных с безопасностью, затрагивающих эти библиотеки. [API MI](/previous-versions/windows/desktop/wmi_v2/windows-management-infrastructure) следует использовать для всех новых разработок.\]

Метод [**CInstance:: GetWBEMINT64**](cinstance-getwbemint64.md) задает свойство 64-разрядного целого числа.
