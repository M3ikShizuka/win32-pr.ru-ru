# <a name="swbemobjectpathsetasclass-property"></a>Свбемобжектпас. Сетаскласс, свойство

Метод **сетаскласс** объекта [**свбемобжектпас**](swbemobjectpath.md) заставляет путь обращаться к классу WMI.

Это свойство доступно только на запись.

## <a name="syntax"></a>Синтаксис


```VB
SWbemObjectPath.SetAsClass
```



## <a name="property-value"></a>Значение свойства

## <a name="error-codes"></a>Коды ошибок

После получения или задания свойства **сетаскласс** объект **Err** может содержать приведенный ниже код ошибки.

<dl> <dt>

**вбемеррфаилед** -2147749889 (0x80041001)
</dt> <dd>

Незаданная ошибка.

</dd> </dl>
