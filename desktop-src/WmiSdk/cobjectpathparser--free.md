# <a name="cobjectpathparserfree-methods"></a>Методы Кобжектпаспарсер:: Free

\[Класс [**кобжектпаспарсер**](/windows/win32/api/objpath/nl-objpath-cobjectpathparser) является частью платформы поставщика WMI, которая теперь считается окончательным состоянием, и дальнейшая разработка, улучшения и обновления не будут доступны для проблем, связанных с безопасностью, затрагивающих эти библиотеки. [API MI](/previous-versions/windows/desktop/wmi_v2/windows-management-infrastructure) следует использовать для всех новых разработок.\]

Перегруженный метод, который освобождает память, содержащую путь.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                     | Описание                                                                          |
|:---------------------------------------------------------------------------|:-------------------------------------------------------------------------------------|
| [**Free (LPWSTR)**](/windows/win32/api/objpath/nf-objpath-cobjectpathparser-free(lpwstr))                     | Освобождает память, содержащую неразобранный путь.<br/>                      |
| [**Бесплатный (Парседобжектпас)**](/windows/win32/api/objpath/nf-objpath-cobjectpathparser-free(parsedobjectpath)) | Освобождает память, содержащую структуру с проанализированным путем.<br/> |
