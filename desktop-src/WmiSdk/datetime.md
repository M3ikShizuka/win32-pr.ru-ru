# <a name="datetime"></a>DATETIME

Тип данных DATETIME — это строка VT BSTR фиксированной длины \_ , представляющая определенную дату и время или интервал. Дополнительные сведения о DATETIME в качестве даты см. в разделе [Формат даты и времени](date-and-time-format.md). Дополнительные сведения о ЗНАЧЕНИи DATETIME в качестве интервала см. в разделе [Формат интервала](interval-format.md).
