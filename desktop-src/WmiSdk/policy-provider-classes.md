# <a name="policy-provider-classes"></a>Классы поставщиков политик

Поставщик политик WMI предоставляет расширения для групповой политики с помощью классов, позволяя выполнять уточнения в приложении политики. Политика задается с помощью язык запросов WMI (WQL) и размещается в Active Directory для простоты развертывания.

В следующей таблице перечислены классы для поставщика политики WMI.



| Класс                                               | Описание                                                       |
|-----------------------------------------------------|-------------------------------------------------------------------|
| [**\_Поставщики MSFT**](/previous-versions/windows/desktop/wmisystemprov/msft-providers) | Предоставляет конфигурацию, относящуюся к экземплярам поставщика.             |
| [**\_Правило MSFT**](/previous-versions/windows/desktop/policmanprov/msft-rule)                | Определяет одно правило в области управления (SOM).         |
| [**MSFT \_ сомфилтер**](/previous-versions/windows/desktop/policmanprov/msft-somfilter)      | Предоставляет список правил, которые оцениваются на целевом компьютере. |
