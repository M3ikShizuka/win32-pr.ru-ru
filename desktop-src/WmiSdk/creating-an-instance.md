# <a name="creating-an-instance"></a>Создание экземпляра

Экземпляр — это вариант класса, который описывает конкретный объект на предприятии. Например, если в классической платформе есть два устройства чтения компакт-дисков, для их описания можно использовать два экземпляра класса [**Win32 \_ кдромдриве**](/windows/desktop/CIMWin32Prov/win32-cdromdrive) . Поскольку экземпляр представляет реальное свойство предприятия, скорее всего, вы создадите больше экземпляров, чем классы. Каждый класс, на основе которого создается экземпляр, имеет одно или несколько ключевых свойств. Нельзя создать экземпляр с более чем 256 ключами.
