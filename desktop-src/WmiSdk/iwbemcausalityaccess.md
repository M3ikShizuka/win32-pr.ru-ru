# <a name="iwbemcausalityaccess-interface"></a>Интерфейс Ивбемкаусалитякцесс

Интерфейс **ивбемкаусалитякцесс** отслеживает дочерние запросы, созданные из родительского запроса. Объект **ивбемкаусалитякцесс** можно получить с помощью интерфейса запроса (QI) для [**ивбемконтекст**](/windows/desktop/api/WbemCli/nn-wbemcli-iwbemcontext). Каждый запрос определяется глобальным уникальным идентификатором (GUID) и может иметь родительский запрос или может быть запросом Top. GUID — это уникальное 128-разрядное число. Например, 5b2fc63a-8AF4-44cb-960c-aefdced908d6.

## <a name="members"></a>Элементы

Интерфейс **ивбемкаусалитякцесс** наследует от интерфейса [**IUnknown**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) . **Ивбемкаусалитякцесс** также имеет следующие типы членов:

-   [Методы](#methods)

### <a name="methods"></a>Методы

Интерфейс **ивбемкаусалитякцесс** содержит следующие методы.



| Метод                                                    | Описание                                                                                                                                                             |
|:----------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**жетрекуестид**](iwbemcausalityaccess-getrequestid.md) | Возвращает уникальный идентификатор GUID для запроса.<br/>                                                                                                              |
| [**исчилдоф**](iwbemcausalityaccess-ischildof.md)       | Определяет, является ли запрос дочерним по отношению к указанному родительскому запросу. Родительский запрос может иметь несколько дочерних запросов, каждый из которых определяется уникальным значением идентификатора GUID.<br/> |
