# <a name="swbemnamedvaluename-property"></a>SWbemNamedValue.Name, свойство

Свойство **Name** объекта [**свбемнамедвалуе**](swbemnamedvalue.md) является уникальной строкой без учета регистра, используемой при получении элемента **свбемнамедвалуе** . Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemNamedValue.Name As String
```
