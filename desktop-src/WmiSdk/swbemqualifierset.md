# <a name="swbemqualifierset-object"></a>Объект Свбемкуалифиерсет

Объект **свбемкуалифиерсет** — это коллекция объектов [**свбемкуалифиер**](swbemqualifier.md) . Элементы добавляются в коллекцию с помощью метода [**Add**](swbemqualifierset-add.md) , полученного из коллекции с помощью метода [**Item**](swbemqualifierset-item.md) , и удаляются из коллекции с помощью метода [**Remove**](swbemqualifierset-remove.md) . Не удается создать этот объект с [помощью вызова функции](creating-an-object-using-vbscript.md) VBScript.

Дополнительные сведения см. [в разделе доступ к коллекции](accessing-a-collection.md).

Объекты [**свбемкуалифиер**](swbemqualifier.md) , составляющие коллекцию **свбемкуалифиерсет** , описывают квалификаторы, присоединенные к классу WMI, экземпляру, методу или параметру метода.

## <a name="members"></a>Элементы

Объект **свбемкуалифиерсет** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Объект **свбемкуалифиерсет** содержит эти методы.



| Метод                                     | Описание                                                                                                 |
|:-------------------------------------------|:------------------------------------------------------------------------------------------------------------|
| [**Включить**](swbemqualifierset-add.md)       | Добавляет объект [**свбемкуалифиер**](swbemqualifier.md) в коллекцию **свбемкуалифиерсет** .<br/> |
| [**Элемент**](swbemqualifierset-item.md)     | Возвращает именованный объект [**свбемкуалифиер**](swbemqualifier.md) из коллекции.<br/>             |
| [**Отменит**](swbemqualifierset-remove.md) | Удаляет именованный квалификатор из коллекции.<br/>                                                   |



 

### <a name="properties"></a>Свойства

Объект **свбемкуалифиерсет** имеет следующие свойства.



| Свойство                                            | Тип доступа          | Описание                                                                     |
|:----------------------------------------------------|:---------------------|:--------------------------------------------------------------------------------|
| [**Расчета**](swbemqualifierset-count.md)<br/> | Только для чтения<br/> | Содержит количество элементов в коллекции **свбемкуалифиерсет** .<br/> |
