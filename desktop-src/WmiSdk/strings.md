# <a name="mof-strings"></a>Строки MOF

Строка — это тип данных, который содержит строку символов, обычно предназначенную для читаемого текста. MOF описывает два типа строк, которые используются для хранения одного или нескольких символов. MOF также имеет ряд правил, описывающих использование кавычек в строке.

В следующей таблице перечислены типы строковых данных для MOF.



| Тип данных  | Тип автоматизации | Описание                                                                            |
|------------|-----------------|----------------------------------------------------------------------------------------|
| **char16** | **VT \_ I2**      | Один 16-разрядный символ Юникода в формате универсального набора символов 2 (UCS-2)<br/> |
| **string** | **VT \_ BSTR**    | Строка символов Юникод<br/>                                                    |



 

При записи строк для MOF используйте следующие рекомендации:

-   Заключите односимвольные константы в одинарные кавычки.

    Если вы не используете одинарные кавычки с одиночными символьными константами, необходимо использовать целочисленное представление значения символа Юникода. При необходимости можно указать символ буквально с \\ escape-последовательностью x из стандарта американский национальный институт стандартов (ANSI) (ANSI) C, как показано ниже.

    ``` syntax
    char16  TestChar1 = '\x4133';
    char16  Testchar2 = 'A';
    ```

    Так как MOF основана на Юникоде, можно также указать 16-разрядные значения.

    Имейте в виду, что константы одного символа в формате ANSI C заключены в двойные кавычки.

-   Заключите в строки символов двойные кавычки.

    ``` syntax
    DTime    = "19940107140332.000000-300";
    ```

-   Объедините последующие строки кавычек с одним или несколькими пробелами.

    ``` syntax
    DString = "This" "becomes a long string";
    ```

-   Чтобы внедрить кавычки в строку, используйте escape-последовательность, начинающуюся с обратной косой черты.

    ``` syntax
    DMyString = "This is an \"embedded quote\" example."
    ```

В следующем примере описывается инициализация строковых свойств и строкового параметра.

``` syntax
class  StringDataClass
{
    [key]  String    Dstring;
    DateTime         DTime;
    char16           CharVal1;
    char16           CharVal2;
    sint32 DiskMethod ([in, Id(0)] string Description = "Disk 1");
};

instance of StringDataClass
{
    Dstring = "this can go on for " " some time"
       " before it is complete";
    DTime    = "19940107140332.000000-300";
    CharVal1 = '\x16';
    CharVal2 = '\x32';
};
```
