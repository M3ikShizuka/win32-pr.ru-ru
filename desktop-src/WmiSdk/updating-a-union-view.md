# <a name="updating-a-union-view"></a>Обновление представления объединения

Можно обновить значения неключевых свойств экземпляров класса представления Union. Изменения, вносимые в экземпляр класса представления, будут переданы обратно экземплярам исходного класса, образуя класс представления Union.

На изменения экземпляров класса представления влияют следующие ограничения.

-   Нельзя создавать новые экземпляры классов представлений.
-   Обновления поддерживаются только в классах Union. представления соединения и ассоциации создают экземпляры представлений только для чтения.
-   Успешное обновление зависит от того, является ли экземпляр источника обновляемым. Если свойство экземпляра представления соответствует свойству исходного экземпляра, доступному только для чтения, попытки изменить свойство View завершатся ошибкой.
