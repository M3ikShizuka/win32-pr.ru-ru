# <a name="chstringformat-methods"></a>Методы Чстринг:: Format

\[Класс [**чстринг**](chstring.md) является частью платформы поставщика WMI, которая теперь считается окончательным состоянием, и дальнейшая разработка, улучшения и обновления не будут доступны для проблем, связанных с безопасностью, затрагивающих эти библиотеки. [API MI](/previous-versions/windows/desktop/wmi_v2/windows-management-infrastructure) следует использовать для всех новых разработок.\]

Метод **Format** форматирует и сохраняет ряд символов и значений в [**чстринг**](chstring.md) строке.

### <a name="overload-list"></a>Список перегрузок



| Метод                                              | Описание                                                                                      |
|:----------------------------------------------------|:-------------------------------------------------------------------------------------------------|
| [**Формат (UINT)**](/previous-versions/windows/desktop/legacy/aa385532(v=vs.85))       | Форматирует этот Чстринг в соответствии с идентификатором ресурса, указанным в параметре **uint**.<br/> |
| [**Формат (ЛПКВСТР)**](/windows/win32/api/chstring/nf-chstring-chstring-format(lpcwstr_---)) | Форматирует этот Чстринг в соответствии с форматом, заданным параметром **лпквстр**.<br/>           |
