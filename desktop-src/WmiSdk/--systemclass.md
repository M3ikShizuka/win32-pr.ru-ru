# <a name="__systemclass-class"></a>\_\_Класс Системкласс

Абстрактный системный класс **\_ \_ системкласс** является базовым классом, от которого наследуют большинство системных классов. Экземпляры этого класса не могут быть созданы.

Приведенный ниже синтаксис является упрощенной версией кода MOF и включает все унаследованные свойства. Свойства перечислены в алфавитном порядке, а не в MOF.

## <a name="syntax"></a>Синтаксис

``` syntax
[abstract]
class __SystemClass
{
};
```

## <a name="members"></a>Члены

Класс **\_ \_ системкласс** не определяет никаких членов.
