# <a name="dynamic-qualifier"></a>Динамический квалификатор

**Динамический** квалификатор указывает класс, экземпляры которого создаются динамически. Значение этого квалификатора должно быть равно **true**.

**Динамический** квалификатор должен быть указан во всех классах, содержащих данные, и для которых экземпляры создаются динамически. Квалификатор [**поставщика**](/windows/desktop/api/Provider/nl-provider-provider) , как правило, также указывается для указания поставщика, ответственного за предоставление данных.

Классы, содержащие только те методы, которые нуждаются в реализации, не требуют **динамического** квалификатора. Для указания имени поставщика, предоставляющего реализацию, требуется только квалификатор [**поставщика**](/windows/desktop/api/Provider/nl-provider-provider) .

Все классы, производные от динамического класса, должны быть динамическими. Нельзя наследовать статический класс от динамического класса.

Если для свойства экземпляра задано значение **dynamic** , также должен быть указан квалификатор **пропертиконтекст** .
