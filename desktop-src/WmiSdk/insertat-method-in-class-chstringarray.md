# <a name="chstringarrayinsertat-methods"></a>Методы Чстрингаррай:: Инсертат

\[Класс [**чстрингаррай**](/windows/win32/api/chstrarr/nl-chstrarr-chstringarray) является частью платформы поставщика WMI, которая теперь считается окончательным состоянием, и дальнейшая разработка, улучшения и обновления не будут доступны для проблем, связанных с безопасностью, затрагивающих эти библиотеки. [API MI](/previous-versions/windows/desktop/wmi_v2/windows-management-infrastructure) следует использовать для всех новых разработок.\]

Метод **инсертат** вставляет элемент (или несколько копий элемента) или все элементы другого массива по указанному индексу.

### <a name="overload-list"></a>Список перегрузок



| Метод                                                                               | Описание                                                                                                             |
|:-------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------|
| [**Инсертат (int, ЛПКВСТР, int)**](/previous-versions/windows/desktop/legacy/aa385383(v=vs.85))       | Вставляет один или несколько элементов по указанному индексу в массиве.<br/>                                               |
| [**Инсертат (int, Чстрингаррай \* )**](/windows/win32/api/chstrarr/nf-chstrarr-chstringarray-insertat(int_chstringarray)) | Вставляет все элементы другого [**чстрингаррай**](/windows/win32/api/chstrarr/nl-chstrarr-chstringarray) с указанным индексом в массиве.<br/> |
