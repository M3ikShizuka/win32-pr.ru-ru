# <a name="swbemlasterrorproperties_-property"></a>Свбемластеррор. Properties, \_ свойство

Свойство **Properties \_** объекта [**свбемластеррор**](swbemlasterror.md) возвращает объект [**SWbemPropertySet**](swbempropertyset.md) , который является коллекцией свойств WMI для текущего класса или экземпляра. Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemLastError.Properties_ As Object
```
