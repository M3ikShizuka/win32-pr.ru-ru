# <a name="namespace-access-rights-constants"></a>Константы прав доступа к пространству имен

WMI содержит константы безопасности, используемые для проверки доступа к пространству имен с помощью [**\_ \_ системсекурити:: жеткаллеракцессригхтс**](--systemsecurity-getcalleraccessrights.md). Сведения о безопасности списков управления доступом (ACL, DACL и SACL) см. в разделе [списки управления доступом (ACL)](/windows/desktop/SecAuthZ/access-control-lists) и [**стандартные права доступа**](/windows/desktop/SecAuthZ/standard-access-rights). Эти константы определяются в перечислении **\_ \_ флагов безопасности WBEM** .

<dl> <dt>

<span id="WBEM_ENABLE"></span><span id="wbem_enable"></span>**\_Включение WBEM**
</dt> <dd> <dl> <dt>

1 (0x1)
</dt> <dt>



Включает учетную запись и предоставляет пользователю разрешения на чтение. Это право доступа по умолчанию для всех пользователей и соответствует разрешению «включение учетной записи» на вкладке «Безопасность» [*элемента управления WMI*](gloss-w.md). Дополнительные сведения см. [в разделе Настройка безопасности пространства имен с помощью элемента управления WMI](setting-namespace-security-with-the-wmi-control.md).


</dt> </dl> </dd> <dt>

<span id="WBEM_METHOD_EXECUTE"></span><span id="wbem_method_execute"></span>**\_выполнение метода \_ WBEM**
</dt> <dd> <dl> <dt>

2 (0x2)
</dt> <dt>



Разрешает выполнение методов. Поставщики могут выполнять дополнительные проверки доступа. Это право доступа по умолчанию для всех пользователей и соответствует разрешению «выполнение методов» на вкладке «Безопасность» элемента управления WMI.


</dt> </dl> </dd> <dt>

<span id="WBEM_FULL_WRITE_REP"></span><span id="wbem_full_write_rep"></span>**Менеджер WBEM с \_ полной \_ записью \_**
</dt> <dd> <dl> <dt>

4 (0x4)
</dt> <dt>



Позволяет учетной записи пользователя выполнять запись в классы в репозитории WMI и экземплярах. Пользователь не может выполнять запись в системные классы. Только члены группы "Администраторы" имеют это разрешение. **WBEM \_ \_ \_ Представитель полной записи** соответствует разрешению на запись для всей записи на вкладке Безопасность элемента управления WMI.


</dt> </dl> </dd> <dt>

<span id="WBEM_PARTIAL_WRITE_REP"></span><span id="wbem_partial_write_rep"></span>**\_представитель с частичным \_ НАПИСАНием WBEM \_**
</dt> <dd> <dl> <dt>

8 (0x8)
</dt> <dt>



Позволяет записывать данные только в экземпляры, а не классы. Пользователь не может записывать классы в [*репозиторий WMI*](gloss-w.md). Это право имеют только члены группы "Администраторы". **WBEM \_ \_ \_ Представитель частичной записи** соответствует разрешению частичной записи на вкладке Безопасность элемента управления WMI.


</dt> </dl> </dd> <dt>

<span id="WBEM_WRITE_PROVIDER"></span><span id="wbem_write_provider"></span>**\_поставщик записи \_ WBEM**
</dt> <dd> <dl> <dt>

16 (0x10)
</dt> <dt>



Позволяет создавать классы и экземпляры для поставщиков. Обратите внимание, что поставщики могут выполнять дополнительные проверки доступа при олицетворении пользователя. Это право доступа по умолчанию для всех пользователей и соответствует разрешению на запись для поставщика на вкладке Безопасность элемента управления WMI.


</dt> </dl> </dd> <dt>

<span id="WBEM_REMOTE_ACCESS"></span><span id="wbem_remote_access"></span>**\_Удаленный \_ доступ WBEM**
</dt> <dd> <dl> <dt>

32 (0x20)
</dt> <dt>



Позволяет учетной записи пользователя удаленно выполнять любые операции, разрешенные описанными выше разрешениями. Это право имеют только члены группы "Администраторы". **WBEM \_ УДАЛЕННЫЙ \_ доступ** соответствует разрешению Remote Enable на вкладке Безопасность элемента управления WMI.


</dt> </dl> </dd> </dl>
