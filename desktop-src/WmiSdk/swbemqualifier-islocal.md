# <a name="swbemqualifierislocal-property"></a>Свойство Свбемкуалифиер. local

Свойство **Local** объекта [**свбемкуалифиер**](swbemqualifier.md) является логическим значением, указывающим, является ли этот квалификатор локальным. Это свойство доступно только для чтения.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

Это свойство доступно только для чтения.

## <a name="syntax"></a>Синтаксис


```VB
SWbemQualifier.IsLocal As Boolean
```
