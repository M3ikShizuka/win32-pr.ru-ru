# <a name="retrieving-wmi-class-or-instance-data"></a>Получение данных класса или экземпляра WMI

| Раздел                                                                   | Описание                                                                       |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| [Получение класса WMI](retrieving-a-class.md)                        | Получение сведений об общих параметрах объекта.                    |
| [Получение экземпляра WMI](retrieving-an-instance.md)                 | Получение сведений, описывающих уникальный отдельный объект на предприятии. |
| [Получение части экземпляра WMI](retrieving-part-of-an-instance.md) | Извлечение одной или нескольких частей экземпляра, но не всего экземпляра.         |
| [Доступ к квалификатору WMI](accessing-a-qualifier.md)                  | Доступ к дополнительным сведениям, содержащимся в квалификаторе.                              |



 

Дополнительные сведения см. в разделе [Управление сведениями о классе и экземпляре](manipulating-class-and-instance-information.md).
