# <a name="__eventdroppedevent-class"></a>\_\_Класс Евентдроппедевент

Класс System **\_ \_ евентдроппедевент** представляет вхождение события, которое удаляется. Удаленное событие — это событие, которое не доставляется потребителю события.

Приведенный ниже синтаксис является упрощенной версией кода MOF и включает все унаследованные свойства. Свойства перечислены в алфавитном порядке, а не в MOF.

## <a name="syntax"></a>Синтаксис

``` syntax
class __EventDroppedEvent : __SystemEvent
{
  __Event             Event;
  __EventConsumer REF IntendedConsumer;
  uint8               SECURITY_DESCRIPTOR[];
  uint64              TIME_CREATED;
};
```
