# <a name="cthreadbase-methods"></a>Методы Ксреадбасе

\[Класс [**ксреадбасе**](/windows/desktop/api/ThrdBase/nl-thrdbase-cthreadbase) является частью платформы поставщика WMI, которая теперь считается окончательным состоянием, и дальнейшая разработка, улучшения и обновления не будут доступны для проблем, связанных с безопасностью, затрагивающих эти библиотеки. [API MI](/previous-versions/windows/desktop/wmi_v2/windows-management-infrastructure) следует использовать для всех новых разработок.\]
