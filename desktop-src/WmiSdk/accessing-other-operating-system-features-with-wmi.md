# <a name="accessing-other-operating-system-features-with-wmi"></a>Доступ к другим функциям операционной системы с помощью инструментария WMI

Инструментарий WMI имеет расширения или поставщики, которые позволяют получить доступ к другим функциям операционной системы, таким как Active Directory, ODBC и SNMP. Можно также получить доступ к данным из драйверов устройств.

Расширения WMI доступны не во всех операционных системах. Дополнительные сведения см. в разделе [доступность операционной системы компонентов WMI](operating-system-availability-of-wmi-components.md).

Следующие разделы содержат сведения о доступе к другим функциям операционной системы.

-   [Доступ к данным из драйверов устройств](accessing-data-from-device-drivers.md)
-   [Доступ к Active Directory](accessing-active-directory.md)
-   [Доступ к SNMP-устройствам](accessing-snmp-devices.md)
