# <a name="errors-1101-through-1110"></a>Ошибки с 1101 по 1110

Описывает ошибки поставщика WMI SNMP с 1101 по 1110.

[Неустранимая ошибка 1101](#fatal-error-1101)

## <a name="fatal-error-1101"></a>Неустранимая ошибка 1101

<dl> <dt>

<span id="_1101__Fatal_____fileName__line____The_length_of_the_range_is_too_long_for_the_compiler__Does_not_fit_into_4_bytes__"></span><span id="_1101__fatal_____filename__line____the_length_of_the_range_is_too_long_for_the_compiler__does_not_fit_into_4_bytes__"></span><span id="_1101__FATAL_____FILENAME__LINE____THE_LENGTH_OF_THE_RANGE_IS_TOO_LONG_FOR_THE_COMPILER__DOES_NOT_FIT_INTO_4_BYTES__"></span>**<1101, Неустранимая>: " <fileName><line \#>: длина диапазона слишком велика для компилятора (не умещается в 4 байта)"**
</dt> <dd>

Семантическая ошибка модуля в спецификации Range или size, характерная для не SNMPv2C. Размер набора диапазонов должен быть меньше 2 ³ ². Размер набора диапазонов определяется как разница между самым большим значением и наименьшим значением в наборе.

</dd> </dl>
