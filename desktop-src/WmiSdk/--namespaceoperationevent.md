# <a name="__namespaceoperationevent-class"></a>\_\_Класс Намеспацеоператионевент

Класс System **\_ \_ намеспацеоператионевент** является базовым классом для всех внутренних событий, связанных с пространством имен.

Приведенный ниже синтаксис является упрощенной версией кода MOF и включает все унаследованные свойства. Свойства перечислены в алфавитном порядке, а не в MOF.

## <a name="syntax"></a>Синтаксис

``` syntax
class __NamespaceOperationEvent : __Event
{
  uint8       SECURITY_DESCRIPTOR[];
  __Namespace TargetNamespace;
  uint64      TIME_CREATED;
};
```

## <a name="members"></a>Члены

Класс **\_ \_ намеспацеоператионевент** имеет следующие типы членов:

-   [Свойства](#properties)

### <a name="properties"></a>Элемент Property

Класс **\_ \_ намеспацеоператионевент** имеет следующие свойства.

<dl> <dt>

**\_дескриптор безопасности**
</dt> <dd> <dl> <dt>

Тип данных: массив **Uint8**
</dt> <dt>

Тип доступа: только для чтения
</dt> </dl>

Дескриптор, используемый поставщиком событий для определения того, какие пользователи могут получить событие. Это свойство наследуется от [**\_ \_ события**](--event.md).

</dd> <dt>

**Имен**
</dt> <dd> <dl> <dt>

Тип данных: **\_ \_ пространство имен**
</dt> <dt>

Тип доступа: только для чтения
</dt> </dl>

Пространство имен, затронутое событием. Для событий создания это только что созданное пространство имен. Для событий изменения это измененное пространство имен. Для событий удаления это удаленное пространство имен.

</dd> <dt>

**ВРЕМЯ \_ создания**
</dt> <dd> <dl> <dt>

Тип данных: **UInt64**
</dt> <dt>

Тип доступа: только для чтения
</dt> </dl>

Уникальное значение, указывающее время создания события. Это 64-разрядное значение, представляющее число 100-наносекундных интервалов после 1 января 1601 г. Эти сведения находятся в формате всеобщего скоординированного времени (UTC). Это свойство наследуется от [**\_ \_ события**](--event.md).

Дополнительные сведения об использовании значений **UInt64** в скриптах см. [в разделе Создание сценариев в WMI](/windows/desktop/WmiSdk/creating-a-wmi-script).

</dd> </dl>

## <a name="remarks"></a>Remarks

Класс **\_ \_ намеспацеоператионевент** является производным от [**\_ \_ события**](--event.md).

Экземпляры этого класса не создаются. Инструментарий WMI создает экземпляры одного из следующих классов, производных от этого класса:

[**\_\_намеспацекреатионевент**](--namespacecreationevent.md)

[**\_\_намеспацемодификатионевент**](--namespacemodificationevent.md)

[**\_\_намеспацеделетионевент**](--namespacedeletionevent.md)
