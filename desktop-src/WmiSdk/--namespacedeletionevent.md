# <a name="__namespacedeletionevent-class"></a>\_\_Класс Намеспацеделетионевент

Класс System **\_ \_ намеспацеделетионевент** сообщает о событии удаления пространства имен, которое представляет собой тип [внутреннего события](determining-the-type-of-event-to-receive.md) , создаваемого при удалении подпространства имен из текущего пространства имен.

Приведенный ниже синтаксис является упрощенной версией кода MOF и включает все унаследованные свойства. Свойства перечислены в алфавитном порядке, а не в MOF.

## <a name="syntax"></a>Синтаксис

``` syntax
class __NamespaceDeletionEvent : __NamespaceOperationEvent
{
  uint8       SECURITY_DESCRIPTOR[];
  __Namespace TargetNamespace;
  uint64      TIME_CREATED;
};
```

Класс **\_ \_ намеспацеделетионевент** имеет следующие свойства.

**Имен**

Тип данных: **\_ \_ пространство имен**

Копия удаляемого экземпляра [**\_ \_ пространства имен**](--namespace.md) . Свойство **Name** экземпляра **\_ \_ пространства имен** определяет пространство имен, которое удаляется. Это свойство наследуется от [**\_ \_ намеспацеоператионевент**](--namespaceoperationevent.md).

</dd> <dt>

**ВРЕМЯ \_ создания**
