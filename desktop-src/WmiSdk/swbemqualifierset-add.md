# <a name="swbemqualifiersetadd-method"></a>Свбемкуалифиерсет. Add, метод

Метод **Add** объекта [**Свбемкуалифиерсет**](swbemqualifierset.md) добавляет объект [**свбемкуалифиер**](swbemqualifier.md) в коллекцию **свбемкуалифиерсет** . Если квалификатор с таким именем уже существует в коллекции, он будет заменен.

Описание этого синтаксиса см. в разделе [соглашения о документе для API скриптов](document-conventions-for-the-scripting-api.md).

## <a name="syntax"></a>Синтаксис


```VB
objQualifier = .Add( _
  ByVal strName, _
  ByVal varVal, _
  [ ByVal bPropagatesToSubclasses ], _
  [ ByVal bPropagatesToInstances ], _
  [ ByVal bOverridable ], _
  [ ByVal iFlags ] _
)
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*strName* \[ окне\]
</dt> <dd>

Обязательный. Имя нового квалификатора.

</dd> <dt>

*варвал* \[ окне\]
</dt> <dd>

Обязательный. Значение Variant нового квалификатора.

</dd> <dt>

*бпропагатестосубклассес* \[ в необязательное\]
</dt> <dd>

Логическое значение, указывающее, распространяется ли новый квалификатор на подклассы. Значение по умолчанию — **true**.

</dd> <dt>

*бпропагатестоинстанцес* \[ в необязательное\]
</dt> <dd>

Логическое значение, указывающее, распространяется ли новый квалификатор на экземпляры. Значение по умолчанию — **true**.

</dd> <dt>

*боверридабле* \[ в необязательное\]
</dt> <dd>

Логическое значение, указывающее, можно ли переопределить этот квалификатор при распространении. Значение по умолчанию — **true**.

</dd> <dt>

*ифлагс* \[ в необязательное\]
</dt> <dd>

Зарезервировано. Значение по умолчанию — 0.

</dd> </dl>

## <a name="return-value"></a>Возвращаемое значение

В случае успеха этот метод возвращает объект [**свбемкуалифиер**](swbemqualifier.md) , представляющий новый квалификатор. В противном случае возвращается **пустой** объект.

## <a name="error-codes"></a>Коды ошибок

После завершения метода **Add** объект **Err** может содержать один из кодов ошибок из следующего списка.

<dl> <dt>

**вбемерринвалидпараметер** -2147749896 (0x80041008)
</dt> <dd>

Недопустимый параметр *ифлагс* .

</dd> <dt>

**вбемеррфаилед** -2147749889 (0x80041001)
</dt> <dd>

Незаданная ошибка.

</dd> <dt>

**вбемеррканнотбекэй** -2147749919 (0x8004101F)
</dt> <dd>

Недопустимая попытка указать квалификатор **ключа** для свойства, которое не может быть ключом. Ключи указываются в определении класса объекта и не могут быть изменены для каждого отдельного экземпляра.

</dd> <dt>

**вбемерринвалидкуалифиертипе** -2147749929 (0x80041029)
</dt> <dd>

Параметр *варвал* не является допустимым типом квалификатора.

**вбемерроверриденоталловед** -2147749914 (0x8004101A)

Невозможно выполнить операцию **свбемкуалифиерсет. Add** над этим квалификатором, так как объект-владелец не допускает переопределения.

</dd> </dl>
