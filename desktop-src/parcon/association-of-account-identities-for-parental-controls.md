# <a name="association-of-account-identities-for-parental-controls"></a>Сопоставление удостоверений учетных записей для родительского контроля

Как обсуждалось в статье Введение, эффективное использование функций настройки, ведения журнала и ограничений, обеспечиваемых инфраструктурой родительских элементов управления Windows Vista, зависит от того, какие учетные записи Windows настроены как обычные пользователи. Рекомендуется, чтобы поставщики, предоставляющие доступ к сети (например, пакеты ISP), создали и поддерживали сопоставление имен пользователей в сети, разрешенных родительским или опекуном для каждой управляемой учетной записи Windows. Таким образом, обход облачных родительских элементов управления только с помощью альтернативных сетевых удостоверений сокращается.
