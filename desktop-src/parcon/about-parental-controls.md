# <a name="about-parental-controls"></a>Об родительских элементах управления

Функция родительского контроля включает в себя следующее:

-   Функция центрального пользовательского интерфейса, интегрированная с панелью управления.
-   Реализации выбранных ограничений.
-   Инфраструктура в операционной системе, которая предоставляет параметры для общих политик и реализованных ограничений, которые предоставляют целевые функции расширяемости.
