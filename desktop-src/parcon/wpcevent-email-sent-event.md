# <a name="wpcevent_email_sent-event"></a>\_Событие отправленного сообщения электронной почты впцевент \_

Событие "на пользователя", формируемое почтовым клиентом при попытке отправить сообщение в родительский контроль.


```C++
const EVENT_DESCRIPTOR WPCEVENT_EMAIL_SENT = {0x5, 0x0, 0x10, 0x4, 0x16, 0x5, 0x8000000000000030};
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*Отправитель* 
</dt> <dd>

Имя учетной записи электронной почты отправляющей сущности.

</dd> <dt>

*AppName* 
</dt> <dd>

Имя приложения электронной почты, создающего событие.

</dd> <dt>

*AppVersion* 
</dt> <dd>

Версия приложения электронной почты, создающего событие.

</dd> <dt>

*Тема* 
</dt> <dd>

Строка темы полученного сообщения.

</dd> <dt>

*Причина* 
</dt> <dd>

Значение перечисления [**впкфлаг \_**](/windows/win32/api/wpcevent/ne-wpcevent-wpcflag_isblocked) , которое указывает сведения о том, какие события заблокированы и какие элементы управления будут использоваться.

</dd> <dt>

*реЦипкаунт* 
</dt> <dd>

Число аддрессис электронной почты, которые получают сообщение и имеющие удостоверения, определенные в поле получателя.

</dd> <dt>

*Recipient* 
</dt> <dd>

Строка с разделителями, содержащая имена учетных записей электронной почты всех получателей сообщения.

</dd> <dt>

*аттачкаунт* 
</dt> <dd>

Число вложений в сообщение.

</dd> <dt>

*аттачментнаме* 
</dt> <dd>

Строка с разделителями, содержащая имена всех вложений в сообщение.

</dd> <dt>

*рецеиведтиме* 
</dt> <dd>

Время, когда было предпринято сообщение.

</dd> <dt>

*емаилаккаунт* 
</dt> <dd>

Имя учетной записи электронной почты для этого пользователя.

</dd> </dl>
