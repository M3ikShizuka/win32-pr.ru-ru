# <a name="parental-controls-sku-availability"></a>Доступность SKU родительского контроля

Пользовательские интерфейсы, функции и API-интерфейсы родительского контроля, описанные в этом разделе, сейчас запланированы на доступ только к номерам SKU, ориентированным на потребителей, например:

-   Windows Vista Starter
-   Windows Vista Home Basic
-   Windows Vista Home Premium
-   Windows Vista Ultimate

Родительский контроль устанавливается только на эти номера SKU. Решения, требующие развертывания на SKU для бизнеса, должны иметь одно из следующих:

-   Обнаруживать и не предоставлять функции родительского контроля в SKU для бизнеса.
-   Обнаружение и предоставление альтернативных средств настройки, ведения журнала и ограничений.

Рекомендуется, чтобы решения определяли доступность инфраструктуры родительского контроля, проверяя успешность выполнения CoCreateInstance COM в API соответствия.

Приложения, поддерживающие родительский контроль, в зависимости от инфраструктуры Windows Vista, также могут подавить отключение любого собственного пользовательского интерфейса, предоставляющего параметры или другие функциональные возможности, когда пользовательский интерфейс родительского контроля подавляется на поддерживаемом SKU. Функция предназначена для определения видимости, которая охватывает такие случаи, как подавление присоединения к домену.
