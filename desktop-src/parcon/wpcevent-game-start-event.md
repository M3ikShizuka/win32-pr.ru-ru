# <a name="wpcevent_game_start-event"></a>\_ \_ Событие запуска игры впцевент

Событие, создаваемое системой при попытке запустить игру. Различные значения полей предоставляются системой обозревателя игр и соответствующими метаданными файла определения игры (ГДФ), предоставляемыми поддерживаемыми играми.


```C++
const EVENT_DESCRIPTOR WPCEVENT_GAME_START = {0x2, 0x0, 0x10, 0x4, 0x16, 0x2, 0x8000000000000030};
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*AppID* 
</dt> <dd>

Идентификатор GUID игры, которая была предпринята для запуска.

</dd> <dt>

*InstanceID* 
</dt> <dd>

Идентификатор GUID, используемый для различения нескольких установок.

</dd> <dt>

*AppVersion* 
</dt> <dd>

Строка версии для игры.

</dd> <dt>

*Путь* 
</dt> <dd>

Путь к основному каталогу установки игры.

</dd> <dt>

*Оценка* 
</dt> <dd>

Строка, определяющая уровень оценки игры в системе оценок.

</dd> <dt>

*ратингсистем* 
</dt> <dd>

Идентификатор GUID, определяющий текущую систему оценок, к которой применяется уровень рейтинга.

</dd> <dt>

*Причина* 
</dt> <dd>

Значение перечисления [**впкфлаг \_**](/windows/win32/api/wpcevent/ne-wpcevent-wpcflag_isblocked) , которое указывает сведения о том, какие события заблокированы и какие элементы управления будут использоваться.

</dd> <dt>

*десккаунт* 
</dt> <dd>

Число дескрипторов, которые имеются в поле "идентификатор".

</dd> <dt>

*Descriptor* 
</dt> <dd>

Строка с разделителями, содержащая дескрипторы, заблокированные для игры.

</dd> <dt>

*PID* 
</dt> <dd>

Идентификатор процесса игры, который используется для сопоставления с завершением процесса оболочки совместимости.

</dd> </dl>
