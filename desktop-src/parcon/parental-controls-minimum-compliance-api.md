# <a name="parental-controls-minimum-compliance-api"></a>API минимального соответствия для родительского контроля

Минимальный API соответствия реализует простые методы интерфейса для приложений, чтобы получать основные сведения о состоянии по заданной учетной записи пользователя с правами родительского управления, параметры веб-ограничений и параметры ограничений игр.

-   [**ивиндовспаренталконтролс**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwindowsparentalcontrols)
-   [**ивпкгамессеттингс**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcgamessettings)
-   [**ивпксеттингс**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcsettings)
-   [**ивпквебсеттингс**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcwebsettings)
