# <a name="wpcevent_web_filedownload-event"></a>\_Событие веб- \_ филедовнлоад впцевент

Событие для каждого пользователя, созданное при попытке скачать файлы из Интернета. Это событие создается приложениями, которые запрашиваются родительскими элементами управления.


```C++
const EVENT_DESCRIPTOR WPCEVENT_WEB_FILEDOWNLOAD = {0xa, 0x0, 0x10, 0x4, 0x18, 0xa, 0x8000000000000030};
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*URL-адрес* 
</dt> <dd>

Источник URL-адреса, который пытается скачаться.

</dd> <dt>

*AppName* 
</dt> <dd>

Имя приложения, создающего событие.

</dd> <dt>

*Version* 
</dt> <dd>

Версия приложения, создающего событие.

</dd> <dt>

*Заблокирован* 
</dt> <dd>

Значение перечисления [**впкфлаг \_**](/windows/win32/api/wpcevent/ne-wpcevent-wpcflag_isblocked) , которое указывает сведения о том, какие события заблокированы и какие элементы управления будут использоваться.

</dd> <dt>

*Путь* 
</dt> <dd>

Целевой путь к файлу.

</dd> </dl>
