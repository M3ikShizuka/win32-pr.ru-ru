# <a name="wpcevent_im_join-event"></a>\_ \_ Событие объединения IM впцевент

Событие "на пользователя", созданное приложением для обмена мгновенными сообщениями, когда сущность пытается присоединиться к выполняемому в данный момент диалогу.


```C++
const EVENT_DESCRIPTOR WPCEVENT_IM_JOIN = {0x8, 0x0, 0x10, 0x4, 0x16, 0x8, 0x8000000000000030};
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*AppName* 
</dt> <dd>

Имя приложения, создающего событие.

</dd> <dt>

*AppVersion* 
</dt> <dd>

Строка версии для приложения, создающего событие.

</dd> <dt>

*AccountName* 
</dt> <dd>

Строка удостоверения учетной записи для обмена мгновенными сообщениями для этого пользователя.

</dd> <dt>

*конвид* 
</dt> <dd>

Идентификатор для этого диалога.

</dd> <dt>

*жоинингип* 
</dt> <dd>

Строка, содержащая IP-адрес компьютера, который присоединяется к данному диалогу.

</dd> <dt>

*жоинингусер* 
</dt> <dd>

Строка идентификатора учетной записи для обмена мгновенными сообщениями для пользователя, который присоединяется.

</dd> <dt>

*Причина* 
</dt> <dd>

Значение перечисления [**впкфлаг \_**](/windows/win32/api/wpcevent/ne-wpcevent-wpcflag_isblocked) , которое указывает сведения о том, какие события заблокированы и какие элементы управления будут использоваться.

</dd> <dt>

*MemberCount* 
</dt> <dd>

Число участников, которые находятся в диалоге и имеют удостоверения, определенные в поле члена.

</dd> <dt>

*Член* 
</dt> <dd>

Строка с разделителями, содержащая строки идентификаторов учетных записей мгновенных сообщений для всех текущих членов этого диалога.

</dd> <dt>

*Отправитель* 
</dt> <dd>

Строка удостоверения учетной записи для обмена мгновенными сообщениями для пользователя, выполняющего запрос на присоединение к беседе.

</dd> </dl>
