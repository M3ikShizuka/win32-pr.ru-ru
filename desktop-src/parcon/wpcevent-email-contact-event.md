# <a name="wpcevent_email_contact-event"></a>\_Событие контакта электронной почты впцевент \_

Событие "на пользователя", созданное клиентом электронной почты, которое регистрируется при добавлении, изменении или удалении контакта в родительских элементах управления.


```C++
const EVENT_DESCRIPTOR WPCEVENT_EMAIL_CONTACT = {0xe, 0x0, 0x10, 0x4, 0x16, 0xe, 0x8000000000000030};
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*AppName* 
</dt> <dd>

Имя приложения электронной почты, создающего событие.

</dd> <dt>

*AppVersion* 
</dt> <dd>

Версия приложения электронной почты, создающего событие.

</dd> <dt>

*OldName* 
</dt> <dd>

Имя предыдущей учетной записи электронной почты, если оно было удалено или изменено.

</dd> <dt>

*олдид* 
</dt> <dd>

Идентификатор, связанный с именем предыдущей учетной записи электронной почты.

</dd> <dt>

*NewName* 
</dt> <dd>

Новое имя учетной записи электронной почты, если оно добавлено или изменено.

</dd> <dt>

*NewID* 
</dt> <dd>

Идентификатор, связанный с новым именем учетной записи электронной почты.

</dd> <dt>

*Причина* 
</dt> <dd>

Значение перечисления [**впкфлаг \_**](/windows/win32/api/wpcevent/ne-wpcevent-wpcflag_isblocked) , которое указывает сведения о том, какие события заблокированы и какие элементы управления будут использоваться.

</dd> <dt>

*емаилаккаунт* 
</dt> <dd>

Имя учетной записи электронной почты для этого пользователя.

</dd> </dl>
