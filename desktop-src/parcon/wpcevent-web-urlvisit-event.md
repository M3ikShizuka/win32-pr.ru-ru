# <a name="wpcevent_web_urlvisit-event"></a>\_Событие веб- \_ урлвисит впцевент

Событие для каждого пользователя, созданное фильтром веб-содержимого из-за попытки доступа к URL-адресу.


```C++
const EVENT_DESCRIPTOR WPCEVENT_WEB_URLVISIT = {0x3, 0x0, 0x10, 0x4, 0x18, 0x3, 0x8000000000000010};
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*URL-адрес* 
</dt> <dd>

URL-адрес, который пытается открыть.

</dd> <dt>

*Приложение* 
</dt> <dd>

Приложение, создающее событие.

</dd> <dt>

*Version* 
</dt> <dd>

Строка версии для приложения.

</dd> <dt>

*Причина* 
</dt> <dd>

Значение перечисления [**впкфлаг \_**](/windows/win32/api/wpcevent/ne-wpcevent-wpcflag_isblocked) , которое указывает сведения о том, какие события заблокированы и какие элементы управления будут использоваться.

</dd> <dt>

*ратингсистемид* 
</dt> <dd>

Идентификатор GUID, определяющий текущую систему оценки.

</dd> <dt>

*каткаунт* 
</dt> <dd>

Число заблокированных записей в поле "Категория".

</dd> <dt>

*Категория* 
</dt> <dd>

Строка с разделителями для идентификаторов категорий, которые заблокированы.

</dd> </dl>
