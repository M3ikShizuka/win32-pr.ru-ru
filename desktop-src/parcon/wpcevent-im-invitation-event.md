# <a name="wpcevent_im_invitation-event"></a>\_ \_ Событие приглашения впцевент IM

Событие для каждого пользователя, предоставляемое для ведения журнала при инициации беседы клиентами мгновенных сообщений.


```C++
const EVENT_DESCRIPTOR WPCEVENT_IM_INVITATION = {0x7, 0x0, 0x10, 0x4, 0x16, 0x7, 0x8000000000000030};
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*AppName* 
</dt> <dd>

Имя приложения, создающего событие.

</dd> <dt>

*AppVersion* 
</dt> <dd>

Строка версии для приложения, создающего событие.

</dd> <dt>

*AccountName* 
</dt> <dd>

Строка идентификатора учетной записи мгновенных сообщений.

</dd> <dt>

*конвид* 
</dt> <dd>

Идентификатор диалога.

</dd> <dt>

*рекуестингип* 
</dt> <dd>

Строка, содержащая IP-адрес компьютера, отправляющего приглашение.

</dd> <dt>

*Отправитель* 
</dt> <dd>

Строка удостоверения учетной записи для обмена мгновенными сообщениями для учетной записи, которая выдает приглашение.

</dd> <dt>

*Причина* 
</dt> <dd>

Значение перечисления [**впкфлаг \_**](/windows/win32/api/wpcevent/ne-wpcevent-wpcflag_isblocked) , которое указывает сведения о том, какие события заблокированы и какие элементы управления будут использоваться.

</dd> <dt>

*реЦипкаунт* 
</dt> <dd>

Число получателей, получивших приглашение, и имеющих удостоверения, определенные в поле получателя.

</dd> <dt>

*Recipient* 
</dt> <dd>

Строка с разделителями, содержащая строки идентификаторов учетных записей мгновенных сообщений для получателей приглашения.

</dd> </dl>
