# <a name="wpcevent_im_contact-event"></a>\_ \_ Контактное событие впцевент IM

Событие "на пользователя", созданное клиентом обмена мгновенными сообщениями при добавлении, изменении или удалении контактной информации в родительских элементах управления.


```C++
const EVENT_DESCRIPTOR WPCEVENT_IM_CONTACT = {0xf, 0x0, 0x10, 0x4, 0x16, 0xf, 0x8000000000000030};
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*AppName* 
</dt> <dd>

Имя приложения для обмена мгновенными сообщениями, создающего событие.

</dd> <dt>

*AppVersion* 
</dt> <dd>

Версия приложения, создающего событие.

</dd> <dt>

*AccountName* 
</dt> <dd>

Имя учетной записи для обмена мгновенными сообщениями для этого пользователя.

</dd> <dt>

*OldName* 
</dt> <dd>

Имя предыдущей учетной записи мгновенных сообщений, если оно было удалено или изменено.

</dd> <dt>

*олдид* 
</dt> <dd>

Идентификатор, связанный с предыдущим именем учетной записи мгновенного обмена сообщениями.

</dd> <dt>

*NewName* 
</dt> <dd>

Новое имя учетной записи для обмена мгновенными сообщениями, если оно добавлено или изменено.

</dd> <dt>

*NewID* 
</dt> <dd>

Идентификатор, связанный с новым именем учетной записи мгновенного обмена сообщениями.

</dd> <dt>

*Причина* 
</dt> <dd>

Значение перечисления [**впкфлаг \_**](/windows/win32/api/wpcevent/ne-wpcevent-wpcflag_isblocked) , которое указывает сведения о том, какие события заблокированы и какие элементы управления будут использоваться.

</dd> </dl>
