# <a name="wpcevent_media_playback-event"></a>\_ \_ Событие воспроизведения мультимедиа впцевент

Событие для каждого пользователя, формируемое попыткой воспроизведения содержимого с помощью приложения мультимедиа, подключенного к родительским элементам управления.


```C++
const EVENT_DESCRIPTOR WPCEVENT_MEDIA_PLAYBACK = {0x6, 0x0, 0x10, 0x4, 0x16, 0x6, 0x8000000000000030};
```



## <a name="parameters"></a>Параметры

<dl> <dt>

*AppName* 
</dt> <dd>

Имя приложения мультимедиа, создающего событие.

</dd> <dt>

*AppVersion* 
</dt> <dd>

Версия приложения мультимедиа, создающего событие.

</dd> <dt>

*Мультимедиа* 
</dt> <dd>

Значение перечисления [**\_ \_ типа мультимедиа WPC**](/windows/win32/api/wpcevent/ne-wpcevent-wpc_media_type) , указывающее сведения о типе доступного файла мультимедиа.

</dd> <dt>

*Путь* 
</dt> <dd>

Путь к источнику содержимого мультимедиа.

</dd> <dt>

*Title* 
</dt> <dd>

Метаданные заголовка для содержимого.

</dd> <dt>

*пмл* 
</dt> <dd>

Уровень родительского управления.

</dd> <dt>

*Музыкальных* 
</dt> <dd>

Метаданные альбома для содержимого.

</dd> <dt>

*Прямая* 
</dt> <dd>

Значение [**\_ \_ явного перечисления мультимедиа WPC**](/windows/win32/api/wpcevent/ne-wpcevent-wpc_media_explicit) , которое указывает сведения о явной оценке файла мультимедиа.

</dd> <dt>

*Причина* 
</dt> <dd>

Значение перечисления [**впкфлаг \_**](/windows/win32/api/wpcevent/ne-wpcevent-wpcflag_isblocked) , которое указывает сведения о том, какие события заблокированы и какие элементы управления будут использоваться.

</dd> </dl>
