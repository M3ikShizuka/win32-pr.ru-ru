# <a name="pointer-device-input-stack"></a>Входной стек устройства указателя

## <a name="purpose"></a>Назначение

Подразделы этого раздела содержат общие сведения о стеке входных данных устройства с указателем Windows. API-интерфейсы [входного стека устройства указателей](/windows/win32/api/_input_pointerdevice/) предоставляют сведения о входных устройствах, подключенных к системе.

## <a name="in-this-section"></a>В этом разделе

| Раздел | Описание |
|---|---|
| [Указатель на входной стек устройства указателя](unified-input-stack-reference.md)<br/> | В подразделах этого раздела приведены справочные сведения о стеке входных данных устройства указателя в Windows 8.<br/> |

## <a name="developer-audience"></a>Аудитория разработчиков

Входной Стаккапис устройства указателя предназначены для разработчиков, которым требуются сведения об устройстве указателя, используемом для взаимодействия с приложением.
