# <a name="pointer-device-input-stack"></a>Входной стек устройства указателя

## <a name="purpose"></a>Назначение

в подразделах этого раздела приводятся общие сведения о стеке входных данных устройства Windows указателя. API-интерфейсы [входного стека устройства указателей](/windows/win32/api/_input_pointerdevice/) предоставляют сведения о входных устройствах, подключенных к системе.

## <a name="in-this-section"></a>В этом разделе

| Раздел | Описание |
|---|---|
| [Указатель на входной стек устройства указателя](unified-input-stack-reference.md)<br/> | В подразделах этого раздела приведены справочные спецификации для входного стека устройства указателя в Windows 8.<br/> |

## <a name="developer-audience"></a>Аудитория разработчиков

Входной Стаккапис устройства указателя предназначены для разработчиков, которым требуются сведения об устройстве указателя, используемом для взаимодействия с приложением.
