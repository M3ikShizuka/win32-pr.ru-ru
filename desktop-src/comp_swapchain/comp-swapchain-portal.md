# <a name="composition-swapchain"></a>Имеющуюся цепочку буферов композиции

## <a name="overview"></a>Обзор

Этот API является первоначальным общедоступным выпуском API композиции имеющуюся цепочку буферов. он позволяет приложениям, использующим интерфейсы api композиции (например, [**Windows. UI. композиция**](/uwp/api/windows.ui.composition) и [**DirectComposition**](/windows/win32/directcomp/directcomposition-portal)), размещать содержимое, которое может быть независимо визуализировано и представлено. Во многих отношениях этот тип представленного содержимого концептуально напоминает имеющуюся цепочку буферов DXGI. Оба предложения предлагают возможность подготовки к просмотру в буфер, а затем представляют этот буфер на экране. Однако API-интерфейс композиции имеющуюся цепочку буферов предлагает возможность представить определенное время (время *пресентат* ), в то время как в DXGI не существует, а API композиции имеющуюся цепочку буферов предлагает больше возможностей, чем DXGI, что обеспечивает упорядочение буферов, доступных для представления.

основное внимание в первоначальной версии этого API заключается в том, чтобы включить платформы мультимедиа, такие как [Windows Media Foundation](/windows/win32/medfound/microsoft-media-foundation-sdk), для эффективного использования представления по времени.
