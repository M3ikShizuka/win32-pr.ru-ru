# <a name="using-more-than-one-context-handle-per-call"></a>Использование нескольких обработчиков контекста на один вызов

возможность использования массивов дескрипторов контекста в том случае, когда параметры становятся доступными в Windows Vista и более поздних операционных системах. Однако эту функцию следует использовать в приложении аккуратно. Например, в ситуации, когда маркер контекста используется в нескольких вызовах, отслеживание его использования становится все труднее.
