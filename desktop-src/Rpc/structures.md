# <a name="structs-rpc"></a>Структуры (RPC)

Обычная семантика C применяется к полям базовых типов. Поля с более сложными типами, такими как указатели, массивы и другие сконструированные типы, могут быть изменены [**\_ атрибутами**](field-attributes.md)типа или поля. Дополнительные сведения см. в разделе [**Структура**](/windows/desktop/Midl/struct).
