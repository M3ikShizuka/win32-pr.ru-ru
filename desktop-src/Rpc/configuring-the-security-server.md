# <a name="configuring-the-security-server"></a>Настройка сервера безопасности

Следующая процедура содержит сведения о настройке сервера безопасности для Microsoft RPC.

**Настройка сервера безопасности для RPC**

1.  Откройте панель управления.
2.  На панели управления щелкните значок **сети** . откроется диалоговое окно **сетевой Параметры** .
3.  В списке **установленное сетевое программное обеспечение** выберите **Конфигурация RPC**. Откроется диалоговое окно **Конфигурация RPC** .
4.  В списке **поставщик службы безопасности** выберите одного или нескольких поставщиков безопасности.
5.  Нажмите кнопку **ОК**.
