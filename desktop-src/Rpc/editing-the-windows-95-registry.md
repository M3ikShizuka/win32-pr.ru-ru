# <a name="editing-the-windows-95-registry"></a>Изменение реестра Windows 95

С помощью Regedit можно изменить реестр Windows 95, указав NSP.

**Назначение поставщика службы имен локатора Майкрософт для Windows 95**

1.  Выберите пункт

    **HKey \_ \_** \\ **Программное обеспечение** локального компьютера \\ **Microsoft** \\ **RPC**

2.  Создайте новый раздел с именем

    **намесервице**

3.  Выбрав ключ **намесервице** , создайте новые имена **StringValue** и измените их, чтобы они содержали данные, как показано в следующей таблице.

    

    | Имя                     | Данные                                                                        |
    |--------------------------|-----------------------------------------------------------------------------|
    | **дефаултсинтакс**        | 3<br/>                                                                |
    | **Протокол**             | нкакн \_ NP<br/>                                                        |
    | **Конечная точка**             | \\\\указатель канала<br/>                                                  |
    | **NetworkAddress**       | MyServer (где MyServer — имя компьютера Windows NT)<br/> |
    | **сервернетворкаддресс** | MyServer<br/>                                                         |

    

     

С помощью Regedit можно изменить реестр Windows 95, чтобы назначить компакт-диски DCE NSP.

**Назначение поставщика службы имен дисков DCE для Windows 95**

1.  Выберите пункт

    **HKey \_ \_** \\ **Программное обеспечение** локального компьютера \\ **Microsoft** \\ **RPC**

2.  Создайте новый раздел с именем

    **намесервице**

3.  Выбрав ключ **намесервице** , создайте новые имена **StringValue** и измените их, чтобы они содержали данные, как показано в следующей таблице.

    

    | Имя                     | Данные                                                             |
    |--------------------------|------------------------------------------------------------------|
    | **дефаултсинтакс**        | 3<br/>                                                     |
    | **Протокол**             | нкакн \_ IP \_ TCP<br/>                                        |
    | **Конечная точка**             | "" (пустая строка)<br/>                                  |
    | **NetworkAddress**       | MyServer (имя главного компьютера, на котором работает нсид)<br/> |
    | **сервернетворкаддресс** | MyServer (имя главного компьютера, на котором работает нсид)<br/> |

    

     

    > [!Note]  
    > Для настройки компакт-дисков DCE в качестве поставщика службы имен требуется продукт службы каталогов компании DCE Digital Equipment Corporation. Сведения о компакт-дисках DCE см. в документации, предоставляемой корпорацией Digital Equipment.
