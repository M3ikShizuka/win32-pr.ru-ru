# <a name="rpc_ep_inq_handle"></a>RPC \_ EP \_ инк, \_ обработчик

Тип **данных \_ \_ \_ Handle инк для RPC EP** объявляет маркер для контекста запроса. Приложения RPC используют его для просмотра сведений об адресе сервера, хранящегося на карте конечных точек.


```C++
typedef I_RPC_HANDLE* RPC_EP_INQ_HANDLE;
```
