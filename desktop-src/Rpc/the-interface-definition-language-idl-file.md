# <a name="the-interface-definition-language-idl-file"></a>Файл языка определения интерфейса (IDL)

IDL-файл содержит одно или несколько определений интерфейса. Каждое определение интерфейса состоит из заголовка интерфейса и тела интерфейса. Заголовок интерфейса демаркатед квадратными скобками. Тело интерфейса содержится в фигурных скобках. Это показано в следующем примере интерфейса:

``` syntax
[
  /*Interface attributes go here. */
]
interface INTERFACENAME
{
  /*The interface body goes here. */
}
```

В этом разделе приводятся общие сведения о компонентах интерфейса. Он состоит из следующих разделов:

-   [Заголовок интерфейса IDL](the-idl-interface-header.md)
-   [Тело интерфейса IDL](the-idl-interface-body.md)
