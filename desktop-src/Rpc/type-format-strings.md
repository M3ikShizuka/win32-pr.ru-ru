# <a name="type-format-strings"></a>Строки формата типов

Форматирование символов обозначают объекты, представляющие собой подсистема доставки ОНД. Существует символ формата для каждого базового типа, для различных сложных типов и описания указателей, упаковки, выравнивания и других прочих объектов. Для составных типов несколько категорий структур и массивов распознаются на основе их свойств производительности. Строка формата для каждой категории начинается с токена FC, идентифицирующего определенную строку формата. Символы формата для структур и категорий массивов могут совместно использовать in-Names в имени ведущего маркера FC, показанного в следующей таблице.



| Символ формата | Описание                                    |
|------------------|------------------------------------------------|
| C                | Указывает сведения о соотношении соответствия в типе. |
| V                | Указывает сведения о дисперсии в типе.    |
| С                | Указывает, что указатели являются частью типа.     |



 

Например, FC \_ кструкт и FC \_ CARRAY определяли согласованную структуру и согласованные дескрипторы массивов соответственно.

Ниже приведены символы форматирования со специальными значениями.



| Символ формата | Описание                                                                         |
|------------------|-------------------------------------------------------------------------------------|
| FC \_ PP           | Указывает начало раздела описания указателя в структуре или массиве. |
