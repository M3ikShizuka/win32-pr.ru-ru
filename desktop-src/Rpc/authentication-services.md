# <a name="authentication-services"></a>Службы проверки подлинности

Интерфейс поставщика поддержки безопасности (SSPI) предоставляет базовые службы проверки подлинности для RPC. Поэтому, когда приложение указывает службу проверки подлинности, выбирается поставщик общих служб. Список SSP, которые в настоящее время поддерживает СППИ, см. в разделе [константы службы проверки подлинности](authentication-service-constants.md). Дополнительные сведения см. в разделе [интерфейс поставщика поддержки безопасности (SSPI)](security-support-provider-interface-sspi-.md).

> [!Note]  
> Поставщик безопасности терминов и служба проверки подлинности являются взаимозаменяемыми.
