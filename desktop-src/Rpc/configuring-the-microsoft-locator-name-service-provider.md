# <a name="configuring-the-microsoft-locator-name-service-provider"></a>Настройка поставщика службы имен локатора Майкрософт

Вы можете изменить указанный поставщик службы имен, изменив файл конфигурации Рпкрег. dat, который содержит параметры NSP и параметры протокола RPC. Используйте текстовый редактор для изменения записей NSP.

**Изменение конфигурации поставщика службы имен локатора Майкрософт**

1.  Откройте файл Рпкрег. dat в текстовом редакторе.

    Рпкрег. dat находится в корневом каталоге, если во время установки не указан другой путь.

2.  Задайте следующие значения для записей реестра. 

    | Запись реестра                                         | Значение                                                                                                                                                 |
    |--------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
    | Программный \\ \\ протокол Microsoft RPC \\ намесервице \\       | Последовательность протокола для используемого протокола. По умолчанию используется **нкакн \_ NP**.                                                                   |
    | Программное обеспечение \\ Microsoft \\ RPC \\ намесервице \\ NetworkAddress | Имя компьютера, на котором работает локатор, используемый клиентами во время операций поиска службы имен. По умолчанию используется основной контроллер домена. |
    | Программное обеспечение \\ \\ \\ \\ Конечная точка Microsoft RPC намесервице       | Имя конечной точки, используемой службой имен. По умолчанию \\ используется \\ указатель канала.                                                                    |

    

     

3.  Сохраните файл и закройте его.
