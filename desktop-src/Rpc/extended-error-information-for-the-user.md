# <a name="extended-error-information-for-the-user"></a>Расширенные сведения об ошибке для пользователя

Если доступны расширенные сведения об ошибке, компоненты, участвующие в создании расширенных сведений об ошибке, должны упростить поиск проблемы. Первым шагом является проверка последней расширенной записи об ошибке и определение того, на каком компьютере и в каком процессе возникла проблема. Это часто вызывает \_ ошибку RPC S \_ \* . Найдите расположение обнаружения и определите, что означают параметры для этого расположения обнаружения. Обычно они указывают на сбой вызова функции или конкретной проверки. Определите причину сбоя функции или проверки и выполните действия по исправлению.

Записи перед последней записью указывают путь, по которому произошла ошибка, и обычно служит в качестве работоспособности проверки, а не телеметрическую в процессе устранения неполадок.
