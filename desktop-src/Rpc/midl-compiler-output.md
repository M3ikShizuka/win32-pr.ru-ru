# <a name="midl-compiler-output"></a>Выходные данные компилятора MIDL

Используя файлы IDL и ACF в качестве входных данных, компилятор MIDL создает до пяти исходных файлов C-Language. По умолчанию компилятор MIDL использует базовое имя файла IDL-файла в составе созданных файлов заглушек. Если в базовом имени файла имеется более шести символов, некоторые файловые системы могут не принимать полное имя заглушки. В следующей таблице показаны соглашения, используемые для имен файлов.



| Файл        | Часть имени базового файла по умолчанию | Пример      |
|-------------|-----------------------------------|--------------|
| IDL-файл    | ---                               | Абкдефгх. idl |
| Заголовок      | .h                                | Abcdef. h     |
| Заглушка клиента | \_c. c                             | Abcdef \_ c. c  |
| Заглушка сервера | \_s. c                             | Abcdef \_ s. c  |
