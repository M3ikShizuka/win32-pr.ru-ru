# <a name="receiving-cancellations"></a>Получение отмен

Серверное приложение может вызвать [**рпксервертестканцел**](/windows/desktop/api/Rpcdce/nf-rpcdce-rpcservertestcancel) с помощью обработчика привязки в вопросе, чтобы выяснить, запросил ли клиент отмену асинхронного вызова. \_ \_ Если Клиент отменил вызов, он вернет RPC S OK.
