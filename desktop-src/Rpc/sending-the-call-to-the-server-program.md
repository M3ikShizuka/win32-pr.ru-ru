# <a name="sending-the-call-to-the-server-program"></a>Отправка вызова в программу сервера

После подключения клиентского времени выполнения RPC к конечной точке сервера он выполняет упаковку аргументов и отправляет их на сервер. Время выполнения RPC сервера дает Упакованные аргументы заглушке, которые исменяют их, а затем передаются в серверные процедуры. Когда серверные процедуры возвращают, заглушка выбирает \[ выходные \] и выходные параметры, \[ \] а также возвращаемое значение, маршалируется и передает их в серверное время выполнения RPC. Время выполнения RPC отправляет данные обратно клиенту, где время выполнения RPC клиента направляет их на клиентскую заглушку, которая рассылает их и возвращает вызывающей стороне.
