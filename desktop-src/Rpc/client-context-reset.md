# <a name="client-context-reset"></a>Сброс контекста клиента

Когда сервер становится недоступным, клиентское приложение может освободить данные контекста, вызвав функцию RPC [**рпкссдестройклиентконтекст**](/windows/desktop/api/Rpcndr/nf-rpcndr-rpcssdestroyclientcontext).
