# <a name="the-string-attribute-in-arrays"></a>\[Строковый \] атрибут в массивах

Можно использовать \[ [строковый](/windows/desktop/Midl/string) \] атрибут для одномерных массивов символов, массивов двухбайтовых символов и массивов байтов, представляющих текстовые строки. Если используется атрибут **\[ String \]** , клиентская заглушка использует функции C-Library **strlen** или **встрлен** для подсчета количества символов в строке. Чтобы избежать возможных несоответствий, MIDL не позволяет использовать атрибут **\[ String \]** одновременно с \[ [первым \_ параметром —](/windows/desktop/Midl/first-is) \] , \[ [Last \_ —](/windows/desktop/Midl/last-is) \] , а \[ [size \_ —](/windows/desktop/Midl/size-is) \] атрибутами.

При использовании строк с завершающим нулем в C необходимо разрешить пространство для нуль-символа в конце строки. Например, при объявлении строки, которая будет содержать до 80 символов, выделите 81 символов. В следующем примере IDL-файла показано, как объявлять массивы с помощью **\[ строкового \]** атрибута.

``` syntax
/* IDL file */
[ 
  uuid(ba209999-0c6c-11d2-97cf-00c04f8eea45),
  version(8.0)
]
interface arraytest
{
  void fArray8([in, out, string] char achArray[]);
}
```
