# <a name="defining-the-interface"></a>Определение интерфейса

Определение интерфейса — это формальное описание того, как клиентское приложение и серверное приложение взаимодействуют друг с другом. Интерфейс определяет, как клиент и сервер распознают друг друга, удаленные процедуры, которые может вызывать клиентское приложение, и типы данных для параметров и возвращаемых значений процедур. Он также указывает, как передаются данные между клиентом и сервером.

Этот интерфейс определяется в язык MIDL (MIDL), который состоит из определений языка C, дополненных ключевыми словами, которые называются атрибутами, описывающими передачу данных по сети.

В файле определения интерфейса (IDL) содержатся определения типов, атрибуты и прототипы функций, описывающие передачу данных в сети. Файл конфигурации приложения (ACF) содержит атрибуты, которые настраивают приложение для конкретной операционной среды, не влияя на характеристики сети.
