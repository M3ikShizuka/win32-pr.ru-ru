# <a name="kinds-of-arrays"></a>Типы массивов

MIDL предоставляет возможность указывать в приложении RPC следующие типы массивов:

-   [Фиксированные массивы](fixed-arrays.md)
-   [Различные массивы](varying-arrays.md)
-   [Согласованные массивы](conformant-arrays.md)

Все три типа массивов можно использовать как \[ [входные](/windows/desktop/Midl/in) \] , \[ [выходные](/windows/desktop/Midl/out-idl) \] или **\[ входные \]** параметры.
