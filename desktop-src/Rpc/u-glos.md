<dl> <dt>

<span id="_rpc_udp_glos"></span><span id="_RPC_UDP_GLOS"></span>**Протокол UDP**
</dt> <dd>

Сетевой транспорт, в котором используются сокеты [*датаграмм*](d-glos.md) без подключения, и он размещается поверх протокола [*IP*](i-glos.md).

</dd> <dt>

<span id="_rpc_unbind_glos"></span><span id="_RPC_UNBIND_GLOS"></span>**отменить привязку**
</dt> <dd>

Процесс завершения логического подключения между клиентом и сервером.

</dd> <dt>

<span id="_rpc_unique_pointer_glos"></span><span id="_RPC_UNIQUE_POINTER_GLOS"></span>**уникальный указатель**
</dt> <dd>

Указатель, который может иметь значение null или указывать на существующие данные, значения которых могут изменяться во время удаленного вызова процедуры. Уникальный указатель не может быть псевдонимом. Атрибут \[ [UNIQUE](/windows/desktop/Midl/unique) \] обозначает уникальный указатель. См. также [*полный*](f-glos.md)указатель, [*указатель на ссылку*](r-glos.md).

</dd> <dt>

<span id="_rpc_uuid_glos"></span><span id="_RPC_UUID_GLOS"></span>**Универсальный уникальный идентификатор (UUID)**
</dt> <dd>

( [**GUID**](/windows/win32/api/guiddef/ns-guiddef-guid)) 128-разрядное значение, используемое при межпроцессном обмене данными для указания таких сущностей, как клиентские и серверные интерфейсы, векторы точек входа диспетчера и объекты RPC. См. также [uuidgen](/windows).

</dd> <dt>

<span id="_rpc_unmarshaling_glos"></span><span id="_RPC_UNMARSHALING_GLOS"></span>**демаршалинга**
</dt> <dd>

Процесс распаковки параметров, отправленных через границы процесса.

</dd> <dt>

<span id="_rpc_uuidgen_glos"></span><span id="_RPC_UUIDGEN_GLOS"></span>**uuidgen**
</dt> <dd>

Служебная программа, предоставляемая вместе с пакетом SDK для платформы, которая использует значение времени и идентификатор сетевой карты компьютера для формирования уникальных идентификаторов [UUID](/windows) .

</dd> </dl>
