# <a name="rpctryfinally"></a>рпктрифиналли

Инструкция **рпктрифиналли** обеспечивает структурированную обработку завершения. Это исключение возникает во время выполнения инструкций блока кода, связанного с инструкцией **рпктрифиналли** , для выполнения инструкций в блоке кода, связанного с инструкцией [**рпкфиналли**](/previous-versions/aa375699(v=vs.80)) . Все инструкции **рпктрифиналли** должны завершаться инструкцией [**рпцендфиналли**](/previous-versions/aa375634(v=vs.80)) .

Дополнительные сведения см. в разделе [**рпкфиналли**](/previous-versions/aa375699(v=vs.80)).
