# <a name="pointer-attributes-applied-to-the-parameter"></a>Атрибуты указателя, применяемые к параметру

Каждый атрибут указателя ( \[ [ref](/windows/desktop/Midl/ref) \] , \[ [UNIQUE](/windows/desktop/Midl/unique) \] и \[ [ptr](/windows/desktop/Midl/ptr) \] ) имеет характеристики, влияющие на выделение памяти. В следующей таблице приведена сводка этих характеристик.



| Атрибут указателя       | Клиент                                                                                                                                                                                                            | Сервер                                                              |
|-------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------|
| Ссылка ( \[ **ref** \] ) | Клиентское приложение должно выделить.                                                                                                                                                                                 | Требуется специальная обработка для нонтоп указателей уровня "только для **\[ выхода \]**". |
| Уникальный ( \[ **уникальный** \] ) | Если параметр, клиентское приложение должно выделить; Если внедрено, может иметь значение null. Переход от NULL к значению, отличному от NULL, приводит к выделению клиентской заглушки; изменение значения, отличного от NULL, может привести к потере.<br/> |                                                                     |
| Full ( \[ **ptr** \] )      | Если параметр, клиентское приложение должно выделить; Если внедрено, может иметь значение null. Переход от NULL к значению, отличному от NULL, приводит к выделению клиентской заглушки; изменение значения, отличного от NULL, может привести к потере.<br/> |                                                                     |



 

Атрибут **\[ ref \]** указывает на то, что указатель указывает на допустимую память. По определению клиентское приложение должно выделить всю память, требуемую для ссылочных указателей.

Уникальный указатель может изменяться со значения NULL на значение, отличное от NULL. Если уникальный указатель меняется со значения NULL на значение, отличное от NULL, на клиенте выделяется новая память. Если уникальный указатель изменяется со значения, отличного от NULL, результат может быть потерян. Дополнительные сведения см. в разделе [потеря памяти](memory-orphaning.md).
