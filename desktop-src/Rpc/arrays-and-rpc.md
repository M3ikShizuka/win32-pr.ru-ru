# <a name="arrays-and-rpc"></a>Массивы и RPC

Языки программирования C и C++ предоставляют по сути два типа массивов: одномерные и многомерные. RPC позволяет разработчикам указывать дополнительные типы массивов с помощью атрибутов MIDL для описания характеристик массивов в распределенных приложениях.

В этом разделе описываются типы массивов, доступных в RPC. В нем также обсуждаются атрибуты MIDL, которые разработчики могут использовать для описания одномерных и многомерных массивов. Этот материал представлен в следующих разделах:

-   [Типы массивов](kinds-of-arrays.md)
-   [Атрибуты массива](array-attributes.md)
-   [Многомерные массивы](multidimensional-arrays.md)
