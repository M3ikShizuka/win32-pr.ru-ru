# <a name="use-the-robust-flag"></a>Использование флага/robust

Всегда компилируйте IDL-файлы с помощью параметра [**/robust**](/windows/desktop/Midl/-robust) . С помощью параметра **/robust** создаются дополнительные сведения, позволяющие подсистеме представления данных сети выполнять проверку ошибок во время выполнения для коррелированных аргументов в динамических массивах, объединениях и в указателях интерфейса в приложениях COM и RPC. Если по не удается скомпилировать программное обеспечение с этим флагом, программное обеспечение становится доступным для атак, которые не могут быть защищены в какой-либо другой области.
