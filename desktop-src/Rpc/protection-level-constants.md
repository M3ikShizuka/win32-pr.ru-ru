# <a name="protection-level-constants"></a>Константы уровня защиты

В спецификации использование/DCE RPC были переименованы константы уровня проверки подлинности. Теперь они называются константами уровня защиты. Microsoft RPC поддерживает как старое, так и новое имя. Дополнительные сведения см. в разделе [константы уровня проверки подлинности](authentication-level-constants.md).
