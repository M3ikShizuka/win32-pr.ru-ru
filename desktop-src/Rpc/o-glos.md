<dl> <dt>

<span id="_rpc_odl_glos"></span><span id="_RPC_ODL_GLOS"></span>**Язык описания объектов (ODL)**
</dt> <dd>

Подмножество атрибутов [*MIDL*](m-glos.md) , ключевых слов, инструкций и директив, используемых для определения библиотек типов для приложений OLE Automation.

</dd> <dt>

<span id="_rpc_open_array_glos"></span><span id="_RPC_OPEN_ARRAY_GLOS"></span>**открыть массив**
</dt> <dd>

Массив, который является и согласованным, и различным; то есть и его размер, и диапазон переданных элементов определяются во время выполнения другими параметрами, структурами или выражениями.

</dd> <dt>

<span id="_rpc_osf_glos"></span><span id="_RPC_OSF_GLOS"></span>**Open Software Foundation (использование)**
</dt> <dd>

Консорциум компаний, сформированный для определения распределенной вычислительной среды (DCE).

</dd> <dt>

<span id="_rpc_orphaned_memory_glos"></span><span id="_RPC_ORPHANED_MEMORY_GLOS"></span>**потерянная память**
</dt> <dd>

Память на клиенте, которая ранее ссылалась на параметр указателя, сброшенный на значение NULL сервером.

</dd> <dt>

<span id="_rpc_output_parameter_glos"></span><span id="_RPC_OUTPUT_PARAMETER_GLOS"></span>**выходной параметр**
</dt> <dd>

Параметр, содержащий данные, которые серверное приложение передает клиентскому приложению. Атрибут \[ [out](/windows/desktop/Midl/out-idl) \] обозначает выходной параметр.

</dd> </dl>
