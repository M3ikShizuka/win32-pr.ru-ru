# <a name="configuring-the-name-service-for-windows-95"></a>Настройка службы имен для Windows 95

В Windows 95 не используется локатор Майкрософт. Чтобы использовать службу имен в приложении Windows 95, компьютер с Windows 95 должен иметь одно из следующих:

-   Входить в рабочую группу или домен, в состав которых входит компьютер под управлением Windows 2000 или Windows NT, служащий поставщиком службы имен прокси-серверов.
-   Подключитесь к главному компьютеру, на котором запущена управляющая программа NSI (нсид), которая служит шлюзом для службы каталогов ячеек Digital Equipment Corporation DCE.
