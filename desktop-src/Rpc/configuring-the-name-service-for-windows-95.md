# <a name="configuring-the-name-service-for-windows-95"></a>настройка службы имен для Windows 95

Windows 95 не использует локатор майкрософт. чтобы использовать службу имен в приложении Windows 95, компьютер с Windows 95 должен иметь одно из следующих:

-   входить в рабочую группу или домен, в состав которых входит компьютер с Windows 2000 или Windows NT, служащий поставщиком службы имен прокси-серверов.
-   Подключитесь к главному компьютеру, на котором запущена управляющая программа NSI (нсид), которая служит шлюзом для службы каталогов ячеек Digital Equipment Corporation DCE.
