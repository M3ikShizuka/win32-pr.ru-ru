# <a name="serialization-styles"></a>Стили сериализации

Существует три стиля, которые можно использовать для управления дескрипторами сериализации. А именно:

-   [Фиксированная сериализация буфера](fixed-buffer-serialization.md)
-   [Динамическая сериализация буфера](dynamic-buffer-serialization.md)
-   [Добавочная сериализация](incremental-serialization.md)

Независимо от используемого стиля необходимо создать обработчик сериализации, сериализовать данные, а затем освободить этот обработчик. Стиль задается, когда программа создает обработчик и определяет способ манипуляции с буфером. Этот обработчик поддерживает соответствующий контекст, связанный с каждым из трех стилей сериализации.
