# <a name="the-application-configuration-file-acf"></a>Файл конфигурации приложения (ACF)

Файл конфигурации приложения (ACF) состоит из двух частей: [заголовка интерфейса](the-acf-header.md), похожего на заголовок интерфейса в IDL-файле, и [тела](the-acf-body.md), который содержит атрибуты конфигурации, применяемые к типам и функциям, определенным в теле IDL-файла. Они рассматриваются в следующих разделах:

-   [Заголовок ACF](the-acf-header.md)
-   [Текст ACF](the-acf-body.md)
