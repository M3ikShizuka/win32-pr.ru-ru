# <a name="counted-character-arrays"></a>Подсчет массивов символов

Атрибут \[ [size \_ имеет](/windows/desktop/Midl/size-is) значение, \] указывающее верхнюю границу массива, а \[ [Длина атрибута length \_ ](/windows/desktop/Midl/length-is) \] указывает число передаваемых элементов массива. В дополнение к массиву, прототип удаленной процедуры должен содержать любые переменные, представляющие длину или размер, которые определяют переданные элементы массива (они могут быть отдельными параметрами или объединены со строкой в структуре). Эти атрибуты можно использовать с массивами двухбайтовых символов или однобайтовых символов точно так же, как массивы других типов.

Сведения в этом разделе описывают прототипы параметров удаленной процедуры для массивов символов. Он разделен на следующие разделы:

-   [\[в, размер \_ и, размер \_ — \] прототип](-in-size-is-and-out-size-is-prototype.md)
