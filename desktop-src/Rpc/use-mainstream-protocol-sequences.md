# <a name="use-mainstream-protocol-sequences"></a>Использование основных последовательностей протоколов

Используйте [**нкакн \_ IP \_ TCP**](/windows/desktop/Midl/ncacn-ip-tcp), [**нкакн \_ http**](/windows/desktop/Midl/ncacn-http) или [**нкакн \_ NP**](/windows/desktop/Midl/ncacn-np) для удаленных вызовов RPC. Используйте [**нкалрпк**](/windows/desktop/Midl/ncalrpc) для ЛОКАЛЬНЫХ вызовов RPC. Эти протоколы наиболее широко используются и получают наиболее тщательное и тестирование.
