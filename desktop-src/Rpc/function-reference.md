# <a name="function-reference"></a>Справочник по функциям

В этом разделе описываются функции времени выполнения удаленного вызова процедур (RPC), поддерживаемые в Microsoft RPC. В этом разделе описывается назначение функции, синтаксис, входные параметры и возвращаемые значения. В нем также содержатся дополнительные сведения, помогающие использовать функции RPC в приложении.

Все указатели, передаваемые в функции RPC, должны включать атрибут **\_ \_ RPC \_ FAR** . Например, маркер привязки RPC с указателем   **\_ \_ \* *_ преобразуется* в значение _ \_ \_ \_ \_ \_ \*  \* \* маркера** **\_ \_ \_ \* \_ \_ \_ привязки RPC. \***
