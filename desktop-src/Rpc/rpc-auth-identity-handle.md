# <a name="rpc_auth_identity_handle"></a>\_ \_ маркер идентификации RPC \_ AUTH

Тип **данных \_ \_ \_ обработчика проверки подлинности RPC** объявляет маркер, указывающий на структуру, содержащую учетные данные проверки подлинности и авторизации клиента, указанные для удаленных вызовов процедур.


```C++
typedef void* RPC_AUTH_IDENTITY_HANDLE;
```
