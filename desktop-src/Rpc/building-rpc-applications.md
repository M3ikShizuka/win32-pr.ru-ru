# <a name="building-rpc-applications"></a>Создание приложений RPC

Процедура создания распределенного приложения удаленного вызова процедур (RPC) немного различается в зависимости от следующих действий.

-   Платформа операционной системы, на которой выполняется разработка.
-   Целевая платформа.
-   Используемая версия компилятора MIDL и C или C++.
-   Используемые библиотеки API.

В этом разделе обсуждается процесс создания клиентских и серверных приложений с помощью удаленного вызова процедур (Майкрософт). Эта информация представлена в следующих разделах:

-   [Общая процедура сборки](general-build-procedure.md)
-   [Обработка исключений](exception-handling.md)
