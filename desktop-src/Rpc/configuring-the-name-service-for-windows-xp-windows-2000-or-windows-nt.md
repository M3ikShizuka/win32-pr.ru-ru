# <a name="configuring-the-name-service"></a>Настройка службы имен

Начиная с Windows 2000, при установке Windows SDK программа установки автоматически выбирает локатор Майкрософт в качестве поставщика службы имен. Поставщик службы имен можно изменить с помощью панели управления.

Главный компьютер, на котором работает нсид, выступает в качестве шлюза для службы каталогов ячеек DCE, передающий вызовы функций NSI между компьютерами, на которых работают операционные системы Майкрософт и компьютеры DCE. Сетевой адрес может содержать до 80 символов, например, 11.1.9.169 является допустимым адресом.

> [!Note]  
> Для настройки компакт-дисков DCE в качестве поставщика службы имен необходимо иметь продукт устройства DCE с цифровым оборудованием. Сведения о компакт-дисках DCE см. в документации, предоставляемой корпорацией Digital Equipment.

 

**Перенастройка поставщика службы имен**

1.  На панели управления щелкните значок " **Сетевые подключения** ". Откроется окно " **Сетевые подключения** ".
2.  Выберите сетевое подключение для настройки, щелкните его правой кнопкой мыши и выберите в контекстном меню пункт **Свойства** . Откроется диалоговое окно **Свойства соединения** .
3.  В диалоговом окне **Свойства соединения** выберите **клиент для сетей Microsoft** и нажмите кнопку **Свойства** . Откроется диалоговое окно **Свойства клиента для сети Microsoft** .
4.  В диалоговом окне **Свойства клиента для сети Microsoft** откройте раскрывающийся список **поставщик службы имен** и выберите поставщика имен из списка.
    1.  При выборе указателя Майкрософт нажмите кнопку ОК. После этого процесс настройки завершится.
    2.  При выборе службы каталогов ячеек DCE в поле **сетевой адрес** введите имя главного компьютера, на котором работает управляющая программа NSI (нсид), и нажмите кнопку ОК.

**Перенастройка поставщика службы имен для Windows 2000**

1.  На панели управления щелкните значок **сети** .

    Откроется диалоговое окно **сеть** .

2.  В диалоговом окне **сеть** нажмите кнопку **настроить**.
3.  В списке **нетворксофтваре** выберите **Конфигурация RPC**.

    Откроется диалоговое окно **Конфигурация поставщика службы имен RPC** .

4.  В диалоговом окне **Конфигурация поставщика службы имен RPC** выберите поставщика службы имен из списка.
    1.  При выборе указателя Майкрософт нажмите кнопку ОК. После этого процесс настройки завершится.
    2.  При выборе службы каталогов ячеек DCE в поле **сетевой адрес** введите имя главного компьютера, на котором работает управляющая программа NSI (нсид), и нажмите кнопку ОК.
