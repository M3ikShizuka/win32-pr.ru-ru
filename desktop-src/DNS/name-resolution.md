# <a name="name-resolution"></a>Разрешение имен

DNS обеспечивает разрешение имен: процесс сравнения имени узла (например, имени компьютера) со списком записей ресурсов и идентификации соответствующего IP-адреса. Обратная разрешение имен — это процесс сравнения IP-адреса со списком записей ресурсов и идентификации соответствующего имени узла. Эти процессы запускаются, когда DNS-сервер получает запрос.
