# <a name="writebackzone-method-of-the-microsoftdns_zone-class"></a>Метод Вритебаккзоне \_ класса зоны микрософтднс

Метод **вритебаккзоне** сохраняет данные зоны в файл зоны.

## <a name="syntax"></a>Синтаксис


```mof
void WriteBackZone();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
