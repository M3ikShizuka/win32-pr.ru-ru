# <a name="dns-wmi-provider-scripting-examples"></a>Примеры сценариев поставщика WMI DNS

В этом разделе приводятся примеры сценариев, демонстрирующие создание скриптов административных задач с помощью поставщика WMI для DNS.

Предоставляются следующие типы примеров сценариев.



| Section                                                                                     | Предоставленные образцы                                                                                                                                                                          |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Управление DNS-сервером](dns-wmi-provider-samples-managing-a-dns-server.md)                 | Содержит примеры, обеспечивающие управление DNS-сервером, включая остановку, запуск или перезапуск сервера, вывод списка зон, добавление или удаление IP-адресов и т. д.                |
| [Управление записями ресурсов DNS](dns-wmi-provider-samples-managing-dns-resource-records.md) | Демонстрирует подключение к поставщику WMI DNS и выполнение различных задач, связанных с управлением записями ресурсов — Добавление, удаление, изменение или перечисление записей ресурсов. |
| [Управление зоны DNS](dns-wmi-provider-samples-managing-dns-zones.md)                       | Примеры включают создание, изменение и удаление зоны, а также приостановку, возобновление, обновление и другие задачи зоны, выполняемые администраторам DNS.                                  |
