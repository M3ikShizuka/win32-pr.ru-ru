# <a name="clearcache-method-of-the-microsoftdns_cache-class"></a>Метод ClearCache \_ класса кэша микрософтднс

Метод **ClearCache** очищает кэш DNS-сервера записей ресурсов.

## <a name="syntax"></a>Синтаксис


```mof
void ClearCache();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
