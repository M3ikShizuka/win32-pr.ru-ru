# <a name="getdistinguishedname-method-of-the-microsoftdns_domain-class"></a>Метод Жетдистингуишеднаме \_ класса домена микрософтднс

Метод **жетдистингуишеднаме** извлекает различающееся имя DS для зоны.

## <a name="syntax"></a>Синтаксис


```mof
string GetDistinguishedName();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Возвращает различающееся имя DS для зоны.
