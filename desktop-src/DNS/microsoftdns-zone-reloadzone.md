# <a name="reloadzone-method-of-the-microsoftdns_zone-class"></a>Метод Релоадзоне \_ класса зоны микрософтднс

Метод **релоадзоне** перезагружает зону DNS из своей базы данных.

## <a name="syntax"></a>Синтаксис


```mof
void ReloadZone();
```



## <a name="parameters"></a>Параметры

Этот метод не имеет параметров.

## <a name="return-value"></a>Возвращаемое значение

Этот метод не возвращает значение.
