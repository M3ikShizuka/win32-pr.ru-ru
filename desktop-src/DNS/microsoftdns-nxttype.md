# <a name="microsoftdns_nxttype-class"></a>\_Класс микрософтднс нксттипе

Подкласс [**микрософтднс \_ ресаурцерекорд**](microsoftdns-resourcerecord.md) , представляющий следующую запись ресурса (NXT).

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
class MicrosoftDNS_NXTType : MicrosoftDNS_ResourceRecord
{
  string NextDomainName;
  string Types;
};
```

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Класс **микрософтднс \_ нксттипе** содержит следующие методы.



| Метод                             | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
|:-----------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **креатеинстанцефромпропертидата** | Создает экземпляр записи ресурса NXT на основе данных в входных параметрах метода: имя DNS-сервера записи, имя контейнера, имя владельца, класс (по умолчанию — IN), значение срока жизни и флаг сопоставления WINS, время ожидания для отмены просмотра, время ожидания кэша WINS и имя домена для добавления. Он возвращает ссылку на новый объект в качестве выходного параметра. <br/> Квалификаторы: реализованные, статические<br/>                                                                                                                                           |
| **Изменить**                         | Обновляет значения TTL, флага сопоставления, времени ожидания при поиске, истечения времени ожидания кэша и домена результата до значений, указанных в качестве входных параметров этого метода. Если новое значение параметра не указано, текущее значение параметра не изменяется. Метод возвращает ссылку на измененный объект в качестве выходного параметра. <br/> Квалификаторы: Реализовано<br/> **Windows Server 2003:** Этот метод также обновляет Некстдомаиннаме и типы значениями, указанными в качестве входных параметров этого метода.<br/> <br/> |
