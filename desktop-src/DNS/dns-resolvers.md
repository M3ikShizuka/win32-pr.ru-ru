# <a name="dns-resolvers"></a>Арбитры DNS

[*сопоставитель*](r-gly.md) — это программный компонент, объединенный в стек протокола TCP/IP Windows 2000, который взаимодействует с DNS-серверами для создания и разрешения запросов на разрешение имен. Пакет также доступен в любом клиенте, поддерживающем DNS, а не на сервере Майкрософт. Арбитры конфликтов существуют на DNS-клиентах и DNS-серверах.
