# <a name="dns-resolvers"></a>Арбитры DNS

[*Сопоставитель*](r-gly.md) — это программный компонент, Объединенный в стек TCP/IP протокола управления передачей Windows 2000, который взаимодействует с DNS-серверами для создания и разрешения запросов на разрешение имен. Пакет также доступен в любом клиенте, поддерживающем DNS, а не на сервере Майкрософт. Арбитры конфликтов существуют на DNS-клиентах и DNS-серверах.
