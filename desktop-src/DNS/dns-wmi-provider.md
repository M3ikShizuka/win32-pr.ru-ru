# <a name="dns-wmi-provider"></a>Поставщик WMI для DNS

поставщик WMI для dns позволяет администраторам и программистам настраивать записи ресурсов (rr) службы доменных имен (dns) и dns-серверы с помощью инструментарий управления Windows (WMI) (WMI).
