# <a name="microsoftdns_atmatype-class"></a>\_Класс микрософтднс атматипе

Подкласс [**микрософтднс \_ ресаурцерекорд**](microsoftdns-resourcerecord.md) , представляющий запись адреса ATM в имя (ATMA).

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
class MicrosoftDNS_ATMAType : MicrosoftDNS_ResourceRecord
{
  uint16 Format;
  string ATMAddress;
};
```

## <a name="members"></a>Члены

Класс **микрософтднс \_ атматипе** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Класс **микрософтднс \_ атматипе** содержит следующие методы.



| Метод                             | Описание                                                                                                                                                                                                                                                                                                                                        |
|:-----------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **креатеинстанцефромпропертидата** | Создает экземпляр записи ресурса ATMA на основе данных в входных параметрах метода: имя DNS-сервера записи, имя контейнера, имя владельца или узла, класс (по умолчанию — IN), значение срока жизни и формат ATM и адрес. Возвращает ссылку на новый объект в качестве выходного параметра. <br/> Квалификаторы: реализованные, статические<br/> |
| **Изменение**                         | Обновляет параметры TTL, Format и ATMA на значения, указанные в качестве входных параметров этого метода. Если новое значение параметра не указано, текущее значение параметра не изменяется. Метод возвращает ссылку на измененный объект в качестве выходного параметра. <br/> Квалификаторы: Реализовано<br/>         |



 

### <a name="properties"></a>Свойства

Класс **микрософтднс \_ атматипе** имеет следующие свойства.

<dl> <dt>

**атмаддресс**
</dt> <dd> <dl> <dt>

Тип данных: **строка**
</dt> <dt>

Тип доступа: только для чтения
</dt> </dl>

Строка переменной длины октетов, содержащая ATM-адрес узла или владельца, к которому относится эта запись ресурса. Первые 4 байта массива используются для хранения размера строки октета. Наиболее значимый байт хранится в байте 0.

</dd> <dt>

**Формат**
</dt> <dd> <dl> <dt>

Тип данных: **UInt16**
</dt> <dt>

Тип доступа: только для чтения
</dt> </dl>

Формат адреса ATM. Допустимые значения: 0, указывающий формат адреса конечной системы (АЕСА) ATM, а 1 — формат E. 164.

</dd> </dl>
