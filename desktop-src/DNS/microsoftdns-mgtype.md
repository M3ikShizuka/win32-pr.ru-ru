# <a name="microsoftdns_mgtype-class"></a>\_Класс микрософтднс мгтипе

Подкласс [**микрософтднс \_ ресаурцерекорд**](microsoftdns-resourcerecord.md) , представляющий запись почтовой группы (MG).

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
class MicrosoftDNS_MGType : MicrosoftDNS_ResourceRecord
{
  string MGMailbox;
};
```

## <a name="members"></a>Члены

Класс **микрософтднс \_ мгтипе** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Класс **микрософтднс \_ мгтипе** содержит следующие методы.



| Метод                             | Описание                                                                                                                                                                                                                                                                                                                                                      |
|:-----------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **креатеинстанцефромпропертидата** | Этот метод создает тип записи ресурса MG на основе данных в входных параметрах метода: имя DNS-сервера записей, имя контейнера, имя владельца почтовой группы, класс (по умолчанию — IN), значение срока жизни и имя почтового ящика. Он возвращает ссылку на новый объект в качестве выходного параметра. <br/> Квалификаторы: реализованные, статические<br/> |
| **Изменить**                         | Этот метод обновляет почтовый ящик TTL и MG значениями, указанными в качестве входных параметров этого метода. Если новое значение параметра не указано, текущее значение параметра не изменяется. Метод возвращает ссылку на измененный объект в качестве выходного параметра. <br/> Квалификаторы: Реализовано<br/>                |
