# <a name="microsoftdns_afsdbtype-class"></a>\_Класс микрософтднс афсдбтипе

Подкласс [**микрософтднс \_ ресаурцерекорд**](microsoftdns-resourcerecord.md) , представляющий запись сервера базы данных AFS (AFSDB).

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
class MicrosoftDNS_AFSDBType : MicrosoftDNS_ResourceRecord
{
  uint16 Subtype;
  string ServerName;
};
```

## <a name="members"></a>Члены

Класс **микрософтднс \_ афсдбтипе** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Класс **микрософтднс \_ афсдбтипе** содержит следующие методы.



| Метод                             | Описание                                                                                                                                                                                                                                                                                                                                                   |
|:-----------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **креатеинстанцефромпропертидата** | Создает экземпляр записи ресурса AFSDB на основе данных в входных параметрах метода: имя DNS-сервера записи, имя контейнера, имя владельца/ячейки, класс (по умолчанию — IN), значение срока жизни, а также тип и имя узла сервера AFS. Возвращает ссылку на новый объект в качестве выходного параметра. <br/> Квалификаторы: реализованные, статические<br/> |
| **Изменить**                         | Этот метод обновляет значения TTL, подтип и имя сервера до значений, указанных в качестве входных параметров этого метода. Если новое значение параметра не указано, текущее значение параметра не изменяется. Метод возвращает ссылку на измененный объект в качестве выходного параметра. <br/> Квалификаторы: Реализовано<br/>   |



 

### <a name="properties"></a>Свойства

Класс **микрософтднс \_ афсдбтипе** имеет следующие свойства.

<dl> <dt>

**ServerName**
</dt> <dd> <dl> <dt>

Тип данных: **строка**
</dt> <dt>

Тип доступа: только для чтения
</dt> </dl>

Полное доменное имя, указывающее узел с сервером для ячейки AFS, указанной в имени владельца.

</dd> <dt>

**Подтип**
</dt> <dd> <dl> <dt>

Тип данных: **UInt16**
</dt> <dt>

Тип доступа: только для чтения
</dt> </dl>

Подтип ведущего сервера AFS. Для подтипа 1 (значение 1) на узле имеется сервер AFS версии 3,0 для именованной ячейки AFS. В случае подтипа 2 (значение = 2) узел имеет прошедший проверку подлинности сервер имен, в котором размещен узел корневого каталога ячеек для именованной ячейки DCE/NCA.

</dd> </dl>
