# <a name="dns-concepts"></a>Основные понятия DNS

Существует несколько основных понятий, на которых работает функциональная структура DNS. Эти основные понятия изменяются от определений пространства имен до разделительной границ обязанностей по разрешению имен. Понимание этих концепций является долгим способом получения надежной рабочей информации о DNS. В этом разделе определены и описаны следующие основные понятия DNS:

-   [Пространство имен](name-space.md)
-   [Домены DNS](dns-domains.md)
-   [Зоны DNS](dns-zones.md)
-   [Корневой](root.md)
-   [Разрешение имен](name-resolution.md)
-   [DNS-серверы](dns-servers.md)
-   [Арбитры DNS](dns-resolvers.md)
-   [Записи ресурсов](resource-records.md)
