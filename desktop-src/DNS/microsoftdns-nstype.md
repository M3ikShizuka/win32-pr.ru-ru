# <a name="microsoftdns_nstype-class"></a>\_Класс микрософтднс нстипе

Подкласс [**микрософтднс \_ ресаурцерекорд**](microsoftdns-resourcerecord.md) , представляющий запись сервера имен (NS).

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
class MicrosoftDNS_NSType : MicrosoftDNS_ResourceRecord
{
  string NSHost;
};
```

## <a name="members"></a>Члены

Класс **микрософтднс \_ нстипе** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Класс **микрософтднс \_ нстипе** содержит следующие методы.



| Метод                             | Описание                                                                                                                                                                                                                                                                                                                                                      |
|:-----------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **креатеинстанцефромпропертидата** | Создает экземпляр записи ресурса NS DNS на основе данных в входных параметрах метода: имя DNS-сервера записи, имя контейнера, имя владельца, класс (по умолчанию — IN), значение срока жизни и узел с полномочиями для домена. Он возвращает ссылку на новый объект в качестве выходного параметра. <br/> Квалификаторы: реализованные, статические<br/> |
| **Изменение**                         | Обновляет значения TTL и узла NS значениями, указанными в качестве входных параметров этого метода. Если новое значение параметра не указано, текущее значение параметра не изменяется. Метод возвращает ссылку на измененный объект в качестве выходного параметра. <br/> Квалификаторы: Реализовано<br/>                               |
