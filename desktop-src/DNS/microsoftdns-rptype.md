# <a name="microsoftdns_rptype-class"></a>\_Класс микрософтднс рптипе

Подкласс [**микрософтднс \_ ресаурцерекорд**](microsoftdns-resourcerecord.md) , представляющий запись ответственного лица (RP).

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
class MicrosoftDNS_RPType : MicrosoftDNS_ResourceRecord
{
  string RPMailbox;
  string TXTDomainName;
};
```

## <a name="members"></a>Члены

Класс **микрософтднс \_ рптипе** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Класс **микрософтднс \_ рптипе** содержит следующие методы.



| Метод                             | Описание                                                                                                                                                                                                                                                                                                                                                                                         |
|:-----------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **креатеинстанцефромпропертидата** | Создает экземпляр типа ресурса RP на основе данных в входных параметрах метода: имя DNS-сервера записи, имя контейнера, имя владельца/ответственного лица, класс (по умолчанию — IN), значение срока жизни и имена доменов для местоположения RR пользователя и TXT-файла. Он возвращает ссылку на новый объект в качестве выходного параметра. <br/> Квалификаторы: реализованные, статические<br/> |
| **Изменение**                         | Этот метод обновляет имя домена "TTL", "почтовый ящик RP" и "TXT" на значения, указанные в качестве входных параметров этого метода. Если новое значение параметра не указано, текущее значение параметра не изменяется. Метод возвращает ссылку на измененный объект в качестве выходного параметра. <br/> Квалификаторы: Реализовано<br/>                                  |
