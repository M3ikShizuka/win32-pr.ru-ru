# <a name="managing-dns-zones"></a>Управление зоны DNS

Зона DNS — это база данных записей RR, которая соответствует части иерархического пространства имен DNS. Зоны DNS используются для определения того, какие DNS-серверы являются полномочными для разрешения запросов разрешения имен для определенного раздела иерархии DNS.

## <a name="administration-steps"></a>Шаги администрирования

Поставщик WMI для DNS позволяет администрировать зоны DNS с самого полномочного DNS-сервера или с удаленных узлов. Общие шаги, необходимые для администрирования зоны с помощью поставщика WMI DNS:

-   Подключение к поставщику WMI DNS
-   Получение экземпляра зоны
-   Перечисление или изменение свойства зоны или с помощью метода
