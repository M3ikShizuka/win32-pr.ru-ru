# <a name="dns-wmi-classes"></a>Классы WMI DNS

Поставщик WMI для DNS определяет следующие классы WMI DNS.



| Класс WMI Microsoft DNS                                                                               | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|-------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_Сервер микрософтднс**](microsoftdns-server.md)                                                   | Описывает DNS-сервер. Каждый экземпляр этого класса может быть связан с одним экземпляром [**\_ кэша микрософтднс**](microsoftdns-cache.md)класса, одним экземпляром класса [**микрософтднс \_ русинтс**](microsoftdns-roothints.md)и несколькими экземплярами класса [**микрософтднс \_ Zone**](microsoftdns-zone.md).                                                                                                                                                                                                                                                                                                                                                                                       |
| [**\_Домен микрософтднс**](microsoftdns-domain.md)                                                   | Представляет домен в дереве иерархии DNS.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |

Зоны могут быть связаны с несколькими экземплярами классов [**микрософтднс \_ domain**](microsoftdns-domain.md) и [**микрософтднс \_ ресаурцерекорд**](microsoftdns-resourcerecord.md).                                                                                                                                                                                                                                                                                                                                                                    |

Этот класс упрощает визуализацию включения DNS-объектов, а не представляет реальный объект. Класс [**микрософтднс \_ Cache**](microsoftdns-cache.md)— это контейнер для [*записей ресурсов*](r-gly.md) , кэшированных DNS-сервером.
