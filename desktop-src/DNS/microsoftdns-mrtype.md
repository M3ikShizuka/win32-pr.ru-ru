# <a name="microsoftdns_mrtype-class"></a>\_Класс микрософтднс мртипе

Подкласс [**микрософтднс \_ ресаурцерекорд**](microsoftdns-resourcerecord.md) , представляющий запись переименования почтового ящика (MR).

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
class MicrosoftDNS_MRType : MicrosoftDNS_ResourceRecord
{
  string MRMailbox;
};
```

## <a name="members"></a>Члены

Класс **микрософтднс \_ мртипе** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Класс **микрософтднс \_ мртипе** содержит следующие методы.



| Метод                             | Описание                                                                                                                                                                                                                                                                                                                                                       |
|:-----------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **креатеинстанцефромпропертидата** | Этот метод создает тип RR типа "MR" на основе данных в входных параметрах метода: имя DNS-сервера записи, имя контейнера, имя владельца почтового ящика, класс (по умолчанию — IN), значение срока жизни и переименование почтового ящика. Он возвращает ссылку на новый объект в качестве выходного параметра. <br/> Квалификаторы: реализованные, статические<br/> |
| **Изменение**                         | Этот метод обновляет в почтовом ящике TTL и MR значения, указанные в качестве входных параметров этого метода. Если новое значение параметра не указано, текущее значение параметра не изменяется. Метод возвращает ссылку на измененный объект в качестве выходного параметра. <br/> Квалификаторы: Реализовано<br/>                 |
