# <a name="dns-domains"></a>Домены DNS

Домены DNS знакомы большинству компьютерных специалистов, использующих Интернет; домены — это узлы в пространстве имен иерархии DNS, а домены можно разделить на поддомены. Домены в вымышленной иерархии gadgets.widgets.microsoft.com, например, включают гаджеты, мини-приложения, домены Microsoft и com.
