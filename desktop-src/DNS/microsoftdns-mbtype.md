# <a name="microsoftdns_mbtype-class"></a>\_Класс микрософтднс мбтипе

Подкласс [**микрософтднс \_ ресаурцерекорд**](microsoftdns-resourcerecord.md) , представляющий запись ресурса почтового ящика (МБ).

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
class MicrosoftDNS_MBType : MicrosoftDNS_ResourceRecord
{
  string MBHost;
};
```

## <a name="members"></a>Члены

Класс **микрософтднс \_ мбтипе** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Класс **микрософтднс \_ мбтипе** содержит следующие методы.



| Метод                             | Описание                                                                                                                                                                                                                                                                                                                               |
|:-----------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **креатеинстанцефромпропертидата** | Создает экземпляр RR в МБ на основе данных в входных параметрах метода: имя DNS-сервера записи, имя контейнера, имя владельца почтового ящика, класс (по умолчанию — IN), значение срока жизни и узел почтового ящика. Он возвращает ссылку на новый объект в качестве выходного параметра. <br/> Квалификаторы: реализованные, статические<br/> |
| **Изменить**                         | Обновляет узлы TTL и MB до значений, указанных в качестве входных параметров этого метода. Если новое значение параметра не указано, текущее значение параметра не изменяется. Метод возвращает ссылку на измененный объект в качестве выходного параметра. <br/> Квалификаторы: Реализовано<br/>        |
