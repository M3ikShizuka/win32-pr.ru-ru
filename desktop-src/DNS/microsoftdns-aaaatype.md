# <a name="microsoftdns_aaaatype-class"></a>\_Класс микрософтднс аааатипе

Подкласс [**микрософтднс \_ ресаурцерекорд**](microsoftdns-resourcerecord.md) , представляющий запись IPv6-адреса (AAAA). Запись AAAA обычно произносится как «четыре» записи.

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
class MicrosoftDNS_AAAAType : MicrosoftDNS_ResourceRecord
{
  string IPv6Address;
};
```

## <a name="members"></a>Члены

Класс **микрософтднс \_ аааатипе** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Класс **микрософтднс \_ аааатипе** содержит следующие методы.



| Метод                             | Описание                                                                                                                                                                                                                                                                                                                                  |
|:-----------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **креатеинстанцефромпропертидата** | Создает экземпляр типа ресурса RR на основе данных в входных параметрах метода: имя DNS-сервера записи, имя контейнера, имя владельца/узла, класс (по умолчанию — IN), значение срока жизни и IPv6-адрес. Он возвращает ссылку на новый объект в качестве выходного параметра. <br/> Квалификаторы: реализованные, статические<br/> |
| **Изменение**                         | Обновляет значения TTL и IPv6 до значений, указанных в качестве входных параметров этого метода. Если новое значение параметра не указано, текущее значение параметра не изменяется. Метод возвращает ссылку на измененный объект в качестве выходного параметра. <br/> Квалификаторы: Реализовано<br/>      |
