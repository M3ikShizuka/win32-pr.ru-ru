# <a name="microsoftdns_soatype-class"></a>\_Класс микрософтднс соатипе

Подкласс [**микрософтднс \_ ресаурцерекорд**](microsoftdns-resourcerecord.md) , представляющий начальную запись зоны (SOA).

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
class MicrosoftDNS_SOAType : MicrosoftDNS_ResourceRecord
{
  uint32 SerialNumber;
  string PrimaryServer;
  string ResponsibleParty;
  uint32 RefreshInterval;
  uint32 RetryDelay;
  uint32 ExpireLimit;
  uint32 MinimumTTL;
};
```

## <a name="members"></a>Члены

Класс **микрософтднс \_ соатипе** имеет следующие типы членов:

-   [Методы](#methods)
-   [Свойства](#properties)

### <a name="methods"></a>Методы

Класс **микрософтднс \_ соатипе** содержит следующие методы.



| Метод     | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|:-----------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Изменить** | Обновляет значение TTL, серийный номер SOA, сервер-источник, ответственный субъект, интервал обновления, задержку повтора, ограничение срока действия и минимальный срок жизни (для зоны) на значения, указанные в качестве входных параметров этого метода. Если новое значение параметра не указано, текущее значение параметра не изменяется. Метод возвращает ссылку на измененный объект в качестве выходного параметра. <br/> Квалификаторы: Реализовано<br/> |
