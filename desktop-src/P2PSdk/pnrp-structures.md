# <a name="pnrp-structures"></a>Структуры PNRP

API поставщика пространства имен PNRP использует следующие структуры:



| Структура                                                             | Описание                                                                                                                                                         |
|-----------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_ \_ сведения о конечной точке PNRP Peer \_**](/windows/desktop/api/P2P/ns-p2p-peer_pnrp_endpoint_info)         | Содержит IP-адреса и данные, связанные с конечной точкой одноранговой сети.                                                                                                 |
| [**\_сведения о одноранговом \_ облаке PNRP \_**](/windows/desktop/api/P2P/ns-p2p-peer_pnrp_cloud_info)               | Содержит сведения о облаке протокола PNRP.                                                                                            |
| [**\_ \_ сведения о регистрации однорангового PNRP \_**](/windows/desktop/api/P2P/ns-p2p-peer_pnrp_registration_info) | Содержит сведения, предоставленные одноранговым удостоверением при регистрации в облаке PNRP.                                                                           |
| [PNRP и BLOB-объект](pnrp-and-blob.md)                                    | Передает данные в структуру [**всакуерисет**](winsock-nsp-reference-links.md) во время вызовов нескольких функций.                                                  |
| [PNRP и ВСАКУЕРИСЕТ](pnrp-and-wsaqueryset.md)                      | Облегчает разрешение имен и перечисление имен и облаков.                                                                                         |
| [**\_ИД облака \_ PNRP**](/windows/desktop/api/Pnrpdef/ns-pnrpdef-pnrp_cloud_id)                              | Содержит значения, определяющие сетевое облако.                                                                                                                    |
| [**пнрпклаудинфо**](/windows/desktop/api/Pnrpns/ns-pnrpns-pnrpcloudinfo)                                | На который указывает элемент **лпблоб** структуры [**всакуерисет**](winsock-nsp-reference-links.md) .                                                            |
| [**ПНРПИНФО \_ v1**](/windows/desktop/api/Pnrpns/ns-pnrpns-pnrpinfo_v1)                                      | Указывает элемент **лпблоб** структуры [**всакуерисет**](winsock-nsp-reference-links.md)                                                             |
| [**ПНРПИНФО \_ v2**](/previous-versions/windows/desktop/legacy/aa371671(v=vs.85))                                   | На который указывает элемент **лпблоб** структуры [**всакуерисет**](winsock-nsp-reference-links.md) , и обеспечивает поддержку непрозрачных данных, зависящих от приложения. |
