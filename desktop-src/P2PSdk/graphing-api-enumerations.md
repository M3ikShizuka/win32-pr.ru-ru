# <a name="graphing-api-constants"></a>Построение диаграмм констант API

API-интерфейс для построения диаграмм использует следующие константы:



| Константа                                                          | Описание                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------|
| [**Тип события однорангового \_ графа \_ \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_event_type)         | Указывает типы одноранговых событий, для которых должно быть уведомлено приложение. |
| [**\_ \_ флаги свойств однорангового графа \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_property_flags) | Задает свойства однорангового графа.                             |
| [**область однорангового \_ графа \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_scope)                    | Указывает область сети однорангового графа.                      |
| [**\_ \_ флаги состояния одноранговых диаграмм \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_status_flags)     | Указывает текущее состояние узла в одноранговой диаграмме.     |
| [**\_ \_ тип изменения однорангового узла \_**](/windows/desktop/api/P2P/ne-p2p-peer_node_change_type)         | Указывает типы изменений состояния графа одноранговых узлов.            |
