# <a name="about-people-near-me"></a>О программе "Соседние пользователи"

"Соседние пользователи" (PNM) — это способ объявить себя и свои интересы пользователям и приложениям для упрощения подключения и совместной работы. «Near» в меню «соседние пользователи» — это набор пользователей, расположенных в одной подсети. PNM — это новая возможность [одноранговых сетей Windows](what-is-peer-networking-.md) , которая входит в состав Windows Vista.

Этот раздел документации содержит сведения о PNM и содержит следующие разделы:

-   [Зачем использовать "Соседние пользователи"?](why-use-people-near-me-.md)
-   [Требования "Соседние пользователи"](people-near-me-requirements.md)

Сведения о базовой архитектуре, а также об интерфейсах API, хранилищах и компонентах, используемых в PNM, см. в [документации по архитектуре "Соседние пользователи](people-near-me-architecture.md)".
