# <a name="common-structures"></a>Общие структуры

Одноранговая инфраструктура использует следующие общие структуры.



| Структура                                                                          | Описание                                                                                                                         |
|------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------|
| [**Одноранговый \_ адрес**](/windows/desktop/api/P2P/ns-p2p-peer_address)                                              | Указывает сведения об IP-адресе.                                                                                     |
| [**\_сведения о одноранговом соединении \_**](/windows/desktop/api/P2P/ns-p2p-peer_connection_info)                             | Содержит сведения о соединении. Эта структура возвращается при перечислении одноранговых диаграмм или группирования соединений. |
| [**Одноранговые \_ данные**](/windows/desktop/api/P2P/ns-p2p-peer_data)                                                    | Содержит двоичные данные.                                                                                                               |
| [**\_ \_ \_ данные изменения подключения \_ однорангового события**](/windows/desktop/api/P2P/ns-p2p-peer_event_connection_change_data) | Содержит обновленную информацию, которая включает изменения в соседа или прямое соединение.                                              |
| [**\_ \_ входные \_ данные однорангового события**](/windows/desktop/api/P2P/ns-p2p-peer_event_incoming_data)                    | Содержит обновленную информацию, которая включает изменения данных, получаемые узлом от соседа или прямого подключения.                       |
| [**данные изменения одноранговой \_ \_ записи событий \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_record_change_data)         | Содержит обновленную информацию о том, что приложение запрашивает изменения данных в записи или типе записи.                              |
| [**Одноранговая \_ запись**](/windows/desktop/api/P2P/ns-p2p-peer_record)                                                | Содержит объект Record, используемый приложением.                                                                                |
| [**данные одноранговой \_ версии \_**](/windows/desktop/api/P2P/ns-p2p-peer_version_data)                                   | Содержит сведения о версии API-интерфейсов одноранговой диаграммы и группирования.                                                         |
