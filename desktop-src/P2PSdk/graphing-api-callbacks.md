# <a name="graphing-api-callbacks"></a>Построение графов обратных вызовов API

API Graph использует следующие обратные вызовы:



| Обратный вызов                                                          | Описание                                                                                                                                                                                                                  |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [*ПФНПИР \_ Бесплатные \_ \_ данные безопасности*](/windows/desktop/api/P2P/nc-p2p-pfnpeer_free_security_data) | Указывает функцию, которую Инфраструктура одноранговых диаграмм вызывает для освобождения данных, возвращаемых [*пфнпир \_ безопасной \_ записью*](/windows/desktop/api/P2P/nc-p2p-pfnpeer_secure_record) и пфнпир проверки обратного вызова [*\_ \_ записи*](/windows/desktop/api/P2P/nc-p2p-pfnpeer_validate_record) . |
| [*ПФНПИР \_ защищенная \_ запись*](/windows/desktop/api/P2P/nc-p2p-pfnpeer_secure_record)            | Указывает функцию, которую Инфраструктура одноранговых диаграмм вызывает для защиты записей.                                                                                                                                        |
| [*\_запись проверки \_ пфнпир*](/windows/desktop/api/P2P/nc-p2p-pfnpeer_validate_record)        | Указывает функцию, которую Инфраструктура одноранговых диаграмм вызывает для проверки записей.                                                                                                                                      |
