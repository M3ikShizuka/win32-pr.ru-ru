# <a name="windows-vista-components-for-people-near-me"></a>Компоненты Windows Vista для "соседних пользователей"

[Соседние пользователи](about-people-near-me.md) (PNM) используют следующие компоненты Windows Vista:

### <a name="web-service-discovery"></a>Обнаружение веб-служб

Компонент обнаружения веб-служб позволяет публиковать и обнаруживать службы в локальной подсети. В PNM пользователь — это служба, которая объявляется в локальной подсети. Дополнительные сведения об обнаружении веб-служб см. в разделе [Обнаружение веб-службы XML](/previous-versions/dotnet/netframework-4.0/fxx6cfx2(v=vs.100)).

### <a name="schannel"></a>Schannel

Компонент SChannel обеспечивает зашифрованный обмен данными между двумя конечными точками.

### <a name="windows-sockets"></a>Сокеты Windows

Компонент сокетов Windows используется приложениями для взаимодействия по протоколу TCP/IP.

### <a name="next-generation-tcpip-stack"></a>Стек TCP/IP следующего поколения

Новый стек TCP/IP в Windows Vista с встроенным и включенным по умолчанию протоколом IPv6. Дополнительные сведения см. [в разделе стек TCP/IP следующего поколения в Windows Vista](https://www.microsoft.com/technet/itsolutions/network/tcpip/default.mspx).
