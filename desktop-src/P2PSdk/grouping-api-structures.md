# <a name="grouping-api-structures"></a>Группирование структур API

API группирования использует следующие структуры.



| Структура                                                                  | Описание                                                                                                   |
|----------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_сведения об учетных данных пиринга \_**](/windows/desktop/api/P2P/ns-p2p-peer_credential_info)                     | Определяет сведения, используемые для получения и выпуска учетных данных безопасности однорангового узла.                                   |
| [**\_ \_ \_ данные изменения члена \_ однорангового события**](/windows/desktop/api/P2P/ns-p2p-peer_event_member_change_data) | Содержит данные, описывающие изменение состояния элемента одноранговой группы.                                   |
| [**\_ \_ данные события одноранговой группы \_**](/windows/win32/api/p2p/ns-p2p-peer_group_event_data-r1)                  | Содержит сведения об определенном событии одноранговой группы.                                     |
| [**\_ \_ Регистрация событий одноранговой группы \_**](/windows/desktop/api/P2P/ns-p2p-peer_group_event_registration)  | Определяет конкретное событие одноранговой группы, для которого может быть зарегистрирован элемент.                                            |
| [**свойства одноранговой \_ группы \_**](/windows/desktop/api/P2P/ns-p2p-peer_group_properties)                   | Содержит данные о политике членства одноранговой группы.                                                    |
| [**\_сведения о приглашении к одноранговому узлу \_**](/windows/desktop/api/P2P/ns-p2p-peer_invitation_info)                     | Определяет сведения о приглашении для присоединения к одноранговой группе. Приглашения представляются в виде строк в Юникоде. |
| [**Одноранговый \_ член**](/windows/desktop/api/P2P/ns-p2p-peer_member)                                        | Содержит сведения, описывающие члена одноранговой группы.                                                 |
