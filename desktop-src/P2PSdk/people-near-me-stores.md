# <a name="people-near-me-stores"></a>Магазины "Соседние пользователи"

Приложения, поддерживающие [соседние пользователи](about-people-near-me.md) (PNM), могут использовать следующие хранилища данных:

### <a name="windows-address-book"></a>Windows Адресная книга

в Windows адресной книге хранятся сведения о контактах и их цифровых сертификатах. контакт "**я**", представляющий текущего пользователя, также хранится в адресной книге Windows.

### <a name="certificate-store"></a>Хранилище сертификатов

Хранилище сертификатов содержит самозаверяющий сертификат для контакта "**Me**".

### <a name="application-store"></a>Магазин приложений

Установщик приложения регистрирует приложение в PNM во время процесса установки. Это объекты, которые могут быть просмотрены другими пользователями при попытке подключиться к пользователю с помощью общего приложения, например компьютерной игры. Для каждого приложения магазин приложений содержит имя приложения, глобальный уникальный идентификатор (GUID), область приложения, описание и путь к исполняемому файлу программы. Области приложения можно присвоить значение None (не объявлено), соседних пользователей (объявленных в локальной подсети), контакты (объявленные только для контактов) или все (объявленные в локальной подсети и для контактов). магазин приложений находится в реестре Windows Vista.
