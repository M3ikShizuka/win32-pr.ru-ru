# <a name="why-use-people-near-me"></a>Зачем использовать "Соседние пользователи"?

Ниже приведены некоторые примеры сценариев, которые могут выполнять [Пользователи с ближайшими пользователями](about-people-near-me.md) (PNM).

-   Инициация разговора с помощью человека в конференц-зале во время собрания. Использование приложения для разговора для динамического обнаружения других пользователей в комнате, выбора конкретного пользователя и начала сеанса разговора для совместной работы или обмена данными.
-   API PNM обеспечивает общий доступ к документам для всех пользователей в конференц-зале. Хотя флэш-накопители (рассматриваемое) универсальной последовательной шины (USB) значительно упрощают перенос документов между компьютерами в закрытом параметре, например на собрании, он по-прежнему занимает много времени и усилий, чтобы подключить UFD к каждому компьютеру и скопировать документ в локальную папку. С помощью приложений, поддерживающих PNM, можно выбрать пользователей в комнате, с которыми вы хотите предоставить общий доступ или отправить документ.
-   Потоковая передача сведений о рабочем столе компьютера на компьютер коллеги для совместного использования ресурсов.
-   Начните играть с сотрудника во время потом в аэропорту Terminal, поддерживающем беспроводное подключение. API PNM позволяет приложению объявить интерес к игре, найти других пользователей в аэропорту, которые также заинтересованы в игре, и запустить игровой сеанс.
-   Объявление интересов на Конференции отрасли и обнаружение других пользователей с помощью беспроводного подключения. После обнаружения пользователи могут общаться и настраивать встречи импровизированном для обмена идеями.
