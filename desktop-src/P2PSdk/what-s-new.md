[Интерфейсы API одноранговой совместной работы](peer-collaboration.md). Эти API предоставляют уровень совместной работы, позволяющий одноранговые клиенты инициировать сеансы совместной работы, обеспечивать бессерверный обмен мгновенными сообщениями и запускать общие сетевые приложения.
