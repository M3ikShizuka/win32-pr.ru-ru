# <a name="graphing-api-structures"></a>Построение диаграмм по структурам API

API-интерфейс для построения диаграмм использует следующие структуры:



| Структура                                                                 | Описание                                                                                                   |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_ \_ \_ данные изменения узла событий \_ в одноранговых узлах**](/windows/desktop/api/P2P/ns-p2p-peer_event_node_change_data)    | Содержит указатель на данные при запуске события **\_ \_ \_ \_ изменения узла однорангового графа** .                  |
| [**\_ \_ синхронизированные данные события однорангового узла \_**](/windows/desktop/api/P2P/ns-p2p-peer_event_synchronized_data)   | Указывает тип синхронизированной записи.                                                      |
| [**данные события однорангового \_ графа \_ \_**](/windows/desktop/api/P2P/ns-p2p-peer_graph_event_data)                 | Содержит данные, связанные с одноранговым событием.                                                                   |
| [**\_ \_ Регистрация события одноранговой диаграммы \_**](/windows/desktop/api/P2P/ns-p2p-peer_graph_event_registration) | Указывает события однорангового узла, для которых приложению требуются уведомления.                                        |
| [**свойства однорангового \_ графа \_**](/windows/desktop/api/P2P/ns-p2p-peer_graph_properties)                  | Содержит данные о политике однорангового графа, идентификатора, области и других сведений.                             |
| [**\_ \_ сведения о одноранговом узле**](/windows/desktop/api/P2P/ns-p2p-peer_node_info)                                | Содержит сведения, относящиеся к определенному узлу в одноранговой диаграмме.                                   |
| [**\_интерфейс безопасности \_ узла**](/windows/desktop/api/P2P/ns-p2p-peer_security_interface)              | Указывает интерфейсы безопасности, которые вызывают API-интерфейсы одноранговой диаграммы, используемые для проверки, защиты и освобождения записей. |
