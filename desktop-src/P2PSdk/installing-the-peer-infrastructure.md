# <a name="installing-the-peer-infrastructure"></a>Установка одноранговой инфраструктуры

Инфраструктура одноранговых сетей Windows — это платформа разработки для создания одноранговых приложений для компьютеров под управлением Windows XP с пакетом обновления 1 (SP1) с расширенным сетевым пакетом, Windows XP с пакетом обновления 2 (SP2), Windows Vista и Windows Server 2008.

На компьютерах под управлением Windows Vista или Windows Server 2008 уже установлена инфраструктура одноранговых сетей Windows. Для компьютеров под управлением Windows XP с пакетом обновления 2 (SP2) необходимо выполнить следующие действия для установки одноранговых сетей Windows.

-   Щелкните **Пуск**, нажмите **Панель управления**, а затем **Установка и удаление программ**.
-   Щелкните элемент **Установка и удаление компонентов Windows**.
-   В окне **компоненты** щелкните **Сетевые службы** (но не выбирайте соответствующий флажок), а затем нажмите кнопку **сведения**.
-   Установите флажок **одноранговый узел** и нажмите кнопку **ОК**.
-   Нажмите кнопку **Далее**, а затем следуйте инструкциям мастера.

Компьютерам под управлением Windows XP с пакетом обновления 1 (SP1) требуется установка [пакета Advanced Networking Pack](https://www.microsoft.com/downloads/details.aspx?FamilyID=e88cc382-8ce6-4739-97c0-1a52a6f005e4), бесплатной загрузки, чтобы обеспечить одноранговую сеть.
