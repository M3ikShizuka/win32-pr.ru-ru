# <a name="installing-the-peer-infrastructure"></a>Установка одноранговой инфраструктуры

инфраструктура Windows одноранговой сети — это платформа разработки для создания одноранговых приложений для компьютеров под управлением Windows XP с пакетом обновления 1 (sp1) с расширенным сетевым пакетом, Windows XP с пакетом обновления 2 (SP2), Windows Vista и Windows Server 2008.

на компьютерах под управлением Windows Vista или Windows Server 2008 уже установлена Windows инфраструктура одноранговых сетей. для компьютеров под управлением Windows XP с пакетом обновления 2 (SP2) для установки Windows одноранговые сети необходимо выполнить следующие действия.

-   Щелкните **Пуск**, нажмите **Панель управления**, а затем **Установка и удаление программ**.
-   щелкните **добавить или удалить Windows компоненты**.
-   В окне **компоненты** щелкните **Сетевые службы** (но не выбирайте соответствующий флажок), а затем нажмите кнопку **сведения**.
-   Установите флажок **одноранговый узел** и нажмите кнопку **ОК**.
-   Нажмите кнопку **Далее**, а затем следуйте инструкциям мастера.

для компьютеров под управлением Windows XP с пакетом обновления 1 (SP1) требуется установка [расширенного сетевого пакета](https://www.microsoft.com/downloads/details.aspx?FamilyID=e88cc382-8ce6-4739-97c0-1a52a6f005e4), бесплатной загрузки, чтобы обеспечить одноранговую сеть.
