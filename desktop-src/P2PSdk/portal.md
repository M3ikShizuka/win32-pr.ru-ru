# <a name="peer-to-peer"></a>Одноранговая сеть

## <a name="purpose"></a>Назначение

Одноранговые технологии используются для упрощения взаимодействия и совместной работы в распределенных сетях в режиме реального времени.

В одноранговой модели без использования серверов Интернета каждый пользователь компьютера может выполнять следующие действия:

-   Exchange данных
-   Общий доступ к ресурсам
-   Обнаружение других пользователей
-   Сообща
-   Непосредственная совместная работа в режиме реального времени

С помощью одноранговых технологий приложения, которые координируют использование циклов ЦП компьютера и хранилища, могут совместно использовать ресурсы в небольших или больших группах компьютеров, подключенных к Интернету.

## <a name="where-applicable"></a>Где применимо

Разработчики могут использовать одноранговую инфраструктуру для создания широкого спектра распределенных, специализированных и одноранговых приложений.

## <a name="developer-audience"></a>Аудитория разработчиков

Разработчики, использующие одноранговую инфраструктуру, должны быть знакомы с концепциями программирования C. Разработчики, использующие поставщик пространства имен Winsock PNRP, должны быть знакомы с API Winsock.

## <a name="run-time-requirements"></a>Требования к среде выполнения

одноранговая инфраструктура поддерживается в Windows Vista, Windows XP с пакетом обновления 2 (sp2) и более поздних версий, а также расширенный сетевой пакет для Windows xp, доступный для Windows xp с пакетом обновления 1 (SP1). Для одноранговой инфраструктуры необходимо установить протокол IPv6 и запустить его, чтобы обеспечить функционирование приложений одноранговой сети. использование одноранговой совместной работы поддерживается только в Windows Vista.
