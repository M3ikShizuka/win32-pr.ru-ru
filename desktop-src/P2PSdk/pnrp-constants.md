# <a name="pnrp-constants"></a>Константы PNRP

API поставщика пространства имен PNRP использует следующие константы.



| Константа                                                 | Описание                                                |
|----------------------------------------------------------|------------------------------------------------------------|
| [**\_ \_ Флаги облака PNRP**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_flags)           | Указывает допустимость имени облака.                    |
| [**\_облачное \_ состояние PNRP**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_state)           | Указывает различные состояния, в которых может находиться облако PNRP.     |
| [**\_критерии разрешения \_ PNRP**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_resolve_criteria) | Указывает критерии, используемые протоколом PNRP для разрешения поиска. |
