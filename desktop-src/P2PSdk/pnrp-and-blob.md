# <a name="pnrp-and-blob"></a>PNRP и BLOB-объект

При вызове нескольких функций PNRP использует структуру [**больших двоичных объектов**](winsock-nsp-reference-links.md) для передачи данных в структуру **всакуерисет** . Для использования с протоколом PNRP структура **большого двоичного объекта** указывает на структуру [**Пнрпинфо**](/windows/desktop/api/Pnrpns/ns-pnrpns-pnrpinfo_v1) или структуру [**пнрпклаудинфо**](/windows/desktop/api/Pnrpns/ns-pnrpns-pnrpcloudinfo) .
