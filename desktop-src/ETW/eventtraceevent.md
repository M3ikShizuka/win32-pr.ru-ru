# <a name="eventtraceevent-class"></a>Класс Евенттрацеевент

Родительский класс для событий заголовка журнала. Этот класс всегда является первым классом трассировки событий, отправляемым потребителю (это событие не отправляется потребителям реального времени).

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
[Guid("{68fdd900-4a3e-11d1-84f4-0000f80464e3}")]
class EventTraceEvent : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Участники

Класс **евенттрацеевент** не определяет никаких членов.

## <a name="requirements"></a>Требования



| Требование | Значение |
|-------------------------------------|------------------------------------------------------|
| Минимальная версия клиента<br/> | Только для \[ классических приложений Windows XP\]<br/>          |
| Минимальная версия сервера<br/> | \[Только для настольных приложений Windows Server 2003\]<br/> |
