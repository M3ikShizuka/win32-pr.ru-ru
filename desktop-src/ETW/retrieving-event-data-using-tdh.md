# <a name="retrieving-event-data-using-tdh"></a>Получение данных о событиях с помощью TDH

Чтобы использовать данные событий, потребитель должен иметь представление о формате данных события. Если поставщик использовал файлы манифеста, MOF или ТМФ для публикации формата данных события, то для анализа данных события можно использовать функции модуля поддержки данных трассировки (TDH). Функции TDH доступны начиная с Windows Vista.

В следующих разделах показано, как использовать TDH для получения данных о событиях и метаданных.

-   [Использование Тдхформатпроперти для использования данных событий](using-tdhformatproperty-to-consume-event-data.md)
-   [Использование Тдхжетпроперти для использования данных событий](using-tdhgetproperty-to-consume-event-data.md)
-   [Получение метаданных события](retrieving-event-metadata.md)
-   [Перечисление поставщиков](enumerating-providers.md)
