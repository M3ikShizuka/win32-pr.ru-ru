# <a name="thread_v1-class"></a>\_Класс Thread v1

Этот класс является родительским для событий потока.

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
[Guid("{3d6fa8d1-fe05-11d0-9dda-00c04fd7ba7c}"), EventVersion(1)]
class Thread_V1 : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Участники

Класс **Thread \_ v1** не определяет никаких членов.
