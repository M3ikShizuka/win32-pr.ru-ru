# <a name="lost_event-class"></a>\_Класс событий Lost

Это родительский класс для событий, потерянных в сеансе в режиме реального времени.

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
[Guid("{6a399ae0-4bc6-4de9-870b-3657f8947e7e}")]
class Lost_Event : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Участники

Класс **\_ событий Lost** не определяет никаких членов.
