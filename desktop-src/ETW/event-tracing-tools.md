# <a name="event-tracing-tools"></a>Средства трассировки событий

Для использования доступны следующие средства трассировки событий.



| Средство     | Описание                                                                                                                                                                                                                                                                                                                                |
|----------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Logman   | Управляет и планирует сбор счетчиков производительности и журналов трассировки событий в локальных и удаленных системах. Это средство можно использовать для перечисления поставщиков, которые [опубликовали макет данных о событиях](publishing-your-event-schema.md) в корневом \\ пространстве имен WMI. Дополнительные сведения об использовании этого средства см. в **центре справки и поддержки**. |
| трацефмт | Форматирует сведения в файле журнала трассировки в удобочитаемой форме. Дополнительные сведения см. в разделе средства для трассировки программного обеспечения в пакете средств разработки драйверов Microsoft Windows (DDK). Используется только трассировкой программного обеспечения WPP.                                                                                                                                      |
| Tracelog | Управляет сеансом трассировки событий. Дополнительные сведения см. в разделе средства для трассировки программного обеспечения в DDK.                                                                                                                                                                                                                                                   |
| трацепдб | Создает файл формата сообщений трассировки, который средство Трацефмт использует для преобразования журнальных сообщений в форму, доступную для восприятия. Дополнительные сведения см. в разделе средства для трассировки программного обеспечения в DDK. Используется только трассировкой программного обеспечения WPP.                                                                                                                            |
| Tracerpt | Обрабатывает журналы трассировки событий или события в режиме реального времени от инструментированных поставщиков трассировки событий и позволяет создавать отчеты анализа трассировки и файлы CSV (с разделителями-запятыми) для создаваемых событий. Дополнительные сведения см. в **центре справки и поддержки**.                                                                                       |
