# <a name="image_v1-class"></a>\_Класс Image v1

Этот класс является родительским для событий загрузки изображения.

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
[Guid("{2cb15d1d-5fc1-11d2-abe1-00a0c911f518}"), EventVersion(1)]
class Image_V1 : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Участники

Класс **image \_ v1** не определяет никаких членов.
