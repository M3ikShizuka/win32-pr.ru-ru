# <a name="writing-wpp-events"></a>Создание событий WPP

процессор Windows software trace (WPP) обеспечивает эффективный механизм записи событий, происходящих во время выполнения приложения или драйвера. Интерфейс WPP использует API ETW.

дополнительные сведения об использовании конвейера wpp для ведения журнала событий см. в разделе часто задаваемые вопросы о [трассировке программного обеспечения](/windows-hardware/drivers/devtest/wpp-software-tracing) и [программном обеспечении](/windows-hardware/drivers/devtest/software-tracing-faq) для Microsoft Windows Driver sdk (DDK).
