# <a name="device-drivers"></a>Драйверы устройств

Эта документация предназначена для приложений пользовательского режима, которые хотят использовать ETW для трассировки событий. Эта документация не предназначена для драйверов устройств, работающих в режиме ядра. Для тех, кто пишет драйверы устройств и приложения в режиме ядра, которые хотят использовать ETW в качестве отладчика с минимальными последствиями для устранения неполадок, связанных со временем, см. сведения об инструментировании драйвера в разделе [Отслеживание событий для Windows](/windows-hardware/drivers/devtest/event-tracing-for-windows--etw-), [Отслеживание программного обеспечения WPP](/windows-hardware/drivers/devtest/wpp-software-tracing)и [Трассировка событий WMI](/windows-hardware/drivers/kernel/wmi-event-tracing) в комплекте драйверов Windows (WDK).
