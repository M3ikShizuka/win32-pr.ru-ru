# <a name="device-drivers"></a>Драйверы устройств

Эта документация предназначена для приложений пользовательского режима, которые хотят использовать ETW для трассировки событий. Эта документация не предназначена для драйверов устройств, работающих в режиме ядра. для тех, кто пишет драйверы устройств и приложения в режиме ядра, которые хотят использовать ETW в качестве отладчика с минимальными последствиями для устранения неполадок, связанных со временем, см. сведения об инструментировании драйвера в разделе [трассировка событий для Windows](/windows-hardware/drivers/devtest/event-tracing-for-windows--etw-), [трассировка программного обеспечения WPP](/windows-hardware/drivers/devtest/wpp-software-tracing)и [трассировка событий WMI](/windows-hardware/drivers/kernel/wmi-event-tracing) в Windowsном наборе драйверов (WDK).
