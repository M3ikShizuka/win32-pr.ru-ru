# <a name="fileio_v0-class"></a>\_Класс FileIo v0

Этот класс является родительским для событий файлового ввода-вывода.

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
[Guid("{90cbdc39-4a3e-11d1-84f4-0000f80464e3}"), EventVersion(0)]
class FileIo_V0 : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Участники

Класс **FileIo \_ v0** не определяет никаких членов.
