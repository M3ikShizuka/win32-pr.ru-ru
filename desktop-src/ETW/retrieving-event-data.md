# <a name="retrieving-event-data"></a>Получение данных о событиях

Чтобы использовать данные событий, потребитель должен иметь представление о формате данных события. Это не является проблемой, если вы используете события от собственного поставщика, так как вы знакомы с форматом данных. Однако для использования события от любого поставщика поставщик должен опубликовать макет данных события. Общие сведения о работе метаданных событий, а также о том, как опубликовать метаданные событий, чтобы другие пользователи могли его использовать, см. в статье [Общие сведения о метаданных событий](event-metadata-overview.md).

Для использования событий в Windows Vista, опубликованных с помощью файлов манифеста, MOF или ТМФ, а также событий Трацелоггинг, см. статью [Получение данных о событиях с помощью TDH](retrieving-event-data-using-tdh.md).

Для использования событий до Windows Vista, опубликованных с помощью MOF, см. статью [Получение данных о событиях с помощью MOF](retrieving-event-data-using-mof.md).
