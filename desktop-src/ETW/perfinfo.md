# <a name="perfinfo-class"></a>Класс Перфинфо

Этот класс является родительским классом для событий счетчика производительности.

Следующий синтаксис упрощен из MOF-кода.

## <a name="syntax"></a>Синтаксис

``` syntax
[Guid("{ce1dbfb4-137e-4da6-87b0-3f59aa102cbc}"), EventVersion(2)]
class PerfInfo : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Члены

Класс **перфинфо** не определяет никаких членов.

## <a name="remarks"></a>Remarks

Чтобы включить события отложенного вызова процедур (DPC) в сеансе ведения журнала ядра NT, при вызове функции [**старттраце**](/windows/win32/api/evntrace/nf-evntrace-starttracea) укажите флаг **\_ трассировки событий \_ \_ DPC** в элементе **енаблефлагс** структуры [**\_ \_ свойств трассировки событий**](/windows/win32/api/evntrace/ns-evntrace-event_trace_properties) . Можно также указать один или несколько из следующих флагов:

-   **\_ \_ прерывание флага трассировки событий \_**
-   **\_ \_ профиль флага трассировки событий \_**
-   **\_ \_ системкалл флага трассировки событий \_**

Потребители трассировки событий могут реализовать специальную обработку событий DPC, вызвав функцию [**сеттрацекаллбакк**](/windows/win32/api/evntrace/nf-evntrace-settracecallback) и указав [**перфинфогуид**](nt-kernel-logger-constants.md) в качестве параметра *пгуид* . Используйте следующие типы событий для обнаружения фактического события при использовании событий.



| Тип события           | Описание                                                                                                          |
|----------------------|----------------------------------------------------------------------------------------------------------------------|
| Значение типа события, 46 | Событие выборки профиля. Класс MOF [**сампледпрофиле**](sampledprofile.md) определяет данные события для этого события. |
| Значение типа события, 51 | Событие ввода системного вызова. Класс MOF [**сискаллентер**](syscallenter.md) определяет данные события для этого события.   |
| Значение типа события, 52 | Событие выхода из системного вызова. Класс MOF [**сискаллексит**](syscallexit.md) определяет данные события для этого события.      |
| Значение типа события, 66 | Потоковое событие DPC. Класс [**DPC**](dpc.md) MOF определяет данные события для этого события.                          |
| Значение типа события, 67 | Событие "процедура службы прерываний" (ISR). Класс MOF [**ISR**](isr.md) определяет данные события для этого события.       |
| Значение типа события, 68 | Событие DPC. Класс [**DPC**](dpc.md) MOF определяет данные события для этого события.                                   |
| Значение типа события, 69 | Событие таймера DPC. Класс [**DPC**](dpc.md) MOF определяет данные события для этого события.                             |
