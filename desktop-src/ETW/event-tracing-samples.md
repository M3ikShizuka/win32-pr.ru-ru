# <a name="event-tracing-samples"></a>Примеры трассировки событий

Пакет средств разработки программного обеспечения платформы (SDK) содержит полные примеры трассировки событий. Эти примеры находятся в каталоге Samples \\ винбасе \\ евенттраце. Корень этого пути является базовым каталогом установки PSDK. Вы можете скачать PSDK из [https://www.microsoft.com/downloads/details.aspx?FamilyId=A55B6B43-E24F-4EA3-A93E-40C0EC4F68E5](https://msdn.microsoft.com/windowsserver/bb980924.aspx) .

В следующей таблице описаны примеры, содержащиеся в каталоге Samples \\ винбасе \\ евенттраце.



| Образец   | Описание                                                                                                                            |
|----------|----------------------------------------------------------------------------------------------------------------------------------------|
| трацедмп | Потребитель трассировки событий. Он декодирует данные события с помощью сведений о формате, полученных из инструментария WMI, и выводит данные в CSV-файл. |
| трацедп  | Использует API поставщика трассировки событий для предоставления данных трассировки событий средству ведения журнала или потребителю.                                             |
| TraceLog | Использует API контроллера трассировки событий для управления сеансами ведения журнала.                                                                        |



 

Подробные описания см. в файле Readme.txt, прилагаемом к этим примерам.
