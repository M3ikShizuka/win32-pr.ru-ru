# <a name="device-host-programming-considerations"></a>Рекомендации по программированию на узле устройств

Пустой URL-адрес презентации в описании устройства не разрешен. Если у устройства нет URL-адреса презентации, удалите теги URL-адресов презентации из описания устройства.
