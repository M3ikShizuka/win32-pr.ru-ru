# <a name="control-point-programming-considerations"></a>Вопросы программирования контрольной точки

Разработчики, создающие приложения на основе UPnP (или контрольные точки), должны использовать эти приложения из \_ клиента апартментсреадед. Существуют известные проблемы при использовании API управляющей точки из \_ многопоточного клиента, выполняющегося под нагрузкой.
