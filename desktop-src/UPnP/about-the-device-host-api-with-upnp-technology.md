# <a name="about-the-device-host-api"></a>Сведения об API узла устройства

API хоста устройств с технологией UPnP — это платформа для реализации функций устройств на основе UPnP на платформе Windows. Разработчики, создающие устройства с помощью API узла устройства с технологией UPnP (называемые [*размещенными устройствами*](h-gly.md)), должны только реализовать основные функциональные возможности устройства. Разработчики могут полагаться на узел устройства для работы с специфичными для UPnP сведениями об обнаружении, описании, управлении, событиях и презентации. Узел устройства проверяет входящие данные от клиентов, основанных на UPnP, и форматирует все исходящие данные с размещенных устройств в соответствии с архитектурой UPnP-устройств.
