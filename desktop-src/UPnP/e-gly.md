# <a name="e-upnp-apis"></a>E (API-интерфейсы UPnP)

Содержит термины, связанные с UPnP, которые начинаются с буквы E.

[A](a-gly.md) B [C](c-gly.md) [D](d-gly.md) E F G [р](h-gly.md) . J K L M [N](n-gly.md) O [P](p-gly.md) Q [R](r-gly.md) [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.e_1_gly"></span><span id="UPNP.E_1_GLY"></span>**журнале**
</dt> <dd>

Событие — это уведомление одного или нескольких изменений в переменных состояния, предоставляемых службой.

</dd> <dt>

<span id="upnp.e_2_gly"></span><span id="UPNP.E_2_GLY"></span>**переменная с событием**
</dt> <dd>

Переменная с событием — это переменная состояния для службы, которая при изменении активирует событие. Все контрольные точки, подписанные на источник событий переменной, получают уведомление об изменении. Переменная состояния без событий никогда не включается в уведомление о событии.

</dd> </dl>
