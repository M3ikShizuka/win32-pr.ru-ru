# <a name="r-upnp-apis"></a>R (API-интерфейсы UPnP)

Содержит термины, относящиеся к стандарту UPnP и начинающиеся с буквы R.

<dl> <dt>

<span id="upnp.r_1_gly"></span><span id="UPNP.R_1_GLY"></span>**Каталог ресурсов**
</dt> <dd>

Каталог ресурсов — это локальный каталог для устройства, где находятся файлы, такие как файлы значков.

</dd> <dt>

<span id="upnp.r_2_gly"></span><span id="UPNP.R_2_GLY"></span>**корневое устройство**
</dt> <dd>

Корневое устройство — это самое верхнее устройство в дереве устройств. Например, при обращении к устройству для комбинирования ВИДЕОМАГНИТОФОНА или телевизора, называемой станцией просмотра, существует три устройства: станция просмотра, ВИДЕОМАГНИТОФОН и телевизор. Станция просмотра считается корневым устройством.

</dd> <dt>

<span id="upnp.r_3_gly"></span><span id="UPNP.R_3_GLY"></span>**выполняющееся устройство**
</dt> <dd>

Запущенное устройство — это устройство, для которого уже был загружен код перед регистрацией.

Это устройства, для которых описание службы неизвестно до регистрации.

Примерами таких устройств являются мосты, например мост SCP. Этот мост регистрирует выполняющиеся устройства на узле устройства.

</dd> </dl>
