# <a name="whats-new-in-the-upnp-apis"></a>Новые возможности API-интерфейсов UPnP

api-интерфейсы™ UPnP имеют новые функции и обновленную документацию по Windows XP с пакетом обновления 2 (SP2). В следующей таблице приведены сведения о новой документации.



| Компонент                                                          | Описание                                                                                                                                                                                                                                                                                                                                                                   |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**иупнпремотиндпоинтинфо**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpremoteendpointinfo)       | Этот новый интерфейс позволяет размещенному устройству получать сведения об инициаторе запроса (то есть контрольной точке) и запросе. Он включает три метода, каждый из которых предоставляет разный тип информации.                                                                                                                                                              |
| [Реализация поведения устройства](implementing-device-behavior.md) | Этот раздел содержит новый раздел, в котором объясняется, как получить сведения о конечной точке с помощью нового интерфейса [**иупнпремотиндпоинтинфо**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpremoteendpointinfo) .                                                                                                                                                                                                     |
| [Параметры конфигурации](configuration-settings.md)             | Этот новый раздел содержит сведения о параметрах реестра, влияющих на поведение API-интерфейсов UPnP.                                                                                                                                                                                                                                                                    |
| [Коды ошибок устройства](device-error-codes.md)                     | В этом новом разделе объясняется, как код ошибки из устройства преобразуется в значение HRESULT и наоборот. Понимание этого процесса необходимо для того, чтобы оценить значение HRESULT, возвращаемое методами [**иупнпсервице:: инвокеактион**](/windows/desktop/api/Upnp/nf-upnp-iupnpservice-invokeaction) и [**Иупнпсервице:: куеристатевариабле**](/windows/desktop/api/Upnp/nf-upnp-iupnpservice-querystatevariable) . |
