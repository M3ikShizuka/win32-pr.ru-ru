# <a name="what-is-the-control-point-object-model"></a>Что такое объектная модель контрольной точки?

На следующем рисунке показана базовая объектная модель контрольной точки.

![Объектная модель точки управления](images/upnp-object-model.png)

Поиск устройств с помощью интерфейса поиска устройств создает коллекцию устройств. Коллекция устройств содержит ноль или более объектов устройств. Приложения могут использовать различные методы коллекции устройств для доступа к отдельным объектам устройств.

Объекты устройств всегда содержат коллекцию служб, содержащую один или несколько объектов службы. Эти объекты службы используются приложениями для взаимодействия с устройствами и управления им.
