# <a name="controlling-a-device"></a>Управление устройством

После обнаружения устройств их можно контролировать.

**Просмотр свойств устройства**

1.  Выберите устройство в списке **найденные устройства** .
2.  Щелкните **Свойства устройства**. Откроется окно **Свойства устройства** с запрошенными сведениями.

> [!Note]  
> Функция **просмотра представления** недоступна в примере кода C++.

 

**Просмотр страницы презентации устройства**

1.  Выберите устройство в списке **найденные устройства** .
2.  Нажмите кнопку **Просмотреть презентацию**. Откроется окно Internet Explorer с запрошенной страницей презентации.

> [!Note]  
> Функция **View Service DESC.** функциональная возможность недоступна в примере кода C++.

 

**Просмотр описания службы**

1.  URL-адрес описания службы можно ввести в поле **Service DESC. Поле URL-адреса** .

    ![URL-адрес описания службы](images/ucp-url.png)

2.  Щелкните **View Service DESC.** Откроется окно **средства просмотра описания службы** . Затем можно просмотреть описание службы с помощью древовидного представления. Эта функция недоступна в примере кода C++.

    ![окно средства просмотра описания службы](images/ucp-serv.png)

    В окне средства просмотра описания служб можно использовать пять команд.



| Кнопка                 | Выполненное действие                                                                                                                                                                      |
|------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Go                     | Загружает файл, показанный в **службе DESC. Поле URL-адреса** .                                                                                                                              |
| Задание действия             | Выберите имя действия в дереве описания службы и щелкните **задать действие**. Имя действия указывается в поле действие при **вызове** основного **универсального окна управления служебной программой** .       |
| Установка значения переменной           | Выберите имя переменной в дереве описания службы и нажмите кнопку **задать переменную**. Имя переменной задается в поле **переменная запроса** основного **универсального окна управления служебной программой (UCP** ). |
| Заполнить список переменных | Загружает все имена переменных службы в список **переменных запроса** основного **универсального окна управления служебной программой (UCP** ).                                                                           |
| Заполнение списка действий   | Загружает все имена действий службы в список **действий Invoke** основного **универсального окна управления служебной программой** .                                                                              |



 

**Управление устройством**

1.  В списке **обнаруженные устройства** выберите устройство, которое требуется контролировать.
2.  В списке **выберите службу** выберите службу, которую нужно вызвать, или переменную состояния, к которой необходимо выполнить запрос.

    ![окно управления устройствами](images/ucp-contr.png)

    > [!Note]  
    > Содержимое **списка служб** основано на устройстве, выбранном в списке **найденные устройства** .

     

3.  Если вы хотите узнать значение переменной состояния для выбранной службы, введите имя переменной в поле " **переменная запроса** " для параметра "служба". Нажмите кнопку **Запрос**. Результаты отображаются в поле **значение состояния запроса/аргументы исходящих действий** .
4.  Если вы хотите вызвать действие для службы, введите имя действия в поле **Действие вызова** и любые аргументы в поле **аргументы действия** . Нажмите кнопку **вызвать**. Результаты отображаются в поле **значение состояния запроса или аргументы исходящих действий** (при наличии).

    Возвращаемое значение, если таковое имеется, содержится в первом аргументе out.

    > [!Note]  
    > Если для действия существует несколько аргументов, разделите их пробелами.

     

5.  Просмотрите сведения о событиях в поле **события** для выбранной службы.
