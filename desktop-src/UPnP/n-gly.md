<span id="upnp.n_1_gly"></span><span id="UPNP.N_1_GLY"></span>**устройство, не работающее**
</dt> <dd>

Неработающее устройство — это устройство, для которого код загружается только при получении запроса на это устройство. Часто это устройства, для которых описания служб известны до регистрации.

Типичным примером неработающего устройства является мягкое устройство, такое как устройство проигрывателя мультимедиа.
