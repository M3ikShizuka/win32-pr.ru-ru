# <a name="about-the-control-point-api"></a>Сведения об API контрольной точки

Технология UPnP обеспечивает динамическую сеть интеллектуальных устройств, ПК и других контрольных точек. Динамические сетевые подключения доступны на различных физических носителях. Реализация API контрольной точки в корпорации Майкрософт с технологией UPnP состоит из набора COM-объектов, которые используются для создания приложений и веб-страниц, позволяющих находить и контролировать устройства на основе UPnP. Например, видеомагнитофон с поддержкой UPnP можно контролировать с веб-страницы, специально предназначенной для управления ей. Чтобы расширить эту концепцию, разработчики могут управлять многими устройствами из центрального, удаленного или однорангового расположения.

Приложения, созданные с помощью API контрольной точки с технологией UPnP, находятся в диапазоне сценариев, внедренных в HTML-страницы, в полноценные программы на языке C++. API контрольной точки с технологией UPnP поддерживает microsoft Visual Basic разработки, microsoft Visual Basic scripting Edition (VBScript) и C++.
