# <a name="device-host-api-reference"></a>Справочник по API узла устройства

Следующие интерфейсы являются частью API узла устройства с технологией UPnP. Узел устройства с технологией UPnP поддерживает Microsoft Visual Basic и C++.

Этот API не обеспечивает поддержку Microsoft Visual Basic Scripting Edition (VBScript).



| Интерфейс                                                  | Описание                                                                                         |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [**иупнпдевицеконтрол**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpdevicecontrol)           | Используется узлом устройства для управления устройствами и связанными службами.                                     |
| [**иупнпдевицепровидер**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpdeviceprovider)         | Используется узлом устройства для запуска и завершения поставщиков устройств.                                         |
| [**иупнпевентсинк**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpeventsink)                   | Используется устройствами для отправки уведомлений о событиях на узел устройства.                                     |
| [**иупнпевентсаурце**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpeventsource)               | Используется узлом устройства для подписки на события, предоставляемых размещенной службой, и отмены подписки на них.       |
| [**иупнпрегистрар**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpregistrar)                   | Используется устройствами для регистрации на узле устройства.                                                   |
| [**иупнпремотиндпоинтинфо**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpremoteendpointinfo) | Используется устройствами для получения сведений об инициаторе запроса (то есть контрольной точке) и запросе. |
| [**иупнпререгистрар**](/windows/desktop/api/Upnphost/nn-upnphost-iupnpreregistrar)               | Используется устройствами для повторной регистрации на узле устройства.                                                |
