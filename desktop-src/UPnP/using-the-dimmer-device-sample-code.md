# <a name="using-the-dimmer-device-sample-code"></a>Использование примера кода устройства DIMM

Пакет средств разработки программного обеспечения платформы (SDK) включает в себя пример кода устройства DIMM. В этом примере кода демонстрируется простое устройство на основе UPnP, реализованное с помощью API узла устройства с технологией UPnP.

Каталог ДКО \_ диммерсервице содержит реализацию для .dll устройства. Каталог RegisterDevice содержит реализацию для регистрации устройства .dll со службой регистратора UPnP на компьютере. Каталог Степбистеп содержит проекты VC 6.0, соответствующие «пошаговому руководству», которые содержатся в файле Readme.txt. «Пошаговое пошаговое руководством» предназначено для разработчиков, которые не имеют опыта работы с ATL и COM.
