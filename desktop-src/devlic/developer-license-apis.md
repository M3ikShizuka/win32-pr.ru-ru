# <a name="developer-license-apis"></a>API лицензий разработчика

Используйте эти API для проверки, получения и удаления лицензии разработчика.

## <a name="in-this-section"></a>В этом разделе



| Раздел                                                                 | Описание                                             |
|-----------------------------------------------------------------------|---------------------------------------------------------|
| [**чеккдевелоперлиценсе**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-checkdeveloperlicense)<br/>     | Проверяет, существует ли лицензия разработчика.<br/> |
| [**аккуиредевелоперлиценсе**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-acquiredeveloperlicense)<br/> | Получает лицензию разработчика.<br/>                |
| [**ремоведевелоперлиценсе**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-removedeveloperlicense)<br/>   | Удаляет лицензию разработчика.<br/>                 |
